# 深度学习数学基础

## 一、线性代数

### 1.1 向量与矩阵基础

深度学习中数据通常表示为张量（Tensor），其数学基础是线性代数。

**核心概念：**

- **向量**：n维空间中的有向线段，深度学习中表示特征
- **矩阵**：线性变换的表示，神经网络的权重矩阵
- **张量**：高维数组的泛化，0阶张量=标量，1阶张量=向量，2阶张量=矩阵

**关键运算：**

| 运算 | 符号 | 应用场景 |
|------|------|----------|
| 矩阵乘法 | $C = AB$ | 前向传播 |
| 转置 | $A^T$ | 反向传播 |
| Hadamard积 | $A \odot B$ | 逐元素运算 |
| 点积 | $\langle a, b \rangle$ | 相似度计算 |

### 1.2 特征值分解与SVD

- **特征值分解**：$Av = \lambda v$，用于PCA降维
- **SVD**：$A = U\Sigma V^T$，用于矩阵近似、推荐系统
- **正定矩阵**：所有特征值大于0，保证优化问题的凸性

### 1.3 范数与距离

- L1范数：$\|x\|_1 = \sum |x_i|$（稀疏性，LASSO正则化）
- L2范数：$\|x\|_2 = \sqrt{\sum x_i^2}$（权重衰减，Ridge正则化）
- Frobenius范数：$\|A\|_F = \sqrt{\sum a_{ij}^2}$（矩阵的L2范数）

## 二、概率与统计

### 2.1 概率基础

- **条件概率**：$P(A|B) = P(A \cap B) / P(B)$
- **贝叶斯定理**：$P(\theta | D) = P(D|\theta)P(\theta) / P(D)$
- **最大似然估计(MLE)**：$\theta^* = \arg\max_\theta P(D|\theta)$
- **最大后验估计(MAP)**：$\theta^* = \arg\max_\theta P(\theta|D)$

### 2.2 常见分布

| 分布 | 公式 | 应用 |
|------|------|------|
| 伯努利 | $P(x) = p^x(1-p)^{1-x}$ | 二分类 |
| 高斯 | $\mathcal{N}(\mu, \sigma^2)$ | 初始化、VAE |
| Softmax | $p_i = e^{z_i}/\sum e^{z_j}$ | 多分类输出 |
| 分类分布 | $Cat(p_1,...,p_K)$ | 采样 |

### 2.3 信息论

- **熵**：$H(X) = -\sum p(x)\log p(x)$（不确定性度量）
- **交叉熵**：$H(p,q) = -\sum p(x)\log q(x)$（分类损失函数）
- **KL散度**：$D_{KL}(p\|q) = \sum p(x)\log\frac{p(x)}{q(x)}$（分布差异）

## 三、微积分与优化

### 3.1 导数与梯度

- **链式法则**：反向传播的核心，$\frac{\partial L}{\partial w} = \frac{\partial L}{\partial z} \cdot \frac{\partial z}{\partial w}$
- **雅可比矩阵**：向量函数的导数矩阵
- **Hessian矩阵**：二阶导数矩阵，用于二阶优化

### 3.2 梯度下降

$$\theta_{t+1} = \theta_t - \eta \nabla_\theta L(\theta_t)$$

变体：
- **SGD**：随机梯度下降，每次用mini-batch
- **Momentum**：$v_t = \beta v_{t-1} + \eta \nabla L$
- **Adam**：结合动量和自适应学习率

### 3.3 凸优化

- **凸函数**：$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$
- **凸优化问题**：局部最优=全局最优
- **拉格朗日乘子法**：处理约束优化

## 四、关键数学直觉

1. **维度匹配**：矩阵乘法要求内维相等，影响网络层设计
2. **梯度方向**：梯度指向函数增长最快的方向
3. **概率归一化**：Softmax将任意实数映射为概率分布
4. **正则化的几何解释**：L1在坐标轴上有尖点，促进稀疏；L2球形约束，均匀缩小权重
