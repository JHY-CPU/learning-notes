# 深度学习笔记

本模块系统整理深度学习的核心理论与实践知识，涵盖从数学基础到前沿模型的完整学习路径。

## 目录

| 章节 | 目录 | 主题 |
|------|------|------|
| 01 | [数学基础](01_数学基础/) | 线性代数、概率统计、微积分、优化理论 |
| 02 | [机器学习基础](02_机器学习基础/) | 监督学习、无监督学习、偏差方差、正则化 |
| 03 | [神经网络核心](03_神经网络核心/) | 前馈网络、反向传播、激活函数、损失函数、优化器 |
| 04 | [计算机视觉](04_计算机视觉/) | CNN架构、目标检测、图像分割、GAN、ViT |
| 05 | [NLP与序列模型](05_NLP与序列模型/) | RNN、LSTM、Seq2Seq、Attention、BERT、GPT |
| 06 | [生成式AI](06_生成式AI/) | VAE、GAN、Diffusion、LLM、Stable Diffusion |
| 07 | [强化学习](07_强化学习/) | MDP、Q-Learning、Policy Gradient、PPO、RLHF |
| 08 | [工程与部署](08_工程与部署/) | 框架对比、训练技巧、分布式训练、模型压缩、推理部署 |

## 学习路线

```
数学基础 → 机器学习基础 → 神经网络核心
    ↓                          ↓
  专项应用                    工程部署
(CV / NLP / RL / 生成式)    (框架 / 优化 / 部署)
```
