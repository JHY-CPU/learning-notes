# 机器学习基础

## 一、学习范式

### 1.1 监督学习

给定标注数据 $\{(x_i, y_i)\}_{i=1}^N$，学习映射 $f: X \rightarrow Y$。

| 任务 | 输出 | 损失函数 | 示例 |
|------|------|----------|------|
| 分类 | 离散标签 | 交叉熵 | 图像分类 |
| 回归 | 连续值 | MSE/MAE | 房价预测 |

**常见算法：**
- 线性回归/逻辑回归
- 决策树/随机森林
- SVM（支持向量机）
- K近邻（KNN）
- 朴素贝叶斯

### 1.2 无监督学习

没有标注数据，发现数据的内在结构。

- **聚类**：K-Means、DBSCAN、层次聚类
- **降维**：PCA、t-SNE、UMAP
- **密度估计**：GMM、VAE
- **异常检测**：Isolation Forest、Autoencoder

### 1.3 强化学习

智能体在环境中通过试错学习最优策略。

- 核心要素：状态、动作、奖励、策略
- 方法：Q-Learning、Policy Gradient、Actor-Critic

## 二、核心概念

### 2.1 偏差-方差权衡

$$\text{Error} = \text{Bias}^2 + \text{Variance} + \text{Noise}$$

- **偏差(Bias)**：模型预测与真实值的偏离，高偏差=欠拟合
- **方差(Variance)**：模型对训练数据的敏感度，高方差=过拟合
- **目标**：找到偏差与方差的最佳平衡点

### 2.2 过拟合与正则化

过拟合的表现：训练误差远小于测试误差。

**解决方法：**
1. **L1正则化**：$L = L_0 + \lambda \sum |w_i|$（稀疏化）
2. **L2正则化**：$L = L_0 + \lambda \sum w_i^2$（权重衰减）
3. **Dropout**：训练时随机丢弃神经元
4. **早停(Early Stopping)**：监控验证集损失
5. **数据增强**：扩充训练数据

### 2.3 交叉验证

- **K折交叉验证**：将数据分为K份，轮流做验证集
- **留一法(LOO)**：K=N，每次留一个样本
- **分层K折**：保持每折中类别比例一致

### 2.4 评估指标

**分类：**
- Accuracy（准确率）
- Precision（精确率）、Recall（召回率）、F1-Score
- AUC-ROC曲线
- 混淆矩阵

**回归：**
- MSE / RMSE / MAE
- R² Score
- MAPE

## 三、特征工程

### 3.1 数据预处理

- **标准化**：$z = (x - \mu) / \sigma$（Z-score）
- **归一化**：$x' = (x - x_{min}) / (x_{max} - x_{min})$（Min-Max）
- **缺失值处理**：均值填充、KNN填充、删除
- **类别编码**：One-Hot、Label Encoding、Target Encoding

### 3.2 特征选择

- 过滤法：方差阈值、相关系数、互信息
- 包裹法：递归特征消除（RFE）
- 嵌入法：L1正则化、树模型特征重要性

---

## 四、模型选择流程

```
数据收集 → 数据清洗 → 特征工程 → 模型选择
    ↓
交叉验证 → 超参数调优 → 模型评估 → 模型部署
    ↑___________________|
      (不满足性能要求)
```

**无免费午餐定理**：没有一种算法在所有问题上都最优。需要根据数据特点、问题类型选择合适模型。
