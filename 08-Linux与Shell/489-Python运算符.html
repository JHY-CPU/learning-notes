🧮 Python 运算符

算术/比较/逻辑/赋值/位/身份/成员运算符、优先级、短路求值。

算术运算符

// ========== 基本算术 ========== a, b = 10, 3 a + b # 13 (加) a - b # 7 (减) a * b # 30 (乘) a / b # 3.333... (除,总是浮点数) a // b # 3 (整除) a % b # 1 (取余) a ** b # 1000 (幂) // ========== 除法细节 ========== 10 / 3 # 3.3333333333333335 (浮点) 10 // 3 # 3 (向下取整) -10 // 3 # -4 (注意: 向下取整!) 10 % 3 # 1 -10 % 3 # 2 (余数非负) // 整除 + 余数 = 被除数 // 3 * 3 + 1 = 10 // ========== 增强赋值 ========== x = 10 x += 3 # x = 13 x -= 2 # x = 11 x *= 2 # x = 22 x /= 4 # x = 5.5 x //= 2 # x = 2.0 x %= 3 # x = 2.0 x **= 3 # x = 8.0 // ========== 数学函数 ========== import math math.sqrt(16) # 4.0 math.ceil(3.2) # 4 (向上取整) math.floor(3.8) # 3 (向下取整) math.pow(2, 3) # 8.0 math.pi # 3.14159... abs(-5) # 5 (绝对值) pow(2, 3, 5) # 3 (2^3 % 5 = 8 % 5 = 3) divmod(10, 3) # (3, 1) (商和余数)

比较运算符

// ========== 比较运算 ========== a, b = 5, 10 a == b # False (等于) a != b # True (不等于) a < b # True (小于) a > b # False (大于) a <= b # True (小于等于) a >= b # False (大于等于) // ========== 链式比较 ========== // Python 支持连续比较 (其他语言少见) x = 15 1 < x < 20 # True (1 < x and x < 20) 10 <= x < 20 # True 5 < x < 10 # False // ========== 字符串比较 ========== "apple" < "banana" # True (字典序) "Hello" == "hello" # False (区分大小写) "10" > "2" # False! (字符串比较: "1" < "2") // ========== 浮点比较 ========== 0.1 + 0.2 == 0.3 # False! (精度问题) // 正确方式: import math math.isclose(0.1 + 0.2, 0.3) # True // 或自己设容忍度: abs((0.1 + 0.2) - 0.3) < 1e-9 # True // ========== 相等 vs 身份 ========== // == 比较"值" (相等) // is 比较"身份" (内存地址) a = [1, 2, 3] b = [1, 2, 3] a == b # True (值相同) a is b # False (不同对象) a is not b # True c = a a is c # True (同一对象) // is 常用于 None 比较: result = None result is None # True (推荐) result == None # True (不推荐)

逻辑运算符

// ========== and / or / not ========== True and True # True True and False # False True or False # True False or False # False not True # False // ========== 短路求值 ========== // and: 遇到第一个 False 就停止 // or: 遇到第一个 True 就停止 False and print("不会执行") # 短路,不执行 print True or print("不会执行") # 短路 // ========== 短路用途 ========== // 默认值 name = user_input or "default" # 如果 user_input 空,用默认值 // 条件执行 is_admin and delete_user() # 只有 admin 才执行 // 安全检查 file and file.read() # file 不为 None 才读 // ========== 逻辑运算的返回值 ========== // and/or 返回最后一个被求值的值 (不一定是布尔值) 0 and 42 # 0 (0 是假,停止) 1 and 42 # 42 (都真,返回最后一个) 0 or 42 # 42 (0 是假,返回 42) 1 or 42 # 1 (1 是真,短路) // 常用于: value = some_list or [1, 2, 3] # 空列表时用默认值 path = os.getenv("PATH") or "/usr/bin" // ========== 优先级 ========== // not > and > or True or False and False # True (相当于 True or (False and False)) // 建议加括号提高可读性: (True or False) and False # False

位运算符与成员运算符

// ========== 位运算符 ========== a, b = 5 (0b0101), 3 (0b0011) a & b # 1 (0b0001) AND a | b # 7 (0b0111) OR a ^ b # 6 (0b0110) XOR ~a # -6 (按位取反) a << 1 # 10 (左移 1 位, 乘 2) a >> 1 # 2 (右移 1 位, 整除 2) // 用途: // 权限控制 (标志位) READ = 4 # 0b100 WRITE = 2 # 0b010 EXEC = 1 # 0b001 perm = READ | WRITE # 6 (0b110) perm & READ # True (有读权限) perm & EXEC # False (无执行权限) // ========== 成员运算符 ========== "he" in "hello" # True (字符串包含) 3 in [1, 2, 3] # True (列表包含) "x" in {"x": 1, "y": 2} # True (字典键) "x" not in [1, 2, 3] # True (不包含) // ========== 运算符优先级 (高→低) ========== // ** // +x, -x, ~x (一元运算符) // *, /, //, % // +, - (二元) // <<, >> // & // ^ // | // ==, !=, <, >, <=, >=, is, in // not // and // or // 建议: 复杂表达式加括号,不要依赖优先级记忆
💡 Python 运算符要点: (1) / 总是返回浮点数,// 是向下取整; (2) is 用于 None 比较,== 用于值比较; (3) and/or 的短路特性可用于简洁的默认值; (4) 浮点比较用 math.isclose() 而非 ==; (5) 链式比较 1 < x < 10 是 Python 特有语法。
