# 等价类划分与边界值分析


## 一、等价类划分（Equivalence Partitioning）


将输入数据划分为若干等价类，从每个等价类中选取一个代表值作为测试用例。假设同一等价类中的数据以相同方式处理。


### 1.1 划分原则


| 类型 | 说明 | 示例（输入范围1~100） |
| --- | --- | --- |
| 有效等价类 | 合理的、有意义的输入数据 | [1, 100] 内的整数 |
| 无效等价类 | 不合理的、无意义的输入数据 | <1 的数、>100 的数、非数字 |


### 1.2 划分方法


- **按范围划分**
   ：如果输入在[a,b]范围内，则有效等价类为[a,b]，无效为(∞,a)和(b,+∞)
- **按个数划分**
   ：如果输入n个值，则有效等价类为n个，无效为少于n个和多于n个
- **按集合划分**
   ：如果输入取值于集合S，则有效等价类为S，无效为非S
- **按布尔划分**
   ：如果输入为布尔值，则真和假各为一个等价类


### 1.3 示例：三角形问题


$$
输入：三条边a, b, c（1~100的整数）
                输出：等边三角形、等腰三角形、一般三角形、非三角形
$$


| 输入 | 有效等价类 | 无效等价类 |
| --- | --- | --- |
| 边a | [1, 100]的整数（有效） | <1 或 >100 或 非整数（无效） |
| 边b | [1, 100]的整数（有效） | <1 或 >100 或 非整数（无效） |
| 边c | [1, 100]的整数（有效） | <1 或 >100 或 非整数（无效） |
| 三角形关系 | a+b>c, a+c>b, b+c>a | 不满足三角形不等式 |
| 三角形类型 | a=b=c（等边）
a=b≠c（等腰）
a≠b≠c（一般） | - |


### 1.4 测试用例设计


| 编号 | a | b | c | 预期输出 | 覆盖的等价类 |
| --- | --- | --- | --- | --- | --- |
| 1 | 5 | 5 | 5 | 等边三角形 | 有效 + 等边 |
| 2 | 5 | 5 | 3 | 等腰三角形 | 有效 + 等腰 |
| 3 | 3 | 4 | 5 | 一般三角形 | 有效 + 一般 |
| 4 | 1 | 2 | 10 | 非三角形 | 有效 + 非三角形 |
| 5 | 0 | 5 | 5 | 无效输入 | 无效（a<1） |
| 6 | 5 | 101 | 5 | 无效输入 | 无效（b>100） |


## 二、边界值分析（Boundary Value Analysis）


大量错误发生在输入域的边界上，因此在等价类边界附近选取测试数据。


### 2.1 基本原则


- 如果输入范围是[a, b]，测试 a-1, a, a+1, b-1, b, b+1
- 如果输入个数是n，测试 n-1, n, n+1
- 如果输入是有序集合，测试第一个和最后一个元素
- 对每个变量都要做边界值分析，然后组合


### 2.2 示例：函数 f(x) = x/10，x范围[1, 100]


| x值 | 类型 | 预期输出 | 说明 |
| --- | --- | --- | --- |
| 0 | 边界外 | 无效 | 下界-1 |
| 1 | 边界上 | 0.1 | 下界 |
| 2 | 边界内 | 0.2 | 下界+1 |
| 99 | 边界内 | 9.9 | 上界-1 |
| 100 | 边界上 | 10 | 上界 |
| 101 | 边界外 | 无效 | 上界+1 |


### 2.3 健壮性测试


在边界值分析的基础上，扩展到无效等价类的边界，测试程序的容错能力。


$$
对n个变量的函数，标准边界值分析需要 4n+1 个测试用例
                健壮性边界值分析需要 6n+1 个测试用例
                最坏情况边界值分析需要 5^n 个测试用例（全排列）
$$


## 三、决策表测试（Decision Table Testing）


决策表是表示条件组合和对应动作的表格，适合处理多条件组合的逻辑。


### 3.1 决策表结构


| 组成部分 | 说明 |
| --- | --- |
| 条件桩 | 列出所有条件 | if语句中的条件 |
| 动作桩 | 列出所有可能的动作 | 执行的操作 |
| 条件项 | 条件的取值组合 | T/F/-（不关心） |
| 动作项 | 满足条件时执行的动作 | X 表示执行 |


### 3.2 示例：打印机决策表


| 规则 | R1 | R2 | R3 | R4 | R5 |
| --- | --- | --- | --- | --- | --- |
| 打印机是否故障 | F | T | F | F | F |
| 文件是否已发送 | T | - | T | T | F |
| 纸张是否充足 | T | - | F | T | - |
| 墨粉是否充足 | T | - | - | F | - |
|  |
| 打印文件 | X |  |  |  |  |
| 显示故障信息 |  | X |  |  |  |
| 显示缺纸信息 |  |  | X |  |  |
| 显示缺墨信息 |  |  |  | X |  |
| 显示未收到文件 |  |  |  |  | X |


## 四、知识要点总结


1. 等价类划分：有效等价类 + 无效等价类
2. 边界值分析：在边界附近选取测试数据（a-1, a, a+1, b-1, b, b+1）
3. 边界值分析是对等价类划分的补充，两者结合使用效果更好
4. 决策表适合处理多条件组合的逻辑
5. n个变量的边界值分析：标准4n+1，健壮性6n+1
