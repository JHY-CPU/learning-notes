量子纠缠与量子隐形传态

量子计算量子纠缠量子通信

量子纠缠是量子力学最奇特的现象之一，也是量子计算和量子通信的核心资源。

Bell 态（贝尔态）

Bell 态是两个量子比特的最大纠缠态，共有四种。

四种 Bell 态： ┌─────────────────────────────────────────────────┐ │ |Φ⁺⟩ = (1/√2)(|00⟩ + |11⟩) │ │ |Φ⁻⟩ = (1/√2)(|00⟩ - |11⟩) │ │ |Ψ⁺⟩ = (1/√2)(|01⟩ + |10⟩) │ │ |Ψ⁻⟩ = (1/√2)(|01⟩ - |10⟩) │ │ │ │ 特性： │ │ - 每个 Bell 态都是最大纠缠态 │ │ - 测量一个粒子会立即确定另一个粒子的状态 │ │ - 四个 Bell 态构成二维希尔伯特空间的正交基 │ └─────────────────────────────────────────────────┘ 制备 Bell 态的电路： |0⟩ ──[H]──●── → 量子比特 A │ |0⟩ ──────[X]── → 量子比特 B 步骤： 1. |00⟩ 2. 对第一个量子比特施加 Hadamard 门：(1/√2)(|0⟩ + |1⟩)|0⟩ 3. 施加 CNOT 门：(1/√2)(|00⟩ + |11⟩) = |Φ⁺⟩ 其他 Bell 态可通过在制备前后施加 X 或 Z 门得到： |Φ⁻⟩ = (I ⊗ Z)|Φ⁺⟩ |Ψ⁺⟩ = (I ⊗ X)|Φ⁺⟩ |Ψ⁻⟩ = (X ⊗ Z)|Φ⁺⟩ Bell 态测量（Bell Measurement）： - 电路：CNOT + H + 测量 - 将两比特联合测量投影到 Bell 基 - 结果是 2 经典比特信息

EPR 悖论与 Bell 不等式

EPR 悖论和 Bell 不等式揭示了量子力学与经典物理的根本差异。

EPR 悖论（Einstein-Podolsky-Rosen, 1935）： ┌─────────────────────────────────────────────────────┐ │ Einstein 等人的论证： │ │ 1. 制备一对纠缠粒子，分发到相距很远的 Alice 和 Bob │ │ 2. Alice 测量自己的粒子，立即知道 Bob 粒子的状态 │ │ 3. 如果量子力学是完备的，信息似乎"瞬间传递" │ │ 4. 这违反相对论的定域性（信息不能超光速） │ │ 结论：量子力学不完备，存在"隐变量" │ └─────────────────────────────────────────────────────┘ Bell 不等式（1964）： ┌─────────────────────────────────────────────────────┐ │ Bell 证明：任何满足定域实在论的隐变量理论 │ │ 都必须满足某个数学不等式 │ │ │ │ CHSH 不等式： │ │ |S| = |E(a,b) - E(a,b') + E(a',b) + E(a',b')| ≤ 2 │ │ │ │ 量子力学预测： │ │ 当选择特定角度时，S = 2√2 ≈ 2.828 > 2 │ │ 量子力学违反 Bell 不等式！ │ │ │ │ 实验验证（Aspect, 1982; 2022 诺贝尔物理学奖）： │ │ 实验结果明确违反 Bell 不等式 │ │ 支持量子力学，否定定域隐变量理论 │ └─────────────────────────────────────────────────────┘ 关键概念： - 定域性（Locality）：信息不能超光速传播 - 实在性（Realism）：测量前物理量有确定值 - 量子力学：放弃定域实在论中的至少一个 - 纠缠≠通信：不能用纠缠传递信息（无超光速通信）
2022 年诺贝尔物理学奖授予 Aspect、Clauser 和 Zeilinger，表彰他们在 Bell 不等式实验验证方面的贡献。

量子隐形传态（Quantum Teleportation）

量子隐形传态利用纠缠和经典通信来传输量子态，是量子通信的基础协议。

量子隐形传态协议（Bennett et al., 1993）： ┌──────────────────────────────────────────────────────┐ │ 目标：Alice 将未知量子态 |ψ⟩ = α|0⟩ + β|1⟩ 传给 Bob │ │ 资源：一对预先共享的纠缠粒子（Bell 态 |Φ⁺⟩） │ │ 约束：不能复制量子态（不可克隆定理） │ │ 副作用：Alice 的原始态被破坏（不是复制） │ └──────────────────────────────────────────────────────┘ 初始状态（3 量子比特）： |ψ⟩_1 ⊗ |Φ⁺⟩_23 = (α|0⟩ + β|1⟩) ⊗ (1/√2)(|00⟩ + |11⟩) = (1/√2)[α|000⟩ + α|011⟩ + β|100⟩ + β|111⟩] Alice 持有比特 1（要传的态）和比特 2（纠缠对一半） Bob 持有比特 3（纠缠对另一半） 协议步骤： ┌──────────────────────────────────────────────────────┐ │ Step 1: Alice 对比特 1 和 2 做 Bell 测量 │ │ |ψ⟩_1 |Φ⁺⟩_23 → Bell 测量 → 经典结果 (i,j) │ │ │ │ Step 2: Alice 通过经典信道发送 2 经典比特 (i,j) 给 Bob│ │ │ │ Step 3: Bob 根据收到的信息应用相应的门： │ │ (0,0) → 不做操作（I） │ │ (0,1) → 施加 X 门 │ │ (1,0) → 施加 Z 门 │ │ (1,1) → 施加 ZX 门 │ │ │ │ 结果：Bob 的比特 3 变为 |ψ⟩ = α|0⟩ + β|1⟩ │ └──────────────────────────────────────────────────────┘ 量子隐形传态电路： |ψ⟩ ──────●──[H]──[M]──→ 经典比特 ──→ [X 条件]──→ |ψ⟩ │ │ |0⟩ ──[H]─●──[M]──────→ 经典比特 ──→ [Z 条件] │ |0⟩ ──────┴────────────────────────────────────── 重要限制： - 需要经典信道传输 2 比特信息 → 不超光速 - 量子态被破坏（no-cloning）→ 不是复制 - 成功概率 100%（确定性协议） - 已在光子、离子阱、超导系统中实验验证

超密编码（Superdense Coding）

超密编码是量子隐形传态的对偶协议，用 1 个量子比特传输 2 经典比特信息。

超密编码协议： ┌────────────────────────────────────────────────────┐ │ 目标：用 1 个量子比特传输 2 个经典比特的信息 │ │ 资源：一对预先共享的纠缠粒子 │ │ 效率：经典需要 2 个量子比特，量子只需 1 个 │ └────────────────────────────────────────────────────┘ 协议步骤： 1. 制备 Bell 态 |Φ⁺⟩ = (1/√2)(|00⟩ + |11⟩) Alice 持有 qubit A，Bob 持有 qubit B 2. Alice 根据要发送的 2 经典比特对 qubit A 操作： ┌──────────┬───────────┬────────────────────┐ │ 消息 │ 操作 │ 结果 Bell 态 │ ├──────────┼───────────┼────────────────────┤ │ 00 │ I（不变） │ |Φ⁺⟩ │ │ 01 │ X（翻转） │ |Ψ⁺⟩ │ │ 10 │ Z（相位） │ |Φ⁻⟩ │ │ 11 │ ZX │ |Ψ⁻⟩ │ └──────────┴───────────┴────────────────────┘ 3. Alice 发送 qubit A 给 Bob 4. Bob 对 A 和 B 做 Bell 测量，得到 2 经典比特 与 Alice 要发送的消息完全一致 超密编码 vs 经典通信： - 经典：传输 2 比特信息需要 2 个比特 - 超密编码：传输 2 比特信息只需 1 个量子比特（+ 共享纠缠） - 纠缠是"通信资源"，可提前准备
量子隐形传态和超密编码是对偶关系：隐形传态用纠缠+经典比特传输量子态，超密编码用纠缠+量子比特传输经典信息。
