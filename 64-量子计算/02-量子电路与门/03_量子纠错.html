量子纠错

量子计算QEC容错

量子纠错通过将逻辑量子比特编码到多个物理量子比特中，检测和纠正退相干和门误差导致的错误，是实现实用量子计算的关键。

量子纠错的挑战

量子纠错 vs 经典纠错： ┌─────────────────────────────────────────────────────────┐ │ 经典纠错的直接方法： │ │ - 重复编码：0 → 000，1 → 111 │ │ - 多数投票：001 → 0（1 位错误，纠正） │ │ - 简单有效 │ │ │ │ 为什么不能直接用于量子？ │ │ 1. 不可克隆定理：量子态不能被复制 │ │ → 不能简单地"重复"编码 │ │ 2. 测量破坏叠加态： │ │ → 不能直接测量量子比特来检查错误 │ │ → 测量会导致坍缩，丢失量子信息 │ │ 3. 错误类型更复杂： │ │ → 经典只有比特翻转 (0↔1) │ │ → 量子有比特翻转 X、相位翻转 Z、两者组合 Y │ │ → 连续错误（任意小旋转）也是可能的 │ │ │ │ 关键洞察： │ │ 不直接测量量子比特的状态 │ │ 而是测量"症状"（error syndrome） │ │ → 知道哪里出了错，但不知道逻辑状态 │ └─────────────────────────────────────────────────────────┘

3-比特比特翻转码

比特翻转码：纠正 X 错误（比特翻转） ┌─────────────────────────────────────────────────────────┐ │ 编码： │ │ |0⟩_L = |000⟩ │ │ |1⟩_L = |111⟩ │ │ α|0⟩ + β|1⟩ → α|000⟩ + β|111⟩（逻辑叠加态） │ │ │ │ 这里利用了量子纠缠而不是简单复制： │ │ 不是 |0⟩ → |0⟩|0⟩|0⟩（违反不可克隆定理） │ │ 而是 α|0⟩ + β|1⟩ → α|000⟩ + β|111⟩（受控的编码） │ │ │ │ 症状测量（Syndrome Measurement）： │ │ 测量算符（稳定子）： │ │ S₁ = Z₁Z₂（比较第 1、2 个量子比特是否相同） │ │ S₂ = Z₂Z₃（比较第 2、3 个量子比特是否相同） │ │ │ │ 使用辅助量子比特进行症状测量： │ │ |0⟩_anc ──H──●──●──H──M S₁ 结果 │ │ │ │ │ │ qubit 1 ────X───┘ │ │ qubit 2 ────────X │ │ │ │ 症状与错误对照表： │ │ ┌────────┬────────┬──────────────────┐ │ │ │ S₁ │ S₂ │ 错误 │ │ │ ├────────┼────────┼──────────────────┤ │ │ │ +1(0) │ +1(0) │ 无错误 │ │ │ │ -1(1) │ +1(0) │ X₁（第1位翻转） │ │ │ │ -1(1) │ -1(1) │ X₂（第2位翻转） │ │ │ │ +1(0) │ -1(1) │ X₃（第3位翻转） │ │ │ └────────┴────────┴──────────────────┘ │ │ │ │ 纠正：根据症状施加相应的 X 门 │ │ 注意：症状测量不泄露 α 和 β 的信息！ │ └─────────────────────────────────────────────────────────┘

3-比特相位翻转码

相位翻转码：纠正 Z 错误（相位翻转） ┌─────────────────────────────────────────────────────────┐ │ 编码（在 X 基中"重复"）： │ │ |0⟩_L = |+++⟩ = (|0⟩+|1⟩)/√2 ⊗ 3 │ │ |1⟩_L = |---⟩ = (|0⟩-|1⟩)/√2 ⊗ 3 │ │ │ │ 编码电路： │ │ α|0⟩+β|1⟩ ──H──●──●── α|+++⟩+β|---⟩ │ │ │ │ │ │ |0⟩ ─────────H─X──┘ │ │ |0⟩ ─────────H─────X │ │ │ │ Z 错误的效果： │ │ Z|+⟩ = |-⟩，Z|-⟩ = |+⟩ │ │ 一个 Z 错误翻转其中一个量子比特的 X 基态 │ │ │ │ 症状测量： │ │ S₁ = X₁X₂（比较第 1、2 位在 X 基是否相同） │ │ S₂ = X₂X₃（比较第 2、3 位在 X 基是否相同） │ │ │ │ 与比特翻转码完全对偶： │ │ 比特翻转码：编码到 Z 基，测量 Z，纠正 X │ │ 相位翻转码：编码到 X 基，测量 X，纠正 Z │ │ （对所有量子比特施加 H 门即可互相转换） │ └─────────────────────────────────────────────────────────┘

Shor 9-比特码

Shor 码：同时纠正比特翻转和相位翻转（第一个完整量子纠错码） ┌─────────────────────────────────────────────────────────┐ │ 思路：级联两个纠错码 │ │ 第一层：相位翻转码（纠正 Z 错误） │ │ 第二层：每个量子比特再用比特翻转码（纠正 X 错误） │ │ │ │ 编码（9 个物理量子比特 → 1 个逻辑量子比特）： │ │ |0⟩_L = (1/2√2)(|000⟩+|111⟩)(|000⟩+|111⟩)(|000⟩+|111⟩) │ │ |1⟩_L = (1/2√2)(|000⟩-|111⟩)(|000⟩-|111⟩)(|000⟩-|111⟩) │ │ │ │ 结构理解： │ │ 3 组 × 3 量子比特 = 9 量子比特 │ │ 组间：相位翻转码（Z 错误保护） │ │ 组内：比特翻转码（X 错误保护） │ │ │ │ 症状测量（8 个稳定子）： │ │ 检测 X 错误（6 个）： │ │ S₁ = Z₁Z₂, S₂ = Z₂Z₃（第 1 组内） │ │ S₃ = Z₄Z₅, S₄ = Z₅Z₆（第 2 组内） │ │ S₅ = Z₇Z₈, S₆ = Z₈Z₉（第 3 组内） │ │ 检测 Z 错误（2 个）： │ │ S₇ = X₁X₂X₃X₄X₅X₆（组 1 和组 2 的相位一致性） │ │ S₈ = X₄X₅X₆X₇X₈X₉（组 2 和组 3 的相位一致性） │ │ │ │ 可纠正的错误： │ │ - 任意单量子比特 X 错误 │ │ - 任意单量子比特 Z 错误 │ │ - 任意单量子比特 Y = XZ 错误 │ │ → 可纠正任意单量子比特错误 │ │ │ │ 距离 d = 3（可纠正 ⌊(d-1)/2⌋ = 1 个错误） │ └─────────────────────────────────────────────────────────┘

Steane 7-比特码

Steane 码：基于经典 Hamming 码的 CSS 码 ┌─────────────────────────────────────────────────────────┐ │ CSS (Calderbank-Shor-Steane) 码框架： │ │ - 利用两个经典线性码 C₁ 和 C₂ 构造 │ │ - C₂ ⊂ C₁，且 C₁ 和 C₂⊥ 都有好的纠错能力 │ │ - X 错误用 C₁ 纠正，Z 错误用 C₂ 纠正 │ │ │ │ Steane 码基于 Hamming 码 [7,4,3]： │ │ - 7 位编码 4 位信息（经典） │ │ - 距离 3，可纠正 1 位错误 │ │ │ │ 编码（7 个物理 → 1 个逻辑）： │ │ |0⟩_L = (1/√8) Σ |v⟩ （对所有偶校验码字 v 求和） │ │ |1⟩_L = X₁X₂X₃...X₇ |0⟩_L（全部翻转） │ │ │ │ 稳定子（6 个）： │ │ X 型稳定子（检测 Z 错误）： │ │ S₁ = X₄X₅X₆X₇ │ │ S₂ = X₂X₃X₆X₇ │ │ S₃ = X₁X₃X₅X₇ │ │ Z 型稳定子（检测 X 错误）： │ │ S₄ = Z₄Z₅Z₆Z₇ │ │ S₅ = Z₂Z₃Z₆Z₇ │ │ S₆ = Z₁Z₃Z₅Z₇ │ │ │ │ 优势（对比 Shor 码）： │ │ - 更少的量子比特（7 vs 9） │ │ - 更高的编码率（1/7 vs 1/9） │ │ - 可以横向实现容错门（transversal） │ │ - 对 X、Y、Z 的处理对称 │ └─────────────────────────────────────────────────────────┘

表面码 (Surface Code)

表面码：最有望实现实用量子计算的纠错码 ┌─────────────────────────────────────────────────────────┐ │ 特点： │ │ - 二维格点结构（容易在芯片上布局） │ │ - 只需最近邻相互作用（适合超导量子比特） │ │ - 高容错阈值（~1%） │ │ - 编码率 O(d²) 量子比特编码 1 个逻辑量子比特 │ └─────────────────────────────────────────────────────────┘ 表面码结构（距离 d=3 的示例）： ┌─────────────────────────────────────────────────────────┐ │ ○───○───○───○───○ ○ = 数据量子比特 │ │ │ Z │ X │ Z │ X │ Z = Z 稳定子（面） │ │ ○───○───○───○───○ X = X 稳定子（面） │ │ │ X │ Z │ X │ Z │ │ │ ○───○───○───○───○ 表面有边界： │ │ │ Z │ X │ Z │ X │ 上/下边界：Z 型 │ │ ○───○───○───○───○ 左/右边界：X 型 │ │ │ │ 稳定子测量： │ │ - Z 稳定子（面）：S_Z = Z₁Z₂Z₃Z₄（4 个数据量子比特的Z）│ │ - X 稳定子（面）：S_X = X₁X₂X₃X₄（4 个数据量子比特的X）│ │ │ │ 使用辅助量子比特进行症状测量： │ │ 每个稳定子面有一个辅助量子比特 │ │ 通过 CNOT / CZ 门与数据量子比特耦合 │ │ 测量辅助量子比特得到症状 │ └─────────────────────────────────────────────────────────┘ 错误检测与纠正： ┌─────────────────────────────────────────────────────────┐ │ 错误链 (Error Chain)： │ │ - 一个 X 错误在 Z 稳定子面上产生两个症状 │ │ - 症状成对出现 │ │ - 连接症状的链即为可能的错误位置 │ │ │ │ 纠错过程： │ │ 1. 测量所有稳定子，得到症状集合 │ │ 2. 将症状配对，找最短的连接链 │ │ 3. 沿链施加纠正操作 │ │ 4. 重复（每轮测量周期 ~1μs） │ │ │ │ 逻辑算符： │ │ - 逻辑 X：连接左右边界的 X 链 │ │ - 逻辑 Z：连接上下边界的 Z 链 │ │ - 逻辑算符可沿多条路径实现 │ │ - 错误构成逻辑算符时 → 逻辑错误（不可纠正） │ │ │ │ 距离 d 的含义： │ │ - 最短的逻辑算符路径有 d 个量子比特 │ │ - 需要 ≥ d/2 个错误才能构成不可纠正的逻辑错误 │ │ - 可纠正 ⌊(d-1)/2⌋ 个任意错误 │ │ - d=3 → 纠正 1 个错误 │ │ - d=5 → 纠正 2 个错误 │ │ - d=7 → 纠正 3 个错误 │ └─────────────────────────────────────────────────────────┘ 容错阈值定理： ┌─────────────────────────────────────────────────────────┐ │ 阈值定理 (Threshold Theorem)： │ │ 如果物理门的错误率 p < p_threshold │ │ 则可以通过增加编码距离 d 来任意降低逻辑错误率 │ │ p_logical ~ (p/p_threshold)^((d+1)/2) │ │ │ │ 表面码的容错阈值： │ │ p_threshold ≈ 1%（目前已知最高的阈值之一） │ │ │ │ 当前超导量子比特门保真度： │ │ - 单比特门：99.9%+ │ │ - 双比特门：99.5%+ │ │ - 测量：99.5%+ │ │ → 已接近表面码阈值！ │ │ │ │ Google 2024 年 Willow 芯片实验： │ │ - 105 量子比特 │ │ - 表面码距离从 3 增加到 7 │ │ - 首次实现逻辑错误率随距离增加而下降 │ │ - 验证了表面码的可扩展性 │ └─────────────────────────────────────────────────────────┘
表面码是目前最主流的量子纠错方案。Google Willow 芯片（2024）首次展示了表面码的可扩展性：增加编码距离可以降低逻辑错误率。实用量子计算可能需要数百万物理量子比特编码数千个逻辑量子比特。
