# 量子机器学习


## 量子机器学习


量子计算量子机器学习QML


量子机器学习探索如何利用量子计算加速机器学习任务，是量子计算最有前景的应用方向之一。


## 量子核方法（Quantum Kernel Methods）


量子核方法利用量子电路将数据映射到高维希尔伯特空间，计算量子核函数。


```
量子核函数定义：
┌─────────────────────────────────────────────────────┐
│  经典核函数：K(x, x') = ⟨φ(x)|φ(x')⟩               │
│  量子核函数：K(x, x') = |⟨φ(x)|φ(x')⟩|²             │
│                                                     │
│  其中 |φ(x)⟩ = U(x)|0⟩ⁿ 是量子特征映射              │
│  U(x)：参数化量子电路，将经典数据编码为量子态         │
└─────────────────────────────────────────────────────┘

量子特征映射电路：
┌─────────────────────────────────────────────┐
│  |0⟩ ──[Rz(x1)]──[Rz(x1*x2)]──[H]──...     │
│  |0⟩ ──[Rz(x2)]──[Rz(x2*x3)]──[H]──...     │
│  |0⟩ ──[Rz(x3)]──[Rz(x3*x1)]──[H]──...     │
│  ↑ ZZ 特征映射（包含交互项）                  │
└─────────────────────────────────────────────┘

量子核 SVM 流程：
1. 对每对训练样本 (x_i, x_j)：
   - 运行 U(x_i) 得到 |φ(x_i)⟩
   - 运行 U(x_j) 得到 |φ(x_j)⟩
   - 计算量子核矩阵 K_ij = |⟨φ(x_i)|φ(x_j)⟩|²
2. 将核矩阵输入经典 SVM 进行分类

潜在优势：
- 量子特征空间可能具有经典方法无法高效表达的特征
- 对某些特定数据结构有指数级优势
- 在 NISQ 设备上可行

挑战：
- 核矩阵计算需要大量量子电路运行
- 特征映射的设计需要领域知识
- 指数级量子特征空间不一定带来实际优势
```


## VQE（变分量子本征求解器）


VQE 是 NISQ 时代最核心的量子-经典混合算法，用于寻找哈密顿量的基态能量。


```
VQE 原理：
┌─────────────────────────────────────────────────────┐
│  目标：找到哈密顿量 H 的最小特征值（基态能量）         │
│  方法：变分原理 E(θ) = ⟨ψ(θ)|H|ψ(θ)⟩ ≥ E_0         │
│                                                     │
│  量子电路                    经典优化器                │
│  ┌─────────┐               ┌──────────┐             │
│  │ |0⟩ ─ U(θ) ─→ |ψ(θ)⟩   │ 最小化 E(θ)│             │
│  │ 测量 ⟨H⟩  → E(θ)  ───→ │ 更新 θ     │             │
│  └─────────┘    ↑          └──────────┘             │
│                 └─── 反馈 ────┘                      │
└─────────────────────────────────────────────────────┘

变分量子电路（Ansatz）设计：
1. 硬件高效 Ansatz（HEA）
   - 使用设备原生门的通用结构
   - 层数可调，参数量 = 量子比特数 × 层数 × 2
   - 问题：梯度消失（Barren Plateaus）

2. 问题启发 Ansatz
   - UCCSD（化学）：模拟酉耦合簇
   - QAOA（优化）：问题哈密顿量和混合哈密顿量交替
   - 更少参数，更好的收敛性

VQE 应用场景：
- 分子能量计算：H₂, LiH, H₂O 等
- 材料模拟：电子结构计算
- 优化问题：作为 QAOA 的特例
- 量子化学：催化剂设计、药物发现
```


## 量子神经网络（QNN / Quantum Neural Networks）


量子神经网络将参数化量子电路类比为神经网络层，实现端到端的量子机器学习。


```
量子神经网络架构：
┌─────────────────────────────────────────────────────┐
│  经典输入 x → 编码层 → 变分层 1 → 变分层 2 → 测量 → 经典输出 │
│                                                     │
│  编码层（Encoding Layer）：                           │
│    将经典数据编码为量子态                             │
│    方法：角度编码、振幅编码、IQP 编码                  │
│                                                     │
│  变分层（Variational Layer）：                        │
│    参数化量子门序列                                   │
│    例：[Ry(θ1), CNOT, Ry(θ2)]                       │
│    类似经典神经网络的隐藏层                            │
│                                                     │
│  测量层（Measurement Layer）：                        │
│    测量量子比特得到期望值                             │
│    输出经典数据                                       │
└─────────────────────────────────────────────────────┘

三种编码方式：
┌─────────────┬──────────────────────────────────────┐
│ 角度编码     │ x → Rx(x) 或 Ry(x)                   │
│ (Angle)     │ 每个特征用一个量子比特                  │
│             │ 编码效率低，但电路简单                  │
├─────────────┼──────────────────────────────────────┤
│ 振幅编码     │ x 直接编码为量子态的振幅                │
│ (Amplitude) │ n 量子比特编码 2ⁿ 经典特征             │
│             │ 指数压缩，但电路深度大                  │
├─────────────┼──────────────────────────────────────┤
│ IQP 编码    │ 对角酉变换编码特征间的交互               │
│ (Instant.   │ 捕获特征交叉项                         │
│ Quantum     │ 与量子核函数密切相关                    │
│ Processor)  │                                      │
└─────────────┴──────────────────────────────────────┘

训练量子神经网络：
- 参数移位规则（Parameter-Shift Rule）计算梯度
  ∂⟨H⟩/∂θ = (⟨H⟩_{θ+π/2} - ⟨H⟩_{θ-π/2}) / 2
- 与经典自动微分框架集成（PennyLane, Qiskit ML）
- 面临挑战：Barren Plateaus（梯度消失）
```


> **Note:** Barren Plateaus 是量子神经网络训练的核心障碍：随着量子比特数增加，梯度的方差指数级衰减到零，导致训练停滞。解决方案包括局部代价函数、浅层电路等。


## QAOA（量子近似优化算法）


QAOA 是用于组合优化问题的变分量子算法，思想源于量子绝热演化。


```
QAOA 算法：
┌─────────────────────────────────────────────────────┐
│  目标：最小化目标函数 C(z)，z ∈ {0,1}ⁿ              │
│                                                     │
│  1. 构建问题哈密顿量 H_C：                           │
│     C(z) = ⟨z|H_C|z⟩                                │
│     例：MaxCut H_C = Σ (1 - Z_i Z_j)/2             │
│                                                     │
│  2. 构建混合哈密顿量 H_B：                           │
│     H_B = Σ X_i（驱动量子态在计算基之间跃迁）         │
│                                                     │
│  3. 变分量子电路（p 层）：                            │
│     |γ, β⟩ = U(β_p)U(γ_p) ... U(β_1)U(γ_1)|+⟩ⁿ   │
│     其中 U(γ) = e^{-iγH_C}, U(β) = e^{-iβH_B}     │
│                                                     │
│  4. 测量期望值 ⟨γ,β|H_C|γ,β⟩                       │
│  5. 经典优化器更新 (γ, β)                            │
└─────────────────────────────────────────────────────┘

QAOA vs 量子绝热计算：
┌──────────────┬────────────────────────────────────┐
│ 绝热计算      │ 连续演化，时间长，需要容错量子计算    │
│ QAOA         │ 离散化为 p 步，可在 NISQ 设备运行    │
│              │ p → ∞ 时趋近绝热解                   │
│              │ 有限 p 可能得到近似最优解              │
└──────────────┴────────────────────────────────────┘

QAOA 应用：
- MaxCut：图的最大割问题
- 图着色、调度问题
- 旅行商问题（TSP）近似
- 组合拍卖
```
