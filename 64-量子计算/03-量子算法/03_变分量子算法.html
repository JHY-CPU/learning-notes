变分量子算法

量子计算VQEQAOA

变分量子算法是 NISQ（含噪声中等规模量子）时代最具实用前景的量子-经典混合算法。

变分量子本征求解器 VQE

VQE 原理（变分法 + 量子-经典混合）： ┌─────────────────────────────────────────────────────────┐ │ 目标：找到哈密顿量 H 的最小特征值 E₀（基态能量） │ │ │ │ 变分原理： │ │ E(θ) = ⟨ψ(θ)|H|ψ(θ)⟩ ≥ E₀ │ │ 对于任意参数化量子态 |ψ(θ)⟩，期望值 ≥ 真实基态能量 │ │ 优化 θ 最小化 E(θ) → 逼近 E₀ │ │ │ │ VQE 算法流程： │ │ ┌─────────────────────────────────────────────────┐ │ │ │ 重复直到收敛： │ │ │ │ 1. 准备参数化态 |ψ(θ)⟩ = U(θ)|0⟩ⁿ │ │ │ │ 2. 测量哈密顿量的期望值 E(θ) │ │ │ │ - 将 H 分解为 Pauli 项之和 │ │ │ │ - H = Σ cᵢ Pᵢ（Pᵢ 为 Pauli 字符串） │ │ │ │ - E(θ) = Σ cᵢ ⟨ψ(θ)|Pᵢ|ψ(θ)⟩ │ │ │ │ - 每个 Pᵢ 单独测量，经典加权求和 │ │ │ │ 3. 经典优化器更新参数 θ │ │ │ └─────────────────────────────────────────────────┘ │ └─────────────────────────────────────────────────────────┘ Ansatz（参数化量子电路）设计： ┌─────────────────────────────────────────────────────────┐ │ 硬件高效 Ansatz (HEA)： │ │ ┌─────────────────────────────────────┐ │ │ │ Layer 1: Rx(θ₁) Rz(θ₂) ●─... │ │ │ │ Layer 1: Rx(θ₃) Rz(θ₄) ─●... │ │ │ │ ... │ │ │ │ Layer 2: Rx(θ₅) Rz(θ₆) ●─... │ │ │ │ ... │ │ │ └─────────────────────────────────────┘ │ │ - 每层：单比特旋转 + 双比特纠缠门 │ │ - 优点：通用性强，适应硬件拓扑 │ │ - 缺点：梯度消失（Barren Plateaus） │ │ │ │ UCCSD Ansatz（量子化学专用）： │ │ |ψ⟩ = e^(T - T†)|Φ₀⟩ │ │ - T = Σ tᵢₐ a†ᵢaₐ + Σ tᵢⱼₐᵦ a†ᵢa†ⱼaᵦaₐ │ │ - 物理意义：对参考态（Hartree-Fock）做激发 │ │ - 参数少，收敛性好 │ │ - 电路深度较大 │ │ │ │ ADAPT-VQE： │ │ - 自适应地添加算符到 Ansatz 中 │ │ - 每步选择使梯度最大的算符 │ │ - 比 UCCSD 更紧凑的表示 │ └─────────────────────────────────────────────────────────┘ VQE 应用：分子能量计算 ┌─────────────────────────────────────────────────────────┐ │ H₂ 分子示例： │ │ 1. 将哈密顿量映射到量子比特 │ │ H = g₀I + g₁Z₀ + g₂Z₁ + g₃Z₀Z₁ + g₄X₀X₁ + g₅Y₀Y₁ │ │ （Jordan-Wigner 或 Bravyi-Kitaev 变换） │ │ │ │ 2. 制备初态 |01⟩（Hartree-Fock 态） │ │ │ │ 3. 应用参数化电路 U(θ) │ │ │ │ 4. 测量各项期望值，计算 E(θ) │ │ │ │ 5. 优化 θ 得到基态能量 │ │ │ │ 已计算的分子：H₂, LiH, H₂O, BeH₂, N₂ │ │ 挑战：大分子需要太多量子比特 │ │ FeMoCo（固氮酶活性中心）：需要 ~100-200 逻辑量子比特 │ └─────────────────────────────────────────────────────────┘

QAOA（量子近似优化算法）

QAOA：用于组合优化问题的变分量子算法 ┌─────────────────────────────────────────────────────────┐ │ 目标：最小化目标函数 C(z)，z ∈ {0,1}ⁿ │ │ │ │ 核心思想：源自量子绝热演化 │ │ 绝热定理：缓慢从简单哈密顿量演化到问题哈密顿量 │ │ → 最终态是问题的基态（最优解） │ │ QAOA：将连续演化离散化为 p 步 │ └─────────────────────────────────────────────────────────┘ QAOA 电路结构： ┌─────────────────────────────────────────────────────────┐ │ 构建哈密顿量： │ │ 问题哈密顿量 H_C：C(z) = ⟨z|H_C|z⟩ │ │ 例：MaxCut 问题 │ │ H_C = (1/2) Σ_(i,j∈E) (1 - Z_iZ_j) │ │ │ │ 混合哈密顿量 H_B = Σᵢ Xᵢ │ │ 驱动量子态在不同计算基之间跃迁 │ │ │ │ p 层 QAOA 电路： │ │ |+⟩ⁿ ──U(γ₁)──U(β₁)──U(γ₂)──U(β₂)──...──U(γₚ)──U(βₚ)──M │ │ │ │ U(γ) = e^(-iγH_C) = 问题演化 │ │ U(β) = e^(-iβH_B) = 混合演化 │ │ │ │ 参数向量：(γ₁, ..., γₚ, β₁, ..., βₚ) 共 2p 个参数 │ │ │ │ MaxCut 示例（图 G = (V,E)）： │ │ 对每条边 (i,j)： │ │ U_C(γ) = Π_(i,j∈E) e^(iγ(1-Z_iZ_j)/2) │ │ = Π_(i,j∈E) [e^(iγ/2) · CNOT──Rz(2γ)──CNOT] │ │ 对每个节点： │ │ U_B(β) = Πᵢ e^(iβXᵢ) = Πᵢ Rx(2β) │ └─────────────────────────────────────────────────────────┘ QAOA 优化过程： ┌─────────────────────────────────────────────────────────┐ │ 1. 初始化参数 (γ, β)（随机或启发式） │ │ │ │ 2. 执行量子电路得到测量结果 z │ │ │ │ 3. 计算目标函数期望值： │ │ ⟨C⟩ = ⟨γ,β|H_C|γ,β⟩ │ │ 通过多次采样估计 │ │ │ │ 4. 经典优化器更新 (γ, β)： │ │ - 梯度下降（参数移位规则计算梯度） │ │ - COBYLA, Nelder-Mead（无梯度） │ │ - 贝叶斯优化 │ │ │ │ 5. 重复 2-4 直到收敛 │ │ │ │ p → ∞ 时 QAOA 逼近量子绝热解（最优解） │ │ 有限 p 得到近似解（可能仍优于经典算法） │ └─────────────────────────────────────────────────────────┘ QAOA 复杂度分析： ┌─────────────────────────────────────────────────────────┐ │ 电路深度：O(p × |E|)（p 为层数，|E| 为边数） │ │ 参数个数：2p（常数于问题规模） │ │ 测量次数：每轮需 O(poly(n)) 次采样估计期望值 │ │ │ │ p=1 的 MaxCut 近似比： │ │ 对于 3-正则图：≥ 0.6924（Goemans-Williamson 的 0.878） │ │ 已证明优于简单随机算法 │ │ │ │ 理论结果： │ │ - 对某些问题，有限 p 的 QAOA 需要 p = Ω(n) │ │ - 是否存在量子优势仍有争议 │ │ - 小规模实验已验证正确性 │ └─────────────────────────────────────────────────────────┘

Barren Plateaus（梯度消失问题）

Barren Plateaus 问题： ┌─────────────────────────────────────────────────────────┐ │ 随着量子比特数 n 增加，损失函数的梯度方差指数衰减： │ │ Var[∂C/∂θₖ] ≤ F(n) │ │ F(n) 随 n 指数衰减到 0 │ │ │ │ 后果： │ │ - 梯度在大部分参数空间接近零 │ │ - 优化器无法找到下降方向 │ │ - 训练停滞 │ │ - 需要指数次测量才能获得有用的梯度估计 │ └─────────────────────────────────────────────────────────┘ Barren Plateaus 的原因： ┌─────────────────────────────────────────────────────────┐ │ 1. 全局代价函数： │ │ 对所有量子比特的联合测量 │ │ 导致梯度方差随 n 指数衰减 │ │ │ │ 2. 随机参数化电路： │ │ - 电路深度足够深时（polynomial in n） │ │ - 随机量子电路构成 2-design 或 t-design │ │ - 参数梯度均匀分布在 0 附近 │ │ │ │ 3. 硬件高效 Ansatz： │ │ - 层数随 n 增加 │ │ - 每层的随机门使整体趋近 Haar 随机酉矩阵 │ │ │ │ 4. 噪声效应： │ │ - 量子噪声使有效电路深度增加 │ │ - 加剧 Barren Plateaus │ └─────────────────────────────────────────────────────────┘ 解决方法： ┌─────────────────────────────────────────────────────────┐ │ 1. 局部代价函数 (Local Cost Functions)： │ │ - 只测量少数量子比特的观测量 │ │ - 梯度方差不再随 n 指数衰减 │ │ - 但表达能力可能受限 │ │ │ │ 2. 问题启发式 Ansatz： │ │ - UCCSD（量子化学）：基于物理直觉的电路结构 │ │ - QAOA：问题哈密顿量 + 混合哈密顿量交替 │ │ - 参数少，梯度性质更好 │ │ │ │ 3. 逐层训练 (Layerwise Training)： │ │ - 先训练浅层电路 │ │ - 逐步增加层数 │ │ - 每层只在已有参数附近优化 │ │ │ │ 4. 参数初始化策略： │ │ - Identity Block 初始化（参数接近 0） │ │ - 从经典近似解初始化 │ │ - 避免随机初始化 │ │ │ │ 5. 经典预处理： │ │ - 用经典算法找到好的初始解 │ │ - 量子电路只做精细化优化 │ │ │ │ 6. 使用浅层电路： │ │ - 限制电路深度在 O(log n) 或 O(1) │ │ - 牺牲部分表达能力换取可训练性 │ └─────────────────────────────────────────────────────────┘
Barren Plateaus 是变分量子算法规模化的主要障碍。解决方案包括使用问题启发式 Ansatz（如 UCCSD）、局部代价函数和逐层训练策略。当前研究热点是如何在大规模系统中保持可训练性。

参数移位规则与量子梯度

参数移位规则 (Parameter-Shift Rule)： ┌─────────────────────────────────────────────────────────┐ │ 在量子硬件上直接计算梯度，无需经典模拟 │ │ │ │ 对于单参数门 U(θ) = e^(-iθG)，其中 G 是生成器： │ │ ∂⟨H⟩/∂θ = (⟨H⟩_{θ+π/2} - ⟨H⟩_{θ-π/2}) / 2 │ │ │ │ 即：只需在 θ±π/2 处运行两次量子电路 │ │ 梯度 = (E(θ+π/2) - E(θ-π/2)) / 2 │ │ │ │ 优势： │ │ - 在真实量子硬件上计算精确梯度 │ │ - 不需要量子态层析 │ │ - 只需 2 次电路执行 per 参数 │ │ │ │ 推广（2-term parameter-shift）： │ │ 对于任意单参数门： │ │ ∂E/∂θ = c₁ E(θ+s₁) + c₂ E(θ+s₂) │ │ 其中 c₁, c₂, s₁, s₂ 取决于 G 的特征值 │ └─────────────────────────────────────────────────────────┘ 量子梯度下降： ┌─────────────────────────────────────────────────────────┐ │ 算法： │ │ 1. 初始化参数 θ │ │ 2. 对每个参数 θₖ： │ │ - 运行 U(θₖ+π/2)，测量期望值 E₊ │ │ - 运行 U(θₖ-π/2)，测量期望值 E₋ │ │ - 梯度 gₖ = (E₊ - E₋) / 2 │ │ 3. 更新：θ ← θ - η × g │ │ 4. 重复 2-3 直到收敛 │ │ │ │ 总电路执行次数：O(N × iterations) │ │ N = 参数个数，每轮需 2N 次执行 │ │ │ │ 改进方法： │ │ - 量子自然梯度：考虑参数空间的量子几何结构（Fubini-Study 度量） │ - SPSA（同时扰动随机近似）：每次只随机扰动两个参数 │ │ - L-BFGS：拟牛顿法，收敛更快 │ │ - ADAM：自适应学习率，适合非平稳目标函数 │ └─────────────────────────────────────────────────────────┘ 量子自动微分： ┌─────────────────────────────────────────────────────────┐ │ PennyLane 框架（Xanadu 开发）： │ │ - 将量子电路集成到经典自动微分框架 │ │ - 支持 PyTorch, TensorFlow, JAX 后端 │ │ - 自动选择最优微分策略 │ │ │ │ 微分方法选择： │ │ ┌────────────────┬─────────────────────────────┐ │ │ │ 参数移位规则 │ 精确梯度，2 次执行/参数 │ │ │ ├────────────────┼─────────────────────────────┤ │ │ │ 伴随微分法 │ 反向传播，1 次执行/参数 │ │ │ │ (Adjoint) │ 但需要模拟器支持 │ │ │ ├────────────────┼─────────────────────────────┤ │ │ │ 有限差分 │ 近似梯度，N+1 次执行 │ │ │ ├────────────────┼─────────────────────────────┤ │ │ │ SPSA │ 随机梯度，2 次执行 │ │ │ └────────────────┴─────────────────────────────┘ │ └─────────────────────────────────────────────────────────┘
