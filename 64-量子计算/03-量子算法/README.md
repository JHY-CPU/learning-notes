# 03-量子算法

量子算法利用叠加、纠缠和干涉实现对经典算法的加速。本章介绍从基础到前沿的核心量子算法。

---

## Deutsch-Jozsa 算法

**问题**：给定函数 f:{0,1}ⁿ→{0,1}，已知它要么是常函数（所有输入输出相同），要么是平衡函数（恰好一半输出 0，一半输出 1）。确定 f 的类型。

**经典复杂度**：确定性算法最坏需 2ⁿ⁻¹+1 次查询。

**量子算法**：
1. 初始化 n+1 个量子比特：|0⟩ⁿ|1⟩
2. 对所有量子比特施加 Hadamard 门
3. 执行 oracle：|x⟩|y⟩→|x⟩|y⊕f(x)⟩
4. 再次对前 n 个量子比特施加 H 门
5. 测量：若全为 0 则常函数，否则平衡函数

仅需 1 次 oracle 查询，展示了量子计算对确定性经典算法的指数级加速。

---

## Bernstein-Vazirani 算法

**问题**：给定 f(x) = s·x (mod 2)，其中 s 是隐藏的 n 位字符串，求 s。

**经典复杂度**：需要 n 次查询（逐位确定）。**量子算法**：一次 oracle 查询即可确定完整 s。

结构与 Deutsch-Jozsa 类似：|0⟩ⁿ|1⟩ → H⊗(n+1) → Oracle → H⊗n → 测量 → s。

---

## Simon 算法

**问题**：给定 f:{0,1}ⁿ→{0,1}ⁿ，存在隐藏 s≠0 使得 f(x)=f(x⊕s)，求 s。

**经典复杂度**：O(2^(n/2)) 次查询。**量子算法**：O(n) 次查询，是第一个展示指数级量子加速的算法（相对于有界误差随机经典算法）。

步骤：制备叠加态查询 oracle，测量得到满足 y·s=0 的 y，重复 O(n) 次，经典线性代数求解 s。此算法直接启发了 Shor 算法的设计。

---

## 量子傅里叶变换（QFT）

QFT 是经典离散傅里叶变换的量子版本：

$$QFT|j\rangle = \frac{1}{\sqrt{N}}\sum_{k=0}^{N-1} e^{2\pi ijk/N}|k\rangle$$

电路实现需 O(n²) 个门（H 门和受控相位旋转门），远优于经典 FFT 的 O(n·2ⁿ)。但 QFT 的指数加速需与态制备和测量配合才能体现，单独使用不能直接加速经典傅里叶变换。

---

## 量子相位估计（QPE）

**问题**：给定酉算符 U 和其本征态 |u⟩（U|u⟩=e^{2πiφ}|u⟩），估计相位 φ。

**算法**：准备 t 个辅助量子比特，施加 H 门，用受控 U 门实施受控幂运算，对辅助量子比特实施逆 QFT，测量得到 φ 的 t 位近似。精度 2⁻ᵗ，成功概率至少 4/π²。

QPE 是 Shor 算法和量子模拟的核心子程序。

---

## Shor 算法（大整数分解）

Shor 算法（1994）是最著名的量子算法，可在多项式时间内分解大整数。

### 原理与步骤

将分解约化为**阶查找**问题：
1. 随机选择与 N 互素的 a
2. 用 QPE 找到 f(x) = aˣ mod N 的阶 r
3. 若 r 为偶数且 a^(r/2) ≠ -1 mod N，gcd(a^(r/2)±1, N) 给出因子

复杂度 O((log N)³)，对比经典最优算法有指数级加速。

### 对 RSA 的威胁

2048 位 RSA 需约 4000 个逻辑量子比特（数百万物理量子比特）。"先收集后解密"威胁促使各国加速部署后量子密码学（PQC），NIST 已于 2024 年标准化首批 PQC 算法。

## Grover 搜索算法

Grover 算法（1996）在无序数据库中搜索目标，实现平方根加速。

### 振幅放大

1. 初始化均匀叠加态 |ψ⟩ = (1/√N)Σ|x⟩
2. Oracle 标记：对目标态引入相位翻转
3. 扩散算符：绕均匀叠加态做反射，增大概率幅
4. 重复约 √N 次后测量

几何解释：在二维子空间中实现旋转，最优迭代次数 k ≈ (π/4)√N。O(√N) 次 oracle 查询，是最优量子搜索算法。

应用：搜索、满足性问题、对称密钥密码分析（密钥长度减半）。

---

## 量子行走

量子行走是经典随机行走的量子类比：

- **离散时间**：引入"硬币"量子比特，用受控移位演化
- **连续时间**：用图的邻接矩阵驱动演化

经典行走扩散速度 O(√t)，量子行走弹道传播 O(t)，提供二次加速。应用于图搜索、元素独特性问题等。

---

## VQE（变分量子本征求解器）

VQE 是 NISQ 时代最重要的量子-经典混合算法：

1. 用参数化量子电路（ansatz）制备试探态 |ψ(θ)⟩
2. 量子计算机测量期望值 ⟨ψ(θ)|H|ψ(θ)⟩
3. 经典优化器更新参数 θ，迭代至收敛

变分原理保证找到的能量是真实基态的上界。应用于量子化学分子能量计算。挑战：barren plateau（梯度消失）、局部极小值。

---

## QAOA（量子近似优化算法）

QAOA 用交替的代价层 e^{-iγC} 和混合层 e^{-iβB} 构造参数化电路，变分优化 (γ,β)，逼近组合优化问题的近似解。

典型应用：MaxCut、图着色、调度优化。有限深度的 QAOA 可超越最简单的经典算法，但是否超越最佳经典算法仍有争议。

## 量子机器学习基础

主要方向：量子核方法、变分量子分类器、量子神经网络（QNN，含数据编码/参数化/测量层）、量子 GAN、量子采样。需注意数据加载瓶颈和与经典基准的严格对比。

## 量子计算复杂性理论

量子计算复杂性类的核心是 **BQP（有界误差量子多项式时间）**：量子计算机可以在多项式时间内以高概率正确求解的问题集合。

已知关系：P ⊆ BQP ⊆ PSPACE。Shor 算法表明整数分解属于 BQP（但不在 P 中，目前未知是否在 NP 完全中）。Grover 搜索说明量子计算对 NP 完全问题仅提供多项式（平方根）加速，而非指数加速。

**量子优势的条件**：需要问题具有特定结构（如周期性、对称性），使量子干涉能有效放大正确答案的概率。并非所有问题都能从量子计算中获益。
