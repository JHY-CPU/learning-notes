量子比特与叠加

量子计算量子比特叠加态

量子比特是量子计算的基本信息单元，叠加和纠缠是量子计算的核心特性。

量子比特 (Qubit)

经典比特 vs 量子比特： ┌─────────────────────────────────────────────┐ │ 经典比特 (Bit)： │ │ 状态：0 或 1（二选一） │ │ n位：2^n个状态中取1个 │ │ │ │ 量子比特 (Qubit)： │ │ 状态：|ψ⟩ = α|0⟩ + β|1⟩（叠加态） │ │ 其中 |α|² + |β|² = 1 │ │ n位：同时处于2^n个状态的叠加 │ └─────────────────────────────────────────────┘ Dirac符号 (Bra-Ket Notation)： - |0⟩ = [1, 0]^T = 基态（列向量） - |1⟩ = [0, 1]^T = 激发态 - |ψ⟩ = α|0⟩ + β|1⟩ = [α, β]^T - ⟨ψ| = [α*, β*] （共轭转置，行向量） - ⟨φ|ψ⟩ = 内积（振幅） Bloch球表示： |ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩ - θ：与z轴的夹角（0到π） - φ：x-y平面上的相位角 - 任何纯态都是Bloch球面上的一个点 - |0⟩在北极，|1⟩在南极

叠加与测量

叠加态 (Superposition)： |ψ⟩ = α|0⟩ + β|1⟩ - 量子比特同时处于|0⟩和|1⟩的叠加 - 不是"不知道是0还是1"，而是"同时是0和1" - 叠加是量子并行性的来源 测量 (Measurement)： - 测量会坍缩量子态到计算基 - 以概率 |α|² 得到 |0⟩ - 以概率 |β|² 得到 |1⟩ - 测量后叠加态被破坏 - 测量是不可逆的！ 示例： |+⟩ = (1/√2)(|0⟩ + |1⟩) → 测量得0概率50%，得1概率50% |−⟩ = (1/√2)(|0⟩ - |1⟩) → 同样50/50，但相位不同 |0⟩ → 测量必定得0 |1⟩ → 测量必定得1 Hadamard门 H： H|0⟩ = |+⟩ = (1/√2)(|0⟩ + |1⟩) 制备叠加态 H|1⟩ = |−⟩ = (1/√2)(|0⟩ - |1⟩) H H = I 应用两次回到原状态

量子纠缠 (Entanglement)

纠缠态：多量子比特的不可分态 Bell态（最大纠缠态）： |Φ+⟩ = (1/√2)(|00⟩ + |11⟩) ← 最常用 |Φ−⟩ = (1/√2)(|00⟩ - |11⟩) |Ψ+⟩ = (1/√2)(|01⟩ + |10⟩) |Ψ−⟩ = (1/√2)(|01⟩ - |10⟩) 制备Bell态 |Φ+⟩： |00⟩ → H⊗I → CNOT → (1/√2)(|00⟩ + |11⟩) 1.对第一个比特施加Hadamard 2.以第一个比特控制，第二个为目标的CNOT门 纠缠的特性： - 测量一个比特立即确定另一个的状态 - 对|Φ+⟩，测第一个得0→第二个必得0 - 对|Φ+⟩，测第一个得1→第二个必得1 - 这种关联不依赖距离（超距作用） - 但不能用于超光速通信（测量结果随机） EPR悖论与Bell不等式： - 爱因斯坦认为纠缠说明量子力学不完备 - Bell不等式证明了局域隐变量理论不成立 - 2022年诺贝尔物理学奖授予纠缠实验验证
纠缠是量子计算超越经典计算的关键资源。量子隐形传态和量子纠错都依赖纠缠。

多量子比特系统

两量子比特系统： ┌─────────────────────────────────────────────────────────┐ │ 状态空间：C⁴（4维复向量空间） │ │ 计算基：|00⟩, |01⟩, |10⟩, |11⟩ │ │ │ │ 一般态：|ψ⟩ = α|00⟩ + β|01⟩ + γ|10⟩ + δ|11⟩ │ │ 归一化：|α|² + |β|² + |γ|² + |δ|² = 1 │ │ │ │ 张量积 (Tensor Product)： │ │ |0⟩ ⊗ |0⟩ = |00⟩ = [1,0,0,0]^T │ │ |0⟩ ⊗ |1⟩ = |01⟩ = [0,1,0,0]^T │ │ |+⟩ ⊗ |0⟩ = (|00⟩ + |10⟩)/√2 │ │ ⊗ 是构造多量子比特系统的基础运算 │ │ │ │ n 量子比特系统： │ │ 状态空间：C^(2^n) │ │ 一般态：|ψ⟩ = Σ αₓ|x⟩（x 遍历所有 n 位二进制串） │ │ 共有 2^n 个复数振幅需要描述 │ │ → 这是经典模拟量子计算的根本困难 │ └─────────────────────────────────────────────────────────┘ 不可分态 (Entangled States)： ┌─────────────────────────────────────────────────────────┐ │ 积态 (Product State)： │ │ |ψ⟩ = |φ⟩⊗|χ⟩ 可以写成两个子系统的张量积 │ │ 例：|00⟩ = |0⟩⊗|0⟩ │ │ 子系统之间没有量子关联 │ │ │ │ 纠缠态 (Entangled State)： │ │ |ψ⟩ ≠ |φ⟩⊗|χ⟩ 不能写成张量积 │ │ 例：(|00⟩ + |11⟩)/√2（Bell 态） │ │ 子系统之间存在非经典关联 │ │ │ │ 判断纠缠： │ │ - Schmidt 分解：任何态可写成 Σ λᵢ|aᵢ⟩⊗|bᵢ⟩ │ │ - 如果 Schmidt 系数 λᵢ 不止一个非零 → 纠缠 │ │ - Schmidt 秩 = 1 → 积态（不纠缠） │ │ - Schmidt 秩 > 1 → 纠缠态 │ └─────────────────────────────────────────────────────────┘
