# 01-量子力学基础

理解量子计算需要先掌握量子力学的核心概念。本章从经典信息论出发，逐步引入量子态、叠加、纠缠等关键思想。

---

## 经典比特 vs 量子比特

经典计算机使用**比特（bit）**作为信息的基本单位，每个比特处于确定的 0 或 1 状态。量子计算机使用**量子比特（qubit）**，其状态可以是 |0⟩ 和 |1⟩ 的任意线性叠加：

$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$$

其中 α、β 为复数，满足归一化条件 |α|² + |β|² = 1。

经典比特是量子比特的特例（α=0 或 β=0）。量子比特的关键特性在于：在测量之前，它同时"处于" 0 和 1 的叠加状态。

---

## 量子态的数学表示

### Dirac 符号（Bra-Ket 记号）

量子力学中使用 **Dirac 符号** 表示量子态：

- **Ket 向量**：|ψ⟩ 表示一个列向量（量子态）
- **Bra 向量**：⟨ψ| 表示 |ψ⟩ 的共轭转置（行向量）
- **内积**：⟨φ|ψ⟩ 表示两个态之间的内积（复数标量）
- **外积**：|ψ⟩⟨φ| 表示一个算符

标准基：
- |0⟩ = [1, 0]ᵀ 对应经典比特 0
- |1⟩ = [0, 1]ᵀ 对应经典比特 1

### 向量表示

单量子比特态可以表示为二维复向量空间中的单位向量：

$$|\psi\rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}$$

该向量属于二维**Hilbert 空间** H = ℂ²，内积定义为 ⟨φ|ψ⟩ = φ†ψ。

## 叠加态（Superposition）

叠加原理是量子力学最基本的原理之一：如果 |0⟩ 和 |1⟩ 都是合法的量子态，那么它们的任意线性组合也是合法的量子态。

叠加态的物理意义：
- 量子系统在测量前同时"探索"所有可能的状态
- 叠加是量子并行性的来源：n 个量子比特可同时表示 2ⁿ 个基态的叠加
- 例：H|0⟩ = (|0⟩ + |1⟩)/√2，即等概率叠加态

叠加并非"同时处于多个状态"的经典理解，而是一种概率幅（probability amplitude）的相干叠加，各分量之间存在**相位关系**，这是量子干涉效应的根源。

---

## Bloch 球

任意单量子比特纯态可以用 **Bloch 球** 上的一个点直观表示：

$$|\psi\rangle = \cos\frac{\theta}{2}|0\rangle + e^{i\phi}\sin\frac{\theta}{2}|1\rangle$$

其中 θ ∈ [0, π]，φ ∈ [0, 2π)。

Bloch 球的几何直观：
- 北极 (θ=0) 对应 |0⟩，南极 (θ=π) 对应 |1⟩
- 赤道上的点对应等概率叠加态（|0⟩+|1⟩)/√2 等
- 整个球面表示所有可能的纯态
- 全局相位不改变物理状态，因此一个纯态对应球面上一个点

Bloch 球的局限：只能表示**纯态**，无法表示混合态（需要密度矩阵）。

## 量子测量与波函数坍缩

量子测量是量子力学中最具争议的核心概念：

### 投影测量（冯诺依曼测量）

对处于态 |ψ⟩ = α|0⟩ + β|1⟩ 的量子比特以计算基 {|0⟩, |1⟩} 进行测量：
- 以概率 |α|² 得到结果 0，态坍缩为 |0⟩
- 以概率 |β|² 得到结果 1，态坍缩为 |1⟩

### 关键性质

1. **不可逆性**：测量导致量子态不可逆地坍缩
2. **概率性**：测量结果本质上是随机的，只能给出概率预测
3. **信息破坏**：测量会破坏叠加态中的相位信息
4. **Born 规则**：概率等于概率幅的模方

### 一般测量

更一般的测量可用 **POVM（正算符值测度）** 描述，投影测量是其特例。测量算符集合 {Mₘ} 满足 Σ Mₘ†Mₘ = I。

## 量子纠缠

量子纠缠是多粒子量子系统中特有的非经典关联。两个量子比特可以处于无法分解为各自独立态之积的联合态。

### Bell 态（EPR 对）

四个最大纠缠态（Bell 态）：

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$
$$|\Phi^-\rangle = \frac{1}{\sqrt{2}}(|00\rangle - |11\rangle)$$
$$|\Psi^+\rangle = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)$$
$$|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)$$

以 |Φ⁺⟩ 为例：单独看每个粒子都是完全随机的（各 50% 概率），但两个粒子的测量结果完全相关——如果一方测得 0，另一方也必定测得 0。

### 纠缠的非局域性

纠缠不传递信息（不能超光速通信），但产生非经典的统计关联。纠缠是量子密钥分发、量子隐形传态和超dense coding的基础。

## EPR 悖论与 Bell 不等式

### EPR 悖论（1935）

Einstein、Podolsky 和 Rosen 提出的思想实验：如果量子力学是完备的，那么纠缠粒子间的瞬时关联似乎意味着超光速作用（"幽灵般的超距作用"），或者量子力学对物理实在的描述不完备——他们认为存在**隐变量**。

### Bell 不等式（1964）

John Bell 推导出一组不等式，任何**局域隐变量理论**都必须满足这些不等式。而量子力学预测这些不等式可以被**违反**。

CHSH 不等式：|S| ≤ 2（经典极限），量子力学可达 2√2 ≈ 2.828（Tsirelson 界限）。

### 实验验证

Aspect 实验（1982）及后续更严格的"无漏洞"实验（2015）一致证实了量子力学的预测，否定了局域隐变量理论。2022 年 Nobel 物理学奖授予 Aspect、Clauser 和 Zeilinger。

## 不可克隆定理

**量子不可克隆定理（No-Cloning Theorem）**：不存在一个幺正算符 U 使得对任意量子态 |ψ⟩ 都有 U|ψ⟩|0⟩ = |ψ⟩|ψ⟩。

证明概要：假设存在这样的 U，对于两个不同态 |ψ⟩ 和 |φ⟩，由 U 的幺正性和内积不变性可导出矛盾（要么 |ψ⟩=|φ⟩，要么 |ψ⟩ 与 |φ⟩ 正交时才能克隆）。

物理意义：
- 量子信息不能被完美复制，这与经典信息可以自由复制形成对比
- 是量子密码学安全性的基础：窃听者无法在不引入扰动的情况下复制量子态
- 与**不可区分定理**（No-Deletion Theorem）对偶

## 密度矩阵基础

密度矩阵（密度算符）提供了一种统一描述纯态和混合态的数学框架。

### 定义

对于纯态 |ψ⟩：ρ = |ψ⟩⟨ψ|

对于混合态（以概率 pᵢ 处于纯态 |ψᵢ⟩）：ρ = Σᵢ pᵢ |ψᵢ⟩⟨ψᵢ|

### 性质

- ρ 是厄米算符（ρ = ρ†）
- Tr(ρ) = 1（归一化）
- ρ ≥ 0（半正定）
- 纯态满足 Tr(ρ²) = 1；混合态满足 Tr(ρ²) < 1

### 约化密度矩阵

对于复合系统 AB 的纠缠态，对子系统 B 求偏迹得到子系统 A 的**约化密度矩阵** ρₐ = Tr_B(ρ_AB)。约化密度矩阵描述了忽略 B 后 A 的有效状态，通常不再是纯态。

---

## 纯态与混合态

### 纯态

可以用一个态矢量 |ψ⟩ 完全描述的量子态。密度矩阵满足 ρ = ρ²（幂等）。

例如：|+⟩ = (|0⟩+|1⟩)/√2 是纯态，其 Bloch 球上的点在赤道上。

### 混合态

由多种纯态按概率混合形成的统计混合。例如：以 50% 概率处于 |0⟩，50% 概率处于 |1⟩。

该混合态的密度矩阵 ρ = (1/2)|0⟩⟨0| + (1/2)|1⟩⟨1| = I/2，位于 Bloch 球的**原点**。

### 关键区别

| 性质 | 纯态 | 混合态 |
|------|------|--------|
| 描述方式 | 态矢量 | 密度矩阵 |
| Tr(ρ²) | 1 | < 1 |
| Bloch 球 | 球面（r=1） | 球内部（r<1） |
| 相干性 | 存在 | 部分或完全丧失 |
| 信息量 | 最大 | 不确定性更高 |

混合态的物理来源：退相干（与环境相互作用）、统计制备过程、部分trace（子系统）。
