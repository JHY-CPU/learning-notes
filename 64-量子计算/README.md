# 62-量子计算

量子计算是利用量子力学原理（如叠加、纠缠和干涉）进行信息处理的计算范式。与经典计算使用比特不同，量子计算使用量子比特（qubit），能够在某些特定问题上实现超越经典计算机的指数级加速。
