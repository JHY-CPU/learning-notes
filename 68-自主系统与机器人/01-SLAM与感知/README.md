# 01-SLAM与感知

## 1. 自主系统概述

自主系统是指能够在无人（或少人）干预下感知环境、做出决策并执行动作的智能系统，主要应用领域包括：

- **自动驾驶**：L2-L5 级别自动驾驶车辆，需要实时感知、定位、规划与控制
- **无人机（UAV）**：航拍、物流配送、农业植保、搜救
- **移动机器人**：仓储物流机器人、服务机器人、家用扫地机器人
- **水下/太空机器人**：深海探测、行星探测车

自主系统的核心技术栈：**感知（Perception）→ 定位（Localization）→ 规划（Planning）→ 控制（Control）**

## 2. 传感器

传感器是自主系统感知外部环境的基础，不同传感器有各自的优势与局限。

### 2.1 摄像头

| 类型 | 特点 | 应用 |
|------|------|------|
| **单目摄像头** | 成本低，但缺乏直接深度信息 | 车道线检测、交通标志识别 |
| **双目摄像头** | 通过视差计算深度，无需主动光源 | 立体匹配、中近距离测距 |
| **鱼眼摄像头** | 超大视场角（>180度），畸变大 | 全景泊车、近场感知 |
| **事件相机（DVS）** | 异步输出亮度变化事件，极高时间分辨率（微秒级）、高动态范围 | 高速运动场景、极端光照 |

### 2.2 激光雷达（LiDAR）

通过发射激光束并测量反射时间来获取精确的 3D 距离信息。

- **机械式 LiDAR**：通过旋转机构实现 360 度扫描，精度高但成本昂贵、体积大（如 Velodyne HDL-64E）
- **固态 LiDAR**：无机械运动部件，使用 MEMS 微镜或 OPA（光学相控阵）实现扫描，成本更低、可靠性更高
- **特点**：不受光照影响，精度高（厘米级），但无法感知颜色和纹理，在雨雾天气性能下降

### 2.3 毫米波雷达

- 工作在 24GHz/77GHz 频段，测量目标的距离、速度和角度
- **优势**：全天候工作，可直接测量多普勒速度，成本低
- **局限**：角分辨率较低，难以区分密集目标，无法识别静止物体类型
- **应用**：自适应巡航控制（ACC）、自动紧急制动（AEB）

### 2.4 超声波传感器

- 近距离（0.2-5m）测距，成本极低
- 不受光照和颜色影响
- **应用**：泊车辅助、低速避障

### 2.5 IMU（惯性测量单元）

- 包含三轴加速度计和三轴陀螺仪，测量线加速度和角速度
- **优势**：高频输出（100-1000Hz），短时精度极高，不受外界干扰
- **局限**：存在漂移误差，长时间积分会导致位置和姿态严重偏离
- **应用**：与 GPS/LiDAR 融合，提供高频位姿估计

### 2.6 GPS/RTK

- **GPS**：全球定位系统，精度约 2-5 米
- **RTK（Real-Time Kinematic）**：通过基站差分修正，精度可达厘米级
- **局限**：依赖卫星信号，在隧道、地下停车场、城市峡谷中信号丢失或严重退化

## 3. SLAM（同时定位与建图）

SLAM（Simultaneous Localization and Mapping）是指机器人在未知环境中，利用传感器数据同时估计自身位姿并构建环境地图的过程。这是自主系统最基础的能力之一。

### 3.1 视觉 SLAM

#### 特征点法

从图像中提取特征点（如 ORB、SIFT），通过特征匹配估计相机运动。

- **ORB-SLAM**：基于 ORB 特征点的单目/双目/RGB-D SLAM 系统
  - 三个并行线程：跟踪（Tracking）、局部建图（Local Mapping）、回环检测（Loop Closing）
  - 使用词袋模型（DBoW2）进行回环检测
  - 代表版本：ORB-SLAM2、ORB-SLAM3（增加视觉惯性支持）
- **优势**：对光照变化和动态物体鲁棒，定位精度高
- **局限**：在纹理缺失区域失效，特征提取和匹配计算量大

#### 直接法

直接利用像素灰度信息，通过最小化光度误差来估计相机运动，无需提取特征点。

- **LSD-SLAM（Large-Scale Direct SLAM）**：半稠密直接法 SLAM，估计具有不确定性的深度
- **DSO（Direct Sparse Odometry）**：稀疏直接法，联合优化光度误差和几何误差
- **优势**：可以利用图像中更多的信息（边缘、纹理渐变），生成更稠密的地图
- **局限****：对光照变化敏感，假设亮度恒定（光度标定要求高）

### 3.2 LiDAR SLAM

利用激光雷达点云数据进行定位和建图。

- **LOAM（Lidar Odometry and Mapping）**：将运动估计分为高频里程计和低频建图两个模块，利用边缘和平面特征进行点云配准
- **LeGO-LOAM**：LOAM 的改进版本，引入地面分割和聚类，在地面机器人上效果更好
- **LIO-SAM**：将 LOAM 与 IMU 紧耦合的 LiDAR-惯性里程计
- **特点**：LiDAR SLAM 精度高、鲁棒性好，是目前自动驾驶的主流定位方案

### 3.3 滤波方法（EKF-SLAM）

基于扩展卡尔曼滤波的 SLAM 方法：

- 状态向量包含机器人位姿和所有路标点的位置
- 每次观测后，通过 EKF 的预测-更新步骤同时估计位姿和路标
- **局限**：状态向量维度随路标数量增长，协方差矩阵为 $O(n^2)$，不适合大规模场景
- **应用**：适合小规模、低维度场景

### 3.4 图优化方法

将 SLAM 问题建模为图（Graph）上的非线性优化问题：

- **节点（Node）**：表示机器人的位姿和路标点
- **边（Edge）**：表示观测约束（里程计约束、回环约束）
- **优化目标**：最小化所有约束的总误差

**常用框架**：

- **g2o（General Graph Optimization）**：通用图优化库，支持各种类型的节点和边
- **GTSAM（Georgia Tech Smoothing and Mapping）**：基于因子图的平滑与建图库，利用稀疏矩阵高效求解
- **Ceres Solver**：Google 开源的非线性最小二乘求解器

图优化方法相比滤波方法的优势：全局一致性更好，适合大规模场景，是现代 SLAM 系统的主流方法。

### 3.5 回环检测

回环检测用于识别机器人是否回到了之前访问过的位置：

- **词袋模型（Bag of Words, BoW）**：将图像描述为视觉单词的直方图，通过词典匹配快速检索相似图像（如 DBoW2/DBoW3）
- **Scan Context**：基于 LiDAR 点云的全局描述子，用于 3D 回环检测
- **深度学习方法**：NetVLAD、LoST 等端到端的地点识别网络

回环检测的作用：消除累积漂移，保证全局地图的一致性。

## 4. 传感器融合

单一传感器无法满足自主系统在所有场景下的感知需求，传感器融合是必然选择。

### 4.1 卡尔曼滤波

**卡尔曼滤波（KF）**：适用于线性高斯系统的最优状态估计器。

- 预测步骤：$\hat{x}_{k|k-1} = F\hat{x}_{k-1} + Bu_k$
- 更新步骤：$\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k(z_k - H\hat{x}_{k|k-1})$
- 卡尔曼增益 $K_k$ 自动平衡预测与观测的置信度

**扩展卡尔曼滤波（EKF）**：对非线性系统在当前估计点进行一阶泰勒展开（雅可比矩阵线性化），然后应用标准 KF 框架。

- 广泛用于 IMU 与其他传感器的融合
- 线性化误差在强非线性场景下可能发散

**无迹卡尔曼滤波（UKF）**：通过确定性采样（Sigma 点）来近似非线性变换后的分布，避免了雅可比矩阵的计算。

- 在强非线性系统中通常优于 EKF
- 计算量略大于 EKF

### 4.2 粒子滤波

通过一组带权重的随机粒子来近似后验概率分布，适用于任意非线性、非高斯系统。

- 不需要对系统做线性化假设
- 可以处理多模态分布（如在多个可能位置同时存在概率质量）
- **局限**：粒子退化（大部分粒子权重趋近于零）、高维空间需要大量粒子（维度灾难）
- **应用**：全局定位（ kidnapped robot problem）、蒙特卡罗定位

### 4.3 多传感器时间同步与标定

**时间同步**：

- 不同传感器的采样频率不同（摄像头 30Hz，LiDAR 10Hz，IMU 200Hz）
- 需要硬件同步（如 PPS 脉冲信号）或软件插值（时间戳对齐）
- 时间误差会导致运动畸变和配准失败

**传感器标定**：

- **内参标定**：摄像头内参矩阵、畸变系数（棋盘格标定法）
- **外参标定**：传感器之间的空间变换关系（LiDAR-相机、相机-IMU）
- **在线标定**：在运行过程中自动估计外参的变化（如振动导致的标定漂移）

## 5. 感知算法

### 5.1 目标检测

- **YOLO 系列**：单阶段检测器，实时性好。从 YOLOv1 到 YOLOv8/v9/v11，精度和速度不断提升
- **PointPillars**：基于 LiDAR 点云的 3D 目标检测，将点云编码为伪图像（柱体化），使用 2D CNN 处理，速度极快
- **CenterPoint**：基于中心点的 3D 目标检测，在 nuScenes 等基准上表现优异
- **BEVFusion**：将摄像头和 LiDAR 特征统一到 BEV（鸟瞰图）空间进行融合检测

### 5.2 语义分割

- 将图像中每个像素分类到预定义的类别（道路、行人、车辆、建筑等）
- **代表模型**：DeepLab 系列、U-Net、SegFormer（基于 Vision Transformer）
- **应用**：可行驶区域检测、场景理解

### 5.3 3D 点云处理

- **PointNet/PointNet++**：直接处理无序点集的深度学习网络，学习逐点特征和全局特征
- **体素化方法**：将点云转化为规则的 3D 体素网格，使用 3D 卷积处理（如 VoxelNet）
- **稀疏卷积**：MinkowskiNet、SparseConvNet，高效处理稀疏 3D 数据
- **点云配准**：ICP（迭代最近点）算法及其变体，用于对齐两片点云

### 5.4 车道线检测

- 传统方法：基于边缘检测 + 霍夫变换
- 深度学习方法：
  - **语义分割方式**：逐像素分类
  - **多项式回归方式**：直接输出车道线参数
  - **LaneATT / CLRNet**：基于锚点的车道线检测，使用注意力机制聚合全局信息
