# 装饰器与数据驱动

## 核心概念

装饰器模式通过将对象包装在装饰器中来动态添加功能，不修改原有对象结构。在游戏开发中，这一模式广泛用于 Buff/Debuff 系统、装备属性叠加、技能强化和天赋树——任何需要"效果可以自由组合叠加"的场景。

### 为什么需要装饰器？

```
传统方式——硬编码属性计算：
float GetFinalDamage()
{
    float dmg = baseDamage;
    if (hasSword) dmg += 20;
    if (hasRing) dmg *= 1.15f;
    if (hasBuff) dmg *= 1.5f;
    if (hasDebuff) dmg *= 0.8f;
    // 新增一个装备？改代码。
    // 新增一个Buff？改代码。
    // 装备叠加规则变化？改代码。
}

装饰器模式——效果即数据：
float GetFinalDamage()
{
    float dmg = baseDamage;
    foreach (var modifier in modifiers) // modifiers 是动态列表
        dmg = modifier.Modify(dmg);
    return dmg;
}
// 新增效果？添加新的 modifier 实例，不改代码。
```

### 属性修饰器系统

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

// ========== 修饰器接口 ==========
public interface IStatModifier
{
    /// <summary>
    /// 修改属性值。输入当前累积值，返回修改后的值。
    /// </summary>
    float Modify(float currentValue);

    /// <summary>
    /// 执行顺序——数值小的先执行
    /// </summary>
    int Order { get; }

    /// <summary>
    /// 来源标识（用于调试和UI显示："攻击力+20 来自 力量之剑"）
    /// </summary>
    string Source { get; }
}

// ========== 修饰器类型 ==========

/// <summary>
/// 固定值加成：直接加/减一个固定值
/// 例：攻击力+20
/// </summary>
public class FlatModifier : IStatModifier
{
    public float Amount;
    public string Source { get; set; }

    public int Order => 0; // 最先执行

    public float Modify(float currentValue) => currentValue + Amount;

    public FlatModifier(float amount, string source = "")
    {
        Amount = amount;
        Source = source;
    }
}

/// <summary>
/// 百分比加法：所有加法百分比累加后一起乘
/// 例：+10%攻击力、+15%攻击力 → 总共+25%
/// </summary>
public class PercentAddModifier : IStatModifier
{
    public float Percent; // 0.1 = +10%, -0.2 = -20%
    public string Source { get; set; }

    public int Order => 10; // 在固定值之后执行

    public float Modify(float currentValue) => currentValue * (1f + Percent);

    public PercentAddModifier(float percent, string source = "")
    {
        Percent = percent;
        Source = source;
    }
}

/// <summary>
/// 乘法修饰：独立乘以一个系数
/// 例：Buff"狂化"：最终伤害×1.5
/// </summary>
public class PercentMultModifier : IStatModifier
{
    public float Multiplier; // 1.5 = ×150%
    public string Source { get; set; }

    public int Order => 20; // 最后执行

    public float Modify(float currentValue) => currentValue * Multiplier;

    public PercentMultModifier(float multiplier, string source = "")
    {
        Multiplier = multiplier;
        Source = source;
    }
}
```

### 完整属性计算器

```csharp
/// <summary>
/// 属性计算器——管理修饰器列表，按优先级计算最终值
/// </summary>
public class StatCalculator
{
    private float baseValue;
    private List<IStatModifier> modifiers = new();
    private bool needsRecalculation = true;
    private float cachedValue;

    public event Action<float> OnValueChanged;

    public StatCalculator(float baseValue)
    {
        this.baseValue = baseValue;
    }

    public float BaseValue
    {
        get => baseValue;
        set
        {
            baseValue = value;
            needsRecalculation = true;
        }
    }

    /// <summary>
    /// 添加修饰器
    /// </summary>
    public void AddModifier(IStatModifier modifier)
    {
        modifiers.Add(modifier);
        // 保持按 Order 排序
        modifiers.Sort((a, b) => a.Order.CompareTo(b.Order));
        needsRecalculation = true;
        OnValueChanged?.Invoke(Calculate());
    }

    /// <summary>
    /// 移除修饰器
    /// </summary>
    public bool RemoveModifier(IStatModifier modifier)
    {
        bool removed = modifiers.Remove(modifier);
        if (removed)
        {
            needsRecalculation = true;
            OnValueChanged?.Invoke(Calculate());
        }
        return removed;
    }

    /// <summary>
    /// 根据来源移除修饰器（如装备被卸下时移除该装备的所有修饰器）
    /// </summary>
    public int RemoveAllFromSource(string source)
    {
        int count = modifiers.RemoveAll(m => m.Source == source);
        if (count > 0)
        {
            needsRecalculation = true;
            OnValueChanged?.Invoke(Calculate());
        }
        return count;
    }

    /// <summary>
    /// 计算最终属性值
    /// </summary>
    public float Calculate()
    {
        if (!needsRecalculation) return cachedValue;

        float value = baseValue;

        // 按 Order 顺序处理修饰器
        // 关键技巧：将同类型修饰器分组处理
        float sumPercentAdd = 0f;

        foreach (var mod in modifiers)
        {
            if (mod is FlatModifier flat)
            {
                // 固定值直接加
                value += flat.Amount;
            }
            else if (mod is PercentAddModifier pctAdd)
            {
                // 累加百分比（不立即乘）
                sumPercentAdd += pctAdd.Percent;
            }
            else if (mod is PercentMultModifier pctMult)
            {
                // 独立乘法立即执行
                value *= pctMult.Multiplier;
            }
        }

        // 所有加法百分比最后一起乘
        value *= (1f + sumPercentAdd);

        cachedValue = value;
        needsRecalculation = false;
        return value;
    }

    /// <summary>
    /// 获取属性计算明细（调试用）
    /// </summary>
    public string GetBreakdown()
    {
        var lines = new List<string>();
        lines.Add($"基础值: {baseValue}");

        float value = baseValue;
        float sumPctAdd = 0f;

        foreach (var mod in modifiers)
        {
            if (mod is FlatModifier flat)
            {
                value += flat.Amount;
                lines.Add($"  + {flat.Amount} ({flat.Source}) = {value}");
            }
            else if (mod is PercentAddModifier pctAdd)
            {
                sumPctAdd += pctAdd.Percent;
                lines.Add($"  +{pctAdd.Percent*100}% ({pctAdd.Source}) [累加至 {sumPctAdd*100}%]");
            }
            else if (mod is PercentMultModifier pctMult)
            {
                value *= pctMult.Multiplier;
                lines.Add($"  ×{pctMult.Multiplier} ({pctMult.Source}) = {value}");
            }
        }

        value *= (1f + sumPctAdd);
        lines.Add($"最终值: {value}");
        return string.Join("\n", lines);
    }
}
```

### 完整属性计算示例

```csharp
// 属性计算过程：
// 基础攻击力: 100
//
// 装备"力量之剑": +20攻击力 (flat, Order=0)
//   100 + 20 = 120
//
// 装备"狂暴戒指": +15%攻击力 (percentAdd, Order=10)
//   累加: 15%
//
// 天赋"战士本能": +10%攻击力 (percentAdd, Order=10)
//   累加: 15% + 10% = 25%
//
// Buff"狂化": ×1.5 (percentMult, Order=20)
//   120 × 1.5 = 180
//
// 最终计算: 180 × (1 + 0.25) = 225

public class CharacterStats
{
    public StatCalculator Attack { get; private set; }
    public StatCalculator Defense { get; private set; }
    public StatCalculator Speed { get; private set; }
    public StatCalculator MaxHealth { get; private set; }

    public CharacterStats(CharacterClass charClass)
    {
        // 根据职业设置基础属性
        var baseStats = CharacterClassData.GetBaseStats(charClass);
        Attack = new StatCalculator(baseStats.Attack);
        Defense = new StatCalculator(baseStats.Defense);
        Speed = new StatCalculator(baseStats.Speed);
        MaxHealth = new StatCalculator(baseStats.Health);
    }

    public void ApplyEquipment(Equipment equip)
    {
        foreach (var mod in equip.Modifiers)
        {
            switch (mod.TargetStat)
            {
                case StatType.Attack: Attack.AddModifier(mod.Modifier); break;
                case StatType.Defense: Defense.AddModifier(mod.Modifier); break;
                case StatType.Speed: Speed.AddModifier(mod.Modifier); break;
                case StatType.Health: MaxHealth.AddModifier(mod.Modifier); break;
            }
        }
    }

    public void RemoveEquipment(Equipment equip)
    {
        Attack.RemoveAllFromSource(equip.Name);
        Defense.RemoveAllFromSource(equip.Name);
        Speed.RemoveAllFromSource(equip.Name);
        MaxHealth.RemoveAllFromSource(equip.Name);
    }
}
```

### Buff/Debuff 系统

```csharp
/// <summary>
/// Buff 系统——时间限制 + 可叠加 + 动态效果
/// </summary>
public class Buff
{
    public string Id;
    public string Name;
    public string Description;
    public float Duration;
    public float RemainingTime;
    public int MaxStacks;
    public int CurrentStacks;
    public BuffType Type;       // Buff 或 Debuff
    public StackRule StackRule;  // 叠加规则
    public List<BuffEffect> Effects;
    public GameObject Source;   // 施加来源

    public bool IsExpired => RemainingTime <= 0;

    /// <summary>
    /// 施加 Buff 到目标
    /// </summary>
    public void Apply(Character target)
    {
        switch (StackRule)
        {
            case StackRule.AddStack:
                // 叠加层数
                if (CurrentStacks < MaxStacks)
                    CurrentStacks++;
                RemainingTime = Duration; // 刷新持续时间
                break;

            case StackRule.RefreshDuration:
                // 不叠加层数，只刷新时间
                RemainingTime = Duration;
                break;

            case StackRule.Independent:
                // 独立计算，每个Buff实例互不影响
                break;

            case StackRule.TakeMax:
                // 取最高值
                break;
        }

        // 应用效果修饰器
        foreach (var effect in Effects)
        {
            var modifier = effect.CreateModifier(CurrentStacks);
            target.Stats.GetStat(effect.TargetStat).AddModifier(modifier);
        }
    }

    /// <summary>
    /// 每帧更新
    /// </summary>
    public void Update(float dt)
    {
        RemainingTime -= dt;

        // 持续伤害/治疗效果（如中毒每秒扣血）
        foreach (var effect in Effects)
        {
            if (effect.TickInterval > 0)
            {
                effect.TickTimer -= dt;
                if (effect.TickTimer <= 0)
                {
                    effect.TickTimer = effect.TickInterval;
                    effect.OnTick?.Invoke();
                }
            }
        }
    }

    /// <summary>
    /// 移除 Buff
    /// </summary>
    public void Remove(Character target)
    {
        foreach (var effect in Effects)
        {
            var modifier = effect.CreateModifier(CurrentStacks);
            target.Stats.GetStat(effect.TargetStat).RemoveModifier(modifier);
        }
    }
}

public enum BuffType { Buff, Debuff }
public enum StackRule { AddStack, RefreshDuration, Independent, TakeMax }

public class BuffEffect
{
    public StatType TargetStat;
    public IStatModifier TemplateModifier;
    public float TickInterval;
    public float TickTimer;
    public Action OnTick;

    public IStatModifier CreateModifier(int stacks)
    {
        // 根据层数缩放效果
        if (TemplateModifier is FlatModifier flat)
            return new FlatModifier(flat.Amount * stacks, flat.Source);
        if (TemplateModifier is PercentAddModifier pctAdd)
            return new PercentAddModifier(pctAdd.Percent * stacks, pctAdd.Source);
        return TemplateModifier;
    }
}

/// <summary>
/// Buff 管理器——管理角色身上的所有 Buff
/// </summary>
public class BuffManager
{
    private List<Buff> activeBuffs = new();
    private Character owner;

    public event Action<Buff> OnBuffApplied;
    public event Action<Buff> OnBuffRemoved;

    public BuffManager(Character owner) { this.owner = owner; }

    public void ApplyBuff(Buff buff)
    {
        // 检查是否已有同类Buff
        var existing = activeBuffs.FirstOrDefault(b => b.Id == buff.Id);
        if (existing != null)
        {
            existing.Apply(owner); // 叠加或刷新
        }
        else
        {
            buff.Apply(owner);
            activeBuffs.Add(buff);
        }
        OnBuffApplied?.Invoke(buff);
    }

    public void RemoveBuff(string buffId)
    {
        var buff = activeBuffs.FirstOrDefault(b => b.Id == buffId);
        if (buff != null)
        {
            buff.Remove(owner);
            activeBuffs.Remove(buff);
            OnBuffRemoved?.Invoke(buff);
        }
    }

    public void Update(float dt)
    {
        // 更新所有Buff
        for (int i = activeBuffs.Count - 1; i >= 0; i--)
        {
            activeBuffs[i].Update(dt);
            if (activeBuffs[i].IsExpired)
            {
                activeBuffs[i].Remove(owner);
                OnBuffRemoved?.Invoke(activeBuffs[i]);
                activeBuffs.RemoveAt(i);
            }
        }
    }

    /// <summary>
    /// 获取所有活跃Buff（用于UI显示）
    /// </summary>
    public List<Buff> GetActiveBuffs() => activeBuffs.ToList();

    public bool HasBuff(string buffId) => activeBuffs.Any(b => b.Id == buffId);
}
```

### 装备属性叠加系统

```csharp
/// <summary>
/// 装备系统——装备作为属性修饰器的载体
/// </summary>
public class Equipment
{
    public string Name;
    public EquipmentSlot Slot;
    public EquipmentRarity Rarity;
    public List<StatModifierEntry> Modifiers = new();

    public void Equip(Character character)
    {
        foreach (var entry in Modifiers)
        {
            entry.Modifier.Source = Name; // 标记来源
            character.Stats.GetStat(entry.TargetStat).AddModifier(entry.Modifier);
        }
    }

    public void Unequip(Character character)
    {
        character.Stats.Attack.RemoveAllFromSource(Name);
        character.Stats.Defense.RemoveAllFromSource(Name);
        character.Stats.Speed.RemoveAllFromSource(Name);
        character.Stats.MaxHealth.RemoveAllFromSource(Name);
    }
}

public class StatModifierEntry
{
    public StatType TargetStat;
    public IStatModifier Modifier;
}

public enum EquipmentSlot { Weapon, Helmet, Armor, Boots, Ring, Necklace }
public enum EquipmentRarity { Common, Uncommon, Rare, Epic, Legendary }
public enum StatType { Attack, Defense, Speed, Health, CritRate, CritDamage }
```

## 方案对比

| 方案 | 灵活性 | 性能 | 调试难度 | 实现复杂度 | 适用场景 |
|------|--------|------|---------|-----------|---------|
| if-else 硬编码 | 低 | 最高 | 低 | 无 | 效果固定不变 |
| 装饰器模式 | 高 | 中 | 中 | 中 | Buff、装备 |
| 数据驱动+修饰器 | 极高 | 中 | 高 | 高 | RPG属性系统 |
| ECS组件 | 极高 | 优秀 | 高 | 高 | 大型项目 |

## 常见陷阱与解决方案

1. **属性来源追踪困难**：不知道某个值是怎么算出来的。解决方案：实现 GetBreakdown() 方法，记录每个修饰器的来源
2. **叠加规则混乱**：不同效果的叠加规则不同。解决方案：明确定义每种修饰器的 Order 和计算方式
3. **Buff互斥/覆盖**：增益和减益效果冲突。解决方案：使用 Buff 分组和优先级系统
4. **性能问题**：大量修饰器导致每帧计算开销。解决方案：缓存计算结果，仅在修饰器变化时重算
5. **数据一致性**：装备修改了但修饰器没同步。解决方案：装备变更时统一触发重算

## 实际使用案例

- **《暗黑破坏神》** 的装备词缀系统本质上是属性装饰器
- **《英雄联盟》** 的 Buff 系统支持多种叠加规则（独立计算/取最大/叠加层数）
- **《流放之路》** 的天赋系统通过装饰器实现数千种效果组合
- **《原神》** 的元素反应系统通过装饰器链式叠加效果
- **《魔兽世界》** 的属性系统：装备+Buff+天赋+种族特性全部通过修饰器组合
