# 分词与词性标注


## 分词与词性标注


NLP分词NER


分词和词性标注是NLP的基础任务，为下游任务提供特征。


## 中文分词


```
中文分词的挑战：
- 词之间没有空格分隔
- 歧义切分：交叉歧义、组合歧义
- 未登录词（OOV）：新词、人名、地名

分词方法：
┌─────────────────────────────────────────────┐
│  1. 基于规则/词典                            │
│     最大正向匹配：从左到右，贪心最长匹配      │
│     最大逆向匹配：从右到左                    │
│     双向匹配：正向+逆向，选词数少的           │
│     优点：简单快速                            │
│     缺点：歧义处理差，OOV无法识别             │
│                                             │
│  2. 基于统计模型                              │
│     HMM分词：状态=BMES，观测=字符             │
│     CRF分词：特征丰富，效果更好               │
│     可学习字符级别的上下文信息                 │
│                                             │
│  3. 基于深度学习                              │
│     BiLSTM-CRF：序列标注                     │
│     BERT+CRF：预训练语言模型+标注             │
│     效果最好，但需要大量训练数据              │
└─────────────────────────────────────────────┘

HMM分词（BMES标注）：
B = 词首, M = 词中, E = 词尾, S = 单字词
"我爱北京天安门" → S B E B M M E
                   我 爱 北 京 天 安 门
利用Viterbi算法求最优标注序列
```


## 词性标注 (POS Tagging)


```
词性标注：为每个词赋予词性标签

中文词性标签集（北大标准部分）：
n-名词, v-动词, a-形容词, d-副词
p-介词, c-连词, u-助词, r-代词
m-数词, q-量词

示例：
我/r 爱/v 北京/ns 天安门/ns 的/u 早晨/t

方法：
┌─────────────────────────────────────────────┐
│  HMM词性标注：                                │
│  隐藏状态：词性标签                           │
│  观测：词语                                   │
│  P(词性序列|句子) ∝ Π P(词|词性)×P(词性|前一个)│
│  用Viterbi解码                                │
│                                             │
│  CRF词性标注：                                │
│  特征函数考虑更多上下文信息                    │
│  如：前后词、前后词性、词本身特征              │
│                                             │
│  Transformer/BERT标注：                       │
│  将词性标注转化为序列标注任务                  │
│  BERT编码 + 分类头                            │
└─────────────────────────────────────────────┘
```


## 命名实体识别 (NER)


```
NER：识别文本中的命名实体并分类

常见实体类型：
- PER（人名）：张三、李四
- LOC（地名）：北京、上海
- ORG（机构）：清华大学、华为
- TIME（时间）：2024年1月
- 数量、货币等

标注体系：BIO / BIOES
B-实体类型：实体开始
I-实体类型：实体内部
O：非实体
E-实体类型：实体结束（BIOES）
S-实体类型：单字实体（BIOES）

示例（BIOES）：
张 三 在 北 京 大 学 学 习
B-PER E-PER O B-LOC M-LOC M-LOC E-LOC O O

现代NER方法：
- BiLSTM-CRF：经典方法
- BERT-NER：微调BERT做序列标注
- 大语言模型：GPT-4等通过提示进行NER

分词工具：
- jieba：简单快速，规则+HMM
- pkuseg：北大，支持领域分词
- LAC：百度，分词+词性+NER
- spaCy：多语言，工业级
```


> **Note:** 大语言模型时代，传统NLP pipeline(分词→词性→NER→解析)在逐渐被端到端模型替代，
>         但理解底层原理仍然重要。
