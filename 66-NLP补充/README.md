# 64-NLP补充 — 自然语言处理进阶笔记

本目录为自然语言处理（NLP）领域的补充学习笔记，涵盖从经典词法句法分析到大语言模型技术的完整知识体系。

## 目录结构

| 序号 | 目录 | 主要内容 |
|------|------|----------|
| 01 | [词法与句法分析](01-词法与句法分析/README.md) | 中文分词、词性标注、命名实体识别、成分/依存句法分析、语义角色标注 |
| 02 | [语义分析与知识图谱](02-语义分析与知识图谱/README.md) | 词义消歧、语义相似度、指代消解、情感分析、知识图谱构建与应用 |

## 学习建议

1. **基础阶段**（01）：掌握NLP基本处理流水线，从分词到句法分析，理解语言的结构化表示。
2. **语义阶段**（02）：在句法基础上深入语义理解，学习知识的结构化表示与推理。
3. **前沿阶段**（03）：学习大语言模型的核心技术，包括预训练、微调、对齐与应用。
