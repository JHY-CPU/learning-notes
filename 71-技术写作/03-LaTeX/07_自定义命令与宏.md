# 自定义命令与宏

## newcommand 基础

`\newcommand` 用于定义新命令，是提高 LaTeX 文档可维护性的核心工具。

### 无参数命令

```latex
% 定义常用符号或短语
\newcommand{\R}{\mathbb{R}}            % 实数集
\newcommand{\N}{\mathbb{N}}            % 自然数集
\newcommand{\Z}{\mathbb{Z}}            % 整数集
\newcommand{\Q}{\mathbb{Q}}            % 有理数集
\newcommand{\C}{\mathbb{C}}            % 复数集

% 使用
设 $x \in \R$，则 $x^2 \geq 0$。
```

### 带参数的命令

```latex
% 单参数命令 #1 表示第一个参数
\newcommand{\abs}[1]{\left| #1 \right|}
\newcommand{\norm}[1]{\left\| #1 \right\|}
\newcommand{\inner}[2]{\left\langle #1, #2 \right\rangle}

% 使用
$\abs{x} \leq \norm{\mathbf{v}} \cdot \inner{u}{v}$

% 双参数命令 #1, #2 分别表示第一和第二个参数
\newcommand{\fracpart}[2]{\dfrac{\partial #1}{\partial #2}}
\newcommand{\ddfrac}[2]{\dfrac{d #1}{d #2}}

% 使用
$\fracpart{f}{x} + \fracpart{f}{y} = 0$
$\ddfrac{y}{x} = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x}$
```

### 带默认参数的命令

```latex
% 第一个参数有默认值，可选
\newcommand{\highlight}[2][yellow]{\colorbox{#1}{#2}}

% 使用
\highlight{重要文字}              % 默认黄色背景
\highlight[red]{警告文字}         % 红色背景

% 另一个例子
\newcommand{\keyword}[1][Definition]{\textbf{#1:}}
\keyword 算法复杂度是指...
\keyword[Theorem] 对于任意...
```

## renewcommand 重定义已有命令

```latex
% 修改已有命令的行为
\renewcommand{\abstractname}{摘\quad 要}   % 修改摘要标题
\renewcommand{\contentsname}{目录}          % 修改目录标题
\renewcommand{\figurename}{图}             % 修改图片标签
\renewcommand{\tablename}{表}              % 修改表格标签
\renewcommand{\refname}{参考文献}          % 修改参考文献标题

% 修改章节编号格式
\renewcommand{\thesection}{\thechapter.\arabic{section}}
\renewcommand{\theequation}{\thechapter.\arabic{equation}}

% 修改列表标记
\renewcommand{\labelitemi}{$\bullet$}
\renewcommand{\labelitemii}{$\circ$}
\renewcommand{\labelitemiii}{$\diamond$}
\renewcommand{\labelitemiv}{$\ast$}
```

## newenvironment 定义新环境

### 基本语法

```latex
% \newenvironment{环境名}[参数个数][默认值]{开始代码}{结束代码}

% 简单示例：高亮框
\newenvironment{important}{%
    \begin{center}
    \begin{tabular}{|p{0.9\textwidth}|}
    \hline
    \textbf{重要提示：} \\
}{%
    \hline
    \end{tabular}
    \end{center}
}

% 使用
\begin{important}
    请在提交前仔细检查所有参考文献。
\end{important}
```

### 带参数的环境

```latex
% 定义一个带标题的注释框
\newenvironment{notebox}[1]{%
    \medskip
    \noindent\textbf{#1}
    \begin{quote}
    \itshape
}{%
    \end{quote}
    \medskip
}

% 使用
\begin{notebox}{注意事项}
    此方法仅适用于线性可分的数据集。
\end{notebox}
```

### 定理类环境

```latex
\usepackage{amsthm}

% 定义新的定理环境
\newtheorem{theorem}{定理}[section]
\newtheorem{lemma}[theorem]{引理}
\newtheorem{corollary}[theorem]{推论}
\newtheorem{proposition}[theorem]{命题}
\newtheorem{definition}{定义}[section]
\newtheorem{remark}{注}[section]

% 不编号的环境
\newtheorem*{claim{claim}{断言}

% 使用
\begin{theorem}[勾股定理]
\label{thm:pythagoras}
在直角三角形中，$a^2 + b^2 = c^2$。
\end{theorem}

\begin{proof}
利用面积法可证明。
\end{proof}
```

## 计数器

### 内置计数器

```latex
% LaTeX 内置计数器
% part, chapter, section, subsection, subsubsection
% paragraph, subparagraph
% page, equation, figure, table
% footnote, mpfootnote

% 修改计数器值
\setcounter{section}{0}        % 重置节计数器
\addtocounter{page}{100}       % 页码加 100

% 获取计数器值
\thesection                    % 当前节编号
\arabic{section}               % 阿拉伯数字
\alph{section}                 % 小写字母 (a, b, c)
\Alph{section}                 % 大写字母 (A, B, C)
\roman{section}                % 小写罗马 (i, ii, iii)
\Roman{section}                % 大写罗马 (I, II, III)
```

### 自定义计数器

```latex
% 定义新计数器
\newcounter{mycounter}
\newcounter{exercise}[section]    % 每节重新编号

% 设置初始值
\setcounter{mycounter}{0}

% 使用计数器
\stepcounter{exercise}            % 递增
\addtocounter{exercise}{2}        % 加 2
\refstepcounter{exercise}         % 递增并允许引用

% 显示计数器值
练习 \theexercise

% 引用计数器
\label{ex:first}
如练习~\ref{ex:first} 所示。
```

### 自定义练习题环境

```latex
\newcounter{exercise}[section]
\renewcommand{\theexercise}{\thesection.\arabic{exercise}}

\newenvironment{exercise}[1][]{%
    \refstepcounter{exercise}%
    \medskip\noindent
    \textbf{练习~\theexercise}%
    \ifx#1\empty\else\ (#1)\fi
    \par\medskip
}{%
    \medskip
}

% 使用
\begin{exercise}[基础]
    计算 $\int_0^1 x^2 \, dx$ 的值。
\end{exercise}

\begin{exercise}[进阶]
    证明 $\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$。
\end{exercise}
```

## 实用自定义命令集合

### 数学常用命令

```latex
% 向量
\newcommand{\vect}[1]{\boldsymbol{#1}}
\newcommand{\mat}[1]{\mathbf{#1}}
\newcommand{\grad}{\nabla}
\newcommand{\laplacian}{\nabla^2}

% 集合
\newcommand{\set}[1]{\{#1\}}
\newcommand{\bigO}{\mathcal{O}}

% 概率统计
\newcommand{\E}{\mathbb{E}}
\newcommand{\Var}{\mathrm{Var}}
\newcommand{\Cov}{\mathrm{Cov}}
\newcommand{\Prob}{\mathbb{P}}

% 微分
\newcommand{\dif}{\mathop{}\!\mathrm{d}}
\newcommand{\pdif}{\mathop{}\!\partial}

% 使用
$\E[X] = \int x \, p(x) \dif x$
$\grad \cdot \vect{E} = \frac{\rho}{\epsilon_0}$
算法复杂度为 $O(n \log n)$。
```

### 编程相关命令

```latex
% 代码关键词
\newcommand{\code}[1]{\texttt{#1}}
\newcommand{\func}[1]{\texttt{#1()}}
\newcommand{\var}[1]{\textit{#1}}
\newcommand{\const}[1]{\textsc{#1}}

% 使用
调用 \func{calculate} 函数，传入参数 \var{x}。
\const{MAX\_SIZE} 的值为 1024。
```

### 审稿与编辑命令

```latex
% 审稿标记（可用于论文修改阶段）
\newcommand{\todo}[1]{\textcolor{red}{[TODO: #1]}}
\newcommand{\changed}[1]{\textcolor{blue}{#1}}
\newcommand{\note}[1]{\marginpar{\footnotesize\textcolor{red}{#1}}}

% 注释掉某段文字
\newcommand{\ignore}[1]{}

% 使用
\todo{需要补充实验数据}
\changed{这是修改后的内容}
\note{检查此处}
```

## 条件定义与 xparse

```latex
% 使用 xparse 宏包定义更灵活的命令（LaTeX 2020+ 已内置）
\NewDocumentCommand{\mycommand}{O{default} m m}{%
    可选参数：#1, 必选参数：#2 和 #3%
}

\mycommand{第一个}{第二个}            % 可选参数使用默认值
\mycommand[自定义]{第一个}{第二个}

% 定义星号版本的命令
\NewDocumentCommand{\mystar}{s m}{%
    \IfBooleanTF{#1}{%
        带星号版本：\textbf{#2}%
    }{%
        普通版本：#2%
    }%
}

\mystar{文字}        % 普通版本
\mystar*{文字}       % 带星号版本
```
