# 学术论文模板

## IEEE 论文模板

IEEE 是电气电子工程师学会，其会议和期刊论文使用统一的模板格式。

```latex
% IEEE 会议论文
\documentclass[conference]{IEEEtran}

% IEEE 期刊论文
% \documentclass[journal]{IEEEtran}

\usepackage{cite}
\usepackage{amsmath,amssymb}
\usepackage{graphicx}
\usepackage{hyperref}
\usepackage[UTF8]{ctex}    % 中文支持（注意兼容性）

\begin{document}

% 标题
\title{论文标题：使用简洁明确的表述}

% 作者信息
\author{
    \IEEEauthorblockN{张三}
    \IEEEauthorblockA{清华大学计算机系\\
        北京 100084, 中国\\
        zhangsan@tsinghua.edu.cn}
    \and
    \IEEEauthorblockN{李四}
    \IEEEauthorblockA{北京大学信息学院\\
        北京 100871, 中国\\
        lisi@pku.edu.cn}
}

\maketitle

% 摘要
\begin{abstract}
本文提出了一种新的方法来解决某某问题。实验结果表明，该方法在三个基准
数据集上取得了最优性能，准确率相比现有方法提升了 5.3 个百分点。
\end{abstract}

% 关键词
\begin{IEEEkeywords}
深度学习, 注意力机制, 自然语言处理
\end{IEEEkeywords}

\section{引言}
\label{sec:intro}

% 使用 cite 宏包的引用
相关研究见文献~\cite{ref1,ref2}。

\section{相关工作}
\label{sec:related}

\section{方法}
\label{sec:method}

\section{实验}
\label{sec:experiment}

\section{结论}
\label{sec:conclusion}

% 参考文献
\bibliographystyle{IEEEtran}
\bibliography{references}

\end{document}
```

## ACM 论文模板

ACM（美国计算机协会）提供基于 acmart 文档类的统一模板。

```latex
\documentclass[manuscript, review]{acmart}
% 选项：
% manuscript  - 单栏投稿格式
% acmtog      - 双栏期刊格式
% sigconf     - 会议格式
% review      - 匿名评审模式

\usepackage{amsmath}
\usepackage{graphicx}

% ACM 元信息
\acmConference[MM '24]{ACM Multimedia}{October 2024}{Melbourne, Australia}
\acmYear{2024}
\copyrightyear{2024}
\acmISBN{978-1-23456-789-0/24/10}
\acmDOI{10.1145/1234567.1234568}

% CCS 概念分类（ACM 要求）
\begin{CCSXML}
<ccs2012>
    <concept>
        <concept_id>10010147.10010178</concept_id>
        <concept_desc>Computing methodologies~Machine learning</concept_desc>
        <concept_significance>500</concept_significance>
    </concept>
</ccs2012>
\end{CCSXML}

\ccsdesc[500]{Computing methodologies~Machine learning}

\begin{document}

\title{ACM 论文标题}

\author{张三}
\affiliation{
    \institution{清华大学}
    \city{北京}
    \country{中国}
}
\email{zhangsan@tsinghua.edu.cn}

\begin{abstract}
摘要内容。
\end{abstract}

\keywords{关键词1, 关键词2}

\maketitle

\section{引言}
正文内容。

% ACM 使用 natbib + abbrvnat
\bibliographystyle{ACM-Reference-Format}
\bibliography{references}

\end{document}
```

## Springer 论文模板

Springer 使用基于 llncs（计算机科学讲义）或 svjour3 文档类的模板。

```latex
% LNCS（计算机科学讲义）会议论文
\documentclass[runningheads]{llncs}

\usepackage{amsmath}
\usepackage{graphicx}
\usepackage{hyperref}
\usepackage[UTF8]{ctex}

\begin{document}

\title{LNCS 格式的论文标题}
\titlerunning{短标题（用于页眉）}

\author{张三\inst{1} \and 李四\inst{2}}
\authorrunning{张三等}
\institute{清华大学计算机系, 北京 100084, 中国
    \email{zhangsan@tsinghua.edu.cn}
    \and
    北京大学信息学院, 北京 100871, 中国}

\maketitle

\begin{abstract}
Springer LNCS 模板的摘要。
\keywords{关键词1 \and 关键词2}
\end{abstract}

\section{引言}

\section{方法}

\subsection{子方法}

\subsubsection{详细说明}

\begin{theorem}
这是一个定理。
\end{theorem}

\begin{proof}
证明过程。
\end{proof}

% LNCS 使用 splncs04 参考文献样式
\bibliographystyle{splncs04}
\bibliography{references}

\end{document}
```

## 作者与单位信息

### 多作者多单位

```latex
% IEEE 风格
\author{
    \IEEEauthorblockN{作者一\thanks{通讯作者}}
    \IEEEauthorblockA{单位一\\email@one.edu}
    \and
    \IEEEauthorblockN{作者二}
    \IEEEauthorblockA{单位二\\email@two.edu}
}

% 使用 authblk 宏包（更灵活）
\usepackage{authblk}
\author[1]{张三\thanks{通讯作者：zhangsan@tsinghua.edu.cn}}
\author[2]{李四}
\author[1,3]{王五}
\affil[1]{清华大学计算机系}
\affil[2]{北京大学信息学院}
\affil[3]{中国科学院计算所}

\renewcommand\Authands{和}
```

## 摘要与关键词

```latex
% 标准 abstract 环境
\begin{abstract}
摘要内容，通常 150-250 个英文单词。摘要应包含研究目的、方法、
主要结果和结论。避免使用缩写和引用。
\end{abstract}

% 关键词设置
\keywords{关键词1, 关键词2, 关键词3}

% IEEE 关键词环境
\begin{IEEEkeywords}
深度学习, 神经网络, 图像识别
\end{IEEEkeywords}

% 单独页面的摘要（某些学位论文要求）
\newpage
\begin{abstract}
这里是摘要内容。
\end{abstract}
\thispagestyle{empty}
```

## 脚注

```latex
% 基本脚注
这是一个带有脚注\footnote{这是脚注内容}的句子。

% 作者脚注（特殊符号）
\author{张三\footnotemark[1] \and 李四\footnotemark[2]}
\footnotetext[1]{通讯作者，邮箱：zhangsan@example.com}
\footnotetext[2]{现在工作单位：某公司}

% 表格脚注
\begin{table}[htbp]
    \centering
    \caption{带脚注的表格}
    \begin{tabular}{lc}
        \toprule
        项目 & 值\textsuperscript{a} \\
        \midrule
        A    & 100 \\
        B    & 200\textsuperscript{b} \\
        \bottomrule
    \end{tabular}
    \begin{flushleft}
        \scriptsize \textsuperscript{a}单位为百分比。
        \textsuperscript{b}估计值。
    \end{flushleft}
\end{table}
```

## 匿名评审模式

```latex
% IEEE 匿名审稿
\documentclass[conference, anonymous]{IEEEtran}

% ACM 匿名审稿
\documentclass[manuscript, review, anonymous]{acmart}

% 手动隐藏作者信息
\author{Anonymous Authors}
\affiliation{Anonymous Institution}
```

## 自定义论文模板结构

```latex
% mythesis.tex - 学位论文主文件
\documentclass[12pt, a4paper]{report}

\usepackage[UTF8]{ctex}
\usepackage{geometry}
\usepackage{amsmath}
\usepackage{graphicx}
\usepackage[numbers]{natbib}
\usepackage{hyperref}

% 自定义命令
\newcommand{\thesistitle}{基于深度学习的图像识别研究}
\newcommand{\thesisauthor}{张三}
\newcommand{\thesisadvisor}{李四教授}

\title{\thesistitle}
\author{\thesisauthor}
\date{\today}

\begin{document}

\frontmatter
\input{chapters/cover}           % 封面
\input{chapters/declaration}     % 声明
\input{chapters/abstract_zh}     % 中文摘要
\input{chapters/abstract_en}     % 英文摘要

\tableofcontents
\listoffigures
\listoftables

\mainmatter
\input{chapters/chap01_intro}    % 第一章
\input{chapters/chap02_related}  % 第二章
\input{chapters/chap03_method}   % 第三章
\input{chapters/chap04_exp}      % 第四章
\input{chapters/chap05_conc}     % 第五章

\backmatter
\bibliographystyle{plainnat}
\bibliography{references}

\appendix
\input{chapters/appendix_a}

\end{document}
```
