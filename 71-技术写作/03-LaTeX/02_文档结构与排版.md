# 文档结构与排版

## 文档类详解

### article 类

article 是最常用的文档类，适合期刊论文、会议论文和技术报告。

```latex
\documentclass[11pt, a4paper, twocolumn]{article}

\usepackage[UTF8]{ctex}
\usepackage{geometry}
\geometry{top=2.5cm, bottom=2.5cm, left=2.5cm, right=2.5cm}

% 文章标题信息
\title{文章标题}
\author{作者姓名 \\ 单位名称 \\ email@example.com}
\date{\today}

\begin{document}
\maketitle

\begin{abstract}
这是一段摘要，用于概述文章的主要内容和贡献。摘要通常不超过 200 字。
\end{abstract}

\keywords{关键词一 \and 关键词二 \and 关键词三}

\section{一级标题}
\subsection{二级标题}
\subsubsection{三级标题}

正文内容。

\end{document}
```

### report 类

report 适合较长的文档，支持 `\chapter` 命令，默认每章从新页开始。

```latex
\documentclass[12pt, a4paper]{report}

\usepackage[UTF8]{ctex}

\begin{document}

\tableofcontents    % 目录
\listoffigures      % 图目录
\listoftables       % 表目录

\chapter{绪论}
这是第一章的内容。

\section{研究背景}
\subsection{国内外研究现状}

\chapter{相关工作}
这是第二章的内容。

\end{document}
```

### book 类

book 类用于排版书籍，支持前后辅文（frontmatter/mainmatter/backmatter）。

```latex
\documentclass[12pt, twoside]{book}

\usepackage[UTF8]{ctex}

\begin{document}

\frontmatter            % 前辅文：页码使用罗马数字
\maketitle
\tableofcontents

\mainmatter             % 正文：页码使用阿拉伯数字，从 1 开始

\part{基础篇}
\chapter{入门介绍}
正文内容。

\part{进阶篇}
\chapter{高级主题}
正文内容。

\backmatter             % 后辅文：不编号的附录等
\appendix
\chapter{附录A}

\bibliography{references}

\end{document}
```

## 页面布局

### geometry 宏包

```latex
\usepackage{geometry}

% 方式一：通过参数设置
\geometry{
    a4paper,
    top=2.5cm,
    bottom=2.5cm,
    left=3cm,
    right=2.5cm,
    headheight=15pt
}

% 方式二：使用预设
\geometry{margin=2.5cm}
\geometry{left=3cm, right=2cm}

% 双面排版时的对称页边距
\geometry{
    a4paper,
    inner=3cm,     % 装订侧
    outer=2.5cm,
    top=2.5cm,
    bottom=2.5cm
}
```

### 页眉页脚（fancyhdr）

```latex
\usepackage{fancyhdr}
\pagestyle{fancy}
\fancyhf{}                          % 清空默认设置
\fancyhead[L]{\leftmark}           % 左页眉：章节名
\fancyhead[R]{\rightmark}          % 右页眉：小节名
\fancyfoot[C]{\thepage}            % 中间页脚：页码
\renewcommand{\headrulewidth}{0.4pt} % 页眉横线宽度

% 章首页使用 plain 样式
\fancypagestyle{plain}{
    \fancyhf{}
    \fancyfoot[C]{\thepage}
    \renewcommand{\headrulewidth}{0pt}
}
```

## 字体设置

### 使用 ctex 宏包（推荐）

```latex
\usepackage[UTF8, fontset=ubuntu]{ctex}
% fontset 可选值：ubuntu, mac, windows, fandol, none

% 自定义字体
\setCJKmainfont{Noto Serif CJK SC}[
    BoldFont = Noto Sans CJK SC Bold,
    ItalicFont = Noto Serif CJK SC
]
\setCJKsansfont{Noto Sans CJK SC}
\setCJKmonofont{Noto Sans Mono CJK SC}
```

### 使用 xeCJK 宏包（更底层的控制）

```latex
\usepackage{xeCJK}
\setCJKmainfont{SimSun}[BoldFont=SimHei, ItalicFont=KaiTi]
\setCJKsansfont{SimHei}
\setCJKmonofont{FangSong}

% 英文字体设置
\setmainfont{Times New Roman}
\setsansfont{Arial}
\setmonofont{Courier New}
```

### 字号与样式命令

```latex
% 字号命令（从小到大）
{\tiny 极小}
{\scriptsize 脚本大小}
{\footnotesize 脚注大小}
{\small 小}
{\normalsize 正常}
{\large 大}
{\Large 更大}
{\LARGE 非常大}
{\huge 巨大}
{\Huge 超大}

% 字体样式
\textbf{粗体}          % Bold
\textit{斜体}          % Italic
\underline{下划线}     % Underline
\textsc{小型大写字母}  % Small Caps
\emph{强调}            % Emphasis（通常是斜体）

% 字体大小指定
\fontsize{14pt}{21pt}\selectfont  % 字号 14pt，行距 21pt
```

## 段落与行距

```latex
% 行距设置
\usepackage{setspace}
\onehalfspacing      % 1.5 倍行距
\doublespacing       % 双倍行距
\setstretch{1.25}    % 自定义倍率

% 段落缩进
\setlength{\parindent}{2em}    % 首行缩进 2 个字符
\setlength{\parskip}{0.5em}    % 段间距

% 取消某段缩进
\noindent 这段不会缩进。

% 强制换行，但不开始新段落
第一行\\第二行

% 分页
\newpage
\clearpage             % 清除浮动体后再分页
```

## 中文排版特殊处理

```latex
\usepackage[UTF8]{ctex}

% 禁用中英文间距
\usepackage[
    autoindent=false,
    checking=false
]{xeCJK}

% 中文数字编号
\ctexset{
    section = {
        name = {第,节},
        number = \chinese{section}
    },
    subsection = {
        name = {,},
        number = \arabic{subsection}
    }
}

% 中文引号
``这是中文引号的用法''
