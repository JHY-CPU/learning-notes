# 矩与矩母函数

*随机变量与分布 · 矩、矩母函数与特征函数*


### 1. 原点矩（Raw Moment）


随机变量 X 的 k 阶原点矩定义为：


$$
μₖ′ = E(Xᵏ)  （离散：Σ xᵢᵏpᵢ   连续：∫ xᵏf(x)dx）
$$


- 一阶原点矩 = 数学期望：μ₁′ = E(X)
- 二阶原点矩：μ₂′ = E(X²) = D(X) + [E(X)]²


### 2. 中心矩（Central Moment）


随机变量 X 的 k 阶中心矩定义为：


$$
μₖ = E[(X - E(X))ᵏ] = E[(X - μ)ᵏ]
$$


- 一阶中心矩 = 0：μ₁ = E(X-μ) = 0
- 二阶中心矩 = 方差：μ₂ = E[(X-μ)²] = D(X) = σ²
- 三阶中心矩与
   **偏度**
   （skewness）相关
- 四阶中心矩与
   **峰度**
   （kurtosis）相关


### 3. 偏度与峰度


$$
偏度：γ₁ = μ₃ / σ³  （衡量分布的对称性）
            峰度：γ₂ = μ₄ / σ⁴  （衡量分布的尖峭程度）
$$


| 偏度 γ₁ | 含义 |
| --- | --- |
| γ₁ = 0 | 对称分布（如正态分布） |
| γ₁ > 0 | 右偏（正偏），长尾在右 |
| γ₁ < 0 | 左偏（负偏），长尾在左 |


正态分布的峰度 γ₂ = 3（超额峰度 = 0）。超额峰度 = γ₂ - 3。


### 4. 原点矩与中心矩的转换


$$
μ₂ = μ₂′ - (μ₁′)²    （方差 = E(X²) - [E(X)]²）
$$


$$
μ₃ = μ₃′ - 3μ₂′μ₁′ + 2(μ₁′)³
$$


### 1. 定义


随机变量 X 的矩母函数定义为：


$$
M(t) = M_X(t) = E(etX)
$$


离散型：M(t) = Σ e^txᵢ^ · pᵢ    连续型：M(t) = ∫ e^tx^ f(x) dx


矩母函数在 t=0 附近的某个邻域内存在（即上述积分/求和收敛）。


### 2. 用 MGF 求矩


矩母函数的核心用途——对其求导并在 t=0 处取值：


$$
M⁽ᵏ⁾(0) = E(Xᵏ) = k阶原点矩
$$


- M′(0) = E(X) = μ
- M″(0) = E(X²)
- D(X) = M″(0) - [M′(0)]²


### 3. MGF 的重要性质


| 性质 | 公式 |
| --- | --- |
| 线性变换 | M~aX+b~(t) = e^bt^ · M~X~(at) |
| 独立和 | M~X+Y~(t) = M~X~(t) · M~Y~(t)（X,Y独立） |
| 唯一性 | M~X~(t) = M~Y~(t) ⟹ F~X~(x) = F~Y~(y)（分布唯一确定） |


### 1. 定义


$$
φ(t) = φ_X(t) = E(eitX) = E(cos tX) + i·E(sin tX)
$$


其中 i = √(-1) 为虚数单位。特征函数是复值函数，对所有分布都存在（永远有定义）。


### 2. 特征函数 vs 矩母函数


| 对比 | 矩母函数 M(t) | 特征函数 φ(t) |
| --- | --- | --- |
| 定义 | E(e^tX^) | E(e^itX^) |
| 存在性 | 不一定存在（可能发散） | 对所有分布都存在 |
| 取值 | 实值 | 复值 |
| 关系 | φ(t) = M(it)（将 MGF 中 t 换为 it） |
| 唯一性 | 有（在存在时） | 有（总是成立） |
| 求矩 | M⁽ᵏ⁾(0) = E(Xᵏ) | φ⁽ᵏ⁾(0) = iᵏE(Xᵏ) |


### 3. 特征函数的性质


- |φ(t)| ≤ 1，且 φ(0) = 1
- φ(-t) = φ̄(t)（共轭对称）
- φ
   ~aX+b~
   (t) = e
   ^ibt^
   · φ
   ~X~
   (at)
- 独立和：φ
   ~X+Y~
   (t) = φ
   ~X~
   (t) · φ
   ~Y~
   (t)
- **连续性定理**
   ：φ
   ~Xₙ~
   (t) → φ
   ~X~
   (t)（逐点收敛）⟹ Xₙ → X（依分布收敛）


### 核心结论


$$
随机变量的分布函数 与 其矩母函数（或特征函数） 一一对应
$$


具体表述：


- **MGF 唯一性**
   ：若 M
   ~X~
   (t) = M
   ~Y~
   (t) 对某邻域内所有 t 成立，则 F
   ~X~
   = F
   ~Y~
- **CF 唯一性**
   ：若 φ
   ~X~
   (t) = φ
   ~Y~
   (t) 对所有 t 成立，则 F
   ~X~
   = F
   ~Y~


这是用矩母函数/特征函数识别分布的理论基础。
