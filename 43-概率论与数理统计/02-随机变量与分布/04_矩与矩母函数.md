# 矩与矩母函数

*随机变量与分布 · 矩、矩母函数与特征函数*


### 1. 原点矩（Raw Moment）


随机变量 X 的 k 阶原点矩定义为：


$$
μₖ′ = E(Xᵏ)  （离散：Σ xᵢᵏpᵢ   连续：∫ xᵏf(x)dx）
$$


- 一阶原点矩 = 数学期望：μ₁′ = E(X)
- 二阶原点矩：μ₂′ = E(X²) = D(X) + [E(X)]²


### 2. 中心矩（Central Moment）


随机变量 X 的 k 阶中心矩定义为：


$$
μₖ = E[(X - E(X))ᵏ] = E[(X - μ)ᵏ]
$$


- 一阶中心矩 = 0：μ₁ = E(X-μ) = 0
- 二阶中心矩 = 方差：μ₂ = E[(X-μ)²] = D(X) = σ²
- 三阶中心矩与
   **偏度**
   （skewness）相关
- 四阶中心矩与
   **峰度**
   （kurtosis）相关


### 3. 偏度与峰度


$$
偏度：γ₁ = μ₃ / σ³  （衡量分布的对称性）
            峰度：γ₂ = μ₄ / σ⁴  （衡量分布的尖峭程度）
$$


| 偏度 γ₁ | 含义 |
| --- | --- |
| γ₁ = 0 | 对称分布（如正态分布） |
| γ₁ > 0 | 右偏（正偏），长尾在右 |
| γ₁ < 0 | 左偏（负偏），长尾在左 |


正态分布的峰度 γ₂ = 3（超额峰度 = 0）。超额峰度 = γ₂ - 3。


### 4. 原点矩与中心矩的转换


$$
μ₂ = μ₂′ - (μ₁′)²    （方差 = E(X²) - [E(X)]²）
$$


$$
μ₃ = μ₃′ - 3μ₂′μ₁′ + 2(μ₁′)³
$$


### 1. 定义


随机变量 X 的矩母函数定义为：


$$
M(t) = M_X(t) = E(etX)
$$


离散型：M(t) = Σ e^txᵢ^ · pᵢ    连续型：M(t) = ∫ e^tx^ f(x) dx


矩母函数在 t=0 附近的某个邻域内存在（即上述积分/求和收敛）。


### 2. 用 MGF 求矩


矩母函数的核心用途——对其求导并在 t=0 处取值：


$$
M⁽ᵏ⁾(0) = E(Xᵏ) = k阶原点矩
$$


- M′(0) = E(X) = μ
- M″(0) = E(X²)
- D(X) = M″(0) - [M′(0)]²


### 3. MGF 的重要性质


| 性质 | 公式 |
| --- | --- |
| 线性变换 | M~aX+b~(t) = e^bt^ · M~X~(at) |
| 独立和 | M~X+Y~(t) = M~X~(t) · M~Y~(t)（X,Y独立） |
| 唯一性 | M~X~(t) = M~Y~(t) ⟹ F~X~(x) = F~Y~(y)（分布唯一确定） |


### 1. 定义


$$
φ(t) = φ_X(t) = E(eitX) = E(cos tX) + i·E(sin tX)
$$


其中 i = √(-1) 为虚数单位。特征函数是复值函数，对所有分布都存在（永远有定义）。


### 2. 特征函数 vs 矩母函数


| 对比 | 矩母函数 M(t) | 特征函数 φ(t) |
| --- | --- | --- |
| 定义 | E(e^tX^) | E(e^itX^) |
| 存在性 | 不一定存在（可能发散） | 对所有分布都存在 |
| 取值 | 实值 | 复值 |
| 关系 | φ(t) = M(it)（将 MGF 中 t 换为 it） |
| 唯一性 | 有（在存在时） | 有（总是成立） |
| 求矩 | M⁽ᵏ⁾(0) = E(Xᵏ) | φ⁽ᵏ⁾(0) = iᵏE(Xᵏ) |


### 3. 特征函数的性质


- |φ(t)| ≤ 1，且 φ(0) = 1
- φ(-t) = φ̄(t)（共轭对称）
- φ
   ~aX+b~
   (t) = e
   ^ibt^
   · φ
   ~X~
   (at)
- 独立和：φ
   ~X+Y~
   (t) = φ
   ~X~
   (t) · φ
   ~Y~
   (t)
- **连续性定理**
   ：φ
   ~Xₙ~
   (t) → φ
   ~X~
   (t)（逐点收敛）⟹ Xₙ → X（依分布收敛）


### 核心结论


$$
随机变量的分布函数 与 其矩母函数（或特征函数） 一一对应
$$


具体表述：


- **MGF 唯一性**
   ：若 M
   ~X~
   (t) = M
   ~Y~
   (t) 对某邻域内所有 t 成立，则 F
   ~X~
   = F
   ~Y~
- **CF 唯一性**
   ：若 φ
   ~X~
   (t) = φ
   ~Y~
   (t) 对所有 t 成立，则 F
   ~X~
   = F
   ~Y~


这是用矩母函数/特征函数识别分布的理论基础。


## Python实现

### 矩、矩母函数与特征函数

```python
import numpy as np
from scipy import stats
from scipy.special import factorial
import sympy as sp

# 用 scipy.stats 计算矩
print("=== 矩的计算 ===")
normal = stats.norm(2, 3)  # N(2, 9)
data = normal.rvs(size=100000, random_state=42)

# 原点矩
for k in [1, 2, 3, 4]:
    raw_moment = np.mean(data**k)
    print(f"{k}阶原点矩 ≈ {raw_moment:.4f}")

# 中心矩
mu = data.mean()
for k in [2, 3, 4]:
    central_moment = np.mean((data - mu)**k)
    print(f"{k}阶中心矩 ≈ {central_moment:.4f}")

# 偏度与峰度
from scipy.stats import skew, kurtosis
print(f"\n偏度 = {skew(data):.4f} (正态=0)")
print(f"超额峰度 = {kurtosis(data):.4f} (正态=0)")
```

### 矩母函数的符号计算

```python
import sympy as sp

t, x, lam = sp.symbols('t x lambda', positive=True)

# 泊松分布 P(λ) 的矩母函数
# M(t) = Σ e^(tk) * (λ^k * e^(-λ)) / k! = e^(λ(e^t - 1))
print("=== 矩母函数 ===")
# 验证泊松分布的 MGF
k = sp.Symbol('k', integer=True, nonneg=True)
# M(t) = e^(λ(e^t - 1))
M_poisson = sp.exp(lam * (sp.exp(t) - 1))
print(f"泊松分布 MGF: M(t) = {M_poisson}")

# 用 MGF 求矩: M'(0) = E(X)
M_prime = sp.diff(M_poisson, t)
E_X = M_prime.subs(t, 0)
print(f"M'(0) = {sp.simplify(E_X)} = λ (期望)")

M_double_prime = sp.diff(M_poisson, t, 2)
E_X2 = M_double_prime.subs(t, 0)
print(f"M''(0) = {sp.simplify(E_X2)} = λ + λ² (二阶矩)")
D_X = sp.simplify(E_X2 - E_X**2)
print(f"D(X) = M''(0) - [M'(0)]² = {D_X} = λ (方差)")

# 特征函数: φ(t) = M(it)
phi_poisson = M_poisson.subs(t, sp.I * sp.Symbol('omega'))
print(f"\n泊松分布特征函数: φ(ω) = e^(λ(e^(iω) - 1))")
```
