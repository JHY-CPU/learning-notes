# 连续型随机变量分布

*随机变量与分布 · PDF、CDF 与常见连续分布*


### 1. 概率密度函数（PDF）


连续型随机变量 X 的概率密度函数 f(x) 满足：


- f(x) ≥ 0（非负性）
- ∫₋∞⁺∞ f(x)dx = 1（归一性）
- P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx


注意：P(X = a) = 0，即连续型随机变量取任何单个值的概率为零。


### 2. 累积分布函数（CDF）


$$
F(x) = P(X ≤ x) = ∫₋∞ˣ f(t)dt
$$


性质：F(x) 单调不减、0 ≤ F(x) ≤ 1、F(-∞)=0、F(+∞)=1、F'(x) = f(x)


### 3. 期望与方差


$$
E(X) = ∫₋∞⁺∞ x·f(x)dx      D(X) = ∫₋∞⁺∞ [x - E(X)]²·f(x)dx
$$


### 定义与公式


随机变量 X 在区间 [a, b] 上均匀取值。


$$
f(x) = { 1/(b-a),   a ≤ x ≤ b
                     { 0,         其他

            F(x) = { 0,              x < a
                     { (x-a)/(b-a),   a ≤ x ≤ b
                     { 1,              x > b

            E(X) = (a+b)/2,   D(X) = (b-a)²/12
$$

X ~ U(a, b)

### 定义与公式


常用于描述等待时间、寿命等。参数 λ > 0。


$$
f(x) = { λe⁻λˣ,   x ≥ 0
                     { 0,        x < 0

            F(x) = { 1 - e⁻λˣ,   x ≥ 0
                     { 0,           x < 0

            E(X) = 1/λ,   D(X) = 1/λ²
$$

X ~ Exp(λ)

### 无记忆性


$$
P(X > s+t | X > s) = P(X > t) = e⁻λᵗ
$$


与几何分布类似，指数分布也具有无记忆性，是连续情形下唯一具有无记忆性的分布。


### 1. 定义与公式


最重要的连续型分布，由均值 μ 和方差 σ² 完全确定。


$$
f(x) = (1 / (σ√(2π))) · e^(-(x-μ)² / (2σ²)),   -∞ < x < +∞

            E(X) = μ,   D(X) = σ²
$$

X ~ N(μ, σ²)

概率密度函数关于 x = μ 对称，呈钟形曲线。


### 2. 标准正态分布 N(0, 1)


当 μ = 0，σ = 1 时，称为标准正态分布，记 Φ(x) 为其 CDF。


$$
φ(x) = (1/√(2π)) · e^(-x²/2)

            Φ(-x) = 1 - Φ(x)   （对称性）
$$


### 3. 标准化


若 X ~ N(μ, σ²)，则：


$$
Z = (X - μ) / σ ~ N(0, 1)

            P(a ≤ X ≤ b) = Φ((b-μ)/σ) - Φ((a-μ)/σ)
$$


### 4. 3σ 原则


| 区间 | 概率 |
| --- | --- |
| (μ - σ, μ + σ) | ≈ 68.27% |
| (μ - 2σ, μ + 2σ) | ≈ 95.45% |
| (μ - 3σ, μ + 3σ) | ≈ 99.73% |


| 分布 | 记号 | 参数 | E(X) | D(X) | 典型场景 |
| --- | --- | --- | --- | --- | --- |
| 均匀分布 | U(a,b) | a, b | (a+b)/2 | (b-a)²/12 | 等可能取值 |
| 指数分布 | Exp(λ) | λ | 1/λ | 1/λ² | 寿命、等待时间 |
| 正态分布 | N(μ,σ²) | μ, σ² | μ | σ² | 自然/社会数据 |


## Python实现

### 常见连续分布 (SciPy)

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 均匀分布 U(a, b)
a, b = 2, 8
uniform = stats.uniform(a, b - a)
print(f"=== 均匀分布 U({a}, {b}) ===")
print(f"P(X<5) = {uniform.cdf(5):.6f}")
print(f"E(X) = {uniform.mean():.2f}, D(X) = {uniform.var():.4f}")

# 指数分布 Exp(λ)
lam = 2.0
exp_dist = stats.expon(scale=1/lam)
print(f"\n=== 指数分布 Exp({lam}) ===")
print(f"P(X>1) = {1 - exp_dist.cdf(1):.6f} = e^(-{lam}) = {np.exp(-lam):.6f}")
print(f"E(X) = {exp_dist.mean():.4f} = 1/λ = {1/lam:.4f}")
# 无记忆性
s, t = 1, 2
P_conditional = (1 - exp_dist.cdf(s+t)) / (1 - exp_dist.cdf(s))
P_original = 1 - exp_dist.cdf(t)
print(f"无记忆性: P(X>{s+t}|X>{s}) = {P_conditional:.6f} = P(X>{t}) = {P_original:.6f}")

# 正态分布 N(μ, σ²)
mu, sigma = 170, 8  # 身高分布
normal = stats.norm(mu, sigma)
print(f"\n=== 正态分布 N({mu}, {sigma}²) ===")
print(f"P(X < 160) = {normal.cdf(160):.6f}")
print(f"P(165 < X < 175) = {normal.cdf(175) - normal.cdf(165):.6f}")

# 3σ 厘则
for k in [1, 2, 3]:
    prob = normal.cdf(mu + k*sigma) - normal.cdf(mu - k*sigma)
    print(f"P(μ-{k}σ < X < μ+{k}σ) = {prob:.6f}")

# 可视化
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
x = np.linspace(0, 10, 200)
axes[0].plot(x, uniform.pdf(x), 'b-')
axes[0].set_title('U(2, 8)')

x = np.linspace(0, 3, 200)
axes[1].plot(x, exp_dist.pdf(x), 'r-')
axes[1].set_title('Exp(2)')

x = np.linspace(140, 200, 200)
axes[2].plot(x, normal.pdf(x), 'g-')
axes[2].set_title('N(170, 64)')
plt.tight_layout()
plt.savefig('continuous_distributions.png', dpi=100)
plt.show()
```
