离散型随机变量分布

随机变量与分布 · 常见离散分布

一、离散型随机变量基本概念

1. 概率质量函数（PMF）

设离散型随机变量 X 的取值为 x₁, x₂, ..., xₙ，取各值的概率为：

P(X = xₖ) = pₖ,   k = 1, 2, ...

满足：pₖ ≥ 0，Σ pₖ = 1

2. 数学期望与方差

E(X) = Σ xₖ · pₖ      （期望 = 加权平均）
D(X) = E(X²) - [E(X)]² = Σ (xₖ - E(X))² · pₖ

性质：E(aX+b) = aE(X)+b，D(aX+b) = a²D(X)

二、伯努利分布（0-1分布）

定义与公式

一次试验只有两种结果：成功（概率 p）或失败（概率 1-p）。

X0（失败）1（成功）
P1-pp
P(X=k) = pᵏ(1-p)¹⁻ᵏ,   k = 0, 1

E(X) = p,   D(X) = p(1-p)
X ~ B(1, p)

三、二项分布

定义与公式

n 次独立重复伯努利试验中，成功次数 X 服从二项分布。

P(X = k) = C(n,k) · pᵏ · (1-p)ⁿ⁻ᵏ,   k = 0, 1, 2, ..., n

E(X) = np,   D(X) = np(1-p)
X ~ B(n, p)

记号：C(n,k) = n! / [k!(n-k)!]

例题 1：二项分布计算

某射手命中率为 0.8，独立射击 5 次，求恰好命中 3 次的概率。

X ~ B(5, 0.8)
P(X=3) = C(5,3) × 0.8³ × 0.2² = 10 × 0.512 × 0.04 = 0.2048
二项分布近似： 当 n 很大、p 很小时（np 适中），二项分布可近似为泊松分布：B(n,p) ≈ Poisson(λ=np)。当 np 和 n(1-p) 都较大时，可近似为正态分布。

四、泊松分布

定义与公式

描述单位时间（或单位空间）内随机事件发生的次数。参数 λ > 0 表示平均发生率。

P(X = k) = (λᵏ · e⁻λ) / k!,   k = 0, 1, 2, ...

E(X) = λ,   D(X) = λ
X ~ P(λ)

泊松分布的期望和方差相等，这是其重要特征。

例题 2：泊松分布应用

某路口平均每小时通过 3 辆救护车，求一小时内恰好通过 5 辆的概率。

X ~ P(λ = 3)
P(X=5) = (3⁵ · e⁻³) / 5! = (243 × 0.0498) / 120 ≈ 0.1008
泊松分布的典型场景： 电话呼叫次数、网站访问量、放射性粒子衰变数、书页印刷错误数、路口到达车辆数等。

五、几何分布

定义与公式

在独立重复伯努利试验中，首次成功所需的试验次数。

P(X = k) = (1-p)ᵏ⁻¹ · p,   k = 1, 2, 3, ...

E(X) = 1/p,   D(X) = (1-p)/p²
X ~ Ge(p)

无记忆性

P(X > s+t | X > s) = P(X > t)

已知前 s 次都没成功，再等 t 次才首次成功的概率与从头开始等 t 次一样。

例题 3：几何分布

某人射击命中率为 0.6，求前 3 次都没命中、第 4 次才命中的概率。

P(X=4) = 0.4³ × 0.6 = 0.064 × 0.6 = 0.0384

六、超几何分布

定义与公式

N 件产品中有 M 件次品，从中不放回抽取 n 件，其中次品数 X 服从超几何分布。

P(X = k) = C(M,k) · C(N-M, n-k) / C(N,n)

E(X) = nM/N,   D(X) = n(M/N)(1-M/N)(N-n)/(N-1)
X ~ H(N, M, n)
超几何 vs 二项分布： 超几何分布对应"不放回"抽样，二项分布对应"有放回"抽样。当 N 远大于 n 时，超几何分布近似于二项分布 B(n, M/N)。

本节总结

  • 伯努利分布：单次试验，B(1,p)，E=p，D=p(1-p)
  • 二项分布：n 次独立重复试验的成功次数，B(n,p)，E=np，D=np(1-p)
  • 泊松分布：单位时间/空间事件发生数，P(λ)，E=D=λ
  • 几何分布：首次成功所需试验数，Ge(p)，E=1/p，有无记忆性
  • 超几何分布：不放回抽样中的次品数，H(N,M,n)
  • 近似关系：泊松近似二项（n大p小），正态近似二项（np和n(1-p)大）
