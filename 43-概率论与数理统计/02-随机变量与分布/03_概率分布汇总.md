# 概率分布汇总

*随机变量与分布 · 常见分布一览表*


### 1. 离散分布对照表


| 分布 | 记号 | PMF P(X=k) | E(X) | D(X) | MGF M(t) |
| --- | --- | --- | --- | --- | --- |
| 伯努利 | B(1,p) | p^k^(1-p)^1-k^, k=0,1 | p | p(1-p) | (1-p)+pe^t^ |
| 二项 | B(n,p) | C(n,k)p^k^(1-p)^n-k^ | np | np(1-p) | [(1-p)+pe^t^]^n^ |
| 泊松 | P(λ) | λ^k^e^-λ^/k! | λ | λ | e^λ(e^t^-1)^ |
| 几何 | Ge(p) | (1-p)^k-1^p, k≥1 | 1/p | (1-p)/p² | pe^t^/[1-(1-p)e^t^] |
| 负二项 | NB(r,p) | C(k-1,r-1)p^r^(1-p)^k-r^ | r/p | r(1-p)/p² | [pe^t^/(1-(1-p)e^t^)]^r^ |
| 超几何 | H(N,M,n) | C(M,k)C(N-M,n-k)/C(N,n) | nM/N | 略 | — |
| 均匀离散 | DU{a,b} | 1/(b-a+1), k=a,...,b | (a+b)/2 | [(b-a+1)²-1]/12 | — |


### 2. 离散分布关系图


- **伯努利 = 二项(n=1)**
   ：B(1,p) 是 B(n,p) 的特例
- **二项 → 泊松**
   ：n→∞, p→0, np=λ 固定时，B(n,p)→P(λ)（泊松定理）
- **几何 = 负二项(r=1)**
   ：NB(1,p) = Ge(p)
- **超几何 → 二项**
   ：N→∞, M/N→p 时，H(N,M,n)→B(n,p)
- **二项 → 正态**
   ：n 大且 np, n(1-p) 都 ≥5 时，B(n,p) ≈ N(np, np(1-p))


### 1. 连续分布对照表


| 分布 | 记号 | PDF f(x) | E(X) | D(X) | 取值范围 |
| --- | --- | --- | --- | --- | --- |
| 均匀 | U(a,b) | 1/(b-a) | (a+b)/2 | (b-a)²/12 | [a,b] |
| 指数 | Exp(λ) | λe^-λx^ | 1/λ | 1/λ² | [0,∞) |
| 正态 | N(μ,σ²) | (1/√(2πσ²))e^-(x-μ)²/(2σ²)^ | μ | σ² | (-∞,+∞) |
| 伽马 | Γ(α,β) | β^α^x^α-1^e^-βx^/Γ(α) | α/β | α/β² | (0,∞) |
| 贝塔 | Beta(a,b) | x^a-1^(1-x)^b-1^/B(a,b) | a/(a+b) | ab/[(a+b)²(a+b+1)] | (0,1) |
| 卡方 | χ²(n) | x^n/2-1^e^-x/2^/(2^n/2^Γ(n/2)) | n | 2n | (0,∞) |
| t分布 | t(n) | 复杂（见正文） | 0 (n>1) | n/(n-2) (n>2) | (-∞,+∞) |
| F分布 | F(m,n) | 复杂（见正文） | n/(n-2) | 略 | (0,∞) |


### 2. 连续分布关系图


- **指数 = 伽马(α=1)**
   ：Exp(λ) = Γ(1, λ)
- **卡方 = 伽马(n/2, 1/2)**
   ：χ²(n) = Γ(n/2, 1/2)
- **卡方可加**
   ：X~χ²(m), Y~χ²(n) 独立，则 X+Y~χ²(m+n)
- **正态平方 = 卡方**
   ：Z~N(0,1)，则 Z²~χ²(1)
- **t分布 → 正态**
   ：n→∞ 时，t(n)→N(0,1)
- **伽马(n,1) = n个独立Exp(1)之和**
- **指数分布无记忆性**
   ：P(X>s+t|X>s) = P(X>t)


### 卡方分布、t分布、F分布


| 分布 | 构造方式 | 自由度 | 关键性质 |
| --- | --- | --- | --- |
| χ²(n) | Z₁²+Z₂²+...+Zₙ²（Zᵢ~N(0,1)独立） | n | E=n, D=2n；可加性 |
| t(n) | Z/√(V/n)，Z~N(0,1)，V~χ²(n)，Z⊥V | n | 关于0对称；n→∞时趋近N(0,1) |
| F(m,n) | (U/m)/(V/n)，U~χ²(m)，V~χ²(n)，U⊥V | (m,n) | F~(n,m) = 1/F(m,n)；t²(n)=F(1,n) |


### 如何选择合适的分布模型


| 数据类型 | 场景 | 推荐分布 |
| --- | --- | --- |
| 二元结果 | 一次试验成功/失败 | 伯努利 B(1,p) |
| 计数（固定次数） | n 次试验中的成功次数 | 二项 B(n,p) |
| 计数（单位时间） | 单位时间内事件发生次数 | 泊松 P(λ) |
| 等待时间（首次成功） | 离散：首次成功所需试验次数 | 几何 Ge(p) |
| 等待时间（连续） | 连续等待时间/寿命 | 指数 Exp(λ) |
| 自然测量数据 | 身高、误差、温度等 | 正态 N(μ,σ²) |
| 比例/概率值 | 取值在(0,1)之间的随机变量 | 贝塔 Beta(a,b) |
| 不放回抽样 | 有限总体抽样中的次品数 | 超几何 H(N,M,n) |


## Python实现

### 分布族关系验证与可视化

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 验证分布族关系
print("=== 分布族关系验证 ===")

# 1. 伽马(α=1) = 指数分布
lam = 2
gamma_as_exp = stats.gamma(a=1, scale=1/lam)
expon = stats.expon(scale=1/lam)
x = 0.5
print(f"Gamma(1, {lam}) 的 PDF({x}) = {gamma_as_exp.pdf(x):.6f}")
print(f"Exp({lam}) 的 PDF({x}) = {expon.pdf(x):.6f}")

# 2. 卡方 = 伽马(n/2, 1/2)
n = 5
chi2 = stats.chi2(n)
gamma_as_chi2 = stats.gamma(a=n/2, scale=2)
x = 3.0
print(f"χ²({n}) 的 PDF({x}) = {chi2.pdf(x):.6f}")
print(f"Gamma({n/2}, 1/2) 的 PDF({x}) = {gamma_as_chi2.pdf(x):.6f}")

# 3. 正态平方 = 卡方(1)
np.random.seed(42)
Z = np.random.randn(100000)
Z_squared = Z**2
print(f"\nZ² 的均值: {Z_squared.mean():.4f} (期望 1)")
print(f"Z² 的方差: {Z_squared.var():.4f} (期望 2)")

# 4. 二项→正态近似
n_binom, p_binom = 100, 0.3
binom = stats.binom(n_binom, p_binom)
normal_approx = stats.norm(n_binom*p_binom, np.sqrt(n_binom*p_binom*(1-p_binom)))
print(f"\n=== 二项分布的正态近似 ===")
for k in [25, 30, 35]:
    exact = binom.pmf(k)
    approx = normal_approx.pdf(k)
    print(f"P(X={k}): 精确={exact:.6f}, 正态近似={approx:.6f}")

# 5. t分布→正态 (自由度增大)
print(f"\n=== t分布趋近正态 ===")
for df in [1, 5, 30, 100]:
    t_val = stats.t.ppf(0.975, df)
    z_val = stats.norm.ppf(0.975)
    print(f"t_0.025({df}) = {t_val:.4f}, z_0.025 = {z_val:.4f}")
```
