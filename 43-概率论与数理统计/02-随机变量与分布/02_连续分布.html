连续型随机变量分布

随机变量与分布 · PDF、CDF 与常见连续分布

一、连续型随机变量基本概念

1. 概率密度函数（PDF）

连续型随机变量 X 的概率密度函数 f(x) 满足：

  • f(x) ≥ 0（非负性）
  • ∫₋∞⁺∞ f(x)dx = 1（归一性）
  • P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx

注意：P(X = a) = 0，即连续型随机变量取任何单个值的概率为零。

2. 累积分布函数（CDF）

F(x) = P(X ≤ x) = ∫₋∞ˣ f(t)dt

性质：F(x) 单调不减、0 ≤ F(x) ≤ 1、F(-∞)=0、F(+∞)=1、F'(x) = f(x)

3. 期望与方差

E(X) = ∫₋∞⁺∞ x·f(x)dx      D(X) = ∫₋∞⁺∞ [x - E(X)]²·f(x)dx

二、均匀分布 U(a, b)

定义与公式

随机变量 X 在区间 [a, b] 上均匀取值。

f(x) = { 1/(b-a),   a ≤ x ≤ b
         { 0,         其他

F(x) = { 0,              x < a
         { (x-a)/(b-a),   a ≤ x ≤ b
         { 1,              x > b

E(X) = (a+b)/2,   D(X) = (b-a)²/12
X ~ U(a, b)
例题 1：均匀分布

公交车每 10 分钟一班，乘客随机到达车站，求等待时间不超过 3 分钟的概率。

X ~ U(0, 10)
P(X ≤ 3) = F(3) = (3-0)/(10-0) = 3/10 = 0.3

三、指数分布 Exp(λ)

定义与公式

常用于描述等待时间、寿命等。参数 λ > 0。

f(x) = { λe⁻λˣ,   x ≥ 0
         { 0,        x < 0

F(x) = { 1 - e⁻λˣ,   x ≥ 0
         { 0,           x < 0

E(X) = 1/λ,   D(X) = 1/λ²
X ~ Exp(λ)

无记忆性

P(X > s+t | X > s) = P(X > t) = e⁻λᵗ

与几何分布类似，指数分布也具有无记忆性，是连续情形下唯一具有无记忆性的分布。

例题 2：指数分布

某电子元件寿命 X（单位：小时）服从 λ = 0.001 的指数分布，求使用寿命超过 1000 小时的概率。

P(X > 1000) = e⁻⁰·⁰⁰¹ˣ¹⁰⁰⁰ = e⁻¹ ≈ 0.3679

四、正态分布 N(μ, σ²)

1. 定义与公式

最重要的连续型分布，由均值 μ 和方差 σ² 完全确定。

f(x) = (1 / (σ√(2π))) · e^(-(x-μ)² / (2σ²)),   -∞ < x < +∞

E(X) = μ,   D(X) = σ²
X ~ N(μ, σ²)

概率密度函数关于 x = μ 对称，呈钟形曲线。

2. 标准正态分布 N(0, 1)

当 μ = 0，σ = 1 时，称为标准正态分布，记 Φ(x) 为其 CDF。

φ(x) = (1/√(2π)) · e^(-x²/2)

Φ(-x) = 1 - Φ(x)   （对称性）

3. 标准化

若 X ~ N(μ, σ²)，则：

Z = (X - μ) / σ ~ N(0, 1)

P(a ≤ X ≤ b) = Φ((b-μ)/σ) - Φ((a-μ)/σ)

4. 3σ 原则

区间概率
(μ - σ, μ + σ)≈ 68.27%
(μ - 2σ, μ + 2σ)≈ 95.45%
(μ - 3σ, μ + 3σ)≈ 99.73%
例题 3：正态分布标准化

某班考试成绩 X ~ N(72, 100)（即 μ=72，σ=10），求成绩在 60~90 分之间的概率。

P(60 ≤ X ≤ 90) = Φ((90-72)/10) - Φ((60-72)/10)
                 = Φ(1.8) - Φ(-1.2)
                 = Φ(1.8) - [1 - Φ(1.2)]
                 = 0.9641 - (1 - 0.8849)
                 = 0.9641 - 0.1151 = 0.8490
例题 4：正态分布求分位数

某指标 X ~ N(100, 225)（σ=15），求 95% 的人低于的值。

需要求 x 使 P(X ≤ x) = 0.95
即 Φ((x-100)/15) = 0.95
查表：Φ(1.645) ≈ 0.95
(x-100)/15 = 1.645 → x = 100 + 15×1.645 = 124.675
正态分布的核心地位：
  • 自然界和社会现象中大量数据近似服从正态分布
  • 中心极限定理保证大量独立随机变量之和近似正态
  • 是许多统计推断方法（t检验、方差分析等）的基础
  • 任何正态分布都可通过标准化转化为标准正态分布

本节总结

  • PDF：f(x) ≥ 0，∫f(x)dx = 1，P(a≤X≤b) = ∫ₐᵇf(x)dx
  • CDF：F(x) = P(X≤x) = ∫₋∞ˣf(t)dt，F'(x) = f(x)
  • 均匀分布 U(a,b)：E=(a+b)/2，D=(b-a)²/12
  • 指数分布 Exp(λ)：E=1/λ，D=1/λ²，有无记忆性
  • 正态分布 N(μ,σ²)：最重要分布，标准化 Z=(X-μ)/σ~N(0,1)
  • 3σ 原则：68.27%、95.45%、99.73%
