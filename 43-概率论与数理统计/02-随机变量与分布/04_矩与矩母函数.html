矩与矩母函数

随机变量与分布 · 矩、矩母函数与特征函数

一、矩（Moments）

1. 原点矩（Raw Moment）

随机变量 X 的 k 阶原点矩定义为：

μₖ′ = E(Xᵏ)  （离散：Σ xᵢᵏpᵢ   连续：∫ xᵏf(x)dx）
  • 一阶原点矩 = 数学期望：μ₁′ = E(X)
  • 二阶原点矩：μ₂′ = E(X²) = D(X) + [E(X)]²

2. 中心矩（Central Moment）

随机变量 X 的 k 阶中心矩定义为：

μₖ = E[(X - E(X))ᵏ] = E[(X - μ)ᵏ]
  • 一阶中心矩 = 0：μ₁ = E(X-μ) = 0
  • 二阶中心矩 = 方差：μ₂ = E[(X-μ)²] = D(X) = σ²
  • 三阶中心矩与偏度（skewness）相关
  • 四阶中心矩与峰度（kurtosis）相关

3. 偏度与峰度

偏度：γ₁ = μ₃ / σ³  （衡量分布的对称性）

峰度：γ₂ = μ₄ / σ⁴  （衡量分布的尖峭程度）
偏度 γ₁含义
γ₁ = 0对称分布（如正态分布）
γ₁ > 0右偏（正偏），长尾在右
γ₁ < 0左偏（负偏），长尾在左

正态分布的峰度 γ₂ = 3（超额峰度 = 0）。超额峰度 = γ₂ - 3。

4. 原点矩与中心矩的转换

μ₂ = μ₂′ - (μ₁′)²    （方差 = E(X²) - [E(X)]²）
μ₃ = μ₃′ - 3μ₂′μ₁′ + 2(μ₁′)³
例题 1：计算各阶矩

X ~ U(0,1)，求各阶原点矩和中心矩。

原点矩：μₖ′ = E(Xᵏ) = ∫₀¹ xᵏ dx = 1/(k+1)
μ₁′ = 1/2（期望），μ₂′ = 1/3

中心矩：μ₂ = μ₂′ - (μ₁′)² = 1/3 - 1/4 = 1/12（方差）

二、矩母函数（Moment Generating Function）

1. 定义

随机变量 X 的矩母函数定义为：

M(t) = M_X(t) = E(etX)

离散型：M(t) = Σ etxᵢ · pᵢ    连续型：M(t) = ∫ etx f(x) dx

矩母函数在 t=0 附近的某个邻域内存在（即上述积分/求和收敛）。

2. 用 MGF 求矩

矩母函数的核心用途——对其求导并在 t=0 处取值：

M⁽ᵏ⁾(0) = E(Xᵏ) = k阶原点矩
  • M′(0) = E(X) = μ
  • M″(0) = E(X²)
  • D(X) = M″(0) - [M′(0)]²

3. MGF 的重要性质

性质公式
线性变换MaX+b(t) = ebt · MX(at)
独立和MX+Y(t) = MX(t) · MY(t)（X,Y独立）
唯一性MX(t) = MY(t) ⟹ FX(x) = FY(y)（分布唯一确定）
例题 2：正态分布的 MGF

求 X ~ N(μ,σ²) 的矩母函数，并用它求 E(X) 和 D(X)。

M(t) = E(etX) = eμt + σ²t²/2

M′(t) = (μ + σ²t) · eμt + σ²t²/2
M′(0) = μ → E(X) = μ

M″(t) = [σ² + (μ + σ²t)²] · eμt + σ²t²/2
M″(0) = σ² + μ² → D(X) = σ² + μ² - μ² = σ²
例题 3：利用 MGF 证明独立和

X~B(n,p)，Y~B(m,p) 且独立，证明 X+Y~B(n+m,p)。

MX(t) = [(1-p)+pet]n
MY(t) = [(1-p)+pet]m
MX+Y(t) = MX(t)·MY(t) = [(1-p)+pet]n+m
这正是 B(n+m,p) 的 MGF，由唯一性，X+Y ~ B(n+m,p)

三、特征函数（Characteristic Function）

1. 定义

φ(t) = φ_X(t) = E(eitX) = E(cos tX) + i·E(sin tX)

其中 i = √(-1) 为虚数单位。特征函数是复值函数，对所有分布都存在（永远有定义）。

2. 特征函数 vs 矩母函数

对比矩母函数 M(t)特征函数 φ(t)
定义E(etX)E(eitX)
存在性不一定存在（可能发散）对所有分布都存在
取值实值复值
关系φ(t) = M(it)（将 MGF 中 t 换为 it）
唯一性有（在存在时）有（总是成立）
求矩M⁽ᵏ⁾(0) = E(Xᵏ)φ⁽ᵏ⁾(0) = iᵏE(Xᵏ)

3. 特征函数的性质

  • |φ(t)| ≤ 1，且 φ(0) = 1
  • φ(-t) = φ̄(t)（共轭对称）
  • φaX+b(t) = eibt · φX(at)
  • 独立和：φX+Y(t) = φX(t) · φY(t)
  • 连续性定理：φXₙ(t) → φX(t)（逐点收敛）⟹ Xₙ → X（依分布收敛）

四、唯一性定理

核心结论

随机变量的分布函数 与 其矩母函数（或特征函数） 一一对应

具体表述：

  • MGF 唯一性：若 MX(t) = MY(t) 对某邻域内所有 t 成立，则 FX = FY
  • CF 唯一性：若 φX(t) = φY(t) 对所有 t 成立，则 FX = FY

这是用矩母函数/特征函数识别分布的理论基础。

常见分布的矩母函数速查：
  • B(n,p)：M(t) = [(1-p)+pet]n
  • P(λ)：M(t) = exp{λ(et-1)}
  • Exp(λ)：M(t) = λ/(λ-t)，t < λ
  • N(μ,σ²)：M(t) = exp{μt + σ²t²/2}
  • U(a,b)：M(t) = (ebt-eat)/[(b-a)t]

本节总结

  • 原点矩：μₖ′ = E(Xᵏ)，一阶=期望，二阶与方差相关
  • 中心矩：μₖ = E[(X-μ)ᵏ]，一阶=0，二阶=方差，三阶→偏度，四阶→峰度
  • 矩母函数：M(t)=E(etX)，M⁽ᵏ⁾(0) = k阶矩，独立和=MGF相乘
  • 特征函数：φ(t)=E(eitX)，永远存在，φ(t)=M(it)
  • 唯一性定理：MGF/CF 唯一确定分布函数
  • 连续性定理：特征函数逐点收敛 ⟹ 依分布收敛
