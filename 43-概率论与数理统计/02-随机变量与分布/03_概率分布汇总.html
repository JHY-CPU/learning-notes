2. 离散分布关系图

  • 伯努利 = 二项(n=1)：B(1,p) 是 B(n,p) 的特例
  • 二项 → 泊松：n→∞, p→0, np=λ 固定时，B(n,p)→P(λ)（泊松定理）
  • 几何 = 负二项(r=1)：NB(1,p) = Ge(p)
  • 超几何 → 二项：N→∞, M/N→p 时，H(N,M,n)→B(n,p)
  • 二项 → 正态：n 大且 np, n(1-p) 都 ≥5 时，B(n,p) ≈ N(np, np(1-p))

2. 连续分布关系图

  • 指数 = 伽马(α=1)：Exp(λ) = Γ(1, λ)
  • 卡方 = 伽马(n/2, 1/2)：χ²(n) = Γ(n/2, 1/2)
  • 卡方可加：X~χ²(m), Y~χ²(n) 独立，则 X+Y~χ²(m+n)
  • 正态平方 = 卡方：Z~N(0,1)，则 Z²~χ²(1)
  • t分布 → 正态：n→∞ 时，t(n)→N(0,1)
  • 伽马(n,1) = n个独立Exp(1)之和
  • 指数分布无记忆性：P(X>s+t|X>s) = P(X>t)

三、三大抽样分布

卡方分布、t分布、F分布

分布构造方式自由度关键性质
χ²(n) Z₁²+Z₂²+...+Zₙ²（Zᵢ~N(0,1)独立） n E=n, D=2n；可加性
t(n) Z/√(V/n)，Z~N(0,1)，V~χ²(n)，Z⊥V n 关于0对称；n→∞时趋近N(0,1)
F(m,n) (U/m)/(V/n)，U~χ²(m)，V~χ²(n)，U⊥V (m,n) F~(n,m) = 1/F(m,n)；t²(n)=F(1,n)
三大抽样分布的关系：
  • t²(n) = F(1,n)：t分布的平方服从F分布
  • 1/F(m,n) ~ F(n,m)：F分布的倒数仍为F分布
  • χ²(n) = Γ(n/2, 1/2)：卡方分布是伽马分布的特例
  • 正态总体下的样本均值和方差的分布必然涉及这三个分布

本节总结

  • 离散分布7种：伯努利、二项、泊松、几何、负二项、超几何、均匀离散
  • 连续分布8种：均匀、指数、正态、伽马、贝塔、卡方、t分布、F分布
  • 关键近似：泊松近似二项（n大p小）、正态近似二项（np和n(1-p)大）、正态近似泊松（λ大）
  • 三大抽样分布：χ²、t、F，由正态总体的样本构造
  • 分布族关系：伯努利⊂二项⊂泊松近似；指数=伽马(1)⊂伽马；χ²⊂伽马
