事件独立性与条件期望

概率基础 · 独立性判定与条件期望

一、事件的独立性

1. 两事件独立

设 A、B 为两个事件，若满足以下等价条件之一，则称 A 与 B 相互独立：

P(AB) = P(A) · P(B)

等价表述：

  • P(A|B) = P(A)，即 B 的发生不改变 A 的概率
  • P(B|A) = P(B)，即 A 的发生不改变 B 的概率

2. 独立性的重要推论

若 A 与 B 相互独立，则以下各对事件也相互独立：

事件对证明思路
A 与 B̄（B的对立事件）P(AB̄) = P(A) - P(AB) = P(A) - P(A)P(B) = P(A)[1-P(B)] = P(A)P(B̄)
Ā 与 B由对称性
Ā 与 B̄P(ĀB̄) = 1 - P(A∪B) = 1 - P(A) - P(B) + P(AB) = [1-P(A)][1-P(B)] = P(Ā)P(B̄)

3. 独立 vs 互斥（易混淆辨析）

概念定义条件能否同时成立
互斥A∩B = ∅P(AB) = 0一般不能同时成立（除非 P(A)=0 或 P(B)=0）
独立发生互不影响P(AB) = P(A)P(B)—
关键区分： 互斥事件"不能同时发生"，因此一个发生就决定了另一个不发生（互相影响），故互斥事件一定不独立（概率均非零时）。独立事件"可以同时发生但互不影响"。

4. 多事件独立

（1）两两独立

对任意 i ≠ j，有 P(AᵢAⱼ) = P(Aᵢ)P(Aⱼ)

（2）相互独立（全局独立）

对任意子集 {i₁, i₂, ..., iₖ}，有：

P(Aᵢ₁Aᵢ₂...Aᵢₖ) = P(Aᵢ₁)·P(Aᵢ₂)·...·P(Aᵢₖ)

两两独立 ≠ 相互独立！两两独立是必要条件，但不是充分条件。

例题 1：两两独立但不相互独立

掷两枚均匀硬币，定义事件 A={第一枚正面}，B={第二枚正面}，C={两枚结果相同}。验证 A,B,C 两两独立但不相互独立。

样本空间：{正正, 正反, 反正, 反反}，各概率 1/4
P(A)=1/2, P(B)=1/2, P(C)=1/2

两两独立验证：
P(AB) = 1/4 = P(A)P(B)   ✓
P(AC) = 1/4 = P(A)P(C)   ✓
P(BC) = 1/4 = P(B)P(C)   ✓

相互独立验证：
P(ABC) = P({正正}) = 1/4
P(A)P(B)P(C) = 1/2 × 1/2 × 1/2 = 1/8
P(ABC) ≠ P(A)P(B)P(C)，故不相互独立

二、条件期望（数学期望的条件版本）

1. 条件期望的定义

给定事件 B（P(B)>0），随机变量 X 关于 B 的条件期望定义为：

E(X|B) = Σ xₖ · P(X = xₖ | B)  （离散型）

E(X|B) = ∫₋∞⁺∞ x · f(x|B) dx  （连续型）

其中 P(X=xₖ|B) = P(X=xₖ, B)/P(B)，f(x|B) 为条件密度函数。

2. 条件期望的基本性质

  • E(aX + b | B) = aE(X|B) + b
  • E(X + Y | B) = E(X|B) + E(Y|B)
  • 若 X 与 B 独立，则 E(X|B) = E(X)
  • E(E(X|B)) = E(X)（需推广形式）
例题 2：条件期望计算

掷一颗均匀骰子，设事件 A = {点数为偶数}，求 X（骰子点数）关于 A 的条件期望。

A = {2, 4, 6}，P(A) = 3/6 = 1/2
P(X=2|A) = P(X=2, A)/P(A) = (1/6)/(1/2) = 1/3
P(X=4|A) = 1/3，P(X=6|A) = 1/3

E(X|A) = 2×(1/3) + 4×(1/3) + 6×(1/3) = 4
（对比无条件期望 E(X) = 3.5）

三、全期望公式（Law of Total Expectation）

1. 离散版本

设 B₁, B₂, ..., Bₙ 是样本空间的一个划分（互斥且穷尽），则：

E(X) = Σ E(X|Bᵢ) · P(Bᵢ)

这是全概率公式在期望层面的推广。

2. 随机变量版本（重期望公式）

设 Y 为随机变量，则以 Y 为条件：

E(X) = E[E(X|Y)]

含义：X 的无条件期望 = X 在每个 Y 值下条件期望的加权平均（权重为 Y 的分布）。

例题 3：全期望公式应用

某公司有两个部门，A部门有300人（平均工资6000元），B部门有200人（平均工资8000元）。随机选一人，求其工资期望。

设 Y 为部门（0=A, 1=B），X 为工资
P(Y=0) = 300/500 = 0.6，P(Y=1) = 200/500 = 0.4
E(X|Y=0) = 6000，E(X|Y=1) = 8000

E(X) = E(X|Y=0)·P(Y=0) + E(X|Y=1)·P(Y=1)
= 6000 × 0.6 + 8000 × 0.4 = 3600 + 3200 = 6800元
例题 4：随机个数之和的期望

某商店每天顾客数 N 服从参数 λ=10 的泊松分布，每个顾客消费金额 Xᵢ 服从均值 200 元的分布且相互独立。求总营业额 S = X₁+X₂+...+Xₙ 的期望。

利用重期望公式：E(S) = E[E(S|N)]
E(S|N=n) = E(X₁+...+Xₙ) = n·E(X₁) = 200n
E(S|N) = 200N
E(S) = E(200N) = 200·E(N) = 200 × 10 = 2000元

推广：若 N 与 Xᵢ 独立，则 E(S) = E(N)·E(X₁)（Wald等式）
全期望公式的应用场景：
  • 混合分布问题：总体期望 = 各层条件期望的加权平均
  • 随机个数随机变量之和：E(S) = E(N)·E(X₁)
  • 递推问题：利用条件期望建立递推关系
  • 优化决策：在不同情境下的期望收益加权

本节总结

  • 事件独立：P(AB) = P(A)P(B)，对立事件也独立
  • 独立 vs 互斥：互斥（不能同时发生）≠ 独立（互不影响），互斥且概率非零则一定不独立
  • 两两独立 ≠ 相互独立：相互独立要求所有子集的联合概率 = 各概率乘积
  • 条件期望：E(X|B) 在缩减样本空间中计算 X 的期望
  • 全期望公式：E(X) = Σ E(X|Bᵢ)P(Bᵢ) = E[E(X|Y)]
  • Wald等式：随机个数独立同分布之和的期望 = E(N)·E(X₁)
