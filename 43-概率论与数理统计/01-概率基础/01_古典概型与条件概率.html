古典概型与条件概率

概率论基础 · 核心概念与公式

一、古典概型（等可能概型）

1. 定义与条件

古典概型满足两个基本条件：

  • 有限性：样本空间中样本点的总数是有限的，即 Ω = {ω₁, ω₂, ..., ωₙ}
  • 等可能性：每个基本事件发生的概率相等，即 P(ω₁) = P(ω₂) = ... = P(ωₙ) = 1/n

2. 古典概率公式

P(A) = |A| / |Ω| = A中包含的基本事件数 / 样本空间中基本事件总数 = m / n

其中 |A| 表示事件 A 包含的样本点数，|Ω| 表示样本空间的总样本点数。

3. 计数方法

方法公式说明
排列（有序选取）P(n,m) = n! / (n-m)!从 n 个中取 m 个并排序
组合（无序选取）C(n,m) = n! / [m!(n-m)!]从 n 个中取 m 个不排序
可重复排列nmm 个球放入 n 个盒子
多重排列n! / (n₁!·n₂!·...·nₖ!)n 个元素分为 k 组
例题 1：掷骰子

同时掷两颗骰子，求点数之和为 7 的概率。

样本空间：|Ω| = 6 × 6 = 36
事件 A（点数之和为 7）：(1,6), (2,5), (3,4), (4,3), (5,2), (6,1)，共 6 个
P(A) = 6/36 = 1/6
例题 2：抽球问题

袋中有 5 个红球、3 个白球，不放回地抽取 3 个球，求恰好有 2 个红球的概率。

总取法：C(8,3) = 56
有利取法：C(5,2) × C(3,1) = 10 × 3 = 30
P(A) = 30/56 = 15/28 ≈ 0.536
注意： 抽球问题中，"有放回"与"不放回"的概率计算不同。不放回用组合，有放回用独立重复试验。

二、条件概率

1. 定义

设 A、B 是两个事件，且 P(B) > 0，则在事件 B 发生的条件下，事件 A 发生的条件概率为：

P(A|B) = P(AB) / P(B)

直观理解：将样本空间从 Ω 缩减到 B，在新的样本空间中 A 所占的比例。

2. 乘法公式

P(AB) = P(A|B) · P(B) = P(B|A) · P(A)

推广到多个事件：

P(A₁A₂...Aₙ) = P(A₁) · P(A₂|A₁) · P(A₃|A₁A₂) · ... · P(Aₙ|A₁A₂...Aₙ₋₁)

3. 独立性

若 P(AB) = P(A)·P(B)，则称事件 A 与 B 相互独立。

  • 独立 ⟹ P(A|B) = P(A)（B的发生不影响A的概率）
  • 独立 ⟹ P(B|A) = P(B)
  • 若 A 与 B 独立，则 A 与 B̄、Ā 与 B、Ā 与 B̄ 也独立
  • 独立 ≠ 互斥：互斥指 P(AB)=0，独立指 P(AB)=P(A)P(B)
例题 3：条件概率计算

某班级 60% 的学生喜欢数学，40% 的学生喜欢数学且喜欢物理，求喜欢数学的学生中也喜欢物理的概率。

设 A = 喜欢数学，B = 喜欢物理
P(A) = 0.6，P(AB) = 0.4
P(B|A) = P(AB)/P(A) = 0.4/0.6 = 2/3

三、贝叶斯定理

1. 全概率公式（预备知识）

设 B₁, B₂, ..., Bₙ 是样本空间 Ω 的一个划分（完备事件组），且 P(Bᵢ) > 0，则对任意事件 A：

P(A) = Σ P(A|Bᵢ) · P(Bᵢ)

2. 贝叶斯公式

在上述条件下：

P(Bⱼ|A) = P(A|Bⱼ) · P(Bⱼ) / Σ P(A|Bᵢ) · P(Bᵢ)

核心思想：由结果推原因。P(Bⱼ) 称为先验概率，P(Bⱼ|A) 称为后验概率。

例题 4：贝叶斯经典问题——检测疾病

某种疾病的发病率为 0.1%。一种检测方法的灵敏度（有病检测阳性）为 99%，特异度（无病检测阴性）为 95%。某人检测为阳性，问此人真正患病的概率是多少？

设 D = 患病，T = 检测阳性
P(D) = 0.001，P(T|D) = 0.99，P(T|D̄) = 1-0.95 = 0.05

全概率：P(T) = P(T|D)·P(D) + P(T|D̄)·P(D̄) = 0.99×0.001 + 0.05×0.999 = 0.05094

P(D|T) = 0.99 × 0.001 / 0.05094 ≈ 1.94%

即使检测结果为阳性，真正患病的概率也只有约 2%！这是因为基础发病率很低。

例题 5：工厂产品问题

三个工厂甲、乙、丙分别生产 25%、35%、40% 的产品，次品率分别为 5%、4%、2%。随机取一件是次品，求它来自甲厂的概率。

设 A = 次品，B₁ = 甲厂，B₂ = 乙厂，B₃ = 丙厂
P(B₁) = 0.25, P(B₂) = 0.35, P(B₃) = 0.40
P(A|B₁) = 0.05, P(A|B₂) = 0.04, P(A|B₃) = 0.02

P(A) = 0.25×0.05 + 0.35×0.04 + 0.40×0.02 = 0.0125 + 0.014 + 0.008 = 0.0345

P(B₁|A) = 0.0125 / 0.0345 ≈ 36.23%
贝叶斯思维要点：
  • 先验概率 P(B)：在获得新信息之前对事件的初始判断
  • 似然度 P(A|B)：在假设 B 成立的情况下，观测到 A 的概率
  • 后验概率 P(B|A)：获得观测数据 A 后，对 B 的更新判断
  • 贝叶斯公式本质上是 概率的更新规则

本节总结

  • 古典概型：P(A) = m/n（有限、等可能）
  • 条件概率：P(A|B) = P(AB)/P(B)
  • 乘法公式：P(AB) = P(A|B)·P(B)
  • 独立性：P(AB) = P(A)·P(B)
  • 全概率公式：P(A) = ΣP(A|Bᵢ)·P(Bᵢ)
  • 贝叶斯公式：P(Bⱼ|A) = P(A|Bⱼ)·P(Bⱼ) / ΣP(A|Bᵢ)·P(Bᵢ)
