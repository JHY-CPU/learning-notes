全概率公式与贝叶斯

概率论基础 · 由原因推结果与由结果推原因

一、全概率公式

1. 完备事件组（样本空间的划分）

设 B₁, B₂, ..., Bₙ 是一组事件，满足：

  • 互斥性：BᵢBⱼ = ∅（i ≠ j），任意两个事件不能同时发生
  • 完备性：B₁ ∪ B₂ ∪ ... ∪ Bₙ = Ω，它们的并集覆盖整个样本空间
  • 非零概率：P(Bᵢ) > 0（i = 1, 2, ..., n）

则 {B₁, B₂, ..., Bₙ} 构成样本空间 Ω 的一个划分。

2. 全概率公式

设 {B₁, B₂, ..., Bₙ} 是 Ω 的一个划分，A 为任一事件，则：

P(A) = P(A|B₁)·P(B₁) + P(A|B₂)·P(B₂) + ... + P(A|Bₙ)·P(Bₙ)

P(A) = Σᵢ₌₁ⁿ P(A|Bᵢ)·P(Bᵢ)

核心思想："化整为零"——将复杂事件 A 的概率分解为在各种"原因" Bᵢ 下 A 发生的概率之和。

┌── B₁ ──→ P(A|B₁) │ Ω ──┤── B₂ ──→ P(A|B₂) ──→ P(A) = Σ P(A|Bᵢ)·P(Bᵢ) │ └── B₃ ──→ P(A|B₃)
例题 1：产品合格率

某工厂有甲、乙、丙三条生产线，产量分别占总产量的 25%、35%、40%。各生产线的合格率分别为 95%、96%、98%。求该工厂产品的总合格率。

设 A = 合格品，B₁ = 甲线，B₂ = 乙线，B₃ = 丙线

P(B₁) = 0.25, P(B₂) = 0.35, P(B₃) = 0.40
P(A|B₁) = 0.95, P(A|B₂) = 0.96, P(A|B₃) = 0.98

P(A) = 0.25×0.95 + 0.35×0.96 + 0.40×0.98
     = 0.2375 + 0.336 + 0.392
     = 0.9655 = 96.55%
例题 2：多阶段抽球

甲袋中有 3 个红球、2 个白球；乙袋中有 1 个红球、4 个白球。先随机选一袋，再从选中的袋中取一球，求取到红球的概率。

设 A = 取到红球，B₁ = 选甲袋，B₂ = 选乙袋

P(B₁) = P(B₂) = 1/2
P(A|B₁) = 3/5, P(A|B₂) = 1/5

P(A) = (1/2)×(3/5) + (1/2)×(1/5) = 3/10 + 1/10 = 2/5
何时使用全概率公式？ 当题目中存在多个"原因"或"途径"导致同一"结果"时，使用全概率公式将结果的概率分解为各原因下的条件概率之和。关键词：多个来源、分阶段、多个类别。

二、贝叶斯公式（逆概率公式）

1. 公式推导

由条件概率定义和全概率公式联立：

P(Bⱼ|A) = P(ABⱼ) / P(A) = P(A|Bⱼ)·P(Bⱼ) / Σᵢ₌₁ⁿ P(A|Bᵢ)·P(Bᵢ)

本质：全概率公式是"由原因推结果"，贝叶斯公式是"由结果推原因"——已知结果 A 已经发生，反推是由原因 Bⱼ 导致的概率。

2. 先验概率与后验概率

概念符号含义
先验概率P(Bⱼ)在观测到结果之前，对原因 Bⱼ 发生概率的先验判断
似然度P(A|Bⱼ)在原因 Bⱼ 下产生结果 A 的可能性
后验概率P(Bⱼ|A)观测到结果 A 后，对原因 Bⱼ 的更新判断

贝叶斯公式的核心：用观测数据（似然度）来修正先验判断，得到后验概率。

例题 3：贝叶斯综合应用——邮件分类

某邮箱中 30% 的邮件是垃圾邮件。垃圾邮件中出现"免费"一词的概率为 80%，正常邮件中出现"免费"一词的概率为 10%。现收到一封含有"免费"的邮件，它是垃圾邮件的概率是多少？

设 S = 垃圾邮件，N = 正常邮件，W = 含"免费"

先验：P(S) = 0.3, P(N) = 0.7
似然：P(W|S) = 0.8, P(W|N) = 0.1

全概率：P(W) = 0.8×0.3 + 0.1×0.7 = 0.24 + 0.07 = 0.31

P(S|W) = 0.24 / 0.31 ≈ 77.4%

先验 30% → 后验 77.4%。观测到"免费"一词使垃圾邮件的概率大幅上升。

例题 4：多重贝叶斯更新

一种疾病的患病率为 1%。一种检测的灵敏度为 99%（有病→阳性），假阳性率为 5%（无病→阳性）。若第一次检测阳性后做第二次独立检测仍为阳性，求真正患病的概率。

设 D = 患病，T₁ = 第一次阳性，T₂ = 第二次阳性

第一次检测后：
P(T₁) = 0.99×0.01 + 0.05×0.99 = 0.0099 + 0.0495 = 0.0594
P(D|T₁) = 0.0099 / 0.0594 ≈ 16.67%

第二次检测——用后验做先验：
新先验：P(D) = 16.67%, P(D̄) = 83.33%
P(T₂) = 0.99×0.1667 + 0.05×0.8333 = 0.1650 + 0.0417 = 0.2067
P(D|T₁T₂) = 0.1650 / 0.2067 ≈ 79.8%

两次阳性后，患病概率从 1% → 16.67% → 79.8%，体现了贝叶斯逐步更新的过程。

例题 5：蒙提霍尔问题（三门问题）

有三扇门，一扇后面是汽车，两扇后面是山羊。你选了一扇门后，主持人打开另一扇有山羊的门。问：换门是否更有利？

设 A = 你最初选的门后是汽车，B = 主持人打开有山羊的门

不换门：P(赢) = P(A) = 1/3

换门：
P(赢|A) = 0（换到山羊）
P(赢|Ā) = 1（只剩汽车那扇）
P(赢) = 0×(1/3) + 1×(2/3) = 2/3

结论：换门获胜概率为 2/3，不换门为 1/3，应该换！
贝叶斯方法的现实应用：
  • 医学诊断：根据检测结果评估疾病概率
  • 垃圾邮件过滤：根据词汇出现频率分类邮件
  • 机器学习：朴素贝叶斯分类器
  • 司法推理：根据证据评估嫌疑概率
  • 搜索引擎：根据查询词判断文档相关性

本节总结

  • 全概率公式：P(A) = Σ P(A|Bᵢ)·P(Bᵢ)，将复杂事件分解为简单事件之和
  • 贝叶斯公式：P(Bⱼ|A) = P(A|Bⱼ)·P(Bⱼ) / Σ P(A|Bᵢ)·P(Bᵢ)，由结果推原因
  • 先验→后验：用观测数据修正初始判断
  • 逐步更新：后验可作为下一次的先验，不断迭代
  • 解题关键：正确识别完备事件组，准确设定条件概率
