# 古典概型与条件概率

*概率论基础 · 核心概念与公式*


### 1. 定义与条件


古典概型满足两个基本条件：


- **有限性**
   ：样本空间中样本点的总数是有限的，即
    $Ω = {ω₁, ω₂, ..., ωₙ}$
- **等可能性**
   ：每个基本事件发生的概率相等，即
    $P(ω₁) = P(ω₂) = ... = P(ωₙ) = 1/n$


### 2. 古典概率公式


$$
P(A) = |A| / |Ω| = A中包含的基本事件数 / 样本空间中基本事件总数 = m / n
$$


其中 |A| 表示事件 A 包含的样本点数，|Ω| 表示样本空间的总样本点数。


### 3. 计数方法


| 方法 | 公式 | 说明 |
| --- | --- | --- |
| 排列（有序选取） | P(n,m) = n! / (n-m)! | 从 n 个中取 m 个并排序 |
| 组合（无序选取） | C(n,m) = n! / [m!(n-m)!] | 从 n 个中取 m 个不排序 |
| 可重复排列 | n^m^ | m 个球放入 n 个盒子 |
| 多重排列 | n! / (n₁!·n₂!·...·nₖ!) | n 个元素分为 k 组 |


### 1. 定义


设 A、B 是两个事件，且 P(B) > 0，则在事件 B 发生的条件下，事件 A 发生的条件概率为：


$$
P(A|B) = P(AB) / P(B)
$$


直观理解：将样本空间从 Ω 缩减到 B，在新的样本空间中 A 所占的比例。


### 2. 乘法公式


$$
P(AB) = P(A|B) · P(B) = P(B|A) · P(A)
$$


推广到多个事件：


$$
P(A₁A₂...Aₙ) = P(A₁) · P(A₂|A₁) · P(A₃|A₁A₂) · ... · P(Aₙ|A₁A₂...Aₙ₋₁)
$$


### 3. 独立性


若 P(AB) = P(A)·P(B)，则称事件 A 与 B **相互独立**。


- 独立 ⟹ P(A|B) = P(A)（B的发生不影响A的概率）
- 独立 ⟹ P(B|A) = P(B)
- 若 A 与 B 独立，则 A 与
    $B̄$
   、
    $Ā$
   与 B、
    $Ā$
   与
    $B̄$
   也独立
- **独立 ≠ 互斥**
   ：互斥指 P(AB)=0，独立指 P(AB)=P(A)P(B)


### 1. 全概率公式（预备知识）


设 B₁, B₂, ..., Bₙ 是样本空间 Ω 的一个划分（完备事件组），且 P(Bᵢ) > 0，则对任意事件 A：


$$
P(A) = Σ P(A|Bᵢ) · P(Bᵢ)
$$


### 2. 贝叶斯公式


在上述条件下：


$$
P(Bⱼ|A) = P(A|Bⱼ) · P(Bⱼ) / Σ P(A|Bᵢ) · P(Bᵢ)
$$


核心思想：**由结果推原因**。P(Bⱼ) 称为先验概率，P(Bⱼ|A) 称为后验概率。


## Python实现

### 古典概型模拟与条件概率

```python
import numpy as np
from itertools import permutations, combinations
from math import factorial, comb

# 古典概率: 从52张牌中抽5张, 恰好有2张红心的概率
total = comb(52, 5)
favorable = comb(13, 2) * comb(39, 3)
prob = favorable / total
print(f"恰好2张红心: P = {favorable}/{total} = {prob:.6f}")

# 模拟验证
np.random.seed(42)
n_trials = 100000
count = 0
for _ in range(n_trials):
    cards = np.zeros(52)
    cards[:13] = 1  # 前13张为红心
    drawn = np.random.choice(52, 5, replace=False)
    if sum(cards[drawn]) == 2:
        count += 1
print(f"模拟结果: {count/n_trials:.6f}")

# 条件概率模拟: P(A|B) = P(AB)/P(B)
# 掷两个骰子, 已知和为7, 求其中一个为3的概率
n_sim = 100000
rolls = np.random.randint(1, 7, (n_sim, 2))
sum7 = rolls[rolls.sum(axis=1) == 7]
has3 = np.any(sum7 == 3, axis=1)
print(f"P(有3 | 和=7) ≈ {has3.mean():.4f}, 精确值 = {2/6:.4f}")
```

### 排列组合计算

```python
from math import factorial, perm, comb

# 排列 P(n, m)
print(f"P(5,3) = {perm(5, 3)}")  # 60
print(f"P(10,4) = {perm(10, 4)}")  # 5040

# 组合 C(n, m)
print(f"C(5,3) = {comb(5, 3)}")  # 10
print(f"C(10,4) = {comb(10, 4)}")  # 210

# 多重排列: 10个人分3组 (4,3,3)
multi = factorial(10) / (factorial(4) * factorial(3) * factorial(3))
print(f"10人分(4,3,3)三组: {int(multi)}")
```
