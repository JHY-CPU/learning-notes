# 事件独立性与条件期望

*概率基础 · 独立性判定与条件期望*


### 1. 两事件独立


设 A、B 为两个事件，若满足以下等价条件之一，则称 A 与 B **相互独立**：


$$
P(AB) = P(A) · P(B)
$$


等价表述：


- P(A|B) = P(A)，即 B 的发生不改变 A 的概率
- P(B|A) = P(B)，即 A 的发生不改变 B 的概率


### 2. 独立性的重要推论


若 A 与 B 相互独立，则以下各对事件也相互独立：


| 事件对 | 证明思路 |
| --- | --- |
| A 与 B̄（B的对立事件） | P(AB̄) = P(A) - P(AB) = P(A) - P(A)P(B) = P(A)[1-P(B)] = P(A)P(B̄) |
| Ā 与 B | 由对称性 |
| Ā 与 B̄ | P(ĀB̄) = 1 - P(A∪B) = 1 - P(A) - P(B) + P(AB) = [1-P(A)][1-P(B)] = P(Ā)P(B̄) |


### 3. 独立 vs 互斥（易混淆辨析）


| 概念 | 定义 | 条件 | 能否同时成立 |
| --- | --- | --- | --- |
| 互斥 | A∩B = ∅ | P(AB) = 0 | 一般不能同时成立（除非 P(A)=0 或 P(B)=0） |
| 独立 | 发生互不影响 | P(AB) = P(A)P(B) | — |


> **Note:** 关键区分：
> 互斥事件"不能同时发生"，因此一个发生就决定了另一个不发生（互相影响），故
> **互斥事件一定不独立**
> （概率均非零时）。独立事件"可以同时发生但互不影响"。


### 4. 多事件独立


### （1）两两独立


对任意 i ≠ j，有 P(AᵢAⱼ) = P(Aᵢ)P(Aⱼ)


### （2）相互独立（全局独立）


对任意子集 {i₁, i₂, ..., iₖ}，有：


$$
P(Aᵢ₁Aᵢ₂...Aᵢₖ) = P(Aᵢ₁)·P(Aᵢ₂)·...·P(Aᵢₖ)
$$


两两独立 ≠ 相互独立！两两独立是必要条件，但不是充分条件。


### 1. 条件期望的定义


给定事件 B（P(B)>0），随机变量 X 关于 B 的条件期望定义为：


$$
E(X|B) = Σ xₖ · P(X = xₖ | B)  （离散型）
            E(X|B) = ∫₋∞⁺∞ x · f(x|B) dx  （连续型）
$$


其中 P(X=xₖ|B) = P(X=xₖ, B)/P(B)，f(x|B) 为条件密度函数。


### 2. 条件期望的基本性质


- E(aX + b | B) = aE(X|B) + b
- E(X + Y | B) = E(X|B) + E(Y|B)
- 若 X 与 B 独立，则 E(X|B) = E(X)
- E(E(X|B)) = E(X)（需推广形式）


### 1. 离散版本


设 B₁, B₂, ..., Bₙ 是样本空间的一个划分（互斥且穷尽），则：


$$
E(X) = Σ E(X|Bᵢ) · P(Bᵢ)
$$


这是全概率公式在期望层面的推广。


### 2. 随机变量版本（重期望公式）


设 Y 为随机变量，则以 Y 为条件：


$$
E(X) = E[E(X|Y)]
$$


含义：X 的无条件期望 = X 在每个 Y 值下条件期望的加权平均（权重为 Y 的分布）。


## Python实现

### 独立性检验与条件期望

```python
import numpy as np

# 模拟独立性检验: 两个独立的骰子
np.random.seed(42)
n = 100000
X = np.random.randint(1, 7, n)
Y = np.random.randint(1, 7, n)

# 检查 P(XY) = P(X)*P(Y)
for xi in range(1, 4):
    for yj in range(1, 4):
        P_X = np.mean(X == xi)
        P_Y = np.mean(Y == yj)
        P_XY = np.mean((X == xi) & (Y == yj))
        print(f"P(X={xi})={P_X:.4f}, P(Y={yj})={P_Y:.4f}, "
              f"P(XY)={P_XY:.4f}, P(X)P(Y)={P_X*P_Y:.4f}")

# 独立 vs 互斥的对比
print("\n=== 独立 vs 互斥 ===")
# 互斥: A∩B = ∅, P(AB) = 0
# 独立: P(AB) = P(A)P(B)
P_A, P_B = 0.3, 0.4
print(f"P(A)={P_A}, P(B)={P_B}")
print(f"若互斥: P(AB) = 0")
print(f"若独立: P(AB) = P(A)P(B) = {P_A*P_B}")
print(f"互斥且概率均非零 → 不能同时独立!")
```

### 条件期望与全期望公式

```python
import numpy as np

# 全期望公式验证: E(X) = E[E(X|Y)]
# X = 掷骰子的点数, Y = X 是否大于3
np.random.seed(42)
n = 1000000
X = np.random.randint(1, 7, n)
Y = (X > 3).astype(int)  # 0: ≤3, 1: >3

E_X = X.mean()
print(f"E(X) = {E_X:.4f} (精确值 3.5)")

# E(X|Y=0): X≤3 时的期望
E_X_given_Y0 = X[Y == 0].mean()
# E(X|Y=1): X>3 时的期望
E_X_given_Y1 = X[Y == 1].mean()
P_Y0 = (Y == 0).mean()
P_Y1 = (Y == 1).mean()

E_X_reconstructed = E_X_given_Y0 * P_Y0 + E_X_given_Y1 * P_Y1
print(f"E(X|Y=0)={E_X_given_Y0:.4f}, E(X|Y=1)={E_X_given_Y1:.4f}")
print(f"E[E(X|Y)] = {E_X_reconstructed:.4f}")
print(f"全期望公式验证: E(X) ≈ E[E(X|Y)]")
```
