# 全概率公式与贝叶斯

*概率论基础 · 由原因推结果与由结果推原因*


### 1. 完备事件组（样本空间的划分）


设 B₁, B₂, ..., Bₙ 是一组事件，满足：


- **互斥性**
   ：BᵢBⱼ = ∅（i ≠ j），任意两个事件不能同时发生
- **完备性**
   ：B₁ ∪ B₂ ∪ ... ∪ Bₙ = Ω，它们的并集覆盖整个样本空间
- **非零概率**
   ：P(Bᵢ) > 0（i = 1, 2, ..., n）


则 {B₁, B₂, ..., Bₙ} 构成样本空间 Ω 的一个划分。


### 2. 全概率公式


设 {B₁, B₂, ..., Bₙ} 是 Ω 的一个划分，A 为任一事件，则：


$$
P(A) = P(A|B₁)·P(B₁) + P(A|B₂)·P(B₂) + ... + P(A|Bₙ)·P(Bₙ)

            P(A) = Σᵢ₌₁ⁿ P(A|Bᵢ)·P(Bᵢ)
$$


**核心思想**："化整为零"——将复杂事件 A 的概率分解为在各种"原因" Bᵢ 下 A 发生的概率之和。


### 1. 公式推导


由条件概率定义和全概率公式联立：


$$
P(Bⱼ|A) = P(ABⱼ) / P(A) = P(A|Bⱼ)·P(Bⱼ) / Σᵢ₌₁ⁿ P(A|Bᵢ)·P(Bᵢ)
$$


**本质**：全概率公式是"由原因推结果"，贝叶斯公式是"由结果推原因"——已知结果 A 已经发生，反推是由原因 Bⱼ 导致的概率。


### 2. 先验概率与后验概率


| 概念 | 符号 | 含义 |
| --- | --- | --- |
| 先验概率 | P(Bⱼ) | 在观测到结果之前，对原因 Bⱼ 发生概率的先验判断 |
| 似然度 | P(A\|Bⱼ) | 在原因 Bⱼ 下产生结果 A 的可能性 |
| 后验概率 | P(Bⱼ\|A) | 观测到结果 A 后，对原因 Bⱼ 的更新判断 |


贝叶斯公式的核心：用观测数据（似然度）来修正先验判断，得到后验概率。


### 3. 经典例题：贝叶斯应用

**例题**：某工厂有3条生产线 A、B、C，分别生产 60%、30%、10% 的产品。已知次品率分别为 2%、3%、5%。随机取一件产品发现是次品，问它来自哪条生产线的概率最大？

**解答**：设事件 D = "取到次品"，B₁/B₂/B₃ = "来自 A/B/C 线"

先验概率：P(B₁)=0.6, P(B₂)=0.3, P(B₃)=0.1
似然度：P(D|B₁)=0.02, P(D|B₂)=0.03, P(D|B₃)=0.05

全概率公式：
$$P(D) = 0.02×0.6 + 0.03×0.3 + 0.05×0.1 = 0.012 + 0.009 + 0.005 = 0.026$$

贝叶斯后验概率：
$$P(B₁|D) = 0.012/0.026 = 12/26 ≈ 0.462$$
$$P(B₂|D) = 0.009/0.026 = 9/26 ≈ 0.346$$
$$P(B₃|D) = 0.005/0.026 = 5/26 ≈ 0.192$$

结论：次品最可能来自 A 线（概率 46.2%）。

### 4. 贝叶斯在机器学习中的应用

贝叶斯定理是许多机器学习算法的核心：

```
后验概率 = (似然度 × 先验概率) / 证据

P(θ|Data) = P(Data|θ) × P(θ) / P(Data)
```

| 算法 | 贝叶斯角色 | 应用 |
|------|-----------|------|
| 朴素贝叶斯分类器 | 用贝叶斯公式计算类别后验概率 | 垃圾邮件过滤 |
| 贝叶斯优化 | 用先验和似然更新超参数分布 | 超参数调优 |
| 贝叶斯神经网络 | 权重的后验分布 | 不确定性估计 |
| 主题模型(LDA) | 文档-主题-词的贝叶斯生成模型 | 文本分析 |

### 5. 常见陷阱与注意事项

1. **贝叶斯公式的分母不能忘**：P(A) = ΣP(A|Bᵢ)P(Bᵢ)，必须计算完整
2. **先验概率的影响**：先验概率越强，需要越多数据才能改变后验判断
3. **不要混淆 P(A|B) 和 P(B|A)**：这是最常见的贝叶斯错误
4. **条件独立假设**：朴素贝叶斯假设各特征条件独立，实际中往往不完全满足但仍有效

## Python实现

### 贝叶斯公式计算

```python
import numpy as np

# 经典例题: 三生产线次品率问题
# P(B1)=0.6, P(B2)=0.3, P(B3)=0.1
# P(D|B1)=0.02, P(D|B2)=0.03, P(D|B3)=0.05
prior = np.array([0.6, 0.3, 0.1])
likelihood = np.array([0.02, 0.03, 0.05])

# 全概率公式: P(D)
P_D = np.sum(likelihood * prior)
print(f"P(D) = {P_D:.4f}")

# 贝叶斯后验概率
posterior = (likelihood * prior) / P_D
print(f"后验概率: P(B1|D)={posterior[0]:.4f}, P(B2|D)={posterior[1]:.4f}, P(B3|D)={posterior[2]:.4f}")
print(f"次品最可能来自: B{np.argmax(posterior)+1}")

# 蒙特卡洛验证
np.random.seed(42)
n = 1000000
# 随机选择生产线
lines = np.random.choice([0, 1, 2], size=n, p=prior)
# 每个产品是否为次品
defective = np.random.rand(n) < likelihood[lines]
# 统计次品来自各线的比例
defective_lines = lines[defective]
for i, name in enumerate(['A', 'B', 'C']):
    count = np.sum(defective_lines == i)
    print(f"  来自{name}线的次品比例: {count/len(defective_lines):.4f}")
```

### 朴素贝叶斯分类器示例

```python
import numpy as np

# 简化版: 垃圾邮件检测
# 特征: 是否包含"免费", "中奖", "紧急"
# P(垃圾邮件) = 0.3
# P("免费"|垃圾) = 0.8, P("免费"|正常) = 0.1
# P("中奖"|垃圾) = 0.6, P("中奖"|正常) = 0.05
# P("紧急"|垃圾) = 0.7, P("紧急|正常) = 0.1

P_spam = 0.3
P_normal = 0.7

# 邮件同时包含"免费"和"中奖"
P_free_spam, P_free_normal = 0.8, 0.1
P_prize_spam, P_prize_normal = 0.6, 0.05

# 贝叶斯: P(垃圾|免费,中奖) ∝ P(免费|垃圾)P(中奖|垃圾)P(垃圾)
P_spam_given = P_free_spam * P_prize_spam * P_spam
P_normal_given = P_free_normal * P_prize_normal * P_normal
total = P_spam_given + P_normal_given

print(f"P(垃圾邮件|含'免费'和'中奖') = {P_spam_given/total:.4f}")
print(f"分类结果: {'垃圾邮件' if P_spam_given > P_normal_given else '正常邮件'}")
```
