# 条件分布与随机变量独立性

*多维随机变量 · 条件分布、独立性与次序统计量*


### 1. 离散型条件分布律


设 (X,Y) 为二维离散型随机变量，在 Y=yⱼ 的条件下，X 的条件分布律为：


$$
P(X=xᵢ | Y=yⱼ) = P(X=xᵢ, Y=yⱼ) / P(Y=yⱼ) = pᵢⱼ / p·ⱼ
$$


其中 p·ⱼ = P(Y=yⱼ) = Σᵢ pᵢⱼ 为 Y 的边缘分布律。


### 2. 连续型条件密度


设 (X,Y) 的联合密度为 f(x,y)，边缘密度为 f_Y(y)，在 Y=y 的条件下，X 的条件密度为：


$$
fX|Y(x|y) = f(x,y) / fY(y),   fY(y) > 0
$$


条件期望：E(X|Y=y) = ∫ x · f~X|Y~(x|y) dx


条件方差：Var(X|Y=y) = E(X²|Y=y) - [E(X|Y=y)]²


### 1. 独立的定义


随机变量 X 与 Y 相互独立，当且仅当对任意实数 x, y：


$$
F(x,y) = FX(x) · FY(y)  （联合CDF = 边缘CDF的乘积）
$$


等价条件（更常用）：


- **离散型**
   ：P(X=xᵢ, Y=yⱼ) = P(X=xᵢ) · P(Y=yⱼ) 对所有 i,j 成立
- **连续型**
   ：f(x,y) = f
   ~X~
   (x) · f
   ~Y~
   (y) 对所有 x,y 成立

==================== 条件分布性质 ====================

## 三、条件分布的重要性质


### 重期望公式与重方差公式


$$
E(X) = E[E(X|Y)]  （全期望公式）
$$


$$
Var(X) = E[Var(X|Y)] + Var[E(X|Y)]  （全方差公式 / Eve定律）
$$


全方差公式的直觉：总方差 = 条件方差的期望 + 条件期望的方差（"期望中的变异 + 期望周围的变异"）


> **Example:** 例题 3：全方差公式
>
> 某保险公司，投保人事故次数 N|λ~Poisson(λ)，其中 λ~Exp(1)。求 Var(N)。
>
>
> > **Step:** 已知 E(N|λ)=λ, Var(N|λ)=λ
> >
> >
> > E(λ)=1, Var(λ)=1（Exp(1)的参数）
> >
> >
> >
> >
> > E(N) = E[E(N|λ)] = E(λ) = 1
> >
> >
> > Var(N) = E[Var(N|λ)] + Var[E(N|λ)]
> >
> >
> > = E(λ) + Var(λ) = 1 + 1 =
> > **2**
> >
> >
> >
> >
> > （对比：若 λ 固定=1，Var(N)=1；随机 λ 导致方差增大）

### 2. 次序统计量的分布


设 X₁,...,Xₙ 独立同分布，CDF 为 F(x)，PDF 为 f(x)：


$$
FX(n)(x) = [F(x)]n    （最大值的CDF）
$$


$$
FX(1)(x) = 1 - [1-F(x)]n    （最小值的CDF）
$$


$$
fX(k)(x) = n! / [(k-1)!(n-k)!] · [F(x)]k-1 · [1-F(x)]n-k · f(x)
$$
