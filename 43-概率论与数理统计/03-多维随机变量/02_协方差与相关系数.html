协方差与相关系数

多维随机变量 · 线性相关性度量

一、协方差

1. 定义

Cov(X, Y) = E[(X - E(X))(Y - E(Y))] = E(XY) - E(X)·E(Y)

协方差衡量两个随机变量的线性相关程度和方向。

2. 基本性质

  • Cov(X, X) = D(X)
  • Cov(X, Y) = Cov(Y, X)（对称性）
  • Cov(aX, bY) = ab · Cov(X, Y)
  • Cov(X₁+X₂, Y) = Cov(X₁, Y) + Cov(X₂, Y)
  • D(X±Y) = D(X) + D(Y) ± 2Cov(X, Y)
  • 若 X, Y 独立，则 Cov(X, Y) = 0（反之不一定成立）
例题 1：计算协方差

设 (X, Y) 的联合分布：P(X=0,Y=0)=0.2, P(X=0,Y=1)=0.3, P(X=1,Y=0)=0.1, P(X=1,Y=1)=0.4。求 Cov(X,Y)。

E(X) = 0×0.5 + 1×0.5 = 0.5
E(Y) = 0×0.3 + 1×0.7 = 0.7
E(XY) = 0×0.2 + 0×0.3 + 0×0.1 + 1×1×0.4 = 0.4

Cov(X,Y) = E(XY) - E(X)E(Y) = 0.4 - 0.5×0.7 = 0.4 - 0.35 = 0.05

二、相关系数

1. 定义

ρ_XY = Cov(X, Y) / (σ_X · σ_Y) = Cov(X, Y) / √(D(X)·D(Y))

相关系数是标准化后的协方差，取值范围 [-1, 1]。

2. 含义解读

ρ 的值含义
ρ = 1完全正线性相关（Y = aX + b, a > 0）
ρ = -1完全负线性相关（Y = aX + b, a < 0）
ρ = 0不相关（无线性关系，但可能有非线性关系）
0 < |ρ| < 1部分线性相关

3. 重要性质

  • |ρ_XY| ≤ 1（柯西-施瓦茨不等式）
  • |ρ| = 1 ⟺ 存在常数 a, b 使 P(Y = aX + b) = 1
  • 独立 ⟹ ρ = 0（不相关）
  • ρ = 0 ⟹ 独立（一般不成立，但对二维正态分布成立）
  • ρ 只反映线性相关性，不反映非线性依赖关系
例题 2：相关系数计算

设 D(X) = 4, D(Y) = 9, Cov(X, Y) = 3，求 ρ_XY 和 D(X+Y)。

ρ_XY = Cov(X,Y) / √(D(X)·D(Y)) = 3 / √(4×9) = 3/6 = 0.5

D(X+Y) = D(X) + D(Y) + 2Cov(X,Y) = 4 + 9 + 2×3 = 19

三、矩与协方差矩阵

1. 矩的概念

类型定义说明
k阶原点矩E(Xᵏ)k=1 即期望
k阶中心矩E[(X-E(X))ᵏ]k=2 即方差
混合矩E(XᵃYᵇ)反映两个变量的联合矩

2. 协方差矩阵

对于 n 维随机向量 X = (X₁, X₂, ..., Xₙ)ᵀ：

Σ = [Cov(Xᵢ, Xⱼ)]ₙₓₙ

对角线元素为各分量的方差，非对角线元素为协方差。协方差矩阵是对称非负定矩阵。

不相关 vs 独立的区别：
  • 不相关（ρ=0）：没有线性关系，但可能有非线性关系
  • 独立：没有任何统计依赖关系
  • 独立 ⟹ 不相关，不相关 ⟹ 独立（除非二维正态等特殊情况）
  • 例：X ~ U(-1,1), Y=X²，则 Cov(X,Y)=0（不相关），但显然不独立

本节总结

  • 协方差：Cov(X,Y) = E(XY) - E(X)E(Y)，衡量线性相关方向和强度
  • 相关系数：ρ = Cov(X,Y) / (σ_X·σ_Y)，标准化的协方差，|ρ| ≤ 1
  • 关键公式：D(X±Y) = D(X) + D(Y) ± 2Cov(X,Y)
  • 独立与不相关：独立 ⟹ 不相关，反之一般不成立
  • 协方差矩阵：对称非负定，对角线为方差
