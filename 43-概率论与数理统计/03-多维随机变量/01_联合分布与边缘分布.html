联合分布与边缘分布

多维随机变量 · 联合、边缘与条件分布

一、二维离散型随机变量

1. 联合分布律

用联合概率表描述 (X, Y) 取各组值的概率：

P(X = xᵢ, Y = yⱼ) = pᵢⱼ,   pᵢⱼ ≥ 0,   Σᵢ Σⱼ pᵢⱼ = 1

2. 边缘分布律

对联合分布表按行/列求和：

P(X = xᵢ) = pᵢ· = Σⱼ pᵢⱼ      （行和）

P(Y = yⱼ) = p·ⱼ = Σᵢ pᵢⱼ      （列和）

P(X = xᵢ | Y = yⱼ) = pᵢⱼ / p·ⱼ      （固定 Y 的列，各元素除以列和）

二、二维连续型随机变量

1. 联合概率密度

f(x, y) 满足：

  • f(x, y) ≥ 0
  • ∫₋∞⁺∞ ∫₋∞⁺∞ f(x, y) dx dy = 1
  • P((X,Y) ∈ D) = ∬_D f(x, y) dx dy

2. 边缘概率密度

f_X(x) = ∫₋∞⁺∞ f(x, y) dy      （对 y 积分）

f_Y(y) = ∫₋∞⁺∞ f(x, y) dx      （对 x 积分）

3. 条件概率密度

f_{X|Y}(x|y) = f(x, y) / f_Y(y)      （f_Y(y) > 0）

三、随机变量的独立性

定义

X 与 Y 相互独立，当且仅当联合分布等于边缘分布的乘积：

离散型：P(X=xᵢ, Y=yⱼ) = P(X=xᵢ) · P(Y=yⱼ),   对所有 i, j

连续型：f(x, y) = f_X(x) · f_Y(y),   对几乎所有 (x, y)

直观理解：知道 Y 的取值不影响 X 的分布。

例题 3：独立性判断

判断例题 1 中的 X, Y 是否独立。

检验 P(X=0, Y=0) = 1/6
P(X=0) · P(Y=0) = (11/24) × (7/24) = 77/576 ≈ 0.1337
P(X=0, Y=0) = 1/6 ≈ 0.1667

因为 1/6 ≠ 77/576，所以 X 与 Y 不独立。
独立性的重要推论：
  • 若 X, Y 独立，则 E(XY) = E(X)·E(Y)
  • 若 X, Y 独立，则 D(X±Y) = D(X) + D(Y)
  • 独立 ⟹ 不相关，但不相关 ⟹ 独立（仅对正态分布等价）

四、常见二维分布

二维正态分布 N(μ₁, μ₂, σ₁², σ₂², ρ)

f(x,y) = (1/(2πσ₁σ₂√(1-ρ²))) · exp{ -1/(2(1-ρ²)) [ (x-μ₁)²/σ₁² - 2ρ(x-μ₁)(y-μ₂)/(σ₁σ₂) + (y-μ₂)²/σ₂² ] }
  • X ~ N(μ₁, σ₁²)，Y ~ N(μ₂, σ₂²)（边缘分布仍是正态）
  • ρ 是相关系数，|ρ| ≤ 1
  • ρ = 0 ⟺ X, Y 独立（二维正态中不相关与独立等价）

本节总结

  • 联合分布：完整描述 (X,Y) 的概率行为
  • 边缘分布：对联合分布求和（离散）或积分（连续）得到单个变量的分布
  • 条件分布：固定一个变量时另一个变量的分布
  • 独立性：联合 = 边缘之积，可用此条件检验
  • 二维正态：边缘仍是正态，ρ=0 与独立等价
