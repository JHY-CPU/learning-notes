# 联合分布与边缘分布

*多维随机变量 · 联合、边缘与条件分布*


### 1. 联合分布律


用联合概率表描述 (X, Y) 取各组值的概率：


$$
P(X = xᵢ, Y = yⱼ) = pᵢⱼ,   pᵢⱼ ≥ 0,   Σᵢ Σⱼ pᵢⱼ = 1
$$


### 2. 边缘分布律


对联合分布表按行/列求和：


$$
P(X = xᵢ) = pᵢ· = Σⱼ pᵢⱼ      （行和）

            P(Y = yⱼ) = p·ⱼ = Σᵢ pᵢⱼ      （列和）
$$


### 3. 条件分布律


$$
P(X = xᵢ | Y = yⱼ) = pᵢⱼ / p·ⱼ      （固定 Y 的列，各元素除以列和）
$$


### 1. 联合概率密度


f(x, y) 满足：


- f(x, y) ≥ 0
- ∫₋∞⁺∞ ∫₋∞⁺∞ f(x, y) dx dy = 1
- P((X,Y) ∈ D) = ∬_D f(x, y) dx dy


### 2. 边缘概率密度


$$
f_X(x) = ∫₋∞⁺∞ f(x, y) dy      （对 y 积分）

            f_Y(y) = ∫₋∞⁺∞ f(x, y) dx      （对 x 积分）
$$


### 3. 条件概率密度


$$
f_{X|Y}(x|y) = f(x, y) / f_Y(y)      （f_Y(y) > 0）
$$


### 定义


X 与 Y 相互独立，当且仅当联合分布等于边缘分布的乘积：


$$
离散型：P(X=xᵢ, Y=yⱼ) = P(X=xᵢ) · P(Y=yⱼ),   对所有 i, j

            连续型：f(x, y) = f_X(x) · f_Y(y),   对几乎所有 (x, y)
$$


直观理解：知道 Y 的取值不影响 X 的分布。


### 二维正态分布 N(μ₁, μ₂, σ₁², σ₂², ρ)


$$
f(x,y) = (1/(2πσ₁σ₂√(1-ρ²))) · exp{ -1/(2(1-ρ²)) [ (x-μ₁)²/σ₁² - 2ρ(x-μ₁)(y-μ₂)/(σ₁σ₂) + (y-μ₂)²/σ₂² ] }
$$


- X ~ N(μ₁, σ₁²)，Y ~ N(μ₂, σ₂²)（边缘分布仍是正态）
- ρ 是相关系数，|ρ| ≤ 1
- **ρ = 0 ⟺ X, Y 独立**
   （二维正态中不相关与独立等价）

### 离散型例题

**例题**：袋中有2个红球、3个白球。不放回地取2次，每次取1个。设 X = 第一次取到红球数(0或1)，Y = 两次共取到红球数。求联合分布和边缘分布。

$$
P(X=0, Y=0) = C(3,2)/C(5,2) = 3/10
$$
$$
P(X=0, Y=1) = 3×2/C(5,2) = 6/10
$$
$$
P(X=1, Y=1) = 2×3/C(5,2) = 6/10 → P(X=1,Y=1) = 0
$$

更正：P(X=0,Y=0) = 3/10, P(X=0,Y=1) = 6/10, P(X=1,Y=1) = 2×3/20 = 3/10, P(X=1,Y=2) = 2×1/20 = 1/10

边缘分布：P(X=0) = 3/10+6/10 = 9/10（应验算修正为正确值）

### 连续型例题

**例题**：设 f(x,y) = k(2x+y)，0<x<1, 0<y<2。求 k、边缘密度、判断独立性。

第一步：求 k
$$\int_0^1 \int_0^2 k(2x+y) \, dy \, dx = k \int_0^1 [2xy + y^2/2]_0^2 dx = k \int_0^1 (4x+2) dx = k[x^2+2x]_0^1 = 3k = 1$$

得 k = 1/3

边缘密度：
$$f_X(x) = \int_0^2 \frac{1}{3}(2x+y) dy = \frac{1}{3}[2xy+y^2/2]_0^2 = \frac{1}{3}(4x+2) = \frac{4x+2}{3}$$

$$f_Y(y) = \int_0^1 \frac{1}{3}(2x+y) dx = \frac{1}{3}[x^2+xy]_0^1 = \frac{1}{3}(1+y) = \frac{1+y}{3}$$

验证独立性：f(x,y) = (2x+y)/3，f_X(x)·f_Y(y) = (4x+2)(1+y)/9。由于 (2x+y)/3 ≠ (4x+2)(1+y)/9，**X 与 Y 不独立**。

### 6. 常见陷阱

1. **离散型和连续型不要混淆**：联合分布律用求和，联合密度用积分
2. **边缘分布由联合分布唯一确定**，但仅凭边缘分布无法还原联合分布
3. **独立性必须对所有 (x,y) 成立**：只要有一个点不满足就不独立
4. **二维正态的特例**：ρ=0 等价于独立，这在一般情况下不成立


## Python实现

### 联合分布与边缘分布

```python
import numpy as np
from scipy import stats

# 离散型: 联合分布表
joint = np.array([[0.3, 0.2],
                   [0.1, 0.4]])
print("联合分布:")
print(joint)
print(f"P(X=0) = {joint[0].sum()} (行和)")
print(f"P(Y=0) = {joint[:, 0].sum()} (列和)")

# 边缘分布
marginal_X = joint.sum(axis=1)  # 行和
marginal_Y = joint.sum(axis=0)  # 列和
print(f"边缘分布 P(X): {marginal_X}")
print(f"边缘分布 P(Y): {marginal_Y}")

# 独立性检验: P(X=x, Y=y) = P(X=x)*P(Y=y)?
is_independent = np.allclose(joint, np.outer(marginal_X, marginal_Y))
print(f"X,Y 独立? {is_independent}")

# 条件分布 P(X|Y=0)
P_X_given_Y0 = joint[:, 0] / marginal_Y[0]
print(f"P(X|Y=0) = {P_X_given_Y0}")
```

### 二维正态分布

```python
import numpy as np
from scipy import stats

# 二维正态分布
mu = [0, 0]
cov = [[1, 0.5], [0.5, 2]]  # 相关系数 ρ = 0.5/√(1*2) ≈ 0.354
rho = 0.5 / np.sqrt(2)
print(f"相关系数 ρ = {rho:.4f}")

# 生成样本
np.random.seed(42)
samples = np.random.multivariate_normal(mu, cov, 100000)
X, Y = samples[:, 0], samples[:, 1]

# 边缘分布 (二维正态的边缘仍是正态)
print(f"E(X) = {X.mean():.4f}, Var(X) = {X.var():.4f} (期望: 0, 1)")
print(f"E(Y) = {Y.mean():.4f}, Var(Y) = {Y.var():.4f} (期望: 0, 2)")

# 协方差和相关系数
print(f"Cov(X,Y) = {np.cov(X, Y)[0,1]:.4f} (期望: {cov[0][1]})")
print(f"ρ = {np.corrcoef(X, Y)[0,1]:.4f} (期望: {rho:.4f})")

# ρ=0 时独立 (二维正态的特有性质)
samples_indep = np.random.multivariate_normal([0,0], [[1,0],[0,1]], 10000)
X_indep, Y_indep = samples_indep[:, 0], samples_indep[:, 1]
# 检验独立性: E(X²Y) ≈ E(X²)*E(Y)
E_X2Y = np.mean(X_indep**2 * Y_indep)
E_X2 = np.mean(X_indep**2)
E_Y = np.mean(Y_indep)
print(f"ρ=0: E(X²Y)={E_X2Y:.4f}, E(X²)E(Y)={E_X2*E_Y:.4f} (应相等)")
```
