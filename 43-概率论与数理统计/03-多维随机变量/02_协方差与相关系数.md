# 协方差与相关系数

*多维随机变量 · 线性相关性度量*


### 1. 定义


$$
Cov(X, Y) = E[(X - E(X))(Y - E(Y))] = E(XY) - E(X)·E(Y)
$$


协方差衡量两个随机变量的**线性相关**程度和方向。


### 2. 基本性质


- Cov(X, X) = D(X)
- Cov(X, Y) = Cov(Y, X)（对称性）
- Cov(aX, bY) = ab · Cov(X, Y)
- Cov(X₁+X₂, Y) = Cov(X₁, Y) + Cov(X₂, Y)
- **D(X±Y) = D(X) + D(Y) ± 2Cov(X, Y)**
- 若 X, Y 独立，则 Cov(X, Y) = 0（反之不一定成立）


### 1. 定义


$$
ρ_XY = Cov(X, Y) / (σ_X · σ_Y) = Cov(X, Y) / √(D(X)·D(Y))
$$


相关系数是**标准化后的协方差**，取值范围 [-1, 1]。


### 2. 含义解读


| ρ 的值 | 含义 |
| --- | --- |
| ρ = 1 | 完全正线性相关（Y = aX + b, a > 0） |
| ρ = -1 | 完全负线性相关（Y = aX + b, a < 0） |
| ρ = 0 | 不相关（无线性关系，但可能有非线性关系） |
| 0 < \|ρ\| < 1 | 部分线性相关 |


### 3. 重要性质


- |ρ_XY| ≤ 1（柯西-施瓦茨不等式）
- |ρ| = 1 ⟺ 存在常数 a, b 使 P(Y = aX + b) = 1
- **独立 ⟹ ρ = 0（不相关）**
- **ρ = 0 ⟹ 独立**
   （一般不成立，但对二维正态分布成立）
- ρ 只反映线性相关性，不反映非线性依赖关系


### 1. 矩的概念


| 类型 | 定义 | 说明 |
| --- | --- | --- |
| k阶原点矩 | E(Xᵏ) | k=1 即期望 |
| k阶中心矩 | E[(X-E(X))ᵏ] | k=2 即方差 |
| 混合矩 | E(XᵃYᵇ) | 反映两个变量的联合矩 |


### 2. 协方差矩阵


对于 n 维随机向量 X = (X₁, X₂, ..., Xₙ)ᵀ：


$$
Σ = [Cov(Xᵢ, Xⱼ)]ₙₓₙ
$$


对角线元素为各分量的方差，非对角线元素为协方差。协方差矩阵是**对称非负定矩阵**。
