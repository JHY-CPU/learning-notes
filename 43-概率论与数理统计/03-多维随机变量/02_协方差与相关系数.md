# 协方差与相关系数

*多维随机变量 · 线性相关性度量*


### 1. 定义


$$
Cov(X, Y) = E[(X - E(X))(Y - E(Y))] = E(XY) - E(X)·E(Y)
$$


协方差衡量两个随机变量的**线性相关**程度和方向。


### 2. 基本性质


- Cov(X, X) = D(X)
- Cov(X, Y) = Cov(Y, X)（对称性）
- Cov(aX, bY) = ab · Cov(X, Y)
- Cov(X₁+X₂, Y) = Cov(X₁, Y) + Cov(X₂, Y)
- **D(X±Y) = D(X) + D(Y) ± 2Cov(X, Y)**
- 若 X, Y 独立，则 Cov(X, Y) = 0（反之不一定成立）


### 1. 定义


$$
ρ_XY = Cov(X, Y) / (σ_X · σ_Y) = Cov(X, Y) / √(D(X)·D(Y))
$$


相关系数是**标准化后的协方差**，取值范围 [-1, 1]。


### 2. 含义解读


| ρ 的值 | 含义 |
| --- | --- |
| ρ = 1 | 完全正线性相关（Y = aX + b, a > 0） |
| ρ = -1 | 完全负线性相关（Y = aX + b, a < 0） |
| ρ = 0 | 不相关（无线性关系，但可能有非线性关系） |
| 0 < \|ρ\| < 1 | 部分线性相关 |


### 3. 重要性质


- |ρ_XY| ≤ 1（柯西-施瓦茨不等式）
- |ρ| = 1 ⟺ 存在常数 a, b 使 P(Y = aX + b) = 1
- **独立 ⟹ ρ = 0（不相关）**
- **ρ = 0 ⟹ 独立**
   （一般不成立，但对二维正态分布成立）
- ρ 只反映线性相关性，不反映非线性依赖关系


### 1. 矩的概念


| 类型 | 定义 | 说明 |
| --- | --- | --- |
| k阶原点矩 | E(Xᵏ) | k=1 即期望 |
| k阶中心矩 | E[(X-E(X))ᵏ] | k=2 即方差 |
| 混合矩 | E(XᵃYᵇ) | 反映两个变量的联合矩 |


### 2. 协方差矩阵


对于 n 维随机向量 X = (X₁, X₂, ..., Xₙ)ᵀ：


$$
Σ = [Cov(Xᵢ, Xⱼ)]ₙₓₙ
$$


对角线元素为各分量的方差，非对角线元素为协方差。协方差矩阵是**对称非负定矩阵**。

### 3. 计算例题

**例题**：设 (X,Y) 的联合分布律如下，求 Cov(X,Y) 和 ρ。

| | Y=0 | Y=1 | P(X=xᵢ) |
|---|---|---|---|
| X=0 | 0.3 | 0.2 | 0.5 |
| X=1 | 0.1 | 0.4 | 0.5 |
| P(Y=yⱼ) | 0.4 | 0.6 | 1 |

E(X) = 0×0.5 + 1×0.5 = 0.5
E(Y) = 0×0.4 + 1×0.6 = 0.6
E(XY) = 1×0.4 = 0.4
Cov(X,Y) = E(XY) - E(X)·E(Y) = 0.4 - 0.5×0.6 = 0.1

D(X) = E(X²) - [E(X)]² = 0.5 - 0.25 = 0.25
D(Y) = E(Y²) - [E(Y)]² = 0.6 - 0.36 = 0.24

ρ = 0.1 / √(0.25×0.24) = 0.1 / √0.06 ≈ 0.408

X 和 Y 存在中等程度的正线性相关。

### 4. 独立、不相关与相关的关系

```
独立 ⟹ 不相关（ρ=0）
不相关 ⟹ 独立（一般不成立）
不相关 ⟹ 独立（二维正态分布中成立）

关键区别：
- 独立：X和Y完全无关，没有任何依赖关系
- 不相关：X和Y没有线性关系，但可能有非线性关系
```

**反例**：设 X ~ U(-1,1), Y = X²。则 E(X)=0, E(XY)=E(X³)=0，Cov(X,Y)=0 即不相关。但 Y = X² 显然由 X 决定，不独立。

### 5. 协方差矩阵的性质

- 协方差矩阵 Σ 是对称矩阵：Σᵢⱼ = Σⱼᵢ
- 协方差矩阵是非负定的：对任意非零向量 a，aᵀΣa ≥ 0
- 若各分量相互独立，Σ 为对角矩阵
- 多元正态分布 N(μ, Σ) 完全由均值向量和协方差矩阵决定

### 6. 应用场景

| 场景 | 协方差/相关系数的作用 |
|------|---------------------|
| 投资组合 | 衡量资产间的联动性，分散风险 |
| 特征选择 | 去除高度相关的特征，减少冗余 |
| 主成分分析(PCA) | 基于协方差矩阵降维 |
| 回归分析 | 检验自变量与因变量的线性关系强度 |


## Python实现

### 协方差、相关系数与协方差矩阵

```python
import numpy as np

# 生成两个相关的随机变量
np.random.seed(42)
n = 10000
X = np.random.randn(n)
Y = 0.5 * X + 0.3 * np.random.randn(n)  # Y 与 X 线性相关

# 手动计算协方差
cov_manual = np.mean((X - X.mean()) * (Y - Y.mean()))
# 等价公式: E(XY) - E(X)E(Y)
cov_formula = np.mean(X * Y) - X.mean() * Y.mean()
print(f"Cov(X,Y) (定义式) = {cov_manual:.6f}")
print(f"Cov(X,Y) (公式) = {cov_formula:.6f}")
print(f"numpy cov = {np.cov(X, Y)[0,1]:.6f}")

# 相关系数
rho = cov_manual / (X.std() * Y.std())
print(f"ρ = {rho:.6f}")
print(f"numpy corrcoef = {np.corrcoef(X, Y)[0,1]:.6f}")

# 独立但不相关: Y = X², X ~ U(-1,1)
X_unif = np.random.uniform(-1, 1, n)
Y_sq = X_unif**2
cov_indep = np.cov(X_unif, Y_sq)[0, 1]
print(f"\nX~U(-1,1), Y=X²: Cov(X,Y) = {cov_indep:.6f} (应≈0)")
print(f"但 Y = X² 显然依赖 X!")
```

### 协方差矩阵与投资组合

```python
import numpy as np

# 三种资产的收益率
np.random.seed(42)
n = 1000
# 模拟三种资产的收益率
r1 = np.random.normal(0.10, 0.20, n)  # 股票
r2 = np.random.normal(0.05, 0.08, n)  # 债券
r3 = 0.7 * r1 + 0.3 * r2 + 0.02*np.random.randn(n)  # 混合资产

returns = np.column_stack([r1, r2, r3])
cov_matrix = np.cov(returns, rowvar=False)
print("协方差矩阵:")
print(np.round(cov_matrix, 4))

# 相关系数矩阵
corr_matrix = np.corrcoef(returns, rowvar=False)
print("\n相关系数矩阵:")
print(np.round(corr_matrix, 4))

# 投资组合方差: Var(w'R) = w'Σw
weights = np.array([0.5, 0.3, 0.2])
port_variance = weights @ cov_matrix @ weights
port_std = np.sqrt(port_variance)
print(f"\n投资组合权重: {weights}")
print(f"组合方差: {port_variance:.6f}")
print(f"组合标准差: {port_std:.6f}")
```
