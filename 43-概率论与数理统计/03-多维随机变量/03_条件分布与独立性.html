条件分布与随机变量独立性

多维随机变量 · 条件分布、独立性与次序统计量

一、条件分布

1. 离散型条件分布律

设 (X,Y) 为二维离散型随机变量，在 Y=yⱼ 的条件下，X 的条件分布律为：

P(X=xᵢ | Y=yⱼ) = P(X=xᵢ, Y=yⱼ) / P(Y=yⱼ) = pᵢⱼ / p·ⱼ

其中 p·ⱼ = P(Y=yⱼ) = Σᵢ pᵢⱼ 为 Y 的边缘分布律。

2. 连续型条件密度

设 (X,Y) 的联合密度为 f(x,y)，边缘密度为 f_Y(y)，在 Y=y 的条件下，X 的条件密度为：

fX|Y(x|y) = f(x,y) / fY(y),   fY(y) > 0

条件期望：E(X|Y=y) = ∫ x · fX|Y(x|y) dx

条件方差：Var(X|Y=y) = E(X²|Y=y) - [E(X|Y=y)]²

例题 1：条件密度计算

(X,Y) 的联合密度 f(x,y) = 24xy（0X|Y(x|0.5) 和 E(X|Y=0.5)。

先求 fY(y) = ∫₀1-y 24xy dx = 24y · (1-y)²/2 = 12y(1-y)², 0
fY(0.5) = 12 × 0.5 × 0.25 = 1.5

fX|Y(x|0.5) = 24x(0.5) / 1.5 = 8x, 0 < x < 0.5

E(X|Y=0.5) = ∫₀0.5 x · 8x dx = 8 × [x³/3]₀0.5 = 8 × 0.125/3 = 1/3
条件期望 vs 回归函数： 条件期望 E(X|Y=y) 是关于 y 的函数，称为 X 关于 Y 的回归函数

二、随机变量的独立性

1. 独立的定义

随机变量 X 与 Y 相互独立，当且仅当对任意实数 x, y：

F(x,y) = FX(x) · FY(y)  （联合CDF = 边缘CDF的乘积）

等价条件（更常用）：

  • 离散型：P(X=xᵢ, Y=yⱼ) = P(X=xᵢ) · P(Y=yⱼ) 对所有 i,j 成立
  • 连续型：f(x,y) = fX(x) · fY(y) 对所有 x,y 成立

2. 独立性的判定方法

方法判据适用场景
定义法f(x,y) = fX(x)fY(y)一般情况
分解定理f(x,y) = g(x)·h(y)，且取值范围为矩形快速判定
协方差=0Cov(X,Y)=0（必要非充分）仅判断相关性
特征函数φXY(t,s) = φX(t)·φY(s)理论证明
重要： 联合密度能写成 g(x)·h(y) 且取值范围为矩形区域（即 a

3. 独立随机变量的函数

若 X 与 Y 独立，则：

  • g(X) 与 h(Y) 独立（任意函数 g, h）
  • E(XY) = E(X)·E(Y)
  • D(X±Y) = D(X) + D(Y)（方差可加，注意无减法）
  • MX+Y(t) = MX(t)·MY(t)（矩母函数可乘）
例题 2：判定独立性

f(x,y) = 6e-2x-3y（x>0, y>0），判断 X,Y 是否独立。

f(x,y) = 6e-2x-3y = (2e-2x) · (3e-3y) = g(x)·h(y)

取值范围 x>0, y>0 是矩形区域。
可以验证：fX(x) = 2e-2x（x>0），fY(y) = 3e-3y（y>0）
f(x,y) = fX(x)·fY(y) ✓

X 与 Y 相互独立，且 X~Exp(2), Y~Exp(3)

三、条件分布的重要性质

重期望公式与重方差公式

E(X) = E[E(X|Y)]  （全期望公式）
Var(X) = E[Var(X|Y)] + Var[E(X|Y)]  （全方差公式 / Eve定律）

全方差公式的直觉：总方差 = 条件方差的期望 + 条件期望的方差（"期望中的变异 + 期望周围的变异"）

例题 3：全方差公式

某保险公司，投保人事故次数 N|λ~Poisson(λ)，其中 λ~Exp(1)。求 Var(N)。

已知 E(N|λ)=λ, Var(N|λ)=λ
E(λ)=1, Var(λ)=1（Exp(1)的参数）

E(N) = E[E(N|λ)] = E(λ) = 1
Var(N) = E[Var(N|λ)] + Var[E(N|λ)]
= E(λ) + Var(λ) = 1 + 1 = 2

（对比：若 λ 固定=1，Var(N)=1；随机 λ 导致方差增大）

四、次序统计量（Order Statistics）

1. 定义

设 X₁, X₂, ..., Xₙ 为来自总体 F 的简单随机样本，将它们从小到大排列：

X(1) ≤ X(2) ≤ ... ≤ X(n)

X(k) 称为第 k 个次序统计量。特别地：

  • X(1) = min(X₁,...,Xₙ)（最小次序统计量）
  • X(n) = max(X₁,...,Xₙ)（最大次序统计量）
  • 极差 R = X(n) - X(1)
  • 样本中位数：n 奇取 X((n+1)/2)，n 偶取 [X(n/2)+X(n/2+1)]/2

2. 次序统计量的分布

设 X₁,...,Xₙ 独立同分布，CDF 为 F(x)，PDF 为 f(x)：

FX(n)(x) = [F(x)]n    （最大值的CDF）
FX(1)(x) = 1 - [1-F(x)]n    （最小值的CDF）
fX(k)(x) = n! / [(k-1)!(n-k)!] · [F(x)]k-1 · [1-F(x)]n-k · f(x)
例题 4：极值分布

X₁,...,Xₙ 独立同服从 U(0,1)，求 X(n) 的密度和期望。

F(x)=x（0 FX(n)(x) = xn
fX(n)(x) = nxn-1, 0
E(X(n)) = ∫₀¹ x·nxn-1 dx = n/(n+1)
特例：n=2 时 E(X(2)) = 2/3
次序统计量的应用：
  • 极值分析：最大洪水量、最大风速等极端事件
  • 质量控制：寿命试验中的最短/最长寿命
  • 非参数统计：中位数检验、秩统计量
  • Bootstrap 方法中的重采样排序

本节总结

  • 条件密度：fX|Y(x|y) = f(x,y)/fY(y)
  • 随机变量独立：f(x,y) = fX(x)fY(y)，取值范围为矩形
  • 分解定理：f(x,y)=g(x)h(y) + 矩形区域 ⟹ 独立
  • 全方差公式：Var(X) = E[Var(X|Y)] + Var[E(X|Y)]
  • 次序统计量：X(1)最小，X(n)最大，各有分布公式
  • 极值分布：最大值CDF=[F(x)]n，最小值CDF=1-[1-F(x)]n
