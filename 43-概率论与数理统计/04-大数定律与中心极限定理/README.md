# 04-大数定律与中心极限定理

> 从切比雪夫不等式到中心极限定理，理解"频率趋近概率"和"和的分布趋近正态"

---

## 1. 切比雪夫不等式

### 1.1 马尔可夫不等式

对于非负随机变量 $X$ 和 $a > 0$：

$$P(X \geq a) \leq \frac{E[X]}{a}$$

这是概率论中最基本的不等式之一，无需知道分布形式。

### 1.2 切比雪夫不等式

设 $E[X] = \mu$，$\text{Var}(X) = \sigma^2 < \infty$，则对任意 $\varepsilon > 0$：

$$P(|X - \mu| \geq \varepsilon) \leq \frac{\sigma^2}{\varepsilon^2}$$

等价形式：$P(|X - \mu| < k\sigma) \geq 1 - \frac{1}{k^2}$

**意义**：无论分布形态如何，数据至少有 $1 - 1/k^2$ 的比例落在均值的 $k$ 个标准差范围内。

**推导**：由马尔可夫不等式应用于 $(X-\mu)^2$ 直接得到。

**局限**：上界通常较松，实际概率往往远好于此界。正态分布在 $k=2$ 时实际概率为95%，而切比雪夫只保证75%。

## 2. 大数定律

### 2.1 依概率收敛

$X_n \xrightarrow{P} X$：对任意 $\varepsilon > 0$，

$$\lim_{n \to \infty} P(|X_n - X| \geq \varepsilon) = 0$$

### 2.2 辛钦大数定律（弱大数定律）

设 $X_1, X_2, \ldots$ 为i.i.d.随机变量序列，$E[X_i] = \mu$ 存在，则：

$$\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \mu$$

**含义**：样本均值依概率收敛于总体均值。无需假设方差存在。

**证明思路**：对 $\bar{X}_n$ 应用切比雪夫不等式（需假设方差存在，但辛钦证明了期望存在即可）。

### 2.3 伯努利大数定律

设 $n_A$ 为 $n$ 次伯努利试验中事件 $A$ 发生的次数，$P(A) = p$，则：

$$\frac{n_A}{n} \xrightarrow{P} p$$

这是频率学派概率定义的理论基础——频率趋近于概率。

### 2.4 辛钦强大数定律（强大数定律）

设 $X_1, X_2, \ldots$ 为i.i.d.随机变量序列，$E[X_i] = \mu$ 存在，则：

$$\bar{X}_n \xrightarrow{a.s.} \mu$$

即 $P\left(\lim_{n\to\infty} \bar{X}_n = \mu\right) = 1$。

**弱大数定律 vs 强大数定律**：

| | 弱大数定律 | 强大数定律 |
|--|-----------|-----------|
| 收敛方式 | 依概率收敛 | 几乎必然收敛 |
| 含义 | $n$ 大时 $\bar{X}_n$ 偏离 $\mu$ 的概率很小 | $\bar{X}_n$ 最终一定收敛到 $\mu$ |
| 强度 | 较弱 | 较强（蕴含弱大数定律） |

### 2.5 大数定律的意义

- **理论**：为频率稳定性提供数学证明
- **蒙特卡洛方法**：样本均值可以逼近期望值，是MCMC的基础
- **ML应用**：经验风险最小化（ERM）的理论依据——训练集上的平均损失逼近期望损失

## 3. 中心极限定理

### 3.1 林德伯格-勒维中心极限定理

设 $X_1, X_2, \ldots$ 为i.i.d.随机变量序列，$E[X_i] = \mu$，$\text{Var}(X_i) = \sigma^2 < \infty$，则：

$$\frac{\sum_{i=1}^n X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)$$

或等价地：

$$\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)$$

**含义**：大量独立同分布随机变量之和（标准化后）近似服从正态分布，与原始分布形态无关。

**近似公式**：当 $n$ 足够大时，

$$\sum_{i=1}^n X_i \approx \mathcal{N}(n\mu,\ n\sigma^2)$$

### 3.2 棣莫弗-拉普拉斯中心极限定理

这是CLT在二项分布上的特例。设 $S_n \sim \text{Binomial}(n, p)$，当 $n \to \infty$ 时：

$$\frac{S_n - np}{\sqrt{np(1-p)}} \xrightarrow{d} \mathcal{N}(0, 1)$$

**近似条件**：$np \geq 5$ 且 $n(1-p) \geq 5$ 时近似效果较好。

**连续性修正**：离散变量近似连续时，$P(S_n = k) \approx \Phi\left(\frac{k+0.5-np}{\sqrt{np(1-p)}}\right) - \Phi\left(\frac{k-0.5-np}{\sqrt{np(1-p)}}\right)$

### 3.3 定理的应用条件

| 条件 | 说明 |
|------|------|
| 独立性 | 各随机变量相互独立（可放宽至弱依赖） |
| 同分布 | 林德伯格-勒维要求同分布（李雅普诺夫CLT可放宽） |
| 方差有限 | $\sigma^2 < \infty$ 是关键条件（柯西分布不满足） |
| $n$ 足够大 | 一般 $n \geq 30$ 即可，但取决于原始分布的偏度和峰度 |

### 3.4 为什么正态分布如此普遍？

CLT解释了自然界中正态分布无处不在的现象：任何受大量微小独立随机因素影响的量，其分布趋近正态。例如：
- 人的身高（受大量基因和环境因素影响）
- 测量误差
- 金融资产收益率

## 4. 定理的实际意义

### 4.1 大数定律 vs 中心极限定理

| | 大数定律 | 中心极限定理 |
|--|---------|-------------|
| 回答问题 | $\bar{X}_n$ 收敛到哪里？ | $\bar{X}_n$ 围绕均值如何波动？ |
| 结论 | $\bar{X}_n \to \mu$ | $\bar{X}_n \approx \mathcal{N}(\mu, \sigma^2/n)$ |
| 信息量 | 只给出收敛方向 | 给出波动的分布形态和速率 |

### 4.2 蒙特卡洛积分

用CLT估计蒙特卡洛误差：

$$\text{SE}(\bar{X}_n) = \frac{\sigma}{\sqrt{n}}$$

误差以 $O(1/\sqrt{n})$ 速率下降——精度提高10倍需要100倍样本。

### 4.3 Bootstrap的理论基础

Bootstrap方法依赖CLT：经验分布的统计量近似正态分布，使得可以用重抽样估计标准误和置信区间。

## 5. 统计推断中的应用

### 5.1 参数估计的渐近正态性

MLE具有渐近正态性：

$$\hat{\theta}_{MLE} \xrightarrow{d} \mathcal{N}\left(\theta_0,\ \frac{1}{nI(\theta_0)}\right)$$

其中 $I(\theta_0)$ 是Fisher信息量。

### 5.2 置信区间的构造

基于CLT，样本均值的95%置信区间：

$$\bar{X}_n \pm z_{0.025} \cdot \frac{S}{\sqrt{n}}$$

其中 $z_{0.025} \approx 1.96$，$S$ 为样本标准差。

### 5.3 假设检验

$Z = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}$ 在 $H_0$ 下近似 $\mathcal{N}(0,1)$，构成 $Z$ 检验的基础。

### 5.4 大样本推断

对于非正态总体，只要 $n$ 足够大，仍可用正态近似进行推断——这正是CLT的威力所在。

**ML应用**：
- 置信区间用于模型评估（如AUC的置信区间）
- A/B测试的样本量计算基于CLT
- SGD的收敛性分析依赖大数定律
