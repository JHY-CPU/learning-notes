中心极限定理

大数定律与中心极限定理 · 为什么正态分布如此重要

一、核心思想

中心极限定理的直观含义

无论单个随机变量服从什么分布，大量独立随机变量之和（或均值）近似服从正态分布。

这就是正态分布如此重要的根本原因——它自然地从大量随机因素的叠加中涌现出来。

二、独立同分布的中心极限定理（林德伯格-列维）

定理表述

设 X₁, X₂, ... i.i.d.，E(Xᵢ) = μ, D(Xᵢ) = σ² (0 < σ² < +∞)，则当 n → ∞ 时：

(∑ᵢ₌₁ⁿ Xᵢ - nμ) / (σ√n) →ᵈ N(0, 1)

等价地，样本均值 X̄ₙ 近似：

X̄ₙ ~ N(μ, σ²/n)     （n 足够大时）

"足够大"的经验法则：通常 n ≥ 30 即可近似。

例题 1：i.i.d. 中心极限定理

某保险公司有 10000 份保单，每份保单索赔额 X（单位：万元）服从 E(X)=1, D(X)=4 的分布。求总索赔额超过 10400 万元的概率。

设 S = X₁ + X₂ + ... + X₁₀₀₀₀
E(S) = 10000 × 1 = 10000
D(S) = 10000 × 4 = 40000, σ_S = 200

由 CLT：S ~ N(10000, 200²) 近似
P(S > 10400) = P(Z > (10400-10000)/200) = P(Z > 2)
= 1 - Φ(2) = 1 - 0.9772 = 0.0228

三、棣莫弗-拉普拉斯定理

定理表述（二项分布的正态近似）

设 X ~ B(n, p)，当 n 足够大时（np 和 n(1-p) 都较大），则：

(X - np) / √(np(1-p)) →ᵈ N(0, 1)

即 X ~ N(np, np(1-p)) 近似。经验法则：np ≥ 5 且 n(1-p) ≥ 5 即可使用。

例题 2：二项分布的正态近似

某工厂产品次品率为 0.05，现从 1000 件产品中随机抽查，求次品数超过 60 的概率。

X ~ B(1000, 0.05), np = 50, np(1-p) = 47.5

正态近似：X ~ N(50, 47.5)
P(X > 60) = P(Z > (60-50)/√47.5) = P(Z > 10/6.892)
= P(Z > 1.451) ≈ 1 - Φ(1.45) = 1 - 0.9265 = 0.0735

四、李雅普诺夫定理（独立不同分布）

定理表述

设 X₁, X₂, ... 相互独立（不必同分布），E(Xᵢ) = μᵢ, D(Xᵢ) = σᵢ²，令 Sₙ = ΣXᵢ，Bₙ² = Σσᵢ²。

若满足李雅普诺夫条件（存在 δ > 0 使三阶矩有界），则：

(Sₙ - Σμᵢ) / Bₙ →ᵈ N(0, 1)

意义：即使各随机变量的分布不同，只要独立且方差有限，它们的和仍然近似正态。

五、中心极限定理的应用

实际应用场景

场景说明
误差分析测量误差是大量微小随机因素之和，近似正态分布
质量控制产品指标受多因素影响，样本均值近似正态
民意调查样本比例的抽样分布近似正态
金融风险投资组合收益是众多资产收益的加权和
信号处理噪声是大量独立小信号之和，近似高斯噪声
例题 3：样本比例的正态近似

某城市选民支持率为 55%，随机调查 400 人，求样本支持率在 50%~60% 之间的概率。

p̂ ~ N(p, p(1-p)/n) 近似
p̂ ~ N(0.55, 0.55×0.45/400) = N(0.55, 0.00061875)
σ_p̂ = √0.00061875 ≈ 0.02487

P(0.50 < p̂ < 0.60) = Φ((0.60-0.55)/0.02487) - Φ((0.50-0.55)/0.02487)
= Φ(2.01) - Φ(-2.01) = 0.9778 - 0.0222 = 0.9556
大数定律 vs 中心极限定理：
  • 大数定律：X̄ₙ → μ（样本均值收敛于期望）——回答"收敛到哪里"
  • 中心极限定理：√n(X̄ₙ - μ) →ᵈ N(0, σ²)（收敛速度和分布形状）——回答"以什么方式收敛"
  • CLT 是 LLN 的精细化：不仅说明收敛，还给出收敛速率是 √n，分布形状是正态

本节总结

  • 核心思想：大量独立随机变量之和近似正态分布
  • i.i.d. CLT：(ΣXᵢ - nμ)/(σ√n) → N(0,1)
  • 棣莫弗-拉普拉斯：二项分布 B(n,p) 近似 N(np, np(1-p))
  • 李雅普诺夫 CLT：独立不同分布之和也近似正态
  • 经验法则：n ≥ 30（一般情况），np≥5 且 n(1-p)≥5（二项分布）
  • CLT 解释了正态分布在自然界和社会科学中普遍存在的原因
