# 中心极限定理

*大数定律与中心极限定理 · 为什么正态分布如此重要*


### 中心极限定理的直观含义


无论单个随机变量服从什么分布，**大量独立随机变量之和（或均值）近似服从正态分布**。


这就是正态分布如此重要的根本原因——它自然地从大量随机因素的叠加中涌现出来。


### 定理表述


设 X₁, X₂, ... i.i.d.，E(Xᵢ) = μ, D(Xᵢ) = σ² (0 < σ² < +∞)，则当 n → ∞ 时：


$$
(∑ᵢ₌₁ⁿ Xᵢ - nμ) / (σ√n) →ᵈ N(0, 1)
$$


等价地，样本均值 X̄ₙ 近似：


$$
X̄ₙ ~ N(μ, σ²/n)     （n 足够大时）
$$


"足够大"的经验法则：通常 n ≥ 30 即可近似。


### 定理表述（二项分布的正态近似）


设 X ~ B(n, p)，当 n 足够大时（np 和 n(1-p) 都较大），则：


$$
(X - np) / √(np(1-p)) →ᵈ N(0, 1)
$$


即 X ~ N(np, np(1-p)) 近似。经验法则：np ≥ 5 且 n(1-p) ≥ 5 即可使用。


### 定理表述


设 X₁, X₂, ... 相互独立（不必同分布），E(Xᵢ) = μᵢ, D(Xᵢ) = σᵢ²，令 Sₙ = ΣXᵢ，Bₙ² = Σσᵢ²。


若满足李雅普诺夫条件（存在 δ > 0 使三阶矩有界），则：


$$
(Sₙ - Σμᵢ) / Bₙ →ᵈ N(0, 1)
$$


意义：即使各随机变量的分布不同，只要独立且方差有限，它们的和仍然近似正态。


### 实际应用场景


| 场景 | 说明 |
| --- | --- |
| 误差分析 | 测量误差是大量微小随机因素之和，近似正态分布 |
| 质量控制 | 产品指标受多因素影响，样本均值近似正态 |
| 民意调查 | 样本比例的抽样分布近似正态 |
| 金融风险 | 投资组合收益是众多资产收益的加权和 |
| 信号处理 | 噪声是大量独立小信号之和，近似高斯噪声 |

### 计算例题

**例题1**：一批零件重量 X~N(50, 25)（单位：g），随机取 100 个装箱，求总重量在 4950g~5050g 之间的概率。

S₁₀₀ = ΣXᵢ ~ N(100×50, 100×25) = N(5000, 2500)

P(4950 < S₁₀₀ < 5050) = P(-1 < Z < 1) = 2Φ(1) - 1 = 2×0.8413 - 1 = 0.6826

**例题2**：保险公司有 10000 份保单，每份索赔概率 0.01，索赔金额相互独立且服从均值 1000 元的指数分布。求总索赔额不超过 120000 的概率。

设第 i 份保单索赔额为 Yᵢ，总索赔 S = ΣYᵢ

E(Yᵢ) = 0.01 × 1000 = 10
D(Yᵢ) = E(Yᵢ²) - [E(Yᵢ)]² = 0.01×(1000²+1000²) - 100 = 19900

由中心极限定理：S ~ N(100000, 199000000)

P(S ≤ 120000) = Φ((120000-100000)/√199000000) = Φ(1.42) ≈ 0.9222

### CLT 与 LLN 的对比

| 对比维度 | 大数定律(LLN) | 中心极限定理(CLT) |
|----------|-------------|-------------------|
| 结论 | X̄ₙ → μ（收敛到期望） | (X̄ₙ-μ)/(σ/√n) 近似 N(0,1) |
| 信息量 | 只知道趋近方向 | 给出趋近的速度和分布形状 |
| 应用 | 频率稳定性、大样本统计 | 置信区间、假设检验 |
| 近似质量 | 无误差信息 | n≥30 通常足够好 |

### 独立不同分布的CLT

**李雅普诺夫 CLT**：X₁, X₂, ... 独立但不必同分布，若存在 δ>0 使得
$$\lim_{n→∞} \frac{1}{B_n^{2+δ}} \sum_{i=1}^n E[|X_i - μ_i|^{2+δ}] = 0$$

则 (Sₙ - Σμᵢ)/Bₙ →ᵈ N(0,1)

**林德伯格-费勒 CLT**：给出更精确的条件——单个随机变量对总和的贡献趋于均匀小。

### 易错点

1. **n≥30 是经验法则**，不是严格标准。对于偏态分布可能需要更大的 n
2. **连续性修正**：用正态近似离散分布时，P(X=k) ≈ P(k-0.5 < Y < k+0.5)
3. **CLT 要求独立性**：非独立变量之和不满足CLT（需要不同的理论）
4. **方差必须有限**：柯西分布不满足CLT条件（方差无穷大）


## Python实现

### 中心极限定理的模拟验证

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

np.random.seed(42)

# 验证: 即使原始分布非正态, 均值的分布趋近正态
distributions = {
    '均匀分布 U(0,1)': lambda size: np.random.uniform(0, 1, size),
    '指数分布 Exp(1)': lambda size: np.random.exponential(1, size),
    '二项分布 B(10,0.3)': lambda size: np.random.binomial(10, 0.3, size),
}

fig, axes = plt.subplots(1, 3, figsize=(15, 4))
for ax, (name, dist_func) in zip(axes, distributions.items()):
    # 抽取大量样本均值
    n = 30  # 每组样本量
    n_samples = 10000
    means = np.array([dist_func(n).mean() for _ in range(n_samples)])

    # 标准化
    mu = means.mean()
    sigma = means.std()
    standardized = (means - mu) / sigma

    ax.hist(standardized, bins=50, density=True, alpha=0.7, label='样本均值')
    x = np.linspace(-4, 4, 100)
    ax.plot(x, stats.norm.pdf(x), 'r-', label='N(0,1)')
    ax.set_title(f'{name}\nn={n}')
    ax.legend()

plt.suptitle('中心极限定理: 样本均值趋近正态分布', fontsize=14)
plt.tight_layout()
plt.savefig('clt_demo.png', dpi=100)
plt.show()
```

### 二项分布的正态近似

```python
import numpy as np
from scipy import stats

# 二项分布 B(n, p) 的正态近似
n, p = 100, 0.3
binom = stats.binom(n, p)
mu_binom = n * p
sigma_binom = np.sqrt(n * p * (1 - p))

print(f"=== 二项分布 B({n}, {p}) 的正态近似 ===")
print(f"np = {mu_binom}, √(np(1-p)) = {sigma_binom:.2f}")

# 连续性修正: P(X=k) ≈ P(k-0.5 < Y < k+0.5)
for k in [25, 30, 35, 40]:
    exact = binom.pmf(k)
    # 不用连续性修正
    approx_no_cc = stats.norm.pdf(k, mu_binom, sigma_binom)
    # 用连续性修正
    approx_cc = stats.norm.cdf(k+0.5, mu_binom, sigma_binom) - \
                stats.norm.cdf(k-0.5, mu_binom, sigma_binom)
    print(f"P(X={k}): 精确={exact:.6f}, 正态(无修正)={approx_no_cc:.6f}, 正态(连续修正)={approx_cc:.6f}")
```
