# 大数定律

*大数定律与中心极限定理 · 频率收敛于概率*


### 公式


设随机变量 X 的期望 E(X) = μ，方差 D(X) = σ²，则对任意 ε > 0：


$$
P(|X - μ| ≥ ε) ≤ σ² / ε²

            P(|X - μ| < ε) ≥ 1 - σ² / ε²
$$


意义：方差越小，随机变量越集中于期望附近。**不依赖分布的具体形式**，只要期望和方差存在即可。


### 核心思想


当试验次数 n 足够大时，随机变量的算术平均值**趋近于**其数学期望。


$$
X̄ₙ = (X₁ + X₂ + ... + Xₙ) / n → E(X)    （依概率或几乎必然）
$$


### 1. 切比雪夫大数定律


设 X₁, X₂, ... 两两不相关，且方差有上界 D(Xᵢ) ≤ C，则


$$
X̄ₙ →ᵖ E(X̄ₙ)   （依概率收敛）
$$


### 2. 辛钦大数定律（独立同分布）


设 X₁, X₂, ... 独立同分布（i.i.d.），E(Xᵢ) = μ 存在，则


$$
X̄ₙ = (1/n)ΣXᵢ →ᵖ μ
$$


这是最重要的大数定律形式，是频率稳定性的理论基础。


### 3. 伯努利大数定律（弱大数定律）


设 nₐ 是 n 次独立重复试验中事件 A 发生的次数，P(A) = p，则


$$
nₐ/n →ᵖ p   （频率收敛于概率）
$$


### 4. 强大数定律（柯尔莫哥洛夫）


设 X₁, X₂, ... i.i.d.，E(Xᵢ) = μ，则


$$
X̄ₙ →ᵃ·ˢ· μ   （几乎必然收敛）
$$


"几乎必然"比"依概率"更强：不仅是概率趋于 0，而是除了概率为零的例外情况外，极限精确成立。


| 定律 | 条件 | 收敛性 |
| --- | --- | --- |
| 切比雪夫 LLN | 两两不相关，方差有界 | 依概率 |
| 辛钦 LLN | i.i.d.，期望存在 | 依概率 |
| 伯努利 LLN | n次伯努利试验 | 依概率 |
| 柯尔莫哥洛夫 LLN | i.i.d.，期望存在 | 几乎必然 |

### 应用实例

**例题1：蒙特卡罗方法求π**

在正方形 [0,1]×[0,1] 内随机撒点，统计落在四分之一圆 x²+y²≤1 内的比例 p，由大数定律 p → π/4：

```python
import numpy as np
n = 10_000_000
x = np.random.uniform(0, 1, n)
y = np.random.uniform(0, 1, n)
inside = np.sum(x**2 + y**2 <= 1)
pi_estimate = 4 * inside / n
# 当 n 足够大时，pi_estimate → 3.14159...
```

**例题2：保险公司定价**

某险种每单赔付 Xᵢ（i.i.d.），E(Xᵢ)=1000元，σ=500元。承保 n=10000 单，由大数定律：
- 平均赔付 X̄ → 1000（概率趋于1）
- 总赔付 nX̄ → 1000万
- 定价策略：保费 = 1000 × (1+利润率)，大数定律保证了盈利确定性

### 依概率收敛与几乎必然收敛的区别

| 收敛类型 | 含义 | 直觉理解 |
|----------|------|---------|
| 依概率收敛 X̄ₙ →ᵖ μ | P(|X̄ₙ-μ|>ε) → 0 | 偶尔偏离，但偏离概率趋于0 |
| 几乎必然收敛 X̄ₙ →ᵃ·ˢ· μ | P(lim X̄ₙ = μ) = 1 | 极限序列精确等于μ |

### 易错点

1. **大数定律不给出收敛速度**：只知道 X̄ₙ 会趋近 μ，但不知道多快（这由中心极限定理给出）
2. **条件不可忽视**：辛钦大数定律要求期望存在，切比雪夫要求方差有界
3. **伯努利大数定律是辛钦定律的特例**：伯努利分布的期望 np/n = p
4. **"频率收敛于概率"是弱大数定律**：强大数定律给出了更强的保证

### 大数定律的模拟验证

```python
import numpy as np

# 蒙特卡洛方法求 π
np.random.seed(42)
print("=== 蒙特卡洛求 π ===")
for n in [100, 1000, 10000, 100000, 1000000]:
    x = np.random.uniform(0, 1, n)
    y = np.random.uniform(0, 1, n)
    inside = np.sum(x**2 + y**2 <= 1)
    pi_est = 4 * inside / n
```
