大数定律

大数定律与中心极限定理 · 频率收敛于概率

一、切比雪夫不等式

公式

设随机变量 X 的期望 E(X) = μ，方差 D(X) = σ²，则对任意 ε > 0：

P(|X - μ| ≥ ε) ≤ σ² / ε²

P(|X - μ| < ε) ≥ 1 - σ² / ε²

意义：方差越小，随机变量越集中于期望附近。不依赖分布的具体形式，只要期望和方差存在即可。

例题 1：切比雪夫不等式应用

已知 D(X) = 4，用切比雪夫不等式估计 P(|X-E(X)| ≥ 5) 的上界。

P(|X-μ| ≥ 5) ≤ σ²/ε² = 4/25 = 0.16

二、大数定律

核心思想

当试验次数 n 足够大时，随机变量的算术平均值趋近于其数学期望。

X̄ₙ = (X₁ + X₂ + ... + Xₙ) / n → E(X)    （依概率或几乎必然）

1. 切比雪夫大数定律

设 X₁, X₂, ... 两两不相关，且方差有上界 D(Xᵢ) ≤ C，则

X̄ₙ →ᵖ E(X̄ₙ)   （依概率收敛）

2. 辛钦大数定律（独立同分布）

设 X₁, X₂, ... 独立同分布（i.i.d.），E(Xᵢ) = μ 存在，则

X̄ₙ = (1/n)ΣXᵢ →ᵖ μ

这是最重要的大数定律形式，是频率稳定性的理论基础。

3. 伯努利大数定律（弱大数定律）

设 nₐ 是 n 次独立重复试验中事件 A 发生的次数，P(A) = p，则

nₐ/n →ᵖ p   （频率收敛于概率）

4. 强大数定律（柯尔莫哥洛夫）

设 X₁, X₂, ... i.i.d.，E(Xᵢ) = μ，则

X̄ₙ →ᵃ·ˢ· μ   （几乎必然收敛）

"几乎必然"比"依概率"更强：不仅是概率趋于 0，而是除了概率为零的例外情况外，极限精确成立。

例题 2：大数定律应用

抛一枚均匀硬币，求至少抛多少次才能使正面频率与 0.5 的偏差小于 0.01 的概率超过 99%？

由辛钦大数定律（或切比雪夫不等式）：
P(|nₐ/n - 0.5| < 0.01) ≥ 1 - D(X̄ₙ)/(0.01)²

X̄ₙ ~ B(n, 0.5)/n, D(X̄ₙ) = 0.25/n
1 - 0.25/(n × 0.0001) ≥ 0.99
0.25/(0.0001n) ≤ 0.01 → n ≥ 250000

至少需要抛 250,000 次

注意：切比雪夫不等式给出的界通常很保守，实际需要的次数远少于这个理论估计。

三、各大数定律对比

定律条件收敛性
切比雪夫 LLN两两不相关，方差有界依概率
辛钦 LLNi.i.d.，期望存在依概率
伯努利 LLNn次伯努利试验依概率
柯尔莫哥洛夫 LLNi.i.d.，期望存在几乎必然
大数定律的现实意义：
  • 频率的稳定性：大量重复试验中频率趋于概率
  • 蒙特卡洛方法的理论基础：用随机样本的均值估计期望值
  • 保险精算：大量保单的平均赔付趋于期望赔付
  • 质量控制：大量产品中次品率趋于真实的次品概率

本节总结

  • 切比雪夫不等式：P(|X-μ|≥ε) ≤ σ²/ε²，适用于任何分布
  • 辛钦大数定律：i.i.d. 时，样本均值依概率收敛于总体期望
  • 伯努利大数定律：频率依概率收敛于概率
  • 强大数定律：几乎必然收敛，比依概率收敛更强
  • 核心含义：大量重复试验下，随机现象呈现出稳定的统计规律
