卡方检验与非参数检验

参数估计与假设检验 · 卡方检验与秩方法

一、卡方拟合优度检验

1. 基本思想

检验观测数据的频数分布是否符合某个理论分布。

原假设 H₀：总体服从指定的分布 F₀(x)

备择 H₁：总体不服从 F₀(x)

2. 检验统计量

χ² = Σ (Oᵢ - Eᵢ)² / Eᵢ ~ χ²(k-1-r)
  • Oᵢ：第 i 组的观测频数
  • Eᵢ：第 i 组的期望频数（= n × 理论概率）
  • k：分组数
  • r：从数据中估计的参数个数
  • 自由度 = k - 1 - r
重要条件： 每组期望频数 Eᵢ ≥ 5。若某组 Eᵢ < 5，需合并相邻组。
例题 1：检验骰子是否均匀

掷骰子 120 次，各面出现次数为：18, 22, 19, 21, 20, 20。检验骰子是否均匀（α=0.05）。

H₀：各面概率均为 1/6
期望频数 Eᵢ = 120/6 = 20（每组）

χ² = Σ(Oᵢ-Eᵢ)²/Eᵢ = [(18-20)²+(22-20)²+(19-20)²+(21-20)²+(20-20)²+(20-20)²]/20
= (4+4+1+1+0+0)/20 = 10/20 = 0.5

自由度 = 6-1 = 5
χ²0.05(5) = 11.07
0.5 < 11.07 → 不拒绝 H₀，没有证据表明骰子不均匀。

二、卡方独立性检验

1. 问题描述

基于 r×c 列联表，检验两个分类变量是否独立。

H₀：两个变量独立    H₁：两个变量不独立（有关联）

2. 检验统计量

χ² = Σᵢ Σⱼ (Oᵢⱼ - Eᵢⱼ)² / Eᵢⱼ ~ χ²((r-1)(c-1))

期望频数：Eᵢⱼ = (第 i 行合计 × 第 j 列合计) / 总计

自由度 = (r-1)(c-1)

例题 2：独立性检验

调查性别与是否喜欢数学的关系：

喜欢不喜欢合计
男302050
女252550
合计5545100
期望频数表：
E(男,喜欢) = 50×55/100 = 27.5
E(男,不喜欢) = 50×45/100 = 22.5
E(女,喜欢) = 50×55/100 = 27.5
E(女,不喜欢) = 50×45/100 = 22.5

χ² = (30-27.5)²/27.5 + (20-22.5)²/22.5 + (25-27.5)²/27.5 + (25-22.5)²/22.5
= 0.227 + 0.278 + 0.227 + 0.278 = 1.01

df = (2-1)(2-1) = 1
χ²0.05(1) = 3.84
1.01 < 3.84 → 不拒绝 H₀，性别与是否喜欢数学无显著关联。

三、非参数检验概述

参数 vs 非参数检验

方面参数检验非参数检验
分布假设需假设总体分布（如正态）不依赖总体分布假设
数据类型定量数据（连续）可处理定序、分类数据
检验效力高（假设满足时）略低（但稳健）
常见方法t检验、F检验秩检验、卡方检验

四、Wilcoxon 符号秩检验

用途与步骤

用于单样本或配对样本的中位数检验（替代单样本 t 检验或配对 t 检验）。

H₀：中位数 = M₀   （或配对差值的中位数 = 0）

  1. 计算 dᵢ = xᵢ - M₀（或配对差值）
  2. 去掉 dᵢ=0 的观测，记剩余个数为 n
  3. 对 |dᵢ| 排秩（相同取平均秩）
  4. 分别计算正秩和 W⁺ 和负秩和 W⁻
  5. 取 W = min(W⁺, W⁻) 为检验统计量
  6. 查 Wilcoxon 符号秩检验表，W 越小越拒绝 H₀
例题 3：Wilcoxon 符号秩检验

9 名患者的治疗前后差值为：3, -1, 4, 2, 5, -2, 6, 1, 3。检验治疗是否有效（中位数是否>0）。

去掉 0（无），n=9
|d| 排秩：|3|=3→6, |-1|=1→2, |4|=4→7, |2|=2→4.5, |5|=5→8, |-2|=2→4.5, |6|=6→9, |1|=1→2, |3|=3→6

正秩和 W⁺ = 6+7+4.5+8+9+2+6 = 42.5
负秩和 W⁻ = 2+4.5 = 6.5
W = min(42.5, 6.5) = 6.5

查表 n=9, α=0.05(单侧): W₀.₀₅=5
6.5 > 5 → 不拒绝 H₀（在α=0.05水平）

五、Mann-Whitney U 检验

用途与步骤

用于两个独立样本的位置比较（替代独立样本 t 检验）。

H₀：两总体分布相同（或中位数相等）

  1. 将两组数据合并，从小到大排秩
  2. 分别计算两组的秩和 R₁, R₂
  3. 计算 U 统计量：
    U₁ = n₁n₂ + n₁(n₁+1)/2 - R₁
    U₂ = n₁n₂ + n₂(n₂+1)/2 - R₂
    U = min(U₁, U₂)
  4. 查 Mann-Whitney 临界值表，U 越小越拒绝 H₀
例题 4：Mann-Whitney U 检验

两组学生成绩：A组(n₁=4): 85, 92, 78, 88；B组(n₂=5): 72, 80, 76, 84, 79。检验两组是否有显著差异。

合并排秩（从小到大）：
72(B,秩1), 76(B,秩2), 78(A,秩3), 79(B,秩4), 80(B,秩5),
84(B,秩6), 85(A,秩7), 88(A,秩8), 92(A,秩9)

R₁(A组) = 3+7+8+9 = 27
R₂(B组) = 1+2+4+5+6 = 18

U₁ = 4×5 + 4×5/2 - 27 = 20+10-27 = 3
U₂ = 4×5 + 5×6/2 - 18 = 20+15-18 = 17
U = min(3, 17) = 3

查表 n₁=4, n₂=5, α=0.05: U₀.₀₅=1
3 > 1 → 不拒绝 H₀（无显著差异）

六、常用非参数检验方法对比

方法对应参数检验数据类型适用场景
Wilcoxon 符号秩单样本/配对 t 检验配对/单样本中位数检验，非正态数据
Mann-Whitney U独立样本 t 检验两独立样本两组位置比较
Kruskal-Wallis单因素 ANOVA多独立样本多组中位数比较
Friedman重复测量 ANOVA多相关样本区组设计
符号检验单样本 t 检验配对/单样本最简单的非参数检验
Spearman 秩相关Pearson 相关两变量非线性单调关系
选择非参数检验的时机：
  • 样本量很小（n < 30）且总体分布未知
  • 数据严重偏态，不满足正态假设
  • 数据含异常值（秩方法对异常值不敏感）
  • 数据是定序型（等级、排名）
  • 参数检验的前提条件无法满足或无法验证

本节总结

  • 卡方拟合优度：χ²=Σ(Oᵢ-Eᵢ)²/Eᵢ ~ χ²(k-1-r)，检验数据是否符合理论分布
  • 卡方独立性：基于列联表检验两个分类变量的关联性，df=(r-1)(c-1)
  • Wilcoxon 符号秩：配对/单样本的非参数替代，基于秩和检验
  • Mann-Whitney U：两独立样本的非参数替代，基于合并排秩
  • Kruskal-Wallis：多组比较的非参数替代（ANOVA 的非参数版本）
  • 共同点：不依赖分布假设，基于秩或频数，对异常值稳健
