# 点估计与区间估计

*参数估计 · 最大似然估计与置信区间*


### 1. 矩估计法


**思想**：用样本矩估计总体矩（用样本均值估计总体均值等）。


**步骤**：令总体的 k 阶原点矩等于样本的 k 阶原点矩，解方程组。


$$
E(Xᵏ) = (1/n) Σ Xᵢᵏ    （k = 1, 2, ..., m，m 为待估参数个数）
$$


### 2. 最大似然估计法（MLE）


**思想**：选取使观测数据出现概率最大的参数值。


**步骤**：


1. 写出似然函数：L(θ) = ∏ f(xᵢ; θ)
2. 取对数：ln L(θ) = Σ ln f(xᵢ; θ)
3. 对 θ 求导并令导数为 0：d(ln L)/dθ = 0
4. 解出 θ 即为 MLE


### 3. 估计量的评价标准


| 标准 | 定义 | 说明 |
| --- | --- | --- |
| 无偏性 | E(θ̂) = θ | 估计量的期望等于真值 |
| 有效性 | D(θ̂₁) < D(θ̂₂) | 方差越小越好 |
| 一致性 | θ̂ →ᵖ θ | n→∞ 时依概率收敛于真值 |


### 1. 基本概念


**置信区间**：以一定的置信水平（通常 95%）包含参数真值的区间。


$$
P(θ̂_L ≤ θ ≤ θ̂_U) = 1 - α
$$


其中 1-α 称为置信水平（如 α=0.05 时，置信水平为 95%）。


注意：置信区间是随机的，参数 θ 是固定的。95% 置信水平意味着：如果反复抽样构造区间，约 95% 的区间会包含真值。


### 2. 正态总体均值的置信区间


| 条件 | 枢轴量 | 置信区间 |
| --- | --- | --- |
| σ² 已知 | Z = (X̄-μ)/(σ/√n) ~ N(0,1) | X̄ ± z_{α/2} · σ/√n |
| σ² 未知 | t = (X̄-μ)/(S/√n) ~ t(n-1) | X̄ ± t_{α/2}(n-1) · S/√n |


S² = (1/(n-1))Σ(Xᵢ-X̄)² 为样本方差。

### 3. MLE 计算例题

**例题**：设 X₁, ..., Xₙ 是来自总体 B(1, p) 的样本（即伯努利分布），求 p 的 MLE。

似然函数：L(p) = ∏p^xᵢ(1-p)^(1-xᵢ) = p^(Σxᵢ)(1-p)^(n-Σxᵢ)

取对数：ln L = (Σxᵢ)ln p + (n-Σxᵢ)ln(1-p)

求导：d(ln L)/dp = (Σxᵢ)/p - (n-Σxᵢ)/(1-p) = 0

解得：p̂ = (Σxᵢ)/n = X̄（样本均值即为 MLE）

**例题**：设 X₁, ..., Xₙ 是来自 N(μ, σ²) 的样本，求 μ 和 σ² 的 MLE。

ln L = -n/2·ln(2π) - n/2·ln(σ²) - Σ(xᵢ-μ)²/(2σ²)

∂(ln L)/∂μ = Σ(xᵢ-μ)/σ² = 0 → μ̂ = X̄

∂(ln L)/∂(σ²) = -n/(2σ²) + Σ(xᵢ-μ)²/(2σ⁴) = 0 → σ̂² = (1/n)Σ(xᵢ-X̄)²

注意：σ² 的 MLE 是有偏估计（除以 n），无偏估计是 S² = (1/(n-1))Σ(xᵢ-X̄)²

### 4. 矩估计与MLE的对比

| 对比维度 | 矩估计 | 最大似然估计 |
|----------|--------|-------------|
| 思路 | 令样本矩=总体矩 | 最大化似然函数 |
| 计算难度 | 通常较简单 | 可能需要求导/数值优化 |
| 有效性 | 通常较低 | 渐近有效（n→∞时达到C-R下界） |
| 一致性 | 是 | 是 |
| 不变性 | 不具有 | 具有不变性：g(θ)的MLE=g(θ̂_MLE) |
| 适用范围 | 总是存在 | 需要密度函数可写 |

### 5. 置信区间的计算例题

**例题**：从正态总体抽取 n=25 的样本，X̄=50, S=10。求 μ 的 95% 置信区间（σ 未知）。

自由度 df = 24，α/2 = 0.025，查 t 分布表 t₀.₀₂₅(24) = 2.064

置信区间 = X̄ ± t_{α/2}·S/√n = 50 ± 2.064×10/5 = 50 ± 4.128 = (45.872, 54.128)

### 6. 常见陷阱

1. **置信区间不是概率陈述**：不能说"μ有95%的概率在区间内"，应该说"95%的置信区间会覆盖真值"
2. **MLE 的有偏性**：正态总体方差的MLE是有偏的，需用 n-1 校正
3. **样本量影响**：置信区间宽度与 √n 成反比，想将宽度减半需4倍样本量
4. **σ 已知与未知**：σ 已知用 Z 分布，未知用 t 分布（小样本时差异显著）


## Python实现

### MLE 与置信区间

```python
import numpy as np
from scipy import stats
from scipy.optimize import minimize_scalar

# MLE: 正态分布的参数估计
np.random.seed(42)
true_mu, true_sigma = 50, 10
data = np.random.normal(true_mu, true_sigma, 100)

# MLE 估计
mu_mle = data.mean()
sigma2_mle = np.mean((data - mu_mle)**2)  # 除以 n (有偏)
sigma2_unbiased = np.var(data, ddof=1)     # 除以 n-1 (无偏)

print(f"=== 点估计 ===")
print(f"真值: μ={true_mu}, σ²={true_sigma**2}")
print(f"MLE: μ̂={mu_mle:.4f}, σ̂²={sigma2_mle:.4f}")
print(f"无偏估计: S²={sigma2_unbiased:.4f}")

# 置信区间 (σ 未知, 用 t 分布)
n = len(data)
x_bar = data.mean()
S = data.std(ddof=1)
alpha = 0.05
t_crit = stats.t.ppf(1 - alpha/2, n - 1)
margin = t_crit * S / np.sqrt(n)
CI = (x_bar - margin, x_bar + margin)

print(f"\n=== 95% 置信区间 ===")
print(f"X̄ = {x_bar:.4f}, S = {S:.4f}")
print(f"t_0.025({n-1}) = {t_crit:.4f}")
print(f"置信区间: ({CI[0]:.4f}, {CI[1]:.4f})")
print(f"真值 {true_mu} 在区间内: {CI[0] <= true_mu <= CI[1]}")

# 用 scipy 直接计算
ci_scipy = stats.t.interval(0.95, df=n-1, loc=x_bar, scale=S/np.sqrt(n))
print(f"scipy 验证: {ci_scipy}")
```

### 置信区间宽度与样本量

```python
import numpy as np
from scipy import stats

# 置信区间宽度 = 2 * t_{α/2} * S/√n
# 想将宽度减半, 需要 4 倍样本量
print("=== 样本量对置信区间的影响 ===")
sigma = 10
for n in [10, 25, 100, 400, 1000]:
    z_crit = stats.norm.ppf(0.975)
    width = 2 * z_crit * sigma / np.sqrt(n)
    print(f"n={n:4d}: 宽度 = {width:.4f}")

# 所需样本量: n = (z_{α/2} * σ / E)^2
desired_margin = 1.0  # 希望误差在 1 以内
n_needed = (z_crit * sigma / desired_margin) ** 2
print(f"\n希望边际误差 E={desired_margin}, 所需样本量 n ≈ {int(np.ceil(n_needed))}")
```
