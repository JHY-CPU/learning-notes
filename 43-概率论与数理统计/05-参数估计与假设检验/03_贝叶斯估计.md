# 贝叶斯估计

*参数估计与假设检验 · 贝叶斯推断*


### 1. 核心思想


经典频率学派将参数 θ 视为固定未知常数；贝叶斯学派将 θ 视为**随机变量**，具有先验分布 π(θ)。


贝叶斯推断的三要素：


- **先验分布**
   π(θ)：在观测数据之前对参数的认知
- **似然函数**
   L(θ|x) = f(x|θ)：给定参数时数据的分布
- **后验分布**
   π(θ|x)：观测数据后对参数的更新认知


### 2. 贝叶斯公式（密度函数形式）


$$
π(θ|x) = f(x|θ) · π(θ) / m(x)
$$


其中边际密度（归一化常数）：


$$
m(x) = ∫ f(x|θ) · π(θ) dθ  （连续参数）
            m(x) = Σ f(x|θᵢ) · π(θᵢ)  （离散参数）
$$


实用形式（去掉常数因子）：


$$
π(θ|x) ∝ L(θ|x) · π(θ)    （后验 ∝ 似然 × 先验）
$$


### 1. 先验分布的类型


| 类型 | 说明 | 示例 |
| --- | --- | --- |
| 信息先验 | 基于历史数据或专家知识 | 以往试验数据拟合的分布 |
| 无信息先验 | 尽量不引入主观信息 | 均匀分布 π(θ)∝1；Jeffreys先验 |
| 共轭先验 | 后验与先验属于同一分布族 | 见下表 |
| 分层先验 | 先验的参数也赋予分布 | 超参数的先验 |


### 2. 共轭先验（Conjugate Prior）


若后验 π(θ|x) 与先验 π(θ) 属于同一分布族，则称 π(θ) 为似然函数的**共轭先验**。


| 似然（数据分布） | 参数 | 共轭先验 | 后验参数更新 |
| --- | --- | --- | --- |
| Bernoulli / Binomial | p（成功概率） | Beta(α,β) | Beta(α+Σxᵢ, β+n-Σxᵢ) |
| Poisson | λ | Gamma(a,b) | Gamma(a+Σxᵢ, b+n) |
| Normal（方差已知） | μ | N(μ₀, τ²) | N(μ₁, σ²₁)，见例题 |
| Normal（均值已知） | σ² | InvGamma(a,b) | InvGamma(a+n/2, b+Σ(xᵢ-μ)²/2) |
| Exponential | λ | Gamma(a,b) | Gamma(a+n, b+Σxᵢ) |


### 后验估计准则


| 估计量 | 定义 | 优化目标 |
| --- | --- | --- |
| 后验均值（Posterior Mean） | E(θ\|x) = ∫ θ·π(θ\|x)dθ | 最小化后验均方误差 |
| 后验中位数 | 后验分布的中位数 | 最小化后验绝对误差 |
| MAP（最大后验估计） | argmax~θ~ π(θ\|x) | 最大化后验密度 |


常用准则为后验均值（贝叶斯估计通常指后验均值）。


### 公式对比


$$
MLE：θ̂ = argmaxθ L(θ|x)    （最大化似然）
$$


$$
MAP：θ̂ = argmaxθ [L(θ|x) · π(θ)]    （最大化 似然×先验）
$$


$$
取对数：MAP = argmaxθ [log L(θ|x) + log π(θ)]
$$


### 详细对比


| 方面 | MLE（极大似然估计） | MAP（最大后验估计） |
| --- | --- | --- |
| 哲学 | 频率学派：参数是固定的 | 贝叶斯学派：参数是随机变量 |
| 目标 | argmax L(θ\|x) | argmax L(θ\|x)π(θ) |
| 先验 | 不使用先验信息 | 使用先验分布 π(θ) |
| 小样本 | 可能不稳定 | 先验起正则化作用，更稳定 |
| 大样本 | 一致、渐近正态、渐近有效 | 随样本增大趋近 MLE（先验影响减弱） |
| 关系 | 先验为均匀分布时，MAP = MLE |
| 正则化视角 | — | MAP 等价于 MLE + 先验的正则项 |


### 可信区间（Credible Interval）


基于后验分布 π(θ|x)，参数 θ 的 1-α 可信区间为满足：


$$
P(θL ≤ θ ≤ θU | x) = 1 - α
$$


常用类型：


- **等尾可信区间**
   ：取后验分布的 α/2 和 1-α/2 分位数
- **最高后验密度区间（HPD）**
   ：区间内后验密度不低于区间外


注意：贝叶斯可信区间 ≠ 频率学派的置信区间！可信区间给出的是参数落在该区间内的概率，置信区间的意义则基于重复抽样。


## Python实现

### 贝叶斯估计: 共轭先验

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 二项分布 + Beta 共轭先验
# 先验: Beta(α, β), 似然: Binomial(n, p), 后验: Beta(α+Σx, β+n-Σx)

# 场景: 抛硬币 30 次, 20 次正面
n_trials, n_heads = 30, 20

# 不同先验的对比
priors = {
    '无信息先验 Beta(1,1)': (1, 1),
    '弱先验 Beta(2,2)': (2, 2),
    '强先验 (偏公平) Beta(10,10)': (10, 10),
}

p = np.linspace(0, 1, 200)
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
for ax, (name, (alpha_prior, beta_prior)) in zip(axes, priors.items()):
    # 先验
    prior = stats.beta(alpha_prior, beta_prior)
    ax.plot(p, prior.pdf(p), 'b--', label='先验', linewidth=2)

    # 后验
    alpha_post = alpha_prior + n_heads
    beta_post = beta_prior + (n_trials - n_heads)
    posterior = stats.beta(alpha_post, beta_post)
    ax.plot(p, posterior.pdf(p), 'r-', label='后验', linewidth=2)

    # MLE
    mle = n_heads / n_trials
    ax.axvline(mle, color='green', linestyle=':', label=f'MLE={mle:.2f}')

    ax.set_title(f'{name}\n后验: Beta({alpha_post},{beta_post})')
    ax.legend(fontsize=8)
    ax.set_xlabel('p')

plt.tight_layout()
plt.savefig('bayesian_estimation.png', dpi=100)
plt.show()

# 打印后验统计量
for name, (a, b) in priors.items():
    post_mean = (a + n_heads) / (a + b + n_trials)
    post_mode = (a + n_heads - 1) / (a + b + n_trials - 2)
    print(f"{name}: 后验均值={post_mean:.4f}, 后验众数={post_mode:.4f}")

print(f"MLE: {n_heads/n_trials:.4f}")
```
