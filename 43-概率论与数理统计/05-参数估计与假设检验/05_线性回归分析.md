# 线性回归分析

*参数估计与假设检验 · 回归建模与诊断*


### 1. 模型


$$
Y = β₀ + β₁X + ε,    ε ~ N(0, σ²)
$$


- β₀：截距（x=0 时 y 的期望）
- β₁：斜率（x 每增加1单位，y 平均变化 β₁）
- ε：随机误差，独立同分布 N(0,σ²)


### 2. 最小二乘估计（OLS）


使残差平方和 Q = Σ(yᵢ - β₀ - β₁xᵢ)² 最小化：


$$
β̂₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)² = Sxy / Sxx = Lxy / Lxx
$$


$$
β̂₀ = ȳ - β̂₁x̄
$$


其中：L~xx~ = Σ(xᵢ-x̄)²，L~xy~ = Σ(xᵢ-x̄)(yᵢ-ȳ)，L~yy~ = Σ(yᵢ-ȳ)²


### 3. 回归系数的方差


$$
Var(β̂₁) = σ² / Lxx
$$


$$
Var(β̂₀) = σ² [1/n + x̄²/Lxx]
$$


$$
σ̂² = SSE / (n-2) = Σ(yᵢ-ŷᵢ)² / (n-2)    （σ² 的无偏估计）
$$


### 三种平方和


$$
SST = SSR + SSE
$$


| 名称 | 记号 | 公式 | 自由度 | 含义 |
| --- | --- | --- | --- | --- |
| 总平方和 | SS~T~=L~yy~ | Σ(yᵢ-ȳ)² | n-1 | y 的总变异 |
| 回归平方和 | SS~R~ | Σ(ŷᵢ-ȳ)² = β̂₁²L~xx~ | 1 | x 可解释的变异 |
| 残差平方和 | SS~E~=SSE | Σ(yᵢ-ŷᵢ)² | n-2 | 未解释的变异（误差） |


### 定义与计算


$$
R² = SSR / SST = 1 - SSE / SST
$$


R² 的含义：回归方程解释了 y 总变异的百分比。


- R² = 1：完美拟合，所有点在回归线上
- R² = 0：回归线没有解释任何变异
- 简单回归中：R² = r²（r 为 Pearson 相关系数）
- 多重回归中：使用调整 R²（adjusted R²）


$$
R²adj = 1 - (SSE/(n-p-1)) / (SST/(n-1))
$$


其中 p 为自变量个数。调整 R² 惩罚变量过多。


### 1. 对 β₁ 的 t 检验


H₀: β₁ = 0（x 与 y 无线性关系）vs H₁: β₁ ≠ 0


$$
t = β̂₁ / SE(β̂₁) = β̂₁ / (σ̂/√Lxx) ~ t(n-2)
$$


若 |t| > t~α/2~(n-2)，拒绝 H₀，认为 x 与 y 有显著线性关系。


### 2. 回归方程的 F 检验


H₀: β₁ = 0 vs H₁: β₁ ≠ 0


$$
F = (SSR/1) / (SSE/(n-2)) = MSR / MSE ~ F(1, n-2)
$$


简单回归中 F = t²（完全等价！）。多重回归中 F 检验检验整体显著性。


### 1. 残差定义


$$
eᵢ = yᵢ - ŷᵢ    （观测值 - 拟合值）
$$


标准化残差：rᵢ = eᵢ / σ̂


### 2. 残差图诊断


| 残差图形态 | 含义 | 处理 |
| --- | --- | --- |
| 随机散布在0附近 | 模型合适 | 无需处理 |
| 漏斗形（发散） | 方差不齐 | 对 y 做变换（log、√） |
| 弯曲趋势 | 非线性关系 | 加入 x² 项或做变换 |
| 残差间相关 | 独立性假设违反 | 时间序列方法 |


### 3. 回归假设检查清单


| 假设 | 检查方法 |
| --- | --- |
| 线性（E(Y) = β₀+β₁x） | 散点图、残差图 |
| 独立性（εᵢ 独立） | Durbin-Watson 检验 |
| 同方差性（Var(ε)=σ² 恒定） | 残差 vs 拟合值图 |
| 正态性（ε ~ N(0,σ²)） | Q-Q图、Shapiro-Wilk检验 |


### 模型


$$
Y = β₀ + β₁X₁ + β₂X₂ + ... + βpXp + ε
$$


矩阵形式：


$$
Y = Xβ + ε,    β̂ = (X'X)⁻¹X'Y
$$


- F 检验：检验整体回归方程是否显著（H₀: β₁=β₂=...=β
   ~p~
   =0）
- t 检验：检验单个回归系数是否显著（H₀: βⱼ=0）
- 调整 R²：考虑变量个数的拟合优度
- 多重共线性：检查 VIF（方差膨胀因子），VIF>10 表示严重共线性


## Python实现

### 线性回归分析

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

np.random.seed(42)

# 生成数据: y = 3 + 2x + ε
x = np.random.uniform(0, 10, 50)
y = 3 + 2 * x + np.random.normal(0, 2, 50)

# OLS 估计
x_bar, y_bar = x.mean(), y.mean()
beta1_hat = np.sum((x - x_bar) * (y - y_bar)) / np.sum((x - x_bar)**2)
beta0_hat = y_bar - beta1_hat * x_bar
print(f"=== 线性回归 ===")
print(f"β̂₀ = {beta0_hat:.4f}, β̂₁ = {beta1_hat:.4f}")
print(f"拟合方程: y = {beta0_hat:.2f} + {beta1_hat:.2f}x")

# 残差分析
y_pred = beta0_hat + beta1_hat * x
residuals = y - y_pred
SSE = np.sum(residuals**2)
SST = np.sum((y - y_bar)**2)
SSR = SST - SSE
R2 = 1 - SSE / SST
n = len(x)
sigma2_hat = SSE / (n - 2)

print(f"\nSSE = {SSE:.2f}, SSR = {SSR:.2f}, SST = {SST:.2f}")
print(f"R² = {R2:.4f}")
print(f"σ̂² = {sigma2_hat:.4f}")

# t 检验: H0: β1 = 0
SE_beta1 = np.sqrt(sigma2_hat / np.sum((x - x_bar)**2))
t_stat = beta1_hat / SE_beta1
p_value = 2 * (1 - stats.t.cdf(abs(t_stat), n - 2))
print(f"\nt = {t_stat:.4f}, p = {p_value:.6f}")
print(f"β₁ 显著不为0: {'是' if p_value < 0.05 else '否'}")

# 用 numpy.polyfit 验证
coeffs = np.polyfit(x, y, 1)
print(f"\npolyfit 验证: y = {coeffs[1]:.4f} + {coeffs[0]:.4f}x")

# 可视化
plt.figure(figsize=(8, 5))
plt.scatter(x, y, alpha=0.6, label='数据点')
x_line = np.linspace(x.min(), x.max(), 100)
plt.plot(x_line, beta0_hat + beta1_hat * x_line, 'r-', linewidth=2, label=f'y = {beta0_hat:.1f} + {beta1_hat:.1f}x')
plt.xlabel('x')
plt.ylabel('y')
plt.title(f'线性回归 (R² = {R2:.3f})')
plt.legend()
plt.savefig('linear_regression.png', dpi=100)
plt.show()
```
