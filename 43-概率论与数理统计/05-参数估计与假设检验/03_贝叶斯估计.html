贝叶斯估计

参数估计与假设检验 · 贝叶斯推断

一、贝叶斯推断框架

1. 核心思想

经典频率学派将参数 θ 视为固定未知常数；贝叶斯学派将 θ 视为随机变量，具有先验分布 π(θ)。

贝叶斯推断的三要素：

  • 先验分布 π(θ)：在观测数据之前对参数的认知
  • 似然函数 L(θ|x) = f(x|θ)：给定参数时数据的分布
  • 后验分布 π(θ|x)：观测数据后对参数的更新认知

2. 贝叶斯公式（密度函数形式）

π(θ|x) = f(x|θ) · π(θ) / m(x)

其中边际密度（归一化常数）：

m(x) = ∫ f(x|θ) · π(θ) dθ  （连续参数）
m(x) = Σ f(x|θᵢ) · π(θᵢ)  （离散参数）

实用形式（去掉常数因子）：

π(θ|x) ∝ L(θ|x) · π(θ)    （后验 ∝ 似然 × 先验）

二、先验分布的选择

1. 先验分布的类型

类型说明示例
信息先验基于历史数据或专家知识以往试验数据拟合的分布
无信息先验尽量不引入主观信息均匀分布 π(θ)∝1；Jeffreys先验
共轭先验后验与先验属于同一分布族见下表
分层先验先验的参数也赋予分布超参数的先验

2. 共轭先验（Conjugate Prior）

若后验 π(θ|x) 与先验 π(θ) 属于同一分布族，则称 π(θ) 为似然函数的共轭先验。

似然（数据分布）参数共轭先验后验参数更新
Bernoulli / Binomialp（成功概率）Beta(α,β)Beta(α+Σxᵢ, β+n-Σxᵢ)
PoissonλGamma(a,b)Gamma(a+Σxᵢ, b+n)
Normal（方差已知）μN(μ₀, τ²)N(μ₁, σ²₁)，见例题
Normal（均值已知）σ²InvGamma(a,b)InvGamma(a+n/2, b+Σ(xᵢ-μ)²/2)
ExponentialλGamma(a,b)Gamma(a+n, b+Σxᵢ)
例题 1：Beta-Binomial 共轭

某产品次品率 p 的先验为 Beta(2,8)。抽取 20 件发现 3 件次品，求 p 的后验分布。

先验：π(p) ~ Beta(2, 8)，即 α₀=2, β₀=8
似然：X|p ~ B(20, p)，观测 Σxᵢ=3

后验：π(p|x) ~ Beta(2+3, 8+20-3) = Beta(5, 25)

后验均值：E(p|x) = 5/(5+25) = 1/6 ≈ 0.167
（先验均值 2/10=0.2，观测频率 3/20=0.15，后验均值在两者之间！）
例题 2：Normal-Normal 共轭

已知 σ²=4，μ 的先验为 N(0,4)。观测数据 x̄=2（n=16），求 μ 的后验分布。

先验：μ ~ N(μ₀=0, τ₀²=4)
数据：x̄|μ ~ N(μ, σ²/n=4/16=0.25)

后验精度 = 先验精度 + 数据精度：
1/τ₁² = 1/4 + 1/0.25 = 0.25 + 4 = 4.25 → τ₁² = 1/4.25 ≈ 0.235

后验均值（精度加权平均）：
μ₁ = (0/4 + 2/0.25) / 4.25 = 8/4.25 ≈ 1.882

后验：μ|x̄ ~ N(1.882, 0.235)

三、贝叶斯点估计

后验估计准则

估计量定义优化目标
后验均值（Posterior Mean）E(θ|x) = ∫ θ·π(θ|x)dθ最小化后验均方误差
后验中位数后验分布的中位数最小化后验绝对误差
MAP（最大后验估计）argmaxθ π(θ|x)最大化后验密度

常用准则为后验均值（贝叶斯估计通常指后验均值）。

四、MAP 与 MLE 的比较

公式对比

MLE：θ̂ = argmaxθ L(θ|x)    （最大化似然）
MAP：θ̂ = argmaxθ [L(θ|x) · π(θ)]    （最大化 似然×先验）
取对数：MAP = argmaxθ [log L(θ|x) + log π(θ)]

详细对比

方面MLE（极大似然估计）MAP（最大后验估计）
哲学频率学派：参数是固定的贝叶斯学派：参数是随机变量
目标argmax L(θ|x)argmax L(θ|x)π(θ)
先验不使用先验信息使用先验分布 π(θ)
小样本可能不稳定先验起正则化作用，更稳定
大样本一致、渐近正态、渐近有效随样本增大趋近 MLE（先验影响减弱）
关系先验为均匀分布时，MAP = MLE
正则化视角—MAP 等价于 MLE + 先验的正则项
例题 3：MAP vs MLE 对比

抛硬币 n 次，正面 x 次，参数 p 为正面概率。先验 p~Beta(a,b)，比较 MLE 和 MAP。

MLE：
L(p) = px(1-p)n-x
log L = x log p + (n-x) log(1-p)
d(log L)/dp = x/p - (n-x)/(1-p) = 0
θ̂MLE = x/n（观测频率）

MAP：
log[π(p|data)] ∝ x log p + (n-x) log(1-p) + (a-1) log p + (b-1) log(1-p)
= (x+a-1) log p + (n-x+b-1) log(1-p)
θ̂MAP = (x+a-1) / (n+a+b-2)

当 a=b=1（均匀先验=Beta(1,1)）时，MAP = x/n = MLE ✓
贝叶斯方法的优势：
  • 自然处理小样本问题：先验信息补偿数据不足
  • 提供完整的后验分布，而非单点估计
  • 可直接计算参数的概率区间（可信区间）
  • 适合序贯更新：新数据到来时，旧后验成为新先验
  • 在机器学习中：MAP 对应 L2 正则化（高斯先验）或 L1 正则化（拉普拉斯先验）

五、贝叶斯可信区间

可信区间（Credible Interval）

基于后验分布 π(θ|x)，参数 θ 的 1-α 可信区间为满足：

P(θL ≤ θ ≤ θU | x) = 1 - α

常用类型：

  • 等尾可信区间：取后验分布的 α/2 和 1-α/2 分位数
  • 最高后验密度区间（HPD）：区间内后验密度不低于区间外

注意：贝叶斯可信区间 ≠ 频率学派的置信区间！可信区间给出的是参数落在该区间内的概率，置信区间的意义则基于重复抽样。

本节总结

  • 贝叶斯公式：后验 ∝ 似然 × 先验，π(θ|x) ∝ L(θ|x)π(θ)
  • 共轭先验：Beta-Binomial、Gamma-Poisson、Normal-Normal 等，后验有解析形式
  • 贝叶斯点估计：后验均值（最小化MSE）、MAP（最大化后验密度）
  • MAP vs MLE：MAP = MLE + 先验正则项；均匀先验时 MAP = MLE
  • 可信区间：基于后验分布的区间，直接给出参数概率
  • 核心优势：小样本、序贯更新、完整不确定性量化
