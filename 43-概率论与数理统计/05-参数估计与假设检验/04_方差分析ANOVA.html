方差分析（ANOVA）

参数估计与假设检验 · 单因素方差分析

一、方差分析的基本思想

1. 问题背景

设有 k 个不同水平（处理/组），每个水平下有 nᵢ 个观测，判断各组均值是否有显著差异。

原假设 H₀：μ₁ = μ₂ = ... = μₖ（所有组均值相等）

备择假设 H₁：至少有一对均值不相等

2. 核心思想：变异分解

总变异 = 组间变异 + 组内变异：

SST = SSB + SSW
  • SST（总平方和）：所有观测值与总均值的偏差平方和
  • SSB（组间平方和）：各组均值与总均值的偏差平方和（反映处理效应）
  • SSW（组内平方和）：各观测值与其组均值的偏差平方和（反映随机误差）

二、单因素方差分析模型

1. 数学模型

Xij = μ + αᵢ + εij
  • Xij：第 i 组的第 j 个观测值
  • μ：总体均值
  • αᵢ = μᵢ - μ：第 i 组的处理效应
  • εij ~ N(0,σ²)：随机误差，相互独立

H₀：α₁ = α₂ = ... = αₖ = 0（无处理效应）

2. 平方和公式

设总样本数 N = Σnᵢ，总均值 x̄ = ΣΣxij/N，各组均值 x̄i：

SST = Σᵢ Σⱼ (xij - x̄)²    （总平方和，df = N-1）
SSB = Σᵢ nᵢ(x̄i - x̄)²    （组间平方和，df = k-1）
SSW = Σᵢ Σⱼ (xij - x̄i)²    （组内平方和/误差，df = N-k）

3. 均方与 F 检验

MSB = SSB / (k-1)    （组间均方 = 处理均方）
MSW = SSW / (N-k)    （组内均方 = 误差均方）
F = MSB / MSW    ~ F(k-1, N-k)  在 H₀ 成立时

判断：若 F > Fα(k-1, N-k)，则拒绝 H₀，认为各组均值有显著差异。

例题 1：单因素方差分析

三种饲料对猪增重的影响，数据如下（单位：kg）。检验三种饲料效果是否有显著差异（α=0.05）。

饲料A：51, 50, 49, 52, 48（n₁=5, x̄₁=50）
饲料B：55, 53, 54, 52, 56（n₂=5, x̄₂=54）
饲料C：48, 47, 49, 50, 46（n₃=5, x̄₃=48）

总均值 x̄ = (250+270+240)/15 = 50.67

SSB = 5(50-50.67)² + 5(54-50.67)² + 5(48-50.67)² = 2.22+55.56+35.56 = 93.33
SSW = Σ(x-x̄i)² = (1+0+1+4+4)+(1+1+0+4+4)+(4+9+1+0+4) = 34.00
SST = 93.33+34.00 = 127.33

MSB = 93.33/2 = 46.67
MSW = 34.00/12 = 2.83
F = 46.67/2.83 = 16.49

F0.05(2,12) = 3.89
16.49 > 3.89 → 拒绝 H₀，三种饲料效果有显著差异。

四、事后多重比较（Post-Hoc Tests）

为什么需要事后检验？

ANOVA 的 F 检验只告诉我们"至少有一对均值不同"，但不告诉具体哪对不同。若 F 检验显著，需用事后检验进行两两比较。

常用方法

方法全称适用场景控制指标
LSD最小显著差异法组数少，初步探索不控制族错误率
Bonferroni邦弗朗尼校正通用，保守族错误率 ≤ α
Tukey HSDTukey诚实显著差异各组样本量相等族错误率 = α
Scheffe谢弗法任意对比，最保守族错误率 = α
DuncanDuncan多重极差组数较多较宽松
Bonferroni 校正原理： 若做 m 次两两比较，将显著性水平调整为 α/m。例如 α=0.05，k=3 组需比较 C(3,2)=3 次，每次用 α'=0.05/3=0.0167。

五、ANOVA 的前提条件与检验

三大前提

前提含义检验方法
独立性各组观测值相互独立实验设计保证
正态性各组数据服从正态分布Shapiro-Wilk检验、Q-Q图
方差齐性各组方差相等Levene检验、Bartlett检验

违反正态性：可使用非参数方法（Kruskal-Wallis检验）

违反方差齐性：可使用Welch ANOVA

六、效应量

η²（Eta Squared）

η² = SSB / SST

表示组间变异占总变异的比例（解释力度）。

η² 值效应大小
0.01小效应
0.06中效应
0.14大效应

例题1中：η² = 93.33/127.33 = 0.73（大效应，饲料因素解释了73%的变异）

本节总结

  • 核心思想：总变异 = 组间变异 + 组内变异，SST = SSB + SSW
  • F 检验：F = MSB / MSW ~ F(k-1, N-k)，判断组间差异是否显著
  • 方差分析表：列出 SS、df、MS、F 值
  • 事后检验：ANOVA 显著后进行两两比较（Tukey、Bonferroni 等）
  • 前提条件：独立性、正态性、方差齐性
  • 效应量：η² = SSB/SST，衡量实际差异大小
