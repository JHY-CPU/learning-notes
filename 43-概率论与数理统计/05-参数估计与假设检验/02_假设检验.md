# 假设检验

*参数估计与假设检验 · 统计推断的核心方法*


### 1. 原假设与备择假设


- **原假设 H₀**
   ：待检验的假设（通常为"无效果""无差异"）
- **备择假设 H₁**
   ：与原假设对立的假设（通常为"有效果""有差异"）


| 类型 | H₀ | H₁ | 检验方向 |
| --- | --- | --- | --- |
| 双侧检验 | μ = μ₀ | μ ≠ μ₀ | 两侧 |
| 左侧检验 | μ ≥ μ₀ | μ < μ₀ | 左侧 |
| 右侧检验 | μ ≤ μ₀ | μ > μ₀ | 右侧 |


### 2. 两类错误


|  | H₀ 为真 | H₀ 为假 |
| --- | --- | --- |
| **拒绝 H₀** | 第一类错误（α）弃真 | 正确（检验力 1-β） |
| **不拒绝 H₀** | 正确 | 第二类错误（β）取伪 |


**显著性水平 α**：控制第一类错误的概率，通常取 0.05 或 0.01。


### 3. 检验步骤


1. 提出假设 H₀ 和 H₁
2. 选择检验统计量
3. 确定拒绝域（根据 α）
4. 计算统计量的观测值
5. 做出判断：统计量落入拒绝域则拒绝 H₀


### 定义与解读


**p 值**：在 H₀ 成立的前提下，出现当前样本（或更极端样本）的概率。


- p < α → 拒绝 H₀（结果具有统计显著性）
- p ≥ α → 不拒绝 H₀（证据不足以拒绝）


$$
p 值越小 → 拒绝 H₀ 的证据越强 → 对 H₀ 越不利
$$


### 1. Z 检验（σ 已知或大样本）


$$
Z = (X̄ - μ₀) / (σ/√n) ~ N(0,1)
$$


### 2. t 检验（σ 未知，小样本正态总体）


$$
t = (X̄ - μ₀) / (S/√n) ~ t(n-1)
$$


### 3. χ² 检验（方差检验）


$$
χ² = (n-1)S² / σ₀² ~ χ²(n-1)
$$

### 假设检验计算例题

**例题1（单样本t检验）**：某厂生产的零件标准长度 μ₀=10cm。现抽取 n=16 个样品，测得 X̄=10.2cm, S=0.4cm。在 α=0.05 下检验零件长度是否达标。

H₀: μ = 10, H₁: μ ≠ 10

t = (10.2-10)/(0.4/√16) = 0.2/0.1 = 2.0

t₀.₀₂₅(15) = 2.131（查t分布表）

由于 |t| = 2.0 < 2.131，不拒绝 H₀，没有足够证据表明零件长度不达标。

**例题2（双样本t检验）**：比较两种教学方法的效果。方法A：n₁=20, X̄₁=78, S₁=8。方法B：n₂=25, X̄₂=82, S₂=6。在 α=0.05 下检验两种方法是否有显著差异。

H₀: μ₁ = μ₂, H₁: μ₁ ≠ μ₂

t = (78-82)/√(8²/20 + 6²/25) = -4/√(3.2+1.44) = -4/2.154 = -1.857

自由度近似（Welch法）：df ≈ 35

t₀.₀₂₅(35) ≈ 2.030，|t| = 1.857 < 2.030，不拒绝 H₀。

### p 值的深入理解

```
p值 = 在H₀成立时，出现当前或更极端样本的概率

p值小 ← 当前样本与H₀矛盾严重 ← 拒绝H₀的证据强
p值大 ← 当前样本与H₀不矛盾 ← 不能拒绝H₀

注意：p值不是H₀为真的概率！
```

### 假设检验与置信区间的关系

- 若 μ₀ 不在 (1-α) 置信区间内，则在 α 水平下拒绝 H₀: μ = μ₀
- 两者是等价的：置信区间给出了所有不被拒绝的 μ₀ 值

### 单侧检验与双侧检验的选择

| 检验类型 | 使用场景 | 示例 |
|----------|---------|------|
| 双侧检验 | 关心是否"不等于" | 新药效果是否与旧药不同 |
| 右侧检验 | 关心是否"大于" | 新工艺是否提高了产量 |
| 左侧检验 | 关心是否"小于" | 新方法是否降低了成本 |

### 常见陷阱

1. **"不拒绝 H₀" ≠ "接受 H₀"**：只是证据不足以拒绝，不等于证明 H₀ 正确
2. **p 值不是效应量**：统计显著 ≠ 实际意义显著（大样本下微小差异也可能显著）
3. **多重比较问题**：同时做多个检验会增加第一类错误率，需要 Bonferroni 校正
4. **样本量影响检验力**：样本越大，越容易检测到微小差异，但也可能检测到无实际意义的差异


## Python实现

### 假设检验的 Python 实现

```python
import numpy as np
from scipy import stats

# 单样本 t 检验
# H0: μ = 10, H1: μ ≠ 10
np.random.seed(42)
data = np.random.normal(10.5, 2, 30)  # 真值 10.5
mu_0 = 10

t_stat, p_value = stats.ttest_1samp(data, mu_0)
print(f"=== 单样本 t 检验 ===")
print(f"样本均值 = {data.mean():.4f}")
print(f"t = {t_stat:.4f}, p = {p_value:.4f}")
print(f"α=0.05 下 {'拒绝' if p_value < 0.05 else '不拒绝'} H0")

# 双样本 t 检验
# H0: μ1 = μ2, H1: μ1 ≠ μ2
group1 = np.random.normal(78, 8, 20)
group2 = np.random.normal(82, 6, 25)
t_stat2, p_value2 = stats.ttest_ind(group1, group2, equal_var=False)
print(f"\n=== 双样本 t 检验 (Welch) ===")
print(f"组1: n={len(group1)}, X̄={group1.mean():.2f}, S={group1.std():.2f}")
print(f"组2: n={len(group2)}, X̄={group2.mean():.2f}, S={group2.std():.2f}")
print(f"t = {t_stat2:.4f}, p = {p_value2:.4f}")
print(f"α=0.05 下 {'拒绝' if p_value2 < 0.05 else '不拒绝'} H0")

# 配对 t 检验
before = np.array([72, 75, 68, 80, 74, 71, 77, 73])
after = np.array([74, 77, 70, 82, 73, 73, 78, 75])
t_paired, p_paired = stats.ttest_rel(before, after)
print(f"\n=== 配对 t 检验 ===")
print(f"差值均值 = {(after-before).mean():.4f}")
print(f"t = {t_paired:.4f}, p = {p_paired:.4f}")
```

### p 值的含义

```python
import numpy as np
from scipy import stats

# 模拟 p 值的分布 (当 H0 为真时, p ~ U(0,1))
np.random.seed(42)
n_sim = 10000
p_values = []
for _ in range(n_sim):
    # 从 H0 下的分布抽样
    sample = np.random.normal(0, 1, 20)
    _, p = stats.ttest_1samp(sample, 0)
    p_values.append(p)

p_values = np.array(p_values)
print("=== p 值分布 (H0 为真时) ===")
print(f"p < 0.05 的比例: {(p_values < 0.05).mean():.4f} (应≈0.05)")
print(f"p < 0.01 的比例: {(p_values < 0.01).mean():.4f} (应≈0.01)")
print(f"p 值的均值: {p_values.mean():.4f} (应≈0.5)")

# 检验力: 当 H1 为真时拒绝 H0 的概率
effect_size = 0.5
power_p_values = []
for _ in range(n_sim):
    sample = np.random.normal(effect_size, 1, 20)
    _, p = stats.ttest_1samp(sample, 0)
    power_p_values.append(p)
power = np.mean(np.array(power_p_values) < 0.05)
print(f"\n检验力 (effect size={effect_size}): {power:.4f}")
```
