点估计与区间估计

参数估计 · 最大似然估计与置信区间

一、点估计

1. 矩估计法

思想：用样本矩估计总体矩（用样本均值估计总体均值等）。

步骤：令总体的 k 阶原点矩等于样本的 k 阶原点矩，解方程组。

E(Xᵏ) = (1/n) Σ Xᵢᵏ    （k = 1, 2, ..., m，m 为待估参数个数）
例题 1：矩估计

设 X ~ U(0, θ)，样本为 2.3, 1.8, 3.5, 2.1, 2.8，求 θ 的矩估计。

E(X) = θ/2，样本均值 x̄ = (2.3+1.8+3.5+2.1+2.8)/5 = 2.5
令 E(X) = x̄ → θ/2 = 2.5 → θ̂ = 5

2. 最大似然估计法（MLE）

思想：选取使观测数据出现概率最大的参数值。

步骤：

  1. 写出似然函数：L(θ) = ∏ f(xᵢ; θ)
  2. 取对数：ln L(θ) = Σ ln f(xᵢ; θ)
  3. 对 θ 求导并令导数为 0：d(ln L)/dθ = 0
  4. 解出 θ 即为 MLE
例题 2：最大似然估计（指数分布）

设 X ~ Exp(λ)，样本观测值为 0.5, 1.2, 0.8, 1.5，求 λ 的 MLE。

似然函数：L(λ) = ∏ λe^(-λxᵢ) = λⁿ · e^(-λΣxᵢ)
对数似然：ln L = n·lnλ - λ·Σxᵢ
求导：d(ln L)/dλ = n/λ - Σxᵢ = 0
λ̂ = n/Σxᵢ = 1/x̄ = 4/(0.5+1.2+0.8+1.5) = 1

3. 估计量的评价标准

标准定义说明
无偏性E(θ̂) = θ估计量的期望等于真值
有效性D(θ̂₁) < D(θ̂₂)方差越小越好
一致性θ̂ →ᵖ θn→∞ 时依概率收敛于真值

二、区间估计（置信区间）

1. 基本概念

置信区间：以一定的置信水平（通常 95%）包含参数真值的区间。

P(θ̂_L ≤ θ ≤ θ̂_U) = 1 - α

其中 1-α 称为置信水平（如 α=0.05 时，置信水平为 95%）。

注意：置信区间是随机的，参数 θ 是固定的。95% 置信水平意味着：如果反复抽样构造区间，约 95% 的区间会包含真值。

2. 正态总体均值的置信区间

条件枢轴量置信区间
σ² 已知 Z = (X̄-μ)/(σ/√n) ~ N(0,1) X̄ ± z_{α/2} · σ/√n
σ² 未知 t = (X̄-μ)/(S/√n) ~ t(n-1) X̄ ± t_{α/2}(n-1) · S/√n

S² = (1/(n-1))Σ(Xᵢ-X̄)² 为样本方差。

例题 3：均值的置信区间

从正态总体抽取 25 个样本，x̄ = 50, S = 10。求 μ 的 95% 置信区间（σ 未知）。

σ 未知，用 t 分布：t_{0.025}(24) ≈ 2.064
置信区间 = x̄ ± t · S/√n = 50 ± 2.064 × 10/5
= 50 ± 4.128
95% CI: (45.87, 54.13)
z 分布 vs t 分布：
  • σ 已知：用 z 分布（标准正态）
  • σ 未知：用 t 分布（用 S 代替 σ，需要修正自由度）
  • t 分布比正态分布尾部更厚，n 越大 t 分布越接近正态
  • n ≥ 30 时，t 分布与正态分布差异很小

本节总结

  • 矩估计：样本矩 = 总体矩，简单但不一定最优
  • 最大似然估计（MLE）：使观测数据概率最大，具有一致性和渐近有效性
  • 评价标准：无偏性、有效性、一致性
  • 置信区间：σ 已知用 Z，σ 未知用 t 分布
  • 95% 置信区间的含义：反复抽样构造的区间中，约 95% 包含真值
