假设检验

参数估计与假设检验 · 统计推断的核心方法

一、假设检验的基本概念

1. 原假设与备择假设

  • 原假设 H₀：待检验的假设（通常为"无效果""无差异"）
  • 备择假设 H₁：与原假设对立的假设（通常为"有效果""有差异"）
类型H₀H₁检验方向
双侧检验μ = μ₀μ ≠ μ₀两侧
左侧检验μ ≥ μ₀μ < μ₀左侧
右侧检验μ ≤ μ₀μ > μ₀右侧

2. 两类错误

H₀ 为真H₀ 为假
拒绝 H₀第一类错误（α）弃真正确（检验力 1-β）
不拒绝 H₀正确第二类错误（β）取伪

显著性水平 α：控制第一类错误的概率，通常取 0.05 或 0.01。

3. 检验步骤

  1. 提出假设 H₀ 和 H₁
  2. 选择检验统计量
  3. 确定拒绝域（根据 α）
  4. 计算统计量的观测值
  5. 做出判断：统计量落入拒绝域则拒绝 H₀

二、p 值

定义与解读

p 值：在 H₀ 成立的前提下，出现当前样本（或更极端样本）的概率。

  • p < α → 拒绝 H₀（结果具有统计显著性）
  • p ≥ α → 不拒绝 H₀（证据不足以拒绝）
p 值越小 → 拒绝 H₀ 的证据越强 → 对 H₀ 越不利
"不拒绝" ≠ "接受"： 假设检验的结论是"拒绝 H₀"或"不拒绝 H₀"，而不是"接受 H₀"。不拒绝 H₀ 可能是因为样本量不够大，不能简单等同于 H₀ 为真。

三、常用检验方法

1. Z 检验（σ 已知或大样本）

Z = (X̄ - μ₀) / (σ/√n) ~ N(0,1)

2. t 检验（σ 未知，小样本正态总体）

t = (X̄ - μ₀) / (S/√n) ~ t(n-1)

3. χ² 检验（方差检验）

χ² = (n-1)S² / σ₀² ~ χ²(n-1)
例题 1：t 检验

某厂声称灯泡平均寿命为 1000 小时。抽取 16 个样本，x̄=980, S=40。在 α=0.05 下检验该说法是否成立。

H₀: μ = 1000, H₁: μ ≠ 1000（双侧检验）

t = (980-1000)/(40/√16) = -20/10 = -2
临界值：t_{0.025}(15) ≈ 2.131

|t| = 2 < 2.131，未落入拒绝域
结论：在 α=0.05 下，不拒绝 H₀，没有足够证据否定厂家的说法。
例题 2：单侧检验

某药物声称能降低血压。10 名患者服药前后血压差均值为 -5mmHg，标准差为 8mmHg。在 α=0.05 下检验药物是否有效。

H₀: μ_d ≥ 0（无效），H₁: μ_d < 0（有效，左侧检验）

t = (-5-0)/(8/√10) = -5/2.53 = -1.976
临界值：-t_{0.05}(9) ≈ -1.833

t = -1.976 < -1.833，落入拒绝域
结论：拒绝 H₀，认为该药物有显著降压效果。
假设检验与置信区间的关系：
  • 双侧检验与置信区间互补：若 μ₀ 不在 (1-α) 置信区间内，则在 α 水平下拒绝 H₀
  • 置信区间提供参数的可能范围，假设检验判断参数是否等于特定值
  • 两者使用相同的统计量和分布，只是表达方式不同

本节总结

  • 假设检验：H₀（原假设）vs H₁（备择假设）
  • 两类错误：α（弃真）、β（取伪），通常控制 α = 0.05
  • p 值：H₀ 下观测到当前或更极端结果的概率，p < α 则拒绝 H₀
  • Z 检验：σ 已知；t 检验：σ 未知；χ² 检验：方差
  • "不拒绝" ≠ "接受"，统计显著 ≠ 实际显著
