线性回归分析

参数估计与假设检验 · 回归建模与诊断

一、简单线性回归

1. 模型

Y = β₀ + β₁X + ε,    ε ~ N(0, σ²)
  • β₀：截距（x=0 时 y 的期望）
  • β₁：斜率（x 每增加1单位，y 平均变化 β₁）
  • ε：随机误差，独立同分布 N(0,σ²)

2. 最小二乘估计（OLS）

使残差平方和 Q = Σ(yᵢ - β₀ - β₁xᵢ)² 最小化：

β̂₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)² = Sxy / Sxx = Lxy / Lxx
β̂₀ = ȳ - β̂₁x̄

其中：Lxx = Σ(xᵢ-x̄)²，Lxy = Σ(xᵢ-x̄)(yᵢ-ȳ)，Lyy = Σ(yᵢ-ȳ)²

3. 回归系数的方差

Var(β̂₁) = σ² / Lxx
Var(β̂₀) = σ² [1/n + x̄²/Lxx]
σ̂² = SSE / (n-2) = Σ(yᵢ-ŷᵢ)² / (n-2)    （σ² 的无偏估计）

二、回归中的平方和分解

三种平方和

SST = SSR + SSE
名称记号公式自由度含义
总平方和SST=LyyΣ(yᵢ-ȳ)²n-1y 的总变异
回归平方和SSRΣ(ŷᵢ-ȳ)² = β̂₁²Lxx1x 可解释的变异
残差平方和SSE=SSEΣ(yᵢ-ŷᵢ)²n-2未解释的变异（误差）

三、判定系数 R²

定义与计算

R² = SSR / SST = 1 - SSE / SST

R² 的含义：回归方程解释了 y 总变异的百分比。

  • R² = 1：完美拟合，所有点在回归线上
  • R² = 0：回归线没有解释任何变异
  • 简单回归中：R² = r²（r 为 Pearson 相关系数）
  • 多重回归中：使用调整 R²（adjusted R²）
R²adj = 1 - (SSE/(n-p-1)) / (SST/(n-1))

其中 p 为自变量个数。调整 R² 惩罚变量过多。

四、回归系数的假设检验

1. 对 β₁ 的 t 检验

H₀: β₁ = 0（x 与 y 无线性关系）vs H₁: β₁ ≠ 0

t = β̂₁ / SE(β̂₁) = β̂₁ / (σ̂/√Lxx) ~ t(n-2)

若 |t| > tα/2(n-2)，拒绝 H₀，认为 x 与 y 有显著线性关系。

2. 回归方程的 F 检验

H₀: β₁ = 0 vs H₁: β₁ ≠ 0

F = (SSR/1) / (SSE/(n-2)) = MSR / MSE ~ F(1, n-2)

简单回归中 F = t²（完全等价！）。多重回归中 F 检验检验整体显著性。

例题 1：简单线性回归

5 组 (x,y) 数据：(1,2), (2,4), (3,5), (4,4), (5,5)。求回归方程和 R²。

计算中间量：
x̄=3, ȳ=4
Lxx = Σ(xᵢ-x̄)² = 4+1+0+1+4 = 10
Lxy = Σ(xᵢ-x̄)(yᵢ-ȳ) = (-1)(-2)+(0)(-2)+(0)(1)+(1)(0)+(2)(1) = 2+0+0+0+2 = 6
Lyy = Σ(yᵢ-ȳ)² = 4+0+1+0+1 = 6

β̂₁ = Lxy/Lxx = 6/10 = 0.6
β̂₀ = ȳ - β̂₁x̄ = 4 - 0.6×3 = 2.2

回归方程：ŷ = 2.2 + 0.6x

SSR = β̂₁²Lxx = 0.36×10 = 3.6
SSE = Lyy - SSR = 6-3.6 = 2.4
R² = 3.6/6 = 0.6（x 解释了 y 60% 的变异）

五、残差分析与模型诊断

1. 残差定义

eᵢ = yᵢ - ŷᵢ    （观测值 - 拟合值）

标准化残差：rᵢ = eᵢ / σ̂

2. 残差图诊断

残差图形态含义处理
随机散布在0附近模型合适无需处理
漏斗形（发散）方差不齐对 y 做变换（log、√）
弯曲趋势非线性关系加入 x² 项或做变换
残差间相关独立性假设违反时间序列方法

3. 回归假设检查清单

假设检查方法
线性（E(Y) = β₀+β₁x）散点图、残差图
独立性（εᵢ 独立）Durbin-Watson 检验
同方差性（Var(ε)=σ² 恒定）残差 vs 拟合值图
正态性（ε ~ N(0,σ²)）Q-Q图、Shapiro-Wilk检验

六、多元线性回归

模型

Y = β₀ + β₁X₁ + β₂X₂ + ... + βpXp + ε

矩阵形式：

Y = Xβ + ε,    β̂ = (X'X)⁻¹X'Y
  • F 检验：检验整体回归方程是否显著（H₀: β₁=β₂=...=βp=0）
  • t 检验：检验单个回归系数是否显著（H₀: βⱼ=0）
  • 调整 R²：考虑变量个数的拟合优度
  • 多重共线性：检查 VIF（方差膨胀因子），VIF>10 表示严重共线性
回归分析的应用步骤：
  1. 绘制散点图，初步判断关系类型
  2. 拟合回归方程（OLS）
  3. 进行假设检验（t检验、F检验）
  4. 计算 R²，评估拟合优度
  5. 残差分析，验证模型假设
  6. 必要时进行变量变换或添加多项式项

本节总结

  • 简单回归：Y=β₀+β₁X+ε，OLS 得 β̂₁=Lxy/Lxx, β̂₀=ȳ-β̂₁x̄
  • 平方和分解：SST=SSR+SSE，F=MSR/MSE
  • R²：回归解释的变异比例，简单回归中 R²=r²
  • 假设检验：t检验（单个系数）、F检验（整体方程），简单回归中 F=t²
  • 残差诊断：线性、独立、同方差、正态四个假设的检验
  • 多元回归：矩阵形式 β̂=(X'X)⁻¹X'Y，需注意多重共线性
