# 卡方检验与非参数检验

*参数估计与假设检验 · 卡方检验与秩方法*


### 1. 基本思想


检验观测数据的频数分布是否符合某个理论分布。


原假设 H₀：总体服从指定的分布 F₀(x)


备择 H₁：总体不服从 F₀(x)


### 2. 检验统计量


$$
χ² = Σ (Oᵢ - Eᵢ)² / Eᵢ ~ χ²(k-1-r)
$$


- Oᵢ：第 i 组的观测频数
- Eᵢ：第 i 组的期望频数（= n × 理论概率）
- k：分组数
- r：从数据中估计的参数个数
- 自由度 = k - 1 - r


> **Note:** 重要条件：
> 每组期望频数 Eᵢ ≥ 5。若某组 Eᵢ < 5，需合并相邻组。


### 1. 问题描述


基于 r×c 列联表，检验两个分类变量是否独立。


H₀：两个变量独立    H₁：两个变量不独立（有关联）


### 2. 检验统计量


$$
χ² = Σᵢ Σⱼ (Oᵢⱼ - Eᵢⱼ)² / Eᵢⱼ ~ χ²((r-1)(c-1))
$$


期望频数：Eᵢⱼ = (第 i 行合计 × 第 j 列合计) / 总计


自由度 = (r-1)(c-1)


### 参数 vs 非参数检验


| 方面 | 参数检验 | 非参数检验 |
| --- | --- | --- |
| 分布假设 | 需假设总体分布（如正态） | 不依赖总体分布假设 |
| 数据类型 | 定量数据（连续） | 可处理定序、分类数据 |
| 检验效力 | 高（假设满足时） | 略低（但稳健） |
| 常见方法 | t检验、F检验 | 秩检验、卡方检验 |


### 用途与步骤


用于单样本或配对样本的中位数检验（替代单样本 t 检验或配对 t 检验）。


H₀：中位数 = M₀   （或配对差值的中位数 = 0）


1. 计算 dᵢ = xᵢ - M₀（或配对差值）
2. 去掉 dᵢ=0 的观测，记剩余个数为 n
3. 对 |dᵢ| 排秩（相同取平均秩）
4. 分别计算正秩和 W⁺ 和负秩和 W⁻
5. 取 W = min(W⁺, W⁻) 为检验统计量
6. 查 Wilcoxon 符号秩检验表，W 越小越拒绝 H₀


### 用途与步骤


用于两个独立样本的位置比较（替代独立样本 t 检验）。


H₀：两总体分布相同（或中位数相等）


1. 将两组数据合并，从小到大排秩
2. 分别计算两组的秩和 R₁, R₂
3. 计算 U 统计量：

   $$
   U₁ = n₁n₂ + n₁(n₁+1)/2 - R₁
                       U₂ = n₁n₂ + n₂(n₂+1)/2 - R₂
                       U = min(U₁, U₂)
   $$
4. 查 Mann-Whitney 临界值表，U 越小越拒绝 H₀


| 方法 | 对应参数检验 | 数据类型 | 适用场景 |
| --- | --- | --- | --- |
| Wilcoxon 符号秩 | 单样本/配对 t 检验 | 配对/单样本 | 中位数检验，非正态数据 |
| Mann-Whitney U | 独立样本 t 检验 | 两独立样本 | 两组位置比较 |
| Kruskal-Wallis | 单因素 ANOVA | 多独立样本 | 多组中位数比较 |
| Friedman | 重复测量 ANOVA | 多相关样本 | 区组设计 |
| 符号检验 | 单样本 t 检验 | 配对/单样本 | 最简单的非参数检验 |
| Spearman 秩相关 | Pearson 相关 | 两变量 | 非线性单调关系 |


## Python实现

### 卡方检验与非参数检验

```python
import numpy as np
from scipy import stats

# 卡方拟合优度检验
# 骰子是否公平? 观察频数
observed = np.array([10, 8, 12, 11, 9, 10])
expected = np.array([10, 10, 10, 10, 10, 10])  # 每面 1/6
chi2_stat, p_value = stats.chisquare(observed, f_exp=expected)
print(f"=== 卡方拟合优度检验 ===")
print(f"χ² = {chi2_stat:.4f}, p = {p_value:.4f}")
print(f"α=0.05 下 {'拒绝' if p_value < 0.05 else '不拒绝'} 骰子公平的假设")

# 卡方独立性检验 (列联表)
# 性别与偏好是否独立
#        喜欢A  喜欢B  喜欢C
# 男性     30     20     10
# 女性     15     30     15
contingency = np.array([[30, 20, 10],
                         [15, 30, 15]])
chi2_ind, p_ind, dof, expected_freq = stats.chi2_contingency(contingency)
print(f"\n=== 卡方独立性检验 ===")
print(f"列联表:\n{contingency}")
print(f"期望频数:\n{np.round(expected_freq, 2)}")
print(f"χ² = {chi2_ind:.4f}, df = {dof}, p = {p_ind:.4f}")
print(f"性别与偏好 {'相关' if p_ind < 0.05 else '独立'}")

# Mann-Whitney U 检验 (非参数)
np.random.seed(42)
group1 = np.random.exponential(2, 30)
group2 = np.random.exponential(3, 35)
U_stat, p_mw = stats.mannwhitneyu(group1, group2, alternative='two-sided')
print(f"\n=== Mann-Whitney U 检验 ===")
print(f"U = {U_stat:.1f}, p = {p_mw:.4f}")

# Wilcoxon 符号秩检验 (配对非参数)
before = np.array([72, 75, 68, 80, 74, 71, 77, 73, 76, 70])
after = np.array([74, 77, 70, 82, 73, 73, 78, 75, 77, 72])
W_stat, p_wilcoxon = stats.wilcoxon(before - after)
print(f"\n=== Wilcoxon 符号秩检验 ===")
print(f"W = {W_stat:.1f}, p = {p_wilcoxon:.4f}")

# Kruskal-Wallis 检验 (多组非参数)
g1 = np.random.normal(50, 5, 20)
g2 = np.random.normal(55, 5, 20)
g3 = np.random.normal(52, 5, 20)
H_stat, p_kw = stats.kruskal(g1, g2, g3)
print(f"\n=== Kruskal-Wallis 检验 ===")
print(f"H = {H_stat:.4f}, p = {p_kw:.4f}")
```
