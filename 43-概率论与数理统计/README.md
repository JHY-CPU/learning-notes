## 学习建议

- **第01-02章** 是基础，务必扎实掌握分布及其性质
- **第03章** 是多变量分析的基础，与线性代数紧密结合
- **第04-05章** 是统计推断的核心，直接支撑机器学习理论
- **第06章** 与深度学习、贝叶斯推断紧密相关

## 与AI/ML的关联

- **概率基础** → 贝叶斯分类器、朴素贝叶斯、概率图模型
- **分布理论** → 生成模型（VAE、GAN、Diffusion）、损失函数设计
- **大数定律/CLT** → 经验风险最小化、batch normalization理论
- **参数估计** → 最大似然 = 交叉熵损失，MAP = 正则化
- **假设检验** → A/B测试、模型比较
- **随机过程** → MCMC采样、变分推断、扩散模型
