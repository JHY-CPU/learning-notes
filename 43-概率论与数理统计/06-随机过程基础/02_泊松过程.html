泊松过程

随机过程基础 · Poisson Process

一、泊松过程的定义

计数过程定义

设 N(t) 表示 [0,t] 时间内事件发生的次数。若满足以下条件，则称 {N(t), t ≥ 0} 为参数 λ 的泊松过程：

  1. N(0) = 0
  2. 独立增量：不相交时间段内的事件数相互独立
  3. 平稳增量：在长度为 t 的任意区间内，事件数服从泊松分布 P(λt)：
    P(N(s+t) - N(s) = k) = (λt)ᵏ e-λt / k!,   k = 0, 1, 2, ...

λ 称为到达率（或强度），表示单位时间内事件发生的平均次数。

等价定义（事件时间间隔）

设 T₁, T₂, T₃, ... 为相邻事件之间的时间间隔。{N(t)} 是速率为 λ 的泊松过程 ⟺ T₁, T₂, T₃, ... 是独立同分布的 Exp(λ)。

fT(t) = λe-λt,   t > 0
E(Tᵢ) = 1/λ,   Var(Tᵢ) = 1/λ²

二、基本性质

1. 均值与方差

E[N(t)] = λt,   Var[N(t)] = λt

泊松分布的特征：均值 = 方差 = λt。

2. 到达时间分布

第 k 个事件的到达时间 Sₖ = T₁+T₂+...+Tₖ 服从 Gamma 分布：

Sₖ ~ Gamma(k, λ)  （形状参数 k，速率参数 λ）
fSₖ(s) = λᵏsk-1e-λs / (k-1)!,   s > 0
  • E(Sₖ) = k/λ
  • Var(Sₖ) = k/λ²
  • 特别地，S₁ ~ Exp(λ)（首次到达时间）
例题 1：到达概率

某路口平均 10 分钟来一辆车（λ=0.1 辆/分钟）。求 30 分钟内恰好来 2 辆车的概率，以及下一辆车在 5 分钟内到达的概率。

30分钟内恰好2辆：
N(30) ~ P(λt = 0.1×30 = 3)
P(N(30)=2) = 3²e⁻³/2! = 9×0.0498/2 ≈ 0.224

下一辆车在5分钟内到达：
T ~ Exp(0.1)
P(T ≤ 5) = 1 - e-0.1×5 = 1 - e⁻⁰·⁵ ≈ 0.393

三、泊松过程的条件特性

1. 给定 N(t)=n 的到达时间分布

若在 [0,t] 内共发生了 n 个事件，则这 n 个到达时间 S₁,S₂,...,Sₙ 的联合分布等价于 [0,t] 上 n 个独立均匀分布随机变量的次序统计量。

(S₁, S₂, ..., Sₙ) | N(t)=n 的分布 = U₁, U₂, ..., Uₙ 的次序统计量，其中 Uᵢ ~ U(0,t)

直观理解：若已知 [0,t] 内来了 n 个事件，这 n 个事件的时间均匀分布在 [0,t] 中。

2. 到达时间与计数的对偶

  • N(t) ≥ k ⟺ Sₖ ≤ t（到 t 时刻至少 k 个事件发生，等价于第 k 个事件在 t 之前到达）
  • P(Sₖ ≤ t) = P(N(t) ≥ k) = Σⱼ₌ₖ∞ (λt)ʲe-λt/j!
  • Sₖ 的 CDF 可用伽马分布或泊松分布的尾概率计算

四、稀释（Thinning）与叠加（Superposition）

1. 稀释（Thinning）

将泊松过程的每个事件独立地以概率 p 标记为"类型1"，以概率 1-p 标记为"类型2"：

N(t) ~ P(λt) ⟹ N₁(t) ~ P(λpt),   N₂(t) ~ P(λ(1-p)t)

且 N₁(t) 与 N₂(t) 相互独立。

实际应用：将顾客按类型分类、将邮件按是否为垃圾邮件分类等。

2. 叠加（Superposition）

将 k 个独立的泊松过程（速率分别为 λ₁, λ₂, ..., λₖ）合并：

N₁(t) ~ P(λ₁t), ..., Nₖ(t) ~ P(λₖt) ⟹ N(t) = ΣNᵢ(t) ~ P((λ₁+...+λₖ)t)

合并后仍是泊松过程，速率 = 各过程速率之和。

特别地：每个到达事件来自第 i 个过程的概率 = λᵢ/(λ₁+...+λₖ)，且独立标记。

例题 2：稀释的应用

某网站每分钟平均访问 100 次（λ=100），其中 30% 为新用户。求 1 分钟内新用户访问数的分布，以及恰好 35 次新用户访问的概率。

新用户访问数 N₁(1) ~ P(λp·t) = P(100×0.3×1) = P(30)

P(N₁=35) = 30³⁵·e⁻³⁰/35! ≈ 0.0453

老用户访问数 N₂(1) ~ P(70)，且 N₁ 与 N₂ 独立。
例题 3：叠加与均匀性

两个独立泊松过程：过程A（λ₁=3/小时），过程B（λ₂=5/小时）。求合并后 2 小时内发生 10 次事件的概率，以及第一个到达事件来自过程A的概率。

合并过程 N(t) ~ P((3+5)t) = P(8t)

P(N(2)=10) = (16)¹⁰·e⁻¹⁶/10! ≈ 0.0341

第一个事件来自过程A的概率 = λ₁/(λ₁+λ₂) = 3/8 = 0.375

五、非齐次泊松过程

定义

当到达率随时间变化时，称为非齐次泊松过程（或非平稳泊松过程）：

P(N(t+h) - N(t) = 1) = λ(t)h + o(h)
E[N(t)] = ∫₀ᵗ λ(s) ds = Λ(t)    （累积强度函数）

Λ(t) 称为累积速率函数（或均值函数），N(t) ~ P(Λ(t))。

泊松过程 vs 马尔可夫链：
  • 泊松过程是连续时间随机过程，事件间等待时间服从指数分布
  • 泊松过程具有无记忆性（指数分布的无记忆性）
  • 泊松过程是连续时间马尔可夫链的特例（只有两个状态或计数型）
  • 事件发生的时间间隔相互独立

本节总结

  • 定义：N(0)=0，独立增量，N(t)~P(λt)
  • 到达间隔：Tᵢ ~ Exp(λ) 独立同分布，E(T)=1/λ
  • 到达时间：Sₖ ~ Gamma(k, λ)，E(Sₖ)=k/λ
  • 条件到达时间：给定 N(t)=n，到达时间等价于 [0,t] 上的均匀次序统计量
  • 稀释：独立以概率 p 标记，得到两个独立泊松过程 P(λp) 和 P(λ(1-p))
  • 叠加：独立泊松过程合并后仍为泊松过程，速率为各速率之和
  • 非齐次：速率 λ(t) 随时间变化，N(t)~P(Λ(t))
