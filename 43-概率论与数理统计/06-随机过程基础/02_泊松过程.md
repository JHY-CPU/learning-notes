# 泊松过程

*随机过程基础 · Poisson Process*


### 计数过程定义


设 N(t) 表示 [0,t] 时间内事件发生的次数。若满足以下条件，则称 {N(t), t ≥ 0} 为参数 λ 的**泊松过程**：


1. **N(0) = 0**
2. **独立增量**
   ：不相交时间段内的事件数相互独立
3. **平稳增量**
   ：在长度为 t 的任意区间内，事件数服从泊松分布 P(λt)：

   $$
   P(N(s+t) - N(s) = k) = (λt)ᵏ e-λt / k!,   k = 0, 1, 2, ...
   $$


λ 称为**到达率**（或强度），表示单位时间内事件发生的平均次数。


### 等价定义（事件时间间隔）


设 T₁, T₂, T₃, ... 为相邻事件之间的时间间隔。{N(t)} 是速率为 λ 的泊松过程 ⟺ T₁, T₂, T₃, ... 是独立同分布的 Exp(λ)。


$$
fT(t) = λe-λt,   t > 0
            E(Tᵢ) = 1/λ,   Var(Tᵢ) = 1/λ²
$$


### 1. 均值与方差


$$
E[N(t)] = λt,   Var[N(t)] = λt
$$


泊松分布的特征：均值 = 方差 = λt。


### 2. 到达时间分布


第 k 个事件的到达时间 Sₖ = T₁+T₂+...+Tₖ 服从 Gamma 分布：


$$
Sₖ ~ Gamma(k, λ)  （形状参数 k，速率参数 λ）
$$


$$
fSₖ(s) = λᵏsk-1e-λs / (k-1)!,   s > 0
$$


- E(Sₖ) = k/λ
- Var(Sₖ) = k/λ²
- 特别地，S₁ ~ Exp(λ)（首次到达时间）


### 1. 给定 N(t)=n 的到达时间分布


若在 [0,t] 内共发生了 n 个事件，则这 n 个到达时间 S₁,S₂,...,Sₙ 的联合分布等价于 [0,t] 上 n 个独立均匀分布随机变量的次序统计量。


$$
(S₁, S₂, ..., Sₙ) | N(t)=n 的分布 = U₁, U₂, ..., Uₙ 的次序统计量，其中 Uᵢ ~ U(0,t)
$$


直观理解：若已知 [0,t] 内来了 n 个事件，这 n 个事件的时间均匀分布在 [0,t] 中。


### 2. 到达时间与计数的对偶


- N(t) ≥ k ⟺ Sₖ ≤ t（到 t 时刻至少 k 个事件发生，等价于第 k 个事件在 t 之前到达）
- P(Sₖ ≤ t) = P(N(t) ≥ k) = Σⱼ₌ₖ∞ (λt)ʲe
   ^-λt^
   /j!
- Sₖ 的 CDF 可用伽马分布或泊松分布的尾概率计算


### 1. 稀释（Thinning）


将泊松过程的每个事件独立地以概率 p 标记为"类型1"，以概率 1-p 标记为"类型2"：


$$
N(t) ~ P(λt) ⟹ N₁(t) ~ P(λpt),   N₂(t) ~ P(λ(1-p)t)
$$


且 N₁(t) 与 N₂(t) **相互独立**。


实际应用：将顾客按类型分类、将邮件按是否为垃圾邮件分类等。


### 2. 叠加（Superposition）


将 k 个独立的泊松过程（速率分别为 λ₁, λ₂, ..., λₖ）合并：


$$
N₁(t) ~ P(λ₁t), ..., Nₖ(t) ~ P(λₖt) ⟹ N(t) = ΣNᵢ(t) ~ P((λ₁+...+λₖ)t)
$$


合并后仍是泊松过程，速率 = 各过程速率之和。


特别地：每个到达事件来自第 i 个过程的概率 = λᵢ/(λ₁+...+λₖ)，且独立标记。


### 定义


当到达率随时间变化时，称为**非齐次泊松过程**（或非平稳泊松过程）：


$$
P(N(t+h) - N(t) = 1) = λ(t)h + o(h)
$$


$$
E[N(t)] = ∫₀ᵗ λ(s) ds = Λ(t)    （累积强度函数）
$$


Λ(t) 称为累积速率函数（或均值函数），N(t) ~ P(Λ(t))。


## Python实现

### 泊松过程模拟

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

np.random.seed(42)
lam = 3  # 到达率

# 方法: 事件间隔 ~ Exp(λ)
T = 0
arrival_times = []
while T < 100:
    T += np.random.exponential(1/lam)
    arrival_times.append(T)
arrival_times = np.array(arrival_times)

# 统计各时间段的事件数
intervals = np.arange(0, 101, 10)
counts = np.histogram(arrival_times, bins=intervals)[0]

print(f"=== 泊松过程模拟 (λ={lam}) ===")
print(f"各10秒区间内事件数: {counts}")
print(f"事件数均值: {counts.mean():.2f} (期望 {lam*10})")
print(f"事件数方差: {counts.var():.2f} (期望 {lam*10})")

# 验证 N(t) ~ Poisson(λt)
t_values = [1, 5, 10, 20]
for t in t_values:
    n_events = np.sum(arrival_times < t)
    expected = lam * t
    print(f"N({t}) = {n_events}, E[N({t})] = {expected}")

# 验证事件间隔 ~ Exp(λ)
intervals = np.diff(arrival_times)
print(f"\n间隔均值: {intervals.mean():.4f} (期望 1/λ = {1/lam:.4f})")

# 可视化
fig, axes = plt.subplots(2, 1, figsize=(12, 6))
# 事件到达过程
axes[0].eventplot(arrival_times[:50], lineoffsets=0, linelengths=0.5)
axes[0].set_title(f'泊松过程 (λ={lam}) - 前50个事件')
axes[0].set_xlabel('时间')

# 间隔分布
axes[1].hist(intervals, bins=30, density=True, alpha=0.7, label='模拟间隔')
x = np.linspace(0, max(intervals), 100)
axes[1].plot(x, stats.expon.pdf(x, scale=1/lam), 'r-', label=f'Exp({lam})')
axes[1].set_title('事件间隔分布')
axes[1].legend()

plt.tight_layout()
plt.savefig('poisson_process.png', dpi=100)
plt.show()
```

### 泊松过程的叠加与稀释

```python
import numpy as np

np.random.seed(42)

# 叠加: 两个独立泊松过程
lam1, lam2 = 2, 3
n_sim = 10000
N1 = np.random.poisson(lam1 * 10, n_sim)  # 10秒内
N2 = np.random.poisson(lam2 * 10, n_sim)
N_total = N1 + N2

print(f"=== 泊松过程叠加 ===")
print(f"N1(t=10): 均值={N1.mean():.2f} (期望 {lam1*10})")
print(f"N2(t=10): 均值={N2.mean():.2f} (期望 {lam2*10})")
print(f"N1+N2: 均值={N_total.mean():.2f} (期望 {(lam1+lam2)*10})")
print(f"N1+N2 方差: {N_total.var():.2f} (期望 {(lam1+lam2)*10})")

# 稀释
lam_total = 5
p = 0.4
N = np.random.poisson(lam_total * 10, n_sim)
N1_thinned = np.array([np.random.binomial(n, p) for n in N])
N2_thinned = N - N1_thinned

print(f"\n=== 泊松过程稀释 ===")
print(f"N1 (稀释后): 均值={N1_thinned.mean():.2f} (期望 {lam_total*p*10})")
print(f"N2 (稀释后): 均值={N2_thinned.mean():.2f} (期望 {lam_total*(1-p)*10})")
```
