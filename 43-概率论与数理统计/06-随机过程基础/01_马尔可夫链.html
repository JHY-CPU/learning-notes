马尔可夫链

随机过程基础 · 无记忆性的随机演化

一、马尔可夫性质

1. 定义

随机过程 {Xₙ, n = 0, 1, 2, ...} 具有马尔可夫性质：未来状态只依赖于当前状态，与过去状态无关。

P(Xₙ₊₁ = j | Xₙ = i, Xₙ₋₁ = iₙ₋₁, ..., X₀ = i₀) = P(Xₙ₊₁ = j | Xₙ = i)

"给定现在，未来与过去独立"——这是马尔可夫链的核心特征。

2. 离散时间马尔可夫链

  • 状态空间：S = {0, 1, 2, ...}（有限或可数集）
  • 时间参数：n = 0, 1, 2, ...（离散）
  • 转移概率：pᵢⱼ = P(Xₙ₊₁ = j | Xₙ = i)

二、转移概率矩阵

1. 一步转移矩阵

将所有转移概率排列成矩阵 P = (pᵢⱼ)：

P = | p₀₀ p₀₁ p₀₂ ... |    | p₁₀ p₁₁ p₁₂ ... |    | p₂₀ p₂₁ p₂₂ ... |    | ... ... ... ... |

性质：每行之和为 1（pᵢⱼ ≥ 0，Σⱼ pᵢⱼ = 1），称为随机矩阵。

2. n 步转移矩阵

P⁽ⁿ⁾ = Pⁿ   （n 步转移矩阵 = 一步转移矩阵的 n 次幂）

切普曼-科尔莫戈罗夫方程：pᵢⱼ⁽ⁿ⁺ᵐ⁾ = Σₖ pᵢₖ⁽ⁿ⁾ · pₖⱼ⁽ᵐ⁾

例题 1：天气模型

晴天后第二天晴天概率 0.7，雨天后第二天雨天概率 0.5。写出转移矩阵并求两天后的天气概率。

状态：S = {晴(0), 雨(1)}

转移矩阵：P = | 0.7 0.3 |
                | 0.5 0.5 |

两步转移矩阵：
P² = | 0.7×0.7+0.3×0.5   0.7×0.3+0.3×0.5 |
     | 0.5×0.7+0.5×0.5   0.5×0.3+0.5×0.5 |
   = | 0.64 0.36 |
     | 0.60 0.40 |

若今天晴天：P(两天后晴) = 0.64，P(两天后雨) = 0.36

三、平稳分布（稳态分布）

1. 定义

概率向量 π = (π₀, π₁, ...) 满足：

πP = π    且    Σ πᵢ = 1

即 π 是矩阵 P 的左特征向量（特征值为 1）。经过足够多步转移后，状态分布趋于 π，与初始分布无关。

2. 存在唯一性条件

有限状态马尔可夫链若不可约且非周期，则存在唯一的平稳分布，且 n→∞ 时 P⁽ⁿ⁾ 的每一行都趋于 π。

  • 不可约：从任何状态可以到达任何其他状态
  • 非周期：返回某状态的时间步长的最大公约数为 1
例题 2：求平稳分布

求例题 1 中天气模型的平稳分布。

设 π = (π₁, π₂)
πP = π → π₁ = 0.7π₁ + 0.5π₂
π₂ = 0.3π₁ + 0.5π₂
π₁ + π₂ = 1

由第一式：0.3π₁ = 0.5π₂ → π₁ = (5/3)π₂
(5/3)π₂ + π₂ = 1 → π₂ = 3/8

平稳分布：π = (5/8, 3/8) ≈ (0.625, 0.375)
长期来看，约 62.5% 的天是晴天，37.5% 是雨天。

四、马尔可夫链的应用

领域应用
自然语言处理N-gram 语言模型、词性标注
搜索引擎PageRank 算法（网页排名的平稳分布）
金融信用评级迁移、市场状态模型
排队论服务系统中的顾客数变化
生物学基因序列分析、种群演化
强化学习马尔可夫决策过程（MDP）
马尔可夫链与 PageRank： Google 的 PageRank 算法将网页浏览建模为马尔可夫链：每个网页是一个状态，超链接决定了转移概率，PageRank 值就是这个马尔可夫链的平稳分布。

本节总结

  • 马尔可夫性质：无记忆性，未来只取决于现在
  • 转移矩阵 P：随机矩阵，行和为 1，pᵢⱼ 表示从 i 到 j 的概率
  • n 步转移：P⁽ⁿ⁾ = Pⁿ
  • 平稳分布：πP = π，Σπᵢ = 1，长期状态分布
  • 不可约 + 非周期 → 存在唯一平稳分布
  • 广泛应用于 NLP、搜索引擎、金融、排队论等领域
