# 马尔可夫链

*随机过程基础 · 无记忆性的随机演化*


### 1. 定义


随机过程 {Xₙ, n = 0, 1, 2, ...} 具有**马尔可夫性质**：未来状态只依赖于当前状态，与过去状态无关。


$$
P(Xₙ₊₁ = j | Xₙ = i, Xₙ₋₁ = iₙ₋₁, ..., X₀ = i₀) = P(Xₙ₊₁ = j | Xₙ = i)
$$


"给定现在，未来与过去独立"——这是马尔可夫链的核心特征。


### 2. 离散时间马尔可夫链


- **状态空间**
   ：S = {0, 1, 2, ...}（有限或可数集）
- **时间参数**
   ：n = 0, 1, 2, ...（离散）
- **转移概率**
   ：pᵢⱼ = P(Xₙ₊₁ = j | Xₙ = i)


### 1. 一步转移矩阵


将所有转移概率排列成矩阵 P = (pᵢⱼ)：

P = | p₀₀  p₀₁  p₀₂  ... |
               | p₁₀  p₁₁  p₁₂  ... |
               | p₂₀  p₂₁  p₂₂  ... |
               | ...  ...  ...  ... |

性质：每行之和为 1（pᵢⱼ ≥ 0，Σⱼ pᵢⱼ = 1），称为**随机矩阵**。


### 2. n 步转移矩阵


$$
P⁽ⁿ⁾ = Pⁿ   （n 步转移矩阵 = 一步转移矩阵的 n 次幂）
$$


切普曼-科尔莫戈罗夫方程：pᵢⱼ⁽ⁿ⁺ᵐ⁾ = Σₖ pᵢₖ⁽ⁿ⁾ · pₖⱼ⁽ᵐ⁾


### 1. 定义


概率向量 π = (π₀, π₁, ...) 满足：


$$
πP = π    且    Σ πᵢ = 1
$$


即 π 是矩阵 P 的左特征向量（特征值为 1）。经过足够多步转移后，状态分布趋于 π，与初始分布无关。


### 2. 存在唯一性条件


有限状态马尔可夫链若**不可约**且**非周期**，则存在唯一的平稳分布，且 n→∞ 时 P⁽ⁿ⁾ 的每一行都趋于 π。


- **不可约**
   ：从任何状态可以到达任何其他状态
- **非周期**
   ：返回某状态的时间步长的最大公约数为 1


| 领域 | 应用 |
| --- | --- |
| 自然语言处理 | N-gram 语言模型、词性标注 |
| 搜索引擎 | PageRank 算法（网页排名的平稳分布） |
| 金融 | 信用评级迁移、市场状态模型 |
| 排队论 | 服务系统中的顾客数变化 |
| 生物学 | 基因序列分析、种群演化 |
| 强化学习 | 马尔可夫决策过程（MDP） |


## Python实现

### 马尔可夫链模拟与平稳分布

```python
import numpy as np

# 天气模型: 晴天(0), 阴天(1), 雨天(2)
P = np.array([[0.7, 0.2, 0.1],   # 从晴天出发
              [0.3, 0.4, 0.3],   # 从阴天出发
              [0.2, 0.3, 0.5]])  # 从雨天出发

print("转移矩阵:")
print(P)

# 验证随机矩阵 (每行之和=1)
print(f"行和: {P.sum(axis=1)}")

# n 步转移矩阵
P_2 = np.linalg.matrix_power(P, 2)
P_5 = np.linalg.matrix_power(P, 5)
P_20 = np.linalg.matrix_power(P, 20)
print(f"\n2步转移矩阵:\n{np.round(P_2, 4)}")
print(f"\n20步转移矩阵:\n{np.round(P_20, 4)}")

# 平稳分布: πP = π, Σπ = 1
# 即求 P^T 的特征值1对应的左特征向量
eigenvalues, eigenvectors = np.linalg.eig(P.T)
idx = np.argmin(np.abs(eigenvalues - 1))
stationary = np.real(eigenvectors[:, idx])
stationary = stationary / stationary.sum()
print(f"\n平稳分布: π = {np.round(stationary, 4)}")

# 蒙特卡洛模拟
np.random.seed(42)
state = 0  # 从晴天开始
states = [state]
for _ in range(10000):
    state = np.random.choice(3, p=P[state])
    states.append(state)

states = np.array(states)
print(f"\n模拟10000步后的状态频率:")
for i, name in enumerate(['晴天', '阴天', '雨天']):
    freq = np.mean(states == i)
    print(f"  {name}: {freq:.4f} (平稳分布: {stationary[i]:.4f})")
```

### PageRank 算法

```python
import numpy as np

# 简化的 PageRank: 4个网页的链接关系
# 转移矩阵 (列归一化)
H = np.array([[0,   0,   0.5, 0],
              [1/3, 0,   0,   0],
              [1/3, 0.5, 0,   1],
              [1/3, 0.5, 0.5, 0]])

d = 0.85  # 阻尼因子
n = 4
G = d * H + (1 - d) / n * np.ones((n, n))  # Google 矩阵

# 迭代求平稳分布
r = np.ones(n) / n
for _ in range(100):
    r_new = G @ r
    if np.allclose(r, r_new):
        break
    r = r_new

print("PageRank 值:")
for i, val in enumerate(r):
    print(f"  网页{i+1}: {val:.4f}")
```
