# 布朗运动

*随机过程基础 · Wiener Process 与金融建模*


### 定义


随机过程 {B(t), t ≥ 0} 若满足以下条件，则称为**标准布朗运动**（或 Wiener 过程）：


1. **B(0) = 0**
2. **独立增量**
   ：对任意 0 ≤ t₁ < t₂ < ... < tₙ，增量 B(t₂)-B(t₁), ..., B(tₙ)-B(tₙ₋₁) 相互独立
3. **平稳增量**
   ：对任意 0 ≤ s < t，增量 B(t)-B(s) ~ N(0, t-s)
4. **连续路径**
   ：B(t) 是 t 的连续函数（概率为 1）


### 基本统计量


$$
E[B(t)] = 0,   Var[B(t)] = t
$$


$$
Cov[B(s), B(t)] = min(s, t)    （s, t ≥ 0）
$$


特别地，Corr[B(s), B(t)] = √(min(s,t)/max(s,t)) = √(s/t)（若 s ≤ t）。


### 1. 马尔可夫性


布朗运动是连续时间马尔可夫过程：未来行为只依赖当前状态，与过去无关。


$$
P(B(t) ≤ x | B(s) = y, 历史) = P(B(t) ≤ x | B(s) = y)
$$


### 2. 鞅性（Martingale）


布朗运动是鞅：在当前信息下，未来值的条件期望等于当前值。


$$
E[B(t) | Fs] = B(s),   对任意 t > s
$$


这意味着布朗运动"没有趋势"——向上和向下的机会均等。


### 3. 自相似性


对任意 c > 0：


$$
{B(ct), t ≥ 0} 的分布 = {√c · B(t), t ≥ 0}
$$


放大时间尺度 c 倍等价于放大空间尺度 √c 倍。


### 4. 路径性质


- **连续但处处不可微**
   ：布朗运动的路径以概率 1 处处连续但处处不可微
- **无限变差**
   ：在任意区间 [0,t] 上，路径的总变差以概率 1 为无穷大
- **二次变差**
   ：在 [0,t] 上的二次变差 = t（即 Σ[B(tᵢ₊₁)-B(tᵢ)]² → t）
- **零点集**
   ：布朗运动在任意区间内有无穷多个零点


### 定义与统计量


$$
X(t) = μt + σB(t)
$$


- μ：漂移率（drift），表示趋势方向
- σ：扩散系数（volatility），表示波动幅度
- E[X(t)] = μt
- Var[X(t)] = σ²t
- X(t) ~ N(μt, σ²t)


### 1. 定义


**几何布朗运动**（Geometric Brownian Motion）定义为：


$$
S(t) = S(0) · exp{(μ - σ²/2)t + σB(t)}
$$


等价的随机微分方程（SDE）形式：


$$
dS(t) = μS(t)dt + σS(t)dB(t)
$$


- μ：漂移率（drift），表示平均收益率
- σ：波动率（volatility），表示价格波动幅度
- S(t) > 0 始终成立（价格非负）


### 2. 统计性质


$$
E[S(t)] = S(0) · eμt
$$


$$
Var[S(t)] = S(0)² · e2μt · (eσ²t - 1)
$$


$$
对数收益：ln(S(t)/S(0)) ~ N((μ-σ²/2)t, σ²t)
$$


关键性质：对数收益服从正态分布（即收益服从对数正态分布）。


### 1. 基本假设


- 股票价格服从几何布朗运动：dS = μSdt + σSdB
- 无交易成本、无税收
- 可以无限制地卖空
- 无风险利率 r 恒定
- 股票不支付股息（基本版）


### 2. Black-Scholes 公式


欧式看涨期权价格（不支付股息）：


$$
C = S₀·N(d₁) - K·e-rT·N(d₂)
$$


$$
d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)
$$


$$
d₂ = d₁ - σ√T
$$


- S₀：股票现价
- K：执行价格（strike price）
- T：到期时间（年为单位）
- r：无风险利率
- σ：波动率
- N(·)：标准正态分布 CDF


### 3. 欧式看跌期权（Put-Call Parity）


$$
P = K·e-rT·N(-d₂) - S₀·N(-d₁)
$$


或由看涨-看跌平价关系：P = C - S₀ + K·e^-rT^
