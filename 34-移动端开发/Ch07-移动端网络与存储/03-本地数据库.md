# 本地数据库

## 一、概念说明

当应用需要存储结构化数据、支持复杂查询时，本地数据库是最佳选择。常用方案包括 SQLite、Hive 和 Isar。

```dart
// 选择指南
/*
SQLite  - 关系型数据，复杂查询
Hive    - 简单 NoSQL，高性能读写
Isar    - Flutter 专用 NoSQL，功能丰富
*/
```

## 二、Isar 数据库

```dart
// pubspec.yaml
dependencies:
  isar: ^4.0.0
  isar_flutter_libs: ^4.0.0
dev_dependencies:
  isar_generator: ^4.0.0
  build_runner: ^2.4.0

// 定义数据模型
@collection
class Task {
  Id id = Isar.autoIncrement;

  late String title;

  @enumerated
  late TaskStatus status;

  DateTime? dueDate;

  @Index()
  late DateTime createdAt;

  bool get isCompleted => status == TaskStatus.completed;
}

enum TaskStatus { todo, inProgress, completed }
```

```dart
// 数据库操作
class TaskRepository {
  late Isar _isar;

  Future<void> init() async {
    _isar = await Isar.open([TaskSchema]);
  }

  // 增
  Future<int> addTask(Task task) async {
    return await _isar.writeTxn(() => _isar.tasks.put(task));
  }

  // 查 - 全部
  Future<List<Task>> getAllTasks() async {
    return await _isar.tasks.where().sortByCreatedAtDesc().findAll();
  }

  // 查 - 条件
  Future<List<Task>> getTasksByStatus(TaskStatus status) async {
    return await _isar.tasks
        .where()
        .filter()
        .statusEqualTo(status)
        .findAll();
  }

  // 查 - 搜索
  Future<List<Task>> searchTasks(String query) async {
    return await _isar.tasks
        .where()
        .titleContains(query, caseSensitive: false)
        .findAll();
  }

  // 改
  Future<void> updateTask(Task task) async {
    await _isar.writeTxn(() => _isar.tasks.put(task));
  }

  // 删
  Future<bool> deleteTask(int id) async {
    return await _isar.writeTxn(() => _isar.tasks.delete(id));
  }

  // 监听变化
  Stream<List<Task>> watchTasks() {
    return _isar.tasks.where().watch(fireImmediately: true);
  }

  // 统计
  Future<int> getCompletedCount() async {
    return await _isar.tasks
        .where()
        .filter()
        .statusEqualTo(TaskStatus.completed)
        .count();
  }

  Future<void> close() async {
    await _isar.close();
  }
}
```

## 三、数据库迁移

```dart
// 数据库版本迁移
class DatabaseHelper {
  static const int _version = 2;

  static Future<Database> get database async {
    final path = join(await getDatabasesPath(), 'app.db');
    return openDatabase(
      path,
      version: _version,
      onCreate: (db, version) async {
        await db.execute('''
          CREATE TABLE tasks (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            title TEXT NOT NULL,
            status INTEGER DEFAULT 0,
            created_at TEXT DEFAULT CURRENT_TIMESTAMP
          )
        ''');
      },
      onUpgrade: (db, oldVersion, newVersion) async {
        if (oldVersion < 2) {
          // 版本 2: 添加 due_date 字段
          await db.execute('ALTER TABLE tasks ADD COLUMN due_date TEXT');
        }
        if (oldVersion < 3) {
          // 版本 3: 添加 priority 字段
          await db.execute('ALTER TABLE tasks ADD COLUMN priority INTEGER DEFAULT 0');
        }
      },
    );
  }
}
```

## 四、注意事项

1. **索引优化**：为频繁查询的字段创建索引
2. **事务处理**：批量操作使用事务提高性能
3. **连接管理**：全局使用单一数据库连接
4. **数据迁移**：版本升级时做好数据迁移
5. **备份恢复**：提供数据导出导入功能
