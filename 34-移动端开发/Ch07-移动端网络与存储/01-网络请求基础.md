# 移动端网络请求基础

## 一、概念说明

移动应用需要与服务器通信获取和提交数据。移动端网络请求需要处理弱网环境、超时、重试和离线缓存。

```dart
*/
```

## 二、请求封装

```dart
// Flutter Dio 封装
class HttpClient {
  static final Dio _dio = Dio(BaseOptions(
    baseUrl: 'https://api.example.com',
    connectTimeout: const Duration(seconds: 10),
    receiveTimeout: const Duration(seconds: 10),
  ));

  static void init() {
    _dio.interceptors.add(InterceptorsWrapper(
      onRequest: (options, handler) async {
        final token = await SecureStorage.read('token');
        if (token != null) {
          options.headers['Authorization'] = 'Bearer $token';
        }
        handler.next(options);
      },
      onError: (error, handler) {
        if (error.response?.statusCode == 401) {
          // 处理 token 过期
        }
        handler.next(error);
      },
    ));
  }

  static Future<Response> get(String path, {Map<String, dynamic>? params}) {
    return _dio.get(path, queryParameters: params);
  }

  static Future<Response> post(String path, {dynamic data}) {
    return _dio.post(path, data: data);
  }
}
```

## 三、错误处理

```javascript
// 统一错误处理
class ApiError {
  static String getMessage(error) {
    if (error.response) {
      switch (error.response.status) {
        case 400: return '请求参数错误';
        case 401: return '请重新登录';
        case 403: return '没有权限访问';
        case 404: return '资源不存在';
        case 500: return '服务器错误';
        default: return '请求失败';
      }
    }
    if (error.code === 'ECONNABORTED') return '请求超时';
    return '网络连接失败';
  }
}
```

## 四、网络状态检测

```dart
// 网络状态监听
class NetworkService {
  static final StreamController<bool> _controller = StreamController.broadcast();
  static Stream<bool> get onConnectivityChanged => _controller.stream;
  static bool _isConnected = true;

  static bool get isConnected => _isConnected;

  static void init() {
    Connectivity().onConnectivityChanged.listen((result) {
      _isConnected = result != ConnectivityResult.none;
      _controller.add(_isConnected);
    });
  }
}

// 使用
StreamBuilder<bool>(
  stream: NetworkService.onConnectivityChanged,
  builder: (context, snapshot) {
    final isConnected = snapshot.data ?? true;
    if (!isConnected) {
      return const Banner(message: '无网络连接', location: BannerLocation.topEnd);
    }
    return const SizedBox();
  },
)
```

## 五、注意事项

1. **HTTPS**：生产环境必须使用 HTTPS
2. **超时设置**：连接和读取超时都需要设置
3. **请求取消**：页面销毁时取消未完成的请求
4. **重试机制**：对可重试的错误自动重试
5. **离线缓存**：网络不可用时使用缓存数据
