# 移动端手势处理

## 一、概念说明

手势识别是移动端交互的核心特性，用户通过点击、滑动、捏合等手势与应用进行交互。合理的手势支持能提供自然流畅的用户体验。

```javascript
// React Native PanResponder 手势示例
import { PanResponder } from 'react-native';

const GestureDemo = () => {
  const pan = useRef(new Animated.ValueXY()).current;

  const panResponder = useRef(
    PanResponder.create({
      onStartShouldSetPanResponder: () => true,
      onPanResponderMove: Animated.event(
        [null, { dx: pan.x, dy: pan.y }],
        { useNativeDriver: false }
      ),
      onPanResponderRelease: () => {
        Animated.spring(pan, {
          toValue: { x: 0, y: 0 },
          useNativeDriver: false,
        }).start();
      },
    })
  ).current;

  return (
    <View style={styles.container}>
      <Animated.View
        style={{
          transform: [{ translateX: pan.x }, { translateY: pan.y }],
        }}
        {...panResponder.panHandlers}
      >
        <View style={styles.box}>
          <Text>拖动我</Text>
        </View>
      </Animated.View>
    </View>
  );
};
```

### 2.2 滑动手势

```javascript
// React Native 滑动检测
const SwipeDemo = () => {
  const [direction, setDirection] = useState('');

  const handleSwipe = (gestureState) => {
    const { dx, dy } = gestureState;
    const minDistance = 50;

    if (Math.abs(dx) > Math.abs(dy)) {
      if (dx > minDistance) setDirection('右滑');
      else if (dx < -minDistance) setDirection('左滑');
    } else {
      if (dy > minDistance) setDirection('下滑');
      else if (dy < -minDistance) setDirection('上滑');
    }
  };

  const panResponder = useRef(
    PanResponder.create({
      onMoveShouldSetPanResponder: (_, gestureState) => {
        return Math.abs(gestureState.dx) > 10 || Math.abs(gestureState.dy) > 10;
      },
      onPanResponderRelease: (_, gestureState) => {
        handleSwipe(gestureState);
      },
    })
  ).current;

  return (
    <View style={styles.swipeArea} {...panResponder.panHandlers}>
      <Text>滑动方向: {direction}</Text>
    </View>
  );
};
```

```dart
// Flutter 缩放手势
class ScaleDemo extends StatefulWidget {
  @override
  _ScaleDemoState createState() => _ScaleDemoState();
}

class _ScaleDemoState extends State<ScaleDemo> {
  double _scale = 1.0;
  double _previousScale = 1.0;

  @override
  Widget build(BuildContext context) {
    return GestureDetector(
      onScaleStart: (details) {
        _previousScale = _scale;
      },
      onScaleUpdate: (details) {
        setState(() {
          _scale = _previousScale * details.scale;
          _scale = _scale.clamp(0.5, 3.0);
        });
      },
      child: Transform.scale(
        scale: _scale,
        child: Container(
          width: 200,
          height: 200,
          color: Colors.orange,
          child: Center(child: Text('捏合缩放')),
        ),
      ),
    );
  }
}
```

### 2.4 轮播与滑动删除

```javascript
// 滑动删除列表项
const SwipeToDelete = () => {
  const [items, setItems] = useState(
    Array.from({ length: 10 }, (_, i) => ({ id: i, text: `项目 ${i + 1}` }))
  );

  const deleteItem = (id) => {
    setItems(items.filter(item => item.id !== id));
  };

  return (
    <FlatList
      data={items}
      keyExtractor={(item) => item.id.toString()}
      renderItem={({ item }) => (
        <Swipeable
          renderRightActions={() => (
            <TouchableOpacity
              style={styles.deleteButton}
              onPress={() => deleteItem(item.id)}
            >
              <Text style={styles.deleteText}>删除</Text>
            </TouchableOpacity>
          )}
        >
          <View style={styles.listItem}>
            <Text>{item.text}</Text>
          </View>
        </Swipeable>
      )}
    />
  );
};
```

const GestureConflictDemo = () => {

    PanResponder.create({
      onStartShouldSetPanResponder: () => false,
      onMoveShouldSetPanResponder: (_, gestureState) => {

      </View>

## 四、注意事项与常见陷阱

1. **手势响应区域**：确保可交互元素有足够的尺寸（至少 44pt/48dp），方便用户操作
2. **手势冲突**：嵌套组件的手势可能冲突，需要合理设置响应优先级
3. **误触防护**：添加手势防抖和最小距离/时间阈值，防止误触
4. **无障碍适配**：手势操作应有替代的按钮操作，方便无障碍用户
5. **手势反馈**：提供视觉或触觉反馈，让用户确认操作已被识别
