# 移动端响应式布局

## 一、概念说明

响应式布局是指应用界面能够自动适应不同屏幕尺寸和方向，保证在各种设备上都有良好的显示效果。移动端设备屏幕尺寸差异大，从 4 英寸小屏到 12 英寸平板都需要兼顾。

```dart
// Flutter 响应式布局基础
import 'package:flutter/material.dart';

class ResponsiveLayout extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return LayoutBuilder(
      builder: (context, constraints) {
        if (constraints.maxWidth > 600) {
          return _buildWideLayout();
        } else {
          return _buildNarrowLayout();
        }
      },
    );
  }

  Widget _buildWideLayout() {
    return Row(
      children: [
        Expanded(flex: 1, child: NavigationPanel()),
        Expanded(flex: 3, child: ContentPanel()),
      ],
    );
  }

  Widget _buildNarrowLayout() {
    return Column(
      children: [
        ContentPanel(),
        BottomNavBar(),
      ],
    );
  }
}
```

### 2.2 屏幕尺寸适配

```javascript
// 适配不同屏幕尺寸的工具函数
import { Dimensions, PixelRatio } from 'react-native';

const { width: SCREEN_WIDTH, height: SCREEN_HEIGHT } = Dimensions.get('window');

// 设计稿基准宽度（以 375px 为基准）
const DESIGN_WIDTH = 375;

// 将设计稿尺寸转换为实际像素
const scale = (size) => (SCREEN_WIDTH / DESIGN_WIDTH) * size;

// 适配字体大小
const normalizeFontSize = (size) => {
  const newSize = scale(size);
  return Math.round(PixelRatio.roundToNearestPixel(newSize));
};

// 使用示例
const styles = StyleSheet.create({
  container: {
    paddingHorizontal: scale(16),
    paddingVertical: scale(12),
  },
  title: {
    fontSize: normalizeFontSize(18),
    marginBottom: scale(8),
  },
  text: {
    fontSize: normalizeFontSize(14),
    lineHeight: normalizeFontSize(22),
  },
});
```

```dart
// Flutter 横竖屏适配
class OrientationDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final orientation = MediaQuery.of(context).orientation;

    return Scaffold(
      body: orientation == Orientation.landscape
          ? _buildLandscapeView()
          : _buildPortraitView(),
    );
  }

  Widget _buildLandscapeView() {
    return Row(
      children: [
        Expanded(child: SidebarWidget()),
        Expanded(flex: 3, child: ContentWidget()),
      ],
    );
  }

  Widget _buildPortraitView() {
    return Column(
      children: [
        Expanded(child: ContentWidget()),
        BottomNavBarWidget(),
      ],
    );
  }
}
```

## 三、媒体查询与断点

```dart
// Flutter 响应式断点系统
class Breakpoints {
  static const double mobile = 600;
  static const double tablet = 900;
  static const double desktop = 1200;
}

class ResponsiveBuilder extends StatelessWidget {
  final Widget mobile;
  final Widget? tablet;
  final Widget? desktop;

  const ResponsiveBuilder({
    required this.mobile,
    this.tablet,
    this.desktop,
  });

  @override
  Widget build(BuildContext context) {
    return LayoutBuilder(
      builder: (context, constraints) {
        if (constraints.maxWidth >= Breakpoints.desktop && desktop != null) {
          return desktop!;
        }
        if (constraints.maxWidth >= Breakpoints.mobile && tablet != null) {
          return tablet!;
        }
        return mobile;
      },
    );
  }
}
```

## 四、注意事项与常见陷阱

1. **不要使用固定像素值**：使用相对单位或百分比，避免在不同屏幕上布局错乱
2. **考虑极端情况**：测试小屏（SE）和大屏（iPad Pro）的显示效果
3. **图片适配**：提供多倍图资源，使用合适的图片加载策略
4. **键盘弹出处理**：输入框获取焦点时键盘弹出可能导致布局挤压，需要适配
5. **折叠屏适配**：新型折叠屏设备需要考虑屏幕折叠状态的变化
