# 移动端开发工具链

## 一、概念说明

移动端开发工具链是指从编码、调试、测试到发布整个过程中使用的工具集合。掌握高效的工具链能显著提升开发效率和代码质量。

主要包括：IDE、调试工具、版本控制、构建工具、CI/CD 和测试框架等。

```bash
# 常用开发工具安装命令
# 安装 React Native CLI
npm install -g react-native-cli

# 安装 Flutter SDK（推荐使用官方安装包）
# Windows: 下载 flutter_windows_xxx.zip
# macOS: 使用 Homebrew
brew install --cask flutter

# 安装 CocoaPods（iOS 依赖管理）
sudo gem install cocoapods

# 安装 Android Studio
# 从 developer.android.com 下载
```

### 2.4 CI/CD 工具

```yaml
# GitHub Actions 移动端 CI/CD 配置示例
name: Build and Deploy

on:
  push:
    branches: [main]

jobs:
  build-android:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-java@v3
        with:
          distribution: 'zulu'
          java-version: '11'
      - uses: subosito/flutter-action@v2
        with:
          flutter-version: '3.10.0'
      - run: flutter pub get
      - run: flutter build apk --release

  build-ios:
    runs-on: macos-latest
    steps:
      - uses: actions/checkout@v3
      - uses: subosito/flutter-action@v2
        with:
          flutter-version: '3.10.0'
      - run: flutter pub get
      - run: flutter build ios --release --no-codesign
```

## 三、测试工具

```javascript
// React Native 单元测试示例
import { render, fireEvent } from '@testing-library/react-native';
import UserScreen from '../screens/UserScreen';

describe('UserScreen', () => {
  it('should display user name', () => {
    const { getByText } = render(<UserScreen userId={1} />);
    expect(getByText('加载中...')).toBeTruthy();
  });

  it('should handle button press', () => {
    const onPress = jest.fn();
    const { getByText } = render(<Button title="提交" onPress={onPress} />);
    fireEvent.press(getByText('提交'));
    expect(onPress).toHaveBeenCalled();
  });
});
```

```dart
// Flutter 测试示例
import 'package:flutter_test/flutter_test.dart';
import 'package:my_app/widgets/counter.dart';

void main() {
  testWidgets('Counter increments smoke test', (WidgetTester tester) async {
    await tester.pumpWidget(const CounterApp());

    expect(find.text('0'), findsOneWidget);
    expect(find.text('1'), findsNothing);

    await tester.tap(find.byIcon(Icons.add));
    await tester.pump();

    expect(find.text('0'), findsNothing);
    expect(find.text('1'), findsOneWidget);
  });
}
```

## 四、注意事项与常见陷阱

1. **版本兼容性**：Node.js、Flutter SDK、Xcode 等工具版本需匹配，升级时注意检查兼容性
2. **缓存问题**：构建失败时可尝试清理缓存（`flutter clean`、`cd ios && pod deintegrate`）
3. **环境变量配置**：确保 ANDROID_HOME、JAVA_HOME 等环境变量正确配置
4. **依赖锁文件**：提交 yarn.lock 或 pubspec.lock 确保团队使用相同依赖版本
5. **模拟器性能**：iOS 模拟器只支持 macOS，Android 模拟器需要开启硬件加速（HAXM）
