# 移动端列表与滚动

## 一、概念说明

列表是移动端最常用的数据展示方式。由于移动设备屏幕空间有限，高效地渲染长列表并支持流畅滚动是移动开发的关键技术点。

```javascript
// React Native FlatList 基础使用
import { FlatList, Text, View } from 'react-native';

const SimpleList = () => {
  const data = Array.from({ length: 100 }, (_, i) => ({
    id: i.toString(),
    title: `项目 ${i + 1}`,
  }));

  const renderItem = ({ item }) => (
    <View style={styles.item}>
      <Text>{item.title}</Text>
    </View>
  );

  return (
    <FlatList
      data={data}
      renderItem={renderItem}
      keyExtractor={(item) => item.id}
      ItemSeparatorComponent={() => <View style={styles.separator} />}
    />
  );
};
```

## 二、列表组件详解

### 2.1 FlatList 高级用法

```javascript
// FlatList 完整功能示例
const AdvancedFlatList = () => {
  const [data, setData] = useState([]);
  const [refreshing, setRefreshing] = useState(false);
  const [loading, setLoading] = useState(false);
  const page = useRef(1);

  // 下拉刷新
  const onRefresh = async () => {
    setRefreshing(true);
    const newData = await fetchData(1);
    setData(newData);
    page.current = 1;
    setRefreshing(false);
  };

  // 上拉加载更多
  const onEndReached = async () => {
    if (loading) return;
    setLoading(true);
    page.current += 1;
    const moreData = await fetchData(page.current);
    setData([...data, ...moreData]);
    setLoading(false);
  };

  // 吸顶标题
  const renderSectionHeader = ({ section }) => (
    <View style={styles.sectionHeader}>
      <Text style={styles.sectionTitle}>{section.title}</Text>
    </View>
  );

  return (
    <FlatList
      data={data}
      renderItem={({ item }) => <ListItem item={item} />}
      keyExtractor={(item) => item.id}
      // 刷新功能
      refreshing={refreshing}
      onRefresh={onRefresh}
      // 分页加载
      onEndReached={onEndReached}
      onEndReachedThreshold={0.5}
      // 性能优化
      initialNumToRender={10}
      maxToRenderPerBatch={10}
      windowSize={5}
      // 头部和尾部
      ListHeaderComponent={<Header />}
      ListFooterComponent={loading ? <ActivityIndicator /> : null}
      ListEmptyComponent={<EmptyView />}
    />
  );
};
```

### 2.3 下拉刷新与上拉加载

```dart
// Flutter RefreshIndicator + ScrollController
class RefreshLoadMoreDemo extends StatefulWidget {
  @override
  _RefreshLoadMoreDemoState createState() => _RefreshLoadMoreDemoState();
}

class _RefreshLoadMoreDemoState extends State<RefreshLoadMoreDemo> {
  List<String> _items = [];
  bool _isLoading = false;
  final ScrollController _scrollController = ScrollController();

  @override
  void initState() {
    super.initState();
    _loadData();
    _scrollController.addListener(() {
      if (_scrollController.position.pixels >=
              _scrollController.position.maxScrollExtent - 200 &&
          !_isLoading) {
        _loadMore();
      }
    });
  }

  Future<void> _loadData() async {
    await Future.delayed(Duration(seconds: 1));
    setState(() {
      _items = List.generate(20, (i) => '项目 ${i + 1}');
    });
  }

  Future<void> _loadMore() async {
    setState(() => _isLoading = true);
    await Future.delayed(Duration(seconds: 1));
    setState(() {
      _items.addAll(
        List.generate(10, (i) => '项目 ${_items.length + i + 1}'),
      );
      _isLoading = false;
    });
  }

  @override
  Widget build(BuildContext context) {
    return RefreshIndicator(
      onRefresh: () async {
        await _loadData();
      },
      child: ListView.builder(
        controller: _scrollController,
        itemCount: _items.length + 1,
        itemBuilder: (context, index) {
          if (index == _items.length) {
            return _isLoading
                ? Center(child: CircularProgressIndicator())
                : SizedBox.shrink();
          }
          return ListTile(title: Text(_items[index]));
        },
      ),
    );
  }
}
```

## 三、性能优化

```javascript
// 列表性能优化技巧
const OptimizedList = () => {
  // 使用 useMemo 缓存数据
  const processedData = useMemo(() => {
    return rawData.map(item => ({
      ...item,
      formatted: formatData(item),
    }));
  }, [rawData]);

  // 使用 useCallback 缓存回调
  const renderItem = useCallback(({ item }) => (
    <MemoizedListItem item={item} onPress={handlePress} />
  ), [handlePress]);

  return (
    <FlatList
      data={processedData}
      renderItem={renderItem}
      keyExtractor={keyExtractor}
      getItemLayout={(data, index) => ({
        length: ITEM_HEIGHT,
        offset: ITEM_HEIGHT * index,
        index,
      })}
      removeClippedSubviews={true}
      initialNumToRender={10}
      maxToRenderPerBatch={5}
      windowSize={5}
    />
  );
};

// 使用 React.memo 优化列表项
const ListItem = React.memo(({ item, onPress }) => (
  <TouchableOpacity onPress={() => onPress(item.id)}>
    <Text>{item.title}</Text>
    <Text>{item.subtitle}</Text>
  </TouchableOpacity>
));
```

## 四、注意事项与常见陷阱

1. **keyExtractor 必须唯一且稳定**：不要使用数组索引作为 key，否则可能导致渲染问题
2. **避免内联函数**：renderItem 中使用内联函数会导致不必要的重渲染
3. **设置 getItemLayout**：固定高度的列表项应提供此属性，提升滚动性能
4. **windowSize 调优**：过大的 windowSize 会占用更多内存，过小可能导致滚动白屏
5. **复杂列表项考虑虚拟化**：对于包含图片、视频的复杂列表项，考虑使用 FlashList 等高性能方案
