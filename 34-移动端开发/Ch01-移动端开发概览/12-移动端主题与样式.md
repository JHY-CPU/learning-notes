# 移动端主题与样式

## 一、概念说明

主题与样式管理是确保应用视觉一致性的关键。良好的主题系统支持亮色/暗色模式切换、自定义配色方案和统一的样式规范。

```dart
// Flutter 主题定义
import 'package:flutter/material.dart';

class AppTheme {
  static ThemeData lightTheme = ThemeData(
    brightness: Brightness.light,
    primarySwatch: Colors.blue,
    scaffoldBackgroundColor: Colors.white,
    appBarTheme: AppBarTheme(
      elevation: 0,
      backgroundColor: Colors.blue,
      foregroundColor: Colors.white,
    ),
    textTheme: TextTheme(
      headlineLarge: TextStyle(fontSize: 28, fontWeight: FontWeight.bold),
      bodyLarge: TextStyle(fontSize: 16, height: 1.5),
      bodyMedium: TextStyle(fontSize: 14, color: Colors.grey[600]),
    ),
    elevatedButtonTheme: ElevatedButtonThemeData(
      style: ElevatedButton.styleFrom(
        padding: EdgeInsets.symmetric(horizontal: 24, vertical: 12),
        shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(8)),
      ),
    ),
  );

  static ThemeData darkTheme = ThemeData(
    brightness: Brightness.dark,
    primarySwatch: Colors.blue,
    scaffoldBackgroundColor: Color(0xFF121212),
    appBarTheme: AppBarTheme(
      backgroundColor: Color(0xFF1E1E1E),
    ),
  );
}
```

### 2.2 主题切换实现

```javascript
// React Native 主题切换
import { useColorScheme } from 'react-native';
import { createContext, useContext, useState } from 'react';

const ThemeContext = createContext();

const lightColors = {
  background: '#ffffff',
  surface: '#f5f5f5',
  text: '#333333',
  textSecondary: '#666666',
  primary: '#3498db',
  accent: '#e74c3c',
  border: '#e0e0e0',
};

const darkColors = {
  background: '#121212',
  surface: '#1e1e1e',
  text: '#ffffff',
  textSecondary: '#aaaaaa',
  primary: '#5dade2',
  accent: '#ff6b6b',
  border: '#333333',
};

export const ThemeProvider = ({ children }) => {
  const systemScheme = useColorScheme();
  const [theme, setTheme] = useState(systemScheme || 'light');

  const colors = theme === 'dark' ? darkColors : lightColors;

  const toggleTheme = () => {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  };

  return (
    <ThemeContext.Provider value={{ theme, colors, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
};

// 使用主题
const ThemedComponent = () => {
  const { theme, colors, toggleTheme } = useContext(ThemeContext);

  return (
    <View style={{ backgroundColor: colors.background }}>
      <Text style={{ color: colors.text }}>当前主题: {theme}</Text>
      <TouchableOpacity onPress={toggleTheme}>
        <Text style={{ color: colors.primary }}>切换主题</Text>
      </TouchableOpacity>
    </View>
  );
};
```

## 三、样式最佳实践

```javascript
// 样式组合工具函数
const combineStyles = (...styles) => {
  return styles.filter(Boolean).reduce((acc, style) => {
    if (Array.isArray(style)) {
      return { ...acc, ...combineStyles(...style) };
    }
    return { ...acc, ...style };
  }, {});
};

// 条件样式
const getCardStyle = (variant, theme) => {
  const baseStyle = { borderRadius: 8, padding: 16 };
  const variants = {
    elevated: { ...baseStyle, ...theme.shadows.md, backgroundColor: '#fff' },
    outlined: { ...baseStyle, borderWidth: 1, borderColor: theme.colors.border },
    filled: { ...baseStyle, backgroundColor: theme.colors.surface },
  };
  return variants[variant] || variants.elevated;
};
```

## 四、注意事项与常见陷阱

1. **避免内联样式滥用**：频繁使用内联样式会影响性能，优先使用 StyleSheet
2. **暗色模式不只是反转颜色**：需要单独设计，确保对比度和可读性
3. **字体大小可访问性**：不要使用固定字体大小，支持系统的字体缩放设置
4. **颜色对比度**：确保文本和背景的对比度符合 WCAG 标准
5. **平台特定样式**：iOS 和 Android 的阴影、字体渲染有差异，需要针对性处理
