# 移动端 UI 框架概览

## 一、概念说明

移动端 UI 框架提供了预构建的界面组件和设计系统，帮助开发者快速构建美观、一致的用户界面。主流框架包括平台原生组件和跨平台 UI 库。

```dart
// Flutter Material Design 示例
import 'package:flutter/material.dart';

  @override

      floatingActionButton: FloatingActionButton(
        onPressed: () {},

    );
  }
}
```

## 二、主流 UI 框架

### 2.1 React Native 组件库

```javascript
// React Native 内置组件
import {
  View,
  Text,
  TextInput,
  TouchableOpacity,
  FlatList,
  Image,
  ScrollView,
  StyleSheet,
  StatusBar,
  SafeAreaView
} from 'react-native';

const ComponentDemo = () => {
  const [text, setText] = useState('');
  const data = [
    { id: '1', title: '列表项一' },
    { id: '2', title: '列表项二' },
    { id: '3', title: '列表项三' },
  ];

  return (
    <SafeAreaView style={styles.container}>
      <StatusBar barStyle="dark-content" />
      <ScrollView>
        {/* 文本输入 */}
        <TextInput
          style={styles.input}
          placeholder="请输入内容"
          value={text}
          onChangeText={setText}
        />

        {/* 图片 */}
        <Image
          source={{ uri: 'https://example.com/image.png' }}
          style={styles.image}
          resizeMode="cover"
        />

        {/* 列表 */}
        <FlatList
          data={data}
          keyExtractor={(item) => item.id}
          renderItem={({ item }) => (
            <TouchableOpacity style={styles.listItem}>
              <Text>{item.title}</Text>
            </TouchableOpacity>
          )}
        />
      </ScrollView>
    </SafeAreaView>
  );
};

const styles = StyleSheet.create({
  container: { flex: 1, backgroundColor: '#fff' },
  input: { borderWidth: 1, borderColor: '#ddd', padding: 12, margin: 16, borderRadius: 8 },
  image: { width: 200, height: 200, alignSelf: 'center' },
  listItem: { padding: 16, borderBottomWidth: 1, borderBottomColor: '#eee' },
});
```

## 三、自定义组件封装

```dart
// Flutter 自定义按钮组件
class CustomButton extends StatelessWidget {
  final String text;
  final VoidCallback onPressed;
  final Color? backgroundColor;
  final bool isLoading;

  const CustomButton({
    required this.text,
    required this.onPressed,
    this.backgroundColor,
    this.isLoading = false,
  });

  @override
  Widget build(BuildContext context) {
    return SizedBox(
      width: double.infinity,
      height: 48,
      child: ElevatedButton(
        onPressed: isLoading ? null : onPressed,
        style: ElevatedButton.styleFrom(
          backgroundColor: backgroundColor ?? Theme.of(context).primaryColor,
          shape: RoundedRectangleBorder(
            borderRadius: BorderRadius.circular(8),
          ),
        ),
        child: isLoading
            ? SizedBox(
                width: 24,
                height: 24,
                child: CircularProgressIndicator(
                  strokeWidth: 2,
                  valueColor: AlwaysStoppedAnimation<Color>(Colors.white),
                ),
              )
            : Text(text, style: TextStyle(fontSize: 16)),
      ),
    );
  }
}
```

## 四、注意事项与常见陷阱

1. **平台差异处理**：iOS 和 Android 的交互习惯不同，如 iOS 滑动返回、Android 返回键等
2. **图片资源适配**：需提供 @1x、@2x、@3x 多倍图，Flutter 使用不同的 DPI 配置
3. **字体和间距**：移动端字体应比 Web 端大，最小可点击区域建议 44pt/48dp
4. **深色模式适配**：现代应用需要同时支持亮色和暗色主题
5. **无障碍支持**：为组件添加语义标签，支持屏幕阅读器访问
