# 移动端组件设计

## 一、概念说明

组件设计是构建一致、可复用 UI 的基础。每个组件应有明确的职责、统一的接口和灵活的配置选项。

```dart
// 设计组件的基本原则
/*
1. 单一职责 - 每个组件只做一件事
2. 可配置 - 通过 props/参数控制外观和行为
3. 可组合 - 小组件组合成复杂界面
4. 无障碍 - 支持屏幕阅读器
*/
```

## 二、按钮组件设计

```dart
// Flutter 自定义按钮组件
enum ButtonVariant { primary, secondary, outline, text }
enum ButtonSize { small, medium, large }

class AppButton extends StatelessWidget {
  final String label;
  final VoidCallback? onPressed;
  final ButtonVariant variant;
  final ButtonSize size;
  final bool loading;
  final IconData? icon;

  const AppButton({
    super.key,
    required this.label,
    this.onPressed,
    this.variant = ButtonVariant.primary,
    this.size = ButtonSize.medium,
    this.loading = false,
    this.icon,
  });

  @override
  Widget build(BuildContext context) {
    final style = _getStyle(context);
    final child = loading
        ? SizedBox(
            width: 20,
            height: 20,
            child: CircularProgressIndicator(
              strokeWidth: 2,
              color: variant == ButtonVariant.primary ? Colors.white : Colors.blue,
            ),
          )
        : Row(
            mainAxisSize: MainAxisSize.min,
            children: [
              if (icon != null) ...[
                Icon(icon, size: _iconSize),
                const SizedBox(width: 8),
              ],
              Text(label),
            ],
          );

    return SizedBox(
      height: _height,
      child: variant == ButtonVariant.text
          ? TextButton(onPressed: loading ? null : onPressed, style: style, child: child)
          : variant == ButtonVariant.outline
              ? OutlinedButton(onPressed: loading ? null : onPressed, style: style, child: child)
              : ElevatedButton(onPressed: loading ? null : onPressed, style: style, child: child),
    );
  }

  double get _height => switch (size) {
        ButtonSize.small => 32,
        ButtonSize.medium => 44,
        ButtonSize.large => 52,
      };

  double get _iconSize => switch (size) {
        ButtonSize.small => 16,
        ButtonSize.medium => 20,
        ButtonSize.large => 24,
      };

  ButtonStyle _getStyle(BuildContext context) { /* ... */ }
}
```

## 三、卡片组件设计

```dart
class AppCard extends StatelessWidget {
  final Widget child;
  final EdgeInsetsGeometry? padding;
  final VoidCallback? onTap;
  final double elevation;

  const AppCard({
    super.key,
    required this.child,
    this.padding,
    this.onTap,
    this.elevation = 2,
  });

  @override
  Widget build(BuildContext context) {
    return Card(
      elevation: elevation,
      shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(12)),
      child: InkWell(
        onTap: onTap,
        borderRadius: BorderRadius.circular(12),
        child: Padding(
          padding: padding ?? const EdgeInsets.all(16),
          child: child,
        ),
      ),
    );
  }
}
```

## 四、输入框组件

```dart
class AppTextField extends StatelessWidget {
  final String? label;
  final String? hint;
  final String? error;
  final TextEditingController? controller;
  final bool obscureText;
  final TextInputType? keyboardType;
  final Widget? prefix;
  final Widget? suffix;
  final ValueChanged<String>? onChanged;

  const AppTextField({
    super.key,
    this.label,
    this.hint,
    this.error,
    this.controller,
    this.obscureText = false,
    this.keyboardType,
    this.prefix,
    this.suffix,
    this.onChanged,
  });

  @override
  Widget build(BuildContext context) {
    return Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        if (label != null) ...[
          Text(label!, style: Theme.of(context).textTheme.bodyMedium),
          const SizedBox(height: 8),
        ],
        TextField(
          controller: controller,
          obscureText: obscureText,
          keyboardType: keyboardType,
          onChanged: onChanged,
          decoration: InputDecoration(
            hintText: hint,
            errorText: error,
            prefixIcon: prefix,
            suffixIcon: suffix,
            border: OutlineInputBorder(borderRadius: BorderRadius.circular(8)),
            contentPadding: const EdgeInsets.symmetric(horizontal: 16, vertical: 12),
          ),
        ),
      ],
    );
  }
}
```

## 五、注意事项

1. **组件文档**：每个组件应有清晰的使用文档和示例
2. **默认值**：提供合理的默认值，减少使用时的配置
3. **主题适配**：组件应使用主题中的颜色和字体，而非硬编码
4. **动画过渡**：状态变化应有平滑的过渡动画
5. **边界情况**：处理空值、超长文本、极端尺寸等边界情况
