# 移动端表单设计

## 一、概念说明

移动端表单设计需要特别考虑输入效率、屏幕空间和键盘交互。好的表单设计能减少用户输入负担，提高完成率。

```dart
// 表单设计原则
/*
1. 减少输入 - 使用选择器替代输入框
2. 分步填写 - 长表单拆分为多个步骤
3. 即时验证 - 输入时实时反馈
4. 键盘适配 - 自动管理键盘遮挡
*/
```

## 二、表单布局

```dart
class RegistrationForm extends StatefulWidget {
  const RegistrationForm({super.key});

  @override
  State<RegistrationForm> createState() => _RegistrationFormState();
}

class _RegistrationFormState extends State<RegistrationForm> {
  final _formKey = GlobalKey<FormState>();

  @override
  Widget build(BuildContext context) {
    return SingleChildScrollView(
      padding: const EdgeInsets.all(16),
      child: Form(
        key: _formKey,
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // 分组标题
            const Text('基本信息', style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold)),
            const SizedBox(height: 16),

            // 姓名输入
            TextFormField(
              decoration: const InputDecoration(
                labelText: '姓名',
                prefixIcon: Icon(Icons.person),
                border: OutlineInputBorder(),
              ),
              validator: (v) => v?.isEmpty == true ? '请输入姓名' : null,
            ),
            const SizedBox(height: 16),

            // 手机号输入
            TextFormField(
              decoration: const InputDecoration(
                labelText: '手机号',
                prefixIcon: Icon(Icons.phone),
                border: OutlineInputBorder(),
              ),
              keyboardType: TextInputType.phone,
              maxLength: 11,
              validator: (v) {
                if (v?.isEmpty == true) return '请输入手机号';
                if (!RegExp(r'^1\d{10}$').hasMatch(v!)) return '手机号格式不正确';
                return null;
              },
            ),
            const SizedBox(height: 16),

            // 选择器替代输入
            DropdownButtonFormField<String>(
              decoration: const InputDecoration(
                labelText: '所在城市',
                prefixIcon: Icon(Icons.location_on),
                border: OutlineInputBorder(),
              ),
              items: ['北京', '上海', '广州', '深圳']
                  .map((c) => DropdownMenuItem(value: c, child: Text(c)))
                  .toList(),
              onChanged: (v) {},
              validator: (v) => v == null ? '请选择城市' : null,
            ),
            const SizedBox(height: 24),

            // 提交按钮
            SizedBox(
              width: double.infinity,
              height: 48,
              child: ElevatedButton(
                onPressed: () {
                  if (_formKey.currentState!.validate()) {
                    // 提交表单
                  }
                },
                child: const Text('提交', style: TextStyle(fontSize: 16)),
              ),
            ),
          ],
        ),
      ),
    );
  }
}
```

## 三、输入优化

```dart
// 自动聚焦下一个输入框
class AutoFocusForm extends StatefulWidget {
  const AutoFocusForm({super.key});

  @override
  State<AutoFocusForm> createState() => _AutoFocusFormState();
}

class _AutoFocusFormState extends State<AutoFocusForm> {
  final _focusNodes = List.generate(3, (_) => FocusNode());
  final _controllers = List.generate(3, (_) => TextEditingController());

  @override
  void dispose() {
    for (var node in _focusNodes) node.dispose();
    for (var c in _controllers) c.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        TextField(
          focusNode: _focusNodes[0],
          controller: _controllers[0],
          decoration: const InputDecoration(labelText: '验证码第1位'),
          keyboardType: TextInputType.number,
          maxLength: 1,
          onChanged: (v) {
            if (v.isNotEmpty) _focusNodes[1].requestFocus();
          },
        ),
        TextField(
          focusNode: _focusNodes[1],
          controller: _controllers[1],
          decoration: const InputDecoration(labelText: '验证码第2位'),
          keyboardType: TextInputType.number,
          maxLength: 1,
          onChanged: (v) {
            if (v.isNotEmpty) _focusNodes[2].requestFocus();
          },
        ),
        TextField(
          focusNode: _focusNodes[2],
          controller: _controllers[2],
          decoration: const InputDecoration(labelText: '验证码第3位'),
          keyboardType: TextInputType.number,
          maxLength: 1,
          onChanged: (v) {
            if (v.isNotEmpty) _focusNodes[2].unfocus();
          },
        ),
      ],
    );
  }
}
```

## 四、注意事项

1. **键盘类型选择**：根据输入内容选择正确的键盘类型
2. **标签位置**：输入框上方放置标签，比 placeholder 更清晰
3. **错误提示**：错误信息紧跟在对应输入框下方
4. **必填标识**：必填字段使用 * 或"必填"标签标识
5. **提交按钮位置**：放在键盘弹出后仍可见的位置
