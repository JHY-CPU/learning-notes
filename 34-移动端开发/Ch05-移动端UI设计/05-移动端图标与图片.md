# 移动端图标与图片

## 一、概念说明

图标和图片是移动端 UI 的重要组成部分。图标用于导航和功能指示，图片用于内容展示和视觉增强。

```dart
// Flutter 图标使用
class IconDemo extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Row(
      mainAxisAlignment: MainAxisAlignment.spaceAround,
      children: [
        // Material Icons
        const Icon(Icons.home, size: 32, color: Colors.blue),
        const Icon(Icons.search, size: 32),
        const Icon(Icons.person, size: 32),

        // 自定义图标
        Image.asset('assets/icons/custom.png', width: 32, height: 32),

        // 网络图标
        Image.network('https://example.com/icon.png', width: 32),
      ],
    );
  }
}
```

## 二、图片处理

```dart
// 图片加载与缓存
class AppImage extends StatelessWidget {
  final String url;
  final double? width;
  final double? height;
  final BoxFit fit;
  final Widget? placeholder;
  final Widget? errorWidget;

  const AppImage({
    super.key,
    required this.url,
    this.width,
    this.height,
    this.fit = BoxFit.cover,
    this.placeholder,
    this.errorWidget,
  });

  @override
  Widget build(BuildContext context) {
    return Image.network(
      url,
      width: width,
      height: height,
      fit: fit,
      loadingBuilder: (context, child, loadingProgress) {
        if (loadingProgress == null) return child;
        return placeholder ??
            Center(
              child: CircularProgressIndicator(
                value: loadingProgress.expectedTotalBytes != null
                    ? loadingProgress.cumulativeBytesLoaded /
                        loadingProgress.expectedTotalBytes!
                    : null,
              ),
            );
      },
      errorBuilder: (context, error, stackTrace) {
        return errorWidget ??
            Container(
              width: width,
              height: height,
              color: Colors.grey[200],
              child: const Icon(Icons.broken_image, color: Colors.grey),
            );
      },
    );
  }
}

// 圆形头像
class Avatar extends StatelessWidget {
  final String? imageUrl;
  final double size;
  final String? name;

  const Avatar({super.key, this.imageUrl, this.size = 48, this.name});

  @override
  Widget build(BuildContext context) {
    return CircleAvatar(
      radius: size / 2,
      backgroundColor: Colors.blue,
      backgroundImage: imageUrl != null ? NetworkImage(imageUrl!) : null,
      child: imageUrl == null && name != null
          ? Text(
              name!.substring(0, 1).toUpperCase(),
              style: TextStyle(fontSize: size * 0.4, color: Colors.white),
            )
          : null,
    );
  }
}
```

## 三、图片适配

```javascript
// React Native 图片适配
import FastImage from 'react-native-fast-image';
import { Image, Platform, PixelRatio } from 'react-native';

// 获取合适分辨率的图片
const getOptimizedImageUrl = (url, width, height) => {
  const pixelRatio = PixelRatio.get();
  const optimizedWidth = Math.round(width * pixelRatio);
  const optimizedHeight = Math.round(height * pixelRatio);
  return `${url}?w=${optimizedWidth}&h=${optimizedHeight}&fit=crop`;
};

// 响应式图片组件
const ResponsiveImage = ({ uri, aspectRatio = 1 }) => (
  <FastImage
    source={{ uri, priority: FastImage.priority.normal }}
    style={{ width: '100%', aspectRatio }}
    resizeMode={FastImage.resizeMode.cover}
  />
);
```

## 四、注意事项

1. **图片格式**：优先使用 WebP（体积小、支持透明），备选 PNG/JPEG
2. **图片压缩**：上传前压缩，展示时使用合适尺寸
3. **占位图**：使用骨架屏或渐变色占位，避免空白闪烁
4. **懒加载**：列表中的图片使用懒加载，减少初始加载时间
5. **内存管理**：大图片及时释放，避免内存溢出
