# 移动端字体设计

## 一、概念说明

字体设计直接影响移动端应用的可读性和视觉效果。移动端字体需要考虑屏幕尺寸、阅读距离和用户的阅读习惯。

```dart
// 字体系统定义
class AppTypography {
  // 大标题
  static const displayLarge = TextStyle(
    fontSize: 32,
    fontWeight: FontWeight.bold,
    height: 1.2,
    letterSpacing: -0.5,
  );

  // 标题
  static const headlineMedium = TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    height: 1.3,
  );

  // 正文
  static const bodyLarge = TextStyle(
    fontSize: 16,
    fontWeight: FontWeight.normal,
    height: 1.6,
  );

  // 小字
  static const labelSmall = TextStyle(
    fontSize: 12,
    fontWeight: FontWeight.w500,
    height: 1.4,
    letterSpacing: 0.5,
  );
}
```

## 二、字体配置

```dart
// 自定义字体配置
class AppFonts {
  static const primary = 'PingFang SC';      // iOS 中文字体
  static const android = 'Noto Sans SC';     // Android 中文字体
  static const monospace = 'Menlo';          // 等宽字体

  static String get platform {
    if (Platform.isIOS) return primary;
    return android;
  }
}

// 在 ThemeData 中配置
ThemeData(
  fontFamily: AppFonts.platform,
  textTheme: const TextTheme(
    headlineLarge: TextStyle(fontSize: 32, fontWeight: FontWeight.bold),
    bodyLarge: TextStyle(fontSize: 16, height: 1.6),
    bodyMedium: TextStyle(fontSize: 14, height: 1.5),
    labelSmall: TextStyle(fontSize: 12, color: Colors.grey),
  ),
)
```

## 三、响应式字体

```dart
// 根据屏幕宽度调整字体大小
double responsiveFontSize(BuildContext context, double baseSize) {
  final screenWidth = MediaQuery.of(context).size.width;
  // 以 375px 为基准
  return baseSize * (screenWidth / 375).clamp(0.8, 1.2);
}

// 使用
Text(
  '响应式文字',
  style: TextStyle(
    fontSize: responsiveFontSize(context, 16),
  ),
)
```

## 四、注意事项

1. **最小字号**：正文不小于 14sp，辅助文字不小于 10sp
2. **行高设置**：正文行高 1.5-1.8，标题行高 1.2-1.4
3. **字体加载**：自定义字体增加包体积，权衡使用
4. **系统字体缩放**：支持系统的字体大小设置
5. **对比度**：文字与背景对比度至少 4.5:1
