# 内存优化

## 一、概念说明

移动端内存有限（通常 2-6GB），内存泄漏和过度使用会导致应用崩溃或被系统杀掉。

```dart
// 内存问题类型
/*
1. 内存泄漏 - 对象不再使用但未被释放
2. 内存抖动 - 频繁分配释放导致 GC 频繁
3. 内存溢出 - 使用超出可用内存
*/
```

## 二、内存泄漏防范

```dart
// 常见泄漏场景及防范
class _MyState extends State<MyWidget> {
  // 1. 控制器未释放
  late AnimationController _controller;
  late ScrollController _scrollController;

  // 2. 订阅未取消
  StreamSubscription? _subscription;
  Timer? _timer;

  // 3. 监听器未移除
  late FocusNode _focusNode;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(vsync: this);
    _scrollController = ScrollController();
    _focusNode = FocusNode();

    _subscription = someStream.listen((data) {});
    _timer = Timer.periodic(const Duration(seconds: 1), (_) {});
  }

  @override
  void dispose() {
    // 必须清理所有资源
    _controller.dispose();
    _scrollController.dispose();
    _focusNode.dispose();
    _subscription?.cancel();
    _timer?.cancel();
    super.dispose();
  }
}
```

## 三、图片内存优化

```dart
// 图片内存管理
class MemoryEfficientImage extends StatelessWidget {
  final String url;
  final double size;

  const MemoryEfficientImage({
    super.key,
    required this.url,
    this.size = 200,
  });

  @override
  Widget build(BuildContext context) {
    final dpr = MediaQuery.of(context).devicePixelRatio;
    return Image.network(
      url,
      // 限制解码尺寸，减少内存占用
      cacheWidth: (size * dpr).toInt(),
      cacheHeight: (size * dpr).toInt(),
      // 使用中等质量
      filterQuality: FilterQuality.medium,
      frameBuilder: (context, child, frame, wasSynchronouslyLoaded) {
        if (wasSynchronouslyLoaded) return child;
        return AnimatedOpacity(
          opacity: frame == null ? 0 : 1,
          duration: const Duration(milliseconds: 300),
          child: child,
        );
      },
    );
  }
}
```

## 四、内存监控

```dart
// 内存监控工具
class MemoryMonitor {
  static void logMemoryUsage(String tag) {
    // Flutter 内存信息
    final info = PaintingBinding.instance.imageCache;
    debugPrint('[$tag] Image缓存: ${info.currentSize}/${info.maximumSize}');
    debugPrint('[$tag] 缓存字节: ${info.currentSizeBytes}');
  }

  static void clearImageCache() {
    PaintingBinding.instance.imageCache.clear();
  }

  static void configureImageCache({
    int maxSize = 100,
    int maxSizeBytes = 50 * 1024 * 1024, // 50MB
  }) {
    final cache = PaintingBinding.instance.imageCache;
    cache.maximumSize = maxSize;
    cache.maximumSizeBytes = maxSizeBytes;
  }
}
```

```dart
import 'dart:isolate';

Future<dynamic> heavyComputation(List<dynamic> data) async {
  final receivePort = ReceivePort();
  await Isolate.spawn(_isolateEntry, [receivePort.sendPort, data]);
  return await receivePort.first;
}

void _isolateEntry(List<dynamic> args) {
  final sendPort = args[0] as SendPort;
  final data = args[1] as List<dynamic>;
  // 在隔离中执行耗时计算
```

## 六、注意事项

1. **及时释放**：dispose 中释放所有控制器、订阅、定时器
2. **图片缓存**：限制图片缓存大小，避免内存溢出
3. **列表优化**：长列表使用懒加载，不可见项及时回收
4. **大文件处理**：大文件分块读写，避免一次性加载
5. **监控工具**：使用 DevTools Memory 面板监控内存使用
