# 渲染性能优化

## 一、概念说明

渲染性能决定了应用的流畅度。60fps 意味着每帧 16.67ms，任何超过这个时间的渲染都会导致掉帧。

```dart
// 渲染优化目标
/*
帧率: 60fps (16.67ms/帧)
避免:
- 长任务 (>16ms)
- 过度重建
- 内存抖动
- 复杂布局
*/
```

## 二、减少重建

```dart
// Flutter 减少重建
// 1. 使用 const 构造函数
class OptimizedPage extends StatelessWidget {
  const OptimizedPage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Column(
      children: [
        Text('静态文本'), // const
        Icon(Icons.star),  // const
      ],
    );
  }
}

// 2. 拆分 Widget，隔离变化
class ParentWidget extends StatefulWidget {
  @override
  State<ParentWidget> createState() => _ParentWidgetState();
}

class _ParentWidgetState extends State<ParentWidget> {
  int _count = 0;

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        const StaticHeader(), // 不会重建
        CounterDisplay(count: _count), // 只有这个重建
        ElevatedButton(
          onPressed: () => setState(() => _count++),
          child: const Text('+1'),
        ),
      ],
    );
  }
}

// 3. 使用 Selector 只订阅需要的部分
Selector<CartModel, int>(
  selector: (_, cart) => cart.itemCount,
  builder: (context, count, _) {
    return Text('商品: $count');
  },
)
```

## 三、列表优化

```dart
// 列表渲染优化
class OptimizedListView extends StatelessWidget {
  final List<Item> items;

  const OptimizedListView({super.key, required this.items});

  @override
  Widget build(BuildContext context) {
    return ListView.builder(
      itemCount: items.length,
      // 固定高度项优化
      itemExtent: 80,
      // 缓存区域
      cacheExtent: 200,
      itemBuilder: (context, index) {
        return _ListItem(item: items[index]);
      },
    );
  }
}

// 列表项 Widget 优化
class _ListItem extends StatelessWidget {
  final Item item;
  const _ListItem({required this.item});

  @override
  Widget build(BuildContext context) {
    return ListTile(
      leading: CachedNetworkImage(
        imageUrl: item.thumbnail,
        width: 50,
        height: 50,
        fit: BoxFit.cover,
      ),
      title: Text(item.title),
      subtitle: Text(item.subtitle),
    );
  }
}
```

## 四、图片优化

```dart
// 图片性能优化
class OptimizedImage extends StatelessWidget {
  final String url;
  final double size;

  const OptimizedImage({super.key, required this.url, this.size = 100});

  @override
  Widget build(BuildContext context) {
    return Image.network(
      url,
      width: size,
      height: size,
      fit: BoxFit.cover,
      // 限制解码尺寸
      cacheWidth: (size * MediaQuery.of(context).devicePixelRatio).toInt(),
      cacheHeight: (size * MediaQuery.of(context).devicePixelRatio).toInt(),
      // 过滤质量
      filterQuality: FilterQuality.medium,
    );
  }
}
```

## 五、动画优化

```dart
// 高性能动画
class PerformantAnimation extends StatefulWidget {
  const PerformantAnimation({super.key});

  @override
  State<PerformantAnimation> createState() => _PerformantAnimationState();
}

class _PerformantAnimationState extends State<PerformantAnimation>
    with SingleTickerProviderStateMixin {
  late AnimationController _controller;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      duration: const Duration(milliseconds: 300),
      vsync: this,
    );
  }

  @override
  Widget build(BuildContext context) {
    return AnimatedBuilder(
      animation: _controller,
      // child 不会重建
      child: const Icon(Icons.star, size: 50),
      builder: (context, child) {
        return Transform.scale(
          scale: _controller.value,
          child: child,
        );
      },
    );
  }
}
```

## 六、注意事项

1. **Profile 模式测试**：Debug 模式有 JIT 开销，用 Profile 模式测性能
2. **DevTools 分析**：使用 Performance 面板分析帧率和重建
3. **避免内联函数**：回调函数使用成员方法而非内联
4. **RepaintBoundary**：使用 RepaintBoundary 隔离重绘区域
5. **懒加载**：不在首屏的内容延迟加载
