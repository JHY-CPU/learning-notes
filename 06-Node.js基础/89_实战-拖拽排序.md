# 实战 - 拖拽排序 (Drag & Drop Sort)

## 项目需求与功能分析

拖拽排序是提升用户体验的常见交互模式。本项目使用 HTML5 原生 Drag API 实现列表项的拖拽排序。

### 核心功能

- 拖拽列表项排序
- 拖拽手柄（仅手柄区域可拖拽）
- 拖拽时的视觉反馈
- 排序结果保存到 localStorage
- 支持动画过渡

```html
<style>
  * { margin: 0; padding: 0; box-sizing: border-box; }
  body { font-family: 'Segoe UI', sans-serif; background: #f5f5f5; display: flex; justify-content: center; padding: 40px 20px; }
  .container { width: 100%; max-width: 450px; }
  h2 { text-align: center; color: #333; margin-bottom: 20px; }
  .sort-list { list-style: none; }
  .sort-item { display: flex; align-items: center; background: #fff; margin-bottom: 8px; padding: 14px 16px; border-radius: 10px; box-shadow: 0 1px 4px rgba(0,0,0,0.06); cursor: grab; transition: all 0.2s; user-select: none; }
  .sort-item:active { cursor: grabbing; }
  .sort-item.dragging { opacity: 0.4; transform: scale(0.98); box-shadow: none; }
  .sort-item.drag-over { border-top: 3px solid #667eea; }
  .handle { margin-right: 12px; color: #ccc; font-size: 18px; cursor: grab; display: flex; flex-direction: column; gap: 2px; }
  .handle span { display: block; width: 16px; height: 2px; background: #ccc; border-radius: 1px; }
  .item-number { width: 28px; height: 28px; background: #667eea; color: #fff; border-radius: 50%; display: flex; align-items: center; justify-content: center; font-size: 13px; font-weight: 600; margin-right: 14px; flex-shrink: 0; }
  .item-content { flex: 1; }
  .item-title { font-size: 15px; color: #333; font-weight: 500; }
  .item-desc { font-size: 12px; color: #999; margin-top: 3px; }
  .status { text-align: center; margin-top: 15px; font-size: 13px; color: #999; }
</style>

  <h2>拖拽排序</h2>
  <ul class="sort-list" id="sortList"></ul>
  <div class="status" id="status">拖拽列表项调整顺序</div>
</div>

<script>
const defaultItems = [
  { id: 1, title: '学习 HTML 基础', desc: '标签、属性、语义化' },
  { id: 2, title: '学习 CSS 布局', desc: 'Flexbox、Grid、响应式' },
  { id: 3, title: '学习 JavaScript', desc: 'ES6+、DOM、异步' },
  { id: 4, title: '学习 Node.js', desc: '模块系统、Express、数据库' },
  { id: 5, title: '学习 React/Vue', desc: '组件化、状态管理、路由' },
  { id: 6, title: '项目实战', desc: '全栈项目开发与部署' },
  { id: 7, title: '算法与数据结构', desc: '刷题、竞赛、面试准备' },
  { id: 8, title: '系统设计', desc: '架构、性能、安全' },
];

class SortableList {
  constructor(container, items) {
    this.container = container;
    this.items = JSON.parse(localStorage.getItem('sortItems')) || items;
    this.draggedEl = null;
    this.render();
  }

  render() {
    this.container.innerHTML = '';
    this.items.forEach((item, index) => {
      const li = document.createElement('li');
      li.className = 'sort-item';
      li.draggable = true;
      li.dataset.index = index;
      li.innerHTML = `
        <div class="handle"><span></span><span></span><span></span></div>
        <div class="item-number">${index + 1}</div>
        <div class="item-content">
          <div class="item-title">${item.title}</div>
          <div class="item-desc">${item.desc}</div>
        </div>`;

      li.addEventListener('dragstart', e => this.onDragStart(e, li));
      li.addEventListener('dragend', e => this.onDragEnd(e, li));
      li.addEventListener('dragover', e => this.onDragOver(e, li));
      li.addEventListener('dragenter', e => this.onDragEnter(e, li));
      li.addEventListener('dragleave', e => this.onDragLeave(e, li));
      li.addEventListener('drop', e => this.onDrop(e, li));

      this.container.appendChild(li);
    });
  }

  onDragStart(e, li) {
    this.draggedEl = li;
    li.classList.add('dragging');
    e.dataTransfer.effectAllowed = 'move';
    e.dataTransfer.setData('text/plain', li.dataset.index);
  }

  onDragEnd(e, li) {
    li.classList.remove('dragging');
    document.querySelectorAll('.sort-item').forEach(el => el.classList.remove('drag-over'));
    this.draggedEl = null;
  }

  onDragOver(e, li) {
    e.preventDefault();
    e.dataTransfer.dropEffect = 'move';
  }

  onDragEnter(e, li) {
    e.preventDefault();
    if (li !== this.draggedEl) li.classList.add('drag-over');
  }

  onDragLeave(e, li) {
    li.classList.remove('drag-over');
  }

  onDrop(e, li) {
    e.preventDefault();
    li.classList.remove('drag-over');
    const fromIndex = parseInt(e.dataTransfer.getData('text/plain'));
    const toIndex = parseInt(li.dataset.index);
    if (fromIndex === toIndex) return;

    // 移动元素
    const [moved] = this.items.splice(fromIndex, 1);
    this.items.splice(toIndex, 0, moved);
    this.save();
    this.render();
    document.getElementById('status').textContent = `已移动: "${moved.title}"`;
  }

  save() {
    localStorage.setItem('sortItems', JSON.stringify(this.items));
  }
}

new SortableList(document.getElementById('sortList'), defaultItems);
</script>
</body>
</html>
```

## 核心技术详解

### HTML5 Drag API

- `dragstart`：开始拖拽，设置数据
- `dragover`：经过目标上方，必须 `preventDefault` 才能 drop
- `drop`：放下，获取数据并重新排序

### 视觉反馈

拖拽中的元素降低透明度，目标位置显示上方边框指示。

## 扩展方向

1. **触摸支持**：添加 touch 事件支持移动端
2. **多列表拖拽**：在不同列表之间拖拽
3. **嵌套排序**：支持父子层级的排序
4. **动画平滑**：使用 FLIP 动画技术实现平滑过渡
5. **虚拟滚动**：大量数据时使用虚拟列表
6. **撤销操作**：支持 Ctrl+Z 撤销排序
7. **批量拖拽**：选中多个项目一起拖拽
