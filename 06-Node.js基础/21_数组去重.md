# 数组去重


## 数组去重


Set/Map/filter+indexOf/sort/哈希表、性能对比、对象去重。


## 数组去重方法


```
// ========== Set 去重 (最简单) ==========
const unique = [...new Set([1, 2, 2, 3, 3, 4])];
// [1, 2, 3, 4]

// ========== filter + indexOf ==========
const unique = arr.filter((v, i) => arr.indexOf(v) === i);

// ========== Map 去重 (推荐对象数组) ==========
const unique = [...new Map(arr.map(v => [v.id, v])).values()];

// ========== reduce 去重 ==========
const unique = arr.reduce((acc, v) => acc.includes(v) ? acc : [...acc, v], []);

// ========== 对象去重 ==========
const unique = arr.filter((v, i, self) =>
    self.findIndex(o => o.id === v.id) === i
);
```

## 各方法性能对比

| 方法 | 时间复杂度 | 适用场景 | 说明 |
|------|-----------|---------|------|
| Set | O(n) | 基础类型 | 最简洁高效 |
| filter+indexOf | O(n²) | 基础类型 | 兼容性好 |
| Map | O(n) | 对象数组 | 按指定key去重 |
| reduce+includes | O(n²) | 小数组 | 可读性一般 |
| findIndex | O(n²) | 对象数组 | 灵活自定义条件 |

## 使用场景

1. **Set 去重**：数字、字符串等基础类型数组，推荐首选方案
2. **Map 去重**：对象数组按某个唯一字段（如id）去重
3. **filter+findIndex**：对象数组按自定义条件去重
4. **哈希表**：超大数据集去重，使用Object/Map做哈希

## 常见坑点

- **NaN去重**：Set中NaN只保留一个，indexOf无法正确处理NaN
- **对象比较**：对象引用不同视为不同元素，需自定义比较逻辑
- **undefined/null**：会被正常去重保留一个

## 深度去重与特殊场景

```javascript
// ========== 深度对象数组去重 ==========
function deepUnique(arr, keyFn) {
    const seen = new Map();
    return arr.filter(item => {
        const key = keyFn(item);
        if (seen.has(key)) return false;
        seen.set(key, true);
        return true;
    });
}

// 按多个字段去重
const users = [
    { id: 1, name: 'Alice', dept: 'IT' },
    { id: 2, name: 'Bob', dept: 'HR' },
    { id: 1, name: 'Alice', dept: 'IT' }, // 重复
];
const uniqueUsers = deepUnique(users, u => `${u.id}-${u.name}`);
// [{ id: 1, name: 'Alice', dept: 'IT' }, { id: 2, name: 'Bob', dept: 'HR' }]

// ========== JSON 序列化去重 (适用于简单对象) ==========
function uniqueByJson(arr) {
    const seen = new Set();
    return arr.filter(item => {
        const key = JSON.stringify(item);
        if (seen.has(key)) return false;
        seen.add(key);
        return true;
    });
}

// ========== 有序数组原地去重 (双指针) ==========
function uniqueSorted(arr) {
    if (arr.length === 0) return 0;
    let slow = 0;
    for (let fast = 1; fast < arr.length; fast++) {
        if (arr[fast] !== arr[slow]) {
            slow++;
            arr[slow] = arr[fast];
        }
    }
    return slow + 1; // 返回去重后长度
}

// ========== 计数去重 (保留出现次数) ==========
function countUnique(arr) {
    const count = new Map();
    for (const item of arr) {
        count.set(item, (count.get(item) || 0) + 1);
    }
    return [...count.entries()].map(([value, count]) => ({ value, count }));
}
// countUnique([1,2,2,3,3,3]) → [{value:1,count:1},{value:2,count:2},{value:3,count:3}]
```

## 性能基准测试

```javascript
// ========== 去重方法性能对比 ==========
function benchmark(name, fn, arr) {
    const start = performance.now();
    for (let i = 0; i < 100; i++) fn([...arr]);
    console.log(`${name}: ${(performance.now() - start).toFixed(2)}ms`);
}

// 生成10万个随机数
const largeArray = Array.from({ length: 100000 }, () => Math.floor(Math.random() * 50000));

benchmark('Set', arr => [...new Set(arr)], largeArray);           // ~15ms
benchmark('filter+indexOf', arr => arr.filter((v,i) => arr.indexOf(v)===i), largeArray); // ~8000ms
benchmark('reduce+includes', arr => arr.reduce((a,v) => a.includes(v)?a:[...a,v],[]), largeArray); // >30000ms
benchmark('Map', arr => [...new Map(arr.map(v=>[v,v])).values()], largeArray); // ~20ms

// 结论: Set/Map O(n) 远快于 filter+indexOf/ reduce+includes O(n²)
```
