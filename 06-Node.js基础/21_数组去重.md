# 数组去重


## 数组去重


Set/Map/filter+indexOf/sort/哈希表、性能对比、对象去重。


## 数组去重方法


```
// ========== Set 去重 (最简单) ==========
const unique = [...new Set([1, 2, 2, 3, 3, 4])];
// [1, 2, 3, 4]

// ========== filter + indexOf ==========
const unique = arr.filter((v, i) => arr.indexOf(v) === i);

// ========== Map 去重 (推荐对象数组) ==========
const unique = [...new Map(arr.map(v => [v.id, v])).values()];

// ========== reduce 去重 ==========
const unique = arr.reduce((acc, v) => acc.includes(v) ? acc : [...acc, v], []);

// ========== 对象去重 ==========
const unique = arr.filter((v, i, self) =>
    self.findIndex(o => o.id === v.id) === i
);
```
