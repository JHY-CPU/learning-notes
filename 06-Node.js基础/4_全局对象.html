全局对象

global 全局对象、process 核心对象、Buffer、__dirname/__filename。

Node.js 全局对象

// ========== global ========== // Node.js 的全局对象 (类似浏览器的 window) global.myVar = 'accessible everywhere'; // ========== process ========== // 进程信息和控制 process.argv // 命令行参数 process.env // 环境变量 process.cwd() // 当前工作目录 process.pid // 进程 ID process.platform // 平台 (win32/darwin/linux) process.version // Node 版本 process.exit(code) // 退出进程 // ========== Buffer ========== // 处理二进制数据 Buffer.from('hello') // 字符串 → Buffer Buffer.alloc(1024) // 分配 1KB Buffer.concat([buf1, buf2]) // 合并 // ========== __dirname / __filename ========== // __dirname — 当前模块目录 // __filename — 当前模块文件路径 // ⚠️ 仅在 CommonJS 中可用 // ESM 中需要: import.meta.url 转换 // ========== 其他全局对象 ========== console // 控制台 setTimeout/clearTimeout setInterval/clearInterval setImmediate/clearImmediate // Node 特有 queueMicrotask console.time / console.timeEnd exports / module / require
