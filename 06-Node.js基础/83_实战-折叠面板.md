# 实战 - 折叠面板 (Accordion)

## 项目需求与功能分析

折叠面板（Accordion）是常见的信息展示组件，用于在有限空间中展示多个可展开/收起的内容区域。本项目实现一个支持手风琴模式、CSS 动画和可访问性的折叠面板组件。

### 核心功能

- 点击标题展开 / 收起内容
- 手风琴模式（同时只展开一个）
- CSS 平滑展开动画
- 支持多个独立面板
- 键盘可访问（Enter / Space）
- 图标旋转指示

class Accordion {
  constructor(container, options = {}) {
    this.container = container;
    this.items = container.querySelectorAll('.accordion-item');
    this.accordion = options.accordion !== false; // 默认手风琴模式
    this.bindEvents();
  }

  bindEvents() {
    this.items.forEach(item => {
      const header = item.querySelector('.accordion-header');
      header.addEventListener('click', () => this.toggle(item));
      header.addEventListener('keydown', e => {
        if (e.key === 'Enter' || e.key === ' ') {
          e.preventDefault();
          this.toggle(item);
        }
      });
    });
  }

  toggle(item) {
    const isActive = item.classList.contains('active');
    if (this.accordion) {
      this.items.forEach(i => i.classList.remove('active'));
    }
    if (!isActive) {
      item.classList.add('active');
    }
  }

  open(index) { this.items[index]?.classList.add('active'); }
  close(index) { this.items[index]?.classList.remove('active'); }
  openAll() { this.items.forEach(i => i.classList.add('active')); }
  closeAll() { this.items.forEach(i => i.classList.remove('active')); }
}

const acc = new Accordion(document.getElementById('accordion'), { accordion: true });
</script>

## 核心技术详解

### CSS 动画展开

使用 `max-height` 过渡实现平滑展开动画。`max-height: 0` 折叠，`max-height: 500px` 展开。

### 手风琴模式

切换时先移除所有面板的 `active` 类，再给当前面板添加，确保只有一个展开。

### 可访问性

- `tabindex="0"` 让面板可聚焦
- 支持 Enter 和 Space 键触发切换
- `focus` 样式提供视觉反馈

## 测试用例

```javascript
describe('Accordion', () => {
  test('toggle 切换', () => {
    const items = [{ active: false }, { active: false }];
    const toggle = (idx) => {
      const isActive = items[idx].active;
      items.forEach(i => i.active = false);
      if (!isActive) items[idx].active = true;
    };
    toggle(0); expect(items[0].active).toBe(true);
    toggle(0); expect(items[0].active).toBe(false);
  });

  test('手风琴互斥', () => {
    const items = [{ active: true }, { active: false }];
    const toggle = (idx) => {
      const isActive = items[idx].active;
      items.forEach(i => i.active = false);
      if (!isActive) items[idx].active = true;
    };
    toggle(1);
    expect(items[0].active).toBe(false);
    expect(items[1].active).toBe(true);
  });
});
```

## 扩展方向

1. **嵌套面板**：支持多层嵌套的折叠面板
2. **异步加载**：展开时从 API 获取内容
3. **拖拽排序**：可拖拽调整面板顺序
4. **自定义图标**：支持 Chevron、Plus/Minus 等图标样式
5. **URL 锚点**：展开面板时更新 URL hash
6. **持久化状态**：记住用户展开的面板
7. **过渡高度自动计算**：使用 `scrollHeight` 替代固定 `max-height`
