# 实战 - 进度条 (Progress Bar)

## 项目需求与功能分析

进度条是展示任务进度的基础 UI 组件。本项目实现线性进度条、环形进度条、定时进度等多种类型。

### 核心功能

- 线性进度条（水平）
- 环形进度条（SVG）
- 上传进度模拟
- 定时进度条
- 条纹动画效果
- 多种颜色和尺寸

// ===== 上传进度模拟 =====
class UploadProgress {
  constructor() {
    this.fill = document.getElementById('uploadFill');
    this.text = document.getElementById('uploadText');
    this.pct = document.getElementById('uploadPct');
    document.getElementById('startUpload').addEventListener('click', () => this.start());
    document.getElementById('resetUpload').addEventListener('click', () => this.reset());
  }

  start() {
    this.reset();
    this.text.textContent = '上传中...';
    let progress = 0;
    const interval = setInterval(() => {
      progress += Math.random() * 15;
      if (progress >= 100) {
        progress = 100;
        clearInterval(interval);
        this.text.textContent = '上传完成!';
      }
      this.fill.style.width = progress + '%';
      this.fill.textContent = Math.round(progress) + '%';
      this.pct.textContent = Math.round(progress) + '%';
    }, 200);
  }

  reset() {
    this.fill.style.width = '0%';
    this.fill.textContent = '';
    this.text.textContent = '等待上传...';
    this.pct.textContent = '0%';
  }
}

// ===== 定时进度 =====
class TimerProgress {
  constructor() {
    this.fill = document.getElementById('timerFill');
    this.text = document.getElementById('timerText');
    this.pct = document.getElementById('timerPct');
    document.getElementById('startTimer').addEventListener('click', () => this.start());
  }

  start(duration = 5000) {
    this.text.textContent = '运行中...';
    const start = performance.now();
    const tick = () => {
      const elapsed = performance.now() - start;
      const progress = Math.min(elapsed / duration * 100, 100);
      this.fill.style.width = progress + '%';
      this.pct.textContent = Math.round(progress) + '%';
      if (progress < 100) {
        requestAnimationFrame(tick);
      } else {
        this.text.textContent = '已完成!';
      }
    };
    requestAnimationFrame(tick);
  }
}

// ===== 环形进度 =====
class CircularProgress {
  constructor(container, items) {
    const circumference = 2 * Math.PI * 40;
    items.forEach(item => {
      const offset = circumference * (1 - item.value / 100);
      const color = item.color || '#667eea';
      container.innerHTML += `
        <div class="circle-container">
          <svg width="100" height="100">
            <circle class="bg" cx="50" cy="50" r="40"/>
            <circle class="fg" cx="50" cy="50" r="40" stroke="${color}"
              stroke-dasharray="${circumference}" stroke-dashoffset="${offset}"/>
          </svg>
          <div class="label">${item.value}%</div>
          <div class="sub">${item.label}</div>
        </div>`;
    });
  }
}

// ===== 系统资源模拟 =====
class SystemMonitor {
  constructor() {
    document.getElementById('refreshSys').addEventListener('click', () => this.refresh());
    this.refresh();
  }

  refresh() {
    const mem = Math.floor(Math.random() * 40 + 50);
    const cpu = Math.floor(Math.random() * 60 + 20);
    const disk = Math.floor(Math.random() * 30 + 60);

    document.getElementById('memFill').style.width = mem + '%';
    document.getElementById('memPct').textContent = mem + '%';
    document.getElementById('cpuFill').style.width = cpu + '%';
    document.getElementById('cpuPct').textContent = cpu + '%';
    document.getElementById('diskFill').style.width = disk + '%';
    document.getElementById('diskPct').textContent = disk + '%';
  }
}

## 核心技术详解

### CSS 动画条纹

使用 `background-image` 渐变 + `background-position` 动画实现流动条纹效果。

### SVG 环形进度

`stroke-dasharray` 设置为周长，`stroke-dashoffset` 控制显示的弧长。

## 扩展方向

1. **步骤进度条**：多步骤流程进度展示
2. **文件上传**：真实文件上传进度（XMLHttpRequest / Fetch）
3. **多色分段**：不同颜色代表不同类别
4. **动画缓动**：使用贝塞尔曲线自定义动画效果
5. **数据可视化**：集成到仪表盘中
6. **响应式**：适配移动端
7. **可交互**：点击进度条跳转进度
