渲染优化 API

// ========== 渲染流水线 ========== // JavaScript → Style → Layout → Paint → Composite // 重排↑ 重绘↑ 合成↑ // ========== 触发重排 (Layout) ========== // 读取/修改几何属性: width/height/top/left/margin/padding // 读取布局信息: offsetHeight/clientHeight/getBoundingClientRect // ========== 触发重绘 (Paint) ========== // color, background, visibility, outline // ========== 仅触发合成 (Composite) ========== // transform, opacity // ✅ 推荐动画属性 // ========== GPU 加速 ========== will-change: transform; // 提示浏览器优化 transform: translateZ(0); // 强制 GPU 层
