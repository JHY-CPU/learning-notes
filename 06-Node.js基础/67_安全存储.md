# 安全存储


## 安全存储


HttpOnly/Secure/SameSite Cookie、敏感信息处理、Token 存储最佳实践。


## 安全存储 API


```
// ========== Cookie 安全属性 ==========
Set-Cookie: session=abc;
    HttpOnly     // 禁止 JS 读取 (防御 XSS)
    Secure       // 仅 HTTPS 传输
    SameSite=Lax // 防 CSRF
    Domain=example.com
    Path=/
    Max-Age=86400

// ========== Token 存储对比 ==========
// localStorage: ❌ XSS 可读取
// sessionStorage: ❌ XSS 可读取
// Cookie (HttpOnly): ✅ 安全
// 内存变量: ✅ 最安全 (刷新丢失)

// ========== 敏感信息处理原则 ==========
// 1. 不在前端存储敏感数据
// 2. 使用短生命周期的 Token
// 3. 敏感操作需二次验证
// 4. HTTPS 传输
```
