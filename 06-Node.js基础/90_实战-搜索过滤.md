# 实战 - 搜索过滤 (Search Filter)

## 项目需求与功能分析

搜索过滤是列表和表格页面的核心功能。本项目实现实时搜索、防抖处理、高亮匹配和分类过滤。

### 核心功能

- 实时搜索过滤
- 输入防抖 (debounce)
- 搜索结果高亮
- 分类标签过滤
- 空状态提示
- 搜索历史

<script>
const data = [

  { title: 'Go 语言编程', desc: '并发编程与微服务', category: '后端' },
  { title: 'TypeScript 实战', desc: '类型系统与工程化', category: '前端' },
  { title: 'Docker 容器化', desc: '容器编排与部署', category: '运维' },
  { title: 'Redis 深度历险', desc: '缓存与分布式锁', category: '后端' },
  { title: '算法导论', desc: '经典算法与数据结构', category: '算法' },

class SearchFilter {
  constructor(items) {
    this.items = items;
    this.query = '';
    this.activeCategory = '全部';
    this.input = document.getElementById('searchInput');
    this.clearBtn = document.getElementById('clearBtn');
    this.tagsEl = document.getElementById('tags');
    this.listEl = document.getElementById('itemList');
    this.countEl = document.getElementById('resultCount');
    this.categories = ['全部', ...new Set(items.map(i => i.category))];
    this.init();
  }

  init() {
    this.renderTags();
    // 防抖搜索
    let timer;
    this.input.addEventListener('input', () => {
      clearTimeout(timer);
      timer = setTimeout(() => {
        this.query = this.input.value.trim();
        this.clearBtn.style.display = this.query ? 'block' : 'none';
        this.render();
      }, 300);
    });
    this.clearBtn.addEventListener('click', () => {
      this.input.value = '';
      this.query = '';
      this.clearBtn.style.display = 'none';
      this.render();
      this.input.focus();
    });
    this.render();
  }

  renderTags() {
    this.tagsEl.innerHTML = this.categories.map(cat =>
      `<button class="tag ${cat === this.activeCategory ? 'active' : ''}" data-cat="${cat}">${cat}</button>`
    ).join('');
    this.tagsEl.querySelectorAll('.tag').forEach(btn => {
      btn.addEventListener('click', () => {
        this.activeCategory = btn.dataset.cat;
        this.tagsEl.querySelector('.active').classList.remove('active');
        btn.classList.add('active');
        this.render();
      });
    });
  }

  filter() {
    return this.items.filter(item => {
      const matchQuery = !this.query ||
        item.title.toLowerCase().includes(this.query.toLowerCase()) ||
        item.desc.toLowerCase().includes(this.query.toLowerCase());
      const matchCat = this.activeCategory === '全部' || item.category === this.activeCategory;
      return matchQuery && matchCat;
    });
  }

  highlight(text) {
    if (!this.query) return text;
    const regex = new RegExp(`(${this.query.replace(/[.*+?^${}()|[\]\\]/g, '\\$&')})`, 'gi');
    return text.replace(regex, '<mark>$1</mark>');
  }

  render() {
    const filtered = this.filter();
    this.countEl.textContent = `${filtered.length} / ${this.items.length} 个项目`;
    if (filtered.length === 0) {
      this.listEl.innerHTML = '<div class="empty"><div class="icon">🔍</div><p>没有找到匹配的项目</p></div>';
      return;
    }
    this.listEl.innerHTML = filtered.map(item => `
      <div class="item">
        <div class="item-title">${this.highlight(item.title)}</div>
        <div class="item-desc">${this.highlight(item.desc)}</div>
        <span class="item-category">${item.category}</span>
      </div>`).join('');
  }
}

new SearchFilter(data);
</script>

## 核心技术详解

### 防抖 (Debounce)

使用 `setTimeout` + `clearTimeout` 实现防抖，用户停止输入 300ms 后才执行搜索，避免频繁过滤。

### 高亮匹配

使用正则表达式将搜索词替换为 `<mark>` 标签包裹。

## 扩展方向

2. **搜索建议**：输入时显示搜索建议下拉列表
3. **搜索历史**：记住最近的搜索词
4. **高级过滤**：支持多条件组合过滤
5. **排序功能**：按名称、日期、相关度排序
6. **URL 同步**：搜索参数同步到 URL
7. **虚拟滚动**：大量结果时使用虚拟列表
