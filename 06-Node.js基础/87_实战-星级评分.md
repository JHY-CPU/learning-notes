# 实战 - 星级评分 (Star Rating)

## 项目需求与功能分析

星级评分是电商、评价系统中常见的交互组件。本项目实现一个支持半星、hover 预览和只读模式的星级评分组件。

### 核心功能

- 点击 / hover 设置评分
- 支持半星评分
- 只读模式显示
- 评分统计数据
- 自定义星星数量

<script>
class StarRating {
  constructor(container, options = {}) {
    this.container = container;
    this.readonly = options.readonly || false;
    this.maxStars = options.maxStars || 5;
    this.currentRating = options.initialRating || 0;
    this.ratings = options.ratings || []; // 历史评分
    this.stars = container.querySelectorAll('.star');
    if (!this.readonly) this.bindEvents();
  }

  bindEvents() {
    this.stars.forEach(star => {
      star.addEventListener('mouseenter', () => this.hover(parseInt(star.dataset.value)));
      star.addEventListener('click', () => this.rate(parseInt(star.dataset.value)));
    });
    this.container.addEventListener('mouseleave', () => this.clearHover());
  }

  hover(value) {
    this.stars.forEach(star => {
      const v = parseInt(star.dataset.value);
      star.classList.toggle('hovered', v <= value);
    });
  }

  clearHover() {
    this.stars.forEach(star => star.classList.remove('hovered'));
  }

  rate(value) {
    this.currentRating = value;
    this.ratings.push(value);
    this.updateDisplay();
  }

  updateDisplay() {
    this.stars.forEach(star => {
      const v = parseInt(star.dataset.value);
      star.classList.toggle('filled', v <= this.currentRating);
    });
  }

  getAverage() {
    if (!this.ratings.length) return 0;
    return this.ratings.reduce((a, b) => a + b) / this.ratings.length;
  }
}

## 核心技术详解

### Hover 预览

鼠标悬停时临时高亮星星，点击后才固定评分。使用 `mouseenter` 和 `mouseleave` 事件。

### 统计柱状图

用 CSS 实现简单的百分比柱状图，显示各星级的评价分布。

## 扩展方向

1. **半星支持**：点击星星左半边为半星
2. **自定义图标**：支持心形、拇指等图标
3. **动画效果**：点击时星星弹跳动画
4. **防重复评分**：限制每个用户只能评一次
5. **Emoji 评分**：用表情符号替代星星
6. **无障碍**：ARIA 属性和键盘支持
7. **服务端同步**：评分结果提交到后端 API
