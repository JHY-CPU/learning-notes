# 实战 - 通知消息 (Toast Notification)

## 项目需求与功能分析

Toast 通知是用于向用户反馈操作结果的轻量级消息组件。本项目实现支持多种类型、自动消失、堆叠排列的 Toast 通知系统。

### 核心功能

- 多种类型（成功、错误、警告、信息）
- 自动消失（可配置时长）
- 手动关闭按钮
- 堆叠排列（多条通知并存）
- 位置可选（四角）
- 进度条指示剩余时间
- 进入 / 退出动画

<script>
class Toast {
  constructor() {
    this.containers = {};
    this.icons = {
      success: '✅', error: '❌', warning: '⚠️', info: 'ℹ️',
    };
  }

  _getContainer(position) {
    if (!this.containers[position]) {
      const el = document.createElement('div');
      el.className = `toast-container ${position}`;
      document.body.appendChild(el);
      this.containers[position] = el;
    }
    return this.containers[position];
  }

  _show(type, title, message, duration, position) {
    const container = this._getContainer(position);
    const el = document.createElement('div');
    el.className = `toast ${type} entering`;
    el.innerHTML = `
      <div class="toast-icon">${this.icons[type]}</div>
      <div class="toast-body">
        <div class="toast-title">${title}</div>
        <div class="toast-message">${message}</div>
      </div>
      <button class="toast-close">&times;</button>
      <div class="toast-progress" style="animation-duration:${duration}ms"></div>`;

    container.appendChild(el);
    setTimeout(() => el.classList.remove('entering'), 300);

    const close = () => {
      el.classList.add('exiting');
      setTimeout(() => el.remove(), 300);
    };

    el.querySelector('.toast-close').addEventListener('click', close);
    const timer = setTimeout(close, duration);

    el.addEventListener('mouseenter', () => clearTimeout(timer));
    el.addEventListener('mouseleave', () => setTimeout(close, duration));
  }

  success(title, message) {
    const pos = document.getElementById('position').value;
    const dur = parseInt(document.getElementById('duration').value) * 1000;
    this._show('success', title, message, dur, pos);
  }

  error(title, message) {
    const pos = document.getElementById('position').value;
    const dur = parseInt(document.getElementById('duration').value) * 1000;
    this._show('error', title, message, dur, pos);
  }

  warning(title, message) {
    const pos = document.getElementById('position').value;
    const dur = parseInt(document.getElementById('duration').value) * 1000;
    this._show('warning', title, message, dur, pos);
  }

  info(title, message) {
    const pos = document.getElementById('position').value;
    const dur = parseInt(document.getElementById('duration').value) * 1000;
    this._show('info', title, message, dur, pos);
  }
}

const toast = new Toast();

## 核心技术详解

### 动画

使用 CSS `@keyframes` 实现滑入 / 滑出动画。`entering` 类添加入场动画，`exiting` 类添加退场动画。

### 进度条

CSS `animation: progress linear forwards` 配合 `animation-duration` 实现倒计时进度条。

### 悬停暂停

鼠标悬停时 `clearTimeout` 暂停自动关闭，移开后重新计时。

## 扩展方向

1. **Promise API**：返回 Promise 支持 await 关闭
2. **确认通知**：带确认和取消按钮的 Toast
3. **图片通知**：支持缩略图展示
4. **声音提示**：不同类型播放不同音效
5. **通知中心**：收起的通知存入通知中心
6. **桌面通知**：集成 Notification API
7. **队列管理**：限制最大显示数量，超出排队
