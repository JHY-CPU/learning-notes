# 实战 - 模态框 (Modal/Dialog)

## 项目需求与功能分析

模态框（Modal）是覆盖在主页面上的对话框，用于需要用户专注操作的场景。本项目实现一个功能完善的模态框组件，包括背景遮罩、焦点陷阱、动画效果等。

### 核心功能

- 打开 / 关闭模态框
- 背景遮罩（点击遮罩关闭）
- 焦点陷阱（Tab 键限制在模态框内）
- Escape 键关闭
- 打开时禁止背景滚动
- 进入 / 退出动画
- 确认和取消回调

<script>
class Modal {
  constructor(overlay) {
    this.overlay = overlay;
    this.modal = overlay.querySelector('.modal');
    this.focusableEls = this.modal.querySelectorAll('button, input, textarea, [tabindex]');
    this.firstFocusable = this.focusableEls[0];
    this.lastFocusable = this.focusableEls[this.focusableEls.length - 1];
    this.previousFocus = null;
    this.bindEvents();
  }

  bindEvents() {
    // 关闭按钮
    this.modal.querySelector('.modal-close')?.addEventListener('click', () => this.close());
    this.modal.querySelector('.modal-cancel')?.addEventListener('click', () => this.close());
    // 点击遮罩关闭
    this.overlay.addEventListener('click', e => {
      if (e.target === this.overlay) this.close();
    });
    // Escape 键关闭
    document.addEventListener('keydown', e => {
      if (e.key === 'Escape' && this.isOpen()) this.close();
      if (e.key === 'Tab' && this.isOpen()) this.trapFocus(e);
    });
  }

  open() {
    this.previousFocus = document.activeElement;
    this.overlay.classList.add('active');
    document.body.classList.add('modal-open');
    this.firstFocusable?.focus();
  }

  close() {
    this.overlay.classList.remove('active');
    document.body.classList.remove('modal-open');
    this.previousFocus?.focus();
  }

  isOpen() {
    return this.overlay.classList.contains('active');
  }

  trapFocus(e) {
    if (e.shiftKey && document.activeElement === this.firstFocusable) {
      e.preventDefault();
      this.lastFocusable.focus();
    } else if (!e.shiftKey && document.activeElement === this.lastFocusable) {
      e.preventDefault();
      this.firstFocusable.focus();
    }
  }

  onConfirm(callback) {
    this.modal.querySelector('.modal-confirm')?.addEventListener('click', () => {
      callback();
      this.close();
    });
  }
}

## 核心技术详解

### 焦点陷阱

Tab 键在模态框内循环：焦点到最后一个可聚焦元素后按 Tab 回到第一个，Shift+Tab 反向。

### 打开状态管理

打开时给 `body` 添加 `modal-open` 类，CSS 中设置 `overflow: hidden` 禁止背景滚动。

### 焦点恢复

关闭时将焦点恢复到打开前的元素，保证可访问性。

## 扩展方向

1. **多尺寸**：支持 small / medium / full-screen 尺寸
2. **嵌套模态框**：支持模态框内再打开模态框
3. **拖拽移动**：拖拽标题栏移动模态框位置
4. **可调整大小**：拖拽边缘调整模态框大小
5. **堆叠管理**：多层模态框的 z-index 管理
6. **Promise API**：返回 Promise 支持 async/await 调用
7. **全局 API**：`Modal.confirm()`, `Modal.alert()` 快捷调用
