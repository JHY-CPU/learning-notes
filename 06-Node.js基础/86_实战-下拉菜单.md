# 实战 - 下拉菜单 (Dropdown)

## 项目需求与功能分析

下拉菜单是常见的导航和操作组件。本项目实现一个支持点击触发、分层子菜单和键盘导航的下拉菜单组件。

### 核心功能

- 点击触发下拉
- 点击外部自动关闭
- 支持禁用项
- 支持分层子菜单
- 键盘导航（上下箭头、Enter、Escape）
- 右侧对齐支持

<script>
class Dropdown {
  constructor(container) {
    this.container = container;
    this.trigger = container.querySelector('.dropdown-trigger');
    this.menu = container.querySelector('.dropdown-menu');
    this.items = container.querySelectorAll(':scope > .dropdown-menu > .dropdown-item:not(.disabled)');
    this.isOpen = false;
    this.focusedIndex = -1;
    this.bindEvents();
  }

  bindEvents() {
    this.trigger.addEventListener('click', e => { e.stopPropagation(); this.toggle(); });
    document.addEventListener('click', () => this.close());
    this.container.addEventListener('keydown', e => this.handleKey(e));
    this.items.forEach(item => {
      item.addEventListener('click', e => {
        if (item.classList.contains('disabled')) return;
        e.stopPropagation();
        this.close();
      });
    });
  }

  toggle() { this.isOpen ? this.close() : this.open(); }

  open() {
    this.isOpen = true;
    this.container.classList.add('open');
    this.focusedIndex = -1;
  }

  close() {
    this.isOpen = false;
    this.container.classList.remove('open');
    this.focusedIndex = -1;
  }

  handleKey(e) {
    if (!this.isOpen) return;
    switch (e.key) {
      case 'Escape': this.close(); this.trigger.focus(); break;
      case 'ArrowDown': e.preventDefault(); this.focusNext(); break;
      case 'ArrowUp': e.preventDefault(); this.focusPrev(); break;
      case 'Enter': e.preventDefault(); this.items[this.focusedIndex]?.click(); break;
    }
  }

  focusNext() {
    this.focusedIndex = (this.focusedIndex + 1) % this.items.length;
    this.items[this.focusedIndex].focus();
  }

  focusPrev() {
    this.focusedIndex = (this.focusedIndex - 1 + this.items.length) % this.items.length;
    this.items[this.focusedIndex].focus();
  }
}

document.querySelectorAll('.dropdown').forEach(el => new Dropdown(el));
</script>

## 核心技术详解

### CSS 动画

使用 `opacity + visibility + transform` 实现平滑出现效果，避免使用 `display`（无法过渡）。

### 子菜单

使用绝对定位将子菜单放在父项右侧，`:hover` 时显示。

### 点击外部关闭

在 `document` 上监听点击事件，任何不在菜单内的点击都关闭菜单。

## 扩展方向

1. **搜索过滤**：下拉菜单中带搜索框
2. **多选菜单**：支持勾选多个选项
3. **虚拟滚动**：大量选项时使用虚拟列表
4. **异步加载**：展开时加载子菜单数据
5. **拖拽排序**：菜单项可拖拽排序
6. **命令面板**：类似 VS Code 的命令面板
7. **上下文菜单**：右键弹出菜单
