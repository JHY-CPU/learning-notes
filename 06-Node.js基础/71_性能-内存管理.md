# 性能-内存管理


## 性能-内存管理


V8 垃圾回收、内存泄漏检测、Chrome DevTools Memory 面板、弱引用。


## 内存管理 API


```
// ========== 垃圾回收 (GC) ==========
// V8 使用: 分代回收 (Generational GC)
// 新生代: Scavenge (频繁, 小)
// 老生代: Mark-Sweep / Mark-Compact (不频繁, 大)

// ========== 内存泄漏常见原因 ==========
// 1. 全局变量 (意外)
// 2. 闭包引用未释放
// 3. 定时器未清理
// 4. DOM 引用未释放
// 5. 事件监听未移除
// 6. console.log (保留对象引用)

// ========== WeakMap / WeakSet ==========
// 弱引用: 不影响 GC
// 适合: 对象额外数据 / 缓存 / DOM 关联数据
```
