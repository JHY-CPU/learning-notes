# 分片上传


## 分片上传


文件分片(slice)、并发上传、进度合并、秒传(hash预检)。


## 分片上传 API


```
// ========== 文件分片 ==========
const CHUNK_SIZE = 5 * 1024 * 1024;  // 5MB
const chunks = [];
let start = 0;
while (start < file.size) {
    const end = Math.min(start + CHUNK_SIZE, file.size);
    chunks.push(file.slice(start, end));
    start = end;
}

// ========== 并发上传 ==========
const tasks = chunks.map((chunk, index) => {
    const formData = new FormData();
    formData.append('chunk', chunk);
    formData.append('index', index);
    formData.append('total', chunks.length);
    formData.append('hash', fileHash);
    return fetch('/upload/chunk', { method: 'POST', body: formData });
});
await Promise.all(tasks);

// ========== 合并请求 ==========
fetch('/upload/merge', {
    method: 'POST',
    body: JSON.stringify({ hash: fileHash, name: file.name, chunks: chunks.length })
});

// ========== 秒传 ==========
// 先计算文件 hash，查询服务器是否已存在
const hash = await calculateHash(file);
const res = await fetch('/upload/check', { body: JSON.stringify({ hash }) });
if (res.exist) { /* 秒传 */ }
```
