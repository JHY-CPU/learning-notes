# 实战 - 无限滚动 (Infinite Scroll)

## 项目需求与功能分析

无限滚动是社交媒体和内容平台的常见模式，用户滚动到底部时自动加载更多内容。本项目使用 IntersectionObserver 实现高性能的无限滚动。

### 核心功能

- IntersectionObserver 检测滚动到底部
- 分页加载数据
- 加载状态指示器
- 防重复加载
- 支持手动触发和自动触发

```html
<script>
class InfiniteScroll {
  constructor(container, options = {}) {
    this.container = container;
    this.pageSize = options.pageSize || 10;
    this.maxPages = options.maxPages || Infinity;
    this.page = 0;
    this.loading = false;
    this.hasMore = true;
    this.loader = document.getElementById('loader');
    this.stats = document.getElementById('stats');
    this.pageInfo = document.getElementById('pageInfo');
    this.items = [];

    // 模拟数据源
    this.dataPool = this.generateData(200);
    this.totalLoaded = 0;

    // IntersectionObserver
    this.observer = new IntersectionObserver(entries => {
      if (entries[0].isIntersecting && !this.loading && this.hasMore) {
        this.loadMore();
      }
    }, { rootMargin: '200px' });

    this.observer.observe(document.getElementById('sentinel'));
    this.loadMore();
  }

  generateData(count) {
    const titles = [
      '深入理解 JavaScript 闭包', 'React Hooks 最佳实践', 'CSS Grid 布局完全指南',
      'Node.js 性能优化技巧', 'Python 爬虫入门教程', 'Docker 容器化部署',
      'Vue 3 组合式 API 详解', 'TypeScript 高级类型', 'MySQL 索引优化',
      'Redis 缓存策略', '微服务架构设计', 'Kubernetes 入门',
    ];
    const descs = [
      '本文详细介绍了核心概念和实际应用场景，适合有一定基础的开发者。',
      '从零开始学习，包含大量代码示例和实战项目，循序渐进。',
      '涵盖最佳实践和常见陷阱，帮助你写出更优雅的代码。',
      '性能对比和优化建议，让你的应用运行更快更稳定。',
    ];
    const colors = ['#667eea','#f093fb','#4facfe','#43e97b','#fa709a','#a18cd1','#fbc2eb','#84fab0'];
    const data = [];
    for (let i = 0; i < count; i++) {
      data.push({
        title: titles[i % titles.length] + ` #${i + 1}`,
        desc: descs[i % descs.length],
        color: colors[i % colors.length],
        views: Math.floor(Math.random() * 10000),
        likes: Math.floor(Math.random() * 500),
        date: `2025-${String(Math.floor(Math.random()*12)+1).padStart(2,'0')}-${String(Math.floor(Math.random()*28)+1).padStart(2,'0')}`,
      });
    }
    return data;
  }

  async loadMore() {
    if (this.loading || !this.hasMore) return;
    this.loading = true;
    this.loader.style.display = 'block';

    // 模拟网络延迟
    await new Promise(resolve => setTimeout(resolve, 800));

    const start = this.page * this.pageSize;
    const end = start + this.pageSize;
    const items = this.dataPool.slice(start, end);

    if (items.length === 0 || this.page >= this.maxPages) {
      this.hasMore = false;
      this.loader.style.display = 'none';
      this.pageInfo.textContent = `已加载全部 ${this.totalLoaded} 条数据`;
      return;
    }

    items.forEach(item => {
      const card = document.createElement('div');
      card.className = 'card';
      card.innerHTML = `
        <div class="card-avatar" style="background:${item.color}20;color:${item.color}">
          ${item.title.charAt(0)}
        </div>
        <div class="card-body">
          <div class="card-title">${item.title}</div>
          <div class="card-text">${item.desc}</div>
          <div class="card-meta">
            <span>📅 ${item.date}</span>
            <span>👁 ${item.views}</span>
            <span>❤ ${item.likes}</span>
          </div>
        </div>`;
      this.container.appendChild(card);
    });

    this.totalLoaded += items.length;
    this.page++;
    this.loading = false;
    this.loader.style.display = 'none';
    this.stats.textContent = `已加载: ${this.totalLoaded} 篇 | 第 ${this.page} 页`;
    this.pageInfo.textContent = `向下滚动加载更多`;
  }

  reset() {
    this.page = 0;
    this.totalLoaded = 0;
    this.hasMore = true;
    this.container.innerHTML = '';
    this.loadMore();
  }
}

new InfiniteScroll(document.getElementById('cardList'), { pageSize: 10 });
</script>
</body>
</html>
```

## 核心技术详解

### IntersectionObserver

`rootMargin: '200px'` 表示在目标元素进入视口前 200px 就开始加载，用户感知不到加载延迟。

### 防重复加载

使用 `loading` 标志确保上一次加载完成前不会触发新的加载请求。

## 扩展方向

1. **骨架屏**：加载时显示骨架屏占位
2. **下拉刷新**：顶部下拉刷新最新数据
3. **虚拟列表**：大数据量时只渲染可见区域
4. **图片懒加载**：图片进入视口才加载
5. **错误重试**：加载失败自动重试
6. **滚动位置恢复**：返回时恢复滚动位置
7. **瀑布流布局**：不同高度的卡片排列
