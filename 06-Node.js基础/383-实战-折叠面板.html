🎨 HTML 是什么? ▼

HTML (HyperText Markup Language) 是构建网页的标准标记语言。它使用标签来描述网页的结构和内容。

🎨 CSS 的作用? ▼

CSS (Cascading Style Sheets) 用于控制网页的视觉呈现，包括布局、颜色、字体、动画等样式。

🎨 JavaScript 特点? ▼

JavaScript 是一种动态编程语言，支持面向对象、函数式和事件驱动编程，是 Web 开发的核心语言。

🎨 什么是 DOM? ▼

DOM (Document Object Model) 是 HTML 文档的编程接口，将文档表示为树结构，允许 JS 动态操作页面。
