# 性能-代码分割


## 性能-代码分割


动态 import() 分包、React.lazy、路由级分割、chunk 命名。


## 代码分割 API


```
// ========== 动态 import() ==========
const module = await import('./module.js');
// 返回 Promise
// Webpack/Vite 自动分包

// ========== React.lazy ==========
const HeavyComponent = React.lazy(() => import('./Heavy'));
}>



// ========== 路由级分割 ==========
const Home = () => import('./pages/Home');
const About = () => import('./pages/About');
const Dashboard = () => import('./pages/Dashboard');

// ========== Chunk 命名 ==========
// Webpack: /* webpackChunkName: "admin" */
// Vite: 自动按目录命名
// 魔法注释: import(/* webpackPrefetch: true */ './a')
```
