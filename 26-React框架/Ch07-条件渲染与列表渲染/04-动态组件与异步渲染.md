# 动态组件与异步渲染

现代 React 应用通常体积庞大。通过动态导入和懒加载，可以将代码拆分成更小的块，在需要时按需加载，显著提升首屏加载速度。

---

## 一、动态 import()

### 1.1 基础动态导入

ES Module 的 `import()` 函数返回 Promise，可以在运行时动态加载模块。

```jsx
// 静态导入：打包时就包含在 bundle 中
import HeavyComponent from './HeavyComponent';

// 动态导入：按需加载
async function loadComponent() {
  const module = await import('./HeavyComponent');
  return module.default;
}
```

### 1.2 条件动态导入

```jsx
function App() {
  const [Component, setComponent] = useState(null);

  const loadAdminPanel = async () => {
    const module = await import('./AdminPanel');
    setComponent(() => module.default);
  };

  return (
    <div>
      <button onClick={loadAdminPanel}>加载管理面板</button>
      {Component && <Component />}
    </div>
  );
}
```

## 二、React.lazy()

`React.lazy()` 是 React 内置的动态导入方案，让组件在首次渲染时才被加载。

### 2.1 基础用法

```jsx
import React, { Suspense } from 'react';

// React.lazy 接受一个返回 Promise 的函数
// 该 Promise 必须 resolve 一个带有 default export 的模块
const LazyComponent = React.lazy(() => import('./HeavyComponent'));

function App() {
  return (
    <Suspense fallback={<div>加载中...</div>}>
      <LazyComponent />
    </Suspense>
  );
}
```

### 2.2 多个懒加载组件

```jsx
const Dashboard = React.lazy(() => import('./pages/Dashboard'));
const Settings = React.lazy(() => import('./pages/Settings'));
const Profile  = React.lazy(() => import('./pages/Profile'));

function App() {
  const [page, setPage] = useState('dashboard');

  const pages = {
    dashboard: Dashboard,
    settings:  Settings,
    profile:   Profile,
  };

  const CurrentPage = pages[page];

  return (
    <Suspense fallback={<PageSpinner />}>
      <nav>
        <button onClick={() => setPage('dashboard')}>Dashboard</button>
        <button onClick={() => setPage('settings')}>Settings</button>
        <button onClick={() => setPage('profile')}>Profile</button>
      </nav>
      <CurrentPage />
    </Suspense>
  );
}
```

### 2.3 命名导出的懒加载

`React.lazy` 默认只能加载 `default` export。加载命名导出需要额外处理：

```jsx
// 如果组件是命名导出的
// components/Chart.js
export const LineChart = () => { /* ... */ };
export const BarChart  = () => { /* ... */ };

// 懒加载命名导出
const LineChart = React.lazy(() =>
  import('./components/Chart').then(module => ({
    default: module.LineChart,
  }))
);

const BarChart = React.lazy(() =>
  import('./components/Chart').then(module => ({
    default: module.BarChart,
  }))
);

// 或者创建通用的懒加载工具函数
function lazyNamed(importFn, name) {
  return React.lazy(() =>
    importFn().then(module => ({ default: module[name] }))
  );
}

const PieChart = lazyNamed(() => import('./components/Chart'), 'PieChart');
```

## 三、Suspense 基础

`Suspense` 是 React 的加载状态管理组件，用于包裹懒加载组件。

### 3.1 基础用法

```jsx
<Suspense fallback={<LoadingSpinner />}>
  <LazyComponent />
</Suspense>
```

### 3.2 多个懒加载组件共享 Suspense

```jsx
// 两个组件都由同一个 Suspense 管理
// 只要任一组件还在加载，就显示 fallback
function Dashboard() {
  return (
    <Suspense fallback={<div>加载仪表盘组件...</div>}>
      <div className="dashboard">
        <ChartPanel />        {/* 懒加载 */}
        <StatsPanel />        {/* 懒加载 */}
        <RecentActivity />    {/* 懒加载 */}
      </div>
    </Suspense>
  );
}
```

### 3.3 嵌套 Suspense

```jsx
function App() {
  return (
    <Suspense fallback={<FullPageSpinner />}>
      <Layout>
        {/* 主内容区有自己的加载状态 */}
        <Suspense fallback={<ContentSkeleton />}>
          <MainContent />
        </Suspense>

        {/* 侧边栏独立加载 */}
        <Suspense fallback={<SidebarSkeleton />}>
          <Sidebar />
        </Suspense>
      </Layout>
    </Suspense>
  );
}
```

### 3.4 自定义加载组件

```jsx
function PageSpinner() {
  return (
    <div className="page-spinner">
      <div className="spinner" />
      <p>页面加载中...</p>
    </div>
  );
}

function ContentSkeleton() {
  return (
    <div className="skeleton">
      <div className="skeleton-header" />
      <div className="skeleton-line" />
      <div className="skeleton-line short" />
      <div className="skeleton-line" />
    </div>
  );
}
```

## 四、错误边界与懒加载

网络请求可能失败，懒加载的组件也可能加载失败。需要错误边界来优雅处理。

### 4.1 错误边界组件

```jsx
import React from 'react';

class LazyLoadErrorBoundary extends React.Component {
  constructor(props) {
    super(props);
    this.state = { hasError: false, error: null };
  }

  static getDerivedStateFromError(error) {
    return { hasError: true, error };
  }

  componentDidCatch(error, errorInfo) {
    console.error('懒加载失败:', error, errorInfo);
  }

  render() {
    if (this.state.hasError) {
      return (
        <div className="error-fallback">
          <h2>组件加载失败</h2>
          <p>{this.state.error?.message}</p>
          <button onClick={() => {
            this.setState({ hasError: false, error: null });
          }}>
            重试
          </button>
        </div>
      );
    }

    return this.props.children;
  }
}
```

### 4.2 配合 Suspense 使用

```jsx
function App() {
  return (
    <LazyLoadErrorBoundary>
      <Suspense fallback={<PageSpinner />}>
        <LazyPage />
      </Suspense>
    </LazyLoadErrorBoundary>
  );
}
```

### 4.3 带重试的懒加载

```jsx
// 自动重试的懒加载函数
function lazyWithRetry(importFn, retries = 3, interval = 1000) {
  return React.lazy(() =>
    new Promise((resolve, reject) => {
      const attempt = (remainingRetries) => {
        importFn()
          .then(resolve)
          .catch((error) => {
            if (remainingRetries <= 0) {
              reject(error);
              return;
            }
            // 等待后重试
            setTimeout(() => attempt(remainingRetries - 1), interval);
          });
      };
      attempt(retries);
    })
  );
}

// 使用
const LazyDashboard = lazyWithRetry(() => import('./pages/Dashboard'), 3, 1000);
```

### 4.4 函数组件错误边界 (react-error-boundary)

```bash
npm install react-error-boundary
```

```jsx
import { ErrorBoundary } from 'react-error-boundary';

function ErrorFallback({ error, resetErrorBoundary }) {
  return (
    <div className="error-fallback">
      <h2>出错了</h2>
      <pre>{error.message}</pre>
      <button onClick={resetErrorBoundary}>重试</button>
    </div>
  );
}

function App() {
  return (
    <ErrorBoundary
      FallbackComponent={ErrorFallback}
      onReset={() => {
        // 重置应用状态
        window.location.reload();
      }}
    >
      <Suspense fallback={<PageSpinner />}>
        <LazyPage />
      </Suspense>
    </ErrorBoundary>
  );
}
```

## 五、条件懒加载

不是所有懒加载都需要立即发生。可以通过条件控制何时加载。

### 5.1 基于用户交互的懒加载

```jsx
const HeavyModal = React.lazy(() => import('./HeavyModal'));

function App() {
  const [showModal, setShowModal] = useState(false);

  return (
    <div>
      <button onClick={() => setShowModal(true)}>
        打开弹窗（首次点击会加载组件）
      </button>

      {showModal && (
        <Suspense fallback={<ModalSpinner />}>
          <HeavyModal onClose={() => setShowModal(false)} />
        </Suspense>
      )}
    </div>
  );
}
```

### 5.2 基于路由的懒加载

```jsx
const Home      = React.lazy(() => import('./pages/Home'));
const About     = React.lazy(() => import('./pages/About'));
const Dashboard = React.lazy(() => import('./pages/Dashboard'));
const Settings  = React.lazy(() => import('./pages/Settings'));

function App() {
  return (
    <Suspense fallback={<RouteSpinner />}>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/dashboard" element={<Dashboard />} />
        <Route path="/settings" element={<Settings />} />
      </Routes>
    </Suspense>
  );
}
```

### 5.3 基于视口的懒加载（进入视口才加载）

```jsx
function LazyOnViewport({ children, fallback = null }) {
  const [shouldLoad, setShouldLoad] = useState(false);
  const ref = useRef(null);

  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setShouldLoad(true);
          observer.disconnect();
        }
      },
      { rootMargin: '200px' } // 提前 200px 开始加载
    );

    if (ref.current) {
      observer.observe(ref.current);
    }

    return () => observer.disconnect();
  }, []);

  return (
    <div ref={ref}>
      {shouldLoad ? children : fallback}
    </div>
  );
}

// 使用
function Page() {
  return (
    <div>
      <AboveFoldContent />

      <LazyOnViewport fallback={<Skeleton />}>
        <Suspense fallback={<Spinner />}>
          <BelowFoldHeavyComponent />
        </Suspense>
      </LazyOnViewport>
    </div>
  );
}
```

## 六、预加载策略

预加载可以在用户即将需要某个组件时提前加载，消除等待时间。

### 6.1 基础预加载

```jsx
// 创建可预加载的 lazy 函数
function lazyWithPreload(importFn) {
  let cachedModule = null;
  let promise = null;

  const load = () => {
    if (cachedModule) return Promise.resolve(cachedModule);
    if (!promise) {
      promise = importFn().then(module => {
        cachedModule = module;
        return module;
      });
    }
    return promise;
  };

  // React.lazy 需要的格式
  const LazyComponent = React.lazy(() => load());

  // 暴露预加载方法
  LazyComponent.preload = load;

  return LazyComponent;
}

// 使用
const Dashboard = lazyWithPreload(() => import('./pages/Dashboard'));

function App() {
  return (
    <nav>
      {/* 鼠标悬停时预加载 */}
      <Link
        to="/dashboard"
        onMouseEnter={() => Dashboard.preload()}
        onFocus={() => Dashboard.preload()}
      >
        Dashboard
      </Link>
    </nav>
  );
}
```

### 6.2 预加载所有路由组件

```jsx
// routeConfig.js
const routes = [
  {
    path: '/',
    component: lazyWithPreload(() => import('./pages/Home')),
  },
  {
    path: '/dashboard',
    component: lazyWithPreload(() => import('./pages/Dashboard')),
  },
  {
    path: '/settings',
    component: lazyWithPreload(() => import('./pages/Settings')),
  },
];

// 在应用空闲时预加载所有路由
function preloadAllRoutes() {
  routes.forEach(route => route.component.preload());
}

// 页面加载完成后，空闲时预加载
if ('requestIdleCallback' in window) {
  requestIdleCallback(preloadAllRoutes);
} else {
  setTimeout(preloadAllRoutes, 2000);
}
```

### 6.3 基于预测的预加载

```jsx
// 根据当前页面预测用户可能访问的下一个页面
const predictions = {
  '/': ['/dashboard', '/about'],
  '/dashboard': ['/dashboard/settings', '/dashboard/analytics'],
  '/products': ['/cart', '/checkout'],
};

function useRoutePrediction(currentPath) {
  useEffect(() => {
    const likelyNext = predictions[currentPath] || [];
    likelyNext.forEach(path => {
      const route = routes.find(r => r.path === path);
      if (route) route.component.preload();
    });
  }, [currentPath]);
}
```

## 七、按路由代码分割

### 7.1 基于 React Router 的代码分割

```jsx
import { BrowserRouter, Routes, Route } from 'react-router-dom';
import { Suspense, lazy } from 'react';

// 路由级代码分割
const Home       = lazy(() => import('./pages/Home'));
const Products   = lazy(() => import('./pages/Products'));
const ProductDetail = lazy(() => import('./pages/ProductDetail'));
const Cart       = lazy(() => import('./pages/Cart'));
const Checkout   = lazy(() => import('./pages/Checkout'));
const NotFound   = lazy(() => import('./pages/NotFound'));

function App() {
  return (
    <BrowserRouter>
      <Suspense fallback={<PageSpinner />}>
        <Routes>
          <Route path="/" element={<Home />} />
          <Route path="/products" element={<Products />} />
          <Route path="/products/:id" element={<ProductDetail />} />
          <Route path="/cart" element={<Cart />} />
          <Route path="/checkout" element={<Checkout />} />
          <Route path="*" element={<NotFound />} />
        </Routes>
      </Suspense>
    </BrowserRouter>
  );
}
```

### 7.2 路由级别的细粒度 Suspense

```jsx
function App() {
  return (
    <BrowserRouter>
      <Layout>
        {/* 导航栏不需要懒加载 */}
        <Navbar />

        {/* 每个路由区域独立 Suspense */}
        <main>
          <Suspense fallback={<ContentSkeleton />}>
            <Routes>
              <Route path="/" element={<Home />} />
              <Route path="/dashboard/*" element={
                <Suspense fallback={<DashboardSkeleton />}>
                  <DashboardRoutes />
                </Suspense>
              } />
              <Route path="/admin/*" element={
                <ErrorBoundary fallback={<AdminError />}>
                  <Suspense fallback={<AdminSkeleton />}>
                    <AdminRoutes />
                  </Suspense>
                </ErrorBoundary>
              } />
            </Routes>
          </Suspense>
        </main>
      </Layout>
    </BrowserRouter>
  );
}

// 子路由也可以懒加载
const DashboardRoutes = React.lazy(() => import('./pages/dashboard/DashboardRoutes'));
const AdminRoutes     = React.lazy(() => import('./pages/admin/AdminRoutes'));
```

## 八、命名 chunk（调试友好）

默认的 chunk 名称是数字 ID，调试困难。使用魔法注释命名：

```jsx
// 使用 webpack 魔法注释命名 chunk
const Dashboard = React.lazy(() =>
  import(
    /* webpackChunkName: "dashboard" */
    /* webpackPrefetch: true */
    './pages/Dashboard'
  )
);

const Settings = React.lazy(() =>
  import(
    /* webpackChunkName: "settings" */
    './pages/Settings'
  )
);
```

Vite 中自动根据文件名命名 chunk，也可以手动指定：

```jsx
const Dashboard = React.lazy(() =>
  import(
    /* @vite-ignore */
    './pages/Dashboard'
  )
);
```

## 九、综合示例：完整的路由懒加载架构

```jsx
// App.jsx
import { BrowserRouter, Routes, Route, useLocation } from 'react-router-dom';
import { Suspense, lazy, useEffect } from 'react';
import { ErrorBoundary } from 'react-error-boundary';

// 布局组件（非懒加载，首屏需要）
import MainLayout from './layouts/MainLayout';

// 页面级懒加载
const HomePage       = lazy(() => import('./pages/Home'));
const ProductsPage   = lazy(() => import('./pages/Products'));
const ProductPage    = lazy(() => import('./pages/Product'));
const CartPage       = lazy(() => import('./pages/Cart'));
const CheckoutPage   = lazy(() => import('./pages/Checkout'));
const AccountPage    = lazy(() => import('./pages/Account'));
const NotFoundPage   = lazy(() => import('./pages/NotFound'));

// 加载组件
function PageLoader() {
  return (
    <div className="page-loader">
      <div className="spinner" />
    </div>
  );
}

// 错误回退
function PageError({ error, resetErrorBoundary }) {
  return (
    <div className="page-error">
      <h2>页面加载失败</h2>
      <p>{error.message}</p>
      <button onClick={resetErrorBoundary}>重试</button>
    </div>
  );
}

// 路由变化时回到顶部
function ScrollToTop() {
  const { pathname } = useLocation();
  useEffect(() => { window.scrollTo(0, 0); }, [pathname]);
  return null;
}

function App() {
  return (
    <BrowserRouter>
      <ScrollToTop />
      <ErrorBoundary FallbackComponent={PageError}>
        <Suspense fallback={<PageLoader />}>
          <Routes>
            <Route element={<MainLayout />}>
              <Route path="/" element={<HomePage />} />
              <Route path="/products" element={<ProductsPage />} />
              <Route path="/products/:id" element={<ProductPage />} />
              <Route path="/cart" element={<CartPage />} />
              <Route path="/checkout" element={<CheckoutPage />} />
              <Route path="/account/*" element={<AccountPage />} />
              <Route path="*" element={<NotFoundPage />} />
            </Route>
          </Routes>
        </Suspense>
      </ErrorBoundary>
    </BrowserRouter>
  );
}

export default App;
```

## 总结

| 技术 | 用途 | 推荐场景 |
|---|---|---|
| `React.lazy()` | 组件级懒加载 | 所有非首屏组件 |
| `Suspense` | 声明式加载状态 | 配合 lazy 使用 |
| 错误边界 | 处理加载失败 | 所有 lazy 组件 |
| 预加载 | 消除加载等待 | 导航链接 hover |
| 路由分割 | 按页面拆分代码 | 多页面应用 |
| 视口懒加载 | 进入视口才加载 | 长页面中的重型组件 |
