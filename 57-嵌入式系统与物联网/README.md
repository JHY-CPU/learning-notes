# 55-嵌入式系统与物联网

嵌入式系统与物联网（IoT）是当今信息技术领域中最具活力的方向之一。嵌入式系统以专用计算机系统为核心，广泛应用于消费电子、工业控制、汽车电子、医疗设备等领域。物联网则将嵌入式设备连接到互联网，实现万物互联。本笔记系统梳理嵌入式系统与物联网的核心知识体系。

## 学习路径建议

### 第一阶段：硬件基础
- 掌握微控制器架构（ARM Cortex-M系列）
- 理解GPIO、中断、定时器等基本外设
- 熟悉UART、SPI、I2C等通信接口

### 第二阶段：软件平台
- 学习实时操作系统（RTOS）的基本原理
- 掌握FreeRTOS或RT-Thread的任务管理与同步机制
- 了解嵌入式Linux的基本概念

### 第三阶段：物联网应用
- 理解物联网三层架构
- 掌握MQTT、CoAP等核心协议
- 熟悉蓝牙、LoRa、NB-IoT等无线通信技术

### 第四阶段：深入优化
- 学习低功耗设计方法
- 掌握能量收集与电池管理技术
- 实践边缘计算与IoT平台集成

## 常见开发板与平台

- **STM32系列**：意法半导体ARM Cortex-M微控制器，生态完善
- **ESP32/ESP8266**：乐鑫科技WiFi/BLE SoC，性价比高
- **Arduino**：开源硬件平台，适合快速原型开发
- **Raspberry Pi**：Linux单板计算机，适合网关和边缘计算
- **Nordic nRF系列**：低功耗蓝牙（BLE）首选方案
