# 输入系统

## 核心概念

Unity有两套输入系统：旧的Input Manager（内置）和新的Input System Package。旧系统简单易用但功能有限，新系统支持动作映射、多设备适配和组合输入。新系统采用事件驱动架构，通过Action Map组织输入逻辑，天然支持多玩家和多设备切换。

## 旧Input Manager详解

基于轮询模式，每帧在Update中读取输入状态。底层调用平台API轮询硬件状态：

```csharp
public class OldInputExample : MonoBehaviour
{
    void Update()
    {
        // ========== 按键检测 ==========
        if (Input.GetKeyDown(KeyCode.Space))    // 按下瞬间（单次触发）
            Debug.Log("空格键按下");
        if (Input.GetKey(KeyCode.W))             // 持续按住（每帧检测）
            transform.Translate(Vector3.forward * Time.deltaTime * 5f);
        if (Input.GetKeyUp(KeyCode.Space))       // 松开瞬间
            Debug.Log("空格键松开");

        // ========== 轴输入（带平滑处理） ==========
        float h = Input.GetAxis("Horizontal");   // -1 到 1，有渐变过渡
        float v = Input.GetAxis("Vertical");
        float rawH = Input.GetAxisRaw("Horizontal"); // -1, 0, 1，无渐变（适合即时响应）

        Vector3 move = new Vector3(h, 0, v).normalized * Time.deltaTime * speed;
        transform.Translate(move);

        // ========== 鼠标输入 ==========
        if (Input.GetMouseButtonDown(0))  // 左键按下
            Debug.Log("鼠标左键");
        if (Input.GetMouseButton(1))       // 右键持续按住
            Debug.Log("右键按住中");
        if (Input.GetMouseButtonUp(0))     // 左键松开
            Debug.Log("左键松开");
        float mouseX = Input.GetAxis("Mouse X"); // 鼠标水平移动量
        float mouseY = Input.GetAxis("Mouse Y"); // 鼠标垂直移动量
        Vector3 mousePos = Input.mousePosition;  // 屏幕坐标（像素）

        // ========== 触摸输入（移动端） ==========
        if (Input.touchCount > 0)
        {
            Touch touch = Input.GetTouch(0);
            if (touch.phase == TouchPhase.Began)
                Debug.Log($"触摸开始: {touch.position}");
            else if (touch.phase == TouchPhase.Moved)
                Debug.Log($"触摸移动: {touch.deltaPosition}");
            else if (touch.phase == TouchPhase.Ended)
                Debug.Log($"触摸结束");
        }

        // ========== 手柄输入 ==========
        float trigger = Input.GetAxis("LT");  // 左扳机
        bool jump = Input.GetButtonDown("Joystick1Button0"); // A键
        // 手柄轴映射: Left Stick X/Y, Right Stick X/Y, LT, RT
    }
}
```

### 轴映射配置详解

Edit -> Project Settings -> Input Manager可配置轴映射：
- Horizontal / Vertical: WASD + 方向键移动
- Fire1 / Fire2 / Fire3: Ctrl/Alt/鼠标动作按钮
- Mouse X / Mouse Y: 鼠标移动增量
- Mouse ScrollWheel: 滚轮
- 每个轴可设置: 正键/负键, 死区(dead zone), 灵敏度(sensitivity), 重力(gravity)
- 最多支持28个轴映射


## 新Input System详解

新系统采用事件驱动+动作映射模式，支持多设备自动切换。架构分为三层：

```
硬件设备层: Keyboard, Mouse, Gamepad, Touch, XR
    ↓ (状态变化)
动作映射层: Action Map → Action → Binding
    ↓ (触发)
业务逻辑层: performed/canceled 回调
```

### InputAction资产配置

1. 创建Input Actions资产: Assets -> Create -> Input Actions
2. 定义Action Map（如Player、UI）
3. 每个Map下添加Action（如Move、Jump、Attack）
4. 为Action添加Binding（绑定到具体按键/轴）
5. 生成C#类: Inspector中勾选"Generate C# Class"

```csharp
using UnityEngine.InputSystem;

public class NewInputExample : MonoBehaviour
{
    private PlayerInputActions inputActions;
    private Vector2 moveInput;
    private bool jumpPressed;
    private bool attackPressed;

    void Awake()
    {
        inputActions = new PlayerInputActions();

        // ========== 事件绑定方式 ==========

        // performed: 动作触发时调用（按下、值变化等）
        inputActions.Player.Move.performed += ctx =>
        {
            moveInput = ctx.ReadValue<Vector2>();
        };

        // canceled: 动作取消时调用（松开、值归零）
        inputActions.Player.Move.canceled += ctx =>
        {
            moveInput = Vector2.zero;
        };

        // 按钮类动作
        inputActions.Player.Jump.performed += ctx => jumpPressed = true;
        inputActions.Player.Attack.performed += ctx => attackPressed = true;
    }

    void OnEnable() => inputActions.Player.Enable();
    void OnDisable() => inputActions.Player.Disable();

    void Update()
    {
        // 移动
        transform.Translate(new Vector3(moveInput.x, 0, moveInput.y)
            * Time.deltaTime * 5f);

        // 跳跃
        if (jumpPressed)
        {
            GetComponent<Rigidbody>().AddForce(Vector3.up * 8f, ForceMode.Impulse);
            jumpPressed = false;
        }

        // 攻击
        if (attackPressed)
        {
            PerformAttack();
            attackPressed = false;
        }
    }
}
```

### Value vs Button vs Pass-Through

```csharp
public class InputActionTypes : MonoBehaviour
{
    private PlayerInputActions inputActions;

    void Awake()
    {
        inputActions = new PlayerInputActions();

        // Value类型: 持续报告当前值（如摇杆轴）
        inputActions.Player.Move.performed += ctx =>
        {
            Vector2 value = ctx.ReadValue<Vector2>();
            // 每帧都可能触发，值是当前状态
        };

        // Button类型: 有pressed/released概念
        inputActions.Player.Jump.performed += ctx =>
        {
            // 只在按下瞬间触发
            float pressPoint = ctx.control.EvaluateMagnitude();
            // pressPoint: 0-1，按钮按下深度
        };

        // Pass-Through类型: 不经过Action处理，直接传递
        // 适合自定义处理所有原始输入
    }
}
```

### Device检测与多设备支持

```csharp
public class DeviceManager : MonoBehaviour
{
    void Start()
    {
        // 监听设备连接/断开
        InputSystem.onDeviceChange += (device, change) =>
        {
            switch (change)
            {
                case InputDeviceChange.Added:
                    Debug.Log($"设备连接: {device.displayName}");
                    break;
                case InputDeviceChange.Removed:
                    Debug.Log($"设备断开: {device.displayName}");
                    break;
                case InputDeviceChange.ConfigurationChanged:
                    Debug.Log($"设备配置变化: {device.displayName}");
                    break;
            }
        };
    }

    void Update()
    {
        // ========== 检测当前活跃设备 ==========

        // 键盘
        var keyboard = Keyboard.current;
        if (keyboard != null && keyboard.spaceKey.wasPressedThisFrame)
        {
            Debug.Log("使用键盘");
        }

        // 鼠标
        var mouse = Mouse.current;
        if (mouse != null)
        {
            Vector2 mousePos = mouse.position.ReadValue();
            Vector2 mouseDelta = mouse.delta.ReadValue();
        }

        // 手柄
        var gamepad = Gamepad.current;
        if (gamepad != null)
        {
            Vector2 leftStick = gamepad.leftStick.ReadValue();
            Vector2 rightStick = gamepad.rightStick.ReadValue();
            float leftTrigger = gamepad.leftTrigger.ReadValue();
            float rightTrigger = gamepad.rightTrigger.ReadValue();

            if (gamepad.buttonSouth.wasPressedThisFrame)
                Debug.Log("A/X按钮按下");

            // 手柄震动
            gamepad.SetMotorSpeeds(0.5f, 0.5f);
            // 停止震动
            gamepad.SetMotorSpeeds(0f, 0f);
        }

        // 触摸屏
        var touchscreen = Touchscreen.current;
        if (touchscreen != null)
        {
            var touch = touchscreen.primaryTouch;
            if (touch.press.isPressed)
            {
                Vector2 pos = touch.position.ReadValue();
            }
        }
    }
}
```

### PlayerInput组件

PlayerInput组件是新Input System的高层封装，简化多玩家管理：

```csharp
// PlayerInput组件配置方式:
// 1. Inspector中指定Input Actions资产
// 2. 选择通知方式: Unity Events / C# Events / Send Messages / Invoke Unity Events
// 3. 自动管理Action Map切换

public class PlayerInputHandler : MonoBehaviour
{
    private PlayerInput playerInput;
    private Vector2 moveInput;

    void Awake()
    {
        playerInput = GetComponent<PlayerInput>();

        // 监听Action Map切换事件
        playerInput.onActionTriggered += OnActionTriggered;
        playerInput.onControlsChanged += OnControlsChanged;
    }

    void OnActionTriggered(InputAction.CallbackContext ctx)
    {
        // 根据Action名称分发
        switch (ctx.action.name)
        {
            case "Move":
                moveInput = ctx.ReadValue<Vector2>();
                break;
            case "Jump":
                if (ctx.performed) Jump();
                break;
        }
    }

    void OnControlsChanged(PlayerInput input)
    {
        Debug.Log($"控制方案切换: {input.currentControlScheme}");
        // 例如: 切换到"Gamepad"或"Keyboard&Mouse"
    }

    // 运行时切换Action Map
    void SwitchToUIMode()
    {
        playerInput.SwitchCurrentActionMap("UI");
    }

    void SwitchToPlayerMode()
    {
        playerInput.SwitchCurrentActionMap("Player");
    }
}
```

### 按键重绑定（Rebinding）

```csharp
public class RebindingUI : MonoBehaviour
{
    private PlayerInputActions inputActions;
    [SerializeField] private TextMeshProUGUI bindingText;

    void Awake()
    {
        inputActions = new PlayerInputActions();
    }

    // 开始重绑定某个Action的某个Binding
    public void StartRebind(string actionName, int bindingIndex)
    {
        var action = inputActions.asset.FindAction(actionName);
        if (action == null) return;

        // 暂时禁用Action
        action.Disable();

        // 开始交互式重绑定
        var rebindOperation = action.PerformInteractiveRebinding(bindingIndex)
            .WithControlsExcluding("Mouse") // 排除鼠标
            .OnMatchWaitForAnother(0.1f)    // 等待组合键
            .OnComplete(operation =>
            {
                operation.Dispose();
                action.Enable();

                // 更新UI显示
                UpdateBindingDisplay(action, bindingIndex);

                // 保存重绑定结果
                SaveBindings();
            })
            .OnCancel(operation =>
            {
                operation.Dispose();
                action.Enable();
            });

        bindingText.text = "按下新按键...";
    }

    void UpdateBindingDisplay(InputAction action, int bindingIndex)
    {
        string path = action.bindings[bindingIndex].effectivePath;
        string displayName = InputControlPath.ToHumanReadableString(
            path, InputControlPath.HumanReadableStringOptions.OmitDevice);
        bindingText.text = displayName;
    }

    // 持久化重绑定结果
    void SaveBindings()
    {
        string rebinds = inputActions.SaveBindingOverridesAsJson();
        PlayerPrefs.SetString("InputBindings", rebinds);
        PlayerPrefs.Save();
    }

    // 加载重绑定结果
    void LoadBindings()
    {
        string rebinds = PlayerPrefs.GetString("InputBindings", "");
        if (!string.IsNullOrEmpty(rebinds))
            inputActions.LoadBindingOverridesFromJson(rebinds);
    }

    // 重置为默认绑定
    public void ResetBindings()
    {
        inputActions.RemoveAllBindingOverrides();
        PlayerPrefs.DeleteKey("InputBindings");
    }
}
```

```csharp
public class PlatformInput : MonoBehaviour
{
    // 抽象层：统一不同平台的输入
    public Vector2 GetMovement()
    {
#if UNITY_EDITOR || UNITY_STANDALONE
        // PC使用键盘/手柄
        float h = Input.GetAxis("Horizontal");
        float v = Input.GetAxis("Vertical");
        return new Vector2(h, v).normalized;
#elif UNITY_ANDROID || UNITY_IOS
        // 移动端使用触摸或虚拟摇杆
        if (Input.touchCount > 0)
            return VirtualJoystick.GetAxis();
        return Vector2.zero;
#elif UNITY_PS4 || UNITY_PS5
        // 主机使用手柄
        return new Vector2(
            Input.GetAxis("PS4_LeftStickX"),
            Input.GetAxis("PS4_LeftStickY")
        ).normalized;
#endif
    }
}

// 虚拟摇杆实现
public class VirtualJoystick : MonoBehaviour, IPointerDownHandler, IDragHandler, IPointerUpHandler
{
    [SerializeField] private RectTransform background;
    [SerializeField] private RectTransform handle;

    private Vector2 inputVector = Vector2.zero;

    public void OnPointerDown(PointerEventData eventData)
    {
        OnDrag(eventData);
    }

    public void OnDrag(PointerEventData eventData)
    {
        Vector2 localPoint;
        if (RectTransformUtility.ScreenPointToLocalPointInRectangle(
            background, eventData.position, eventData.pressEventCamera, out localPoint))
        {
            Vector2 normalized = localPoint / (background.sizeDelta * 0.5f);
            inputVector = normalized.magnitude > 1 ? normalized.normalized : normalized;
            handle.anchoredPosition = inputVector * (background.sizeDelta * 0.5f) * 0.8f;
        }
    }

    public void OnPointerUp(PointerEventData eventData)
    {
        inputVector = Vector2.zero;
        handle.anchoredPosition = Vector2.zero;
    }

    public Vector2 GetAxis() => inputVector;
}
```

## 输入缓冲系统

格斗游戏需要输入缓冲来允许玩家提前输入招式：

```csharp
public class InputBuffer : MonoBehaviour
{
    [System.Serializable]
    public struct BufferedInput
    {
        public string actionName;
        public float timestamp;
    }

    private Queue<BufferedInput> buffer = new Queue<BufferedInput>();
    [SerializeField] private float bufferWindow = 0.15f; // 缓冲窗口（秒）

    void Update()
    {
        // 清除过期的缓冲输入
        while (buffer.Count > 0 && Time.time - buffer.Peek().timestamp > bufferWindow)
            buffer.Dequeue();
    }

    public void BufferInput(string action)
    {
        buffer.Enqueue(new BufferedInput
        {
            actionName = action,
            timestamp = Time.time
        });
    }

    // 检查是否有特定缓冲输入
    public bool ConsumeBuffer(string action)
    {
        var items = buffer.ToArray();
        for (int i = 0; i < items.Length; i++)
        {
            if (items[i].actionName == action)
            {
                // 重建队列（移除已消耗的输入）
                buffer.Clear();
                for (int j = 0; j < items.Length; j++)
                {
                    if (j != i)
                        buffer.Enqueue(items[j]);
                }
                return true;
            }
        }
        return false;
    }
}

// 组合技输入检测
public class ComboDetector : MonoBehaviour
{
    [SerializeField] private float comboWindow = 0.5f;
    private List<string> recentInputs = new List<string>();
    private float lastInputTime;

    void Update()
    {
        // 清除超时的输入
        if (Time.time - lastInputTime > comboWindow)
            recentInputs.Clear();
    }

    public void RecordInput(string direction, string button)
    {
        string combo = direction + "_" + button;
        recentInputs.Add(combo);
        lastInputTime = Time.time;

        // 检查组合技
        CheckCombo();
    }

    void CheckCombo()
    {
        // 例如: 检查 "下前拳" (Hadouken)
        if (HasSequence("Down_Punch", "Forward_Punch"))
            ExecuteHadouken();
    }

    bool HasSequence(params string[] sequence)
    {
        if (recentInputs.Count < sequence.Length) return false;
        int startIdx = recentInputs.Count - sequence.Length;
        for (int i = 0; i < sequence.Length; i++)
        {
            if (recentInputs[startIdx + i] != sequence[i])
                return false;
        }
        return true;
    }

    void ExecuteHadouken() { Debug.Log("Hadouken!"); }
}
```

## 常见陷阱与最佳实践

1. **旧系统必须在Update中读取**: Input在FixedUpdate中可能丢失输入事件，因为FixedUpdate可能跳帧
2. **新系统用performed/canceled**: 不要每帧轮询，用事件驱动更精确且节省性能
3. **轴的死区问题**: 摇杆漂移会产生微小输入，需设置死区阈值（通常0.1-0.2）
4. **跨平台输入差异**: 移动端没有物理键盘，需提供虚拟按钮或手势替代
5. **输入缓冲**: 格斗/动作游戏需要输入缓冲来提升手感，窗口通常100-200ms
6. **Action Map切换**: UI模式下需切换到UI Action Map，防止游戏输入干扰UI操作
7. **重绑定持久化**: 使用PlayerPrefs或文件保存重绑定结果，确保重启后保留
8. **新旧系统不兼容**: 同一项目中不要混用两套输入系统，会产生冲突

## 性能分析

| 操作 | 开销 | 说明 |
|------|------|------|
| Input.GetKey (旧系统) | 极低 | 直接查询状态缓存 |
| Input.GetKeyDown | 低 | 需要检测状态变化 |
| InputAction.performed | 低 | 事件驱动，仅触发时执行 |
| PlayerInput组件 | 低中 | 有额外的管理开销 |
| PerformInteractiveRebinding | 中高 | 交互式重绑定有UI和监听开销 |
| Touchscreen查询 | 低 | 每帧查询触摸状态 |

## 与其他系统的关联

- **UGUI**: EventSystem依赖输入系统处理UI交互（点击、拖拽）
- **动画系统**: 输入触发的动画状态切换
- **多人游戏**: 新Input System原生支持多玩家设备分配
- **Cinemachine**: 相机输入轴通过Input System获取
