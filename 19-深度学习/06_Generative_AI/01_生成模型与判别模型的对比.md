# 01_生成模型与判别模型的对比

## 核心概念

- **生成模型**：学习数据分布 $P(X)$（或联合分布 $P(X,Y)$），能够从学习到的分布中采样生成新样本。典型代表有 GAN、VAE、扩散模型。
- **判别模型**：学习类别边界，即直接建模条件概率 $P(Y|X)$，用于分类或回归任务。典型代表有 SVM、逻辑回归、CNN 分类器。
- **核心区别**：生成模型关注"数据是如何产生的"，判别模型关注"决策边界在哪里"。生成模型能做"创造"而判别模型只能做"判断"。
- **对偶关系**：从贝叶斯视角，生成模型 + 先验可通过贝叶斯定理得到判别模型：$P(Y|X) = P(X|Y)P(Y)/P(X)$，反之不成立（判别模型不包含数据分布信息）。
- **半监督学习**：生成模型可以利用无标签数据学习数据分布，从而提升少量标签数据下的分类性能，这是判别模型难以做到的。
- **异常检测**：生成模型可以估计样本的似然值，低似然样本被视为异常；判别模型需要重新训练才能处理未见过的类别。

## 数学推导

**联合分布 vs 条件分布**：

生成模型建模联合分布 $P(X,Y)$ 或数据分布 $P(X)$：

$$
P(X) = \int P(X|Y)P(Y) dY \quad \text{（连续情形）}
$$

判别模型直接建模条件分布：

$$
P(Y|X) = \frac{P(X|Y)P(Y)}{P(X)}
$$

**最大似然估计的差异**：

判别模型最大化条件对数似然：

$$
\theta^* = \arg\max_\theta \sum_{i=1}^N \log P_\theta(y_i | x_i)
$$

生成模型最大化联合对数似然（或边缘对数似然）：

$$
\theta^* = \arg\max_\theta \sum_{i=1}^N \log P_\theta(x_i, y_i) \quad \text{或} \quad \theta^* = \arg\max_\theta \sum_{i=1}^N \log P_\theta(x_i)
$$

## 直观理解

- **判别模型**就像一位艺术品鉴定专家，他能精准分辨一幅画是真迹还是赝品，但他自己不会画画。
- **生成模型**就像一位画家，他学习了大量画作的风格和技巧后，能够创作出全新的作品。
- 如果用一个比喻：判别模型在学习"猫和狗的区别"，而生成模型在学习"猫到底是什么样子"。前者只需要找到决策边界，后者需要掌握完整的概率分布。
- **信息含量**：生成模型包含更多信息（完整的分布），因此更难训练，但能做的事情更多（生成、填充、异常检测等）。

## 代码示例

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.distributions import Normal

# 判别模型示例：简单的二分类器
class DiscriminativeModel(nn.Module):
    def __init__(self, input_dim=2):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, 32),
            nn.ReLU(),
            nn.Linear(32, 1),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        return self.net(x)  # P(Y=1|X)

# 生成模型示例：高斯混合模型（用神经网络参数化）
class GenerativeModel(nn.Module):
    def __init__(self, latent_dim=2, data_dim=2):
        super().__init__()
        self.latent_dim = latent_dim
        self.generator = nn.Sequential(
            nn.Linear(latent_dim, 32),
            nn.ReLU(),
            nn.Linear(32, data_dim)
        )
    
    def forward(self, z):
        return self.generator(z)  # 从隐变量生成数据
    
    def sample(self, n):
        z = torch.randn(n, self.latent_dim)
        return self.forward(z)

# 使用示例
disc = DiscriminativeModel()
gen = GenerativeModel()
x_real = torch.randn(64, 2)
print(f"判别模型输出: {disc(x_real).shape}")
print(f"生成模型采样: {gen.sample(10).shape}")
```

## 深度学习关联

- **GAN 框架**：生成器（生成模型）和判别器（判别模型）的直接对抗，完美体现了两种模型的互补性。
- **自监督学习**：MAE、SimCLR 等方法利用生成式目标（掩码重建、对比预测）学习表征，再在下游任务中作为判别模型微调，模糊了两者的边界。
- **扩散模型用于分类**：最新的研究（如 Diffusion Classifier）直接用预训练扩散模型计算条件似然 $P(x|c)$ 进行分类，证明了生成模型完全可以替代判别模型。
- **多模态大模型**：GPT-4V、Gemini 等模型同时具备生成能力（生成文本/图像）和判别能力（理解、分类、推理），代表了两种范式的融合趋势。
