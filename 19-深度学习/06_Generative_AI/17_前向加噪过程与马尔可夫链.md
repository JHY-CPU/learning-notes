# 17_前向加噪过程与马尔可夫链

## 核心概念

- **马尔可夫链**：前向加噪过程是一个马尔可夫链——$x_t$ 只依赖于 $x_{t-1}$，不依赖 $t-1$ 之前的状态。这简化了概率建模。
- **高斯转移核**：每一步的加噪是高斯转移：$q(x_t|x_{t-1}) = \mathcal{N}(x_t; \sqrt{1-\beta_t}x_{t-1}, \beta_t I)$。
- **方差调度 (Variance Schedule)**：$\beta_1, \beta_2, ..., \beta_T$ 决定了每一步添加噪声的强度。常见的调度包括线性、余弦、平方根等。
- **闭式跳跃 (Closed-form Jump)**：马尔可夫链的性质使得 $x_t$ 可以直接从 $x_0$ 计算，无需迭代 $t$ 步：$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1-\bar{\alpha}_t}\epsilon$。
- **信噪比 (Signal-to-Noise Ratio, SNR)**：在时间步 $t$，$x_t$ 的信噪比为 $\text{SNR}(t) = \bar{\alpha}_t / (1-\bar{\alpha}_t)$，随着 $t$ 增加单调递减。
- **稳态分布**：当 $\bar{\alpha}_T \to 0$ 时，$x_T$ 的分布趋近于标准正态分布 $\mathcal{N}(0, I)$，与原始数据 $x_0$ 无关。

## 数学推导

**马尔可夫链的详细定义**：

前向过程定义了联合分布 $q(x_{0:T})$：

$$
q(x_{0:T}) = q(x_0) \prod_{t=1}^T q(x_t|x_{t-1})
$$

转移核：

$$
q(x_t|x_{t-1}) = \mathcal{N}(x_t; \sqrt{1-\beta_t} x_{t-1}, \beta_t I)
$$

**重参数化推导**：

$$
x_t = \sqrt{1-\beta_t} x_{t-1} + \sqrt{\beta_t} \epsilon_{t-1}, \quad \epsilon_{t-1} \sim \mathcal{N}(0, I)
$$

递推展开：

$$
x_t = \sqrt{\alpha_t \alpha_{t-1} ... \alpha_1} x_0 + \sqrt{1 - \alpha_t \alpha_{t-1} ... \alpha_1} \bar{\epsilon}
$$

其中 $\bar{\epsilon}$ 是合并后的高斯噪声（高斯变量的加权和仍为高斯），$\alpha_t = 1-\beta_t$。

定义 $\bar{\alpha}_t = \prod_{s=1}^t \alpha_s$：

$$
x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)
$$

**对数信噪比 (log SNR)**：

$$
\log\text{SNR}(t) = \log \frac{\bar{\alpha}_t}{1-\bar{\alpha}_t}
$$

$\log\text{SNR}(t)$ 从初始的大值（主要由信号主导）单调递减到 $t=T$ 时的大负值（主要由噪声主导）。

**不同噪声调度的对比**：

- 线性调度：$\beta_t = \beta_1 + (t-1)\frac{\beta_T - \beta_1}{T-1}$ —— 信号衰减快
- 余弦调度：$\bar{\alpha}_t = \frac{f(t)}{f(0)}$，$f(t) = \cos^2\left(\frac{t/T + s}{1+s} \cdot \frac{\pi}{2}\right)$ —— 信号衰减慢，信息保留更长
- 平方根调度：$\beta_t = (\sqrt{\beta_1} + (t-1)\frac{\sqrt{\beta_T} - \sqrt{\beta_1}}{T-1})^2$

## 直观理解

- **马尔可夫链假设 = 每一步只取一滴毒药**：图像从清晰到完全被噪声淹没的过程，马尔可夫假设意味着"今天的图像只取决于昨天的图像，和前天无关"。这很合理——每次只加一点点噪声，不需要记住整个过程。
- **闭式跳跃的关键好处**：如果没有闭式跳跃公式，训练时需要依次计算 1000 步才能得到 $x_{1000}$——完全不可行。闭式公式让我们可以从 $x_0$ 直接跳到任意 $t$，训练效率提升了 1000 倍。
- **信噪比的类比**：想像你在打电话，信号逐渐被噪音干扰。初始时（$t=0$）你说话很清楚（SNR 高），随着时间推移噪音越来越大（SNR 低），最后完全听不到你的声音（$x_T \approx$ 纯噪音）。
- **方差调度的意义**：线性调度就像匀速加噪，但因为图像内容在早期被快速破坏，后期加噪的实际效果不大。余弦调度则在早期加噪更慢，给内容更多"存在时间"。

## 代码示例

```python
import torch
import torch.nn.functional as F
import math
import matplotlib.pyplot as plt

class MarkovForwardProcess:
    """前向加噪马尔可夫链的完整实现"""
    def __init__(self, timesteps=1000, schedule_type='linear'):
        self.timesteps = timesteps
        self.schedule_type = schedule_type
        self._init_schedule()
    
    def _init_schedule(self):
        """初始化噪声调度"""
        if self.schedule_type == 'linear':
            self.betas = torch.linspace(1e-4, 0.02, self.timesteps)
        elif self.schedule_type == 'cosine':
            s = 0.008
            steps = torch.linspace(0, self.timesteps, self.timesteps + 1)
            f_t = torch.cos((steps / self.timesteps + s) / (1 + s) * math.pi / 2) ** 2
            alphas_bar = f_t / f_t[0]
            self.betas = 1 - alphas_bar[1:] / alphas_bar[:-1]
            self.betas = torch.clip(self.betas, 0.0, 0.999)
        elif self.schedule_type == 'sqrt':
            self.betas = (torch.linspace(
                math.sqrt(1e-4), math.sqrt(0.02), self.timesteps
            ) ** 2)
        
        self.alphas = 1.0 - self.betas
        self.alpha_bars = torch.cumprod(self.alphas, dim=0)
        self.sqrt_alpha_bars = torch.sqrt(self.alpha_bars)
        self.sqrt_one_minus_alpha_bars = torch.sqrt(1.0 - self.alpha_bars)
    
    def q_sample(self, x_0, t, noise=None):
        """q(x_t|x_0): 从 x_0 直接计算 x_t"""
        if noise is None:
            noise = torch.randn_like(x_0)
        
        # 维度适配
        sqrt_alpha_bar = self.sqrt_alpha_bars[t].view(-1, 1, 1, 1)
        sqrt_one_minus = self.sqrt_one_minus_alpha_bars[t].view(-1, 1, 1, 1)
        
        return sqrt_alpha_bar * x_0 + sqrt_one_minus * noise, noise
    
    def q_sample_step(self, x_t_1, t):
        """q(x_t|x_{t-1}): 马尔可夫链的逐步转移"""
        beta = self.betas[t]
        noise = torch.randn_like(x_t_1)
        return torch.sqrt(1 - beta) * x_t_1 + torch.sqrt(beta) * noise, noise
    
    def compute_snr(self, t):
        """计算时间步 t 的信噪比"""
        return self.alpha_bars[t] / (1.0 - self.alpha_bars[t])
    
    def simulate_markov_chain(self, x_0):
        """模拟完整的马尔可夫链（逐步加噪）"""
        chain = [x_0]
        x_t = x_0
        for t in range(self.timesteps):
            x_t, _ = self.q_sample_step(x_t, t)
            chain.append(x_t)
        return chain  # [x_0, x_1, ..., x_T]
    
    def compare_jump_vs_iterative(self, x_0, t):
        """对比闭式跳跃与迭代计算的结果（应一致）"""
        # 闭式跳跃
        x_t_jump, noise = self.q_sample(x_0, torch.tensor([t]))
        
        # 迭代计算
        x_t_iter = x_0
        for i in range(t):
            x_t_iter, _ = self.q_sample_step(x_t_iter, i)
        
        diff = (x_t_jump - x_t_iter).abs().mean().item()
        return diff, x_t_jump, x_t_iter

# 演示前向过程
process = MarkovForwardProcess(timesteps=100, schedule_type='linear')

# 打印各阶段的噪声强度
print("=== 前向加噪过程的噪声分析 ===")
for t in [0, 10, 25, 50, 75, 99]:
    beta_t = process.betas[t].item()
    alpha_bar_t = process.alpha_bars[t].item()
    snr_t = process.compute_snr(t).item()
    signal_ratio = alpha_bar_t  # 信号保留比例
    noise_ratio = 1 - alpha_bar_t  # 噪声占比
    
    print(f"t={t:3d}: beta={beta_t:.5f}, alpha_bar={alpha_bar_t:.4f}, "
          f"SNR={snr_t:.3f}, 信号={signal_ratio:.1%}, 噪声={noise_ratio:.1%}")

# 验证闭式跳跃与迭代计算的一致性
x_0 = torch.randn(1, 3, 32, 32)
diff, _, _ = process.compare_jump_vs_iterative(x_0, t=50)
print(f"\n闭式跳跃 vs 迭代计算的差异: {diff:.8f} (应接近 0)")

# 计算不同调度下的信噪比衰减
print("\n=== 不同噪声调度下的 SNR 对比 ===")
for schedule in ['linear', 'cosine', 'sqrt']:
    p = MarkovForwardProcess(timesteps=100, schedule_type=schedule)
    snr_t_50 = p.compute_snr(torch.tensor(50)).item()
    print(f"{schedule:8s}: t=50 时 SNR = {snr_t_50:.3f}")
```

## 深度学习关联

- **噪声调度的学习**：后续工作（如 VDM, Diffusion with Learned Noise Schedule）将噪声调度作为可学习参数，让模型自己决定每个时间步的最佳加噪量。
- **连续时间扩散 (Score SDE)**：将离散的马尔可夫链推广为连续的随机微分方程，噪声调度对应于 SDE 的扩散系数函数 $\beta(t)$。
- **非马尔可夫前向过程 (DDIM)**：DDIM 证明了马尔可夫假设不是必须的——通过设计非马尔可夫的前向过程族，可以用同一个训练好的 DDPM 模型做更快的确定性采样。
- **变分下界视角**：前向过程的马尔可夫性质使得变分下界（ELBO）可以用 KL 散度项的和来表示，每项对应一个时间步，可以逐项分析训练质量。
