# 33_时间序列特征：滞后项与滑动窗口

## 核心概念

- **时间序列特征工程**：将时间序列数据转化为监督学习格式，通过构造滞后特征、滑动窗口统计量等，让标准 ML 模型能处理时间依赖性。
- **滞后项 (Lag Features)**：将过去时刻的观测值作为当前时刻的特征。$Lag_k = y_{t-k}$ 表示 $k$ 个时间步前的值。
- **滑动窗口统计量**：在固定大小的窗口内计算统计量（均值、标准差、最小值、最大值、斜率等）作为特征，捕捉局部趋势和波动性。
- **差分 (Differencing)**：$y'_t = y_t - y_{t-1}$，用于去除趋势，使非平稳序列变得平稳。
- **时间编码**：将时间本身作为特征——小时、星期几、月份、节假日标志等，捕捉周期性模式。
- **信息泄露风险**：时间序列特征构造时必须确保不使用未来信息——滑动窗口只能使用当前时间点"之前"的数据。

## 数学推导

**滞后特征**：
给定时间序列 $\{y_1, y_2, \dots, y_T\}$，构造监督学习样本：
$$
X_t = [y_{t-1}, y_{t-2}, \dots, y_{t-L}], \quad y_t \text{ 为目标}
$$
其中 $L$ 是最大滞后阶数。

**滑动平均 (Moving Average)**：
$$
MA_k(t) = \frac{1}{k} \sum_{i=1}^k y_{t-i}
$$

**指数加权移动平均 (EWMA)**：
$$
EWMA_\alpha(t) = \alpha y_{t-1} + (1-\alpha) EWMA_\alpha(t-1)
$$
$\alpha$ 越大，近期观测的权重越大。

**差分算子**：
一阶差分：$\nabla y_t = y_t - y_{t-1}$
$d$ 阶差分：$\nabla^d y_t = \nabla^{d-1} y_t - \nabla^{d-1} y_{t-1}$

**季节性差分**：如果序列有周期 $S$（如年度 $S=12$ 月或 $S=7$ 日）：
$$
\nabla_S y_t = y_t - y_{t-S}
$$

**时间编码特征**：
- 正弦/余弦编码（捕捉循环时间如小时、月份）：
$$
\sin(2\pi \times hour / 24), \quad \cos(2\pi \times hour / 24)
$$
- 用一对正余弦编码可以保持时间的循环距离（23:00 和 1:00 的距离小）。

## 直观理解

- **滞后项 = "回头看"**：预测明天的股票价格，昨天和今天的价格是最有力的特征。滞后项就是"回头看几步"——回头看 1 步就是 $y_{t-1}$，回头看 7 步就是 $y_{t-7}$（对周周期数据很重要）。
- **滑动窗口 = "局部平滑"**：单日股价波动可能只是噪声，但 20 日均线能揭示趋势。滑动窗口就像给时间序列加了一个"模糊滤镜"——过滤掉短期波动，看到更稳定的模式。
- **差分 = "去掉趋势看变化"**：一个股票从 100 涨到了 200 再到 150，直接看价格很难建模，但如果看每日涨跌额（差分），序列变得平稳且有规律（涨多了回调，跌多了反弹）。许多模型对平稳序列的预测效果远好于原始序列。

## 代码示例

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import train_test_split

# 生成模拟时间序列数据
np.random.seed(42)
dates = pd.date_range('2020-01-01', periods=500, freq='D')
trend = np.linspace(0, 5, 500)
seasonality = 2 * np.sin(2 * np.pi * np.arange(500) / 30)  # 30天周期
noise = np.random.randn(500) * 0.5
y = trend + seasonality + noise

df = pd.DataFrame({'date': dates, 'y': y})

# 构造时间序列特征
def create_ts_features(df, lags=7, window=7):
    for lag in range(1, lags + 1):
        df[f'lag_{lag}'] = df['y'].shift(lag)
    df['rolling_mean'] = df['y'].rolling(window=window).mean()
    df['rolling_std'] = df['y'].rolling(window=window).std()
    df['hour'] = df['date'].dt.hour
    df['dayofweek'] = df['date'].dt.dayofweek
    df['month'] = df['date'].dt.month
    df['diff'] = df['y'].diff()
    return df

df_feat = create_ts_features(df)
df_feat = df_feat.dropna()

# 监督学习
feature_cols = [c for c in df_feat.columns if c not in ['date', 'y']]
X, y_ts = df_feat[feature_cols].values, df_feat['y'].values
X_train, X_test, y_train, y_test = train_test_split(
    X, y_ts, test_size=0.2, shuffle=False)  # 时序数据不打乱！

rf = RandomForestRegressor(n_estimators=100, random_state=42)
rf.fit(X_train, y_train)
print(f"测试集 R^2: {rf.score(X_test, y_test):.4f}")

# 特征重要性
for name, imp in sorted(zip(feature_cols, rf.feature_importances_),
                        key=lambda x: x[1], reverse=True)[:5]:
    print(f"  {name}: {imp:.4f}")
```

## 深度学习关联

- **LSTM/GRU 的隐式滞后**：循环神经网络 (RNN) 通过循环连接自动学习时间依赖关系，无需手动构造滞后特征。LSTM 的记忆单元可以学习多长的历史信息需要保留、多长的信息需要遗忘——这是对滑动窗口思想的柔性扩展。
- **Transformer 的位置编码与时间特征**：时间序列 Transformer (如 Informer、Autoformer) 使用位置编码或时间戳嵌入来捕捉时间信息，配合自注意力机制自动学习不同时间步之间的依赖关系，相当于自适应地学习最优的"滞后阶数"和"窗口大小"。
- **时序特征在预测中的应用**：在电力负荷预测、交通流量预测、股票预测等深度时序模型中，尽管 LSTM/Transformer 能自动学习时序依赖，但手动添加的滞后特征和日历特征（节假日、工作日等）仍然显著提升预测性能——领域知识特征工程跟深度特征提取是互补的。
