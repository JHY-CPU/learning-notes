# 27_L2 正则化与权重衰减的等价性证明

## 核心概念

- **L2 正则化的定义**：L2 正则化在损失函数中添加权重的平方和作为惩罚项：$\tilde{L} = L + \frac{\lambda}{2}\|\theta\|^2$，其中 $\lambda$ 是正则化强度。目的是限制权重的尺度，防止过拟合。
- **权重衰减的定义**：权重衰减（Weight Decay）是在参数更新时，直接将参数向零衰减：$\theta_{t+1} = \theta_t - \eta \nabla L - \eta \lambda \theta_t$。第一项是正常的梯度下降，第二项是衰减项。
- **SGD 中的等价性**：在使用标准 SGD 优化器时，L2 正则化和权重衰减完全等价。这是因为将 L2 正则化代入 SGD 的更新公式后，恰好得到 $\theta_{t+1} = (1-\eta\lambda)\theta_t - \eta \nabla L$，与权重衰减的更新一致。
- **不再成立的情况**：在 Adam、RMSProp 等自适应学习率优化器中，L2 正则化和权重衰减不再等价。因为自适应学习率对 L2 梯度项和损失梯度项做了相同缩放，而权重衰减直接作用于参数，不经过学习率缩放。

## 数学推导

**SGD 更新规则**：

标准梯度下降：

$$
\theta_{t+1} = \theta_t - \eta \nabla L(\theta_t)
$$

**L2 正则化的损失函数**：

$$
\tilde{L}(\theta) = L(\theta) + \frac{\lambda}{2}\|\theta\|^2
$$

梯度：

$$
\nabla \tilde{L}(\theta) = \nabla L(\theta) + \lambda \theta
$$

代入 SGD：

$$
\theta_{t+1} = \theta_t - \eta (\nabla L(\theta_t) + \lambda \theta_t)
$$

$$
= \theta_t - \eta \nabla L(\theta_t) - \eta \lambda \theta_t
$$

$$
= (1 - \eta \lambda) \theta_t - \eta \nabla L(\theta_t)
$$

**权重衰减的更新规则**：

$$
\theta_{t+1} = \theta_t - \eta \nabla L(\theta_t) - \eta \lambda \theta_t
$$

这正是 L2 正则化得到的相同公式。因此，在标准 SGD 下，两者完全等价。

**在 Adam 中两者不等价**：

Adam 中 L2 正则化的更新：

$$
\tilde{g}_t = \nabla L(\theta_t) + \lambda \theta_t
$$

$$
\theta_{t+1} = \theta_t - \eta \frac{\hat{m}_t}{\sqrt{\hat{v}_t} + \epsilon}, \quad
\text{其中 } m_t, v_t \text{ 基于 } \tilde{g}_t
$$

Adam 中权重衰减（AdamW 风格）的更新：

$$
g_t = \nabla L(\theta_t) \quad (\text{不含 L2 项})
$$

$$
\theta_{t+1} = \theta_t - \eta \frac{\hat{m}_t}{\sqrt{\hat{v}_t} + \epsilon} - \eta \lambda \theta_t
$$

两者不同，因为 L2 正则化项 $\lambda \theta_t$ 经过了 $\sqrt{\hat{v}_t}$ 的自适应缩放，而权重衰减没有。

## 直观理解

L2 正则化和权重衰减在 SGD 中的等价性可以理解为"殊途同归"：L2 正则化是从损失函数的角度施加惩罚（目标函数中加一项），权重衰减是从优化过程的角度施加惩罚（更新公式中减一项）。因为 SGD 的更新就是沿着损失梯度的反方向，所以两种方式自然等价。

形象地说，L2 正则化像"在薪水里直接扣税"（在损失中直接惩罚），而权重衰减像"发薪前就扣除"（在更新时直接衰减）。在 SGD 下，两种方式扣除的金额完全一样。

但在 Adam 中，事情发生了变化。L2 正则化相当于"根据调整后的收入比例扣税"，而权重衰减是"固定扣除"。由于 Adam 的税率（学习率）对每个参数不同，两种方式的最终效果不再相同。

## 代码示例

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 证明 SGD 下 L2 正则化与权重衰减的等价性

# 创建相同的模型
torch.manual_seed(42)

model_l2 = nn.Linear(10, 2)
model_wd = nn.Linear(10, 2)
# 确保参数相同
model_wd.weight.data = model_l2.weight.data.clone()
model_wd.bias.data = model_l2.bias.data.clone()

# 两种方式设置 L2 正则化
lr = 0.01
wd = 0.001

# 方式一: 在损失中加入 L2 正则项
optimizer_l2 = optim.SGD(model_l2.parameters(), lr=lr)

# 方式二: 使用 weight_decay 参数
optimizer_wd = optim.SGD(model_wd.parameters(), lr=lr, weight_decay=wd)

# 相同的输入数据
x = torch.randn(32, 10)
y = torch.randint(0, 2, (32,))
criterion = nn.CrossEntropyLoss()

# 训练一步检查参数一致性
def train_one_step(model, optimizer, use_l2=False):
    optimizer.zero_grad()
    pred = model(x)
    loss = criterion(pred, y)
    if use_l2:
        # 手动添加 L2 正则项
        l2_reg = sum(p.pow(2).sum() for p in model.parameters())
        loss = loss + wd/2 * l2_reg
    loss.backward()
    optimizer.step()
    return loss.item()

print("SGD 下 L2 正则化 vs 权重衰减:")
print("初始参数是否一致:", torch.allclose(model_l2.weight, model_wd.weight))

# 多次训练步骤
for step in range(10):
    loss_l2 = train_one_step(model_l2, optimizer_l2, use_l2=True)
    loss_wd = train_one_step(model_wd, optimizer_wd, use_l2=False)

params_equal = torch.allclose(model_l2.weight, model_wd.weight, atol=1e-6)
print(f"Step {step+1}: 参数一致 = {params_equal}, L2 loss={loss_l2:.4f}, WD loss={loss_wd:.4f}")

# 证明 Adam 下两者不等价
print("\nAdam 下 L2 正则化 vs 权重衰减:")
model_adam_l2 = nn.Linear(10, 2)
model_adam_wd = nn.Linear(10, 2)
model_adam_wd.weight.data = model_adam_l2.weight.data.clone()
model_adam_wd.bias.data = model_adam_l2.bias.data.clone()

adam_l2 = optim.Adam(model_adam_l2.parameters(), lr=0.001)
adam_wd = optim.Adam(model_adam_wd.parameters(), lr=0.001, weight_decay=wd)

for step in range(10):
    # L2 方式
    adam_l2.zero_grad()
    pred = model_adam_l2(x)
    loss = criterion(pred, y) + wd/2 * sum(p.pow(2).sum() for p in model_adam_l2.parameters())
    loss.backward()
    adam_l2.step()

    # 权重衰减方式 (Adam 内置)
    adam_wd.zero_grad()
    pred = model_adam_wd(x)
    loss = criterion(pred, y)
    loss.backward()
    adam_wd.step()

    params_equal = torch.allclose(model_adam_l2.weight, model_adam_wd.weight, atol=1e-4)
    if step == 0:
        print(f"  Step 1: 参数一致 = {params_equal} (很快就会不同)")

print(f"  最终参数一致: {torch.allclose(model_adam_l2.weight, model_adam_wd.weight, atol=1e-3)}")

# 演示权重衰减对参数的影响
print("\n权重衰减的效果演示:")
model_noreg = nn.Linear(10, 2)
model_reg = nn.Linear(10, 2)
model_reg.weight.data = model_noreg.weight.data.clone()

opt_noreg = optim.SGD(model_noreg.parameters(), lr=0.01)
opt_reg = optim.SGD(model_reg.parameters(), lr=0.01, weight_decay=0.01)

for _ in range(50):
    x = torch.randn(32, 10)
    y = torch.randint(0, 2, (32,))

    for opt, m in [(opt_noreg, model_noreg), (opt_reg, model_reg)]:
        opt.zero_grad()
        loss = criterion(m(x), y)
        loss.backward()
        opt.step()

print(f"  no reg weight norm: {model_noreg.weight.norm():.4f}")
print(f"  w/ weight decay:    {model_reg.weight.norm():.4f}")
```

## 深度学习关联

- **在 SGD 优化器中的普及**：几乎所有深度学习框架的 SGD 优化器都通过 `weight_decay` 参数提供 L2 正则化功能。由于在 SGD 中等价，用户不需要区分两者。
- **AdamW 的提出背景**：在 Adam 中 L2 正则化和权重衰减不再等价的发现，直接催生了 AdamW 优化器。AdamW 的作者证明了在自适应优化器中解耦权重衰减的重要性，这在 NLP 领域的大模型训练中得到了广泛验证。
- **贝叶斯视角**：从贝叶斯角度看，L2 正则化等价于在权重上施加高斯先验 $\theta \sim \mathcal{N}(0, 1/\lambda)$。权重衰减的更新规则对应于最大后验估计（MAP）的优化过程。这为理解正则化的概率意义提供了更深层的视角。
