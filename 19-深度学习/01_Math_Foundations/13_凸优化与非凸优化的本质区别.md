# 13_凸优化与非凸优化的本质区别

## 核心概念

- **凸集 (Convex Set)**：集合 $C$ 中任意两点连线上的所有点仍在 $C$ 中。形式化为：对任意 $\mathbf{x}, \mathbf{y} \in C$ 和 $\theta \in [0,1]$，有 $\theta\mathbf{x} + (1-\theta)\mathbf{y} \in C$。
- **凸函数 (Convex Function)**：$f(\theta\mathbf{x} + (1-\theta)\mathbf{y}) \leq \theta f(\mathbf{x}) + (1-\theta) f(\mathbf{y})$。凸函数的等高线是凸集，局部极小值必为全局极小值。
- **凸优化**：优化凸函数在凸集上的最小值。关键特性：任何局部最优解都是全局最优解；一阶条件 $\nabla f(\mathbf{x}^*) = \mathbf{0}$ 是最优的充要条件。
- **非凸优化**：优化目标函数非凸或约束集非凸的问题。可能存在大量局部极小值、鞍点和平坦区域。找到全局最优是 NP-难的，通常以找到"足够好"的局部解为目标。
- **凸 vs 非凸的判定**：凸函数的充要条件是 Hessian 矩阵半正定。非凸函数的 Hessian 在某些区域可能负定（有凹区域），这是产生鞍点和多局部极小值的根源。

## 数学推导

凸函数的定义：
$$
f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y), \quad \forall \theta \in [0,1]
$$

一阶充要条件（可微凸函数）：
$$
f(y) \geq f(x) + \nabla f(x)^T(y-x), \quad \forall x, y
$$

二阶充要条件（二次可微凸函数）：
$$
\nabla^2 f(x) \succeq 0 \quad (\text{Hessian 半正定}), \quad \forall x
$$

凸优化问题标准形式（全局最优性保证）：
$$
\min_{\mathbf{x}} f(\mathbf{x}) \quad \text{s.t.} \quad g_i(\mathbf{x}) \leq 0, \; h_j(\mathbf{x}) = 0
$$
其中 $f$ 和所有 $g_i$ 是凸函数，$h_j$ 是仿射函数。

非凸问题与之对比——局部极小值 $\mathbf{x}^*$ 仅满足：
$$
\nabla f(\mathbf{x}^*) = 0, \quad \nabla^2 f(\mathbf{x}^*) \succeq 0
$$
这只是局部最优的必要条件，不是全局最优的保证。

## 直观理解

- **碗 vs 山地**：凸函数像一个光滑的碗——从任何位置开始，沿着梯度下降都能到达同一个最低点。非凸函数像真实的山地——有多个山谷（局部极小值）、山脊（鞍点）和不连续的悬崖。不同的起点可能到达不同的山谷。
- **橡皮膜比喻**：想象用橡皮膜覆盖山峰。凸函数的橡皮膜是拉紧的、没有褶皱的（任何两点间的膜都在两点上方）。非凸函数的橡皮膜有褶皱和凹陷，形成了多个"洼地"。
- **深度学习的"反直觉"**：传统观点认为非凸优化比凸优化困难很多，但深度学习的实践表明，在高维空间中，局部极小值的数量虽多，但"好的"局部极小值（泛化能力强）也很多，且 SGD 倾向于收敛到这类好的极小值。

## 代码示例

```python
import numpy as np

# 凸函数判定
def is_convex_function(f, hessian, x_range=(-5, 5), n_points=100):
    """通过 Hessian 检查凸性"""
    xs = np.linspace(x_range[0], x_range[1], n_points)
    for x in xs:
        H = hessian(x)
        # 检查所有特征值 >= 0
        eigvals = np.linalg.eigvalsh(H)
        if np.min(eigvals) < -1e-10:
            return False
    return True

# 凸函数
f_convex = lambda x: x**2
hess_convex = lambda x: np.array([[2]])

# 非凸函数
f_nonconvex = lambda x: x**4 - 4*x**2 + x  # 有多个局部极值
def hess_nonconvex(x):
    return np.array([[12*x**2 - 8]])

print(f"f(x)=x^2 是凸函数? {is_convex_function(f_convex, hess_convex)}")
print(f"f(x)=x^4-4x^2+x 是凸函数? {is_convex_function(f_nonconvex, hess_nonconvex)}")

# 多维凸 vs 非凸
def hessian_check():
    # 凸: H 半正定
    A_convex = np.array([[2, 0], [0, 3]])  # 特征值 2, 3 > 0
    # 非凸: H 不定
    A_nonconvex = np.array([[2, 0], [0, -1]])  # 特征值 2, -1

    print(f"\n凸Hessian特征值: {np.linalg.eigvalsh(A_convex)}")
    print(f"非凸Hessian特征值: {np.linalg.eigvalsh(A_nonconvex)}")

    # 验证二次型 f(x) = x^T A x 的凸性
    x = np.array([1.0, 1.0])
    print(f"凸二次型 x^T A_convex x = {x @ A_convex @ x:.2f}")
    print(f"非凸二次型 x^T A_nonconvex x = {x @ A_nonconvex @ x:.2f}")

hessian_check()

# 凸优化保证：梯度为零 => 全局最优
# 非凸优化反例
print("\n--- 凸 vs 非凸优化示例 ---")
# 凸问题: 梯度下降收敛到全局最优
f_conv = lambda x: x**2
grad_conv = lambda x: 2*x

# 非凸问题: 梯度下降可能陷入局部最优
f_nc = lambda x: 0.1*x**4 - 0.5*x**2 - 0.3*x  # 有两个局部极小值
grad_nc = lambda x: 0.4*x**3 - x - 0.3

for x0 in [-3, -1, 0, 2, 3]:
    x = x0
    for _ in range(500):
        x = x - 0.05 * grad_nc(x)
    val = f_nc(x)
    print(f"初始 x0={x0:4.1f} -> 终点 x={x:.4f}, f(x)={val:.4f}")
```

## 深度学习关联

- **深度网络的损失景观**：深度神经网络的损失函数高度非凸，但实践发现"大部分局部极小值在损失值上接近全局最小值"。理论研究表明，在网络宽度足够大时，所有的局部极小值都是全局极小值，这是因为过度参数化使得 Hessian 在极小值处几乎处处正定。
- **鞍点 vs 局部极小值**：高维非凸优化中主要的困难不是局部极小值而是鞍点。在鞍点处，梯度为零但 Hessian 有正有负，优化会停滞。幸运的是，SGD 的随机性提供了一种天然的"逃离"机制——梯度噪声可以在负曲率方向推动优化器离开鞍点。
- **凸代理损失 (Convex Surrogate)**：虽然深度网络整体非凸，但常用的损失函数（如交叉熵）相对于模型输出的最后一层是凸的。训练时只优化最后一层的输出（固定前面层），就变成了凸优化问题。这个性质在理论分析中经常被利用。
- **优化与泛化的解耦**：在深度学习中，找到训练损失的"全局极小值"并不保证好的泛化。非凸优化中找到的局部极小值的"平坦度"（Hessian 特征值的大小）反而与泛化能力相关——平坦极小值通常泛化更好。这是凸优化中没有的现象。
