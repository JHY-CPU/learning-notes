# 38_共轭先验与后验分布的解析解

## 核心概念

- **共轭先验 (Conjugate Prior)**：对于似然函数 $p(x|\theta)$，如果先验 $p(\theta)$ 和后验 $p(\theta|x)$ 属于同一分布族，则称该先验为共轭先验。共轭性保证了后验有解析表达式，极大简化了贝叶斯计算。
- **共轭性的本质**：先验和后验属于同一分布族，意味着更新只涉及参数的改变。数据的作用就是将先验参数 "更新" 为后验参数，这通常表现为简单的加法运算。
- **常见共轭对**：
  | 似然 | 共轭先验 | 后验参数更新 |
  |------|---------|------------|
  | Bernoulli | Beta | $\alpha + \sum x_i, \beta + n - \sum x_i$ |
  | Gaussian (已知 $\sigma^2$) | Gaussian | 精度加权平均 |
  | Poisson | Gamma | $\alpha + \sum x_i, \beta + n$ |
  | Multinomial | Dirichlet | $\alpha_i + \sum \mathbf{1}(x=i)$ |
  | Exponential | Gamma | $\alpha + n, \beta + \sum x_i$ |
- **指数族与共轭性**：指数族分布的对数似然 $\log p(x|\eta) = \eta^T T(x) - A(\eta) + \text{const}$ 是 $\eta$ 的线性函数，其共轭先验自然是指数族形式本身。
- **共轭先验在深度学习中的角色**：尽管现代深度学习中很少显式使用共轭先验，但理解共轭性对于理解变分推断、贝叶斯神经网络和概率编程至关重要。

## 数学推导

一般形式的共轭：对于指数族似然 $p(x|\eta) \propto \exp(\eta^T T(x))$，其共轭先验为：
$$
p(\eta|\tau, \nu) \propto \exp(\eta^T \tau - \nu A(\eta))
$$

其中 $\tau, \nu$ 是超参数（分别对应先验的"充分统计量"和"伪计数"）。后验为：
$$
p(\eta|x_{1:n}) \propto \exp\left( \eta^T \left(\tau + \sum_{i=1}^n T(x_i) \right) - (\nu + n) A(\eta) \right)
$$

### Beta-Bernoulli 共轭
似然：$x_i \sim \text{Bernoulli}(p)$，先验：$p \sim \text{Beta}(\alpha, \beta)$

$$
p(p|x) \propto p^{\alpha-1}(1-p)^{\beta-1} \prod_{i=1}^n p^{x_i}(1-p)^{1-x_i}
$$
$$
= p^{\alpha + n\bar{x} - 1}(1-p)^{\beta + n(1-\bar{x}) - 1}
$$

所以后验：$p|x \sim \text{Beta}(\alpha + n\bar{x}, \beta + n - n\bar{x})$

### Gaussian-Gaussian 共轭
似然：$x_i \sim \mathcal{N}(\mu, \sigma^2)$（已知 $\sigma^2$），先验：$\mu \sim \mathcal{N}(\mu_0, \sigma_0^2)$

后验均值：
$$
\mu_n = \frac{\sigma_0^2}{\sigma_0^2 + \sigma^2/n} \bar{x} + \frac{\sigma^2/n}{\sigma_0^2 + \sigma^2/n} \mu_0
$$
后验方差：
$$
\sigma_n^2 = \frac{\sigma_0^2 \sigma^2}{n\sigma_0^2 + \sigma^2}
$$

## 直观理解

- **"伪计数"的直觉**：Beta 先验 $\text{Beta}(\alpha, \beta)$ 可以理解为"预先看到的 $\alpha-1$ 次成功和 $\beta-1$ 次失败"。后验更新就是把这些"伪计数"加上实际观测，得到总的成功和失败次数。
- **更新过程就像学习**：先验代表初始知识（或偏见），数据每到来一条，后验参数就更新一次。随着数据增多，先验的影响逐渐减弱，后验趋近于 MLE。这完美体现了"从经验中学习"的过程。
- **共轭的威力**：如果没有共轭性，每次更新后验都需要做高维积分（通常不可行）。共轭性使得一轮贝叶斯更新只需要几行代数运算——先验参数 + 数据统计量 = 后验参数。

## 代码示例

```python
import numpy as np
from scipy.stats import beta, norm, gamma

# 1. Beta-Bernoulli 共轭：硬币抛掷
np.random.seed(42)

# 真实参数
true_p = 0.35
n_tosses = 100
data = np.random.binomial(1, true_p, n_tosses)
n_success = np.sum(data)
n_failure = n_tosses - n_success

# 先验 Beta(2, 2) -- 认为硬币大致公平
alpha_prior, beta_prior = 2, 2

# 后验 Beta(alpha_prior + 成功数, beta_prior + 失败数)
alpha_post = alpha_prior + n_success
beta_post = beta_prior + n_failure

print("Beta-Bernoulli 共轭:")
print(f"  真实 p = {true_p}")
print(f"  观测: {n_success} 正, {n_failure} 反")
print(f"  先验: Beta({alpha_prior}, {beta_prior})")
print(f"  后验: Beta({alpha_post}, {beta_post})")
print(f"  后验均值: {alpha_post/(alpha_post+beta_post):.4f} (MLE={n_success/n_tosses:.4f})")

# 序贯更新
print("\n序贯贝叶斯更新:")
alpha, beta = 2, 2
for i, x in enumerate(data[:10]):
    alpha += x
    beta += 1 - x
    if i < 5 or i == 9:
        post_mean = alpha / (alpha + beta)
        print(f"  前 {i+1:2d} 次后: 后验均值≈{post_mean:.4f}")

# 2. Gaussian-Gaussian 共轭
true_mu = 3.0
sigma = 1.5
n_data = 20
data = np.random.randn(n_data) * sigma + true_mu

# 先验 mu ~ N(0, 10^2) (弱先验)
mu0, sigma0 = 0.0, 10.0

# 后验
x_bar = np.mean(data)
sigma_n_sq = 1 / (1/sigma0**2 + n_data/sigma**2)
mu_n = sigma_n_sq * (mu0/sigma0**2 + n_data*x_bar/sigma**2)

print(f"\nGaussian-Gaussian 共轭:")
print(f"  真实 μ = {true_mu}")
print(f"  样本均值 = {x_bar:.4f}")
print(f"  先验 μ~N({mu0}, {sigma0}^2)")
print(f"  后验 μ~N({mu_n:.4f}, {np.sqrt(sigma_n_sq):.4f})")

# 3. Gamma-Poisson 共轭
true_lambda = 2.5
n_pois = 30
data_pois = np.random.poisson(true_lambda, n_pois)

# 先验 Gamma(1, 1) (弱先验)
alpha_prior_g, beta_prior_g = 1, 1

# 后验 Gamma(alpha + sum(x), beta + n)
alpha_post_g = alpha_prior_g + np.sum(data_pois)
beta_post_g = beta_prior_g + n_pois

print(f"\nGamma-Poisson 共轭:")
print(f"  真实 λ = {true_lambda}")
print(f"  样本均值 = {np.mean(data_pois):.4f}")
print(f"  后验均值 = {alpha_post_g/beta_post_g:.4f}")

# 4. 先验强度的影响
print("\n先验强度的影响 (Beta-Bernoulli, 3正1反):")
for alpha0, beta0 in [(1, 1), (5, 2), (100, 100)]:
    pa = alpha0 + 3
    pb = beta0 + 1
    post_mean = pa / (pa + pb)
    print(f"  先验 Beta({alpha0},{beta0}): 后验均值={post_mean:.4f} (MLE=0.75)")
```

## 深度学习关联

- **贝叶斯与深度学习的桥梁**：虽然深度学习主要使用点估计（MAP），但共轭先验的理解可以帮助设计更好的正则化和先验结构。例如，高斯先验（L2 正则化）对应于在权重上施加高斯先验，而拉普拉斯先验（L1 正则化）对应拉普拉斯先验。共轭性告诉我们，在贝叶斯设置中，某些损失函数有自然的对应先验分布。
- **概率编程语言**：Pyro、Stan、TensorFlow Probability 等概率编程语言在后端使用共轭性加速推断。当检测到共轭结构时，系统自动使用解析更新而非采样或 VI。理解共轭性有助于理解这些框架的优化策略。
- **主题模型 (LDA)**：LDA 利用共轭性——文档主题分布的共轭先验是 Dirichlet 分布，主题词分布的共轭先验也是 Dirichlet。这使得 Gibbs 采样和变分推断可以高效地处理 LDA，每一轮更新只需要简单的计数和采样操作。
- **贝叶斯非参数模型**：Dirichlet 过程（DP）作为 Dirichlet 分布的无限维推广，保持了多项分布的共轭性。贝叶斯非参数模型中的 Chinese Restaurant Process 采样器利用了这一性质。在深度学习中，贝叶斯非参数方法可用于自动确定网络宽度和深度。
