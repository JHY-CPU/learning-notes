# 29_高维空间中的距离度量：欧氏距离与余弦相似度

## 核心概念

- **欧氏距离 (Euclidean Distance)**：$\|\mathbf{x} - \mathbf{y}\|_2 = \sqrt{\sum_i (x_i - y_i)^2}$，是最直观的空间直线距离。在低维空间符合直觉，但在高维空间中会遭遇"维度灾难"。
- **余弦相似度 (Cosine Similarity)**：$\cos(\mathbf{x}, \mathbf{y}) = \frac{\mathbf{x} \cdot \mathbf{y}}{\|\mathbf{x}\| \|\mathbf{y}\|}$，衡量两个向量的方向一致性，不考虑长度。取值范围 $[-1, 1]$，1 表示方向完全相同，-1 表示方向相反。
- **欧氏距离与余弦相似度的关系**：$\|\mathbf{x} - \mathbf{y}\|_2^2 = \|\mathbf{x}\|_2^2 + \|\mathbf{y}\|_2^2 - 2 \mathbf{x} \cdot \mathbf{y}$。当向量已归一化（$\|\mathbf{x}\| = \|\mathbf{y}\| = 1$）时，欧氏距离与余弦相似度单调相关：$\|\mathbf{x} - \mathbf{y}\|_2^2 = 2(1 - \cos(\mathbf{x},\mathbf{y}))$。
- **维度灾难 (Curse of Dimensionality)**：在高维空间中，所有点对之间的距离都趋向于相等，导致距离度量失去区分能力。具体地，$\lim_{d \to \infty} \frac{\|x\|_{\max} - \|x\|_{\min}}{\|x\|_{\min}} \to 0$。
- **曼哈顿距离 (Manhattan Distance)**：$\|\mathbf{x} - \mathbf{y}\|_1 = \sum_i |x_i - y_i|$，在高维空间中比欧氏距离更稳健（对异常值不那么敏感）。
- **马氏距离 (Mahalanobis Distance)**：考虑协方差结构的距离，$\sqrt{(\mathbf{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\mathbf{x} - \boldsymbol{\mu})}$，消除了特征间的相关性影响。

## 数学推导

欧氏距离：
$$
d_{\text{Euclidean}}(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{i=1}^d (x_i - y_i)^2} = \|\mathbf{x} - \mathbf{y}\|_2
$$

余弦相似度：
$$
\cos(\mathbf{x}, \mathbf{y}) = \frac{\mathbf{x} \cdot \mathbf{y}}{\|\mathbf{x}\|_2 \|\mathbf{y}\|_2} = \frac{\sum_{i=1}^d x_i y_i}{\sqrt{\sum_{i=1}^d x_i^2} \sqrt{\sum_{i=1}^d y_i^2}}
$$

余弦距离：
$$
d_{\text{cosine}}(\mathbf{x}, \mathbf{y}) = 1 - \cos(\mathbf{x}, \mathbf{y}) \in [0, 2]
$$

高维空间中距离的趋同现象（设 $\mathbf{x}, \mathbf{y} \sim \mathcal{U}([0,1]^d)$）：
$$
\mathbb{E}[\|\mathbf{x} - \mathbf{y}\|_2^2] = \frac{d}{6}
$$
$$
\text{Var}[\|\mathbf{x} - \mathbf{y}\|_2^2] = \frac{7d}{180}
$$
随着维数 $d$ 增长，标准差与均值的比值 $\propto 1/\sqrt{d} \to 0$。

## 直观理解

- **欧氏距离是"直线"**：就像在两点之间拉一根绳子，测量绳子的长度。简单直观，但在高维中所有绳子长度都变得差不多，失去了区分度。
- **余弦相似度是"角度"**：不关心向量的长度，只关心它们是否指向相同的方向。在文本分析中，一篇长文章和一篇短文章可能讨论同一主题（余弦相似度高），但欧氏距离可能很大（长度差异大）。
- **归一化的效果**：将所有向量归一化到单位球面上后，欧氏距离和余弦相似度等价。这就像把所有文本缩放到相同长度，只看内容方向的差异，这在很多深度表示学习中广泛采用。
- **维度灾难的直觉**：想象在一个 $d$ 维超立方体中随机取点。当 $d$ 很大时，大部分点都落在立方体的角落（离中心很远），中心区域几乎是空的。所有点对之间的距离都接近 $\sqrt{d/6}$，使得最近邻与最远邻的距离几乎一样。

## 代码示例

```python
import numpy as np

def euclidean_distance(x, y):
    return np.linalg.norm(x - y)

def cosine_similarity(x, y):
    return np.dot(x, y) / (np.linalg.norm(x) * np.linalg.norm(y))

def cosine_distance(x, y):
    return 1 - cosine_similarity(x, y)

# 1. 基础对比
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])
z = np.array([2, 4, 6])  # 2*x，方向相同

print("距离度量对比:")
print(f"  d_euc(x, y) = {euclidean_distance(x, y):.4f}")
print(f"  d_euc(x, z) = {euclidean_distance(x, z):.4f} (长度不同)")
print(f"  cos(x, y) = {cosine_similarity(x, y):.4f}")
print(f"  cos(x, z) = {cosine_similarity(x, z):.4f} (方向相同!)")

# 2. 归一化后欧氏距离与余弦距离等价
x_norm = x / np.linalg.norm(x)
z_norm = z / np.linalg.norm(z)
print(f"\n归一化后:")
print(f"  d_euc(x_norm, z_norm) = {euclidean_distance(x_norm, z_norm):.4f}")
print(f"  sqrt(2-2*cos) = {np.sqrt(2 - 2*cosine_similarity(x, z)):.4f}")

# 3. 维度灾难演示
np.random.seed(42)
dimensions = [2, 5, 10, 50, 100, 500, 1000]
print("\n维度灾难:")
for d in dimensions:
    # 在 [0,1]^d 中随机采样 100 个点
    points = np.random.rand(100, d)
    # 计算所有点对距离
    dists = []
    for i in range(50):
        for j in range(i+1, 50):
            dists.append(euclidean_distance(points[i], points[j]))
    dists = np.array(dists)
    ratio = (np.max(dists) - np.min(dists)) / np.min(dists)
    print(f"  维度 {d:4d}: 距离范围=[{np.min(dists):.4f}, {np.max(dists):.4f}], "
          f"变异比={ratio:.4f}")

# 4. 文本向量的距离度量
doc1 = np.array([1, 0, 1, 1, 0])  # 词袋向量
doc2 = np.array([2, 0, 2, 2, 0])  # 同一主题，篇幅更长
doc3 = np.array([0, 1, 0, 0, 1])  # 不同主题

print(f"\n文本向量:")
print(f"  cos(doc1, doc2) = {cosine_similarity(doc1, doc2):.4f} (同主题)")
print(f"  cos(doc1, doc3) = {cosine_similarity(doc1, doc3):.4f} (不同主题)")
print(f"  d_euc(doc1, doc2) = {euclidean_distance(doc1, doc2):.4f} (长度差异大)")
print(f"  d_euc(doc1, doc3) = {euclidean_distance(doc1, doc3):.4f}")

# 5. 选择合适的度量
# 模拟 embedding 向量（归一化的）
embeddings = np.random.randn(1000, 128)
embeddings = embeddings / np.linalg.norm(embeddings, axis=1, keepdims=True)

# 在归一化向量中，cosine 和 L2 等价
query = embeddings[0]
l2_dists = np.linalg.norm(embeddings - query, axis=1)
cos_sims = embeddings @ query

# 最近邻应该一致
l2_nn = np.argsort(l2_dists)[1]  # 排除自身
cos_nn = np.argsort(-cos_sims)[1]
print(f"\n归一化 embedding 中:")
print(f"  L2最近邻索引: {l2_nn}, Cosine最近邻索引: {cos_nn}")
print(f"  一致? {l2_nn == cos_nn}")
```

## 深度学习关联

- **表示学习中的度量选择**：在人脸识别（FaceNet）和度量学习中，通常使用余弦相似度作为特征向量的相似性度量。训练时使用 Triplet Loss 拉近同类样本的余弦距离，推远不同类样本。Softmax 分类器也等价于计算输入与权重的余弦相似度（当输入和权重都归一化时）。
- **注意力机制 (Attention)**：自注意力中的 $QK^T$ 计算就是点积相似度（等价于余弦相似度当向量已归一化）。Scaled Dot-Product Attention $\text{softmax}(QK^T / \sqrt{d_k})$ 通过 $\sqrt{d_k}$ 缩放来缓解高维点积的方差膨胀问题。
- **嵌入层的向量检索**：在大规模检索系统中（如语义搜索、推荐系统），物品和查询被编码为向量（embedding），使用余弦相似度或内积进行近似最近邻搜索（ANN）。FAISS、HNSW 等高效索引库支持在十亿级向量空间中快速检索。
- **对比学习 (Contrastive Learning)**：SimCLR、MoCo 等对比学习方法使用余弦相似度作为正负样本对的距离度量。InfoNCE 损失函数 $-\log \frac{\exp(\text{sim}(z_i, z_j)/\tau)}{\sum_k \exp(\text{sim}(z_i, z_k)/\tau)}$ 基于余弦相似度，通过温度参数 $\tau$ 控制相似度的锐利程度。
