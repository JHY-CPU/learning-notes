# 15_随机变量与概率分布族

## 核心概念

- **随机变量 (Random Variable)**：将随机试验的结果映射到实数空间的函数。离散随机变量取可数多个值（如抛硬币的结果），连续随机变量取某个区间内的值（如身高）。
- **概率分布 (Probability Distribution)**：描述随机变量取各个可能值的概率。离散型用概率质量函数 (PMF) 描述，连续型用概率密度函数 (PDF) 描述。
- **分布族 (Distribution Family)**：由一组参数决定的分布集合，如高斯族由均值和方差参数化。不同分布族有不同的适用场景和数学性质。
- **累积分布函数 (CDF)**：$F(x) = P(X \leq x)$，单调非减，取值在 $[0,1]$ 区间。PDF 是 CDF 的导数（对连续变量）。
- **概率分布的变换**：如果 $X \sim p_X(x)$，$Y = g(X)$，则 $Y$ 的分布可以通过变量变换公式得到。这涉及雅可比行列式。
- **指数族分布 (Exponential Family)**：形式为 $p(x|\theta) = h(x)\exp(\eta(\theta)^T T(x) - A(\eta))$ 的分布类，包括高斯、伯努利、泊松、Gamma 等。具有良好的数学性质。

## 数学推导

概率质量函数与概率密度函数的关系：
$$
\text{离散: } P(X = x_i) = p_i, \quad \sum_i p_i = 1
$$
$$
\text{连续: } P(a \leq X \leq b) = \int_a^b f_X(x)dx, \quad \int_{-\infty}^{\infty} f_X(x)dx = 1
$$

随机变量的期望与方差：
$$
\mathbb{E}[X] = \begin{cases} \sum_i x_i p_i & \text{离散} \\ \int x f_X(x) dx & \text{连续} \end{cases}
$$
$$
\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2
$$

随机变量变换的分布（一维，$g$ 单调）：
$$
f_Y(y) = f_X(g^{-1}(y)) \left|\frac{d}{dy} g^{-1}(y)\right|
$$

## 直观理解

- **随机变量不是变量，是函数**：随机变量 $X$ 不是一个数值，而是一个从样本空间到实数的映射。"$X = x$" 表示"随机试验的结果对应的数值为 $x$"。把随机变量理解为"试验结果的一个数值编码"会更准确。
- **分布的物理类比**：PMF/PDF 就像"概率的质量/密度分布"。概率大的区域质量/密度高，概率小的区域质量/密度低。总质量/积分恒等于 1。
- **分布族就像工具包**：不同的分布族适用于不同的数据和场景。高斯分布适合连续对称数据，伯努利适合二元数据，泊松适合计数数据。选择合适的分布族是统计建模的核心步骤。

## 代码示例

```python
import numpy as np
import scipy.stats as stats

# 1. 离散随机变量：伯努利分布
p = 0.6
bernoulli = stats.bernoulli(p)
samples = bernoulli.rvs(size=1000)
print(f"伯努利(p=0.6): P(X=1)≈{np.mean(samples):.3f} (理论={p})")

# 2. 连续随机变量：高斯分布
mu, sigma = 0, 1
normal = stats.norm(mu, sigma)
samples = normal.rvs(size=10000)
print(f"\nN(0,1): 均值={np.mean(samples):.3f} (理论={mu})")
print(f"         方差={np.var(samples):.3f} (理论={sigma**2})")

# 3. PDF、CDF 和分位数
x = 1.5
pdf_val = normal.pdf(x)   # 概率密度
cdf_val = normal.cdf(x)   # P(X <= x)
ppf_val = normal.ppf(0.975)  # 分位数（0.975 对应约 1.96）
print(f"\nN(0,1): PDF(1.5)={pdf_val:.4f}, CDF(1.5)={cdf_val:.4f}")
print(f"        0.975 分位数={ppf_val:.4f} (理论=1.96)")

# 4. 随机变量的变换
X = stats.norm(0, 1)
xs = X.rvs(100000)
ys = xs**2  # Y = X^2 ~ Chi-square(1)
print(f"\nY=X^2 (df=1): 均值={np.mean(ys):.3f} (理论=1)")
print(f"              方差={np.var(ys):.3f} (理论=2)")

# 5. 比较理论分布与经验分布
from scipy.stats import kstest
data = normal.rvs(size=500)
stat, p_val = kstest(data, 'norm')
print(f"\nK-S 检验: 统计量={stat:.4f}, p值={p_val:.4f}")
print("数据符合正态分布!" if p_val > 0.05 else "数据不符合正态分布!")

# 6. 常见分布族的参数
distributions = {
    'Normal': stats.norm(0, 1),
    'Exponential': stats.expon(scale=1.0),
    'Gamma': stats.gamma(a=2.0, scale=1.0),
    'Beta': stats.beta(a=2, b=5),
}

print("\n--- 各分布基本统计量 ---")
for name, dist in distributions.items():
    s = dist.rvs(10000)
    print(f"{name:15s}: 均值={np.mean(s):.3f}, 方差={np.var(s):.3f}, "
          f"偏度={stats.skew(s):.3f}, 峰度={stats.kurtosis(s):.3f}")
```

## 深度学习关联

- **生成模型中的分布建模**：GAN 和 VAE 的核心是学习数据分布 $p_{\text{data}}(\mathbf{x})$。生成器将简单分布（如标准正态）映射到复杂分布，判别器则判别样本来自真实分布还是生成分布。概率分布是生成式深度学习的数学基础。
- **变分自编码器 (VAE)**：VAE 假设隐变量服从标准正态先验 $p(\mathbf{z}) = \mathcal{N}(0, I)$，编码器学习后验 $q(\mathbf{z}|\mathbf{x})$（通常也是高斯），解码器学习条件分布 $p(\mathbf{x}|\mathbf{z})$。概率分布的选择直接影响生成质量。
- **Dropout 作为贝叶斯近似**：Dropout 可以解释为在神经网络权重上施加伯努利分布的近似贝叶斯推断。在测试时保持 Dropout 开启（Monte Carlo Dropout）可以得到预测的不确定性估计，这是将随机变量引入确定性网络的重要应用。
- **输出层的分布选择**：神经网络最后一层的激活函数和损失函数由输出变量的分布决定：回归用高斯 + MSE 损失、二分类用伯努利 + 交叉熵、多分类用类别分布 + 交叉熵。正确地匹配分布和损失函数是深度学习实践的基本要求。
