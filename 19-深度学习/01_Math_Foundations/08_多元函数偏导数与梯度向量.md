# 08_多元函数偏导数与梯度向量

## 核心概念

- **偏导数 (Partial Derivative)**：对多元函数 $f(x_1, x_2, \dots, x_n)$，偏导数 $\partial f / \partial x_i$ 衡量当其它变量固定时，$f$ 沿 $x_i$ 方向的变化率。计算时将其它变量视为常数，对目标变量求导。
- **梯度向量 (Gradient Vector)**：梯度的定义为 $\nabla f = (\partial f / \partial x_1, \partial f / \partial x_2, \dots, \partial f / \partial x_n)^T$，是各偏导数组成的向量。梯度指向函数值增长最快的方向。
- **方向导数**：函数在方向 $\mathbf{u}$（单位向量）上的方向导数为 $\nabla f \cdot \mathbf{u}$，表示 $\mathbf{u}$ 方向上的变化率。梯度方向的方向导数最大，等于 $\|\nabla f\|$。负梯度方向的方向导数最小，即下降最快的方向。
- **全微分**：$df = \frac{\partial f}{\partial x_1}dx_1 + \frac{\partial f}{\partial x_2}dx_2 + \dots + \frac{\partial f}{\partial x_n}dx_n$，表示当所有自变量都有微小变化时，函数值的线性近似变化。
- **梯度与等值面的关系**：梯度 $\nabla f$ 在几何上与函数等值面（等高线/等值面）正交。这一性质在约束优化和几何学习中非常重要。

## 数学推导

偏导数的极限定义：
$$
\frac{\partial f}{\partial x_i}(x_1, \dots, x_n) = \lim_{h \to 0} \frac{f(x_1, \dots, x_i + h, \dots, x_n) - f(x_1, \dots, x_n)}{h}
$$

梯度定义与方向导数的关系：
$$
\nabla f(\mathbf{x}) = \begin{pmatrix}
\frac{\partial f}{\partial x_1} \\
\frac{\partial f}{\partial x_2} \\
\vdots \\
\frac{\partial f}{\partial x_n}
\end{pmatrix}, \quad
D_{\mathbf{u}} f(\mathbf{x}) = \nabla f(\mathbf{x}) \cdot \mathbf{u}
$$

梯度的方向性：
$$
\max_{\|\mathbf{u}\|=1} D_{\mathbf{u}} f(\mathbf{x}) = \|\nabla f(\mathbf{x})\|, \quad
\min_{\|\mathbf{u}\|=1} D_{\mathbf{u}} f(\mathbf{x}) = -\|\nabla f(\mathbf{x})\|
$$

复合函数的梯度（链式法则）：
$$
\frac{\partial}{\partial x_i} f(g_1(\mathbf{x}), \dots, g_m(\mathbf{x})) = \sum_{j=1}^m \frac{\partial f}{\partial g_j} \cdot \frac{\partial g_j}{\partial x_i}
$$

## 直观理解

- **登山类比**：将函数值想象成山的高度。站在山上某点，梯度指向最陡的上坡方向，其大小表示坡度。负梯度方向就是最陡的下坡方向——这正是梯度下降法的核心思想。
- **等高线视角**：在地形图上，等高线密集的地方坡度大。梯度方向垂直于等高线（即切线方向），因为沿着等高线移动高度不变，方向导数为零，而梯度是方向导数最大的方向。
- **偏导数的"切片"理解**：$\partial f / \partial x$ 相当于在 $y$ 固定的截面上，看 $f$ 随 $x$ 的变化曲线在该点处的斜率。就像用刀切蛋糕，得到一个切面，看切面的斜率。

## 代码示例

```python
import numpy as np

# 定义函数 f(x, y) = x^2 + 2xy + y^3
def f(x, y):
    return x**2 + 2*x*y + y**3

# 数值计算梯度（用中心差分）
def numerical_gradient(f, x, y, h=1e-6):
    df_dx = (f(x + h, y) - f(x - h, y)) / (2 * h)
    df_dy = (f(x, y + h) - f(x, y - h)) / (2 * h)
    return np.array([df_dx, df_dy])

# 解析梯度：df/dx = 2x + 2y, df/dy = 2x + 3y^2
def analytical_gradient(x, y):
    return np.array([2*x + 2*y, 2*x + 3*y**2])

# 在多个点验证
points = [(1, 1), (0, 0), (-1, 2)]
for p in points:
    ng = numerical_gradient(f, p[0], p[1])
    ag = analytical_gradient(p[0], p[1])
    print(f"在 {p} 处: 数值={ng}, 解析={ag}, 误差={np.linalg.norm(ng-ag):.2e}")

# 梯度下降法求最小值
x, y = 2.0, 2.0
lr = 0.01
path = [(x, y)]
for _ in range(1000):
    g = analytical_gradient(x, y)
    x = x - lr * g[0]
    y = y - lr * g[1]
    path.append((x, y))

print(f"\n梯度下降终点: ({x:.4f}, {y:.4f})")
print(f"终点函数值: {f(x, y):.4f}")

# 验证梯度与等高线正交
# 等高线 f(x,y) = c 在 (x0,y0) 处的切线方向
x0, y0 = 1.0, 1.0
grad = analytical_gradient(x0, y0)
# 等高线切线方向: (-df/dy, df/dx)
tangent = np.array([-grad[1], grad[0]])
print(f"\n梯度: {grad}")
print(f"等高线切线: {tangent}")
print(f"点积 (应接近0): {np.dot(grad, tangent):.6f}")
```

## 深度学习关联

- **反向传播 (Backpropagation)**：反向传播的核心就是应用链式法则逐层计算损失函数对各层参数的梯度。$\partial L / \partial w^{(l)} = \partial L / \partial a^{(L)} \cdot \prod_{k=l+1}^{L} \partial a^{(k)} / \partial a^{(k-1)} \cdot \partial a^{(l)} / \partial w^{(l)}$。每个神经元的局部梯度通过链式相乘得到全局梯度。
- **梯度下降变体**：随机梯度下降 (SGD)、带动量的 SGD、Adam 等优化算法都是在梯度信息的基础上添加不同的修正策略。梯度是驱动所有深度学习优化的核心动力——没有梯度就没有学习。
- **梯度消失/爆炸**：在深层网络中，连续矩阵乘法导致梯度指数级衰减或增长。具体地，$\|\partial L / \partial h^{(l)}\| \propto \prod_{k=l}^{L} \|W^{(k)}\|$。这是激活函数选择（ReLU 替代 sigmoid）和残差连接（提供梯度捷径）的主要动机。
- **可解释性中的梯度**：Saliency Map、Grad-CAM 等可视化方法利用输入对输出的梯度 $\partial y_c / \partial x_{ij}$ 来指示输入中哪些像素对分类决策最重要，将梯度用作"关注度"指标。
