# 21_信息熵与自信息的定义

## 核心概念

- **自信息 (Self-information)**：事件 $X = x$ 发生后所携带的信息量，定义为 $I(x) = -\log P(x)$。概率越小的事件发生，带来的信息量越大。确定事件（$p=1$）的信息量为 0。
- **信息熵 (Information Entropy)**：随机变量 $X$ 的平均信息量，$H(X) = \mathbb{E}[-\log P(X)] = -\sum_i p_i \log p_i$。熵衡量了系统的不确定性或"混乱程度"。
- **对数的底数**：以 2 为底时单位是比特 (bit)，以 $e$ 为底时单位是纳特 (nat)，以 10 为底时单位是哈特 (hart)。深度学习中使用自然对数（纳特）。
- **最大熵原理**：在给定约束下，熵最大的分布是最"无偏"的（最能反映我们对未知的无知）。例如，只有均值约束时最大熵分布是指数分布，有均值和方差约束时是高斯分布。
- **联合熵与条件熵**：$H(X,Y) = -\sum\sum p(x,y)\log p(x,y)$ 衡量两个变量的联合不确定性。$H(Y|X) = \mathbb{E}[-\log p(Y|X)]$ 衡量已知 $X$ 后 $Y$ 的剩余不确定性。
- **熵与编码长度**：信息论中，熵 $H(X)$ 是对 $X$ 进行最优编码时每个符号所需的平均比特数的下界（香农源编码定理）。

## 数学推导

自信息：
$$
I(x) = -\log p(x) \quad (\text{以 2 为底时单位为比特})
$$

信息熵：
$$
H(X) = -\sum_{x \in \mathcal{X}} p(x) \log p(x) = \mathbb{E}_{x \sim p}[-\log p(x)]
$$

对于连续随机变量（微分熵）：
$$
h(X) = -\int p(x) \log p(x) dx
$$

条件熵：
$$
H(Y|X) = -\sum_x \sum_y p(x,y) \log p(y|x) = H(X,Y) - H(X)
$$

熵的性质：
- $H(X) \geq 0$，等号成立当 $X$ 是确定性的
- $H(X) \leq \log|\mathcal{X}|$，等号成立当 $X$ 均匀分布
- 熵是凹函数：$H(\lambda p + (1-\lambda)q) \geq \lambda H(p) + (1-\lambda)H(q)$

## 直观理解

- **"惊喜度"度量**：自信息量化了"意外程度"。听到"太阳从东边升起"（$p\approx 1$）不带来信息；听到"明天会下钞票"（$p\approx 0$）带来巨大信息量。
- **不确定性 = 熵**：一个均匀分布的骰子有高熵（$\log 6$ 比特），因为每次掷出的结果都很难预测。一个总是正面朝上的硬币熵为 0，因为结果完全确定。
- **图书馆的比喻**：想象一个图书馆里每本书都是独特的（高熵），找特定信息很难，但每本书都带来新信息。如果图书馆里 90% 的书都相同（低熵），大部分书都不提供新信息。

## 代码示例

```python
import numpy as np

def entropy(p, base=np.exp(1)):
    """计算离散分布的熵"""
    p = np.array(p)
    p = p[p > 0]  # 忽略概率为零的项
    return -np.sum(p * np.log(p)) / np.log(base)

# 不同分布的熵
uniform = [1/6] * 6  # 均匀骰子
biased = [0.7, 0.1, 0.05, 0.05, 0.05, 0.05]  # 有偏骰子
deterministic = [1.0, 0, 0, 0, 0, 0]  # 确定事件

print(f"均匀骰子熵: {entropy(uniform):.4f} 比特 (最大={np.log2(6):.4f})")
print(f"有偏骰子熵: {entropy(biased):.4f} 比特")
print(f"确定事件熵: {entropy(deterministic):.4f} 比特")

# 伯努利分布熵函数 H(p) = -p log p - (1-p) log (1-p)
def bernoulli_entropy(p):
    p = np.array(p)
    p = np.clip(p, 1e-10, 1-1e-10)  # 防止 log(0)
    return -p * np.log2(p) - (1-p) * np.log2(1-p)

ps = np.linspace(0, 1, 11)
for p in ps:
    h = bernoulli_entropy(p)
    print(f"Bernoulli(p={p:.1f}): H={h:.4f}")

# 联合熵与条件熵
# 假设 X, Y 的联合分布
joint = np.array([[0.1, 0.2], [0.3, 0.4]])
p_x = joint.sum(axis=1)  # P(X)
p_y = joint.sum(axis=0)  # P(Y)

H_XY = entropy(joint.flatten())
H_X = entropy(p_x)
H_Y = entropy(p_y)

# 条件熵 H(Y|X) = H(X,Y) - H(X)
H_Y_given_X = H_XY - H_X
H_X_given_Y = H_XY - H_Y

print(f"\n联合熵 H(X,Y) = {H_XY:.4f}")
print(f"边缘熵 H(X) = {H_X:.4f}, H(Y) = {H_Y:.4f}")
print(f"条件熵 H(Y|X) = {H_Y_given_X:.4f}")
print(f"条件熵 H(X|Y) = {H_X_given_Y:.4f}")

# 验证链式法则: H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)
print(f"H(X) + H(Y|X) = {H_X + H_Y_given_X:.4f} = H(X,Y)")
```

## 深度学习关联

- **经验风险最小化的信息论视角**：训练神经网络最小化负对数似然等价于最小化交叉熵（后文详述），其本质是最小化模型分布与数据分布之间的 KL 散度。熵 $H(p_{\text{data}})$ 是损失的下界。
- **Softmax 的温度参数 (Temperature)**：在知识蒸馏中，教师网络的 Softmax 输出使用温度 $\tau$ 缩放：$\text{Softmax}(z/\tau)$。高温使输出分布更均匀（熵更高），保留了更多的"暗知识"（类别间细粒度关系）。
- **置信度校准 (Confidence Calibration)**：良好的分类器应该输出与正确率匹配的置信度。熵可以衡量模型的不确定性——分布熵过高表示不确定（接近均匀分布），熵过低可能表示过度自信。期望校准误差 (ECE) 是评估这一性质的常用指标。
- **最大熵强化学习**：在强化学习中，最大熵方法（如 SAC）在奖励之外还最大化策略的熵 $H(\pi(\cdot|s))$，鼓励智能体保持探索行为，避免过早收敛到次优策略。
