# 生产环境调优清单

## 一、配置检查

```bash
# 1. 内存设置
maxmemory 4gb
maxmemory-policy allkeys-lru

# 2. 持久化
appendonly yes
aof-use-rdb-preamble yes
appendfsync everysec

# 3. 安全
requirepass strongpassword
rename-command FLUSHALL ""
rename-command FLUSHDB ""

# 4. 网络
bind 192.168.1.100
protected-mode yes
tcp-backlog 511
timeout 0
```

## 二、性能优化

```bash
# 1. 连接池
maxclients 10000

# 2. 线程（Redis 6.0+）
io-threads 4
io-threads-do-reads yes

# 3. 碎片整理
activedefrag yes

# 4. 内存分配
# 使用jemalloc
```

## 三、监控配置

```bash
# 1. 慢查询
slowlog-log-slower-than 10000
slowlog-max-len 128

# 2. 键空间通知（按需开启）
notify-keyspace-events ""

# 3. 内存碎片
activedefrag yes
```

## 四、高可用

```bash
# 1. 主从复制
min-replicas-to-write 1
min-replicas-max-lag 10

# 2. 哨兵（如使用）
sentinel monitor mymaster 192.168.1.100 6379 2

# 3. 集群（如使用）
cluster-enabled yes
cluster-node-timeout 5000
```
