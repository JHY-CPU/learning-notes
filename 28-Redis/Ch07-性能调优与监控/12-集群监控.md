# 集群监控

## 一、概念说明

监控Redis Cluster的状态，包括节点状态、槽位分布、数据迁移等。

## 二、集群状态查询

```bash
# 集群信息
redis-cli -c CLUSTER INFO
# cluster_state:ok
# cluster_slots_assigned:16384
# cluster_known_nodes:6
# cluster_size:3

# 节点信息
redis-cli -c CLUSTER NODES

# 槽位信息
redis-cli -c CLUSTER SLOTS
```

## 三、关键指标

```bash
# 集群状态
cluster_state:ok  # 正常
cluster_state:fail  # 异常

# 槽位覆盖
cluster_slots_assigned:16384  # 所有槽位已分配

# 节点状态
# master/fail?/fail/握手状态

# 数据迁移状态
cluster_slots_ok:16384
cluster_slots_pfail:0
cluster_slots_fail:0
```

## 四、监控脚本

```python
def check_cluster():
    r = redis.Redis(host='192.168.1.100', port=7000)
    
    info = r.cluster_info()
    
    if info['cluster_state'] != 'ok':
        alert("集群状态异常")
    
    if info['cluster_slots_assigned'] != 16384:
        alert("槽位未完全分配")
    
    if info['cluster_slots_pfail'] > 0:
        alert("有节点疑似下线")
```

## 五、注意事项

1. **槽位均衡**：监控槽位是否均匀分布
2. **节点状态**：监控所有节点状态
3. **数据迁移**：迁移期间监控性能
## 六、集群详细监控

```python
import redis

class ClusterMonitor:
    """Redis集群监控工具"""
    
    def __init__(self, host='localhost', port=7000):
        self.r = redis.Redis(host=host, port=port)
    
    def get_cluster_info(self):
        """获取集群信息"""
        return self.r.cluster_info()
    
    def get_cluster_nodes(self):
        """获取节点信息"""
        nodes = self.r.cluster_nodes()
        result = []
        for line in nodes.split('\n'):
            if line.strip():
                parts = line.split()
                if len(parts) >= 8:
                    result.append({
                        'id': parts[0],
                        'address': parts[1],
                        'flags': parts[2],
                        'master': parts[3] if parts[3] != '-' else None,
                        'ping_sent': parts[4],
                        'pong_recv': parts[5],
                        'config_epoch': parts[6],
                        'link_state': parts[7],
                        'slots': parts[8:] if len(parts) > 8 else []
                    })
        return result
    
    def get_slot_distribution(self):
        """获取槽位分布"""
        slots = self.r.cluster_slots()
        result = []
        for slot_range in slots:
            start = slot_range[0]
            end = slot_range[1]
            master = slot_range[2]
            result.append({
                'start': start,
                'end': end,
                'master': f"{master[0].decode()}:{master[1]}",
                'slot_count': end - start + 1
            })
        return result
    
    def check_health(self):
        """检查集群健康状态"""
        info = self.get_cluster_info()
        
        health = {
            'state': info['cluster_state'],
            'slots_assigned': int(info['cluster_slots_assigned']),
            'slots_ok': int(info['cluster_slots_ok']),
            'slots_pfail': int(info['cluster_slots_pfail']),
            'slots_fail': int(info['cluster_slots_fail']),
            'known_nodes': int(info['cluster_known_nodes']),
            'size': int(info['cluster_size']),
            'current_epoch': int(info['cluster_current_epoch']),
        }
        
        # 判断健康状态
        if health['state'] == 'ok' and health['slots_fail'] == 0:
            health['status'] = 'healthy'
        elif health['state'] == 'fail':
            health['status'] = 'critical'
        elif health['slots_pfail'] > 0:
            health['status'] = 'warning'
        else:
            health['status'] = 'unknown'
        
        return health
    
    def print_report(self):
        """打印报告"""
        health = self.check_health()
        
        print("Redis集群监控报告")
        print("=" * 50)
        print(f"状态: {health['state']}")
        print(f"健康: {health['status']}")
        print(f"已分配槽位: {health['slots_assigned']}/16384")
        print(f"正常槽位: {health['slots_ok']}")
        print(f"疑似故障槽位: {health['slots_pfail']}")
        print(f"故障槽位: {health['slots_fail']}")
        print(f"已知节点: {health['known_nodes']}")
        print(f"集群大小: {health['size']}")
        
        # 节点信息
        nodes = self.get_cluster_nodes()
        print(f"\n节点列表:")
        for node in nodes:
            print(f"  {node['address']} - {node['flags']} - {node['link_state']}")
        
        # 槽位分布
        slots = self.get_slot_distribution()
        print(f"\n槽位分布:")
        for slot in slots:
            print(f"  [{slot['start']}-{slot['end']}] → {slot['master']} ({slot['slot_count']} slots)")

# 使用
monitor = ClusterMonitor(host='192.168.1.100', port=7000)
monitor.print_report()
```

## 七、集群关键指标

```bash
# 集群状态
cluster_state:ok              # ok或fail
cluster_slots_assigned:16384  # 已分配槽位数
cluster_slots_ok:16384        # 正常槽位数
cluster_slots_pfail:0         # 疑似故障槽位
cluster_slots_fail:0          # 确认故障槽位

# 节点状态
cluster_known_nodes:6         # 已知节点数
cluster_size:3                # 主节点数
cluster_current_epoch:6       # 当前配置纪元

# 故障检测
# pfail: 疑似下线（可能网络抖动）
# fail: 确认下线（多数节点确认）

# 槽位迁移
cluster_slots_ok == 16384     # 所有槽位正常
cluster_slots_pfail == 0      # 无疑似故障
cluster_slots_fail == 0       # 无确认故障
```
