# 内存优化

## 一、概念说明

Redis内存优化包括选择合适的数据结构、控制Key大小、减少内存碎片等。

## 二、数据结构选择

```bash
# Hash使用ziplist编码（字段<128，值<64字节）
hash-max-ziplist-entries 128
hash-max-ziplist-value 64

# Set使用intset编码（元素都是整数且<512个）
set-max-intset-entries 512

# ZSet使用ziplist编码
zset-max-ziplist-entries 128
zset-max-ziplist-value 64

# 查看编码
OBJECT ENCODING key_name
```

## 三、内存优化技巧

```bash
# 1. 缩短Key名称
# 不好: user_profile_information_1001
# 好: u:1001:p

# 2. 使用Hash代替多个String
# 不好: SET user:1001:name "张三"
#        SET user:1001:age 25
# 好: HSET user:1001 name "张三" age 25

# 3. 设置过期时间
SET cache:key value EX 3600

# 4. 压缩Value
# 使用gzip/msgpack压缩大Value
```

## 四、内存碎片处理

```bash
# 查看碎片率
INFO memory | grep mem_fragmentation_ratio
# 正常: 1.0 - 1.5
# 过高: > 1.5（需要整理）
# 过低: < 1.0（可能使用了swap）

# 碎片整理（Redis 4.0+）
CONFIG SET activedefrag yes
# 自动整理内存碎片

# 手动触发
MEMORY PURGE
```

## 五、注意事项

1. **ziplist自动转换**：超过阈值自动转为hashtable
2. **碎片率监控**：定期检查碎片率
3. **swap检查**：确保Redis不使用swap
## 六、Value压缩策略

```python
import json
import gzip
import msgpack

class ValueCompressor:
    """Value压缩工具"""
    
    @staticmethod
    def compress_json(data):
        """JSON压缩"""
        json_str = json.dumps(data, ensure_ascii=False)
        return gzip.compress(json_str.encode('utf-8'))
    
    @staticmethod
    def decompress_json(compressed):
        """JSON解压"""
        json_str = gzip.decompress(compressed).decode('utf-8')
        return json.loads(json_str)
    
    @staticmethod
    def compress_msgpack(data):
        """MessagePack压缩"""
        return msgpack.packb(data, use_bin_type=True)
    
    @staticmethod
    def decompress_msgpack(compressed):
        """MessagePack解压"""
        return msgpack.unpackb(compressed, raw=False)

# 内存对比
# 原始JSON: {"name":"张三","age":25,"address":"北京市朝阳区"}
# 大小: 约70 bytes

# GZIP压缩后: 约50 bytes (节省28%)
# MessagePack: 约45 bytes (节省36%)

# 使用场景
# 大Value（>1KB）→ GZIP压缩
# 小Value（<1KB）→ MessagePack
# 极小Value → 不压缩
```

## 七、Key优化最佳实践

```bash
# Key命名规范
# 格式: 业务:模块:标识
# 不好: user_profile_information_1001 (30 bytes)
# 好: u:1001:p (9 bytes)

# Key长度对比
# 长Key: order_detail_information_for_user_1001 (45 bytes)
# 短Key: o:1001:d (9 bytes)
# 节省: 36 bytes/key

# 100万Key节省
# 100万 * 36 bytes = 36 MB

# 但不要过度压缩
# 可读性 > 极致压缩
# 建议Key长度: 10-50 bytes
```

## 八、数据结构编码优化

```bash
# Hash编码优化
# ziplist条件（Redis 7.0+使用listpack）
hash-max-listpack-entries 128
hash-max-listpack-value 64
# 字段数<=128 且 值<=64字节 → listpack编码
# 超过条件 → hashtable编码

# Set编码优化
set-max-intset-entries 512
# 元素都是整数 且 数量<=512 → intset编码
# 否则 → hashtable编码

# ZSet编码优化
zset-max-listpack-entries 128
zset-max-listpack-value 64
# 元素数<=128 且 值<=64字节 → listpack编码
# 否则 → skiplist编码

# List编码优化
list-max-listpack-size -2
# -2: 每个节点最多8192字节
# -1: 每个节点最多4096字节
# 正数: 每个节点最多N个元素

# 查看编码
OBJECT ENCODING key_name
```

## 九、内存优化Python工具

```python
import redis

r = redis.Redis()

def analyze_memory():
    """分析Redis内存使用"""
    info = r.info('memory')
    
    print("内存分析报告")
    print("=" * 50)
    print(f"已用内存: {info['used_memory_human']}")
    print(f"峰值内存: {info['used_memory_peak_human']}")
    print(f"碎片率: {info['mem_fragmentation_ratio']:.2f}")
    print(f"内存分配器: {info.get('mem_allocator', 'N/A')}")
    
    # 分析各数据类型
    type_stats = {}
    cursor = 0
    sample_size = 10000
    
    for _ in range(sample_size):
        cursor, keys = r.scan(cursor, count=100)
        for key in keys:
            key_type = r.type(key).decode()
            memory = r.memory_usage(key) or 0
            encoding = r.object('encoding', key).decode()
            
            if key_type not in type_stats:
                type_stats[key_type] = {'count': 0, 'memory': 0, 'encodings': {}}
            
            type_stats[key_type]['count'] += 1
            type_stats[key_type]['memory'] += memory
            
            if encoding not in type_stats[key_type]['encodings']:
                type_stats[key_type]['encodings'][encoding] = 0
            type_stats[key_type]['encodings'][encoding] += 1
        
        if cursor == 0:
            break
    
    print("\n数据类型统计:")
    for key_type, stats in type_stats.items():
        avg_memory = stats['memory'] / stats['count'] if stats['count'] else 0
        print(f"  {key_type}: {stats['count']} 个, "
              f"总内存 {stats['memory']/1024:.1f}KB, "
              f"平均 {avg_memory:.0f} bytes/key")
        for encoding, count in stats['encodings'].items():
            print(f"    {encoding}: {count}")

analyze_memory()
```
