# 分布式协调

## 一、概念说明

Redis可用于分布式系统的协调任务，如服务注册发现、配置同步、分布式通知等。

## 二、服务注册与发现

```bash
# 服务注册
HSET service:api:node1 host 192.168.1.1
HSET service:api:node1 port 8080
HSET service:api:node1 status healthy
EXPIRE service:api:node1 30

# 心跳续期
# 每10秒续期
EXPIRE service:api:node1 30

# 服务发现
HKEYS service:api:*
# 获取所有api服务节点

# 获取健康节点
HSCAN service:api: 0 MATCH * COUNT 100
```

## 三、配置同步

```bash
# 存储配置
HSET config:app max_connections 100
HSET config:app timeout 30

# 配置变更通知
PUBLISH config:changed "max_connections"

# 客户端订阅
SUBSCRIBE config:changed
# 收到通知后重新加载配置
```

## 四、分布式通知

```bash
# 简单通知
LPUSH notifications:user:1001 "有新消息"
EXPIRE notifications:user:1001 86400

# 批量通知
PUBLISH channel:all "系统维护通知"

# 事件通知
XADD events * type "deploy" service "api" version "1.0.0"
```

## 五、分布式Leader选举

```bash
# 竞选Leader
SET leader "node1" NX EX 30
# 获取成功成为Leader

# Leader续期
EXPIRE leader 30

# 其他节点检查
GET leader
# 返回当前Leader
```

## 六、注意事项

1. **过期时间**：服务注册必须设置过期
2. **心跳机制**：定期续期防止服务过期
3. **一致性**：最终一致性，不保证强一致
4. **性能**：大量服务注册时的性能考虑
5. **替代方案**：ZooKeeper/etcd更适合协调任务

## 七、分布式信号量

```python
import redis
import time
import uuid

r = redis.Redis()

class DistributedSemaphore:
    def __init__(self, key, permits, expire=30):
        self.key = f"semaphore:{key}"
        self.permits = permits
        self.expire = expire
        self.token = str(uuid.uuid4())
    
    def acquire(self, timeout=10):
        """获取许可"""
        end_time = time.time() + timeout
        
        while time.time() < end_time:
            # 使用ZSet存储许可
            # score为过期时间
            now = time.time()
            
            # 清理过期许可
            r.zremrangebyscore(self.key, 0, now)
            
            # 检查是否还有许可
            current_count = r.zcard(self.key)
            if current_count < self.permits:
                # 尝试获取许可
                if r.zadd(self.key, {self.token: now + self.expire}):
                    return True
            
            time.sleep(0.1)
        
        return False
    
    def release(self):
        """释放许可"""
        r.zrem(self.key, self.token)
    
    def available(self):
        """获取可用许可数"""
        now = time.time()
        r.zremrangebyscore(self.key, 0, now)
        return self.permits - r.zcard(self.key)

# 使用示例
semaphore = DistributedSemaphore("database", permits=10)

if semaphore.acquire(timeout=5):
    try:
        print("获取许可，执行数据库操作")
    finally:
        semaphore.release()
else:
    print("获取许可超时")
```

## 八、分布式栅栏

```python
class DistributedBarrier:
    def __init__(self, key, parties):
        self.key = f"barrier:{key}"
        self.parties = parties
    
    def wait(self, timeout=30):
        """等待所有参与者到达"""
        # 原子性递增并获取当前值
        lua = """
        local current = redis.call('INCR', KEYS[1])
        if current == 1 then
            redis.call('EXPIRE', KEYS[1], ARGV[1])
        end
        return current
        """
        current = r.eval(lua, 1, self.key, timeout)
        
        if current >= self.parties:
            # 所有参与者已到达
            return True
        
        # 等待
        end_time = time.time() + timeout
        while time.time() < end_time:
            count = int(r.get(self.key) or 0)
            if count >= self.parties:
                return True
            time.sleep(0.1)
        
        return False
    
    def reset(self):
        """重置栅栏"""
        r.delete(self.key)

# 使用示例
barrier = DistributedBarrier("task", parties=3)

# 在各节点执行
if barrier.wait(timeout=60):
    print("所有节点已到达，继续执行")
else:
    print("等待超时")
```

## 九、分布式锁优化：Redlock

```python
class Redlock:
    """多节点分布式锁，避免单点故障"""
    def __init__(self, nodes):
        self.nodes = [redis.Redis(host=h, port=p) for h, p in nodes]
        self.quorum = len(nodes) // 2 + 1
    
    def lock(self, resource, ttl=10000):
        token = str(uuid.uuid4())
        start = time.time() * 1000
        
        acquired = sum(
            1 for node in self.nodes
            if node.set(resource, token, nx=True, px=ttl)
        )
        
        elapsed = time.time() * 1000 - start
        
        if acquired >= self.quorum and elapsed < ttl:
            return token
        
        self.unlock(resource, token)
        return None
    
    def unlock(self, resource, token):
        lua = """
        if redis.call('get', KEYS[1]) == ARGV[1] then
            return redis.call('del', KEYS[1])
        end
        return 0
        """
        for node in self.nodes:
            try:
                node.eval(lua, 1, resource, token)
            except:
                pass
```
