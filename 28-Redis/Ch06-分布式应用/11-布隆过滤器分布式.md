# 布隆过滤器分布式

## 一、概念说明

Redis布隆过滤器用于分布式系统中的去重和存在性判断，内存效率极高，有极小的误判率。

## 二、基本操作

```bash
# 加载模块
MODULE LOAD redisbloom.so

# 创建布隆过滤器
BF.RESERVE myfilter 0.01 1000000
# 0.01: 1%误判率
# 1000000: 预计100万元素

# 添加元素
BF.ADD myfilter "user:1001"
BF.MADD myfilter "user:1002" "user:1003"

# 检查元素
BF.EXISTS myfilter "user:1001"  # 1(可能存在)
BF.EXISTS myfilter "user:9999"  # 0(一定不存在)
```

## 四、Cuckoo Filter

```bash
# 支持删除的布隆过滤器
CF.RESERVE mycuckoo 1000000
CF.ADD mycuckoo "item1"
CF.EXISTS mycuckoo "item1"  # 1
CF.DEL mycuckoo "item1"
CF.EXISTS mycuckoo "item1"  # 0
```

## 五、注意事项

1. **误判率**：存在一定误判，可能误判存在
2. **不支持删除**：标准布隆过滤器不支持
3. **容量规划**：预估准确的元素数量
4. **集群部署**：所有节点共享同一个布隆过滤器
5. **内存效率**：比Set节省90%以上内存

## 六、容量规划与内存计算

```bash
# 公式
# m = -(n * ln(p)) / (ln(2))^2
# k = (m / n) * ln(2)

# 示例计算
# n = 100万, p = 0.01 (1%)
# m = -(1000000 * ln(0.01)) / (ln(2))^2
# m = -(1000000 * -4.605) / 0.480
# m = 9593750 bits ≈ 1.14 MB
# k = (9593750 / 1000000) * 0.693 ≈ 7

# 内存对比
# 布隆过滤器：1.14 MB 存储100万元素
# Set：约 40 MB 存储100万元素（40字节/元素）
# Cuckoo：约 2 MB 存储100万元素
# 节省约 97% 的内存
```

## 七、与应用集成

```python
from redis import Redis
from redisbloom.client import Client

class BloomFilterCache:
    """布隆过滤器+缓存集成"""
    
    def __init__(self):
        self.rb = Client()
        self.r = Redis()
        self.filter_name = "cache_filter"
        
        # 创建布隆过滤器
        try:
            self.rb.bfCreate(self.filter_name, 0.01, 1000000)
        except:
            pass  # 已存在
    
    def get_or_set(self, key, db_query_func, ttl=3600):
        """带布隆过滤器的缓存查询"""
        # 1. 检查布隆过滤器
        if not self.rb.bfExists(self.filter_name, key):
            return None  # 一定不存在
        
        # 2. 查缓存
        cached = self.r.get(key)
        if cached:
            if cached == b'NULL':
                return None
            return json.loads(cached)
        
        # 3. 查数据库
        value = db_query_func()
        
        if value:
            self.r.setex(key, ttl, json.dumps(value))
        else:
            # 缓存空值
            self.r.setex(key, 300, 'NULL')
        
        return value
    
    def add(self, key):
        """添加Key到布隆过滤器"""
        self.rb.bfAdd(self.filter_name, key)
    
    def add_batch(self, keys):
        """批量添加"""
        self.rb.bfMAdd(self.filter_name, *keys)

# 使用
cache = BloomFilterCache()

# 注册存在的Key
cache.add("user:1001")
cache.add("user:1002")

# 查询
result = cache.get_or_set("user:1001", lambda: query_user(1001))
result = cache.get_or_set("user:9999", lambda: query_user(9999))  # 布隆过滤器拦截
```

## 八、定时重建策略

```python
class BloomFilterRebuilder:
    """布隆过滤器定时重建"""
    
    def __init__(self):
        self.rb = Client()
        self.r = Redis()
    
    def rebuild(self, data_source):
        """重建布隆过滤器"""
        # 删除旧过滤器
        try:
            self.rb.bfCreate("new_filter", 0.01, 1000000)
        except:
            pass
        
        # 批量添加所有Key
        batch = []
        for key in data_source:
            batch.append(key)
            if len(batch) >= 1000:
                self.rb.bfMAdd("new_filter", *batch)
                batch = []
        
        if batch:
            self.rb.bfMAdd("new_filter", *batch)
        
        # 原子替换
        self.r.rename("new_filter", "cache_filter")
        print("布隆过滤器重建完成")

# 定时任务
import schedule

rebuilder = BloomFilterRebuilder()

def rebuild_task():
    # 从数据库获取所有有效Key
    valid_keys = get_all_valid_keys_from_db()
    rebuilder.rebuild(valid_keys)

# 每天凌晨3点重建
schedule.every().day.at("03:00").do(rebuild_task)
```
