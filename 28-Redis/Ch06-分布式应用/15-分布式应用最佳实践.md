```bash
# 1. 缓存层设计
# 应用 → 本地缓存 → Redis → 数据库

# 2. 数据分片
# 按业务模块分库
# user:* → db0
# order:* → db1

# 3. 读写分离
# 写主节点，读从节点
```

## 二、数据一致性

```bash
# 最终一致性方案
# 1. 先更新DB，再删缓存
# 2. 使用消息队列同步
# 3. Canal监听binlog

# 强一致性方案
# 1. 分布式锁
# 2. 数据库为主，Redis为辅
```
