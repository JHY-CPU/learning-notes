# 缓存监控

## 一、概念说明

缓存监控是跟踪Redis缓存的性能和健康状态，包括命中率、内存使用、连接数等关键指标。

## 二、关键指标

### 命中率

```bash
# 查看命中率
INFO stats
# keyspace_hits:100000
# keyspace_misses:10000

# 计算命中率
# 命中率 = hits / (hits + misses)
# 100000 / 110000 ≈ 90.9%

# 健康标准
# > 90% 优秀
# 80-90% 正常
# < 80% 需要优化
```

### 内存使用

```bash
INFO memory
# used_memory_human:2.50GB
# maxmemory_human:4.00GB
# mem_fragmentation_ratio:1.20

# 内存使用率 = used / max
# 2.50 / 4.00 = 62.5%

# 碎片率
# 1.0-1.5 正常
# > 1.5 需要关注
# < 1.0 可能使用了swap
```

### 连接数

```bash
INFO clients
# connected_clients:100
# blocked_clients:5

# 设置最大连接数
maxclients 10000

# 连接数过多可能是连接池泄漏
```

## 五、告警规则

```bash
# 命中率低于80%
keysace_hits / (keyspace_hits + keypace_misses) < 0.8

# 内存使用超过80%
used_memory / maxmemory > 0.8

# 连接数超过80%
connected_clients / maxclients > 0.8

# 慢查询突增
slowlog_len > 100
```

## 六、注意事项

1. **定期检查**：每天检查关键指标
2. **趋势分析**：关注指标变化趋势
3. **基线对比**：建立性能基线
4. **告警响应**：及时响应告警
5. **容量预警**：提前预警容量不足

## 七、Python监控脚本

```python
import redis
import time
import json
from datetime import datetime

class RedisMonitor:
    def __init__(self, host='localhost', port=6379):
        self.r = redis.Redis(host=host, port=port)
        self.metrics_history = []
    
    def collect_metrics(self):
        """收集所有关键指标"""
        info = self.r.info()
        
        metrics = {
            'timestamp': datetime.now().isoformat(),
            # 内存指标
            'used_memory': info['used_memory'],
            'used_memory_human': info['used_memory_human'],
            'maxmemory': info.get('maxmemory', 0),
            'mem_fragmentation_ratio': info['mem_fragmentation_ratio'],
            
            # 性能指标
            'instantaneous_ops_per_sec': info['instantaneous_ops_per_sec'],
            'keyspace_hits': info['keyspace_hits'],
            'keyspace_misses': info['keyspace_misses'],
            
            # 连接指标
            'connected_clients': info['connected_clients'],
            'blocked_clients': info['blocked_clients'],
            
            # 持久化指标
            'rdb_last_bgsave_status': info['rdb_last_bgsave_status'],
            'aof_last_rewrite_status': info.get('aof_last_bgrewrite_status', 'N/A'),
            
            # 复制指标
            'role': info['role'],
            'connected_slaves': info.get('connected_slaves', 0),
        }
        
        # 计算命中率
        total = metrics['keyspace_hits'] + metrics['keyspace_misses']
        metrics['hit_rate'] = metrics['keyspace_hits'] / total * 100 if total > 0 else 0
        
        # 计算内存使用率
        if metrics['maxmemory'] > 0:
            metrics['memory_usage_rate'] = metrics['used_memory'] / metrics['maxmemory'] * 100
        else:
            metrics['memory_usage_rate'] = 0
        
        return metrics
    
    def check_alerts(self, metrics):
        """检查告警条件"""
        alerts = []
        
        if metrics['hit_rate'] < 80:
            alerts.append(f"命中率过低: {metrics['hit_rate']:.1f}%")
        
        if metrics['memory_usage_rate'] > 80:
            alerts.append(f"内存使用过高: {metrics['memory_usage_rate']:.1f}%")
        
        if metrics['mem_fragmentation_ratio'] > 1.5:
            alerts.append(f"内存碎片过高: {metrics['mem_fragmentation_ratio']:.2f}")
        
        if metrics['connected_clients'] > 5000:
            alerts.append(f"连接数过多: {metrics['connected_clients']}")
        
        if metrics['blocked_clients'] > 100:
            alerts.append(f"阻塞客户端过多: {metrics['blocked_clients']}")
        
        return alerts
    
    def print_report(self, metrics):
        """打印监控报告"""
        print("=" * 50)
        print(f"Redis监控报告 - {metrics['timestamp']}")
        print("=" * 50)
        print(f"内存使用: {metrics['used_memory_human']}")
        print(f"内存使用率: {metrics['memory_usage_rate']:.1f}%")
        print(f"碎片率: {metrics['mem_fragmentation_ratio']:.2f}")
        print(f"命中率: {metrics['hit_rate']:.1f}%")
        print(f"QPS: {metrics['instantaneous_ops_per_sec']}")
        print(f"连接数: {metrics['connected_clients']}")
        print(f"阻塞客户端: {metrics['blocked_clients']}")
        
        alerts = self.check_alerts(metrics)
        if alerts:
            print("\n告警:")
            for alert in alerts:
                print(f"  - {alert}")
        print("=" * 50)
    
    def run(self, interval=60):
        """持续监控"""
        while True:
            metrics = self.collect_metrics()
            self.print_report(metrics)
            self.metrics_history.append(metrics)
            time.sleep(interval)

# 使用
monitor = RedisMonitor()
monitor.run(interval=60)
```

## 八、告警配置示例

```yaml
# Prometheus告警规则
groups:
  - name: redis_alerts
    rules:
      - alert: RedisHitRateLow
        expr: redis_keyspace_hits_total / (redis_keyspace_hits_total + redis_keyspace_misses_total) < 0.8
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "Redis命中率低于80%"
      
      - alert: RedisMemoryHigh
        expr: redis_memory_used_bytes / redis_memory_max_bytes > 0.8
        for: 5m
        labels:
          severity: critical
        annotations:
          summary: "Redis内存使用超过80%"
      
      - alert: RedisConnectionsHigh
        expr: redis_connected_clients > 5000
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "Redis连接数超过5000"
```
