# 加载优化

## 核心概念

资源加载是游戏体验的关键环节。加载时间过长会严重影响玩家体验，而加载峰值可能导致帧率骤降。优化方向包括减少加载时间、隐藏加载过程和预测性加载。

### 加载方式对比

| 方式 | 特点 | 适用场景 | 推荐度 |
|------|------|---------|--------|
| Resources.Load | 同步加载，打包时全量打包，无法热更 | 小型项目、原型 | 不推荐 |
| AssetBundle | 可热更，按需加载，手动管理依赖 | 中大型项目(旧方案) | 可选 |
| Addressables | 基于引用计数，自动管理依赖和热更 | 所有新项目 | **强烈推荐** |
| AssetDatabase | 仅Editor可用 | 开发调试 | 仅编辑器 |

### Addressables核心概念

- **Address**：资源的逻辑地址（字符串），与物理路径解耦
- **Label**：标签，用于批量加载一组资源
- **Group**：分组，控制打包策略和加载方式
- **Reference Counting**：引用计数，自动管理资源生命周期
- **Catalog**：资源目录，记录所有资源的地址和依赖关系

### 加载策略分类

1. **预加载(Preload)**：进入场景前加载所有必需资源，适合小场景
2. **流式加载(Streaming)**：根据玩家位置动态加载/卸载区域，适合开放世界
3. **按需加载(On-Demand)**：使用时才加载，减少初始内存占用
4. **预测性加载(Predictive)**：根据玩家行为预测下一步需要的资源

## 具体实现方法

### Addressables异步加载完整系统

```csharp
using System;
using System.Collections.Generic;
using System.Threading.Tasks;
using UnityEngine;
using UnityEngine.AddressableAssets;
using UnityEngine.ResourceManagement.AsyncOperations;

/// <summary>
/// 完整的资源加载管理器
/// 基于Addressables，支持异步加载、预加载、引用计数
/// </summary>
public class AssetLoader : MonoBehaviour
{
    public static AssetLoader Instance { get; private set; }

    // 已加载资源缓存（地址 -> 资源）
    private Dictionary<string, object> loadedAssets
        = new Dictionary<string, object>();

    // 加载中句柄（用于取消和跟踪）
    private Dictionary<string, AsyncOperationHandle> loadingHandles
        = new Dictionary<string, AsyncOperationHandle>();

    void Awake()
    {
        if (Instance != null) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
    }

    /// <summary>
    /// 异步加载单个资源
    /// </summary>
    public async Task<T> LoadAsync<T>(string address) where T : class
    {
        // 已缓存则直接返回
        if (loadedAssets.TryGetValue(address, out var cached))
            return cached as T;

        // 正在加载中则等待
        if (loadingHandles.TryGetValue(address, out var existingHandle))
        {
            await existingHandle.Task;
            return loadedAssets[address] as T;
        }

        // 开始加载
        var handle = Addressables.LoadAssetAsync<T>(address);
        loadingHandles[address] = handle;

        await handle.Task;

        loadingHandles.Remove(address);

        if (handle.Status == AsyncOperationStatus.Succeeded)
        {
            loadedAssets[address] = handle.Result;
            return handle.Result;
        }

        Debug.LogError($"Failed to load asset: {address}, Error: {handle.OperationException}");
        return null;
    }

    /// <summary>
    /// 异步实例化GameObject
    /// </summary>
    public async Task<GameObject> InstantiateAsync(string address,
        Vector3 position, Quaternion rotation, Transform parent = null)
    {
        var handle = Addressables.InstantiateAsync(address, position, rotation, parent);
        await handle.Task;

        if (handle.Status == AsyncOperationStatus.Succeeded)
            return handle.Result;

        Debug.LogError($"Failed to instantiate: {address}");
        return null;
    }

    /// <summary>
    /// 预加载Label下的所有资源
    /// </summary>
    public async Task PreloadByLabel(string label,
        Action<float> onProgress = null)
    {
        // 获取所有资源位置
        var locationsHandle
            = Addressables.LoadResourceLocationsAsync(label);
        await locationsHandle.Task;

        if (locationsHandle.Status != AsyncOperationStatus.Succeeded)
        {
            Debug.LogError($"Failed to get locations for label: {label}");
            return;
        }

        var locations = locationsHandle.Result;
        int total = locations.Count;
        int loaded = 0;

        // 并行加载（限制并发数）
        var tasks = new List<Task>();
        int maxConcurrent = 4;
        int currentConcurrent = 0;

        foreach (var loc in locations)
        {
            // 等待并发槽位
            while (currentConcurrent >= maxConcurrent)
            {
                await Task.Delay(10);
            }

            currentConcurrent++;
            var task = Task.Run(async () =>
            {
                var handle = Addressables.LoadAssetAsync<object>(loc);
                await handle.Task;

                if (handle.Status == AsyncOperationStatus.Succeeded)
                    loadedAssets[loc.PrimaryKey] = handle.Result;

                loaded++;
                onProgress?.Invoke((float)loaded / total);
                currentConcurrent--;
            });
            tasks.Add(task);
        }

        await Task.WhenAll(tasks);
        Addressables.Release(locationsHandle);
    }

    /// <summary>
    /// 释放资源
    /// </summary>
    public void Release(string address)
    {
        if (loadedAssets.TryGetValue(address, out var asset))
        {
            Addressables.Release(asset);
            loadedAssets.Remove(address);
        }
    }

    /// <summary>
    /// 释放所有资源
    /// </summary>
    public void ReleaseAll()
    {
        foreach (var asset in loadedAssets.Values)
            Addressables.Release(asset);
        loadedAssets.Clear();

        foreach (var handle in loadingHandles.Values)
            if (handle.IsValid()) Addressables.Release(handle);
        loadingHandles.Clear();
    }

    void OnDestroy()
    {
        ReleaseAll();
    }
}
```

### 场景流式加载系统

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;
using UnityEngine;
using UnityEngine.AddressableAssets;
using UnityEngine.ResourceManagement.AsyncOperations;

/// <summary>
/// 开放世界流式加载系统
/// 根据玩家位置动态加载/卸载世界区域
/// </summary>
public class WorldStreamer : MonoBehaviour
{
    [Header("配置")]
    [SerializeField] private Transform player;
    [SerializeField] private float loadRadius = 200f;
    [SerializeField] private float unloadRadius = 300f;
    [SerializeField] private float checkInterval = 1f;
    [SerializeField] private int maxLoadsPerFrame = 2;

    [Header("区域定义")]
    [SerializeField] private WorldChunk[] chunks;

    [System.Serializable]
    public class WorldChunk
    {
        public string chunkId;
        public string address;     // Addressable地址
        public Vector3 center;
        public float radius;
        public bool isLoaded;
        public GameObject instance;
    }

    // 已加载区域的句柄
    private Dictionary<string, AsyncOperationHandle<GameObject>> loadedChunks
        = new Dictionary<string, AsyncOperationHandle<GameObject>>();

    private float checkTimer;
    private Queue<WorldChunk> loadQueue = new Queue<WorldChunk>();
    private Queue<WorldChunk> unloadQueue = new Queue<WorldChunk>();

    void Update()
    {
        if (player == null) return;

        checkTimer += Time.deltaTime;
        if (checkTimer < checkInterval) return;
        checkTimer = 0;

        // 检查需要加载/卸载的区域
        Vector3 playerPos = player.position;
        foreach (var chunk in chunks)
        {
            float dist = Vector3.Distance(playerPos, chunk.center);

            if (dist < loadRadius && !chunk.isLoaded)
                loadQueue.Enqueue(chunk);
            else if (dist > unloadRadius && chunk.isLoaded)
                unloadQueue.Enqueue(chunk);
        }

        // 分帧处理加载队列
        ProcessLoadQueue();
        ProcessUnloadQueue();
    }

    async void ProcessLoadQueue()
    {
        int loaded = 0;
        while (loadQueue.Count > 0 && loaded < maxLoadsPerFrame)
        {
            var chunk = loadQueue.Dequeue();
            if (chunk.isLoaded) continue;

            var handle = Addressables.InstantiateAsync(chunk.address);
            await handle.Task;

            if (handle.Status == AsyncOperationStatus.Succeeded)
            {
                chunk.instance = handle.Result;
                chunk.isLoaded = true;
                loadedChunks[chunk.chunkId] = handle;
            }
            loaded++;
        }
    }

    void ProcessUnloadQueue()
    {
        while (unloadQueue.Count > 0)
        {
            var chunk = unloadQueue.Dequeue();
            if (!chunk.isLoaded) continue;

            if (loadedChunks.TryGetValue(chunk.chunkId, out var handle))
            {
                Addressables.ReleaseInstance(handle);
                loadedChunks.Remove(chunk.chunkId);
            }

            chunk.instance = null;
            chunk.isLoaded = false;
        }
    }
}
```

### 加载进度UI

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;
using UnityEngine;
using UnityEngine.AddressableAssets;
using UnityEngine.UI;
using TMPro;

/// <summary>
/// 带进度显示的加载界面
/// </summary>
public class LoadingScreen : MonoBehaviour
{
    [SerializeField] private Slider progressBar;
    [SerializeField] private TextMeshProUGUI progressText;
    [SerializeField] private TextMeshProUGUI tipText;
    [SerializeField] private Image backgroundImage;

    private string[] loadingTips = {
        "提示: 合理使用道具可以让你事半功倍",
        "提示: 探索隐藏区域可以获得稀有装备",
        "提示: 升级技能可以提高战斗效率"
    };

    public async Task LoadWithProgress(List<string> addresses)
    {
        gameObject.SetActive(true);
        tipText.text = loadingTips[Random.Range(0, loadingTips.Length)];

        int total = addresses.Count;
        int loaded = 0;

        foreach (var addr in addresses)
        {
            await Addressables.LoadAssetAsync<Object>(addr).Task;
            loaded++;

            float progress = (float)loaded / total;
            progressBar.value = progress;
            progressText.text = $"Loading... {progress * 100:F0}% ({loaded}/{total})";

            // 随机更新提示
            if (loaded % 5 == 0)
                tipText.text = loadingTips[Random.Range(0, loadingTips.Length)];
        }

        progressText.text = "Complete!";
        await Task.Delay(500); // 短暂显示完成状态
        gameObject.SetActive(false);
    }
}
```

### 资源依赖分析工具

```csharp
#if UNITY_EDITOR
using UnityEditor;
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// 资源依赖分析工具
/// 识别冗余打包和间接依赖
/// </summary>
public class DependencyAnalyzer : EditorWindow
{
    [MenuItem("Tools/Dependency Analyzer")]
    static void Open() => GetWindow<DependencyAnalyzer>();

    private Object targetAsset;
    private Vector2 scrollPos;
    private List<string> dependencies = new List<string>();

    void OnGUI()
    {
        targetAsset = EditorGUILayout.ObjectField(
            "Target Asset", targetAsset, typeof(Object), false);

        if (GUILayout.Button("Analyze Dependencies"))
            Analyze();

        if (GUILayout.Button("Find Unused Assets"))
            FindUnused();

        scrollPos = EditorGUILayout.BeginScrollView(scrollPos);
        foreach (var dep in dependencies)
        {
            EditorGUILayout.BeginHorizontal();
            EditorGUILayout.LabelField(dep);

            // 显示文件大小
            if (System.IO.File.Exists(dep))
            {
                long size = new System.IO.FileInfo(dep).Length;
                EditorGUILayout.LabelField(
                    $"{size / 1024}KB", GUILayout.Width(80));
            }
            EditorGUILayout.EndHorizontal();
        }
        EditorGUILayout.EndScrollView();
    }

    void Analyze()
    {
        dependencies.Clear();
        if (targetAsset == null) return;

        string path = AssetDatabase.GetAssetPath(targetAsset);
        string[] deps = AssetDatabase.GetDependencies(path, true);
        dependencies.AddRange(deps);

        long totalSize = 0;
        foreach (var dep in deps)
        {
            if (System.IO.File.Exists(dep))
                totalSize += new System.IO.FileInfo(dep).Length;
        }

        Debug.Log($"Dependencies of {path}: {deps.Length} files, " +
            $"Total size: {totalSize / (1024 * 1024)}MB");
    }

    void FindUnused()
    {
        dependencies.Clear();
        string[] allAssets = AssetDatabase.GetAllAssetPaths();
        var usedAssets = new HashSet<string>();

        foreach (string scene in EditorBuildSettings.scenes)
        {
            if (!scene.enabled) continue;
            string[] deps = AssetDatabase.GetDependencies(scene.path, true);
            foreach (string dep in deps)
                usedAssets.Add(dep);
        }

        foreach (string asset in allAssets)
        {
            if (asset.StartsWith("Assets/")
                && !usedAssets.Contains(asset)
                && !AssetDatabase.IsValidFolder(asset))
            {
                dependencies.Add(asset);
            }
        }

        Debug.Log($"Found {dependencies.Count} potentially unused assets");
    }
}
#endif
```

## 最佳实践

- 使用Addressables代替Resources和手动AssetBundle管理
- 场景切换时使用加载界面遮挡加载过程
- 在玩家不会立即到达的区域使用流式加载
- 资源按依赖关系分析工具定期审计，消除冗余
- 使用AsyncOperation的priority参数控制加载优先级
- 关键资源（主角模型、基础特效）设为不卸载
- 加载失败时有降级方案（显示错误提示、使用替代资源）
- 使用预加载标签(Preload Label)批量加载场景所需资源

## 常见陷阱与修复

**陷阱1：Addressables引用计数管理不当**
- 症状：资源提前释放（MissingReferenceException）或内存泄漏
- 修复：每次LoadAssetAsync必须对应一次Release，使用引用计数管理

**陷阱2：同步加载大量资源导致主线程卡顿**
- 症状：场景切换时卡顿数秒
- 修复：全部使用异步加载，显示加载界面

**陷阱3：未处理加载失败的情况**
- 症状：网络中断或磁盘空间不足时游戏崩溃
- 修复：检查AsyncOperationHandle.Status，提供降级方案

**陷阱4：预加载过多资源导致内存峰值过高**
- 症状：预加载时内存占用暴增，可能触发OOM
- 修复：限制预加载资源总量，使用优先级加载

**陷阱5：资源依赖分析遗漏间接依赖**
- 症状：运行时发现资源缺失
- 修复：使用`AssetDatabase.GetDependencies(path, true)`递归分析
