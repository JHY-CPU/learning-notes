# 数据隐私法规


## 数据隐私法规与AI伦理


伦理GDPRAI伦理


数据隐私法规保护个人信息，AI伦理确保AI的公平和透明。


## 主要隐私法规


```
GDPR（欧盟通用数据保护条例，2018年）：
- 适用：处理欧盟居民数据的所有组织
- 核心原则：
  - 合法性、公平性、透明性
  - 目的限制、数据最小化
  - 准确性、存储限制
  - 完整性和保密性
- 用户权利：
  - 访问权：获取个人数据副本
  - 更正权：修正不准确数据
  - 删除权（被遗忘权）：要求删除数据
  - 数据可携权：导出数据到其他服务
  - 反对权：反对数据处理
- 罚款：最高2000万欧元或全球营收4%

CCPA（加州消费者隐私法，2020年）：
- 适用：处理加州居民数据的大型企业
- 消费者权利：
  - 知道收集了哪些数据
  - 删除个人数据
  - 拒绝出售数据
- 比GDPR宽松，侧重于"出售"数据

PIPL（中国个人信息保护法，2021年）：
- 适用：处理中国境内个人信息的活动
- 知情同意原则
- 最小必要原则
- 跨境传输需安全评估
- 最高5000万元或营收5%罚款
```


## AI伦理


```
AI伦理核心议题：
┌─────────────────────────────────────────────┐
│  公平性 (Fairness)                           │
│  - 算法不应基于种族、性别等歧视               │
│  - AI招聘工具歧视女性案例                     │
│  - 面部识别对深色皮肤准确率低                 │
│                                             │
│  透明性 (Transparency)                       │
│  - AI决策过程应可解释                         │
│  - 用户有权知道正在与AI交互                   │
│  - 自动化决策的影响告知                       │
│                                             │
│  可问责性 (Accountability)                    │
│  - AI出错谁负责？开发者/部署者/使用者？        │
│  - 需要明确的责任链条                         │
│                                             │
│  隐私保护                                    │
│  - 训练数据的隐私保护                         │
│  - 联邦学习/差分隐私等技术手段                 │
│                                             │
│  安全性                                      │
│  - 对抗攻击的鲁棒性                           │
│  - 防止恶意使用                               │
└─────────────────────────────────────────────┘

EU AI Act（欧盟AI法案，2024年）：
- 基于风险的分级监管
- 不可接受风险：禁止（社会评分、实时面部识别）
- 高风险：严格监管（医疗、招聘、信贷）
- 有限风险：透明义务（聊天机器人）
- 最低风险：不限制（游戏、垃圾邮件过滤）

AI治理实践：
- AI影响评估
- 算法审计
- 人工监督机制
- AI红队测试
```


## 开发者的隐私实践


```
技术措施：
- 数据加密：传输中(TLS) + 静态(AES)
- 数据最小化：只收集必要数据
- 匿名化/假名化：减少可识别信息
- 访问控制：最小权限原则
- 审计日志：记录数据访问

组织措施：
- 数据保护官(DPO)
- 隐私影响评估(PIA)
- 数据泄露响应计划
- 员工隐私培训
- 供应商数据处理协议

设计原则(Privacy by Design)：
1. 隐私是默认设置
2. 隐私嵌入设计
3. 全生命周期保护
4. 正和而非零和
5. 端到端安全
6. 可见性和透明性
7. 尊重用户隐私
```


> **Note:** 开发者应践行Privacy by Design，在系统设计阶段就考虑隐私保护。
>         了解适用的隐私法规（GDPR/CCPA/PIPL）是合规的基础。
