# 软件专利与商业秘密


## 软件专利与商业秘密


知识产权专利商业秘密


软件专利和商业秘密是技术企业保护核心竞争力的两大法律工具。


## 软件专利基础


```
软件专利的特殊性：
- 软件本身（代码）通常受版权保护，不是专利客体
- 软件专利保护的是"技术方案"或"方法"
- 各国对软件专利的态度不同

可专利性条件（中国）：
1. 新颖性：申请日前未被公开
2. 创造性：与现有技术相比有突出的实质性特点
3. 实用性：能够制造或使用，并产生积极效果
4. 属于技术方案：解决技术问题、采用技术手段、获得技术效果

不可专利的对象：
- 智力活动的规则和方法（纯算法）
- 疾病的诊断和治疗方法
- 动物和植物品种
- 用原子核变换方法获得的物质
- 平面印刷品的图案/色彩

软件专利申请策略：
- 将算法与具体应用场景结合
- 强调技术效果和解决的技术问题
- 用流程图+文字描述权利要求
- 撰写多个从属权利要求（扩大保护范围）
```


## 专利流氓与防御策略


```
专利流氓 (Patent Troll / NPE)：
- 定义：不生产产品，专门通过专利诉讼获利的实体
- 特征：持有大量专利，寻找侵权目标，发起诉讼或威胁
- 影响：增加企业合规成本，抑制创新

典型案例：
- Uniloc：起诉Microsoft激活系统，获赔数亿美元
- VirnetX：起诉Apple的FaceTime和VPN技术
- Intellectual Ventures：持有数万专利的"专利银行"

防御策略：
1. 专利交叉许可（Cross-Licensing）
   - 双方互相授权使用对方专利
   - 减少诉讼风险，降低许可费用

2. 防御性专利池
   - Open Invention Network (OIN)：Linux生态防御联盟
   - LOT Network：成员承诺不向NPE转让专利
   - 企业间共享防御专利

3. 专利无效挑战
   - IPR (Inter Partes Review)：美国PTAB的专利复审
   - 提供现有技术(Prior Art)证明专利无效
   - 成本低于专利诉讼

4. 开源优先策略
   - 使用Apache 2.0许可证（含专利授权）
   - 参与专利承诺组织
   - 专利公开 (Defensive Publication)
```


## 商业秘密保护


```
商业秘密三要素（TRIPS协议）：
1. 秘密性 (Secrecy)：不为公众所知悉
2. 价值性 (Value)：具有商业价值
3. 保密性 (Measures)：权利人采取了合理保密措施

商业秘密 vs 专利：

┌──────────┬──────────────────┬──────────────────┐
│ 维度       │ 商业秘密           │ 专利              │
├──────────┼──────────────────┼──────────────────┤
│ 保护期限   │ 无期限（只要保密）  │ 20年（发明）      │
│ 保护成本   │ 低（内部管理）      │ 高（申请+维护费）  │
│ 公开要求   │ 不公开            │ 必须公开技术细节   │
│ 保护力度   │ 弱（反不正当竞争）  │ 强（排他性权利）   │
│ 侵权证明   │ 困难              │ 相对容易           │
│ 适用场景   │ 配方、算法、客户名单 │ 可逆向工程的技术   │
└──────────┴──────────────────┴──────────────────┘

商业秘密保护措施：
- 法律措施：保密协议(NDA)、竞业限制条款
- 技术措施：代码加密、访问控制、水印追踪
- 管理措施：分级管理、知密范围限制、离职审计
- 物理措施：门禁、监控、文件锁

Open Core模型：
- 核心产品开源（吸引用户和贡献者）
- 企业功能闭源收费（高级功能、支持服务）
- 典型案例：GitLab, Elastic, Confluent, MongoDB
- 风险：需平衡社区利益和商业利益
```


> **Note:** 软件企业应根据技术特点选择保护策略：容易逆向的技术申请专利，难以逆向的算法/配方用商业秘密保护。Apache 2.0许可证的专利条款提供了良好的防御基础。


## 专利实务操作


```
专利申请流程（中国）：
1. 专利检索：确认新颖性（CNIPA数据库）
2. 撰写申请文件：
   - 权利要求书（定义保护范围）
   - 说明书（详细技术描述）
   - 摘要和附图
3. 提交申请：CNIPA（国家知识产权局）
4. 初步审查 → 公开（18个月）
5. 实质审查（请求后3年内）
6. 授权/驳回 → 复审（可选）

权利要求撰写技巧：
- 独立权利要求：尽量宽泛（最大保护范围）
- 从属权利要求：逐步具体化（层层退守）
- 方法权利要求 + 装置权利要求（双重保护）

国际专利申请：
- PCT途径：一份申请覆盖150+国家
- 巴黎公约：12个月内可向多国申请优先权
- 成本：PCT约3-5万元，进入各国另需费用

专利组合管理：
- 核心专利：保护关键技术
- 外围专利：保护改进和应用
- 防御专利：防止竞争对手布局
- 定期评估：淘汰低价值专利（节省维护费）
```
