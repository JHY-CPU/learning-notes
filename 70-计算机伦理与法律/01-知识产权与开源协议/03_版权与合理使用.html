版权与合理使用

版权合理使用Creative Commons

理解版权法是开发者避免法律风险的基本功。

软件版权基础

版权 (Copyright) 自动产生： - 作品完成即自动获得版权，无需注册 - 保护表达形式（代码文本），不保护思想/算法 - 保护期限：作者终生 + 死后50年（中国）/ 70年（美国） 软件版权保护范围： - 源代码和目标代码的文本 - 用户界面的独创性表达 - 软件文档和技术手册 - 数据库结构的独创性编排 不受保护的部分： - 编程语言本身（语法和关键字） - 算法和数据结构（属于思想） - 标准化的接口规范 - 功能性需求 职务作品 vs 个人作品： - 职务作品：工作中开发的软件，版权归雇主（约定优先） - 个人作品：个人时间开发，版权归个人 - 注意劳动合同中的知识产权条款！

合理使用 (Fair Use)

美国合理使用的四要素测试： ┌──────────┬────────────────────────────────────┐ │ 要素 │ 说明 │ ├──────────┼────────────────────────────────────┤ │ 使用目的 │ 教育、研究、评论、新闻报道 > 商业用途 │ │ 作品性质 │ 事实性作品 > 创造性作品 │ │ 使用比例 │ 使用少量 > 使用大量 │ │ 市场影响 │ 不影响原作品市场 > 替代原作品市场 │ └──────────┴────────────────────────────────────┘ 中国著作权法合理使用（第24条）： 1. 个人学习、研究或欣赏 2. 介绍、评论或说明（适当引用） 3. 新闻报道中不可避免的引用 4. 课堂教学或科学研究 5. 国家机关执行公务 6. 图书馆等保存版本的复制 7. 免费表演已发表作品 8. 将已发表作品翻译成少数民族语言 9. 将已发表作品改为盲文 软件领域的合理使用： - 反向工程：为互操作性目的的反编译（欧盟合法） - 安全研究：为发现漏洞的代码分析 - 兼容性测试：验证接口兼容性 - 代码片段引用：教学和文档中的少量引用

Creative Commons 许可

CC许可组件： ┌──────────┬──────────────────────────────────┐ │ 组件 │ 含义 │ ├──────────┼──────────────────────────────────┤ │ BY (署名) │ 必须注明原作者 │ │ SA (相同方式)│ 衍生作品必须使用相同许可 │ │ NC (非商业) │ 禁止商业使用 │ │ ND (禁止演绎)│ 禁止修改原作品 │ └──────────┴──────────────────────────────────┘ CC许可组合（从宽松到严格）： - CC0：完全放弃权利，进入公共领域 - CC BY：只需署名 - CC BY-SA：署名 + 相同方式共享 - CC BY-NC：署名 + 非商业使用 - CC BY-NC-SA：署名 + 非商业 + 相同方式 - CC BY-ND：署名 + 禁止演绎 - CC BY-NC-ND：最严格，仅允许署名传播 适用场景： - CC0：数据集、API文档 - CC BY：教程、博客文章 - CC BY-SA：维基百科(Wikipedia) - CC BY-NC：学术论文、教育资源 注意：CC许可不适合软件！ - 软件应使用专门的开源许可证（MIT/GPL/Apache等） - CC许可未提供源代码分发条款 - CC许可不包含专利授权

DMCA与数字版权

DMCA (数字千年版权法) 核心条款： 1. 反规避条款 (§1201)： - 禁止绕过技术保护措施 (TPM) - 例外：安全研究、互操作性、加密研究 - 争议：DRM限制合理使用 2. 安全港条款 (§512)： - 平台不因用户上传侵权内容承担责任 - 条件：不知情 + 及时移除 (Notice & Takedown) - 适用：YouTube, GitHub, Cloudflare等 3. DMCA Takedown流程： 版权持有人 → 发送侵权通知 → 平台移除内容 用户 → 发送反通知(Counter-Notice) → 10-14天后恢复 → 版权持有人未起诉 → 内容保留 4. GitHub DMCA实践： - 公开的DMCA通知库（透明度） - 代码仓库可被完整禁用 - Fork链也会被影响 - 反通知流程需要真实身份 其他地区的类似法律： - 欧盟：数字单一市场版权指令 (DSM Directive) - 中国：信息网络传播权保护条例 - 日本：不正竞争防止法
开发者应了解：代码自动受版权保护；合理使用需综合四要素判断；CC许可适合文档和内容，不适合软件；DMCA安全港机制保护平台但要求及时响应侵权通知。
