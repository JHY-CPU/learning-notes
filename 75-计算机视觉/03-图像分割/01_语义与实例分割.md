# 语义分割与实例分割

*从FCN到SAM — 像素级视觉理解的核心技术*


**图像分割**是将图像划分为若干有意义区域的任务，是计算机视觉中像素级理解的核心。按照粒度不同分为三类：


| 类型 | 目标 | 是否区分同类实例 | 代表方法 |
| --- | --- | --- | --- |
| 语义分割 | 为每个像素分配类别标签 | 否（同类别合并） | FCN, DeepLab, U-Net |
| 实例分割 | 检测每个物体实例并给出mask | 是（同类分实例） | Mask R-CNN, YOLACT |
| 全景分割 | 语义+实例的统一 | 是（stuff+thing） | Panoptic FPN, Mask2Former |


**评估指标：**


- **mIoU（Mean IoU）**
   ：各类别IoU的平均值，语义分割最核心指标
- **PQ（Panoptic Quality）**
   ：PQ = SQ × RQ，用于全景分割
- **mAP**
   ：不同IoU阈值下的平均精度，用于实例分割


$$
IoU = |预测∩真实| / |预测∪真实|,　mIoU = (1/C) Σ IoU_c
$$


### 2.1 核心思想


FCN（2015年）是语义分割的开山之作，将分类网络中的全连接层替换为卷积层，使网络可以接受任意尺寸的输入并输出同尺寸的分割图。

输入图像
H×W×3
→
卷积骨干
(VGG/ResNet)
→
1×1 Conv
通道数=类别数
→
转置卷积
上采样
→
输出分割图
H×W×C

### 2.2 跳跃连接结构


```
FCN-32s: pool5上采样32倍（最粗糙）
FCN-16s: pool5上采样2倍 + pool4，再上采样16倍
FCN-8s:  pool5上采样2倍 + pool4上采样2倍 + pool3，再上采样8倍

# FCN-8s效果最好，融合了不同层次的特征
# 浅层特征：精细边缘
# 深层特征：语义信息
```


### 2.3 转置卷积（Transposed Convolution）


也叫反卷积（Deconvolution），用于将低分辨率特征图上采样到高分辨率：


- 输出尺寸 = (输入尺寸-1)×步长 + 核尺寸 - 2×padding
- 可能存在"棋盘效应"（Checkerboard Artifacts）
- 替代方案：双线性插值上采样 + 卷积


### 3.1 架构


U-Net（2015年）为医学图像分割设计，呈U型对称结构：


- **编码器（Contracting Path）**
   ：逐步下采样，提取语义特征
- **解码器（Expanding Path）**
   ：逐步上采样，恢复空间分辨率
- **跳跃连接**
   ：将编码器各层特征拼接到解码器对应层


```
编码器: Conv→Conv→Pool→Conv→Conv→Pool→...→Bottleneck
          ↓skip    ↓skip              ↓skip
解码器: UpConv→Concat→Conv→Conv→UpConv→Concat→Conv→Conv→...→1×1Conv→Output

# U-Net特点：
# 1. 拼接(Concat)而非相加(Add) — 保留更多特征
# 2. 数据增强策略：弹性变形(elastic deformation)
# 3. 加权损失函数：分离接触的同类物体
```


### 3.2 U-Net变体


| 变体 | 改进 |
| --- | --- |
| U-Net++ | 密集跳跃连接（nested dense skip connections） |
| U-Net 3+ | 全尺度跳跃连接，融合多尺度特征 |
| Attention U-Net | 跳跃连接中加入注意力门控 |
| TransUNet | 编码器用Transformer，解码器用CNN |
| Swin-UNet | 纯Swin Transformer的U型结构 |
| nnU-Net | 自适应框架，自动配置网络和训练策略 |


### 4.1 DeepLab v1（2015）


- **空洞卷积（Atrous/Dilated Convolution）**
   ：在不增加参数的情况下扩大感受野
- **CRF（条件随机场）**
   ：后处理细化分割边界


```
# 空洞卷积：rate=d 表示每隔d-1个像素采样
# 3×3卷积，rate=1: 感受野3×3（标准卷积）
# 3×3卷积，rate=2: 感受野5×5
# 3×3卷积，rate=4: 感受野9×9

# 输出尺寸不变，参数量不变，但感受野增大
```


### 4.2 DeepLab v2 — ASPP模块


**空洞空间金字塔池化（Atrous Spatial Pyramid Pooling）**：并行使用多个不同rate的空洞卷积，捕获多尺度上下文信息。


```
ASPP模块:
         ┌─ 1×1 Conv ─┐
Input → ├─ Atrous Conv (rate=6)  ├─→ Concat → 1×1 Conv → Output
         ├─ Atrous Conv (rate=12) │
         ├─ Atrous Conv (rate=18) │
         └─ Global Avg Pool ──────┘
```


### 4.3 DeepLab v3/v3+


- **v3改进**
   ：ASPP加入图像级特征（全局平均池化），去掉CRF
- **v3+改进**
   ：增加编码器-解码器结构，解码器细化分割边界
- 骨干网络采用ResNet-101或Xception
- 多尺度输入（Multi-grid策略）


```
# DeepLab v3+ 编码器-解码器结构
编码器: 骨干网络 + ASPP
解码器: 低层特征(4×下采样) → 1×1Conv → 上采样4×
        与编码器输出(上采样4×)拼接
        → 3×3Conv → 上采样4× → 输出
```

### 5.2 Mask分支


- 对每个ROI预测一个 K×m×m 的二值mask（K为类别数，m通常为28）
- 每个类别独立预测mask（解耦分类和分割）
- 使用
   **ROI Align**
   替代ROI Pooling，避免量化误差
- 损失函数：L = L_cls + L_box + L_mask


### 5.3 实例分割mask头


```
# Mask分支网络结构
ROI Align (14×14×256)
  → 4层 3×3 Conv (256通道)
  → 2层 3×3 Transposed Conv (256通道)
  → 1×1 Conv (K通道, Sigmoid)
  → 输出: K×28×28 二值mask
```


### 6.1 任务定义


全景分割（Panoptic Segmentation）统一了语义分割和实例分割：


- **Stuff（不可计数）**
   ：天空、道路、草地等无固定形状的区域 → 语义分割
- **Thing（可计数）**
   ：人、车、动物等有明确边界的物体 → 实例分割


$$
PQ = SQ × RQ = (∑ IoU / |TP|) × (|TP| / (|TP| + ½|FP| + ½|FN|))
$$


### 6.2 代表方法


| 方法 | 策略 |
| --- | --- |
| Panoptic FPN | 共享FPN特征，分别做语义和实例分割，后融合 |
| Mask2Former | 统一的Transformer解码器，同时处理stuff和thing |
| OneFormer | 一个模型通过任务token切换语义/实例/全景模式 |


### 7.1 核心思想


SAM（2023年，Meta）是视觉基础模型，能够根据**提示（prompt）**分割任意物体，实现"分割一切"。


- **支持的提示**
   ：点、框、文本、掩码
- **零样本迁移**
   ：无需针对新类别微调
- **训练数据**
   ：SA-1B数据集（1100万图像，10亿+ mask）


### 7.2 架构


- **图像编码器**
   ：ViT-H（预训练的Vision Transformer），提取图像嵌入
- **提示编码器**
   ：将点/框/文本编码为向量
- **mask解码器**
   ：轻量级Transformer解码器，融合图像和提示嵌入，输出mask


### 7.3 SAM 2（2024）


- 扩展到视频分割，引入
   **记忆机制**
   （Memory Bank）
- Hiera骨干网络（层次化ViT）
- 支持跨帧的时序一致性分割
