# 语义分割与实例分割

*从FCN到SAM — 像素级视觉理解的核心技术*

## 一、图像分割概述

**图像分割**是将图像划分为若干有意义区域的任务，是计算机视觉中像素级理解的核心。按照粒度不同分为三类：

| 类型 | 目标 | 是否区分同类实例 | 代表方法 |
| --- | --- | --- | --- |
| 语义分割 | 为每个像素分配类别标签 | 否（同类别合并） | FCN, DeepLab, U-Net |
| 实例分割 | 检测每个物体实例并给出mask | 是（同类分实例） | Mask R-CNN, YOLACT |
| 全景分割 | 语义+实例的统一 | 是（stuff+thing） | Panoptic FPN, Mask2Former |

**评估指标：**
- **mIoU（Mean IoU）**：各类别IoU的平均值，语义分割最核心指标
- **PQ（Panoptic Quality）**：PQ = SQ × RQ，用于全景分割

$$
IoU = |预测∩真实| / |预测∪真实|,　mIoU = (1/C) Σ IoU_c
$$

## 二、FCN（Fully Convolutional Network）

### 核心思想

FCN（2015年）是语义分割的开山之作，将分类网络中的全连接层替换为卷积层，使网络可以接受任意尺寸的输入并输出同尺寸的分割图。

### 跳跃连接结构

```
FCN-32s: pool5上采样32倍（最粗糙）
FCN-16s: pool5上采样2倍 + pool4，再上采样16倍
FCN-8s:  pool5上采样2倍 + pool4上采样2倍 + pool3，再上采样8倍
```

## 三、U-Net

U-Net（2015年）为医学图像分割设计，呈U型对称结构：

- **编码器（Contracting Path）**：逐步下采样，提取语义特征
- **解码器（Expanding Path）**：逐步上采样，恢复空间分辨率
- **跳跃连接**：将编码器各层特征拼接到解码器对应层

```
编码器: Conv→Conv→Pool→Conv→Conv→Pool→...→Bottleneck
          ↓skip    ↓skip              ↓skip
解码器: UpConv→Concat→Conv→Conv→UpConv→Concat→Conv→Conv→...→1×1Conv→Output
```

### U-Net 变体

| 变体 | 改进 |
| --- | --- |
| U-Net++ | 密集跳跃连接（nested dense skip connections） |
| TransUNet | 编码器用Transformer，解码器用CNN |
| Swin-UNet | 纯Swin Transformer的U型结构 |
| nnU-Net | 自适应框架，自动配置网络和训练策略 |

## 四、DeepLab 系列

### DeepLab v1（2015）

- **空洞卷积（Atrous Convolution）**：在不增加参数的情况下扩大感受野
- **CRF（条件随机场）**：后处理细化分割边界

```
# 空洞卷积：rate=d 表示每隔d-1个像素采样
# 3×3卷积，rate=1: 感受野3×3（标准卷积）
# 3×3卷积，rate=2: 感受野5×5
# 3×3卷积，rate=4: 感受野9×9
```

### DeepLab v2 — ASPP 模块

```
ASPP模块:
         ┌─ 1×1 Conv ─┐
Input → ├─ Atrous Conv (rate=6)  ├─→ Concat → 1×1 Conv → Output
         ├─ Atrous Conv (rate=12) │
         ├─ Atrous Conv (rate=18) │
         └─ Global Avg Pool ──────┘
```

### DeepLab v3+

```
编码器: 骨干网络 + ASPP
解码器: 低层特征(4×下采样) → 1×1Conv → 上采样4×
        与编码器输出(上采样4×)拼接
        → 3×3Conv → 上采样4× → 输出
```

## 五、Mask R-CNN

Mask R-CNN在Faster R-CNN基础上增加了**mask预测分支**，同时实现目标检测和实例分割。

```
骨干网络 + FPN → RPN → ROI Align → ┬─ 分类头
                                    ├─ 回归头
                                    └─ Mask头
```

### Mask 分支

```python
# Mask分支网络结构
# ROI Align (14×14×256)
#   → 4层 3×3 Conv (256通道)
#   → 2层 3×3 Transposed Conv (256通道)
#   → 1×1 Conv (K通道, Sigmoid)
#   → 输出: K×28×28 二值mask
```

## 六、全景分割

- **Stuff（不可计数）**：天空、道路、草地等无固定形状的区域
- **Thing（可计数）**：人、车、动物等有明确边界的物体

$$
PQ = SQ × RQ = (∑ IoU / |TP|) × (|TP| / (|TP| + ½|FP| + ½|FN|))
$$

## 七、SAM（Segment Anything）

SAM（2023年，Meta）是视觉基础模型，能够根据**提示（prompt）**分割任意物体。

- 支持点、框、文本、掩码作为提示
- 零样本迁移，无需微调
- 训练数据：SA-1B（1100万图像，10亿+ mask）

## 八、Python 实战：分割模型训练

### 示例：U-Net 语义分割训练

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class UNet(nn.Module):
    """简化的 U-Net 语义分割模型"""

    def __init__(self, in_channels=3, num_classes=21):
        super().__init__()

        # 编码器
        self.enc1 = self._double_conv(in_channels, 64)
        self.enc2 = self._double_conv(64, 128)
        self.enc3 = self._double_conv(128, 256)
        self.enc4 = self._double_conv(256, 512)

        self.pool = nn.MaxPool2d(2)

        # Bottleneck
        self.bottleneck = self._double_conv(512, 1024)

        # 解码器
        self.up4 = nn.ConvTranspose2d(1024, 512, 2, stride=2)
        self.dec4 = self._double_conv(1024, 512)
        self.up3 = nn.ConvTranspose2d(512, 256, 2, stride=2)
        self.dec3 = self._double_conv(512, 256)
        self.up2 = nn.ConvTranspose2d(256, 128, 2, stride=2)
        self.dec2 = self._double_conv(256, 128)
        self.up1 = nn.ConvTranspose2d(128, 64, 2, stride=2)
        self.dec1 = self._double_conv(128, 64)

        # 输出层
        self.out_conv = nn.Conv2d(64, num_classes, 1)

    def _double_conv(self, in_ch, out_ch):
        return nn.Sequential(
            nn.Conv2d(in_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True),
            nn.Conv2d(out_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True),
        )

    def forward(self, x):
        # 编码
        e1 = self.enc1(x)
        e2 = self.enc2(self.pool(e1))
        e3 = self.enc3(self.pool(e2))
        e4 = self.enc4(self.pool(e3))

        # 瓶颈
        b = self.bottleneck(self.pool(e4))

        # 解码 + 跳跃连接
        d4 = self.dec4(torch.cat([self.up4(b), e4], dim=1))
        d3 = self.dec3(torch.cat([self.up3(d4), e3], dim=1))
        d2 = self.dec2(torch.cat([self.up2(d3), e2], dim=1))
        d1 = self.dec1(torch.cat([self.up1(d2), e1], dim=1))

        return self.out_conv(d1)

# 训练函数
def train_segmentation(model, train_loader, num_classes, epochs=50, lr=1e-4):
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    model = model.to(device)

    optimizer = torch.optim.AdamW(model.parameters(), lr=lr)
    criterion = nn.CrossEntropyLoss(ignore_index=255)

    for epoch in range(epochs):
        model.train()
        total_loss = 0

        for images, masks in train_loader:
            images = images.to(device)
            masks = masks.to(device).long()

            optimizer.zero_grad()
            outputs = model(images)  # [B, C, H, W]
            loss = criterion(outputs, masks)
            loss.backward()
            optimizer.step()
            total_loss += loss.item()

        # 计算 mIoU
        if (epoch + 1) % 10 == 0:
            model.eval()
            ious = []
            with torch.no_grad():
                for images, masks in train_loader:
                    images = images.to(device)
                    preds = model(images).argmax(dim=1)
                    for cls in range(num_classes):
                        pred_mask = (preds == cls)
                        true_mask = (masks.to(device) == cls)
                        inter = (pred_mask & true_mask).sum().float()
                        union = (pred_mask | true_mask).sum().float()
                        if union > 0:
                            ious.append((inter / union).item())

            miou = sum(ious) / len(ious) if ious else 0
            print(f"Epoch {epoch+1}/{epochs}, Loss: {total_loss/len(train_loader):.4f}, mIoU: {miou:.4f}")

    return model
```

### 示例：使用 Torchvision 进行 Mask R-CNN 实例分割

```python
from torchvision.models.detection import maskrcnn_resnet50_fpn
from torchvision.models.detection.faster_rcnn import FastRCNNPredictor
from torchvision.models.detection.mask_rcnn import MaskRCNNPredictor

def get_mask_rcnn_model(num_classes):
    model = maskrcnn_resnet50_fpn(pretrained=True)

    # 替换 box predictor
    in_features = model.roi_heads.box_predictor.cls_score.in_features
    model.roi_heads.box_predictor = FastRCNNPredictor(in_features, num_classes)

    # 替换 mask predictor
    in_features_mask = model.roi_heads.mask_predictor.conv5_mask.in_channels
    model.roi_heads.mask_predictor = MaskRCNNPredictor(
        in_features_mask, 256, num_classes
    )

    return model

# 输出包含 boxes, labels, scores, masks
```

## 总结

- 语义分割为每个像素分配类别（FCN、DeepLab、U-Net），实例分割还要区分同类的不同个体（Mask R-CNN）
- DeepLab 系列的空洞卷积和 ASPP 模块是多尺度上下文信息聚合的经典方案
- U-Net 的编码器-解码器+跳跃连接结构是医学图像分割的标准范式
- SAM 将分割推向"分割一切"的新范式，支持零样本迁移
- 全景分割统一了语义分割和实例分割，Mask2Former 是当前最强方法之一
- 实践中使用 Torchvision 的预训练 Mask R-CNN 可以快速构建实例分割系统
