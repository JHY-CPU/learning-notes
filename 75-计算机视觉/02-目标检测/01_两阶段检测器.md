# 两阶段目标检测器

*R-CNN系列演进：从Region Proposal到端到端训练*

## 一、目标检测概述

**目标检测（Object Detection）**不仅要识别图像中物体的类别，还要定位其位置（通常用边界框Bounding Box表示）。两阶段检测器的核心思想是**先生成候选区域，再对每个区域分类和回归**。

### 检测任务的基本要素

- **分类（Classification）**：判断候选区域属于哪个类别
- **定位（Localization）**：预测边界框坐标 (x, y, w, h) 或 (x1, y1, x2, y2)
- **置信度（Confidence）**：评估检测结果的可信度

### 评估指标

| 指标 | 说明 |
| --- | --- |
| IoU（Intersection over Union） | 预测框与真实框的交并比，IoU ≥ 阈值（通常0.5）视为正确检测 |
| Precision（精确率） | TP / (TP + FP)，检测出的目标中正确的比例 |
| Recall（召回率） | TP / (TP + FN)，所有真实目标中被检测到的比例 |
| AP（Average Precision） | P-R曲线下面积，单类别的综合指标 |
| mAP（mean AP） | 所有类别AP的平均值，目标检测的核心指标 |

## 二、R-CNN

### 核心思想

R-CNN（Regions with CNN features）首次将CNN特征引入目标检测，开启了深度学习检测时代。

输入图像 → Selective Search 生成~2000个候选框 → 每个候选框 warp到227×227 → AlexNet 提取特征 → SVM分类 + 边框回归

### 缺陷

- 每个候选框独立过CNN前向传播（~2000次），速度极慢（47秒/张）
- 训练分多步，流程复杂
- Selective Search耗时且不可学习

## 三、Fast R-CNN

### 核心改进：共享卷积计算

输入图像 → CNN骨干网络（整图一次前向） → ROI Pooling → FC层 → 分类 + 回归（多任务联合训练）

### ROI Pooling

```python
from torchvision.ops import roi_pool

# ROI Pooling将不同大小的候选区域特征统一到固定尺寸
output = roi_pool(feature_maps, rois, output_size=(7, 7), spatial_scale=1/16)
```

### 多任务损失

$$
L = L_cls(p, u) + λ·[u≥1]·L_loc(t, v)
$$

## 四、Faster R-CNN

### RPN（Region Proposal Network）

Faster R-CNN用**区域提议网络（RPN）**替代Selective Search，实现端到端可学习的候选框生成。

```
特征图 → 3×3 Conv(256) → ┬─ 1×1 Conv(2×k) → 分类（前景/背景）
                          └─ 1×1 Conv(4×k) → 边框回归
```

### ROI Align

```python
from torchvision.ops import roi_align

output = roi_align(feature_maps, rois, output_size=(7, 7),
                   spatial_scale=1/16, sampling_ratio=2)
```

### FPN（Feature Pyramid Network）

```
自底向上路径（骨干网络）:
C1(1/2) → C2(1/4) → C3(1/8) → C4(1/16) → C5(1/32)

自顶向下路径 + 横向连接:
P5 = 1×1 Conv(C5)
P4 = 1×1 Conv(C4) + Upsample(P5)
P3 = 1×1 Conv(C3) + Upsample(P4)
P2 = 1×1 Conv(C2) + Upsample(P3)
```

## 五、Cascade R-CNN 与改进

使用多个检测头级联训练，每个阶段使用更高的IoU阈值，逐步精修检测框。

| 模型 | 核心改进 | 效果 |
| --- | --- | --- |
| Mask R-CNN | 增加mask分支，ROI Align | 同时做检测+分割 |
| Libra R-CNN | 均衡采样、特征融合、损失函数 | 解决样本和特征不平衡 |
| Grid R-CNN | 网格点预测替代边框回归 | 更好的空间定位 |

### 性能对比

| 模型 | 骨干网络 | 速度(FPS) | COCO mAP |
| --- | --- | --- | --- |
| Faster R-CNN | VGG-16 | 5 | 21.9 |
| Faster R-CNN+FPN | ResNet-101 | 5 | 36.2 |
| Cascade R-CNN | ResNet-101-FPN | 3.5 | 42.8 |

## 六、Python 实战：Faster R-CNN 训练

### 示例：使用 Torchvision 训练 Faster R-CNN

```python
import torch
import torchvision
from torchvision.models.detection import fasterrcnn_resnet50_fpn
from torchvision.models.detection.faster_rcnn import FastRCNNPredictor
from torch.utils.data import DataLoader
from PIL import Image
import numpy as np

# 1. 加载预训练模型并修改类别数
def get_faster_rcnn_model(num_classes):
    model = fasterrcnn_resnet50_fpn(pretrained=True)
    # 替换分类头
    in_features = model.roi_heads.box_predictor.cls_score.in_features
    model.roi_heads.box_predictor = FastRCNNPredictor(in_features, num_classes)
    return model

# 2. 自定义数据集格式
class DetectionDataset(torch.utils.data.Dataset):
    def __init__(self, image_paths, annotations, transforms=None):
        self.image_paths = image_paths
        self.annotations = annotations
        self.transforms = transforms

    def __len__(self):
        return len(self.image_paths)

    def __getitem__(self, idx):
        img = Image.open(self.image_paths[idx]).convert("RGB")
        target = self.annotations[idx]
        # target 格式: {"boxes": [[x1,y1,x2,y2], ...], "labels": [1, 2, ...]}
        target["boxes"] = torch.tensor(target["boxes"], dtype=torch.float32)
        target["labels"] = torch.tensor(target["labels"], dtype=torch.int64)

        if self.transforms:
            img = self.transforms(img)

        return img, target

# 3. 训练函数
def train_detector(model, train_loader, num_epochs=10, lr=0.005):
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    model.to(device)

    optimizer = torch.optim.SGD(model.parameters(), lr=lr,
                                 momentum=0.9, weight_decay=0.0005)
    lr_scheduler = torch.optim.lr_scheduler.StepLR(optimizer,
                                                     step_size=3, gamma=0.1)

    for epoch in range(num_epochs):
        model.train()
        total_loss = 0

        for images, targets in train_loader:
            images = [img.to(device) for img in images]
            targets = [{k: v.to(device) for k, v in t.items()} for t in targets]

            # Faster R-CNN 在训练时返回损失字典
            loss_dict = model(images, targets)
            losses = sum(loss for loss in loss_dict.values())

            optimizer.zero_grad()
            losses.backward()
            torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=10.0)
            optimizer.step()

            total_loss += losses.item()

        lr_scheduler.step()
        print(f"Epoch {epoch+1}/{num_epochs}, Loss: {total_loss/len(train_loader):.4f}")

    return model

# 4. 推理
def detect_objects(model, image_path, confidence_threshold=0.5):
    model.eval()
    device = next(model.parameters()).device

    from torchvision import transforms
    transform = transforms.ToTensor()
    img = Image.open(image_path).convert("RGB")
    img_tensor = transform(img).unsqueeze(0).to(device)

    with torch.no_grad():
        predictions = model(img_tensor)

    # 过滤低置信度检测
    pred = predictions[0]
    mask = pred["scores"] > confidence_threshold
    boxes = pred["boxes"][mask].cpu().numpy()
    labels = pred["labels"][mask].cpu().numpy()
    scores = pred["scores"][mask].cpu().numpy()

    return boxes, labels, scores
```

### 示例：自定义 Faster R-CNN 模块

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimplifiedRPN(nn.Module):
    """简化版 Region Proposal Network"""

    def __init__(self, in_channels=256, num_anchors=9):
        super().__init__()
        self.conv = nn.Conv2d(in_channels, in_channels, 3, padding=1)
        self.cls_layer = nn.Conv2d(in_channels, num_anchors * 2, 1)
        self.reg_layer = nn.Conv2d(in_channels, num_anchors * 4, 1)

        # 初始化
        for layer in [self.conv, self.cls_layer, self.reg_layer]:
            nn.init.normal_(layer.weight, std=0.01)
            nn.init.constant_(layer.bias, 0)

    def forward(self, feature_map):
        """
        Args:
            feature_map: [B, C, H, W] 骨干网络输出的特征图
        Returns:
            objectness: [B, H*W*num_anchors, 2] 前景/背景分数
            pred_boxes: [B, H*W*num_anchors, 4] 预测的框偏移量
        """
        t = F.relu(self.conv(feature_map))
        objectness = self.cls_layer(t)    # [B, 2*A, H, W]
        pred_boxes = self.reg_layer(t)    # [B, 4*A, H, W]

        # reshape
        B, _, H, W = objectness.shape
        objectness = objectness.permute(0, 2, 3, 1).reshape(B, -1, 2)
        pred_boxes = pred_boxes.permute(0, 2, 3, 1).reshape(B, -1, 4)

        return objectness, pred_boxes
```

## 总结

- 两阶段检测器的核心是"先提议后分类"：RPN 生成候选框 → ROI Pooling/Align 提取特征 → 分类和回归
- R-CNN → Fast R-CNN → Faster R-CNN 的演进实现了端到端训练和实时速度
- ROI Align 解决了 ROI Pooling 的量化误差，对实例分割尤为重要
- FPN 通过多尺度特征融合提升小目标检测能力
- Cascade R-CNN 通过级联多个检测头逐步提升定位精度
- 实践中推荐使用 Torchvision 的预训练 Faster R-CNN 进行微调
