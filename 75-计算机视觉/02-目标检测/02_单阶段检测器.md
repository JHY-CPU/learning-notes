# 单阶段目标检测器

*YOLO系列、SSD与Anchor-Free方法全解析*

## 一、单阶段检测器概述

与两阶段检测器（先生成候选框再分类）不同，**单阶段检测器（One-Stage Detector）**直接在特征图上同时预测类别和边界框，省去了区域提议步骤，因此**速度更快**，适合实时应用。

| 对比维度 | 两阶段检测器 | 单阶段检测器 |
| --- | --- | --- |
| 速度 | 较慢（5-20 FPS） | 更快（30-150+ FPS） |
| 精度 | 通常更高 | 略低（差距在缩小） |
| 小目标检测 | 更优 | 相对困难 |
| 密集目标 | 较弱 | 更强 |
| 代表模型 | Faster R-CNN, Cascade R-CNN | YOLO, SSD, RetinaNet |

## 二、YOLO 系列演进

### YOLO v1（2016）

YOLO（You Only Look Once）将目标检测重新定义为**回归问题**：将图像划分为S×S网格，每个网格预测B个边界框和C个类别概率。

```
# YOLO v1 输出: S×S×(B×5+C)
# S=7, B=2, C=20 (VOC)
# 输出张量: 7×7×30
```

### YOLO v2 / YOLO9000（2017）

| 改进 | 说明 |
| --- | --- |
| Batch Normalization | 所有卷积层后加BN，去掉Dropout |
| 高分辨率分类器 | 先在448×448上预训练分类，再微调检测 |
| Anchor Boxes | 借鉴Faster R-CNN使用anchor |
| Dimension Clusters | 用K-means在训练集上聚类anchor尺寸 |
| Passthrough层 | 将26×26特征拼接到13×13 |

### YOLO v3（2018）

引入多尺度预测，使用Darknet-53骨干网络。

```
# 多尺度检测
# 大尺度 (13×13): 检测大物体
# 中尺度 (26×26): 检测中等物体
# 小尺度 (52×52): 检测小物体
```

### YOLO v4-v8

| 版本 | 发布方 | 核心改进 |
| --- | --- | --- |
| YOLOv4 | Bochkovskiy | Mosaic增强, CSPDarknet53, SPP, PANet |
| YOLOv5 | Ultralytics | 自适应anchor, 模型缩放(n/s/m/l/x) |
| YOLOv6 | 美团 | EfficientRep + RepVGG风格重参数化 |
| YOLOv7 | WongKinYiu | E-ELAN结构、复合模型缩放 |
| YOLOv8 | Ultralytics | Anchor-free、解耦头、C2f模块 |

YOLOv8是目前最流行的YOLO版本，采用**Anchor-Free**设计，使用**解耦头（Decoupled Head）**分别做分类和回归。

## 三、SSD

SSD在多个不同尺度的特征图上同时检测目标。

```
conv4_3 (38×38): 检测小物体
conv7   (19×19): 检测中小物体
conv8_2 (10×10): 检测中物体
conv9_2 (5×5):   检测中大物体
conv10_2(3×3):   检测大物体
conv11_2(1×1):   检测超大物体
```

## 四、Focal Loss 与 RetinaNet

### 正负样本不平衡问题

$$
FL(p_t) = -α_t · (1 - p_t)^γ · log(p_t)
$$

```python
import torch.nn.functional as F

def focal_loss(pred, target, alpha=0.25, gamma=2.0):
    ce_loss = F.cross_entropy(pred, target, reduction='none')
    pt = torch.exp(-ce_loss)
    focal_weight = alpha * (1 - pt) ** gamma
    return (focal_weight * ce_loss).mean()
```

## 五、Anchor-Free 方法

### FCOS（Fully Convolutional One-Stage）

```python
# FCOS预测目标
# 每个位置预测:
#   - 分类: C维向量（类别概率）
#   - 回归: 4维向量 (l, t, r, b) — 到左、上、右、下边的距离
#   - Centerness: 1维标量（中心度）

# Centerness计算
# centerness = sqrt(min(l,r)/max(l,r) × min(t,b)/max(t,b))
```

### CenterNet

```python
# CenterNet输出
# 热力图 (H×W×C): 每个类别的中心点概率
# 尺寸回归 (H×W×2): 目标的宽和高
# 偏移回归 (H×W×2): 中心点的量化误差补偿
```

### Anchor-Free 方法对比

| 方法 | 预测形式 | 是否需要NMS |
| --- | --- | --- |
| FCOS | 逐像素预测到四边距离 | 需要 |
| CenterNet | 中心点热力图+尺寸 | 不需要 |
| YOLOv8 | 关键点+距离预测 | 需要 |

## 六、NMS 与损失函数

### NMS 变体

| 方法 | 说明 | 适用场景 |
| --- | --- | --- |
| Soft-NMS | 不直接删除重叠框，而是降低其置信度 | 密集目标场景 |
| DIoU-NMS | 使用DIoU替代IoU作为抑制准则 | 考虑中心点距离 |

### 损失函数演进

| 损失函数 | 特点 | 优势 |
| --- | --- | --- |
| IoU Loss | 1 - IoU | 直接优化IoU |
| GIoU Loss | 1 - (IoU - \|C\(A∪B)\|/\|C\|) | 解决不相交框梯度为0 |
| DIoU Loss | 1 - IoU + ρ²(b,b_gt)/c² | 考虑中心点距离 |
| CIoU Loss | 1 - IoU + ρ²/c² + αv | 同时考虑重叠、距离、宽高比 |

### 示例：手动实现 Focal Loss 和 IoU Loss

```python
import torch
import torch.nn as nn

class FocalLoss(nn.Module):
    def __init__(self, alpha=0.25, gamma=2.0):
        super().__init__()
        self.alpha = alpha
        self.gamma = gamma

    def forward(self, pred, target):
        ce_loss = nn.functional.cross_entropy(pred, target, reduction='none')
        pt = torch.exp(-ce_loss)
        focal_weight = self.alpha * (1 - pt) ** self.gamma
        return (focal_weight * ce_loss).mean()


class CIoULoss(nn.Module):
    """Complete IoU Loss"""
    def forward(self, pred_boxes, target_boxes):
        # pred_boxes, target_boxes: [N, 4] (x1, y1, x2, y2)
        xi1 = torch.max(pred_boxes[:, 0], target_boxes[:, 0])
        yi1 = torch.max(pred_boxes[:, 1], target_boxes[:, 1])
        xi2 = torch.min(pred_boxes[:, 2], target_boxes[:, 2])
        yi2 = torch.min(pred_boxes[:, 3], target_boxes[:, 3])
        inter = (xi2 - xi1).clamp(0) * (yi2 - yi1).clamp(0)

        # Union area
        pred_area = (pred_boxes[:, 2] - pred_boxes[:, 0]) * (pred_boxes[:, 3] - pred_boxes[:, 1])
        target_area = (target_boxes[:, 2] - target_boxes[:, 0]) * (target_boxes[:, 3] - target_boxes[:, 1])
        union = pred_area + target_area - inter

        iou = inter / (union + 1e-6)

        # Enclosing box
        xc1 = torch.min(pred_boxes[:, 0], target_boxes[:, 0])
        yc1 = torch.min(pred_boxes[:, 1], target_boxes[:, 1])
        xc2 = torch.max(pred_boxes[:, 2], target_boxes[:, 2])
        yc2 = torch.max(pred_boxes[:, 3], target_boxes[:, 3])
        c_area = (xc2 - xc1) * (yc2 - yc1)

        # Center distance
        pred_cx = (pred_boxes[:, 0] + pred_boxes[:, 2]) / 2
        pred_cy = (pred_boxes[:, 1] + pred_boxes[:, 3]) / 2
        target_cx = (target_boxes[:, 0] + target_boxes[:, 2]) / 2
        target_cy = (target_boxes[:, 1] + target_boxes[:, 3]) / 2
        center_dist = (pred_cx - target_cx)**2 + (pred_cy - target_cy)**2

        # Diagonal of enclosing box
        diag = (xc2 - xc1)**2 + (yc2 - yc1)**2 + 1e-6

        # Aspect ratio
        v = (4 / (3.14159**2)) * (
            torch.atan((target_boxes[:, 2] - target_boxes[:, 0]) /
                       (target_boxes[:, 3] - target_boxes[:, 1] + 1e-6)) -
            torch.atan((pred_boxes[:, 2] - pred_boxes[:, 0]) /
                       (pred_boxes[:, 3] - pred_boxes[:, 1] + 1e-6))
        )**2

        with torch.no_grad():
            alpha = v / (1 - iou + v + 1e-6)

        ciou = iou - center_dist / diag - alpha * v
        return (1 - ciou).mean()
```

## 总结

- 单阶段检测器直接从特征图预测类别和框，速度快适合实时应用
- YOLO系列从v1到v8不断演进，v8采用Anchor-Free设计
- Focal Loss解决了正负样本不平衡问题，是单阶段检测器精度提升的关键
- Anchor-Free方法（FCOS、CenterNet）简化了检测器设计
- CIoU Loss同时优化重叠度、中心距离和宽高比，定位精度更高
- 实践中推荐使用 Ultralytics YOLOv8 进行快速部署
