# 信息熵与互信息 - 信息论笔记


### 1.1 自信息（Self-Information）


自信息衡量一个事件发生所带来的"惊讶程度"或"信息量"。事件越不可能发生，发生时带来的信息量越大。


$$
I(x) = -log₂ P(x) = log₂ (1/P(x))
$$


- **I(x)**
   ：事件x发生时的自信息量
- **P(x)**
   ：事件x发生的概率
- **单位**
   ：以2为底时单位为比特（bit），以e为底时单位为奈特（nat），以10为底时单位为哈特利（hartley）


> **Example:** **示例：**
>
> - 掷硬币正面朝上：P=0.5，I = -log₂(0.5) = 1 bit
> - 掷骰子出6点：P=1/6，I = -log₂(1/6) = log₂6 ≈ 2.585 bits
> - 太阳从东方升起：P≈1，I ≈ 0 bits（几乎没有信息量）


### 1.2 信息熵（Shannon Entropy）


信息熵是随机变量的**平均自信息量**，描述随机变量的不确定性或信息的平均量。


$$
H(X) = -Σ P(x) log₂ P(x)
$$


- **H(X) ≥ 0**
   ：信息熵非负
- **H(X) = 0**
   ：当且仅当X为确定性事件时（P=1的情况）
- **H(X) ≤ log₂|X|**
   ：当且仅当X均匀分布时取等号（最大熵原理）


> **Example:** **计算示例：掷硬币**
>
> P(正)=0.5, P(反)=0.5
>
>
> H(X) = -(0.5 × log₂0.5 + 0.5 × log₂0.5) = -(0.5×(-1) + 0.5×(-1)) = 1 bit
>
>
> 这表示掷一次公平硬币平均产生1比特的信息。


> **Example:** **计算示例：不公平硬币**
>
> P(正)=0.9, P(反)=0.1
>
>
> H(X) = -(0.9 × log₂0.9 + 0.1 × log₂0.1) = -(0.9×(-0.152) + 0.1×(-3.322)) ≈ 0.469 bits
>
>
> 不公平硬币的信息熵小于公平硬币，因为结果更确定，不确定性更小。


### 1.3 熵的基本性质


| 性质 | 数学表达 | 说明 |
| --- | --- | --- |
| 非负性 | H(X) ≥ 0 | 信息熵不可能为负 |
| 确定性 | H(X)=0 ⟺ P(x)=1 对某个x | 确定事件的熵为0 |
| 最大熵 | H(X) ≤ log₂\|X\| | 均匀分布时熵最大 |
| 对称性 | H(X)只与概率分布有关 | 与事件标签无关 |
| 可加性 | H(X,Y) = H(X) + H(Y\|X) | 链式法则 |


### 2.1 联合熵（Joint Entropy）


联合熵描述两个随机变量联合分布的不确定性。


$$
H(X,Y) = -Σ Σ P(x,y) log₂ P(x,y)
              = H(X) + H(Y|X)
$$


### 2.2 条件熵（Conditional Entropy）


条件熵表示在已知Y的条件下，X仍剩余的不确定性。


$$
H(X|Y) = -Σ P(y) Σ P(x|y) log₂ P(x|y)
             = H(X,Y) - H(Y)
$$


### 2.3 链式法则


$$
H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)
$$


```
┌──────────────────────────────────────┐
  │  H(X,Y)                             │
  │  ┌──────────────┐ ┌───────────────┐ │
  │  │    H(X)      │ │   H(Y|X)     │ │
  │  │              │ └───────────────┘ │
  │  └──────────────┘                    │
  └──────────────────────────────────────┘

  也等于：

  ┌──────────────────────────────────────┐
  │  H(X,Y)                             │
  │  ┌──────────────┐ ┌───────────────┐ │
  │  │    H(Y)      │ │   H(X|Y)     │ │
  │  │              │ └───────────────┘ │
  │  └──────────────┘                    │
  └──────────────────────────────────────┘
```

**重要结论：**
知道额外信息Y不会增加X的不确定性，即 H(X|Y) ≤ H(X)。等号成立当且仅当X与Y相互独立。

### 3.1 定义


互信息衡量两个随机变量之间的相互依赖程度，即知道一个变量后对另一个变量不确定性减少的量。


$$
I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)
             = H(X) + H(Y) - H(X,Y)
             = Σ Σ P(x,y) log₂ [P(x,y) / (P(x)·P(y))]
$$


### 3.2 性质


- **I(X;Y) ≥ 0**
   ：互信息非负
- **I(X;Y) = 0 ⟺ X与Y独立**
- **I(X;Y) = I(Y;X)**
   ：对称性
- **I(X;X) = H(X)**
   ：自信息等于熵
- **I(X;Y) ≤ min(H(X), H(Y))**
   ：互信息不超过任一变量的熵


### 3.3 信息图（Venn Diagram类比）


```
┌─────────────────────────────────────────┐
      │                  H(X,Y)                  │
      │                                           │
      │    ┌──────────┐       ┌──────────┐       │
      │    │          │       │          │       │
      │    │  H(X|Y)  │ I(X;Y)│  H(Y|X)  │       │
      │    │          │       │          │       │
      │    │          │       │          │       │
      │    └──────────┘       └──────────┘       │
      │         H(X)              H(Y)           │
      └─────────────────────────────────────────┘

  H(X) = H(X|Y) + I(X;Y)     （X的不确定 = X独有 + X与Y共享）
  H(Y) = H(Y|X) + I(X;Y)     （Y的不确定 = Y独有 + X与Y共享）
  H(X,Y) = H(X|Y) + I(X;Y) + H(Y|X)  （联合 = 三部分之和）
```


> **Example:** **互信息示例：**
>
> 设X表示天气（晴/雨），Y表示地面状况（干/湿）。
>
>
> - H(X) = 1 bit（等概率天气）
> - H(X|Y) = 0.2 bits（看到地面后天气就很确定了）
> - I(X;Y) = H(X) - H(X|Y) = 0.8 bits（地面状况提供了0.8比特关于天气的信息）


### 4.1 定义


KL散度衡量两个概率分布之间的差异，也称为相对熵。它表示用分布Q来近似分布P时损失的信息量。


$$
D_KL(P || Q) = Σ P(x) log₂ [P(x) / Q(x)]
                    = E_P[log P(X) - log Q(X)]
$$


### 4.2 性质


- **非负性**
   ：D_KL(P || Q) ≥ 0，等号当且仅当P = Q（Gibbs不等式）
- **不对称性**
   ：D_KL(P || Q) ≠ D_KL(Q || P)，因此不是真正的"距离"
- **与互信息的关系**
   ：I(X;Y) = D_KL(P(X,Y) || P(X)P(Y))


> **Example:** **KL散度的直观理解：**
>
> 假设真实分布P表示一封信中各字母的出现频率，Q是我们猜测的频率分布。
>
>
> D_KL(P || Q)表示如果我们使用Q来编码这封信，相比使用最优编码P多花费的平均比特数。


### 4.3 交叉熵与KL散度


$$
H(P,Q) = -Σ P(x) log₂ Q(x) = H(P) + D_KL(P || Q)
$$


交叉熵 = 真实分布的熵 + KL散度。在机器学习中，最小化交叉熵等价于最小化KL散度（因为H(P)是常数）。

**机器学习中的应用：**

- 分类任务的损失函数通常使用交叉熵损失
- 知识蒸馏中通过最小化KL散度对齐教师和学生模型的输出分布
- 变分自编码器（VAE）中用KL散度约束潜在空间分布
- GAN训练中多种损失函数涉及KL/JS散度


### 5.1 马尔可夫链


若随机变量X、Y、Z构成马尔可夫链 X → Y → Z，即Z仅通过Y依赖X（P(z|x,y) = P(z|y)）。


### 5.2 数据处理不等式


$$
若 X → Y → Z 构成马尔可夫链，则：
        I(X;Z) ≤ I(X;Y)  且  I(X;Z) ≤ I(Y;Z)
$$


含义：**对数据进行处理（无论多么复杂）不会增加信息量**。数据处理只会丢失信息，不会创造信息。


```
数据处理不等式的直观理解：

  原始数据X → 处理Y → 再处理Z
  I(X;Z) ≤ I(X;Y) ≤ I(X;X) = H(X)

  信息只会减少或保持不变，不会增加

  实例：
  原始信号X → 传感器测量Y → 软件处理Z
  知道Z获取的关于X的信息 ≤ 知道Y获取的关于X的信息
```


### 5.3 应用意义


- **信息不增原理**
   ：任何后续处理无法恢复已经被丢失的信息
- **通信系统设计**
   ：编码和解码的极限由信道决定，无法通过后处理突破
- **机器学习**
   ：特征提取和模型推理本质上是数据处理，不可能从数据中提取出数据本身不包含的信息
- **因果推断**
   ：处理观测数据不能推断出超过数据本身蕴含的因果关系

**重要推论：**
充分统计量是数据处理不等式的特殊情况。T是参数θ关于数据X的充分统计量 ⟺ I(θ;T) = I(θ;X)，即充分统计量保留了原始数据关于参数的全部信息。

### 6.1 Renyi熵


Renyi熵是Shannon熵的参数化推广，引入参数α控制对概率分布不同部分的敏感度。


$$
H_α(X) = (1 / (1-α)) log₂ Σ P(x)^α
$$


- α → 1 时：Renyi熵收敛为Shannon熵
- α = 0 时：Hartley熵 = log|X|（只关心支持集大小）
- α = 2 时：碰撞熵（Collision Entropy），常用于密码学
- α → ∞ 时：最小熵（Min-Entropy），最保守的信息度量


### 6.2 微分熵（Differential Entropy）


将信息熵从离散随机变量推广到连续随机变量。


$$
h(X) = -∫ f(x) log₂ f(x) dx
$$


注意：微分熵可以为负值（如均匀分布），且值依赖于测量单位，因此不能直接等同于信息量。


### 6.3 核心概念关系总结


| 概念 | 符号 | 含义 |
| --- | --- | --- |
| 信息熵 | H(X) | 随机变量的平均不确定性 |
| 联合熵 | H(X,Y) | 联合分布的不确定性 |
| 条件熵 | H(X\|Y) | 已知Y后X的剩余不确定性 |
| 互信息 | I(X;Y) | X和Y共享的信息量 |
| KL散度 | D_KL(P\|\|Q) | 用Q近似P的信息损失 |
| 交叉熵 | H(P,Q) | 用Q编码P的平均比特数 |
