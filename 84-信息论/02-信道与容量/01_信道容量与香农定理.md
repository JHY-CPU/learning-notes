# 信道容量与香农定理 - 信息论笔记


### 1.1 通信系统基本模型


```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
  │  信源    │───→│  编码器   │───→│  信道    │───→│  解码器   │───→│  信宿    │
  │ (X)      │    │          │    │ (噪声)   │    │          │    │ (Y)      │
  └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
       信息         信源编码        传输介质         信道解码        恢复信息
                   信道编码                        信源解码

  香农分离定理：信源编码和信道编码可以分开设计，且不会损失最优性
```


### 1.2 离散无记忆信道（DMC）


离散无记忆信道由输入字母表、输出字母表和转移概率矩阵定义。


$$
信道由转移概率矩阵 P(Y|X) = {p(yⱼ|xᵢ)} 完全描述
        其中 p(yⱼ|xᵢ) = P(Y=yⱼ | X=xᵢ)
$$


### 1.3 转移概率矩阵


> **Example:** **二进制对称信道（BSC）的转移矩阵：**
>
> | P(Y\|X) | Y=0 | Y=1 |
> | --- | --- | --- |
> | **X=0** | 1-p | p |
> | **X=1** | p | 1-p |
>
>
> 其中p为错误概率（交叉概率），1-p为正确传输概率。


### 2.1 定义


信道容量C是信道在单位时间内能够可靠传输的最大信息量，即信道的最大互信息。


$$
C = max_{P(X)} I(X;Y)    （比特/信道使用）
$$


信道容量取决于信道本身的特性（转移概率），与输入分布无关，但通过优化输入分布可以达到容量。


### 2.2 计算信道容量的方法


对于对称信道，当输入均匀分布时互信息达到最大值。

**对称信道的特殊性质：**
如果信道转移矩阵的每一行都是其他行的排列，每一列也是其他列的排列，则称为对称信道。对于对称信道，C = log₂|Y| - H(转移矩阵的一行)。

### 3.1 二进制对称信道（BSC）


```
BSC (Binary Symmetric Channel):

        X ─────→ [ BSC(p) ] ─────→ Y

        0 ──── 1-p ────→ 0
          ──── p ────→ 1

        1 ──── p ────→ 0
          ──── 1-p ────→ 1
```


$$
C_BSC = 1 - H(p) = 1 + p·log₂(p) + (1-p)·log₂(1-p)    （比特/信道使用）
$$


- p = 0 时：C = 1 bit（完美信道）
- p = 0.5 时：C = 0 bit（完全随机，无法传输任何信息）
- p = 1 时：C = 1 bit（虽然每次翻转，但是确定性的，可以纠正）


> **Example:** **BSC容量计算示例：**
>
> 设p = 0.1，则 H(0.1) = -(0.1×log₂0.1 + 0.9×log₂0.9) = 0.469 bits
>
>
> C = 1 - 0.469 = 0.531 bits/信道使用
>
>
> 这意味着即使有10%的错误率，通过适当的编码仍能以0.531比特/次的速率可靠传输。


### 3.2 二进制删除信道（BEC）


```
BEC (Binary Erasure Channel):

        0 ──── 1-α ────→ 0
          ──── α ────→ ? (删除)

        1 ──── 1-α ────→ 1
          ──── α ────→ ? (删除)
```


$$
C_BEC = 1 - α    （比特/信道使用）
$$


BEC比BSC更容易处理，因为接收端知道哪些符号被删除了，可以直接请求重传或用纠错码恢复。


### 3.3 AWGN信道（加性高斯白噪声信道）


AWGN信道是连续信道中最重要的模型，广泛用于无线通信和数字通信系统。


$$
Y = X + N

        其中 X 为输入信号，N ~ N(0, σ²) 为高斯白噪声，Y 为输出信号

        功率约束：E[X²] ≤ P

        C_AWGN = (1/2) log₂(1 + P/σ²) = (1/2) log₂(1 + SNR)    （比特/信道使用）
$$


- **SNR = P/σ²**
   ：信噪比（Signal-to-Noise Ratio）
- SNR通常以dB表示：SNR_dB = 10 × log₁₀(SNR)
- 容量与带宽成正比（乘以带宽B得到比特/秒）


### 3.4 离散时间AWGN信道的香农公式


$$
C = B × log₂(1 + S/N)    （比特/秒）

        其中：
        C = 信道容量（bps）
        B = 信道带宽（Hz）
        S = 信号功率（W）
        N = 噪声功率（W），N = N₀ × B
        N₀ = 噪声功率谱密度（W/Hz）
$$


> **Example:** **AWGN容量计算示例：**
>
> 设 B = 1 MHz，SNR = 20 dB（即 SNR = 100）
>
>
> C = 10⁶ × log₂(1 + 100) = 10⁶ × 6.658 = 6.658 Mbps
>
>
> 这意味着在这个信道条件下，理论最大传输速率为6.658 Mbps。


### 4.1 定理表述（1948年）


$$
香农第二定理（有噪信道编码定理）：

        对于任意离散无记忆信道，其信道容量为C。

        1. 若信息传输率 R < C，则存在编码方案使得错误概率任意小（趋于0）；
        2. 若信息传输率 R > C，则不存在任何编码方案能使错误概率趋于0。

        即：在R < C时，可以实现"可靠"通信；
            在R > C时，不可能实现可靠通信。
$$


### 4.2 定理的深层含义


- **存在性定理**
   ：证明了好的编码方案的存在，但没有给出具体构造方法
- **渐进性**
   ：编码长度趋于无穷时才能达到容量，实际系统中码长有限会有性能损失
- **分离原理**
   ：信源编码和信道编码可以独立设计
- **噪声不是障碍**
   ：只要速率不超过容量，噪声可以通过编码克服


### 4.3 编码增益与信道编码


```
┌─────────────────────────────────────────────────────────┐
  │  无编码系统：BER随SNR缓慢下降                             │
  │  有编码系统：在相同BER下，所需SNR更低（编码增益）          │
  │                                                          │
  │     BER                                                  │
  │      │╲                                                  │
  │      │ ╲  无编码                                         │
  │      │  ╲                                                │
  │      │   ╲_____                                          │
  │      │    ╲     ╲___                                     │
  │      │     ╲  有编码 ╲___                                │
  │      │      ╲          ╲___                              │
  │      │───────┼──────────────→ SNR                       │
  │             编码增益                                      │
  └─────────────────────────────────────────────────────────┘
```


### 4.4 经典信道编码方案


| 编码类型 | 代表 | 特点 | 与香农限距离 |
| --- | --- | --- | --- |
| 分组码 | 汉明码、BCH码、RS码 | 结构简单、代数译码 | 较远（2-3 dB） |
| 卷积码 | 卷积码 + Viterbi译码 | 连续编码、性能较好 | 中等（1-2 dB） |
| Turbo码 | 并行级联卷积码 | 迭代译码、接近香农限 | 0.5-1 dB |
| LDPC码 | 低密度奇偶校验码 | 稀疏图、并行译码 | 0.1-0.5 dB |
| Polar码 | 极化码 | 理论上可达容量、5G控制信道 | 接近香农限 |

**编码里程碑：**

- 1948年：Shannon证明容量极限存在
- 1950年：Hamming发明汉明码
- 1955年：Elias发明卷积码
- 1960年：Reed-Solomon码
- 1966年：Viterbi算法（卷积码最优译码）
- 1962年：Gallager发明LDPC码（被遗忘30多年）
- 1993年：Berrou发明Turbo码（首次接近香农限）
- 1996年：MacKay重新发现LDPC码
- 2008年：Arikan发明Polar码（理论上可达信道容量）
- 2016年：Polar码被选为5G NR控制信道编码方案


### 5.1 极限推导


当带宽B趋于无穷大时，AWGN信道的容量有极限：


$$
lim_{B→∞} C = lim_{B→∞} B × log₂(1 + S/(N₀B)) = S / (N₀ × ln2) ≈ 1.44 × S/N₀    （比特/秒）
$$


这给出了在任意带宽下，要达到速率R所需的最小比特信噪比：


$$
E_b/N₀ ≥ ln(2) ≈ 0.693    （即 -1.59 dB）

        这 -1.59 dB 称为香农极限
$$


### 5.2 香农极限的物理意义


- 在-1.59 dB以下，无论编码多么复杂、带宽多么大，都不可能实现可靠通信
- 实际系统的Eb/N0需要大于这个极限，差值就是编码增益的损耗
- 现代编码方案（Turbo码、LDPC码）可以在距香农限0.5-1 dB内工作

### 6.1 香农三大定理总结


| 定理 | 内容 | 意义 |
| --- | --- | --- |
| **第一定理**
信源编码定理 | 无损压缩的下界是信源熵H(X) | 数据压缩的理论基础 |
| **第二定理**
有噪信道编码定理 | 速率R < C时可实现可靠通信 | 信道编码的理论基础 |
| **第三定理**
率失真定理 | 有损压缩在给定失真下的最小速率 | 有损压缩（音视频编码）的理论基础 |


### 6.2 实际应用领域


- **5G通信**
   ：LDPC码（数据信道）+ Polar码（控制信道），逼近香农限
- **WiFi 6/7**
   ：LDPC码，高阶调制（1024-QAM）
- **卫星通信**
   ：Turbo码/RS码，在低信噪比下实现可靠通信
- **深空通信**
   ：极低Eb/N0下的编码方案（旅行者号使用(7,1,6)卷积码+RS码）
- **存储系统**
   ：SSD中的LDPC码、硬盘中的RS码
- **MIMO系统**
   ：多天线信道容量 C = log₂ det(I + HH^H·SNR/n)


### 6.3 信道容量的扩展


| 信道类型 | 容量公式 | 应用场景 |
| --- | --- | --- |
| 并行信道 | C = Σ Cᵢ | OFDM子载波 |
| MIMO信道 | C = log₂ det(I + HQH^H) | 4G/5MIMO |
| 衰落信道 | C = E[log₂(1 + \|h\|²·SNR)] | 无线通信 |
| 多址接入信道 | C = log₂(1 + ΣSNRᵢ) | 上行链路 |
| 广播信道 | 容量区域（非单值） | 下行链路 |

**信息论的核心启示：**

1. 存在理论极限（香农限），这是物理规律决定的，不可突破
2. 编码理论的任务就是无限逼近这些极限
3. 从1948年到2016年，人类用了近70年终于找到了接近容量极限的实用编码方案（Polar码和LDPC码）
4. 信息论的思维已经超越通信领域，广泛应用于机器学习、密码学、生物信息学等
