发布订阅模式 Pub-Sub

通过事件中心（Event Bus）解耦发布者和订阅者。发布者触发事件，订阅者接收通知，双方无需直接引用。

完整 Event Bus 实现

支持 on/emit/off/once，命名空间事件，通配符事件，以及内存泄漏警告。

class EventBus { on(event, handler) { /* ... */ } emit(event, data) { /* ... */ } off(event, handler) { /* ... */ } once(event, handler) { /* ... */ } }
