拖放 Drag & Drop

拖放 API：元素拖拽排序、文件拖放上传。

基本用法

// 可拖拽元素（设置 draggable） element.setAttribute('draggable', 'true'); // 拖拽元素的事件 element.addEventListener('dragstart', (e) => { e.dataTransfer.setData('text/plain', '数据'); e.dataTransfer.effectAllowed = 'move'; // 'move' | 'copy' | 'link' | 'none' e.dataTransfer.setDragImage(img, x, y); // 自定义拖拽图标 }); element.addEventListener('drag', (e) => {}); // 拖拽中 element.addEventListener('dragend', (e) => {}); // 拖拽结束 // 放置目标的事件 target.addEventListener('dragenter', (e) => {}); // 进入目标 target.addEventListener('dragover', (e) => { e.preventDefault(); // 必须阻止默认行为才能 drop e.dataTransfer.dropEffect = 'move'; }); target.addEventListener('dragleave', (e) => {}); // 离开目标 target.addEventListener('drop', (e) => { e.preventDefault(); const data = e.dataTransfer.getData('text/plain'); const files = e.dataTransfer.files; // 拖拽的文件 // 处理放置 }); // DataTransfer 属性和方法 e.dataTransfer.files; // 拖拽的文件列表（File API） e.dataTransfer.items; // 拖拽的数据项列表（DataTransferItem） e.dataTransfer.types; // 数据类型列表（['text/plain', 'Files']） e.dataTransfer.setData(type, data); e.dataTransfer.getData(type); e.dataTransfer.clearData();
⚠️ dragover 必须 e.preventDefault() 才能触发 drop 事件。
⚠️ dragenter/dragover 在子元素上也会频繁触发，需注意判断。
💡 文件拖放不需要 draggable 属性，从系统文件管理器拖入即可。
