Web Animations API

Element.animate() — 用 JavaScript 控制 CSS 动画。

基本用法

// 1. 创建动画 const animation = element.animate(keyframes, options); // keyframes — 关键帧数组 const keyframes = [ { transform: 'translateX(0)', opacity: 1 }, { transform: 'translateX(200px)', opacity: 0.5 }, { offset: 0.3 }, // 控制在 30% 处 { transform: 'translateX(0)', opacity: 1 }, ]; // 或使用对象格式 const keyframes = { transform: ['translateX(0)', 'translateX(200px)', 'translateX(0)'], opacity: [1, 0.5, 1], }; // options — 选项 const options = { duration: 1000, // 持续时间 (ms) delay: 200, // 延迟 (ms) endDelay: 0, // 结束后延迟 iterations: Infinity, // 循环次数 (Infinity = 无限) direction: 'normal', // 'normal' | 'reverse' | 'alternate' | 'alternate-reverse' easing: 'ease-in-out', // 缓动函数 fill: 'forwards', // 'none' | 'forwards' | 'backwards' | 'both' iterationStart: 0, }; // 2. 控制动画 animation.play(); // 播放 animation.pause(); // 暂停 animation.reverse(); // 反向播放 animation.finish(); // 跳到结束 animation.cancel(); // 取消 // 3. 读取状态 animation.playState; // 'idle' | 'running' | 'paused' | 'finished' animation.currentTime; // 当前进度 (ms) animation.timeline; // 时间线（默认 document.timeline） // 4. 事件 animation.onfinish = () => {}; animation.oncancel = () => {}; animation.onremove = () => {}; // 5. 播放速率 (2 = 双倍速, 0.5 = 慢放) animation.playbackRate = 1; // 6. 获取元素所有动画 element.getAnimations(); // 返回 Animation[]
