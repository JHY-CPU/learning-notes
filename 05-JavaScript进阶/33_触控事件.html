触控事件 Touch Events

触摸屏设备的触摸事件：touchstart/touchmove/touchend/touchcancel。

基本用法

// 触摸事件类型 element.addEventListener('touchstart', handler); // 手指触摸屏幕 element.addEventListener('touchmove', handler); // 手指在屏幕上滑动 element.addEventListener('touchend', handler); // 手指离开屏幕 element.addEventListener('touchcancel', handler); // 触摸被中断（来电/弹窗） // TouchEvent 对象 e.touches; // 所有触摸点（手指）的列表 e.targetTouches; // 在同一目标元素上的触摸点 e.changedTouches; // 发生变化的触摸点 // Touch 对象 touch.identifier; // 唯一标识符（用于多点触控追踪） touch.clientX; // 触摸点在视口的 X 坐标 touch.clientY; // 触摸点在视口的 Y 坐标 touch.pageX; // 触摸点在页面的 X 坐标 touch.pageY; // 触摸点在页面的 Y 坐标 touch.screenX; // 触摸点在屏幕的 X 坐标 touch.screenY; // 触摸点在屏幕的 Y 坐标 touch.force; // 按压力度（iOS 支持） touch.radiusX; // 触摸区域半径 X touch.radiusY; // 触摸区域半径 Y touch.rotationAngle; // 旋转角度 // 处理多点触控（缩放手势） element.addEventListener('touchstart', (e) => { if (e.touches.length === 2) { const dist = Math.hypot( e.touches[0].pageX - e.touches[1].pageX, e.touches[0].pageY - e.touches[1].pageY, ); // 记录初始距离 } }); // CSS touch-action // touch-action: none; // 禁用浏览器默认触摸行为 // touch-action: pan-x; // 允许水平滚动 // touch-action: pan-y; // 允许垂直滚动 // touch-action: manipulation; // 允许滚动 + 缩放（禁止 dblclick zoom）
💡 桌面端可以用 DevTools 的"切换设备模拟"来测试触控事件。
⚠️ 记得设置 touch-action: none 防止浏览器干扰自定义手势。
💡 用 identifier 追踪每个触摸点（在 touchend 时 changedTouches 的 identifier 与 touchstart 一致）。
