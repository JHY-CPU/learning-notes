网络状态检测

检测浏览器在线/离线状态及网络类型（3G/4G/WiFi 等）。

基本用法

// 1. 在线/离线状态 navigator.onLine; // true = 在线, false = 离线 window.addEventListener('online', () => { console.log('网络已连接'); }); window.addEventListener('offline', () => { console.log('网络已断开'); }); // 2. Network Information API（非标准，但有广泛支持） const connection = navigator.connection || navigator.mozConnection || navigator.webkitConnection; connection.effectiveType; // 'slow-2g' | '2g' | '3g' | '4g' (5g 也可能) connection.downlink; // 下行速度（Mbps） connection.rtt; // 往返时延（ms） connection.saveData; // 是否开启省流量模式 // 监听网络类型变化 connection.addEventListener('change', () => { console.log('网络变化:', connection.effectiveType); }); // 用途：根据网络质量调整资源加载 if (connection?.effectiveType === '4g') { loadHDImages(); } else { loadLowResImages(); }
⚠️ navigator.onLine 并不完全可靠（可能返回 false 但实际能联网）。
⚠️ NetworkInformation API 是实验性功能，部分浏览器不支持。
💡 最佳实践：online/offline + 实际请求（fetch）双重检测。
