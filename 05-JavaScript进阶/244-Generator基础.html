Generator 基础

function*、yield、next()、done/value、手动控制迭代流程。

Generator 语法

// ========== Generator 函数定义 ========== function* genFunc() { yield 1; yield 2; yield 3; } // Generator 表达式 const genFunc2 = function*() { yield 'a'; yield 'b'; }; // 对象方法简写 const obj = { *gen() { yield 'x'; } }; // 类方法 class MyClass { *gen() { yield 'y'; } } // ========== 调用方式 ========== const gen = genFunc(); // 返回 Generator 对象（不执行函数体） console.log(gen.next()); // { value: 1, done: false } console.log(gen.next()); // { value: 2, done: false } console.log(gen.next()); // { value: 3, done: false } console.log(gen.next()); // { value: undefined, done: true } // ========== Generator 是可迭代对象 ========== for (const val of genFunc()) { console.log(val); // 1, 2, 3 } // 展开运算符 const arr = [...genFunc()]; // [1, 2, 3] // 解构 const [a, b, c] = genFunc(); // ========== Generator 特点 ========== // ✅ 暂停执行 — 遇到 yield 暂停 // ✅ 惰性求值 — 按需产生值 // ✅ 双向通信 — next() 可传值 // ✅ 可无限序列 — 不一次性计算所有值
