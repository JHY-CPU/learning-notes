ResizeObserver

监听元素的尺寸变化（比 window.resize 更精准，可观察任意元素）。

基本用法

// 创建观察器 const observer = new ResizeObserver((entries) => { entries.forEach(entry => { entry.contentRect; // 内容区域尺寸 {width/height/top/left} entry.target; // 发生变化的元素 entry.borderBoxSize; // border-box 尺寸（最新 API） entry.contentBoxSize; // content-box 尺寸 entry.devicePixelContentBoxSize; // 物理像素尺寸 }); }); // 开始/停止观察 observer.observe(element); observer.observe(element, { box: 'border-box' }); // 指定观察的盒模型 observer.unobserve(element); observer.disconnect(); // 对比 window.resize: // ✅ 观察任意元素，不是只能观察 window // ✅ 不频繁触发，只在实际尺寸变化时 // ✅ 回调自带尺寸信息，无需强行读取 layout
💡 contentRect 是标准属性，但 borderBoxSize 需要使用 entry.borderBoxSize[0].blockSize/inlineSize。
⚠️ 旧版本浏览器可能只支持 contentRect，注意兼容性。
