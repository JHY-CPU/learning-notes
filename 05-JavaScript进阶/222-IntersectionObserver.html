IntersectionObserver

异步观察元素与视口（或父元素）的交叉状态，用于懒加载、无限滚动、曝光统计等。

基本用法

// 创建观察器 const observer = new IntersectionObserver((entries) => { entries.forEach(entry => { entry.isIntersecting; // 是否交叉（进入视口） entry.intersectionRatio; // 可见比例 (0~1) entry.boundingClientRect; // 元素的视口坐标 entry.intersectionRect; // 交叉区域坐标 entry.rootBounds; // 根元素的视口坐标 entry.target; // 被观察的元素 }); }, { root: null, // 父容器（null = 视口） rootMargin: '0px', // 扩大的边界（如 '-50px' 延迟触发） threshold: [0, 0.5, 1], // 触发阈值 (0~1 数组) }); // 开始/停止观察 observer.observe(element); observer.unobserve(element); observer.disconnect(); // 获取所有被观察的元素 const observed = observer.takeRecords();
💡 rootMargin 可以提前触发："100px" = 元素距离视口 100px 时触发，适合预加载。
💡 threshold: [0, 0.5, 1] = 进入/一半/完全可见时各触发一次。
