迭代器协议

Symbol.iterator、自定义可迭代对象、可迭代与类数组。

迭代器协议语法

// ========== 迭代器协议 (Iterator Protocol) ========== // 实现 next() 方法，返回 { value, done } const iterator = { next() { if (/* 还有值 */) return { value: 'x', done: false }; return { value: undefined, done: true }; } }; // ========== 可迭代协议 (Iterable Protocol) ========== // 实现 [Symbol.iterator]() 方法，返回 iterator const iterable = { [Symbol.iterator]() { let i = 0; return { next() { if (i < 3) return { value: i++, done: false }; return { value: undefined, done: true }; } }; } }; // 使用 for (const val of iterable) console.log(val); // 0, 1, 2 const arr = [...iterable]; // [0, 1, 2] // ========== 内置可迭代对象 ========== // Array, String, Map, Set, TypedArray, arguments, NodeList // ❌ Object 默认不可迭代 // ========== 可迭代对象 vs 类数组 ========== // 可迭代: 有 [Symbol.iterator] // 类数组: 有 length 和数字索引 // Array.from() 两者都支持 const arrayLike = { 0: 'a', 1: 'b', length: 2 }; Array.from(arrayLike); // ['a', 'b']
