Notification API

浏览器桌面通知推送。

基本用法

// 检查支持 if ('Notification' in window) { // 支持桌面通知 } // 请求权限（必须用户手势触发） Notification.requestPermission().then(permission => { // 'granted' | 'denied' | 'default' }); // 发送通知 const notif = new Notification('标题', { body: '通知正文内容', // 正文 icon: '/icon.png', // 图标 URL image: '/image.png', // 大图 URL badge: '/badge.png', // 标记图标（移动端） tag: 'unique-tag', // 标签（相同 tag 会替换前一个） data: { key: 'value' }, // 自定义数据 vibrate: [200, 100, 200], // 震动模式 [振, 停, 振] silent: false, // 静音 requireInteraction: true, // 不自动关闭 actions: [ // 按钮（移动端） { action: 'open', title: '打开' }, { action: 'close', title: '关闭' }, ], }); // 事件 notif.onshow = () => {}; // 已显示（已弃用） notif.onclick = () => {}; // 点击通知 notif.onclose = () => {}; // 关闭通知 notif.onerror = () => {}; // 出错 // 关闭通知 notif.close(); // 检查当前权限 Notification.permission; // 'granted' | 'denied' | 'default'
⚠️ 必须通过用户手势（click/touch）请求权限。
⚠️ HTTPS 下才能使用（localhost 开发环境可以）。
⚠️ 移动端支持有限，iOS Safari 不支持。
💡 tag 属性可用于去重：多条通知用相同 tag 只会显示最新一条。
