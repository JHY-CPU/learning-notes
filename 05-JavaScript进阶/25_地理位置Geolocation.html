Geolocation API

获取用户的地理位置（经度/纬度/海拔/速度等）。

基本用法

// 检查可用性 if ('geolocation' in navigator) { // 支持 } else { // 不支持 } // 获取当前位置 navigator.geolocation.getCurrentPosition(success, error, options); // 成功回调 function success(pos) { pos.coords.latitude; // 纬度 pos.coords.longitude; // 经度 pos.coords.accuracy; // 精度（米） pos.coords.altitude; // 海拔（米） pos.coords.altitudeAccuracy; // 海拔精度 pos.coords.heading; // 朝向（度，0=北） pos.coords.speed; // 速度（m/s） pos.timestamp; // 时间戳 } // 错误回调 function error(err) { err.code; // 1=PERMISSION_DENIED, 2=POSITION_UNAVAILABLE, 3=TIMEOUT err.message; // 错误描述 } // 选项 const options = { enableHighAccuracy: true, // 高精度（GPS，耗电） timeout: 10000, // 超时（ms） maximumAge: 60000, // 缓存时间（ms） }; // 持续监听位置变化 const watchId = navigator.geolocation.watchPosition(success, error, options); navigator.geolocation.clearWatch(watchId);
⚠️ 需要用户授权才能使用（浏览器会弹出权限请求）。
⚠️ 仅在 HTTPS 下可用（本地 localhost 也可以）。
⚠️ enableHighAccuracy:true 会增加耗电，请按需使用。
