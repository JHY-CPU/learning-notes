函数组合 compose / pipe

将多个函数组合成一个新函数。compose 从右向左执行（数学复合 f∘g），pipe 从左向右执行（管道/流水线）。

compose vs pipe

compose(f, g)(x)=f(g(x))
pipe(f, g)(x)=g(f(x))

Point-free 风格（无参风格）

通过组合子函数避免显式声明参数。使代码更声明式、更易于阅读和测试。

// 有参: const process = arr => sort(unique(filter(arr))); // Point-free: const process = pipe(filter, unique, sort);

异步中间件管道

Koa 风格的 async (ctx, next) => ... 中间件链，通过 compose 实现洋葱模型。
