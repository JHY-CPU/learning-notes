指针事件 Pointer Events

统一处理鼠标/触控/触控笔事件的 API。

基本用法

// 指针事件（统一鼠标/触摸/笔） element.addEventListener('pointerdown', handler); // 按下 element.addEventListener('pointermove', handler); // 移动 element.addEventListener('pointerup', handler); // 抬起 element.addEventListener('pointercancel', handler); // 取消 element.addEventListener('pointerover', handler); // 进入 element.addEventListener('pointerout', handler); // 离开 element.addEventListener('pointerenter', handler); // 进入（不冒泡） element.addEventListener('pointerleave', handler); // 离开（不冒泡） // PointerEvent 属性 e.pointerId; // 唯一标识符 e.pointerType; // 'mouse' | 'touch' | 'pen' e.clientX/Y; // 视口坐标 e.pageX/Y; // 页面坐标 e.screenX/Y; // 屏幕坐标 e.button; // 0=左键, 1=中键, 2=右键 e.buttons; // 位掩码（1=左,2=右,4=中,8=后退,16=前进） e.pressure; // 压力 (0~1) e.tiltX/Y; // 笔倾斜角度 e.twist; // 笔旋转角度 e.width/height; // 触摸区域大小 e.isPrimary; // 是否为主指针（多点触控时区分） // 指针捕获（将指针事件锁定到指定元素） element.setPointerCapture(e.pointerId); element.releasePointerCapture(e.pointerId); element.hasPointerCapture(e.pointerId); // CSS touch-action // touch-action: none; // 禁用所有默认手势 // touch-action: pinch-zoom; // 仅允许缩放 // touch-action: pan-x pan-y; // 允许滚动
💡 Pointer Events 是推荐的事件 API，比分别处理 mouse/touch 更方便。
⚠️ 使用 setPointerCapture 后，即使指针移出元素也能收到事件。
💡 pointerType 可用来区分鼠标和触摸，提供不同交互体验。
