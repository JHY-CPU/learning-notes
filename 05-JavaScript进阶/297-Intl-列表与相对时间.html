Intl 列表与相对时间

ListFormat 列表连接、RelativeTimeFormat 相对时间。

ListFormat 和 RelativeTimeFormat

// ========== Intl.ListFormat (ES2021) ========== // 格式化列表 (如 "A, B 和 C") const lf = new Intl.ListFormat(locale, options); // 选项 { type: 'conjunction', // 'conjunction' (A, B 和 C) // 'disjunction' (A, B 或 C) // 'unit' (A, B, C) style: 'long', // 'long' / 'short' / 'narrow' } // 用法 lf.format(['苹果', '香蕉', '橙子']); // '苹果、香蕉和橙子' (zh-CN) // '苹果, 香蕉, 橙子' (短格式) // ========== Intl.RelativeTimeFormat (ES2020) ========== // 格式化相对时间 (如 "3天前"、"5分钟后") const rtf = new Intl.RelativeTimeFormat(locale, options); // 选项 { numeric: 'always', // 'always' / 'auto' (auto 会使用 "昨天" 而不是 "1天前") style: 'long', // 'long' / 'short' / 'narrow' } // 用法 rtf.format(-1, 'day'); // '1天前' rtf.format(3, 'month'); // '3个月后' rtf.format(-5, 'minute'); // '5分钟前' // 可用单位 // 'year' / 'month' / 'week' / 'day' / 'hour' / 'minute' / 'second' / 'quarter'
