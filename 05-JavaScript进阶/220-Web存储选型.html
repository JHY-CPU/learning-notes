选型决策

// 选型决策树 // 1. 需要 HTTP 自动发送？ → Cookie // ✅ 适合: 会话标识、认证 Token // ❌ 不适合: 大量数据、API 响应缓存 // 2. 简单键值对，容量 < 5MB？ // a) 仅当前标签页 → sessionStorage // ✅ 适合: 表单草稿、临时状态 // b) 需要持久化 → localStorage // ✅ 适合: 用户偏好、主题、语言设置 // 3. 需要存储结构化/大量数据？ // a) 离线数据 > 5MB → IndexedDB // ✅ 适合: 离线数据、文件、日志、复杂查询 // b) HTTP 响应缓存 → Cache-API (SW) // ✅ 适合: 离线页面、静态资源缓存 // 4. 需要跨标签通信？ // → BroadcastChannel + localStorage/storage 事件 // 5. 安全性需求 // 敏感信息 → HttpOnly Cookie (服务器设置) // 非敏感 → localStorage 即可
