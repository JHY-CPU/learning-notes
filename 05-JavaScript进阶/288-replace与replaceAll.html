replace 与 replaceAll

String.replace / replaceAll、$ 引用、函数作为 replacer。

replace 语法

// ========== String.replace() ========== // 替换第一个匹配 (字符串或正则) 'Hello World'.replace('World', 'JS'); // 'Hello JS' 'Hello World'.replace(/World/, 'JS'); // 'Hello JS' 'Hello World'.replace(/l/g, 'L'); // 'HeLLo WorLd' (全局) // ========== String.replaceAll() (ES2021) ========== // 替换所有匹配 (必须全局或字符串) 'Hello World'.replaceAll('o', 'O'); // 'HellO WOrld' 'Hello World'.replaceAll(/o/g, 'O'); // 'HellO WOrld' // ========== $ 引用模式 ========== // $& — 匹配的子串 // $` — 匹配左侧文本 // $' — 匹配右侧文本 // $n — 第 n 个捕获组 // $ — 命名捕获组 // $$ — $ 字面量 // ========== 函数 replacer ========== '2024-01-15'.replace(/(\d{4})-(\d{2})-(\d{2})/g, (match, year, month, day, offset, string) => { return `${month}/${day}/${year}`; } ); // '01/15/2024' // 命名组 replacer str.replace(/(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/g, (...args) => { const groups = args.at(-1); // 最后一个参数是 groups return `${groups.month}/${groups.day}/${groups.year}`; } );
