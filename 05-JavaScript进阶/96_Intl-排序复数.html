Intl 排序与复数

Collator 排序（按拼音/笔画）、PluralRules 复数规则。

Collator 和 PluralRules

// ========== Intl.Collator (排序) ========== const collator = new Intl.Collator(locale, options); // 选项 { sensitivity: 'base', // 'base' / 'accent' / 'case' / 'variant' ignorePunctuation: true, // 忽略标点 numeric: true, // 数字排序 (1,2,10 而不是 1,10,2) caseFirst: 'upper', // 'upper' / 'lower' / 'false' } // 用法 const sorted = arr.sort(collator.compare); // 中文排序 const zhCollator = new Intl.Collator('zh-CN'); // 默认按拼音排序 // zh-CN-u-co-pinyin — 拼音 (默认) // zh-CN-u-co-stroke — 按笔画 // ========== Intl.PluralRules (复数) ========== const pr = new Intl.PluralRules(locale, options); // 方法 pr.select(1); // 'one' pr.select(2); // 'other' // 复数类别: // 'zero', 'one', 'two', 'few', 'many', 'other' // 中文: 'other' (没有复数变化) // 英语: 'one' / 'other' // 俄语: 'one' / 'few' / 'many' / 'other' // 阿拉伯语: 'zero' / 'one' / 'two' / 'few' / 'many' / 'other'
