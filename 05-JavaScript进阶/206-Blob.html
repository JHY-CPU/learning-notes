Blob

Binary Large Object — 二进制数据对象。

创建与属性

// 创建 Blob const blob = new Blob(array, options); // array — 数据源（ArrayBuffer / TypedArray / Blob / DOMString 的数组） const blob1 = new Blob(['Hello World']); // 文本 const blob2 = new Blob(['...'], { type: 'text/html' }); // HTML const blob3 = new Blob([JSON.stringify({a:1})], { type: 'application/json' }); const blob4 = new Blob([uint8Array], { type: 'image/png' }); // 二进制 // options // type: MIME 类型（'text/plain' | 'text/html' | 'application/json' | 'image/png'...） // endings: 'transparent'（默认）| 'native'（换行符处理） // 属性 blob.size; // 字节大小 blob.type; // MIME 类型 // 方法 blob.slice(start, end, type); // 截取部分（创建新的 Blob，用于分片上传） blob.arrayBuffer(); // 转为 ArrayBuffer（返回 Promise） blob.text(); // 转为文本（返回 Promise） blob.stream(); // 转为 ReadableStream

Blob URL

// 创建 Blob URL const url = URL.createObjectURL(blob); // blob:http://... // 使用（可以用于 img/a/video/iframe 的 src） img.src = url; a.href = url; a.download = 'filename.txt'; // 指定下载文件名 // 释放 URL.revokeObjectURL(url); // 释放内存 // ⚠️ 每次 createObjectURL 都占用内存 // ⚠️ 用完必须 revoke，否则内存泄漏 // ⚠️ 同一个 blob 多次 createObjectURL 会得到不同 URL
