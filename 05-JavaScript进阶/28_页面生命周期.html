页面生命周期

页面从加载到卸载的完整生命周期事件。

生命周期阶段

// 各阶段的转换： // 加载 → active（活跃） ↔ passive（不可见） ↔ frozen（冻结） → terminated（终止） // // active: 页面可见，用户可交互 // passive: 页面不可见（切换标签/最小化），但可能仍在运行 // frozen: 页面资源被冻结（移动端 Safari / 后台标签） // terminated: 页面被关闭或导航离开 // 关键事件： document.addEventListener('readystatechange', () => { document.readyState; // 'loading' → 'interactive' → 'complete' }); window.addEventListener('DOMContentLoaded', () => { // HTML 解析完成，DOM 可用 }); window.addEventListener('load', () => { // 所有资源（图片/样式）加载完成 }); document.addEventListener('visibilitychange', () => { document.visibilityState; // 'visible' | 'hidden' document.hidden; // true/false }); window.addEventListener('beforeunload', (e) => { // 用户即将离开页面（可阻止） e.preventDefault(); e.returnValue = ''; }); window.addEventListener('unload', () => { // 页面正在卸载（可以做最后的数据保存） // ⚠️ 只能做同步操作（async 不保证执行） }); // 页面冻结/恢复（Safari / Chrome 后台） document.addEventListener('freeze', () => { }); document.addEventListener('resume', () => { }); // 页面往返缓存（bfcache） window.addEventListener('pageshow', (e) => { if (e.persisted) console.log('从 bfcache 恢复'); }); window.addEventListener('pagehide', (e) => { if (e.persisted) console.log('页面被存入 bfcache'); });
💡 visibilitychange 是推荐的事件：可以暂停/恢复视频、动画、WebSocket。
⚠️ beforeunload 大部分浏览器已忽略自定义消息。
💡 bfcache (Back-Forward Cache) 使页面可以在返回时瞬间恢复。
