量词

* / + / ? / {n,m}、贪婪模式、懒惰模式。

量词大全

// ========== 量词 ========== // * — 0 次或多次 {0,} // + — 1 次或多次 {1,} // ? — 0 次或 1 次 {0,1} // {n} — 恰好 n 次 // {n,} — 至少 n 次 // {n,m} — n 到 m 次 // ========== 贪婪模式 (默认) ========== // 尽量匹配更多字符 /a+/ // 对 'aaa' ⇒ 'aaa' /\d{2,4}/ // 对 '12345' ⇒ '1234' (取最多) // ========== 懒惰模式 (加 ?) ========== // 尽量匹配更少字符 /a+?/ // 对 'aaa' ⇒ 'a' /\d{2,4}?/ // 对 '12345' ⇒ '12' (取最少) /a*?b/ // 对 'aab' ⇒ 'aab' (找到即停) /.*?/ // 尽量少匹配 (可能为空) // ========== 占有量词 (JS 不支持) ========== // JS 不支持 *+ / ++ / ?+ (占有量词) // 但可用原子组 (?>...) 替代 (部分引擎) // ========== 常见陷阱 ========== // /<.+>/ 对 '
text
' ⇒ 整个字符串（贪婪） // /<.+?>/ 对 '
text
' ⇒ '
'（懒惰） // 对比 'abbb'.match(/ab+/) // ['abb'] 贪婪 'abbb'.match(/ab+?/) // ['ab'] 懒惰
