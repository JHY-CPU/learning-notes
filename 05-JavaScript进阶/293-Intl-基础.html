Intl 简介

// ========== Intl 国际化 API ========== // Intl 是 JS 内置的国际化对象 // 提供日期、数字、排序、复数等本地化功能 // ========== Locale 语言标签 ========== // 格式: language[-script][-region][-extensions] // 'zh' — 中文 // 'zh-CN' — 中文 (中国) // 'zh-Hans-CN' — 中文 (简体, 中国) // 'en' — 英语 // 'en-US' — 英语 (美国) // 'en-GB' — 英语 (英国) // 'ja-JP' — 日语 (日本) // 'de-DE' — 德语 (德国) // 'ar-EG' — 阿拉伯语 (埃及) // ========== Intl 主要构造器 ========== Intl.DateTimeFormat // 日期格式化 Intl.NumberFormat // 数字格式化 Intl.Collator // 字符串排序/比较 Intl.ListFormat // 列表连接 (ES2021) Intl.RelativeTimeFormat // 相对时间 (ES2020) Intl.PluralRules // 复数规则 Intl.Segmenter // 文本分割 (ES2022) // ========== 基本用法 ========== // 所有 Intl 构造器用法一致: const formatter = new Intl.XXX(locale, options); formatter.format(value); // ========== 检测支持 ========== if (Intl) console.log('Intl 支持'); if (typeof Intl.DateTimeFormat !== 'undefined') { // 浏览器支持 }
