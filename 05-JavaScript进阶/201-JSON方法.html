JSON 方法

JSON.stringify / JSON.parse — 序列化与反序列化。

JSON.stringify

// 序列化为 JSON 字符串 JSON.stringify(value); JSON.stringify(value, replacer); JSON.stringify(value, replacer, space); // 基本 JSON.stringify({ a: 1, b: 2 }); // '{"a":1,"b":2}' JSON.stringify([1, 2, 3]); // '[1,2,3]' JSON.stringify('hello'); // '"hello"' JSON.stringify(42); // '42' JSON.stringify(true); // 'true' JSON.stringify(null); // 'null' // replacer — 过滤或转换 JSON.stringify(obj, ['key1', 'key2']); // 只序列化指定键 JSON.stringify(obj, (key, value) => { // 函数处理 if (key === 'password') return undefined; // 排除 if (typeof value === 'bigint') return value.toString(); return value; }); // space — 美化缩进 JSON.stringify(obj, null, 2); // 2 空格缩进 JSON.stringify(obj, null, '\t'); // Tab 缩进 // toJSON — 自定义对象的序列化 const obj = { data: 42, toJSON() { return { custom: this.data }; } }; // JSON.stringify(obj) → '{"custom":42}' // ⚠️ 不支持: undefined, Function, Symbol（会被忽略或转为 null） // ⚠️ BigInt 会报错（需要 replacer 处理） // ⚠️ 循环引用会抛错

JSON.parse

// 解析 JSON 字符串 JSON.parse(text); JSON.parse(text, reviver); // 基本 JSON.parse('{"a":1,"b":2}'); // { a: 1, b: 2 } JSON.parse('[1,2,3]'); // [1, 2, 3] JSON.parse('"hello"'); // 'hello' JSON.parse('42'); // 42 // reviver — 转换解析后的值 JSON.parse('{"date":"2024-01-01"}', (key, value) => { if (key === 'date') return new Date(value); return value; }); // ⚠️ JSON.parse 不会解析 undefined/Symbol/Function // ⚠️ 不是真正的 deep clone（Date → string, undefined 丢失, 循环引用崩溃） // ⚠️ 安全风险：永远不要解析不可信的 JSON（会执行 reviver）
