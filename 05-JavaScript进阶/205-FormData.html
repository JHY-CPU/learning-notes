FormData

构建表单数据，用于文件上传和表单提交。

基本用法

// 创建 const fd = new FormData(); // 空 const fd2 = new FormData(formElement); // 从表单元素创建 // 操作 fd.append('username', 'Alice'); // 追加（允许同名） fd.set('username', 'Bob'); // 设置（覆盖同名） fd.get('username'); // 获取第一个值 fd.getAll('username'); // 获取所有值（数组） fd.has('username'); // 是否存在 fd.delete('username'); // 删除 fd.keys(); // 所有键 fd.values(); // 所有值 fd.entries(); // 所有 [键, 值] 对 // 遍历 for (const [key, value] of fd.entries()) { console.log(key, value); } // 或 for (const [key, value] of fd) { } // 文件上传 fd.append('avatar', fileInput.files[0]); fd.append('photos', fileInput.files[0], 'custom-name.jpg'); // 指定文件名 // 字符串值 vs Blob fd.append('note', 'hello'); // 字符串 fd.append('doc', new Blob(['content']), 'doc.txt'); // Blob // 使用 fetch 发送 fetch('/upload', { method: 'POST', body: fd, // 自动设置 Content-Type: multipart/form-data // ⚠️ 不要手动设置 Content-Type（浏览器会自动处理 boundary） });
💡 FormData 的值可以是字符串、Blob、File。
⚠️ 通过 fetch 发送 FormData 时不要手动设置 Content-Type！
💡 append 和 set 的区别：append 允许同名多个值，set 覆盖之前所有同名值。

从表单元素创建

// HTML 表单 // <form id="myForm"> // <input name="username" value="Alice"> // <input name="email" value="a@test.com"> // <select name="role">...</select> // </form> const form = document.getElementById('myForm'); const fd = new FormData(form); // 注意： // - 只包含有 name 属性的表单元素 // - checkbox 只包含勾选的 // - file input 包含 File 对象 // - button 的 name/value 不会包含
