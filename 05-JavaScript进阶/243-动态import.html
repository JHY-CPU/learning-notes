动态 import

import() 按需加载、条件加载、模块预加载。

动态 import 语法

// ========== 动态 import() ========== // import() 返回 Promise<Module> const module = await import('./module.js'); // ========== 基本用法 ========== // 按需加载 button.addEventListener('click', async () => { const { showToast } = await import('./toast.js'); showToast('Hello!'); }); // 条件加载 if (locale === 'zh') { const { formatDate } = await import('./date-zh.js'); } else { const { formatDate } = await import('./date-en.js'); } // ========== 解构导入 ========== const { default: React, useState } = await import('react'); // ========== 命名空间导入 ========== const math = await import('./math.js'); math.sum(1, 2); // ========== 动态路径 ========== const path = `./lang/${lang}.js`; const messages = await import(path); // ========== 预加载 (Webpack magic comment) ========== // webpackPrefetch: true — 浏览器空闲时预加载 // webpackPreload: true — 与父模块并行加载 const Chart = await import(/* webpackPrefetch: true */ './Chart.js'); // ========== 与静态 import 对比 ========== // 静态 import: 编译时解析，提升到顶部，必须写在顶层 // 动态 import: 运行时解析，可在任何位置，返回 Promise // ✅ 动态 import 支持: 按需 / 条件 / 变量路径 // ❌ 动态 import 不能静态分析 (tree-shaking 失效)
