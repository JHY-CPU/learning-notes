全屏 Fullscreen API

将元素或页面切换至全屏模式。

基本用法

// 进入全屏（必须由用户手势触发） element.requestFullscreen(); element.requestFullscreen({ navigationUI: 'hide' }); // 隐藏导航栏 // 退出全屏 document.exitFullscreen(); // 检查当前全屏元素 document.fullscreenElement; // 当前全屏的元素（null 表示未全屏） // 检查是否支持全屏 if (document.fullscreenEnabled) { // 支持全屏 } // 事件 document.addEventListener('fullscreenchange', () => { if (document.fullscreenElement) { console.log('进入全屏'); } else { console.log('退出全屏'); } }); document.addEventListener('fullscreenerror', () => { console.error('全屏切换失败'); }); // CSS 伪类（全屏时生效） // :fullscreen { ... } // ::backdrop { background: black; } // 全屏背景 // ⚠️ 浏览器前缀（旧版） // element.webkitRequestFullscreen() // document.webkitExitFullscreen() // document.webkitFullscreenElement
⚠️ 必须由用户手势（click/keydown）触发，不能自动进入全屏。
⚠️ 不同浏览器有前缀（webkit），注意兼容。
💡 按 ESC 或 F11 也可以退出全屏。
