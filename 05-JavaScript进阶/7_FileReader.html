FileReader

读取文件内容（文本/DataURL/ArrayBuffer）。

基本用法

// 创建 const reader = new FileReader(); // 读取方法 reader.readAsText(file, encoding); // 读取为文本（encoding 默认 'UTF-8'） reader.readAsDataURL(file); // 读取为 Data URL（base64） reader.readAsArrayBuffer(file); // 读取为 ArrayBuffer（二进制） reader.readAsBinaryString(file); // 读取为二进制字符串（已弃用） // 事件 reader.onload = () => { reader.result; // 读取结果（根据读取方法不同） reader.readyState; // 0=EMPTY, 1=LOADING, 2=DONE }; reader.onerror = () => { reader.error; // DOMException }; reader.onprogress = (e) => { e.loaded; // 已读取 e.total; // 总大小 }; reader.onloadstart = () => {}; // 开始 reader.onloadend = () => {}; // 完成（无论成功/失败） // 中止 reader.abort(); // 注意：FileReader 只能读取用户选择的文件（安全限制） // 通常通过 <input type="file"> 或拖放获得 File 对象
