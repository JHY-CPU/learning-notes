不可变数据 Immutability

数据一旦创建就不能被修改。任何"修改"操作都返回新的数据对象，原始数据保持不变。带来可预测性、方便变更检测、便于调试。

实现不可变的技术

Object.freeze： 浅层冻结。展开运算符： {...obj} 创建浅拷贝。structuredClone： 深拷贝。自定义 update 函数： 使用路径语法更新嵌套属性。

Immer 原理

Immer 库的 produce() 函数允许你"修改"一个 draft 对象，实际返回基于原始数据的新对象。通过 Proxy 记录所有修改操作并生成不可变结果。

const nextState = produce(baseState, draft => { draft.count++ });
