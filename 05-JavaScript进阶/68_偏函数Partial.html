偏函数 Partial Application

固定函数的部分参数，返回一个接受剩余参数的新函数。与柯里化的区别：偏函数预设任意位置的参数，柯里化将多参数函数转为单参数函数链。

偏函数 vs 柯里化

偏函数 (Partial)

  • 固定部分参数，剩余参数稍后传入
  • 不要求每次只传一个参数
  • 可以对任意位置的参数进行预设
  • 示例: partial(fn, 1, _, 3)(2)

柯里化 (Currying)

  • 将 f(a,b,c) 转为 f(a)(b)(c)
  • 每次只接收一个参数
  • 严格从左到右逐个接收
  • 示例: curry(fn)(1)(2)(3)

使用 bind() 实现偏函数

Function.prototype.bind() 不仅可以绑定 this，还能固定部分参数，是内置的偏函数工具。

应用场景

固定地区/货币的格式化函数、预设 HTTP 请求头、API 请求封装、事件处理函数参数预设。
