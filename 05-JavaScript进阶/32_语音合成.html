语音合成 Speech Synthesis

浏览器文字转语音（TTS）API。

基本用法

// 创建语音实例 const utterance = new SpeechSynthesisUtterance('要朗读的文本'); // 配置 utterance.text = '你好世界'; // 文本内容 utterance.lang = 'zh-CN'; // 语言（zh-CN/zh-TW/en-US/ja-JP） utterance.rate = 1.0; // 语速 (0.1 ~ 10, 1=正常) utterance.pitch = 1.0; // 音调 (0 ~ 2, 1=正常) utterance.volume = 1.0; // 音量 (0 ~ 1) utterance.voice = selectedVoice; // 指定语音（从 getVoices() 选择） // 事件 utterance.onstart = () => {}; // 开始朗读 utterance.onend = () => {}; // 朗读结束 utterance.onpause = () => {}; // 暂停 utterance.onresume = () => {}; // 恢复朗读 utterance.onboundary = (e) => { // 词/句边界 e.charIndex; // 字符索引 e.name; // 'word' | 'sentence' }; utterance.onerror = (e) => {}; // 出错 // 控制 speechSynthesis.speak(utterance); // 开始/加入队列 speechSynthesis.pause(); // 暂停 speechSynthesis.resume(); // 恢复 speechSynthesis.cancel(); // 取消所有 speechSynthesis.speaking; // 是否正在朗读 speechSynthesis.paused; // 是否暂停 speechSynthesis.pending; // 是否有待朗读 // 获取可用语音（需异步） const voices = speechSynthesis.getVoices(); // 语音对象: voice.name, voice.lang, voice.localService
⚠️ 不同浏览器/系统的语音列表不同。
⚠️ getVoices() 在页面加载时可能为空，需要监听 voiceschanged 事件。
💡 Chrome 需要用户手势触发第一次 speak()。
