Reflect API

Reflect 是 ES6 提供的全局对象，封装了底层操作（属性读写、函数调用、原型操作等）。所有方法都是静态的，与 Proxy 的处理器陷阱一一对应。

Reflect vs Object 方法

返回值差异：Object.defineProperty 失败抛异常，Reflect.defineProperty 返回 false。Receiver 参数：Reflect.get/set 支持 receiver 参数，正确传递 this 上下文。

在 Proxy 中始终使用 Reflect

在 Proxy 陷阱中调用 Reflect 的对应方法可确保默认行为正确，避免破坏原生语义。特别是正确传递 receiver 参数。
