跨标签通信

BroadcastChannel / postMessage / storage 事件 — 标签页间传递数据。

BroadcastChannel

// BroadcastChannel — 同源下多个标签页通信（推荐） // 创建（同一个频道名 = 同一组） const channel = new BroadcastChannel('my-channel'); // 发送 channel.postMessage({ type: 'update', data: 'Hello' }); // 接收 channel.onmessage = (event) => { console.log('收到:', event.data); }; // 或 channel.addEventListener('message', handler) // 错误 channel.onmessageerror = (event) => {}; // 关闭连接 channel.close(); // 特点： // ✅ 同源下任意标签页/iframe 间通信 // ✅ 双向通信 // ✅ API 简洁 // ❌ 不支持跨域

postMessage (跨窗口/iframe)

// 跨窗口消息（支持跨域） // 发送消息 targetWindow.postMessage(data, targetOrigin); targetWindow.postMessage({ type: 'greet' }, 'https://example.com'); targetWindow.postMessage('data', '*'); // ⚠️ 避免用 *（安全风险） // 接收消息 window.addEventListener('message', (event) => { // ⚠️ 必须验证来源！ if (event.origin !== 'https://trusted.com') return; console.log('收到:', event.data); console.log('来源:', event.origin); console.log('来源窗口:', event.source); // window }); // targetWindow 可以是: // - iframe.contentWindow // - window.open() 的返回值 // - event.source（来自收到的消息）

storage 事件

// localStorage 变化时触发（其他标签页） // ⚠️ 当前页面不触发，只有同源的其他标签页触发 window.addEventListener('storage', (event) => { console.log('键:', event.key); console.log('旧值:', event.oldValue); console.log('新值:', event.newValue); console.log('URL:', event.url); console.log('存储区:', event.storageArea); }); // 用途：跨标签同步状态
