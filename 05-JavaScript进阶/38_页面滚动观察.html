高级滚动检测

// 1. scrollend 事件（浏览器原生，Chrome 105+） // 用户在滚动完全停止后触发（手指抬起 + 滚动惯性结束） element.addEventListener('scrollend', () => { console.log('滚动结束'); }); // 2. IntersectionObserver 高级配置 // threshold: [0, 0.25, 0.5, 0.75, 1] — 多阈值触发 const observer = new IntersectionObserver(callback, { root: scrollContainer, // 滚动容器（null = 视口） rootMargin: '100px 0px', // 扩大触发区域（提前 100px 触发） threshold: [0, 0.5, 1], }); // 3. CSS scroll-snap（纯 CSS 滚动吸附） // scroll-snap-type: y mandatory; /* 容器 */ // scroll-snap-align: start; /* 子元素 */ // 4. 检测滚动方向 let lastScrollY = 0; window.addEventListener('scroll', () => { const direction = window.scrollY > lastScrollY ? '⬇️ 向下' : '⬆️ 向上'; lastScrollY = window.scrollY; }, { passive: true }); // 5. 检测是否滚动到底部 const isBottom = window.innerHeight + window.scrollY >= document.documentElement.scrollHeight - 10; // 6. 检测元素是否完全可见 function isFullyVisible(el) { const rect = el.getBoundingClientRect(); return rect.top >= 0 && rect.bottom <= window.innerHeight; }
