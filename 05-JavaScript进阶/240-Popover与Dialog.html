Popover 与 Dialog

浏览器原生弹出面板和对话框 API。

Popover API

// HTML 声明式（最简用法） // <button popovertarget="my-popover">打开</button> // <div id="my-popover" popover>内容</div> // JS API const popover = document.getElementById('my-popover'); popover.showPopover(); // 打开 popover.hidePopover(); // 关闭 popover.togglePopover(); // 切换 // popover 属性值 // popover="auto" — 默认，点击外部/ESC 自动关闭（一次只能打开一个） // popover="manual" — 必须手动关闭 // 事件 popover.addEventListener('beforetoggle', (e) => { if (e.newState === 'open') { } if (e.newState === 'closed') { } }); popover.addEventListener('toggle', (e) => { }); // CSS 伪类 // [popover]:popover-open { } — 打开状态 // [popover]::backdrop { } — 背景层

Dialog API

// 创建 const dialog = document.getElementById('my-dialog'); // 打开 dialog.show(); // 非模态（可操作外部） dialog.showModal(); // 模态（阻止外部操作，焦点自动管理） // 关闭 dialog.close(); // 关闭 dialog.close('result'); // 带返回值 // 事件 dialog.addEventListener('close', () => { console.log(dialog.returnValue); // 获取关闭时的返回值 }); dialog.addEventListener('cancel', () => { // ESC 键关闭时触发 }); // 表单集成 // <form method="dialog"> 提交时自动关闭 dialog // <button value="ok">确定</button> // CSS 伪类 // dialog::backdrop { background: rgba(0,0,0,0.5); }
💡 showModal() 自动处理焦点陷阱（Tab 循环在对话框内）。
💡 Popover 适合轻量提示/tooltip，Dialog 适合正式对话框/表单。
🤝 Popover API 较新（Chrome 114+），Dialog 支持更广泛。
