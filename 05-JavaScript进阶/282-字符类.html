字符类

\d / \w / \s / . / [abc] / [^abc] / Unicode 类 \p{}。

字符类大全

// ========== 预定义字符类 ========== // \d — 数字 [0-9] // \D — 非数字 [^0-9] // \w — 单词字符 [a-zA-Z0-9_] // \W — 非单词字符 [^a-zA-Z0-9_] // \s — 空白字符 [ \t\n\r\f\v] // \S — 非空白字符 // . — 任意字符（除换行），dotAll 模式 (s) 包含换行 // ========== 自定义字符类 [...] ========== [abc] // a 或 b 或 c [^abc] // 除了 a/b/c 之外的任何字符 [a-z] // 小写字母 a-z [0-9] // 数字 [a-zA-Z] // 所有字母 [0-9a-f] // 十六进制数字 [一-鿿] // 中文字符 (Unicode 范围) // ========== Unicode 字符类 \p{} ========== // 需要 u flag \p{L} // 任意字母 (Letter) \p{N} // 任意数字 (Number) \p{P} // 标点符号 (Punctuation) \p{S} // 符号 (Symbol) \p{Z} // 分隔符 (Separator) \p{Sc} // 货币符号 ($, €, ¥) \p{Han} // 汉字 (CJK 表意文字) // ========== 非 Unicode 类 ========== // \t — 制表符 // \n — 换行符 // \r — 回车符 // \f — 换页符 // \v — 垂直制表符 // \0 — null 字符
