正则基础

/pattern/flags 创建、test/exec 方法、常用 flag。

正则创建与基础方法

// ========== 创建正则 ========== // 字面量语法（推荐） const re1 = /hello/; const re2 = /hello/gi; // g全局 i不区分大小写 // 构造函数（支持变量） const pattern = 'hello'; const re3 = new RegExp(pattern, 'gi'); // ========== 实例方法 ========== // test() — 是否匹配（返回 boolean） /hello/.test('hello world'); // true /hello/.test('hi world'); // false // exec() — 返回匹配结果数组（或 null） const match = /l{2}/.exec('hello'); // ['ll', index: 2, input: 'hello', groups: undefined] // 全局模式下 exec 会记住 lastIndex const globalRe = /l/g; globalRe.exec('hello'); // ['l', index: 2] globalRe.exec('hello'); // ['l', index: 3] globalRe.exec('hello'); // null // ========== 字符串的正则方法 ========== 'Hello World'.match(/l/g); // ['l', 'l', 'l'] 'Hello World'.search(/W/); // 6 (索引) 'Hello World'.replace(/l/g, 'L'); // 'HeLLo WorLd' 'a,b,c'.split(/,/); // ['a', 'b', 'c'] // ========== 常用 flag ========== // g — 全局匹配 (global) // i — 不区分大小写 (ignoreCase) // m — 多行模式 (multiline) // s — dotAll (. 匹配换行) // u — Unicode 模式 // y — 粘性匹配 (sticky) // d — 匹配索引 (hasIndices)
