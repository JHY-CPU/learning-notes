单例模式 Singleton

确保一个类只有一个实例，并提供全局访问点。适用于全局状态管理、日志记录、配置管理等场景。

IIFE闭包实现

利用立即执行函数和闭包保存实例引用，通过 getInstance 方法获取唯一实例。

const Singleton = (function() { let instance; function init() { return { name: '实例' }; } return { getInstance() { if (!instance) instance = init(); return instance; } }; })();

Class静态方法实现

ES6 class 配合 static getInstance 方法，实现更优雅的单例。

class Singleton { static #instance; constructor() { if (Singleton.#instance) return Singleton.#instance; Singleton.#instance = this; } }

模块单例与懒加载

ES Module 天然支持单例（模块只执行一次），结合懒加载避免不必要的资源消耗。
