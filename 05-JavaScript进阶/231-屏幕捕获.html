屏幕捕获 getUserMedia

获取摄像头/麦克风/屏幕的媒体流。

基本用法

// 获取摄像头 + 麦克风 const stream = await navigator.mediaDevices.getUserMedia({ video: { width: { ideal: 1280 }, height: { ideal: 720 }, facingMode: 'user', // 'user'前置 | 'environment'后置 frameRate: { ideal: 30 }, }, audio: true, }); // 获取屏幕共享 const stream = await navigator.mediaDevices.getDisplayMedia({ video: { cursor: 'always' }, // 'always' | 'motion' | 'never' audio: true, // 是否包含系统音频 }); // 枚举设备 const devices = await navigator.mediaDevices.enumerateDevices(); // device.kind: 'videoinput' | 'audioinput' | 'audiooutput' // device.label: 设备名称（需授权后才有） // 显示到 video 元素 const video = document.getElementById('video'); video.srcObject = stream; video.play(); // 停止所有轨道 stream.getTracks().forEach(track => track.stop()); // 截图（canvas） const canvas = document.createElement('canvas'); canvas.width = video.videoWidth; canvas.height = video.videoHeight; canvas.getContext('2d').drawImage(video, 0, 0); const screenshot = canvas.toDataURL('image/png');
⚠️ getUserMedia 需要 HTTPS（localhost 可以）。
⚠️ 必须由用户手势触发（click 等）。
⚠️ 移动端需要用户授予摄像头/麦克风权限。
💡 enumerateDevices() 在授权前返回的 label 为空。
