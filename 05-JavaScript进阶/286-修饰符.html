修饰符

i / g / m / s / u / y / d 修饰符及其组合效果。

修饰符大全

// ========== 修饰符 ========== // i — ignoreCase 不区分大小写 // /hello/i.test('HELLO') // true // g — global 全局匹配 // 'a1b2c3'.match(/\d/g) // ['1','2','3'] // m — multiline 多行模式 // ^ 和 $ 匹配每行的开头/结尾 // 'a\nb'.match(/^\w$/gm) // ['a','b'] // s — dotAll (ES2018) // . 匹配换行符 // 'a\nb'.match(/a.b/s) // ['a\nb'] // u — unicode (ES2015) // 启用 Unicode 匹配 // \p{L} 等 Unicode 属性需要 u 标志 // '😀'.match(/./u) // 正确匹配 4 字节字符 // y — sticky (ES2015) // 粘性匹配，从 lastIndex 开始 // const re = /a/y; re.lastIndex = 1; 'aba'.match(re) // null // d — hasIndices (ES2022) // 匹配结果包含索引 // 'abc'.match(/(b)/d) // indices: [[1,2],[1,2]] // ========== 组合效果 ========== /hello/gim // 全局 + 不区分大小写 + 多行 /./gs // dotAll + 全局 /\p{L}+/gu // Unicode 字母 + 全局 // ========== 修饰符顺序不影响结果 ========== /hello/gim === /hello/mig // true (等效)
