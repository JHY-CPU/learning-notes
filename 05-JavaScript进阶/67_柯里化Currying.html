柯里化 Currying

将接收多个参数的函数转换为每次只接收一个参数的嵌套函数序列。f(a,b,c) → f(a)(b)(c)。核心思想：参数复用和延迟计算。

通用 curry 实现

支持任意元数函数的自动柯里化，直到收集够所有参数才执行原函数。

function curry(fn) { return function curried(...args) { if (args.length >= fn.length) return fn(...args); return (...more) => curried(...args, ...more); }; }

占位符支持

使用 _ 作为占位符，允许跳过某些参数，稍后再填充。

const curried = curry(fn, _); curried(1, _, 3)(2); // 占位符被 2 替换

实际应用场景

日志级别函数（log('INFO')('消息')）、URL 构建器（buildUrl(base)(path)）、事件处理器配置等。
