# 分组与捕获

## 一、概念说明

分组（Group）使用圆括号 `()` 将多个字符组合成一个单元，可以对其应用量词或提取匹配内容。捕获组会保存匹配结果供后续使用。

```javascript
// 分组的作用
/*
1. 将多个字符作为一个单元
2. 捕获匹配内容
3. 后向引用
4. 分组命名
*/
```

## 二、基本分组

```javascript
// 将多个字符作为一个单元
/abc+/.test('abccc');      // true - c 重复
/(abc)+/.test('abcabc');   // true - abc 整体重复

// 分组应用量词
/(ab)+/.test('ababab');    // true
/(ab){2,3}/.test('abab');  // true

// 可选分组
/hello( world)?/.test('hello');        // true
/hello( world)?/.test('hello world');  // true
```

## 三、捕获组

```javascript
// 获取捕获内容
const date = '2024-01-15';
const match = date.match(/(\d{4})-(\d{2})-(\d{2})/);
// match[0] = '2024-01-15'
// match[1] = '2024'  (年)
// match[2] = '01'    (月)
// match[3] = '15'    (日)

// 使用 exec 获取所有捕获
const regex = /(\d{4})-(\d{2})-(\d{2})/;
const result = regex.exec('日期: 2024-01-15');
// result[1] = '2024'
// result.index = 4

// matchAll 获取所有匹配
const text = '2024-01-15, 2024-02-20';
const matches = [...text.matchAll(/(\d{4})-(\d{2})-(\d{2})/g)];
// [['2024-01-15', '2024', '01', '15'], ['2024-02-20', '2024', '02', '20']]
```

### 3.1 命名捕获组

```javascript
// 命名捕获组 (?<name>pattern)
const date = '2024-01-15';
const match = date.match(/(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/);

console.log(match.groups.year);   // '2024'
console.log(match.groups.month);  // '01'
console.log(match.groups.day);    // '15'

// 在 replace 中使用
'2024-01-15'.replace(
  /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/,
  '$<month>/$<day>/$<year>'
);
// '01/15/2024'
```

### 3.2 非捕获组

```javascript
// 非捕获组 (?:pattern) - 只分组不捕获
const text = 'hello world hello dart';

// 捕获组 - 会创建额外的捕获
const withCapture = text.match(/(hello) (world|dart)/);
// ['hello world', 'hello', 'world']

// 非捕获组 - 不创建捕获
const withoutCapture = text.match(/(hello) (?:world|dart)/);
// ['hello world', 'hello']

// 实际应用：只关心外层捕获
const prices = '价格: $10.5, $20.0';
const matches = [...prices.matchAll(/\$(\d+(?:\.\d{2})?)/g)];
// [['$10.5', '10.5'], ['$20.0', '20.0']]
```

## 四、后向引用

```javascript
// 后向引用 \1, \2...
// 匹配重复的单词
const text = 'the the quick quick fox';
const duplicates = text.match(/\b(\w+)\s+\1\b/gi);
// ['the the', 'quick quick']

// HTML 标签匹配
const html = '<div>content</div>';
const tagMatch = html.match(/<(\w+)>(.*?)<\/\1>/);
// ['<div>content</div>', 'div', 'content']

// 命名后向引用 \k<name>
const pattern = /(?<word>\w+)\s+\k<word>/;
pattern.test('hello hello');  // true
```

## 五、实践示例

```javascript
// 提取 URL 各部分
const url = 'https://www.example.com:8080/path/to/page?key=value#section';
const urlMatch = url.match(
  /^(?<protocol>https?):\/\/(?<host>[\w.]+)(?::(?<port>\d+))?(?<path>\/[^?#]*)?(?:\?(?<query>[^#]*))?(?:#(?<hash>.*))?$/
);
// groups: { protocol: 'https', host: 'www.example.com', port: '8080', ... }

// 解析 key=value 对
const query = 'name=张三&age=25&city=北京';
const params = Object.fromEntries(
  [...query.matchAll(/(?<key>\w+)=(?<value>[^&]+)/g)]
    .map(m => [m.groups.key, decodeURIComponent(m.groups.value)])
);
// { name: '张三', age: '25', city: '北京' }

// 替换日期格式
'日期: 2024-01-15'.replace(
  /(?<y>\d{4})-(?<m>\d{2})-(?<d>\d{2})/,
  '$<m>月$<d>日$<y>年'
);
// '日期: 01月15日2024年'
```

## 六、注意事项

1. **捕获组编号**：按左括号从左到右编号，从 1 开始
2. **嵌套分组**：外层编号大于内层
3. **非捕获组性能**：不需要捕获时使用 `(?:)` 减少开销
4. **后向引用限制**：只能引用已捕获的内容
5. **命名冲突**：避免在同一正则中使用相同的组名
