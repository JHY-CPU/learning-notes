# 字符匹配基础

## 一、概念说明

正则表达式的核心是字符匹配。最基本的匹配就是精确匹配一个字符或使用元字符匹配一类字符。

```javascript
// 基本字符匹配
/a/.test('abc');           // true - 精确匹配 'a'
/abc/.test('abcdef');      // true - 匹配连续字符
```

## 二、元字符

### 2.1 常用元字符

```javascript
// 常用元字符
/*
.   匹配任意字符（除换行符）
\d  匹配数字 [0-9]
\D  匹配非数字 [^0-9]
\w  匹配单词字符 [a-zA-Z0-9_]
\W  匷配非单词字符
\s  匹配空白字符（空格、制表符、换行）
\S  匹配非空白字符
*/

// 示例
/\d/.test('abc');          // false
/\d/.test('abc123');       // true
/\w/.test('hello');        // true
/\s/.test('hello world');  // true
/./.test('any');           // true
```

### 2.2 转义字符

```javascript
// 转义特殊字符
/\./.test('a.b');          // true - 匹配点号
/\//.test('a/b');          // true - 匹配斜杠
/\(/.test('a(b');          // true - 匹配左括号
/\\/.test('a\\b');         // true - 匹配反斜杠

// 常见需要转义的字符
// . * + ? ^ $ { } [ ] ( ) | \ /
```

## 三、字符类

```javascript
// 字符类 [abc]
/[abc]/.test('apple');     // true - 包含 a
/[abc]/.test('dog');       // false

// 范围 [a-z]
/[a-z]/.test('hello');     // true
/[A-Z]/.test('Hello');     // true
/[0-9]/.test('123');       // true
/[a-zA-Z0-9]/.test('a1'); // true

// 否定 [^abc]
/[^abc]/.test('def');      // true - 包含非 a/b/c 的字符
/[^0-9]/.test('abc');      // true

// 预定义字符类
/\d/.test('123');          // 等价于 [0-9]
/\D/.test('abc');          // 等价于 [^0-9]
/\w/.test('hello');        // 等价于 [a-zA-Z0-9_]
/\W/.test('!@#');          // 等价于 [^a-zA-Z0-9_]
/\s/.test(' ');            // 匹配空白
/\S/.test('a');            // 匹配非空白
```

## 四、实践示例

```javascript
// 匹配中国手机号（简化版）
const isPhone = (str) => /^1[3-9]\d{9}$/.test(str);
isPhone('13812345678');     // true
isPhone('12345678901');     // false

// 匹配纯数字
const isNumeric = (str) => /^\d+$/.test(str);
isNumeric('12345');         // true
isNumeric('123abc');        // false

// 匹配字母和数字组合
const isAlphanumeric = (str) => /^[a-zA-Z0-9]+$/.test(str);
isAlphanumeric('abc123');   // true
isAlphanumeric('abc@123');  // false

// 提取所有数字
const extractNumbers = (str) => str.match(/\d+/g);
extractNumbers('abc123def456'); // ['123', '456']
```

## 五、注意事项

1. **区分大小写**：默认区分大小写，使用 `i` 标志忽略大小写
2. **点号不匹配换行**：`.` 默认不匹配换行符，使用 `s` 标志（dotAll）
3. **字符类中的特殊字符**：在 `[]` 内，大部分特殊字符失去特殊含义
4. **连字符位置**：`[a-z]` 中 `-` 表示范围，放在开头或结尾表示字面量
5. **性能**：尽量使用具体的字符类而非 `.` 通配符
