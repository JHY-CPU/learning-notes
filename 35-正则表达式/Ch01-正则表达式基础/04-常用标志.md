# 常用标志

## 一、概念说明

标志（Flags/Modifiers）用于修改正则表达式的匹配行为，如全局匹配、忽略大小写、多行模式等。

```javascript
// JavaScript 正则标志
/*
g - 全局匹配（查找所有匹配）
i - 忽略大小写
m - 多行模式（^$ 匹配每行）
s - dotAll（. 匹配换行符）
u - Unicode 模式
y - 粘性匹配
*/
```

## 二、标志详解

### 2.1 全局标志 g

```javascript
// 无 g - 只匹配第一个
'abc abc abc'.match(/abc/);
// ['abc']

// 有 g - 匹配所有
'abc abc abc'.match(/abc/g);
// ['abc', 'abc', 'abc']

// replace 中的区别
'aaa'.replace(/a/, 'b');    // 'baa' - 只替换第一个
'aaa'.replace(/a/g, 'b');   // 'bbb' - 替换所有
```

### 2.2 忽略大小写 i

```javascript
// 默认区分大小写
/hello/.test('Hello');       // false

// 忽略大小写
/hello/i.test('Hello');      // true
/hello/i.test('HELLO');      // true

// 在替换中使用
'Hello HELLO'.replace(/hello/gi, 'hi');
// 'hi hi'
```

### 2.3 多行模式 m

```javascript
const text = 'line1\nline2\nline3';

// 默认模式
/^line/.test(text);          // true - 只检查第一行

// 多行模式
/^line/m.test(text);         // true - 每行检查
/text$/m.test(text);         // true
/^line\d$/m.test(text);      // true - 每行都匹配
```

### 2.4 dotAll s

```javascript
// 默认 . 不匹配换行
/hello.+world/.test('hello\nworld');   // false

// dotAll 模式
/hello.+world/s.test('hello\nworld');  // true
```

### 2.5 Unicode u

```javascript
// 处理 Unicode 字符
// 默认可能将 emoji 拆分
'😀'.match(/./gu);          // ['😀'] - 正确处理
'😀'.match(/./g);           // ['\uD83D', '\uDE00'] - 被拆分

// Unicode 属性
/\p{Letter}/u.test('中');   // true - 匹配任何语言的字母
/\p{Script=Han}/u.test('中'); // true - 匹配汉字
```

## 三、标志组合

```javascript
// 组合使用
const regex = /hello/gim;    // 全局 + 忽略大小写 + 多行

// 动态添加标志
const pattern = 'hello';
const regex = new RegExp(pattern, 'gi');

// 检查标志
const regex = /hello/gim;
regex.global;      // true
regex.ignoreCase;  // true
regex.multiline;   // true
regex.dotAll;      // false
regex.flags;       // 'gim'
```

## 四、实践示例

```javascript
// 大小写不敏感的搜索
const search = (text, keyword) => {
  const regex = new RegExp(keyword, 'gi');
  return text.match(regex) || [];
};
search('Hello HELLO hello', 'hello');
// ['Hello', 'HELLO', 'hello']

// 多行文本处理
const addLineNumbers = (text) => {
  let lineNum = 1;
  return text.replace(/^/gm, () => `${lineNum++}. `);
};
addLineNumbers('a\nb\nc');
// '1. a\n2. b\n3. c'

// 提取所有注释
const code = `
// comment 1
const x = 1;
// comment 2
`;
const comments = code.match(/\/\/.*$/gm);
// ['// comment 1', '// comment 2']
```

## 五、注意事项

1. **g 标志与 exec**：带 g 标志的 exec 每次返回下一个匹配
2. **m 影响 ^$**：开启 m 后 `^$` 匹配每行首尾而非整个字符串
3. **u 标志必要性**：处理 emoji 和 Unicode 字符时必须使用 u 标志
4. **性能影响**：g 标志会搜索所有匹配，比只找第一个慢
5. **标志不可变**：正则创建后无法修改标志
