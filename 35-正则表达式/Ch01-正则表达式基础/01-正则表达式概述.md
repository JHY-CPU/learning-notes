# 正则表达式概述

## 一、概念说明

正则表达式（Regular Expression，简称 regex 或 regexp）是一种用于描述字符串匹配模式的工具，广泛应用于文本搜索、替换、验证和数据提取。

```javascript
// 正则表达式基本概念
/*
正则表达式 = 模式字符串 + 匹配规则
用途:
1. 验证 - 检查字符串是否符合格式（邮箱、手机号）
2. 搜索 - 在文本中查找匹配内容
3. 替换 - 批量替换文本
4. 提取 - 从文本中提取特定信息
*/
```

## 二、基本语法

### 2.1 创建正则表达式

```javascript
// JavaScript 中创建正则的两种方式
// 1. 字面量
const regex1 = /hello/;

// 2. 构造函数
const regex2 = new RegExp('hello');

// 带标志
const regex3 = /hello/gi;  // g-全局, i-忽略大小写
```

```python
# Python 中使用正则
import re

# 编译正则
pattern = re.compile(r'hello')

# 直接使用
result = re.search(r'hello', 'hello world')
```

```dart
// Dart 中使用正则
final regex = RegExp(r'hello');
final hasMatch = regex.hasMatch('hello world');
```

### 2.2 基本匹配

```javascript
// 精确匹配
/hello/.test('hello world');     // true
/hello/.test('hi world');        // false

// 大小写不敏感
/hello/i.test('Hello World');    // true

// 全局匹配
const matches = 'hello hello'.match(/hello/g);
// ['hello', 'hello']
```

## 三、常用方法

```javascript
// JavaScript 正则方法

// test - 检测匹配
/d+/.test('abc123');             // true

// exec - 获取匹配详情
/\d+/.exec('abc123def456');     // ['123', index: 3]

// String.match
'abc123'.match(/\d+/);          // ['123']

// String.matchAll
[...'abc123def456'.matchAll(/\d+/g)];
// [['123'], ['456']]

// String.replace
'hello world'.replace(/world/, 'dart');
// 'hello dart'

// String.split
'a,b,c'.split(/,/);            // ['a', 'b', 'c']
```

## 四、注意事项

1. **贪婪与懒惰**：默认贪婪匹配，使用 `?` 切换为懒惰匹配
2. **转义字符**：特殊字符需要转义（如 `\.` 匹配点号）
3. **性能考虑**：避免过度复杂的正则，可能引起回溯灾难
4. **可读性**：复杂正则应添加注释或拆分为多个简单表达式
5. **跨语言差异**：不同语言的正则语法有细微差异
