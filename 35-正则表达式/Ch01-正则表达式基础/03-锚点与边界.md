# 锚点与边界

## 一、概念说明

锚点（Anchor）用于匹配位置而非字符，它们不消耗字符，只断言某个位置是否满足条件。

```javascript
// 锚点类型
/*
^   行/字符串开头
$   行/字符串结尾
\b  单词边界
\B  非单词边界
*/
```

## 二、行首和行尾

```javascript
// ^ 匹配开头
/^hello/.test('hello world');    // true
/^hello/.test('say hello');      // false

// $ 匹配结尾
/world$/.test('hello world');    // true
/world$/.test('world hello');    // false

// 完全匹配（最常用）
/^\d+$/.test('123');             // true - 整个字符串是数字
/^\d+$/.test('123abc');          // false

// 邮箱验证（简化）
/^\w+@\w+\.\w+$/.test('test@example.com');  // true
```

### 2.1 多行模式

```javascript
// 默认模式：^ 和 $ 匹配整个字符串
const text = 'line1\nline2\nline3';
/^line/.test(text);               // true - 只匹配第一行

// 多行模式 m 标志
/^line/m.test(text);              // true - 每行都检查开头
/line$/m.test(text);              // true - 每行都检查结尾

// 匹配所有行首
const lines = text.match(/^.*$/gm);
// ['line1', 'line2', 'line3']
```

## 三、单词边界

```javascript
// \b 匹配单词边界（\w 与 \W 之间的位置）
/\bcat\b/.test('the cat sat');    // true - cat 是独立单词
/\bcat\b/.test('concatenate');    // false - cat 不是独立单词
/\bcat/.test('category');         // true - cat 在单词开头
/cat\b/.test('bobcat');           // true - cat 在单词结尾

// 实际应用：精确替换
'cat and concatenate'.replace(/\bcat\b/g, 'dog');
// 'dog and concatenate' - 只替换了独立的 cat

// \B 非单词边界
/\Bcat/.test('concatenate');      // true - cat 不在单词开头
/cat\B/.test('category');         // true - cat 不在单词结尾
```

## 四、实践示例

```javascript
// 验证用户名（字母开头，允许字母数字下划线）
const isValidUsername = (name) => /^[a-zA-Z]\w{4,19}$/.test(name);
isValidUsername('user_01');       // true
isValidUsername('1user');         // false - 不能数字开头

// 验证密码（至少包含大小写字母和数字）
const isStrongPassword = (pwd) =>
  /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}$/.test(pwd);
isStrongPassword('Abc12345');    // true
isStrongPassword('abc12345');    // false - 无大写

// 提取独立单词
const findWord = (text, word) => {
  const regex = new RegExp(`\\b${word}\\b`, 'gi');
  return text.match(regex);
};
findWord('The cat sat on the concatenate', 'cat');
// ['cat']

// 清理行首行尾空格
const trimLines = (text) =>
  text.replace(/^\s+|\s+$/gm, '');
trimLines('  line1  \n  line2  ');
// 'line1\nline2'
```

## 五、注意事项

1. **^$ 在不同上下文中的含义**：在字符类 `[^abc]` 中 `^` 表示否定
2. **多行模式影响**：开启 `m` 后 `^$` 匹配每行的首尾
3. **单词边界适用范围**：`\b` 只识别 ASCII 单词字符
4. **Unicode 边界**：某些语言支持 `\b` 的 Unicode 版本
5. **性能**：锚点不消耗字符，对性能影响极小
