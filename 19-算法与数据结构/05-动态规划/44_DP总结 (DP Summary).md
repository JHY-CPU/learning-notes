## 2. 解题步骤总结

### 2.1 DP五步法

```
1. 定义状态 — dp[i] 的含义
2. 转移方程 — 状态之间的关系
3. 初始条件 — 最小子问题
4. 遍历顺序 — 保证子问题先求解
5. 返回答案 — dp[n] 或 max(dp)
```

### 2.2 快速判断

```
问：求最值？方案数？可行性？
问：是否有重叠子问题？（画递归树）
问：是否有最优子结构？
→ 是DP问题

问：状态有几个维度？
问：每个状态有多少种选择？
→ 确定状态和转移
```

## 3. 类型速查表

| 类型 | 状态定义 | 转移特征 | 典型问题 |
|------|---------|---------|---------|
| 线性DP | dp[i] | 从dp[i-1]转移 | 爬楼梯、打家劫舍 |
| 网格DP | dp[i][j] | 从上方/左方转移 | 不同路径 |
| 背包DP | dp[j] | 枚举物品 | 0-1背包 |
| 区间DP | dp[i][j] | 枚举分割点 | 石子合并 |
| 树形DP | dp[u] | 从子节点转移 | 树的直径 |
| 状压DP | dp[mask] | 位运算 | TSP |
| 数位DP | dfs(pos,state) | 逐位枚举 | 统计满足条件的数 |
| 股票DP | dp[i][0/1] | 状态机转移 | 买卖股票 |

## 4. 常用技巧速查

### 4.1 空间优化

```
二维→一维：只保留最近一行
一维→常数：只保留最近几个变量
背包问题：0-1逆序，完全正序
```

### 4.2 时间优化

```
单调队列：滑动窗口最值
前缀和：区间和查询
二分查找：LIS等
四边形不等式：区间DP优化
```

### 4.3 遍历方向

```
标准：从左到右、从小到大
背包：0-1逆序、完全正序
区间：按长度递增
网格：从左上到右下（或反向）
股票：交易次数逆序
```

## 5. 代码模板速查

### 5.1 一维DP

```python
dp = [0] * (n + 1)
dp[0] = 基础值
for i in range(1, n + 1):
    dp[i] = 转移
return dp[n]
```

### 5.2 二维DP

```python
dp = [[0] * n for _ in range(m)]
初始化第一行和第一列
for i in range(1, m):
    for j in range(1, n):
        dp[i][j] = 转移
return dp[m-1][n-1]
```

### 5.3 背包

```python
dp = [0] * (W + 1)
for i in range(n):
    for j in range(W, w[i]-1, -1):  # 0-1逆序
        dp[j] = max(dp[j], dp[j-w[i]] + v[i])
```

### 5.4 区间DP

```python
for length in range(2, n+1):
    for i in range(n - length + 1):
        j = i + length - 1
        for k in range(i, j):
            dp[i][j] = 最优(dp[i][k] + dp[k+1][j] + cost)
```

### 5.5 记忆化搜索

```python
@lru_cache(maxsize=None)
def dp(状态):
    if 边界: return 基础值
    return 最优(dp(新状态) for 所有选择)
```

## 6. 学习路线

```
阶段1：基础
  斐波那契 → 爬楼梯 → 打家劫舍 → 最大子数组和
  重点：理解状态定义和转移方程

阶段2：二维DP
  不同路径 → 最小路径和 → 最大正方形 → 编辑距离 → LCS
  重点：二维状态的定义和转移

阶段3：背包问题
  0-1背包 → 完全背包 → 多重背包 → 目标和 → 零钱兑换
  重点：遍历顺序和空间优化

阶段4：进阶DP
  区间DP → 树形DP → 状压DP → 数位DP
  重点：状态设计和优化技巧

阶段5：综合应用
  股票系列 → 字符串系列 → 博弈DP → 概率DP
  重点：灵活运用各种DP模型
```

## 8. 总结

动态规划的核心是**状态定义**和**转移方程**。掌握DP的关键：
1. 理解最优子结构和重叠子问题
2. 熟练写出各种DP模板
3. 能灵活选择实现方式（记忆化/表格）
4. 掌握常见优化技巧
5. 大量练习，形成直觉
