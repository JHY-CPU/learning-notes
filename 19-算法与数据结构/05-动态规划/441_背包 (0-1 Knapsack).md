# 06-0-1背包 (0-1 Knapsack)

## 核心概念

- **0-1 背包问题**：给定 n 个物品，每个物品有重量 w[i] 和价值 v[i]，背包容量为 W。每个物品只能选或不选，求在不超过容量的前提下总价值最大。
- **状态定义**：`dp[j]` 表示容量为 j 的背包能装的最大价值。
- **状态转移**：`dp[j] = max(dp[j], dp[j - w[i]] + v[i])`，必须**逆序遍历** j（从 W 到 w[i]），确保每个物品只选一次。
- **为什么逆序**：正序遍历时 `dp[j - w[i]]` 可能已经包含了物品 i（当前轮次更新过的值），等同于物品可以重复选。逆序保证 `dp[j - w[i]]` 是上一轮（未选物品 i）的值。

## 原理分析

**初始化**：`dp[0...W] = 0`

**遍历**：外层循环遍历每个物品 i（0 到 n-1），内层循环逆序遍历容量 j（W 到 w[i]）。

**时间复杂度**：$O(n \times W)$
**空间复杂度**：$O(W)$（优化后）

## 代码示例

```javascript
function knapsack(weights, values, W) {
  const n = weights.length;
  const dp = new Array(W + 1).fill(0);

  for (let i = 0; i < n; i++) {
    // 必须逆序！保证每个物品只选一次
    for (let j = W; j >= weights[i]; j--) {
      dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
    }
  }
  return dp[W];
}

// 示例
const w = [2, 3, 4, 5];
const v = [3, 4, 5, 6];
const W = 8;
console.log(knapsack(w, v, W)); // 10（选物品 1,2,3: 重量3+4+5=12超了；选0,1,2: 2+3+4=9超了；选0,2,3: 2+4+5=11超了；选1,2: 3+4=7, 价值9；选0,1,3: 2+3+5=10超了；选0,3: 2+5=7, 价值9；选2,3: 4+5=9超了；选0,1,2: 超了；最优: 选0,1: 2+3=5, 价值7；或选1,3: 3+5=8, 价值10）
```

## 完全背包 vs 0-1 背包

| 区别 | 0-1 背包 | 完全背包 |
|------|---------|---------|
| 每个物品 | 最多选一次 | 可选无限次 |
| 内层循环 | **逆序** W→w[i] | **正序** w[i]→W |
| 转移方程 | 相同 | 相同，但正序实现 |

## 应用与变体

- **多重背包**：每个物品有数量限制，可拆分为多个 0-1 背包或用二进制优化
- **分组背包**：每组选一个物品，组内枚举
- **二维费用背包**：增加一维限制（如重量+体积），dp 从一维变二维
