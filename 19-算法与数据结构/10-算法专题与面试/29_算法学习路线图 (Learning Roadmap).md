# 算法学习路线图 (Learning Roadmap)

## 一、学习阶段总览

```
第1阶段(2-3周) → 第2阶段(4-6周) → 第3阶段(2-3周) → 第4阶段(持续)
  数据结构基础      算法专题突破       模拟面试训练       持续提升
```

---

## 二、第一阶段：数据结构基础 (2-3周)

### 2.1 第一周：线性结构

- [ ] 数组 — 遍历、双指针、滑动窗口
- [ ] 链表 — 翻转、合并、快慢指针
- [ ] 栈 — 单调栈、表达式求值
- [ ] 队列 — BFS、优先队列
- [ ] 哈希表 — 两数之和、去重、计数

**每日任务：** 每种结构刷 3-5 道 Easy 题

### 2.2 第二周：树与图

- [ ] 二叉树 — 三种遍历、层序遍历
- [ ] BST — 插入、删除、验证
- [ ] 图 — BFS、DFS、连通分量
- [ ] 并查集 — 基本操作

**每日任务：** 每种结构刷 3-5 道 Medium 题

### 2.3 第三周：高级结构

- [ ] 堆 — TopK问题
- [ ] Trie — 前缀匹配
- [ ] 线段树 — 区间查询（了解即可）

## 三、第二阶段：算法专题 (4-6周)

### 3.1 第3-4周：核心算法

- [ ] 二分查找 — 标准模板、二分答案（10题）
- [ ] 动态规划 — 背包、LIS、LCS、区间DP（20题）
- [ ] 回溯 — 排列组合、N皇后（15题）

### 3.2 第5-6周：进阶算法

- [ ] 贪心 — 区间调度、跳跃游戏（10题）
- [ ] 图论 — 拓扑排序、最短路、并查集（10题）
- [ ] 字符串 — KMP、Trie、滑动窗口（10题）

### 3.3 第7-8周：补充专题

- [ ] 排序算法 — 快排、归并、堆排序
- [ ] 位运算 — 状态压缩、位技巧
- [ ] 数学 — 素数、组合、快速幂

## 六、学习方法

### 6.1 刷题四步法

```
1. 独立思考 15分钟
2. 看提示但不看代码
3. 看懂后自己写
4. 隔天重写一遍
```

### 6.2 复习策略

- **3-7-14 天法则：** 做完的题分别在第3、7、14天复习
- **错题本：** 记录做错的题目和原因
- **模板库：** 每类题型总结通用代码模板

### 6.3 避免的误区

- 不要只看不写 — 必须亲手编码
- 不要只做 Easy — Medium 才是面试主战场
- 不要追求数量 — 理解比数量更重要
- 不要忽略复杂度 — 面试必问
