# Stack Applications

### 栈的典型应用场景

栈在计算机科学中有广泛应用：括号匹配检验表达式合法性，表达式求值处理中缀/后缀转换，函数调用管理执行流程，撤销/重做操作实现用户交互。

### 关键特性

- **括号匹配**：遇到左括号入栈，右括号出栈匹配，最后栈为空则合法
- **表达式求值**：中缀转后缀用运算符栈，后缀求值用操作数栈
- **单调栈**：维护栈内元素单调性，高效解决"下一个更大元素"类问题

### 时间与空间复杂度

| 应用 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 括号匹配 | O(n) | O(n) |
| 中缀转后缀 | O(n) | O(n) |
| 后缀表达式求值 | O(n) | O(n) |
| 下一个更大元素 | O(n) | O(n) |

### 适用场景 vs 替代方案

- 括号匹配：栈是最佳选择，无法用其他结构高效替代
- 表达式求值：递归下降也是选择，但栈方法更直观
- DFS 遍历：可用递归（隐式栈）或显式栈，显式栈避免栈溢出

### 常见陷阱

- 括号匹配时忽略嵌套层级，导致 "({)}" 被误判为合法
- 表达式求值未处理运算符优先级，结果错误
- 忘记检查栈是否为空就执行 pop 操作

```
// 括号匹配
function isBalanced(s) {
  const stack = [];
  const map = {')': '(', '}': '{', ']': '['};
  for (const ch of s) {
    if ('({['.includes(ch)) stack.push(ch);
    else if (map[ch] !== stack.pop()) return false;
  }
  return stack.length === 0;
}
console.log(isBalanced("(){}[]")); // true
console.log(isBalanced("({)}"));   // false
```


### 实际应用

- **代码编辑器**：IDE 用栈匹配高亮括号，实时检测语法错误
- **计算器程序**：科学计算器通过栈处理运算符优先级和括号
- **路由导航**：浏览器前进/后退功能用两个栈实现
- **编译器**：语法分析阶段用栈构建抽象语法树（AST）
