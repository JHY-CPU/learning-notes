# Monotonic Stack

### 什么是单调栈

单调栈是一种特殊的栈结构，栈内元素始终保持单调递增或单调递减的顺序。主要用于高效解决"下一个更大/更小元素"类问题，将暴力 O(n²) 优化到 O(n)。

### 关键特性

- **单调递增栈**：从栈底到栈顶递增，用于找下一个更小元素
- **单调递减栈**：从栈底到栈顶递减，用于找下一个更大元素
- **每个元素最多入栈出栈各一次**，因此总时间复杂度为 O(n)

### 时间与空间复杂度

| 操作 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 构建单调栈 | O(n) | O(n) |
| 查找下一个更大元素 | O(n) | O(n) |
| 查找下一个更小元素 | O(n) | O(n) |

### 适用场景 vs 替代方案

- **适用**：下一个更大/更小元素、柱状图最大矩形、接雨水
- **不适用**：需要全局排序的场景（直接排序更简单）
- **替代**：暴力双循环 O(n²) 也能解，但效率低

### 常见陷阱

- 栈中存索引还是值容易混淆，通常存索引更灵活
- 遍历方向（正序/倒序）决定结果含义，需要根据题意选择
- 边界条件：所有元素都没有更大/更小元素时，结果为 -1

```
// 下一个更大元素
function nextGreater(arr) {
  const res = new Array(arr.length).fill(-1);
  const stack = [];
  for (let i = 0; i < arr.length; i++) {
    while (stack.length && arr[stack[stack.length-1]] < arr[i]) {
      res[stack.pop()] = arr[i];
    }
    stack.push(i);
  }
  return res;
}
console.log(nextGreater([2,1,3,4,2])); // [3,3,4,-1,-1]
```

- **温度预测**：LeetCode 739 每日温度问题，找距离下一个更高温度的天数
- **股票分析**：找到价格上升趋势中的买入卖出时机
- **编译器优化**：计算表达式树中节点的影响范围
- **图像处理**：直方图均衡化中的峰值检测
