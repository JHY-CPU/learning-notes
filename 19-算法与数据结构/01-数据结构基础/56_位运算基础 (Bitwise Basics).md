# 位运算基础 (Bitwise Basics)

## 一、六种基本运算

### 1.1 运算符详解

| 运算符 | 名称 | 规则 | 示例 |
|--------|------|------|------|
| `&` | 按位与 | 两位都为1则1 | `5 & 3 = 1` (101 & 011 = 001) |
| `\|` | 按位或 | 至少一位为1则1 | `5 \| 3 = 7` (101 \| 011 = 111) |
| `^` | 异或 | 两位不同则1 | `5 ^ 3 = 6` (101 ^ 011 = 110) |
| `~` | 取反 | 0变1，1变0 | `~5 = -6` |
| `<<` | 左移 | 左移n位，相当于乘2^n | `5 << 1 = 10` |
| `>>` | 右移 | 右移n位，相当于除2^n | `5 >> 1 = 2` |

### 1.2 Python与C++的差异

```python
# Python 整数无限精度，右移是算术右移
print(-5 >> 1)  # -3（符号位扩展）

# C++ 中，负数右移行为取决于实现（通常算术右移）
# 左移负数是未定义行为
```

## 二、常用位运算技巧

### 2.1 基本操作

```python
# 判断奇偶
n & 1          # 1为奇数，0为偶数

# 乘以/除以2的幂
n << k         # n * 2^k
n >> k         # n / 2^k（整数除法）

# 交换两数（不需要额外变量）
a ^= b
b ^= a
a ^= b

# 取绝对值（适用于32位整数）
mask = n >> 31
abs_n = (n ^ mask) - mask
```

### 2.2 高级操作

```python
# 取最低位的1 — lowbit操作
lowbit = n & (-n)

# 去掉最低位的1
n & (n - 1)

# 将最低位的0变成1
n | (n + 1)

# 判断是否为2的幂
n > 0 and (n & (n - 1)) == 0

# 将最右边的1变成0
n & (n - 1)
```

### 2.3 集合操作

```python
# 用整数表示集合（最多32/64个元素）
S = 0b1011  # 表示集合 {0, 1, 3}

# 检查元素i是否在集合中
(S >> i) & 1
S & (1 << i)

# 添加元素i
S | (1 << i)

# 删除元素i
S & ~(1 << i)

# 切换元素i
S ^ (1 << i)

# 枚举所有子集
sub = S
while sub:
    # 处理 sub
    sub = (sub - 1) & S
```

## 三、代码示例

### 3.1 C++ 实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a = 0b1100;  // 12
    int b = 0b1010;  // 10

    cout << "AND: " << (a & b) << endl;    // 8 (0b1000)
    cout << "OR: " << (a | b) << endl;     // 14 (0b1110)
    cout << "XOR: " << (a ^ b) << endl;    // 6 (0b0110)
    cout << "NOT a: " << (~a) << endl;     // -13
    cout << "Left shift: " << (a << 1) << endl;  // 24
    cout << "Right shift: " << (a >> 1) << endl; // 6

    // 判断第i位是否为1
    int n = 13;  // 1101
    for (int i = 0; i < 4; i++) {
        cout << "Bit " << i << ": " << ((n >> i) & 1) << endl;
    }

    // 统计1的个数
    int count = __builtin_popcount(n);  // GCC内置

    // lowbit
    int lowbit = n & (-n);
    cout << "Lowest bit of " << n << ": " << lowbit << endl; // 1

    return 0;
}
```

### 3.2 Python 实现

```python
a = 0b1100  # 12
b = 0b1010  # 10

print(f"AND: {a & b}")       # 8
print(f"OR: {a | b}")        # 14
print(f"XOR: {a ^ b}")       # 6
print(f"NOT a: {~a}")        # -13
print(f"Binary: {bin(a)}")   # '0b1100'

# 统计1的个数
def count_bits(n):
    count = 0
    while n:
        n &= n - 1  # 去掉最低位的1
        count += 1
    return count

print(count_bits(0b1101))  # 3
```

## 四、复杂度分析

| 操作 | 时间复杂度 |
|------|-----------|
| 基本位运算 | $O(1)$ |
| 统计1的个数 | $O(k)$ k为1的个数 |
| 枚举子集 | $O(2^k)$ k为集合大小 |
| lowbit | $O(1)$ |
