# Monotonic Stack Advanced

### 单调栈的进阶应用

单调栈在进阶题目中常用于解决涉及"区间最值"和"面积计算"的问题。核心思想是：对每个元素，快速找到它作为边界时能影响的最远范围。

### 关键特性

- **接雨水**：每个位置能接的水量 = min(左边最高, 右边最高) - 当前高度
- **柱状图最大矩形**：以每个柱子为高，找左右边界确定宽度
- **双指针优化**：接雨水可用左右指针替代单调栈，空间降到 O(1)

### 时间与空间复杂度

| 问题 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 接雨水（双指针） | O(n) | O(1) |
| 接雨水（单调栈） | O(n) | O(n) |
| 柱状图最大矩形 | O(n) | O(n) |
| 去除重复字母 | O(n) | O(26) |

### 适用场景 vs 替代方案

- **接雨水**：双指针法更省空间，单调栈思路更直观
- **最大矩形**：单调栈是标准解法，暴力 O(n²) 不可接受
- **环形数组**：通过遍历两遍数组（或取模）配合单调栈解决

### 常见陷阱

- 接雨水时左右边界条件容易遗漏（最左最右无法蓄水）
- 柱状图最大矩形需考虑栈为空时的左边界处理
- 环形问题中索引取模避免越界

```
// 接雨水
function trap(height) {
  let l = 0, r = height.length - 1;
  let lMax = 0, rMax = 0, ans = 0;
  while (l < r) {
    if (height[l] < height[r]) {
      height[l] >= lMax ? lMax = height[l] : ans += lMax - height[l];
      l++;
    } else {
      height[r] >= rMax ? rMax = height[r] : ans += rMax - height[r];
      r--;
    }
  }
  return ans;
}
console.log(trap([0,1,0,2,1,0,1,3,2,1,2,1])); // 6
```


### 实际应用

- **雨水收集系统**：计算建筑物之间的积水容量
- **数据可视化**：柱状图中寻找最大可用矩形区域
- **资源调度**：在时间轴上找到最大的连续可用区间
- **图像处理**：计算直方图中最大矩形区域
