## String Basics


```javascript
字符串是字符序列，在 JavaScript 中不可变。常见操作包括拼接、截取、查找、替换。```


```
const s = "Hello, 算法世界!";
console.log(s.length);
console.log(s.charAt(0));
console.log(s.indexOf("算法"));
console.log(s.slice(7, 9));
console.log(s.includes("Hello"));
console.log(s.split(","));
console.log(s.replace("算法", "Algorithm"));```
