## Bitwise Tricks


```javascript
位运算技巧包括判断奇偶、交换变量、清除最低位1、2的幂判断等。```


```
// 常用位运算技巧
console.log(`奇数? 7&1: ${7&1}`);  // 1
console.log(`偶数? 8&1: ${8&1}`);  // 0
console.log(`2的幂? 16&(16-1): ${16&(16-1)}`); // 0
// 清除最低位的1
const x = 0b1011000;
console.log(`清除最低位1: ${x & (x-1)}`); // 1010000
// 取最低位1
console.log(`取最低位1: ${x & (-x)}`); // 1000
// 交换变量
let p=3, q=5; p^=q; q^=p; p^=q;
console.log(`交换后: p=${p}, q=${q}`); // p=5, q=3```
