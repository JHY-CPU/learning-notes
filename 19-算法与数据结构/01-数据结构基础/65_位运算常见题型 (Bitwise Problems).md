# Bitwise Problems

### 位运算常见面试题型

位运算在面试中常用于考察对二进制的理解和巧妙的算法设计。核心思路是利用异或消除重复、利用与运算提取特征。

### 关键特性

- **只出现一次的数字**：全部异或，相同的数异或为 0
- **2 的幂判断**：n > 0 && (n & (n-1)) === 0
- **比特位计数**：dp[i] = dp[i>>1] + (i&1)，利用已计算结果
- **汉明距离**：异或后统计 1 的个数

### 时间与空间复杂度

| 问题 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 只出现一次的数字 | O(n) | O(1) |
| 比特位计数 | O(n) | O(n) |
| 2 的幂 | O(1) | O(1) |
| 汉明距离 | O(1) | O(1) |
| 颠倒二进制位 | O(1) | O(1) |

### 适用场景 vs 替代方案

- **空间 O(1) 要求**：位运算是唯一选择（如只出现一次的数字）
- **位计数**：Brian Kernighan 算法 O(k) vs 查表法 O(1)
- **对称问题**：异或天然满足交换律和结合律
- **替代**：可读性优先时用 Set 或 Map

### 常见陷阱

- 忘记处理 0 和负数的特殊情况
- JavaScript 中 1 << 31 会得到负数（32 位有符号）
- 异或的恒等值是 0，初始化应为 0 而非其他值

```
// 只出现一次的数字
function singleNumber(nums) {
  return nums.reduce((a,b) => a ^ b, 0);
}
// 比特位计数
function countBits(n) {
  const res = new Array(n+1).fill(0);
  for (let i = 1; i <= n; i++) res[i] = res[i>>1] + (i&1);
  return res;
}
// 2的幂
function isPowerOfTwo(n) { return n > 0 && (n & (n-1)) === 0; }
console.log(singleNumber([4,1,2,1,2])); // 4
console.log(countBits(5)); // [0,1,1,2,1,2]
```


### 实际应用

- **错误检测**：奇偶校验位利用异或检测传输错误
- **数据去重**：日志中找出唯一出现的错误码
- **性能监控**：汉明距离衡量两个状态的差异程度
- **图像处理**：快速计算两个像素值的差异
