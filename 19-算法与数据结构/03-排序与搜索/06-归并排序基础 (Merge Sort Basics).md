## 归并排序基础 (Merge Sort Basics)

  归并排序是建立在归并操作上的一种有效排序算法。它采用**分治策略**（Divide and Conquer），将已有序的子序列合并，得到完全有序的序列。


>
    **核心思想：**分而治之 — 先将数组不断二分，直到每个子序列只有一个元素（天然有序），再两两合并成有序序列。


  ## 分治过程

  [38, 27, 43, 3, 9, 82, 10]
        /              \
[38, 27, 43, 3]    [9, 82, 10]
   /       \          /      \
[38, 27]  [43, 3]  [9, 82]  [10]
 /    \    /    \   /    \
[38] [27] [43] [3] [9] [82]
  \   /    \   /     \   /
[27, 38]  [3, 43]  [9, 82]
   \       /          |
  [3, 27, 38, 43]    [9, 10, 82]
        \              /
     [3, 9, 10, 27, 38, 43, 82]

  ## 归并操作详解

  归并操作将两个有序数组合并为一个有序数组：


    - 同时遍历两个有序数组，比较当前元素

    - 将较小的元素放入结果数组

    - 指针后移，继续比较

    - 将剩余元素全部追加到结果数组
