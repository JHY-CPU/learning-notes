## 排序算法对比 (Sorting Comparison)

  本页面提供全面的排序算法对比，包含理论分析和实际性能基准测试。

  ## 理论对比













  | 算法 | 平均 | 最坏 | 最好 | 空间 | 稳定 | 原地 |
| --- | --- | --- | --- | --- | --- | --- |
| 冒泡排序 | O(n^2) | O(n^2) | O(n) | O(1) | 是 | 是 |

| 插入排序 | O(n^2) | O(n^2) | O(n) | O(1) | 是 | 是 |
| 希尔排序 | O(n^1.3) | O(n^2) | O(n) | O(1) | 否 | 是 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 是 | 否 |
| 快速排序 | O(n log n) | O(n^2) | O(n log n) | O(log n) | 否 | 是 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 否 | 是 |
| 计数排序 | O(n+k) | O(n+k) | O(n+k) | O(k) | 是 | 否 |
| 桶排序 | O(n+k) | O(n^2) | O(n) | O(n) | 是 | 否 |
| 基数排序 | O(d(n+k)) | O(d(n+k)) | O(d(n+k)) | O(n+k) | 是 | 否 |
