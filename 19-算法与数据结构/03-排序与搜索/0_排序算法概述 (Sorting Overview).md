## 排序算法概述

  排序算法是计算机科学中最基础的算法之一，用于将一组数据按特定顺序重新排列。排序算法种类繁多，各有优劣。

  ## 分类标准


    - **比较排序 vs 非比较排序**：比较排序通过元素间比较决定顺序，非比较排序利用元素本身特性（如计数排序、基数排序）。

    - **稳定排序 vs 不稳定排序**：稳定排序保持相等元素的原始相对顺序。

    - **内部排序 vs 外部排序**：内部排序数据可全部加载到内存，外部排序处理内存放不下的大数据。

    - **原地排序 vs 非原地排序**：原地排序只需 O(1) 额外空间。



  ## 常用排序算法对比













  | 算法 | 平均时间复杂度 | 最坏时间复杂度 | 空间复杂度 | 稳定性 |
| --- | --- | --- | --- | --- |
| 冒泡排序 | O(n^2) | O(n^2) | O(1) | 稳定 |
| 选择排序 | O(n^2) | O(n^2) | O(1) | 不稳定 |
| 插入排序 | O(n^2) | O(n^2) | O(1) | 稳定 |
| 希尔排序 | O(n log n)~O(n^2) | O(n^2) | O(1) | 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) | O(n^2) | O(log n) | 不稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(1) | 不稳定 |
| 计数排序 | O(n+k) | O(n+k) | O(k) | 稳定 |
| 桶排序 | O(n+k) | O(n^2) | O(n) | 稳定 |
| 基数排序 | O(d*(n+k)) | O(d*(n+k)) | O(n+k) | 稳定 |

  ## 如何选择排序算法


    - **小规模数据（n<100）**：插入排序，简单高效。

    - **中等规模数据**：快速排序通常最快。

    - **大规模数据**：归并排序或堆排序。

    - **数据范围有限**：计数排序、基数排序可达线性时间。

    - **需要稳定性**：归并排序、插入排序。

    - **内存受限**：堆排序、原地快速排序。
