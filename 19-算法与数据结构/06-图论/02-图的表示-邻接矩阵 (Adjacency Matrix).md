## 邻接矩阵

  使用 N x N 的二维数组表示图。matrix[i][j] = 1 表示存在从 i 到 j 的边（无权图），或存储权重值。


```javascript
// 无向无权图（4个顶点）
const matrix = [
  [0, 1, 1, 0],  // A
  [1, 0, 1, 0],  // B
  [1, 1, 0, 1],  // C
  [0, 0, 1, 0]   // D
];```

  ## 优缺点


    - **优点：**判断边是否存在 O(1)，添加/删除边 O(1)

    - **优点：**实现简单直观

    - **缺点：**空间复杂度 O(V²)，不适合稀疏图

    - **缺点：**遍历所有边需要 O(V²)
