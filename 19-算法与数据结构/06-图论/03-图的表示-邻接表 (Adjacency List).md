## 邻接表

  为每个顶点维护一个邻接顶点列表（或数组）。适合稀疏图，空间复杂度 O(V+E)。


```javascript
// 无向图邻接表
const graph = {
  A: ['B', 'C', 'D'],
  B: ['A', 'C'],
  C: ['A', 'B', 'D'],
  D: ['A', 'C']
};```

  ## 优缺点


    - **优点：**空间省，O(V+E)，适合稀疏图

    - **优点：**遍历某顶点的所有邻边快速

    - **缺点：**判断两顶点是否相邻需要 O(degree)

    - **缺点：**删除边操作较慢
