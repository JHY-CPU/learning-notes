# 28-算法的正确性证明 (Correctness Proof)

算法正确性证明确保算法在所有合法输入下都能产生正确结果。

## 三种主要方法

### 1. 循环不变式

```javascript
// 插入排序的循环不变式：
// 每次迭代开始时，arr[0..i-1] 已经有序
function insertionSort(arr) {
  // 初始化：arr[0..0] 有序
  for (let i = 1; i < arr.length; i++) {
    const key = arr[i];
    let j = i - 1;
    // 保持：arr[0..i] 有序
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
  // 终止：arr[0..n-1] 有序，即整个数组有序
  return arr;
}
```

### 2. 数学归纳法

```javascript
// 证明归并排序正确性
// 基础：长度为 1 的数组已排序
// 归纳：假设 arr[l..m] 和 arr[m+1..r] 都已排序
//       合并后 arr[l..r] 有序（因为合并过程比较每对元素）
```

### 3. 交换论证法（贪心）

```javascript
// 证明区间调度贪心正确性
// 假设最优解 OPT 不等于贪心解 GREEDY
// 设第一个不同位置：OPT 选了区间 A，GREEDY 选了区间 B
// 由于 B 结束时间 <= A 结束时间（贪心选最早结束）
// 将 OPT 中的 A 替换为 B，不会减少可选区间数
// 因此存在一个最优解与 GREEDY 一致
```

## 证明步骤

| 方法 | 步骤 |
|------|------|
| 循环不变式 | 初始化 -> 保持 -> 终止 |
| 数学归纳法 | 基础情况 -> 归纳假设 -> 归纳步骤 |
| 交换论证 | 假设最优解不同 -> 逐步交换 -> 证明不差 |

## 各算法的证明方法

| 算法 | 证明方法 |
|------|---------|
| 排序算法 | 循环不变式 |
| 贪心算法 | 交换论证法 |
| 分治算法 | 数学归纳法 |
| 动态规划 | 最优子结构证明 |

## 常见陷阱

1. **只证最好情况**：必须证明所有输入都正确
2. **忽略边界**：空输入、单元素的处理
3. **循环不变式太弱**：不能推导出终止时的正确性
4. **贪心不证**：贪心算法必须证明贪心选择性质
