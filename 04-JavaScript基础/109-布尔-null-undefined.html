布尔值 / null / undefined

三个表示"真/假"和"空"的基础类型。

boolean 布尔值

只有两个值：true（真）和 false（假）

let isReady = true; let isDone = false; // 通常由比较产生 let isGreater = 5 > 3; // true let isEqual = 1 === 2; // false

逻辑运算结果

console.log(true && false); // false（AND） console.log(true || false); // true（OR） console.log(!true); // false（NOT） console.log(!!'hello'); // true（转boolean）

undefined

表示变量声明了但没有赋值，由 JavaScript 引擎自动产生。

let x; console.log(x); // undefined // 访问对象不存在的属性 let obj = {}; console.log(obj.foo); // undefined // 函数没有返回值时 function doNothing() {} console.log(doNothing()); // undefined // 数组越界 let arr = [1, 2, 3]; console.log(arr[10]); // undefined

null

表示"空值"或"没有对象"，需要手动赋值。

let empty = null; // 主动设为"空" let user = null; // 表示没有用户数据 // 常见用法：重置变量 let data = fetchData(); // 获取数据 data = null; // 清空数据

实际开发建议

// null 和 undefined 都可以表示"没有值" // 约定：主动设置"空"用 null，不要手动赋 undefined // ✅ 推荐 let user = null; // 明确表示"没有用户" let data = null; // 明确表示"没有数据" // ❌ 不推荐 let user = undefined; // 应该用 null 表示"主动为空" // ✅ 检查是否存在 if (user === null) { /* 处理空值 */ } if (user == null) { /* 同时匹配 null 和 undefined */ }
💡 不要手动把变量赋值为 undefined，那是引擎的事。需要用"空值"时用 null。
