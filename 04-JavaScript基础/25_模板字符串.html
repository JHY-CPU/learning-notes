模板字符串（Template Literals）

使用反引号 `` 定义，支持插值、多行、表达式。

基本用法

// 普通字符串（可以包含双引号、单引号） let str = `这是一个模板字符串`; // 可以写单引号/双引号而不用转义 let withQuotes = `他说"你好"我说'嗨'`; // ✅

字符串插值 ${}

let name = 'Alice'; let age = 25; // 旧方式：拼接 let oldWay = '我的名字是 ' + name + '，年龄 ' + age; // 模板字符串 let newWay = `我的名字是 ${name}，年龄 ${age}`; // "我的名字是 Alice，年龄 25" // 可以放表达式 let info = `${name} 明年 ${age + 1} 岁`; // "Alice 明年 26 岁" // 可以放函数调用 let upper = `${name.toUpperCase()}`; // "ALICE"
💡 ${} 里可以放任何 JS 表达式：变量、运算、函数调用、三元运算符等。

多行字符串

// 旧方式：用 \n let old = '第一行\n第二行\n第三行'; // 模板字符串：直接换行 let multi = `第一行 第二行 第三行`; // 常用于生成HTML let html = `

${title}

${content}

`;

表达式在 ${} 中的应用

let price = 100; let count = 3; let isMember = true; // 运算 let total = `总价: ${price * count} 元`; // "总价: 300 元" // 三元运算符 let discount = `会员: ${isMember ? '享受折扣' : '无折扣'}`; // 方法调用 let items = ['苹果', '香蕉', '橘子']; let list = `商品: ${items.join(', ')}`; // 嵌套模板 let cssClass = `btn ${isActive ? 'btn-active' : ''}`;

标签模板（Tagged Templates）

// 高级用法：用函数处理模板字符串 function highlight(strings, ...values) { return strings.reduce((result, str, i) => { return result + str + (values[i] ? `${values[i]}` : ''); }, ''); } let name = 'Alice'; let score = 95; let result = highlight`学生 ${name} 考了 ${score} 分`; // "学生 Alice 考了 95 分"
标签模板可以自定义字符串的处理方式，常用于国际化、转义等。
