Set — 集合

存储唯一值的有序集合，自动去重。

创建与基本操作

// 创建 const set = new Set(); const set2 = new Set([1, 2, 3, 3, 2]); // [1, 2, 3]（去重） const set3 = new Set('hello'); // {'h', 'e', 'l', 'o'} // 添加 set.add(1); set.add(2); set.add(2); // 不会重复添加 // 删除 set.delete(1); // 检查存在 set.has(2); // true set.has(1); // false（删除了） // 清空 set.clear(); // 大小 set.size; // 2

遍历 Set

const set = new Set(['apple', 'banana', 'cherry']); // for...of（推荐） for (let item of set) { console.log(item); } // forEach set.forEach((value, key) => { console.log(value); // key === value（Set没有键） }); // 转数组 const arr = [...set]; const arr2 = Array.from(set);

主要用途：去重

// 数组去重 const nums = [1, 2, 2, 3, 3, 3, 4]; const unique = [...new Set(nums)]; // [1, 2, 3, 4] // 字符串去重 const str = 'hello world'; const chars = [...new Set(str)].join(''); // 'helo wrd' // 合并去重 const arr1 = [1, 2, 3]; const arr2 = [2, 3, 4]; const merged = [...new Set([...arr1, ...arr2])]; // [1, 2, 3, 4]

Set 运算

const setA = new Set([1, 2, 3, 4]); const setB = new Set([3, 4, 5, 6]); // 交集 const intersection = new Set( [...setA].filter(x => setB.has(x)) ); // {3, 4} // 差集（A有B没有） const difference = new Set( [...setA].filter(x => !setB.has(x)) ); // {1, 2} // 并集 const union = new Set([...setA, ...setB]); // {1, 2, 3, 4, 5, 6}
