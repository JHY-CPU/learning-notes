展开运算符（Spread Operator）

用 `...` 将数组/对象"展开"。

数组展开

const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; // 合并数组 const merged = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6] // 在中间插入 const withMiddle = [...arr1, 99, ...arr2]; // [1, 2, 3, 99, 4, 5, 6] // 复制数组（浅拷贝） const copy = [...arr1];

对象展开

const obj1 = { a: 1, b: 2 }; const obj2 = { c: 3, d: 4 }; // 合并对象 const merged = { ...obj1, ...obj2 }; // { a: 1, b: 2, c: 3, d: 4 } // 覆盖属性 const withOverride = { ...obj1, b: 99, c: 100 }; // { a: 1, b: 99, c: 100 } // 复制对象（浅拷贝） const copy = { ...obj1 }; // 添加新属性 const withExtra = { ...obj1, extra: 'new' };
💡 展开对象的顺序很重要！后面的属性会覆盖前面的同名属性。

实际应用

// 1. 函数参数展开 const nums = [3, 1, 4, 1, 5]; Math.max(...nums); // 5 // 2. 字符串转数组 const chars = [...'hello']; // ['h', 'e', 'l', 'l', 'o'] // 3. Set/Map 转数组 const unique = [...new Set([1,2,2,3])]; // [1, 2, 3] // 4. 不可变更新 const todos = [ { id: 1, text: '学习JS', done: false }, { id: 2, text: '学习React', done: false } ]; // 添加 const added = [...todos, { id: 3, text: '学习Vue', done: false }]; // 更新 const updated = todos.map(t => t.id === 1 ? { ...t, done: true } : t ); // 删除 const deleted = todos.filter(t => t.id !== 2);

展开 vs rest

// 展开：在赋值/传参时展开数组/对象 const arr = [1, 2, 3]; console.log(...arr); // 1 2 3（展开） // Rest：在解构时收集剩余项 const [first, ...rest] = arr; // first=1, rest=[2,3]（收集） // 函数参数 function sum(...nums) { // rest（收集参数） return nums.reduce((a,c)=>a+c, 0); } const nums = [1, 2, 3]; sum(...nums); // spread（展开数组）
