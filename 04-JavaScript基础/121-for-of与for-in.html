for...of 和 for...in

两种遍历语法，用途不同。

for...of — 遍历值

遍历可迭代对象（数组、字符串、Set、Map等）的值。

// 遍历数组 let fruits = ['apple', 'banana', 'cherry']; for (let fruit of fruits) { console.log(fruit); } // 'apple', 'banana', 'cherry' // 遍历字符串 for (let char of 'Hello') { console.log(char); } // 'H', 'e', 'l', 'l', 'o'
💡 for...of 用于获取集合的值，比 for 循环更简洁。

for...in — 遍历键

遍历对象的可枚举属性名（键）。

// 遍历对象 let user = { name: 'Alice', age: 25, role: 'admin' }; for (let key in user) { console.log(key, ':', user[key]); } // 'name : Alice', 'age : 25', 'role : admin' // 遍历数组（不推荐！） let arr = ['a', 'b', 'c']; for (let index in arr) { console.log(index); // '0', '1', '2'（字符串！不是数字） }

for...in 的注意事项

// 1. 遍历的顺序不保证（虽然现在几乎所有浏览器都按创建顺序） // 2. 会遍历继承的可枚举属性 // 3. 不遍历 Symbol 键 let obj = { b: 2, a: 1, c: 3 }; for (let key in obj) { console.log(key); // 'b', 'a', 'c'（按创建顺序） } // 只遍历自身的属性（排除继承的） for (let key in obj) { if (obj.hasOwnProperty(key)) { console.log(key, '是自己的属性'); } }

对比：遍历数组的四种方式

let arr = ['a', 'b', 'c']; // 1. for 循环（最灵活，需要索引） for (let i = 0; i < arr.length; i++) { console.log(i, arr[i]); // 0 'a', 1 'b', 2 'c' } // 2. for...of（直接取值，不需要索引） for (let val of arr) { console.log(val); // 'a', 'b', 'c' } // 3. for...in（不推荐数组用） for (let idx in arr) { console.log(idx, arr[idx]); // '0' 'a', '1' 'b', '2' 'c' } // 4. forEach（数组方法） arr.forEach((val, idx) => console.log(idx, val));
数组遍历推荐：需要索引用 forEach，只需要值用 for...of。

for...of 的更多用法

// Set let set = new Set([1, 2, 3]); for (let val of set) { console.log(val); // 1, 2, 3 } // Map let map = new Map([['a', 1], ['b', 2]]); for (let [key, val] of map) { console.log(key, val); // 'a' 1, 'b' 2 } // 字符串 for (let char of '你好') { console.log(char); // '你', '好' } // arguments（函数内） function showArgs() { for (let arg of arguments) { console.log(arg); } }
