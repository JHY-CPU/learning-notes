Math 对象

提供数学常量和函数，所有属性和方法都是静态的。

常量

Math.E // 2.718281828459045（自然常数） Math.PI // 3.141592653589793（圆周率） Math.LN2 // 0.6931471805599453（2的自然对数） Math.SQRT2 // 1.4142135623730951（2的平方根）

取整方法

Math.floor(3.7); // 3（向下取整） Math.ceil(3.2); // 4（向上取整） Math.round(3.5); // 4（四舍五入） Math.trunc(3.7); // 3（截断小数部分，ES6） Math.trunc(-3.7); // -3（截断，而 floor(-3.7)=-4） // 对比： // floor(3.7)=3, floor(-3.7)=-4 // trunc(3.7)=3, trunc(-3.7)=-3

随机数 Math.random()

Math.random(); // 0~1之间的随机小数（含0不含1） // 随机整数 [0, max) Math.floor(Math.random() * 10); // 0~9 // 随机整数 [1, max] Math.floor(Math.random() * 10) + 1; // 1~10 // 随机整数 [min, max] function getRandomInt(min, max) { return Math.floor(Math.random() * (max - min + 1)) + min; }
💡 Math.random() 是伪随机，不适合安全场景（用 crypto.getRandomValues）。

最大值/最小值/绝对值

Math.max(10, 5, 20, 8); // 20 Math.min(10, 5, 20, 8); // 5 Math.abs(-5); // 5 Math.abs(5); // 5 // 用于数组 Math.max(...[10, 5, 20]); // 20 Math.min(...[10, 5, 20]); // 5

幂、平方根、其他

Math.pow(2, 10); // 1024（2的10次方）← 可用 2**10 Math.sqrt(16); // 4（平方根） Math.cbrt(27); // 3（立方根，ES6） Math.hypot(3, 4); // 5（勾股定理：√(3²+4²)） Math.log(10); // 2.302...（自然对数） Math.log10(100); // 2（以10为底的对数） Math.log2(8); // 3（以2为底的对数） Math.sin(Math.PI / 2); // 1（正弦） Math.cos(0); // 1（余弦） Math.tan(Math.PI / 4); // 1（正切）
