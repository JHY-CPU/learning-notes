类型转换 — 转数字

一共4种方式将其他类型转为 number。

方式一：Number() 函数

Number('42'); // 42 Number('3.14'); // 3.14 Number(''); // 0 Number(' '); // 0 Number(' 123 '); // 123（自动去空格） Number('123abc'); // NaN（不能完全转换） Number('abc'); // NaN Number(true); // 1 Number(false); // 0 Number(null); // 0 Number(undefined); // NaN Number([]); // 0 Number([1]); // 1 Number([1, 2]); // NaN
Number() 最严格：整个字符串必须是合法的数字格式

方式二：parseInt() / parseFloat()

parseInt('42px'); // 42（可以含单位） parseInt(' 100 '); // 100 parseInt('3.14'); // 3（只取整数部分） parseInt('abc'); // NaN parseInt('100px200'); // 100（从开始读到不能读为止） parseFloat('3.14em'); // 3.14 parseFloat('3.14.15'); // 3.14（只认第一个小数点） parseFloat('100px'); // 100 // parseInt 第二个参数：进制 parseInt('1010', 2); // 10（二进制 → 十进制） parseInt('FF', 16); // 255（十六进制 → 十进制） parseInt('77', 8); // 63（八进制 → 十进制）

方式三：一元 + 运算符

+'42'; // 42 +'3.14'; // 3.14 +''; // 0 +' '; // 0 +'123abc'; // NaN +true; // 1 +false; // 0 +null; // 0 +undefined; // NaN
+ 和 Number() 行为一致，是更简洁的写法

方式四：隐式转换（运算符触发）

// 减法/乘法/除法会触发转数字 '5' - 3; // 2 '6' * '2'; // 12 '10' / '2'; // 5 // 但加法会优先做字符串拼接 '5' + 3; // '53'（不是8！） 5 + '3'; // '53' // 技巧：先减0或乘1来转数字 '5' - 0; // 5 '3.14' * 1; // 3.14
⚠️ + 运算符的一个陷阱：如果一边是字符串，就变成拼接！
