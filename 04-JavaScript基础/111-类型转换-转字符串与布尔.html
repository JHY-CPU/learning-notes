类型转换 — 转字符串 / 转布尔

其他类型转字符串和布尔值。

转字符串 — String()

String(42); // "42" String(3.14); // "3.14" String(true); // "true" String(false); // "false" String(null); // "null" String(undefined); // "undefined" String([1, 2, 3]); // "1,2,3" String({name: 'JS'}); // "[object Object]"

转字符串 — toString() 方法

(42).toString(); // "42" (3.14).toString(); // "3.14" true.toString(); // "true" // 数字转不同进制 (255).toString(16); // "ff"（十六进制） (10).toString(2); // "1010"（二进制） // ⚠️ null 和 undefined 没有 toString() // null.toString(); // ❌ 报错！
null 和 undefined 只能用 String()，不能用 .toString()

转字符串 — 空字符串拼接

42 + ''; // "42" ← 最常用！ 3.14 + ''; // "3.14" true + ''; // "true" null + ''; // "null" undefined + ''; // "undefined"
💡 '' + value 是最简洁的转字符串方式

转布尔值 — Boolean()

Boolean(1); // true Boolean(0); // false Boolean('hello'); // true Boolean(''); // false Boolean(null); // false Boolean(undefined); // false Boolean(NaN); // false Boolean([]); // true（空数组是 true!） Boolean({}); // true（空对象是 true!）

转布尔值 — !! 双感叹号

!!1; // true !!0; // false !!'hello'; // true !!''; // false !!null; // false !!undefined; // false !!NaN; // false
💡 !!value 是最简洁的转布尔方式

Truthy 和 Falsy 值

Falsy（假值） — 转布尔时为 false 的 6 个值：

false // 布尔false 0 // 数字零 -0 // 负零 '' // 空字符串 null // 空 undefined // 未定义 NaN // 非数字

Truthy（真值） — 除了上面 7 个，其他都是 truthy：

true 'hello' // 非空字符串 42 // 非零数字 [] {} // 空对象也是 truthy！ 'false' // 字符串"false"也是 truthy！ '0' // 字符串"0"也是 truthy！ Infinity
⚠️ 'false' 是 truthy！'0' 是 truthy！只有空字符串 '' 是 falsy。
