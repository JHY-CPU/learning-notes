数组排序 sort() 与反转 reverse()

修改原数组，返回排序后的数组。

reverse() — 反转数组

let arr = [1, 2, 3, 4, 5]; arr.reverse(); // [5, 4, 3, 2, 1] // ⚠️ 修改原数组！

sort() — 默认排序（按字符串）

let arr = ['banana', 'apple', 'cherry', 'date']; arr.sort(); // ['apple', 'banana', 'cherry', 'date'] // ⚠️ 默认按字符串排序，数字会有问题！ [1, 10, 2, 20].sort(); // [1, 10, 2, 20] ← 不是按数字大小！ // 因为转成字符串比较："1" < "10" < "2" < "20"
⚠️ sort() 默认按字符串 Unicode 编码比较！数字排序需要传比较函数。

sort() — 自定义比较函数

// 比较函数：接收两个参数 a, b // 返回负数 → a 排在 b 前 // 返回正数 → b 排在 a 前 // 返回 0 → 位置不变 // 升序 [1, 10, 2, 20].sort((a, b) => a - b); // [1, 2, 10, 20] // 降序 [1, 10, 2, 20].sort((a, b) => b - a); // [20, 10, 2, 1] // 对象数组排序 let users = [ { name: 'Alice', age: 30 }, { name: 'Bob', age: 25 }, { name: 'Charlie', age: 35 } ]; users.sort((a, b) => a.age - b.age); // 按年龄升序

localeCompare — 字符串排序

// 中文/带重音符号的字符串排序 ['äpple', 'apple', 'banana'].sort(); // ['apple', 'banana', 'äpple'] ['äpple', 'apple', 'banana'].sort((a, b) => a.localeCompare(b)); // ['apple', 'äpple', 'banana'] ← 正确排序

toSorted() / toReversed() — 不修改原数组

// ES2023 新增，返回新数组，不修改原数组 let arr = [3, 1, 4, 1, 5]; let sorted = arr.toSorted((a, b) => a - b); // [1, 1, 3, 4, 5] let reversed = arr.toReversed(); // [5, 1, 4, 1, 3] console.log(arr); // [3, 1, 4, 1, 5]（不变）
