赋值与自增自减运算符

赋值运算符用于修改变量的值，自增/自减是最常见的简写。

赋值运算符

运算符名称示例等价于
=赋值x = 5x = 5
+=加后赋值x += 3x = x + 3
-=减后赋值x -= 3x = x - 3
*=乘后赋值x *= 3x = x * 3
/=除后赋值x /= 2x = x / 2
%=取余后赋值x %= 2x = x % 2
**=幂后赋值x **= 2x = x ** 2
let x = 10; x += 5; // x = 15 x -= 3; // x = 12 x *= 2; // x = 24 x /= 4; // x = 6 x %= 4; // x = 2 x **= 3; // x = 8

自增 ++ 和自减 --

用于给变量加1或减1，是 +=1 和 -=1 的简写。

let count = 0; count++; // count = 1（++在后） count--; // count = 0（--在后）

前置 vs 后置

let a = 5; // 后置++：先用原值，再加1 let b = a++; // b = 5（先赋值），a = 6（后加1） // 前置++：先加1，再用新值 let c = ++a; // a = 7（先加1），c = 7（后赋值） // 同理： let d = 5; let e = d--; // e = 5, d = 4 let f = --d; // d = 3, f = 3
记忆技巧：
++ 在 前 → 先加再赋值
++ 在 后 → 后加再赋值

常见使用场景

// 1. 循环计数（最常用） for (let i = 0; i < 10; i++) { // 每次循环 i 自增1 } // 2. 统计数量 let likes = 0; function like() { likes++; // 点赞数+1 } // 3. 数组索引遍历 let index = 0; while (index < arr.length) { console.log(arr[index++]); // 先取值再自增 }
