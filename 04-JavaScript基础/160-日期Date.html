Date — 日期与时间

JavaScript 的日期对象，基于 Unix 时间戳（毫秒）。

创建 Date 对象

// 当前时间 const now = new Date(); // 无参数 new Date(); // 当前时间 // 毫秒时间戳（1970-01-01 以来的毫秒数） new Date(0); // 1970-01-01T00:00:00.000Z new Date(1000 * 60 * 60 * 24); // 1970-01-02 // 字符串解析 new Date('2024-01-01'); new Date('2024-01-01T12:30:00'); new Date('2024/01/01 12:30:00'); // 不同格式 // 参数列表（年，月[0-11]，日，时，分，秒，毫秒） new Date(2024, 0, 1); // 2024-01-01（月份 0=一月） new Date(2024, 0, 1, 12, 30, 0, 0); // 2024-01-01 12:30:00 // ⚠️ 月份范围 0-11（0=一月，11=十二月） // ⚠️ 日期越界时会自动进位

Getter 方法

const d = new Date(2024, 5, 15, 10, 30, 45, 123); d.getFullYear(); // 2024（年份） d.getMonth(); // 5（月份 0-11，6月） d.getDate(); // 15（日 1-31） d.getDay(); // 6（星期几 0-6，0=周日） d.getHours(); // 10（时 0-23） d.getMinutes(); // 30（分 0-59） d.getSeconds(); // 45（秒 0-59） d.getMilliseconds(); // 123（毫秒 0-999） d.getTime(); // 毫秒时间戳 d.getTimezoneOffset(); // 时区偏移（分钟，东八区=-480） // UTC 版本 d.getUTCFullYear(); // UTC 年份 d.getUTCMonth(); // UTC 月份 d.getUTCDate(); // UTC 日 // ... getUTCHours, getUTCMinutes 等

Setter 方法

const d = new Date(); d.setFullYear(2025); d.setMonth(11); // 12月（0-11） d.setDate(25); // 日 d.setHours(15); // 时 d.setMinutes(0); // 分 d.setSeconds(0); // 秒 d.setMilliseconds(0); // 毫秒 d.setTime(1700000000000); // 设置时间戳 // 也支持 UTC 版本 d.setUTCFullYear(2025); // ⚠️ setMonth(12) 会自动进到下一年 // ⚠️ setDate(0) 会变成上个月最后一天

格式化输出

const d = new Date(2024, 0, 1, 14, 30, 0); // 字符串格式 d.toString(); // 'Mon Jan 01 2024 14:30:00 GMT+0800 (CST)' d.toDateString(); // 'Mon Jan 01 2024' d.toTimeString(); // '14:30:00 GMT+0800 (CST)' // 本地化格式 d.toLocaleString(); // '2024/1/1 14:30:00'（取决于浏览器语言） d.toLocaleDateString(); // '2024/1/1' d.toLocaleTimeString(); // '14:30:00' // 带选项的本地化 d.toLocaleDateString('zh-CN', { year: 'numeric', month: 'long', day: 'numeric', weekday: 'long' }); // '2024年1月1日星期一' d.toLocaleString('en-US', { month: 'short', day: 'numeric', hour: '2-digit', minute: '2-digit' }); // 'Jan 1, 02:30 PM' // ISO 格式 d.toISOString(); // '2024-01-01T06:30:00.000Z'（UTC） d.toJSON(); // '2024-01-01T06:30:00.000Z' // UTC 格式 d.toUTCString(); // 'Mon, 01 Jan 2024 06:30:00 GMT'
💡 toLocaleString 的选项：weekday(short/long/narrow)、year(numeric/2-digit)、month(numeric/2-digit/short/long/narrow)、day、hour、minute、second、timeZone、timeZoneName。

时间计算

// 日期加减（时间戳毫秒） const now = new Date(); const tomorrow = new Date(now.getTime() + 24 * 60 * 60 * 1000); const nextWeek = new Date(now.getTime() + 7 * 24 * 60 * 60 * 1000); // setDate 自动进位 const d = new Date(2024, 0, 31); d.setDate(d.getDate() + 1); // 2024-02-01（自动进位） // 计算时间差 const start = new Date(2024, 0, 1); const end = new Date(2024, 11, 31); const diff = end.getTime() - start.getTime(); const days = diff / (1000 * 60 * 60 * 24); // 相差天数 // 比较日期（可以直接用比较运算符） date1 < date2; // true/false date1 - date2; // 毫秒差 // 月份加减（注意自动进位） const d2 = new Date(2024, 0, 31); d2.setMonth(d2.getMonth() + 1); // 2024-03-02（没有2月31日，自动进到3月2日）

静态方法

// 当前时间戳（毫秒） Date.now(); // 1700000000000（等价于 new Date().getTime()） // 解析日期字符串（不推荐，不同浏览器可能有差异） Date.parse('2024-01-01'); // 时间戳 Date.parse('2024/01/01'); // UTC 时间创建（参数同构造函数，但按 UTC 解释） Date.UTC(2024, 0, 1); // 时间戳（UTC 下的 2024-01-01） // 性能测量（高精度时间，不受系统时间调整影响） const start2 = performance.now(); // ... 执行代码 ... const end2 = performance.now(); const elapsed = end2 - start2; // 毫秒（含小数）
