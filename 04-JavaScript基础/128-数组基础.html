数组基础

数组是有序的数据集合，可以存放任意类型。

创建数组

// 字面量（推荐） let arr1 = [1, 2, 3, 4, 5]; let arr2 = ['apple', 'banana', 'cherry']; let arr3 = [1, 'hello', true, null, [2, 3]]; // 混合类型 // Array() 构造函数 let arr4 = new Array(5); // [empty × 5]（长度为5的空数组） let arr5 = new Array(1, 2, 3); // [1, 2, 3] // 推荐：始终用字面量 []

访问与修改

let fruits = ['apple', 'banana', 'cherry']; fruits[0]; // 'apple'（索引从0开始） fruits[1]; // 'banana' fruits[3]; // undefined（越界） fruits[fruits.length - 1]; // 'cherry'（最后一个） // 修改 fruits[1] = 'blueberry'; // ['apple', 'blueberry', 'cherry'] // 添加超出长度 fruits[5] = 'date'; // ['apple', 'blueberry', 'cherry', empty×2, 'date']

length 属性

let arr = [1, 2, 3, 4, 5]; arr.length; // 5 // length 可写！截断数组 arr.length = 3; // [1, 2, 3]（后面被截断） // 清空数组 arr.length = 0; // [] // length 总是比最大索引大1

遍历数组

let colors = ['red', 'green', 'blue']; // for 循环（需要索引） for (let i = 0; i < colors.length; i++) { console.log(i, colors[i]); } // for...of（不需要索引） for (let color of colors) { console.log(color); } // forEach（推荐） colors.forEach((color, index) => { console.log(index, color); });

检测数组

Array.isArray([1, 2, 3]); // true Array.isArray('hello'); // false Array.isArray({}); // false // instanceof 检查（但跨 iframe 会失效） [1,2,3] instanceof Array; // true
💡 推荐用 Array.isArray() 检测数组，跨 iframe 也有效。
