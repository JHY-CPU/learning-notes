数组静态方法与 ES6+ 新方法

Array 的静态工具方法和数组的新式方法。

Array.isArray() — 检测数组

Array.isArray([1, 2, 3]); // true Array.isArray('hello'); // false Array.isArray({ 0: 'a' }); // false Array.isArray(new Array(5)); // true // 比 instanceof 更可靠（跨 iframe 也有效）

Array.from() — 类数组转数组

// 1. 字符串转数组 Array.from('hello'); // ['h','e','l','l','o'] // 2. Set 转数组 Array.from(new Set([1,2,3])); // [1,2,3] // 3. arguments 转数组 function showArgs() { return Array.from(arguments); } showArgs(1, 2, 3); // [1, 2, 3] // 4. 第二个参数：map 函数 Array.from('hello', c => c.toUpperCase()); // ['H','E','L','L','O'] // 5. 创建指定长度的序列 Array.from({length: 5}, (_, i) => i + 1); // [1, 2, 3, 4, 5]

Array.of() — 创建数组

Array.of(7); // [7]（而不是 new Array(7) 的 [empty×7]） Array.of(1, 2, 3); // [1, 2, 3] Array.of('hello'); // ['hello']

flat() / flatMap() — 展平

// flat — 展平嵌套数组 [1, [2, [3, [4]]]].flat(); // [1, 2, [3, [4]]]（默认展平1层） [1, [2, [3, [4]]]].flat(2); // [1, 2, 3, [4]] [1, [2, [3, [4]]]].flat(Infinity); // [1, 2, 3, 4] // flatMap — map + flat 一步完成 ['hello world', 'hi there'].flatMap(s => s.split(' ')); // ['hello', 'world', 'hi', 'there'] // 相当于： ['hello world', 'hi there'].map(s => s.split(' ')).flat(); // ['hello', 'world', 'hi', 'there']
