节点遍历

在 DOM 树中上下左右移动，访问父/子/兄弟节点。

父节点访问

// parentNode — 父节点（可能不是元素） const parent = element.parentNode; // 如果元素没有父节点，返回 null // document.documentElement.parentNode === document // parentElement — 父元素节点 const parentEl = element.parentElement; // 如果父节点不是元素，返回 null // document.body.parentElement === document.documentElement (<html>) // document.documentElement.parentElement === null // closest — 向上查找最近的匹配选择器的祖先 const closest = element.closest('.card'); const closest2 = element.closest('#container'); // 从元素自身开始查找，沿父级向上 // 没找到返回 null // 常用：事件委托中找目标元素

子节点访问

// childNodes — 所有子节点（包括文本和注释节点） const allChildren = element.childNodes; // NodeList（live） // children — 所有元素子节点（只包括 Element） const elementChildren = element.children; // HTMLCollection（live） // firstChild — 第一个子节点（可能是文本） const first = element.firstChild; // lastChild — 最后一个子节点 const last = element.lastChild; // firstElementChild — 第一个元素子节点 const firstEl = element.firstElementChild; // lastElementChild — 最后一个元素子节点 const lastEl = element.lastElementChild; // childElementCount — 子元素数量 const count = element.childElementCount; // 等价于 element.children.length

兄弟节点访问

// previousSibling — 上一个兄弟节点（可能是文本） const prev = element.previousSibling; // nextSibling — 下一个兄弟节点（可能是文本） const next = element.nextSibling; // previousElementSibling — 上一个元素兄弟 const prevEl = element.previousElementSibling; // nextElementSibling — 下一个元素兄弟 const nextEl = element.nextElementSibling; // 遍历兄弟元素 let current = element.firstElementChild; while (current) { console.log(current.textContent); current = current.nextElementSibling; } // 反向遍历兄弟 let current = element.lastElementChild; while (current) { console.log(current.textContent); current = current.previousElementSibling; }

节点类型检测

// nodeType — 节点类型数字 element.nodeType; // 1 — 元素 // 属性 nodeType: 2 text.nodeType; // 3 — 文本 comment.nodeType; // 8 — 注释 document.nodeType; // 9 — 文档 // nodeName — 节点名称 element.nodeName; // 'DIV', 'SPAN', 'P'（大写） text.nodeName; // '#text' comment.nodeName; // '#comment' // tagName — 元素标签名（仅元素节点有效） element.tagName; // 'DIV', 'SPAN' // nodeValue — 节点值 element.nodeValue; // null text.nodeValue; // 文本内容 comment.nodeValue; // 注释内容
