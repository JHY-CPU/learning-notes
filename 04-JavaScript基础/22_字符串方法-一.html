字符串方法(一) — 查找与判断

判断字符串的包含关系、位置等信息。

length — 字符串长度

'hello'.length; // 5 '你好世界'.length; // 4（中文算一个字符） ' '.length; // 3（空格算字符） ''.length; // 0

indexOf() / lastIndexOf() — 查找位置

let str = 'hello world hello'; str.indexOf('hello'); // 0（从开头找） str.indexOf('world'); // 6 str.indexOf('xyz'); // -1（找不到） str.indexOf('hello', 1); // 12（从索引1开始找） str.lastIndexOf('hello'); // 12（从末尾找） str.lastIndexOf('world'); // 6
找不到返回 -1。可以用来检测是否存在：str.indexOf('xxx') !== -1

includes() — 是否包含

let str = 'hello world'; str.includes('hello'); // true str.includes('xyz'); // false str.includes(''); // true（空字符串总是包含） str.includes('Hello'); // false（区分大小写）
💡 includes() 比 indexOf !== -1 更语义化。

startsWith() / endsWith() — 是否以...开头/结尾

let url = 'https://example.com/images/photo.jpg'; url.startsWith('https'); // true url.startsWith('http://'); // false url.startsWith('example', 8); // true（从索引8开始） url.endsWith('.jpg'); // true url.endsWith('.png'); // false url.endsWith('images/', 23); // true（只看前23个字符）

charAt() / at() — 获取字符

let str = 'JavaScript'; str.charAt(0); // 'J' str.charAt(4); // 'S' str.charAt(100); // ''（越界返回空字符串） str.at(0); // 'J'（ES2022新增） str.at(-1); // 't'（负索引从末尾数） str.at(-2); // 'p'
at() 支持负索引，比 charAt 更强大。

charCodeAt() / codePointAt() — Unicode编码

'A'.charCodeAt(0); // 65（ASCII码） 'a'.charCodeAt(0); // 97 '0'.charCodeAt(0); // 48 '你'.charCodeAt(0); // 20320（Unicode编码） '😀'.charCodeAt(0); // 55357（只返回前半个编码） '😀'.codePointAt(0); // 128512（正确返回完整 emoji）
