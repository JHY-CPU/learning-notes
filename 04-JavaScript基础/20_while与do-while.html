while 和 do-while 循环

两种基于条件的循环，和 for 循环可以互相转换。

while 循环

先判断条件，再执行循环体。条件为 false 时一次都不执行。

while (条件) { // 条件为 true 时重复执行 } let i = 0; while (i < 5) { console.log(i); i++; // 很重要！条件要变化 } // 输出: 0, 1, 2, 3, 4
⚠️ 忘记更新条件 → 死循环！

while 遍历数组

let fruits = ['apple', 'banana', 'cherry']; let i = 0; while (i < fruits.length) { console.log(fruits[i]); i++; } // 'apple', 'banana', 'cherry'

do-while 循环

先执行一次循环体，再判断条件。至少执行一次。

do { // 先执行一次（不管条件是否成立） } while (条件); let i = 0; do { console.log(i); i++; } while (i < 5); // 输出: 0, 1, 2, 3, 4

while vs do-while

// while：可能一次都不执行 let x = 10; while (x < 5) { console.log('不会执行'); } // do-while：至少执行一次 let y = 10; do { console.log('至少执行一次'); } while (y < 5); // 输出: '至少执行一次'

while 的实际应用

// 1. 不知道循环次数时（如读取数据直到结束） let data = fetchChunk(); while (data) { process(data); data = fetchChunk(); } // 2. 数字数位处理 let num = 12345; while (num > 0) { let digit = num % 10; console.log(digit); // 5, 4, 3, 2, 1 num = Math.floor(num / 10); } // 3. 用户输入验证（至少输入一次） let input; do { input = prompt('请输入一个数字:'); } while (isNaN(input));

for vs while 选择

// 知道循环次数 → 用 for for (let i = 0; i < arr.length; i++) { ... } // 不知道循环次数 → 用 while while (data.next()) { ... } // 至少执行一次 → 用 do-while do { // ... } while (条件);

break 和 continue 同样适用

// break 退出循环 let i = 0; while (true) { // 故意写死循环 if (i >= 5) { break; // 条件满足时退出 } console.log(i); i++; } // continue 跳过当前次 let j = 0; while (j < 5) { j++; if (j === 3) { continue; // 跳过3 } console.log(j); // 1, 2, 4, 5 }
