数组 filter() 与 reduce()

筛选和聚合数据。

filter() — 筛选

let nums = [1, 2, 3, 4, 5, 6]; // 筛选偶数 let evens = nums.filter(n => n % 2 === 0); // [2, 4, 6] // 筛选大于3的数 let big = nums.filter(n => n > 3); // [4, 5, 6] // 对象数组筛选 let products = [ { name: '手机', price: 3000 }, { name: '键盘', price: 200 }, { name: '显示器', price: 1500 } ]; let cheap = products.filter(p => p.price < 1000); // [{ name: '键盘', price: 200 }]
💡 filter() 返回新数组，不修改原数组。回调返回 true 保留，false 过滤。

reduce() — 累积

let nums = [1, 2, 3, 4, 5]; // 求和（初始值 0） let sum = nums.reduce((acc, cur) => acc + cur, 0); // 15 // 求最大值（不设初始值，用第一个元素） let max = nums.reduce((acc, cur) => acc > cur ? acc : cur); // 5 // 数组 → 对象 let fruits = ['apple', 'banana', 'apple', 'orange', 'banana', 'apple']; let count = fruits.reduce((acc, fruit) => { acc[fruit] = (acc[fruit] || 0) + 1; return acc; }, {}); // { apple: 3, banana: 2, orange: 1 }

reduce 参数详解

arr.reduce((accumulator, currentValue, index, array) => { // accumulator: 上一次回调的返回值（或初始值） // currentValue: 当前元素 // index: 当前索引 // array: 原数组 }, initialValue); // 如果没有 initialValue，第一次 accumulator = arr[0]，从 arr[1] 开始

链式调用

let nums = [1, 2, 3, 4, 5, 6]; // 先筛选偶数，再每个乘以3，然后求和 let result = nums .filter(n => n % 2 === 0) // [2, 4, 6] .map(n => n * 3) // [6, 12, 18] .reduce((sum, n) => sum + n, 0); // 36 // 一行完成复杂数据处理！
