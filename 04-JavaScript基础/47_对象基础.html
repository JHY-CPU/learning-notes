对象基础

对象是键值对的集合，JS 中最核心的数据结构。

创建对象

// 字面量（推荐） const user = { name: 'Alice', age: 25, 'full name': 'Alice Wang', email: 'alice@example.com' }; // new Object() const obj = new Object(); obj.key = 'value'; // 空对象 const empty = {};

访问属性

const user = { name: 'Alice', age: 25 }; // 点语法（推荐，属性名必须是合法标识符） user.name; // 'Alice' user.age; // 25 // 方括号语法（属性名含特殊字符或动态） user['name']; // 'Alice' user['full name']; // 'Alice Wang'（点语法不能访问） // 动态键名 const key = 'age'; user[key]; // 25

修改 / 添加 / 删除

const user = { name: 'Alice' }; // 修改 user.name = 'Bob'; // 添加 user.age = 30; user['email'] = 'bob@example.com'; // 删除 delete user.email; delete user['email']; // 方括号也可以 // 检查是否存在 'name' in user; // true（in 运算符） 'age' in user; // true 'email' in user; // false（删除了） user.hasOwnProperty('name'); // true（自身属性，不含继承的） user.hasOwnProperty('toString'); // false（继承自原型）

属性简写

const name = 'Alice'; const age = 25; // 旧写法 const user1 = { name: name, age: age }; // ES6 简写（变量名和属性名一样时） const user2 = { name, age }; // 方法简写 const obj = { // 旧 sayHi: function() { console.log('Hi'); }, // 新（ES6） sayHello() { console.log('Hello'); } };
