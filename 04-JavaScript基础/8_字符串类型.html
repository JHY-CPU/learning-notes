字符串类型 string

字符串表示文本数据，是不可变的（创建后不能修改）。

三种引号

let single = '单引号字符串'; let double = "双引号字符串"; let backtick = `反引号字符串（模板字符串）`; // 单双引号功能相同，只是风格区别 // 反引号支持插值（${}）和多行

转义字符

console.log('换行: \n 新行'); console.log('制表符: \t 缩进'); console.log('反斜杠: \\'); console.log('单引号: \''); console.log('双引号: \"'); console.log('Unicode: ©'); // © console.log('Unicode: \u{1F600}'); // 😀
在字符串中放特殊字符需要用 \ 转义

字符串的 length 属性

'hello'.length; // 5 '你好世界'.length; // 4（中文算1个字符） ' '.length; // 3（空格算字符） ''.length; // 0（空字符串） '\\n'.length; // 2（\n 是两个字符）

通过索引访问字符

let str = 'JavaScript'; str[0]; // 'J' str[4]; // 'S' str[9]; // 't' str[100]; // undefined（超出范围） str[-1]; // undefined（不支持负数） // 也可以用 charAt（越界返回空字符串） str.charAt(0); // 'J' str.charAt(100); // ''（空字符串）

字符串是不可变的

let s = 'Hello'; s[0] = 'h'; // ❌ 不会改变 console.log(s); // 还是 'Hello' // 要修改只能创建新字符串 s = 'h' + s.slice(1); // 'hello'

字符串比较

// 按字典顺序（Unicode编码）比较 'a' < 'b'; // true 'apple' > 'banana'; // false '2' > '10'; // true（字符 '2' > '1'，不是数值比较！） // 字符串和数字比较时，字符串会转数字 '5' > 2; // true（'5' → 5） 'abc' > 0; // false（'abc' → NaN，NaN比较都是false）
⚠️ 比较字符串用 localeCompare 更可靠
