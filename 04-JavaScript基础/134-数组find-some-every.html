数组 find / some / every

查找符合条件的元素或判断条件。

find() — 查找第一个匹配元素

let users = [ { id: 1, name: 'Alice', age: 25 }, { id: 2, name: 'Bob', age: 30 }, { id: 3, name: 'Charlie', age: 35 } ]; let user = users.find(u => u.id === 2); // { id: 2, name: 'Bob', age: 30 } let minor = users.find(u => u.age < 18); // undefined（没找到） // 找到就停止，不继续遍历
💡 find() 返回第一个匹配的元素，找不到返回 undefined。

findIndex() — 查找第一个匹配的索引

let nums = [5, 12, 8, 130, 44]; let idx = nums.findIndex(n => n > 10); // 1（12的索引） let notFound = nums.findIndex(n => n > 200); // -1（找不到）

some() — 是否有元素符合条件

let nums = [1, 2, 3, 4, 5]; nums.some(n => n > 3); // true（有大于3的） nums.some(n => n > 10); // false（没有大于10的） nums.some(n => n % 2 === 0);// true（有偶数） // 只要找到一个符合条件的就返回 true

every() — 是否所有元素都符合条件

let nums = [2, 4, 6, 8, 10]; nums.every(n => n % 2 === 0); // true（全是偶数） nums.every(n => n > 5); // false（2和4不大于5） let ages = [25, 30, 18, 22]; ages.every(a => a >= 18); // true（全部成年）

对比总结

// find → 找第一个匹配的"元素" // findIndex → 找第一个匹配的"索引" // some → 是否有匹配（返回 true/false） // every → 是否全部匹配（返回 true/false） // filter → 找所有匹配的"数组" // indexOf → 找精确值的索引（不能用回调） let nums = [1, 2, 3, 4, 5]; nums.find(n => n > 3); // 4 nums.findIndex(n => n > 3); // 3 nums.some(n => n > 3); // true nums.every(n => n > 3); // false nums.filter(n => n > 3); // [4, 5]
