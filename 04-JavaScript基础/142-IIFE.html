IIFE — 立即执行函数表达式

Immediately Invoked Function Expression，定义后立即执行的函数。

IIFE 语法

// 写法1（推荐）：包裹在 () 中，末尾调用 (function() { console.log('立即执行'); })(); // 写法2：调用括号在外面（Douglas Crockford 风格） (function() { console.log('立即执行'); }()); // 箭头函数 IIFE (() => { console.log('箭头函数 IIFE'); })(); // 带参数 (function(name) { console.log('Hello, ' + name); })('Alice');

通过 IIFE 创建私有作用域

// IIFE 内部的变量不会污染全局 (function() { let privateVar = '只有IIFE内部能访问'; console.log(privateVar); // ✅ })(); // console.log(privateVar); // ❌ 报错！ // 多个 IIFE 互不干扰 (function() { let counter = 0; console.log('IIFE 1:', ++counter); // 1 })(); (function() { let counter = 0; console.log('IIFE 2:', ++counter); // 1（互不影响） })();

IIFE 模块模式

const Counter = (function() { // 私有变量 let count = 0; // 私有函数 function log(msg) { console.log(`[Counter] ${msg}`); } // 公开接口 return { increment() { count++; log(`count → ${count}`); }, decrement() { count--; log(`count → ${count}`); }, getCount() { return count; } }; })(); Counter.increment(); // [Counter] count → 1 Counter.increment(); // [Counter] count → 2 console.log(Counter.getCount()); // 2 // console.log(Counter.count); // undefined（不能访问）
💡 IIFE 模块模式是 ES6 模块出现前最常用的封装方式。

经典应用：循环中的闭包

// 用 IIFE 创建独立作用域 for (var i = 0; i < 3; i++) { (function(j) { setTimeout(function() { console.log(j); // 0, 1, 2 }, 100); })(i); } // 现在直接用 let 即可 for (let i = 0; i < 3; i++) { setTimeout(() => console.log(i), 100); // 0, 1, 2 }
