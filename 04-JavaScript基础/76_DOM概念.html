DOM 概念

Document Object Model — 将 HTML 文档表示为树形结构的对象。

DOM 树结构

// 每个 HTML 页面被浏览器解析成一棵树 // 树的每个节点都是一个 DOM 节点 // document → html → head → title // → body → h1 // → p // → div → span // 节点类型（Node.nodeType）： // 1 — ELEMENT_NODE（元素节点，如 div, p） // 2 — ATTRIBUTE_NODE（属性节点，已不常用） // 3 — TEXT_NODE（文本节点） // 8 — COMMENT_NODE（注释节点） // 9 — DOCUMENT_NODE（document 节点） // 10 — DOCUMENT_TYPE_NODE（） // 11 — DOCUMENT_FRAGMENT_NODE（文档片段） // document 是 DOM 的入口 console.log(document); // document 对象 console.log(document.documentElement); // <html> console.log(document.body); // <body> console.log(document.head); // <head> console.log(document.title); // 页面标题

节点关系

// DOM 节点之间的关系（类似家族树） // parentNode // │ // ┌────┴────┐ // │ │ // child child // │ │ // └── sibling ──┘ // 常用属性： // parentNode — 父节点 // childNodes — 所有子节点（包含文本节点） // children — 所有元素子节点 // firstChild — 第一个子节点 // lastChild — 最后一个子节点 // firstElementChild — 第一个元素子节点 // lastElementChild — 最后一个元素子节点 // previousSibling — 上一个兄弟节点 // nextSibling — 下一个兄弟节点 // previousElementSibling — 上一个兄弟元素节点 // nextElementSibling — 下一个兄弟元素节点 // 示例： const body = document.body; // console.log(body.parentNode); // <html> // console.log(body.children); // body 的子元素 // console.log(body.firstElementChild); // 第一个子元素

DOM 操作分类

// DOM 操作主要分为： // 1. 查询/选择 document.getElementById('id'); document.querySelector('.class'); document.querySelectorAll('div'); // 2. 创建节点 document.createElement('div'); document.createTextNode('文本'); // 3. 插入/追加 parent.appendChild(child); parent.insertBefore(newNode, refNode); element.prepend(text); element.append(child); // 4. 删除/替换 parent.removeChild(child); element.remove(); parent.replaceChild(newChild, oldChild); // 5. 修改内容 element.innerHTML = '<b>HTML</b>'; element.textContent = '纯文本'; element.setAttribute('class', 'foo'); // 6. 样式操作 element.style.color = 'red'; element.classList.add('active'); // 7. 事件 element.addEventListener('click', handler);

重要特性

// 1. DOM 是"活的" // 获取 NodeList 后，DOM 变化会实时反映（部分集合是动态的） const liveList = document.getElementsByTagName('div'); // HTMLCollection 是动态的（live） // NodeList 通常是静态的（querySelectorAll） // 2. DOM 操作可能触发重排（reflow）和重绘（repaint） // 重排：布局变化，开销大（修改宽高、增删元素） // 重绘：样式变化，开销较小（修改颜色） // 3. 批量操作优化 // 使用 DocumentFragment 或 display:none 批量修改 const fragment = document.createDocumentFragment(); // 在 fragment 上多次操作 // 一次性插入到 DOM（只触发一次重排） // 4. DOM 不是 JavaScript 的一部分 // DOM 是浏览器提供的 Web API（用 JS 调用） // 其他语言也可以操作 DOM
