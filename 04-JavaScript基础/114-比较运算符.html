比较运算符

比较两个值，返回布尔值（true / false）。

比较运算符列表

运算符名称示例结果
===严格等于5 === '5'false
!==严格不等于5 !== '5'true
==宽松等于5 == '5'true
!=宽松不等于5 != '5'false
>大于5 > 3true
<小于5 < 3false
>=大于等于5 >= 5true
<=小于等于5 <= 3false

⚠️ 严格等于 === vs 宽松等于 ==

// === 严格等于：先比较类型，类型不同直接false 5 === 5; // true 5 === '5'; // false（类型不同：number vs string） 0 === false; // false（类型不同） null === undefined; // false（类型不同） // == 宽松等于：先转类型再比较 5 == '5'; // true（字符串'5'转成数字5） 0 == false; // true（false转成0） '' == 0; // true（空字符串转成0） null == undefined; // true（特殊规则） [] == false; // true（复杂规则）
🚨 永远使用 === 和 !==，不要使用 == 和 !=！

== 的隐式转换陷阱

// == 的转换规则非常复杂，容易出bug '' == '0'; // false 0 == ''; // true 0 == '0'; // true false == 'false'; // false（'false'转数字是NaN） false == '0'; // true [] == ![]; // true ← wtf? [] == 0; // true [1] == 1; // true [1,2] == NaN; // false
💡 记住：永远用 ===，== 有太多隐式转换陷阱！

比较不同类型

// 数字 vs 字符串：字符串转数字 5 > '3'; // true（'3' → 3） '10' > 5; // true（'10' → 10） 'abc' > 0; // false（'abc' → NaN, NaN比较返回false） // 字符串比较：按字典顺序（Unicode编码） 'a' < 'b'; // true 'apple' < 'banana'; // true 'Z' < 'a'; // true（大写字母编码小于小写） // null 和 undefined null > 0; // false null == 0; // false（==不转换null） null >= 0; // true（>=会转换null为0） undefined > 0; // false undefined == 0; // false
