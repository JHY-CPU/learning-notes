事件类型

常见 DOM 事件：鼠标、键盘、表单、窗口等。

鼠标事件

// 鼠标事件类型 click — 单击（完整点击：mousedown + mouseup） dblclick — 双击 mousedown — 鼠标按下 mouseup — 鼠标松开 mousemove — 鼠标移动 mouseover — 鼠标进入元素（冒泡） mouseout — 鼠标离开元素（冒泡） mouseenter — 鼠标进入元素（不冒泡） mouseleave — 鼠标离开元素（不冒泡） contextmenu — 右键菜单 // mouseover vs mouseenter： // mouseover 会冒泡，从子元素移入也会触发父元素 // mouseenter 不冒泡，只进入元素本身时触发 // 鼠标事件对象属性： event.clientX / event.clientY — 视口坐标 event.pageX / event.pageY — 页面坐标 event.screenX / event.screenY — 屏幕坐标 event.offsetX / event.offsetY — 元素内坐标 event.button — 0=左键 1=中键 2=右键 event.ctrlKey / shiftKey / altKey / metaKey

触摸与指针事件

// 触摸事件（移动端） touchstart — 触摸开始 touchmove — 触摸移动 touchend — 触摸结束 touchcancel — 触摸中断 // 触摸事件对象属性： event.touches — 当前所有触摸点 event.targetTouches — 目标元素上的触摸点 event.changedTouches — 发生变化的触摸点 // 每个 touch 对象有 clientX/Y, pageX/Y, identifier // 指针事件（统一鼠标/触摸/笔） pointerdown — 指针按下 pointerup — 指针松开 pointermove — 指针移动 pointerover — 指针进入 pointerout — 指针离开 // ⚠️ 优先使用指针事件（兼容鼠标和触摸）

焦点与表单事件

// 焦点事件 focus — 获得焦点（不冒泡） blur — 失去焦点（不冒泡） focusin — 获得焦点（冒泡版） focusout — 失去焦点（冒泡版） // 表单事件 submit — 表单提交 reset — 表单重置 change — 值改变并失去焦点 input — 输入值变化（实时） select — 选中文本 invalid — 验证不通过 // 窗口事件 load — 页面/资源加载完成 DOMContentLoaded — HTML 解析完成（不等待图片） resize — 窗口大小变化 scroll — 滚动 beforeunload — 页面卸载前 unload — 页面卸载 hashchange — URL hash 变化 popstate — 历史记录变化
