加法 + 的两种作用

// 1. 数学加法（两边都是数字） 1 + 2; // 3 // 2. 字符串拼接（只要一边是字符串） 'Hello ' + 'World'; // "Hello World" 1 + '2'; // "12" '3' + 4; // "34" 1 + 2 + '3'; // "33"（先算 1+2=3，然后 3+'3'="33"） '1' + 2 + 3; // "123"（先拼接 "1"+"2"="12"，再+"3"="123"）
⚠️ 加号是 JS 中最容易出错的运算符！数值相加会被误拼成字符串。

除法和取余

// 除法总是返回浮点数 5 / 2; // 2.5 10 / 3; // 3.3333... 0 / 5; // 0 5 / 0; // Infinity -5 / 0; // -Infinity 0 / 0; // NaN // 取余（模运算）— 判断能否整除 10 % 3; // 1 10 % 2; // 0（能整除） 7 % 5; // 2

幂运算 **

2 ** 3; // 8（2的3次方） 10 ** 2; // 100 2 ** 10; // 1024 // 相当于 Math.pow(2, 3)

运算符优先级

2 + 3 * 4; // 14（乘优先于加） (2 + 3) * 4; // 20（括号优先） 2 ** 3 * 2; // 16（** 优先于 *） 2 + 3 + '4'; // "54"（从左到右） '2' + 3 + 4; // "234"（从左到右拼接）

优先级从高到低：

1. () 括号 2. ** 幂运算 3. * / % 乘除取余 4. + - 加减
💡 不确定优先级时用括号明确，代码更可读。
