数组增删方法

在数组两端或任意位置添加/删除元素。

push() / pop() — 末尾操作

let arr = [1, 2, 3]; arr.push(4); // 返回 4（新长度），arr → [1,2,3,4] arr.push(5, 6); // 返回 6，arr → [1,2,3,4,5,6] arr.pop(); // 返回 6（删除的元素），arr → [1,2,3,4,5] arr.pop(); // 返回 5，arr → [1,2,3,4]

unshift() / shift() — 开头操作

let arr = [1, 2, 3]; arr.unshift(0); // 返回 4（新长度），arr → [0,1,2,3] arr.unshift(-2, -1); // 返回 6，arr → [-2,-1,0,1,2,3] arr.shift(); // 返回 -2（删除的元素），arr → [-1,0,1,2,3] arr.shift(); // 返回 -1，arr → [0,1,2,3] // ⚠️ shift/unshift 比 push/pop 慢！ // 因为要移动所有元素的位置
💡 性能：push/pop O(1)，shift/unshift O(n)。

splice() — 万能增删改

let arr = ['a', 'b', 'c', 'd', 'e']; // 删除：splice(开始位置, 删除数量) arr.splice(1, 2); // ['b', 'c']（被删除的），arr → ['a','d','e'] // 插入：splice(开始位置, 0, ...插入元素) arr.splice(1, 0, 'x', 'y'); // []（没删），arr → ['a','x','y','d','e'] // 替换：splice(开始位置, 删除数量, ...插入元素) arr.splice(2, 1, 'z'); // ['y']，arr → ['a','x','z','d','e'] // ⚠️ splice 会修改原数组！

fill() — 填充

let arr = [1, 2, 3, 4, 5]; arr.fill(0); // [0,0,0,0,0]（全填0） arr.fill(1, 2); // [0,0,1,1,1]（从索引2开始填1） arr.fill(2, 1, 3); // [0,2,2,1,1]（索引1到3填2） // 创建指定长度的数组 new Array(5).fill(0); // [0,0,0,0,0]
