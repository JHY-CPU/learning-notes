字符串方法(三) — 格式与转换

大小写转换、去空格、填充等。

大小写转换

'Hello World'.toUpperCase(); // "HELLO WORLD" 'Hello World'.toLowerCase(); // "hello world" // 常用于校验（不区分大小写） 'admin@email.com'.toLowerCase() === 'Admin@Email.com'.toLowerCase(); // true

trim() — 去除首尾空格

' hello '.trim(); // "hello" ' hello '.trimStart(); // "hello " ' hello '.trimEnd(); // " hello" 'hello'.trim(); // "hello"（无变化） // 常用于用户输入处理 let input = ' user@example.com '; let email = input.trim(); // "user@example.com"
💡 处理表单输入时一定要 trim()，用户可能不小心打了空格。

padStart() / padEnd() — 填充

// padStart(目标长度, 填充字符) '5'.padStart(3, '0'); // "005" '123'.padStart(5, '0'); // "00123" 'hello'.padStart(10, '-'); // "-----hello" 'hello'.padStart(3, '-'); // "hello"（原长度超过目标长度） // padEnd '5'.padEnd(3, '0'); // "500" 'hello'.padEnd(8, '.'); // "hello..."

repeat() — 重复

'*'.repeat(5); // "*****" 'abc'.repeat(3); // "abcabcabc" '😀'.repeat(2); // "😀😀" ''.repeat(5); // "" 'x'.repeat(0); // "" // 'x'.repeat(-1); // ❌ 报错

localeCompare() — 本地化比较

// 按字典顺序比较字符串，考虑本地语言规则 'ä'.localeCompare('z'); // -1（ä 在德语中排在 z 前） // 比 < > 更可靠，考虑本地语言规则 ['ä', 'a', 'z'].sort(); // ["a", "z", "ä"] ['ä', 'a', 'z'].sort((a, b) => a.localeCompare(b)); // ["a", "ä", "z"]

实用组合

// 生成唯一ID function genId() { return Date.now().toString(36) + Math.random().toString(36).slice(2); } // 隐藏邮箱中间部分 function maskEmail(email) { let [name, domain] = email.split('@'); return name[0] + '*'.repeat(name.length - 2) + name.slice(-1) + '@' + domain; } console.log(maskEmail('alice@example.com')); // "a***e@example.com"
