数组高阶方法 — forEach 与 map

函数式编程风格的数组遍历方法。

forEach() — 遍历

let arr = ['a', 'b', 'c']; arr.forEach(function(item, index, array) { console.log(item, index); }); // 'a' 0, 'b' 1, 'c' 2 // 箭头函数写法 arr.forEach((item, index) => console.log(item, index)); // ⚠️ forEach 不返回新数组，不能 break/continue

map() — 映射

let nums = [1, 2, 3, 4, 5]; let doubled = nums.map(n => n * 2); // [2, 4, 6, 8, 10] let squared = nums.map(n => n ** 2); // [1, 4, 9, 16, 25] // 对象转换 let users = [ { name: 'Alice', age: 25 }, { name: 'Bob', age: 30 } ]; let names = users.map(user => user.name); // ['Alice', 'Bob']
💡 map() 返回与原数组等长的新数组，不修改原数组。

forEach vs map 对比

// forEach — 用于"执行操作" arr.forEach(item => console.log(item)); // 打印每个元素 // map — 用于"转换数据" let newArr = arr.map(item => item * 2); // 生成新数组 // forEach 的返回值是 undefined！ let result = arr.forEach(x => x * 2); // undefined let mapped = arr.map(x => x * 2); // [2,4,6,...]
