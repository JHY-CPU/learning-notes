对象方法

作为对象属性的函数称为方法。

定义方法

const user = { name: 'Alice', // 方式一：传统 sayHi: function() { console.log('Hi!'); }, // 方式二：ES6 简写（推荐） sayHello() { console.log('Hello!'); }, // 方式三：箭头函数（注意 this 问题） greet: () => { console.log('Greetings!'); } }; user.sayHi(); // 'Hi!' user.sayHello(); // 'Hello!'

方法中的 this

const user = { name: 'Alice', age: 25, // 方法中的 this 指向调用方法的对象 introduce() { console.log(`我叫${this.name}，今年${this.age}岁`); }, // 箭头函数没有自己的 this // this 指向外层（这里指向 window/全局） arrowIntro: () => { // console.log(this.name); // undefined } }; user.introduce(); // '我叫Alice，今年25岁' // this 的值取决于如何调用 const fn = user.introduce; fn(); // 此时 this 是 undefined（严格模式）或 window

链式调用

// 方法返回 this 就可以链式调用 const calculator = { value: 0, add(n) { this.value += n; return this; // 返回自身 }, subtract(n) { this.value -= n; return this; }, multiply(n) { this.value *= n; return this; }, getResult() { return this.value; } }; calculator.add(5).subtract(2).multiply(3).getResult(); // (0 + 5 - 2) × 3 = 9

动态添加方法

const car = { brand: 'Toyota' }; // 添加方法 car.drive = function() { console.log(`${this.brand} 在行驶`); }; car.drive(); // 'Toyota 在行驶' // 也可以用方括号 car['stop'] = function() { console.log('刹车'); };
