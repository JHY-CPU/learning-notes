this — 默认绑定与隐式绑定

this 的值取决于函数的调用方式（而不是定义位置）。

默认绑定（单独调用函数）

// 非严格模式：this → window（全局对象） function showThis() { console.log(this); // window（浏览器）/ global（Node） } showThis(); // 严格模式：this → undefined function showThisStrict() { 'use strict'; console.log(this); // undefined } showThisStrict(); // 函数表达式同理 const fn = function() { console.log(this); // window（非严格） }; fn(); // ⚠️ 嵌套函数中的 this const obj = { name: 'Alice', say() { function inner() { console.log(this); // window（不是 obj！） } inner(); } };
💡 默认绑定规则：函数独立调用时，非严格模式 this 指向全局对象，严格模式指向 undefined。

隐式绑定（作为对象方法调用）

const user = { name: 'Alice', greet() { console.log(this.name); } }; user.greet(); // 'Alice'（this → user） // 方法中的 this 指向调用者 const user2 = { name: 'Bob' }; user2.greet = user.greet; user2.greet(); // 'Bob'（this → user2） // ⚠️ 隐式丢失 const greet = user.greet; greet(); // undefined（this → window，window.name 不存在） // ⚠️ 回调函数中的 this setTimeout(user.greet, 100); // undefined（作为独立函数调用） // ⚠️ 链式调用 const objChain = { name: 'outer', inner: { name: 'inner', greet() { console.log(this.name); } } }; objChain.inner.greet(); // 'inner'（this 指向最近的调用者 .inner）

this 绑定优先级

// 优先级：new > 显式绑定 > 隐式绑定 > 默认绑定 // 1. 默认绑定（最低） function show() { console.log(this); } show(); // window // 2. 隐式绑定 const obj = { name: 'obj', show }; obj.show(); // obj // 3. 显式绑定（call/apply/bind） obj.show.call({ name: 'explicit' }); // { name: 'explicit' } // 4. new 绑定（最高） function Person(name) { this.name = name; } const p = new Person('Alice'); // this → 新创建的对象 // 箭头函数不绑定 this（穿透作用域）
