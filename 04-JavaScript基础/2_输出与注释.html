JavaScript 输出方式

打开浏览器控制台 (F12 → Console) 查看输出

1. console.log() — 控制台打印

console.log('这是一条普通日志');

最常用的调试方式，可以打印任何类型的数据。

2. console.warn() — 警告

console.warn('这是一条警告信息');

3. console.error() — 错误

console.error('这是一条错误信息');

4. console.table() — 表格形式

console.table(['apple', 'banana', 'cherry']);

5. alert() — 弹窗

alert('这是一个弹窗');

会阻塞页面，用户体验差，仅用于调试。

6. document.write() — 写入文档

document.write('<p>直接写入页面</p>');

⚠️ 会覆盖已加载的页面内容，不推荐使用。

7. 输出到页面元素

document.getElementById('output').textContent = 'Hello!';

8. 注释写法

// 单行注释 - 用于简短说明 /* * 多行注释 * 用于长篇说明 * 或暂时禁用代码 */ /* 也可以写成一行 */

注释不会被浏览器执行，仅用于说明代码。

快捷键：Ctrl + / （单行注释）/ Alt + Shift + A（块注释）

其他 console 方法

console.info('信息'); // 打印信息 console.debug('调试'); // 打印调试信息 console.time('计时'); // 开始计时 console.timeEnd('计时'); // 结束计时 console.count('计数'); // 计数 console.group('分组'); // 分组折叠 console.groupEnd(); // 结束分组 console.trace(); // 打印调用栈
