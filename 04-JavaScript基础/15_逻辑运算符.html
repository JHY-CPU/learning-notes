逻辑运算符

用于组合多个条件，控制程序的逻辑流程。

三种基本逻辑运算符

运算符名称含义示例
&&AND（与）所有条件都为 truea > 0 && b > 0
||OR（或）至少一个条件为 truea > 0 || b > 0
!NOT（非）取反!isActive

基本用法

// && — 全部为true才返回true true && true; // true true && false; // false false && false; // false // || — 只要有一个true就返回true true || false; // true false || false; // false false || true; // true // ! — 取反 !true; // false !false; // true !!'hello'; // true（转布尔） // 实际条件判断 let age = 20; age >= 18 && age <= 60; // true（成年且未退休） age < 18 || age > 60; // false（不在区间外）

短路求值（Short-circuit Evaluation）

逻辑运算符不一定返回 true/false，而是返回某个操作数的值。

// && 短路：遇到 falsy 就停止，返回该值 0 && 'hello'; // 0（第一个就是 false，不继续了） 1 && 'hello'; // 'hello'（两个都是 truthy，返回最后一个） null && 'hello'; // null 'hello' && 'world'; // 'world' // || 短路：遇到 truthy 就停止，返回该值 0 || 'hello'; // 'hello'（第一个是 false，继续找） 1 || 'hello'; // 1（第一个就是 true，不继续了） '' || 'default'; // 'default' null || undefined; // undefined（都是 falsy，返回最后一个）
💡 || 常用于给变量设置默认值

实际应用：默认值

// 给参数设置默认值（旧方式） function greet(name) { name = name || '游客'; // 如果 name 是 falsy，使用'游客' console.log('你好, ' + name); } greet(); // "你好, 游客" greet('Alice'); // "你好, Alice" // ⚠️ || 的缺陷：如果合法值是空字符串/0，也会被覆盖 let count = 0; let result = count || 10; // 10（0被当成falsy了！） // 此时应该用 ??（空值合并运算符）: // let result = count ?? 10; // 0（只有 null/undefined 才使用默认值）

实际应用：条件执行

// && 短路执行（如果条件为真才执行） isLoggedIn && renderDashboard(); // 只有登录了才渲染 // 相当于： if (isLoggedIn) { renderDashboard(); } // React 中常用： // {isLoading && } // {!isLoading && }

逻辑非 ! 的常见用法

// 取反 let isActive = false; if (!isActive) { /* 执行 */ } // 转布尔（双感叹号） !!'hello'; // true !!0; // false !!null; // false // 检查变量是否存在 if (!user) { /* 用户不存在 */ }
