选择器 — getElement 系列

传统的 DOM 选择方法，返回 HTMLCollection（动态集合）。

getElementById

// 通过 id 获取单个元素（最常用、最快） // 返回 Element | null const header = document.getElementById('header'); // 如果没找到，返回 null // ⚠️ id 在页面中应该是唯一的 // ⚠️ 不需要写 # 前缀（区别于 CSS 选择器） // 也可以直接用 window.idName（不推荐） // 浏览器会将部分 id 作为 window 的属性

getElementsByClassName

// 通过 class 获取元素集合 // 返回 HTMLCollection（动态集合 — DOM 变化时自动更新） const items = document.getElementsByClassName('item'); // 所有 class 包含 'item' 的元素 // 可以在指定元素下查找 const container = document.getElementById('container'); const containerItems = container.getElementsByClassName('item'); // HTMLCollection 特征： // 1. 类数组，有 length，可以下标访问 [0] // 2. 动态的（live）— DOM 增删自动反映 // 3. 不是数组（不能用 forEach/map/filter） // 4. 转数组：[...items] 或 Array.from(items) // ⚠️ HTMLCollection 没有 forEach！ // 需要用 for...of 或 for 循环遍历 for (let item of items) { console.log(item.textContent); } // 多个 class 名顺序不重要，空格分隔 // getElementsByClassName('foo bar') 匹配 class="foo bar" 或 class="bar foo"

getElementsByTagName

// 通过标签名获取元素集合 // 返回 HTMLCollection（动态集合） const allDivs = document.getElementsByTagName('div'); const allPs = document.getElementsByTagName('p'); const allSpans = document.getElementsByTagName('span'); // 获取所有元素（*） const allElements = document.getElementsByTagName('*'); // 不区分大小写 getElementsByTagName('DIV') === getElementsByTagName('div'); // true // 可以在指定元素下查找 const main = document.getElementById('main'); const mainDivs = main.getElementsByTagName('div');

HTMLCollection vs NodeList

// HTMLCollection（getElement 系列返回） // — 动态集合（live），DOM 变化自动更新 // — 只有元素节点（Element） // — 可以用 name/id 索引（如 form.elements） // — 没有 forEach // NodeList（querySelectorAll 返回） // — 静态集合（static），不会自动更新 // — 包含所有节点类型（元素+文本+注释） // — 有 forEach（现代浏览器） // — 可以用 for...of 遍历 // 示例对比： const liveCollection = document.getElementsByTagName('div'); const staticList = document.querySelectorAll('div'); console.log('HTMLCollection 动态:', liveCollection.length); console.log('NodeList 静态:', staticList.length); // 新增一个 div 后： document.body.appendChild(document.createElement('div')); console.log('HTMLCollection 自动更新:', liveCollection.length); // +1 console.log('NodeList 不变:', staticList.length); // 不变
