键盘与表单事件

keydown/keyup、submit/input/change/focus/blur。

键盘事件

// 键盘事件类型： // keydown — 按下任意键（包括功能键，长按会重复触发） // keyup — 松开键 // keypress — 已废弃（不要使用） // 重要属性： event.key — 按键的字符值（'a', 'A', '1', 'Enter', 'Escape'） event.code — 物理键代码（'KeyA', 'Digit1', 'Enter', 'ArrowUp'） event.ctrlKey — 是否按了 Ctrl event.shiftKey — 是否按了 Shift event.altKey — 是否按了 Alt event.metaKey — 是否按了 Meta（Windows键/Command键） event.repeat — 是否重复触发（长按） // key vs code 区别： // 按 Shift + a（大写A）： // event.key = 'A'（字符值） // event.code = 'KeyA'（物理键） // 常用组合键： if (e.ctrlKey && e.key === 's') { // Ctrl+S e.preventDefault(); save(); } if (e.key === 'Escape') { // 按 Esc closeModal(); } if (e.key === 'Enter') { // 按回车 submit(); }

表单事件

// submit — 表单提交 form.addEventListener('submit', function(e) { e.preventDefault(); // 阻止提交（常用于验证） const data = new FormData(form); console.log('提交数据:', Object.fromEntries(data)); }); // input — 输入变化（实时触发） input.addEventListener('input', function() { console.log('当前值:', this.value); }); // change — 值改变并失去焦点 input.addEventListener('change', function() { console.log('改变为:', this.value); }); // focus / blur — 获得/失去焦点 input.addEventListener('focus', function() { this.style.borderColor = '#3498db'; }); input.addEventListener('blur', function() { this.style.borderColor = '#ddd'; }); // select — 选中文本 input.addEventListener('select', function() { console.log('选中:', window.getSelection().toString()); });
