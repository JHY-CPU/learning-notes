DOM 样式 — style 属性

通过 JavaScript 操作元素的行内样式。

style 属性（行内样式）

// style 属性是 CSSStyleDeclaration 对象 // 可读写元素的行内样式（非计算样式） const el = document.getElementById('demo'); // 设置单个样式属性（驼峰命名） el.style.color = 'red'; el.style.backgroundColor = '#f0f0f0'; // background-color → backgroundColor el.style.fontSize = '24px'; // font-size → fontSize el.style.borderRadius = '8px'; // border-radius → borderRadius el.style.marginTop = '10px'; el.style.display = 'flex'; // CSS 属性名转换规则： // CSS: background-color → JS: backgroundColor // CSS: font-size → JS: fontSize // CSS: margin-left → JS: marginLeft // CSS: border-top-width → JS: borderTopWidth // CSS: float → JS: cssFloat（float 是保留字） // 特殊值 el.style.display = 'none'; // 隐藏 el.style.visibility = 'hidden'; // 隐藏（保留空间） el.style.opacity = '0.5'; // 半透明 // ⚠️ 设置的值必须是字符串（带单位） el.style.width = 100; // ❌ 无效 el.style.width = '100px'; // ✅ // ⚠️ 读取未设置的样式返回空字符串 console.log(el.style.marginLeft); // ''（如果没设过）

cssText — 批量设置样式

// cssText — 用字符串设置多个样式 // 会覆盖所有已存在的行内样式 // 批量设置（替换所有行内样式） el.style.cssText = 'color: red; background: blue; font-size: 20px;'; // 追加（不推荐，会覆盖） el.style.cssText += '; border: 1px solid black'; // 对比：分别设置（推荐，不影响其他样式） el.style.color = 'red'; el.style.background = 'blue'; el.style.fontSize = '20px'; // 清除行内样式 el.style.cssText = ''; // 方式1 el.removeAttribute('style'); // 方式2

getComputedStyle — 获取计算样式

// style 只能读取行内样式 // getComputedStyle 获取最终计算后的样式（包括 CSS 文件中的） const el = document.getElementById('demo'); // 获取计算样式（只读） const computed = getComputedStyle(el); // 或 window.getComputedStyle(el) console.log(computed.color); // 'rgb(255, 0, 0)' console.log(computed.fontSize); // '24px' console.log(computed.backgroundColor); // 'rgb(240, 240, 240)' // 获取伪元素样式 const beforeStyle = getComputedStyle(el, '::before'); const afterStyle = getComputedStyle(el, '::after'); console.log(beforeStyle.content); // 伪元素的 content // ⚠️ getComputedStyle 是只读的！ // ⚠️ 返回的值可能是相对值（如 em、%）经过计算后的绝对值（px） // ⚠️ 颜色值标准化为 rgb/rgba 格式

style vs getComputedStyle

// style（行内样式） // ✅ 可读写 // ✅ 只反映行内样式（style 属性） // ❌ 不能读取 CSS 文件中的样式 // ❌ 未设置的属性返回空字符串 // getComputedStyle（计算样式） // ✅ 最终生效样式（CSS + 行内 + 继承） // ✅ 伪元素样式 // ❌ 只读 // ❌ 触发重排（性能开销） // 使用场景： // 设置样式 → style // 读取最终样式 → getComputedStyle // 只需要行内样式 → style // 要判断元素是否隐藏 → getComputedStyle(el).display
