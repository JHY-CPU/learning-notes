Object 静态方法

Object 对象自带的工具方法。

Object.assign() — 合并对象

const target = { a: 1 }; const source1 = { b: 2 }; const source2 = { c: 3, a: 99 }; // a 会覆盖 Object.assign(target, source1, source2); // target → { a: 99, b: 2, c: 3 } // 常用：创建副本 const copy = Object.assign({}, original); // 合并默认配置 const defaults = { theme: 'light', lang: 'zh' }; const userConfig = { theme: 'dark' }; const config = Object.assign({}, defaults, userConfig); // { theme: 'dark', lang: 'zh' }

Object.freeze() — 冻结对象

const obj = Object.freeze({ name: 'Alice' }); obj.name = 'Bob'; // 静默失败（严格模式报错） obj.age = 25; // 不能添加 delete obj.name; // 不能删除 // 只读，不能修改、添加、删除 Object.isFrozen(obj); // true（检查是否冻结） // ⚠️ 浅冻结！嵌套对象仍可修改 const frozen = Object.freeze({ inner: { x: 1 } }); frozen.inner.x = 100; // ✅ 内层对象可以修改
💡 freeze() 可以让对象真正不可变（但只针对第一层）。

Object.seal() — 密封对象

const obj = Object.seal({ name: 'Alice', age: 25 }); obj.name = 'Bob'; // ✅ 可以修改 obj.job = 'dev'; // ❌ 不能添加 delete obj.age; // ❌ 不能删除 Object.isSealed(obj); // true // freeze vs seal： // freeze: 不能改/增/删 // seal: 可以改，不能增/删

Object.defineProperty() — 定义属性

const obj = {}; Object.defineProperty(obj, 'name', { value: 'Alice', writable: true, // 可修改 enumerable: true, // 可遍历 configurable: true // 可删除/配置 }); // 定义 getter/setter Object.defineProperty(obj, 'fullName', { get() { return this._firstName + ' ' + this._lastName; }, set(val) { [this._firstName, this._lastName] = val.split(' '); } });
