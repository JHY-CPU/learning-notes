JavaScript 数据类型

JS 是动态类型语言，变量类型由运行时值决定。

基本类型（8 种）

类型typeof 结果示例
number"number"42, 3.14, NaN, Infinity
string"string"'hello', "world", `template`
boolean"boolean"true, false
undefined"undefined"undefined
null"object"null（typeof 是历史遗留bug）
bigint"bigint"9007199254740991n
symbol"symbol"Symbol('id')
object"object"{}, [], new Date()

number 数字类型

let int = 42; // 整数 let float = 3.14; // 浮点数 let negative = -10; // 负数 let hex = 0xff; // 十六进制 let octal = 0o77; // 八进制 let binary = 0b1010; // 二进制 let big = 1e6; // 科学计数法 = 1000000 let nan = NaN; // Not a Number let inf = Infinity; // 无穷大

string 字符串类型

let single = '单引号'; let double = "双引号"; let backtick = `模板字符串`; let multi = `多行 字符串`; let escape = "换行符: \n 制表符: \t";

boolean / undefined / null

// boolean let isActive = true; let isDone = false; // undefined - 变量声明但未赋值 let notAssigned; console.log(notAssigned); // undefined // null - 表示"空"（主动赋值） let empty = null; // 区别： // undefined → 未定义（系统产生） // null → 定义了但为空（人为设置）

bigint 大整数

let bigInt = 9007199254740991n; // 末尾加 n let fromNum = BigInt(123); // BigInt() 转换 let fromStr = BigInt('0x1a'); // 字符串转 // 不能和普通 number 混合运算 // console.log(bigInt + 1); // ❌ 报错 console.log(bigInt + 1n); // ✅ 必须同类型

typeof 运算符

typeof 42 // "number" typeof 'hello' // "string" typeof true // "boolean" typeof undefined // "undefined" typeof null // "object" ← 这是 JS 的著名 bug！ typeof 1n // "bigint" typeof Symbol() // "symbol" typeof {} // "object" typeof [] // "object" ← 数组也是 object typeof function(){} // "function"
typeof null === "object" 是 JS 从诞生就有的 bug，但为了兼容性一直没修。

动态类型

let value = 42; // number value = 'hello'; // 变成 string value = true; // 变成 boolean value = { name: 'JS' };// 变成 object // JS 变量可以随时改变类型，这就是"动态类型"
