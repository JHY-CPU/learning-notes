箭头函数 Arrow Function

ES6 引入的更简洁的函数写法。

基本语法

// 无参数 const f1 = () => console.log('Hello'); // 一个参数（可以省略括号） const f2 = x => x * 2; // 多个参数（必须括号） const f3 = (a, b) => a + b; // 函数体多行（需要 {} 和 return） const f4 = (a, b) => { const result = a + b; return result; };

从普通函数到箭头函数

// 普通函数 function add(a, b) { return a + b; } // 箭头函数（逐步简化） const add = (a, b) => { return a + b; }; // 去掉 function，加 => const add = (a, b) => a + b; // 去掉 {} 和 return // 一个参数可以去掉括号 const double = x => x * 2; // 没有参数必须写空括号 const random = () => Math.random();

返回值规则

// 隐式返回：函数体只有一行表达式时 const sum = (a, b) => a + b; // 自动 return a + b // 返回对象：需要加括号 const getUser = (name) => ({ name: name, age: 25 }); // ⚠️ 不加括号 → { 被当做函数体 // const getUser = (name) => { name: name }; // 不会 return // 显式 return：函数体有多行 const process = (a, b) => { const result = a * b; return result; };

箭头函数没有 arguments

// 普通函数有 arguments function regular() { console.log(arguments); // [1, 2, 3] } regular(1, 2, 3); // 箭头函数没有 arguments const arrow = () => { // console.log(arguments); // ❌ 报错！ }; // 用 rest 参数代替 const arrow2 = (...args) => console.log(args);

箭头函数的 this

// 箭头函数不绑定自己的 this // 它捕获定义时的外部 this const obj = { name: 'Alice', regular: function() { console.log(this.name); // 'Alice'（obj） }, arrow: () => { console.log(this.name); // undefined（window/全局） } }; // 箭头函数不能用作构造函数 // const p = new Person(); // ❌ 箭头函数不能 new
💡 箭头函数没有自己的 this、arguments、super、new.target。
