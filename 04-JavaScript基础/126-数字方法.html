数字方法与 Number 属性

JS 提供数字的实例方法和 Number 对象的静态方法。

实例方法 — toFixed()

let n = 123.456789; n.toFixed(0); // "123"（四舍五入到整数） n.toFixed(1); // "123.5" n.toFixed(2); // "123.46" n.toFixed(5); // "123.45679" // ⚠️ 返回的是字符串，不是数字！ // 浮点数精度问题： (0.1 + 0.2).toFixed(1); // "0.3"（修复精度问题）

实例方法 — toPrecision() / toString()

let n = 123.456; n.toPrecision(4); // "123.5"（总有效数字4位） n.toPrecision(2); // "1.2e+2"（科学计数法） // toString(radix) — 转成指定进制字符串 n.toString(); // "123.456" n.toString(2); // "1111011.011101..."（二进制） n.toString(16); // "7b.74bc..."（十六进制） (255).toString(16); // "ff"

判断方法 — Number.isNaN() / isFinite()

// isNaN vs Number.isNaN isNaN('hello'); // true（先转数字，'hello'→NaN） Number.isNaN('hello'); // false（不转换，直接比较） // isFinite — 判断是否为有限数字 Number.isFinite(42); // true Number.isFinite(Infinity); // false Number.isFinite('42'); // false（不转类型） isFinite('42'); // true（先转类型）

判断方法 — isInteger() / isSafeInteger()

Number.isInteger(42); // true Number.isInteger(3.14); // false Number.isInteger('42'); // false（不转类型） Number.isSafeInteger(42); // true Number.isSafeInteger(9007199254740991); // true（MAX_SAFE_INTEGER） Number.isSafeInteger(9007199254740992); // false

Number 常量属性

Number.MAX_VALUE; // 1.7976931348623157e+308（最大值） Number.MIN_VALUE; // 5e-324（最小正数） Number.MAX_SAFE_INTEGER; // 9007199254740991（最大安全整数） Number.MIN_SAFE_INTEGER; // -9007199254740991 Number.EPSILON; // 2.220446049250313e-16（精度） Number.NaN; // NaN Number.POSITIVE_INFINITY; // Infinity Number.NEGATIVE_INFINITY; // -Infinity
