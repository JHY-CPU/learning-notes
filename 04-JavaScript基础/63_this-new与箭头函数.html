this — new 绑定与箭头函数

new 创建新对象并绑定 this，箭头函数不绑定 this。

new 绑定

// new 做了四件事： // 1. 创建空对象 // 2. 该对象链接到构造函数的 prototype // 3. this 指向该对象 // 4. 返回该对象（除非构造函数返回对象） function Person(name, age) { // this = {}（new 创建的空对象） this.name = name; this.age = age; // return this（默认） } const p1 = new Person('Alice', 25); console.log(p1.name); // 'Alice' console.log(p1 instanceof Person); // true // ⚠️ 构造函数也有返回值优先级 function Special(name) { this.name = name; return { custom: true }; // 如果返回对象，则返回这个对象而不是 this } const s = new Special('test'); console.log(s.name); // undefined console.log(s.custom); // true // 如果返回基本类型，则忽略 function Normal(name) { this.name = name; return 123; // 基本类型被忽略 } const n = new Normal('ok'); console.log(n.name); // 'ok'

箭头函数 — 没有自己的 this

// 箭头函数不会创建自己的 this // 它从外层作用域继承 this（词法 this） const obj = { name: 'Alice', // 普通函数：有自己的 this normal() { console.log(this.name); // 取决于调用方式 }, // 箭头函数：没有自己的 this arrow: () => { console.log(this.name); // 指向外层（定义时的 this） } }; obj.normal(); // 'Alice' obj.arrow(); // undefined（window.name，如果外层是全局） // ✅ 箭头函数修复嵌套 this const outer = { name: 'outer', say() { const inner = () => { console.log(this.name); // 'outer'（从 say 继承 this） }; inner(); } }; outer.say(); // 'outer' // ❌ 普通函数嵌套问题 const outer2 = { name: 'outer2', say() { function inner() { console.log(this.name); // undefined（this 是 window） } inner(); } }; outer2.say(); // undefined

箭头函数不能做的事

// 1. 不能用作构造函数（不能 new） const Fn = () => {}; // new Fn(); // ❌ TypeError: Fn is not a constructor // 2. 没有 prototype 属性 console.log(Fn.prototype); // undefined // 3. 不能使用 arguments const fn = () => { // console.log(arguments); // ❌ ReferenceError console.log(...arguments); // ❌ 也会报错 }; // 需要改用 rest 参数 const fn2 = (...args) => { console.log(args); }; // 4. 不能 call/apply/bind 改变 this const obj = { x: 1 }; const arrow = () => console.log(this); arrow.call(obj); // this 不会被改变（仍然是外层 this） // 5. 没有 super // class 中箭头函数的 this 固定为实例

箭头函数的应用场景

// 1. 回调函数（数组方法） const nums = [1, 2, 3, 4, 5]; const doubled = nums.map(n => n * 2); // 2. 定时器/事件监听中保持 this class Timer { constructor() { this.seconds = 0; } start() { // 箭头函数自动捕获外层的 this（Timer 实例） setInterval(() => { this.seconds++; console.log(this.seconds); }, 1000); } } // 3. 简写小函数 const add = (a, b) => a + b; // 4. 返回对象时用 () 包裹 const getUser = () => ({ name: 'Alice', age: 25 }); // 5. Promise 链中保持 this class Fetcher { constructor(url) { this.url = url; } fetch() { return fetch(this.url) .then(res => res.json()) .then(data => { this.data = data; // this 指向实例 }); } }
