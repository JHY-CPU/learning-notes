多重积分

多元微积分 · 二重积分、三重积分与变量替换

一、二重积分

1. 定义

∬_D f(x, y) dA = lim Σ f(ξᵢ, ηᵢ) · ΔAᵢ

几何意义：以 D 为底、z = f(x,y) 为顶的曲顶柱体的有向体积。

2. 直角坐标系下的计算

X-型区域（先 y 后 x）：D: a ≤ x ≤ b, φ₁(x) ≤ y ≤ φ₂(x)

∬_D f(x,y) dA = ∫ₐᵇ [ ∫_{φ₁(x)}^{φ₂(x)} f(x,y) dy ] dx

Y-型区域（先 x 后 y）：D: c ≤ y ≤ d, ψ₁(y) ≤ x ≤ ψ₂(y)

∬_D f(x,y) dA = ∫ᶜᵈ [ ∫_{ψ₁(y)}^{ψ₂(y)} f(x,y) dx ] dy

3. 极坐标系下的计算

令 x = rcosθ, y = rsinθ，dA = r dr dθ

∬_D f(x,y) dA = ∬ f(rcosθ, rsinθ) · r dr dθ

适用场景：积分区域为圆形、扇形，或被积函数含 x²+y²。

例题 1：二重积分

计算 ∬_D (x+y) dA，其中 D 是由 y=x, y=0, x=1 围成的三角形。

D: 0 ≤ x ≤ 1, 0 ≤ y ≤ x
= ∫₀¹ [∫₀ˣ (x+y) dy] dx
= ∫₀¹ [xy + y²/2]₀ˣ dx
= ∫₀¹ (x² + x²/2) dx = ∫₀¹ (3x²/2) dx
= [x³/2]₀¹ = 1/2
例题 2：极坐标二重积分

计算 ∬_D e^(-x²-y²) dA，其中 D: x²+y² ≤ a²。

极坐标：x²+y² = r², dA = r dr dθ
= ∫₀²ᵖ ∫₀ᵃ e^(-r²) · r dr dθ
= 2π ∫₀ᵃ r·e^(-r²) dr
= 2π · [-e^(-r²)/2]₀ᵃ = 2π(1-e^(-a²))/2
= π(1 - e^(-a²))

二、三重积分

定义

∭_Ω f(x, y, z) dV

计算方式：直角坐标（先一后二或先二后一）、柱坐标、球坐标。

柱坐标与球坐标

坐标系变换体积元素适用场景
柱坐标x=rcosθ, y=rsinθ, z=zdV = r dr dθ dz柱对称
球坐标x=ρsinφcosθ, y=ρsinφsinθ, z=ρcosφdV = ρ²sinφ dρ dφ dθ球对称

三、变量替换与雅可比行列式

1. 二重积分换元

令 x = x(u, v), y = y(u, v)，则

∬_D f(x,y) dxdy = ∬_{D'} f(x(u,v), y(u,v)) · |J| dudv

其中 J 为雅可比行列式：

J = ∂(x,y)/∂(u,v) = | ∂x/∂u   ∂x/∂v |                           | ∂y/∂u   ∂y/∂v | = (∂x/∂u)(∂y/∂v) - (∂x/∂v)(∂y/∂u)

2. 雅可比行列式的性质

  • 极坐标变换：J = r（这就是极坐标面积元素多一个 r 的原因）
  • 球坐标变换：J = ρ²sinφ
  • |J| 是面积（体积）的伸缩因子
多重积分计算技巧：
  • 先画积分区域图，确定积分限
  • 选择合适的坐标系（直角/极/柱/球）
  • 利用对称性简化计算（奇偶函数、区域对称）
  • 交换积分次序有时能简化计算

本节总结

  • 二重积分：直角坐标或极坐标，dA = r dr dθ
  • 三重积分：直角、柱坐标、球坐标
  • 变量替换：∬f dxdy = ∬f·|J| dudv
  • 雅可比行列式 J：面积/体积的伸缩因子
  • 极坐标 J = r，球坐标 J = ρ²sinφ
  • 画图定限是计算多重积分的关键
