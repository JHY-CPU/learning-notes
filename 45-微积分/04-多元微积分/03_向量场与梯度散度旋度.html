一、向量场（Vector Field）

向量场是空间中每个点对应一个向量的映射。在 ℝ³ 中：

F(x, y, z) = P(x,y,z) i + Q(x,y,z) j + R(x,y,z) k

例如：速度场、力场、电场、磁场都是向量场。

1.1 相关概念

  • 保守场（Conservative Field）：若存在标量函数 f 使得 F = ∇f，则 F 为保守场
  • 势函数（Potential）：上述函数 f 称为 F 的势函数
  • 无旋场（Irrotational）：curl F = 0
  • 无源场（Solenoidal）：div F = 0

二、梯度（Gradient）

标量场 f(x, y, z) 的梯度：

grad f = ∇f = (∂f/∂x) i + (∂f/∂y) j + (∂f/∂z) k

方向导数：函数 f 在单位向量 u 方向上的变化率

Duf = ∇f · u = |∇f| cos θ

当 u 与 ∇f 同向（θ = 0）时，方向导数最大，值为 |∇f|

梯度的方向是函数增长最快的方向，梯度的大小是最大变化率。

梯度的重要性质：

  • ∇f 垂直于等值面 f(x,y,z) = c
  • ∇(fg) = f∇g + g∇f
  • ∇(f/g) = (g∇f − f∇g) / g²
  • 若 F = ∇f，则 F 是保守场 ⟺ ∮F · dr = 0（沿任意闭曲线）

例：求 f(x,y,z) = x²y + yz³ 的梯度

∂f/∂x = 2xy, ∂f/∂y = x² + z³, ∂f/∂z = 3yz²

∇f = (2xy, x² + z³, 3yz²)

三、散度（Divergence）

向量场 F = (P, Q, R) 的散度是一个标量：

div F = ∇ · F = ∂P/∂x + ∂Q/∂y + ∂R/∂z

物理意义——流体流动：

  • div F > 0：该点是源（source），流体从该点流出多于流入
  • div F < 0：该点是汇（sink），流体流入多于流出
  • div F = 0：该场为无源场（不可压缩流体的速度场）

电磁学：电场的散度与电荷密度成正比——麦克斯韦方程 ∇ · E = ρ/ε₀

散度的运算法则：

  • ∇ · (F + G) = ∇ · F + ∇ · G
  • ∇ · (fF) = f(∇ · F) + F · ∇f
  • ∇ · (∇f) = ∇²f = Δf（散度的梯度 = 拉普拉斯算子）

例：求 F = (x²y, yz, xz²) 的散度

div F = ∂(x²y)/∂x + ∂(yz)/∂y + ∂(xz²)/∂z

= 2xy + z + 2xz

= 2xy + z + 2xz

四、旋度（Curl）

向量场 F = (P, Q, R) 的旋度是一个向量：

curl F = ∇ × F

= (∂R/∂y − ∂Q/∂z) i + (∂P/∂z − ∂R/∂x) j + (∂Q/∂x − ∂P/∂y) k

可用行列式记忆：

∇ × F =

|i  j  k|
|∂/∂x∂/∂y∂/∂z|
| P  Q  R |

物理意义——旋转与环量：

  • curl F 的方向：旋转轴方向（右手定则）
  • curl F 的大小：旋转角速度的两倍
  • 流体：旋度描述流体的涡旋程度；curl F = 0 表示无旋流（层流）
  • 电磁学：磁场的旋度与电流密度成正比——∇ × B = μ₀J

旋度的重要性质：

  • ∇ × (∇f) = 0（梯度的旋度恒为零——保守场无旋）
  • ∇ · (∇ × F) = 0（旋度的散度恒为零）
  • ∇ × (fF) = f(∇ × F) + (∇f) × F

例：求 F = (yz, xz, xy) 的旋度

(∂R/∂y − ∂Q/∂z) = x − x = 0

(∂P/∂z − ∂R/∂x) = y − y = 0

(∂Q/∂x − ∂P/∂y) = z − z = 0

curl F = 0，说明 F 是保守场

（事实上 F = ∇(xyz)）

五、拉普拉斯算子（Laplacian）

拉普拉斯算子是梯度的散度：

Δf = ∇²f = ∇ · (∇f) = ∂²f/∂x² + ∂²f/∂y² + ∂²f/∂z²

拉普拉斯方程与泊松方程

拉普拉斯方程：∇²f = 0（调和函数）

泊松方程：∇²f = g（源项 g）

应用：热传导方程、波动方程、薛定谔方程、引力势、静电势等都涉及拉普拉斯算子。

核心要点汇总

  • 梯度 ∇f：标量 → 向量，指向最大增长方向，垂直等值面
  • 散度 ∇ · F：向量 → 标量，衡量"源"或"汇"的强度
  • 旋度 ∇ × F：向量 → 向量，衡量旋转/涡旋程度
  • 拉普拉斯 ∇²f：梯度的散度，出现在热传导、波动、薛定谔方程中
  • 关键恒等式：curl(grad f) = 0；div(curl F) = 0
  • 保守场判据：curl F = 0（在单连通区域上） ⟺ F = ∇f
