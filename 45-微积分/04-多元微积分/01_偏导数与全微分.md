# 偏导数与全微分

*多元微积分 · 多变量函数的微分*


### 1. 定义


二元函数 z = f(x, y) 在点 (x₀, y₀) 处对 x 的偏导数：


$$
∂z/∂x = fₓ(x₀, y₀) = lim(Δx→0) [f(x₀+Δx, y₀) - f(x₀, y₀)] / Δx
$$


计算方法：求 ∂f/∂x 时，将 y 视为常数，对 x 求导。


### 2. 高阶偏导数


$$
∂²f/∂x² = fₓₓ,   ∂²f/∂x∂y = fₓᵧ,   ∂²f/∂y∂x = fᵧₓ,   ∂²f/∂y² = fᵧᵧ
$$


**克莱罗定理**：若 fₓᵧ 和 fᵧₓ 连续，则 fₓᵧ = fᵧₓ（混合偏导数与求导顺序无关）。


### 定义


若 z = f(x, y) 在点 (x, y) 处的全增量可表示为：


$$
Δz = A·Δx + B·Δy + o(ρ)   （ρ = √(Δx²+Δy²)）
$$


则称 f 在该点**可微**，全微分为：


$$
dz = (∂z/∂x)dx + (∂z/∂y)dy
$$


### 可微、可偏导、连续的关系


- **可微 ⟹ 可偏导**
   （且 dz = fₓdx + fᵧdy）
- **可微 ⟹ 连续**
- **可偏导 ⟹ 连续**
   （反例：f(x,y) 在 (0,0) 处可偏导但不连续）
- **偏导数连续 ⟹ 可微**
   （充分条件）


### 链式法则


设 z = f(u, v), u = φ(x, y), v = ψ(x, y)，则：


$$
∂z/∂x = (∂z/∂u)(∂u/∂x) + (∂z/∂v)(∂v/∂x)

            ∂z/∂y = (∂z/∂u)(∂u/∂y) + (∂z/∂v)(∂v/∂y)
$$


口诀：**沿线相乘，分线相加**——沿着每条路径将偏导数相乘，不同路径的结果相加。


### 公式法


由 F(x, y) = 0 确定的隐函数 y = y(x)：


$$
dy/dx = -Fₓ / Fᵧ   （Fᵧ ≠ 0）
$$


由 F(x, y, z) = 0 确定的隐函数 z = z(x, y)：


$$
∂z/∂x = -Fₓ / F_z,    ∂z/∂y = -Fᵧ / F_z   （F_z ≠ 0）
$$


### 1. 方向导数


函数 f 在点 P 沿方向 l 的变化率：


$$
∂f/∂l = fₓ·cosα + fᵧ·cosβ   （二维）
$$


其中 (cosα, cosβ) 是方向 l 的方向余弦。


### 2. 梯度


$$
grad f = ∇f = (∂f/∂x, ∂f/∂y)   （二维）

            grad f = (∂f/∂x, ∂f/∂y, ∂f/∂z)   （三维）
$$


梯度的方向是函数增长最快的方向，其模长为最大方向导数值。


## Python实现

### 偏导数与全微分的符号计算

```python
import sympy as sp

x, y = sp.symbols('x y')
z = x**2 * sp.sin(y) + sp.exp(x*y)

# 偏导数
print("=== 偏导数 ===")
print(f"∂z/∂x = {sp.diff(z, x)}")
print(f"∂z/∂y = {sp.diff(z, y)}")

# 高阶偏导数
print(f"\n=== 高阶偏导数 ===")
print(f"∂²z/∂x² = {sp.diff(z, x, 2)}")
print(f"∂²z/∂y² = {sp.diff(z, y, 2)}")
print(f"∂²z/∂x∂y = {sp.diff(z, x, y)}")
print(f"∂²z/∂y∂x = {sp.diff(z, y, x)}")

# 验证克莱罗定理: f_xy = f_yx
f_xy = sp.diff(z, x, y)
f_yx = sp.diff(z, y, x)
print(f"克莱罗定理验证: f_xy == f_yx ? {sp.simplify(f_xy - f_yx) == 0}")

# 梯度
grad = [sp.diff(z, var) for var in [x, y]]
print(f"\n∇z = ({grad[0]}, {grad[1]})")

# 全微分
dz = sp.diff(z, x) * sp.Symbol('dx') + sp.diff(z, y) * sp.Symbol('dy')
print(f"dz = {dz}")

# 隐函数求导: x^2 + y^2 + z^2 = 1 → ∂z/∂x = -x/z
F = x**2 + y**2 + sp.Symbol('z')**2 - 1
print(f"\nx²+y²+z²=1: ∂z/∂x = -x/z, ∂z/∂y = -y/z")
```

### 链式法则验证

```python
import sympy as sp

x, y, t = sp.symbols('x y t')

# z = x^2 + y^2, x = cos(t), y = sin(t)
# dz/dt = ∂z/∂x * dx/dt + ∂z/∂y * dy/dt
z = x**2 + y**2
x_func = sp.cos(t)
y_func = sp.sin(t)

# 直接求导
z_of_t = z.subs([(x, x_func), (y, y_func)])
direct = sp.diff(z_of_t, t)
print(f"z(t) = cos²(t) + sin²(t) = {sp.simplify(z_of_t)}")
print(f"dz/dt (直接) = {sp.simplify(direct)}")

# 链式法则
dz_dx = sp.diff(z, x)
dz_dy = sp.diff(z, y)
dx_dt = sp.diff(x_func, t)
dy_dt = sp.diff(y_func, t)
chain = dz_dx.subs(x, x_func) * dx_dt + dz_dy.subs(y, y_func) * dy_dt
print(f"dz/dt (链式法则) = {sp.simplify(chain)}")
print(f"两种方法结果相同: {sp.simplify(direct - chain) == 0}")
```
