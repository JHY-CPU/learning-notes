多元函数极值

Extrema of Multivariable Functions — 多元微积分 · 第五节

一、无条件极值（Critical Points）

1.1 必要条件

若 f(x, y) 在点 (a, b) 处取极值，且 f 在该点可微，则：

fx(a, b) = 0 且 fy(a, b) = 0

满足此条件的点称为驻点（临界点 / critical point）。

推广到 n 元：∇f = 0。

1.2 二阶充分条件——Hessian 矩阵

设 (a, b) 是 f 的驻点，记：

A = fxx(a,b),   B = fxy(a,b),   C = fyy(a,b)

Hessian 矩阵：

H = | A   B |
    | B   C |

判别式：D = AC − B² = det(H)

条件 结论
D > 0 且 A > 0 极小值
D > 0 且 A < 0 极大值
D < 0 鞍点（非极值）
D = 0 需用其他方法判断

例：求 f(x, y) = x³ + y³ − 3xy 的极值

fx = 3x² − 3y = 0, fy = 3y² − 3x = 0

驻点：(0, 0) 和 (1, 1)

fxx = 6x, fxy = −3, fyy = 6y

在 (0, 0)：D = 0·0 − 9 = −9 < 0 → 鞍点

在 (1, 1)：D = 6·6 − 9 = 27 > 0, A = 6 > 0 → 极小值 f(1,1) = −1

多元函数的 Hessian 推广：

对 n 元函数 f(x₁, …, xn)，Hessian 矩阵 H 为 n×n 矩阵：

Hij = ∂²f / ∂xi∂xj

H 正定 ⟹ 极小值；H 负定 ⟹ 极大值；H 不定 ⟹ 鞍点

二、条件极值与拉格朗日乘数法

2.1 问题描述

求函数 f(x, y) 在约束条件 g(x, y) = 0 下的极值。

几何意义：在曲线 g = 0 上找 f 的最值点。

拉格朗日乘数法（Lagrange Multipliers）

设 f 和 g 有连续偏导数，且 ∇g ≠ 0。若 f 在约束 g(x,y) = 0 下取极值，则存在常数 λ 使得：

∇f = λ ∇g

即：

∂f/∂x = λ · ∂g/∂x

∂f/∂y = λ · ∂g/∂y

g(x, y) = 0

三个方程，三个未知数 (x, y, λ)。

2.2 多个约束条件

求 f(x, y, z) 在约束 g(x,y,z) = 0 和 h(x,y,z) = 0 下的极值：

∇f = λ∇g + μ∇h

联立 g = 0, h = 0，共五个方程，五个未知数 (x, y, z, λ, μ)

例：求 f(x, y) = x² + y² 在约束 x + y = 1 下的最小值

g(x,y) = x + y − 1 = 0

∇f = (2x, 2y), ∇g = (1, 1)

2x = λ, 2y = λ → x = y

x + y = 1 → x = y = 1/2

最小值 = (1/2)² + (1/2)² = 1/2

例：求 xyz 在 x² + y² + z² = 12 条件下的最大值

∇f = (yz, xz, xy), ∇g = (2x, 2y, 2z)

yz = 2λx, xz = 2λy, xy = 2λz

三式相乘：x²y²z² = 8λ³xyz → 若 xyz ≠ 0：xyz = 8λ³

由对称性：x² = y² = z² = 4，即 x = y = z = 2

最大值 = 2 × 2 × 2 = 8

三、拉格朗日函数

3.1 构造拉格朗日函数

L(x, y, λ) = f(x, y) − λ · g(x, y)

将条件极值问题转化为 L 的无条件极值问题：

∂L/∂x = 0 → fx = λgx

∂L/∂y = 0 → fy = λgy

∂L/∂λ = 0 → g = 0

拉格朗日乘数 λ 的意义：

若将约束写为 g(x,y) = c，则 λ = ∂f*/∂c，其中 f* 是 f 在约束下的最优值。
即：λ 表示约束条件的微小变化对最优值的边际影响（影子价格）。

四、不等式约束与 KKT 条件

4.1 Karush-Kuhn-Tucker（KKT）条件

求 min f(x) 满足 g(x) ≤ 0, h(x) = 0：

构造：L = f(x) + λg(x) + μh(x)

KKT 条件：

① ∇xL = 0（平稳性）

② g(x) ≤ 0, h(x) = 0（原始可行性）

③ λ ≥ 0（对偶可行性）

④ λg(x) = 0（互补松弛性）

互补松弛性：要么约束起作用（g = 0），要么乘数为零（λ = 0）。

KKT 条件是拉格朗日乘数法在不等式约束下的推广，是最优化理论的基础，在机器学习（SVM）、经济学、工程设计中广泛应用。

五、闭区域上的最值

5.1 求解步骤

  1. 求区域内部的驻点（∇f = 0），计算函数值
  2. 求区域边界上的最值（参数化边界或拉格朗日乘数法）
  3. 比较所有候选点的函数值，最大者为最大值，最小者为最小值

例：求 f(x,y) = x² + 2y² − x 在圆盘 x² + y² ≤ 4 上的最大值和最小值

内部：fx = 2x − 1 = 0, fy = 4y = 0 → (1/2, 0), f = −1/4

边界：g = x² + y² − 4 = 0

∇f = λ∇g → (2x−1, 4y) = λ(2x, 2y)

若 y ≠ 0：4y = 2λy → λ = 2, 2x−1 = 4x → x = −1/2, y² = 15/4

f(−1/2, ±√(15)/2) = 1/4 + 15/2 + 1/2 = 33/4

若 y = 0：x = ±2, f(2,0) = 2, f(−2,0) = 6

最小值 = −1/4（在 (1/2, 0)），最大值 = 33/4（在 (−1/2, ±√(15)/2)）

核心要点汇总

  • 驻点：∇f = 0 是极值的必要条件
  • Hessian 判别：D = AC − B²，D > 0 判定极值，D < 0 为鞍点
  • 拉格朗日乘数法：∇f = λ∇g，将条件极值化为方程组
  • 拉格朗日函数：L = f − λg，对各变量和 λ 求偏导
  • 多约束：∇f = λ∇g + μ∇h
  • 闭区域最值：比较内部驻点和边界上的候选值
  • KKT 条件：拉格朗日乘数法对不等式约束的推广
