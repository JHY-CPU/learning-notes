偏导数与全微分

多元微积分 · 多变量函数的微分

一、偏导数

1. 定义

二元函数 z = f(x, y) 在点 (x₀, y₀) 处对 x 的偏导数：

∂z/∂x = fₓ(x₀, y₀) = lim(Δx→0) [f(x₀+Δx, y₀) - f(x₀, y₀)] / Δx

计算方法：求 ∂f/∂x 时，将 y 视为常数，对 x 求导。

2. 高阶偏导数

∂²f/∂x² = fₓₓ,   ∂²f/∂x∂y = fₓᵧ,   ∂²f/∂y∂x = fᵧₓ,   ∂²f/∂y² = fᵧᵧ

克莱罗定理：若 fₓᵧ 和 fᵧₓ 连续，则 fₓᵧ = fᵧₓ（混合偏导数与求导顺序无关）。

例题 1：偏导数计算

设 f(x, y) = x³y² + sin(xy)，求 fₓ, fᵧ, fₓᵧ。

fₓ = 3x²y² + y·cos(xy)   （y 视为常数）
fᵧ = 2x³y + x·cos(xy)   （x 视为常数）
fₓᵧ = ∂(fₓ)/∂y = 6x²y + cos(xy) - xy·sin(xy)

二、全微分

定义

若 z = f(x, y) 在点 (x, y) 处的全增量可表示为：

Δz = A·Δx + B·Δy + o(ρ)   （ρ = √(Δx²+Δy²)）

则称 f 在该点可微，全微分为：

dz = (∂z/∂x)dx + (∂z/∂y)dy

可微、可偏导、连续的关系

  • 可微 ⟹ 可偏导（且 dz = fₓdx + fᵧdy）
  • 可微 ⟹ 连续
  • 可偏导 ⟹ 连续（反例：f(x,y) 在 (0,0) 处可偏导但不连续）
  • 偏导数连续 ⟹ 可微（充分条件）

三、多元复合函数求导（链式法则）

链式法则

设 z = f(u, v), u = φ(x, y), v = ψ(x, y)，则：

∂z/∂x = (∂z/∂u)(∂u/∂x) + (∂z/∂v)(∂v/∂x)

∂z/∂y = (∂z/∂u)(∂u/∂y) + (∂z/∂v)(∂v/∂y)

口诀：沿线相乘，分线相加——沿着每条路径将偏导数相乘，不同路径的结果相加。

例题 2：链式法则

设 z = eᵘsinv, u = xy, v = x+y，求 ∂z/∂x。

∂z/∂u = eᵘsinv, ∂z/∂v = eᵘcosv
∂u/∂x = y, ∂v/∂x = 1

∂z/∂x = eᵘsinv · y + eᵘcosv · 1
= eˣʸ[ysin(x+y) + cos(x+y)]

四、隐函数求导

公式法

由 F(x, y) = 0 确定的隐函数 y = y(x)：

dy/dx = -Fₓ / Fᵧ   （Fᵧ ≠ 0）

由 F(x, y, z) = 0 确定的隐函数 z = z(x, y)：

∂z/∂x = -Fₓ / F_z,    ∂z/∂y = -Fᵧ / F_z   （F_z ≠ 0）
例题 3：隐函数求导

设 x² + y² + z² = 1，求 ∂z/∂x 和 ∂z/∂y。

F(x,y,z) = x²+y²+z²-1
Fₓ = 2x, Fᵧ = 2y, F_z = 2z
∂z/∂x = -x/z, ∂z/∂y = -y/z

五、方向导数与梯度

1. 方向导数

函数 f 在点 P 沿方向 l 的变化率：

∂f/∂l = fₓ·cosα + fᵧ·cosβ   （二维）

其中 (cosα, cosβ) 是方向 l 的方向余弦。

2. 梯度

grad f = ∇f = (∂f/∂x, ∂f/∂y)   （二维）

grad f = (∂f/∂x, ∂f/∂y, ∂f/∂z)   （三维）

梯度的方向是函数增长最快的方向，其模长为最大方向导数值。

偏导数与梯度的关系： 梯度是一个向量，由各方向的偏导数组成。方向导数 = 梯度在该方向上的投影。沿梯度方向方向导数最大（增长最快），沿梯度反方向方向导数最小（下降最快）。

本节总结

  • 偏导数：固定其他变量，对某一个变量求导
  • 全微分：dz = fₓdx + fᵧdy，可微 ⟹ 可偏导且连续
  • 链式法则：沿线相乘，分线相加
  • 隐函数求导：∂z/∂x = -Fₓ/F_z
  • 梯度：∇f = (fₓ, fᵧ)，指向函数增长最快的方向
  • 混合偏导连续时与求导顺序无关（克莱罗定理）
