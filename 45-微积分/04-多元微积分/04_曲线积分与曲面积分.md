# 曲线积分与曲面积分 - 多元微积分

*Line Integrals and Surface Integrals — 多元微积分 · 第四节*


### 1.1 计算方法


参数化曲线 C：x = x(t), y = y(t), t ∈ [α, β]，则


$$
∫C f ds = ∫αβ f(x(t), y(t)) √([x′(t)]² + [y′(t)]²) dt
$$


关键：ds = √(dx² + dy²)，积分始终从较小参数到较大参数（**不依赖方向**）。


### 2.1 计算方法


$$
参数化 C：r(t) = (x(t), y(t), z(t)), t ∈ [α, β]
$$


$$
∫C F · dr = ∫αβ F(r(t)) · r′(t) dt
$$


### 3.1 格林公式的应用


- **计算平面面积**
   ：A = (1/2) ∮
   ~C~
   (x dy − y dx)
- **判断保守场**
   ：在单连通区域上，∂P/∂y = ∂Q/∂x ⟺ 积分与路径无关
- **简化曲线积分**
   ：将复杂曲线上的积分转化为区域上的二重积分


### 4.1 计算方法


若曲面 Σ：z = z(x,y)，(x,y) ∈ D：


$$
dS = √(1 + zx² + zy²) dA
$$


$$
∬Σ f dS = ∬D f(x, y, z(x,y)) √(1 + zx² + zy²) dx dy
$$


### 5.1 计算方法


若 Σ：z = z(x,y)，取上侧（**n** 的 z 分量 > 0）：


$$
dS = (−zx, −zy, 1) dx dy
$$


$$
∬Σ F · dS = ∬D (−P zx − Q zy + R) dx dy
$$


### 7.1 散度定理的物理意义


流出闭曲面的总通量 = 曲面内部所有源的总和。


- **流体力学**
   ：流出体积的流体质量 = 体积内源的总产生率
- **电学**
   ：∮
   **E**
   · d
   **S**
   = Q
   ~enclosed~
   /ε₀（高斯定律）

### 格林公式

在平面闭区域 D 上，若 P、Q 具有连续偏导数：

$$\oint_C (P dx + Q dy) = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dA$$

其中 C 是 D 的正向边界曲线。

### 斯托克斯公式

在空间曲面 Σ 上，若 P、Q、R 具有连续偏导数：

$$\oint_C (P dx + Q dy + R dz) = \iint_\Sigma \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right) dy dz + \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right) dz dx + \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dx dy$$

等价形式：∮_C F·dr = ∬_Σ (curl F)·dS

### 散度定理（高斯公式）

$$\oiint_\Sigma (P dy dz + Q dz dx + R dx dy) = \iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right) dV$$

等价形式：∬_Σ F·dS = ∭_Ω div F dV

### 三大公式的关系

```
牛顿-莱布尼茨公式：∫ₐᵇ f'(x)dx = f(b) - f(a)
    ↓ 推广到平面
格林公式：∬_D (∂Q/∂x - ∂P/∂y)dA = ∮_C Pdx + Qdy
    ↓ 推广到空间
斯托克斯公式：∬_Σ curl F·dS = ∮_C F·dr
    ↓ 取闭曲面
散度定理：∭_Ω div F dV = ∮_Σ F·dS
```

这三个公式本质上都是"微积分基本定理"在不同维度的推广：区域内部的积分 = 边界上的积分。

### 计算例题

**例题（第一类曲线积分）**：计算 ∮_C (x²+y²) ds，C 为圆 x²+y²=a²

参数化：x = acost, y = asint, t ∈ [0,2π]
ds = √((-asint)² + (acost)²) dt = a dt

∫₀²π a² · a dt = 2πa³

**例题（格林公式）**：计算 ∮_C (x²-y)dx + (x+y²)dy，C 为 x²+y²=1 的正向

∂Q/∂x = 1, ∂P/∂y = -1
∮ = ∬_D (1-(-1)) dA = 2·π·1² = 2π

### 易错点

1. **第一类曲线积分与方向无关**，第二类曲线积分与方向有关
2. **格林公式要求 P,Q 在 D 内有连续偏导**：有奇点时不能直接用
3. **散度定理要求闭曲面**：非闭曲面需补面
4. **曲面的侧**：第二类曲面积分需确定正侧（法向量方向）


## Python实现

### 格林公式验证

```python
import numpy as np
from scipy.integrate import dblquad, quad
import sympy as sp

x, y = sp.symbols('x y')

# 格林公式: ∮_C (P dx + Q dy) = ∬_D (∂Q/∂x - ∂P/∂y) dA
# P = -y, Q = x, C 为单位圆 (逆时针)
P, Q = -y, x

# 右边: 二重积分
green_integrand = sp.diff(Q, x) - sp.diff(P, y)  # 1 - (-1) = 2
print(f"∂Q/∂x - ∂P/∂y = {green_integrand}")
area = np.pi * 1**2  # 单位圆面积
double_integral = 2 * area
print(f"∬ (∂Q/∂x - ∂P/∂y) dA = {double_integral:.4f} = 2π")

# 左边: 曲线积分 (参数化 x=cos(t), y=sin(t))
# ∮ (-sin(t)(-sin(t)) + cos(t)cos(t)) dt = ∮ 1 dt = 2π
t = sp.Symbol('t')
x_param, y_param = sp.cos(t), sp.sin(t)
dx_dt = sp.diff(x_param, t)
dy_dt = sp.diff(y_param, t)
integrand = P.subs([(x, x_param), (y, y_param)]) * dx_dt + \
            Q.subs([(x, x_param), (y, y_param)]) * dy_dt
line_integral = sp.integrate(integrand, (t, 0, 2*sp.pi))
print(f"∮_C (P dx + Q dy) = {line_integral}")
print(f"格林公式验证: 两边相等")
```

### 数值曲线积分

```python
import numpy as np
from scipy.integrate import quad

# 第一类曲线积分: ∫_C (x^2 + y^2) ds, C 为圆 x²+y²=a²
a = 3
# 参数化: x = a*cos(t), y = a*sin(t), t ∈ [0, 2π]
# ds = a dt
result, _ = quad(lambda t: a**2 * a, 0, 2*np.pi)  # (a²) * a = a³, ∫0^{2π} = 2πa³
print(f"∮_C (x²+y²) ds = {result:.4f} = 2π·27 ≈ {2*np.pi*27:.4f}")

# 向量场的线积分: ∫_C F·dr
# F = (y, -x), C: x=cos(t), y=sin(t), t∈[0,2π]
result, _ = quad(lambda t: np.sin(t)*(-np.sin(t)) + (-np.cos(t))*np.cos(t),
                  0, 2*np.pi)
print(f"∮_C F·dr = {result:.4f} = -2π")

# 验证散度定理: ∬_S F·dS = ∭_Ω div F dV
# F = (x, y, z), Ω 为单位球
# div F = 3, ∭ 3 dV = 3 * (4π/3) = 4π
from scipy.integrate import tplquad
div_integral = 3 * (4/3 * np.pi * 1**3)
print(f"散度定理: ∭ div F dV = {div_integral:.4f} = 4π")
```
