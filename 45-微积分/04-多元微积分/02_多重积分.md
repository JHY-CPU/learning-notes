# 多重积分

*多元微积分 · 二重积分、三重积分与变量替换*


### 1. 定义


$$
∬_D f(x, y) dA = lim Σ f(ξᵢ, ηᵢ) · ΔAᵢ
$$


几何意义：以 D 为底、z = f(x,y) 为顶的曲顶柱体的有向体积。


### 2. 直角坐标系下的计算


**X-型区域**（先 y 后 x）：D: a ≤ x ≤ b, φ₁(x) ≤ y ≤ φ₂(x)


$$
∬_D f(x,y) dA = ∫ₐᵇ [ ∫_{φ₁(x)}^{φ₂(x)} f(x,y) dy ] dx
$$


**Y-型区域**（先 x 后 y）：D: c ≤ y ≤ d, ψ₁(y) ≤ x ≤ ψ₂(y)


$$
∬_D f(x,y) dA = ∫ᶜᵈ [ ∫_{ψ₁(y)}^{ψ₂(y)} f(x,y) dx ] dy
$$


### 3. 极坐标系下的计算


令 x = rcosθ, y = rsinθ，dA = r dr dθ


$$
∬_D f(x,y) dA = ∬ f(rcosθ, rsinθ) · r dr dθ
$$


适用场景：积分区域为圆形、扇形，或被积函数含 x²+y²。


### 定义


$$
∭_Ω f(x, y, z) dV
$$


计算方式：直角坐标（先一后二或先二后一）、柱坐标、球坐标。


### 柱坐标与球坐标


| 坐标系 | 变换 | 体积元素 | 适用场景 |
| --- | --- | --- | --- |
| 柱坐标 | x=rcosθ, y=rsinθ, z=z | dV = r dr dθ dz | 柱对称 |
| 球坐标 | x=ρsinφcosθ, y=ρsinφsinθ, z=ρcosφ | dV = ρ²sinφ dρ dφ dθ | 球对称 |


### 1. 二重积分换元


令 x = x(u, v), y = y(u, v)，则


$$
∬_D f(x,y) dxdy = ∬_{D'} f(x(u,v), y(u,v)) · |J| dudv
$$


其中 J 为**雅可比行列式**：


$$
J = ∂(x,y)/∂(u,v) = | ∂x/∂u   ∂x/∂v |
                                      | ∂y/∂u   ∂y/∂v |
            = (∂x/∂u)(∂y/∂v) - (∂x/∂v)(∂y/∂u)
$$


### 2. 雅可比行列式的性质


- 极坐标变换：J = r（这就是极坐标面积元素多一个 r 的原因）
- 球坐标变换：J = ρ²sinφ
- |J| 是面积（体积）的伸缩因子

### 4. 常见坐标变换的雅可比

| 变换 | 雅可比行列式 | |J| |
|------|-------------|-----|
| 极坐标 (r,θ) → (x,y) | r | r |
| 柱坐标 (r,θ,z) → (x,y,z) | r | r |
| 球坐标 (ρ,φ,θ) → (x,y,z) | ρ²sin φ | ρ²sin φ |

### 5. 二重积分例题

**例题**：计算 ∬_D (x²+y²) dA，D 为圆域 x²+y² ≤ a²

使用极坐标：x = rcosθ, y = rsinθ
$$\int_0^{2\pi} \int_0^a r^2 \cdot r \, dr \, d\theta = \int_0^{2\pi} d\theta \int_0^a r^3 dr = 2\pi \cdot [r^4/4]_0^a = \pi a^4/2$$

**例题**：计算 ∬_D xy dA，D 由 y=x, y=0, x=1 围成

$$\int_0^1 \int_0^x xy \, dy \, dx = \int_0^1 x \cdot [y^2/2]_0^x dx = \int_0^1 x^3/2 dx = [x^4/8]_0^1 = 1/8$$

### 6. 三重积分例题

**例题**：计算 ∭_Ω z dV，Ω 为上半球 x²+y²+z² ≤ a², z ≥ 0

使用球坐标：x = ρsinφcosθ, y = ρsinφsinθ, z = ρcosφ

$$\int_0^{2\pi} \int_0^{\pi/2} \int_0^a (\rho\cos\phi) \cdot \rho^2\sin\phi \, d\rho \, d\phi \, d\theta$$
$$= 2\pi \int_0^{\pi/2} \cos\phi\sin\phi \, d\phi \int_0^a \rho^3 d\rho$$
$$= 2\pi \cdot \frac{1}{2} \cdot \frac{a^4}{4} = \frac{\pi a^4}{4}$$

### 7. 选择坐标系的策略

| 积分区域/被积函数 | 推荐坐标系 |
|-------------------|-----------|
| 矩形、三角形 | 直角坐标 |
| 圆形、扇形、含 x²+y² | 极坐标 |
| 圆柱、旋转体 | 柱坐标 |
| 球形、含 x²+y²+z² | 球坐标 |

### 8. 常见陷阱

1. **极坐标别忘 r**：dA = r dr dθ，不是 dr dθ
2. **球坐标别忘 ρ²sinφ**：dV = ρ²sinφ dρ dφ dθ
3. **积分限要对应坐标系**：极坐标 θ 范围可能是 0 到 2π 或其他
4. **交换积分次序**：需要重新确定积分限，画图辅助


## Python实现

### 二重积分数值计算

```python
import numpy as np
from scipy.integrate import dblquad, tplquad

# 二重积分: ∬_D (x^2 + y^2) dA, D: x²+y² ≤ 1
# 用极坐标: = ∫_0^{2π} ∫_0^1 r^2 * r dr dθ
result, _ = dblquad(lambda y, x: x**2 + y**2,
                     -1, 1,                             # x 的范围
                     lambda x: -np.sqrt(1-x**2),        # y 下限
                     lambda x: np.sqrt(1-x**2))         # y 上限
print(f"∬ (x²+y²) dA = {result:.6f} = π/2 ≈ {np.pi/2:.6f}")

# 极坐标验证
result_polar, _ = dblquad(lambda r, theta: r**2 * r,    # f(r,θ) * r (雅可比)
                            0, 2*np.pi,                   # θ
                            0, 1)                         # r
print(f"极坐标验证: {result_polar:.6f}")

# 二重积分: ∬ xy dA, D 由 y=x, y=0, x=1 围成
result, _ = dblquad(lambda y, x: x*y,
                     0, 1,       # x: 0 to 1
                     0, lambda x: x)  # y: 0 to x
print(f"∬ xy dA = {result:.6f} = 1/8")

# 三重积分: ∭ z dV, 上半球 x²+y²+z² ≤ 1, z ≥ 0
# 用球坐标
result, _ = tplquad(lambda z, y, x: z,
                     -1, 1,                                # x
                     lambda x: -np.sqrt(1-x**2),          # y 下限
                     lambda x: np.sqrt(1-x**2),           # y 上限
                     lambda x, y: 0,                       # z 下限
                     lambda x, y: np.sqrt(1-x**2-y**2))   # z 上限
print(f"∭ z dV = {result:.6f} = π/4 ≈ {np.pi/4:.6f}")
```

### 蒙特卡洛方法计算多重积分

```python
import numpy as np

def monte_carlo_double(f, x_range, y_range, n_samples=100000):
    """蒙特卡洛方法计算二重积分"""
    x = np.random.uniform(x_range[0], x_range[1], n_samples)
    y = np.random.uniform(y_range[0], y_range[1], n_samples)
    area = (x_range[1] - x_range[0]) * (y_range[1] - y_range[0])
    return area * np.mean(f(x, y))

# 示例: ∬_D e^(-x²-y²) dA, D: x²+y² ≤ 4
# 转换为 [-2,2]×[-2,2] 上的积分 (乘以圆内指示函数)
def f(x, y):
    mask = x**2 + y**2 <= 4
    return np.exp(-x**2 - y**2) * mask

result_mc = monte_carlo_double(f, (-2, 2), (-2, 2), 1000000)
exact = np.pi * (1 - np.exp(-4))  # 解析值
print(f"蒙特卡洛: ∬ e^(-x²-y²) dA ≈ {result_mc:.4f}")
print(f"精确值: {exact:.4f}")
```
