# 多元函数极值 - 多元微积分

*Extrema of Multivariable Functions — 多元微积分 · 第五节*


### 1.1 必要条件


> **Theorem:** 若 f(x, y) 在点 (a, b) 处取极值，且 f 在该点可微，则：
>
>
> $$
> fx(a, b) = 0 且 fy(a, b) = 0
> $$
>
>
> 满足此条件的点称为**驻点（临界点 / critical point）**。
>
>
> 推广到 n 元：∇f = **0**。


### 1.2 二阶充分条件——Hessian 矩阵


> **Theorem:** 设 (a, b) 是 f 的驻点，记：
>
>
> $$
> A = fxx(a,b),   B = fxy(a,b),   C = fyy(a,b)
> $$
>
>
> $$
> Hessian 矩阵：
> $$
>
>
> $$
> H = | A   B |
>                         | B   C |
> $$
>
>
> $$
> 判别式：D = AC − B² = det(H)
> $$
>
>
> | 条件 | 结论 |
> | --- | --- |
> | D > 0 且 A > 0 | 极小值 |
> | D > 0 且 A < 0 | 极大值 |
> | D < 0 | 鞍点（非极值） |
> | D = 0 | 需用其他方法判断 |


### 2.1 问题描述


求函数 f(x, y) 在约束条件 g(x, y) = 0 下的极值。


几何意义：在曲线 g = 0 上找 f 的最值点。


### 2.2 多个约束条件


求 f(x, y, z) 在约束 g(x,y,z) = 0 和 h(x,y,z) = 0 下的极值：


$$
∇f = λ∇g + μ∇h
$$


$$
联立 g = 0, h = 0，共五个方程，五个未知数 (x, y, z, λ, μ)
$$


### 3.1 构造拉格朗日函数


$$
L(x, y, λ) = f(x, y) − λ · g(x, y)
$$


将条件极值问题转化为 L 的无条件极值问题：


$$
∂L/∂x = 0 → fx = λgx
$$


$$
∂L/∂y = 0 → fy = λgy
$$


$$
∂L/∂λ = 0 → g = 0
$$


### 4.1 Karush-Kuhn-Tucker（KKT）条件


求 min f(x) 满足 g(x) ≤ 0, h(x) = 0：


$$
构造：L = f(x) + λg(x) + μh(x)
$$


$$
KKT 条件：
$$


$$
① ∇xL = 0（平稳性）
$$


$$
② g(x) ≤ 0, h(x) = 0（原始可行性）
$$


$$
③ λ ≥ 0（对偶可行性）
$$


$$
④ λg(x) = 0（互补松弛性）
$$


互补松弛性：要么约束起作用（g = 0），要么乘数为零（λ = 0）。


### 5.1 求解步骤


1. 求区域
   **内部**
   的驻点（∇f = 0），计算函数值
2. 求区域
   **边界**
   上的最值（参数化边界或拉格朗日乘数法）
3. 比较所有候选点的函数值，最大者为最大值，最小者为最小值


## Python实现

### 多元函数极值的符号求解

```python
import sympy as sp

x, y = sp.symbols('x y')
f = x**3 + y**3 - 3*x*y

# 求驻点: ∇f = 0
fx = sp.diff(f, x)
fy = sp.diff(f, y)
critical_points = sp.solve([fx, fy], [x, y])
print(f"f = {f}")
print(f"∇f = ({fx}, {fy})")
print(f"驻点: {critical_points}")

# Hessian 矩阵判别
fxx = sp.diff(f, x, 2)
fxy = sp.diff(f, x, y)
fyy = sp.diff(f, y, 2)
D = fxx * fyy - fxy**2

for cp in critical_points:
    A = float(fxx.subs([(x, cp[0]), (y, cp[1])]))
    D_val = float(D.subs([(x, cp[0]), (y, cp[1])]))
    f_val = float(f.subs([(x, cp[0]), (y, cp[1])]))
    print(f"\n驻点 ({cp[0]}, {cp[1]}):")
    print(f"  D = AC-B² = {D_val:.1f}, A = {A:.1f}")
    if D_val > 0 and A > 0:
        print(f"  → 极小值, f = {f_val}")
    elif D_val > 0 and A < 0:
        print(f"  → 极大值, f = {f_val}")
    elif D_val < 0:
        print(f"  → 鞍点, f = {f_val}")
```

### 拉格朗日乘数法

```python
import numpy as np
from scipy.optimize import minimize

# 求 f(x,y) = x*y 在约束 x²+y²=1 下的极值
# 拉格朗日函数 L = xy - λ(x²+y²-1)
x, y, lam = sp.symbols('x y lambda')
f = x * y
g = x**2 + y**2 - 1
L = f - lam * g

equations = [sp.diff(L, var) for var in [x, y, lam]]
solutions = sp.solve(equations, [x, y, lam])
print("拉格朗日乘数法求解:")
for sol in solutions:
    f_val = float(f.subs([(x, sol[0]), (y, sol[1])]))
    print(f"  x={sol[0]}, y={sol[1]}, λ={sol[2]}, f={f_val}")

# 数值方法验证
from scipy.optimize import minimize

def objective(vars):
    return -vars[0] * vars[1]  # 取负号转为求最大值

def constraint(vars):
    return vars[0]**2 + vars[1]**2 - 1

from scipy.optimize import NonlinearConstraint
nlc = NonlinearConstraint(constraint, 0, 0)
result = minimize(objective, [0.5, 0.5], constraints={'type': 'eq', 'fun': constraint})
print(f"\n数值方法: x={result.x[0]:.4f}, y={result.x[1]:.4f}, f={-result.fun:.4f}")
```
