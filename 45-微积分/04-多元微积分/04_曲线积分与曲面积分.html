曲线积分与曲面积分

Line Integrals and Surface Integrals — 多元微积分 · 第四节

一、第一型曲线积分（对弧长的曲线积分）

设 C 为平面上的光滑曲线，f 为定义在 C 上的函数：

∫C f(x,y) ds = limλ→0 Σ f(ξi, ηi) Δsi

物理意义：曲线形细杆（线密度 f）的质量。

1.1 计算方法

参数化曲线 C：x = x(t), y = y(t), t ∈ [α, β]，则

∫C f ds = ∫αβ f(x(t), y(t)) √([x′(t)]² + [y′(t)]²) dt

关键：ds = √(dx² + dy²)，积分始终从较小参数到较大参数（不依赖方向）。

二、第二型曲线积分（对坐标的曲线积分）

向量场 F = (P, Q) 沿有向曲线 C 的积分：

∫C F · dr = ∫C P dx + Q dy

物理意义：力 F 沿曲线 C 所做的功。

2.1 计算方法

参数化 C：r(t) = (x(t), y(t), z(t)), t ∈ [α, β]

∫C F · dr = ∫αβ F(r(t)) · r′(t) dt

第一型与第二型的区别：

第一型曲线积分 第二型曲线积分
被积函数 标量函数 f 向量场 F
积分元素 ds（弧长） dr（位移向量）
方向依赖 不依赖方向 依赖方向（反向变号）
物理意义 质量、重心 功、环量

三、格林公式（Green's Theorem）

格林公式

设 D 是由正向简单闭曲线 C 围成的区域，P、Q 在 D 上有连续偏导数：

∮C P dx + Q dy = ∬D (∂Q/∂x − ∂P/∂y) dA

将曲线积分转化为二重积分。

3.1 格林公式的应用

  • 计算平面面积：A = (1/2) ∮C (x dy − y dx)
  • 判断保守场：在单连通区域上，∂P/∂y = ∂Q/∂x ⟺ 积分与路径无关
  • 简化曲线积分：将复杂曲线上的积分转化为区域上的二重积分

例：用格林公式计算 ∮C (x² − y) dx + (y² + x) dy，C 为单位圆（逆时针）

∂Q/∂x = 1, ∂P/∂y = −1

原式 = ∬D (1 − (−1)) dA = 2 ∬D dA = 2π

四、第一型曲面积分（对面积的曲面积分）

设 Σ 为光滑曲面，f 为定义在 Σ 上的函数：

∬Σ f(x,y,z) dS = limλ→0 Σ f(ξi, ηi, ζi) ΔSi

物理意义：曲面薄壳（面密度 f）的质量。

4.1 计算方法

若曲面 Σ：z = z(x,y)，(x,y) ∈ D：

dS = √(1 + zx² + zy²) dA

∬Σ f dS = ∬D f(x, y, z(x,y)) √(1 + zx² + zy²) dx dy

五、第二型曲面积分（对坐标的曲面积分）

向量场 F 在有向曲面 Σ 上的积分（通量/流量）：

∬Σ F · dS = ∬Σ F · n dS

其中 n 为曲面的单位法向量

物理意义：流体穿过曲面 Σ 的流量（flux）。

5.1 计算方法

若 Σ：z = z(x,y)，取上侧（n 的 z 分量 > 0）：

dS = (−zx, −zy, 1) dx dy

∬Σ F · dS = ∬D (−P zx − Q zy + R) dx dy

六、斯托克斯公式（Stokes' Theorem）

斯托克斯公式

设 Σ 为有向曲面，∂Σ 为其正向边界曲线：

∮∂Σ F · dr = ∬Σ (∇ × F) · dS

将空间曲线积分转化为曲面积分。

格林公式是斯托克斯公式在平面情形的特例。

例：验证斯托克斯公式：∮ xy dx + x² dy，C 为矩形边界

curl F = ∇ × (xy, x², 0) = (0, 0, 2x − x) = (0, 0, x)

∬ x dA = ∫₀¹ ∫₀¹ x dy dx = 1/2

七、高斯公式 / 散度定理（Divergence Theorem）

高斯公式（散度定理）

设 Ω 为由闭曲面 Σ 围成的空间区域，F 在 Ω 上有连续偏导数：

∬Σ F · dS = ∭Ω (∇ · F) dV

将闭曲面上的曲面积分转化为三重积分。

7.1 散度定理的物理意义

流出闭曲面的总通量 = 曲面内部所有源的总和。

  • 流体力学：流出体积的流体质量 = 体积内源的总产生率
  • 电学：∮ E · dS = Qenclosed/ε₀（高斯定律）

例：求 ∬Σ (x, y, z) · dS，Σ 为单位球面外侧

∇ · F = 1 + 1 + 1 = 3

∭Ω 3 dV = 3 × (4π/3) = 4π

八、三大定理的统一视角

定理 等式 维度 联系
格林公式 ∮C = ∬D 1D → 2D 边界曲线 ↔ 区域
斯托克斯公式 ∮∂Σ = ∬Σ 1D → 2D 边界曲线 ↔ 曲面
高斯公式 ∬Σ = ∭Ω 2D → 3D 边界曲面 ↔ 体积

这三大定理都是斯托克斯型定理的特例，统一表述为：
"区域上的微分算子的积分 = 边界上的积分"——这是微积分基本定理在高维的推广。

核心要点汇总

  • 第一型（对弧长/面积）：被积函数为标量，积分与方向无关
  • 第二型（对坐标）：被积函数为向量场，积分依赖方向
  • 格林公式：∮C P dx + Q dy = ∬(∂Q/∂x − ∂P/∂y) dA
  • 斯托克斯公式：∮∂Σ F · dr = ∬Σ (∇ × F) · dS
  • 高斯公式（散度定理）：∬Σ F · dS = ∭Ω (∇ · F) dV
  • 三大定理的本质：边界上的积分 = 内部的微分算子的积分
