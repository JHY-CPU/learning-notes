# 向量场与梯度散度旋度 - 多元微积分

*Vector Fields, Gradient, Divergence & Curl — 多元微积分 · 第三节*


### 1.1 相关概念


- **保守场（Conservative Field）**
   ：若存在标量函数 f 使得
   **F**
   = ∇f，则
   **F**
   为保守场
- **势函数（Potential）**
   ：上述函数 f 称为
   **F**
   的势函数
- **无旋场（Irrotational）**
   ：curl
   **F**
   =
   **0**
- **无源场（Solenoidal）**
   ：div
   **F**
   = 0


### 拉普拉斯方程与泊松方程


$$
拉普拉斯方程：∇²f = 0（调和函数）
$$


$$
泊松方程：∇²f = g（源项 g）
$$


应用：热传导方程、波动方程、薛定谔方程、引力势、静电势等都涉及拉普拉斯算子。

### 梯度、散度、旋度的定义与公式

**梯度 (Gradient)**：

$$\nabla f = \frac{\partial f}{\partial x} \mathbf{i} + \frac{\partial f}{\partial y} \mathbf{j} + \frac{\partial f}{\partial z} \mathbf{k}$$

- 梯度是一个向量，指向函数增长最快的方向
- 梯度的模等于该方向上的方向导数最大值
- 等值面的法线方向即为梯度方向

**散度 (Divergence)**：

$$\text{div} \mathbf{F} = \nabla \cdot \mathbf{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$$

其中 F = Pi + Qj + Rk

- 散度是一个标量
- 散度 > 0：该点是"源"（向外发散）
- 散度 < 0：该点是"汇"（向内汇聚）
- 散度 = 0：无源场（不可压缩流体）

**旋度 (Curl)**：

$$\text{curl} \mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \partial/\partial x & \partial/\partial y & \partial/\partial z \\ P & Q & R \end{vmatrix}$$

- 旋度是一个向量
- 旋度的方向表示旋转轴方向（右手定则）
- 旋度的模表示旋转的强度
- 旋度 = 0：无旋场（保守场）

### 保守场的判定

向量场 F 为保守场的等价条件：

1. 存在势函数 f 使 F = ∇f
2. 沿任意闭合曲线的线积分为零：∮ F·dr = 0
3. 积分与路径无关
4. curl F = 0（在单连通区域上）

### 常用向量恒等式

```
∇·(∇×F) = 0                    散度的旋度恒为零
∇×(∇f) = 0                     梯度的旋度恒为零
∇²f = ∇·(∇f)                   拉普拉斯 = 散度的梯度
∇·(fF) = f∇·F + F·∇f           乘积的散度
∇×(fF) = f∇×F + (∇f)×F         乘积的旋度
∇·(F×G) = G·(∇×F) - F·(∇×G)    叉乘的散度
```

### 物理应用

| 概念 | 物理意义 | 应用领域 |
|------|---------|---------|
| 梯度 | 势能的空间变化率 | 重力场、电场、温度场 |
| 散度 | 源/汇的密度 | 流体力学、电磁学（高斯定律） |
| 旋度 | 旋转/涡旋强度 | 流体力学（涡量）、电磁学（法拉第定律） |
| 拉普拉斯 | 稳态分布 | 热传导、静电势、波动 |

### 易错点

1. **散度是标量，旋度是向量**：不要搞混
2. **保守场要求单连通**：R³ 中挖去原点的区域不是单连通的，curl F = 0 不保证保守
3. **拉普拉斯算子 ∇² = Δ**：两种记号等价
4. **梯度只对标量函数定义**：不能对向量场求梯度


## Python实现

### 梯度、散度、旋度的符号计算

```python
import sympy as sp

x, y, z = sp.symbols('x y z')

# 标量场的梯度
f = x**2 * y + sp.sin(z)
grad_f = [sp.diff(f, var) for var in [x, y, z]]
print(f"f = {f}")
print(f"∇f = ({grad_f[0]}, {grad_f[1]}, {grad_f[2]})")

# 向量场的散度
P = x*y
Q = y*z
R = z*x
div_F = sp.diff(P, x) + sp.diff(Q, y) + sp.diff(R, z)
print(f"\nF = (xy, yz, zx)")
print(f"div F = ∇·F = {div_F}")

# 向量场的旋度
curl_F = [
    sp.diff(R, y) - sp.diff(Q, z),  # ∂R/∂y - ∂Q/∂z
    sp.diff(P, z) - sp.diff(R, x),  # ∂P/∂z - ∂R/∂x
    sp.diff(Q, x) - sp.diff(P, y)   # ∂Q/∂x - ∂P/∂y
]
print(f"curl F = ∇×F = ({curl_F[0]}, {curl_F[1]}, {curl_F[2]})")

# 拉普拉斯算子: ∇²f = ∂²f/∂x² + ∂²f/∂y² + ∂²f/∂z²
laplacian = sum(sp.diff(f, var, 2) for var in [x, y, z])
print(f"∇²f = {sp.simplify(laplacian)}")

# 验证恒等式: ∇×(∇f) = 0
curl_grad = [
    sp.diff(grad_f[2], y) - sp.diff(grad_f[1], z),
    sp.diff(grad_f[0], z) - sp.diff(grad_f[2], x),
    sp.diff(grad_f[1], x) - sp.diff(grad_f[0], y)
]
print(f"\n∇×(∇f) = ({sp.simplify(curl_grad[0])}, {sp.simplify(curl_grad[1])}, {sp.simplify(curl_grad[2])})")
print("验证: ∇×(∇f) = 0")
```

### 保守场验证

```python
import sympy as sp

x, y = sp.symbols('x y')

# 向量场 F = (2xy, x^2)
P, Q = 2*x*y, x**2

# 检查是否保守: ∂P/∂y = ∂Q/∂x
dP_dy = sp.diff(P, y)
dQ_dx = sp.diff(Q, x)
print(f"∂P/∂y = {dP_dy}, ∂Q/∂x = {dQ_dx}")
print(f"保守场? {dP_dy == dQ_dx}")

# 求势函数: φ = ∫P dx = x^2*y + g(y)
# ∂φ/∂y = x^2 + g'(y) = Q = x^2 → g'(y) = 0 → g(y) = C
phi = x**2 * y
print(f"势函数 φ = {phi}")
print(f"∇φ = ({sp.diff(phi, x)}, {sp.diff(phi, y)}) = F ?")
```
