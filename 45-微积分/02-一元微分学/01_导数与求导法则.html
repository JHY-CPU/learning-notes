导数与求导法则

一元微分学 · 变化率的核心概念

一、导数的定义

1. 定义

函数 y = f(x) 在点 x₀ 处的导数定义为：

f'(x₀) = lim(Δx→0) [f(x₀+Δx) - f(x₀)] / Δx = lim(Δx→0) Δy / Δx

几何意义：曲线 y = f(x) 在点 (x₀, f(x₀)) 处切线的斜率。

2. 可导与连续的关系

  • 可导 ⟹ 连续（可导一定连续）
  • 连续 ⟹ 可导（连续不一定可导，如 y = |x| 在 x=0）

3. 左导数与右导数

f'₋(x₀) = lim(Δx→0⁻) [f(x₀+Δx) - f(x₀)] / Δx
f'₊(x₀) = lim(Δx→0⁺) [f(x₀+Δx) - f(x₀)] / Δx

f'(x₀) 存在 ⟺ f'₋(x₀) = f'₊(x₀)

二、基本求导公式

函数导数函数导数
(C)' = 0(xⁿ)' = nxⁿ⁻¹
(sinx)' = cosx(cosx)' = -sinx
(tanx)' = sec²x(cotx)' = -csc²x
(eˣ)' = eˣ(aˣ)' = aˣ·ln a
(ln x)' = 1/x(log_a x)' = 1/(x·ln a)
(arcsin x)' = 1/√(1-x²)(arctan x)' = 1/(1+x²)

三、求导法则

1. 四则运算法则

[u ± v]' = u' ± v'

[u · v]' = u'v + uv'      （乘法法则）

[u/v]' = (u'v - uv') / v²      （商法则, v ≠ 0）

2. 链式法则（复合函数求导）

若 y = f(u), u = g(x)，则：

dy/dx = (dy/du) · (du/dx) = f'(u) · g'(x) = f'(g(x)) · g'(x)

核心：由外向内，逐层求导，相乘。

3. 隐函数求导

方程 F(x, y) = 0 确定了隐函数 y = y(x)。对方程两边同时对 x 求导，注意 y 是 x 的函数。

4. 参数方程求导

设 x = φ(t), y = ψ(t)，则：

dy/dx = (dy/dt) / (dx/dt) = ψ'(t) / φ'(t)

d²y/dx² = [d(dy/dx)/dt] / (dx/dt)
例题 1：链式法则

求 y = sin(x² + 1) 的导数。

令 u = x² + 1，则 y = sin u
dy/dx = cos(u) · du/dx = cos(x²+1) · 2x = 2x·cos(x²+1)
例题 2：隐函数求导

设 x² + y² = 25，求 dy/dx。

两边对 x 求导：2x + 2y·(dy/dx) = 0
dy/dx = -x/y
例题 3：参数方程求导

设 x = acost, y = bsint，求 dy/dx 和 d²y/dx²。

dy/dt = bcost, dx/dt = -asint
dy/dx = bcost/(-asint) = -b·cott/a

d²y/dx² = [d(-bcott/a)/dt] / (-asint)
         = [b·csc²t/a] / (-asint)
         = -b/(a²·sin³t)
求导技巧总结：
  • 先化简再求导（如分式化简、对数化简）
  • 对数求导法：对 y = f(x) 两边取 ln，再求导（适合幂指函数、连乘/连除）
  • 隐函数求导：对等式两边同时对 x 求导，y 视为 x 的函数
  • 参数方程求导：dy/dx = (dy/dt)/(dx/dt)

本节总结

  • 导数定义：f'(x₀) = lim Δy/Δx，几何意义是切线斜率
  • 可导 ⟹ 连续，连续 ⟹ 可导
  • 四则法则：(uv)' = u'v + uv'，(u/v)' = (u'v-uv')/v²
  • 链式法则：(f(g(x)))' = f'(g(x))·g'(x)
  • 隐函数求导：等式两边对 x 求导
  • 参数方程：dy/dx = (dy/dt)/(dx/dt)
