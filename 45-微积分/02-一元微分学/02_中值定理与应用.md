# 中值定理与应用

*一元微分学 · 微分学核心定理*


### 定理内容


若 f(x) 满足：


1. 在闭区间 [a, b] 上连续
2. 在开区间 (a, b) 内可导
3. f(a) = f(b)


则存在 ξ ∈ (a, b)，使得 **f'(ξ) = 0**。


$$
几何意义：在弧 AB 上至少有一点处的切线平行于 x 轴。
$$


### 定理内容


若 f(x) 在 [a, b] 上连续，在 (a, b) 内可导，则存在 ξ ∈ (a, b)，使得：


$$
f'(ξ) = [f(b) - f(a)] / (b - a)

            即 f(b) - f(a) = f'(ξ)(b - a)
$$


几何意义：弧 AB 上至少有一点的切线平行于弦 AB。


罗尔定理是拉格朗日定理在 f(a)=f(b) 时的特例。


### 重要推论


- 若在 (a,b) 上 f'(x) ≡ 0，则 f(x) 在 [a,b] 上为常数
- 若在 (a,b) 上 f'(x) = g'(x)，则 f(x) = g(x) + C
- 若 f'(x) > 0，则 f(x) 严格递增；f'(x) < 0 则严格递减


### 定理内容


若 f(x), g(x) 在 [a, b] 上连续，在 (a, b) 内可导，且 g'(x) ≠ 0，则存在 ξ ∈ (a, b) 使得：


$$
[f(b) - f(a)] / [g(b) - g(a)] = f'(ξ) / g'(ξ)
$$


令 g(x) = x 即得拉格朗日中值定理。


### 适用条件与公式


适用于 0/0 型或 ∞/∞ 型不定式：


$$
若 lim f(x) = lim g(x) = 0（或 ±∞），且 lim f'(x)/g'(x) 存在，则

            lim f(x)/g(x) = lim f'(x)/g'(x)
$$


其他不定式（0·∞, ∞-∞, 0⁰, 1^∞, ∞⁰）需先转化为 0/0 或 ∞/∞ 型。


### 泰勒展开


若 f(x) 在 x₀ 处有 n 阶导数，则：


$$
f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)/2! · (x-x₀)² + ... + f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ + Rₙ(x)
$$


当 x₀ = 0 时称为**麦克劳林公式**。


### 常用麦克劳林展开


| f(x) | 展开式 |
| --- | --- |
| eˣ | 1 + x + x²/2! + x³/3! + ... + xⁿ/n! |
| sin x | x - x³/3! + x⁵/5! - ... + (-1)ⁿx²ⁿ⁺¹/(2n+1)! |
| cos x | 1 - x²/2! + x⁴/4! - ... + (-1)ⁿx²ⁿ/(2n)! |
| ln(1+x) | x - x²/2 + x³/3 - ... + (-1)ⁿ⁻¹xⁿ/n |
| 1/(1-x) | 1 + x + x² + x³ + ... + xⁿ |
| (1+x)^α | 1 + αx + α(α-1)x²/2! + ... |


## Python实现

### 洛必达法则与泰勒展开

```python
import sympy as sp

x = sp.Symbol('x')

# 洛必达法则: 0/0 型
print("=== 洛必达法则 ===")
# lim(x→0) sin(x)/x
f, g = sp.sin(x), x
print(f"lim sin(x)/x = {sp.limit(f/g, x, 0)}")

# lim(x→0) (e^x - 1)/x
f, g = sp.exp(x) - 1, x
print(f"lim (e^x-1)/x = {sp.limit(f/g, x, 0)}")

# ∞/∞ 型: lim(x→∞) x^2 / e^x
print(f"lim x^2/e^x = {sp.limit(x**2 / sp.exp(x), x, sp.oo)}")

# 0*∞ 型: lim(x→0+) x*ln(x)
print(f"lim x*ln(x) = {sp.limit(x*sp.ln(x), x, 0, '+')}")
```

### 泰勒展开 (Taylor Series)

```python
import sympy as sp
import numpy as np

x = sp.Symbol('x')

# 常用函数的麦克劳林展开
print("=== 麦克劳林级数 ===")
for name, func in [("e^x", sp.exp(x)),
                    ("sin(x)", sp.sin(x)),
                    ("cos(x)", sp.cos(x)),
                    ("ln(1+x)", sp.ln(1+x))]:
    series = sp.series(func, x, 0, 6)
    print(f"{name} = {series}")

# 泰勒展开的数值验证
print("\n=== 泰勒近似精度 ===")
# sin(x) 在 x=1 处用不同阶数近似
x_val = 1.0
exact = np.sin(x_val)
for n in [3, 5, 7, 9]:
    approx = float(sp.sin(x).series(x, 0, n).removeO().subs(x, x_val))
    print(f"sin(1) ≈ {approx:.10f} (n={n}), 误差 = {abs(approx - exact):.2e}")
```

### 中值定理的数值验证

```python
import numpy as np
from scipy.optimize import fsolve

# 拉格朗日中值定理: f(b) - f(a) = f'(ξ)(b-a)
# 示例: f(x) = x^3 - 3x, [0, 2]
f = lambda x: x**3 - 3*x
df = lambda x: 3*x**2 - 3

a, b = 0, 2
slope = (f(b) - f(a)) / (b - a)
print(f"弦 AB 的斜率 = {slope:.4f}")

# 找 ξ 使 f'(ξ) = slope
xi = fsolve(lambda x: df(x) - slope, 1.0)[0]
print(f"ξ = {xi:.6f}, f'(ξ) = {df(xi):.4f}")
print(f"验证 f'(ξ) = 弦斜率: {abs(df(xi) - slope) < 1e-10}")

# 可视化
import matplotlib.pyplot as plt
x = np.linspace(-0.5, 2.5, 300)
plt.plot(x, f(x), 'b-', label='f(x) = x³ - 3x')
plt.plot([a, b], [f(a), f(b)], 'r--', label='弦 AB')
plt.plot(xi, f(xi), 'ro', markersize=8)
plt.axhline(y=0, color='gray', linewidth=0.5)
plt.legend()
plt.title(f"拉格朗日中值定理: ξ ≈ {xi:.4f}")
plt.savefig('lagrange_mvt.png', dpi=100)
plt.show()
```
