# 导数应用 - 一元微分学

*Applications of Derivatives — 一元微分学 · 第三节*


### 1.1 基本思想


当两个或多个变量通过某个方程关联，且它们都是时间 *t* 的函数时，对等式两边关于 *t* 求导，得到变化率之间的关系。


1. 建立变量之间的几何/物理关系式
2. 对等式两边关于时间
   *t*
   求导（链式法则）
3. 代入已知瞬时值，求解未知变化率


### 2.1 线性近似公式


$$
函数 f 在点 a 处的线性近似（切线近似）：
$$


$$
f(x) ≈ f(a) + f′(a)(x − a)
$$


当 x 接近 a 时，用切线代替曲线


### 2.2 微分记号


$$
dy = f′(x) dx
$$


$$
Δy ≈ dy = f′(x) · Δx
$$


其中 dx = Δx 是自变量的微分（增量），dy 是因变量的**微分**（线性主部）。


### 4.1 单调性判定


$$
在区间 I 上：
        f′(x) > 0 ⟹ f(x) 单调递增
        f′(x) < 0 ⟹ f(x) 单调递减
$$


### 4.2 凹凸性与拐点


$$
在区间 I 上：
f″(x) > 0 ⟹ f(x) 凹（下凸 ⌣）
f″(x) < 0 ⟹ f(x) 凸（上凸 ⌢）
$$


$$
凹凸性改变的点称为拐点（Inflection Point）
$$


### 4.3 极值判别法


| 方法 | 条件 | 结论 |
| --- | --- | --- |
| 一阶导数法 | f′ 在 a 左正右负 / 左负右正 | 极大值 / 极小值 |
| 二阶导数法 | f′(a) = 0, f″(a) < 0 / > 0 | 极大值 / 极小值 |
| 高阶导数法 | f′(a) = f″(a) = … = f^(n-1)^(a) = 0, f^(n)^(a) ≠ 0 | n 为偶数则极值，n 为奇数则拐点 |


### 4.4 渐近线


| 类型 | 条件 | 方程 |
| --- | --- | --- |
| 垂直渐近线 | lim~x→a~ f(x) = ±∞ | x = a |
| 水平渐近线 | lim~x→±∞~ f(x) = b | y = b |
| 斜渐近线 | lim~x→±∞~ [f(x) − (ax+b)] = 0 | y = ax + b |


斜渐近线系数：**a** = lim~x→∞~ f(x)/x，**b** = lim~x→∞~ [f(x) − ax]


1. **定义域**
   ：确定函数的定义域，检查对称性（奇偶性）、周期性
2. **一阶导数**
   ：求 f′(x)，找驻点和不可导点，判断单调区间
3. **极值**
   ：利用驻点计算极值
4. **二阶导数**
   ：求 f″(x)，找拐点，判断凹凸区间
5. **渐近线**
   ：确定垂直、水平或斜渐近线
6. **特殊点**
   ：计算与坐标轴交点
7. **描绘**
   ：综合以上信息画出曲线


## Python实现

### 极值与单调性分析

```python
import numpy as np
from scipy.optimize import minimize_scalar, brentq
import matplotlib.pyplot as plt

# f(x) = x^3 - 6x^2 + 9x + 1
f = lambda x: x**3 - 6*x**2 + 9*x + 1
df = lambda x: 3*x**2 - 12*x + 9    # f'(x) = 3(x-1)(x-3)
d2f = lambda x: 6*x - 12             # f''(x)

# 求驻点: f'(x) = 0
from sympy import symbols, diff, solve, factor
x = symbols('x')
f_sym = x**3 - 6*x**2 + 9*x + 1
df_sym = diff(f_sym, x)
d2f_sym = diff(f_sym, x, 2)
critical_points = solve(df_sym, x)
print(f"f'(x) = {factor(df_sym)}")
print(f"驻点: {critical_points}")

# 二阶导数判别法
for cp in critical_points:
    d2val = float(d2f_sym.subs(x, cp))
    fval = float(f_sym.subs(x, cp))
    if d2val > 0:
        print(f"x={cp}: f''={d2val} > 0 → 极小值 f({cp})={fval}")
    elif d2val < 0:
        print(f"x={cp}: f''={d2val} < 0 → 极大值 f({cp})={fval}")

# 拐点: f''(x) = 0 且 f'' 变号
inflection = solve(d2f_sym, x)
print(f"拐点: x = {inflection}")

# 可视化
x_plot = np.linspace(-1, 5, 300)
fig, axes = plt.subplots(1, 2, figsize=(12, 5))
axes[0].plot(x_plot, f(x_plot), 'b-', label='f(x)')
axes[0].plot(x_plot, df(x_plot), 'r--', label="f'(x)")
axes[0].set_title('函数与导数')
axes[0].legend()
axes[0].grid(True)

# 渐近线分析
# f(x) = x/(x-1): 垂直渐近线 x=1, 水平渐近线 y=1
x_asym = np.linspace(-3, 5, 500)
x_asym = x_asym[abs(x_asym - 1) > 0.05]
axes[1].plot(x_asym, x_asym/(x_asym-1), 'b-')
axes[1].axvline(x=1, color='red', linestyle='--', label='x=1 (垂直渐近线)')
axes[1].axhline(y=1, color='green', linestyle='--', label='y=1 (水平渐近线)')
axes[1].set_ylim(-10, 10)
axes[1].set_title('渐近线分析')
axes[1].legend()
plt.tight_layout()
plt.savefig('derivative_applications.png', dpi=100)
plt.show()
```
