# 导数与求导法则

*一元微分学 · 变化率的核心概念*


### 1. 定义


函数 y = f(x) 在点 x₀ 处的导数定义为：


$$
f'(x₀) = lim(Δx→0) [f(x₀+Δx) - f(x₀)] / Δx = lim(Δx→0) Δy / Δx
$$


几何意义：曲线 y = f(x) 在点 (x₀, f(x₀)) 处切线的斜率。


### 2. 可导与连续的关系


- **可导 ⟹ 连续**
   （可导一定连续）
- **连续 ⟹ 可导**
   （连续不一定可导，如 y = |x| 在 x=0）


### 3. 左导数与右导数


$$
f'₋(x₀) = lim(Δx→0⁻) [f(x₀+Δx) - f(x₀)] / Δx

            f'₊(x₀) = lim(Δx→0⁺) [f(x₀+Δx) - f(x₀)] / Δx

            f'(x₀) 存在 ⟺ f'₋(x₀) = f'₊(x₀)
$$


| 函数 | 导数 | 函数 | 导数 |
| --- | --- | --- | --- |
| (C)' = | 0 | (xⁿ)' = | nxⁿ⁻¹ |
| (sinx)' = | cosx | (cosx)' = | -sinx |
| (tanx)' = | sec²x | (cotx)' = | -csc²x |
| (eˣ)' = | eˣ | (aˣ)' = | aˣ·ln a |
| (ln x)' = | 1/x | (log_a x)' = | 1/(x·ln a) |
| (arcsin x)' = | 1/√(1-x²) | (arctan x)' = | 1/(1+x²) |


### 1. 四则运算法则


$$
[u ± v]' = u' ± v'

            [u · v]' = u'v + uv'      （乘法法则）

            [u/v]' = (u'v - uv') / v²      （商法则, v ≠ 0）
$$


### 2. 链式法则（复合函数求导）


若 y = f(u), u = g(x)，则：


$$
dy/dx = (dy/du) · (du/dx) = f'(u) · g'(x) = f'(g(x)) · g'(x)
$$


核心：**由外向内，逐层求导，相乘**。


### 3. 隐函数求导


方程 F(x, y) = 0 确定了隐函数 y = y(x)。对方程两边同时对 x 求导，注意 y 是 x 的函数。


### 4. 参数方程求导


设 x = φ(t), y = ψ(t)，则：


$$
dy/dx = (dy/dt) / (dx/dt) = ψ'(t) / φ'(t)

            d²y/dx² = [d(dy/dx)/dt] / (dx/dt)
$$

### 5. 高阶导数

$$
f''(x) = d²y/dx² = d/dx[dy/dx]
$$

常见函数的高阶导数公式：

| 函数 | n阶导数 |
|------|---------|
| xⁿ | n! |
| eˣ | eˣ |
| aˣ | aˣ·(ln a)ⁿ |
| ln(x) | (-1)ⁿ⁻¹·(n-1)!/xⁿ |
| sin(x) | sin(x + nπ/2) |
| cos(x) | cos(x + nπ/2) |

**莱布尼茨公式**：(uv)⁽ⁿ⁾ = Σ C(n,k) · u⁽ⁿ⁻ᵏ⁾ · v⁽ᵏ⁾

### 6. 导数的应用

**单调性判断**：
- f'(x) > 0 → f(x) 在该区间单调递增
- f'(x) < 0 → f(x) 在该区间单调递减

**极值判断**：
- 第一充分条件：f'(x₀)=0，f' 在 x₀ 左右变号 → x₀ 为极值点
- 第二充分条件：f'(x₀)=0 且 f''(x₀)≠0
  - f''(x₀) > 0 → 极小值
  - f''(x₀) < 0 → 极大值

**凹凸性与拐点**：
- f''(x) > 0 → 凹（开口向上）
- f''(x) < 0 → 凸（开口向下）
- f''(x₀)=0 且 f'' 在 x₀ 左右变号 → (x₀, f(x₀)) 为拐点

### 7. 中值定理

**罗尔定理**：f 在 [a,b] 连续，(a,b) 可导，f(a)=f(b)，则 ∃ξ∈(a,b) 使 f'(ξ)=0

**拉格朗日中值定理**：f 在 [a,b] 连续，(a,b) 可导，则 ∃ξ∈(a,b) 使：
$$f'(ξ) = [f(b)-f(a)]/(b-a)$$

**柯西中值定理**：f,g 在 [a,b] 连续，(a,b) 可导，g'(x)≠0，则 ∃ξ∈(a,b) 使：
$$[f(b)-f(a)]/[g(b)-g(a)] = f'(ξ)/g'(ξ)$$

### 8. 洛必达法则

对于 0/0 或 ∞/∞ 型未定式：

$$\lim_{x→a} f(x)/g(x) = \lim_{x→a} f'(x)/g'(x)$$

其他未定式需先转化为 0/0 或 ∞/∞：
- 0·∞ → 化为 0/(1/∞)
- ∞-∞ → 通分或有理化
- 0⁰, 1^∞, ∞⁰ → 取对数

### 9. 常见易错点

1. **可导与连续**：可导一定连续，连续不一定可导（如 |x| 在 x=0）
2. **链式法则的�数**：复合多少层就乘多少个导数
3. **隐函数求导**：y 是 x 的函数，对 y 的函数求导时要乘 dy/dx
4. **参数方程二阶导**：d²y/dx² ≠ ψ''(t)/φ''(t)，需用 d(dy/dx)/dt 除以 dx/dt


## Python实现

### 符号求导 (SymPy)

```python
import sympy as sp

x = sp.Symbol('x')

# 基本求导
print("=== 基本求导法则 ===")
print(f"(x^3)' = {sp.diff(x**3, x)}")
print(f"(sin(x))' = {sp.diff(sp.sin(x), x)}")
print(f"(e^x)' = {sp.diff(sp.exp(x), x)}")
print(f"(ln(x))' = {sp.diff(sp.ln(x), x)}")
print(f"(arctan(x))' = {sp.diff(sp.atan(x), x)}")

# 乘法法则: (x^2 * sin(x))'
print(f"\n=== 乘法法则 ===")
print(f"(x^2 * sin(x))' = {sp.diff(x**2 * sp.sin(x), x)}")

# 链式法则: (sin(x^2))'
print(f"\n=== 链式法则 ===")
print(f"(sin(x^2))' = {sp.diff(sp.sin(x**2), x)}")

# 高阶导数
print(f"\n=== 高阶导数 ===")
print(f"(sin(x))'' = {sp.diff(sp.sin(x), x, 2)}")
print(f"(sin(x))''' = {sp.diff(sp.sin(x), x, 3)}")
print(f"(e^x)^(n) = {sp.diff(sp.exp(x), x, 5)}")  # 仍为 e^x

# 隐函数求导: x^2 + y^2 = 1 → dy/dx = -x/y
print(f"\n=== 隐函数求导 ===")
y = sp.Function('y')(x)
eq = x**2 + y**2 - 1
dy_dx = -sp.diff(eq, x) / sp.diff(eq, y)
print(f"x^2 + y^2 = 1 的 dy/dx = {sp.simplify(dy_dx)}")
```

### 数值求导

```python
import numpy as np

def numerical_derivative(f, x, h=1e-7):
    """中心差分法数值求导"""
    return (f(x + h) - f(x - h)) / (2 * h)

# 示例
f = lambda x: np.sin(x**2)
x_val = 1.0
approx = numerical_derivative(f, x_val)
exact = 2 * x_val * np.cos(x_val**2)  # 解析导数
print(f"f'(1) ≈ {approx:.10f}, 精确值 = {exact:.10f}")
print(f"误差 = {abs(approx - exact):.2e}")

# 验证可导与连续的关系
# y = |x| 在 x=0 处连续但不可导
x_vals = np.linspace(-0.1, 0.1, 1000)
f_abs = lambda x: abs(x)
left_deriv = (f_abs(0 - 1e-7) - f_abs(0)) / (-1e-7)   # ≈ -1
right_deriv = (f_abs(0 + 1e-7) - f_abs(0)) / (1e-7)   # ≈ +1
print(f"\n|x| 在 x=0: 左导数 ≈ {left_deriv:.1f}, 右导数 ≈ {right_deriv:.1f}")
print(f"左右导数不等, 不可导")
```
