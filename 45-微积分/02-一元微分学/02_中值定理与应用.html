中值定理与应用

一元微分学 · 微分学核心定理

一、罗尔定理（Rolle）

定理内容

若 f(x) 满足：

  1. 在闭区间 [a, b] 上连续
  2. 在开区间 (a, b) 内可导
  3. f(a) = f(b)

则存在 ξ ∈ (a, b)，使得 f'(ξ) = 0。

几何意义：在弧 AB 上至少有一点处的切线平行于 x 轴。

二、拉格朗日中值定理（Lagrange MVT）

定理内容

若 f(x) 在 [a, b] 上连续，在 (a, b) 内可导，则存在 ξ ∈ (a, b)，使得：

f'(ξ) = [f(b) - f(a)] / (b - a)

即 f(b) - f(a) = f'(ξ)(b - a)

几何意义：弧 AB 上至少有一点的切线平行于弦 AB。

罗尔定理是拉格朗日定理在 f(a)=f(b) 时的特例。

重要推论

  • 若在 (a,b) 上 f'(x) ≡ 0，则 f(x) 在 [a,b] 上为常数
  • 若在 (a,b) 上 f'(x) = g'(x)，则 f(x) = g(x) + C
  • 若 f'(x) > 0，则 f(x) 严格递增；f'(x) < 0 则严格递减
例题 1：拉格朗日中值定理

设 f(x) = x³ - 3x，在 [0, 2] 上验证拉格朗日中值定理并求 ξ。

f'(x) = 3x² - 3
f(2) - f(0) = (8-6) - 0 = 2
f'(ξ) = [f(2)-f(0)]/(2-0) = 2/2 = 1
3ξ² - 3 = 1 → ξ² = 4/3 → ξ = 2/√3 ∈ (0,2) ✓

三、柯西中值定理

定理内容

若 f(x), g(x) 在 [a, b] 上连续，在 (a, b) 内可导，且 g'(x) ≠ 0，则存在 ξ ∈ (a, b) 使得：

[f(b) - f(a)] / [g(b) - g(a)] = f'(ξ) / g'(ξ)

令 g(x) = x 即得拉格朗日中值定理。

四、洛必达法则（L'Hopital）

适用条件与公式

适用于 0/0 型或 ∞/∞ 型不定式：

若 lim f(x) = lim g(x) = 0（或 ±∞），且 lim f'(x)/g'(x) 存在，则

lim f(x)/g(x) = lim f'(x)/g'(x)

其他不定式（0·∞, ∞-∞, 0⁰, 1^∞, ∞⁰）需先转化为 0/0 或 ∞/∞ 型。

例题 2：洛必达法则

求 lim(x→0) (eˣ - 1 - x) / x²。

0/0 型，使用洛必达：
= lim (eˣ - 1) / (2x)   （仍为 0/0）
= lim eˣ / 2 = 1/2
例题 3：∞-∞ 型

求 lim(x→0⁺) (1/x - 1/sinx)。

通分：= lim (sinx - x) / (x·sinx)   （0/0 型）
= lim (cosx - 1) / (sinx + x·cosx)   （0/0）
= lim (-sinx) / (cosx + cosx - x·sinx)
= lim (-sinx) / (2cosx - x·sinx)
= 0 / 2 = 0

五、泰勒公式（Taylor）

泰勒展开

若 f(x) 在 x₀ 处有 n 阶导数，则：

f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)/2! · (x-x₀)² + ... + f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ + Rₙ(x)

当 x₀ = 0 时称为麦克劳林公式。

例题 4：泰勒展开求极限

求 lim(x→0) (cosx - e^(-x²/2)) / x⁴。

cosx = 1 - x²/2 + x⁴/24 + o(x⁴)
e^(-x²/2) = 1 + (-x²/2) + (-x²/2)²/2 + o(x⁴) = 1 - x²/2 + x⁴/8 + o(x⁴)

分子 = (1 - x²/2 + x⁴/24) - (1 - x²/2 + x⁴/8) + o(x⁴)
      = x⁴(1/24 - 1/8) + o(x⁴) = -x⁴/12 + o(x⁴)

lim = -1/12
三大中值定理的关系：
  • 罗尔定理：拉格朗日定理的特例（f(a)=f(b)）
  • 拉格朗日定理：柯西定理的特例（g(x)=x）
  • 柯西定理：是洛必达法则证明的基础
  • 泰勒公式：拉格朗日中值定理的高阶推广

本节总结

  • 罗尔定理：f(a)=f(b) → f'(ξ)=0
  • 拉格朗日中值定理：f'(ξ) = [f(b)-f(a)]/(b-a)
  • 柯西中值定理：[f(b)-f(a)]/[g(b)-g(a)] = f'(ξ)/g'(ξ)
  • 洛必达法则：0/0 或 ∞/∞ 型，lim f/g = lim f'/g'
  • 泰勒公式：用多项式逼近函数，常用展开要熟记
