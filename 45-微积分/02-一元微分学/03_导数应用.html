导数应用

Applications of Derivatives — 一元微分学 · 第三节

一、相关变化率（Related Rates）

1.1 基本思想

当两个或多个变量通过某个方程关联，且它们都是时间 t 的函数时，对等式两边关于 t 求导，得到变化率之间的关系。

  1. 建立变量之间的几何/物理关系式
  2. 对等式两边关于时间 t 求导（链式法则）
  3. 代入已知瞬时值，求解未知变化率

若 F(x, y) = 0，且 x = x(t), y = y(t)，则

∂F/∂x · dx/dt + ∂F/∂y · dy/dt = 0

隐函数求导法推导出变化率关系

问题：一个半径为 r 的球体，水以恒定速率 dV/dt = 5 cm³/s 注入，当 r = 10 cm 时，半径的增长速率 dr/dt 是多少？

V = (4/3)πr³

dV/dt = 4πr² · dr/dt

5 = 4π(10)² · dr/dt

dr/dt = 1/(80π) ≈ 0.00398 cm/s

问题：一架飞机以 500 km/h 的速度在 3 km 高空水平飞行，求飞机飞过观察者正上方 4 km 时，与观察者距离的变化率。

设飞机水平距离为 x，与观察者距离为 s，高度 h = 3

s² = x² + h²

2s · ds/dt = 2x · dx/dt

当 x = 4 时，s = 5

ds/dt = (x/s)·(dx/dt) = (4/5)(500) = 400 km/h

二、线性近似与微分（Linear Approximation）

2.1 线性近似公式

函数 f 在点 a 处的线性近似（切线近似）：

f(x) ≈ f(a) + f′(a)(x − a)

当 x 接近 a 时，用切线代替曲线

2.2 微分记号

dy = f′(x) dx

Δy ≈ dy = f′(x) · Δx

其中 dx = Δx 是自变量的微分（增量），dy 是因变量的微分（线性主部）。

例：估算 √4.02

f(x) = √x, a = 4, f(4) = 2, f′(x) = 1/(2√x), f′(4) = 1/4

√4.02 ≈ 2 + (1/4)(0.02) = 2.005

（精确值 ≈ 2.00499…）

常见近似（当 |x| 很小时）：
(1+x)n ≈ 1 + nx,   sin x ≈ x,   tan x ≈ x,   ex ≈ 1+x,   ln(1+x) ≈ x

三、洛必达法则（L'Hopital's Rule）

设 f(x) 和 g(x) 满足：

  • limx→a f(x) = 0 且 limx→a g(x) = 0（0/0 型）
    或 limx→a f(x) = ±∞ 且 limx→a g(x) = ±∞（∞/∞ 型）
  • f(x) 和 g(x) 在 a 的某去心邻域内可导，且 g′(x) ≠ 0

limx→a f(x)/g(x) = limx→a f′(x)/g′(x)

（若右边极限存在或为无穷大）

不定式类型 转化方法 形式
0/0 或 ∞/∞ 直接使用洛必达法则 f′/g′
0 · ∞ 化为 0/0 或 ∞/∞ f/(1/g) 或 g/(1/f)
∞ − ∞ 通分或有理化 化为 0/0 型
00, 1∞, ∞0 取对数 ln y = g · ln f

例：求 limx→0 (sin x − x) / x³

原式为 0/0 型

= lim (cos x − 1) / 3x²（仍为 0/0）

= lim (−sin x) / 6x（仍为 0/0）

= lim (−cos x) / 6

= −1/6

例：求 limx→0⁺ xx（00 型）

y = xx, ln y = x · ln x = ln x / (1/x)（∞/∞ 型）

= lim (1/x) / (−1/x²) = lim (−x) = 0

ln y → 0, 所以 y → e⁰ = 1

使用洛必达法则时的注意事项：

  • 必须先验证是不定式
  • 对分子分母分别求导，不是对整个分式用商的求导法则
  • 若 lim f′/g′ 不存在，不能断定原极限不存在（需用其他方法）
  • 可多次使用，每次都要验证不定式条件

四、曲线描绘（Curve Sketching）

4.1 单调性判定

在区间 I 上：
f′(x) > 0 ⟹ f(x) 单调递增
f′(x) < 0 ⟹ f(x) 单调递减

4.2 凹凸性与拐点

在区间 I 上：
f″(x) > 0 ⟹ f(x) 凹（下凸 ⌣）
f″(x) < 0 ⟹ f(x) 凸（上凸 ⌢）

凹凸性改变的点称为拐点（Inflection Point）

4.3 极值判别法

方法 条件 结论
一阶导数法 f′ 在 a 左正右负 / 左负右正 极大值 / 极小值
二阶导数法 f′(a) = 0, f″(a) < 0 / > 0 极大值 / 极小值
高阶导数法 f′(a) = f″(a) = … = f(n-1)(a) = 0, f(n)(a) ≠ 0 n 为偶数则极值，n 为奇数则拐点

4.4 渐近线

类型 条件 方程
垂直渐近线 limx→a f(x) = ±∞ x = a
水平渐近线 limx→±∞ f(x) = b y = b
斜渐近线 limx→±∞ [f(x) − (ax+b)] = 0 y = ax + b

斜渐近线系数：a = limx→∞ f(x)/x，b = limx→∞ [f(x) − ax]

例：描绘 f(x) = x³ − 3x² + 2 的图像

f′(x) = 3x² − 6x = 3x(x − 2)

驻点：x = 0（极大值 f(0) = 2），x = 2（极小值 f(2) = −2）

f″(x) = 6x − 6 = 6(x − 1)

拐点：x = 1, f(1) = 0

x < 1：凸（f″ < 0）；x > 1：凹（f″ > 0）

五、曲线描绘完整步骤

  1. 定义域：确定函数的定义域，检查对称性（奇偶性）、周期性
  2. 一阶导数：求 f′(x)，找驻点和不可导点，判断单调区间
  3. 极值：利用驻点计算极值
  4. 二阶导数：求 f″(x)，找拐点，判断凹凸区间
  5. 渐近线：确定垂直、水平或斜渐近线
  6. 特殊点：计算与坐标轴交点
  7. 描绘：综合以上信息画出曲线

核心要点汇总

  • 相关变化率：建立关系 → 对 t 求导 → 代入求解
  • 线性近似：f(x) ≈ f(a) + f′(a)(x − a)，近似公式（sin x ≈ x, ex ≈ 1+x 等）
  • 洛必达法则：适用于 0/0 和 ∞/∞，其他类型需先转化；可多次使用
  • 单调性：f′ 的符号决定；凹凸性：f″ 的符号决定
  • 极值：一阶导数法或二阶导数法；拐点：f″ 变号的点
  • 渐近线：垂直（间断点）、水平（x→∞ 极限）、斜（一次近似）
