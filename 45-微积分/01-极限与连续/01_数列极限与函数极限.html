数列极限与函数极限

极限与连续 · 微积分的基石

一、数列极限

1. ε-N 定义

设 {aₙ} 为数列，a 为常数。若对任意 ε > 0，存在正整数 N，当 n > N 时恒有 |aₙ - a| < ε，则称数列 {aₙ} 的极限为 a，记为：

lim(n→∞) aₙ = a

直观含义：n 足够大时，aₙ 可以任意接近 a。

2. 收敛数列的性质

  • 唯一性：收敛数列的极限唯一
  • 有界性：收敛数列必有界
  • 保号性：若 lim aₙ = a > 0，则存在 N，当 n > N 时 aₙ > 0
  • 夹逼准则：若 aₙ ≤ bₙ ≤ cₙ 且 lim aₙ = lim cₙ = a，则 lim bₙ = a
例题 1：用定义证明极限

证明 lim(n→∞) (2n+1)/(n+1) = 2。

|(2n+1)/(n+1) - 2| = |(2n+1-2n-2)/(n+1)| = 1/(n+1)
要使 1/(n+1) < ε，只需 n > 1/ε - 1
取 N = [1/ε]，当 n > N 时，|(2n+1)/(n+1) - 2| < ε
由定义，极限为 2。

二、函数极限

1. x→x₀ 的 ε-δ 定义

若对任意 ε > 0，存在 δ > 0，当 0 < |x - x₀| < δ 时恒有 |f(x) - A| < ε，则

lim(x→x₀) f(x) = A

注意：0 < |x-x₀| 排除了 x = x₀ 这一点，极限与函数在该点是否有定义、取何值无关。

2. x→∞ 的定义

lim(x→+∞) f(x) = A：对 ∀ε>0, ∃X>0, 当 x>X 时 |f(x)-A|<ε

lim(x→∞) f(x) = A ⟺ lim(x→+∞) f(x) = lim(x→-∞) f(x) = A

3. 极限运算法则

设 lim f(x) = A, lim g(x) = B，且极限均存在：

  • lim [f(x) ± g(x)] = A ± B
  • lim [f(x) · g(x)] = A · B
  • lim [f(x)/g(x)] = A/B（B ≠ 0）
  • lim [f(x)]ⁿ = Aⁿ
  • lim c·f(x) = c·A
例题 2：基本极限计算

求 lim(x→1) (x² - 1)/(x - 1)。

直接代入得 0/0（不定式）
因式分解：(x²-1)/(x-1) = (x+1)(x-1)/(x-1) = x+1   (x ≠ 1)
lim(x→1) (x²-1)/(x-1) = lim(x→1) (x+1) = 2

三、两个重要极限

第一个重要极限

lim(x→0) sin(x)/x = 1

推广形式：lim(u→0) sin(u)/u = 1，其中 u 可以是趋于 0 的任意表达式。

第二个重要极限

lim(n→∞) (1 + 1/n)ⁿ = e

lim(x→∞) (1 + 1/x)ˣ = e

lim(x→0) (1 + x)^(1/x) = e

推广：lim(u→0) (1+u)^(1/u) = e，常用于连续复利、自然增长模型。

例题 3：利用重要极限

求 lim(x→0) sin(3x)/x 和 lim(x→∞) (1+2/x)ˣ。

lim sin(3x)/x = lim 3·sin(3x)/(3x) = 3（令 u=3x）

lim (1+2/x)ˣ = lim [(1+2/x)^(x/2)]² = e²（令 u=2/x）

四、夹逼准则

定理

若在 x₀ 的某去心邻域内（或 n > N 时），恒有 g(x) ≤ f(x) ≤ h(x)，且 lim g(x) = lim h(x) = A，则 lim f(x) = A。

例题 4：夹逼准则

求 lim(n→∞) (1 + 2 + ... + n)/n²。

(1+2+...+n)/n² = n(n+1)/(2n²) = (n+1)/(2n) = (1+1/n)/2
lim = 1/2

夹逼方法：n/n² ≤ (1+2+...+n)/n² ≤ n·n/n²
1/n ≤ S ≤ 1，两头都趋于 1/2（需要更精确的夹逼）
更直接：直接化简 = (n+1)/(2n) → 1/2
极限计算的常用技巧：
  • 有理化（分子或分母有理化）
  • 因式分解（消除不定式）
  • 等价无穷小替换（sinx~x, tanx~x, 1-cosx~x²/2, eˣ-1~x, ln(1+x)~x）
  • 洛必达法则（0/0 或 ∞/∞ 型）
  • 泰勒展开

本节总结

  • 数列极限 ε-N 定义：n > N 时 |aₙ - a| < ε
  • 函数极限 ε-δ 定义：0 < |x-x₀| < δ 时 |f(x)-A| < ε
  • 运算法则：和差积商的极限 = 极限的和差积商（极限存在时）
  • 第一重要极限：lim sinx/x = 1
  • 第二重要极限：lim (1+1/x)ˣ = e
  • 夹逼准则：g(x) ≤ f(x) ≤ h(x)，两边极限相同则中间相同
