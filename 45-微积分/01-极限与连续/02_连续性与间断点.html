连续性与间断点

极限与连续 · 函数的连续性分析

一、连续性

1. 定义

函数 f(x) 在点 x₀ 处连续，需满足三个条件：

  1. f(x₀) 有定义
  2. lim(x→x₀) f(x) 存在
  3. lim(x→x₀) f(x) = f(x₀)
等价定义：lim(Δx→0) Δy = lim(Δx→0) [f(x₀+Δx) - f(x₀)] = 0

2. 左连续与右连续

左连续：lim(x→x₀⁻) f(x) = f(x₀)
右连续：lim(x→x₀⁺) f(x) = f(x₀)

连续 ⟺ 左连续 且 右连续

3. 连续函数的运算

  • 连续函数的和、差、积、商（分母不为零）仍连续
  • 连续函数的复合函数仍连续
  • 初等函数在其定义区间内连续

二、间断点分类

分类标准

若 f(x) 在 x₀ 处不连续，则 x₀ 为间断点。分类依据：左极限和右极限是否存在。

类型左极限右极限示例
可去间断点存在且相等存在且相等f(x)=sinx/x 在 x=0
跳跃间断点存在存在但不等符号函数 sgn(x) 在 x=0
无穷间断点∞（或其中之一）f(x)=1/x 在 x=0
振荡间断点不存在不存在f(x)=sin(1/x) 在 x=0

第一类间断点：左右极限都存在（可去或跳跃）。第二类间断点：至少一侧极限不存在（无穷或振荡）。

例题 1：间断点判断

判断 f(x) = (x²-1)/(x-1) 在 x=1 处的间断点类型。

f(1) 无定义
lim(x→1) (x²-1)/(x-1) = lim(x→1) (x+1) = 2
左右极限都存在且相等
可去间断点（可补充定义 f(1)=2 使其连续）
例题 2：判断间断点

判断 f(x) = (e^(1/x) - 1)/(e^(1/x) + 1) 在 x=0 处的间断点类型。

x→0⁺: 1/x → +∞, e^(1/x) → +∞
f → (+∞-1)/(+∞+1) → 1

x→0⁻: 1/x → -∞, e^(1/x) → 0
f → (0-1)/(0+1) = -1

左极限 = -1，右极限 = 1，不相等
跳跃间断点（第一类）

三、闭区间上连续函数的性质

1. 最值定理（Weierstrass）

若 f(x) 在闭区间 [a, b] 上连续，则 f(x) 在 [a, b] 上必取得最大值 M 和最小值 m。

∃ x₁, x₂ ∈ [a, b]，使得 f(x₁) = M（最大值），f(x₂) = m（最小值）

2. 介值定理（IVT）

若 f(x) 在 [a, b] 上连续，且 f(a) ≠ f(b)，则对 f(a) 与 f(b) 之间的任意值 c，存在 ξ ∈ (a, b) 使得 f(ξ) = c。

f(a) < c < f(b) ⟹ ∃ ξ ∈ (a, b), f(ξ) = c

3. 零点定理

介值定理的特例：若 f(a)·f(b) < 0，则存在 ξ ∈ (a, b) 使得 f(ξ) = 0。

f(a)·f(b) < 0 ⟹ ∃ ξ ∈ (a, b), f(ξ) = 0
例题 3：零点定理应用

证明方程 x³ - 4x + 1 = 0 在 (0, 1) 内至少有一个实根。

设 f(x) = x³ - 4x + 1
f(0) = 1 > 0, f(1) = 1 - 4 + 1 = -2 < 0
f 在 [0,1] 上连续，f(0)·f(1) < 0
由零点定理，存在 ξ ∈ (0,1) 使 f(ξ) = 0，即方程在 (0,1) 内有实根。
重要提醒：
  • 最值定理和介值定理要求 闭区间 + 连续，缺一不可
  • 开区间上连续的函数不一定有最大值（如 f(x)=x 在 (0,1) 上）
  • 零点定理是证明方程存在根的常用工具
  • 初等函数在其定义区间内处处连续，间断点通常出现在分母为零、对数的真数为零等位置

本节总结

  • 连续性三条件：有定义、极限存在、极限值 = 函数值
  • 间断点分类：第一类（可去/跳跃）、第二类（无穷/振荡）
  • 最值定理：闭区间连续函数必有最大最小值
  • 介值定理（IVT）：连续函数取遍最大值与最小值之间的所有值
  • 零点定理：f(a)·f(b) < 0 则 (a,b) 内有零点
  • 初等函数在其定义区间内连续
