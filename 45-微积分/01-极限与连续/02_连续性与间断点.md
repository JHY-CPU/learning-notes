# 连续性与间断点

*极限与连续 · 函数的连续性分析*


### 1. 定义


函数 f(x) 在点 x₀ 处连续，需满足三个条件：


1. f(x₀) 有定义
2. lim(x→x₀) f(x) 存在
3. lim(x→x₀) f(x) = f(x₀)


$$
等价定义：lim(Δx→0) Δy = lim(Δx→0) [f(x₀+Δx) - f(x₀)] = 0
$$


### 2. 左连续与右连续


$$
左连续：lim(x→x₀⁻) f(x) = f(x₀)

            右连续：lim(x→x₀⁺) f(x) = f(x₀)

            连续 ⟺ 左连续 且 右连续
$$


### 3. 连续函数的运算


- 连续函数的和、差、积、商（分母不为零）仍连续
- 连续函数的复合函数仍连续
- 初等函数在其定义区间内连续


### 分类标准


若 f(x) 在 x₀ 处不连续，则 x₀ 为间断点。分类依据：左极限和右极限是否存在。


| 类型 | 左极限 | 右极限 | 示例 |
| --- | --- | --- | --- |
| 可去间断点 | 存在且相等 | 存在且相等 | f(x)=sinx/x 在 x=0 |
| 跳跃间断点 | 存在 | 存在但不等 | 符号函数 sgn(x) 在 x=0 |
| 无穷间断点 | ∞ | （或其中之一） | f(x)=1/x 在 x=0 |
| 振荡间断点 | 不存在 | 不存在 | f(x)=sin(1/x) 在 x=0 |


第一类间断点：左右极限都存在（可去或跳跃）。第二类间断点：至少一侧极限不存在（无穷或振荡）。


### 1. 最值定理（Weierstrass）


若 f(x) 在闭区间 [a, b] 上连续，则 f(x) 在 [a, b] 上必取得最大值 M 和最小值 m。


$$
∃ x₁, x₂ ∈ [a, b]，使得 f(x₁) = M（最大值），f(x₂) = m（最小值）
$$


### 2. 介值定理（IVT）


若 f(x) 在 [a, b] 上连续，且 f(a) ≠ f(b)，则对 f(a) 与 f(b) 之间的任意值 c，存在 ξ ∈ (a, b) 使得 f(ξ) = c。


$$
f(a) < c < f(b) ⟹ ∃ ξ ∈ (a, b), f(ξ) = c
$$


### 3. 零点定理


介值定理的特例：若 f(a)·f(b) < 0，则存在 ξ ∈ (a, b) 使得 f(ξ) = 0。


$$
f(a)·f(b) < 0 ⟹ ∃ ξ ∈ (a, b), f(ξ) = 0
$$


## Python实现

### 间断点类型判断与可视化

```python
import numpy as np
import matplotlib.pyplot as plt

fig, axes = plt.subplots(2, 2, figsize=(10, 8))

# 可去间断点: f(x) = sin(x)/x 在 x=0
ax = axes[0, 0]
x = np.linspace(-3, 3, 500)
x = x[x != 0]
y = np.sin(x) / x
ax.plot(x, y, 'b-')
ax.plot(0, 1, 'ro', markersize=8)  # 补充定义 f(0)=1
ax.set_title('可去间断点: sin(x)/x')
ax.axhline(y=1, color='gray', linestyle='--', alpha=0.5)

# 跳跃间断点: 符号函数
ax = axes[0, 1]
x = np.linspace(-2, 2, 500)
y = np.sign(x)
ax.plot(x, y, 'b-')
ax.plot(0, 1, 'bo')   # 右极限
ax.plot(0, -1, 'bo')  # 左极限
ax.set_title('跳跃间断点: sign(x)')

# 无穷间断点: f(x) = 1/x
ax = axes[1, 0]
x1 = np.linspace(-3, -0.01, 200)
x2 = np.linspace(0.01, 3, 200)
ax.plot(x1, 1/x1, 'b-')
ax.plot(x2, 1/x2, 'b-')
ax.set_ylim(-10, 10)
ax.set_title('无穷间断点: 1/x')
ax.axvline(x=0, color='red', linestyle='--', alpha=0.5)

# 振荡间断点: f(x) = sin(1/x)
ax = axes[1, 1]
x = np.linspace(0.001, 1, 5000)
y = np.sin(1/x)
ax.plot(x, y, 'b-', linewidth=0.5)
ax.set_title('振荡间断点: sin(1/x)')

plt.tight_layout()
plt.savefig('discontinuity_types.png', dpi=100)
plt.show()
```

### 零点定理的数值求解

```python
import numpy as np
from scipy.optimize import brentq

# 二分法求解方程的根 (基于零点定理)
def bisection(f, a, b, tol=1e-8, max_iter=100):
    """二分法求 f(x)=0 在 [a,b] 内的根"""
    if f(a) * f(b) > 0:
        raise ValueError("f(a) 和 f(b) 异号才可使用零点定理")
    for i in range(max_iter):
        c = (a + b) / 2
        if abs(f(c)) < tol or (b - a) / 2 < tol:
            print(f"第 {i+1} 次迭代收敛, 根 ≈ {c:.8f}")
            return c
        if f(a) * f(c) < 0:
            b = c
        else:
            a = c
    return c

# 示例: x^3 - x - 1 = 0, 验证 f(1)=-1, f(2)=5, 零点在 (1,2)
f = lambda x: x**3 - x - 1
print(f"f(1) = {f(1)}, f(2) = {f(2)}")
root = bisection(f, 1, 2)

# 用 scipy 验证
root_scipy = brentq(f, 1, 2)
print(f"scipy 结果: {root_scipy:.8f}")
```
