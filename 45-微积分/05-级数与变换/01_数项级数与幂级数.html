数项级数与幂级数

级数与变换 · 收敛性判别与函数展开

一、数项级数的基本概念

1. 级数的收敛与发散

级数 Σaₙ = a₁ + a₂ + ... 的部分和 Sₙ = a₁ + a₂ + ... + aₙ。

级数收敛 ⟺ lim(n→∞) Sₙ 存在且有限

2. 收敛的必要条件

级数 Σaₙ 收敛 ⟹ lim(n→∞) aₙ = 0

注意：lim aₙ = 0 不保证级数收敛（如调和级数 Σ1/n 发散）。但 lim aₙ ≠ 0 时级数一定发散。

二、收敛判别法

正项级数判别法

方法条件结论
比较判别法0 ≤ aₙ ≤ bₙΣbₙ 收敛 ⟹ Σaₙ 收敛
比值判别法（D'Alembert）lim aₙ₊₁/aₙ = ρρ<1 收敛，ρ>1 发散，ρ=1 不确定
根值判别法（Cauchy）lim ⁿ√(aₙ) = ρρ<1 收敛，ρ>1 发散，ρ=1 不确定
积分判别法f(n) = aₙ, f 递减正函数∫₁^∞ f(x)dx 与 Σaₙ 同敛散
极限比较法lim aₙ/bₙ = L (0<L<∞)Σaₙ 与 Σbₙ 同敛散

交错级数判别法（Leibniz）

对于交错级数 Σ(-1)ⁿ⁻¹aₙ（aₙ > 0），若

  1. aₙ 单调递减：aₙ₊₁ ≤ aₙ
  2. lim aₙ = 0

则级数收敛，且 |余项| ≤ aₙ₊₁。

绝对收敛与条件收敛

  • 绝对收敛：Σ|aₙ| 收敛 ⟹ Σaₙ 收敛
  • 条件收敛：Σaₙ 收敛但 Σ|aₙ| 发散
  • 绝对收敛级数可任意重排，条件收敛级数不可
例题 1：级数敛散性判断

判断 Σ n!/nⁿ 的敛散性。

比值判别法：
aₙ₊₁/aₙ = [(n+1)!/(n+1)ⁿ⁺¹] / [n!/nⁿ]
= (n+1)·nⁿ / (n+1)ⁿ⁺¹ = nⁿ/(n+1)ⁿ = [n/(n+1)]ⁿ
= 1/(1+1/n)ⁿ → 1/e < 1
由比值判别法，级数收敛。

三、幂级数

1. 形式

Σ aₙ(x - x₀)ⁿ = a₀ + a₁(x-x₀) + a₂(x-x₀)² + ...

2. 收敛半径

R = 1 / lim |aₙ₊₁/aₙ|   （比值法）

R = 1 / lim ⁿ√|aₙ|   （根值法）

收敛区间为 (x₀-R, x₀+R)，端点需单独判断。R=0 时仅在 x₀ 收敛，R=+∞ 时处处收敛。

3. 幂级数的运算性质

  • 逐项求导：(Σaₙxⁿ)' = Σnaₙxⁿ⁻¹，收敛半径不变
  • 逐项积分：∫(Σaₙxⁿ)dx = Σ(aₙ/(n+1))xⁿ⁺¹ + C，收敛半径不变
  • 在收敛区间内，幂级数可逐项求导和逐项积分
例题 2：收敛半径

求 Σ xⁿ/n 的收敛半径和收敛域。

aₙ = 1/n
R = 1/lim|aₙ₊₁/aₙ| = 1/lim[n/(n+1)] = 1

收敛区间 (-1, 1)，端点判断：
x=1: Σ1/n（调和级数，发散）
x=-1: Σ(-1)ⁿ/n（交错级数，Leibniz 判别法收敛）

收敛域：[-1, 1)

四、泰勒级数与麦克劳林级数

泰勒级数展开

f(x) = Σ f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ

当 x₀ = 0 时称为麦克劳林级数。

常用麦克劳林级数（熟记）

f(x)展开式收敛域
eˣΣ xⁿ/n!(-∞, +∞)
sin xΣ (-1)ⁿx²ⁿ⁺¹/(2n+1)!(-∞, +∞)
cos xΣ (-1)ⁿx²ⁿ/(2n)!(-∞, +∞)
ln(1+x)Σ (-1)ⁿ⁻¹xⁿ/n(-1, 1]
1/(1-x)Σ xⁿ(-1, 1)
(1+x)^αΣ C(α,n)xⁿ视 α 而定
arctan xΣ (-1)ⁿx²ⁿ⁺¹/(2n+1)[-1, 1]
例题 3：函数展开

将 f(x) = 1/(1+x²) 展开为麦克劳林级数。

利用 1/(1-u) = Σuⁿ（|u|<1）
令 u = -x²：
1/(1+x²) = Σ (-1)ⁿx²ⁿ = 1 - x² + x⁴ - x⁶ + ... , |x|<1
级数在近似计算中的应用：
  • eˣ ≈ 1 + x + x²/2 + x³/6 + ...（取前几项近似）
  • sin x ≈ x - x³/6（小角度近似，工程常用）
  • 数值积分中用级数展开计算复杂积分
  • 微分方程求解中用幂级数方法

本节总结

  • 收敛必要条件：lim aₙ = 0（不充分）
  • 正项级数判别：比较、比值、根值、积分判别法
  • 交错级数：Leibniz 判别法（递减且趋于0）
  • 幂级数收敛半径：R = 1/lim|aₙ₊₁/aₙ|
  • 逐项求导/积分：收敛半径不变
  • 泰勒/麦克劳林级数：常用展开必须熟记
