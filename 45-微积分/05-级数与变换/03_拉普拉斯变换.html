拉普拉斯变换

Laplace Transform — 级数与变换 · 第三节

一、拉普拉斯变换的定义

函数 f(t)（t ≥ 0）的拉普拉斯变换定义为：

F(s) = L{f(t)} = ∫0∞ f(t) e−st dt

其中 s 是复变量（通常取实部足够大的复数）

1.1 存在条件

若 f(t) 满足：

  • 在 [0, +∞) 上分段连续
  • 为指数阶函数（即存在常数 M, a 使 |f(t)| ≤ Meat）

则 L{f(t)} 在 Re(s) > a 时存在。

二、常用拉普拉斯变换表

f(t), t ≥ 0 F(s) = L{f(t)} 条件
1 1/s s > 0
tn n! / sn+1 s > 0, n ∈ ℕ
ta Γ(a+1) / sa+1 s > 0, a > −1
eat 1 / (s − a) s > a
sin(ωt) ω / (s² + ω²) s > 0
cos(ωt) s / (s² + ω²) s > 0
eatsin(ωt) ω / ((s−a)² + ω²) s > a
eatcos(ωt) (s−a) / ((s−a)² + ω²) s > a
t · eat 1 / (s−a)² s > a
δ(t) （单位脉冲） 1 所有 s

三、基本性质

3.1 线性性质

L{af(t) + bg(t)} = aF(s) + bG(s)

3.2 位移性质（频移）

L{eat f(t)} = F(s − a)

3.3 延迟性质（时移）

L{f(t − a) u(t − a)} = e−as F(s)

其中 u(t−a) 是单位阶跃函数

3.4 微分性质（关键性质）

L{f′(t)} = sF(s) − f(0)

L{f″(t)} = s²F(s) − sf(0) − f′(0)

L{f(n)(t)} = snF(s) − sn−1f(0) − … − f(n−1)(0)

这就是拉普拉斯变换能将微分方程化为代数方程的核心原因。

3.5 积分性质

L{∫0t f(τ) dτ} = F(s) / s

四、卷积定理（Convolution Theorem）

卷积的定义：

(f * g)(t) = ∫0t f(τ) g(t − τ) dτ

卷积定理

L{f * g} = F(s) · G(s)

即：L−1{F(s)G(s)} = (f * g)(t)

卷积定理将 s 域中的乘法转化为 t 域中的卷积积分。

卷积的性质：

  • 交换律：f * g = g * f
  • 结合律：(f * g) * h = f * (g * h)
  • 分配律：f * (g + h) = f * g + f * h

五、拉普拉斯逆变换

5.1 逆变换定义

f(t) = L−1{F(s)} = (1/2πi) ∫γ−i∞γ+i∞ F(s) est ds

实际计算中，通常利用变换表和部分分式展开，而非直接计算复积分。

5.2 部分分式展开法

将 F(s) 分解为简单分式的和，逐项查表求逆变换。

F(s) = N(s)/D(s)，其中 deg(N) < deg(D)

对 D(s) 的每个因式展开：

1/(s−a) → eat

1/(s−a)² → t·eat

1/(s²+ω²) → (1/ω)sin(ωt)

s/(s²+ω²) → cos(ωt)

例：求 F(s) = (s+3) / (s² + 2s + 5) 的逆变换

分母配方：s² + 2s + 5 = (s+1)² + 4

分子拆分：s + 3 = (s + 1) + 2

F(s) = (s+1)/((s+1)²+4) + 2/((s+1)²+4)

第一项 → e−tcos(2t)（频移 + 余弦）

第二项 → e−tsin(2t)（频移 + 正弦，系数 2/2 = 1）

f(t) = e−t[cos(2t) + sin(2t)]

六、用拉普拉斯变换解常微分方程

6.1 求解步骤

  1. 对 ODE 两边取拉普拉斯变换，利用微分性质将导数转化为 s 的代数式
  2. 代入初始条件
  3. 求解代数方程，得到 Y(s)
  4. 对 Y(s) 取拉普拉斯逆变换，得到 y(t)

例：求解 y″ + 3y′ + 2y = 0, y(0) = 1, y′(0) = 0

取拉普拉斯变换：[s²Y − s·1 − 0] + 3[sY − 1] + 2Y = 0

(s² + 3s + 2)Y = s + 3

Y(s) = (s+3) / ((s+1)(s+2))

部分分式：Y = 2/(s+1) − 1/(s+2)

y(t) = 2e−t − e−2t

例：求解 y″ + y = sin(t), y(0) = 0, y′(0) = 0

s²Y + Y = 1/(s²+1)

Y = 1/((s²+1)²)

由卷积定理或查表：y(t) = (1/2)[sin(t) − t·cos(t)]

拉普拉斯变换的优势：

  • 自动处理初始条件，无需额外确定积分常数
  • 将微分方程转化为代数方程，大幅简化求解
  • 特别适用于分段函数和脉冲函数驱动的系统
  • 在控制理论、信号处理、电路分析中是标准工具

核心要点汇总

  • 定义：F(s) = ∫₀∞ f(t)e−st dt
  • 微分性质：L{f′(t)} = sF(s) − f(0)，这是解 ODE 的关键
  • 频移：L{eatf(t)} = F(s−a)；时移：L{f(t−a)u(t−a)} = e−asF(s)
  • 卷积定理：L{f*g} = F(s)·G(s)
  • 解 ODE 流程：取变换 → 代入初始条件 → 解代数方程 → 逆变换
  • 常用变换：1↔1/s, eat↔1/(s−a), sin↔ω/(s²+ω²), cos↔s/(s²+ω²)
