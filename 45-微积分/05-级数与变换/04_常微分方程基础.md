# 常微分方程基础 - 级数与变换

*Ordinary Differential Equations — 级数与变换 · 第四节*


### 1.1 ODE 分类概览


| 类型 | 一般形式 | 解法 |
| --- | --- | --- |
| 可分离变量 | dy/dx = f(x)g(y) | 分离积分 |
| 一阶线性 | y′ + P(x)y = Q(x) | 积分因子 |
| 恰当方程 | M dx + N dy = 0, M~y~=N~x~ | 求势函数 |
| 二阶常系数线性 | ay″ + by′ + cy = f(x) | 特征方程 + 特解 |


### 5.1 一般形式


$$
a(x)y″ + b(x)y′ + c(x)y = f(x)
$$


f(x) = 0 为**齐次**，f(x) ≠ 0 为**非齐次**。


### 7.1 待定系数法（Method of Undetermined Coefficients）


适用于 f(x) 为多项式、指数函数、三角函数及其组合。


| f(x) 的形式 | 特解 y~p~ 的假设形式 |
| --- | --- |
| P~n~(x)（n 次多项式） | x^s^ · A~n~(x)（A~n~ 为 n 次多项式） |
| e^αx^ | x^s^ · A e^αx^ |
| cos βx 或 sin βx | x^s^ · (A cos βx + B sin βx) |
| e^αx^(A cos βx + B sin βx) | x^s^ · e^αx^(C cos βx + D sin βx) |


**s 的取值**：若假设形式的最低次项是齐次方程的解，则乘以 x；若是 k 重根，则乘以 x^k^。否则 s = 0。


### Wronskian 行列式


$$
W(y₁, y₂) = |y₁   y₂| = y₁y₂′ − y₂y₁′
$$


$$
|y₁′ y₂′|
$$


$$
u₁′ = −y₂ f / (aW),   u₂′ = y₁ f / (aW)
$$


| 方程类型 | 首选方法 | 备选方法 |
| --- | --- | --- |
| 一阶可分离 | 分离变量 | 积分因子 |
| 一阶线性 | 积分因子 | 常数变易 |
| 恰当方程 | 求势函数 | 积分因子转化 |
| 二阶常系数齐次 | 特征方程 | — |
| 二阶常系数非齐次 | 待定系数 | 常数变易 |
| 二阶变系数 | 常数变易 | 幂级数法 |
