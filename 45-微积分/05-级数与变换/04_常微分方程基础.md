# 常微分方程基础 - 级数与变换

*Ordinary Differential Equations — 级数与变换 · 第四节*


### 1.1 ODE 分类概览


| 类型 | 一般形式 | 解法 |
| --- | --- | --- |
| 可分离变量 | dy/dx = f(x)g(y) | 分离积分 |
| 一阶线性 | y′ + P(x)y = Q(x) | 积分因子 |
| 恰当方程 | M dx + N dy = 0, M~y~=N~x~ | 求势函数 |
| 二阶常系数线性 | ay″ + by′ + cy = f(x) | 特征方程 + 特解 |


### 5.1 一般形式


$$
a(x)y″ + b(x)y′ + c(x)y = f(x)
$$


f(x) = 0 为**齐次**，f(x) ≠ 0 为**非齐次**。


### 7.1 待定系数法（Method of Undetermined Coefficients）


适用于 f(x) 为多项式、指数函数、三角函数及其组合。


| f(x) 的形式 | 特解 y~p~ 的假设形式 |
| --- | --- |
| P~n~(x)（n 次多项式） | x^s^ · A~n~(x)（A~n~ 为 n 次多项式） |
| e^αx^ | x^s^ · A e^αx^ |
| cos βx 或 sin βx | x^s^ · (A cos βx + B sin βx) |
| e^αx^(A cos βx + B sin βx) | x^s^ · e^αx^(C cos βx + D sin βx) |


**s 的取值**：若假设形式的最低次项是齐次方程的解，则乘以 x；若是 k 重根，则乘以 x^k^。否则 s = 0。


### Wronskian 行列式


$$
W(y₁, y₂) = |y₁   y₂| = y₁y₂′ − y₂y₁′
$$


$$
|y₁′ y₂′|
$$


$$
u₁′ = −y₂ f / (aW),   u₂′ = y₁ f / (aW)
$$


| 方程类型 | 首选方法 | 备选方法 |
| --- | --- | --- |
| 一阶可分离 | 分离变量 | 积分因子 |
| 一阶线性 | 积分因子 | 常数变易 |
| 恰当方程 | 求势函数 | 积分因子转化 |
| 二阶常系数齐次 | 特征方程 | — |
| 二阶常系数非齐次 | 待定系数 | 常数变易 |
| 二阶变系数 | 常数变易 | 幂级数法 |

### 可分离变量方程详解

形式：dy/dx = f(x)g(y)

解法：分离变量后积分
$$\int \frac{1}{g(y)} dy = \int f(x) dx + C$$

**例题**：dy/dx = xy, y(0)=1

分离：dy/y = x dx
积分：ln|y| = x²/2 + C
代入初值：y = e^{x²/2}

### 一阶线性方程详解

形式：y' + P(x)y = Q(x)

积分因子：μ(x) = e^{∫P(x)dx}

通解：y = (1/μ) [∫μQ dx + C]

**例题**：y' + (2/x)y = x, y(1)=0

P(x) = 2/x, μ = e^{∫2/x dx} = e^{2lnx} = x²

y = (1/x²) [∫x²·x dx + C] = (1/x²)[x⁴/4 + C]

代入 y(1)=0：0 = 1/4 + C → C = -1/4

y = x²/4 - 1/(4x²)

### 二阶常系数齐次方程详解

形式：ay'' + by' + cy = 0

特征方程：ar² + br + c = 0

| 特征根情况 | 通解 |
|-----------|------|
| 两不等实根 r₁≠r₂ | y = C₁e^{r₁x} + C₂e^{r₂x} |
| 两相等实根 r₁=r₂=r | y = (C₁+C₂x)e^{rx} |
| 共轭复根 r = α±βi | y = e^{αx}(C₁cosβx + C₂sinβx) |

**例题**：y'' - 3y' + 2y = 0

特征方程：r²-3r+2 = 0 → r₁=1, r₂=2
通解：y = C₁e^x + C₂e^{2x}

### 二阶常系数非齐次方程

通解 = 齐次通解 + 特解

**例题**：y'' - 3y' + 2y = e^{3x}

齐次通解已知：y_h = C₁e^x + C₂e^{2x}

设特解 y_p = Ae^{3x}（因为 3 不是特征根）
y_p' = 3Ae^{3x}, y_p'' = 9Ae^{3x}

代入：9A - 9A + 2A = 1 → A = 1/2

通解：y = C₁e^x + C₂e^{2x} + (1/2)e^{3x}

### 常数变易法

当待定系数法不适用时（f(x) 不是多项式/指数/三角的组合），使用常数变易法：

1. 先求齐次方程的两个线性无关解 y₁, y₂
2. 设特解 y_p = u₁(x)y₁ + u₂(x)y₂
3. 解方程组求 u₁', u₂'

### 易错点

1. **特征方程是 ar²+br+c=0**：不是 ar²+br+c
2. **重根情况不要遗漏 x 因子**：y = (C₁+C₂x)e^{rx}
3. **特解形式要与齐次解比较**：若假设形式含齐次解，需乘以 x
4. **常数变易法的约束条件**：u₁'y₁ + u₂'y₂ = 0


## Python实现

### 常微分方程的符号求解

```python
import sympy as sp

x = sp.Symbol('x')
y = sp.Function('y')

# 可分离变量方程: dy/dx = x*y
eq1 = sp.Eq(y(x).diff(x), x * y(x))
sol1 = sp.dsolve(eq1, y(x))
print(f"dy/dx = xy → {sol1}")

# 一阶线性方程: y' + 2y = e^(-x)
eq2 = sp.Eq(y(x).diff(x) + 2*y(x), sp.exp(-x))
sol2 = sp.dsolve(eq2, y(x))
print(f"y' + 2y = e^(-x) → {sol2}")

# 二阶常系数齐次: y'' - 3y' + 2y = 0
eq3 = sp.Eq(y(x).diff(x, 2) - 3*y(x).diff(x) + 2*y(x), 0)
sol3 = sp.dsolve(eq3, y(x))
print(f"y'' - 3y' + 2y = 0 → {sol3}")

# 二阶常系数非齐次: y'' + y = sin(x)
eq4 = sp.Eq(y(x).diff(x, 2) + y(x), sp.sin(x))
sol4 = sp.dsolve(eq4, y(x))
print(f"y'' + y = sin(x) → {sol4}")

# 带初始条件的求解
eq5 = sp.Eq(y(x).diff(x, 2) + y(x), 0)
sol5 = sp.dsolve(eq5, y(x), ics={y(0): 1, y(x).subs(x, 0).diff(x): 0})
# 简化写法
eq_ic = sp.Eq(y(x).diff(x, 2) + y(x), 0)
sol_ic = sp.dsolve(eq_ic, y(x), ics={y(0): 2, sp.diff(y(x), x).subs(x, 0): 1})
print(f"y'' + y = 0, y(0)=2, y'(0)=1 → {sol_ic}")
```

### ODE 的数值求解

```python
import numpy as np
from scipy.integrate import odeint, solve_ivp
import matplotlib.pyplot as plt

# y' = -2y + sin(x), y(0) = 1
def f(y, x):
    return -2*y + np.sin(x)

# 方法1: odeint
x_span = np.linspace(0, 10, 200)
y0 = 1.0
y_odeint = odeint(f, y0, x_span).flatten()

# 方法2: solve_ivp (更现代)
def f_ivp(x, y):
    return -2*y + np.sin(x)

sol = solve_ivp(f_ivp, [0, 10], [y0], t_eval=x_span, method='RK45')
y_rk45 = sol.y[0]

# 可视化
plt.figure(figsize=(10, 5))
plt.plot(x_span, y_odeint, 'b-', label='odeint (LSODA)', linewidth=2)
plt.plot(x_span, y_rk45, 'r--', label='solve_ivp (RK45)')
plt.xlabel('x')
plt.ylabel('y')
plt.title("ODE 数值解: y' = -2y + sin(x), y(0)=1")
plt.legend()
plt.grid(True, alpha=0.3)
plt.savefig('ode_numerical.png', dpi=100)
plt.show()
```

### 特征方程法求解二阶 ODE

```python
import numpy as np

# y'' + 3y' + 2y = 0 → 特征方程: r² + 3r + 2 = 0
coeffs = [1, 3, 2]  # a, b, c
roots = np.roots(coeffs)
print(f"特征方程: r² + 3r + 2 = 0")
print(f"特征根: r1 = {roots[0]}, r2 = {roots[1]}")
print(f"通解: y = C1*e^({roots[0]}x) + C2*e^({roots[1]}x)")

# 重根情况: y'' + 2y' + y = 0
coeffs = [1, 2, 1]
roots = np.roots(coeffs)
print(f"\n特征方程: r² + 2r + 1 = 0")
print(f"特征根: r = {roots[0]} (重根)")
print(f"通解: y = (C1 + C2*x)*e^({roots[0]}x)")

# 复根情况: y'' + 2y' + 5y = 0
coeffs = [1, 2, 5]
roots = np.roots(coeffs)
alpha = np.real(roots[0])
beta = np.imag(roots[0])
print(f"\n特征方程: r² + 2r + 5 = 0")
print(f"特征根: r = {roots[0]}, {roots[1]}")
print(f"通解: y = e^({alpha:.0f}x)(C1*cos({abs(beta):.0f}x) + C2*sin({abs(beta):.0f}x))")
```
