# 傅里叶级数与变换

*级数与变换 · 信号的频域分析*


### 三角函数系


{1, cos x, sin x, cos 2x, sin 2x, ..., cos nx, sin nx, ...}


在区间 [-π, π] 上，任意两个不同函数的乘积积分等于零：


$$
∫₋πᵖ cos(mx)cos(nx)dx = 0   （m ≠ n）

            ∫₋πᵖ sin(mx)sin(nx)dx = 0   （m ≠ n）

            ∫₋πᵖ sin(mx)cos(nx)dx = 0   （对任意 m, n）
$$


### 1. 傅里叶级数展开


周期为 2π 的函数 f(x) 的傅里叶级数：


$$
f(x) ~ a₀/2 + Σ [aₙcos(nx) + bₙsin(nx)]
$$


其中傅里叶系数：


$$
aₙ = (1/π) ∫₋πᵖ f(x)cos(nx)dx   （n = 0, 1, 2, ...）

            bₙ = (1/π) ∫₋πᵖ f(x)sin(nx)dx   （n = 1, 2, 3, ...）
$$


### 2. 收敛定理（Dirichlet 条件）


若 f(x) 在 [-π, π] 上满足：


1. 连续或只有有限个第一类间断点
2. 只有有限个极值点


则傅里叶级数收敛于：


- x 为连续点：f(x)
- x 为间断点：[f(x⁺) + f(x⁻)] / 2
- x = ±π：[f(-π⁺) + f(π⁻)] / 2


### 3. 奇偶函数的傅里叶级数


| f(x) 的对称性 | 傅里叶系数 | 级数形式 |
| --- | --- | --- |
| 偶函数 f(-x)=f(x) | bₙ=0 | 余弦级数：a₀/2 + Σaₙcos(nx) |
| 奇函数 f(-x)=-f(x) | aₙ=0 | 正弦级数：Σbₙsin(nx) |


### 公式


$$
f(x) ~ a₀/2 + Σ [aₙcos(nπx/L) + bₙsin(nπx/L)]

            aₙ = (1/L) ∫₋ᴸᴸ f(x)cos(nπx/L)dx

            bₙ = (1/L) ∫₋ᴸᴸ f(x)sin(nπx/L)dx
$$


### 1. 从傅里叶级数到傅里叶变换


当周期 T → ∞ 时，离散的频率 n/T 变为连续的频率 ω，级数变为积分：


$$
傅里叶变换（时域 → 频域）：

            F(ω) = ∫₋∞⁺∞ f(t) e⁻ʲωᵗ dt

傅里叶逆变换（频域 → 时域）：

            f(t) = (1/2π) ∫₋∞⁺∞ F(ω) eʲωᵗ dω
$$


### 2. 基本性质


| 性质 | 时域 | 频域 |
| --- | --- | --- |
| 线性 | af(t) + bg(t) | aF(ω) + bG(ω) |
| 时移 | f(t - t₀) | F(ω)e⁻ʲωᵗ⁰ |
| 频移 | f(t)eʲω₀ᵗ | F(ω - ω₀) |
| 尺度变换 | f(at) | (1/\|a\|)F(ω/a) |
| 时域微分 | f'(t) | jωF(ω) |
| 卷积 | f(t)*g(t) | F(ω)·G(ω) |


卷积定理：**时域卷积 = 频域乘积**，这是傅里叶变换最重要的性质之一。


## Python实现

### 傅里叶级数计算与可视化

```python
import numpy as np
import matplotlib.pyplot as plt

def fourier_coefficients(f, n_terms, L=np.pi):
    """计算傅里叶系数 a_0, a_n, b_n"""
    from scipy.integrate import quad

    # a_0 = (1/L) ∫_{-L}^{L} f(x) dx
    a0, _ = quad(lambda x: f(x), -L, L) / L

    an = []
    bn = []
    for n in range(1, n_terms + 1):
        # a_n = (1/L) ∫ f(x)*cos(nπx/L) dx
        a, _ = quad(lambda x: f(x) * np.cos(n * np.pi * x / L), -L, L)
        an.append(a / L)
        # b_n = (1/L) ∫ f(x)*sin(nπx/L) dx
        b, _ = quad(lambda x: f(x) * np.sin(n * np.pi * x / L), -L, L)
        bn.append(b / L)

    return a0, an, bn

def fourier_series(x, a0, an, bn, L=np.pi):
    """计算傅里叶级数在 x 处的值"""
    result = a0 / 2
    for n, (a, b) in enumerate(zip(an, bn), 1):
        result += a * np.cos(n * np.pi * x / L) + b * np.sin(n * np.pi * x / L)
    return result

# 示例: 方波函数 f(x) = sign(x) 在 [-π, π]
f_square = lambda x: np.sign(x) if abs(x) > 1e-10 else 0

fig, axes = plt.subplots(2, 2, figsize=(12, 10))
x_plot = np.linspace(-np.pi, np.pi, 500)

for idx, n_terms in enumerate([1, 3, 5, 20]):
    ax = axes[idx // 2, idx % 2]
    a0, an, bn = fourier_coefficients(f_square, n_terms)

    y_exact = [f_square(xi) for xi in x_plot]
    y_fourier = [fourier_series(xi, a0, an, bn) for xi in x_plot]

    ax.plot(x_plot, y_exact, 'b-', label='原函数', linewidth=2)
    ax.plot(x_plot, y_fourier, 'r--', label=f'傅里叶级数 (n={n_terms})')
    ax.set_title(f'n = {n_terms} 项')
    ax.legend()
    ax.set_ylim(-1.5, 1.5)
    ax.grid(True, alpha=0.3)

plt.suptitle('方波函数的傅里叶级数逼近', fontsize=14)
plt.tight_layout()
plt.savefig('fourier_series.png', dpi=100)
plt.show()

# 打印方波的傅里叶系数
a0, an, bn = fourier_coefficients(f_square, 6)
print(f"a_0 = {a0:.4f}")
for i, (a, b) in enumerate(zip(an, bn), 1):
    print(f"a_{i} = {a:.6f}, b_{i} = {b:.6f}")
# 方波只有 b_n 项: b_n = 2/(nπ) * (1-(-1)^n)
```

### FFT (快速傅里叶变换)

```python
import numpy as np

# 生成信号: 包含 50Hz 和 120Hz 的正弦波 + 噪声
fs = 1000  # 采样率
t = np.linspace(0, 1, fs, endpoint=False)
signal = np.sin(2*np.pi*50*t) + 0.5*np.sin(2*np.pi*120*t) + 0.3*np.random.randn(fs)

# FFT
fft_result = np.fft.fft(signal)
freqs = np.fft.fftfreq(len(signal), 1/fs)
magnitude = np.abs(fft_result) / len(signal)

# 只取正频率部分
pos_mask = freqs > 0
freqs_pos = freqs[pos_mask]
mag_pos = 2 * magnitude[pos_mask]  # 乘2是因为对称

# 找到主要频率分量
peak_idx = np.argsort(mag_pos)[-2:]  # 取最大的两个
print("主要频率分量:")
for idx in sorted(peak_idx, reverse=True):
    print(f"  频率 = {freqs_pos[idx]:.1f} Hz, 幅值 = {mag_pos[idx]:.4f}")
```
