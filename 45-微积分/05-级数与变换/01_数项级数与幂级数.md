# 数项级数与幂级数

*级数与变换 · 收敛性判别与函数展开*


### 1. 级数的收敛与发散


级数 Σaₙ = a₁ + a₂ + ... 的部分和 Sₙ = a₁ + a₂ + ... + aₙ。


$$
级数收敛 ⟺ lim(n→∞) Sₙ 存在且有限
$$


### 2. 收敛的必要条件


$$
级数 Σaₙ 收敛 ⟹ lim(n→∞) aₙ = 0
$$


注意：lim aₙ = 0 不保证级数收敛（如调和级数 Σ1/n 发散）。但 lim aₙ ≠ 0 时级数一定发散。


### 正项级数判别法


| 方法 | 条件 | 结论 |
| --- | --- | --- |
| 比较判别法 | 0 ≤ aₙ ≤ bₙ | Σbₙ 收敛 ⟹ Σaₙ 收敛 |
| 比值判别法（D'Alembert） | lim aₙ₊₁/aₙ = ρ | ρ<1 收敛，ρ>1 发散，ρ=1 不确定 |
| 根值判别法（Cauchy） | lim ⁿ√(aₙ) = ρ | ρ<1 收敛，ρ>1 发散，ρ=1 不确定 |
| 积分判别法 | f(n) = aₙ, f 递减正函数 | ∫₁^∞ f(x)dx 与 Σaₙ 同敛散 |
| 极限比较法 | lim aₙ/bₙ = L (0<L<∞) | Σaₙ 与 Σbₙ 同敛散 |


### 交错级数判别法（Leibniz）


对于交错级数 Σ(-1)ⁿ⁻¹aₙ（aₙ > 0），若


1. aₙ 单调递减：aₙ₊₁ ≤ aₙ
2. lim aₙ = 0


则级数收敛，且 |余项| ≤ aₙ₊₁。


### 绝对收敛与条件收敛


- **绝对收敛**
   ：Σ|aₙ| 收敛 ⟹ Σaₙ 收敛
- **条件收敛**
   ：Σaₙ 收敛但 Σ|aₙ| 发散
- 绝对收敛级数可任意重排，条件收敛级数不可


### 1. 形式


$$
Σ aₙ(x - x₀)ⁿ = a₀ + a₁(x-x₀) + a₂(x-x₀)² + ...
$$


### 2. 收敛半径


$$
R = 1 / lim |aₙ₊₁/aₙ|   （比值法）

            R = 1 / lim ⁿ√|aₙ|   （根值法）
$$


收敛区间为 (x₀-R, x₀+R)，端点需单独判断。R=0 时仅在 x₀ 收敛，R=+∞ 时处处收敛。


### 3. 幂级数的运算性质


- 逐项求导：(Σaₙxⁿ)' = Σnaₙxⁿ⁻¹，收敛半径不变
- 逐项积分：∫(Σaₙxⁿ)dx = Σ(aₙ/(n+1))xⁿ⁺¹ + C，收敛半径不变
- 在收敛区间内，幂级数可逐项求导和逐项积分


### 泰勒级数展开


$$
f(x) = Σ f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ
$$


当 x₀ = 0 时称为麦克劳林级数。


### 常用麦克劳林级数（熟记）


| f(x) | 展开式 | 收敛域 |
| --- | --- | --- |
| eˣ | Σ xⁿ/n! | (-∞, +∞) |
| sin x | Σ (-1)ⁿx²ⁿ⁺¹/(2n+1)! | (-∞, +∞) |
| cos x | Σ (-1)ⁿx²ⁿ/(2n)! | (-∞, +∞) |
| ln(1+x) | Σ (-1)ⁿ⁻¹xⁿ/n | (-1, 1] |
| 1/(1-x) | Σ xⁿ | (-1, 1) |
| (1+x)^α | Σ C(α,n)xⁿ | 视 α 而定 |
| arctan x | Σ (-1)ⁿx²ⁿ⁺¹/(2n+1) | [-1, 1] |
