# 数项级数与幂级数

*级数与变换 · 收敛性判别与函数展开*


### 1. 级数的收敛与发散


级数 Σaₙ = a₁ + a₂ + ... 的部分和 Sₙ = a₁ + a₂ + ... + aₙ。


$$
级数收敛 ⟺ lim(n→∞) Sₙ 存在且有限
$$


### 2. 收敛的必要条件


$$
级数 Σaₙ 收敛 ⟹ lim(n→∞) aₙ = 0
$$


注意：lim aₙ = 0 不保证级数收敛（如调和级数 Σ1/n 发散）。但 lim aₙ ≠ 0 时级数一定发散。


### 正项级数判别法


| 方法 | 条件 | 结论 |
| --- | --- | --- |
| 比较判别法 | 0 ≤ aₙ ≤ bₙ | Σbₙ 收敛 ⟹ Σaₙ 收敛 |
| 比值判别法（D'Alembert） | lim aₙ₊₁/aₙ = ρ | ρ<1 收敛，ρ>1 发散，ρ=1 不确定 |
| 根值判别法（Cauchy） | lim ⁿ√(aₙ) = ρ | ρ<1 收敛，ρ>1 发散，ρ=1 不确定 |
| 积分判别法 | f(n) = aₙ, f 递减正函数 | ∫₁^∞ f(x)dx 与 Σaₙ 同敛散 |
| 极限比较法 | lim aₙ/bₙ = L (0<L<∞) | Σaₙ 与 Σbₙ 同敛散 |


### 交错级数判别法（Leibniz）


对于交错级数 Σ(-1)ⁿ⁻¹aₙ（aₙ > 0），若


1. aₙ 单调递减：aₙ₊₁ ≤ aₙ
2. lim aₙ = 0


则级数收敛，且 |余项| ≤ aₙ₊₁。


### 绝对收敛与条件收敛


- **绝对收敛**
   ：Σ|aₙ| 收敛 ⟹ Σaₙ 收敛
- **条件收敛**
   ：Σaₙ 收敛但 Σ|aₙ| 发散
- 绝对收敛级数可任意重排，条件收敛级数不可


### 1. 形式


$$
Σ aₙ(x - x₀)ⁿ = a₀ + a₁(x-x₀) + a₂(x-x₀)² + ...
$$


### 2. 收敛半径


$$
R = 1 / lim |aₙ₊₁/aₙ|   （比值法）

            R = 1 / lim ⁿ√|aₙ|   （根值法）
$$


收敛区间为 (x₀-R, x₀+R)，端点需单独判断。R=0 时仅在 x₀ 收敛，R=+∞ 时处处收敛。


### 3. 幂级数的运算性质


- 逐项求导：(Σaₙxⁿ)' = Σnaₙxⁿ⁻¹，收敛半径不变
- 逐项积分：∫(Σaₙxⁿ)dx = Σ(aₙ/(n+1))xⁿ⁺¹ + C，收敛半径不变
- 在收敛区间内，幂级数可逐项求导和逐项积分


### 泰勒级数展开


$$
f(x) = Σ f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ
$$


当 x₀ = 0 时称为麦克劳林级数。


### 常用麦克劳林级数（熟记）


| f(x) | 展开式 | 收敛域 |
| --- | --- | --- |
| eˣ | Σ xⁿ/n! | (-∞, +∞) |
| sin x | Σ (-1)ⁿx²ⁿ⁺¹/(2n+1)! | (-∞, +∞) |
| cos x | Σ (-1)ⁿx²ⁿ/(2n)! | (-∞, +∞) |
| ln(1+x) | Σ (-1)ⁿ⁻¹xⁿ/n | (-1, 1] |
| 1/(1-x) | Σ xⁿ | (-1, 1) |
| (1+x)^α | Σ C(α,n)xⁿ | 视 α 而定 |
| arctan x | Σ (-1)ⁿx²ⁿ⁺¹/(2n+1) | [-1, 1] |


## Python实现

### 级数收敛性判断

```python
import numpy as np

# 比值判别法: Σ a_n, 计算 lim |a_{n+1}/a_n|
def ratio_test(a_n_func, n_max=1000):
    """比值判别法"""
    ratios = []
    for n in range(1, n_max):
        r = abs(a_n_func(n+1) / a_n_func(n))
        ratios.append(r)
    rho = ratios[-1]
    print(f"  ρ ≈ {rho:.6f}", end="")
    if rho < 1:
        print(" → 收敛")
    elif rho > 1:
        print(" → 发散")
    else:
        print(" → 不确定")
    return rho

# 根值判别法
def root_test(a_n_func, n_max=1000):
    """根值判别法"""
    n = n_max
    rho = abs(a_n_func(n)) ** (1/n)
    print(f"  ⁿ√(a_n) ≈ {rho:.6f}", end="")
    if rho < 1:
        print(" → 收敛")
    elif rho > 1:
        print(" → 发散")

# 测试不同级数
print("=== 比值判别法 ===")
print("Σ 1/n! (收敛):")
ratio_test(lambda n: 1/np.math.factorial(n))

print("Σ n!/n^n (收敛):")
ratio_test(lambda n: np.math.factorial(n) / n**n)

print("Σ 2^n/n! (收敛):")
ratio_test(lambda n: 2**n / np.math.factorial(n))

# 交错级数 Leibniz 判别法
print("\n=== 交错级数 ===")
# Σ (-1)^n / n, a_n = 1/n 单调递减趋于0
partial_sums = []
S = 0
for n in range(1, 1001):
    S += (-1)**(n-1) / n
    if n in [10, 100, 1000]:
        partial_sums.append(S)
        print(f"S_{n} = {S:.8f}, ln(2) = {np.log(2):.8f}")
```

### 泰勒级数展开与近似

```python
import sympy as sp
import numpy as np

x = sp.Symbol('x')

# 泰勒级数展开
print("=== 泰勒级数展开 ===")
for name, func in [("e^x", sp.exp(x)),
                    ("sin(x)", sp.sin(x)),
                    ("cos(x)", sp.cos(x)),
                    ("ln(1+x)", sp.ln(1+x))]:
    series = sp.series(func, x, 0, 8)
    print(f"{name} ≈ {series}")

# 收敛半径计算
print("\n=== 收敛半径 ===")
# a_n = 1/n!
from sympy import factorial
n = sp.Symbol('n', positive=True, integer=True)
a_n = 1/factorial(n)
ratio = sp.limit(abs(a_n.subs(n, n+1) / a_n), n, sp.oo)
R = 1/ratio if ratio != 0 else sp.oo
print(f"Σ x^n/n! 的收敛半径 R = {R}")

# 幂级数数值近似
print("\n=== 幂级数近似精度 ===")
x_val = 1.0
exact = np.exp(x_val)
for order in [3, 5, 7, 10]:
    # 用 sympy 提取系数手动求和
    series = sp.series(sp.exp(x), x, 0, order+1)
    approx = float(series.removeO().subs(x, x_val))
    print(f"e^1 ≈ {approx:.10f} (n={order}), 误差 = {abs(approx - exact):.2e}")
```
