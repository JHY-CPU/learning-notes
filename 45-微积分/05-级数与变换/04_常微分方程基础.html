常微分方程基础

Ordinary Differential Equations — 级数与变换 · 第四节

一、基本概念

常微分方程（ODE）：含有未知函数及其导数的方程。

  • 阶：方程中出现的最高阶导数的阶数
  • 线性：未知函数及其各阶导数均为一次的
  • 通解：含 n 个独立任意常数的解（n = 阶数）
  • 特解：由初始条件确定常数后的解

1.1 ODE 分类概览

类型 一般形式 解法
可分离变量 dy/dx = f(x)g(y) 分离积分
一阶线性 y′ + P(x)y = Q(x) 积分因子
恰当方程 M dx + N dy = 0, My=Nx 求势函数
二阶常系数线性 ay″ + by′ + cy = f(x) 特征方程 + 特解

二、可分离变量方程

形如 dy/dx = f(x) · g(y) 的方程：

分离变量：dy/g(y) = f(x) dx

两边积分：∫ dy/g(y) = ∫ f(x) dx + C

例：求解 dy/dx = 2xy

dy/y = 2x dx

ln|y| = x² + C₁

y = Cex²（其中 C = ±eC₁，也包含 C = 0 的情况）

例：求解 dy/dx = y² sin(x)

dy/y² = sin(x) dx

−1/y = −cos(x) + C

y = 1/(cos(x) − C)

三、一阶线性方程

标准形式：y′ + P(x)y = Q(x)

若 Q(x) = 0，称为齐次；否则为非齐次。

3.1 积分因子法

积分因子：μ(x) = e∫P(x)dx

通解公式：

y = (1/μ) [∫ μ(x) Q(x) dx + C]

例：求解 y′ + (2/x)y = x

P(x) = 2/x, μ = e∫2/x dx = e2ln x = x²

y = (1/x²)[∫ x² · x dx + C] = (1/x²)[x⁴/4 + C]

y = x²/4 + C/x²

四、恰当方程（全微分方程）

方程 M(x,y)dx + N(x,y)dy = 0，若存在函数 Φ(x,y) 使得 ∂Φ/∂x = M, ∂Φ/∂y = N，则称为恰当方程。

判别条件：∂M/∂y = ∂N/∂x

通解：Φ(x, y) = C

求解步骤

  1. 验证 ∂M/∂y = ∂N/∂x
  2. 由 ∂Φ/∂x = M 对 x 积分：Φ = ∫M dx + h(y)
  3. 由 ∂Φ/∂y = N 确定 h(y)
  4. 写出通解 Φ(x, y) = C

例：求解 (2xy + 3)dx + (x² + 4y)dy = 0

M = 2xy + 3, N = x² + 4y

∂M/∂y = 2x, ∂N/∂x = 2x → 恰当方程 ✓

Φ = ∫(2xy + 3)dx = x²y + 3x + h(y)

∂Φ/∂y = x² + h′(y) = x² + 4y → h′(y) = 4y → h(y) = 2y²

通解：x²y + 3x + 2y² = C

非恰当方程的处理：若 ∂M/∂y ≠ ∂N/∂x，可寻找积分因子 μ(x) 或 μ(y) 使方程变为恰当方程。

五、二阶线性常微分方程

5.1 一般形式

a(x)y″ + b(x)y′ + c(x)y = f(x)

f(x) = 0 为齐次，f(x) ≠ 0 为非齐次。

5.2 叠加原理

若 y₁ 和 y₂ 是齐次方程的两个线性无关解，则通解为：

yh = C₁y₁ + C₂y₂

非齐次方程的通解 = 齐次通解 + 一个特解：y = yh + yp

六、常系数齐次线性方程

ay″ + by′ + cy = 0（a, b, c 为常数）

特征方程：ar² + br + c = 0

特征根：r = (−b ± √(b²−4ac)) / 2a

特征根情况 判别式 通解形式
两个不等实根 r₁ ≠ r₂ Δ = b²−4ac > 0 y = C₁er₁x + C₂er₂x
重根 r₁ = r₂ = r Δ = 0 y = (C₁ + C₂x)erx
共轭复根 α ± βi Δ < 0 y = eαx(C₁cos βx + C₂sin βx)

例：求解 y″ − 5y′ + 6y = 0

特征方程：r² − 5r + 6 = 0 → (r−2)(r−3) = 0

r₁ = 2, r₂ = 3

y = C₁e2x + C₂e3x

例：求解 y″ + 4y′ + 4y = 0

r² + 4r + 4 = 0 → (r+2)² = 0, r = −2（重根）

y = (C₁ + C₂x)e−2x

例：求解 y″ + 2y′ + 5y = 0

r² + 2r + 5 = 0 → r = −1 ± 2i

y = e−x(C₁cos 2x + C₂sin 2x)

七、非齐次方程——待定系数法

7.1 待定系数法（Method of Undetermined Coefficients）

适用于 f(x) 为多项式、指数函数、三角函数及其组合。

f(x) 的形式 特解 yp 的假设形式
Pn(x)（n 次多项式） xs · An(x)（An 为 n 次多项式）
eαx xs · A eαx
cos βx 或 sin βx xs · (A cos βx + B sin βx)
eαx(A cos βx + B sin βx) xs · eαx(C cos βx + D sin βx)

s 的取值：若假设形式的最低次项是齐次方程的解，则乘以 x；若是 k 重根，则乘以 xk。否则 s = 0。

例：求解 y″ − 3y′ + 2y = e3x

齐次通解：r²−3r+2 = 0 → r = 1,2，yh = C₁ex + C₂e2x

设 yp = Ae3x（3 不是特征根，s = 0）

代入：9A − 9A + 2A = 1 → A = 1/2

y = C₁ex + C₂e2x + (1/2)e3x

八、常数变易法（Variation of Parameters）

适用于任意 f(x)，比待定系数法更通用。

设齐次方程的两个线性无关解为 y₁, y₂，令：

yp = u₁(x) y₁(x) + u₂(x) y₂(x)

其中 u₁, u₂ 满足：

u₁′ y₁ + u₂′ y₂ = 0

u₁′ y₁′ + u₂′ y₂′ = f/a

解出 u₁′, u₂′ 后积分得到 u₁, u₂。

Wronskian 行列式

W(y₁, y₂) = |y₁   y₂| = y₁y₂′ − y₂y₁′

              |y₁′ y₂′|

u₁′ = −y₂ f / (aW),   u₂′ = y₁ f / (aW)

例：求解 y″ + y = sec(x)（用常数变易法）

齐次通解：yh = C₁cos x + C₂sin x

y₁ = cos x, y₂ = sin x, W = cos²x + sin²x = 1

u₁′ = −sin x · sec x / 1 = −tan x

u₂′ = cos x · sec x / 1 = 1

u₁ = ln|cos x|, u₂ = x

y = C₁cos x + C₂sin x + cos x · ln|cos x| + x sin x

九、ODE 解题策略总结

方程类型 首选方法 备选方法

一阶线性 积分因子 常数变易
恰当方程 求势函数 积分因子转化
二阶常系数齐次 特征方程 —
二阶常系数非齐次 待定系数 常数变易
二阶变系数 常数变易 幂级数法

核心要点汇总

  • 可分离变量：dy/g(y) = f(x)dx，两边积分
  • 一阶线性：y′ + Py = Q，积分因子 μ = e∫Pdx，通解 y = (1/μ)[∫μQ dx + C]
  • 恰当方程：My = Nx，求势函数 Φ(x,y) = C
  • 二阶常系数齐次：ar² + br + c = 0，实根/重根/复根对应三种通解形式
  • 非齐次：通解 = 齐次通解 + 特解；待定系数法或常数变易法求特解
  • 常数变易法：用 Wronskian 求 u₁′, u₂′，适用于任意右端项
