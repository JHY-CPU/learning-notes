### 6.1 求解步骤


1. 对 ODE 两边取拉普拉斯变换，利用微分性质将导数转化为 s 的代数式
2. 代入初始条件
3. 求解代数方程，得到 Y(s)
4. 对 Y(s) 取拉普拉斯逆变换，得到 y(t)

### 拉普拉斯变换的定义

$$F(s) = L\{f(t)\} = \int_0^\infty f(t) e^{-st} dt$$

其中 s = σ + jω 为复变量。变换将时域函数 f(t) 映射到复频域函数 F(s)。

### 常用拉普拉斯变换对

| f(t) | F(s) | 适用条件 |
|------|------|---------|
| 1 | 1/s | s > 0 |
| t | 1/s² | s > 0 |
| tⁿ | n!/s^{n+1} | s > 0 |
| e^{at} | 1/(s-a) | s > a |
| sin(ωt) | ω/(s²+ω²) | s > 0 |
| cos(ωt) | s/(s²+ω²) | s > 0 |
| e^{at}sin(ωt) | ω/[(s-a)²+ω²] | s > a |
| δ(t) | 1 | 全s平面 |

### 拉普拉斯变换的重要性质

**线性性质**：L{af(t)+bg(t)} = aF(s) + bG(s)

**时域微分**：
$$L\{f'(t)\} = sF(s) - f(0)$$
$$L\{f''(t)\} = s²F(s) - sf(0) - f'(0)$$

这是用拉普拉斯变换求解ODE的关键：将微分方程转化为代数方程。

**时域平移**：L{f(t-a)u(t-a)} = e^{-as}F(s)

**频域平移**：L{e^{at}f(t)} = F(s-a)

**卷积定理**：L{f*g} = F(s)·G(s)，其中 (f*g)(t) = ∫₀ᵗ f(τ)g(t-τ)dτ

### 求解ODE例题

**例题**：求解 y'' + 3y' + 2y = 0, y(0)=1, y'(0)=0

取拉普拉斯变换：
s²Y - s·1 - 0 + 3(sY - 1) + 2Y = 0
(s²+3s+2)Y = s+3
Y = (s+3)/[(s+1)(s+2)]

部分分式展开：
(s+3)/[(s+1)(s+2)] = A/(s+1) + B/(s+2)

s=-1: 2 = A·1 → A = 2
s=-2: 1 = B·(-1) → B = -1

Y = 2/(s+1) - 1/(s+2)

逆变换：y(t) = 2e^{-t} - e^{-2t}

### 拉普拉斯变换在工程中的应用

| 领域 | 应用 |
|------|------|
| 控制系统 | 传递函数 G(s) = Y(s)/U(s)，分析系统稳定性 |
| 电路分析 | RLC 电路的 s 域分析，阻抗 Z(s) |
| 信号处理 | 滤波器设计，系统频率响应 |
| 机械振动 | 阻尼振动的解析解 |

### 易错点

1. **变换存在的条件**：f(t) 需满足指数阶增长条件
2. **初始条件必须代入**：L{f'(t)} = sF(s) - f(0)，不要忘记 f(0)
3. **部分分式展开要准确**：重根情况需 A/(s-a) + B/(s-a)² 形式
4. **逆变换要查表**：复杂表达式可能需要用卷积定理


## Python实现

### 拉普拉斯变换的符号计算

```python
import sympy as sp

t, s = sp.symbols('t s', positive=True)

# 常用拉普拉斯变换
print("=== 拉普拉斯变换 ===")
transforms = [
    (1, "1"),
    (t, "t"),
    (t**2, "t^2"),
    (sp.exp(-2*t), "e^(-2t)"),
    (sp.sin(3*t), "sin(3t)"),
    (sp.cos(3*t), "cos(3t)"),
    (t * sp.exp(-t), "t*e^(-t)"),
]
for f, name in transforms:
    F = sp.laplace_transform(f, t, s, noconds=True)
    print(f"L{{{name}}} = {sp.simplify(F)}")

# 拉普拉斯逆变换
print("\n=== 拉普拉斯逆变换 ===")
inv_transforms = [
    (1/s, "1/s → 1"),
    (1/s**2, "1/s² → t"),
    (1/(s+2), "1/(s+2) → e^(-2t)"),
    (3/(s**2+9), "3/(s²+9) → sin(3t)"),
    (s/(s**2+9), "s/(s²+9) → cos(3t)"),
]
for F, desc in inv_transforms:
    f = sp.inverse_laplace_transform(F, s, t)
    print(f"L^(-1){{{desc}}} = {f}")
```

### 用拉普拉斯变换求解 ODE

```python
import sympy as sp

t, s = sp.symbols('t s')
Y = sp.Function('Y')

# 例: y'' + 3y' + 2y = 0, y(0)=1, y'(0)=0
# 变换: (s²Y - s*1 - 0) + 3(sY - 1) + 2Y = 0
# (s² + 3s + 2)Y = s + 3

Y_s = sp.Symbol('Y_s')
eq = (s**2 + 3*s + 2) * Y_s - s - 3
Y_solved = sp.solve(eq, Y_s)[0]
print(f"Y(s) = {sp.factor(Y_solved)}")

# 部分分式展开
Y_partial = sp.apart(Y_solved, s)
print(f"部分分式: Y(s) = {Y_partial}")

# 逆变换
y_t = sp.inverse_laplace_transform(Y_solved, s, t)
print(f"y(t) = {y_t}")

# 验证
y = 2*sp.exp(-t) - sp.exp(-2*t)
y_prime = sp.diff(y, t)
y_double_prime = sp.diff(y, t, 2)
residual = y_double_prime + 3*y_prime + 2*y
print(f"验证: y'' + 3y' + 2y = {sp.simplify(residual)}")
```

### 数值拉普拉斯逆变换

```python
import numpy as np
from scipy.integrate import quad

def numerical_laplace_inverse(F_s, t_val, N=50, sigma=1.0):
    """数值拉普拉斯逆变换 (基于梯形法则)
    f(t) ≈ (e^(σt)/π) * Re[ Σ F(σ + j*k*π/T) * e^(j*k*π*t/T) ]
    """
    result = 0
    for k in range(-N, N+1):
        s_val = complex(sigma, k * np.pi / t_val)
        result += F_s(s_val) * np.exp(1j * k * np.pi)
    return np.exp(sigma * t_val) / t_val * np.real(result)

# 示例: L^{-1}{1/(s+1)} = e^(-t)
F = lambda s: 1 / (s + 1)
for t_val in [0.5, 1.0, 2.0]:
    approx = numerical_laplace_inverse(F, t_val)
    exact = np.exp(-t_val)
    print(f"t={t_val}: f(t) ≈ {approx:.6f}, 精确值 = {exact:.6f}")
```
