傅里叶级数与变换

级数与变换 · 信号的频域分析

一、三角函数系的正交性

三角函数系

{1, cos x, sin x, cos 2x, sin 2x, ..., cos nx, sin nx, ...}

在区间 [-π, π] 上，任意两个不同函数的乘积积分等于零：

∫₋πᵖ cos(mx)cos(nx)dx = 0   （m ≠ n）
∫₋πᵖ sin(mx)sin(nx)dx = 0   （m ≠ n）
∫₋πᵖ sin(mx)cos(nx)dx = 0   （对任意 m, n）

二、傅里叶级数

1. 傅里叶级数展开

周期为 2π 的函数 f(x) 的傅里叶级数：

f(x) ~ a₀/2 + Σ [aₙcos(nx) + bₙsin(nx)]

其中傅里叶系数：

aₙ = (1/π) ∫₋πᵖ f(x)cos(nx)dx   （n = 0, 1, 2, ...）

bₙ = (1/π) ∫₋πᵖ f(x)sin(nx)dx   （n = 1, 2, 3, ...）

2. 收敛定理（Dirichlet 条件）

若 f(x) 在 [-π, π] 上满足：

  1. 连续或只有有限个第一类间断点
  2. 只有有限个极值点

则傅里叶级数收敛于：

  • x 为连续点：f(x)
  • x 为间断点：[f(x⁺) + f(x⁻)] / 2
  • x = ±π：[f(-π⁺) + f(π⁻)] / 2

3. 奇偶函数的傅里叶级数

f(x) 的对称性傅里叶系数级数形式
偶函数 f(-x)=f(x)bₙ=0余弦级数：a₀/2 + Σaₙcos(nx)
奇函数 f(-x)=-f(x)aₙ=0正弦级数：Σbₙsin(nx)
例题 1：方波的傅里叶级数

将 f(x) = { 1, 0<x<π; -1, -π<x<0 } 展开为傅里叶级数。

f(x) 是奇函数 → aₙ = 0
bₙ = (2/π)∫₀ᵖ sin(nx)dx = (2/π)[-cos(nx)/n]₀ᵖ
= (2/(nπ))(1 - cos(nπ)) = (2/(nπ))(1-(-1)ⁿ)

n 为奇数时 bₙ = 4/(nπ)，n 为偶数时 bₙ = 0

f(x) = (4/π)[sinx + sin(3x)/3 + sin(5x)/5 + ...]

三、周期为 2L 的傅里叶级数

公式

f(x) ~ a₀/2 + Σ [aₙcos(nπx/L) + bₙsin(nπx/L)]

aₙ = (1/L) ∫₋ᴸᴸ f(x)cos(nπx/L)dx

bₙ = (1/L) ∫₋ᴸᴸ f(x)sin(nπx/L)dx

四、傅里叶变换

1. 从傅里叶级数到傅里叶变换

当周期 T → ∞ 时，离散的频率 n/T 变为连续的频率 ω，级数变为积分：

傅里叶变换（时域 → 频域）：

F(ω) = ∫₋∞⁺∞ f(t) e⁻ʲωᵗ dt

傅里叶逆变换（频域 → 时域）：

f(t) = (1/2π) ∫₋∞⁺∞ F(ω) eʲωᵗ dω

2. 基本性质

性质时域频域
线性af(t) + bg(t)aF(ω) + bG(ω)
时移f(t - t₀)F(ω)e⁻ʲωᵗ⁰
频移f(t)eʲω₀ᵗF(ω - ω₀)
尺度变换f(at)(1/|a|)F(ω/a)
时域微分f'(t)jωF(ω)
卷积f(t)*g(t)F(ω)·G(ω)

卷积定理：时域卷积 = 频域乘积，这是傅里叶变换最重要的性质之一。

例题 2：矩形脉冲的傅里叶变换

求 f(t) = { 1, |t| ≤ T; 0, |t| > T } 的傅里叶变换。

F(ω) = ∫₋ᵀᵀ 1 · e⁻ʲωᵗ dt = [e⁻ʲωᵗ/(-jω)]₋ᵀᵀ
= (e⁻ʲωᵀ - eʲωᵀ)/(-jω) = 2sin(ωT)/ω
F(ω) = 2T · sin(ωT)/(ωT) = 2T · sinc(ωT)
傅里叶变换的广泛应用：
  • 信号处理：频谱分析、滤波、信号压缩
  • 图像处理：2D-FFT 用于图像滤波、压缩（JPEG）
  • 通信：调制解调、OFDM、信道分析
  • 量子力学：动量空间与位置空间的变换
  • 偏微分方程：将 PDE 转化为 ODE 求解
  • 数据压缩：MP3（音频）、JPEG（图像）基于 DCT（离散余弦变换）

本节总结

  • 傅里叶级数：将周期函数分解为三角函数之和
  • 傅里叶系数：aₙ = (1/π)∫f(x)cos(nx)dx, bₙ = (1/π)∫f(x)sin(nx)dx
  • 奇偶函数：偶函数只有余弦项，奇函数只有正弦项
  • 傅里叶变换：F(ω) = ∫f(t)e⁻ʲωᵗdt，时域与频域之间的桥梁
  • 卷积定理：时域卷积 = 频域乘积
  • 傅里叶分析是现代信号处理、通信、图像处理的数学基础
