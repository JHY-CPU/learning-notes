# 43-微积分

## 课程概述

本笔记系统整理微积分的核心内容，涵盖从极限基础到多元微积分、级数与变换的完整知识体系。微积分是现代数学的基石，也是机器学习、信号处理、物理学、工程学等领域的核心工具。本系列笔记不仅注重数学理论的严谨性，同时强调在计算机科学与人工智能中的实际应用。

## 01-极限与连续

极限是微积分的出发点。本模块深入讲解：
- **ε-N定义**：数列极限的严格定义，用以精确刻画"任意接近"的概念
- **ε-δ定义**：函数极限的严格定义，统一处理双侧极限
- **极限运算法则**：四则运算、夹逼准则、单调有界定理
- **两个重要极限**：$\lim_{x\to0}\frac{\sin x}{x}=1$ 与 $\lim_{x\to\infty}(1+\frac{1}{x})^x=e$
- **无穷小与无穷大**：阶的比较（高阶、低阶、同阶、k阶）、等价无穷小替换
- **连续性与间断点**：左/右连续、第一类（可去/跳跃）与第二类（无穷/振荡）间断点
- **闭区间性质**：有界性定理、最值定理、零点定理、介值定理

## 02-一元微分学

微分学研究函数的变化率。本模块涵盖：
- **导数定义**：从切线斜率与瞬时速度出发，建立导数的严格定义
- **求导法则**：四则运算、链式法则、反函数求导、隐函数求导、参数方程求导
- **高阶导数**：Leibniz公式 $(uv)^{(n)}=\sum_{k=0}^{n}\binom{n}{k}u^{(k)}v^{(n-k)}$
- **微分**：线性近似 $f(x+\Delta x)\approx f(x)+f'(x)\Delta x$
- **中值定理**：Rolle定理、Lagrange中值定理、Cauchy中值定理的递进关系
- **Taylor公式**：函数的多项式逼近，常见函数的Maclaurin展开式
- **L'Hôpital法则**：$\frac{0}{0}$与$\frac{\infty}{\infty}$型未定式的处理
- **函数性态**：单调性判别、极值、凹凸性、拐点、渐近线、曲率

## 03-一元积分学

积分学是微分学的逆运算。本模块涵盖：
- **不定积分**：原函数的概念、基本积分表、积分公式推导
- **换元积分法**：第一类（凑微分法）、第二类（变量替换法）
- **分部积分法**：$\int u\,dv=uv-\int v\,du$ 及选u的LIATE原则
- **有理函数积分**：部分分式分解的系统方法
- **定积分（Riemann积分）**：从分割、取点、求和到取极限的严格构造
- **微积分基本定理**：Newton-Leibniz公式 $\int_a^bf(x)dx=F(b)-F(a)$
- **反常积分**：无穷区间积分（$\int_a^{+\infty}$）与无界函数积分（瑕积分）
- **积分应用**：平面面积、旋转体体积、弧长、旋转曲面积
- **Gamma与Beta函数**：$\Gamma(n)=\int_0^{+\infty}x^{n-1}e^{-x}dx$，在概率统计中的重要性

## 04-多元微积分

从一元推广到多元，几何直觉与代数工具并重。本模块涵盖：
- **多元函数极限与连续**：$(x,y)\to(x_0,y_0)$ 与路径无关性
- **偏导数与全微分**：$\frac{\partial f}{\partial x}$, $df=\frac{\partial f}{\partial x}dx+\frac{\partial f}{\partial y}dy$
- **链式法则**：复合函数 $z=f(u(x,y),v(x,y))$ 的求导
- **隐函数求导**：$F(x,y)=0$ 确定 $y=y(x)$ 与方程组情形
- **方向导数与梯度**：$\nabla f=\left(\frac{\partial f}{\partial x_1},\ldots,\frac{\partial f}{\partial x_n}\right)$
- **极值与条件极值**：Hessian矩阵判别、Lagrange乘数法
- **重积分**：二重积分（Fubini定理换序）、三重积分（柱/球坐标变换）
- **曲线积分与曲面积分**：第一型（标量场）、第二型（向量场）
- **场论三大公式**：Green、Stokes、Gauss公式——微积分在高维的统一
- **机器学习应用**：梯度下降、凸优化基础、Jacobian与Hessian矩阵

## 05-级数与变换

级数是无穷求和，变换是函数的工具性改写。本模块涵盖：
- **数项级数**：收敛的必要条件、比较/比值/根值/积分判别法
- **正项级数与交错级数**：Leibniz判别法、绝对收敛与条件收敛
- **幂级数**：收敛半径（Cauchy-Hadamard公式）、逐项求导与逐项积分
- **函数的幂级数展开**：Taylor级数的构造与收敛性
- **Fourier级数**：三角级数展开、Dirichlet收敛条件、Parseval等式
- **Fourier变换**：$F(\omega)=\int_{-\infty}^{+\infty}f(t)e^{-i\omega t}dt$ 及其性质
- **Laplace变换**：$F(s)=\int_0^{+\infty}f(t)e^{-st}dt$，解微分方程的有力工具
- **Z变换**：离散信号的Laplace变换，数字信号处理的基础
- **应用**：频谱分析、卷积定理、深度学习中的注意力机制与位置编码

## 学习建议

1. **先理解定义再做题**：每个概念的严格定义是理解后续推导的前提
2. **注重几何直觉**：极限、导数、积分都有清晰的几何含义，善用图像辅助理解
3. **掌握计算技巧**：微积分有大量计算，需要通过练习形成"计算直觉"
4. **联系应用**：将数学概念与机器学习、信号处理等实际问题联系，加深理解
5. **建立知识网络**：理解各模块之间的内在联系（如微分与积分的互逆、级数与变换的统一）
