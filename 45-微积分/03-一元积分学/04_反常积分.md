# 反常积分 - 一元积分学

*Improper Integrals — 一元积分学 · 第四节*


### 1.1 基本类型


$$
∫a+∞ f(x) dx = limt→+∞ ∫at f(x) dx
$$


$$
∫−∞b f(x) dx = limt→−∞ ∫tb f(x) dx
$$


$$
∫−∞+∞ f(x) dx = ∫−∞c f(x) dx + ∫c+∞ f(x) dx
$$


（需两部分都收敛，取任意中间点 c）


### 2.1 常见瑕点位置


- **左端点瑕点**
   ：lim
   ~x→a⁺~
   f(x) = ∞，取 lim
   ~t→a⁺~
   ∫
   ~t~
   ^b^
- **右端点瑕点**
   ：lim
   ~x→b⁻~
   f(x) = ∞，取 lim
   ~t→b⁻~
   ∫
   ~a~
   ^t^
- **内部瑕点**
   ：c ∈ (a,b) 使得 lim
   ~x→c~
   f(x) = ∞，拆分为 ∫
   ~a~
   ^c^
   + ∫
   ~c~
   ^b^


### 3.1 比较判别法（Comparison Test）


> **Theorem:** 设 0 ≤ f(x) ≤ g(x)（x ≥ a）：
>
>
> - 若 ∫
>    ~a~
>    ^∞^
>    g(x) dx 收敛 ⟹ ∫
>    ~a~
>    ^∞^
>    f(x) dx
>    收敛
> - 若 ∫
>    ~a~
>    ^∞^
>    f(x) dx 发散 ⟹ ∫
>    ~a~
>    ^∞^
>    g(x) dx
>    发散


### 3.2 极限比较判别法


> **Theorem:** 设 f(x) ≥ 0, g(x) > 0，若 lim~x→∞~ f(x)/g(x) = L：
>
>
> - 0 < L < +∞：两者
>    **同敛散**
> - L = 0 且 g 收敛 ⟹ f 收敛
> - L = +∞ 且 g 发散 ⟹ f 发散


### 3.3 绝对收敛与条件收敛


> **Theorem:** 若 ∫~a~^∞^ |f(x)| dx 收敛 ⟹ ∫~a~^∞^ f(x) dx 收敛（**绝对收敛**）
>
>
> 反向不成立：∫ f(x) dx 收敛但 ∫ |f(x)| dx 发散 → **条件收敛**


### 4.1 基本性质


$$
Γ(s+1) = s · Γ(s)（递推公式）
$$


$$
Γ(n+1) = n! （n 为正整数）
$$


$$
Γ(1) = 1
$$


$$
Γ(1/2) = √π
$$


### 4.2 其他表示


$$
令 x = t²，得：Γ(s) = 2 ∫0∞ t2s−1 e−t² dt
$$


$$
特别地：Γ(1/2) = 2 ∫0∞ e−t² dt = √π
$$


$$
高斯积分：∫−∞+∞ e−x² dx = √π
$$


### 5.1 Beta 函数的性质


$$
对称性：B(p, q) = B(q, p)
$$


$$
与 Gamma 的关系：B(p, q) = Γ(p) · Γ(q) / Γ(p+q)
$$


$$
三角函数形式：B(p, q) = 2 ∫0π/2 sin2p−1θ · cos2q−1θ dθ
$$


## Python实现

### 反常积分的数值计算与符号求解

```python
import numpy as np
from scipy.integrate import quad
import sympy as sp

x = sp.Symbol('x')

# 无穷区间积分
print("=== 无穷区间反常积分 ===")

# ∫_1^∞ 1/x^2 dx = 1 (收敛)
result, _ = quad(lambda t: 1/t**2, 1, np.inf)
print(f"∫_1^∞ 1/x² dx = {result:.6f}")

# ∫_1^∞ 1/x dx = ∞ (发散)
result, _ = quad(lambda t: 1/t, 1, np.inf, limit=200)
print(f"∫_1^∞ 1/x dx = {result:.2f} (发散, 积分值很大)")

# Γ函数: Γ(n) = ∫_0^∞ t^(n-1) e^(-t) dt
print(f"\n=== Γ 函数 ===")
from scipy.special import gamma
for n in [0.5, 1, 2, 3, 4, 5]:
    print(f"Γ({n}) = {gamma(n):.6f}")
print(f"Γ(1/2) = √π ≈ {np.sqrt(np.pi):.6f}")

# 高斯积分: ∫_{-∞}^{+∞} e^(-x^2) dx = √π
result, _ = quad(lambda t: np.exp(-t**2), -np.inf, np.inf)
print(f"\n高斯积分 = {result:.10f}, √π = {np.sqrt(np.pi):.10f}")

# 无界函数积分 (瑕积分)
print(f"\n=== 瑕积分 ===")
# ∫_0^1 1/√x dx = 2 (收敛, p=1/2 < 1)
result, _ = quad(lambda t: 1/np.sqrt(t), 0, 1)
print(f"∫_0^1 1/√x dx = {result:.6f}")

# 用 sympy 求反常积分
print(f"\n=== SymPy 求解 ===")
print(f"∫_1^∞ 1/x^2 dx = {sp.integrate(1/x**2, (x, 1, sp.oo))}")
print(f"∫_0^1 1/√x dx = {sp.integrate(1/sp.sqrt(x), (x, 0, 1))}")
```
