反常积分

Improper Integrals — 一元积分学 · 第四节

一、第一类反常积分（无穷限积分）

积分限为无穷的积分。设 f(x) 在 [a, +∞) 上连续：

∫a+∞ f(x) dx = limt→+∞ ∫at f(x) dx

若极限存在，则称反常积分收敛；否则称发散。

1.1 基本类型

∫a+∞ f(x) dx = limt→+∞ ∫at f(x) dx

∫−∞b f(x) dx = limt→−∞ ∫tb f(x) dx

∫−∞+∞ f(x) dx = ∫−∞c f(x) dx + ∫c+∞ f(x) dx

（需两部分都收敛，取任意中间点 c）

例：讨论 ∫1+∞ 1/xp dx 的收敛性

p ≠ 1：∫1t x−p dx = [x1−p/(1−p)]1t

p > 1：t1−p → 0，积分 = 1/(p−1) 收敛

p < 1：t1−p → +∞，积分 发散

p = 1：∫ 1/x dx = ln t → +∞，发散

结论：p > 1 收敛，p ≤ 1 发散

二、第二类反常积分（无界函数积分 / 瑕积分）

被积函数在积分区间内某点无界（瑕点）。设 f(x) 在 (a, b] 上连续，且 limx→a⁺ f(x) = ∞：

∫ab f(x) dx = limt→a⁺ ∫tb f(x) dx

2.1 常见瑕点位置

  • 左端点瑕点：limx→a⁺ f(x) = ∞，取 limt→a⁺ ∫tb
  • 右端点瑕点：limx→b⁻ f(x) = ∞，取 limt→b⁻ ∫at
  • 内部瑕点：c ∈ (a,b) 使得 limx→c f(x) = ∞，拆分为 ∫ac + ∫cb

例：讨论 ∫01 1/xq dx 的收敛性

x = 0 为瑕点

q < 1：收敛，积分值 = 1/(1−q)

q ≥ 1：发散

第一类与第二类的对比记忆：

类型 问题 p-积分判别
第一类（无穷限） ∫₁∞ 1/xp p > 1 收敛
第二类（瑕积分） ∫₀1 1/xq q < 1 收敛

三、收敛性判别法

3.1 比较判别法（Comparison Test）

设 0 ≤ f(x) ≤ g(x)（x ≥ a）：

  • 若 ∫a∞ g(x) dx 收敛 ⟹ ∫a∞ f(x) dx 收敛
  • 若 ∫a∞ f(x) dx 发散 ⟹ ∫a∞ g(x) dx 发散

3.2 极限比较判别法

设 f(x) ≥ 0, g(x) > 0，若 limx→∞ f(x)/g(x) = L：

  • 0 < L < +∞：两者同敛散
  • L = 0 且 g 收敛 ⟹ f 收敛
  • L = +∞ 且 g 发散 ⟹ f 发散

3.3 绝对收敛与条件收敛

若 ∫a∞ |f(x)| dx 收敛 ⟹ ∫a∞ f(x) dx 收敛（绝对收敛）

反向不成立：∫ f(x) dx 收敛但 ∫ |f(x)| dx 发散 → 条件收敛

例：判断 ∫1∞ sin(x)/x² dx 的收敛性

|sin(x)/x²| ≤ 1/x²

∫1∞ 1/x² dx 收敛（p = 2 > 1）

由比较判别法，原积分 绝对收敛

四、Gamma 函数（Γ 函数）

Gamma 函数是反常积分的一个重要应用：

Γ(s) = ∫0+∞ xs−1 e−x dx，(s > 0)

4.1 基本性质

Γ(s+1) = s · Γ(s)（递推公式）

Γ(n+1) = n! （n 为正整数）

Γ(1) = 1

Γ(1/2) = √π

4.2 其他表示

令 x = t²，得：Γ(s) = 2 ∫0∞ t2s−1 e−t² dt

特别地：Γ(1/2) = 2 ∫0∞ e−t² dt = √π

高斯积分：∫−∞+∞ e−x² dx = √π

五、Beta 函数（B 函数）

Beta 函数定义为：

B(p, q) = ∫01 xp−1 (1−x)q−1 dx，(p > 0, q > 0)

5.1 Beta 函数的性质

对称性：B(p, q) = B(q, p)

与 Gamma 的关系：B(p, q) = Γ(p) · Γ(q) / Γ(p+q)

三角函数形式：B(p, q) = 2 ∫0π/2 sin2p−1θ · cos2q−1θ dθ

例：计算 ∫01 x³(1−x)² dx

= B(4, 3) = Γ(4)Γ(3)/Γ(7) = 3! · 2! / 6! = 12/720 = 1/60

Gamma 和 Beta 函数在概率论（Beta 分布、Gamma 分布）、物理学和工程学中有广泛应用。

六、常用反常积分速查

积分 结论 条件
∫1∞ 1/xp dx p > 1 收敛，p ≤ 1 发散 无穷限
∫01 1/xq dx q < 1 收敛，q ≥ 1 发散 瑕积分
∫a∞ e−kx dx k > 0 收敛 指数衰减
∫1∞ 1/(x(ln x)p) dx p > 1 收敛 对数判别
∫0∞ e−x² dx 收敛 = √π/2 高斯积分

核心要点汇总

  • 第一类（无穷限）：取极限 limt→∞ ∫at f dx
  • 第二类（瑕积分）：瑕点处取极限 limt→a⁺ ∫tb f dx
  • p-积分判别：∫₁∞ 1/xp 在 p > 1 收敛；∫₀1 1/xq 在 q < 1 收敛
  • 比较判别法：用已知敛散性的"标杆"函数进行比较
  • 绝对收敛 ⟹ 收敛（但反之不一定）
  • Γ(s) = ∫₀∞ xs-1e-xdx，Γ(n+1) = n!，Γ(1/2) = √π
  • B(p,q) = Γ(p)Γ(q)/Γ(p+q)
