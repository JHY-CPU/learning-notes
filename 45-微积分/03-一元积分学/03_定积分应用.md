# 定积分应用 - 一元积分学

*Applications of Definite Integrals — 一元积分学 · 第三节*


### 1.1 直角坐标系


由 y = f(x), y = g(x)（f(x) ≥ g(x)）, x = a, x = b 围成的面积：


$$
A = ∫ab [f(x) − g(x)] dx
$$


由 x = φ(y), x = ψ(y)（φ(y) ≥ ψ(y)）, y = c, y = d 围成的面积（对 y 积分）：


$$
A = ∫cd [φ(y) − ψ(y)] dy
$$


### 1.2 参数方程形式


曲线由参数方程 x = x(t), y = y(t) 给出，t ∈ [α, β]：


$$
A = ∫αβ |y(t) · x′(t)| dt
$$


### 1.3 极坐标形式


极坐标曲线 r = r(θ)，θ ∈ [α, β] 围成的扇形面积：


$$
A = (1/2) ∫αβ [r(θ)]² dθ
$$


### 3.1 圆盘法（Disk Method）


y = f(x) 绕 x 轴旋转，x ∈ [a, b]：


$$
V = π ∫ab [f(x)]² dx
$$


### 3.2 垫圈法（Washer Method）


y = f(x)（外半径）和 y = g(x)（内半径）绕 x 轴旋转：


$$
V = π ∫ab ([f(x)]² − [g(x)]²) dx
$$


### 3.3 壳层法（Shell Method）


y = f(x) 绕 y 轴旋转，x ∈ [a, b]：


$$
V = 2π ∫ab x · f(x) dx
$$


物理直觉：薄壳体积 = 周长(2πx) × 高度(f(x)) × 厚度(dx)


### 参数方程形式


$$
S = 2π ∫αβ |y(t)| √([x′(t)]² + [y′(t)]²) dt
$$


### 5.1 变力做功


$$
力 F(x) 沿 x 方向从 a 到 b 做的功：
$$


$$
W = ∫ab F(x) dx
$$


### 5.2 液体压力


$$
浸入液体的薄板受到的总压力：
$$


$$
P = ρg ∫ab y · w(y) dy
$$


ρ 为液体密度，g 为重力加速度，w(y) 为深度 y 处的板宽


### 5.3 质心与形心


$$
线密度为 ρ(x) 的细棒的质心：
$$


$$
x̄ = ∫ x · ρ(x) dx / ∫ ρ(x) dx
$$


$$
平面图形的形心（面积加权）：
$$


$$
x̄ = ∫ab x[f(x)−g(x)] dx / A
$$


$$
ȳ = (1/2) ∫ab [f²(x)−g²(x)] dx / A
$$


### 5.4 函数平均值


$$
favg = (1/(b−a)) ∫ab f(x) dx
$$


## Python实现

### 用积分计算面积和体积

```python
import numpy as np
from scipy.integrate import quad
import matplotlib.pyplot as plt

# 1. 计算面积: y=x^2 和 y=x 围成的区域
# 交点: x^2 = x → x=0, x=1
area, _ = quad(lambda x: x - x**2, 0, 1)
print(f"y=x² 和 y=x 围成的面积 = {area:.6f} (精确值 1/6)")

# 2. 旋转体体积 (绕 x 轴)
# y = sqrt(x), x ∈ [0,4], 绕 x 轴旋转
V_disk, _ = quad(lambda x: np.pi * x, 0, 4)  # π * (sqrt(x))^2 = πx
print(f"旋转体体积 (圆盘法) = {V_disk:.4f}")

# 3. 壳层法: y=x^2, x∈[0,1], 绕 y 轴旋转
V_shell, _ = quad(lambda x: 2*np.pi*x * x**2, 0, 1)
print(f"旋转体体积 (壳层法) = {V_shell:.6f} = π/2")

# 4. 弧长: y = (2/3)*x^(3/2), x∈[0,3]
# L = ∫√(1 + f'(x)^2) dx = ∫√(1+x) dx
arc_length, _ = quad(lambda x: np.sqrt(1 + x), 0, 3)
print(f"弧长 = {arc_length:.6f}")

# 5. 质心: 均匀半圆盘 x^2+y^2 ≤ 1, y≥0
# y_bar = (1/A) * (1/2) * ∫_{-1}^1 (1-x^2) dx
A_half_circle = np.pi / 2  # 半圆面积
y_bar, _ = quad(lambda x: 0.5*(1-x**2), -1, 1)
y_bar /= A_half_circle
print(f"半圆盘形心 y_bar = {y_bar:.6f} = 4/(3π)")
```

### 极坐标面积

```python
import numpy as np
from scipy.integrate import quad

# 心形线 r = 1 + cos(θ), θ ∈ [0, 2π]
# 面积 A = (1/2) ∫ r(θ)^2 dθ
r_func = lambda theta: 1 + np.cos(theta)
area, _ = quad(lambda theta: 0.5 * r_func(theta)**2, 0, 2*np.pi)
print(f"心形线面积 = {area:.6f} = 3π/2 ≈ {3*np.pi/2:.6f}")

# 可视化
theta = np.linspace(0, 2*np.pi, 500)
r = 1 + np.cos(theta)
x = r * np.cos(theta)
y = r * np.sin(theta)
plt.figure(figsize=(6, 6))
plt.fill(x, y, alpha=0.3, color='red')
plt.plot(x, y, 'b-')
plt.title('心形线 r = 1 + cos(θ)')
plt.axis('equal')
plt.savefig('cardioid.png', dpi=100)
plt.show()
```
