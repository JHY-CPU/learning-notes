不定积分与定积分

一元积分学 · 积分的基本理论

一、不定积分

1. 定义

若 F'(x) = f(x)，则称 F(x) 是 f(x) 的一个原函数。f(x) 的所有原函数称为 f(x) 的不定积分：

∫ f(x) dx = F(x) + C

几何意义：f(x) 的一族积分曲线，它们之间相差常数 C。

3. 运算法则

∫ [f(x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx

∫ k·f(x) dx = k · ∫ f(x) dx   （k 为常数）

二、定积分

1. 黎曼和与定积分定义

将 [a, b] 分割为 n 个小区间，取每个小区间中的点 ξᵢ，构造黎曼和：

∫ₐᵇ f(x) dx = lim(n→∞) Σ f(ξᵢ) · Δxᵢ

几何意义：曲线 y=f(x) 与 x 轴之间、x=a 到 x=b 所围成的有向面积（x 轴上方为正，下方为负）。

2. 定积分的性质

  • ∫ₐᵇ f(x)dx = -∫ᵇₐ f(x)dx
  • ∫ₐᵃ f(x)dx = 0
  • ∫ₐᵇ [f(x)±g(x)]dx = ∫ₐᵇ f(x)dx ± ∫ₐᵇ g(x)dx
  • 积分中值定理：∃ ξ ∈ [a,b] 使 ∫ₐᵇ f(x)dx = f(ξ)(b-a)
  • 若 f(x) ≤ g(x)，则 ∫ₐᵇ f(x)dx ≤ ∫ₐᵇ g(x)dx

三、牛顿-莱布尼茨公式（微积分基本定理）

公式

若 F'(x) = f(x)，f(x) 在 [a, b] 上连续，则：

∫ₐᵇ f(x) dx = F(b) - F(a) = [F(x)]ₐᵇ

意义：定积分 = 原函数在上下限处的值之差。将积分问题转化为求原函数问题。

变上限积分

Φ(x) = ∫ₐˣ f(t)dt    则    Φ'(x) = f(x)

变上限积分的导数等于被积函数在上限处的值。这是连接积分与微分的桥梁。

例题 1：牛顿-莱布尼茨公式

计算 ∫₀² (3x² + 2x) dx。

原函数 F(x) = x³ + x²
F(2) - F(0) = (8 + 4) - 0 = 12
例题 2：变上限积分求导

设 F(x) = ∫₀ˣ² e^(-t²) dt，求 F'(x)。

令 u = x²，则 F(x) = ∫₀ᵘ e^(-t²) dt
链式法则：F'(x) = [d/du ∫₀ᵘ e^(-t²)dt] · du/dx
= e^(-u²) · 2x = 2x·e^(-x⁴)
不定积分 vs 定积分：
  • 不定积分：原函数族 F(x)+C，结果是函数
  • 定积分：黎曼和的极限，结果是一个数
  • 牛顿-莱布尼茨公式用不定积分（原函数）来计算定积分
  • 不定积分是"逆微分"，定积分是"求面积"

本节总结

  • 不定积分：∫f(x)dx = F(x)+C，原函数族
  • 定积分：黎曼和的极限，有向面积
  • 牛顿-莱布尼茨公式：∫ₐᵇf(x)dx = F(b)-F(a)
  • 变上限积分：d/dx ∫ₐˣf(t)dt = f(x)
  • 基本积分公式和求导公式互逆，要熟记
