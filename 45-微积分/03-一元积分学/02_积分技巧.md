# 积分技巧

*一元积分学 · 换元法、分部积分、有理函数积分*


### 公式


若 ∫f(u)du = F(u) + C，则


$$
∫ f(φ(x)) · φ'(x) dx = ∫ f(u) du = F(u) + C = F(φ(x)) + C
$$


关键：将被积函数凑成 f(φ(x))·φ'(x) 的形式，令 u = φ(x)。


### 公式


令 x = φ(t)，φ'(t) ≠ 0，则


$$
∫ f(x) dx = ∫ f(φ(t)) · φ'(t) dt
$$


换元后求出关于 t 的原函数，再将 t = φ⁻¹(x) 代回。


### 常用三角换元


| 被积函数含 | 换元 | 恒等式 |
| --- | --- | --- |
| √(a²-x²) | x = asint | 1-sin²t = cos²t |
| √(a²+x²) | x = atant | 1+tan²t = sec²t |
| √(x²-a²) | x = asect | sec²t-1 = tan²t |


### 公式


$$
∫ u dv = uv - ∫ v du
$$


关键在于选择 u 和 dv。口诀："反对幂指三"——对数、反三角、幂函数、指数、三角，排在前面的优先选作 u。


### 选择 u 的经验法则（LIATE）


- **L**
   ogarithmic（对数函数）：ln x, log x
- **I**
   nverse trig（反三角函数）：arcsin x, arctan x
- **A**
   lgebraic（代数函数）：xⁿ, 多项式
- **T**
   rigonometric（三角函数）：sin x, cos x
- **E**
   xponential（指数函数）：eˣ, aˣ


排在前面的优先选作 u（因为它求导后会简化）。


### 步骤


1. 若分子次数 ≥ 分母次数，先做多项式除法
2. 将分母因式分解（实数范围内分解为一次因式和不可约二次因式）
3. 拆成部分分式
4. 逐项积分


### 部分分式形式


| 分母因子 | 部分分式形式 |
| --- | --- |
| (x-a) | A/(x-a) |
| (x-a)ᵏ | A₁/(x-a) + A₂/(x-a)² + ... + Aₖ/(x-a)ᵏ |
| x²+px+q（不可约） | (Ax+B)/(x²+px+q) |

### 常见积分技巧总结

**三角函数积分**：
- ∫sin²x dx = x/2 - sin(2x)/4 + C
- ∫cos²x dx = x/2 + sin(2x)/4 + C
- ∫tan x dx = -ln|cos x| + C
- ∫sec x dx = ln|sec x + tan x| + C

**含根式的积分**：
- ∫1/√(a²-x²) dx = arcsin(x/a) + C
- ∫1/(a²+x²) dx = (1/a)arctan(x/a) + C
- ∫1/√(x²+a²) dx = ln|x+√(x²+a²)| + C
- ∫1/√(x²-a²) dx = ln|x+√(x²-a²)| + C

**递推公式法**：

对于 Iₙ = ∫sinⁿx dx 或 ∫cosⁿx dx，利用分部积分建立递推：
$$Iₙ = -1/n · sin^{n-1}x · cos x + (n-1)/n · I_{n-2}$$

### 分部积分的 LIATE 选择实战

**例题**：求 ∫x²eˣ dx

选择 u = x² (A-代数), dv = eˣdx (E-指数)
du = 2x dx, v = eˣ
∫x²eˣ dx = x²eˣ - ∫2xeˣ dx

再做一次：∫2xeˣ dx = 2xeˣ - 2eˣ + C

结果：∫x²eˣ dx = (x²-2x+2)eˣ + C

### 有理函数积分完整流程

**例题**：求 ∫(x+1)/(x²+x-2) dx

1. 因式分解：x²+x-2 = (x+2)(x-1)
2. 部分分式：(x+1)/[(x+2)(x-1)] = A/(x+2) + B/(x-1)
3. 通分求系数：x+1 = A(x-1) + B(x+2)
   - x=1: 2 = 3B → B = 2/3
   - x=-2: -1 = -3A → A = 1/3
4. 积分：(1/3)ln|x+2| + (2/3)ln|x-1| + C

### 易错点

1. **凑微分要保持等号**：∫f(g(x))g'(x)dx 中 g'(x) 必须完整出现
2. **三角换元后要回代**：t = arcsin(x/a) 代回原变量
3. **分部积分可能循环**：如 ∫eˣsinx dx 需要做两次分部，然后解方程
4. **部分分式系数不能出错**：建议代入特殊值法验证


## Python实现

### 分部积分与换元法验证

```python
import sympy as sp

x = sp.Symbol('x')

# 分部积分: ∫x*e^x dx = (x-1)e^x + C
result = sp.integrate(x * sp.exp(x), x)
print(f"∫x*e^x dx = {result}")

# 分部积分: ∫x^2*sin(x) dx
result = sp.integrate(x**2 * sp.sin(x), x)
print(f"∫x^2*sin(x) dx = {sp.expand(result)}")

# 三角换元: ∫1/sqrt(1-x^2) dx = arcsin(x)
result = sp.integrate(1/sp.sqrt(1-x**2), x)
print(f"∫1/√(1-x²) dx = {result}")

# 有理函数积分 (部分分式)
result = sp.integrate((x+1)/(x**2+x-2), x)
print(f"∫(x+1)/(x²+x-2) dx = {result}")

# 验证: 对结果求导应等于被积函数
deriv = sp.diff(result, x)
simplified = sp.simplify(deriv)
print(f"验证 (对结果求导): {simplified} = (x+1)/(x²+x-2)?")

# 复杂积分示例
print("\n=== 更多积分 ===")
print(f"∫tan(x) dx = {sp.integrate(sp.tan(x), x)}")
print(f"∫sec(x) dx = {sp.integrate(1/sp.cos(x), x)}")
print(f"∫1/(1+x^2) dx = {sp.integrate(1/(1+x**2), x)}")
print(f"∫ln(x) dx = {sp.integrate(sp.ln(x), x)}")

# 定积分: 换元后需换限
# ∫_0^1 x*sqrt(1-x^2) dx, 令 u=1-x^2
result = sp.integrate(x*sp.sqrt(1-x**2), (x, 0, 1))
print(f"\n∫_0^1 x√(1-x²) dx = {result}")
```

### 数值验证积分技巧

```python
import numpy as np
from scipy.integrate import quad

# 验证分部积分: ∫_0^π x*sin(x) dx = π
result, _ = quad(lambda x: x*np.sin(x), 0, np.pi)
print(f"∫_0^π x*sin(x) dx = {result:.6f} (精确值 = {np.pi:.6f})")

# 验证对称区间奇偶性
# ∫_{-1}^1 x^3 dx = 0 (奇函数)
result, _ = quad(lambda x: x**3, -1, 1)
print(f"∫_(-1)^1 x³ dx = {result:.10f} (奇函数, 应为0)")

# ∫_{-1}^1 x^2 dx = 2/3 (偶函数)
result, _ = quad(lambda x: x**2, -1, 1)
print(f"∫_(-1)^1 x² dx = {result:.6f} = 2∫_0^1 x² dx = {2*1/3:.6f}")
```
