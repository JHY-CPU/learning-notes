定积分应用

Applications of Definite Integrals — 一元积分学 · 第三节

一、平面图形的面积

1.1 直角坐标系

由 y = f(x), y = g(x)（f(x) ≥ g(x)）, x = a, x = b 围成的面积：

A = ∫ab [f(x) − g(x)] dx

由 x = φ(y), x = ψ(y)（φ(y) ≥ ψ(y)）, y = c, y = d 围成的面积（对 y 积分）：

A = ∫cd [φ(y) − ψ(y)] dy

1.2 参数方程形式

曲线由参数方程 x = x(t), y = y(t) 给出，t ∈ [α, β]：

A = ∫αβ |y(t) · x′(t)| dt

1.3 极坐标形式

极坐标曲线 r = r(θ)，θ ∈ [α, β] 围成的扇形面积：

A = (1/2) ∫αβ [r(θ)]² dθ

例：求 r = 2a cos θ 围成的面积（圆）

A = (1/2) ∫−π/2π/2 (2a cos θ)² dθ = (1/2) · 4a² ∫ cos²θ dθ

= 2a² · (π/2) = πa²（验证了 r = 2a cos θ 是半径为 a 的圆）

二、弧长（Arc Length）

曲线形式 弧长公式
y = f(x), x ∈ [a, b] s = ∫ab √(1 + [f′(x)]²) dx
x = x(t), y = y(t), t ∈ [α, β] s = ∫αβ √([x′(t)]² + [y′(t)]²) dt
r = r(θ), θ ∈ [α, β] s = ∫αβ √([r(θ)]² + [r′(θ)]²) dθ

微分弧长元素（统一形式）：

ds = √(dx² + dy²)

例：求悬链线 y = cosh x 在 [−1, 1] 上的弧长

y′ = sinh x, 1 + (y′)² = 1 + sinh²x = cosh²x

s = ∫−11 cosh x dx = [sinh x]−11 = 2 sinh 1 ≈ 2.350

3.1 圆盘法（Disk Method）

y = f(x) 绕 x 轴旋转，x ∈ [a, b]：

V = π ∫ab [f(x)]² dx

3.2 垫圈法（Washer Method）

y = f(x)（外半径）和 y = g(x)（内半径）绕 x 轴旋转：

V = π ∫ab ([f(x)]² − [g(x)]²) dx

3.3 壳层法（Shell Method）

y = f(x) 绕 y 轴旋转，x ∈ [a, b]：

V = 2π ∫ab x · f(x) dx

物理直觉：薄壳体积 = 周长(2πx) × 高度(f(x)) × 厚度(dx)

例：求 y = √x 绕 x 轴旋转，x ∈ [0, 4] 的体积

V = π ∫04 (√x)² dx = π ∫04 x dx = π[x²/2]04 = 8π

选择方法的建议：

  • 绕 x 轴旋转 → 用圆盘法（对 x 积分）
  • 绕 y 轴旋转 → 用壳层法（对 x 积分）或圆盘法（对 y 积分）
  • 被积函数简单优先：选择使积分更容易的方法

四、旋转体表面积（Surface Area of Revolution）

y = f(x) ≥ 0 绕 x 轴旋转的曲面面积：

S = 2π ∫ab f(x) √(1 + [f′(x)]²) dx

= ∫ 2πy ds （周长 × 弧长元素）

参数方程形式

S = 2π ∫αβ |y(t)| √([x′(t)]² + [y′(t)]²) dt

例：求球面面积（半径 R 的圆绕直径旋转）

y = √(R² − x²), y′ = −x/√(R² − x²)

1 + (y′)² = R²/(R² − x²)

S = 2π ∫−RR √(R² − x²) · R/√(R² − x²) dx = 2πR ∫−RR dx = 4πR²

五、物理应用

5.1 变力做功

力 F(x) 沿 x 方向从 a 到 b 做的功：

W = ∫ab F(x) dx

5.2 液体压力

浸入液体的薄板受到的总压力：

P = ρg ∫ab y · w(y) dy

ρ 为液体密度，g 为重力加速度，w(y) 为深度 y 处的板宽

5.3 质心与形心

线密度为 ρ(x) 的细棒的质心：

x̄ = ∫ x · ρ(x) dx / ∫ ρ(x) dx

平面图形的形心（面积加权）：

x̄ = ∫ab x[f(x)−g(x)] dx / A

ȳ = (1/2) ∫ab [f²(x)−g²(x)] dx / A

5.4 函数平均值

favg = (1/(b−a)) ∫ab f(x) dx

核心公式汇总

  • 面积：A = ∫[f−g] dx（直角坐标）；A = (1/2)∫r² dθ（极坐标）
  • 弧长：ds = √(dx² + dy²)，直角/参数/极坐标各有对应公式
  • 体积：圆盘法 V = π∫f² dx；壳层法 V = 2π∫x·f(x) dx
  • 表面积：S = 2π∫y ds = 2π∫f(x)√(1+f′²) dx
  • 物理应用：功 W = ∫F dx；压力 P = ρg∫y·w dy；平均值 f_avg = (1/(b−a))∫f dx
