积分技巧

一元积分学 · 换元法、分部积分、有理函数积分

一、第一类换元法（凑微分法）

公式

若 ∫f(u)du = F(u) + C，则

∫ f(φ(x)) · φ'(x) dx = ∫ f(u) du = F(u) + C = F(φ(x)) + C

关键：将被积函数凑成 f(φ(x))·φ'(x) 的形式，令 u = φ(x)。

例题 1：凑微分

求 ∫ x·e^(x²) dx。

凑微分：x dx = (1/2) d(x²)
= (1/2) ∫ e^(x²) d(x²) = (1/2) e^(x²) + C
令 u = x²：= (1/2)e^u + C = (1/2)e^(x²) + C

二、第二类换元法

公式

令 x = φ(t)，φ'(t) ≠ 0，则

∫ f(x) dx = ∫ f(φ(t)) · φ'(t) dt

换元后求出关于 t 的原函数，再将 t = φ⁻¹(x) 代回。

常用三角换元

被积函数含换元恒等式
√(a²-x²)x = asint1-sin²t = cos²t
√(a²+x²)x = atant1+tan²t = sec²t
√(x²-a²)x = asectsec²t-1 = tan²t
例题 2：三角换元

求 ∫ √(1-x²) dx。

令 x = sint, dx = cost dt
= ∫ √(1-sin²t) · cost dt = ∫ cos²t dt
= ∫ (1+cos2t)/2 dt = t/2 + sin2t/4 + C
= t/2 + sint·cost/2 + C
= (arcsinx)/2 + x·√(1-x²)/2 + C

三、分部积分法

公式

∫ u dv = uv - ∫ v du

关键在于选择 u 和 dv。口诀："反对幂指三"——对数、反三角、幂函数、指数、三角，排在前面的优先选作 u。

选择 u 的经验法则（LIATE）

  • Logarithmic（对数函数）：ln x, log x
  • Inverse trig（反三角函数）：arcsin x, arctan x
  • Algebraic（代数函数）：xⁿ, 多项式
  • Trigonometric（三角函数）：sin x, cos x
  • Exponential（指数函数）：eˣ, aˣ

排在前面的优先选作 u（因为它求导后会简化）。

例题 3：分部积分

求 ∫ x·eˣ dx。

u = x, dv = eˣdx → du = dx, v = eˣ
∫ x·eˣ dx = x·eˣ - ∫ eˣ dx = x·eˣ - eˣ + C
= (x-1)eˣ + C
例题 4：循环型分部积分

求 ∫ eˣ·sinx dx。

I = ∫ eˣ·sinx dx
分部（u=sinx, dv=eˣdx）：I = eˣ·sinx - ∫ eˣ·cosx dx
再分部（u=cosx, dv=eˣdx）：∫eˣcosx dx = eˣcosx + ∫eˣsinx dx

I = eˣsinx - eˣcosx - I
2I = eˣ(sinx - cosx)
I = eˣ(sinx - cosx)/2 + C

四、有理函数积分（部分分式法）

步骤

  1. 若分子次数 ≥ 分母次数，先做多项式除法
  2. 将分母因式分解（实数范围内分解为一次因式和不可约二次因式）
  3. 拆成部分分式
  4. 逐项积分

部分分式形式

分母因子部分分式形式
(x-a)A/(x-a)
(x-a)ᵏA₁/(x-a) + A₂/(x-a)² + ... + Aₖ/(x-a)ᵏ
x²+px+q（不可约）(Ax+B)/(x²+px+q)
例题 5：部分分式

求 ∫ 1/(x²-1) dx。

1/(x²-1) = 1/[(x-1)(x+1)] = A/(x-1) + B/(x+1)
1 = A(x+1) + B(x-1)
令 x=1: 1 = 2A → A = 1/2
令 x=-1: 1 = -2B → B = -1/2

= (1/2)∫1/(x-1)dx - (1/2)∫1/(x+1)dx
= (1/2)ln|x-1| - (1/2)ln|x+1| + C = (1/2)ln|(x-1)/(x+1)| + C
积分技巧选择顺序：
  1. 先看能否直接套基本公式
  2. 尝试凑微分（第一类换元）
  3. 有根号时考虑三角换元
  4. 有对数/反三角/幂×三角等乘积时用分部积分
  5. 有理函数用部分分式
  6. 以上都不行时考虑特殊技巧或数值方法

本节总结

  • 第一类换元（凑微分）：∫f(φ(x))φ'(x)dx = F(φ(x))+C
  • 第二类换元：x = φ(t)，常用三角换元去根号
  • 分部积分：∫udv = uv - ∫vdu，选 u 遵循 LIATE 原则
  • 有理函数积分：部分分式分解后逐项积分
  • 技巧选择：先简单后复杂，先凑微分再换元再分部
