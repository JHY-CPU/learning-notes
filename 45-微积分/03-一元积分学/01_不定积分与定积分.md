# 不定积分与定积分

*一元积分学 · 积分的基本理论*


### 1. 定义


若 F'(x) = f(x)，则称 F(x) 是 f(x) 的一个原函数。f(x) 的所有原函数称为 f(x) 的不定积分：


$$
∫ f(x) dx = F(x) + C
$$


几何意义：f(x) 的一族积分曲线，它们之间相差常数 C。


### 2. 基本积分公式


| f(x) | ∫f(x)dx | f(x) | ∫f(x)dx |
| --- | --- | --- | --- |
| xⁿ | xⁿ⁺¹/(n+1)+C | 1/x | ln\|x\|+C |
| eˣ | eˣ+C | aˣ | aˣ/ln a+C |
| sin x | -cos x+C | cos x | sin x+C |
| sec²x | tan x+C | csc²x | -cot x+C |
| 1/(1+x²) | arctan x+C | 1/√(1-x²) | arcsin x+C |


### 3. 运算法则


$$
∫ [f(x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx

            ∫ k·f(x) dx = k · ∫ f(x) dx   （k 为常数）
$$


### 1. 黎曼和与定积分定义


将 [a, b] 分割为 n 个小区间，取每个小区间中的点 ξᵢ，构造黎曼和：


$$
∫ₐᵇ f(x) dx = lim(n→∞) Σ f(ξᵢ) · Δxᵢ
$$


几何意义：曲线 y=f(x) 与 x 轴之间、x=a 到 x=b 所围成的有向面积（x 轴上方为正，下方为负）。


### 2. 定积分的性质


- ∫ₐᵇ f(x)dx = -∫ᵇₐ f(x)dx
- ∫ₐᵃ f(x)dx = 0
- ∫ₐᵇ [f(x)±g(x)]dx = ∫ₐᵇ f(x)dx ± ∫ₐᵇ g(x)dx
- **积分中值定理**
   ：∃ ξ ∈ [a,b] 使 ∫ₐᵇ f(x)dx = f(ξ)(b-a)
- 若 f(x) ≤ g(x)，则 ∫ₐᵇ f(x)dx ≤ ∫ₐᵇ g(x)dx


### 公式


若 F'(x) = f(x)，f(x) 在 [a, b] 上连续，则：


$$
∫ₐᵇ f(x) dx = F(b) - F(a) = [F(x)]ₐᵇ
$$


意义：定积分 = 原函数在上下限处的值之差。将积分问题转化为求原函数问题。


### 变上限积分


$$
Φ(x) = ∫ₐˣ f(t)dt    则    Φ'(x) = f(x)
$$


变上限积分的导数等于被积函数在上限处的值。这是连接积分与微分的桥梁。

### 积分的应用

**求面积**：
$$A = ∫ₐᵇ |f(x)| dx$$

**求旋转体体积**：
- 绕 x 轴旋转：V = π∫ₐᵇ [f(x)]² dx
- 绕 y 轴旋转：V = 2π∫ₐᵇ x|f(x)| dx（壳层法）

**求弧长**：
$$L = ∫ₐᵇ √(1 + [f'(x)]²) dx$$

### 广义积分（反常积分）

**无穷区间上的积分**：
$$∫ₐ^∞ f(x)dx = lim_{t→∞} ∫ₐᵗ f(x)dx$$

若极限存在则收敛，否则发散。

**无界函数的积分**（瑕积分）：
若 f(x) 在 x=a 处无界：
$$∫ₐᵇ f(x)dx = lim_{ε→0⁺} ∫_{a+ε}ᵇ f(x)dx$$

常用判敛准则：
- ∫₁^∞ 1/xᵖ dx：p>1 收敛，p≤1 发散
- ∫₀¹ 1/xᵖ dx：p<1 收敛，p≥1 发散

### 定积分的换元和分部

定积分的换元法：换元后必须换积分上下限
$$∫ₐᵇ f(x)dx = ∫_α^β f(φ(t))φ'(t)dt$$
其中 a=φ(α), b=φ(β)

定积分的分部积分：
$$∫ₐᵇ u dv = [uv]ₐᵇ - ∫ₐᵇ v du$$

### 常见易错点

1. **不定积分不要漏掉 +C**：定积分不需要 C
2. **定积分换元要换限**：x=a 对应 t=α，x=b 对应 t=β
3. **变上限积分的链式法则**：d/dx[∫₀^{g(x)} f(t)dt] = f(g(x))·g'(x)
4. **对称区间奇偶性**：∫₋ₐᵃ f(x)dx，f 奇函数则为0，f 偶函数则为 2∫₀ᵃ


## Python实现

### 符号积分 (SymPy)

```python
import sympy as sp

x = sp.Symbol('x')

# 不定积分
print("=== 不定积分 ===")
funcs = [
    (x**3, "x^3"),
    (sp.sin(x), "sin(x)"),
    (sp.exp(x), "e^x"),
    (1/x, "1/x"),
    (1/(1+x**2), "1/(1+x^2)"),
    (sp.sqrt(1-x**2), "sqrt(1-x^2)")
]
for f, name in funcs:
    print(f"∫{name} dx = {sp.integrate(f, x)} + C")

# 定积分
print("\n=== 定积分 ===")
print(f"∫_0^1 x^2 dx = {sp.integrate(x**2, (x, 0, 1))}")
print(f"∫_0^π sin(x) dx = {sp.integrate(sp.sin(x), (x, 0, sp.pi))}")
print(f"∫_{-1}^1 1/(1+x^2) dx = {sp.integrate(1/(1+x**2), (x, -1, 1))}")
```

### 数值积分

```python
import numpy as np
from scipy import integrate

# 黎曼和近似
def riemann_sum(f, a, b, n, method='midpoint'):
    """黎曼和近似定积分"""
    x = np.linspace(a, b, n+1)
    if method == 'left':
        return np.sum(f(x[:-1]) * (b-a)/n)
    elif method == 'right':
        return np.sum(f(x[1:] * (b-a)/n))
    else:  # midpoint
        mid = (x[:-1] + x[1:]) / 2
        return np.sum(f(mid) * (b-a)/n)

# 示例: ∫_0^1 x^2 dx = 1/3
f = lambda x: x**2
for n in [10, 100, 1000]:
    approx = riemann_sum(f, 0, 1, n)
    print(f"n={n}: ∫x^2 ≈ {approx:.8f}, 误差 = {abs(approx - 1/3):.2e}")

# scipy 数值积分 (高精度)
result, error = integrate.quad(lambda x: np.exp(-x**2), 0, np.inf)
print(f"\n∫_0^∞ e^(-x^2) dx = {result:.10f} (精确值 √π/2 = {np.sqrt(np.pi)/2:.10f})")

# 变上限积分求导: d/dx ∫_0^x sin(t^2) dt = sin(x^2)
from scipy.integrate import quad
def phi(x):
    result, _ = quad(lambda t: np.sin(t**2), 0, x)
    return result

x_val = 1.0
h = 1e-7
approx_deriv = (phi(x_val+h) - phi(x_val-h)) / (2*h)
exact_deriv = np.sin(x_val**2)
print(f"Φ'(1) ≈ {approx_deriv:.8f}, sin(1) = {exact_deriv:.8f}")
```
