# 文本渲染与本地化

## 核心概念

游戏文本涉及渲染质量和多语言支持两个核心问题。文本渲染质量直接影响游戏观感，而本地化系统决定了游戏能否触达全球市场。TextMeshPro是Unity推荐的文本解决方案，已内置到Unity 2021+中。

### TextMeshPro SDF渲染原理

SDF（Signed Distance Field，有向距离场）是一种革命性的文字渲染技术。传统位图字体存储每个像素的颜色，而SDF存储每个像素到字形边缘的距离值：

```
传统位图：  像素值 = 颜色 (RGBA)
SDF字体：   像素值 = 到边缘的有向距离 (正值=内部, 负值=外部)
```

SDF的优势：
- **任意缩放不失真**：因为存储的是距离而非颜色，放大时通过距离值插值重新计算边缘
- **完美的轮廓**：即使在极小尺寸下也保持锐利的边缘
- **高效描边和阴影**：通过距离值阈值即可实现，无需额外Pass
- **字体图集更小**：每个字符只需较低分辨率的SDF纹理

SDF的局限：
- 不适合复杂装饰性字体（如书法字体的飞白效果）
- 极端放大时可能丢失细节（距离场精度有限）
- 需要生成专用的SDF字体图集

### TMP字体图集详解

TMP的字体图集生成流程：
```
源字体文件(.ttf/.otf)
    ↓ Font Asset Creator
指定采样字符集(常用3500汉字 + ASCII + 标点)
    ↓
指定图集分辨率(2048x2048或4096x4096)
    ↓
生成SDF/SDFA图集(.asset)
    ↓
运行时使用Shader采样图集渲染文字
```

关键参数选择：
- **Atlas Resolution**：移动平台推荐2048x2048，PC端可用4096x4096
- **Character Set**：中文游戏建议覆盖GB2312的3755个一级汉字+ASCII
- **Padding**：字符间距，影响描边效果质量，推荐5-8
- **Render Mode**：SDFA（带Alpha通道的SDF）适合大部分场景

## 具体实现方法

### 多语言本地化系统（完整生产级实现）

```csharp
using System;
using System.Collections.Generic;
using UnityEngine;

/// <summary>
/// 本地化管理器
/// 支持多语言切换、参数替换、复数形式、RTL语言
/// </summary>
public class LocalizationManager : MonoBehaviour
{
    public static LocalizationManager Instance { get; private set; }

    public event Action OnLanguageChanged;

    private Dictionary<string, string> currentDict = new Dictionary<string, string>();
    private string currentLanguage = "zh_cn";
    private const string MISSING_KEY_FORMAT = "[Missing: {0}]";

    public string CurrentLanguage => currentLanguage;

    void Awake()
    {
        if (Instance != null) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
        LoadLanguage(currentLanguage);
    }

    /// <summary>
    /// 加载指定语言的翻译数据
    /// 支持JSON和CSV格式
    /// </summary>
    public void LoadLanguage(string langCode)
    {
        currentLanguage = langCode;
        currentDict.Clear();

        // 从JSON加载
        TextAsset json = Resources.Load<TextAsset>($"Localization/{langCode}");
        if (json != null)
        {
            var data = JsonUtility.FromJson<LanguageData>(json.text);
            foreach (var entry in data.entries)
                currentDict[entry.key] = entry.value;
        }

        OnLanguageChanged?.Invoke();
        Debug.Log($"Loaded language: {langCode}, {currentDict.Count} entries");
    }

    /// <summary>
    /// 获取翻译文本
    /// </summary>
    public string GetText(string key)
    {
        if (currentDict.TryGetValue(key, out string value))
            return value;
        return string.Format(MISSING_KEY_FORMAT, key);
    }

    /// <summary>
    /// 获取带参数的格式化文本
    /// 用法: GetText("level_info", level, exp, maxExp)
    /// 翻译文件中: "level_info": "等级: {0} | 经验: {1}/{2}"
    /// </summary>
    public string GetText(string key, params object[] args)
    {
        string template = GetText(key);
        try
        {
            return string.Format(template, args);
        }
        catch (FormatException)
        {
            Debug.LogWarning($"Localization format error: {key}");
            return template;
        }
    }

    /// <summary>
    /// 获取带复数形式的文本
    /// </summary>
    public string GetPluralText(string key, int count)
    {
        string pluralKey = count == 1 ? $"{key}_one" : $"{key}_other";
        return GetText(pluralKey, count);
    }

    public void SetLanguage(string langCode)
    {
        LoadLanguage(langCode);
        PlayerPrefs.SetString("Language", langCode);
    }

    [System.Serializable]
    public class LanguageData
    {
        public List<LangEntry> entries = new List<LangEntry>();
    }

    [System.Serializable]
    public class LangEntry
    {
        public string key;
        public string value;
    }
}
```

### 本地化文本组件（自动刷新版）

```csharp
/// <summary>
/// 本地化文本组件
/// 自动监听语言切换事件并刷新显示
/// 支持参数替换和富文本
/// </summary>
[RequireComponent(typeof(TextMeshProUGUI))]
public class LocalizedText : MonoBehaviour
{
    [SerializeField] private string localizationKey;
    [SerializeField] private object[] formatArgs; // Inspector中不可序列化，需代码设置

    private TextMeshProUGUI tmpText;

    public string Key
    {
        get => localizationKey;
        set { localizationKey = value; UpdateText(); }
    }

    void Start()
    {
        tmpText = GetComponent<TextMeshProUGUI>();
        UpdateText();

        if (LocalizationManager.Instance != null)
            LocalizationManager.Instance.OnLanguageChanged += UpdateText;
    }

    void OnDestroy()
    {
        if (LocalizationManager.Instance != null)
            LocalizationManager.Instance.OnLanguageChanged -= UpdateText;
    }

    /// <summary>
    /// 设置带参数的本地化文本
    /// </summary>
    public void SetText(string key, params object[] args)
    {
        localizationKey = key;
        formatArgs = args;
        UpdateText();
    }

    void UpdateText()
    {
        if (tmpText == null || LocalizationManager.Instance == null) return;

        if (formatArgs != null && formatArgs.Length > 0)
            tmpText.text = LocalizationManager.Instance.GetText(localizationKey, formatArgs);
        else
            tmpText.text = LocalizationManager.Instance.GetText(localizationKey);
    }
}
```

public class FontFallbackSetup : MonoBehaviour
{
    [SerializeField] private TMP_FontAsset primaryFont;
    [SerializeField] private TMP_FontAsset[] fallbackFonts;

    void Start()
    {
        // 配置Fallback链
        primaryFont.fallbackFontAssetTable = new List<TMP_FontAsset>(fallbackFonts);

        // 设置动态图集参数
        primaryFont.atlasPopulationMode = AtlasPopulationMode.Dynamic;
        primaryFont.freeGlyphRects = true;
    }
}

### 打字机效果（对话系统常用）

```csharp
/// <summary>
/// 打字机效果组件
/// 逐字符显示文本，支持跳过和暂停
/// </summary>
public class TypewriterEffect : MonoBehaviour
{
    [SerializeField] private TextMeshProUGUI tmpText;
    [SerializeField] private float charInterval = 0.05f; // 每字符间隔（秒）
    [SerializeField] private float punctuationPause = 0.2f; // 标点符号额外停顿

    private string fullText;
    private int currentIndex;
    private bool isTyping;
    private Coroutine typingCoroutine;

    public bool IsTyping => isTyping;
    public event Action OnTypingComplete;

    /// <summary>
    /// 开始打字效果
    /// </summary>
    public void StartTyping(string text)
    {
        fullText = text;
        currentIndex = 0;
        tmpText.text = "";
        tmpText.maxVisibleCharacters = 0;

        if (typingCoroutine != null)
            StopCoroutine(typingCoroutine);
        typingCoroutine = StartCoroutine(TypeCoroutine());
    }

    IEnumerator TypeCoroutine()
    {
        isTyping = true;
        tmpText.text = fullText; // 先设置完整文本（TMP需要完整文本才能正确排版）
        tmpText.maxVisibleCharacters = 0;

        for (int i = 0; i < fullText.Length; i++)
        {
            tmpText.maxVisibleCharacters = i + 1;
            currentIndex = i;

            // 标点符号停顿
            char c = fullText[i];
            if (c == '。' || c == '！' || c == '？' || c == '.' || c == '!' || c == '?')
                yield return new WaitForSeconds(charInterval + punctuationPause);
            else if (c == '，' || c == '、' || c == ',' || c == ';')
                yield return new WaitForSeconds(charInterval + punctuationPause * 0.5f);
            else
                yield return new WaitForSeconds(charInterval);
        }

        tmpText.maxVisibleCharacters = fullText.Length;
        isTyping = false;
        OnTypingComplete?.Invoke();
    }

    /// <summary>
    /// 立即显示全部文本（跳过打字效果）
    /// </summary>
    public void Skip()
    {
        if (typingCoroutine != null)
        {
            StopCoroutine(typingCoroutine);
            typingCoroutine = null;
        }

        tmpText.maxVisibleCharacters = fullText.Length;
        isTyping = false;
        OnTypingComplete?.Invoke();
    }
}
```

## 最佳实践

- 使用SDF字体代替传统位图字体，确保任意缩放质量
- 中文字体图集优先覆盖常用3500汉字以减少动态加载和运行时内存
- 本地化键值使用语义化命名（如`ui_btn_confirm`而非`str_001`），方便维护
- 文本更新仅在值变化时进行，避免每帧刷新触发Canvas Rebuild
- 对话系统使用富文本配合TMP.maxVisibleCharacters实现打字机效果
- 准备阿拉伯语等RTL（从右到左）语言的排版方案（TMP支持RTL）
- 同一界面中尽量只使用1-2个字体Asset，避免字体图集切换打断合批
- 为所有本地化文本提供Fallback，缺失键显示[Missing: key]便于调试

## 常见陷阱与修复

**陷阱1：中文字体全部导入导致图集过大**
- 症状：字体图集占用50-100MB内存
- 修复：按需选择字符子集（GB2312一级3755字+ASCII+数字+标点），约2048x2048即可

**陷阱2：动态字体模式下未设置Fallback Font**
- 症状：某些字符显示为空白方块（tofu字符）
- 修复：配置Fallback Font Assets链，确保所有语言字符都有对应字体

**陷阱3：本地化文本未考虑文字长度差异**
- 症状：英文界面翻译成德语后文字溢出按钮边界
- 修复：UI设计预留20-30%的文字长度余量，使用auto-size或自动换行

**陷阱4：富文本标签未闭合**
- 症状：后续文本样式异常，全变成红色或粗体
- 修复：TMP会自动处理未闭合标签，但最好在编辑器中使用TMP的标签验证工具

**陷阱5：字体图集分辨率过低**
- 症状：大字号文字出现锯齿
- 修复：增大Atlas Resolution到4096，或拆分字符到多个图集
