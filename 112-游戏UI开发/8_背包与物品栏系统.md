# 背包与物品栏系统

## 核心概念

背包系统是RPG和大多数游戏的核心UI系统之一，涉及数据绑定、拖拽交互、视觉反馈和性能优化。一个完善的背包系统需要处理格子布局、物品拖拽交换、堆叠拆分、分类筛选、排序、Tooltip预览等多种功能。

### 格子布局系统

- **固定格子**：GridLayoutGroup自动排列，每格固定大小，实现简单但不灵活
- **动态格子**：根据物品大小占据不同数量格子（如《暗黑破坏神》系列的变体）
- **分类标签页**：按物品类型分Tab显示（全部/武器/防具/消耗品/材料）
- **虚拟列表格子**：格子总数远大于物品数量，使用对象池复用

```csharp
/// <summary>
/// 物品类型枚举
/// </summary>
public enum ItemType
{
    Weapon,      // 武器
    Armor,       // 防具
    Consumable,  // 消耗品（药水、卷轴）
    Material,    // 材料（合成素材）
    Quest,       // 任务物品
    Currency     // 货币
}

/// <summary>
/// 物品稀有度
/// </summary>
public enum Rarity
{
    Common = 0,      // 白色 - 普通
    Uncommon = 1,    // 绿色 - 优秀
    Rare = 2,        // 蓝色 - 稀有
    Epic = 3,        // 紫色 - 史诗
    Legendary = 4    // 橙色 - 传说
}

/// <summary>
/// 物品数据定义（ScriptableObject，策划配置）
/// </summary>
[CreateAssetMenu(fileName = "ItemDef", menuName = "Game/Item Definition")]
public class ItemDefinition : ScriptableObject
{
    public string itemId;
    public string itemName;
    public string description;
    public Sprite icon;
    public ItemType type;
    public Rarity rarity;
    public int maxStack = 99;
    public int buyPrice;
    public int sellPrice;

    // 属性加成（武器/防具）
    public List<StatBonus> statBonuses = new List<StatBonus>();

    [System.Serializable]
    public class StatBonus
    {
        public string statName; // "atk", "def", "hp"
        public int value;
    }
}

/// <summary>
/// 物品实例数据（运行时，玩家背包中的实际物品）
/// </summary>
[System.Serializable]
public class ItemInstance
{
    public string instanceId;       // 唯一实例ID
    public ItemDefinition definition; // 指向物品定义
    public int stackCount;          // 当前堆叠数
    public int slotIndex;           // 所在格子索引（-1表示未在背包中）

    public bool IsFullStack => stackCount >= definition.maxStack;
    public int RemainingSpace => definition.maxStack - stackCount;
}
```

## 具体实现方法

### 背包数据管理器

```csharp
/// <summary>
/// 背包数据管理器
/// 纯数据逻辑，不依赖UI
/// </summary>
public class InventoryData
{
    private ItemInstance[] slots;
    private int capacity;

    public event Action<int, ItemInstance> OnSlotChanged; // 格子索引, 新物品
    public event Action OnInventoryChanged;

    public int Capacity => capacity;
    public ItemInstance this[int index] => slots[index];

    public InventoryData(int capacity)
    {
        this.capacity = capacity;
        slots = new ItemInstance[capacity];
    }

    /// <summary>
    /// 添加物品到背包（自动堆叠）
    /// 返回未能添加的数量
    /// </summary>
    public int AddItem(ItemDefinition definition, int count = 1)
    {
        int remaining = count;

        // 第一轮：尝试堆叠到已有格子
        if (definition.maxStack > 1)
        {
            for (int i = 0; i < capacity && remaining > 0; i++)
            {
                if (slots[i] != null
                    && slots[i].definition == definition
                    && !slots[i].IsFullStack)
                {
                    int add = Mathf.Min(remaining, slots[i].RemainingSpace);
                    slots[i].stackCount += add;
                    remaining -= add;
                    OnSlotChanged?.Invoke(i, slots[i]);
                }
            }
        }

        // 第二轮：放入空格子
        for (int i = 0; i < capacity && remaining > 0; i++)
        {
            if (slots[i] == null)
            {
                ItemInstance newItem = new ItemInstance
                {
                    instanceId = System.Guid.NewGuid().ToString(),
                    definition = definition,
                    stackCount = Mathf.Min(remaining, definition.maxStack),
                    slotIndex = i
                };
                slots[i] = newItem;
                remaining -= newItem.stackCount;
                OnSlotChanged?.Invoke(i, slots[i]);
            }
        }

        if (remaining < count)
            OnInventoryChanged?.Invoke();

        return remaining; // 返回未能添加的数量
    }

    /// <summary>
    /// 移除指定数量的物品
    /// </summary>
    public bool RemoveItem(string itemId, int count = 1)
    {
        int total = CountItem(itemId);
        if (total < count) return false;

        int remaining = count;
        for (int i = 0; i < capacity && remaining > 0; i++)
        {
            if (slots[i] != null && slots[i].definition.itemId == itemId)
            {
                int remove = Mathf.Min(remaining, slots[i].stackCount);
                slots[i].stackCount -= remove;
                remaining -= remove;

                if (slots[i].stackCount <= 0)
                    slots[i] = null;

                OnSlotChanged?.Invoke(i, slots[i]);
            }
        }

        OnInventoryChanged?.Invoke();
        return true;
    }

    /// <summary>
    /// 交换两个格子的物品
    /// </summary>
    public void SwapSlots(int indexA, int indexB)
    {
        ItemInstance temp = slots[indexA];
        slots[indexA] = slots[indexB];
        slots[indexB] = temp;

        if (slots[indexA] != null) slots[indexA].slotIndex = indexA;
        if (slots[indexB] != null) slots[indexB].slotIndex = indexB;

        OnSlotChanged?.Invoke(indexA, slots[indexA]);
        OnSlotChanged?.Invoke(indexB, slots[indexB]);
        OnInventoryChanged?.Invoke();
    }

    /// <summary>
    /// 统计某种物品的总数
    /// </summary>
    public int CountItem(string itemId)
    {
        int total = 0;
        for (int i = 0; i < capacity; i++)
        {
            if (slots[i] != null && slots[i].definition.itemId == itemId)
                total += slots[i].stackCount;
        }
        return total;
    }

    public bool HasItem(string itemId, int count = 1)
    {
        return CountItem(itemId) >= count;
    }

    /// <summary>
    /// 获取所有物品（用于UI显示）
    /// </summary>
    public List<ItemInstance> GetAllItems()
    {
        var list = new List<ItemInstance>();
        for (int i = 0; i < capacity; i++)
            if (slots[i] != null) list.Add(slots[i]);
        return list;
    }
}
```

### 背包格子UI组件（完整版）

```csharp
/// <summary>
/// 背包格子UI组件
/// 支持物品显示、拖拽交换、稀有度边框、右键使用
/// </summary>
public class InventorySlotUI : MonoBehaviour,
    IDropHandler, IPointerClickHandler, IPointerEnterHandler, IPointerExitHandler
{
    [SerializeField] private Image icon;
    [SerializeField] private TextMeshProUGUI countText;
    [SerializeField] private Image borderFrame;
    [SerializeField] private Image selectionHighlight;
    [SerializeField] private Image lockOverlay;

    public int SlotIndex { get; set; }
    public ItemInstance CurrentItem { get; private set; }
    public bool IsLocked { get; set; }

    // 稀有度对应颜色
    private static readonly Color[] RarityColors = {
        new Color(0.7f, 0.7f, 0.7f),   // Common - 灰白
        new Color(0.2f, 0.8f, 0.2f),   // Uncommon - 绿色
        new Color(0.2f, 0.4f, 1f),     // Rare - 蓝色
        new Color(0.6f, 0.2f, 0.9f),   // Epic - 紫色
        new Color(1f, 0.6f, 0f)        // Legendary - 橙色
    };

    public void SetItem(ItemInstance item)
    {
        CurrentItem = item;
        if (item == null)
        {
            Clear();
            return;
        }

        icon.sprite = item.definition.icon;
        icon.gameObject.SetActive(true);
        icon.SetNativeSize();

        countText.text = item.stackCount > 1 ? item.stackCount.ToString() : "";
        borderFrame.color = RarityColors[(int)item.definition.rarity];
        borderFrame.gameObject.SetActive(true);
    }

    public void Clear()
    {
        CurrentItem = null;
        icon.gameObject.SetActive(false);
        countText.text = "";
        borderFrame.gameObject.SetActive(false);
        if (selectionHighlight != null)
            selectionHighlight.gameObject.SetActive(false);
    }

    // 拖拽放置
    public void OnDrop(PointerEventData eventData)
    {
        if (IsLocked) return;

        DraggableItem dragged = eventData.pointerDrag?.GetComponent<DraggableItem>();
        if (dragged != null && dragged.SourceSlot != this)
        {
            // 触发背包数据层的交换
            InventoryManager.Instance.SwapSlots(
                dragged.SourceSlot.SlotIndex, this.SlotIndex);
        }
    }

    // 点击
    public void OnPointerClick(PointerEventData eventData)
    {
        if (eventData.button == PointerEventData.InputButton.Right)
        {
            // 右键使用/装备
            if (CurrentItem != null)
                InventoryManager.Instance.UseItem(CurrentItem);
        }
        else if (eventData.button == PointerEventData.InputButton.Left)
        {
            // 左键选中
            InventoryUI.Instance.SelectSlot(this);
        }
    }

    // 悬停
    public void OnPointerEnter(PointerEventData eventData)
    {
        if (CurrentItem != null)
            TooltipManager.Instance.Show(CurrentItem, GetComponent<RectTransform>());
    }

    public void OnPointerExit(PointerEventData eventData)
    {
        TooltipManager.Instance.Hide();
    }
}
```

### 拖拽物品组件（完整版）

```csharp
/// <summary>
/// 可拖拽物品组件
/// 支持拖拽跟随、半透明效果、回到原位
/// </summary>
public class DraggableItem : MonoBehaviour,
    IBeginDragHandler, IDragHandler, IEndDragHandler
{
    public InventorySlotUI SourceSlot { get; set; }

    private RectTransform rectTransform;
    private CanvasGroup canvasGroup;
    private Canvas rootCanvas;
    private Transform originalParent;
    private Vector3 originalPosition;
    private int originalSiblingIndex;

    void Awake()
    {
        rectTransform = GetComponent<RectTransform>();
        canvasGroup = GetComponent<CanvasGroup>();
    }

    public void OnBeginDrag(PointerEventData eventData)
    {
        rootCanvas = GetComponentInParent<Canvas>().rootCanvas;
        originalParent = transform.parent;
        originalPosition = rectTransform.position;
        originalSiblingIndex = transform.GetSiblingIndex();

        // 提升到Canvas根节点确保渲染在最上层
        transform.SetParent(rootCanvas.transform);
        transform.SetAsLastSibling();

        canvasGroup.alpha = 0.6f;
        canvasGroup.blocksRaycasts = false; // 允许穿透到下方格子
    }

    public void OnDrag(PointerEventData eventData)
    {
        if (RectTransformUtility.ScreenPointToWorldPointInRectangle(
            rootCanvas.transform as RectTransform,
            eventData.position, eventData.pressEventCamera, out Vector3 worldPos))
        {
            rectTransform.position = worldPos;
        }
    }

    public void OnEndDrag(PointerEventData eventData)
    {
        canvasGroup.alpha = 1f;
        canvasGroup.blocksRaycasts = true;

        // 回到原位（如果未被OnDrop接收）
        transform.SetParent(originalParent);
        rectTransform.position = originalPosition;
        transform.SetSiblingIndex(originalSiblingIndex);
    }
}
```

### Tooltip系统

```csharp
/// <summary>
/// 物品详情Tooltip
/// 显示物品名称、描述、属性、稀有度
/// </summary>
public class TooltipManager : MonoBehaviour
{
    public static TooltipManager Instance { get; private set; }

    [SerializeField] private GameObject tooltipPanel;
    [SerializeField] private TextMeshProUGUI nameText;
    [SerializeField] private TextMeshProUGUI typeText;
    [SerializeField] private TextMeshProUGUI descText;
    [SerializeField] private TextMeshProUGUI statsText;
    [SerializeField] private Image rarityBorder;

    private RectTransform tooltipRect;
    private Canvas parentCanvas;
    private bool isShowing;

    void Awake()
    {
        Instance = this;
        tooltipRect = tooltipPanel.GetComponent<RectTransform>();
        parentCanvas = GetComponentInParent<Canvas>();
        tooltipPanel.SetActive(false);
    }

    public void Show(ItemInstance item, RectTransform targetRect)
    {
        if (item == null) return;

        ItemDefinition def = item.definition;
        nameText.text = def.itemName;
        nameText.color = InventorySlotUI.RarityColors[(int)def.rarity];
        typeText.text = def.type.ToString();
        descText.text = def.description;

        // 属性加成
        if (def.statBonuses.Count > 0)
        {
            System.Text.TextBuilder sb = new System.Text.StringBuilder();
            foreach (var bonus in def.statBonuses)
                sb.AppendLine($"+{bonus.value} {bonus.statName}");
            statsText.text = sb.ToString();
        }
        else
        {
            statsText.text = "";
        }

        rarityBorder.color = InventorySlotUI.RarityColors[(int)def.rarity];

        tooltipPanel.SetActive(true);
        isShowing = true;

        // 定位：在目标右侧显示，超出屏幕则在左侧
        PositionTooltip(targetRect);
    }

    public void Hide()
    {
        tooltipPanel.SetActive(false);
        isShowing = false;
    }

    void PositionTooltip(RectTransform target)
    {
        Vector3[] targetCorners = new Vector3[3];
        target.GetWorldCorners(targetCorners);

        // 尝试放在目标右侧
        Vector3 worldPos = targetCorners[2] + Vector3.right * 10f;
        tooltipRect.position = worldPos;

        // 检测是否超出屏幕，超出则放左侧
        Canvas.ForceUpdateCanvases();
        Vector3[] tooltipCorners = new Vector3[3];
        tooltipRect.GetWorldCorners(tooltipCorners);

        if (tooltipCorners[2].x > Screen.width)
        {
            worldPos = targetCorners[0] + Vector3.left * 10f;
            tooltipRect.position = worldPos;
        }
    }
}
```

## 最佳实践

- 使用GridLayoutGroup自动处理格子排列，避免硬编码位置
- 物品图标使用图集加载，减少Draw Call
- 堆叠物品仅显示数字，不重复创建图标实例
- 拖拽时生成半透明Ghost图标跟随鼠标，原位置保持显示（不立即移除）
- 背包容量变化时复用已有格子对象池，避免频繁Instantiate/Destroy
- 数据层与UI层分离：InventoryData不依赖MonoBehaviour，便于序列化保存
- 使用事件机制驱动UI刷新，而非轮询检测变化

## 常见陷阱与修复

**陷阱1：拖拽结束时未恢复parent导致UI层级错乱**
- 症状：物品拖拽后永远显示在最上层，遮挡其他UI
- 修复：OnEndDrag中恢复originalParent和originalSiblingIndex

**陷阱2：物品交换逻辑未处理堆叠物品的拆分合并**
- 症状：拖动5个药水到已有3个药水的格子，出现物品丢失或重复
- 修复：交换时判断堆叠兼容性，支持部分堆叠转移

**陷阱3：稀有度颜色硬编码，不支持策划后期调整**
- 症状：上线后想修改颜色方案需要改代码
- 修复：使用ScriptableObject或配置表定义稀有度颜色

**陷阱4：背包打开时一次性实例化所有格子导致卡顿**
- 症状：100个格子的背包打开时卡顿50-100ms
- 修复：使用虚拟列表只实例化可见区域的格子，或使用对象池预热

**陷阱5：未处理背包数据和UI状态不一致**
- 症状：数据层物品已移除但UI仍显示旧图标
- 修复：所有数据变更必须触发OnSlotChanged事件，UI监听事件刷新
