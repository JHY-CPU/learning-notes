02 - 矩阵的基本运算

一、矩阵的定义

矩阵是一个 m×n 的数表，由 m 行 n 列元素排列而成：

A = a₁₁ a₁₂ a₁₃ a₂₁ a₂₂ a₂₃ 记作 A = (aᵢⱼ)_{m×n}

特殊矩阵

名称定义示例
零矩阵 O 所有元素为0 0 0 0 0
单位矩阵 I 对角线为1，其余为0 1 0 0 0 1 0 0 0 1
对角矩阵 非对角元素全为0 diag(d₁, d₂, ..., dₙ)
对称矩阵 A = Aᵀ（aᵢⱼ = aⱼᵢ） 1 3 3 2

二、矩阵加法

同型矩阵（行列数分别相等）对应元素相加：

(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ
【示例】
1 2 3 4 + 5 6 7 8 = 6 8 10 12
加法运算律：
- 交换律：A + B = B + A
- 结合律：(A + B) + C = A + (B + C)
- A + O = A（零矩阵是加法单位元）
- A + (-A) = O（-A 为 A 的负矩阵）

三、数乘矩阵（标量乘法）

用标量 k 乘以矩阵的每个元素：

(kA)ᵢⱼ = k · aᵢⱼ
数乘运算律：
- k(A + B) = kA + kB
- (k + l)A = kA + lA
- (kl)A = k(lA)
- 1·A = A，0·A = O

四、矩阵乘法 核心重点

定义

A(m×p) × B(p×n) = C(m×n)，其中：

cᵢⱼ = Σ (k=1→p) aᵢₖ · bₖⱼ = aᵢ₁b₁ⱼ + aᵢ₂b₂ⱼ + ... + aᵢₚbₚⱼ

关键条件：A 的列数 = B 的行数

【示例】 1 2 3 4 × 5 6 7 8 = 19 22 43 50

c₁₁ = 1×5 + 2×7 = 19
c₁₂ = 1×6 + 2×8 = 22
c₂₁ = 3×5 + 4×7 = 43
c₂₂ = 3×6 + 4×8 = 50

乘法运算律

- 结合律：(AB)C = A(BC)
- 分配律：A(B + C) = AB + AC，(A + B)C = AC + BC
- k(AB) = (kA)B = A(kB)
- AI = IA = A（单位矩阵是乘法单位元）
重要警告：矩阵乘法不满足交换律！
- AB 不一定等于 BA（甚至可能无法相乘）
- AB = O 不能推出 A = O 或 B = O
- AB = AC 不能推出 B = C（不能随意消去）

五、矩阵的转置

将矩阵的行与列互换：(Aᵀ)ᵢⱼ = aⱼᵢ

1 2 3 4 5 6 T = 1 4 2 5 3 6
转置运算律：
- (Aᵀ)ᵀ = A
- (A + B)ᵀ = Aᵀ + Bᵀ
- (kA)ᵀ = kAᵀ
- (AB)ᵀ = BᵀAᵀ（注意顺序反转！）

六、矩阵的迹（Trace）

方阵对角线元素之和：

tr(A) = Σ (i=1→n) aᵢᵢ = a₁₁ + a₂₂ + ... + aₙₙ
迹的运算律：
- tr(A + B) = tr(A) + tr(B)
- tr(kA) = k · tr(A)
- tr(AB) = tr(BA)
- tr(A) = 特征值之和

七、矩阵多项式

对于 n 阶方阵 A：

f(A) = aₘAᵐ + aₘ₋₁Aᵐ⁻¹ + ... + a₁A + a₀I
【示例】若 f(x) = x² - 3x + 2，则 f(A) = A² - 3A + 2I
注意：矩阵多项式中常数项要用 a₀I（单位矩阵），不能写成标量 a₀。
