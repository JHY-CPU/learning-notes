# 矩阵的基本运算


## 一、矩阵的定义

$$

## 二、矩阵加法


同型矩阵（行列数分别相等）对应元素相加：


$$
(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ
$$

**加法运算律：**


- 交换律：A + B = B + A


- 结合律：(A + B) + C = A + (B + C)


- A + O = A（零矩阵是加法单位元）


- A + (-A) = O（-A 为 A 的负矩阵）

## 三、数乘矩阵（标量乘法）


用标量 k 乘以矩阵的每个元素：


$$
(kA)ᵢⱼ = k · aᵢⱼ
$$

**数乘运算律：**


- k(A + B) = kA + kB


- (k + l)A = kA + lA


- (kl)A = k(lA)


- 1·A = A，0·A = O

## 四、矩阵乘法 核心重点


### 定义


A(m×p) × B(p×n) = C(m×n)，其中：


$$

$$


**关键条件：**A 的列数 = B 的行数

### 乘法运算律

- 结合律：(AB)C = A(BC)


- 分配律：A(B + C) = AB + AC，(A + B)C = AC + BC


- k(AB) = (kA)B = A(kB)


- AI = IA = A（单位矩阵是乘法单位元）

> **Warning:** **重要警告：矩阵乘法不满足交换律！**
>
>
> - AB 不一定等于 BA（甚至可能无法相乘）
>
>
> - AB = O 不能推出 A = O 或 B = O
>
>
> - AB = AC 不能推出 B = C（不能随意消去）


## 五、矩阵的转置


将矩阵的行与列互换：`(Aᵀ)ᵢⱼ = aⱼᵢ`


> **Example:** 1
> 2
> 3
> 4
> 5
> 6
>
> ^T^
> =
>
> 1
> 4
> 2
> 5
> 3
> 6

**转置运算律：**


- (Aᵀ)ᵀ = A


- (A + B)ᵀ = Aᵀ + Bᵀ


- (kA)ᵀ = kAᵀ


-
**(AB)ᵀ = BᵀAᵀ**
（注意顺序反转！）

## 六、矩阵的迹（Trace）


方阵对角线元素之和：


$$
tr(A) = Σ (i=1→n) aᵢᵢ = a₁₁ + a₂₂ + ... + aₙₙ
$$

**迹的运算律：**


- tr(A + B) = tr(A) + tr(B)


- tr(kA) = k · tr(A)


- tr(AB) = tr(BA)


- tr(A) = 特征值之和

## 七、矩阵多项式


对于 n 阶方阵 A：


$$
f(A) = aₘAᵐ + aₘ₋₁Aᵐ⁻¹ + ... + a₁A + a₀I
$$


> **Example:** **【示例】**
> 若 f(x) = x² - 3x + 2，则 f(A) = A² - 3A + 2I


> **Warning:** **注意：**
> 矩阵多项式中常数项要用 a₀I（单位矩阵），不能写成标量 a₀。


## Python实现

### 矩阵基本运算 (NumPy)

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# 矩阵加法
print("A + B =")
print(A + B)

# 数乘矩阵
print("3A =")
print(3 * A)

# 矩阵乘法
print("A @ B =")
print(A @ B)  # 等价于 np.matmul(A, B)

# 验证矩阵乘法不满足交换律: AB vs BA
print("AB =")
print(A @ B)
print("BA =")
print(B @ A)
print(f"AB == BA? {np.allclose(A @ B, B @ A)}")  # False

# 矩阵转置
print("A^T =")
print(A.T)

# 验证 (AB)^T = B^T A^T
AB_T = (A @ B).T
BT_AT = B.T @ A.T
print(f"(AB)^T == B^T A^T? {np.allclose(AB_T, BT_AT)}")  # True

# 矩阵的迹
print(f"tr(A) = {np.trace(A)}")  # 1 + 4 = 5

# 验证 tr(AB) = tr(BA)
print(f"tr(AB) = {np.trace(A @ B)}, tr(BA) = {np.trace(B @ A)}")

# 矩阵多项式: f(A) = A^2 - 3A + 2I
I = np.eye(2)
f_A = A @ A - 3 * A + 2 * I
print(f"f(A) = A^2 - 3A + 2I =\n{f_A}")
```

### 特殊矩阵的创建

```python
import numpy as np

n = 4
# 零矩阵
O = np.zeros((n, n))
# 单位矩阵
I = np.eye(n)
# 对角矩阵
D = np.diag([1, 2, 3, 4])
# 随机对称矩阵
M = np.random.randn(3, 3)
S = (M + M.T) / 2  # 对称矩阵 S = S^T
print(f"对称矩阵:\n{S}")
print(f"S == S^T? {np.allclose(S, S.T)}")
```
