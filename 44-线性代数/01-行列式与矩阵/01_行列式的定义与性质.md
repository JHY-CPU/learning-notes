# 行列式的定义与性质


## 一、行列式的定义


### 1. 二阶行列式


二阶行列式表示 2x2 矩阵的一个标量值：


$$

$$


### 2. 三阶行列式（沙路法 / 对角线法则）


三阶行列式可用对角线法则计算：


$$

$$


### 3. n 阶行列式（递归定义）


n 阶行列式通过**余子式**递归定义：


$$
det(A) = Σ (j=1→n) aᵢⱼ · (-1)^(i+j) · Mᵢⱼ   （按第 i 行展开）
$$


其中 `Mᵢⱼ` 是元素 `aᵢⱼ` 的**余子式**（删去第 i 行第 j 列后的 n-1 阶行列式）。


`Cᵢⱼ = (-1)^(i+j) · Mᵢⱼ` 称为**代数余子式**。


## 二、行列式的重要性质

**性质1：**
行列式与其转置行列式相等，即
`|A| = |Aᵀ|`
**性质2：**
互换行列式的两行（列），行列式变号
**性质3：**
行列式中某一行（列）有公因子 k，可以提到行列式外面
**性质4：**
若行列式中某一行（列）的元素全为零，则行列式为零
**性质5：**
若行列式中两行（列）相同，则行列式为零
**性质6：**
若行列式中两行（列）成比例，则行列式为零
**性质7：**
行列式某行（列）元素可拆分为两数之和，则行列式可拆分为两个行列式之和
**性质8：**
将行列式某行（列）的 k 倍加到另一行（列），行列式值不变

## 三、余子式展开定理（拉普拉斯展开）


### 按行展开


$$
det(A) = Σ (j=1→n) aᵢⱼ · Cᵢⱼ   （i 为任意固定行）
$$


### 按列展开


$$
det(A) = Σ (i=1→n) aᵢⱼ · Cᵢⱼ   （j 为任意固定列）
$$


### 展开行/列与非展开行/列的关系


$$
Σ (j=1→n) aᵢⱼ · Cₖⱼ = 0   （当 i ≠ k 时）
$$

## 四、特殊矩阵的行列式


| 矩阵类型 | 行列式值 |
| --- | --- |
| 对角矩阵 diag(d₁, d₂, ..., dₙ) | d₁ × d₂ × ... × dₙ |
| 上/下三角矩阵 | 对角线元素之积 |
| 单位矩阵 Iₙ | 1 |
| 零矩阵 O | 0（n ≥ 1） |
| 数量矩阵 kIₙ | kⁿ |
| 正交矩阵 Q（QᵀQ = I） | ±1 |


## 五、行列式的运算律

**乘法定理：**
`|AB| = |A| · |B|`
（同阶方阵）
**幂运算：**
`|Aⁿ| = |A|ⁿ`
**数乘运算：**
`|kA| = kⁿ · |A|`
（A 为 n 阶方阵）
**逆矩阵：**
`|A⁻¹| = 1/|A|`
（当 |A| ≠ 0）
**伴随矩阵：**
`|A*| = |A|^(n-1)`
（A* 为 A 的伴随矩阵）

## 六、行列式的应用


### 1. 克拉默法则（Cramer's Rule）


对于线性方程组 Ax = b，当 |A| ≠ 0 时：


$$
xᵢ = |Aᵢ| / |A|
$$


其中 `Aᵢ` 是将 A 的第 i 列替换为 b 后得到的矩阵。


### 2. 判断矩阵可逆性


`A 可逆 ⟺ |A| ≠ 0`


### 3. 计算面积/体积


二阶行列式的绝对值 = 平行四边形面积；三阶行列式的绝对值 = 平行六面体体积。

>
> 行列式的核心计算方法：利用**性质8**将行列式化为上三角矩阵，再取对角线元素之积。


## Python实现

### 使用 NumPy 计算行列式

```python
import numpy as np

# 创建矩阵
A = np.array([[1, 2, 3],
              [0, 1, 4],
              [0, 0, 5]])

# 计算行列式
det_A = np.linalg.det(A)
print(f"det(A) = {det_A:.1f}")  # 输出: det(A) = 5.0

# 验证上三角矩阵行列式 = 对角线元素之积
diag_product = np.prod(np.diag(A))
print(f"对角线元素之积 = {diag_product}")  # 输出: 5

# 验证性质: |AB| = |A| * |B|
B = np.array([[2, 1], [3, 4]])
C = np.array([[1, 0], [0, 2]])
det_BC = np.linalg.det(B @ C)
det_B_det_C = np.linalg.det(B) * np.linalg.det(C)
print(f"|BC| = {det_BC:.1f}, |B|*|C| = {det_B_det_C:.1f}")  # 两者相等
```

### 克拉默法则求解线性方程组

```python
import numpy as np

def cramers_rule(A, b):
    """使用克拉默法则求解 Ax = b"""
    n = len(b)
    det_A = np.linalg.det(A)
    if abs(det_A) < 1e-10:
        raise ValueError("矩阵不可逆 (|A| = 0)")
    x = np.zeros(n)
    for i in range(n):
        A_i = A.copy()
        A_i[:, i] = b  # 将第 i 列替换为 b
        x[i] = np.linalg.det(A_i) / det_A
    return x

# 示例: 求解 2x + y = 5, x + 3y = 10
A = np.array([[2, 1], [1, 3]], dtype=float)
b = np.array([5, 10], dtype=float)
x = cramers_rule(A, b)
print(f"解: x = {x[0]:.1f}, y = {x[1]:.1f}")  # x = 1.0, y = 3.0

# 验证: 用 numpy 直接求解
x_np = np.linalg.solve(A, b)
print(f"验证: {x_np}")  # [1. 3.]
```

### 行列式性质验证

```python
import numpy as np

A = np.array([[1, 2, 3], [4, 5, 6], [7, 8, 10]])

# 性质1: |A| = |A^T|
print(f"|A| = {np.linalg.det(A):.1f}")
print(f"|A^T| = {np.linalg.det(A.T):.1f}")  # 相等

# 性质3: |kA| = k^n * |A| (n为阶数)
k = 3
n = A.shape[0]
det_kA = np.linalg.det(k * A)
det_k_relation = k**n * np.linalg.det(A)
print(f"|3A| = {det_kA:.1f}, 3^3 * |A| = {det_k_relation:.1f}")

# 性质8: 某行的k倍加到另一行, 行列式不变
A_modified = A.copy().astype(float)
A_modified[1] = A_modified[1] + 2 * A_modified[0]  # r2 + 2*r1
print(f"|A| = {np.linalg.det(A):.1f}, |A变换后| = {np.linalg.det(A_modified):.1f}")
```
