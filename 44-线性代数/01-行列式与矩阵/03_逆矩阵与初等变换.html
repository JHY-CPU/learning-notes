03 - 逆矩阵与初等变换

一、逆矩阵的定义 核心概念

对于 n 阶方阵 A，若存在 n 阶方阵 B 使得：

AB = BA = Iₙ

则称 A 是可逆矩阵（非奇异矩阵），B 称为 A 的逆矩阵，记作 A⁻¹。

矩阵可逆的等价条件

以下条件等价（即满足其一即满足全部）：
1. A 可逆（存在 A⁻¹）
2. |A| ≠ 0（A 是非奇异矩阵）
3. rank(A) = n（A 满秩）
4. A 的行（列）向量线性无关
5. Ax = 0 只有零解
6. A 的所有特征值均不为零
7. A 可以表示为有限个初等矩阵的乘积

二、逆矩阵的性质与公式

运算性质

- (A⁻¹)⁻¹ = A
- (kA)⁻¹ = (1/k)A⁻¹（k ≠ 0）
- (AB)⁻¹ = B⁻¹A⁻¹（注意顺序反转！）
- (Aᵀ)⁻¹ = (A⁻¹)ᵀ
- |A⁻¹| = 1/|A|

求逆公式

A⁻¹ = (1/|A|) · A*

其中 A* 是 A 的伴随矩阵，其第 j 行第 i 列元素 = aᵢⱼ 的代数余子式 Cᵢⱼ。

【二阶矩阵逆矩阵速记公式】
若 A = a b c d ，则 A⁻¹ = (1/(ad-bc)) d -b -c a

口诀：主对角线互换，副对角线变号，除以行列式

三、初等变换与初等矩阵

三种初等行变换

变换类型操作对应初等矩阵
交换两行 rᵢ ↔ rⱼ E(i,j) — 交换 I 的第 i、j 行
某行乘以非零常数 rᵢ × k（k ≠ 0） E(i(k)) — I 的第 i 行乘 k
某行的倍数加到另一行 rⱼ + k·rᵢ E(i,j(k)) — I 的 (j,i) 位置填 k
初等矩阵的性质：
- 初等矩阵均可逆，且逆仍为初等矩阵
- 对 A 做初等行变换 = 用初等矩阵左乘 A
- 对 A 做初等列变换 = 用初等矩阵右乘 A

四、高斯消元法（初等行变换求逆） 必考方法

步骤

将增广矩阵 [A | I] 通过初等行变换变为 [I | A⁻¹]。

【例题】求矩阵 A 的逆
A = 1 2 3 7

增广矩阵： 1 2 |1 0 3 7 |0 1

r₂ - 3r₁ → r₂：
1 2 |1 0 0 1 |-3 1
r₁ - 2r₂ → r₁：
1 0 |7 -2 0 1 |-3 1
结果：A⁻¹ = 7 -2 -3 1

五、高斯-约当消元法求解线性方程组

行阶梯形与行最简形

行阶梯形：非零行的首个非零元素（主元）所在列号严格递增
行最简形：在行阶梯形基础上，每个主元为1且主元所在列的其他元素全为0

解的判断

条件解的情况
rank(A) = rank([A|b]) = n唯一解
rank(A) = rank([A|b]) < n无穷多解（含自由变量）
rank(A) < rank([A|b])无解（矛盾方程组）

六、分块矩阵的逆

对角分块

若 A = diag(A₁, A₂, ..., Aₖ)，则 A⁻¹ = diag(A₁⁻¹, A₂⁻¹, ..., Aₖ⁻¹)

二阶分块求逆

若 A = B C O D （B、D 可逆），则 A⁻¹ = B⁻¹ -B⁻¹CD⁻¹ O D⁻¹
注意：矩阵分块求逆时，各子块之间的乘法顺序不可随意交换。
