# 内积与正交


## 一、内积（Inner Product）的定义 核心概念


### 实向量的内积


设 α = (a₁, a₂, ..., aₙ)ᵀ，β = (b₁, b₂, ..., bₙ)ᵀ，则：


$$
(α, β) = αᵀβ = a₁b₁ + a₂b₂ + ... + aₙbₙ = Σ aᵢbᵢ
$$


### 内积的公理（抽象定义）


| 公理 | 描述 |
| --- | --- |
| 对称性 | (α, β) = (β, α) |
| 线性 | (kα + lβ, γ) = k(α, γ) + l(β, γ) |
| 正定性 | (α, α) ≥ 0，且 (α, α) = 0 ⟺ α = 0 |


### 范数（向量长度）


$$
‖α‖ = √(α, α) = √(a₁² + a₂² + ... + aₙ²)
$$


### 夹角


$$
cos θ = (α, β) / (‖α‖ · ‖β‖)
$$


### 柯西-施瓦茨不等式


$$
|(α, β)| ≤ ‖α‖ · ‖β‖
$$


## 二、正交（Orthogonality）


### 正交的定义


$$
α 与 β 正交 ⟺ (α, β) = 0 ⟺ αᵀβ = 0
$$


### 正交向量组


向量组 α₁, α₂, ..., αₛ 中两两正交，即 (αᵢ, αⱼ) = 0（i ≠ j）。

**重要性质：**


- 非零的正交向量组一定
**线性无关**


- 两两正交的非零向量组称为
**正交基**


- 正交基中每个向量都单位化后称为
**标准正交基（规范正交基）**

### 标准正交基


ε₁, ε₂, ..., εₙ 是标准正交基 ⟺


$$
(εᵢ, εⱼ) = δᵢⱼ = { 1, 当 i=j; 0, 当 i≠j }
$$


> **Example:** **【示例】**
> R³ 的标准正交基：
>
>
> e₁ = (1, 0, 0), e₂ = (0, 1, 0), e₃ = (0, 0, 1)
>
>
>
>
> 另一组标准正交基：
>
>
> u₁ = (1/√2, 1/√2, 0), u₂ = (1/√2, -1/√2, 0), u₃ = (0, 0, 1)


## 三、Gram-Schmidt 正交化 核心算法


将线性无关向量组 α₁, α₂, ..., αₛ 化为正交向量组 β₁, β₂, ..., βₛ：


$$
正交化：
        β₁ = α₁
        β₂ = α₂ - (α₂, β₁)/(β₁, β₁) · β₁
        β₃ = α₃ - (α₃, β₁)/(β₁, β₁) · β₁ - (α₃, β₂)/(β₂, β₂) · β₂
        ...
        βₖ = αₖ - Σ (j=1→k-1) (αₖ, βⱼ)/(βⱼ, βⱼ) · βⱼ
$$


$$
单位化：
        εₖ = βₖ / ‖βₖ‖
$$


> **Example:** **【例题】**
> 将 α₁=(1,1,0), α₂=(1,0,1), α₃=(0,1,1) 标准正交化。
>
>
>
>
>
> > **Step:** 正交化：
> >
> >
> > β₁ = α₁ = (1, 1, 0)
> >
> >
> > β₂ = α₂ - (α₂·β₁)/(β₁·β₁) · β₁ = (1,0,1) - 1/2·(1,1,0) = (1/2, -1/2, 1)
> >
> >
> > β₃ = α₃ - (α₃·β₁)/(β₁·β₁)·β₁ - (α₃·β₂)/(β₂·β₂)·β₂
> >
> >
> > = (0,1,1) - 1/2·(1,1,0) - 1/3·(1/2,-1/2,1)
> >
> >
> > = (-2/3, 2/3, 2/3)
>
>
> > **Step:** 单位化：
> >
> >
> > ε₁ = (1/√2, 1/√2, 0)
> >
> >
> > ε₂ = (1/√6, -1/√6, 2/√6)
> >
> >
> > ε₃ = (-1/√3, 1/√3, 1/√3)


## 四、正交投影 重要应用


### 向量到子空间的投影


设 W = Span{u₁, u₂, ..., uₖ}（uᵢ 标准正交），向量 y 在 W 上的正交投影为：


$$
proj_W(y) = (y·u₁)u₁ + (y·u₂)u₂ + ... + (y·uₖ)uₖ
$$


### 投影矩阵


$$
P = UUᵀ   （U 的列为 W 的标准正交基）
        proj_W(y) = P · y
$$

**投影矩阵的性质：**


- P² = P（幂等矩阵）


- Pᵀ = P（对称矩阵）


- rank(P) = dim(W)


- 正交补上的投影：proj_{W⊥}(y) = y - proj_W(y) = (I - P)y

## 五、正交矩阵


n 阶方阵 Q 满足 `QᵀQ = QQᵀ = I` 时称为**正交矩阵**。

**等价条件：**


1. QᵀQ = I


2. Q⁻¹ = Qᵀ


3. Q 的列（行）向量是 Rⁿ 的标准正交基


4. |Q| = ±1

### 常见正交矩阵


| 类型 | 示例 |
| --- | --- |
| 旋转矩阵 R(θ) | cosθ
-sinθ
sinθ
cosθ

                （\|R\| = 1） |
| 反射矩阵 H | H = I - 2uuᵀ（u 为单位向量，\|H\| = -1） |
