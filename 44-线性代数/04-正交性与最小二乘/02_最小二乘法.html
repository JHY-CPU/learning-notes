02 - 最小二乘法

一、问题背景：超定方程组

当线性方程组 Ax = b 的方程数 m 大于未知数个数 n 时（m > n），通常无精确解。此时求"最接近"的解。

超定方程组：A 是 m×n 矩阵（m > n），rank(A) = n（列满秩）
Ax = b 无解 ⟺ b 不在 Col(A) 中
目标：找 x 使 ‖Ax - b‖² 最小（残差最小）

二、最小二乘解（Least Squares Solution）核心方法

定义

最小二乘问题是求 x 使得残差的平方和最小：

minimize ‖Ax - b‖² = Σᵢ (aᵢᵀx - bᵢ)²

正规方程（Normal Equations）

最小二乘解满足：

AᵀAx = Aᵀb

若 A 列满秩（rank(A) = n），则 AᵀA 可逆，最小二乘解唯一：

x = (AᵀA)⁻¹Aᵀb

几何解释

最小二乘解 = b 在 Col(A) 上的正交投影

r = b - Ax = b - A(AᵀA)⁻¹Aᵀb = b - Pb
其中 P = A(AᵀA)⁻¹Aᵀ 是投影矩阵
残差 r ⊥ Col(A)，即 Aᵀr = Aᵀ(b - Ax) = 0

三、计算步骤

方法1：正规方程
Step 1：计算 AᵀA（n×n 矩阵）
Step 2：计算 Aᵀb（n×1 向量）
Step 3：解方程组 AᵀAx = Aᵀb
方法2：QR分解
若 A = QR（QR 分解），则：
x = R⁻¹Qᵀb
数值稳定性更好。
方法3：SVD分解
若 A = UΣVᵀ，则：
x = VΣ⁺Uᵀb（Σ⁺ 为伪逆）
适用于 rank(A) < n 的情况（此时解不唯一，取范数最小的解）。
【例题：线性回归】拟合数据点 (1,1), (2,1), (3,2) 的直线 y = c₀ + c₁x

构造方程组： 1 1 1 2 1 3 c₀ = 1 1 2

AᵀA = 3 6 6 14 , Aᵀb = 4 9

解正规方程得：c₀ = 1/6, c₁ = 1/2
最佳拟合直线：y = 1/6 + x/2

四、最小二乘在线性回归中的应用

线性回归模型

y = Xβ + ε

其中 X 是设计矩阵（m×n），β 是回归系数（n×1），ε 是误差向量。

β̂ = (XᵀX)⁻¹Xᵀy

预测与残差

- 预测值：ŷ = Xβ̂ = Py（P = X(XᵀX)⁻¹Xᵀ 为投影矩阵/"帽子矩阵"）
- 残差：e = y - ŷ
- 残差平方和：SSE = ‖e‖² = eᵀe
- 总平方和：SST = ‖y - ȳ·1‖²
- 决定系数：R² = 1 - SSE/SST（越接近1拟合越好）

五、加权最小二乘

当各观测值的精度不同时，引入权重矩阵 W（对角矩阵，对角线为权重）：

minimize (Ax - b)ᵀW(Ax - b)
x = (AᵀWA)⁻¹AᵀWb
特例：
- 普通最小二乘：W = I
- 广义最小二乘：W = Σ⁻¹（Σ 为误差协方差矩阵）

六、最小二乘的其他应用

应用模型说明
多项式拟合 y = a₀ + a₁x + a₂x² + ... X 的列为 1, x, x², ...
曲线拟合 y = a·eᵇˣ, y = a·xᵇ 等 取对数转化为线性问题
信号处理 y = Σ cᵢφᵢ(x) 基函数展开
机器学习 y = wᵀx + b 线性回归的参数估计
