# 最小二乘法


## 一、问题背景：超定方程组


当线性方程组 Ax = b 的方程数 m 大于未知数个数 n 时（m > n），通常**无精确解**。此时求"最接近"的解。

**超定方程组：**
A 是 m×n 矩阵（m > n），rank(A) = n（列满秩）


Ax = b 无解 ⟺ b 不在 Col(A) 中


**目标：**
找 x 使 ‖Ax - b‖² 最小（残差最小）

## 二、最小二乘解（Least Squares Solution）核心方法


### 定义


最小二乘问题是求 x 使得残差的平方和最小：


$$
minimize ‖Ax - b‖² = Σᵢ (aᵢᵀx - bᵢ)²
$$


### 正规方程（Normal Equations）


最小二乘解满足：


$$
AᵀAx = Aᵀb
$$


若 A 列满秩（rank(A) = n），则 AᵀA 可逆，最小二乘解唯一：


$$
x = (AᵀA)⁻¹Aᵀb
$$


### 几何解释

最小二乘解 = b 在 Col(A) 上的正交投影


r = b - Ax = b - A(AᵀA)⁻¹Aᵀb = b - Pb


其中 P = A(AᵀA)⁻¹Aᵀ 是投影矩阵


残差 r ⊥ Col(A)，即 Aᵀr = Aᵀ(b - Ax) = 0

## 三、计算步骤

**方法1：正规方程**


Step 1：计算 AᵀA（n×n 矩阵）


Step 2：计算 Aᵀb（n×1 向量）


Step 3：解方程组 AᵀAx = Aᵀb
**方法2：QR分解**


若 A = QR（QR 分解），则：


x = R⁻¹Qᵀb


数值稳定性更好。
**方法3：SVD分解**


若 A = UΣVᵀ，则：


x = VΣ⁺Uᵀb（Σ⁺ 为伪逆）


适用于 rank(A) < n 的情况（此时解不唯一，取范数最小的解）。

## 四、最小二乘在线性回归中的应用


### 线性回归模型


$$
y = Xβ + ε
$$


其中 X 是设计矩阵（m×n），β 是回归系数（n×1），ε 是误差向量。


$$
β̂ = (XᵀX)⁻¹Xᵀy
$$


### 预测与残差

- 预测值：ŷ = Xβ̂ = Py（P = X(XᵀX)⁻¹Xᵀ 为投影矩阵/"帽子矩阵"）


- 残差：e = y - ŷ


- 残差平方和：SSE = ‖e‖² = eᵀe


- 总平方和：SST = ‖y - ȳ·1‖²


- 决定系数：R² = 1 - SSE/SST（越接近1拟合越好）

## 五、加权最小二乘


当各观测值的精度不同时，引入权重矩阵 W（对角矩阵，对角线为权重）：


$$
minimize (Ax - b)ᵀW(Ax - b)
$$


$$
x = (AᵀWA)⁻¹AᵀWb
$$

**特例：**


- 普通最小二乘：W = I


- 广义最小二乘：W = Σ⁻¹（Σ 为误差协方差矩阵）

## 六、最小二乘的其他应用


| 应用 | 模型 | 说明 |
| --- | --- | --- |
| 多项式拟合 | y = a₀ + a₁x + a₂x² + ... | X 的列为 1, x, x², ... |
| 曲线拟合 | y = a·eᵇˣ, y = a·xᵇ 等 | 取对数转化为线性问题 |
| 信号处理 | y = Σ cᵢφᵢ(x) | 基函数展开 |
| 机器学习 | y = wᵀx + b | 线性回归的参数估计 |


## Python实现

### 最小二乘法求解

```python
import numpy as np

# 数据点: (1,1), (2,1), (3,2) 拟合 y = c0 + c1*x
x_data = np.array([1, 2, 3], dtype=float)
y_data = np.array([1, 1, 2], dtype=float)

# 构造设计矩阵 A
A = np.column_stack([np.ones_like(x_data), x_data])
print(f"设计矩阵 A =\n{A}")

# 方法1: 正规方程 A^T A x = A^T b
ATA = A.T @ A
ATb = A.T @ y_data
x_normal = np.linalg.solve(ATA, ATb)
print(f"正规方程解: c0 = {x_normal[0]:.4f}, c1 = {x_normal[1]:.4f}")

# 方法2: numpy lstsq (推荐)
x_lstsq, residuals, rank, sv = np.linalg.lstsq(A, y_data, rcond=None)
print(f"lstsq 解: c0 = {x_lstsq[0]:.4f}, c1 = {x_lstsq[1]:.4f}")

# 方法3: QR 分解
Q, R = np.linalg.qr(A)
x_qr = np.linalg.solve(R, Q.T @ y_data)
print(f"QR 分解解: c0 = {x_qr[0]:.4f}, c1 = {x_qr[1]:.4f}")

# 计算 R^2 (决定系数)
y_pred = A @ x_lstsq
SSE = np.sum((y_data - y_pred)**2)
SST = np.sum((y_data - np.mean(y_data))**2)
R_squared = 1 - SSE / SST
print(f"R^2 = {R_squared:.4f}")
```

### 多项式拟合

```python
import numpy as np

# 用最小二乘做多项式拟合
x_data = np.array([0, 1, 2, 3, 4, 5], dtype=float)
y_data = np.array([1, 2.5, 4.2, 8.1, 15.5, 26.0], dtype=float)

# 拟合 y = a0 + a1*x + a2*x^2
degree = 2
A = np.column_stack([x_data**i for i in range(degree + 1)])
coeffs, _, _, _ = np.linalg.lstsq(A, y_data, rcond=None)

print(f"拟合系数: {np.round(coeffs, 4)}")
print(f"拟合方程: y = {coeffs[0]:.2f} + {coeffs[1]:.2f}x + {coeffs[2]:.2f}x^2")

# 预测
y_pred = A @ coeffs
print(f"拟合值: {np.round(y_pred, 2)}")
print(f"残差: {np.round(y_data - y_pred, 4)}")

# 投影矩阵 (帽子矩阵)
P = A @ np.linalg.inv(A.T @ A) @ A.T
print(f"帽子矩阵迹 (应等于 degree+1): {np.trace(P):.1f}")
```

### 加权最小二乘

```python
import numpy as np

A = np.array([[1, 1], [1, 2], [1, 3], [1, 4]], dtype=float)
b = np.array([2.1, 3.9, 6.2, 7.8], dtype=float)

# 权重矩阵: 前两个观测精度更高
W = np.diag([10, 10, 1, 1])
x_weighted = np.linalg.solve(A.T @ W @ A, A.T @ W @ b)
x_ordinary = np.linalg.lstsq(A, b, rcond=None)[0]

print(f"普通最小二乘: {np.round(x_ordinary, 4)}")
print(f"加权最小二乘: {np.round(x_weighted, 4)}")
```
