# 矩阵求导与应用


## 一、标量对向量求导（梯度）基础


### 定义


设 f: Rⁿ → R 是标量函数，x = (x₁, x₂, ..., xₙ)ᵀ，定义：


$$
∂f/∂x = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)ᵀ
$$


即 f 对 x 的梯度（gradient），记作 ∇f 或 grad f。

**布局约定：**


- 分子布局（numerator layout）：∂f/∂x 是列向量（与 x 同维）


- 分母布局（denominator layout）：∂f/∂x 是行向量


- 本文采用
**分母布局**
（∂f/∂x 为列向量）

## 二、常用矩阵求导公式 必记


### 向量运算


| 函数 f(x) | ∂f/∂x | 结果维度 |
| --- | --- | --- |
| aᵀx（线性函数） | a | n×1 |
| xᵀa（同上） | a | n×1 |
| xᵀx = ‖x‖² | 2x | n×1 |
| xᵀAx（二次型） | (A + Aᵀ)x | n×1 |
| xᵀAx（A 对称） | 2Ax | n×1 |


### 矩阵运算


| 函数 | 导数 |
| --- | --- |
| ∂(aᵀx)/∂x | a |
| ∂(xᵀAx)/∂x | (A + Aᵀ)x |
| ∂(tr(AX))/∂X | Aᵀ |
| ∂(tr(XᵀBX))/∂X | (B + Bᵀ)X |
| ∂(\|X\|)/∂X | \|X\| · (X⁻¹)ᵀ |
| ∂(ln\|X\|)/∂X | (X⁻¹)ᵀ |


## 三、Hessian 矩阵（二阶导数）

$$

## 四、梯度下降法（Gradient Descent）ML核心


### 基本思想


沿梯度的反方向移动使函数值下降：


$$
xₖ₊₁ = xₖ - α · ∇f(xₖ)
$$


其中 α > 0 是学习率（步长）。


### 变种


| 方法 | 更新公式 | 特点 |
| --- | --- | --- |
| 批量梯度下降 (BGD) | θ = θ - α·∇J(θ) | 用全部数据计算梯度 |
| 随机梯度下降 (SGD) | θ = θ - α·∇Jᵢ(θ) | 每次用一个样本 |
| 小批量梯度下降 | θ = θ - α·(1/b)Σ∇Jᵢ(θ) | 每次用一批样本 |
| 动量法 (Momentum) | v = βv + α∇J; θ = θ - v | 加速收敛，减少振荡 |
| Adam | 结合动量 + 自适应学习率 | 最常用优化器 |


> **Example:** **【线性回归的梯度下降】**
>
>
> 损失函数：J(w) = (1/2m)‖Xw - y‖²
>
>
> 梯度：∇J(w) = (1/m)Xᵀ(Xw - y)
>
>
> 更新：w ← w - α · (1/m)Xᵀ(Xw - y)


## 五、矩阵微分在机器学习中的应用


### 1. 线性回归


$$
最小化 J(w) = (1/2m)‖Xw - y‖²
        解析解：w* = (XᵀX)⁻¹Xᵀy
$$


### 2. 逻辑回归


$$
损失函数：J(θ) = -(1/m)Σ[yᵢlog(hᵢ) + (1-yᵢ)log(1-hᵢ)]
        其中 hᵢ = σ(θᵀxᵢ) = 1/(1 + e^(-θᵀxᵢ))
        梯度：∇J(θ) = (1/m)Xᵀ(h - y)
$$


### 3. 神经网络反向传播

链式法则的矩阵形式：


∂L/∂W⁽ˡ⁾ = ∂L/∂z⁽ˡ⁾ · (a⁽ˡ⁻¹⁾)ᵀ


∂L/∂a⁽ˡ⁻¹⁾ = (W⁽ˡ⁾)ᵀ · ∂L/∂z⁽ˡ⁾


反向传播 = 矩阵微分 + 链式法则

### 4. 协方差矩阵与PCA


$$
协方差矩阵：C = (1/n)XXᵀ（X 已中心化）
        PCA：对 C 做特征分解，取前 k 个最大特征值对应的特征向量
$$
