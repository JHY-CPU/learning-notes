# 矩阵求导与应用


## 一、标量对向量求导（梯度）基础


### 定义


设 f: Rⁿ → R 是标量函数，x = (x₁, x₂, ..., xₙ)ᵀ，定义：


$$
∂f/∂x = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)ᵀ
$$


即 f 对 x 的梯度（gradient），记作 ∇f 或 grad f。

**布局约定：**


- 分子布局（numerator layout）：∂f/∂x 是列向量（与 x 同维）


- 分母布局（denominator layout）：∂f/∂x 是行向量


- 本文采用
**分母布局**
（∂f/∂x 为列向量）

## 二、常用矩阵求导公式 必记


### 向量运算


| 函数 f(x) | ∂f/∂x | 结果维度 |
| --- | --- | --- |
| aᵀx（线性函数） | a | n×1 |
| xᵀa（同上） | a | n×1 |
| xᵀx = ‖x‖² | 2x | n×1 |
| xᵀAx（二次型） | (A + Aᵀ)x | n×1 |
| xᵀAx（A 对称） | 2Ax | n×1 |


### 矩阵运算


| 函数 | 导数 |
| --- | --- |
| ∂(aᵀx)/∂x | a |
| ∂(xᵀAx)/∂x | (A + Aᵀ)x |
| ∂(tr(AX))/∂X | Aᵀ |
| ∂(tr(XᵀBX))/∂X | (B + Bᵀ)X |
| ∂(\|X\|)/∂X | \|X\| · (X⁻¹)ᵀ |
| ∂(ln\|X\|)/∂X | (X⁻¹)ᵀ |


## 三、Hessian 矩阵（二阶导数）


设 f: Rⁿ → R，f 的 Hessian 矩阵是 n×n 对称矩阵：


$$
H = ∇²f = [∂²f/∂xᵢ∂xⱼ]_{n×n}
$$

## 四、梯度下降法（Gradient Descent）ML核心


### 基本思想


沿梯度的反方向移动使函数值下降：


$$
xₖ₊₁ = xₖ - α · ∇f(xₖ)
$$


其中 α > 0 是学习率（步长）。


### 变种


| 方法 | 更新公式 | 特点 |
| --- | --- | --- |
| 批量梯度下降 (BGD) | θ = θ - α·∇J(θ) | 用全部数据计算梯度 |
| 随机梯度下降 (SGD) | θ = θ - α·∇Jᵢ(θ) | 每次用一个样本 |
| 小批量梯度下降 | θ = θ - α·(1/b)Σ∇Jᵢ(θ) | 每次用一批样本 |
| 动量法 (Momentum) | v = βv + α∇J; θ = θ - v | 加速收敛，减少振荡 |
| Adam | 结合动量 + 自适应学习率 | 最常用优化器 |


> **Example:** **【线性回归的梯度下降】**
>
>
> 损失函数：J(w) = (1/2m)‖Xw - y‖²
>
>
> 梯度：∇J(w) = (1/m)Xᵀ(Xw - y)
>
>
> 更新：w ← w - α · (1/m)Xᵀ(Xw - y)


## 五、矩阵微分在机器学习中的应用


### 1. 线性回归


$$
最小化 J(w) = (1/2m)‖Xw - y‖²
        解析解：w* = (XᵀX)⁻¹Xᵀy
$$


### 2. 逻辑回归


$$
损失函数：J(θ) = -(1/m)Σ[yᵢlog(hᵢ) + (1-yᵢ)log(1-hᵢ)]
        其中 hᵢ = σ(θᵀxᵢ) = 1/(1 + e^(-θᵀxᵢ))
        梯度：∇J(θ) = (1/m)Xᵀ(h - y)
$$


### 3. 神经网络反向传播

链式法则的矩阵形式：


∂L/∂W⁽ˡ⁾ = ∂L/∂z⁽ˡ⁾ · (a⁽ˡ⁻¹⁾)ᵀ


∂L/∂a⁽ˡ⁻¹⁾ = (W⁽ˡ⁾)ᵀ · ∂L/∂z⁽ˡ⁾


反向传播 = 矩阵微分 + 链式法则

### 4. 协方差矩阵与PCA


$$
协方差矩阵：C = (1/n)XXᵀ（X 已中心化）
        PCA：对 C 做特征分解，取前 k 个最大特征值对应的特征向量
$$


## 六、矩阵微分速查表


| 表达式 | 对 x 的导数 | 维度 |
| --- | --- | --- |
| c（常数） | 0 | 与 x 同维 |
| aᵀx | a | n×1 |
| Ax | Aᵀ | n×m |
| xᵀAx | (A+Aᵀ)x | n×1 |
| ‖Ax - b‖² | 2Aᵀ(Ax-b) | n×1 |
| xᵀx | 2x | n×1 |
| log(xᵀAx) | 2Ax/(xᵀAx) | n×1 |
| exp(aᵀx) | exp(aᵀx)·a | n×1 |

## Python实现

### 梯度与 Hessian 矩阵的符号计算

```python
import sympy as sp

# 定义符号变量
x1, x2 = sp.symbols('x1 x2')
x = sp.Matrix([x1, x2])

# 标量函数 f(x) = x1^2 + 2*x1*x2 + 3*x2^2
f = x1**2 + 2*x1*x2 + 3*x2**2

# 梯度 ∇f
grad_f = sp.Matrix([sp.diff(f, var) for var in [x1, x2]])
print(f"f = {f}")
print(f"∇f = {grad_f.T}")

# Hessian 矩阵
H = sp.hessian(f, [x1, x2])
print(f"Hessian H =\n{H}")

# 验证: 二次函数 f = (1/2) x^T A x - b^T x + c 的 Hessian = A
A_sym = sp.Matrix([[2, 2], [2, 6]])  # 对称矩阵
f_quad = sp.Rational(1, 2) * (x.T @ A_sym @ x)[0]
H_quad = sp.hessian(f_quad, [x1, x2])
print(f"二次型 Hessian:\n{H_quad}")
```

### 梯度下降法实现

```python
import numpy as np

def gradient_descent(f_grad, x0, lr=0.1, max_iter=1000, tol=1e-6):
    """梯度下降法"""
    x = x0.copy().astype(float)
    history = [x.copy()]
    for i in range(max_iter):
        grad = f_grad(x)
        x = x - lr * grad
        history.append(x.copy())
        if np.linalg.norm(grad) < tol:
            print(f"收敛于第 {i+1} 次迭代")
            break
    return x, history

# 示例: 最小化 f(x,y) = (x-3)^2 + (y-2)^2
# 梯度: ∇f = [2(x-3), 2(y-2)]
f_grad = lambda x: np.array([2*(x[0]-3), 2*(x[1]-2)])

x0 = np.array([0.0, 0.0])
x_opt, history = gradient_descent(f_grad, x0, lr=0.1)
print(f"最优解: x = {x_opt}")  # [3, 2]
print(f"最优值: f = {(x_opt[0]-3)**2 + (x_opt[1]-2)**2:.8f}")
```

### 线性回归的矩阵微分

```python
import numpy as np

# 线性回归: y = Xw, 最小化 J(w) = (1/2m)||Xw - y||^2
np.random.seed(42)
X = np.column_stack([np.ones(100), np.random.randn(100, 2)])  # 100x3
true_w = np.array([1.5, -2.0, 0.5])
y = X @ true_w + 0.1 * np.random.randn(100)

# 解析解: w* = (X^T X)^(-1) X^T y
w_analytic = np.linalg.solve(X.T @ X, X.T @ y)
print(f"解析解: {np.round(w_analytic, 4)}")

# 梯度下降解
def linear_regression_gd(X, y, lr=0.01, epochs=1000):
    m, n = X.shape
    w = np.zeros(n)
    for epoch in range(epochs):
        gradient = (1/m) * X.T @ (X @ w - y)
        w -= lr * gradient
    return w

w_gd = linear_regression_gd(X, y)
print(f"梯度下降解: {np.round(w_gd, 4)}")
print(f"真值: {true_w}")
print(f"解接近真值: {np.allclose(w_analytic, true_w, atol=0.2)}")
```

### PCA 与协方差矩阵

```python
import numpy as np

# 生成二维数据
np.random.seed(42)
data = np.random.randn(100, 3) @ np.array([[3, 1, 0], [0, 2, 1], [0, 0, 1]])

# 中心化
data_centered = data - np.mean(data, axis=0)

# 协方差矩阵
cov_matrix = (1 / len(data)) * (data_centered.T @ data_centered)
print(f"协方差矩阵:\n{np.round(cov_matrix, 2)}")

# PCA: 特征分解
eigenvalues, eigenvectors = np.linalg.eigh(cov_matrix)
idx = np.argsort(eigenvalues)[::-1]
eigenvalues = eigenvalues[idx]
eigenvectors = eigenvectors[:, idx]

print(f"特征值(方差): {np.round(eigenvalues, 2)}")
print(f"方差解释比: {np.round(eigenvalues / np.sum(eigenvalues) * 100, 1)}%")
```
