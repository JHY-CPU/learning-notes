# 特征值与特征向量


## 一、特征值与特征向量的定义 核心概念


设 A 是 n 阶方阵，若存在数 λ 和非零向量 x 使得：


$$
Ax = λx
$$


则称 λ 是 A 的**特征值**，x 是对应于 λ 的**特征向量**。

**几何意义：**
特征向量在变换后方向不变（可能反向），只发生伸缩。伸缩倍数就是特征值。

### 特征多项式


$$
f(λ) = |λI - A| = λⁿ - tr(A)λⁿ⁻¹ + ... + (-1)ⁿ|A|
$$


特征多项式的根就是特征值。


### 特征方程


$$
|λI - A| = 0
$$


## 二、求解特征值与特征向量的步骤

**Step 1：**
计算特征多项式 |λI - A| = 0，解出特征值 λ₁, λ₂, ..., λₖ


**Step 2：**
对每个特征值 λᵢ，求齐次方程组 (λᵢI - A)x = 0 的基础解系


**Step 3：**
基础解系的非零线性组合就是对应于 λᵢ 的所有特征向量

> **Example:** **【例题】**
> 求 A =
>
> 3
> 1
> 1
> 3
>
> 的特征值与特征向量。
>
>
>

>

> λ-3
> -1
> -1
> λ-3
>
> = (λ-3)² - 1 = λ² - 6λ + 8 = (λ-4)(λ-2) = 0
>
>
> 特征值：λ₁ = 4, λ₂ = 2
>
>
>
>

>
> 1

>

>
>

> **Step 2b（λ₂ = 2）：**
>
>
> (2I - A)x = 0 → x₁ = -x₂，特征向量：x = k(1,-1)ᵀ（k ≠ 0）


## 三、特征值的重要性质 必记


| 性质 | 说明 |
| --- | --- |
| 特征值之和 = 迹 | λ₁ + λ₂ + ... + λₙ = tr(A) = a₁₁ + a₂₂ + ... + aₙₙ |
| 特征值之积 = 行列式 | λ₁ × λ₂ × ... × λₙ = \|A\| |
| A 可逆 ⟺ 所有 λᵢ ≠ 0 | \|A\| ≠ 0 ⟺ 无零特征值 |
| A 和 Aᵀ 有相同的特征值 | \|λI - A\| = \|λI - Aᵀ\| |
| 不同特征值的特征向量线性无关 | 重要结论 |


### 特征值的运算性质


$$
若 A 的特征值为 λ，特征向量为 x：
        - A² 的特征值为 λ²，特征向量仍为 x
        - A⁻¹ 的特征值为 1/λ（A 可逆时）
        - A + kI 的特征值为 λ + k
        - kA 的特征值为 kλ
        - f(A) 的特征值为 f(λ)（f 为多项式）
        - A*（伴随矩阵）的特征值为 |A|/λ
$$


## 四、特征子空间（特征空间）


对应于特征值 λ 的所有特征向量加上零向量构成的子空间：


$$
Eλ = Ker(λI - A) = {x | (λI - A)x = 0}
$$


Eλ 的维数称为特征值 λ 的**几何重数**（geometric multiplicity）。

**代数重数 vs 几何重数：**


- 代数重数：λ 作为特征多项式根的重数


- 几何重数：dim(Eλ) = n - rank(λI - A)


-
**几何重数 ≤ 代数重数**
（等号成立时可对角化）

> **Warning:** **注意：**
> 一个特征值对应的特征向量有无穷多个（非零标量倍数），但特征子空间的维数有限。


## 五、特殊矩阵的特征值


| 矩阵类型 | 特征值特点 |
| --- | --- |
| 对称矩阵 A = Aᵀ | 所有特征值为实数 |
| 正交矩阵 QᵀQ = I | \|λ\| = 1（在单位圆上） |
| 幂等矩阵 A² = A | 特征值为 0 或 1 |
| 对合矩阵 A² = I | 特征值为 1 或 -1 |
| 幂零矩阵 Aᵏ = 0 | 所有特征值为 0 |
| 上/下三角矩阵 | 特征值 = 对角线元素 |


## Python实现

### 特征值与特征向量的计算

```python
import numpy as np

A = np.array([[3, 1], [1, 3]])

# 计算特征值和特征向量
eigenvalues, eigenvectors = np.linalg.eig(A)
print(f"特征值: {eigenvalues}")           # [4, 2]
print(f"特征向量矩阵:\n{eigenvectors}")    # 每列为一个特征向量

# 验证 Ax = λx
for i, lam in enumerate(eigenvalues):
    v = eigenvectors[:, i]
    lhs = A @ v
    rhs = lam * v
    print(f"λ={lam}: Ax = {np.round(lhs, 4)}, λx = {np.round(rhs, 4)}")
    print(f"  验证: {np.allclose(lhs, rhs)}")

# 验证性质: 特征值之和 = 迹, 特征值之积 = 行列式
print(f"特征值之和 = {np.sum(eigenvalues):.1f}, tr(A) = {np.trace(A)}")
print(f"特征值之积 = {np.prod(eigenvalues):.1f}, |A| = {np.linalg.det(A):.1f}")
```

### 特征值的运算性质验证

```python
import numpy as np

A = np.array([[3, 1], [1, 3]])
eigenvalues, eigenvectors = np.linalg.eig(A)
lam = eigenvalues[0]   # 特征值 4
v = eigenvectors[:, 0]  # 对应的特征向量

# A^2 的特征值为 λ^2
A2 = A @ A
eig_A2 = np.linalg.eigvals(A2)
print(f"A^2 的特征值: {np.sort(eig_A2)}")        # [4, 16] = [2^2, 4^2]
print(f"λ^2: {np.sort(eigenvalues**2)}")

# A^(-1) 的特征值为 1/λ
A_inv = np.linalg.inv(A)
eig_inv = np.linalg.eigvals(A_inv)
print(f"A^(-1) 的特征值: {np.sort(eig_inv)}")    # [1/4, 1/2]

# A + kI 的特征值为 λ + k
k = 5
eig_shifted = np.linalg.eigvals(A + k * np.eye(2))
print(f"A + 5I 的特征值: {np.sort(eig_shifted)}")  # [7, 9] = [2+5, 4+5]
```

### 特殊矩阵特征值验证

```python
import numpy as np

# 对称矩阵: 特征值为实数
S = np.array([[1, 2], [2, 3]])
eig_S = np.linalg.eigvals(S)
print(f"对称矩阵特征值: {eig_S}, 全为实数: {np.all(np.isreal(eig_S))}")

# 幂等矩阵 A^2 = A: 特征值为 0 或 1
P = np.array([[0.5, 0.5], [0.5, 0.5]])  # 投影矩阵
print(f"P^2 = P? {np.allclose(P @ P, P)}")
eig_P = np.linalg.eigvals(P)
print(f"幂等矩阵特征值: {np.round(eig_P, 4)}")  # [0, 1]

# 正交矩阵: |λ| = 1
Q = np.array([[np.cos(1), -np.sin(1)], [np.sin(1), np.cos(1)]])
eig_Q = np.linalg.eigvals(Q)
print(f"正交矩阵特征值模长: {np.abs(eig_Q)}")  # [1, 1]
```
