02 - 矩阵对角化

一、矩阵对角化的定义 核心目标

若存在可逆矩阵 P 使得：

P⁻¹AP = Λ = diag(λ₁, λ₂, ..., λₙ)

则称 A 是可对角化的。其中 Λ 是对角矩阵，对角线元素为 A 的特征值。

P 和 Λ 的构成：
- P 的各列为 A 的 n 个线性无关的特征向量 p₁, p₂, ..., pₙ
- Λ 的对角线元素为对应的特征值 λ₁, λ₂, ..., λₙ
- P 的列的排列顺序要与 Λ 的对角线元素一一对应

二、可对角化的条件 必考

A 可对角化的充要条件（以下等价）：

1. A 有 n 个线性无关的特征向量
2. 每个特征值的几何重数 = 代数重数
    即 dim(Eλᵢ) = 重数(λᵢ)，对所有 i
3. A 的最小多项式无重根

充分条件

充分条件1：A 有 n 个互不相同的特征值 → A 可对角化
充分条件2：A 是实对称矩阵 → A 可对角化（且可正交对角化）
常见错误："有 n 个不同特征值" 是充分不必要条件。即使有重特征值，只要几何重数等于代数重数，仍然可对角化。

三、对角化的步骤

Step 1：求特征值：解 |λI - A| = 0，得到 λ₁, λ₂, ..., λₖ
Step 2：对每个 λᵢ，求特征向量：解 (λᵢI - A)x = 0，得到基础解系
Step 3：检验：检查线性无关的特征向量总数是否为 n
Step 4：构造：P = [p₁ | p₂ | ... | pₙ]，Λ = diag(λ₁, λ₂, ..., λₙ)
【例题】对角化 A = 4 -2 1 1

特征多项式：(λ-3)(λ-2) = 0，λ₁=3, λ₂=2
λ₁=3: p₁ = (2,1)ᵀ
λ₂=2: p₂ = (1,1)ᵀ

P = 2 1 1 1 , P⁻¹AP = 3 0 0 2

四、对角化的应用

矩阵幂的快速计算

Aⁿ = PΛⁿP⁻¹ = P · diag(λ₁ⁿ, λ₂ⁿ, ..., λₙⁿ) · P⁻¹

矩阵函数

f(A) = Pf(Λ)P⁻¹ = P · diag(f(λ₁), f(λ₂), ..., f(λₙ)) · P⁻¹
【示例】计算 A¹⁰⁰
A = PΛP⁻¹ → A¹⁰⁰ = PΛ¹⁰⁰P⁻¹
只需对对角矩阵求100次幂，再做两次矩阵乘法。

线性递推关系

xₙ₊₁ = Axₙ → xₙ = Aⁿx₀ = PΛⁿP⁻¹x₀

五、正交对角化（实对称矩阵）

实对称矩阵 A = Aᵀ 一定可以正交对角化：

Q⁻¹AQ = QᵀAQ = Λ

其中 Q 是正交矩阵（QᵀQ = I），即特征向量经过正交规范化（Schmidt正交化 + 单位化）。

实对称矩阵的特殊性质：
1. 所有特征值都是实数
2. 不同特征值对应的特征向量相互正交
3. 每个特征值的几何重数 = 代数重数（必可对角化）
4. A² = O ⟹ A = O（对称矩阵没有非零的幂零矩阵）
