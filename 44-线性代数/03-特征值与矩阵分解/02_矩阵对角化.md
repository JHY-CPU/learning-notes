# 矩阵对角化


## 一、矩阵对角化的定义 核心目标


若存在可逆矩阵 P 使得：


$$
P⁻¹AP = Λ = diag(λ₁, λ₂, ..., λₙ)
$$


则称 A 是**可对角化**的。其中 Λ 是对角矩阵，对角线元素为 A 的特征值。

**P 和 Λ 的构成：**


- P 的各列为 A 的 n 个线性无关的特征向量 p₁, p₂, ..., pₙ


- Λ 的对角线元素为对应的特征值 λ₁, λ₂, ..., λₙ


- P 的列的排列顺序要与 Λ 的对角线元素一一对应

## 二、可对角化的条件 必考


$$
A 可对角化的充要条件（以下等价）：
        1. A 有 n 个线性无关的特征向量
        2. 每个特征值的几何重数 = 代数重数
            即 dim(Eλᵢ) = 重数(λᵢ)，对所有 i
        3. A 的最小多项式无重根
$$


### 充分条件

**充分条件1：**
A 有 n 个
**互不相同**
的特征值 → A 可对角化


**充分条件2：**
A 是实对称矩阵 → A 可对角化（且可正交对角化）

> **Warning:** **常见错误：**
> "有 n 个不同特征值" 是充分不必要条件。即使有重特征值，只要几何重数等于代数重数，仍然可对角化。


## 三、对角化的步骤

**Step 1：**
求特征值：解 |λI - A| = 0，得到 λ₁, λ₂, ..., λₖ


**Step 2：**
对每个 λᵢ，求特征向量：解 (λᵢI - A)x = 0，得到基础解系


**Step 3：**
检验：检查线性无关的特征向量总数是否为 n


**Step 4：**
构造：P = [p₁ | p₂ | ... | pₙ]，Λ = diag(λ₁, λ₂, ..., λₙ)

## 四、对角化的应用


### 矩阵幂的快速计算


$$
Aⁿ = PΛⁿP⁻¹ = P · diag(λ₁ⁿ, λ₂ⁿ, ..., λₙⁿ) · P⁻¹
$$


### 矩阵函数


$$
f(A) = Pf(Λ)P⁻¹ = P · diag(f(λ₁), f(λ₂), ..., f(λₙ)) · P⁻¹
$$


> **Example:** **【示例】**
> 计算 A¹⁰⁰
>
>
> A = PΛP⁻¹ → A¹⁰⁰ = PΛ¹⁰⁰P⁻¹
>
>
> 只需对对角矩阵求100次幂，再做两次矩阵乘法。


### 线性递推关系


$$
xₙ₊₁ = Axₙ → xₙ = Aⁿx₀ = PΛⁿP⁻¹x₀
$$


## 五、正交对角化（实对称矩阵）


实对称矩阵 A = Aᵀ 一定可以**正交对角化**：


$$
Q⁻¹AQ = QᵀAQ = Λ
$$


其中 Q 是正交矩阵（QᵀQ = I），即特征向量经过正交规范化（Schmidt正交化 + 单位化）。

**实对称矩阵的特殊性质：**


1. 所有特征值都是实数


2. 不同特征值对应的特征向量相互正交


3. 每个特征值的几何重数 = 代数重数（必可对角化）


4. A² = O ⟹ A = O（对称矩阵没有非零的幂零矩阵）

## 六、相似矩阵总结


A 与 B 相似（A ~ B）：存在可逆 P 使得 B = P⁻¹AP

**相似等价于：**
同一个线性变换在不同基下的矩阵表示

| 不变量（A ~ B 则相等） | 可变量 |
| --- | --- |
| 特征多项式 \|λI-A\| | 矩阵元素本身 |
| 特征值（含重数） |
| 行列式 \|A\| |
| 迹 tr(A) |
| 秩 rank(A) |


## Python实现

### 矩阵对角化

```python
import numpy as np

A = np.array([[4, -2], [1, 1]])

# 特征分解
eigenvalues, P = np.linalg.eig(A)
print(f"特征值: {eigenvalues}")    # [3, 2]
print(f"P =\n{P}")

# 对角化验证: P^(-1) A P = Λ
P_inv = np.linalg.inv(P)
Lambda = P_inv @ A @ P
print(f"P^(-1) A P = Λ =\n{np.round(Lambda, 4)}")  # 对角矩阵

# 验证 A = P Λ P^(-1)
A_reconstructed = P @ Lambda @ P_inv
print(f"A 重构: {np.allclose(A, A_reconstructed)}")
```

### 矩阵幂的快速计算

```python
import numpy as np

A = np.array([[4, -2], [1, 1]])
eigenvalues, P = np.linalg.eig(A)
P_inv = np.linalg.inv(P)

# 方法1: 利用对角化 A^n = P Λ^n P^(-1)
n = 100
Lambda_n = np.diag(eigenvalues ** n)
A_n = P @ Lambda_n @ P_inv
print(f"A^100 (对角化法) =\n{np.round(A_n, 2)}")

# 方法2: 直接用 numpy
A_n_direct = np.linalg.matrix_power(A, n)
print(f"A^100 (直接法) =\n{np.round(A_n_direct, 2)}")
print(f"两种方法一致: {np.allclose(A_n, A_n_direct)}")
```

### 正交对角化 (实对称矩阵)

```python
import numpy as np

# 实对称矩阵一定可以正交对角化
A = np.array([[2, 1], [1, 2]])
eigenvalues, Q = np.linalg.eigh(A)  # eigh 专用于对称矩阵

print(f"特征值: {eigenvalues}")
print(f"正交矩阵 Q =\n{np.round(Q, 4)}")

# 验证 Q 正交: Q^T Q = I
print(f"Q^T Q = I? {np.allclose(Q.T @ Q, np.eye(2))}")

# 验证 Q^T A Q = Λ
Lambda = Q.T @ A @ Q
print(f"Q^T A Q =\n{np.round(Lambda, 4)}")

# 应用: 用对角化解线性递推 x_{n+1} = A x_n
x0 = np.array([1, 0])
A_power = P @ np.diag(eigenvalues ** 10) @ P_inv
x10 = A_power @ x0
print(f"x_10 = A^10 x_0 = {np.round(x10, 4)}")
```
