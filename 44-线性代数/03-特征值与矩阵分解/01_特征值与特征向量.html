01 - 特征值与特征向量

一、特征值与特征向量的定义 核心概念

设 A 是 n 阶方阵，若存在数 λ 和非零向量 x 使得：

Ax = λx

则称 λ 是 A 的特征值，x 是对应于 λ 的特征向量。

几何意义：特征向量在变换后方向不变（可能反向），只发生伸缩。伸缩倍数就是特征值。

特征多项式

f(λ) = |λI - A| = λⁿ - tr(A)λⁿ⁻¹ + ... + (-1)ⁿ|A|

特征多项式的根就是特征值。

特征方程

|λI - A| = 0

二、求解特征值与特征向量的步骤

Step 1：计算特征多项式 |λI - A| = 0，解出特征值 λ₁, λ₂, ..., λₖ
Step 2：对每个特征值 λᵢ，求齐次方程组 (λᵢI - A)x = 0 的基础解系
Step 3：基础解系的非零线性组合就是对应于 λᵢ 的所有特征向量
【例题】求 A = 3 1 1 3 的特征值与特征向量。

Step 1：
|λI - A| = λ-3 -1 -1 λ-3 = (λ-3)² - 1 = λ² - 6λ + 8 = (λ-4)(λ-2) = 0
特征值：λ₁ = 4, λ₂ = 2

Step 2a（λ₁ = 4）：
(4I - A)x = 0 → 1 -1 -1 1 x = 0
x₁ = x₂，特征向量：x = k(1,1)ᵀ（k ≠ 0）

Step 2b（λ₂ = 2）：
(2I - A)x = 0 → x₁ = -x₂，特征向量：x = k(1,-1)ᵀ（k ≠ 0）

三、特征值的重要性质 必记

性质说明
特征值之和 = 迹 λ₁ + λ₂ + ... + λₙ = tr(A) = a₁₁ + a₂₂ + ... + aₙₙ
特征值之积 = 行列式 λ₁ × λ₂ × ... × λₙ = |A|
A 可逆 ⟺ 所有 λᵢ ≠ 0 |A| ≠ 0 ⟺ 无零特征值
A 和 Aᵀ 有相同的特征值 |λI - A| = |λI - Aᵀ|
不同特征值的特征向量线性无关 重要结论

特征值的运算性质

若 A 的特征值为 λ，特征向量为 x：

- A² 的特征值为 λ²，特征向量仍为 x
- A⁻¹ 的特征值为 1/λ（A 可逆时）
- A + kI 的特征值为 λ + k
- kA 的特征值为 kλ
- f(A) 的特征值为 f(λ)（f 为多项式）
- A*（伴随矩阵）的特征值为 |A|/λ

四、特征子空间（特征空间）

对应于特征值 λ 的所有特征向量加上零向量构成的子空间：

Eλ = Ker(λI - A) = {x | (λI - A)x = 0}

Eλ 的维数称为特征值 λ 的几何重数（geometric multiplicity）。

代数重数 vs 几何重数：
- 代数重数：λ 作为特征多项式根的重数
- 几何重数：dim(Eλ) = n - rank(λI - A)
- 几何重数 ≤ 代数重数（等号成立时可对角化）
注意：一个特征值对应的特征向量有无穷多个（非零标量倍数），但特征子空间的维数有限。

五、特殊矩阵的特征值

矩阵类型特征值特点
对称矩阵 A = Aᵀ所有特征值为实数
正交矩阵 QᵀQ = I|λ| = 1（在单位圆上）
幂等矩阵 A² = A特征值为 0 或 1
对合矩阵 A² = I特征值为 1 或 -1
幂零矩阵 Aᵏ = 0所有特征值为 0
上/下三角矩阵特征值 = 对角线元素
