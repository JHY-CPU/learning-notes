# 基与维数


## 一、基（Basis）的定义 核心概念


向量空间 V 的一个**基**是 V 中满足以下两个条件的向量组 α₁, α₂, ..., αₙ：


$$
1. α₁, α₂, ..., αₙ 线性无关
        2. Span{α₁, α₂, ..., αₙ} = V（生成整个空间）
$$

**基的等价刻画：**


V 中 n 个向量是基 ⟺ V 中每个向量都能被这 n 个向量
**唯一地**
线性表示

### 标准基


Rⁿ 的标准基：


$$
e₁ = (1, 0, ..., 0), e₂ = (0, 1, ..., 0), ..., eₙ = (0, 0, ..., 1)
$$


## 二、维数（Dimension）


向量空间 V 的**维数** dim(V) 是 V 中基所含向量的个数。

**维数的基本性质：**


- 同一个空间中所有基包含相同个数的向量


- dim(Rⁿ) = n


- dim(M_{m×n}) = mn


- dim(Pₙ) = n + 1（基为 1, x, x², ..., xⁿ）


- 零空间 {0} 的维数为 0

> **Warning:** **重要结论：**
> 若 dim(V) = n，则 V 中任意 n 个线性无关的向量构成一组基；任意 n 个能生成 V 的向量也构成一组基。


## 三、坐标与坐标变换


### 坐标


设 {α₁, α₂, ..., αₙ} 是 V 的一组基，向量 β 的坐标定义为满足 β = x₁α₁ + x₂α₂ + ... + xₙαₙ 的系数：


$$
[β]ₛ = (x₁, x₂, ..., xₙ)ᵀ
$$


### 基变换与过渡矩阵


设旧基 A = {α₁, ..., αₙ}，新基 B = {β₁, ..., βₙ}，则：


$$
(β₁, ..., βₙ) = (α₁, ..., αₙ) · P
$$


P 称为从基 A 到基 B 的**过渡矩阵**（基变换矩阵）。


### 坐标变换公式


$$
[x]ₛ = P · [x]ₐ   （新坐标 = 过渡矩阵 × 旧坐标）
$$


## 四、秩-零化度定理 核心定理


设 A 是 m×n 矩阵，则：


$$
rank(A) + nullity(A) = n
        即：dim(Col(A)) + dim(N(A)) = n
$$


其中 nullity(A) = dim(N(A)) = dim({x | Ax = 0}) 是零空间的维数（自由变量的个数）。


> **Example:** **【示例】**
> A 是 3×5 矩阵，rank(A) = 2
>
>
> nullity(A) = 5 - 2 = 3
>
>
> 说明 Ax = 0 的解空间是 3 维的（有 3 个自由变量）


### 秩的性质

- 0 ≤ rank(A) ≤ min(m, n)


- rank(A) = rank(Aᵀ)


- rank(A) = A 的行秩 = A 的列秩


- rank(AB) ≤ min(rank(A), rank(B))


- rank(A + B) ≤ rank(A) + rank(B)


- rank(A) = r ⟺ A 有 r 个线性无关的行/列向量

## 五、子空间的和与直和


### 子空间的和


$$
W₁ + W₂ = {w₁ + w₂ | w₁ ∈ W₁, w₂ ∈ W₂}
$$


### 维数公式


$$
dim(W₁ + W₂) = dim(W₁) + dim(W₂) - dim(W₁ ∩ W₂)
$$


### 直和


若 W₁ ∩ W₂ = {0}，则 W₁ + W₂ 称为**直和**，记作 W₁ ⊕ W₂。

**直和的等价条件：**


- W₁ ∩ W₂ = {0}


- dim(W₁ + W₂) = dim(W₁) + dim(W₂)


- W₁ + W₂ 中每个向量的分解 w₁ + w₂ 唯一


## Python实现

### 秩-零化度定理验证与基变换

```python
import numpy as np

# 秩-零化度定理: rank(A) + nullity(A) = n
A = np.array([[1, 2, 3, 4, 5],
              [2, 4, 6, 8, 10],
              [1, 3, 5, 7, 9]], dtype=float)

n = A.shape[1]  # 列数
rank_A = np.linalg.matrix_rank(A)

# 计算零空间维数
from scipy.linalg import null_space
ns = null_space(A)
nullity_A = ns.shape[1]

print(f"矩阵 A: {A.shape}")
print(f"rank(A) = {rank_A}")
print(f"nullity(A) = {nullity_A}")
print(f"rank + nullity = {rank_A + nullity_A}, n = {n}")
print(f"验证 rank + nullity = n: {rank_A + nullity_A == n}")
```

### 基变换与坐标变换

```python
import numpy as np

# 旧基: 标准基
# 新基: v1 = (1,1), v2 = (1,-1)
# 过渡矩阵 P: 新基在旧基下的坐标作为列
P = np.array([[1, 1], [1, -1]], dtype=float)

# 向量 x 在旧基下的坐标
x_old = np.array([3, 1], dtype=float)

# 坐标变换: x_new = P^(-1) * x_old
P_inv = np.linalg.inv(P)
x_new = P_inv @ x_old
print(f"旧基坐标: {x_old}")
print(f"新基坐标: {x_new}")

# 验证: 用新基坐标还原向量
v1, v2 = np.array([1, 1]), np.array([1, -1])
x_verify = x_new[0] * v1 + x_new[1] * v2
print(f"验证还原: {x_verify}")  # 应等于 [3, 1]

# 求多项式空间 P_2 中的坐标变换
# 旧基: {1, x, x^2}, 新基: {1, 1+x, (1+x)^2}
from sympy import symbols, expand
x_sym = symbols('x')
new_basis = [1, 1 + x_sym, (1 + x_sym)**2]
print("新基展开:")
for b in new_basis:
    print(f"  {expand(b)}")
```

### 子空间的和与直和

```python
import numpy as np

# 子空间 W1 = Span{(1,0,0), (0,1,0)} (xy平面)
# 子空间 W2 = Span{(0,0,1)} (z轴)
W1 = np.array([[1, 0, 0], [0, 1, 0]], dtype=float).T
W2 = np.array([[0, 0, 1]], dtype=float).T

# W1 + W2 的基 = 合并后取极大无关组
combined = np.column_stack([W1, W2])
rank_sum = np.linalg.matrix_rank(combined)
print(f"dim(W1) = {np.linalg.matrix_rank(W1)}")
print(f"dim(W2) = {np.linalg.matrix_rank(W2)}")
print(f"dim(W1 + W2) = {rank_sum}")
print(f"dim(W1) + dim(W2) = {np.linalg.matrix_rank(W1) + np.linalg.matrix_rank(W2)}")

# 验证直和: dim(W1+W2) = dim(W1) + dim(W2) 当且仅当 W1 ∩ W2 = {0}
is_direct_sum = rank_sum == np.linalg.matrix_rank(W1) + np.linalg.matrix_rank(W2)
print(f"是否为直和: {is_direct_sum}")
```
