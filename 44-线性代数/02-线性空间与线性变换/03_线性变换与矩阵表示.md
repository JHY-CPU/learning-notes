# 线性变换与矩阵表示


## 一、线性变换的定义 核心概念


设 T: V → W 是向量空间之间的映射，若满足：


$$
1. 加法保持：T(α + β) = T(α) + T(β)
        2. 数乘保持：T(kα) = kT(α)
$$


则称 T 为从 V 到 W 的**线性变换**（线性映射）。

**等价条件：**
T 是线性变换 ⟺ T(k₁α₁ + k₂α₂) = k₁T(α₁) + k₂T(α₂)


（线性变换把线性组合映射为线性组合）

### 线性变换的基本性质

- T(0) = 0（零向量映射为零向量）


- T(-α) = -T(α)


- 线性相关的向量组的像仍然线性相关


- 线性无关的向量组的像不一定线性无关

## 二、核（Kernel）与像（Image） 重要概念


### 核（零空间）


$$
Ker(T) = {α ∈ V | T(α) = 0}
$$


核是 V 的子空间。dim(Ker(T)) 称为 T 的**零化度**（nullity）。


### 像（值域）


$$
Im(T) = {T(α) | α ∈ V} = T(V)
$$


像是 W 的子空间。dim(Im(T)) 称为 T 的**秩**（rank）。


### 秩-零化度定理


$$
rank(T) + nullity(T) = dim(V)
        dim(Im(T)) + dim(Ker(T)) = dim(V)
$$

**单射与满射的判别：**


- T 是单射 ⟺ Ker(T) = {0} ⟺ nullity(T) = 0


- T 是满射 ⟺ Im(T) = W ⟺ rank(T) = dim(W)


- T 是双射（同构）⟺ T 既是单射又是满射 ⟺ dim(V) = dim(W) 且 T 可逆

## 三、线性变换的矩阵表示 核心方法


### 定义


设 T: V → V 是线性变换，{ε₁, ε₂, ..., εₙ} 是 V 的一组基。若：


$$
T(εⱼ) = a₁ⱼε₁ + a₂ⱼε₂ + ... + aₙⱼεₙ   （j = 1, 2, ..., n）
$$


则矩阵 A = (aᵢⱼ)ₙₓₙ 称为 T 在基 {ε₁, ..., εₙ} 下的**矩阵表示**。

**构造方法：**
将 T(ε₁), T(ε₂), ..., T(εₙ) 在基下的坐标作为矩阵 A 的
**各列**
。


即 A 的第 j 列 = [T(εⱼ)]ₛ（T(εⱼ) 在基 S 下的坐标）。

### 坐标公式


$$
[T(α)]ₛ = A · [α]ₛ
$$


即：变换后的坐标 = 矩阵表示 × 原始坐标。

## 四、基变换与相似矩阵


### 相似变换


同一线性变换在不同基下的矩阵表示**相似**：


$$
B = P⁻¹AP
$$


其中 P 是从旧基到新基的过渡矩阵，A 是旧基下的矩阵，B 是新基下的矩阵。

**相似矩阵的不变量（相似不变量）：**


- 秩：rank(A) = rank(B)


- 行列式：|A| = |B|


- 迹：tr(A) = tr(B)


- 特征多项式：|λI - A| = |λI - B|


- 特征值（重数也相同）

## 五、常见线性变换

| 恒等变换 | T(α) = α | 单位矩阵 I |
| 数乘变换 | T(α) = kα | kI（数量矩阵） |
| 旋转变换(θ) | 逆时针旋转 θ | cosθ
-sinθ
sinθ
cosθ |
| 投影变换 | 投影到子空间 | 幂等矩阵（A² = A） |
| 镜像变换 | 关于子空间的镜像 | 对合矩阵（A² = I） |

## 六、线性变换的运算

**运算定义：**


- 加法：(T₁ + T₂)(α) = T₁(α) + T₂(α)


- 数乘：(kT)(α) = k · T(α)


- 复合（乘法）：(T₁T₂)(α) = T₁(T₂(α))


- 逆变换：T⁻¹(T(α)) = T(T⁻¹(α)) = α
**运算对应的矩阵运算：**


设 T₁ 对应矩阵 A，T₂ 对应矩阵 B，则：


- T₁ + T₂ 对应 A + B


- kT₁ 对应 kA


- T₁T₂ 对应 AB


- T⁻¹ 对应 A⁻¹


## Python实现

### 线性变换的矩阵表示

```python
import numpy as np

# 定义线性变换 T: R^2 -> R^2, T(x,y) = (2x+y, x-y)
# 在标准基下的矩阵表示
def T(v):
    return np.array([2*v[0] + v[1], v[0] - v[1]])

# 求矩阵表示: T(e1) 和 T(e2) 作为列
e1 = np.array([1, 0])
e2 = np.array([0, 1])
col1 = T(e1)  # [2, 1]
col2 = T(e2)  # [1, -1]
A = np.column_stack([col1, col2])
print(f"矩阵表示 A =\n{A}")

# 验证: T(v) = A @ v
v = np.array([3, 5])
print(f"T({v}) = {T(v)}")
print(f"A @ {v} = {A @ v}")

# 核 (Kernel) 的求解: T(x) = 0 即 Ax = 0
from scipy.linalg import null_space
kernel_basis = null_space(A)
print(f"Ker(T) 的基:\n{kernel_basis}")
print(f"nullity(T) = {kernel_basis.shape[1]}")
print(f"rank(T) = {np.linalg.matrix_rank(A)}")
print(f"rank + nullity = {np.linalg.matrix_rank(A) + kernel_basis.shape[1]}")  # = dim(V) = 2
```

### 相似变换验证

```python
import numpy as np

# 验证相似矩阵的不变量
A = np.array([[4, -2], [1, 1]])
P = np.array([[2, 1], [1, 1]], dtype=float)
P_inv = np.linalg.inv(P)
B = P_inv @ A @ P  # B ~ A

print(f"A =\n{A}")
print(f"B = P^(-1)AP =\n{np.round(B, 4)}")

# 相似不变量
print(f"tr(A) = {np.trace(A)}, tr(B) = {np.trace(B)}")         # 迹相同
print(f"|A| = {np.linalg.det(A):.1f}, |B| = {np.linalg.det(B):.1f}")  # 行列式相同
print(f"rank(A) = {np.linalg.matrix_rank(A)}, rank(B) = {np.linalg.matrix_rank(B)}")
```

### 旋转变换的矩阵

```python
import numpy as np

theta = np.pi / 4  # 45度
R = np.array([[np.cos(theta), -np.sin(theta)],
              [np.sin(theta),  np.cos(theta)]])
print(f"旋转矩阵 R(45°) =\n{np.round(R, 4)}")

# 验证正交性: R^T R = I
print(f"R^T R = I? {np.allclose(R.T @ R, np.eye(2))}")

# 旋转向量
v = np.array([1, 0])
v_rotated = R @ v
print(f"v = {v}, 旋转后 = {np.round(v_rotated, 4)}")  # (cos45, sin45)

# 连续旋转: R^4 = 旋转360度 = I
R_360 = np.linalg.matrix_power(R, 4)
print(f"R^4 = I? {np.allclose(R_360, np.eye(2))}")
```
