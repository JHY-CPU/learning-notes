# 向量空间与子空间


## 一、向量空间（线性空间）的定义 核心公理


设 V 是一个非空集合，其元素称为**向量**，定义了两种运算：


- **加法**
   ：V × V → V
- **标量乘法**
   ：F × V → V（F 为数域）


### 八大公理


| 类别 | 公理 |
| --- | --- |
| 加法交换律 | α + β = β + α |
| 加法结合律 | (α + β) + γ = α + (β + γ) |
| 零元素 | 存在 0 ∈ V，使 α + 0 = α |
| 负元素 | 对每个 α，存在 -α 使 α + (-α) = 0 |
| 标量单位元 | 1 · α = α |
| 标量结合律 | k(lα) = (kl)α |
| 分配律(标量) | (k + l)α = kα + lα |
| 分配律(向量) | k(α + β) = kα + kβ |

**常见向量空间：**


- Rⁿ：所有 n 维实向量构成的空间


- C[a,b]：区间 [a,b] 上的连续函数空间


- Pₙ：次数不超过 n 的多项式空间


- M_{m×n}：m×n 实矩阵空间

## 二、线性组合与生成空间（张成空间）


### 线性组合


向量 β 是向量组 α₁, α₂, ..., αₛ 的**线性组合**，若存在标量 k₁, k₂, ..., kₛ 使得：


$$
β = k₁α₁ + k₂α₂ + ... + kₛαₛ = Σ kᵢαᵢ
$$


### 生成空间（张成空间）


向量组 α₁, α₂, ..., αₛ 的所有线性组合构成的集合：


$$
Span{α₁, α₂, ..., αₛ} = {Σ kᵢαᵢ | kᵢ ∈ F}
$$


生成空间一定是向量空间（子空间）。


> **Example:** **【示例】**
> R³ 中的 e₁ = (1,0,0), e₂ = (0,1,0), e₃ = (0,0,1)
>
>
> Span{e₁, e₂, e₃} = R³（整个三维空间）
>
>
> Span{e₁, e₂} = xy 平面（二维子空间）


## 三、线性相关与线性无关 重要概念


### 定义

**线性相关：**
存在不全为零的标量 k₁, k₂, ..., kₛ，使得 k₁α₁ + k₂α₂ + ... + kₛαₛ = 0


**线性无关：**
只有当 k₁ = k₂ = ... = kₛ = 0 时，才有 k₁α₁ + k₂α₂ + ... + kₛαₛ = 0

### 判别方法

1.
**n 个 n 维向量：**
组成矩阵 A，若 |A| ≠ 0 则线性无关，|A| = 0 则线性相关


2.
**含零向量：**
一定线性相关


3.
**含相等/成比例向量：**
一定线性相关


4.
**向量个数 > 维数：**
一定线性相关


5.
**部分相关则整体相关，整体无关则部分无关**

> **Example:** **【示例】**
> 判断 α₁=(1,2,3), α₂=(2,3,4), α₃=(3,5,7) 的线性相关性
>
>
>
>
> 构造矩阵 A =
>
> 1
> 2
> 3
> 2
> 3
> 5
> 3
> 4
> 7
>
>
>
> det(A) = 1(21-20) - 2(14-15) + 3(8-9) = 1 + 2 - 3 =
> **0**
>
>
> 因此 α₁, α₂, α₃
> **线性相关**
> 。
>
>
> 验证：α₃ = α₁ + α₂


## 四、子空间


### 定义


设 W 是向量空间 V 的非空子集，若 W 对 V 中的加法和标量乘法也构成向量空间，则 W 是 V 的**子空间**。


### 子空间判定定理


$$
W 是 V 的子空间 ⟺
        1. 0 ∈ W（非空，含零向量）
        2. α, β ∈ W ⟹ α + β ∈ W（对加法封闭）
        3. α ∈ W, k ∈ F ⟹ kα ∈ W（对标量乘法封闭）
$$


### 常见子空间


| 名称 | 定义 | 维数 |
| --- | --- | --- |
| 列空间 Col(A) | A 的列向量的所有线性组合 | rank(A) |
| 行空间 Row(A) | A 的行向量的所有线性组合 | rank(A) |
| 零空间 N(A) | {x \| Ax = 0} | n - rank(A) |
| 左零空间 N(Aᵀ) | {y \| Aᵀy = 0} | m - rank(A) |


> **Warning:** **注意：**
> 两个子空间的并集 W₁ ∪ W₂ 一般
> **不是**
> 子空间（除非其中一个包含另一个）。但 W₁ + W₂ 和 W₁ ∩ W₂ 都是子空间。


## 五、矩阵的四个基本子空间 核心定理


对于 m×n 矩阵 A（rank(A) = r）：


$$
正交补关系：
        Col(A) ⊥ N(Aᵀ)   （列空间 ⊥ 左零空间）
        Row(A) ⊥ N(A)   （行空间 ⊥ 零空间）
维数关系：
        dim(Col(A)) + dim(N(A)) = n
        dim(Row(A)) + dim(N(Aᵀ)) = m
        dim(Col(A)) = dim(Row(A)) = r
$$


## Python实现

### 线性相关性判断与四个基本子空间

```python
import numpy as np

# 判断向量组的线性相关性
def check_linear_independence(vectors):
    """通过矩阵的秩判断向量组是否线性无关"""
    A = np.column_stack(vectors)
    rank = np.linalg.matrix_rank(A)
    n = len(vectors)
    if rank == n:
        print(f"秩 = {rank}, 向量个数 = {n}: 线性无关")
    else:
        print(f"秩 = {rank}, 向量个数 = {n}: 线性相关")
    return rank == n

# 示例: 判断 (1,2,3), (2,3,4), (3,5,7) 的线性相关性
v1 = np.array([1, 2, 3])
v2 = np.array([2, 3, 4])
v3 = np.array([3, 5, 7])
check_linear_independence([v1, v2, v3])  # 线性相关: rank=2 < 3

# 线性无关的向量组
u1 = np.array([1, 0, 0])
u2 = np.array([0, 1, 0])
u3 = np.array([0, 0, 1])
check_linear_independence([u1, u2, u3])  # 线性无关

# 求四个基本子空间
A = np.array([[1, 2, 3, 4],
              [2, 4, 6, 8],
              [1, 3, 5, 7]], dtype=float)

print(f"矩阵 A 的形状: {A.shape}")
print(f"rank(A) = {np.linalg.matrix_rank(A)}")

# 列空间 (Column Space) - 通过 SVD 找基
U, S, Vt = np.linalg.svd(A)
rank = np.linalg.matrix_rank(A)
col_space_basis = U[:, :rank]
print(f"列空间的维数: {rank}")

# 零空间 (Null Space) - Ax=0 的解空间
from scipy.linalg import null_space
ns = null_space(A)
print(f"零空间的维数: {ns.shape[1]}")
if ns.shape[1] > 0:
    print(f"零空间的一组基:\n{ns}")

# 用 sympy 求精确的零空间基
from sympy import Matrix
M = Matrix([[1, 2, 3, 4], [2, 4, 6, 8], [1, 3, 5, 7]])
nullspace = M.nullspace()
print(f"精确零空间基: {nullspace}")
```

### 生成空间的验证

```python
import numpy as np

# 验证 Span{e1, e2} = xy平面
e1 = np.array([1, 0, 0])
e2 = np.array([0, 1, 0])

# R3中的任意向量 (a, b, 0) 在 Span{e1, e2} 中
# 用最小二乘验证投影
for v in [np.array([3, 5, 0]), np.array([1, 2, 3])]:
    E = np.column_stack([e1, e2])
    coeffs, residuals, _, _ = np.linalg.lstsq(E, v, rcond=None)
    projection = E @ coeffs
    residual = np.linalg.norm(v - projection)
    print(f"v = {v}, 投影 = {projection}, 残差 = {residual:.4f}")
    # 残差为0说明在span中
```
