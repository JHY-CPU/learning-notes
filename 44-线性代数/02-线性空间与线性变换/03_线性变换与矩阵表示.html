03 - 线性变换与矩阵表示

一、线性变换的定义 核心概念

设 T: V → W 是向量空间之间的映射，若满足：

1. 加法保持：T(α + β) = T(α) + T(β)
2. 数乘保持：T(kα) = kT(α)

则称 T 为从 V 到 W 的线性变换（线性映射）。

等价条件：T 是线性变换 ⟺ T(k₁α₁ + k₂α₂) = k₁T(α₁) + k₂T(α₂)
（线性变换把线性组合映射为线性组合）

线性变换的基本性质

- T(0) = 0（零向量映射为零向量）
- T(-α) = -T(α)
- 线性相关的向量组的像仍然线性相关
- 线性无关的向量组的像不一定线性无关

二、核（Kernel）与像（Image） 重要概念

核（零空间）

Ker(T) = {α ∈ V | T(α) = 0}

核是 V 的子空间。dim(Ker(T)) 称为 T 的零化度（nullity）。

像（值域）

Im(T) = {T(α) | α ∈ V} = T(V)

像是 W 的子空间。dim(Im(T)) 称为 T 的秩（rank）。

秩-零化度定理

rank(T) + nullity(T) = dim(V)
dim(Im(T)) + dim(Ker(T)) = dim(V)
单射与满射的判别：
- T 是单射 ⟺ Ker(T) = {0} ⟺ nullity(T) = 0
- T 是满射 ⟺ Im(T) = W ⟺ rank(T) = dim(W)
- T 是双射（同构）⟺ T 既是单射又是满射 ⟺ dim(V) = dim(W) 且 T 可逆

三、线性变换的矩阵表示 核心方法

定义

设 T: V → V 是线性变换，{ε₁, ε₂, ..., εₙ} 是 V 的一组基。若：

T(εⱼ) = a₁ⱼε₁ + a₂ⱼε₂ + ... + aₙⱼεₙ   （j = 1, 2, ..., n）

则矩阵 A = (aᵢⱼ)ₙₓₙ 称为 T 在基 {ε₁, ..., εₙ} 下的矩阵表示。

构造方法：将 T(ε₁), T(ε₂), ..., T(εₙ) 在基下的坐标作为矩阵 A 的各列。
即 A 的第 j 列 = [T(εⱼ)]ₛ（T(εⱼ) 在基 S 下的坐标）。

坐标公式

[T(α)]ₛ = A · [α]ₛ

即：变换后的坐标 = 矩阵表示 × 原始坐标。

【示例】设 T: R² → R²，T(x, y) = (2x + y, x - y)，在标准基下：

T(e₁) = T(1,0) = (2,1) → 第一列为 (2,1)ᵀ
T(e₂) = T(0,1) = (1,-1) → 第二列为 (1,-1)ᵀ

矩阵表示：A = 2 1 1 -1

四、基变换与相似矩阵

相似变换

同一线性变换在不同基下的矩阵表示相似：

B = P⁻¹AP

其中 P 是从旧基到新基的过渡矩阵，A 是旧基下的矩阵，B 是新基下的矩阵。

相似矩阵的不变量（相似不变量）：
- 秩：rank(A) = rank(B)
- 行列式：|A| = |B|
- 迹：tr(A) = tr(B)
- 特征多项式：|λI - A| = |λI - B|
- 特征值（重数也相同）

五、常见线性变换

名称定义矩阵表示（标准基）
恒等变换 T(α) = α 单位矩阵 I
数乘变换 T(α) = kα kI（数量矩阵）
旋转变换(θ) 逆时针旋转 θ cosθ -sinθ sinθ cosθ
投影变换 投影到子空间 幂等矩阵（A² = A）
镜像变换 关于子空间的镜像 对合矩阵（A² = I）
微分算子 D D(f) = f' 依赖基的选择
注意：微分算子在多项式空间 Pₙ 中是线性变换，但在整个函数空间上可能不是（不满足有界性）。

六、线性变换的运算

运算定义：
- 加法：(T₁ + T₂)(α) = T₁(α) + T₂(α)
- 数乘：(kT)(α) = k · T(α)
- 复合（乘法）：(T₁T₂)(α) = T₁(T₂(α))
- 逆变换：T⁻¹(T(α)) = T(T⁻¹(α)) = α
运算对应的矩阵运算：
设 T₁ 对应矩阵 A，T₂ 对应矩阵 B，则：
- T₁ + T₂ 对应 A + B
- kT₁ 对应 kA
- T₁T₂ 对应 AB
- T⁻¹ 对应 A⁻¹
