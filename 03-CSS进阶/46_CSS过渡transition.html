1. transition 简写

transition: all 0.5s ease 0s;
hover
所有属性0.5s过渡，ease曲线，无延迟

2. transition-property 指定属性

transition-property: background-color; /* width不过渡 */
hover看宽度

width 瞬间变化，background-color 平滑过渡

3. transition-duration 不同持续时间

transition-duration: 0.3s, 0.6s, 1s;
hover

4. transition-timing-function 速度曲线

ease（默认）
ease
linear
linear
ease-in
in
ease-out
out
ease-in-out
inout
悬停任意轨道，所有球同时开始移动，观察速度差异

5. transition-delay 延迟

transition: all 0.5s ease 0.3s;
延迟0.3s

6. 多属性分别设置

transition:
 width 0.5s ease,
 background-color 1s ease-in-out 0.2s,
 border-radius 2s;
hover
