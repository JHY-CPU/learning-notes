1. calc() — 数学计算

calc(100% - 40px)

calc() 支持 + - * /，可混合不同单位（px, %, vw等）

calc((100% - 40px) / 3)
calc((100% - 40px) / 3)
calc((100% - 40px) / 3)
⚠️ 运算符两侧必须有空格：calc(100% - 20px) ✓

2. min() — 取最小值

min(48px, 8vw) 响应式字号
min(400px, 100%) — 不超过400px

min(A, B) = A和B中较小的值，常用于限制最大宽度

3. max() — 取最大值

max(300px, 50%) — 至少300px

max(A, B) = A和B中较大的值，常用于保证最小尺寸

4. clamp() — 范围限制

clamp(16px, 3.5vw, 32px) — 响应式字号，安全范围
clamp(200px, 50%, 600px)

clamp(MIN, 首选值, MAX) — 首选值在MIN和MAX之间
