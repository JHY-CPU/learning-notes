1. 基础动画 — 来回移动

animation: move-right 2s ease infinite alternate;
move

2. 动画方向与循环

animation-direction: normal; ← 正向播放（默认）
animation-direction: reverse; ← 反向播放
animation-direction: alternate; ← 来回播放
→
←
↔

3. 弹跳动画 — 多关键帧

@keyframes bounce {
 0% { transform: translateY(0); }
 30% { transform: translateY(-60px); }
 50% { transform: translateY(-30px); }
 70% { transform: translateY(-60px); }
 100% { transform: translateY(0); }
}
bounce

4. animation-fill-mode 填充模式

animation-fill-mode: forwards; /* 停在最后一帧 */

延迟1秒开始，2秒动画结束后停在终点

forwards
none=默认不填充 | forwards=停最后一帧 | backwards=延迟期用第一帧 | both=同时forwards+backwards

5. animation-play-state 控制播放/暂停

animation-play-state: paused; /* 暂停动画 */
spin

6. steps() 步进动画

animation: step-move 2s steps(4) infinite alternate;
step
steps(4) 将动画分成4步，无平滑过渡，类似帧动画效果

7. 多动画叠加

animation:
 move-right 3s ease infinite alternate,
 fade-in 2s ease,
 pulse 1.5s ease infinite;
multi
