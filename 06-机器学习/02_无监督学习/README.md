# 无监督学习

## 一、聚类

### 1.1 K-Means

迭代将数据点分配到最近的聚类中心，然后更新中心。

**算法流程：**
1. 随机初始化K个聚类中心
2. 将每个点分配到最近的中心
3. 更新每个簇的中心（均值）
4. 重复2-3直到收敛

**评估指标：**
- 轮廓系数（Silhouette Score）
- Calinski-Harabasz指数
- 肘部法则（Elbow Method）选择K

### 1.2 DBSCAN

基于密度的聚类，能发现任意形状的簇，自动识别噪声。

- 核心参数：$\epsilon$（邻域半径）、MinPts（最小点数）
- 优势：不需要预先指定簇数量
- 劣势：对密度不均匀的数据效果差

### 1.3 层次聚类

- 凝聚式：自底向上合并
- 分裂式：自顶向下分裂
- 可视化：树状图（Dendrogram）

## 二、降维

### 2.1 PCA（主成分分析）

找到数据方差最大的方向，将数据投影到低维空间。

**步骤：**
1. 数据中心化
2. 计算协方差矩阵
3. 特征值分解
4. 选择前k个特征向量

**方差解释率**：$\frac{\sum_{i=1}^{k}\lambda_i}{\sum_{i=1}^{d}\lambda_i}$

### 2.2 t-SNE

非线性降维，擅长可视化高维数据。

- 保留局部结构
- 适合2D/3D可视化
- 计算复杂度高，不适合大规模数据

### 2.3 UMAP

t-SNE的改进版，更快且更好保留全局结构。

## 三、密度估计

### 3.1 高斯混合模型（GMM）

$$p(x) = \sum_{k=1}^{K} \pi_k \mathcal{N}(x|\mu_k, \Sigma_k)$$

- 使用EM算法求解
- 软聚类：输出属于各簇的概率
- 比K-Means更灵活（椭圆形簇）
