# 机器学习

本模块系统整理机器学习的核心理论与实践，从经典算法到深度学习前沿。

## 学习路线

```
监督/无监督学习 → 集成学习 → 特征工程 → 模型评估
        ↓
  深度学习基础 → CNN → RNN → Transformer → NLP
        ↓
  迁移学习 → 强化学习 → 大模型时代
```
