# 特征工程

## 一、数据预处理

### 1.1 数值特征

- **标准化**：$z = (x - \mu) / \sigma$，适合大多数算法
- **归一化**：$x' = (x - x_{min}) / (x_{max} - x_{min})$，适合神经网络
- **对数变换**：$x' = \log(1+x)$，处理长尾分布
- **Box-Cox变换**：使数据接近正态分布

### 1.2 类别特征

- **Label Encoding**：整数编码（有序类别）
- **One-Hot Encoding**：每个类别一个二值列
- **Target Encoding**：用目标变量均值编码（注意数据泄漏）
- **Frequency Encoding**：用类别频率编码

### 1.3 缺失值处理

- 删除：缺失比例高时考虑删除列
- 均值/中位数/众数填充
- KNN填充
- 模型预测填充
- 将缺失值作为单独类别

## 二、特征构造

### 2.1 交互特征

- 乘积特征：$x_1 \times x_2$
- 比值特征：$x_1 / x_2$
- 多项式特征：sklearn PolynomialFeatures

### 2.2 时间特征

- 年、月、日、星期几、小时
- 是否周末、是否节假日
- 距特定日期的距离

### 2.3 文本特征

- TF-IDF
- 文本长度、词数
- 词嵌入平均

---

## 三、特征选择

### 3.1 过滤法

- 方差阈值
- 相关系数
- 卡方检验
- 互信息

### 3.2 包裹法

- 递归特征消除（RFE）
- 前向/后向选择

### 3.3 嵌入法

- L1正则化自动选特征
- 树模型特征重要性
- Permutation Importance

---

## 四、数据泄漏

常见泄漏来源：
- 使用未来信息（时间序列）
- 测试集信息混入训练集
- Target Encoding时使用了验证集信息
- 预处理时使用了全量数据的统计量

**原则**：所有变换只能用训练集的信息。
