# 循环神经网络

## 一、RNN

### 1.1 基本结构

$$h_t = \tanh(W_{xh} x_t + W_{hh} h_{t-1} + b_h)$$

隐藏状态 $h_t$ 编码了历史信息。

### 1.2 问题

- **梯度消失**：长序列中梯度指数衰减
- **梯度爆炸**：梯度指数增长
- 解决方法：梯度裁剪、LSTM/GRU

---

## 二、LSTM

引入三个门和细胞状态：

- **遗忘门**：$f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$
- **输入门**：$i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$
- **候选细胞**：$\tilde{C}_t = \tanh(W_c [h_{t-1}, x_t] + b_c)$
- **细胞状态**：$C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t$
- **输出门**：$o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$
- **隐藏状态**：$h_t = o_t \odot \tanh(C_t)$

关键：细胞状态的线性传播使梯度可以长距离流动。

---

## 三、GRU

LSTM的简化版，两个门：

- **更新门**：$z_t = \sigma(W_z [h_{t-1}, x_t])$
- **重置门**：$r_t = \sigma(W_r [h_{t-1}, x_t])$
- **候选状态**：$\tilde{h}_t = \tanh(W [r_t \odot h_{t-1}, x_t])$
- **隐藏状态**：$h_t = (1-z_t) \odot h_{t-1} + z_t \odot \tilde{h}_t$

参数更少，训练更快，效果与LSTM接近。

---

## 四、Seq2Seq

编码器-解码器结构：
- **编码器**：RNN将输入序列编码为固定长度向量
- **解码器**：RNN从编码向量生成输出序列

**问题**：信息瓶颈——编码向量难以保留长序列的全部信息。

**解决方案**：注意力机制让解码器在每一步都能访问编码器的所有隐藏状态。
