# 深度学习基础

## 一、神经网络基础

### 1.1 感知机

最基本的神经元模型：$y = f(\sum w_i x_i + b)$

单层感知机只能解决线性可分问题（如AND、OR），无法解决XOR问题。

### 1.2 多层感知机（MLP）

通过增加隐藏层和非线性激活函数，MLP可以拟合任意连续函数（万能近似定理）。

前向传播：
$$z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$$
$$a^{(l)} = f(z^{(l)})$$

---

## 二、反向传播

### 2.1 链式法则

反向传播的核心是链式法则，逐层计算损失对参数的梯度。

### 2.2 自动微分

现代框架（PyTorch、TensorFlow）使用计算图自动求导：
- **前向模式**：适合输入维度少
- **反向模式**：适合输出维度少（深度学习常用）

---

## 三、激活函数

- **Sigmoid**：$\sigma(x) = 1/(1+e^{-x})$，梯度消失问题
- **ReLU**：$\max(0, x)$，可能神经元死亡
- **Leaky ReLU**：$\max(0.01x, x)$，缓解死亡问题
- **GELU**：$x\Phi(x)$，Transformer常用
- **Swish**：$x\sigma(\beta x)$

---

## 四、优化器

| 算法 | 特点 |
|------|------|
| SGD | 基础，需调学习率 |
| SGD + Momentum | 加速收敛 |
| Adam | 自适应学习率，最常用 |
| AdamW | 解耦权重衰减 |
| LAMB | 大batch训练 |

---

## 五、正则化

- **Dropout**：随机置零，防止共适应
- **BatchNorm**：标准化激活值
- **LayerNorm**：Transformer常用
- **权重衰减**：L2正则化
- **早停**：监控验证损失
