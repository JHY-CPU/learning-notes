# 监督学习

## 一、线性模型

### 1.1 线性回归

$$\hat{y} = w^T x + b$$

**损失函数**：最小二乘法
$$L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

**闭式解**：$w^* = (X^TX)^{-1}X^Ty$

**正则化变体：**
- Ridge（L2）：$L + \lambda \|w\|_2^2$
- LASSO（L1）：$L + \lambda \|w\|_1$（产生稀疏解）
- ElasticNet：同时使用L1和L2

### 1.2 逻辑回归

用于二分类，输出概率：
$$P(y=1|x) = \sigma(w^Tx + b) = \frac{1}{1 + e^{-(w^Tx + b)}}$$

损失函数：交叉熵
$$L = -\frac{1}{n}\sum [y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]$$

## 二、支持向量机（SVM）

寻找最大间隔超平面：
$$\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^Tx_i + b) \geq 1$$

- **软间隔**：引入松弛变量 $\xi_i$
- **核技巧**：通过核函数隐式映射到高维空间
  - 线性核：$K(x,y) = x^Ty$
  - RBF核：$K(x,y) = \exp(-\gamma\|x-y\|^2)$
  - 多项式核：$K(x,y) = (x^Ty + c)^d$

## 三、决策树

### 3.1 划分准则

- **信息增益（ID3）**：$Gain(D, a) = H(D) - \sum \frac{|D_v|}{|D|} H(D_v)$
- **增益率（C4.5）**：信息增益 / 属性熵
- **基尼指数（CART）**：$Gini(D) = 1 - \sum p_k^2$

### 3.2 剪枝

- 预剪枝：限制深度、叶节点最小样本数
- 后剪枝：先生成完整树，再自底向上剪枝

---

## 四、K近邻（KNN）

根据K个最近邻的标签进行预测。

- 距离度量：欧氏距离、曼哈顿距离、闵可夫斯基距离
- K值选择：K小易过拟合，K大易欠拟合
- 无训练过程（懒学习），预测时计算量大
