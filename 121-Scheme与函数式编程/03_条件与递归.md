# 条件与递归

## 条件表达式

Scheme 提供了多种条件表达式，其中 `cond` 是最通用和最常用的。

### if 表达式

`if` 是最基本的条件形式，只能有两个分支：

```scheme
;; if 语法: (if 条件 真分支 假分支)

;; 基本用法
(if (> 3 2)
    "3 大于 2"
    "3 不大于 2")
; => "3 大于 2"

;; if 可以省略假分支（返回未指定值）
(if #t
    (display "条件为真\n"))

;; 嵌套 if（不推荐，用 cond 更清晰）
(if (> x 0)
    "正数"
    (if (= x 0)
        "零"
        "负数"))

;; 用 and/or 实现短路求值
(define (safe-divide a b)
  (and (not (= b 0)) (/ a b)))

(safe-divide 10 3)   ; => 10/3
(safe-divide 10 0)   ; => #f
```

### cond 表达式

`cond` 是 Scheme 中处理多分支条件的首选方式：

```scheme
;; cond 语法: (cond [条件1 表达式1] [条件2 表达式2] ... [else 默认])

;; 数字分类
(define (number-type n)
  (cond
    [(positive? n) "正数"]
    [(negative? n) "负数"]
    [else          "零"]))

(number-type 42)   ; => "正数"
(number-type -7)   ; => "负数"
(number-type 0)    ; => "零"

;; cond 可以有多条表达式在条件之后
(define (grade score)
  (cond
    [(>= score 90) (display "优秀\n") "A"]
    [(>= score 80) (display "良好\n") "B"]
    [(>= score 70) (display "中等\n") "C"]
    [(>= score 60) (display "及格\n") "D"]
    [else          (display "不及格\n") "F"]))

;; => 语法（R5RS+）: 条件 => 函数，将条件的值传给函数
(cond
  [(assq 'b '((a 1) (b 2) (c 3))) => cadr]
  [else #f])
; => 2
```

### case 表达式

`case` 基于值的匹配进行分支，类似其他语言的 switch：

```scheme
;; case 语法: (case 表达式 [(值1 值2 ...) 结果] ... [else 默认])

(define (day-name n)
  (case n
    [(1)    "星期一"]
    [(2)    "星期二"]
    [(3)    "星期三"]
    [(4)    "星期四"]
    [(5)    "星期五"]
    [(6 7)  "周末"]
    [else   "无效"]))

(day-name 3)    ; => "星期三"
(day-name 6)    ; => "周末"

;; case 使用 eqv? 进行比较，适合符号和整数
(define (describe x)
  (case x
    [(apple)    "苹果"]
    [(banana)   "香蕉"]
    [(cherry)   "樱桃"]
    [else       "未知水果"]))
```

## 递归

递归是 Scheme 中实现循环的核心机制。Scheme 要求实现**尾调用优化**（Tail Call Optimization, TCO），使得递归在空间上等价于迭代。

### 线性递归

线性递归是最直观的递归形式，但调用栈随输入增长：

```scheme
;; 阶乘的线性递归
(define (factorial n)
  (if (<= n 1)
      1
      (* n (factorial (- n 1)))))

;; 求值过程：展开和收缩
;; (factorial 5)
;; (* 5 (factorial 4))
;; (* 5 (* 4 (factorial 3)))
;; (* 5 (* 4 (* 3 (factorial 2))))
;; (* 5 (* 4 (* 3 (* 2 (factorial 1)))))
;; (* 5 (* 4 (* 3 (* 2 1))))
;; (* 5 (* 4 (* 3 2)))
;; (* 5 (* 4 6))
;; (* 5 24)
;; 120

;; 斐波那契数列（指数时间复杂度）
(define (fib n)
  (if (< n 2)
      n
      (+ (fib (- n 1)) (fib (- n 2)))))
```

### 尾递归

尾递归是 Scheme 编程的核心技巧。尾调用是指函数的最后一个操作就是调用自身（或另一个函数），没有后续计算。

```scheme
;; 尾递归阶乘：使用累加器
(define (factorial-tail n [acc 1])
  (if (<= n 1)
      acc
      (factorial-tail (- n 1) (* n acc))))

;; 求值过程：不需要展开
;; (factorial-tail 5 1)
;; (factorial-tail 4 5)
;; (factorial-tail 3 20)
;; (factorial-tail 2 60)
;; (factorial-tail 1 120)
;; 120

;; 尾递归斐波那契
(define (fib-tail n [a 0] [b 1])
  (if (= n 0)
      a
      (fib-tail (- n 1) b (+ a b))))

(fib-tail 10)  ; => 55

;; 列表求和的尾递归版本
(define (sum-list lst [acc 0])
  (if (null? lst)
      acc
      (sum-list (cdr lst) (+ acc (car lst)))))

(sum-list '(1 2 3 4 5))  ; => 15

;; 使用命名 let 的惯用写法
(define (factorial n)
  (let loop ([i n] [acc 1])
    (if (<= i 1)
        acc
        (loop (- i 1) (* i acc)))))
```

### 通过递归实现迭代

Scheme 没有传统的循环结构（for、while），所有循环都通过递归实现：

```scheme
;; 使用递归遍历列表
(define (for-each* proc lst)
  (unless (null? lst)
    (proc (car lst))
    (for-each* proc (cdr lst))))

(for-each* display '("hello" " " "world" "\n"))

;; 使用递归过滤列表
(define (filter* pred lst)
  (cond
    [(null? lst) '()]
    [(pred (car lst))
     (cons (car lst) (filter* pred (cdr lst)))]
    [else
     (filter* pred (cdr lst))]))

(filter* even? '(1 2 3 4 5 6 7 8))  ; => (2 4 6 8)

;; 使用递归构建列表
(define (iota n)
  (let loop ([i 0] [acc '()])
    (if (= i n)
        (reverse acc)
        (loop (+ i 1) (cons i acc)))))

(iota 5)  ; => (0 1 2 3 4)
```

## 绑定表达式

Scheme 提供了三种 `let` 形式来创建局部绑定。

### let

`let` 同时绑定所有变量（并行绑定），绑定表达式中不能引用其他绑定变量：

```scheme
;; let 语法: (let ([变量1 值1] [变量2 值2] ...) 表达式)

(let ([x 10]
      [y 20])
  (+ x y))
; => 30

;; 注意：并行绑定，下面的写法是错误的
;; (let ([x 10]
;;       [y (+ x 1)])   ; 错误！此时 x 还未绑定
;;   y)

;; 命名 let：创建尾递归循环的惯用模式
(define (countdown n)
  (let loop ([n n])
    (when (>= n 0)
      (display n)
      (display " ")
      (loop (- n 1)))))
;; (countdown 5) 输出: 5 4 3 2 1 0
```

### let*

`let*` 按顺序绑定变量（串行绑定），后续绑定可以引用前面的变量：

```scheme
;; let* 语法：绑定按从左到右的顺序求值
(let* ([x 10]
       [y (* x 2)]    ; y 可以引用 x
       [z (+ x y)])   ; z 可以引用 x 和 y
  (+ x y z))
; => 10 + 20 + 30 = 60

;; let* 等价于嵌套的 let
(let ([x 10])
  (let ([y (* x 2)])
    (let ([z (+ x y)])
      (+ x y z))))
```

### letrec

`letrec` 允许绑定之间相互引用，主要用于定义相互递归的函数：

```scheme
;; letrec 语法：所有绑定在所有表达式中可见
(letrec ([even? (lambda (n)
                  (if (= n 0)
                      #t
                      (odd? (- n 1))))]
         [odd?  (lambda (n)
                  (if (= n 0)
                      #f
                      (even? (- n 1))))])
  (even? 10))  ; => #t

;; letrec 常用于定义局部辅助函数
(define (quicksort lst)
  (letrec ([sort (lambda (lst)
                   (if (null? lst)
                       '()
                       (let* ([pivot (car lst)]
                              [rest  (cdr lst)]
                              [less  (filter (lambda (x) (< x pivot)) rest)]
                              [more  (filter (lambda (x) (>= x pivot)) rest)])
                         (append (sort less) (list pivot) (sort more)))))])
    (sort lst)))

(quicksort '(3 1 4 1 5 9 2 6))  ; => (1 1 2 3 4 5 6 9)
```
