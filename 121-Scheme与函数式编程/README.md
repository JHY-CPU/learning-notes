# 121 - Scheme 与函数式编程

本目录包含 Scheme 编程语言及函数式编程范式的系统学习笔记。

## 目录

| 编号 | 文件 | 主题 |
|------|------|------|
| 01 | `01_Scheme语言概述.md` | Scheme 语言历史、标准与开发环境 |
| 02 | `02_基本表达式与求值.md` | 前缀表达式、原子、列表与求值模型 |
| 03 | `03_条件与递归.md` | 条件分支、递归与绑定表达式 |
| 04 | `04_高阶函数.md` | 一等函数、map/filter/fold、柯里化 |
| 05 | `05_闭包与词法作用域.md` | 闭包创建、词法作用域与自由变量 |
| 06 | `06_数据结构.md` | 对偶、列表、关联列表、树与向量 |
| 07 | `07_延续Continuation.md` | call/cc、延续传递风格、逃逸延续 |
| 08 | `08_宏系统.md` | define-syntax、syntax-rules、卫生宏 |
| 09 | `09_SICP核心概念.md` | 流处理、惰性求值、元循环求值器 |
| 10 | `10_惰性求值与流.md` | delay/force、无穷流、素数筛 |
| 11 | `11_Typed_Racket.md` | 类型注解、多态、出现类型 |
| 12 | `12_函数式编程思想总结.md` | 不可变性、引用透明性、与 Haskell 对比 |

## 推荐学习路线

1. **入门阶段（01-03）**：掌握 Scheme 语法基础、求值规则和递归思维
2. **核心阶段（04-06）**：深入高阶函数、闭包机制和数据结构
3. **进阶阶段（07-08）**：理解延续和宏系统，这是 Scheme 的精髓所在
4. **综合阶段（09-12）**：学习 SICP 经典内容、流处理、类型系统及函数式编程哲学
