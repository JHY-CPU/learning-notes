# S3对象存储

## 一、概念说明

S3（Simple Storage Service）是AWS的对象存储服务，提供高可用、高持久性的数据存储。适用于静态网站托管、备份、大数据分析等场景。

| 存储类别 | 访问频率 | 最短保留 | 成本 |
|----------|----------|----------|------|
| Standard | 频繁访问 | 无 | 最高 |
| Standard-IA | 不频繁访问 | 30天 | 中等 |
| Glacier | 归档 | 90天 | 低 |
| Glacier Deep Archive | 长期归档 | 180天 | 最低 |

## 二、具体用法

### 基本操作

```bash
# 创建存储桶
aws s3 mb s3://my-unique-bucket-name --region us-east-1

# 上传文件
aws s3 cp myfile.txt s3://my-bucket/
aws s3 cp ./myfolder s3://my-bucket/myfolder/ --recursive

# 下载文件
aws s3 cp s3://my-bucket/myfile.txt ./

# 列出对象
aws s3 ls s3://my-bucket/
aws s3 ls s3://my-bucket/myfolder/ --recursive

# 删除对象
aws s3 rm s3://my-bucket/myfile.txt
aws s3 rm s3://my-bucket/myfolder/ --recursive
```

### 静态网站托管

```bash
# 配置静态网站托管
aws s3 website s3://my-bucket \
    --index-document index.html \
    --error-document error.html

# 设置存储桶策略允许公开读取
aws s3api put-bucket-policy --bucket my-bucket --policy '{
    "Version": "2012-10-17",
    "Statement": [{
        "Sid": "PublicRead",
        "Effect": "Allow",
        "Principal": "*",
        "Action": "s3:GetObject",
        "Resource": "arn:aws:s3:::my-bucket/*"
    }]
}'
```

### 生命周期规则

```json
{
    "Rules": [
        {
            "ID": "ArchiveRule",
            "Status": "Enabled",
            "Transitions": [
                {
                    "Days": 30,
                    "StorageClass": "STANDARD_IA"
                },
                {
                    "Days": 90,
                    "StorageClass": "GLACIER"
                }
            ],
            "Expiration": {
                "Days": 365
            }
        }
    ]
}
```

### 版本控制和跨区域复制

```bash
# 启用版本控制
aws s3api put-bucket-versioning \
    --bucket my-bucket \
    --versioning-configuration Status=Enabled

# 配置跨区域复制
aws s3api put-bucket-replication \
    --bucket source-bucket \
    --replication-configuration file://replication.json
```

## 三、注意事项与常见陷阱

1. **存储桶命名**：名称全局唯一，仅小写字母、数字和连字符
2. **权限设置**：默认私有，公开访问需显式配置
3. **存储类别选择**：根据访问模式选择合适的存储类别
4. **生命周期管理**：设置自动转换规则降低存储成本
5. **大文件上传**：大于100MB使用分段上传
6. **版本控制费用**：启用版本控制后删除对象不会释放空间
7. **传输加速**：跨地域上传考虑使用Transfer Acceleration
