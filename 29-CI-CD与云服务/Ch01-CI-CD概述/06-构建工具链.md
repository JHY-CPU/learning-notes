# 构建工具链

## 一、概念说明

构建工具将源代码编译、打包成可部署的制品。不同语言有不同的构建工具。

## 二、常用构建工具

```bash
# JavaScript/Node.js
npm install        # 安装依赖
npm run build      # 构建
npm test           # 测试
npm run lint       # 代码检查

# Python
pip install -r requirements.txt
python setup.py build
pytest

# Java/Maven
mvn clean install
mvn test
mvn package

# Go
go build ./...
go test ./...
go mod download
```

## 三、CI集成示例

```yaml
# Node.js构建
- run: npm ci
- run: npm run lint
- run: npm test
- run: npm run build

# Maven构建
- uses: actions/setup-java@v4
  with:
    java-version: '17'
- run: mvn clean install

# Go构建
- uses: actions/setup-go@v5
  with:
    go-version: '1.21'
- run: go build ./...
- run: go test ./...
```

## 四、注意事项

1. **版本锁定**：使用lock文件锁定依赖版本
2. **缓存依赖**：缓存node_modules/.m2等
3. **并行构建**：利用多核CPU并行编译
4. **构建优化**：增量编译，避免全量构建
