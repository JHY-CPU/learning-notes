# 状态徽章

## 一、概念说明

状态徽章显示CI/CD的当前状态，可以嵌入到README中，直观展示构建状态。

## 二、基本徽章

```markdown
![CI](https://github.com/owner/repo/actions/workflows/ci.yml/badge.svg)
```

## 三、带分支的徽章

```markdown
![CI](https://github.com/owner/repo/actions/workflows/ci.yml/badge.svg?branch=main)
```

## 四、其他徽章

```markdown
# 测试覆盖率
[![codecov](https://codecov.io/gh/owner/repo/branch/main/graph/badge.svg)](https://codecov.io/gh/owner/repo)

# npm版本
[![npm](https://img.shields.io/npm/v/package-name)](https://www.npmjs.com/package/package-name)

# 许可证
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
```

## 五、自定义徽章

```markdown
# Shields.io自定义
[![custom](https://img.shields.io/badge/custom-badge-blue)](https://example.com)
```

## 六、注意事项

1. **公共仓库**：徽章只对公共仓库可见
2. **缓存**：徽章有缓存，更新有延迟
3. **分支**：指定正确的分支
4. **风格统一**：保持徽章风格统一
