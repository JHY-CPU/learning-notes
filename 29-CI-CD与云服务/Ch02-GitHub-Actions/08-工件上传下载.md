# 工件上传下载

## 一、概念说明

工件（Artifact）是构建过程中产生的文件，可以在Job之间共享或下载。

## 二、上传工件

```yaml
- uses: actions/upload-artifact@v4
  with:
    name: build-output
    path: |
      dist/
      coverage/
    retention-days: 7
```

## 三、下载工件

```yaml
- uses: actions/download-artifact@v4
  with:
    name: build-output
    path: ./output
```

## 四、跨Job共享

```yaml
jobs:
  build:
    steps:
      - run: npm run build
      - uses: actions/upload-artifact@v4
        with:
          name: app
          path: dist/

  deploy:
    needs: build
    steps:
      - uses: actions/download-artifact@v4
        with:
          name: app
          path: ./app
      - run: ./deploy.sh
```

## 五、注意事项

1. **大小限制**：单个工件最大512MB
2. **保留时间**：默认90天，可设置retention-days
3. **格式**：支持ZIP格式
4. **安全性**：工件可以被有权限的人下载
