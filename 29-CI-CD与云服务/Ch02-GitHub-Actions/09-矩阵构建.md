# 矩阵构建

## 一、概念说明

矩阵构建允许在多个配置组合下并行运行测试，如不同Node.js版本、不同操作系统等。

## 二、基本用法

```yaml
strategy:
  matrix:
    node: [16, 18, 20]
    os: [ubuntu-latest, windows-latest]
runs-on: ${{ matrix.os }}
steps:
  - uses: actions/setup-node@v4
    with:
      node-version: ${{ matrix.node }}
  - run: npm test
```

## 三、排除和包含

```yaml
strategy:
  matrix:
    node: [16, 18, 20]
    os: [ubuntu-latest, windows-latest]
    exclude:
      - os: windows-latest
        node: 16
    include:
      - os: macos-latest
        node: 20
```

## 四、fail-fast

```yaml
strategy:
  fail-fast: false  # 不因一个失败而取消其他
  matrix:
    node: [16, 18, 20]
```

## 五、注意事项

1. **并行数量**：最多256个并行Job
2. **资源消耗**：矩阵构建消耗更多额度
3. **fail-fast**：默认一个失败会取消其他
4. **命名清晰**：给Job起描述性名称
