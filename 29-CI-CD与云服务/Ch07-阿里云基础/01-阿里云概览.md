# 阿里云概览

## 一、概念说明

阿里云（Alibaba Cloud）是阿里巴巴集团旗下的云计算服务，是中国最大的公有云服务商。提供计算、存储、数据库、网络、安全等全栈云服务。

| 服务类别 | AWS对应 | 阿里云服务 |
|----------|---------|-----------|
| 计算 | EC2 | ECS云服务器 |
| 容器 | ECS/EKS | ACK容器服务 |
| 对象存储 | S3 | OSS |
| 数据库 | RDS | RDS云数据库 |
| 负载均衡 | ALB/NLB | SLB负载均衡 |
| DNS | Route 53 | 云解析DNS |
| CDN | CloudFront | CDN加速 |
| 无服务器 | Lambda | 函数计算FC |

## 二、具体用法

### 安装和配置CLI

```bash
# 安装阿里云CLI
pip install aliyun-cli

# 配置账号
aliyun configure
# AccessKey ID
# AccessKey Secret
# Default Region Id
# Default Output Format
# Default Language (zh/en)

# 验证配置
aliyun ecs DescribeRegions
```

### 基本操作

```bash
# 查看区域列表
aliyun ecs DescribeRegions

# 查看可用区
aliyun ecs DescribeZones --RegionId cn-hangzhou

# 创建ECS实例
aliyun ecs RunInstances \
    --RegionId cn-hangzhou \
    --ImageId ubuntu_20_04_x64_20G_alibase_20210420.vhd \
    --InstanceType ecs.t6-c1m1.large \
    --SecurityGroupId sg-bp1xxxxxxxx \
    --VSwitchId vsw-bp1xxxxxxxx \
    --InstanceName my-server \
    --InternetMaxBandwidthOut 5

# 查看实例列表
aliyun ecs DescribeInstances --RegionId cn-hangzhou
```

### RAM用户管理

```bash
# 创建RAM用户
aliyun ram CreateUser \
    --UserName developer \
    --DisplayName "开发人员"

# 创建访问密钥
aliyun ram CreateLoginProfile \
    --UserName developer \
    --Password "StrongPassword123!" \
    --MFABindRequired true

# 授权策略
aliyun ram AttachPolicyToUser \
    --PolicyType System \
    --PolicyName AliyunECSFullAccess \
    --UserName developer
```

## 三、注意事项与常见陷阱

1. **区域选择**：国内用户选择就近区域，考虑备案要求
2. **AccessKey安全**：避免硬编码，使用RAM角色或环境变量
3. **ICP备案**：面向国内用户提供Web服务需要ICP备案
4. **网络计费**：注意公网带宽计费方式（按流量/按带宽）
5. **安全组**：阿里云安全组默认出方向全开，入方向全拒绝
6. **镜像选择**：选择官方镜像或经过安全扫描的自定义镜像
7. **资源释放**：开发测试资源及时释放避免不必要的费用
