# 函数计算FC

## 一、概念说明

函数计算（Function Compute，FC）是阿里云的无服务器计算服务，按需执行代码，无需管理服务器。支持事件触发和HTTP请求触发。

| 概念 | 说明 |
|------|------|
| 服务 | 函数的管理和权限单元 |
| 函数 | 可执行代码单元 |
| 触发器 | 事件触发规则 |
| 版本 | 函数版本快照 |
| 别名 | 版本的逻辑指针 |

### 函数代码

```python
# index.py - HTTP触发器处理函数
import json

def handler(environ, start_response):
    """HTTP触发器入口"""
    path = environ.get('PATH_INFO', '/')

    if path == '/api/health':
        status = '200 OK'
        body = json.dumps({"status": "healthy"})
    elif path == '/api/data':
        status = '200 OK'
        body = json.dumps({"data": [1, 2, 3, 4, 5]})
    else:
        status = '404 Not Found'
        body = json.dumps({"error": "Not found"})

    response_headers = [('Content-Type', 'application/json')]
    start_response(status, response_headers)
    return [body.encode()]
```

### 命令行操作

```bash
# 调用函数
aliyun fc InvokeFunction \
    --serviceName my-service \
    --functionName hello-world \
    --payload '{"name": "test"}'

# 查看函数日志
aliyun fc GetFunctionLogs \
    --serviceName my-service \
    --functionName hello-world \
    --limit 20

# 发布版本
aliyun fc PublishServiceVersion \
    --serviceName my-service \
    --description "v1.0 发布版本"

# 创建别名
aliyun fc CreateAlias \
    --serviceName my-service \
    --aliasName prod \
    --versionId 1 \
    --additionalVersionWeight '{"2": 0.1}'
```

## 三、注意事项与常见陷阱

1. **冷启动**：首次调用或实例回收后会有冷启动延迟
2. **执行超时**：最大执行时间600秒，注意超时设置
3. **包大小限制**：代码包50MB压缩/250MB解压
4. **并发限制**：注意服务级和函数级并发限制
5. **VPC访问**：访问VPC资源需配置网络配置
6. **环境变量**：敏感信息使用加密环境变量
7. **日志配置**：配置日志服务便于问题排查
