# 流水线配置

## 一、Job定义

```yaml
build:
  stage: build
  image: node:20-alpine
  script:
    - npm install
    - npm run build
  artifacts:
    paths:
      - dist/
    expire_in: 1 week
  cache:
    paths:
      - node_modules/
```

## 二、Stage定义

```yaml
stages:
  - build
  - test
  - deploy

build:
  stage: build
  script: echo "Building..."

test:unit:
  stage: test
  script: echo "Unit test..."

test:e2e:
  stage: test
  script: echo "E2E test..."

deploy:
  stage: deploy
  script: echo "Deploying..."
```

## 三、触发条件

```yaml
# 仅main分支
only:
  - main

# 除了main分支
except:
  - main

# 规则（推荐）
rules:
  - if: $CI_COMMIT_BRANCH == "main"
  - if: $CI_PIPELINE_SOURCE == "merge_request_event"
```

## 四、环境变量

```yaml
variables:
  NODE_ENV: production

build:
  variables:
    DEBUG: "false"
  script:
    - echo $NODE_ENV
    - echo $DEBUG
```

## 五、注意事项

1. **依赖关系**：使用needs定义Job依赖
2. **并行执行**：同一Stage的Job并行执行
3. **缓存配置**：缓存依赖加速构建
4. **制品管理**：保留必要的构建制品
