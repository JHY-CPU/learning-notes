# 容器注册表

## 一、概念说明

GitLab Container Registry是GitLab内置的Docker镜像仓库，与CI/CD深度集成。

## 二、构建和推送镜像

```yaml
build-image:
  stage: build
  image: docker:24.0.5
  services:
    - docker:24.0.5-dind
  variables:
    DOCKER_TLS_CERTDIR: "/certs"
  before_script:
    - docker login -u $CI_REGISTRY_USER -p $CI_REGISTRY_PASSWORD $CI_REGISTRY
  script:
    - docker build -t $CI_REGISTRY_IMAGE:$CI_COMMIT_SHA .
    - docker push $CI_REGISTRY_IMAGE:$CI_COMMIT_SHA
    - docker tag $CI_REGISTRY_IMAGE:$CI_COMMIT_SHA $CI_REGISTRY_IMAGE:latest
    - docker push $CI_REGISTRY_IMAGE:latest
```

## 三、使用预构建镜像

```yaml
# 使用GitLab CI模板
build:
  image: docker:24.0.5
  services:
    - docker:24.0.5-dind

# 使用项目镜像
deploy:
  image: $CI_REGISTRY_IMAGE:latest
  script:
    - ./deploy.sh
```


## 五、注意事项

1. **认证**：使用CI/CD变量自动认证
2. **命名规范**：镜像名使用项目路径
3. **标签策略**：使用Git SHA作为标签
4. **存储限制**：注意镜像存储空间
