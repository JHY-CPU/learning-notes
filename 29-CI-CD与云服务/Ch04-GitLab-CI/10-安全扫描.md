## 六、注意事项

1. **扫描配置**：根据项目需要配置扫描类型
2. **误报处理**：标记和管理误报
3. **安全门禁**：严重漏洞阻止合并
4. **定期扫描**：定期运行安全扫描

## 七、完整安全扫描配置

```yaml
include:
  - template: Security/SAST.gitlab-ci.yml
  - template: Security/Secret-Detection.gitlab-ci.yml
  - template: Security/Dependency-Scanning.gitlab-ci.yml
  - template: Security/Container-Scanning.gitlab-ci.yml
  - template: Security/DAST.gitlab-ci.yml

stages:
  - build
  - test
  - security
  - deploy

variables:
  SAST_EXCLUDED_PATHS: "spec, test, tests, tmp, vendor"
  DS_EXCLUDED_PATHS: "test, spec"
  CS_IMAGE: $CI_REGISTRY_IMAGE:$CI_COMMIT_SHA
  DAST_TARGET_URL: "https://staging.example.com"

# 自定义安全门禁
security-gate:
  stage: security
  script:
    - echo "Checking security reports..."
  needs:
    - job: sast
      artifacts: true
    - job: dependency_scanning
      artifacts: true
    - job: container_scanning
      artifacts: true
  allow_failure: false
  rules:
    - if: $CI_COMMIT_BRANCH == "main"
```

## 八、SAST配置详解

```yaml
sast:
  stage: test
  variables:
    SAST_GOSEC_LEVEL: 2
    SAST_SEMGREP_RULESETS: "p/ci"
    SAST_EXCLUDED_PATHS: "docs, vendor, node_modules"
  artifacts:
    reports:
      sast: gl-sast-report.json
  rules:
    - if: $SAST_DISABLED == "true"
      when: never
    - if: $CI_PIPELINE_SOURCE == "merge_request_event"
    - if: $CI_COMMIT_BRANCH == $CI_DEFAULT_BRANCH
```

## 九、License合规扫描

```yaml
# License扫描
include:
  - template: Security/License-Scanning.gitlab-ci.yml

license_scanning:
  variables:
    LICENSE_FINDER_OPTS: "--no-prepare --pip-d44-path requirements.txt"
  artifacts:
    reports:
      license_scanning: gl-license-scanning-report.json

# 自定义许可证门禁
license-gate:
  stage: security
  script:
    - |
      FORBIDDEN_LICENSES="GPL-3.0 AGPL-3.0"
      REPORT=gl-license-scanning-report.json
      for LICENSE in $FORBIDDEN_LICENSES; do
        if jq -e ".licenses[] | select(.id == \"$LICENSE\")" "$REPORT" > /dev/null; then
          echo "ERROR: Found forbidden license: $LICENSE"
          exit 1
        fi
      done
  needs:
    - job: license_scanning
      artifacts: true
```
