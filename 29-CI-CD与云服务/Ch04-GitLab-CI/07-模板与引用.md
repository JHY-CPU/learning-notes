# 模板与引用

## 一、extends继承

```yaml
# 定义模板
.test-template:
  image: node:20-alpine
  cache:
    paths:
      - node_modules/
  before_script:
    - npm ci

# 继承模板
test:unit:
  extends: .test-template
  script:
    - npm run test:unit

test:e2e:
  extends: .test-template
  script:
    - npm run test:e2e
```

## 二、include引入

```yaml
# 引入本地文件
include:
  - local: '/ci/build.yml'
  - local: '/ci/test.yml'

# 引入远程文件
include:
  - remote: 'https://gitlab.com/templates/ci.yml'

# 引入模板
include:
  - template: Auto-DevOps.gitlab-ci.yml
```

## 三、YAML锚点

```yaml
# 定义锚点
.build-template: &build
  stage: build
  image: node:20
  script:
    - npm install
    - npm run build

# 使用锚点
build:linux:
  <<: *build
  tags: [linux]

build:windows:
  <<: *build
  tags: [windows]
```

## 四、注意事项

1. **模板复用**：提取公共配置为模板
2. **版本控制**：远程模板指定版本
3. **覆盖继承**：子Job可以覆盖父配置
4. **文档清晰**：模板应有清晰文档
