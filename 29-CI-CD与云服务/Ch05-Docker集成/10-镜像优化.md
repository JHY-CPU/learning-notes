# 镜像优化

## 一、基础镜像选择

```dockerfile
# 最小镜像选择
# Alpine - 最小，约5MB
FROM alpine:3.18

# Distroless - 无shell，更安全
FROM gcr.io/distroless/nodejs20-debian12

# Ubuntu - 功能完整，较大
FROM ubuntu:22.04
```

## 二、镜像大小优化

```dockerfile
# 1. 使用多阶段构建
FROM node:20-alpine AS builder
# ... 构建阶段

FROM node:20-alpine AS production
# ... 只复制必要文件

# 2. 合并RUN命令
RUN apk add --no-cache git && \
    npm install && \
    apk del git

# 3. 清理缓存
RUN apt-get update && \
    apt-get install -y package && \
    rm -rf /var/lib/apt/lists/*
```

## 三、优化对比

```bash
# 优化前
ubuntu:22.04 + full install → 1.2GB

# 优化后
alpine:3.18 + minimal install → 150MB

# 进一步优化
distroless + 只复制二进制 → 50MB
```

## 四、注意事项

1. **功能兼容**：确保选择的基础镜像功能够用
2. **安全更新**：定期更新基础镜像
3. **调试困难**：distroless镜像调试较困难
4. **测试验证**：优化后充分测试
