# 编排与部署

## 一、Docker Compose部署

```yaml
# 生产环境配置
version: '3.8'
services:
  app:
    image: myregistry/myapp:latest
    deploy:
      replicas: 3
      resources:
        limits:
          cpus: '0.5'
          memory: 512M
    restart: always
  
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    depends_on:
      - app
```

## 二、Docker Swarm

```yaml
# Stack部署
docker stack deploy -c docker-compose.yml myapp

# 服务管理
docker service ls
docker service scale myapp_app=5
```

## 三、CI/CD部署

```yaml
deploy:
  script:
    - ssh deploy@server "cd /app && docker-compose pull && docker-compose up -d"
```

## 四、滚动更新

```yaml
deploy:
  update_config:
    parallelism: 1
    delay: 10s
    order: start-first
```

## 五、注意事项

1. **健康检查**：配置健康检查确保服务就绪
2. **资源限制**：设置CPU和内存限制
3. **日志管理**：配置集中日志收集
4. **监控告警**：部署后监控服务状态
