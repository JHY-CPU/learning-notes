
## 二、安装插件

```bash
# Web界面
# Manage Jenkins > Manage Plugins > Available

# 命令行
java -jar jenkins-cli.jar -s http://localhost:8080/ install-plugin plugin-name

# Docker
# 在Dockerfile中预装
RUN jenkins-plugin-cli --plugins git pipeline docker-workflow
```

## 四、注意事项

1. **定期更新**：保持插件最新
2. **兼容性**：注意插件与Jenkins版本兼容
3. **最小化**：只安装必要的插件
4. **备份**：插件配置纳入备份

## 五、Docker中预装插件

```dockerfile
# 自定义Jenkins镜像，预装插件
FROM jenkins/jenkins:lts-jdk17

# 使用jenkins-plugin-cli安装插件
RUN jenkins-plugin-cli --plugins \
    git \
    workflow-aggregator \
    docker-workflow \
    blueocean \
    credentials-binding \
    slack \
    email-ext \
    sonar \
    jacoco \
    configuration-as-code \
    job-dsl \
    pipeline-stage-view \
    build-timeout \
    timestamper \
    ws-cleanup
```

## 六、插件管理CLI

```bash
# 列出已安装插件
java -jar jenkins-cli.jar -s http://localhost:8080/ list-plugins

# 安装插件
java -jar jenkins-cli.jar -s http://localhost:8080/ install-plugin git workflow-aggregator

# 升级插件
java -jar jenkins-cli.jar -s http://localhost:8080/ install-plugin -restart git

# 卸载插件
java -jar jenkins-cli.jar -s http://localhost:8080/ safe-restart

# 批量安装（从文件）
cat plugins.txt | xargs java -jar jenkins-cli.jar -s http://localhost:8080/ install-plugin
```

```text
# plugins.txt - 插件列表
git:latest
workflow-aggregator:latest
docker-workflow:latest
blueocean:latest
credentials-binding:latest
slack:latest
configuration-as-code:latest
```
