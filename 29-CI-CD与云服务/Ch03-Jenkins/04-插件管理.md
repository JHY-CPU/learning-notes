# 插件管理

## 一、常用插件

```bash
# 必装插件
# Git - Git集成
# Pipeline - 流水线支持
# Blue Ocean - 可视化界面
# Docker Pipeline - Docker集成
# Credentials Binding - 凭据管理

# 推荐插件
# Slack Notification - Slack通知
# Email Extension - 邮件通知
# SonarQube Scanner - 代码质量
# Jacoco - 测试覆盖率
# AnsiColor - 彩色日志
```

## 二、安装插件

```bash
# Web界面
# Manage Jenkins > Manage Plugins > Available

# 命令行
java -jar jenkins-cli.jar -s http://localhost:8080/ install-plugin plugin-name

# Docker
# 在Dockerfile中预装
RUN jenkins-plugin-cli --plugins git pipeline docker-workflow
```

## 三、插件配置

```bash
# 全局配置
# Manage Jenkins > Configure System

# 工具配置
# Manage Jenkins > Global Tool Configuration
# 配置JDK、Maven、Node.js等
```

## 四、注意事项

1. **定期更新**：保持插件最新
2. **兼容性**：注意插件与Jenkins版本兼容
3. **最小化**：只安装必要的插件
4. **备份**：插件配置纳入备份
