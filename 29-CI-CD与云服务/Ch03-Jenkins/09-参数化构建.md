# 参数化构建

## 一、参数类型

```groovy
pipeline {
    parameters {
        string(name: 'VERSION', defaultValue: '1.0.0', description: 'Version')
        choice(name: 'ENV', choices: ['dev', 'staging', 'prod'], description: 'Environment')
        booleanParam(name: 'DEPLOY', defaultValue: false, description: 'Deploy?')
        password(name: 'SECRET', defaultValue: '', description: 'Secret')
    }
    stages {
        stage('Deploy') {
            when { expression { params.DEPLOY } }
            steps {
                sh "./deploy.sh ${params.VERSION} ${params.ENV}"
            }
        }
    }
}
```

## 二、使用参数

```groovy
steps {
    echo "Version: ${params.VERSION}"
    echo "Environment: ${params.ENV}"
}
```

## 三、触发参数化构建

```bash
# URL触发
http://jenkins-url/job/myjob/buildWithParameters?VERSION=1.0.0&ENV=prod

# CLI触发
java -jar jenkins-cli.jar build myjob -p VERSION=1.0.0
```

## 四、注意事项

1. **默认值**：为参数设置合理的默认值
2. **验证**：在Pipeline中验证参数值
3. **安全**：敏感参数使用password类型
4. **文档**：为每个参数添加描述
