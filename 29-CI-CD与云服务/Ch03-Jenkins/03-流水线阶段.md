# 流水线阶段

## 一、阶段定义

```groovy
stages {
    stage('Build') {
        steps {
            sh 'mvn clean package'
        }
    }
    
    stage('Test') {
        parallel {
            stage('Unit Test') {
                steps { sh 'mvn test' }
            }
            stage('Integration Test') {
                steps { sh 'mvn verify' }
            }
        }
    }
    
    stage('Deploy') {
        when { branch 'main' }
        steps { sh './deploy.sh' }
    }
}
```

## 二、条件执行

```groovy
stage('Deploy') {
    when {
        branch 'main'
        expression { params.DEPLOY == 'true' }
        changeRequest()
    }
    steps { sh './deploy.sh' }
}
```

## 三、Post操作

```groovy
post {
    always { echo 'Always runs' }
    success { notifySlack('Success') }
    failure { notifySlack('Failed') }
    unstable { echo 'Tests failed' }
    cleanup { cleanWs() }
}
```

## 四、注意事项

1. **阶段命名**：使用描述性名称
2. **并行阶段**：独立测试并行运行
3. **条件逻辑**：使用when控制执行
4. **清理资源**：post中清理临时文件
