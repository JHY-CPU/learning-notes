# 消息加密

## 一、传输加密

```java
// Kafka SSL 配置
props.put("security.protocol", "SSL");
props.put("ssl.truststore.location", "/path/truststore.jks");
props.put("ssl.truststore.password", "password");
props.put("ssl.keystore.location", "/path/keystore.jks");
props.put("ssl.keystore.password", "password");

// RabbitMQ TLS 配置
ConnectionFactory factory = new ConnectionFactory();
factory.useSslProtocol("TLSv1.3");
factory.setSslContextFactory(() -> {
    SSLContext ctx = SSLContext.getInstance("TLSv1.3");
    ctx.init(null, trustAllCerts, new SecureRandom());
    return ctx;
});
```

## 二、消息体加密

```java
// 应用层加密
public class EncryptedProducer {
    @Autowired
    private KafkaTemplate<String, String> kafkaTemplate;

    public void sendEncrypted(String topic, String key, Object data) {
        String json = JSON.toJSONString(data);
        String encrypted = AESUtil.encrypt(json, secretKey);
        kafkaTemplate.send(topic, key, encrypted);
    }
}

public class EncryptedConsumer {
    @KafkaListener(topics = "encrypted-topic")
    public void consume(ConsumerRecord<String, String> record) {
        String decrypted = AESUtil.decrypt(record.value(), secretKey);
        Order order = JSON.parseObject(decrypted, Order.class);
        process(order);
    }
}
```

## 三、注意事项

1. **传输加密用 SSL/TLS**，消息体加密用 AES/RSA
2. **密钥管理要用专业的密钥服务（KMS）**
3. **加密会增加 CPU 开销**，大消息影响更明显
4. **Kafka 支持 SASL 认证 + SSL 加密**
5. **RabbitMQ 支持 TLS + OAuth2 认证**
