# 消息优先级

## 一、RabbitMQ 优先级队列

```java
// 声明优先级队列
@Bean
public Queue priorityQueue() {
    return QueueBuilder.durable("priority-queue")
        .withArgument("x-max-priority", 10)
        .build();
}

// 发送优先级消息
rabbitTemplate.convertAndSend("exchange", "key", msg, m -> {
    m.getMessageProperties().setPriority(10);  // 高优先级
    return m;
});
```

## 二、RocketMQ 优先级

```java
// RocketMQ 不原生支持优先级
// 方案: 使用多个 Topic 或 Tag 模拟

// VIP 订单 -> 高优先级 Topic
rocketMQTemplate.syncSend("OrderTopic:VIP", order);
// 普通订单 -> 低优先级 Topic
rocketMQTemplate.syncSend("OrderTopic:NORMAL", order);
```

## 三、Kafka 优先级

```java
// Kafka 不原生支持优先级
// 方案: 多 Topic + 优先消费

// 高优先级 Topic 优先消费
@KafkaListener(topics = "high-priority-topic", groupId = "priority-group")
public void consumeHigh(ConsumerRecord<String, String> record) {
    process(record);
}
```

## 四、注意事项

1. **只有 RabbitMQ 原生支持优先级**
2. **优先级队列有性能开销**
3. **x-max-priority 建议不超过 10**
4. **多 Topic 方案更适合 Kafka 和 RocketMQ**
5. **优先级只在队列积压时有意义**
