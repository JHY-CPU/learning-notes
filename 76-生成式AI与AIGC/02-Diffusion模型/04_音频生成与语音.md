# 音频生成与语音

*VALL-E、Bark、TTS 技术演进、语音克隆、音乐生成与 AudioLDM*


### 1.1 TTS 发展阶段


| 阶段 | 技术 | 特点 | 代表 |
| --- | --- | --- | --- |
| 拼接合成 | 录音片段拼接 | 自然但不灵活 | 早期语音助手 |
| 参数合成 | 声码器+参数模型 | 灵活但音质一般 | Tacotron |

### 2.1 核心思想


VALL-E（微软，2023）将语音合成为语言建模任务：用文本和 3 秒音频提示作为条件，自回归地预测音频的离散 token 序列。


### 2.2 音频离散化：EnCodec


VALL-E 使用 EnCodec（Meta）将连续音频波形编码为离散 token 序列：

音频波形
(24kHz)
→
EnCodec 编码器
卷积+残差向量量化
→
离散 token
(多层 RVQ)

- **RVQ（残差向量量化）**
   ：将连续向量量化为 8 层离散 codebook 索引
- **码率**
   ：24kHz 音频 → 每秒 75 个 token（每层），8 层共 600 tokens/s
- **层间关系**
   ：第一层编码主要信息，后续层补充细节


### 2.3 模型架构


```
# VALL-E 架构
# 基于 GPT-2 风格的自回归 Transformer

# 输入序列构造：
# [文本token] [分隔符] [音频提示token] [预测目标token]

# 训练：
# 1. 使用 EnCodec 将音频编码为离散 token
# 2. 文本 + 音频 token 序列作为自回归模型的输入
# 3. 预测下一层的音频 token（逐层生成）

# 推理（零样本语音克隆）：
# 输入：目标文本 + 3秒参考音频
# 1. 编码参考音频为 token
# 2. 文本 + 参考音频 token → 自回归生成目标音频 token
# 3. EnCodec 解码器将 token 还原为波形

# 核心创新：
# - 零样本：只需 3 秒参考音频，无需微调即可克隆声音
# - 多样性：同一文本+参考音频可以生成不同音色的变体
# - 韵律保持：保留参考音频的说话风格和韵律
```


### 3.1 Bark（Suno AI）


Bark 是 Suno 开源的**文本到音频**模型，不仅能生成语音，还能生成音乐、音效和非语言声音。


| 特性 | 说明 |
| --- | --- |
| 三阶段生成 | 文本→语义 token →音频 token →波形 |
| 多模态输出 | 语音、音乐、笑声、叹息、音效 |
| 多语言 | 支持 100+ 语言自动检测 |
| 声音克隆 | 支持预设音色或自定义音色 |

### 4.2 MusicGen 架构


```
# MusicGen（Meta）架构
# 基于 EnCodec 的音频离散化 + 自回归 Transformer

# 1. 音频编码：EnCodec 将音乐编码为多层 RVQ token
# 2. 文本编码：T5 编码文本描述
# 3. 条件生成：Transformer 自回归预测音频 token
# 4. 音频解码：EnCodec 解码器还原波形

# 关键创新：延迟模式（Delay Pattern）
# 多层 RVQ 的 token 不是并行生成，而是交错延迟生成
# 第0层: [t0, t1, t2, t3, ...]
# 第1层: [_, t0, t1, t2, ...]  (延迟1步)
# 第2层: [_, _, t0, t1, ...]  (延迟2步)
# 这种模式让高层的生成可以参考低层信息
```


### 5.1 核心思想


AudioLDM 受 Latent Diffusion 启发，在音频的潜空间中做扩散，实现高质量的文本到音频/音乐生成。


### 5.2 架构

文本输入
→
CLAP 文本编码器
→
潜空间扩散
去噪生成
→
VAE 解码
音频

- **CLAP 编码器**
   ：对比学习的音频-文本对齐模型（类似 CLIP 之于图像）
- **音频 VAE**
   ：将梅尔频谱图压缩到潜空间
- **潜空间扩散**
   ：在压缩后的潜空间中做 DDPM 扩散


### 5.3 AudioLDM 2 改进


- 使用 GPT-2 作为语义 token 生成器，再用扩散模型生成细节
- 支持音频、音乐、语音三种模态统一生成
- 引入 LLM 理解复杂文本指令

### 6.2 跨语言语音克隆


跨语言克隆需要将说话人的音色特征与语言内容解耦。主要方法：


- **内容-说话人分离：**
   分别编码内容（语言无关）和说话人特征
- **多语言预训练：**
   在多语言数据上预训练，学习语言无关的音色表示
- **音色迁移：**
   将源说话人的音色特征注入目标语言的合成过程
