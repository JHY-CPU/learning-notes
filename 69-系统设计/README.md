# 67 - 系统设计

本模块涵盖大规模分布式系统设计的核心知识，分为高可用架构、可扩展架构、典型设计案例三大部分。

## 目录

### [01-高可用架构](./01-高可用架构/README.md)

- 高可用的定义与量化（SLA、可用性等级）
- 冗余设计（主从复制、主主复制、双活多活）
- 故障转移（Failover）、VIP漂移、脑裂问题
- 负载均衡（四层/七层、算法、会话保持）
- 容错设计（重试、熔断、降级、限流、超时控制）
- 数据备份与恢复（RPO/RTO、灾难恢复计划）

### [02-可扩展架构](./02-可扩展架构/README.md)

- 可扩展性的维度（垂直扩展 vs 水平扩展）
- 数据库扩展（读写分离、分库分表、在线扩容）
- 缓存策略（多级缓存、穿透/击穿/雪崩、一致性方案）
- 消息队列（异步解耦、削峰填谷、顺序保证、幂等性）
- CDN与边缘计算
- 微服务拆分原则
- 数据最终一致性方案（事件驱动、Event Sourcing、CDC）
