# 文本输入绑定

## 一、概念说明

`v-model`在文本输入框上创建双向绑定，输入值自动同步到data。支持trim、lazy等修饰符。

```vue
<template>
  <div>
    <input v-model="name" placeholder="请输入姓名" />
    <p>你输入了: {{ name }}</p>

    <!-- trim自动去除首尾空格 -->
    <input v-model.trim="trimmedName" placeholder="自动去空格" />

    <!-- 输入框类型 -->
    <input v-model="email" type="email" placeholder="邮箱" />
    <input v-model="password" type="password" placeholder="密码" />
  </div>
</template>

<script setup>
import { ref } from 'vue'

const name = ref('')
const trimmedName = ref('')
const email = ref('')
const password = ref('')
</script>
```

## 二、具体用法

### 多个输入框绑定对象

```vue
<script setup>
import { reactive } from 'vue'

const form = reactive({
  firstName: '',
  lastName: '',
  email: '',
  phone: ''
})
</script>

<template>
  <input v-model="form.firstName" placeholder="姓" />
  <input v-model="form.lastName" placeholder="名" />
  <input v-model="form.email" type="email" />
  <input v-model="form.phone" type="tel" />
</template>
```

### 实时格式化

```vue
<script setup>
import { ref, watch } from 'vue'

const phone = ref('')

watch(phone, (val) => {
  // 只允许数字
  phone.value = val.replace(/\D/g, '').slice(0, 11)
})
</script>

<template>
  <input v-model="phone" type="tel" placeholder="手机号" />
</template>
```

## 三、注意事项与常见陷阱

1. `v-model`默认使用`input`事件（实时更新）
2. 使用`.lazy`变为`change`事件（失焦更新）
3. `ref`值在模板中自动解包，不需要`.value`
4. `<input type="file">`不能用`v-model`，需用`@change`
5. 输入框值始终是字符串类型
