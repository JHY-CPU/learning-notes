# 选择框绑定

## 一、概念说明

`<select>`使用`v-model`绑定选中值。单选时绑定字符串，多选时绑定数组（需加`multiple`属性）。

```vue
<template>
  <!-- 单选下拉 -->
  <select v-model="city">
    <option disabled value="">请选择城市</option>
    <option value="beijing">北京</option>
    <option value="shanghai">上海</option>
    <option value="guangzhou">广州</option>
  </select>
  <p>选择的城市: {{ city }}</p>

  <!-- 多选下拉 -->
  <select v-model="skills" multiple>
    <option value="vue">Vue</option>
    <option value="react">React</option>
    <option value="angular">Angular</option>
  </select>
  <p>选择的技能: {{ skills }}</p>
</template>

<script setup>
import { ref } from 'vue'

const city = ref('')
const skills = ref([])
</script>
```

## 二、具体用法

### 动态选项

```vue
<template>
  <select v-model="selectedCategory">
    <option value="">全部分类</option>
    <option v-for="cat in categories" :key="cat.id" :value="cat.id">
      {{ cat.name }}
    </option>
  </select>
</template>

<script setup>
import { ref } from 'vue'

const categories = [
  { id: 1, name: '电子产品' },
  { id: 2, name: '服装' },
  { id: 3, name: '食品' }
]
const selectedCategory = ref('')
</script>
```

### 对象值绑定

```vue
<script setup>
const selectedCity = ref(null)
const cities = [
  { id: 1, name: '北京', code: '010' },
  { id: 2, name: '上海', code: '021' }
]
</script>

<template>
  <select v-model="selectedCity">
    <option :value="null" disabled>请选择</option>
    <option v-for="city in cities" :key="city.id" :value="city">
      {{ city.name }}
    </option>
  </select>
</template>
```

## 四、级联选择

```vue
<script setup>
import { ref, watch } from 'vue'

const provinces = [
  { id: 1, name: '北京', cities: ['朝阳区', '海淀区', '东城区'] },
  { id: 2, name: '上海', cities: ['浦东新区', '黄浦区', '静安区'] }
]

const selectedProvince = ref('')
const cities = ref([])

watch(selectedProvince, (val) => {
  const province = provinces.find(p => p.name === val)
  cities.value = province ? province.cities : []
  selectedCity.value = ''
})

const selectedCity = ref('')
</script>

<template>
  <select v-model="selectedProvince">
    <option value="" disabled>选择省份</option>
    <option v-for="p in provinces" :key="p.id">{{ p.name }}</option>
  </select>
  <select v-model="selectedCity" :disabled="!selectedProvince">
    <option value="" disabled>选择城市</option>
    <option v-for="c in cities" :key="c">{{ c }}</option>
  </select>
</template>
```

## 五、多选框的高级用法

```vue
<script setup>
import { ref, computed } from 'vue'

const selectedSkills = ref([])

const allSkills = ['JavaScript', 'TypeScript', 'Vue', 'React', 'Node.js']

const isAllSelected = computed(() =>
  selectedSkills.value.length === allSkills.length
)

const toggleAll = () => {
  selectedSkills.value = isAllSelected.value ? [] : [...allSkills]
}
</script>

<template>
  <select v-model="selectedSkills" multiple size="5">
    <option v-for="skill in allSkills" :key="skill" :value="skill">
      {{ skill }}
    </option>
  </select>
  <button @click="toggleAll" type="button">
    {{ isAllSelected ? '取消全选' : '全选' }}
  </button>
  <p>已选: {{ selectedSkills.join(', ') }}</p>
</template>
```

## 三、注意事项与常见陷阱

1. 单选`<select>`绑定字符串/对象值
2. 多选`<select>`绑定数组，需加`multiple`属性
3. 第一个`<option>`设为`disabled`作提示选项
4. 动态选项使用`v-for`渲染`<option>`
5. 绑定对象值时，使用`:value`绑定整个对象
6. 级联选择时，切换上级选项需清空下级选择
7. 多选框建议设置`size`属性控制可见行数
