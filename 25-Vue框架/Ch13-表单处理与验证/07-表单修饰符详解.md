# 表单修饰符详解

## 一、概念说明

Vue提供三个`v-model`修饰符：`.lazy`（失焦更新）、`.number`（自动转数字）、`.trim`（去除空格）。

```vue
<template>
  <!-- .lazy: change事件触发而非input -->
  <input v-model.lazy="msg" />
  <p>失焦后才更新: {{ msg }}</p>

  <!-- .number: 自动转为数字 -->
  <input v-model.number="age" type="number" />
  <p>类型: {{ typeof age }}</p>

  <!-- .trim: 自动去除首尾空格 -->
  <input v-model.trim="username" />
  <p>长度: {{ username.length }}</p>

  <!-- 修饰符可链式使用 -->
  <input v-model.lazy.trim="query" />
</template>

<script setup>
import { ref } from 'vue'

const msg = ref('')
const age = ref(0)
const username = ref('')
const query = ref('')
</script>
```

## 二、具体用法

### .lazy

```vue
<!-- 实时更新（input事件） -->
<input v-model="name" />

<!-- 失焦更新（change事件） -->
<input v-model.lazy="name" />
<!-- 适合搜索框（减少请求频率） -->
```

### .number

```vue
<!-- 不加.number: age是字符串"25" -->
<input v-model="age" type="number" />

<!-- 加.number: age是数字25 -->
<input v-model.number="age" type="number" />

<!-- 注意：非数字输入会被parseFloat解析 -->
<input v-model.number="price" /> <!-- "abc" -> NaN -->
```

### .trim

```vue
<!-- 自动去除首尾空格 -->
<input v-model.trim="email" />
<!-- 用户输入"  test@example.com  " -> "test@example.com" -->
```

## 三、注意事项与常见陷阱

1. `.lazy`使更新时机从`input`变为`change`（失焦时）
2. `.number`使用`parseFloat()`转换，非数字变NaN
3. `.trim`只去除首尾空格，不影响中间空格
4. 修饰符可以组合使用：`v-model.lazy.trim`
5. `.number`对`type="number"`特别有用，确保值是数字类型

## 四、修饰符的组合使用

```vue
<template>
  <!-- 组合：先 trim，再 number -->
  <input v-model.number.trim="price" />
  <!-- 输入 "  100  " → price = 100 (number) -->

  <!-- 组合：lazy + trim -->
  <input v-model.lazy.trim="searchQuery" />
  <!-- 失焦后更新，且去除空格 -->

  <!-- 不支持的组合 -->
  <!-- v-model.lazy.number.trim 是支持的 -->
  <!-- 但顺序固定：先 lazy → 再 number → 最后 trim -->
</template>
```

## 五、自定义修饰符

Vue 组件支持自定义 v-model 修饰符：

```vue
<!-- 父组件 -->
<template>
  <CustomInput v-model.capitalize="text" />
</template>

<!-- 子组件 -->
<script setup>
const props = defineProps({
  modelValue: String,
  modelModifiers: { default: () => ({}) }
})

const emit = defineEmits(['update:modelValue'])

function handleInput(e) {
  let value = e.target.value
  if (props.modelModifiers.capitalize) {
    value = value.charAt(0).toUpperCase() + value.slice(1)
  }
  emit('update:modelValue', value)
}
</script>

<template>
  <input :value="modelValue" @input="handleInput" />
</template>
```

## 六、实际场景

```vue
<template>
  <!-- 搜索框：lazy + trim 减少请求 -->
  <input v-model.lazy.trim="searchQuery" placeholder="搜索..." />

  <!-- 价格输入：number 确保数字类型 -->
  <input v-model.number="price" type="number" min="0" step="0.01" />

  <!-- 用户名：trim 去除空格 -->
  <input v-model.trim="username" maxlength="20" />
</template>
```
