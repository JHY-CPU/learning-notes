# 多行文本绑定

## 一、概念说明

`<textarea>`使用`v-model`绑定多行文本。注意不能在`<textarea>`标签内写插值，必须用`v-model`。

```vue
<template>
  <textarea v-model="content" rows="5" placeholder="请输入内容"></textarea>
  <p>字符数: {{ content.length }}</p>
  <p>预览:</p>
  <div class="preview">{{ content }}</div>
</template>

<script setup>
import { ref } from 'vue'

const content = ref('')
</script>
```

## 二、具体用法

### 带字数限制

```vue
<template>
  <textarea
    v-model="bio"
    :maxlength="200"
    rows="4"
    placeholder="个人简介（最多200字）"
  ></textarea>
  <p>{{ bio.length }} / 200</p>
</template>

<script setup>
import { ref } from 'vue'
const bio = ref('')
</script>
```

### 与HTML渲染

```vue
<template>
  <textarea v-model="markdown" rows="6"></textarea>
  <div v-html="renderedContent"></div>
</template>

<script setup>
import { ref, computed } from 'vue'

const markdown = ref('# 标题\n\n这是内容')

const renderedContent = computed(() => {
  // 简单的换行转<br>
  return markdown.value.replace(/\n/g, '<br>')
})
</script>
```

## 三、注意事项与常见陷阱

1. `<textarea>`不能写插值：`<textarea>{{ text }}</textarea>`是错误的
2. 必须使用`v-model`绑定textarea的值
3. `rows`属性控制默认行数，不影响最大行数
4. 换行符`\n`在HTML中不显示，需要转换
5. 大文本绑定可能影响性能，考虑防抖
