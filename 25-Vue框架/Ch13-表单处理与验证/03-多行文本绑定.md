# 多行文本绑定

## 一、概念说明

`<textarea>`使用`v-model`绑定多行文本。注意不能在`<textarea>`标签内写插值，必须用`v-model`。

```vue
<template>
  <textarea v-model="content" rows="5" placeholder="请输入内容"></textarea>
  <p>字符数: {{ content.length }}</p>
  <p>预览:</p>
  <div class="preview">{{ content }}</div>
</template>

<script setup>
import { ref } from 'vue'

const content = ref('')
</script>
```

## 二、具体用法

### 带字数限制

```vue
<template>
  <textarea
    v-model="bio"
    :maxlength="200"
    rows="4"
    placeholder="个人简介（最多200字）"
  ></textarea>
  <p>{{ bio.length }} / 200</p>
</template>

<script setup>
import { ref } from 'vue'
const bio = ref('')
</script>
```

### 与HTML渲染

```vue
<template>
  <textarea v-model="markdown" rows="6"></textarea>
  <div v-html="renderedContent"></div>
</template>

<script setup>
import { ref, computed } from 'vue'

const markdown = ref('# 标题\n\n这是内容')

const renderedContent = computed(() => {
  // 简单的换行转<br>
  return markdown.value.replace(/\n/g, '<br>')
})
</script>
```

## 四、自动调整高度

```vue
<script setup>
import { ref, watch, nextTick } from 'vue'

const content = ref('')
const textareaRef = ref(null)

const autoResize = () => {
  const el = textareaRef.value
  if (!el) return
  el.style.height = 'auto'
  el.style.height = el.scrollHeight + 'px'
}

watch(content, () => {
  nextTick(autoResize)
})
</script>

<template>
  <textarea
    ref="textareaRef"
    v-model="content"
    @input="autoResize"
    rows="2"
    placeholder="输入内容自动扩展高度..."
    style="overflow: hidden; resize: none;"
  ></textarea>
</template>
```

## 五、编辑器功能增强

```vue
<script setup>
import { ref, computed } from 'vue'

const content = ref('')
const maxChars = 500

// 字数统计
const charCount = computed(() => content.value.length)
const charRemaining = computed(() => maxChars - charCount.value)
const isOverLimit = computed(() => charCount.value > maxChars)

// 行数统计
const lineCount = computed(() => {
  if (!content.value) return 0
  return content.value.split('\n').length
})

// 清空内容
const clearContent = () => { content.value = '' }
</script>

<template>
  <div class="editor">
    <textarea v-model="content" :maxlength="maxChars" rows="8"></textarea>
    <div class="status-bar">
      <span>{{ charCount }} 字符 | {{ lineCount }} 行</span>
      <span :class="{ warning: charRemaining < 50 }">
        剩余 {{ charRemaining }} 字符
      </span>
      <button @click="clearContent" type="button">清空</button>
    </div>
  </div>
</template>
```

## 六、与v-model修饰符配合

```vue
<template>
  <!-- .lazy：失焦时更新 -->
  <textarea v-model.lazy="lazyContent" rows="4"></textarea>
  <p>失焦后才更新: {{ lazyContent }}</p>

  <!-- .trim：自动去除首尾空格 -->
  <textarea v-model.trim="trimmedContent" rows="4"></textarea>
  <p>自动去空格: "{{ trimmedContent }}"</p>
</template>

<script setup>
import { ref } from 'vue'
const lazyContent = ref('')
const trimmedContent = ref('')
</script>
```

## 三、注意事项与常见陷阱

1. `<textarea>`不能写插值：`<textarea>{{ text }}</textarea>`是错误的
2. 必须使用`v-model`绑定textarea的值
3. `rows`属性控制默认行数，不影响最大行数
4. 换行符`\n`在HTML中不显示，需要转换
5. 大文本绑定可能影响性能，考虑防抖
6. 自动调整高度需在nextTick后计算scrollHeight
7. 字数限制建议同时使用maxlength属性和JS判断
