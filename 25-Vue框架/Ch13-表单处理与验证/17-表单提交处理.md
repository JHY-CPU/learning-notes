# 表单提交处理

## 一、概念说明

表单提交需要处理：防重复提交、loading状态、错误处理、提交后重置等。使用`@submit.prevent`阻止默认提交。

```vue
<template>
  <form @submit.prevent="handleSubmit">
    <input v-model="form.name" placeholder="姓名" />
    <input v-model="form.email" placeholder="邮箱" />

    <button type="submit" :disabled="submitting">
      {{ submitting ? '提交中...' : '提交' }}
    </button>

    <p v-if="submitError" class="error">{{ submitError }}</p>
    <p v-if="success" class="success">提交成功！</p>
  </form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const form = reactive({ name: '', email: '' })
const submitting = ref(false)
const submitError = ref('')
const success = ref(false)

const handleSubmit = async () => {
  if (submitting.value) return  // 防重复提交

  submitting.value = true
  submitError.value = ''
  success.value = false

  try {
    await api.submit(form)
    success.value = true
    // 重置表单
    form.name = ''
    form.email = ''
  } catch (e) {
    submitError.value = e.message || '提交失败，请重试'
  } finally {
    submitting.value = false
  }
}
</script>
```

## 二、具体用法

### 提交前验证

```js
const handleSubmit = async () => {
  if (!validateForm()) return
  // 提交逻辑
}
```

### 提交后处理

```js
const handleSubmit = async () => {
  const result = await api.submit(form)
  // 方式1：跳转页面
  router.push('/success')
  // 方式2：显示成功消息
  showNotification('提交成功')
  // 方式3：触发父组件事件
  emit('submitted', result)
}
```

## 四、乐观更新与回滚

```vue
<script setup>
import { reactive, ref } from 'vue'

const todos = reactive([...])
const pendingTodo = ref(null)

const addTodo = async (todo) => {
  // 乐观更新：先添加到列表
  const tempId = Date.now()
  const optimisticTodo = { ...todo, id: tempId }
  todos.push(optimisticTodo)
  pendingTodo.value = tempId

  try {
    const result = await api.addTodo(todo)
    // 用服务器返回的真实数据替换
    const index = todos.findIndex(t => t.id === tempId)
    if (index > -1) todos[index] = result
  } catch (e) {
    // 回滚：移除乐观添加的项
    const index = todos.findIndex(t => t.id === tempId)
    if (index > -1) todos.splice(index, 1)
    alert('添加失败: ' + e.message)
  } finally {
    pendingTodo.value = null
  }
}
</script>
```

## 五、带进度的文件上传

```vue
<script setup>
import { ref } from 'vue'

const file = ref(null)
const progress = ref(0)
const uploading = ref(false)

const handleSubmit = async () => {
  if (!file.value) return
  uploading.value = true
  progress.value = 0

  const formData = new FormData()
  formData.append('file', file.value)

  try {
    await fetch('/api/upload', {
      method: 'POST',
      body: formData,
      // XMLHttpRequest支持进度监听，fetch不支持
      // 使用axios时有onUploadProgress
    })
  } finally {
    uploading.value = false
  }
}
</script>
```

## 六、提交后的用户反馈

```vue
<script setup>
import { ref } from 'vue'

const submitStatus = ref('idle')  // idle | submitting | success | error

const handleSubmit = async () => {
  submitStatus.value = 'submitting'
  try {
    await api.submit(form)
    submitStatus.value = 'success'
    // 3秒后重置状态
    setTimeout(() => { submitStatus.value = 'idle' }, 3000)
  } catch (e) {
    submitStatus.value = 'error'
  }
}
</script>

<template>
  <button
    type="submit"
    :disabled="submitStatus === 'submitting'"
    :class="submitStatus"
  >
    <span v-if="submitStatus === 'idle'">提交</span>
    <span v-else-if="submitStatus === 'submitting'">提交中...</span>
    <span v-else-if="submitStatus === 'success'">提交成功</span>
    <span v-else>提交失败，请重试</span>
  </button>
</template>
```

## 三、注意事项与常见陷阱

1. 使用`.prevent`阻止表单默认提交行为
2. `submitting`状态防止重复提交
3. try/catch捕获网络错误，友好提示用户
4. 提交成功后可重置表单或跳转页面
5. 提交按钮禁用时机：提交中或表单无效时
6. 乐观更新适合非关键操作，需实现回滚逻辑
7. 文件上传建议使用XMLHttpRequest或axios获取进度
