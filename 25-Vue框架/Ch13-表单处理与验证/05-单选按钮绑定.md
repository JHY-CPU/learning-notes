# 单选按钮绑定

## 一、概念说明

单选按钮使用`v-model`绑定同一个变量，同一组radio只能选中一个。绑定的值是选中radio的`value`。

```vue
<template>
  <label>
    <input type="radio" v-model="gender" value="male" />
    男
  </label>
  <label>
    <input type="radio" v-model="gender" value="female" />
    女
  </label>
  <p>选择的性别: {{ gender }}</p>
</template>

<script setup>
import { ref } from 'vue'

const gender = ref('male')  // 默认选中男
</script>
```

## 二、具体用法

### 动态生成选项

```vue
<template>
  <div v-for="plan in plans" :key="plan.value">
    <label>
      <input type="radio" v-model="selectedPlan" :value="plan.value" />
      {{ plan.label }} - {{ plan.price }}
    </label>
  </div>
  <p>已选: {{ selectedPlan }}</p>
</template>

<script setup>
import { ref } from 'vue'

const plans = [
  { value: 'free', label: '免费版', price: '¥0' },
  { value: 'pro', label: '专业版', price: '¥99/月' },
  { value: 'enterprise', label: '企业版', price: '¥999/月' }
]
const selectedPlan = ref('free')
</script>
```

### 对象值绑定

```vue
<script setup>
const selectedUser = ref(null)

const users = [
  { id: 1, name: '张三' },
  { id: 2, name: '李四' }
]
</script>

<template>
  <label v-for="user in users" :key="user.id">
    <input type="radio" v-model="selectedUser" :value="user" />
    {{ user.name }}
  </label>
</template>
```

## 四、按钮样式单选

```vue
<template>
  <div class="radio-group">
    <button
      v-for="opt in options"
      :key="opt.value"
      :class="{ active: selected === opt.value }"
      @click="selected = opt.value"
      type="button"
    >
      {{ opt.label }}
    </button>
  </div>
</template>

<script setup>
import { ref } from 'vue'
const options = [
  { label: '日', value: 'day' },
  { label: '周', value: 'week' },
  { label: '月', value: 'month' }
]
const selected = ref('day')
</script>

<style scoped>
.radio-group button.active {
  background: #409eff;
  color: white;
}
</style>
```

## 五、条件联动

```vue
<script setup>
import { ref, watch } from 'vue'

const paymentMethod = ref('alipay')

watch(paymentMethod, (val) => {
  // 切换支付方式时重置相关字段
  if (val === 'bank') {
    cardNumber.value = ''
  }
})

const cardNumber = ref('')
</script>

<template>
  <label><input type="radio" v-model="paymentMethod" value="alipay" /> 支付宝</label>
  <label><input type="radio" v-model="paymentMethod" value="wechat" /> 微信</label>
  <label><input type="radio" v-model="paymentMethod" value="bank" /> 银行卡</label>

  <div v-if="paymentMethod === 'bank'">
    <input v-model="cardNumber" placeholder="银行卡号" />
  </div>
</template>
```

## 六、radio与表单验证

```vue
<script setup>
import { ref, computed } from 'vue'

const shipping = ref('')
const shippingError = computed(() => {
  return shipping.value ? '' : '请选择配送方式'
})
const isValid = computed(() => !!shipping.value)
</script>

<template>
  <fieldset>
    <legend>配送方式 <span class="required">*</span></legend>
    <label><input type="radio" v-model="shipping" value="express" /> 快递</label>
    <label><input type="radio" v-model="shipping" value="pickup" /> 自提</label>
    <span v-if="shippingError" class="error">{{ shippingError }}</span>
  </fieldset>
  <button :disabled="!isValid">提交订单</button>
</template>
```

## 三、注意事项与常见陷阱

1. 同一组radio的`v-model`必须绑定同一变量
2. 每个radio必须有`value`属性
3. 绑定的值是选中radio的value（字符串）
4. 使用`:value`绑定对象或数字
5. 默认选中通过设置v-model的初始值实现
6. radio的`name`属性在Vue中可以省略（v-model自动管理同组）
7. 绑定对象值时，Vue使用引用比较判断选中状态
