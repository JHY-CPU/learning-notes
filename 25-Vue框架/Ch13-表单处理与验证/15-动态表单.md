# 动态表单

## 一、概念说明

动态表单根据配置数据生成表单字段，适用于表单构建器、配置页面等场景。

```vue
<template>
  <form @submit.prevent="handleSubmit">
    <div v-for="field in fields" :key="field.name">
      <label>{{ field.label }}</label>

      <input v-if="field.type === 'text'" v-model="formData[field.name]" />
      <select v-else-if="field.type === 'select'" v-model="formData[field.name]">
        <option v-for="opt in field.options" :key="opt.value" :value="opt.value">
          {{ opt.label }}
        </option>
      </select>
      <textarea v-else-if="field.type === 'textarea'" v-model="formData[field.name]" />

      <span v-if="errors[field.name]" class="error">{{ errors[field.name] }}</span>
    </div>
    <button type="submit">提交</button>
  </form>
</template>

<script setup>
import { reactive } from 'vue'

const fields = [
  { name: 'name', label: '姓名', type: 'text', required: true },
  { name: 'email', label: '邮箱', type: 'text', required: true },
  { name: 'role', label: '角色', type: 'select', options: [
    { value: 'admin', label: '管理员' },
    { value: 'user', label: '普通用户' }
  ]}
]

const formData = reactive({})
fields.forEach(f => { formData[f.name] = '' })
const errors = reactive({})

const handleSubmit = () => {
  // 验证并提交
}
</script>
```

## 二、具体用法

### 动态验证

```js
const validate = () => {
  fields.forEach(field => {
    if (field.required && !formData[field.name]) {
      errors[field.name] = `${field.label}不能为空`
    } else {
      errors[field.name] = ''
    }
  })
}
```

## 三、注意事项与常见陷阱

1. 使用`v-if`/`v-else-if`根据不同类型渲染不同控件
2. 动态v-model绑定使用计算属性名：`v-model="formData[field.name]"`
3. 配置数据可从后端获取，实现后端驱动表单
4. 动态表单的验证规则也应动态配置
5. 组件化每个字段类型，提高复用性

## 四、组件化的动态表单

```vue
<!-- FormField.vue：通用字段组件 -->
<template>
  <div class="form-field" :class="{ 'has-error': error }">
    <label v-if="label">{{ label }}<span v-if="required" class="required">*</span></label>

    <input v-if="type === 'text' || type === 'email' || type === 'password'"
           :type="type" :value="modelValue"
           @input="$emit('update:modelValue', $event.target.value)"
           :placeholder="placeholder" />

    <select v-else-if="type === 'select'" :value="modelValue"
            @change="$emit('update:modelValue', $event.target.value)">
      <option value="" disabled>请选择</option>
      <option v-for="opt in options" :key="opt.value" :value="opt.value">
        {{ opt.label }}
      </option>
    </select>

    <textarea v-else-if="type === 'textarea'" :value="modelValue"
              @input="$emit('update:modelValue', $event.target.value)"
              :rows="rows" :placeholder="placeholder" />

    <span v-if="error" class="error-text">{{ error }}</span>
  </div>
</template>

<script setup>
defineProps({
  type: { type: String, default: 'text' },
  label: String,
  modelValue: [String, Number],
  placeholder: String,
  required: Boolean,
  error: String,
  options: Array,
  rows: { type: Number, default: 3 }
})

defineEmits(['update:modelValue'])
</script>
```

```vue
<!-- DynamicForm.vue -->
<template>
  <form @submit.prevent="handleSubmit">
    <FormField
      v-for="field in fields"
      :key="field.name"
      v-bind="field"
      v-model="formData[field.name]"
      :error="errors[field.name]"
    />
    <button :disabled="isSubmitting">
      {{ isSubmitting ? '提交中...' : '提交' }}
    </button>
  </form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const props = defineProps({
  fields: { type: Array, required: true },
  initialValues: { type: Object, default: () => ({}) }
})

const emit = defineEmits(['submit'])

const formData = reactive({})
props.fields.forEach(f => {
  formData[f.name] = props.initialValues[f.name] ?? ''
})

const errors = reactive({})
const isSubmitting = ref(false)

function validate() {
  Object.keys(errors).forEach(k => delete errors[k])
  props.fields.forEach(f => {
    if (f.required && !formData[f.name]) {
      errors[f.name] = `${f.label}不能为空`
    }
  })
  return Object.keys(errors).length === 0
}

async function handleSubmit() {
  if (!validate()) return
  isSubmitting.value = true
  try {
    emit('submit', { ...formData })
  } finally {
    isSubmitting.value = false
  }
}
</script>
```
