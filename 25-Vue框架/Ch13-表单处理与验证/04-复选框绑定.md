# 复选框绑定

## 一、概念说明

复选框有两种使用方式：**单个**（绑定boolean）和**多个**（绑定数组）。`v-model`自动处理checked和value。

```vue
<template>
  <!-- 单个复选框 -->
  <label>
    <input type="checkbox" v-model="agree" />
    同意协议
  </label>
  <p>是否同意: {{ agree }}</p>

  <!-- 多个复选框 -->
  <label><input type="checkbox" v-model="fruits" value="apple" />苹果</label>
  <label><input type="checkbox" v-model="fruits" value="banana" />香蕉</label>
  <label><input type="checkbox" v-model="fruits" value="orange" />橙子</label>
  <p>选择的水果: {{ fruits }}</p>
</template>

<script setup>
import { ref } from 'vue'

const agree = ref(false)
const fruits = ref([])
</script>
```

## 二、具体用法

### 动态生成复选框

```vue
<template>
  <label v-for="item in options" :key="item.value">
    <input type="checkbox" v-model="selected" :value="item.value" />
    {{ item.label }}
  </label>
  <p>已选: {{ selected }}</p>
</template>

<script setup>
import { ref } from 'vue'

const options = [
  { label: '读', value: 'read' },
  { label: '写', value: 'write' },
  { label: '执行', value: 'execute' }
]
const selected = ref([])
</script>
```

### 全选/取消全选

```vue
<script setup>
import { ref, computed } from 'vue'

const allOptions = ['A', 'B', 'C', 'D']
const selected = ref([])

const isAllSelected = computed(() => selected.value.length === allOptions.length)

const toggleAll = () => {
  selected.value = isAllSelected.value ? [] : [...allOptions]
}
</script>
```

## 三、注意事项与常见陷阱

1. 单个checkbox绑定boolean值
2. 多个checkbox绑定数组，`value`决定数组中的值
3. `true-value`/`false-value`可自定义绑定值
4. 全选功能使用computed判断是否全部选中
5. 复选框的值变化会立即触发`v-model`更新
