# 文件上传

## 一、概念说明

文件上传使用 `FormData` 对象将文件和附加数据一起发送到服务器。Axios 能自动设置 `multipart/form-data` Content-Type。

```vue
<script setup>
import { ref } from 'vue'
import axios from 'axios'

const file = ref(null)
const progress = ref(0)
const uploading = ref(false)

async function upload() {
  if (!file.value) return
  const formData = new FormData()
  formData.append('file', file.value)
  formData.append('category', 'images')

  uploading.value = true
  try {
    await axios.post('/api/upload', formData, {
      headers: { 'Content-Type': 'multipart/form-data' },
      onUploadProgress: (e) => {
        progress.value = Math.round((e.loaded / e.total) * 100)
      }
    })
  } finally {
    uploading.value = false
  }
}
</script>

<template>
  <input type="file" @change="file = $event.target.files[0]" />
  <button @click="upload" :disabled="uploading">
    {{ uploading ? `上传中 ${progress}%` : '上传' }}
  </button>
  <progress v-if="uploading" :value="progress" max="100" />
</template>
```

## 二、具体用法

### 2.1 多文件上传

```js
function uploadMultiple(files) {
  const formData = new FormData()
  files.forEach(f => formData.append('files', f))
  return axios.post('/api/upload/batch', formData)
}
```

### 2.2 上传进度

```js
axios.post('/api/upload', formData, {
  onUploadProgress: (e) => {
    const percent = Math.round((e.loaded / e.total) * 100)
    console.log(`上传进度: ${percent}%`)
  }
})
```

### 2.3 拖拽上传

```vue
<script setup>
const isDragOver = ref(false)

function onDrop(e) {
  isDragOver.value = false
  const droppedFiles = e.dataTransfer.files
  // 处理文件...
}
</script>

<template>
  <div
    @dragover.prevent="isDragOver = true"
    @dragleave="isDragOver = false"
    @drop.prevent="onDrop"
    :class="{ dragover: isDragOver }"
    class="drop-zone"
  >
    拖拽文件到此处
  </div>
</template>
```

## 三、注意事项与常见陷阱

- 不要手动设置 `Content-Type` 的 `boundary`，让浏览器自动处理
- 大文件上传考虑分片上传（chunked upload）
- 文件类型和大小需前后端双重校验
