# 文件下载

## 一、概念说明

浏览器端文件下载通常通过将响应数据转为 Blob 对象，再创建临时链接触发下载实现。Axios 需要设置 `responseType: 'blob'`。

```vue
<script setup>
import axios from 'axios'

async function downloadFile(id) {
  const res = await axios.get(`/api/files/${id}`, {
    responseType: 'blob'
  })
  const blob = new Blob([res.data])
  const url = window.URL.createObjectURL(blob)
  const a = document.createElement('a')
  a.href = url
  a.download = `file-${id}.pdf`
  a.click()
  window.URL.revokeObjectURL(url)
}
</script>

<template>
  <button @click="downloadFile(1)">下载文件</button>
</template>
```

## 二、具体用法

### 2.1 从响应头获取文件名

```js
async function download(url) {
  const res = await axios.get(url, { responseType: 'blob' })
  // Content-Disposition: attachment; filename="report.xlsx"
  const disposition = res.headers['content-disposition']
  const filename = disposition?.match(/filename=(.+)/)?.[1] || 'download'

  const blob = new Blob([res.data])
  const link = document.createElement('a')
  link.href = URL.createObjectURL(blob)
  link.download = decodeURIComponent(filename)
  link.click()
  URL.revokeObjectURL(link.href)
}
```

### 2.2 下载进度

```js
axios.get('/api/download', {
  responseType: 'blob',
  onDownloadProgress: (e) => {
    if (e.total) {
      const percent = Math.round((e.loaded / e.total) * 100)
      console.log(`下载进度: ${percent}%`)
    }
  }
})
```

### 2.3 浏览器直接打开

```js
// 在新窗口中打开，而非下载
const url = URL.createObjectURL(blob)
window.open(url)
```

## 四、多文件打包下载

```js
import JSZip from 'jszip'
import axios from 'axios'

async function downloadZip(files) {
  const zip = new JSZip()

  for (const file of files) {
    const res = await axios.get(file.url, { responseType: 'blob' })
    zip.file(file.name, res.data)
  }

  const blob = await zip.generateAsync({ type: 'blob' })
  const url = URL.createObjectURL(blob)
  const a = document.createElement('a')
  a.href = url
  a.download = 'package.zip'
  a.click()
  URL.revokeObjectURL(url)
}
```

## 五、带进度的下载组件

```vue
<script setup>
import { ref } from 'vue'
import axios from 'axios'

const progress = ref(0)
const downloading = ref(false)
const fileName = ref('')

async function download(url, name) {
  downloading.value = true
  fileName.value = name
  progress.value = 0

  try {
    const res = await axios.get(url, {
      responseType: 'blob',
      onDownloadProgress: (e) => {
        if (e.total) {
          progress.value = Math.round((e.loaded / e.total) * 100)
        }
      }
    })

    const blob = new Blob([res.data])
    const link = document.createElement('a')
    link.href = URL.createObjectURL(blob)
    link.download = name
    link.click()
    URL.revokeObjectURL(link.href)
  } finally {
    downloading.value = false
  }
}
</script>

<template>
  <button @click="download('/api/report.pdf', 'report.pdf')" :disabled="downloading">
    {{ downloading ? `下载中 ${progress}%` : '下载报告' }}
  </button>
  <div v-if="downloading">
    <div class="progress-bar">
      <div :style="{ width: progress + '%' }"></div>
    </div>
    <span>{{ fileName }} - {{ progress }}%</span>
  </div>
</template>
```

## 六、大文件流式下载

```js
// 使用 ReadableStream 处理超大文件，避免内存溢出
async function streamDownload(url, filename) {
  const res = await fetch(url)
  const contentLength = res.headers.get('Content-Length')
  const reader = res.body.getReader()

  const chunks = []
  let receivedLength = 0

  while (true) {
    const { done, value } = await reader.read()
    if (done) break

    chunks.push(value)
    receivedLength += value.length

    if (contentLength) {
      console.log(`进度: ${Math.round(receivedLength / contentLength * 100)}%`)
    }
  }

  const blob = new Blob(chunks)
  const link = document.createElement('a')
  link.href = URL.createObjectURL(blob)
  link.download = filename
  link.click()
}
```

## 三、注意事项与常见陷阱

- `responseType` 必须设为 `'blob'`，否则数据被当作文本解析
- 下载后记得调用 `URL.revokeObjectURL()` 释放内存
- 文件名中的中文需要 `decodeURIComponent` 解码
- 大文件下载建议使用流式处理，避免一次性加载到内存
- 下载进度回调在 axios 中可用，原生 fetch 需要手动实现
