# 测试路由

## 一、概念说明

测试涉及 Vue Router 的组件时，需要 mock 路由环境或创建真实的内存路由。主要测试路由导航、路由参数获取和导航守卫。

```vue
<!-- NavHeader.vue -->
<script setup>
import { useRoute, useRouter } from 'vue-router'

const route = useRoute()
const router = useRouter()

function goBack() {
  router.back()
}
</script>

<template>
  <nav>
    <span>当前路径: {{ route.path }}</span>
    <button @click="goBack">返回</button>
    <RouterLink to="/home">首页</RouterLink>
  </nav>
</template>
```

```js
import { mount } from '@vue/test-utils'
import { createRouter, createMemoryHistory } from 'vue-router'
import NavHeader from '../NavHeader.vue'

const router = createRouter({
  history: createMemoryHistory(),
  routes: [
    { path: '/home', component: { template: '<div>首页</div>' } },
    { path: '/about', component: { template: '<div>关于</div>' } }
  ]
})

describe('NavHeader', () => {
  it('显示当前路径', async () => {
    await router.push('/about')
    await router.isReady()

    const wrapper = mount(NavHeader, {
      global: { plugins: [router] }
    })
    expect(wrapper.text()).toContain('/about')
  })

  it('返回按钮调用 router.back', async () => {
    const spy = vi.spyOn(router, 'back')
    const wrapper = mount(NavHeader, {
      global: { plugins: [router] }
    })
    await wrapper.find('button').trigger('click')
    expect(spy).toHaveBeenCalled()
  })
})
```

## 二、具体用法

### 2.1 创建内存路由

```js
import { createRouter, createMemoryHistory } from 'vue-router'

const router = createRouter({
  history: createMemoryHistory(),
  routes: [/* 路由配置 */]
})
```

### 2.2 Stub 路由组件

```js
const wrapper = mount(Component, {
  global: {
    stubs: {
      RouterLink: true,
      RouterView: true
    }
  }
})
```

### 2.3 测试路由守卫

```js
it('未登录跳转到登录页', async () => {
  const guard = vi.fn((to, from) => {
    if (to.meta.requiresAuth) return '/login'
  })
  router.beforeEach(guard)
  await router.push('/dashboard')
  expect(guard).toHaveBeenCalled()
})
```

## 三、注意事项与常见陷阱

- 使用 `createMemoryHistory` 而非 `createWebHistory`，避免浏览器依赖
- `push` 后需要 `await router.isReady()` 确保路由就绪
- 每个测试文件可以共享一个 router 实例
