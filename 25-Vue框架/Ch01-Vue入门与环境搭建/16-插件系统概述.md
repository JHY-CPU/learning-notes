# 插件系统概述

## 一、概念说明

Vue 插件是为 Vue 应用添加全局功能的工具。插件通过 `app.use()` 安装，可以添加全局组件、指令、mixin、provide 值等。常见的插件有 Vue Router、Pinia、Vue I18n 等。

插件是一个对象或函数，必须暴露一个 `install` 方法。

```vue
<script setup>
// 使用插件: main.js
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'
import { createPinia } from 'pinia'

const app = createApp(App)
app.use(createPinia())  // 安装 Pinia 插件
app.use(router)         // 安装路由插件
app.mount('#app')
</script>
```

## 二、具体用法

### 2.1 编写简单插件

```js
// plugins/myPlugin.js
export default {
  install(app, options) {
    // 添加全局方法
    app.config.globalProperties.$hello = (name) => {
      console.log(`Hello, ${name}!`)
    }

    // 添加全局指令
    app.directive('focus', {
      mounted(el) {
        el.focus()
      }
    })

    // 添加全局 provide
    app.provided.myOptions = options

    // 添加全局组件
    app.component('MyButton', {
      template: '<button><slot /></button>'
    })
  }
}
```

### 2.2 安装插件

```js
import myPlugin from './plugins/myPlugin'

const app = createApp(App)
app.use(myPlugin, { /* 传入选项 */ })
```

### 2.3 组合式插件

```js
// 插件也可以是函数
export function myComposablePlugin(app) {
  app.provide('theme', { primary: '#42b883', dark: false })
}
```

## 三、注意事项与常见陷阱

- `app.use()` 必须在 `app.mount()` 之前调用
- 插件的 `install` 方法接收 `(app, options)` 两个参数
- 重复调用 `app.use()` 同一插件不会报错但也不必要
- TypeScript 项目中使用插件可能需要扩展全局类型
- 推荐使用 provide/inject 而非 globalProperties
