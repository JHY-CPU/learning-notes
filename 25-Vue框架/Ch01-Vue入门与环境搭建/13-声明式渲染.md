# 声明式渲染

## 一、概念说明

**声明式渲染**是 Vue 的核心理念。开发者声明数据与 DOM 的绑定关系，Vue 自动将数据渲染到 DOM 中。与命令式编程（手动操作 DOM）不同，声明式渲染让开发者只需关注**数据是什么**，而非**如何更新 DOM**。

```vue
<script setup>
import { ref } from 'vue'

const items = ref(['Vue', 'React', 'Angular'])
const newItem = ref('')

function addItem() {
  if (newItem.value.trim()) {
    items.value.push(newItem.value)
    newItem.value = ''
  }
}
</script>

<template>
  <input v-model="newItem" placeholder="输入框架名" />
  <button @click="addItem">添加</button>
  <ul>
    <li v-for="(item, index) in items" :key="index">{{ item }}</li>
  </ul>
</template>
```

## 二、具体用法

### 2.1 文本插值 `{{ }}`

```vue
<script setup>
import { ref } from 'vue'
const name = ref('Vue 3')
const version = ref(3.4)
const count = ref(10)
const items = ref(['A', 'B', 'C'])
</script>

<template>
  <h1>{{ name }}</h1>
  <p>版本: {{ version }}</p>
  <p>双倍: {{ count * 2 }}</p>
  <p>三元: {{ count > 0 ? '有值' : '无值' }}</p>
  <p>方法: {{ items.reverse().join(', ') }}</p>
</template>
```

### 2.2 数据驱动更新

```vue
<script setup>
import { ref } from 'vue'
const time = ref(new Date().toLocaleTimeString())

setInterval(() => {
  time.value = new Date().toLocaleTimeString()
}, 1000)
</script>

<template>
  <p>当前时间: {{ time }}</p>
</template>
```

### 2.3 属性绑定

```vue
<script setup>
import { ref } from 'vue'
const imageUrl = ref('/logo.png')
const isDisabled = ref(true)
const inputId = ref('username')
</script>

<template>
  <!-- v-bind 简写为 : -->
  <img :src="imageUrl" alt="Logo" />
  <button :disabled="isDisabled">提交</button>
  <input :id="inputId" />
</template>
```

### 2.4 声明式 vs 命令式对比

```js
// 命令式（jQuery 风格）—— 手动操作 DOM
$('#btn').on('click', function() {
  const val = $('#input').val()
  $('#list').append('<li>' + val + '</li>')
  $('#input').val('')
})
```

```vue
<!-- 声明式（Vue 风格）—— 描述数据与视图的关系 -->
<script setup>
import { ref } from 'vue'
const items = ref([])
const input = ref('')

function addItem() {
  items.value.push(input.value)
  input.value = ''
}
</script>

<template>
  <input v-model="input" />
  <button @click="addItem">添加</button>
  <li v-for="item in items" :key="item">{{ item }}</li>
</template>
```

## 三、常见用例

### 3.1 条件显示

```vue
<script setup>
import { ref } from 'vue'
const isLoggedIn = ref(false)
const user = ref({ name: '张三', role: 'admin' })
</script>

<template>
  <p v-if="isLoggedIn">欢迎, {{ user.name }}</p>
  <p v-else>请先登录</p>
  <span v-show="user.role === 'admin'">管理员标识</span>
</template>
```

### 3.2 计算属性声明

```vue
<script setup>
import { ref, computed } from 'vue'
const firstName = ref('张')
const lastName = ref('三')

const fullName = computed(() => `${firstName.value}${lastName.value}`)
</script>

<template>
  <p>全名: {{ fullName }}</p>
</template>
```

## 四、注意事项与常见陷阱

- `{{ }}` 内只能放**单个表达式**，不能放语句（如 `if`、`for`）
- 模板中的表达式被限制在当前组件实例的作用域内
- 不要在插值中产生副作用（如修改状态）
- `{{ }}` 对 `null`/`undefined` 会显示空字符串
- 响应式数据变化后，DOM 更新是**异步**的（nextTick 后生效）
- 用 `v-html` 渲染原始 HTML，但要防范 XSS 攻击
