# 声明式渲染

## 一、概念说明

**声明式渲染**是 Vue 的核心理念。开发者声明数据与 DOM 的绑定关系，Vue 自动将数据渲染到 DOM 中。与命令式编程（手动操作 DOM）不同，声明式渲染让开发者只需关注**数据是什么**，而非**如何更新 DOM**。

```vue
<script setup>
import { ref } from 'vue'

// 声明数据
const items = ref(['Vue', 'React', 'Angular'])
const newItem = ref('')

function addItem() {
  if (newItem.value.trim()) {
    items.value.push(newItem.value)
    newItem.value = ''
  }
}
</script>

<template>
  <!-- 声明式地描述数据与视图的关系 -->
  <input v-model="newItem" placeholder="输入框架名" />
  <button @click="addItem">添加</button>
  <ul>
    <li v-for="(item, index) in items" :key="index">{{ item }}</li>
  </ul>
</template>
```

## 二、具体用法

### 2.1 文本插值 `{{ }}`

```vue
<script setup>
import { ref } from 'vue'
const name = ref('Vue 3')
const version = ref(3.4)
const count = ref(10)
</script>

<template>
  <!-- 基本插值 -->
  <h1>{{ name }}</h1>

  <!-- 表达式 -->
  <p>版本: {{ version }}</p>
  <p>双倍: {{ count * 2 }}</p>
  <p>三元: {{ count > 0 ? '有值' : '无值' }}</p>
  <p>方法: {{ items.reverse().join(', ') }}</p>
</template>
```

### 2.2 数据驱动更新

```vue
<script setup>
import { ref } from 'vue'
const time = ref(new Date().toLocaleTimeString())

setInterval(() => {
  time.value = new Date().toLocaleTimeString() // 数据变化 → 视图自动更新
}, 1000)
</script>

<template>
  <p>当前时间: {{ time }}</p>
</template>
```

## 三、注意事项与常见陷阱

- `{{ }}` 内只能放**单个表达式**，不能放语句（如 `if`、`for`）
- 模板中的表达式被限制在当前组件实例的作用域内
- 不要在插值中产生副作用（如修改状态）
- `{{ }}` 对 `null`/`undefined` 会显示空字符串
- 响应式数据变化后，DOM 更新是**异步**的（nextTick 后生效）
