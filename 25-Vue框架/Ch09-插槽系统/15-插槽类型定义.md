# 插槽类型定义

## 一、概念说明
在 TypeScript + Vue 3 项目中，需要为插槽提供**类型定义**以获得完整的类型检查和智能提示。

## 二、具体用法

### 2.1 组件插槽类型
```vue
<!-- List.vue -->
<script setup lang="ts" generic="T">
interface Props {
  items: T[]
}
defineProps<Props>()
</script>

<template>
  <ul>
    <li v-for="(item, index) in items" :key="index">
      <slot :item="item" :index="index" />
    </li>
  </ul>
</template>
```

### 2.2 类型声明文件
```ts
// shims-components.d.ts
import type { Slots } from 'vue'

declare module '*.vue' {
  import type { DefineComponent } from 'vue'
  const component: DefineComponent<{}, {}, any>
  export default component
}

// 为特定组件声明插槽类型
declare module './components/List.vue' {
  interface ListSlots<T> {
    default: (props: { item: T; index: number }) => any
    empty: () => any
  }
}
```

### 2.3 使用 Slots 类型
```ts
import type { Slots } from 'vue'

function hasSlot(slots: Slots, name: string): boolean {
  return !!slots[name]
}
```

### 2.4 泛型组件中的插槽类型
```vue
<script setup lang="ts" generic="T extends { id: number }">
defineProps<{
  items: T[]
  selected?: number[]
}>()

defineSlots<{
  default(props: { item: T; index: number; isSelected: boolean }): any
  header(props: { total: number }): any
  empty(): any
}>()
</script>
```

### 2.5 在父组件中获得类型检查
```vue
<script setup lang="ts">
import List from './List.vue'
</script>

<template>
  <List :items="users" v-slot="{ item, index }">
    <!-- item 自动推断为 User 类型 -->
    <!-- index 自动推断为 number 类型 -->
    {{ item.name }}
  </List>
</template>
```

## 三、注意事项与常见陷阱
- Vue 3.3+ 支持 `defineSlots` 宏
- 泛型组件需要 `generic` 属性
- 插槽函数的返回值类型用 `any` 即可
- 使用 `defineSlots` 可以不写 `<script setup>` 中的其他类型声明

## 六、defineSlots 的完整用法

```vue
<script setup lang="ts" generic="T extends { id: number | string }">
interface SlotProps {
  item: T
  index: number
  isSelected: boolean
}

defineProps<{
  items: T[]
  selected?: Array<number | string>
}>()

// defineSlots 提供完整的类型检查
defineSlots<{
  default(props: SlotProps): any
  header(props: { total: number; selectedCount: number }): any
  empty(): any
  footer(props: { page: number; totalPages: number }): any
}>()
</script>

<template>
  <div>
    <slot name="header" :total="items.length" :selectedCount="selected?.length || 0" />
    <slot v-if="items.length === 0" name="empty" />
    <ul v-else>
      <li v-for="(item, index) in items" :key="item.id">
        <slot :item="item" :index="index" :isSelected="selected?.includes(item.id)" />
      </li>
    </ul>
    <slot name="footer" :page="1" :totalPages="1" />
  </div>
</template>
```

## 七、没有 defineSlots 时的类型处理

```ts
// 在 .d.ts 文件中手动声明
import type { DefineComponent, SlotsType } from 'vue'

declare const List: DefineComponent<
  { items: any[] },
  {},
  {},
  {},
  {},
  {},
  {},
  {},
  {},
  {},
  {},
  {},
  SlotsType<{
    default: (props: { item: any; index: number }) => any
    empty: () => any
  }>
>

export default List
```
