# 渲染函数中的插槽

## 一、概念说明
在**渲染函数**（`h()`）和 JSX 中也可以使用插槽。通过 `setup` 函数的第二个参数 `slots` 来访问和使用插槽内容。

## 二、具体用法

### 2.1 使用 h() 函数访问插槽
```js
import { h } from 'vue'

export default {
  setup(props, { slots }) {
    return () => h('div', { class: 'wrapper' }, [
      h('header', null, slots.header?.()),
      h('main', null, slots.default?.()),
      h('footer', null, slots.footer?.())
    ])
  }
}
```

### 2.2 传递数据给插槽
```js
import { h } from 'vue'

export default {
  setup(props) {
    const items = [{ id: 1, name: '张三' }, { id: 2, name: '李四' }]

    return () => h('ul', null,
      items.map(item =>
        h('li', { key: item.id },
          slots.default?.({ item, index: items.indexOf(item) })
        )
      )
    )
  }
}
```

### 2.3 JSX 中使用插槽
```jsx
export default {
  setup(props, { slots }) {
    return () => (
      <div class="card">
        <header>{slots.header?.()}</header>
        <main>{slots.default?.()}</main>
        <footer>{slots.footer?.()}</footer>
      </div>
    )
  }
}
```

### 2.4 检查插槽是否存在
```js
export default {
  setup(props, { slots }) {
    const hasHeader = !!slots.header
    const hasFooter = !!slots.footer

    return () => h('div', null, [
      hasHeader && h('header', null, slots.header()),
      h('main', null, slots.default?.()),
      hasFooter && h('footer', null, slots.footer())
    ])
  }
}
```

### 2.5 在 script setup 中使用渲染函数
```vue
<script setup>
import { h, useSlots } from 'vue'

const slots = useSlots()

// 使用渲染函数替代 template
const render = () => h('div', { class: 'container' }, [
  slots.header?.(),
  slots.default?.()
])
</script>
```

## 三、注意事项与常见陷阱
- `slots.xxx?.()` 使用可选链，防止插槽不存在时报错
- 在 `setup` 中不要在返回函数外调用 `slots.xxx()`
- JSX 中插槽调用方式与 h() 函数一致
- 混用 template 和渲染函数时注意插槽的传递
