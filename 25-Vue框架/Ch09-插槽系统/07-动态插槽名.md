# 动态插槽名

## 一、概念说明
插槽名可以是**动态的**，通过 JavaScript 表达式来决定使用哪个插槽。使用方括号语法 `v-slot:[dynamicName]` 或简写 `#[dynamicName]`。

## 二、具体用法

### 2.1 基本用法
```vue
<template>
  <Layout>
    <template #[currentSlot]>
      <p>动态插槽内容</p>
    </template>
  </Layout>
</template>
<script setup>
import { ref } from 'vue'
const currentSlot = ref('header')
// 切换为 'footer' 时内容会移到页脚
</script>
```

### 2.2 配合用户选择
```vue
<template>
  <div>
    <select v-model="slotTarget">
      <option value="header">头部</option>
      <option value="body">主体</option>
      <option value="footer">尾部</option>
    </select>

    <Layout>
      <template #[slotTarget]>
        <p>这段内容会显示在: {{ slotTarget }}</p>
      </template>
    </Layout>
  </div>
</template>
<script setup>
import { ref } from 'vue'
const slotTarget = ref('header')
</script>
```

### 2.3 动态作用域插槽
```vue
<template>
  <Table :data="rows">
    <template v-slot:[getCellSlot(column)]="{ value }">
      <!-- 根据列类型动态选择渲染方式 -->
      <span>{{ formatValue(value, column.type) }}</span>
    </template>
  </Table>
</template>
<script setup>
function getCellSlot(column) {
  return `cell-${column.key}`
}
</script>
```

### 2.4 条件性插槽
```vue
<template>
  <Layout>
    <template #[showSidebar ? 'sidebar' : 'default']>
      <nav>导航内容</nav>
    </template>
  </Layout>
</template>
<script setup>
import { ref } from 'vue'
const showSidebar = ref(true)
</script>
```

## 三、注意事项与常见陷阱
- 动态插槽名必须是有效的 JavaScript 表达式
- 表达式的结果必须是字符串或 `null`/`undefined`
- 与缩写组合时：`#[dynamicName]`，方括号内不能有空格
- 如果表达式为 `null`，该插槽不被应用

## 四、实际应用场景

### 4.1 动态表单字段
```vue
<template>
  <FormBuilder :fields="formFields">
    <template v-for="field in formFields" :key="field.name"
              #[field.name]="{ value, update }">
      <!-- 根据字段类型动态选择渲染方式 -->
      <input v-if="field.type === 'text'"
             :value="value" @input="update($event.target.value)" />
      <select v-else-if="field.type === 'select'"
              :value="value" @change="update($event.target.value)">
        <option v-for="opt in field.options" :key="opt.value" :value="opt.value">
          {{ opt.label }}
        </option>
      </select>
    </template>
  </FormBuilder>
</template>
```

### 4.2 根据权限显示不同插槽
```vue
<script setup>
import { ref, computed } from 'vue'

const userRole = ref('admin')
const contentSlot = computed(() => {
  return userRole.value === 'admin' ? 'admin-content'
    : userRole.value === 'editor' ? 'editor-content'
    : 'viewer-content'
})
</script>

<template>
  <Dashboard>
    <template #[contentSlot]>
      <AdminPanel v-if="userRole === 'admin'" />
      <EditorPanel v-else-if="userRole === 'editor'" />
      <ViewerPanel v-else />
    </template>
  </Dashboard>
</template>
```

### 4.3 循环生成插槽
```vue
<template>
  <Table :columns="columns" :data="rows">
    <template v-for="col in columns" :key="col.key"
              #[`cell-${col.key}`]="{ value, row }">
      <!-- 自定义每列的渲染 -->
      <CustomCell :value="value" :config="col" :row="row" />
    </template>
  </Table>
</template>
```
