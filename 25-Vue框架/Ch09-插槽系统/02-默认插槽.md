# 默认插槽

## 一、概念说明
默认插槽是**未指定名称的插槽**，子组件中的 `<slot />` 标签定义了父组件内容的插入位置。一个组件只能有一个默认插槽。

## 二、具体用法

### 2.1 基本用法
```vue
<!-- 子组件 Alert.vue -->
<template>
  <div class="alert">
    <slot>默认提示信息</slot>
  </div>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <!-- 使用默认内容 -->
  <Alert />

  <!-- 覆盖默认内容 -->
  <Alert>
    <p>自定义警告内容</p>
  </Alert>
</template>
```

### 2.2 多个元素作为插槽内容
```vue
<template>
  <Card>
    <h2>标题</h2>
    <p>段落1</p>
    <p>段落2</p>
    <button>操作</button>
  </Card>
</template>
```

### 2.3 组件作为插槽内容
```vue
<template>
  <Container>
    <UserProfile />
    <SettingPanel />
  </Container>
</template>
```

### 2.4 带后备内容（默认值）
```vue
<!-- 子组件 Button.vue -->
<template>
  <button class="btn">
    <slot>
      <!-- 没有传内容时显示 -->
      <span>默认按钮</span>
    </slot>
  </button>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <Button />  <!-- 显示"默认按钮" -->
  <Button>保存</Button>  <!-- 显示"保存" -->
  <Button><Icon /> 提交</Button>  <!-- 显示图标+文字 -->
</template>
```

## 三、注意事项与常见陷阱
- 如果子组件没有 `<slot>` 标签，父组件传入的内容会被**丢弃**
- 插槽内容在**父组件的作用域**中编译，访问父组件的数据
- 空的 `<slot>` 也会渲染，只是没有后备内容
- `<slot>` 标签上不要添加其他属性（具名插槽用 `name`）

## 四、插槽内容的编译位置

```vue
<!-- 子组件 -->
<template>
  <div class="child">
    <p>子组件内容</p>
    <slot />  <!-- 插槽占位符 -->
    <p>子组件底部</p>
  </div>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <Child>
    <p>插槽内容</p>
  </Child>
</template>
<!-- 渲染结果：
<div class="child">
  <p>子组件内容</p>
  <p>插槽内容</p>     ← 插入到 <slot> 的位置
  <p>子组件底部</p>
</div>
-->
```

## 五、嵌套默认插槽

```vue
<!-- 多层嵌套 -->
<template>
  <Card>
    <Panel>
      <Alert>
        <p>最深层的内容</p>
      </Alert>
    </Panel>
  </Card>
</template>
```

## 六、检查插槽是否存在

```vue
<script setup>
import { useSlots, computed } from 'vue'

const slots = useSlots()

// 检查默认插槽是否有内容
const hasDefaultSlot = computed(() => !!slots.default)

// 检查默认插槽是否有实际内容（非空格）
const hasRealContent = computed(() => {
  const children = slots.default?.()
  return children?.some(vnode =>
    vnode.type !== Comment &&
    (typeof vnode.children !== 'string' || vnode.children.trim() !== '')
  )
})
</script>

<template>
  <div class="card">
    <div v-if="hasDefaultSlot" class="card-body">
      <slot />
    </div>
    <div v-else class="card-empty">
      暂无内容
    </div>
  </div>
</template>
```
