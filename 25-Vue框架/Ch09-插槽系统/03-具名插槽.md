# 具名插槽

## 一、概念说明
具名插槽通过 `name` 属性来**区分不同的插槽位置**。父组件使用 `v-slot:name` 或 `#name` 将内容分发到对应的插槽。

## 二、具体用法

### 2.1 基本用法
```vue
<!-- 子组件 Layout.vue -->
<template>
  <div class="layout">
    <header><slot name="header" /></header>
    <main><slot /></main>        <!-- 隐含 name="default" -->
    <aside><slot name="sidebar" /></aside>
    <footer><slot name="footer" /></footer>
  </div>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <Layout>
    <template v-slot:header>
      <h1>网站标题</h1>
    </template>

    <template v-slot:default>
      <p>主内容区域</p>
    </template>

    <template v-slot:sidebar>
      <nav>导航菜单</nav>
    </template>

    <template v-slot:footer>
      <p>版权信息</p>
    </template>
  </Layout>
</template>
```

### 2.2 缩写语法
```vue
<template>
  <Layout>
    <template #header>标题</template>
    <template #default>内容</template>
    <template #sidebar>侧边栏</template>
    <template #footer>页脚</template>
  </Layout>
</template>
```

### 2.3 默认插槽的简写
```vue
<template>
  <Layout>
    <template #header>标题</template>

    <!-- 默认插槽可以不用 template 包裹 -->
    <p>直接写内容，自动进入默认插槽</p>

    <template #footer>页脚</template>
  </Layout>
</template>
```

### 2.4 动态插槽名
```vue
<template>
  <Layout>
    <template #[dynamicSlotName]>
      <p>动态内容</p>
    </template>
  </Layout>
</template>
<script setup>
import { ref } from 'vue'
const dynamicSlotName = ref('header')
</script>
```

## 三、注意事项与常见陷阱
- 默认插槽的隐含名称是 `default`
- 只有**默认插槽**内容可以不用 `<template>` 包裹
- 如果同时有具名插槽和默认插槽，所有内容都必须用 `v-slot` 指定
- 动态插槽名用方括号语法：`#[name]`
