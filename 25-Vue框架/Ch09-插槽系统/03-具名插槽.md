# 具名插槽

## 一、概念说明
具名插槽通过 `name` 属性来**区分不同的插槽位置**。父组件使用 `v-slot:name` 或 `#name` 将内容分发到对应的插槽。

## 二、具体用法

### 2.1 基本用法
```vue
<!-- 子组件 Layout.vue -->
<template>
  <div class="layout">
    <header><slot name="header" /></header>
    <main><slot /></main>        <!-- 隐含 name="default" -->
    <aside><slot name="sidebar" /></aside>
    <footer><slot name="footer" /></footer>
  </div>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <Layout>
    <template v-slot:header>
      <h1>网站标题</h1>
    </template>

    <template v-slot:default>
      <p>主内容区域</p>
    </template>

    <template v-slot:sidebar>
      <nav>导航菜单</nav>
    </template>

    <template v-slot:footer>
      <p>版权信息</p>
    </template>
  </Layout>
</template>
```

### 2.2 缩写语法
```vue
<template>
  <Layout>
    <template #header>标题</template>
    <template #default>内容</template>
    <template #sidebar>侧边栏</template>
    <template #footer>页脚</template>
  </Layout>
</template>
```

### 2.3 默认插槽的简写
```vue
<template>
  <Layout>
    <template #header>标题</template>

    <!-- 默认插槽可以不用 template 包裹 -->
    <p>直接写内容，自动进入默认插槽</p>

    <template #footer>页脚</template>
  </Layout>
</template>
```

### 2.4 动态插槽名
```vue
<template>
  <Layout>
    <template #[dynamicSlotName]>
      <p>动态内容</p>
    </template>
  </Layout>
</template>
<script setup>
import { ref } from 'vue'
const dynamicSlotName = ref('header')
</script>
```

## 三、注意事项与常见陷阱
- 默认插槽的隐含名称是 `default`
- 只有**默认插槽**内容可以不用 `<template>` 包裹
- 如果同时有具名插槽和默认插槽，所有内容都必须用 `v-slot` 指定
- 动态插槽名用方括号语法：`#[name]`

## 四、实际应用场景

### 4.1 通用表格组件
```vue
<!-- DataTable.vue -->
<template>
  <div class="data-table">
    <header>
      <slot name="toolbar">
        <button @click="refresh">刷新</button>
      </slot>
    </header>
    <table>
      <thead>
        <tr>
          <slot name="header-row">
            <th v-for="col in columns" :key="col.key">{{ col.label }}</th>
          </slot>
        </tr>
      </thead>
      <tbody>
        <tr v-for="row in data" :key="row.id">
          <slot name="row" :row="row" />
        </tr>
      </tbody>
      <tfoot v-if="$slots.footer">
        <tr><slot name="footer" /></tr>
      </tfoot>
    </table>
    <footer>
      <slot name="pagination" :page="page" :total="total" />
    </footer>
  </div>
</template>
```

### 4.2 模态框组件
```vue
<!-- Modal.vue -->
<template>
  <Teleport to="body">
    <div v-if="visible" class="modal-overlay" @click.self="$emit('close')">
      <div class="modal">
        <header>
          <slot name="header">
            <h3>{{ title }}</h3>
          </slot>
          <button @click="$emit('close')">×</button>
        </header>
        <main>
          <slot />
        </main>
        <footer v-if="$slots.footer">
          <slot name="footer" />
        </footer>
      </div>
    </div>
  </Teleport>
</template>
```

### 4.3 检查具名插槽是否存在
```vue
<script setup>
import { useSlots } from 'vue'
const slots = useSlots()
</script>

<template>
  <div class="card">
    <header v-if="$slots.header">
      <slot name="header" />
    </header>
    <main>
      <slot />
    </main>
    <footer v-if="$slots.footer">
      <slot name="footer" />
    </footer>
  </div>
</template>
```
