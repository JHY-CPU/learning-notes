# 布局插槽模式

## 一、概念说明
布局组件通过插槽定义**页面结构骨架**（header/sidebar/footer），让不同页面共享统一的布局，同时允许每个页面自定义内容。

## 二、具体用法

### 2.1 基础布局组件
```vue
<!-- MainLayout.vue -->
<template>
  <div class="layout">
    <header class="layout-header">
      <slot name="header">
        <nav>默认导航栏</nav>
      </slot>
    </header>

    <div class="layout-body">
      <aside v-if="$slots.sidebar" class="layout-sidebar">
        <slot name="sidebar" />
      </aside>

      <main class="layout-content">
        <slot />
      </main>
    </div>

    <footer class="layout-footer">
      <slot name="footer">
        <p>&copy; 2024</p>
      </slot>
    </footer>
  </div>
</template>
```

### 2.2 页面使用布局
```vue
<!-- HomePage.vue -->
<template>
  <MainLayout>
    <template #header>
      <nav>
        <a href="/">首页</a>
        <a href="/about">关于</a>
      </nav>
    </template>

    <template #sidebar>
      <ul>
        <li>菜单项1</li>
        <li>菜单项2</li>
      </ul>
    </template>

    <h1>欢迎来到首页</h1>
    <p>这是主要内容区域</p>
  </MainLayout>
</template>
```

### 2.3 响应式布局（带侧边栏切换）
```vue
<script setup>
import { useSlots } from 'vue'
const slots = useSlots()
</script>

<template>
  <div :class="['layout', { 'has-sidebar': !!slots.sidebar }]">
    <aside v-if="slots.sidebar">
      <slot name="sidebar" />
    </aside>
    <main><slot /></main>
  </div>
</template>
```

### 2.4 嵌套布局
```vue
<template>
  <AppLayout>
    <DashboardLayout>
      <template #widgets>
        <StatsWidget />
      </template>
      <p>仪表盘内容</p>
    </DashboardLayout>
  </AppLayout>
</template>
```

## 三、注意事项与常见陷阱
- 布局组件应尽量轻量，只负责结构
- 默认插槽放主内容，具名插槽放导航/侧边栏/页脚
- 使用 `$slots` 检查插槽是否存在来条件性渲染区域
- 考虑用 Vue Router 的路由嵌套代替嵌套布局组件
