# 高阶组件与插槽

## 一、概念说明
**Renderless 组件（无渲染组件）**是一种高阶组件模式：组件只提供数据和逻辑，不包含任何模板。通过作用域插槽将数据暴露给父组件，由父组件控制渲染。

## 二、具体用法

### 2.1 Renderless 计数器
```vue
<!-- Counter.vue（无渲染） -->
<script setup>
import { ref } from 'vue'

const count = ref(0)
const increment = () => count.value++
const decrement = () => count.value--
const reset = () => { count.value = 0 }
</script>

<template>
  <slot :count="count" :increment="increment" :decrement="decrement" :reset="reset" />
</template>
```

```vue
<!-- 父组件控制渲染 -->
<template>
  <Counter v-slot="{ count, increment, decrement, reset }">
    <div class="custom-counter">
      <button @click="decrement">-</button>
      <span class="count">{{ count }}</span>
      <button @click="increment">+</button>
      <button @click="reset">重置</button>
    </div>
  </Counter>
</template>
```

### 2.2 Renderless 数据请求
```vue
<!-- Fetcher.vue -->
<script setup>
import { ref, onMounted } from 'vue'

const props = defineProps<{ url: string }>()
const data = ref(null)
const loading = ref(true)
const error = ref(null)

onMounted(async () => {
  try {
    const res = await fetch(props.url)
    data.value = await res.json()
  } catch (e) {
    error.value = e
  } finally {
    loading.value = false
  }
})

const refresh = async () => { /* 重新请求 */ }
</script>

<template>
  <slot :data="data" :loading="loading" :error="error" :refresh="refresh" />
</template>
```

```vue
<!-- 父组件自定义渲染 -->
<template>
  <Fetcher url="/api/users" v-slot="{ data, loading, error }">
    <div v-if="loading">加载中...</div>
    <div v-else-if="error">错误: {{ error.message }}</div>
    <ul v-else>
      <li v-for="u in data" :key="u.id">{{ u.name }}</li>
    </ul>
  </Fetcher>
</template>
```

### 2.3 Renderless 表单验证
```vue
<!-- FormValidator.vue -->
<script setup>
import { reactive, computed } from 'vue'
const form = reactive({ email: '', password: '' })
const errors = reactive({ email: '', password: '' })
const isValid = computed(() => !errors.email && !errors.password)
</script>

<template>
  <slot :form="form" :errors="errors" :isValid="isValid" />
</template>
```

## 三、注意事项与常见陷阱
- Renderless 组件的 `<template>` 中只有 `<slot>` 标签
- 也可以用纯 JS（无 `<template>`）配合 `render` 函数实现
- Vue 3 中组合式函数可以替代部分 renderless 模式
- 适合需要复用逻辑但渲染不同的场景
