# 自定义指令修饰符

## 一、概念说明
指令**修饰符**是附加在指令后的特殊后缀（如 `.prevent`），通过 `binding.modifiers` 对象访问。可以用来配置指令的行为。

## 二、具体用法

### 2.1 基本修饰符
```vue
<script setup>
const vHighlight = {
  mounted: (el, binding) => {
    // binding.modifiers = { bold: true, italic: true }
    if (binding.modifiers.bold) el.style.fontWeight = 'bold'
    if (binding.modifiers.italic) el.style.fontStyle = 'italic'
    el.style.backgroundColor = binding.value || 'yellow'
  }
}
</script>

<template>
  <p v-highlight.bold>粗体高亮</p>
  <p v-highlight.italic="'lightblue'">斜体蓝色高亮</p>
  <p v-highlight.bold.italic>粗斜体高亮</p>
</template>
```

### 2.2 修饰符与 arg 组合
```vue
<script setup>
const vBind = {
  mounted: (el, binding) => {
    // v-bind:color.prevent → arg = 'color', modifiers = { prevent: true }
    const prop = binding.arg
    const value = binding.value
    const options = binding.modifiers

    if (prop === 'color') {
      el.style.color = value
    }
    if (options.prevent) {
      el.addEventListener('click', (e) => e.preventDefault())
    }
  }
}
</script>
```

### 2.3 数字键修饰符
```vue
<script setup>
const vDelay = {
  mounted: (el, binding) => {
    // v-delay.1000 → modifiers = { 1000: true }
    const delay = Object.keys(binding.modifiers)[0] || 300
    el.style.transition = `opacity ${delay}ms`
  }
}
</script>

<template>
  <div v-delay.500>500ms 延迟</div>
  <div v-delay.1000>1000ms 延迟</div>
</template>
```

### 2.4 实际应用：格式化输入
```vue
<script setup>
const vFormat = {
  mounted: (el, binding) => {
    el.addEventListener('input', (e) => {
      let val = e.target.value
      if (binding.modifiers.uppercase) val = val.toUpperCase()
      if (binding.modifiers.trim) val = val.trim()
      if (binding.modifiers.digits) val = val.replace(/\D/g, '')
      e.target.value = val
    })
  }
}
</script>

<template>
  <input v-format.uppercase placeholder="自动大写" />
  <input v-format.digits placeholder="只允许数字" />
  <input v-format.trim.uppercase placeholder="去空格+大写" />
</template>
```

## 三、注意事项与常见陷阱
- 修饰符在 `binding.modifiers` 中是键为 `true` 的对象
- 数字修饰符会变成对象的键：`.1000` → `{ 1000: true }`
- 修饰符和 arg 可以组合使用
- 命名要语义化，让使用指令的人一目了然
