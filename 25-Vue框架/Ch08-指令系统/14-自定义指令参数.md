# 自定义指令参数

## 一、概念说明
`binding` 对象包含指令的**参数、值和修饰符**等信息。通过 `binding.value` 获取指令值，`binding.arg` 获取指令参数。

## 二、具体用法

### 2.1 binding 对象属性
```js
// v-my-directive:foo.bar="value"
// binding 对象:
{
  value: 'value',        // 指令的值
  oldValue: 'oldValue',  // 上一次的值（updated 中）
  arg: 'foo',            // 参数（冒号后的部分）
  modifiers: { bar: true },  // 修饰符
  instance: null,        // 使用该指令的组件实例
  dir: {}                // 指令定义对象
}
```

### 2.2 使用 binding.value
```vue
<script setup>
const vColor = {
  mounted: (el, binding) => {
    el.style.color = binding.value
  },
  updated: (el, binding) => {
    el.style.color = binding.value
  }
}
</script>

<template>
  <p v-color="textColor">彩色文字</p>
</template>
<script setup>
import { ref } from 'vue'
const textColor = ref('blue')
</script>
```

### 2.3 使用 binding.arg（动态属性名）
```vue
<script setup>
const vStyle = {
  mounted: (el, binding) => {
    // binding.arg = 'color' 或 'fontSize' 等
    el.style[binding.arg] = binding.value
  },
  updated: (el, binding) => {
    el.style[binding.arg] = binding.value
  }
}
</script>

<template>
  <p v-style:color="'red'">红色</p>
  <p v-style:fontSize="'20px'">大字</p>
  <p v-style:[dynamicProp]="'blue'">动态属性</p>
</template>
<script setup>
import { ref } from 'vue'
const dynamicProp = ref('color')
</script>
```

### 2.4 对象值
```vue
<script setup>
const vStyle = {
  mounted: (el, binding) => {
    Object.entries(binding.value).forEach(([prop, val]) => {
      el.style[prop] = val
    })
  }
}
</script>

<template>
  <p v-style="{ color: 'red', fontSize: '18px', fontWeight: 'bold' }">样式</p>
</template>
```

## 三、注意事项与常见陷阱
- `binding.value` 可以是任何类型（字符串、数字、对象、函数）
- `binding.arg` 在动态参数 `:[arg]` 中是响应式的
- `binding.oldValue` 只在 `updated` 钩子中有意义
- 不要在指令中直接修改 `binding` 对象
