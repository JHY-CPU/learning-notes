# 指令系统概述

## 一、概念说明
Vue 指令是带有 `v-` 前缀的特殊 attribute，用于在模板中对 DOM 元素应用**响应式行为**。Vue 提供了丰富的内置指令，也支持创建自定义指令。

## 二、指令分类

### 2.1 内置指令列表
```
模板语法: v-bind, v-on, v-model, v-slot
条件渲染: v-if, v-else-if, v-else, v-show
列表渲染: v-for
一次性: v-once, v-memo
其他: v-pre, v-cloak
```

### 2.2 指令基本语法
```vue
<template>
  <!-- 完整语法 -->
  <div v-directive:arg.modifier="value"></div>

  <!-- 示例 -->
  <a v-bind:href="url">链接</a>
  <a :href="url">缩写</a>

  <button v-on:click="handler">按钮</button>
  <button @click="handler">缩写</button>
</template>
```

### 2.3 自定义指令
```vue
<script setup>
// 注册自定义指令
const vFocus = {
  mounted: (el) => el.focus()
}
</script>

<template>
  <input v-focus />
</template>
```

### 2.4 指令的生命周期钩子
```js
const myDirective = {
  created(el, binding, vnode, prevVnode) {},
  beforeMount(el, binding, vnode, prevVnode) {},
  mounted(el, binding, vnode, prevVnode) {},
  beforeUpdate(el, binding, vnode, prevVnode) {},
  updated(el, binding, vnode, prevVnode) {},
  beforeUnmount(el, binding, vnode, prevVnode) {},
  unmounted(el, binding, vnode, prevVnode) {}
}
```

## 三、注意事项与常见陷阱
- 指令应在 DOM 操作层面工作，复杂逻辑用组件
- 每个指令的值只能是单个表达式
- 自定义指令在 `<script setup>` 中使用小写 v 前缀命名
- 全局注册用 `app.directive()`，局部注册用变量

## 四、指令的参数与修饰符

### 4.1 参数（argument）
```vue
<template>
  <!-- : 后面是参数 -->
  <a v-bind:href="url">链接</a>
  <button v-on:click="handle">按钮</button>

  <!-- 动态参数 -->
  <a v-bind:[attributeName]="value">动态属性</a>
  <button v-on:[eventName]="handler">动态事件</button>
</template>
```

### 4.2 修饰符（modifier）
```vue
<template>
  <!-- . 后面是修饰符 -->
  <form @submit.prevent="onSubmit">阻止默认</form>
  <div @click.stop="handle">阻止冒泡</div>
  <input @keyup.enter="submit">按键修饰符</div>
  <input v-model.lazy="value">延迟更新</div>
</template>
```

### 4.3 指令的完整语法结构
```
v-directive:argument.modifier1.modifier2="expression"
    │           │        │              │
    │           │        │              └── JavaScript 表达式
    │           │        └── 修饰符（可多个）
    │           └── 参数（可选）
    └── 指令名
```

## 五、指令 vs 过滤器 vs 方法

| 特性 | 指令 | 方法 | computed |
|------|------|------|----------|
| 目的 | DOM 操作 | 逻辑复用 | 派生数据 |
| 返回值 | 无 | 可有 | 有 |
| 缓存 | 无 | 无 | 有 |
| 适用 | 底层 DOM | 通用逻辑 | 数据计算 |
