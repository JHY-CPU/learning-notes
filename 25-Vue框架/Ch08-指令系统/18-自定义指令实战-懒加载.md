# 自定义指令实战 - 懒加载

## 一、概念说明
`v-lazy` 用于实现图片**懒加载**：图片进入可视区域时才加载，减少初始加载时间和带宽消耗。利用 IntersectionObserver API 实现。

## 二、具体用法

### 2.1 基础懒加载指令
```js
// directives/lazy.js
export const lazyDirective = {
  mounted(el, binding) {
    const defaultSrc = 'data:image/gif;base64,R0lGOD...'  // 占位图
    el.src = defaultSrc

    const observer = new IntersectionObserver((entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          // 进入可视区域，加载真实图片
          el.src = binding.value
          el.onload = () => el.classList.add('loaded')
          observer.unobserve(el)
        }
      })
    }, { threshold: 0.1 })

    observer.observe(el)
    el._observer = observer
  },
  unmounted(el) {
    el._observer?.disconnect()
  }
}
```

### 2.2 使用指令
```vue
<template>
  <div class="gallery">
    <img v-lazy="item.url" v-for="item in images" :key="item.id" />
  </div>
</template>
<script setup>
import { ref } from 'vue'
const images = ref([
  { id: 1, url: '/images/photo1.jpg' },
  { id: 2, url: '/images/photo2.jpg' },
  { id: 3, url: '/images/photo3.jpg' }
])
</script>
```

### 2.3 支持背景图懒加载
```js
export const lazyBgDirective = {
  mounted(el, binding) {
    const observer = new IntersectionObserver((entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          el.style.backgroundImage = `url(${binding.value})`
          observer.unobserve(el)
        }
      })
    })

    observer.observe(el)
    el._observer = observer
  },
  unmounted(el) {
    el._observer?.disconnect()
  }
}
```

```vue
<template>
  <div v-lazy-bg="'/images/hero.jpg'" class="hero-section"></div>
</template>
```

### 2.4 加载失败处理
```js
el.onerror = () => {
  el.src = '/images/fallback.png'  // 加载失败显示备用图
}
```

## 三、注意事项与常见陷阱
- **务必在 `unmounted` 中调用 `observer.disconnect()`**，防止内存泄漏
- IntersectionObserver 的 threshold 设置为 0.1 表示 10% 可见时触发
- 不支持 IntersectionObserver 的浏览器需要 polyfill
- 生产环境建议使用 vue-lazyload 等成熟库
