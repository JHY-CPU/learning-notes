# 路由过渡动画

## 一、概念说明

在 Vue Router 中，路由切换本质上是 `<RouterView>` 中组件的替换。将 `<RouterView>` 包裹在 `<Transition>` 中，即可为路由切换添加过渡动画。

```vue
<!-- App.vue -->
<script setup>
</script>

<template>
  <nav>
    <RouterLink to="/">首页</RouterLink>
    <RouterLink to="/about">关于</RouterLink>
  </nav>
  <RouterView v-slot="{ Component }">
    <Transition name="route" mode="out-in">
      <component :is="Component" />
    </Transition>
  </RouterView>
</template>

<style>
.route-enter-active { transition: all 0.3s ease-out; }
.route-leave-active { transition: all 0.2s ease-in; }
.route-enter-from { opacity: 0; transform: translateY(20px); }
.route-leave-to { opacity: 0; transform: translateY(-20px); }
</style>
```

## 二、具体用法

### 2.1 使用插槽获取组件

必须使用 `<RouterView v-slot="{ Component }">` 获取动态组件，再通过 `<component :is="Component">` 渲染。

### 2.2 不同路由不同动画

```vue
<script setup>
import { useRoute } from 'vue-router'
const route = useRoute()
</script>

<template>
  <RouterView v-slot="{ Component }">
    <Transition :name="route.meta.transition || 'fade'" mode="out-in">
      <component :is="Component" />
    </Transition>
  </RouterView>
</template>
```

```js
// router/index.js
const routes = [
  { path: '/', component: Home, meta: { transition: 'slide-left' } },
  { path: '/about', component: About, meta: { transition: 'slide-right' } }
]
```

### 2.3 KeepAlive 结合

```vue
<RouterView v-slot="{ Component }">
  <Transition name="fade" mode="out-in">
    <KeepAlive>
      <component :is="Component" />
    </KeepAlive>
  </Transition>
</RouterView>
```

## 四、前进后退方向动画

```vue
<script setup>
import { ref } from 'vue'
import { useRouter } from 'vue-router'

const router = useRouter()
const transitionName = ref('slide-left')

// 根据路由层级判断方向
router.beforeEach((to, from) => {
  const toDepth = to.path.split('/').length
  const fromDepth = from.path.split('/').length
  transitionName.value = toDepth > fromDepth ? 'slide-left' : 'slide-right'
})
</script>

<template>
  <RouterView v-slot="{ Component }">
    <Transition :name="transitionName" mode="out-in">
      <component :is="Component" />
    </Transition>
  </RouterView>
</template>

<style>
.slide-left-enter-active, .slide-left-leave-active,
.slide-right-enter-active, .slide-right-leave-active {
  transition: all 0.3s ease;
}
.slide-left-enter-from { transform: translateX(100%); opacity: 0; }
.slide-left-leave-to { transform: translateX(-100%); opacity: 0; }
.slide-right-enter-from { transform: translateX(-100%); opacity: 0; }
.slide-right-leave-to { transform: translateX(100%); opacity: 0; }
</style>
```

## 五、结合 KeepAlive + Transition

```vue
<template>
  <RouterView v-slot="{ Component, route }">
    <Transition :name="route.meta.transition || 'fade'" mode="out-in">
      <KeepAlive :include="cachedViews">
        <component :is="Component" :key="route.path" />
      </KeepAlive>
    </Transition>
  </RouterView>
</template>
```

> `KeepAlive` 必须放在 `Transition` 内部，`component` 外部。`:key="route.path"` 确保不同路由参数的同一组件也能正确更新。

## 六、每个路由独立动画

```js
// router/index.js
const routes = [
  { path: '/', component: Home, meta: { transition: 'fade' } },
  { path: '/list', component: List, meta: { transition: 'slide-up' } },
  { path: '/detail/:id', component: Detail, meta: { transition: 'slide-left' } },
  { path: '/settings', component: Settings, meta: { transition: 'zoom' } }
]
```

## 三、注意事项与常见陷阱

- `<RouterView>` 不能直接放在 `<Transition>` 中，必须通过插槽获取组件
- 使用 `mode="out-in"` 防止新旧页面同时显示
- `<KeepAlive>` 要放在 `<Transition>` 内部、`<component>` 外部
- 前进后退方向动画需要在 `beforeEach` 守卫中动态设置
- 路由层级判断方向可能不准确，可通过 `meta.direction` 手动标记
