# 自定义过渡类名

## 一、概念说明

Vue 允许通过 `enter-from-class`、`enter-active-class` 等属性自定义过渡类名。这在使用第三方 CSS 动画库（如 Animate.css）时非常有用，因为这些库使用自己的类名体系。

```vue
<script setup>
import { ref } from 'vue'
const show = ref(true)
</script>

<template>
  <button @click="show = !show">切换</button>
  <Transition
    enter-from-class="fade-in-start"
    enter-active-class="fade-in-active"
    enter-to-class="fade-in-end"
    leave-from-class="fade-out-start"
    leave-active-class="fade-out-active"
    leave-to-class="fade-out-end"
  >
    <div v-if="show">自定义类名</div>
  </Transition>
</template>
```

## 二、具体用法

### 2.1 可用属性列表

| 属性 | 说明 |
|------|------|
| `enter-from-class` | 替代 `{name}-enter-from` |
| `enter-active-class` | 替代 `{name}-enter-active` |
| `enter-to-class` | 替代 `{name}-enter-to` |
| `leave-from-class` | 替代 `{name}-leave-from` |
| `leave-active-class` | 替代 `{name}-leave-active` |
| `leave-to-class` | 替代 `{name}-leave-to` |

### 2.2 结合 Animate.css

```vue
<template>
  <Transition
    enter-active-class="animate__animated animate__bounceIn"
    leave-active-class="animate__animated animate__bounceOut"
  >
    <div v-if="show">Animate.css 效果</div>
  </Transition>
</template>
```

### 2.3 结合自定义样式库

```vue
<Transition
  enter-active-class="custom-enter"
  leave-active-class="custom-leave"
>
  <Component v-if="show" />
</Transition>
```

## 四、结合第三方库的完整示例

```vue
<script setup>
import { ref } from 'vue'
import 'animate.css'

const show = ref(true)
const animationType = ref('bounce')
</script>

<template>
  <select v-model="animationType">
    <option value="bounce">弹跳</option>
    <option value="fade">淡入</option>
    <option value="slide">滑入</option>
    <option value="flip">翻转</option>
  </select>
  <button @click="show = !show">切换</button>

  <Transition
    :enter-active-class="`animate__animated animate__${animationType}In`"
    :leave-active-class="`animate__animated animate__${animationType}Out`"
  >
    <div v-if="show" class="box">动态动画效果</div>
  </Transition>
</template>
```

## 五、enter-class vs enter-from-class 区别

| 属性 | 何时添加 | 何时移除 | 作用 |
|------|---------|---------|------|
| `enter-from-class` | 元素插入前 | 元素插入后下一帧 | 起始状态 |
| `enter-active-class` | 元素插入前 | 动画完成时 | 过渡/动画定义 |
| `enter-to-class` | 元素插入后下一帧 | 动画完成时 | 结束状态 |

Vue 的处理流程：
1. 添加 `active-class` 和 `from-class`
2. 下一帧：移除 `from-class`，添加 `to-class`
3. 动画结束：移除 `active-class` 和 `to-class`

## 六、JavaScript 钩子配合自定义类名

```vue
<Transition
  enter-active-class="animate__animated animate__bounceIn"
  leave-active-class="animate__animated animate__bounceOut"
  @after-enter="onAfterEnter"
  @after-leave="onAfterLeave"
>
  <div v-if="show">内容</div>
</Transition>
```

> 自定义类名和 JS 钩子可以同时使用。CSS 类名负责视觉效果，JS 钩子处理业务逻辑。

## 三、注意事项与常见陷阱

- 自定义类名会**完全替换**默认类名，而非合并
- 设置了自定义类名后，`name` 属性对应的默认类名将不再生效
- 确保自定义类名中的 `active` 类包含正确的 `transition` 或 `animation` 声明
- 使用 `:enter-active-class` 动态绑定可以实现切换不同动画效果
- 第三方库的 `animated` 基础类不要遗漏
