# 自定义过渡类名

## 一、概念说明

Vue 允许通过 `enter-from-class`、`enter-active-class` 等属性自定义过渡类名。这在使用第三方 CSS 动画库（如 Animate.css）时非常有用，因为这些库使用自己的类名体系。

```vue
<script setup>
import { ref } from 'vue'
const show = ref(true)
</script>

<template>
  <button @click="show = !show">切换</button>
  <Transition
    enter-from-class="fade-in-start"
    enter-active-class="fade-in-active"
    enter-to-class="fade-in-end"
    leave-from-class="fade-out-start"
    leave-active-class="fade-out-active"
    leave-to-class="fade-out-end"
  >
    <div v-if="show">自定义类名</div>
  </Transition>
</template>
```

## 二、具体用法

### 2.1 可用属性列表

| 属性 | 说明 |
|------|------|
| `enter-from-class` | 替代 `{name}-enter-from` |
| `enter-active-class` | 替代 `{name}-enter-active` |
| `enter-to-class` | 替代 `{name}-enter-to` |
| `leave-from-class` | 替代 `{name}-leave-from` |
| `leave-active-class` | 替代 `{name}-leave-active` |
| `leave-to-class` | 替代 `{name}-leave-to` |

### 2.2 结合 Animate.css

```vue
<template>
  <Transition
    enter-active-class="animate__animated animate__bounceIn"
    leave-active-class="animate__animated animate__bounceOut"
  >
    <div v-if="show">Animate.css 效果</div>
  </Transition>
</template>
```

### 2.3 结合自定义样式库

```vue
<Transition
  enter-active-class="custom-enter"
  leave-active-class="custom-leave"
>
  <Component v-if="show" />
</Transition>
```

## 三、注意事项与常见陷阱

- 自定义类名会**完全替换**默认类名，而非合并
- 设置了自定义类名后，`name` 属性对应的默认类名将不再生效
- 确保自定义类名中的 `active` 类包含正确的 `transition` 或 `animation` 声明
