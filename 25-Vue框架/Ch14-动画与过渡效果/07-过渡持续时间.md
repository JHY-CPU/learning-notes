# 过渡持续时间

## 一、概念说明

Vue 默认通过监听 `transitionend` 或 `animationend` 事件来判断过渡是否结束。但某些场景下（如宽高变化的 `height` 过渡），事件可能提前触发。`duration` 属性允许手动指定过渡持续时间。

```vue
<script setup>
import { ref } from 'vue'
const show = ref(true)
</script>

<template>
  <button @click="show = !show">切换</button>
  <Transition name="fade" :duration="500">
    <div v-if="show">500ms 淡入淡出</div>
  </Transition>
</template>

<style>
.fade-enter-active, .fade-leave-active {
  transition: opacity 0.5s ease;
}
.fade-enter-from, .fade-leave-to { opacity: 0; }
</style>
```

## 二、具体用法

### 2.1 统一持续时间

```vue
<!-- 进入和离开都是 500ms -->
<Transition :duration="500">
  <div v-if="show">内容</div>
</Transition>
```

### 2.2 分别设置进入/离开时长

```vue
<!-- 进入 800ms，离开 300ms -->
<Transition :duration="{ enter: 800, leave: 300 }">
  <div v-if="show">内容</div>
</Transition>
```

### 2.3 何时需要设置

- CSS 中使用了 `animation` 但没有正确触发 `animationend`
- 过渡涉及多个子元素的级联动画
- 需要精确控制进入和离开的时间差

## 三、注意事项与常见陷阱

- `duration` 只影响 Vue 等待的时间，不影响 CSS 中定义的实际动画时长
- 如果 `duration` 小于 CSS 动画实际时长，动画会被提前截断
- 建议 `duration` 值与 CSS 中的过渡/动画时长保持一致
