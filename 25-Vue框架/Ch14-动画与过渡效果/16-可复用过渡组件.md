# 可复用过渡组件

## 一、概念说明

将过渡效果封装为可复用组件，可以在项目中多处使用统一的动画效果，减少重复代码。可复用过渡组件通常接受插槽（slot）内容和配置参数。

```vue
<!-- FadeTransition.vue -->
<script setup>
defineProps({
  duration: { type: Number, default: 300 },
  mode: { type: String, default: 'out-in' }
})
</script>

<template>
  <Transition
    name="fade"
    :mode="mode"
    :style="{ '--duration': duration + 'ms' }"
  >
    <slot />
  </Transition>
</template>

<style scoped>
.fade-enter-active,
.fade-leave-active {
  transition: opacity var(--duration) ease;
}
.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>
```

```vue
<!-- 使用方式 -->
<script setup>
import FadeTransition from './FadeTransition.vue'
import { ref } from 'vue'
const show = ref(true)
</script>

<template>
  <FadeTransition :duration="500" mode="out-in">
    <div v-if="show" key="a">内容 A</div>
    <div v-else key="b">内容 B</div>
  </FadeTransition>
</template>
```

## 二、具体用法

### 2.1 滑动过渡组件

```vue
<!-- SlideTransition.vue -->
<template>
  <Transition name="slide" mode="out-in">
    <slot />
  </Transition>
</template>
<style scoped>
.slide-enter-active, .slide-leave-active { transition: all 0.3s; }
.slide-enter-from { transform: translateX(100%); opacity: 0; }
.slide-leave-to { transform: translateX(-100%); opacity: 0; }
</style>
```

### 2.2 缩放过渡组件

```vue
<!-- ScaleTransition.vue -->
<template>
  <Transition name="scale">
    <slot />
  </Transition>
</template>
<style scoped>
.scale-enter-active, .scale-leave-active { transition: all 0.3s ease; }
.scale-enter-from, .scale-leave-to { transform: scale(0.8); opacity: 0; }
</style>
```

## 三、注意事项与常见陷阱

- 使用 `<slot />` 传递内容，确保 `<Transition>` 内只有一个根元素
- `scoped` 样式中的过渡类名可能不生效，需使用 `:deep()` 选择器
- 传递 `duration` 等配置通过 CSS 变量是最简洁的方式
