# 动态过渡

## 一、概念说明

动态过渡是指通过动态绑定 `name` 属性或使用 `:css` 属性，让组件根据运行时条件切换不同的过渡效果。这在需要根据用户操作或路由方向改变动画时非常有用。

```vue
<script setup>
import { ref, computed } from 'vue'
const direction = ref('left')
const show = ref(true)

const transitionName = computed(() => `slide-${direction.value}`)

function toggle(dir) {
  direction.value = dir
  show.value = !show.value
}
</script>

<template>
  <button @click="toggle('left')">左滑</button>
  <button @click="toggle('right')">右滑</button>
  <button @click="toggle('up')">上滑</button>

  <Transition :name="transitionName" mode="out-in">
    <div v-if="show" key="a">内容 A</div>
    <div v-else key="b">内容 B</div>
  </Transition>
</template>

<style>
.slide-left-enter-active, .slide-left-leave-active { transition: all 0.3s; }
.slide-left-enter-from { transform: translateX(-100%); opacity: 0; }
.slide-left-leave-to { transform: translateX(100%); opacity: 0; }

.slide-right-enter-active, .slide-right-leave-active { transition: all 0.3s; }
.slide-right-enter-from { transform: translateX(100%); opacity: 0; }
.slide-right-leave-to { transform: translateX(-100%); opacity: 0; }

.slide-up-enter-active, .slide-up-leave-active { transition: all 0.3s; }
.slide-up-enter-from { transform: translateY(100%); opacity: 0; }
.slide-up-leave-to { transform: translateY(-100%); opacity: 0; }
</style>
```

## 二、具体用法

### 2.1 动态 name

```vue
<Transition :name="isSlide ? 'slide' : 'fade'">
  <div v-if="show">内容</div>
</Transition>
```

### 2.2 :css="false" 纯 JS 动画

```vue
<!-- 告诉 Vue 跳过 CSS 检测，纯用 JavaScript 钩子 -->
<Transition :css="false" @enter="onEnter" @leave="onLeave">
  <div v-if="show">纯 JS 动画</div>
</Transition>
```

### 2.3 结合路由方向

```js
// 判断路由方向决定动画方向
const transitionName = computed(() => {
  return route.meta.depth > prevDepth ? 'slide-left' : 'slide-right'
})
```

## 三、注意事项与常见陷阱

- 动态切换 `name` 时，确保对应的 CSS 类名都已定义
- `:css="false"` 时，Vue 不会添加任何 CSS 类，所有动画必须在 JS 钩子中完成
- 频繁切换过渡类型可能导致视觉混乱，建议添加防抖
