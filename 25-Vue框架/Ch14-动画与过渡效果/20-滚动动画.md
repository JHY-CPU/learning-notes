# 滚动动画

## 一、概念说明

滚动动画是指元素进入视口（viewport）时触发的动画效果。使用 Intersection Observer API 可以高效地检测元素是否进入视口，避免传统 `scroll` 事件的性能问题。

```vue
<script setup>
import { ref, onMounted } from 'vue'
const cards = ref([
  { id: 1, visible: false },
  { id: 2, visible: false },
  { id: 3, visible: false },
])

onMounted(() => {
  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        const id = Number(entry.target.dataset.id)
        const card = cards.value.find(c => c.id === id)
        if (card) card.visible = true
      }
    })
  }, { threshold: 0.2 })

  document.querySelectorAll('.scroll-card').forEach(el => {
    observer.observe(el)
  })
})
</script>

<template>
  <div
    v-for="card in cards"
    :key="card.id"
    :data-id="card.id"
    class="scroll-card"
    :class="{ 'is-visible': card.visible }"
  >
    卡片 {{ card.id }}
  </div>
</template>

<style>
.scroll-card {
  opacity: 0;
  transform: translateY(40px);
  transition: all 0.6s ease-out;
}
.scroll-card.is-visible {
  opacity: 1;
  transform: translateY(0);
}
</style>
```

## 二、具体用法

### 2.1 Intersection Observer 配置

```js
const observer = new IntersectionObserver(callback, {
  root: null,        // 视口为根
  rootMargin: '0px', // 边距
  threshold: 0.1     // 10% 可见时触发
})
```

### 2.2 封装为组合式函数

```js
// useScrollReveal.js
import { ref, onMounted, onUnmounted } from 'vue'

export function useScrollReveal() {
  const isVisible = ref(false)
  const elRef = ref(null)
  let observer = null

  onMounted(() => {
    observer = new IntersectionObserver(([entry]) => {
      if (entry.isIntersecting) isVisible.value = true
    }, { threshold: 0.2 })
    if (elRef.value) observer.observe(elRef.value)
  })

  onUnmounted(() => observer?.disconnect())

  return { isVisible, elRef }
}
```

## 三、注意事项与常见陷阱

- 记得在 `onUnmounted` 中调用 `observer.disconnect()` 清理
- `threshold` 设为 0 只要 1 像素可见就触发，设为 1 则需完全可见
- Intersection Observer 兼容性良好，但 IE 不支持
