# 初始渲染动画

## 一、概念说明

默认情况下，`<Transition>` 只在元素的条件状态变化时触发动画。如果希望元素在**首次渲染**时也播放动画，需要添加 `appear` 属性。

```vue
<script setup>
import { ref } from 'vue'
const show = ref(true)
</script>

<template>
  <Transition name="fade" appear>
    <div v-if="show">首次渲染时也会淡入</div>
  </Transition>
</template>

<style>
.fade-enter-active { transition: opacity 0.5s ease; }
.fade-enter-from { opacity: 0; }
</style>
```

## 二、具体用法

### 2.1 appear 属性

```vue
<!-- 基本用法 -->
<Transition name="slide" appear>
  <div v-if="visible">页面加载时就会滑入</div>
</Transition>
```

### 2.2 appear 配合 mode

```vue
<!-- 首次渲染动画 + 离开/进入模式 -->
<Transition name="fade" mode="out-in" appear>
  <component :is="currentView" />
</Transition>
```

### 2.3 appear 使用独立类名

初始渲染时使用与进入相同的类名体系：`*-enter-from`、`*-enter-active`、`*-enter-to`。

## 三、注意事项与常见陷阱

- `appear` 只在**首次渲染**时生效，后续的 `v-if` 切换使用正常的进入/离开类名
- 如果组件一开始就满足 `v-if="true"`，不加 `appear` 则无任何动画
- `appear` 对 `v-show` 同样有效
