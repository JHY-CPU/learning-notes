# 状态过渡动画

## 一、概念说明

除了 DOM 元素的进入/离开动画，Vue 也能为**数字、颜色、对象属性**等数据的变化添加平滑过渡效果。这通常通过侦听器（watch）或计算属性结合 JavaScript 动画库实现。

```vue
<script setup>
import { ref, watch } from 'vue'

const number = ref(0)
const displayNumber = ref(0)

watch(number, (newVal, oldVal) => {
  const duration = 500
  const start = performance.now()
  function animate(now) {
    const progress = Math.min((now - start) / duration, 1)
    displayNumber.value = Math.round(oldVal + (newVal - oldVal) * progress)
    if (progress < 1) requestAnimationFrame(animate)
  }
  requestAnimationFrame(animate)
})
</script>

<template>
  <div class="display">{{ displayNumber }}</div>
  <button @click="number += 100">+100</button>
  <button @click="number -= 50">-50</button>
  <button @click="number = 0">重置</button>
</template>
```

## 二、具体用法

### 2.1 数字动画过渡

使用 `requestAnimationFrame` 在新旧值之间插值，实现平滑数字跳动效果。

### 2.2 颜色过渡

```js
const color = ref('#42b883')
const displayColor = ref('#42b883')

// 通过 watch 监听颜色变化，使用 Lerp 插值实现颜色渐变
watch(color, (newColor) => {
  displayColor.value = newColor // 简化示例，可结合颜色插值库
})
```

### 2.3 对象属性过渡

```js
const style = ref({ width: 100, height: 100 })
// 使用 watch 逐个属性做插值过渡
```

## 四、颜色过渡动画

```vue
<script setup>
import { ref, watch } from 'vue'

const targetColor = ref('#42b883')
const displayColor = ref('#42b883')

// 颜色插值
function lerpColor(a, b, t) {
  const parseHex = (hex) => {
    const r = parseInt(hex.slice(1, 3), 16)
    const g = parseInt(hex.slice(3, 5), 16)
    const bl = parseInt(hex.slice(5, 7), 16)
    return [r, g, bl]
  }
  const [ar, ag, ab] = parseHex(a)
  const [br, bg, bb] = parseHex(b)
  const r = Math.round(ar + (br - ar) * t)
  const g = Math.round(ag + (bg - ag) * t)
  const bl = Math.round(ab + (bb - ab) * t)
  return `#${r.toString(16).padStart(2, '0')}${g.toString(16).padStart(2, '0')}${bl.toString(16).padStart(2, '0')}`
}

watch(targetColor, (newVal, oldVal) => {
  const duration = 600
  const start = performance.now()
  function animate(now) {
    const progress = Math.min((now - start) / duration, 1)
    displayColor.value = lerpColor(oldVal, newVal, progress)
    if (progress < 1) requestAnimationFrame(animate)
  }
  requestAnimationFrame(animate)
})
</script>

<template>
  <div :style="{ backgroundColor: displayColor, color: 'white', padding: '20px' }">
    当前颜色: {{ displayColor }}
  </div>
  <button @click="targetColor = '#e74c3c'">红色</button>
  <button @click="targetColor = '#3498db'">蓝色</button>
  <button @click="targetColor = '#42b883'">绿色</button>
</template>
```

## 五、SVG 属性过渡

```vue
<script setup>
import { ref, watch } from 'vue'

const progress = ref(0)
const displayProgress = ref(0)

watch(progress, (newVal, oldVal) => {
  const duration = 400
  const start = performance.now()
  function animate(now) {
    const t = Math.min((now - start) / duration, 1)
    displayProgress.value = oldVal + (newVal - oldVal) * t
    if (t < 1) requestAnimationFrame(animate)
  }
  requestAnimationFrame(animate)
})

const dashOffset = computed(() => 314 - (314 * displayProgress.value / 100))
</script>

<template>
  <svg width="120" height="120">
    <circle cx="60" cy="60" r="50" fill="none" stroke="#eee" stroke-width="8" />
    <circle
      cx="60" cy="60" r="50" fill="none"
      stroke="#42b883" stroke-width="8"
      :stroke-dasharray="314"
      :stroke-dashoffset="dashOffset"
      transform="rotate(-90 60 60)"
    />
    <text x="60" y="65" text-anchor="middle">{{ Math.round(displayProgress) }}%</text>
  </svg>
  <button @click="progress = 75">75%</button>
  <button @click="progress = 100">100%</button>
</template>
```

## 三、注意事项与常见陷阱

- 复杂的颜色过渡建议使用专门的库（如 GSAP、chroma.js）
- 数字精度问题：浮点数插值可能导致小数位误差
- `requestAnimationFrame` 的频率为 60fps，确保动画足够流畅
- 状态动画需要在组件卸载时取消 `requestAnimationFrame`（使用 `onUnmounted`）
- 数字过渡要考虑格式化（如千分位、货币符号）
