# 状态过渡动画

## 一、概念说明

除了 DOM 元素的进入/离开动画，Vue 也能为**数字、颜色、对象属性**等数据的变化添加平滑过渡效果。这通常通过侦听器（watch）或计算属性结合 JavaScript 动画库实现。

```vue
<script setup>
import { ref, watch } from 'vue'

const number = ref(0)
const displayNumber = ref(0)

watch(number, (newVal, oldVal) => {
  const duration = 500
  const start = performance.now()
  function animate(now) {
    const progress = Math.min((now - start) / duration, 1)
    displayNumber.value = Math.round(oldVal + (newVal - oldVal) * progress)
    if (progress < 1) requestAnimationFrame(animate)
  }
  requestAnimationFrame(animate)
})
</script>

<template>
  <div class="display">{{ displayNumber }}</div>
  <button @click="number += 100">+100</button>
  <button @click="number -= 50">-50</button>
  <button @click="number = 0">重置</button>
</template>
```

## 二、具体用法

### 2.1 数字动画过渡

使用 `requestAnimationFrame` 在新旧值之间插值，实现平滑数字跳动效果。

### 2.2 颜色过渡

```js
const color = ref('#42b883')
const displayColor = ref('#42b883')

// 通过 watch 监听颜色变化，使用 Lerp 插值实现颜色渐变
watch(color, (newColor) => {
  displayColor.value = newColor // 简化示例，可结合颜色插值库
})
```

### 2.3 对象属性过渡

```js
const style = ref({ width: 100, height: 100 })
// 使用 watch 逐个属性做插值过渡
```

## 三、注意事项与常见陷阱

- 复杂的颜色过渡建议使用专门的库（如 GSAP、chroma.js）
- 数字精度问题：浮点数插值可能导致小数位误差
- `requestAnimationFrame` 的频率为 60fps，确保动画足够流畅
