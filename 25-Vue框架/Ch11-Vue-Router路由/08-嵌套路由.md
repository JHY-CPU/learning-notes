# 嵌套路由

## 一、概念说明

嵌套路由用于在页面中创建嵌套的UI结构。父路由组件包含`<RouterView>`，子路由组件在此渲染。

```js
const routes = [
  {
    path: '/dashboard',
    component: Dashboard,
    children: [
      { path: '', component: DashboardHome },
      { path: 'profile', component: Profile },
      { path: 'settings', component: Settings }
    ]
  }
]
```

```vue
<!-- Dashboard.vue -->
<template>
  <div class="dashboard">
    <aside>
      <RouterLink to="/dashboard">概览</RouterLink>
      <RouterLink to="/dashboard/profile">个人资料</RouterLink>
      <RouterLink to="/dashboard/settings">设置</RouterLink>
    </aside>
    <main>
      <RouterView />  <!-- 子路由渲染位置 -->
    </main>
  </div>
</template>
```

## 二、具体用法

### 多层嵌套

```js
const routes = [
  {
    path: '/admin',
    component: AdminLayout,
    children: [
      {
        path: 'users',
        component: UserManagement,
        children: [
          { path: '', component: UserList },
          { path: ':id', component: UserDetail }
        ]
      }
    ]
  }
]
// URL: /admin/users/123
```

### 默认子路由

```js
children: [
  { path: '', component: DefaultContent },     // /parent
  { path: 'child', component: ChildContent }   // /parent/child
]
```

## 三、注意事项与常见陷阱

1. 子路由的`path`不以`/`开头，会拼接父路径
2. 空路径`''`匹配父路径（默认子路由）
3. 每层嵌套需要对应的`<RouterView>`
4. 嵌套深度没有限制，但建议不超过3层
5. 父路由组件不显示内容时，给父路由一个重定向到默认子路由

## 四、实际应用场景

### 4.1 后台管理系统
```js
const routes = [
  {
    path: '/admin',
    component: () => import('@/views/admin/Layout.vue'),
    meta: { requiresAuth: true },
    children: [
      { path: '', redirect: '/admin/dashboard' },
      { path: 'dashboard', name: 'AdminDashboard', component: () => import('@/views/admin/Dashboard.vue') },
      { path: 'users', name: 'AdminUsers', component: () => import('@/views/admin/Users.vue') },
      {
        path: 'users/:id',
        name: 'AdminUserDetail',
        component: () => import('@/views/admin/UserDetail.vue'),
        props: true
      },
      { path: 'settings', name: 'AdminSettings', component: () => import('@/views/admin/Settings.vue') }
    ]
  }
]
```

### 4.2 用户中心
```js
const routes = [
  {
    path: '/user/:userId',
    component: () => import('@/views/user/Layout.vue'),
    props: true,
    children: [
      { path: '', name: 'UserProfile', component: () => import('@/views/user/Profile.vue') },
      { path: 'posts', name: 'UserPosts', component: () => import('@/views/user/Posts.vue') },
      { path: 'followers', name: 'UserFollowers', component: () => import('@/views/user/Followers.vue') }
    ]
  }
]
```

## 五、嵌套路由与 keep-alive

```vue
<template>
  <router-view v-slot="{ Component, route }">
    <keep-alive :include="cachedViews">
      <component :is="Component" :key="route.path" />
    </keep-alive>
  </router-view>
</template>

<script setup>
import { ref } from 'vue'
const cachedViews = ref(['UserList', 'AdminDashboard'])
</script>
```
