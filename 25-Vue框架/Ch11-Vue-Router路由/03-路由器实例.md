# 路由器实例

## 一、概念说明

`createRouter()`创建路由器实例，配置路由规则和历史模式。实例提供导航方法、守卫注册等功能。

```js
import { createRouter, createWebHistory } from 'vue-router'

const router = createRouter({
  history: createWebHistory(import.meta.env.BASE_URL),
  routes: [/* ... */],
  scrollBehavior(to, from, savedPosition) {
    return { top: 0 }
  }
})

// 全局前置守卫
router.beforeEach((to, from) => {
  console.log(`从 ${from.path} 导航到 ${to.path}`)
  return true
})

export default router
```

## 二、具体用法

### 路由器实例属性和方法

```js
// 获取当前路由
router.currentRoute  // Ref<RouteLocationNormalized>

// 导航方法
router.push('/about')           // 跳转（添加历史记录）
router.replace('/login')        // 替换（不添加历史记录）
router.go(-1)                   // 后退一步
router.back()                   // 后退
router.forward()                // 前进

// 注册守卫
router.beforeEach(guard)
router.beforeResolve(guard)
router.afterEach(hook)

// 动态添加路由
router.addRoute({ path: '/new', component: NewPage })
router.removeRoute('routeName')
router.hasRoute('routeName')
router.getRoutes()
```

### 在setup中使用

```vue
<script setup>
import { useRouter, useRoute } from 'vue-router'

const router = useRouter()
const route = useRoute()

const goHome = () => router.push('/')
const goBack = () => router.go(-1)

console.log('当前路径:', route.path)
console.log('路由参数:', route.params)
</script>
```

## 三、注意事项与常见陷阱

1. `router`是全局单例，`route`是响应式的当前路由信息
2. `router.push()`返回Promise，可等待导航完成
3. 在模板中可直接使用`$router`和`$route`（全局属性）
4. 选项式API通过`this.$router`访问，组合式API使用`useRouter()`
5. `createWebHistory()`参数是base路径，通常从环境变量获取
