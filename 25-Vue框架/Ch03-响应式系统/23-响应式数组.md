# 响应式数组

## 一、概念说明

Vue 3 的响应式数组支持所有变异方法（push、pop、splice 等）以及索引赋值和 length 修改。这是 Vue 3 Proxy 相比 Vue 2 的重要改进。

```vue
<script setup>
import { ref, reactive } from 'vue'

const items = ref(['苹果', '香蕉', '橙子'])
const list = reactive([1, 2, 3])

// 以下操作都是响应式的
items.value.push('葡萄')        // 添加
items.value.pop()               // 删除最后一个
items.value.splice(0, 1)        // 删除指定位置
items.value[0] = '西瓜'         // 索引赋值（Vue 3 支持！）
items.value.length = 0          // 清空数组
</script>
```

## 二、具体用法

### 2.1 变异方法

```vue
<script setup>
import { ref } from 'vue'

const list = ref([1, 2, 3, 4, 5])

// 变异方法（会改变原数组，触发更新）
list.value.push(6)           // [1,2,3,4,5,6]
list.value.pop()             // [1,2,3,4,5]
list.value.shift()           // [2,3,4,5]
list.value.unshift(0)        // [0,2,3,4,5]
list.value.splice(1, 1)      // [0,3,4,5]
list.value.reverse()         // [5,4,3,0]
list.value.sort((a, b) => a - b) // [0,3,4,5]
</script>
```

### 2.2 非变异方法（替换数组）

```vue
<script setup>
import { ref } from 'vue'

const list = ref([3, 1, 4, 1, 5])

// 非变异方法返回新数组，需要重新赋值
list.value = list.value.filter(n => n > 1)    // [3,4,1,5]
list.value = list.value.map(n => n * 2)       // [6,8,2,10]
list.value = list.value.concat([9, 2])         // [6,8,2,10,9,2]
list.value = [...list.value, ...[7, 8]]        // 展开运算符
</script>
```

### 2.3 v-for 与数组更新

```vue
<script setup>
import { ref } from 'vue'

const todos = ref([
  { id: 1, text: '学习 Vue', done: false },
  { id: 2, text: '写代码', done: false }
])

function toggleTodo(id) {
  const todo = todos.value.find(t => t.id === id)
  if (todo) todo.done = !todo.done
}
</script>

<template>
  <div v-for="todo in todos" :key="todo.id">
    <input type="checkbox" :checked="todo.done" @change="toggleTodo(todo.id)" />
    <span :class="{ done: todo.done }">{{ todo.text }}</span>
  </div>
</template>
```

## 三、注意事项与常见陷阱

- Vue 3 支持索引赋值（`arr[0] = x`），但 Vue 2 不支持
- 非变异方法需要重新赋值整个数组
- 使用 `:key` 绑定唯一 id 而非索引
- `sort()` 和 `reverse()` 会改变原数组，注意是否有其他引用
- 大型数组频繁操作考虑性能优化
