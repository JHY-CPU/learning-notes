# 动态参数

## 一、概念说明

Vue 允许在指令参数中使用 JavaScript 表达式，称为**动态参数**。语法为 `:[attributeName]` 或 `@[eventName]`，其中方括号内的值可以是任意表达式。

```vue
<script setup>
import { ref } from 'vue'

const attributeName = ref('href')
const url = ref('https://vuejs.org')
const eventName = ref('click')
</script>

<template>
  <a :[attributeName]="url">动态链接</a>
  <!-- 渲染为: <a href="https://vuejs.org">动态链接</a> -->

  <button @[eventName]="handleEvent">动态事件</button>
</template>
```

## 二、具体用法

### 2.1 动态属性名

```vue
<script setup>
import { ref } from 'vue'

const dynamicAttr = ref('title')
const attrValue = ref('鼠标悬停提示')
const attributeName = ref('placeholder')
const placeholderText = ref('请输入内容')
</script>

<template>
  <div :[dynamicAttr]="attrValue">鼠标悬停查看</div>
  <input :[attributeName]="placeholderText" />
</template>
```

### 2.2 动态事件名

```vue
<script setup>
import { ref } from 'vue'
const eventType = ref('mouseenter')

function handleEvent(event) {
  console.log('触发事件:', event.type)
}
</script>

<template>
  <div @[eventType]="handleEvent">动态事件监听</div>
</template>
```

### 2.3 动态插槽名

```vue
<script setup>
import { ref } from 'vue'
const slotName = ref('header')
</script>

<template>
  <MyComponent>
    <template #[slotName]>
      <h1>动态插槽内容</h1>
    </template>
  </MyComponent>
</template>
```

### 2.4 条件切换属性

```vue
<script setup>
import { ref, computed } from 'vue'
const isLink = ref(true)
const url = ref('https://vuejs.org')
const onClick = () => console.log('clicked')

const dynamicAttr = computed(() => isLink.value ? 'href' : 'title')
const dynamicValue = computed(() => isLink.value ? url.value : '不是链接')
</script>

<template>
  <div :[dynamicAttr]="dynamicValue">
    {{ isLink ? '点击跳转' : '仅提示' }}
  </div>
  <button @click="isLink = !isLink">切换模式</button>
</template>
```

### 2.5 动态样式属性

```vue
<script setup>
import { ref } from 'vue'
const prop = ref('backgroundColor')
const value = ref('#42b883')
</script>

<template>
  <div :style="{ [prop]: value }">动态背景色</div>
</template>
```

## 三、常见用例

### 3.1 可配置的输入组件

```vue
<script setup>
import { ref } from 'vue'

const props = defineProps({
  inputType: { type: String, default: 'text' },
  inputName: { type: String, default: 'field' }
})

const dynamicType = computed(() => props.inputType)
const dynamicName = computed(() => props.inputName)
</script>

<template>
  <input :type="dynamicType" :name="dynamicName" />
</template>
```

## 四、注意事项与常见陷阱

- 动态参数的表达式必须是**单个字符串**（或 `null`）
- 表达式结果为 `null` 时会移除绑定
- 动态参数不能有空格和引号，使用计算属性处理复杂情况
- 在 DOM 模板中，属性名会被强制转为小写，避免大小写敏感问题
- 动态参数与静态参数不能同时使用在同一属性上
- 建议使用计算属性包裹复杂逻辑，保持模板简洁
- 动态事件名的使用场景较少，通常直接绑定多个事件更清晰
