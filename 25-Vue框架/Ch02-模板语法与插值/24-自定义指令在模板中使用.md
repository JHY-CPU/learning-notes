# 自定义指令在模板中使用

## 一、概念说明

Vue 允许创建自定义指令来封装对 DOM 元素的底层操作。自定义指令以 `v-` 前缀使用，在模板中像内置指令（v-if、v-model）一样调用。

```vue
<script setup>
import { ref } from 'vue'

// 注册局部自定义指令
const vFocus = {
  mounted: (el) => el.focus()
}

const vHighlight = {
  mounted: (el, binding) => {
    el.style.backgroundColor = binding.value || 'yellow'
  }
}
</script>

<template>
  <!-- 使用自定义指令 -->
  <input v-focus placeholder="自动聚焦" />
  <p v-highlight="'#ffeb3b'">高亮文本</p>
</template>
```

## 二、具体用法

### 2.1 指令的生命周期钩子

```js
const myDirective = {
  // 绑定到元素前
  created(el, binding, vnode, prevVnode) {},

  // 绑定到父元素时
  beforeMount(el, binding) {},

  // 元素插入父 DOM 后
  mounted(el, binding) {},

  // 元素更新前
  beforeUpdate(el, binding) {},

  // 元素更新后
  updated(el, binding) {},

  // 元素卸载前
  beforeUnmount(el, binding) {},

  // 元素卸载后
  unmounted(el, binding) {}
}
```

### 2.2 指令参数和修饰符

```vue
<script setup>
const vPin = {
  mounted(el, binding) {
    // binding.value: 传入的值
    // binding.arg: 参数（: 后面的部分）
    // binding.modifiers: 修饰符对象
    el.style.position = binding.arg || 'fixed'
    el.style.top = binding.value + 'px'
    if (binding.modifiers.away) {
      el.style.right = '0'
    }
  }
}
</script>

<template>
  <div v-pin:fixed="20">固定在顶部 20px</div>
  <div v-pin.away="20">固定在右上角</div>
</template>
```

### 2.3 全局注册指令

```js
// main.js
app.directive('focus', {
  mounted: (el) => el.focus()
})

app.directive('click-outside', {
  mounted(el, binding) {
    el._clickOutside = (event) => {
      if (!el.contains(event.target)) {
        binding.value()
      }
    }
    document.addEventListener('click', el._clickOutside)
  },
  unmounted(el) {
    document.removeEventListener('click', el._clickOutside)
  }
})
```

## 三、注意事项与常见陷阱

- 自定义指令不能在 `<script setup>` 之外使用（全局注册除外）
- 指令的 `binding.value` 可以是任何类型
- 在 SSR 中指令不会执行（因为没有 DOM）
- 推荐优先使用组件而非指令封装逻辑
- 指令命名在模板中使用 kebab-case（`v-my-directive`）
