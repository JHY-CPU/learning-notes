# 文本插值

## 一、概念说明

**文本插值**是 Vue 中最基本的数据绑定方式，使用"Mustache"语法（双大括号 `{{ }}`）将响应式数据渲染到模板中。插值会自动追踪数据变化并更新 DOM。

```vue
<script setup>
import { ref } from 'vue'

const message = ref('你好，Vue 3！')
const count = ref(42)
const user = ref({ name: '张三' })
</script>

<template>
  <!-- 基本文本插值 -->
  <h1>{{ message }}</h1>

  <!-- 数字 -->
  <p>计数: {{ count }}</p>

  <!-- 表达式 -->
  <p>双倍: {{ count * 2 }}</p>
  <p>格式化: {{ count.toFixed(2) }}</p>

  <!-- 对象属性 -->
  <p>用户名: {{ user.name }}</p>
</template>
```

## 二、具体用法

### 2.1 JavaScript 表达式

```vue
<template>
  <!-- 算术运算 -->
  <p>{{ price * quantity }}</p>

  <!-- 三元表达式 -->
  <p>{{ isLoggedIn ? '欢迎回来' : '请登录' }}</p>

  <!-- 方法调用 -->
  <p>{{ message.toUpperCase() }}</p>

  <!-- 数组操作 -->
  <p>{{ items.join(', ') }}</p>

  <!-- 模板字面量 -->
  <p>{{ `你好，${name}！` }}</p>
</template>
```

### 2.2 v-once 一次性插值

```vue
<template>
  <!-- 数据变化后不再更新 -->
  <span v-once>{{ staticMessage }}</span>
</template>
```

### 2.3 v-text 等效写法

```vue
<template>
  <!-- 与 {{ }} 等效 -->
  <span v-text="message"></span>
  <!-- 等同于 -->
  <span>{{ message }}</span>
</template>
```

## 三、注意事项与常见陷阱

- `{{ }}` 内只能放**单个表达式**，不能放流程控制语句
- `undefined` 和 `null` 在插值中显示为空字符串
- 插值会在每次数据变化时重新计算，复杂表达式用 computed 优化
- 模板中自动解包 ref（不需要 `.value`）
- 不要在 `{{ }}` 中产生副作用（如修改响应式数据）
