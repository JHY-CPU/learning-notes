# 文本插值

## 一、概念说明

**文本插值**是 Vue 中最基本的数据绑定方式，使用"Mustache"语法（双大括号 `{{ }}`）将响应式数据渲染到模板中。插值会自动追踪数据变化并更新 DOM。

```vue
<script setup>
import { ref } from 'vue'

const message = ref('你好，Vue 3！')
const count = ref(42)
const user = ref({ name: '张三' })
</script>

<template>
  <h1>{{ message }}</h1>
  <p>计数: {{ count }}</p>
  <p>双倍: {{ count * 2 }}</p>
  <p>格式化: {{ count.toFixed(2) }}</p>
  <p>用户名: {{ user.name }}</p>
</template>
```

## 二、具体用法

### 2.1 JavaScript 表达式

```vue
<template>
  <!-- 算术运算 -->
  <p>{{ price * quantity }}</p>

  <!-- 三元表达式 -->
  <p>{{ isLoggedIn ? '欢迎回来' : '请登录' }}</p>

  <!-- 方法调用 -->
  <p>{{ message.toUpperCase() }}</p>

  <!-- 数组操作 -->
  <p>{{ items.join(', ') }}</p>

  <!-- 模板字面量 -->
  <p>{{ `你好，${name}！` }}</p>
</template>
```

### 2.2 v-once 一次性插值

```vue
<template>
  <!-- 数据变化后不再更新 -->
  <span v-once>{{ staticMessage }}</span>
</template>
```

### 2.3 v-text 等效写法

```vue
<template>
  <span v-text="message"></span>
  <!-- 等同于 -->
  <span>{{ message }}</span>
</template>
```

### 2.4 文本插值中的空值处理

```vue
<script setup>
import { ref } from 'vue'
const name = ref(null)
const age = ref(undefined)
const city = ref('')
</script>

<template>
  <!-- null 和 undefined 显示为空字符串 -->
  <p>姓名: {{ name }}</p>       <!-- 输出: 姓名: -->
  <p>年龄: {{ age }}</p>       <!-- 输出: 年龄: -->
  <p>城市: {{ city }}</p>      <!-- 输出: 城市: -->

  <!-- 提供默认值 -->
  <p>姓名: {{ name ?? '未填写' }}</p>
</template>
```

### 2.5 与 CSS v-bind 配合

```vue
<script setup>
import { ref } from 'vue'
const fontSize = ref(16)
const color = ref('#42b883')
</script>

<template>
  <p :style="{ fontSize: fontSize + 'px', color }">
    {{ fontSize }}px 的文字
  </p>
</template>
```

## 三、常见用例

### 3.1 格式化显示

```vue
<script setup>
import { ref, computed } from 'vue'
const price = ref(1234.5)
const date = ref(new Date())

const formattedPrice = computed(() => `¥${price.value.toFixed(2)}`)
const formattedDate = computed(() => date.value.toLocaleDateString('zh-CN'))
</script>

<template>
  <p>价格: {{ formattedPrice }}</p>
  <p>日期: {{ formattedDate }}</p>
</template>
```

## 四、注意事项与常见陷阱

- `{{ }}` 内只能放**单个表达式**，不能放流程控制语句
- `undefined` 和 `null` 在插值中显示为空字符串
- 插值会在每次数据变化时重新计算，复杂表达式用 computed 优化
- 模板中自动解包 ref（不需要 `.value`）
- 不要在 `{{ }}` 中产生副作用（如修改响应式数据）
- 空字符串 `''` 和数字 `0` 会正常显示（只有 null/undefined 为空）
- 使用 `||` 或 `??` 运算符提供默认值
