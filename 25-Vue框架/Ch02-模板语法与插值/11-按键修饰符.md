# 按键修饰符

## 一、概念说明

Vue 提供按键修饰符来监听特定按键的键盘事件。可以使用按键别名（如 `.enter`、`.esc`）或按键代码来绑定事件。Vue 3 还支持组合键修饰符。

```vue
<script setup>
function onEnter() {
  console.log('按下了 Enter 键')
}

function onEscape() {
  console.log('按下了 Escape 键')
}
</script>

<template>
  <!-- 按键别名 -->
  <input @keyup.enter="onEnter" placeholder="按 Enter 提交" />
  <input @keyup.escape="onEscape" placeholder="按 Esc 清除" />
</template>
```

## 二、具体用法

### 2.1 按键别名

```vue
<template>
  <!-- 常用按键别名 -->
  <input @keyup.enter="submit" />      <!-- 回车 -->
  <input @keyup.tab="onTab" />         <!-- Tab -->
  <input @keyup.delete="onDelete" />   <!-- Delete/Backspace -->
  <input @keyup.esc="cancel" />        <!-- Escape -->
  <input @keyup.space="onSpace" />     <!-- 空格 -->
  <input @keyup.up="moveUp" />         <!-- 上箭头 -->
  <input @keyup.down="moveDown" />     <!-- 下箭头 -->
  <input @keyup.left="moveLeft" />     <!-- 左箭头 -->
  <input @keyup.right="moveRight" />   <!-- 右箭头 -->
</template>
```

### 2.2 系统修饰键

```vue
<template>
  <!-- 组合键 -->
  <button @keyup.ctrl.enter="submit">Ctrl + Enter</button>
  <button @keyup.ctrl.s.prevent="save">Ctrl + S</button>
  <button @keyup.alt.delete="clearAll">Alt + Delete</button>

  <!-- .exact 精确匹配 -->
  <button @click.ctrl.exact="onCtrlClick">仅 Ctrl+点击</button>
  <button @click.exact="onExactClick">仅点击（无修饰键）</button>
</template>
```

### 2.3 鼠标按钮修饰符

```vue
<template>
  <div @click.left="onLeftClick">左键点击</div>
  <div @click.middle="onMiddleClick">中键点击</div>
  <div @click.right.prevent="onRightClick">右键点击</div>
</template>
```

## 三、注意事项与常见陷阱

- Vue 3 中按键修饰符区分大小写（`.enter` 不等于 `.Enter`）
- 系统修饰键（Ctrl、Alt、Shift、Meta）在 `keyup` 中需要配合其他按键
- `.exact` 可以精确控制组合键的匹配
- 自定义按键码在 Vue 3 中不再支持，使用按键名称
- 在 `<input>` 上常用 `@keyup.enter` 替代 `@keydown.enter` 避免重复触发
