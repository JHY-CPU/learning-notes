# 模板中的类型推断

## 一、概念说明

当使用 TypeScript 和 `<script setup lang="ts">` 时，Vue 编译器和 IDE 可以在模板中提供类型推断和智能提示。Volar 插件利用 Vue 的语言服务，在模板中实现类似 TypeScript 的类型检查。

```vue
<script setup lang="ts">
interface User {
  name: string
  age: number
  email: string
}

const user: User = {
  name: '张三',
  age: 25,
  email: 'zhang@example.com'
}
</script>

<template>
  <!-- Volar 能推断 user 的类型 -->
  <p>{{ user.name }}</p>    <!-- 提示: string -->
  <p>{{ user.age }}</p>     <!-- 提示: number -->
  <p>{{ user.email }}</p>   <!-- 提示: string -->
</template>
```

## 二、具体用法

### 2.1 defineProps 类型推断

```vue
<script setup lang="ts">
// 运行时声明 + 类型声明
interface Props {
  title: string
  count?: number
  items: Array<{ id: number; label: string }>
}

const props = defineProps<Props>()
</script>

<template>
  <!-- 模板中 props 有完整的类型提示 -->
  <h2>{{ props.title }}</h2>
  <p>数量: {{ props.count ?? 0 }}</p>
  <ul>
    <li v-for="item in props.items" :key="item.id">
      {{ item.label }}  <!-- item 的类型被推断 -->
    </li>
  </ul>
</template>
```

### 2.2 defineEmits 类型推断

```vue
<script setup lang="ts">
const emit = defineEmits<{
  change: [id: number, value: string]
  submit: [data: { name: string; email: string }]
}>()
</script>

<template>
  <!-- emit 调用有类型检查 -->
  <button @click="emit('change', 1, 'hello')">改变</button>
  <button @click="emit('submit', { name: 'test', email: 'a@b.com' })">提交</button>
</template>
```

### 2.3 计算属性和 ref 类型

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'

const count = ref<number>(0)
const doubled = computed(() => count.value * 2) // 推断为 ComputedRef<number>
</script>

<template>
  <!-- 模板中自动解包，类型推断为 number -->
  <p>{{ doubled }}</p>
</template>
```

## 三、注意事项与常见陷阱

- 类型推断需要安装 Volar 插件并启用 Take Over Mode
- `defineProps` 使用泛型时不能同时使用运行时声明
- 模板中的 ref 自动解包，类型从 `Ref<T>` 变为 `T`
- 复杂类型可能需要显式标注以获得更好的推断
- 模板中的 `$event` 类型可能需要手动标注
