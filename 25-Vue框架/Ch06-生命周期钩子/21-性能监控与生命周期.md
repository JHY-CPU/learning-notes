# 性能监控与生命周期

## 一、概念说明
利用生命周期钩子可以进行**性能监控**，测量组件的挂载时间、更新耗时等指标，帮助发现性能瓶颈。

## 二、具体用法

### 2.1 测量挂载耗时
```vue
<script setup>
import { onBeforeMount, onMounted } from 'vue'

let startTime

onBeforeMount(() => {
  startTime = performance.now()
})

onMounted(() => {
  const duration = performance.now() - startTime
  console.log(`组件挂载耗时: ${duration.toFixed(2)}ms`)
})
</script>
```

### 2.2 测量每次更新耗时
```vue
<script setup>
import { onBeforeUpdate, onUpdated } from 'vue'

let updateStart

onBeforeUpdate(() => {
  updateStart = performance.now()
})

onUpdated(() => {
  const duration = performance.now() - updateStart
  if (duration > 16) {  // 超过一帧时间 (60fps)
    console.warn(`更新耗时过长: ${duration.toFixed(2)}ms`)
  }
})
</script>
```

### 2.3 性能监控 Composable
```js
// composables/usePerfMonitor.js
import { onMounted, onUpdated, ref } from 'vue'

export function usePerfMonitor(componentName) {
  const metrics = ref({ mounts: 0, updates: 0, totalMountTime: 0, totalUpdateTime: 0 })
  let startTime

  onMounted(() => {
    metrics.value.mounts++
    metrics.value.totalMountTime += performance.now() - startTime
  })

  onUpdated(() => {
    metrics.value.updates++
    metrics.value.totalUpdateTime += performance.now() - startTime
  })

  return metrics
}
```

### 2.4 使用 performance.mark
```vue
<script setup>
import { onMounted, onUpdated } from 'vue'

onMounted(() => {
  performance.mark('component-mounted')
  performance.measure('mount-duration', 'component-start', 'component-mounted')
})
</script>
```

## 三、注意事项与常见陷阱
- `performance.now()` 精度高，适合性能测量
- 生产环境应避免过多的 console 输出
- 考虑使用 Web Vitals 等标准性能指标
- 性能数据应汇总上报而非逐次打印
