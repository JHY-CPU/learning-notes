# 侦听多个来源

## 一、概念说明
Vue 支持使用**数组形式**同时侦听多个响应式数据源。当其中任何一个源变化时，回调都会被触发。

## 二、具体用法

### 2.1 侦听多个 ref
```vue
<script setup>
import { ref, watch } from 'vue'

const firstName = ref('张')
const lastName = ref('三')

watch([firstName, lastName], ([newFirst, newLast], [oldFirst, oldLast]) => {
  console.log(`${oldFirst} ${oldLast} → ${newFirst} ${newLast}`)
})
</script>
```

### 2.2 混合侦听 ref 和 getter
```vue
<script setup>
import { ref, reactive, watch } from 'vue'

const keyword = ref('')
const state = reactive({ page: 1, size: 10 })

watch(
  [keyword, () => state.page, () => state.size],
  ([newKeyword, newPage, newSize], [oldKeyword, oldPage, oldSize]) => {
    fetchData({ keyword: newKeyword, page: newPage, size: newSize })
  }
)
</script>
```

### 2.3 watchEffect 的等价写法
```vue
<script setup>
import { ref, watch, watchEffect } from 'vue'

const a = ref(1)
const b = ref(2)

// watch 多源
watch([a, b], () => {
  console.log('a 或 b 变化了:', a.value + b.value)
})

// 等价的 watchEffect
watchEffect(() => {
  console.log('a 或 b 变化了:', a.value + b.value)
})
```

### 2.4 只有部分源变化时获取旧值
```vue
<script setup>
import { ref, watch } from 'vue'

const x = ref(0)
const y = ref(0)

watch([x, y], ([newX, newY], [oldX, oldY]) => {
  if (newX !== oldX) console.log('X 变化了')
  if (newY !== oldY) console.log('Y 变化了')
})
</script>
```

## 三、注意事项与常见陷阱
- 回调的参数是数组形式：`([newVals], [oldVals])`
- 任意一个源变化都会触发回调
- 如果只需要自动追踪，`watchEffect` 更简洁
- 如果需要 oldValue，使用 `watch([多个源], callback)`
