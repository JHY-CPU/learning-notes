# 动画库集成

## 一、概念说明

Vue 3 内置 `<Transition>` 和 `<TransitionGroup>` 组件处理基本动画。复杂动画可集成 GSAP（GreenSock）、Framer Motion 或 VueUse Motion。GSAP 功能强大适合精细控制，VueUse Motion 声明式更符合 Vue 风格。

## 二、具体用法

### VueUse Motion

```bash
npm install @vueuse/motion
```

```ts
// main.ts
import { MotionPlugin } from '@vueuse/motion'
app.use(MotionPlugin)
```

```vue
<template>
  <!-- 声明式动画 -->
  <div
    v-motion
    :initial="{ opacity: 0, y: 100 }"
    :enter="{ opacity: 1, y: 0, transition: { duration: 500 } }"
    :delay="200"
  >
    <h1>滑入动画</h1>
  </div>

  <!-- 滚动触发 -->
  <div
    v-motion-slide-visible-once-bottom
  >
    <p>滚动到视口时触发</p>
  </div>
</template>
```

### GSAP 集成

```bash
npm install gsap
```

```vue
<script setup lang="ts">
import { ref, onMounted } from 'vue'
import gsap from 'gsap'

const boxRef = ref<HTMLDivElement | null>(null)
const items = ref([1, 2, 3, 4, 5])

onMounted(() => {
  // 元素动画
  gsap.from(boxRef.value, {
    duration: 1,
    x: -200,
    opacity: 0,
    ease: 'power3.out'
  })
  // 从左侧滑入，淡入效果

  // 时间轴动画
  const tl = gsap.timeline()
  tl.to('.item', {
    duration: 0.5,
    y: 0,
    opacity: 1,
    stagger: 0.1,
    ease: 'back.out(1.7)'
  })
  // 5个元素依次弹入，间隔0.1秒
})

function animateIn() {
  gsap.to(boxRef.value, {
    duration: 0.6,
    scale: 1.2,
    rotation: 360,
    backgroundColor: '#42b883',
    ease: 'elastic.out(1, 0.3)'
  })
}
</script>

<template>
  <div ref="boxRef" class="box" @click="animateIn">
    点击动画
  </div>
  <div v-for="item in items" :key="item" class="item">
    项目 {{ item }}
  </div>
</template>
```

### CSS 过渡与 Vue Transition

```vue
<template>
  <button @click="show = !show">切换</button>

  <Transition
    name="fade"
    enter-active-class="animate__animated animate__bounceIn"
    leave-active-class="animate__animated animate__bounceOut"
  >
    <div v-if="show" class="content">
      <p>动画内容</p>
    </div>
  </Transition>

  <TransitionGroup name="list" tag="ul">
    <li v-for="item in items" :key="item.id">
      {{ item.text }}
    </li>
  </TransitionGroup>
</template>

<script setup lang="ts">
const show = ref(true)
const items = ref([
  { id: 1, text: '项目1' },
  { id: 2, text: '项目2' }
])
</script>

<style>
.list-enter-active,
.list-leave-active {
  transition: all 0.5s ease;
}
.list-enter-from,
.list-leave-to {
  opacity: 0;
  transform: translateX(-30px);
}
</style>
```

## 三、注意事项与常见陷阱

1. **GSAP 是付费库**：核心功能免费，高级插件（MorphSVG等）需付费
2. **动画库不应阻塞首次渲染**：在 onMounted 中初始化动画
3. **VueUse Motion 适合简单场景**：复杂动画序列仍推荐 GSAP
4. **Transition 的 name 对应 CSS 类名**：`name="fade"` 对应 `.fade-enter-active`
5. **性能注意**：同时动画大量元素可能卡顿，使用 `will-change` CSS 属性优化
