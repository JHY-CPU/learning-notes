# 移动端方案

## 一、概念说明

Vue 3 移动端开发主要有三个方向：**H5 移动网页**（Vant 组件库）、**跨平台小程序**（uni-app）、**原生应用**（Capacitor）。Vant 适合纯 H5 场景，uni-app 可一套代码编译到多个平台，Capacitor 将 Web 应用打包为原生 App。

## 二、具体用法

### Vant 4（移动端 H5）

```bash
npm install vant
```

```ts
// main.ts - 按需引入
import { Button, Cell, Toast, Dialog } from 'vant'
import 'vant/lib/index.css'

app.use(Button).use(Cell).use(Toast).use(Dialog)
```

```vue
<script setup lang="ts">
import { showToast, showDialog } from 'vant'

function onClick() {
  showToast('操作成功')
  // 底部弹出轻提示
}

function onConfirm() {
  showDialog({
    title: '提示',
    message: '确定要删除吗？'
  }).then(() => {
    showToast('已删除')
  })
}
</script>

<template>
  <div class="mobile-page">
    <van-nav-bar title="我的应用" left-arrow />
    <van-cell-group inset>
      <van-cell title="用户信息" is-link @click="onClick" />
      <van-cell title="设置" is-link />
    </van-cell-group>
    <van-button type="primary" block @click="onConfirm">
      确认操作
    </van-button>

    <van-tabbar v-model="active">
      <van-tabbar-item icon="home-o">首页</van-tabbar-item>
      <van-tabbar-item icon="search">搜索</van-tabbar-item>
      <van-tabbar-item icon="setting-o">我的</van-tabbar-item>
    </van-tabbar>
  </div>
</template>
```

### uni-app（跨平台）

```vue
<!-- pages/index/index.vue -->
<script setup lang="ts">
import { ref } from 'vue'

const list = ref([
  { id: 1, name: '项目1' },
  { id: 2, name: '项目2' }
])

function navigateTo(id: number) {
  uni.navigateTo({ url: `/pages/detail/detail?id=${id}` })
  // 一套代码编译到 H5、微信小程序、App
}
</script>

<template>
  <view class="container">
    <view class="item" v-for="item in list" :key="item.id" @click="navigateTo(item.id)">
      <text>{{ item.name }}</text>
    </view>
  </view>
</template>
```

### Capacitor（原生 App）

```bash
npm install @capacitor/core @capacitor/cli
npx cap init MyApp com.example.myapp
npx cap add android ios
npx cap sync
npx cap open android
# 在 Android Studio 中运行
```

```ts
// 使用 Capacitor 插件访问原生功能
import { Camera, CameraResultType } from '@capacitor/camera'
import { Geolocation } from '@capacitor/geolocation'

async function takePhoto() {
  const photo = await Camera.getPhoto({
    resultType: CameraResultType.Uri
  })
  console.log('照片路径:', photo.webPath)
}

async function getLocation() {
  const position = await Geolocation.getCurrentPosition()
  console.log('位置:', position.coords.latitude, position.coords.longitude)
}
```

## 三、注意事项与常见陷阱

1. **Vant 不支持桌面端**：专为移动端设计，在大屏幕显示效果差
2. **uni-app 语法与标准 Vue 有差异**：使用 `<view>` 替代 `<div>`，API 不同
3. **Capacitor 需要 Android Studio / Xcode**：纯 Web 开发者需要学习成本
4. **移动端适配用 rem/vw**：配合 `postcss-pxtorem` 或 `postcss-px-to-viewport`
5. **真机调试必不可少**：模拟器不能完全替代真机测试
