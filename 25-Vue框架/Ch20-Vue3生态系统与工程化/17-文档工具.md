# 文档工具VitePress

## 一、概念说明

VitePress 是 Vue 官方的静态文档站生成器，基于 Vite 和 Vue 3。它将 Markdown 文件渲染为静态网站，支持 Vue 组件嵌入、代码高亮、搜索等功能。适合组件文档、技术博客、API 文档等场景。

## 二、具体用法

### 安装初始化

```bash
npm install -D vitepress
npx vitepress init
# 选择目录、标题、描述等
```

### 目录结构

```
docs/
├── .vitepress/
│   └── config.ts      # VitePress 配置
├── index.md           # 首页
├── guide/
│   ├── index.md       # 指南首页
│   ├── getting-started.md
│   └── components.md
└── api/
    └── reference.md
```

### 配置文件

```ts
// docs/.vitepress/config.ts
import { defineConfig } from 'vitepress'

export default defineConfig({
  title: '我的组件库',
  description: '基于 Vue 3 的组件库文档',
  themeConfig: {
    nav: [
      { text: '指南', link: '/guide/' },
      { text: 'API', link: '/api/' }
    ],
    sidebar: {
      '/guide/': [
        {
          text: '入门',
          items: [
            { text: '快速开始', link: '/guide/getting-started' },
            { text: '组件', link: '/guide/components' }
          ]
        }
      ]
    },
    socialLinks: [
      { icon: 'github', link: 'https://github.com/your/repo' }
    ],
    search: {
      provider: 'local'  // 本地搜索
    }
  }
})
```

### Markdown 中使用 Vue 组件

````markdown
<!-- docs/guide/components.md -->
# 组件文档

## MyButton 按钮

基础用法：

<MyButton type="primary">主要按钮</MyButton>

<script setup>
import MyButton from '../../src/components/MyButton.vue'
</script>

```vue
<MyButton type="primary">主要按钮</MyButton>
<MyButton type="secondary">次要按钮</MyButton>
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| type | 'primary' \| 'secondary' | 'primary' | 按钮类型 |
| disabled | boolean | false | 是否禁用 |
````

### 构建和部署

```json
// package.json
{
  "scripts": {
    "docs:dev": "vitepress dev docs",
    "docs:build": "vitepress build docs",
    "docs:preview": "vitepress preview docs"
  }
}
```

```yaml
# GitHub Actions 自动部署
name: Deploy Docs
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: npm ci
      - run: npm run docs:build
      - uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: docs/.vitepress/dist
```

## 三、注意事项与常见陷阱

1. **首页需要 frontmatter 配置**：`index.md` 需要 `layout: home` 设置
2. **组件导入路径是相对的**：Markdown 中 import 使用相对路径
3. **搜索默认不启用**：需要配置 `search.provider` 为 `'local'` 或 algolia
4. **自定义主题需扩展默认主题**：不能完全替换
5. **构建产物是纯静态文件**：可部署到任意静态托管
