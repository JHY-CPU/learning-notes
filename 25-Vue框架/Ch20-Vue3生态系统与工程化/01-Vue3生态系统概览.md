# Vue3生态系统概览

## 一、概念说明

Vue 3 拥有成熟的生态系统，涵盖 UI 组件库、状态管理、路由、构建工具、测试框架等。核心生态由 Vue 官方维护（Vue Router、Pinia、Vite），社区生态提供丰富的 UI 库和工具。了解全景图有助于做出正确的技术选型。

## 二、具体用法

### 核心生态

```text
Vue 3 核心生态：

框架层
├── Vue 3          → 核心库（响应式、组件、模板）
├── Vite           → 构建工具（开发服务器 + Rollup 打包）
├── Vue Router     → 官方路由
├── Pinia          → 官方状态管理
└── Vue DevTools   → 浏览器调试工具

SSR/全栈
├── Nuxt 3         → 全栈框架（SSR/SSG/ISR）
├── VitePress      → 文档站/静态站生成器
└── Quasar          → 全栈 UI 框架
```

## 三、注意事项与常见陷阱

1. **不要安装过多库**：每个依赖都有维护成本，按需引入
2. **UI 库一旦选定难以更换**：选择前充分评估，考虑设计规范和团队熟悉度
3. **VueUse 是必备工具**：大部分常用功能已被封装，避免重复造轮子
4. **Vite 已成为事实标准**：新项目直接用 Vite，不需要 Webpack
5. **生态版本要匹配**：确保各库支持 Vue 3，Vue 2 的库可能不兼容
