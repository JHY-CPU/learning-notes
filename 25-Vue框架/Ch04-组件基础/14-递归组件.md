# 递归组件

## 一、概念说明

递归组件是指组件在自己的模板中调用自身的组件。常用于树形结构、评论嵌套、文件目录等场景。递归组件需要有终止条件，防止无限递归。

```vue
<!-- TreeNode.vue -->
<script setup>
defineProps({
  node: {
    type: Object,
    required: true
  },
  depth: {
    type: Number,
    default: 0
  }
})
</script>

<template>
  <div class="tree-node" :style="{ marginLeft: depth * 20 + 'px' }">
    <span>{{ node.name }}</span>
    <!-- 递归调用自身 -->
    <TreeNode
      v-for="child in node.children"
      :key="child.id"
      :node="child"
      :depth="depth + 1"
    />
  </div>
</template>
```

## 二、具体用法

### 2.1 文件目录树

```vue
<!-- FileTree.vue -->
<script setup>
defineProps({
  items: Array
})
</script>

<template>
  <ul>
    <li v-for="item in items" :key="item.name">
      <span>{{ item.type === 'folder' ? '📁' : '📄' }} {{ item.name }}</span>
      <FileTree v-if="item.children" :items="item.children" />
    </li>
  </ul>
</template>
```

### 2.2 评论嵌套

```vue
<!-- CommentThread.vue -->
<script setup>
defineProps({
  comment: Object,
  depth: { type: Number, default: 0 }
})
</script>

<template>
  <div class="comment" :style="{ marginLeft: depth * 16 + 'px' }">
    <p>{{ comment.author }}: {{ comment.text }}</p>
    <CommentThread
      v-for="reply in comment.replies"
      :key="reply.id"
      :comment="reply"
      :depth="depth + 1"
    />
  </div>
</template>
```

### 2.3 限制递归深度

```vue
<script setup>
defineProps({
  node: Object,
  depth: { type: Number, default: 0 },
  maxDepth: { type: Number, default: 10 }
})
</script>

<template>
  <div>
    <span>{{ node.name }}</span>
    <template v-if="depth < maxDepth && node.children">
      <MyTree
        v-for="child in node.children"
        :key="child.id"
        :node="child"
        :depth="depth + 1"
      />
    </template>
  </div>
</template>
```

## 三、注意事项与常见陷阱

- 递归组件**必须有终止条件**（数据耗尽或达到最大深度）
- 递归组件通过文件名推断 name（如 `TreeNode.vue` 的 name 为 `TreeNode`）
- 或通过 `defineOptions({ name: 'TreeNode' })` 显式设置
- 深度递归可能有性能问题，考虑虚拟滚动
- 确保递归数据是树形结构（无循环引用）
