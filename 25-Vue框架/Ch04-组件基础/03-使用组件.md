# 使用组件

## 一、概念说明

使用组件需要先**导入**，然后在模板中用 PascalCase 或 kebab-case 使用。`<script setup>` 中导入的组件自动注册，无需额外配置。

```vue
<script setup>
import MyComponent from './MyComponent.vue'
import UserCard from './UserCard.vue'
</script>

<template>
  <MyComponent />
  <UserCard name="张三" />
  <my-component />  <!-- kebab-case 也支持 -->
</template>
```

## 二、具体用法

### 2.1 局部导入

```vue
<script setup>
import ChildComponent from './ChildComponent.vue'
import AnotherChild from './AnotherChild.vue'
</script>

<template>
  <div>
    <ChildComponent title="子组件 A" />
    <ChildComponent title="子组件 B" />
    <AnotherChild />
  </div>
</template>
```

### 2.2 动态组件

```vue
<script setup>
import { ref, shallowRef } from 'vue'
import ComponentA from './ComponentA.vue'
import ComponentB from './ComponentB.vue'

const currentComponent = shallowRef(ComponentA)
</script>

<template>
  <button @click="currentComponent = ComponentA">显示 A</button>
  <button @click="currentComponent = ComponentB">显示 B</button>
  <component :is="currentComponent" />
</template>
```

### 2.3 递归组件

```vue
<!-- TreeNode.vue -->
<script setup>
defineProps({
  node: Object
})

// 递归组件需要通过文件名推断 name 或使用 defineOptions
defineOptions({ name: 'TreeNode' })
</script>

<template>
  <div>
    <span>{{ node.name }}</span>
    <div v-if="node.children" style="margin-left: 20px">
      <TreeNode v-for="child in node.children" :key="child.id" :node="child" />
    </div>
  </div>
</template>
```

### 2.4 异步组件

```vue
<script setup>
import { defineAsyncComponent } from 'vue'

// 懒加载组件
const HeavyComponent = defineAsyncComponent(() =>
  import('./HeavyComponent.vue')
)

// 带加载和错误状态
const AsyncChart = defineAsyncComponent({
  loader: () => import('./Chart.vue'),
  loadingComponent: LoadingSpinner,
  errorComponent: ErrorDisplay,
  delay: 200,
  timeout: 5000
})
</script>

<template>
  <Suspense>
    <template #default>
      <HeavyComponent />
    </template>
    <template #fallback>
      <p>加载中...</p>
    </template>
  </Suspense>
</template>
```

## 三、常见用例

### 3.1 组件命名对比

| 位置 | 推荐格式 | 示例 |
|------|---------|------|
| 文件名 | PascalCase | `UserCard.vue` |
| 模板中 | PascalCase | `<UserCard />` |
| DOM 模板 | kebab-case | `<user-card />` |

## 四、注意事项与常见陷阱

- 模板中可以使用 PascalCase 或 kebab-case，推荐 PascalCase
- 在 DOM 模板中必须使用 kebab-case（因为 HTML 大小写不敏感）
- 动态组件使用 `:is` 绑定组件对象
- 递归组件必须有 `name` 选项或通过文件名推断
- `<script setup>` 中导入的组件自动注册，不需要 `components` 选项
- 异步组件需要 `<Suspense>` 或手动处理加载状态
- 使用 `shallowRef` 存储动态组件对象比 `ref` 更高效
