# 纯展示组件

## 一、概念说明

纯展示组件（也称无状态组件或哑组件）只负责接收 props 并渲染 UI，不包含业务逻辑或状态管理。它们有清晰的输入（props）和输出（events/slots），易于测试和复用。

```vue
<!-- 纯展示组件: Badge.vue -->
<script setup>
defineProps({
  label: { type: String, required: true },
  color: { type: String, default: 'blue' },
  size: { type: String, default: 'medium' }
})
</script>

<template>
  <span class="badge" :class="[color, size]">
    {{ label }}
  </span>
</template>

<style scoped>
.badge {
  display: inline-block;
  padding: 2px 8px;
  border-radius: 4px;
  font-size: 12px;
}
.blue { background: #e3f2fd; color: #1976d2; }
.red { background: #ffebee; color: #c62828; }
.green { background: #e8f5e9; color: #2e7d32; }
.small { font-size: 10px; }
.medium { font-size: 12px; }
.large { font-size: 14px; }
</style>
```

## 二、具体用法

### 2.1 设计原则

```vue
<!-- 好: 纯展示组件，只接收 props 并渲染 -->
<!-- Avatar.vue -->
<script setup>
defineProps({
  src: String,
  alt: { type: String, default: '头像' },
  size: { type: Number, default: 40 }
})
</script>

<template>
  <img
    :src="src"
    :alt="alt"
    :width="size"
    :height="size"
    class="avatar"
    :style="{ width: size + 'px', height: size + 'px' }"
  />
</template>
```

### 2.2 复用性示例

```vue
<!-- StatCard.vue -->
<script setup>
defineProps({
  title: String,
  value: [String, Number],
  icon: String,
  trend: { type: String, default: '' } // 'up' | 'down' | ''
})
</script>

<template>
  <div class="stat-card">
    <span class="icon">{{ icon }}</span>
    <div class="info">
      <p class="title">{{ title }}</p>
      <p class="value">{{ value }}</p>
      <span v-if="trend" :class="trend">{{ trend === 'up' ? '↑' : '↓' }}</span>
    </div>
  </div>
</template>
```

## 三、注意事项与常见陷阱

- 纯展示组件不应该有 `ref`、`reactive`、`watch` 等响应式 API
- 组件名应该描述它展示的内容
- 保持接口（props/events/slots）简洁明确
- 纯展示组件应该可以被任何项目复用
- 样式应该尽量独立，避免依赖外部 CSS
