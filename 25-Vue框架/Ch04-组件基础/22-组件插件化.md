# 组件插件化

## 一、概念说明

将组件打包为 Vue 插件，方便全局注册和分发。插件是一个包含 `install` 方法的对象，通过 `app.use()` 安装后自动注册组件和指令。

```vue
<!-- 插件入口: MyUIPlugin.js -->
import MyButton from './components/MyButton.vue'
import MyInput from './components/MyInput.vue'
import MyModal from './components/MyModal.vue'

export default {
  install(app, options = {}) {
    // 注册全局组件
    app.component('MyButton', MyButton)
    app.component('MyInput', MyInput)
    app.component('MyModal', MyModal)

    // 提供全局配置
    app.provide('myUIConfig', {
      theme: options.theme || 'light',
      size: options.size || 'medium'
    })
  }
}
```

## 二、具体用法

### 2.1 安装和使用

```js
// main.js
import { createApp } from 'vue'
import App from './App.vue'
import MyUI from './plugins/my-ui'

const app = createApp(App)
app.use(MyUI, { theme: 'dark', size: 'large' })
app.mount('#app')
```

```vue
<!-- 任意组件中直接使用 -->
<template>
  <MyButton>点击</MyButton>
  <MyInput v-model="value" />
</template>
```

### 2.2 完整插件结构

```js
// my-ui/index.js
import Button from './Button.vue'
import Input from './Input.vue'

// 支持按需导入
export { Button, Input }

// 默认导出插件
export default {
  install(app, options) {
    // 全局组件
    app.component(options.prefix ? `${options.prefix}Button` : 'MyButton', Button)
    app.component(options.prefix ? `${options.prefix}Input` : 'MyInput', Input)

    // 全局指令
    app.directive('focus', {
      mounted: (el) => el.focus()
    })

    // 全局方法
    app.config.globalProperties.$myUtil = {
      format: (val) => val.toString()
    }
  }
}
```

### 2.3 TypeScript 支持

```ts
// types.ts
import type { Component } from 'vue'

declare module 'vue' {
  interface GlobalComponents {
    MyButton: typeof import('./components/Button.vue').default
    MyInput: typeof import('./components/Input.vue').default
  }
}
```

## 三、注意事项与常见陷阱

- 插件的 `install` 方法在 `app.mount()` 之前调用
- 全局注册的组件无法 Tree-shaking
- 提供按需导入选项让用户选择
- 全局组件名应该有命名空间前缀
- 使用 TypeScript 时需要扩展全局类型声明
