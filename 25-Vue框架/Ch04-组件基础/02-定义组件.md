# 定义组件

## 一、概念说明

Vue 3 组件通过 `.vue` 单文件组件定义。使用 `<script setup>` 语法糖是最简洁的方式。组件名应该由多个单词组成（如 `UserProfile`），避免与 HTML 元素冲突。

```vue
<!-- UserProfile.vue -->
<script setup>
import { ref } from 'vue'

const props = defineProps({
  name: String,
  age: Number
})

const emit = defineEmits(['update'])
</script>

<template>
  <div class="user-profile">
    <h3>{{ name }}</h3>
    <p>年龄: {{ age }}</p>
    <button @click="emit('update', name)">更新</button>
  </div>
</template>
```

## 二、具体用法

### 2.1 `<script setup>` 定义

```vue
<!-- GreetingCard.vue -->
<script setup>
import { ref, computed } from 'vue'

const greeting = ref('你好')
const formattedGreeting = computed(() => `${greeting.value}，欢迎！`)

const sayHello = () => {
  console.log(formattedGreeting.value)
}
</script>

<template>
  <div>
    <h2>{{ formattedGreeting }}</h2>
    <button @click="sayHello">打招呼</button>
  </div>
</template>
```

### 2.2 带 TypeScript 的定义

```vue
<script setup lang="ts">
interface Props {
  title: string
  count?: number
  items: string[]
}

const props = withDefaults(defineProps<Props>(), {
  count: 0
})

const emit = defineEmits<{
  change: [value: string]
  submit: [data: { title: string }]
}>()
</script>
```

### 2.3 defineOptions（Vue 3.3+）

```vue
<script setup>
defineOptions({
  name: 'CustomName',
  inheritAttrs: false
})
</script>
```

### 2.4 defineExpose 暴露方法

```vue
<script setup>
import { ref } from 'vue'

const count = ref(0)
const reset = () => { count.value = 0 }

// 暴露给父组件通过模板 ref 访问
defineExpose({ count, reset })
</script>
```

## 三、常见用例

### 3.1 完整组件结构

```vue
<script setup>
// 1. 导入
import { ref, computed, onMounted } from 'vue'
import { useApi } from '@/composables/useApi'

// 2. 编译器宏（不需要 import）
const props = defineProps({ id: Number })
const emit = defineEmits(['close'])

// 3. 组合式函数
const { data, loading, fetch } = useApi()

// 4. 响应式状态
const isVisible = ref(true)

// 5. 计算属性
const displayData = computed(() => data.value?.filter(Boolean) ?? [])

// 6. 方法
function handleClose() {
  isVisible.value = false
  emit('close')
}

// 7. 生命周期
onMounted(() => fetch(props.id))
</script>

<template>
  <div v-if="isVisible">
    <p v-if="loading">加载中...</p>
    <ul v-else>
      <li v-for="item in displayData" :key="item.id">{{ item.name }}</li>
    </ul>
    <button @click="handleClose">关闭</button>
  </div>
</template>

<style scoped>
/* 样式 */
</style>
```

## 四、注意事项与常见陷阱

- `<script setup>` 中的顶层绑定自动暴露给模板
- 组件名建议使用多个单词（`UserProfile` 而非 `User`）
- `defineProps` 和 `defineEmits` 是编译宏，不需要 import
- `defineProps` 和 `defineEmits` 必须在 `<script setup>` 的顶层使用
- 使用 `defineExpose` 暴露方法给父组件
- `<script setup>` 不能与普通的 `<script>` 同时使用（除了 `defineOptions`）
- 编译宏（defineProps、defineEmits 等）在 `<script setup>` 中可用，不需要引入
