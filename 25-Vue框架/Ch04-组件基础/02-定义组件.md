# 定义组件

## 一、概念说明

Vue 3 组件通过 `.vue` 单文件组件定义。使用 `<script setup>` 语法糖是最简洁的方式。组件名应该由多个单词组成（如 `UserProfile`），避免与 HTML 元素冲突。

```vue
<!-- UserProfile.vue -->
<script setup>
import { ref } from 'vue'

const props = defineProps({
  name: String,
  age: Number
})

const emit = defineEmits(['update'])
</script>

<template>
  <div class="user-profile">
    <h3>{{ name }}</h3>
    <p>年龄: {{ age }}</p>
    <button @click="emit('update', name)">更新</button>
  </div>
</template>
```

## 二、具体用法

### 2.1 `<script setup>` 定义

```vue
<!-- GreetingCard.vue -->
<script setup>
import { ref, computed } from 'vue'

// 响应式数据
const greeting = ref('你好')

// 计算属性
const formattedGreeting = computed(() => `${greeting.value}，欢迎！`)

// 方法
const sayHello = () => {
  console.log(formattedGreeting.value)
}
</script>

<template>
  <div>
    <h2>{{ formattedGreeting }}</h2>
    <button @click="sayHello">打招呼</button>
  </div>
</template>
```

### 2.2 带 TypeScript 的定义

```vue
<script setup lang="ts">
interface Props {
  title: string
  count?: number
  items: string[]
}

const props = withDefaults(defineProps<Props>(), {
  count: 0
})

const emit = defineEmits<{
  change: [value: string]
  submit: [data: { title: string }]
}>()
</script>
```

### 2.3 defineOptions

```vue
<script setup>
// 设置组件选项（Vue 3.3+）
defineOptions({
  name: 'CustomName',
  inheritAttrs: false
})
</script>
```

## 三、注意事项与常见陷阱

- `<script setup>` 中的顶层绑定自动暴露给模板
- 组件名建议使用多个单词（`UserProfile` 而非 `User`）
- `defineProps` 和 `defineEmits` 是编译宏，不需要 import
- `defineProps` 和 `defineEmits` 必须在 `<script setup>` 的顶层使用
- 使用 `defineExpose` 暴露方法给父组件
