# 图片优化

## 一、概念说明

图片通常是网页中最大的资源。优化图片可以显著减少加载时间和带宽消耗。关键策略包括：选择正确格式、懒加载、响应式图片和 CDN。

```vue
<script setup>
import { ref } from 'vue'

// 原生懒加载
// <img loading="lazy" src="..." />

// 自定义懒加载指令
const vLazy = {
  mounted(el, binding) {
    const observer = new IntersectionObserver(([entry]) => {
      if (entry.isIntersecting) {
        el.src = binding.value
        observer.unobserve(el)
      }
    })
    observer.observe(el)
  }
}

const images = ref([
  '/images/photo1.webp',
  '/images/photo2.webp',
  '/images/photo3.webp',
])
</script>

<template>
  <img
    v-for="(src, i) in images"
    :key="i"
    v-lazy="src"
    alt="照片"
    width="400"
    height="300"
  />
</template>
```

## 二、具体用法

### 2.1 图片格式选择

| 格式 | 适用场景 | 特点 |
|------|---------|------|
| WebP | 通用场景 | 体积小，兼容性好 |
| AVIF | 现代浏览器 | 体积最小 |
| SVG | 图标、Logo | 矢量不失真 |
| JPEG | 照片 | 压缩率高 |
| PNG | 需要透明度 | 无损压缩 |

### 2.2 响应式图片

```html
<img
  srcset="photo-400.webp 400w, photo-800.webp 800w, photo-1200.webp 1200w"
  sizes="(max-width: 600px) 400px, (max-width: 1024px) 800px, 1200px"
  src="photo-800.webp"
  alt="响应式图片"
/>
```

### 2.3 占位图方案

```vue
<template>
  <img
    :src="loaded ? realSrc : placeholder"
    @load="loaded = true"
    :class="{ loaded }"
    class="lazy-img"
  />
</template>

<style>
.lazy-img {
  filter: blur(10px);
  transition: filter 0.3s;
}
.lazy-img.loaded {
  filter: blur(0);
}
</style>
```

## 三、注意事项与常见陷阱

- 设置 `width` 和 `height` 属性避免 CLS（布局偏移）
- 图片懒加载时使用低质量占位图（LQIP）
- 重要首屏图片不应懒加载，且使用 `<link rel="preload">`
- 避免在 CSS 中使用 `background-image` 加载大图（不支持懒加载）
