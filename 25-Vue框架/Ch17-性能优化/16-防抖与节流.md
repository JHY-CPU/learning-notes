# 防抖与节流

## 一、概念说明

防抖（Debounce）和节流（Throttle）是控制事件触发频率的两种策略。用于优化高频事件（滚动、输入、窗口调整）的处理性能。

- **防抖**：事件停止触发后，等待一段时间再执行。适用于搜索框输入。
- **节流**：固定时间间隔执行一次。适用于滚动事件。

```vue
<script setup>
import { ref, watch } from 'vue'
import { useDebounceFn, useThrottleFn } from '@vueuse/core'

const keyword = ref('')
const results = ref([])

// 防抖搜索：输入停止 300ms 后才发请求
const debouncedSearch = useDebounceFn(async () => {
  if (!keyword.value) return
  const res = await fetch(`/api/search?q=${keyword.value}`)
  results.value = await res.json()
}, 300)

watch(keyword, debouncedSearch)

// 节流滚动：每 100ms 最多执行一次
const onScroll = useThrottleFn(() => {
  console.log('滚动位置:', window.scrollY)
}, 100)
</script>

<template>
  <input v-model="keyword" placeholder="搜索..." />
  <ul>
    <li v-for="r in results" :key="r.id">{{ r.name }}</li>
  </ul>
</template>
```

## 二、具体用法

### 2.1 手动实现防抖

```js
function debounce(fn, delay = 300) {
  let timer = null
  return function (...args) {
    clearTimeout(timer)
    timer = setTimeout(() => fn.apply(this, args), delay)
  }
}

const handleInput = debounce((e) => {
  console.log('搜索:', e.target.value)
}, 300)
```

### 2.2 手动实现节流

```js
function throttle(fn, interval = 100) {
  let last = 0
  return function (...args) {
    const now = Date.now()
    if (now - last >= interval) {
      last = now
      fn.apply(this, args)
    }
  }
}
```

```js
// 防抖函数
const debouncedFn = useDebounceFn(fn, 500)

// 节流函数
const throttledFn = useThrottleFn(fn, 200)

const debouncedValue = useDebounce(keyword, 300)
```

### 2.4 对比区别

| 特性 | 防抖 | 节流 |
|------|------|------|
| 执行时机 | 停止触发后延迟执行 | 固定间隔执行 |
| 适用场景 | 搜索框、表单验证 | 滚动、resize、拖拽 |
| 特点 | 只执行最后一次 | 均匀执行 |

## 三、注意事项与常见陷阱

- 防抖延迟不宜过长，否则用户感知明显
- 组件卸载时要 `clearTimeout` 避免更新已销毁的组件
- 使用 VueUse 的组合式函数可自动处理清理
