# 虚拟滚动

## 一、概念说明

虚拟滚动（Virtual Scroll）只渲染可视区域内的列表项，即使有上万条数据也能保持流畅。它是处理大列表的最有效方案。

```vue
<script setup>
import { ref } from 'vue'
import { RecycleScroller } from 'vue-virtual-scroller'
import 'vue-virtual-scroller/dist/vue-virtual-scroller.css'

const items = ref(
  Array.from({ length: 10000 }, (_, i) => ({
    id: i,
    text: `项目 ${i}`,
    height: Math.random() > 0.7 ? 80 : 50 // 动态高度
  }))
)
</script>

<template>
  <RecycleScroller
    :items="items"
    :item-size="50"
    key-field="id"
    v-slot="{ item }"
    style="height: 400px"
  >
    <div class="list-item">
      <strong>{{ item.text }}</strong>
      <p>这是第 {{ item.id }} 项的内容</p>
    </div>
  </RecycleScroller>
</template>

<style>
.list-item {
  padding: 12px;
  border-bottom: 1px solid #eee;
}
</style>
```

## 二、具体用法

### 2.1 安装 vue-virtual-scroller

```bash
npm install vue-virtual-scroller
```

### 2.2 固定高度 vs 动态高度

```vue
<!-- 固定高度（性能最好） -->
<RecycleScroller :items="items" :item-size="50" v-slot="{ item }">
  <div>{{ item.text }}</div>
</RecycleScroller>

<!-- 动态高度 -->
<RecycleScroller
  :items="items"
  :item-size="50"
  :buffer="200"
  v-slot="{ item }"
>
  <div :style="{ minHeight: '50px' }">{{ item.text }}</div>
</RecycleScroller>
```

### 2.3 手动实现简易版

```vue
<script setup>
import { ref, computed } from 'vue'

const props = defineProps({ items: Array, itemHeight: Number, containerHeight: Number })
const scrollTop = ref(0)

const visibleItems = computed(() => {
  const start = Math.floor(scrollTop.value / props.itemHeight)
  const count = Math.ceil(props.containerHeight / props.itemHeight) + 2
  return props.items.slice(start, start + count)
})
</script>
```

## 三、注意事项与常见陷阱

- 列表项必须有唯一的 `key`（`key-field` 指定）
- 动态高度列表性能不如固定高度
- 虚拟滚动中不能使用 CSS `position: sticky`
- 滚动容器必须有固定高度
