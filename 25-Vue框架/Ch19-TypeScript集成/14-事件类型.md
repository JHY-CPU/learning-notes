# 事件类型

## 一、概念说明

Vue 模板中的事件处理器有对应的 TypeScript 类型：DOM 事件用 `EventHandler<T>`，具体事件如 `MouseEvent`、`KeyboardEvent`、`InputEvent` 等。正确标注事件类型能获得精确的属性自动补全。

## 二、具体用法

### 基本事件类型

```vue
<script setup lang="ts">
import type { EventHandler } from 'vue'

// 鼠标事件
function handleClick(event: MouseEvent) {
  console.log('点击位置:', event.clientX, event.clientY)
  // 输出：点击位置: 150 300
  console.log('目标元素:', (event.target as HTMLElement).tagName)
}

// 键盘事件
function handleKeyup(event: KeyboardEvent) {
  if (event.key === 'Enter') {
    console.log('按下回车键')
  }
  if (event.ctrlKey && event.key === 's') {
    event.preventDefault()
    console.log('Ctrl+S 被按下')
  }
}

// 输入事件
function handleInput(event: Event) {
  const target = event.target as HTMLInputElement
  console.log('输入值:', target.value)
  // 输出：输入值: 用户刚输入的内容
}

// 滚动事件
function handleScroll(event: Event) {
  const target = event.target as HTMLElement
  console.log('滚动位置:', target.scrollTop)
}

// 显式事件处理器类型
const onSubmit: EventHandler<SubmitEvent> = (event) => {
  event.preventDefault()
  console.log('表单提交')
}
</script>

<template>
  <div>
    <button @click="handleClick">点击我</button>
    <input @keyup="handleKeyup" placeholder="按Enter测试" />
    <input @input="handleInput" placeholder="输入内容" />
    <div @scroll="handleScroll" style="height: 200px; overflow: auto">
      <div style="height: 1000px">滚动区域</div>
    </div>
    <form @submit="onSubmit">
      <button type="submit">提交</button>
    </form>
  </div>
</template>
```

### 事件类型集合

```ts
// 常用 DOM 事件类型
MouseEvent      // click, dblclick, mousedown, mouseup
KeyboardEvent   // keydown, keyup, keypress
InputEvent      // input（不同于 Event）
FocusEvent      // focus, blur
TouchEvent      // touchstart, touchmove, touchend
WheelEvent      // wheel
DragEvent       // drag, dragstart, dragend
SubmitEvent     // submit
ChangeEvent     // change
ClipboardEvent  // copy, paste, cut
```

### 自定义事件类型

```vue
<!-- CustomList.vue -->
<script setup lang="ts">
// 事件对象的类型定义
interface ItemClickEvent {
  item: { id: number; name: string }
  index: number
  originalEvent: MouseEvent
}

const emit = defineEmits<{
  itemClick: [event: ItemClickEvent]
}>()

function handleClick(item: { id: number; name: string }, index: number, e: MouseEvent) {
  emit('itemClick', { item, index, originalEvent: e })
}
</script>

<template>
  <ul>
    <li
      v-for="(item, index) in items"
      :key="item.id"
      @click="handleClick(item, index, $event)"
    >
      {{ item.name }}
    </li>
  </ul>
</template>
```

```vue
<!-- 父组件 -->
<script setup lang="ts">
function onItemClick(event: { item: { id: number; name: string }; index: number }) {
  console.log('点击了:', event.item.name, '索引:', event.index)
  // 输出：点击了: 产品A 索引: 0
}
</script>

<template>
  <CustomList @item-click="onItemClick" />
</template>
```

## 三、注意事项与常见陷阱

1. **内联事件处理器不需要类型标注**：`@click="count++"` 自动推断
2. **`$event` 在模板中可用**：`@click="handler($event)"` 传递原生事件
3. **InputEvent 和 Event 不同**：`@input` 用 InputEvent，`@change` 用 Event
4. **事件修饰符不影响类型**：`@click.prevent` 仍然接收 MouseEvent
5. **第三方组件的事件类型需查看文档**：自定义事件参数由组件作者定义
