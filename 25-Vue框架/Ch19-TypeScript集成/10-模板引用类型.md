# 模板引用类型

## 一、概念说明

模板引用（template ref）通过 `ref()` 获取 DOM 元素或子组件实例的引用。TypeScript 需要知道引用的类型才能提供正确的自动补全。DOM 元素用 `Ref<HTMLElement | null>`，子组件用 `Ref<InstanceType<typeof Component> | null>`。

## 二、具体用法

### DOM 元素引用

```vue
<script setup lang="ts">
import { ref, onMounted } from 'vue'

// 基本 DOM 引用
const inputRef = ref<HTMLInputElement | null>(null)
const containerRef = ref<HTMLDivElement | null>(null)

onMounted(() => {
  // 类型安全的 DOM 操作
  inputRef.value?.focus()
  // TypeScript 知道 focus() 是 HTMLInputElement 的方法

  const width = containerRef.value?.offsetWidth
  console.log('容器宽度:', width)
  // 输出：容器宽度: 800（具体值取决于CSS）

  // inputRef.value?.play()
  // 类型错误：HTMLInputElement 没有 play 方法
})

function selectText() {
  inputRef.value?.select()
  // 选中输入框中的文本
}
</script>

<template>
  <div ref="containerRef">
    <input ref="inputRef" type="text" value="默认文本" />
    <button @click="selectText">选中文本</button>
  </div>
</template>
```

### 子组件引用

```vue
<!-- CustomInput.vue -->
<script setup lang="ts">
const modelValue = defineModel<string>()

function focus() {
  // 内部聚焦逻辑
  console.log('CustomInput 聚焦')
}

function clear() {
  modelValue.value = ''
}

defineExpose({ focus, clear })
</script>
```

```vue
<!-- 父组件 -->
<script setup lang="ts">
import { ref } from 'vue'
import CustomInput from './CustomInput.vue'

// 方式1：使用 InstanceType
const customInputRef = ref<InstanceType<typeof CustomInput> | null>(null)

// 方式2：使用 ComponentExposed（Vue 3.3+）
// import type { ComponentExposed } from 'vue'
// const customInputRef = ref<ComponentExposed<typeof CustomInput> | null>(null)

function handleFocus() {
  customInputRef.value?.focus()
  // TypeScript 知道 focus 方法存在
}

function handleClear() {
  customInputRef.value?.clear()
}
</script>

<template>
  <CustomInput ref="customInputRef" v-model="text" />
  <button @click="handleFocus">聚焦</button>
  <button @click="handleClear">清空</button>
</template>
```

### 列表引用

```vue
<script setup lang="ts">
import { ref } from 'vue'

const items = ref([1, 2, 3, 4, 5])
const itemRefs = ref<HTMLLIElement[]>([])

function scrollToItem(index: number) {
  itemRefs.value[index]?.scrollIntoView({ behavior: 'smooth' })
  // itemRefs.value 是 HTMLLIElement 数组
}
</script>

<template>
  <ul>
    <li
      v-for="(item, index) in items"
      :key="item"
      :ref="el => { if (el) itemRefs[index] = el as HTMLLIElement }"
    >
      项目 {{ item }}
    </li>
  </ul>
  <button @click="scrollToItem(2)">滚动到第3项</button>
</template>
```

## 三、注意事项与常见陷阱

1. **初始值必须为 null**：SSR 时没有 DOM，ref 必须初始为 null
2. **模板中 ref 名必须与变量名一致**：`ref="myRef"` 对应 `const myRef = ref()`
3. **v-for 中的 ref 是数组**：多个元素同名 ref 自动收集为数组
4. **访问 ref 需要可选链**：`inputRef.value?.focus()` 避免 null 报错
5. **defineExpose 决定实例类型**：未暴露的方法在 ref 中不可见
