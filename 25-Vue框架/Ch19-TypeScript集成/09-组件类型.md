# 组件类型

## 一、概念说明

Vue 3 提供 `DefineComponent`、`Component`、`FunctionalComponent` 等组件类型。在动态组件、异步组件、JSX 等场景中需要显式使用这些类型来获得类型检查。`DefineComponent` 是最通用的组件类型。

## 二、具体用法

### DefineComponent 基本用法

```ts
import type { DefineComponent, Component } from 'vue'

// 定义组件类型
type MyButton = DefineComponent<{
  label: string
  size?: 'small' | 'medium' | 'large'
}, {}, any>

// 在数组或对象中存储组件
const components: Record<string, Component> = {
  MyButton: defineAsyncComponent(() => import('./MyButton.vue')),
  MyInput: defineAsyncComponent(() => import('./MyInput.vue'))
}
```

### 动态组件类型

```vue
<script setup lang="ts">
import { shallowRef, type Component } from 'vue'
import Dashboard from './Dashboard.vue'
import Settings from './Settings.vue'
import Profile from './Profile.vue'

// 组件映射表
const componentMap: Record<string, Component> = {
  Dashboard,
  Settings,
  Profile
}

const currentView = shallowRef<Component>(Dashboard)

function switchView(name: keyof typeof componentMap) {
  currentView.value = componentMap[name]
  // switchView('Dashboard') → 切换到 Dashboard 组件
  // switchView('Invalid')   → 类型错误：不在映射表中
}
</script>

<template>
  <div>
    <nav>
      <button @click="switchView('Dashboard')">仪表盘</button>
      <button @click="switchView('Settings')">设置</button>
      <button @click="switchView('Profile')">个人资料</button>
    </nav>
    <component :is="currentView" />
  </div>
</template>
```

### 组件实例类型

```vue
<!-- MyForm.vue -->
<script setup lang="ts">
const props = defineProps<{ title: string }>()

// 暴露的方法
function validate(): boolean {
  return true
}
function reset(): void {
  console.log('表单已重置')
}

defineExpose({ validate, reset })
</script>
```

```vue
<!-- 父组件 -->
<script setup lang="ts">
import { ref, type ComponentExposed } from 'vue'
import MyForm from './MyForm.vue'

// 获取组件实例类型
type MyFormExposed = ComponentExposed<typeof MyForm>
// 或使用 InstanceType
type MyFormInstance = InstanceType<typeof MyForm>

const formRef = ref<MyFormExposed | null>(null)

function handleValidate() {
  formRef.value?.validate()
  // TypeScript 知道 validate 返回 boolean
}
</script>

<template>
  <MyForm ref="formRef" title="用户信息" />
  <button @click="handleValidate">验证表单</button>
</template>
```

### 异步组件

```ts
import { defineAsyncComponent, type Component } from 'vue'

// 带类型的异步组件
const AsyncChart = defineAsyncComponent({
  loader: () => import('./Chart.vue'),
  loadingComponent: {
    template: '<div>加载图表中...</div>'
  } as Component,
  errorComponent: {
    template: '<div>图表加载失败</div>'
  } as Component,
  delay: 200,
  timeout: 5000
})
```

## 三、注意事项与常见陷阱

1. **`<MyComponent />` 模板中自动推断类型**：通常不需要显式声明组件类型
2. **DefineComponent 的泛型参数复杂**：一般直接用 `as Component` 做类型断言
3. **shallowRef 用于存储组件**：避免 Vue 深度代理组件对象
4. **defineExpose 控制实例类型**：不暴露的属性在 ref 中不可访问
5. **异步组件的类型需要手动标注**：loadingComponent 等需要 `as Component`
