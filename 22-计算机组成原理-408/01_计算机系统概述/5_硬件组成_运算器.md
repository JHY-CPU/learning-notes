# 06_硬件组成_运算器

## 核心概念

### 运算器的功能

运算器（Arithmetic and Logic Unit，ALU所在的部件）是计算机中执行算术运算和逻辑运算的核心部件。它是数据加工处理的中心。

### 运算器的核心组成

| 部件 | 英文缩写 | 功能 |
|------|---------|------|
| 算术逻辑单元 | ALU | 执行加减乘除和逻辑运算（与、或、非、异或） |
| 累加器 | ACC（Accumulator） | 存放操作数和运算结果，是ALU的核心寄存器 |
| 乘商寄存器 | MQ（Multiplier-Quotient） | 乘法时存放乘数，除法时存放商 |
| 操作数寄存器 | X | 存放参加运算的操作数 |
| 通用寄存器组 | R0, R1, R2... | 存放各种数据，可灵活使用 |
| 状态字寄存器 | PSW | 存放运算状态标志（零标志、溢出标志等） |

### 各寄存器在运算中的作用

**加法运算**：
$$
ACC + X \rightarrow ACC
$$
- ACC存放被加数（兼放结果），X存放加数
- ALU执行加法，结果存回ACC

**乘法运算**：
$$
X \times MQ \rightarrow ACC \text{（高位）}, MQ \text{（低位）}
$$
- X存放被乘数，MQ存放乘数
- 乘积的高位存入ACC，低位存入MQ

**除法运算**：
$$
ACC \div X \rightarrow ACC \text{（余数）}, MQ \text{（商）}
$$
- ACC存放被除数，X存放除数
- 商存入MQ，余数存入ACC

## 原理分析

### 运算器内部数据流

```
通用寄存器组 ──→  ALU  ──→ 结果
                ↑   ↓
              ACC   MQ
                ↑
                X
```

ALU是一个组合逻辑电路，具有两个输入端和一个输出端：
- 输入端A：通常来自ACC
- 输入端B：通常来自X或MQ
- 输出端：结果通常送回ACC

ALU由**加法器**为核心构成：
- 加法运算：直接输出和
- 减法运算：将减数取补码后做加法
- 乘法运算：多次加法和移位
- 除法运算：多次减法和移位

### 状态字寄存器（PSW）标志位

| 标志位 | 名称 | 含义 |
|-------|------|------|
| Z | 零标志 | 运算结果为0时置1 |
| N/V | 溢出标志 | 运算溢出时置1 |
| C | 进位标志 | 最高位有进位时置1 |
| S | 符号标志 | 运算结果为负时置1 |

## 直观理解

### 生活类比

**运算器就像一个算盘**：
- ALU是算盘的运算部分（珠子和档位）
- ACC就像你的左手指，一直扶着当前数字
- MQ就像你的右手指，专门处理乘除
- X就像纸条上记着的第二个操作数

**加法**：左手（ACC）的数加上纸条（X）的数，结果放在左手上
**乘法**：纸条（X）的数乘以右手（MQ）的数，高位结果放左手，低位放右手

### 记忆技巧

运算器寄存器：「阿（ALU）才（ACC）没去（MQ）西（X）安」
- ALU：运算
- ACC：累加（核心寄存器）
- MQ：乘商
- X：操作数

### 常考组合

1. ACC/MQ/X在加法、乘法、除法中的不同作用（常考选择题）
2. ALU是组合逻辑电路还是时序电路（组合逻辑电路）
3. 运算器中哪些是程序员可见的（通用寄存器通常可见，但具体取决于ISA）

### 易错陷阱

1. **ALU是组合电路**，没有记忆功能；ACC、MQ等寄存器是时序电路
2. **乘法结果占用两个寄存器**（ACC和MQ），除法也占用两个
3. **状态字寄存器PSW**不是运算器独有的，它属于CPU的公共寄存器
4. **不同计算机的运算器组成可能不同**：上述是经典的典型模型
