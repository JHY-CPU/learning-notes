# 35_跨章联系_概述与各章

## 第1章与后续各章的联系

### 与第2章（数据的表示和运算）的联系

| 第1章概念 | 第2章深入内容 |
|----------|-------------|
| 二进制表示 | 原码、反码、补码、移码的详细定义和运算 |
| ALU功能 | ALU的详细设计，加法器的实现 |
| 存储字长 | 定点数和浮点数的表示格式 |
| 运算器 | 定点运算和浮点运算的硬件实现 |

**关键联系**：冯诺依曼结构要求"采用二进制"，第2章深入讲解二进制的各种编码方式和运算方法。

### 与第3章（存储系统）的联系

| 第1章概念 | 第3章深入内容 |
|----------|-------------|
| 存储器层次概述 | Cache的详细设计（映射、替换、写策略） |
| MAR/MDR | 主存的详细结构和扩展方法 |
| 存储容量计算 | 存储器的组织和扩展（位扩展、字扩展） |
| 虚拟机概念 | 虚拟存储器的实现（页式、段式、段页式） |

**关键联系**：第1章的存储器层次是宏观概述，第3章深入讲解每一层的具体实现。

### 与第4章（指令系统）的联系

| 第1章概念 | 第4章深入内容 |
|----------|-------------|
| 指令格式（OP+A） | 指令的详细格式（零地址、一地址、二地址等） |

| 操作码 | 操作码的编码方式（定长、变长） |
| RISC/CISC | 两种指令集设计哲学的详细对比 |

**关键联系**：第1章介绍了指令的基本格式，第4章深入讲解指令系统的设计。

### 与第5章（中央处理器）的联系

| 第1章概念 | 第5章深入内容 |
|----------|-------------|
| 运算器和控制器概述 | CPU内部数据通路的详细设计 |

| 取指-分析-执行 | 指令流水线的工作原理 |

**关键联系**：第1章是CPU的概览，第5章深入讲解CPU的内部实现和流水线技术。

### 与第6章（总线）的联系

| 第1章概念 | 第6章深入内容 |

### 与第7章（IO系统）的联系

## 学习建议

### 从第1章出发的学习路径

```
第1章（概述）
  ├── 第2章：理解数据如何在计算机中表示和运算
  ├── 第3章：理解存储器层次和各级存储器的实现
  ├── 第4章：理解指令系统的设计
  ├── 第5章：理解CPU的内部结构和工作原理
  ├── 第6章：理解各部件之间的通信机制
  └── 第7章：理解计算机与外部世界的交互方式
```

### 复习建议

1. **第1章是总纲**：先学好第1章，建立整体框架
2. **后续章节是细节**：在学习后续章节时，经常回看第1章的概述
3. **跨章联系很重要**：408考试经常考跨章节的综合题
4. **性能计算贯穿始终**：第1章的性能指标在各章中都有应用

### 408考试中的跨章联系

**常见组合**：
1. 存储器层次 + Cache命中率 + 性能计算
2. 指令格式 + CPI计算 + CPU执行时间
3. 总线带宽 + 存储器访问 + 性能分析
4. IO控制方式 + 中断处理 + 系统性能
