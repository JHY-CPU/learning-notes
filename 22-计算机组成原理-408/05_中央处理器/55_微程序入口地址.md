# 55_微程序入口地址

## 核心概念

### 微程序入口地址的定义
微程序入口地址是指**每条机器指令对应的微程序在控制存储器（CM）中的起始地址**。当CPU取到一条机器指令后，需要根据指令的操作码（opcode）找到对应的微程序入口地址，才能开始执行该指令的微操作序列。

### 入口地址映射问题
一条机器指令的操作码是一个n位的二进制数，需要将其转换为控制存储器中的微地址（m位）。这个转换过程称为**操作码→微地址映射**。

设操作码位数为 $k$，微地址位数为 $m$：
- 最多有 $2^k$ 条不同指令
- 控制存储器最多有 $2^m$ 个单元

### 地址映射方法

#### 1. 直接法（直接映射）
**原理**：将操作码直接拼接固定前缀，形成微程序入口地址。

**公式**：
$$\text{微地址} = \text{固定前缀} \parallel \text{操作码} \parallel 0\ldots0$$

**例子**：操作码为3位，CM地址为8位
- 指令ADD：操作码=001 → 入口地址=00000 001（即CM的第1号单元）
- 指令SUB：操作码=010 → 入口地址=00000 010（即CM的第2号单元）
- 指令LOAD：操作码=011 → 入口地址=00000 011（即CM的第3号单元）

**特点**：
- 实现简单，速度快（一次映射）
- 各指令微程序入口地址连续，可能造成存储空间浪费
- 不够灵活

#### 2. 间接法（间接映射/查找表法）
**原理**：用操作码作为索引，查找一个**入口地址表**，表中存放真正的微程序入口地址。

**流程**：
```
操作码 → 查找入口地址表（ROM）→ 微程序入口地址
```

**公式**：
$$\text{微地址} = \text{入口地址表}[\text{操作码}]$$

**例子**：
| 操作码 | 表中存储的入口地址 |
|--------|-------------------|
| 000 | 00100000 |
| 001 | 00100010 |
| 010 | 00110000 |
| 011 | 01000000 |

**特点**：
- 灵活性高，各微程序可以分散在CM任意位置
- 需要额外的ROM存储入口地址表
- 需要两次访问（先查表再取微指令），速度较慢

#### 3. PLA法（可编程逻辑阵列法）
**原理**：用PLA实现操作码到微地址的直接转换。

**特点**：
- 可以实现复杂的映射关系
- 硬件开销介于直接法和间接法之间
- 灵活性好

## 原理分析

### 直接映射法的硬件实现

设操作码为 $OP[7:0]$，微地址为 $A[9:0]$，固定前缀为 $00$：

$$A[9:0] = 00 \parallel OP[7:0]$$

实际电路中，就是一个简单的**拼接操作**：
```
微地址 = 00 + 操作码
```

### 间接映射法的硬件实现

```
                    ┌──────────────┐
  操作码 ──────────→│  入口地址ROM  │──────→ 微程序入口地址
  （地址输入）      │  （存储地址表）│
                    └──────────────┘
```

### 取指微程序与入口地址

取指微程序是**所有指令共享**的，其入口地址固定（通常为CM的0号单元）：
$$\text{取指微程序入口} = 000\ldots0$$

执行完取指微程序后，根据**当前指令的操作码**计算该指令微程序的入口地址。

完整流程：
```
取指微程序（入口固定）→ 微指令IR ← M[PC]
    ↓
根据IR中的操作码 → 计算该指令微程序入口地址
    ↓
执行该指令的微程序
    ↓
执行完毕 → 返回取指微程序（入口固定）
```

## 直观理解

### 生活类比：酒店找房间
- **直接法**：房间号直接由客人编号决定，如100号客人住100号房间。简单但浪费空间。
- **间接法**：客人编号先查前台的登记表，表上写着实际房间号。灵活但多一步。

### 记忆技巧
**"直接拼，间接查"**：
- 直接法：操作码直接拼到地址中
- 间接法：操作码查表得地址

### 常考组合与易错陷阱

**易错陷阱1**：
- 取指微程序的入口地址是**固定的**（通常是0号地址），不是由操作码决定的
- 操作码决定的是**各指令执行阶段**微程序的入口地址

**易错陷阱2**：
- 直接法中，操作码位数决定了CM中**预留的地址空间**
- 如果操作码有3位，直接法需要为8条指令预留地址空间（不管实际用几条）

**408常见考法**：
- 给定操作码和CM地址位数，计算各指令微程序入口地址
- 判断某种映射方法的优缺点
- 直接法和间接法的对比选择
