# 64_流水线性能指标_吞吐率

## 核心概念

- **吞吐率（Throughput, TP）**：单位时间内流水线完成的指令数
- 吞吐率是衡量流水线**整体效率**的关键指标
- 理想吞吐率 = 每时钟周期完成 1 条指令

## 原理分析

### 一、吞吐率的定义

$$TP = \frac{n}{T_{k}}$$

其中：
- $n$：完成的指令条数
- $T_k$：流水线完成 $n$ 条指令的总时间

### 二、理想吞吐率

对于 $k$ 段流水线，执行 $n$ 条指令：

$$T_k = (n + k - 1) \times \Delta t$$

$$TP = \frac{n}{(n + k - 1) \times \Delta t}$$

当 $n \to \infty$ 时：

$$TP_{max} = \frac{1}{\Delta t}$$

即**理想吞吐率 = 1/时钟周期**，每周期完成一条指令。

### 三、实际吞吐率

实际吞吐率受以下因素影响：

1. **流水线装填期**：前 $k-1$ 个周期没有指令完成
2. **流水线排空期**：最后 $k-1$ 个周期指令逐条完成
3. **冲突导致的暂停**：气泡减少了有效吞吐

$$TP_{实际} = \frac{n}{(n + k - 1 + \text{暂停周期数}) \times \Delta t}$$

### 四、最大吞吐率

$$TP_{max} = \lim_{n \to \infty} \frac{n}{(n + k - 1) \times \Delta t} = \frac{1}{\Delta t}$$

- 与流水线**段数 $k$ 无关**
- 只取决于**时钟周期 $\Delta t$**
- $\Delta t$ 取决于**最慢段的延迟**

### 五、吞吐率的瓶颈

当流水线各段延迟不等时：

$$\Delta t = \max(t_1, t_2, \ldots, t_k)$$

- 最慢段成为**瓶颈**
- 其他段会空闲等待
- 解决方案：将慢段细分，或使用缓冲

## 直观理解

**吞吐率**就像高速公路的车流量：

- 5 条车道（5 段流水线）
- 每辆车通过收费站需 1 分钟（时钟周期）
- 理想：每分钟通过 1 辆车（TP = 1）
- 实际：开始几辆车还在路上，最后几辆车还没到终点
- 车越多（$n$ 越大），实际吞吐率越接近理想值

### 简单例题

某 4 段流水线，时钟周期 100ns，执行 100 条指令。

$$T = (100 + 4 - 1) \times 100 = 10300 \text{ ns}$$

$$TP = \frac{100}{10300} = \frac{1}{103} \text{ 条/ns} \approx 9.7 \times 10^6 \text{ 条/s}$$

接近理想吞吐率 $1/100$ 条/ns $= 10^7$ 条/s。

## 代码/模拟

### Python计算流水线性能指标

```python
"""流水线性能指标计算 - 适用于408考研复习"""

def pipeline_performance(n, k, dt=1, stall_cycles=0):
    """
    计算流水线各项性能指标
    :param n: 指令条数
    :param k: 流水线段数
    :param dt: 每段延迟（默认1个时间单位）
    :param stall_cycles: 因冲突插入的暂停周期总数
    """
    # 流水线执行时间
    T_pipe = (n + k - 1 + stall_cycles) * dt
    # 非流水线执行时间
    T_seq = n * k * dt

    # 吞吐率
    TP = n / T_pipe          # 实际吞吐率
    TP_max = 1 / dt          # 理想最大吞吐率

    # 加速比
    S = T_seq / T_pipe

    # 效率
    eta = (n * k) / (k * T_pipe)

    print(f"流水线参数: {k}段, {n}条指令, dt={dt}, 暂停={stall_cycles}周期")
    print(f"  执行时间: T = ({n} + {k} - 1 + {stall_cycles}) × {dt} = {T_pipe}")
    print(f"  非流水线: T' = {n} × {k} × {dt} = {T_seq}")
    print(f"  吞吐率:   TP = {n}/{T_pipe} = {TP:.4f} (理想={TP_max:.4f})")
    print(f"  加速比:   S  = {T_seq}/{T_pipe} = {S:.2f} (理论最大={k})")
    print(f"  效率:     η  = {eta:.2%}")
    return TP, S, eta

print("=== 基本性能计算 ===")
pipeline_performance(n=100, k=5, dt=1)

print("\n=== 含暂停的性能影响 ===")
pipeline_performance(n=100, k=5, dt=1, stall_cycles=20)

print("\n=== 指令数对吞吐率的影响 ===")
for n in [5, 10, 50, 100, 1000]:
    T_pipe = (n + 5 - 1) * 1
    TP = n / T_pipe
    print(f"  n={n:>4}: TP={TP:.4f} (趋近理想值1.0)")
```
