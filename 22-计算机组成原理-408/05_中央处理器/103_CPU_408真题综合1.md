# 103_CPU_408真题综合1

## 核心概念

- 本节综合数据通路、微操作、控制信号的 408 真题
- 覆盖 CPU 设计的核心考法

## 原理分析

### 真题 1：数据通路 + 控制信号

**题目**：某机数据通路如下（单总线）：
- 功能部件：ALU（ADD/SUB）、MAR、MDR、IR、PC、R0-R7（通用寄存器）
- ALU 的两个输入来自总线和寄存器 A
- 控制信号：PCout, PCin, MARin, MDRin, MDRout, IRin, Rin, Rout, Ain, Cout, ADD, SUB, Read, Write

要求写出指令 `ADD R1, R2` 的微操作序列和控制信号。

**解**：

| 时钟 | 微操作 | 控制信号 |
|------|--------|---------|
| T0 | PC → MAR | PCout, MARin |
| T1 | M(MAR) → MDR, PC+1 → PC | Read, PC+1 |
| T2 | MDR → IR | MDRout, IRin |
| T3 | R2 → A | R2out, Ain |
| T4 | A + R1 → R1 | R1out, ADD, R1in |

### 真题 2：间址寻址微操作

**题目**：指令 `ADD R1, (R2)`（R2 中存放的是操作数地址），写出执行周期微操作。

**解**：

| 时钟 | 微操作 | 说明 |
|------|--------|------|
| T0 | R2 → MAR | 取操作数地址 |
| T1 | M(MAR) → MDR | 读取操作数 |
| T2 | MDR → A | 操作数送 A |
| T3 | A + R1 → R1 | 运算结果送 R1 |

### 真题 3：转移指令微操作

**题目**：指令 `BEQ R1, R2, offset`，写出微操作序列。

**解**：

| 时钟 | 微操作 | 说明 |
|------|--------|------|
| T0 | R1 → A | 取第一操作数 |
| T1 | A - R2 → Z | 比较，结果送零标志 Z |
| T2 | if (Z=1) PC+4+offset → PC | 相等则转移 |

### 真题 4：控制信号表

**题目**：填写以下指令的控制信号（部分）：

| 信号 | ADD R1,R2 | LD R1,0(R2) | ST R1,0(R2) | BEQ |
|------|-----------|-------------|-------------|-----|
| PCout | 1 | 1 | 1 | 1 |
| MARin | 1 | 1 | 1 | 1 |
| Read | 1 | 1 | 1 | 1 |
| IRin | 1 | 1 | 1 | 1 |
| RegRead | 1 | 1 | 1 | 1 |
| ALUop | ADD | ADD | ADD | SUB |
| MemWrite | 0 | 0 | **1** | 0 |
| RegWrite | **1** | **1** | 0 | 0 |
| Branch | 0 | 0 | 0 | **1** |
