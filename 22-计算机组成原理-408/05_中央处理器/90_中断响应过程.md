# 90_中断响应过程

## 核心概念

- **中断响应**：CPU 检测到中断请求后，暂停当前程序，转去执行中断服务程序
- 响应过程包括：**检查请求**、**关中断**、**保存现场**、**转向服务程序**
- 中断响应发生在**每条指令执行完毕后**

## 原理分析

### 一、中断响应的条件

#### 对于外部中断（INTR）

$$\text{响应条件} = (\text{有中断请求}) \land (IF = 1) \land (\text{当前指令执行完毕})$$

- 必须**开中断**（$IF = 1$）
- 必须在**指令边界**（当前指令执行完毕）

#### 对于 NMI

$$\text{响应条件} = (\text{有 NMI 请求}) \land (\text{当前指令执行完毕})$$

- 不受 IF 位限制

#### 对于异常

- 异常是**同步的**，在执行指令过程中就检测到
- 立即响应，不需要等到指令边界

### 二、中断响应过程（详细步骤）

#### 步骤 1：中断请求检测

- 每条指令结束时，CPU 检查中断请求线
- 外部中断：检查 INTR 和 NMI 引脚
- 内部中断：在执行过程中检测

#### 步骤 2：中断判优

- 多个中断同时请求时，按**优先级**排队
- 硬件排队器或软件查询

#### 步骤 3：关中断

$$IF \leftarrow 0$$

- 防止在保存现场过程中被新的中断打断
- 保证中断处理的**原子性**

#### 步骤 4：保存断点

- **保存 PC**：当前指令的下一条地址（断点）
- **保存 PSW**：当前程序状态字（标志寄存器）

$$\text{栈} \leftarrow PC, \quad \text{栈} \leftarrow PSW$$

#### 步骤 5：识别中断源

- 确定是哪个中断源发出的请求
- 获取**中断类型号**（向量号）

#### 步骤 6：转入中断服务程序

- 根据中断类型号，查**中断向量表**
- 获取中断服务程序入口地址

$$PC \leftarrow \text{中断向量表[中断类型号]}$$

### 三、中断隐指令

**中断隐指令**不是真正的指令，而是**硬件自动完成**的一系列操作：

$$\text{隐指令} = \text{关中断} + \text{保存 PC} + \text{保存 PSW} + \text{取服务程序入口}$$

- 由**中断响应周期**（中断周期）的硬件自动执行
- 程序员**不可见**，无对应机器指令

### 四、中断响应的时序

```
指令执行 ─→ 指令结束 ─→ 检查中断请求 ─→ 有请求?
                                    │
                              ┌──是─┤
                              │     ↓
                              │  关中断 (IF←0)
                              │     ↓
                              │  保存 PC → 栈
                              │     ↓
                              │  保存 PSW → 栈
                              │     ↓
                              │  取中断类型号
                              │     ↓
                              │  查向量表
                              │     ↓
                              │  PC ← 服务程序入口
                              │     ↓
                              └→ 执行中断服务程序
```

## 直观理解

中断响应像**办公室接电话**：

1. 听到电话铃响（检测请求）
2. 先把手头的事情写个便签（保存 PC/PSW）
3. 设定"请勿打扰"（关中断）
4. 看来电显示确定是谁（识别中断源）
5. 接电话处理事情（执行服务程序）

## 知识关联

- 与**中断周期**（第 17 节）的关系：中断周期就是执行隐指令的阶段
- 与**中断向量表**（第 92 节）的关系：步骤 6 需要查向量表
- 与**中断优先级**（第 93 节）的关系：步骤 2 需要判优
- 与**中断屏蔽**（第 94 节）的关系：步骤 3 关中断就是屏蔽
