# 89_中断源分类

## 核心概念

- **中断源**：引起中断的事件或设备
- 分类维度：内部/外部、可屏蔽/不可屏蔽、同步/异步
- 理清中断源分类是理解中断系统的**基础**

## 原理分析

### 一、按来源分类

#### 内部中断源（异常）

| 中断源 | 类型 | 说明 |
|--------|------|------|
| 除法溢出 | 故障 | 除数为 0 |
| 缺页 | 故障 | 页面不在内存 |
| 段越界 | 故障 | 访问超出段界限 |
| 系统调用 | 陷阱 | INT 指令 |
| 断点 | 陷阱 | INT 3 指令 |
| 溢出 | 陷阱 | INTO 指令 |
| 非法指令 | 终止 | 操作码不存在 |
| 硬件故障 | 终止 | 机器检查异常 |

#### 外部中断源（中断）

| 中断源 | 类型 | 说明 |
|--------|------|------|
| 键盘 | INTR | 按键事件 |
| 鼠标 | INTR | 移动/点击 |
| 硬盘 | INTR | IO 完成 |
| 网卡 | INTR | 数据到达 |
| 定时器 | INTR | 时间片到 |
| 电源掉电 | NMI | 紧急断电 |
| 内存校验错 | NMI | 奇偶校验错误 |

### 二、按可屏蔽性分类

| 类型 | 可否屏蔽 | 控制方式 | 优先级 |
|------|---------|---------|--------|
| NMI | **不可屏蔽** | 无 | 最高 |
| INTR | **可屏蔽** | IF 位 | 中 |
| 异常 | 不可屏蔽 | 无 | 高于 INTR |

### 三、按同步/异步分类

| 类型 | 同步/异步 | 特点 |
|------|----------|------|
| 异常 | **同步** | 与指令执行绑定，特定指令必然触发 |
| INTR | **异步** | 随时可能发生，与当前指令无关 |
| NMI | **异步** | 随时可能发生 |

### 四、优先级排序

$$\text{NMI} > \text{硬件故障(终止)} > \text{故障} > \text{陷阱} > \text{INTR}$$

## 直观理解

把中断源类比为**医院急诊分类**：

- **NMI**：心脏骤停，必须立即抢救
- **终止异常**：重伤，需要紧急处理
- **故障异常**：骨折，处理完可以继续活动（重新执行）
- **陷阱异常**：体检，有意识的行为（执行下一条）
- **INTR**：普通感冒，可以等一等（可屏蔽）
