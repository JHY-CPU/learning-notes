## 典型考题解析

### 考点1：CPU组成判断

**题目**：以下哪些部件属于CPU？（多选）
A. ALU  B. 通用寄存器  C. MAR  D. 主存储器  E. CU  F. Cache

**答案**：A、B、C、E

**解析**：
- A. ALU属于运算器，在CPU内 ✓
- B. 通用寄存器组在CPU内 ✓
- C. MAR通常算在CPU内（现代设计）✓
- D. 主存储器在CPU外 ✗
- E. CU属于控制器，在CPU内 ✓
- F. Cache不在CPU基本组成中（虽然芯片内可能集成）✗

### 考点2：用户可见 vs 透明寄存器

**题目**：以下哪个寄存器对用户透明？
A. PC  B. 通用寄存器R0  C. PSW  D. MAR

**答案**：D

**解析**：
- PC：用户可通过转移指令间接修改 → 可见
- R0：用户可通过指令直接读写 → 可见
- PSW：用户可通过特定指令访问（如PUSHF） → 可见
- MAR：用户无法通过任何指令直接访问 → **透明**

### 考点3：取指数据流

**题目**：在取指周期中，数据流的正确顺序是？

**答案**：PC → MAR → 地址总线 → 主存 → 数据总线 → MDR → IR

**详细微操作**：
1. PC_out, MAR_in（PC的内容送MAR）
2. Read（向主存发出读命令）
3. 主存将数据送到数据总线 → MDR_in
4. MDR_out, IR_in（指令送IR）
5. PC + 1 → PC

### 考点4：乘法寄存器使用

**题目**：执行乘法运算 X × Y 时，被乘数应放在哪个寄存器？乘数呢？结果如何存放？

**答案**：
- 被乘数 → X
- 乘数 → MQ
- 高位积 → ACC
- 低位积 → MQ

**记忆公式**：$X \times MQ \to ACC \| MQ$

### 考点5：MAR位数计算

**题目**：某机MAR为24位，MDR为32位，按字节编址。求：
(1) 可寻址空间大小
(2) 存储容量

**解答**：
(1) 可寻址存储单元数 = $2^{24}$ = 16M 个单元

(2) 因为按字节编址，每个单元8位（1字节）：
$$\text{容量} = 2^{24} \times 1 \text{ B} = 16 \text{ MB}$$

若按字编址（32位/字）：
$$\text{容量} = 2^{24} \times 4 \text{ B} = 64 \text{ MB}$$

### 考点6：内部总线与执行时间

**题目**：单总线结构的CPU中，执行"R1 ← R0 + R2"需要几个时钟周期？

**解答**：至少需要**3个时钟周期**：
- T0：R0 → 总线 → Y（ALU输入锁存器）
- T1：R2 → 总线 → ALU
- T2：ALU加法结果 → 总线 → R1

若用双总线，可将R0和R2分别送上两条总线，只需**2个时钟周期**。

## 直观理解

### 答题技巧

1. **选择题**：看到"CPU组成"，首先想到「运算器+控制器」，不要加主存。
2. **判断题**：看到"用户可见"，先排除MAR/MDR/IR（这三个是透明的）。
3. **计算题**：MAR位数 → $2^n$ → 乘以MDR位数（或字节数）→ 容量。
4. **大题**：画数据流图时，先确定数据通路类型（单/双/专用），再分步列出微操作。

### 常见陷阱汇总

| 陷阱 | 正确理解 |
|------|----------|
| CPU包含主存 | CPU = 运算器 + 控制器，不含主存 |
| PC是透明的 | PC是用户可见的（可通过指令修改） |
| MAR/MDR在CPU外 | 现代设计在CPU内，考试看题目 |
| MAR位数=机器字长 | MAR位数=地址总线宽度，MDR位数≈机器字长 |
| 乘法被乘数放MQ | 被乘数放X，乘数放MQ |
| 除法被除数放X | 被除数放ACC，除数放X |
