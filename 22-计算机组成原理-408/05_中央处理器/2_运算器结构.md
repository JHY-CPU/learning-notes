# 03_运算器结构

## 核心概念

- **运算器**：CPU中负责数据加工的部件，核心是ALU，配合各类寄存器和数据通路。
- **ALU（Arithmetic Logic Unit）**：算术逻辑单元，执行加减乘除和逻辑运算（与、或、非、异或等）。
- **累加器ACC（Accumulator）**：存放一个操作数和运算结果的专用寄存器。
- **乘商寄存器MQ（Multiplier-Quotient Register）**：乘法时存乘数和低位积，除法时存商。
- **操作数寄存器X**：ALU的通用输入寄存器，存放另一个操作数。
- **通用寄存器组**：R0~R(n-1)，可编程访问，存放操作数和中间结果。

### 运算器组成

| 部件 | 位宽 | 功能 |
|------|------|------|
| ALU | 与机器字长一致 | 核心运算部件，由组合逻辑电路实现 |
| ACC | n位 | 累加器，存放操作数和结果 |
| MQ | n位 | 乘商寄存器 |
| X | n位 | 操作数寄存器（ALU输入） |
| R0~R(n-1) | n位 | 通用寄存器组 |
| 数据通路 | n位宽 | 连接各寄存器与ALU |

## 原理分析

### ALU的内部实现

ALU是一个**组合逻辑电路**，输入为两个n位操作数和操作控制信号，输出为n位结果和状态标志。

$$F_i = f(A_i, B_i, C_0, C_1, ..., C_{m-1})$$

其中 $A_i$、$B_i$ 为两个操作数的第 $i$ 位，$C_0 \sim C_{m-1}$ 为控制信号（选择加/减/与/或等操作），$F_i$ 为输出的第 $i$ 位。

**状态标志位（PSW的一部分）**：
- **CF（Carry Flag）**：进位/借位标志
- **ZF（Zero Flag）**：结果为零标志
- **SF（Sign Flag）**：符号标志（结果最高位）
- **OF（Overflow Flag）**：溢出标志

### 各运算中寄存器的使用

#### 加法运算：ACC + X → ACC

微操作序列（单总线）：
1. ACC → 总线（将被加数送上总线，暂存于ALU输入锁存器Y）
2. X → 总线（将加数送上总线）
3. ALU 执行加法，结果 → 总线 → ACC

#### 乘法运算：X × MQ → ACC(高位) + MQ(低位)

$$\text{被乘数} \times \text{乘数} = \text{高位积} \| \text{低位积}$$

- 被乘数 → X
- 乘数 → MQ
- 运算后：高位积 → ACC，低位积 → MQ

#### 除法运算：ACC ÷ X → MQ(商) + ACC(余数)

$$\text{被除数} \div \text{除数} = \text{商}\cdots\text{余数}$$

- 被除数 → ACC
- 除数 → X
- 运算后：商 → MQ，余数 → ACC

### 数据通路与ALU的连接

```
通用寄存器组    ACC     MQ      X
  R0~R3         │       │       │
   │            │       │       │
   ▼            ▼       ▼       ▼
┌──────────────────────────────────┐
│         内部总线（Bus）           │
└───────────────┬──────────────────┘
                │
                ▼
          ┌───────────┐
    Y ──→ │           │
          │    ALU    │──→ 结果 → 总线 → 目标寄存器
    X ──→ │           │
          └───────────┘
                │
                ▼
          状态标志 (CF/ZF/SF/OF)
```

- Y是ALU的输入锁存器，从总线接收第一个操作数并保持。
- 第二个操作数直接从总线送入ALU的另一个输入端。
- ALU运算结果送上总线，写入目标寄存器。

## 直观理解

### 生活类比

运算器就像**一个计算器工作室**：
- **ALU**= 计算器核心芯片，能做各种运算
- **ACC**= 计算器的显示屏，显示当前数字和结果
- **MQ**= 记账本的辅助栏，乘除法时需要额外记录
- **X**= 另一个输入区，存放第二个数
- **通用寄存器**= 桌上的便签纸，临时记几个数字
- **数据通路**= 传送带，把数字从一个地方送到另一个地方

### 记忆技巧

**寄存器角色记忆口诀**（加减乘除）：
- 「**加减ACC为被操作，X为操作数，结果回ACC**」
- 「**乘法X被乘MQ乘，高位ACC低位MQ**」
- 「**除法ACC被除X除，商去MQ余留ACC**」
- 简化：加减看ACC+X→ACC；乘看X×MQ→ACC+MQ；除看ACC÷X→MQ+ACC

### 易错陷阱

1. **陷阱**：乘法中"被乘数"和"乘数"分别在X和MQ，而不是都在通用寄存器中。经典题目会考"执行乘法前各寄存器应如何设置"。
2. **陷阱**：单总线结构中同一时刻总线上只能有一个数据，因此两个操作数需要分两个时钟周期送上总线。
## 代码/模拟

### Verilog实现4位ALU

```verilog
// ============================================
// 4位ALU模块 - 对应408运算器结构知识点
// 支持: ADD, SUB, AND, OR, XOR, NOT
// ============================================

module alu_4bit(
    input  [3:0] a,          // 操作数A
    input  [3:0] b,          // 操作数B
    input  [2:0] alu_op,     // 操作选择
    output reg [3:0] result, // 运算结果
    output zero,             // 零标志 ZF
    output sign,             // 符号标志 SF
    output carry,            // 进位标志 CF
    output overflow          // 溢出标志 OF
);

    wire [4:0] sum_ext;      // 扩展一位用于进位检测

    always @(*) begin
        case (alu_op)
            3'b000: result = a + b;          // ADD
            3'b001: result = a - b;          // SUB
            3'b010: result = a & b;          // AND
            3'b011: result = a | b;          // OR
            3'b100: result = a ^ b;          // XOR
            3'b101: result = ~a;             // NOT
            default: result = 4'b0000;
        endcase
    end

    // 标志位计算
    assign zero    = (result == 4'b0000);           // ZF
    assign sign    = result[3];                      // SF = 最高位
    assign sum_ext = {1'b0, a} + {1'b0, b};
    assign carry   = sum_ext[4];                     // CF = 最高位进位
    // OF: 同号相加结果符号不同
    assign overflow = (a[3] == b[3]) && (result[3] != a[3]);

endmodule
```

### Python模拟运算器各运算的寄存器使用

```python
"""运算器寄存器分工模拟 - 适用于408考研复习"""

class ALURegisters:
    """模拟运算器中ACC、MQ、X各寄存器的使用"""

    def __init__(self, word_bits=8):
        self.ACC = 0   # 累加器
        self.MQ = 0    # 乘商寄存器
        self.X = 0     # 操作数寄存器
        self.bits = word_bits
        self.mask = (1 << word_bits) - 1

    def add(self, acc_val, x_val):
        """加法: ACC + X → ACC"""
        self.ACC = acc_val
        self.X = x_val
        print(f"加法: ACC={self.ACC}, X={self.X}")
        print(f"  微操作: ACC → Y(锁存), X → ALU另一输入")
        self.ACC = (self.ACC + self.X) & self.mask
        print(f"  结果: ACC ← {self.ACC}")
        return self.ACC

    def multiply(self, x_val, mq_val, n_bits=4):
        """乘法: X × MQ → ACC(高位) + MQ(低位)"""
        self.X = x_val
        self.MQ = mq_val
        print(f"乘法: X(被乘数)={self.X}, MQ(乘数)={self.MQ}")
        product = self.X * self.MQ
        full_bits = n_bits * 2
        self.ACC = (product >> n_bits) & self.mask  # 高位积
        self.MQ = product & self.mask               # 低位积
        print(f"  结果: ACC(高位积)={self.ACC}, MQ(低位积)={self.MQ}")
        print(f"  完整乘积={product}")
        return product

    def divide(self, acc_val, x_val):
        """除法: ACC ÷ X → MQ(商) + ACC(余数)"""
        self.ACC = acc_val  # 被除数
        self.X = x_val      # 除数
        print(f"除法: ACC(被除数)={self.ACC}, X(除数)={self.X}")
        if self.X == 0:
            print("  错误: 除数为零!")
            return None
        self.MQ = self.ACC // self.X    # 商
        self.ACC = self.ACC % self.X    # 余数
        print(f"  结果: MQ(商)={self.MQ}, ACC(余数)={self.ACC}")
        return (self.MQ, self.ACC)

# 运行示例
print("=== 运算器寄存器分工模拟 ===\n")
alu = ALURegisters(word_bits=8)

print("--- 加法: 15 + 20 ---")
alu.add(15, 20)

print("\n--- 乘法: 7 × 6 (4位) ---")
alu.multiply(7, 6, n_bits=4)

print("\n--- 除法: 23 ÷ 5 ---")
alu.divide(23, 5)

print("\n=== 寄存器角色速记表 ===")
print(f"{'运算':<8} {'被操作数':<10} {'操作数':<10} {'结果高位':<10} {'结果低位':<10}")
print("-" * 50)
print(f"{'加法':<8} {'ACC':<10} {'X':<10} {'ACC(和)':<10} {'—':<10}")
print(f"{'减法':<8} {'ACC':<10} {'X':<10} {'ACC(差)':<10} {'—':<10}")
print(f"{'乘法':<8} {'X':<10} {'MQ':<10} {'ACC(高位积)':<10} {'MQ(低位积)':<10}")
print(f"{'除法':<8} {'ACC':<10} {'X':<10} {'ACC(余数)':<10} {'MQ(商)':<10}")
```
