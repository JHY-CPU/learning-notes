# 98_中断处理_408考点

## 核心概念

- 中断处理是 408 的**核心高频考点**
- 常考内容：保存内容、中断隐指令、响应时机、中断类型号
- 题型：选择题 + 大题中结合数据通路出题

## 原理分析

### 一、408 必背知识点

#### 1. 中断隐指令自动完成的内容

- [x] 关中断（$IF \leftarrow 0$）
- [x] 保存断点（PC → 栈）
- [x] 保存 PSW → 栈
- [x] 识别中断源，获取中断类型号
- [x] 查中断向量表，获取服务程序入口

#### 2. 中断服务程序完成的内容

- [x] 保存通用寄存器（保护现场）
- [x] 开中断（允许嵌套）
- [x] 执行中断服务
- [x] 关中断
- [x] 恢复通用寄存器
- [x] 恢复 PSW
- [x] 恢复 PC
- [x] 开中断
- [x] 中断返回

#### 3. 保存内容总结

| 内容 | 保存者 | 保存位置 | 时机 |
|------|--------|---------|------|
| PC | 硬件（隐指令） | 栈 | 响应时 |
| PSW | 硬件（隐指令） | 栈 | 响应时 |
| 通用寄存器 | 软件（服务程序） | 栈 | 进入服务后 |

### 二、常见 408 选择题

**题 1**：中断响应时，保存的 PC 值是？

A. 当前正在执行的指令地址（错误）
B. **下一条指令的地址**（正确）
C. 中断服务程序入口地址（错误）
D. 中断向量表地址（错误）

**题 2**：以下哪个不是中断隐指令完成的？

A. 关中断（是）
B. 保存 PC（是）
C. **保存通用寄存器**（不是！这是服务程序做的）
D. 查向量表（是）

**题 3**：中断服务程序中两次关中断分别在什么时候？

A. 进入时和退出前
B. **响应时（隐指令）和恢复现场前**
C. 保存 PC 前和恢复 PC 后
D. 开中断后和中断返回前

**答案**：B（第一次在隐指令中自动关，第二次在服务程序恢复前手动关）

### 三、408 大题常见考法

**考法 1**：结合数据通路，写出中断响应的微操作序列

**考法 2**：给定指令执行时序，判断中断何时响应

**考法 3**：计算中断响应时间

### 四、易错点

1. **"中断隐指令保存通用寄存器"** —— 错误！通用寄存器由服务程序保存
2. **"关中断在服务程序中才做"** —— 错误！隐指令就已经关中断了
3. **"异常也是在指令边界响应"** —— 错误！异常在执行过程中就检测到
4. **"NMI 可以被 IF 屏蔽"** —— 错误！NMI 不可屏蔽
