# 33_DMA_408考点

## 核心概念

本节汇总408考研中DMA相关的高频考点。

## 原理分析

### 考点一：DMA传送方式选择

**常考选择题**：

1. **周期挪用是最常用的DMA传送方式** — 对
2. **停止CPU方式中CPU完全停止** — 不完全对(Cache可能继续工作)
3. **交替访问不需要总线仲裁** — 错(仍需仲裁)
4. **周期挪用每次挪用一个数据块** — 错(一个存取周期传一个字)

### 考点二：周期挪用原理

**408经典考法**：给出CPU执行序列，判断DMA可以挪用多少个周期。

**例题**：某CPU执行一条指令需要3个时钟周期(取指1周期+执行2周期)。取指必须访存，执行阶段第1个周期访存(读操作数)，第2个周期不访存(ALU运算)。CPU时钟频率400MHz，总线宽度32位。求DMA的最大可用带宽。

**解答**：
- 每条指令3个周期，其中2个必须访存，1个空闲
- 每秒指令数 = $400MHz / 3 = 133.3M$ 条
- 每秒可挪用周期数 = $133.3M \times 1 = 133.3M$ 个
- DMA最大带宽 = $133.3M \times 4B = 533.3MB/s$

### 考点三：DMA优先级

**DMA优先级通常高于中断**：
- DMA有时间限制(如磁盘窗口期)
- 中断可以等待(推迟几微秒影响不大)
- 如果DMA错过窗口(如磁盘扇区转过)，数据丢失

**优先级排序**(从高到低)：
1. DMA(最高)
2. 不可屏蔽中断(NMI)
3. 可屏蔽中断(INTR)
4. 程序查询

### 考点四：DMA与中断的本质区别

| 区别 | DMA | 中断 |
|------|-----|------|
| 数据传送 | 硬件直接(DMA控制器) | 软件(CPU执行指令) |
| 响应时机 | 每个数据(或每个数据块) | 数据块完成后 |
| 数据通路 | 设备→主存(不经CPU) | 设备→CPU→主存 |
| 适合数据量 | 大批量 | 小量 |
| CPU开销 | 仅预/后处理 | 每个数据都有开销 |

## 直观理解

**DMA优先级高的原因类比**：
- 磁盘就像旋转的唱片机，数据窗口转过去就没了(时间敏感)
- 中断就像等电话，晚几秒接不影响(可延迟)

## 知识关联

### 408典型选择题

**题1**：关于DMA方式，以下说法正确的是？
- A. DMA方式完全不需要CPU参与 → 错(预处理和后处理需要CPU)
- B. DMA传送过程中CPU不能执行任何指令 → 错(CPU可以执行不访存的指令)
- C. DMA优先级高于中断 → 对
- D. DMA控制器有独立的指令系统 → 错(没有)

**题2**：周期挪用方式的特点是？
- A. DMA独占总线 → 错
- B. 每次挪用一个存取周期 → 对
- C. CPU被完全暂停 → 错
- D. 需要双端口存储器 → 错(交替访问才需要)

### 易错陷阱

1. **DMA不是零CPU开销**：预处理和后处理需要CPU
2. **周期挪用不等于DMA效率低**：只是DMA可用带宽受CPU影响
3. **DMA不需要中断**：数据传送不需要，但后处理通知需要
4. **多个DMA可以同时存在**：如磁盘DMA和网卡DMA，通过优先级仲裁
