# 50_IO综合_408真题2

## 核心概念

本节继续汇总408考研中IO系统的综合真题。

## 原理分析

### 真题1：中断响应条件判断

**题目**：CPU响应中断请求的条件是（ ）。
A. 当前指令执行完毕
B. 中断允许触发器IE=1
C. 有中断请求
D. 以上都是

**答案**：D

**解析**：中断响应需要同时满足四个条件：
1. 有中断请求(INTR=1)
2. 中断允许(IE=1)
3. 当前指令执行完毕
4. 无更高优先级中断正在处理
- 选项A、B、C分别对应条件3、2、1，全部正确 → D

### 真题2：中断屏蔽字

**题目**：某系统有4个中断源I1~I4，硬件响应优先级I1>I2>I3>I4。要求中断处理次序为I4>I2>I3>I1。各中断源的屏蔽字应设为（ ）。

**解答**：

处理次序 $I_4 > I_2 > I_3 > I_1$：
- $I_4$最先处理，不被任何中断打断 → 屏蔽字 = 1111
- $I_2$第二个处理，只允许$I_4$打断 → 屏蔽字 = 0111 (I4位=0)
- $I_3$第三个处理，允许$I_4, I_2$打断 → 屏蔽字 = 0111 → 修正：允许$I_4$和$I_2$打断 → I4位=0, I2位=0 → 0101... 

等等，让我重新分析。处理次序$I_4 > I_2 > I_3 > I_1$意味着：
- 处理$I_1$时：允许$I_2, I_3, I_4$打断(它们都应该先处理完) → 屏蔽字 = 1000
- 处理$I_2$时：允许$I_4$打断(先于$I_2$处理) → 屏蔽字 = 0110
- 处理$I_3$时：允许$I_4, I_2$打断 → 屏蔽字 = 0101
- 处理$I_4$时：不被任何打断 → 屏蔽字 = 1111

| 中断源 | 处理顺序 | 屏蔽字(1 2 3 4) |
|--------|----------|----------------|
| $I_1$ | 第4 | 1000 |
| $I_2$ | 第2 | 0110 |
| $I_3$ | 第3 | 0101 |
| $I_4$ | 第1 | 1111 |

### 真题3：DMA传送时间计算

**题目**：某系统总线宽度64位，总线时钟频率200MHz。用DMA方式传送8KB数据，采用停止CPU方式。求：
(1) DMA需要多少个总线周期？
(2) CPU被暂停多长时间？

**解答**：
- 总线宽度64位 = 8字节
- 数据量8KB = 8192字节
- (1) 传送次数 = $8192 / 8 = 1024$ 次 → 1024个总线周期
- 总线周期 = $1 / 200MHz = 5ns$
- (2) 暂停时间 = $1024 \times 5ns = 5120ns = 5.12\mu s$

### 真题4：中断处理现场保存

**题目**：在中断处理过程中，以下哪些内容是由硬件自动保存的（ ）。
A. 通用寄存器
B. 程序计数器(PC)
C. 程序状态字(PSW)
D. 堆栈指针(SP)

**答案**：B、C

**解析**：
- PC和PSW：由中断响应阶段硬件自动压栈保存 → B、C对
- 通用寄存器：由中断服务程序中软件(PUSH指令)保存 → A错
- SP(堆栈指针)：一般不需要保存(SP本身就是用来保存其他内容的) → D错

### 真题5：IO端口编址

**题目**：采用统一编址方式的计算机系统中（ ）。
A. IO端口地址与存储器地址重叠
B. IO端口占用主存地址空间
C. 需要专门的IO指令
D. IO端口不能用MOV指令访问

**答案**：B

**解析**：
- A错：不重叠，是共享同一地址空间的不同区域
- **B对**：统一编址的定义就是IO端口占用主存地址空间
- C错：统一编址不需要专门IO指令，用MOV即可
- D错：统一编址方式用MOV访问IO端口

### 真题6：通道类型选择

**题目**：某大型机有多个IO设备，包括键盘(低速)、磁带机(中速)、高速磁盘(高速)。应分别使用什么类型的通道？

**解答**：
- 键盘(低速) → **字节多路通道**(分时共享，适合多个低速设备)
- 磁带机(中速) → **数组多路通道**(成组分时，适合中速设备)
- 高速磁盘(高速) → **选择通道**(独占，适合高速设备)

## 直观理解

**做题口诀**：
- 保存现场：硬件保PC+PSW，软件保通用寄存器
- 统一编址：用MOV，不需IO指令
- 独立编址：用IN/OUT，需专用IO指令
- 通道类型：低速字节多路，中速数组多路，高速选择通道

## 知识关联

### 易错总结

1. **中断响应vs中断处理**：响应是硬件，处理是软件
2. **统一编址不是地址重叠**：是共享，不同区域不重叠
3. **DMA暂停时间取决于数据量和总线宽度**：不是固定的
4. **屏蔽字构造要明确位序**：高位对应高优先级中断源
