# 32_DMA三种方式对比

## 核心概念

DMA的三种数据传送方式是408考试的高频对比考点。

## 原理分析

### 三种方式全面对比表(必背)

| 对比项 | 停止CPU访存 | 周期挪用(窃取) | 交替访问 |
|--------|------------|---------------|---------|
| **基本思想** | DMA独占总线 | DMA利用CPU空闲周期 | CPU和DMA各用一半周期 |
| **CPU暂停情况** | 完全暂停(整个传送期间) | 暂停一个存取周期 | 不暂停 |
| **CPU受影响程度** | 最大 | 较小 | 无 |
| **DMA可用带宽** | 最高(100%) | 中等(视空闲情况) | 较低(50%) |
| **CPU可用带宽** | 0%(暂停期间) | 剩余空闲周期 | 50% |
| **存储器利用率** | 高(DMA独占) | 较高 | 100%(但分时) |
| **硬件复杂度** | 最简单 | 中等 | 最复杂(需双端口) |
| **适用场景** | 数据量小 | 最通用 | 高性能系统 |
| **典型应用** | 简单嵌入式 | PC、服务器 | 高端服务器 |

### 时间线对比

**停止CPU方式**：
```
├── CPU ──┤────── DMA(传送整个数据块) ──────┤── CPU ──→
          │←──── CPU长时间暂停 ────→│
```

**周期挪用方式**：
```
├CPU取指─┤CPU执行─┤CPU取指─┤CPU执行─┤CPU取指─┤
         │DMA挪用│        │DMA挪用│        │
         (暂停1个)        (暂停1个)
```

**交替访问方式**：
```
├CPU|DMA├CPU|DMA├CPU|DMA├CPU|DMA├CPU|DMA├CPU|DMA├
 各用半个存取周期，完全不暂停CPU
```

### 数据传输率对比

设存储器存取周期为 $T_m$，存储器宽度为 $W$ 位。

**停止CPU**：
$$R_{\text{DMA}} = \frac{W}{T_m}$$

**周期挪用**（假设每 $k$ 个CPU周期可挪用1个）：
$$R_{\text{DMA}} = \frac{W}{k \times T_m}$$

**交替访问**：
$$R_{\text{DMA}} = \frac{W}{2T_m}$$

### CPU性能影响

**停止CPU**：
- $T_{\text{暂停}} = N \times T_m$（N为传送字数）
- CPU执行延迟 = $T_{\text{暂停}}$
- 如果CPU有Cache且命中率高，影响可降低

**周期挪用**：
- 总延迟 = $n \times T_m$（n为实际挪用次数）
- 一般 $n < N$（不是每个周期都能挪用）
- 对CPU的影响较小

**交替访问**：
- CPU不暂停，但存取带宽减半
- CPU访存延迟 = $2 \times T_m$（等两个半周期才能轮到自己）

## 直观理解

**三种方式的类比**：

1. **停止CPU** = 高速公路封路施工，所有车辆等待
2. **周期挪用** = 施工人在红灯间隙快速修一块，绿灯让车过
3. **交替访问** = 单行道分时通行，各走各的时间段

**选择建议**：
- 小数据量、简单系统 → 停止CPU
- 通用系统 → 周期挪用(最常用)
- 高端系统、需要CPU零延迟 → 交替访问

## 知识关联

### 408典型题

**题1**：DMA三种传送方式中，CPU利用率最高的是？ — 交替访问(不暂停CPU)

**题2**：DMA三种传送方式中，硬件最简单的是？ — 停止CPU访存

**题3**：周期挪用方式中，DMA每次挪用？ — 一个存取周期

**题4**：交替访问方式中，DMA可用带宽为存储器总带宽的？ — 50%

### 易错陷阱

1. **"停止CPU"不等于关闭CPU**：CPU可能继续从Cache执行
2. **周期挪用的次数不固定**：取决于CPU的访存模式
3. **交替访问不是50%吞吐量**：存储器利用率100%，只是每个设备50%
4. **三种方式都需要总线仲裁**：只是仲裁方式不同
