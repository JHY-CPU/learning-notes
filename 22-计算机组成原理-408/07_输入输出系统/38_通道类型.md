# 39_通道类型

## 核心概念

通道根据数据传送方式和设备管理方式分为三种类型。

### 三种通道类型

| 类型 | 英文名 | 管理方式 | 适合设备 | 数据传输率 |
|------|--------|----------|----------|-----------|
| **字节多路通道** | Byte Multiplexer Channel | 分时共享 | 低速设备 | 最低 |
| **数组多路通道** | Block Multiplexer Channel | 成组分时 | 中高速设备 | 中等 |
| **选择通道** | Selector Channel | 独占 | 最高速设备 | 最高 |

## 原理分析

### 1. 字节多路通道(Byte Multiplexer Channel)

**工作方式**：为每个连接的设备轮流分配一个字节的传送时间。

```
时间轴:
├── 设备A字节1 ──┤── 设备B字节1 ──┤── 设备C字节1 ──┤── 设备A字节2 ──┤── ...
   (一个时间片)     (一个时间片)     (一个时间片)     (一个时间片)
```

**特点**：
- 多个低速设备分时共享通道
- 每个设备每次只传一个字节(或一个字)
- 通道在设备之间快速切换
- 适合速度差异大的多个低速设备

**类比**：老师轮流给多个学生辅导，每次给一个学生讲一道题

### 2. 数组多路通道(Block Multiplexer Channel)

**工作方式**：为每个设备轮流分配一个数据块(数组)的传送时间。

```
时间轴:
├── 设备A传数据块1 ──────────────┤── 设备B传数据块1 ──────────────┤── ...
   (传送N个字节，然后切换)           (传送N个字节，然后切换)
```

**特点**：
- 每个设备每次传送一个数据块
- 传完一个数据块后切换到下一个设备
- 适合多个中高速设备
- 兼顾了传输效率和设备共享

**类比**：快递员给多个小区轮流送件，每次给一个小区送完一批再转下一个

### 3. 选择通道(Selector Channel)

**工作方式**：通道一次只连接一个设备，独占通道直到传送完毕。

```
时间轴:
├── 设备A传送整个数据 ──────────────────────┤── 设备B传送整个数据 ────────────┤
   (通道被设备A独占)                           (通道被设备B独占)
```

**特点**：
- 一次只连接一个设备
- 设备独占通道，直到整个IO操作完成
- 数据传输率最高
- 适合最高速的设备(如高速磁盘)

**类比**：高速公路专用车道，一辆车(设备)独占整条路直到到达目的地

### 三种类型对比表(必背)

| 对比项 | 字节多路通道 | 数组多路通道 | 选择通道 |
|--------|-------------|-------------|---------|
| 数据单位 | 字节(1个) | 数据块(若干字节) | 整个数据 |
| 设备连接数 | 多个同时 | 多个轮流 | 一次一个 |
| 利用方式 | 分时(时间片) | 分时(成组) | 独占 |
| 通道利用率 | 低(频繁切换) | 较高 | 最高 |
| 适合设备 | 键盘、鼠标 | 磁带机 | 高速磁盘 |
| 传输率 | 低 | 中 | 高 |
| 设备速度 | < 几KB/s | 几KB~几MB/s | > 几MB/s |

## 直观理解

**三种通道的类比**：

1. **字节多路通道** = 公共汽车
   - 多个乘客(设备)轮流上下车
   - 每站(时间片)只处理一个乘客的一个需求
   - 适合乘客多但每人需求少的情况

2. **数组多路通道** = 大巴旅游团
   - 多个旅游团(设备)轮流用车
   - 每个团用完一整天(传完一个数据块)再换下一个
   - 适合多个中等规模需求

3. **选择通道** = 包车
   - 一个客户(设备)包下整辆车
   - 直到行程结束才释放
   - 适合大批量紧急运输

## 知识关联

### 408常考

1. **字节多路通道适合低速设备** — 常考选择题
2. **选择通道传输率最高** — 但只能服务一个设备
3. **数组多路通道是折中方案** — 兼顾效率和共享

### 易错陷阱

1. **字节多路不是同时传多个字节**：是分时轮流传，每次一个字节
2. **选择通道不是"选择数据"**：是独占方式，"选择"指一次选一个设备
3. **数组多路不是真正的并行**：仍然是分时的，只是每个设备分到的时间较长
4. **实际系统可能混合使用**：一个大型机可能同时有多种类型的通道
