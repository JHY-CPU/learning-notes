# 37_DMA综合_408真题

## 核心概念

本节汇总408考研中DMA相关的综合真题。

## 原理分析

### 真题1：DMA传送方式分析

**题目**：某计算机系统中，CPU主频为500MHz，采用DMA方式与硬盘交换数据。硬盘数据传输率为4MB/s，每次DMA传送的数据宽度为32位。采用周期挪用方式。CPU平均每条指令有2个存取周期，每个存取周期1个时钟周期。求：
(1) 每秒需要多少次DMA周期挪用？
(2) 由于周期挪用，CPU速度降低了多少？

**解答**：

(1) 硬盘数据传输率 = 4MB/s，每次传送4字节(32位)
- 每秒传送次数 = $4MB / 4B = 1M$ 次
- 每秒需要1M次周期挪用

(2) CPU每秒执行指令数 = $500MHz / 2 = 250M$ 条
- 每秒总存取周期数 = $250M \times 2 = 500M$ 个
- DMA挪用1M个周期
- CPU速度降低 = $1M / 500M = 0.2\%$
- **结论**：对CPU速度影响极小

### 真题2：DMA时间计算

**题目**：某系统用DMA方式从磁盘读取数据。磁盘转速7200RPM，每磁道512个扇区，每扇区512字节。总线宽度32位，总线频率100MHz。采用停止CPU方式传送一个扇区。求：
(1) 磁盘数据传输率
(2) 传送一个扇区的DMA时间
(3) CPU被暂停的时间

**解答**：

(1) 磁盘转速 = 7200RPM = 120转/秒
- 每转数据量 = $512 \times 512 = 262144$ 字节 = 256KB
- 数据传输率 = $120 \times 256KB = 30720KB/s \approx 30MB/s$

(2) 一个扇区 = 512字节
- 总线宽度 = 4字节
- 传送次数 = $512 / 4 = 128$ 次
- 总线周期 = $1 / 100MHz = 10ns$
- DMA传送时间 = $128 \times 10ns = 1280ns = 1.28\mu s$

(3) 采用停止CPU方式，CPU暂停时间 = DMA传送时间 = 1.28μs

### 真题3：中断与DMA对比

**题目**：某IO设备每秒产生1000个数据，每个数据1字节。CPU时钟频率1GHz，每次中断开销500个时钟周期。比较中断方式和DMA方式的CPU利用率。

**解答**：

**中断方式**：
- 每秒中断1000次
- 每次中断开销 = $500 / 1GHz = 500ns$
- 每秒中断总开销 = $1000 \times 500ns = 500\mu s$
- CPU用于IO的比例 = $500\mu s / 1s = 0.05\%$

**DMA方式**：
- 预处理 + 后处理 = 约2次中断的开销 = $2 \times 500ns = 1\mu s$ (假设每秒传完一次)
- 如果DMA每秒传完一次(1000字节)：CPU开销 = $1\mu s / 1s = 0.0001\%$
- **结论**：中断方式CPU开销是DMA的500倍

### 真题4：周期挪用计算

**题目**：某计算机CPU主频800MHz，采用周期挪用DMA。每条指令2个存取周期(取指+执行)，取指必须访存，执行周期40%需要访存。总线宽度64位。求DMA最大可用带宽。

**解答**：
- 每条指令可挪用周期数 = $2 - 1 - 0.4 \times 1 = 0.6$ 个
- (取指必须访存=1个，执行周期有0.4个需要访存，所以0.6个空闲)
- 每秒指令数 = $800MHz / 2 = 400M$ 条
- 每秒可挪用 = $400M \times 0.6 = 240M$ 个周期
- 每次挪用传8字节(64位)
- DMA最大带宽 = $240M \times 8B = 1920MB/s = 1.92GB/s$

## 直观理解

**DMA计算关键思路**：
1. 确定数据宽度(每次传多少)
2. 确定传送次数(总数据量/每次量)
3. 确定每次时间(总线周期)
4. 算总时间或带宽

## 知识关联

### 易错陷阱

1. **数据宽度决定传送次数**：32位总线每次传4字节，不是1字节
2. **周期挪用次数与CPU访存模式相关**：不是固定值
3. **停止CPU方式DMA带宽=存储器带宽**：不是总线带宽
4. **注意单位换算**：MB=1024KB=1048576B(二进制)或MB=1000KB(十进制，看题意)
