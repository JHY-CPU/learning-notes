# 46_LCD显示器

## 核心概念

- **LCD(Liquid Crystal Display，液晶显示器)**：利用液晶分子的光电效应来控制光的通过与否
- **TFT(Thin Film Transistor，薄膜晶体管)**：有源矩阵驱动方式，每个像素有独立的晶体管
- **当前主流显示器技术**

## 原理分析

### LCD显示原理

```
背光层 → 偏振片(水平) → 液晶层 → 偏振片(垂直) → 观察者
                              ↑
                         电压控制液晶
                         分子旋转角度
                         控制通过的光量
```

**工作原理**：
1. 背光层发出白光(光源)
2. 第一个偏振片只允许水平偏振光通过
3. 液晶层：施加电压改变液晶分子排列，旋转光的偏振方向
4. 第二个偏振片只允许垂直偏振光通过
5. 通过控制电压，控制每个像素的亮度

### TFT有源矩阵

- **无源矩阵**：简单但响应慢，对比度低(早期LCD)
- **有源矩阵(TFT)**：每个像素有一个TFT晶体管控制
  - 响应速度快
  - 对比度高
  - 色彩准确
  - 主流技术

```
扫描线 →─┬── TFT ──┬── 数据线
          │   │    │
         栅极 源极 漏极
               │
          像素电容(存储电压，保持亮度)
```

### LCD vs CRT对比

| 对比项 | CRT | LCD |
|--------|-----|-----|
| 显像原理 | 电子束扫描荧光屏 | 液晶控制光通过 |
| 体积/重量 | 大而重 | 薄而轻 |
| 功耗 | 高 | 低 |
| 辐射 | 有(电磁辐射) | 几乎没有 |
| 刷新率要求 | 需要高刷新率(防闪烁) | 不需要(像素持续发光) |
| 可视角度 | 好(各方向相同) | 早期较差，现在改善 |
| 响应时间 | 快(微秒级) | 慢(毫秒级，已改善) |
| 分辨率 | 固定(由点距决定) | 固定(物理像素数) |
| 亮度均匀性 | 中心亮边缘暗 | 均匀 |

### LCD的刷新机制

- **CRT**：电子束必须不断扫描，否则荧光粉亮度衰减 → 需要高刷新率
- **LCD**：像素电压保持不变时亮度不变 → 不需要不断刷新
- LCD驱动电路仍然按刷新率更新，但主要是为了接收新帧数据

## 直观理解

**LCD类比**：百叶窗
- 背光 = 阳光
- 液晶层 = 可调节角度的百叶窗叶片
- 加电压 = 调节叶片角度
- 叶片全开 = 像素亮(光全通过)
- 叶片全关 = 像素暗(光被挡住)
- 不同角度 = 不同亮度(灰度)

## 知识关联

### 408常考

1. **LCD不需要高刷新率**：像素持续发光，CRT才需要
2. **TFT是有源矩阵驱动**：每个像素一个晶体管
3. **LCD功耗低于CRT**：液晶本身不发光
4. **LCD响应时间慢于CRT**：液晶分子转动需要时间

### 易错陷阱

1. **LCD有背光**：液晶本身不发光，需要背光
2. **LCD也有刷新率**：但意义与CRT不同，主要是数据更新频率
3. **LCD的"坏点"**：TFT晶体管故障导致像素异常
4. **LCD分辨率是物理固定的**：不像CRT可以灵活调整
