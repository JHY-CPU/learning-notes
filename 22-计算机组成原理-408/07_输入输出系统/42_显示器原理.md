# 43_显示器原理

## 核心概念

- **显示器**：最重要的输出设备，将计算机处理结果以视觉形式呈现
- **显示系统** = 显示器(硬件) + 显示控制器(显卡) + 显存(帧缓冲)

### 显示器关键参数

| 参数 | 含义 | 常见值 |
|------|------|--------|
| **分辨率** | 屏幕像素数(宽×高) | 1920×1080, 2560×1440, 3840×2160 |
| **刷新率** | 每秒重绘次数(Hz) | 60Hz, 120Hz, 144Hz |
| **颜色深度** | 每像素用几位表示颜色 | 8位(256色), 24位(真彩色), 32位 |
| **点距** | 相邻像素中心距离 | 0.25~0.30mm |
| **响应时间** | 像素从一种颜色切换到另一种的时间 | 1~5ms(液晶) |

### 显存容量计算

$$\text{显存容量} = \text{分辨率(像素数)} \times \text{颜色深度(位)} / 8$$

**例**：1920×1080分辨率，24位真彩色

$$\text{显存} = 1920 \times 1080 \times 24 / 8 = 1920 \times 1080 \times 3 = 6,220,800 \text{字节} \approx 6MB$$

**注意**：实际显存还需要考虑双缓冲(两个帧缓冲交替)和Z缓冲(3D渲染)，实际需要量更大。

$$\text{实际显存} \approx 2 \times \text{帧缓冲} + \text{其他缓冲}$$

### 显示控制器(显卡)

- **功能**：将显存中的图像数据转换为显示器能接受的视频信号
- **组成**：
  - 显示控制芯片(GPU)
  - 显存(帧缓冲存储器)
  - 数模转换器(DAC，模拟显示器用)
  - 视频输出接口(VGA/HDMI/DP)

```
CPU → 系统总线 → 显卡(显示控制器) → 视频信号 → 显示器
                   ├── GPU
                   ├── 显存(帧缓冲)
                   └── DAC/数字接口
```

## 原理分析

### CRT显示器工作原理

- **CRT(Cathode Ray Tube，阴极射线管)**：传统的CRT显示器
- **电子束扫描**：电子枪发出电子束，逐行扫描荧光屏
- **扫描方式**：
  - **逐行扫描**：从上到下逐行扫描
  - **隔行扫描**：先扫奇数行，再扫偶数行(减少闪烁)
- **刷新**：电子束不断重复扫描，保持图像亮度
- **刷新率**：60Hz表示每秒扫描60次

### 帧缓冲与显示

```
帧缓冲(显存)              显示屏幕
┌──────────────────┐    ┌──────────────────┐
│ P(0,0) P(1,0)...│    │ ●  ●  ●  ●  ...  │
│ P(0,1) P(1,1)...│ →  │ ●  ●  ●  ●  ...  │
│ ...              │    │ ...              │
└──────────────────┘    └──────────────────┘
每个像素 = 帧缓冲中一个存储单元
```

- 帧缓冲中每个单元对应屏幕上一个像素
- 显示控制器按扫描顺序读取帧缓冲
- 读出的像素值转换为颜色信号输出到显示器

## 直观理解

**帧缓冲类比**：马赛克拼图
- 显存 = 每个小格子的颜料配方(颜色编号)
- 显示器 = 按配方拼出来的马赛克画
- 刷新 = 不断重新读配方、重新上色(保持画面)

## 知识关联

### 显存带宽需求

$$\text{显存带宽} = \text{分辨率} \times \text{颜色深度} \times \text{刷新率}$$

**例**：1920×1080 × 24位 × 60Hz

$$= 1920 \times 1080 \times 3 \times 60 = 373,248,000 \text{B/s} \approx 356 \text{MB/s}$$

### 408常考

1. **显存容量计算**：分辨率×颜色深度
2. **帧缓冲就是显存的核心**：存储待显示的图像数据
3. **刷新率与闪烁**：低于60Hz会感到闪烁

### 易错陷阱

1. **显存容量≠分辨率×颜色深度**：要考虑双缓冲、3D缓冲等
2. **刷新率不是帧率**：显示器刷新率固定，帧率取决于GPU渲染速度
3. **CRT和LCD的刷新原理不同**：CRT靠电子束扫描，LCD靠像素持续发光
