# 34_DMA_计算详解

## 核心概念

DMA相关的计算题是408的高频考点，主要包括传送时间、带宽、周期挪用数等。

## 原理分析

### 基本公式汇总

| 计算项 | 公式 |
|--------|------|
| 存取周期 | $T_m = \frac{1}{f_m}$（$f_m$为存储器频率） |
| 传送一个字的时间 | $T_{\text{word}} = T_m$ |
| 传送N个字的时间(停止CPU) | $T = N \times T_m$ |
| 数据传输率(停止CPU) | $R = \frac{W}{T_m} = W \times f_m$ |
| 周期挪用DMA带宽 | $R_{\text{DMA}} = \frac{n}{T_{\text{inst}}} \times W$ |
| 交替访问DMA带宽 | $R_{\text{DMA}} = \frac{W}{2T_m}$ |

其中 $W$ 为总线宽度(位)，$n$ 为每条指令可挪用周期数，$T_{\text{inst}}$ 为指令周期。

### 例题1：DMA传送时间计算

某系统总线宽度32位，总线时钟频率100MHz。用DMA方式传送4KB数据，采用停止CPU方式。求：
(1) 传送需要多少个总线周期？
(2) 传送总时间是多少？

**解答**：
- 总线宽度32位 = 4字节
- 数据量 = 4KB = 4096字节
- (1) 传送次数 = $4096 / 4 = 1024$ 次
- 每次1个总线周期 → 共1024个总线周期
- (2) 总线周期 = $1 / 100MHz = 10ns$
- 总时间 = $1024 \times 10ns = 10240ns = 10.24\mu s$

### 例题2：周期挪用带宽计算

CPU主频600MHz，平均每条指令2个时钟周期。每条指令有1个取指周期(必须访存)和1个执行周期(可能访存)。执行周期中，50%的指令需要访存(读操作数)，50%不需要(ALU运算)。总线宽度64位。求DMA的最大可用带宽。

**解答**：
- 每条指令2个周期
- 每条指令必须访存1次(取指)
- 50%的指令执行周期不访存 → 可挪用
- 每条指令平均可挪用数 = $0.5 \times 1 = 0.5$ 个周期
- 每秒指令数 = $600MHz / 2 = 300M$ 条
- 每秒可挪用周期数 = $300M \times 0.5 = 150M$ 个
- DMA最大带宽 = $150M \times 8B = 1200MB/s = 1.2GB/s$

### 例题3：DMA与中断方式时间对比

从磁盘读取1MB数据到主存。磁盘数据传输率40MB/s。比较中断方式和DMA方式的CPU开销。

**中断方式**：
- 磁盘数据传输率40MB/s → 每字节间隔 = $1 / 40M = 25ns$
- 若每次中断开销20μs，每次传1字节
- 1MB = 1048576次中断
- CPU总中断开销 = $1048576 \times 20\mu s = 20.97s$
- 实际传输时间 = $1MB / 40MB/s = 25ms$
- CPU因IO被占用20.97s(远超传输时间)

**DMA方式**：
- 预处理：10μs
- 后处理(1次中断)：20μs
- CPU总开销 = 30μs
- **节省比**：$20.97s / 30\mu s = 699000$ 倍

### 例题4：交替访问方式带宽

某存储器存取周期200ns，数据宽度64位。采用交替访问方式。求DMA可用带宽。

**解答**：
- 存储器带宽 = $64bit / 200ns = 320Mbit/s = 40MB/s$
- DMA可用带宽 = $40MB/s / 2 = 20MB/s$
- (交替方式下DMA只能用一半存储器带宽)

## 直观理解

**计算技巧**：
1. **先确定数据宽度**：总线宽度决定每次传送多少字节
2. **再确定传送次数**：总数据量 / 每次传送量
3. **最后算时间**：次数 × 每次时间

## 知识关联

### 易错陷阱

1. **KB和Kb不同**：1KB = 8Kb，注意单位
2. **MHz是10^6 Hz**：100MHz = 100 × 10^6 Hz
3. **停止CPU方式DMA带宽 = 存储器带宽**：不是总线带宽
4. **周期挪用受CPU访存模式影响**：实际可用带宽可能小于理论最大值
5. **数据宽度注意字节对齐**：32位总线每次传4字节
