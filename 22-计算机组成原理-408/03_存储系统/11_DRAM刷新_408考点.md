# 11_DRAM刷新_408考点

## 核心概念

本文件汇总DRAM刷新相关的408常考计算题和易错点。

### 刷新的基本公式

- **刷新周期** $T_{\text{refresh}}$：完成所有行一轮刷新的最大允许时间（通常2ms或64ms）
- **存储矩阵行数** $n$：需要刷新的行数
- **存取周期** $t_c$：一次读写操作的时间
- **每次刷新一行的时间**：等于1个存取周期 $t_c$
- **刷新所有行的总时间**：$n \times t_c$

## 原理分析

### 考点一：集中刷新的死时间计算

**例题1**：某DRAM有128行，存取周期为0.5μs，刷新周期为2ms。求集中刷新的死时间和死时间占比。

**解答**：
- 刷新128行需要的时间 = $128 \times 0.5 = 64\,\mu\text{s}$
- 死时间 = $64\,\mu\text{s}$
- 2ms内总存取周期数 = $2000 / 0.5 = 4000$ 个
- 可用于正常读写的周期数 = $4000 - 128 = 3872$ 个
- 死时间占比 = $128 / 4000 = 3.2\%$

**计算公式**：

$$\text{死时间占比} = \frac{n}{T_{\text{refresh}} / t_c} = \frac{n \times t_c}{T_{\text{refresh}}}$$

### 考点二：分散刷新的存取周期

**例题2**：某DRAM有256行，原存取周期为0.5μs。采用分散刷新后，每个存取周期变为多少？多长时间完成一轮刷新？

**解答**：
- 分散刷新将每个存取周期分为两半：前半读写，后半刷新一行
- 新存取周期 = $0.5 + 0.5 = 1.0\,\mu\text{s}$
- 一轮刷新时间 = $256 \times 1.0 = 256\,\mu\text{s}$
- 远小于2ms，满足要求

### 考点三：异步刷新的刷新间隔

**例题3**：某DRAM有512行，刷新周期为2ms。求异步刷新的刷新间隔。

**解答**：
$$\text{刷新间隔} = \frac{2\,\text{ms}}{512} = \frac{2000\,\mu\text{s}}{512} = 3.906\,\mu\text{s}$$

即每隔约3.906μs刷新一行。

### 考点四：给定参数求刷新方式相关

**例题4**：某DRAM芯片容量为16M×1位，存储矩阵为4096行×4096列，存取周期为100ns，刷新周期为64ms。求：
(1) 集中刷新的死时间
(2) 异步刷新的刷新间隔

**解答**：
(1) 集中刷新死时间 = $4096 \times 100\,\text{ns} = 409600\,\text{ns} = 0.4096\,\text{ms}$
(2) 异步刷新间隔 = $64\,\text{ms} / 4096 = 15.625\,\mu\text{s}$

### 考点五：判断刷新是否满足要求

**例题5**：某DRAM有1024行，存取周期为200ns，刷新周期为2ms。采用分散刷新是否可行？每轮刷新需要多长时间？

**解答**：
- 分散刷新后每个存取周期 = $200 + 200 = 400\,\text{ns}$
- 一轮刷新 = $1024 \times 400 = 409600\,\text{ns} = 0.4096\,\text{ms}$
- $0.4096\,\text{ms} < 2\,\text{ms}$，满足要求

### 考点六：CPU占用率（有效带宽损失）

**例题6**：某DRAM有128行，存取周期为500ns，刷新周期为2ms。集中刷新方式下，CPU可用带宽损失多少？

**解答**：
- 总周期数 = $2000000 / 500 = 4000$ 个
- 刷新占用 = 128 个
- 带宽损失 = $128 / 4000 = 3.2\%$

## 直观理解

### 刷新计算的速记口诀

```
集中刷新算死区：行数 × 存取周期
分散刷新周期翻：原周期 × 2
异步刷新间隔：刷新周期 ÷ 行数
死时间占比：死时间 ÷ 刷新周期
```

### 关键数字记忆

| 参数 | 典型值 |
|------|--------|
| 刷新周期 | 2ms（传统）或64ms（现代） |
| 一行刷新时间 | 1个存取周期 |
| 刷新行数 | 取决于芯片（64~8192行） |

## 知识关联

### 常考组合

1. **刷新方式对比**（选择题）：三种方式的死时间、速度影响
2. **死时间/刷新间隔计算**（计算题）
3. **存取周期变化**（分散刷新使存取周期翻倍）
4. **带宽损失计算**（死时间占比 = 带宽损失比例）

### 易错陷阱

1. **刷新一行的时间 = 1个存取周期**，不是半个
2. **分散刷新后存取周期翻倍**，但没有"死时间"——因为读写和刷新交替进行
3. **集中刷新的死时间是连续的**，期间CPU完全不能访存
4. **异步刷新每行的死时间 = 1个存取周期**，但分散在各处
5. **刷新周期（2ms）和存取周期（纳秒级）差好几个数量级**，单位换算要注意
6. **存储矩阵的行数 ≠ 芯片容量**：行数 × 列数 = 容量，但题目可能直接给行数
7. **刷新由硬件完成，不占用CPU指令周期**，但占用访存时间（CPU需等待）
