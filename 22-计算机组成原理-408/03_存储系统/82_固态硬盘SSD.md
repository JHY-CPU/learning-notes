# 83_固态硬盘SSD

## 核心概念
- **SSD (Solid State Drive)**: 基于闪存芯片的固态存储设备
- **无机械部件**: 没有磁头臂和旋转盘片, 全电子化
- **闪存(Flash Memory)**: 非易失性半导体存储器
- **NAND Flash**: SSD的主流存储介质

### SSD vs HDD对比

| 对比项 | HDD | SSD |
|--------|-----|-----|
| 存储介质 | 磁性盘片 | NAND闪存芯片 |
| 机械部件 | 有(磁头臂、主轴) | 无 |
| 随机读取 | ~10ms | ~0.1ms |
| 顺序读取 | ~200MB/s | ~3500MB/s(NVMe) |
| 抗震性 | 差(怕摔) | 好(无机械部件) |
| 功耗 | 高(5~10W) | 低(2~5W) |
| 噪音 | 有(旋转声) | 无 |
| 容量/价格 | 低(便宜) | 高(较贵) |
| 写入寿命 | 理论无限 | 有限(擦写次数) |

## 原理分析

### SSD内部结构

```
┌──────────────────────────────────────┐
│              SSD控制器                │
│  (FTL映射、磨损均衡、垃圾回收...)      │
├──────┬──────┬──────┬──────┬──────────┤
│闪存  │闪存  │闪存  │闪存  │  DRAM    │
│芯片0 │芯片1 │芯片2 │芯片3 │  缓存    │
│      │      │      │      │(映射表)  │
└──────┴──────┴──────┴──────┴──────────┘
         多通道并行访问
```

### NAND Flash存储原理

**基本单元**: 浮栅晶体管(Floating Gate Transistor)

```
控制栅(Control Gate)
    |
绝缘层(隧穿氧化层)
    |
浮栅(Floating Gate) ← 电子存储在这里
    |
绝缘层
    |
源极 ──── 沟道 ──── 漏极
```

**存储原理**:
- 浮栅中有电子 → 阈值电压高 → 代表"0"
- 浮栅中无电子 → 阈值电压低 → 代表"1"

**SLC/MLC/TLC/QLC**:

| 类型 | 每单元位数 | 电压状态 | 寿命 | 速度 | 成本 |
|------|-----------|---------|------|------|------|
| SLC | 1 | 2 | 最长(10万次) | 最快 | 最贵 |
| MLC | 2 | 4 | 较长(1万次) | 快 | 中等 |
| TLC | 3 | 8 | 一般(3000次) | 较慢 | 便宜 |
| QLC | 4 | 16 | 短(1000次) | 慢 | 最便宜 |

### 读写擦除特性

SSD的基本操作与HDD完全不同:

| 操作 | HDD | SSD |
|------|-----|-----|
| 读 | 磁头感应磁场 | 读取晶体管阈值电压 |
| 写 | 磁头改变磁场 | 注入电子到浮栅 |
| 擦除 | 无需(直接覆写) | 必须先擦除才能写 |

**关键问题: 写前必须擦除!**

```
SSD的最小单位:
- 页(Page): 最小读写单位(4KB~16KB)
- 块(Block): 最小擦除单位(含64~256页)

不能写单个页!
必须: 擦除整个块 → 再写入页
```

**读快写慢的原因**:
- 读: 直接检测电压(快, ~25μs)
- 写: 编程注入电子(较慢, ~200μs)
- 擦除: 需要整个块(最慢, ~2ms)

### 读写性能不对称

| 操作 | 延迟 |
|------|------|
| 页读取 | ~25 μs |
| 页编程(写) | ~200 μs |
| 块擦除 | ~2 ms |

**写操作比读慢8倍, 擦除比写慢10倍!**

这导致SSD的随机写性能问题:
- 新数据写入: 需要擦除旧块 → 写入新块
- 频繁小写: 大量块擦除, 性能急剧下降

## 直观理解

**图书馆 vs 电子图书馆类比**:
- HDD = 传统图书馆, 找书要走(寻道), 翻页(旋转), 效率低
- SSD = 电子图书馆, 输入书名直接跳转(无寻道), 但删除书需要整排清空(擦除块)

### 记忆口诀
> **"SSD无机械, 读快写较慢; 写前必须擦, 擦以块为单位"**

## 知识关联

### FTL层 (Flash Translation Layer)

FTL是SSD控制器中的关键软件层:
- **功能**: 将逻辑页地址映射到物理页地址
- **作用**: 隐藏擦除前必须擦除的限制, 对外呈现类似HDD的接口
- **类比**: 类似虚拟存储中的页表

### 常考对比
1. HDD存取时间 = 寻道 + 旋转 + 传输
2. SSD存取时间 = 控制器开销 + 闪存访问(无寻道无旋转)
3. SSD的"寻道时间" ≈ 0
4. SSD顺序读写比随机读写快(因为控制器优化)

### 408考点
- SSD的基本结构(控制器+闪存芯片)
- 读写擦除的单位不同(页读写, 块擦除)
- SSD vs HDD的性能特点
- 写入寿命有限的原因
