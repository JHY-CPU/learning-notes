# 02_存储器分类

## 核心概念

存储器有多种分类方式，408考试要求掌握每种分类的依据和典型代表。

### 一、按存储介质分类

| 类型 | 存储原理 | 典型代表 | 易失性 |
|------|---------|---------|--------|
| **半导体存储器** | 双稳态触发器(SRAM)或电容电荷(DRAM) | Cache、内存条 | SRAM/DRAM易失 |
| **磁表面存储器** | 磁性材料磁化方向 | 硬盘、磁带 | 非易失 |
| **光存储器** | 激光改变介质反射特性 | CD、DVD、蓝光 | 非易失 |

### 二、按存取方式分类

- **随机存取存储器（RAM, Random Access Memory）**
  - 特点：存取时间与存储单元的物理位置**无关**
  - 代表：主存储器（DRAM）、Cache（SRAM）
  - 任何地址的访问时间相同

- **顺序存取存储器（SAM, Sequential Access Memory）**
  - 特点：存取时间与存储单元的物理位置**有关**，必须按顺序查找
  - 代表：磁带
  - 访问时间 = 寻道时间 + 等待时间 + 读写时间

- **直接存取存储器（DAM, Direct Access Memory）**
  - 特点：先直接定位到小区域（寻道），再在区域内顺序查找
  - 代表：机械硬盘（HDD）
  - 是"随机+顺序"的折中：磁头直接移到目标磁道，再等盘片旋转到目标扇区

> **考试重点**：硬盘是**直接存取**存储器，不是随机存取也不是顺序存取！这个区分是408高频考点。

- **相联存储器（Associative Memory）/ 按内容访问存储器（CAM）**
  - 特点：不是按地址访问，而是按**内容**访问——给出数据内容，返回匹配的地址
  - 代表：TLB（快表）、Cache的标记比较
  - 速度极快但成本高

### 三、按可改写性分类

| 类型 | 全称 | 特点 |
|------|------|------|
| **ROM** | Read-Only Memory | 出厂写入，用户只能读不能写 |
| **PROM** | Programmable ROM | 用户可一次性写入（熔丝编程） |
| **EPROM** | Erasable PROM | 紫外线擦除，可多次重写 |
| **EEPROM** | Electrically Erasable PROM | 电擦除，可多次重写，可按字节擦写 |
| **Flash ROM** | 闪存 | 电擦除，按块擦除，U盘/SSD使用 |

### 四、按信息易失性分类

- **易失性存储器（Volatile）**：断电后信息丢失
  - 典型：SRAM、DRAM
  - 需要持续供电才能保持数据

- **非易失性存储器（Non-volatile）**：断电后信息不丢失
  - 典型：ROM系列、磁盘、光盘、Flash

> **易错点**：ROM是非易失性的，但**不是所有非易失性存储器都是ROM**。磁盘和光盘也是非易失性的，但它们不是ROM。

## 原理分析

### SRAM存储原理（简介，详见第8讲）
- 利用**双稳态触发器**存储1位信息
- 有电就有稳定状态，不需要刷新
- 6个晶体管存储1位，集成度低

### DRAM存储原理（简介，详见第9讲）
- 利用**电容**充放电存储1位信息
- 电容会漏电，必须定期**刷新**
- 1个晶体管+1个电容存储1位，集成度高

### 磁盘存储原理
- 利用磁性材料的**剩磁**特性
- 磁化方向表示0/1（正向=1，反向=0）
- 非易失性因为磁化状态不会自动消失

### ROM为什么不可改写（掩膜ROM）
- 掩膜ROM在制造时通过**光刻掩膜**决定每个存储单元是0还是1
- 物理结构已固定，无法改变
- 成本最低，适合大批量生产（如BIOS芯片）

## 直观理解

**按存取方式的类比**：
- **随机存取** = 你有一排储物柜，知道编号就能直接打开（图书馆的开放书架）
- **顺序存取** = 一卷磁带，想找第10首歌必须快进到那里（录音带）
- **直接存取** = 硬盘像唱片机——唱臂直接移到目标轨道（快），但要等盘片转到正确位置（等）

**按可改写性的类比**：
- **ROM** = 石碑上刻的字（永久）
- **PROM** = 一次性写字的纸（写完不能改）
- **EPROM** = 用橡皮擦了重新写的纸（擦除麻烦）
- **EEPROM** = 电子白板（按字节擦除，方便）
- **Flash** = U盘（按块擦除，现代主流）

### 易错陷阱
1. **硬盘不是随机存取存储器**！它是直接存取存储器（DAM）
2. **Flash ROM是ROM家族的**，但它可以多次擦写，不是"只读"的
3. **Cache用SRAM，主存用DRAM**——不要搞反
4. **U盘/SSD属于Flash存储器**，是非易失性的，但不是ROM（ROM特指不可在线改写的）
5. **"按内容访问"不是"按地址访问"的反义词**——相联存储器是通过比较内容来查找地址
