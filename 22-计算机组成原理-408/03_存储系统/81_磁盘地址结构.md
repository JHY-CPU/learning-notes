# 81_磁盘地址结构

## 核心概念
- **CHS寻址**: 柱面号(Cylinder) + 磁头号(Head) + 扇区号(Sector)
- **LBA寻址**: 逻辑块地址(Linear Block Addressing), 线性编号
- **寻址过程**: 先定位柱面 → 再选磁头 → 最后等扇区

### CHS地址结构

$$\text{物理地址} = (\text{柱面号 C}, \text{磁头号 H}, \text{扇区号 S})$$

| 字段 | 范围 | 作用 |
|------|------|------|
| 柱面号 C | 0 ~ (柱面数-1) | 控制磁头臂移动(寻道) |
| 磁头号 H | 0 ~ (磁头数-1) | 选择哪个盘面的磁头 |
| 扇区号 S | 0 ~ (扇区数-1) | 选择磁道上的哪个扇区 |

### 为什么先定位柱面?

```
访问顺序:
1. 移动磁头臂 → 定位到目标柱面(最慢, 寻道时间)
2. 选择磁头   → 切换电子开关(极快, 几乎0时间)
3. 等待扇区   → 等磁盘旋转到目标扇区(中等, 旋转延迟)
```

**原因**:
- 磁头臂移动是**机械运动**, 需要几毫秒
- 磁头选择是**电子开关**, 只需几纳秒
- 所以先做最慢的操作(寻道), 再做最快的操作(选磁头)

## 原理分析

### CHS寻址过程

```
CPU给出CHS地址(C, H, S)
        |
        v
磁盘控制器接收
        |
        v
1. 移动磁头臂到第C柱面 ← 寻道时间(ms级)
        |
        v
2. 激活第H号磁头 ← 电子开关(ns级)
        |
        v
3. 等待第S号扇区旋转到磁头下方 ← 旋转延迟(ms级)
        |
        v
4. 读/写数据 ← 传输时间(μs级)
```

### LBA到CHS的转换

现代磁盘使用LBA寻址, 控制器内部转换为CHS:

$$\text{LBA} = C \times (H_{\text{总}} \times S_{\text{每道}}) + H \times S_{\text{每道}} + S$$

反向转换:
$$C = \text{LBA} \div (H_{\text{总}} \times S_{\text{每道}})$$
$$H = (\text{LBA} \bmod (H_{\text{总}} \times S_{\text{每道}})) \div S_{\text{每道}}$$
$$S = \text{LBA} \bmod S_{\text{每道}}$$

**例**: 磁盘有1000柱面, 8磁头, 每磁道50扇区
- CHS = (2, 3, 10)
- LBA = $2 \times (8 \times 50) + 3 \times 50 + 10 = 800 + 150 + 10 = 960$

### CHS寻址的限制

**容量限制**:
- BIOS INT 13h: 柱面10bit(0~1023), 磁头4bit(0~15), 扇区6bit(1~63)
- 最大容量 = $1024 \times 16 \times 63 \times 512$B $\approx 8$ GB
- 这就是老BIOS只能识别8GB硬盘的原因

**扩展CHS**:
- LBA-assisted: 利用LBA突破CHS限制
- INT 13h扩展: 使用LBA代替CHS

### 现代LBA寻址

```
LBA 0 ─→ 扇区0 (柱面0, 磁头0, 扇区0)
LBA 1 ─→ 扇区1 (柱面0, 磁头0, 扇区1)
...
LBA N-1 ─→ 最后一个扇区
```

**LBA优势**:
- 简单线性编号, 不需要关心物理结构
- 突破CHS的容量限制
- 支持逻辑扇区重映射(坏块管理)

**LBA48**: 48位LBA地址, 支持最大 $2^{48} \times 512$B = 128 PB

## 直观理解

**图书馆找书类比**:
- **柱面号** = 第几个书架区(需要走路过去)
- **磁头号** = 书架区的第几层(按按钮选择, 很快)
- **扇区号** = 第几本书(等传送带把书转过来)

先走到正确的书架区(最慢), 再按按钮选层(最快), 最后等书转到面前(中等速度)。

### 记忆口诀
> **"CHS三层定位, 先柱面(寻道), 再磁头(电子), 最后扇区(旋转)"**

### 常见陷阱
- 扇区号从1开始还是从0开始? (老式CHS从1开始, LBA从0开始)
- 磁头切换时间: 实际上不是0, 但相对于寻道可忽略
- 同一柱面的不同磁道不需要寻道, 这是扇区交错存取的基础
