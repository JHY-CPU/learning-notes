# 80_磁盘格式化

## 核心概念
- **格式化(Formatting)**: 在磁盘上建立数据结构, 使其能存储数据
- **低级格式化(物理格式化)**: 划分磁道和扇区, 写入扇区标记
- **分区(Partitioning)**: 将磁盘划分为多个逻辑区域
- **高级格式化(逻辑格式化)**: 在分区内建立文件系统(目录、FAT等)

### 格式化层次

```
全新磁盘(出厂)
    |
    v
低级格式化(物理格式化)
    ├── 划分磁道
    ├── 划分扇区
    ├── 写入扇区头部信息(ID字段、同步字段、间隙)
    └── 建立扇区标记(地址标记、CRC校验)
    |
    v
分区
    ├── 将磁盘分为多个分区(如C盘、D盘)
    └── 写入分区表(MBR或GPT)
    |
    v
高级格式化(逻辑格式化)
    ├── 建立文件系统(FAT32, NTFS, ext4...)
    ├── 建立根目录
    ├── 初始化FAT表或inode表
    └── 标记坏扇区(映射到备用扇区)
    |
    v
可用磁盘(可存储文件)
```

## 原理分析

### 低级格式化(Physical Format)

**目的**: 在空白磁性介质上建立物理存储结构

**操作**:
1. 将每个磁道划分为固定数量的扇区
2. 每个扇区写入:
   - **头部间隙(GAP)**: 同步信号
   - **ID字段**: 磁道号、扇区号、校验码(CRC)
   - **数据字段**: 空白(或出厂测试数据)
   - **尾部间隙**

**扇区结构**:
```
┌──────┬──────────┬──────────┬──────┐
│ SYNC │ ID+HEADER│   DATA   │ GAP  │
│(同步)│ (扇区头) │ (数据区) │ (间隙)│
└──────┴──────────┴──────────┴──────┘
```

**容量损失**: 低级格式化后, 有效容量 < 标称容量
- 每扇区需要额外字节存放ID、校验、间隙
- 典型: 标称512B扇区, 实际占用约571B

### 分区(Partitioning)

**目的**: 将物理磁盘划分为逻辑区域

**分区表**:
- **MBR**(主引导记录): 位于磁盘0柱面0磁头1扇区, 支持4个主分区
- **GPT**(GUID分区表): 现代标准, 支持128个分区, 支持>2TB磁盘

**MBR结构**:
```
┌──────────────────────────────┐
│  446B 引导代码                │
├──────────────────────────────┤
│  64B 分区表(4×16B)           │
├──────────────────────────────┤
│  2B 结束标记(0x55AA)         │
└──────────────────────────────┘
```

### 高级格式化(Logical Format)

**目的**: 在分区内建立文件系统

**操作**:
1. 创建文件系统数据结构
   - **FAT32**: 文件分配表(FAT) + 根目录
   - **NTFS**: MFT(主文件表)
   - **ext4**: inode表 + 块位图 + inode位图
2. 建立根目录
3. 标记坏块(映射到保留扇区)
4. 写入引导扇区信息

**格式化后可用容量** < 分区容量
- 文件系统本身需要占用空间
- 簇/块的内部碎片

## 直观理解

**盖房子类比**:
- **低级格式化** = 在空地上画好每个房间的轮廓(划分扇区)
- **分区** = 把地皮分成几个区域(客厅区、卧室区...)
- **高级格式化** = 在每个区域里布置家具(建立文件系统)

### 关键数字

| 格式化层次 | 耗时 | 数据可恢复性 |
|-----------|------|-------------|
| 低级格式化 | 数小时 | 极难恢复(物理覆盖) |
| 分区 | 秒级 | 可恢复(只是改分区表) |
| 高级格式化 | 分钟级 | 可恢复(只是改文件系统) |

### 格式化与容量

**标称容量**: 厂商标注(如1TB)
**低级格式化后**: 约标称容量的93%~97%
**高级格式化后**: 再扣除文件系统开销

### 常考问题

**Q: 低级格式化会破坏数据吗?**
A: 会, 它直接在磁介质上写入扇区标记, 覆盖原有数据。

**Q: 高级格式化后数据能恢复吗?**
A: 可以, 高级格式化只修改文件系统元数据, 实际数据仍在磁盘上。

**Q: 为什么新硬盘需要格式化?**
A: 出厂时磁盘是空白介质, 需要低级格式化建立扇区结构, 再分区和高级格式化才能使用。

### 现代磁盘的格式化
- 现代硬盘出厂时已完成低级格式化
- 用户只需分区和高级格式化
- 低级格式化会严重缩短磁盘寿命(写入量大)
