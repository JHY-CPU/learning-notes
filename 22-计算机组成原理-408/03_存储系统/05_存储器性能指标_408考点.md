# 05_存储器性能指标_408考点

## 核心概念

本文件汇总存储器性能指标相关的408常考题型和易错点。

## 原理分析

### 考点一：给MAR/MDR求容量

**例题1**：某计算机主存按字节编址，MAR为20位，MDR为8位，求主存容量。

$$\text{容量} = 2^{20} \times 8\,\text{bit} = 1\,\text{MB}$$

**例题2**：MAR为24位，MDR为16位，按字编址，求主存容量。

$$\text{容量} = 2^{24} \times 16\,\text{bit} = 2^{24} \times 2\,\text{B} = 32\,\text{MB}$$

**注意**：按字编址时，一个地址对应一个字（16位=2字节），不是1字节。

### 考点二：给容量反求MAR位数

**例题3**：某机主存容量为128KB，按字节编址，求MAR至少需要多少位？

$$128\,\text{KB} = 128 \times 1024 = 2^{17}\,\text{B}$$

按字节编址，需要 $2^{17}$ 个地址，因此MAR至少需要 **17位**。

**例题4**：主存容量为64K×16位，按字编址，求MAR位数。

- $64\text{K} = 2^{16}$ 个字
- 按字编址，需要16位MAR

**如果按字节编址呢？**
- 每字16位 = 2字节
- 字节数 = $2^{16} \times 2 = 2^{17}$
- 按字节编址需要17位MAR

### 考点三：给芯片求需要几片

**例题5**：用 $8\text{K} \times 8$ 位的存储芯片，组成 $32\text{K} \times 16$ 位的存储器。

- 字方向：$32\text{K} / 8\text{K} = 4$ 组
- 位方向：$16 / 8 = 2$ 片/组
- 总片数 = $4 \times 2 = 8$ 片

**例题6**：用 $1\text{K} \times 4$ 位芯片组成 $4\text{K} \times 8$ 位存储器。

- 字方向：$4\text{K} / 1\text{K} = 4$ 组
- 位方向：$8 / 4 = 2$ 片/组
- 总片数 = $4 \times 2 = 8$ 片

**解题模板**：

$$\text{字方向扩展组数} = \frac{\text{目标存储单元数}}{\text{单片存储单元数}}$$

$$\text{位方向每组片数} = \frac{\text{目标字长}}{\text{单片字长}}$$

$$\text{总片数} = \text{字方向组数} \times \text{位方向每组片数}$$

### 考点四：存储器带宽计算

**例题7**：某存储器存取周期为250ns，数据总线宽度为32位，求带宽。

$$B_m = \frac{32\,\text{bit}}{250\,\text{ns}} = \frac{32}{250 \times 10^{-9}} = 1.28 \times 10^8\,\text{bit/s} = 16\,\text{MB/s}$$

**例题8**：存储芯片存取时间为100ns，恢复时间为50ns，数据宽度为64位。求带宽。

- 存取周期 $T_c = 100 + 50 = 150\,\text{ns}$
- 带宽 $B_m = \frac{64}{150 \times 10^{-9}} = 426.7\,\text{MB/s}$

### 考点五：存取时间与存取周期的辨析

**例题9**（选择题）：下列说法正确的是？
- A. 存取时间就是存取周期
- B. 存取时间大于存取周期
- C. 存取周期包含存取时间和恢复时间
- D. 存取周期等于恢复时间

**答案：C**。存取周期 = 存取时间 + 恢复时间，所以存取周期 > 存取时间。

## 直观理解

### 常用幂次速查表

| 幂次 | 值 | 应用 |
|------|-----|------|
| $2^{10}$ | 1K = 1024 | KB |
| $2^{20}$ | 1M = 1048576 | MB |
| $2^{30}$ | 1G = 1073741824 | GB |
| $2^{32}$ | 4G | 32位地址空间 |
| $2^{16}$ | 64K | 16位地址空间 |

### 位数与容量的快速对应

- 10位 → 1K
- 12位 → 4K
- 16位 → 64K
- 20位 → 1M
- 24位 → 16M
- 30位 → 1G
- 32位 → 4G

### 带宽计算单位换算

$$1\,\text{GB/s} = 1024\,\text{MB/s} = 1024 \times 1024\,\text{KB/s}$$

或使用10进制（存储器带宽通常用10进制）：

$$1\,\text{GB/s} = 10^9\,\text{B/s}$$

## 知识关联

### 常考陷阱汇总

1. **按字节编址 vs 按字编址**：同一容量，不同编址方式，MAR位数不同
2. **K = 1024还是1000**：计算机领域 K = $2^{10} = 1024$（除硬盘厂商用1000）
3. **容量单位B vs b**：B = Byte（字节），b = bit（位），差8倍
4. **芯片标注格式**："$m \times n$" 表示 m个存储单元，每单元n位
5. **存取时间与存取周期**：不能混用，存取周期 > 存取时间
6. **MAR位数的最小值**：容量为 $2^n$ 时MAR需要n位；容量不是2的幂次时，MAR取向上取整

### 典型选择题

**题**：某计算机字长32位，主存容量为16MB，按字节编址，MAR至少需要多少位？
- A. 20  B. 24  C. 28  D. 32

$$16\,\text{MB} = 16 \times 2^{20} = 2^4 \times 2^{20} = 2^{24}\,\text{B}$$

按字节编址，需要 $2^{24}$ 个地址，MAR至少24位。

**答案：B**

**注意**：字长32位是干扰项！按字节编址时MAR只和字节数有关，与字长无关。

### 典型计算题

**题**：某存储系统，主存存取时间为200ns，Cache存取时间为10ns，命中率95%，求平均访问时间。

$$t_{\text{avg}} = H \times t_c + (1-H) \times (t_c + t_m)$$
$$= 0.95 \times 10 + 0.05 \times (10 + 200) = 9.5 + 10.5 = 20\,\text{ns}$$

注意：公式中的 $t_m$ 是否包含访问Cache的时间，取决于题目给出的主存存取时间是"单独"还是"从CPU出发"的总时间。考试中务必看清题目定义！
