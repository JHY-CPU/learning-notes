# 25_高位交叉vs低位交叉

## 核心概念

高位交叉和低位交叉是多模块存储器的两种编址方式，目的和效果完全不同。

### 对比总表

| 对比项 | 高位交叉 | 低位交叉 |
|--------|---------|---------|
| **模块号** | 高位地址决定 | 低位地址决定 |
| **连续地址分布** | 在**同一模块**内 | 在**不同模块**间 |
| **并行访问** | **不支持** | **支持**（流水访问） |
| **带宽** | 等于单模块 | 理论上 $m$ 倍 |
| **主要目的** | **容量扩展** | **带宽提升** |
| **模块独立性** | 各模块完全独立 | 各模块协同工作 |
| **扩展难度** | 容易（类似字扩展） | 较复杂 |
| **适用场景** | 大容量存储系统 | 高带宽需求系统 |

## 原理分析

### 高位交叉详解

**编址方式**：将地址的高位作为模块号，低位作为模块内地址。

4个模块，每模块容量为1K的地址分配：

| 模块 | 地址范围 | 模块号（高位） |
|------|---------|--------------|
| 模块0 | 0000~1023 | 00 |
| 模块1 | 1024~2047 | 01 |
| 模块2 | 2048~3071 | 10 |
| 模块3 | 3072~4095 | 11 |

**特点**：
- 每个模块有独立的连续地址空间
- 模块之间互不干扰
- 同一时刻只有一个模块被访问（因为连续地址在同一模块）
- 类似于字扩展

**时间线**：
```
模块0：[访问地址0][访问地址1][访问地址2]...[访问地址1023]
模块1：                                         [空闲]
模块2：                                         [空闲]
模块3：                                         [空闲]
```

### 低位交叉详解

**编址方式**：将地址的低位作为模块号，高位作为模块内地址。

4个模块，地址0~15的分配：

| 模块 | 负责的地址 | 模块内地址 |
|------|-----------|-----------|
| 模块0 | 0, 4, 8, 12 | 0, 1, 2, 3 |
| 模块1 | 1, 5, 9, 13 | 0, 1, 2, 3 |
| 模块2 | 2, 6, 10, 14 | 0, 1, 2, 3 |
| 模块3 | 3, 7, 11, 15 | 0, 1, 2, 3 |

**特点**：
- 连续地址分布在不同模块
- 可以流水（重叠）访问
- 带宽是单模块的 $m$ 倍（理想情况下）
- 需要 $T = m \times t$ 的条件

**时间线**（4模块，$T=4t$）：
```
模块0：[存取0][    ][    ][传0][存取4]...
模块1：  [    ][存取1][    ][传1][存取5]...
模块2：  [    ][    ][存取2][传2][存取6]...
模块3：  [    ][    ][    ][传3][存取7]...
总线：                      [传0][传1][传2][传3]...
```

每个 $t$ 时刻总线都在传输数据！

### 为什么高位交叉不能并行？

因为连续地址在同一模块内，当CPU顺序执行指令时（空间局部性），访问的都是同一模块的地址，其他模块闲置。

**例**：CPU顺序访问地址0、1、2、3，都在模块0中，模块1、2、3完全空闲。

### 为什么低位交叉能并行？

因为连续地址在不同模块上，CPU顺序访问时，各模块轮流工作，形成流水。

**例**：CPU顺序访问地址0、1、2、3，分别在模块0、1、2、3中。模块0完成存取后，模块1正好也完成，依次通过总线输出。

## 直观理解

**发牌类比**：
- **低位交叉** = 扑克牌轮流发给4个人（1张给甲，1张给乙...），每人拿到的牌不连续
- **高位交叉** = 把牌分成4叠，每叠给一个人（前13张给甲，接下来13张给乙...），每人拿到的牌连续

**流水线类比**：
- **低位交叉** = 工厂流水线，4个工人同时作业，效率最高
- **高位交叉** = 4个工人各做各的，无法流水

### 易错陷阱

1. **高位交叉 = 字扩展**：目的相同（增大容量），但说法不同
2. **低位交叉不能简单理解为"轮流访问"**：是流水重叠，不是轮流
3. **高位交叉的模块也可以同时被不同CPU访问**（如果有多条总线），但不提高单CPU的带宽
4. **低位交叉需要 $T \geq m \times t$ 才能理想运行**：否则流水线断流
