# 77_磁盘调度算法_CSCAN

## 核心概念
- **C-SCAN (Circular SCAN)**: 循环扫描算法
- **原理**: 磁头只沿一个方向服务, 到达末端后直接跳回起点(不服务), 继续单向扫描
- **改进**: 解决SCAN两端等待时间不均匀的问题
- **变体**: C-LOOK —— 到最远请求即跳回(不到末端)

## 原理分析

### C-SCAN vs SCAN 关键区别

| 对比项 | SCAN | C-SCAN |
|--------|------|--------|
| 反向方式 | 掉头, 反向服务 | 跳回起点, 不服务 |
| 方向 | 双向都服务 | 只单向服务 |
| 公平性 | 两端不均匀 | 各位置更均匀 |
| 磁道等效 | 线性(0~199) | 环形(0→199→0) |

### C-SCAN算法

```
1. 磁头沿一个方向移动, 服务沿途请求
2. 到达末端后, 直接跳到另一端(不服务)
3. 继续沿同一方向扫描
4. 重复直到所有请求完成
```

### 示例

**题目**: 请求队列: 98, 183, 37, 122, 14, 124, 65, 67。磁头位置: 53, 方向: 向外。磁道范围: 0~199。

**解**:

向外扫描: 53 → 65 → 67 → 98 → 122 → 124 → 183 → 199(末端)
跳回: 199 → 0 (不服务)
继续向外: 0 → 14 → 37

访问顺序: 53 → 65 → 67 → 98 → 122 → 124 → 183 → [跳到0] → 14 → 37

```
磁道: 0    14 37     53  65 67   98     122 124    183  199
      |     |  |      |   |  |    |       |   |      |    |
      ●     ●  ●      ●→  ●  ●    ●       ●   ●      ● → ●
      |←────────────── |── |──|────|───────|───|────── |   |
      |     (跳回,不服务)   |  |    |       |   |      |    |
```

**总移动距离**:
- 向外: $|65-53| + |67-65| + |98-67| + |122-98| + |124-122| + |183-124| + |199-183|$
  $= 12 + 2 + 31 + 24 + 2 + 59 + 16 = 146$
- 跳回: $199 - 0 = 199$ (跳回不计入寻道距离! 瞬间完成)
- 继续向外: $|14-0| + |37-14| = 14 + 23 = 37$

> **注意**: 跳回是否计入总距离, 不同教材有不同约定。
> - 有些认为跳回是瞬间的, 不计入距离
> - 有些认为需要计入从末端到另一端的距离
> - **408考试一般不计入跳回距离**(或题目会明确说明)

总距离(不含跳回): $146 + 37 = 183$ 磁道
总距离(含跳回): $146 + 199 + 37 = 382$ 磁道

### C-LOOK变体

**C-LOOK**: 不扫描到物理末端, 到最远请求即跳回

```
53 → 65 → 67 → 98 → 122 → 124 → 183 → [跳到14] → 14 → 37
```

总距离(不含跳回): $12+2+31+24+2+59 + 14+23 = 167$ 磁道

### 公平性分析

**SCAN的不公平**: 刚被反向经过的请求等待时间 = 磁头走到末端 + 再回来的时间

**C-SCAN的改进**: 等待时间差异更小
- 假设磁头从53向外扫描
- 磁道14的请求: 等磁头到199, 跳回0, 再到14
- 磁道37的请求: 等磁头到199, 跳回0, 再到37
- 两者等待时间差异很小(只差23个磁道)

**SCAN**: 14和37等待时间差异可能很大(取决于扫描方向)

## 直观理解

**环形跑道类比**:
- SCAN = 往返跑: 从A跑到B, 再从B跑回A
- C-SCAN = 环形跑: 从A跑到B, 瞬间传送回A, 再从A跑到B
- 环形跑中, 每个人被服务的时间间隔更均匀

### 记忆口诀
> **"C-SCAN单向服务, 跳回不服务; 比SCAN更公平, 但距离可能更长"**

## 知识关联

### 四种调度算法总结

| 算法 | 策略 | 饥饿 | 公平性 | 寻道距离 |
|------|------|------|--------|----------|
| FCFS | 按到达顺序 | 无 | 最公平 | 最长 |
| SSTF | 最近优先 | 有 | 差 | 最短 |
| SCAN | 电梯(双向) | 无 | 中等 | 中等 |
| C-SCAN | 循环(单向) | 无 | 较好 | 中等偏长 |
