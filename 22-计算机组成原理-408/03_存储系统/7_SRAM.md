# 08_SRAM

## 核心概念

### SRAM的定义

**SRAM（Static Random Access Memory，静态随机存取存储器）**：利用**双稳态触发器（Bistable Flip-Flop）**来存储一位信息的半导体存储器。

### SRAM的基本存储单元

每个存储1位需要 **6个MOS管**（6T存储单元）：
- 2个交叉耦合的反相器构成双稳态触发器
- 2个访问管（字线选通时导通）
- 也有4T方案（省面积但需要预充电电路）

### SRAM的核心特性

| 特性 | 说明 |
|------|------|
| **存储原理** | 双稳态触发器（两个稳定状态表示0和1） |
| **是否需要刷新** | **不需要**——只要有电就有稳定状态 |
| **存取速度** | **快**（1~10ns级） |
| **集成度** | **低**（每bit需要6个晶体管） |
| **功耗** | 静态功耗低，但单位面积功耗高 |
| **价格** | **贵**（每位价格高） |
| **主要用途** | **Cache**、CPU寄存器 |

## 原理分析

### 双稳态触发器工作原理

```
        Vcc (+)
        ─┬─
         │
    ┌────┴────┐
    │         │
   T1管      T3管
    │         │
    ├── Q ────┤
    │         │
   T2管      T4管
    │         │
    └────┬────┘
         │
        GND (-)
```

- T1-T2构成一个反相器，T3-T4构成另一个反相器
- 两个反相器交叉耦合，形成正反馈环路
- **状态1（存1）**：Q = 1, Q̄ = 0（T2导通，T4截止）
- **状态2（存0）**：Q = 0, Q̄ = 1（T2截止，T4导通）
- 只要供电，状态**自动保持稳定**——这就是"静态"的含义

### 读操作

1. 字线（Word Line）被激活（选中该单元）
2. 两个访问管（T5、T6）导通
3. Q和Q̄的电平通过位线（Bit Line）和位线反（Bit Line Bar）输出
4. 读出放大器检测两根位线上的电压差
5. **非破坏性读取**：读操作不改变存储状态

### 写操作

1. 字线激活
2. 在位线和位线反上施加要写入的数据（互补电平）
3. 强制覆盖触发器的当前状态
4. 字线关闭后，新状态自动保持

### 为什么SRAM不需要刷新？

双稳态触发器是一个**正反馈环路**：
- 如果一个节点电位稍高，经过反相器反馈后会更高
- 最终自动稳定在某个确定状态
- 只要供电，就会一直保持——不像DRAM的电容会漏电

### SRAM芯片的典型参数

| 型号 | 容量 | 存取时间 | 用途 |
|------|------|---------|------|
| 高速SRAM | 256KB~1MB | 5~15ns | L1 Cache |
| 中速SRAM | 1~8MB | 10~30ns | L2 Cache |
| 嵌入式SRAM | 几十KB | 1~5ns | CPU寄存器文件 |

## 直观理解

**跷跷板类比**：
- 双稳态触发器就像一个跷跷板，要么左边高（状态0），要么右边高（状态1）
- 一旦稳定在某个状态，只要不施加外力（断电），就会永远保持
- 读操作 = 看看哪边高（不碰跷跷板）
- 写操作 = 强行把跷跷板推到另一边

**为什么SRAM贵？**
- 每bit需要6个晶体管，而DRAM只需1个晶体管+1个电容
- 晶体管多 = 芯片面积大 = 每片芯片容量小 = 成本高
- 所以SRAM用于Cache（容量小但要求速度快），DRAM用于主存（容量大且要求便宜）

### SRAM vs 你家的记事本

- SRAM就像用**粉笔写在黑板**上的内容——只要不擦，一直都在
- DRAM就像用**铅笔写在纸**上的字——墨迹会慢慢变淡，需要定期描一遍（刷新）

## 知识关联

### 跨章节联系
- **第5章（CPU）**：CPU中的通用寄存器、PC、IR等都是SRAM实现的
- **第5章**：L1 Cache集成在CPU芯片内，用的是片上SRAM
- **第3章（存储系统）**：Cache用SRAM是因为速度快，主存用DRAM是因为容量大且便宜

### SRAM在Cache中的应用

现代CPU的典型配置：
- **L1 Cache**：32~64 KB，片上SRAM，访问时间 ~1ns
- **L2 Cache**：256KB~1MB，片上SRAM，访问时间 ~5ns
- **L3 Cache**：4~64 MB，片上SRAM，访问时间 ~15ns

### 易错陷阱

1. **SRAM不是不需要供电**——只是不需要刷新！断电一样丢数据（易失性存储器）
2. **"静态"不是指速度慢**——恰恰相反，SRAM比DRAM快。"静态"指的是不需要周期性刷新
3. **SRAM成本高不是因为工艺复杂**——是因为每个bit需要的晶体管数量多
4. **CPU内部的寄存器也是SRAM**，但讨论存储层次时通常只讨论Cache-主存-辅存
