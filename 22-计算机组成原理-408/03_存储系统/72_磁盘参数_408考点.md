# 73_磁盘参数_408考点

## 核心考点
- 磁盘容量计算
- 平均存取时间 = 寻道 + 旋转延迟 + 传输时间
- 数据传输率计算
- 连续读写多个扇区的优化

## 原理分析

### 公式汇总

**容量**:
$$C = \text{盘面数} \times \text{磁道数/面} \times \text{扇区数/道} \times \text{字节数/扇区}$$

**平均存取时间**:
$$T = T_{\text{寻道}} + T_{\text{旋转延迟}} + T_{\text{传输}}$$

其中:
- $T_{\text{寻道}}$ = 平均寻道时间(题目给定或取最大值的一半)
- $T_{\text{旋转延迟}}$ = $\dfrac{1}{2} \times \dfrac{60}{\text{转速(RPM)}}$ (单位: 秒)
- $T_{\text{传输}}$ = $\dfrac{\text{传输字节数}}{N_{\text{扇}} \times B_{\text{扇}} \times \frac{n}{60}}$

**数据传输率**:
$$R = N_{\text{扇}} \times B_{\text{扇}} \times \frac{n}{60}$$

## 例题详解

### 例题1: 基本容量和存取时间

**题目**: 某磁盘有4个盘片(8个记录面), 每面有800个磁道, 每磁道有100个扇区, 每扇区512字节。磁盘转速7200RPM, 平均寻道时间8ms。

(1) 求磁盘总容量
(2) 求数据传输率
(3) 求读取一个扇区的平均存取时间

**解**:

**(1) 容量**:
$$C = 8 \times 800 \times 100 \times 512 = 327,680,000 \text{ B} \approx 312.5 \text{ MB}$$

**(2) 数据传输率**:
每磁道数据量 = $100 \times 512 = 51,200$ B = 50 KB
$$R = 51,200 \times \frac{7200}{60} = 51,200 \times 120 = 6,144,000 \text{ B/s} \approx 6 \text{ MB/s}$$

**(3) 平均存取时间**:
- 寻道时间 = 8 ms
- 平均旋转延迟 = $\frac{1}{2} \times \frac{60}{7200} = 4.17$ ms
- 传输1个扇区时间 = $\frac{512}{6,144,000} = 0.083$ ms
- 总时间 = $8 + 4.17 + 0.083 = 12.25$ ms

---

### 例题2: 读取多个扇区

**题目**: 同例题1的磁盘参数。现要读取连续存放的25个扇区(在同一磁道上), 求总读取时间。

**解**:

- 寻道时间 = 8 ms (只需一次)
- 旋转延迟 = 4.17 ms (只需一次, 等到第一个扇区)
- 传输25个扇区 = $\frac{25 \times 512}{6,144,000} = 2.08$ ms

总时间 = $8 + 4.17 + 2.08 = 14.25$ ms

**对比**: 如果25个扇区分散在不同磁道:
- 最坏情况: 需要25次寻道 + 25次旋转延迟 + 25次传输
- 总时间 = $25 \times (8 + 4.17 + 0.083) = 25 \times 12.25 = 306.25$ ms

**结论**: 连续存放比分散存放快**21倍**!

### 例题3: 不同转速对比

**题目**: 两种磁盘:
- A: 5400 RPM, 平均寻道10ms, 每磁道600扇区, 每扇区512B
- B: 10000 RPM, 平均寻道6ms, 每磁道800扇区, 每扇区512B

求各自读取一个扇区的平均存取时间。

**解**:

**磁盘A**:
- 旋转延迟 = $\frac{1}{2} \times \frac{60}{5400} = 5.56$ ms
- 传输率 = $600 \times 512 \times \frac{5400}{60} = 27,648,000$ B/s $\approx 26.4$ MB/s
- 传输时间 = $\frac{512}{27,648,000} = 0.0185$ ms
- 存取时间 = $10 + 5.56 + 0.0185 = 15.58$ ms

**磁盘B**:
- 旋转延迟 = $\frac{1}{2} \times \frac{60}{10000} = 3.0$ ms
- 传输率 = $800 \times 512 \times \frac{10000}{60} = 68,266,667$ B/s $\approx 65.1$ MB/s
- 传输时间 = $\frac{512}{68,266,667} = 0.0075$ ms
- 存取时间 = $6 + 3.0 + 0.0075 = 9.01$ ms

### 例题4: 最大旋转延迟

**题目**: 某磁盘转速15000RPM, 磁头刚到达目标磁道时, 目标扇区刚好离开磁头。求最坏情况下的旋转延迟。

**解**:
- 最坏情况: 需要转接近一整圈
- 旋转延迟 ≈ $\frac{60}{15000} = 4$ ms (一圈时间)

---

### 例题5: 容量与寻址

**题目**: 某磁盘有2个盘片, 每面1024个磁道, 每磁道256个扇区, 每扇区4096字节。用LBA寻址, LBA地址需要多少位?

**解**:
- 总扇区数 = 4面 × 1024道/面 × 256扇/道 = $2^2 \times 2^{10} \times 2^8 = 2^{20}$ = 1,048,576
- LBA位数 = $\log_2 2^{20}$ = **20位**

## 易错陷阱

1. **盘面数**: 4个盘片 = 8个盘面(双面), 但如果题目说"上下两面不使用", 则只有6面
2. **旋转延迟**: 一定是**半圈**, 不是一圈
3. **单位换算**: 1MB = $10^6$ B 还是 $2^{20}$ B? 磁盘厂商用10进制, 操作系统用2进制
4. **传输时间很小**: 实际上传输一个扇区的时间通常远小于寻道和旋转延迟, 有时可忽略
5. **连续扇区**: 同一磁道上的连续扇区不需要再寻道和等旋转
