# 12_SRAM_vs_DRAM

## 核心概念

SRAM和DRAM是408考试的高频对比考点。两者在结构、性能、用途上完全不同。

### 全面对比表

| 对比项目 | SRAM | DRAM |
|---------|------|------|
| **存储原理** | 双稳态触发器 | 电容充放电 |
| **每bit晶体管数** | 6个（或4个） | 1个 + 1个电容 |
| **是否需要刷新** | **不需要** | **必须刷新** |
| **读出方式** | 非破坏性读取 | **破坏性读出**（需回写） |
| **存取速度** | **快**（1~10ns） | **慢**（50~100ns） |
| **集成度** | **低** | **高** |
| **单位价格** | **贵** | **便宜** |
| **功耗** | 较高（静态功耗） | 较低（动态功耗） |
| **地址线** | 不复用 | **复用**（行列分时） |
| **主要用途** | **Cache**、寄存器 | **主存**（内存条） |
| **易失性** | 易失 | 易失 |

## 原理分析

### 为什么Cache用SRAM？

1. **速度要求**：Cache要尽可能接近CPU速度，SRAM的1~10ns访问时间适合
2. **不需要刷新**：Cache由硬件管理，刷新会增加复杂性
3. **非破坏性读取**：读操作不需要回写，节省时间
4. **容量小**：Cache容量只需几MB，SRAM的高成本可以接受

### 为什么主存用DRAM？

1. **容量需求大**：主存需要几GB到几十GB，DRAM的高集成度适合
2. **成本考虑**：同样容量，DRAM比SRAM便宜几十倍
3. **速度要求可放宽**：有Cache缓冲，主存速度不必极致
4. **功耗考虑**：DRAM单位容量功耗更低

### 成本对比的量级

假设同样1GB容量：
- DRAM方案：成本约为几十元
- SRAM方案：成本可能高达数千元

这就是为什么不用SRAM做主存。

### 速度对比的量级

- SRAM存取时间：1~10ns（与CPU时钟周期同级）
- DRAM存取时间：50~100ns（比CPU慢10~100倍）
- 如果主存用SRAM，不需要Cache；但成本不现实

## 直观理解

**SRAM = 豪华跑车**：
- 快、性能好、但贵、体积大（集成度低）
- 用在关键位置（Cache = 你家门口的停车位）

**DRAM = 经济家用车**：
- 速度一般、但便宜、空间大（集成度高）
- 用在需要大容量的地方（主存 = 大型停车场）

**记忆口诀**：
```
SRAM：触发器存数据，不需要刷新，速度快，价格贵，做Cache
DRAM：电容存电荷，必须常刷新，速度慢，价格低，做主存
```

### 一个形象的对比

| 特征 | SRAM | DRAM |
|------|------|------|
| 比喻 | 笔记本（随时记录，快速翻阅） | 黑板（容量大，但要定期维护） |
| 存储 | 铅笔写纸上（字迹会变淡要描） | 石碑刻字（永久但不便修改） |
| 类比 | 大脑短期记忆（快但容量小） | 笔记本（容量大但翻找慢） |

### 易错陷阱

1. **SRAM和DRAM都是易失性存储器**——断电都丢数据
2. **"SRAM快"不意味着SRAM做主存好**——成本太高
3. **DRAM需要刷新不是"缺点"**——而是特性，刷新机制已经很成熟
4. **SRAM地址线不复用，DRAM地址线复用**——考试常作为区分点
5. **不要将SRAM/DRAM与ROM混淆**——SRAM和DRAM都是RAM（可读可写），ROM是只读的
6. **现代计算机中**：CPU内部寄存器（SRAM）→ Cache（SRAM）→ 主存（DRAM）→ 辅存（Flash/磁盘）
