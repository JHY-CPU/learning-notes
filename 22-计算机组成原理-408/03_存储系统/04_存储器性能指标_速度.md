# 04_存储器性能指标_速度

## 核心概念

### 1. 存取时间（Access Time / 访问时间）$T_a$

- **定义**：从CPU发出读/写命令到完成一次读/写操作所需的时间
- **读操作**：从发出地址到数据稳定出现在数据线上
- **写操作**：从发出地址到数据被写入存储单元
- **单位**：纳秒（ns）
- **特点**：只描述单次操作的"延迟"，不包含恢复时间

### 2. 存取周期（Memory Cycle Time）$T_c$

- **定义**：连续两次访问存储器所需的**最小间隔时间**
- **关系**：$T_c > T_a$
- **原因**：存取周期 = 存取时间 + **恢复时间**
  - 读操作后，存储单元需要恢复（如DRAM的预充电）
  - 写操作后，需要等待数据稳定
- **类比**：拍照后需要等闪光灯充电才能拍下一张

$$T_c = T_a + T_{\text{恢复}}$$

### 3. 存储器带宽（Memory Bandwidth）$B_m$

- **定义**：单位时间内存储器能传送的数据量
- **公式**：

$$B_m = \frac{\text{数据宽度（每次传送的位数）}}{T_c}$$

或等价地：

$$B_m = \frac{\text{数据宽度}}{\text{存取周期}}$$

- **单位**：bit/s、B/s、MB/s等
- **意义**：带宽越大，数据吞吐能力越强

### 三个指标的关系总结

| 指标 | 含义 | 说明 |
|------|------|------|
| 存取时间 $T_a$ | 单次读写操作耗时 | 越短越好 |
| 存取周期 $T_c$ | 连续两次操作的最小间隔 | $T_c > T_a$ |
| 存储器带宽 $B_m$ | 单位时间数据传输量 | $B_m = \text{数据宽度} / T_c$ |

## 原理分析

### 存取时间 vs 存取周期的区别（常考！）

假设某个DRAM芯片：
- 存取时间 $T_a = 100\,\text{ns}$
- 恢复时间 $T_{\text{恢复}} = 50\,\text{ns}$
- 存取周期 $T_c = 150\,\text{ns}$

含义：
- 发出读命令后，**100ns后**数据出现在数据线上（存取时间）
- 但要等**再过50ns**（总共150ns），才能发起下一次读/写操作（存取周期）

### 带宽计算实例

**例1**：某存储器数据总线宽度为32位，存取周期为200ns，求存储器带宽。

$$B_m = \frac{32\,\text{bit}}{200\,\text{ns}} = \frac{32}{200 \times 10^{-9}} = 1.6 \times 10^8\,\text{bit/s} = 160\,\text{Mbit/s} = 20\,\text{MB/s}$$

**例2**：存储器总线宽度为64位，存取周期为50ns。

$$B_m = \frac{64\,\text{bit}}{50\,\text{ns}} = \frac{64}{50 \times 10^{-9}} = 1.28 \times 10^9\,\text{bit/s} = 1.28\,\text{Gbit/s} = 160\,\text{MB/s}$$

**关键公式**：

$$\text{每秒访问次数} = \frac{1}{T_c}$$

$$\text{带宽} = \text{每秒访问次数} \times \text{数据宽度} = \frac{\text{数据宽度}}{T_c}$$

### 提高带宽的方法

1. **缩短存取周期** $T_c$：使用更快的存储介质（SRAM > DRAM）
2. **增大存储字长**（数据宽度）：位扩展，一次读写更多数据
3. **多模块并行**：多模块交叉存储器，多个模块同时工作（详见第23讲）

### 从CPU角度看存储器速度

假设CPU主频为1GHz（时钟周期1ns），存储器存取时间为50ns：
- 一次存储器访问需要50个CPU时钟周期
- 如果没有Cache，CPU在这50个周期内只能等待
- 这就是为什么需要Cache——减少CPU等待时间

## 直观理解

**ATM取钱类比**：
- **存取时间**：你插卡、输入密码、ATM吐出钞票的总时间（比如30秒）
- **存取周期**：你取完钱后，ATM准备好接待下一位顾客的完整时间（比如45秒，含屏幕重置、清点等）
- **带宽**：ATM每分钟能处理多少人的取款业务（多台ATM = 多模块交叉）

**带宽类比**：
- 数据宽度 = 公路的车道数（4车道 vs 8车道）
- 存取周期 = 每辆车通过的时间
- 带宽 = 单位时间通过的车辆总数

### 易错陷阱
1. **存取时间 $\neq$ 存取周期**！存取周期 > 存取时间（含恢复时间）
2. **带宽单位**：题目可能用 bit/s 或 B/s，注意1 Byte = 8 bit
3. **"存取周期为200ns"不等于"每秒访问500万次"**——要确认单位换算正确：$1 / (200 \times 10^{-9}) = 5 \times 10^6 = 500\,\text{万次/秒}$
4. **数据宽度**可能是存储字长也可能是总线宽度，题目中要明确
5. **多模块交叉可以提高带宽但不改变单模块的存取时间**
