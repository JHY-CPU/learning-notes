# 20_容量扩展_计算详解

## 核心概念

本文件通过5道例题，详细展示存储器容量扩展的完整解题过程。

## 原理分析

### 解题通用公式

$$\text{总芯片数} = \frac{\text{目标容量}}{\text{单片容量}} = \frac{\text{目标单元数}}{\text{单片单元数}} \times \frac{\text{目标字长}}{\text{单片字长}}$$

$$\text{芯片内地址线} = \log_2(\text{单片存储单元数})$$

$$\text{片选地址线} = \log_2(\text{字方向组数})$$

### 例题1（纯位扩展）

**题目**：用 $8\text{K} \times 4$ 位的芯片组成 $8\text{K} \times 16$ 位的存储器。

**解答**：
- 字方向：$8\text{K} / 8\text{K} = 1$ 组（不需要字扩展）
- 位方向：$16 / 4 = 4$ 片/组
- 总片数 = $1 \times 4 = 4$ 片

**连接**：
- 地址线：A0~A12（13位）并联到4片芯片
- 数据线：每片接4位
  - 芯片1：D0~D3
  - 芯片2：D4~D7
  - 芯片3：D8~D11
  - 芯片4：D12~D15
- 片选：4片共用同一个CE信号
- 读写控制：WE并联

---

### 例题2（纯字扩展）

**题目**：用 $8\text{K} \times 8$ 位的芯片组成 $32\text{K} \times 8$ 位的存储器。

**解答**：
- 字方向：$32\text{K} / 8\text{K} = 4$ 组
- 位方向：$8 / 8 = 1$ 片/组
- 总片数 = $4 \times 1 = 4$ 片

**地址分配**：
- 芯片内地址线：$\log_2(8\text{K}) = 13$ 位（A0~A12）
- 片选地址线：$\log_2(4) = 2$ 位（A13、A14）
- 总地址线：15位（A0~A14）

| 芯片 | 地址范围 | A14A13 |
|------|---------|--------|
| 芯片0 | 0000H~1FFFH | 00 |
| 芯片1 | 2000H~3FFFH | 01 |
| 芯片2 | 4000H~5FFFH | 10 |
| 芯片3 | 6000H~7FFFH | 11 |

**连接**：
- A0~A12 并联到4片芯片的地址引脚
- A13、A14 接2:4译码器，产生4个片选信号
- 数据线 D0~D7 并联到4片芯片（同一时刻只有一片的数据有效）
- WE并联

---

### 例题3（字位同时扩展）

**题目**：用 $4\text{K} \times 4$ 位的芯片组成 $16\text{K} \times 8$ 位的存储器。

**解答**：
- 字方向：$16\text{K} / 4\text{K} = 4$ 组
- 位方向：$8 / 4 = 2$ 片/组
- 总片数 = $4 \times 2 = 8$ 片

**地址分配**：
- 芯片内地址线：$\log_2(4\text{K}) = 12$ 位（A0~A11）
- 片选地址线：$\log_2(4) = 2$ 位（A12、A13）
- 总地址线：14位（A0~A13）

**连接方式**：
```
A0~A11 ──→ 所有8片芯片（共用）
A12,A13 ──→ 2:4译码器 ──→ CE0,CE1,CE2,CE3

第0组（CE0有效时）：
  芯片0a：D0~D3
  芯片0b：D4~D7

第1组（CE1有效时）：
  芯片1a：D0~D3
  芯片1b：D4~D7

第2组、第3组类似...
```

### 例题4（给定地址范围求扩展方案）

**题目**：某系统需要地址空间 0000H~3FFFH 的存储器，数据总线8位，用 $1\text{K} \times 4$ 位芯片实现。

**解答**：

**Step 1：确定目标容量**
- 地址范围 = $3\text{FFFH} + 1 = 4000\text{H} = 16384 = 16\text{K}$ 个字节
- 目标：$16\text{K} \times 8$ 位

**Step 2：计算芯片数**
- 字方向：$16\text{K} / 1\text{K} = 16$ 组
- 位方向：$8 / 4 = 2$ 片/组
- 总片数 = $16 \times 2 = 32$ 片

**Step 3：地址线分配**
- 芯片内地址线：$\log_2(1\text{K}) = 10$ 位（A0~A9）
- 片选地址线：$\log_2(16) = 4$ 位（A10~A13）
- 总地址线：14位（A0~A13）

### 例题5（综合题：含画连接图）

**题目**：某计算机地址总线16位（A0~A15），数据总线8位。用 $4\text{K} \times 8$ 位ROM芯片（地址范围0000H~0FFFH）和 $4\text{K} \times 8$ 位RAM芯片（地址范围1000H~5FFFH）构成主存。求各需要多少片，画出连接图。

**解答**：

**Step 1：分析地址空间**
- ROM：0000H~0FFFH = 4K × 8位 → 需要 1 片 $4\text{K} \times 8$ ROM
- RAM：1000H~5FFFH = 20K × 8位 → 需要 $20\text{K} / 4\text{K} = 5$ 片 $4\text{K} \times 8$ RAM

**Step 2：地址线分配**
- 芯片内地址线：A0~A11（12位，对应4K）
- 片选地址线：A12~A15

**Step 3：片选逻辑**
- ROM：A15~A12 = 0000 时选中
- RAM0：A15~A12 = 0001 时选中
- RAM1：A15~A12 = 0010 时选中
- RAM2：A15~A12 = 0011 时选中
- RAM3：A15~A12 = 0100 时选中
- RAM4：A15~A12 = 0101 时选中

## 直观理解

### 解题步骤速记

```
第一步：算芯片数 = (目标单元/单片单元) × (目标字长/单片字长)
第二步：算地址线 = log₂(单片单元数) = 芯片内地址线
第三步：算片选线 = log₂(字方向组数) = 高位地址线
第四步：画连接图 = 低位共用 + 高位译码 + 数据分配
```

## 知识关联

### 易错陷阱

1. **地址范围转容量**：末地址 - 首地址 + 1 = 单元数
2. **地址线数量**：$\log_2$ 向上取整（如3K需要12位，不是11.55位）
3. **十六进制计算**：$4000\text{H} = 4 \times 16^3 = 16384 = 16\text{K}$
4. **ROM和RAM可能混用**：ROM只读、RAM可读可写，片选逻辑需分开
5. **数据线是双向的**：读时数据从芯片到CPU，写时从CPU到芯片
