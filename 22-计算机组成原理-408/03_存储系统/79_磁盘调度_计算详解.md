# 79_磁盘调度_计算详解

## 例题1: 四种算法完整对比

**题目**: 磁盘请求队列: 98, 183, 37, 122, 14, 124, 65, 67。磁头在53号磁道, 方向: 向外。磁道范围: 0~199。用四种算法调度, 计算总移动距离。

### FCFS

顺序: 53 → 98 → 183 → 37 → 122 → 14 → 124 → 65 → 67

$$d = |98-53| + |183-98| + |37-183| + |122-37| + |14-122| + |124-14| + |65-124| + |67-65|$$
$$= 45 + 85 + 146 + 85 + 108 + 110 + 59 + 2 = \mathbf{640}$$

### SSTF

每次选最近的:
- 53 → 65 (12) → 67 (2) → 37 (30) → 14 (23) → 98 (84) → 122 (24) → 124 (2) → 183 (59)

$$d = 12 + 2 + 30 + 23 + 84 + 24 + 2 + 59 = \mathbf{236}$$

### SCAN (到最远请求即反向)

向外: 53 → 65 → 67 → 98 → 122 → 124 → 183
向内: 183 → 37 → 14

$$d_{\text{向外}} = 12 + 2 + 31 + 24 + 2 + 59 = 130$$
$$d_{\text{向内}} = |183-37| + |37-14| = 146 + 23 = 169$$
$$d = 130 + 169 = \mathbf{299}$$

### SCAN (到物理末端199再反向)

向外: 53 → 65 → 67 → 98 → 122 → 124 → 183 → 199
向内: 199 → 37 → 14

$$d_{\text{向外}} = 12 + 2 + 31 + 24 + 2 + 59 + 16 = 146$$
$$d_{\text{向内}} = |199-37| + |37-14| = 162 + 23 = 185$$
$$d = 146 + 185 = \mathbf{331}$$

### C-SCAN (到最远请求即跳回)

向外: 53 → 65 → 67 → 98 → 122 → 124 → 183
跳回: 183 → 14 (跳回不计入)
继续向外: 14 → 37

$$d = 12+2+31+24+2+59 + |14-14| + |37-14| = 130 + 0 + 23 = \mathbf{153}$$

等等, 跳到14后直接服务14, 再到37:

向外: 53 → 65(12) → 67(2) → 98(31) → 122(24) → 124(2) → 183(59) = 130
跳到14(不计) → 37(23) = 23
总距离 = 130 + 23 = **153**

### C-SCAN (到物理末端199跳回0)

向外: 53 → 65 → 67 → 98 → 122 → 124 → 183 → 199 = 146
跳回0(不计) → 14(14) → 37(23) = 37
总距离 = 146 + 37 = **183**

### 结果汇总

| 算法 | 总移动距离 | 备注 |
|------|-----------|------|
| FCFS | 640 | 最长, 但最公平 |
| SSTF | 236 | 最短, 但可能饥饿 |
| SCAN(到最远) | 299 | 无饥饿 |
| SCAN(到末端) | 331 | 无饥饿 |
| C-SCAN(到最远) | 153 | 较短, 跳回不计 |
| C-SCAN(到末端) | 183 | 较短 |

## 例题2: 特殊方向

**题目**: 磁头在53, 方向: 向内(磁道号减小方向)。请求: 98, 183, 37, 122, 14, 124, 65, 67。用SCAN(到最远请求)。

**解**:

向内扫描: 53 → 37 → 14
向外扫描: 14 → 65 → 67 → 98 → 122 → 124 → 183

$$d = |53-37| + |37-14| + |65-14| + |67-65| + |98-67| + |122-98| + |124-122| + |183-124|$$
$$= 16 + 23 + 51 + 2 + 31 + 24 + 2 + 59 = 208$$

## 例题3: 请求很少的情况

**题目**: 磁头在50, 请求: 10, 90。分别用FCFS和SSTF。

**FCFS**: 50 → 10 → 90, 距离 = 40 + 80 = 120
**SSTF**: 50 → 10 → 90 或 50 → 90 → 10
- 50→10(40)→90(80) = 120
- 50→90(40)→10(80) = 120
两者相同!

**结论**: 只有两个请求且磁头在中间时, FCFS和SSTF结果相同。

## 例题4: C-SCAN公平性分析

**题目**: 磁头在100, 请求均匀分布在0~199。比较SCAN和C-SCAN各位置的等待时间差异。

**分析**:

假设连续多次扫描:

**SCAN**:
- 磁道50(刚被反向经过): 等待时间 = 走到0 + 反向回来 = 100 + 50 = 150磁道时间
- 磁道150(刚被正向经过): 等待时间 = 走到0 + 反向到150 = 250 + 150 = 400磁道时间
- 差异巨大!

**C-SCAN**:
- 磁道50: 等待时间 = 走到199 + 跳回 + 走到50 = 99 + 0 + 50 = 149
- 磁道150: 等待时间 = 走到199 + 跳回 + 走到150 = 99 + 0 + 150 = 249
- 差异 = 100 (固定差值, 更均匀)

**结论**: C-SCAN的等待时间差异更小, 公平性更好。
