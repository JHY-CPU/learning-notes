# 09_DRAM

## 核心概念

### DRAM的定义

**DRAM（Dynamic Random Access Memory，动态随机存取存储器）**：利用**电容**存储电荷来表示一位信息的半导体存储器。

### DRAM的基本存储单元

每个存储1位只需 **1个晶体管 + 1个电容**（1T1C结构）：
- 电容充电 = 存储1（高电平）
- 电容放电 = 存储0（低电平）
- 晶体管作为开关，由字线控制通断

### DRAM的核心特性

| 特性 | 说明 |
|------|------|
| **存储原理** | 电容充放电（电荷存储） |
| **是否需要刷新** | **必须刷新**——电容会漏电 |
| **存取速度** | 较慢（50~100ns级） |
| **集成度** | **高**（每bit只需1T1C） |
| **功耗** | 动态功耗（充放电），整体较低 |
| **价格** | **便宜**（每位价格低） |
| **主要用途** | **主存储器**（内存条） |

## 原理分析

### 1T1C存储单元结构

```
        数据线(D)
          │
        ┌─┴─┐
        │ T │ ← 字线(W)控制
        └─┬─┘
          │
        ┌─┴─┐
        │ C │ ← 电容
        └─┬─┘
          │
        GND
```

- **字线（Word Line）**：控制晶体管T的导通/截止
- **数据线（Bit Line / 位线）**：读写数据的通道
- **电容C**：存储电荷，有电荷=1，无电荷=0

### 读操作（破坏性读出！）

1. 字线激活，晶体管导通
2. 电容与数据线相连
3. 读出放大器检测数据线上的**微小电压变化**
4. **关键**：读操作会使电容放电，**破坏原有信息**
5. 读出后必须**回写（恢复）**数据

> **考试重点**：DRAM是**破坏性读出**，读完必须回写；SRAM是**非破坏性读取**。

### 写操作

1. 字线激活
2. 在数据线上施加要写入的电平
3. 电容充电或放电到目标状态
4. 字线关闭，电容保持电荷

### 为什么DRAM需要刷新？

电容存在**漏电流**，存储的电荷会逐渐流失：
- 电容充满电（存1）后，即使不读写，电荷也会慢慢泄漏
- 如果不及时补充，1会变成0
- **刷新** = 定期重新读出并回写（充电），维持数据

**刷新周期**：
- 标准DRAM：**2ms**（或64ms，取决于具体规格）
- 必须在刷新周期内完成对所有行的刷新

### DRAM的地址线复用

DRAM芯片的地址线是**复用**的：
- 先送**行地址**（Row Address），用 RAS（Row Address Strobe）锁存
- 再送**列地址**（Column Address），用 CAS（Column Address Strobe）锁存
- 这样地址线数量减半，引脚数减少

例如：64Mbit的DRAM，$2^{26}$ 个地址。
- 不复用需要26根地址线
- 复用后只需13根地址线（行地址13位 + 列地址13位）

> **考试重点**：SRAM地址线不复用，DRAM地址线复用。这是重要区分点。

## 直观理解

**水桶类比**：
- 电容 = 水桶，水位高 = 1，水位低 = 0
- 桶底有小孔（漏电），水会慢慢漏掉
- **刷新** = 定期往桶里加水，维持水位
- **读操作** = 看看水位——但看的时候要把水倒出来量（破坏性读出），然后倒回去

**为什么DRAM便宜？**
- 每bit只需1个晶体管+1个电容
- 面积小 → 集成度高 → 每片容量大 → 单位成本低
- 所以适合做主存（需要大容量）

**为什么DRAM慢？**
- 电容充放电比触发器翻转慢
- 需要读出放大器检测微小电压变化（电容电压变化小）
- 破坏性读出需要额外的回写时间
- 地址复用需要分两次送地址

### 现代DRAM技术发展

| 技术 | 特点 |
|------|------|
| **SDRAM** | 同步DRAM，与CPU时钟同步 |
| **DDR SDRAM** | 双倍数据速率，每个时钟周期传两次数据 |
| **DDR2/DDR3/DDR4/DDR5** | 速率不断提升，电压不断降低 |

### 常考组合
- DRAM破坏性读出 vs SRAM非破坏性读取（选择题）
- DRAM地址复用（行/列地址分时传送）
- DRAM需要刷新，刷新周期（2ms/64ms）

### 易错陷阱

1. **"动态"不是指速度快**——"动态"指电荷会流失（需要刷新），与速度无关
2. **DRAM的"破坏性读出"**——读操作会放电，必须回写；SRAM没有这个问题
3. **地址复用是DRAM的特征**——SRAM不需要地址复用
4. **刷新操作由刷新控制器自动完成**——不需要CPU指令控制
5. **刷新期间CPU不能访问存储器**——会产生"死时间"（详见第10讲）
