# 71_磁盘结构

## 核心概念
- **磁盘(Disk)**: 利用磁性材料存储数据的旋转式存储设备
- **盘片(Platter)**: 磁盘的核心存储介质, 双面涂磁
- **磁头(Head)**: 读写数据的装置, 每个盘面一个
- **磁道(Track)**: 盘面上同心圆环形存储区域
- **扇区(Sector)**: 磁道上等分的弧段, 是磁盘最小读写单位
- **柱面(Cylinder)**: 所有盘面上相同半径的磁道组成的圆柱面

### 物理结构

```
          盘片组(多个盘片叠在同一主轴上)
          ┌─────────────────────────┐
          │  ┌───┐  ┌───┐  ┌───┐   │
          │  │盘1│  │盘2│  │盘3│   │  ← 盘片
          │  └─┬─┘  └─┬─┘  └─┬─┘   │
          │    │      │      │     │
          │   [H1]   [H2]   [H3]   │  ← 磁头(每个盘面一个)
          │    ↑      ↑      ↑     │
          │    └──────┼──────┘     │
          │         主轴           │
          └─────────────────────────┘
                    ↓
              高速旋转(如7200RPM)
```

### 盘面布局

```
          盘面俯视图
          ┌─────────────────┐
          │    ╭──────╮     │
          │   ╭┤磁道5 ├╮    │
          │  ╭┤├──────┤├╮   │
          │  │││磁道3 │││   │  ← 磁道(同心圆)
          │  ╰┤├──────┤├╯   │
          │   ╰┤磁道1 ├╯    │
          │    ╰──┬───╯     │
          │       │         │
          │     主轴(中心)   │
          └─────────────────┘

          单个磁道展开
          ┌──┬──┬──┬──┬──┬──┐
          │S1│S2│S3│S4│S5│S6│  ← 扇区
          └──┴──┴──┴──┴──┴──┘
          扇区间有间隙(gap)
```

### 关键术语

| 术语 | 说明 |
|------|------|
| 盘片(Platter) | 圆形薄片, 双面涂磁性材料 |
| 磁头(Head) | 悬浮在盘面上方, 读/写磁信号 |
| 磁道(Track) | 盘面上的一个同心圆 |
| 扇区(Sector) | 磁道上的一段弧, 最小读写单位(通常512B或4KB) |
| 柱面(Cylinder) | 同一半径的所有磁道的集合 |
| 磁道密度 | 单位长度上的磁道数 |
| 位密度 | 单位磁道长度上的数据位数 |

## 原理分析

### 磁盘控制器(Disk Controller)

磁盘控制器是CPU与磁盘驱动器之间的接口:
- **功能**: 接收命令, 控制寻道, 管理扇区读写, 数据缓存
- **接口标准**: SATA, SAS, NVMe(固态), IDE(老式)
- **缓存**: 控制器上有少量DRAM缓存(通常64MB~256MB)

### 寻址方式

**CHS寻址** (老式):
$$\text{物理地址} = (\text{柱面号}, \text{磁头号}, \text{扇区号})$$

**LBA寻址** (现代):
$$\text{逻辑块地址} = 0, 1, 2, \ldots, N-1$$
- 控制器内部将LBA转换为CHS
- 对操作系统透明

### 磁道格式

每个磁道的扇区布局:
```
┌────┬────┬────┬────┬────┬────┬────┐
│GAP │ID  │GAP │DATA│GAP │ID  │... │
│    │字段│    │字段│    │字段│    │
└────┴────┴────┴────┴────┴────┴────┘
```
- **ID字段**: 磁道号、扇区号、校验信息
- **DATA字段**: 实际存储的数据
- **GAP**: 间隔, 用于同步

### 磁盘操作流程

```
1. CPU发送I/O请求到磁盘控制器
        |
        v
2. 控制器解析LBA → CHS(柱面、磁头、扇区)
        |
        v
3. 寻道: 磁头臂移动到目标柱面
        |
        v
4. 旋转等待: 等待目标扇区转到磁头下方
        |
        v
5. 数据传输: 磁头读/写数据
        |
        v
6. 控制器通知CPU完成
```

## 直观理解

**唱片机类比**:
- 盘片 = 黑胶唱片的盘面
- 磁道 = 唱片上的纹路(同心圆)
- 扇区 = 纹路上的一小段
- 磁头 = 唱针
- 主轴 = 唱片中心轴
- 寻道 = 唱针移动到正确的纹路上
- 旋转等待 = 等唱片转到正确位置

### 多盘片结构理解

3个盘片 = 6个盘面 = 6个磁头

```
磁头0 ←→ 盘面0(顶面)
磁头1 ←→ 盘面1(盘片1底面)
磁头2 ←→ 盘面2(盘片2顶面)
磁头3 ←→ 盘面3(盘片2底面)
磁头4 ←→ 盘面4(盘片3顶面)
磁头5 ←→ 盘面5(底面)
```

同一柱面上: 6个磁道(每面1个), 磁头臂联动, 一起移动

## 知识关联

### 跨章节联系
- **计组第5章(总线)**: 磁盘通过总线与CPU通信
- **操作系统**: 磁盘调度算法, I/O管理
- **计组第6章(I/O)**: 中断方式、DMA方式访问磁盘

### 关键数字
- 典型硬盘: 7200 RPM(转/分)
- 每转时间: $60 / 7200 = 8.33$ ms
- 典型扇区大小: 4KB(高级格式化)
- 典型容量: 1TB ~ 16TB

### 常考点
- 柱面号、磁头号、扇区号的寻址顺序
- 同一柱面不需要移动磁头臂(寻道时间为0)
- 磁道从外到内编号还是从内到外编号(通常外到内)
