# 85_光盘与磁带

## 核心概念
- **光盘(Optical Disc)**: 利用激光读写数据的存储介质
- **磁带(Tape)**: 利用磁性材料的顺序存取存储设备
- 两者都是**辅助存储器(外存)**, 但用途不同

### 光盘系列

| 类型 | 全称 | 容量 | 读写特性 |
|------|------|------|----------|
| CD-ROM | 只读光盘 | ~700MB | 只读 |
| CD-R | 一次性写入 | ~700MB | 写一次 |
| CD-RW | 可重写 | ~700MB | 可多次写 |
| DVD-ROM | 数字多功能只读 | 4.7GB(单层) | 只读 |
| DVD-R/RW | DVD写入/重写 | 4.7GB | 写一次/多次 |
| Blu-ray | 蓝光光盘 | 25GB(单层) | 多种 |

## 原理分析

### CD/DVD读写原理

**CD结构**:
```
激光照射方向
    ↓
┌──────────┐
│ 保护层    │
├──────────┤
│ 反射层    │ ← 凹坑(Pit)和陆地(Land)反射不同
├──────────┤
│ 数据层    │ ← 凹坑和陆地编码数据
├──────────┤
│ 基板      │
└──────────┘
```

**读取原理**:
- **凹坑(Pit)**: 激光照射后散射, 反射光弱
- **陆地(Land)**: 激光照射后正常反射, 反射光强
- 光电检测器检测反射光强度变化 → 解码为0和1

**写入原理(CD-R)**:
- 有机染料层: 激光照射区域变暗(不可逆), 形成"凹坑"
- CD-RW: 使用相变材料, 激光可使其在晶态和非晶态间切换

### 光盘容量计算

**CD**: 螺旋轨道, 轨道间距1.6μm
$$\text{轨道长度} \approx 2\pi r_{\text{平均}} \times \text{圈数}$$

**DVD**: 轨道间距0.74μm (CD的一半不到), 使用更短波长激光(650nm vs 780nm)

**Blu-ray**: 轨道间距0.32μm, 使用405nm蓝紫色激光, 单层25GB

### 光盘与磁盘对比

| 对比项 | 光盘 | 磁盘 |
|--------|------|------|
| 存储密度 | 较低 | 较高 |
| 读写速度 | 慢 | 快 |
| 介质成本 | 低 | 高 |
| 可擦写性 | CD-RW/DVD-RW有限 | 完全可擦写 |
| 主要用途 | 分发、备份 | 日常存储 |
| 寿命 | 较长(物理介质稳定) | 较短(机械磨损) |

### 磁带(Tape)

**结构**:
```
┌────────────────────────────┐
│     磁带(长条, 磁性涂覆)     │
│  ┌──┬──┬──┬──┬──┬──┬──┐   │
│  │T1│T2│T3│T4│T5│T6│...│   │ ← 多条磁道并行
│  └──┴──┴──┴──┴──┴──┴──┘   │
│     卷轴 ←──────→ 卷轴      │
│     (供给)        (收带)     │
└────────────────────────────┘
```

**特点**:
- **顺序存取**: 必须从头转到目标位置(不像磁盘可随机定位)
- **容量大**: 现代LTO磁带单盘可达18TB(未压缩)
- **成本低**: 单位容量成本远低于磁盘
- **速度慢**: 随机访问极慢, 但顺序读写可达300MB/s+
- **主要用途**: 数据备份、归档、冷存储

### 磁带存取时间

$$T_{\text{磁带}} = T_{\text{加载}} + T_{\text{定位}} + T_{\text{传输}}$$

- $T_{\text{加载}}$: 将磁带加载到驱动器(数十秒)
- $T_{\text{定位}}$: 转到目标位置(数秒~数分钟)
- $T_{\text{传输}}$: 读写数据本身

## 直观理解

**CD/DVD** = 唱片(激光读取凹坑)
**磁带** = 录音带(必须快进到目标位置才能听)

### 记忆要点
> **"光盘靠激光, 凹坑陆地0和1; 磁带顺序存, 备份归档用"**

## 知识关联

### 408考点
1. 光盘读写原理(凹坑和陆地的光学特性)
2. CD/DVD/Blu-ray容量差异的原因(激光波长、轨道间距)
3. 磁带的顺序存取特性
4. 各种外存的适用场景对比

### 外存层次总结

| 存储器 | 存取方式 | 速度 | 容量 | 用途 |
|--------|----------|------|------|------|
| SSD | 随机 | 快 | 中等 | 系统盘、热数据 |
| HDD | 随机 | 中等 | 大 | 日常存储 |
| 光盘 | 随机(但慢) | 慢 | 小 | 分发、备份 |
| 磁带 | 顺序 | 极慢 | 极大 | 归档、冷备份 |
