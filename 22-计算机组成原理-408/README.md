## 笔记格式

每个Markdown文件采用四段式结构：

- **核心概念** — 定义、术语、关键公式
- **原理分析** — 工作流程、推导过程、LaTeX计算题
- **直观理解** — 生活类比和记忆技巧
- **知识关联** — 跨章节联系、常考组合、易错陷阱

### 文件命名规则

| 后缀 | 含义 |
|------|------|
| `NN_中文标题.md` | 基础知识点（概念+原理+直观理解+知识关联） |
| `NNa_中文标题_408考点.md` | 408真题题型分析、易错辨析 |
| `NNa_中文标题_计算详解.md` | 详细计算步骤、公式推导、例题 |
| `NN_自测题.md` | 练习题+答案 |

## 学习建议

1. **按顺序学习**：第1章→第2章→第3章→第4章→第5章→第6章→第7章
2. **重点关注**：第2章(数据表示)、第3章(Cache/虚存)、第5章(流水线/微程序)是408高频考点
3. **计算题多练**：第2章的补码运算、IEEE 754转换，第3章的Cache地址划分和命中率计算是必考大题
4. **对比记忆**：直接映射vs全相联vs组相联、硬布线vs微程序、DMA vs 中断
