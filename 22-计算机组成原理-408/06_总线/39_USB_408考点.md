# 39_USB_408考点

## 408真题题型分析

### 题型1：USB传输类型

**典型题目**：区分四种USB传输类型的特点和适用场景。

| 传输类型 | 保证带宽 | 保证正确性 | 典型应用 |
|---------|---------|-----------|---------|
| 控制传输 | 否 | 是 | 设备配置 |
| 批量传输 | 否 | 是 | 文件传输 |
| 中断传输 | 是 | 是 | 键盘鼠标 |
| 等时传输 | 是 | 否 | 音视频 |

### 题型2：USB的特点

**典型题目**：判断关于USB的描述是否正确。

### 题型3：USB拓扑结构

**典型题目**：USB的拓扑结构类型。

## 典型例题

### 例题1

**题目**：以下关于USB的描述中，正确的是（）。

A. USB采用并行传输方式
B. USB采用树形拓扑结构
C. USB设备之间可以直接通信
D. USB不支持热插拔

**答案**：B

**解析**：
- A错误：USB是串行传输
- B正确：USB采用树形拓扑
- C错误：USB是主从结构，设备间不能直接通信
- D错误：USB支持热插拔

### 例题2

**题目**：USB键盘使用哪种传输类型？

A. 控制传输
B. 批量传输
C. 中断传输
D. 等时传输

**答案**：C

**解析**：键盘是人机交互设备，需要定期查询状态，使用中断传输。

### 例题3

**题目**：以下关于USB传输类型的描述，错误的是（）。

A. 等时传输保证带宽但不保证数据正确性
B. 批量传输用于大量数据的可靠传送
C. 控制传输用于设备的初始化和配置
D. 中断传输不保证带宽

**答案**：D

**解析**：中断传输保证带宽和低延迟，用于定期查询设备。

### 例题4

**题目**：USB 2.0的最高传输速率为（）。

A. 12Mbps
B. 480Mbps
C. 5Gbps
D. 480MBps

**答案**：B

**解析**：USB 2.0最高速率为480Mbps（注意是bit/s，不是Byte/s）。

## 易错辨析

### 易错点1：Mbps和MBps

- Mbps = Megabit per second（兆比特每秒）
- MBps = Megabyte per second（兆字节每秒）
- 480Mbps = 60MBps

### 易错点2：USB是主从结构

USB通信由主机发起，设备只能响应。设备之间不能直接通信。

### 易错点3：四种传输类型的选择

- 需要配置设备 → 控制传输
- 传文件 → 批量传输
- 键盘鼠标 → 中断传输
- 音视频 → 等时传输
