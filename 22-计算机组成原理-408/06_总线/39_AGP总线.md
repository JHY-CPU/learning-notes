# 40_AGP总线

## 核心概念

### AGP总线概述

**AGP（Accelerated Graphics Port）**：加速图形端口，是一种专门为显卡设计的高速总线接口。

### AGP的特点

| 特点 | 说明 |
|------|------|
| 专用接口 | 专门用于显卡 |
| 点对点 | 只连接显卡，不共享 |
| 高带宽 | 比PCI带宽高 |
| 直接访问内存 | 可以直接访问系统内存作为纹理内存 |

### AGP版本

| 版本 | 频率 | 带宽 | 电压 |
|------|------|------|------|
| AGP 1x | 66MHz | 266MB/s | 3.3V |
| AGP 2x | 66MHz | 533MB/s | 3.3V |
| AGP 4x | 66MHz | 1066MB/s | 1.5V |
| AGP 8x | 66MHz | 2133MB/s | 0.8V |

## 原理分析

### AGP与PCI的关系

```
CPU ←→ 北桥 ←→ AGP总线 ←→ 显卡
              ↕
           PCI总线 ←→ 其他设备
```

- AGP是北桥的专用高速通道
- PCI是共享总线
- 显卡使用AGP而不是PCI，避免占用PCI带宽

### AGP被淘汰的原因

1. **PCIe的出现**：PCIe x16带宽远超AGP 8x
2. **统一接口**：PCIe可以连接各种设备
3. **更好的扩展性**：PCIe支持热插拔、更灵活的配置

## 直观理解

### 生活类比

AGP就像**显卡专用快速通道**：
- 只有显卡能走这条路
- 速度比普通公路（PCI）快
- 但后来建了更快的高速公路（PCIe），这条路就被废弃了

### 记忆技巧

AGP：「显卡专用，已被PCIe取代」

## 知识关联

### AGP vs PCI vs PCIe

| 对比 | PCI | AGP | PCIe |
|------|-----|-----|------|
| 用途 | 通用 | 显卡专用 | 通用 |
| 带宽 | 132MB/s | 2133MB/s | 16GB/s+ |
| 状态 | 淘汰中 | 已淘汰 | 主流 |
| 架构 | 共享总线 | 点对点 | 点对点串行 |

### AGP的工作模式

AGP支持多种工作模式来优化图形数据传输：

| 模式 | 说明 | 应用场景 |
|------|------|----------|
| DMA模式 | 直接内存访问，将纹理从系统内存传到显存 | 常规纹理加载 |
| Execute模式 | GPU直接执行系统内存中的纹理数据，无需复制到显存 | 大纹理场景，节省显存 |
| 快写模式(Fast Writes) | CPU写入数据绕过北桥缓存直达GPU | 减少延迟 |

### AGP的DIME技术

**DIME（Direct Memory Execute）**允许显卡直接使用系统内存存储纹理数据：

```
传统方式：系统内存 → 显存 → GPU处理
DIME方式：系统内存 → GPU直接处理（跳过显存复制）
```

优势：
- 突破显存容量限制，可用系统内存作为扩展显存
- 减少数据复制开销
- AGP 8x的DIME带宽可达2.1GB/s

### AGP物理接口特征

- **插槽颜色**：通常为棕色，区别于白色PCI插槽
- **防呆设计**：插槽有定位键(Key)，防止插错方向
- **供电规范**：AGP 1x/2x使用3.3V，AGP 4x使用1.5V，AGP 8x使用0.8V
- **向下兼容**：AGP 4x插槽可兼容AGP 2x显卡（但需注意电压匹配）

### AGP被PCIe取代的技术原因

| 对比维度 | AGP 8x | PCIe x16 |
|----------|--------|----------|
| 带宽 | 2.1GB/s（单向） | 8GB/s（双向各8GB/s） |
| 架构 | 共享总线式 | 点对点串行 |
| 热插拔 | 不支持 | 支持 |
| 扩展性 | 仅限显卡 | 通用接口 |
| 功耗管理 | 基础 | 高级电源管理 |

### 易错陷阱

1. **AGP已被PCIe完全取代**
2. **AGP是点对点的**：不像PCI是共享总线
3. **AGP可以直接访问系统内存**：PCI不可以
4. **AGP不同版本电压不同**：不能混插，否则可能损坏硬件
5. **AGP总线宽度为32位**：与PCI相同，但通过更高频率和双倍/四倍数据速率提升带宽
