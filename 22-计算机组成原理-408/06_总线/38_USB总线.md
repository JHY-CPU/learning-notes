# 38_USB总线

## 核心概念

### USB总线概述

**USB（Universal Serial Bus）**：通用串行总线，用于连接外部设备的标准接口。

### USB的特点

| 特点 | 说明 |
|------|------|
| 即插即用 | 连接后自动识别和配置 |
| 热插拔 | 支持运行时插拔 |
| 串行传输 | 数据逐位传送 |
| 总线供电 | 可以为设备供电 |
| 拓扑结构 | 树形拓扑（通过Hub扩展） |
| 标准化 | 统一的接口标准 |

### USB版本

| 版本 | 年份 | 速度 | 商业名称 |
|------|------|------|---------|
| USB 1.0 | 1996 | 1.5/12Mbps | Full Speed |
| USB 2.0 | 2000 | 480Mbps | High Speed |
| USB 3.0 | 2008 | 5Gbps | Super Speed |
| USB 3.1 | 2013 | 10Gbps | Super Speed+ |
| USB 3.2 | 2017 | 20Gbps | Super Speed+ |
| USB4 | 2019 | 40Gbps | — |

### USB的拓扑结构

```
        主机（Host）
            │
         根集线器
        ┌───┼───┐
      Hub  设备  设备
     ┌──┼──┐
   设备 设备 设备
```

USB采用**树形拓扑**：
- 根节点是主机（Host）
- 通过集线器（Hub）扩展
- 最多5层Hub级联
- 最多127个设备

## 原理分析

### USB的传输类型

| 传输类型 | 特点 | 典型应用 |
|---------|------|---------|
| 控制传输 | 用于设备配置和控制 | 设备枚举 |
| 批量传输 | 大量数据，无时间要求 | 打印机、存储设备 |
| 中断传输 | 定期查询，低延迟 | 键盘、鼠标 |
| 等时传输 | 实时数据流，固定带宽 | 音频、视频 |

### USB的通信模型

```
应用层
  ↓
USB设备驱动
  ↓
USB总线驱动
  ↓
主控制器驱动（EHCI/xHCI）
  ↓
USB硬件
```

## 直观理解

### 生活类比

USB就像**标准快递系统**：
- 统一的包裹规格（标准接口）
- 有快递站（Hub）可以分发包裹
- 有普通快递（批量）、定时快递（中断）、实时快递（等时）
- 任何人都可以用（即插即用）

### 记忆技巧

USB：「通用、串行、即插即用」
- U = Universal（通用）
- S = Serial（串行）
- B = Bus（总线）

传输类型记忆：「控批中等」
- 控制传输
- 批量传输
- 中断传输
- 等时传输

### 易错陷阱

1. **USB是主从结构**：主机控制所有通信
2. **USB设备之间不能直接通信**
3. **USB的"速度"是理论最大值**：实际速度更低
4. **等时传输不保证数据正确**：只保证带宽
