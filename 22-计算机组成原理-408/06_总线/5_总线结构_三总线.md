# 06_总线结构_三总线

## 核心概念

### 三总线结构定义

**三总线结构**：在双总线基础上增加一条DMA总线，为高速IO设备提供专用数据传送通道。

```
         ┌────── 主存总线 ────── 主存
         │
   CPU ──┤
         │                        ┌── IO设备1（低速）
         └────── IO总线 ──────── IO接口
                                  └── IO设备2（低速）

   DMA控制器 ─────── DMA总线 ────── 主存
                      │
                高速IO设备（如磁盘）
```

### 三条总线的功能

| 总线 | 连接 | 功能 |
|------|------|------|
| 主存总线 | CPU ↔ 主存 | CPU读写主存 |
| IO总线 | CPU ↔ IO接口 | CPU控制IO设备 |
| DMA总线 | DMA控制器 ↔ 主存 | 高速IO设备直接与主存传送数据 |

## 原理分析

### 三总线结构的工作方式

1. **CPU正常工作**：通过主存总线访问主存，通过IO总线控制IO设备
2. **高速IO传送**：DMA控制器通过DMA总线直接在IO设备和主存之间传送数据
3. **并行工作**：CPU、高速IO、低速IO可以同时工作（在不同总线上）

### 三总线的性能优势

$$
\text{系统总吞吐量} = B_{\text{主存总线}} + B_{\text{IO总线}} + B_{\text{DMA总线}}
$$

三总线结构允许：
- CPU通过主存总线取指令和读写数据
- 低速IO设备通过IO总线传送数据
- 高速IO设备通过DMA总线传送数据

三条总线可以**同时工作**，系统并行度最高。

## 直观理解

### 生活类比

**三总线就像城市有了三条独立的道路**：
- 高速公路（主存总线）：核心区域之间的快速通道
- 普通公路（IO总线）：连接一般设施
- 专线公路（DMA总线）：专门为高速运输车辆准备的专用通道

### 记忆技巧

三总线：「主IO-DMA」
- **主**存总线：CPU和主存
- **IO**总线：CPU和低速IO
- **DMA**总线：高速IO和主存

## 知识关联

### 三种总线结构对比

| 对比维度 | 单总线 | 双总线 | 三总线 |
|---------|--------|--------|--------|
| 总线数量 | 1 | 2 | 3 |
| 并行度 | 最低 | 中等 | 最高 |
| IO效率 | 低 | 较高 | 高 |
| 硬件复杂度 | 最低 | 中等 | 最高 |
| 适用场景 | 简单系统 | 一般系统 | 高性能系统 |

### 易错陷阱

1. **三总线结构中，高速IO走DMA总线，低速IO走IO总线**
2. **三条总线可以同时工作**，这是三总线的核心优势
3. **DMA总线不需要总线仲裁**：通常DMA总线专用，不存在竞争
