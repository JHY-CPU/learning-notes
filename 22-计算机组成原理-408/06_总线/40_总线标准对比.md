# 41_总线标准对比

## 核心概念

### 主要总线标准对比表

| 总线标准 | 数据宽度 | 频率 | 带宽 | 架构 | 状态 |
|---------|---------|------|------|------|------|
| ISA | 8/16位 | 8MHz | 8-16MB/s | 并行共享 | 淘汰 |
| EISA | 32位 | 8MHz | 33MB/s | 并行共享 | 淘汰 |
| VESA | 32位 | 33MHz | 132MB/s | 并行共享 | 淘汰 |
| PCI | 32/64位 | 33/66MHz | 132-528MB/s | 并行共享 | 逐渐淘汰 |
| AGP | 32位 | 66MHz | 266-2133MB/s | 并行点对点 | 淘汰 |
| PCIe | 串行 | GHz级 | 最高64GB/s | 串行点对点 | 主流 |
| USB | 串行 | — | 1.5Mbps-40Gbps | 串行主从 | 主流 |

### 总线标准的发展趋势

```
ISA (8MHz, 16bit)
  ↓
EISA (8MHz, 32bit)
  ↓
VESA (33MHz, 32bit)
  ↓
PCI (66MHz, 64bit)     AGP (显卡专用)
  ↓                        ↓
  └──── PCIe ←─────────────┘  (统一取代)
```

### 关键特征对比

| 对比维度 | ISA | PCI | PCIe | USB |
|---------|-----|-----|------|-----|
| 传输方式 | 并行 | 并行 | 串行 | 串行 |
| 拓扑结构 | 共享 | 共享 | 点对点 | 主从树形 |
| 即插即用 | 否 | 是 | 是 | 是 |
| 热插拔 | 否 | 否 | 是 | 是 |
| 最大带宽 | 16MB/s | 528MB/s | 64GB/s | 40Gbps |
| 主要用途 | 早期扩展卡 | 通用扩展 | 通用扩展（主流） | 外部设备 |

## 直观理解

### 记忆技巧

总线发展：「IEPVU」（ISA→EISA→VESA→PCI→USB）
核心趋势：「并行→串行，共享→点对点」

### 总线标准选择

| 应用场景 | 推荐总线 |
|---------|---------|
| 显卡 | PCIe x16 |
| 高速SSD | PCIe x4 (NVMe) |
| 网卡 | PCIe x1/x4 |
| 键盘鼠标 | USB |
| 打印机 | USB |
| 外接硬盘 | USB 3.x |

### 易错陷阱

1. **AGP已被PCIe取代**
2. **PCIe虽然串行，但比并行的PCI快**
3. **USB是外部总线标准**
4. **ISA/EISA/VESA都已淘汰**
