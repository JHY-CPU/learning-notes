# 07_总线结构_408考点

## 408真题题型分析

### 题型1：总线结构的对比

**典型题目**：比较单总线、双总线、三总线的特点。

**解题关键**：
- 单总线：所有设备共享一条总线，简单但有瓶颈
- 双总线：分离CPU-主存和CPU-IO，减少竞争
- 三总线：增加DMA总线，高速IO专用通道

### 题型2：画总线结构图

**典型题目**：画出某种总线结构的框图，并说明各总线的作用。

### 题型3：性能分析

**典型题目**：在某种总线结构下，分析系统的最大并行度。

## 典型例题

### 例题1

**题目**：以下关于总线结构的描述中，错误的是（）。

A. 单总线结构中，所有设备共享同一组总线
B. 双总线结构通常以CPU为中心
C. 三总线结构中，三条总线不能同时工作
D. 单总线结构简单，但存在总线瓶颈

**答案**：C

**解析**：三总线结构的优势就是三条总线可以同时工作。C的描述错误。

### 例题2

**题目**：在三总线结构中，高速IO设备（如磁盘）通过哪条总线与主存通信？

A. 主存总线
B. IO总线
C. DMA总线
D. 系统总线

**答案**：C

**解析**：高速IO设备通过DMA总线直接与主存进行数据传送，不占用CPU总线。

### 例题3

**题目**：某系统采用单总线结构，总线带宽为100MB/s，有5个设备需要通信。若各设备平均分配总线带宽，每个设备的带宽是多少？若改为三总线结构（主存总线100MB/s，IO总线50MB/s，DMA总线100MB/s），系统总带宽是多少？

**解答**：

单总线：每个设备平均带宽 = $100 / 5 = 20$MB/s

三总线：系统总带宽 = $100 + 50 + 100 = 250$MB/s（三条总线可同时工作）

### 例题4

**题目**：画出三总线结构的框图，标注CPU、主存、高速IO设备、低速IO设备的位置，以及三条总线的名称。

**参考答案**：

```
        ┌──── 主存总线 ──── 主存
        │
  CPU ──┤
        │                     ┌── 低速IO设备
        └──── IO总线 ────── IO接口
                             └── 低速IO设备

  DMA控制器 ─── DMA总线 ──── 主存
                 │
           高速IO设备
```

## 易错辨析

### 易错点1：总线数量和并行度

- 单总线：只有一对设备能通信
- 双总线：CPU-主存和CPU-IO可以并行
- 三总线：CPU-主存、CPU-IO、DMA-主存三者都可以并行

### 易错点2：DMA总线的方向

DMA总线主要用于高速IO设备与主存之间的数据传送。它是一组完整的总线（数据+地址+控制），由DMA控制器控制。

### 易错点3：双总线中IO是否能访问主存

在以CPU为中心的双总线结构中：
- IO设备不能直接访问主存
- 必须经过CPU中转

在以存储器为中心的双总线结构中：
- IO设备可以通过总线直接访问主存

考试中通常指以CPU为中心的双总线。
