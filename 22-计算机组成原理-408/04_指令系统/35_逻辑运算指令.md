# 36_逻辑运算指令

## 核心概念

### 常见逻辑运算指令

| 指令 | 格式 | 含义 | 说明 |
|:---|:---|:---|:---|
| **AND** | `AND dst, src` | $dst \leftarrow dst \wedge src$ | 按位与 |
| **OR** | `OR dst, src` | $dst \leftarrow dst \vee src$ | 按位或 |
| **NOT** | `NOT dst` | $dst \leftarrow \neg dst$ | 按位取反 |
| **XOR** | `XOR dst, src` | $dst \leftarrow dst \oplus src$ | 按位异或 |
| **TEST** | `TEST dst, src` | $dst \wedge src$（不保存结果） | 测试，只设标志 |

### 移位指令

| 指令 | 名称 | 含义 |
|:---|:---|:---|
| **SHL/SAL** | 逻辑左移/算术左移 | 左移，低位补0 |
| **SHR** | 逻辑右移 | 右移，高位补0 |
| **SAR** | 算术右移 | 右移，高位补符号位 |
| **ROL** | 循环左移 | 最高位循环到最低位 |
| **ROR** | 循环右移 | 最低位循环到最高位 |
| **RCL** | 带进位循环左移 | CF参与循环 |
| **RCR** | 带进位循环右移 | CF参与循环 |

### 逻辑运算的典型用途

| 操作 | 指令组合 | 效果 |
|:---|:---|:---|
| **清零** | `XOR R, R` | $R \leftarrow 0$ |
| **置位** | `OR R, mask` | 指定位设为1 |
| **清位** | `AND R, \neg mask` | 指定位清零 |
| **取反** | `XOR R, mask` | 指定位取反 |
| **测试位** | `TEST R, mask` | 测试指定位是否为1 |
| **乘以2^n** | `SHL R, n` | 左移n位 |
| **除以2^n** | `SAR R, n` | 右移n位（有符号） |

## 原理分析

### 标志位设置

逻辑运算指令对标志位的影响：

| 标志 | 说明 |
|:---:|:---|
| **ZF** | 结果为0则置1 |
| **SF** | 结果最高位为1则置1 |
| **PF** | 结果低8位中1的个数为偶数则置1 |
| **CF** | 清零（逻辑运算不产生进位） |
| **OF** | 清零 |

### 移位操作的标志位

| 标志 | 说明 |
|:---:|:---|
| **CF** | 最后移出的位 |
| **OF** | 移位后最高两位不同则置1（仅移1位时有意义） |
| **ZF/SF** | 根据结果设置 |

### 逻辑运算的计算示例

**AND掩码**：提取指定位

```
R1 = 1011 0110
AND  0000 1111  (mask = 0FH)
=    0000 0110  (只保留低4位)
```

**OR置位**：将第3位置1

```
R1 = 1011 0110
OR   0000 1000  (mask = 08H)
=    1011 1110
```

**XOR取反**：将低4位取反

```
R1 = 1011 0110
XOR  0000 1111  (mask = 0FH)
=    1011 1001
```

### 易错陷阱
1. **AND清位需要用反码**：`AND R, 11110111` 清第3位
2. **算术右移≠逻辑右移**：有符号数用SAR，无符号用SHR
3. **XOR R, R是清零最快方式**：比MOV R, #0快
4. **循环移位与移位不同**：循环移位不丢失位
