# 05_指令格式_408考点

## 核心概念

### 408考纲要求
- 指令格式的分类与设计
- 操作码与地址码的关系
- 指令字长与寻址范围的计算
- 扩展操作码的方案设计

### 高频考点一览

1. **给定指令字长和地址数，求操作码位数**
2. **给定操作码位数，求最大指令条数**
3. **给定指令字长和地址格式，求寻址范围**
4. **扩展操作码方案设计与验证**
5. **指令字长与存储字长的关系**

## 原理分析

### 考点一：OP位数计算

**基本公式**：

$$操作码位数 = 指令字长 - 地址字段个数 \times 每个地址字段位数$$

**例题1**：某机指令字长16位，采用二地址格式，每个地址字段占6位，求操作码位数。

解：$OP位数 = 16 - 2 \times 6 = 4$位

最多表示 $2^4 = 16$ 种不同操作。

### 考点二：地址空间计算

**例题2**：某机指令字长32位，操作码8位，三地址格式，求每个地址字段位数和寻址范围。

解：
$$地址字段位数 = \frac{32 - 8}{3} = 8位$$
$$寻址范围 = 2^8 = 256个存储单元$$

### 考点三：混合地址格式的位数分配

**例题3**：某机指令字长16位，操作码4位。现有三种指令格式：
- 三地址格式：每个地址 $n_3$ 位
- 二地址格式：每个地址 $n_2$ 位
- 一地址格式：地址 $n_1$ 位

求 $n_3$、$n_2$、$n_1$。

解：
$$n_3 = \frac{16 - 4}{3} = 4位$$
$$n_2 = \frac{16 - 4}{2} = 6位$$
$$n_1 = 16 - 4 = 12位$$

### 考点四：存储器容量与地址码位数的关系

设存储器有 $N$ 个存储单元，地址码需要 $\lceil \log_2 N \rceil$ 位。

**例题4**：某机主存容量64KB，按字节编址，指令采用直接寻址，求地址码至少多少位？

解：
$$64KB = 2^{16}B，需\lceil \log_2 2^{16} \rceil = 16位$$

### 考点五：指令条数与地址格式的综合计算

**例题5**：某机指令字长16位，操作码占4位，采用扩展操作码：
- 15条三地址指令
- 15条二地址指令
- 15条一地址指令
- 16条零地址指令

验证方案可行性。

解：
- 三地址：4位OP，$2^4 - 1 = 15$条（保留1111），地址各 $(16-4)/3 = 4$位。✓
- 二地址：OP扩展到8位（前4位=1111），新4位 $2^4-1=15$条（保留1111），地址各 $(16-8)/2 = 4$位。✓
- 一地址：OP扩展到12位（前8位=11111111），新4位 $2^4-1=15$条（保留1111），地址 $16-12=4$位。✓
- 零地址：OP全部16位，$2^4=16$条。✓

总计 $15+15+15+16 = 61$ 条。**方案可行。**

### 考点六：给定指令条数求OP位数

**例题6**：某机有128条指令，问操作码至少需要几位？

解：
$$\lceil \log_2 128 \rceil = 7位$$

注意：$2^7 = 128$，刚好7位够用。若要129条指令，则需8位。

## 直观理解

### 常用公式速查

| 求什么 | 公式 |
|:---|:---|
| 操作码位数 | $k = \text{指令字长} - m \times n$（m为地址数，n为每位地址位数） |
| 最大指令条数 | $2^k$（定长OP）或按扩展方案计算 |
| 每字段位数 | $n = (L - k) / m$ |
| 寻址范围 | $2^n$ |
| 地址码总位数 | $L - k$ |
| 存储器地址位数 | $\lceil \log_2 N \rceil$（N为存储单元数） |

### 记忆技巧

**"三个变量知二求一"**：指令字长 $L$、操作码位数 $k$、地址码总位数 $m \times n$，知道任意两个可求第三个。

## 知识关联

### 跨章节联系
- **第3章**：存储容量→地址位数→地址码设计
- **第5章**：操作码译码→控制信号→指令执行

### 易错陷阱
1. **操作码位数 vs 最大指令条数**：$k$ 位OP最多 $2^k$ 条（不是 $k$ 条）
2. **"至少几位"要用向上取整**：$\lceil \log_2 n \rceil$ 而非 $\lfloor \log_2 n \rfloor$
3. **扩展操作码保留码点**：每级扩展需留1个编码，不是满编 $2^m$
4. **地址字段位数≠可寻址单元数**：$n$ 位→$2^n$ 个单元，不要混淆
5. **指令字长=机器字长？不一定**：指令字长可以是机器字长的整数倍
