# 26_寻址方式_计算详解

## 核心概念

### EA计算的关键要素

- 各寄存器的当前值（BR、IX、PC、SP、通用寄存器）
- 形式地址A的值和编码方式（无符号/补码）
- 指令长度（用于PC计算）
- 存储器编址方式（按字节/按字）

## 例题详解

### 例题1：基础EA计算

**题目**：某机按字节编址，BR = 3000H，IX = 0200H，PC = 1000H（当前指令2字节）。存储器内容：M[1000H] = 4000H，M[3200H] = 5678H，M[4000H] = ABCDH。求各指令EA和操作数。

**（1）直接寻址：`LOAD R1, [1000H]`**

$$EA = A = 1000H$$
$$操作数 = M[1000H] = 4000H$$
$$访存次数 = 1$$

**（2）间接寻址：`LOAD R1, @1000H`**（@表示间接）

$$EA = M[A] = M[1000H] = 4000H$$
$$操作数 = M[4000H] = ABCDH$$
$$访存次数 = 2$$

**（3）基址寻址：`LOAD R1, [BR + 200H]`**

$$EA = BR + 200H = 3000H + 200H = 3200H$$
$$操作数 = M[3200H] = 5678H$$
$$访存次数 = 1$$

**（4）变址寻址：`LOAD R1, [1000H + IX]`**

$$EA = A + IX = 1000H + 0200H = 1200H$$
$$操作数 = M[1200H]$$
$$访存次数 = 1$$

**（5）相对寻址：`JMP +20H`**

$$PC_{取指后} = 1000H + 2 = 1002H$$
$$EA = PC + 20H = 1002H + 20H = 1022H$$
$$PC \leftarrow 1022H（跳转目标）$$
$$访存次数 = 0（跳转不取操作数）$$

### 例题2：基址+变址组合

**题目**：BR = 5000H，IX = 0300H，指令 `LOAD R1, [BR + IX + 10H]`。求EA。

$$EA = BR + IX + A = 5000H + 0300H + 10H = 5310H$$

$$操作数 = M[5310H]$$
$$访存次数 = 1$$

### 例题3：间接寻址的深度计算

**题目**：A = 2000H，M[2000H] = 3000H，M[3000H] = 4000H，M[4000H] = 5000H。

**一次间接**：
$$EA = M[2000H] = 3000H$$
$$操作数 = M[3000H] = 4000H$$
$$访存次数 = 2$$

**二次间接**：
$$EA = M[M[2000H]] = M[3000H] = 4000H$$
$$操作数 = M[4000H] = 5000H$$
$$访存次数 = 3$$

### 例题4：相对寻址的PC值计算

**题目**：指令JMP位于地址2000H处，指令长度4字节，偏移量A = 08H。求跳转目标。

$$PC_{取指后} = 2000H + 4 = 2004H$$
$$EA = 2004H + 08H = 200CH$$

**若偏移量为补码F8H（-8）**：

$$EA = 2004H + (-8) = 2004H - 8 = 1FFCH$$

### 例题5：堆栈操作

**题目**：SP = 1000H（满递减堆栈），执行PUSH R1（R1 = ABCDH）。

**执行过程**：
1. $SP \leftarrow SP - 4 = 0FFCH$（先减4字节）
2. $M[0FFCH] \leftarrow ABCDH$（写入栈顶）

$$访存次数 = 1（写入M[SP]）$$

执行POP R2：
1. $R2 \leftarrow M[SP] = M[0FFCH] = ABCDH$
2. $SP \leftarrow SP + 4 = 1000H$

$$访存次数 = 1（读取M[SP]）$$

## 直观理解

### 解题流程图

```
拿到指令 → 识别寻址方式 → 套EA公式 → 代入寄存器值 → 计算EA → 查M[EA]得操作数
                                                    │
                                            统计访存次数
```

### 易错点总结

| 易错点 | 正确理解 |
|:---|:---|
| PC值 | 取指后PC已指向下一条指令 |
| 访存次数 | 不含取指令的访存 |
| 补码偏移 | 相对寻址偏移量可正可负 |
| 间接层数 | 一次间接=2次访存，二次=3次访存 |
| 堆栈方向 | 递减栈PUSH时SP减，递增栈PUSH时SP加 |
