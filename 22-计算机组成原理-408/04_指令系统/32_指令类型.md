# 33_指令类型

## 核心概念

### 指令系统的功能分类

一台完备的指令系统应包含以下基本类型的指令：

| 类别 | 功能 | 典型指令 |
|:---|:---|:---|
| **数据传送** | 数据在寄存器/存储器间传送 | MOV、LOAD、STORE、PUSH、POP |
| **算术运算** | 数值计算 | ADD、SUB、MUL、DIV、INC、DEC |
| **逻辑运算** | 位级操作 | AND、OR、NOT、XOR、移位 |
| **程序控制** | 改变执行顺序 | JMP、Jcc、CALL、RET、LOOP |
| **输入输出** | 与外设交互 | IN、OUT |
| **其他** | 系统控制 | NOP、HLT、置标志、特权指令 |

### 各类指令的必要性

1. **数据传送指令**：最基础，没有数据传送其他操作无从谈起
2. **算术运算指令**：完成数值计算（加减乘除）
3. **逻辑运算指令**：位操作、掩码、比较
4. **程序控制指令**：实现分支、循环、函数调用
5. **I/O指令**：与外部设备通信
6. **其他指令**：系统管理、状态控制

### 指令系统完备性的基本要求

$$\text{完备性} = \text{数据传送} + \text{算术运算} + \text{逻辑运算} + \text{程序控制} + \text{I/O}$$

一个指令系统如果缺少上述任何一类，就不能支持完整的计算。

## 原理分析

### 指令类型与数据通路的关系

```
存储器 ←──数据传送──→ 寄存器 ←──算术/逻辑运算──→ ALU
                          │
                    程序控制修改PC
                          │
                    I/O指令访问外设
```

- **数据传送**：数据在存储器、寄存器之间流动
- **算术/逻辑**：数据在ALU中处理
- **程序控制**：修改PC改变执行流
- **I/O**：数据在CPU和外设之间流动

### 各类指令的比例

根据实际程序统计：

| 指令类型 | 执行频率（近似） |
|:---|:---|
| 数据传送 | 30~40% |
| 算术运算 | 20~30% |
| 比较/测试 | 10~20% |
| 程序控制（跳转） | 10~20% |
| 逻辑运算 | 5~10% |
| I/O | < 5% |

数据传送和算术运算是最常用的两类。

## 直观理解

### 生活类比

把指令系统比作**工厂的操作手册**：
- **数据传送** = 搬运工（把东西搬到该去的地方）
- **算术运算** = 计算员（加减乘除）
- **逻辑运算** = 质检员（判断是/否、对/错）
- **程序控制** = 调度员（决定下一步做什么）
- **I/O** = 仓库管理员（与外部交接物资）

### 记忆技巧

> **"传算逻控IO，五类缺一不可"**

## 知识关联

### 跨章节联系
- **第5章（CPU）**：各类指令对应的微操作不同
- **第6章（总线）**：数据传送经数据总线，I/O经I/O总线
- **高级语言**：赋值→传送，表达式→算术，if→控制，位操作→逻辑

### 易错陷阱
1. **"数据传送"不只是MOV**：LOAD/STORE/PUSH/POP都是
2. **比较指令属于逻辑运算类**：CMP本质是减法但不保存结果
3. **I/O指令是独立类型**：不能归入数据传送
## 代码/模拟

### MIPS汇编演示五类指令

```mips
# ============================================
# 五类指令的MIPS汇编示例 - 适用于408考研复习
# ============================================

# ===== 1. 数据传送指令 =====
# 寄存器之间、寄存器与存储器之间的数据搬移

lw   $t0, 0($sp)       # Load Word: 从内存加载到寄存器
sw   $t0, 8($sp)       # Store Word: 从寄存器存储到内存
move $t0, $t1          # 寄存器间传送 (伪指令, 实际为 add $t0,$t1,$zero)
lui  $t0, 0x1234       # 加载立即数到高16位

# ===== 2. 算术运算指令 =====
# 加减乘除

add  $t0, $t1, $t2     # $t0 = $t1 + $t2  (加法)
sub  $t0, $t1, $t2     # $t0 = $t1 - $t2  (减法)
addi $t0, $t1, 100     # $t0 = $t1 + 100  (加立即数)
mult $t1, $t2          # HI:LO = $t1 × $t2 (乘法)
div  $t1, $t2          # LO = 商, HI = 余数 (除法)
mflo $t0               # $t0 = LO (取商)
mfhi $t0               # $t0 = HI (取余数)

# ===== 3. 逻辑运算指令 =====
# 与、或、非、异或、移位

and  $t0, $t1, $t2     # $t0 = $t1 & $t2  (按位与)
or   $t0, $t1, $t2     # $t0 = $t1 | $t2  (按位或)
nor  $t0, $t1, $zero   # $t0 = ~$t1       (按位取反)
xor  $t0, $t1, $t2     # $t0 = $t1 ^ $t2  (异或)
sll  $t0, $t1, 4       # $t0 = $t1 << 4   (逻辑左移)
srl  $t0, $t1, 4       # $t0 = $t1 >> 4   (逻辑右移)
sra  $t0, $t1, 4       # $t0 = $t1 >>> 4  (算术右移)
slt  $t0, $t1, $t2     # $t0 = ($t1 < $t2)?1:0 (小于置位)

# ===== 4. 程序控制指令 =====
# 转移、跳转、子程序调用

beq  $t0, $t1, label   # if ($t0 == $t1) goto label (相等转移)
bne  $t0, $t1, label   # if ($t0 != $t1) goto label (不等转移)
j    target             # goto target (无条件跳转)
jal  func               # $ra = PC+4; goto func (子程序调用)
jr   $ra                # goto $ra (子程序返回)

# ===== 5. I/O指令 =====
# MIPS通过内存映射I/O, 没有专门的I/O指令
# 用 lw/sw 访问设备寄存器地址

lw   $t0, 0xFFFF0000   # 从键盘数据寄存器读取按键
sw   $t0, 0xFFFF000C   # 向显示器数据寄存器写入字符

# 汇总统计
print("指令类型使用频率:")
types = {
    "数据传送": "30-40% (MOV/LOAD/STORE/PUSH/POP)",
    "算术运算": "25-35% (ADD/SUB/MUL/DIV)",
    "逻辑运算": "5-10%  (AND/OR/NOT/XOR/SHIFT)",
    "程序控制": "10-20% (JMP/CALL/RET/BRANCH)",
    "I/O":      "<5%    (IN/OUT)",
}
```

4. **"其他指令"不是可有可无**：HLT、NOP、中断指令都很重要
