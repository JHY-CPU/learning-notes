# 27_定点小数表示

## 核心概念

### 定点数的定义

**定点数**（Fixed-Point Number）是指小数点位置固定不变的数。与浮点数（小数点位置可变）相对。

### 定点数的两种格式

#### 定点整数
小数点固定在最低数值位之后。

```
| 符号位 | 数值位(整数部分) | . (隐含小数点)
```

例如8位定点整数：$X = -42$，$[-42]_{\text{补}} = 11010110$

表示的真值范围：$-2^{n-1} \sim +(2^{n-1}-1)$（补码）

#### 定点小数（纯小数）
小数点固定在符号位之后、数值位之前。

```
| 符号位 | . (隐含小数点) | 数值位(小数部分) |
```

例如8位定点小数（1位符号位 + 7位小数位）：

$X = 0.75$，$[0.75]_{\text{补}} = 0.1100000$（小数点隐含在符号位后）

$X = -0.625$，$[-0.625]_{\text{补}} = 1.0110000$

### 定点小数的表示范围

$n$ 位定点小数（1位符号位 + $n-1$ 位小数位）：

**补码范围**：$-1 \leq X \leq +(1 - 2^{-(n-1)})$

**原码范围**：$-(1 - 2^{-(n-1)}) \leq X \leq +(1 - 2^{-(n-1)})$

| 位数 | 最小值(补码) | 最大值(补码) |
|------|-------------|-------------|
| 8位 | -1 | +0.9921875 |
| 16位 | -1 | +0.9999695 |

## 原理分析

### 定点小数的二进制权值

小数点后的第 $k$ 位权值为 $2^{-k}$：

$$X = x_0 \cdot 2^0 + x_{-1} \cdot 2^{-1} + x_{-2} \cdot 2^{-2} + \cdots + x_{-(n-1)} \cdot 2^{-(n-1)}$$

对于定点小数，$x_0$ 是符号位，$x_{-1} \sim x_{-(n-1)}$ 是小数部分：

$$X = -x_0 + \sum_{i=1}^{n-1} x_{-i} \cdot 2^{-i}$$

### 示例：8位定点小数（补码）

**$0.75_{10}$ 的编码**：
- $0.75 = 0.5 + 0.25 = 2^{-1} + 2^{-2}$
- 二进制：$0.1100000$
- 补码（正数不变）：$0.1100000$

**$-0.625_{10}$ 的编码**：
- $0.625 = 0.5 + 0.125 = 2^{-1} + 2^{-3}$
- 二进制绝对值：$0.1010000$
- 原码：$1.1010000$
- 数值位取反：$1.0101111$
- 加1：$1.0110000$（补码）

**验证**：$1.0110000$（补码）→ 数值位取反+1 = $0.1001111 + 0.0000001 = 0.1010000 = 0.625$，符号为负，所以 $X = -0.625$ ✓

### 定点小数与定点整数的统一理解

定点数的本质是**小数点位置固定**：
- 定点整数：小数点在最后一位之后（权值为 $2^0, 2^1, \ldots$）
- 定点小数：小数点在第一位之后（权值为 $2^{-1}, 2^{-2}, \ldots$）

在硬件中，两者没有本质区别，只是对同一组二进制位的不同解释。

## 直观理解

### 生活类比
定点小数就像一把只有固定刻度的尺子：
- 刻度是固定的（如每格0.125）
- 不能表示刻度之间的值
- 范围和精度由位数决定

### 记忆要点
- 定点小数的"点"在符号位后面
- 定点小数只能表示 $[-1, +1)$ 范围内的值
- 定点小数的精度 = $2^{-(n-1)}$
- 定点整数和定点小数的编码规则完全相同，只是权值解释不同

## 知识关联

### 与浮点数的对比

| 特性 | 定点数 | 浮点数 |
|------|--------|--------|
| 小数点位置 | 固定 | 浮动 |
| 表示范围 | 有限 | 极大 |
| 精度 | 均匀 | 相对均匀 |
| 硬件复杂度 | 低 | 高 |
| 应用场景 | 整数运算、DSP | 科学计算 |

### 408考试要点
- 定点小数的格式和权值
- 定点小数的表示范围
- 定点小数的编码（原码/补码）
- 与定点整数的异同

### 易错陷阱
1. **定点小数的范围不是 $-1 \sim +1$**（原码最大不到1，补码可以到-1）
2. **定点小数的权值是负指数**，不要和定点整数混淆
3. **补码中 $-1$ 的表示**：$1.0000000$（注意：原码无法表示 $-1.0$）
