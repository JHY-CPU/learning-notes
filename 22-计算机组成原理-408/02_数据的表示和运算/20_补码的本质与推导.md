# 21_补码的本质与推导

## 核心概念

### 补码的本质

补码的本质是**模运算**。在模 $M$ 的运算系统中，每个负数都有一个等价的正数（即它的"补数"），使得两者相加模 $M$ 为0。

**定义**：对于模 $M$ 的系统，$X$ 的补码 $= M + X$（当 $X < 0$ 时）

### 从模运算推导补码

以8位二进制系统为例（模 $M = 2^8 = 256$）：

对于负数 $X$（$-128 \leq X < 0$）：

$$[X]_{\text{补}} = 256 + X$$

**示例**：
- $[-3]_{\text{补}} = 256 + (-3) = 253 = 11111101_2$
- $[-42]_{\text{补}} = 256 + (-42) = 214 = 11010110_2$
- $[-128]_{\text{补}} = 256 + (-128) = 128 = 10000000_2$

## 原理分析

### 时钟类比的严格推导

时钟系统：模 $M = 12$

- 当前3点，要回退5个小时
- $3 - 5 = -2$
- 在模12系统中：$-2 \equiv 10 \pmod{12}$（因为 $-2 + 12 = 10$）
- 所以"回退5小时"等价于"前进7小时"，因为 $-5 \equiv 7 \pmod{12}$
- $-5$ 的"补码"就是 $12 + (-5) = 7$

推广到二进制：模 $M = 2^n$，$-B$ 的补码 $= 2^n - B$。

### 为什么减法可以变成加法？

要计算 $A - B$（假设 $A, B > 0$ 且 $A < B$）：

$$A - B = A + (-B)$$

在模 $2^n$ 系统中：

$$A + (-B) \equiv A + (2^n - B) \pmod{2^n}$$

$$= (A - B) + 2^n$$

因为 $A - B < 0$，所以 $(A - B) + 2^n$ 就是 $A - B$ 的补码表示！

**关键推导**：

$$A - B = A + (2^n - B) \pmod{2^n}$$

$2^n - B$ 就是 $-B$ 的补码。所以：

$$[A - B]_{\text{补}} = [A]_{\text{补}} + [-B]_{\text{补}} \pmod{2^n}$$

这就是**补码减法统一为加法**的数学证明。

### 补码的另一种推导：从原码到补码

对于负数 $-X$（$X > 0$）：

**原码表示**：符号位1 + $X$ 的二进制

**反码表示**：符号位1 + $X$ 的二进制每位取反

**补码**：反码 + 1

**为什么取反+1等于模+真值？**

设 $X$ 的二进制为 $b_{n-2} b_{n-3} \cdots b_0$（$n-1$ 位）

取反后：$\bar{b}_{n-2} \bar{b}_{n-3} \cdots \bar{b}_0$

$$b_i + \bar{b}_i = 1$$

所以：$X + \bar{X} = 2^{n-1} - 1$（所有位都是1）

取反 + 1：
$$\bar{X} + 1 = (2^{n-1} - 1 - X) + 1 = 2^{n-1} - X$$

加上符号位（$2^{n-1}$）：

$$[−X]_{\text{补}} = 2^{n-1} + (2^{n-1} - X) = 2^n - X = 2^n + (-X)$$

这正是补码的定义！

## 直观理解

### 深层理解

补码不只是一个"技巧"，它是模运算系统的自然结果：
- 计算机的 $n$ 位寄存器就是一个模 $2^n$ 的系统
- 在这个系统中，负数和它的补码是完全等价的
- "减法变加法"不是近似，而是精确的数学等价

### 记忆技巧

1. **补码 = 补数**：在模系统中，一个数和它的"补"相加等于模
2. **时钟直觉**：-5的补码就是"模-5"，在12小时制中就是7点
3. **取反+1**：是计算补码的快速方法，但本质是"模+真值"
