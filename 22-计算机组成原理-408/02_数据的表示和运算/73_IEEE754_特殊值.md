# 73_IEEE754_特殊值

## 核心概念

### IEEE 754 特殊值总览

IEEE 754 通过保留**阶码全0**和**阶码全1**来表示特殊值：

| 阶码 | 尾数 | 含义 |
|------|------|------|
| 0 | 0 | **零**（±0） |
| 0 | ≠0 | **非规格化数**（Denormalized/Subnormal） |
| 1~254 | 任意 | **规格化数** |
| 255 | 0 | **无穷大**（±∞） |
| 255 | ≠0 | **NaN**（Not a Number） |

### 零（Zero）

**表示**：阶码 = 0，尾数 = 0

```
+0: 0 00000000 00000000000000000000000
-0: 1 00000000 00000000000000000000000
```

**特点**：
- $+0$ 和 $-0$ 是两个不同的表示
- 但 $+0 = -0$ 在比较时为真
- $1/+0 = +\infty$，$1/-0 = -\infty$
- 用于表示非常小的数（下溢到零）

### 无穷大（Infinity）

**表示**：阶码 = 全1，尾数 = 0

```
+∞: 0 11111111 00000000000000000000000
-∞: 1 11111111 00000000000000000000000
```

**产生原因**：
- 上溢：$10^{300} \times 10^{300}$（double范围外）
- 除以零：$1.0 / 0.0 = +\infty$
- $\infty + \infty = \infty$，$\infty \times \infty = \infty$

**运算规则**：
- $\infty +$ 有限数 $= \infty$
- $\infty \times \infty = \infty$
- $\infty - \infty =$ **NaN**
- $0 \times \infty =$ **NaN**

### NaN（Not a Number）

**表示**：阶码 = 全1，尾数 ≠ 0

```
NaN: S 11111111 MMMMMMMMMMMMMMMMMMMMMMM (M≠0)
```

**产生原因**：
- $\infty - \infty$
- $0 \times \infty$
- $\sqrt{-1}$
- $0 / 0$
- NaN 参与的任何运算

**特点**：
- NaN ≠ NaN（任何比较都为假，包括自身比较）
- NaN 参与运算结果仍为 NaN
- 用于标记无效运算

### 非规格化数（Denormalized Number）

**表示**：阶码 = 0，尾数 ≠ 0

**真值公式**：

$$N = (-1)^S \times 0.M \times 2^{1-Bias}$$

注意：没有隐含的前导1！使用 $0.M$ 而非 $1.M$。

**作用**：
- 提供**渐进下溢**（Gradual Underflow）
- 在零和最小规格化数之间填充数值
- 避免突然的精度丢失

**示例**（float）：
- 最小规格化正数：$1.0 \times 2^{-126}$
- 最大非规格化正数：$(1 - 2^{-23}) \times 2^{-126}$
- 最小非规格化正数：$2^{-23} \times 2^{-126} = 2^{-149}$

## 原理分析

### IEEE 754 值的完整分类

```
IEEE 754 32位浮点数
├── 阶码 = 0
│   ├── 尾数 = 0 → ±零
│   └── 尾数 ≠ 0 → 非规格化数（0.M × 2⁻¹²⁶）
├── 阶码 = 1~254
│   └── 规格化数（1.M × 2^(e-127)）
└── 阶码 = 255
    ├── 尾数 = 0 → ±∞
    └── 尾数 ≠ 0 → NaN
```

### 规格化数 vs 非规格化数

| 特征 | 规格化数 | 非规格化数 |
|------|---------|-----------|
| 阶码 | 1~254 | 0 |
| 尾数形式 | $1.M$（隐含1） | $0.M$（无隐含1） |
| 精度 | 满精度 | 精度递减 |
| 范围 | $2^{-126}$ 到 $2^{127}$ | $2^{-149}$ 到 $2^{-126}$ |

### float 的最小和最大值

- 最大有限值：$(2 - 2^{-23}) \times 2^{127} \approx 3.4028235 \times 10^{38}$
- 最小正规格化数：$2^{-126} \approx 1.17549435 \times 10^{-38}$
- 最小正非规格化数：$2^{-149} \approx 1.40129846 \times 10^{-45}$

## 直观理解

### 记忆要点

- **阶码全0**：零或非规格化数
- **阶码全1**：无穷大或NaN
- **NaN不等于自己**：特殊的"无效"标记
- **非规格化数没有隐含1**
- **±0不同但相等**

### 类比

- **零**：精确的"什么都没有"
- **非规格化数**：接近零但还不为零的"微小值"
- **无穷大**：超出范围的"无限大"
- **NaN**：计算出错的"无效结果"
