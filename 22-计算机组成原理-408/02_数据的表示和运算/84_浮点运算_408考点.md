# 84_浮点运算_408考点

## 核心概念

浮点运算是 408 考试的**综合大题**重点，涵盖加减乘除、规格化、舍入、溢出等全部知识点。

## 408高频考点

### 考点1：浮点加法完整过程（大题高频）

**典型格式**：给定两个浮点数的编码，要求完整写出五步过程。

**答题模板**：

```
已知：A = 0.1011 × 2^(010), B = 0.1101 × 2^(000)

(1) 对阶
    ΔE = |E_A - E_B| = 2
    小阶向大阶看齐：B 尾数右移2位，阶码加2
    对阶后：B = 0.0011 × 2^(010)

(2) 尾数加法
    0.1011 + 0.0011 = 0.1110

(3) 规格化
    0.1110 符号位0，最高数值位1 → 已规格化

(4) 舍入
    不需要

(5) 溢出判断
    阶码 010(+2) 在正常范围内

结果：A + B = 0.1110 × 2^(010)
```

### 考点2：阶码溢出判断

**典型问法**：浮点乘法中，阶码相加后如何判断溢出？

**答案**：
- 规格化数的阶码范围 $[E_{min}, E_{max}]$
- 阶码相加/相减后，检查结果是否在此范围内
- 超出 → 上溢；不足 → 下溢

### 考点3：尾数溢出与规格化的关系

**典型问法**：尾数加法溢出（绝对值 $\geq 1$）是阶码溢出吗？

**答案**：不是。尾数溢出通过**右规**解决（尾数右移，阶码加1）。只有阶码超出范围才是真正的溢出。

### 考点4：规格化后溢出判断

**典型问法**：右规后阶码超出了范围，属于什么情况？

**答案**：上溢。

### 考点5：浮点乘法中规格化方向

**典型问法**：两个规格化正尾数相乘，结果需要左规还是右规？

**答案**：最多需要**左移 1 位**。

原因：$[0.5, 1) \times [0.5, 1) = [0.25, 1)$，可能 $< 0.5$，需要左移 1 位。

### 考点6：浮点除法中规格化方向

**典型问法**：两个规格化正尾数相除，结果需要左规还是右规？

**答案**：最多需要**右移 1 位**。

原因：$[0.5, 1) / [0.5, 1) = (0.5, 2)$，可能 $\geq 1$，需要右移 1 位。

### 考点7：综合计算题

**典型大题**：给定格式和两个浮点数，求加法或乘法结果。

## 典型真题精选

### 真题1（大题）

设浮点数格式：阶码 4 位补码（含阶符），尾数 6 位补码（含数符），$R = 2$。

$A = 0.11010 \times 2^{0011}$（阶码 $0011 = +3$），$B = 0.10110 \times 2^{0001}$（阶码 $0001 = +1$）

用浮点加法求 $A + B$。

**解答**：

**（1）对阶**
$E_A = +3$，$E_B = +1$，$\Delta E = 2$
$B$ 尾数右移 2 位：$0.10110 \to 0.001011 \to 0.00110$（截断为 5 位尾数）
$B$ 阶码改为 $0011$

**（2）尾数加法**
$0.11010 + 0.00110 = 0.100000$（溢出进位）

实际上：$0.11010 + 0.00110 = 1.00000$（有进位到符号位前）

$= 0.10000$（5位尾数，溢出），需要右规。

**（3）规格化**
右移 1 位：$0.10000 \to 0.01000$（进位1处理为阶码调整）

实际处理：$1.00000$ 右移 → $0.10000$，阶码加 1：$+3 + 1 = +4$，阶码 $0100$

**（4）舍入**
无需

**（5）溢出判断**
阶码 $0100 = +4$，在 $[-8, +7]$ 范围内，无溢出。

结果：$A + B = 0.10000 \times 2^{0100}$

验证：$A = 0.8125 \times 8 = 6.5$，$B = 0.6875 \times 2 = 1.375$，$A + B = 7.875$

$0.10000 \times 2^4 = 0.5 \times 16 = 8$（有截断误差）

### 真题2（选择题）

浮点乘法中，阶码相加可能产生溢出。以下说法正确的是：

A. 只有上溢没有下溢
B. 既有上溢也有下溢
C. 不会产生溢出
D. 只在除法中产生溢出

**答案**：B

### 真题3（选择题）

两个规格化尾数相乘后，结果最多需要左移几位？

A. 0　　B. 1　　C. 2　　D. n

**答案**：B（最多左移 1 位）

### 真题4（判断题）

浮点加法中，对阶右移时使用的是逻辑右移。

**答案**：错误。对阶右移应使用**算术右移**（补码运算时高位补符号位）。

## 易错总结

1. **尾数溢出 $\neq$ 阶码溢出**：尾数溢出用右规解决
2. **右规后才判断阶码溢出**
3. **对阶用算术右移**
4. **乘法最多左规1位，除法最多右规1位**
5. **加法左规可能多次**
6. **舍入可能引起再次规格化**
