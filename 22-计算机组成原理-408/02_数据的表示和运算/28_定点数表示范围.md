# 28_定点数表示范围

## 核心概念

### n位定点整数的表示范围

#### 原码

$$-(2^{n-1} - 1) \leq X \leq +(2^{n-1} - 1)$$

- 最大正数：$0\underbrace{11\cdots1}_{n-1} = +(2^{n-1}-1)$
- 最小负数：$1\underbrace{11\cdots1}_{n-1} = -(2^{n-1}-1)$
- 编码总数：$2^n - 1$（因为+0和-0占两个编码表示同一个值）

#### 补码

$$-2^{n-1} \leq X \leq +(2^{n-1} - 1)$$

- 最大正数：$0\underbrace{11\cdots1}_{n-1} = +(2^{n-1}-1)$
- 最小负数：$1\underbrace{00\cdots0}_{n-1} = -2^{n-1}$
- 编码总数：$2^n$（所有编码都有效利用）

#### 反码

$$-(2^{n-1} - 1) \leq X \leq +(2^{n-1} - 1)$$

与原码相同。

### n位定点小数的表示范围

#### 原码

$$-(1 - 2^{-(n-1)}) \leq X \leq +(1 - 2^{-(n-1)})$$

#### 补码

$$-1 \leq X \leq +(1 - 2^{-(n-1)})$$

#### 反码

$$-(1 - 2^{-(n-1)}) \leq X \leq +(1 - 2^{-(n-1)})$$

---

## 原理分析

### 表示范围汇总表

| 编码 | 定点整数(n位) | 定点小数(n位) |
|------|-------------|-------------|
| 原码 | $-(2^{n-1}-1) \sim +(2^{n-1}-1)$ | $-(1-2^{-(n-1)}) \sim +(1-2^{-(n-1)})$ |
| 反码 | $-(2^{n-1}-1) \sim +(2^{n-1}-1)$ | $-(1-2^{-(n-1)}) \sim +(1-2^{-(n-1)})$ |
| 补码 | $-2^{n-1} \sim +(2^{n-1}-1)$ | $-1 \sim +(1-2^{-(n-1)})$ |
| 无符号 | $0 \sim (2^n-1)$ | — |

### 具体数值（8位）

| 编码 | 定点整数范围 | 定点小数范围 |
|------|------------|------------|
| 原码 | $-127 \sim +127$ | $-(1-2^{-7}) \sim +(1-2^{-7})$ |
| 反码 | $-127 \sim +127$ | 同上 |
| 补码 | $-128 \sim +127$ | $-1 \sim +(1-2^{-7})$ |
| 无符号 | $0 \sim 255$ | — |

### 关键公式记忆

$n$ 位补码：
- 最大正数：$2^{n-1} - 1$
- 最小负数：$-2^{n-1}$
- 可表示个数：$2^n$
- 比原码多表示一个数：$-2^{n-1}$

$n$ 位定点小数补码：
- 最大正数：$1 - 2^{-(n-1)}$
- 最小负数：$-1$

---

## 直观理解

### 数轴表示

8位补码定点整数的数轴：
```
-128          -1    0           +127
 |____________|_____|____________|
 10000000   11111111 00000000   01111111
```

8位原码定点整数的数轴：
```
-127          -0 +0           +127
 |____________|___|____________|
 11111111   10000000 00000000 01111111
```

### 记忆技巧
1. **补码比原码多一个数**：多的是最小负数 $-2^{n-1}$
2. **原码和反码范围对称**（除0之外）
3. **补码范围不对称**：负数比正数多一个（$-2^{n-1}$）
4. **无符号数没有负数**：全部用于表示正数

---

## 知识关联

### 408考试要点
- 给定 $n$ 位求各编码的表示范围
- 补码比原码多一个数的原因（只有一个0）
- 定点整数和定点小数范围的计算公式
- 溢出判断（结果超出表示范围）

### 易错陷阱
1. **8位补码范围**：$-128 \sim +127$（不是 $-128 \sim +128$）
2. **定点小数补码最小值**：$-1$（不是 $-1 + 2^{-(n-1)}$）
3. **原码没有 $-2^{n-1}$**：原码最小负数是 $-(2^{n-1}-1)$
4. **无符号数范围**：$0 \sim 2^n-1$（不是 $0 \sim 2^n$）
