# 58_除法_408考点

## 核心概念

### 408考试中除法的出题形式

- **选择题**：判断除法过程中的寄存器状态、商和余数的符号
- **大题**：给定被除数和除数，画出完整的加减交替法过程表格

## 408高频考点

### 考点1：原码除法过程表格

**典型大题**：用加减交替法计算 $0.1011 \div 0.1101$，画出运算过程表。

**标准解答格式**：

符号位：$s = 0 \oplus 0 = 0$

X = 01101（除数，5位含符号扩展）

| 轮次 | 操作类型 | 操作 | ACC | MQ（商） |
|------|---------|------|-----|---------|
| 初始 | — | — | 01011 | 0000 |
| 1 | 减 | ACC - X | 11110 | 0000 |
| 1 | — | 左移 | 11100 | 0000 |
| 2 | 加 | ACC + X | 01001 | 0001 |
| 2 | — | 左移 | 10010 | 0010 |
| 3 | 减 | ACC - X | 00101 | 0011 |
| 3 | — | 左移 | 01010 | 0110 |
| 4 | 减 | ACC - X | 11101 | 0110 |
| 4 | — | 左移 | 11010 | 1100 |
| 最后 | 恢复 | ACC + X | 00111 | 1100 |

结果：商 = 0.1100，余数需调整

### 考点2：商的符号

**典型问法**：$[x]_{原} = 1.1011$，$[y]_{原} = 0.1101$，则商的符号位为？

**解法**：$s = x_s \oplus y_s = 1 \oplus 0 = 1$（负数）

**易错点**：
- 商的符号用**异或**计算
- 余数的符号 = 被除数的符号（余数与被除数同号）

### 考点3：余数的符号规则

**重要规则**：
- **余数的符号与被除数相同**
- 若被除数为正，余数必须为正；若被除数为负，余数必须为负
- 这与数学中的"余数非负"约定不同！

**408陷阱**：问"余数的符号与什么相同？"
- 正确答案：与被除数相同
- 错误答案：与除数相同 / 与商相同

### 考点4：迭代次数

**典型问法**：n 位原码除法需要几轮迭代？

**答案**：
- 数值部分 n 位 → **n 轮**
- 每轮一次加法或减法
- 最多额外 1 次恢复余数操作

### 考点5：加减交替法的关键规则

**典型选择题**：以下关于加减交替法的说法，正确的是？

常见选项：
- "每轮都做减法" → **错误**，根据上轮符号决定加或减
- "上轮够减，本轮做减法" → **正确**
- "最后余数为负不需要恢复" → **错误**，需要恢复
- "商0时下轮做加法" → **正确**

### 考点6：恢复余数法 vs 加减交替法

**典型问法**：两种方法的区别？

| 对比项 | 恢复余数法 | 加减交替法 |
|--------|-----------|-----------|
| 不够减时 | 加回除数恢复 | 不恢复，下轮改加法 |
| 最坏情况加法次数 | 2n | n+1 |
| 硬件复杂度 | 较复杂 | 较简单 |
| 结果 | 相同 | 相同 |

## 典型真题精选

### 真题1（选择题）

原码除法中，余数的符号与什么相同？

A. 商　　B. 除数　　C. 被除数　　D. 固定为正

**答案**：C

**解析**：余数的符号与被除数相同。例如 $(-7) \div 3 = (-2)$ 余 $(-1)$，余数与被除数同号。

### 真题2（选择题）

用加减交替法进行 4 位原码除法，最多需要几次加/减法操作？

A. 4　　B. 5　　C. 8　　D. 3

**答案**：B（4次迭代操作 + 最多1次恢复 = 5次）

### 真题3（大题节选）

已知 $x = 0.1001$，$y = 0.1101$，用加减交替法求 $x \div y$。

**解答**：

符号位：$s = 0 \oplus 0 = 0$

X = 01101, ACC = 01001

| 轮次 | 操作 | ACC | 商 |
|------|------|-----|-----|
| 1 | ACC - X | 11100 | 0 |
| 1 | 左移 | 11000 | 0 |
| 2 | ACC + X | 00101 | 1 |
| 2 | 左移 | 01010 | 1 |
| 3 | ACC - X | 11101 | 0 |
| 3 | 左移 | 11010 | 0 |
| 4 | ACC + X | 00111 | 1 |
| 4 | — | 00111 | 1 |

商 = 0.1011，余数（需调整）= $0.00111 \times 2^{-4}$

### 真题4（选择题）

原码除法中，商的符号由什么决定？

A. 被除数符号　　B. 除数符号　　C. 被除数与除数符号异或　　D. 被除数与除数符号同或

**答案**：C

## 易错总结

1. **余数符号 = 被除数符号**，不是除数符号
2. **商符号 = 被除数符号 XOR 除数符号**
3. **加减交替法第一轮固定做减法**
4. **最后一轮余数为负需要恢复**
5. **除法是左移，不是右移**（与乘法相反）
6. **溢出判断**：定点小数除法中，若 $|$被除数$|$ ≥ $|$除数$|$，则商 ≥ 1，超出小数范围
