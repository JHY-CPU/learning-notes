# 16_CRC_计算详解

## 原理分析

### 模2除法规则

**模2运算**：加法 = 减法 = 异或（XOR），没有进位和借位。

**模2除法步骤**：
1. 从被除数最高位开始，取与除数相同位数
2. 最高位为1则商1，做异或；最高位为0则商0，异或000...0
3. 拉下一位，组成新的被除数部分（保持与除数等长）
4. 重复直到所有位处理完，最后 $r$ 位即余数

---

### 例题1：数据 $M = 101001$，$G(x) = 1101$

$r = 3$，补3个0：$M' = 101001000$

```
               111010  （商）
             __________
  1101 ) 101001000
         1101               （1010 ⊕ 1101，商1）
         ----
          1110              （余011，拉下0 → 1110）
          1101              （1110 ⊕ 1101，商1）
          ----
           0111             （余0011，拉下1 → 0111）
           0000             （0111最高位0，商0，⊕0000）
           ----
            1110            （余111，拉下0 → 1110）
            1101            （1110 ⊕ 1101，商1）
            ----
             0110           （余011，拉下0 → 0110）
             0000           （0110最高位0，商0，⊕0000）
             ----
              1100          （余110，拉下0 → 1100）
              1101          （1100 ⊕ 1101，商1）
              ----
               001          （余数 R = 001）
```

**余数 $R = 001$，CRC编码 = $101001\mathbf{001}$**

**验证**：$101001001 \div 1101$，余数应为0（留给读者练习）。

---

### 例题2：数据 $M = 1101011$，$G(x) = 1011$（$x^3+x+1$）

$r = 3$，补3个0：$M' = 1101011000$

```
              1110110
            ___________
  1011 ) 1101011000
         1011             （1101 ⊕ 1011，商1）
         ----
          1100            （余110，拉下0 → 1100）
          1011            （1100 ⊕ 1011，商1）
          ----
           1111           （余111，拉下1 → 1111）
           1011           （1111 ⊕ 1011，商1）
           ----
            1000          （余100，拉下0 → 1000）
            1011          （1000 ⊕ 1011，商1）
            ----
             0110         （余011，拉下0 → 0110）
             0000         （最高位0，商0）
             ----
              1100        （余110，拉下0 → 1100）
              1011        （1100 ⊕ 1011，商1）
              ----
               1110       （余111，拉下0 → 1110）
               1011       （1110 ⊕ 1011，商1）
               ----
                101       （余数 R = 101）
```

**CRC编码 = $1101011\mathbf{101}$**

### 例题3：数据 $M = 10111$，$G(x) = x^3 + 1 = 1001$

$r = 3$，补3个0：$M' = 10111000$

```
             10101
           ________
  1001 ) 10111000
         1001          （1011 ⊕ 1001，商1）
         ----
           0101        （余010，拉下1 → 0101）
           0000        （最高位0，商0）
           ----
            1010       （余101，拉下0 → 1010）
            1001       （1010 ⊕ 1001，商1）
            ----
              0110     （余011，拉下0 → 0110）
              0000     （最高位0，商0）
              ----
               1100    （余110，拉下0 → 1100）
               1001    （1100 ⊕ 1001，商1）
               ----
                 101   （余数 R = 101）
```

**CRC编码 = $10111\mathbf{101}$**

---

### 例题4：检错验证

**题目**：接收到数据 $10111101$，$G(x) = 1001$，判断是否有错。

用 $1001$ 对 $10111101$ 做模2除法：

```
             10101
           ________
  1001 ) 10111101
         1001          （商1）
         ----
           0101        （余010，拉下1 → 0101）
           0000        （商0）
           ----
            1011       （余101，拉下1 → 1011）
            1001       （商1）
            ----
              0100     （余010，拉下0 → 0100）
              0000     （商0）
              ----
               1001    （余100，拉下1 → 1001）
               1001    （商1）
               ----
                 000   （余数 = 000）
```

余数为000，**无错误** ✓

## 直观理解

### 计算技巧
1. **补零数** = 除数位数 - 1 = 生成多项式次数 $r$
2. **余数位数** = $r$ 位（不足 $r$ 位前面补0）
3. **CRC编码长度** = 数据长度 + $r$
4. 模2运算中只有XOR，手算比十进制除法简单

### 常见错误
- 异或运算写成减法导致借位（模2运算没有借位）
- 补零数量错误（应补 $r$ 个，不是 $r+1$ 个）
- 最终结果忘记拼接余数，或把商当成答案

## 知识关联

### 408答题模板
1. 确定 $r$（生成多项式次数）
2. 数据末尾补 $r$ 个0
3. 做模2除法（XOR），写出详细步骤
4. 取 $r$ 位余数
5. CRC编码 = 数据 + 余数
6. 验证：用CRC编码再做一次模2除法，余数应为0
