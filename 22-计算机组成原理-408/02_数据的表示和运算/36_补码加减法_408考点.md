# 37_补码加减法_408考点

## 核心概念

### 408大题高频考点
1. 给定 $x, y$（真值），用补码计算 $x + y$ 或 $x - y$
2. 判断运算是否溢出
3. 写出完整的编码和运算过程
4. 将结果转回真值

### 答题模板

1. **写补码**：写出两个操作数的补码表示
2. **减法处理**：如需减法，先求 $[-y]_{\text{补}}$（全部取反+1）
3. **二进制加法**：按位相加，符号位参与运算
4. **舍弃进位**：最高位进位丢弃
5. **判断溢出**：同号相加异号结果则溢出
6. **转真值**：将结果补码转回真值

## 典型考题

### 考题1：补码加法（408真题风格）

**题目**：设机器字长8位（含1位符号位），用补码计算 $x + y$，其中 $x = +46$，$y = +29$。

**解**：

$[x]_{\text{补}} = 00101110$（$46 = 32 + 8 + 4 + 2 = 00101110$）

$[y]_{\text{补}} = 00011101$（$29 = 16 + 8 + 4 + 1 = 00011101$）

```
  00101110
+ 00011101
----------
  01001011
```

结果 $01001011$（补码），符号位0，为正数。

$01001011 = 64 + 8 + 2 + 1 = +75$

$46 + 29 = 75$ ✓，无溢出。

### 考题2：补码减法

**题目**：用8位补码计算 $x - y$，其中 $x = +46$，$y = +29$。

**解**：

$[x]_{\text{补}} = 00101110$

$[y]_{\text{补}} = 00011101$

$[-y]_{\text{补}} = \overline{00011101} + 1 = 11100010 + 1 = 11100011$

```
  00101110
+ 11100011
----------
 100010001  （9位）
```

舍弃进位：$00010001 = +17$

$46 - 29 = 17$ ✓

### 考题3：有溢出的加法

**题目**：用8位补码计算 $x + y$，其中 $x = +100$，$y = +50$。

**解**：

$[x]_{\text{补}} = 01100100$

$[y]_{\text{补}} = 00110010$

```
  01100100
+ 00110010
----------
  10010110
```

结果符号位为1（负数），但两个正数相加不可能为负——**上溢**。

$100 + 50 = 150 > 127$（8位补码最大正数）。

### 考题4：负数运算

**题目**：用8位补码计算 $x + y$，其中 $x = -46$，$y = -29$。

**解**：

$[x]_{\text{补}} = 11010010$（$-46$ 的补码）

$[y]_{\text{补}} = 11100011$（$-29$ 的补码）

```
  11010010
+ 11100011
----------
 110110101  （9位）
```

舍弃进位：$10110101$（补码，符号位1，负数）

验证：取反+1 = `01001010` + 1 = `01001011` = 75，结果 = $-75$。

$-46 + (-29) = -75$ ✓

$-75 \geq -128$，未超出范围，无溢出。

## 易错陷阱

### 陷阱1：二进制加法 vs 十进制加法
- 不要先转十进制计算再转回二进制
- 必须直接做二进制补码加法
- 408大题需要写出二进制运算过程

### 陷阱2：$[-y]_{\text{补}}$ 的求法
- **全部位**取反+1（包括符号位）
- 不要只取反数值位

### 陷阱3：溢出不等于进位
- 最高位有进位 ≠ 溢出
- 溢出 = 同号相加结果异号
- 异号相加的进位是正常现象

### 陷阱4：结果的解读
- 结果仍然是补码形式
- 如果符号位为1，需要取反+1才能得到真值的绝对值
- 不要直接把补码当作真值
