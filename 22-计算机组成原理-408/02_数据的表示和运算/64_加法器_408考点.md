# 64_加法器_408考点

## 核心概念

### 408考试中加法器的出题形式

- **选择题**：延迟分析、超前进位原理、$G_i$/$P_i$ 定义
- **大题**：计算给定输入下的各进位和和、写出 $C_i$ 展开式

## 408高频考点

### 考点1：全加器表达式

**必记公式**：

$$S_i = A_i \oplus B_i \oplus C_{i-1}$$

$$C_i = A_i B_i + (A_i \oplus B_i) C_{i-1}$$

**典型问法**：以下哪个是全加器进位的正确表达式？

### 考点2：进位产生和进位传递

**必记**：
- $G_i = A_i \cdot B_i$（产生：两个都是1，必然有进位）
- $P_i = A_i \oplus B_i$（传递：恰好一个1时，传递进位）
- $C_i = G_i + P_i \cdot C_{i-1}$

**典型问法**：给定 $A_i = 1, B_i = 1$，问 $G_i$ 和 $P_i$ 的值。

$G_i = 1, P_i = 0$（两个1必然产生进位，没有传递）

### 考点3：延迟分析

**串行进位延迟**：$n$ 位 = $n$ 级全加器延迟

**超前进位延迟**：$n$ 位 ≈ $\lceil \log_2 n \rceil + 3$ 级门延迟

**典型问法**：
- 32 位串行进位加法器的延迟是多少？ → $32 t_{FA}$
- 4 位超前进位加法器的延迟？ → 约 4-5 级门延迟

### 考点4：4位超前进位展开式

**必记** $C_0$ 到 $C_3$ 的展开式（见第62节完整推导）

**典型考法**：写出 $C_2$ 的展开式。

$$C_2 = G_2 + P_2 G_1 + P_2 P_1 G_0 + P_2 P_1 P_0 C_{in}$$

### 考点5：CLA 分组

**典型问法**：
- 16 位 CLA 分为 4 组，每组 4 位，组间进位如何计算？
- $BG_k$ 和 $BP_k$ 的定义？

### 考点6：加法器类型对比

| 类型 | 延迟 | 硬件 | 适用场景 |
|------|------|------|---------|
| 串行进位 | $O(n)$ | 简单 | 低速、低成本 |
| 超前进位 | $O(\log n)$ | 复杂 | 高速 |
| CLA分组 | $O(\log n)$ | 中等 | 实用方案 |
| 选择进位 | $O(\sqrt{n})$ | 中等 | 中速 |

### 考点7：计算给定输入的各值

**典型大题**：$A = 1101$，$B = 1011$，$C_{in} = 0$，用4位超前进位求 $S$ 和 $C_{out}$。

**解答过程**：

第一步：计算 $G_i, P_i$
- $G_0 = 1 \cdot 1 = 1, P_0 = 1 \oplus 1 = 0$
- $G_1 = 0 \cdot 1 = 0, P_1 = 0 \oplus 1 = 1$
- $G_2 = 1 \cdot 0 = 0, P_2 = 1 \oplus 0 = 1$
- $G_3 = 1 \cdot 1 = 1, P_3 = 1 \oplus 1 = 0$

第二步：计算 $C_i$
- $C_0 = G_0 + P_0 C_{in} = 1 + 0 = 1$
- $C_1 = G_1 + P_1 G_0 + P_1 P_0 C_{in} = 0 + 1 + 0 = 1$
- $C_2 = G_2 + P_2 G_1 + P_2 P_1 G_0 + P_2 P_1 P_0 C_{in} = 0 + 0 + 1 + 0 = 1$
- $C_3 = G_3 + P_3 G_2 + P_3 P_2 G_1 + P_3 P_2 P_1 G_0 + 0 = 1 + 0 + 0 + 0 = 1$

第三步：计算 $S_i$
- $S_0 = P_0 \oplus C_{in} = 0 \oplus 0 = 0$
- $S_1 = P_1 \oplus C_0 = 1 \oplus 1 = 0$
- $S_2 = P_2 \oplus C_1 = 1 \oplus 1 = 0$
- $S_3 = P_3 \oplus C_2 = 0 \oplus 1 = 1$

结果：$S = 1000, C_{out} = 1$

验证：$1101 + 1011 = 11000$（$13 + 11 = 24$）✓

## 典型真题精选

### 真题1（选择题）

4位串行进位加法器中，$S_3$ 的输出需要等待几个全加器的延迟？

A. 1　　B. 2　　C. 3　　D. 4

**答案**：D

**解析**：$S_3$ 依赖 $C_2$，$C_2$ 依赖 $C_1$，$C_1$ 依赖 $C_0$，所以需要等 4 级全加器延迟。

### 真题2（选择题）

超前进位中，$P_i = A_i \oplus B_i$ 的含义是？

A. 必然产生进位
B. 必然不产生进位
C. 可能传递进位
D. 必然传递进位

**答案**：C

**解析**：$P_i = 1$ 时，若低位有进位则传递，不是"必然"传递。

### 真题3（大题节选）

写出4位超前进位中 $C_2$ 的表达式，并说明其物理意义。

**答案**：

$$C_2 = G_2 + P_2 G_1 + P_2 P_1 G_0 + P_2 P_1 P_0 C_{in}$$

物理意义：
- 第2位产生进位（$G_2 = 1$），或
- 第1位产生且第2位传递（$P_2 G_1$），或
- 第0位产生且第1、2位传递（$P_2 P_1 G_0$），或
- 输入进位且第0、1、2位全部传递（$P_2 P_1 P_0 C_{in}$）

### 真题4（选择题）

n 位超前进位加法器的门延迟约为？

A. $O(n)$　　B. $O(n^2)$　　C. $O(\log n)$　　D. $O(n \log n)$

**答案**：C

## 易错总结

1. **$P_i$ 的定义**：唐朔飞版用 $A_i \oplus B_i$，部分教材用 $A_i + B_i$
2. **$C_i$ 展开式中的项数**：$C_i$ 有 $i+2$ 项（含 $C_{in}$ 项）
3. **延迟的计算**：串行是 $n \times t_{FA}$，超前是 $O(\log n)$ 级门延迟
4. **$S_i$ 用的是 $C_{i-1}$ 不是 $C_i$**
5. **分组 CLA 的 $BG$ 和 $BP$ 的计算**
