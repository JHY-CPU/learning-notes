# 67_浮点数表示范围

## 核心概念

### 浮点数表示范围的四个极值

浮点数 $N = M \times R^E$，其中：
- 尾数 $M$ 为规格化纯小数：$1/R \leq |M| < 1$（原码规格化时 $1/2 \leq |M| < 1$）
- 阶码 $E$ 范围：$E_{min} \leq E \leq E_{max}$

### 最大正数

$$N_{max+} = M_{max} \times R^{E_{max}}$$

- $M_{max} = (1 - 2^{-m})$（尾数最大值，m 位尾数）
- $E_{max}$：阶码最大正值

**示例**：阶码 4 位补码（含阶符），尾数 12 位原码（含数符），$R = 2$
- $E_{max} = +7$（补码 0,111）
- $M_{max} = +(1 - 2^{-11})$
- $N_{max+} = +(1 - 2^{-11}) \times 2^7$

### 最小正数

$$N_{min+} = M_{min+} \times R^{E_{min}}$$

- $M_{min+} = +2^{-m}$（规格化后尾数最小正值，即 $0.100...0$）
- $E_{min}$：阶码最小值

**示例**：同上格式
- $E_{min} = -8$（补码 1,000）
- $M_{min+} = +2^{-11}$
- $N_{min+} = +2^{-11} \times 2^{-8} = 2^{-19}$

### 最大负数

$$N_{max-} = M_{max-} \times R^{E_{min}}$$

- $M_{max-} = -2^{-m}$（规格化尾数的最大负数）
- $E_{min}$：阶码最小值

**示例**：$N_{max-} = -2^{-11} \times 2^{-8} = -2^{-19}$

### 最小负数

$$N_{min-} = M_{min-} \times R^{E_{max}}$$

- $M_{min-} = -(1 - 2^{-m})$
- $E_{max}$：阶码最大正值

**示例**：$N_{min-} = -(1 - 2^{-11}) \times 2^7$

### 表示范围示意图

```
      N_min-         N_max-    0    N_min+         N_max+
  ←──────────────────────┤    │    ├──────────────────────→
  负数区                    ↑↑↑    正数区
                      （不可表示）
```

### 上溢与下溢

**上溢（Overflow）**：
- $|N| > |N_{max}|$ 时发生
- 数太大，超出表示范围
- 需要中断处理

**下溢（Underflow）**：
- $0 < |N| < |N_{min}|$ 时发生
- 数太小，可以用 0 近似或非规格化数表示

## 原理分析

### 详细计算公式

设浮点数格式：阶码 $n$ 位（补码，含阶符），尾数 $m$ 位（原码，含数符），$R = 2$

| 极值 | 阶码 | 尾数 | 真值 |
|------|------|------|------|
| 最大正数 | $2^{n-1}-1$ | $+(1-2^{-(m-1)})$ | $+(1-2^{-(m-1)}) \times 2^{2^{n-1}-1}$ |
| 最小正数 | $-2^{n-1}$ | $+2^{-(m-1)}$ | $+2^{-(m-1)} \times 2^{-2^{n-1}}$ |
| 最大负数 | $-2^{n-1}$ | $-2^{-(m-1)}$ | $-2^{-(m-1)} \times 2^{-2^{n-1}}$ |
| 最小负数 | $2^{n-1}-1$ | $-(1-2^{-(m-1)})$ | $-(1-2^{-(m-1)}) \times 2^{2^{n-1}-1}$ |

> 注：具体数值取决于阶码是补码还是移码，以及尾数是否含数符位。

## 直观理解

### 记忆技巧

- **最大正数 = 最大尾数 × 最大阶码**
- **最小正数 = 最小正尾数 × 最小阶码**
- **最大/最小负数与正数对称（符号相反）**
- **上溢 = 太大，下溢 = 太小**

### 常见陷阱

- 尾数的"最小正值"是规格化后的，不是 $2^{-m}$ 而是 $2^{-(m-1)}$
- 阶码范围：补码含阶符时，负的范围比正的多1

## 知识关联

- 与 IEEE 754 的范围计算方法一致
- 上溢/下溢在浮点运算中需要特殊处理
- 408考试中常以计算题出现
