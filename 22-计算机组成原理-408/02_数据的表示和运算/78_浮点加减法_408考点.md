# 79_浮点加减法_408考点

## 核心概念

浮点加减法是 408 考试的**大题高频考点**，要求能完整写出五步过程。

## 408高频考点

### 考点1：五步流程（必考）

**典型大题**：给定两个浮点数的编码，要求用浮点加法求和，写出每步过程。

**标准解答格式**：

```
已知 A = 0.1011 × 2^010, B = 0.1101 × 2^000

（1）对阶
    E_A = 010(+2), E_B = 000(0)
    ΔE = 2 - 0 = 2
    B 尾数右移2位：0.1101 → 0.0011
    B 阶码改为 010

（2）尾数加法
    0.1011 + 0.0011 = 0.1110

（3）规格化
    0.1110 符号位0，最高数值位1，已规格化

（4）舍入
    无需舍入

（5）溢出判断
    阶码010(+2)在范围内，无溢出

结果：A + B = 0.1110 × 2^010
```

### 考点2：对阶方向

**典型问法**：对阶时，是小阶向大阶看齐还是大阶向小阶看齐？

**答案**：小阶向大阶看齐

**原因**：
- 大阶向小阶看齐 = 阶码减小，尾数左移 → 可能丢失高位（严重错误）
- 小阶向大阶看齐 = 阶码增大，尾数右移 → 只丢低位（精度损失，但不会出错）

### 考点3：规格化方向

| 情况 | 操作 | 阶码变化 |
|------|------|---------|
| 左规 | 尾数左移 | 阶码减小 |
| 右规 | 尾数右移 | 阶码增大 |

**典型问法**：浮点加法后尾数溢出，需要左规还是右规？

**答案**：右规（尾数绝对值超过1时右移）

### 考点4：舍入方法

**0舍1入**：
- 右移出去的最高位为 0：直接丢弃
- 右移出去的最高位为 1：尾数末位加 1

**典型问法**：对阶右移 2 位，移出位为 10，用 0 舍 1 入后如何处理？

答：最低移出位为 0，直接舍去。

### 考点5：溢出判断

**典型问法**：浮点加法什么情况下产生上溢/下溢？

- **上溢**：规格化后阶码 > 最大阶码
- **下溢**：规格化后阶码 < 最小阶码

### 考点6：左规可能多次，右规最多一次

这是高频判断题/选择题。

**原因**：
- 加法溢出最多产生一个进位 → 右规最多一次
- 两个相近数相减可能产生多个前导零 → 左规可能多次

## 典型真题精选

### 真题1（大题）

设浮点数的阶码 3 位（补码含阶符），尾数 5 位（补码含数符），$R = 2$。

$A = 0.1011 \times 2^{001}$，$B = (-0.1101) \times 2^{010}$

用浮点加法求 $A + B$。

**解答**：

$B$ 的尾数补码：$1.0011$（$-0.1101$ 的补码）

**（1）对阶**
$E_A = 001(+1)$，$E_B = 010(+2)$
$\Delta E = 2 - 1 = 1$
$A$ 尾数右移 1 位：$0.1011 \to 0.0101$（舍入后 $0.0101$）
$A$ 阶码改为 $010$

**（2）尾数加法**
$0.0101 + 1.0011 = 1.1000$

**（3）规格化**
$1.1000$：符号位 1，最高数值位 1，同号，未规格化！
左移 1 位：$1.1000 \to 1.0000$
阶码减 1：$010 \to 001$（$+1$）

**（4）舍入**
无

**（5）溢出判断**
阶码 $001$ 在范围内

结果：$A + B = 1.0000 \times 2^{001}$（即 $-1 \times 2^1 = -2$）

验证：$A = 0.1011 \times 2 = 0.6875 \times 2 = 1.375$
$B = -0.1101 \times 4 = -0.8125 \times 4 = -3.25$
$A + B = -1.875$

$1.0000$（补码）$= -1$，$-1 \times 2^1 = -2$... 有误差，来自对阶截断。

### 真题2（选择题）

浮点加法中，对阶操作的结果是：

A. 尾数右移，阶码减小
B. 尾数左移，阶码增大
C. 尾数右移，阶码增大
D. 尾数左移，阶码减小

**答案**：C

### 真题3（选择题）

浮点加法后，尾数溢出需要：

A. 左规　　B. 右规　　C. 舍入　　D. 重新对阶

**答案**：B

## 易错总结

1. **对阶方向**：小阶向大阶，不是大阶向小阶
2. **对阶是右移不是左移**
3. **左规是阶码减，右规是阶码加**
4. **补码右移高位补符号位**
5. **舍入可能引起再次溢出**
6. **浮点减法 = 对阶后尾数做减法**
