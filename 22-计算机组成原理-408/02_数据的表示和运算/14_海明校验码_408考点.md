# 14_海明校验码_408考点

## 核心概念

### 408常考题型
1. 给定数据位数，求最少需要几位校验位
2. 给定数据，写出完整的海明码编码过程
3. 给定接收码，进行检错和纠错
4. 海明码的最小码距与检错/纠错能力关系

### 最小码距

**最小码距**（Minimum Hamming Distance）：任意两个合法编码之间的最小不同位数。

海明码的最小码距 $d_{\min} = 3$，这保证了：
- 检测 $e$ 位错误：$d_{\min} \geq e + 1$
- 纠正 $t$ 位错误：$d_{\min} \geq 2t + 1$

海明码（$d_{\min} = 3$）：
- 能检测最多 $d_{\min} - 1 = 2$ 位错误
- 能纠正最多 $\lfloor(d_{\min} - 1)/2\rfloor = 1$ 位错误

## 典型考题

### 考题1：求校验位数量

**题目**：信息位8位，采用海明校验，至少需要几位校验位？

**解**：设校验位 $r$ 位，满足 $2^r \geq 8 + r + 1$：
- $r = 3$：$8 \geq 12$？不成立
- $r = 4$：$16 \geq 13$？成立 ✓

**答案**：至少需要4位校验位。

### 考题2：给数据求海明码

**题目**：数据 $1010$，求海明码（偶校验）。

**分析**：$n = 4$，$r = 3$，$2^3 = 8 \geq 8$ 成立。

| 位置 | 1($P_1$) | 2($P_2$) | 3($D_1$) | 4($P_3$) | 5($D_2$) | 6($D_3$) | 7($D_4$) |
|------|-----------|-----------|-----------|-----------|-----------|-----------|-----------|
| 值 | ? | ? | 1 | ? | 0 | 1 | 0 |

- $P_1$ 覆盖 1,3,5,7：$1 \oplus 0 \oplus 0 = 1$，所以 $P_1 = 1$
- $P_2$ 覆盖 2,3,6,7：$1 \oplus 1 \oplus 0 = 0$，所以 $P_2 = 0$
- $P_3$ 覆盖 4,5,6,7：$0 \oplus 1 \oplus 0 = 1$，所以 $P_3 = 1$

**答案**：$1011010$

### 考题3：检错纠错

**题目**：接收到海明码 $1011110$，采用偶校验，判断是否有错，若有错则纠正。

**解**：计算校验子：
- $S_1 = P_1 \oplus D_1 \oplus D_2 \oplus D_4 = 1 \oplus 1 \oplus 1 \oplus 0 = 1$
- $S_2 = P_2 \oplus D_1 \oplus D_3 \oplus D_4 = 0 \oplus 1 \oplus 1 \oplus 0 = 0$
- $S_3 = P_3 \oplus D_2 \oplus D_3 \oplus D_4 = 1 \oplus 1 \oplus 1 \oplus 0 = 1$

校验子 $S_3 S_2 S_1 = 101_2 = 5$。

**位置5出错**，将位置5取反：$1 \to 0$。

纠正后：$1011\mathbf{0}10$。

### 考题4：最小码距与检错能力

**题目**：海明码最小码距为3，下列说法正确的是（）

A. 能纠正1位错误
B. 能检测3位错误
C. 能同时纠正1位错误和检测1位错误
D. 能纠正2位错误

**分析**：
- A：$d_{\min} \geq 2 \times 1 + 1 = 3$，能纠正1位 ✓
- B：$d_{\min} \geq e + 1$，$3 \geq e + 1$，$e \leq 2$，最多检测2位 ✗
- C：$d_{\min} \geq t + e + 1$，$3 \geq 1 + 1 + 1 = 3$，成立 ✓
- D：$d_{\min} \geq 2 \times 2 + 1 = 5$，$3 < 5$，不能 ✗

**答案**：A、C

### 考题5：编码效率

**题目**：8位数据采用海明校验，编码效率是多少？

**解**：8位数据需要4位校验位（$2^4 = 16 \geq 8 + 4 + 1 = 13$）。

$$\text{编码效率} = \frac{8}{8+4} = \frac{8}{12} = 66.7\%$$

## 易错陷阱

### 陷阱1：公式变形
- $2^r \geq n + r + 1$ 与 $2^r - 1 \geq n + r$ 等价
- 注意不要把 $n$ 和 $r$ 的关系搞反

### 陷阱2：位置编号从1开始
- 位置编号是1, 2, 3, ..., 不是0, 1, 2, ...
- 二进制表示时，位置1 = 001, 位置2 = 010, 位置3 = 011, ...

### 陷阱3：校验子全0表示无错
- 校验子 $S_3 S_2 S_1 = 000$ 表示没有错误
- 不要误认为000表示位置0出错（位置从1开始编号）

### 陷阱4：覆盖范围包含校验位自身
- 每个校验位在计算自身值时，其自身不参与异或
- 但检错时，校验位参与校验子的计算

### 陷阱5：偶校验 vs 奇校验
- 偶校验：覆盖范围内所有位异或 = 0 为正确
- 奇校验：覆盖范围内所有位异或 = 1 为正确
- 408考试通常指定偶校验，但要留意题目说明
