# 63_先行进位加法器

## 核心概念

### CLA（Carry Lookahead Adder）的定义

**先行进位加法器**（CLA）是一种通过并行计算进位来提高加法速度的加法器。它在超前进位的基础上，采用**分组**策略，兼顾速度和硬件复杂度。

### 为什么需要分组？

直接的超前进位加法器，当位数很大时：
- $C_i$ 的表达式项数 = $i + 1$，急剧增长
- 高位需要的门扇入（fan-in）太大
- 硬件实现困难

**分组策略**：将 n 位分成若干组，组内用超前进位，组间也用超前进位。

### 两级分组 CLA（以16位为例）

**第一级**：将 16 位分为 4 组，每组 4 位
```
组0: 位0-3    组1: 位4-7    组2: 位8-11    组3: 位12-15
```

**第二级**：组间也用超前进位

### 组进位产生和组进位传递

对于第 $k$ 组（设每组 4 位），定义：

**组进位产生（Block Generate）**：
$$BG_k = G_{4k+3} + P_{4k+3} G_{4k+2} + P_{4k+3} P_{4k+2} G_{4k+1} + P_{4k+3} P_{4k+2} P_{4k+1} G_{4k}$$

**组进位传递（Block Propagate）**：
$$BP_k = P_{4k+3} \cdot P_{4k+2} \cdot P_{4k+1} \cdot P_{4k}$$

含义：
- $BG_k$：第 $k$ 组**自身能产生**进位（不需要等待外部进位）
- $BP_k$：第 $k$ 组**能传递**进位（4 位全部是传递状态时才传递）

### 组间超前进位

用 $BG_k$ 和 $BP_k$ 类似于单个 $G_i$ 和 $P_i$，组间进位：

$$C_3 = BG_0 + BP_0 \cdot C_{in}$$

$$C_7 = BG_1 + BP_1 \cdot BG_0 + BP_1 \cdot BP_0 \cdot C_{in}$$

$$C_{11} = BG_2 + BP_2 \cdot BG_1 + BP_2 \cdot BP_1 \cdot BG_0 + BP_2 \cdot BP_1 \cdot BP_0 \cdot C_{in}$$

$$C_{15} = BG_3 + BP_3 \cdot BG_2 + BP_3 \cdot BP_2 \cdot BG_1 + BP_3 \cdot BP_2 \cdot BP_1 \cdot BG_0 + BP_3 \cdot BP_2 \cdot BP_1 \cdot BP_0 \cdot C_{in}$$

### CLA 的完整推导（4位组为例）

**组内**（以组0为例，位0-3）：

$$G_0 = A_0 B_0, \quad P_0 = A_0 \oplus B_0$$
$$G_1 = A_1 B_1, \quad P_1 = A_1 \oplus B_1$$
$$G_2 = A_2 B_2, \quad P_2 = A_2 \oplus B_2$$
$$G_3 = A_3 B_3, \quad P_3 = A_3 \oplus B_3$$

组内进位：
$$C_0 = G_0 + P_0 C_{in}$$
$$C_1 = G_1 + P_1 G_0 + P_1 P_0 C_{in}$$
$$C_2 = G_2 + P_2 G_1 + P_2 P_1 G_0 + P_2 P_1 P_0 C_{in}$$
$$C_3 = G_3 + P_3 G_2 + P_3 P_2 G_1 + P_3 P_2 P_1 G_0 + P_3 P_2 P_1 P_0 C_{in}$$

组进位：
$$BG_0 = G_3 + P_3 G_2 + P_3 P_2 G_1 + P_3 P_2 P_1 G_0$$
$$BP_0 = P_3 P_2 P_1 P_0$$
$$C_3 = BG_0 + BP_0 \cdot C_{in}$$

组内和：
$$S_i = P_i \oplus C_{i-1}$$

**组间**：

得到 $C_3$ 后，组1 内部的 $C_4, C_5, C_6, C_7$ 可以用组1 的 $G, P$ 和 $C_3$ 计算。

### 延迟分析

对于 16 位两级 CLA（4 组，每组 4 位）：
- 产生所有 $G_i, P_i$：1 级延迟
- 组内 $BG_k, BP_k$：2 级延迟
- 组间超前进位：2 级延迟
- 组内各进位：2 级延迟
- 和 $S_i$：1 级延迟

**总延迟**：约 5-6 级门延迟，远小于串行进位的 16 级。

## 原理分析

### 4位 CLA 的电路框图

```
输入: A₃A₂A₁A₀, B₃B₂B₁B₀, C_in

第一层: 产生 Gᵢ, Pᵢ (i=0,1,2,3)
  Gᵢ = Aᵢ·Bᵢ,  Pᵢ = Aᵢ⊕Bᵢ

第二层: 超前进位逻辑
  C₀ = G₀ + P₀·C_in
  C₁ = G₁ + P₁·G₀ + P₁·P₀·C_in
  C₂ = G₂ + P₂·G₁ + P₂·P₁·G₀ + P₂·P₁·P₀·C_in
  C₃ = G₃ + P₃·G₂ + P₃·P₂·G₁ + P₃·P₂·P₁·G₀ + P₃·P₂·P₁·P₀·C_in

第三层: 产生和
  Sᵢ = Pᵢ ⊕ Cᵢ₋₁

输出: S₃S₂S₁S₀, C_out = C₃, BG, BP
```

### 多级分组的扩展

- **三级 CLA**：位数更多时，再加一层组间超前进位
- **64位 CLA**：可以分为 4 大组，每大组 4 小组，每小组 4 位（三级分组）

## 直观理解

### 类比

分组 CLA 就像"公司层级管理"：
- 组内超前进位 = 组内员工同时汇报给组长
- 组间超前进位 = 组长同时汇报给经理
- 不用逐级等，而是层级并行

### 记忆要点

- **$BG$ = 组进位产生**，$BP$ = 组进位传递
- **$BP = P_3 \cdot P_2 \cdot P_1 \cdot P_0$**（四个传递的AND）
- **两级 CLA 兼顾速度和硬件**
- **延迟 $O(\log n)$，硬件 $O(n \log n)$**

## 知识关联

- 超前进位（第62节）是 CLA 的理论基础
- 串行进位（第61节）是 CLA 的对比对象
- CLA 是现代处理器中 ALU 的标准实现方式

## 408考试要点

### 常考题型

1. 给定 $A$ 和 $B$，计算各 $G_i, P_i, C_i, S_i$
2. 分析 CLA 的延迟
3. 写出 $C_3$ 的完整展开式

### 典型例题

**题目**：用 4 位 CLA 计算 $A = 1011$，$B = 0111$，$C_{in} = 0$。

**解答**：

$G_0 = 1 \cdot 1 = 1$，$P_0 = 1 \oplus 1 = 0$
$G_1 = 1 \cdot 1 = 1$，$P_1 = 1 \oplus 1 = 0$
$G_2 = 0 \cdot 1 = 0$，$P_2 = 0 \oplus 1 = 1$
$G_3 = 1 \cdot 0 = 0$，$P_3 = 1 \oplus 0 = 1$

$C_0 = G_0 + P_0 \cdot 0 = 1 + 0 = 1$
$C_1 = G_1 + P_1 G_0 + P_1 P_0 \cdot 0 = 1 + 0 + 0 = 1$
$C_2 = G_2 + P_2 G_1 + P_2 P_1 G_0 + P_2 P_1 P_0 \cdot 0 = 0 + 1 + 0 + 0 = 1$
$C_3 = G_3 + P_3 G_2 + P_3 P_2 G_1 + P_3 P_2 P_1 G_0 + 0 = 0 + 0 + 1 + 0 = 1$

$S_0 = P_0 \oplus 0 = 0 \oplus 0 = 0$
$S_1 = P_1 \oplus C_0 = 0 \oplus 1 = 1$
$S_2 = P_2 \oplus C_1 = 1 \oplus 1 = 0$
$S_3 = P_3 \oplus C_2 = 1 \oplus 1 = 0$

结果：$S = 0010$，$C_{out} = C_3 = 1$

验证：$1011 + 0111 = 10010$（$11 + 7 = 18$）✓
