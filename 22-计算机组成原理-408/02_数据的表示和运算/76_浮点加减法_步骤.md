# 77_浮点加减法_步骤

## 核心概念

### 浮点加减法的五个步骤

浮点加减法是 408 考试的**必考大题**，需要掌握完整的五步流程。

$$A = M_A \times 2^{E_A}, \quad B = M_B \times 2^{E_B}$$

求 $A \pm B$：

### 第一步：对阶（Exponent Alignment）

**目标**：使两个浮点数的阶码相同

**方法**：**小阶向大阶看齐**

1. 比较 $E_A$ 和 $E_B$，找到较大的阶 $E_{max} = \max(E_A, E_B)$
2. 将较小阶的尾数**右移** $|E_A - E_B|$ 位
3. 将较小阶的阶码改为 $E_{max}$

**示例**：
$$A = 0.1011 \times 2^3, \quad B = 0.1100 \times 2^1$$

对阶：$B$ 的阶码小，右移 2 位
$$B = 0.001100 \times 2^3$$

**为什么右移？** 右移等价于除以 2，保持数值不变：
$$0.1100 \times 2^1 = 0.0011 \times 2^3$$

**注意**：对阶可能丢失精度（低位被移出）

### 第二步：尾数加减

**目标**：对齐后，执行尾数的加法或减法

$$M_{结果} = M_A \pm M_B$$

**注意**：
- 加减法用**补码**运算
- 如果结果的绝对值 $\geq 1$（溢出），需要右规

### 第三步：规格化

**目标**：将尾数调整为规格化形式

**左规**：尾数最高数值位为 0
- 尾数左移，阶码减 1
- 可能需要多次

**右规**：尾数溢出（绝对值 $\geq 1$）
- 尾数右移，阶码加 1
- 最多一次

### 第四步：舍入（Rounding）

**目标**：右移或截断后处理低位

**常见舍入方法**：
- **0舍1入**：类似于十进制的四舍五入
- **恒舍入**：直接截断
- **恒置1**：最低位强制置1
- **就近舍入**：IEEE 754 默认方式

**0舍1入**：
- 右移出的最高位为 0：直接舍去
- 右移出的最高位为 1：尾数加 1（可能再次溢出）

### 第五步：溢出判断

**目标**：检查阶码是否超出表示范围

**上溢**：阶码 $> E_{max}$，结果太大，需中断处理
**下溢**：阶码 $< E_{min}$，结果太小，当作 0 处理

### 完整流程图

```
开始
  │
对阶（小阶向大阶对齐）
  │
尾数加减
  │
规格化
  ├─ 需要右规？ → 右移，阶码加1 → 舍入 → 可能再规格化
  │
  ├─ 需要左规？ → 左移，阶码减1 → 可能多次
  │
  └─ 已规格化？ → 继续
  │
溢出判断
  ├─ 阶码上溢？ → 中断/置∞
  ├─ 阶码下溢？ → 置0
  └─ 正常 → 输出结果
  │
结束
```

## 直观理解

### 类比

浮点加法就像"科学计数法的加法"：
1. 对齐指数（对阶）
2. 有效数字相加（尾数加减）
3. 调整为标准形式（规格化）
4. 处理多余位（舍入）
5. 检查是否超出范围（溢出判断）

### 记忆口诀

- **"对阶、加减、规、舍、溢"**（五步顺序）
- **"小阶向大阶看齐"**（对阶方向）
- **"左规阶减，右规阶加"**（规格化方向）
- **"0舍1入"**（常用舍入方法）

## 代码/模拟

### Python模拟浮点加减法五步骤

```python
"""浮点加减法模拟 - 适用于408考研复习"""

def float_add_sub(x, y, op='+', mantissa_bits=6):
    """
    模拟浮点加减法的五个步骤
    :param x, y: 浮点数 (阶码, 尾数), 如 (3, 0.10110) 表示 0.10110×2^3
    :param op: '+' 或 '-'
    :param mantissa_bits: 尾数位数
    """
    ex, mx = x
    ey, my = y

    # 如果是减法, 尾数取反
    if op == '-':
        my = -my

    print(f"操作: {mx}×2^{ex} {op} ({abs(my)}×2^{ey})")
    print(f"{'='*50}")

    # === 第一步: 对阶 ===
    print(f"\n【步骤1: 对阶】")
    if ex != ey:
        small_e, big_e = (ex, ey) if ex < ey else (ey, ex)
        small_m, big_m = (mx, my) if ex < ey else (my, mx)
        diff = big_e - small_m if False else big_e - small_e
        # 小阶向大阶看齐: 尾数右移diff位, 阶码增大到大阶
        shifted_m = small_m / (2 ** diff)
        result_e = big_e
        print(f"  小阶{small_e}向大阶{big_e}对齐: 右移{diff}位")
        print(f"  对阶后: ({shifted_m:.{mantissa_bits}f}×2^{result_e}, {big_m}×2^{big_e})")
    else:
        shifted_m = mx
        big_m = my
        result_e = ex
        diff = 0
        print(f"  阶码相同({ex}), 无需对阶")

    # === 第二步: 尾数加减 ===
    print(f"\n【步骤2: 尾数加减】")
    result_m = shifted_m + big_m
    print(f"  {shifted_m:.{mantissa_bits}f} + ({big_m:.{mantissa_bits}f}) = {result_m:.{mantissa_bits}f}")

    # === 第三步: 规格化 ===
    print(f"\n【步骤3: 规格化】")
    if result_m == 0:
        print(f"  结果为0")
        return (0, 0.0)

    original_m = result_m
    while abs(result_m) >= 1.0:
        # 右规
        result_m /= 2
        result_e += 1
        print(f"  右规: 尾数={result_m:.{mantissa_bits}f}, 阶码={result_e}")
    while abs(result_m) < 0.5 and result_m != 0:
        # 左规
        result_m *= 2
        result_e -= 1
        print(f"  左规: 尾数={result_m:.{mantissa_bits}f}, 阶码={result_e}")

    # === 第四步: 舍入 ===
    print(f"\n【步骤4: 舍入】")
    # 0舍1入 (简化处理)
    rounded_m = round(result_m, mantissa_bits - 1)
    print(f"  舍入前: {result_m:.{mantissa_bits}f}")
    print(f"  舍入后: {rounded_m:.{mantissa_bits}f}")

    # === 第五步: 溢出判断 ===
    print(f"\n【步骤5: 溢出判断】")
    E_max = 7   # 假设4位阶码, 偏置=7
    E_min = -8
    if result_e > E_max:
        print(f"  阶码上溢! E={result_e} > Emax={E_max}")
    elif result_e < E_min:
        print(f"  阶码下溢! E={result_e} < Emin={E_min}")
    else:
        print(f"  无溢出, 阶码={result_e} 在 [{E_min}, {E_max}] 范围内")

    print(f"\n最终结果: {rounded_m}×2^{result_e} = {rounded_m * (2**result_e)}")
    return (result_e, rounded_m)

# 示例: 0.11010×2^3 + 0.10010×2^1
print("=== 浮点加法示例 ===")
float_add_sub((3, 0.11010), (1, 0.10010), op='+')

print("\n=== 浮点减法示例 ===")
float_add_sub((2, 0.10100), (2, 0.10010), op='-')
```
