# 11_奇偶校验码_408考点

## 核心概念

### 408常见题型
1. 给定数据计算奇/偶校验位
2. 给定接收码判断是否出错
3. 与海明码、CRC的对比选择
4. 分析检错能力

### 检错能力总结

| 错误类型 | 能否检测 |
|----------|----------|
| 0位错误 | 无错误 |
| 1位错误 | 能 |
| 2位错误 | 不能 |
| 3位错误 | 能 |
| 4位错误 | 不能 |
| n位错误 | 奇数位能，偶数位不能 |

## 原理分析

### 考题1：计算校验位

**题目**：设数据为 $1101011$，分别求奇校验和偶校验的校验位。

**解**：
- 数据中1的个数 = 5（奇数）
- 偶校验：需要总个数为偶数，校验位 $P = 1$
- 奇校验：需要总个数为奇数，校验位 $P = 0$

偶校验编码：$1101011\mathbf{1}$（6个1，偶数 ✓）

奇校验编码：$1101011\mathbf{0}$（5个1，奇数 ✓）

### 考题2：检错判断

**题目**：采用偶校验，收到数据 $10011010$，判断是否出错。

**解**：
- 1的个数 = 4（偶数）
- 偶校验要求1的总数为偶数
- 4是偶数，**未检测到错误**（可能是正确数据，也可能是偶数位出错）

**题目**：采用偶校验，收到数据 $10011011$，判断是否出错。

**解**：
- 1的个数 = 5（奇数）
- 偶校验要求1的总数为偶数
- 5是奇数，**检测到错误**

### 考题3：与海明码对比

**题目**：下列关于奇偶校验码的叙述中，正确的是（）

A. 奇偶校验码能检测任意位数的错误
B. 奇偶校验码能纠正1位错误
C. 奇偶校验码能检测奇数位错误
D. 奇偶校验码的编码效率高于海明码

**分析**：
- A：错，偶数位错误检测不到
- B：错，只能检错不能纠错
- C：正确，奇数位出错必然改变奇偶性
- D：正确，奇偶校验只加1位校验位，海明码需要多位

**答案**：C、D

### 考题4：编码效率计算

**题目**：8位数据加1位奇偶校验位，编码效率是多少？

**解**：
$$\text{编码效率} = \frac{\text{信息位数}}{\text{总位数}} = \frac{8}{9} \approx 88.9\%$$

与海明码对比：8位数据用海明码需要4位校验位，效率 = $8/12 = 66.7\%$

## 易错陷阱

### 陷阱1：偶数位错误不可检测
如果同时有2位（或任意偶数位）出错，1的个数奇偶性不变，校验检查通过但数据实际是错的。

### 陷阱2：校验位位置
408题目中校验位可能放在最高位或最低位，务必看清题目描述。

### 陷阱3：奇校验和偶校验混用
- 奇校验：$P = 1 \oplus (d_1 \oplus d_2 \oplus \cdots \oplus d_n)$
- 偶校验：$P = d_1 \oplus d_2 \oplus \cdots \oplus d_n$

### 陷阱4：与海明码的本质区别
- 奇偶校验：只有1个校验位，只能检错
- 海明码：多个校验位，交叉分组，能检错并纠错
- 奇偶校验是海明码的基础思想（海明码使用多组奇偶校验）

## 知识关联

### 编码效率与检错/纠错能力的权衡

| 校验码类型 | 校验位数 | 检错能力 | 纠错能力 | 编码效率 |
|-----------|---------|---------|---------|---------|
| 奇偶校验 | 1 | 奇数位错误 | 无 | 最高 |
| 海明码 | r位 ($2^r \geq n+r+1$) | 2位错误 | 1位错误 | 中等 |
| CRC | r位（生成多项式次数） | ≤r位突发错误 | 无 | 较高 |

### 408选择题常见混淆
- "奇偶校验码能纠正1位错误" → **错误**，只能检测
- "海明码能检测2位错误" → **正确**（但只能纠正1位）
- "CRC能检测所有错误" → **错误**，有检错概率限制
