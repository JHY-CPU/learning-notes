## 典型考题

### 考题1：补码算术移位

**题目**：设 $[X]_{\text{补}} = 10110110$，分别求算术左移1位和算术右移1位的结果。

**解**：

$[X]_{\text{补}} = 10110110$（$X = -74$）

**算术左移1位**：$01101100$

符号位从1变为0 → **溢出**（下溢方向）

验证：$-74 \times 2 = -148 < -128$（超出8位补码范围）

**算术右移1位**：$11011011$

最高位补符号位1。

验证：$11011011$（补码）→ 取反+1 = `00100100` + 1 = `00100101` = 37，结果 $= -37$。

$-74 \div 2 = -37$ ✓

### 考题2：无符号数逻辑移位

**题目**：设无符号数 $X = 11010110 = 214$，求逻辑左移1位和逻辑右移1位的结果。

**解**：

**逻辑左移1位**：$10101100 = 172$

最高位1丢失，结果 $\neq 214 \times 2$（溢出）

**逻辑右移1位**：$01101011 = 107$

最高位补0，$214 \div 2 = 107$ ✓

### 考题3：移位与乘除法

**题目**：用移位操作实现 $X \times 6$（$X$ 为8位有符号数）。

**解**：

$$X \times 6 = X \times 4 + X \times 2 = (X \ll 2) + (X \ll 1)$$

即：将 $X$ 左移2位加上左移1位。

**验证**：设 $X = 5$

$5 \times 2 = 10$（左移1位）
$5 \times 4 = 20$（左移2位）
$10 + 20 = 30 = 5 \times 6$ ✓

### 考题4：移位结果判断

**题目**：$[X]_{\text{补}} = 01001100$，算术左移1位后，判断是否溢出。

**解**：

左移1位：$10011000$

原符号位0，新符号位1 → 符号位变化 → **上溢**

$X = +76$，$76 \times 2 = 152 > 127$（8位补码最大正数），确认溢出。

### 考题5：算术右移取整方向

**题目**：$[-7]_{\text{补}} = 11111001$，算术右移1位的结果是多少？

**解**：

算术右移1位：$11111100$

验证：$11111100$（补码）→ 取反+1 = `00000011` + 1 = `00000100` = 4，结果 $= -4$。

$-7 \div 2 = -3.5$，向下取整 $= -4$ ✓

**注意**：算术右移对负数是**向下取整**（向负无穷方向），不是向0取整。

## 左移右移关系总结

| 操作 | 无符号数 | 有符号数（补码） |
|------|---------|---------------|
| 左移1位 | $\times 2$（可能溢出） | $\times 2$（可能溢出） |
| 右移1位（逻辑） | $\div 2$（向0取整） | 不适用 |
| 右移1位（算术） | 不适用 | $\div 2$（向下取整） |

### 取整方向对比

| 真值 | $\div 2$ 精确值 | 算术右移（向下取整） | 向0取整 |
|------|---------------|------------------|--------|
| $+7$ | $+3.5$ | $+3$ | $+3$ |
| $-7$ | $-3.5$ | $-4$ | $-3$ |
| $+5$ | $+2.5$ | $+2$ | $+2$ |
| $-5$ | $-2.5$ | $-3$ | $-2$ |

算术右移对负数的取整方向是**向下（向负无穷）**，这是需要注意的细节。

## 易错陷阱

### 陷阱1：算术左移和逻辑左移相同
- 对于左移，算术移位和逻辑移位规则**完全相同**：低位补0
- 区别只在右移：算术右移补符号位，逻辑右移补0

### 陷阱2：左移溢出
- 左移可能导致溢出（符号位被改变）
- 检验方法：移位后符号位与原符号位不同则溢出

### 陷阱3：右移的取整方向
- 算术右移：向下取整（向负无穷方向）
- $-5$ 算术右移1位 $= -3$，不是 $-2$

### 陷阱4：多次移位
- 左移 $k$ 位 ≈ $\times 2^k$
- 右移 $k$ 位 ≈ $\div 2^k$
- 多次移位的溢出风险更大

## 知识关联

### 与编译器优化的关系
- 编译器常将乘以2的幂优化为移位
- `x * 8` 优化为 `x << 3`
- `x / 4` 优化为 `x >> 2`（对无符号用逻辑右移，对有符号用算术右移）

### 408考试规律
- 移位结果的计算在选择题中常见
- 移位和乘除法的关系是理解考点
- 算术右移对负数的取整方向可能出现在选择题中
- 移位溢出判断与前面的溢出检测内容呼应

### 自测练习

1. $[+25]_{\text{补}} = 00011001$，左移1位 $= ?$
2. $[-25]_{\text{补}} = 11100111$，算术右移1位 $= ?$
3. 无符号数 $00110110 = 54$，逻辑左移2位 $= ?$
4. $[-13]_{\text{补}} = 11110011$，算术右移1位 $= ?$

**答案**：
1. $00110010 = +50$ ✓
2. $11110011 = -13$（验证：取反+1 = `00001101` = 13，$= -13$... 等等）

让我重算：$[-25]_{\text{补}} = 11100111$，右移1位补符号位1：$11110011$（补码）

$11110011$ → 取反+1 = `00001100` + 1 = `00001101` = 13，$= -13$。

$-25 \div 2 = -12.5$，向下取整 $= -13$ ✓

3. $11011000 = 216$（逻辑左移2位，低位补2个0）
4. $[-13]_{\text{补}} = 11110011$，算术右移1位：$11111001$（补码）

$11111001$ → 取反+1 = `00000110` + 1 = `00000111` = 7，$= -7$。

$-13 \div 2 = -6.5$，向下取整 $= -7$ ✓
