# 36_补码减法

## 核心概念

### 补码减法公式

$$[X - Y]_{\text{补}} = [X]_{\text{补}} + [-Y]_{\text{补}} \pmod{2^n}$$

**关键**：减法统一为加法！只需将减数取负后与被减数相加。

### 求 $[-Y]_{\text{补}}$ 的方法

对 $[Y]_{\text{补}}$ **所有位（包括符号位）取反再加1**：

$$[-Y]_{\text{补}} = \overline{[Y]_{\text{补}}} + 1$$

**示例**：
- $[Y]_{\text{补}} = 00101010$（+42）
- 取反：$11010101$
- 加1：$11010110$
- $[-Y]_{\text{补}} = 11010110$（-42）

---

## 原理分析

### 例题1：正数 - 正数（结果为正）

计算 $65 - 23$（8位补码）

$$[65]_{\text{补}} = 01000001$$
$$[23]_{\text{补}} = 00010111$$
$$[-23]_{\text{补}} = \overline{00010111} + 1 = 11101000 + 1 = 11101001$$

```
  01000001
+ 11101001
----------
 100101010  （9位）
```

舍弃进位：$00101010 = +42$

$65 - 23 = 42$ ✓

### 例题2：正数 - 正数（结果为负）

计算 $23 - 65$（8位补码）

$$[23]_{\text{补}} = 00010111$$
$$[-65]_{\text{补}} = \overline{01000001} + 1 = 10111110 + 1 = 10111111$$

```
  00010111
+ 10111111
----------
  11010110  （8位）
```

结果 $11010110$（补码，符号位1，为负数）

验证：取反+1 = `00101001` + 1 = `00101010` = 42，所以结果 = $-42$。

$23 - 65 = -42$ ✓

### 例题3：负数 - 负数

计算 $(-23) - (-65)$（8位补码）

$$[-23]_{\text{补}} = 11101001$$
$$[-(-65)]_{\text{补}} = [+65]_{\text{补}} = 01000001$$

（$-(-65) = +65$，直接用+65的补码）

```
  11101001
+ 01000001
----------
 100101010  （9位）
```

舍弃进位：$00101010 = +42$

$(-23) - (-65) = -23 + 65 = +42$ ✓

### 例题4：负数 - 正数

计算 $(-42) - 23$（8位补码）

$$[-42]_{\text{补}} = 11010110$$
$$[-23]_{\text{补}} = 11101001$$

```
  11010110
+ 11101001
----------
 110111111  （9位）
```

舍弃进位：$10111111$（补码，符号位1，负数）

验证：取反+1 = `01000000` + 1 = `01000001` = 65，结果 = $-65$。

$(-42) - 23 = -65$ ✓

## 直观理解

### 运算步骤总结

1. 将被减数和减数都写成补码形式
2. 对减数的补码**全部取反+1**，得到 $[-Y]_{\text{补}}$
3. 将被减数的补码与 $[-Y]_{\text{补}}$ **相加**
4. 舍弃最高位进位
5. 结果即为差的补码

### 硬件优势

补码减法在硬件上与加法使用**同一个加法器**：
- 加法：输入 $[X]_{\text{补}}$ 和 $[Y]_{\text{补}}$
- 减法：输入 $[X]_{\text{补}}$ 和 $\overline{[Y]_{\text{补}}} + 1$（取反加1可以用加法器的进位输入实现）

这就是补码的最大优势：**加减法共用同一套硬件**。

## 知识关联

### 408考试要点
- 补码减法公式
- $[-Y]_{\text{补}}$ 的求法（全部取反+1）
- 运算过程和结果验证
- 溢出判断

### 易错陷阱
1. **取反+1包括符号位**：不要只取反数值位
2. **减法变成加法**：不要做二进制减法，而是取负后做加法
3. **进位舍弃**：与加法一样，最高位进位要舍弃
4. **$[-Y]_{\text{补}}$ 的 $+1$ 不会产生溢出**：最坏情况就是全部位变为0（$Y=0$ 时）
