# 19_有符号整数_原码

## 核心概念

### 定义
**原码**（Sign-Magnitude）是最直观的有符号数表示方法：**最高位为符号位**（0正1负），其余位为**数值的绝对值**（真值的二进制表示）。

### 编码规则

设 $n$ 位机器字长，其中1位符号位，$n-1$ 位数值位：

$$[X]_{\text{原}} = \begin{cases} 0\underbrace{x_1 x_2 \cdots x_{n-1}}_{\text{绝对值}} & \text{当 } X \geq 0 \\ 1\underbrace{x_1 x_2 \cdots x_{n-1}}_{\text{绝对值}} & \text{当 } X < 0 \end{cases}$$

### 表示范围

$n$ 位原码的表示范围：

$$-(2^{n-1} - 1) \leq X \leq +(2^{n-1} - 1)$$

| 位数 | 范围 |
|------|------|
| 8位 | $-127 \sim +127$ |
| 16位 | $-32767 \sim +32767$ |
| $n$位 | $-(2^{n-1}-1) \sim +(2^{n-1}-1)$ |

### 特殊值：+0 和 -0

原码中有**两个零**：
- $+0$：$00000000$（8位）
- $-0$：$10000000$（8位）

这是原码的一个缺点——浪费了一个编码位置。

## 原理分析

### 原码示例（8位）

| 真值 | 原码 | 说明 |
|------|------|------|
| $+0$ | `00000000` | 符号位0 + 7个0 |
| $-0$ | `10000000` | 符号位1 + 7个0 |
| $+1$ | `00000001` | 符号位0 + 0000001 |
| $-1$ | `10000001` | 符号位1 + 0000001 |
| $+127$ | `01111111` | 符号位0 + 1111111 |
| $-127$ | `11111111` | 符号位1 + 1111111 |
| $+42$ | `00101010` | 符号位0 + 0101010 |
| $-42$ | `10101010` | 符号位1 + 0101010 |

**规律**：正数和负数的原码只有符号位不同，数值位完全相同。

### 原码的加减法问题

原码最大的缺点：**加减法实现复杂**。

做加法时需要：
1. 判断两个操作数的符号
2. 如果同号：绝对值相加，结果符号与操作数相同
3. 如果异号：绝对值相减，结果符号与绝对值大的相同

这需要复杂的硬件判断逻辑，因此计算机内部**不使用原码进行运算**。

**示例**：用原码计算 $(+5) + (-3)$

1. $[+5]_{\text{原}} = 00000101$，$[-3]_{\text{原}} = 10000011$
2. 符号不同，需要做减法
3. $|5| - |3| = 5 - 3 = 2$
4. 结果符号取绝对值大的（5 > 3，取正）
5. 结果 $= +2$，$[+2]_{\text{原}} = 00000010$

对比补码：只需做加法，简单得多！

## 直观理解

### 生活类比
原码就像温度计：
- 零度以上标记"+"
- 零度以下标记"-"
- 但"+0度"和"-0度"都表示同一个温度，却有两个不同的标记

### 记忆要点
- 原码 = 符号位 + 绝对值
- 正数和负数只有符号位不同
- 两个零（+0和-0）
- 运算复杂，实际不用于计算机内部运算

## 知识关联

### 与其他编码的对比

| 特性 | 原码 | 补码 | 反码 |
|------|------|------|------|
| 正数表示 | 同真值 | 同真值 | 同真值 |
| 负数表示 | 符号+绝对值 | 取反+1 | 取反 |
| 零的个数 | 2个（+0,-0） | 1个 | 2个 |
| 加减法 | 需要判断符号 | 统一为加法 | 需要循环进位 |
| 范围(8位) | -127~+127 | -128~+127 | -127~+127 |

### 408考试要点
- 原码的定义和表示方法
- 原码的表示范围（比补码少一个数）
- 原码+0和-0的问题
- 原码加减法的复杂性是引入补码的原因
- 浮点数的尾数用原码表示（重要关联）

### 易错陷阱
1. **原码和真值的转换**：负数的原码不是简单地在前面加1，而是符号位1 + 绝对值的二进制
2. **原码取反不是求负**：原码取反只是改变了数值位，不是求相反数
3. **范围计算**：8位原码范围是-127~+127，不是-128~+127（补码才是）
