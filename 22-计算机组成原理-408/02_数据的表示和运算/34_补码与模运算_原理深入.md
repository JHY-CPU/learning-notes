# 34_补码与模运算_原理深入

## 核心概念

### 补码的模运算本质

在 $n$ 位二进制系统中，模 $M = 2^n$。补码的定义：

$$[X]_{\text{补}} = \begin{cases} X & 0 \leq X < 2^{n-1} \\ 2^n + X & -2^{n-1} \leq X < 0 \end{cases}$$

本质上：$[X]_{\text{补}} = X \mod 2^n$（在模 $2^n$ 意义下）

### 为什么减法可以变成加法？

要计算 $A - B$（$A, B > 0$，$A < B$，结果为负）：

在模 $2^n$ 系统中：

$$A - B \equiv A + (2^n - B) \pmod{2^n}$$

$2^n - B$ 就是 $-B$ 的补码表示！

所以：$A - B = A + [−B]_{\text{补}} \pmod{2^n}$

这就是补码加法统一减法的严格数学证明。

## 原理分析

### 严格推导

设 $A$ 和 $B$ 都是 $n$ 位二进制数，$A, B > 0$，$A < B$。

**目标**：计算 $A - B$（结果为负数）

**Step 1**：在模 $2^n$ 系统中，$-B \equiv 2^n - B \pmod{2^n}$

**Step 2**：$2^n - B$ 就是 $-B$ 的补码表示（因为 $B > 0$，所以 $-B < 0$，补码 $= 2^n + (-B) = 2^n - B$）

**Step 3**：

$$A - B \equiv A + (2^n - B) \pmod{2^n}$$

$$= (A - B) + 2^n$$

因为 $A - B < 0$，所以 $0 < (A - B) + 2^n < 2^n$。

$(A - B) + 2^n$ 正好是 $A - B$ 这个负数的补码表示！

**结论**：$[A - B]_{\text{补}} = [A]_{\text{补}} + [-B]_{\text{补}} \pmod{2^n}$

### 具体示例

计算 $5 - 3$（8位补码）：

$$[5]_{\text{补}} = 00000101$$
$$[-3]_{\text{补}} = 256 - 3 = 253 = 11111101$$

$$[5]_{\text{补}} + [-3]_{\text{补}} = 00000101 + 11111101 = 1\ 00000010$$

模 $2^8$（截断最高位）：$000000010 = +2$ ✓

### 溢出的模运算解释

当两个正数相加结果 $\geq 2^{n-1}$ 时：

$$A + B \geq 2^{n-1}$$

在补码中，结果被解释为负数（因为最高位变为1），这就是**上溢**。

当两个负数相加结果 $< -2^{n-1}$ 时：

$$A + B < -2^{n-1}$$

在模 $2^n$ 系统中，$(A + B) + 2^n < 2^{n-1}$，结果被解释为正数，这就是**下溢**。

### 补码加法的进位舍弃

在模 $2^n$ 系统中，超过 $2^n$ 的部分自动被"截断"（取模）：

$$A + B \mod 2^n = \begin{cases} A + B & \text{if } A + B < 2^n \\ A + B - 2^n & \text{if } A + B \geq 2^n \end{cases}$$

最高位的进位恰好就是 $A + B \geq 2^n$ 的标志，舍弃它等价于减去 $2^n$。

## 直观理解

### 时钟的深度类比

**设定**：12小时制时钟（模12）

**问题**：3点往回拨5小时是几点？

**方法1**（减法）：$3 - 5 = -2$（不存在"负2点"）

**方法2**（补码加法）：
- $-5$ 在模12下的补码 $= 12 + (-5) = 7$
- $3 + 7 = 10$
- 结果：10点 ✓

**关键**：补码就是把负数转换成它的"模等价值"，使减法变为加法。

### 二进制时钟

$n$ 位寄存器就像一个从0到 $2^n - 1$ 的圆形刻度盘：
- 顺时针转（加法）：到达 $2^n$ 后自动回到0
- 逆时针转（减法）：到达0后自动跳到 $2^n - 1$
- 这就是模运算的循环特性

## 知识关联

### 补码运算的三大规则

1. **$[X + Y]_{\text{补}} = [X]_{\text{补}} + [Y]_{\text{补}}$**（直接相加，舍弃进位）
2. **$[X - Y]_{\text{补}} = [X]_{\text{补}} + [-Y]_{\text{补}}$**（减法变加法）
3. **$[-Y]_{\text{补}}$ = $[Y]_{\text{补}}$ 全部取反 + 1**（求相反数的补码）

### 易错陷阱
1. **模运算中没有溢出**：模运算的结果总是"合法"的，只是解释可能不同
2. **计算机中的溢出**：是结果超出了有符号整数的范围（$-2^{n-1} \sim 2^{n-1}-1$）
3. **模运算的结果被截断**：最高位进位舍弃 = 取模操作
