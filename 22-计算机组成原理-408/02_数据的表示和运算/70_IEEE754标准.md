# 71_IEEE754标准

## 核心概念

### IEEE 754 标准的定义

**IEEE 754** 是国际电气和电子工程师协会（IEEE）制定的**浮点数算术标准**，于1985年发布，是目前几乎所有计算机系统使用的浮点数标准。

### IEEE 754 的基本格式

```
┌────┬────────────┬───────────────────────────────┐
│ S  │  Exponent  │         Mantissa              │
│符号│    阶码     │           尾数                │
└────┴────────────┴───────────────────────────────┘
 1位    e位              m位
```

**关键创新**：**隐含的前导1（Hidden Leading 1）**

由于规格化数的尾数最高位总是 1，IEEE 754 不存储这一位，从而多获得 1 位精度。

实际尾数 = $1.M$（隐含的 1 + 存储的尾数位）

### IEEE 754 的三种主要格式

| 格式 | 总位数 | 符号位 | 阶码位数 | 尾数位数 | 偏置值 | 精度 |
|------|--------|--------|---------|---------|--------|------|
| **float**（单精度） | 32 | 1 | 8 | 23 | 127 | ~7位十进制 |
| **double**（双精度） | 64 | 1 | 11 | 52 | 1023 | ~16位十进制 |
| **half**（半精度） | 16 | 1 | 5 | 10 | 15 | ~3位十进制 |

### IEEE 754 float（32位）详细格式

```
 31   30────23   22──────────────0
┌───┬─────────┬───────────────────┐
│ S │  e₇~e₀  │    m₂₂~m₀         │
│符号│ 8位阶码 │   23位尾数         │
└───┴─────────┴───────────────────┘
```

### 阶码的偏置值

IEEE 754 使用**偏置值（Bias）**来表示阶码：

$$E_{真值} = e_{存储} - Bias$$

| 格式 | 阶码位数 | Bias | 阶码存储值范围 | 真值范围 |
|------|---------|------|--------------|---------|
| float | 8 | 127 | 1~254 | -126~+127 |
| double | 11 | 1023 | 1~2046 | -1022~+1023 |

### 尾数与隐含1

IEEE 754 的规格化数，尾数实际值为：

$$M = 1.m_{m-1}m_{m-2}\cdots m_0$$

即隐含一个 1 在小数点前。

**示例**：
- 存储的尾数：$101000...0$
- 实际尾数：$1.101000...0 = 1 + 0.5 + 0.125 = 1.625$

### IEEE 754 浮点数真值公式

**规格化数**：

$$N = (-1)^S \times 1.M \times 2^{e - Bias}$$

**示例**：$S = 0$，$e = 10000010_2 = 130$，$M = 101000...0$

$$N = (+1) \times 1.101 \times 2^{130-127} = 1.625 \times 2^3 = 13.0$$

## 原理分析

### IEEE 754 的设计精妙之处

1. **隐含1**：多1位精度，23位尾数实际有24位精度
2. **偏置值**：使得阶码可以用无符号数比较
3. **特殊值保留**：阶码全0和全1表示特殊值
4. **非规格化数**：阶码全0时提供渐进下溢

### 阶码编码的含义

| 阶码（存储值） | 含义 |
|--------------|------|
| 0 | 非规格化数 / 零 |
| 1~254 | 规格化数 |
| 255 | 特殊值（无穷大 / NaN） |

### 符号位

- $S = 0$：正数
- $S = 1$：负数
- 注意：$+0$ 和 $-0$ 是不同的（$S$ 不同）

## 直观理解

### 记忆要点

- **IEEE 754 float = 1位符号 + 8位阶码 + 23位尾数**
- **偏置值 = 127（float），1023（double）**
- **隐含1：$1.M$**
- **真值 = $(-1)^S \times 1.M \times 2^{e-127}$**
- **阶码全0和全1是特殊值**

### 类比

IEEE 754 就像一个标准化的科学计数法格式：
- 符号 → 正负号
- 阶码 → 指数（用偏置值让负指数变成正数存储）
- 尾数 → 有效数字（省略最高位的1来多存1位精度）
