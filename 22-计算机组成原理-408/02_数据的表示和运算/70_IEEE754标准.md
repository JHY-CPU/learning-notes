# 71_IEEE754标准

## 核心概念

### IEEE 754 标准的定义

**IEEE 754** 是国际电气和电子工程师协会（IEEE）制定的**浮点数算术标准**，于1985年发布，是目前几乎所有计算机系统使用的浮点数标准。

### IEEE 754 的基本格式

```
┌────┬────────────┬───────────────────────────────┐
│ S  │  Exponent  │         Mantissa              │
│符号│    阶码     │           尾数                │
└────┴────────────┴───────────────────────────────┘
 1位    e位              m位
```

**关键创新**：**隐含的前导1（Hidden Leading 1）**

由于规格化数的尾数最高位总是 1，IEEE 754 不存储这一位，从而多获得 1 位精度。

实际尾数 = $1.M$（隐含的 1 + 存储的尾数位）

### IEEE 754 的三种主要格式

| 格式 | 总位数 | 符号位 | 阶码位数 | 尾数位数 | 偏置值 | 精度 |
|------|--------|--------|---------|---------|--------|------|
| **float**（单精度） | 32 | 1 | 8 | 23 | 127 | ~7位十进制 |
| **double**（双精度） | 64 | 1 | 11 | 52 | 1023 | ~16位十进制 |
| **half**（半精度） | 16 | 1 | 5 | 10 | 15 | ~3位十进制 |

### IEEE 754 float（32位）详细格式

```
 31   30────23   22──────────────0
┌───┬─────────┬───────────────────┐
│ S │  e₇~e₀  │    m₂₂~m₀         │
│符号│ 8位阶码 │   23位尾数         │
└───┴─────────┴───────────────────┘
```

### 阶码的偏置值

IEEE 754 使用**偏置值（Bias）**来表示阶码：

$$E_{真值} = e_{存储} - Bias$$

| 格式 | 阶码位数 | Bias | 阶码存储值范围 | 真值范围 |
|------|---------|------|--------------|---------|
| float | 8 | 127 | 1~254 | -126~+127 |
| double | 11 | 1023 | 1~2046 | -1022~+1023 |

### 尾数与隐含1

IEEE 754 的规格化数，尾数实际值为：

$$M = 1.m_{m-1}m_{m-2}\cdots m_0$$

即隐含一个 1 在小数点前。

**示例**：
- 存储的尾数：$101000...0$
- 实际尾数：$1.101000...0 = 1 + 0.5 + 0.125 = 1.625$

### IEEE 754 浮点数真值公式

**规格化数**：

$$N = (-1)^S \times 1.M \times 2^{e - Bias}$$

**示例**：$S = 0$，$e = 10000010_2 = 130$，$M = 101000...0$

$$N = (+1) \times 1.101 \times 2^{130-127} = 1.625 \times 2^3 = 13.0$$

## 原理分析

### IEEE 754 的设计精妙之处

1. **隐含1**：多1位精度，23位尾数实际有24位精度
2. **偏置值**：使得阶码可以用无符号数比较
3. **特殊值保留**：阶码全0和全1表示特殊值
4. **非规格化数**：阶码全0时提供渐进下溢

### 阶码编码的含义

| 阶码（存储值） | 含义 |
|--------------|------|
| 0 | 非规格化数 / 零 |
| 1~254 | 规格化数 |
| 255 | 特殊值（无穷大 / NaN） |

### 符号位

- $S = 0$：正数
- $S = 1$：负数
- 注意：$+0$ 和 $-0$ 是不同的（$S$ 不同）

## 直观理解

### 记忆要点

- **IEEE 754 float = 1位符号 + 8位阶码 + 23位尾数**
- **偏置值 = 127（float），1023（double）**
- **隐含1：$1.M$**
- **真值 = $(-1)^S \times 1.M \times 2^{e-127}$**
- **阶码全0和全1是特殊值**

### 类比

IEEE 754 就像一个标准化的科学计数法格式：
- 符号 → 正负号
- 阶码 → 指数（用偏置值让负指数变成正数存储）
- 尾数 → 有效数字（省略最高位的1来多存1位精度）

## 代码/模拟

### Python实现IEEE 754编码与解码

```python
"""IEEE 754浮点数编码/解码模拟 - 适用于408考研复习"""
import struct

def float_to_ieee754(value):
    """十进制浮点数 → IEEE 754单精度二进制"""
    # 使用struct模块获取IEEE 754的二进制表示
    packed = struct.pack('>f', value)
    bits = int.from_bytes(packed, 'big')
    binary = format(bits, '032b')

    # 解析各字段
    sign = int(binary[0])
    exponent = int(binary[1:9], 2)
    mantissa = binary[9:]

    # 计算实际值
    bias = 127
    if exponent == 0:
        actual_exp = 1 - bias
        actual_mantissa = '0.' + mantissa
    else:
        actual_exp = exponent - bias
        actual_mantissa = '1.' + mantissa

    print(f"十进制: {value}")
    print(f"IEEE 754单精度: {binary}")
    print(f"  符号位 S = {sign} ({'负' if sign else '正'})")
    print(f"  阶码 E  = {exponent:08b} = {exponent} (偏置={bias})")
    print(f"  尾数 M  = {mantissa}")
    print(f"  实际值 = (-1)^{sign} × {actual_mantissa} × 2^{actual_exp}")
    print(f"  = {value}")
    return binary

def ieee754_to_float(binary_str):
    """IEEE 754单精度二进制 → 十进制浮点数"""
    bits = int(binary_str, 2)
    packed = bits.to_bytes(4, 'big')
    value = struct.unpack('>f', packed)[0]

    sign = int(binary_str[0])
    exponent = int(binary_str[1:9], 2)
    mantissa = binary_str[9:]

    print(f"\nIEEE 754: {binary_str}")
    print(f"  S={sign}, E={exponent}, M={mantissa}")
    if exponent == 0 and mantissa == '0'*23:
        print(f"  → 零: {value}")
    elif exponent == 255:
        if mantissa == '0'*23:
            print(f"  → 无穷大: {'+' if sign==0 else '-'}∞")
        else:
            print(f"  → NaN")
    else:
        print(f"  → {value}")
    return value

# 示例
print("=== IEEE 754 编码示例 ===\n")
float_to_ieee754(6.75)
float_to_ieee754(-0.75)
float_to_ieee754(0.1)

print("\n=== IEEE 754 解码示例 ===")
ieee754_to_float('01000000110110000000000000000000')  # 6.75
ieee754_to_float('10111111010000000000000000000000')  # -0.75

# 手动编码过程演示
print("\n=== 手动编码 6.75 过程 ===")
print("1. 整数部分: 6 = 110")
print("2. 小数部分: 0.75 × 2 = 1.5 → 1")
print("              0.5 × 2 = 1.0 → 1")
print("3. 二进制: 110.11")
print("4. 规格化: 1.1011 × 2^2")
print("5. S=0 (正数)")
print("6. E=127+2=129=10000001")
print("7. M=10110000000000000000000 (隐含前导1)")
print("8. 结果: 0 10000001 10110000000000000000000")
```
