# 75_IEEE754_408考点

## 核心概念

IEEE 754 是 408 考试的**高频必考内容**，几乎每年都有涉及。

## 408高频考点

### 考点1：IEEE 754 表示的识别

**典型问法**：给定 32 位十六进制数，问它表示什么类型的 IEEE 754 数。

**判断流程**：
1. 转为二进制，拆分字段
2. 检查阶码：
   - 全0 → 零或非规格化数
   - 全1 → 无穷大或NaN
   - 其他 → 规格化数

**典型真题**：`0x40000000` 表示什么？

二进制：`0 10000000 00000000000000000000000`
- $S = 0$, $e = 128$, $E = 1$, 尾数 = $1.0$
- $N = 1.0 \times 2^1 = 2.0$

### 考点2：特殊值的识别

**典型问法**：以下哪个是 NaN / 无穷大 / 零？

| 十六进制 | 含义 |
|---------|------|
| `0x00000000` | +0 |
| `0x80000000` | -0 |
| `0x7F800000` | +∞ |
| `0xFF800000` | -∞ |
| `0x7FC00000` | NaN |
| `0x7F7FFFFF` | 最大有限正数 |
| `0x00800000` | 最小正规格化数 |
| `0x00000001` | 最小正非规格化数 |

### 考点3：阶码偏置值

**典型问法**：float 的阶码偏置值是多少？

**答案**：127

**问**：阶码真值范围？

**答案**：$[-126, +127]$（规格化数）

### 考点4：隐含1

**典型问法**：IEEE 754 中为什么 23 位尾数有 24 位精度？

**答案**：因为规格化数的尾数隐含前导 1，存储的是 $1.M$ 中的 $M$ 部分。

### 考点5：十进制与 IEEE 754 互转

见第74节详细计算过程。

### 考点6：表示范围

**典型问法**：float 的最大有限值是多少？

$$(2 - 2^{-23}) \times 2^{127} \approx 3.40 \times 10^{38}$$

**典型问法**：float 的最小正规格化数是多少？

$$2^{-126} \approx 1.18 \times 10^{-38}$$

### 考点7：易错细节

1. **阶码 0 和 255 保留**：实际可用阶码 1~254
2. **隐含1 只用于规格化数**：非规格化数没有隐含1
3. **±0 不同但相等**
4. **NaN ≠ NaN**
5. **0.1 不能精确表示**

## 典型真题精选

### 真题1（选择题）

IEEE 754 单精度浮点数中，阶码的偏置值是：

A. 128　　B. 127　　C. 255　　D. 1023

**答案**：B

### 真题2（选择题）

IEEE 754 单精度浮点数 `0x42C80000` 表示的十进制值是：

A. 100.0　　B. 50.0　　C. 200.0　　D. 25.0

**答案**：A

解析：
```
0100 0010 1100 1000 0000 0000 0000 0000
S=0, e=10000101=133, E=133-127=6
M=1001000...0, 1.1001000 = 1+0.5+0.0625=1.5625
N=1.5625×2⁶=1.5625×64=100.0
```

### 真题3（选择题）

IEEE 754 单精度格式中，以下哪个数表示正无穷大？

A. `0x7F800000`　　B. `0x7FC00000`　　C. `0xFF800000`　　D. `0x00000000`

**答案**：A（阶码全1，尾数全0，符号0）

### 真题4（选择题）

IEEE 754 单精度格式中，能表示的最小正数（非规格化）是：

A. $2^{-126}$　　B. $2^{-127}$　　C. $2^{-149}$　　D. $2^{-150}$

**答案**：C

解析：最小非规格化正数 = $2^{-23} \times 2^{-126} = 2^{-149}$

### 真题5（大题节选）

将 $-12.5$ 转为 IEEE 754 单精度表示。

解答：
1. $12.5 = 1100.1_2$
2. 规格化：$1.1001 \times 2^3$
3. $S = 1$, $e = 3 + 127 = 130 = 10000010_2$
4. 尾数：$10010000000000000000000$
5. 结果：`1 10000010 10010000000000000000000`
6. 十六进制：`0xC1480000`

## 易错总结

1. **阶码是无符号整数**：存储时不需要符号位
2. **偏置值是 127 不是 128**
3. **阶码 0 和 255 是特殊值**：可用范围 1~254
4. **非规格化数的阶码真值是 $1 - Bias$ 不是 $0 - Bias$**
5. **尾数位数是存储位数**：实际精度多1位
6. **转换时注意舍入**：不是所有十进制数都能精确表示
