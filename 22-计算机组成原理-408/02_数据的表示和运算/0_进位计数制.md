# 01_进位计数制

## 核心概念

### 定义
**进位计数制**（Positional Number System）是一种用**固定基数**和**位权**表示数值的方法。每一位数字的值不仅取决于数字本身，还取决于它所在的位置。

### 核心要素
- **基数（Base/Radix）**：每一位数字可能取值的个数，记为 $r$
- **位权（Weight）**：每一位对应的权值，第 $i$ 位的权值为 $r^i$
- **数位（Digit）**：表示每一位上的具体数值

### 常见进制

| 进制 | 基数 | 使用数字 | 计算机中用途 |
|------|------|----------|-------------|
| 二进制（Binary） | 2 | 0,1 | 机器内部直接使用 |
| 八进制（Octal） | 8 | 0~7 | 程序设计中的简写 |
| 十进制（Decimal） | 10 | 0~9 | 人类日常使用 |
| 十六进制（Hexadecimal） | 16 | 0~9,A~F | 地址、颜色码等 |

### 数值展开通式

对于 $r$ 进制数 $N$，其按权展开为：

$$N = \sum_{i=m}^{n} d_i \times r^i$$

其中 $d_i$ 为第 $i$ 位的数字，$r$ 为基数，$i$ 从右到左依次增大（小数部分为负）。

**示例**：二进制数 $1011.101_2$ 的展开：

$$1 \times 2^3 + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0 + 1 \times 2^{-1} + 0 \times 2^{-2} + 1 \times 2^{-3}$$

$$= 8 + 0 + 2 + 1 + 0.5 + 0 + 0.125 = 11.625_{10}$$

## 原理分析

### 二进制（Binary）
- 基数 $r = 2$，每位取值为 0 或 1
- 位权从低位到高位依次为 $2^0, 2^1, 2^2, \ldots$
- 计算机内部一切数据（数值、字符、指令）最终都以二进制形式存储
- 电子元件容易实现：高电平=1，低电平=0

### 八进制（Octal）
- 基数 $r = 8$，每位取值 0~7
- 与二进制关系：**1位八进制 = 3位二进制**
- 例：$753_8 = 111\ 101\ 011_2$

### 十进制（Decimal）
- 基数 $r = 10$，每位取值 0~9
- 人类最熟悉的计数方式
- 计算机中通常用 BCD 码来表示十进制数

### 十六进制（Hexadecimal）
- 基数 $r = 16$，每位取值 0~9, A~F（A=10, B=11, ..., F=15）
- 与二进制关系：**1位十六进制 = 4位二进制**
- 广泛用于内存地址、颜色码（如 `#FF0000`）

## 直观理解

### 生活类比
- **十进制**就像我们用10个手指头计数，满了10就进位
- **二进制**就像一个只有开/关两种状态的开关系统
- **时钟**：秒逢60进位，分逢60进位，时逢12/24进位——这就是非十进制的例子
- **十二进制**：一打鸡蛋=12个

### 记忆技巧
1. **二进制位权序列**：1, 2, 4, 8, 16, 32, 64, 128（每次乘2）
2. **十六进制与二进制**：记住4位二进制对应1位十六进制
3. **八进制与二进制**：记住3位二进制对应1位八进制
4. 快速判断一个二进制数的值：看哪些位为1，把对应的位权加起来

### 位权记忆口诀
```
2^0=1, 2^1=2, 2^2=4, 2^3=8
2^4=16, 2^5=32, 2^6=64, 2^7=128
2^8=256, 2^9=512, 2^10=1024
```

## 知识关联

### 跨章节联系
- **第一章 计算机系统概述**：计算机内部所有数据以二进制表示是冯·诺依曼体系结构的核心特征之一
- **存储器章节**：内存地址常用十六进制表示
- **指令系统**：操作码、地址码均为二进制编码
- **数据通路**：数据在总线上传输时以二进制位串形式

### 常考组合
1. 进制转换（十进制↔二进制↔十六进制）是后续所有计算的基础
2. 二进制→补码运算需要熟练掌握二进制
3. 浮点数表示需要理解位权和科学计数法

### 易错陷阱
1. **书写混淆**：不要将不同进制的数混写。如 $101$ 在二进制是5，在十进制是101
2. **下标遗漏**：表示数值时必须标明基数，如 $101_2$ 或 $(101)_2$
3. **八进制/十六进制的误读**：$10_8 = 8_{10}$，$10_{16} = 16_{10}$，不要误认为是十进制的10
4. **小数位权为负指数**：小数点后第 $k$ 位的权值是 $r^{-k}$，不是 $r^{k}$

### 408考试要点
- 进位计数制的定义和性质属于基础概念，常在选择题中出现
- 熟练的进制转换能力是完成后续补码计算、浮点数计算的前提
- 理解"位权"的概念，是理解原码、补码等编码方式的理论基础
