# 02_二进制与十六进制转换

## 核心概念

### 核心关系
**1位十六进制数字 = 4位二进制数字**，这是因为 $16 = 2^4$。

同理，**1位八进制数字 = 3位二进制数字**，因为 $8 = 2^3$。

### 转换对照表（必记）

| 十六进制 | 二进制 | 十六进制 | 二进制 |
|----------|--------|----------|--------|
| 0 | 0000 | 8 | 1000 |
| 1 | 0001 | 9 | 1001 |
| 2 | 0010 | A | 1010 |
| 3 | 0011 | B | 1011 |
| 4 | 0100 | C | 1100 |
| 5 | 0101 | D | 1101 |
| 6 | 0110 | E | 1110 |
| 7 | 0111 | F | 1111 |

### 考试重点
- 这是408中极其高频的基础操作，几乎所有计算题都可能涉及
- 转换速度直接影响答题效率

## 原理分析

### 二进制 → 十六进制（从右向左，每4位一组）

**步骤**：
1. 从小数点开始，向左每4位分为一组（整数部分）
2. 从小数点开始，向右每4位分为一组（小数部分）
3. 不足4位的在最外侧补0
4. 每组替换为对应的十六进制数字

**例题1**：$10110110_2 = ?_{16}$

$$1011\ 0110 \xrightarrow{每4位一组} B\ 6$$

所以 $10110110_2 = \text{B6}_{16}$

**例题2**：$11011.101_2 = ?_{16}$

整数部分：$1\ 1011$ → 补0为 $0001\ 1011$ → $1\text{B}$

小数部分：$101$ → 补0为 $1010$ → $\text{A}$

所以 $11011.101_2 = 1\text{B}.\text{A}_{16}$

### 十六进制 → 二进制（每位展开为4位）

**步骤**：
1. 每位十六进制数字替换为4位二进制
2. 整数部分最高位的0可以省略，小数部分最低位的0可以省略

**例题3**：$\text{2F.C}_{16} = ?_2$

$$2 \to 0010, \quad F \to 1111, \quad C \to 1100$$

$$\text{2F.C}_{16} = 10\ 1111.11_2 \quad (\text{去掉前导0和末尾0})$$

### 二进制 ↔ 八进制

原理相同：**每3位二进制对应1位八进制**

**例题4**：$10110110_2 = ?_8$

$$10\ 110\ 110 \to 2\ 6\ 6$$

所以 $10110110_2 = 266_8$

**例题5**：$753_8 = ?_2$

$$7 \to 111, \quad 5 \to 101, \quad 3 \to 011$$

$$753_8 = 111\ 101\ 011_2$$

## 直观理解

### 记忆技巧

1. **十六进制记忆口诀**：
   - 0~7的二进制：从 $0000$ 到 $0111$，最高位都是0
   - 8~F的二进制：从 $1000$ 到 $1111$，最高位都是1
   - 换句话说：8~F的二进制就是0~7的二进制最高位取反

2. **分组口诀**：
   - 十六进制：4位一组（因为 $2^4 = 16$）
   - 八进制：3位一组（因为 $2^3 = 8$）
   - 记住"指数=每组位数"

3. **补零方向**：
   - 整数部分：在**左边**补0（不影响数值）
   - 小数部分：在**右边**补0（不影响数值）

### 常见错误
- 整数分组**从左往右**分是错误的！必须**从右往左**（从低位开始）
- 小数分组**从右往左**是错误的！必须**从左往右**（从低位开始）
- 记忆方法：分组都是从**小数点向外**扩展

## 知识关联

### 跨章节联系
- **存储器寻址**：内存地址几乎都用十六进制表示，如 `0x7FFF FFFF`
- **浮点数表示**：IEEE 754格式常用十六进制简写
- **指令编码**：机器指令用十六进制/二进制表示
- **网络**：MAC地址用十六进制表示（如 `00:1A:2B:3C:4D:5E`）

### 408考试应用
- 进制转换是几乎所有计算题的第一步
- 补码运算需要频繁在二进制和十进制之间转换
- 浮点数的十六进制表示需要转为二进制来分析各字段
- 校验码（海明码、CRC）的计算也需要进制转换

### 易错陷阱
1. **分组起始位置**：整数从右向左，小数从左向右
2. **前导零和末尾零**：书写结果时通常省略，但计算过程中不能省略
3. **A~F大小写**：考试中大小写均可，但要统一
4. **八进制分组**：3位一组，不是4位；不要与十六进制混淆
