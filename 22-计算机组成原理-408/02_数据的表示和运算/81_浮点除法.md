# 81_浮点除法

## 核心概念

### 浮点除法的公式

设 $A = M_A \times 2^{E_A}$，$B = M_B \times 2^{E_B}$（$B \neq 0$）

$$A \div B = \frac{M_A}{M_B} \times 2^{(E_A - E_B)}$$

### 浮点除法的步骤

**第一步：阶码相减**

$$E_{结果} = E_A - E_B$$

**第二步：尾数相除**

$$M_{结果} = M_A \div M_B$$

使用定点除法（恢复余数法或加减交替法）。

**第三步：规格化**

尾数商可能需要规格化：
- 如果 $|M_A| < |M_B|$，商的绝对值 $< 1$，可能已规格化
- 如果 $|M_A| \geq |M_B|$，商的绝对值 $\geq 1$，需要右规

实际上，两个规格化数相除：
- 商的范围 $[1/2, 2)$
- 如果 $\geq 1$，需右移 1 位，阶码加 1

**第四步：舍入**

除法结果可能有余数，需要根据精度要求舍入。

**第五步：溢出判断和除零检查**

- **除零检查**：若 $B = 0$，产生异常
- **上溢**：$E_{结果} > E_{max}$
- **下溢**：$E_{结果} < E_{min}$

### 浮点除法流程图

```
开始
  │
检查 B ≠ 0？── 否 → 除零异常
  │是
阶码相减：E = E_A - E_B
  │
尾数相除：M = M_A ÷ M_B
  │
规格化
  │
舍入
  │
溢出判断
  │
结束
```

### 示例

$A = 0.1100 \times 2^5$，$B = 0.1010 \times 2^2$

**阶码相减**：$E = 5 - 2 = 3$

**尾数相除**：$0.1100 \div 0.1010$

用加减交替法：
$0.1100 \div 0.1010$

ACC = 01100, X = 01010

| 轮次 | 操作 | ACC | 商 |
|------|------|-----|-----|
| 1 | ACC - X | 00010 | 1 |
| 1 | 左移 | 00100 | 1 |
| 2 | ACC - X | 11010 | 0 |
| 2 | ACC + X | 00100 | 0 |
| 2 | 左移 | 01000 | 10 |
| 3 | ACC - X | 00110 | 1 |
| 3 | 左移 | 01100 | 101 |
| 4 | ACC - X | 00010 | 1 |

商约为 $0.1001$（近似）

更精确：$0.1100 / 0.1010 = 0.75 / 0.625 = 1.2 = 1.00110011..._2$

商 $> 1$，需要右规。

**规格化**：$1.0011...$ 右移 1 位：$0.1001...$，阶码加 1：$3 + 1 = 4$

**结果**：$A \div B \approx 0.1001 \times 2^4$

验证：$A = 0.75 \times 32 = 24$，$B = 0.625 \times 4 = 2.5$，$A / B = 9.6$

$0.1001 \times 16 = 0.5625 \times 16 = 9$（近似值，余数部分省略）

### 除零检查

若 $B = 0$（$M_B = 0$ 或阶码太小）：
- $A / 0 = +\infty$（$A > 0$）
- $A / 0 = -\infty$（$A < 0$）
- $0 / 0 = $ NaN

## 直观理解

### 记忆要点

- **浮点除法 = 阶码相减 + 尾数相除**
- **先检查除数是否为零**
- **商可能 $\geq 1$，需要右规**
- **阶码相减可能产生上溢或下溢**

### 与浮点乘法的对比

| 特征 | 浮点乘法 | 浮点除法 |
|------|---------|---------|
| 阶码 | 相加 | 相减 |
| 尾数 | 相乘 | 相除 |
| 规格化 | 最多左移1位 | 最多右移1位 |
| 特殊检查 | 无 | 除零 |
