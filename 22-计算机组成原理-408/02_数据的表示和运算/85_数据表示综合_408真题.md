# 85_数据表示综合_408真题

## 核心概念

本文件汇集 408 真题中关于数据表示和运算的综合题目，涵盖定点、浮点、编码、运算等全部知识点。

## 真题精选

### 真题 1（编码与表示）

已知 $[x]_{补} = 1.1011$，$[y]_{补} = 0.1101$，求 $[x - y]_{补}$。

**解答**：

$[x - y]_{补} = [x]_{补} + [-y]_{补}$

$[-y]_{补} = 1.0011$（$[y]_{补}$ 取反加 1：$0.1101 \to 1.0010 \to 1.0011$）

$[x]_{补} + [-y]_{补} = 1.1011 + 1.0011 = 0.1110$（溢出进位丢弃）

$[x - y]_{补} = 0.1110$

验证：$x = -0.0101 = -5/16$，$y = +0.1101 = 13/16$，$x - y = -18/16 = -9/8$

$0.1110$（补码，正数）$= 0.875 = 14/16$... 不对。

重新算：$[x]_{补} = 1.1011$，$x = -(0.0100 + 0.0001) = -0.0101 = -5/16$

$[-y]_{补} = -(0.1101)$ 的补码 $= 1.0011$

$1.1011 + 1.0011 = (1)0.1110$（进位 1 丢弃）$= 0.1110$

$0.1110 = 14/16 = 7/8$

但 $x - y = -5/16 - 13/16 = -18/16$，溢出了 1 位小数的表示范围 $[-1, 1-2^{-4}]$。

这里发生了**溢出**，结果需要更多位来表示。

### 真题 2（原码一位乘）

用原码一位乘法计算 $x \times y$，其中 $x = 0.1101$，$y = 0.1011$。

**解答**：

符号位：$0 \oplus 0 = 0$

数值部分（X = 1101，ACC = 0000，MQ = 1011）：

| 轮次 | MQ最低位 | 操作 | ACC | MQ |
|------|---------|------|-----|-----|
| 初始 | — | — | 0000 | 1011 |
| 1 | 1 | +X | 1101 | 1011 |
| 1 | — | 右移 | 0110 | 1101 |
| 2 | 1 | +X | 10011 | 1101 |
| 2 | — | 右移 | 01001 | 1110 |
| 3 | 0 | 不加 | 01001 | 1110 |
| 3 | — | 右移 | 00100 | 1111 |
| 4 | 1 | +X | 10001 | 1111 |
| 4 | — | 右移 | 01000 | 1111 |

结果：$0.10001111$

### 真题 3（IEEE 754 浮点数）

IEEE 754 单精度浮点数 `0x42F00000` 表示的十进制数是多少？

**解答**：

```
0x42F00000 = 0100 0010 1111 0000 0000 0000 0000 0000

S = 0
阶码 = 10000101 = 133
E = 133 - 127 = 6
尾数 = 1110000...0
实际尾数 = 1.111 = 1 + 0.5 + 0.25 + 0.125 = 1.875

N = 1.875 × 2^6 = 1.875 × 64 = 120.0
```

### 真题 4（浮点加法）

设浮点数格式：阶码 3 位补码，尾数 5 位原码（含数符），$R = 2$。

$A = 0.1011 \times 2^{010}$，$B = 0.1101 \times 2^{000}$

求 $A + B$。

**解答**：

**（1）对阶**
$E_A = 010 = +2$，$E_B = 000 = 0$
$\Delta E = 2$，$B$ 右移 2 位：$0.1101 \to 0.001101 \to 0.0011$（截断）
$B$ 阶码改为 $010$

**（2）尾数加法**
$0.1011 + 0.0011 = 0.1110$

**（3）规格化**
$0.1110$：最高数值位为 1，已规格化。

**（4）溢出判断**
阶码 $+2$ 在 $[-4, +3]$ 范围内。

结果：$A + B = 0.1110 \times 2^{010}$

验证：$A = 0.6875 \times 4 = 2.75$，$B = 0.8125 \times 1 = 0.8125$
$A + B = 3.5625$
$0.1110 \times 4 = 0.875 \times 4 = 3.5$（截断误差）

### 真题 5（Booth 算法）

用 Booth 算法计算 $[x \times y]_{补}$，其中 $[x]_{补} = 1.0011$，$[y]_{补} = 0.1010$。

**解答**：

$[x]_{补} = 10011$（5 位），$[-x]_{补} = 01101$

ACC = 00000，MQ = 1010，附加位 = 0

| 轮次 | $(y_n, y_{n+1})$ | 操作 | ACC | MQ | 附加位 |
|------|-------------------|------|-----|-----|--------|
| 初始 | — | — | 00000 | 1010 | 0 |
| 1 | (0,0) | 不加，右移 | 00000 | 0101 | 0 |
| 2 | (1,0) | +$[-x]_补$=01101，右移 | 01101→00110 | 1010 | 1 |
| 3 | (0,1) | +$[x]_补$=10011，右移 | 11001→11100 | 1101 | 0 |
| 4 | (1,0) | +$[-x]_补$=01101，右移 | 01001→00100 | 1110 | 1 |

结果：$[x \times y]_{补} = 001001110$（9 位）

### 真题 6（加法器）

用 4 位超前进位加法器计算 $A = 1010$，$B = 0110$，$C_{in} = 0$。

**解答**：

$G_0 = 0, P_0 = 0 \oplus 1 = 1$
$G_1 = 1, P_1 = 1 \oplus 1 = 0$
$G_2 = 0, P_2 = 1 \oplus 0 = 1$
$G_3 = 1, P_3 = 0 \oplus 0 = 0$

$C_0 = 0 + 1 \cdot 0 = 0$
$C_1 = 1 + 0 = 1$
$C_2 = 0 + 1 \cdot 1 + 1 \cdot 1 \cdot 0 = 1$
$C_3 = 1 + 0 = 1$

$S_0 = P_0 \oplus 0 = 1$
$S_1 = P_1 \oplus C_0 = 0$
$S_2 = P_2 \oplus C_1 = 1 \oplus 1 = 0$
$S_3 = P_3 \oplus C_2 = 0 \oplus 1 = 1$

$S = 1001$, $C_{out} = 1$

验证：$1010 + 0110 = 10000$（$10 + 6 = 16$）✓

### 真题 7（综合概念）

以下说法正确的是：

A. 原码乘法中，符号位参与运算
B. 补码除法中，商的末位恒置 1
C. 浮点加法中，对阶时大阶向小阶看齐
D. IEEE 754 中，规格化数的尾数隐含 0

**答案**：B

A 错误：原码乘法符号位单独异或处理
B 正确：补码除法末位恒置 1
C 错误：小阶向大阶看齐
D 错误：隐含 1，不是 0
