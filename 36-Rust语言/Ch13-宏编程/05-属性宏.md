# 属性宏

## 一、概念说明

属性宏通过自定义属性扩展功能，可以修改或包装被标注的项。

```rust
#[route(GET, "/")]
fn index() -> String { "首页".to_string() }
```

## 二、具体用法

### 2.1 基本属性宏

```rust
use proc_macro::TokenStream;

#[proc_macro_attribute]
pub fn log_call(_attr: TokenStream, item: TokenStream) -> TokenStream {
    let input = syn::parse_macro_input!(item as syn::ItemFn);
    let name = &input.sig.ident;
    let block = &input.block;
    let vis = &input.vis;
    let sig = &input.sig;

    let gen = quote::quote! {
        #vis #sig {
            println!("调用函数: {}", stringify!(#name));
            let result = (|| #block)();
            println!("函数 {} 返回", stringify!(#name));
            result
        }
    };

    gen.into()
}
```

### 2.2 带参数的属性宏

```rust
#[proc_macro_attribute]
pub fn retry(attr: TokenStream, item: TokenStream) -> TokenStream {
    let max_retries: syn::LitInt = syn::parse(attr).unwrap();
    let input = syn::parse_macro_input!(item as syn::ItemFn);

    let gen = quote::quote! {
        #input

        fn retry_wrapper() -> Result<_, String> {
            for i in 0..#max_retries {
                match #input {
                    Ok(v) => return Ok(v),
                    Err(e) if i == #max_retries - 1 => return Err(e),
                    _ => continue,
                }
            }
            unreachable!()
        }
    };

    gen.into()
}
```

## 三、注意事项与常见陷阱

1. **属性语法**：属性参数需符合 Rust 语法
2. **上下文保留**：保留原有的可见性、泛型等信息
3. **错误报告**：使用 syn::Error 报告编译错误
4. **文档继承**：考虑保留或处理文档属性
5. **IDE 支持**：属性宏的 IDE 支持可能有限
