# 函数宏

## 一、概念说明

函数宏类似于函数但接收 TokenStream 作为输入，可以实现更灵活的代码生成。

```rust
// 类似 println! 但自定义
my_macro!(input tokens here);
```

## 二、具体用法

### 2.1 基本函数宏

```rust
use proc_macro::TokenStream;

#[proc_macro]
pub fn make_answer(input: TokenStream) -> TokenStream {
    "fn answer() -> u32 { 42 }".parse().unwrap()
}
```

### 2.2 带参数的函数宏

```rust
#[proc_macro]
pub fn sql_query(input: TokenStream) -> TokenStream {
    let query = input.to_string();

    // 简单的 SQL 验证
    if !query.to_lowercase().starts_with("select") {
        panic!("只支持 SELECT 查询");
    }

    let gen = quote::quote! {
        fn execute_query() -> String {
            #query.to_string()
        }
    };

    gen.into()
}
```

## 三、注意事项与常见陷阱

1. **输入解析**：TokenStream 解析需要 syn crate
2. **错误处理**：提供清晰的编译错误信息
3. **字符串处理**：注意字符串中的特殊字符
4. **性能**：宏展开在编译时，不影响运行时性能
5. **测试**：为宏编写测试用例
