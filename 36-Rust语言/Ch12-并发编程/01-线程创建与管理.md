# 线程创建与管理

## 一、概念说明

Rust 通过标准库 `std::thread` 模块提供多线程支持。Rust 的所有权系统和类型系统在编译时保证线程安全，防止数据竞争。

```rust
use std::thread;

let handle = thread::spawn(|| {
    println!("新线程执行中");
});

handle.join().unwrap(); // 等待线程完成
```

## 二、具体用法

### 2.1 创建线程

```rust
use std::thread;

// 基本线程创建
let handle = thread::spawn(|| {
    for i in 1..5 {
        println!("线程: {}", i);
    }
});

// 主线程继续执行
for i in 1..3 {
    println!("主线程: {}", i);
}

handle.join().unwrap();
```

### 2.2 传递数据

```rust
use std::thread;

let data = vec![1, 2, 3, 4, 5];

// move 转移所有权到新线程
let handle = thread::spawn(move || {
    println!("数据: {:?}", data);
    data.iter().sum::<i32>()
});

// data 已被移动，主线程不能再使用
let sum = handle.join().unwrap();
println!("总和: {}", sum);
```

### 2.3 线程配置

```rust
use std::thread;

let builder = thread::Builder::new()
    .name("工作线程".into())
    .stack_size(4 * 1024 * 1024); // 4MB 栈

let handle = builder.spawn(|| {
    println!("线程名: {:?}", thread::current().name());
}).unwrap();

handle.join().unwrap();
```

### 2.4 线程局部存储

```rust
use std::cell::RefCell;

thread_local! {
    static THREAD_DATA: RefCell<Vec<i32>> = RefCell::new(vec![]);
}

THREAD_DATA.with(|data| {
    data.borrow_mut().push(42);
});

thread::spawn(|| {
    THREAD_DATA.with(|data| {
        data.borrow_mut().push(100);
    });
}).join().unwrap();
```

## 三、注意事项与常见陷阱

1. **join 重要性**：不调用 join 可能导致主线程提前退出
2. **所有权转移**：跨线程数据需要 move 或 Send trait
3. **线程开销**：创建线程有开销，考虑使用线程池
4. **panic 传播**：子线程 panic 通过 join 传播到主线程
5. **无序执行**：线程执行顺序不确定，不要依赖特定顺序
