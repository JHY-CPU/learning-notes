# 并发安全数据结构

## 一、概念说明

Rust 标准库提供了一些并发安全的数据结构，第三方 crate 如 `crossbeam` 提供了更多高效选择。

```rust
use std::sync::mpsc;
use crossbeam::queue::SegQueue;

// 无锁队列
let queue = SegQueue::new();
queue.push(1);
queue.push(2);
```

## 二、具体用法

### 2.1 crossbeam 无锁队列

```rust
// Cargo.toml: crossbeam = "0.8"
use crossbeam::queue::{ArrayQueue, SegQueue};

// 有界队列
let bounded = ArrayQueue::new(100);
bounded.push(1).unwrap();
let val = bounded.pop();

// 无界队列
let unbounded = SegQueue::new();
unbounded.push(1);
unbounded.push(2);
while let Some(val) = unbounded.pop() {
    println!("{}", val);
}
```

### 2.2 crossbeam scope

```rust
use crossbeam::thread;

fn scoped_threads() {
    let data = vec![1, 2, 3, 4, 5];

    thread::scope(|s| {
        s.spawn(|_| {
            println!("线程1: {:?}", data);
        });
        s.spawn(|_| {
            println!("线程2: {:?}", data);
        });
    }).unwrap();
    // data 在此处仍可用
}
```

### 2.3 DashMap 并发哈希表

```rust
// Cargo.toml: dashmap = "5"
use dashmap::DashMap;
use std::sync::Arc;

fn dashmap_example() {
    let map = Arc::new(DashMap::new());
    let map_clone = Arc::clone(&map);

    let handle = std::thread::spawn(move || {
        map_clone.insert("key1", 100);
    });

    map.insert("key2", 200);
    handle.join().unwrap();

    println!("{:?}", map.get("key1"));
}
```

## 三、注意事项与常见陷阱

1. **无锁 vs 有锁**：无锁数据结构在高竞争下性能更好
2. **正确性保证**：无锁结构可能比锁更难正确使用
3. **内存管理**：无锁结构的内存回收更复杂
4. **适用场景**：简单场景用标准库，高性能需求用 crossbeam
5. **测试挑战**：并发数据结构需要大量压力测试
