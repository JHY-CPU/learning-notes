# 智能指针实战案例

## 一、概念说明

综合运用各种智能指针解决实际问题，包括观察者模式、缓存系统、树形结构等常见场景。

```rust
use std::rc::{Rc, Weak};
use std::cell::RefCell;
use std::collections::HashMap;

// 综合案例：事件系统
struct EventEmitter {
    listeners: HashMap<String, Vec<Weak<dyn Listener>>>,
}
```

## 二、具体用法

### 2.1 观察者模式

```rust
use std::rc::{Rc, Weak};
use std::cell::RefCell;

trait Listener {
    fn on_event(&self, event: &str);
}

struct EventEmitter {
    listeners: RefCell<Vec<Weak<dyn Listener>>>,
}

impl EventEmitter {
    fn new() -> Self {
        EventEmitter { listeners: RefCell::new(vec![]) }
    }

    fn subscribe(&self, listener: Weak<dyn Listener>) {
        self.listeners.borrow_mut().push(listener);
    }

    fn emit(&self, event: &str) {
        let mut listeners = self.listeners.borrow_mut();
        listeners.retain(|weak| {
            if let Some(listener) = weak.upgrade() {
                listener.on_event(event);
                true
            } else {
                false
            }
        });
    }
}
```

### 2.2 LRU 缓存

```rust
use std::collections::HashMap;
use std::rc::{Rc, Weak};
use std::cell::RefCell;

struct LRUNode {
    key: String,
    value: String,
    prev: RefCell<Option<Rc<LRUNode>>>,
    next: RefCell<Option<Rc<LRUNode>>>,
}

struct LRUCache {
    capacity: usize,
    map: RefCell<HashMap<String, Rc<LRUNode>>>,
    head: RefCell<Option<Rc<LRUNode>>>,
    tail: RefCell<Option<Weak<LRUNode>>>,
}
```

### 2.3 文件系统树

```rust
use std::rc::{Rc, Weak};
use std::cell::RefCell;
use std::collections::HashMap;

struct Directory {
    name: String,
    parent: Option<Weak<Directory>>,
    children: RefCell<HashMap<String, Rc<Directory>>>,
    files: RefCell<Vec<String>>,
}

impl Directory {
    fn new(name: &str, parent: Option<Weak<Directory>>) -> Rc<Self> {
        Rc::new(Directory {
            name: name.to_string(),
            parent,
            children: RefCell::new(HashMap::new()),
            files: RefCell::new(vec![]),
        })
    }

    fn add_subdir(self: &Rc<Self>, name: &str) -> Rc<Self> {
        let child = Directory::new(name, Some(Rc::downgrade(self)));
        self.children.borrow_mut().insert(name.to_string(), Rc::clone(&child));
        child
    }
}
```

## 三、注意事项与常见陷阱

1. **引用管理**：父子关系中子对父使用 Weak
2. **清理机制**：定期清理无效的 Weak 引用
3. **性能考虑**：大量小对象考虑使用 arena 分配器
4. **调试技巧**：实现 Debug trait 打印内部状态
5. **测试覆盖**：智能指针逻辑需要充分测试
