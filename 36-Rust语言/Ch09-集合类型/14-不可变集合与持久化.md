# 不可变集合与持久化

## 一、概念说明

不可变集合是创建后不可修改的数据结构。修改操作会返回新的集合，原集合保持不变。这种模式在函数式编程中常见，称为"持久化数据结构"。

```rust
// 使用 im crate 实现不可变集合
// Cargo.toml: im = "15"
use im::{Vector, HashMap as ImHashMap};

let v1 = vector![1, 2, 3];
let v2 = v1.push_back(4); // v1不变，返回新的v2
```

## 二、具体用法

### 2.1 Copy-on-Write模式

```rust
use std::rc::Rc;

#[derive(Clone)]
struct ImmutableVec<T: Clone> {
    data: Rc<Vec<T>>,
}

impl<T: Clone> ImmutableVec<T> {
    fn new() -> Self {
        ImmutableVec { data: Rc::new(Vec::new()) }
    }

    fn push(&self, item: T) -> Self {
        let mut new_data = (*self.data).clone();
        new_data.push(item);
        ImmutableVec { data: Rc::new(new_data) }
    }

    fn get(&self, index: usize) -> Option<&T> {
        self.data.get(index)
    }
}
```

### 2.2 分层Map实现

```rust
use std::collections::HashMap;
use std::sync::Arc;

#[derive(Clone)]
struct PersistentMap<K: Eq + std::hash::Hash + Clone, V: Clone> {
    base: Arc<HashMap<K, V>>,
    changes: HashMap<K, Option<V>>,
}

impl<K: Eq + std::hash::Hash + Clone, V: Clone> PersistentMap<K, V> {
    fn set(&self, key: K, value: V) -> Self {
        let mut new = self.clone();
        new.changes.insert(key, Some(value));
        new
    }

    fn remove(&self, key: K) -> Self {
        let mut new = self.clone();
        new.changes.insert(key, None);
        new
    }

    fn get(&self, key: &K) -> Option<&V> {
        if let Some(change) = self.changes.get(key) {
            return change.as_ref();
        }
        self.base.get(key)
    }
}
```

### 2.3 使用 im crate

```rust
use im::{Vector, OrdMap, HashSet as ImHashSet};

// 不可变向量
let v1 = vector![1, 2, 3];
let v2 = v1.push_back(4);
let v3 = v2.update(0, 10);
// v1=[1,2,3], v2=[1,2,3,4], v3=[10,2,3,4]

// 不可变有序Map
let m1 = ordmap! {1 => "一", 2 => "二"};
let m2 = m1.update(3, "三");
// m1={1:"一",2:"二"}, m2={1:"一",2:"二",3:"三"}
```

## 三、注意事项与常见陷阱

1. **性能权衡**：不可变集合使用结构共享，修改操作通常 O(log n)
2. **内存使用**：共享结构减少内存，但引用计数有开销
3. **并发安全**：不可变天然线程安全，无需锁
4. **适用场景**：函数式风格、undo/redo、并发场景
5. **选择库**：im crate 是 Rust 中最成熟的不可变集合库
