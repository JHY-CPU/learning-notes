## 二、具体用法

### 2.1 LRU 缓存实现

```rust
use std::collections::{HashMap, LinkedList};

struct LRUCache<K: Eq + std::hash::Hash + Clone, V> {
    capacity: usize,
    map: HashMap<K, (V, std::collections::linked_list::Cursor<()>)>,
    order: LinkedList<K>,
}

impl<K: Eq + std::hash::Hash + Clone, V> LRUCache<K, V> {
    fn new(capacity: usize) -> Self {
        LRUCache {
            capacity,
            map: HashMap::new(),
            order: LinkedList::new(),
        }
    }

    fn get(&mut self, key: &K) -> Option<&V> {
        self.map.get(key).map(|(v, _)| v)
    }

    fn put(&mut self, key: K, value: V) {
        if self.map.len() >= self.capacity {
            if let Some(old_key) = self.order.pop_back() {
                self.map.remove(&old_key);
            }
        }
        self.order.push_front(key.clone());
        self.map.insert(key, (value, self.order.cursor_front()));
    }
}
```

### 2.2 图算法应用

```rust
use std::collections::{HashMap, HashSet, VecDeque};

struct Graph {
    adj: HashMap<String, Vec<String>>,
}

impl Graph {
    fn new() -> Self {
        Graph { adj: HashMap::new() }
    }

    fn add_edge(&mut self, from: &str, to: &str) {
        self.adj.entry(from.to_string()).or_default().push(to.to_string());
    }

    fn bfs(&self, start: &str) -> Vec<String> {
        let mut visited = HashSet::new();
        let mut queue = VecDeque::new();
        let mut result = Vec::new();

        queue.push_back(start.to_string());
        visited.insert(start.to_string());

        while let Some(node) = queue.pop_front() {
            result.push(node.clone());
            if let Some(neighbors) = self.adj.get(&node) {
                for neighbor in neighbors {
                    if !visited.contains(neighbor) {
                        visited.insert(neighbor.clone());
                        queue.push_back(neighbor.clone());
                    }
                }
            }
        }
        result
    }
}
```

### 2.3 数据聚合分析

```rust
fn analyze_logs(logs: &[(&str, &str, u64)]) -> Vec<(String, u64, f64)> {
    let mut stats: HashMap<&str, Vec<u64>> = HashMap::new();

    for (service, _, duration) in logs {
        stats.entry(*service).or_default().push(*duration);
    }

    let mut result: Vec<_> = stats.into_iter()
        .map(|(svc, durations)| {
            let sum: u64 = durations.iter().sum();
            let avg = sum as f64 / durations.len() as f64;
            (svc.to_string(), durations.len() as u64, avg)
        })
        .collect();

    result.sort_by(|a, b| b.2.partial_cmp(&a.2).unwrap());
    result
}
```

## 三、注意事项与常见陷阱

1. **组合使用**：实际项目中通常组合多种集合
2. **性能测量**：用 criterion crate 做基准测试
3. **内存监控**：大集合使用 memory_stats 观察内存占用
4. **边界测试**：空集合、单元素、大数据量的测试
5. **选择标准**：先实现功能，再根据性能需求调整集合类型
