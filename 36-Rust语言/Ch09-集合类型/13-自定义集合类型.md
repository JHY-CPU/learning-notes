# 自定义集合类型

## 一、概念说明

Rust 允许通过实现标准库的 trait 来创建自定义集合类型。常用 trait 包括 `Iterator`、`IntoIterator`、`FromIterator`、`Extend` 等，让自定义类型无缝融入 Rust 生态。

```rust
struct RingBuffer<T> {
    data: Vec<Option<T>>,
    capacity: usize,
    head: usize,
    tail: usize,
    count: usize,
}
```

## 二、具体用法

### 2.1 实现环形缓冲区

```rust
impl<T: Clone> RingBuffer<T> {
    fn new(capacity: usize) -> Self {
        RingBuffer {
            data: vec![None; capacity],
            capacity,
            head: 0,
            tail: 0,
            count: 0,
        }
    }

    fn push(&mut self, item: T) {
        self.data[self.tail] = Some(item);
        self.tail = (self.tail + 1) % self.capacity;
        if self.count < self.capacity {
            self.count += 1;
        } else {
            self.head = (self.head + 1) % self.capacity;
        }
    }

    fn pop(&mut self) -> Option<T> {
        if self.count == 0 {
            return None;
        }
        let item = self.data[self.head].take();
        self.head = (self.head + 1) % self.capacity;
        self.count -= 1;
        item
    }
}
```

### 2.2 实现 IntoIterator

```rust
impl<T: Clone> IntoIterator for RingBuffer<T> {
    type Item = T;
    type IntoIter = RingBufferIter<T>;

    fn into_iter(self) -> Self::IntoIter {
        RingBufferIter {
            buffer: self,
            index: 0,
        }
    }
}

struct RingBufferIter<T> {
    buffer: RingBuffer<T>,
    index: usize,
}

impl<T: Clone> Iterator for RingBufferIter<T> {
    type Item = T;

    fn next(&mut self) -> Option<Self::Item> {
        if self.index >= self.buffer.count {
            return None;
        }
        let pos = (self.buffer.head + self.index) % self.buffer.capacity;
        self.index += 1;
        self.buffer.data[pos].clone()
    }
}
```

### 2.3 实现 FromIterator

```rust
impl<T: Clone> FromIterator<T> for RingBuffer<T> {
    fn from_iter<I: IntoIterator<Item = T>>(iter: I) -> Self {
        let items: Vec<T> = iter.into_iter().collect();
        let mut buf = RingBuffer::new(items.len());
        for item in items {
            buf.push(item);
        }
        buf
    }
}

// 使用 collect 创建
let buf: RingBuffer<i32> = vec![1, 2, 3, 4, 5].into_iter().collect();
```

### 2.4 实现 Extend

```rust
impl<T: Clone> Extend<T> for RingBuffer<T> {
    fn extend<I: IntoIterator<Item = T>>(&mut self, iter: I) {
        for item in iter {
            self.push(item);
        }
    }
}
```

## 三、注意事项与常见陷阱

1. **trait 要求**：实现 IntoIterator 需同时实现自定义迭代器类型
2. **泛型约束**：合理添加 `Clone`、`Debug` 等约束
3. **内存安全**：手动管理索引时注意溢出和边界检查
4. **Drop 实现**：自定义集合可能需要实现 Drop 清理资源
5. **测试覆盖**：自定义集合需充分测试边界条件
