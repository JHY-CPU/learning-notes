# 集合序列化与反序列化

## 一、概念说明

Rust 中通过 serde crate 实现集合的序列化与反序列化。serde 支持 JSON、YAML、TOML、MessagePack 等多种格式，几乎所有标准库集合类型都实现了 Serialize/Deserialize。

```rust
use serde::{Serialize, Deserialize};

#[derive(Serialize, Deserialize, Debug)]
struct User {
    name: String,
    scores: Vec<f64>,
    metadata: std::collections::HashMap<String, String>,
}
```

## 二、具体用法

### 2.1 JSON 序列化

```rust
use serde_json;
use std::collections::HashMap;

fn json_example() {
    let mut data = HashMap::new();
    data.insert("name", "张三");
    data.insert("age", "25");

    // 序列化为 JSON 字符串
    let json = serde_json::to_string(&data).unwrap();

    // 美化输出
    let pretty = serde_json::to_string_pretty(&data).unwrap();

    // 反序列化
    let parsed: HashMap<String, String> = serde_json::from_str(&json).unwrap();
}
```

### 2.2 自定义序列化

```rust
use serde::{Serialize, Deserialize, Serializer, Deserializer};

#[derive(Serialize, Deserialize)]
struct Config {
    #[serde(rename = "db_url")]
    database_url: String,

    #[serde(default)]
    max_connections: u32,

    #[serde(skip)]
    internal_state: String,

    #[serde(serialize_with = "serialize_sorted_vec")]
    items: Vec<String>,
}

fn serialize_sorted_vec<S>(items: &[String], serializer: S) -> Result<S::Ok, S::Error>
where
    S: Serializer,
{
    let mut sorted = items.to_vec();
    sorted.sort();
    sorted.serialize(serializer)
}
```

### 2.3 枚举序列化

```rust
use serde::{Serialize, Deserialize};

#[derive(Serialize, Deserialize, Debug)]
#[serde(tag = "type")]
enum Message {
    Text { content: String },
    Image { url: String, width: u32, height: u32 },
    Audio { url: String, duration: f64 },
}
```

## 三、注意事项与常见陷阱

1. **特征派生**：需同时派生 Serialize 和 Deserialize
2. **生命周期**：反序列化的引用需使用 `#[serde(borrow)]`
3. **版本兼容**：使用 `#[serde(default)]` 处理新增字段
4. **性能优化**：大数据量考虑 MessagePack 或 Bincode 格式
5. **类型安全**：枚举使用 `#[serde(tag = "...")]` 确保类型识别
