# 常用数据类型

## 一、概念说明

Rust 的类型系统非常丰富，包括标量类型（整数、浮点、布尔、字符）和复合类型（元组、数组）。

```rust
fn main() {
    // 整数类型
    let a: i8 = -128;       // 8 位有符号
    let b: u8 = 255;        // 8 位无符号
    let c: i32 = 42;        // 32 位有符号（默认）
    let d: i64 = 1000000;   // 64 位有符号
    let e: isize = 100;     // 指针大小有符号
    let f: usize = 100;     // 指针大小无符号

    // 浮点类型
    let x: f64 = 3.14;      // 64 位（默认）
    let y: f32 = 2.718;     // 32 位

    // 布尔类型
    let t: bool = true;
    let f: bool = false;

    // 字符类型（4 字节 Unicode）
    let c: char = 'z';
    let heart: char = '❤';
    let emoji: char = '😀';
}
```

## 二、复合类型

```rust
fn main() {
    // 元组
    let tup: (i32, f64, u8) = (500, 6.4, 1);
    let (x, y, z) = tup;  // 解构
    println!("第一个: {}", tup.0);

    // 单元元组
    let unit: () = ();

    // 数组（固定长度）
    let arr: [i32; 5] = [1, 2, 3, 4, 5];
    let zeros = [0; 10];  // 10 个 0
    let first = arr[0];   // 索引访问

    // 切片（引用数组的一部分）
    let slice: &[i32] = &arr[1..3];  // [2, 3]
    let slice: &[i32] = &arr[..];    // 整个数组
}
```

## 三、字符串类型

```rust
fn main() {
    // &str - 字符串切片（不可变引用）
    let s: &str = "hello";
    let s: &'static str = "hello";  // 字符串字面量是静态的

    // String - 可增长的字符串
    let mut s = String::new();
    let s = String::from("hello");
    let s = "hello".to_string();

    // 转换
    let s: &str = &String::from("hello");
    let s: String = "hello".to_string();
}
```

## 四、类型转换

```rust
fn main() {
    // as 转换（基本类型）
    let x: i32 = 5;
    let y: i64 = x as i64;
    let z: u8 = x as u8;

    // From/Into trait
    let s = String::from("hello");
    let bytes: Vec<u8> = s.into_bytes();

    // TryFrom/TryInto（可能失败）
    let x: i32 = 256;
    let y: Result<u8, _> = u8::try_from(x);  // 溢出，返回 Err

    // parse 字符串解析
    let num: i32 = "42".parse().unwrap();
    let num: f64 = "3.14".parse().unwrap();
}
```

## 五、Option 和 Result

```rust
fn main() {
    // Option<T> - 可能为空的值
    let some_value: Option<i32> = Some(42);
    let no_value: Option<i32> = None;

    match some_value {
        Some(v) => println!("值: {}", v),
        None => println!("空"),
    }

    // Result<T, E> - 可能出错的值
    let ok: Result<i32, &str> = Ok(42);
    let err: Result<i32, &str> = Err("出错了");

    match ok {
        Ok(v) => println!("成功: {}", v),
        Err(e) => println!("错误: {}", e),
    }
}
```

## 六、注意事项

1. **整数溢出**：debug 模式会 panic，release 模式环绕
2. **默认类型**：整数默认 i32，浮点数默认 f64
3. **字符串编码**：Rust 字符串是 UTF-8 编码
4. **数组越界**：运行时会 panic
5. **类型推断**：大多数情况下编译器能推断类型
