# 函数与控制流

## 一、概念说明

Rust 使用 `fn` 定义函数，函数参数必须标注类型，返回值使用 `->` 指定。Rust 中的表达式和语句有明确区分。

```rust
// 函数定义
fn add(x: i32, y: i32) -> i32 {
    x + y  // 表达式作为返回值（无分号）
}

fn main() {
    let result = add(5, 3);
    println!("5 + 3 = {}", result);
}
```

## 二、函数详解

```rust
// 无返回值函数
fn greet(name: &str) {
    println!("你好，{}！", name);
}

// 多参数函数
fn create_user(name: String, age: u32, email: String) -> User {
    User { name, age, email }
}

// 返回元组
fn divide(x: f64, y: f64) -> (f64, f64) {
    let quotient = x / y;
    let remainder = x % y;
    (quotient, remainder)
}

// 语句 vs 表达式
fn example() {
    // 语句 - 不返回值
    let x = 5;       // 语句
    let y = {        // 块表达式
        let inner = 3;
        inner + 1    // 表达式（无分号），返回 4
    };
}

// 高阶函数
fn apply(f: fn(i32) -> i32, x: i32) -> i32 {
    f(x)
}

fn double(x: i32) -> i32 { x * 2 }

fn main() {
    let result = apply(double, 5);
    println!("结果: {}", result);
}
```

## 三、条件控制

```rust
fn main() {
    let number = 6;

    // if-else
    if number % 4 == 0 {
        println!("能被 4 整除");
    } else if number % 3 == 0 {
        println!("能被 3 整除");
    } else {
        println!("不能被 4 或 3 整除");
    }

    // if 作为表达式
    let condition = true;
    let value = if condition { 5 } else { 6 };
    println!("value: {}", value);

    // match 匹配
    let number = 13;
    match number {
        1 => println!("一"),
        2 | 3 | 5 | 7 | 11 | 13 => println!("质数"),
        13..=19 => println!("青少年"),
        _ => println!("其他"),
    }

    // match 解构
    let point = (3, 5);
    match point {
        (0, y) => println!("在 Y 轴上: {}", y),
        (x, 0) => println!("在 X 轴上: {}", x),
        (x, y) => println!("坐标: ({}, {})", x, y),
    }

    // match 守卫
    let num = Some(4);
    match num {
        Some(x) if x < 5 => println!("小于 5: {}", x),
        Some(x) => println!("大于等于 5: {}", x),
        None => println!("空值"),
    }
}
```

## 四、循环控制

```rust
fn main() {
    // loop 无限循环
    let mut counter = 0;
    let result = loop {
        counter += 1;
        if counter == 10 {
            break counter * 2;  // 循环可以返回值
        }
    };
    println!("结果: {}", result);

    // while 循环
    let mut number = 3;
    while number != 0 {
        println!("{}!", number);
        number -= 1;
    }

    // for 循环
    let arr = [10, 20, 30, 40, 50];
    for element in arr {
        println!("值: {}", element);
    }

    // 范围循环
    for i in 0..5 {
        println!("索引: {}", i);
    }

    // 反向范围
    for i in (1..4).rev() {
        println!("倒计时: {}!", i);
    }

    // enumerate 带索引
    let fruits = ["苹果", "香蕉", "橙子"];
    for (index, fruit) in fruits.iter().enumerate() {
        println!("{}: {}", index, fruit);
    }

    // 循环标签（嵌套循环控制）
    'outer: for i in 0..3 {
        for j in 0..3 {
            if i == 1 && j == 1 {
                break 'outer;  // 跳出外层循环
            }
            println!("({}, {})", i, j);
        }
    }
}
```

## 五、注意事项

1. **表达式无分号**：函数最后一行表达式无分号则作为返回值
2. **条件必须是 bool**：Rust 不会自动转换非布尔值
3. **match 穷尽**：match 必须覆盖所有可能的值
4. **循环返回值**：loop 可以通过 break 返回值
5. **变量遮蔽**：在循环内可以遮蔽外部变量
