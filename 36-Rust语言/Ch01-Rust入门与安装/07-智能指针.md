# 智能指针

## 一、概念说明

智能指针是拥有数据并提供额外功能的指针类型。Rust 中最常用的智能指针有 Box、Rc 和 RefCell。

```rust
// 智能指针与普通引用的区别
/*
引用: 只借用数据
智能指针: 拥有数据的所有权

常见智能指针:
Box<T>   - 堆分配
Rc<T>    - 引用计数（多所有者）
RefCell<T> - 内部可变性
Arc<T>   - 原子引用计数（线程安全）
Mutex<T> - 互斥锁
*/
```

## 二、Box 堆分配

```rust
fn main() {
    // Box 基础使用
    let b = Box::new(5);
    println!("b = {}", b);

    // 递归类型
    enum List {
        Cons(i32, Box<List>),
        Nil,
    }

    use List::{Cons, Nil};

    let list = Cons(1, Box::new(
        Cons(2, Box::new(
            Cons(3, Box::new(Nil))
        ))
    ));

    // Box 用于 trait 对象
    let items: Vec<Box<dyn std::fmt::Display>> = vec![
        Box::new(42),
        Box::new("hello"),
        Box::new(3.14),
    ];
}
```

## 三、Rc 引用计数

```rust
use std::rc::Rc;

fn main() {
    // Rc 允许多个所有者
    let a = Rc::new(vec![1, 2, 3]);
    let b = Rc::clone(&a);  // 增加引用计数
    let c = Rc::clone(&a);

    println!("引用计数: {}", Rc::strong_count(&a));  // 3
    println!("a = {:?}", a);
    println!("b = {:?}", b);

    // Rc 只能读取，不能修改
    // a.push(4);  // 错误！Rc 是不可变的

    // 配合 RefCell 实现可变
    use std::cell::RefCell;
    let shared = Rc::new(RefCell::new(vec![1, 2, 3]));
    shared.borrow_mut().push(4);
}
```

## 四、RefCell 内部可变性

```rust
use std::cell::RefCell;

fn main() {
    // RefCell 在运行时检查借用规则
    let data = RefCell::new(vec![1, 2, 3]);

    // 不可变借用
    let borrowed = data.borrow();
    println!("{:?}", borrowed);
    drop(borrowed);  // 释放借用

    // 可变借用
    data.borrow_mut().push(4);
    println!("{:?}", data.borrow());

    // 运行时检查：同时多个可变借用会 panic
    // let mut ref1 = data.borrow_mut();
    // let mut ref2 = data.borrow_mut();  // panic!
}
```

## 五、Arc 和 Mutex

```rust
use std::sync::{Arc, Mutex};
use std::thread;

fn main() {
    // Arc - 线程安全的引用计数
    // Mutex - 线程安全的内部可变性
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];

    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }

    for handle in handles {
        handle.join().unwrap();
    }

    println!("结果: {}", *counter.lock().unwrap());  // 10
}
```

## 六、注意事项

1. **Rc 不线程安全**：多线程使用 Arc
2. **RefCell 运行时检查**：借用违规会 panic，不是编译错误
3. **避免循环引用**：使用 Weak 打破循环
4. **性能考虑**：智能指针有额外开销
5. **选择指南**：单线程用 Rc<RefCell>，多线程用 Arc<Mutex>
