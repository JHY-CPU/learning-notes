# 闭包基础语法

## 一、概念说明

闭包（Closure）是可以捕获其所在环境变量的匿名函数。Rust 的闭包采用 `||` 语法，可以捕获环境中的变量，具有灵活的移动语义控制。

```rust
// 基本闭包定义
let add = |x: i32, y: i32| -> i32 { x + y };
let result = add(3, 5); // 8

// 省略类型（编译器自动推断）
let double = |x| x * 2;
let result = double(5); // 10

// 无参数闭包
let greeting = || println!("你好，世界！");
greeting();
```

## 二、具体用法

### 2.1 捕获环境变量

```rust
let name = String::from("张三");

// 不可变借用捕获
let greet = || println!("你好，{}！", name);
greet();
greet(); // 可以多次调用

// 可变借用捕获
let mut count = 0;
let mut increment = || {
    count += 1;
    println!("计数: {}", count);
};
increment(); // 计数: 1
increment(); // 计数: 2

// 所有权转移捕获
let message = String::from("再见");
let say_goodbye = move || {
    println!("{}", message);
};
say_goodbye();
// println!("{}", message); // 编译错误：所有权已转移
```

### 2.2 闭包作为函数参数

```rust
// 使用 Fn trait
fn apply<F: Fn(i32) -> i32>(f: F, value: i32) -> i32 {
    f(value)
}

let result = apply(|x| x * x, 5); // 25

// 使用 impl Trait 简化
fn process(value: i32, transform: impl Fn(i32) -> i32) -> i32 {
    transform(value)
}
```

### 2.3 返回闭包

```rust
fn make_adder(x: i32) -> impl Fn(i32) -> i32 {
    move |y| x + y
}

let add_five = make_adder(5);
println!("{}", add_five(3));  // 8
println!("{}", add_five(10)); // 15
```

## 三、注意事项与常见陷阱

1. **类型推断**：闭包参数类型通常需要从上下文推断
2. **move 关键字**：强制闭包获取捕获变量的所有权
3. **Fn trait**：Fn > FnMut > FnOnce（约束从强到弱）
4. **生命周期**：闭包的生命周期不超过捕获变量的生命周期
5. **多次调用**：FnMut 和 FnOnce 闭包的调用次数有区别
