# 闭包作为返回值

## 一、概念说明

Rust 中可以将闭包作为函数的返回值。由于每个闭包都有唯一的匿名类型，需要使用 `impl Trait` 或 trait 对象来处理返回的闭包类型。

```rust
// 返回 impl Fn
fn make_greeter(name: &str) -> impl Fn() + '_ {
    move || println!("你好，{}！", name)
}

// 返回 Box<dyn Fn>
fn make_operation(op: &str) -> Box<dyn Fn(i32, i32) -> i32> {
    match op {
        "add" => Box::new(|a, b| a + b),
        "mul" => Box::new(|a, b| a * b),
        _ => Box::new(|a, b| a),
    }
}
```

## 二、具体用法

### 2.1 impl Trait 返回

```rust
fn counter(start: i32) -> impl FnMut() -> i32 {
    let mut count = start;
    move || {
        count += 1;
        count
    }
}

let mut c = counter(0);
println!("{}", c()); // 1
println!("{}", c()); // 2

// 使用生命周期约束
fn make_ref_printer<'a>(data: &'a str) -> impl Fn() + 'a {
    move || println!("{}", data)
}
```

### 2.2 dyn Trait 返回

```rust
fn get_filter(kind: &str) -> Box<dyn Fn(&i32) -> bool> {
    match kind {
        "positive" => Box::new(|x| *x > 0),
        "negative" => Box::new(|x| *x < 0),
        "even" => Box::new(|x| x % 2 == 0),
        _ => Box::new(|_| true),
    }
}

let data = vec![-2, -1, 0, 1, 2];
let filter = get_filter("positive");
let positives: Vec<&i32> = data.iter().filter(|x| filter(x)).collect();
// [1, 2]
```

### 2.3 闭包工厂模式

```rust
fn make_validator(min: i32, max: i32) -> impl Fn(i32) -> Result<i32, String> {
    move |value| {
        if value < min {
            Err(format!("值 {} 小于最小值 {}", value, min))
        } else if value > max {
            Err(format!("值 {} 大于最大值 {}", value, max))
        } else {
            Ok(value)
        }
    }
}

let validate_age = make_validator(0, 150);
match validate_age(25) {
    Ok(age) => println!("有效年龄: {}", age),
    Err(e) => println!("错误: {}", e),
}
```

## 三、注意事项与常见陷阱

1. **impl Trait 限制**：同一函数不能返回不同类型的 impl Fn
2. **生命周期约束**：捕获引用的闭包需要关联生命周期
3. **性能选择**：impl Fn 静态分发性能好，dyn Fn 灵活性高
4. **Send 约束**：跨线程使用需 `Box<dyn Fn() + Send>`
5. **迭代器返回**：返回 Map<..., impl Fn> 比返回 Vec 更高效
