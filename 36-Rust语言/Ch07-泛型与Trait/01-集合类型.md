# 集合类型

## 一、概念说明

Rust 标准库提供了多种集合类型，用于存储和操作数据。最常用的有 Vec、HashMap 和 String。

```rust
// 常用集合
/*
Vec<T>     - 动态数组
String     - UTF-8 字符串
HashMap<K, V> - 哈希映射
HashSet<T> - 哈希集合
BTreeMap<K, V> - 有序映射
VecDeque<T> - 双端队列
LinkedList<T> - 双向链表
*/
```

## 二、Vec 动态数组

```rust
fn main() {
    // 创建 Vec
    let mut v: Vec<i32> = Vec::new();
    let v2 = vec![1, 2, 3];  // 使用宏创建

    // 添加元素
    v.push(1);
    v.push(2);
    v.push(3);

    // 读取元素
    let third: &i32 = &v[2];     // 索引访问（越界会 panic）
    let third: Option<&i32> = v.get(2);  // get 方法（越界返回 None）

    // 遍历
    for i in &v {
        println!("{}", i);
    }

    // 可变遍历
    for i in &mut v {
        *i += 50;
    }

    // 使用枚举存储不同类型
    enum Cell {
        Int(i32),
        Float(f64),
        Text(String),
    }
    let row = vec![
        Cell::Int(3),
        Cell::Text(String::from("blue")),
        Cell::Float(10.12),
    ];
}
```

## 三、String 字符串

```rust
fn main() {
    // 创建 String
    let mut s = String::new();
    let s = String::from("hello");
    let s = "hello".to_string();

    // 追加
    s.push_str(" world");  // 追加字符串切片
    s.push('!');            // 追加字符

    // 拼接
    let s1 = String::from("Hello, ");
    let s2 = String::from("world!");
    let s3 = s1 + &s2;  // s1 被移动，不再有效

    // format! 宏（推荐）
    let s = format!("{}-{}-{}", 2024, 1, 15);

    // 注意：String 不支持索引
    // let c = s[0];  // 错误！

    // 遍历字符
    for c in "你好世界".chars() {
        println!("{}", c);  // 你 好 世 界
    }

    // 遍历字节
    for b in "hello".bytes() {
        println!("{}", b);  // 104 101 108 108 111
    }

    // 切片（注意字符边界）
    let s = "你好世界";
    let hello = &s[0..6];  // "你好"（每个汉字 3 字节）
}
```

## 四、HashMap

```rust
use std::collections::HashMap;

fn main() {
    // 创建 HashMap
    let mut scores = HashMap::new();

    // 插入
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    // 或从元组迭代器创建
    let teams = vec![String::from("Blue"), String::from("Yellow")];
    let initial_scores = vec![10, 50];
    let scores: HashMap<_, _> = teams.into_iter().zip(initial_scores.into_iter()).collect();

    // 访问
    let score = scores.get("Blue");  // Option<&V>
    match score {
        Some(&s) => println!("分数: {}", s),
        None => println!("没有分数"),
    }

    // 只在键不存在时插入
    scores.entry(String::from("Blue")).or_insert(50);

    // 基于旧值更新
    let text = "hello world wonderful world";
    let mut word_count = HashMap::new();
    for word in text.split_whitespace() {
        let count = word_count.entry(word).or_insert(0);
        *count += 1;
    }
    println!("{:?}", word_count);

    // 遍历
    for (key, value) in &scores {
        println!("{}: {}", key, value);
    }
}
```

## 五、其他集合

```rust
use std::collections::{HashSet, BTreeMap, VecDeque};

fn main() {
    // HashSet - 不重复集合
    let mut books = HashSet::new();
    books.insert("Rust编程");
    books.insert("深入理解计算机系统");
    books.insert("Rust编程");  // 重复，不会添加

    println!("书籍数: {}", books.len());  // 2

    // 集合运算
    let a: HashSet<i32> = vec![1, 2, 3].into_iter().collect();
    let b: HashSet<i32> = vec![2, 3, 4].into_iter().collect();

    let union: HashSet<_> = a.union(&b).collect();       // {1, 2, 3, 4}
    let intersection: HashSet<_> = a.intersection(&b).collect();  // {2, 3}

    // BTreeMap - 有序映射
    let mut map = BTreeMap::new();
    map.insert(3, "three");
    map.insert(1, "one");
    map.insert(2, "two");
    // 按键排序: 1, 2, 3

    // VecDeque - 双端队列
    let mut deque = VecDeque::new();
    deque.push_back(1);
    deque.push_front(0);
    deque.push_back(2);
    // [0, 1, 2]
}
```

## 六、注意事项

1. **所有权**：集合拥有其元素的所有权
2. **借用规则**：不能在持有可变引用的同时持有不可变引用
3. **字符串索引**：String 不支持直接索引，使用 chars() 或切片
4. **HashMap 键要求**：键类型必须实现 Hash 和 Eq trait
5. **集合选择**：根据使用场景选择合适的集合类型
