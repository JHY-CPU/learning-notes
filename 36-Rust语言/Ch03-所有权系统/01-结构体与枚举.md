# 结构体与枚举

## 一、概念说明

结构体（Struct）用于创建自定义数据类型，将多个相关值组合在一起。枚举（Enum）表示一个值可能是一组值中的一个。

```rust
// 结构体定义
struct User {
    username: String,
    email: String,
    age: u32,
    active: bool,
}

// 枚举定义
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
```

## 二、结构体详解

```rust
// 元组结构体
struct Color(i32, i32, i32);
struct Point(f64, f64);

// 单元结构体
struct AlwaysEqual;

// 结构体方法
impl User {
    // 关联函数（构造器）
    fn new(username: String, email: String) -> Self {
        Self {
            username,
            email,
            age: 0,
            active: true,
        }
    }

    // 方法（有 self）
    fn greeting(&self) -> String {
        format!("你好，我是 {}", self.username)
    }

    // 可变方法
    fn set_age(&mut self, age: u32) {
        self.age = age;
    }

    // 消耗 self 的方法
    fn into_email(self) -> String {
        self.email
    }
}

fn main() {
    let mut user = User::new(
        String::from("张三"),
        String::from("zhangsan@example.com"),
    );
    user.set_age(25);
    println!("{}", user.greeting());

    // 结构体更新语法
    let user2 = User {
        username: String::from("李四"),
        ..user
    };
}
```

## 三、枚举详解

```rust
// 枚举方法
impl Message {
    fn process(&self) {
        match self {
            Message::Quit => println!("退出"),
            Message::Move { x, y } => println!("移动到 ({}, {})", x, y),
            Message::Write(text) => println!("写入: {}", text),
            Message::ChangeColor(r, g, b) => println!("颜色: ({}, {}, {})", r, g, b),
        }
    }
}

// Option 枚举（标准库）
fn divide(x: f64, y: f64) -> Option<f64> {
    if y == 0.0 {
        None
    } else {
        Some(x / y)
    }
}

fn main() {
    let msg = Message::Write(String::from("你好"));
    msg.process();

    // Option 处理
    match divide(10.0, 3.0) {
        Some(result) => println!("结果: {}", result),
        None => println!("不能除以零"),
    }

    // if let 简化匹配
    if let Some(result) = divide(10.0, 2.0) {
        println!("结果: {}", result);
    }

    // unwrap_or 提供默认值
    let result = divide(10.0, 0.0).unwrap_or(0.0);
    println!("结果: {}", result);
}
```

## 四、模式匹配

```rust
// match 完整示例
enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter(UsState),
}

#[derive(Debug)]
enum UsState {
    Alabama,
    Alaska,
    California,
}

fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => 1,
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter(state) => {
            println!("来自 {:?} 的 25 美分", state);
            25
        }
    }
}

fn main() {
    let coin = Coin::Quarter(UsState::California);
    println!("价值: {} 美分", value_in_cents(coin));
}
```

## 五、注意事项

1. **所有权与结构体**：结构体字段可以拥有数据或引用（需要生命周期）
2. **枚举大小**：枚举大小取决于最大变体
3. **match 穷尽**：match 必须覆盖所有可能的值
4. **Option 处理**：优先使用 match 或 if let，而非 unwrap
5. **derive 宏**：使用 `#[derive(Debug, Clone)]` 自动实现常用 trait
