# 模板引擎

## 一、概念说明

Rust 有多种模板引擎：Tera、Askama、Handlebars 等，用于服务端渲染 HTML。

```rust
use tera::{Tera, Context};

let tera = Tera::new("templates/**/*").unwrap();
let mut context = Context::new();
context.insert("name", "张三");
let html = tera.render("index.html", &context).unwrap();
```

## 二、具体用法

### 2.1 Askama 编译时模板

```rust
use askama::Template;

#[derive(Template)]
#[template(path = "hello.html")]
struct HelloTemplate<'a> {
    name: &'a str,
}

async fn hello() -> impl axum::response::IntoResponse {
    let template = HelloTemplate { name: "世界" };
    Html(template.render().unwrap())
}
```

### 2.2 Tera 模板语法

```html
<!-- templates/base.html -->
<!DOCTYPE html>
<html>
<head><title>{{ title }}</title></head>
<body>
{% block content %}{% endblock %}
</body>
</html>

<!-- templates/index.html -->
{% extends "base.html" %}
{% block content %}
<h1>你好，{{ name }}！</h1>
{% for item in items %}
<p>{{ item }}</p>
{% endfor %}
{% endblock %}
```

## 三、注意事项与常见陷阱

1. **XSS 防护**：模板引擎默认转义输出
2. **性能**：编译时模板（Askama）比运行时模板更快
3. **错误处理**：模板渲染错误需妥善处理
4. **继承复用**：使用模板继承减少重复
5. **静态资源**：配置静态文件服务
