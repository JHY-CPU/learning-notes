# 模板引擎

## 一、概念说明

Rust 有多种模板引擎：Tera、Askama、Handlebars 等，用于服务端渲染 HTML。

```rust
use tera::{Tera, Context};

let tera = Tera::new("templates/**/*").unwrap();
let mut context = Context::new();
context.insert("name", "张三");
let html = tera.render("index.html", &context).unwrap();
```

## 二、具体用法

### 2.1 Askama 编译时模板

```rust
use askama::Template;

#[derive(Template)]
#[template(path = "hello.html")]
struct HelloTemplate<'a> {
    name: &'a str,
}

async fn hello() -> impl axum::response::IntoResponse {
    let template = HelloTemplate { name: "世界" };
    Html(template.render().unwrap())
}
```

### 2.2 Tera 模板语法

```html
<!-- templates/base.html -->
<!DOCTYPE html>
<html>
<head><title>{{ title }}</title></head>
<body>
{% block content %}{% endblock %}
</body>
</html>

<!-- templates/index.html -->
{% extends "base.html" %}
{% block content %}
<h1>你好，{{ name }}！</h1>
{% for item in items %}
<p>{{ item }}</p>
{% endfor %}
{% endblock %}
```

### 2.3 MiniJinja 模板

```rust
use minijinja::{Environment, context};

fn minijinja_example() {
    let mut env = Environment::new();
    env.add_template("hello", "你好，{{ name }}！").unwrap();

    let tmpl = env.get_template("hello").unwrap();
    let html = tmpl.render(context!(name => "张三")).unwrap();
    println!("{}", html);
}
```

## 四、模板引擎对比

| 引擎 | 类型 | 性能 | 特点 |
|------|------|------|------|
| Askama | 编译时 | 最快 | 类型安全、编译时检查 |
| Tera | 运行时 | 中等 | Jinja2 语法、灵活 |
| Handlebars | 运行时 | 中等 | 通用、跨语言 |
| MiniJinja | 运行时 | 快 | 轻量级、API 简洁 |

## 五、注意事项与常见陷阱

1. **XSS 防护**：模板引擎默认转义输出，使用 `|safe` 过滤器标记安全内容
2. **性能**：编译时模板（Askama）比运行时模板更快，推荐生产环境使用
3. **错误处理**：模板渲染错误需妥善处理，提供有用的错误信息
4. **继承复用**：使用模板继承减少重复，定义 base 模板
5. **静态资源**：配置静态文件服务和缓存策略
6. **模板组织**：合理组织模板目录结构，使用模块化
7. **国际化**：考虑模板的国际化支持
