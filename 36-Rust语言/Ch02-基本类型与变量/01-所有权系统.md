# 所有权系统

## 一、概念说明

所有权（Ownership）是 Rust 最核心的特性，它在编译时保证内存安全，无需垃圾回收。每个值都有一个所有者，当所有者离开作用域时值被释放。

```rust
// 所有权三大规则
/*
1. Rust 中每个值都有一个所有者（owner）
2. 同一时刻只能有一个所有者
3. 当所有者离开作用域，值被丢弃（drop）
*/
```

## 二、所有权转移

```rust
fn main() {
    // 基本类型（Copy）- 赋值时复制
    let x = 5;
    let y = x;  // 复制值
    println!("x = {}, y = {}", x, y); // 两个都有效

    // String 类型（非 Copy）- 赋值时移动
    let s1 = String::from("hello");
    let s2 = s1;  // 所有权移动到 s2
    // println!("{}", s1);  // 错误！s1 已失效
    println!("{}", s2);

    // 克隆（深拷贝）
    let s1 = String::from("hello");
    let s2 = s1.clone();
    println!("s1 = {}, s2 = {}", s1, s2); // 两个都有效

    // 函数传参 - 所有权转移
    let s = String::from("hello");
    takes_ownership(s);
    // println!("{}", s);  // 错误！s 已移动

    // 函数返回 - 所有权转移回来
    let s1 = gives_ownership();
    let s2 = String::from("world");
    let s3 = takes_and_gives_back(s2);
}

fn takes_ownership(s: String) {
    println!("{}", s);
} // s 离开作用域，内存释放

fn gives_ownership() -> String {
    String::from("hello")
}

fn takes_and_gives_back(s: String) -> String {
    s
}
```

## 三、引用与借用

```rust
fn main() {
    // 不可变引用（借用）
    let s1 = String::from("hello");
    let len = calculate_length(&s1);  // 借用 s1
    println!("'{}' 的长度是 {}", s1, len); // s1 仍然有效

    // 可变引用
    let mut s = String::from("hello");
    change(&mut s);  // 可变借用
    println!("{}", s);

    // 引用规则
    let mut s = String::from("hello");

    // 可以有多个不可变引用
    let r1 = &s;
    let r2 = &s;
    println!("{}, {}", r1, r2);
    // r1, r2 不再使用

    // 只能有一个可变引用
    let r3 = &mut s;
    println!("{}", r3);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}

fn change(s: &mut String) {
    s.push_str(", world");
}
```

## 四、所有权与函数

```rust
// 返回所有权
fn first_word(s: &String) -> &str {
    let bytes = s.as_bytes();
    for (i, &item) in bytes.iter().enumerate() {
        if item == b' ' {
            return &s[0..i];
        }
    }
    &s[..]
}

// 多个引用
fn analyze_string(s: &String) -> (usize, &str, &str) {
    let len = s.len();
    let first = &s[0..1];
    let last = &s[len-1..];
    (len, first, last)
}

fn main() {
    let mut s = String::from("hello world");
    let word = first_word(&s);
    println!("第一个单词: {}", word);

    // 在使用不可变引用期间不能修改
    // s.clear(); // 错误！word 还在使用
}
```

## 五、生命周期基础

```rust
// 生命周期标注
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}

fn main() {
    let string1 = String::from("long string");
    let result;
    {
        let string2 = String::from("xyz");
        result = longest(string1.as_str(), string2.as_str());
        println!("更长的: {}", result);
    }
    // string2 的生命周期在此结束
    // result 不能在此使用
}
```

## 六、注意事项

1. **移动语义**：非 Copy 类型赋值会移动所有权
2. **借用规则**：同一时刻只能有多个不可变引用或一个可变引用
3. **悬垂引用**：Rust 编译器防止引用指向已释放的内存
4. **Copy trait**：基本类型实现了 Copy，赋值时复制而非移动
5. **生命周期**：引用的生命周期不能超过被引用值的生命周期
