# 游戏开发入门

## 一、概念说明

Rust 在游戏开发中越来越受欢迎，Bevy 是最流行的 Rust 游戏引擎。

```rust
use bevy::prelude::*;

fn main() {
    App::new()
        .add_plugins(DefaultPlugins)
        .add_systems(Startup, setup)
        .add_systems(Update, move_player)
        .run();
}

fn setup(mut commands: Commands) {
    commands.spawn(Camera2dBundle::default());
}
```

## 二、具体用法

### 2.1 ECS 架构

```rust
use bevy::prelude::*;

#[derive(Component)]
struct Player;

#[derive(Component)]
struct Velocity(Vec2);

fn movement_system(
    mut query: Query<(&mut Transform, &Velocity), With<Player>>,
    time: Res<Time>,
) {
    for (mut transform, velocity) in &mut query {
        transform.translation.x += velocity.0.x * time.delta_seconds();
        transform.translation.y += velocity.0.y * time.delta_seconds();
    }
}
```

### 2.2 输入处理

```rust
fn keyboard_input(
    keyboard: Res<Input<KeyCode>>,
    mut query: Query<&mut Velocity, With<Player>>,
) {
    let mut velocity = query.single_mut();
    if keyboard.pressed(KeyCode::W) {
        velocity.0.y = 100.0;
    }
    if keyboard.pressed(KeyCode::S) {
        velocity.0.y = -100.0;
    }
}
```

### 2.3 碰撞检测

```rust
use bevy::prelude::*;

fn collision_detection(
    query1: Query<&Transform, With<Player>>,
    query2: Query<&Transform, With<Enemy>>,
) {
    let player = query1.single();
    for enemy in &query2 {
        let distance = player.translation.distance(enemy.translation);
        if distance < 50.0 {
            println!("碰撞！");
        }
    }
}
```

### 2.4 资源与资产管理

```rust
use bevy::prelude::*;

#[derive(Resource)]
struct GameScore(u32);

#[derive(Resource)]
struct GameSettings {
    volume: f32,
    difficulty: u8,
}

fn setup_resources(mut commands: Commands, asset_server: Res<AssetServer>) {
    commands.insert_resource(GameScore(0));
    commands.insert_resource(GameSettings {
        volume: 0.8,
        difficulty: 2,
    });

    // 加载资产
    let texture: Handle<Image> = asset_server.load("sprites/player.png");
}
```

### 2.5 状态管理

```rust
use bevy::prelude::*;

#[derive(States, Debug, Clone, Eq, PartialEq, Hash, Default)]
enum GameState {
    #[default]
    MainMenu,
    Playing,
    Paused,
    GameOver,
}

fn state_transitions(
    mut next_state: ResMut<NextState<GameState>>,
    keyboard: Res<Input<KeyCode>>,
) {
    if keyboard.just_pressed(KeyCode::Escape) {
        next_state.set(GameState::Paused);
    }
}
```

## 四、游戏引擎对比

| 引擎 | 特点 | 适用场景 |
|------|------|---------|
| Bevy | ECS、现代、社区活跃 | 2D/3D 游戏 |
| macroquad | 轻量、简单 | 小游戏、原型 |
| ggez | 简单、2D 专注 | 2D 游戏 |
| wgpu | 底层图形 | 自定义渲染 |

## 五、注意事项与常见陷阱

1. **ECS 学习曲线**：ECS 架构需要时间理解，从简单系统开始
2. **性能优化**：合理使用查询和系统，避免每帧查询大量实体
3. **资产管理**：使用 Bevy 的资产管理器加载和管理资产
4. **物理引擎**：使用 bevy_rapier 处理物理，或 bevy_xpbd
5. **社区资源**：利用 Bevy 社区资源学习，参考官方示例
6. **帧率控制**：配置固定时间步长和最大帧率
7. **多平台测试**：在目标平台上测试游戏性能
