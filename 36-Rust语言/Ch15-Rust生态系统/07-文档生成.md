# 文档生成

## 一、概念说明

Rust 使用 rustdoc 从代码注释生成文档，支持 Markdown 格式和代码示例测试。

```rust
/// 计算两个数的和
///
/// # 示例
///
/// ```
/// let result = my_crate::add(2, 3);
/// assert_eq!(result, 5);
/// ```
pub fn add(a: i32, b: i32) -> i32 {
    a + b
}
```

## 二、具体用法

### 2.1 文档注释

```rust
/// 结构体文档
///
/// # 示例
///
/// ```
/// use my_crate::MyStruct;
/// let s = MyStruct::new();
/// ```
pub struct MyStruct {
    /// 字段文档
    pub field: i32,
}

// 模块级文档
//! 模块级文档注释
//! 用于描述模块的功能
```

### 2.2 文档属性

```rust
/// 内部函数，不在公共文档中显示
#[doc(hidden)]
pub fn internal_function() {}

// 文档中不显示
#[cfg(not(doctest))]
pub fn test_only() {}
```

### 2.3 生成文档

```bash
# 生成并打开文档
cargo doc --open

# 包含私有项
cargo doc --document-private-items

# 无依赖文档
cargo doc --no-deps
```

## 三、注意事项与常见陷阱

1. **示例测试**：文档中的代码示例会被测试
2. **完整性**：为所有公共 API 提供文档
3. **链接**：使用链接引用相关项
4. **版本信息**：添加版本历史信息
5. **国际化**：考虑文档的国际化
