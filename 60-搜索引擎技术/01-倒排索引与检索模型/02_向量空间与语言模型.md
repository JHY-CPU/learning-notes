# 向量空间与语言模型


## 向量空间模型与语言模型


VSM语言模型相关性反馈


向量空间模型(VSM)和查询似然模型是两种经典的检索模型。


## 向量空间模型 (Vector Space Model)


```
核心思想：
- 将文档和查询都表示为高维向量
- 每个维度对应一个词条
- 通过向量相似度衡量相关性

文档向量化：
d = (w1, w2, w3, ..., wn)
其中 wi = TF-IDF值（词条i在文档d中的权重）

相似度计算：余弦相似度
sim(q, d) = cos(θ) = (q · d) / (|q| × |d|)
= Σ(qi × di) / sqrt(Σqi²) × sqrt(Σdi²)

示例：
词表：[搜索引擎, 倒排索引, 算法, 数据结构]
Query: [1.0, 0.8, 0.0, 0.0]
Doc1:  [0.5, 0.6, 0.3, 0.1]
Doc2:  [0.9, 0.1, 0.2, 0.4]

cos(Query, Doc1) = (1.0×0.5 + 0.8×0.6) / (|q|×|d1|)
cos(Query, Doc2) = (1.0×0.9 + 0.8×0.1) / (|q|×|d2|)
→ Doc1的相关度更高

VSM的优缺点：
+ 简单直观，数学基础扎实
+ 支持部分匹配（不要求精确匹配）
- 假设词条独立（词袋模型，丢失语义）
- 维度灾难（词表可能有百万级词条）
- 无法处理同义词/多义词
```


## 查询似然模型 (Query Likelihood Model)


```
语言模型方法：
- 为每个文档建立一个语言模型 P(w|d)
- 查询的生成概率 = 各词条概率的乘积
- 假设查询词之间独立

核心公式：
P(q|d) = P(q1|d) × P(q2|d) × ... × P(qn|d)
其中 P(qi|d) = c(qi, d) / |d|（词条在文档中的频率/文档长度）

问题：如果查询词不在文档中，概率为0！
解决方案：平滑 (Smoothing)

Jelinek-Mercer 平滑：
P(qi|d) = λ × P_ml(qi|d) + (1-λ) × P_ml(qi|C)
- P_ml(qi|d)：词条在文档中的最大似然估计
- P_ml(qi|C)：词条在整个语料中的概率
- λ：平滑参数（0到1之间）

Dirichlet 平滑：
P(qi|d) = [c(qi,d) + μ×P(qi|C)] / [|d| + μ]
- μ：平滑参数（通常取2000左右）
- 文档越短，平滑效果越强

两者的区别：
- Jelinek-Mercer：固定混合比例
- Dirichlet：根据文档长度自适应调整
```


## 相关性反馈 (Relevance Feedback)


```
相关性反馈：利用用户的反馈改进查询

三种反馈方式：
┌─────────────────────────────────────────────┐
│  1. 显式反馈 (Explicit Feedback)             │
│     用户手动标注相关/不相关文档               │
│     → Rocchio算法修改查询向量                 │
│                                             │
│  2. 隐式反馈 (Implicit Feedback)             │
│     通过用户行为推断相关性                    │
│     - 点击、停留时间、滚动深度                │
│     - 点击模型（Click Model）                 │
│                                             │
│  3. 伪相关反馈 (Pseudo-Relevance Feedback)   │
│     自动假设初始Top-K结果相关                 │
│     用这些文档扩展查询                        │
│     → 通常有效但可能引入噪声                  │
└─────────────────────────────────────────────┘

Rocchio 算法：
q_new = α × q_old + β/|Dr| × Σd∈Dr d - γ/|Dnr| × Σd∈Dnr d
- Dr：相关文档集合
- Dnr：不相关文档集合
- α, β, γ：权重参数
- 将查询向相关文档方向移动，远离不相关文档
```


> **Note:** 现代搜索引擎广泛使用隐式反馈（点击数据），结合点击模型优化排序。伪相关反馈在学术搜索中常用。
