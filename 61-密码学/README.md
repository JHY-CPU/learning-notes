# 密码学（Cryptography）

密码学是研究信息安全传输与存储的学科，涵盖加密、解密、认证、完整性校验等核心理论与技术。本系列笔记系统梳理从经典到现代的密码学知识体系。

## 目录结构

| 序号 | 目录 | 主要内容 |
|------|------|----------|
| 01 | [对称加密](./01-对称加密/README.md) | DES、AES、分组密码工作模式、流密码 |
| 02 | [非对称加密](./02-非对称加密/README.md) | RSA、Diffie-Hellman、ECC、后量子密码 |
| 03 | [哈希与数字签名](./03-哈希与数字签名/README.md) | SHA系列、HMAC、数字签名、PKI |
| 04 | [高级密码学](./04-高级密码学/README.md) | 零知识证明、同态加密、秘密共享、MPC |

## 学习路线建议

1. **对称加密** -- 理解最基本的加解密思想，掌握AES及其工作模式
2. **非对称加密** -- 学习公钥密码体系，理解密钥交换与数字签名的数学基础
3. **哈希与数字签名** -- 掌握消息完整性保障机制与PKI体系
4. **高级密码学** -- 了解前沿密码学协议与应用场景
