# 常见凸优化问题 — 最优化理论笔记


## 一、线性规划（Linear Programming, LP）


### 1.1 标准形式


$$
\(\min_{x} \quad c^T x\)
                \(\text{s.t.} \quad Ax = b, \quad x \ge 0\)
$$


等价地，一般形式为：


$$
\(\min \quad c^T x \quad \text{s.t.} \quad A_{\text{eq}}x = b_{\text{eq}}, \; A_{\text{in}}x \le b_{\text{in}}, \; l \le x \le u\)
$$


### 1.2 关键性质


- 目标函数和约束全是仿射函数，必为凸优化
- 可行域是多面体（有限个半空间的交）
- 最优解一定出现在可行域的
   **顶点**
   上（如果存在最优解）
- 图解法：在2D中，画出可行域，沿梯度反方向移动直到边界
- 求解算法：
   **单纯形法**
   （沿顶点搜索）、
   **内点法**
   （穿过可行域内部）
- 强对偶成立：LP的对偶仍然是LP，且对偶间隙为零


### 1.3 应用场景


- 资源分配与生产计划
- 运输问题、指派问题
- 网络流问题
- 线性分类（感知机可转化为LP）


## 二、二次规划（Quadratic Programming, QP）


### 2.1 标准形式


$$
\(\min_{x} \quad \frac{1}{2}x^T Q x + c^T x\)
                \(\text{s.t.} \quad Ax \le b, \; A_{\text{eq}}x = b_{\text{eq}}\)
$$


当 \( Q \succeq 0 \)（半正定）时，目标函数是凸函数，问题是凸QP。


### 2.2 典型应用


| 应用 | Q 矩阵 | 说明 |
| --- | --- | --- |
| 最小二乘（OLS） | \( Q = A^TA \)（半正定） | \( \min \\|Ax-b\\|^2 \) |
| SVM | \( Q_{ij} = y_iy_j x_i^Tx_j \) | 最大化间隔 |
| 带约束最小二乘 | 同OLS | 添加 \( x \ge 0 \) 等约束 |
| Markowitz投资组合 | 协方差矩阵 | 最小化风险给定收益 |


## 三、半定规划（Semidefinite Programming, SDP）与正则化方法


### 3.1 半定规划


$$
\(\min \quad \text{tr}(CX) \quad \text{s.t.} \quad \text{tr}(A_i X) = b_i, \; X \succeq 0\)
$$


SDP是LP的推广——用半正定锥代替非负象限。应用包括：特征值优化、控制理论中的Lyapunov稳定性、组合优化的松弛等。


### 3.2 LASSO回归（L1正则化）


$$
\(\min_\beta \frac{1}{2}\|y - X\beta\|_2^2 + \lambda \|\beta\|_1\)
$$


等价形式（约束版）：


$$
\(\min_\beta \frac{1}{2}\|y - X\beta\|_2^2 \quad \text{s.t.} \quad \|\beta\|_1 \le t\)
$$


- L1正则化产生
   **稀疏解**
   ：许多系数恰好为零
- 几何直觉：L1球有尖角，等高线容易在坐标轴上相切
- 求解方法：坐标下降法、ADMM、近端梯度法（ISTA/FISTA）
- 本质上是NP难组合优化问题的凸松弛


### 3.3 Ridge回归（L2正则化）


$$
\(\min_\beta \frac{1}{2}\|y - X\beta\|_2^2 + \lambda \|\beta\|_2^2\)
$$


- L2正则化解的所有分量都非零（不产生稀疏性）
- 几何直觉：L2球是圆形，等高线不太可能在坐标轴上相切
- 有闭式解：\( \beta^* = (X^TX + \lambda I)^{-1}X^Ty \)
- 解决多重共线性问题，提高数值稳定性


> **Note:** **Elastic Net：**
> 结合L1和L2：\( \lambda_1\|\beta\|_1 + \lambda_2\|\beta\|_2^2 \)，兼有稀疏性和稳定性。


## 四、投影梯度法（Projected Gradient Descent）


### 4.1 算法思路


对于约束优化问题 \( \min f(x) \text{ s.t. } x \in \mathcal{C} \)（\( \mathcal{C} \) 为凸集）：


$$
\(x_{t+1} = \text{Proj}_{\mathcal{C}}(x_t - \alpha \nabla f(x_t))\)
$$


其中投影算子 \( \text{Proj}_{\mathcal{C}}(z) = \arg\min_{x \in \mathcal{C}} \|x - z\|_2^2 \)。


### 4.2 常见投影


| 约束集 \( \mathcal{C} \) | 投影操作 |
| --- | --- |
| 非负象限 \( x \ge 0 \) | \( \max(x, 0) \)（逐元素） |
| 盒子约束 \( l \le x \le u \) | \( \text{clip}(x, l, u) \) |
| L1球 \( \\|x\\|_1 \le t \) | 软阈值算子（Soft Thresholding） |
| 概率单纯形 | 排序后截断归一化 |
| L2球 \( \\|x\\|_2 \le t \) | \( x \cdot \min(1, t/\\|x\\|_2) \) |


### 4.3 ISTA / FISTA


**ISTA（Iterative Shrinkage-Thresholding Algorithm）**是投影梯度法在LASSO问题上的特例：


$$
\(x_{t+1} = S_{\lambda/L}(x_t - \frac{1}{L}\nabla f(x_t))\)
$$


其中 \( S_\tau(z) = \text{sign}(z)\max(|z|-\tau, 0) \) 是软阈值算子。


**FISTA**在ISTA基础上加入Nesterov加速，收敛速度从 \( O(1/k) \) 提升到 \( O(1/k^2) \)。

def soft_threshold(x, lam):
    """软阈值算子（L1近端算子）"""
    return np.sign(x) * np.maximum(np.abs(x) - lam, 0)

def lasso_proximal_gradient(X, y, lam=0.1, lr=0.01, max_iter=1000):
    """
    近端梯度下降求解LASSO:
    min 0.5 * ||y - Xw||^2 + lam * ||w||_1
    """
    n, p = X.shape
    w = np.zeros(p)
    losses = []

    for i in range(max_iter):
        # 梯度下降步（光滑部分）
        residual = X @ w - y
        grad = X.T @ residual
        w_temp = w - lr * grad

        # 近端步（非光滑部分：L1正则化）
        w = soft_threshold(w_temp, lr * lam)

        # 记录损失
        loss = 0.5 * np.sum(residual**2) + lam * np.sum(np.abs(w))
        losses.append(loss)

    return w, losses

# 生成数据
np.random.seed(42)
n, p = 100, 20
X = np.random.randn(n, p)
true_w = np.zeros(p)
true_w[[0, 3, 7]] = [3, -2, 1.5]  # 稀疏真实权重
y = X @ true_w + 0.5 * np.random.randn(n)

# 求解
w_lasso, losses = lasso_proximal_gradient(X, y, lam=0.5, lr=0.005)
print("=== LASSO回归（近端梯度法）===")
print(f"非零系数个数: {np.sum(np.abs(w_lasso) > 1e-4)} (真实: 3)")
print(f"恢复的非零位置: {np.where(np.abs(w_lasso) > 1e-4)[0]}")
print(f"对应的系数值: {w_lasso[np.abs(w_lasso) > 1e-4].round(4)}")
print(f"最终损失: {losses[-1]:.4f}")

# ==========================================
# 示例3: 投影到概率单纯形
# ==========================================
def project_simplex(v):
    """将向量v投影到概率单纯形 {x | x>=0, sum(x)=1}"""
    u = np.sort(v)[::-1]
    cumsum = np.cumsum(u)
    rho = np.where(u + (1 - cumsum) / np.arange(1, len(v)+1) > 0)[0][-1]
    theta = (cumsum[rho] - 1) / (rho + 1)
    return np.maximum(v - theta, 0)

v = np.array([0.5, 0.3, -0.1, 0.8])
proj = project_simplex(v)
print(f"\n=== 投影到概率单纯形 ===")
print(f"原始向量: {v}")
print(f"投影结果: {proj.round(4)}")
print(f"验证: sum={proj.sum():.4f}, all>=0={np.all(proj >= -1e-10)}")

## 六、凸优化问题分类总结


| 问题类型 | 目标函数 | 约束类型 | 求解方法 | 复杂度 |
| --- | --- | --- | --- | --- |
| LP | 线性 | 线性 | 单纯形/内点法 | 多项式 |
| QP | 二次（凸） | 线性 | 有效集/内点法 | 多项式 |
| QCQP | 二次（凸） | 二次（凸） | SDP松弛/内点法 | 多项式 |
| SDP | 线性（矩阵） | 半正定锥 | 内点法 | 多项式（较慢） |
| SOCP | 线性 | 二阶锥 | 内点法 | 多项式 |


> **Important:** **包含关系：**
> LP ⊂ QP ⊂ QCQP ⊂ SOCP ⊂ SDP ⊂ 凸优化
>
>
> 越往右越一般，但也越难求解。实践中应使用尽可能具体的结构。


> **Note:** **总结：**
> 线性规划、二次规划是工程中最常见的凸优化问题。LASSO和Ridge是机器学习中正则化的两种范式。投影梯度法将梯度下降推广到约束优化，是处理L1正则化等问题的基础工具。cvxpy等建模语言使得求解凸优化问题变得非常方便。
