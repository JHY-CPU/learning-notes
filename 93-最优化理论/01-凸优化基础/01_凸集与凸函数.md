# 凸集与凸函数 — 最优化理论笔记


## 一、凸集（Convex Set）


### 1.1 凸集的定义


集合 \( C \subseteq \mathbb{R}^n \) 是**凸集**，当且仅当对于任意 \( x, y \in C \) 和任意 \( \lambda \in [0, 1] \)，都有：


$$
\(\lambda x + (1 - \lambda) y \in C\)
$$


直观理解：集合中任意两点的连线段完全落在集合内部。


> **Note:** **几何直觉：**
> 凸集没有"凹陷"。圆、正方形、三角形是凸集；五角星、月牙形不是凸集。


### 1.2 凸集的判定方法


- **定义法：**
   验证任意两点连线仍在集合内
- **交集法：**
   任意多个凸集的交集仍是凸集（常用判定手段）
- **函数上/下水平集：**

   - 若 \( f \) 是凸函数，则其上水平集 \( \{x \mid f(x) \ge \alpha\} \) 是凸集
   - 若 \( f \) 是凹函数，则其下水平集 \( \{x \mid f(x) \le \alpha\} \) 是凸集


## 二、常见凸集


| 凸集类型 | 数学表示 | 说明 |
| --- | --- | --- |
| 超平面（Hyperplane） | \( \{x \mid a^T x = b\} \) | n维空间中的"平面"，将空间分成两半 |
| 半空间（Halfspace） | \( \{x \mid a^T x \le b\} \) | 超平面的一侧，最基本的凸集之一 |
| 球（Ball） | \( \{x \mid \\|x - x_c\\|_2 \le r\} \) | 欧氏球，也叫 2-范数球 |
| 范数球（Norm Ball） | \( \{x \mid \\|x - x_c\\| \le r\} \) | 任意范数定义的球都是凸集 |
| 多面体（Polyhedron） | \( \{x \mid Ax \preceq b,\ Cx = d\} \) | 有限个半空间和超平面的交集 |
| 单纯形（Simplex） | \( \text{conv}\{v_0, v_1, \dots, v_k\} \) | 一组点的凸包，三角形是2维单纯形 |
| 概率单纯形 | \( \{x \in \mathbb{R}^n \mid x_i \ge 0, \sum x_i = 1\} \) | 机器学习中常见的约束集合 |
| 半正定锥 | \( \mathbb{S}^n_+ = \{X \in \mathbb{S}^n \mid X \succeq 0\} \) | 所有半正定矩阵的集合，半定规划的基础 |


> **Example:** **例题：**
> 证明两个凸集 \( C_1, C_2 \) 的交集 \( C_1 \cap C_2 \) 仍是凸集。
>
>
>
>
> **证明：**
> 设 \( x, y \in C_1 \cap C_2 \)，则 \( x, y \in C_1 \) 且 \( x, y \in C_2 \)。对任意 \( \lambda \in [0,1] \)：
>
>
> 由 \( C_1 \) 是凸集，\( \lambda x + (1-\lambda)y \in C_1 \)。
>
>
> 由 \( C_2 \) 是凸集，\( \lambda x + (1-\lambda)y \in C_2 \)。
>
>
> 因此 \( \lambda x + (1-\lambda)y \in C_1 \cap C_2 \)，即交集是凸集。∎


## 三、凸函数（Convex Function）


### 3.1 凸函数的定义


函数 \( f: \mathbb{R}^n \to \mathbb{R} \) 是定义在凸集 \( C \) 上的**凸函数**，当且仅当对任意 \( x, y \in C \) 和 \( \lambda \in [0, 1] \)：


$$
\(f(\lambda x + (1-\lambda)y) \le \lambda f(x) + (1-\lambda) f(y)\)
$$


若不等号严格成立（\( x \ne y, \lambda \in (0,1) \)），则称 \( f \) 为**严格凸函数**。


若不等号反向，则称为**凹函数**（concave function），即 \( -f \) 为凸函数。


### 3.2 判定定理


**一阶条件（First-order condition）：**


$$
\(f(y) \ge f(x) + \nabla f(x)^T (y - x)\)
$$


即凸函数在任意点的切平面（超平面）是函数的全局下界。这提供了凸函数的线性下近似。


**二阶条件（Second-order condition）：**


$$
\(\nabla^2 f(x) \succeq 0 \quad \text{（Hessian矩阵半正定）}\)
$$


对于标量函数 \( f: \mathbb{R} \to \mathbb{R} \)，二阶条件退化为 \( f''(x) \ge 0 \)。


> **Important:** **核心定理：**
> 二阶条件是判断函数凸性的最常用工具。计算Hessian矩阵并验证其半正定性即可。


### 3.3 常见凸函数与非凸函数


| 凸函数 | 非凸函数 |
| --- | --- |
| \( ax + b \)（仿射函数） | \( -\log x \)（在 \( x>0 \) 上凹） |
| \( x^2, \; x^4 \)（偶次幂） | \( \sin x \) |
| \( e^{ax} \) | \( x^3 \) |
| \( \\|x\\|_p \; (p \ge 1) \) | \( \frac{1}{x} \; (x>0) \) |
| \( x\log x \; (x>0) \) | \( \log(1+e^{-x}) \) 的负值 |
| \( \max(x_1, \dots, x_n) \) | 多层神经网络损失函数（一般非凸） |
| \( \\|x\\|_2^2 \) | 含有 ReLU 的复合非凸组合 |


## 四、Jensen不等式与凸函数性质


### 4.1 Jensen不等式


设 \( f \) 是凸函数，\( \lambda_1, \dots, \lambda_k \ge 0 \) 且 \( \sum \lambda_i = 1 \)，则：


$$
\(f\left(\sum_{i=1}^k \lambda_i x_i\right) \le \sum_{i=1}^k \lambda_i f(x_i)\)
$$


这是凸函数定义在多点上的推广。Jensen不等式是概率论与信息论中许多重要不等式的基础。


> **Example:** **概率形式：**
> 若 \( f \) 是凸函数，则 \( f(\mathbb{E}[X]) \le \mathbb{E}[f(X)] \)。
>
>
>
>
> **应用：**
> 由此可推导出 \( \log \) 的凹性给出 \( \mathbb{E}[\log X] \le \log \mathbb{E}[X] \)，即算术-几何均值不等式。


### 4.2 凸函数的极值性质


> **Important:** **核心结论：**
> 凸函数的
> **局部极小值一定是全局极小值**
> 。
>
>
>
>
> 对于严格凸函数，全局极小值如果存在则
> **唯一**
> 。
>
>
>
>
> 这是凸优化如此重要的根本原因：我们不需要担心陷入局部最优。


- **一阶最优性条件：**
   若 \( f \) 可微且凸，则 \( x^* \) 是全局极小值 \(\iff\) \( \nabla f(x^*) = 0 \)
- **无约束凸优化：**
   令梯度等于零即可求解（如果是解析形式）
- **有约束凸优化：**
   需引入KKT条件（后续章节详述）


### 4.3 保凸运算


以下操作保持函数的凸性：


- 非负加权求和：\( \sum \alpha_i f_i(x) \)（\( \alpha_i \ge 0 \)，\( f_i \) 凸）
- 与仿射函数复合：\( f(Ax + b) \)（\( f \) 凸）
- 逐点最大值：\( \max(f_1(x), \dots, f_k(x)) \)（每个 \( f_i \) 凸）
- 逐点上确界：\( \sup_{y \in \mathcal{Y}} f(x, y) \)（对每个 \( y \)，\( f(\cdot, y) \) 凸）
- 凸函数的复合：\( g(f(x)) \)（当 \( g \) 凸且非减，\( f \) 凸时）


## 五、Python代码示例


### 5.1 用numpy判断函数的凸性（Hessian半正定）


```
import numpy as np

def check_convexity_by_hessian(func_hessian, x_samples):
    """
    通过采样点计算Hessian矩阵并验证半正定性来判断函数凸性。
    参数：
        func_hessian: 返回Hessian矩阵的函数
        x_samples: 采样点列表
    返回：
        is_convex: 是否在所有采样点处Hessian半正定
    """
    all_positive_semidefinite = True
    for x in x_samples:
        H = func_hessian(x)
        eigenvalues = np.linalg.eigvalsh(H)
        if np.any(eigenvalues < -1e-8):
            all_positive_semidefinite = False
            print(f"  x={x}, 特征值={eigenvalues}, 非凸!")
        else:
            print(f"  x={x}, 特征值={eigenvalues}, 凸 ✓")
    return all_positive_semidefinite

# 示例1: f(x) = x^2 + 3y^2 （凸函数）
def hessian_f1(x):
    return np.array([[2, 0], [0, 6]])

# 示例2: f(x) = sin(x) （非凸函数, 1D用2x2矩阵近似）
def hessian_f2(x):
    val = -np.sin(x[0])
    return np.array([[val, 0], [0, 1e-6]])

print("=== 检验 f(x,y) = x² + 3y² 的凸性 ===")
samples = [np.array([x, y]) for x in [-2, 0, 2] for y in [-2, 0, 2]]
result1 = check_convexity_by_hessian(hessian_f1, samples)
print(f"结果: {'凸函数' if result1 else '非凸函数'}\n")

print("=== 检验 f(x) = sin(x) 的凸性（在 [-π, π] 采样）===")
samples2 = [np.array([t, 0]) for t in np.linspace(-np.pi, np.pi, 7)]
result2 = check_convexity_by_hessian(hessian_f2, samples2)
print(f"结果: {'凸函数' if result2 else '非凸函数'}")
```


### 5.2 可视化凸函数与非凸函数


```
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 500)

# 凸函数
f_convex = x**2
f_exp = np.exp(x)

# 非凸函数
f_sin = np.sin(x)
f_cubic = x**3 - 3*x

fig, axes = plt.subplots(2, 2, figsize=(12, 10))

# 凸函数可视化
axes[0, 0].plot(x, f_convex, 'b-', linewidth=2)
axes[0, 0].set_title('凸函数: f(x) = x²', fontsize=14)
# 画两点连线验证凸性
x1, x2 = -1.5, 1.5
lam = 0.5
x_mid = lam * x1 + (1-lam) * x2
axes[0, 0].plot([x1, x2], [x1**2, x2**2], 'r--', label='弦（chord）')
axes[0, 0].scatter([x_mid], [x_mid**2], color='green', s=100, zorder=5,
                   label='f(λx₁+(1-λ)x₂) ≤ 弦上的值')
axes[0, 0].legend(fontsize=10)

axes[0, 1].plot(x, f_exp, 'b-', linewidth=2)
axes[0, 1].set_title('凸函数: f(x) = eˣ', fontsize=14)
axes[0, 1].set_ylim(-1, 10)

# 非凸函数可视化
axes[1, 0].plot(x, f_sin, 'r-', linewidth=2)
axes[1, 0].set_title('非凸函数: f(x) = sin(x)', fontsize=14)
axes[1, 0].axhline(y=0, color='gray', linestyle='--')

axes[1, 1].plot(x, f_cubic, 'r-', linewidth=2)
axes[1, 1].set_title('非凸函数: f(x) = x³ - 3x', fontsize=14)
axes[1, 1].axhline(y=0, color='gray', linestyle='--')

for ax in axes.flat:
    ax.grid(True, alpha=0.3)
    ax.set_xlabel('x', fontsize=12)

plt.tight_layout()
plt.savefig('convex_vs_nonconvex.png', dpi=150)
plt.show()
```


## 六、凸集与凸函数在机器学习中的意义


- **损失函数设计：**
   逻辑回归使用交叉熵损失（凸函数），保证梯度下降能收敛到全局最优
- **SVM：**
   目标函数是凸二次函数，约束是凸集（半空间），整个问题是凸优化
- **LASSO / Ridge：**
   目标函数 = 凸损失 + 凸正则项，整体仍是凸函数
- **深度学习的挑战：**
   神经网络损失函数一般非凸，因此需要精心设计优化器和初始化策略
- **凸松弛：**
   将非凸问题松弛为凸问题是求解困难问题的重要手段（如LASSO是组合优化的松弛）


> **Note:** **总结：**
> 凸优化理论是现代优化方法的基石。虽然深度学习中的问题通常是非凸的，但凸优化中发展出的方法（梯度下降、对偶理论、KKT条件等）仍然是解决非凸问题的重要工具。理解凸优化，是理解更复杂优化算法的必经之路。
