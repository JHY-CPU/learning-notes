# 对偶理论 — 最优化理论笔记


## 一、原始问题与对偶问题


### 1.1 原始问题（Primal Problem）


$$
\(p^* = \min_{x} f(x) \quad \text{s.t.} \quad g_i(x) \le 0, \; h_j(x) = 0\)
$$


其拉格朗日函数为：


$$
\(\mathcal{L}(x, \lambda, \nu) = f(x) + \sum_i \lambda_i g_i(x) + \sum_j \nu_j h_j(x)\)
$$


原始问题可以等价地写为：


$$
\(p^* = \min_x \max_{\lambda \ge 0, \nu} \mathcal{L}(x, \lambda, \nu)\)
$$


### 1.2 对偶函数（Dual Function）


定义**对偶函数**为拉格朗日函数关于 \( x \) 的下确界：


$$
\(g(\lambda, \nu) = \inf_x \mathcal{L}(x, \lambda, \nu)\)
$$


关键性质：


- 对偶函数一定是
   **凹函数**
   （无论原始问题是否凸）——因为它是关于 \( (\lambda, \nu) \) 的仿射函数族的逐点下确界
- 对偶函数给出了原始最优值的
   **下界**
   ：\( g(\lambda, \nu) \le p^* \)（对所有 \( \lambda \ge 0 \)）


### 1.3 对偶问题（Dual Problem）


$$
\(d^* = \max_{\lambda \ge 0, \nu} g(\lambda, \nu) = \max_{\lambda \ge 0, \nu} \inf_x \mathcal{L}(x, \lambda, \nu)\)
$$


对偶问题是寻找拉格朗日函数下确界的最大值——即在所有下界中找最紧的那个。


> **Note:** **核心直觉：**
> 原始问题是从上方逼近最优值（找可行解中最小的目标值），对偶问题是从下方逼近（找最紧的下界）。两者"夹逼"时，就得到了最优值。


## 二、弱对偶定理与强对偶定理


### 2.1 弱对偶定理（Weak Duality）


> **Important:** **弱对偶定理：**
> 对于任意优化问题，都有 \( d^* \le p^* \)。
>
>
> 即对偶问题的最优值不超过原始问题的最优值。这个定理
> **无条件成立**
> 。


定义**对偶间隙（Duality Gap）**：


$$
\(\text{Gap} = p^* - d^* \ge 0\)
$$


弱对偶定理保证对偶间隙非负。它在以下场景特别有用：


- 即使找不到原始最优解 \( p^* \)，对偶最优值 \( d^* \) 也给出了一个下界，可以评估当前解的质量
- 近似算法中用对偶间隙衡量近似比


### 2.2 强对偶定理（Strong Duality）


> **Important:** **强对偶定理：**
> 若原始问题是凸优化问题且满足某些正则性条件，则 \( d^* = p^* \)，即对偶间隙为零。


强对偶的意义：


- 可以通过求解对偶问题来间接求解原始问题
- 对偶问题可能比原始问题更容易求解（例如维度更低、无约束等）
- KKT条件的推导依赖于强对偶


## 三、Slater条件


**Slater条件**是保证凸优化问题强对偶成立的充分条件：


$$
\(\exists \, \tilde{x} \in \text{relint}(\mathcal{D}) : g_i(\tilde{x}) < 0, \; h_j(\tilde{x}) = 0, \; \forall i, j\)
$$


其中 relint 表示相对内点。直观来说，Slater条件要求存在一个**严格可行点**——满足所有不等式约束严格成立的点。


| 条件 | 含义 | 是否保证强对偶 |
| --- | --- | --- |
| 一般问题 | 无特殊结构 | 不保证 |
| 凸问题 + Slater条件 | 存在严格可行内点 | 保证强对偶 |
| 凸问题（仅含仿射不等式约束） | \( g_i \) 为仿射函数 | Slater条件自动满足 |
| 线性规划（LP） | 所有函数仿射 | 只要可行非空，强对偶即成立 |


> **Example:** **例：**
> 问题 \( \min x \text{ s.t. } x^2 \le 0 \)。唯一可行解是 \( x=0 \)，但不存在严格可行点（\( x^2 < 0 \) 无解）。Slater条件不满足，强对偶不一定成立。


## 四、对偶问题的构造方法与应用


### 4.1 构造步骤


1. 写出拉格朗日函数 \( \mathcal{L}(x, \lambda, \nu) \)
2. 固定 \( \lambda, \nu \)，求 \( \inf_x \mathcal{L}(x, \lambda, \nu) \) 得到对偶函数 \( g(\lambda, \nu) \)
3. 求解 \( \max_{\lambda \ge 0} g(\lambda, \nu) \) 得到对偶问题


### 4.2 对偶问题的优势


- 对偶问题总是凹优化（最大化凹函数），即使原始问题是非凸的
- 对偶问题的约束通常更简单（\( \lambda \ge 0 \)）
- 某些问题的对偶形式有特殊结构，可用高效算法求解
- 对偶变量具有经济/物理含义（影子价格、灵敏度）


### 4.3 SVM中的对偶问题


SVM原始问题：


$$
\(\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^T x_i + b) \ge 1, \; \forall i\)
$$


对偶问题（通过对偶变换得到）：


$$
\(\max_\alpha \sum_i \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j x_i^T x_j\)
                \(\text{s.t.} \quad \alpha_i \ge 0, \quad \sum_i \alpha_i y_i = 0\)
$$


其中 \( \alpha_i \) 是对偶变量（拉格朗日乘子）。互补松弛性 \( \alpha_i(y_i(w^Tx_i+b)-1)=0 \) 告诉我们：只有 \( \alpha_i > 0 \) 的样本是**支持向量**，位于分类边界上。


> **Note:** **核技巧：**
> 对偶问题中数据仅以内积 \( x_i^T x_j \) 的形式出现，可以用核函数 \( K(x_i, x_j) \) 替换，实现非线性分类。这是对偶理论在机器学习中最经典的应用。


## 五、Python代码：对偶问题求解


```
import numpy as np
from scipy.optimize import minimize

# ==========================================
# 原始问题: min x1^2 + x2^2  s.t. x1 + x2 >= 2
# 等价: min x1^2 + x2^2  s.t. 2 - x1 - x2 <= 0
# ==========================================

# --- 原始问题求解 ---
def primal_objective(x):
    return x[0]**2 + x[1]**2

def primal_constraint(x):
    return x[0] + x[1] - 2  # >= 0

result_primal = minimize(primal_objective, [0, 0],
                         constraints={'type': 'ineq', 'fun': primal_constraint})
print("=== 原始问题 ===")
print(f"最优解: ({result_primal.x[0]:.4f}, {result_primal.x[1]:.4f})")
print(f"最优值 p* = {result_primal.fun:.4f}")

# --- 对偶问题分析 ---
# L(x, λ) = x1^2 + x2^2 + λ(2 - x1 - x2)
# ∂L/∂x1 = 2x1 - λ = 0 => x1 = λ/2
# ∂L/∂x2 = 2x2 - λ = 0 => x2 = λ/2
# g(λ) = L(λ/2, λ/2, λ) = λ²/2 - λ²/2 + 2λ - λ²/2 = 2λ - λ²/2
# 对偶问题: max 2λ - λ²/2  s.t. λ >= 0
# 最优: λ* = 2, d* = 2

def dual_function(lam):
    return -(2*lam - lam**2/2)  # 取负因为minimize

result_dual = minimize(dual_function, [1.0], bounds=[(0, None)])
lambda_star = result_dual.x[0]
d_star = -result_dual.fun

print(f"\n=== 对偶问题 ===")
print(f"最优乘子: λ* = {lambda_star:.4f}")
print(f"最优值 d* = {d_star:.4f}")
print(f"对偶间隙: {result_primal.fun - d_star:.6f}")
print(f"强对偶成立: {abs(result_primal.fun - d_star) < 1e-4}")

# --- 验证互补松弛 ---
x_star = result_primal.x
g_val = 2 - x_star[0] - x_star[1]
print(f"\n=== KKT验证 ===")
print(f"g(x*) = {g_val:.6f}")
print(f"λ* * g(x*) = {lambda_star * g_val:.6f} (互补松弛)")
print(f"∇f(x*) = ({2*x_star[0]:.4f}, {2*x_star[1]:.4f})")
print(f"λ* * ∇g = ({-lambda_star:.4f}, {-lambda_star:.4f})")
print(f"∇f + λ*∇g = ({2*x_star[0]-lambda_star:.4f}, {2*x_star[1]-lambda_star:.4f}) (应为0)")
```


## 六、对偶理论的核心要点


| 概念 | 关键结论 |
| --- | --- |
| 弱对偶 | \( d^* \le p^* \) 恒成立 |
| 强对偶 | \( d^* = p^* \)，凸+Slater条件保证 |
| 对偶函数 | 必为凹函数，给出原始最优值的下界 |
| 对偶间隙 | 可用于评估近似解质量 |
| Slater条件 | 存在严格可行点，保证强对偶 |
| KKT + 强对偶 | 最优解满足KKT条件，对偶变量有实际意义 |


> **Note:** **总结：**
> 对偶理论将原始约束优化问题转化为（通常是更简单的）对偶问题。弱对偶提供了下界，强对偶使得两个问题等价。Slater条件是判断强对偶是否成立的关键工具。在SVM、信号处理、经济学等领域，对偶理论都是核心方法。
