# 蒙特卡洛方法


## 蒙特卡洛方法 (Monte Carlo)


强化学习MC无模型


蒙特卡洛方法通过采样完整的回合（episode）来估计价值函数，不需要环境的转移概率模型。


## MC基本思想


```
核心思想：用实际回报的平均值来估计价值函数

价值函数定义：
V^π(s) = E_π[ Gt | St = s ]
= E_π[ R(t+1) + γR(t+2) + γ²R(t+3) + ... | St = s ]

MC估计：
V^π(s) ≈ 所有经过s的回合的回报Gt的平均值

特点：
+ 不需要环境模型（Model-Free）
+ 从实际经验中学习
+ 只能用于episodic任务（有终止状态）
- 必须等到回合结束才能更新
- 无法在线实时学习
```


## 首次访问 vs 每次访问


```
首次访问MC (First-Visit MC)：
- 每个回合中，只用第一次访问状态s时的回报来更新V(s)
- 统计无偏，理论性质更好
- 每个回合对每个状态最多更新一次

每次访问MC (Every-Visit MC)：
- 每个回合中，每次访问状态s都用回报更新V(s)
- 实现更简单
- 统计有偏但一致性仍然成立

首次访问算法：
for each episode:
    生成回合: S0, A0, R1, S1, A1, R2, ..., ST
    G = 0
    for t = T-1, T-2, ..., 0:
        G = γG + R(t+1)
        除非 St 在 S0, S1, ..., S(t-1) 中出现过:
            returns(St).append(G)
            V(St) = average(returns(St))
```


## 动作价值MC (MC for Action Values)


```
问题：MC估计Q(s,a)需要知道在状态s下选择动作a的概率
如果策略是确定性的，可能永远不会访问某些(s,a)对

解决方案：探索性起步 (Exploring Starts)
- 每个回合随机选择初始状态和动作
- 确保所有(s,a)对都有机会被访问

另一种方案：ε-软策略
- 策略始终有ε概率随机选择动作
- 保证所有动作都有至少 ε/|A| 的选择概率

Q值更新：
Q(s,a) = average(Gt) for all t where St=s, At=a

最优策略：
π*(s) = argmax_a Q(s,a)
```


> **Note:** 探索性起步假设在实践中很难满足。实际中更常用ε-贪婪策略来保证充分探索。


## 同策略 vs 异策略


```
同策略 (On-Policy)：
- 用于采样的策略（行为策略）= 要评估/改进的策略（目标策略）
- 更简单、更稳定
- 但可能收敛到ε-软策略而非最优策略

异策略 (Off-Policy)：
- 行为策略 b(a|s) 用于采样
- 目标策略 π(a|s) 是要学习的策略
- 可以从其他策略的数据中学习
- 可以学习最优策略同时用探索性策略采样

异策略的关键：重要性采样 (Importance Sampling)
ρ(t:T-1) = Π(π(At|St) / b(At|St))
  t到T-1每步的π/b的乘积

加权重要性采样：
V(s) = Σ ρ(t:T-1) × Gt / Σ ρ(t:T-1)
- 用ρ的比值作为权重
- 普通重要性采样：无偏但方差极大
- 加权重要性采样：有偏但方差小得多
```


## 重要性采样 (Importance Sampling)


```
目标：用分布b的样本估计分布π的期望
E_π[f(x)] = Σ π(x) × f(x)
           = Σ b(x) × (π(x)/b(x)) × f(x)
           = E_b[(π(x)/b(x)) × f(x)]

两种估计器：

普通重要性采样 (Ordinary IS)：
V(s) = (1/|returns(s)|) × Σ ρ(t:T-1) × Gt
- 无偏估计
- 方差可能非常大（ρ是连乘积）

加权重要性采样 (Weighted IS)：
V(s) = Σ ρ(t:T-1) × Gt / Σ ρ(t:T-1)
- 有偏估计（偏差随样本增加趋于0）
- 方差小得多
- 实际中更常用

增量更新（加权IS版本）：
Q(s,a) ← Q(s,a) + (ρ / C(s,a)) × [Gt - Q(s,a)]
C(s,a) ← C(s,a) + ρ
其中ρ = π(a|s) / b(a|s)
```


## 增量MC与常数步长


```
增量平均更新：
V(St) ← V(St) + α × [Gt - V(St)]

常数步长α的优点：
- 无需存储所有回报
- 更重视最近的经验
- 适用于非平稳环境

α的选择：
- α = 1/N(s)：等价于样本平均
- α = 常数（如0.01）：指数加权平均
  Vn+1 = Vn + α(Rn - Vn)
       = (1-α)Vn + αRn
       = (1-α)ⁿV1 + αΣ(1-α)^(n-k)Rk

等效窗口长度 ≈ 1/α
- α=0.1 → 约10步有效窗口
- α=0.01 → 约100步有效窗口
```


> **Note:** 常数步长在非平稳环境中更优，因为它更重视最近的经验。样本平均适用于平稳环境。
