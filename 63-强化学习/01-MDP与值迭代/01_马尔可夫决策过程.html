马尔可夫决策过程 (MDP)

强化学习MDPBellman

MDP是强化学习的数学框架，描述智能体与环境的交互过程。

MDP核心要素

MDP五元组 (S, A, P, R, γ)： S：状态空间 (State Space) 环境所有可能状态的集合 例：棋盘位置、游戏画面、机器人位姿 A：动作空间 (Action Space) 每个状态下可用动作的集合 例：上下左右、买入卖出、加速减速 P：状态转移概率 (Transition Probability) P(s'|s, a) = 在状态s执行动作a后转移到s'的概率 描述了环境的动态特性 R：奖励函数 (Reward Function) R(s, a, s') = 在状态s执行动作a转移到s'获得的即时奖励 设计奖励函数是RL的核心挑战 γ：折扣因子 (Discount Factor) 0 ≤ γ ≤ 1 γ=0：只关心即时奖励（贪婪） γ=1：同等关心所有未来奖励 γ=0.99：远期奖励略微折扣 马尔可夫性质： P(St+1 | St, At, St-1, At-1, ...) = P(St+1 | St, At) → 下一状态只依赖于当前状态和动作 → 与历史路径无关

策略与价值函数

策略 (Policy) π(a|s)： - 在状态s下选择动作a的概率分布 - 确定性策略：π(s) = a（直接给出动作） - 随机性策略：π(a|s) = P(A=a|S=s) 价值函数： ┌─────────────────────────────────────────────┐ │ 状态价值函数 V^π(s)： │ │ 从状态s出发，按策略π行动，期望累积奖励 │ │ │ │ V^π(s) = E_π[ Σ γ^t × Rt+1 | S0=s ] │ │ = E_π[ R + γ × V^π(S') | S=s ] │ │ │ │ 动作价值函数 Q^π(s,a)： │ │ 在状态s执行动作a，然后按策略π行动的期望累积奖励│ │ │ │ Q^π(s,a) = E_π[ Σ γ^t × Rt+1 | S0=s, A0=a ]│ │ = R(s,a) + γ × Σ P(s'|s,a) V^π(s')│ │ │ │ 关系：V^π(s) = Σ π(a|s) × Q^π(s,a) │ └─────────────────────────────────────────────┘ 最优策略 π*： - 最大化所有状态的价值 - V*(s) = max_π V^π(s) - 最优策略一定存在（至少一个）

Bellman方程

Bellman期望方程（给定策略π）： V^π(s) = Σ_a π(a|s) × [R(s,a) + γ × Σ_s' P(s'|s,a) V^π(s')] Q^π(s,a) = R(s,a) + γ × Σ_s' P(s'|s,a) × Σ_a' π(a'|s') Q^π(s',a') Bellman最优方程（求最优价值）： V*(s) = max_a [ R(s,a) + γ × Σ_s' P(s'|s,a) V*(s') ] Q*(s,a) = R(s,a) + γ × Σ_s' P(s'|s,a) × max_a' Q*(s',a') 最优策略： π*(s) = argmax_a Q*(s,a) = argmax_a [ R(s,a) + γ × Σ_s' P(s'|s,a) V*(s') ] 理解Bellman方程： 当前价值 = 即时奖励 + 折扣后的未来价值 → 递归关系，当前状态价值依赖于后继状态价值 → 通过迭代求解
Bellman方程是所有RL算法的基础。值迭代和策略迭代直接求解Bellman方程， Q-learning和SARSA用采样近似求解。

值迭代与策略迭代

值迭代 (Value Iteration)： 重复更新直到收敛： V_{k+1}(s) = max_a [ R(s,a) + γ × Σ_s' P(s'|s,a) V_k(s') ] 收敛后：π*(s) = argmax_a [ R(s,a) + γ × Σ P(s'|s,a) V*(s') ] 策略迭代 (Policy Iteration)： 1. 初始化随机策略 π 2. 策略评估：计算 V^π(s)（求解线性方程组或迭代） 3. 策略改进：π'(s) = argmax_a Q^π(s,a) 4. 如果π'=π，停止；否则π=π'，回到2 对比： ┌──────────┬──────────────┬──────────────────┐ │ │ 值迭代 │ 策略迭代 │ ├──────────┼──────────────┼──────────────────┤ │ 每步 │ 更新值函数 │ 评估+改进策略 │ │ 收敛速度 │ 较慢 │ 较快（迭代次数少）│ │ 计算量 │ 每步简单 │ 每步较重 │ │ 适用 │ 状态空间小 │ 策略空间小 │ └──────────┴──────────────┴──────────────────┘ 两者都需要知道完整的P和R（模型已知 → Model-Based） 当模型未知时，使用Q-learning等Model-Free方法
