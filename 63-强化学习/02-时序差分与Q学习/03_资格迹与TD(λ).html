资格迹与TD(λ)

强化学习资格迹TD(λ)

资格迹统一了MC和TD方法，通过λ参数在两者之间连续调节，是n-step TD的加权组合。

资格迹动机

MC vs TD的权衡： - MC：使用完整回报，无偏但高方差，需要等回合结束 - TD(0)：使用一步TD误差，低方差但有偏，在线实时 n-step TD： G(t:t+n) = R(t+1) + γR(t+2) + ... + γ^n V(St+n) - n=1：TD(0) - n=∞：MC - 中间值：偏差和方差的折中 TD(λ) = 所有n-step回报的加权平均： G^λ(t) = (1-λ) × Σ λ^(n-1) × G(t:t+n) - λ=0：等价于TD(0) - λ=1：等价于MC - 0 < λ < 1：两者之间的插值

资格迹 (Eligibility Trace)

资格迹 e_t(s) 记录了每个状态"最近被访问的程度" 当TD误差发生时： δ(t) = R(t+1) + γV(St+1) - V(St) 所有状态按照其资格迹的值获得更新： V(s) ← V(s) + α × δ(t) × e_t(s) 这意味着： - 最近访问的状态获得最大的更新 - 更早访问的状态获得衰减后的更新 - 未访问的状态不更新（资格迹=0） 等效效果： 资格迹自动实现了n-step回报的加权组合 不需要显式存储n步数据

前向视角 vs 后向视角

前向视角 (Forward View)： - 看未来的n步回报 - 需要等到知道未来的奖励和状态 - 理论上清晰，但实际难以实现 - TD(λ)回报 = (1-λ) × Σ λ^(n-1) × G(t:t+n) 后向视角 (Backward View)： - 使用资格迹在线更新 - 不需要等待未来信息 - 是前向视角的在线实现 - 每步都做更新 后向视角算法： 初始化 V(s)，对所有 s 初始化 e(s) = 0 for each step of episode: 选择动作A，观察R, S' δ = R + γV(S') - V(S) e(S) ← e(S) + 1 (或 e(S) ← 1) for all s: V(s) ← V(s) + α × δ × e(s) e(s) ← γ × λ × e(s) (资格迹衰减) S ← S'

累积迹 vs 替换迹

累积资格迹 (Accumulating Trace)： e(s) ← e(s) + 1 （每次访问加1） 问题：频繁访问时资格迹可能非常大 替换资格迹 (Replacing Trace)： e(s) ← 1 （每次访问直接设为1） 优势：资格迹有界，更稳定 比较： 状态s被连续访问3次： 累积迹： e(s) → 1 → 2 → 3 → 衰减 替换迹： e(s) → 1 → 1 → 1 → 衰减 实际中替换迹往往表现更好 （对线性函数逼近也更稳定） Dutch Trace (荷兰迹)： e(s) ← (1-α) × e(s) + 1 - 累积迹和替换迹之间的折中 - 在非线性函数逼近中表现更好 - PPO等现代算法常用
替换迹在表格型方法中通常优于累积迹。Dutch trace在函数逼近中更优。

SARSA(λ) 与 Watkins's Q(λ)

SARSA(λ)： 将资格迹应用到SARSA，同策略学习 for each step: 选择动作A（ε-greedy），观察R, S' 选择动作A'（ε-greedy） δ = R + γQ(S',A') - Q(S,A) e(S,A) ← e(S,A) + 1 (替换迹) for all s,a: Q(s,a) ← Q(s,a) + α × δ × e(s,a) e(s,a) ← γ × λ × e(s,a) S ← S', A ← A' Watkins's Q(λ)： 将资格迹应用到Q-Learning，异策略学习 问题：Q-Learning使用max操作，当选择的动作≠greedy动作时怎么办？ 解决方案：清除资格迹 if A' was chosen by ε-greedy (not max): δ = R + γQ(S',A') - Q(S,A) // SARSA更新 else: δ = R + γ × max_a Q(S',a) - Q(S,A) // Q-Learning更新 for all s,a: Q(s,a) ← Q(s,a) + α × δ × e(s,a) e(s,a) ← γ × λ × e(s,a) × (是否greedy) 如果A'不是greedy动作：e(s,a)全部清零

λ参数选择

λ的含义： - 控制看多远的未来 - λ=0：只看一步（TD(0)），偏差最大方差最小 - λ=1：看完整回报（MC），无偏方差最大 - 0.9~0.99：常用范围 有效视野（Effective Horizon）： ≈ 1 / (1 - λ) - λ=0.9 → 约10步 - λ=0.99 → 约100步 - λ=0.999 → 约1000步 经验法则： - 短期依赖任务：λ较小（0.5~0.8） - 长期依赖任务：λ较大（0.9~0.99） - 在线学习：λ较小（快速响应） - 离线学习：λ可以更大 λ与α的交互： - λ大时需要α较小（因为方差增大） - λ小时可以用较大的α - 实际中需要联合调参
TD(λ)用一个参数λ统一了MC和TD的频谱，是强化学习中连接两种方法的桥梁。资格迹使更新更加"即时"且"宽泛"。
