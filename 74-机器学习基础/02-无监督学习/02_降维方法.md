## 1. 为什么需要降维


- **维度灾难**
   ：高维空间中数据稀疏，距离度量失效
- **可视化**
   ：将数据降到 2D/3D 便于观察
- **去噪**
   ：去除噪声维度，保留主要信息
- **加速计算**
   ：减少特征数，提高模型训练速度
- **减少存储**
   ：压缩数据表示


### 降维的分类


| 类型 | 方法 | 特点 |
| --- | --- | --- |
| 线性 | PCA、LDA | 线性变换，可解释性好 |
| 非线性 | t-SNE、UMAP | 捕捉非线性结构 |
| 有监督 | LDA | 利用类别标签信息 |
| 无监督 | PCA、t-SNE、UMAP | 不需要标签 |


## 2. PCA 主成分分析


PCA（Principal Component Analysis）是最经典的线性降维方法，通过找到数据方差最大的方向进行投影。


### 2.1 核心思想


找到一组正交基（主成分），使得数据在这些方向上的投影方差最大。第一主成分方向方差最大，第二主成分与第一正交且方差次大，以此类推。


### 2.2 直觉理解


- 将数据的坐标系旋转到方差最大的方向
- 保留前 k 个主成分，丢弃其余维度
- 等价于在最小化重构误差的意义下找到最优的 k 维子空间


## 3. PCA 数学推导


### 3.1 算法步骤


1. **数据中心化**
   ：x̃ = x - μ（每列减去均值）
2. **计算协方差矩阵**
   ：C = (1/m) X̃ᵀX̃
3. **特征分解**
   ：C = UΛUᵀ，U 为特征向量矩阵，Λ 为特征值对角矩阵
4. **选择主成分**
   ：取前 k 个最大特征值对应的特征向量
5. **投影**
   ：Z = X̃ · Uₖ（Uₖ 为前 k 个特征向量组成的矩阵）


### 3.2 协方差矩阵


$$
C = (1/m) Σᵢ₌₁ᵐ (x⁽ⁱ⁾ - μ)(x⁽ⁱ⁾ - μ)ᵀ
$$


协方差矩阵 C 是对称半正定矩阵，其特征值 λᵢ 表示第 i 个主成分方向上的方差。


### 3.3 方差解释率


$$
累积方差贡献率 = Σᵢ₌₁ᵏ λᵢ / Σⱼ₌₁ᵈ λⱼ
$$


通常选择累积贡献率达到 90%~95% 的最小 k 值。


### 3.4 SVD 方法


实际中常用 SVD（奇异值分解）代替特征分解：


$$
X̃ = UΣVᵀ
$$


- V 的前 k 列即为前 k 个主成分方向
- Σ²/m 即为特征值
- SVD 更数值稳定，适用于特征数多的情况


### 3.5 核 PCA


标准 PCA 只能捕捉线性结构。核 PCA 通过核技巧将数据映射到高维空间再做 PCA，能捕捉非线性结构。


## 4. t-SNE


t-SNE（t-distributed Stochastic Neighbor Embedding）是一种非线性降维方法，特别适合高维数据的可视化。


### 4.1 核心思想


将高维空间中的相似性（概率分布）转换为低维空间中的相似性，并最小化两个分布之间的 KL 散度。


### 4.2 高维空间的相似性


$$
pⱼ|ᵢ = exp(-||xᵢ - xⱼ||² / 2σᵢ²) / Σₖ≠ᵢ exp(-||xᵢ - xₖ||² / 2σᵢ²)
$$


使用高斯核将距离转换为条件概率。σᵢ 通过困惑度（Perplexity）参数自动确定。


### 4.3 低维空间的相似性


$$
qᵢⱼ = (1 + ||yᵢ - yⱼ||²)⁻¹ / Σₖ≠ₗ (1 + ||yₖ - yₗ||²)⁻¹
$$


使用 t 分布（自由度为 1）代替高斯分布，解决"拥挤问题"（低维空间中点倾向于聚集在中心）。


### 4.4 优化目标


最小化高维和低维分布之间的 KL 散度：


$$
KL(P||Q) = Σᵢ Σⱼ pᵢⱼ log(pᵢⱼ / qᵢⱼ)
$$


### 4.5 t-SNE 的特点


- **优点**
   ：能很好地保留局部结构，可视化效果出色
- **缺点**
   ：计算复杂度 O(n²)，不能用于新数据投影，不同运行结果不同
- **参数**
   ：困惑度（Perplexity，通常 5-50）、学习率、迭代次数
- **注意**
   ：簇间距离和簇大小没有意义，不要过度解读


## 5. UMAP


UMAP（Uniform Manifold Approximation and Projection）是基于流形学习和拓扑学的降维方法，比 t-SNE 更快且更好地保留全局结构。


### 5.1 核心思想


1. 在高维空间中构建数据的模糊单纯复形（fuzzy simplicial complex）
2. 在低维空间中构建类似的结构
3. 最小化两个结构之间的交叉熵


### 5.2 UMAP vs t-SNE


| 特性 | t-SNE | UMAP |
| --- | --- | --- |
| 速度 | 慢 O(n²) | 快 O(n^1.14) |
| 全局结构 | 保留较差 | 保留较好 |
| 新数据投影 | 不支持 | 支持 transform |
| 理论基础 | 信息论 | 黎曼几何/代数拓扑 |
| 可调参数 | 困惑度 | n_neighbors, min_dist |


### 5.3 关键参数


- **n_neighbors**
   ：局部邻域大小，越大保留越多全局结构（5-50）
- **min_dist**
   ：低维空间中点的最小距离，越小簇越紧凑（0.01-0.5）


## 6. LDA 线性判别分析


LDA（Linear Discriminant Analysis）是一种有监督的降维方法，目标是找到能最大化类别间分离的投影方向。


### 6.1 与 PCA 的区别


- PCA 最大化
   **方差**
   （无监督），LDA 最大化
   **类别可分性**
   （有监督）
- PCA 降维到任意维度，LDA 最多降到 K-1 维（K 为类别数）


### 6.2 优化目标


最大化类间散布与类内散布之比（Fisher 准则）：


$$
J(w) = (wᵀSᴮw) / (wᵀSᵂw)
$$


其中：


- **Sᴮ**
   ：类间散布矩阵 = Σₖ nₖ(μₖ - μ)(μₖ - μ)ᵀ
- **Sᵂ**
   ：类内散布矩阵 = Σₖ Σ_{x∈Cₖ} (x - μₖ)(x - μₖ)ᵀ


### 6.3 求解


最优 w 是 Sᵂ⁻¹Sᴮ 的最大特征值对应的特征向量。实际中先用 PCA 降维到合适的维度（避免 Sᵂ 不可逆），再用 LDA。


## 7. 方法对比


| 方法 | 类型 | 速度 | 可视化 | 特征工程 | 适用场景 |
| --- | --- | --- | --- | --- | --- |
| PCA | 线性/无监督 | 快 | 一般 | 优秀 | 通用降维、去噪 |
| t-SNE | 非线性/无监督 | 慢 | 优秀 | 不适用 | 数据可视化 |
| UMAP | 非线性/无监督 | 快 | 优秀 | 尚可 | 可视化+特征工程 |
| LDA | 线性/有监督 | 快 | 一般 | 优秀 | 分类任务降维 |


> **Tip:** **选型指南：**
> 需要特征降维用 PCA；需要可视化用 UMAP（大数据）或 t-SNE（小数据）；分类任务的降维用 LDA。


## 8. Python 代码实现


### 8.1 PCA


```
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

# 标准化后做 PCA
X_scaled = StandardScaler().fit_transform(X)
pca = PCA(n_components=0.95)  # 保留 95% 方差
X_pca = pca.fit_transform(X_scaled)

print(f"降维后维度: {X_pca.shape[1]}")
print(f"各主成分方差比: {pca.explained_variance_ratio_}")
print(f"累积方差比: {pca.explained_variance_ratio_.cumsum()}")
```


### 8.2 t-SNE


```
from sklearn.manifold import TSNE

tsne = TSNE(n_components=2, perplexity=30, learning_rate='auto',
            n_iter=1000, random_state=42)
X_tsne = tsne.fit_transform(X_scaled)
# 通常先用 PCA 降到 50 维再用 t-SNE
```


### 8.3 UMAP


```
# pip install umap-learn
import umap

reducer = umap.UMAP(n_neighbors=15, min_dist=0.1, n_components=2)
X_umap = reducer.fit_transform(X_scaled)

# 支持新数据投影
X_new_umap = reducer.transform(X_new_scaled)
```


### 8.4 LDA


```
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis as LDA

lda = LDA(n_components=2)  # 最多 K-1 维
X_lda = lda.fit_transform(X_scaled, y)  # 需要标签
print(f"各类别均值: {lda.means_}")
```


## 9. 总结


1. **PCA**
   通过特征分解找到方差最大的方向，是最通用的线性降维方法
2. 选择主成分数目时关注
   **累积方差贡献率**
   （通常 90%~95%）
3. **t-SNE**
   和
   **UMAP**
   用于可视化，前者更经典，后者更快且保留全局结构
4. **LDA**
   是有监督降维，利用标签信息最大化类别可分性
5. 降维前通常需要
   **特征标准化**
