## 1. 感知机 (Perceptron)


感知机是神经网络最基本的组成单元，由 Frank Rosenblatt 于 1957 年提出。它是一个线性二分类模型。


### 1.1 模型定义


$$
f(x) = sign(wᵀx + b)
$$


其中 sign 是符号函数：当 wᵀx + b ≥ 0 时输出 +1，否则输出 -1。


### 1.2 学习算法


感知机使用错误驱动的在线学习：


1. 初始化 w = 0, b = 0
2. 对每个样本 (x⁽ⁱ⁾, y⁽ⁱ⁾)：如果 y⁽ⁱ⁾(wᵀx⁽ⁱ⁾ + b) ≤ 0（误分类），则更新：


$$
w := w + η·y⁽ⁱ⁾·x⁽ⁱ⁾
                b := b + η·y⁽ⁱ⁾
$$


其中 η 是学习率。


### 1.3 收敛定理


若数据线性可分，感知机学习算法必在有限步内收敛。但**解不唯一**（取决于初始值和样本顺序）。


### 1.4 感知机的局限


- 只能解决线性可分问题（如无法解决 XOR 异或问题）
- 单层感知机的表达能力有限
- 解决方案：堆叠多层感知机


## 2. 多层感知机 (MLP)


多层感知机（Multi-Layer Perceptron）通过堆叠多个感知机层，引入非线性激活函数，克服了单层感知机的局限。


### 2.1 网络结构


- **输入层**
   ：接收原始特征，不做计算
- **隐藏层**
   ：一个或多个，进行特征变换
- **输出层**
   ：产生最终预测


### 2.2 前向传播


每一层的计算过程：


$$
z⁽ˡ⁾ = W⁽ˡ⁾a⁽ˡ⁻¹⁾ + b⁽ˡ⁾
                a⁽ˡ⁾ = σ(z⁽ˡ⁾)
$$


其中 l 表示层数，W⁽ˡ⁾ 是权重矩阵，b⁽ˡ⁾ 是偏置向量，σ 是激活函数，a⁽⁰⁾ = x（输入）。


### 2.3 输出层设计


| 任务类型 | 输出层激活 | 损失函数 |
| --- | --- | --- |
| 二分类 | Sigmoid (1 个神经元) | 二元交叉熵 |
| 多分类 | Softmax (K 个神经元) | 多类交叉熵 |
| 回归 | 无激活 (线性) | MSE / MAE |


## 3. 激活函数


激活函数为神经网络引入非线性。没有激活函数，多层网络等价于单层线性变换。


### 3.1 Sigmoid


$$
σ(z) = 1 / (1 + e⁻ᶻ)
$$


- 输出范围 (0, 1)，适合概率输出
- 缺点：梯度消失（|σ'(z)| ≤ 0.25）、非零中心、exp 计算慢


### 3.2 Tanh


$$
tanh(z) = (eᶻ - e⁻ᶻ) / (eᶻ + e⁻ᶻ)
$$


- 输出范围 (-1, 1)，零中心
- 比 Sigmoid 好，但仍存在梯度消失


### 3.3 ReLU (Rectified Linear Unit)


$$
ReLU(z) = max(0, z)
$$


- 计算高效，正区间梯度恒为 1，缓解梯度消失
- 缺点：负区间梯度为 0（"死亡 ReLU"问题）
- 是隐藏层最常用的激活函数


### 3.4 Leaky ReLU / PReLU


$$
LeakyReLU(z) = max(αz, z)，α 通常为 0.01
$$


- 解决死亡 ReLU 问题，负区间保留微小梯度
- PReLU 中 α 作为可学习参数


### 3.5 GELU / Swish


$$
GELU(z) = z · Φ(z) = z · 0.5 · [1 + erf(z/√2)]
                Swish(z) = z · σ(βz)
$$


- GELU：Transformer 模型的默认激活函数
- Swish：Google 提出，β=1 时等价于 SiLU
- 平滑、非单调，在深层网络中表现更好


### 3.6 激活函数对比


| 函数 | 输出范围 | 零中心 | 梯度消失 | 常用位置 |
| --- | --- | --- | --- | --- |
| Sigmoid | (0,1) | 否 | 严重 | 输出层(二分类) |
| Tanh | (-1,1) | 是 | 中等 | RNN 隐藏层 |
| ReLU | [0,∞) | 否 | 不 | 隐藏层(默认) |
| Leaky ReLU | (-∞,∞) | 否 | 不 | 隐藏层 |
| GELU | (-0.17,∞) | 近似 | 不 | Transformer |


## 4. 反向传播算法


反向传播（Backpropagation）是训练神经网络的核心算法，利用链式法则高效计算损失函数对所有参数的梯度。


### 4.1 链式法则


对于复合函数 y = f(g(x))，有 dy/dx = (df/dg) · (dg/dx)。反向传播递归地从输出层向输入层传播梯度。


### 4.2 计算过程


1. **前向传播**
   ：计算每层的 z⁽ˡ⁾ 和 a⁽ˡ⁾，直到输出层
2. **计算损失**
   ：L = loss(a⁽ᴸ⁾, y)
3. **反向传播**
   ：从输出层开始，逐层计算梯度


### 4.3 梯度计算


输出层误差：


$$
δ⁽ᴸ⁾ = ∂L/∂a⁽ᴸ⁾ ⊙ σ'(z⁽ᴸ⁾)
$$


隐藏层误差（递推）：


$$
δ⁽ˡ⁾ = (W⁽ˡ⁺¹⁾)ᵀδ⁽ˡ⁺¹⁾ ⊙ σ'(z⁽ˡ⁾)
$$


参数梯度：


$$
∂L/∂W⁽ˡ⁾ = δ⁽ˡ⁾(a⁽ˡ⁻¹⁾)ᵀ
                ∂L/∂b⁽ˡ⁾ = δ⁽ˡ⁾
$$


### 4.4 计算图


反向传播本质是计算图上的自动微分。现代框架（PyTorch、TensorFlow）通过计算图自动实现反向传播，无需手动推导梯度。


## 5. 梯度消失与爆炸


### 5.1 问题描述


在深层网络中，梯度经过多层传播后：


- **梯度消失**
   ：梯度指数级缩小→靠近输入层的参数几乎不更新→浅层学不到特征
- **梯度爆炸**
   ：梯度指数级放大→参数更新过大→训练不稳定


数学原因：梯度 = 各层雅可比矩阵的连乘。若每层的梯度范数 < 1，连乘后趋于 0；若 > 1，连乘后趋于无穷。


### 5.2 解决方案


| 方案 | 解决什么 | 原理 |
| --- | --- | --- |
| ReLU 激活 | 梯度消失 | 正区间梯度恒为 1 |
| Batch Normalization | 两者 | 规范化中间层输出，稳定梯度分布 |
| 残差连接 (ResNet) | 梯度消失 | 提供梯度的"直通路径" |
| 梯度裁剪 | 梯度爆炸 | 限制梯度最大范数 |
| LSTM/GRU | 梯度消失 (RNN) | 门控机制控制信息流 |
| 合理初始化 | 两者 | Xavier/He 初始化保持方差稳定 |


### 5.3 权重初始化


- **Xavier/Glorot**
   ：适用于 Sigmoid/Tanh，Var(w) = 2/(n_in + n_out)
- **He 初始化**
   ：适用于 ReLU，Var(w) = 2/n_in
- 目标：保持各层输出的方差稳定，避免信号逐层衰减或放大


## 6. 万能逼近定理


万能逼近定理（Universal Approximation Theorem）：一个包含足够多隐藏神经元的单隐层前馈网络，能以任意精度逼近任何连续函数。


### 6.1 定理表述


对于任意连续函数 f: [0,1]ⁿ → ℝ 和任意 ε > 0，存在一个单隐层 MLP（使用非线性激活函数），使得对所有 x ∈ [0,1]ⁿ，|F(x) - f(x)| < ε。


### 6.2 重要含义


- 神经网络具有
   **理论上的**
   强大表达能力
- 但定理不保证能找到这样的参数（训练可能困难）
- 深层网络比浅层网络在实践中更高效（参数更少，泛化更好）
- 这是神经网络能工作的理论基础


> **Note:** **理论 vs 实践：**
> 万能逼近定理是存在性定理，实际中深层窄网络通常比浅层宽网络更高效。深度提供了"层次化特征学习"的能力。


## 7. 训练技巧


### 7.1 优化器


| 优化器 | 特点 |
| --- | --- |
| SGD | 基础随机梯度下降，可能震荡 |
| SGD + Momentum | 加入动量，加速收敛，减少震荡 |
| AdaGrad | 自适应学习率，适合稀疏梯度 |
| RMSProp | 解决 AdaGrad 学习率过快衰减 |
| Adam | 结合 Momentum 和 RMSProp，最常用 |
| AdamW | 修正的权重衰减，推荐使用 |


### 7.2 Batch Normalization


对每个 mini-batch 的输出进行标准化：


$$
μ_B = (1/m) Σxᵢ
                σ²_B = (1/m) Σ(xᵢ - μ_B)²
                x̂ᵢ = (xᵢ - μ_B) / √(σ²_B + ε)
                yᵢ = γ·x̂ᵢ + β  （可学习的缩放和平移）
$$


- 加速训练，允许更大学习率
- 有轻微正则化效果
- 减小对初始化的敏感度


### 7.3 Dropout


训练时以概率 p 随机将神经元输出置零，测试时使用全部神经元并乘以 (1-p)。


- 防止过拟合，等价于训练多个子网络的集成
- 常用 p = 0.2 ~ 0.5


## 8. Python 代码实现


### 8.1 使用 PyTorch 构建 MLP


```
import torch
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.BatchNorm1d(hidden_dim),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(hidden_dim, hidden_dim),
            nn.BatchNorm1d(hidden_dim),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(hidden_dim, output_dim)
        )

    def forward(self, x):
        return self.net(x)

# 训练循环
model = MLP(784, 256, 10)
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
```

## 9. 总结


1. **感知机**
   是最基本的神经元，只能解决线性可分问题
2. **MLP**
   通过多层堆叠和非线性激活实现复杂函数逼近
3. **激活函数**
   引入非线性：ReLU 是隐藏层默认选择，Sigmoid/Softmax 用于输出层
4. **反向传播**
   利用链式法则高效计算梯度，是训练的核心
5. **梯度消失/爆炸**
   通过 ReLU、BN、残差连接、合理初始化来解决
6. **万能逼近定理**
   保证了神经网络的理论表达能力
