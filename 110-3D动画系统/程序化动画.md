# 程序化动画

## 核心概念

程序化动画（Procedural Animation）通过物理模拟或数学计算在运行时生成动画，而非播放预录制的动画片段。它使角色能对环境变化做出实时响应，如布娃娃物理、弹簧晃动、程序化步态等。程序化动画通常与预录制动画叠加使用，在保留艺术控制的同时增加自然感。

## 数学基础

### 阻尼弹簧方程

弹簧-阻尼系统是程序化动画的核心数学工具：

$$m \ddot{x} = -k(x - x_{target}) - c \dot{x}$$

其中：
- $m$ 是质量（通常设为 1）
- $k$ 是弹簧刚度（stiffness），值越大回弹越快
- $c$ 是阻尼系数（damping），值越大衰减越快
- $x_{target}$ 是目标位置

在离散时间步中，使用半隐式欧拉积分：

$$v_{n+1} = v_n + (-k(x_n - x_{target}) - c \cdot v_n) \cdot \Delta t$$

$$x_{n+1} = x_n + v_{n+1} \cdot \Delta t$$

### 临界阻尼分析

系统的阻尼比 $\zeta = \frac{c}{2\sqrt{mk}}$ 决定了行为：
- $\zeta < 1$：欠阻尼，系统会振荡
- $\zeta = 1$：临界阻尼，最快回到目标位置无振荡
- $\zeta > 1$：过阻尼，缓慢回到目标位置

游戏中的典型设置：$\zeta \approx 0.5-0.8$（轻微欠阻尼，有弹性感）

### 弹簧参数与帧率无关

为确保不同帧率下表现一致，使用帧率无关的参数：

```cpp
// 将频率和阻尼比转换为 k 和 c
// frequency: 弹簧的自然频率（Hz）
// dampingRatio: 阻尼比
float k = (2 * PI * frequency) * (2 * PI * frequency);
float c = 2 * dampingRatio * sqrt(k);
```

## 完整 C++ 程序化动画系统

### 弹簧向量系统

```cpp
#include <glm/glm.hpp>
#include <glm/gtc/quaternion.hpp>
#include <glm/gtx/quaternion.hpp>
#include <cmath>
#include <vector>

using Vec3 = glm::vec3;
using Quat = glm::quat;

// ===== 弹簧向量 =====
class SpringVector {
public:
    Vec3 position;
    Vec3 velocity;
    Vec3 target;

    float frequency = 3.0f;       // 自然频率 (Hz)
    float dampingRatio = 0.5f;    // 阻尼比

    SpringVector() : position(0), velocity(0), target(0) {}
    SpringVector(Vec3 pos, float freq = 3.0f, float damp = 0.5f)
        : position(pos), velocity(0), target(pos), frequency(freq), dampingRatio(damp) {}

    void Update(float dt) {
        // 帧率无关的弹簧计算
        float k = (2.0f * 3.14159265f * frequency);
        k = k * k;  // k = omega^2
        float c = 2.0f * dampingRatio * std::sqrt(k);

        Vec3 displacement = position - target;
        Vec3 springForce = -k * displacement;
        Vec3 dampingForce = -c * velocity;
        Vec3 acceleration = springForce + dampingForce;

        // 半隐式欧拉积分
        velocity += acceleration * dt;
        position += velocity * dt;
    }

    void SetTarget(Vec3 newTarget) {
        target = newTarget;
    }

    void SetTargetInstant(Vec3 newTarget) {
        target = newTarget;
        position = newTarget;
        velocity = Vec3(0);
    }

    bool IsSettled(float threshold = 0.001f) const {
        return glm::length(velocity) < threshold &&
               glm::length(position - target) < threshold;
    }
};

// ===== 弹簧四元数 =====
class SpringQuaternion {
public:
    Quat rotation;
    Vec3 angularVelocity;
    Quat target;

    float frequency = 3.0f;
    float dampingRatio = 0.5f;

    SpringQuaternion() : rotation(1, 0, 0, 0), angularVelocity(0), target(1, 0, 0, 0) {}

    void Update(float dt) {
        // 计算旋转差（用对数映射将四元数差映射到角速度空间）
        Quat delta = target * glm::inverse(rotation);

        // 确保最短路径
        if (delta.w < 0) {
            delta = Quat(-delta.w, -delta.x, -delta.y, -delta.z);
        }

        // 将四元数差转换为角位移（轴角表示的角部分）
        float halfAngle = std::acos(glm::clamp(delta.w, -1.0f, 1.0f));
        Vec3 angularDisplacement;
        if (halfAngle > 0.001f) {
            float s = std::sin(halfAngle);
            angularDisplacement = Vec3(delta.x / s, delta.y / s, delta.z / s) * (2.0f * halfAngle);
        } else {
            angularDisplacement = Vec3(delta.x, delta.y, delta.z) * 2.0f;
        }

        // 弹簧力学
        float k = (2.0f * 3.14159265f * frequency);
        k = k * k;
        float c = 2.0f * dampingRatio * std::sqrt(k);

        Vec3 springTorque = -k * angularDisplacement;
        Vec3 dampingTorque = -c * angularVelocity;
        Vec3 angularAccel = springTorque + dampingTorque;

        angularVelocity += angularAccel * dt;
        Vec3 frameRotation = angularVelocity * dt;

        // 将角速度转换为四元数增量
        float angle = glm::length(frameRotation);
        if (angle > 0.0001f) {
            Vec3 axis = frameRotation / angle;
            Quat deltaRot = glm::angleAxis(angle, axis);
            rotation = glm::normalize(deltaRot * rotation);
        }
    }
};
```

### 程序化尾巴/耳朵系统

```cpp
// ===== 程序化骨骼链跟随系统 =====
class ProceduralChain {
public:
    struct ChainSegment {
        Vec3 position;
        Vec3 prevPosition;
        Vec3 velocity;
        float length;          // 到下一个段的长度
    };

    std::vector<ChainSegment> segments;
    float stiffness = 8.0f;
    float damping = 0.6f;
    float gravity = -9.8f;
    Vec3 externalForce = Vec3(0);

    void Initialize(const std::vector<Vec3>& initialPositions) {
        segments.clear();
        for (size_t i = 0; i < initialPositions.size(); i++) {
            ChainSegment seg;
            seg.position = initialPositions[i];
            seg.prevPosition = initialPositions[i];
            seg.velocity = Vec3(0);
            seg.length = (i < initialPositions.size() - 1) ?
                glm::length(initialPositions[i + 1] - initialPositions[i]) : 0;
            segments.push_back(seg);
        }
    }

    void Update(Vec3 rootPosition, float dt) {
        if (segments.empty()) return;

        // 1. 将根段移动到根位置
        segments[0].position = rootPosition;

        // 2. 逐段更新
        for (size_t i = 1; i < segments.size(); i++) {
            ChainSegment& seg = segments[i];
            ChainSegment& parent = segments[i - 1];

            // Verlet 积分
            Vec3 newPos = seg.position + (seg.position - seg.prevPosition) * (1.0f - damping)
                         + (externalForce + Vec3(0, gravity, 0)) * (dt * dt);

            // 弹簧力：拉向父段
            Vec3 toParent = parent.position - newPos;
            float dist = glm::length(toParent);
            if (dist > 0.001f) {
                Vec3 springDir = toParent / dist;
                float stretch = dist - seg.length;
                newPos += springDir * stretch * stiffness * dt * dt;
            }

            seg.prevPosition = seg.position;
            seg.position = newPos;
        }

        // 3. 距离约束迭代（保持骨骼长度）
        for (int iter = 0; iter < 3; iter++) {
            for (size_t i = 1; i < segments.size(); i++) {
                Vec3 toParent = segments[i - 1].position - segments[i].position;
                float dist = glm::length(toParent);
                if (dist > 0.001f && dist != segments[i].length) {
                    Vec3 correction = toParent * (1.0f - segments[i].length / dist) * 0.5f;
                    segments[i].position += correction;
                    if (i > 1) segments[i - 1].position -= correction;
                }
            }
            // 重新固定根段
            segments[0].position = rootPosition;
        }
    }

    // 获取每段的旋转（用于驱动骨骼）
    std::vector<Quat> GetRotations() const {
        std::vector<Quat> rotations;
        Vec3 defaultDir(0, -1, 0);  // 默认朝下

        for (size_t i = 0; i < segments.size() - 1; i++) {
            Vec3 dir = glm::normalize(segments[i + 1].position - segments[i].position);
            Quat rot = RotationBetweenVectors(defaultDir, dir);
            rotations.push_back(rot);
        }
        return rotations;
    }

private:
    static Quat RotationBetweenVectors(Vec3 v1, Vec3 v2) {
        v1 = glm::normalize(v1);
        v2 = glm::normalize(v2);
        float dot = glm::clamp(glm::dot(v1, v2), -1.0f, 1.0f);
        if (dot > 0.9999f) return Quat(1, 0, 0, 0);
        Vec3 axis = glm::normalize(glm::cross(v1, v2));
        return glm::angleAxis(std::acos(dot), axis);
    }
};
```

### 布娃娃物理系统

```cpp
// ===== 布娃娃系统 =====
class RagdollSystem {
public:
    struct RagdollBone {
        int jointId;
        std::string boneName;

        // 物理属性
        Vec3 position;
        Quat rotation;
        Vec3 velocity;
        Vec3 angularVelocity;
        float mass;
        float radius;         // 碰撞体半径

        // 约束
        int parentBoneIndex;  // -1 表示无父级
        Vec3 constraintAxis;  // 关节约束轴
        float minAngle;       // 最小弯曲角度
        float maxAngle;       // 最大弯曲角度
    };

    std::vector<RagdollBone> bones;
    bool active = false;
    float gravity = -9.8f;
    float linearDamping = 0.05f;
    float angularDamping = 0.1f;

    // 从当前动画姿势初始化布娃娃
    void Enable(const std::vector<Vec3>& bonePositions,
                const std::vector<Quat>& boneRotations) {
        for (size_t i = 0; i < bones.size(); i++) {
            bones[i].position = bonePositions[i];
            bones[i].rotation = boneRotations[i];
            bones[i].velocity = Vec3(0);
            bones[i].angularVelocity = Vec3(0);
        }
        active = true;
    }

    void Disable() {
        active = false;
    }

    void Update(float dt) {
        if (!active) return;

        // 1. 施加重力和阻尼
        for (auto& bone : bones) {
            bone.velocity += Vec3(0, gravity, 0) * dt;
            bone.velocity *= (1.0f - linearDamping);
            bone.angularVelocity *= (1.0f - angularDamping);
        }

        // 2. 更新位置和旋转
        for (auto& bone : bones) {
            bone.position += bone.velocity * dt;

            float angle = glm::length(bone.angularVelocity) * dt;
            if (angle > 0.0001f) {
                Vec3 axis = bone.angularVelocity / glm::length(bone.angularVelocity);
                Quat deltaRot = glm::angleAxis(angle, axis);
                bone.rotation = glm::normalize(deltaRot * bone.rotation);
            }
        }

        // 3. 约束求解
        for (int iter = 0; iter < 4; iter++) {
            for (auto& bone : bones) {
                if (bone.parentBoneIndex < 0) continue;
                auto& parent = bones[bone.parentBoneIndex];

                // 铰链约束：限制父子骨骼之间的弯曲角度
                Vec3 parentDir = glm::rotate(parent.rotation, Vec3(0, -1, 0));
                Vec3 childDir = glm::rotate(bone.rotation, Vec3(0, -1, 0));
                float angle = std::acos(glm::clamp(glm::dot(parentDir, childDir), -1.0f, 1.0f));

                if (angle < bone.minAngle || angle > bone.maxAngle) {
                    float targetAngle = glm::clamp(angle, bone.minAngle, bone.maxAngle);
                    Vec3 correctionAxis = glm::normalize(glm::cross(parentDir, childDir));
                    Quat correction = glm::angleAxis(targetAngle - angle, correctionAxis);
                    bone.rotation = glm::normalize(correction * bone.rotation);
                }
            }
        }

        // 4. 简单碰撞检测
        for (size_t i = 0; i < bones.size(); i++) {
            if (bones[i].position.y < bones[i].radius) {
                bones[i].position.y = bones[i].radius;
                bones[i].velocity.y = -bones[i].velocity.y * 0.3f;  // 反弹+摩擦
            }
        }
    }

    // 获取骨骼姿势用于渲染
    void GetPose(std::vector<Vec3>& positions, std::vector<Quat>& rotations) const {
        positions.resize(bones.size());
        rotations.resize(bones.size());
        for (size_t i = 0; i < bones.size(); i++) {
            positions[i] = bones[i].position;
            rotations[i] = bones[i].rotation;
        }
    }
};
```

### LookAt 程序化追踪

```cpp
// ===== 程序化头部追踪 =====
class ProceduralLookAt {
public:
    SpringVector eyeTarget;       // 眼睛追踪的弹簧
    SpringVector headTarget;      // 头部追踪的弹簧
    Vec3 currentLookDir;
    Vec3 desiredLookDir;

    float maxEyeAngle = 30.0f;    // 眼睛最大偏转角度
    float maxHeadAngle = 60.0f;   // 头部最大偏转角度
    float maxSpineAngle = 20.0f;  // 脊柱最大偏转角度

    float eyeWeight = 1.0f;       // 眼睛追踪权重
    float headWeight = 0.8f;      // 头部追踪权重
    float spineWeight = 0.3f;     // 脊柱追踪权重

    void Initialize() {
        eyeTarget = SpringVector(Vec3(0), 8.0f, 0.6f);    // 眼睛快速响应
        headTarget = SpringVector(Vec3(0), 4.0f, 0.7f);   // 头部中等响应
    }

    void Update(Character* character, const Vec3& worldTarget, float dt) {
        // 计算眼睛的目标方向
        Vec3 eyePos = character->GetBoneWorldPosition("LeftEye");
        Vec3 eyeDir = glm::normalize(worldTarget - eyePos);
        eyeTarget.SetTarget(eyeDir);
        eyeTarget.Update(dt);

        // 限制眼睛偏转角度
        Vec3 baseForward = character->GetForward();
        Vec3 clampedEyeDir = ClampAngle(eyeTarget.position, baseForward, maxEyeAngle);

        // 应用到眼睛骨骼
        ApplyEyeRotation(character, "LeftEye", clampedEyeDir, eyeWeight);
        ApplyEyeRotation(character, "RightEye", clampedEyeDir, eyeWeight);

        // 头部追踪（更慢，更平滑）
        Vec3 headPos = character->GetBoneWorldPosition("Head");
        Vec3 headDir = glm::normalize(worldTarget - headPos);
        headTarget.SetTarget(headDir);
        headTarget.Update(dt);

        Vec3 clampedHeadDir = ClampAngle(headTarget.position, baseForward, maxHeadAngle);
        ApplyHeadRotation(character, clampedHeadDir, headWeight);

        // 脊柱微调
        ApplySpineRotation(character, clampedHeadDir, spineWeight);
    }

private:
    Vec3 ClampAngle(const Vec3& dir, const Vec3& forward, float maxAngleDeg) {
        float angle = std::acos(glm::clamp(glm::dot(glm::normalize(dir), forward), -1.0f, 1.0f));
        float maxAngle = glm::radians(maxAngleDeg);
        if (angle > maxAngle) {
            return glm::normalize(glm::mix(forward, dir, maxAngle / angle));
        }
        return glm::normalize(dir);
    }

    void ApplyEyeRotation(Character* character, const std::string& boneName,
                          const Vec3& dir, float weight) {
        Quat currentRot = character->GetBoneRotation(boneName);
        Quat lookRot = RotationBetweenVectors(character->GetForward(), dir);
        Quat blendedRot = glm::slerp(currentRot, lookRot * currentRot, weight);
        character->SetBoneRotation(boneName, blendedRot);
    }

    void ApplyHeadRotation(Character* character, const Vec3& dir, float weight) {
        Quat currentRot = character->GetBoneRotation("Head");
        Quat lookRot = RotationBetweenVectors(character->GetForward(), dir);
        Quat blendedRot = glm::slerp(currentRot, lookRot * currentRot, weight);
        character->SetBoneRotation("Head", blendedRot);
    }

    void ApplySpineRotation(Character* character, const Vec3& dir, float weight) {
        // 分散到3节脊柱上
        float perSpine = weight / 3.0f;
        Quat lookRot = RotationBetweenVectors(character->GetForward(), dir);

        for (const std::string& bone : {"Spine", "Spine1", "Spine2"}) {
            Quat currentRot = character->GetBoneRotation(bone);
            Quat blendedRot = glm::slerp(currentRot, lookRot * currentRot, perSpine);
            character->SetBoneRotation(bone, blendedRot);
        }
    }

    static Quat RotationBetweenVectors(Vec3 v1, Vec3 v2) {
        v1 = glm::normalize(v1);
        v2 = glm::normalize(v2);
        float dot = glm::clamp(glm::dot(v1, v2), -1.0f, 1.0f);
        if (dot > 0.9999f) return Quat(1, 0, 0, 0);
        Vec3 axis = glm::normalize(glm::cross(v1, v2));
        return glm::angleAxis(std::acos(dot), axis);
    }
};
```

## 程序化步态（四足/爬行）

```cpp
// ===== 程序化步态系统 =====
class ProceduralGait {
public:
    struct Leg {
        std::string hipBone, kneeBone, footBone;
        Vec3 footPosition;
        Vec3 footTarget;
        Vec3 lastStablePos;
        bool isStepping = false;
        float stepProgress = 0;
    };

    std::vector<Leg> legs;
    float stepHeight = 0.2f;       // 抬脚高度
    float stepDuration = 0.3f;     // 单步时长
    float footSpacing = 0.3f;      // 脚间距
    float bodyHeight = 0.8f;       // 身体高度

    // 四足步态顺序：左前+右后 -> 右前+左后
    int phase = 0;
    float phaseTimer = 0;

    void Update(Character* character, Vec3 moveDir, float speed, float dt) {
        // 根据速度调整步态频率
        float stepFreq = speed * 2.0f;  // 速度越快步频越高

        phaseTimer += dt * stepFreq;
        if (phaseTimer >= 1.0f) {
            phaseTimer -= 1.0f;
            phase = (phase + 1) % 2;
        }

        // 更新每条腿
        for (int i = 0; i < (int)legs.size(); i++) {
            bool shouldStep = (phase == 0 && (i == 0 || i == 3)) ||
                              (phase == 1 && (i == 1 || i == 2));

            if (shouldStep && !legs[i].isStepping) {
                // 开始迈步
                legs[i].isStepping = true;
                legs[i].stepProgress = 0;
                // 计算目标位置（前方一定距离）
                legs[i].footTarget = character->GetPosition() +
                    moveDir * footSpacing +
                    character->GetLegOffset(i);
            }

            if (legs[i].isStepping) {
                legs[i].stepProgress += dt / stepDuration;
                if (legs[i].stepProgress >= 1.0f) {
                    legs[i].isStepping = false;
                    legs[i].lastStablePos = legs[i].footTarget;
                    legs[i].stepProgress = 1.0f;
                }

                // 抬脚曲线（抛物线）
                float t = legs[i].stepProgress;
                float height = stepHeight * 4.0f * t * (1.0f - t);

                // 插值脚部位置
                legs[i].footPosition = glm::mix(legs[i].lastStablePos, legs[i].footTarget, t);
                legs[i].footPosition.y += height;
            }

            // 通过 IK 将脚放到目标位置
            ApplyFootIK(character, legs[i]);
        }

        // 根据脚步调整身体高度和倾斜
        UpdateBodyBalance(character);
    }

private:
    void ApplyFootIK(Character* character, Leg& leg) {
        // 使用 Two-Bone IK 将脚放到 footPosition
        // （参见 IK 章节）
    }

    void UpdateBodyBalance(Character* character) {
        // 计算所有着地脚的重心
        Vec3 centroid(0);
        int groundedCount = 0;
        for (const auto& leg : legs) {
            if (!leg.isStepping) {
                centroid += leg.footPosition;
                groundedCount++;
            }
        }
        if (groundedCount > 0) {
            centroid /= (float)groundedCount;
            // 身体在重心上方 bodyHeight 处
            Vec3 bodyTarget = centroid + Vec3(0, bodyHeight, 0);
            character->AdjustBodyPosition(bodyTarget);
        }
    }
};
```

## 程序化动画到动画过渡（Ragdoll-to-Animation）

```cpp
// ===== 布娃娃到动画的平滑过渡 =====
class RagdollToAnimationBlend {
public:
    float blendDuration = 0.5f;
    float blendProgress = 0;
    bool blending = false;

    std::vector<Vec3> ragdollPositions;
    std::vector<Quat> ragdollRotations;

    void StartBlend(const std::vector<Vec3>& ragdollPos,
                    const std::vector<Quat>& ragdollRot) {
        ragdollPositions = ragdollPos;
        ragdollRotations = ragdollRot;
        blendProgress = 0;
        blending = true;
    }

    SkeletonPose Update(float dt, const SkeletonPose& targetAnimPose) {
        if (!blending) return targetAnimPose;

        blendProgress += dt / blendDuration;
        if (blendProgress >= 1.0f) {
            blending = false;
            return targetAnimPose;
        }

        // 平滑插值：从布娃娃姿势过渡到动画姿势
        float t = SmoothStep(blendProgress);  // 使用 SmoothStep 而非线性插值

        SkeletonPose blendedPose;
        blendedPose.joints.resize(targetAnimPose.joints.size());

        for (size_t i = 0; i < targetAnimPose.joints.size(); i++) {
            blendedPose.joints[i].position = glm::mix(
                ragdollPositions[i], targetAnimPose.joints[i].position, t);
            blendedPose.joints[i].rotation = glm::slerp(
                ragdollRotations[i], targetAnimPose.joints[i].rotation, t);
            blendedPose.joints[i].scale = targetAnimPose.joints[i].scale;
        }

        return blendedPose;
    }

private:
    float SmoothStep(float t) {
        return t * t * (3.0f - 2.0f * t);
    }
};
```

## 常见陷阱与解决方案

1. **弹簧参数导致抖动**：刚度过高或阻尼过低会导致振荡。使用临界阻尼比 ($\zeta \approx 0.7$) 作为起点调整。

2. **帧率不一致**：弹簧在不同帧率下表现不同。使用帧率无关的参数（频率+阻尼比），或使用固定时间步长。

3. **程序化层穿模**：弹簧或布娃娃可能使肢体穿过地面或墙壁。添加碰撞检测和约束求解。

4. **布娃娃恢复动画不自然**：直接插值到动画姿势会导致肢体扭曲。使用 `SmoothStep` 过渡，并在过渡期间禁用物理碰撞。
