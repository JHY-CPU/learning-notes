# 110-3D动画系统

3D动画系统是游戏引擎中驱动角色和物体运动的核心子系统。本模块覆盖从骨骼绑定到高级程序化动画的完整技术栈。

## 目录

1. [骨骼绑定基础](./骨骼绑定基础.md) - 骨骼层级与关节变换
2. [蒙皮(Skinning)](./蒙皮(Skinning).md) - 网格与骨骼的绑定技术
3. [动画状态机](./动画状态机.md) - 动画片段的状态管理
4. [混合树(Blend Tree)](./混合树(Blend%20Tree).md) - 多动画融合技术
5. [动画事件与脚本集成](./动画事件与脚本集成.md) - 动画与游戏逻辑的桥梁
6. [IK逆向运动学](./IK逆向运动学.md) - 末端执行器驱动的骨骼求解
7. [动画层与遮罩](./动画层与遮罩.md) - 分层动画叠加系统
8. [程序化动画](./程序化动画.md) - 物理驱动的动态动画
9. [动画性能优化](./动画性能优化.md) - 大规模角色动画性能策略
10. [Motion Matching](./MotionMatching.md) - 数据驱动的现代动画技术

## 学习路径

```
基础层: 骨骼绑定 -> 蒙皮 -> 动画状态机
进阶层: 混合树 -> 动画事件 -> IK逆向运动学
高级层: 动画层 -> 程序化动画 -> Motion Matching
```

## 推荐工具

- **绑定**: Maya HumanIK、Blender Armature
- **动画**: Maya、Blender、3ds Max
- **引擎**: Unity Animator、Unreal Animation Blueprint
