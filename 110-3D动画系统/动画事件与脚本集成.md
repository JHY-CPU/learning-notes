# 动画事件与脚本集成

## 核心概念

动画事件（Animation Event）是动画与游戏逻辑之间的桥梁，允许在动画播放到特定时刻时触发预定义的回调函数。通过事件系统，美术制作的动画可以驱动音效播放、伤害判定、特效生成、脚步声等游戏逻辑，实现美术与程序的解耦。

## 系统设计

### 事件的分层架构

```
动画资源层 (Animation Clip)
    |
    +-- 时间轴上嵌入事件标记
    |     t=0.3s -> PlayFootstepSound({foot: "left"})
    |     t=0.5s -> SpawnDamageHitbox({weapon: "sword", damage: 25})
    |     t=0.7s -> EnableWeaponTrail({effect: "fire"})
    |     t=0.9s -> OnAttackEnd()
    |
    v
动画播放层 (Animator)
    |
    +-- 检测当前时间是否跨越事件时间点
    +-- 收集所有待触发事件
    |
    v
事件分发层 (Event Dispatcher)
    |
    +-- 按事件名路由到注册的回调
    |
    v
游戏逻辑层 (Game Systems)
    +-- Audio System: 播放脚步声
    +-- Combat System: 伤害判定
    +-- VFX System: 生成特效
    +-- AI System: 通知动画状态变化
```

### 动画曲线作为数据

除了离散事件，动画系统还支持用曲线（Animation Curve）在动画片段中嵌入连续数据：

```
Curve: "FootStepVolume" -> t=[0.0, 0.5, 1.0] -> v=[0.0, 1.0, 0.0]
Curve: "AttackWindow"   -> t=[0.0, 0.3, 0.6, 1.0] -> v=[0.0, 1.0, 1.0, 0.0]
Curve: "BlendUpperBody" -> t=[0.0, 0.2, 0.8, 1.0] -> v=[0.0, 1.0, 1.0, 0.0]
```

## 完整 C++ 事件系统实现

```cpp
#include <vector>
#include <string>
#include <functional>
#include <unordered_map>
#include <variant>
#include <cstdint>

// ===== 事件参数 =====
struct EventParam {
    std::variant<int, float, std::string> value;

    int GetInt() const { return std::get<int>(value); }
    float GetFloat() const { return std::get<float>(value); }
    const std::string& GetString() const { return std::get<std::string>(value); }
};

// ===== 动画事件定义 =====
struct AnimationEvent {
    float normalizedTime;     // 归一化时间 (0.0 - 1.0)
    std::string functionName; // 回调函数名
    int intParam = 0;
    float floatParam = 0.0f;
    std::string stringParam;
    uint32_t eventHash = 0;   // 函数名的哈希值（加速查找）

    AnimationEvent() = default;
    AnimationEvent(float time, const std::string& funcName)
        : normalizedTime(time), functionName(funcName) {
        eventHash = HashString(funcName);
    }

    static uint32_t HashString(const std::string& str) {
        uint32_t hash = 5381;
        for (char c : str) hash = ((hash << 5) + hash) + c;
        return hash;
    }
};

// ===== 动画曲线 =====
struct AnimationCurve {
    std::string name;

    struct Keyframe {
        float time;   // 归一化时间
        float value;
        float inTangent = 0;
        float outTangent = 0;
    };
    std::vector<Keyframe> keys;

    float Evaluate(float normalizedTime) const {
        if (keys.empty()) return 0.0f;
        if (keys.size() == 1) return keys[0].value;

        // 找到包围时间的两个关键帧
        for (int i = 0; i < (int)keys.size() - 1; i++) {
            if (normalizedTime >= keys[i].time && normalizedTime <= keys[i + 1].time) {
                float t = (normalizedTime - keys[i].time) / (keys[i + 1].time - keys[i].time);
                // Hermite 插值
                float t2 = t * t;
                float t3 = t2 * t;
                float h00 = 2 * t3 - 3 * t2 + 1;
                float h10 = t3 - 2 * t2 + t;
                float h01 = -2 * t3 + 3 * t2;
                float h11 = t3 - t2;

                return h00 * keys[i].value +
                       h10 * keys[i].outTangent * (keys[i + 1].time - keys[i].time) +
                       h01 * keys[i + 1].value +
                       h11 * keys[i + 1].inTangent * (keys[i + 1].time - keys[i].time);
            }
        }
        return keys.back().value;
    }
};

// ===== 动画片段（含事件和曲线） =====
class AnimationClip {
public:
    std::string name;
    float duration = 1.0f;
    float frameRate = 30.0f;
    bool loop = true;

    std::vector<AnimationEvent> events;
    std::vector<AnimationCurve> curves;

    // 获取指定时间范围内触发的事件
    void GetEventsInRange(float prevTime, float currentTime,
                          std::vector<const AnimationEvent*>& outEvents) const {
        float prevNorm = NormalizeTime(prevTime);
        float currNorm = NormalizeTime(currentTime);

        for (const auto& evt : events) {
            // 处理循环情况
            if (prevNorm <= currNorm) {
                // 正常情况：未跨越循环点
                if (evt.normalizedTime > prevNorm && evt.normalizedTime <= currNorm) {
                    outEvents.push_back(&evt);
                }
            } else {
                // 跨越循环点
                if (evt.normalizedTime > prevNorm || evt.normalizedTime <= currNorm) {
                    outEvents.push_back(&evt);
                }
            }
        }
    }

    float EvaluateCurve(const std::string& curveName, float time) const {
        for (const auto& curve : curves) {
            if (curve.name == curveName) {
                return curve.Evaluate(NormalizeTime(time));
            }
        }
        return 0.0f;
    }

private:
    float NormalizeTime(float time) const {
        if (duration <= 0) return 0;
        return loop ? fmod(time / duration, 1.0f) : std::clamp(time / duration, 0.0f, 1.0f);
    }
};

// ===== 事件回调系统 =====
using EventCallback = std::function<void(const AnimationEvent&)>;

class EventDispatcher {
    // 事件名 -> 回调列表
    std::unordered_map<uint32_t, std::vector<EventCallback>> handlers;
    // 通配符回调（接收所有事件）
    std::vector<EventCallback> wildcardHandlers;

public:
    void RegisterHandler(const std::string& eventName, EventCallback callback) {
        uint32_t hash = AnimationEvent::HashString(eventName);
        handlers[hash].push_back(std::move(callback));
    }

    void RegisterWildcard(EventCallback callback) {
        wildcardHandlers.push_back(std::move(callback));
    }

    void Dispatch(const AnimationEvent& evt) const {
        // 先执行通配符回调
        for (const auto& cb : wildcardHandlers) {
            cb(evt);
        }

        // 再执行特定事件回调
        auto it = handlers.find(evt.eventHash);
        if (it != handlers.end()) {
            for (const auto& cb : it->second) {
                cb(evt);
            }
        }
    }

    void Clear() {
        handlers.clear();
        wildcardHandlers.clear();
    }
};

// ===== 动画播放器 =====
class AnimationPlayer {
public:
    AnimationClip* currentClip = nullptr;
    float currentTime = 0.0f;
    float prevTime = 0.0f;
    float playbackSpeed = 1.0f;
    bool playing = false;

    EventDispatcher* eventDispatcher = nullptr;

    void Play(AnimationClip* clip, float speed = 1.0f) {
        currentClip = clip;
        currentTime = 0.0f;
        prevTime = 0.0f;
        playbackSpeed = speed;
        playing = true;
    }

    void Update(float dt) {
        if (!currentClip || !playing) return;

        prevTime = currentTime;
        currentTime += dt * playbackSpeed;

        // 循环处理
        if (currentClip->loop) {
            currentTime = fmod(currentTime, currentClip->duration);
            if (currentTime < 0) currentTime += currentClip->duration;
        } else {
            currentTime = std::clamp(currentTime, 0.0f, currentClip->duration);
            if (currentTime >= currentClip->duration) playing = false;
        }

        // 检查并触发事件
        if (eventDispatcher) {
            std::vector<const AnimationEvent*> triggeredEvents;
            currentClip->GetEventsInRange(prevTime, currentTime, triggeredEvents);
            for (const auto* evt : triggeredEvents) {
                eventDispatcher->Dispatch(*evt);
            }
        }
    }

    // 采样当前曲线值
    float GetCurveValue(const std::string& curveName) const {
        return currentClip ? currentClip->EvaluateCurve(curveName, currentTime) : 0.0f;
    }
};
```

## 游戏侧事件处理器

```cpp
// ===== 完整的角色动画事件处理器 =====
class CharacterAnimEventHandler {
public:
    Character* character;
    EventDispatcher dispatcher;

    void Initialize(Character* owner) {
        character = owner;

        // 注册所有事件处理回调
        dispatcher.RegisterHandler("PlayFootstep", [this](const AnimationEvent& evt) {
            OnFootstep(evt);
        });

        dispatcher.RegisterHandler("AttackHit", [this](const AnimationEvent& evt) {
            OnAttackHit(evt);
        });

        dispatcher.RegisterHandler("SpawnEffect", [this](const AnimationEvent& evt) {
            OnSpawnEffect(evt);
        });

        dispatcher.RegisterHandler("EnableTrail", [this](const AnimationEvent& evt) {
            OnEnableTrail(evt);
        });

        dispatcher.RegisterHandler("DisableTrail", [this](const AnimationEvent& evt) {
            OnDisableTrail(evt);
        });

        dispatcher.RegisterHandler("PlaySound", [this](const AnimationEvent& evt) {
            OnPlaySound(evt);
        });

        dispatcher.RegisterHandler("EnableRagdoll", [this](const AnimationEvent& evt) {
            OnEnableRagdoll(evt);
        });
    }

private:
    void OnFootstep(const AnimationEvent& evt) {
        // intParam: 0=左脚, 1=右脚
        Vec3 footPos = (evt.intParam == 0) ?
            character->GetLeftFootPos() : character->GetRightFootPos();

        // 射线检测地面类型
        RaycastHit hit;
        if (Physics::Raycast(footPos + Vec3(0, 0.5f, 0), Vec3(0, -1, 0), 1.0f, hit)) {
            SurfaceType surface = GetSurfaceType(hit.material);
            float volume = 0.5f + character->GetVelocity().length() * 0.1f;
            AudioManager::PlayFootstep(surface, volume, footPos);
        }
    }

    void OnAttackHit(const AnimationEvent& evt) {
        // stringParam: 攻击碰撞体名称
        // floatParam: 伤害倍率
        Vec3 weaponPos = character->GetWeaponPosition(evt.stringParam);
        Vec3 weaponDir = character->GetWeaponDirection(evt.stringParam);
        float weaponRange = character->GetWeaponRange(evt.stringParam);

        // 执行攻击判定
        auto hits = Physics::CapsuleCast(weaponPos, weaponDir, weaponRange,
                                          character->GetWeaponRadius());

        float baseDamage = character->GetBaseDamage();
        float finalDamage = baseDamage * evt.floatParam;

        for (auto& hit : hits) {
            if (hit.entity != character && hit.entity->HasComponent<HealthComponent>()) {
                hit.entity->GetComponent<HealthComponent>()->TakeDamage(
                    finalDamage, character, hit.point, hit.normal);
            }
        }
    }

    void OnSpawnEffect(const AnimationEvent& evt) {
        // stringParam: 特效资源路径
        // intParam: 挂载点 (0=右手, 1=左手, 2=武器, 3=头部)
        Transform attachPoint = character->GetAttachPoint(evt.intParam);
        EffectManager::Spawn(evt.stringParam, attachPoint);
    }

    void OnEnableTrail(const AnimationEvent& evt) {
        character->SetWeaponTrailActive(true);
    }

    void OnDisableTrail(const AnimationEvent& evt) {
        character->SetWeaponTrailActive(false);
    }

    void OnPlaySound(const AnimationEvent& evt) {
        // stringParam: 音频资源路径
        // floatParam: 音量
        AudioManager::Play3D(evt.stringParam, character->GetPosition(), evt.floatParam);
    }

    void OnEnableRagdoll(const AnimationEvent& evt) {
        character->EnableRagdoll();
    }

    SurfaceType GetSurfaceType(Material* mat) {
        // 根据材质名称或属性判断地面类型
        if (!mat) return SurfaceType::Concrete;
        std::string name = mat->GetName();
        if (name.find("Grass") != std::string::npos) return SurfaceType::Grass;
        if (name.find("Wood") != std::string::npos) return SurfaceType::Wood;
        if (name.find("Metal") != std::string::npos) return SurfaceType::Metal;
        if (name.find("Water") != std::string::npos) return SurfaceType::Water;
        return SurfaceType::Concrete;
    }
};
```

```csharp
using UnityEngine;

// ===== Unity Animation Event 使用 =====
public class CharacterAnimEvents : MonoBehaviour {
    Animator animator;
    AudioSource audioSource;
    ParticleSystem weaponTrail;

    void Start() {
        animator = GetComponent<Animator>();
        audioSource = GetComponent<AudioSource>();
    }

    // 这些方法名必须与 Animation 窗口中设置的事件函数名完全一致

    // 脚步声事件 - 在动画窗口中添加事件，调用此方法
    void PlayFootstep(int footIndex) {
        // footIndex: 0=左脚, 1=右脚
        Transform foot = (footIndex == 0) ?
            animator.GetBoneTransform(HumanBodyBones.LeftFoot) :
            animator.GetBoneTransform(HumanBodyBones.RightFoot);

        // 检测地面材质
        RaycastHit hit;
        if (Physics.Raycast(foot.position + Vector3.up * 0.3f, Vector3.down, out hit, 0.6f)) {
            // 根据地面材质播放不同音效
            string surface = hit.collider.tag;
            AudioClip clip = GetFootstepClip(surface);
            audioSource.PlayOneShot(clip, 0.5f);
        }
    }

    // 攻击判定事件
    void AttackHitCheck() {
        // 获取武器碰撞体并检测
        Transform weapon = animator.GetBoneTransform(HumanBodyBones.RightHand);
        Collider[] hits = Physics.OverlapSphere(weapon.position, 0.5f);

        foreach (var hit in hits) {
            if (hit.gameObject != gameObject) {
                var health = hit.GetComponent<HealthSystem>();
                if (health != null) {
                    health.TakeDamage(25, transform.forward);
                }
            }
        }
    }

    // 武器拖尾效果
    void EnableWeaponTrail() {
        if (weaponTrail) weaponTrail.Play();
    }

    void DisableWeaponTrail() {
        if (weaponTrail) weaponTrail.Stop();
    }

    // 动画结束事件
    void OnAttackAnimationEnd() {
        animator.SetBool("IsAttacking", false);
    }

    AudioClip GetFootstepClip(string surface) {
        // 返回对应材质的脚步声
        return Resources.Load<AudioClip>($"Footsteps/{surface}");
    }
}
```

```cpp
// UE5 C++ - 动画通知系统

// ===== 基础动画通知 =====
UCLASS()
class UAnimNotify_PlayFootstep : public UAnimNotify {
    GENERATED_BODY()

public:
    UPROPERTY(EditAnywhere, Category = "Footstep")
    EFoot Foot = EFoot::Left;

    virtual void Notify(USkeletalMeshComponent* MeshComp,
                        UAnimSequenceBase* Animation,
                        const FAnimNotifyEventReference& EventReference) override {
        if (!MeshComp) return;

        ACharacter* Character = Cast<ACharacter>(MeshComp->GetOwner());
        if (!Character) return;

        // 获取脚部位置
        FName BoneName = (Foot == EFoot::Left) ? FName("foot_l") : FName("foot_r");
        FVector FootPos = MeshComp->GetBoneLocation(BoneName);

        // 射线检测地面
        FHitResult Hit;
        FVector Start = FootPos + FVector(0, 0, 30);
        FVector End = FootPos - FVector(0, 0, 30);

        if (Character->GetWorld()->LineTraceSingleByChannel(Hit, Start, End, ECC_WorldStatic)) {
            // 根据物理材质播放脚步声
            UPhysicalMaterial* PhysMat = Hit.PhysMaterial.Get();
            if (PhysMat) {
                // 播放对应材质的脚步声
                UGameplayStatics::PlaySoundAtLocation(
                    MeshComp->GetWorld(),
                    GetFootstepSound(PhysMat->SurfaceType),
                    Hit.ImpactPoint);
            }
        }
    }

private:
    USoundBase* GetFootstepSound(EPhysicalSurface SurfaceType) {
        // 根据地面类型返回音效
        switch (SurfaceType) {
            case SurfaceType1: return GrassSound;
            case SurfaceType2: return MetalSound;
            case SurfaceType3: return WoodSound;
            default: return DefaultSound;
        }
    }

    UPROPERTY(EditAnywhere) USoundBase* GrassSound;
    UPROPERTY(EditAnywhere) USoundBase* MetalSound;
    UPROPERTY(EditAnywhere) USoundBase* WoodSound;
    UPROPERTY(EditAnywhere) USoundBase* DefaultSound;
};

        // 每帧执行攻击判定

        if (Character && Character->bInAttackWindow) {
            Character->PerformAttackHitCheck();
        }

                           UAnimSequenceBase* Animation,
                           const FAnimNotifyEventReference& EventReference) override {
        ABaseCharacter* Character = Cast<ABaseCharacter>(MeshComp->GetOwner());
        if (Character) {
            Character->bInAttackWindow = false;
            Character->ResetHitTargets();  // 清除已命中的目标，允许再次伤害
        }
    }
};
```

```cpp
// ===== 用动画曲线驱动碰撞检测窗口 =====
class CurveDrivenCombatSystem {
public:
    void Update(Character* character, AnimationPlayer* player) {
        // 曲线 "AttackDamageWindow" 的值为 1.0 时进行伤害判定
        float damageWindow = player->GetCurveValue("AttackDamageWindow");

        if (damageWindow > 0.5f && !attackActive) {
            attackActive = true;
            BeginAttack(character);
        } else if (damageWindow < 0.5f && attackActive) {
            attackActive = false;
            EndAttack(character);
        }

        // 曲线 "AttackDamageMultiplier" 控制伤害倍率
        if (attackActive) {
            float multiplier = player->GetCurveValue("AttackDamageMultiplier");
            character->SetDamageMultiplier(multiplier);
        }

        // 曲线 "UpperBodyBlendWeight" 控制上半身动画层权重
        float upperBlend = player->GetCurveValue("UpperBodyBlendWeight");
        character->SetLayerWeight(1, upperBlend);
    }

private:
    bool attackActive = false;

    void BeginAttack(Character* character) {
        character->GetCombatComponent()->EnableHitDetection();
    }

    void EndAttack(Character* character) {
        character->GetCombatComponent()->DisableHitDetection();
        character->GetCombatComponent()->ClearHitTargets();
    }
};
```

## 性能与最佳实践

### 事件处理性能

| 操作 | 开销 | 优化建议 |
|------|------|---------|
| 字符串事件查找 | ~100ns/次 | 使用哈希值代替字符串比较 |
| 回调函数调用 | ~50ns/次 | 避免在回调中执行重型逻辑 |
| 曲线评估 | ~200ns/次 | 缓存曲线查找结果 |
| 事件范围检测 | ~100ns/次 | 利用排序和二分查找 |

### 常见陷阱与解决方案

1. **循环播放时事件重复触发**：在循环动画中，同一事件每次循环都会触发。解决方案：使用标志位记录上一帧是否已触发，或使用 `ConsumeTrigger` 模式。

2. **动画被中途切换时残留事件**：状态切换后，新状态可能仍然触发上一个动画的事件。解决方案：在状态切换时重置事件检测状态。

3. **事件回调执行耗时操作**：在事件回调中执行网络请求或物理模拟会阻塞动画更新。解决方案：将耗时操作放入消息队列异步处理。

4. **跨平台导出时事件参数类型不一致**：不同引擎对事件参数的类型支持不同。解决方案：统一使用字符串参数，由游戏逻辑层解析。

5. **曲线精度不足**：关键帧太少导致曲线评估不平滑。解决方案：确保关键帧密度足够，使用 Hermite 插值保证平滑。

## 实际游戏案例

- **Unity**：在 Animation 窗口中添加事件，通过 MonoBehaviour 方法接收。支持 int/float/string/Object 参数。
- **Unreal Engine**：AnimNotify（瞬时通知）和 AnimNotifyState（持续通知）系统，支持 C++ 和蓝图实现。
- **黑暗之魂系列**：攻击动画事件精确驱动伤害判定窗口的开关，翻滚动画事件控制无敌帧。
- **怪物猎人**：武器动画的精确打击判定完全依赖动画事件和曲线，不同武器有不同的判定时序。
- **战神 (2018)**：斧头的投掷/召回动画使用事件同步物理状态切换和特效触发。
- **只狼**：弹刀判定窗口由动画曲线精确控制，完美弹刀的时机窗口仅约 12 帧（0.2 秒）。
