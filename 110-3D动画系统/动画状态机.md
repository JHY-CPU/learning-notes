# 动画状态机

## 核心概念

动画状态机（Animation State Machine, ASM）是管理角色动画片段之间切换逻辑的核心系统。每个状态对应一个或多个动画片段或混合树，状态之间的转换（Transition）由条件参数驱动（布尔值、触发器、浮点数）。状态机负责决定当前播放哪个动画、何时切换、如何过渡，是角色动画播放的核心调度器。

## 数学基础：状态转换的混合过渡

### 线性插值过渡

当从状态 A 过渡到状态 B 时，在过渡期间的每一帧，最终姿势由两个状态的采样结果按比例混合：

$$P_{final}(t) = (1 - \alpha(t)) \cdot P_A + \alpha(t) \cdot P_B$$

其中 $\alpha(t)$ 是过渡进度函数，$t \in [0, 1]$：

$$\alpha(t) = \text{clamp}\left(\frac{t - t_{start}}{t_{duration}}, 0, 1\right)$$

### 缓动函数

为避免过渡生硬，使用缓动函数平滑 $\alpha$ 的变化：

```
线性:       α = t
平滑步:     α = t²(3 - 2t)          // Hermite 插值
平滑开始:   α = t²
平滑结束:   α = 1 - (1-t)²
三次方:     α = 4t³                 (t<0.5时)
```

平滑步（Smoothstep）的数学定义：

$$smoothstep(t) = 3t^2 - 2t^3$$

### 骨骼级插值

每个关节的插值需要分别处理位置、旋转和缩放：

$$pos_{blend} = (1-\alpha) \cdot pos_A + \alpha \cdot pos_B$$

$$rot_{blend} = \text{Slerp}(rot_A, rot_B, \alpha)$$

$$scale_{blend} = (1-\alpha) \cdot scale_A + \alpha \cdot scale_B$$

Slerp（球面线性插值）公式：

$$\text{Slerp}(q_1, q_2, t) = \frac{\sin((1-t)\theta)}{\sin(\theta)} q_1 + \frac{\sin(t\theta)}{\sin(\theta)} q_2$$

其中 $\theta = \arccos(q_1 \cdot q_2)$。

```cpp
#include <vector>
#include <string>
#include <unordered_map>
#include <functional>
#include <variant>
#include <glm/glm.hpp>
#include <glm/gtc/quaternion.hpp>

using Vec3 = glm::vec3;
using Quat = glm::quat;

// ===== 参数系统 =====
enum class ParamType { Bool, Trigger, Float, Int };

struct ParamValue {
    std::variant<bool, float, int> value;

    bool GetBool() const { return std::get<bool>(value); }
    float GetFloat() const { return std::get<float>(value); }
    int GetInt() const { return std::get<int>(value); }

    void SetBool(bool v) { value = v; }
    void SetFloat(float v) { value = v; }
    void SetInt(int v) { value = v; }
};

class ParameterTable {
    std::unordered_map<std::string, ParamValue> params;

public:
    void SetBool(const std::string& name, bool val) {
        params[name].SetBool(val);
    }
    void SetFloat(const std::string& name, float val) {
        params[name].SetFloat(val);
    }
    void SetInt(const std::string& name, int val) {
        params[name].SetInt(val);
    }
    void SetTrigger(const std::string& name) {
        params[name].SetBool(true);  // 触发器是一次性的布尔值
    }

    bool GetBool(const std::string& name) const {
        auto it = params.find(name);
        return (it != params.end()) ? it->second.GetBool() : false;
    }
    float GetFloat(const std::string& name) const {
        auto it = params.find(name);
        return (it != params.end()) ? it->second.GetFloat() : 0.0f;
    }
    int GetInt(const std::string& name) const {
        auto it = params.find(name);
        return (it != params.end()) ? it->second.GetInt() : 0;
    }

    // 消耗触发器（一次性触发后自动关闭）
    bool ConsumeTrigger(const std::string& name) {
        auto it = params.find(name);
        if (it != params.end() && it->second.GetBool()) {
            it->second.SetBool(false);
            return true;
        }
        return false;
    }
};

// ===== 转换条件 =====
enum class CompareMode { Greater, Less, Equals, NotEquals, GreaterOrEqual, LessOrEqual };

struct TransitionCondition {
    std::string paramName;
    ParamType paramType;
    CompareMode compare;
    float threshold;      // 用于 Float 比较
    int intThreshold;     // 用于 Int 比较
    bool boolValue;       // 用于 Bool 比较
};

// ===== 骨骼姿势 =====
struct JointPose {
    Vec3 position;
    Quat rotation;
    Vec3 scale = Vec3(1.0f);
};

struct SkeletonPose {
    std::vector<JointPose> joints;

    void BlendWith(const SkeletonPose& other, float t) {
        for (size_t i = 0; i < joints.size(); i++) {
            joints[i].position = glm::mix(joints[i].position, other.joints[i].position, t);
            joints[i].rotation = glm::slerp(joints[i].rotation, other.joints[i].rotation, t);
            joints[i].scale = glm::mix(joints[i].scale, other.joints[i].scale, t);
        }
    }

    static SkeletonPose Lerp(const SkeletonPose& a, const SkeletonPose& b, float t) {
        SkeletonPose result;
        result.joints.resize(a.joints.size());
        for (size_t i = 0; i < a.joints.size(); i++) {
            result.joints[i].position = glm::mix(a.joints[i].position, b.joints[i].position, t);
            result.joints[i].rotation = glm::slerp(a.joints[i].rotation, b.joints[i].rotation, t);
            result.joints[i].scale = glm::mix(a.joints[i].scale, b.joints[i].scale, t);
        }
        return result;
    }
};

// ===== 动画片段 =====
class AnimationClip {
public:
    std::string name;
    float duration;           // 动画时长（秒）
    float frameRate = 30.0f;
    bool loop = true;

    struct Frame {
        SkeletonPose pose;
    };
    std::vector<Frame> frames;

    // 采样指定时间的姿势
    SkeletonPose Sample(float time) const {
        if (frames.empty()) return SkeletonPose();

        float t = loop ? fmod(time, duration) : glm::clamp(time, 0.0f, duration);
        float frameFloat = t * frameRate;
        int frameA = (int)frameFloat % frames.size();
        int frameB = (frameA + 1) % frames.size();
        float fraction = frameFloat - (int)frameFloat;

        return SkeletonPose::Lerp(frames[frameA].pose, frames[frameB].pose, fraction);
    }
};

// ===== 转换定义 =====
class AnimState;  // 前向声明

struct Transition {
    AnimState* fromState = nullptr;
    AnimState* toState = nullptr;
    std::vector<TransitionCondition> conditions;
    float exitTime = -1.0f;        // 归一化退出时间（0-1），-1表示不使用
    float duration = 0.25f;        // 混合过渡时长（秒）
    bool hasExitTime = false;      // 是否必须等当前动画播完指定位置
    int priority = 0;              // 优先级（高优先级的转换优先检查）

    // 缓动类型
    enum EaseType { LINEAR, SMOOTH_STEP, EASE_IN, EASE_OUT };
    EaseType easeType = SMOOTH_STEP;

    float EvaluateEase(float t) const {
        switch (easeType) {
            case LINEAR:     return t;
            case SMOOTH_STEP: return t * t * (3.0f - 2.0f * t);
            case EASE_IN:    return t * t;
            case EASE_OUT:   return 1.0f - (1.0f - t) * (1.0f - t);
            default:         return t;
        }
    }
};

// ===== 动画状态 =====
class AnimState {
public:
    std::string name;
    int id;
    AnimationClip* clip = nullptr;
    float speed = 1.0f;
    bool loop = true;

    std::vector<Transition*> outTransitions;  // 出向转换列表

    float currentTime = 0.0f;  // 当前播放时间

    void Update(float dt) {
        if (!clip) return;
        currentTime += dt * speed;
        if (loop && clip->duration > 0) {
            currentTime = fmod(currentTime, clip->duration);
        } else {
            currentTime = glm::clamp(currentTime, 0.0f, clip->duration);
        }
    }

    SkeletonPose GetCurrentPose() const {
        return clip ? clip->Sample(currentTime) : SkeletonPose();
    }

    float GetNormalizedTime() const {
        return (clip && clip->duration > 0) ? currentTime / clip->duration : 0.0f;
    }
};

// ===== 过渡状态（两个状态之间的混合过程） =====
class TransitionState {
public:
    Transition* transition;
    AnimState* fromState;
    AnimState* toState;
    float elapsed = 0.0f;
    bool active = false;

    void Start(Transition* trans) {
        transition = trans;
        fromState = trans->fromState;
        toState = trans->toState;
        elapsed = 0.0f;
        active = true;
        toState->currentTime = 0.0f;  // 新状态从头开始播放
    }

    void Update(float dt) {
        if (!active) return;
        elapsed += dt;
        fromState->Update(dt);
        toState->Update(dt);
    }

    bool IsComplete() const {
        return elapsed >= transition->duration;
    }

    float GetBlendAlpha() const {
        float raw = elapsed / transition->duration;
        return transition->EvaluateEase(glm::clamp(raw, 0.0f, 1.0f));
    }

    SkeletonPose GetBlendedPose() const {
        SkeletonPose poseA = fromState->GetCurrentPose();
        SkeletonPose poseB = toState->GetCurrentPose();
        return SkeletonPose::Lerp(poseA, poseB, GetBlendAlpha());
    }

    void Stop() { active = false; }
};

// ===== 动画状态机 =====
class AnimatorStateMachine {
public:
    std::vector<AnimState*> states;
    std::vector<Transition*> transitions;
    std::vector<Transition*> anyStateTransitions;

    AnimState* currentState = nullptr;
    TransitionState currentTransition;
    ParameterTable parameters;

    AnimState* AddState(const std::string& name, AnimationClip* clip) {
        auto* state = new AnimState();
        state->name = name;
        state->id = (int)states.size();
        state->clip = clip;
        states.push_back(state);
        if (!currentState) currentState = state;
        return state;
    }

    Transition* AddTransition(AnimState* from, AnimState* to) {
        auto* trans = new Transition();
        trans->fromState = from;
        trans->toState = to;
        transitions.push_back(trans);
        from->outTransitions.push_back(trans);
        return trans;
    }

    Transition* AddAnyStateTransition(AnimState* to) {
        auto* trans = new Transition();
        trans->fromState = nullptr;  // Any State
        trans->toState = to;
        anyStateTransitions.push_back(trans);
        return trans;
    }

    // 核心更新函数
    void Update(float dt) {
        // 1. 如果正在过渡中
        if (currentTransition.active) {
            currentTransition.Update(dt);
            if (currentTransition.IsComplete()) {
                // 过渡完成
                currentState = currentTransition.toState;
                currentTransition.Stop();
            }
            return;  // 过渡中不检查新转换
        }

        // 2. 更新当前状态
        currentState->Update(dt);

        // 3. 检查当前状态的出向转换（按优先级排序）
        // 先检查 Any State 转换（高优先级打断）
        for (auto* trans : anyStateTransitions) {
            if (trans->fromState == currentState) continue;  // 不从自身转换
            if (EvaluateTransition(trans)) {
                StartTransition(trans);
                return;
            }
        }

        // 再检查当前状态的出向转换
        for (auto* trans : currentState->outTransitions) {
            if (EvaluateTransition(trans)) {
                StartTransition(trans);
                return;
            }
        }
    }

    // 获取当前最终姿势
    SkeletonPose GetCurrentPose() const {
        if (currentTransition.active) {
            return currentTransition.GetBlendedPose();
        }
        return currentState->GetCurrentPose();
    }

private:
    bool EvaluateTransition(const Transition* trans) const {
        // 检查退出时间条件
        if (trans->hasExitTime) {
            float normTime = currentState->GetNormalizedTime();
            if (normTime < trans->exitTime) return false;
        }

        // 检查所有条件
        for (const auto& cond : trans->conditions) {
            if (!EvaluateCondition(cond)) return false;
        }
        return true;
    }

    bool EvaluateCondition(const TransitionCondition& cond) const {
        switch (cond.paramType) {
            case ParamType::Bool:
                return parameters.GetBool(cond.paramName) == cond.boolValue;

            case ParamType::Trigger:
                return parameters.ConsumeTrigger(cond.paramName);  // 消耗触发器

            case ParamType::Float: {
                float val = parameters.GetFloat(cond.paramName);
                switch (cond.compare) {
                    case CompareMode::Greater: return val > cond.threshold;
                    case CompareMode::Less: return val < cond.threshold;
                    case CompareMode::Equals: return std::abs(val - cond.threshold) < 0.001f;
                    case CompareMode::GreaterOrEqual: return val >= cond.threshold;
                    case CompareMode::LessOrEqual: return val <= cond.threshold;
                    default: return false;
                }
            }

            case ParamType::Int: {
                int val = parameters.GetInt(cond.paramName);
                switch (cond.compare) {
                    case CompareMode::Equals: return val == cond.intThreshold;
                    case CompareMode::NotEquals: return val != cond.intThreshold;
                    case CompareMode::Greater: return val > cond.intThreshold;
                    case CompareMode::Less: return val < cond.intThreshold;
                    default: return false;
                }
            }
        }
        return false;
    }

    void StartTransition(Transition* trans) {
        currentTransition.Start(trans);
    }
};
```

## 子状态机

子状态机（Sub-State Machine）将复杂的状态分组管理，降低主状态机的复杂度：

```cpp
class SubStateMachine : public AnimState {
public:
    AnimatorStateMachine innerMachine;

    // 子状态机作为一个整体参与外部转换
    void Update(float dt) override {
        innerMachine.Update(dt);
    }

    SkeletonPose GetCurrentPose() const override {
        return innerMachine.GetCurrentPose();
    }

    // 子状态机入口：指定初始状态
    void SetEntryState(AnimState* state) {
        innerMachine.currentState = state;
    }
};

// 使用示例
void SetupPlayerStateMachine(AnimatorStateMachine& asm) {
    // 创建子状态机
    auto* locomotion = new SubStateMachine();
    locomotion->name = "Locomotion";

    // 在子状态机中添加状态
    auto* idle = locomotion->innerMachine.AddState("Idle", idleClip);
    auto* walk = locomotion->innerMachine.AddState("Walk", walkClip);
    auto* run = locomotion->innerMachine.AddState("Run", runClip);

    // 子状态机内部转换
    auto* idleToWalk = locomotion->innerMachine.AddTransition(idle, walk);
    idleToWalk->conditions.push_back({"Speed", ParamType::Float, CompareMode::Greater, 0.1f});

    // 主状态机添加子状态机作为状态
    asm.states.push_back(locomotion);

    // 从子状态机到攻击状态的转换
    auto* toAttack = asm.AddTransition(locomotion, attackState);
    toAttack->conditions.push_back({"Attack", ParamType::Trigger, {}, 0});
}
```

using UnityEngine;

public class PlayerAnimator : MonoBehaviour {

    CharacterController controller;

    static readonly int SpeedHash = Animator.StringToHash("Speed");
    static readonly int IsGroundedHash = Animator.StringToHash("IsGrounded");
    static readonly int AttackHash = Animator.StringToHash("Attack");
    static readonly int HitHash = Animator.StringToHash("Hit");

    void Start() {
        animator = GetComponent<Animator>();
        controller = GetComponent<CharacterController>();
    }

    void Update() {
        Vector3 velocity = controller.velocity;
        float speed = new Vector3(velocity.x, 0, velocity.z).magnitude;

        // 设置状态机参数
        animator.SetFloat(SpeedHash, speed, 0.1f, Time.deltaTime);  // 平滑过渡
        animator.SetBool(IsGroundedHash, controller.isGrounded);

        // 攻击触发
        if (Input.GetButtonDown("Fire1")) {
            animator.SetTrigger(AttackHash);
        }

        // 蹲下
        animator.SetBool(IsCrouchingHash, Input.GetKey(KeyCode.C));
    }

    // 在动画被击中时调用
    public void TriggerHit() {
        animator.SetTrigger(HitHash);
    }

    // 通过脚本直接控制状态转换（不经过参数系统）
    public void PlayAnimationDirect(string stateName, int layer = 0, float transitionTime = 0.1f) {
        animator.CrossFade(stateName, transitionTime, layer);
    }

    // 获取当前动画状态信息
    public void LogCurrentState() {
        AnimatorStateInfo info = animator.GetCurrentAnimatorStateInfo(0);
        Debug.Log($"当前状态: {info.fullPathHash}, 进度: {info.normalizedTime:F2}, 是否在转换: {animator.IsInTransition(0)}");

        if (animator.IsInTransition(0)) {
            AnimatorTransitionInfo transInfo = animator.GetAnimatorTransitionInfo(0);
            Debug.Log($"过渡中, 持续时间: {transInfo.duration:F2}s");
        }
    }
}

## 状态机性能优化

### 状态转换缓存

```cpp
class OptimizedStateMachine {
    // 缓存条件评估结果，避免每帧重复计算
    struct TransitionCache {
        Transition* transition;
        bool lastConditionResult;
        int lastEvaluatedFrame;
    };
    std::vector<TransitionCache> transitionCache;

    // 批量评估条件
    void BatchEvaluateConditions(int frameNumber) {
        for (auto& cache : transitionCache) {
            if (cache.lastEvaluatedFrame != frameNumber) {
                cache.lastConditionResult = EvaluateAllConditions(cache.transition);
                cache.lastEvaluatedFrame = frameNumber;
            }
        }
    }
};
```

### 避免的陷阱

| 陷阱 | 现象 | 解决方案 |
|------|------|---------|
| 转换振荡 | 两个状态在边界参数附近反复切换 | 添加迟滞（Hysteresis），如进入条件 Speed>2，退出条件 Speed<1.5 |
| 触发器累积 | 同一帧多次设置触发器未被消耗 | 使用 ConsumeTrigger 模式，一次触发后自动清除 |
| 过渡打断过渡 | 过渡过程中触发新转换导致闪烁 | 过渡期间禁止新的状态转换（或设置最小过渡时间） |
| 忽略 hasExitTime | 动画被意外打断，动作不完整 | 关键动作（攻击、翻滚）设置 hasExitTime=true |
| 层级过深 | 子状态机嵌套超过3层导致调试困难 | 将深层逻辑重构为并行层或独立状态机 |

## 实际游戏案例

- **Unity Animator (Mecanim)**：最广泛使用的状态机系统，支持子状态机、混合树、Avatar Mask，可视化编辑器
- **Unreal AnimGraph**：Animation Blueprint 中的节点图系统，支持状态机、混合节点、IK 节点
- **黑暗之魂系列**：精确的状态转换控制，攻击/翻滚/走路之间的严格状态管理，hasExitTime 用于防止攻击被打断
- **塞尔达：旷野之息**：复杂的状态机管理林克的攀爬、游泳、滑翔、骑马等多套状态系统
- **艾尔登法环**：改进的动画状态机，支持更多无缝过渡，武器架势作为子状态机
- **战神 (2018)**：奎托斯的斧头投掷/召回涉及多个状态机的同步协调
