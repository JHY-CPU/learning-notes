# 动画性能优化

## 核心概念

当场景中存在大量动画角色（数百甚至数千）时，动画系统的性能成为瓶颈。优化策略包括降低远处角色的动画精度、减少骨骼采样频率、利用GPU并行计算等，核心思路是在视觉质量可接受的前提下最大化性能。

## 数学基础：LOD 距离模型

### 动画 LOD 分级

基于距离的 LOD 分级是最基础的优化手段：

$$LOD(d) = \begin{cases} 0 & d < d_0 \\ 1 & d_0 \le d < d_1 \\ 2 & d_1 \le d < d_2 \\ 3 & d \ge d_2 \end{cases}$$

其中 $d$ 是相机到角色的距离，$d_0, d_1, d_2$ 是分级阈值。典型值：$d_0 = 10m$, $d_1 = 30m$, $d_2 = 80m$。

### 帧率分配公式

同屏 $N$ 个角色时，总动画预算为 $B$ 毫秒（通常 2-5ms），每个角色分到的预算：

$$t_{per\_entity} = \frac{B}{N}$$

对于 LOD $i$ 的角色，其更新频率：

$$fps_i = \frac{1}{t_{per\_entity} \cdot cost_i}$$

其中 $cost_i$ 是该 LOD 级别的单次更新开销。通过降频实现：

$$frameSkip_i = \left\lceil \frac{target\_fps}{fps_i} \right\rceil$$

### GPU Skinning 收益模型

CPU 蒙皮 vs GPU 蒙皮的性能对比：

| 顶点数 | CPU 蒙皮耗时 | GPU 蒙皮耗时 (Compute) | GPU 矩阵上传 | GPU 总计 |
|--------|-------------|---------------------|-------------|---------|
| 10K | 0.3ms | 0.015ms | 0.005ms | 0.02ms |
| 100K | 3.0ms | 0.06ms | 0.05ms | 0.11ms |
| 1M | 30ms | 0.4ms | 0.5ms | 0.9ms |

GPU 蒙皮的优势在于并行处理：RTX 3070 有 5888 个 CUDA 核心，可同时处理数千个顶点。

## 完整 C++ 动画 LOD 系统

```cpp
#include <vector>
#include <algorithm>
#include <glm/glm.hpp>
#include <cmath>

using Vec3 = glm::vec3;

// ===== LOD 级别定义 =====
enum AnimLOD {
    LOD_FULL,       // 全精度: 完整骨骼，每帧更新 (近距离)
    LOD_HIGH,       // 高精度: 精简骨骼，每帧更新 (中近距离)
    LOD_REDUCED,    // 精简: 减少关节，隔帧更新 (中距离)
    LOD_MINIMAL,    // 最低: 仅主干关节，每4帧更新 (远距离)
    LOD_CULLED,     // 不可见: 完全不更新 (超远距离)
    LOD_COUNT
};

// ===== LOD 配置 =====
struct LODConfig {
    float distanceThresholds[LOD_COUNT] = {5.0f, 15.0f, 30.0f, 80.0f, FLT_MAX};
    int frameSkip[LOD_COUNT] = {1, 1, 2, 4, 0};     // 每隔N帧更新一次
    int maxJoints[LOD_COUNT] = {65, 45, 25, 10, 0};  // 最大关节数
    bool enableIK[LOD_COUNT] = {true, true, false, false, false};
    bool enableEvents[LOD_COUNT] = {true, true, true, false, false};
};

// ===== 动画实体 =====
struct AnimatedEntity {
    uint32_t id;
    Vec3 position;
    AnimLOD currentLOD = LOD_FULL;
    int importance = 1;               // 重要性权重（玩家>队友>NPC>群众）
    bool isVisible = true;
    bool isInCameraFrustum = true;
    float animTime = 0.0f;            // 动画累积时间
    int frameCounter = 0;             // 帧计数器
};

// ===== LOD 管理器 =====
class AnimationLODManager {
public:
    LODConfig config;
    std::vector<AnimatedEntity*> entities;

    // 每帧更新所有实体的 LOD 级别
    void UpdateLODs(const Vec3& cameraPos, const Vec3& cameraForward,
                    float fovAngle, int currentFrame) {
        for (auto* entity : entities) {
            // 1. 计算距离
            float distance = glm::length(entity->position - cameraPos);

            // 2. 检查是否在视锥内
            Vec3 toEntity = glm::normalize(entity->position - cameraPos);
            float dot = glm::dot(toEntity, cameraForward);
            float halfFov = std::cos(fovAngle * 0.5f);
            entity->isInCameraFrustum = (dot > halfFov);

            // 3. 根据距离和可见性确定 LOD
            if (!entity->isInCameraFrustum) {
                entity->currentLOD = LOD_CULLED;
            } else {
                entity->currentLOD = CalculateLOD(distance, entity->importance);
            }
        }
    }

    // 判断指定实体本帧是否需要更新
    bool ShouldUpdate(const AnimatedEntity* entity, int currentFrame) const {
        if (entity->currentLOD == LOD_CULLED) return false;
        int skip = config.frameSkip[entity->currentLOD];
        if (skip <= 1) return true;
        return (currentFrame % skip) == (entity->id % skip);  // 错开更新
    }

    // 获取指定 LOD 的动画时间步长
    float GetDeltaTime(const AnimatedEntity* entity, float baseDt) const {
        int skip = config.frameSkip[entity->currentLOD];
        return baseDt * std::max(1, skip);
    }

    int GetMaxJoints(AnimLOD lod) const {
        return config.maxJoints[lod];
    }

    // 统计信息
    struct LODStats {
        int countPerLOD[LOD_COUNT] = {0};
        int totalEntities = 0;
        int updatedThisFrame = 0;
    };

    LODStats GetStats(int currentFrame) const {
        LODStats stats;
        stats.totalEntities = (int)entities.size();
        for (auto* entity : entities) {
            stats.countPerLOD[entity->currentLOD]++;
            if (ShouldUpdate(entity, currentFrame)) {
                stats.updatedThisFrame++;
            }
        }
        return stats;
    }

private:
    AnimLOD CalculateLOD(float distance, int importance) const {
        // 重要性影响 LOD 选择：重要角色使用更精细的 LOD
        float adjustedDist = distance / std::max(1, importance);

        for (int i = 0; i < LOD_COUNT; i++) {
            if (adjustedDist < config.distanceThresholds[i]) {
                return (AnimLOD)i;
            }
        }
        return LOD_CULLED;
    }
};
```

```cpp
        skel.reducedToFull.resize(keep.size());

            skel.fullToReduced[keep[i]] = i;
            skel.reducedToFull[i] = keep[i];
        }
        return skel;
    }

    static ReducedSkeleton CreateLOD2() {  // 最低 15 关节
        // 仅保留主干 + 四肢根部
        ReducedSkeleton skel;
        std::vector<int> keep = {
            0, 1, 2, 3, 4, 7,   // 根 -> 头部（跳过中间脊柱）
            8, 10,              // 左臂（跳过前臂和手）
            12, 14,             // 右臂
            16, 18,             // 左腿
            19, 21,             // 右腿
        };

        skel.fullToReduced.resize(65, -1);
        skel.reducedToFull.resize(keep.size());

            skel.fullToReduced[keep[i]] = i;
            skel.reducedToFull[i] = keep[i];
        }
        return skel;

    // 将完整骨骼姿势映射到精简骨骼
    static void ReducePose(
        const std::vector<glm::mat4>& fullSkinMatrices,
        const ReducedSkeleton& reduced,
        std::vector<glm::mat4>& outReducedMatrices)
    {
        outReducedMatrices.resize(reduced.activeJoints.size());
        for (int i = 0; i < (int)reduced.activeJoints.size(); i++) {
            outReducedMatrices[i] = fullSkinMatrices[reduced.activeJoints[i]];
        }
    }
};
```

## GPU Skinning 实现

### Compute Shader 蒙皮

```glsl
// ===== compute_skinning.glsl =====
#version 450 core

layout(local_size_x = 256) in;

// 每个角色的数据
struct CharacterData {
    uint meshOffset;       // 顶点缓冲偏移
    uint vertexCount;      // 顶点数量
    uint jointOffset;      // 关节矩阵偏移
    uint jointCount;       // 关节数量
    mat4 modelMatrix;      // 模型变换矩阵
};

layout(std430, binding = 0) readonly buffer CharacterBuffer {
    CharacterData characters[];
};

layout(std430, binding = 1) readonly buffer JointMatrixBuffer {
    mat4 jointMatrices[];  // 所有关节矩阵的连续数组
};

struct VertexInput {
    vec3 position;
    vec3 normal;
    vec2 texCoord;
    uvec4 jointIndices;   // 4个关节索引（打包为uint）
    vec4 jointWeights;
};

layout(std430, binding = 2) readonly buffer VertexInputBuffer {
    VertexInput inputVertices[];
};

struct VertexOutput {
    vec4 position;         // 蒙皮后世界空间位置
    vec4 normal;           // 蒙皮后世界空间法线
    vec2 texCoord;
};

layout(std430, binding = 3) writeonly buffer VertexOutputBuffer {
    VertexOutput outputVertices[];
};

void main() {
    uint globalIdx = gl_GlobalInvocationID.x;

    // 确定这个顶点属于哪个角色（需要额外的数据结构）
    // 简化版本：假设每个角色的顶点连续排列
    uint charIdx = 0;  // 实际需要通过偏移查找

    CharacterData character = characters[charIdx];
    uint localVertexIdx = globalIdx - character.meshOffset;

    if (localVertexIdx >= character.vertexCount) return;

    VertexInput v = inputVertices[globalIdx];

    // 蒙皮计算
    vec4 skinnedPos = vec4(0.0);
    vec3 skinnedNormal = vec3(0.0);

    for (int i = 0; i < 4; i++) {
        float w = v.jointWeights[i];
        if (w > 0.001) {
            uint jointGlobalIdx = character.jointOffset + v.jointIndices[i];
            mat4 jointMat = jointMatrices[jointGlobalIdx];

            skinnedPos += w * jointMat * vec4(v.position, 1.0);
            skinnedNormal += w * mat3(jointMat) * v.normal;
        }
    }

    // 应用模型变换
    vec4 worldPos = character.modelMatrix * skinnedPos;
    vec3 worldNormal = mat3(character.modelMatrix) * skinnedNormal;

    outputVertices[globalIdx].position = worldPos;
    outputVertices[globalIdx].normal = vec4(normalize(worldNormal), 0.0);
    outputVertices[globalIdx].texCoord = v.texCoord;
}
```

### C++ 调度器

```cpp
// ===== GPU 蒙皮调度器 =====
class GPUSkinningDispatcher {
public:
    struct CharacterInstance {
        uint32_t meshBufferOffset;
        uint32_t vertexCount;
        uint32_t jointBufferOffset;
        uint32_t jointCount;
        glm::mat4 modelMatrix;
        std::vector<glm::mat4> jointMatrices;  // CPU 端更新
    };

    std::vector<CharacterInstance> characters;
    GLuint characterSSBO;
    GLuint jointMatrixSSBO;
    GLuint inputVertexSSBO;
    GLuint outputVertexSSBO;

    // 上传关节矩阵到 GPU
    void UploadJointMatrices() {
        // 收集所有角色的关节矩阵到一个连续数组
        std::vector<glm::mat4> allMatrices;
        for (auto& character : characters) {
            character.jointBufferOffset = (uint32_t)allMatrices.size();
            allMatrices.insert(allMatrices.end(),
                              character.jointMatrices.begin(),
                              character.jointMatrices.end());
        }

        // 上传到 SSBO
        glBindBuffer(GL_SHADER_STORAGE_BUFFER, jointMatrixSSBO);
        glBufferData(GL_SHADER_STORAGE_BUFFER,
                    allMatrices.size() * sizeof(glm::mat4),
                    allMatrices.data(), GL_DYNAMIC_DRAW);
    }

    // 执行 GPU 蒙皮
    void Dispatch() {
        UploadJointMatrices();

        // 绑定 SSBO
        glBindBufferBase(GL_SHADER_STORAGE_BUFFER, 0, characterSSBO);
        glBindBufferBase(GL_SHADER_STORAGE_BUFFER, 1, jointMatrixSSBO);
        glBindBufferBase(GL_SHADER_STORAGE_BUFFER, 2, inputVertexSSBO);
        glBindBufferBase(GL_SHADER_STORAGE_BUFFER, 3, outputVertexSSBO);

        // 计算总顶点数和工作组数
        uint32_t totalVertices = 0;
        for (auto& c : characters) totalVertices += c.vertexCount;

        uint32_t workGroups = (totalVertices + 255) / 256;

        // 执行 Compute Shader
        glDispatchCompute(workGroups, 1, 1);

        // 内存屏障
        glMemoryBarrier(GL_SHADER_STORAGE_BARRIER_BIT);
    }
};
```

```cpp
// ===== 帧交错动画更新系统 =====
class AnimationScheduler {
public:
    struct ScheduleEntry {
        AnimatedEntity* entity;
        int lastUpdateFrame;
        float accumulatedDt;
    };

    std::vector<ScheduleEntry> schedule;
    int maxUpdatesPerFrame = 50;  // 每帧最多更新的角色数

    // 按优先级排序更新
    void UpdateFrame(int currentFrame, float dt, AnimationLODManager& lodMgr) {
        // 1. 累积时间
        for (auto& entry : schedule) {
            if (lodMgr.ShouldUpdate(entry.entity, currentFrame)) {
                entry.accumulatedDt += dt;
            }
        }

        // 2. 选择本帧需要更新的角色（优先级排序）
        std::vector<ScheduleEntry*> toUpdate;
        for (auto& entry : schedule) {
            if (entry.accumulatedDt > 0) {
                toUpdate.push_back(&entry);
            }
        }

        // 按优先级排序：重要性 * 距离因子
        std::sort(toUpdate.begin(), toUpdate.end(),
            [](const ScheduleEntry* a, const ScheduleEntry* b) {
                return a->entity->importance > b->entity->importance;
            });

        // 3. 执行更新（限制数量）
        int updateCount = std::min((int)toUpdate.size(), maxUpdatesPerFrame);
        for (int i = 0; i < updateCount; i++) {
            auto* entry = toUpdate[i];

            // 更新动画
            float effectiveDt = entry->accumulatedDt;
            // TODO: 实际调用动画系统更新

            entry->lastUpdateFrame = currentFrame;
            entry->accumulatedDt = 0;
        }
    }
};
```

## LOD 切换平滑处理

```cpp
// ===== LOD 切换平滑系统 =====
class LODTransitionManager {
public:
    struct LODTransition {
        uint32_t entityId;
        AnimLOD fromLOD;
        AnimLOD toLOD;
        float blendProgress = 0;
        float blendDuration = 0.3f;
        bool active = false;

        // 保存切换前的姿势用于混合
        std::vector<glm::mat4> previousMatrices;
    };

    std::vector<LODTransition> transitions;

    void StartTransition(uint32_t entityId, AnimLOD from, AnimLOD to,
                         const std::vector<glm::mat4>& currentMatrices) {
        LODTransition trans;
        trans.entityId = entityId;
        trans.fromLOD = from;
        trans.toLOD = to;
        trans.blendProgress = 0;
        trans.active = true;
        trans.previousMatrices = currentMatrices;
        transitions.push_back(trans);
    }

    void Update(float dt) {
        for (auto it = transitions.begin(); it != transitions.end(); ) {
            it->blendProgress += dt / it->blendDuration;
            if (it->blendProgress >= 1.0f) {
                it = transitions.erase(it);
            } else {
                ++it;
            }
        }
    }

    float GetBlendAlpha(uint32_t entityId) const {
        for (const auto& trans : transitions) {
            if (trans.entityId == entityId && trans.active) {
                return SmoothStep(trans.blendProgress);
            }
        }
        return 1.0f;  // 无过渡
    }

private:
    float SmoothStep(float t) {
        return t * t * (3.0f - 2.0f * t);
    }
};
```

```csharp
using UnityEngine;

public class AnimationOptimizer : MonoBehaviour {
    Animator animator;
    float distanceToCamera;

    [Header("LOD Settings")]
    public float lod0Distance = 10f;
    public float lod1Distance = 30f;
    public float lod2Distance = 80f;
    public float cullDistance = 120f;

    [Header("Culling")]
    public bool enableAnimatorCulling = true;
    public bool enableOffscreenCulling = true;

    SkinnedMeshRenderer skinnedRenderer;

    void Start() {
        animator = GetComponent<Animator>();
        skinnedRenderer = GetComponent<SkinnedMeshRenderer>();

        // Unity 内置的动画剔除
        animator.cullingMode = enableAnimatorCulling ?
            AnimatorCullingMode.CullUpdateTransforms :
            AnimatorCullingMode.AlwaysAnimate;
    }

    void Update() {
        distanceToCamera = Vector3.Distance(transform.position, Camera.main.transform.position);

        // 距离 LOD
        if (distanceToCamera > cullDistance) {
            animator.enabled = false;
            skinnedRenderer.enabled = false;
            return;
        }

        animator.enabled = true;
        skinnedRenderer.enabled = true;

        if (distanceToCamera > lod2Distance) {
            // LOD2: 最低质量
            animator.speed = 0.25f;  // 4倍降频（通过降低更新速度实现）
            skinnedRenderer.quality = SkinQuality.Bone2;  // 每顶点最多2个骨骼
        } else if (distanceToCamera > lod1Distance) {
            // LOD1: 中等质量
            animator.speed = 0.5f;   // 2倍降频
            skinnedRenderer.quality = SkinQuality.Bone2;
        } else if (distanceToCamera > lod0Distance) {
            // LOD0: 高质量
            animator.speed = 1.0f;
            skinnedRenderer.quality = SkinQuality.Bone4;
        } else {
            // 超近距离：全精度
            animator.speed = 1.0f;
            skinnedRenderer.quality = SkinQuality.Bone4;
        }
    }

    // 动画事件 LOD 控制
    void OnBecameVisible() {
        animator.enabled = true;
    }

    void OnBecameInvisible() {
        if (enableOffscreenCulling) {
            animator.enabled = false;
        }
    }
}
```

## 常见陷阱与解决方案

1. **LOD 切换时动画跳跃**：切换 LOD 级别时骨骼数量变化导致姿势突变。使用过渡混合（在 0.2-0.3 秒内混合两个 LOD 的姿势）。

2. **采样频率降低导致动画事件漏触发**：降频更新时可能跳过事件触发点。解决方案：事件检测使用连续时间范围检查，而非逐帧检查。

3. **GPU 蒙皮与 CPU 碰撞检测数据不一致**：碰撞检测在 CPU 上使用原始网格。解决方案：CPU 维护简化版蒙皮用于碰撞，或使用 Compute Shader 输出碰撞数据。

4. **骨骼精简后法线变形导致光照异常**：精简骨骼的法线变换不正确。解决方案：为精简骨骼单独计算法线变换矩阵，或在精简时保留法线关键骨骼。
