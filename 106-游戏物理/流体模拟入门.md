# 流体模拟入门

## 核心概念

游戏中的流体模拟主要用于水、岩浆、烟雾等效果。完整物理模拟计算量巨大，游戏通常使用简化方法。

### SPH（Smoothed Particle Hydrodynamics）

基于粒子的方法，每个粒子代表一小团流体。通过邻域粒子的加权平均计算密度、压力和粘性力。

**核心公式**：
- **密度**：`ρ(r) = Σ m × W(r - ri, h)`（核函数加权求和）
- **压力力**：`F_pressure = -m × (pi/ρi² + pj/ρj²) × ∇W`
- **粘性力**：`F_viscosity = μ × m × (vj - vi)/ρj × ∇²W`

常用核函数：Poly6核（密度）、Spiky核（压力梯度）、Viscosity核（粘性拉普拉斯）。

### 网格法（Eulerian）

将空间划分为网格，每个格子存储流体属性。通过Navier-Stokes方程求解：
1. 施加外力
2. 求解压力泊松方程确保不可压缩
3. 半拉格朗日平流

### 2D水面效果

简化的水面模拟：一维高度场或正弦波叠加。

## 代码示例

```cpp
// SPH粒子
struct SPHParticle {
    Vec3 position, velocity;
    float density;
    float pressure;
};

// Poly6核函数
float poly6Kernel(float r2, float h) {
    if (r2 >= h*h) return 0;
    float diff = h*h - r2;
    return 315.0f / (64.0f * M_PI * pow(h, 9)) * diff * diff * diff;
}

// Spiky核梯度
Vec3 spikyGradient(Vec3 r, float h, float rLen) {
    if (rLen >= h || rLen < 0.0001f) return {0,0,0};
    float coeff = -45.0f / (M_PI * pow(h, 6)) * (h - rLen) * (h - rLen);
    return r * (coeff / rLen);
}

// SPH模拟器（简化）
class SPHSimulation {
    std::vector<SPHParticle> particles;
    float smoothingRadius = 0.1f;
    float restDensity = 1000.0f;
    float stiffness = 200.0f; // 气体常数
    float viscosity = 0.01f;

    void step(float dt) {
        // 1. 计算密度和压力
        for (auto& pi : particles) {
            pi.density = 0;
            for (auto& pj : particles) {
                Vec3 diff = pi.position - pj.position;
                float r2 = diff.dot(diff);
                pi.density += poly6Kernel(r2, smoothingRadius);
            }
            pi.pressure = stiffness * (pi.density - restDensity);
        }

        // 2. 计算力
        for (auto& pi : particles) {
            Vec3 force = {0, -9.8f * pi.density, 0}; // 重力

            for (auto& pj : particles) {
                if (&pi == &pj) continue;
                Vec3 diff = pi.position - pj.position;
                float rLen = diff.length();
                if (rLen >= smoothingRadius) continue;

                // 压力力
                float pTerm = (pi.pressure / (pi.density*pi.density))
                            + (pj.pressure / (pj.density*pj.density));
                force -= spikyGradient(diff, smoothingRadius, rLen) * pTerm;

                // 粸性力
                force += (pj.velocity - pi.velocity) * viscosity / pj.density;
            }

            pi.velocity += force / pi.density * dt;
            pi.position += pi.velocity * dt;
        }
    }
};

// 2D水面高度场（简化版）
class WaterSurface {
    std::vector<float> heights, velocities;
    std::vector<float> prevHeights;
    int width;
    float springK = 0.03f;
    float damping = 0.025f;
    float spread = 0.25f;

public:
    void init(int w, float h) {
        width = w;
        heights.resize(w, h);
        velocities.resize(w, 0);
        prevHeights.resize(w, h);
    }

    void update(float dt) {
        // 弹簧-阻尼模型
        for (int i = 0; i < width; i++) {
            float force = -springK * (heights[i] - waterLevel);
            force += damping * (prevHeights[i] - heights[i]);
            velocities[i] += force;
        }

        // 波的传播（左右邻居的影响）
        std::vector<float> leftDeltas(width), rightDeltas(width);
        for (int i = 0; i < width; i++) {
            if (i > 0)
                leftDeltas[i] = spread * (heights[i] - heights[i-1]);
            if (i < width-1)
                rightDeltas[i] = spread * (heights[i] - heights[i+1]);
        }
        for (int i = 0; i < width; i++) {
            if (i > 0) velocities[i-1] += leftDeltas[i];
            if (i < width-1) velocities[i+1] += rightDeltas[i];
        }

        // 积分更新高度
        prevHeights = heights;
        for (int i = 0; i < width; i++)
            heights[i] += velocities[i] * dt;
    }

    // 外部扰动（物体落水）
    void splash(int index, float force) {
        velocities[index] += force;
    }
};
```

## 游戏应用场景

1. **水花效果**：物体落水时用高度场产生波纹
2. **岩浆流**：2D网格法模拟岩浆在通道中流动
3. **烟雾/蒸汽**：SPH粒子渲染为半透明广告牌
4. **洪水/海啸**：大规模2D浅水方程模拟

## 常见陷阱

- **SPH性能**：邻域搜索是瓶颈，需用空间哈希加速，粒子数通常限制在数千
- **网格法内存**：3D网格内存消耗巨大，游戏常用2D或稀疏网格
- **时间步长**：流体模拟要求小时间步长，否则不稳定
- **视觉效果优先**：很多游戏的"水"其实是粒子+着色器+法线贴图，而非真实物理
