# 布料与软体模拟

## 核心概念

布料和软体模拟用质点-弹簧系统将连续体离散化为粒子网格，通过约束保持形状，通过积分更新位置。

### 质点-弹簧模型

将布料表示为NxM的质点网格，质点间用弹簧连接：
- **结构弹簧**：连接相邻水平和垂直质点
- **剪切弹簧**：连接对角质点，抵抗剪切变形
- **弯曲弹簧**：隔一个质点连接，抵抗弯曲

### Verlet积分

相比欧拉积分更稳定，适合布料模拟：

`x(t+dt) = 2×x(t) - x(t-dt) + a×dt²`

不需要显式存储速度，速度隐含在位置差中。对约束的满足更直观。

### 约束求解

距离约束：`|p1 - p2| = restLength`

每次迭代时检查实际距离，将质点向平衡位置推/拉。多次迭代提高精度。

### 碰撞处理

布料与世界物体的碰撞：检测质点穿透碰撞体，将质点推到碰撞体表面。

// Verlet质点
struct VerletParticle {
    Vec3 position;
    Vec3 oldPosition;
    Vec3 acceleration;
    bool pinned; // 固定质点（如布料顶端）

    void update(float dt) {
        if (pinned) return;
        Vec3 velocity = position - oldPosition;
        oldPosition = position;
        position = position + velocity + acceleration * dt * dt;
        acceleration = {0, 0, 0};
    }
};

// 距离约束
struct DistanceConstraint {
    int p1, p2;
    float restLength;

    void satisfy(std::vector<VerletParticle>& particles) {
        Vec3 diff = particles[p2].position - particles[p1].position;
        float dist = diff.length();
        if (dist < 0.0001f) return;
        Vec3 correction = diff * ((dist - restLength) / dist) * 0.5f;

        if (!particles[p1].pinned) particles[p1].position += correction;
        if (!particles[p2].pinned) particles[p2].position -= correction;
    }
};

// 布料模拟器
class ClothSimulation {
    std::vector<VerletParticle> particles;
    std::vector<DistanceConstraint> constraints;
    int width, height;
    float spacing;

public:
    void init(int w, int h, float sp, Vec3 origin) {
        width = w; height = h; spacing = sp;

        // 创建质点网格
        for (int y = 0; y < h; y++)
            for (int x = 0; x < w; x++) {
                VerletParticle p;
                p.position = origin + Vec3{(float)x * sp, 0, (float)y * sp};
                p.oldPosition = p.position;
                p.pinned = (y == 0 && x % 4 == 0); // 顶部每隔4个固定
                particles.push_back(p);
            }

        // 创建约束
        for (int y = 0; y < h; y++)
            for (int x = 0; x < w; x++) {
                int idx = y * w + x;
                // 结构弹簧
                if (x < w-1) constraints.push_back({idx, idx+1, sp});
                if (y < h-1) constraints.push_back({idx, idx+w, sp});
                // 剪切弹簧
                if (x < w-1 && y < h-1) {
                    constraints.push_back({idx, idx+w+1, sp*1.414f});
                    constraints.push_back({idx+1, idx+w, sp*1.414f});
                }
                // 弯曲弹簧
                if (x < w-2) constraints.push_back({idx, idx+2, sp*2});
                if (y < h-2) constraints.push_back({idx, idx+w*2, sp*2});
            }
    }

    void update(float dt) {
        // 施加重力
        for (auto& p : particles)
            p.acceleration += Vec3{0, -9.8f, 0};

        // Verlet积分
        for (auto& p : particles)
            p.update(dt);

        // 迭代约束求解
        for (int iter = 0; iter < 5; iter++)
            for (auto& c : constraints)
                c.satisfy(particles);
    }

    // 地面碰撞
    void collideWithGround(float groundY) {
        for (auto& p : particles) {
            if (p.position.y < groundY) {
                p.position.y = groundY;
                // 可选：添加摩擦
                Vec3 delta = p.position - p.oldPosition;
                p.oldPosition = p.position - delta * 0.8f; // 地面摩擦
            }
        }
    }
};

## 游戏应用场景

1. **披风/旗帜**：布料模拟实现飘动效果
2. **帐篷/帷幕**：可交互的布料（风吹、角色碰撞）
3. **软体怪物**：果冻状敌人使用质点-弹簧球体
4. **可变形表面**：蹦床、沙地脚印的软体效果

## 常见陷阱

- **约束迭代不足**：布料看起来像橡胶一样拉伸
- **时间步长过大**：布料"爆炸"（质点飞散），需减小dt或增加子步
- **碰撞穿透**：薄布料快速移动时可能穿透碰撞体
- **渲染性能**：高质量布料网格顶点多，需LOD或简化
- **替代方案**：很多游戏用顶点动画或着色器伪布料替代真实物理模拟
