# 碰撞检测算法

## 核心概念

高效的碰撞检测是物理引擎的核心，通常分为两个阶段：Broad Phase（粗检测，快速排除不可能碰撞的对）和Narrow Phase（精检测，精确判断碰撞并计算碰撞信息）。

### 检测流水线

```
所有刚体 (N个)
    ↓
Broad Phase: O(n²) → O(n log n) ~ O(n)
  → 空间划分 / BVH / 空间哈希 / 扫描修剪
  → 输出: 可能碰撞的物体对列表 (M对, M << N²)
    ↓
Narrow Phase: O(1) per pair
  → GJK / SAT / 特定形状相交测试
  → 输出: 碰撞信息（接触点、法线、穿透深度）
    ↓
碰撞响应
```

## Broad Phase详解

### 包围体层次（BVH）

构建树形结构，每个节点包含其子节点的AABB。遍历时如果两个节点AABB不相交，则跳过整棵子树：

```cpp
struct AABB {
    Vec3 min, max;
    void expand(const AABB& other) {
        min.x = std::fmin(min.x, other.min.x);
        min.y = std::fmin(min.y, other.min.y);
        min.z = std::fmin(min.z, other.min.z);
        max.x = std::fmax(max.x, other.max.x);
        max.y = std::fmax(max.y, other.max.y);
        max.z = std::fmax(max.z, other.max.z);
    }
    float surfaceArea() const {
        Vec3 d = max - min;
        return 2 * (d.x*d.y + d.y*d.z + d.z*d.x);
    }
    bool overlaps(const AABB& other) const {
        return min.x <= other.max.x && max.x >= other.min.x
            && min.y <= other.max.y && max.y >= other.min.y
            && min.z <= other.max.z && max.z >= other.min.z;
    }
};

// BVH节点
struct BVHNode {
    AABB bounds;
    int bodyIndex;      // 叶节点: 刚体索引，内部节点: -1
    int leftChild;      // 左子节点索引
    int rightChild;     // 右子节点索引
    int parent;         // 父节点索引
    bool isLeaf;
};

// SAH (Surface Area Heuristic) BVH构建
class BVH {
    std::vector<BVHNode> nodes;
    int rootIndex;

public:
    // 使用SAH构建最优BVH
    int buildSAH(std::vector<int>& bodyIndices,
                 const std::vector<AABB>& bodyBounds, int start, int end) {

        if (end - start == 1) {
            // 叶节点
            int nodeIdx = (int)nodes.size();
            BVHNode leaf;
            leaf.bounds = bodyBounds[bodyIndices[start]];
            leaf.bodyIndex = bodyIndices[start];
            leaf.isLeaf = true;
            leaf.leftChild = leaf.rightChild = -1;
            nodes.push_back(leaf);
            return nodeIdx;
        }

        // 计算所有物体的总体AABB
        AABB totalBounds = bodyBounds[bodyIndices[start]];
        for (int i = start + 1; i < end; i++)
            totalBounds.expand(bodyBounds[bodyIndices[i]]);

        // 尝试每个轴的每个分割位置，找到SAH代价最小的
        float bestCost = FLT_MAX;
        int bestAxis = -1;
        int bestSplit = -1;

        for (int axis = 0; axis < 3; axis++) {
            // 按当前轴的中心排序
            std::sort(bodyIndices.begin() + start, bodyIndices.begin() + end,
                [&](int a, int b) {
                    float ca = (bodyBounds[a].min[axis] + bodyBounds[a].max[axis]) * 0.5f;
                    float cb = (bodyBounds[b].min[axis] + bodyBounds[b].max[axis]) * 0.5f;
                    return ca < cb;
                });

            // 对每个分割位置计算SAH代价
            for (int split = start + 1; split < end; split++) {
                AABB leftBounds = bodyBounds[bodyIndices[start]];
                for (int i = start + 1; i < split; i++)
                    leftBounds.expand(bodyBounds[bodyIndices[i]]);

                AABB rightBounds = bodyBounds[bodyIndices[split]];
                for (int i = split + 1; i < end; i++)
                    rightBounds.expand(bodyBounds[bodyIndices[i]]);

                int leftCount = split - start;
                int rightCount = end - split;
                float cost = leftCount * leftBounds.surfaceArea()
                           + rightCount * rightBounds.surfaceArea();

                if (cost < bestCost) {
                    bestCost = cost;
                    bestAxis = axis;
                    bestSplit = split;
                }
            }
        }

        // 按最优轴重新排序
        if (bestAxis >= 0) {
            std::sort(bodyIndices.begin() + start, bodyIndices.begin() + end,
                [&](int a, int b) {
                    float ca = (bodyBounds[a].min[bestAxis] + bodyBounds[a].max[bestAxis]) * 0.5f;
                    float cb = (bodyBounds[b].min[bestAxis] + bodyBounds[b].max[bestAxis]) * 0.5f;
                    return ca < cb;
                });
        }

        // 递归构建
        int nodeIdx = (int)nodes.size();
        BVHNode internalNode;
        internalNode.isLeaf = false;
        internalNode.bodyIndex = -1;
        nodes.push_back(internalNode);

        int left = buildSAH(bodyIndices, bodyBounds, start, bestSplit);
        int right = buildSAH(bodyIndices, bodyBounds, bestSplit, end);

        nodes[nodeIdx].leftChild = left;
        nodes[nodeIdx].rightChild = right;
        nodes[nodeIdx].bounds = nodes[left].bounds;
        nodes[nodeIdx].bounds.expand(nodes[right].bounds);

        nodes[left].parent = nodeIdx;
        nodes[right].parent = nodeIdx;

        return nodeIdx;
    }

    // BVH遍历：查找潜在碰撞对
    void query(int nodeA, int nodeB, std::vector<std::pair<int,int>>& pairs) {
        if (!nodes[nodeA].bounds.overlaps(nodes[nodeB].bounds))
            return;

        if (nodes[nodeA].isLeaf && nodes[nodeB].isLeaf) {
            if (nodes[nodeA].bodyIndex != nodes[nodeB].bodyIndex)
                pairs.push_back({nodes[nodeA].bodyIndex, nodes[nodeB].bodyIndex});
        } else if (nodes[nodeA].isLeaf) {
            query(nodeA, nodes[nodeB].leftChild, pairs);
            query(nodeA, nodes[nodeB].rightChild, pairs);
        } else {
            query(nodes[nodeA].leftChild, nodeB, pairs);
            query(nodes[nodeA].rightChild, nodeB, pairs);
        }
    }
};
```

### 空间哈希（Spatial Hash）

将空间划分为网格，每个物体放入其覆盖的格子：

```cpp
class SpatialHashBroadPhase {
    float cellSize;
    std::unordered_map<int64_t, std::vector<int>> grid;

    int64_t hashKey(int cx, int cy, int cz) {
        return ((int64_t)cx * 73856093) ^ ((int64_t)cy * 19349663) ^ ((int64_t)cz * 83492791);
    }

public:
    SpatialHashBroadPhase(float cell) : cellSize(cell) {}

    void clear() { grid.clear(); }

    void insert(int bodyIndex, AABB aabb) {
        int minCX = (int)floor(aabb.min.x / cellSize);
        int minCY = (int)floor(aabb.min.y / cellSize);
        int minCZ = (int)floor(aabb.min.z / cellSize);
        int maxCX = (int)floor(aabb.max.x / cellSize);
        int maxCY = (int)floor(aabb.max.y / cellSize);
        int maxCZ = (int)floor(aabb.max.z / cellSize);

        for (int x = minCX; x <= maxCX; x++)
            for (int y = minCY; y <= maxCY; y++)
                for (int z = minCZ; z <= maxCZ; z++)
                    grid[hashKey(x, y, z)].push_back(bodyIndex);
    }

    std::set<std::pair<int,int>> getPotentialPairs() {
        std::set<std::pair<int,int>> pairs;
        for (auto& [key, bodies] : grid) {
            for (size_t i = 0; i < bodies.size(); i++)
                for (size_t j = i+1; j < bodies.size(); j++)
                    pairs.insert({std::min(bodies[i], bodies[j]),
                                  std::max(bodies[i], bodies[j])});
        }
        return pairs;
    }
};
```

### 扫描-修剪（Sweep and Prune）

对每个轴的AABB区间排序，当两物体在所有三个轴上的区间都重叠时标记为碰撞对：

```cpp
class SweepAndPrune {
    struct Endpoint {
        float value;
        int bodyIndex;
        bool isMin; // true=最小值, false=最大值
    };

    std::vector<Endpoint> axisX, axisY, axisZ;
    std::set<std::pair<int,int>> overlapX, overlapY, overlapZ;

public:
    // 插入排序（利用帧间连续性）
    void update(const std::vector<AABB>& bounds) {
        // 对每个轴的端点排序
        for (int axis = 0; axis < 3; axis++) {
            auto& endpoints = (axis == 0) ? axisX : (axis == 1) ? axisY : axisZ;
            auto& overlaps = (axis == 0) ? overlapX : (axis == 1) ? overlapY : overlapZ;

            // 插入排序
            for (int i = 1; i < (int)endpoints.size(); i++) {
                Endpoint key = endpoints[i];
                int j = i - 1;
                while (j >= 0 && endpoints[j].value > key.value) {
                    endpoints[j+1] = endpoints[j];

                    // 更新重叠状态
                    if (endpoints[j].isMin != key.isMin) {
                        auto pair = std::minmax(endpoints[j].bodyIndex, key.bodyIndex);
                        if (endpoints[j].isMin)
                            overlaps.erase(pair);
                        else
                            overlaps.insert(pair);
                    }
                    j--;
                }
                endpoints[j+1] = key;
            }
        }
    }

    // 三个轴都重叠的对才是真正的潜在碰撞对
    std::set<std::pair<int,int>> getPotentialPairs() {
        std::set<std::pair<int,int>> result;
        for (auto& pair : overlapX) {
            if (overlapY.count(pair) && overlapZ.count(pair))
                result.insert(pair);
        }
        return result;
    }
};
```

## Narrow Phase详解

### GJK完整实现（3D）

```cpp
// Support函数
Vec3 support(const ConvexShape& a, const ConvexShape& b, Vec3 dir) {
    return a.furthestPoint(dir) - b.furthestPoint(-dir);
}

// 3D GJK
bool gjk3D(const ConvexShape& a, const ConvexShape& b) {
    Vec3 simplex[4];
    int count = 0;

    Vec3 dir = {1, 0, 0}; // 初始搜索方向
    simplex[0] = support(a, b, dir);
    count = 1;

    if (simplex[0].dot(dir) < 0) return false;
    dir = -simplex[0];

    for (int iter = 0; iter < 30; iter++) {
        Vec3 newPt = support(a, b, dir);
        if (newPt.dot(dir) < 0) return false;

        simplex[count++] = newPt;
        if (updateSimplex(simplex, count, dir)) return true;
    }
    return false;
}

// 更新单纯形和搜索方向
bool updateSimplex(Vec3* simplex, int& count, Vec3& dir) {
    switch (count) {
    case 2: { // 线段AB
        Vec3 AB = simplex[1] - simplex[0];
        Vec3 AO = -simplex[0];
        dir = AB.cross(AO).cross(AB);
        if (dir.lengthSq() < 1e-10f) dir = {0, 1, 0};
        dir = dir.normalized();
        return false;
    }
    case 3: { // 三角形ABC
        Vec3 A = simplex[2], B = simplex[1], C = simplex[0];
        Vec3 AB = B - A, AC = C - A, AO = -A;
        Vec3 n = AB.cross(AC);

        // 检查AB边
        Vec3 ABperp = AB.cross(n);
        if (ABperp.dot(AO) > 0) {
            simplex[0] = A; simplex[1] = B; count = 2;
            dir = AB.cross(AO).cross(AB).normalized();
            return false;
        }
        // 检查AC边
        Vec3 ACperp = n.cross(AC);
        if (ACperp.dot(AO) > 0) {
            simplex[0] = A; simplex[1] = C; count = 2;
            dir = AC.cross(AO).cross(AC).normalized();
            return false;
        }
        // 原点在三角形上方或下方
        if (n.dot(AO) > 0)
            dir = n.normalized();
        else {
            dir = -n.normalized();
            std::swap(simplex[0], simplex[1]); // 翻转三角形
        }
        return false;
    }
    case 4: { // 四面体ABCD
        Vec3 D = simplex[3], C = simplex[2], B = simplex[1], A = simplex[0];
        Vec3 DO = -D;

        // 检查三个面
        Vec3 DC = C - D, DB = B - D, DA = A - D;
        Vec3 nDCB = DC.cross(DB);
        if (nDCB.dot(DO) > 0) {
            simplex[0] = D; simplex[1] = C; simplex[2] = B; count = 3;
            dir = nDCB.normalized();
            return false;
        }
        Vec3 nDBA = DB.cross(DA);
        if (nDBA.dot(DO) > 0) {
            simplex[0] = D; simplex[1] = B; simplex[2] = A; count = 3;
            dir = nDBA.normalized();
            return false;
        }
        Vec3 nDAC = DA.cross(DC);
        if (nDAC.dot(DO) > 0) {
            simplex[0] = D; simplex[1] = A; simplex[2] = C; count = 3;
            dir = nDAC.normalized();
            return false;
        }
        return true; // 原点在四面体内
    }
    }
    return false;
}
```

### EPA穿透深度计算

GJK之后使用EPA扩展单纯形到Minkowski差集的最近表面：

```cpp
struct ContactManifold {
    Vec3 normal;        // 碰撞法线
    float penetration;  // 穿透深度
    Vec3 contactPoint;  // 接触点
};

ContactManifold epa(const ConvexShape& a, const ConvexShape& b,
                     Vec3* simplex, int simplexCount) {
    // 将单纯形的面存储为三角形列表
    struct Face { int a, b, c; Vec3 normal; float dist; };
    std::vector<Face> faces;
    std::vector<Vec3> polytope(simplex, simplex + simplexCount);

    // 初始化四面体的四个面
    auto addFace = [&](int a, int b, int c) {
        Vec3 n = (polytope[b] - polytope[a]).cross(polytope[c] - polytope[a]).normalized();
        float d = n.dot(polytope[a]);
        if (d < 0) { std::swap(b, c); n = -n; d = -d; }
        faces.push_back({a, b, c, n, d});
    };

    addFace(0, 1, 2);
    addFace(0, 3, 1);
    addFace(0, 2, 3);
    addFace(1, 3, 2);

    const int MAX_ITER = 64;
    for (int iter = 0; iter < MAX_ITER; iter++) {
        // 找到离原点最近的面
        int closestIdx = 0;
        float minDist = faces[0].dist;
        for (int i = 1; i < (int)faces.size(); i++) {
            if (faces[i].dist < minDist) {
                minDist = faces[i].dist;
                closestIdx = i;
            }
        }

        Vec3 searchDir = faces[closestIdx].normal;
        Vec3 newPt = support(a, b, searchDir);

        float newDist = newPt.dot(searchDir);
        if (newDist - minDist < 0.001f) {
            // 收敛：返回最近面的信息
            return {searchDir, minDist,
                    (polytope[faces[closestIdx].a] +
                     polytope[faces[closestIdx].b] +
                     polytope[faces[closestIdx].c]) * (1.0f/3.0f)};
        }

        // 扩展多面体
        polytope.push_back(newPt);
        int newIdx = (int)polytope.size() - 1;

        // 移除所有面向新点的面
        std::vector<Face> newFaces;
        for (auto& face : faces) {
            if ((polytope[newIdx] - polytope[face.a]).dot(face.normal) > 0) {
                continue; // 面向新点，移除
            }
            newFaces.push_back(face);
        }
        faces = newFaces;

        // 为新点和边界边创建新面
        // ... (边界边检测和新面创建)
    }

    return {searchDir, minDist, {0, 0, 0}};
}
```

### Swept Sphere连续碰撞检测

```cpp
float sweptSphereSphere(Vec3 p0, Vec3 p1, float r0,
                        Vec3 p2, Vec3 p3, float r2) {
    Vec3 s = p1 - p0;       // A的运动轨迹
    Vec3 st = p3 - p2;      // B的运动轨迹
    Vec3 relVel = s - st;   // 相对速度
    Vec3 relPos = p0 - p2;  // 相对位置
    float radiusSum = r0 + r2;

    float a = relVel.dot(relVel);
    float b = 2 * relPos.dot(relVel);
    float c = relPos.dot(relPos) - radiusSum * radiusSum;
    float disc = b*b - 4*a*c;

    if (disc < 0) return -1.0f; // 无碰撞
    float t = (-b - sqrt(disc)) / (2*a);
    return (t >= 0 && t <= 1) ? t : -1.0f;
}
```

## 游戏应用场景

1. **大量粒子碰撞**：空间哈希快速筛选碰撞对（O(n)平均复杂度）
2. **精确物理**：GJK+EPA用于任意凸多面体碰撞，获取精确穿透深度
3. **子弹穿透**：CCD防止高速子弹穿过薄墙
4. **大规模场景**：BVH用于开放世界的碰撞检测加速
5. **物理引擎**：Bullet/PhysX/Chaos都使用BVH+Narrow Phase

## 常见陷阱

- **BVH重建**：动态场景需要定期重建或增量更新BVH，否则效率下降
- **GJK退化**：原点在单纯形边上时方向计算不稳定，需特殊处理零向量
- **CCD开销**：连续碰撞检测计算量大，仅对高速物体启用
- **浮点误差**：两物体"刚好接触"时的判据需引入epsilon容差（约0.001）
- **空间哈希碰撞**：不同格子可能映射到同一桶，需处理哈希冲突
- **扫描修剪的插入排序**：利用帧间连续性，但如果物体大幅移动效率退化
