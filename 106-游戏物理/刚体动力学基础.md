# 刚体动力学基础

## 核心概念

刚体是物理引擎中的基本单元，假设物体在受力时不发生形变。核心变量包括线性运动（位置、速度、加速度）和旋转运动（朝向、角速度、角加速度）。理解刚体动力学是实现真实物理模拟的基础。

### 基本物理量

- **质量（Mass）**：m，物体惯性的量度，决定力产生加速度的大小
- **质心（Center of Mass）**：刚体的"平衡点"，所有力的效果可以视为作用在质心上
- **惯性张量（Inertia Tensor）**：3x3矩阵，描述刚体绕各轴旋转的难易程度，类似于旋转运动中的"质量"
- **逆质量（Inverse Mass）**：1/m，静态物体为0，用于避免除零
- **逆惯性张量（Inverse Inertia）**：I^(-1)，用于从力矩直接计算角加速度

### 牛顿第二定律（线性）

```
F = m * a
a = F / m

速度和位置的积分（欧拉法）:
v(t+dt) = v(t) + a * dt
x(t+dt) = x(t) + v(t+dt) * dt

更精确的Verlet积分:
x(t+dt) = 2*x(t) - x(t-dt) + a * dt^2
```

### 欧拉方程（旋转）

```
τ = I * α + ω × (I * ω)

其中:
τ = 力矩（扭矩）
I = 惯性张量（世界空间）
α = 角加速度
ω = 角速度

对于各向同性刚体（球体等）简化为:
α = I^(-1) * τ
```

### 力矩

力矩 = 力臂 × 力：`τ = r × F`

力作用点偏离质心越远，力矩越大。这是开门时在门边缘推比在门轴附近推更省力的原因。

## 积分方法对比

```
1. 显式欧拉 (Explicit Euler):
   v(t+dt) = v(t) + a(t) * dt
   x(t+dt) = x(t) + v(t) * dt
   问题: 能量不守恒，容易爆炸

2. 半隐式欧拉 (Semi-Implicit Euler / Symplectic Euler):
   v(t+dt) = v(t) + a(t) * dt
   x(t+dt) = x(t) + v(t+dt) * dt    ← 用新速度更新位置
   优点: 能量守恒好，推荐用于游戏

3. Verlet积分:
   x(t+dt) = 2*x(t) - x(t-dt) + a * dt^2
   优点: 无需存储速度，天然适合约束求解
   缺点: 速度隐式存储，不易施加力

4. RK4 (四阶Runge-Kutta):
   k1 = f(x, v)
   k2 = f(x + v*dt/2, v + k1*dt/2)
   k3 = f(x + (v+k1*dt/2)*dt/2, v + k2*dt/2)
   k4 = f(x + (v+k2*dt/2)*dt, v + k3*dt)
   x(t+dt) = x(t) + (v + (k1+2*k2+2*k3+k4)/6) * dt
   优点: 高精度，适合物理模拟
   缺点: 计算量大（4倍于欧拉法）
```

#include <cmath>

struct Vec3 {
    float x, y, z;
    Vec3() : x(0), y(0), z(0) {}
    Vec3(float x, float y, float z) : x(x), y(y), z(z) {}
    Vec3 operator+(const Vec3& o) const { return {x+o.x, y+o.y, z+o.z}; }
    Vec3 operator-(const Vec3& o) const { return {x-o.x, y-o.y, z-o.z}; }
    Vec3 operator*(float s) const { return {x*s, y*s, z*s}; }
    Vec3 operator-() const { return {-x, -y, -z}; }
    Vec3& operator+=(const Vec3& o) { x+=o.x; y+=o.y; z+=o.z; return *this; }
    Vec3& operator-=(const Vec3& o) { x-=o.x; y-=o.y; z-=o.z; return *this; }
    float dot(const Vec3& o) const { return x*o.x + y*o.y + z*o.z; }
    Vec3 cross(const Vec3& o) const {
        return {y*o.z-z*o.y, z*o.x-x*o.z, x*o.y-y*o.x};
    }
    float lengthSq() const { return x*x+y*y+z*z; }
    float length() const { return sqrt(lengthSq()); }
    Vec3 normalized() const { float l=length(); return l>1e-6f?Vec3{x/l,y/l,z/l}:Vec3{0,0,0}; }
};

struct Quaternion {
    float w, x, y, z;
    Quaternion() : w(1), x(0), y(0), z(0) {}
    Quaternion(float w, float x, float y, float z) : w(w), x(x), y(y), z(z) {}
    Quaternion operator*(const Quaternion& q) const {
        return {w*q.w-x*q.x-y*q.y-z*q.z, w*q.x+x*q.w+y*q.z-z*q.y,
                w*q.y-x*q.z+y*q.w+z*q.x, w*q.z+x*q.y-y*q.x+z*q.w};
    }
    Quaternion operator+(const Quaternion& q) const { return {w+q.w,x+q.x,y+q.y,z+q.z}; }
    Quaternion operator*(float s) const { return {w*s,x*s,y*s,z*s}; }
    Quaternion conjugate() const { return {w,-x,-y,-z}; }
    Quaternion normalized() const {
        float l=sqrt(w*w+x*x+y*y+z*z);
        return l>1e-6f?Quaternion{w/l,x/l,y/l,z/l}:Quaternion{1,0,0,0};
    }
    Vec3 rotate(const Vec3& v) const {
        Quaternion qv={0,v.x,v.y,v.z};
        Quaternion r=(*this)*qv*conjugate();
        return {r.x,r.y,r.z};
    }
};

struct Mat3 {
    float m[3][3];
    Mat3() { memset(m, 0, sizeof(m)); }
    static Mat3 diagonal(float a, float b, float c) {
        Mat3 r; r.m[0][0]=a; r.m[1][1]=b; r.m[2][2]=c; return r;
    }
    Vec3 transform(const Vec3& v) const {
        return {m[0][0]*v.x+m[1][0]*v.y+m[2][0]*v.z,
                m[0][1]*v.x+m[1][1]*v.y+m[2][1]*v.z,
                m[0][2]*v.x+m[1][2]*v.y+m[2][2]*v.z};
    }
    Mat3 operator*(const Mat3& b) const {
        Mat3 r;
        for(int i=0;i<3;i++) for(int j=0;j<3;j++) {
            r.m[i][j]=0;
            for(int k=0;k<3;k++) r.m[i][j]+=m[k][j]*b.m[i][k];
        }
        return r;
    }
    Mat3 transposed() const {
        Mat3 r;
        for(int i=0;i<3;i++) for(int j=0;j<3;j++) r.m[i][j]=m[j][i];
        return r;
    }
};

// ========== 完整刚体结构 ==========
struct RigidBody {
    // 线性属性
    Vec3 position;
    Vec3 velocity;
    Vec3 forceAccum;        // 力累加器
    float mass;
    float inverseMass;      // 1/mass，静态物体为0

    // 旋转属性
    Quaternion orientation;
    Vec3 angularVelocity;
    Vec3 torqueAccum;       // 力矩累加器
    Mat3 inertiaTensor;     // 体空间惯性张量
    Mat3 inverseInertia;    // 体空间惯性张量逆

    // 阻尼（模拟空气阻力等）
    float linearDamping;
    float angularDamping;

    // 睡眠状态
    bool isAwake;
    float sleepTimer;
    static constexpr float SLEEP_THRESHOLD = 0.01f;
    static constexpr float SLEEP_TIME = 1.0f;

    // AABB（用于Broad Phase）
    Vec3 aabbMin, aabbMax;

    // 是否为静态物体
    bool isStatic() const { return inverseMass == 0; }

    RigidBody()
        : position(0,0,0), velocity(0,0,0), forceAccum(0,0,0),
          mass(1), inverseMass(1),
          angularVelocity(0,0,0), torqueAccum(0,0,0),
          linearDamping(0.99f), angularDamping(0.99f),
          isAwake(true), sleepTimer(0)
    {
        inertiaTensor = Mat3::identity();
        inverseInertia = Mat3::identity();
    }

    // ========== 积分方法 ==========

    // 半隐式欧拉积分（推荐）
    void integrate(float dt) {
        if (isStatic()) return;

        // 线性积分
        Vec3 acceleration = forceAccum * inverseMass;
        velocity = velocity * linearDamping + acceleration * dt;
        position = position + velocity * dt;

        // 旋转积分（完整的欧拉方程）
        // α = I^(-1) * (τ - ω × (I * ω))
        Vec3 inertiaEffect = inertiaTensor.transform(angularVelocity);
        Vec3 gyroscopicTorque = angularVelocity.cross(inertiaEffect);
        Vec3 effectiveTorque = torqueAccum - gyroscopicTorque;
        Vec3 angAccel = inverseInertia.transform(effectiveTorque);

        angularVelocity = angularVelocity * angularDamping + angAccel * dt;

        // 四元数积分: q(t+dt) = q(t) + 0.5 * dt * ω * q(t)
        Quaternion spin = {0, angularVelocity.x, angularVelocity.y, angularVelocity.z};
        orientation = orientation + (spin * orientation) * (dt * 0.5f);
        orientation = orientation.normalized();

        // 更新睡眠状态
        updateSleepState(dt);

        // 清除累加器
        forceAccum = {0, 0, 0};
        torqueAccum = {0, 0, 0};
    }

    // RK4积分（高精度，适合物理演示）
    struct State { Vec3 pos, vel; };

    void integrateRK4(float dt) {
        if (isStatic()) return;

        State s0 = {position, velocity};
        Vec3 accel = forceAccum * inverseMass;

        State k1 = {s0.vel, accel};
        State k2 = {s0.vel + k1.vel * (dt*0.5f), accel};
        State k3 = {s0.vel + k2.vel * (dt*0.5f), accel};
        State k4 = {s0.vel + k3.vel * dt, accel};

        position = s0.pos + (k1.vel + k2.vel*2 + k3.vel*2 + k4.vel) * (dt/6.0f);
        velocity = s0.vel + accel * dt; // 简化，实际应用中也应用RK4到速度
    }

    // ========== 力的施加 ==========

    void applyForce(Vec3 force) {
        if (isStatic()) return;
        forceAccum += force;
        wakeUp();
    }

    void applyForceAtPoint(Vec3 force, Vec3 worldPoint) {
        if (isStatic()) return;
        forceAccum += force;
        Vec3 r = worldPoint - position;
        torqueAccum += r.cross(force);
        wakeUp();
    }

    void applyImpulse(Vec3 impulse) {
        if (isStatic()) return;
        velocity += impulse * inverseMass;
        wakeUp();
    }

    void applyImpulseAtPoint(Vec3 impulse, Vec3 worldPoint) {
        if (isStatic()) return;
        velocity += impulse * inverseMass;
        Vec3 r = worldPoint - position;
        angularVelocity += inverseInertia.transform(r.cross(impulse));
        wakeUp();
    }

    void applyTorque(Vec3 torque) {
        if (isStatic()) return;
        torqueAccum += torque;
        wakeUp();
    }

    // ========== 惯性张量变换 ==========

    // 将体空间惯性张量变换到世界空间
    Mat3 getWorldInverseInertia() const {
        // I_world^(-1) = R * I_body^(-1) * R^T
        // 从四元数构建旋转矩阵
        float xx=orientation.x*orientation.x, yy=orientation.y*orientation.y;
        float zz=orientation.z*orientation.z;
        float xy=orientation.x*orientation.y, xz=orientation.x*orientation.z;
        float yz=orientation.y*orientation.z;
        float wx=orientation.w*orientation.x, wy=orientation.w*orientation.y;
        float wz=orientation.w*orientation.z;

        Mat3 R;
        R.m[0][0]=1-2*(yy+zz); R.m[1][0]=2*(xy-wz);   R.m[2][0]=2*(xz+wy);
        R.m[0][1]=2*(xy+wz);   R.m[1][1]=1-2*(xx+zz); R.m[2][1]=2*(yz-wx);
        R.m[0][2]=2*(xz-wy);   R.m[1][2]=2*(yz+wx);   R.m[2][2]=1-2*(xx+yy);

        return R * inverseInertia * R.transposed();
    }

    // ========== 睡眠管理 ==========

    void updateSleepState(float dt) {
        float speedSq = velocity.lengthSq();
        float angSpeedSq = angularVelocity.lengthSq();

        if (speedSq < SLEEP_THRESHOLD * SLEEP_THRESHOLD &&
            angSpeedSq < SLEEP_THRESHOLD * SLEEP_THRESHOLD) {
            sleepTimer += dt;
            if (sleepTimer > SLEEP_TIME) {
                isAwake = false;
                velocity = {0, 0, 0};
                angularVelocity = {0, 0, 0};
            }
        } else {
            sleepTimer = 0;
            isAwake = true;
        }
    }

    void wakeUp() {
        isAwake = true;
        sleepTimer = 0;
    }

    // ========== 动能计算 ==========

    float getKineticEnergy() const {
        float linearKE = 0.5f * mass * velocity.lengthSq();
        // 旋转动能: 0.5 * ω^T * I * ω
        Vec3 Iw = inertiaTensor.transform(angularVelocity);
        float angularKE = 0.5f * angularVelocity.dot(Iw);
        return linearKE + angularKE;
    }

    float getMomentum() const {
        return mass * velocity.length();
    }
};

// ========== 常见形状的惯性张量 ==========

// 长方体: 质量m，半尺寸hx, hy, hz
Mat3 boxInertia(float mass, float hx, float hy, float hz) {
    float m12 = mass / 12.0f;
    Mat3 I;
    I.m[0][0] = m12 * (hy*hy + hz*hz);
    I.m[1][1] = m12 * (hx*hx + hz*hz);
    I.m[2][2] = m12 * (hx*hx + hy*hy);
    return I;
}

// 球体: 质量m，半径r
Mat3 sphereInertia(float mass, float radius) {
    float I = 0.4f * mass * radius * radius;
    return Mat3::diagonal(I, I, I);
}

// 胶囊体: 质量m，半径r，半高h
Mat3 capsuleInertia(float mass, float radius, float halfHeight) {
    // 近似为圆柱+两个半球
    float cylinderMass = mass * (2 * halfHeight) / (2 * halfHeight + 4.0f/3.0f * radius);
    float sphereMass = mass - cylinderMass;

    float cylIxx = 0.25f * cylinderMass * radius * radius +
                   1.0f/12.0f * cylinderMass * (2*halfHeight) * (2*halfHeight);
    float cylIyy = 0.5f * cylinderMass * radius * radius;

    float sphIxx = 0.4f * sphereMass * radius * radius +
                   sphereMass * (halfHeight * halfHeight + 0.375f * radius * radius);

    Mat3 I;
    I.m[0][0] = cylIxx + sphIxx;
    I.m[1][1] = cylIyy + 0.4f * sphereMass * radius * radius;
    I.m[2][2] = I.m[0][0];
    return I;
}

// 圆柱体: 质量m，半径r，半高h
Mat3 cylinderInertia(float mass, float radius, float halfHeight) {
    Mat3 I;
    I.m[0][0] = 0.25f * mass * radius * radius +
                1.0f/12.0f * mass * (2*halfHeight) * (2*halfHeight);
    I.m[1][1] = 0.5f * mass * radius * radius;
    I.m[2][2] = I.m[0][0];
    return I;
}

public class RigidbodyController : MonoBehaviour

        // ========== 施加力 ==========
        rb.AddForce(Vector3.forward * 10f);                    // 持续力
        rb.AddForce(Vector3.up * 500f, ForceMode.Impulse);     // 瞬时冲量
        rb.AddForce(Vector3.right * 10f, ForceMode.VelocityChange); // 速度变化
        rb.AddForce(Vector3.up * 10f, ForceMode.Acceleration); // 加速度（忽略质量）

        // 在指定点施加力（产生力矩）
        rb.AddForceAtPosition(Vector3.up * 100f, transform.position + Vector3.right);

        // 相对质心施加力矩
        rb.AddTorque(Vector3.up * 10f);

        // ========== 直接设置速度 ==========
        rb.velocity = Vector3.forward * 5f;

        rb.centerOfMass = Vector3.zero; // 设置质心

        if (other.CompareTag("Pickup"))

## 游戏应用场景

1. **抛射物**：子弹、箭矢仅需线性积分（无旋转），或使用射线检测代替
2. **载具物理**：车辆的四轮分别施加力和力矩，需要正确的惯性张量
3. **布娃娃系统**：每个骨骼段作为刚体，通过关节约束连接
4. **可破坏物体**：物体碎裂后各碎片独立作为刚体运动
5. **投掷物**：球类、手雷等需要旋转的动力学模拟
6. **飞行器**：飞机/无人机的升力、阻力、推力模型

struct CarPhysics {
    RigidBody chassis;
    Vec3 wheelPositions[4]; // 四轮相对位置
    float wheelRadius;
    float engineForce;
    float steeringAngle;

    void update(float dt) {
        // 计算四轮的接地力
        for (int i = 0; i < 4; i++) {
            Vec3 wheelWorldPos = chassis.position +
                chassis.orientation.rotate(wheelPositions[i]);

            // Raycast向下检测地面
            float suspensionForce = calculateSuspension(wheelWorldPos);
            chassis.applyForceAtPoint(
                Vec3{0, suspensionForce, 0}, wheelWorldPos);

            // 驱动力（后轮）
            if (i >= 2) {
                Vec3 forward = chassis.orientation.rotate(Vec3{0,0,1});
                chassis.applyForceAtPoint(
                    forward * engineForce, wheelWorldPos);
            }

            // 转向力（前轮）
            if (i < 2) {
                // 简化的转向模型
                Vec3 right = chassis.orientation.rotate(Vec3{1,0,0});
                float lateralSpeed = chassis.velocity.dot(right);
                chassis.applyForceAtPoint(
                    right * (-lateralSpeed * 50.0f), wheelWorldPos);
            }
        }
    }
};

## 常见陷阱

- **积分方法**：显式欧拉法不稳定，推荐半隐式欧拉或Verlet积分
- **力累加器**：每帧必须清除，否则力无限累积导致物体飞出场景
- **惯性张量变换**：旋转后需要将体空间惯性张量变换到世界空间：`I_world = R * I_body * R^T`
- **阻尼系数**：过大的阻尼导致物体运动像在水中，过小则能量守恒导致不稳定振荡
- **dt变化**：物理时间步长必须固定，使用可变dt会导致不一致行为
- **质量比悬殊**：质量差距过大的物体碰撞可能导致数值不稳定
- **静态物体的inverseMass**：必须设为0而非极小值，否则仍会被力影响
- **四元数漂移**：连续积分后四元数可能偏离单位长度，需定期归一化
