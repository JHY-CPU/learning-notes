# 常见物理效果实现

## 核心概念

将物理原理应用于具体的游戏效果实现，包括抛射体、弹簧、力场和布娃娃系统。

### 抛射体运动

忽略空气阻力的弹道：抛物线运动
```
x(t) = x0 + v0×cos(θ)×t
y(t) = y0 + v0×sin(θ)×t - 0.5×g×t²
```

需要考虑空气阻力时，使用数值积分。

### 弹簧效果

胡克定律：`F = -k × (x - x_rest)` + 阻尼项 `F_d = -c × v`

弹簧连接两点，产生恢复力和阻尼力。可用于相机跟随、UI回弹、绳索物理。

### 力场区域

在特定空间区域施加持续的力：
- 风力区域：沿方向的持续推力
- 引力点：向中心的吸引力
- 排斥力：远离中心的推力

### Ragdoll布娃娃系统

角色死亡或受击时切换到物理驱动：骨骼段变为刚体，关节约束连接各段。动画系统混合到物理模拟。

## 代码示例

```cpp
// 抛射体物理（带空气阻力）
struct Projectile {
    Vec3 position, velocity;
    float drag = 0.01f;

    void update(float dt) {
        // 重力
        Vec3 gravity = {0, -9.8f, 0};
        // 空气阻力：与速度平方成正比
        Vec3 dragForce = -velocity * (drag * velocity.length());

        Vec3 acceleration = gravity + dragForce / mass;
        velocity += acceleration * dt;
        position += velocity * dt;
    }

    // 计算命中目标所需的发射角度
    static float calcLaunchAngle(Vec3 start, Vec3 target, float speed) {
        Vec3 diff = target - start;
        float dx = Vec2{diff.x, diff.z}.length();
        float dy = diff.y;
        float g = 9.8f;

        float v2 = speed * speed;
        float v4 = v2 * v2;
        float under = v4 - g * (g * dx * dx + 2 * dy * v2);

        if (under < 0) return -1; // 无法命中（超出射程）
        float angle = atan((v2 - sqrt(under)) / (g * dx));
        return angle;
    }
};

// 弹簧相机跟随
class SpringCamera {
    Vec3 position, velocity;
    Vec3 target;
    float stiffness = 80.0f;
    float damping = 12.0f;

public:
    void update(float dt) {
        Vec3 displacement = position - target;
        Vec3 springForce = displacement * (-stiffness);
        Vec3 dampForce = velocity * (-damping);
        Vec3 acceleration = springForce + dampForce;

        velocity += acceleration * dt;
        position += velocity * dt;
    }

    void setTarget(Vec3 t) { target = t; }
};

// 风力区域
class WindZone {
    AABB bounds;
    Vec3 windDirection;
    float windStrength;

    void applyWind(std::vector<RigidBody>& bodies) {
        for (auto& body : bodies) {
            if (bounds.contains(body.position)) {
                Vec3 force = windDirection * windStrength;
                // 简化：力的大小与物体表面积成正比
                body.applyForce(force * body.surfaceArea);
            }
        }
    }
};

// 引力点（如黑洞效果）
class GravityWell {
    Vec3 position;
    float strength;
    float radius;

    void applyForce(RigidBody& body) {
        Vec3 toCenter = position - body.position;
        float dist = toCenter.length();
        if (dist > radius || dist < 0.1f) return;

        // 与距离平方成反比的引力
        float forceMag = strength / (dist * dist);
        Vec3 force = toCenter.normalized() * forceMag;
        body.applyForce(force);
    }
};

// 布娃娃系统骨架
struct RagdollBone {
    RigidBody body;
    int parentIndex;
    Vec3 jointLocalA, jointLocalB; // 关节在两段骨骼上的局部位置
    ConstraintType jointType;      // BallSocket, Hinge等
};

class RagdollSystem {
    std::vector<RagdollBone> bones;
    std::vector<Constraint*> jointConstraints;

public:
    void initFromSkeleton(const Skeleton& skeleton) {
        for (auto& bone : skeleton.bones) {
            RagdollBone rb;
            // 根据骨骼长度和粗细创建胶囊体碰撞体
            float length = bone.length;
            float radius = bone.width * 0.5f;
            rb.body = createCapsuleBody(length, radius);
            rb.body.position = bone.worldPosition;
            rb.body.orientation = bone.worldRotation;
            rb.parentIndex = bone.parentIndex;
            bones.push_back(rb);
        }

        // 创建关节约束
        for (size_t i = 1; i < bones.size(); i++) {
            auto constraint = new BallSocketConstraint(
                bones[bones[i].parentIndex].body,
                bones[i].body,
                bones[i].jointLocalA,
                bones[i].jointLocalB
            );
            jointConstraints.push_back(constraint);
        }
    }

    // 将物理结果同步回骨骼（用于动画混合）
    void syncToSkeleton(Skeleton& skeleton, float blendFactor) {
        for (size_t i = 0; i < bones.size(); i++) {
            skeleton.bones[i].worldPosition = Vec3::lerp(
                skeleton.bones[i].animatedPosition,
                bones[i].body.position,
                blendFactor
            );
            skeleton.bones[i].worldRotation = Quaternion::slerp(
                skeleton.bones[i].animatedRotation,
                bones[i].body.orientation,
                blendFactor
            );
        }
    }

    void activate() {
        // 激活布娃娃：从动画状态初始化物理状态
        for (auto& b : bones) {
            b.body.velocity = {0, 0, 0};
            b.body.angularVelocity = {0, 0, 0};
        }
    }
};
```

## 游戏应用场景

1. **炮台瞄准**：抛射体公式计算发射角度，实现精确抛物线射击
2. **相机效果**：弹簧相机跟随、后坐力晃动、受击震动
3. **环境效果**：风区吹动旗帜和角色，重力区改变运动方向
4. **角色死亡**：切换到ragdoll物理模拟，产生自然的倒地效果
5. **抓钩系统**：弹簧连接角色与锚点，物理拉伸效果

## 常见陷阱

- **抛射体角度**：atan2有两个解，需根据游戏上下文选择（高弧/低弧）
- **弹簧震荡**：stiffness和damping比值不当导致持续震荡
- **布娃娃穿模**：骨骼穿透地板需要逐质点碰撞修正
- **力场叠加**：多个力场同时影响时力可能过大，需限制最大力
- **混合动画与物理**：blendFactor的过渡需要平滑插值，避免突变
