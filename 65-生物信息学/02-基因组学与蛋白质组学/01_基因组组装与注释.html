基因组组装与注释

生物信息学基因组组装

基因组组装将短序列片段拼接为完整基因组，基因注释识别其中的功能元件。

测序技术

测序技术发展： ┌──────────┬────────────┬────────────┬──────────┐ │ 代数 │ 读长 │ 准确率 │ 代表 │ ├──────────┼────────────┼────────────┼──────────┤ │ 一代(Sanger)│ ~1000bp │ 99.99% │ ABI 3730 │ │ 二代(NGS) │ 100-300bp│ 99.9% │ Illumina │ │ 三代(长读) │ 10-100kb │ 85-99% │ PacBio │ │ │ │ │ Oxford Nanopore│ └──────────┴────────────┴────────────┴──────────┘ Illumina (二代测序)： - 边合成边测序 - 短读长(150-300bp)，高通量 - 准确率高(>99.9%) - 读长限制导致重复区域难以组装 PacBio HiFi (三代测序)： - 单分子实时测序 - 长读长(10-25kb)，HiFi模式准确率>99% - 可跨越重复区域 - 成本较高 覆盖深度(Coverage)： C = N × L / G - N：读段数量 - L：读长 - G：基因组大小 - 通常30x-50x覆盖深度足够

从头组装 (De Novo Assembly)

组装策略： ┌─────────────────────────────────────────────┐ │ 1. Overlap-Layout-Consensus (OLC) │ │ 适用：长读长(三代) │ │ - Overlap：找读段间的重叠 │ │ - Layout：构建重叠图 │ │ - Consensus：生成一致序列 │ │ 工具：Canu, Flye, wtdbg2 │ │ │ │ 2. de Bruijn图方法 │ │ 适用：短读长(二代) │ │ - 将读段切分为k-mer │ │ - 构建de Bruijn图（节点=kmer，边=重叠k-1）│ │ - 寻找欧拉路径/汉密尔顿路径 │ │ 工具：SPAdes, Velvet, MEGAHIT │ │ │ │ 3. 混合组装 │ │ 结合二代(高准确)和三代(长读长)的优势 │ │ 工具：MaSuRCA, Unicycler │ └─────────────────────────────────────────────┘ 组装质量评估： - N50：50%的组装序列长度≥N50（越大越好） - Contig：连续的组装片段 - Scaffold：包含gap的contig连接 - BUSCO：保守基因的完整性评估

基因预测与注释

基因注释流程： 1. 重复序列屏蔽（RepeatMasker） 2. 从头基因预测（Ab initio） 3. 同源基因搜索（Homology-based） 4. RNA-seq证据整合 5. 功能注释 开放阅读框 (ORF)： - 起始密码子(ATG)到终止密码子(TAA/TAG/TGA) - 原核生物：通常>100bp的ORF - 真核生物：需要考虑外显子结构 基因预测工具： ┌──────────┬──────────────────────────────────┐ │ 工具 │ 方法 │ ├──────────┼──────────────────────────────────┤ │ GeneMark │ 隐马尔可夫模型(HMM) │ │ AUGUSTUS │ 基于证据的概率模型 │ │ Glimmer │ 插值马尔可夫模型(原核) │ │ Prodigal │ 原核基因预测 │ │ BRAKER │ 整合RNA-seq和蛋白质证据 │ └──────────┴──────────────────────────────────┘ 功能注释： - GO (Gene Ontology)：功能分类（分子功能、生物过程、细胞组分） - KEGG：代谢通路 - InterPro：蛋白质结构域和家族 - COG/KOG：直系同源基因分组 AlphaFold (蛋白质结构预测)： - DeepMind 2020年发布 - 从氨基酸序列预测3D结构 - 准确度接近实验方法 - 2024年诺贝尔化学奖
现代基因组项目通常使用三代测序(PacBio HiFi)进行组装，然后用RNA-seq和同源证据进行注释。
