蛋白质结构预测

生物信息学蛋白质结构预测

蛋白质结构预测是从氨基酸序列推断三维结构的计算方法，是结构生物学的核心问题。

蛋白质折叠问题

Anfinsen 假说（1973 诺贝尔化学奖）： ┌─────────────────────────────────────────────────────┐ │ 蛋白质的天然三维结构由其氨基酸序列唯一决定 │ │ 即：序列 → 结构（理论上可以从序列预测结构） │ └─────────────────────────────────────────────────────┘ Levinthal 悖论： - 一个 100 个残基的蛋白质 - 每个残基有 10 种构象 - 总构象数 10^100 - 如果随机搜索，宇宙年龄也不够 - 实际折叠在毫秒到秒级完成 - 蛋白质折叠不是随机搜索，而是沿着能量漏斗 蛋白质结构层级： ┌──────────────┬────────────────────────────────────┐ │ 一级结构 │ 氨基酸序列 │ ├──────────────┼────────────────────────────────────┤ │ 二级结构 │ α-螺旋、β-折叠、无规卷曲 │ ├──────────────┼────────────────────────────────────┤ │ 三级结构 │ 整条链的三维折叠 │ ├──────────────┼────────────────────────────────────┤ │ 四级结构 │ 多条链的组装（蛋白质复合物） │ └──────────────┴────────────────────────────────────┘ 实验方法 vs 计算方法： ┌──────────────┬──────────┬──────────┬──────────────┐ │ 方法 │ 分辨率 │ 速度 │ 限制 │ ├──────────────┼──────────┼──────────┼──────────────┤ │ X 射线晶体学 │ 高 │ 慢（年） │ 需要结晶 │ │ 冷冻电镜 │ 高 │ 中（月） │ 需要纯化样品 │ │ NMR │ 中 │ 中 │ 限于小蛋白 │ │ AlphaFold │ 高 │ 快（分） │ 依赖 MSA 质量 │ └──────────────┴──────────┴──────────┴──────────────┘

AlphaFold（深度学习革命）

AlphaFold 2 在 CASP14 中以压倒性优势解决了蛋白质结构预测问题，2024 年获诺贝尔化学奖。

AlphaFold 2 架构（Jumper et al., Nature 2021）： ┌─────────────────────────────────────────────────────┐ │ 输入：氨基酸序列 │ │ │ │ 1. MSA 搜索（多序列比对） │ │ JackHMMER / HHBlits 搜索 UniRef, BFD, MGNify │ │ 获取同源序列的 MSA 矩阵 │ │ │ │ 2. 特征提取 │ │ MSA representation：序列间进化关系 │ │ Pair representation：残基对关系 │ │ 结构模板（如果可用） │ │ │ │ 3. Evoformer（48 层） │ │ MSA 行注意力 + 列注意力 │ │ MSA → Pair 信息传递 │ │ Pair → Pair 自注意力 │ │ 迭代更新 MSA 和 Pair 表示 │ │ │ │ 4. Structure Module │ │ 使用不变点注意力（IPA） │ │ 输出 3D 坐标（每个残基的骨架原子） │ │ │ │ 5. 输出 │ │ 3D 坐标 + 置信度分数（pLDDT） │ │ 预测对齐误差（PAE）矩阵 │ └─────────────────────────────────────────────────────┘ 关键创新： - Evoformer：双向信息流动（MSA ↔ Pair） - 不变点注意力（IPA）：尊重旋转平移对称性 - 端到端训练：从序列直接到坐标 - 迭代回收（recycling）：多次精化预测 pLDDT 置信度： - > 90：非常高置信度 - 70-90：高置信度 - 50-70：低置信度 - < 50：可能是无序区域（IDR）
AlphaFold 3（2024）扩展到蛋白质-蛋白质、蛋白质-核酸、蛋白质-小分子复合物的结构预测，使用扩散模型替代结构模块。

同源建模（Homology Modeling）

同源建模利用已知结构的同源蛋白质作为模板来预测目标蛋白的结构。

同源建模流程： ┌─────────────────────────────────────────────────────┐ │ Step 1: 模板搜索 │ │ 用 BLAST/PSI-BLAST/HHpred 在 PDB 中搜索同源序列 │ │ 序列一致性 >30% 通常可获得好的模型 │ │ │ │ Step 2: 序列比对 │ │ 将目标序列与模板序列进行精确比对 │ │ 这是最关键的步骤，比对错误导致模型错误 │ │ │ │ Step 3: 骨架构建 │ │ 从模板结构复制保守区域的坐标 │ │ 可变区域使用片段数据库或构象搜索 │ │ │ │ Step 4: 环区建模 │ │ 插入/缺失区域（Loop）的构建 │ │ 使用数据库搜索或 ab initio 方法 │ │ │ │ Step 5: 侧链建模 │ │ 预测可变残基的侧链构象 │ │ 旋转异构体库 + 能量优化 │ │ │ │ Step 6: 模型优化与评估 │ │ 分子动力学松弛 │ │ 用 PROCHECK, MolProbity 等工具评估 │ └─────────────────────────────────────────────────────┘ 常用工具： - SWISS-MODEL：在线自动化同源建模服务器 - MODELLER：最广泛使用的同源建模程序 - Phyre2：隐马尔可夫模型辅助建模 - I-TASSER：综合多种方法的预测平台

从头预测（Ab Initio）与分子动力学

Ab Initio 预测： ┌─────────────────────────────────────────────────────┐ │ 不依赖已知结构模板，从物理原理出发 │ │ │ │ 能量函数方法： │ │ E_total = E_bond + E_angle + E_dihedral │ │ + E_vdW + E_elec + E_solvation │ │ │ │ Rosetta： │ │ - 片段组装策略 │ │ - 从片段库中组装初始结构 │ │ - 蒙特卡罗优化能量函数 │ │ - 对短蛋白（<100残基）效果较好 │ │ │ │ 困难： │ │ - 能量函数精度有限 │ │ - 构象空间极大 │ │ - 对长蛋白效果差 │ └─────────────────────────────────────────────────────┘ 分子动力学（MD）模拟： ┌─────────────────────────────────────────────────────┐ │ 原理：基于牛顿力学方程，模拟原子随时间的运动 │ │ │ │ F_i = -∇E(r_1,...,r_N) │ │ m_i * a_i = F_i │ │ │ │ 力场：AMBER, CHARMM, GROMOS, OPLS │ │ 时间步长：1-2 飞秒 │ │ 模拟时长：纳秒到毫秒（专用硬件可达毫秒级） │ │ │ │ 应用： │ │ - 蛋白质折叠路径研究 │ │ - 蛋白质-配体结合自由能计算 │ │ - 药物分子对接（Docking） │ │ - 酶催化机制研究 │ │ │ │ 软件：GROMACS, AMBER, NAMD, OpenMM │ └─────────────────────────────────────────────────────┘ 分子对接（Docking）： - 预测小分子配体与蛋白质受体的结合模式 - 打分函数评估结合亲和力 - 工具：AutoDock, Glide, GOLD - 应用：药物发现和虚拟筛选
AlphaFold 的出现极大地改变了蛋白质结构预测领域，但分子动力学在研究蛋白质动态行为、配体结合等方面仍然不可替代。
