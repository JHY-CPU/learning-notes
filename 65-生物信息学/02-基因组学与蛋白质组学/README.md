# 02-基因组学与蛋白质组学

本章涵盖从基因组测序到蛋白质功能分析的完整流程，是现代组学研究的核心内容。

---

## 基因组组装

### 从 reads 到 contigs 到 scaffolds

**reads** 是测序仪产生的原始 DNA 片段（Illumina 150-300bp，PacBio/Nanopore 数千到数万 bp）。**contigs** 通过重叠 reads 拼接而成的连续序列。**scaffolds** 利用配对末端 reads 的距离信息排列 contigs，中间可能有 gap。

组装分为从头组装（de novo）和参考引导组装两种策略。

### De Bruijn 图方法

Illumina 短读长组装的主流方法：
1. 将 reads 切割为 k-mer
2. 构建 De Bruijn 图（k-mer 为节点，(k-1)-mer 重叠为边）
3. 寻找欧拉路径
4. 处理重复序列和测序错误引起的分支

重复序列是组装的主要挑战。单倍体覆盖度是关键参数。

### 基因组组装工具

- **SPAdes**：多 k-mer 策略，内置纠错，支持混合组装，适用于细菌和宏基因组
- **Hifiasm**：PacBio HiFi 数据的主流组装器
- **Flye**：超长 Nanopore reads 的快速组装器
- **Canu**：经典长读长组装器，含纠错-修剪-组装三阶段

## 基因预测与注释

### 原核生物基因预测

基于 ORF 检测（起始密码子到终止密码子）、核糖体结合位点和密码子使用偏好。工具：Prodigal、Glimmer。

### 真核生物基因预测

真核基因含内含子-外显子结构，更为复杂：
- **从头预测**：基于 HMM/CRF（AUGUSTUS、GeneMark-ES）
- **同源性辅助**：BLASTX 比对蛋白质数据库
- **RNA-Seq 辅助**：StringTie 组装转录本
- **整合方法**：MAKER、BRAKER

### 功能注释

- 序列相似性搜索（BLAST → nr、UniProt）
- 结构域识别（InterProScan、Pfam）
- GO 注释（分子功能、生物过程、细胞组分）
- 通路注释（KEGG、Reactome）

---

## 比较基因组学

比较不同物种基因组揭示结构和进化规律：

- **共线性分析**：识别保守基因排列（MCScanX）
- **基因家族动态**：扩张与收缩分析（CAFE）
- **直系同源**：物种分化产生的功能对应基因（OrthoFinder）
- **泛基因组**：一个物种所有个体的基因全集，含核心基因组和附属基因组

## 基因表达分析

### 微阵列数据处理

背景校正 → 归一化（分位数/LOESS） → 探针汇总 → 差异表达分析（limma）。已被 RNA-Seq 大幅取代。

### RNA-Seq 数据处理

1. 质量控制（FastQC、Trimmomatic）
2. 比对到参考基因组（STAR、HISAT2）
3. 定量（featureCounts）
4. 归一化（TPM/RPKM）

无参考基因组时使用 Trinity 进行 de novo 转录组组装。

### 差异表达分析

- **DESeq2**：负二项分布模型，shrinkage 估计离散度，适合小样本
- **edgeR**：经验贝叶斯方法，支持 GLM 处理复杂设计

结果可视化：火山图（log₂FC vs -log₁₀p-value）、热图。结合 GO/KEGG 富集分析解释功能意义。

## 蛋白质结构

### 一级到四级结构

**一级**：氨基酸线性序列。**二级**：局部折叠模式（α 螺旋、β 折叠、转角）。**三级**：整条链的三维折叠（疏水作用、氢键、盐桥稳定）。**四级**：多亚基组装（如血红蛋白 α₂β₂）。

### 蛋白质结构预测

- **从头预测**：Rosetta，基于物理力场和统计势
- **同源建模**：SWISS-MODEL，序列相似性 > 30% 时可靠
- **AlphaFold2**（2020）：Evoformer 注意力机制处理 MSA 和残基关系，端到端训练，已预测超 2 亿个蛋白质结构
- **AlphaFold3**（2024）：扩展到蛋白质-核酸和蛋白质-小分子复合物
- **ESMFold**：蛋白质语言模型实现极快速单序列预测

## 蛋白质-蛋白质相互作用网络

### 实验检测

酵母双杂交（高通量但假阳性高）、免疫共沉淀-质谱（生理条件）、交联质谱（残基级接触）。

### 网络分析

无标度网络的度分布、Hub 蛋白质（往往是必需基因）、功能模块检测（MCODE）。数据库：STRING、BioGRID、IntAct。

## 药物设计与虚拟筛选

**基于结构**：获取靶标结构 → 结合位点识别（fpocket） → 分子对接（AutoDock Vina、Glide） → 打分评估 → 先导优化。

**基于配体**：药效团建模、QSAR 建模、相似性搜索。

**虚拟筛选流程**：化合物库构建 → 类药五规则过滤 → 对接排序 → ADMET 预测 → 分子动力学验证 → 实验验证。

**AI 驱动**：生成式模型（VAE、扩散模型）生成新分子，图神经网络预测性质，强化学习优化分子设计。

## 单细胞测序分析基础

### 技术平台

10x Genomics Chromium（液滴法，高通量）、Smart-seq2/3（板式，高灵敏度）、CITE-Seq（转录组+表面蛋白）。

### scRNA-Seq 分析流程

1. 原始数据处理（Cell Ranger）
2. 质量控制（过滤低质量细胞）
3. 归一化（SCTransform）
4. 特征选择（高变异基因 HVGs）
5. 降维（PCA → UMAP/t-SNE）
6. 聚类（Louvain/Leiden）
7. 细胞类型注释（基于 marker 基因或参考数据集）
8. 差异表达分析
9. 轨迹推断（RNA velocity、伪时间分析）

其他技术：scATAC-Seq（染色质可及性）、空间转录组学（Visium、Stereo-seq）。主要挑战：数据稀疏性、批次效应整合（Harmony、scVI）。
