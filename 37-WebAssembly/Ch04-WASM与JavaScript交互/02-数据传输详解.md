# 数据传输详解

## 一、概念说明

WASM 与 JavaScript 之间的数据传输包括基本类型、字符串、数组和复杂对象。

```javascript
// 基本类型直接传递
const result = instance.exports.add(1, 2);

// 复杂数据需要内存操作
const ptr = instance.exports.alloc(size);
const memory = new Uint8Array(instance.exports.memory.buffer);
memory.set(data, ptr);
```

## 二、具体用法

### 2.1 字符串传输

```javascript
// JavaScript 到 WASM
function stringToWasm(instance, str) {
  const bytes = new TextEncoder().encode(str);
  const ptr = instance.exports.alloc(bytes.length);
  const memory = new Uint8Array(instance.exports.memory.buffer);
  memory.set(bytes, ptr);
  return { ptr, len: bytes.length };
}

// WASM 到 JavaScript
function wasmToString(instance, ptr, len) {
  const memory = new Uint8Array(instance.exports.memory.buffer);
  const bytes = memory.slice(ptr, ptr + len);
  return new TextDecoder().decode(bytes);
}
```

### 2.2 数组传输

```javascript
// Float64Array 到 WASM
function float64ToWasm(instance, arr) {
  const ptr = instance.exports.alloc(arr.length * 8);
  const memory = new Float64Array(instance.exports.memory.buffer);
  memory.set(arr, ptr / 8);
  return ptr;
}

// WASM 到 Float64Array
function wasmToFloat64(instance, ptr, len) {
  const memory = new Float64Array(instance.exports.memory.buffer);
  return memory.slice(ptr / 8, ptr / 8 + len / 8);
}
```

### 2.3 对象序列化

```javascript
// JSON 序列化
function objectToWasm(instance, obj) {
  const json = JSON.stringify(obj);
  return stringToWasm(instance, json);
}

function wasmToObject(instance, ptr, len) {
  const json = wasmToString(instance, ptr, len);
  return JSON.parse(json);
}
```

### 2.4 批量数据处理

```javascript
// 批量处理减少边界开销
function processBatch(instance, data) {
  const memory = new Uint8Array(instance.exports.memory.buffer);
  const ptr = instance.exports.alloc(data.length);
  memory.set(data, ptr);
  instance.exports.process_batch(ptr, data.length);
  const result = memory.slice(ptr, ptr + data.length);
  instance.exports.dealloc(ptr, data.length);
  return result;
}
```

## 三、注意事项与常见陷阱

1. **内存对齐**：注意数据对齐
2. **字节序**：WASM 使用小端字节序
3. **内存释放**：释放分配的内存
4. **性能优化**：批量传输减少开销
5. **类型安全**：确保类型正确
