# 类型映射与转换

## 一、概念说明

Rust 类型与 JavaScript 类型之间需要通过 wasm-bindgen 进行自动或手动转换。

```rust
use wasm_bindgen::prelude::*;

// 自动转换：Rust 基本类型 ↔ JS 类型
#[wasm_bindgen]
pub fn add(a: i32, b: i32) -> i32 { a + b }     // number ↔ i32
#[wasm_bindgen]
pub fn greet(name: &str) -> String { ... }        // string ↔ &str/String
#[wasm_bindgen]
pub fn is_valid(v: bool) -> bool { v }            // boolean ↔ bool
```

## 二、具体用法

### 2.1 基本类型映射

```rust
// Rust → JavaScript
// i8/i16/i32/u8/u16/u32/f32/f64 → number
// i64/u64 → BigInt
// bool → boolean
// String/&str → string
// () → undefined
// JsValue → any

#[wasm_bindgen]
pub fn types_demo(n: i32, big: i64, flag: bool, text: &str) -> String {
    format!("{} {} {} {}", n, big, flag, text)
}
```

### 2.2 JsValue 动态类型

```rust
use wasm_bindgen::JsValue;

#[wasm_bindgen]
pub fn accept_any(value: JsValue) {
    if value.is_string() {
        let s = value.as_string().unwrap();
        web_sys::console::log_1(&format!("字符串: {}", s).into());
    } else if value.is_number() {
        let n = value.as_f64().unwrap();
        web_sys::console::log_1(&format!("数字: {}", n).into());
    } else if value.is_null() || value.is_undefined() {
        web_sys::console::log_1(&"空值".into());
    }
}
```

### 2.3 数组与对象

```rust
#[wasm_bindgen]
pub fn handle_array(arr: &js_sys::Array) -> js_sys::Array {
    let result = js_sys::Array::new();
    for i in 0..arr.length() {
        let val = arr.get(i);
        result.push(&JsValue::from_f64(val.as_f64().unwrap() * 2.0));
    }
    result
}

#[wasm_bindgen]
pub fn handle_object(obj: &js_sys::Object) -> js_sys::Object {
    let keys = js_sys::Object::keys(obj);
    web_sys::console::log_1(&keys.into());
    obj.clone()
}
```

### 2.4 自定义结构体导出

```rust
#[wasm_bindgen]
pub struct Point {
    x: f64,
    y: f64,
}

#[wasm_bindgen]
impl Point {
    #[wasm_bindgen(constructor)]
    pub fn new(x: f64, y: f64) -> Point {
        Point { x, y }
    }

    #[wasm_bindgen(getter)]
    pub fn x(&self) -> f64 { self.x }

    #[wasm_bindgen(getter)]
    pub fn y(&self) -> f64 { self.y }

    pub fn distance(&self, other: &Point) -> f64 {
        ((self.x - other.x).powi(2) + (self.y - other.y).powi(2)).sqrt()
    }
}
```

### 2.5 Result 与异常

```rust
#[wasm_bindgen]
pub fn may_fail(should_fail: bool) -> Result<String, JsValue> {
    if should_fail {
        Err(JsValue::from_str("操作失败"))
    } else {
        Ok("成功".to_string())
    }
}
// JavaScript 侧：try { may_fail(true) } catch(e) { /* e = "操作失败" */ }
```

## 三、注意事项与常见陷阱

1. **i64 限制**：Rust i64/u64 在 JS 中映射为 BigInt，不能直接与 number 运算
2. **浮点精度**：f32 传递时会被提升为 f64
3. **字符串开销**：每次字符串传递都需要复制和编码转换
4. **Option 类型**：Option<T> 映射为 T | undefined
5. **Vec<u8>**：映射为 Uint8Array，但默认会复制数据
