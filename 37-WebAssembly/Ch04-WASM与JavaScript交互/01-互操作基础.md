# 互操作基础

## 一、概念说明

WASM 与 JavaScript 通过导入/导出机制进行交互。

```javascript
// JavaScript 调用 WASM
const { instance } = await WebAssembly.instantiate(bytes, imports);
const result = instance.exports.add(1, 2);

// WASM 调用 JavaScript
(module
  (import "console" "log" (func $log (param i32)))
  (func $main
    i32.const 42
    call $log))
```

## 二、具体用法

### 2.1 数据传递

```javascript
// 传递数字
const result = instance.exports.add(1, 2);

// 传递字符串（需要内存操作）
const encoder = new TextEncoder();
const bytes = encoder.encode('Hello');
const ptr = instance.exports.alloc(bytes.length);
const memory = new Uint8Array(instance.exports.memory.buffer);
memory.set(bytes, ptr);
instance.exports.process(ptr, bytes.length);
```

### 2.2 类型转换

```javascript
// 自动转换
// JavaScript number → WASM i32/f64
// JavaScript BigInt → WASM i64
// JavaScript boolean → WASM i32

// 手动转换
function jsStringToWasm(instance, str) {
  const bytes = new TextEncoder().encode(str);
  const ptr = instance.exports.alloc(bytes.length + 1);
  const memory = new Uint8Array(instance.exports.memory.buffer);
  memory.set(bytes, ptr);
  memory[ptr + bytes.length] = 0; // null terminator
  return ptr;
}
```

### 2.3 回调函数

```javascript
// JavaScript 函数作为 WASM 回调
const imports = {
  env: {
    callback: (value) => {
      console.log('回调值:', value);
      return value * 2;
    }
  }
};

const { instance } = await WebAssembly.instantiate(bytes, imports);
instance.exports.process_with_callback(42);
```

### 2.4 内存共享

```javascript
// 共享内存
const memory = new WebAssembly.Memory({ initial: 1 });
const imports = {
  env: { memory: memory }
};

// 写入内存
const view = new Uint8Array(memory.buffer);
view[0] = 42;

// WASM 读取
const value = instance.exports.read_byte(0);
```

## 三、注意事项与常见陷阱

1. **类型安全**：确保类型正确转换
2. **内存管理**：正确管理共享内存
3. **性能开销**：互操作有性能开销
4. **错误处理**：处理互操作错误
5. **生命周期**：注意对象生命周期
