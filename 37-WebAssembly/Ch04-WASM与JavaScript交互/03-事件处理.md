# 事件处理

## 一、概念说明

WASM 处理 JavaScript 事件需要使用闭包和回调机制。

```rust
use wasm_bindgen::prelude::*;
use web_sys::EventTarget;

#[wasm_bindgen]
pub fn add_click_handler(element: &EventTarget) {
    let closure = Closure::wrap(Box::new(move || {
        console::log_1(&"点击事件触发".into());
    }) as Box<dyn Fn()>);

    element.add_event_listener_with_callback("click", closure.as_ref().unchecked_ref()).unwrap();
    closure.forget();
}
```

## 二、具体用法

### 2.1 基本事件

```rust
use wasm_bindgen::prelude::*;
use web_sys::{HtmlElement, MouseEvent};

#[wasm_bindgen]
pub fn setup_mouse_events(element: &HtmlElement) {
    let onmousedown = Closure::wrap(Box::new(move |event: MouseEvent| {
        console::log_1(&format!("鼠标按下: ({}, {})", event.client_x(), event.client_y()).into());
    }) as Box<dyn FnMut(MouseEvent)>);

    element.add_event_listener_with_callback("mousedown", onmousedown.as_ref().unchecked_ref()).unwrap();
    onmousedown.forget();
}
```

### 2.2 键盘事件

```rust
use web_sys::KeyboardEvent;

#[wasm_bindgen]
pub fn setup_keyboard_events() {
    let window = web_sys::window().unwrap();

    let onkeydown = Closure::wrap(Box::new(move |event: KeyboardEvent| {
        let key = event.key();
        console::log_1(&format!("按键: {}", key).into());
    }) as Box<dyn FnMut(KeyboardEvent)>);

    window.add_event_listener_with_callback("keydown", onkeydown.as_ref().unchecked_ref()).unwrap();
    onkeydown.forget();
}
```

### 2.3 触摸事件

```rust
use web_sys::TouchEvent;

#[wasm_bindgen]
pub fn setup_touch_events(element: &HtmlElement) {
    let ontouchstart = Closure::wrap(Box::new(move |event: TouchEvent| {
        let touches = event.touches();
        if touches.length() > 0 {
            let touch = touches.get(0).unwrap();
            console::log_1(&format!("触摸: ({}, {})", touch.client_x(), touch.client_y()).into());
        }
    }) as Box<dyn FnMut(TouchEvent)>);

    element.add_event_listener_with_callback("touchstart", ontouchstart.as_ref().unchecked_ref()).unwrap();
    ontouchstart.forget();
}
```

### 2.4 自定义事件

```rust
use web_sys::CustomEvent;

#[wasm_bindgen]
pub fn dispatch_custom_event(element: &HtmlElement, detail: &str) {
    let event = CustomEvent::new_with_event_init_dict(
        "custom-event",
        CustomEventInit::new().detail(&JsValue::from_str(detail)),
    ).unwrap();

    element.dispatch_event(&event).unwrap();
}
```

## 三、注意事项与常见陷阱

1. **闭包生命周期**：使用 forget 防止闭包被回收
2. **事件冒泡**：注意事件冒泡和捕获
3. **性能**：事件处理有性能开销
4. **内存泄漏**：正确清理事件监听器
5. **跨浏览器**：注意浏览器兼容性
