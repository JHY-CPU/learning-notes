# 音视频编码

## 一、概念说明

WASM 可移植 C/C++ 编解码库（如 FFmpeg、libvpx）到浏览器，实现客户端音视频处理。

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub struct AudioEncoder {
    sample_rate: u32,
    channels: u32,
    buffer: Vec<f32>,
}

#[wasm_bindgen]
impl AudioEncoder {
    #[wasm_bindgen(constructor)]
    pub fn new(sample_rate: u32, channels: u32) -> AudioEncoder {
        AudioEncoder {
            sample_rate,
            channels,
            buffer: Vec::with_capacity(sample_rate as usize * channels as usize),
        }
    }

    pub fn encode(&mut self, samples: &[f32]) -> Vec<u8> {
        self.buffer.extend_from_slice(samples);
        // 编码逻辑
        vec![]
    }
}
```

## 二、具体用法

### 2.1 音频处理

```rust
#[wasm_bindgen]
pub fn resample(input: &[f32], from_rate: u32, to_rate: u32) -> Vec<f32> {
    let ratio = from_rate as f64 / to_rate as f64;
    let output_len = (input.len() as f64 / ratio) as usize;
    let mut output = Vec::with_capacity(output_len);

    for i in 0..output_len {
        let src_pos = i as f64 * ratio;
        let idx = src_pos as usize;
        let frac = src_pos - idx as f64;

        if idx + 1 < input.len() {
            // 线性插值
            let sample = input[idx] as f64 * (1.0 - frac) + input[idx + 1] as f64 * frac;
            output.push(sample as f32);
        }
    }
    output
}

#[wasm_bindgen]
pub fn apply_volume(samples: &mut [f32], volume: f32) {
    for sample in samples.iter_mut() {
        *sample *= volume;
    }
}
```

### 2.2 WAV 编码

```rust
#[wasm_bindgen]
pub fn encode_wav(samples: &[f32], sample_rate: u32) -> Vec<u8> {
    let mut output = Vec::new();

    // RIFF 头
    output.extend_from_slice(b"RIFF");
    let data_size = samples.len() as u32 * 2; // 16-bit
    output.extend_from_slice(&(36 + data_size).to_le_bytes());
    output.extend_from_slice(b"WAVE");

    // fmt 块
    output.extend_from_slice(b"fmt ");
    output.extend_from_slice(&16u32.to_le_bytes()); // 块大小
    output.extend_from_slice(&1u16.to_le_bytes());  // PCM
    output.extend_from_slice(&1u16.to_le_bytes());  // 单声道
    output.extend_from_slice(&sample_rate.to_le_bytes());
    output.extend_from_slice(&(sample_rate * 2).to_le_bytes()); // 字节率
    output.extend_from_slice(&2u16.to_le_bytes());  // 块对齐
    output.extend_from_slice(&16u16.to_le_bytes()); // 位深

    // data 块
    output.extend_from_slice(b"data");
    output.extend_from_slice(&data_size.to_le_bytes());
    for &sample in samples {
        let pcm = (sample.clamp(-1.0, 1.0) * 32767.0) as i16;
        output.extend_from_slice(&pcm.to_le_bytes());
    }

    output
}
```

### 2.3 视频帧处理

```rust
#[wasm_bindgen]
pub fn process_frame(
    input: &[u8],
    output: &mut [u8],
    width: u32,
    height: u32,
    brightness: f32,
) {
    for i in (0..input.len()).step_by(4) {
        output[i] = (input[i] as f32 * brightness).min(255.0) as u8;
        output[i+1] = (input[i+1] as f32 * brightness).min(255.0) as u8;
        output[i+2] = (input[i+2] as f32 * brightness).min(255.0) as u8;
        output[i+3] = input[i+3]; // alpha 不变
    }
}
```

### 2.4 Web Audio 集成

```javascript
async function createWasmAudioProcessor(wasmModule) {
    const audioCtx = new AudioContext();
    const processor = audioCtx.createScriptProcessor(4096, 1, 1);

    const wasmPtr = wasmModule.exports.alloc(4096 * 4);

    processor.onaudioprocess = (e) => {
        const input = e.inputBuffer.getChannelData(0);
        const output = e.outputBuffer.getChannelData(0);

        // 写入 WASM 内存
        const wasmMemory = new Float32Array(
            wasmModule.exports.memory.buffer, wasmPtr, 4096
        );
        wasmMemory.set(input);

        // WASM 处理
        wasmModule.exports.process_audio(wasmPtr, 4096);

        // 读回
        output.set(wasmMemory);
    };

    return processor;
}
```

## 三、注意事项与常见陷阱

1. **实时性要求**：音频处理需要在缓冲区内完成，避免卡顿
2. **编解码许可**：某些编解码器有专利限制（如 H.264/AAC）
3. **CPU 占用**：视频编解码非常消耗 CPU，建议用 Worker
4. **WASM 文件大小**：嵌入 FFmpeg 会使 WASM 体积很大
5. **浏览器 API 优先**：WebCodecs API 已可用，优先使用原生 API
