# 数据分析

## 一、概念说明

WASM 可在浏览器中进行大数据集处理、CSV 解析、统计分析，无需后端服务。

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn parse_csv(data: &str) -> JsValue {
    let mut reader = csv::Reader::from_reader(data.as_bytes());
    let headers: Vec<String> = reader.headers()
        .unwrap()
        .iter()
        .map(|s| s.to_string())
        .collect();

    let mut records = Vec::new();
    for result in reader.records() {
        let record = result.unwrap();
        let row: serde_json::Map<String, serde_json::Value> = headers.iter()
            .zip(record.iter())
            .map(|(h, v)| (h.clone(), serde_json::Value::String(v.to_string())))
            .collect();
        records.push(serde_json::Value::Object(row));
    }

    serde_wasm_bindgen::to_value(&records).unwrap()
}
```

## 二、具体用法

### 2.1 数据聚合

```rust
use std::collections::HashMap;

#[wasm_bindgen]
pub fn group_by_sum(data: &[f64], groups: &[u32]) -> JsValue {
    let mut sums: HashMap<u32, f64> = HashMap::new();

    for (i, &value) in data.iter().enumerate() {
        let group = groups[i];
        *sums.entry(group).or_insert(0.0) += value;
    }

    serde_wasm_bindgen::to_value(&sums).unwrap()
}

#[wasm_bindgen]
pub fn rolling_average(data: &[f64], window: usize) -> Vec<f64> {
    if data.len() < window {
        return vec![];
    }

    let mut result = Vec::with_capacity(data.len() - window + 1);
    let mut sum: f64 = data[..window].iter().sum();
    result.push(sum / window as f64);

    for i in window..data.len() {
        sum += data[i] - data[i - window];
        result.push(sum / window as f64);
    }

    result
}
```

### 2.2 数据过滤与转换

```rust
#[wasm_bindgen]
pub fn filter_and_transform(
    data: &[f64],
    min_threshold: f64,
    max_threshold: f64,
    transform: &str,
) -> Vec<f64> {
    data.iter()
        .filter(|&&x| x >= min_threshold && x <= max_threshold)
        .map(|&x| match transform {
            "log" => x.ln(),
            "sqrt" => x.sqrt(),
            "square" => x * x,
            "normalize" => x, // 需要额外归一化
            _ => x,
        })
        .collect()
}
```

### 2.3 相关性分析

```rust
#[wasm_bindgen]
pub fn pearson_correlation(x: &[f64], y: &[f64]) -> f64 {
    assert_eq!(x.len(), y.len());
    let n = x.len() as f64;

    let mean_x = x.iter().sum::<f64>() / n;
    let mean_y = y.iter().sum::<f64>() / n;

    let mut cov = 0.0;
    let mut var_x = 0.0;
    let mut var_y = 0.0;

    for i in 0..x.len() {
        let dx = x[i] - mean_x;
        let dy = y[i] - mean_y;
        cov += dx * dy;
        var_x += dx * dx;
        var_y += dy * dy;
    }

    cov / (var_x * var_y).sqrt()
}
```

### 2.4 大数据分块处理

```rust
#[wasm_bindgen]
pub struct DataProcessor {
    chunk_size: usize,
    buffer: Vec<f64>,
    results: Vec<f64>,
}

#[wasm_bindgen]
impl DataProcessor {
    #[wasm_bindgen(constructor)]
    pub fn new(chunk_size: usize) -> DataProcessor {
        DataProcessor {
            chunk_size,
            buffer: Vec::with_capacity(chunk_size),
            results: Vec::new(),
        }
    }

    pub fn feed(&mut self, data: &[f64]) {
        self.buffer.extend_from_slice(data);
        while self.buffer.len() >= self.chunk_size {
            let chunk: Vec<f64> = self.buffer.drain(..self.chunk_size).collect();
            let result = chunk.iter().sum::<f64>() / chunk.len() as f64;
            self.results.push(result);
        }
    }

    pub fn flush(&mut self) -> Vec<f64> {
        std::mem::take(&mut self.results)
    }
}
```

### 2.5 可视化数据准备

```rust
#[wasm_bindgen]
pub fn prepare_histogram(data: &[f64], bins: usize) -> JsValue {
    let min = data.iter().copied().fold(f64::INFINITY, f64::min);
    let max = data.iter().copied().fold(f64::NEG_INFINITY, f64::max);
    let step = (max - min) / bins as f64;

    let mut counts = vec![0u32; bins];
    for &value in data {
        let bin = ((value - min) / step).floor() as usize;
        let bin = bin.min(bins - 1);
        counts[bin] += 1;
    }

    let edges: Vec<f64> = (0..=bins).map(|i| min + i as f64 * step).collect();
    serde_wasm_bindgen::to_value(&serde_json::json!({
        "counts": counts,
        "edges": edges,
    })).unwrap()
}
```

## 三、注意事项与常见陷阱

1. **数据大小**：超大数据集应分块处理，避免阻塞主线程
2. **类型推断**：CSV 解析时注意数字/字符串类型推断
3. **缺失值**：数据分析中需处理 NaN 和空值
4. **内存管理**：处理完数据后释放 WASM 内存
5. **Web Worker**：大量计算应在 Worker 中进行
