# 科学计算

## 一、概念说明

WASM 适合科学计算：矩阵运算、数值模拟、数据分析等 CPU 密集型任务。

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn matrix_multiply(a: &[f64], b: &[f64], n: usize) -> Vec<f64> {
    let mut result = vec![0.0; n * n];
    for i in 0..n {
        for k in 0..n {
            let a_ik = a[i * n + k];
            for j in 0..n {
                result[i * n + j] += a_ik * b[k * n + j];
            }
        }
    }
    result
}
```

## 二、具体用法

### 2.1 向量运算

```rust
#[wasm_bindgen]
pub struct Vector {
    data: Vec<f64>,
}

#[wasm_bindgen]
impl Vector {
    #[wasm_bindgen(constructor)]
    pub fn new(data: &[f64]) -> Vector {
        Vector { data: data.to_vec() }
    }

    pub fn dot(&self, other: &Vector) -> f64 {
        self.data.iter().zip(&other.data).map(|(a, b)| a * b).sum()
    }

    pub fn norm(&self) -> f64 {
        self.data.iter().map(|x| x * x).sum::<f64>().sqrt()
    }

    pub fn normalize(&self) -> Vector {
        let n = self.norm();
        Vector { data: self.data.iter().map(|x| x / n).collect() }
    }

    pub fn add(&self, other: &Vector) -> Vector {
        Vector {
            data: self.data.iter().zip(&other.data).map(|(a, b)| a + b).collect(),
        }
    }
}
```

### 2.2 FFT 变换

```rust
#[wasm_bindgen]
pub fn fft_real(input: &[f64]) -> Vec<f64> {
    let n = input.len();
    let mut real = input.to_vec();
    let mut imag = vec![0.0f64; n];

    // Cooley-Tukey FFT
    let mut j = 0;
    for i in 0..n {
        if i < j {
            real.swap(i, j);
        }
        let mut m = n >> 1;
        while m >= 1 && j >= m {
            j -= m;
            m >>= 1;
        }
        j += m;
    }

    let mut size = 2;
    while size <= n {
        let half = size / 2;
        let angle = -2.0 * std::f64::consts::PI / size as f64;
        let w_real = angle.cos();
        let w_imag = angle.sin();

        for i in (0..n).step_by(size) {
            let mut cur_real = 1.0;
            let mut cur_imag = 0.0;
            for j in 0..half {
                let t_real = cur_real * real[i+j+half] - cur_imag * imag[i+j+half];
                let t_imag = cur_real * imag[i+j+half] + cur_imag * real[i+j+half];
                real[i+j+half] = real[i+j] - t_real;
                imag[i+j+half] = imag[i+j] - t_imag;
                real[i+j] += t_real;
                imag[i+j] += t_imag;
                let tmp = cur_real * w_real - cur_imag * w_imag;
                cur_imag = cur_real * w_imag + cur_imag * w_real;
                cur_real = tmp;
            }
        }
        size *= 2;
    }

    // 返回幅度
    real.iter().zip(&imag).map(|(r, i)| (r*r + i*i).sqrt()).collect()
}
```

### 2.3 数值积分

```rust
#[wasm_bindgen]
pub fn simpson_integration(f: &js_sys::Function, a: f64, b: f64, n: usize) -> f64 {
    let h = (b - a) / n as f64;
    let mut sum = eval_f(f, a) + eval_f(f, b);

    for i in 1..n {
        let x = a + i as f64 * h;
        let fx = eval_f(f, x);
        sum += if i % 2 == 0 { 2.0 * fx } else { 4.0 * fx };
    }

    sum * h / 3.0
}

fn eval_f(f: &js_sys::Function, x: f64) -> f64 {
    f.call1(&JsValue::null(), &JsValue::from_f64(x))
        .unwrap()
        .as_f64()
        .unwrap()
}
```

### 2.4 统计分析

```rust
#[wasm_bindgen]
pub fn statistics(data: &[f64]) -> JsValue {
    let n = data.len() as f64;
    let mean = data.iter().sum::<f64>() / n;
    let variance = data.iter().map(|x| (x - mean).powi(2)).sum::<f64>() / n;
    let std_dev = variance.sqrt();

    let mut sorted = data.to_vec();
    sorted.sort_by(|a, b| a.partial_cmp(b).unwrap());

    let result = js_sys::Object::new();
    js_sys::Reflect::set(&result, &"mean".into(), &mean.into()).unwrap();
    js_sys::Reflect::set(&result, &"variance".into(), &variance.into()).unwrap();
    js_sys::Reflect::set(&result, &"stdDev".into(), &std_dev.into()).unwrap();
    js_sys::Reflect::set(&result, &"median".into(), &sorted[n as usize / 2].into()).unwrap();
    js_sys::Reflect::set(&result, &"min".into(), &sorted[0].into()).unwrap();
    js_sys::Reflect::set(&result, &"max".into(), &sorted[sorted.len()-1].into()).unwrap();
    result.into()
}
```

## 三、注意事项与常见陷阱

1. **数值精度**：f64 精度足够大部分科学计算，但需注意累加误差
2. **内存限制**：大矩阵运算受 WASM 内存限制
3. **SIMD 加速**：矩阵/向量运算可显著受益于 SIMD
4. **并行计算**：矩阵乘法等可分解为并行任务
5. **BLAS 库**：考虑使用 wasm-numeric 等 BLAS 绑定库
