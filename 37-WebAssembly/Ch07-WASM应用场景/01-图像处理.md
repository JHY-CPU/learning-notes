# 图像处理

## 一、概念说明

WASM 适合图像处理场景：计算密集、需要 SIMD 加速、数据密集型操作。

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn apply_brightness(pixels: &mut [u8], adjustment: i16) {
    for pixel in pixels.chunks_exact_mut(4) {
        // RGBA 处理
        for i in 0..3 {
            let val = pixel[i] as i16 + adjustment;
            pixel[i] = val.clamp(0, 255) as u8;
        }
    }
}
```

### 2.1 Canvas 像素处理

```javascript
async function processCanvas(canvas, wasmModule) {
    const ctx = canvas.getContext('2d');
    const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);

    // 将像素数据传入 WASM
    const ptr = wasmModule.exports.alloc(imageData.data.length);
    const wasmMemory = new Uint8Array(wasmModule.exports.memory.buffer);
    wasmMemory.set(imageData.data, ptr);

    // WASM 处理
    wasmModule.exports.apply_grayscale(ptr, imageData.data.length);

    // 读回结果
    imageData.data.set(wasmMemory.subarray(ptr, ptr + imageData.data.length));
    ctx.putImageData(imageData, 0, 0);

    wasmModule.exports.dealloc(ptr, imageData.data.length);
}
```

### 2.2 滤镜效果

```rust
#[wasm_bindgen]
pub fn apply_sepia(pixels: &mut [u8]) {
    for pixel in pixels.chunks_exact_mut(4) {
        let r = pixel[0] as f32;
        let g = pixel[1] as f32;
        let b = pixel[2] as f32;

        pixel[0] = (r * 0.393 + g * 0.769 + b * 0.189).min(255.0) as u8;
        pixel[1] = (r * 0.349 + g * 0.686 + b * 0.168).min(255.0) as u8;
        pixel[2] = (r * 0.272 + g * 0.534 + b * 0.131).min(255.0) as u8;
    }
}

#[wasm_bindgen]
pub fn apply_sepia_inplace(ptr: *mut u8, len: usize) {
    let pixels = unsafe { std::slice::from_raw_parts_mut(ptr, len) };
    apply_sepia(pixels);
}
```

### 2.3 高斯模糊

```rust
#[wasm_bindgen]
pub fn gaussian_blur(pixels: &mut [u8], width: usize, height: usize, radius: usize) {
    let kernel = gaussian_kernel(radius);
    let mut temp = pixels.to_vec();

    // 水平模糊
    for y in 0..height {
        for x in 0..width {
            let mut r = 0.0f32;
            let mut g = 0.0f32;
            let mut b = 0.0f32;
            for kx in 0..kernel.len() {
                let sx = (x as isize + kx as isize - radius as isize)
                    .clamp(0, width as isize - 1) as usize;
                let idx = (y * width + sx) * 4;
                r += pixels[idx] as f32 * kernel[kx];
                g += pixels[idx + 1] as f32 * kernel[kx];
                b += pixels[idx + 2] as f32 * kernel[kx];
            }
            let idx = (y * width + x) * 4;
            temp[idx] = r as u8;
            temp[idx + 1] = g as u8;
            temp[idx + 2] = b as u8;
        }
    }
    pixels.copy_from_slice(&temp);
}

fn gaussian_kernel(radius: usize) -> Vec<f32> {
    let size = radius * 2 + 1;
    let sigma = radius as f32 / 3.0;
    let mut kernel = vec![0.0f32; size];
    let mut sum = 0.0f32;
    for i in 0..size {
        let x = (i as f32 - radius as f32).abs();
        kernel[i] = (-x * x / (2.0 * sigma * sigma)).exp();
        sum += kernel[i];
    }
    for k in kernel.iter_mut() { *k /= sum; }
    kernel
}
```

### 2.4 边缘检测

```rust
#[wasm_bindgen]
pub fn sobel_edge_detection(input: &[u8], output: &mut [u8], width: usize, height: usize) {
    let gx = [-1, 0, 1, -2, 0, 2, -1, 0, 1];
    let gy = [-1, -2, -1, 0, 0, 0, 1, 2, 1];

    for y in 1..height-1 {
        for x in 1..width-1 {
            let mut sum_x = 0i32;
            let mut sum_y = 0i32;
            for ky in 0..3 {
                for kx in 0..3 {
                    let idx = ((y + ky - 1) * width + x + kx - 1) * 4;
                    let gray = (input[idx] as f32 * 0.299
                        + input[idx+1] as f32 * 0.587
                        + input[idx+2] as f32 * 0.114) as i32;
                    let ki = ky * 3 + kx;
                    sum_x += gray * gx[ki];
                    sum_y += gray * gy[ki];
                }
            }
            let magnitude = ((sum_x * sum_x + sum_y * sum_y) as f32).sqrt() as u8;
            let idx = (y * width + x) * 4;
            output[idx] = magnitude;
            output[idx+1] = magnitude;
            output[idx+2] = magnitude;
            output[idx+3] = 255;
        }
    }
}
```

## 三、注意事项与常见陷阱

1. **像素格式**：Canvas 默认 RGBA，注意 alpha 通道处理
2. **边界处理**：滤镜操作需要处理图像边界像素
3. **内存传递**：大图像在 WASM-JS 间传递有开销
4. **SIMD 加速**：像素操作可受益于 SIMD 指令
5. **Web Worker**：大图像处理应在 Worker 中进行
