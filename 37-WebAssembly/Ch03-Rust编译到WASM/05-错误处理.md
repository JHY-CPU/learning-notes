# 错误处理

## 一、概念说明

Rust WASM 中的错误处理需要将 Rust Result 转换为 JavaScript 异常。

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn divide(a: f64, b: f64) -> Result<f64, JsValue> {
    if b == 0.0 {
        Err(JsValue::from_str("除数不能为零"))
    } else {
        Ok(a / b)
    }
}
```

## 二、具体用法

### 2.1 基本错误处理

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn parse_number(s: &str) -> Result<f64, JsValue> {
    s.parse::<f64>()
        .map_err(|e| JsValue::from_str(&e.to_string()))
}

#[wasm_bindgen]
pub fn validate_input(input: &str) -> Result<String, JsValue> {
    if input.is_empty() {
        return Err(JsValue::from_str("输入不能为空"));
    }
    Ok(input.to_uppercase())
}
```

### 2.2 自定义错误类型

```rust
use wasm_bindgen::prelude::*;
use std::fmt;

#[derive(Debug)]
struct AppError {
    message: String,
    code: u32,
}

impl fmt::Display for AppError {
    fn fmt(&self, f: &mut fmt::Formatter) -> fmt::Result {
        write!(f, "错误 {}: {}", self.code, self.message)
    }
}

impl From<AppError> for JsValue {
    fn from(error: AppError) -> JsValue {
        JsValue::from_str(&error.to_string())
    }
}

#[wasm_bindgen]
pub fn process_with_error() -> Result<String, JsValue> {
    Err(AppError {
        message: "处理失败".to_string(),
        code: 500,
    }.into())
}
```

### 2.3 JavaScript 捕获

```javascript
import init, { divide } from './pkg/my_lib.js';

async function main() {
  await init();

  try {
    const result = divide(10, 0);
    console.log(result);
  } catch (error) {
    console.error('错误:', error);
  }
}

main();
```

### 2.4 Result 链式处理

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn chain_operations(input: &str) -> Result<String, JsValue> {
    let parsed = parse_number(input)?;
    let doubled = multiply(parsed, 2.0)?;
    let formatted = format_result(doubled)?;
    Ok(formatted)
}

fn multiply(x: f64, y: f64) -> Result<f64, JsValue> {
    Ok(x * y)
}

fn format_result(x: f64) -> Result<String, JsValue> {
    Ok(format!("结果: {:.2}", x))
}
```

## 三、注意事项与常见陷阱

1. **类型转换**：Rust 错误转换为 JsValue
2. **错误信息**：提供有用的错误信息
3. **错误码**：考虑使用错误码
4. **日志记录**：记录错误日志
5. **用户友好**：用户友好的错误消息
