## 二、具体用法

### 2.1 编译问题

```bash

# 解决：添加目标

# 问题：链接错误
# 解决：检查 crate-type
[lib]

# 问题：类型不支持 wasm-bindgen
# 解决：使用 JsValue 包装
#[wasm_bindgen]

    value
}
```

## 六、所有权和生命周期问题

```rust
// 问题：不能在 wasm-bindgen 函数中使用引用返回
// 错误：
#[wasm_bindgen]
pub fn get_string() -> &str {
    "hello"  // 编译错误：生命周期问题
}

// 解决方案 1：返回 String
#[wasm_bindgen]
pub fn get_string() -> String {
    "hello".to_string()
}

// 解决方案 2：返回 JsValue
#[wasm_bindgen]
pub fn get_string() -> JsValue {
    JsValue::from_str("hello")
}

// 问题：结构体字段不能有引用类型
// 错误：
// struct MyStruct<'a> {
//     data: &'a str,  // wasm-bindgen 不支持
// }

// 解决方案：使用 owned 类型
#[wasm_bindgen]
pub struct MyStruct {
    data: String,
}
```

## 七、Web API 兼容性问题

```rust
use wasm_bindgen::prelude::*;
use web_sys::{console, window, Window};

// 问题：某些 Web API 在 Worker 中不可用
// 解决：运行时检测
#[wasm_bindgen]
pub fn safe_dom_operation() -> Result<(), JsValue> {
    let window = window().ok_or_else(|| JsValue::from_str("No window"))?;
    let document = window
        .document()
        .ok_or_else(|| JsValue::from_str("No document"))?;

    // 检测是否在 Worker 中运行
    if document.body().is_none() {
        console::warn_1(&"Running in Worker, skipping DOM ops".into());
        return Ok(());
    }

    // 安全地操作 DOM
    let body = document.body().unwrap();
    body.set_inner_html("<p>Hello</p>");
    Ok(())
}
```

## 八、编译警告和 Clippy

```bash
# 常见 Clippy 警告修复

# 问题：unused import
# 解决：添加条件编译
#[cfg(target_arch = "wasm32")]
use wasm_bindgen::prelude::*;

# 问题：dead_code
# 解决：标记为 pub 或添加 #[allow(dead_code)]
#[allow(dead_code)]
fn internal_helper() { }

# 问题：needless_return
# 解决：移除 return 关键字
fn add(a: i32, b: i32) -> i32 {
    a + b  // 不需要 return
}

# 运行 clippy
cargo clippy --target wasm32-unknown-unknown -- -D warnings
```

## 十、性能相关问题

```rust
// 问题：频繁的 JS ↔ WASM 边界调用很慢
// 解决：批量处理

// 不好：
#[wasm_bindgen]
pub fn process_one(x: i32) -> i32 {
    x * 2
}
// JS 端：for (let i = 0; i < 10000; i++) result.push(process_one(i));

// 好：批量处理
#[wasm_bindgen]
pub fn process_batch(data: &[i32]) -> Vec<i32> {
    data.iter().map(|x| x * 2).collect()
}

// 问题：字符串传递开销大
// 解决：使用内存直接操作
#[wasm_bindgen]
pub fn get_buffer_ptr() -> *mut u8 {
    // 返回内部缓冲区的指针
    // JavaScript 直接操作 WASM 内存
    std::ptr::null_mut()
}

// 问题：内存碎片化
// 解决：使用 wee_alloc 或预分配
#[global_allocator]
static ALLOC: wee_alloc::WeeAlloc = wee_alloc::WeeAlloc::INIT;
```
