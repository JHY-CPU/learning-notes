## 二、具体用法

### 2.1 技术栈总结

```toml
# 核心依赖
wasm-bindgen = "0.2"      # JavaScript 绑定
web-sys = "0.3"           # Web API 绑定
js-sys = "0.3"            # JavaScript 对象
wasm-bindgen-futures = "0.4"  # 异步支持
serde = { version = "1.0", features = ["derive"] }  # 序列化
serde_json = "1.0"        # JSON 处理
```

### 2.2 工具链总结

```bash
# 核心工具
wasm-pack              # 构建和打包
wasm-bindgen           # 生成绑定
wasm-opt               # 优化
wasm2wat / wat2wasm    # 转换工具
```
