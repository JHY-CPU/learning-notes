# 性能优化

## 一、概念说明

Rust WASM 性能优化包括编译优化、代码优化和运行时优化。

```rust
// 使用 #[inline] 优化热点函数
#[inline]
fn hot_function(x: i32) -> i32 {
    x * x
}

// 使用 unsafe 避免边界检查
#[wasm_bindgen]
pub unsafe fn fast_process(data: &mut [u8]) {
    for i in 0..data.len() {
        *data.get_unchecked_mut(i) = data.get_unchecked(i).wrapping_mul(2);
    }
}
```

### 2.2 代码优化

```rust
// 批量处理
#[wasm_bindgen]
pub fn process_batch(data: &[u8]) -> Vec<u8> {
    data.iter()
        .map(|&x| x.wrapping_mul(2))
        .collect()
}

// 避免分配
#[wasm_bindgen]
pub fn process_in_place(data: &mut [u8]) {
    for x in data.iter_mut() {
        *x = x.wrapping_mul(2);
    }
}

// 使用 SmallVec
use smallvec::SmallVec;

#[wasm_bindgen]
pub fn small_vec_example() -> Vec<u8> {
    let mut vec: SmallVec<[u8; 16]> = SmallVec::new();
    vec.extend_from_slice(&[1, 2, 3, 4, 5]);
    vec.into_vec()
}
```

### 2.3 SIMD 优化

```rust
use std::arch::wasm32::*;

#[wasm_bindgen]
pub unsafe fn simd_add(a: &[f32], b: &[f32], result: &mut [f32]) {
    let chunks = a.len() / 4;
    for i in 0..chunks {
        let offset = i * 4;
        let va = v128_load(a.as_ptr().add(offset) as *const v128);
        let vb = v128_load(b.as_ptr().add(offset) as *const v128);
        let vr = f32x4_add(va, vb);
        v128_store(result.as_mut_ptr().add(offset) as *mut v128, vr);
    }
}
```

### 2.4 内存优化

```rust
// 内存池
#[wasm_bindgen]
pub struct Pool {
    buffers: Vec<Vec<u8>>,
}

#[wasm_bindgen]
impl Pool {
    pub fn get_buffer(&mut self, size: usize) -> usize {
        if let Some(mut buf) = self.buffers.pop() {
            buf.resize(size, 0);
            self.buffers.push(buf);
            self.buffers.len() - 1
        } else {
            self.buffers.push(vec![0u8; size]);
            self.buffers.len() - 1
        }
    }
}
```

## 三、注意事项与常见陷阱

1. **测量优先**：先测量再优化
2. **热点识别**：使用分析工具找热点
3. **权衡考虑**：权衡代码大小和性能
4. **浏览器差异**：不同浏览器性能不同
5. **持续优化**：持续监控和优化性能
