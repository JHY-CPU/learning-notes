# 异步编程

## 一、概念说明

Rust WASM 支持异步编程，使用 wasm-bindgen-futures 桥接 Rust 和 JavaScript 的异步。

```rust
use wasm_bindgen::prelude::*;
use wasm_bindgen_futures::spawn_local;

#[wasm_bindgen]
pub fn start() {
    spawn_local(async {
        // 异步代码
    });
}
```

## 二、具体用法

### 2.1 基本异步

```rust
use wasm_bindgen::prelude::*;
use wasm_bindgen_futures::JsFuture;
use web_sys::{Request, RequestInit};

#[wasm_bindgen]
pub async fn fetch_url(url: &str) -> Result<String, JsValue> {
    let mut opts = RequestInit::new();
    opts.method("GET");

    let request = Request::new_with_str_and_init(url, &opts)?;
    let window = web_sys::window().unwrap();
    let resp = JsFuture::from(window.fetch_with_request(&request)).await?;
    let resp: web_sys::Response = resp.dyn_into()?;
    let text = JsFuture::from(resp.text()?).await?;
    Ok(text.as_string().unwrap())
}
```

### 2.2 并发执行

```rust
use futures::future::join_all;

#[wasm_bindgen]
pub async fn fetch_multiple(urls: &js_sys::Array) -> js_sys::Array {
    let mut futures = vec![];

    for i in 0..urls.length() {
        let url = urls.get(i).as_string().unwrap();
        futures.push(fetch_url(&url));
    }

    let results = join_all(futures).await;
    let arr = js_sys::Array::new();

    for result in results {
        if let Ok(text) = result {
            arr.push(&text.into());
        }
    }

    arr
}
```

### 2.3 定时器

```rust
use wasm_bindgen::prelude::*;
use wasm_bindgen_futures::JsFuture;

#[wasm_bindgen]
pub async fn delay(ms: i32) {
    let promise = js_sys::Promise::new(&mut |resolve, _| {
        web_sys::window()
            .unwrap()
            .set_timeout_with_callback_and_timeout_and_arguments_0(&resolve, ms)
            .unwrap();
    });

    JsFuture::from(promise).await.unwrap();
}

#[wasm_bindgen]
pub async fn run_with_delay() {
    delay(1000).await;
    console::log_1(&"1秒后执行".into());
}
```

### 2.4 动画循环

```rust
use wasm_bindgen::prelude::*;
use wasm_bindgen::JsCast;

#[wasm_bindgen]
pub fn start_animation_loop() {
    let f: Closure<dyn FnMut()> = Closure::wrap(Box::new(move || {
        // 更新动画
        web_sys::window()
            .unwrap()
            .request_animation_frame(f.as_ref().unchecked_ref())
            .unwrap();
    }));

    web_sys::window()
        .unwrap()
        .request_animation_frame(f.as_ref().unchecked_ref())
        .unwrap();

    f.forget();
}
```

## 三、注意事项与常见陷阱

1. **生命周期**：异步函数的生命周期
2. **错误处理**：异步错误处理
3. **取消支持**：取消异步操作
4. **性能**：异步操作有开销
5. **调试**：异步调试更困难
