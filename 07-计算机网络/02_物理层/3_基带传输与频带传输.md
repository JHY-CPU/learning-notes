# 04_基带传输与频带传输

## 核心概念

- **基带传输**：将基带信号直接在信道中传输，不经过调制
  - 基带信号：来自信源的原始信号，通常包含低频分量甚至直流分量
  - 典型应用：局域网中的以太网（10BASE-T等）
- **频带传输**：将基带信号通过调制搬到高频段后传输
  - 需要在发送端调制，接收端解调
  - 典型应用：ADSL、传统电话拨号上网

### 为什么要频带传输

- 基带信号含有低频和直流分量，许多信道（如无线信道、电话线）不能传输低频分量
- 一条信道频带较宽，不调制则只能传一路基带信号
- 调制后可以将多路信号搬移到不同频段，实现**频分复用**

### 调制的基本方式

| 调制方式 | 英文 | 原理 | 抗噪声 |
|----------|------|------|--------|
| 调幅（AM） | Amplitude Modulation | 载波振幅随基带信号变化 | 差 |
| 调频（FM） | Frequency Modulation | 载波频率随基带信号变化 | 较好 |
| 调相（PM） | Phase Modulation | 载波相位随基带信号变化 | 较好 |

## 原理分析

### 基带传输

```
数字比特流 → 编码 → 基带信号 → [信道] → 解码 → 数字比特流
```

- 不需要调制和解调
- 编码方式：曼彻斯特编码、差分曼彻斯特编码、NRZ、NRZI等
- 适用于距离较短、带宽充足的信道

### 频带传输

```
数字比特流 → 调制 → 频带信号 → [信道] → 解调 → 数字比特流
```

- 调制方式：ASK、FSK、PSK、QAM等
- 适用于长距离传输或需要频分复用的场景

### 正交幅度调制（QAM）

QAM结合了调幅和调相，能携带更多信息：

$$R_b = R_B \cdot \log_2 M$$

例如 16-QAM：$M=16$，每个码元携带 $\log_2 16 = 4$ 比特。

## 直观理解

### 类比：寄信

- **基带传输**好比直接走路去送信——简单直接，但距离有限
- **频带传输**好比把信件交给快递公司（调制），快递走高速公路（高频信道），到达后再取出来（解调）
- 多路频带信号共用一条信道，好比多辆快递车走同一条高速的不同车道

### 记忆要点

- **基带** = 基本频带 = 原始信号频带（低频段）
- **频带** = 调制后的频带（高频段）
- 基带信号：频谱从零开始
- 频带信号：频谱从某个非零频率开始

## 协议关联

### 与物理层编码的关系

- 基带传输依赖于**线路编码**（曼彻斯特编码等）
- 频带传输依赖于**数字调制**（ASK/FSK/PSK/QAM）

### 与复用技术的关系

- 基带传输通常与**时分复用（TDM）**配合
- 频带传输通常与**频分复用（FDM）**配合

### 考试常见考点

- 问：什么情况下用基带传输？答：局域网、短距离
- 问：什么情况下用频带传输？答：长距离、需要频分复用、通过模拟信道传数字数据
- 问：ADSL用什么传输方式？答：频带传输（使用DMT调制技术）
