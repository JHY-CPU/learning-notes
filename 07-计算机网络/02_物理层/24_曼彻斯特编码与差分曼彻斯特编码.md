# 25_曼彻斯特编码与差分曼彻斯特编码

## 核心概念

### 曼彻斯特编码（Manchester Encoding）

- **定义**：每个比特的中间必定有一次电平跳变
- **编码规则**：
  - 比特"1"：从高电平跳变到低电平（高→低）
  - 比特"0"：从低电平跳变到低高电平（低→高）
- **注意**：谢希仁教材中 1 和 0 的定义可能与其他教材相反，408以谢希仁教材为准
- **特点**：自带时钟信号，具有自同步能力

### 差分曼彻斯特编码（Differential Manchester Encoding）

- **定义**：每个比特中间必有一次跳变，比特开始处有无跳变表示数据
- **编码规则**：
  - 比特"1"：比特**开始处无跳变**（电平延续上一比特的末尾）
  - 比特"0"：比特**开始处有跳变**（电平与上一比特末尾相反）
- **特点**：比曼彻斯特编码有更好的抗干扰性（看相对变化而非绝对电平）

### 两者对比

| 特性 | 曼彻斯特 | 差分曼彻斯特 |
|------|----------|-------------|
| 比特中间跳变 | 有（必须） | 有（必须） |
| 比特开始跳变 | 不一定 | 有=0，无=1 |
| 编码依据 | 绝对电平 | 相对跳变 |
| 自同步能力 | 有 | 有 |
| 抗干扰 | 一般 | 更好 |
| 典型应用 | 10M以太网 | 令牌环网 |

## 原理分析

### 曼彻斯特编码的波形

数据 **10110** 的编码：

```
电平
 高 ┌─┐   ┌─────┐ ┌─┐
    │ │   │     │ │ │
 低 ─┘ └───┘     └─┘ └─────
    1   0    1    1    0

规则：1=高→低（比特中间从高跳到低）
      0=低→高（比特中间从低跳到高）
```

### 差分曼彻斯特编码的波形

数据 **10110** 的编码（设初始状态为高电平）：

```
电平
 高 ┌───┐ ┌─┐ ┌───────┐
    │   │ │ │ │       │
 低 ─┘   └─┘ └─┘       └─────
    1   0   1    1     0

规则：1=开始处无跳变，中间有跳变
      0=开始处有跳变，中间有跳变
```

### 编码效率计算

- 曼彻斯特编码：1个数据比特需要2个电平状态（或说线路速率是数据速率的2倍）
- 编码效率 = 数据速率 / 线路速率 = 50%
- 例如：10 Mbit/s数据，需要 20 Mbaud 的线路速率

## 直观理解

### 曼彻斯特编码类比：心跳

- 每个比特中间必须跳变，好比人的心跳——每一下都有规律的跳动
- 如果某一段时间没有跳变（如NRZ的连续0），好比心跳停止——接收端就知道出问题了
- 跳变就是"我还活着"的信号

### 差分曼彻斯特类比：火车信号灯

- 不看灯的绝对颜色，而是看灯有没有**变化**
- 变了 = 0，没变 = 1
- 这样即使有人把灯的颜色搞反了（极性反转），也不会影响解码

### 为什么以太网选择曼彻斯特

1. 自同步能力强：不需要额外的时钟线
2. 无直流分量：适合变压器耦合
3. 实现简单：可用异或门实现编码
4. 但代价是带宽效率只有50%

## 协议关联

### 与以太网标准的关系

- **10BASE5、10BASE2、10BASE-T**（所有10M以太网）都使用**曼彻斯特编码**
- 曼彻斯特编码保证了以太网的自同步能力
- 线路速率为 20 Mbaud（数据速率 10 Mbit/s 的2倍）

### 与令牌环网的关系

- IEEE 802.5 令牌环网使用**差分曼彻斯特编码**
- 差分曼彻斯特的抗干扰性更好，适合环形拓扑

### 考试要点

1. 曼彻斯特编码每个比特中间**必定**有跳变（这是判断依据）
2. 曼彻斯特编码的线路速率 = 2 × 数据速率
3. 差分曼彻斯特看"比特开始处有无跳变"
4. 10M以太网用曼彻斯特编码（408常考）
5. 注意区分编码规则（以谢希仁教材为准：1=高→低，0=低→高）
