# 19_传输介质_概念_考试

## 核心概念

### 传输介质分类总览

```
传输介质
├─ 有导向介质（有线）
│  ├─ 双绞线
│  ├─ 同轴电缆
│  └─ 光纤
└─ 无导向介质（无线）
   ├─ 无线电波
   ├─ 微波
   ├─ 红外线
   └─ 卫星
```

### 三种有线介质对比（408高频考点）

| 特性 | 双绞线 | 同轴电缆 | 光纤 |
|------|--------|----------|------|
| 带宽 | 低（~数百MHz） | 中（~数GHz） | 极高（~数十THz） |
| 衰减 | 高 | 中 | 极低 |
| 抗电磁干扰 | 差 | 好 | 不受影响 |
| 成本 | 最低 | 中等 | 较高 |
| 安装难度 | 简单 | 中等 | 较难 |
| 典型距离 | 100m | 数百m~数km | 数十km |
| 应用 | 局域网 | 有线电视 | 骨干网 |

### 传输介质与信号类型

| 介质 | 支持的信号类型 |
|------|---------------|
| 双绞线 | 数字、模拟均可 |
| 同轴电缆 | 数字（50Ω）、模拟（75Ω） |
| 光纤 | 光脉冲（本质上是数字） |
| 无线 | 模拟为主（需调制） |

## 原理分析

### 常考参数对比

**以太网标准与介质**：

| 标准 | 介质 | 速率 | 最大距离 |
|------|------|------|----------|
| 10BASE-T | 双绞线 | 10 Mbit/s | 100m |
| 100BASE-TX | 双绞线 | 100 Mbit/s | 100m |
| 1000BASE-T | 双绞线 | 1 Gbit/s | 100m |
| 10BASE5 | 同轴电缆 | 10 Mbit/s | 500m |
| 10BASE2 | 同轴电缆 | 10 Mbit/s | 185m |
| 1000BASE-SX | 多模光纤 | 1 Gbit/s | 550m |
| 1000BASE-LX | 单模光纤 | 1 Gbit/s | 5km |

### 典型选择题

**题目**：下列传输介质中，抗电磁干扰能力最强的是？
A. 双绞线  B. 同轴电缆  C. 光纤  D. 地面微波

**答案**：C. 光纤。光纤利用光信号传输，完全不受电磁干扰影响。

**题目**：10BASE-T以太网使用什么传输介质？
A. 粗同轴电缆  B. 细同轴电缆  C. 双绞线  D. 光纤

**答案**：C. 双绞线。"T"代表Twisted Pair。

## 直观理解

### 记忆口诀

> 双绞便宜距离短，同轴屏蔽中间选
> 光纤带宽抗干扰，无线灵活不安全

### 选择介质的考虑因素

1. **带宽需求**：高带宽选光纤
2. **传输距离**：长距离选光纤或无线
3. **环境干扰**：强电磁干扰选光纤
4. **成本预算**：低成本选双绞线
5. **移动性**：需要移动选无线

## 协议关联

### 与OSI各层的关系

- 传输介质属于物理层
- 数据链路层的帧格式与传输介质无关
- 物理层标准规定了特定介质上的信号编码方式

### 考试常见陷阱

1. **光纤不受电磁干扰**——这是光纤最大的优势之一
2. **双绞线绞合是为了抗干扰**，不是为了提高带宽
3. **同轴电缆需要终端电阻**防止信号反射
4. **无线是无导向介质**，不属于有线传输
5. **10BASE5的"5"代表约500m**，不是5km
