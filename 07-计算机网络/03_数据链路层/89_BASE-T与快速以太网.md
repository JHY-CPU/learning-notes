# 49_10BASE-T与快速以太网

## 核心概念

- **10BASE-T**：10Mbps以太网，使用双绞线，星形拓扑，最大距离100m
- **快速以太网（Fast Ethernet）**：100Mbps，IEEE 802.3u标准（1995年）
- **千兆以太网**：1Gbps，IEEE 802.3z/ab标准（1998年）

### 10BASE-T特点

| 特性 | 值 |
|------|-----|
| 速率 | 10 Mbps |
| 介质 | 3类/5类双绞线 |
| 拓扑 | 星形（集线器） |
| 最大距离 | 100m |
| 编码 | 曼彻斯特编码 |
| 连接器 | RJ-45 |

### 快速以太网（100Mbps）

| 标准 | 介质 | 最大距离 | 编码 |
|------|------|---------|------|
| 100BASE-TX | 5类双绞线 | 100m | 4B/5B + MLT-3 |
| 100BASE-FX | 多模光纤 | 412m | 4B/5B |
| 100BASE-T4 | 3类双绞线 | 100m | 8B/6T |

### 快速以太网的改进

- 速率从10Mbps提升到100Mbps
- 帧格式不变（向后兼容）
- 帧间间隔从9.6μs变为0.96μs
- 最小帧长不变（64字节）→ 最大网络直径缩小10倍

### 半双工 vs 全双工

| 模式 | 特点 | CSMA/CD |
|------|------|---------|
| 半双工 | 同一时间只能收或发 | 需要 |
| 全双工 | 可同时收发 | 不需要 |

- 集线器连接：半双工（共享冲突域）
- 交换机连接：全双工（点对点，无冲突）

### 考试重点

- 快速以太网与10M以太网的兼容性
- 半双工和全双工的区别
- 速率提高对网络直径的影响

## 原理分析

### 速率提升的影响

$$L_{\min} = 2\tau \times R$$

当 $R$ 增大10倍（10M→100M），$L_{\min}$ 不变时，$2\tau$ 必须缩小10倍：

$$\tau_{100M} = \tau_{10M} / 10$$

这意味着最大网络直径缩小10倍。

### 100BASE-TX的规范

- 使用2对5类双绞线（一对发，一对收）
- 4B/5B编码（每4位数据编码为5位符号）
- MLT-3电平转换（减少带宽需求）

## 直观理解

- **快速以太网 = 升级版以太网**：路修得更宽（更高带宽），但车（帧）的规格不变
- **全双工 = 双向四车道**：来去各走各的，不会撞车

## 协议关联

- 快速以太网继承了10M以太网的帧格式和CSMA/CD
- 全双工模式使CSMA/CD成为可选
- 交换机的普及使全双工成为主流
- 100M→1G→10G的演进不断推动以太网进入广域网领域
