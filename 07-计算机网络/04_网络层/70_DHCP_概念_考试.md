# 34_DHCP_概念_考试

## 核心概念

- **408 DHCP考点**：
  1. DHCP基于UDP协议（端口67/68）
  2. DHCP的DORA四步过程
  3. DHCPDISCOVER使用广播
  4. DHCP支持跨网段（中继代理）
  5. DHCP租约机制

## 原理分析

### 真题1：DHCP协议基础

**题**：以下关于DHCP的说法，正确的是？
A. DHCP基于TCP协议
B. DHCP服务器端口号为80
C. DHCP基于UDP协议
D. DHCP只能在局域网内使用

**解**：C。DHCP基于UDP，服务器端口67，客户端端口68。通过中继代理可跨网段使用。

### 真题2：DHCP交互过程

**题**：DHCP客户端首次获取IP地址时，发送的第一个报文是什么？目的IP是什么？

**解**：
- 第一个报文：**DHCPDISCOVER**
- 源IP：0.0.0.0（还没有IP）
- 目的IP：255.255.255.255（受限广播）
- 封装在UDP中，UDP封装在IP数据报中

### 真题3：DHCP vs ARP

**题**：DHCP和ARP都使用广播，它们的主要区别是什么？

**解**：
| 对比 | DHCP | ARP |
|------|------|-----|
| 功能 | 动态分配IP地址 | 解析IP→MAC |
| 协议层次 | 应用层（基于UDP） | 网络层 |
| 广播类型 | 受限广播（255.255.255.255） | 数据链路层广播（FF:FF:FF:FF:FF:FF） |
| 跨网段 | 支持（中继代理） | 不支持 |

### 真题4：DHCP租约

**题**：DHCP客户端租约时间为86400秒（24小时），客户端在什么时间尝试续约？

**解**：
- 第一次续约尝试：50% 租期 = 43200秒（12小时）
- 若失败，第二次尝试：87.5% 租期 = 75600秒（21小时）
- 若仍失败，到期后释放IP，重新发送DHCPDISCOVER

### 真题5：综合判断

**题**：某主机接入网络后需要获取以下信息：IP地址、子网掩码、默认网关、DNS服务器地址。应使用什么协议？

**解**：**DHCP**
- DHCP可以一次分配所有这些信息
- RARP只能获取IP地址
- ARP是获取MAC地址

## 直观理解

**DHCP报文类型速记**：
| 报文 | 作用 | 方向 |
|------|------|------|
| DISCOVER | 寻找服务器 | 客户端→广播 |
| OFFER | 提供IP地址 | 服务器→客户端 |
| REQUEST | 请求使用IP | 客户端→广播 |
| ACK | 确认分配 | 服务器→客户端 |
| NAK | 拒绝请求 | 服务器→客户端 |
| RELEASE | 释放IP | 客户端→服务器 |

**口诀**："D找O给R要A确认，租约到期续一半"

## 协议关联

- DHCP与UDP：DHCP报文封装在UDP段中
- DHCP与IP：客户端初始没有IP，使用0.0.0.0作为源地址
- DHCP与广播：DISCOVER使用受限广播，路由器不转发
- DHCP中继：路由器配置中继代理后可转发DHCP报文到其他网段的DHCP服务器
