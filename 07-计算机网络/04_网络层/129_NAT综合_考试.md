# NAT综合考试题

## 核心概念

### 高频考点
- NAT三种类型：静态NAT、动态NAT、NAPT/PAT
- NAPT使用端口号区分不同连接（多对一映射）
- NAT转换表维护私有地址与公有地址的映射关系
- NAT工作在网络层，NAPT还需查看传输层端口号

### 常见陷阱
- NAT不是路由协议，不参与路由选择
- 静态NAT和动态NAT不使用端口号（一对一映射）
- NAPT也称为PAT或地址端口复用

## 原理分析

### 典型选择题
**Q1**: NAPT使用什么来区分不同的内部主机？
A. IP地址 B. 端口号 C. MAC地址 D. 协议号

**答**: B。NAPT通过端口号区分共享同一公有地址的不同内部主机。

**Q2**: 一个NAPT路由器拥有1个公有IP地址，最多支持多少个TCP连接？
A. 1个 B. 256个 C. 65535个 D. 无限个

**答**: C。端口号16位，理论最大值$2^{16}-1 = 65535$个（减去端口0）。

**Q3**: 下列关于NAT的描述，正确的是？
A. NAT是一种路由协议 B. NAT可以解决IPv4地址不足的问题
C. NAT不需要转换表 D. NAT只工作在传输层

**答**: B。NAT通过地址转换让多个私有地址共享公有地址，解决地址不足。

### 计算题
某NAPT路由器有1个公有地址202.100.1.1，内部网络使用192.168.1.0/24
- 内部最多254台主机（192.168.1.1~254）
- 每台主机最多64512个并发连接（端口1024~65535）
- 总理论连接数：254 × 64512 ≈ 1638万

### 更多考试题

**Q4**: 以下哪种NAT类型不需要查看端口号？
A. 静态NAT B. NAPT C. PAT D. 动态NAPT

**答**：A。静态NAT是一对一映射，只看IP地址，不需要端口号。

**Q5**: NAT破坏了互联网的什么基本原则？
A. 分层原则 B. 端到端原则 C. 最长前缀匹配 D. 尽力而为

**答**：B。NAT使内部主机不能直接被外部访问，破坏了端到端可达性。

**Q6**: 以下哪些协议与NAT存在兼容性问题？
A. TCP B. IPsec AH C. UDP D. HTTP

**答**：B。IPsec AH对IP首部进行认证，NAT修改地址后认证失败。

### NAT类型选择指南
| 场景 | 推荐NAT类型 |
|------|------------|
| 服务器对外提供服务 | 静态NAT |
| 大量用户少量地址 | NAPT |
| 临时地址分配 | 动态NAT |
| P2P应用 | 需要NAT穿越技术 |

### 私有地址范围速查
```
10.0.0.0/8       → A类私有（1677万地址）
172.16.0.0/12    → B类私有（104万地址）
192.168.0.0/16   → C类私有（6.5万地址）
```

### 408易错点
- NAT不参与路由选择（只做地址转换）
- NAPT也叫PAT或地址端口复用
- 静态NAT和动态NAT不用端口号
- NAT工作在2.5层（查看IP和端口）
- NAT穿越是VoIP和P2P的核心问题

## 直观理解
- NAT像"公司前台"——内部用分机号，对外统一用总机号
- NAPT像"合租房门牌"——一个地址多个住户，用房间号区分
- NAT穿越像"快递代收"——找不到具体门牌时通过中转

## 协议关联
- NAT与防火墙常集成使用
- NAT穿越是P2P应用的核心问题
- IPv6的普及将减少NAT的使用需求
- STUN/TURN/ICE是NAT穿越的常用方案
