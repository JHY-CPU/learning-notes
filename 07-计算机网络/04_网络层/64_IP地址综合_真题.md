# 28_IP地址综合_真题

## 核心概念

- **408综合真题特点**：
  - 通常融合地址分类、子网划分、分片等多个知识点
  - 要求写出计算过程
  - 注意单位换算和边界条件

## 原理分析

### 真题1：综合计算

**题**：某主机IP地址为172.16.100.5，子网掩码为255.255.192.0。
（1）该IP属于哪类地址？
（2）求该子网的网络地址。
（3）该子网的广播地址是什么？
（4）该子网最多可容纳多少台主机？

**解**：
（1）172在128~191之间 → **B类地址**
（2）192 = 11000000
- 第三字节：100 AND 192 = 01100100 AND 11000000 = 01000000 = 64
- 网络地址 = **172.16.64.0**
（3）第三字节：64 OR (255-192) = 64 OR 63 = 01111111 = 127
广播地址 = **172.16.127.255**
（4）主机位 = 14位（第三字节6位 + 第四字节8位）
可用主机数 = $2^{14} - 2 = 16382$

### 真题2：路由转发

**题**：路由器收到目的地址为192.168.3.100的分组，路由表如下：

| 目的网络 | 子网掩码 | 下一跳 |
|---------|---------|--------|
| 192.168.0.0 | 255.255.252.0 | R1 |
| 192.168.3.0 | 255.255.255.0 | R2 |
| 192.168.2.0 | 255.255.254.0 | R3 |
| 0.0.0.0 | 0.0.0.0 | R4 |

问：分组从哪个下一跳转发？

**解**：
- 匹配/22（255.255.252.0）：192.168.3.100 AND 255.255.252.0 = 192.168.0.0 ✓
- 匹配/24（255.255.255.0）：192.168.3.100 AND 255.255.255.0 = 192.168.3.0 ✓
- 匹配/23（255.255.254.0）：192.168.3.100 AND 255.255.254.0 = 192.168.2.0 ✓
  - 验证：3=00000011, 2=00000010，前7位相同 ✓
- 匹配/0：默认 ✓

最长匹配：**/24 → R2**

### 真题3：子网划分设计

**题**：某公司申请到C类地址202.110.0.0/24（实际为202.110.x.x，设为202.110.1.0/24），需要划分5个子网，每个子网至少25台主机。
（1）最小子网掩码是多少？
（2）每个子网可用主机数？
（3）第3个子网的网络地址和广播地址？

**解**：
（1）$2^n \geq 5$，$n=3$，借3位
剩余主机位 = 5，$2^5 - 2 = 30 \geq 25$ ✓
子网掩码 = **255.255.255.224（/27）**
（2）每子网可用主机 = $2^5 - 2 = 30$
（3）间隔 = 32
第3个子网（索引2）：网络地址 = 202.110.1.64
广播地址 = 202.110.1.95

### 真题4：CIDR分配

**题**：ISP拥有地址块200.10.8.0/21，需分配给4个部门，每个部门至少200个地址。求分配方案。

**解**：
- $2^m - 2 \geq 200$，$m = 8$，每部门需/24
- /21包含 $2^{11} = 2048$ 地址，可分8个/24
- 分配4个/24：200.10.8.0/24, 200.10.9.0/24, 200.10.10.0/24, 200.10.11.0/24

## 直观理解

**408综合题解题策略**：
1. 仔细审题，识别考查哪些知识点
2. 按步骤计算，每步验证
3. 注意单位（4字节/字节/8字节）
4. 最后交叉验证
