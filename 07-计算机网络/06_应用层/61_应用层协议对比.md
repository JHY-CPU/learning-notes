# 62_应用层协议对比

## 核心概念

- **应用层协议对比**：对比各应用层协议的特点
- **对比维度**：端口号、传输层、模式、状态、用途
- **408考试重点**：协议对比是常考题型

## 原理分析

### 主要协议对比

| 协议 | 端口 | 传输层 | 模式 | 状态 | 用途 |
|------|------|--------|------|------|------|
| HTTP | 80 | TCP | C/S | 无状态 | Web |
| HTTPS | 443 | TCP | C/S | 无状态 | 安全Web |
| FTP | 20/21 | TCP | C/S | 有状态 | 文件传输 |
| SMTP | 25 | TCP | C/S | - | 发送邮件 |
| POP3 | 110 | TCP | C/S | - | 接收邮件 |
| IMAP | 143 | TCP | C/S | - | 接收邮件 |
| DNS | 53 | UDP/TCP | C/S | - | 域名解析 |
| DHCP | 67/68 | UDP | C/S | - | 动态IP |
| TELNET | 23 | TCP | C/S | 有状态 | 远程终端 |
| SSH | 22 | TCP | C/S | 有状态 | 安全远程 |
| SNMP | 161/162 | UDP | C/S | - | 网络管理 |

### HTTP vs FTP

| 特性 | HTTP | FTP |
|------|------|-----|
| 连接数 | 1个TCP | 2个TCP |
| 状态 | 无状态 | 有状态 |
| 端口 | 80 | 20/21 |
| 用途 | Web文档 | 文件传输 |

### SMTP vs POP3 vs IMAP

| 特性 | SMTP | POP3 | IMAP |
|------|------|------|------|
| 端口 | 25 | 110 | 143 |
| 方向 | 推 | 拉 | 拉 |
| 邮件存储 | - | 下载删除 | 服务器保留 |
| 多设备 | - | 不支持 | 支持 |

### DNS vs DHCP

| 特性 | DNS | DHCP |
|------|-----|------|
| 端口 | 53 | 67/68 |
| 传输层 | UDP/TCP | UDP |
| 功能 | 域名→IP | 分配IP |
| 过程 | 查询 | DORA |

## 直观理解

**记忆技巧**：
- HTTP无状态，FTP有状态
- SMTP推，POP3/IMAP拉
- DNS查域名，DHCP分IP
- HTTP单连接，FTP双连接

### 安全协议对比

| 协议 | 端口 | 安全替代 | 安全机制 |
|------|------|----------|----------|
| HTTP | 80 | HTTPS(443) | SSL/TLS加密 |
| TELNET | 23 | SSH(22) | 加密+认证 |
| FTP | 20/21 | SFTP(22) | SSH隧道 |
| SMTP | 25 | SMTPS(465) | SSL/TLS |
| POP3 | 110 | POP3S(995) | SSL/TLS |
| IMAP | 143 | IMAPS(993) | SSL/TLS |

### HTTP版本对比

| 版本 | 连接 | 默认行为 |
|------|------|----------|
| HTTP/1.0 | 非持久 | 每个对象新TCP连接 |
| HTTP/1.1 | 持久 | 复用TCP连接 |
| HTTP/2 | 持久+多路复用 | 同一连接并行传输 |
| HTTP/3 | QUIC(UDP) | 0-RTT，连接迁移 |

### FTP两种模式
- **主动模式**：服务器主动连接客户端的数据端口（端口20）
- **被动模式**：服务器开放端口等待客户端连接（适应防火墙）

### 邮件协议详细对比

| 特性 | SMTP | POP3 | IMAP |
|------|------|------|------|
| 端口 | 25 | 110 | 143 |
| 功能 | 发送邮件 | 接收邮件 | 接收邮件 |
| 模式 | 推 | 拉 | 拉 |
| 存储 | - | 下载删除 | 服务器保留 |
| 多设备 | - | 不支持 | 支持 |
| 离线 | - | 支持 | 部分支持 |

### 408常考要点
- HTTP无状态，FTP有状态
- SMTP发送，POP3/IMAP接收
- DNS默认UDP，大报文TCP
- FTP双连接（控制+数据），HTTP单连接
- 安全协议：HTTPS=HTTP+SSL/TLS，SSH替代TELNET
- 端口号必须熟记

## 协议关联

- **协议与传输层**：TCP还是UDP
- **协议与安全**：各协议的安全替代方案
- **协议与应用**：各协议的适用场景
- **408常见组合**：协议对比题是高频考点
