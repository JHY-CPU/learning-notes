# 61_应用层端口号_考试

## 核心概念

- **408常考题型**：选择题为主，必须熟记端口号
- **高频考点**：常用协议的端口号
- **出题规律**：直接考端口号或给出端口号判断协议

## 原理分析

### 典型考题1：端口号匹配

**题目**：HTTP使用的端口号是（  ）
A. 21
B. 25
C. 53
D. 80

**答案**：D

### 典型考题2：协议判断

**题目**：端口号25对应的协议是（  ）
A. FTP
B. SMTP
C. POP3
D. DNS

**答案**：B

### 典型考题3：UDP端口

**题目**：以下使用UDP的协议是（  ）
A. HTTP
B. FTP
C. DNS
D. SMTP

**答案**：C

**解析**：
- HTTP、FTP、SMTP都使用TCP
- DNS主要使用UDP

### 典型考题4：端口填空

**题目**：完成以下表格：
| 协议 | 端口号 |
|------|--------|
| FTP控制 | ? |
| SMTP | ? |
| POP3 | ? |
| DNS | ? |
| HTTPS | ? |

**答案**：
| 协议 | 端口号 |
|------|--------|
| FTP控制 | 21 |
| SMTP | 25 |
| POP3 | 110 |
| DNS | 53 |
| HTTPS | 443 |

## 直观理解

**做题技巧**：
- "21控制20数据，25发邮件，53域名"
- "80看网页，443安全"
- "110取邮件，143也取"
- "67/68动态IP，161/162管理"

### 典型考题5：端口号分类

**题目**：以下端口号中，属于熟知端口的是（  ）
A. 8080
B. 80
C. 49152
D. 1024

**答案**：B

**解析**：
| 范围 | 类型 | 说明 |
|------|------|------|
| 0~1023 | 熟知端口 | 分配给标准服务 |
| 1024~49151 | 注册端口 | 分配给用户进程 |
| 49152~65535 | 动态端口 | 临时分配 |

### 典型考题6：端口号判断

**题目**：以下哪个端口号不属于标准应用层协议？
A. 21
B. 25
C. 8080
D. 110

**答案**：C

**解析**：
- 21 = FTP控制，25 = SMTP，110 = POP3
- 8080是常用的代理/备选端口，不是标准端口

### 完整端口号速查表

| 端口 | 协议 | 传输层 | 用途 |
|------|------|--------|------|
| 20 | FTP数据 | TCP | 文件传输数据通道 |
| 21 | FTP控制 | TCP | 文件传输控制通道 |
| 22 | SSH | TCP | 安全远程登录 |

| 67 | DHCP服务器 | UDP | 动态IP分配 |
| 68 | DHCP客户端 | UDP | 动态IP请求 |
| 80 | HTTP | TCP | Web服务 |
| 110 | POP3 | TCP | 接收邮件 |
| 143 | IMAP | TCP | 接收邮件 |

| 554 | RTSP | TCP | 实时流控制 |

### 408常考要点
- 熟知端口0~1023，注册端口1024~49151
- 端口号16位（0~65535）
- TCP和UDP的端口号空间独立（同名端口不同协议含义不同）
- DNS默认UDP 53，大报文用TCP
- DHCP使用67（服务器）和68（客户端）
- 端口号是408送分题，必须熟记

## 协议关联

- **端口与传输层**：端口标识应用层服务
- **端口与应用层**：每个协议有固定端口
- **端口与TCP/UDP**：TCP和UDP有独立的端口号空间
- **408必考**：端口号是送分题，必须记住
