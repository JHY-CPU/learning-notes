# 81_DNS_HTTP_FTP综合_考试

## 核心概念

- **408常考题型**：选择题为主，涉及DNS/HTTP/FTP综合对比
- **关键考点**：
  - 各协议端口号
  - 传输层选择
  - 协议特点对比
- **出题规律**：给出特征，判断协议

## 原理分析

### 典型考题1：端口判断

**题目**：端口号80对应的协议是（  ）
A. DNS
B. HTTP
C. FTP
D. SMTP

**答案**：B

### 典型考题2：传输层判断

**题目**：以下使用UDP的协议是（  ）
A. HTTP
B. FTP
C. DNS
D. SMTP

**答案**：C

### 典型考题3：协议特点

**题目**：以下协议中有状态的是（  ）
A. HTTP
B. DNS
C. FTP
D. SMTP

**答案**：C

### 典型考题4：综合判断

**题目**：关于应用层协议，以下说法正确的是（  ）
A. HTTP使用两个TCP连接
B. FTP是无状态协议
C. DNS使用UDP端口53
D. HTTP使用端口21

**答案**：C

## 直观理解

**做题技巧**：
- DNS 53/UDP，HTTP 80/TCP，FTP 20+21/TCP
- HTTP无状态，FTP有状态
- DNS用UDP，HTTP/FTP用TCP

### 典型考题5：FTP模式

**题目**：FTP主动模式中，服务器使用的数据端口是（  ）
A. 21
B. 20
C. 22
D. 80

**答案**：B

**解析**：
- FTP控制连接端口：21
- FTP主动模式数据连接端口：20
- FTP被动模式数据连接端口：服务器随机分配

### 典型考题6：DNS传输层

**题目**：关于DNS使用的传输层协议，以下说法正确的是（  ）
A. DNS只使用UDP
B. DNS只使用TCP
C. DNS默认UDP，大报文用TCP
D. DNS使用ICMP

**答案**：C

**解析**：
- DNS默认使用UDP（快速、简单）
- 响应超过512字节时使用TCP
- 区域传送使用TCP

### 典型考题7：协议特征判断

**题目**：以下说法正确的是（  ）（多选）
A. HTTP是无状态协议
B. FTP是无状态协议
C. DNS使用UDP端口53
D. FTP使用一个TCP连接

**答案**：A, C

**解析**：
- HTTP无状态（A正确）
- FTP有状态（B错误）
- DNS使用UDP 53（C正确）
- FTP使用两个TCP连接（D错误）

### 408易错点
- DNS既用UDP也用TCP（不是只用一种）
- FTP有两个TCP连接（控制+数据）
- HTTP无状态，FTP有状态
- HTTP/1.1默认持久连接，1.0默认非持久
- DNS查询需要先于HTTP/FTP进行
- FTP主动模式和被动模式的数据连接建立方式不同

## 协议关联

- **DNS + HTTP + FTP** = 应用层核心协议
- **端口号 + 传输层** = 必考内容
- **协议特征** = 无状态/有状态、单连接/双连接
- **408常见组合**：协议对比题是高频考点
