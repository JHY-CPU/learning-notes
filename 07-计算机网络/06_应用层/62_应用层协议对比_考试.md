# 63_应用层协议对比_考试

## 核心概念

- **408常考题型**：选择题为主，对比各协议特点
- **关键考点**：协议端口号、传输层、状态特性
- **出题规律**：给出特征，判断协议

## 原理分析

### 典型考题1：协议对比

**题目**：以下使用TCP的协议有（  ）（多选）
A. HTTP
B. DNS
C. FTP
D. DHCP

**答案**：A, C

**解析**：
- HTTP使用TCP（A正确）
- DNS主要使用UDP（B错误）
- FTP使用TCP（C正确）
- DHCP使用UDP（D错误）

### 典型考题2：协议特性

**题目**：以下协议中有状态的是（  ）
A. HTTP
B. FTP
C. DNS
D. SMTP

**答案**：B

**解析**：
- HTTP无状态
- FTP有状态
- DNS无状态
- SMTP无状态

### 典型考题3：协议用途

**题目**：用于发送邮件的协议是（  ）
A. POP3
B. IMAP
C. SMTP
D. HTTP

**答案**：C

### 典型考题4：协议端口

**题目**：HTTPS的端口号是（  ）
A. 80
B. 443
C. 8080
D. 21

**答案**：B

## 直观理解

**做题技巧**：
- TCP协议：HTTP(80)、FTP(20/21)、SMTP(25)、POP3(110)、HTTPS(443)
- UDP协议：DNS(53)、DHCP(67/68)、SNMP(161)
- 有状态：FTP
- 无状态：HTTP

### 典型考题5：安全协议

**题目**：以下协议中，提供加密传输的是（  ）（多选）
A. HTTP
B. HTTPS
C. TELNET
D. SSH

**答案**：B, D

**解析**：
- HTTP明文传输（A错误）
- HTTPS使用SSL/TLS加密（B正确）
- TELNET明文传输（C错误）
- SSH加密传输（D正确）

### 典型考题6：拉与推

**题目**：以下协议中，属于"推"协议的是（  ）
A. POP3
B. IMAP
C. SMTP
D. HTTP

**答案**：C

**解析**：
- SMTP是推协议（发送方主动推送邮件）
- POP3/IMAP是拉协议（接收方主动拉取邮件）
- HTTP可以是拉（GET）或推（POST）

### 典型考题7：协议特征判断

**题目**：以下说法正确的是（  ）
A. DNS只使用UDP
B. FTP使用一个TCP连接
C. HTTP/1.1默认持久连接
D. SNMP使用TCP

**答案**：C

**解析**：
- DNS默认UDP，大报文用TCP（A错误）
- FTP使用两个TCP连接（控制+数据）（B错误）
- HTTP/1.1默认持久连接（C正确）
- SNMP使用UDP（D错误）

### 408易错点
- DNS既用UDP也用TCP（不是只用UDP）
- FTP有两个TCP连接（控制+数据）
- HTTP无状态，FTP有状态
- SMTP是推，POP3/IMAP是拉
- HTTP/1.1默认持久连接，1.0默认非持久
- 所有邮件协议（SMTP/POP3/IMAP）都用TCP

## 协议关联

- **协议与传输层**：TCP还是UDP
- **协议与端口**：每个协议有固定端口
- **协议与安全**：安全协议 vs 明文协议
- **408常见组合**：协议对比 + 端口号 = 选择题
