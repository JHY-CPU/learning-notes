# 71_CDN_考试

## 核心概念

- **408常考题型**：选择题为主，涉及CDN基本概念
- **关键考点**：
  - CDN的作用：减少延迟、减轻源服务器
  - CDN通过DNS将用户导向最近的服务器
  - CDN是分布式Web缓存
- **易混淆点**：
  - CDN不只是缓存，还包含负载均衡等功能

## 原理分析

### 典型考题1：CDN作用

**题目**：CDN的主要作用是（  ）
A. 加密网络数据
B. 减少访问延迟
C. 管理用户认证
D. 路由网络数据

**答案**：B

**解析**：
- CDN将内容缓存到边缘服务器
- 用户从最近的服务器获取内容
- 显著减少访问延迟

### 典型考题2：CDN与DNS

**题目**：CDN通过什么技术将用户导向最近的服务器（  ）
A. TCP
B. DNS
C. HTTP
D. FTP

**答案**：B

**解析**：
- CDN使用DNS（CNAME）将用户导向最近的CDN服务器
- 根据用户位置返回最优服务器IP

### 典型考题3：CDN特点

**题目**：关于CDN，以下说法正确的是（  ）
A. CDN是集中式的
B. CDN可以减轻源服务器负担
C. CDN只用于视频
D. CDN不使用缓存

**答案**：B

## 直观理解

**做题技巧**：
- CDN = 减少延迟 + 减轻源服务器
- CDN通过DNS实现
- CDN是分布式缓存

### 典型考题4：CDN工作原理

**题目**：描述用户访问CDN加速网站的完整过程。

**参考答案**：
1. 用户请求`www.example.com`
2. DNS解析返回CNAME（CDN域名）
3. CDN DNS根据用户位置选择最优边缘服务器
4. 返回边缘服务器IP
5. 用户向边缘服务器请求内容
6. 边缘服务器命中缓存则直接返回
7. 未命中则回源获取并缓存

### 典型考题5：CDN缓存策略

**题目**：CDN的缓存失效（Cache Invalidation）策略有哪些？

**参考答案**：
1. **TTL过期**：设置缓存过期时间
2. **主动推送**：源服务器主动通知CDN更新
3. **按需刷新**：手动触发缓存清除
4. **版本控制**：URL中包含版本号，更新时改变URL

### 典型考题6：CDN优势

**题目**：CDN的主要优势包括（  ）（多选）
A. 减少访问延迟
B. 减轻源服务器负担
C. 提高可用性
D. 提供加密服务

**答案**：A, B, C

**解析**：
- CDN将内容缓存到边缘，减少延迟（A正确）
- 边缘服务器分担请求，减轻源服务器（B正确）
- 多个边缘服务器提供冗余，提高可用性（C正确）
- CDN本身不提供加密（D错误），但可配合HTTPS

### CDN选择策略

| 策略 | 说明 |
|------|------|
| 地理位置 | 选择距离最近的服务器 |
| 网络延迟 | 选择延迟最低的服务器 |
| 服务器负载 | 选择负载最低的服务器 |
| 带宽成本 | 选择带宽成本最低的服务器 |

### CDN与DNS重定向
- 源站域名配置CNAME指向CDN
- CDN的DNS服务器根据用户位置返回最优IP
- 用户直接连接CDN边缘服务器
- 整个过程对用户透明

### 408常考要点
- CDN通过DNS将用户导向最近的边缘服务器
- CDN是分布式Web缓存
- CDN减少延迟、减轻源服务器负担
- CDN使用CNAME重定向
- 缓存命中率是CDN效率的关键指标

- **CDN与HTTP**：CDN缓存HTTP内容

- **CDN与HTTPS**：CDN可配置SSL证书
