# 时间序列异常检测


## 一、异常类型分类


时间序列中的异常（Anomaly）是指与正常模式显著不同的观测值或子序列。根据异常的形态和持续时间，可以分为以下三类：


| 异常类型 | 描述 | 示例 | 检测难度 |
| --- | --- | --- | --- |
| **点异常** | 单个数据点显著偏离正常值 | 服务器CPU突然飙到100% | 低 |
| **上下文异常** | 在特定上下文中异常，但孤立看未必异常 | 30度在夏天正常，冬天异常 | 中 |
| **集体异常** | 连续多个数据点组合起来形成异常模式 | 心电图突然出现异常波形 | 高 |


> **Note:** **算法工程师视角：**
> 在实际业务中，点异常最容易检测但往往价值最低，集体异常最难检测但通常代表真正的系统故障或模式变化。选择检测方法时需考虑异常类型和业务需求。


## 二、统计方法


### 2.1 3σ 法则（三西格玛法则）


假设数据服从正态分布，落在均值 ± 3σ 之外的数据点被判定为异常。


$$
异常判定：|x - μ| > 3σ
                覆盖率：正态分布下约 99.7% 的数据落在 [μ-3σ, μ+3σ] 内
$$


### 2.2 箱线图法（IQR法）


基于四分位数，不假设数据分布，更鲁棒。


$$
下界 = Q1 - 1.5 × IQR
                上界 = Q3 + 1.5 × IQR
                其中 IQR = Q3 - Q1（四分位距）
$$


### 2.3 Grubbs 检验


用于检测单变量数据集中是否存在单个异常值的统计检验方法。


$$
G = max|xi - x̄| / s
                临界值：Gcrit = (n-1)/√n × √t²/(n-2+t²)
                若 G > Gcrit，则最大偏差的点为异常值
$$


| 方法 | 优点 | 缺点 |
| --- | --- | --- |
| 3σ | 简单直观 | 要求正态分布，对离群值敏感 |
| IQR | 不需要分布假设，鲁棒 | 忽略时间顺序 |
| Grubbs | 有统计显著性保证 | 只能检测一个异常值 |


## 三、基于距离的方法


### 3.1 KNN 异常检测


计算每个点到其第K个最近邻的距离（K-distance），距离大的点为异常。适合检测点异常。


### 3.2 LOF（局部离群因子）


LOF通过比较一个点的局部密度与其邻居的局部密度来判断异常程度。


$$
LOFk(p) = ∑o∈Nk(p) lrdk(o) / (|Nk(p)| · lrdk(p))
                lrdk(p)：点p的局部可达密度
                LOF ≈ 1：正常点
                LOF > 1：可能是异常点
                LOF >> 1：高度异常
$$


> **Note:** **LOF的优势：**
> 能够处理不同密度的簇。在某些区域密度低但正常的点不会被误判为异常，而同一区域中密度显著偏低的点才会被检测出来。


## 四、基于重构的方法 — 自编码器异常检测


自编码器（Autoencoder）学习数据的低维表示，正常数据的重构误差小，异常数据的重构误差大。


### 4.1 基本原理


1. 用正常数据训练自编码器（LSTM-AE 或 Transformer-AE）
2. 计算训练集的重构误差分布
3. 设定阈值（如误差均值 + 3倍标准差）
4. 新数据的重构误差超过阈值则判定为异常


### 4.2 异常分数定义


$$
A(x) = ||x - AE(x)||²
                阈值 θ = μA + α · σA
                其中 μA 和 σA 为训练集异常分数的均值和标准差
$$


## 五、基于预测的方法与孤立森林


### 5.1 基于预测偏差法


用时间序列模型（ARIMA、LSTM等）做预测，如果实际值与预测值的偏差超过阈值，则判定为异常。核心思想是：模型学到正常模式后，异常会导致预测失败。


### 5.2 孤立森林（Isolation Forest）


孤立森林是一种高效的异常检测算法，核心思想是：异常点因为与大多数点不同，更容易被随机划分"孤立"。


$$
异常分数：
                s(x, n) = 2-E(h(x)) / c(n)
                h(x)：点x在随机树中的路径长度
                c(n) = 2H(n-1) - 2(n-1)/n（平均路径长度的归一化因子）
                s → 1：异常；s < 0.5：正常
$$


| 方法类别 | 代表算法 | 适用场景 | 复杂度 |
| --- | --- | --- | --- |
| 统计方法 | 3σ、IQR、Grubbs | 简单点异常 | O(n) |
| 基于距离 | KNN、LOF | 低维数据 | O(n²) |
| 基于重构 | LSTM-AE、VAE | 复杂序列模式 | O(n·d²) |
| 基于预测 | ARIMA残差、LSTM残差 | 有规律的时序 | 取决于模型 |
| 树方法 | 孤立森林 | 大数据集、高维 | O(n log n) |


## 六、Python 实战：自编码器异常检测


> **Example:** ### 示例：LSTM自编码器时间序列异常检测
>
>
> ```
> import numpy as np
> import torch
> import torch.nn as nn
> from torch.utils.data import DataLoader, TensorDataset
> import matplotlib.pyplot as plt
>
> # 1. 生成带异常的时间序列数据
> np.random.seed(42)
> n = 2000
> t = np.arange(n)
> # 正常模式：正弦波 + 趋势
> normal_data = np.sin(0.05 * t) + 0.001 * t + np.random.normal(0, 0.1, n)
>
> # 注入异常
> data = normal_data.copy()
> anomaly_indices = np.random.choice(range(500, 1800), 30, replace=False)
> data[anomaly_indices] += np.random.choice([-3, 3], 30) * np.random.uniform(1, 3, 30)
>
> # 2. 数据预处理 — 构造滑动窗口序列
> WINDOW = 30
>
> def create_windows(data, window):
>     windows = []
>     for i in range(len(data) - window):
>         windows.append(data[i:i + window])
>     return np.array(windows)
>
> # 用前1000个点（正常数据）构造训练集
> train_windows = create_windows(data[:1000], WINDOW)
> all_windows = create_windows(data, WINDOW)
>
> train_t = torch.FloatTensor(train_windows).unsqueeze(-1)
> all_t = torch.FloatTensor(all_windows).unsqueeze(-1)
>
> # 3. 定义LSTM自编码器
> class LSTMAutoencoder(nn.Module):
>     def __init__(self, input_size=1, hidden_size=32, num_layers=1):
>         super().__init__()
>         self.encoder = nn.LSTM(input_size, hidden_size, num_layers,
>                                batch_first=True)
>         self.decoder = nn.LSTM(hidden_size, hidden_size, num_layers,
>                                batch_first=True)
>         self.output = nn.Linear(hidden_size, input_size)
>
>     def forward(self, x):
>         # 编码
>         _, (h, c) = self.encoder(x)
>         # 重复编码状态作为decoder输入
>         decoder_input = h[-1].unsqueeze(1).repeat(1, x.size(1), 1)
>         decoder_out, _ = self.decoder(decoder_input, (h, c))
>         reconstructed = self.output(decoder_out)
>         return reconstructed
>
> # 4. 训练
> model = LSTMAutoencoder(hidden_size=32)
> criterion = nn.MSELoss(reduction='none')
> optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
>
> dataset = TensorDataset(train_t, train_t)
> loader = DataLoader(dataset, batch_size=64, shuffle=True)
>
> EPOCHS = 50
> for epoch in range(EPOCHS):
>     model.train()
>     total_loss = 0
>     for batch_x, _ in loader:
>         optimizer.zero_grad()
>         output = model(batch_x)
>         loss = criterion(output, batch_x).mean()
>         loss.backward()
>         optimizer.step()
>         total_loss += loss.item()
>     if (epoch + 1) % 10 == 0:
>         print(f"Epoch {epoch+1}/{EPOCHS}, Loss: {total_loss/len(loader):.6f}")
>
> # 5. 计算所有窗口的重构误差
> model.eval()
> with torch.no_grad():
>     reconstructed = model(all_t)
>     # 每个窗口的MSE作为异常分数
>     errors = criterion(reconstructed, all_t).mean(dim=(1, 2)).numpy()
>
> # 6. 设定阈值（训练集误差的均值+3倍标准差）
> train_errors = errors[:1000 - WINDOW]
> threshold = train_errors.mean() + 3 * train_errors.std()
> print(f"\n异常阈值: {threshold:.6f}")
>
> # 7. 检测异常
> is_anomaly = errors > threshold
> anomaly_count = is_anomaly.sum()
> print(f"检测到 {anomaly_count} 个异常窗口")
>
> # 8. 可视化
> fig, axes = plt.subplots(3, 1, figsize=(14, 10))
>
> axes[0].plot(data, alpha=0.7)
> axes[0].scatter(anomaly_indices, data[anomaly_indices],
>                color='red', s=30, label='真实异常', zorder=5)
> axes[0].set_title("原始时间序列（红色为注入的异常）")
> axes[0].legend()
>
> axes[1].plot(errors, alpha=0.7)
> axes[1].axhline(y=threshold, color='red', linestyle='--',
>                label=f'阈值={threshold:.4f}')
> axes[1].set_title("重构误差（异常分数）")
> axes[1].legend()
>
> # 标记检测到的异常
> detected_indices = np.where(is_anomaly)[0] + WINDOW // 2
> axes[2].plot(data, alpha=0.5)
> axes[2].scatter(detected_indices, data[detected_indices],
>                color='red', s=15, label='检测到的异常', zorder=5)
> axes[2].set_title("异常检测结果")
> axes[2].legend()
>
> plt.tight_layout()
> plt.savefig("anomaly_detection.png", dpi=150)
> plt.show()
>
> # 9. 评估
> true_positive = len(set(detected_indices) & set(anomaly_indices))
> precision = true_positive / max(len(detected_indices), 1)
> recall = true_positive / len(anomaly_indices)
> print(f"\n=== 检测评估 ===")
> print(f"精确率: {precision:.4f}")
> print(f"召回率: {recall:.4f}")
> ```


## 总结


- 时间序列异常分为点异常、上下文异常和集体异常三类
- 3σ和IQR方法简单高效，适合初步筛查
- LOF能够处理不同密度区域，适合局部异常检测
- 自编码器通过学习正常模式来检测异常，适合复杂序列模式
- 孤立森林以O(n log n)复杂度高效处理大数据集
- 异常阈值的设定需要平衡精确率和召回率
- 实际应用中通常组合多种方法以提高检测效果
