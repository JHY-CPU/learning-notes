# 时间序列基础与平稳性


## 一、时间序列的定义与组成


时间序列（Time Series）是按时间顺序排列的一系列观测值，记为 **{X~t~, t = 1, 2, ..., T}**。时间序列分析的核心目标是从历史数据中提取规律，用于预测未来值或理解数据的动态特性。


### 1.1 时间序列的四个分量


经典的时间序列分解模型将序列拆解为四个组成部分：


| 分量 | 符号 | 含义 | 示例 |
| --- | --- | --- | --- |
| **趋势（Trend）** | T~t~ | 长期的上升或下降趋势 | GDP逐年增长 |
| **季节性（Seasonality）** | S~t~ | 固定周期内的规律波动 | 冰淇淋夏季销量高 |
| **周期性（Cycle）** | C~t~ | 非固定周期的波动 | 经济周期 |
| **随机性（Noise）** | ε~t~ | 不可预测的随机波动 | 突发事件影响 |


$$
加法模型： Xt = Tt + St + Ct + εt
乘法模型： Xt = Tt × St × Ct × εt
$$


> **Note:** **选择提示：**
> 当季节性波动的幅度不随趋势变化时使用加法模型；当季节性波动的幅度随趋势增大而增大时使用乘法模型。乘法模型可通过取对数转化为加法模型。


## 二、平稳性（Stationarity）


平稳性是时间序列分析中最重要的概念之一。大多数经典时间序列模型都要求数据是平稳的。


### 2.1 严平稳（Strict Stationarity）


如果对于任意的时滞 k 和任意的时间点 t~1~, t~2~, ..., t~n~，联合分布 (X~t1~, X~t2~, ..., X~tn~) 与 (X~t1+k~, X~t2+k~, ..., X~tn+k~) 相同，则称该时间序列是严平稳的。严平稳要求所有阶矩都不随时间变化，条件非常严格。


### 2.2 弱平稳（Weak Stationarity）


在实际应用中，我们通常使用弱平稳的概念，也称为二阶平稳或协方差平稳：


$$
弱平稳的三个条件：
                ① E(Xt) = μ （均值为常数，不随时间变化）
                ② Var(Xt) = σ² （方差为常数，不随时间变化）
                ③ Cov(Xt, Xt+k) = γ(k) （自协方差只与时间间隔 k 有关，与时间 t 无关）
$$


> **Important:** **重要性：**
> 如果时间序列不平稳，会导致伪回归（Spurious Regression）问题——两个不相关的非平稳序列可能表现出极高的相关性，导致统计推断完全失效。


### 2.3 平稳性检验方法


| 检验方法 | 原假设 | 判别标准 |
| --- | --- | --- |
| ADF检验（Augmented Dickey-Fuller） | 存在单位根（非平稳） | p值 < 0.05 则拒绝原假设，认为平稳 |
| KPSS检验 | 序列是平稳的 | p值 < 0.05 则拒绝原假设，认为非平稳 |
| PP检验（Phillips-Perron） | 存在单位根（非平稳） | p值 < 0.05 则拒绝原假设，认为平稳 |


## 三、自相关函数（ACF）与偏自相关函数（PACF）


### 3.1 自协方差与自相关


自协方差函数衡量时间序列与其自身滞后值之间的线性关系：


$$
γ(k) = Cov(Xt, Xt+k) = E[(Xt - μ)(Xt+k - μ)]
$$


自相关函数（ACF）是自协方差的标准化形式：


$$
ρ(k) = γ(k) / γ(0) = Cov(Xt, Xt+k) / Var(Xt)
$$


### 3.2 ACF 与 PACF 的区别


| 指标 | 定义 | 反映的关系 | AR模型特征 | MA模型特征 |
| --- | --- | --- | --- | --- |
| **ACF** | X~t~ 与 X~t-k~ 的相关系数 | 直接+间接相关 | 拖尾（指数衰减） | p阶截尾 |
| **PACF** | 剔除中间变量后的偏相关系数 | 仅直接相关 | p阶截尾 | 拖尾（指数衰减） |


> **Note:** **模型识别口诀：**
> ACF截尾PACF拖尾 → MA模型；ACF拖尾PACF截尾 → AR模型；ACF和PACF都拖尾 → ARMA模型。截尾是指在某阶之后突然变为0（或在置信区间内）。


## 四、白噪声检验


白噪声（White Noise）是指均值为0、方差恒定且不同时间点的观测值互不相关的随机序列。白噪声序列不包含任何可预测的信息。


$$
白噪声序列满足：E(Xt) = 0, Var(Xt) = σ², Cov(Xt, Xs) = 0 (t ≠ s)
$$


### Ljung-Box 检验


Ljung-Box检验是最常用的白噪声检验方法，检验统计量为：


$$
Q = T(T+2) ∑k=1m ρ(k)² / (T-k) ∼ χ²(m)
$$


其中 T 为样本量，m 为检验的最大滞后阶数，ρ(k) 为 k 阶自相关系数。若 p值 < 0.05，则拒绝白噪声假设，说明序列存在自相关，有分析价值。


> **Important:** **实践要点：**
> 在进行任何时间序列建模之前，必须先检验原始序列是否为白噪声。如果是白噪声，则没有必要进一步建模分析。


## 五、Python 实战：ACF/PACF 分析


> **Example:** ### 示例：使用 statsmodels 进行时间序列分析
>
>
> ```
> import numpy as np
> import pandas as pd
> import matplotlib.pyplot as plt
> from statsmodels.tsa.stattools import adfuller, acf, pacf
> from statsmodels.graphics.tsaplots import plot_acf, plot_pacf
> from statsmodels.stats.diagnostic import acorr_ljungbox
>
> # 1. 生成模拟时间序列数据
> np.random.seed(42)
> n = 500
>
> # 非平稳序列（带趋势）
> trend = np.linspace(0, 50, n)
> seasonal = 10 * np.sin(2 * np.pi * np.arange(n) / 50)
> noise = np.random.normal(0, 3, n)
> ts_nonstationary = trend + seasonal + noise
>
> # 平稳序列（AR(1)过程）
> ts_stationary = np.zeros(n)
> for t in range(1, n):
>     ts_stationary[t] = 0.7 * ts_stationary[t-1] + np.random.normal(0, 1)
>
> # 2. ADF平稳性检验
> def adf_test(series, name="序列"):
>     result = adfuller(series, autolag='AIC')
>     print(f"\n=== {name} ADF检验 ===")
>     print(f"ADF统计量: {result[0]:.4f}")
>     print(f"p值: {result[1]:.6f}")
>     print(f"滞后阶数: {result[2]}")
>     for key, value in result[4].items():
>         print(f"临界值 ({key}): {value:.4f}")
>     if result[1] < 0.05:
>         print("结论: 序列是平稳的 (p < 0.05)")
>     else:
>         print("结论: 序列是非平稳的 (p >= 0.05)")
>
> adf_test(ts_nonstationary, "非平稳序列")
> adf_test(ts_stationary, "AR(1)平稳序列")
>
> # 差分后检验
> diff_series = np.diff(ts_nonstationary)
> adf_test(diff_series, "一阶差分后的序列")
>
> # 3. Ljung-Box白噪声检验
> lb_result = acorr_ljungbox(ts_stationary, lags=[10], return_df=True)
> print("\n=== Ljung-Box白噪声检验 ===")
> print(lb_result)
>
> # 4. 绘制ACF和PACF图
> fig, axes = plt.subplots(2, 2, figsize=(14, 10))
>
> # 非平稳序列
> plot_acf(ts_nonstationary, lags=40, ax=axes[0, 0], title="非平稳序列 ACF")
> plot_pacf(ts_nonstationary, lags=40, ax=axes[0, 1], title="非平稳序列 PACF")
>
> # 平稳序列
> plot_acf(ts_stationary, lags=40, ax=axes[1, 0], title="AR(1)平稳序列 ACF")
> plot_pacf(ts_stationary, lags=40, ax=axes[1, 1], title="AR(1)平稳序列 PACF")
>
> plt.tight_layout()
> plt.savefig("acf_pacf_analysis.png", dpi=150)
> plt.show()
>
> # 5. 计算并打印自相关系数
> acf_vals = acf(ts_stationary, nlags=10)
> pacf_vals = pacf(ts_stationary, nlags=10)
> print("\n=== 自相关与偏自相关系数 ===")
> print(f"{'滞后':<6} {'ACF':>10} {'PACF':>10}")
> print("-" * 28)
> for i in range(11):
>     print(f"{i:<6} {acf_vals[i]:>10.4f} {pacf_vals[i]:>10.4f}")
> ```


## 六、非平稳序列的处理方法


当时间序列不平稳时，需要对其进行变换以满足平稳性要求：


| 方法 | 适用场景 | 操作 |
| --- | --- | --- |
| 对数变换 | 方差随均值增大 | Y~t~ = ln(X~t~) |
| 一阶差分 | 存在线性趋势 | ∇X~t~ = X~t~ - X~t-1~ |
| 二阶差分 | 存在二次趋势 | ∇²X~t~ = ∇X~t~ - ∇X~t-1~ |
| 季节差分 | 存在季节性 | ∇~s~X~t~ = X~t~ - X~t-s~ |
| Box-Cox变换 | 方差不稳定 | 自动选择最优变换参数 λ |


> **Note:** **差分阶数 d 的选择：**
> 对原始序列依次做1阶、2阶差分，每次差分后进行ADF检验，直到序列变为平稳。一般 d ≤ 2，过多差分会导致过度拟合和信息损失。


## 总结


- 时间序列由趋势、季节性、周期性和随机性四个分量组成
- 弱平稳要求均值、方差恒定，自协方差仅与时间间隔有关
- ADF检验是最常用的平稳性检验方法
- ACF/PACF是识别ARIMA模型阶数的核心工具
- 差分是处理非平稳序列的基本手段
- 白噪声检验是时间序列分析的第一步——确认数据是否值得分析
