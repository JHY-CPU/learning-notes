## 第1题 (2分)
以下关于线性表的叙述中，正确的是（）

A. 顺序表可以实现随机访问，链表不可以
B. 链表的插入和删除操作一定比顺序表快
C. 顺序表和链表都可以在O(1)时间内获得表长
D. 顺序表要求连续的存储空间，链表不需要

**答案：D**

**解析：** 顺序表使用连续的存储空间存储数据元素，链表通过指针将分散的结点链接起来，不需要连续存储空间。A选项不完全正确——链表虽然不能通过下标直接访问，但也可以按序号查找实现随机访问，只是时间复杂度为O(n)。B选项错误——链表的插入删除虽然不需要移动元素，但需要先找到插入位置，查找的时间可能比顺序表的移动还长。C选项正确——顺序表有length字段，链表也可以维护length字段，都能在O(1)获得表长。但D更准确地描述了存储结构的本质区别。因此D正确。

## 第2题 (2分)
设一个双端队列（Deque）初始为空，元素1, 2, 3, 4, 5依次进行以下操作：从前端插入1，从前端插入2，从后端插入3，从前端插入4，从后端插入5。若从前到后遍历该队列，输出序列为（）

A. 4, 2, 1, 3, 5  B. 2, 4, 1, 3, 5  C. 1, 2, 3, 4, 5  D. 5, 3, 1, 2, 4

**答案：A**

**解析：** 逐个模拟操作（从前端插入意味着新元素成为第一个）：
- 前端插入1：队列=[1]
- 前端插入2：队列=[2, 1]
- 后端插入3：队列=[2, 1, 3]
- 前端插入4：队列=[4, 2, 1, 3]
- 后端插入5：队列=[4, 2, 1, 3, 5]

从前到后遍历输出：4, 2, 1, 3, 5。因此A正确。

## 第3题 (2分)
将中缀表达式a+b*(c-d)-e/f转换为后缀表达式，结果为（）

A. abcd-*+ef/-  B. abc-d*+ef/-  C. ab+cd-*-ef/  D. abcd-*+ef/-

**答案：A**

**解析：** 使用栈将中缀表达式转换为后缀表达式的过程：
- a → 输出a
- + → 栈[+]
- b → 输出ab
- * → 栈[+,*]
- ( → 栈[+,*,(]
- c → 输出abc
- - → 栈[+,*,(,-]
- d → 输出abcd
- ) → 弹出到( → 输出abcd-，栈[+,*]
- *弹出 → 输出abcd-*，栈[+]（因为-优先级低于*，先不弹+）
- - → 弹出+ → 输出abcd-*+，栈[-]
- e → 输出abcd-*+e
- / → 栈[-,/]
- f → 输出abcd-*+ef
- 结束 → 弹出栈中剩余 → 输出abcd-*+ef/-

结果为：abcd-*+ef/-。因此A或B？让我重新检查。

更精确的过程：
1. 输出a
2. +入栈
3. 输出b → ab
4. *入栈（优先级高于+）
5. (入栈
6. 输出c → abc
7. -入栈
8. 输出d → abcd
9. ) 弹出到( → 输出- → abcd-，栈[+,*]
10. 遇到-（减号），*优先级高于-，弹出*，输出* → abcd-*，+优先级等于-，弹出+，输出+ → abcd-*+，-入栈
11. 输出e → abcd-*+e
12. /入栈（优先级高于-）
13. 输出f → abcd-*+ef
14. 弹出/ → abcd-*+ef/，弹出- → abcd-*+ef/-

结果：**abcd-*+ef/-**。因此A正确。

     B

- B的左子：后序D，中序D，根D。

但是选项中有ABCDEF。让我验证后序：D、E、B、F、C、A = DEBFCA ✓。中序：D、B、E、A、C、F = DBEACF ✓。

## 第6题 (2分)
设有向图G有n个顶点和e条边。若采用邻接矩阵存储，则求单源最短路径的Dijkstra算法的时间复杂度为（）

A. O(n)  B. O(n²)  C. O(n+e)  D. O(n×e)

**答案：B**

**解析：** 采用邻接矩阵存储的Dijkstra算法中，每一轮从未确定的顶点中找出距离最小的顶点（O(n)），然后对其邻接点进行松弛操作（O(n)）。共进行n轮，总时间复杂度为O(n²)。若采用邻接表存储并配合优先队列（最小堆），时间复杂度可优化为O((n+e)log n)。因此B正确。

## 第7题 (2分)
设有无向图G，使用Prim算法从顶点A出发构造最小生成树。若图G有n个顶点和e条边，使用邻接矩阵存储，则Prim算法的时间复杂度为（）

A. O(n)  B. O(n²)  C. O(elog e)  D. O(nlog n)

**答案：B**

**解析：** Prim算法使用邻接矩阵存储时，每次选择一个距已选集合最近的顶点加入，需要扫描距离数组（O(n)），然后更新未选顶点到已选集合的距离（O(n)）。共进行n轮，总时间复杂度为O(n²)。若使用邻接表加最小堆，可优化到O(e log n)。因此B正确。

## 第8题 (2分)
对关键字序列{25, 84, 21, 47, 15, 27, 68, 35, 20}进行堆排序（从小到大排序，建立大根堆）。建堆完成后，堆顶元素和第一趟排序后序列的最后一个元素分别是（）

A. 84, 15  B. 84, 20  C. 68, 25  D. 25, 84

**答案：B（84, 20）**

**解析：** 建立大根堆后，堆顶元素为关键字最大的元素，即84。第一趟排序：将堆顶84与最后一个元素交换，然后调整剩余元素为大根堆。原序列最后一个元素与84交换后，84到达序列末尾。第一趟排序后的最后位置是84。但题目问的是第一趟排序后序列的最后一个元素。

等等，堆排序的过程：建堆后84在堆顶。第一趟：交换堆顶(84)和堆末，堆缩小1，调整。84到达最终位置（数组末尾）。然后继续第二趟... 但题目说的是"第一趟排序后序列的最后一个元素"。

堆排序每趟将堆顶（最大值）放到数组末尾。第一趟后，数组最后一个位置是84。但选项中没有84作为最后一个元素的。

让我重新理解题目：建堆完成后堆顶元素=84。第一趟排序：将堆顶与当前堆的最后一个元素交换，堆缩小1，调整。交换后原来的堆顶84放到位置n-1。堆中剩余n-1个元素调整为大根堆。

"第一趟排序后序列的最后一个元素"应该是指整个序列（含已排序部分）的最后一个元素。第一趟后84在最后位置。

选项A说堆顶84，最后元素15。但第一趟后84应该在最后。

等等，题目可能理解为建堆后的堆顶，和第一趟排序交换前的堆的最后一个元素（与84交换的那个元素）。

建堆后，堆的最后一个元素是哪个？需要建立大根堆后确定。在建立大根堆过程中，最后一个位置的元素会发生变化。

让我算一下建立大根堆后的序列。初始：[25, 84, 21, 47, 15, 27, 68, 35, 20]

用数组表示的完全二叉树（下标从0）：
```
          25(0)
        /       \
      84(1)     21(2)
     /   \      /   \
   47(3) 15(4) 27(5) 68(6)
   / \
  35(7) 20(8)
```

从最后一个非叶子结点开始调整（n/2-1=3）：
- i=3: 47的子35,20，47>35和20，不用调整
- i=2: 21的子27,68，68>21，交换21和68
  [25, 84, 68, 47, 15, 27, 21, 35, 20]
- i=1: 84的子47,15，84>47和15，不用调整
- i=0: 25的子84,68，84>25，交换25和84
  [84, 25, 68, 47, 15, 27, 21, 35, 20]
  调整位置1: 25的子47,15，47>25，交换
  [84, 47, 68, 25, 15, 27, 21, 35, 20]
  调整位置3: 25的子35,20，35>25，交换
  [84, 47, 68, 35, 15, 27, 21, 25, 20]

大根堆：[84, 47, 68, 35, 15, 27, 21, 25, 20]。堆顶=84，堆的最后一个元素=20。

第一趟排序：交换堆顶84和堆末20→[20,47,68,35,15,27,21,25,84]，调整前8个元素为大根堆。

但题目说"第一趟排序后序列的最后一个元素"。第一趟后整个序列的最后元素是84。不在选项中。

如果理解为"与堆顶交换的那个元素（原来堆的最后一个元素）"，那就是20。**答案B：84, 20**。

但更合理的理解：建堆后堆顶=84，第一趟排序将84放到最终位置，最后一个位置的元素变为84。题目选项的解读可能是"建堆后的堆顶元素和建堆后堆的最后一个元素"。建堆后堆顶=84，最后一个=20。

**答案：B（84, 20）**

## 第10题 (2分)
在平衡二叉树（AVL树）中，插入一个新结点后可能导致不平衡。若不平衡是由某结点的左孩子的右子树过高引起的，应进行（）

A. LL型旋转  B. RR型旋转  C. LR型旋转  D. RL型旋转

**答案：C**

**解析：** AVL树的四种旋转类型：
- LL型：左孩子的左子树过高，进行右单旋转。
- RR型：右孩子的右子树过高，进行左单旋转。
- LR型：左孩子的右子树过高，先对左孩子左旋，再对当前结点右旋（双旋转）。
- RL型：右孩子的左子树过高，先对右孩子右旋，再对当前结点左旋（双旋转）。

题目描述"左孩子的右子树过高"属于LR型。因此C正确。

## 第11题 (2分)
在B-树中，若一个结点有n个关键字，则该结点有（）个子树

A. n  B. n+1  C. n-1  D. 2n

**答案：B**

**解析：** B-树的定义：每个非根结点至少有⌈m/2⌉-1个关键字，至多有m-1个关键字。每个有n个关键字的非叶子结点恰有n+1个子树（n+1棵子树之间被n个关键字分隔）。这是B-树的基本性质。因此B正确。
