## 第1题 (2分)
下列关于算法时间复杂度的叙述中，正确的是（）

A. 算法的时间复杂度取决于问题的规模和待处理数据的初始状态
B. 算法的时间复杂度是指执行算法所需要的绝对时间
C. 同一个算法用不同的语言实现，其时间复杂度一定不同
D. 算法的时间复杂度与编译程序的性能无关

**答案：A**

**解析：** 算法的时间复杂度不仅取决于问题的规模n，还与待处理数据的初始状态有关（例如快速排序的最坏情况与数据初始排列有关）。时间复杂度是用基本操作的执行次数来衡量的，而非绝对时间，绝对时间还与机器性能、编译程序等因素有关。用不同语言实现同一算法，其渐进时间复杂度是一样的。因此A正确，B、C、D错误。

## 第2题 (2分)
设有一个带头结点的单链表，头指针为head，要在p所指结点之后插入一个由q所指的新结点，则正确的操作序列是（）

A. q->next = p->next; p->next = q;
B. p->next = q; q->next = p->next;
C. q->next = p; p->next = q;
D. p->next = q->next; q->next = p->next;

**答案：A**

**解析：** 在p所指结点之后插入q所指结点，需要先将q的next指向p的后继结点(q->next = p->next)，再将p的next指向q(p->next = q)。这两个步骤的顺序不能颠倒，否则p的原后继结点会丢失。选项B先修改了p->next，再执行q->next = p->next时，q->next会指向q自身，形成环。因此A正确。

## 第3题 (2分)
一个栈的入栈序列为1, 2, 3, ..., n，其出栈序列为p1, p2, p3, ..., pn。若p1 = n，则pi为（）

A. i  B. n-i  C. n-i+1  D. 不确定

**答案：C**

**解析：** 若p1 = n，说明n是第一个出栈的元素。由于入栈序列为1, 2, 3, ..., n，要使n第一个出栈，必须将1, 2, ..., n全部入栈后再依次出栈。因此出栈序列为n, n-1, n-2, ..., 2, 1，即pi = n - i + 1。因此C正确。

## 第4题 (2分)
已知循环队列的存储空间为数组data[0..20]，front指向队头元素的前一个位置，rear指向队尾元素。若front = 8，rear = 3，则队列中元素的个数为（）

A. 12  B. 13  C. 15  D. 16

**答案：C**

**解析：** 循环队列中元素个数的计算公式为：(rear - front + MAXSIZE) % MAXSIZE。其中MAXSIZE = 21（数组下标0到20共21个位置）。代入得：(3 - 8 + 21) % 21 = 16 % 21 = 16。等等，让我重新计算：rear=3指向队尾，front=8指向队头前一个，有效存储空间从front+1=9到rear=3（循环），个数为(3-8+21)=16。但若使用另一种约定（牺牲一个存储单元区分队空队满），MAXSIZE仍为21。重新按标准公式：(rear - front + MAXSIZE) % MAXSIZE = (3 - 8 + 21) % 21 = 16。因此队列中元素个数为16个。答案修正为D。

重新审视：若数组大小为21（下标0~20），(rear - front + 21) % 21 = (3-8+21) % 21 = 16 % 21 = 16。

**答案修正：D（16个）**

## 第5题 (2分)
将一棵二叉树的前序遍历序列和中序遍历序列分别存入数组A和数组B中。已知前序序列为{1,2,4,5,3,6,7}，中序序列为{4,2,5,1,6,3,7}，则该二叉树的后序遍历序列为（）

A. 4,5,2,6,7,3,1  B. 4,2,5,6,3,7,1  C. 4,5,2,6,3,7,1  D. 4,2,5,7,6,3,1

**答案：A**

**解析：** 由前序遍历可知根结点为1。在中序遍历中找到1的位置，左半部分{4,2,5}为左子树的中序序列，右半部分{6,3,7}为右子树的中序序列。前序中1后面跟着{2,4,5}（左子树前序）和{3,6,7}（右子树前序）。递归分析：左子树根为2，2的左子为4，右子为5；右子树根为3，3的左子为6，右子为7。后序遍历为：左子树后序(4,5,2) + 右子树后序(6,7,3) + 根(1) = 4,5,2,6,7,3,1。因此A正确。

## 第6题 (2分)
一棵完全二叉树有2022个结点，则其叶子结点的个数为（）

A. 505  B. 506  C. 1011  D. 1012

**答案：B**

**解析：** 完全二叉树有n=2022个结点。度为1的结点个数n1的取值为0或1。由n = n0 + n1 + n2 且 n0 = n2 + 1（二叉树性质），得n = 2n0 - 1 + n1，即2022 = 2n0 - 1 + n1。若n1 = 0，则2n0 = 2023，n0非整数，不合理。若n1 = 1，则2n0 = 2022，n0 = 1011。但这是完全二叉树，n=2022为偶数，所以最后一个是左孩子，n1=1。n0 = (n+1)/2 = 2023/2不整除。换公式：n0 = floor((n+1)/2) = floor(2023/2) = 1011，n1 = n - 2*n0 + 1 = 2022 - 2022 + 1 = 1。所以叶子结点个数为1011。但需要再核实。

更简洁的方法：完全二叉树叶子结点个数n0 = ceil(n/2) = ceil(2022/2) = 1011。

**答案修正：C（1011）**

## 第7题 (2分)
对n个互不相同的元素进行冒泡排序，在最好情况下的比较次数为（）

A. n  B. n-1  C. n(n-1)/2  D. n(n+1)/2

**答案：B**

**解析：** 冒泡排序的最好情况是待排序序列已经有序。此时只需进行一趟冒泡，进行n-1次相邻元素的比较，如果没有发生交换则排序结束。因此最好情况下比较次数为n-1。因此B正确。

## 第8题 (2分)
设有向图G的邻接矩阵A，若A[i][j] = 1，则说明（）

A. 顶点i到顶点j之间有一条边  B. 顶点i和顶点j之间有一条边
C. 顶点j到顶点i之间有一条边  D. 不能确定

**答案：A**

**解析：** 有向图的邻接矩阵A中，A[i][j] = 1表示从顶点i到顶点j存在一条有向边（弧）。有向图的边有方向性，A[i][j] = 1不能说明存在从j到i的边。因此A正确。

## 第9题 (2分)
对线性表进行折半查找时，要求线性表必须（）

A. 以顺序方式存储  B. 以链接方式存储
C. 以顺序方式存储，且数据元素有序  D. 以链接方式存储，且数据元素有序

**答案：C**

**解析：** 折半查找（二分查找）需要能够通过下标随机访问中间元素，因此必须使用顺序存储结构（数组）。同时，折半查找的前提是数据元素必须按关键字有序排列。因此C正确。链接方式存储无法实现随机访问，不适用于折半查找。

## 第10题 (2分)
已知散列表的表长为11，散列函数为H(key) = key % 11。用线性探测法处理冲突，将关键字序列{15, 36, 57, 22, 18, 43}依次插入空散列表中，则元素43的散列地址为（）

A. 5  B. 6  C. 8  D. 10

**答案：D**

**解析：** 逐步插入：
- 15: H(15) = 15 % 11 = 4，地址4为空，插入4。
- 36: H(36) = 36 % 11 = 3，地址3为空，插入3。
- 57: H(57) = 57 % 11 = 2，地址2为空，插入2。
- 22: H(22) = 22 % 11 = 0，地址0为空，插入0。
- 18: H(18) = 18 % 11 = 7，地址7为空，插入7。
- 43: H(43) = 43 % 11 = 10，地址10为空，插入10。

因此43的散列地址为10。D正确。

## 第11题 (2分)
对序列{50, 30, 70, 20, 40, 60, 80}建立大根堆，从最后一个非叶子结点开始调整，最终得到的大根堆的根结点值为（）

A. 50  B. 70  C. 80  D. 60

**答案：C**

**解析：** 对序列建立大根堆，需要从最后一个非叶子结点（下标n/2 = 3，值为20）开始，自下而上进行调整（筛选）。
- 调整位置3（值20，左子40，右子60）：20与60交换→{50,30,70,20,40,60,80}不对，需要重新梳理。

将序列看作完全二叉树（下标从1开始）：
```
       50(1)
      /     \
   30(2)    70(3)
   / \      / \
 20(4) 40(5) 60(6) 80(7)
```
- 从i=3开始，70的子结点60和80，80>70，交换→80放在3，70放在7
- i=2，30的子结点20和40，40>30，交换→40放在2，30放在5
- i=1，50的子结点40（位置2）和80（位置3），80>50，交换→80放在1，50放在3
- 调整位置3，50的子结点60和70，70>50，交换→70放在3，50放在7

最终大根堆：{80, 40, 70, 20, 30, 60, 50}，根结点值为80。C正确。
