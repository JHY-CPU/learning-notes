# 53-软件过程与管理

软件过程与管理关注如何高效、可控地组织和管理软件开发活动。本模块覆盖从传统到敏捷的各类过程模型、项目管理方法和软件度量体系。

## 目录结构

| 子目录 | 主题 | 说明 |
|--------|------|------|
| [01-过程模型](./01-过程模型/README.md) | 过程模型 | 瀑布、敏捷、DevOps等各类软件开发过程模型 |
| [02-项目管理](./02-项目管理/README.md) | 项目管理 | WBS、甘特图、风险管理、敏捷估算等项目管理方法 |
| [03-软件度量](./03-软件度量/README.md) | 软件度量 | 代码度量、功能点、DORA指标、CMMI等度量体系 |

## 学习路径

1. **过程模型**：理解不同过程模型的原理和适用场景
2. **项目管理**：掌握项目管理的核心工具和方法
3. **软件度量**：学会用数据度量和改进软件开发过程
