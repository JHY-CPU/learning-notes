# 资产管线与流水线

## 核心概念

资产管线是将美术资源从制作工具导入游戏引擎的标准化流程。自动化程度直接影响团队效率。一个10人美术团队每天可能产生数百个资源文件，如果每个都需要手动配置导入设置，将浪费大量时间且容易出错。

### Asset导入规范

- **统一命名规范**：`类型_名称_变体`格式，如`CHR_Hero_01`、`TEX_Wall_Diffuse`、`MAT_Metal_Shiny`
- **标准目录结构**：按类型和功能分目录组织
- **预设导入设置**：根据目录自动应用纹理压缩、模型缩放、动画设置
- **自动化标签和分组**：Addressables Group自动管理

### 命名规范参考

```
前缀:
CHR_ = 角色(Character)
ENV_ = 环境(Environment)
PROP_ = 道具(Prop)
TEX_ = 纹理(Texture)
MAT_ = 材质(Material)
SFX_ = 音效(Sound Effect)
BGM_ = 背景音乐
ANM_ = 动画(Animation)
VFX_ = 视觉特效
UI_ = UI元素
SHD_ = Shader

后缀:
_D = Diffuse贴图
_N = Normal贴图
_M = Metallic贴图
_R = Roughness贴图
_AO = Ambient Occlusion贴图
_E = Emission贴图
_LOD0/LOD1/LOD2 = LOD级别
```

## 具体实现方法

### 自动化导入处理器（完整版）

```csharp
#if UNITY_EDITOR
using UnityEditor;
using UnityEngine;

/// <summary>
/// 自动化资源导入处理器
/// 根据资源路径自动应用导入设置
/// </summary>
public class AssetImportHandler : AssetPostprocessor
{
    // === 纹理导入自动设置 ===
    void OnPreprocessTexture()
    {
        TextureImporter importer = (TextureImporter)assetImporter;
        string path = assetPath.ToLower();

        // UI纹理
        if (path.Contains("/ui/") || path.Contains("/sprites/"))
        {
            importer.textureType = TextureImporterType.Sprite;
            importer.mipmapEnabled = false;
            importer.alphaIsTransparency = true;
            importer.maxTextureSize = 1024;
            SetPlatformSettings(importer, "Android", 1024, TextureImporterFormat.ASTC_6x6);
            SetPlatformSettings(importer, "iPhone", 1024, TextureImporterFormat.ASTC_6x6);
        }
        // 3D纹理
        else if (path.Contains("/3d/") || path.Contains("/textures/"))
        {
            importer.textureType = TextureImporterType.Default;
            importer.mipmapEnabled = true;
            importer.maxTextureSize = 2048;
            importer.wrapMode = TextureWrapMode.Repeat;
            SetPlatformSettings(importer, "Android", 2048, TextureImporterFormat.ASTC_6x6);
            SetPlatformSettings(importer, "iPhone", 2048, TextureImporterFormat.ASTC_6x6);
        }
        // 法线贴图
        else if (path.Contains("_n.") || path.Contains("normal"))
        {
            importer.textureType = TextureImporterType.NormalMap;
            importer.mipmapEnabled = true;
            importer.maxTextureSize = 2048;
        }
        // Lightmap
        else if (path.Contains("lightmap"))
        {
            importer.textureType = TextureImporterType.Default;
            importer.mipmapEnabled = false;
            importer.wrapMode = TextureWrapMode.Clamp;
            importer.maxTextureSize = 2048;
        }
        // HDR/Cubemap
        else if (path.Contains("skybox") || path.Contains("cubemap"))
        {
            importer.textureType = TextureImporterType.Default;
            importer.textureShape = TextureImporterShape.TextureCube;
            importer.mipmapEnabled = false;
        }
    }

    // === 模型导入自动设置 ===
    void OnPreprocessModel()
    {
        ModelImporter importer = (ModelImporter)assetImporter;
        string path = assetPath.ToLower();

        // 角色模型
        if (path.Contains("/characters/"))
        {
            importer.importAnimation = true;
            importer.animationType = ModelImporterAnimationType.Human;
            importer.materialImportMode = ModelImporterMaterialImportMode.None;
            importer.importNormals = ModelImporterNormals.Import;
            importer.importTangents = ModelImporterTangents.Import;
            importer.optimizeMesh = true;
        }
        // 道具模型
        else if (path.Contains("/props/"))
        {
            importer.importAnimation = false;
            importer.generateSecondaryData = true;
            importer.importNormals = ModelImporterNormals.Calculate;
            importer.materialImportMode = ModelImporterMaterialImportMode.None;
            importer.optimizeMesh = true;
        }
        // 环境模型
        else if (path.Contains("/environment/"))
        {
            importer.importAnimation = false;
            importer.generateSecondaryData = true;
            importer.importNormals = ModelImporterNormals.Import;
            importer.materialImportMode = ModelImporterMaterialImportMode.None;
            importer.optimizeMesh = true;
            // 环境模型通常需要合并
            importer.isReadable = false; // 关闭Read/Write节省内存
        }
    }

    // === 音频导入自动设置 ===
    void OnPreprocessAudio()
    {
        AudioImporter importer = (AudioImporter)assetImporter;
        string path = assetPath.ToLower();

        if (path.Contains("/bgm/") || path.Contains("/music/"))
        {
            // BGM: Streaming模式，中等质量
            importer.loadType = AudioClipLoadType.Streaming;
            importer.preloadAudioData = false;
            SetAudioCompression(importer, 70);
        }
        else if (path.Contains("/sfx/") || path.Contains("/sounds/"))
        {
            // SFX: Decompress on Load，高质量
            importer.loadType = AudioClipLoadType.DecompressOnLoad;
            importer.preloadAudioData = true;
            SetAudioCompression(importer, 100);
        }
        else if (path.Contains("/ambient/"))
        {
            // 环境音: Streaming模式
            importer.loadType = AudioClipLoadType.Streaming;
            importer.preloadAudioData = false;
            SetAudioCompression(importer, 60);
        }
    }

    void SetPlatformSettings(TextureImporter importer, string platform,
        int maxSize, TextureImporterFormat format)
    {
        var settings = new TextureImporterPlatformSettings();
        settings.name = platform;
        settings.maxTextureSize = maxSize;
        settings.format = format;
        settings.compressionQuality = 50;
        settings.overridden = true;
        importer.SetPlatformTextureSettings(settings);
    }

    void SetAudioCompression(AudioImporter importer, int quality)
    {
        var settings = importer.defaultSampleSettings;
        settings.compressionQuality = quality;
        importer.defaultSampleSettings = settings;
    }
}
#endif
```

### 资源命名规范检查器

```csharp
#if UNITY_EDITOR
using UnityEditor;
using UnityEngine;
using System.Collections.Generic;
using System.IO;
using System.Text.RegularExpressions;

/// <summary>
/// 资源命名规范检查工具
/// 自动检查所有资源是否符合命名规范
/// </summary>
public class NamingConventionChecker : EditorWindow
{
    [MenuItem("Tools/Asset Pipeline/Naming Convention Checker")]
    static void Open() => GetWindow<NamingConventionChecker>();

    private List<Violation> violations = new List<Violation>();

    struct Violation
    {
        public string path;
        public string reason;
        public MessageType severity;
    }

    void OnGUI()
    {
        if (GUILayout.Button("检查所有资源"))
            CheckAllAssets();

        EditorGUILayout.Space();
        EditorGUILayout.LabelField($"发现 {violations.Count} 个问题");

        foreach (var v in violations)
        {
            EditorGUILayout.HelpBox($"{v.path}\n{v.reason}", v.severity);
        }
    }

    void CheckAllAssets()
    {
        violations.Clear();

        string[] guids = AssetDatabase.FindAssets("",
            new[] { "Assets/Art", "Assets/Audio", "Assets/Prefabs" });

        foreach (string guid in guids)
        {
            string path = AssetDatabase.GUIDToAssetPath(guid);
            if (AssetDatabase.IsValidFolder(path)) continue;

            string fileName = Path.GetFileNameWithoutExtension(path);

            // 检查命名规范
            if (!Regex.IsMatch(fileName, @"^[A-Z][a-zA-Z0-9_]+$"))
            {
                violations.Add(new Violation
                {
                    path = path,
                    reason = "命名不规范：应使用PascalCase，仅包含字母数字下划线",
                    severity = MessageType.Warning
                });
            }

            // 检查是否有中文
            if (Regex.IsMatch(fileName, @"[一-龥]"))
            {
                violations.Add(new Violation
                {
                    path = path,
                    reason = "文件名包含中文，建议使用英文",
                    severity = MessageType.Warning
                });
            }

            // 检查空格
            if (fileName.Contains(" "))
            {
                violations.Add(new Violation
                {
                    path = path,
                    reason = "文件名包含空格，建议使用下划线",
                    severity = MessageType.Warning
                });
            }
        }

        Debug.Log($"命名检查完成: {violations.Count} 个问题");
    }
}
#endif
```

### Addressables Group管理规范

```
推荐的Group结构:
├── Default Local Group (本地必须资源，打包时内嵌)
│   ├── 核心Shader
│   ├── 启动场景资源
│   └── 全局配置
├── Characters (角色模型、动画)
├── Environments (场景环境资源，按区域分组)
│   ├── Env_Forest
│   ├── Env_Dungeon
│   └── Env_Castle
├── UI_Sprites (UI图集)
├── Audio_BGM (背景音乐)
├── Audio_SFX (音效)
├── Shared_Materials (共享材质和Shader)
└── Preload_Common (预加载的公共资源)
```

### 自动化打包脚本

```csharp
#if UNITY_EDITOR
using UnityEditor;
using System.Linq;

/// <summary>
/// 自动化打包脚本
/// 支持一键打包和CI/CD集成
/// </summary>
public class BuildAutomation
{
    [MenuItem("Build/Android Release")]
    static void BuildAndroid()
    {
        PlayerSettings.Android.bundleVersionCode++;
        BuildPlayerOptions options = new BuildPlayerOptions
        {
            scenes = GetEnabledScenes(),
            locationPathName = "Builds/Android/game.apk",
            target = BuildTarget.Android,
            options = BuildOptions.None
        };
        BuildPipeline.BuildPlayer(options);
    }

    [MenuItem("Build/iOS Release")]
    static void BuildiOS()
    {
        PlayerSettings.iOS.buildNumber =
            (int.Parse(PlayerSettings.iOS.buildNumber) + 1).ToString();
        BuildPlayerOptions options = new BuildPlayerOptions
        {
            scenes = GetEnabledScenes(),
            locationPathName = "Builds/iOS",
            target = BuildTarget.iOS,
            options = BuildOptions.None
        };
        BuildPipeline.BuildPlayer(options);
    }

    [MenuItem("Build/Windows Release")]
    static void BuildWindows()
    {
        BuildPlayerOptions options = new BuildPlayerOptions
        {
            scenes = GetEnabledScenes(),
            locationPathName = "Builds/Windows/game.exe",
            target = BuildTarget.StandaloneWindows64,
            options = BuildOptions.None
        };
        BuildPipeline.BuildPlayer(options);
    }

    static string[] GetEnabledScenes()
    {
        return EditorBuildSettings.scenes
            .Where(s => s.enabled)
            .Select(s => s.path)
            .ToArray();
    }
}
#endif
```

## 最佳实践

- 建立团队统一的命名规范并用工具强制执行
- 纹理压缩格式按平台预设，避免手动设置
- 使用AssetPostprocessor自动应用导入规则
- 定期运行资源审计工具检查命名、冗余和未使用资源
- Addressables Group按功能模块划分，便于管理和热更新
- 模型导入时关闭Read/Write以节省运行时内存
- 音频按类型设置不同的加载模式和压缩质量

## 常见陷阱与修复

**陷阱1：导入设置不统一**
- 症状：同一类型的资源压缩格式不同，包体不一致
- 修复：使用AssetPostprocessor根据目录自动设置

**陷阱2：资源命名随意**
- 症状：查找和管理困难
- 修复：使用命名规范检查工具定期审计

**陷阱3：循环依赖导致Addressables打包失败**
- 症状：打包时报循环依赖错误
- 修复：共享资源（材质、Shader）独立分组，避免交叉引用

**陷阱4：未设置平台特定的纹理压缩格式**
- 症状：移动端包体过大
- 修复：使用SetPlatformTextureSettings为Android/iOS分别设置ASTC

**陷阱5：AssetPostprocessor逻辑错误批量修改了不该修改的资源**
- 症状：大量资源的导入设置被错误修改
- 修复：添加严格的路径判断，修改前确认路径匹配规则
