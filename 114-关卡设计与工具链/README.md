# 114-关卡设计与工具链

## 模块概述

本模块涵盖游戏关卡设计的核心理念与工具链开发，从设计原则到编辑器扩展，全面掌握关卡制作流程。

## 目录

1. 关卡设计原则 - 引导设计、节奏控制、难度曲线、玩家心流理论
2. 地图编辑器使用 - Tilemap/地形系统/笔刷工具、Unity与UE实现
3. 程序化生成(Procedural Generation) - 随机地牢、Wave Function Collapse
4. 关卡白盒与灰盒 - Blockout流程、碰撞占位、快速迭代
5. 触发器与事件区域 - 区域触发、剧情触发、检查点存档
6. 光照与氛围设计 - 实时vs烘焙光照、Light Probe、体积雾
7. 环境音效与音乐触发 - BGM切换、环境音混合、音频遮挡
8. 编辑器扩展开发 - Unity EditorWindow、UE Editor Utility Widget
9. 资产管线与流水线 - Asset导入设置、自动化脚本、资源规范
10. 关卡测试与迭代 - Playtest方法、遥测数据、热力图分析
