# 环境音效与音乐触发

## 核心概念

声音设计是关卡沉浸感的重要组成部分，合理的音频系统让玩家通过听觉感知空间和状态变化。研究表明，玩家对视觉质量的感知有30%受声音影响。好的音频设计可以让简单的画面显得高级，差的音频设计会让精美的画面显得廉价。

### 音频系统架构

```
音频管理器(AudioManager)
├── BGM管理器(BGMManager) - 背景音乐
│   ├── 探索层
│   ├── 战斗层
│   └── Boss层
├── 环境音管理器(AmbientManager) - 环境音
│   ├── 风声层
│   ├── 鸟鸣层
│   └── 水声层
├── SFX管理器(SFXManager) - 音效
│   ├── 3D音效(空间化)
│   └── 2D音效(UI/系统)
└── Audio Mixer - 混音控制
    ├── Master
    ├── BGM Group
    ├── SFX Group
    └── Ambient Group
```

### 区域BGM切换策略

- 不同区域播放不同的背景音乐
- 区域之间实现平滑交叉过渡（Cross Fade），避免音乐断裂
- 战斗BGM与探索BGM动态切换，使用多层音轨实现无缝过渡
- 音乐Stinger：特定事件触发的短音乐片段（获得宝物、进入新区域）

### 环境音层混合

- 多个环境音源按权重混合
- 根据玩家位置和状态动态调整各层音量
- 使用AnimationCurve控制昼夜周期的音量变化（早晨鸟鸣增加，夜晚虫鸣增加）

### 音频遮挡（Audio Occlusion）

- 墙壁等障碍物阻挡声音传播
- 被遮挡的声音降低音量和高频（低通滤波器）
- 增强空间感和真实感
- 只对重要的3D音效启用（NPC对话、机关声），避免每帧射线检测开销

## 具体实现方法

### BGM管理器（完整版）

```csharp
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.Audio;

/// <summary>
/// BGM管理器
/// 支持多层音轨、交叉渐变、音乐Stinger
/// </summary>
public class BGMManager : MonoBehaviour
{
    public static BGMManager Instance { get; private set; }

    [System.Serializable]
    public class BGMLayer
    {
        public string name;
        public AudioSource source;
        public AudioClip clip;
        [Range(0, 1)] public float targetVolume = 1f;
        [HideInInspector] public float startVolume;
    }

    [Header("音轨层")]
    [SerializeField] private BGMLayer[] layers;

    [Header("设置")]
    [SerializeField] private float crossfadeDuration = 2f;
    [SerializeField] private AudioMixerGroup bgmMixerGroup;

    [Header("Stinger")]
    [SerializeField] private AudioClip discoveryStinger; // 进入新区域
    [SerializeField] private AudioClip victoryStinger;   // 胜利
    [SerializeField] private AudioClip dangerStinger;    // 危险

    void Awake()
    {
        if (Instance != null) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);

        // 初始化所有音轨
        foreach (var layer in layers)
        {
            if (layer.source == null)
            {
                layer.source = gameObject.AddComponent<AudioSource>();
                layer.source.loop = true;
                layer.source.playOnAwake = false;
                if (bgmMixerGroup != null)
                    layer.source.outputAudioMixerGroup = bgmMixerGroup;
            }
            layer.source.volume = 0;
            if (layer.clip != null)
            {
                layer.source.clip = layer.clip;
                layer.source.Play();
            }
        }
    }

    /// <summary>
    /// 激活指定音轨，其他音轨淡出
    /// </summary>
    public void TransitionTo(string layerName)
    {
        foreach (var layer in layers)
        {
            float target = layer.name == layerName ? layer.targetVolume : 0f;
            StartCoroutine(FadeLayer(layer, target));
        }
    }

    /// <summary>
    /// 多层同时激活（战斗时叠加战斗层）
    /// </summary>
    public void EnableLayer(string layerName)
    {
        foreach (var layer in layers)
        {
            if (layer.name == layerName)
                StartCoroutine(FadeLayer(layer, layer.targetVolume));
        }
    }

    public void DisableLayer(string layerName)
    {
        foreach (var layer in layers)
        {
            if (layer.name == layerName)
                StartCoroutine(FadeLayer(layer, 0f));
        }
    }

    /// <summary>
    /// 播放音乐Stinger
    /// </summary>
    public void PlayStinger(AudioClip stinger)
    {
        if (stinger != null)
        {
            // 临时降低BGM
            StartCoroutine(StingerCoroutine(stinger));
        }
    }

    IEnumerator StingerCoroutine(AudioClip stinger)
    {
        // 降低所有BGM音量
        foreach (var layer in layers)
            layer.startVolume = layer.source.volume;

        float elapsed = 0;
        while (elapsed < 0.3f)
        {
            elapsed += Time.unscaledDeltaTime;
            foreach (var layer in layers)
                layer.source.volume = Mathf.Lerp(
                    layer.startVolume, layer.startVolume * 0.3f, elapsed / 0.3f);
            yield return null;
        }

        // 播放Stinger
        AudioSource.PlayClipAtPoint(stinger, Vector3.zero);
        yield return new WaitForSecondsRealtime(stinger.length);

        // 恢复BGM音量
        elapsed = 0;
        while (elapsed < 0.5f)
        {
            elapsed += Time.unscaledDeltaTime;
            foreach (var layer in layers)
                layer.source.volume = Mathf.Lerp(
                    layer.source.volume, layer.startVolume, elapsed / 0.5f);
            yield return null;
        }
    }

    IEnumerator FadeLayer(BGMLayer layer, float targetVolume)
    {
        layer.startVolume = layer.source.volume;
        float elapsed = 0;

        while (elapsed < crossfadeDuration)
        {
            elapsed += Time.unscaledDeltaTime;
            layer.source.volume = Mathf.Lerp(
                layer.startVolume, targetVolume,
                elapsed / crossfadeDuration);
            yield return null;
        }

        layer.source.volume = targetVolume;

        // 音量为0时暂停播放节省CPU
        if (targetVolume <= 0.001f)
            layer.source.Pause();
        else if (!layer.source.isPlaying)
            layer.source.UnPause();
    }

    public void PlayDiscoveryStinger() => PlayStinger(discoveryStinger);
    public void PlayVictoryStinger() => PlayStinger(victoryStinger);
    public void PlayDangerStinger() => PlayStinger(dangerStinger);
}
```

### 环境音层混合系统

```csharp
using UnityEngine;

/// <summary>
/// 环境音混合器
/// 多层环境音按时间和位置动态混合
/// </summary>
public class AmbientMixer : MonoBehaviour
{
    [System.Serializable]
    public class AmbientLayer
    {
        public string name;
        public AudioSource source;
        public AnimationCurve volumeByTime; // 一天中的音量变化(0-1)
        public float baseVolume = 1f;
        public float spatialBlend = 0f; // 0=2D, 1=完全3D
    }

    [SerializeField] private AmbientLayer[] layers;
    [SerializeField] private float updateInterval = 0.5f; // 更新间隔

    private float updateTimer;

    void Update()
    {
        updateTimer += Time.deltaTime;
        if (updateTimer < updateInterval) return;
        updateTimer = 0;

        if (DayNightCycle.Instance == null) return;

        float timeOfDay = DayNightCycle.Instance.timeOfDay;

        foreach (var layer in layers)
        {
            if (layer.source == null) continue;

            float timeMultiplier = layer.volumeByTime.Evaluate(timeOfDay);
            layer.source.volume = layer.baseVolume * timeMultiplier;
            layer.source.spatialBlend = layer.spatialBlend;
        }
    }
}
```

### 音频遮挡系统

```csharp
using UnityEngine;

/// <summary>
/// 音频遮挡系统
/// 墙壁等障碍物阻挡声音传播
/// </summary>
public class AudioOcclusion : MonoBehaviour
{
    [SerializeField] private AudioSource audioSource;
    [SerializeField] private Transform listener;
    [SerializeField] private float occlusionAmount = 0.3f; // 遮挡后的音量比例
    [SerializeField] private float lowPassCutoff = 800f;   // 遮挡后的低通截止频率
    [SerializeField] private LayerMask occluderLayer;
    [SerializeField] private float updateInterval = 0.2f;   // 检测间隔

    private AudioLowPassFilter lowPassFilter;
    private AudioHighPassFilter highPassFilter;
    private float originalVolume;
    private float updateTimer;
    private bool isOccluded;

    void Start()
    {
        originalVolume = audioSource.volume;
        lowPassFilter = gameObject.AddComponent<AudioLowPassFilter>();
        lowPassFilter.cutoffFrequency = 22000f; // 默认全通
    }

    void Update()
    {
        updateTimer += Time.deltaTime;
        if (updateTimer < updateInterval) return;
        updateTimer = 0;

        if (listener == null)
        {
            listener = Camera.main?.transform;
            if (listener == null) return;
        }

        Vector3 direction = listener.position - transform.position;
        float distance = direction.magnitude;

        // 多条射线检测（更准确）
        int rayCount = 3;
        int occludedCount = 0;

        for (int i = 0; i < rayCount; i++)
        {
            Vector3 offset = Random.insideUnitSphere * 0.5f;
            Vector3 rayStart = transform.position + offset;
            Vector3 rayDir = (listener.position - rayStart).normalized;

            if (Physics.Raycast(rayStart, rayDir, distance, occluderLayer))
                occludedCount++;
        }

        isOccluded = occludedCount > rayCount / 2;

        // 平滑过渡
        float targetVolume = isOccluded
            ? originalVolume * occlusionAmount
            : originalVolume;
        float targetCutoff = isOccluded ? lowPassCutoff : 22000f;

        audioSource.volume = Mathf.Lerp(audioSource.volume, targetVolume, 0.1f);
        lowPassFilter.cutoffFrequency = Mathf.Lerp(
            lowPassFilter.cutoffFrequency, targetCutoff, 0.1f);
    }
}
```

### 触发器音频切换

```csharp
using UnityEngine;

/// <summary>
/// 区域音频触发器
/// 进入区域时自动切换BGM、环境音和混响
/// </summary>
public class ZoneAudioTrigger : MonoBehaviour
{
    [Header("BGM")]
    [SerializeField] private string bgmLayerName;
    [SerializeField] private bool enableBgmLayer = true;

    [Header("环境音")]
    [SerializeField] private AudioClip zoneAmbience;
    [SerializeField] private float ambienceFadeDuration = 2f;

    [Header("混响")]
    [SerializeField] private AudioReverbPreset reverbPreset;

    [Header("音乐Stinger")]
    [SerializeField] private AudioClip enterStinger;

    void OnTriggerEnter(Collider other)
    {
        if (!other.CompareTag("Player")) return;

        // 切换BGM
        if (!string.IsNullOrEmpty(bgmLayerName))
        {
            if (enableBgmLayer)
                BGMManager.Instance.TransitionTo(bgmLayerName);
        }

        // 切换环境音
        if (zoneAmbience != null)
            AmbientManager.Instance.CrossFade(zoneAmbience, ambienceFadeDuration);

        // 切换混响
        AudioReverbZone reverbZone = GetComponent<AudioReverbZone>();
        if (reverbZone != null)
            reverbZone.reverbPreset = reverbPreset;

        // 播放Stinger
        if (enterStinger != null)
            BGMManager.Instance.PlayStinger(enterStinger);
    }
}
```

## 最佳实践

- BGM使用多层音轨实现动态混合（探索层、战斗层、Boss层），避免整首切换
- 环境音至少3-4层混合以避免重复感（风声+鸟鸣+虫鸣+水声）
- 音频遮挡仅对重要的3D音效启用，避免每帧射线检测的开销
- 使用Audio Mixer的Snapshot实现全局混音状态切换（战斗混音、室内混音）
- 所有音频过渡使用协程平滑渐变，避免突变（渐变时长0.5-2秒）
- 预加载即将进入区域的音频资源
- 移动端使用Vorbis/ADPCM压缩格式减少内存占用
- 3D音效设置合理的Rolloff曲线，避免远处完全无声

## 常见陷阱与修复

**陷阱1：BGM切换时直接Stop再Play**
- 症状：音乐断裂，听感不连贯
- 修复：使用Cross Fade（交叉渐变），新旧音乐同时播放渐变

**陷阱2：环境音源过多导致音频混叠和CPU开销**
- 症状：场景中有50+AudioSource
- 修复：限制同时播放的音源数量，远处音源暂停播放

**陷阱3：未考虑移动端的音频格式压缩**
- 症状：音频文件占用大量内存
- 修复：移动端使用Vorbis压缩，质量设为70%

**陷阱4：3D音效的Rolloff曲线未调**
- 症状：远处音量过小或过大的变化不自然
- 修复：自定义Rolloff曲线，设置合理的Min/Max Distance

**陷阱5：音频遮挡的射线检测每帧执行**
- 症状：CPU开销过高
- 修复：设置检测间隔0.1-0.3秒，使用节流
