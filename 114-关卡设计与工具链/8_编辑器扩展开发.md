# 编辑器扩展开发

## 核心概念

编辑器扩展工具能大幅提升关卡设计师的工作效率，将重复性操作自动化。一个好的编辑器工具可以将数小时的手动操作缩短到几秒钟。

### Unity编辑器扩展类型

| 类型 | 用途 | 复杂度 | 典型场景 |
|------|------|--------|---------|
| MenuItem | 菜单栏扩展 | 简单 | 批量操作、快捷功能 |
| EditorWindow | 自定义窗口 | 中等 | 复杂工具、关卡生成 |
| Custom Inspector | 自定义检视面板 | 中等 | 增强组件显示 |
| Property Drawer | 自定义属性绘制 | 简单 | 美化特定字段 |
| Editor Tool | Scene视图工具 | 中等 | 可视化操作 |
| AssetPostprocessor | 资源导入处理 | 中等 | 自动化导入设置 |

### UE Editor Utility Widget

- 基于UMG构建的编辑器工具界面
- 支持蓝图逻辑驱动
- 通过Editor Utility Blueprint实现自动化
- 可以发布为Editor Utility Tool供团队使用

## 具体实现方法

### 自定义EditorWindow（完整工具）

```csharp
#if UNITY_EDITOR
using UnityEditor;
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// 关卡设计工具窗口
/// 集成关卡生成、批量操作、检查工具
/// </summary>
public class LevelDesignTool : EditorWindow
{
    // 状态
    private string levelName = "New Level";
    private int gridWidth = 20, gridHeight = 20;
    private GameObject wallPrefab;
    private GameObject floorPrefab;
    private GameObject enemyPrefab;
    private int enemyCount = 5;

    // Tab页
    private int selectedTab;
    private string[] tabs = { "生成", "批量操作", "检查", "设置" };

    [MenuItem("Tools/Level Design Tool")]
    static void Open()
    {
        var window = GetWindow<LevelDesignTool>("Level Design");
        window.minSize = new Vector2(400, 300);
    }

    void OnGUI()
    {
        selectedTab = GUILayout.Toolbar(selectedTab, tabs);
        EditorGUILayout.Space();

        switch (selectedTab)
        {
            case 0: DrawGenerateTab(); break;
            case 1: DrawBatchTab(); break;
            case 2: DrawCheckTab(); break;
            case 3: DrawSettingsTab(); break;
        }
    }

    void DrawGenerateTab()
    {
        GUILayout.Label("关卡生成工具", EditorStyles.boldLabel);

        levelName = EditorGUILayout.TextField("关卡名称", levelName);
        gridWidth = EditorGUILayout.IntField("宽度", gridWidth);
        gridHeight = EditorGUILayout.IntField("高度", gridHeight);
        wallPrefab = (GameObject)EditorGUILayout.ObjectField(
            "墙壁预制体", wallPrefab, typeof(GameObject), false);
        floorPrefab = (GameObject)EditorGUILayout.ObjectField(
            "地板预制体", floorPrefab, typeof(GameObject), false);

        EditorGUILayout.Space();

        EditorGUILayout.BeginHorizontal();
        if (GUILayout.Button("生成关卡"))
            GenerateLevel();
        if (GUILayout.Button("清空关卡"))
            ClearLevel();
        EditorGUILayout.EndHorizontal();

        if (GUILayout.Button("生成随机房间"))
            GenerateRandomRooms();
    }

    void DrawBatchTab()
    {
        GUILayout.Label("批量操作", EditorStyles.boldLabel);

        if (GUILayout.Button("批量替换选中物体为预制体"))
            BatchReplaceSelected();

        if (GUILayout.Button("批量标记Static"))
            BatchMarkStatic();

        if (GUILayout.Button("批量添加碰撞体"))
            BatchAddColliders();

        if (GUILayout.Button("批量设置Layer"))
            BatchSetLayer();
    }

    void DrawCheckTab()
    {
        GUILayout.Label("关卡检查", EditorStyles.boldLabel);

        if (GUILayout.Button("检查悬空物体"))
            CheckFloatingObjects();

        if (GUILayout.Button("检查重叠物体"))
            CheckOverlappingObjects();

        if (GUILayout.Button("检查缺少碰撞体的物体"))
            CheckMissingColliders();

        if (GUILayout.Button("检查光照设置"))
            CheckLightingSetup();
    }

    void DrawSettingsTab()
    {
        GUILayout.Label("设置", EditorStyles.boldLabel);
        enemyPrefab = (GameObject)EditorGUILayout.ObjectField(
            "敌人预制体", enemyPrefab, typeof(GameObject), false);
        enemyCount = EditorGUILayout.IntSlider("敌人数量", enemyCount, 1, 20);
    }

    // === 生成方法 ===
    void GenerateLevel()
    {
        GameObject root = new GameObject(levelName);
        Undo.RegisterCreatedObjectUndo(root, "Generate Level");

        for (int x = 0; x < gridWidth; x++)
        for (int y = 0; y < gridHeight; y++)
        {
            bool isEdge = x == 0 || x == gridWidth - 1 ||
                          y == 0 || y == gridHeight - 1;
            GameObject prefab = isEdge ? wallPrefab : floorPrefab;
            if (prefab == null) continue;

            GameObject go = (GameObject)PrefabUtility
                .InstantiatePrefab(prefab, root.transform);
            go.transform.position = new Vector3(x, 0, y);
            Undo.RegisterCreatedObjectUndo(go, "Place Tile");
        }
    }

    void GenerateRandomRooms()
    {
        // 使用BSP生成随机房间
        var bsp = new BSPDungeon();
        bsp.Generate(gridWidth, gridHeight, 4, 5);

        GameObject root = new GameObject(levelName + "_BSP");
        Undo.RegisterCreatedObjectUndo(root, "Generate BSP");

        foreach (var tile in bsp.floorTiles)
        {
            if (floorPrefab != null)
            {
                GameObject go = (GameObject)PrefabUtility
                    .InstantiatePrefab(floorPrefab, root.transform);
                go.transform.position = new Vector3(tile.x, 0, tile.y);
                Undo.RegisterCreatedObjectUndo(go, "Place Floor");
            }
        }
    }

    void ClearLevel()
    {
        if (EditorUtility.DisplayDialog("确认",
            "确定要删除所有生成的关卡物体吗？", "确定", "取消"))
        {
            // 删除关卡根节点
        }
    }

    // === 批量操作方法 ===
    void BatchReplaceSelected()
    {
        if (Selection.gameObjects.Length == 0)
        {
            EditorUtility.DisplayDialog("提示", "请先选中物体", "OK");
            return;
        }

        string path = EditorUtility.OpenFilePanel("选择预制体", "Assets", "prefab");
        if (string.IsNullOrEmpty(path)) return;

        path = "Assets" + path.Substring(Application.dataPath.Length);
        GameObject replacement = AssetDatabase.LoadAssetAtPath<GameObject>(path);

        if (replacement == null) return;

        var selected = new List<GameObject>(Selection.gameObjects);
        foreach (var go in selected)
        {
            Vector3 pos = go.transform.position;
            Quaternion rot = go.transform.rotation;
            Transform parent = go.transform.parent;
            Undo.DestroyObjectImmediate(go);

            GameObject newGo = (GameObject)PrefabUtility
                .InstantiatePrefab(replacement, parent);
            newGo.transform.position = pos;
            newGo.transform.rotation = rot;
            Undo.RegisterCreatedObjectUndo(newGo, "Replace Prefab");
        }
    }

    void BatchMarkStatic()
    {
        foreach (var go in Selection.gameObjects)
        {
            Undo.RecordObject(go, "Mark Static");
            go.isStatic = true;
        }
    }

    void BatchAddColliders()
    {
        foreach (var go in Selection.gameObjects)
        {
            if (go.GetComponent<Collider>() == null)
            {
                Undo.AddComponent<BoxCollider>(go);
            }
        }
    }

    void BatchSetLayer()
    {
        string layerName = "Ground";
        int layer = LayerMask.NameToLayer(layerName);
        foreach (var go in Selection.gameObjects)
        {
            Undo.RecordObject(go, "Set Layer");
            go.layer = layer;
        }
    }

    // === 检查方法 ===
    void CheckFloatingObjects()
    {
        int count = 0;
        foreach (var go in FindObjectsOfType<GameObject>())
        {
            if (go.isStatic && !Physics.Raycast(
                go.transform.position + Vector3.up * 0.5f,
                Vector3.down, 2f))
            {
                Debug.LogWarning($"可能悬空: {go.name} at {go.transform.position}", go);
                count++;
            }
        }
        Debug.Log($"检查完成，发现 {count} 个可能悬空的物体");
    }

    void CheckOverlappingObjects()
    {
        var renderers = FindObjectsOfType<Renderer>();
        int count = 0;
        for (int i = 0; i < renderers.Length; i++)
        for (int j = i + 1; j < renderers.Length; j++)
        {
            if (renderers[i].bounds.Intersects(renderers[j].bounds))
            {
                Debug.LogWarning($"重叠: {renderers[i].name} 和 {renderers[j].name}");
                count++;
            }
        }
        Debug.Log($"发现 {count} 对重叠物体");
    }

    void CheckMissingColliders()
    {
        int count = 0;
        foreach (var renderer in FindObjectsOfType<Renderer>())
        {
            if (renderer.gameObject.isStatic
                && renderer.GetComponent<Collider>() == null)
            {
                Debug.LogWarning($"缺少碰撞体: {renderer.name}", renderer.gameObject);
                count++;
            }
        }
        Debug.Log($"发现 {count} 个缺少碰撞体的静态物体");
    }

    void CheckLightingSetup()
    {
        Light[] lights = FindObjectsOfType<Light>();
        int realtimeCount = 0;
        foreach (var light in lights)
        {
            if (light.lightmapBakeType == LightmapBakeType.Realtime)
                realtimeCount++;
        }
        Debug.Log($"灯光总数: {lights.Length}, 实时灯光: {realtimeCount}");
        if (realtimeCount > 3)
            Debug.LogWarning("实时灯光过多(>3)，移动端建议使用烘焙光照");
    }
}
#endif
```

### Custom Inspector

```csharp
#if UNITY_EDITOR
using UnityEditor;
using UnityEngine;

/// <summary>
/// 生成点自定义Inspector
/// 增强SpawnPoint组件的编辑体验
/// </summary>
[CustomEditor(typeof(SpawnPoint))]
public class SpawnPointEditor : Editor
{
    public override void OnInspectorGUI()
    {
        DrawDefaultInspector();

        SpawnPoint sp = (SpawnPoint)target;

        EditorGUILayout.Space();
        EditorGUILayout.LabelField("快捷操作", EditorStyles.boldLabel);

        EditorGUILayout.BeginHorizontal();
        if (GUILayout.Button("预览范围"))
            SceneView.RepaintAll();

        if (GUILayout.Button("测试生成"))
            sp.TestSpawn();

        if (GUILayout.Button("清除生成物"))
            sp.ClearSpawned();
        EditorGUILayout.EndHorizontal();

        // 显示统计信息
        EditorGUILayout.Space();
        EditorGUILayout.LabelField("统计", EditorStyles.boldLabel);
        EditorGUILayout.LabelField($"已生成: {sp.SpawnedCount}");
        EditorGUILayout.LabelField($"存活: {sp.AliveCount}");
    }

    void OnSceneGUI()
    {
        SpawnPoint sp = (SpawnPoint)target;

        // 绘制生成范围
        Handles.color = new Color(1, 1, 0, 0.2f);
        Handles.DrawSolidDisc(sp.transform.position, Vector3.up, sp.spawnRadius);
        Handles.color = Color.yellow;
        Handles.DrawWireDisc(sp.transform.position, Vector3.up, sp.spawnRadius);

        // 可编辑的半径手柄
        float newRadius = Handles.RadiusHandle(
            Quaternion.identity, sp.transform.position, sp.spawnRadius);
        if (!Mathf.Approximately(newRadius, sp.spawnRadius))
        {
            Undo.RecordObject(sp, "Change Spawn Radius");
            sp.spawnRadius = newRadius;
        }

        // 标签
        Handles.Label(sp.transform.position + Vector3.up * 2,
            $"Spawn: {sp.spawnRadius}m\nCount: {sp.SpawnedCount}");
    }
}
#endif
```

### Property Drawer

```csharp
#if UNITY_EDITOR
using UnityEditor;
using UnityEngine;

/// <summary>
/// MinMax范围滑块Property Drawer
/// 在Inspector中显示为可拖拽的范围滑块
/// </summary>
[CustomPropertyDrawer(typeof(MinMaxRange))]
public class MinMaxRangeDrawer : PropertyDrawer
{
    public override void OnGUI(Rect position,
        SerializedProperty property, GUIContent label)
    {
        SerializedProperty minProp = property.FindPropertyRelative("min");
        SerializedProperty maxProp = property.FindPropertyRelative("max");
        SerializedProperty rangeProp = property.FindPropertyRelative("range");

        float min = minProp.floatValue;
        float max = maxProp.floatValue;
        Vector2 range = rangeProp.vector2Value;

        position = EditorGUI.PrefixLabel(position, label);
        float fieldWidth = 45;
        float sliderWidth = position.width - fieldWidth * 2 - 10;

        Rect minRect = new Rect(position.x, position.y, fieldWidth, position.height);
        Rect sliderRect = new Rect(position.x + fieldWidth + 5, position.y,
            sliderWidth, position.height);
        Rect maxRect = new Rect(position.x + fieldWidth + sliderWidth + 10,
            position.y, fieldWidth, position.height);

        min = EditorGUI.FloatField(minRect, min);
        EditorGUI.MinMaxSlider(sliderRect, ref min, ref max, range.x, range.y);
        max = EditorGUI.FloatField(maxRect, max);

        minProp.floatValue = Mathf.Clamp(min, range.x, max);
        maxProp.floatValue = Mathf.Clamp(max, min, range.y);
    }
}

[System.Serializable]
public class MinMaxRange
{
    public float min;
    public float max;
    public Vector2 range = new Vector2(0, 100);
}
#endif
```

## 最佳实践

- 所有编辑器操作使用Undo系统注册，支持Ctrl+Z撤销
- 编辑器代码放在Editor文件夹或使用`#if UNITY_EDITOR`包裹
- 为团队编写工具时提供清晰的UI标签和提示信息
- 频繁使用的工具注册为快捷键（通过MenuItem的快捷键参数）
- 工具输出的日志信息应清晰有用，包含位置引用方便定位
- EditorWindow使用EditorGUILayout简化布局代码
- 大批量操作使用ProgressBar显示进度

## 常见陷阱与修复

**陷阱1：编辑器代码未使用`#if UNITY_EDITOR`包裹**
- 症状：打包时报编译错误
- 修复：所有Editor相关using和代码用`#if UNITY_EDITOR`包裹

**陷阱2：未注册Undo操作**
- 症状：用户无法Ctrl+Z撤销工具操作
- 修复：使用Undo.RegisterCreatedObjectUndo, Undo.RecordObject, Undo.AddComponent

**陷阱3：EditorWindow中执行耗时操作导致编辑器卡顿**
- 症状：点击按钮后编辑器冻结数秒
- 修复：使用EditorCoroutine或ProgressBar显示进度，分帧处理

**陷阱4：Custom Inspector中修改了序列化数据但未调用ApplyModifiedProperties**
- 症状：修改不保存
- 修复：在OnInspectorGUI末尾调用`serializedObject.ApplyModifiedProperties()`

**陷阱5：Scene视图中的Handles绘制消耗过多性能**
- 痖状：选中物体后Scene视图卡顿
- 修复：减少绘制的Handles数量，使用Handles.matrix批量绘制
