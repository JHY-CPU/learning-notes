# 触发器与事件区域

## 核心概念

触发器是关卡设计中连接空间与游戏逻辑的桥梁，控制事件的发生时机和位置。它们是关卡的"神经系统"，决定了玩家在何时何地触发何种事件。

### 触发器类型详解

| 类型 | 触发条件 | 典型用途 | 性能开销 |
|------|---------|---------|---------|
| 区域触发器 | 进入/离开Collider | 切换BGM、触发战斗 | 极低 |
| 剧情触发点 | 到达指定位置 | 对话、过场动画 | 低 |
| 检查点存档 | 到达指定位置 | 保存进度 | 极低 |
| 安全区 | 进入/离开 | 禁止战斗、恢复HP | 极低 |
| 加载触发器 | 进入区域 | 异步加载相邻场景 | 中等 |
| 条件触发器 | 满足特定条件 | 解锁门、开启机关 | 低 |
| 定时触发器 | 时间到达 | 限时事件、波次刷新 | 极低 |

### 触发器设计原则

- **不可逆触发器需要状态持久化**：玩家重入区域时不应重复触发
- **触发区域大小应比视觉提示略大**：避免玩家擦边错过
- **使用Tag/Layer过滤触发对象**：避免无关物体（掉落物、子弹）触发事件
- **一次触发器应有明确的视觉/音频反馈**：让玩家知道事件已触发

## 具体实现方法

### 触发器基础系统（完整版）

```csharp
using UnityEngine;
using UnityEngine.Events;

/// <summary>
/// 通用区域触发器
/// 支持多种触发模式和条件过滤
/// </summary>
public class AreaTrigger : MonoBehaviour
{
    public enum TriggerMode { EnterOnce, EnterRepeat, ExitOnce, EnterAndExit, Continuous }

    [Header("基本配置")]
    [SerializeField] private TriggerMode mode = TriggerMode.EnterOnce;
    [SerializeField] private string requiredTag = "Player";
    [SerializeField] private LayerMask requiredLayer = -1;
    [SerializeField] private float cooldown = 0f; // 冷却时间

    [Header("事件")]
    [SerializeField] private UnityEvent onTriggerEnter;
    [SerializeField] private UnityEvent onTriggerExit;
    [SerializeField] private UnityEvent onTriggerStay;

    [Header("调试")]
    [SerializeField] private bool showDebugGizmo = true;
    [SerializeField] private Color gizmoColor = new Color(0, 1, 0, 0.3f);

    private bool hasTriggered = false;
    private float lastTriggerTime;
    private bool isPlayerInside = false;

    void OnTriggerEnter(Collider other)
    {
        if (!ValidateTrigger(other)) return;
        if (Time.time - lastTriggerTime < cooldown) return;

        isPlayerInside = true;

        switch (mode)
        {
            case TriggerMode.EnterOnce:
                if (hasTriggered) return;
                hasTriggered = true;
                lastTriggerTime = Time.time;
                onTriggerEnter?.Invoke();
                break;

            case TriggerMode.EnterRepeat:
            case TriggerMode.EnterAndExit:
                lastTriggerTime = Time.time;
                onTriggerEnter?.Invoke();
                break;
        }
    }

    void OnTriggerExit(Collider other)
    {
        if (!ValidateTrigger(other)) return;
        isPlayerInside = false;

        switch (mode)
        {
            case TriggerMode.ExitOnce:
                if (hasTriggered) return;
                hasTriggered = true;
                onTriggerExit?.Invoke();
                break;

            case TriggerMode.EnterAndExit:
                onTriggerExit?.Invoke();
                break;
        }
    }

    void OnTriggerStay(Collider other)
    {
        if (!ValidateTrigger(other)) return;
        if (mode == TriggerMode.Continuous)
            onTriggerStay?.Invoke();
    }

    bool ValidateTrigger(Collider other)
    {
        if (!other.CompareTag(requiredTag)) return false;
        if ((requiredLayer.value & (1 << other.gameObject.layer)) == 0) return false;
        return true;
    }

    /// <summary>
    /// 重置触发器状态（允许再次触发）
    /// </summary>
    public void ResetTrigger()
    {
        hasTriggered = false;
    }

    void OnDrawGizmos()
    {
        if (!showDebugGizmo) return;

        Gizmos.color = hasTriggered ? Color.gray : gizmoColor;
        Collider col = GetComponent<Collider>();
        if (col is BoxCollider box)
        {
            Gizmos.matrix = transform.localToWorldMatrix;
            Gizmos.DrawCube(box.center, box.size);
            Gizmos.DrawWireCube(box.center, box.size);
        }
        else if (col is SphereCollider sphere)
        {
            Gizmos.DrawSphere(transform.position + sphere.center, sphere.radius);
        }
    }
}
```

### 剧情触发系统

```csharp
/// <summary>
/// 剧情触发系统
/// 到达指定位置触发对话或过场动画
/// </summary>
public class StoryTrigger : MonoBehaviour
{
    [Header("剧情配置")]
    [SerializeField] private string dialogueKey;
    [SerializeField] private string cutsceneId;
    [SerializeField] private bool pauseGame = true;
    [SerializeField] private bool playOnce = true;
    [SerializeField] private float delayBeforePlay = 0f;

    [Header("条件")]
    [SerializeField] private string[] prerequisiteQuests;
    [SerializeField] private string[] requiredItems;

    private bool played = false;
    private bool savedInSaveData = false;

    void OnTriggerEnter(Collider other)
    {
        if (!other.CompareTag("Player") || played) return;

        // 检查前置条件
        if (!CheckPrerequisites()) return;

        if (playOnce) played = true;

        if (delayBeforePlay > 0)
        {
            Invoke(nameof(PlayStory), delayBeforePlay);
        }
        else
        {
            PlayStory();
        }
    }

    bool CheckPrerequisites()
    {
        foreach (var quest in prerequisiteQuests)
        {
            if (!QuestManager.Instance.IsQuestCompleted(quest))
                return false;
        }

        foreach (var item in requiredItems)
        {
            if (!InventoryManager.Instance.HasItem(item))
                return false;
        }

        return true;
    }

    void PlayStory()
    {
        if (pauseGame) Time.timeScale = 0f;

        if (!string.IsNullOrEmpty(dialogueKey))
        {
            DialogueManager.Instance.Play(dialogueKey, () =>
            {
                if (pauseGame) Time.timeScale = 1f;
            });
        }
        else if (!string.IsNullOrEmpty(cutsceneId))
        {
            CutsceneManager.Instance.Play(cutsceneId, () =>
            {
                if (pauseGame) Time.timeScale = 1f;
            });
        }
    }

    /// <summary>
    /// 从存档加载触发状态
    /// </summary>
    public void LoadState(bool hasPlayed)
    {
        played = hasPlayed;
    }
}
```

### 检查点系统（完整版）

```csharp
using UnityEngine;
using UnityEngine.Events;

/// <summary>
/// 检查点系统
/// 保存玩家进度和重生位置
/// </summary>
public class Checkpoint : MonoBehaviour
{
    [Header("检查点配置")]
    [SerializeField] private string checkpointId;
    [SerializeField] private Transform respawnPoint;
    [SerializeField] private bool autoSave = true;

    [Header("视觉反馈")]
    [SerializeField] private GameObject deactivatedEffect; // 未激活的火焰效果
    [SerializeField] private GameObject activatedEffect;   // 激活后的火焰效果
    [SerializeField] private ParticleSystem activationVFX;
    [SerializeField] private AudioClip activationSound;

    [Header("事件")]
    [SerializeField] private UnityEvent onActivated;

    private bool isActivated = false;

    void Start()
    {
        // 检查是否已经从存档中激活过
        if (SaveManager.Instance != null)
        {
            isActivated = SaveManager.Instance.IsCheckpointActivated(checkpointId);
            UpdateVisuals();
        }
    }

    void OnTriggerEnter(Collider other)
    {
        if (!other.CompareTag("Player") || isActivated) return;

        Activate();
    }

    void Activate()
    {
        isActivated = true;

        // 保存检查点数据
        SaveData data = SaveManager.Instance.CurrentData;
        data.lastCheckpoint = checkpointId;
        data.playerPosition = respawnPoint.position;
        data.playerRotation = respawnPoint.rotation;
        data.playerHP = PlayerStats.Instance.CurrentHP;
        data.playerMP = PlayerStats.Instance.CurrentMP;

        // 保存世界状态
        SaveWorldState(data);

        if (autoSave)
            SaveManager.Instance.Save();

        // 更新重生点
        GameManager.Instance.SetRespawnPoint(respawnPoint, checkpointId);

        // 视觉和音频反馈
        UpdateVisuals();
        PlayActivationEffects();

        onActivated?.Invoke();

        Debug.Log($"Checkpoint activated: {checkpointId}");
    }

    void SaveWorldState(SaveData data)
    {
        // 保存敌人状态
        data.enemyStates.Clear();
        foreach (var enemy in FindObjectsOfType<Enemy>())
        {
            data.enemyStates.Add(new SaveData.EnemyState
            {
                enemyId = enemy.EnemyId,
                isDead = enemy.IsDead,
                position = enemy.transform.position
            });
        }

        // 保存物品拾取状态
        data.collectedItems.Clear();
        foreach (var pickup in FindObjectsOfType<ItemPickup>())
        {
            if (pickup.IsCollected)
                data.collectedItems.Add(pickup.PickupId);
        }
    }

    void UpdateVisuals()
    {
        if (deactivatedEffect != null)
            deactivatedEffect.SetActive(!isActivated);
        if (activatedEffect != null)
            activatedEffect.SetActive(isActivated);
    }

    void PlayActivationEffects()
    {
        if (activationVFX != null) activationVFX.Play();
        if (activationSound != null)
            AudioSource.PlayClipAtPoint(activationSound, transform.position);
    }
}
```

### 区域管理器（综合系统）

```csharp
/// <summary>
/// 区域管理器
/// 管理不同区域的环境配置切换
/// </summary>
public class ZoneManager : MonoBehaviour
{
    [System.Serializable]
    public class Zone
    {
        public string name;
        public Collider triggerArea;
        public ZoneConfig config;
        public UnityEvent onEnter;
        public UnityEvent onExit;
    }

    [System.Serializable]
    public class ZoneConfig
    {
        public string bgmName;
        public AudioReverbPreset reverbPreset;
        public Color ambientColor = Color.white;
        public float fogDensity = 0.01f;
        public Color fogColor = Color.gray;
        public float temperature = 20f; // 温度（影响色调）
    }

    [SerializeField] private Zone[] zones;
    private Zone currentZone;

    public Zone CurrentZone => currentZone;

    public void OnPlayerEnterZone(Zone zone)
    {
        if (zone == currentZone) return;

        currentZone = zone;

        // 切换背景音乐
        if (!string.IsNullOrEmpty(zone.config.bgmName))
            BGMManager.Instance.TransitionTo(zone.config.bgmName);

        // 切换环境音
        AudioReverbManager.Instance.SetReverb(zone.config.reverbPreset);

        // 切换环境光照
        RenderSettings.ambientLight = zone.config.ambientColor;
        RenderSettings.fogDensity = zone.config.fogDensity;
        RenderSettings.fogColor = zone.config.fogColor;

        // 触发自定义事件
        zone.onEnter?.Invoke();

        Debug.Log($"Entered zone: {zone.name}");
    }

    public void OnPlayerExitZone(Zone zone)
    {
        zone.onExit?.Invoke();
    }
}
```

## 最佳实践

- 触发器Collider设置为Trigger模式（isTrigger = true）
- 使用Tag或Layer过滤触发对象，避免无关物体触发事件
- 一次性触发器务必有状态持久化，防止玩家重入区域时重复触发
- 检查点密度合理：太稀疏导致挫败感（死亡重玩5分钟），太密降低紧张感
- 触发区域大小应比视觉提示略大（大10-20%），避免玩家擦边错过
- 为触发器添加Gizmo可视化，方便设计师在编辑器中调整位置和大小

## 常见陷阱与修复

**陷阱1：触发器Collider过大导致意外触发**
- 症状：玩家在走廊经过就触发了远处房间的事件
- 修复：精确调整Collider大小，使用Tag过滤

**陷阱2：未保存触发状态导致重复播放剧情**
- 症状：玩家返回已触发区域再次播放剧情
- 修复：将触发状态保存到SaveData

**陷阱3：检查点未保存所有必要状态**
- 症状：读档后NPC位置、任务进度丢失
- 修复：SaveWorldState保存所有关键世界状态

**陷阱4：多个触发器在同一位置竞争**
- 症状：两个触发器同时触发导致事件冲突
- 修复：使用优先级系统，或确保触发区域不重叠

**陷阱5：触发事件中的异步操作未正确处理**
- 症状：对话未完成就触发了下一个事件
- 修复：使用回调确保异步操作完成后再继续
