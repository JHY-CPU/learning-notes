# 关卡测试与迭代

## 核心概念

关卡测试是验证设计意图是否落地的关键环节，数据驱动的迭代方法比主观感觉更可靠。"没有测试的设计只是假设"——这是关卡设计领域最重要的原则之一。

### Playtest方法分类

| 方法 | 参与者 | 时机 | 目的 | 样本量 |
|------|--------|------|------|--------|
| 内部Playtest | 团队成员 | 每日 | 快速发现问题 | 5-20人 |
| 外部Playtest | 目标用户 | 里程碑 | 获取真实反馈 | 20-50人 |
| 盲测 | 不了解项目的人 | 关键节点 | 观察自然行为 | 10-30人 |
| 竞品分析 | 设计师 | 设计阶段 | 学习优秀做法 | N/A |
| A/B测试 | 在线用户 | 运营阶段 | 数据对比决策 | 1000+人 |

### 遥测数据类型

- **移动热力图**：玩家在关卡中花费最多时间的位置
- **死亡/失败点统计**：哪些位置玩家死亡最多
- **关卡完成时间和通关率**：整体难度和时长评估
- **可收集物品的发现率**：可选内容的可见度
- **玩家卡关时长**：在哪里迷路或卡住
- **技能/武器使用率**：玩家偏好和平衡性

### 数据驱动迭代流程

```
设计假设 → 白盒实现 → Playtest → 收集数据 → 分析数据
    ↓                                      ↓
  下一轮迭代 ← 调整设计 ← 识别问题 ← 数据报告
```

## 具体实现方法

### 热力图系统（完整版）

```csharp
using System.Collections.Generic;
using UnityEngine;

/// <summary>
/// 玩家移动热力图追踪器
/// 记录玩家位置并生成热力图数据
/// </summary>
public class HeatmapTracker : MonoBehaviour
{
    [Header("配置")]
    [SerializeField] private float recordInterval = 0.5f;
    [SerializeField] private string levelId;
    [SerializeField] private bool autoSaveOnExit = true;

    private float timer;
    private List<HeatmapPoint> dataPoints = new List<HeatmapPoint>();

    [System.Serializable]
    public struct HeatmapPoint
    {
        public Vector3 position;
        public float time;
        public string state; // "explore", "combat", "death", "idle"
    }

    [System.Serializable]
    public class HeatmapData
    {
        public string levelId;
        public string sessionId;
        public float totalTime;
        public List<HeatmapPoint> points = new List<HeatmapPoint>();
    }

    void Update()
    {
        timer += Time.deltaTime;
        if (timer < recordInterval) return;
        timer = 0;

        dataPoints.Add(new HeatmapPoint
        {
            position = transform.position,
            time = Time.timeSinceLevelLoad,
            state = GetCurrentState()
        });
    }

    string GetCurrentState()
    {
        // 根据游戏状态判断
        if (PlayerController.Instance.IsInCombat) return "combat";
        if (PlayerController.Instance.IsDead) return "death";
        if (PlayerController.Instance.Velocity.magnitude < 0.1f) return "idle";
        return "explore";
    }

    void OnDestroy()
    {
        if (autoSaveOnExit) SaveHeatmapData();
    }

    public void SaveHeatmapData()
    {
        if (dataPoints.Count == 0) return;

        var data = new HeatmapData
        {
            levelId = levelId,
            sessionId = System.Guid.NewGuid().ToString(),
            totalTime = Time.timeSinceLevelLoad,
            points = dataPoints
        };

        string json = JsonUtility.ToJson(data, true);
        string path = Application.persistentDataPath +
            $"/heatmap_{levelId}_{System.DateTime.Now:yyyyMMdd_HHmmss}.json";
        System.IO.File.WriteAllText(path, json);

        Debug.Log($"Heatmap saved: {path} ({dataPoints.Count} points)");
    }
}
```

### 死亡/卡关检测器

```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

/// <summary>
/// 死亡和卡关追踪系统
/// 识别关卡中的难点区域
/// </summary>
public class DeathTracker : MonoBehaviour
{
    public static DeathTracker Instance { get; private set; }

    [System.Serializable]
    public struct DeathRecord
    {
        public Vector3 position;
        public string cause;
        public float time;
        public int attemptNumber;
    }

    private Dictionary<Vector3Int, int> deathCountByLocation
        = new Dictionary<Vector3Int, int>();
    private List<DeathRecord> deathHistory = new List<DeathRecord>();
    private int currentAttempt = 1;

    [Header("告警阈值")]
    [SerializeField] private int hotSpotThreshold = 3;

    public int TotalDeaths => deathHistory.Count;

    void Awake()
    {
        Instance = this;
    }

    public void RecordDeath(Vector3 position, string cause)
    {
        Vector3Int key = Vector3Int.FloorToInt(position / 2f) * 2; // 2米网格

        if (!deathCountByLocation.ContainsKey(key))
            deathCountByLocation[key] = 0;
        deathCountByLocation[key]++;

        deathHistory.Add(new DeathRecord
        {
            position = position,
            cause = cause,
            time = Time.timeSinceLevelLoad,
            attemptNumber = currentAttempt
        });

        // 检查是否是热点区域
        if (deathCountByLocation[key] >= hotSpotThreshold)
        {
            Debug.LogWarning($"<color=red>热点区域</color>: " +
                $"位置 {key} 已发生 {deathCountByLocation[key]} 次死亡 " +
                $"(原因: {cause})");
        }
    }

    public void OnRespawn()
    {
        currentAttempt++;
    }

    /// <summary>
    /// 生成死亡热点报告
    /// </summary>
    public void GenerateReport()
    {
        var sorted = deathCountByLocation
            .OrderByDescending(kvp => kvp.Value);

        Debug.Log("=== 死亡热点报告 ===");
        Debug.Log($"总死亡次数: {deathHistory.Count}");
        Debug.Log($"尝试次数: {currentAttempt}");
        Debug.Log("");

        foreach (var entry in sorted.Take(10))
        {
            float deathRate = (float)entry.Value / currentAttempt;
            Debug.Log($"位置 {entry.Key}: {entry.Value} 次死亡 " +
                $"(死亡率: {deathRate:P0})");
        }

        // 按原因分类
        var byCause = deathHistory.GroupBy(d => d.cause)
            .OrderByDescending(g => g.Count());
        Debug.Log("\n=== 死亡原因分布 ===");
        foreach (var group in byCause)
            Debug.Log($"{group.Key}: {group.Count()} 次 ({(float)group.Count() / deathHistory.Count:P0})");
    }
}
```

### 遥测数据收集系统

```csharp
using System.Collections.Generic;
using UnityEngine;

/// <summary>
/// 遥测数据收集系统
/// 收集各种游戏事件用于分析
/// </summary>
public class TelemetrySystem : MonoBehaviour
{
    public static TelemetrySystem Instance { get; private set; }

    [SerializeField] private string serverUrl;
    [SerializeField] private int batchSize = 50;
    [SerializeField] private float uploadInterval = 30f;

    private Queue<TelemetryEvent> eventQueue = new Queue<TelemetryEvent>();
    private float uploadTimer;

    [System.Serializable]
    public class TelemetryEvent
    {
        public string eventType;
        public string levelId;
        public Vector3 position;
        public float timestamp;
        public string metadata; // JSON字符串
    }

    void Awake()
    {
        Instance = this;
        DontDestroyOnLoad(gameObject);
    }

    /// <summary>
    /// 记录游戏事件
    /// </summary>
    public void LogEvent(string eventType, Dictionary<string, object> metadata = null)
    {
        string metaJson = metadata != null
            ? JsonUtility.ToJson(new SerializableDict(metadata))
            : "{}";

        eventQueue.Enqueue(new TelemetryEvent
        {
            eventType = eventType,
            levelId = GameManager.Instance?.CurrentLevel ?? "unknown",
            position = PlayerController.Instance?.transform.position ?? Vector3.zero,
            timestamp = Time.realtimeSinceStartup,
            metadata = metaJson
        });
    }

    void Update()
    {
        uploadTimer += Time.deltaTime;
        if (uploadTimer >= uploadInterval && eventQueue.Count > 0)
        {
            uploadTimer = 0;
            UploadBatch();
        }
    }

    async void UploadBatch()
    {
        var batch = new List<TelemetryEvent>();
        while (eventQueue.Count > 0 && batch.Count < batchSize)
            batch.Add(eventQueue.Dequeue());

        if (batch.Count == 0) return;

        string json = JsonUtility.ToJson(batch);

        // 保存到本地（作为备份）
        string localPath = Application.persistentDataPath +
            $"/telemetry_{System.DateTime.Now:yyyyMMdd_HHmmss}.json";
        System.IO.File.AppendAllText(localPath, json + "\n");

        // 上传到服务器（如果可用）
        // 使用UnityWebRequest发送POST请求
        Debug.Log($"Telemetry: {batch.Count} events uploaded");
    }

    // 常用事件快捷方法
    public void LogLevelStart(string levelId) =>
        LogEvent("level_start");
    public void LogLevelComplete(string levelId, float time) =>
        LogEvent("level_complete");
    public void LogPlayerDeath(string cause) =>
        LogEvent("player_death", new Dictionary<string, object> { { "cause", cause } });
    public void LogItemCollected(string itemId) =>
        LogEvent("item_collected", new Dictionary<string, object> { { "item", itemId } });
    public void LogCheckpointReached(string checkpointId) =>
        LogEvent("checkpoint", new Dictionary<string, object> { { "checkpoint", checkpointId } });

    [System.Serializable]
    private class SerializableDict
    {
        public List<string> keys = new List<string>();
        public List<string> values = new List<string>();

        public SerializableDict(Dictionary<string, object> dict)
        {
            foreach (var kvp in dict)
            {
                keys.Add(kvp.Key);
                values.Add(kvp.Value?.ToString() ?? "");
            }
        }
    }
}
```

### A/B测试框架

```csharp
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// A/B测试管理器
/// 根据玩家ID分配测试组
/// </summary>
public class ABTestManager : MonoBehaviour
{
    public static ABTestManager Instance { get; private set; }

    [System.Serializable]
    public class TestVariant
    {
        public string name;
        public GameObject levelPrefab;
        [Range(0, 1)] public float trafficRatio;
    }

    [SerializeField] private TestVariant[] variants;
    private string assignedVariant;

    void Awake()
    {
        Instance = this;
    }

    void Start()
    {
        // 根据玩家ID哈希分配测试组
        string playerId = GameManager.Instance?.PlayerId ?? "default";
        int hash = playerId.GetHashCode();
        float roll = (hash & 0xFFFF) / 65536f;

        float cumulative = 0;
        foreach (var variant in variants)
        {
            cumulative += variant.trafficRatio;
            if (roll <= cumulative)
            {
                assignedVariant = variant.name;
                LoadVariant(variant);

                TelemetrySystem.Instance?.LogEvent("ab_test_assigned",
                    new Dictionary<string, object>
                    { { "variant", variant.name } });
                break;
            }
        }

        Debug.Log($"A/B Test: Assigned to variant '{assignedVariant}'");
    }

    void LoadVariant(TestVariant variant)
    {
        if (variant.levelPrefab != null)
        {
            // 实例化对应的关卡预制体
            Instantiate(variant.levelPrefab);
        }
    }

    /// <summary>
    /// 记录测试指标
    /// </summary>
    public void LogMetric(string metricName, float value)
    {
        TelemetrySystem.Instance?.LogEvent("ab_test_metric",
            new Dictionary<string, object>
            {
                { "variant", assignedVariant },
                { "metric", metricName },
                { "value", value }
            });
    }
}
```

### 关卡分析报告生成

```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

/// <summary>
/// 关卡分析报告生成器
/// 从遥测数据中提取洞察
/// </summary>
public class LevelAnalyticsReport
{
    public static void Generate(string levelId, List<TelemetrySystem.TelemetryEvent> events)
    {
        Debug.Log($"=== 关卡 {levelId} 分析报告 ===");

        // 通关时间分析
        var completes = events.Where(e => e.eventType == "level_complete").ToList();
        if (completes.Count > 0)
        {
            float avgTime = completes.Average(e => e.timestamp);
            float minTime = completes.Min(e => e.timestamp);
            float maxTime = completes.Max(e => e.timestamp);
            Debug.Log($"通关时间: 平均{avgTime:F1}s, 最短{minTime:F1}s, 最长{maxTime:F1}s");
        }

        // 死亡分析
        int totalDeaths = events.Count(e => e.eventType == "player_death");
        int totalAttempts = events.Count(e => e.eventType == "level_start");
        float completionRate = totalAttempts > 0
            ? (float)completes.Count / totalAttempts : 0;

        Debug.Log($"总尝试: {totalAttempts}, 通关: {completes.Count} ({completionRate:P1})");
        Debug.Log($"总死亡: {totalDeaths}, 平均死亡次数: {(float)totalDeaths / Mathf.Max(1, totalAttempts):F1}");

        // 收集率
        int itemsCollected = events.Count(e => e.eventType == "item_collected");
        Debug.Log($"物品收集: {itemsCollected}");

        // 卡关识别
        var deathLocations = events
            .Where(e => e.eventType == "player_death")
            .GroupBy(e => Vector3Int.FloorToInt(e.position / 2f) * 2)
            .OrderByDescending(g => g.Count())
            .Take(5);

        Debug.Log("\n死亡热点:");
        foreach (var loc in deathLocations)
            Debug.Log($"  {loc.Key}: {loc.Count()} 次");

        // 难度评估
        string difficulty;
        if (completionRate > 0.8f) difficulty = "太简单";
        else if (completionRate > 0.5f) difficulty = "适中";
        else if (completionRate > 0.3f) difficulty = "偏难";
        else difficulty = "太难";

        Debug.Log($"\n难度评估: {difficulty} (通关率 {completionRate:P1})");
    }
}
```

## 最佳实践

- 每次关卡改动后至少进行一次内部Playtest
- 邀请不熟悉项目的外部人员进行盲测，观察自然行为
- 收集定量数据（时间、次数、位置）而非仅依赖定性反馈
- 热力图可视化帮助发现设计盲区（玩家走错路、忽略可选区域）
- A/B测试只同时运行一个变量，确保结论可靠
- 建立回归测试清单，确保修改不破坏已有功能
- 将遥测数据可视化为图表便于分析

## 常见陷阱与修复

**陷阱1：仅依赖设计师的个人体验判断**
- 症状：设计师觉得好的关卡，玩家反馈很差
- 修复：结合定量数据和定性反馈做决策

**陷阱2：样本量过小导致结论不可靠**
- 症状：3个玩家的测试结果不能代表整体
- 修复：A/B测试至少需要数百个样本

**陷阱3：收集了大量数据但未建立分析流程**
- 症状：数据存在硬盘中从未查看
- 修复：建立定期分析报告流程，使用可视化工具

**陷阱4：过度优化单一指标**
- 症状：为了提高通关率降低了难度，但玩家觉得无聊
- 修复：综合考虑多个指标（通关率、重玩率、满意度）

**陷阱5：测试环境与正式环境差异**
- 症状：测试通过但正式上线后发现问题
- 修复：确保测试环境与正式环境一致（帧率、设备、网络）
