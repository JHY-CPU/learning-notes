# 07-格与布尔代数

格与布尔代数是代数结构与序理论的交汇点，在数字电路设计、逻辑设计、数据库理论中有广泛应用。布尔代数更是命题逻辑的代数化表达，是计算机硬件设计的数学基础。

---

## 1. 偏序集与格的等价定义

### 1.1 偏序集

**偏序集** $(P, \preceq)$ 由集合 $P$ 和偏序关系 $\preceq$（自反、反对称、传递）组成。

在偏序集中，$a$ 与 $b$ 的**下确界**（meet）记为 $a \wedge b = \inf\{a, b\}$，**上确界**（join）记为 $a \vee b = \sup\{a, b\}$。

### 1.2 格的两种等价定义

**代数定义**：$(L, \vee, \wedge)$ 是格，如果对 $\vee$ 和 $\wedge$ 满足交换律、结合律、幂等律和吸收律。

**偏序定义**：$(L, \preceq)$ 是格，如果 $L$ 中任意两个元素都有上确界和下确界。

**从偏序到代数**：$a \vee b = \sup\{a, b\}$，$a \wedge b = \inf\{a, b\}$。

**从代数到偏序**：定义 $a \preceq b \Leftrightarrow a \wedge b = a \Leftrightarrow a \vee b = b$。

### 1.3 格的基本性质

对格中任意元素 $a, b, c$：

| 性质 | 公式 |
|------|------|
| 幂等律 | $a \vee a = a$，$a \wedge a = a$ |
| 交换律 | $a \vee b = b \vee a$，$a \wedge b = b \wedge a$ |
| 结合律 | $(a \vee b) \vee c = a \vee (b \vee c)$ |
| 吸收律 | $a \vee (a \wedge b) = a$，$a \wedge (a \vee b) = a$ |
| 保序性 | $a \preceq b \Rightarrow a \vee c \preceq b \vee c$，$a \wedge c \preceq b \wedge c$ |

**注意**：格中一般**不满足分配律**。

## 2. 子格与格同态

### 2.1 子格

设 $(L, \vee, \wedge)$ 是格，$S \subseteq L$。若 $S$ 对 $\vee$ 和 $\wedge$ 封闭，则 $(S, \vee, \wedge)$ 是 $L$ 的**子格**。

**注意**：子格必须对两个运算都封闭。子格的子集在继承偏序关系后不一定保持子格性质（可能缺少上确界或下确界）。

### 2.2 格同态

映射 $\phi: L_1 \rightarrow L_2$ 是**格同态**，如果：

$$\phi(a \vee b) = \phi(a) \vee \phi(b), \quad \phi(a \wedge b) = \phi(a) \wedge \phi(b)$$

**格同态保持保序性**：若 $\phi$ 是格同态，则 $a \preceq b \Rightarrow \phi(a) \preceq \phi(b)$。

**注意**：保序映射不一定是格同态。格同态要求两个运算都保。

### 2.3 格的直积

设 $(L_1, \vee_1, \wedge_1)$ 和 $(L_2, \vee_2, \wedge_2)$ 是格，则 $L_1 \times L_2$ 在逐分量运算下构成格：

$$(a_1, a_2) \vee (b_1, b_2) = (a_1 \vee_1 b_1, a_2 \vee_2 b_2)$$
$$(a_1, a_2) \wedge (b_1, b_2) = (a_1 \wedge_1 b_1, a_2 \wedge_2 b_2)$$

## 3. 分配格的性质与判定

### 3.1 分配格定义

格 $L$ 是**分配格**，如果对任意 $a, b, c \in L$：

$$a \wedge (b \vee c) = (a \wedge b) \vee (a \wedge c)$$
$$a \vee (b \wedge c) = (a \vee b) \wedge (a \vee c)$$

注意：在格中，这两个分配律是等价的（一个成立可以推出另一个）。

### 3.2 分配格的判定

**定理**：格 $L$ 是分配格当且仅当 $L$ 不含有同构于以下两个格的子格：

**$N_5$（五角格）**：5 个元素 $\{0, a, b, c, 1\}$，其中 $a \wedge b = a \wedge c = 0$，$a \vee b = a \vee c = 1$，$b$ 和 $c$ 不可比。

**$M_3$（菱形格）**：5 个元素 $\{0, a, b, c, 1\}$，其中 $a, b, c$ 两两不可比，$a \vee b = a \vee c = b \vee c = 1$，$a \wedge b = a \wedge c = b \wedge c = 0$。

### 3.3 分配格的性质

- 每个全序集都是分配格。
- 分配格中补元若存在则唯一。
- 分配格中若 $a \wedge c = b \wedge c$ 且 $a \vee c = b \vee c$，则 $a = b$（消去律的格版本）。

---

## 4. 有界格与有补格

### 4.1 有界格

**有界格**是有最大元 $1$（全上界）和最小元 $0$（全下界）的格。

对任意 $a \in L$：
- $a \vee 1 = 1$，$a \wedge 1 = a$
- $a \vee 0 = a$，$a \wedge 0 = 0$

**有限格一定有界**：最大元 = 所有元素的 $\vee$，最小元 = 所有元素的 $\wedge$。

### 4.2 有补格

在有界格 $L$ 中，元素 $b$ 是 $a$ 的**补元**，如果：

$$a \vee b = 1, \quad a \wedge b = 0$$

**有补格**：有界格中每个元素都至少有一个补元。

**注意**：
- 非分配格中，一个元素可能有多个补元（如 $M_3$ 中 $a, b, c$ 互为补元）。
- 分配格中，补元若存在则唯一。

## 5. 布尔代数的定义与性质

### 5.1 定义

**布尔代数**是有补分配格，即同时满足分配律和有补性的有界格。

等价地，布尔代数 $(B, \vee, \wedge, \neg, 0, 1)$ 满足：
1. $(B, \vee, \wedge)$ 是有界分配格
2. 存在一元运算 $\neg$（补运算）：$a \vee \neg a = 1$，$a \wedge \neg a = 0$

### 5.2 基本性质

| 性质 | 公式 |
|------|------|
| 双重否定律 | $\neg(\neg a) = a$ |
| De Morgan 律 | $\neg(a \vee b) = \neg a \wedge \neg b$，$\neg(a \wedge b) = \neg a \vee \neg b$ |
| 零律 | $a \vee 1 = 1$，$a \wedge 0 = 0$ |
| 同一律 | $a \vee 0 = a$，$a \wedge 1 = a$ |
| 排中律 | $a \vee \neg a = 1$ |
| 矛盾律 | $a \wedge \neg a = 0$ |
| 幂等律 | $a \vee a = a$，$a \wedge a = a$ |
| 吸收律 | $a \vee (a \wedge b) = a$ |
| 分配律 | $a \wedge (b \vee c) = (a \wedge b) \vee (a \wedge c)$ |

### 5.3 重要例子

- **二元布尔代数** $\{0, 1\}$：最简单的布尔代数。
- **幂集布尔代数** $(\mathcal{P}(S), \cup, \cap, \bar{\cdot}, \emptyset, S)$：集合 $S$ 的幂集在并、交、补运算下构成布尔代数。
- **命题逻辑的等价类**：命题公式在逻辑等价关系下的商代数。

## 6. 有限布尔代数的结构定理（Stone 表示定理）

### 6.1 原子

布尔代数 $B$ 中的非零元素 $a$ 是**原子**（极小元），如果不存在 $0 < b < a$。

### 6.2 Stone 表示定理

**定理**：每个有限布尔代数 $B$ 都同构于某个有限集合 $S$ 的幂集布尔代数 $\mathcal{P}(S)$。

**推论**：
- 有限布尔代数的元素个数必为 $2^n$（$n$ 为原子个数）。
- 任意两个具有相同元素个数的有限布尔代数同构。
- 原子集 $S$ 的每个子集对应布尔代数中的一个元素（子集的各原子取 $\vee$）。

**证明思路**：
1. 设 $B$ 的原子集为 $S = \{s_1, s_2, \ldots, s_n\}$。
2. 定义映射 $\phi: \mathcal{P}(S) \rightarrow B$，$\phi(A) = \bigvee_{s \in A} s$（约定 $\phi(\emptyset) = 0$）。
3. 证明 $\phi$ 是双射且保运算。

### 6.3 结构的唯一性

$2$ 个元素的布尔代数只有一种（$\{0, 1\}$），$4$ 个元素的也只有一种（$\mathcal{P}(\{a\}) \cong \mathcal{P}(\{b\})$）。一般地，$2^n$ 个元素的布尔代数在同构意义下唯一。

## 7. 布尔代数与命题逻辑的关系

### 7.1 对应关系

| 布尔代数 | 命题逻辑 |
|----------|----------|
| $a \vee b$ | $P \vee Q$（析取） |
| $a \wedge b$ | $P \wedge Q$（合取） |
| $\neg a$ | $\neg P$（否定） |
| $0$ | 矛盾式（F） |
| $1$ | 重言式（T） |
| $a \preceq b$ | $A \Rightarrow B$（逻辑蕴含） |

### 7.2 等价类

命题公式在逻辑等价关系下构成布尔代数。每个等价类可由其主析取范式（或主合取范式）唯一确定。$n$ 个命题变元对应 $2^{2^n}$ 个不同的等价类（真值表有 $2^n$ 行，每行可取 T 或 F）。

### 7.3 对偶原理

在布尔代数中，将 $\vee$ 与 $\wedge$ 互换、$0$ 与 $1$ 互换，得到的对偶命题仍然成立。这正是命题逻辑中对偶律的代数表达。

## 8. 布尔代数在数字电路设计中的应用

### 8.1 基本逻辑门

| 逻辑门 | 布尔运算 | 符号 |
|--------|----------|------|
| 与门（AND） | $A \wedge B$ | 输出为 1 当且仅当所有输入为 1 |
| 或门（OR） | $A \vee B$ | 输出为 1 当且仅当至少一个输入为 1 |
| 非门（NOT） | $\neg A$ | 输出与输入相反 |
| 与非门（NAND） | $\neg(A \wedge B)$ | 与门后接非门 |
| 或非门（NOR） | $\neg(A \vee B)$ | 或门后接非门 |
| 异或门（XOR） | $A \oplus B = (A \wedge \neg B) \vee (\neg A \wedge B)$ | 输入不同时输出为 1 |

### 8.2 逻辑函数化简

利用布尔代数恒等式（特别是吸收律、分配律、De Morgan 律）化简逻辑表达式，减少门电路数量。

**卡诺图法**（K-Map）：
1. 将逻辑函数的真值表填入 $2^n$ 格的卡诺图。
2. 将相邻的 1 格合并为矩形块（每块大小为 $2^k$）。
3. 每个矩形块对应一个最简项，所有块的析取即为最简表达式。

**例**：化简 $F(A, B, C) = \sum m(1, 3, 5, 7)$：
- 卡诺图中 m1, m3, m5, m7 全为 1，可合并为 $C$（$A, B$ 任意）。
- 最简形式：$F = C$。

### 8.3 组合电路设计流程

1. 根据需求列出真值表。
2. 写出逻辑函数的表达式（最小项之和或最大项之积）。
3. 用卡诺图或代数方法化简。
4. 用逻辑门实现。

### 8.4 完备性

$\{\wedge, \vee, \neg\}$ 是**功能完备**的——任何布尔函数都可以用这三个运算表示。

**更小的完备集**：
- $\{\text{NAND}\}$：与非门单独就是功能完备的。
- $\{\text{NOR}\}$：或非门单独也是功能完备的。

这在实际电路设计中极为重要——只需一种门即可实现所有逻辑功能。
