01 - 格与布尔代数

离散数学 | 格与布尔代数

1. 格（Lattice）的定义

偏序定义

偏序集 (L, ≤) 是一个格，如果 L 中任意两个元素都存在上确界（LUB）和下确界（GLB）。

  • a ∨ b = sup{a, b} = LUB（上确界，也叫"并"或"join"）
  • a ∧ b = inf{a, b} = GLB（下确界，也叫"交"或"meet"）

代数定义

格也可以从代数角度定义：代数系统 (L, ∨, ∧) 满足以下性质：

律∨（并/join）∧（交/meet）
交换律a ∨ b = b ∨ aa ∧ b = b ∧ a
结合律(a∨b)∨c = a∨(b∨c)(a∧b)∧c = a∧(b∧c)
吸收律a ∨ (a ∧ b) = a   ;   a ∧ (a ∨ b) = a
幂等律a ∨ a = aa ∧ a = a

2. 格的示例

示例1：P({a,b,c}) 的子集格

集合 {a,b,c} 的幂集 P({a,b,c})，偏序为子集关系 ⊆。

任意两个子集的 LUB = 并集 ∪，GLB = 交集 ∩。

Hasse 图呈菱形对称。最上面是全集 {a,b,c}（最大元 1），最下面是空集（最小元 0）。

示例2：正整数的整除格

Z+ 的子集 Dn（n 的所有正因子），偏序为整除关系 |。

LUB = 最小公倍数 lcm，GLB = 最大公约数 gcd。

D12 = {1, 2, 3, 4, 6, 12}

3. 特殊格

类型条件示例
有界格存在最大元 1 和最小元 0(P(S), ⊆)，1=S, 0=∅
分配格满足分配律：a∧(b∨c)=(a∧b)∨(a∧c)链、P(S)
模格满足模律：a≥c ⇒ a∧(b∨c)=(a∧b)∨c分配格都是模格
有补格有界格中每个元素都有补元P(S) 中 A 的补 = S-A
布尔代数有补分配格(P(S), ∪, ∩,')

非分配格示例：钻石格 M5 和五角格 N5

M5 = {0, a, b, c, 1}（a, b, c 两两不可比较）和 N5 是最小的非分配格。

一个格是分配格 ↔ 它不包含 M5 或 N5 作为子格。

4. 布尔代数（Boolean Algebra）

定义

一个有补分配格称为布尔代数。代数系统 (B, ∨, ∧, ', 0, 1) 是布尔代数，满足：

律公式
交换律a∨b = b∨a  ;  a∧b = b∧a
结合律(a∨b)∨c = a∨(b∨c)
分配律a∧(b∨c) = (a∧b)∨(a∧c)
同一律a∨0 = a  ;  a∧1 = a
互补律a∨a' = 1  ;  a∧a' = 0
对偶（零一律）a∨1 = 1  ;  a∧0 = 0

5. 布尔代数中的德摩根律

德摩根律（De Morgan's Laws）

(a ∨ b)' = a' ∧ b'
(a ∧ b)' = a' ∨ b'

"并的补等于补的交，交的补等于补的并"

推广：(a₁ ∨ a₂ ∨ ... ∨ aₙ)' = a₁' ∧ a₂' ∧ ... ∧ aₙ'

德摩根律在数字逻辑电路设计（与非门、或非门）中有直接应用。
