1. 布尔代数回顾

布尔代数 (B, +, ·, ', 0, 1) 是有补分配格，满足：
  • 交换律、结合律、分配律
  • 同一律：a+0=a, a·1=a
  • 互补律：a+a'=1, a·a'=0
  • 幂等律、吸收律、对偶律(De Morgan)

3. 布尔函数的标准形式

3.1 最小项与最大项

最小项 (Minterm)：n 个变量的乘积项，每个变量以原形或补形出现恰好一次。
n 个变量有 2ⁿ 个最小项，记为 m₀, m₁, ..., m_(2ⁿ-1)。
恰好有一个输入使最小项为 1。

最大项 (Maxterm)：n 个变量的和项，每个变量以原形或补形出现恰好一次。
恰好有一个输入使最大项为 0。
最大项 Mᵢ = mᵢ'。

3.2 标准形式

SOP (积之和 / 标准与或式)：f = Σ mᵢ (对使 f=1 的所有 i)
POS (和之积 / 标准或与式)：f = Π Mᵢ (对使 f=0 的所有 i)

4. 卡诺图 (K-Map) 化简

4.1 基本原理

卡诺图是布尔函数的图形化表示，利用相邻最小项可合并的性质进行化简。
相邻性：两个最小项仅有一个变量不同，可合并消去该变量。
规则：圈 2ᵏ 个相邻的 1，圈越大消去的变量越多。

4.3 三变量卡诺图化简示例

f = Σ(1,3,5,6)，三变量卡诺图：
y'z' (00)y'z (01)yz (11)yz' (10)
x=001 (m₁)1 (m₃)0
x=101 (m₅)01 (m₆)

圈1：{m₁, m₃, m₅} = 3个1（注意 m₃m₅ 不相邻）→ 实际圈 {m₁,m₃} 和 {m₁,m₅} 和 {m₆}
{m₁,m₃} = x'z，{m₅} 不能与 m₁ 合并（跨越边界）→ {m₁,m₅} = y'z（图的左右相邻）
最简形式：f = y'z + x'z + xyz'（需进一步化简得 f = z(x'+y') + xyz'）

4.4 化简原则

  1. 圈必须是矩形，包含 2ᵏ 个格子
  2. 圈尽可能大（消去更多变量）
  3. 每个 1 至少被圈一次
  4. 圈的总数尽可能少
  5. 允许圈重叠（重叠可产生更简表达式）
  6. 卡诺图的左右和上下边界相邻（Gray码排列）

5. 无关项 (Don't Care)

某些输入组合在实际中不会出现，或输出值不影响系统功能，这些项称为无关项，标记为 d 或 X。
化简时可将无关项视为 0 或 1，取使表达式最简的那个值。
BCD 码（0000~1001）中，1010~1111 是无关项。
化简时可利用这些无关项得到更简单的电路。

6. 电路最小化

6.1 两级电路 (SOP / POS)

SOP 实现：与门 → 或门（AND-OR），适合"多数输入使输出为1"的函数。
POS 实现：或门 → 与门（OR-AND），适合"多数输入使输出为0"的函数。
选择标准：比较 SOP 和 POS 形式的复杂度。

6.2 多级电路优化

有时多级电路比两级更简洁：
因式分解：xy + xz = x(y+z) — 提取公共因子
代入法：f(x,y,z) = g(h(x,y), z) — 用子函数代替

7. Reed-Muller 展开

7.1 定义

Reed-Muller 展开（XOR 展开 / 正极性 Reed-Muller 形式）：
任何布尔函数可唯一表示为：
f(x₁,...,xₙ) = a₀ ⊕ a₁x₁ ⊕ a₂x₂ ⊕ a₃x₁x₂ ⊕ ... ⊕ aₖx₁x₂···xₙ
其中 aᵢ ∈ {0,1}，所有运算在 GF(2) 上进行（⊕ 为异或）。

7.2 特点

特性说明
唯一性每个布尔函数有唯一的 RM 展开
无冒险XOR 门无冒险（glitch-free）
易于测试结构规则，测试向量简单
FPGA 友好某些 FPGA 原生支持 XOR
自对偶检测a₀=1 时函数是自对偶的
f(x,y) = x ⊕ y ⊕ xy
真值表：f(0,0)=0, f(0,1)=1, f(1,0)=1, f(1,1)=1 — 即 OR 门！
但用 AND-NOT 形式：x+y = x ⊕ y ⊕ xy。

f(x,y) = x ⊕ y — 即 XOR 门（不含乘积项）。

7.3 从真值表到 RM 展开

方法一（代数替换）：将 SOP 中的 + 替换为 ⊕，再用 xy = x·y 化简。
注意：a+b = a⊕b⊕ab，所以 SOP → RM 需要展开所有 OR。

方法二（快速变换）：对真值表做 Hadamard 变换（类似 FFT）。

8. 门的完备性

完备集：一组逻辑门称为完备的，如果任意布尔函数都可用这组门实现。

完备集：
  • {AND, OR, NOT} — 经典完备集
  • {NAND} — 单门完备（万能门）
  • {NOR} — 单门完备
  • {AND, NOT} 或 {OR, NOT}
非完备集：{AND, OR}（缺少 NOT 无法实现取反）
NAND 的万能性：
NOT x = x NAND x
x AND y = NOT(x NAND y) = (x NAND y) NAND (x NAND y)
x OR y = (x NAND x) NAND (y NAND y)

本节总结

  • 布尔函数：f: {0,1}ⁿ → {0,1}，共 2^(2ⁿ) 个
  • 标准形式：SOP (Σmᵢ, 积之和) 和 POS (ΠMᵢ, 和之积)
  • 卡诺图化简：圈 2ᵏ 个相邻 1，利用 Gray 码相邻性，可含无关项
  • 电路最小化：两级 (AND-OR / OR-AND)，多级 (因式分解)
  • Reed-Muller 展开：f = a₀ ⊕ a₁x₁ ⊕ a₂x₂ ⊕ ...，XOR 形式唯一、无冒险
  • 门完备性：{NAND} 和 {NOR} 各自完备；{AND, OR} 不完备
