## 03 - 函数的性质


离散数学 | 关系与函数

==================== 1. 函数的定义 ====================

## 1. 函数的定义


### 函数是一种特殊的关系


**函数（Function）** f: A → B 是 A × B 的一个子集，满足：


- **存在性（处处有定义）：**
   对每个 a ∈ A，存在 b ∈ B 使得 (a,b) ∈ f
- **唯一性（单值性）：**
   对每个 a ∈ A，至多有一个 b ∈ B 使得 (a,b) ∈ f


即：**每个输入恰好对应一个输出**。

f: A → B    其中 A 称为
**定义域**
（Domain），B 称为
**陪域**
（Codomain）

> **Warning:** ### 函数 vs 一般关系
>
>
> | 特征 | 函数 | 一般关系 R ⊆ A×B |
> | --- | --- | --- |
> | 每个a是否必须有对应 | 是（处处有定义） | 否 |
> | 一个a能否对应多个b | 否（单值性） | 是 |
> | 一个b能否对应多个a | 是 | 是 |

==================== 2. 像与值域 ====================

## 2. 像（Image）与值域（Range）


**像：**f(a) = b 表示 a 映射到 b，b 称为 a 在 f 下的像。


**值域（Range）：**ran(f) = {f(a) | a ∈ A} ⊆ B — 函数所有输出值的集合。


**注意：**值域 ⊆ 陪域（值域是陪域的子集，不一定等于陪域）


**示例：**f: R → R, f(x) = x²，值域 = [0, +∞) ⊂ R（陪域）

==================== 3. 三种函数类型 ====================

## 3. 函数的三种性质


## 3.1 单射（Injective / One-to-One）


### 定义


f: A → B 是**单射的** ↔ 不同的输入一定映射到不同的输出。

∀a₁, a₂ ∈ A: f(a₁) = f(a₂) ⇒ a₁ = a₂

**等价表述：**a₁ ≠ a₂ ⇒ f(a₁) ≠ f(a₂)


**示例：**f(x) = 2x + 1 是单射；f(x) = x² 不是单射（f(2) = f(-2)）


**必要条件：**|A| ≤ |B|


## 3.2 满射（Surjective / Onto）


### 定义


f: A → B 是**满射的** ↔ 陪域中的每个元素都是某个输入的像。

∀b ∈ B, ∃a ∈ A: f(a) = b

**等价表述：**值域 = 陪域，即 ran(f) = B


**示例：**f: R → [0,+∞), f(x) = x² 是满射（到[0,+∞)）


**必要条件：**|A| ≥ |B|（有限集情况）


## 3.3 双射（Bijective / One-to-One Correspondence）


### 定义


f: A → B 是**双射的** ↔ f 既是单射又是满射。

双射 = 单射 + 满射

**含义：**每个A中的元素恰好对应B中的一个元素，反之亦然 — 一一对应。


**示例：**f: R → R, f(x) = 2x + 1 是双射


**必要条件：**|A| = |B|（有限集情况）


**重要性质：**双射函数必存在反函数。

==================== 4. 性质对比 ====================

## 4. 三种性质对比总结


| 性质 | 定义 | 条件 | 有限集条件 | 反函数？ |
| --- | --- | --- | --- | --- |
| **单射** | 不同输入不同输出 | f(a₁)=f(a₂) ⇒ a₁=a₂ | \|A\| ≤ \|B\| | 左逆存在 |
| **满射** | 陪域中每个元素都被映射到 | ∀b∈B, ∃a: f(a)=b | \|A\| ≥ \|B\| | 右逆存在 |
| **双射** | 一一对应 | 单射且满射 | \|A\| = \|B\| | 反函数存在且唯一 |


#### 示例判断


f: R→R, f(x) = x²


f(2) = f(-2) = 4 → 不是单射


-1 不在值域中 → 不是满射（到R）


结论：既非单射也非满射


#### 示例判断


f: R→R, f(x) = 2x+1


2x₁+1 = 2x₂+1 → x₁=x₂ → 单射


对任意y，x=(y-1)/2满足 → 满射


结论：双射，f⁻¹(y) = (y-1)/2

==================== 5. 函数复合 ====================

## 5. 函数复合（Composition）


### 定义


设 f: A → B，g: B → C，则**复合函数** g ∘ f: A → C 定义为：

(g ∘ f)(a) = g(f(a))

**执行顺序：**先执行 f，再执行 g（从右到左）。


### 复合函数的性质


| 性质 | 结论 |
| --- | --- |
| f 单射，g 单射 | g ∘ f 单射 |
| f 满射，g 满射 | g ∘ f 满射 |
| f 双射，g 双射 | g ∘ f 双射 |
| 交换律 | 不满足：g ∘ f ≠ f ∘ g（一般情况） |
| 结合律 | 满足：h ∘ (g ∘ f) = (h ∘ g) ∘ f |


## 5.1 复合示例


f: R → R, f(x) = x + 1


g: R → R, g(x) = 2x


**g ∘ f** = g(f(x)) = g(x+1) = 2(x+1) = `2x + 2`


**f ∘ g** = f(g(x)) = f(2x) = 2x + 1 = `2x + 1`


g ∘ f ≠ f ∘ g — 说明函数复合不满足交换律。

==================== 6. 反函数 ====================

## 6. 反函数（Inverse Function）


### 定义


如果 f: A → B 是**双射**，则存在唯一的**反函数** f⁻¹: B → A，满足：

f⁻¹(b) = a ↔ f(a) = b

**性质：**


- f ∘ f⁻¹ = I
   ~B~
   （B 上的恒等函数）
- f⁻¹ ∘ f = I
   ~A~
   （A 上的恒等函数）
- (f⁻¹)⁻¹ = f
- (g ∘ f)⁻¹ = f⁻¹ ∘ g⁻¹


> **Warning:** ### 只有双射才有反函数
>
>
> **非单射：**无法确定 f⁻¹(b) 应该返回哪个 a（多对一导致逆映射不唯一）
>
>
> **非满射：**某些 b ∈ B 没有对应的 a，f⁻¹(b) 未定义
>
>
> **示例：**f(x) = x² 不是双射（到R），所以没有反函数 f⁻¹: R → R。
>
>
> 但如果限制定义域：f: [0,+∞) → [0,+∞), f(x) = x² 是双射，反函数为 f⁻¹(x) = √x。

==================== 8. 有限集基数与函数 ====================

## 8. 有限集上的函数计数


| 函数类型 | 计数公式（\|A\|=n, \|B\|=m） |
| --- | --- |
| 所有函数 A→B | m^n^ |
| 单射 A→B（n≤m） | m! / (m-n)! = P(m,n) 排列数 |
| 满射 A→B（n≥m） | m! × S(n,m)，S为第二类Stirling数 |
| 双射 A→B（n=m） | n!（排列数） |

/.container
==========================================
    总结：
    1. 函数是满足"处处有定义"和"单值性"的特殊关系
    2. 三种性质：
       - 单射（Injective）：不同输入 → 不同输出（一对一）
       - 满射（Surjective）：每个陪域元素都被映射到（值域=陪域）
       - 双射（Bijective）：单射 + 满射（一一对应）
    3. 函数复合 g∘f：先f后g，不满足交换律但满足结合律
    4. 反函数 f⁻¹ 存在 ⟺ f是双射
    5. 有限集上：|A|=n,|B|=m，函数总数 m^n，双射数 n!
    ==========================================


## Python实现

### 函数性质判定（单射、满射、双射）

```python
from collections import Counter

def analyze_function(func, domain, codomain):
    """分析函数的单射、满射、双射性质"""
    images = {a: func(a) for a in domain}
    image_set = set(images.values())

    # 单射: 不同输入不同输出
    injective = len(image_set) == len(domain)

    # 满射: 值域 = 陪域
    surjective = image_set == codomain

    # 双射
    bijective = injective and surjective

    return {
        "单射 (Injective)": injective,
        "满射 (Surjective)": surjective,
        "双射 (Bijective)": bijective,
        "值域": image_set,
    }

# 示例1: f(x) = 2x + 1, R→R (有限模拟)
print("=== f(x) = 2x + 1, {-3..3} → {-3..3} ===")
domain = set(range(-3, 4))
codomain = set(range(-3, 4))
f1 = lambda x: 2 * x + 1
result = analyze_function(f1, domain, codomain)
for k, v in result.items():
    print(f"  {k}: {v}")

# 示例2: f(x) = x², R→R
print("\n=== f(x) = x², {-3..3} → {-3..3} ===")
f2 = lambda x: x ** 2
result = analyze_function(f2, domain, codomain)
for k, v in result.items():
    print(f"  {k}: {v}")

# 示例3: f(x) = x², {0..3} → {0..4} (限制定义域)
print("\n=== f(x) = x², {0..3} → {0..9} (限制后为满射) ===")
domain3 = {0, 1, 2, 3}
codomain3 = {0, 1, 4, 9}
result = analyze_function(f3 := lambda x: x**2, domain3, codomain3)
for k, v in result.items():
    print(f"  {k}: {v}")

# 反函数验证
print("\n=== 反函数验证 ===")
domain4 = set(range(1, 6))
f4 = lambda x: 2 * x + 1
f4_inv = lambda y: (y - 1) // 2

print("f(x) = 2x+1 的反函数验证:")
for x in domain4:
    y = f4(x)
    x_back = f4_inv(y)
    print(f"  f({x}) = {y}, f⁻¹({y}) = {x_back}, 完美还原: {x == x_back}")
```

### 函数复合与有限集计数

```python
from math import factorial, perm

# 函数复合
print("=== 函数复合 ===")
f = lambda x: x + 1
g = lambda x: 2 * x

# g ∘ f = g(f(x)) = 2(x+1) = 2x+2
# f ∘ g = f(g(x)) = 2x+1
print("f(x) = x+1, g(x) = 2x")
for x in range(5):
    gf = g(f(x))
    fg = f(g(x))
    print(f"  g∘f({x}) = g(f({x})) = {gf},  f∘g({x}) = f(g({x})) = {fg}")
print(f"g∘f ≠ f∘g: {g(f(3)) != f(g(3))}")

# 验证结合律: h∘(g∘f) = (h∘g)∘f
h = lambda x: x ** 2
x = 3
left = h(g(f(x)))       # h(g(f(x)))
right = (lambda x: h(g(x)))(f(x))  # (h∘g)(f(x))
print(f"\n结合律 h(g(f({x}))) = (h∘g)(f({x})): {left == right} ({left} = {right})")

# 有限集函数计数
print("\n=== 有限集函数计数 ===")
n, m = 4, 3
all_funcs = m ** n
injections = perm(m, n) if n <= m else 0
bijections = factorial(n) if n == m else 0

print(f"|A| = {n}, |B| = {m}")
print(f"所有函数 A→B: {m}^{n} = {all_funcs}")
print(f"单射 A→B: P({m},{n}) = {injections}")
print(f"双射 A→B (仅当n=m): {bijections}")

# 枚举验证 (小例子)
print("\n=== 枚举验证: A={1,2}, B={a,b,c} ===")
from itertools import product
A = [1, 2]
B = ['a', 'b', 'c']
all_count = 0
injective_count = 0
for assignments in product(B, repeat=len(A)):
    all_count += 1
    if len(set(assignments)) == len(assignments):
        injective_count += 1
print(f"所有函数: {all_count} (理论值 {len(B)}^{len(A)} = {len(B)**len(A)})")
print(f"单射: {injective_count} (理论值 P({len(B)},{len(A)}) = {perm(len(B), len(A))})")
```
