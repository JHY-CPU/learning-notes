03 - 函数的性质

离散数学 | 关系与函数

1. 函数的定义

函数是一种特殊的关系

函数（Function） f: A → B 是 A × B 的一个子集，满足：

  • 存在性（处处有定义）：对每个 a ∈ A，存在 b ∈ B 使得 (a,b) ∈ f
  • 唯一性（单值性）：对每个 a ∈ A，至多有一个 b ∈ B 使得 (a,b) ∈ f

即：每个输入恰好对应一个输出。

f: A → B    其中 A 称为定义域（Domain），B 称为陪域（Codomain）

函数 vs 一般关系

特征函数一般关系 R ⊆ A×B
每个a是否必须有对应是（处处有定义）否
一个a能否对应多个b否（单值性）是
一个b能否对应多个a是是

2. 像（Image）与值域（Range）

像：f(a) = b 表示 a 映射到 b，b 称为 a 在 f 下的像。

值域（Range）：ran(f) = {f(a) | a ∈ A} ⊆ B — 函数所有输出值的集合。

注意：值域 ⊆ 陪域（值域是陪域的子集，不一定等于陪域）

示例：f: R → R, f(x) = x²，值域 = [0, +∞) ⊂ R（陪域）

3. 函数的三种性质

3.1 单射（Injective / One-to-One）

定义

f: A → B 是单射的 ↔ 不同的输入一定映射到不同的输出。

∀a₁, a₂ ∈ A: f(a₁) = f(a₂) ⇒ a₁ = a₂

等价表述：a₁ ≠ a₂ ⇒ f(a₁) ≠ f(a₂)

示例：f(x) = 2x + 1 是单射；f(x) = x² 不是单射（f(2) = f(-2)）

必要条件：|A| ≤ |B|

3.2 满射（Surjective / Onto）

定义

f: A → B 是满射的 ↔ 陪域中的每个元素都是某个输入的像。

∀b ∈ B, ∃a ∈ A: f(a) = b

等价表述：值域 = 陪域，即 ran(f) = B

示例：f: R → [0,+∞), f(x) = x² 是满射（到[0,+∞)）

必要条件：|A| ≥ |B|（有限集情况）

3.3 双射（Bijective / One-to-One Correspondence）

定义

f: A → B 是双射的 ↔ f 既是单射又是满射。

双射 = 单射 + 满射

含义：每个A中的元素恰好对应B中的一个元素，反之亦然 — 一一对应。

示例：f: R → R, f(x) = 2x + 1 是双射

必要条件：|A| = |B|（有限集情况）

重要性质：双射函数必存在反函数。

4. 三种性质对比总结

性质定义条件有限集条件反函数？
单射 不同输入不同输出 f(a₁)=f(a₂) ⇒ a₁=a₂ |A| ≤ |B| 左逆存在
满射 陪域中每个元素都被映射到 ∀b∈B, ∃a: f(a)=b |A| ≥ |B| 右逆存在
双射 一一对应 单射且满射 |A| = |B| 反函数存在且唯一

示例判断

f: R→R, f(x) = x²

f(2) = f(-2) = 4 → 不是单射

-1 不在值域中 → 不是满射（到R）

结论：既非单射也非满射

示例判断

f: R→R, f(x) = 2x+1

2x₁+1 = 2x₂+1 → x₁=x₂ → 单射

对任意y，x=(y-1)/2满足 → 满射

结论：双射，f⁻¹(y) = (y-1)/2

5. 函数复合（Composition）

定义

设 f: A → B，g: B → C，则复合函数 g ∘ f: A → C 定义为：

(g ∘ f)(a) = g(f(a))

执行顺序：先执行 f，再执行 g（从右到左）。

复合函数的性质

性质结论
f 单射，g 单射g ∘ f 单射
f 满射，g 满射g ∘ f 满射
f 双射，g 双射g ∘ f 双射
交换律不满足：g ∘ f ≠ f ∘ g（一般情况）
结合律满足：h ∘ (g ∘ f) = (h ∘ g) ∘ f

5.1 复合示例

f: R → R, f(x) = x + 1

g: R → R, g(x) = 2x

g ∘ f = g(f(x)) = g(x+1) = 2(x+1) = 2x + 2

f ∘ g = f(g(x)) = f(2x) = 2x + 1 = 2x + 1

g ∘ f ≠ f ∘ g — 说明函数复合不满足交换律。

6. 反函数（Inverse Function）

定义

如果 f: A → B 是双射，则存在唯一的反函数 f⁻¹: B → A，满足：

f⁻¹(b) = a ↔ f(a) = b

性质：

  • f ∘ f⁻¹ = IB（B 上的恒等函数）
  • f⁻¹ ∘ f = IA（A 上的恒等函数）
  • (f⁻¹)⁻¹ = f
  • (g ∘ f)⁻¹ = f⁻¹ ∘ g⁻¹

只有双射才有反函数

非单射：无法确定 f⁻¹(b) 应该返回哪个 a（多对一导致逆映射不唯一）

非满射：某些 b ∈ B 没有对应的 a，f⁻¹(b) 未定义

示例：f(x) = x² 不是双射（到R），所以没有反函数 f⁻¹: R → R。

但如果限制定义域：f: [0,+∞) → [0,+∞), f(x) = x² 是双射，反函数为 f⁻¹(x) = √x。
