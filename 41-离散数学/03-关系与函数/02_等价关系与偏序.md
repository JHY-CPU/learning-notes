==================== 1. 等价关系 ====================

## 1. 等价关系（Equivalence Relation）


### 定义


集合 A 上的关系 R 如果同时满足**自反性**、**对称性**和**传递性**，则称 R 为**等价关系**。

R 是等价关系 ↔ R 是自反的 ∧ R 是对称的 ∧ R 是传递的

### 等价关系示例


| 关系 | 自反 | 对称 | 传递 | 等价关系？ |
| --- | --- | --- | --- | --- |
| 整数上的"模n同余" | ✓ | ✓ | ✓ | 是 |
| "同龄"关系 | ✓ | ✓ | ✓ | 是 |
| "等于"关系 = | ✓ | ✓ | ✓ | 是 |
| "小于等于" ≤ | ✓ | × | ✓ | 否 |
| "朋友"关系 | × | ✓ | × | 否 |

==================== 2. 等价类 ====================

## 2. 等价类（Equivalence Class）


### 定义


设 R 是 A 上的等价关系，对任意 a ∈ A，a 的**等价类** [a]~R~ 定义为：

[a]
~R~
= {x ∈ A | (x, a) ∈ R}

即所有与 a 有等价关系 R 的元素组成的集合。


### 示例：模3同余


A = Z（整数集），R = "模3同余"（即 a-b 能被3整除）


等价类：

[0] = 3的倍数
{..., -6, -3, 0, 3, 6, ...}
[1] = 余1
{..., -5, -2, 1, 4, 7, ...}
[2] = 余2
{..., -4, -1, 2, 5, 8, ...}

注意：[0] = [3] = [6] = ...（同一等价类的不同代表元）


### 等价类的重要性质


- 对任意 a，a ∈ [a]（由自反性保证）
- [a] ≠ ∅（等价类非空）
- 若 (a,b) ∈ R，则 [a] = [b]（由对称性和传递性保证）
- 若 (a,b) ∉ R，则 [a] ∩ [b] = ∅

==================== 3. 商集 ====================

## 3. 商集（Quotient Set）


### 定义


等价关系 R 的所有不同等价类组成的集合称为 A 关于 R 的**商集**，记为 A/R。

A/R = {[a]
~R~
| a ∈ A}

**划分：**等价关系 R 将集合 A 划分为若干互不相交的等价类，这些等价类的并集等于 A。


**示例：**Z/（模3同余）= {[0], [1], [2]}，即 3 个等价类。


### 等价关系与划分的对应


**一一对应关系：**


- A 上的每个等价关系 ↔ A 的一个划分
- A 的每个划分 ↔ A 上的一个等价关系


即：**等价关系 ≡ 划分**

==================== 4. 偏序关系 ====================

## 4. 偏序关系（Partial Order）


### 定义


集合 A 上的关系 R 如果同时满足**自反性**、**反对称性**和**传递性**，则称 R 为**偏序关系**（简称偏序）。

R 是偏序 ↔ R 是自反的 ∧ R 是反对称的 ∧ R 是传递的

记作 (A, ≤) 或 (A, R)，称 A 为**偏序集**（Poset）。


### 偏序关系示例


| 偏序关系 | 说明 |
| --- | --- |
| (N, ≤) | 自然数集上的小于等于关系 |
| (P(S), ⊆) | 集合S的幂集上的子集关系 |
| (Z^+^, \| ) | 正整数集上的整除关系 |
| (R, ≤) | 实数集上的小于等于关系 |

==================== 5. 全序与偏序的区别 ====================

## 5. 全序（Total Order）vs 偏序


> **Warning:** ### 全序的定义
>
>
> 偏序 (A, ≤) 如果还满足**完全性**（任意两个元素都可比较），则称为**全序**（或线序）。
>
> ∀a, b ∈ A: a ≤ b 或 b ≤ a（至少成立一个）
>
> **全序 = 偏序 + 完全性**


| 偏序集 | 是否全序 | 说明 |
| --- | --- | --- |
| (N, ≤) | 是 | 任意两个自然数可比较大小 |
| (P({a,b}), ⊆) | 否 | {a} 和 {b} 不可比较 |
| (Z^+^, \| ) | 否 | 2 和 3 互不整除 |

==================== 6. Hasse 图 ====================

## 6. Hasse 图（哈斯图）


### 定义


**Hasse 图**是偏序关系的可视化表示，通过"覆盖关系"简化偏序图。


**覆盖（Cover）：**a 覆盖 b（记为 b ≺ a），如果 b < a 且不存在 c 使得 b < c < a。


**绘制规则：**


- 去掉自环（自反边）
- 去掉传递边（只保留"直接覆盖"边）
- 较小元素画在下方，省略箭头（默认向上）

==================== 7. 偏序中的特殊元素 ====================

## 7. 偏序中的特殊元素


| 概念 | 定义 | 示例（N, ≤） |
| --- | --- | --- |
| **最大元** | b 使得对所有 a，有 a ≤ b | 有限子集有最大元，N无最大元 |
| **最小元** | b 使得对所有 a，有 b ≤ a | 0（在N中） |
| **极大元** | b 使得不存在 a 满足 b < a | 有限子集的最后一个元素 |
| **极小元** | b 使得不存在 a 满足 a < b | 0（在N中） |
| **上界** | c 使得对子集 S 中所有 a，有 a ≤ c | S={2,3} 的上界 ∈ {3,4,5,...} |
| **下界** | c 使得对子集 S 中所有 a，有 c ≤ a | S={2,3} 的下界 ∈ {0,1,2} |
| **上确界（LUB）** | 最小的上界 | S={2,3} 的上确界 = 3 |
| **下确界（GLB）** | 最大的下界 | S={2,3} 的下确界 = 2 |


> **Warning:** ### 格（Lattice）的定义
>
>
> 如果偏序集中**任意两个元素**都存在上确界和下确界，则该偏序集称为**格**。
>
>
> **示例：**(P(S), ⊆) 是格 — 任意两个子集的上确界为并集，下确界为交集。
>
>
> (Z^+^, |) 是格 — 任意两个正整数的上确界为最小公倍数(LCM)，下确界为最大公约数(GCD)。

### 等价类与商集计算

```python
from collections import defaultdict

def equivalence_classes(A, R):
    """计算等价关系 R 在集合 A 上的等价类"""
    # 构建邻接表
    adj = defaultdict(set)
    for (a, b) in R:
        adj[a].add(b)
        adj[b].add(a)

    visited = set()
    classes = []
    for a in A:
        if a not in visited:
            # BFS 找连通分量
            queue = [a]
            cls = set()
            while queue:
                node = queue.pop(0)
                if node not in visited:
                    visited.add(node)
                    cls.add(node)
                    queue.extend(adj[node] - visited)
            classes.append(cls)
    return classes

# 示例: 模5同余
print("=== 模5同余等价类 ===")
A = set(range(15))
R_mod5 = {(a, b) for a in A for b in A if (a - b) % 5 == 0}
classes = equivalence_classes(A, R_mod5)
for i, cls in enumerate(sorted(classes, key=min)):
    print(f"  [{min(cls)}] = {sorted(cls)}")

# 验证等价关系性质
print(f"\n自反性: 所有 (a,a) ∈ R: {all((a, a) in R_mod5 for a in A)}")
print(f"对称性: (a,b)∈R → (b,a)∈R: {all((b, a) in R_mod5 for a, b in R_mod5)}")
print(f"传递性验证中...")
# 检查传递性
transitive = True
for a in A:
    for b in A:
        if (a, b) in R_mod5:
            for c in A:
                if (b, c) in R_mod5 and (a, c) not in R_mod5:
                    transitive = False
print(f"传递性: {transitive}")

# 验证等价类的划分性质
print(f"\n等价类互不相交: {all(c1 & c2 == set() or c1 == c2 for c1 in classes for c2 in classes)}")
print(f"等价类的并 = A: {set().union(*classes) == A}")
```

```python
def is_partial_order(A, R):
    """检查 R 是否是 A 上的偏序关系"""
    reflexive = all((a, a) in R for a in A)
    antisymmetric = all(
        a == b for a, b in R if (b, a) in R
    )
    transitive = all(
        (a, c) in R for a, b in R for b2, c in R if b == b2
    )
    return reflexive, antisymmetric, transitive

# 整除偏序
print("=== 整除偏序 ===")
D12 = {1, 2, 3, 4, 6, 12}
R_div = {(a, b) for a in D12 for b in D12 if b % a == 0}
refl, antisym, trans = is_partial_order(D12, R_div)
print(f"D_12 = {sorted(D12)}")
print(f"自反: {refl}, 反对称: {antisym}, 传递: {trans}")
print(f"是偏序: {refl and antisym and trans}")

# 找最大元、最小元
def find_special_elements(A, R):
    """找偏序集中的最大元、最小元、极大元、极小元"""
    # 最大元: 所有元素都 ≤ b
    max_elems = [b for b in A if all((a, b) in R for a in A)]
    # 最小元: b ≤ 所有元素
    min_elems = [b for b in A if all((b, a) in R for a in A)]
    # 极大元: 不存在 a > b
    maximal = [b for b in A if not any((b, a) in R and (a, b) not in R for a in A if a != b)]
    # 极小元: 不存在 a < b
    minimal = [b for b in A if not any((a, b) in R and (b, a) not in R for a in A if a != b)]
    return max_elems, min_elems, maximal, minimal

max_e, min_e, maxl, minl = find_special_elements(D12, R_div)
print(f"最大元: {max_e}, 最小元: {min_e}")
print(f"极大元: {maxl}, 极小元: {minl}")

# 上确界 (lcm) 和下确界 (gcd)
from math import gcd
def lcm(a, b):
    return a * b // gcd(a, b)

print(f"\n=== 上下确界 ===")
print(f"sup{{4, 6}} = lcm(4,6) = {lcm(4, 6)}")
print(f"inf{{4, 6}} = gcd(4,6) = {gcd(4, 6)}")
print(f"sup{{4,6}} ∈ D_12: {lcm(4, 6) in D12}")
print(f"inf{{4,6}} ∈ D_12: {gcd(4, 6) in D12}")
```
