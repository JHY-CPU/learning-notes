==================== 1. 代数系统 ====================

## 1. 代数系统概述


### 什么是代数结构？


**代数结构**是一个非空集合 S 连同定义在 S 上的一个或多个运算组成的系统。


例如：(Z, +) 是整数集连同加法运算组成的代数系统。


代数结构的研究关注**运算的性质**（结合律、交换律、单位元、逆元等），而非具体元素。

==================== 2. 二元运算 ====================

## 2. 二元运算的性质


**二元运算** *: S × S → S，将 S 中的两个元素映射到 S 中的一个元素。


| 性质 | 定义 | 示例 |
| --- | --- | --- |
| 封闭性 | ∀a,b∈S: a*b∈S | 整数加法封闭 |
| 结合律 | (a*b)*c = a*(b*c) | 加法、乘法满足 |
| 交换律 | a*b = b*a | 加法满足，减法不满足 |
| 单位元（幺元） | ∃e: a*e = e*a = a | 加法的0，乘法的1 |
| 逆元 | ∀a, ∃a⁻¹: a*a⁻¹ = a⁻¹*a = e | 加法中-a是a的逆元 |
| 消去律 | a*b=a*c ⇒ b=c（a可消去） | 整数乘法满足 |
| 分配律 | a*(b+c) = a*b + a*c | 乘法对加法分配 |
| 幂等律 | a*a = a | 集合的并和交满足 |

==================== 3. 群的定义 ====================

## 3. 群（Group）的定义


### 群的公理


代数系统 (G, *) 是一个**群**，如果满足以下四条公理：


| 公理 | 名称 | 表述 |
| --- | --- | --- |
| G1 | 封闭性 | ∀a,b∈G: a*b∈G |
| G2 | 结合律 | ∀a,b,c∈G: (a*b)*c = a*(b*c) |
| G3 | 单位元 | ∃e∈G: ∀a, a*e = e*a = a |
| G4 | 逆元 | ∀a∈G, ∃a⁻¹∈G: a*a⁻¹ = a⁻¹*a = e |


如果还满足**交换律**（G5: a*b = b*a），则称为**阿贝尔群（Abelian Group）**或交换群。


> **Warning:** ### 群的层次结构
>
> 代数系统 ⊃ 半群（封闭+结合）⊃ 独异点/幺半群（+单位元）⊃ 群（+逆元）⊃ 阿贝尔群（+交换律）
>
> | 结构 | 封闭 | 结合 | 单位元 | 逆元 | 交换 |
> | --- | --- | --- | --- | --- | --- |
> | 原群（Magma） | ✓ |  |  |  |  |
> | 半群（Semigroup） | ✓ | ✓ |  |  |  |
> | 幺半群（Monoid） | ✓ | ✓ | ✓ |  |  |
> | 群（Group） | ✓ | ✓ | ✓ | ✓ |  |
> | 阿贝尔群 | ✓ | ✓ | ✓ | ✓ | ✓ |

==================== 4. 群的示例 ====================

## 4. 重要群的示例


## 4.1 整数加法群 (Z, +)


- **集合：**
   Z = {..., -2, -1, 0, 1, 2, ...}
- **运算：**
   普通加法 +
- **封闭性：**
   两个整数之和仍为整数
- **结合律：**
   (a+b)+c = a+(b+c)
- **单位元：**
   0（a+0 = 0+a = a）
- **逆元：**
   a 的逆元是 -a（a + (-a) = 0）
- **交换律：**
   a+b = b+a →
   **阿贝尔群**
- **阶：**
   无限群（|Z| = ∞）


## 4.2 模 n 整数加法群 (Z~n~, +~n~)


- **集合：**
   Z
   ~n~
   = {0, 1, 2, ..., n-1}
- **运算：**
   a +
   ~n~
   b = (a + b) mod n
- **单位元：**
   0
- **逆元：**
   a 的逆元是 n-a（当 a ≠ 0）
- **阶：**
   n（有限群）
- **是阿贝尔群**

Cayley 表：Z₆

## 4.3 对称群 S~n~（置换群）


> **Warning:** ### 定义
>
>
> **对称群 S~n~** 是集合 {1,2,...,n} 上所有双射（置换）组成的群，运算为函数复合。
>
>
> **阶：**|S~n~| = n!
>
>
> **单位元：**恒等置换 id = (1)(2)...(n)
>
>
> **逆元：**置换 σ 的逆置换 σ⁻¹
>
>
> **S~3~（3阶对称群）**有 3! = 6 个元素，是最小的非阿贝尔群。


### S~3~ 的元素（用轮换记号表示）


| 置换 | 轮换记号 | 映射 | 阶 |
| --- | --- | --- | --- |
| e | (1)(2)(3) | 1→1, 2→2, 3→3 | 1 |
| (1 2) | (1 2)(3) | 1→2, 2→1, 3→3 | 2 |
| (1 3) | (1 3)(2) | 1→3, 2→2, 3→1 | 2 |
| (2 3) | (1)(2 3) | 1→1, 2→3, 3→2 | 2 |
| (1 2 3) | (1 2 3) | 1→2, 2→3, 3→1 | 3 |
| (1 3 2) | (1 3 2) | 1→3, 3→2, 2→1 | 3 |


**非交换性：**(1 2) ∘ (1 3) = (1 3 2)，而 (1 3) ∘ (1 2) = (1 2 3)。不相等！

==================== 5. 群的重要性质 ====================

## 5. 群的重要性质


| 性质 | 内容 |
| --- | --- |
| 单位元唯一 | 群中的单位元 e 是唯一的 |
| 逆元唯一 | 每个元素的逆元是唯一的 |
| (a⁻¹)⁻¹ = a | 逆元的逆元是自身 |
| (a*b)⁻¹ = b⁻¹*a⁻¹ | 积的逆 = 逆的积（反序） |
| 消去律 | a*b = a*c ⇒ b = c（左消去律） |
| 方程可解 | a*x = b 有唯一解 x = a⁻¹*b |
| 元素的阶 | 使 a^m^ = e 的最小正整数 m |

==================== 6. 子群 ====================

## 6. 子群（Subgroup）


### 定义


群 G 的非空子集 H，如果对 G 的运算也构成群，则称 H 是 G 的**子群**，记为 H ≤ G。


**判定定理：**H 是 G 的子群 ↔


- ∀a,b ∈ H: a*b ∈ H（封闭性）
- ∀a ∈ H: a⁻¹ ∈ H（逆元在H中）


**示例：**({0, 2, 4}, +~6~) 是 (Z~6~, +~6~) 的子群。
