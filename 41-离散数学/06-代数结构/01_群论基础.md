==================== 1. 代数系统 ====================

## 1. 代数系统概述


### 什么是代数结构？


**代数结构**是一个非空集合 S 连同定义在 S 上的一个或多个运算组成的系统。


例如：(Z, +) 是整数集连同加法运算组成的代数系统。


代数结构的研究关注**运算的性质**（结合律、交换律、单位元、逆元等），而非具体元素。

==================== 2. 二元运算 ====================

## 2. 二元运算的性质


**二元运算** *: S × S → S，将 S 中的两个元素映射到 S 中的一个元素。


| 性质 | 定义 | 示例 |
| --- | --- | --- |
| 封闭性 | ∀a,b∈S: a*b∈S | 整数加法封闭 |
| 结合律 | (a*b)*c = a*(b*c) | 加法、乘法满足 |
| 交换律 | a*b = b*a | 加法满足，减法不满足 |
| 单位元（幺元） | ∃e: a*e = e*a = a | 加法的0，乘法的1 |
| 逆元 | ∀a, ∃a⁻¹: a*a⁻¹ = a⁻¹*a = e | 加法中-a是a的逆元 |
| 消去律 | a*b=a*c ⇒ b=c（a可消去） | 整数乘法满足 |
| 分配律 | a*(b+c) = a*b + a*c | 乘法对加法分配 |
| 幂等律 | a*a = a | 集合的并和交满足 |

==================== 3. 群的定义 ====================

## 3. 群（Group）的定义


### 群的公理


代数系统 (G, *) 是一个**群**，如果满足以下四条公理：


| 公理 | 名称 | 表述 |
| --- | --- | --- |
| G1 | 封闭性 | ∀a,b∈G: a*b∈G |
| G2 | 结合律 | ∀a,b,c∈G: (a*b)*c = a*(b*c) |
| G3 | 单位元 | ∃e∈G: ∀a, a*e = e*a = a |
| G4 | 逆元 | ∀a∈G, ∃a⁻¹∈G: a*a⁻¹ = a⁻¹*a = e |


如果还满足**交换律**（G5: a*b = b*a），则称为**阿贝尔群（Abelian Group）**或交换群。


> **Warning:** ### 群的层次结构
>
> 代数系统 ⊃ 半群（封闭+结合）⊃ 独异点/幺半群（+单位元）⊃ 群（+逆元）⊃ 阿贝尔群（+交换律）
>
> | 结构 | 封闭 | 结合 | 单位元 | 逆元 | 交换 |
> | --- | --- | --- | --- | --- | --- |
> | 原群（Magma） | ✓ |  |  |  |  |
> | 半群（Semigroup） | ✓ | ✓ |  |  |  |
> | 幺半群（Monoid） | ✓ | ✓ | ✓ |  |  |
> | 群（Group） | ✓ | ✓ | ✓ | ✓ |  |
> | 阿贝尔群 | ✓ | ✓ | ✓ | ✓ | ✓ |

==================== 4. 群的示例 ====================

## 4. 重要群的示例


## 4.1 整数加法群 (Z, +)


- **集合：**
   Z = {..., -2, -1, 0, 1, 2, ...}
- **运算：**
   普通加法 +
- **封闭性：**
   两个整数之和仍为整数
- **结合律：**
   (a+b)+c = a+(b+c)
- **单位元：**
   0（a+0 = 0+a = a）
- **逆元：**
   a 的逆元是 -a（a + (-a) = 0）
- **交换律：**
   a+b = b+a →
   **阿贝尔群**
- **阶：**
   无限群（|Z| = ∞）


## 4.2 模 n 整数加法群 (Z~n~, +~n~)


- **集合：**
   Z
   ~n~
   = {0, 1, 2, ..., n-1}
- **运算：**
   a +
   ~n~
   b = (a + b) mod n
- **单位元：**
   0
- **逆元：**
   a 的逆元是 n-a（当 a ≠ 0）
- **阶：**
   n（有限群）
- **是阿贝尔群**

Cayley 表：Z₆

## 4.3 对称群 S~n~（置换群）


> **Warning:** ### 定义
>
>
> **对称群 S~n~** 是集合 {1,2,...,n} 上所有双射（置换）组成的群，运算为函数复合。
>
>
> **阶：**|S~n~| = n!
>
>
> **单位元：**恒等置换 id = (1)(2)...(n)
>
>
> **逆元：**置换 σ 的逆置换 σ⁻¹
>
>
> **S~3~（3阶对称群）**有 3! = 6 个元素，是最小的非阿贝尔群。


### S~3~ 的元素（用轮换记号表示）


| 置换 | 轮换记号 | 映射 | 阶 |
| --- | --- | --- | --- |
| e | (1)(2)(3) | 1→1, 2→2, 3→3 | 1 |
| (1 2) | (1 2)(3) | 1→2, 2→1, 3→3 | 2 |
| (1 3) | (1 3)(2) | 1→3, 2→2, 3→1 | 2 |
| (2 3) | (1)(2 3) | 1→1, 2→3, 3→2 | 2 |
| (1 2 3) | (1 2 3) | 1→2, 2→3, 3→1 | 3 |
| (1 3 2) | (1 3 2) | 1→3, 3→2, 2→1 | 3 |


**非交换性：**(1 2) ∘ (1 3) = (1 3 2)，而 (1 3) ∘ (1 2) = (1 2 3)。不相等！

==================== 5. 群的重要性质 ====================

## 5. 群的重要性质


| 性质 | 内容 |
| --- | --- |
| 单位元唯一 | 群中的单位元 e 是唯一的 |
| 逆元唯一 | 每个元素的逆元是唯一的 |
| (a⁻¹)⁻¹ = a | 逆元的逆元是自身 |
| (a*b)⁻¹ = b⁻¹*a⁻¹ | 积的逆 = 逆的积（反序） |
| 消去律 | a*b = a*c ⇒ b = c（左消去律） |
| 方程可解 | a*x = b 有唯一解 x = a⁻¹*b |
| 元素的阶 | 使 a^m^ = e 的最小正整数 m |

==================== 6. 子群 ====================

## 6. 子群（Subgroup）


### 定义


群 G 的非空子集 H，如果对 G 的运算也构成群，则称 H 是 G 的**子群**，记为 H ≤ G。


**判定定理：**H 是 G 的子群 ↔


- ∀a,b ∈ H: a*b ∈ H（封闭性）
- ∀a ∈ H: a⁻¹ ∈ H（逆元在H中）


**示例：**({0, 2, 4}, +~6~) 是 (Z~6~, +~6~) 的子群。

### 群的性质验证与 Cayley 表

```python
import numpy as np

class FiniteGroup:
    """有限群的通用实现"""
    def __init__(self, elements, operation, name="G"):
        self.elements = elements
        self.op = operation
        self.name = name
        self.n = len(elements)
        self.idx = {e: i for i, e in enumerate(elements)}

    def cayley_table(self):
        """生成 Cayley 表"""
        table = np.empty((self.n, self.n), dtype=object)
        for i, a in enumerate(self.elements):
            for j, b in enumerate(self.elements):
                table[i][j] = self.op(a, b)
        return table

    def is_closed(self):
        """封闭性"""
        table = self.cayley_table()
        return all(table[i][j] in self.elements
                   for i in range(self.n) for j in range(self.n))

    def is_associative(self):
        """结合律"""
        for a in self.elements:
            for b in self.elements:
                for c in self.elements:
                    if self.op(self.op(a, b), c) != self.op(a, self.op(b, c)):
                        return False
        return True

    def identity(self):
        """单位元"""
        for e in self.elements:
            if all(self.op(e, a) == a and self.op(a, e) == a for a in self.elements):
                return e
        return None

    def inverse(self, a):
        """逆元"""
        e = self.identity()
        for b in self.elements:
            if self.op(a, b) == e and self.op(b, a) == e:
                return b
        return None

    def is_abelian(self):
        """交换律"""
        return all(self.op(a, b) == self.op(b, a)
                   for a in self.elements for b in self.elements)

    def verify_group(self):
        """验证群的四条公理"""
        print(f"\n=== {self.name} 群性质验证 ===")
        print(f"  封闭性: {self.is_closed()}")
        print(f"  结合律: {self.is_associative()}")
        e = self.identity()
        print(f"  单位元: {e}")
        print(f"  逆元存在: {all(self.inverse(a) is not None for a in self.elements)}")
        print(f"  阿贝尔群: {self.is_abelian()}")

# 模6加法群 (Z6, +6)
print("=== 模6加法群 Z6 ===")
Z6_elements = list(range(6))
Z6_op = lambda a, b: (a + b) % 6
Z6 = FiniteGroup(Z6_elements, Z6_op, "Z6")
Z6.verify_group()

# 打印 Cayley 表
table = Z6.cayley_table()
print("\nCayley 表:")
print(f"{'+6':>4}", end="")
for e in Z6_elements:
    print(f" {e:>3}", end="")
print()
for i, a in enumerate(Z6_elements):
    print(f"{a:>4}", end="")
    for j in range(Z6.n):
        print(f" {table[i][j]:>3}", end="")
    print()

# S3 对称群 (最小非阿贝尔群)
print("\n=== S3 对称群 ===")
from itertools import permutations
S3_elements = [tuple(p) for p in permutations(range(3))]
S3_op = lambda a, b: tuple(a[b[i]] for i in range(3))
S3 = FiniteGroup(S3_elements, S3_op, "S3")
S3.verify_group()

# 子群检测
print("\n=== 子群检测: {0,2,4} ≤ Z6 ===")
sub_elements = [0, 2, 4]
sub_closed = all(Z6_op(a, b) in sub_elements for a in sub_elements for b in sub_elements)
sub_inverses = all(Z6.inverse(a) in sub_elements for a in sub_elements)
print(f"  封闭性: {sub_closed}")
print(f"  逆元在子集中: {sub_inverses}")
print(f"  是子群: {sub_closed and sub_inverses}")
```

```python
def element_order(a, operation, identity):
    """计算元素 a 的阶"""
    current = a
    for n in range(1, 1000):
        if current == identity:
            return n
        current = operation(current, a)
    return None
```
