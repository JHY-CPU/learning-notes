# 环与域 - 离散数学笔记


## 02 - 环与域


离散数学 | 代数结构

==================== 1. 环的定义 ====================

## 1. 环（Ring）的定义


### 环的公理


代数系统 (R, +, ·) 是一个**环**，如果满足：


| 类别 | 公理 | 说明 |
| --- | --- | --- |
| **(R, +) 是阿贝尔群** | 加法封闭性 | ∀a,b: a+b∈R |
| 加法结合律 | (a+b)+c = a+(b+c) |
| 加法单位元 | ∃0: a+0 = 0+a = a |
| 加法逆元 | ∀a, ∃(-a): a+(-a) = 0 |
| **(R, ·) 是半群** | 乘法封闭性 | ∀a,b: a·b∈R |
| 乘法结合律 | (a·b)·c = a·(b·c) |
| 加法交换律 | a+b = b+a |
| **分配律** | a·(b+c) = a·b + a·c  ;  (b+c)·a = b·a + c·a |


> **Warning:** ### 环的附加性质
>
>
> | 名称 | 条件 | 示例 |
> | --- | --- | --- |
> | 交换环 | 乘法满足交换律 | (Z, +, ·) |
> | 含幺环 | 乘法有单位元 1 | (Z, +, ·) 含幺元 1 |
> | 整环 | 交换 + 含幺 + 无零因子 | (Z, +, ·) |
> | 除环/体 | 非零元素对乘法构成群 | (Q, +, ·) |
> | 域 | 交换除环 | (Q, R, C) |
>
>
> **零因子：**在环 R 中，若 a≠0, b≠0 但 a·b=0，则称 a, b 为零因子。

==================== 2. 环的示例 ====================

## 2. 重要的环


### 示例对比


| 代数系统 | 交换环 | 含幺 | 零因子 | 整环 | 域 |
| --- | --- | --- | --- | --- | --- |
| (Z, +, ·) | ✓ | ✓ | 无 | ✓ | × |
| (Q, +, ·) | ✓ | ✓ | 无 | ✓ | ✓ |
| (R, +, ·) | ✓ | ✓ | 无 | ✓ | ✓ |
| (Z~6~, +~6~, ·~6~) | ✓ | ✓ | 有 (2·3=0) | × | × |
| (Z~5~, +~5~, ·~5~) | ✓ | ✓ | 无 | ✓ | ✓ |
| (M~n~(R), +, ·) | × | ✓ | 有 | × | × |


### Z~6~ 的乘法表（展示零因子）


| ·~6~ | 0 | 1 | 2 | 3 | 4 | 5 |
| --- | --- | --- | --- | --- | --- | --- |
| **0** | 0 | 0 | 0 | 0 | 0 | 0 |
| **1** | 0 | 1 | 2 | 3 | 4 | 5 |
| **2** | 0 | 2 | 4 | 0 | 2 | 4 |
| **3** | 0 | 3 | 0 | 3 | 0 | 3 |
| **4** | 0 | 4 | 2 | 0 | 4 | 2 |

红色高亮：2 · 3 = 0（mod 6），说明 2 和 3 都是零因子。因此 Z~6~ 不是整环。

==================== 3. 域 ====================

## 3. 域（Field）的定义


### 域的公理


(F, +, ·) 是一个**域**，如果满足：


- (F, +) 是阿贝尔群（加法群，单位元记为 0）
- (F - {0}, ·) 是阿贝尔群（乘法群，单位元记为 1）
- 乘法对加法满足分配律

域 = 交换整环 + 每个非零元素都有乘法逆元

**直观理解：**域是一个可以进行加减乘除（除数非零）运算的系统。


### 常见域


| 域 | 说明 | 特征 |
| --- | --- | --- |
| (Q, +, ·) | 有理数域 | 0（特征零） |
| (R, +, ·) | 实数域 | 0 |
| (C, +, ·) | 复数域 | 0 |
| (Z~p~, +~p~, ·~p~) | p为素数时是域 | p |
| GF(2^n^) | 有限域/Galois域 | 2 |


**重要结论：**Z~n~ 是域 ↔ n 是素数。这就是为什么 Z~5~ 是域但 Z~6~ 不是。


## 3.1 Z~5~ 的乘法表（域，无零因子）


| ·~5~ | 0 | 1 | 2 | 3 | 4 |
| --- | --- | --- | --- | --- | --- |
| **0** | 0 | 0 | 0 | 0 | 0 |
| **1** | 0 | 1 | 2 | 3 | 4 |
| **2** | 0 | 2 | 4 | 1 | 3 |
| **3** | 0 | 3 | 1 | 4 | 2 |
| **4** | 0 | 4 | 3 | 2 | 1 |


非零元素 {1,2,3,4} 对乘法构成阿贝尔群。每个非零元素都有逆元：1⁻¹=1, 2⁻¹=3, 3⁻¹=2, 4⁻¹=4。

==================== 4. 层次关系 ====================

## 4. 代数结构层次关系


> **Warning:** 群 ⊂ 环 ⊂ 整环 ⊂ 域
>
>
> （从群到域，要求越来越多）
>
> | 结构 | 加法群 | 乘法 | 分配律 | 交换乘法 | 无零因子 | 乘法逆元 |

> | 环 | ✓阿贝尔 | 半群 | ✓ | 可选 | 可选 | - |

> | 域 | ✓阿贝尔 | ✓阿贝尔 | ✓ | ✓ | ✓ | ✓ |

==================== 5. 理想 ====================

## 5. 理想（Ideal）简介


### 定义


环 R 的非空子集 I 是 R 的**理想**，如果：


- (I, +) 是 (R, +) 的子群
- ∀r∈R, ∀a∈I: r·a ∈ I 且 a·r ∈ I（"吸收性"）


**主理想：**由一个元素 a 生成的理想 (a) = {ra | r ∈ R}。


**示例：**nZ = {..., -2n, -n, 0, n, 2n, ...} 是 Z 的主理想。


**重要定理：**环 R 的理想 I 对应一个商环 R/I。整环的极大理想对应域。

    总结：
    1. 环 = 加法阿贝尔群 + 乘法半群 + 分配律
    2. 整环 = 交换含幺环 + 无零因子
    3. 域 = 交换整环 + 每个非零元素有乘法逆元（可做四则运算）
    4. Zn 是域 ⟺ n 是素数
    5. 零因子：a≠0, b≠0 但 ab=0，存在零因子则不是整环
    6. 常见域：Q, R, C（特征0），Zp（p为素数，特征p），GF(2^n)

## Python实现

### 环与域性质验证

```python
class FiniteRing:
    """有限环的性质验证"""
    def __init__(self, elements, add_op, mul_op, name="R"):
        self.elements = elements
        self.add = add_op
        self.mul = mul_op
        self.name = name
        self.n = len(elements)

    def is_abelian_additive(self):
        """加法构成阿贝尔群"""
        closed = all(self.add(a, b) in self.elements
                     for a in self.elements for b in self.elements)
        associative = all(
            self.add(self.add(a, b), c) == self.add(a, self.add(b, c))
            for a in self.elements for b in self.elements for c in self.elements)
        identity = any(
            all(self.add(e, a) == a for a in self.elements) for e in self.elements)
        commutative = all(
            self.add(a, b) == self.add(b, a)
            for a in self.elements for b in self.elements)
        return closed and associative and identity and commutative

    def is_commutative(self):
        """乘法交换律"""
        return all(self.mul(a, b) == self.mul(b, a)
                   for a in self.elements for b in self.elements)

    def has_multiplicative_identity(self):
        """含幺"""
        for e in self.elements:
            if all(self.mul(e, a) == a and self.mul(a, e) == a for a in self.elements):
                return e
        return None

    def zero_divisors(self):
        """找零因子"""
        zeros = []
        for a in self.elements:
            if a == 0:
                continue
            for b in self.elements:
                if b == 0:
                    continue
                if self.mul(a, b) == 0:
                    zeros.append((a, b))
        return zeros

    def is_field(self):
        """是否为域"""
        unit = self.has_multiplicative_identity()
        if unit is None:
            return False
        # 每个非零元素都有乘法逆元
        for a in self.elements:
            if a == 0:
                continue
            has_inv = any(self.mul(a, b) == unit for b in self.elements)
            if not has_inv:
                return False
        return True

# Z6 验证
print("=== Z6 环验证 ===")
Z6_add = lambda a, b: (a + b) % 6
Z6_mul = lambda a, b: (a * b) % 6
Z6 = FiniteRing(range(6), Z6_add, Z6_mul, "Z6")

print(f"加法阿贝尔群: {Z6.is_abelian_additive()}")
print(f"乘法交换: {Z6.is_commutative()}")
print(f"含幺: {Z6.has_multiplicative_identity()}")
print(f"零因子: {Z6.zero_divisors()}")
print(f"是整环: {len(Z6.zero_divisors()) == 0}")
print(f"是域: {Z6.is_field()}")

# Z5 验证 (素数, 应为域)
print("\n=== Z5 环验证 (素数模) ===")
Z5_add = lambda a, b: (a + b) % 5
Z5_mul = lambda a, b: (a * b) % 5
Z5 = FiniteRing(range(5), Z5_add, Z5_mul, "Z5")
print(f"含幺: {Z5.has_multiplicative_identity()}")
print(f"零因子: {Z5.zero_divisors()}")
print(f"是域: {Z5.is_field()}")

# 打印 Z5 乘法表
print("\nZ5 乘法表:")
for a in range(5):
    for b in range(5):
        print(f"{Z5_mul(a, b):>3}", end="")
    print()

# 计算逆元
print("\nZ5 非零元素的乘法逆元:")
for a in range(1, 5):
    for b in range(5):
        if Z5_mul(a, b) == 1:
            print(f"  {a}⁻¹ = {b}")
            break

# 代数结构层次验证
print("\n=== 代数结构层次 ===")
structures = [
    ("Z6", Z6, "交换环, 含幺, 有零因子, 非整环, 非域"),
    ("Z5", Z5, "交换环, 含幺, 无零因子, 整环, 域"),
]
for name, ring, desc in structures:
    print(f"{name}: {desc}")
```
