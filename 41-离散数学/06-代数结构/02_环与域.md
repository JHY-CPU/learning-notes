# 02 - 环与域


离散数学 | 代数结构

==================== 1. 环的定义 ====================

## 1. 环（Ring）的定义


### 环的公理


代数系统 (R, +, ·) 是一个**环**，如果满足：


| 类别 | 公理 | 说明 |
| --- | --- | --- |
| **(R, +) 是阿贝尔群** | 加法封闭性 | ∀a,b: a+b∈R |
| 加法结合律 | (a+b)+c = a+(b+c) |
| 加法单位元 | ∃0: a+0 = 0+a = a |
| 加法逆元 | ∀a, ∃(-a): a+(-a) = 0 |
| **(R, ·) 是半群** | 乘法封闭性 | ∀a,b: a·b∈R |
| 乘法结合律 | (a·b)·c = a·(b·c) |
| 加法交换律 | a+b = b+a |
| **分配律** | a·(b+c) = a·b + a·c  ;  (b+c)·a = b·a + c·a |


> **Warning:** ### 环的附加性质
>
>
> | 名称 | 条件 | 示例 |
> | --- | --- | --- |
> | 交换环 | 乘法满足交换律 | (Z, +, ·) |
> | 含幺环 | 乘法有单位元 1 | (Z, +, ·) 含幺元 1 |
> | 整环 | 交换 + 含幺 + 无零因子 | (Z, +, ·) |
> | 除环/体 | 非零元素对乘法构成群 | (Q, +, ·) |
> | 域 | 交换除环 | (Q, R, C) |
>
>
> **零因子：**在环 R 中，若 a≠0, b≠0 但 a·b=0，则称 a, b 为零因子。

## 2. 重要的环


### 示例对比


| 代数系统 | 交换环 | 含幺 | 零因子 | 整环 | 域 |
| --- | --- | --- | --- | --- | --- |
| (Z, +, ·) | ✓ | ✓ | 无 | ✓ | × |
| (Q, +, ·) | ✓ | ✓ | 无 | ✓ | ✓ |
| (R, +, ·) | ✓ | ✓ | 无 | ✓ | ✓ |

| (Z~5~, +~5~, ·~5~) | ✓ | ✓ | 无 | ✓ | ✓ |

==================== 3. 域 ====================

## 3. 域（Field）的定义


### 域的公理


(F, +, ·) 是一个**域**，如果满足：


- (F, +) 是阿贝尔群（加法群，单位元记为 0）
- (F - {0}, ·) 是阿贝尔群（乘法群，单位元记为 1）
- 乘法对加法满足分配律

域 = 交换整环 + 每个非零元素都有乘法逆元

**直观理解：**域是一个可以进行加减乘除（除数非零）运算的系统。


### 常见域


| 域 | 说明 | 特征 |
| --- | --- | --- |
| (Q, +, ·) | 有理数域 | 0（特征零） |
| (R, +, ·) | 实数域 | 0 |
| (C, +, ·) | 复数域 | 0 |
| (Z~p~, +~p~, ·~p~) | p为素数时是域 | p |
| GF(2^n^) | 有限域/Galois域 | 2 |


**重要结论：**Z~n~ 是域 ↔ n 是素数。这就是为什么 Z~5~ 是域但 Z~6~ 不是。


## 3.1 Z~5~ 的乘法表（域，无零因子）


| ·~5~ | 0 | 1 | 2 | 3 | 4 |
| --- | --- | --- | --- | --- | --- |
| **0** | 0 | 0 | 0 | 0 | 0 |
| **1** | 0 | 1 | 2 | 3 | 4 |
| **2** | 0 | 2 | 4 | 1 | 3 |
| **3** | 0 | 3 | 1 | 4 | 2 |
| **4** | 0 | 4 | 3 | 2 | 1 |


非零元素 {1,2,3,4} 对乘法构成阿贝尔群。每个非零元素都有逆元：1⁻¹=1, 2⁻¹=3, 3⁻¹=2, 4⁻¹=4。

==================== 4. 层次关系 ====================

## 4. 代数结构层次关系


> **Warning:** 群 ⊂ 环 ⊂ 整环 ⊂ 域
>
>
> （从群到域，要求越来越多）
>
> | 结构 | 加法群 | 乘法 | 分配律 | 交换乘法 | 无零因子 | 乘法逆元 |
> | --- | --- | --- | --- | --- | --- | --- |
> | 群 | ✓ | - | - | 可选 | - | - |
> | 环 | ✓阿贝尔 | 半群 | ✓ | 可选 | 可选 | - |

> | 域 | ✓阿贝尔 | ✓阿贝尔 | ✓ | ✓ | ✓ | ✓ |

==================== 5. 理想 ====================

## 5. 理想（Ideal）简介


### 定义


环 R 的非空子集 I 是 R 的**理想**，如果：


- (I, +) 是 (R, +) 的子群
- ∀r∈R, ∀a∈I: r·a ∈ I 且 a·r ∈ I（"吸收性"）


**主理想：**由一个元素 a 生成的理想 (a) = {ra | r ∈ R}。


**示例：**nZ = {..., -2n, -n, 0, n, 2n, ...} 是 Z 的主理想。


**重要定理：**环 R 的理想 I 对应一个商环 R/I。整环的极大理想对应域。
