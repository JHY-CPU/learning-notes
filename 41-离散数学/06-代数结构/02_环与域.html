1. 环（Ring）的定义

环的公理

代数系统 (R, +, ·) 是一个环，如果满足：

类别公理说明
(R, +) 是阿贝尔群加法封闭性∀a,b: a+b∈R
加法结合律(a+b)+c = a+(b+c)
加法单位元∃0: a+0 = 0+a = a
加法逆元∀a, ∃(-a): a+(-a) = 0
(R, ·) 是半群乘法封闭性∀a,b: a·b∈R
乘法结合律(a·b)·c = a·(b·c)
加法交换律a+b = b+a
分配律a·(b+c) = a·b + a·c  ;  (b+c)·a = b·a + c·a

环的附加性质

名称条件示例
交换环乘法满足交换律(Z, +, ·)
含幺环乘法有单位元 1(Z, +, ·) 含幺元 1
整环交换 + 含幺 + 无零因子(Z, +, ·)
除环/体非零元素对乘法构成群(Q, +, ·)
域交换除环(Q, R, C)

零因子：在环 R 中，若 a≠0, b≠0 但 a·b=0，则称 a, b 为零因子。

3. 域（Field）的定义

域的公理

(F, +, ·) 是一个域，如果满足：

  • (F, +) 是阿贝尔群（加法群，单位元记为 0）
  • (F - {0}, ·) 是阿贝尔群（乘法群，单位元记为 1）
  • 乘法对加法满足分配律
域 = 交换整环 + 每个非零元素都有乘法逆元

直观理解：域是一个可以进行加减乘除（除数非零）运算的系统。

常见域

域说明特征
(Q, +, ·)有理数域0（特征零）
(R, +, ·)实数域0
(C, +, ·)复数域0
(Zp, +p, ·p)p为素数时是域p
GF(2n)有限域/Galois域2

重要结论：Zn 是域 ↔ n 是素数。这就是为什么 Z5 是域但 Z6 不是。

5. 理想（Ideal）简介

定义

环 R 的非空子集 I 是 R 的理想，如果：

  • (I, +) 是 (R, +) 的子群
  • ∀r∈R, ∀a∈I: r·a ∈ I 且 a·r ∈ I（"吸收性"）

主理想：由一个元素 a 生成的理想 (a) = {ra | r ∈ R}。

示例：nZ = {..., -2n, -n, 0, n, 2n, ...} 是 Z 的主理想。

重要定理：环 R 的理想 I 对应一个商环 R/I。整环的极大理想对应域。
