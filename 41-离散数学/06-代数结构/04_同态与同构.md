## 同态与同构


离散数学 · 代数结构 · 第4节

==================== 1. 同态 ====================

## 1. 同态映射 (Homomorphism)


### 1.1 定义


> **Note:** 设 (G, *) 和 (H, ∘) 是两个代数系统。映射 φ: G → H 称为
> **同态映射**
> ，若对任意 a, b ∈ G：
>
>
>
> $$
> φ(a * b) = φ(a) ∘ φ(b)
> $$
>
> 即：先运算再映射 = 先映射再运算（
> **保持运算结构**
> ）。


### 1.2 同态的类型


| 类型 | 条件 | 性质 |
| --- | --- | --- |
| 同态 (Homomorphism) | φ(a*b) = φ(a)∘φ(b) | 保持运算 |
| 单同态 (Monomorphism) | 同态 + 单射 | 保持运算且一一对应到像 |
| 满同态 (Epimorphism) | 同态 + 满射 | 保持运算且映满 H |
| 同构 (Isomorphism) | 同态 + 双射 | 结构完全相同 |
| 自同态 (Endomorphism) | G=H 的同态 | 映射到自身 |
| 自同构 (Automorphism) | G=H 的同构 | 结构的对称性 |

> **Example:** **例：**
> φ: (Z, +) → (Zₙ, +ₙ)，φ(x) = x mod n。
>
>
> 验证：φ(a+b) = (a+b) mod n = (a mod n + b mod n) mod n = φ(a) +ₙ φ(b) ✓
>
>
> 这是满同态，核 ker(φ) = nZ = {..., -2n, -n, 0, n, 2n, ...}。

==================== 2. 核与像 ====================

## 2. 同态的核与像


### 2.1 定义


> **Note:** 设 φ: G → H 是群同态。
>
>
> **像 (Image)：**
> Im(φ) = {φ(g) : g ∈ G} ⊆ H
>
>
> **核 (Kernel)：**
> ker(φ) = {g ∈ G : φ(g) = e_H} ⊆ G
>
>
> 其中 e_H 是 H 的单位元。


### 2.2 基本性质


> **Theorem:** 设 φ: G → H 是群同态，则：
>
> - φ(e_G) = e_H（单位元映射到单位元）
> - φ(g⁻¹) = (φ(g))⁻¹（逆元映射到逆元）
> - ker(φ) 是 G 的
>    **正规子群**
> - Im(φ) 是 H 的
>    **子群**
> - φ 是单射 ⟺ ker(φ) = {e_G}
> - |G| = |ker(φ)| · |Im(φ)|（当 G 有限时）

==================== 3. 商群与自然同态 ====================

## 3. 商结构与自然同态


### 3.1 商群


> **Note:** 设 N 是群 G 的正规子群。
> **商群**
> G/N = {gN : g ∈ G}，运算定义为：
>
>
>
> $$
> (aN) · (bN) = (ab)N
> $$
>
> 单位元是 N，(gN)⁻¹ = g⁻¹N。


### 3.2 自然同态


> **Theorem:** **自然同态 (Canonical/Natural Homomorphism)：**
>
>
> π: G → G/N，π(g) = gN
>
>
> 这是满同态，ker(π) = N。


> **Example:** (Z, +) 中，nZ 是正规子群。
>
>
> Z/nZ = {0+nZ, 1+nZ, ..., (n-1)+nZ} ≅ Zₙ
>
>
> 自然同态 π: Z → Zₙ，π(x) = x mod n。

==================== 4. 同构定理 ====================

## 4. 同构定理 (Isomorphism Theorems)


### 4.1 第一同构定理


> **Theorem:** **第一同构定理 (商同构定理)：**
>
>
> 设 φ: G → H 是群同态，则：
>
>
>
> $$
> G / ker(φ) ≅ Im(φ)
> $$
>
> 即：G 模掉核同构于像。
>
>
>
>
> 更精确地，存在同构 ψ: G/ker(φ) → Im(φ)，ψ(g·ker(φ)) = φ(g)。


> **Example:** φ: Z → Zₙ, φ(x) = x mod n。
>
>
> ker(φ) = nZ，Im(φ) = Zₙ。
>
>
> 由第一同构定理：Z/nZ ≅ Zₙ ✓


### 4.2 第二同构定理 (钻石定理)


> **Theorem:** **第二同构定理：**
>
>
> 设 H 是 G 的子群，N 是 G 的正规子群，则：
>
>
>
> $$
> HN / N ≅ H / (H ∩ N)
> $$
>
> 其中 HN = {hn : h ∈ H, n ∈ N}。


### 4.3 第三同构定理


> **Theorem:** **第三同构定理：**
>
>
> 设 N ⊆ K ⊆ G，N 和 K 都是 G 的正规子群，则：
>
>
>
> $$
> (G/N) / (K/N) ≅ G/K
> $$

==================== 5. 同构 ====================

## 5. 同构 (Isomorphism)


### 5.1 定义与意义


> **Note:** 两个代数系统 (G, *) 和 (H, ∘)
> **同构**
> ，记 G ≅ H，若存在双射同态 φ: G → H。
>
>
>
>
> 同构意味着两个系统在结构上
> **完全相同**
> ——只是元素的"名字"不同。
>
>
> 同构的系统有相同的乘法表（只是重命名行列）。


### 5.2 同构不变量


若 G ≅ H，则它们共享以下性质（同构不变量）：


| 不变量 | 说明 |
| --- | --- |
| 阶 \|G\| | 元素个数相同 |
| 交换性 | G 交换 ⟺ H 交换 |
| 循环性 | G 循环 ⟺ H 循环 |
| 子群格结构 | 子群的偏序结构相同 |
| 元素阶的分布 | 各阶元素个数相同 |
| 中心 \|Z(G)\| | 中心大小相同 |


### 5.3 Cayley 定理


> **Theorem:** **Cayley 定理：**
> 每个群 G 都同构于某个置换群。
>
>
> 即：G ≅ Sₙ 的某个子群（n = |G|）。
>
>
> 这说明置换群是所有群的"原型"。

==================== 6. 自同构群 ====================

## 6. 自同构群


> **Note:** 群 G 的所有自同构的集合 Aut(G) 在映射复合下构成群，称为
> **自同构群**
> 。
>
>
> Aut(G) 衡量了 G 的"对称性"。


| 群 G | Aut(G) | 说明 |
| --- | --- | --- |
| Zₙ | U(n)（模n乘法群） | φ(k): x→kx, k与n互素 |
| Z | {id, neg} ≅ Z₂ | 只有恒等和取反 |
| S₃ | S₃ | 内自同构群 Inn(S₃) = S₃ |
| V₄ (Klein四元群) | S₃ | 3个非单位元可任意置换 |


### 6.1 内自同构


> **Note:** 对 g ∈ G，映射 φ_g: x → gxg⁻¹ 称为
> **内自同构**
> 。
>
>
> 所有内自同构构成 Inn(G) ◁ Aut(G)。
>
>
> **第一同构定理应用：**
> G/Z(G) ≅ Inn(G)。

==================== 总结 ====================

> **Summary:** ## 本节总结
>
>
> - **同态：**
>    φ(a*b) = φ(a)∘φ(b)，保持运算结构的映射
> - **核与像：**
>    ker(φ) 是正规子群，Im(φ) 是子群；单射 ⟺ ker={e}
> - **第一同构定理：**
>    G/ker(φ) ≅ Im(φ) — 最核心的同构定理
> - **第二同构定理：**
>    HN/N ≅ H/(H∩N)
> - **第三同构定理：**
>    (G/N)/(K/N) ≅ G/K
> - **同构：**
>    G ≅ H 意味着结构完全相同；Cayley定理：每个群同构于置换群
> - **自同构群：**
>    Aut(G) 衡量对称性；内自同构 Inn(G) ≅ G/Z(G)

## Python实现

### 同态映射与同构定理验证

```python
class GroupHomomorphism:
    """群同态验证"""
    def __init__(self, G_elements, G_op, H_elements, H_op, phi, name="φ"):
        self.G = G_elements
        self.G_op = G_op
        self.H = H_elements
        self.H_op = H_op
        self.phi = phi
        self.name = name

    def is_homomorphism(self):
        """验证同态性质: φ(a*b) = φ(a)∘φ(b)"""
        for a in self.G:
            for b in self.G:
                lhs = self.phi(self.G_op(a, b))
                rhs = self.H_op(self.phi(a), self.phi(b))
                if lhs != rhs:
                    return False, f"反例: φ({a}*{b}) = {lhs} ≠ φ({a})∘φ({b}) = {rhs}"
        return True, "是同态映射"

    def kernel(self):
        """计算核 ker(φ)"""
        # 找 H 的单位元
        identity_H = None
        for e in self.H:
            if all(self.H_op(e, h) == h for h in self.H):
                identity_H = e
                break
        return [g for g in self.G if self.phi(g) == identity_H]

    def image(self):
        """计算像 Im(φ)"""
        return set(self.phi(g) for g in self.G)

    def is_injective(self):
        """单射 ⟺ ker = {e_G}"""
        return len(self.kernel()) == 1

    def is_surjective(self):
        """满射 ⟺ Im(φ) = H"""
        return self.image() == set(self.H)

# 示例: φ: Z → Zₙ, φ(x) = x mod n
print("=== 同态: φ: Z_12 → Z_4, φ(x) = x mod 4 ===")
G = list(range(12))
G_op = lambda a, b: (a + b) % 12
H = list(range(4))
H_op = lambda a, b: (a + b) % 4
phi = lambda x: x % 4

hom = GroupHomomorphism(G, G_op, H, H_op, phi)
is_hom, msg = hom.is_homomorphism()
print(f"  同态性质: {msg}")
print(f"  ker(φ) = {hom.kernel()}")
print(f"  Im(φ) = {hom.image()}")
print(f"  单射: {hom.is_injective()}")
print(f"  满射: {hom.is_surjective()}")

# 第一同构定理: G/ker(φ) ≅ Im(φ)
print(f"\n=== 第一同构定理验证 ===")
ker = hom.kernel()
im = hom.image()
print(f"  |G| = {len(G)}, |ker(φ)| = {len(ker)}, |Im(φ)| = {len(im)}")
print(f"  |G| = |ker| × |Im|: {len(G) == len(ker) * len(im)}")
print(f"  |G/ker| = {len(G)//len(ker)}, |Im| = {len(im)}")
print(f"  G/ker(φ) ≅ Im(φ): {len(G)//len(ker) == len(im)}")

# 自同构群: Aut(Zₙ) ≅ U(n)
print("\n=== Aut(Z_5) ≅ U(5) ===")
# Z5 的自同构: φ_k(x) = kx mod 5, 其中 gcd(k,5)=1
for k in range(1, 5):
    if gcd(k, 5) == 1:
        mapping = [(x, (k * x) % 5) for x in range(5)]
        print(f"  φ_{k}(x) = {k}x mod 5: {dict(mapping)}")
```

### Cayley 定理验证

```python
from itertools import permutations

print("=== Cayley 定理: 每个群同构于置换群的子群 ===")

# Z3 = {0, 1, 2}, +3
Z3 = list(range(3))
Z3_op = lambda a, b: (a + b) % 3

# 通过左乘构造置换
S3 = list(permutations(range(3)))
compose = lambda p, q: tuple(p[q[i]] for i in range(3))

print("Z3 → S3 的嵌入 (左乘置换):")
embeddings = {}
for a in Z3:
    perm = tuple((a + x) % 3 for x in range(3))
    embeddings[a] = perm
    print(f"  a={a}: 左乘置换 = {perm}")

# 验证嵌入是同态
print("\n验证同态性质:")
for a in Z3:
    for b in Z3:
        ab = Z3_op(a, b)
        lhs = embeddings[ab]
        rhs = compose(embeddings[a], embeddings[b])
        assert lhs == rhs, f"同态验证失败: {a}+{b}"
print("  所有验证通过 ✓")
print(f"  Z3 嵌入为 S3 的子群: {{{', '.join(str(e) for e in embeddings.values())}}}")
```
