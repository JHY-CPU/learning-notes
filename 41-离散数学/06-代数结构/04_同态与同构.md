# 同态与同构


离散数学 · 代数结构 · 第4节

==================== 1. 同态 ====================

## 1. 同态映射 (Homomorphism)


### 1.1 定义


> **Note:** 设 (G, *) 和 (H, ∘) 是两个代数系统。映射 φ: G → H 称为
> **同态映射**
> ，若对任意 a, b ∈ G：
>
>
>
> $$
> φ(a * b) = φ(a) ∘ φ(b)
> $$
>
> 即：先运算再映射 = 先映射再运算（
> **保持运算结构**
> ）。


### 1.2 同态的类型


| 类型 | 条件 | 性质 |
| --- | --- | --- |
| 同态 (Homomorphism) | φ(a*b) = φ(a)∘φ(b) | 保持运算 |
| 单同态 (Monomorphism) | 同态 + 单射 | 保持运算且一一对应到像 |
| 满同态 (Epimorphism) | 同态 + 满射 | 保持运算且映满 H |
| 同构 (Isomorphism) | 同态 + 双射 | 结构完全相同 |
| 自同态 (Endomorphism) | G=H 的同态 | 映射到自身 |
| 自同构 (Automorphism) | G=H 的同构 | 结构的对称性 |

> **Example:** **例：**
> φ: (Z, +) → (Zₙ, +ₙ)，φ(x) = x mod n。
>
>
> 验证：φ(a+b) = (a+b) mod n = (a mod n + b mod n) mod n = φ(a) +ₙ φ(b) ✓
>
>
> 这是满同态，核 ker(φ) = nZ = {..., -2n, -n, 0, n, 2n, ...}。

==================== 2. 核与像 ====================

## 2. 同态的核与像


### 2.1 定义


> **Note:** 设 φ: G → H 是群同态。
>
>
> **像 (Image)：**
> Im(φ) = {φ(g) : g ∈ G} ⊆ H
>
>
> **核 (Kernel)：**
> ker(φ) = {g ∈ G : φ(g) = e_H} ⊆ G
>
>
> 其中 e_H 是 H 的单位元。


### 2.2 基本性质


> **Theorem:** 设 φ: G → H 是群同态，则：
>
> - φ(e_G) = e_H（单位元映射到单位元）
> - φ(g⁻¹) = (φ(g))⁻¹（逆元映射到逆元）
> - ker(φ) 是 G 的
>    **正规子群**
> - Im(φ) 是 H 的
>    **子群**
> - φ 是单射 ⟺ ker(φ) = {e_G}
> - |G| = |ker(φ)| · |Im(φ)|（当 G 有限时）

==================== 3. 商群与自然同态 ====================

## 3. 商结构与自然同态


### 3.1 商群


> **Note:** 设 N 是群 G 的正规子群。
> **商群**
> G/N = {gN : g ∈ G}，运算定义为：
>
>
>
> $$
> (aN) · (bN) = (ab)N
> $$
>
> 单位元是 N，(gN)⁻¹ = g⁻¹N。


### 3.2 自然同态


> **Theorem:** **自然同态 (Canonical/Natural Homomorphism)：**
>
>
> π: G → G/N，π(g) = gN
>
>
> 这是满同态，ker(π) = N。


> **Example:** (Z, +) 中，nZ 是正规子群。
>
>
> Z/nZ = {0+nZ, 1+nZ, ..., (n-1)+nZ} ≅ Zₙ
>
>
> 自然同态 π: Z → Zₙ，π(x) = x mod n。

==================== 4. 同构定理 ====================

## 4. 同构定理 (Isomorphism Theorems)


### 4.1 第一同构定理


> **Theorem:** **第一同构定理 (商同构定理)：**
>
>
> 设 φ: G → H 是群同态，则：
>
>
>
> $$
> G / ker(φ) ≅ Im(φ)
> $$
>
> 即：G 模掉核同构于像。
>
>
>
>
> 更精确地，存在同构 ψ: G/ker(φ) → Im(φ)，ψ(g·ker(φ)) = φ(g)。


> **Example:** φ: Z → Zₙ, φ(x) = x mod n。
>
>
> ker(φ) = nZ，Im(φ) = Zₙ。
>
>
> 由第一同构定理：Z/nZ ≅ Zₙ ✓


### 4.2 第二同构定理 (钻石定理)


> **Theorem:** **第二同构定理：**
>
>
> 设 H 是 G 的子群，N 是 G 的正规子群，则：
>
>
>
> $$
> HN / N ≅ H / (H ∩ N)
> $$
>
> 其中 HN = {hn : h ∈ H, n ∈ N}。


### 4.3 第三同构定理


> **Theorem:** **第三同构定理：**
>
>
> 设 N ⊆ K ⊆ G，N 和 K 都是 G 的正规子群，则：
>
>
>
> $$
> (G/N) / (K/N) ≅ G/K
> $$

==================== 5. 同构 ====================

## 5. 同构 (Isomorphism)


### 5.1 定义与意义


> **Note:** 两个代数系统 (G, *) 和 (H, ∘)
> **同构**
> ，记 G ≅ H，若存在双射同态 φ: G → H。
>
>
>
>
> 同构意味着两个系统在结构上
> **完全相同**
> ——只是元素的"名字"不同。
>
>
> 同构的系统有相同的乘法表（只是重命名行列）。


### 5.2 同构不变量


若 G ≅ H，则它们共享以下性质（同构不变量）：


| 不变量 | 说明 |
| --- | --- |
| 阶 \|G\| | 元素个数相同 |
| 交换性 | G 交换 ⟺ H 交换 |
| 循环性 | G 循环 ⟺ H 循环 |
| 子群格结构 | 子群的偏序结构相同 |
| 元素阶的分布 | 各阶元素个数相同 |
| 中心 \|Z(G)\| | 中心大小相同 |


### 5.3 Cayley 定理


> **Theorem:** **Cayley 定理：**
> 每个群 G 都同构于某个置换群。
>
>
> 即：G ≅ Sₙ 的某个子群（n = |G|）。
>
>
> 这说明置换群是所有群的"原型"。

==================== 6. 自同构群 ====================

## 6. 自同构群


> **Note:** 群 G 的所有自同构的集合 Aut(G) 在映射复合下构成群，称为
> **自同构群**
> 。
>
>
> Aut(G) 衡量了 G 的"对称性"。


| 群 G | Aut(G) | 说明 |
| --- | --- | --- |
| Zₙ | U(n)（模n乘法群） | φ(k): x→kx, k与n互素 |
| Z | {id, neg} ≅ Z₂ | 只有恒等和取反 |
| S₃ | S₃ | 内自同构群 Inn(S₃) = S₃ |
| V₄ (Klein四元群) | S₃ | 3个非单位元可任意置换 |


### 6.1 内自同构


> **Note:** 对 g ∈ G，映射 φ_g: x → gxg⁻¹ 称为
> **内自同构**
> 。
>
>
> 所有内自同构构成 Inn(G) ◁ Aut(G)。
>
>
> **第一同构定理应用：**
> G/Z(G) ≅ Inn(G)。

==================== 总结 ====================

> **Summary:** ## 本节总结
>
>
> - **同态：**
>    φ(a*b) = φ(a)∘φ(b)，保持运算结构的映射
> - **核与像：**
>    ker(φ) 是正规子群，Im(φ) 是子群；单射 ⟺ ker={e}
> - **第一同构定理：**
>    G/ker(φ) ≅ Im(φ) — 最核心的同构定理
> - **第二同构定理：**
>    HN/N ≅ H/(H∩N)
> - **第三同构定理：**
>    (G/N)/(K/N) ≅ G/K
> - **同构：**
>    G ≅ H 意味着结构完全相同；Cayley定理：每个群同构于置换群
> - **自同构群：**
>    Aut(G) 衡量对称性；内自同构 Inn(G) ≅ G/Z(G)
