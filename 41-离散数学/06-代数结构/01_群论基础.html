离散数学 | 代数结构

1. 代数系统概述

什么是代数结构？

代数结构是一个非空集合 S 连同定义在 S 上的一个或多个运算组成的系统。

例如：(Z, +) 是整数集连同加法运算组成的代数系统。

代数结构的研究关注运算的性质（结合律、交换律、单位元、逆元等），而非具体元素。

2. 二元运算的性质

二元运算 *: S × S → S，将 S 中的两个元素映射到 S 中的一个元素。

性质定义示例
封闭性∀a,b∈S: a*b∈S整数加法封闭
结合律(a*b)*c = a*(b*c)加法、乘法满足
交换律a*b = b*a加法满足，减法不满足
单位元（幺元）∃e: a*e = e*a = a加法的0，乘法的1
逆元∀a, ∃a⁻¹: a*a⁻¹ = a⁻¹*a = e加法中-a是a的逆元
消去律a*b=a*c ⇒ b=c（a可消去）整数乘法满足
分配律a*(b+c) = a*b + a*c乘法对加法分配
幂等律a*a = a集合的并和交满足

3. 群（Group）的定义

群的公理

代数系统 (G, *) 是一个群，如果满足以下四条公理：

公理名称表述
G1封闭性∀a,b∈G: a*b∈G
G2结合律∀a,b,c∈G: (a*b)*c = a*(b*c)
G3单位元∃e∈G: ∀a, a*e = e*a = a
G4逆元∀a∈G, ∃a⁻¹∈G: a*a⁻¹ = a⁻¹*a = e

如果还满足交换律（G5: a*b = b*a），则称为阿贝尔群（Abelian Group）或交换群。

群的层次结构

代数系统 ⊃ 半群（封闭+结合）⊃ 独异点/幺半群（+单位元）⊃ 群（+逆元）⊃ 阿贝尔群（+交换律）
结构封闭结合单位元逆元交换
原群（Magma）✓
半群（Semigroup）✓✓
幺半群（Monoid）✓✓✓
群（Group）✓✓✓✓

4. 重要群的示例

4.1 整数加法群 (Z, +)

  • 集合：Z = {..., -2, -1, 0, 1, 2, ...}
  • 运算：普通加法 +
  • 封闭性：两个整数之和仍为整数
  • 结合律：(a+b)+c = a+(b+c)
  • 单位元：0（a+0 = 0+a = a）
  • 逆元：a 的逆元是 -a（a + (-a) = 0）
  • 交换律：a+b = b+a → 阿贝尔群
  • 阶：无限群（|Z| = ∞）

4.2 模 n 整数加法群 (Zn, +n)

  • 集合：Zn = {0, 1, 2, ..., n-1}
  • 运算：a +n b = (a + b) mod n
  • 单位元：0
  • 逆元：a 的逆元是 n-a（当 a ≠ 0）
  • 阶：n（有限群）
  • 是阿贝尔群

4.3 对称群 Sn（置换群）

定义

对称群 Sn 是集合 {1,2,...,n} 上所有双射（置换）组成的群，运算为函数复合。

阶：|Sn| = n!

单位元：恒等置换 id = (1)(2)...(n)

逆元：置换 σ 的逆置换 σ⁻¹

S3（3阶对称群）有 3! = 6 个元素，是最小的非阿贝尔群。

S3 的元素（用轮换记号表示）

置换轮换记号映射阶
e(1)(2)(3)1→1, 2→2, 3→31
(1 2)(1 2)(3)1→2, 2→1, 3→32
(1 3)(1 3)(2)1→3, 2→2, 3→12
(2 3)(1)(2 3)1→1, 2→3, 3→22
(1 2 3)(1 2 3)1→2, 2→3, 3→13
(1 3 2)(1 3 2)1→3, 3→2, 2→13

非交换性：(1 2) ∘ (1 3) = (1 3 2)，而 (1 3) ∘ (1 2) = (1 2 3)。不相等！

6. 子群（Subgroup）

定义

群 G 的非空子集 H，如果对 G 的运算也构成群，则称 H 是 G 的子群，记为 H ≤ G。

判定定理：H 是 G 的子群 ↔

  • ∀a,b ∈ H: a*b ∈ H（封闭性）
  • ∀a ∈ H: a⁻¹ ∈ H（逆元在H中）

示例：({0, 2, 4}, +6) 是 (Z6, +6) 的子群。
