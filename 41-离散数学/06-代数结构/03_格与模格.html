格与模格

离散数学 · 代数结构 · 第3节

1. 从偏序集到格

1.1 偏序集 (Poset) 回顾

偏序集 (P, ≤)：集合 P 配上满足自反、反对称、传递的二元关系 ≤。
上界/下界：对 S⊆P，u 是上界若 ∀s∈S: s≤u；l 是下界若 ∀s∈S: l≤s。
最小上界 (LUB/Join)：上界中的最小元，记 sup(S) 或 ∨S。
最大下界 (GLB/Meet)：下界中的最大元，记 inf(S) 或 ∧S。

2. 格 (Lattice)

2.1 偏序集定义

格的定义（偏序集方式）：偏序集 (L, ≤) 称为格，如果 L 中任意两个元素都有最小上界和最大下界。

对任意 a, b ∈ L：
a ∨ b = sup{a, b}（join，最小上界）
a ∧ b = inf{a, b}（meet，最大下界）

2.2 代数定义

格的定义（代数方式）：代数系统 (L, ∨, ∧) 称为格，若对任意 a,b,c ∈ L 满足：
  • 交换律：a∨b = b∨a，a∧b = b∧a
  • 结合律：(a∨b)∨c = a∨(b∨c)，(a∧b)∧c = a∧(b∧c)
  • 吸收律：a∨(a∧b) = a，a∧(a∨b) = a
  • 幂等律：a∨a = a，a∧a = a
注意：分配律 a∧(b∨c) = (a∧b)∨(a∧c) 不一定成立——这正是分配格的特殊之处。

2.3 常见格的例子

3. 分配格 (Distributive Lattice)

分配格定义：格 L 是分配格，若对任意 a,b,c ∈ L 满足分配律：
a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)
a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)
两个分配律等价（在格中可互相推导）。

3.1 判定定理

格 L 是分配格 ⟺ L 不含与 M₃（钻石格）或 N₅（五边形格）同构的子格。
格是否分配格原因
幂集格 (P(S), ∩, ∪)是集合运算满足分配律
整除格 (Z⁺, gcd, lcm)是gcd, lcm 满足分配律
子群格一般不是可能含 M₃ 或 N₅
M₃ (钻石格)否含 M₃ 自身
N₅ (五边形格)否含 N₅ 自身
链 (全序集)是总有一个运算退化

3.2 分配格中的消去律

在分配格中：若 a∧b = a∧c 且 a∨b = a∨c，则 b = c。

4. 模格 (Modular Lattice)

模格定义：格 L 是模格，若对任意 a, b, c ∈ L 满足：
若 a ≤ c，则 a ∨ (b ∧ c) = (a ∨ b) ∧ c
即：当 a ≤ c 时，分配律成立。
分配格与模格的关系：
分配格 ⊂ 模格 ⊂ 一般格
  • 所有分配格都是模格
  • 模格不一定分配（M₃ 是模格但非分配格）
  • 非模格一定含 N₅ 作为子格

4.1 模格的判定

格 L 是模格 ⟺ L 不含与 N₅（五边形格）同构的子格。
N₅ 中取 a ≤ c，检验模律：
a ∨ (b ∧ c) = a ∨ O = a
(a ∨ b) ∧ c = I ∧ c = c
a ≠ c，所以模律不成立，N₅ 不是模格。

4.2 模格的例子

5. 子格 (Sublattice)

子格定义：格 L 的子集 S 称为子格，若 S 在 L 的 meet 和 join 运算下封闭（即 S 自身也是格）。

注意：S 在 L 的偏序下的理想/滤子不一定是子格；子格必须对两个运算都封闭。
关键区别：子格 (sublattice) vs 子代数 vs 子集的格结构。
M₃ 是某个格的子格意味着那个格不是分配格。

6. 有界格与有补格

6.1 有界格

格 L 是有界格，若存在最大元 1（全上界）和最小元 0（全下界），使得对所有 a ∈ L：
a ∨ 0 = a，a ∧ 1 = a，a ∨ 1 = 1，a ∧ 0 = 0

6.2 有补格

在有界格 L 中，元素 b 是 a 的补，若 a ∨ b = 1 且 a ∧ b = 0。
L 是有补格，若每个元素至少有一个补。

有补分配格 = 布尔代数（详见格与布尔代数章节）。

本节总结

  • 格：任意两元素有 ∨(join) 和 ∧(meet) 的偏序集
  • 格的公理：交换律、结合律、吸收律、幂等律
  • 分配格：满足分配律；等价于不含 M₃ 和 N₅ 子格
  • 模格：若 a≤c 则 a∨(b∧c)=(a∨b)∧c；等价于不含 N₅ 子格
  • 关系链：分配格 ⊂ 模格 ⊂ 一般格
  • 常见格：幂集格、整除格、子群格、向量空间子空间格
  • 有界格：有最大元1和最小元0；有补分配格 = 布尔代数
