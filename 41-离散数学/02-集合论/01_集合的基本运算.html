1. 集合的基本概念

集合的定义

集合（Set）是具有某种特定性质的、确定的、彼此不同的事物的总体。集合中的每个事物称为集合的元素（Element）。

  • 确定性：给定一个集合，任何元素要么属于它，要么不属于它
  • 互异性：集合中的元素各不相同
  • 无序性：集合中的元素没有顺序

2. 子集与包含关系

子集（Subset）：A ⊆ B 当且仅当 A 中的每个元素都属于 B。

真子集（Proper Subset）：A ⊂ B 当且仅当 A ⊆ B 且 A ≠ B。

集合相等：A = B ↔ A ⊆ B 且 B ⊆ A

空集性质：∅ ⊆ A（空集是任何集合的子集）

3. 集合的基本运算

3.1 并集 A ∪ B（Union）

定义：A ∪ B = {x | x ∈ A 或 x ∈ B}

把两个集合的所有元素合在一起（重复元素只算一次）。

示例：A = {1, 2, 3}，B = {3, 4, 5} → A ∪ B = {1, 2, 3, 4, 5}

性质：A ∪ A = A（幂等律）；A ∪ ∅ = A（同一律）；A ∪ U = U

3.2 交集 A ∩ B（Intersection）

定义：A ∩ B = {x | x ∈ A 且 x ∈ B}

取出两个集合共有的元素。

示例：A = {1, 2, 3}，B = {3, 4, 5} → A ∩ B = {3}

不相交：A ∩ B = ∅ 时，称A和B不相交（Disjoint）

性质：A ∩ A = A；A ∩ ∅ = ∅；A ∩ U = A

3.3 差集 A - B（Difference）

定义：A - B = {x | x ∈ A 且 x ∉ B}

从A中去掉A和B共有的元素。

示例：A = {1, 2, 3, 4}，B = {3, 4, 5} → A - B = {1, 2}

注意：A - B ≠ B - A（差运算不满足交换律）

3.4 补集 A′ 或 A̅（Complement）

定义：A′ = U - A = {x | x ∈ U 且 x ∉ A}

全集中不属于A的所有元素组成的集合。

示例：U = {1,2,3,4,5}，A = {1,2} → A′ = {3,4,5}

性质：(A′)′ = A；A ∪ A′ = U；A ∩ A′ = ∅

3.5 对称差 A ⊕ B（Symmetric Difference）

定义：A ⊕ B = (A - B) ∪ (B - A) = (A ∪ B) - (A ∩ B)

属于A或属于B，但不同时属于A和B的元素。

示例：A = {1, 2, 3}，B = {3, 4, 5} → A ⊕ B = {1, 2, 4, 5}

5. 集合运算律

运算律并集 ∪交集 ∩
交换律A∪B = B∪AA∩B = B∩A
结合律(A∪B)∪C = A∪(B∪C)(A∩B)∩C = A∩(B∩C)
分配律A∪(B∩C) = (A∪B)∩(A∪C)A∩(B∪C) = (A∩B)∪(A∩C)
幂等律A∪A = AA∩A = A
同一律A∪∅ = AA∩U = A
零律A∪U = UA∩∅ = ∅
吸收律A∪(A∩B) = A   ;   A∩(A∪B) = A

德摩根律（De Morgan's Laws）

(A ∪ B)′ = A′ ∩ B′

(A ∩ B)′ = A′ ∪ B′

"并的补等于补的交，交的补等于补的并" — 与命题逻辑的德摩根律完全对应。

6. 集合元素个数公式

容斥公式（两个集合）

|A ∪ B| = |A| + |B| - |A ∩ B|

示例：某班30人选数学，25人选物理，10人同时选了两门。则选了数学或物理的有 30+25-10 = 45 人。

容斥公式（三个集合）

|A∪B∪C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|
