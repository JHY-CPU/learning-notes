==================== 1. 集合的基本概念 ====================

## 1. 集合的基本概念


### 集合的定义


**集合（Set）**是具有某种特定性质的、确定的、彼此不同的事物的总体。集合中的每个事物称为集合的**元素（Element）**。


- **确定性：**
   给定一个集合，任何元素要么属于它，要么不属于它
- **互异性：**
   集合中的元素各不相同
- **无序性：**
   集合中的元素没有顺序


### 集合的表示方法


| 方法 | 示例 | 说明 |
| --- | --- | --- |
| 列举法 | A = {1, 2, 3, 4} | 直接列出所有元素 |
| 描述法 | B = {x \| x是偶数且 x > 0} | 用谓词描述元素特征 |
| 文氏图 | 用圆表示集合 | 直观可视化工具 |


### 特殊集合


| 符号 | 名称 | 说明 |
| --- | --- | --- |
| ∅ | 空集 | 不包含任何元素的集合，唯一确定 |
| N | 自然数集 | {0, 1, 2, 3, ...} 或 {1, 2, 3, ...} |
| Z | 整数集 | {..., -2, -1, 0, 1, 2, ...} |
| Q | 有理数集 | 可表示为分数的数 |
| R | 实数集 | 所有实数 |
| U | 全集 | 当前讨论范围内的所有元素组成的集合 |

==================== 2. 子集与包含 ====================

## 2. 子集与包含关系


**子集（Subset）：**A ⊆ B 当且仅当 A 中的每个元素都属于 B。


**真子集（Proper Subset）：**A ⊂ B 当且仅当 A ⊆ B 且 A ≠ B。


**集合相等：**A = B ↔ A ⊆ B 且 B ⊆ A


**空集性质：**∅ ⊆ A（空集是任何集合的子集）

==================== 3. 集合的基本运算 ====================

## 3. 集合的基本运算


## 3.1 并集 A ∪ B（Union）


**定义：**A ∪ B = {x | x ∈ A 或 x ∈ B}


把两个集合的所有元素合在一起（重复元素只算一次）。


**示例：**A = {1, 2, 3}，B = {3, 4, 5} → A ∪ B = {1, 2, 3, 4, 5}


**性质：**A ∪ A = A（幂等律）；A ∪ ∅ = A（同一律）；A ∪ U = U


## 3.2 交集 A ∩ B（Intersection）


**定义：**A ∩ B = {x | x ∈ A 且 x ∈ B}


取出两个集合共有的元素。


**示例：**A = {1, 2, 3}，B = {3, 4, 5} → A ∩ B = {3}


**不相交：**A ∩ B = ∅ 时，称A和B不相交（Disjoint）


**性质：**A ∩ A = A；A ∩ ∅ = ∅；A ∩ U = A


## 3.3 差集 A - B（Difference）


> **Warning:** **定义：**A - B = {x | x ∈ A 且 x ∉ B}
>
>
> 从A中去掉A和B共有的元素。
>
>
> **示例：**A = {1, 2, 3, 4}，B = {3, 4, 5} → A - B = {1, 2}
>
>
> **注意：**A - B ≠ B - A（差运算不满足交换律）


## 3.4 补集 A′ 或 A̅（Complement）


> **Warning:** **定义：**A′ = U - A = {x | x ∈ U 且 x ∉ A}
>
>
> 全集中不属于A的所有元素组成的集合。
>
>
> **示例：**U = {1,2,3,4,5}，A = {1,2} → A′ = {3,4,5}
>
>
> **性质：**(A′)′ = A；A ∪ A′ = U；A ∩ A′ = ∅


## 3.5 对称差 A ⊕ B（Symmetric Difference）


**定义：**A ⊕ B = (A - B) ∪ (B - A) = (A ∪ B) - (A ∩ B)


属于A或属于B，但不同时属于A和B的元素。


**示例：**A = {1, 2, 3}，B = {3, 4, 5} → A ⊕ B = {1, 2, 4, 5}

==================== 5. 运算律 ====================

## 5. 集合运算律


| 运算律 | 并集 ∪ | 交集 ∩ |
| --- | --- | --- |
| 交换律 | A∪B = B∪A | A∩B = B∩A |
| 结合律 | (A∪B)∪C = A∪(B∪C) | (A∩B)∩C = A∩(B∩C) |
| 分配律 | A∪(B∩C) = (A∪B)∩(A∪C) | A∩(B∪C) = (A∩B)∪(A∩C) |
| 幂等律 | A∪A = A | A∩A = A |
| 同一律 | A∪∅ = A | A∩U = A |
| 零律 | A∪U = U | A∩∅ = ∅ |
| 吸收律 | A∪(A∩B) = A   ;   A∩(A∪B) = A |


### 德摩根律（De Morgan's Laws）


`(A ∪ B)′ = A′ ∩ B′`


`(A ∩ B)′ = A′ ∪ B′`


"并的补等于补的交，交的补等于补的并" — 与命题逻辑的德摩根律完全对应。

==================== 6. 交并运算规模计算 ====================

## 6. 集合元素个数公式


### 容斥公式（两个集合）

|A ∪ B| = |A| + |B| - |A ∩ B|

**示例：**某班30人选数学，25人选物理，10人同时选了两门。则选了数学或物理的有 30+25-10 = 45 人。


### 容斥公式（三个集合）

|A∪B∪C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|

==========================================
    总结：
    1. 集合是离散数学的基础数据结构，具有确定性、互异性、无序性
    2. 五种基本运算：
       - 并集 A∪B：所有元素合在一起
       - 交集 A∩B：共有的元素
       - 差集 A-B：属于A但不属于B的元素
       - 补集 A'：全集中不属于A的元素
       - 对称差 A⊕B：属于其一但不同时属于两者的元素
    3. 核心运算律：交换律、结合律、分配律、吸收律、幂等律
    4. 德摩根律：(A∪B)' = A'∩B'，(A∩B)' = A'∪B'
    5. 容斥公式：|A∪B| = |A| + |B| - |A∩B|

## Python实现

### 集合基本运算与性质验证

```python
# Python 的 set 类型原生支持集合运算
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
U = set(range(1, 8))  # 全集

print("=== 集合基本运算 ===")
print(f"A = {A}")
print(f"B = {B}")
print(f"全集 U = {U}")
print(f"A ∪ B = {A | B}")           # 并集
print(f"A ∩ B = {A & B}")           # 交集
print(f"A - B = {A - B}")           # 差集
print(f"B - A = {B - A}")
print(f"A ⊕ B = {A ^ B}")           # 对称差
print(f"A' (补集) = {U - A}")

# 验证运算律
C = {4, 5, 6, 7}
print("\n=== 运算律验证 ===")
print(f"交换律 A∪B=B∪A: {(A | B) == (B | A)}")
print(f"结合律 (A∪B)∪C=A∪(B∪C): {((A | B) | C) == (A | (B | C))}")
print(f"分配律 A∩(B∪C)=(A∩B)∪(A∩C): {(A & (B | C)) == ((A & B) | (A & C))}")

# 德摩根律验证
print(f"\n德摩根律 (A∪B)' = A'∩B':")
print(f"  (A∪B)' = {U - (A | B)}")
print(f"  A'∩B'  = {(U - A) & (U - B)}")
print(f"  相等: {(U - (A | B)) == ((U - A) & (U - B))}")

print(f"\n德摩根律 (A∩B)' = A'∪B':")
print(f"  (A∩B)' = {U - (A & B)}")
print(f"  A'∪B'  = {(U - A) | (U - B)}")
print(f"  相等: {(U - (A & B)) == ((U - A) | (U - B))}")

# 容斥公式验证
print("\n=== 容斥公式验证 ===")
A2 = {1, 2, 3, 4, 5}
B2 = {3, 4, 5, 6, 7}
union_count = len(A2 | B2)
formula_count = len(A2) + len(B2) - len(A2 & B2)
print(f"|A∪B| = {union_count}, |A|+|B|-|A∩B| = {formula_count}, 相等: {union_count == formula_count}")
```

### 利用集合解决实际问题

```python
# 问题：某班调查结果
# 选数学: 30人, 选物理: 25人, 同时选两门: 10人
math_students = set(range(1, 31))           # 学号1-30选数学
physics_students = set(range(21, 46))       # 学号21-45选物理

both = math_students & physics_students
either = math_students | physics_students
math_only = math_students - physics_students
physics_only = physics_students - math_students
neither = set(range(1, 51)) - either

print("=== 实际问题：课程选择 ===")
print(f"选数学: {len(math_students)}人")
print(f"选物理: {len(physics_students)}人")
print(f"同时选两门: {len(both)}人")
print(f"至少选一门: {len(either)}人")
print(f"只选数学: {len(math_only)}人")
print(f"只选物理: {len(physics_only)}人")
```
