树与生成树

离散数学 · 图论 · 第6节

1. 树的定义与等价条件

1.1 基本定义

树 (Tree)：连通且无圈的无向图。
森林 (Forest)：无圈的无向图（若干棵树的并）。
叶节点 (Leaf)：度数为 1 的顶点。

1.2 树的等价条件

以下条件对于 n 个顶点的无向图 G 互相等价：

#条件
1G 是树（连通且无圈）
2G 连通且有 n-1 条边
3G 无圈且有 n-1 条边
4G 中任意两顶点之间有唯一路径
5G 连通，但删去任意一条边后不连通
6G 无圈，但添加任意一条边后产生恰好一个圈
基本性质：
  • n 个顶点的树有 n-1 条边
  • 非平凡树至少有 2 个叶节点
  • 树是二分图（无奇圈）
  • 树是平面图，且是自身唯一的生成树

2. 生成树 (Spanning Tree)

2.1 定义

图 G 的生成树是包含 G 的所有顶点的一棵树（即 G 的连通生成子图且是树）。

存在性：图 G 有生成树 ⟺ G 连通。
边数：n 个顶点的生成树恰好有 n-1 条边。

2.2 构造方法

方法一：破圈法

不断删除图中的圈上的任意一条边，直到无圈。剩余图即为生成树。

方法二：广度/深度优先搜索 (BFS/DFS)

用 BFS 或 DFS 遍历图，遍历过程中经过的边构成一棵生成树。

方法三：Prim 算法 (最小生成树)

贪心策略：从任意顶点开始，每次选择连接"已选顶点集"和"未选顶点集"的最小权边。
时间复杂度：O(V²)（朴素）或 O(E log V)（堆优化）。

方法四：Kruskal 算法 (最小生成树)

按边权从小到大排序，依次选择不构成圈的边，直到选够 n-1 条边。
用并查集判断是否构成圈。时间复杂度：O(E log E)。

2.3 最小生成树的唯一性

若图中所有边权互不相同，则最小生成树唯一。
反之不一定唯一。
Prim 算法示例：
图有顶点 {A,B,C,D}，边权：AB=1, AC=3, AD=4, BC=2, BD=5, CD=6。
步骤1：选A，候选边 {AB:1, AC:3, AD:4} → 选 AB
步骤2：已选{A,B}，候选 {AC:3, AD:4, BC:2} → 选 BC
步骤3：已选{A,B,C}，候选 {AD:4, CD:6} → 选 AD
最小生成树：{AB, BC, AD}，总权 = 1+2+4 = 7

Prufer 序列证明思路：建立 n 个标号顶点的树与长度为 n-2 的 Prufer 序列（值域 {1,2,...,n}）之间的双射。
每棵树对应唯一的序列，每个序列对应唯一的树，故总数为 nn-2。

4. 有根树与二叉树

4.1 有根树 (Rooted Tree)

指定一个顶点为根 (root) 的树。从根到每个顶点有唯一路径，定义了层次关系。

术语：
  • 父节点 (Parent)：路径上的上一个节点
  • 子节点 (Child)：路径上的下一个节点
  • 深度 (Depth)：从根到该节点的路径长度
  • 高度 (Height)：从该节点到最深叶节点的最长路径
  • 层数：深度 + 1（根在第1层）

4.2 二叉树 (Binary Tree)

每个节点最多有2个子节点（左子和右子）的有根树。

二叉树性质：
  • 高度为 h 的二叉树最多有 2h+1 - 1 个节点
  • n 个节点的二叉树最小高度为 ⌊log₂n⌋
  • n 个节点的不同二叉树数目 = 第 n 个Catalan 数：C(n) = C(2n,n)/(n+1)
  • 叶节点数 = 度为2的节点数 + 1

4.3 完全二叉树与满二叉树

类型定义节点数 (高度h)
满二叉树每个内部节点恰好有2个子节点，所有叶节点在同一层2h+1-1
完全二叉树除最后一层外都满，最后一层靠左排列2h ~ 2h+1-1
平衡二叉树左右子树高度差 ≤ 1O(log n) 高度

4.4 二叉树遍历

遍历方式顺序应用
前序 (Pre-order)根 → 左 → 右复制表达式树
中序 (In-order)左 → 根 → 右表达式求值
后序 (Post-order)左 → 右 → 根释放内存、后缀表达式
层序 (Level-order)逐层从左到右BFS
表达式 (a + b) * c 的表达式树：
前序：* + a b c    中序：a + b * c    后序：a b + c *

5. 应用与扩展

5.1 生成树的数目

矩阵树定理 (Kirchhoff)：图 G 的生成树数目 = 其拉普拉斯矩阵 L 的任意一个代数余子式的值。
L = D - A，其中 D 是度数矩阵，A 是邻接矩阵。

5.2 实际应用

  • 网络设计：用最少的线缆连接所有节点（最小生成树）
  • 聚类算法：最小生成树去掉 k-1 条最长边得到 k 个聚类
  • 决策树：机器学习中的分类与回归
  • Huffman 编码：基于二叉树的最优前缀编码

本节总结

  • 树的等价条件：连通无圈 ⟺ 连通且 E=V-1 ⟺ 无圈且 E=V-1 ⟺ 任意两点唯一路径
  • 生成树：连通图必有生成树；最小生成树可用 Prim / Kruskal 算法
  • Prim 算法：顶点扩展，O(V²) 或 O(E log V)
  • Kruskal 算法：边排序 + 并查集，O(E log E)
  • Cayley 公式：n 个标号顶点的树有 nn-2 棵
  • 二叉树：Catalan 数计数，h 高度最多 2h+1-1 个节点
  • 矩阵树定理：拉普拉斯矩阵代数余子式 = 生成树数目
