==================== 1. 图的定义 ====================

## 1. 图的定义


### 图（Graph）的数学定义


**图** G = (V, E) 由两个集合组成：


- **V（Vertex）：**
   非空的顶点集合，V ≠ ∅
- **E（Edge）：**
   边的集合，每条边连接两个顶点


**无向图：**边是无序对 (u, v)，表示 u 和 v 之间的连接。


**有向图：**边是有序对 ，表示从 u 到 v 的有向连接。

==================== 2. 图的分类 ====================

## 2. 图的分类


| 类型 | 边的性质 | 示例 |
| --- | --- | --- |
| 无向图（Undirected Graph） | 边是无序对 | 社交网络（好友关系） |
| 有向图（Directed Graph / Digraph） | 边是有序对 | 网页链接、关注关系 |
| 多重图（Multigraph） | 允许平行边（多条边连接同一对顶点） | 城市间多条航线 |
| 简单图（Simple Graph） | 无平行边、无自环 | 本笔记主要讨论对象 |
| 加权图（Weighted Graph） | 边带权重 | 道路网络（距离/时间） |
| 完全图 K~n~ | 每对顶点之间都有边 | K₅ — 5个顶点的完全图 |

==================== 4. 度 ====================

## 4. 度（Degree）


### 无向图的度


顶点 v 的**度** deg(v) 是与 v 相关联的边的数目（自环算两次）。

握手定理（Handshaking Lemma）：


Σ deg(v) = 2|E|


所有顶点的度之和 = 边数的两倍

**推论：**度为奇数的顶点个数一定是偶数。


> **Warning:** ### 有向图的度
>
>
> 每个顶点有两类度：
>
>
> - **入度 deg⁻(v)：**
>    指向 v 的边的数量（以 v 为终点）
> - **出度 deg⁺(v)：**
>    从 v 出发的边的数量（以 v 为起点）
>
>
> deg(v) = deg⁻(v) + deg⁺(v)
>
> 有向图握手定理：
>
>
> Σ deg⁺(v) = Σ deg⁻(v) = |E|


## 4.1 度序列计算示例


以上面的无向图为例：V = {A, B, C, D, E}，E = {(A,B),(A,C),(B,C),(C,D),(D,E)}


| 顶点 | 相邻边 | 度数 | 奇偶 |
| --- | --- | --- | --- |
| A | (A,B), (A,C) | 2 | 偶 |
| B | (A,B), (B,C) | 2 | 偶 |
| C | (A,C), (B,C), (C,D) | 3 | **奇** |
| D | (C,D), (D,E) | 2 | 偶 |
| E | (D,E) | 1 | **奇** |
| 总和 | 10 | 2个奇度顶点 |


验证：Σdeg(v) = 2+2+3+2+1 = 10 = 2×5 = 2|E|


奇度顶点数 = 2（偶数），符合握手定理推论。

==================== 5. 特殊图 ====================

## 5. 特殊图


### 完全图 K~n~


n 个顶点的简单图中，每对顶点之间都有边相连的图。


**边数：**|E| = C(n,2) = n(n-1)/2


**每个顶点的度：**n-1


### 正则图（Regular Graph）


所有顶点的度都相等的图。**k-正则图**表示每个顶点的度都为 k。


完全图 K~n~ 是 (n-1)-正则图。


3-正则图也称为**立方图**。


> **Warning:** ### 二部图（Bipartite Graph）
>
>
> 顶点集可以划分为两个不相交的子集 V₁ 和 V₂，使得每条边的两个端点分别在 V₁ 和 V₂ 中。
>
>
> **等价条件：**G 是二部图 ↔ G 中不含奇数长度的环
>
>
> **完全二部图 K~m,n~：**V₁ 有 m 个顶点，V₂ 有 n 个顶点，每对跨集顶点之间都有边。
>
>
> |E(K~m,n~)| = m × n

## Python实现

### 图的基本操作与性质验证

```python
import numpy as np

def adjacency_matrix(vertices, edges, directed=False):
    """构建邻接矩阵"""
    idx = {v: i for i, v in enumerate(vertices)}
    n = len(vertices)
    A = np.zeros((n, n), dtype=int)
    for u, v in edges:
        A[idx[u]][idx[v]] = 1
        if not directed:
            A[idx[v]][idx[u]] = 1
    return A

# 无向图示例
V = ['A', 'B', 'C', 'D', 'E']
E = [('A', 'B'), ('A', 'C'), ('B', 'C'), ('C', 'D'), ('D', 'E')]
A = adjacency_matrix(V, E)

print("=== 无向图 ===")
print(f"顶点: {V}")
print(f"边: {E}")
print(f"邻接矩阵:\n{A}")
print(f"矩阵对称: {np.array_equal(A, A.T)}")

# 度数计算
degrees = A.sum(axis=1)
print(f"\n各顶点度数:")
for v, d in zip(V, degrees):
    print(f"  deg({v}) = {d}")
print(f"度数总和 = {sum(degrees)}, 2|E| = {2 * len(E)}")
print(f"握手定理验证: {sum(degrees) == 2 * len(E)}")
print(f"奇度顶点数: {sum(1 for d in degrees if d % 2 == 1)} (必为偶数)")

# 完全图 K_n
print("\n=== 完全图 K_n ===")
for n in [3, 4, 5]:
    edges_count = n * (n - 1) // 2
    print(f"K_{n}: {n}个顶点, {edges_count}条边, 每点度数={n-1}")

# 验证邻接矩阵性质
print("\n=== 邻接矩阵性质 ===")
A_K4 = adjacency_matrix(
    [1, 2, 3, 4],
    [(1, 2), (1, 3), (1, 4), (2, 3), (2, 4), (3, 4)]
)
print(f"K4 邻接矩阵:\n{A_K4}")
print(f"A² (2步路径数):\n{A_K4 @ A_K4}")
```

### 有向图的入度与出度

```python
import numpy as np

# 有向图示例
V = ['A', 'B', 'C', 'D']
E_dir = [('A', 'B'), ('B', 'C'), ('C', 'D'), ('A', 'C'), ('B', 'D')]
A_dir = adjacency_matrix(V, E_dir, directed=True)

print("=== 有向图 ===")
print(f"顶点: {V}")
print(f"有向边: {E_dir}")
print(f"邻接矩阵:\n{A_dir}")
print(f"矩阵不对称: {not np.array_equal(A_dir, A_dir.T)}")

# 入度 = 列和, 出度 = 行和
out_degrees = A_dir.sum(axis=1)
in_degrees = A_dir.sum(axis=0)

print(f"\n{'顶点':>4} | {'出度':>4} | {'入度':>4} | {'总度':>4}")
print("-" * 25)
for i, v in enumerate(V):
    print(f"{v:>4} | {out_degrees[i]:>4} | {in_degrees[i]:>4} | {out_degrees[i] + in_degrees[i]:>4}")

print(f"\n出度之和 = {sum(out_degrees)}, 入度之和 = {sum(in_degrees)}, |E| = {len(E_dir)}")
print(f"Σdeg⁺ = Σdeg⁻ = |E|: {sum(out_degrees) == sum(in_degrees) == len(E_dir)}")
```
