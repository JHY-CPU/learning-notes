==================== 1. 遍历概述 ====================

## 1. 图的遍历概述


### 什么是图的遍历？


**图的遍历（Graph Traversal）**是从图中的某个顶点出发，按照某种策略访问图中所有顶点恰好一次的过程。


两种基本策略：


- **BFS（广度优先搜索）：**
   Breadth-First Search — 按层遍历，先近后远
- **DFS（深度优先搜索）：**
   Depth-First Search — 一路深入，回溯探索

==================== 3. BFS ====================

## 3. 广度优先搜索（BFS）


### BFS 算法思想


从起始顶点出发，**先访问所有距离为1的顶点，再访问距离为2的顶点**，以此类推 — "逐层扩展"。


**数据结构：**队列（Queue）— 先进先出


**特点：**


- 找到的路径是最短路径（无权图中）
- 适合求最短路径、层序遍历
- 空间复杂度较高（需存储整层顶点）


### BFS 伪代码


```
BFS(G, start):
    创建队列 Q
    将 start 加入 Q
    标记 start 为已访问

    while Q 不为空:
        v = Q.dequeue()        // 取出队首
        访问 v
        for v 的每个邻居 w:
            if w 未访问:
                标记 w 为已访问
                Q.enqueue(w)   // 加入队尾
```


## 3.1 BFS 执行过程（从 A 出发）


| 步骤 | 当前访问 | 队列状态 | 已访问集合 | 操作说明 |
| --- | --- | --- | --- | --- |
| 初始化 | - | [A] | {} | A 入队 |
| 1 | A | [B, D] | {A} | 访问A，邻居B、D入队 |
| 2 | B | [D, E, C] | {A, B} | 访问B，邻居E、C入队 |
| 3 | D | [E, C, G] | {A, B, D} | 访问D，邻居G入队 |
| 4 | E | [C, G] | {A, B, D, E} | 访问E，无新邻居 |
| 5 | C | [G, F] | {A, B, D, E, C} | 访问C，邻居F入队 |
| 6 | G | [F] | {A, B, D, E, C, G} | 访问G，无新邻居 |
| 7 | F | []（空） | {A, B, D, E, C, G, F} | 访问F，队列空，结束 |


**BFS 序列（层序）：**A → B → D → E → C → G → F


**BFS 树（按层）：**第1层: {A}，第2层: {B, D}，第3层: {E, C, G}，第4层: {F}

==================== 4. DFS ====================

## 4. 深度优先搜索（DFS）


### DFS 算法思想


从起始顶点出发，**沿着一条路径尽可能深入**，直到无法继续时**回溯**到上一个分叉点，继续探索其他路径。


**数据结构：**栈（Stack）或递归调用栈 — 后进先出


**特点：**


- 适合判断连通性、检测环、拓扑排序
- 空间复杂度较低（只需存储当前路径）
- 找到的路径不一定是最短的


### DFS 伪代码


```
DFS(G, v):
    标记 v 为已访问
    访问 v

    for v 的每个邻居 w:
        if w 未访问:
            DFS(G, w)          // 递归深入
```


## 4.1 DFS 执行过程（从 A 出发）


| 步骤 | 当前访问 | 递归栈（底→顶） | 已访问集合 | 操作说明 |
| --- | --- | --- | --- | --- |
| 1 | A | [A] | {A} | 从A出发，探索邻居B |
| 2 | B | [A, B] | {A, B} | 从B出发，探索邻居E |
| 3 | E | [A, B, E] | {A, B, E} | E无未访问邻居，回溯到B |
| 4 | C | [A, B, C] | {A, B, E, C} | 从B继续，探索C |
| 5 | F | [A, B, C, F] | {A, B, E, C, F} | 从C出发，探索F |
| 6 | 回溯 | [A, B, C] → [A, B] → [A] | {A, B, E, C, F} | F/C/B无未访问邻居，回溯到A |
| 7 | D | [A, D] | {A, B, E, C, F, D} | 从A继续，探索D |
| 8 | G | [A, D, G] | {A, B, E, C, F, D, G} | 从D出发，探索G |


**DFS 序列：**A → B → E → C → F → D → G


**注意：**DFS序列不唯一，取决于邻居的访问顺序。上面按字母序探索。

==================== 5. BFS vs DFS 对比 ====================

## 5. BFS vs DFS 对比


| 特性 | BFS（广度优先） | DFS（深度优先） |
| --- | --- | --- |
| 数据结构 | 队列（Queue） | 栈（Stack）/ 递归 |
| 搜索方式 | 逐层扩展 | 一路深入再回溯 |
| 最短路径 | 保证（无权图） | 不保证 |
| 空间复杂度 | O(b^d^) — 较大 | O(b·m) — 较小 |
| 时间复杂度 | O(V + E) | O(V + E) |
| 适用场景 | 最短路径、层序 | 连通性、环检测、拓扑排序 |
| 完备性 | 完备（能找到解） | 可能陷入深路径 |

==========================================
    总结：
    1. 图的遍历：从某顶点出发访问所有顶点恰好一次
    2. BFS（广度优先）：队列实现，逐层扩展，保证最短路径，时间O(V+E)
    3. DFS（深度优先）：栈/递归实现，一路深入再回溯，适合连通性和环检测
    4. 两种遍历的时间复杂度都是 O(V+E)，区别在搜索策略和适用场景
    5. BFS 序列和 DFS 序列都不唯一（取决于邻居访问顺序）

## Python实现

### BFS 与 DFS 实现

```python
from collections import deque

def bfs(graph, start):
    """广度优先搜索"""
    visited = set()
    queue = deque([start])
    visited.add(start)
    order = []

    while queue:
        v = queue.popleft()
        order.append(v)
        for neighbor in graph[v]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return order

def dfs(graph, start):
    """深度优先搜索 (递归)"""
    visited = set()
    order = []

    def _dfs(v):
        visited.add(v)
        order.append(v)
        for neighbor in graph[v]:
            if neighbor not in visited:
                _dfs(neighbor)

    _dfs(start)
    return order

def dfs_iterative(graph, start):
    """深度优先搜索 (迭代)"""
    visited = set()
    stack = [start]
    order = []

    while stack:
        v = stack.pop()
        if v not in visited:
            visited.add(v)
            order.append(v)
            # 反序入栈以保证字母序
            for neighbor in reversed(graph[v]):
                if neighbor not in visited:
                    stack.append(neighbor)
    return order

# 构建示例图
graph = {
    'A': ['B', 'D'],
    'B': ['A', 'E', 'C'],
    'C': ['B', 'F'],
    'D': ['A', 'G'],
    'E': ['B'],
    'F': ['C'],
    'G': ['D']
}

print("=== BFS (从 A 出发) ===")
bfs_order = bfs(graph, 'A')
print(f"遍历序列: {' → '.join(bfs_order)}")

print("\n=== DFS (从 A 出发) ===")
dfs_order = dfs(graph, 'A')
print(f"遍历序列: {' → '.join(dfs_order)}")

# BFS 求最短路径 (无权图)
def bfs_shortest_path(graph, start, end):
    """BFS 求无权图最短路径"""
    visited = {start: None}
    queue = deque([start])

    while queue:
        v = queue.popleft()
        if v == end:
            # 回溯路径
            path = []
            node = end
            while node is not None:
                path.append(node)
                node = visited[node]
            return path[::-1]
        for neighbor in graph[v]:
            if neighbor not in visited:
                visited[neighbor] = v
                queue.append(neighbor)
    return None

print("\n=== BFS 最短路径 ===")
path = bfs_shortest_path(graph, 'A', 'F')
print(f"A → F 最短路径: {' → '.join(path)}, 长度 = {len(path) - 1}")

# 验证连通性
def is_connected(graph):
    """判断图是否连通"""
    start = next(iter(graph))
    visited = set()
    queue = deque([start])
    while queue:
        v = queue.popleft()
        if v not in visited:
            visited.add(v)
            queue.extend(n for n in graph[v] if n not in visited)
    return len(visited) == len(graph)

print(f"\n图连通: {is_connected(graph)}")
```

### 拓扑排序 (DFS 应用)

```python
def topological_sort(graph_directed):
    """基于 DFS 的拓扑排序"""
    visited = set()
    stack = []

    def dfs_topo(v):
        visited.add(v)
        for neighbor in graph_directed.get(v, []):
            if neighbor not in visited:
                dfs_topo(neighbor)
        stack.append(v)  # 后序添加

    for v in graph_directed:
        if v not in visited:
            dfs_topo(v)

    return stack[::-1]
```
