==================== 1. 欧拉路径与回路 ====================

## 1. 欧拉路径与欧拉回路


### 1.1 基本定义


> **Note:** **欧拉路径 (Eulerian Path)：**
> 经过图中
> **每条边恰好一次**
> 的路径。
>
>
> **欧拉回路 (Eulerian Circuit)：**
> 起点和终点相同的欧拉路径。
>
>
> **欧拉图：**
> 含有欧拉回路的图。
>
>
>
>
> 历史背景：源于 1736 年欧拉解决的"哥尼斯堡七桥问题"。

### 1.2 欧拉定理 (度数条件)


> **Theorem:** 对于
> **无向连通图**
> G：
>
>
>
> - G 有
>    **欧拉回路**
>    ⟺
>    **所有顶点度数为偶数**
> - G 有
>    **欧拉路径**
>    （非回路）⟺
>    **恰好 2 个顶点度数为奇数**
>    ，其余为偶数


> **Theorem:** 对于
> **有向连通图**
> G：
>
>
>
> - G 有
>    **有向欧拉回路**
>    ⟺ 每个顶点入度 = 出度
> - G 有
>    **有向欧拉路径**
>    ⟺ 恰好一个顶点出度-入度=1，一个入度-出度=1，其余相等


### 1.3 Fleury 算法 (构造欧拉回路)


> **Note:** **算法步骤：**
>
> 1. 从任意奇度顶点出发（若无奇度顶点则任意出发）
> 2. 沿未访问的边行走，
>    **除非没有其他选择，不走桥（割边）**
> 3. 直到所有边都被访问
>
> 时间复杂度：O(E²)。Hierholzer 算法可优化到 O(E)。

==================== 2. 哈密顿路径与回路 ====================

## 2. 哈密顿路径与哈密顿回路


### 2.1 基本定义


> **Note:** **哈密顿路径 (Hamiltonian Path)：**
> 经过图中
> **每个顶点恰好一次**
> 的路径。
>
>
> **哈密顿回路 (Hamiltonian Circuit)：**
> 起点和终点相同的哈密顿路径。
>
>
> **哈密顿图：**
> 含有哈密顿回路的图。


> **Warning:** **核心区别：**
>
> - 欧拉路径：遍历所有
>    **边**
>    （度数条件可判定）
> - 哈密顿路径：遍历所有
>    **顶点**
>    （无简洁充要条件，是 NP-完全问题）


### 2.2 充分条件


> **Theorem:** **Dirac 定理：**
> 若简单图 G 有 n≥3 个顶点，且每个顶点度数 ≥ n/2，则 G 有哈密顿回路。


> **Theorem:** **Ore 定理：**
> 若简单图 G 有 n≥3 个顶点，且对每对不相邻顶点 u,v 有 deg(u)+deg(v) ≥ n，则 G 有哈密顿回路。


| 图 | 哈密顿路径 | 哈密顿回路 |
| --- | --- | --- |
| 完全图 Kₙ | 有 (任意排列) | 有 ((n-1)!/2 个) |
| 二分图 Kₘ,ₙ | 有 (若 \|m-n\|≤1) | 有 (仅当 m=n) |
| 彼得森图 | 有 | 有 |
| 偶圈 C₂ₙ | 有 | 有 (就是自身) |

==================== 3. 旅行商问题 ====================

## 3. 旅行商问题 (TSP)


### 3.1 问题描述


给定 n 个城市和城市间的距离（或费用），找到访问每个城市恰好一次并回到起点的**最短回路**。


$$
TSP 的解空间大小 = (n-1)! / 2 （对称情况）
$$


### 3.2 复杂度与求解方法


| 方法 | 类型 | 时间复杂度 | 近似比 |
| --- | --- | --- | --- |
| 暴力枚举 | 精确 | O(n!) | 最优 |
| 动态规划 (Held-Karp) | 精确 | O(n²·2ⁿ) | 最优 |
| 分支限界 | 精确 | 指数级（实践较快） | 最优 |
| 最近邻法 | 贪心近似 | O(n²) | 无保证 |
| Christofides | 近似 | O(n³) | ≤1.5倍最优 |
| 模拟退火/遗传算法 | 启发式 | 视参数 | 通常较好 |


> **Example:** 4 个城市的 TSP：
>
>
> 距离矩阵 D = [[0,10,15,20],[10,0,35,25],[15,35,0,30],[20,25,30,0]]
>
>
> 枚举 (4-1)!/2 = 3 条不同回路：
>
>
> A→B→C→D→A: 10+35+30+20 = 95
>
>
> A→B→D→C→A: 10+25+30+15 = 80 ✓ 最优
>
>
> A→C→B→D→A: 15+35+25+20 = 95


### 3.3 TSP 与哈密顿回路的关系


> **Note:** 哈密顿回路问题 = TSP 中所有边权为 1 的特殊情况。
>
>
> 由于哈密顿回路是 NP-完全的，TSP 至少也是 NP-难的。
>
>
> TSP 的优化版本不在 NP 中（验证最优解需要更长时间），但判定版本（"是否存在费用 ≤ k 的回路"）是 NP-完全的。

==================== 4. 两类路径的对比 ====================

## 4. 欧拉路径 vs 哈密顿路径 对比


| 特性 | 欧拉路径 | 哈密顿路径 |
| --- | --- | --- |
| 遍历对象 | 每条**边**恰好一次 | 每个**顶点**恰好一次 |
| 充要条件 | 简洁（度数条件） | 不存在已知的简洁充要条件 |
| 判定复杂度 | **P** (多项式时间) | **NP-完全** |
| 构造算法 | Fleury / Hierholzer | 回溯、动态规划、近似 |
| 应用场景 | 邮路问题、一笔画 | TSP、排课问题 |
| 历史 | 1736 欧拉 | 1857 哈密顿 |

==================== 总结 ====================

> **Summary:** ## 本节总结
>
>
> - **欧拉回路存在条件：**
>    无向连通图所有顶点度数为偶数
> - **欧拉路径存在条件：**
>    恰好 2 个奇度顶点
> - **有向欧拉回路：**
>    每个顶点入度 = 出度
> - **Fleury 算法：**
>    避免走桥，O(E²)；Hierholzer 算法 O(E)
> - **哈密顿路径：**
>    遍历每个顶点恰好一次，NP-完全
> - **Dirac 定理：**
>    δ(G) ≥ n/2 → 哈密顿
> - **Ore 定理：**
>    不相邻顶点度数和 ≥ n → 哈密顿
> - **TSP：**
>    NP-难，精确算法 O(n²·2ⁿ) (Held-Karp)，Christofides 近似比 1.5

## Python实现

### 欧拉路径/回路判定与构造

```python
from collections import defaultdict

def find_euler_circuit(vertices, edges):
    """Hierholzer 算法找欧拉回路"""
    adj = defaultdict(list)
    edge_count = defaultdict(int)
    for u, v in edges:
        adj[u].append(v)
        adj[v].append(u)
        edge_count[tuple(sorted([u, v]))] += 1

    # 检查欧拉回路条件: 所有顶点度数为偶数
    degrees = defaultdict(int)
    for u, v in edges:
        degrees[u] += 1
        degrees[v] += 1

    odd_vertices = [v for v in degrees if degrees[v] % 2 == 1]
    if len(odd_vertices) == 0:
        path_type = "欧拉回路"
    elif len(odd_vertices) == 2:
        path_type = "欧拉路径"
    else:
        return None, f"无欧拉路径/回路 (奇度顶点: {odd_vertices})"

    # Hierholzer 算法
    stack = [vertices[0]]
    circuit = []
    adj_copy = {v: list(adj[v]) for v in adj}

    while stack:
        v = stack[-1]
        if adj_copy[v]:
            u = adj_copy[v].pop()
            adj_copy[u].remove(v)
            stack.append(u)
        else:
            circuit.append(stack.pop())

    return circuit[::-1], path_type

# 示例1: 有欧拉回路的图
print("=== 欧拉回路检测 ===")
V1 = ['A', 'B', 'C', 'D']
E1 = [('A', 'B'), ('A', 'C'), ('B', 'D'), ('C', 'D'), ('A', 'D'), ('B', 'C')]
circuit, ptype = find_euler_circuit(V1, E1)
print(f"图: {E1}")
print(f"类型: {ptype}")
print(f"路径: {' → '.join(circuit)}")

# 示例2: 有欧拉路径的图
print("\n=== 欧拉路径检测 ===")
V2 = ['A', 'B', 'C', 'D']
E2 = [('A', 'B'), ('A', 'C'), ('B', 'C'), ('C', 'D')]
circuit, ptype = find_euler_circuit(V2, E2)
print(f"图: {E2}")
print(f"类型: {ptype}")
print(f"路径: {' → '.join(circuit)}")
```

### TSP 暴力求解

```python
from itertools import permutations

def tsp_brute_force(dist_matrix):
    """TSP 暴力枚举求解"""
    n = len(dist_matrix)
    cities = list(range(1, n))  # 固定从城市0出发
    best_cost = float('inf')
    best_path = None

    for perm in permutations(cities):
        path = (0,) + perm + (0,)
        cost = sum(dist_matrix[path[i]][path[i + 1]] for i in range(n))
        if cost < best_cost:
            best_cost = cost
            best_path = path

    return best_path, best_cost

# 示例: 4个城市
D = [
    [0, 10, 15, 20],
    [10, 0, 35, 25],
    [15, 35, 0, 30],
    [20, 25, 30, 0]
]

print("=== TSP 暴力求解 ===")
print(f"距离矩阵:")
for i, row in enumerate(D):
    print(f"  城市{i}: {row}")

path, cost = tsp_brute_force(D)
city_names = ['A', 'B', 'C', 'D']
print(f"\n最优路径: {' → '.join(city_names[i] for i in path)}")
print(f"最优费用: {cost}")

# 列举所有回路
from itertools import permutations
print(f"\n所有回路 (共 {(len(D)-1)//2} 种不同回路):")
seen = set()
for perm in permutations(range(1, len(D))):
    cost = sum(D[perm[i - 1]][perm[i]] for i in range(len(perm)))
    cost += D[0][perm[0]] + D[perm[-1]][0]
    route = f"A→{'→'.join(chr(65 + p) for p in perm)}→A"
    if route not in seen:
        print(f"  {route}: {cost}")
        seen.add(route)
```
