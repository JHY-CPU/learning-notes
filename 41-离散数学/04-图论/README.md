# 04-图论

图论是离散数学中与计算机科学联系最紧密的分支，广泛应用于网络分析、路径规划、编译优化、社交网络等领域。本章系统介绍图论的核心概念与经典算法。

---

## 1. 图的基本概念

### 1.1 图的定义

**图** $G = (V, E)$ 由**顶点集** $V$ 和**边集** $E$ 组成。

- **无向图**：边是无序对，$e = \{u, v\}$
- **有向图**：边是有序对，$e = (u, v)$
- **简单图**：无自环、无重边的图
- **多重图**：允许重边的图
- **完全图** $K_n$：任意两个不同顶点之间都有边

### 1.2 度

- **无向图的度**：$\deg(v)$ = 与 $v$ 关联的边数（自环算两次）
- **有向图的度**：入度 $deg^-(v)$ + 出度 $deg^+(v)$

**握手定理**：在无向图中，所有顶点的度之和 = $2|E|$。

**推论**：度为奇数的顶点个数必为偶数。

**有向图握手定理**：$\sum deg^+(v) = \sum deg^-(v) = |E|$。

### 1.3 路径与回路

- **路径（通路）**：顶点序列 $v_0, v_1, \ldots, v_k$，相邻顶点之间有边。
- **简单路径**：路径上的边不重复。
- **基本路径**（初级路径）：路径上的顶点不重复。
- **回路（圈）**：起点和终点相同的路径。
- **连通**：若存在从 $u$ 到 $v$ 的路径，则 $u$ 与 $v$ 连通。

---

## 2. 图的表示

### 2.1 邻接矩阵

$A = [a_{ij}]_{n \times n}$，其中：
$$a_{ij} = \begin{cases} 1, & (v_i, v_j) \in E \\ 0, & \text{其他} \end{cases}$$

- 无向图的邻接矩阵是对称矩阵。
- 顶点 $v_i$ 的度 = 第 $i$ 行（或列）之和。
- $A^k$ 的 $(i,j)$ 元素表示从 $v_i$ 到 $v_j$ 长度为 $k$ 的路径数。
- 空间复杂度：$O(n^2)$，适合稠密图。

### 2.2 邻接表

为每个顶点维护一个链表，存储其邻接顶点。

- 空间复杂度：$O(n + m)$，适合稀疏图。
- 无向图中每条边存储两次。

### 2.3 关联矩阵

$M = [m_{ij}]_{n \times m}$（$n$ 个顶点，$m$ 条边）：
$$m_{ij} = \begin{cases} 1, & v_i \text{ 与 } e_j \text{ 关联} \\ 0, & \text{其他} \end{cases}$$

有向图中，$m_{ij} = 1$（$v_i$ 是 $e_j$ 的起点），$m_{ij} = -1$（$v_i$ 是 $e_j$ 的终点）。

## 3. 连通性

### 3.1 无向图的连通性

- **连通图**：任意两个顶点之间存在路径。
- **连通分量**：极大的连通子图。

**割点**：删除该顶点后连通分量数增加。
**割边（桥）**：删除该边后连通分量数增加。

**定理**：$e = \{u, v\}$ 是割边当且仅当 $e$ 不在任何回路上。

### 3.2 有向图的连通性

- **强连通**：任意两点 $u, v$ 之间同时存在 $u \to v$ 和 $v \to u$ 的路径。
- **强连通分量**（SCC）：极大的强连通子图。

**算法**：Tarjan 算法或 Kosaraju 算法可在 $O(V+E)$ 时间内求出所有 SCC。

### 3.3 点连通度与边连通度

- **点连通度** $\kappa(G)$：使图不连通或变为平凡图所需删除的最少顶点数。
- **边连通度** $\lambda(G)$：使图不连通所需删除的最少边数。
- **Whitney 不等式**：$\kappa(G) \leq \lambda(G) \leq \delta(G)$（$\delta$ 为最小度）。

---

## 4. 欧拉图

### 4.1 欧拉回路与欧拉路径

- **欧拉回路**：经过图中每条边恰好一次且起点=终点的回路。
- **欧拉路径**：经过图中每条边恰好一次的路径（起点≠终点）。

### 4.2 欧拉定理

**无向图**存在欧拉回路 $\Leftrightarrow$ 图连通且每个顶点的度均为偶数。

**无向图**存在欧拉路径（非回路）$\Leftrightarrow$ 图连通且恰有两个度为奇数的顶点。

**有向图**存在欧拉回路 $\Leftrightarrow$ 图弱连通且每个顶点的入度=出度。

**Fleury 算法**：从某顶点出发，沿着边走，尽量不走桥（除非没有其他选择），时间复杂度 $O(m^2)$。

**Hierholzer 算法**：更高效的 $O(m)$ 算法，利用 DFS 找回路并拼接。

---

## 5. 哈密顿图

### 5.1 哈密顿回路与哈密顿路径

- **哈密顿回路**：经过每个顶点恰好一次（除起点/终点外）的回路。
- **哈密顿路径**：经过每个顶点恰好一次的路径。

### 5.2 充分条件

**Dirac 定理**：若 $n \geq 3$ 的简单图中每个顶点的度 $\geq n/2$，则存在哈密顿回路。

**Ore 定理**：若 $n \geq 3$ 的简单图中，对每对不相邻顶点 $u, v$，有 $\deg(u) + \deg(v) \geq n$，则存在哈密顿回路。

### 5.3 重要区分

- **判定欧拉图**有多项式时间算法（检查度的奇偶性）。
- **判定哈密顿图**是 NP 完全问题，无已知多项式时间算法。

## 6. 树与生成树

### 6.1 树的定义

**树**是连通无回路的无向图。

**等价定义**（以下对 $n$ 个顶点的图等价）：
1. 连通且无回路。
2. 连通且有 $n-1$ 条边。
3. 无回路且有 $n-1$ 条边。
4. 任意两个顶点之间有且仅有一条基本路径。

**森林**：无回路的无向图（若干棵树的并）。

### 6.2 生成树

**生成树**是包含图中所有顶点的树子图。

- 连通图一定存在生成树。
- $n$ 个顶点的连通图，生成树有 $n-1$ 条边。

### 6.3 最小生成树

**带权连通图**的**最小生成树**（MST）是边权之和最小的生成树。

**Prim 算法**（加点法）：
1. 从任意顶点开始，将顶点逐个加入已构建的树。
2. 每次选择连接"树内"与"树外"的最小权边。
3. 时间复杂度：$O(n^2)$（朴素实现）或 $O(m \log n)$（堆优化）。

**Kruskal 算法**（加边法）：
1. 将所有边按权值排序。
2. 依次添加最小权边，若形成回路则跳过。
3. 使用并查集判断是否形成回路。
4. 时间复杂度：$O(m \log m)$。

## 7. 平面图与欧拉公式

### 7.1 平面图

**平面图**是可以画在平面上使得边仅在顶点处相交的图。

**欧拉公式**：连通平面图 $G = (V, E)$，面数为 $F$（包含外部无限面），则：

$$|V| - |E| + |F| = 2$$

### 7.2 平面图的必要条件

若 $G$ 是至少有 3 个顶点的简单连通平面图，则：

$$|E| \leq 3|V| - 6$$

**推论**：$K_5$ 不是平面图（$|E| = 10 > 3 \times 5 - 6 = 9$）。

**若图中不含三角形**：$|E| \leq 2|V| - 4$（故 $K_{3,3}$ 不是平面图）。

### 7.3 Kuratowski 定理

图 $G$ 是平面图 $\Leftrightarrow$ $G$ 不包含 $K_5$ 或 $K_{3,3}$ 的细分（subdivision）。

## 8. 图着色

### 8.1 着色与色数

**正常着色**：给图的每个顶点分配一个颜色，使得相邻顶点颜色不同。

**色数** $\chi(G)$：使 $G$ 正常着色所需的最少颜色数。

**基本结论**：
- $\chi(K_n) = n$
- 二分图（偶图）$\chi(G) = 2$ 或 $1$
- 回路 $C_n$：$\chi(C_n) = 2$（$n$ 为偶数），$\chi(C_n) = 3$（$n$ 为奇数，$n \geq 3$）
- 树 $\chi(T) = 2$（非平凡树）

### 8.2 四色定理

任何平面图的色数不超过 4，即 $\chi(G) \leq 4$。

**Brooks 定理**：若 $G$ 不是完全图也不是奇回路，则 $\chi(G) \leq \Delta(G)$（$\Delta$ 为最大度）。

### 8.3 边着色

**边色数** $\chi'(G)$：使图中相邻边颜色不同的最少颜色数。

**Vizing 定理**：$\Delta(G) \leq \chi'(G) \leq \Delta(G) + 1$。

## 9. 匹配

### 9.1 基本概念

**匹配**：图 $G$ 中一组互不相邻的边。

**最大匹配**：包含边数最多的匹配。

**完美匹配**：覆盖所有顶点的匹配（每个顶点恰好关联一条匹配边）。

### 9.2 二分图匹配

**Hall 定理**（婚姻定理）：二分图 $G = (X \cup Y, E)$ 中存在从 $X$ 到 $Y$ 的完全匹配 $\Leftrightarrow$ 对任意 $S \subseteq X$，有 $|N(S)| \geq |S|$（$N(S)$ 为 $S$ 的邻域）。

**推论**：$k$-正则二分图存在完美匹配。

**匈牙利算法**：在二分图中寻找最大匹配，基于增广路思想，时间复杂度 $O(VE)$。

**增广路定理**：匹配 $M$ 是最大匹配 $\Leftrightarrow$ 不存在 $M$-增广路。

## 10. 网络流基础

### 10.1 网络与流

**网络**是有向图 $G = (V, E)$，附带容量函数 $c: E \rightarrow \mathbb{R}^+$、源点 $s$ 和汇点 $t$。

**可行流** $f$ 满足：
1. 容量约束：$0 \leq f(e) \leq c(e)$
2. 流量守恒：除 $s, t$ 外，每个顶点的净流入 = 净流出

**流的值** $|f| = \sum_{v} f(s, v) - \sum_{v} f(v, s)$。

### 10.2 最大流最小割定理

**割** $(S, T)$：将顶点划分为 $S$（含 $s$）和 $T$（含 $t$）两部分。割的**容量**为从 $S$ 到 $T$ 的边的容量之和。

**最大流最小割定理**：网络中的最大流值 = 最小割容量。

**Ford-Fulkerson 方法**：反复找增广路并增流，直到无增广路。

**Edmonds-Karp 算法**：用 BFS 找增广路（最短增广路），时间复杂度 $O(VE^2)$。

**Dinic 算法**：分层图 + 阻塞流，时间复杂度 $O(V^2 E)$。
