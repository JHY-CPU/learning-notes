==================== 1. 命题的基本概念 ====================

## 1. 命题的基本概念


### 什么是命题？


**命题（Proposition）**是一个能判断真假的陈述句。命题的取值只有两个：


- **真（True, T, 1）**
   — 命题所描述的事实成立
- **假（False, F, 0）**
   — 命题所描述的事实不成立


注意：感叹句、疑问句、祈使句、悖论句都不是命题。


### 命题示例


| 语句 | 是否为命题 | 理由 |
| --- | --- | --- |
| 北京是中国的首都 | 是（真） | 陈述句，可判断真假 |
| 2 + 2 = 5 | 是（假） | 陈述句，虽为假但仍为命题 |
| x + 3 > 5 | 不是 | 含自由变元，真假取决于x的值（开语句） |
| 今天天气真好啊！ | 不是 | 感叹句 |
| 请把门关上 | 不是 | 祈使句 |

===== 2.1 否定 =====

## 2.1 否定 ¬P（Negation）


**定义：**若P为命题，则 ¬P（读作"非P"）也是一个命题。当P为真时，¬P为假；当P为假时，¬P为真。


**符号：**`¬P` 或 `~P` 或 `!P`


**示例：**P = "今天下雨"，则 ¬P = "今天不下雨"


| P | ¬P |
| --- | --- |
| T | F |
| F | T |

===== 2.2 合取 =====

## 2.2 合取 P ∧ Q（Conjunction / AND）


**定义：**P ∧ Q（读作"P与Q"）为真当且仅当P和Q同时为真。


**记忆口诀：**"有假即假，全真才真"


**示例：**P = "2是偶数"，Q = "3是奇数"，则 P ∧ Q = "2是偶数且3是奇数" 为真


| P | Q | P ∧ Q |
| --- | --- | --- |
| T | T | T |
| T | F | F |
| F | T | F |
| F | F | F |

===== 2.3 析取 =====

## 2.3 析取 P ∨ Q（Disjunction / OR）


**定义：**P ∨ Q（读作"P或Q"）为假当且仅当P和Q同时为假。这里的"或"是**相容或（inclusive or）**。


**记忆口诀：**"有真即真，全假才假"


**示例：**"我去图书馆或我去食堂" — 两者都去也为真


| P | Q | P ∨ Q |
| --- | --- | --- |
| T | T | T |
| T | F | T |
| F | T | T |
| F | F | F |

===== 2.4 蕴含 =====

## 2.4 蕴含 P → Q（Implication / Conditional）


> **Warning:** **定义：**P → Q（读作"若P则Q"）为假当且仅当P为真而Q为假。
>
>
> **关键理解：**
>
>
> - P称为
>    **前件**
>    ，Q称为
>    **后件**
> - 当前件为假时，整个蕴含式
>    **恒为真**
>    （"空真" — 空前提的蕴含视为真）
> - "如果太阳从西边出来，那么我请你吃饭" — 前件为假，整句为真
>
>
> **记忆口诀：**"前真后假才假，其余皆真"


| P | Q | P → Q | 说明 |
| --- | --- | --- | --- |
| T | T | T | 前真后真，显然成立 |
| T | F | F | 前真后假，**唯一为假**的情况 |
| F | T | T | 前件为假，空真 |
| F | F | T | 前件为假，空真 |


### 蕴含的等价形式


`P → Q ≡ ¬P ∨ Q`


这是将蕴含转化为析取的重要等价公式，在逻辑化简中经常使用。

===== 2.5 等价 =====

## 2.5 等价 P ↔ Q（Biconditional）


**定义：**P ↔ Q（读作"P当且仅当Q"）为真当且仅当P和Q的真值相同。


**等价表达：**`P ↔ Q ≡ (P → Q) ∧ (Q → P)`


**示例：**"三角形是等边三角形当且仅当三边相等"


| P | Q | P ↔ Q |
| --- | --- | --- |
| T | T | T |
| T | F | F |
| F | T | F |
| F | F | T |

===== 2.6 异或 =====

## 2.6 异或 P ⊕ Q（Exclusive OR / XOR）


**定义：**P ⊕ Q 为真当且仅当P和Q的真值不同。


**等价表达：**`P ⊕ Q ≡ (P ∧ ¬Q) ∨ (¬P ∧ Q)`


**与析取的关系：**`P ⊕ Q ≡ (P ∨ Q) ∧ ¬(P ∧ Q)`


| P | Q | P ⊕ Q |
| --- | --- | --- |
| T | T | F |
| T | F | T |
| F | T | T |
| F | F | F |

==================== 5. 优先级 ====================

## 5. 联结词优先级


### 从高到低的优先级顺序


| 优先级 | 联结词 | 说明 |
| --- | --- | --- |
| 1（最高） | ¬ | 否定 |
| 2 | ∧ | 合取 |
| 3 | ∨ | 析取 |
| 4 | → | 蕴含 |
| 5（最低） | ↔ | 等价 |


**示例：**`¬P ∨ Q ∧ R → S` 等价于 `((¬P) ∨ (Q ∧ R)) → S`

==================== 6. 常用等价公式 ====================

## 6. 常用等价公式速查


| 名称 | 公式 |
| --- | --- |
| 双重否定律 | ¬¬P ≡ P |
| 交换律 | P ∧ Q ≡ Q ∧ P  ;  P ∨ Q ≡ Q ∨ P |
| 结合律 | (P∧Q)∧R ≡ P∧(Q∧R) |
| 分配律 | P∧(Q∨R) ≡ (P∧Q)∨(P∧R) |
| 德摩根律 | ¬(P∧Q) ≡ ¬P ∨ ¬Q  ;  ¬(P∨Q) ≡ ¬P ∧ ¬Q |
| 蕴含等价 | P→Q ≡ ¬P ∨ Q |
| 逆否命题 | P→Q ≡ ¬Q→¬P |
| 吸收律 | P∧(P∨Q) ≡ P  ;  P∨(P∧Q) ≡ P |

==========================================
    总结：
    1. 命题是能判断真假的陈述句，有且仅有真(T)和假(F)两个取值
    2. 五种基本联结词：
       - 否定 ¬P：真值取反
       - 合取 P∧Q：有假即假（AND）
       - 析取 P∨Q：有真即真（OR，相容或）
       - 蕴含 P→Q：前真后假才假，其余皆真
       - 等价 P↔Q：同真值为真
    3. 联结词优先级：¬ > ∧ > ∨ > → > ↔
    4. 重要等价：P→Q ≡ ¬P∨Q，逆否命题 P→Q ≡ ¬Q→¬P
    5. 德摩根律在逻辑化简中极为重要

## Python实现

### 真值表生成与逻辑联结词计算

```python
import itertools

def truth_table(variables, expression_func, expr_name="Expr"):
    """生成逻辑表达式的真值表"""
    header = variables + [expr_name]
    print(" | ".join(f"{h:>6}" for h in header))
    print("-" * (8 * len(header)))

    for values in itertools.product([True, False], repeat=len(variables)):
        env = dict(zip(variables, values))
        result = expression_func(env)
        row = list(values) + [result]
        print(" | ".join(f"{str(v):>6}" for v in row))

# 定义联结词
NOT = lambda p: not p
AND = lambda p, q: p and q
OR = lambda p, q: p or q
IMPLIES = lambda p, q: (not p) or q
IFF = lambda p, q: p == q
XOR = lambda p, q: p != q

# 生成完整真值表
print("=== 完整真值表 ===")
print(f"{'P':>6} | {'Q':>6} | {'¬P':>6} | {'P∧Q':>6} | {'P∨Q':>6} | {'P→Q':>6} | {'P↔Q':>6} | {'P⊕Q':>6}")
print("-" * 70)
for P, Q in itertools.product([True, False], repeat=2):
    print(f"{str(P):>6} | {str(Q):>6} | {str(NOT(P)):>6} | {str(AND(P,Q)):>6} | "
          f"{str(OR(P,Q)):>6} | {str(IMPLIES(P,Q)):>6} | {str(IFF(P,Q)):>6} | {str(XOR(P,Q)):>6}")

# 验证等价公式: P→Q ≡ ¬P∨Q
print("\n=== 验证 P→Q ≡ ¬P∨Q ===")
for P, Q in itertools.product([True, False], repeat=2):
    left = IMPLIES(P, Q)
    right = OR(NOT(P), Q)
    assert left == right, f"不等价! P={P}, Q={Q}"
print("所有情况等价 ✓")

# 验证逆否命题: P→Q ≡ ¬Q→¬P
print("\n=== 验证逆否命题 ===")
for P, Q in itertools.product([True, False], repeat=2):
    assert IMPLIES(P, Q) == IMPLIES(NOT(Q), NOT(P))
print("逆否命题等价 ✓")

# 验证德摩根律
print("\n=== 验证德摩根律 ===")
for P, Q in itertools.product([True, False], repeat=2):
    assert NOT(AND(P, Q)) == OR(NOT(P), NOT(Q))
    assert NOT(OR(P, Q)) == AND(NOT(P), NOT(Q))
print("德摩根律验证通过 ✓")
```

### 逻辑表达式自动求值

```python
import itertools

def eval_logic_expr(expr, env):
    """解析并求值逻辑表达式（支持 and, or, not, implies, iff）"""
    # 简单实现：用 Python 运算符
    expr = expr.replace("¬", " not ").replace("∧", " and ")
    expr = expr.replace("∨", " or ").replace("→", " <= ")  # P→Q 等价于 (not P) or Q
    return eval(expr, {"__builtins__": {}}, env)

# 三段论验证
print("=== 三段论验证 ===")
# 前提1: P→Q (所有人会死)
# 前提2: P (苏格拉底是人)
# 结论: Q (苏格拉底会死)
for P, Q in itertools.product([True, False], repeat=2):
    premise1 = IMPLIES(P, Q)  # P→Q
    premise2 = P               # P
    conclusion = Q              # Q
    if premise1 and premise2:
        print(f"P={P}, Q={Q}: 前提成立时, 结论 Q={conclusion}")
        assert conclusion == True, "三段论无效!"
print("三段论验证通过 ✓")

# 复杂逻辑表达式求值
print("\n=== 复杂表达式求值 ===")
# ¬P ∨ Q ∧ R → S，按优先级等价于 ((¬P) ∨ (Q ∧ R)) → S
for vals in itertools.product([True, False], repeat=4):
    P, Q, R, S = vals
    result = IMPLIES(OR(NOT(P), AND(Q, R)), S)
    # 打印使表达式为 False 的情况
    if not result:
        print(f"反例: P={P}, Q={Q}, R={R}, S={S} → 表达式为 False")
```
