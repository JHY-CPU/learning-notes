02 - 谓词逻辑与量词

离散数学 | 命题逻辑与谓词逻辑

1. 为什么需要谓词逻辑？

命题逻辑的局限

命题逻辑将简单命题作为最小单位，无法分析命题的内部结构。例如：

前提："所有人都是会死的" 和 "苏格拉底是人"

结论："苏格拉底是会死的"

这个经典的三段论推理，在命题逻辑中无法表达和证明，因为每个前提都是独立命题，无法建立它们之间的逻辑联系。

谓词逻辑通过引入谓词、个体和量词解决了这一问题。

2. 谓词与个体

基本概念

个体（Individual）：所研究对象中的具体事物或抽象概念，也称客体。通常用小写字母表示，如 a, b, c, x, y。

谓词（Predicate）：表示个体性质或个体之间关系的词。通常用大写字母表示，如 P, Q, R。

表达式含义类型
P(x): "x 是素数"一元谓词（性质）P(7) = T, P(6) = F
L(x, y): "x 大于 y"二元谓词（关系）L(5, 3) = T, L(3, 5) = F
B(x, y, z): "x 在 y 和 z 之间"三元谓词（关系）B(2, 1, 3) = T
命题函数：P(x) 本身不是命题，只有对 x 指定具体值或用量词约束后才成为命题

3. 量词（Quantifiers）

3.1 全称量词 ∀（Universal Quantifier）

定义

∀x P(x) 读作"对所有x，P(x)成立"或"任意x，P(x)"

含义：在给定论域中，对每一个个体x，P(x)都为真。

真值规则：

  • ∀x P(x) 为真 ↔ 论域中每个个体都使 P(x) 为真
  • ∀x P(x) 为假 ↔ 论域中至少存在一个个体使 P(x) 为假

否定：¬∀x P(x) ≡ ∃x ¬P(x)

全称量词示例

设论域 D = {2, 4, 6}，P(x): "x 是偶数"

∀x P(x)：P(2) ∧ P(4) ∧ P(6) = T ∧ T ∧ T = True

设论域 D = {2, 3, 4}，P(x): "x 是偶数"

∀x P(x)：P(2) ∧ P(3) ∧ P(4) = T ∧ F ∧ T = False

3.2 存在量词 ∃（Existential Quantifier）

定义

∃x P(x) 读作"存在x使得P(x)成立"或"至少有一个x，P(x)"

含义：在给定论域中，至少存在一个个体x使得P(x)为真。

真值规则：

  • ∃x P(x) 为真 ↔ 论域中至少有一个个体使 P(x) 为真
  • ∃x P(x) 为假 ↔ 论域中所有个体都使 P(x) 为假

否定：¬∃x P(x) ≡ ∀x ¬P(x)

存在量词示例

设论域 D = {1, 2, 3, 4}，P(x): "x 是素数"

∃x P(x)：P(1) ∨ P(2) ∨ P(3) ∨ P(4) = F ∨ T ∨ T ∨ F = True

设论域 D = {4, 6, 8}，P(x): "x 是素数"

∃x P(x)：P(4) ∨ P(6) ∨ P(8) = F ∨ F ∨ F = False

4. 论域（Domain of Discourse）

论域的重要性

论域是量词所约束的变量的取值范围。同一个谓词公式在不同论域下可能有不同的真值。

谓词公式论域真值
∀x (x² ≥ 0)实数 RTrue
∀x (x² ≥ 0)虚数 iFalse
∃x (x + 1 = 0)自然数 NFalse
∃x (x + 1 = 0)整数 ZTrue

5. 量词的辖域与变元

辖域（Scope）

量词的辖域是该量词所作用的范围。在公式 ∀x P(x) 或 ∃x P(x) 中，P(x) 就是量词的辖域。

  • 约束变元：在量词辖域内、被该量词约束的变元
  • 自由变元：不受任何量词约束的变元

示例分析

∀x (P(x) ∧ ∃y Q(x, y))

  • ∀x 的辖域：(P(x) ∧ ∃y Q(x, y))
  • ∃y 的辖域：Q(x, y)
  • x 是约束变元（被 ∀x 约束）
  • y 是约束变元（被 ∃y 约束）

自由变元示例

P(x) ∧ ∀y Q(y)

  • ∀y 的辖域：Q(y)
  • x 是自由变元（没有量词约束）
  • y 是约束变元
  • 含自由变元的表达式不是命题

换名规则

当辖域重叠时，可以通过换名规则避免混淆：

∀x P(x) ∧ ∃x Q(x) 中两个x互不相关

可以改名为：∀x P(x) ∧ ∃y Q(y)

约束变元可以改名，但不能与辖域内其他变元冲突。

6. 量词的否定（德摩根推广）

量词否定等价公式

否定形式等价形式含义
¬∀x P(x)∃x ¬P(x)"并非所有x都满足P" ≡ "存在x不满足P"
¬∃x P(x)∀x ¬P(x)"不存在x满足P" ≡ "所有x都不满足P"

记忆方法：否定量词时，∀ 变 ∃，∃ 变 ∀，同时否定后面的谓词。

实例："不是所有人都喜欢数学" ≡ "存在有人不喜欢数学"
¬∀x Likes(x, Math) ≡ ∃x ¬Likes(x, Math)

7. 多重量词与嵌套量词

量词顺序的重要性

当多个量词同时出现时，顺序非常重要，不同顺序可能表示不同含义。

公式含义真值（论域为整数）
∀x∃y (x + y = 0)对每个x，存在y使x+y=0True（y = -x）
∃y∀x (x + y = 0)存在y，使得所有x都满足x+y=0False
∀x∀y (x + y = y + x)对所有x和y，加法交换律True
∃x∃y (x + y = 0)存在x和y使x+y=0True（x=1,y=-1）

关键规则：相同类型的相邻量词可以交换顺序（∀∀ 或 ∃∃），但不同类型的量词一般不能交换顺序。

9. 前束范式（PNF）简介

前束范式定义

一个谓词公式如果形如 Q₁x₁ Q₂x₂ ... Qₙxₙ M，其中 Q 为量词（∀或∃），M 为不含量词的公式，则称为前束范式。

示例：

  • ∀x∃y (P(x,y) → Q(y)) — 前束范式
  • ∀x P(x) ∧ ∃y Q(y) — 前束范式
  • (∀x P(x)) → Q — 不是前束范式（量词在蕴含式左边）

任何谓词公式都可以通过等价变换化为前束范式。
