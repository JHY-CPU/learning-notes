# 递推关系


离散数学 · 组合数学 · 第4节

==================== 1. 递推关系概述 ====================

## 1. 递推关系概述


### 1.1 定义


> **Note:** **递推关系 (Recurrence Relation)：**
> 将序列的第 n 项表示为前面若干项的函数的等式。
>
>
> 递推关系 +
> **初始条件**
> = 完整的序列定义。


| 名称 | 递推关系 | 初始条件 | 解 |
| --- | --- | --- | --- |
| Fibonacci | Fₙ = Fₙ₋₁ + Fₙ₋₂ | F₀=0, F₁=1 | (φⁿ-ψⁿ)/√5 |
| 阶乘 | n! = n·(n-1)! | 0!=1 | n! |
| Hanoi塔 | Tₙ = 2Tₙ₋₁ + 1 | T₁=1 | 2ⁿ-1 |
| 错排 | Dₙ = (n-1)(Dₙ₋₁+Dₙ₋₂) | D₁=0, D₂=1 | n!Σ(-1)ᵏ/k! |
| Catalan | Cₙ = Σ CₖCₙ₋₁₋ₖ | C₀=1 | C(2n,n)/(n+1) |
| 二分查找 | T(n) = T(n/2)+1 | T(1)=1 | ⌊log₂n⌋+1 |

==================== 2. 线性常系数递推 ====================

## 2. 线性常系数齐次递推


### 2.1 标准形式


$$
aₙ + c₁aₙ₋₁ + c₂aₙ₋₂ + ... + cₖaₙ₋ₖ = 0
$$


其中 c₁, c₂, ..., cₖ 为常数，k 为递推的阶数。


### 2.2 特征方程法


> **Note:** **求解步骤：**
>
> 1. 写出
>    **特征方程**
>    ：rᵏ + c₁rᵏ⁻¹ + ... + cₖ = 0
> 2. 求特征根 r₁, r₂, ..., rₖ
> 3. 根据根的类型写出通解
> 4. 利用初始条件确定系数


### 2.3 根的情况与通解形式


| 特征根情况 | 通解形式 |
| --- | --- |
| k 个不同实根 r₁,...,rₖ | aₙ = A₁r₁ⁿ + A₂r₂ⁿ + ... + Aₖrₖⁿ |
| r 是 m 重根 | 对应项为 (A₁ + A₂n + ... + Aₘnᵐ⁻¹)rⁿ |
| 共轭复根 α±βi | 对应项为 ρⁿ(A cos nθ + B sin nθ)，其中 ρ=\|α+βi\|, θ=arg(α+βi) |


> **Example:** **例：Fibonacci 数列**
> Fₙ = Fₙ₋₁ + Fₙ₋₂，F₀=0, F₁=1。
>
>
> 特征方程：r² = r + 1，即 r² - r - 1 = 0
>
>
> 特征根：r₁ = (1+√5)/2 = φ，r₂ = (1-√5)/2 = ψ
>
>
> 通解：Fₙ = A·φⁿ + B·ψⁿ
>
>
> 代入初始条件：F₀ = A+B = 0，F₁ = Aφ+Bψ = 1
>
>
> 解得：A = 1/√5，B = -1/√5
>
>
>
> $$
> Fₙ = (φⁿ - ψⁿ)/√5    (Binet 公式)
> $$


> **Example:** **例：Hanoi塔**
> Tₙ = 2Tₙ₋₁ + 1，T₁=1。
>
>
> 这是非齐次递推。先求齐次解：r-2=0 → r=2，Tₙ⁽ʰ⁾ = A·2ⁿ
>
>
> 特解设为常数 C：C = 2C + 1 → C = -1
>
>
> 通解：Tₙ = A·2ⁿ - 1
>
>
> T₁ = 2A - 1 = 1 → A = 1
>
>
>
> $$
> Tₙ = 2ⁿ - 1
> $$

==================== 3. 非齐次递推 ====================

## 3. 线性常系数非齐次递推


$$
aₙ + c₁aₙ₋₁ + ... + cₖaₙ₋ₖ = f(n)
$$


> **Note:** **求解方法：**
> 通解 = 齐次通解 + 非齐次特解。
>
>
> 特解的猜测形式取决于 f(n) 的类型：


| f(n) 的形式 | 特解猜测 | 特殊情况 |
| --- | --- | --- |
| 常数 d | 常数 C | 若 r=1 是特征根，乘以 n |
| d·n | C₁n+C₀ | 若 r=1 是 m 重根，乘以 nᵐ |
| d·nᵖ | Cₚnᵖ+...+C₁n+C₀ | 同上 |
| d·sⁿ | C·sⁿ | 若 s 是 m 重特征根，乘以 nᵐ |
| d·sⁿ·nᵖ | (Cₚnᵖ+...+C₀)sⁿ | 同上 |

==================== 4. Fibonacci 的推广 ====================

## 4. Fibonacci 数列的推广


### 4.1 广义 Fibonacci (k-步)


$$
Fₙ⁽ᵏ⁾ = Fₙ₋₁⁽ᵏ⁾ + Fₙ₋₂⁽ᵏ⁾ + ... + Fₙ₋ₖ⁽ᵏ⁾
$$


Tribonacci (k=3)：Tₙ = Tₙ₋₁ + Tₙ₋₂ + Tₙ₋₃


### 4.2 Fibonacci 的重要性质


| 性质 | 公式 |
| --- | --- |
| Cassini 恒等式 | Fₙ₊₁·Fₙ₋₁ - Fₙ² = (-1)ⁿ |
| 求和 | Σ(k=0→n) Fₖ = Fₙ₊₂ - 1 |
| 平方和 | Σ(k=1→n) Fₖ² = Fₙ·Fₙ₊₁ |
| 相邻比值 | lim Fₙ₊₁/Fₙ = φ (黄金比例) |
| 与组合数 | Fₙ = Σ C(n-1-k, k) (k从0到⌊(n-1)/2⌋) |
| GCD | gcd(Fₘ, Fₙ) = F_(gcd(m,n)) |

==================== 5. Master 定理 ====================

## 5. Master 定理 (主定理)


用于分析分治算法的递推关系：


$$
T(n) = a·T(n/b) + f(n)
$$


其中 a ≥ 1, b > 1 为常数，f(n) 为渐进正函数。


> **Theorem:** **Master 定理：**
> 设 f(n) = Θ(nᶜ · logᵏn)，比较 c 与 log_b(a)：
>
>
>
>
> **情况1：**
> 若 c < log_b(a)，则 T(n) =
> **Θ(n^(log_b(a)))**
>
>
> **情况2：**
> 若 c = log_b(a)，则 T(n) =
> **Θ(nᶜ · log^(k+1)n)**
>
>
> 特别地 k=0 时 T(n) = Θ(nᶜ · log n)
>
>
> **情况3：**
> 若 c > log_b(a)，则 T(n) =
> **Θ(nᶜ)**


> **Example:** **例1：二分查找**
> T(n) = T(n/2) + 1
>
>
> a=1, b=2, f(n)=1=Θ(n⁰)。c=0, log₂1=0。
>
>
> 属于情况2 (c=log_b(a), k=0)：
> **T(n) = Θ(log n)**
> ✓


> **Example:** **例2：归并排序**
> T(n) = 2T(n/2) + n
>
>
> a=2, b=2, f(n)=n=Θ(n¹)。c=1, log₂2=1。
>
>
> 属于情况2 (c=log_b(a), k=0)：
> **T(n) = Θ(n log n)**
> ✓


> **Example:** **例3：Strassen 矩阵乘法**
> T(n) = 7T(n/2) + Θ(n²)
>
>
> a=7, b=2, f(n)=Θ(n²)。c=2, log₂7 ≈ 2.807。
>
>
> 属于情况1 (c < log_b(a))：
> **T(n) = Θ(n^(log₂7)) ≈ Θ(n^2.807)**
> ✓

==================== 6. 其他解法 ====================

## 6. 其他求解方法


### 6.1 迭代法 (展开法)


> **Example:** T(n) = 2T(n-1) + 1, T(0)=0
>
>
> T(n) = 2[2T(n-2)+1]+1 = 2²T(n-2)+2+1
>
>
> = 2³T(n-3)+4+2+1
>
>
> = ...
>
>
> = 2ⁿT(0) + 2ⁿ⁻¹+2ⁿ⁻²+...+1
>
>
> = 0 + 2ⁿ-1 =
> **2ⁿ-1**


### 6.2 换元法


> **Note:** 令 n = 2ᵐ（或 bᵐ），将 T(n) 转化为关于 m 的递推。
>
>
> 例：T(n) = 2T(√n) + log n，令 n=2ᵐ 得 S(m) = 2S(m/2)+m，用 Master 定理解。


### 6.3 生成函数法


> **Note:** 设 G(x)=Σ aₙxⁿ，将递推关系转化为 G(x) 的函数方程，求解后提取系数。
>
>
> 详见"生成函数"一节。
