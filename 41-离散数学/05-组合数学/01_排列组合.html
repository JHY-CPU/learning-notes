01 - 排列组合

离散数学 | 组合数学

1. 基本计数原理

1.1 加法原理（分类计数）

做一件事有 k 类方法，第 i 类有 ni 种方法，则总共有 n₁ + n₂ + ... + nₖ 种方法。

N = n₁ + n₂ + ... + nₖ（分类 — "或"的关系）

示例：从北京到上海可以坐飞机(3班)或高铁(5班)，共 3+5 = 8 种选择。

1.2 乘法原理（分步计数）

做一件事需要 k 个步骤，第 i 步有 ni 种方法，则总共有 n₁ × n₂ × ... × nₖ 种方法。

N = n₁ × n₂ × ... × nₖ（分步 — "且"的关系）

示例：选一件上衣(4种)和一条裤子(3种)搭配，共 4×3 = 12 种搭配。

2. 排列（Permutation）

定义

从 n 个不同元素中取出 r 个（0 ≤ r ≤ n），按一定顺序排成一列，称为从 n 中取 r 的排列。

P(n, r) = A(n, r) = n! / (n-r)! = n × (n-1) × ... × (n-r+1)

关键：顺序重要 — (A,B) 和 (B,A) 是不同的排列。

推导

第1个位置有 n 种选择，第2个位置有 n-1 种选择，...，第 r 个位置有 n-r+1 种选择。

由乘法原理：P(n,r) = n(n-1)...(n-r+1) = n!/(n-r)!

特殊排列

类型公式说明
全排列P(n, n) = n!n个元素的全部排列方式
圆排列P(n, n) / n = (n-1)!首尾相接的环形排列
含重复的排列n! / (n₁! n₂! ... nₖ!)n₁个相同，n₂个相同，...

3. 组合（Combination）

定义

从 n 个不同元素中取出 r 个（0 ≤ r ≤ n），不考虑顺序组成一组，称为从 n 中取 r 的组合。

C(n, r) = n! / (r! × (n-r)!) = P(n, r) / r!

关键：顺序不重要 — {A,B} 和 {B,A} 是同一个组合。

关系：C(n,r) = P(n,r) / r!（排列数除以 r! 去除顺序因素）

组合的重要性质

性质公式说明
对称性C(n, r) = C(n, n-r)选r个等价于不选n-r个
递推（Pascal恒等式）C(n, r) = C(n-1, r-1) + C(n-1, r)包含某元素 + 不包含
边界C(n, 0) = C(n, n) = 1选0个或选n个各只有1种
求和Σ C(n, r) = 2n所有组合数之和等于幂集大小

4. 排列与组合对比

特性排列 P(n,r)组合 C(n,r)
顺序重要不重要
公式n!/(n-r)!n!/(r!(n-r)!)
关系P(n,r) = C(n,r) × r!C(n,r) = P(n,r) / r!
示例选班长、副班长选两个代表
大小关系P(n,r) ≥ C(n,r)等号当 r=0 或 r=n 时成立

4.1 经典例题

例1：5个人站一排照相有多少种排法？

全排列：P(5,5) = 5! = 120 种

例2：5个人选3个人当班长、副班长、学习委员？

排列（有顺序）：P(5,3) = 5!/(5-3)! = 5×4×3 = 60 种

例3：5个人选3个人参加比赛？

组合（无顺序）：C(5,3) = 5!/(3!×2!) = 10 种

5. Pascal 三角形（杨辉三角）

Pascal 三角形的第 n 行、第 r 个数 = C(n, r)。每行两端为1，每个数等于上一行相邻两数之和。

6. 二项式定理（Binomial Theorem）

(a + b)n = Σk=0n C(n, k) · an-k · bk

展开式中 an-kbk 的系数恰好是 C(n, k)，对应 Pascal 三角形第 n 行的第 k 个数。

示例：(a+b)³ = C(3,0)a³ + C(3,1)a²b + C(3,2)ab² + C(3,3)b³

= a³ + 3a²b + 3ab² + b³
