## 01 - 排列组合


离散数学 | 组合数学

==================== 1. 基本计数原理 ====================

## 1. 基本计数原理


## 1.1 加法原理（分类计数）


做一件事有 k 类方法，第 i 类有 n~i~ 种方法，则总共有 **n₁ + n₂ + ... + nₖ** 种方法。

N = n₁ + n₂ + ... + nₖ（分类 — "或"的关系）

**示例：**从北京到上海可以坐飞机(3班)或高铁(5班)，共 3+5 = 8 种选择。


## 1.2 乘法原理（分步计数）


做一件事需要 k 个步骤，第 i 步有 n~i~ 种方法，则总共有 **n₁ × n₂ × ... × nₖ** 种方法。

N = n₁ × n₂ × ... × nₖ（分步 — "且"的关系）

**示例：**选一件上衣(4种)和一条裤子(3种)搭配，共 4×3 = 12 种搭配。

==================== 2. 排列 ====================

## 2. 排列（Permutation）


### 定义


从 n 个**不同**元素中取出 r 个（0 ≤ r ≤ n），**按一定顺序**排成一列，称为从 n 中取 r 的**排列**。

P(n, r) = A(n, r) = n! / (n-r)! = n × (n-1) × ... × (n-r+1)

**关键：**顺序重要 — (A,B) 和 (B,A) 是不同的排列。


### 推导


第1个位置有 n 种选择，第2个位置有 n-1 种选择，...，第 r 个位置有 n-r+1 种选择。


由乘法原理：P(n,r) = n(n-1)...(n-r+1) = n!/(n-r)!


> **Warning:** ### 特殊排列
>
>
> | 类型 | 公式 | 说明 |
> | --- | --- | --- |
> | 全排列 | P(n, n) = n! | n个元素的全部排列方式 |
> | 圆排列 | P(n, n) / n = (n-1)! | 首尾相接的环形排列 |
> | 含重复的排列 | n! / (n₁! n₂! ... nₖ!) | n₁个相同，n₂个相同，... |

==================== 3. 组合 ====================

## 3. 组合（Combination）


### 定义


从 n 个**不同**元素中取出 r 个（0 ≤ r ≤ n），**不考虑顺序**组成一组，称为从 n 中取 r 的**组合**。

C(n, r) = n! / (r! × (n-r)!) = P(n, r) / r!

**关键：**顺序不重要 — {A,B} 和 {B,A} 是同一个组合。


**关系：**C(n,r) = P(n,r) / r!（排列数除以 r! 去除顺序因素）


### 组合的重要性质


| 性质 | 公式 | 说明 |
| --- | --- | --- |
| 对称性 | C(n, r) = C(n, n-r) | 选r个等价于不选n-r个 |
| 递推（Pascal恒等式） | C(n, r) = C(n-1, r-1) + C(n-1, r) | 包含某元素 + 不包含 |
| 边界 | C(n, 0) = C(n, n) = 1 | 选0个或选n个各只有1种 |
| 求和 | Σ C(n, r) = 2^n^ | 所有组合数之和等于幂集大小 |

==================== 4. 排列 vs 组合对比 ====================

## 4. 排列与组合对比


| 特性 | 排列 P(n,r) | 组合 C(n,r) |
| --- | --- | --- |
| 顺序 | 重要 | 不重要 |
| 公式 | n!/(n-r)! | n!/(r!(n-r)!) |
| 关系 | P(n,r) = C(n,r) × r! | C(n,r) = P(n,r) / r! |
| 示例 | 选班长、副班长 | 选两个代表 |
| 大小关系 | P(n,r) ≥ C(n,r) | 等号当 r=0 或 r=n 时成立 |


## 4.1 经典例题


### 例1：5个人站一排照相有多少种排法？


**全排列：**P(5,5) = 5! = 120 种


### 例2：5个人选3个人当班长、副班长、学习委员？


**排列（有顺序）：**P(5,3) = 5!/(5-3)! = 5×4×3 = 60 种


### 例3：5个人选3个人参加比赛？


**组合（无顺序）：**C(5,3) = 5!/(3!×2!) = 10 种

==================== 5. Pascal 三角形 ====================

## 5. Pascal 三角形（杨辉三角）


Pascal 三角形的第 n 行、第 r 个数 = C(n, r)。每行两端为1，每个数等于上一行相邻两数之和。

## Python实现

### 排列组合计算与 Pascal 三角形

```python
from math import factorial, comb, perm
from itertools import combinations, permutations

# 基本排列组合
print("=== 排列与组合 ===")
n, r = 5, 3
print(f"P({n},{r}) = {n}!/{n-r}! = {perm(n, r)}")
print(f"C({n},{r}) = {n}!/({r}!×{n-r}!) = {comb(n, r)}")

# 验证: C(n,r) = C(n, n-r)
print(f"\n对称性: C({n},{r}) = C({n},{n-r}): {comb(n, r) == comb(n, n-r)}")

# Pascal 恒等式: C(n,r) = C(n-1,r-1) + C(n-1,r)
n, r = 6, 3
lhs = comb(n, r)
rhs = comb(n - 1, r - 1) + comb(n - 1, r)
print(f"Pascal恒等式: C({n},{r}) = C({n-1},{r-1}) + C({n-1},{r})")
print(f"  {lhs} = {comb(n-1, r-1)} + {comb(n-1, r)} = {rhs}, 相等: {lhs == rhs}")

# Pascal 三角形
print("\n=== Pascal 三角形 (杨辉三角) ===")
for row in range(8):
    vals = [comb(row, k) for k in range(row + 1)]
    print("  " * (7 - row) + "  ".join(f"{v:4}" for v in vals))

# 二项式定理验证
print("\n=== 二项式定理验证 ===")
a, b, n = 2, 3, 4
# (a+b)^n
direct = (a + b) ** n
# Σ C(n,k) a^(n-k) b^k
expanded = sum(comb(n, k) * a ** (n - k) * b ** k for k in range(n + 1))
print(f"(a+b)^n = ({a}+{b})^{n} = {direct}")
print(f"Σ C({n},k) a^({n}-k) b^k = {expanded}")
print(f"相等: {direct == expanded}")

# 枚举排列和组合
print("\n=== 枚举: 5人中选3人 ===")
people = ['A', 'B', 'C', 'D', 'E']
print(f"排列 P(5,3) (有序): {len(list(permutations(people, 3)))} 种")
for p in permutations(people, 3):
    print(f"  {''.join(p)}", end="  ")
print()
print(f"\n组合 C(5,3) (无序): {len(list(combinations(people, 3)))} 种")
for c in combinations(people, 3):
    print(f"  {''.join(c)}", end="  ")

# 特殊排列
print("\n\n\n=== 特殊排列 ===")
print(f"全排列 5!: {factorial(5)}")
print(f"圆排列 (5-1)!: {factorial(5 - 1)} (5人围圆桌)")
print(f"含重复 AABBC: {factorial(5) // (factorial(2) * factorial(2) * factorial(1))}")

# 函数计数
print("\n=== 有限集函数计数 ===")
n, m = 3, 4
print(f"|A|={n}, |B|={m}")
print(f"所有函数: {m}^{n} = {m**n}")
print(f"单射: P({m},{n}) = {perm(m, n)}")
```
