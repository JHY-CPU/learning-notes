生成函数

离散数学 · 组合数学 · 第3节

1. 普通生成函数 (OGF)

1.1 定义

序列 {a₀, a₁, a₂, ...} 的普通生成函数 (Ordinary Generating Function) 定义为：

G(x) = a₀ + a₁x + a₂x² + a₃x³ + ... = Σ(n=0→∞) aₙxⁿ
核心思想：将序列编码为多项式/幂级数的系数，通过多项式运算（加法、乘法、复合等）来处理计数问题。
x 是形式变量，本身没有数值意义；我们只关心系数。

1.2 基本生成函数

序列 {aₙ}OGF G(x)收敛域
1, 1, 1, 1, ...1/(1-x)|x| < 1
1, a, a², a³, ...1/(1-ax)|x| < 1/|a|
0, 1, 1, 1, ...x/(1-x)|x| < 1
C(n,k) 固定 k(1+x)ⁿ (有限项)多项式
C(n+k-1, k)1/(1-x)ⁿ|x| < 1
1, 2, 3, 4, ...1/(1-x)²|x| < 1
1, n, n², ... (固定 n)(1+x)ⁿ多项式

1.3 运算规则

运算序列层面生成函数层面
加法{aₙ} + {bₙ} = {aₙ+bₙ}A(x) + B(x)
标量乘法c·{aₙ}c·A(x)
移位{0, a₀, a₁, ...}x·A(x)
卷积cₙ = Σ aₖbₙ₋ₖA(x)·B(x)
前缀和{Σ(k=0→n) aₖ}A(x)/(1-x)

2. 生成函数的应用

2.1 组合计数

例1：整数拆分
将 n 拆分为若干正整数之和的方法数（不考虑顺序）：
Π(k=1→∞) 1/(1-x^k)
因式 1/(1-x^k) = 1 + x^k + x^(2k) + ... 表示整数 k 可以出现 0, 1, 2, ... 次。
例2：多重集组合
从 {2个a, 3个b, 1个c} 中选 4 个元素的方案数：
生成函数 = (1+x+x²)(1+x+x²+x³)(1+x)
展开后 x⁴ 的系数 = 3
方案：{a,a,b,b}, {a,a,b,c}, {a,b,b,c}（还需验证实际展开）

2.2 硬币找零问题

用 1元、2元、5元硬币凑出 10 元的方案数：
生成函数 = 1/((1-x)(1-x²)(1-x⁵))
展开求 x¹⁰ 的系数。实际计算得 10 种方案。

2.3 二项式定理的推广

负二项式定理：
(1+x)^α = Σ(n=0→∞) C(α,n) xⁿ
其中 C(α,n) = α(α-1)···(α-n+1)/n! （广义二项式系数）。

特别地，取 α = -m（m 为正整数）：
1/(1-x)^m = Σ(n=0→∞) C(m+n-1, n) xⁿ

3. 指数生成函数 (EGF)

3.1 定义

序列 {a₀, a₁, a₂, ...} 的指数生成函数 (Exponential Generating Function)：

E(x) = a₀ + a₁x/1! + a₂x²/2! + a₃x³/3! + ... = Σ(n=0→∞) aₙ · xⁿ/n!

3.2 何时使用 EGF

OGF：用于组合问题（无序选取）。
EGF：用于排列问题（有序排列），或者涉及标号对象的计数。

卷积公式不同：
OGF 卷积：cₙ = Σ aₖbₙ₋ₖ（普通卷积）
EGF 卷积：cₙ = Σ C(n,k) aₖbₙ₋ₖ（二项式卷积）

3.3 基本 EGF

序列EGF
1, 1, 1, 1, ...eˣ = Σ xⁿ/n!
1, 0, 1, 0, ... (奇数项0)(eˣ + e⁻ˣ)/2 = cosh(x)
0, 1, 0, 1, ... (偶数项0)(eˣ - e⁻ˣ)/2 = sinh(x)
1, k, k², k³, ... (固定 k)e^(kx)
排列数 n!1/(1-x)
错排数 Dₙe⁻ˣ/(1-x)
Bell 数 Bₙe^(eˣ-1)
Stirling 第二类 S(n,k) 固定 k(eˣ-1)^k / k!
错排数的 EGF 推导：
Dₙ 是 n 个元素的排列中没有元素在原来位置的方案数。
EGF = e⁻ˣ/(1-x) = (Σ(-1)ⁿxⁿ/n!) · (Σ xⁿ)
展开取 [xⁿ/n!] 系数：Dₙ = n! Σ(k=0→n) (-1)ᵏ/k!

4. 用生成函数解递推关系

Fibonacci 数列：Fₙ = Fₙ₋₁ + Fₙ₋₂，F₀=0, F₁=1。

设 G(x) = Σ Fₙxⁿ。
G(x) = F₀ + F₁x + Σ(n≥2) Fₙxⁿ
= x + Σ(n≥2) (Fₙ₋₁+Fₙ₋₂)xⁿ
= x + x·(G(x)-F₀) + x²·G(x)
= x + xG(x) + x²G(x)
解得：
G(x) = x/(1-x-x²)

部分分式分解后可得 Binet 公式：Fₙ = (φⁿ-ψⁿ)/√5。

4.1 解题步骤

  1. 设 G(x) = Σ aₙxⁿ 为待求序列的生成函数
  2. 利用递推关系，将 Σ aₙxⁿ 的递推式转化为 G(x) 的方程
  3. 解方程得到 G(x) 的封闭形式
  4. 对 G(x) 进行部分分式分解或泰勒展开，提取系数 aₙ

本节总结

  • OGF：G(x)=Σ aₙxⁿ，用于组合计数，卷积为普通卷积
  • EGF：E(x)=Σ aₙxⁿ/n!，用于排列计数，卷积含二项式系数
  • 核心公式：1/(1-x)=Σxⁿ, 1/(1-x)ⁿ=ΣC(n+k-1,k)xⁿ, eˣ=Σxⁿ/n!
  • 乘法原理：OGF/EGF 的乘积对应计数中的"分步选择"
  • 解递推：序列递推 → 生成函数方程 → 部分分式分解 → 提取系数
  • 整数拆分：Π 1/(1-x^k)；错排：e⁻ˣ/(1-x)
