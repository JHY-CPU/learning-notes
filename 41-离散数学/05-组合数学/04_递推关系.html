递推关系

离散数学 · 组合数学 · 第4节

1. 递推关系概述

1.1 定义

递推关系 (Recurrence Relation)：将序列的第 n 项表示为前面若干项的函数的等式。
递推关系 + 初始条件 = 完整的序列定义。
名称递推关系初始条件解
FibonacciFₙ = Fₙ₋₁ + Fₙ₋₂F₀=0, F₁=1(φⁿ-ψⁿ)/√5
阶乘n! = n·(n-1)!0!=1n!
Hanoi塔Tₙ = 2Tₙ₋₁ + 1T₁=12ⁿ-1
错排Dₙ = (n-1)(Dₙ₋₁+Dₙ₋₂)D₁=0, D₂=1n!Σ(-1)ᵏ/k!
CatalanCₙ = Σ CₖCₙ₋₁₋ₖC₀=1C(2n,n)/(n+1)
二分查找T(n) = T(n/2)+1T(1)=1⌊log₂n⌋+1

2. 线性常系数齐次递推

2.1 标准形式

aₙ + c₁aₙ₋₁ + c₂aₙ₋₂ + ... + cₖaₙ₋ₖ = 0

其中 c₁, c₂, ..., cₖ 为常数，k 为递推的阶数。

2.2 特征方程法

求解步骤：
  1. 写出特征方程：rᵏ + c₁rᵏ⁻¹ + ... + cₖ = 0
  2. 求特征根 r₁, r₂, ..., rₖ
  3. 根据根的类型写出通解
  4. 利用初始条件确定系数

2.3 根的情况与通解形式

特征根情况通解形式
k 个不同实根 r₁,...,rₖaₙ = A₁r₁ⁿ + A₂r₂ⁿ + ... + Aₖrₖⁿ
r 是 m 重根对应项为 (A₁ + A₂n + ... + Aₘnᵐ⁻¹)rⁿ
共轭复根 α±βi对应项为 ρⁿ(A cos nθ + B sin nθ)，其中 ρ=|α+βi|, θ=arg(α+βi)
例：Fibonacci 数列 Fₙ = Fₙ₋₁ + Fₙ₋₂，F₀=0, F₁=1。
特征方程：r² = r + 1，即 r² - r - 1 = 0
特征根：r₁ = (1+√5)/2 = φ，r₂ = (1-√5)/2 = ψ
通解：Fₙ = A·φⁿ + B·ψⁿ
代入初始条件：F₀ = A+B = 0，F₁ = Aφ+Bψ = 1
解得：A = 1/√5，B = -1/√5
Fₙ = (φⁿ - ψⁿ)/√5    (Binet 公式)
例：Hanoi塔 Tₙ = 2Tₙ₋₁ + 1，T₁=1。
这是非齐次递推。先求齐次解：r-2=0 → r=2，Tₙ⁽ʰ⁾ = A·2ⁿ
特解设为常数 C：C = 2C + 1 → C = -1
通解：Tₙ = A·2ⁿ - 1
T₁ = 2A - 1 = 1 → A = 1
Tₙ = 2ⁿ - 1

3. 线性常系数非齐次递推

aₙ + c₁aₙ₋₁ + ... + cₖaₙ₋ₖ = f(n)
求解方法：通解 = 齐次通解 + 非齐次特解。
特解的猜测形式取决于 f(n) 的类型：
f(n) 的形式特解猜测特殊情况
常数 d常数 C若 r=1 是特征根，乘以 n
d·nC₁n+C₀若 r=1 是 m 重根，乘以 nᵐ
d·nᵖCₚnᵖ+...+C₁n+C₀同上
d·sⁿC·sⁿ若 s 是 m 重特征根，乘以 nᵐ
d·sⁿ·nᵖ(Cₚnᵖ+...+C₀)sⁿ同上

4. Fibonacci 数列的推广

4.1 广义 Fibonacci (k-步)

Fₙ⁽ᵏ⁾ = Fₙ₋₁⁽ᵏ⁾ + Fₙ₋₂⁽ᵏ⁾ + ... + Fₙ₋ₖ⁽ᵏ⁾

Tribonacci (k=3)：Tₙ = Tₙ₋₁ + Tₙ₋₂ + Tₙ₋₃

4.2 Fibonacci 的重要性质

性质公式
Cassini 恒等式Fₙ₊₁·Fₙ₋₁ - Fₙ² = (-1)ⁿ
求和Σ(k=0→n) Fₖ = Fₙ₊₂ - 1
平方和Σ(k=1→n) Fₖ² = Fₙ·Fₙ₊₁
相邻比值lim Fₙ₊₁/Fₙ = φ (黄金比例)
与组合数Fₙ = Σ C(n-1-k, k) (k从0到⌊(n-1)/2⌋)
GCDgcd(Fₘ, Fₙ) = F_(gcd(m,n))

5. Master 定理 (主定理)

用于分析分治算法的递推关系：

T(n) = a·T(n/b) + f(n)

其中 a ≥ 1, b > 1 为常数，f(n) 为渐进正函数。

Master 定理：设 f(n) = Θ(nᶜ · logᵏn)，比较 c 与 log_b(a)：

情况1：若 c < log_b(a)，则 T(n) = Θ(n^(log_b(a)))
情况2：若 c = log_b(a)，则 T(n) = Θ(nᶜ · log^(k+1)n)
        特别地 k=0 时 T(n) = Θ(nᶜ · log n)
情况3：若 c > log_b(a)，则 T(n) = Θ(nᶜ)
例1：二分查找 T(n) = T(n/2) + 1
a=1, b=2, f(n)=1=Θ(n⁰)。c=0, log₂1=0。
属于情况2 (c=log_b(a), k=0)：T(n) = Θ(log n) ✓
例2：归并排序 T(n) = 2T(n/2) + n
a=2, b=2, f(n)=n=Θ(n¹)。c=1, log₂2=1。
属于情况2 (c=log_b(a), k=0)：T(n) = Θ(n log n) ✓
例3：Strassen 矩阵乘法 T(n) = 7T(n/2) + Θ(n²)
a=7, b=2, f(n)=Θ(n²)。c=2, log₂7 ≈ 2.807。
属于情况1 (c < log_b(a))：T(n) = Θ(n^(log₂7)) ≈ Θ(n^2.807) ✓

6. 其他求解方法

6.1 迭代法 (展开法)

T(n) = 2T(n-1) + 1, T(0)=0
T(n) = 2[2T(n-2)+1]+1 = 2²T(n-2)+2+1
= 2³T(n-3)+4+2+1
= ...
= 2ⁿT(0) + 2ⁿ⁻¹+2ⁿ⁻²+...+1
= 0 + 2ⁿ-1 = 2ⁿ-1

6.2 换元法

令 n = 2ᵐ（或 bᵐ），将 T(n) 转化为关于 m 的递推。
例：T(n) = 2T(√n) + log n，令 n=2ᵐ 得 S(m) = 2S(m/2)+m，用 Master 定理解。

6.3 生成函数法

设 G(x)=Σ aₙxⁿ，将递推关系转化为 G(x) 的函数方程，求解后提取系数。
详见"生成函数"一节。

本节总结

  • 线性齐次递推：特征方程法 → 不同根/重根/复根的通解形式
  • 非齐次递推：通解 = 齐次通解 + 特解（待定系数法）
  • Fibonacci：Binet 公式 Fₙ=(φⁿ-ψⁿ)/√5，Cassini 恒等式，gcd 性质
  • Master 定理：T(n)=aT(n/b)+f(n)，比较 c 与 log_b(a) 三种情况
  • 其他方法：迭代展开、换元法、生成函数法
  • 常见递推：Hanoi T(n)=2ⁿ-1, 错排 Dₙ=n!Σ(-1)ᵏ/k!, Catalan C(n)=(C(2n,n))/(n+1)
