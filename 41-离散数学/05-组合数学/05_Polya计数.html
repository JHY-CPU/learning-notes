Polya 计数理论

离散数学 · 组合数学 · 第5节

1. 问题背景：等价类计数

核心问题：对 n 个对象用 k 种颜色染色，在某种对称操作下视为"相同"的方案有多少种？

例：用红蓝两色给正方形的 4 个顶点染色，旋转后相同的算一种。不考虑对称有 2⁴=16 种，考虑旋转后有多少种？

2. 群作用与轨道

2.1 基本定义

群作用 (Group Action)：群 G 作用在集合 X 上，对 g∈G, x∈X，定义 g·x ∈ X，满足：
  • e·x = x（单位元不动）
  • (gh)·x = g·(h·x)（结合律）
轨道 (Orbit)：x 的轨道 Orb(x) = {g·x : g ∈ G}
稳定子 (Stabilizer)：x 的稳定子 Stab(x) = {g ∈ G : g·x = x}
轨道-稳定子定理：|Orb(x)| · |Stab(x)| = |G|

3. Burnside 引理

Burnside 引理：设有限群 G 作用在有限集合 X 上，则轨道数目（即不等价的方案数）为：
N = (1/|G|) · Σ(g∈G) |Xᵍ|
其中 Xᵍ = {x ∈ X : g·x = x} 是被 g 固定的元素集合（g 作用下不变的元素）。
例：正方形 4 顶点 2 色染色（旋转）
G = {0°, 90°, 180°, 270°}，|G| = 4，|X| = 2⁴ = 16

旋转 g循环结构|Xᵍ| (不动染色数)
0° (恒等)(1)(2)(3)(4) — 4 个循环2⁴ = 16
90°(1 2 3 4) — 1 个循环2¹ = 2
180°(1 3)(2 4) — 2 个循环2² = 4
270°(1 4 3 2) — 1 个循环2¹ = 2

N = (1/4)(16 + 2 + 4 + 2) = 6 种不等价染色方案。
关键洞察：g 的置换分解为循环后，每个循环内的位置必须同色，所以 |Xᵍ| = k^(#循环)，其中 k 是颜色数。

4. Polya 计数定理

Polya 定理：设群 G 作用在位置集合 {1,...,n} 上，用 k 种颜色染色。若 g ∈ G 的循环分解有 c₁(g) 个1-循环, c₂(g) 个2-循环, ..., 则不等价染色方案数为：
N = (1/|G|) · Σ(g∈G) k^(c₁(g)+c₂(g)+...+cₙ(g))
即 N = (1/|G|) · Σ(g∈G) k^(#g的循环数)

4.1 循环指标多项式

P_G(z₁, z₂, ..., zₙ) = (1/|G|) · Σ(g∈G) z₁^{c₁(g)} · z₂^{c₂(g)} · ... · zₙ^{cₙ(g)}

将 z₁ = z₂ = ... = zₙ = k 代入，即得 k 色染色的方案数。

正方形旋转群的循环指标：
P_G = (1/4)(z₁⁴ + 2z₄ + z₂²)
  • 0°: 4个1-循环 → z₁⁴
  • 90°, 270°: 各1个4-循环 → 2z₄
  • 180°: 2个2-循环 → z₂²
令所有 zᵢ = k: N = (1/4)(k⁴ + 2k + k²)
k=2: N = (16+4+4)/4 = 6 ✓

5. 应用实例

5.1 项链问题

用 3 种颜色给 6 颗珠子的项链染色（旋转+翻转视为相同）。
二面体群 D₆，|G| = 12。
循环指标：P_D₆ = (1/12)(z₁⁶ + 2z₆ + 2z₃² + z₂³ + 3z₂³ + 3z₁²z₂²)
（需精确计算每个操作的循环数）
令 zᵢ = 3，得 N = 92 种不等价项链。

化学中用于计算分子异构体数目：
例：苯环 C₆H₆ 的二氯取代物（6个碳位置选2个放 Cl）：
D₆ 群作用，用 Polya 定理计算不等价位置对数 = 3 种（邻位、间位、对位）。

本节总结

  • 问题本质：在群作用下计数轨道（等价类）数目
  • Burnside 引理：N = (1/|G|) Σ |Xᵍ|，对每个群元素数不动点
  • Polya 定理：N = (1/|G|) Σ k^(#循环)，利用置换的循环结构
  • 循环指标：P_G(z₁,...,zₙ) = (1/|G|) Σ Π zᵢ^{cᵢ(g)}
  • 关键技巧：确定对称群 G → 分析每个 g 的循环结构 → 代入公式
  • 应用：项链染色、化学异构体、棋盘染色、编码理论
