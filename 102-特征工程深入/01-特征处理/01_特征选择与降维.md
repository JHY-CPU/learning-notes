# 特征选择与降维 - 特征工程深入


## 1. 特征选择的意义与方法分类


特征选择是从原始特征集中选出最相关、最有信息量的子集的过程，是特征工程中最重要的步骤之一。


### 为什么要特征选择


- **减少过拟合：**
   移除无关特征降低模型复杂度
- **提升训练速度：**
   特征越少，训练越快
- **提高可解释性：**
   精简特征集便于理解和调试
- **降低存储和计算成本：**
   线上推理需要更少资源
- **缓解维度灾难：**
   高维空间中距离度量失效


| 方法类别 | 原理 | 优势 | 劣势 |
| --- | --- | --- | --- |
| 过滤法（Filter） | 基于统计指标独立评估每个特征 | 速度快，与模型无关 | 忽略特征间交互 |
| 包装法（Wrapper） | 用模型性能评估特征子集 | 考虑特征交互，效果好 | 计算开销大，易过拟合 |
| 嵌入法（Embedded） | 模型训练过程中自动选择特征 | 效率与效果平衡 | 依赖特定模型 |


## 2. 过滤法（Filter Methods）


### 方差阈值


移除方差低于阈值的特征，因为方差低意味着该特征几乎没有区分度。


### 相关系数


$$
Pearson相关系数：r = Σ(xᵢ - x̄)(yᵢ - ȳ) / √(Σ(xᵢ-x̄)² × Σ(yᵢ-ȳ)²)
                |r| > 0.7：强相关；0.3 < |r| < 0.7：中等相关；|r| < 0.3：弱相关
$$


### 互信息（Mutual Information）


衡量特征与目标变量之间的非线性关系，值越大表示特征越重要。


$$
MI(X; Y) = Σ p(x,y) × log(p(x,y) / (p(x)×p(y)))
$$


### 卡方检验


适用于类别型特征与类别型目标变量，检验特征与目标是否独立。


| 方法 | 适用数据 | 捕获关系 | 计算复杂度 |
| --- | --- | --- | --- |
| 方差阈值 | 数值型 | 方差大小 | O(n) |
| Pearson相关 | 数值型→数值型 | 线性关系 | O(n) |
| Spearman相关 | 数值型→数值型 | 单调关系 | O(n log n) |
| 互信息 | 通用 | 任意关系 | O(n log n) |
| 卡方检验 | 类别型→类别型 | 独立性 | O(n) |


## 3. 包装法与嵌入法


### 包装法（Wrapper Methods）


- **前向选择：**
   从空集开始，每次添加一个使模型性能提升最大的特征
- **后向消除：**
   从全集开始，每次移除一个对模型性能影响最小的特征
- **递归特征消除（RFE）：**
   反复训练模型，每次移除重要性最低的特征


### 嵌入法（Embedded Methods）


- **L1正则化（Lasso）：**
   L1惩罚项将不重要特征的系数压缩为0
- **树模型特征重要性：**
   基于信息增益或Gini不纯度评估特征重要性
- **Elastic Net：**
   L1+L2混合正则化


> **Important:** 包装法虽然效果好，但计算开销随特征数指数增长。对于大规模特征集（1000+特征），优先使用过滤法或嵌入法。


## 4. PCA降维与t-SNE可视化


### PCA（主成分分析）


通过线性变换将高维数据投影到低维空间，保留最大方差方向。


$$
步骤：
                1. 数据标准化（均值为0，方差为1）
                2. 计算协方差矩阵 C = XᵀX / (n-1)
                3. 对C进行特征值分解
                4. 选取前k个最大特征值对应的特征向量
                5. X_reduced = X × W_k（W_k为前k个特征向量组成的矩阵）
$$


> **Note:** **选择k的原则：**
> 累积方差贡献率达到85%-95%。例如，前10个主成分解释了90%的方差，则可以用10维代替原始100维特征。


### t-SNE可视化


t-SNE是一种非线性降维方法，主要用于高维数据的二维或三维可视化。


- 保留局部结构：相似的样本在低维空间中仍然接近
- 不保留全局距离：不同簇之间的距离没有意义
- 不适合降维后的特征用于训练（perplexity参数敏感）


## 5. Python代码：sklearn特征选择


```
from sklearn.datasets import load_breast_cancer
from sklearn.feature_selection import (
    SelectKBest, f_classif, mutual_info_classif,
    RFE, SelectFromModel, VarianceThreshold
)
from sklearn.ensemble import RandomForestClassifier
from sklearn.linear_model import LassoCV
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
import numpy as np

# 加载数据
X, y = load_breast_cancer(return_X_y=True)
feature_names = load_breast_cancer().feature_names
print(f"原始特征数: {X.shape[1]}")

# ========== 1. 方差阈值 ==========
selector_var = VarianceThreshold(threshold=0.1)
X_var = selector_var.fit_transform(X)
print(f"方差阈值后特征数: {X_var.shape[1]}")

# ========== 2. 互信息特征选择 ==========
mi_scores = mutual_info_classif(X, y, random_state=42)
mi_ranking = sorted(zip(feature_names, mi_scores), key=lambda x: -x[1])
print("\n互信息Top-5特征:")
for name, score in mi_ranking[:5]:
    print(f"  {name}: {score:.4f}")

# ========== 3. SelectKBest（F统计量）==========
selector_kbest = SelectKBest(f_classif, k=10)
X_kbest = selector_kbest.fit_transform(X, y)
selected_mask = selector_kbest.get_support()
print(f"\nSelectKBest选择的特征: {feature_names[selected_mask]}")

# ========== 4. RFE（递归特征消除）==========
rf = RandomForestClassifier(n_estimators=100, random_state=42)
rfe = RFE(rf, n_features_to_select=10)
rfe.fit(X, y)
print(f"\nRFE选择的特征: {feature_names[rfe.support_]}")
print(f"特征排名: {dict(zip(feature_names, rfe.ranking_))}")

# ========== 5. L1正则化（Lasso）特征选择 ==========
X_scaled = StandardScaler().fit_transform(X)
lasso = LassoCV(cv=5, random_state=42)
lasso.fit(X_scaled, y)
important_features = np.abs(lasso.coef_) > 0.01
print(f"\nLasso选择的特征数: {important_features.sum()}")
print(f"选择的特征: {feature_names[important_features]}")

# ========== 6. PCA降维 ==========
pca = PCA(n_components=0.95)  # 保留95%方差
X_pca = pca.fit_transform(X_scaled)
print(f"\nPCA降维: {X.shape[1]}维 → {X_pca.shape[1]}维")
print(f"各主成分方差解释比: {pca.explained_variance_ratio_}")
print(f"累积方差解释比: {pca.explained_variance_ratio_.cumsum()}")

# ========== 7. 树模型特征重要性 ==========
rf.fit(X, y)
importance = sorted(zip(feature_names, rf.feature_importances_),
                     key=lambda x: -x[1])
print("\n随机森林特征重要性Top-5:")
for name, imp in importance[:5]:
    print(f"  {name}: {imp:.4f}")
```


## 总结


- 特征选择的三大方法：过滤法（快但粗略）、包装法（准但慢）、嵌入法（平衡）
- 过滤法包括方差阈值、相关系数、互信息、卡方检验等统计指标
- RFE是最常用的包装法，结合模型重要性递归消除特征
- L1正则化（Lasso）和树模型重要性是嵌入法的代表
- PCA通过保留最大方差方向降维，通常保留85%-95%方差
- t-SNE适合可视化但不适合训练用的降维
