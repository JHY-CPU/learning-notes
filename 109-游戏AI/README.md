# 109-游戏AI

游戏AI是现代电子游戏中赋予非玩家角色（NPC）智能行为的核心系统。本模块覆盖从经典寻路算法到高级决策系统的完整知识体系。

## 目录

1. [A星寻路算法](./A星寻路算法.md) - 经典启发式图搜索算法
2. [导航网格(NavMesh)](./导航网格(NavMesh).md) - 3D场景寻路基础设施
3. [有限状态机(FSM)](./有限状态机(FSM).md) - 状态驱动的行为控制
4. [行为树(Behavior Tree)](./行为树(Behavior%20Tree).md) - 模块化AI决策框架
5. [GOAP目标导向行动规划](./GOAP目标导向行动规划.md) - 自动规划AI行为序列
6. [感知系统](./感知系统.md) - 视觉、听觉等感官模拟
7. [群体AI与蜂群行为](./群体AI与蜂群行为.md) - 大规模群体运动控制
8. [决策树与效用系统](./决策树与效用系统.md) - 基于效用的智能决策
9. [对话树与任务系统](./对话树与任务系统.md) - NPC交互与任务管理
10. [AI调试与可视化](./AI调试与可视化.md) - AI行为调试工具与方法

## 学习路径

```
基础层: A星寻路 -> NavMesh -> FSM
进阶层: 行为树 -> GOAP -> 感知系统
高级层: 群体AI -> 效用系统 -> 调试可视化
```

## 推荐引擎/框架

- **Unity**: Navigation、Behavior Designer
- **Unreal Engine**: AI Controller、Behavior Tree、EQS
- **Godot**: NavigationRegion3D、自定义行为树
