# 感知系统

## 1. 核心理论

### 1.1 什么是感知系统

感知系统（Perception System）模拟AI角色对游戏世界的感官能力，是AI"知道什么"与"不知道什么"的边界。一个完善的感知系统需要回答三个核心问题：

1. **我能感知到什么？** —— 感知检测（视觉、听觉、触觉）
2. **我记住了什么？** —— 记忆系统（位置、时间、置信度）
3. **我怎么解读这些信息？** —— 感知评估（威胁等级、优先级排序）

感知系统的设计直接决定了AI的"智能感"：感知太强显得作弊（超自然的感知），感知太弱显得愚蠢（看不见面前的敌人）。

### 1.2 感知模型分层

```
原始刺激（Raw Stimulus）
  │  声音事件、视觉线索、伤害事件
  ▼
感知过滤（Perception Filtering）
  │  距离衰减、遮挡检测、注意力限制
  ▼
感知刺激（Perception Stimulus）
  │  过滤后的有效感知
  ▼
感知记忆（Perception Memory）
  │  时间衰减、置信度更新
  ▼
感知评估（Perception Evaluation）
  │  威胁评估、优先级排序
  ▼
决策输入（Decision Input）
  → 提供给行为树/FSM/GOAP
```

## 2. 视觉感知

### 2.1 视锥检测原理

视觉感知的核心是**视锥检测**（Field of View / Vision Cone），模拟生物的视觉范围：

- **视野距离（Sight Range）**：能看多远
- **视野角度（FOV）**：能看多宽（通常用半角表示）
- **余光视野（Peripheral Vision）**：侧面的模糊感知
- **遮挡检测（Occlusion）**：视线是否被墙壁等障碍物阻挡

### 2.2 视锥检测实现

```cpp
#include <cmath>
#include <vector>
#include <algorithm>

struct Vec3 {
    float x, y, z;
    Vec3 operator-(const Vec3& o) const { return {x-o.x, y-o.y, z-o.z}; }
    Vec3 operator+(const Vec3& o) const { return {x+o.x, y+o.y, z+o.z}; }
    Vec3 operator*(float s) const { return {x*s, y*s, z*s}; }
    float Length() const { return sqrtf(x*x + y*y + z*z); }
    Vec3 Normalized() const {
        float len = Length();
        return len > 0 ? Vec3{x/len, y/len, z/len} : Vec3{0,0,0};
    }
    static float Dot(const Vec3& a, const Vec3& b) {
        return a.x*b.x + a.y*b.y + a.z*b.z;
    }
};

struct VisionConfig {
    float sightRange;           // 主视野距离（如20米）
    float fovAngle;             // 主视野半角（如60度，转弧度）
    float peripheralRange;      // 余光距离（如8米）
    float peripheralAngle;      // 余光半角（如110度）
    float eyeHeightOffset;      // 眼睛高度偏移
};

struct RaycastResult {
    bool hit;
    float distance;
    void* hitEntity;  // 射线击中的实体
};

class VisionSensor {
public:
    // 完整的视觉检测
    static bool CanSee(
        const Vec3& viewerPos,
        const Vec3& viewerForward,
        const Vec3& targetPos,
        const VisionConfig& config,
        std::function<RaycastResult(Vec3, Vec3, float)> raycast)
    {
        Vec3 eyePos = viewerPos;
        eyePos.y += config.eyeHeightOffset;

        Vec3 dir = targetPos - eyePos;
        float dist = dir.Length();

        // 阶段1：距离检查
        if (dist > config.sightRange) return false;

        // 阶段2：方向检查（视锥内判断）
        Vec3 dirNorm = dir.Normalized();
        float cosAngle = Vec3::Dot(dirNorm, viewerForward);
        float angleDeg = acosf(std::max(-1.0f, std::min(1.0f, cosAngle))) * 180.0f / 3.14159f;

        // 判断是否在主视野或余光内
        bool inMainVision = (dist <= config.sightRange && angleDeg <= config.fovAngle);
        bool inPeripheral = (dist <= config.peripheralRange && angleDeg <= config.peripheralAngle);

        if (!inMainVision && !inPeripheral) return false;

        // 阶段3：余光检测需要更低的感知质量
        if (inPeripheral && !inMainVision) {
            // 余光区域：距离更近才更容易被看到
            float peripheralEffectiveness = 1.0f - (dist / config.peripheralRange);
            if (peripheralEffectiveness < 0.3f) return false;
        }

        // 阶段4：遮挡检测（射线检测）
        RaycastResult result = raycast(eyePos, dirNorm, dist);
        if (result.hit && result.distance < dist - 0.5f) {
            return false; // 视线被遮挡
        }

        return true;
    }

    // 计算感知质量（0-1）
    static float GetVisionQuality(
        float distance,
        float angleDeg,
        const VisionConfig& config)
    {
        // 距离衰减：越远质量越低
        float distFactor = 1.0f - (distance / config.sightRange);
        distFactor = std::max(0.0f, distFactor);

        // 角度衰减：偏离中心轴质量降低
        float angleFactor = 1.0f - (angleDeg / config.fovAngle);
        angleFactor = std::max(0.0f, angleFactor);

        // 综合质量
        return distFactor * 0.6f + angleFactor * 0.4f;
    }
};
```

## 3. 听觉感知

### 3.1 声音模型

听觉感知的实现比视觉简单，因为声音是**全向传播**的（不考虑朝向），但需要考虑：

- **声音强度**：不同动作产生不同音量（走路 < 跑步 < 射击 < 爆炸）
- **距离衰减**：声音强度随距离衰减
- **墙壁衰减**：障碍物可降低声音传播距离
- **声音类型**：脚步声、枪声、碰撞声等不同类别

### 3.2 完整听觉系统

```cpp
enum SoundType {
    SOUND_FOOTSTEP,
    SOUND_GUNSHOT,
    SOUND_EXPLOSION,
    SOUND_VOICE,
    SOUND_COLLISION,
    SOUND_RELOAD
};

struct SoundEvent {
    Vec3 origin;         // 声音来源位置
    SoundType type;      // 声音类型
    float intensity;     // 原始强度（0-1）
    float maxRange;      // 最大传播距离
    float timestamp;     // 发生时间
    void* sourceEntity;  // 声音来源实体
};

struct HearingConfig {
    float hearingRange;              // 基础听觉距离
    float focusMultiplier = 1.5f;    // 专注时听觉增强
    float wallAttenuation = 0.5f;    // 墙壁衰减系数
    float typeMultiplier[6];         // 各声音类型的敏感度
};

class HearingSensor {
public:
    static bool CanHear(
        const Vec3& listenerPos,
        const SoundEvent& event,
        const HearingConfig& config,
        std::function<float(Vec3, Vec3)> getWallThickness) // 返回两位置间墙体厚度
    {
        float distance = (event.origin - listenerPos).Length();

        // 计算声音传播后的有效强度
        float effectiveRange = event.maxRange * event.intensity *
                              config.typeMultiplier[event.type];

        // 墙壁衰减
        float wallThickness = getWallThickness(listenerPos, event.origin);
        if (wallThickness > 0) {
            effectiveRange *= powf(config.wallAttenuation, wallThickness);
        }

        // 判断是否在听觉范围内
        return distance <= effectiveRange * config.hearingRange;
    }

    // 计算声音定位精度（0-1）
    static float GetHearingAccuracy(float distance, float effectiveRange, SoundType type) {
        float distFactor = 1.0f - (distance / effectiveRange);
        // 枪声比脚步声更容易定位
        float typeFactor = (type == SOUND_GUNSHOT || type == SOUND_EXPLOSION) ? 1.0f : 0.7f;
        return std::max(0.0f, distFactor * typeFactor);
    }
};

// 声音事件管理器
class SoundEventManager {
    std::vector<SoundEvent> recentSounds;
    float soundLifetime = 5.0f; // 声音事件保留时间

public:
    void RegisterSound(const SoundEvent& event) {
        recentSounds.push_back(event);
    }

    void Update(float currentTime) {
        // 清理过期声音
        recentSounds.erase(
            std::remove_if(recentSounds.begin(), recentSounds.end(),
                [currentTime, this](const SoundEvent& e) {
                    return currentTime - e.timestamp > soundLifetime;
                }),
            recentSounds.end());
    }

    // 查询指定位置的AI能听到的声音
    std::vector<const SoundEvent*> GetAudibleSounds(
        const Vec3& listenerPos, const HearingConfig& config) const
    {
        std::vector<const SoundEvent*> result;
        for (const auto& sound : recentSounds) {
            float distance = (sound.origin - listenerPos).Length();
            float effectiveRange = sound.maxRange * sound.intensity *
                                  config.typeMultiplier[sound.type];
            if (distance <= effectiveRange * config.hearingRange) {
                result.push_back(&sound);
            }
        }
        // 按强度排序（最近/最响的优先）
        std::sort(result.begin(), result.end(),
            [listenerPos](const SoundEvent* a, const SoundEvent* b) {
                float da = (a->origin - listenerPos).Length();
                float db = (b->origin - listenerPos).Length();
                return da / a->intensity < db / b->intensity;
            });
        return result;
    }
};
```

## 4. 伤害感知

伤害感知是最高优先级的感知类型——AI受到伤害时应立即反应。

```cpp
struct DamageEvent {
    float damage;
    Vec3 damageOrigin;     // 伤害来源方向
    void* attacker;        // 攻击者
    float timestamp;
    DamageType type;       // 近战/远程/爆炸
};

class DamageSensor {
public:
    static void OnTakeDamage(Entity* entity, const DamageEvent& event) {
        // 立即更新感知记忆
        PerceptionMemory memory;
        memory.target = event.attacker;
        memory.lastKnownPosition = event.damageOrigin;
        memory.lastSeenTime = event.timestamp;
        memory.confidence = 1.0f; // 伤害感知置信度最高
        memory.perceptionType = PERCEPTION_DAMAGE;

        entity->UpdatePerceptionMemory(memory);

        // 如果无法看到攻击者，根据伤害方向估计来源位置
        if (!entity->CanSeeEntity(event.attacker)) {
            Vec3 estimatedPos = entity->position + (event.damageOrigin - entity->position).Normalized() * 10.0f;
            memory.lastKnownPosition = estimatedPos;
            memory.confidence = 0.7f; // 方向估计，置信度略低
        }
    }
};
```

## 5. 记忆系统

### 5.1 记忆模型

```cpp
enum PerceptionType {
    PERCEPTION_VISUAL,
    PERCEPTION_AUDITORY,
    PERCEPTION_DAMAGE
};

struct PerceptionMemory {
    void* target;                   // 目标实体
    Vec3 lastKnownPosition;         // 最后已知位置
    Vec3 lastKnownVelocity;         // 最后已知移动方向
    float lastSeenTime;             // 最后看到时间
    float lastHeardTime;            // 最后听到时间
    float lastDamageTime;           // 最后被攻击时间
    float confidence;               // 置信度 (0-1)
    PerceptionType perceptionType;  // 最后感知类型
    bool bIsHostile;                // 是否为敌对目标
    int timesSpotted;               // 被看到的次数

    PerceptionMemory()
        : target(nullptr), lastSeenTime(0), lastHeardTime(0),
          lastDamageTime(0), confidence(0), bIsHostile(false), timesSpotted(0) {}
};
```

### 5.2 记忆管理器

```cpp
class PerceptionMemoryManager {
    std::unordered_map<void*, PerceptionMemory> memories;
    float memoryDecayRate;      // 置信度衰减速率（每秒）
    float memoryForgetThreshold; // 置信度低于此值时清除记忆

public:
    PerceptionMemoryManager(float decayRate = 0.1f, float forgetThreshold = 0.1f)
        : memoryDecayRate(decayRate), memoryForgetThreshold(forgetThreshold) {}

    // 更新视觉感知
    void UpdateVisual(void* target, const Vec3& position, const Vec3& velocity,
                      float currentTime, bool isHostile) {
        PerceptionMemory& mem = memories[target];
        mem.target = target;
        mem.lastKnownPosition = position;
        mem.lastKnownVelocity = velocity;
        mem.lastSeenTime = currentTime;
        mem.confidence = 1.0f; // 直接看到，完全置信
        mem.perceptionType = PERCEPTION_VISUAL;
        mem.bIsHostile = isHostile;
        mem.timesSpotted++;
    }

    // 更新听觉感知
    void UpdateAuditory(void* target, const Vec3& estimatedPos, float currentTime) {
        auto it = memories.find(target);
        if (it != memories.end()) {
            // 已有记忆，更新位置估计
            it->second.lastKnownPosition = estimatedPos;
            it->second.lastHeardTime = currentTime;
            // 听觉不覆盖视觉置信度
            if (it->second.confidence < 0.5f) {
                it->second.confidence = 0.5f;
            }
        } else {
            // 新记忆
            PerceptionMemory mem;
            mem.target = target;
            mem.lastKnownPosition = estimatedPos;
            mem.lastHeardTime = currentTime;
            mem.confidence = 0.5f; // 听觉置信度中等
            mem.perceptionType = PERCEPTION_AUDITORY;
            mem.bIsHostile = true; // 默认敌对
            memories[target] = mem;
        }
    }

    // 每帧更新（衰减记忆）
    void Tick(float deltaTime, float currentTime) {
        std::vector<void*> toRemove;

        for (auto& [key, mem] : memories) {
            // 如果当前仍能看到目标，置信度不衰减
            float timeSinceLastSeen = currentTime - mem.lastSeenTime;
            if (timeSinceLastSeen > 1.0f) {
                // 脱离视野后，置信度随时间衰减
                mem.confidence -= memoryDecayRate * deltaTime;

                // 根据看到次数调整衰减速度（频繁看到的敌人记忆更持久）
                float persistenceBonus = std::min(mem.timesSpotted * 0.05f, 0.3f);
                mem.confidence += persistenceBonus * deltaTime;
            }

            mem.confidence = std::max(0.0f, std::min(1.0f, mem.confidence));

            // 置信度过低，清除记忆
            if (mem.confidence < memoryForgetThreshold) {
                toRemove.push_back(key);
            }
        }

        for (void* key : toRemove) {
            memories.erase(key);
        }
    }

    // 查询记忆
    const PerceptionMemory* GetMemory(void* target) const {
        auto it = memories.find(target);
        return it != memories.end() ? &it->second : nullptr;
    }

    // 获取最高威胁目标
    const PerceptionMemory* GetHighestThreat() const {
        const PerceptionMemory* best = nullptr;
        float bestThreat = 0;

        for (const auto& [key, mem] : memories) {
            if (!mem.bIsHostile) continue;
            float threat = mem.confidence;

            // 距离因素：位置估计越久远，威胁越低
            // （可以根据lastKnownPosition与自己的距离进一步计算）

            if (threat > bestThreat) {
                bestThreat = threat;
                best = &mem;
            }
        }
        return best;
    }

    // 获取所有已知目标
    std::vector<const PerceptionMemory*> GetAllMemories() const {
        std::vector<const PerceptionMemory*> result;
        for (const auto& [key, mem] : memories) {
            result.push_back(&mem);
        }
        // 按置信度排序
        std::sort(result.begin(), result.end(),
            [](const PerceptionMemory* a, const PerceptionMemory* b) {
                return a->confidence > b->confidence;
            });
        return result;
    }

    void Clear() { memories.clear(); }
};
```

### 5.3 置信度衰减曲线

不同的衰减曲线产生不同的AI"性格"：

```cpp
// 指数衰减：自然，适合大多数场景
float ExponentialDecay(float initialConfidence, float timeSinceLastSeen, float decayRate) {
    return initialConfidence * expf(-decayRate * timeSinceLastSeen);
}

// 线性衰减：可预测，调试方便
float LinearDecay(float initialConfidence, float timeSinceLastSeen, float decayRate) {
    return std::max(0.0f, initialConfidence - decayRate * timeSinceLastSeen);
}

// 阶梯衰减：模拟"突然忘记"
float StepDecay(float initialConfidence, float timeSinceLastSeen) {
    if (timeSinceLastSeen < 3.0f) return initialConfidence;
    if (timeSinceLastSeen < 8.0f) return initialConfidence * 0.5f;
    if (timeSinceLastSeen < 15.0f) return initialConfidence * 0.2f;
    return 0.0f;
}

// S型衰减：初始保持，然后快速衰减
float SigmoidDecay(float initialConfidence, float timeSinceLastSeen,
                   float midpoint = 10.0f, float steepness = 1.0f) {
    return initialConfidence / (1.0f + expf(steepness * (timeSinceLastSeen - midpoint)));
}
```

## 6. 感知优先级与评估

### 6.1 刺激评估

```cpp
struct PerceptionStimulus {
    void* source;
    PerceptionType type;
    Vec3 location;
    float strength;         // 刺激强度
    float timestamp;
    float importance;       // 重要性评分
};

class PerceptionEvaluator {
public:
    // 计算刺激的重要性评分
    static float EvaluateImportance(
        const PerceptionStimulus& stimulus,
        const Vec3& aiPosition,
        float aiHealthPercent)
    {
        float score = 0;

        // 1. 感知类型权重
        switch (stimulus.type) {
            case PERCEPTION_DAMAGE: score += 100.0f; break;   // 伤害最高优先
            case PERCEPTION_VISUAL: score += 50.0f; break;    // 视觉优先
            case PERCEPTION_AUDITORY: score += 30.0f; break;  // 听觉次之
        }

        // 2. 距离因素：越近越重要
        float distance = (stimulus.location - aiPosition).Length();
        score += std::max(0.0f, 50.0f - distance * 2.0f);

        // 3. 刺激强度
        score += stimulus.strength * 20.0f;

        // 4. 自身状态：血量越低越敏感
        score += (1.0f - aiHealthPercent) * 15.0f;

        // 5. 时间因素：越新的刺激越重要
        float age = /*当前时间*/ - stimulus.timestamp;
        score += std::max(0.0f, 10.0f - age);

        return score;
    }

    // 对所有刺激排序
    static std::vector<PerceptionStimulus> Prioritize(
        const std::vector<PerceptionStimulus>& stimuli,
        const Vec3& aiPosition, float aiHealthPercent)
    {
        std::vector<PerceptionStimulus> sorted = stimuli;
        for (auto& s : sorted) {
            s.importance = EvaluateImportance(s, aiPosition, aiHealthPercent);
        }
        std::sort(sorted.begin(), sorted.end(),
            [](const PerceptionStimulus& a, const PerceptionStimulus& b) {
                return a.importance > b.importance;
            });
        return sorted;
    }
};
```

## 7. 团队感知共享

### 7.1 共享机制

团队中的AI共享感知信息，形成集体情报：

```cpp
class TeamPerceptionManager {
    struct TeamMember {
        void* aiEntity;
        PerceptionMemoryManager* perception;
        Vec3 position;
    };

    std::vector<TeamMember> members;
    float shareRadius = 30.0f;  // 感知共享半径

public:
    void AddMember(void* ai, PerceptionMemoryManager* pm) {
        members.push_back({ai, pm, {}});
    }

    // 每帧同步感知信息
    void SyncPerceptions() {
        for (size_t i = 0; i < members.size(); i++) {
            for (size_t j = i + 1; j < members.size(); j++) {
                float dist = (members[i].position - members[j].position).Length();
                if (dist > shareRadius) continue;

                // 双向共享
                ShareMemories(members[i], members[j]);
                ShareMemories(members[j], members[i]);
            }
        }
    }

    void ShareMemories(TeamMember& from, TeamMember& to) {
        auto fromMemories = from.perception->GetAllMemories();
        for (const auto* mem : fromMemories) {
            if (mem->confidence < 0.3f) continue; // 低置信度不共享

            // 共享时置信度打折（二手信息不太可靠）
            float sharedConfidence = mem->confidence * 0.7f;
            // 只共享位置信息，不覆盖视觉置信度
            to.perception->UpdateAuditory(
                mem->target, mem->lastKnownPosition, mem->lastSeenTime);
        }
    }
};
```

public class PerceptionSystem : MonoBehaviour {

    [Header("记忆")]
    public float memoryDecayRate = 0.1f;
    public float forgetThreshold = 0.1f;

    private Dictionary<GameObject, PerceptionMemory> memories =
        new Dictionary<GameObject, PerceptionMemory>();

    void Update() {
        // 视觉检测
        Collider[] visibleTargets = Physics.OverlapSphere(transform.position, sightRange);
        foreach (var col in visibleTargets) {
            if (col.gameObject == gameObject) continue;

            Vector3 dir = (col.transform.position - transform.position).normalized;
            float angle = Vector3.Angle(transform.forward, dir);

            if (angle <= fovAngle) {
                float dist = Vector3.Distance(transform.position, col.transform.position);
                // 遮挡检测
                if (!Physics.Raycast(transform.position, dir, dist, obstacleMask)) {
                    UpdateMemory(col.gameObject, col.transform.position, PerceptionType.Visual);
                }
            }
        }

        // 记忆衰减
        DecayMemories(Time.deltaTime);
    }

    // 外部调用：听到声音
    public void OnHearSound(Vector3 sourcePos, GameObject source) {
        if (memories.ContainsKey(source)) {
            memories[source].lastKnownPosition = sourcePos;
            memories[source].lastHeardTime = Time.time;
        } else {
            memories[source] = new PerceptionMemory {
                target = source,
                lastKnownPosition = sourcePos,
                lastHeardTime = Time.time,
                confidence = 0.5f,
                type = PerceptionType.Auditory
            };
        }
    }

    void UpdateMemory(GameObject target, Vector3 pos, PerceptionType type) {
        if (!memories.ContainsKey(target)) memories[target] = new PerceptionMemory();
        var mem = memories[target];
        mem.target = target;
        mem.lastKnownPosition = pos;
        mem.lastSeenTime = Time.time;
        mem.confidence = 1f;
        mem.type = type;
    }

    void DecayMemories(float dt) {
        List<GameObject> toRemove = new List<GameObject>();
        foreach (var kvp in memories) {
            float timeSince = Time.time - kvp.Value.lastSeenTime;
            if (timeSince > 1f) {
                kvp.Value.confidence -= memoryDecayRate * dt;
            }
            if (kvp.Value.confidence < forgetThreshold) toRemove.Add(kvp.Key);
        }
        foreach (var go in toRemove) memories.Remove(go);
    }

    // Gizmos可视化
    void OnDrawGizmosSelected() {
        Gizmos.color = Color.yellow;
        Gizmos.DrawWireSphere(transform.position, sightRange);

        // 视锥
        Vector3 leftBound = Quaternion.Euler(0, -fovAngle, 0) * transform.forward;
        Vector3 rightBound = Quaternion.Euler(0, fovAngle, 0) * transform.forward;
        Gizmos.DrawRay(transform.position, leftBound * sightRange);
        Gizmos.DrawRay(transform.position, rightBound * sightRange);

        // 听觉范围
        Gizmos.color = Color.cyan;
        Gizmos.DrawWireSphere(transform.position, hearingRange);

        // 记忆可视化
        Gizmos.color = Color.red;
        foreach (var mem in memories.Values) {
            Gizmos.DrawWireSphere(mem.lastKnownPosition, 0.5f * mem.confidence);
        }
    }
}

public enum PerceptionType { Visual, Auditory, Damage }

public class PerceptionMemory {
    public GameObject target;
    public Vector3 lastKnownPosition;
    public float lastSeenTime;
    public float lastHeardTime;
    public float confidence;
    public PerceptionType type;
}

## 9. 性能分析

### 9.1 时间复杂度

| 操作 | 复杂度 | 说明 |
|------|--------|------|
| 视觉检测（单个AI对N个实体） | O(N) | 遍历所有潜在目标 |
| 听觉查询 | O(S) | S为活跃声音事件数 |
| 记忆更新 | O(1) | 哈希表操作 |
| 记忆衰减 | O(M) | M为记忆条目数 |
| 团队共享 | O(M * K) | M为记忆数，K为队友数 |

### 9.2 性能优化策略

1. **分帧处理**：将感知检测分散到多帧执行，每帧只检测一部分AI
2. **LOD降级**：远处AI降低感知检测频率和精度
3. **空间分区**：只检测附近区域的实体
4. **感知缓存**：视觉检测结果缓存，多AI共享同一视野检测
5. **异步射线**：遮挡检测的射线投射异步执行

## 10. 常见陷阱

1. **感知太强（作弊感）**：AI通过墙壁"看到"玩家。解决方案：严格的遮挡检测+感知质量限制
2. **感知太弱**：AI对眼前的事物无反应。解决方案：设置最小感知范围和默认灵敏度
3. **记忆衰减不当**：衰减太快导致AI"秒忘"，衰减太慢导致AI"永不忘记"
4. **性能瓶颈**：大量AI同时执行射线检测。解决方案：分帧+空间分区
5. **团队感知不一致**：队友共享信息导致AI"心灵感应"。解决方案：共享延迟+距离限制+信息衰减
6. **声音事件风暴**：大量枪声同时触发。解决方案：声音合并+强度过滤

## 11. 实际游戏案例

### 案例1：合金装备系列

合金装备的警报系统是感知系统的经典实现：
- 三个警戒等级：无警戒（绿色）、怀疑（黄色）、警报（红色）
- 视觉锥检测 + 听觉范围
- 警卫发现异常后进入调查状态，不直接进入战斗
- 团队感知共享：一个警卫发现异常会通知附近队友
- 玩家可利用噪音制造声东击西

### 案例2：刺客信条系列

守卫的感知系统包含多层检测：
- 视觉锥检测 + 视野距离分级
- 搜索状态：守卫发现异常后进入搜索，逐步扩大搜索范围
- 高处优势：屋顶上的玩家更容易被发现
- 人群掩护：玩家在人群中降低被发现概率
- 感知层级：怀疑 -> 搜索 -> 警报 -> 追击

### 案例3：Left 4 Dead系列

AI Director使用感知系统驱动游戏节奏：
- 感知玩家的位置、血量、弹药、移动速度
- 根据团队状态调整特殊感染者的生成
- 团队压力感知：玩家分散时生成更多威胁
- 记忆系统记录玩家"最近的压力水平"，动态调节难度

### 案例4：F.E.A.R.

F.E.A.R.的感知系统与GOAP深度集成：
- 敌人通过视觉感知获取玩家位置
- 听觉感知允许敌人"听到"枪声并前往调查
- 团队感知共享：敌人通过广播系统汇报玩家位置
- 感知记忆驱动GOAP规划：使用"最后已知位置"进行搜索
