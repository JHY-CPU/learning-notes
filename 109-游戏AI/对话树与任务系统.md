# 对话树与任务系统

## 1. 核心理论

### 1.1 对话系统概述

对话树（Dialogue Tree）是NPC与玩家之间的交互叙事结构，是RPG、冒险游戏和叙事驱动游戏的核心组件。每个对话节点包含文本、说话者、选项和条件分支，玩家通过选择选项推进对话。

对话系统的设计目标：
- **分支叙事**：玩家的选择影响对话走向和剧情发展
- **角色扮演**：玩家通过对话选择塑造角色性格
- **信息传递**：NPC通过对话提供任务、线索和世界观信息
- **情感反馈**：对话系统反映NPC与玩家的关系变化

### 1.2 任务系统概述

任务系统（Quest System）管理游戏中的目标链、进度追踪和奖励发放。任务系统与对话系统紧密耦合：对话触发任务，任务进度影响对话分支。

### 1.3 两者的关系

```
对话系统 ←→ 任务系统
   │            │
   │  ┌─────────┘
   │  │
   ▼  ▼
游戏世界状态（Flags/Variables）
```

- 对话中可以触发任务（"你能帮我找一把剑吗？"）
- 任务进度影响对话选项（"剑已经找到了！"）
- 对话和任务共享全局标记（"已告知NPC真相"）

### 2.1 数据结构

```cpp
#include <string>
#include <vector>
#include <unordered_map>
#include <functional>
#include <memory>
#include <set>

enum class DialogueNodeType {
    Normal,      // 普通对话
    Branch,      // 条件分支
    Event,       // 触发事件
    End          // 对话结束
};

struct DialogueCondition {
    std::function<bool()> check;
    std::string description; // 调试用

    bool Evaluate() const { return check ? check() : true; }
};

struct DialogueEffect {
    std::function<void()> execute;
    std::string description;

    void Apply() const { if (execute) execute(); }
};

struct DialogueOption {
    int id;
    std::string text;                          // 显示文本
    int nextNodeId;                            // 下一个节点ID
    std::vector<DialogueCondition> conditions; // 显示条件（全部满足才显示）
    std::vector<DialogueEffect> effects;       // 选择后的效果
    bool bEnabled = true;                      // 是否可选
    bool bOnceOnly = false;                    // 是否只能选一次
    bool bHasBeenChosen = false;               // 运行时：是否已被选择

    bool IsAvailable() const {
        if (!bEnabled) return false;
        if (bOnceOnly && bHasBeenChosen) return false;
        for (const auto& cond : conditions) {
            if (!cond.Evaluate()) return false;
        }
        return true;
    }
};

struct DialogueNode {
    int id;
    DialogueNodeType type;
    std::string speaker;                      // 说话者名称
    std::string text;                         // 对话文本
    std::vector<DialogueOption> options;      // 选项列表
    std::vector<DialogueEffect> enterEffects; // 进入节点时执行的效果
    std::string emotion;                      // 表情/语气标记
    std::string voiceLine;                    // 配音文件路径
    float autoAdvanceTime = -1;               // 自动推进时间（-1不自动推进）

    // 条件分支：按条件自动选择下一个节点
    struct ConditionalBranch {
        DialogueCondition condition;
        int targetNodeId;
    };
    std::vector<ConditionalBranch> branches;
};
```

### 2.2 对话引擎

```cpp
class DialogueEngine {
    std::unordered_map<int, DialogueNode> nodes;
    int currentNodeId = -1;
    int startNodeId = -1;
    bool bIsActive = false;

    // 回调
    std::function<void(const std::string& speaker, const std::string& text,
                       const std::string& emotion)> onShowText;
    std::function<void(const std::vector<DialogueOption*>& options)> onShowOptions;
    std::function<void()> onDialogueEnd;
    std::function<void(int nodeId)> onNodeEnter;

public:
    void LoadFromData(const std::vector<DialogueNode>& dialogueNodes, int startId) {
        nodes.clear();
        for (const auto& node : dialogueNodes) {
            nodes[node.id] = node;
        }
        startNodeId = startId;
    }

    void Start() {
        if (!nodes.count(startNodeId)) return;
        bIsActive = true;
        EnterNode(startNodeId);
    }

    void EnterNode(int nodeId) {
        if (!nodes.count(nodeId)) {
            EndDialogue();
            return;
        }

        currentNodeId = nodeId;
        DialogueNode& node = nodes[nodeId];

        // 执行进入效果
        for (const auto& effect : node.enterEffects) {
            effect.Apply();
        }

        if (onNodeEnter) onNodeEnter(nodeId);

        // 检查条件分支
        if (!node.branches.empty()) {
            for (const auto& branch : node.branches) {
                if (branch.condition.Evaluate()) {
                    EnterNode(branch.targetNodeId);
                    return;
                }
            }
        }

        // 显示文本
        if (onShowText) {
            onShowText(node.speaker, node.text, node.emotion);
        }

        // 收集可用选项
        std::vector<DialogueOption*> availableOptions;
        for (auto& opt : node.options) {
            if (opt.IsAvailable()) {
                availableOptions.push_back(&opt);
            }
        }

        if (availableOptions.empty()) {
            // 无可用选项：自动结束或进入默认分支
            if (node.type == DialogueNodeType::End) {
                EndDialogue();
            }
        } else {
            if (onShowOptions) onShowOptions(availableOptions);
        }
    }

    void SelectOption(int optionId) {
        if (!bIsActive || currentNodeId < 0) return;

        DialogueNode& node = nodes[currentNodeId];
        for (auto& opt : node.options) {
            if (opt.id == optionId && opt.IsAvailable()) {
                opt.bHasBeenChosen = true;

                // 执行选项效果
                for (const auto& effect : opt.effects) {
                    effect.Apply();
                }

                // 进入下一个节点
                if (opt.nextNodeId >= 0) {
                    EnterNode(opt.nextNodeId);
                } else {
                    EndDialogue();
                }
                return;
            }
        }
    }

    void EndDialogue() {
        bIsActive = false;
        currentNodeId = -1;
        if (onDialogueEnd) onDialogueEnd();
    }

    // 设置回调
    void SetOnShowText(std::function<void(const std::string&, const std::string&,
                                          const std::string&)> cb) { onShowText = cb; }
    void SetOnShowOptions(std::function<void(const std::vector<DialogueOption*>&)> cb) { onShowOptions = cb; }
    void SetOnDialogueEnd(std::function<void()> cb) { onDialogueEnd = cb; }
    void SetOnNodeEnter(std::function<void(int)> cb) { onNodeEnter = cb; }

    bool IsActive() const { return bIsActive; }
    int GetCurrentNodeId() const { return currentNodeId; }
};
```

## 3. 任务系统

### 3.1 任务数据结构

```cpp
enum class QuestState {
    Locked,         // 未解锁
    Available,      // 可接取
    Active,         // 进行中
    Completed,      // 已完成（目标全部达成）
    TurnedIn,       // 已交付
    Failed          // 已失败
};

enum class ObjectiveType {
    Kill,           // 击杀
    Collect,        // 收集
    Reach,          // 到达
    Talk,           // 对话
    Escort,         // 护送
    Survive,        // 生存
    Custom          // 自定义
};

struct QuestObjective {
    std::string id;
    std::string description;
    ObjectiveType type;
    std::string targetId;     // 目标ID（怪物类型/物品ID/NPC ID等）
    int requiredAmount;       // 需要完成的数量
    int currentAmount = 0;    // 当前进度
    bool bOptional = false;   // 是否为可选目标
    bool bHidden = false;     // 是否对玩家隐藏
    bool bCompleted = false;

    float GetProgress() const {
        return requiredAmount > 0 ?
            (float)currentAmount / requiredAmount : 1.0f;
    }

    bool IsComplete() const { return currentAmount >= requiredAmount; }
};

struct QuestReward {
    int gold = 0;
    int experience = 0;
    std::vector<std::string> items;       // 物品ID列表
    std::vector<std::string> unlockFlags; // 解锁的标记
};

struct Quest {
    std::string id;
    std::string title;
    std::string description;
    QuestState state = QuestState::Locked;
    int level;                    // 推荐等级
    bool bMainQuest;              // 是否为主线任务
    bool bRepeatable;             // 是否可重复

    std::vector<QuestObjective> objectives;
    QuestReward reward;

    // 前置条件
    std::vector<std::string> prerequisiteQuestIds; // 前置任务
    std::vector<std::string> prerequisiteFlags;    // 前置标记
    int minLevel = 0;                              // 最低等级

    // 任务链
    std::string nextQuestId;      // 完成后自动触发的任务

    // 失败条件
    float timeLimit = -1;         // 时间限制（秒），-1=无限制
    bool bFailOnPlayerDeath = false;

    bool ArePrerequisitesMet(const std::set<std::string>& completedQuests,
                             const std::set<std::string>& flags,
                             int playerLevel) const {
        if (playerLevel < minLevel) return false;
        for (const auto& preReq : prerequisiteQuestIds) {
            if (!completedQuests.count(preReq)) return false;
        }
        for (const auto& flag : prerequisiteFlags) {
            if (!flags.count(flag)) return false;
        }
        return true;
    }

    void UpdateProgress(ObjectiveType type, const std::string& targetId, int amount = 1) {
        for (auto& obj : objectives) {
            if (obj.type == type && obj.targetId == targetId && !obj.IsComplete()) {
                obj.currentAmount = std::min(obj.currentAmount + amount, obj.requiredAmount);
                if (obj.IsComplete()) obj.bCompleted = true;
            }
        }
        // 检查所有非可选目标是否完成
        bool allDone = true;
        for (const auto& obj : objectives) {
            if (!obj.bOptional && !obj.IsComplete()) { allDone = false; break; }
        }
        if (allDone) state = QuestState::Completed;
    }
};
```

### 3.2 任务管理器

```cpp
class QuestManager {
    std::unordered_map<std::string, Quest> questDatabase;  // 所有任务定义
    std::set<std::string> completedQuests;
    std::set<std::string> globalFlags;
    int playerLevel = 1;

    // 回调
    std::function<void(const Quest&)> onQuestAvailable;
    std::function<void(const Quest&)> onQuestAccepted;
    std::function<void(const Quest&)> onQuestCompleted;
    std::function<void(const Quest&)> onQuestTurnedIn;
    std::function<void(const Quest&)> onQuestFailed;
    std::function<void(const QuestObjective&)> onObjectiveCompleted;

public:
    void RegisterQuest(const Quest& quest) {
        questDatabase[quest.id] = quest;
    }

    void Initialize() {
        // 检查哪些任务变为可用
        for (auto& [id, quest] : questDatabase) {
            if (quest.state == QuestState::Locked) {
                if (quest.ArePrerequisitesMet(completedQuests, globalFlags, playerLevel)) {
                    quest.state = QuestState::Available;
                    if (onQuestAvailable) onQuestAvailable(quest);
                }
            }
        }
    }

    bool AcceptQuest(const std::string& questId) {
        auto it = questDatabase.find(questId);
        if (it == questDatabase.end()) return false;
        Quest& quest = it->second;
        if (quest.state != QuestState::Available) return false;

        quest.state = QuestState::Active;
        if (onQuestAccepted) onQuestAccepted(quest);
        return true;
    }

    void UpdateObjective(ObjectiveType type, const std::string& targetId, int amount = 1) {
        for (auto& [id, quest] : questDatabase) {
            if (quest.state != QuestState::Active) continue;

            // 记录更新前完成的目标
            std::set<std::string> beforeCompleted;
            for (const auto& obj : quest.objectives) {
                if (obj.bCompleted) beforeCompleted.insert(obj.id);
            }

            quest.UpdateProgress(type, targetId, amount);

            // 通知新完成的目标
            for (const auto& obj : quest.objectives) {
                if (obj.bCompleted && !beforeCompleted.count(obj.id)) {
                    if (onObjectiveCompleted) onObjectiveCompleted(obj);
                }
            }

            if (quest.state == QuestState::Completed) {
                if (onQuestCompleted) onQuestCompleted(quest);
            }
        }
    }

    bool TurnInQuest(const std::string& questId) {
        auto it = questDatabase.find(questId);
        if (it == questDatabase.end()) return false;
        Quest& quest = it->second;
        if (quest.state != QuestState::Completed) return false;

        quest.state = QuestState::TurnedIn;
        completedQuests.insert(questId);

        // 发放奖励
        GiveReward(quest.reward);

        // 解锁后续任务
        if (!quest.nextQuestId.empty()) {
            auto nextIt = questDatabase.find(quest.nextQuestId);
            if (nextIt != questDatabase.end()) {
                nextIt->second.state = QuestState::Available;
                if (onQuestAvailable) onQuestAvailable(nextIt->second);
            }
        }

        // 检查是否有新任务变为可用
        Initialize();

        if (onQuestTurnedIn) onQuestTurnedIn(quest);
        return true;
    }

    void AddFlag(const std::string& flag) { globalFlags.insert(flag); }
    bool HasFlag(const std::string& flag) const { return globalFlags.count(flag) > 0; }

    std::vector<const Quest*> GetActiveQuests() const {
        std::vector<const Quest*> result;
        for (const auto& [id, q] : questDatabase) {
            if (q.state == QuestState::Active) result.push_back(&q);
        }
        return result;
    }

private:
    void GiveReward(const QuestReward& reward) {
        // 发金币、经验、物品
        // 实际实现需要集成经济系统和背包系统
    }
};
```

### 4.1 标记管理

```cpp
class FlagManager {
    std::unordered_map<std::string, bool> boolFlags;
    std::unordered_map<std::string, int> intFlags;
    std::unordered_map<std::string, float> floatFlags;
    std::unordered_map<std::string, std::string> stringFlags;

public:
    // 布尔标记
    void SetBool(const std::string& key, bool value) { boolFlags[key] = value; }
    bool GetBool(const std::string& key, bool def = false) const {
        auto it = boolFlags.find(key);
        return it != boolFlags.end() ? it->second : def;
    }

    // 整数标记
    void SetInt(const std::string& key, int value) { intFlags[key] = value; }
    int GetInt(const std::string& key, int def = 0) const {
        auto it = intFlags.find(key);
        return it != intFlags.end() ? it->second : def;
    }
    void IncrementInt(const std::string& key, int amount = 1) {
        intFlags[key] += amount;
    }

    // 好感度系统
    void ModifyDisposition(const std::string& npcId, int delta) {
        IncrementInt("disposition_" + npcId, delta);
    }
    int GetDisposition(const std::string& npcId) const {
        return GetInt("disposition_" + npcId, 0);
    }

    // 序列化（存档）
    void Serialize(std::ostream& os) const {
        // 简化示例
        for (const auto& [k, v] : boolFlags) os << "B:" << k << "=" << v << "\n";
        for (const auto& [k, v] : intFlags) os << "I:" << k << "=" << v << "\n";
    }

    void Deserialize(std::istream& is) {
        std::string line;
        while (std::getline(is, line)) {
            // 解析格式 "B:key=value" 或 "I:key=value"
            size_t colonPos = line.find(':');
            size_t eqPos = line.find('=');
            if (colonPos == std::string::npos || eqPos == std::string::npos) continue;

            char type = line[0];
            std::string key = line.substr(colonPos + 1, eqPos - colonPos - 1);
            std::string val = line.substr(eqPos + 1);

            if (type == 'B') boolFlags[key] = (val == "1");
            else if (type == 'I') intFlags[key] = std::stoi(val);
        }
    }
};
```

### 5.1 条件构建器

```cpp
class ConditionBuilder {
public:
    static DialogueCondition HasItem(const std::string& itemId, int amount = 1) {
        return {
            [itemId, amount]() {
                return playerInventory->GetItemCount(itemId) >= amount;
            },
            "拥有 " + itemId + " x" + std::to_string(amount)
        };
    }

    static DialogueCondition HasFlag(const std::string& flag) {
        return {
            [flag]() { return flagManager->HasFlag(flag); },
            "标记: " + flag
        };
    }

    static DialogueCondition NotHasFlag(const std::string& flag) {
        return {
            [flag]() { return !flagManager->HasFlag(flag); },
            "无标记: " + flag
        };
    }

    static DialogueCondition MinLevel(int level) {
        return {
            [level]() { return playerLevel >= level; },
            "等级 >= " + std::to_string(level)
        };
    }

    static DialogueCondition MinDisposition(const std::string& npcId, int min) {
        return {
            [npcId, min]() {
                return flagManager->GetDisposition(npcId) >= min;
            },
            npcId + " 好感度 >= " + std::to_string(min)
        };
    }

    static DialogueCondition QuestState(const std::string& questId, QuestState state) {
        return {
            [questId, state]() {
                // 检查任务状态
                return questManager->GetQuestState(questId) == state;
            },
            "任务 " + questId + " 状态匹配"
        };
    }

    // 组合条件
    static DialogueCondition And(DialogueCondition a, DialogueCondition b) {
        return {
            [a = std::move(a), b = std::move(b)]() {
                return a.Evaluate() && b.Evaluate();
            },
            "(" + a.description + " AND " + b.description + ")"
        };
    }

    static DialogueCondition Or(DialogueCondition a, DialogueCondition b) {
        return {
            [a = std::move(a), b = std::move(b)]() {
                return a.Evaluate() || b.Evaluate();
            },
            "(" + a.description + " OR " + b.description + ")"
        };
    }
};
```

## 6. 存档系统集成

```cpp
class DialogueSaveData {
public:
    struct NodeState {
        int nodeId;
        std::set<int> chosenOptionIds; // 已选择的选项
        bool bVisited = false;
    };

    std::unordered_map<int, NodeState> nodeStates;

    void RecordChoice(int nodeId, int optionId) {
        nodeStates[nodeId].nodeId = nodeId;
        nodeStates[nodeId].chosenOptionIds.insert(optionId);
        nodeStates[nodeId].bVisited = true;
    }

    bool HasChosen(int nodeId, int optionId) const {
        auto it = nodeStates.find(nodeId);
        if (it == nodeStates.end()) return false;
        return it->second.chosenOptionIds.count(optionId) > 0;
    }

    bool HasVisited(int nodeId) const {
        auto it = nodeStates.find(nodeId);
        return it != nodeStates.end() && it->second.bVisited;
    }
};
```

## 8. 性能分析

### 8.1 时间复杂度

| 操作 | 复杂度 | 说明 |
|------|--------|------|
| 进入节点 | O(O*C) | O=选项数，C=条件数 |
| 条件评估 | O(1) | 每个条件 |
| 任务进度更新 | O(Q*O) | Q=进行中任务数，O=目标数 |
| 标记查询 | O(1) | 哈希表 |

### 8.2 内存分析

| 组件 | 大小 |
|------|------|
| 对话节点 | 128-512字节（含文本） |
| 任务定义 | 256-1024字节（含目标） |
| 标记条目 | 32-64字节 |

## 9. 常见陷阱

1. **对话节点引用断裂**：删除节点后未更新引用该节点的选项
2. **条件循环依赖**：条件A依赖条件B的结果，B依赖A
3. **任务进度不匹配**：任务要求击杀10只怪，但对话只检查"已杀5只"
4. **存档数据不完整**：动态生成的对话节点未正确序列化
5. **本地化问题**：对话文本硬编码，无法支持多语言
6. **内存泄漏**：对话中的Lambda捕获了外部对象的裸指针

## 10. 实际游戏案例

### 案例1：巫师3

巫师3的对话系统是业界标杆：
- 复杂的对话分支影响剧情走向和任务结局
- "灰色选择"：没有绝对对错的选项
- 对话选择影响角色关系和世界状态
- 任务系统支持多结局链：同一任务不同完成方式触发不同后续

### 案例2：上古卷轴5

天际的对话系统特点：
- Radiant系统动态生成任务目标（"在X地杀Y只Z怪"）
- 对话选项与技能/属性关联（说服、威吓、贿赂）
- NPC对话根据玩家种族/装备/声望动态调整
- Radiant Story系统根据玩家行为生成个性化任务

### 案例3：质量效应系列

质量效应的Paragon/Renegade系统：
- 对话轮盘（上善下恶中间中立）
- 好感度系统：队友对话选项影响忠诚度
- 关键选择影响续作剧情（跨游戏状态传承）
- 中断系统：实时对话中的快速行动选项

### 案例4：Undertale

Undertale的对话系统创新：
- 对话选择直接影响战斗系统
- "仁慈"选项在战斗中通过对话触发
- 每个NPC都有独特的对话风格
- 游戏全程追踪玩家的"杀戮值"，影响所有对话和结局
