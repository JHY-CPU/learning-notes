# 决策树与效用系统

## 1. 核心理论

### 1.1 决策树基础

决策树（Decision Tree）是一种树形决策模型，每个内部节点代表一个属性测试，每个分支代表测试结果，每个叶节点代表最终决策。决策树在游戏AI中用于离散条件下的分类决策。

### 1.2 效用系统基础

效用系统（Utility AI）是一种基于评分的决策模型，为每个候选行为计算一个效用分数（Utility Score），选择分数最高的行为执行。效用系统的哲学是：**AI的行为不应该是"对或错"，而应该是"好到什么程度"**。

核心概念：
- **效用（Utility）**：行为的"期望价值"，归一化到[0,1]区间
- **考虑因素（Consideration）**：影响效用的单一因素（如血量、距离、弹药）
- **响应曲线（Response Curve）**：将输入映射到[0,1]的函数
- **评分函数**：综合所有考虑因素计算最终效用

### 1.3 效用系统 vs 其他AI模型

| 维度 | FSM | 行为树 | 效用系统 |
|------|-----|--------|----------|
| 决策逻辑 | 条件分支 | 树形评估 | 连续评分 |
| 新增行为 | 添加状态+转换 | 添加节点 | 添加行为+考虑因素 |
| 多因素处理 | 需要复合条件 | 需要嵌套选择器 | 天然支持多维加权 |
| 调参难度 | 中 | 中 | 低（曲线可调） |
| 可解释性 | 高 | 高 | 中（需要可视化工具） |
| 适用场景 | 简单角色 | 结构化行为 | 复杂决策、NPC需求 |

## 2. 决策树

### 2.1 决策树结构

```
[生命值 < 30%?]
├── Yes -> [有掩体附近?]
│   ├── Yes -> [队友在附近?]
│   │   ├── Yes -> 请求支援
│   │   └── No -> 撤退到掩体
│   └── No -> 冲刺撤退
└── No -> [弹药 > 20%?]
    ├── Yes -> [敌人距离 < 10m?]
    │   ├── Yes -> 近战攻击
    │   └── No -> 射击
    └── No -> 搜索弹药
```

### 2.2 决策树实现

```cpp
#include <string>
#include <functional>
#include <vector>
#include <memory>

// 决策树节点基类
class DecisionNode {
public:
    virtual ~DecisionNode() = default;
    virtual std::string Evaluate(class AIBlackboard& bb) = 0;
};

// 条件分支节点
class ConditionNode : public DecisionNode {
    std::function<bool(AIBlackboard&)> condition;
    std::unique_ptr<DecisionNode> trueBranch;
    std::unique_ptr<DecisionNode> falseBranch;

public:
    ConditionNode(std::function<bool(AIBlackboard&)> cond,
                  std::unique_ptr<DecisionNode> trueNode,
                  std::unique_ptr<DecisionNode> falseNode)
        : condition(std::move(cond)),
          trueBranch(std::move(trueNode)),
          falseBranch(std::move(falseNode)) {}

    std::string Evaluate(AIBlackboard& bb) override {
        if (condition(bb)) {
            return trueBranch ? trueBranch->Evaluate(bb) : "";
        } else {
            return falseBranch ? falseBranch->Evaluate(bb) : "";
        }
    }
};

// 行动叶节点
class ActionLeaf : public DecisionNode {
    std::string actionName;
    std::function<void()> action;

public:
    ActionLeaf(const std::string& name, std::function<void()> act = nullptr)
        : actionName(name), action(std::move(act)) {}

    std::string Evaluate(AIBlackboard& bb) override {
        if (action) action();
        return actionName;
    }
};

// 决策树构建器
class DecisionTreeBuilder {
public:
    static std::unique_ptr<ConditionNode> If(
        std::function<bool(AIBlackboard&)> cond,
        std::unique_ptr<DecisionNode> trueNode,
        std::unique_ptr<DecisionNode> falseNode)
    {
        return std::make_unique<ConditionNode>(std::move(cond),
            std::move(trueNode), std::move(falseNode));
    }

    static std::unique_ptr<ActionLeaf> Do(const std::string& name,
                                           std::function<void()> act = nullptr) {
        return std::make_unique<ActionLeaf>(name, std::move(act));
    }
};

// 使用示例
auto combatTree = DecisionTreeBuilder::If(
    [](AIBlackboard& bb) { return bb.GetFloat("health") < 0.3f; },
    // 血量低分支
    DecisionTreeBuilder::If(
        [](AIBlackboard& bb) { return bb.GetBool("nearCover"); },
        DecisionTreeBuilder::Do("撤退到掩体"),
        DecisionTreeBuilder::Do("冲刺撤退")
    ),
    // 血量正常分支
    DecisionTreeBuilder::If(
        [](AIBlackboard& bb) { return bb.GetFloat("ammo") > 0.2f; },
        DecisionTreeBuilder::If(
            [](AIBlackboard& bb) { return bb.GetFloat("distanceToEnemy") < 10.0f; },
            DecisionTreeBuilder::Do("近战攻击"),
            DecisionTreeBuilder::Do("射击")
        ),
        DecisionTreeBuilder::Do("搜索弹药")
    )
);
```

### 2.3 连续值决策树（Regression Tree）

决策树可以处理连续值输入，用于预测或评估：

```cpp
class RegressionNode {
public:
    virtual float Predict(const std::vector<float>& features) = 0;
};

class SplitNode : public RegressionNode {
    int featureIndex;
    float threshold;
    std::unique_ptr<RegressionNode> leftChild;   // <= threshold
    std::unique_ptr<RegressionNode> rightChild;  // > threshold

public:
    SplitNode(int idx, float thresh,
              std::unique_ptr<RegressionNode> left,
              std::unique_ptr<RegressionNode> right)
        : featureIndex(idx), threshold(thresh),
          leftChild(std::move(left)), rightChild(std::move(right)) {}

    float Predict(const std::vector<float>& features) override {
        if (features[featureIndex] <= threshold) {
            return leftChild->Predict(features);
        }
        return rightChild->Predict(features);
    }
};

class LeafNode : public RegressionNode {
    float value;
public:
    LeafNode(float v) : value(v) {}
    float Predict(const std::vector<float>&) override { return value; }
};
```

## 3. 效用系统核心

### 3.1 响应曲线（Response Curve）

响应曲线将原始输入值映射到[0,1]区间，是效用系统的核心组件。

```cpp
class ResponseCurve {
public:
    virtual ~ResponseCurve() = default;
    virtual float Evaluate(float x) const = 0;
    virtual std::string GetName() const = 0;
};

// 线性曲线：y = x
class LinearCurve : public ResponseCurve {
    float slope, intercept;
public:
    LinearCurve(float s = 1.0f, float i = 0.0f) : slope(s), intercept(i) {}
    float Evaluate(float x) const override {
        return std::max(0.0f, std::min(1.0f, slope * x + intercept));
    }
    std::string GetName() const override { return "Linear"; }
};

// 指数曲线：y = x^n（放大高值差异）
class PolynomialCurve : public ResponseCurve {
    float exponent;
public:
    PolynomialCurve(float exp = 2.0f) : exponent(exp) {}
    float Evaluate(float x) const override {
        x = std::max(0.0f, std::min(1.0f, x));
        return powf(x, exponent);
    }
    std::string GetName() const override { return "Polynomial"; }
};

// 对数曲线：y = log(x+1)/log(2)（弱化极端值）
class LogCurve : public ResponseCurve {
    float base;
public:
    LogCurve(float b = 2.0f) : base(b) {}
    float Evaluate(float x) const override {
        x = std::max(0.0f, std::min(1.0f, x));
        return logf(x * (base - 1.0f) + 1.0f) / logf(base);
    }
    std::string GetName() const override { return "Logarithmic"; }
};

// S曲线（Sigmoid）：模拟阈值效应
class SCurve : public ResponseCurve {
    float midpoint, steepness;
public:
    SCurve(float mid = 0.5f, float k = 10.0f) : midpoint(mid), steepness(k) {}
    float Evaluate(float x) const override {
        x = std::max(0.0f, std::min(1.0f, x));
        return 1.0f / (1.0f + expf(-steepness * (x - midpoint)));
    }
    std::string GetName() const override { return "Sigmoid"; }
};

// 反向S曲线：低值区域敏感，高值区域饱和
class InverseSCurve : public ResponseCurve {
    float midpoint, steepness;
public:
    InverseSCurve(float mid = 0.5f, float k = 10.0f) : midpoint(mid), steepness(k) {}
    float Evaluate(float x) const override {
        x = std::max(0.0f, std::min(1.0f, x));
        return 1.0f - (1.0f / (1.0f + expf(-steepness * (x - midpoint))));
    }
    std::string GetName() const override { return "InverseSigmoid"; }
};

// 自定义曲线（分段线性）
class PiecewiseLinearCurve : public ResponseCurve {
    std::vector<std::pair<float, float>> points; // (x, y) 控制点
public:
    PiecewiseLinearCurve(std::vector<std::pair<float, float>> pts)
        : points(std::move(pts)) {
        std::sort(points.begin(), points.end(),
                  [](auto& a, auto& b) { return a.first < b.first; });
    }

    float Evaluate(float x) const override {
        x = std::max(0.0f, std::min(1.0f, x));
        if (x <= points.front().first) return points.front().second;
        if (x >= points.back().first) return points.back().second;

        for (size_t i = 0; i < points.size() - 1; i++) {
            if (x >= points[i].first && x <= points[i+1].first) {
                float t = (x - points[i].first) / (points[i+1].first - points[i].first);
                return points[i].second + t * (points[i+1].second - points[i].second);
            }
        }
        return 0.0f;
    }
    std::string GetName() const override { return "PiecewiseLinear"; }
};
```

## 4. 考虑因素架构

### 4.1 Consideration类

```cpp
class Consideration {
    std::string name;
    std::function<float()> inputGetter;  // 获取输入值的回调
    std::unique_ptr<ResponseCurve> curve; // 响应曲线
    float weight;                         // 权重
    bool bInvert;                         // 是否反转

public:
    Consideration(const std::string& n,
                  std::function<float()> getter,
                  std::unique_ptr<ResponseCurve> c,
                  float w = 1.0f,
                  bool invert = false)
        : name(n), inputGetter(std::move(getter)),
          curve(std::move(c)), weight(w), bInvert(invert) {}

    // 获取评分
    float Score() const {
        float input = inputGetter();
        input = std::max(0.0f, std::min(1.0f, input)); // 归一化到[0,1]
        if (bInvert) input = 1.0f - input;
        return curve->Evaluate(input) * weight;
    }

    const std::string& GetName() const { return name; }
    float GetRawInput() const { return inputGetter(); }
    std::string GetCurveName() const { return curve->GetName(); }
};
```

### 4.2 行为类

```cpp
class UtilityAction {
    std::string name;
    std::vector<std::unique_ptr<Consideration>> considerations;
    std::function<void()> executeCallback;

    // 评分策略
    enum class ScoringMethod {
        Multiply,     // 连乘（默认）
        Average,      // 平均
        Max,          // 最高分
        Min,          // 最低分
        WeightedAvg   // 加权平均
    };
    ScoringMethod scoringMethod = ScoringMethod::Multiply;

    // 运行时状态
    float lastScore = 0;
    bool bIsActive = false;

public:
    UtilityAction(const std::string& n) : name(n) {}

    void AddConsideration(std::unique_ptr<Consideration> c) {
        considerations.push_back(std::move(c));
    }

    void SetScoringMethod(ScoringMethod method) { scoringMethod = method; }

    // 计算效用分数
    float Score() const {
        if (considerations.empty()) return 0.0f;

        float result = 0;

        switch (scoringMethod) {
            case ScoringMethod::Multiply: {
                // 连乘：任一因素为0则整体为0
                result = 1.0f;
                // 补偿因子：降低连乘导致的过度惩罚
                float compensationFactor = 1.0f / (1.0f + (0.05f * considerations.size()));

                for (const auto& c : considerations) {
                    float s = c->Score();
                    float modification = (1.0f - s) * compensationFactor;
                    result *= (1.0f - modification);
                }
                break;
            }

            case ScoringMethod::Average: {
                for (const auto& c : considerations) result += c->Score();
                result /= considerations.size();
                break;
            }

            case ScoringMethod::Max: {
                result = 0;
                for (const auto& c : considerations)
                    result = std::max(result, c->Score());
                break;
            }

            case ScoringMethod::Min: {
                result = 1.0f;
                for (const auto& c : considerations)
                    result = std::min(result, c->Score());
                break;
            }
        }

        lastScore = result;
        return result;
    }

    void Execute() {
        if (executeCallback) executeCallback();
    }

    const std::string& GetName() const { return name; }
    float GetLastScore() const { return lastScore; }
};
```

### 4.3 效用选择器

```cpp
class UtilitySelector {
    std::vector<std::unique_ptr<UtilityAction>> actions;
    UtilityAction* currentAction = nullptr;
    float switchThreshold = 0.1f; // 切换阈值（防抖振）

public:
    void AddAction(std::unique_ptr<UtilityAction> action) {
        actions.push_back(std::move(action));
    }

    // 评估并选择最优行为
    UtilityAction* SelectBest() {
        UtilityAction* best = nullptr;
        float bestScore = 0;

        // 计分
        for (auto& action : actions) {
            float score = action->Score();
            if (score > bestScore) {
                bestScore = score;
                best = action.get();
            }
        }

        // 防抖振：当前行为分数仍然足够高则不切换
        if (currentAction && best != currentAction) {
            float currentScore = currentAction->GetLastScore();
            if (currentScore + switchThreshold > bestScore) {
                return currentAction; // 保持当前行为
            }
        }

        currentAction = best;
        return best;
    }

    // 获取所有行为分数（调试用）
    struct ScoreInfo {
        std::string name;
        float score;
        std::vector<std::pair<std::string, float>> considerations;
    };

    std::vector<ScoreInfo> GetAllScores() const {
        std::vector<ScoreInfo> result;
        for (const auto& action : actions) {
            ScoreInfo info;
            info.name = action->GetName();
            info.score = action->GetLastScore();
            result.push_back(info);
        }
        std::sort(result.begin(), result.end(),
                  [](const ScoreInfo& a, const ScoreInfo& b) {
                      return a.score > b.score;
                  });
        return result;
    }

    UtilityAction* GetCurrentAction() const { return currentAction; }
};
```

## 5. 完整示例：敌人AI效用系统

```cpp
class EnemyAI {
    UtilitySelector selector;
    AIBlackboard blackboard;

public:
    void Setup() {
        // 射击行为
        auto shoot = std::make_unique<UtilityAction>("射击");
        shoot->AddConsideration(std::make_unique<Consideration>(
            "有弹药",
            [this]() { return blackboard.GetFloat("ammoPercent"); },
            std::make_unique<SCurve>(0.2f, 10.0f),
            1.0f
        ));
        shoot->AddConsideration(std::make_unique<Consideration>(
            "敌人可见",
            [this]() { return blackboard.GetBool("canSeeEnemy") ? 1.0f : 0.0f; },
            std::make_unique<LinearCurve>(),
            1.5f
        ));
        shoot->AddConsideration(std::make_unique<Consideration>(
            "距离适中",
            [this]() {
                float dist = blackboard.GetFloat("distanceToEnemy");
                // 5-20米最佳距离
                if (dist < 5.0f) return dist / 5.0f;
                if (dist < 20.0f) return 1.0f;
                return 1.0f - (dist - 20.0f) / 30.0f;
            },
            std::make_piecewise_linear({{0,0},{0.2,0.5},{0.5,1},{0.8,0.8},{1,0.3}}),
            1.0f
        ));
        shoot->SetScoringMethod(UtilityAction::ScoringMethod::Multiply);
        selector.AddAction(std::move(shoot));

        // 撤退行为
        auto retreat = std::make_unique<UtilityAction>("撤退");
        retreat->AddConsideration(std::make_unique<Consideration>(
            "血量低",
            [this]() { return 1.0f - blackboard.GetFloat("healthPercent"); },
            std::make_unique<PolynomialCurve>(2.0f), // 血量越低，效用增长越快
            2.0f
        ));
        retreat->AddConsideration(std::make_unique<Consideration>(
            "附近有掩体",
            [this]() { return blackboard.GetBool("nearCover") ? 1.0f : 0.3f; },
            std::make_unique<SCurve>(0.5f, 8.0f),
            0.8f
        ));
        selector.AddAction(std::move(retreat));

        // 装弹行为
        auto reload = std::make_unique<UtilityAction>("装弹");
        reload->AddConsideration(std::make_unique<Consideration>(
            "弹药不足",
            [this]() { return 1.0f - blackboard.GetFloat("ammoPercent"); },
            std::make_unique<InverseSCurve>(0.3f, 10.0f),
            1.0f
        ));
        reload->AddConsideration(std::make_unique<Consideration>(
            "安全装弹",
            [this]() { return blackboard.GetBool("inSafeZone") ? 1.0f : 0.4f; },
            std::make_unique<LinearCurve>(),
            0.6f
        ));
        selector.AddAction(std::move(reload));

        // 巡逻行为（默认行为，始终有低分数）
        auto patrol = std::make_unique<UtilityAction>("巡逻");
        patrol->AddConsideration(std::make_unique<Consideration>(
            "默认巡逻",
            []() { return 0.3f; }, // 恒定低分数
            std::make_unique<LinearCurve>(),
            1.0f
        ));
        selector.AddAction(std::move(patrol));
    }

    void Update(float dt) {
        // 更新黑板
        UpdateBlackboard(dt);

        // 选择行为
        UtilityAction* best = selector.SelectBest();
        if (best) {
            best->Execute();
        }
    }

    void UpdateBlackboard(float dt) {
        // 更新感知数据到黑板
        blackboard.SetFloat("healthPercent", GetHealthPercent());
        blackboard.SetFloat("ammoPercent", GetAmmoPercent());
        blackboard.SetFloat("distanceToEnemy", GetDistanceToEnemy());
        blackboard.SetBool("canSeeEnemy", CanSeeEnemy());
        blackboard.SetBool("nearCover", IsNearCover());
        blackboard.SetBool("inSafeZone", IsInSafeZone());
    }
};
```

## 6. 考虑因素的数学组合

### 6.1 连乘补偿

标准连乘有一个问题：考虑因素越多，分数越低。解决方案是使用补偿因子：

```cpp
// 标准连乘（有偏差）
float score = 1.0f;
for (auto& c : considerations) score *= c->Score();
// 问题：3个0.5分 → 0.125，4个0.5分 → 0.0625

// 补偿连乘
float score = 1.0f;
float compensation = 1.0f / (1.0f + 0.05f * considerations.size());
for (auto& c : considerations) {
    float s = c->Score();
    float mod = (1.0f - s) * compensation;
    score *= (1.0f - mod);
}
// 3个0.5分 → 0.41，4个0.5分 → 0.37（更合理）
```

### 6.2 不同组合方式的效果

```
输入：[0.8, 0.6, 0.4]（三个考虑因素的分数）

连乘：    0.8 * 0.6 * 0.4 = 0.192（惩罚重）
平均：    (0.8+0.6+0.4) / 3 = 0.600
最高：    max(0.8,0.6,0.4) = 0.800
最低：    min(0.8,0.6,0.4) = 0.400
补偿连乘：~0.453（较平衡）
```

### 7.1 效用调试器

```cpp
class UtilityDebugger {
public:
    static void PrintUtilityScores(const UtilitySelector& selector) {
        auto scores = selector.GetAllScores();

        std::cout << "\n=== 效用评分 ===\n";
        for (const auto& info : scores) {
            std::string bar(info.score * 50, '=');
            std::cout << info.name << ": " << bar << " " << info.score << "\n";
        }
        if (selector.GetCurrentAction()) {
            std::cout << "当前行为: " << selector.GetCurrentAction()->GetName() << "\n";
        }
        std::cout << "================\n";
    }

    // 生成ASCII响应曲线
    static void DrawCurve(const ResponseCurve& curve, int width = 50, int height = 20) {
        std::vector<std::string> screen(height, std::string(width, ' '));

        for (int x = 0; x < width; x++) {
            float input = (float)x / width;
            float output = curve.Evaluate(input);
            int y = (int)((1.0f - output) * (height - 1));
            y = std::max(0, std::min(height - 1, y));
            screen[y][x] = '*';
        }

        std::cout << "响应曲线: " << curve.GetName() << "\n";
        for (const auto& line : screen) {
            std::cout << "|" << line << "|\n";
        }
    }
};
```

## 9. 常见陷阱

1. **响应曲线调优缺乏方法论**：依赖试错。解决方案：使用可视化工具，建立数据驱动的调参流程
2. **分数计算过度复杂**：过多考虑因素导致帧率影响。解决方案：限制每个行为的考虑因素数量（不超过5-6个）
3. **行为抖振（Thrashing）**：分数接近的行为频繁切换。解决方案：防抖振阈值 + 最小执行时间
4. **缺少默认行为**：所有行为分数为0时AI无事可做。解决方案：始终有一个低分数的"默认行为"
5. **连乘偏差**：考虑因素越多分数越低。解决方案：使用补偿连乘或加权平均
6. **响应曲线过度拟合**：为特定场景过度调参。解决方案：使用简单曲线，避免过于复杂的形状

## 10. 实际游戏案例

### 案例1：模拟人生系列

模拟人生是效用系统最经典的应用：
- NPC有多种"需求"：饥饿、社交、卫生、娱乐、精力、上厕所
- 每种需求是一个考虑因素，使用S曲线或反向S曲线
- 行为（如"做饭"）满足"饥饿"需求但消耗"精力"
- 需求越紧迫，对应行为的效用越高
- NPC自然地在需求间"切换"，产生真实的行为模式

### 案例2：辐射4

辐射4的敌人AI使用效用系统选择战术：
- 射击效用 = 弹药量 * 视线可见 * 距离合适度 * 敌人暴露度
- 掩护效用 = 血量低 * 掩体可用性 * 敌人火力强度
- 冲锋效用 = 近战武器 * 敌人距离近 * 敌人血量低
- 通过调整曲线形状，不同敌人类型（超级变种人 vs 掠夺者）产生不同的战术偏好

### 案例3：无主之地系列

无主之地的敌人AI使用效用系统处理多维决策：
- 不同敌人类型有不同的考虑因素权重
- Badass敌人：攻击效用更高，撤退效用更低
- 普通敌人：更容易被压制，更倾向于撤退
- 环境因素（掩体、高地）影响效用计算

### 案例4：Stellaris（群星）

群星中的AI帝国决策使用效用系统：
- 外交行为（结盟、宣战、贸易）的效用基于多因素
- 考虑因素：军事实力对比、经济状况、地理位置、性格特质
- 不同AI性格（好战、和平、贪婪）使用不同的响应曲线
- 产生丰富多样的外交行为，避免AI行为"千篇一律"
