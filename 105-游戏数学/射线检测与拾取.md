# 射线检测与拾取

## 核心概念

射线检测（Raycast）是从一点沿某一方向发射射线，检测它与场景中物体的碰撞。这是游戏中最常用的交互检测方式——鼠标拾取、射击判定、视线检测、地面检测等都依赖射线检测。

### 射线的数学表示

```
Ray(t) = Origin + t × Direction

其中：
- Origin: 射线起点（Vec3）
- Direction: 射线方向（单位向量）
- t: 参数，t >= 0 表示射线正方向上的点

不同的 t 值对应射线上的不同位置：
- t = 0: 起点本身
- t = 10: 起点沿方向走10个单位的位置
- t = 最近交点的参数值: 碰撞点
```

## 射线-球体相交

### 数学推导

```
球方程：|P - Center|² = r²
代入射线方程 P = O + tD：
|O + tD - C|² = r²

令 OC = O - C（从球心到射线起点的向量）：
|tD + OC|² = r²
(D·D)t² + 2(D·OC)t + (OC·OC - r²) = 0

一元二次方程 at² + bt + c = 0，其中：
a = D·D = 1（单位向量）
b = 2 × D · OC
c = OC·OC - r² = |OC|² - r²

判别式 Δ = b² - 4ac
Δ < 0: 无交点（射线不穿过球）
Δ = 0: 一个交点（射线切过球）
Δ > 0: 两个交点（射线穿过球）

解：t = (-b ± √Δ) / (2a)
取最小的正 t 值作为最近交点
```

### C++ 完整实现

```cpp
struct Vec3 {
    float x, y, z;
    Vec3 operator-(const Vec3& o) const { return {x-o.x, y-o.y, z-o.z}; }
    Vec3 operator+(const Vec3& o) const { return {x+o.x, y+o.y, z+o.z}; }
    Vec3 operator*(float s) const { return {x*s, y*s, z*s}; }
    float dot(const Vec3& o) const { return x*o.x + y*o.y + z*o.z; }
    float lengthSquared() const { return x*x + y*y + z*z; }
    Vec3 normalized() const { float l = sqrtf(lengthSquared()); return {x/l, y/l, z/l}; }
};

struct Ray {
    Vec3 origin;
    Vec3 direction; // 必须是单位向量
};

struct Sphere {
    Vec3 center;
    float radius;
};

struct HitResult {
    bool hit;
    float distance;    // 沿射线的距离
    Vec3 point;        // 碰撞点世界坐标
    Vec3 normal;       // 碰撞点法线
};

// 射线与球体相交
HitResult raySphere(const Ray& ray, const Sphere& sphere) {
    HitResult result = {false, 0, {0,0,0}, {0,0,0}};

    Vec3 oc = ray.origin - sphere.center;
    float b = ray.direction.dot(oc); // 注意：如果a=1，则b实际是2*b/2
    float c = oc.dot(oc) - sphere.radius * sphere.radius;

    // 判别式（简化形式，a=1所以省略4a）
    // 原方程：t² + 2bt + c = 0（这里b已经包含了1/2系数）
    // 判别式：b² - c
    float discriminant = b * b - c;

    if (discriminant < 0)
        return result; // 无交点

    float sqrtD = sqrtf(discriminant);

    // 取最近的正交点
    float t = -b - sqrtD;
    if (t < 0)
        t = -b + sqrtD;
    if (t < 0)
        return result; // 球在射线后方

    result.hit = true;
    result.distance = t;
    result.point = ray.origin + ray.direction * t;
    result.normal = (result.point - sphere.center).normalized();
    return result;
}
```

## 射线-AABB 包围盒（Slab 方法）

### 数学原理

```
AABB 由三个轴对齐的"板对"（slab pair）定义：
X轴：min.x <= x <= max.x
Y轴：min.y <= y <= max.y
Z轴：min.z <= z <= max.z

射线与每个"板对"相交得到一个 t 区间 [tmin_i, tmax_i]
射线与 AABB 相交 = 所有三个区间的交集非空

区间计算（以X轴为例）：
射线方程：x = ox + t × dx
板的范围：min.x <= x <= max.x
所以：min.x <= ox + t × dx <= max.x
当 dx != 0 时：
  t1 = (min.x - ox) / dx
  t2 = (max.x - ox) / dx
  区间 = [min(t1,t2), max(t1,t2)]

三个轴的区间求交集：
tmin = max(tmin_x, tmin_y, tmin_z)
tmax = min(tmax_x, tmax_y, tmax_z)
有交点 iff tmin <= tmax 且 tmax >= 0
```

### C++ 实现

```cpp
struct AABB {
    Vec3 min, max;

    Vec3 center() const { return (min + max) * 0.5f; }
    Vec3 size() const { return max - min; }
};

// 射线与AABB相交（Slab方法）
HitResult rayAABB(const Ray& ray, const AABB& box) {
    HitResult result = {false, 0, {0,0,0}, {0,0,0}};

    float tmin = -FLT_MAX;
    float tmax = FLT_MAX;
    Vec3 normal = {0,0,0};
    int hitAxis = -1;

    // 对每个轴独立计算区间
    const float* o = &ray.origin.x;
    const float* d = &ray.direction.x;
    const float* bmin = &box.min.x;
    const float* bmax = &box.max.x;

    for (int i = 0; i < 3; i++) {
        if (fabsf(d[i]) < 1e-8f) {
            // 射线平行于该轴的板
            if (o[i] < bmin[i] || o[i] > bmax[i])
                return result; // 射线在板的外侧
            continue; // 射线在板的范围内，该轴不限制
        }

        float invD = 1.0f / d[i];
        float t1 = (bmin[i] - o[i]) * invD;
        float t2 = (bmax[i] - o[i]) * invD;

        if (t1 > t2) std::swap(t1, t2);

        // 更新法线（进入面的法线）
        if (t1 > tmin) {
            tmin = t1;
            hitAxis = i;
            normal = {0, 0, 0};
            (&normal.x)[i] = (d[i] < 0) ? 1.0f : -1.0f;
        }

        tmax = std::min(tmax, t2);

        if (tmin > tmax)
            return result; // 区间无交集
    }

    // 确定最近的有效交点
    float t = (tmin > 0) ? tmin : tmax;
    if (t < 0) return result; // AABB在射线后方

    result.hit = true;
    result.distance = t;
    result.point = ray.origin + ray.direction * t;
    result.normal = normal;
    return result;
}
```

## 射线-三角形相交（Moller-Trumbore 算法）

### 数学原理

```
三角形由三个顶点 V0, V1, V2 定义。
三角形上的任意点 P 可表示为重心坐标 (u, v, w)：
P = w*V0 + u*V1 + v*V2，其中 w = 1-u-v，且 u>=0, v>=0, u+v<=1

射线方程：O + tD = (1-u-v)*V0 + u*V1 + v*V2

整理为线性方程组：
[-D, E1, E2] × [t, u, v]^T = [O - V0]

其中 E1 = V1 - V0，E2 = V2 - V0

使用克莱姆法则求解：
```

### C++ 实现

```cpp
struct Triangle {
    Vec3 v0, v1, v2;
};

// Moller-Trumbore 射线-三角形相交算法
HitResult rayTriangle(const Ray& ray, const Triangle& tri, bool backfaceCull = false) {
    HitResult result = {false, 0, {0,0,0}, {0,0,0}};

    const float EPSILON = 1e-6f;

    Vec3 edge1 = tri.v1 - tri.v0;
    Vec3 edge2 = tri.v2 - tri.v0;

    // 开始计算行列式
    Vec3 h = ray.direction.cross(edge2);
    float a = edge1.dot(h);

    if (fabsf(a) < EPSILON)
        return result; // 射线与三角形平行

    if (backfaceCull && a < 0)
        return result; // 背面剔除

    float f = 1.0f / a;
    Vec3 s = ray.origin - tri.v0;
    float u = f * s.dot(h);

    if (u < 0.0f || u > 1.0f)
        return result; // 交点不在三角形内

    Vec3 q = s.cross(edge1);
    float v = f * ray.direction.dot(q);

    if (v < 0.0f || u + v > 1.0f)
        return result; // 交点不在三角形内

    // 计算 t
    float t = f * edge2.dot(q);

    if (t < EPSILON)
        return result; // 交点在射线后方

    result.hit = true;
    result.distance = t;
    result.point = ray.origin + ray.direction * t;

    // 计算法线
    Vec3 normal = edge1.cross(edge2).normalized();
    if (a < 0) normal = Vec3{0,0,0} - normal; // 确保法线朝外
    result.normal = normal;

    return result;
}
```

## 射线-平面相交

```cpp
struct Plane {
    Vec3 normal;  // 单位法线
    float d;      // 平面方程: normal·P + d = 0
};

HitResult rayPlane(const Ray& ray, const Plane& plane) {
    HitResult result = {false, 0, {0,0,0}, {0,0,0}};

    float denom = plane.normal.dot(ray.direction);

    if (fabsf(denom) < 1e-6f)
        return result; // 射线平行于平面

    float t = -(plane.normal.dot(ray.origin) + plane.d) / denom;

    if (t < 0)
        return result; // 平面在射线后方

    result.hit = true;
    result.distance = t;
    result.point = ray.origin + ray.direction * t;
    result.normal = plane.normal;
    return result;
}
```

## 屏幕坐标到世界射线（鼠标拾取管线）

### 坐标变换管线

```
屏幕坐标 → NDC坐标 → 裁剪空间 → 世界空间

1. 屏幕坐标 (mouseX, mouseY) 转 NDC [-1, 1]:
   ndcX = 2 * mouseX / screenWidth - 1
   ndcY = 1 - 2 * mouseY / screenHeight    // Y轴翻转（屏幕Y向下，NDC Y向上）

2. NDC坐标乘以逆投影矩阵得到相机空间的近/远平面点

3. 相机空间点乘以逆视图矩阵得到世界空间的近/远平面点

4. 射线方向 = normalize(远点 - 近点)
   射线起点 = 近点（或相机位置）
```

```cpp
// 屏幕坐标转世界空间射线
Ray screenToRay(
    float mouseX, float mouseY,
    int screenWidth, int screenHeight,
    const Mat4& viewMatrix,
    const Mat4& projMatrix)
{
    // 1. 屏幕坐标 → NDC
    float ndcX = (2.0f * mouseX) / screenWidth - 1.0f;
    float ndcY = 1.0f - (2.0f * mouseY) / screenHeight;

    // 2. 计算逆 VP 矩阵
    Mat4 viewProj = projMatrix * viewMatrix;
    Mat4 invVP = viewProj.inverse();

    // 3. NDC → 世界空间
    // 近平面点（z=-1 in NDC）
    Vec4 nearNDC = {ndcX, ndcY, -1.0f, 1.0f};
    Vec4 nearWorld4 = invVP * nearNDC;
    Vec3 nearWorld = {nearWorld4.x / nearWorld4.w,
                      nearWorld4.y / nearWorld4.w,
                      nearWorld4.z / nearWorld4.w};

    // 远平面点（z=1 in NDC）
    Vec4 farNDC = {ndcX, ndcY, 1.0f, 1.0f};
    Vec4 farWorld4 = invVP * farNDC;
    Vec3 farWorld = {farWorld4.x / farWorld4.w,
                     farWorld4.y / farWorld4.w,
                     farWorld4.z / farWorld4.w};

    // 4. 构建射线
    Vec3 dir = (farWorld - nearWorld).normalized();
    return {nearWorld, dir};
}
```

### 完整鼠标拾取管线

```cpp
// 鼠标拾取：点击屏幕选中3D物体
class MousePickingSystem {
    Camera camera;
    std::vector<GameObject*> sceneObjects;

    GameObject* Pick(int mouseX, int mouseY, int screenW, int screenH) {
        // 1. 从屏幕坐标生成射线
        Ray ray = screenToRay(mouseX, mouseY, screenW, screenH,
                              camera.GetViewMatrix(),
                              camera.GetProjMatrix());

        // 2. 遍历场景物体进行射线检测
        float closestDist = FLT_MAX;
        GameObject* closestObj = nullptr;

        for (auto* obj : sceneObjects) {
            // 先用包围盒做粗检测
            HitResult hit = rayAABB(ray, obj->GetWorldAABB());
            if (hit.hit && hit.distance < closestDist) {
                // 精确检测（射线 vs 网格三角形）
                if (rayMesh(ray, obj->GetMesh(), obj->GetWorldMatrix())) {
                    closestDist = hit.distance;
                    closestObj = obj;
                }
            }
        }

        return closestObj; // 返回最近的被选中物体
    }

    // 射线与网格的精确检测（遍历所有三角形）
    bool rayMesh(const Ray& ray, const Mesh& mesh, const Mat4& worldMatrix) {
        // 将射线变换到模型空间
        Mat4 invWorld = worldMatrix.inverse();
        Ray localRay;
        localRay.origin = invWorld.transformPoint(ray.origin);
        localRay.direction = invWorld.transformDirection(ray.direction).normalized();

        for (int i = 0; i < mesh.triangleCount; i++) {
            Triangle tri = mesh.GetTriangle(i);
            if (rayTriangle(localRay, tri).hit)
                return true;
        }
        return false;
    }
};
```

## 空间划分加速射线检测

```cpp
// 使用 BVH 加速射线检测
class BVHRaycast {
    BVHNode* root;

    // 射线遍历 BVH 树
    void Raycast(const Ray& ray, float maxDist, std::vector<HitResult>& results) {
        RaycastNode(root, ray, maxDist, results);
    }

    void RaycastNode(BVHNode* node, const Ray& ray, float maxDist,
                     std::vector<HitResult>& results) {
        if (node == nullptr) return;

        // 快速排除：射线不与节点包围盒相交
        if (!rayAABB(ray, node->bounds).hit)
            return;

        if (node->isLeaf()) {
            // 叶节点：精确检测
            HitResult hit = rayTriangle(ray, node->triangle);
            if (hit.hit && hit.distance <= maxDist)
                results.push_back(hit);
        } else {
            // 递归检测子节点
            RaycastNode(node->left, ray, maxDist, results);
            RaycastNode(node->right, ray, maxDist, results);
        }
    }
};
```

```cpp
void FireWeapon(Player& player, const Camera& camera) {

    Ray bulletRay;
    bulletRay.origin = camera.position;
    bulletRay.direction = camera.forward;

    // 添加散布（后坐力）
    float spread = player.GetSpreadAngle();

        (rand() / (float)RAND_MAX - 0.5f) * spread,
        0
```

```cpp
// AI 判断是否能看到目标
bool CanSeeTarget(const AICharacter& ai, const Character& target) {
    Vec3 toTarget = target.position - ai.position;
    float distance = toTarget.length();

    // 距离检查
    if (distance > ai.visionRange)
        return false;

    // 视野角度检查
    float cosAngle = ai.forward.dot(toTarget.normalized());
    if (cosAngle < cosf(ai.fovHalfAngle * DEG2RAD))
        return false;

    // 遮挡检测
    Ray sightRay;
    sightRay.origin = ai.position + Vec3{0, 1.5f, 0}; // 眼睛高度
    sightRay.direction = toTarget.normalized();

    HitResult hit = Physics::Raycast(sightRay, distance);
    if (hit.hit && hit.distance < distance - 0.1f) {
        return false; // 被障碍物遮挡
    }

    return true;
}
```

## 方案对比

| 检测类型 | 精度 | 性能 | 用途 |
|---------|------|------|------|
| 射线-Sphere | 数学精确 | 极快 | 简单碰撞、目标选择 |
| 射线-AABB | 数学精确 | 极快 | 粗检测、UI检测 |
| 射线-OBB | 数学精确 | 快 | 倾斜物体检测 |
| 射线-Triangle | 数学精确 | 中 | 精确拾取、地形检测 |
| 射线-BVH | 数学精确 | 快 | 大量三角形加速 |
| 射线-网格 | 近似 | 慢 | 复杂模型检测 |

## 常见陷阱与解决方案

1. **原点在AABB内部**：Slab方法的 tmin 可能为负值。解决方案：取 t = max(tmin, 0) 或 t = max(tmin, tmax)
2. **射线方向未归一化**：t 值不再等于实际距离。解决方案：始终归一化射线方向，或在结果中乘以方向长度
3. **精度问题**：碰撞点可能略在表面下方，导致重复碰撞。解决方案：将碰撞点沿法线偏移一个小量（epsilon = 0.001f）
4. **自碰撞**：射线与发射它的物体碰撞。解决方案：使用 Layer/Mask 过滤，或在碰撞后偏移射线起点
5. **性能瓶颈**：大量射线检测拖慢帧率。解决方案：使用空间划分结构（BVH、八叉树）加速，先粗检测再精检测
