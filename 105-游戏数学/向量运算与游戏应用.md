# 向量运算与游戏应用

## 核心概念

向量是游戏数学的基石，表示方向和大小。游戏中的位置、速度、力、法线、光照方向等都用向量表示。掌握向量运算是游戏编程的基本功。

### 向量基础

```
向量 = 方向 + 大小
A = (Ax, Ay, Az)

长度（模）：|A| = sqrt(Ax² + Ay² + Az²)
单位向量：A_hat = A / |A|  （长度为1的方向向量）
零向量：(0, 0, 0) —— 无方向无大小
```

---

## 点积（Dot Product）

### 数学定义

```
A · B = |A| × |B| × cos(θ)
     = Ax×Bx + Ay×By + Az×Bz
     = |A| × |B| × cos(θ)
```

### 几何意义

```
A · B 的值 = A在B方向上的投影长度 × |B|

当 |B| = 1（B是单位向量）时：
A · B = A在B方向上的投影长度

物理意义：一个力F沿斜面方向的分量 = F · 斜面方向
```

### 点积的符号判断

```
A · B > 0  →  cos(θ) > 0  →  θ < 90°  →  两向量大致同向
A · B = 0  →  cos(θ) = 0  →  θ = 90°  →  两向量垂直
A · B < 0  →  cos(θ) < 0  →  θ > 90°  →  两向量大致反向
```

### 点积推导（二维简化证明）

```
设 A = (Ax, Ay)，B = (Bx, By)
极坐标转换：
  A = (|A|cosα, |A|sinα)
  B = (|B|cosβ, |B|sinβ)

A · B = |A||B|cosαcosβ + |A||B|sinαsinβ
      = |A||B|(cosαcosβ + sinαsinβ)
      = |A||B|cos(α - β)    // 三角恒等式
      = |A||B|cos(θ)        // θ = α - β 是两向量夹角
```

---

## 叉积（Cross Product）

### 数学定义

```
A × B = (Ay×Bz - Az×By, Az×Bx - Ax×Bz, Ax×By - Ay×Bx)
```

### 几何意义

```
|A × B| = |A| × |B| × sin(θ)
        = 以A和B为边的平行四边形面积

A × B 的方向 = 垂直于A和B所在平面（遵循右手/左手定则）
```

### 叉积在游戏中的关键用途

```
1. 判断点在直线的哪一侧（2D）
2. 计算三角形法线（3D）
3. 判断旋转方向（顺时针/逆时针）
4. 计算力矩
```

## 完整 Vec3 类实现

```cpp
#include <cmath>
#include <algorithm>

struct Vec3 {
    float x, y, z;

    Vec3() : x(0), y(0), z(0) {}
    Vec3(float x, float y, float z) : x(x), y(y), z(z) {}

    // ===== 基础运算 =====
    Vec3 operator+(const Vec3& other) const { return {x + other.x, y + other.y, z + other.z}; }
    Vec3 operator-(const Vec3& other) const { return {x - other.x, y - other.y, z - other.z}; }
    Vec3 operator*(float s) const { return {x * s, y * s, z * s}; }
    Vec3 operator/(float s) const { float inv = 1.0f / s; return {x * inv, y * inv, z * inv}; }
    Vec3 operator-() const { return {-x, -y, -z}; }

    Vec3& operator+=(const Vec3& o) { x += o.x; y += o.y; z += o.z; return *this; }
    Vec3& operator-=(const Vec3& o) { x -= o.x; y -= o.y; z -= o.z; return *this; }
    Vec3& operator*=(float s) { x *= s; y *= s; z *= s; return *this; }

    // ===== 点积 =====
    float dot(const Vec3& other) const {
        return x * other.x + y * other.y + z * other.z;
    }

    // ===== 叉积 =====
    Vec3 cross(const Vec3& other) const {
        return {
            y * other.z - z * other.y,
            z * other.x - x * other.z,
            x * other.y - y * other.x
        };
    }

    // ===== 长度 =====
    float lengthSquared() const { return x * x + y * y + z * z; }
    float length() const { return std::sqrt(lengthSquared()); }

    // ===== 归一化 =====
    Vec3 normalized() const {
        float len = length();
        if (len < 1e-6f) return {0, 0, 0}; // 防止除零
        return *this / len;
    }

    void normalize() {
        float len = length();
        if (len > 1e-6f) { x /= len; y /= len; z /= len; }
    }

    // ===== 距离 =====
    static float distance(const Vec3& a, const Vec3& b) {
        return (a - b).length();
    }

    static float distanceSquared(const Vec3& a, const Vec3& b) {
        return (a - b).lengthSquared();
    }

    // ===== 向量投影 =====
    // 将 this 投影到 target 上
    Vec3 projectOnto(const Vec3& target) const {
        float targetLenSq = target.lengthSquared();
        if (targetLenSq < 1e-6f) return {0, 0, 0};
        return target * (dot(target) / targetLenSq);
    }

    // 将 this 从 target 方向上减去投影（得到垂直分量）
    Vec3 rejectFrom(const Vec3& target) const {
        return *this - projectOnto(target);
    }

    // ===== 反射 =====
    // this 是入射方向（指向表面），normal 是表面法线（单位向量）
    // 返回反射方向
    Vec3 reflect(const Vec3& normal) const {
        return *this - normal * (2.0f * dot(normal));
    }

    // ===== 插值 =====
    static Vec3 lerp(const Vec3& a, const Vec3& b, float t) {
        return a + (b - a) * t;
    }

    // ===== 常用向量 =====
    static Vec3 zero()  { return {0, 0, 0}; }
    static Vec3 one()   { return {1, 1, 1}; }
    static Vec3 up()    { return {0, 1, 0}; }
    static Vec3 down()  { return {0, -1, 0}; }
    static Vec3 left()  { return {-1, 0, 0}; }
    static Vec3 right() { return {1, 0, 0}; }
    static Vec3 forward()  { return {0, 0, 1}; }
    static Vec3 backward() { return {0, 0, -1}; }

    // ===== 夹角 =====
    static float angle(const Vec3& a, const Vec3& b) {
        float cosTheta = a.normalized().dot(b.normalized());
        cosTheta = std::max(-1.0f, std::min(1.0f, cosTheta)); // 钳制防止NaN
        return std::acos(cosTheta) * (180.0f / 3.14159265f);  // 返回角度
    }

    // ===== 带符号夹角（2D，绕Z轴） =====
    static float signedAngle2D(const Vec3& a, const Vec3& b) {
        float angle = std::atan2(b.y, b.x) - std::atan2(a.y, a.x);
        if (angle > 3.14159f) angle -= 2 * 3.14159f;
        if (angle < -3.14159f) angle += 2 * 3.14159f;
        return angle * (180.0f / 3.14159f);
    }
};
```

## 游戏应用场景详解

### 1. 视野锥判断（FOV）

```cpp
// 判断敌人是否在玩家前方的视野锥内
// playerForward: 玩家朝向（单位向量）
// toEnemy: 从玩家指向敌人的向量
// fovHalfAngle: 视野半角（如60度FOV → 半角30度）
bool isInFieldOfView(Vec3 playerForward, Vec3 toEnemy, float fovHalfAngle) {
    Vec3 dir = toEnemy.normalized();
    float cosAngle = playerForward.dot(dir);
    float cosHalfFov = std::cos(fovHalfAngle * 3.14159f / 180.0f);
    return cosAngle > cosHalfFov;
    // 不需要 acos！直接比较 cos 值即可
    // cos是单调递减的：cos(30°) > cos(60°)，所以 cosAngle > cos(halfFov) 即角度 < halfFov
}

// 使用示例
void UpdateAI(Enemy& enemy, Player& player) {
    Vec3 toPlayer = player.position - enemy.position;
    bool canSee = isInFieldOfView(enemy.forward, toPlayer, 60.0f);

    if (canSee && toPlayer.length() < detectionRange) {
        enemy.SetState(Chase);
    }
}
```

### 2. 光照漫反射计算

```cpp
// 漫反射强度 = max(0, N · L)
// N = 表面法线（单位向量）
// L = 从表面指向光源的方向（单位向量）
float diffuseIntensity(Vec3 normal, Vec3 lightDir) {
    return std::max(0.0f, normal.dot(lightDir));
    // 为什么 max(0,...)？
    // N·L < 0 表示光线来自表面背面（被遮挡），应该无光照
}

// 冯氏光照模型（Phong）
Vec3 calculatePhong(Vec3 normal, Vec3 lightDir, Vec3 viewDir,
                    Vec3 lightColor, float shininess) {
    // 漫反射
    float diff = std::max(0.0f, normal.dot(lightDir));

    // 镜面反射
    Vec3 reflectDir = (normal * 2.0f * normal.dot(lightDir) - lightDir).normalized();
    float spec = std::pow(std::max(0.0f, viewDir.dot(reflectDir)), shininess);

    // 合成
    Vec3 ambient = lightColor * 0.1f;
    Vec3 diffuse = lightColor * diff * 0.7f;
    Vec3 specular = lightColor * spec * 0.5f;

    return ambient + diffuse + specular;
}
```

### 3. 速度反射（弹球物理）

```cpp
// 弹球从墙面反弹：V_reflected = V - 2(V·N)N
// V: 入射速度向量
// N: 墙面法线（单位向量，指向球来的方向）
Vec3 reflectVelocity(Vec3 velocity, Vec3 wallNormal) {
    // reflect 公式推导：
    // V 在法线方向的分量 = (V·N)N
    // 反弹需要反转法线方向的分量
    // V_ref = V - 2(V·N)N
    return velocity - wallNormal * (2.0f * velocity.dot(wallNormal));
}

// 完整的弹球物理
class Ball {
    Vec3 position;
    Vec3 velocity;

    void Update(float dt, const std::vector<Wall>& walls) {
        position += velocity * dt;

        for (const auto& wall : walls) {
            // 检测碰撞
            float dist = (position - wall.point).dot(wall.normal);
            if (dist < radius && velocity.dot(wall.normal) < 0) {
                // 球正在穿过墙面
                position += wall.normal * (radius - dist); // 推回表面

                // 反射速度（带能量损失）
                velocity = reflectVelocity(velocity, wall.normal) * bounciness;

                // 确保不再朝墙面移动
                if (velocity.dot(wall.normal) < 0)
                    velocity -= wall.normal * velocity.dot(wall.normal);
            }
        }
    }
};
```

### 4. 平面方程与点到平面距离

```cpp
// 平面由法线 N 和平面上一点 P0 定义
// 平面方程：N · (P - P0) = 0
// 展开：Nx*x + Ny*y + Nz*z + d = 0，其中 d = -N·P0

struct Plane {
    Vec3 normal;  // 单位法线
    float d;      // -N·P0

    Plane(Vec3 n, Vec3 point) : normal(n.normalized()) {
        d = -normal.dot(point);
    }

    // 点到平面的有符号距离
    float distanceTo(Vec3 point) const {
        return normal.dot(point) + d;
        // > 0: 点在法线指向的一侧
        // < 0: 点在法线背向的一侧
        // = 0: 点在平面上
    }

    // 点在平面的哪一侧
    bool isAbove(Vec3 point) const { return distanceTo(point) > 0; }

    // 将点投影到平面上
    Vec3 project(Vec3 point) const {
        return point - normal * distanceTo(point);
    }
};

// 应用：视锥体裁剪
// 视锥体由6个平面组成（近、远、左、右、上、下）
bool isInsideFrustum(Vec3 point, const Plane frustum[6]) {
    for (int i = 0; i < 6; i++) {
        if (frustum[i].distanceTo(point) < 0)
            return false; // 在某个平面外侧
    }
    return true;
}
```

### 5. 向量插值与朝向平滑

```cpp
// 平滑旋转朝向目标
Vec3 smoothLookAt(Vec3 currentForward, Vec3 targetDir, float maxAngleDelta) {
    // 球面线性插值的简化版
    float cosAngle = currentForward.dot(targetDir);

    // 如果角度很小，直接返回目标方向
    if (cosAngle > 0.9999f)
        return targetDir;

    // 限制每帧最大旋转角度
    float angle = std::acos(std::max(-1.0f, std::min(1.0f, cosAngle)));
    float t = std::min(1.0f, maxAngleDelta / angle);

    // 线性插值 + 重新归一化（近似球面插值）
    Vec3 result = Vec3::lerp(currentForward, targetDir, t);
    return result.normalized();
}

// 使用
void UpdateTurret(float dt) {
    Vec3 toTarget = (target.position - turret.position).normalized();
    turret.forward = smoothLookAt(turret.forward, toTarget, rotationSpeed * dt);
}
```

### 6. 2D 叉积判断点在直线哪一侧

```cpp
// 在2D中（忽略z），叉积的z分量可以判断方向
// 叉积 > 0: B在A的逆时针方向（左侧）
// 叉积 < 0: B在A的顺时针方向（右侧）
// 叉积 = 0: A和B共线

float cross2D(Vec3 a, Vec3 b) {
    return a.x * b.y - a.y * b.x;
}

// 判断点P是否在有向线段AB的左侧
bool isOnLeftSide(Vec3 A, Vec3 B, Vec3 P) {
    Vec3 AB = B - A;
    Vec3 AP = P - A;
    return cross2D(AB, AP) > 0;
}

// 应用：判断多边形是顺时针还是逆时针
bool isPolygonClockwise(const std::vector<Vec3>& vertices) {
    float sum = 0;
    for (size_t i = 0; i < vertices.size(); i++) {
        Vec3 a = vertices[i];
        Vec3 b = vertices[(i + 1) % vertices.size()];
        sum += cross2D(a, b);
    }
    return sum < 0; // 负值表示顺时针
}
```

## 方案对比

| 运算 | 时间复杂度 | 主要用途 | 返回类型 |
|------|-----------|---------|---------|
| 点积 | O(1) | 角度判断、投影、光照 | 标量 |
| 叉积 | O(1) | 法线、面积、方向判断 | 向量 |
| 归一化 | O(1) | 获取纯方向 | 向量 |
| 投影 | O(1) | 分解力/速度 | 向量 |
| 反射 | O(1) | 弹球、光线追踪 | 向量 |
| Lerp | O(1) | 插值移动 | 向量 |

## 常见陷阱与解决方案

1. **除零问题**：归一化零向量会导致 NaN。解决方案：归一化前检查 `length() > epsilon`
2. **叉积顺序**：左手系和右手系的叉积方向相反。解决方案：确认引擎的坐标系，统一使用
3. **浮点精度**：点积判断正负时使用容差（epsilon = 0.0001f），不要直接与 0 比较
4. **比较 cos 而非角度**：`cos(a) > cos(b)` 等价于 `a < b`（在 [0, pi] 范围内），避免昂贵的 acos
5. **向量比较**：浮点误差导致两个"应该相等"的向量判断为不等。解决方案：用 `distanceSquared < epsilon` 比较

## 实际使用案例

- **视野判断**：点积判断目标是否在玩家前方180度/120度/90度范围内
- **光照计算**：漫反射 = 法线·光线方向，是所有实时渲染的基础
- **弹球/反弹**：向量反射公式 `V - 2(V·N)N`
- **移动/寻路**：归一化获取方向，乘以速度得到位移向量
- **法线计算**：三角形两边叉积得到面法线，用于光照和碰撞
- **力的分解**：将重力沿斜面法线投影和切向投影
