# 插值与缓动函数

## 核心概念

插值是在两个值之间平滑过渡的技术，缓动函数控制过渡的速度曲线。它们是动画和游戏感觉（Game Feel）的关键——UI 弹窗、相机跟随、血条变化、角色移动都需要插值来实现平滑自然的视觉效果。

### 线性插值（Lerp）

```
Lerp(a, b, t) = a + (b - a) × t    其中 t ∈ [0, 1]

t = 0 → 返回 a（起点）
t = 0.5 → 返回 a 和 b 的中点
t = 1 → 返回 b（终点）

几何意义：在 a 和 b 之间，按比例 t 找到中间点
```

## 基础插值函数

```cpp
#include <cmath>
#include <algorithm>

// ===== 标量插值 =====
float lerp(float a, float b, float t) {
    return a + (b - a) * t;
}

// ===== 向量插值 =====
struct Vec3 {
    float x, y, z;
    Vec3 operator+(const Vec3& o) const { return {x+o.x, y+o.y, z+o.z}; }
    Vec3 operator-(const Vec3& o) const { return {x-o.x, y-o.y, z-o.z}; }
    Vec3 operator*(float s) const { return {x*s, y*s, z*s}; }
};

Vec3 lerpVec3(const Vec3& a, const Vec3& b, float t) {
    return {
        lerp(a.x, b.x, t),
        lerp(a.y, b.y, t),
        lerp(a.z, b.z, t)
    };
}

// ===== 颜色插值 =====
struct Color { float r, g, b, a; };

Color lerpColor(const Color& a, const Color& b, float t) {
    return {
        lerp(a.r, b.r, t),
        lerp(a.g, b.g, t),
        lerp(a.b, b.b, t),
        lerp(a.a, b.a, t)
    };
}

// ===== 反向插值 =====
// 给定值 v 和范围 [a, b]，求 t
float inverseLerp(float a, float b, float v) {
    if (fabsf(b - a) < 1e-6f) return 0.0f; // 防止除零
    return (v - a) / (b - a);
}

// ===== 重映射 =====
// 将值从一个范围映射到另一个范围
float remap(float v, float inMin, float inMax, float outMin, float outMax) {
    float t = inverseLerp(inMin, inMax, v);
    return lerp(outMin, outMax, t);
}

// 示例：将角色生命值 [0, 100] 映射到血条宽度 [0, 200像素]
float healthBarWidth = remap(currentHealth, 0, 100, 0, 200);
```

## 三次 Hermite 插值

三次 Hermite 插值提供端点的位置和速度控制，实现更平滑的过渡。

```
Hermite(t) = (2t³ - 3t² + 1) × p0 + (t³ - 2t² + t) × m0
           + (-2t³ + 3t²) × p1 + (t³ - t²) × m1

其中：
p0, p1: 起点和终点的值
m0, m1: 起点和终点的切线（速度）

边界条件：
Hermite(0) = p0,  Hermite'(0) = m0
Hermite(1) = p1,  Hermite'(1) = m1
```

```cpp
// 三次 Hermite 插值
float hermite(float p0, float p1, float m0, float m1, float t) {
    float t2 = t * t;
    float t3 = t2 * t;

    float h00 = 2*t3 - 3*t2 + 1;  // p0 的基函数
    float h10 = t3 - 2*t2 + t;     // m0 的基函数
    float h01 = -2*t3 + 3*t2;      // p1 的基函数
    float h11 = t3 - t2;           // m1 的基函数

    return h00 * p0 + h10 * m0 + h01 * p1 + h11 * m1;
}

// Catmull-Rom 样条（Hermite 的特例：自动计算切线）
// 使用前后控制点自动计算切线：m = (p_{i+1} - p_{i-1}) / 2
float catmullRom(float p0, float p1, float p2, float p3, float t) {
    float m0 = (p2 - p0) * 0.5f;  // p1 点的切线
    float m1 = (p3 - p1) * 0.5f;  // p2 点的切线
    return hermite(p1, p2, m0, m1, t);
}

// Catmull-Rom 曲线上的点（用于路径平滑）
Vec3 catmullRomPoint(
    const Vec3& p0, const Vec3& p1,
    const Vec3& p2, const Vec3& p3,
    float t)
{
    return {
        catmullRom(p0.x, p1.x, p2.x, p3.x, t),
        catmullRom(p0.y, p1.y, p2.y, p3.y, t),
        catmullRom(p0.z, p1.z, p2.z, p3.z, t)
    };
}

// 沿 Catmull-Rom 路径移动
Vec3 samplePath(const std::vector<Vec3>& points, float progress) {
    int n = points.size();
    float totalSegments = (float)(n - 1);
    float scaledT = progress * totalSegments;
    int segment = (int)scaledT;
    float localT = scaledT - segment;

    segment = std::max(0, std::min(segment, n - 1));

    int i0 = std::max(0, segment - 1);
    int i1 = segment;
    int i2 = std::min(n - 1, segment + 1);
    int i3 = std::min(n - 1, segment + 2);

    return catmullRomPoint(points[i0], points[i1], points[i2], points[i3], localT);
}
```

## Bezier 曲线与缓动

### 二次 Bezier

```
B(t) = (1-t)² × P0 + 2(1-t)t × P1 + t² × P2
```

### 三次 Bezier

```
B(t) = (1-t)³ × P0 + 3(1-t)²t × P1 + 3(1-t)t² × P2 + t³ × P3
```

```cpp
// 二次 Bezier
Vec3 quadraticBezier(const Vec3& p0, const Vec3& p1, const Vec3& p2, float t) {
    float u = 1.0f - t;
    return p0 * (u * u) + p1 * (2.0f * u * t) + p2 * (t * t);
}

// 三次 Bezier
Vec3 cubicBezier(const Vec3& p0, const Vec3& p1, const Vec3& p2, const Vec3& p3, float t) {
    float u = 1.0f - t;
    float uu = u * u;
    float uuu = uu * u;
    float tt = t * t;
    float ttt = tt * t;

    return p0 * uuu + p1 * (3.0f * uu * t) + p2 * (3.0f * u * tt) + p3 * ttt;
}

// Bezier 曲线导数（用于获取切线/速度）
Vec3 cubicBezierDerivative(const Vec3& p0, const Vec3& p1,
                            const Vec3& p2, const Vec3& p3, float t) {
    float u = 1.0f - t;
    return (p1 - p0) * (3.0f * u * u) +
           (p2 - p1) * (6.0f * u * t) +
           (p3 - p2) * (3.0f * t * t);
}

// Bezier 缓动函数（CSS标准）
// 三次 Bezier 的特殊情况：P0=(0,0), P3=(1,1)，通过 P1 和 P2 控制曲线形状
// CSS 中的 cubic-bezier(x1, y1, x2, y2) 就是这种形式
float bezierEase(float t, float x1, float y1, float x2, float y2) {
    // 通过 x(t) 反求 t，再计算 y(t)
    // x(t) = 3(1-t)²t*x1 + 3(1-t)t²*x2 + t³
    // 需要数值方法求解 x(t) = inputT 的 t 值

    // Newton-Raphson 迭代法
    float guess = t;
    for (int i = 0; i < 8; i++) {
        float x = cubicBezierCoord(0, x1, x2, 1, guess) - t;
        float dx = cubicBezierDerivativeCoord(0, x1, x2, 1, guess);
        if (fabsf(dx) < 1e-6f) break;
        guess -= x / dx;
        guess = std::max(0.0f, std::min(1.0f, guess));
    }

    return cubicBezierCoord(0, y1, y2, 1, guess);
}

float cubicBezierCoord(float p0, float p1, float p2, float p3, float t) {
    float u = 1.0f - t;
    return u*u*u*p0 + 3*u*u*t*p1 + 3*u*t*t*p2 + t*t*t*p3;
}

float cubicBezierDerivativeCoord(float p0, float p1, float p2, float p3, float t) {
    float u = 1.0f - t;
    return 3*u*u*(p1-p0) + 6*u*t*(p2-p1) + 3*t*t*(p3-p2);
}
```

## 缓动函数大全

缓动函数控制 `t` 的变换，决定动画的速度曲线：

```cpp
// ===== 基础缓动 =====

// Ease In —— 慢起步，快结束（加速）
float easeInQuad(float t)   { return t * t; }
float easeInCubic(float t)  { return t * t * t; }
float easeInQuart(float t)  { return t * t * t * t; }
float easeInQuint(float t)  { return t * t * t * t * t; }

// Ease Out —— 快起步，慢结束（减速）
float easeOutQuad(float t)  { return 1.0f - (1.0f-t)*(1.0f-t); }
float easeOutCubic(float t) { float u = 1.0f-t; return 1.0f - u*u*u; }
float easeOutQuart(float t) { float u = 1.0f-t; return 1.0f - u*u*u*u; }
float easeOutQuint(float t) { float u = 1.0f-t; return 1.0f - u*u*u*u*u; }

// Ease In-Out —— 两端慢，中间快
float easeInOutQuad(float t) {
    return t < 0.5f ? 2*t*t : 1 - powf(-2*t + 2, 2) / 2;
}
float easeInOutCubic(float t) {
    return t < 0.5f ? 4*t*t*t : 1 - powf(-2*t + 2, 3) / 2;
}
float easeInOutQuart(float t) {
    return t < 0.5f ? 8*t*t*t*t : 1 - powf(-2*t + 2, 4) / 2;
}

// ===== 指数缓动 =====
float easeInExpo(float t) {
    return t == 0 ? 0 : powf(2, 10 * t - 10);
}
float easeOutExpo(float t) {
    return t == 1 ? 1 : 1 - powf(2, -10 * t);
}
float easeInOutExpo(float t) {
    if (t == 0 || t == 1) return t;
    return t < 0.5f ? powf(2, 20*t - 10) / 2 : (2 - powf(2, -20*t + 10)) / 2;
}

// ===== 弹性缓动 =====
float easeOutElastic(float t) {
    if (t == 0 || t == 1) return t;
    float p = 0.3f;        // 周期
    float s = p / 4.0f;    // 相位偏移
    return powf(2, -10*t) * sinf((t - s) * (2 * M_PI) / p) + 1;
}

float easeInElastic(float t) {
    if (t == 0 || t == 1) return t;
    float p = 0.3f;
    float s = p / 4.0f;
    return -powf(2, 10*t - 10) * sinf((t * 10 - 10.75f) * (2 * M_PI) / p);
}

// ===== 回弹缓动（Overshoot） =====
float easeOutBack(float t) {
    float c1 = 1.70158f;   // 过冲量
    float c3 = c1 + 1;
    float u = t - 1;
    return 1 + c3 * u*u*u + c1 * u*u;
}

float easeInOutBack(float t) {
    float c1 = 1.70158f;
    float c2 = c1 * 1.525f;
    return t < 0.5f
        ? (powf(2*t, 2) * ((c2 + 1) * 2*t - c2)) / 2
        : (powf(2*t - 2, 2) * ((c2 + 1) * (t*2 - 2) + c2) + 2) / 2;
}

// ===== 弹跳缓动 =====
float easeOutBounce(float t) {
    float n1 = 7.5625f;
    float d1 = 2.75f;

    if (t < 1 / d1) return n1 * t * t;
    if (t < 2 / d1) { t -= 1.5f / d1; return n1 * t * t + 0.75f; }
    if (t < 2.5f / d1) { t -= 2.25f / d1; return n1 * t * t + 0.9375f; }
    t -= 2.625f / d1;
    return n1 * t * t + 0.984375f;
}
```

## 弹簧阻尼系统（Spring Damper）

弹簧物理模拟用于实现自然的跟随和回弹效果。

```
弹簧力：F_spring = -k × (x - target)    // k = 弹簧常数，越大越"硬"
阻尼力：F_damp = -d × v                  // d = 阻尼系数，越大越"粘"
合力：  F = F_spring + F_damp
加速度：a = F / m
速度：  v += a × dt
位置：  x += v × dt
```

```cpp
// 一维弹簧阻尼系统
class SpringDamper1D {
    float position;
    float velocity;
    float stiffness;   // 弹簧刚度 k（越大回弹越快）
    float damping;     // 阻尼 d（越大振动越少）

public:
    SpringDamper1D(float stiffness = 200.0f, float damping = 10.0f)
        : position(0), velocity(0), stiffness(stiffness), damping(damping) {}

    // 每帧调用，朝目标值更新
    float update(float target, float dt) {
        float displacement = position - target;
        float springForce = -stiffness * displacement;
        float dampingForce = -damping * velocity;
        float acceleration = springForce + dampingForce;

        velocity += acceleration * dt;
        position += velocity * dt;

        return position;
    }

    // 设置阻尼比（1.0 = 临界阻尼，<1.0 = 欠阻尼振荡，>1.0 = 过阻尼缓慢到达）
    void setDampingRatio(float ratio, float frequency) {
        stiffness = frequency * frequency;              // k = ω²
        damping = 2.0f * ratio * frequency;             // d = 2ζω
    }

    float getPosition() const { return position; }
    float getVelocity() const { return velocity; }
    void setPosition(float p) { position = p; }
};

// 三维弹簧阻尼系统（相机跟随、UI弹性）
class SpringDamper3D {
    Vec3 position;
    Vec3 velocity;
    float stiffness;
    float damping;

public:
    SpringDamper3D(float stiffness = 200.0f, float damping = 10.0f)
        : position{0,0,0}, velocity{0,0,0},
          stiffness(stiffness), damping(damping) {}

    Vec3 update(Vec3 target, float dt) {
        Vec3 displacement = position - target;
        Vec3 springForce = displacement * (-stiffness);
        Vec3 dampingForce = velocity * (-damping);
        Vec3 acceleration = springForce + dampingForce;

        velocity = velocity + acceleration * dt;
        position = position + velocity * dt;

        return position;
    }

    void setDampingRatio(float ratio, float frequency) {
        stiffness = frequency * frequency;
        damping = 2.0f * ratio * frequency;
    }

    Vec3 getPosition() const { return position; }
    void setPosition(Vec3 p) { position = p; }
};
```

## 基于缓动的动画系统

```cpp
// 动画轨道
struct Animation {
    Vec3 startPos, endPos;
    float duration;        // 总时长（秒）
    float elapsed;         // 已过时间
    float (*easing)(float); // 缓动函数指针
    bool loop;

    Vec3 update(float dt) {
        elapsed += dt;
        float rawT = elapsed / duration;
        if (loop) rawT = fmodf(rawT, 1.0f);
        else rawT = std::min(rawT, 1.0f);

        float easedT = easing(rawT);
        return lerpVec3(startPos, endPos, easedT);
    }

    bool isDone() const { return !loop && elapsed >= duration; }
    float getProgress() const { return std::min(elapsed / duration, 1.0f); }
};

// 动画系统管理器
class AnimationSystem {
    std::vector<Animation*> activeAnimations;

public:
    Animation* Animate(Vec3 from, Vec3 to, float duration,
                       float (*easing)(float) = easeInOutCubic,
                       bool loop = false)
    {
        auto* anim = new Animation{from, to, duration, 0, easing, loop};
        activeAnimations.push_back(anim);
        return anim;
    }

    void Update(float dt) {
        for (int i = activeAnimations.size() - 1; i >= 0; i--) {
            activeAnimations[i]->update(dt);
            if (activeAnimations[i]->isDone()) {
                delete activeAnimations[i];
                activeAnimations.erase(activeAnimations.begin() + i);
            }
        }
    }

    // 链式动画（序列）
    void AnimateSequence(std::vector<Animation> sequence) {
        // 一个完成后触发下一个
    }
};
```

## 使用缓动的 UI 动画示例

```cpp
// UI 弹窗弹出动画
class UIPopup {
    Vec3 scale;
    float alpha;
    SpringDamper1D scaleSpring;
    Animation* fadeInAnim;

public:
    void Show() {
        // 缩放弹簧动画：从0弹到1
        scaleSpring.setDampingRatio(0.5f, 8.0f); // 欠阻尼，有弹性效果
        scaleSpring.setPosition(0.0f);

        // 透明度渐变
        fadeInAnim = animationSystem.Animate(0.0f, 1.0f, 0.3f, easeOutCubic);
    }

    void Hide() {
        scaleSpring.setDampingRatio(1.0f, 10.0f); // 过阻尼，平滑消失
        fadeInAnim = animationSystem.Animate(alpha, 0.0f, 0.2f, easeInCubic);
    }

    void Update(float dt) {
        scale = Vec3{1,1,1} * scaleSpring.update(1.0f, dt);
        alpha = fadeInAnim->update(dt).x; // 简化
    }
};

// 血条变化动画
class HealthBar {
    float displayHealth;   // 显示的血量（缓动）
    float actualHealth;    // 实际血量（立即变化）
    SpringDamper1D healthSpring;

public:
    HealthBar() {
        healthSpring.setDampingRatio(0.7f, 5.0f);
    }

    void TakeDamage(float amount) {
        actualHealth -= amount;
        actualHealth = std::max(0.0f, actualHealth);
        // 实际血量立即下降，显示血量缓动跟随
    }

    void Update(float dt) {
        displayHealth = healthSpring.update(actualHealth, dt);
    }

    // 视觉效果：实际血量白色条立即变化，显示血量红色条缓动下降
};
```

## 方案对比

| 方案 | 平滑度 | 性能 | 可控性 | 物理真实感 | 典型用途 |
|------|--------|------|--------|-----------|---------|
| 线性插值 | 低 | 最高 | 低 | 无 | 进度条 |
| Ease In/Out | 中 | 高 | 中 | 低 | UI动画 |
| Bezier 缓动 | 高 | 中 | 高 | 低 | CSS动画 |
| 弹簧阻尼 | 极高 | 中 | 中 | 高 | 相机跟随 |
| Hermite/样条 | 极高 | 中 | 极高 | 中 | 路径动画 |
| 物理模拟 | 极高 | 低 | 低 | 极高 | 布料、绳索 |

## 常见陷阱与解决方案

1. **帧率依赖**：直接 `t += dt` 在不同帧率下动画速度不同。解决方案：`t = elapsed / duration`，用累计时间而非增量
2. **Lerp 不等于缓动**：`Lerp(a, b, t)` 是线性的，`Lerp(a, b, ease(t))` 才是缓动。不要混淆
3. **过度缓动**：UI 操作使用过多缓动会让界面感觉"粘滞"。关键UI操作应快速响应
4. **t 超出 [0,1]**：某些缓动函数在 t>1 时行为异常（如弹性缓动的过冲）。解决方案：clamp t 或设计好动画结束条件
5. **弹簧发散**：弹簧刚度太大或时间步太大会导致弹簧系统发散。解决方案：使用隐式积分或减小 dt
6. **Lerp 持续逼近问题**：`pos = Lerp(pos, target, 0.1f)` 永远到不了 target。解决方案：判断距离小于阈值时直接设为目标值

```csharp
IEnumerator FadeIn(CanvasGroup canvas, float duration) {
    float elapsed = 0;
    while (elapsed < duration) {
        elapsed += Time.deltaTime;
        float t = Mathf.Clamp01(elapsed / duration);
        canvas.alpha = Mathf.Lerp(0, 1, t * t); // EaseIn
        yield return null;
    }
    canvas.alpha = 1;
}
```

## 实际使用案例

- **UI动画**：菜单弹出（EaseOutBack）、按钮缩放（EaseOutQuad）、通知渐入（EaseOutCubic）
- **相机跟随**：SmoothDamp / 弹簧阻尼实现平滑跟随
- **血条变化**：实际扣血立即生效，显示血量缓动下降
- **角色移动曲线**：跳跃弧线、冲刺加速/减速
- **武器后坐力**：EaseOutExpo 快速回弹效果
- **关卡切换**：屏幕渐黑渐白使用线性或 EaseInOut
- **技能释放**：施法条缓动填充、冷却时间旋转指示器
