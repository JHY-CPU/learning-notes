# 四元数与旋转

## 核心概念

四元数是一种扩展复数，由一个实部和三个虚部组成：`q = w + xi + yj + zk`，表示为`(w, x, y, z)`。单位四元数可以表示任意3D旋转，避免了欧拉角的万向锁问题，是游戏引擎中旋转的首选表示方式。

### 为什么不用欧拉角

1. **万向锁（Gimbal Lock）**：当某个旋转轴旋转90度后，另外两个轴重合，丢失一个自由度。例如：俯仰角为90度时，偏航和滚转等效
2. **插值困难**：欧拉角的线性插值路径不自然，可能绕远路
3. **组合复杂**：多个旋转组合需要大量三角函数计算
4. **歧义性**：同一旋转有多种欧拉角表示（0度和360度等效，但数值不同）

### 四元数基本性质

- **旋转角度θ，绕单位轴n**：`q = (cos(θ/2), sin(θ/2) * n)`
- **单位四元数**：`|q| = 1`（表示旋转必须归一化）
- **逆四元数**：`q^(-1) = (w, -x, -y, -z)`（对单位四元数）
- **共轭**：虚部取反
- **恒等旋转**：`q = (1, 0, 0, 0)`

### 四元数乘法

旋转合成：`q_total = q2 * q1`（先应用q1，再应用q2）

乘法公式：
```
w = w1*w2 - x1*x2 - y1*y2 - z1*z2
x = w1*x2 + x1*w2 + y1*z2 - z1*y2
y = w1*y2 - x1*z2 + y1*w2 + z1*x2
z = w1*z2 + x1*y2 - y1*x2 + z1*w2
```

### Slerp球面线性插值

在两个旋转之间沿最短球面路径插值：

`Slerp(q1, q2, t) = (sin((1-t)*omega) / sin(omega)) * q1 + (sin(t*omega) / sin(omega)) * q2`

其中 `omega = arccos(q1 . q2)`

### 旋转应用

对向量v的旋转：`v' = q * v * q^(-1)`（等价于旋转矩阵乘法但更高效）

## 完整四元数库实现

```cpp
#include <cmath>
#include <cassert>

struct Vec3 {
    float x, y, z;
    Vec3() : x(0), y(0), z(0) {}
    Vec3(float x, float y, float z) : x(x), y(y), z(z) {}

    Vec3 operator+(const Vec3& o) const { return {x+o.x, y+o.y, z+o.z}; }
    Vec3 operator-(const Vec3& o) const { return {x-o.x, y-o.y, z-o.z}; }
    Vec3 operator*(float s) const { return {x*s, y*s, z*s}; }
    Vec3 operator-() const { return {-x, -y, -z}; }

    float dot(const Vec3& o) const { return x*o.x + y*o.y + z*o.z; }
    Vec3 cross(const Vec3& o) const {
        return {y*o.z - z*o.y, z*o.x - x*o.z, x*o.y - y*o.x};
    }
    float lengthSq() const { return x*x + y*y + z*z; }
    float length() const { return sqrt(lengthSq()); }
    Vec3 normalized() const {
        float l = length();
        return l > 1e-6f ? Vec3{x/l, y/l, z/l} : Vec3{0, 0, 0};
    }
};

struct Quaternion {
    float w, x, y, z;

    Quaternion() : w(1), x(0), y(0), z(0) {}
    Quaternion(float w, float x, float y, float z) : w(w), x(x), y(y), z(z) {}

    // ========== 基本运算 ==========

    Quaternion operator*(const Quaternion& q) const {
        return {
            w*q.w - x*q.x - y*q.y - z*q.z,
            w*q.x + x*q.w + y*q.z - z*q.y,
            w*q.y - x*q.z + y*q.w + z*q.x,
            w*q.z + x*q.y - y*q.x + z*q.w
        };
    }

    Quaternion operator+(const Quaternion& q) const {
        return {w+q.w, x+q.x, y+q.y, z+q.z};
    }

    Quaternion operator*(float s) const {
        return {w*s, x*s, y*s, z*s};
    }

    float dot(const Quaternion& q) const {
        return w*q.w + x*q.x + y*q.y + z*q.z;
    }

    Quaternion conjugate() const { return {w, -x, -y, -z}; }

    float lengthSq() const { return w*w + x*x + y*y + z*z; }
    float length() const { return sqrt(lengthSq()); }

    Quaternion normalized() const {
        float len = length();
        if (len < 1e-6f) return {1, 0, 0, 0};
        return {w/len, x/len, y/len, z/len};
    }

    Quaternion inverse() const {
        float lenSq = lengthSq();
        if (lenSq < 1e-6f) return {1, 0, 0, 0};
        return {w/lenSq, -x/lenSq, -y/lenSq, -z/lenSq};
    }

    // ========== 旋转应用 ==========

    // 对向量的旋转：v' = q * v * q^(-1)
    // 展开为矩阵形式更高效
    Vec3 rotate(const Vec3& v) const {
        // q * v * conj(q) 的展开形式
        float xx = x*x, yy = y*y, zz = z*z;
        float xy = x*y, xz = x*z, yz = y*z;
        float wx = w*x, wy = w*y, wz = w*z;

        return {
            (1.0f - 2.0f*(yy + zz)) * v.x + 2.0f*(xy - wz) * v.y + 2.0f*(xz + wy) * v.z,
            2.0f*(xy + wz) * v.x + (1.0f - 2.0f*(xx + zz)) * v.y + 2.0f*(yz - wx) * v.z,
            2.0f*(xz - wy) * v.x + 2.0f*(yz + wx) * v.y + (1.0f - 2.0f*(xx + yy)) * v.z
        };
    }

    // 获取旋转轴和角度
    Vec3 getAxis() const {
        float s = sqrt(1.0f - w*w);
        if (s < 1e-6f) return {1, 0, 0}; // 无旋转
        return {x/s, y/s, z/s};
    }

    float getAngle() const {
        return 2.0f * acos(std::fmax(-1.0f, std::fmin(1.0f, w)));
    }

    // ========== 构造方法 ==========

    static Quaternion fromAxisAngle(Vec3 axis, float angleRad) {
        float half = angleRad * 0.5f;
        float s = sin(half);
        Vec3 a = axis.normalized();
        return {cos(half), a.x * s, a.y * s, a.z * s};
    }

    static Quaternion fromEuler(float pitchRad, float yawRad, float rollRad) {
        // 顺序: Z(roll) * Y(yaw) * X(pitch)
        float cp = cos(pitchRad * 0.5f), sp = sin(pitchRad * 0.5f);
        float cy = cos(yawRad * 0.5f),   sy = sin(yawRad * 0.5f);
        float cr = cos(rollRad * 0.5f),  sr = sin(rollRad * 0.5f);

        return {
            cr*cp*cy + sr*sp*sy,
            sr*cp*cy - cr*sp*sy,
            cr*sp*cy + sr*cp*sy,
            cr*cp*sy - sr*sp*cy
        };
    }

    // 从两个向量构造旋转四元数（从from方向旋转到to方向）
    static Quaternion fromTwoVectors(Vec3 from, Vec3 to) {
        from = from.normalized();
        to = to.normalized();

        float d = from.dot(to);

        // 向量几乎相反（180度旋转）
        if (d < -0.999999f) {
            // 找一个与from不共线的向量
            Vec3 ortho = {1, 0, 0};
            if (abs(from.x) > 0.9f)
                ortho = {0, 1, 0};
            Vec3 axis = from.cross(ortho).normalized();
            return fromAxisAngle(axis, M_PI);
        }

        // 向量几乎相同（无旋转）
        if (d > 0.999999f) {
            return {1, 0, 0, 0};
        }

        Vec3 cross = from.cross(to);
        return {
            1.0f + d,
            cross.x,
            cross.y,
            cross.z
        }.normalized();
    }

    // 从旋转矩阵构造
    static Quaternion fromMatrix(float m[3][3]) {
        float tr = m[0][0] + m[1][1] + m[2][2];
        Quaternion q;

        if (tr > 0) {
            float s = sqrt(tr + 1.0f) * 2.0f;
            q.w = 0.25f * s;
            q.x = (m[2][1] - m[1][2]) / s;
            q.y = (m[0][2] - m[2][0]) / s;
            q.z = (m[1][0] - m[0][1]) / s;
        } else if (m[0][0] > m[1][1] && m[0][0] > m[2][2]) {
            float s = sqrt(1.0f + m[0][0] - m[1][1] - m[2][2]) * 2.0f;
            q.w = (m[2][1] - m[1][2]) / s;
            q.x = 0.25f * s;
            q.y = (m[0][1] + m[1][0]) / s;
            q.z = (m[0][2] + m[2][0]) / s;
        } else if (m[1][1] > m[2][2]) {
            float s = sqrt(1.0f + m[1][1] - m[0][0] - m[2][2]) * 2.0f;
            q.w = (m[0][2] - m[2][0]) / s;
            q.x = (m[0][1] + m[1][0]) / s;
            q.y = 0.25f * s;
            q.z = (m[1][2] + m[2][1]) / s;
        } else {
            float s = sqrt(1.0f + m[2][2] - m[0][0] - m[1][1]) * 2.0f;
            q.w = (m[1][0] - m[0][1]) / s;
            q.x = (m[0][2] + m[2][0]) / s;
            q.y = (m[1][2] + m[2][1]) / s;
            q.z = 0.25f * s;
        }
        return q;
    }

    // 前方向和上方向构造
    static Quaternion fromLookRotation(Vec3 forward, Vec3 up = {0,1,0}) {
        Vec3 f = forward.normalized();
        Vec3 r = up.cross(f).normalized();
        Vec3 u = f.cross(r);

        float rotMatrix[3][3] = {
            {r.x, u.x, f.x},
            {r.y, u.y, f.y},
            {r.z, u.z, f.z}
        };
        return fromMatrix(rotMatrix);
    }
};
```

## 插值算法详解

```cpp
// ========== Slerp球面线性插值 ==========
Quaternion slerp(Quaternion q1, Quaternion q2, float t) {
    float dot = q1.dot(q2);

    // 确保走最短路径（q和-q表示同一旋转）
    if (dot < 0) {
        q2 = {-q2.w, -q2.x, -q2.y, -q2.z};
        dot = -dot;
    }

    // 当两四元数非常接近时，用线性插值避免除零
    if (dot > 0.9995f) {
        Quaternion r = q1 + (q2 - q1) * t;
        return r.normalized();
    }

    float theta = acos(dot);
    float sinTheta = sin(theta);
    float w1 = sin((1-t) * theta) / sinTheta;
    float w2 = sin(t * theta) / sinTheta;

    return {
        w1*q1.w + w2*q2.w,
        w1*q1.x + w2*q2.x,
        w1*q1.y + w2*q2.y,
        w1*q1.z + w2*q2.z
    };
}

// ========== Nlerp归一化线性插值（更快但不等速） ==========
Quaternion nlerp(Quaternion q1, Quaternion q2, float t) {
    float dot = q1.dot(q2);
    if (dot < 0) {
        q2 = {-q2.w, -q2.x, -q2.y, -q2.z};
    }
    Quaternion r = q1 + (q2 - q1) * t;
    return r.normalized();
}

// ========== Squad球面四次插值（用于平滑路径） ==========
// 用于动画中沿一系列旋转的平滑过渡
Quaternion squad(Quaternion q0, Quaternion q1, Quaternion q2, Quaternion q3, float t) {
    // 计算中间控制四元数
    Quaternion s1 = q1 * (q1.inverse() * q2).log().exp() * 0.25f;
    Quaternion s2 = q2 * (q2.inverse() * q3).log().exp() * 0.25f;

    // 两次Slerp的Slerp
    Quaternion slerp1 = slerp(q1, q2, t);
    Quaternion slerp2 = slerp(s1, s2, t);
    return slerp(slerp1, slerp2, 2*t*(1-t));
}

// ========== 角速度转四元数 ==========
// 角速度向量omega（方向=旋转轴，大小=角速度rad/s）
// 转换为时间dt内的旋转四元数
Quaternion angularVelocityToQuaternion(Vec3 omega, float dt) {
    float angle = omega.length() * dt;
    if (angle < 1e-6f) return {1, 0, 0, 0}; // 无旋转
    Vec3 axis = omega.normalized();
    return Quaternion::fromAxisAngle(axis, angle);
}

// ========== 四元数转角速度 ==========
Vec3 quaternionToAngularVelocity(Quaternion q, float dt) {
    if (dt < 1e-6f) return {0, 0, 0};
    float angle = q.getAngle();
    Vec3 axis = q.getAxis();
    return axis * (angle / dt);
}
```

## 四元数转欧拉角

```cpp
// 将四元数转换为欧拉角（弧度）
Vec3 toEulerAngles(Quaternion q) {
    Vec3 euler;

    // Pitch (X轴旋转)
    float sinp = 2.0f * (q.w*q.x + q.y*q.z);
    float cosp = 1.0f - 2.0f*(q.x*q.x + q.y*q.y);
    euler.x = atan2(sinp, cosp);

    // Yaw (Y轴旋转)
    float siny = 2.0f * (q.w*q.y - q.z*q.x);
    if (abs(siny) >= 1.0f)
        euler.y = copysign(M_PI * 0.5f, siny); // 万向锁
    else
        euler.y = asin(siny);

    // Roll (Z轴旋转)
    float sinr = 2.0f * (q.w*q.z + q.x*q.y);
    float cosr = 1.0f - 2.0f*(q.y*q.y + q.z*q.z);
    euler.z = atan2(sinr, cosr);

    return euler; // 弧度
}

// 转为角度
Vec3 toEulerDegrees(Quaternion q) {
    Vec3 rad = toEulerAngles(q);
    return {
        rad.x * 180.0f / M_PI,
        rad.y * 180.0f / M_PI,
        rad.z * 180.0f / M_PI
    };
}
```

## 游戏应用场景

1. **相机旋转**：第一人称相机的平滑旋转控制，避免万向锁
2. **角色朝向**：3D角色平滑转向目标方向，使用Slerp/Nlerp
3. **动画插值**：骨骼动画中关节旋转的关键帧插值
4. **物理旋转**：刚体角速度的积分表示
5. **摄像机轨道**：过场动画中相机沿四元数路径运动
6. **IK解算**：逆运动学中关节旋转的迭代计算

## 实际游戏案例

### 案例：第三人称相机的平滑旋转

```cpp
class ThirdPersonCamera {
    Quaternion currentRotation;
    float rotationSpeed = 5.0f;

    void update(float dt, Vec3 playerPos, Vec2 mouseInput) {
        // 从鼠标输入计算目标旋转
        float yawDelta = mouseInput.x * sensitivity * dt;
        float pitchDelta = mouseInput.y * sensitivity * dt;

        Quaternion yawRot = Quaternion::fromAxisAngle({0,1,0}, yawDelta);
        Quaternion pitchRot = Quaternion::fromAxisAngle({1,0,0}, pitchDelta);

        // 累积旋转
        currentRotation = yawRot * currentRotation * pitchRot;

        // 限制俯仰角
        Vec3 euler = toEulerAngles(currentRotation);
        euler.x = std::fmax(-80.0f * DEG2RAD, std::fmin(80.0f * DEG2RAD, euler.x));
        currentRotation = Quaternion::fromEuler(euler.x, euler.y, 0);

        // 平滑插值到目标旋转
        Quaternion targetRot = currentRotation;
        currentRotation = slerp(currentRotation, targetRot, dt * rotationSpeed);

        // 应用旋转
        Vec3 offset = currentRotation.rotate(Vec3{0, 2, -5});
        cameraPos = playerPos + offset;
    }
};
```

void smoothLookAt(Vec3 targetPos, float dt) {
    Vec3 dir = (targetPos - GetPosition()).normalized();
    if (dir.lengthSq() < 1e-6f) return;

    // 目标旋转
    Quaternion targetRot = Quaternion::fromLookRotation(dir);

    // 插值速度：角度越大转越快
    float angle = acosf(std::fmin(1.0f, currentRotation.dot(targetRot)));
    float speed = std::fmax(2.0f, angle * 3.0f); // 自适应速度

    currentRotation = slerp(currentRotation, targetRot, dt * speed);
    SetRotation(currentRotation);
}

## 常见陷阱

- **四元数必须归一化**：使用过程中需定期归一化，浮点误差累积会导致缩放变形
- **Slerp的dot阈值**：当两四元数非常接近时，Slerp分母趋近零，需切换到Nlerp
- **乘法顺序**：`q2*q1` 表示先q1后q2，与矩阵乘法顺序相反（左乘 vs 右乘）
- **双覆盖**：`q`和`-q`表示同一旋转，但Slerp路径不同。需检查点积符号决定走短弧还是长弧
- **万向锁转换**：四元数本身没有万向锁，但转换为欧拉角时仍会在pitch=90度时出现奇点
- **归一化累积误差**：连续旋转操作后应定期归一化，或使用乘法的归一化版本
- **从两个向量构造的退化情况**：当两向量方向相反时，叉积为零，需特殊处理
