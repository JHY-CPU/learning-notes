# 104-游戏引擎Unreal

## 模块概述

本模块系统学习Unreal Engine的核心知识体系，涵盖从引擎架构到高级功能的完整路径。Unreal Engine以高端画质和完整的工具链著称，广泛应用于3A游戏、影视和建筑可视化领域。

## 知识体系

| 编号 | 主题 | 核心内容 |
|------|------|----------|
| 01 | Unreal引擎架构概览 | 模块系统、引擎启动流程、编辑器界面 |
| 02 | C++与蓝图 | 蓝图可视化脚本、C++与蓝图互操作 |
| 03 | Actor与Component | Actor生命周期、SceneComponent层级、Tick机制 |
| 04 | UObject与反射系统 | UPROPERTY/UFUNCTION宏、垃圾回收、反射 |
| 05 | GamePlay框架 | GameMode/GameState/PlayerController/Pawn |
| 06 | 物理系统(Chaos) | Chaos物理引擎、碰撞响应、Constraint |
| 07 | UMG UI系统 | Widget Blueprint、UMG组件、UI动画 |
| 08 | 动画系统 | AnimGraph、State Machine、Montage、IK |
| 09 | AI系统 | AIController、Behavior Tree、Blackboard、EQS |
| 10 | 网络同步 | Replication、RPC、Ownership、Relevancy |
| 11 | 材质与渲染 | Material Editor、PBR材质、材质实例、LOD |
| 12 | 插件与模块开发 | 模块结构、Build.cs、插件创建、Slate UI |

## 学习路径建议

1. **基础阶段** (01-04): 理解引擎架构、C++与蓝图、对象模型
2. **框架阶段** (05-06): 掌握GamePlay框架和物理系统
3. **表现阶段** (07-08): UI系统和动画系统
4. **进阶阶段** (09-12): AI、网络、材质和模块开发

## 技术栈

- 引擎版本: Unreal Engine 5.4+
- 编程语言: C++20、Blueprints
- 物理引擎: Chaos Physics
- 渲染: Nanite、Lumen、Virtual Shadow Maps
- 构建系统: Unreal Build Tool (UBT)
