# 材质与渲染

## 核心概念

Unreal材质系统基于Material Editor可视化节点编辑，使用PBR（基于物理的渲染）工作流。材质定义表面如何与光照交互，通过材质实例实现参数化复用。

## Material Editor

材质通过节点图定义，核心输入引脚：

```cpp
// 材质核心输入（Base Color, Metallic, Roughness, Normal, Emissive）
// Base Color - 表面基础颜色（sRGB，非HDR）
// Metallic - 金属度 (0=非金属, 1=金属)
// Roughness - 粗糙度 (0=光滑镜面, 1=完全粗糙)
// Normal - 法线贴图（切线空间）
// Emissive Color - 自发光（HDR值，可>1）
// Opacity - 不透明度（半透明材质）
// World Position Offset - 顶点位移

// 常用材质节点
// Texture Sample - 采样纹理
// Lerp - 线性插值
// Power - 幂运算
// Fresnel - 菲涅尔效果
// Panner - UV平移（滚动纹理）
// Rotator - UV旋转
// Append - 合并标量为向量
// Component Mask - 分量提取
// Multiply - 乘法
// Add - 加法
// Clamp - 限制范围
// Saturate - 限制到0-1
```

```cpp
// C++动态创建和修改材质
UCLASS()
class AMaterialController : public AActor
{
    GENERATED_BODY()

    void CreateDynamicMaterial()
    {
        // 创建动态材质实例
        UMaterialInstanceDynamic* DynMat = UMaterialInstanceDynamic::Create(
            BaseMaterial, this);

        // 设置参数
        DynMat->SetVectorParameterValue(TEXT("BaseColor"), FLinearColor::Red);
        DynMat->SetScalarParameterValue(TEXT("Roughness"), 0.5f);
        DynMat->SetTextureParameterValue(TEXT("NormalMap"), NormalTexture);

        // 应用到Mesh
        MeshComp->SetMaterial(0, DynMat);
    }

    // Material Parameter Collection（跨材质共享参数）
    void UpdateGlobalParams()
    {
        UMaterialParameterCollection* Collection = LoadObject<UMaterialParameterCollection>(
            nullptr, TEXT("/Game/Materials/MPC_Global"));

        if (Collection)
        {
            // 设置全局标量参数（如时间、天气强度）
            GetWorld()->GetParameterCollectionInstance(Collection)
                ->SetScalarParameterValue(TEXT("RainIntensity"), 0.7f);
        }
    }

    UPROPERTY(EditDefaultsOnly)
    TObjectPtr<UMaterialInterface> BaseMaterial;

    UPROPERTY(VisibleAnywhere)
    TObjectPtr<UStaticMeshComponent> MeshComp;
};
```

## 常见陷阱与最佳实践

1. **材质实例优于动态材质**: 静态材质实例开销极低，尽量预创建
2. **贴图压缩格式**: BaseColor用BC7/DXT1，Normal用BC5，不要用默认压缩
3. **材质变体过多**: 每个独特材质增加Draw Call，合理合并材质
4. **移动端注意**: 移动端对材质复杂度有硬限制，简化数学运算和贴图采样数
5. **材质排序**: 半透明材质按深度排序消耗大，尽量减少重叠

## 与其他系统的关联

- **光照系统**: 材质与光照类型（方向光、点光、聚光）交互
- **后处理**: 后处理体积影响材质的最终呈现
- **Niagara**: 粒子系统可引用材质参数
