// 图论算法复杂度汇总

//   Dijkstra: O(V²) 或 O(E log V)

//   SPFA: O(kE) 平均，O(VE) 最坏
// 最小生成树:
//   Kruskal: O(E log E)
//   Prim: O(V²) 或 O(E log V)
// 强连通分量: O(V+E)
// 最大流:
//   Dinic: O(V²E)
//   对单位容量图: O(min(V^{2/3}, E^{1/2}) E)
// 二分图匹配: O(VE)
