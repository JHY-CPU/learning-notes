尾递归优化

什么是尾递归

尾递归（Tail Recursion）是指递归调用发生在函数的最后一步（即 return 语句中只有递归调用，没有额外的计算操作）。

非尾递归：递归后还有乘法操作 function factorial(n) { if (n <= 1) return 1; return n * factorial(n - 1); // 有额外乘法 } 尾递归：递归调用是最后一步 function factorialTail(n, acc = 1) { if (n <= 1) return acc; return factorialTail(n - 1, n * acc); // 直接返回递归结果 }

尾递归优化（TCO）

支持 TCO 的编译器/解释器可以复用当前的栈帧，而不是创建新的栈帧，从而将递归的空间复杂度从 O(n) 降为 O(1)。

JavaScript 在严格模式（"use strict"）下，部分引擎（如 Safari）支持尾递归优化。V8 引擎（Chrome/Node.js）目前不直接支持 TCO。

尾递归改写技巧

将普通递归改写为尾递归的通用方法：

1. 引入累加器参数：在函数签名中增加一个参数来保存中间结果

2. 将计算结果作为参数传递：把当前计算的结果作为参数传递给下一层递归

3. 保证最后一步只有递归调用：return 语句中只包含递归函数调用
