递归基础

什么是递归

递归是一种通过让函数调用自身来解决问题的方法。它将一个大问题分解为一个或多个相似的子问题，直到子问题简单到可以直接求解（到达基本情况）。

递归的核心思想：将问题分解为更小的相同问题，直到问题规模小到可以直接求解。

经典示例：阶乘

n! = n x (n-1) x (n-2) x ... x 1，且 0! = 1

function factorial(n) { // 基本情况：0! = 1 if (n <= 1) return 1; // 递归步骤：n! = n x (n-1)! return n * factorial(n - 1); }

递归的执行过程

以 factorial(5) 为例，递归的执行过程分为"递推"和"回归"两个阶段：

factorial(5) = 5 * factorial(4)
           = 5 * (4 * factorial(3))
           = 5 * (4 * (3 * factorial(2)))
           = 5 * (4 * (3 * (2 * factorial(1))))
           = 5 * (4 * (3 * (2 * 1)))
           = 5 * (4 * (3 * 2))
           = 5 * (4 * 6)
           = 5 * 24
           = 120
