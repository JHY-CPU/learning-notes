递归树分析

什么是递归树

递归树是一种可视化递归算法执行过程的方法。树中的每个节点代表一次递归调用，节点的子节点代表该调用产生的子调用。通过递归树可以直观地分析算法的时间复杂度。

递归树的高度代表递归深度，宽度（每层节点数）代表该层的子问题数量，每层总代价之和即为总时间复杂度。

归并排序递归树

归并排序递推式：T(n) = 2T(n/2) + n, T(1) = 1

n (代价: n)
/ \
n/2
n/2 (总代价: n)
/ \ / \
n/4 n/4 n/4 n/4 (总代价: n)
... 共 log₂(n) 层 ...

每层总代价均为 n，共 log₂(n) 层，故总代价为 n * log₂(n)。
