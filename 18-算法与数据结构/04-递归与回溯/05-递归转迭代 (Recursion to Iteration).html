递归转迭代

递归转迭代概述

任何递归算法都可以转换为迭代算法（使用循环和显式栈）。迭代通常更高效，但代码可能更复杂。

转换动机：避免栈溢出、提高性能、节省内存。在嵌入式系统或栈空间有限的环境中尤为重要。

方法一：直接循环转换

对于尾递归或简单递归，可以直接用循环替代。

// 递归 function sum(n) { if (n <= 0) return 0; return n + sum(n - 1); } // 迭代 function sumIter(n) { let result = 0; for (let i = 1; i <= n; i++) { result += i; } return result; }

方法二：使用显式栈

对于非尾递归，可以模拟调用栈进行转换。

// 模拟栈帧结构 function factorialIter(n) { const stack = []; let result = 1; while (n > 0) { stack.push(n); n--; } while (stack.length > 0) { result *= stack.pop(); } return result; }
