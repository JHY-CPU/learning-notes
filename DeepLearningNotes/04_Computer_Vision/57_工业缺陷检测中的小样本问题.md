# 57_工业缺陷检测中的小样本问题

## 核心概念

- **工业缺陷检测（Industrial Defect Detection）**：在生产线中自动检测产品表面的划痕、凹坑、气泡、裂纹等缺陷。通常使用视觉检测系统，但面临"正常品多、缺陷品极少"的极端数据不平衡问题。
- **小样本问题（Few-Shot Problem）**：在工业场景中，严重缺陷很少出现，导致可用标注数据极度稀缺（每个类别可能只有几个样本）。传统深度学习方法在大数据下有效，小样本下容易过拟合。
- **数据增强**：针对缺陷样本少的问题，使用旋转、翻转、色彩抖动、弹性变形等几何和光度变换扩充缺陷样本。CutMix和Mixup也被用于合成新的缺陷样本。
- **迁移学习（Transfer Learning）**：在大规模预训练数据集（ImageNet）上预训练的分类/检测模型，在工业检测的小样本场景下微调。预训练模型学到的基础视觉特征（边缘、纹理、形状）可以迁移到缺陷检测任务。
- **异常检测方法（Anomaly Detection）**：将缺陷检测视为异常检测问题——只用正常样本训练，学习"正常"的数据分布，任何偏离正常分布的样本被视为缺陷。代表性方法：SPADE、PaDiM、PatchCore。
- **元学习（Meta-Learning）**：通过"学会学习"的方式，在大量相似任务（如不同产品线）上训练，使模型能够从少量缺陷样本中快速适应新的缺陷类别。

## 数学推导

**小样本学习的定义：**
在 $N$-way $K$-shot 设置中，训练包含 $N$ 个类别，每个类别 $K$ 个标注样本（$K$ 通常很小，如1-5）。模型需要泛化到测试集中的新样本。

**原型网络（Prototypical Networks）用于小样本缺陷检测：**
计算每个类别的原型（prototype）——该类所有支持样本特征的平均：
$$
c_n = \frac{1}{K} \sum_{i=1}^K f_\theta(x_i^{(n)})
$$

其中 $f_\theta$ 是特征提取网络，$x_i^{(n)}$ 是类别 $n$ 的第 $i$ 个支持样本。

查询样本 $x_q$ 属于类别 $n$ 的概率：
$$
p(y = n | x_q) = \frac{\exp(-d(f_\theta(x_q), c_n))}{\sum_{n'=1}^N \exp(-d(f_\theta(x_q), c_{n'}))}
$$

其中 $d$ 是距离度量（如欧氏距离）。

**PatchCore（基于memory bank的异常检测）：**
从训练集的正常图像中提取所有patch特征，构建memory bank $\mathcal{M}$：
$$
\mathcal{M} = \{f(I)[i,j] | I \in \mathcal{I}_{train}, (i,j) \in \text{grid}\}
$$

测试时，对于图像 $I$ 的每个patch特征 $f(I)[i,j]$，找到其在memory bank中的最近邻，计算异常得分：
$$
s(i,j) = \max_{m \in \mathcal{M}} d(f(I)[i,j], m)
$$

其中 $d$ 通常是余弦距离或L2距离。得分高于阈值的patch被视为缺陷区域。

**知识蒸馏用于缺陷检测：**
教师网络（在正常数据上预训练）指导学生网络（从正常+缺陷数据训练），缺陷区域的师生差异会大于正常区域：
$$
\mathcal{L} = \|f_{student}(I) - f_{teacher}(I)\|_2^2
$$

缺陷区域因为不在教师的"正常知识"范围内，师生差距较大，从而被检测出来。

## 直观理解

工业缺陷检测的小样本问题可以类比为"教一个质检员从5个案例中学会识别所有类型的瑕疵"。传统深度学习需要成千上万的标注数据，但工业中某些严重缺陷可能几个月才出现一次。

解决这一问题的思路有三条：
1. **迁移学习**：让质检员先去ImageNet（通用视觉数据集）上"实习"（预训练），学会看边缘、纹理、形状等基础视觉模式，再到你的工厂做"岗前培训"（微调），只需要少量缺陷样本。
2. **异常检测**：不教质检员"什么是缺陷"，只教他"什么是正常产品"。任何看起来"不对劲"的都标记为异常。这就像只告诉你"正常的螺丝表面是光滑的"，表面有划痕或色斑就是异常——不需要知道具体异常长什么样。
3. **元学习**：让质检员在多种不同类型的产品线（手机壳、电路板、轴承）上积累经验，学会"快速适应"新产品线。这相当于培养了一个"万能质检员"——给他看几个新产品的缺陷样例，他就能在新产品线上工作了。

## 代码示例

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class PatchCore(nn.Module):
    """PatchCore 异常检测 (简化版)"""
    def __init__(self, backbone='resnet18'):
        super().__init__()
        # 使用预训练ResNet作为特征提取器
        resnet = torchvision.models.resnet18(pretrained=True)
        self.features = nn.Sequential(*list(resnet.children())[:-2])  # 到layer3
        
        # Memory Bank (用于存储正常patch特征)
        self.memory_bank = []
        self.register_buffer('memory_tensor', torch.zeros(0))

    def extract_patches(self, x):
        """提取patch级特征"""
        feat = self.features(x)  # (B, 512, H/32, W/32)
        B, C, H, W = feat.shape
        # 每个空间位置作为一个patch特征
        patches = feat.permute(0, 2, 3, 1).reshape(B * H * W, C)
        return patches, (H, W)

    @torch.no_grad()
    def fit(self, dataloader):
        """用正常图像构建memory bank"""
        features = []
        for x, _ in dataloader:
            patches, _ = self.extract_patches(x)
            features.append(patches)
        all_features = torch.cat(features, dim=0)  # (N_patches, C)
        
        # 随机子采样减少memory bank大小 (coreset sampling的简化)
        n_samples = min(len(all_features), 10000)
        indices = torch.randperm(len(all_features))[:n_samples]
        self.memory_tensor = all_features[indices]
        print(f"Memory Bank: {self.memory_tensor.shape}")

    @torch.no_grad()
    def detect(self, x):
        """检测异常区域"""
        patches, (H, W) = self.extract_patches(x)
        
        # 对每个patch, 找到在memory bank中的最近邻距离
        # (B*H*W, D) @ (D, M) -> (B*H*W, M)
        dists = torch.cdist(patches, self.memory_tensor)
        min_dists, _ = dists.min(dim=1)  # (B*H*W,)
        
        # 重构为图像
        score_map = min_dists.reshape(-1, H, W)
        # 上采样到原图大小
        score_map = F.interpolate(
            score_map.unsqueeze(1), size=x.shape[-2:], mode='bilinear'
        )
        return score_map

# 小样本学习的原型网络（用于缺陷分类）
class PrototypicalNetwork(nn.Module):
    """原型网络：从少量样本中学习"""
    def __init__(self, in_channels=3, embed_dim=512):
        super().__init__()
        # 特征提取器
        self.encoder = nn.Sequential(
            nn.Conv2d(in_channels, 64, 3, padding=1), nn.BatchNorm2d(64), nn.ReLU(),
            nn.Conv2d(64, 64, 3, padding=1), nn.BatchNorm2d(64), nn.ReLU(),
            nn.MaxPool2d(2),
            nn.Conv2d(64, 128, 3, padding=1), nn.BatchNorm2d(128), nn.ReLU(),
            nn.Conv2d(128, 128, 3, padding=1), nn.BatchNorm2d(128), nn.ReLU(),
            nn.MaxPool2d(2),
            nn.Conv2d(128, embed_dim, 3, padding=1),
            nn.AdaptiveAvgPool2d(1),
        )

    def forward(self, x):
        x = self.encoder(x).squeeze(-1).squeeze(-1)
        return F.normalize(x, dim=1)  # 单位向量

    def few_shot_classify(self, support_images, support_labels, query_images):
        """
        support: (N_way * K_shot, C, H, W) 支持集
        query: (N_query, C, H, W) 查询集
        """
        support_features = self.forward(support_images)
        query_features = self.forward(query_images)
        
        # 计算每个类别的原型
        n_way = support_labels.max() + 1
        prototypes = []
        for c in range(n_way):
            mask = (support_labels == c)
            prototype = support_features[mask].mean(0)
            prototypes.append(prototype)
        prototypes = torch.stack(prototypes)
        
        # 计算查询样本到每个原型的距离
        distances = torch.cdist(query_features, prototypes)
        logits = -distances  # 负距离作为logits
        return logits

# 测试
import torchvision
patchcore = PatchCore()
x_normal = torch.randn(4, 3, 224, 224)
patches, _ = patchcore.extract_patches(x_normal)
print(f"Patch特征尺寸: {patches.shape}")

# 模拟异常检测
score_map = patchcore.detect(x_normal)
print(f"异常得分图: {score_map.shape}")

# 原型网络测试
proto_net = PrototypicalNetwork()
support = torch.randn(10, 3, 64, 64)  # 2类 × 5样本
labels = torch.tensor([0]*5 + [1]*5)
query = torch.randn(3, 3, 64, 64)
logits = proto_net.few_shot_classify(support, labels, query)
print(f"小样本分类logits: {logits.shape}")
```

## 深度学习关联

- **异常检测的工业化**：PatchCore和PaDiM等基于特征memory bank的异常检测方法已成为工业视觉检测的主流技术，在MVTec AD数据集上达到了SOTA性能，且不需要缺陷样本训练。
- **生成模型补充缺陷样本**：近年来，生成对抗网络（GAN）和扩散模型被用于生成合成的缺陷样本，以缓解小样本问题。Defect-GAN、DRAEM等方法通过将缺陷"粘贴"到正常图像上生成多样化的训练数据。
- **小样本学习与持续学习的结合**：工业产线中新产品不断上线，对模型提出了"持续学习"的要求——在不遗忘旧产品缺陷知识的前提下，快速学习新产品仅有的少量缺陷样本。这是小样本学习和增量学习的交叉前沿。
