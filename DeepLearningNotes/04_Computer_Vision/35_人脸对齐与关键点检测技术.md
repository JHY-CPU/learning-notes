# 35_人脸对齐与关键点检测技术

## 核心概念

- **人脸关键点检测（Facial Landmark Detection）**：自动定位人脸图像中预定义的关键点（如眼睛中心、鼻尖、嘴角、眉毛、下巴轮廓等）。68点标记是最常用的标准。
- **人脸对齐（Face Alignment）**：基于检测到的关键点，通过几何变换（相似变换、仿射变换）将人脸归一化到标准姿态，消除旋转、缩放和平移变化。
- **级联回归（Cascade Regression）**：早期方法（如ESR、SDM、LBF）通过级联多个回归器，逐步细化关键点位置。每个回归器基于当前关键点周围的局部特征预测更新量。
- **热图回归（Heatmap Regression）**：深度学习方法（如Hourglass Network、HRNet）将关键点检测转化为热图预测问题——为每个关键点预测一张高斯热图，热图的峰值位置即为关键点坐标。
- **Hourglass Network（沙漏网络）**：对称的编码器-解码器结构，通过重复的"下采样-上采样"过程捕获多尺度上下文信息，适用于需要精确位置信息的任务。
- **HRNet（High-Resolution Network）**：在整个网络中保持高分辨率特征图，通过多尺度并行分支和反复的多尺度融合，同时保持高分辨率空间信息和丰富语义信息。

## 数学推导

**人脸对齐中的相似变换：**
给定检测到的关键点坐标，通过相似变换将人脸映射到标准位置：
$$
\begin{bmatrix} x' \\ y' \end{bmatrix} = s \begin{bmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix} + \begin{bmatrix} t_x \\ t_y \end{bmatrix}
$$

参数 $(s, \theta, t_x, t_y)$ 可通过最小二乘法从检测关键点与模板关键点的对应中求解。

**热图回归的损失函数：**
对于关键点 $k$，真实热图为 $H_k \in \mathbb{R}^{H \times W}$，其中像素 $(i,j)$ 的值由高斯分布给出：
$$
H_k(i,j) = \exp\left(-\frac{(i - i_k^*)^2 + (j - j_k^*)}{2\sigma^2}\right)
$$

预测热图 $\hat{H}_k$ 与真实热图之间的MSE损失：
$$
\mathcal{L}_{heatmap} = \frac{1}{K} \sum_{k=1}^K \sum_{i=1}^H \sum_{j=1}^W (H_k(i,j) - \hat{H}_k(i,j))^2
$$

**归一化均方误差（NME）评估指标：**
$$
\text{NME} = \frac{1}{K \cdot d} \sum_{k=1}^K \|p_k - p_k^*\|_2
$$

其中 $d$ 是归一化因子（通常用眼间距或瞳孔间距）。

## 直观理解

人脸关键点检测可以理解为"在人脸上画地图"。68个关键点就像是地图上的"地标"——眼睛、鼻子、嘴角是主要地标，下巴轮廓是地形线。有了这些地标，就可以对人脸进行"对齐"：把每个人的脸旋转、缩放到一个标准位置，就像在手机上把人脸放到相框的固定位置一样。

热图回归的方法，相当于在图像上放置"热量探测器"——每个关键点周围会形成一个"热点"（高斯分布的中心），网络的任务就是预测这些热点的分布。热图方法比直接回归坐标更鲁棒，因为它提供了空间上的"软"监督：即使预测位置不完全准确，只要接近真实位置，就会部分正确。

## 代码示例

```python
import torch
import torch.nn as nn

class HourglassBlock(nn.Module):
    """简化的沙漏模块"""
    def __init__(self, in_channels, out_channels):
        super().__init__()
        # 下采样路径
        self.down = nn.Sequential(
            nn.Conv2d(in_channels, out_channels, 3, stride=2, padding=1),
            nn.BatchNorm2d(out_channels),
            nn.ReLU(inplace=True),
            nn.Conv2d(out_channels, out_channels, 3, padding=1),
            nn.BatchNorm2d(out_channels),
            nn.ReLU(inplace=True),
        )
        # 上采样路径
        self.up = nn.Sequential(
            nn.ConvTranspose2d(out_channels, out_channels, 4, stride=2, padding=1),
            nn.BatchNorm2d(out_channels),
            nn.ReLU(inplace=True),
        )
        # 跳跃连接
        self.skip = nn.Sequential(
            nn.Conv2d(in_channels, out_channels, 1),
            nn.BatchNorm2d(out_channels),
        )

    def forward(self, x):
        skip = self.skip(x)
        x = self.down(x)
        x = self.up(x)
        # 确保尺寸匹配
        if x.shape[-2:] != skip.shape[-2:]:
            x = nn.functional.interpolate(x, size=skip.shape[-2:], mode='bilinear', align_corners=True)
        return x + skip

class FaceLandmarkDetector(nn.Module):
    """人脸关键点检测网络 (68点)"""
    def __init__(self, num_landmarks=68):
        super().__init__()
        self.pre = nn.Sequential(
            nn.Conv2d(3, 64, 7, stride=2, padding=3),
            nn.BatchNorm2d(64),
            nn.ReLU(inplace=True),
        )
        self.hg1 = HourglassBlock(64, 128)
        self.hg2 = HourglassBlock(128, 256)
        self.hg3 = HourglassBlock(256, 256)
        self.head = nn.Sequential(
            nn.Conv2d(256, 256, 3, padding=1),
            nn.ReLU(inplace=True),
            nn.Conv2d(256, num_landmarks, 1),  # 输出68个热图
        )

    def forward(self, x):
        x = self.pre(x)
        x = self.hg1(x)
        x = self.hg2(x)
        x = self.hg3(x)
        heatmaps = self.head(x)
        return heatmaps  # (B, 68, H, W)

# 从热图提取坐标
def heatmap_to_coords(heatmaps):
    """从热图中提取关键点坐标"""
    B, K, H, W = heatmaps.shape
    heatmaps_flat = heatmaps.view(B, K, -1)
    max_idx = heatmaps_flat.argmax(dim=2)
    y = max_idx // W
    x = max_idx % W
    return torch.stack([x, y], dim=2).float()  # (B, K, 2)

model = FaceLandmarkDetector(num_landmarks=68)
x = torch.randn(1, 3, 256, 256)
heatmaps = model(x)
coords = heatmap_to_coords(heatmaps)
print(f"热图输出: {heatmaps.shape}")
print(f"关键点坐标: {coords.shape}")
print(f"参数量: {sum(p.numel() for p in model.parameters()):,}")
```

## 深度学习关联

- **人脸分析的基础组件**：人脸关键点检测是所有人脸分析任务（人脸识别、人脸验证、表情分析、3D人脸重建、换脸）的前置步骤。高质量的关键点检测直接决定了后续任务的性能上限。
- **级联回归到端到端学习的演变**：人脸关键点检测经历了从手工特征+级联回归（ESR、SDM、LBF）到深度热图回归（Hourglass、HRNet、MobileFaceNet）的演变，精度和速度都得到极大提升。
- **HRNet的广泛应用**：HRNet不仅在人脸关键点检测中取得SOTA，还被广泛用于人体姿态估计、语义分割、目标检测等任务，其"高分辨率特征保持"的设计思想影响了神经网络的骨干网络设计。
