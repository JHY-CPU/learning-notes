# 01 感知机模型与逻辑门实现

## 核心概念

- **感知机定义**：感知机（Perceptron）是神经网络的基本单元，由 Frank Rosenblatt 于 1957 年提出。它接收多个二进制输入，通过加权求和后经过阶跃函数输出一个二进制结果，是最简单的线性二分类模型。

- **线性可分性**：感知机只能解决线性可分问题，即存在一个超平面能够将正负样本完全分开。对于异或（XOR）这类线性不可分问题，单层感知机无能为力，这直接导致了神经网络的第一次寒冬。

- **权重更新规则**：感知机的学习规则基于错误驱动——只有当分类错误时才更新权重。更新公式为 $w_i \leftarrow w_i + \eta \cdot (y - \hat{y}) \cdot x_i$，其中 $\eta$ 是学习率，$(y - \hat{y})$ 是预测误差。

- **逻辑门实现**：与门（AND）、或门（OR）、与非门（NAND）都可以用单层感知机实现，因为它们是线性可分的。而异或门（XOR）必须通过多层感知机组合实现。

- **感知机收敛定理**：如果训练数据是线性可分的，感知机算法保证在有限步数内收敛到一个解。收敛步数与间隔（margin）的平方成反比。

## 数学推导

感知机的数学形式如下：

$$
f(x) = \text{sign}(w^T x + b) = \begin{cases}
1 & \text{if } w^T x + b > 0 \\
0 & \text{otherwise}
\end{cases}
$$

其中 $w \in \mathbb{R}^n$ 是权重向量，$b \in \mathbb{R}$ 是偏置项，$\text{sign}(\cdot)$ 是阶跃激活函数。

权重更新规则推导：定义损失函数为误分类点到超平面的距离：

$$
L(w, b) = -\sum_{x_i \in M} y_i (w^T x_i + b)
$$

其中 $M$ 是被误分类的样本集合。对 $w$ 求梯度：

$$
\frac{\partial L}{\partial w} = -\sum_{x_i \in M} y_i x_i
$$

采用随机梯度下降，每次随机选取一个误分类点进行更新：

$$
w \leftarrow w + \eta y_i x_i
$$

这就是感知机的标准学习规则。注意这里的学习规则是"错误驱动"的——只有当预测错误时才更新权重。

逻辑门实现的权重示例（输入为 $x_1, x_2$）：

- AND 门：$w_1 = 1, w_2 = 1, b = -1.5$，即 $f = \mathbb{I}(x_1 + x_2 > 1.5)$
- OR 门：$w_1 = 1, w_2 = 1, b = -0.5$，即 $f = \mathbb{I}(x_1 + x_2 > 0.5)$
- NAND 门：$w_1 = -1, w_2 = -1, b = 1.5$，即 $f = \mathbb{I}(-x_1 - x_2 + 1.5 > 0)$

## 直观理解

感知机可以想象为一个"投票系统"：每个输入特征 $x_i$ 投出权重为 $w_i$ 的票，偏置 $b$ 相当于内置的倾向性。当总票数超过阈值时，输出"通过"（1），否则"不通过"（0）。

从几何角度看，感知机在 $n$ 维空间中画出了一个决策超平面 $w^T x + b = 0$。超平面的一侧是正类，另一侧是负类。权重的方向 $w/||w||$ 决定了超平面的法线方向，而 $b/||w||$ 决定了超平面到原点的距离。

AND 门可以理解为"两个条件必须同时满足"——只有当两个输入都为 1 时，加权和才超过阈值。OR 门则是"只要满足一个条件即可"。

## 代码示例

```python
import torch

class Perceptron:
    """感知机模型的 PyTorch 实现"""
    def __init__(self, n_features, lr=0.01):
        self.w = torch.randn(n_features, 1) * 0.01
        self.b = torch.zeros(1)
        self.lr = lr

    def forward(self, x):
        return torch.where(x @ self.w + self.b > 0, 1.0, 0.0)

    def train_step(self, x, y):
        # 逐样本训练，仅当预测错误时更新
        pred = self.forward(x)
        if pred.item() != y.item():
            error = y - pred
            self.w += self.lr * error * x.T
            self.b += self.lr * error
        return self.forward(x)

# 演示：AND 门
x = torch.tensor([[0., 0.], [0., 1.], [1., 0.], [1., 1.]])
y_and = torch.tensor([[0.], [0.], [0.], [1.]])
y_or  = torch.tensor([[0.], [1.], [1.], [1.]])
y_xor = torch.tensor([[0.], [1.], [1.], [0.]])

p = Perceptron(2, lr=0.1)
print("AND 门训练:")
for epoch in range(10):
    errors = 0
    for i in range(4):
        pred = p.train_step(x[i:i+1], y_and[i:i+1])
        if pred.item() != y_and[i].item():
            errors += 1
    if errors == 0:
        print(f"  epoch {epoch}: 收敛!")
        break

# 验证
with torch.no_grad():
    print(f"  AND(0,0)={p.forward(x[0:1]).item():.0f}, AND(0,1)={p.forward(x[1:2]).item():.0f}, AND(1,0)={p.forward(x[2:3]).item():.0f}, AND(1,1)={p.forward(x[3:4]).item():.0f}")
```

## 深度学习关联

- **神经网络的基本单元**：感知机是现代神经网络的基石。单层感知机相当于一个没有隐层的神经网络，而多层感知机（MLP）通过堆叠多个感知机层并引入非线性激活函数，克服了单层感知机无法处理线性不可分问题的根本缺陷。

- **与逻辑回归的关系**：感知机与逻辑回归非常相似，区别仅在于激活函数——感知机使用阶跃函数（不可导），而逻辑回归使用 Sigmoid 函数（可导）。这一可导性差异使得逻辑回归可以用梯度下降进行优化，而原始感知机只能使用错误驱动的离散更新规则。

- **支持向量机的前身**：感知机的收敛定理和最大间隔思想直接启发了支持向量机（SVM）。SVM 通过引入间隔最大化和核技巧，本质上是对感知机思想的扩展和深化，解决了感知机只能处理线性可分数据的问题。
