# 贝叶斯定理与后验概率更新

## 核心概念
- **贝叶斯定理 (Bayes' Theorem)**：描述了在已知观测数据 $D$ 后，参数 $\theta$ 的概率如何更新：
  $$P(\theta|D) = \frac{P(D|\theta) P(\theta)}{P(D)}$$
  它统一了先验知识（先验 $P(\theta)$）和观测证据（似然 $P(D|\theta)$）来得到后验 $P(\theta|D)$。
- **先验概率 (Prior)**：在看到数据前对参数 $\theta$ 的信念。可以是信息先验（基于领域知识）或无信息先验（均匀分布）。先验的选择对后验有影响，尤其在数据量少时。
- **似然函数 (Likelihood)**：给定参数 $\theta$ 时观测到数据 $D$ 的概率 $P(D|\theta)$。似然不是 $\theta$ 的概率分布，而是关于 $\theta$ 的函数。最大似然估计就是寻找使似然最大的 $\theta$。
- **后验概率 (Posterior)**：结合先验和观测数据后对 $\theta$ 的更新信念。后验正比于先验乘以似然：$P(\theta|D) \propto P(D|\theta) P(\theta)$。
- **证据 (Evidence)**：$P(D) = \int P(D|\theta) P(\theta) d\theta$，归一化常数，确保后验积分为 1。计算证据通常是高维贝叶斯推断的主要挑战。
- **贝叶斯更新**：随着更多数据到来，当前的后验可以作为下一轮的先验，实现贝叶斯推断的序贯更新。

## 数学推导
贝叶斯定理的完整形式：
$$
P(\theta|D) = \frac{P(D|\theta) P(\theta)}{P(D)} = \frac{P(D|\theta) P(\theta)}{\int P(D|\theta) P(\theta) d\theta}
$$

贝叶斯定理的推导（从条件概率定义出发）：
$$
P(\theta|D) = \frac{P(\theta, D)}{P(D)}, \quad P(D|\theta) = \frac{P(D, \theta)}{P(\theta)}
$$
因此 $P(\theta, D) = P(\theta|D)P(D) = P(D|\theta)P(\theta)$，整理得贝叶斯定理。

贝叶斯更新（序贯形式）：给定两个独立观测 $D_1, D_2$：
$$
P(\theta|D_1, D_2) \propto P(D_2|\theta) P(\theta|D_1)
$$
即 $D_1$ 后的后验成为 $D_2$ 前的先验。

贝叶斯预测分布（对新的数据点 $\tilde{x}$）：
$$
P(\tilde{x}|D) = \int P(\tilde{x}|\theta) P(\theta|D) d\theta
$$

## 直观理解
- **学习即信念更新**：贝叶斯定理量化了"从经验中学习"的过程。先验是你的初始信念，数据是新的经验，后验是更新后的信念。如果有足够的数据，后验最终会主导先验——数据让你"改变想法"。
- **侦探的推理**：想象一位侦探在破案。先验概率是"随机选一个人是凶手的可能性"。看到证据（指纹、动机）后，用似然评估证据在凶手和普通人中出现的概率比，更新后验概率。这就是贝叶斯推理。
- **先验 vs 数据的权衡**：先验就像"偏见"，数据是"客观事实"。有了 100 个数据点，先验的影响几乎消失（后验 ≈ 似然）；但只有 1-2 个数据点时，先验起重要作用。这在少样本学习中至关重要。

## 代码示例
```python
import numpy as np
import matplotlib.pyplot as plt

# 经典 Beta-Bernoulli 共轭模型：估计硬币正面概率
# 先验：Beta(a, b)，似然：Bernoulli，后验：Beta(a + heads, b + tails)

def beta_binomial_posterior(prior_a, prior_b, heads, tails):
    """计算后验 Beta 分布的参数"""
    return prior_a + heads, prior_b + tails

# 场景：先验认为硬币可能是公平的 (Beta(2,2))
prior_a, prior_b = 2, 2

# 做 10 次抛掷，得到 7 次正面，3 次反面
heads, tails = 7, 3
post_a, post_b = beta_binomial_posterior(prior_a, prior_b, heads, tails)

print(f"先验: Beta({prior_a}, {prior_b})")
print(f"观测: {heads} 正面, {tails} 反面")
print(f"后验: Beta({post_a}, {post_b})")
print(f"后验均值: {post_a/(post_a+post_b):.4f} (MLE={heads/(heads+tails):.4f})")

# 序贯更新：新观测
more_heads, more_tails = 20, 10
post_a2, post_b2 = beta_binomial_posterior(post_a, post_b, more_heads, more_tails)
print(f"\n再观测: {more_heads} 正面, {more_tails} 反面")
print(f"最终后验: Beta({post_a2}, {post_b2})")
print(f"最终后验均值: {post_a2/(post_a2+post_b2):.4f}")

# 贝叶斯预测分布
mean_pred = post_a / (post_a + post_b)
var_pred = post_a * post_b / ((post_a + post_b)**2 * (post_a + post_b + 1))
print(f"\n预测下一次抛掷为正面的概率: {mean_pred:.4f}")
print(f"预测方差: {var_pred:.4f}")

# 不同先验的影响
print("\n--- 不同先验的影响 (10次实验, 7正3反) ---")
for prior in [(1,1), (2,2), (5,2), (10,1)]:
    a, b = prior
    pa, pb = beta_binomial_posterior(a, b, 7, 3)
    post_mean = pa / (pa + pb)
    mle = 7/10
    print(f"先验 Beta{a}: 后验均值={post_mean:.4f} (MLE={mle:.4f})")
```

## 深度学习关联
- **贝叶斯神经网络 (BNN)**：BNN 对网络权重 $\mathbf{w}$ 赋予先验分布 $p(\mathbf{w})$，通过数据计算后验 $p(\mathbf{w}|D)$。BNN 不仅能给出预测值，还能给出预测的不确定性。由于精确后验不可计算，通常使用变分推断近似后验。
- **主动学习 (Active Learning)**：贝叶斯定理天然适用于主动学习——模型选择信息量最大的未标记样本进行标注。常用的采集函数包括：预测熵最大（最不确定）、期望模型变化最大、贝叶斯主动学习通过后验不确定性指导采样。
- **贝叶斯模型平均**：传统深度学习使用单一参数设置做预测。贝叶斯方法通过对后验分布积分 $p(y|x, D) = \int p(y|x, w) p(w|D) dw$，隐式地集成了无限多个模型的预测，往往比单个模型更鲁棒。SWAG、MC Dropout 等方法近似实现了这一思想。
- **元学习 (Meta-Learning)**：MAML 等元学习算法可以解释为分层贝叶斯模型——先验 $p(\theta)$ 在元训练阶段学习，在新任务上通过少量梯度更新（近似后验推断）快速适应。贝叶斯视角为理解"学会学习"提供了理论框架。
