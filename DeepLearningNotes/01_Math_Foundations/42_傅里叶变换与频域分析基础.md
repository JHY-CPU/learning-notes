# 傅里叶变换与频域分析基础

## 核心概念
- **傅里叶级数 (Fourier Series)**：任何周期函数可以分解为不同频率正弦和余弦波的加权和。$f(t) = a_0 + \sum_{n=1}^{\infty} (a_n \cos(n\omega t) + b_n \sin(n\omega t))$。
- **傅里叶变换 (Fourier Transform)**：将时域（或空域）信号 $f(t)$ 变换到频域 $F(\omega)$，揭示信号在不同频率上的能量分布：
  $$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt$$
  逆变换：$f(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} F(\omega) e^{i\omega t} d\omega$
- **频谱与相位谱**：$F(\omega) = |F(\omega)| e^{i\phi(\omega)}$，其中 $|F(\omega)|$ 是振幅谱（幅度），$\phi(\omega)$ 是相位谱。幅度谱显示各频率分量的强度，相位谱显示各分量的时间偏移。
- **离散傅里叶变换 (DFT)**：对有限长离散信号的傅里叶变换：
  $$X_k = \sum_{n=0}^{N-1} x_n e^{-i 2\pi k n / N}, \quad k = 0, 1, \dots, N-1$$
- **快速傅里叶变换 (FFT)**：$O(N\log N)$ 的 DFT 高效算法，广泛用于信号处理。Cooley-Tukey 算法通过递归分治实现。
- **奈奎斯特采样定理 (Nyquist-Shannon)**：为了无失真重建信号，采样频率必须大于信号最高频率的两倍。这是数字信号处理的基本前提。

## 数学推导
傅里叶变换（连续形式）：
$$
F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt = \int_{-\infty}^{\infty} f(t) [\cos(\omega t) - i\sin(\omega t)] dt
$$

傅里叶变换的卷积定理：
$$
\mathcal{F}\{f * g\}(t) = F(\omega) \cdot G(\omega)
$$
时域卷积等于频域相乘。这是卷积定理的核心。

帕塞瓦尔定理 (Parseval's Theorem)：
$$
\int_{-\infty}^{\infty} |f(t)|^2 dt = \frac{1}{2\pi} \int_{-\infty}^{\infty} |F(\omega)|^2 d\omega
$$
信号在时域的总能量等于频域的总能量。

离散傅里叶变换（DFT）：
$$
X_k = \sum_{n=0}^{N-1} x_n e^{-i 2\pi k n / N} = \sum_{n=0}^{N-1} x_n \left(\cos\frac{2\pi kn}{N} - i\sin\frac{2\pi kn}{N}\right)
$$

## 直观理解
- **"棱镜分解光"的类比**：傅里叶变换就像把白光（时域信号）通过棱镜分解成彩虹光谱（频域）。每种颜色的频率对应光谱的一个分量。低频成分（红色）对应信号中的缓慢变化，高频成分（蓝色）对应快速变化（如边缘）。
- **时域 vs 频域**：时域告诉我们信号在"什么时间"变化，频域告诉我们信号以"什么频率"变化。就像乐谱——时域是随时间波动的声音信号，频域是不同音符（频率）的强度和持续时间。
- **为什么频率分析有用**：许多在时域中复杂的问题在频域中变得简单。例如，噪声通常是高频的，因此只需在频域中滤除高频成分就可以去噪。

## 代码示例
```python
import numpy as np

# 1. 生成并分析合成信号
np.random.seed(42)
sr = 1000  # 采样率
T = 1.0    # 持续时间
t = np.linspace(0, T, int(sr*T), endpoint=False)

# 合成信号：50Hz + 120Hz + 噪声
f1, f2 = 50, 120
signal = np.sin(2*np.pi*f1*t) + 0.5*np.sin(2*np.pi*f2*t) + 0.3*np.random.randn(len(t))

# FFT
fft = np.fft.fft(signal)
freqs = np.fft.fftfreq(len(t), 1/sr)
magnitude = np.abs(fft)[:len(fft)//2]
freqs_plot = freqs[:len(freqs)//2]

# 找到主要频率
peak_indices = np.argsort(magnitude)[-5:]
print("主要频率成分:")
for idx in reversed(peak_indices):
    print(f"  {freqs_plot[idx]:.1f} Hz: 幅度={magnitude[idx]:.1f}")

# 2. 低通滤波
cutoff = 80  # 截止频率 (Hz)
# 在频域中滤除高频
fft_filtered = fft.copy()
fft_filtered[np.abs(freqs) > cutoff] = 0
filtered_signal = np.fft.ifft(fft_filtered).real

# 比较滤波前后的标准差
print(f"\n低通滤波 (截止={cutoff}Hz):")
print(f"  原始信号标准差: {np.std(signal):.4f}")
print(f"  滤波后标准差: {np.std(filtered_signal):.4f}")

# 3. 频谱能量分布
total_energy = np.sum(magnitude**2)
energy_low = np.sum(magnitude[freqs_plot <= 80]**2)
energy_high = total_energy - energy_low
print(f"\n频谱能量分布:")
print(f"  低频能量 (<80Hz): {energy_low/total_energy:.1%}")
print(f"  高频能量 (>80Hz): {energy_high/total_energy:.1%}")

# 4. 卷积定理验证
kernel = np.array([1, 2, 1]) / 4  # 平滑核
signal_padded = np.pad(signal, (len(kernel)//2,), mode='constant')

# 时域卷积
conv_time = np.convolve(signal, kernel, mode='same')

# 频域乘法
fft_signal = np.fft.fft(signal)
fft_kernel = np.fft.fft(np.pad(kernel, (0, len(signal)-len(kernel)), mode='constant'))
conv_freq = np.fft.ifft(fft_signal * fft_kernel).real

print(f"\n卷积定理验证:")
print(f"  时域卷积前10个值: {conv_time[:5]}")
print(f"  频域乘法前10个值: {conv_freq[:5]}")
print(f"  两组结果一致? {np.allclose(conv_time, conv_freq, atol=1e-10)}")

# 5. 图像上的二维 FFT（降噪演示）
# 创建简单图像
image = np.zeros((64, 64))
image[20:44, 20:44] = 1.0  # 白色方块
image_noisy = image + 0.3 * np.random.randn(64, 64)

# 2D FFT
fft2 = np.fft.fft2(image_noisy)
fft2_shifted = np.fft.fftshift(fft2)
# 低通
h, w = fft2.shape
mask = np.zeros((h, w))
center = (h//2, w//2)
r = 10
y, x = np.ogrid[:h, :w]
mask[(y-center[0])**2 + (x-center[1])**2 <= r**2] = 1

fft2_filtered = fft2_shifted * mask
image_denoised = np.fft.ifft2(np.fft.ifftshift(fft2_filtered)).real

print(f"\n图像去噪:")
print(f"  原始方块噪声: {np.std(image_noisy - image):.4f}")
print(f"  FFT去噪后误差: {np.std(image_denoised - image):.4f}")
```

## 深度学习关联
- **卷积神经网络的频域解释**：CNN 的卷积核可以看作频域滤波器。浅层卷积核学习高频特征（边缘、纹理），深层学习低频特征（形状、语义）。频域分析可以解释 CNN 的一些行为，如对抗攻击（高频扰动）和图像先验（低频偏好）。
- **傅里叶特征 (Fourier Features)**：NeRF (Neural Radiance Fields) 使用位置编码（将坐标映射到傅里叶特征 $\gamma(x) = [\sin(2\pi x), \cos(2\pi x), \sin(4\pi x), \dots]$）来增强网络对高频细节的学习能力。这本质上是在帮助 MLP 拟合高频函数。类似的编码在 Transformer 的位置编码中也有广泛使用。
- **FFT 加速卷积**：当卷积核很大时（如 $31 \times 31$），利用卷积定理在频域计算卷积的复杂度从 $O(n^2)$ 降至 $O(n \log n)$。在深度学习的早期（AlexNet 时代），FFT 曾用来加速大规模卷积计算。现代小卷积核（$3\times3$）策略更依赖 GPU 优化的 im2col 算法。
- **频谱偏置 (Spectral Bias)**：研究表明，神经网络（特别是 MLP）在学习过程中偏好低频函数——称为"频谱偏置"或"F-principle"。这意味着网络先学习数据的低频模式，再逐步学习高频细节。这一发现与傅里叶分析直接相关，启发了一些改进训练的方法，如渐进增长频率或使用傅里叶特征。
