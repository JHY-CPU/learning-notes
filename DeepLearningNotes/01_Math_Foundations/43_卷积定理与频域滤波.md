# 卷积定理与频域滤波

## 核心概念
- **卷积定理 (Convolution Theorem)**：两个信号在时域的卷积等价于它们在频域的乘积：
  $$f(t) * g(t) \iff F(\omega) \cdot G(\omega)$$
  这是信号处理和深度学习中最核心的定理之一。
- **互相关定理**：$\mathcal{F}\{f \star g\} = \overline{F(\omega)} \cdot G(\omega)$，其中 $\star$ 表示互相关，$\overline{F}$ 表示复共轭。
- **频域滤波 (Frequency Domain Filtering)**：在频域中对信号频谱进行处理（如保留或抑制某些频率成分），然后通过逆变换回时域。常见操作包括低通（平滑）、高通（锐化/边缘检测）、带通（保留特定频段）。
- **低通滤波**：保留低频成分，抑制高频成分。效果：平滑信号、去噪、去除细节。在图像中产生模糊效果。
- **高通滤波**：保留高频成分，抑制低频成分。效果：边缘检测、锐化、提取细节。在图像中突出纹理和边缘。
- **带通/带阻滤波**：保留/抑制特定频率范围的成分。在音频处理中用于音调调节，在图像中用于提取特定尺度的特征。
- **线性时不变系统 (LTI)**：卷积定理适用于 LTI 系统——系统的输出是输入与冲激响应的卷积。在频域中，输出谱 = 输入谱 $\times$ 频率响应。

## 数学推导
卷积的定义：
$$
(f * g)(t) = \int_{-\infty}^{\infty} f(\tau) g(t - \tau) d\tau
$$

卷积定理的证明（一维）：
$$
\mathcal{F}\{f * g\}(\omega) = \int_{-\infty}^{\infty} \left[ \int_{-\infty}^{\infty} f(\tau) g(t - \tau) d\tau \right] e^{-i\omega t} dt
$$
交换积分顺序，令 $u = t - \tau$：
$$
= \int_{-\infty}^{\infty} f(\tau) e^{-i\omega \tau} \left[ \int_{-\infty}^{\infty} g(u) e^{-i\omega u} du \right] d\tau
= F(\omega) \cdot G(\omega)
$$

二维卷积定理（图像处理）：
$$
(f * g)(x, y) \iff F(u, v) \cdot G(u, v)
$$

## 直观理解
- **"盖印章"的操作**：卷积相当于用一个"印章"（卷积核）在信号上滑动，每个位置取加权平均。频域乘法将这个过程简化为"元素对元素的相乘"——就像在频域中一次性地对所有位置施加相同的滤波器。
- **为什么频域滤波效率高**：时域卷积需要 $O(N^2)$ 操作（对每个输出位置计算加权和），而 FFT + 频域乘法 + IFFT 只需 $O(N \log N)$。当卷积核很大时，节省的计算量非常可观。
- **滤波器就像"频率筛子"**：低通滤波器是"粗筛"，让缓慢变化的信息通过，筛掉快速变化的噪声。高通滤波器是"细筛"，抓住变化快的边缘和细节，放过平坦区域。

## 代码示例
```python
import numpy as np

# 1. 一维卷积定理验证
np.random.seed(42)
n = 1000
signal = np.random.randn(n)
kernel = np.exp(-np.linspace(-3, 3, 50)**2)  # 高斯核
kernel = kernel / kernel.sum()

# 时域卷积
conv_time = np.convolve(signal, kernel, mode='same')

# 频域乘法
fft_signal = np.fft.fft(signal)
fft_kernel = np.fft.fft(kernel, n)  # 补零到相同长度
conv_freq = np.fft.ifft(fft_signal * fft_kernel).real

print("一维卷积定理验证:")
print(f"  时域卷积与频域乘法结果一致? {np.allclose(conv_time, conv_freq, atol=1e-8)}")
print(f"  最大绝对误差: {np.max(np.abs(conv_time - conv_freq)):.2e}")

# 2. 图像的频域滤波
image = np.zeros((64, 64))
image[15:50, 15:50] = 1.0
# 添加噪声
image_noisy = image + 0.3 * np.random.randn(64, 64)

# 2D FFT
F = np.fft.fft2(image_noisy)
F_shifted = np.fft.fftshift(F)

# 创建各种滤波器
h, w = image.shape
center = (h//2, w//2)
Y, X = np.ogrid[:h, :w]
r = np.sqrt((X - center[1])**2 + (Y - center[0])**2)

# 低通滤波器 (高斯)
sigma_low = 8
lowpass = np.exp(-r**2 / (2*sigma_low**2))

# 高通滤波器 (1 - 低通)
sigma_high = 15
highpass = 1 - np.exp(-r**2 / (2*sigma_high**2))

# 带通滤波器 (保留中间频率)
r1, r2 = 5, 15
bandpass = ((r >= r1) & (r <= r2)).astype(float)

# 应用滤波器
def apply_filter(F_shifted, filt):
    F_filtered = F_shifted * filt
    return np.fft.ifft2(np.fft.ifftshift(F_filtered)).real

img_low = apply_filter(F_shifted, lowpass)
img_high = apply_filter(F_shifted, highpass)
img_band = apply_filter(F_shifted, bandpass)

print("\n频域滤波效果 (图像):")
print(f"  低通滤波 - 保留能量: {np.sum(lowpass) / (h*w):.1%}")
print(f"  高通滤波 - 保留能量: {np.sum(highpass) / (h*w):.1%}")

# 3. 频域去噪效果测量
original_energy = np.sum(image**2)
denoised_energy = np.sum(img_low**2)
print(f"\n去噪效果:")
print(f"  噪声图像误差: {np.sqrt(np.mean((image_noisy - image)**2)):.4f}")
print(f"  低通去噪误差: {np.sqrt(np.mean((img_low - image)**2)):.4f}")

# 4. 边缘检测 (高通滤波)
edges = np.abs(img_high)
threshold = np.percentile(edges, 95)
edge_mask = edges > threshold
print(f"\n边缘检测:")
print(f"  边缘像素比例: {np.mean(edge_mask):.1%}")

# 5. 计算复杂度对比
import time

n_repeats = 10
signal_len = 10000
kernel_len = 1001
sig = np.random.randn(signal_len)
kern = np.random.randn(kernel_len)
kern = kern / kern.sum()

# 时域卷积
t0 = time.time()
for _ in range(n_repeats):
    np.convolve(sig, kern, mode='same')
t_conv = (time.time() - t0) / n_repeats

# 频域乘法
t0 = time.time()
for _ in range(n_repeats):
    fft_s = np.fft.fft(sig)
    fft_k = np.fft.fft(kern, signal_len)
    np.fft.ifft(fft_s * fft_k)
t_fft = (time.time() - t0) / n_repeats

print(f"\n计算复杂度对比 (n={signal_len}, kernel={kernel_len}):")
print(f"  时域卷积: {t_conv*1000:.3f} ms")
print(f"  频域乘法: {t_fft*1000:.3f} ms")
print(f"  加速比: {t_conv/t_fft:.1f}x")
```

## 深度学习关联
- **CNN 与频域滤波的等价性**：CNN 的卷积层本质上是学习数据的频域滤波器。训练好的卷积核在频域中表现为特定的"频率响应模式"。研究表明，ImageNet 上训练的 CNN 第一层通常学到 Gabor 滤波器（方向性带通滤波器），这跟频域滤波中的带通滤波器非常相似。
- **频域数据增强**：在频域中进行数据增强比时域更有效。例如，随机在频域中擦除某些频率带（FreqAug、CutSpecAug）可以增强模型对不同频率特征的鲁棒性。特别是在音频分类（SpecAugment）中，频域掩码已成为标准做法。
- **图像超分辨率与频域学习**：超分辨率任务本质上是恢复图像的高频细节。SRGAN 和 ESRGAN 等网络通过感知损失增强高频质量。傅里叶域损失（频域 MSE）可以直接约束模型在各频率上的重建质量，相比时域 MSE 对高频更敏感。
- **对抗攻击的频域视角**：对抗扰动通常是高频的（人眼不易察觉但模型敏感）。频域滤波可以防御某些对抗攻击——低通滤波可以去除高频对抗噪声。反过来，频域攻击方法（如 FGSM 的频域变体）专门攻击特定频率段来逃避防御。
