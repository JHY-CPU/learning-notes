# 常见分布：高斯、伯努利、多项式分布

## 核心概念
- **伯努利分布 (Bernoulli Distribution)**：单次试验成功/失败的概率分布。$X \in \{0,1\}$，$P(X=1) = p$，$P(X=0) = 1-p$。均值为 $p$，方差为 $p(1-p)$。
- **二项分布 (Binomial Distribution)**：$n$ 次独立伯努利试验的成功次数。$X \sim \text{Binomial}(n, p)$，PMF 为 $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$。
- **高斯分布 (Gaussian/Normal Distribution)**：自然界中最常见的连续分布。$X \sim \mathcal{N}(\mu, \sigma^2)$，PDF 为对称的钟形曲线。均值为 $\mu$，方差为 $\sigma^2$。中心极限定理保证了大量独立随机变量的和趋近高斯分布。
- **多元高斯分布**：$d$ 维随机向量 $\mathbf{X} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)$，PDF 涉及协方差矩阵 $\Sigma$ 的逆和行列式。等密度面是椭球。
- **多项式分布 (Multinomial Distribution)**：二项分布在多类别上的推广。每次试验有 $K$ 个可能结果，概率为 $p_1, \dots, p_K$，$n$ 次试验后各类别出现次数的联合分布。
- **类别分布 (Categorical Distribution)**：多项式分布在 $n=1$（单次试验）时的特例，是离散输出的"软最大化"目标。

## 数学推导
高斯分布的概率密度函数（一元）：
$$
p(x|\mu, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$

多元高斯分布：
$$
p(\mathbf{x}|\boldsymbol{\mu}, \Sigma) = \frac{1}{(2\pi)^{d/2}|\Sigma|^{1/2}} \exp\left(-\frac{1}{2}(\mathbf{x}-\boldsymbol{\mu})^T \Sigma^{-1}(\mathbf{x}-\boldsymbol{\mu})\right)
$$

伯努利分布：
$$
P(X=x) = p^x (1-p)^{1-x}, \quad x \in \{0,1\}
$$

多项式分布（$n$ 次试验，$K$ 个类别）：
$$
P(\mathbf{x}|\mathbf{p}, n) = \frac{n!}{x_1! \dots x_K!} p_1^{x_1} \dots p_K^{x_K}, \quad \sum_{i=1}^K x_i = n
$$

## 直观理解
- **高斯分布的"68-95-99.7"法则**：在正态分布中，约 68% 的数据落在 $\mu \pm \sigma$ 内，95% 在 $\mu \pm 2\sigma$ 内，99.7% 在 $\mu \pm 3\sigma$ 内。这为识别异常值提供了实用准则。
- **伯努利 = 抛硬币**：伯努利分布就是单次抛硬币（可能不均匀）。二项分布 = 多次抛硬币看总共有几次正面。多项式分布 = 多次抛骰子看每个面出现几次。
- **为什么高斯无处不在**：中心极限定理说，大量独立随机变量的总和（或均值）趋近高斯分布，无论原始分布是什么。这就是为什么测量误差、人群身高、智商分数等都近似高斯分布。在深度学习中，参数的初始化权重也常使用高斯分布。

## 代码示例
```python
import numpy as np
from scipy import stats
import math

# 1. 伯努利分布
p = 0.3
bernoulli = stats.bernoulli(p)
samples = bernoulli.rvs(size=10000)
print(f"伯努利(p={p}): 均值≈{np.mean(samples):.3f} (理论={p})")
print(f"              方差≈{np.var(samples):.3f} (理论={p*(1-p):.3f})")

# 2. 二项分布
n, p = 20, 0.3
binom = stats.binom(n, p)
samples = binom.rvs(size=10000)
print(f"\n二项(n={n}, p={p}): 均值≈{np.mean(samples):.2f} (理论={n*p})")
print(f"                  方差≈{np.var(samples):.2f} (理论={n*p*(1-p):.2f})")
print(f"                  P(X=6)≈{np.mean(samples==6):.3f} (理论={binom.pmf(6):.3f})")

# 3. 高斯分布
mu, sigma = 1.0, 2.0
norm = stats.norm(mu, sigma)
samples = norm.rvs(size=100000)
print(f"\n高斯(μ={mu}, σ={sigma}): 均值≈{np.mean(samples):.3f}")
print(f"                       标准差≈{np.std(samples):.3f}")
print(f"                       68%区间≈({np.percentile(samples, 16):.2f}, {np.percentile(samples, 84):.2f})")

# CDF 验证 68-95-99.7
within_1 = np.sum(np.abs(samples - mu) <= sigma) / len(samples)
within_2 = np.sum(np.abs(samples - mu) <= 2*sigma) / len(samples)
within_3 = np.sum(np.abs(samples - mu) <= 3*sigma) / len(samples)
print(f"\n68-95-99.7 法则验证:")
print(f"  1σ内: {within_1:.3f} (理论≈0.683)")
print(f"  2σ内: {within_2:.3f} (理论≈0.954)")
print(f"  3σ内: {within_3:.3f} (理论≈0.997)")

# 4. 多元高斯分布
mean = np.array([0, 0])
cov = np.array([[1, 0.8], [0.8, 2]])
multi_normal = stats.multivariate_normal(mean, cov)
samples = multi_normal.rvs(size=5000)
print(f"\n多元高斯: 样本均值={np.mean(samples, axis=0)}")
print(f"         样本协方差=\n{np.cov(samples, rowvar=False)}")

# 5. 多项式分布
n, p_vals = 10, [0.2, 0.3, 0.5]
multinomial_samples = np.random.multinomial(n, p_vals, size=10000)
print(f"\n多项式(n={n}, p={p_vals}):")
print(f"  样本均值: {np.mean(multinomial_samples, axis=0)}")
print(f"  理论均值: {[n*p for p in p_vals]}")
```

## 深度学习关联
- **高斯分布与权重初始化**：神经网络权重通常从高斯分布 $\mathcal{N}(0, \sigma^2)$ 或均匀分布中采样。Xavier 初始化设置 $\sigma = \sqrt{2/(n_{\text{in}} + n_{\text{out}})}$，Kaiming 初始化设置 $\sigma = \sqrt{2/n_{\text{in}}}$。正确的初始化分布选择对训练深层网络至关重要。
- **伯努利分布与 Dropout**：Dropout 训练时每个神经元以概率 $p$ 保留（伯努利抽样），从而防止过拟合。预测时使用全部神经元但乘以 $p$（期望等效）。现代变体如 Concrete Dropout 实现了可学习的丢弃概率。
- **多项式分布与分类任务**：多分类问题的输出层使用 Softmax 函数将 logits 转换为概率向量，这等价于多项式分布（单次试验，即类别分布）。交叉熵损失函数正是最大化观测类别的多项式对数似然。
- **高斯分布与变分自编码器 (VAE)**：VAE 假设隐变量 $z \sim \mathcal{N}(0, I)$，编码器输出后验高斯参数 $(\mu(x), \sigma(x)^2)$。重参数化技巧 $z = \mu + \sigma \cdot \epsilon$（其中 $\epsilon \sim \mathcal{N}(0, I)$）使得采样过程可微。
- **高斯过程 (Gaussian Process)**：高斯过程是深度学习中贝叶斯优化的核心工具，用于超参数调优。它将函数视为无限维高斯分布，通过核函数控制函数的平滑性。
