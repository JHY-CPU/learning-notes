# 置信区间与贝叶斯可信区间

## 核心概念
- **置信区间 (Confidence Interval, CI)**：频率学派概念。参数 $\theta$ 的 95% 置信区间是一个随机区间 $[L(X), U(X)]$，使得 $P(L(X) \leq \theta \leq U(X)) = 0.95$。注意随机的是区间本身，不是 $\theta$。如果重复实验 100 次，大约 95 个区间会包含真实 $\theta$。
- **贝叶斯可信区间 (Credible Interval)**：贝叶斯学派概念。参数 $\theta$ 的 95% 可信区间满足 $P(\theta \in [a,b]|X) = 0.95$。这里 $\theta$ 是随机的，区间是固定的（给定数据后）。可以直接说"$\theta$ 有 95% 的概率落在这个区间内"。
- **两种区间的本质区别**：
  - 置信区间：频率概率，区间随机，参数固定
  - 可信区间：主观概率，参数随机，区间固定
- **置信区间的构造**：通常通过枢轴量（pivotal quantity）构造。对正态分布均值，$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim \mathcal{N}(0,1)$，以此构建区间。
- **Bootstrap 置信区间**：通过重采样数据，计算各 Bootstrap 样本的统计量，取 2.5% 和 97.5% 分位数作为 95% 置信区间。不需要分布假设，适用范围广。
- **区间宽度的影响因素**：样本量 $n$ 增大使区间变窄；置信水平增高使区间变宽；数据方差增大使区间变宽。

## 数学推导
正态分布均值的置信区间（已知 $\sigma$）：
$$
\bar{x} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}}
$$

其中 $z_{\alpha/2}$ 是标准正态分布的 $\alpha/2$ 上分位数。

正态分布均值的置信区间（未知 $\sigma$，使用 t 分布）：
$$
\bar{x} \pm t_{n-1, \alpha/2} \frac{s}{\sqrt{n}}
$$

两个独立正态总体均值差的置信区间：
$$
(\bar{x}_1 - \bar{x}_2) \pm t_{\nu, \alpha/2} \sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}
$$

贝叶斯可信区间（已知方差的正态均值）：
后验分布 $\mu|x \sim \mathcal{N}(\mu_n, \sigma_n^2)$，95% 可信区间为：
$$
\mu_n \pm 1.96 \sigma_n
$$

## 直观理解
- **置信区间的类比——捕鱼网**：置信区间就像一张随机撒下的渔网。参数 $\theta$ 是固定的鱼群位置。95% 置信区间意味着"如果撒 100 次网，大约 95 次能捕到鱼"。对于某一次特定实验，不能说有 95% 的概率捕到鱼（网要么捕到要么没捕到）。
- **可信区间的类比——地图上的范围**：贝叶斯可信区间就像在地图上标出的搜索区域。给定一个固定区域，可以说"鱼有 95% 的概率在这个区域内"。这符合直觉——给定数据，我们对参数的位置有了概率化的认识。
- **理解"区间宽度"**：区间宽度是"认知不确定性的度量"。大量数据 → 窄区间 → 知识更确定。小样本 → 宽区间 → 知识不确定。区间宽度比点估计提供了更丰富的信息。

## 代码示例
```python
import numpy as np
from scipy import stats

# 1. 正态分布均值的置信区间
np.random.seed(42)
n = 50
data = np.random.randn(n) * 2 + 5  # N(5, 4)

mean = np.mean(data)
se = stats.sem(data)  # 标准误 = s/sqrt(n)
ci = stats.t.interval(0.95, df=n-1, loc=mean, scale=se)

print(f"样本均值 = {mean:.4f}")
print(f"95% 置信区间 = [{ci[0]:.4f}, {ci[1]:.4f}]")
print(f"区间宽度 = {ci[1] - ci[0]:.4f}")

# 2. 置信区间的频率解释：重复实验
print("\n重复实验中的置信区间覆盖率 (1000次):")
true_mu = 5.0
cover_count = 0
for _ in range(1000):
    sample = np.random.randn(20) * 2 + true_mu
    m = np.mean(sample)
    se_s = stats.sem(sample)
    ci = stats.t.interval(0.95, df=19, loc=m, scale=se_s)
    if ci[0] <= true_mu <= ci[1]:
        cover_count += 1

print(f"  真实参数包含在 CI 中的比例: {cover_count/1000:.3f} (应≈0.95)")

# 3. Bootstrap 置信区间
def bootstrap_ci(data, stat_func=np.mean, n_bootstrap=10000, ci=0.95):
    """Bootstrap 置信区间（百分位法）"""
    n = len(data)
    bootstrap_stats = []
    for _ in range(n_bootstrap):
        resample = np.random.choice(data, n, replace=True)
        bootstrap_stats.append(stat_func(resample))
    
    alpha = (1 - ci) / 2
    lower = np.percentile(bootstrap_stats, alpha * 100)
    upper = np.percentile(bootstrap_stats, (1 - alpha) * 100)
    return lower, upper

data_skewed = np.random.exponential(scale=2, size=50)  # 偏态分布
ci_boot = bootstrap_ci(data_skewed, np.mean)
ci_t = stats.t.interval(0.95, df=49, loc=np.mean(data_skewed), scale=stats.sem(data_skewed))

print(f"\nBootstrap vs t-分布置信区间 (指数分布数据):")
print(f"  Bootstrap CI: [{ci_boot[0]:.4f}, {ci_boot[1]:.4f}]")
print(f"  t-分布 CI:    [{ci_t[0]:.4f}, {ci_t[1]:.4f}]")

# 4. 贝叶斯可信区间
# 使用 Beta-Bernoulli 共轭
alpha_prior, beta_prior = 2, 2
n_success, n_fail = 8, 2  # 10 次试验中 8 次成功

# 后验 Beta(10, 4)
alpha_post = alpha_prior + n_success
beta_post = beta_prior + n_fail

# 计算 95% 可信区间
credible_interval = stats.beta.interval(0.95, alpha_post, beta_post)
print(f"\n贝叶斯可信区间 (Beta后验):")
print(f"  后验 Beta({alpha_post}, {beta_post})")
print(f"  95% 可信区间: [{credible_interval[0]:.4f}, {credible_interval[1]:.4f}]")
print(f"  后验均值: {alpha_post/(alpha_post+beta_post):.4f}")

# 5. 置信区间 vs 贝叶斯可信区间
print("\n置信区间 vs 可信区间对比:")
print("频率置信区间: '长期频率中95%的区间包含真实值'")
print("贝叶斯可信区间: '给定数据，参数有95%概率在此区间'")

# 6. 区间宽度与样本量
print("\n样本量对区间宽度的影响:")
for n in [10, 50, 200, 1000]:
    sample = np.random.randn(n) * 3 + 10
    ci = stats.t.interval(0.95, df=n-1, loc=np.mean(sample), scale=stats.sem(sample))
    width = ci[1] - ci[0]
    print(f"  n={n:4d}: 宽度={width:.4f}")
```

## 深度学习关联
- **不确定性量化 (Uncertainty Quantification)**：在风险敏感场景（自动驾驶、医疗诊断）中，仅给出点预测是不够的，还需要量化预测的不确定性。MC Dropout、深度集成 (Deep Ensembles) 等方法可以提供预测的置信区间。例如，在贝叶斯神经网络中，预测可信区间可以通过后验采样得到。
- **置信区间与模型比较**：在比较两个模型的测试精度时，报告精度差值的置信区间比仅报告 p-value 更有信息量。例如，"模型 A 准确率比模型 B 高 2.1% (95% CI: [0.8%, 3.4%])" 不仅说明差异显著，还量化了效应量的范围。
- **校准 (Calibration)**：概率分类器输出的置信度应该与正确率一致。如果模型预测 90% 置信度的样本实际正确率只有 80%，则模型校准不良。可靠性图 (Reliability Diagram) 和期望校准误差 (ECE) 量化了这一偏差，本质上是在评估置信区间的校准质量。
- **深度概率编程中的可信区间**：Pyro 和 TensorFlow Probability 等概率编程框架自动提供参数的后验可信区间。理解贝叶斯可信区间有助于正确解读这些框架的输出，避免将频率置信区间的解释错误地套用到贝叶斯可信区间上。
