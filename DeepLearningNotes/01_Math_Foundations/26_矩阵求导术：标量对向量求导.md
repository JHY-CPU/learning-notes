# 矩阵求导术：标量对向量求导

## 核心概念
- **标量对向量求导 (Gradient)**：函数 $f: \mathbb{R}^n \to \mathbb{R}$，其导数是一个行向量或列向量（梯度），$\frac{\partial f}{\partial \mathbf{x}} \in \mathbb{R}^n$。梯度指向 $f$ 增长最快的方向。
- **布局约定**：分母布局（结果与分母同形）和分子布局（结果与分子同形）。本文采用分母布局（梯度为列向量）。
- **分子布局与分母布局**：设 $y = f(\mathbf{x})$，$y$ 是标量，$\mathbf{x} \in \mathbb{R}^n$：
  - 分母布局：$\frac{\partial y}{\partial \mathbf{x}} \in \mathbb{R}^{n \times 1}$（列向量）
  - 分子布局：$\frac{\partial y}{\partial \mathbf{x}} \in \mathbb{R}^{1 \times n}$（行向量）
- **常见公式**：
  - $\frac{\partial}{\partial \mathbf{x}} (\mathbf{a}^T \mathbf{x}) = \mathbf{a}$
  - $\frac{\partial}{\partial \mathbf{x}} (\mathbf{x}^T A \mathbf{x}) = (A + A^T) \mathbf{x}$
  - $\frac{\partial}{\partial \mathbf{x}} \|\mathbf{x}\|_2^2 = 2\mathbf{x}$
- **链式法则**：如果 $y = f(g(\mathbf{x}))$，则 $\frac{\partial y}{\partial \mathbf{x}} = \frac{dy}{dg} \cdot \frac{\partial g}{\partial \mathbf{x}}$。

## 数学推导
基本形式的推导：

$\mathbf{a}^T \mathbf{x} = \sum_i a_i x_i$，所以 $\frac{\partial}{\partial x_i} (\mathbf{a}^T \mathbf{x}) = a_i$，因此：
$$
\frac{\partial}{\partial \mathbf{x}} (\mathbf{a}^T \mathbf{x}) = \mathbf{a}
$$

二次型 $\mathbf{x}^T A \mathbf{x} = \sum_i \sum_j x_i A_{ij} x_j$，对 $x_k$ 求导：
$$
\frac{\partial}{\partial x_k} (\mathbf{x}^T A \mathbf{x}) = \sum_j A_{kj} x_j + \sum_i A_{ik} x_i = (A\mathbf{x})_k + (A^T \mathbf{x})_k
$$
因此：
$$
\frac{\partial}{\partial \mathbf{x}} (\mathbf{x}^T A \mathbf{x}) = (A + A^T) \mathbf{x}
$$

当 $A$ 是对称矩阵时，$A = A^T$，则：
$$
\frac{\partial}{\partial \mathbf{x}} (\mathbf{x}^T A \mathbf{x}) = 2A\mathbf{x}
$$

最小二乘的梯度：
$$
L(\mathbf{w}) = \frac{1}{2}\|X\mathbf{w} - \mathbf{y}\|_2^2
$$
$$
\frac{\partial L}{\partial \mathbf{w}} = X^T (X\mathbf{w} - \mathbf{y})
$$

## 直观理解
- **梯度即斜率**：标量对向量求导是单变量微积分中"斜率"概念的自然推广。每个分量 $\partial f / \partial x_i$ 就是 $f$ 沿着 $x_i$ 方向的"斜率"。
- **布局约定的记忆**：分母布局（分子写成行，分母写成列）是矩阵求导中更常见的约定。在反向传播中使用分母布局可以省去转置操作。
- **二次型的梯度"$2A\mathbf{x}$"**：对于对称 $A$，$f = \mathbf{x}^T A \mathbf{x}$ 的梯度是 $2A\mathbf{x}$，这与标量情况 $f = ax^2$ 的导数 $2ax$ 完美对应。

## 代码示例
```python
import numpy as np

# 1. 验证二次型梯度
def quadratic_gradient(A, x):
    """f(x) = x^T A x 的解析梯度"""
    return (A + A.T) @ x

def numerical_gradient(f, x, h=1e-6):
    """数值梯度（中心差分）"""
    grad = np.zeros_like(x)
    for i in range(len(x)):
        x_plus = x.copy()
        x_minus = x.copy()
        x_plus[i] += h
        x_minus[i] -= h
        grad[i] = (f(x_plus) - f(x_minus)) / (2 * h)
    return grad

# 测试
np.random.seed(42)
n = 5
A = np.random.randn(n, n)
x = np.random.randn(n)

f = lambda x: x @ A @ x
grad_analytical = quadratic_gradient(A, x)
grad_numerical = numerical_gradient(f, x)

print("二次型梯度验证:")
print(f"  解析梯度: {grad_analytical}")
print(f"  数值梯度: {grad_numerical}")
print(f"  误差: {np.linalg.norm(grad_analytical - grad_numerical):.2e}")

# 2. 最小二乘梯度
X = np.random.randn(100, 10)
true_w = np.random.randn(10)
y = X @ true_w + 0.1 * np.random.randn(100)

def least_squares_loss(w):
    return 0.5 * np.mean((X @ w - y)**2)

def least_squares_grad(w):
    return X.T @ (X @ w - y) / len(y)

w = np.random.randn(10)
g_analytic = least_squares_grad(w)
g_numeric = numerical_gradient(least_squares_loss, w)
print(f"\n最小二乘梯度误差: {np.linalg.norm(g_analytic - g_numeric):.2e}")

# 3. 梯度下降求解最小二乘
w_opt = np.zeros(10)
lr = 0.1
for t in range(200):
    w_opt = w_opt - lr * least_squares_grad(w_opt)

print(f"\n梯度下降解 vs 解析解:")
w_exact = np.linalg.lstsq(X, y, rcond=None)[0]
print(f"  GD解: {w_opt[:5]}")
print(f"  解析解: {w_exact[:5]}")
print(f"  误差: {np.linalg.norm(w_opt - w_exact):.2e}")

# 4. 常见公式验证
a = np.random.randn(5)
print(f"\nd(a^T x)/dx = a: {np.allclose(quadratic_gradient(np.zeros((5,5)), x), np.zeros(5))}")
# 验证 d(a^T x)/dx = a
f_linear = lambda x: a @ x
g_linear = numerical_gradient(f_linear, x)
print(f"d(a^T x)/dx = a? {np.allclose(g_linear, a)}")
```

## 深度学习关联
- **反向传播的梯度计算**：在深度学习中，损失函数 $L$ 对权重矩阵 $W$ 的梯度就是矩阵求导的核心应用。例如，$\frac{\partial L}{\partial W} = \frac{\partial L}{\partial \mathbf{y}} \mathbf{x}^T$（其中 $\mathbf{y} = W\mathbf{x} + \mathbf{b}$）。这个公式是每个深度学习框架中全连接层反向传播的基础。
- **梯度下降优化**：标量损失对大量参数的求导构成了梯度更新的基础。SGD、Adam 等优化器尽管实现复杂，但其核心都依赖于梯度 $\nabla_{\theta} L$。正确的高效梯度计算是深度学习的"引擎"。
- **正则化项的梯度**：L2 正则化 $\frac{\lambda}{2}\|\mathbf{w}\|^2$ 的梯度为 $\lambda \mathbf{w}$，可以直接加到权重梯度上。这个简洁表达式来自 $\frac{\partial}{\partial \mathbf{w}} \|\mathbf{w}\|^2 = 2\mathbf{w}$，是矩阵求导公式的直接应用。
