# 期望、方差与协方差矩阵

## 核心概念
- **期望 (Expectation)**：$\mathbb{E}[X]$ 是随机变量 $X$ 的概率加权平均值，衡量分布的"中心"位置。离散：$\mathbb{E}[X] = \sum_i x_i p_i$，连续：$\mathbb{E}[X] = \int x f_X(x) dx$。
- **方差 (Variance)**：$\text{Var}(X) = \mathbb{E}[(X-\mu)^2]$，衡量随机变量围绕均值的离散程度。标准差 $\sigma = \sqrt{\text{Var}(X)}$ 与 $X$ 量纲相同。
- **协方差 (Covariance)**：$\text{Cov}(X,Y) = \mathbb{E}[(X-\mu_X)(Y-\mu_Y)]$，衡量两个随机变量共同变化的程度和方向。正值表示同向变化，负值表示反向变化。
- **协方差矩阵 (Covariance Matrix)**：对 $d$ 维随机向量 $\mathbf{X}$，协方差矩阵 $\Sigma \in \mathbb{R}^{d \times d}$ 的第 $(i,j)$ 元素为 $\Sigma_{ij} = \text{Cov}(X_i, X_j)$。对角线是各分量的方差，对称且半正定。
- **相关系数 (Correlation Coefficient)**：$\rho_{XY} = \text{Cov}(X,Y) / (\sigma_X \sigma_Y)$，取值在 $[-1,1]$ 之间，是无量纲的标准化协方差。
- **期望的线性性**：$\mathbb{E}[aX + bY + c] = a\mathbb{E}[X] + b\mathbb{E}[Y] + c$，这一性质在推导各种估计量的性质时极为重要。

## 数学推导
期望的基本性质：
$$
\mathbb{E}[X + Y] = \mathbb{E}[X] + \mathbb{E}[Y] \quad (\text{线性性})
$$
$$
\mathbb{E}[XY] = \mathbb{E}[X]\mathbb{E}[Y] + \text{Cov}(X,Y)
$$

方差的性质：
$$
\text{Var}(aX + b) = a^2 \text{Var}(X)
$$
$$
\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)
$$

协方差矩阵的定义：
$$
\Sigma = \mathbb{E}[(\mathbf{X} - \boldsymbol{\mu})(\mathbf{X} - \boldsymbol{\mu})^T] = 
\begin{pmatrix}
\text{Var}(X_1) & \text{Cov}(X_1, X_2) & \dots & \text{Cov}(X_1, X_d) \\
\text{Cov}(X_2, X_1) & \text{Var}(X_2) & \dots & \text{Cov}(X_2, X_d) \\
\vdots & \vdots & \ddots & \vdots \\
\text{Cov}(X_d, X_1) & \text{Cov}(X_d, X_2) & \dots & \text{Var}(X_d)
\end{pmatrix}
$$

线性变换的协方差：对 $\mathbf{Y} = A\mathbf{X} + \mathbf{b}$，
$$
\mathbb{E}[\mathbf{Y}] = A\boldsymbol{\mu} + \mathbf{b}, \quad \text{Cov}(\mathbf{Y}) = A\Sigma A^T
$$

## 直观理解
- **期望即重心**：期望是概率分布的"质心"或"重心"。如果把概率分布想象成分布在数轴上的质量，期望就是平衡点的位置。
- **方差即散布程度**：方差衡量分布的"宽度"。两个分布可能均值相同（如 $N(0,1)$ 和 $N(0,5)$），但方差大的分布更"胖"、更分散，不确定性更大。
- **协方差即协同变化**：协方差衡量两个变量的"同步性"。身高和体重一般正相关（协方差为正），学习时间和错误数负相关（协方差为负）。协方差矩阵将这种关系推广到了高维。
- **协方差矩阵的几何**：协方差矩阵的特征向量指向数据分布的主轴方向，特征值是对应方向的方差。如果 $\Sigma$ 是球形的（特征值相等），数据呈球形分布；如果特征值差异大，数据呈椭球分布。

## 代码示例
```python
import numpy as np

# 样本统计量计算
np.random.seed(42)
data = np.random.randn(1000, 3)  # 1000个三维样本

# 样本均值
mean = np.mean(data, axis=0)
print(f"样本均值: {mean}")

# 样本协方差矩阵
cov = np.cov(data, rowvar=False)
print(f"样本协方差矩阵:\n{cov}")

# 样本相关系数矩阵
corr = np.corrcoef(data, rowvar=False)
print(f"相关系数矩阵:\n{corr}")

# 验证协方差与相关系数的关系
std = np.std(data, axis=0)
corr_from_cov = cov / np.outer(std, std)
print(f"从协方差计算的相关系数:\n{corr_from_cov}")

# 期望的线性性验证
X = np.random.randn(100000)
Y = np.random.randn(100000)
a, b, c = 2, 3, 1

E_X = np.mean(X)
E_Y = np.mean(Y)
E_aX_bY_c = np.mean(a*X + b*Y + c)
formula = a*E_X + b*E_Y + c

print(f"\n验证 E[aX+bY+c] = aE[X] + bE[Y] + c")
print(f"  计算值: {E_aX_bY_c:.4f}")
print(f"  公式值: {formula:.4f}")

# 协方差矩阵的特征分解
eigvals, eigvecs = np.linalg.eigh(cov)
print(f"\n协方差矩阵特征值: {eigvals}")
print(f"总方差 (迹): {np.trace(cov):.4f} = 特征值和: {np.sum(eigvals):.4f}")

# 引入相关性
mean_vec = np.array([1, 2])
cov_matrix = np.array([[1, 0.8], [0.8, 2]])
correlated_data = np.random.multivariate_normal(mean_vec, cov_matrix, 500)
empirical_cov = np.cov(correlated_data, rowvar=False)
print(f"\n相关数据:")
print(f"  理论协方差:\n{cov_matrix}")
print(f"  经验协方差:\n{empirical_cov}")
```

## 深度学习关联
- **特征白化 (Whitening)**：将数据 $\mathbf{X}$ 变换为协方差为单位矩阵的 $\mathbf{Z} = \Sigma^{-1/2}(\mathbf{X} - \boldsymbol{\mu})$。ZCA 白化在图像预处理中去除像素间的二阶相关性，使网络各层输入更各向同性，加速收敛。BatchNorm 在一定程度上起到了类似作用。
- **表示的相关性分析**：CCA (典型相关分析) 和 CKA (中心核对齐) 利用协方差矩阵分析不同神经网络层的表示相似性。通过比较特征协方差矩阵的特征值谱，可以判断网络是否学到了紧凑的表示。
- **PCA 与协方差**：PCA 的核心是对数据协方差矩阵做特征分解。最大特征值对应的特征向量是数据方差最大的方向（第一主成分）。在深度学习中，PCA 可用于特征可视化、降维预处理和噪声去除。
- **风险最小化中的协方差**：在投资组合优化和马科维茨理论中，协方差矩阵用于衡量资产间的风险关联。类似地，在多任务学习中，任务间的协方差结构可以指导任务权重分配和梯度融合策略。
