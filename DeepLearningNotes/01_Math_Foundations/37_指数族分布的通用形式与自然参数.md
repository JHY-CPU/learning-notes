# 指数族分布的通用形式与自然参数

## 核心概念
- **指数族分布 (Exponential Family)**：形式为 $p(x|\theta) = h(x) \exp(\eta(\theta)^T T(x) - A(\eta))$ 的分布类。包括高斯、伯努利、泊松、Gamma、Beta、狄利克雷等最常见分布。
- **自然参数 (Natural Parameter)**：$\eta$ 是分布的"自然"参数化形式。不同于均值参数 $\mu$ 或常用参数（如高斯中的 $\mu, \sigma^2$），自然参数使分布呈线性指数形式。从标准参数到自然参数的映射有时（如高斯）涉及复杂变换。
- **充分统计量 (Sufficient Statistic)**：$T(x)$ 包含数据 $x$ 中关于参数的所有信息。给定 $T(x)$，原始数据 $x$ 不再包含额外信息，这是充分性的核心。例如，高斯分布的充分统计量是 $T(x) = (x, x^2)^T$。
- **对数配分函数 (Log-partition Function)**：$A(\eta)$ 是归一化常数，满足 $A(\eta) = \log \int h(x) \exp(\eta^T T(x)) dx$。$A(\eta)$ 的导数可以给出分布的矩：$\nabla_\eta A(\eta) = \mathbb{E}[T(X)]$，$\nabla^2_\eta A(\eta) = \text{Cov}[T(X)]$。
- **期望参数 (Expectation Parameter)**：$\mu = \mathbb{E}[T(X)] = \nabla A(\eta)$。自然参数 $\eta$ 和期望参数 $\mu$ 之间通过 Legendre 变换互为对偶。这个对偶性在变分推断中很重要。
- **凸性**：$A(\eta)$ 是关于 $\eta$ 的凸函数，指数族分布是凸的。这保证了 MLE 有唯一解（在可识别条件下）。

## 数学推导
指数族的通用形式：
$$
p(x|\theta) = h(x) \exp\left( \eta(\theta)^T T(x) - A(\eta(\theta)) \right)
$$

其中：
- $h(x)$：基础测度（base measure），与参数无关
- $\eta(\theta)$：自然参数（vector）
- $T(x)$：充分统计量（vector）
- $A(\eta)$：对数配分函数

**伯努利分布**的指数族形式：
$$
p(x|p) = p^x (1-p)^{1-x} = \exp\left( x \log\frac{p}{1-p} + \log(1-p) \right)
$$
自然参数：$\eta = \log\frac{p}{1-p}$（log-odds），充分统计量 $T(x) = x$，$A(\eta) = \log(1+e^\eta)$

**高斯分布**的指数族形式（已知方差 $\sigma^2$）：
$$
p(x|\mu) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left( \frac{\mu}{\sigma^2} x - \frac{\mu^2}{2\sigma^2} \right)
$$
自然参数：$\eta = \mu/\sigma^2$

对数配分函数的性质：
$$
\mathbb{E}[T(X)] = \nabla A(\eta), \quad \text{Cov}[T(X)] = \nabla^2 A(\eta)
$$

## 直观理解
- **"线性结构"的简洁性**：指数族的核心是将分布写成 $\exp(\eta^T T(x))$ 的形式，这意味着对数概率是充分统计量的线性函数。这简化了推断、学习和计算。
- **为什么"指数族"无处不在**：指数族包括了绝大多数我们熟悉的分布。更重要的是，最大熵原理指出：在给定矩约束下（如均值和方差），熵最大的分布正是指数族分布。所以指数族是"最不确定但尊重约束"的分布。
- **自然参数是"深层参数"**：通常参数（如 $p$ 或 $\mu$）更直观，但自然参数 $\eta$ 在数学上更优美——它让分布呈线性结构，简化了梯度计算和贝叶斯更新。

## 代码示例
```python
import numpy as np
from scipy.special import expit, logit

# 1. 伯努利指数族
p = 0.3
eta = logit(p)  # 自然参数
print(f"伯努利: p={p}, 自然参数 η=logit(p)={eta:.4f}")

# 对数配分函数 A(η) = log(1 + e^η)
def A_bernoulli(eta):
    return np.log(1 + np.exp(eta))

# 验证 E[T(x)] = dA/dη = e^η/(1+e^η) = sigmoid(η) = p
def expected_T_bernoulli(eta):
    return expit(eta)  # sigmoid

print(f"A(η) = {A_bernoulli(eta):.4f}")
print(f"dA/dη = {expected_T_bernoulli(eta):.4f} = p")

# 2. 高斯指数族 (已知方差)
mu, sigma = 1.0, 2.0
eta_gauss = mu / sigma**2  # 自然参数
print(f"\n高斯N({mu},{sigma}^2): 自然参数 η={eta_gauss:.4f}")

# 3. 从指数族采样并 MLE
np.random.seed(42)
n = 1000
true_p = 0.3
# 从伯努利采样
x = np.random.binomial(1, true_p, n)

# MLE: p_hat = mean(x)
p_mle = np.mean(x)
eta_mle = logit(p_mle)
print(f"\nMLE: p_hat={p_mle:.4f}, η_hat={eta_mle:.4f}")
print(f"  A(η_hat) = {A_bernoulli(eta_mle):.4f}")
print(f"  dA/dη(η_hat) = {expected_T_bernoulli(eta_mle):.4f} = p_hat")

# 4. 指数族的充分统计量
# 高斯分布的充分统计量
data = np.random.randn(100)
T1 = np.mean(data)  # 一阶矩
T2 = np.mean(data**2)  # 二阶矩
print(f"\n高斯充分统计量: T1(均值)={T1:.4f}, T2(二阶矩)={T2:.4f}")

# 5. 对数配分函数的凸性验证
etas = np.linspace(-5, 5, 100)
A_vals = [A_bernoulli(e) for e in etas]

# 检查是否凸: A''(η) >= 0
def A_second_deriv(eta):
    s = expit(eta)
    return s * (1 - s)  # sigmoid'(eta) = sigmoid*(1-sigmoid)

d2_vals = [A_second_deriv(e) for e in etas]
print(f"\nA''(η) 最小值: {min(d2_vals):.6f} >= 0, 凸性成立!")

# 6. 广义线性模型 (GLM) 的指数族视角
# GLM 假设 y|x 服从指数族分布
# 线性预测器 η = w^T x, 通过链接函数 g 映射到均值: g(E[y]) = w^T x
print(f"\nGLM 的指数族连接:")
print(f"  Bernoulli: 链接函数 = logit (Logistic回归)")
print(f"  Poisson: 链接函数 = log (泊松回归)")
print(f"  Gaussian: 链接函数 = identity (线性回归)")
```

## 深度学习关联
- **Softmax 输出与类别分布**：分类神经网络的 Softmax 输出层可以看作指数族分布中的类别分布（Categorical）的自然参数表示。输入 logits $z_c$ 就是自然参数 $\eta_c$，Softmax 是 $\eta$ 到期望参数 $\mu$ 的变换（即对数配分函数的梯度）。
- **能量基模型 (EBM)**：指数族分布的形式 $p(x) \propto \exp(-E(x))$ 将能量函数 $E(x)$ 与自然参数联系起来。基于能量的模型（如 RBM、Hopfield 网络、EBM）使用神经网络参数化能量函数 $E_\theta(x)$，其对数配分函数的梯度计算是训练的核心难点。
- **变分推理中的指数族**：变分推断常假设变分分布 $q(z)$ 属于指数族。这是因为指数族具有良好的性质——ELBO 中对 $q$ 的 KL 散度有闭式表达式 $D_{\text{KL}}(q\|p)$，且指数族分布的期望参数和自然参数之间的对偶性简化了变分更新。
- **自然梯度下降 (Natural Gradient)**：自然梯度使用 Fisher 信息矩阵 $F(\theta)$（等价于 $\nabla^2 A(\eta)$）调整梯度方向：$\theta_{t+1} = \theta_t - \eta F^{-1} \nabla L$。对于指数族分布，自然梯度具有简洁的形式——它相当于在自然参数空间中进行标准梯度下降，这解释了为什么自然梯度在某些情况下收敛更快。
