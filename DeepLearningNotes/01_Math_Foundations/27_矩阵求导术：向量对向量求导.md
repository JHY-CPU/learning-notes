# 矩阵求导术：向量对向量求导

## 核心概念
- **向量对向量求导**：函数 $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$，其导数是一个 $m \times n$ 矩阵，即 Jacobian 矩阵。
- **Jacobian 矩阵**：$J_{ij} = \frac{\partial f_i}{\partial x_j}$，每个输出分量对所有输入分量的偏导。
- **分母布局 vs 分子布局**：
  - 分母布局：$\frac{\partial \mathbf{f}}{\partial \mathbf{x}} \in \mathbb{R}^{m \times n}$（Jacobian，最常见的约定）
  - 分子布局：$\frac{\partial \mathbf{f}}{\partial \mathbf{x}} \in \mathbb{R}^{n \times m}$（梯度矩阵）
- **常见向量函数求导**：
  - $\frac{\partial}{\partial \mathbf{x}} (A\mathbf{x}) = A$（线性变换）
  - $\frac{\partial}{\partial \mathbf{x}} (A\mathbf{x} + \mathbf{b}) = A$（仿射变换）
  - $\frac{\partial}{\partial \mathbf{x}} (\mathbf{x}^T A) = A^T$（行向量）
  - $\frac{\partial}{\partial \mathbf{x}} \|\mathbf{x}\|_2 = \frac{\mathbf{x}}{\|\mathbf{x}\|}$（仅当 $\mathbf{x} \neq 0$）
- **逐元素函数的 Jacobian**：对于 $\mathbf{f}(\mathbf{x}) = [f(x_1), \dots, f(x_n)]^T$，Jacobian 是对角矩阵 $\text{diag}(f'(x_1), \dots, f'(x_n))$。

## 数学推导
线性变换 $\mathbf{y} = A\mathbf{x}$ 的 Jacobian：
$$
\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = A
$$

因为 $y_i = \sum_j A_{ij} x_j$，所以 $\frac{\partial y_i}{\partial x_j} = A_{ij}$。

内积运算 $\mathbf{y} = \mathbf{a}^T \mathbf{x}$（输出为标量）：
$$
\frac{\partial y}{\partial \mathbf{x}} = \mathbf{a}^T \quad (\text{分子布局行向量})
$$

逐元素激活函数 $\mathbf{y} = \sigma(\mathbf{x})$：
$$
\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = \text{diag}(\sigma'(\mathbf{x}_1), \dots, \sigma'(\mathbf{x}_n))
$$

链式法则（向量对向量）：
如果 $\mathbf{y} = \mathbf{f}(\mathbf{g}(\mathbf{x}))$，则：
$$
\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = \frac{\partial \mathbf{f}}{\partial \mathbf{g}} \cdot \frac{\partial \mathbf{g}}{\partial \mathbf{x}}
$$

## 直观理解
- **Jacobian 是放大镜**：Jacobian 矩阵告诉你，当输入 $\mathbf{x}$ 有微小变化 $\Delta \mathbf{x}$ 时，输出 $\mathbf{y}$ 的变化近似为 $\Delta \mathbf{y} \approx J \Delta \mathbf{x}$。这是"局部线性近似"的向量版本。
- **激活函数的 Jacobian**：对于 ReLU 激活函数 $\text{ReLU}(x) = \max(0, x)$，其导数是 0（$x<0$）或 1（$x>0$）。所以 Jacobian 是对角矩阵，对角线元素为 0 或 1。在反向传播中，这相当于"门控"——当 $x<0$ 时梯度被阻断。
- **线性层的 Jacobian**：全连接层 $\mathbf{y} = W\mathbf{x} + \mathbf{b}$ 的 Jacobian 就是 $W$ 本身。这意味着反向传播中误差信号左乘 $W^T$ 即可传播到前一层——即反向传播的"矩阵转置"性质。

## 代码示例
```python
import numpy as np

# 1. 验证线性变换的 Jacobian
W = np.array([[2, 1], [1, 3]])
x = np.array([1.0, 2.0])

f_linear = lambda x: W @ x

# 数值 Jacobian
def numerical_jacobian(f, x, h=1e-6):
    m = len(f(x))
    n = len(x)
    J = np.zeros((m, n))
    f0 = f(x)
    for j in range(n):
        x_plus = x.copy()
        x_plus[j] += h
        J[:, j] = (f(x_plus) - f0) / h
    return J

J_numerical = numerical_jacobian(f_linear, x)
print(f"线性变换 Wx 的 Jacobian:")
print(f"  理论值:\n{W}")
print(f"  数值值:\n{J_numerical}")

# 2. 逐元素函数的 Jacobian
def elementwise_fn(x):
    return np.array([x[0]**2, np.sin(x[1]), np.exp(x[2])])

x = np.array([2.0, np.pi/4, 0.5])
J_numeric = numerical_jacobian(elementwise_fn, x)
J_analytic = np.diag([2*x[0], np.cos(x[1]), np.exp(x[2])])
print(f"\n逐元素函数 Jacobian:")
print(f"  解析 (对角):\n{np.diag(J_analytic)}")
print(f"  数值:\n{J_numeric}")
print(f"  误差: {np.linalg.norm(J_analytic - J_numeric):.2e}")

# 3. 链式法则验证
def g(x):
    return np.array([x[0]**2, x[1]**3])

def f(y):
    return np.array([y[0] + y[1], y[0] * y[1]])

def fg(x):
    return f(g(x))

x = np.array([2.0, 1.5])

# 直接数值 Jacobian
J_fg_numerical = numerical_jacobian(fg, x)

# 链式法则
y = g(x)
J_f = numerical_jacobian(f, y)
J_g = numerical_jacobian(g, x)
J_fg_chain = J_f @ J_g

print(f"\n链式法则验证:")
print(f"  直接数值:\n{J_fg_numerical}")
print(f"  链式法则:\n{J_fg_chain}")
print(f"  误差: {np.linalg.norm(J_fg_numerical - J_fg_chain):.2e}")

# 4. 神经网络层的前向与反向 Jacobian
class LinearLayer:
    def __init__(self, W, b):
        self.W = W
        self.b = b
    def forward(self, x):
        return self.W @ x + self.b
    def backward(self, grad_output):
        # grad_output 是上一层的梯度 (标量对输出 y 的 Jacobian)
        # 返回梯度对输入 x 的 Jacobian
        return grad_output @ self.W  # 或 self.W.T @ grad_output (取决于布局)

W = np.random.randn(3, 4)
b = np.random.randn(3)
layer = LinearLayer(W, b)
x = np.random.randn(4)

# 反向传播的 Jacobian
grad_output = np.random.randn(3)  # dL/dy
grad_input = layer.backward(grad_output)  # dL/dx
print(f"\n反向传播: grad_input = {grad_input}")
print(f"        = W^T @ grad_output = {W.T @ grad_output}")
print(f"  一致? {np.allclose(grad_input, W.T @ grad_output)}")
```

## 深度学习关联
- **反向传播的核心公式**：向量对向量 Jacobian 是反向传播的数学基础。对于网络层 $\mathbf{h}^{(l)} = f^{(l)}(\mathbf{h}^{(l-1)})$，反向传播公式为：
  $$\frac{\partial L}{\partial \mathbf{h}^{(l-1)}} = \frac{\partial L}{\partial \mathbf{h}^{(l)}} \cdot \frac{\partial \mathbf{h}^{(l)}}{\partial \mathbf{h}^{(l-1)}}$$
  这里的 $\frac{\partial \mathbf{h}^{(l)}}{\partial \mathbf{h}^{(l-1)}}$ 就是 Jacobian 矩阵。

- **自动微分 (Autograd)**：现代深度学习框架（PyTorch、TensorFlow）自动跟踪计算图并计算梯度。每个操作的 backward 函数本质上就是在计算 Jacobian-vector 乘积（VJP）或 vector-Jacobian 乘积（VJP），无需显式构造完整的 Jacobian。

- **Jacobian 正则化**：在一些工作中，对网络输入到输出的 Jacobian 矩阵施加惩罚 $\|J_f(\mathbf{x})\|_F^2$ 可以增强模型的鲁棒性，因为它鼓励函数在输入附近保持平坦（对小的输入变化不敏感），从而提高对抗鲁棒性。

- **神经正切核 (NTK)**：神经正切核 $K(\mathbf{x}_i, \mathbf{x}_j) = \nabla_\theta f(\mathbf{x}_i)^T \nabla_\theta f(\mathbf{x}_j)$ 使用网络输出对参数的 Jacobian（而不是对输入的 Jacobian）来研究无限宽网络的训练动态。NTK 理论揭示了深度网络与核方法之间的深刻联系。
