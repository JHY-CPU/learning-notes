# 拉格朗日乘数法与约束优化

## 核心概念
- **约束优化问题**：在满足等式约束 $h_i(\mathbf{x}) = 0$ 和不等式约束 $g_j(\mathbf{x}) \leq 0$ 的条件下，最小化目标函数 $f(\mathbf{x})$。约束优化无处不在：资源分配、物理模拟、机器学习模型设计。
- **拉格朗日乘数法 (Lagrange Multiplier)**：将等式约束优化转化为无约束优化的方法。构造拉格朗日函数 $L(\mathbf{x}, \lambda) = f(\mathbf{x}) + \sum_i \lambda_i h_i(\mathbf{x})$，最优解满足 $\nabla L = 0$。
- **KKT 条件 (Karush-Kuhn-Tucker)**：不等式约束优化的充要条件（在约束规格下）。KKT 条件包括：梯度为零、原始可行、互补松弛和对偶可行。
- **对偶问题 (Dual Problem)**：原始问题的对偶为 $\max_{\lambda \succeq 0} \min_{\mathbf{x}} L(\mathbf{x}, \lambda)$。强对偶性（对偶间隙为零）在凸优化中成立。对偶问题是理解 SVM 和约束优化的关键。
- **惩罚法 vs 拉格朗日法**：惩罚法将约束作为惩罚项加入目标函数，近似满足约束。拉格朗日法则通过乘子精确满足约束。增广拉格朗日法结合了两者的优点。

## 数学推导
等式约束的拉格朗日乘数法：
$$
\min_{\mathbf{x}} f(\mathbf{x}) \quad \text{s.t.} \quad h_i(\mathbf{x}) = 0, \; i=1,\dots,m
$$

拉格朗日函数：
$$
L(\mathbf{x}, \boldsymbol{\lambda}) = f(\mathbf{x}) + \sum_{i=1}^m \lambda_i h_i(\mathbf{x})
$$

最优性条件：
$$
\nabla_{\mathbf{x}} L = \nabla f(\mathbf{x}^*) + \sum_{i=1}^m \lambda_i^* \nabla h_i(\mathbf{x}^*) = 0
$$
$$
\nabla_{\lambda_i} L = h_i(\mathbf{x}^*) = 0
$$

不等式约束的 KKT 条件：
$$
\nabla f(\mathbf{x}^*) + \sum_i \lambda_i^* \nabla g_i(\mathbf{x}^*) + \sum_j \mu_j^* \nabla h_j(\mathbf{x}^*) = 0
$$
$$
g_i(\mathbf{x}^*) \leq 0, \quad h_j(\mathbf{x}^*) = 0 \quad (\text{原始可行})
$$
$$
\lambda_i^* \geq 0 \quad (\text{对偶可行})
$$
$$
\lambda_i^* g_i(\mathbf{x}^*) = 0 \quad (\text{互补松弛})
$$

互补松弛的直观含义：如果约束 $g_i(\mathbf{x}^*) < 0$ 是"松的"（不在边界上），则 $\lambda_i^* = 0$；如果 $\lambda_i^* > 0$，则约束 $g_i(\mathbf{x}^*) = 0$ 是"紧的"（在边界上）。

## 直观理解
- **等高线相切**：在约束 $h(x,y)=0$ 下最小化 $f(x,y)$，最优解出现在目标函数等值线与约束曲线相切的位置。此时，$f$ 的梯度和 $h$ 的梯度平行，所以 $\nabla f = \lambda \nabla h$。$\lambda$ 就是"拉格朗日乘子"。
- **经济解释**：如果把 $f$ 看作成本，$h(x)=0$ 看作资源约束，$\lambda$ 就是约束的"影子价格"——约束稍微放松一点，目标函数能改善多少。$\lambda$ 大意味着这个约束是主要的瓶颈。
- **互补松弛的直觉**：不等式约束就像预算约束。如果你的钱没花完（约束松了），说明这个约束不影响你的决策（乘子为0）。如果你觉得钱不够花（约束紧），说明预算确实限制了你的选择（乘子大于0）。

## 代码示例
```python
import numpy as np
from scipy.optimize import minimize

# 等式约束优化: 在 x^2 + y^2 = 1 上最大化 x + y
# 等价于在单位圆上求 x+y 的最大值

def f(xy):
    return -xy[0] - xy[1]  # 负号 => 最大化

def constraint_eq(xy):
    return xy[0]**2 + xy[1]**2 - 1

# 使用 scipy 求解
x0 = np.array([0.0, 1.0])
cons = {'type': 'eq', 'fun': constraint_eq}
result = minimize(f, x0, constraints=cons)
print(f"等式约束优化结果: x={result.x[0]:.4f}, y={result.x[1]:.4f}")
print(f"最优值 x+y = {-result.fun:.4f}")

# 拉格朗日乘数法手动求解
# L(x,y,λ) = -x - y + λ(x^2 + y^2 - 1)
# ∂L/∂x = -1 + 2λx = 0 => x = 1/(2λ)
# ∂L/∂y = -1 + 2λy = 0 => y = 1/(2λ)
# x^2 + y^2 = 1 => 2*(1/(2λ))^2 = 1 => λ = 1/√2
lam = 1 / np.sqrt(2)
x_opt = 1 / (2*lam)
y_opt = 1 / (2*lam)
print(f"解析解: x={x_opt:.4f}, y={y_opt:.4f}, x+y={x_opt+y_opt:.4f}")

# 不等式约束: 在 x >= 0, y >= 0, x + y <= 1 下最大化 f
def constraint_ineq(xy):
    return 1 - xy[0] - xy[1]  # x + y <= 1

cons2 = [{'type': 'ineq', 'fun': lambda xy: xy[0]},  # x >= 0
         {'type': 'ineq', 'fun': lambda xy: xy[1]},  # y >= 0
         {'type': 'ineq', 'fun': constraint_ineq}]    # x + y <= 1

f2 = lambda xy: -xy[0]*xy[1]  # 最大化 xy
x0 = np.array([0.3, 0.3])
result2 = minimize(f2, x0, constraints=cons2)
print(f"\n不等式约束优化: x={result2.x[0]:.4f}, y={result2.x[1]:.4f}")
print(f"最优值 xy = {-result2.fun:.4f}")

# 验证 KKT 条件
x_opt, y_opt = result2.x
print(f"KKT: x={x_opt:.4f}>=0, y={y_opt:.4f}>=0, x+y={x_opt+y_opt:.4f}<=1")
```

## 深度学习关联
- **SVM 与拉格朗日对偶**：支持向量机的训练就是一个带不等式约束的二次规划问题。通过拉格朗日对偶性，将原始问题转化为对偶问题，从而引入核技巧并得到稀疏解（仅支持向量对应的 $\lambda_i > 0$）。SVM 是拉格朗日乘数法在 ML 中最经典的应用。
- **正则化的约束优化视角**：L2 正则化 $\min L(\theta) + \lambda\|\theta\|_2^2$ 等价于约束优化 $\min L(\theta)$ s.t. $\|\theta\|_2^2 \leq C$。$\lambda$ 相当于约束的拉格朗日乘子。这种视角解释了为什么增大 $\lambda$ 会"压缩"权重——等效地收紧了对权重的约束。
- **变分自编码器 (VAE)**：VAE 的 ELBO 推导中最大化证据下界可以看作约束优化问题。重参数化技巧和 KL 散度项之间的平衡也可以用拉格朗日乘数法理解——$\beta$-VAE 显式地用 $\beta$ 控制 KL 项的权重。
- **约束优化在神经网络训练中的应用**：在实际工程中，有时需要对网络输出施加约束（如公平性约束、安全性约束）。拉格朗日方法通过构建增广损失函数 $L + \sum_i \lambda_i g_i$ 来"软性"满足约束，$\lambda_i$ 在训练过程中自适应调整（如使用对偶梯度下降）。
