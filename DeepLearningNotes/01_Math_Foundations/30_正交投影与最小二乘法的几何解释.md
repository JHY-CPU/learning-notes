# 正交投影与最小二乘法的几何解释

## 核心概念
- **正交投影 (Orthogonal Projection)**：将向量 $\mathbf{y}$ 投影到子空间 $S$ 上，得到 $\hat{\mathbf{y}} = \text{proj}_S(\mathbf{y})$，使得 $\mathbf{y} - \hat{\mathbf{y}}$ 与 $S$ 正交。投影残差是所有可能近似中范数最小的。
- **投影矩阵**：投影到 $A$ 的列空间的投影矩阵为 $P = A(A^T A)^{-1} A^T$。其性质包括幂等性 $P^2 = P$ 和对称性 $P^T = P$。
- **最小二乘问题 (Least Squares)**：寻找 $\mathbf{x}$ 使得 $\|A\mathbf{x} - \mathbf{b}\|_2^2$ 最小。当方程 $A\mathbf{x} = \mathbf{b}$ 无解时（$b$ 不在 $A$ 的列空间中），最小二乘解给出"最佳近似"。
- **正规方程 (Normal Equation)**：最小二乘解满足 $A^T A \mathbf{x} = A^T \mathbf{b}$。几何意义是残差向量 $\mathbf{b} - A\mathbf{x}$ 与 $A$ 的列空间正交。
- **几何解释**：最小二乘法等价于将 $\mathbf{b}$ 正交投影到 $A$ 的列空间上，投影向量 $\hat{\mathbf{b}} = A\mathbf{x}_{\text{LS}}$ 就是 $A$ 的列空间中离 $\mathbf{b}$ 最近的点。

## 数学推导
最小二乘问题：
$$
\min_{\mathbf{x}} \| A\mathbf{x} - \mathbf{b} \|_2^2
$$

展开目标函数：
$$
\|A\mathbf{x} - \mathbf{b}\|^2 = (A\mathbf{x} - \mathbf{b})^T (A\mathbf{x} - \mathbf{b}) = \mathbf{x}^T A^T A \mathbf{x} - 2\mathbf{x}^T A^T \mathbf{b} + \mathbf{b}^T \mathbf{b}
$$

对 $\mathbf{x}$ 求导并设为零：
$$
\frac{\partial}{\partial \mathbf{x}} \|A\mathbf{x} - \mathbf{b}\|^2 = 2A^T A \mathbf{x} - 2A^T \mathbf{b} = 0
$$

得到正规方程：
$$
A^T A \mathbf{x} = A^T \mathbf{b}
$$

解为（当 $A^T A$ 可逆时）：
$$
\mathbf{x}_{\text{LS}} = (A^T A)^{-1} A^T \mathbf{b}
$$

投影到列空间的向量：
$$
\hat{\mathbf{b}} = A \mathbf{x}_{\text{LS}} = A(A^T A)^{-1} A^T \mathbf{b} = P \mathbf{b}
$$

正交性条件（残差与列空间正交）：
$$
A^T (\mathbf{b} - A\mathbf{x}_{\text{LS}}) = \mathbf{0}
$$

## 直观理解
- **"最近的"近似解**：如果 $A\mathbf{x} = \mathbf{b}$ 没有精确解（$b$ 不在列空间中），最小二乘法给出在该空间中"最近"的近似。就像在一个房间里找一个点，使其到天花板上某点的直线最短——实际上是垂直投影到天花板。
- **投影的物理意义**：想象一个手电筒从正上方照射向量 $\mathbf{b}$，$\mathbf{b}$ 在列空间平面的"影子"就是投影 $\hat{\mathbf{b}}$。这个影子是所有列空间向量中与 $\mathbf{b}$ 距离最近的。
- **残差与列空间正交**：从 $\mathbf{b}$ 到 $\hat{\mathbf{b}}$ 的连线垂直于列空间平面。这就像地面上竖立的一根杆子——杆子垂直于地面（列空间），长度就是残差范数。

## 代码示例
```python
import numpy as np

# 1. 二维空间的投影
# 列空间：过原点的直线方向 v = [1, 2]
v = np.array([1, 2]).reshape(-1, 1)
b = np.array([3, 1])

# 投影矩阵
P = v @ np.linalg.inv(v.T @ v) @ v.T
b_hat = P @ b

print("二维投影:")
print(f"  原向量 b = {b}")
print(f"  投影到v得到 b_hat = {b_hat}")
print(f"  残差 = {b - b_hat}")
print(f"  残差与v正交? {np.dot(b - b_hat, v.flatten()):.6f}")

# 2. 最小二乘线性回归
np.random.seed(42)
n = 50
X = np.column_stack([np.ones(n), np.random.randn(n, 2)])  # 加截距项
true_w = np.array([1.5, 2.0, -1.0])
y = X @ true_w + 0.5 * np.random.randn(n)

# 正规方程求解
w_ls = np.linalg.inv(X.T @ X) @ X.T @ y
y_pred = X @ w_ls
residuals = y - y_pred

print(f"\n最小二乘回归:")
print(f"  真实权重: {true_w}")
print(f"  估计权重: {w_ls}")
print(f"  残差范数: {np.linalg.norm(residuals):.4f}")

# 验证正交性：残差与 X 的列空间正交
print(f"  X^T @ 残差 ≈ 0? {np.max(np.abs(X.T @ residuals)):.2e}")

# 3. QR 分解求解最小二乘（数值更稳定）
Q, R = np.linalg.qr(X)
w_qr = np.linalg.solve(R, Q.T @ y)
print(f"\nQR解与正规方程解一致? {np.allclose(w_ls, w_qr)}")

# 4. 投影矩阵的幂等性验证
P = X @ np.linalg.inv(X.T @ X) @ X.T
print(f"\n投影矩阵性质:")
print(f"  P^2 = P? {np.allclose(P @ P, P)}")
print(f"  P^T = P? {np.allclose(P.T, P)}")
print(f"  迹 = rank? {np.trace(P):.4f} (秩={np.linalg.matrix_rank(X)})")

# 5. 几何可视化辅助：不同维度的投影
print("\n--- 不同维度的投影示例 ---")
# 一维子空间（直线）
A_1d = np.array([[1], [2], [3]])
b_3d = np.array([2, 1, 4])
x_1d = np.linalg.lstsq(A_1d, b_3d, rcond=None)[0]
b_proj_1d = A_1d @ x_1d
print(f"3D点到直线的投影: b_proj = {b_proj_1d}")
print(f"  残差: {np.linalg.norm(b_3d - b_proj_1d):.4f}")

# 二维子空间（平面）
A_2d = np.array([[1, 0], [0, 1], [1, 1]])
b_3d = np.array([2, 1, 4])
x_2d = np.linalg.lstsq(A_2d, b_3d, rcond=None)[0]
b_proj_2d = A_2d @ x_2d
print(f"3D点到平面的投影: b_proj = {b_proj_2d}")
print(f"  残差: {np.linalg.norm(b_3d - b_proj_2d):.4f}")
```

## 深度学习关联
- **线性层与最小二乘**：线性回归模型 $\hat{y} = W\mathbf{x} + \mathbf{b}$ 的训练等价于最小二乘问题（当使用 MSE 损失时）。正规方程 $X^T X W = X^T Y$ 给出了解析解。但在深度学习中，由于特征维度巨大和 batch 训练，一般使用梯度下降而非直接求解正规方程。
- **残差网络 (ResNet) 的投影解释**：ResNet 的瓶颈设计 $y = F(x) + x$ 可以看作在恒等映射附近的扰动。$F(x)$ 学习的是残差，就像最小二乘中的残差项。当最优映射接近恒等映射时，残差比原始映射更容易学习。
- **PCA 与正交投影**：PCA 将高维数据投影到由前 $k$ 个主成分张成的子空间上，使投影后的方差最大。这是另一种正交投影——最大化方差等价于最小化投影误差（根据 PCA 的双重性）。PCA 与最小二乘共享相同的数学结构（SVD / 特征分解）。
- **注意力机制作为软投影**：Transformer 中的多头注意力可以看作一种"软"正交投影。每个注意力头计算值的加权平均，权重由查询和键的相似度决定。这类似于将查询投影到由键向量张成的子空间上，但使用 Softmax 加权而非严格正交。
