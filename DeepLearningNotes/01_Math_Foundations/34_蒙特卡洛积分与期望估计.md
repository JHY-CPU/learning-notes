# 蒙特卡洛积分与期望估计

## 核心概念
- **蒙特卡洛方法 (Monte Carlo Method)**：使用随机采样来估计确定性量（如积分、期望）的计算方法。基本思想：用样本均值近似总体期望。
- **蒙特卡洛积分**：对 $\mathbb{E}[f(X)] = \int f(x) p(x) dx$，取 $n$ 个独立样本 $x_i \sim p(x)$，估计为 $\hat{I} = \frac{1}{n} \sum_{i=1}^n f(x_i)$。
- **大数定律保证**：当 $n \to \infty$ 时，$\hat{I} \xrightarrow{p} \mathbb{E}[f(X)]$（依概率收敛），即估计是一致的。
- **中心极限定理提供误差**：$\sqrt{n}(\hat{I} - \mathbb{E}[f(X)]) \xrightarrow{d} \mathcal{N}(0, \text{Var}[f(X)])$。蒙特卡洛估计的误差以 $O(1/\sqrt{n})$ 速度收敛，与维度无关！
- **方差缩减技术**：重要性采样、控制变量法、对偶变量法、分层采样等可以减少估计方差，加快收敛。
- **与确定性数值积分的对比**：在低维（$d \leq 3$）中，确定性方法（如梯形法）收敛速度 $O(n^{-2/d})$ 优于蒙特卡洛。但在高维中，蒙特卡洛的 $O(1/\sqrt{n})$ 独立于维度，成为唯一可行的方法。

## 数学推导
蒙特卡洛估计：
$$
I = \int f(x) p(x) dx = \mathbb{E}_{p}[f(X)]
$$
$$
\hat{I}_n = \frac{1}{n} \sum_{i=1}^n f(x_i), \quad x_i \sim p(x)
$$

估计的方差：
$$
\text{Var}(\hat{I}_n) = \frac{1}{n} \text{Var}_p[f(X)] = \frac{1}{n} \left( \mathbb{E}[f(X)^2] - \mathbb{E}[f(X)]^2 \right)
$$

误差分析：
$$
\mathbb{E}[(\hat{I}_n - I)^2] = \frac{\text{Var}[f(X)]}{n} = O\left(\frac{1}{n}\right)
$$

标准误差：
$$
\text{SE}(\hat{I}_n) = \frac{\sigma_f}{\sqrt{n}}
$$

要减少一半的误差，需要将样本量增加 4 倍。

## 直观理解
- **"抽奖"估计积分**：想象一个曲线下的面积就是击中概率。在函数定义域内随机撒点，统计落在曲线下的点比例，乘以总面积就得到积分估计。点撒得越多，估计越准。
- **$O(1/\sqrt{n})$ 的含义**：要减少一位小数的误差，需要增加 100 倍的样本量。这是蒙特卡洛的主要缺点——收敛慢。但好消息是，无论积分是 1 维还是 1000 维，这个速率不变。
- **为什么对高维有效**：确定性方法在 $d$ 维空间中需要 $n^d$ 个网格点来维持精度（维度灾难）。蒙特卡洛方法不在乎维度——只需要在空间中随机采样，采样数量与维度无关。

## 代码示例
```python
import numpy as np

# 1. 蒙特卡洛积分: ∫_0^1 x^2 dx = 1/3
def mc_integrate(f, n_samples, d=1):
    samples = np.random.rand(n_samples, d)
    values = f(samples)
    return np.mean(values)  # [0,1]^d 上均匀分布，体积=1

def f_square(x):
    return x**2

true_val = 1/3
print("蒙特卡洛积分 ∫_0^1 x^2 dx:")
for n in [100, 1000, 10000, 100000]:
    est = mc_integrate(f_square, n)
    error = np.abs(est - true_val)
    print(f"  n={n:6d}: 估计={est:.6f}, 真实={true_val:.6f}, 误差={error:.6f}")

# 2. 误差收敛速度演示
np.random.seed(42)
ns = np.logspace(1, 5, 20, dtype=int)
errors = []
for n in ns:
    est = mc_integrate(f_square, n)
    errors.append(np.abs(est - true_val))

# 将最后几个 n 的误差打印出来
for n, err in zip(ns[-5:], errors[-5:]):
    print(f"  n={n:6d}: 误差={err:.6f}, 理论趋势≈{1/np.sqrt(n):.6f}")

# 3. 高维积分（蒙特卡洛 shine!）
def high_dim_integral(d=10):
    """在 [0,1]^d 上积分 f(x) = sum(x_i^2), 真实值 = d/3"""
    def f(x):
        return np.sum(x**2, axis=1)
    true_val = d / 3
    
    for n in [1000, 10000]:
        est = mc_integrate(f, n, d=d)
        err = np.abs(est - true_val)
        print(f"  d={d}, n={n:6d}: 估计={est:.4f}, 真实={true_val:.4f}, "
              f"误差={err:.4f}")

print("\n高维积分:")
high_dim_integral(5)
high_dim_integral(20)

# 4. 方差与收敛
def estimate_pi(n):
    """用蒙特卡洛估计π: 单位圆面积的比例"""
    points = np.random.rand(n, 2)
    inside = np.sum(points[:, 0]**2 + points[:, 1]**2 <= 1)
    return 4 * inside / n

print("\n蒙特卡洛估计 π:")
for n in [100, 1000, 10000, 100000]:
    pi_est = estimate_pi(n)
    print(f"  n={n:6d}: π≈{pi_est:.4f}, 误差={abs(pi_est - np.pi):.4f}")

# 5. 多次实验估计标准差（验证 CLT）
n_trials = 500
n_samples = 1000
estimates = [estimate_pi(n_samples) for _ in range(n_trials)]
print(f"\n{n_trials} 次重复实验 (n={n_samples}):")
print(f"  π 估计均值: {np.mean(estimates):.4f}")
print(f"  标准差: {np.std(estimates):.4f}")
print(f"  理论标准差: {np.sqrt(np.pi*(4-np.pi)/n_samples):.4f}")
```

## 深度学习关联
- **Dropout 作为贝叶斯近似 (MC Dropout)**：在测试时保持 Dropout 开启并进行多次前向传播，得到蒙特卡洛样本 $\hat{y}^{(1)}, \dots, \hat{y}^{(T)}$。预测均值和方差分别作为最终预测和不确定性估计。这本质上是利用蒙特卡洛积分近似贝叶斯推断：
  $$p(y|x, D) \approx \frac{1}{T} \sum_{t=1}^T p(y|x, w^{(t)}), \quad w^{(t)} \sim q_\theta(w)$$

- **变分推断中的蒙特卡洛梯度**：在 VAE 和贝叶斯神经网络中，ELBO 的梯度通常涉及难以解析计算的期望 $\nabla_\theta \mathbb{E}_{q_\theta}[\log p(x|z)]$。重参数化技巧将随机性转移到噪声变量 $\epsilon$，使梯度可以通过蒙特卡洛样本估计：
  $$\nabla_\theta \mathbb{E}_{q_\theta}[f(z)] \approx \frac{1}{M} \sum_{i=1}^M \nabla_\theta f(g_\theta(\epsilon_i))$$

- **基于分数的生成模型**：得分匹配和扩散模型在训练中需要估计期望 $\mathbb{E}_{x \sim p_{\text{data}}}[\nabla_x \log p(x)]$，这通过蒙特卡洛采样训练数据批次来实现。采样生成过程（反向扩散）也是通过 Langevin 动力学的蒙特卡洛模拟完成的。

- **强化学习中的策略梯度**：REINFORCE 算法使用蒙特卡洛方法估计期望回报的梯度：
  $$\nabla_\theta J(\theta) \approx \frac{1}{T} \sum_{t=1}^T \nabla_\theta \log \pi_\theta(a_t|s_t) \cdot G_t$$
  其中 $G_t$ 是蒙特卡洛采样的累积回报。这是蒙特卡洛思想在序列决策中的直接应用。
