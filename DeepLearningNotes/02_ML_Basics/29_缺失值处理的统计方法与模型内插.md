# 缺失值处理的统计方法与模型内插

## 核心概念
- **缺失值机制**：① **MCAR** (完全随机缺失)——缺失与任何变量无关；② **MAR** (随机缺失)——缺失与其他可观测变量相关；③ **MNAR** (非随机缺失)——缺失与缺失值本身相关。处理策略取决于缺失机制。
- **删除法**：直接删除含缺失值的样本（行删除）或特征（列删除）。简单但有信息损失，仅在 MCAR 且缺失比例小时有效。
- **统计填补**：用均值、中位数、众数等简单统计量填补缺失值。计算快但忽略变量间关系，会扭曲数据分布。
- **模型内插 (Model-based Imputation)**：以缺失变量为目标，用其他完整变量建立预测模型来估计缺失值。常见方法有回归填补、KNN填补、MICE (链式方程多重填补)。
- **多重填补 (Multiple Imputation)**：生成多个填补数据集，在每个数据集上分析，最后合并结果。考虑填补的不确定性，是统计上最严谨的方法。
- **矩阵分解填补**：将数据矩阵分解为低秩矩阵的乘积（如 SVD 填补、软阈值 SVD），适合高维稀疏数据的缺失值恢复。

## 数学推导
**均值填补的局限性**：
假定 $x_{ij}$ 缺失，用该特征已知值的均值 $\bar{x}_j$ 填补。填补后的方差：
$$
\hat{\sigma}_j^2 = \frac{1}{m} \sum_{i=1}^m (x_{ij}^{(imp)} - \bar{x}_j)^2
$$
由于填补值不引入任何变异性，实际方差被低估，相关系数也被扭曲。

**回归填补**：
对于缺失特征 $x_j$，用其他完整特征 $X_{-j}$ 建立回归模型：
$$
x_{ij} = \beta_0 + \sum_{k \neq j} \beta_k x_{ik} + \varepsilon_i
$$
在观察到的样本上估计 $\hat{\beta}$，然后用 $\hat{x}_{ij} = \hat{\beta}_0 + \sum_{k \neq j} \hat{\beta}_k x_{ik}$ 填补缺失值。

**KNN 填补**：
$$
\hat{x}_{ij} = \frac{1}{K} \sum_{k \in \mathcal{N}_K(i)} x_{kj}
$$
其中 $\mathcal{N}_K(i)$ 是与样本 $i$ 最相似的 $K$ 个样本（基于非缺失特征的距离）。

**MICE (链式方程多重填补)**：
对每个缺失变量轮流用条件分布 $p(x_j | x_{-j}, y)$ 抽样填补，迭代直到收敛。重复 $M$ 次得到 $M$ 个完整数据集。最终参数估计：
$$
\hat{\theta} = \frac{1}{M} \sum_{m=1}^M \hat{\theta}_m
$$
总方差 = 组内方差 + 组间方差（Rubin 规则）。

## 直观理解
- **均值填补的"收缩"效应**：均值填补就像知道一个班级的平均分是 75 分后，把缺考学生的分数全都填 75 分。看起来公平，但这样会人为缩小分数分布范围（方差被低估），同时低估缺考分数与其他指标（如出勤率）的相关性。
- **MICE 的"接力赛"**：MICE 像接力赛——先给所有缺失值填个初始值，然后对变量 A 的缺失值建模预测（用其他变量做特征），更新 A；再用更新后的 A 去预测变量 B 的缺失值……迭代多轮，每次模型都越填越准。整个过程重复多次生成多个数据集，最后综合结果——这就是"多重"的含义。
- **矩阵分解填补的"协同过滤"**：Netflix 推荐系统中使用的矩阵分解——将用户-电影评分矩阵分解为用户因子和电影因子——天然适合缺失值填补。因为评分矩阵极度稀疏（绝大多数用户没有评过大多数电影），但分解后的低秩结构能预测缺失评分。

## 代码示例
```python
import numpy as np
import pandas as pd
from sklearn.impute import SimpleImputer, KNNImputer
from sklearn.ensemble import RandomForestRegressor
from sklearn.datasets import load_diabetes
from sklearn.model_selection import cross_val_score

# 模拟缺失数据
data = load_diabetes()
X_full = pd.DataFrame(data.data, columns=data.feature_names)
print(f"原始数据形状: {X_full.shape}")

# 随机引入缺失值
np.random.seed(42)
X_missing = X_full.copy()
mask = np.random.random(X_missing.shape) < 0.15  # 15% 缺失
X_missing[mask] = np.nan
print(f"缺失值数量: {mask.sum()}")

# 1. 均值填补
mean_imputer = SimpleImputer(strategy='mean')
X_mean = mean_imputer.fit_transform(X_missing)
score_mean = cross_val_score(RandomForestRegressor(), X_mean,
                             data.target, cv=3).mean()
print(f"均值填补 CV R^2: {score_mean:.4f}")

# 2. 中位数填补
med_imputer = SimpleImputer(strategy='median')
X_med = med_imputer.fit_transform(X_missing)
score_med = cross_val_score(RandomForestRegressor(), X_med,
                            data.target, cv=3).mean()
print(f"中位数填补 CV R^2: {score_med:.4f}")

# 3. KNN 填补
knn_imputer = KNNImputer(n_neighbors=5)
X_knn = knn_imputer.fit_transform(X_missing)
score_knn = cross_val_score(RandomForestRegressor(), X_knn,
                            data.target, cv=3).mean()
print(f"KNN(k=5)填补 CV R^2: {score_knn:.4f}")

# 4. 对比：原始完整数据的性能（作为上限参考）
score_full = cross_val_score(RandomForestRegressor(), X_full.values,
                             data.target, cv=3).mean()
print(f"完整数据（参考上限）CV R^2: {score_full:.4f}")
```

## 深度学习关联
- **缺失值在深度模型中的处理**：深度学习模型通常不能直接处理 NaN 值。神经网络中常见的做法是用可学习的嵌入向量填补缺失值（将缺失视为一个特殊值，让模型自动学习其表示），或用 Masking 机制让模型跳过缺失位置（如 RNN 中处理变长序列的 padding）。
- **去噪自编码器 (Denoising AE)**：DAE 在训练时人为对输入加噪声（包括随机掩盖部分输入），要求重建原始输入。这与缺失值填补问题高度相似——DAE 学会根据部分观测恢复完整数据，是深度学习中做缺失值填补的有效方法。
- **图神经网络填补图表数据 (GRAPE)**：GRAPE 将缺失值填补建模为图上节点分类问题——样本和特征作为图的节点，已知值为边。用 GNN 在图上进行信息传播来预测缺失值，是目前最先进的表格数据缺失值填补方法之一。
