# 特征选择：过滤法、包裹法与嵌入法

## 核心概念
- **特征选择**：从原始特征集合中选出最优特征子集，降低维度、减少过拟合、提高模型可解释性和训练速度。
- **过滤法 (Filter)**：先对特征进行排序或评分，独立于任何学习算法。常见指标包括相关系数、卡方检验、互信息、方差阈值等。
- **包裹法 (Wrapper)**：将特征选择过程包裹在特定学习算法外，用模型的性能来评估特征子集。常见方法有前向选择、后向消除、递归特征消除 (RFE)。
- **嵌入法 (Embedded)**：在模型训练过程中自动进行特征选择，特征选择和模型训练同时完成。典型代表有 Lasso (L1 正则化)、决策树特征重要性。
- **过滤法 vs 包裹法**：过滤法计算快但忽略特征交互，包裹法考虑特征组合但计算成本高（需训练 $2^d$ 次模型，通常用贪心策略）。
- **维度灾难**：特征数越多，数据在高维空间越稀疏，需要的样本量呈指数增长。特征选择是缓解维度灾难的关键手段。

## 数学推导
**过滤法常见指标**：

1. **皮尔逊相关系数**（特征与目标，适用于回归）：
$$
\rho(X_j, y) = \frac{\text{Cov}(X_j, y)}{\sigma_{X_j} \sigma_y}
$$
选择 $|\rho|$ 大的特征。

2. **互信息**（捕捉非线性关系）：
$$
I(X_j; y) = \sum_{x_j} \sum_{y} p(x_j, y) \log \frac{p(x_j, y)}{p(x_j) p(y)}
$$
互信息越大，特征与目标的相关性越强。

3. **卡方检验**（分类任务中特征与类别的独立性检验）：
$$
\chi^2 = \sum_{i=1}^m \sum_{c=1}^C \frac{(O_{ic} - E_{ic})^2}{E_{ic}}
$$
其中 $O_{ic}$ 是观测频数，$E_{ic}$ 是期望频数（假设独立）。

**包裹法——递归特征消除 (RFE)**：
1. 用全部特征训练模型
2. 按特征重要性排序（如模型系数绝对值）
3. 移除最不重要的特征
4. 在剩余特征上重复 1-3 步，直到达到目标特征数

**嵌入法——Lasso (L1 正则化)**：
$$
J(w) = \sum_{i=1}^m (y_i - w^T x_i)^2 + \lambda \sum_{j=1}^d |w_j|
$$
L1 正则化使部分特征的系数精确为零，实现自动特征选择。$\lambda$ 越大，被置零的系数越多。

## 直观理解
- **过滤法 = 海选简历**：HR 根据硬性指标（学历、工作年限等）快速筛选简历——速度快但可能错过优秀的复合型人才。过滤法只看单维指标，不考虑特征组合的协同效应。
- **包裹法 = 实战考核**：让候选人在模拟工作中实操考核，评估实际能力——准确但耗时昂贵。RFE 像淘汰赛：先所有人上场比赛，表现最差的淘汰，剩下的人再比，直到选出最优团队。
- **嵌入法 = 入职试用**：在正式工作中自然淘汰——Lasso 让不重要的特征系数自动归零，就像员工在试用期被证明不适合然后自动离开。
- **何时用哪种？**：数据量大（百万样本）时用过滤法快速筛选；特征数少（几十个）且资源充足时用包裹法精挑细选；通用场景用嵌入法兼顾效率和效果。

## 代码示例
```python
import numpy as np
from sklearn.datasets import load_digits
from sklearn.feature_selection import (SelectKBest, chi2, f_classif,
                                       mutual_info_classif, RFE)
from sklearn.svm import SVC
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import cross_val_score

# 加载高维数据（64 维）
data = load_digits()
X, y = data.data, data.target
print(f"原始特征数: {X.shape[1]}")

# 过滤法：卡方检验选 top-20 特征
chi2_selector = SelectKBest(chi2, k=20)
X_chi2 = chi2_selector.fit_transform(X, y)
scores_chi2 = cross_val_score(SVC(), X_chi2, y, cv=3)
print(f"过滤法(Chi2, k=20) CV准确率: {scores_chi2.mean():.4f}")

# 过滤法：互信息选 top-20
mi_selector = SelectKBest(mutual_info_classif, k=20)
X_mi = mi_selector.fit_transform(X, y)
scores_mi = cross_val_score(SVC(), X_mi, y, cv=3)
print(f"过滤法(MI, k=20) CV准确率: {scores_mi.mean():.4f}")

# 包裹法：RFE (使用 SVM)
rfe = RFE(estimator=SVC(kernel='linear'), n_features_to_select=20)
X_rfe = rfe.fit_transform(X, y)
scores_rfe = cross_val_score(SVC(), X_rfe, y, cv=3)
print(f"包裹法(RFE, k=20) CV准确率: {scores_rfe.mean():.4f}")

# 嵌入法：Lasso (逻辑回归 L1)
l1_model = LogisticRegression(penalty='l1', C=0.1, solver='liblinear')
l1_model.fit(X, y)
n_selected = np.sum(l1_model.coef_ != 0)
print(f"\n嵌入法(Lasso) 选中的特征数: {n_selected}")
```

## 深度学习关联
- **注意力机制作为嵌入特征选择**：Transformer 的注意力权重可以被视为一种软性特征选择——注意力机制自动给输入的不同部分分配不同的权重，突出重要特征的贡献。类似地，SENet (Squeeze-and-Excitation) 通过学习通道注意力来选择性地强调重要特征通道。
- **稀疏化训练 (Pruning)**：深度网络剪枝相当于在参数空间做"特征选择"——通过将不重要的权重置零来精简网络结构，减少计算量和存储。这与 Lasso 的嵌入法思想一致。
- **特征选择在 AutoML 中**：AutoML 系统中的特征工程组件通常整合过滤法、包裹法和嵌入法，自动评估特征对模型性能的贡献，是神经架构搜索 (NAS) 和数据预处理流水线的重要组成部分。
