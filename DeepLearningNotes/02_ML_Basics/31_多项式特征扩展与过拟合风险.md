# 多项式特征扩展与过拟合风险

## 核心概念
- **多项式特征扩展**：在原始特征基础上添加高次项（$x^2, x^3, \dots$）和特征乘积项（$x_i x_j$），使线性模型能捕捉非线性关系。
- **作用机制**：线性模型 $y = w_0 + w_1 x$ 通过添加 $x^2$ 变为 $y = w_0 + w_1 x + w_2 x^2$，在 $x$ 仍然是线性输入的情况下，输出对 $x$ 的依赖变成二次函数，可以拟合曲线。
- **维度爆炸**：给定 $d$ 个原始特征和最高次 $p$，生成的多项式特征数为 $C(d+p, p)$，随 $d$ 和 $p$ 指数增长。
- **过拟合风险**：高次多项式容易产生振荡的决策边界，在训练数据之外剧烈波动——对边界附近的微小变化高度敏感。
- **Ridge/Lasso 正则化**：对多项式特征使用正则化（$L_2$ 或 $L_1$）可以控制系数的大小或稀疏性，是缓解过拟合的关键手段。
- **与核方法的联系**：多项式核 $K(x_i, x_j) = (x_i^T x_j + c)^d$ 隐式地计算了高维多项式特征空间中的内积，但无需显式构造特征。

## 数学推导
**多项式特征构造**：
对于 $d$ 个原始特征 $x_1, x_2, \dots, x_d$，$p$ 次多项式扩展包含所有形如 $\prod_{i=1}^d x_i^{e_i}$ 的项，其中 $\sum_{i=1}^d e_i \leq p$。

例如，$d=2, p=2$：
$$
\phi(x) = [1, x_1, x_2, x_1^2, x_1 x_2, x_2^2]
$$

**特征数量**：
$$
N = \binom{d + p}{p} = \frac{(d+p)!}{d! \, p!}
$$

当 $d=10, p=5$ 时，$N = \binom{15}{5} = 3003$，比原始特征增长了 300 倍。

**多项式回归的模型复杂度**：
$$
y = \sum_{j=0}^p w_j x^j + \varepsilon
$$

当 $p$ 增大时，模型有更多自由度来拟合训练数据中的每一个点（包括噪声），导致：
- 训练误差趋近于 0
- 测试误差在某个 $p$ 之后急剧上升

**Runge 现象**：在等距点上用高次多项式插值时，区间端点附近会出现剧烈振荡。这是高次多项式过拟合的经典例子。

**正则化后的 Ridge 回归**：
$$
J(w) = \sum_{i=1}^m (y_i - \phi(x_i)^T w)^2 + \lambda \|w\|^2
$$
$\lambda$ 越大，权重系数被压缩得越小，决策边界越平滑。

## 直观理解
- **多项式 = 给线性模型"开挂"**：线性模型只能画直线（一维）或平面（二维）。多项式扩展让线性模型能画曲线、波浪线和复杂形状的等高线。但权力越大，风险越大——模型可能"画蛇添足"在噪声点上剧烈扭曲。
- **过拟合的"蛇形"决策边界**：想象用越来越多次数的多项式拟合几个散点——一次（直线）偏差大；三次（S 形）大致贴合趋势；十次（剧烈蛇形）精确穿过每个点，但两个点之间震荡剧烈。这种"蛇形"在未见数据上通常表现很差。
- **正则化的"牵制"作用**：正则化相当于给多项式系数套上"缰绳"——系数不能太大，防止模型"放飞自我"去精确拟合每个噪声点。合适的 $\lambda$ 让曲线既能拟合趋势又不至于剧烈震荡。

## 代码示例
```python
import numpy as np
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import Ridge, LinearRegression
from sklearn.pipeline import Pipeline
from sklearn.model_selection import train_test_split

# 生成非线性数据
np.random.seed(42)
X = np.linspace(0, 6, 100).reshape(-1, 1)
y = np.sin(X).ravel() + np.random.randn(100) * 0.15

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)

# 不同多项式次数的对比
for degree in [1, 4, 15]:
    # 无正则化
    pipe = Pipeline([
        ('poly', PolynomialFeatures(degree=degree)),
        ('lr', LinearRegression())
    ])
    pipe.fit(X_train, y_train)
    print(f"次数={degree:2d} (无正则): "
          f"训练 Error={np.mean((y_train - pipe.predict(X_train))**2):.4f}, "
          f"测试 Error={np.mean((y_test - pipe.predict(X_test))**2):.4f}")

# Ridge 正则化的效果
for alpha in [0, 0.01, 1.0, 100.0]:
    pipe_ridge = Pipeline([
        ('poly', PolynomialFeatures(degree=15)),
        ('ridge', Ridge(alpha=alpha))
    ])
    pipe_ridge.fit(X_train, y_train)
    print(f"次数=15 ridge(alpha={alpha:5.1f}): "
          f"测试 Error={np.mean((y_test - pipe_ridge.predict(X_test))**2):.4f}")
```

## 深度学习关联
- **神经网络的通用近似定理**：深度神经网络不需要显式构造多项式特征——通过层次化的非线性激活函数组合，网络可以自动学习特征的高阶交互和非线性变换。这比手动构造多项式特征更灵活、更高效。
- **傅里叶特征与位置编码**：在 NeRF (Neural Radiance Fields) 和 Transformer 的位置编码中，会显式地将输入坐标映射到高频傅里叶特征（不同频率的正余弦函数），这与多项式特征扩展的思路类似——通过特征变换让模型能捕捉高频细节。
- **网络的隐式正则化**：深度网络训练中的隐式正则化（如 SGD 的隐式偏差、参数范数约束）起到类似 Ridge 正则化的作用，防止模型在高维特征空间过拟合。理解多项式特征的正则化有助于理解深度网络的泛化。
