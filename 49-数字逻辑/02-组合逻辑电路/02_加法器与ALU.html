加法器与ALU

一、半加器（Half Adder）

半加器是最简单的加法电路，将两个一位二进制数相加，产生和（Sum）与进位（Carry）。

1.1 真值表

ABS（和）C（进位）
0000
0110
1010
1101

1.2 逻辑表达式

S = A ⊕ B（异或）
C = A · B（与）
局限性：半加器无法处理来自低位的进位输入，只能用于最低位的加法。

二、全加器（Full Adder）

全加器能处理三个输入：两个加数 A、B 和来自低位的进位 Cin。

2.1 真值表

ABCinS（和）Cout（进位）
00000
00110
01010
01101
10010
10101
11001
11111

2.2 逻辑表达式

S = A ⊕ B ⊕ Cin
Cout = A·B + (A⊕B)·Cin
也可写为：Cout = A·B + A·Cin + B·Cin

2.3 全加器的实现

  • 用两个半加器和一个或门实现：S = HA1.S ⊕ Cin，Cout = HA1.C + HA2.C
  • 用与非门实现（考试可能涉及）

三、行波进位加法器（Ripple Carry Adder）

将多个全加器级联，低位的Cout接高位的Cin，实现多位加法。

3.1 4位行波进位加法器结构

A₃B₃ ── [FA₃] ── S₃
       ↑ C₃
A₂B₂ ── [FA₂] ── S₂
       ↑ C₂
A₁B₁ ── [FA₁] ── S₁
       ↑ C₁
A₀B₀ ── [FA₀] ── S₀
       ↑ C₀=0

3.2 特点分析

  • 优点：结构简单，易于实现
  • 缺点：进位信号逐级传递，速度慢
  • 延迟：n位加法器需要 n × tFA 的时间
关键问题：进位传播延迟是限制加法器速度的主要因素。对于32位加法器，最坏情况进位需要传播32级全加器的延迟。

四、超前进位加法器（Carry Lookahead Adder / CLA）

通过提前计算所有进位信号，消除行波进位的串行延迟。

4.1 基本思想

定义两个辅助变量：

  • 生成（Generate）：Gi = Ai · Bi（本位能自己产生进位）
  • 传播（Propagate）：Pi = Ai ⊕ Bi（本位能传播来自低位的进位）

4.2 超前进位公式

C₀ = 输入进位（通常为0）
C₁ = G₀ + P₀·C₀
C₂ = G₁ + P₁·G₀ + P₁·P₀·C₀
C₃ = G₂ + P₂·G₁ + P₂·P₁·G₀ + P₂·P₁·P₀·C₀
C₄ = G₃ + P₃·G₂ + P₃·P₂·G₁ + P₃·P₂·P₁·G₀ + P₃·P₂·P₁·P₀·C₀

4.3 特点

  • 优点：所有进位并行计算，速度快
  • 缺点：门数量和连线随位数指数增长
  • 实际应用：通常用分组CLA（如4位一组），在速度和复杂度之间折中

五、算术逻辑单元（ALU）

ALU是CPU的核心部件，能执行算术运算（加、减）和逻辑运算（与、或、非、异或）。

5.1 ALU的基本结构

A(n位) ──┐
B(n位) ──┤ ALU ├── F(n位)
操作码 ──┘    ├── 标志位
            └── (ZF,CF,OF,SF)

5.2 74181 ALU

经典的4位ALU芯片，能执行16种算术和16种逻辑运算。

S₃S₂S₁S₀M=1（逻辑）M=0（算术）
0000F = A̅F = A
0001F = A̅+B̅ = (AB)̅F = A+B
0010F = A̅BF = A+B̅
0011F = 0F = 减1（全1）
0110F = A⊕BF = A-B
1001F = A⊙BF = A+B+Cin
1100F = AF = A加A
1111F = 1F = A加A加1

5.3 标志位（Flags）

标志位名称含义
ZF零标志结果为0时置1
CF进位标志无符号运算溢出时置1
OF溢出标志有符号运算溢出时置1
SF符号标志结果为负时置1（等于最高位）

六、减法的实现

利用补码运算，减法可以转换为加法：A - B = A + (-B) = A + B̅ + 1

6.1 加减法统一电路

  • 加一个控制信号 M：M=0做加法，M=1做减法
  • 减法时：B通过异或门（B⊕1=B̅），进位置1
  • 加法时：B直通（B⊕0=B），进位置0
溢出判断：有符号数溢出 = Cn ⊕ Cn-1（最高位进位与次高位进位异或）

七、知识要点总结

  1. 半加器：S=A⊕B，C=AB（不考虑进位输入）
  2. 全加器：S=A⊕B⊕Cin，Cout=AB+(A⊕B)Cin
  3. 行波进位加法器：简单但慢，延迟与位数成正比
  4. 超前进位加法器：快但复杂，利用G和P并行计算进位
  5. ALU是CPU核心，执行算术和逻辑运算
  6. 减法通过补码转化为加法实现
