编码器与译码器

一、编码器（Encoder）

编码器的功能是将 2n 个输入信号编码为 n 位二进制代码输出。它是译码器的逆过程。

1.1 普通编码器

任意时刻只有一个输入有效，输出对应的二进制编码。

8线-3线编码器（74148简化版）

I₇I₆I₅I₄I₃I₂I₁I₀Y₂Y₁Y₀
00000001000
0000001×001

00001×××011
0001××××100

01××××××110
1×××××××111

1.2 优先编码器（Priority Encoder）

允许多个输入同时有效，按优先级对最高优先级的输入进行编码。

74148：8线-3线优先编码器，I₇优先级最高，I₀优先级最低。
关键信号：EI（使能输入）、EO（使能输出）、GS（组信号，表示有编码请求）
级联：通过EI/EO实现多片级联，扩展编码范围

二、译码器（Decoder）

译码器的功能是将 n 位二进制输入译码为 2n 个输出信号中的一个有效输出。它是编码器的逆过程。

2.1 2线-4线译码器

A₁ ──┐
A₀ ──┤ 2:4 ├── Y₀,Y₁,Y₂,Y₃
EN ──┘ Decoder ┘
ENA₁A₀Y₃Y₂Y₁Y₀
0××0000
1000001
1010010
1100100
1111000
输出表达式（低电平有效）：
Y₀ = (A₁̅·A₀̅)̅
Y₁ = (A₁̅·A₀)̅
Y₂ = (A₁·A₀̅)̅
Y₃ = (A₁·A₀)̅

2.2 3线-8线译码器（74138）

最常用的译码器芯片，3个输入，8个输出（低电平有效）。

使能输入输出

1 0 000001111111
1 0 000110111111
1 0 011111111110
0 × ××全1（不译码）

三、数据选择器（Multiplexer / MUX）

数据选择器又称多路复用器，从多个输入中选择一个送到输出端。可以看作一个"多位开关"。

3.1 4选1数据选择器

S₁S₀输出Y
00D₀
01D₁
10D₂
11D₃
Y = S₁̅S₀̅D₀ + S₁̅S₀D₁ + S₁S₀̅D₂ + S₁S₀D₃

3.2 74151（8选1 MUX）

  • 8个数据输入 D₀~D₇
  • 3个选择输入 S₂S₁S₀
  • 使能端 E̅（低电平有效）
  • 互补输出 Y 和 W（W = Y̅）

3.3 用MUX实现组合逻辑函数

方法：用选择输入接函数变量，数据输入接常量0或1，即可实现任意组合逻辑。
示例：用8选1 MUX实现 F(A,B,C) = Σm(1,3,5,7)
将A,B,C分别接 S₂,S₁,S₀，D₁=D₃=D₅=D₇=1，其余D=0。

四、数据分配器（Demultiplexer / DEMUX）

数据分配器是数据选择器的逆过程，将一个输入数据分配到多个输出中的某一个。

4.1 1路-4路数据分配器

S₁S₀Y₃Y₂Y₁Y₀
00000D
0100D0
100D00
11D000
重要关系：n线-2ⁿ线译码器加上使能端后，可以作为数据分配器使用。例如74138译码器用数据端接G₁，即可作为1路-8路DEMUX。

五、知识要点总结

  1. 编码器：2ⁿ输入 → n输出，优先编码器允许多输入同时有效
  2. 译码器：n输入 → 2ⁿ输出，74138是3-8译码器
  3. 数据选择器（MUX）：多输入选一输出，可用于实现组合逻辑
  4. 数据分配器（DEMUX）：一输入分配到多输出之一
  5. 译码器 + 使能端 = 数据分配器
  6. MUX和DEMUX互为逆操作
