触发器与寄存器

一、SR触发器（SR Flip-Flop）

SR触发器是最基本的触发器，有两个输入S（Set）和R（Reset），具有存储一位二进制信息的能力。

1.1 基本SR锁存器（与非门构成）

特征方程：Qn+1 = S + R̅·Qn
约束条件：S·R = 0（S和R不能同时为1）

1.2 真值表（特性表）

SRQn+1功能
00Qn保持（不变）
010置0（Reset）
101置1（Set）
11不确定禁止状态
注意：用或非门构成的SR触发器，有效电平是高电平；用与非门构成的，有效电平是低电平（S̅和R̅）。

二、D触发器（D Flip-Flop）

D触发器通过将S和R反相连接，消除了SR触发器的不确定状态问题。

2.1 特征方程

Qn+1 = D
（在时钟有效沿到来时，输出Q等于输入D）

2.2 特性表

DQn+1功能
00置0
11置1

2.3 应用

  • 数据锁存：一个D触发器可以存储1位数据
  • 移位寄存器：多个D触发器级联
  • 计数器：分频电路
  • 同步电路设计的基础元件

三、JK触发器（JK Flip-Flop）

JK触发器是功能最全的触发器，解决了SR触发器的不确定状态问题。当J=K=1时，输出翻转。

3.1 特征方程

Qn+1 = J·Q̅n + K̅·Qn

3.2 特性表

JKQn+1功能
00Qn保持
010置0
101置1
11Q̅n翻转（Toggle）

3.3 JK触发器的特殊性质

  • J=K=0时保持，不会出现不确定状态
  • J=K=1时翻转，可用于构建计数器
  • 可以实现SR触发器和D触发器的功能

四、T触发器（T Flip-Flop）

T触发器是JK触发器的特例，将J和K连接在一起作为T输入。

4.1 特征方程

Qn+1 = T·Q̅n + T̅·Qn = T ⊕ Qn

4.2 特性表

TQn+1功能
0Qn保持
1Q̅n翻转

五、触发方式

5.1 触发方式对比

触发方式符号特点代表
电平触发无特殊标记时钟高/低电平期间响应输入基本SR锁存器
边沿触发（上升沿）CLK端有 ">" 标记仅在CLK上升沿采样上升沿D触发器
边沿触发（下降沿）CLK端有 ">○" 标记仅在CLK下降沿采样下降沿JK触发器
主从触发有主从结构标记分两次触发，一次变化主从JK触发器
重要：边沿触发的触发器在时钟沿到来时采样输入，之后即使输入变化也不影响输出，避免了空翻现象。

六、寄存器（Register）

寄存器由一组触发器组成，用于存储多位二进制数据。

6.1 基本寄存器

  • n个D触发器并行连接，共用同一个时钟
  • 在CLK上升沿同时将n位数据存入
  • 例如：74273是8位D触发器（带清零）

6.2 移位寄存器（Shift Register）

数据在时钟控制下依次移动，用于串-并转换、数据移位等。

四种移位类型

类型数据方向应用
左移寄存器低位→高位乘以2n
右移寄存器高位→低位除以2n
双向移位寄存器可左可右灵活数据处理
环形移位寄存器首尾相连循环循环控制

6.3 74194（4位双向通用移位寄存器）

  • 功能选择 S₁S₀：00保持，01右移，10左移，11并行置数
  • 串行输入 DSR（右移输入）、DSL（左移输入）
  • 并行输入 D₀~D₃
  • 异步清零 C̅R

七、知识要点总结

  1. SR触发器：有约束条件S·R=0，禁止同时为1
  2. D触发器：Qn+1=D，无不确定状态
  3. JK触发器：功能最全，J=K=1时翻转
  4. T触发器：T=1翻转，T=0保持
  5. 边沿触发避免空翻，是现代电路设计的主流
  6. 寄存器由触发器组成，移位寄存器可实现串并转换
