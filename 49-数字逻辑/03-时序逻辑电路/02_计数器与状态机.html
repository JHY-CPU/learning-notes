计数器与状态机

一、同步计数器（Synchronous Counter）

所有触发器共用同一个时钟信号，状态变化同时发生，速度快且无毛刺。

1.1 同步二进制计数器设计步骤

  1. 确定计数模值和状态数
  2. 画出状态转换图和状态转换表
  3. 选择触发器类型（JK或D）
  4. 用卡诺图化简得到激励方程
  5. 画出逻辑电路图

1.2 4位同步二进制加法计数器

CLKQ₃Q₂Q₁Q₀十进制
000000
100011
200102
..................
15111115
1600000（循环）

1.3 激励方程（用JK触发器）

J₀ = K₀ = 1（每时钟翻转）
J₁ = K₁ = Q₀
J₂ = K₂ = Q₁·Q₀
J₃ = K₃ = Q₂·Q₁·Q₀
规律：JK触发器实现二进制计数器，Ji = Ki = Qi-1·Qi-2···Q0

二、异步计数器（Asynchronous / Ripple Counter）

只有最低位触发器接收外部时钟，高位的时钟来自低位的输出。进位信号像波纹一样逐级传播。

2.1 特点

  • 优点：电路简单，连线少
  • 缺点：逐级延迟累积，速度慢，有毛刺
  • 需要使用下降沿触发的触发器（或上升沿但Q̅输出）

2.2 4位异步二进制加法计数器

CLK ── [FF₀] ── Q₀ ── [FF₁] ── Q₁ ── [FF₂] ── Q₂ ── [FF₃] ── Q₃
       J₀=K₀=1      J₁=K₁=1      J₂=K₂=1      J₃=K₃=1

2.3 任意模值计数器

模N计数器有N个有效状态。实现方法：

  • 反馈置零法：计数到N时，通过门电路产生清零信号
  • 反馈置数法：计数到某值时，预置初始值
  • 例：模6计数器：000→001→010→011→100→101→(清零)→000

三、74160系列集成计数器

3.1 芯片功能对比

型号计数方式模值特点
74160同步模10（BCD）同步置数、异步清零
74161同步模16（4位二进制）同步置数、异步清零
74162同步模10（BCD）同步置数、同步清零
74163同步模16（4位二进制）同步置数、同步清零
74190同步模10可逆计数（加/减）
74191同步模16可逆计数（加/减）

3.2 关键控制信号

  • C̅R：异步清零（低有效），不受CLK控制
  • LD̅：同步预置（低有效），在CLK上升沿加载
  • EP, ET：使能端，控制计数
  • RCO：进位输出，RCO = ET·Q₃·Q₂·Q₁·Q₀（74161）

四、有限状态机（FSM）

有限状态机是时序逻辑电路的抽象模型，由状态寄存器和组合逻辑组成。

4.1 Moore型与Mealy型

类型输出依赖特点
Moore型仅依赖当前状态输出稳定，无毛刺；状态数可能更多
Mealy型依赖当前状态和输入状态数更少；输出可能有毛刺

4.2 Moore型框图

输入 ──> ┌──────────┐ 次态 ──> ┌──────────┐ 当前状态
         │ 组合逻辑 │────────│ 状态寄存 │──────> 输出
         └──────────┘       │ (触发器) │      (仅依赖状态)
                           └──────────┘
CLK ──────────────────> CLK

4.3 Mealy型框图

输入 ──┬──> ┌──────────┐ 次态 ──> ┌──────────┐ 当前状态
       │    │ 组合逻辑 │────────│ 状态寄存 │──────┐
       │    └──────────┘       │ (触发器) │     │
       │          ↑             └──────────┘     │
       └──────────────────────────────────────────────┴──> 输出
                                                       (依赖状态+输入)

五、状态图与状态表

5.1 状态图的表示

  • 圆圈表示状态，圈内写状态名/编码
  • 箭头表示状态转换
  • Moore：箭头旁标注"输入/输出"或"输入"，输出标在状态圈旁
  • Mealy：箭头旁标注"输入/输出"

5.2 示例：序列检测器（检测"1101"）

S₀ ──1/0──> S₁ ──1/0──> S₂ ──0/0──> S₃ ──1/1──> S₀
当前状态输入=0输入=1输出（Moore）
S₀（初始）S₀/0S₁/00
S₁（收到1）S₀/0S₂/00
S₂（收到11）S₃/0S₂/00
S₃（收到110）S₀/0S₀/10
FSM设计五步骤：
1. 根据题意画状态转换图
2. 状态化简（等价状态合并）
3. 状态编码（分配二进制码）
4. 求激励方程和输出方程
5. 画电路图并检查自启动

六、知识要点总结

  1. 同步计数器：所有触发器同时触发，速度快
  2. 异步计数器：逐级延迟，电路简单
  3. 74160系列：同步计数器芯片，注意异步/同步清零的区别
  4. 模N计数器：反馈置零法和反馈置数法
  5. Moore型：输出仅依赖状态；Mealy型：输出依赖状态和输入
  6. FSM设计：画图→化简→编码→求方程→画电路
