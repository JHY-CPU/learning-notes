布尔化简与卡诺图

一、布尔表达式的标准形式

1.1 最小项（Minterm）与最大项（Maxterm）

概念定义记号特点
最小项 n个变量的乘积项，每个变量以原变量或反变量出现一次 mi 使最小项为1的变量组合只有一个
最大项 n个变量的和项，每个变量以原变量或反变量出现一次 Mi 使最大项为0的变量组合只有一个

1.2 SOP与POS形式

形式全称结构与最小项/最大项的关系
SOP Sum of Products（与或式） 若干乘积项相或 标准SOP由最小项之和构成
POS Product of Sums（或与式） 若干和项相与 标准POS由最大项之积构成
SOP示例：F = A̅B + AB̅ + ABC
POS示例：F = (A+B)·(A̅+C)·(B+C̅)

二、代数化简法

利用布尔代数的基本定律对逻辑表达式进行化简，常用方法包括：

2.1 常用化简技巧

  1. 并项法：AB + AB̅ = A(B+B̅) = A
  2. 吸收法：A + AB = A(1+B) = A
  3. 消去法：A + A̅B = A + B
  4. 配项法：添加多余项进行化简

2.2 化简示例

原式：F = ABC + ABC̅ + AB̅C + A̅BC
= AB(C+C̅) + AB̅C + A̅BC ——并项
= AB + AB̅C + A̅BC
= A(B+B̅C) + A̅BC
= A(B+C) + A̅BC ——消去
= AB + AC + A̅BC
= AB + C(A+A̅B) ——消去
= AB + C(A+B)
= AB + AC + BC
局限性：代数化简法依赖经验和技巧，难以判断是否已化到最简。卡诺图法可以系统地找到最简表达式。

三、卡诺图（Karnaugh Map）

卡诺图是一种图形化的化简工具，将真值表按格雷码排列，通过圈出相邻的"1"格来化简表达式。

3.1 卡诺图的结构

  • n个变量需要 2n 个格子
  • 变量按格雷码排列（相邻格只有一个变量不同）
  • 相邻包括：上下相邻、左右相邻、首尾相邻

3.2 二变量卡诺图

A\B01
0A̅B̅ (m₀)A̅B (m₁)
1AB̅ (m₂)AB (m₃)

3.3 三变量卡诺图

A\BC00011110
0m₀m₁m₃m₂
1m₄m₅m₇m₆

3.4 四变量卡诺图

AB\CD00011110
00m₀m₁m₃m₂
01m₄m₅m₇m₆
11m₁₂m₁₃m₁₅m₁₄
10m₈m₉m₁₁m₁₀

四、卡诺图化简规则

4.1 圈"1"的规则

  1. 圈必须是矩形（包括正方形），大小为 2k（k=0,1,2,...）
  2. 圈越大越好，但圈内必须全是"1"
  3. 每个"1"至少被圈一次
  4. 圈的总数越少越好
  5. 可以重复圈（同一个"1"被多个圈包含）
  6. 卡诺图的上下、左右边缘是相邻的（首尾相邻）

4.2 圈的大小与化简结果

圈的大小包含格数化简后项数消去变量数
1格1n个文字0
2格2n-1个文字1
4格4n-2个文字2
8格8n-3个文字3

4.3 化简示例

F(A,B,C,D) = Σm(0,2,5,7,8,10,13,15)

四变量卡诺图填入1：
AB\CD | 00 01 11 10
00 | 1 0 0 1
01 | 0 1 1 0
11 | 0 1 1 0
10 | 1 0 0 1

圈1：m₀,m₂,m₈,m₁₀ → B̅D̅
圈2：m₅,m₇,m₁₃,m₁₅ → BD

最简表达式：F = B̅D̅ + BD = B⊙D（同或）

五、无关项（Don't Care）

某些输入组合在实际中不会出现，或者输出可以是任意值，这些项称为无关项，在卡诺图中标记为"×"或"d"。

5.1 无关项的利用

  • 无关项可以当作"1"来圈，也可以当作"0"忽略
  • 目标是让圈更大、圈数更少
  • 不能单独圈无关项

5.2 示例

F(A,B,C) = Σm(1,3,5) + d(6,7)
即 m₁,m₃,m₅ 为1，m₆,m₇ 为无关项

A\BC | 00 01 11 10
0 | 0 1 1 0
1 | 0 1 × ×

圈1：m₁,m₃,m₅,m₇ → C

最简表达式：F = C

六、知识要点总结

  1. 标准形式：SOP（最小项之和）和 POS（最大项之积）
  2. 代数化简法：并项法、吸收法、消去法、配项法
  3. 卡诺图是系统化简逻辑函数的图形方法
  4. 圈"1"规则：2k大小、全1、越大越好、越少越好
  5. 首尾相邻：卡诺图的边缘是连通的
  6. 无关项可当作0或1使用，以获得更简表达式
