布尔代数与基本逻辑门

一、布尔代数基础

布尔代数（Boolean Algebra）是由英国数学家乔治·布尔（George Boole）于1854年提出的，是数字逻辑电路设计的数学基础。

1.1 基本概念

  • 布尔变量：只能取 0 或 1 两个值的变量
  • 布尔常量：0（假/False）和 1（真/True）
  • 布尔运算：与（AND）、或（OR）、非（NOT）

1.2 基本运算

运算符号表达式含义
与 AND· 或 ∧Y = A · B全1出1，有0出0
或 OR+ 或 ∨Y = A + B有1出1，全0出0
非 NOT¬ 或 ‾Y = A̅取反，0变1，1变0

二、基本逻辑门

2.1 六种基本逻辑门

AND 与门

Y = A · B

OR 或门

Y = A + B

NOT 非门

Y = A̅

NAND 与非门

Y = (A·B)̅

NOR 或非门

Y = (A+B)̅

XOR 异或门

Y = A⊕B

三、真值表

真值表是列出所有输入组合及其对应输出的表格，是分析和设计逻辑电路的基本工具。

四、德摩根定律（De Morgan's Laws）

德摩根定律是布尔代数中最重要的定理之一，用于逻辑表达式的变换和化简。

第一定律：(A · B)̅ = A̅ + B̅
第二定律：(A + B)̅ = A̅ · B̅

4.1 德摩根定律的直观理解

第二定律或的非 = 非的与括号内加变乘，每个变量取反

五、布尔代数基本定律

5.1 基本定律

定律名称与运算形式或运算形式
交换律A·B = B·AA+B = B+A
结合律(A·B)·C = A·(B·C)(A+B)+C = A+(B+C)
分配律A·(B+C) = A·B+A·CA+(B·C) = (A+B)·(A+C)
吸收律A·(A+B) = AA+A·B = A
同一律A·1 = AA+0 = A
互补律A·A̅ = 0A+A̅ = 1
幂等律A·A = AA+A = A
对合律A̅̅ = A（双重否定）
零一律A·0 = 0A+1 = 1
考试重点：分配律的两种形式都需要掌握，特别是 A+(B·C) = (A+B)·(A+C) 这个"或对与的分配律"容易被忽略。

六、通用逻辑门

NAND门和NOR门被称为"通用门"（Universal Gates），因为任何逻辑函数都可以仅用NAND门或仅用NOR门来实现。

6.1 用NAND实现其他门

  • NOT：将NAND的两个输入接在一起 → Y = (A·A)̅ = A̅
  • AND：NAND后面接一个NOT → Y = (A·B)̅̅ = A·B
  • OR：对A和B分别取NAND（做NOT），再NAND → Y = (A̅·B̅)̅ = A+B

6.2 用NOR实现其他门

  • NOT：将NOR的两个输入接在一起 → Y = (A+A)̅ = A̅
  • OR：NOR后面接一个NOT → Y = (A+B)̅̅ = A+B
  • AND：对A和B分别取NOR（做NOT），再NOR → Y = (A̅+B̅)̅ = A·B

七、知识要点总结

  1. 布尔代数只有0和1两种取值，对应数字电路中的高/低电平
  2. 六种基本逻辑门：AND、OR、NOT、NAND、NOR、XOR
  3. 真值表是分析逻辑电路的基础工具
  4. 德摩根定律：与非互变，变量取反
  5. NAND和NOR是通用门，可以实现任何逻辑功能
  6. 布尔代数基本定律是逻辑化简的基础
