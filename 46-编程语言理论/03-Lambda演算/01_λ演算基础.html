Lambda演算基础 (λ-Calculus)

一、Lambda演算概述

Lambda演算是阿隆佐·邱奇 (Alonzo Church) 在1930年代创造的形式系统，是函数式编程语言的理论基础。

  • 是最小的通用编程语言——仅用函数抽象和函数应用即可表达所有可计算函数
  • 与图灵机等价（Church-Turing论题）
  • 现代函数式语言 (Haskell, ML, Lisp) 的理论根基

三种基本构造

构造语法含义
变量x一个标识符
λ抽象λx. M定义函数（参数x，体M）
函数应用M N将函数M应用于参数N

除此之外没有别的——没有数字、布尔值、if、循环等，一切都从函数推导出来。

二、语法约定

  • λx. M — λ抽象：创建一个参数为 x、函数体为 M 的匿名函数
  • M N — 应用：将 M 应用于 N（左结合：M N P = (M N) P）
  • λ的绑定范围延伸到尽可能右边：λx. M N = λx. (M N)
  • 多参数用柯里化：λx. λy. M 等价于数学中的 f(x, y)

自由变量与约束变量

  • 约束变量：在 λx 中的 x 被约束
  • 自由变量：不在任何 λ 抽影中的变量

在 λx. x y 中：x 是约束变量，y 是自由变量

在 (λx. x) y 中：第一个 x 约束，y 自由

FV(λx. M) = FV(M) \ {x}

三、λ归约规则

α-归约（变量重命名）

λx. M →α λy. M[y/x] — 重命名约束变量，避免名称冲突

λx. x = λy. y = λz. z（都是恒等函数）

β-归约（函数求值，最重要）

(λx. M) N →β M[N/x] — 将实参 N 替换到函数体 M 中的 x

β-归约示例

(λx. x + 1) 5

  →β x+1 中 x 替换为 5

5 + 1

  = 6

η-归约（外延等价）

λx. M x →η M（当 x 不在 M 中自由出现时）

含义：如果两个函数对所有输入给出相同输出，则它们等价。

λx. (+ 1) x →η + 1（η-归约）

四、Church-Rosser定理与求值策略

Church-Rosser定理（合流性）：如果 M →* N₁ 且 M →* N₂，则存在 N 使得 N₁ →* N 且 N₂ →* N。

即：归约的顺序不重要，最终结果（如果存在）是唯一的。

求值策略

策略何时归约函数体传值？语言
正则序 (Normal)先展开函数体传名 (Call-by-name)Haskell (惰性)
应用序 (Applicative)先计算参数值传值 (Call-by-value)ML, Scheme

区别示例

(λx. λy. x) ((λx. x x) (λx. x x)) 42

-- 正则序（先展开）：
-- 先用函数体 λy. x，x = 未归约的参数
-- 展开后 λy. ((λx. x x) (λx. x x))，应用到42
-- 结果：42（第二项永远不会被求值）

-- 应用序（先算参数）：
-- 先算 ((λx. x x) (λx. x x)) → (λx. x x)(λx. x x) → ... 死循环！
-- 永远无法得到结果

五、Church编码：用纯函数编码一切

在纯λ演算中，没有数字、布尔值等原语——全部用函数编码：

Church布尔值

true  = λt. λf. t        -- 选择第一个参数
false = λt. λf. f        -- 选择第二个参数
and   = λp. λq. p q p    -- and True False = False
or    = λp. λq. p p q    -- or False True = True
not   = λp. p false true  -- not True = False

Church自然数（Church数）

0 = λf. λx. x              -- 应用f零次
1 = λf. λx. f x            -- 应用f一次
2 = λf. λx. f (f x)        -- 应用f两次
3 = λf. λx. f (f (f x))    -- 应用f三次

-- 后继函数
succ = λn. λf. λx. f (n f x)

-- 加法
add = λm. λn. λf. λx. m f (n f x)

-- 乘法
mult = λm. λn. λf. m (n f)

-- 零检测
iszero = λn. n (λx. false) true

Church对

pair = λx. λy. λf. f x y    -- 构造对
fst  = λp. p (λx. λy. x)    -- 取第一个
snd  = λp. p (λx. λy. y)    -- 取第二个
