# 07 - 条件编译

## 条件编译概述

条件编译是预处理器最重要的功能之一，它允许程序员根据条件决定哪些代码段被编译、哪些被跳过。条件编译在编译之前进行评估，条件必须是**预处理器常量表达式**（编译时可确定的值）。

## 基本条件编译指令

### #if / #endif

```c
#if 条件表达式
    // 条件为真时编译的代码
#endif
```

```c
#define DEBUG 1

#if DEBUG
    printf("调试模式已启用\n");
#endif
```

### #if / #else / #endif

```c
#if 条件
    // 条件为真
#else
    // 条件为假
#endif
```

```c
#define RELEASE 1

#if RELEASE
    #define LOG(msg)  // 发布版本不输出日志
#else
    #define LOG(msg) printf("[LOG] %s\n", msg)
#endif
```

### #if / #elif / #else / #endif

```c
#define PLATFORM 2

#if PLATFORM == 1
    printf("Windows 平台\n");
#elif PLATFORM == 2
    printf("Linux 平台\n");
#elif PLATFORM == 3
    printf("macOS 平台\n");
#else
    printf("未知平台\n");
#endif
```

## #ifdef 和 #ifndef

### #ifdef — 如果已定义

```c
#ifdef 标识符
    // 如果标识符已定义，编译此代码
#endif
```

```c
#define FEATURE_ENABLED

#ifdef FEATURE_ENABLED
    printf("功能已启用\n");
#endif
```

### #ifndef — 如果未定义

```c
#ifndef 标识符
    // 如果标识符未定义，编译此代码
#endif
```

```c
// 常见用法：头文件守卫
#ifndef MYHEADER_H
#define MYHEADER_H

// 头文件内容...

#endif /* MYHEADER_H */
```

### #ifdef 等价于 #if defined()

```c
// 这两种写法完全等价
#ifdef DEBUG
    // ...
#endif

#if defined(DEBUG)
    // ...
#endif
```

## defined() 运算符

`defined` 是一个特殊的预处理器运算符，用于检查宏是否已定义：

```c
// 单个检查
#if defined(WIN32)
    // Windows 代码
#endif

// 组合条件
#if defined(WIN32) || defined(_WIN32) || defined(__WIN32__)
    #define IS_WINDOWS 1
#endif

// 取反检查
#if !defined(NDEBUG)
    // 如果没有定义 NDEBUG（调试模式）
    #define DEBUG 1
#endif
```

## 条件表达式

### 支持的运算

预处理器条件表达式支持：

```c
// 算术运算
#if 2 + 3 * 4 == 14
    // true
#endif

// 比较运算
#if MAX_SIZE > 100 && MAX_SIZE < 1000
    // ...
#endif

// 逻辑运算
#if defined(DEBUG) || defined(VERBOSE)
    // ...
#endif

// 三元运算符
#define BUFFER_SIZE (DEBUG ? 64 : 1024)
```

### 未定义宏的行为

在条件表达式中，未定义的宏会被替换为 0：

```c
// UNDEFINED_MACRO 未定义
#if UNDEFINED_MACRO
    printf("这不会被执行\n");  // 因为 UNDEFINED_MACRO 被替换为 0
#endif

#if UNDEFINED_MACRO == 0
    printf("这会被执行\n");    // 0 == 0 为真
#endif
```

## 嵌套条件编译

```c
#if defined(UNIX)
    #include <unistd.h>
    #if defined(USE_THREADS)
        #include <pthread.h>
    #endif
#elif defined(WINDOWS)
    #include <windows.h>
    #if defined(USE_THREADS)
        #include <processthreadsapi.h>
    #endif
#else
    #error "Unsupported platform"
#endif
```

## #elif 和 #elifdef（C23）

```c
// 标准的 elif
#if VERSION == 1
    printf("版本 1\n");
#elif VERSION == 2
    printf("版本 2\n");
#elif VERSION == 3
    printf("版本 3\n");
#endif

// C23 引入的 elifdef（某些编译器已提前支持）
#ifdef FEATURE_A
    // ...
#elifdef FEATURE_B
    // ...
#elifndef FEATURE_C
    // ...
#endif
```

## 实际应用场景

### 1. 调试/发布模式切换

```c
#ifdef DEBUG
    #define DBG_PRINT(fmt, ...) \
        fprintf(stderr, "[DEBUG %s:%d] " fmt "\n", \
                __FILE__, __LINE__, ##__VA_ARGS__)
    #define DBG_ASSERT(cond) assert(cond)
#else
    #define DBG_PRINT(fmt, ...)  // 空操作
    #define DBG_ASSERT(cond)     // 空操作
#endif
```

### 2. 编译时配置

```c
// config.h
#define CONFIG_MAX_CONNECTIONS 100
#define CONFIG_BUFFER_SIZE     4096
#define CONFIG_ENABLE_LOGGING  1
#define CONFIG_LOG_LEVEL       2  // 0=OFF, 1=ERROR, 2=INFO, 3=DEBUG

// 使用配置
#if CONFIG_LOG_LEVEL >= 1
    #define LOG_ERROR(...) log_message(LOG_ERR, __VA_ARGS__)
#else
    #define LOG_ERROR(...)
#endif

#if CONFIG_LOG_LEVEL >= 2
    #define LOG_INFO(...) log_message(LOG_INFO, __VA_ARGS__)
#else
    #define LOG_INFO(...)
#endif
```

### 3. API 版本适配

```c
#if __STDC_VERSION__ >= 201112L
    // C11: 使用 _Alignas 和 _Alignof
    _Alignas(16) int aligned_var;
    size_t alignment = _Alignof(int);
#elif defined(__GNUC__)
    // GCC 扩展
    int aligned_var __attribute__((aligned(16)));
#else
    // 无对齐支持，使用普通变量
    int aligned_var;
#endif
```

## 常见错误

### 错误1：#if 后面的宏名拼写错误

```c
#define DEBUG 1

#if DEBIG  // 拼写错误！
    printf("调试模式\n");  // 不会执行，因为 DEBIG 未定义 → 替换为 0
#endif
```

### 错误2：忘记 #endif

```c
#if defined(WIN32)
    #include <windows.h>
// 忘记写 #endif，后续代码全部被条件编译影响
```

### 错误3：在条件编译块中使用分号

```c
// 不好
#ifdef DEBUG;
    // 编译错误：预处理器不认识分号
#endif

// 正确
#ifdef DEBUG
    // ...
#endif
```

## 重要注意事项

> **关键点总结**：
> 1. `#ifdef X` 等价于 `#if defined(X)`，后者更灵活（支持组合条件）
> 2. 未定义宏在 `#if` 中被替换为 0
> 3. 条件编译是**编译时**决定的，不是运行时
> 4. 始终确保 `#if` 有对应的 `#endif`
> 5. 条件表达式只支持整数常量表达式
> 6. 善用条件编译实现跨平台代码和调试开关
