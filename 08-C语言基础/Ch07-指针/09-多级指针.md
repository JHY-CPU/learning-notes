# 多级指针

## 一、什么是多级指针

多级指针是指**指向指针的指针**。二级指针存储的是一级指针的地址，三级指针存储的是二级指针的地址，以此类推。

```c
int x = 42;
int *p = &x;      // 一级指针，指向 int
int **pp = &p;    // 二级指针，指向 int*

printf("x  = %d\n", x);       // 42
printf("*p = %d\n", *p);      // 42
printf("**pp = %d\n", **pp);  // 42
```

### 内存模型

```
pp  ->  p   ->  x
(二级)  (一级)  (变量)
```

## 二、二级指针详解

### 定义与使用

```c
int num = 100;
int *ptr = &num;
int **pptr = &ptr;

// 读取
printf("num = %d\n", **pptr);  // 100

// 修改
**pptr = 200;
printf("num = %d\n", num);     // 200
```

### 为什么需要二级指针

**核心原因：函数需要修改指针本身**。

```c
// 场景：函数内分配内存并返回
// 如果只用一级指针，无法修改调用者的指针
void allocate(int **pp) {
    *pp = (int*)malloc(sizeof(int));
    **pp = 42;
}

int main() {
    int *p = NULL;
    allocate(&p);       // 传入指针的地址
    printf("%d\n", *p); // 42
    free(p);
    return 0;
}
```

## 三、二级指针的典型应用场景

### 1. 修改指针本身

```c
void set_null(int **pp) {
    *pp = NULL;
}

int main() {
    int x = 10;
    int *p = &x;
    printf("p 是否为空: %s\n", p == NULL ? "是" : "否");
    set_null(&p);
    printf("p 是否为空: %s\n", p == NULL ? "是" : "否");
    return 0;
}
```

### 2. 动态二维数组

```c
int rows = 3, cols = 4;
int **matrix = (int**)malloc(rows * sizeof(int*));
for (int i = 0; i < rows; i++) {
    matrix[i] = (int*)malloc(cols * sizeof(int));
}

// 使用
matrix[1][2] = 42;

// 释放
for (int i = 0; i < rows; i++) {
    free(matrix[i]);
}
free(matrix);
```

### 3. 函数中修改字符串指针

```c
void change_string(char **str) {
    *str = "Hello, World!";
}

int main() {
    char *msg = "Original";
    printf("Before: %s\n", msg);
    change_string(&msg);
    printf("After: %s\n", msg);  // Hello, World!
    return 0;
}
```

### 4. 链表头节点修改

```c
struct Node {
    int data;
    struct Node *next;
};

// 使用二级指针修改链表头
void insert_front(struct Node **head, int val) {
    struct Node *new_node = malloc(sizeof(struct Node));
    new_node->data = val;
    new_node->next = *head;
    *head = new_node;  // 修改头指针
}
```

## 四、三级及更高级别指针

虽然技术上可以定义三级甚至更多级指针，但实际编程中极少使用。

```c
int x = 10;
int *p = &x;
int **pp = &p;
int ***ppp = &pp;

printf("%d\n", ***ppp);  // 10
```

一般规则：
- 二级指针在实际中广泛使用
- 三级指针偶尔出现（如修改二级指针）
- 更高级别几乎不用，应该重构代码

## 五、多级指针与数组

```c
// 字符串数组
const char *names[] = {"Alice", "Bob", "Charlie"};
const char **pp = names;  // pp 指向 names[0]

for (int i = 0; i < 3; i++) {
    printf("%s\n", pp[i]);
}
```

## 六、常见错误

```c
// 错误：类型不匹配
int x = 10;
int **pp = &x;    // 错误！&x 是 int*，不是 int**

// 正确
int *p = &x;
int **pp = &p;    // 正确
```

## 七、关键要点总结

> **核心概念**
> - 二级指针 `int **pp` 存储一级指针的地址
> - `**pp` 两次解引用才能访问原始数据
> - 主要用途是函数中修改指针本身

> **使用原则**
> - 需要在函数内分配/释放内存时使用二级指针
> - 需要在函数内修改头指针（链表）时使用二级指针
> - 尽量避免使用三级以上指针
