# 指针与动态二维数组

## 一、概述

C 语言中静态二维数组的大小必须在编译时确定。使用指针和动态内存分配，可以创建**运行时决定大小**的二维数组。

## 二、方法一：指针数组方式（不连续存储）

### 分配

```c
#include <stdio.h>
#include <stdlib.h>

int** create_matrix(int rows, int cols) {
    // 分配行指针数组
    int **matrix = (int**)malloc(rows * sizeof(int*));
    if (matrix == NULL) return NULL;

    for (int i = 0; i < rows; i++) {
        matrix[i] = (int*)malloc(cols * sizeof(int));
        if (matrix[i] == NULL) {
            // 分配失败，释放已分配的行
            for (int j = 0; j < i; j++) {
                free(matrix[j]);
            }
            free(matrix);
            return NULL;
        }
    }
    return matrix;
}
```

### 使用

```c
int main() {
    int rows = 3, cols = 4;
    int **matrix = create_matrix(rows, cols);
    if (matrix == NULL) return 1;

    // 像二维数组一样使用
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            matrix[i][j] = i * cols + j + 1;
            printf("%3d ", matrix[i][j]);
        }
        printf("\n");
    }

    // 释放（每行单独释放）
    for (int i = 0; i < rows; i++) {
        free(matrix[i]);
    }
    free(matrix);
    return 0;
}
```

### 内存布局

```
matrix -> [ptr0] -> [1][2][3][4]
          [ptr1] -> [5][6][7][8]
          [ptr2] -> [9][10][11][12]
```

各行**不一定连续**。

## 三、方法二：连续分配方式

### 分配

```c
int* create_flat_matrix(int rows, int cols) {
    int *matrix = (int*)malloc(rows * cols * sizeof(int));
    return matrix;  // 一维数组模拟二维
}

// 访问宏
#define M(m, r, c, cols) ((m)[(r) * (cols) + (c)])
```

### 使用

```c
int main() {
    int rows = 3, cols = 4;
    int *matrix = create_flat_matrix(rows, cols);
    if (matrix == NULL) return 1;

    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            M(matrix, i, j, cols) = i * cols + j + 1;
            printf("%3d ", M(matrix, i, j, cols));
        }
        printf("\n");
    }

    free(matrix);
    return 0;
}
```

## 四、方法三：行指针 + 连续内存（推荐）

兼具 `matrix[i][j]` 的语法和内存连续性。

```c
int** create_contiguous_matrix(int rows, int cols) {
    // 分配所有数据（一次性连续分配）
    int *data = (int*)malloc(rows * cols * sizeof(int));
    if (data == NULL) return NULL;

    // 分配行指针数组
    int **matrix = (int**)malloc(rows * sizeof(int*));
    if (matrix == NULL) {
        free(data);
        return NULL;
    }

    // 设置每行指针
    for (int i = 0; i < rows; i++) {
        matrix[i] = &data[i * cols];
    }

    return matrix;
}

void free_contiguous_matrix(int **matrix) {
    if (matrix != NULL) {
        free(matrix[0]);  // 释放数据区
        free(matrix);     // 释放行指针数组
    }
}
```

### 使用

```c
int main() {
    int rows = 3, cols = 4;
    int **matrix = create_contiguous_matrix(rows, cols);

    // 可以使用 matrix[i][j] 语法
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            matrix[i][j] = i * cols + j + 1;
        }
    }

    // 也可以用一维指针遍历
    int *flat = matrix[0];
    for (int i = 0; i < rows * cols; i++) {
        printf("%d ", flat[i]);
    }

    free_contiguous_matrix(matrix);
    return 0;
}
```

## 五、三种方法对比

| 特性 | 指针数组 | 连续内存 | 行指针+连续 |
|------|---------|---------|------------|
| 语法 | `matrix[i][j]` | `M(m,i,j,c)` | `matrix[i][j]` |
| 内存连续性 | 不连续 | 连续 | 连续 |
| 分配次数 | rows+1 次 | 1 次 | 2 次 |
| 释放次数 | rows+1 次 | 1 次 | 2 次 |
| cache 友好性 | 差 | 好 | 好 |
| 每行可独立大小 | 可以 | 不可以 | 不可以 |

## 六、作为函数参数传递

```c
// 方法一：传递 int**
void process_matrix1(int **matrix, int rows, int cols);

// 方法二：传递一维数组
void process_matrix2(int *matrix, int rows, int cols);

// 方法三：传递行指针（需要知道列数）
void process_matrix3(int (*matrix)[cols], int rows, int cols);  // VLA
```

## 七、常见错误

```c
// 错误一：忘记释放每行
int **m = create_matrix(3, 4);
free(m);  // 泄漏！只释放了行指针数组

// 正确
for (int i = 0; i < 3; i++) free(m[i]);
free(m);

// 错误二：混合释放方式
int **m = create_contiguous_matrix(3, 4);
for (int i = 0; i < 3; i++) free(m[i]);  // 错误！m[i] 不是单独分配的
```

## 八、关键要点总结

> **三种创建方式**
> - 指针数组：灵活但不连续
> - 连续内存：高效但语法不便
> - 行指针+连续：兼顾语法和效率（推荐）

> **核心原则**
> - 分配和释放方式必须匹配
> - 分配失败时需要回滚已分配的资源
> - 大矩阵推荐连续分配，cache 友好
