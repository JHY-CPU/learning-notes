# 字符串指针

## 一、字符串的存储方式

在 C 语言中，字符串有两种主要的表示方式：**字符数组**和**字符串指针**。

```c
// 方式一：字符数组
char str1[] = "Hello";

// 方式二：字符串指针
const char *str2 = "Hello";
```

### 内存布局差异

```
字符数组 str1:
栈内存: [H][e][l][l][o][\0]  (可修改)

字符串指针 str2:
栈: [指针] -> 只读数据区: [H][e][l][l][o][\0]  (不可修改)
```

## 二、字符串指针的定义与使用

### 基本定义

```c
const char *str = "Hello, World!";

// 遍历字符串
for (const char *p = str; *p != '\0'; p++) {
    printf("%c", *p);
}
printf("\n");
```

### 访问字符

```c
const char *str = "Hello";

printf("%c\n", str[0]);    // 'H'  下标访问
printf("%c\n", *(str + 1)); // 'e'  指针偏移访问
```

## 三、字符数组 vs 字符串指针

### 关键区别

```c
// 字符数组：可以修改内容
char arr[] = "Hello";
arr[0] = 'h';       // OK
printf("%s\n", arr); // "hello"

// 字符串指针：指向字符串字面量，不可修改
const char *ptr = "Hello";
// ptr[0] = 'h';    // 未定义行为！可能崩溃

// 但指针本身可以重新赋值
ptr = "World";      // OK
```

### 对比总结

| 特性 | `char arr[]` | `const char *ptr` |
|------|-------------|-------------------|
| 内存位置 | 栈（局部）或数据段（全局） | 只读数据区 |
| 可否修改内容 | 可以 | 不可以 |
| 可否重新赋值 | 不可以（数组名是常量） | 可以 |
| sizeof 大小 | 字符串长度+1 | 指针大小（4或8字节） |

```c
char arr[] = "Hello";
const char *ptr = "Hello";

printf("sizeof(arr) = %zu\n", sizeof(arr));  // 6
printf("sizeof(ptr) = %zu\n", sizeof(ptr));  // 8 (64位)
```

## 四、字符串指针的常见操作

### 1. 字符串长度

```c
size_t my_strlen(const char *s) {
    const char *p = s;
    while (*p) p++;
    return p - s;
}
```

### 2. 字符串复制

```c
void my_strcpy(char *dest, const char *src) {
    while ((*dest++ = *src++) != '\0');
}
```

### 3. 字符串比较

```c
int my_strcmp(const char *s1, const char *s2) {
    while (*s1 && *s1 == *s2) {
        s1++;
        s2++;
    }
    return (unsigned char)*s1 - (unsigned char)*s2;
}
```

### 4. 字符串连接

```c
char* my_strcat(char *dest, const char *src) {
    char *p = dest;
    while (*p) p++;          // 找到 dest 末尾
    while ((*p++ = *src++)); // 复制 src
    return dest;
}
```

## 五、字符串指针数组

```c
const char *days[] = {
    "Monday", "Tuesday", "Wednesday",
    "Thursday", "Friday", "Saturday", "Sunday"
};

for (int i = 0; i < 7; i++) {
    printf("Day %d: %s\n", i + 1, days[i]);
}
```

### 与二维字符数组的对比

```c
// 指针数组：更节省内存，长度灵活
const char *names1[] = {"Alice", "Bob", "Christopher"};

// 二维数组：每行等长，可修改
char names2[3][20] = {"Alice", "Bob", "Christopher"};
```

## 六、动态字符串

```c
#include <stdlib.h>
#include <string.h>

// 动态分配字符串
char* create_string(const char *src) {
    char *str = (char*)malloc(strlen(src) + 1);
    if (str) {
        strcpy(str, src);
    }
    return str;
}

int main() {
    char *msg = create_string("Hello, World!");
    if (msg) {
        msg[0] = 'h';  // 可以修改
        printf("%s\n", msg);
        free(msg);
    }
    return 0;
}
```

## 七、字符串字面量的存储

```c
// 相同的字符串字面量可能共享同一地址（编译器优化）
const char *s1 = "Hello";
const char *s2 = "Hello";

printf("s1 = %p\n", (void*)s1);
printf("s2 = %p\n", (void*)s2);
// 地址可能相同（实现定义）
```

## 八、关键要点总结

> **核心区别**
> - 字符数组可修改内容，字符串指针指向字面量不可修改
> - sizeof 对数组返回总大小，对指针返回指针大小
> - 数组名不可重新赋值，指针可以

> **编程规范**
> - 字符串指针使用 `const char*` 保护
> - 动态字符串使用 malloc + strcpy 创建
> - 函数参数用 `const char*` 接收只读字符串

> **常见错误**
> - 试图修改字符串字面量
> - 返回局部字符数组的地址
> - 忘记给动态字符串分配 `\0` 的空间
