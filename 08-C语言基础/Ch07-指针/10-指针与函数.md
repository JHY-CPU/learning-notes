# 指针与函数

## 一、指针作为函数参数（传址调用）

C 语言中函数参数默认是**传值调用**（值传递），函数内修改形参不影响实参。要让函数修改外部变量，需要传递**指针**（传址调用）。

### 值传递 vs 传址传递

```c
// 值传递：无法交换外部变量
void swap_wrong(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

// 传址传递：正确交换
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap_wrong(x, y);   // 无效，x、y 不变
    swap(&x, &y);       // 有效，x、y 被交换
    printf("x=%d, y=%d\n", x, y);  // x=20, y=10
    return 0;
}
```

## 二、常见的传址模式

### 1. 修改基本类型

```c
void increment(int *n) {
    (*n)++;
}

void set_zero(int *n) {
    *n = 0;
}
```

### 2. 修改数组内容

```c
void fill_array(int *arr, int n, int value) {
    for (int i = 0; i < n; i++) {
        arr[i] = value;
    }
}

// 使用
int arr[10];
fill_array(arr, 10, -1);  // 所有元素设为 -1
```

### 3. 返回多个值

```c
// C 函数只能有一个返回值
// 通过指针参数可以"返回"多个结果
void divide(int a, int b, int *quotient, int *remainder) {
    *quotient = a / b;
    *remainder = a % b;
}

int main() {
    int q, r;
    divide(17, 5, &q, &r);
    printf("商: %d, 余数: %d\n", q, r);  // 商: 3, 余数: 2
    return 0;
}
```

### 4. 修改指针本身

```c
// 需要二级指针
void allocate_int(int **pp, int value) {
    *pp = (int*)malloc(sizeof(int));
    if (*pp != NULL) {
        **pp = value;
    }
}

int main() {
    int *p = NULL;
    allocate_int(&p, 42);
    printf("%d\n", *p);  // 42
    free(p);
    return 0;
}
```

## 三、指针作为函数返回值

函数可以返回指针，但必须确保指向的内存仍然有效。

### 正确做法

```c
// 返回静态变量的地址
int* get_counter() {
    static int count = 0;
    count++;
    return &count;
}

// 返回动态分配的内存
int* create_int(int value) {
    int *p = (int*)malloc(sizeof(int));
    *p = value;
    return p;
}

// 返回参数中的某个地址
int* find_max(int *arr, int n) {
    int *max = &arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > *max) {
            max = &arr[i];
        }
    }
    return max;
}
```

### 错误做法

```c
// 错误：返回局部变量的地址
int* bad_return() {
    int local = 42;
    return &local;  // local 在函数返回后销毁！
}

// 错误：返回局部数组的地址
char* bad_string() {
    char buf[] = "hello";
    return buf;  // buf 在函数返回后销毁！
}
```

### 字符串字面量的特殊性

```c
// 正确：字符串字面量存储在只读区，生命周期贯穿整个程序
const char* get_greeting() {
    return "Hello";  // OK
}

// 正确：动态分配
char* get_string() {
    char *s = (char*)malloc(6);
    strcpy(s, "hello");
    return s;  // 调用者需负责 free
}
```

## 四、const 指针参数

使用 `const` 修饰指针参数，防止函数意外修改数据。

```c
// 不修改传入的数据
void print_array(const int *arr, int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
        // arr[i] = 0;  // 编译错误！const 保护
    }
    printf("\n");
}

// 可以修改
void zero_array(int *arr, int n) {
    for (int i = 0; i < n; i++) {
        arr[i] = 0;
    }
}
```

## 五、函数参数的设计原则

1. **输入参数**：使用 `const` 指针，只读不写
2. **输出参数**：使用普通指针，函数内写入结果
3. **输入输出参数**：使用普通指针，既读又写

```c
// 输入：src 只读
// 输出：dst 只写
// n: 输入参数（值传递）
void copy_array(const int *src, int *dst, int n) {
    for (int i = 0; i < n; i++) {
        dst[i] = src[i];
    }
}
```

## 六、关键要点总结

> **核心概念**
> - 值传递不修改原数据，传址传递可以修改
> - 指针参数让函数可以"返回"多个值
> - 返回指针时必须确保指向的内存有效

> **安全原则**
> - 不要返回局部变量的地址
> - 输入参数加 const 保护
> - 动态分配的返回值要注明谁负责释放
