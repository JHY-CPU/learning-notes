# 指针的运算

## 一、指针运算概述

指针变量存储的是内存地址，C 语言允许对指针进行有限的运算操作。指针运算不是简单的整数加减，而是**根据指针所指向类型的大小自动缩放**的。

## 二、指针加法

指针加上一个整数 n，实际移动的字节数为 `n * sizeof(指向类型)`。

```c
#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *p = arr;  // 指向 arr[0]

    printf("p   指向: %p, 值: %d\n", (void*)p, *p);
    printf("p+1 指向: %p, 值: %d\n", (void*)(p+1), *(p+1));
    printf("p+2 指向: %p, 值: %d\n", (void*)(p+2), *(p+2));

    printf("指针大小差: %ld 字节\n", (char*)(p+1) - (char*)p);
    // 输出 4（在 int 为 4 字节的系统上）

    return 0;
}
```

### 运算规则

```c
int *pi;
// pi + n 实际上移动了 n * sizeof(int) 个字节

char *pc;
// pc + n 实际上移动了 n * sizeof(char) = n 个字节

double *pd;
// pd + n 实际上移动了 n * sizeof(double) 个字节
```

## 三、指针减法（整数）

指针减去一个整数，同样按指向类型的大小缩放。

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p = &arr[4];  // 指向 arr[4]

printf("%d\n", *p);      // 50
printf("%d\n", *(p-1));  // 40
printf("%d\n", *(p-2));  // 30
```

## 四、两个指针相减

两个**指向同一数组（或同一内存块）** 的指针可以相减，结果是它们之间的**元素个数**。

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p1 = &arr[1];
int *p2 = &arr[4];

ptrdiff_t diff = p2 - p1;
printf("差值: %td\n", diff);  // 输出 3

// ptrdiff_t 是有符号整数类型，专用于指针差值
```

### 注意事项

- 两个指针必须指向同一数组/内存块
- 结果类型是 `ptrdiff_t`（有符号类型）
- 不同数组的指针相减是未定义行为

```c
int a[5], b[5];
int *pa = &a[0];
int *pb = &b[0];
// ptrdiff_t d = pa - pb;  // 未定义行为！指向不同数组
```

## 五、指针比较运算

指针可以用关系运算符进行比较（`<`、`<=`、`>`、`>=`、`==`、`!=`）。

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p = arr;
int *q = &arr[3];

if (p < q) {
    printf("p 在 q 之前\n");
}

if (p == arr) {
    printf("p 指向数组首元素\n");
}

if (q != NULL) {
    printf("q 不是空指针\n");
}
```

### 比较的规则

- 同一数组内的指针比较有意义
- 不同数组的指针比较是未定义行为（除了 `==` 和 `!=`）
- 任何指针都可以与 NULL 比较

## 六、指针与自增自减

```c
int arr[3] = {10, 20, 30};
int *p = arr;

printf("%d\n", *p++);  // 输出 10，然后 p 指向 arr[1]
printf("%d\n", *p);    // 输出 20

printf("%d\n", ++*p);  // 先将 *p 加1（arr[1] 变为 21），再输出 21
printf("%d\n", *++p);  // 先移动 p 到 arr[2]，再解引用输出 30
```

### 优先级表

| 表达式 | 含义 |
|--------|------|
| `*p++` | `*(p++)` 先解引用当前 p，再 p 加 1 |
| `*++p` | `*(++p)` 先 p 加 1，再解引用 |
| `(*p)++` | 解引用后，值自增 1 |
| `++*p` | 解引用后，值先自增 1，再使用 |

## 七、指针运算的实际应用

### 遍历数组

```c
int arr[] = {1, 2, 3, 4, 5};
int n = sizeof(arr) / sizeof(arr[0]);

// 方法一：下标法
for (int i = 0; i < n; i++) {
    printf("%d ", arr[i]);
}

// 方法二：指针运算
for (int *p = arr; p < arr + n; p++) {
    printf("%d ", *p);
}
```

### 计算字符串长度

```c
size_t my_strlen(const char *s) {
    const char *p = s;
    while (*p != '\0') {
        p++;
    }
    return p - s;  // 指针相减得到长度
}
```

## 八、指针运算的边界限制

```c
int arr[5] = {1, 2, 3, 4, 5};
int *p = arr;

// 合法：p 可以指向 arr[0] 到 arr[4]
// 合法但不可解引用：p 可以等于 arr + 5（指向末尾之后）
// 非法：p 不能等于 arr + 6（超出范围）
// 非法：p 不能小于 arr（不能指向数组之前）
```

## 九、关键要点总结

> **核心规则**
> - 指针加减 n，实际移动 `n * sizeof(类型)` 字节
> - 同一数组的指针可以相减，结果是元素个数
> - 同一数组的指针可以用关系运算符比较
> - `*p++` 等表达式要注意优先级

> **注意事项**
> - 超出数组边界的指针运算是未定义行为
> - 不同数组的指针相减或比较（除 `==`/`!=`）是未定义行为
> - `++` 和 `*` 的优先级相同，结合性从右到左
