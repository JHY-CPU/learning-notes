# 指针与地址

## 一、`&` 取地址运算符

`&` 运算符用于获取变量的内存地址。

```c
int x = 42;
printf("x 的地址: %p\n", (void*)&x);

double d = 3.14;
printf("d 的地址: %p\n", (void*)&d);

char c = 'A';
printf("c 的地址: %p\n", (void*)&c);
```

### 地址的本质

- 地址是一个整数，表示内存中某个字节的位置编号
- 地址的大小与系统架构有关：32 位系统为 4 字节，64 位系统为 8 字节
- 我们通常用十六进制来表示地址

### 不能取地址的情况

```c
// 错误：不能对常量取地址
// int *p = &42;     // 编译错误

// 错误：不能对表达式取地址
// int a = 1, b = 2;
// int *p = &(a + b); // 编译错误

// 正确：可以对变量取地址
int a = 1;
int *p = &a;         // OK
```

## 二、地址的类型

每个地址都带有**类型信息**，这是 C 语言指针安全的基础。

```c
int n = 100;
float f = 3.14f;

// &n 的类型是 int*
// &f 的类型是 float*
```

类型决定了：
1. 从该地址开始读取多少字节
2. 如何解释这些字节的数据

## 三、指针类型匹配

### 类型必须匹配

```c
int n = 42;
int *ip = &n;      // 正确：类型匹配

// float *fp = &n; // 警告：类型不匹配
```

### 为什么类型匹配重要

```c
int n = 0x40490fdb;  // IEEE 754 表示 3.14159 的二进制
int *ip = &n;
float *fp = (float*)&n;  // 强制转换

printf("%d\n", *ip);     // 1078523387（整数解释）
printf("%f\n", *fp);     // 3.141593（浮点解释）
```

同一个地址，不同的指针类型，读取到的值完全不同！

## 四、void 指针（通用指针）

### 定义

`void*` 是一种特殊的指针类型，它可以指向**任何类型**的数据，但它本身没有类型信息。

```c
int x = 42;
void *vp = &x;    // void* 可以指向任何类型
```

### void* 的特点

1. **可以存储任意类型的地址**
2. **不能直接解引用**（因为不知道读几个字节）
3. **可以与任何指针类型互相转换**

```c
int x = 42;
void *vp = &x;

// *vp;           // 错误！void* 不能解引用
int *ip = (int*)vp;  // 必须先转换为具体类型
printf("%d\n", *ip); // 正确：输出 42
```

### void* 的常见用途

```c
// 1. malloc 返回 void*
int *p = (int*)malloc(sizeof(int));

// 2. 通用函数参数
void print_value(void *ptr, char type) {
    switch (type) {
        case 'i': printf("%d\n", *(int*)ptr); break;
        case 'f': printf("%f\n", *(float*)ptr); break;
        case 'c': printf("%c\n", *(char*)ptr); break;
    }
}

// 3. 泛型数据结构（如链表中存储任意类型数据）
```

## 五、地址与指针的区别

| 概念 | 说明 |
|------|------|
| 地址 | 内存中某个位置的编号（整数值） |
| 指针 | 存储地址的变量，带有类型信息 |
| `&x` | 表达式，结果是地址，有类型 |
| `int *p` | 指针变量，存储 int 类型的地址 |

```c
int x = 10;
int *p = &x;

// &x 是表达式，类型为 int*，值为 x 的地址
// p  是变量，类型为 int*，值为 &x
// &x 和 p 的值相同
```

## 六、地址的打印与格式化

```c
#include <stdio.h>

int main() {
    int x = 42;

    printf("标准打印: %p\n", (void*)&x);
    printf("作为无符号整数: %lu\n", (unsigned long)(void*)&x);

    // 指针的指针
    int *p = &x;
    int **pp = &p;
    printf("p 的地址: %p\n", (void*)&p);
    printf("pp 的值: %p\n", (void*)pp);

    return 0;
}
```

## 七、关键要点总结

> **核心概念**
> - `&` 取地址，结果带有类型信息
> - 指针类型决定了解引用时的行为
> - `void*` 是通用指针，但不能直接解引用
> - 地址是值，指针是存储地址的变量

> **注意事项**
> - 不能对常量或表达式取地址
> - `void*` 在 C 中可以隐式转换（但在 C++ 中需要显式转换）
> - 打印地址使用 `%p` 格式符，并强制转换为 `void*`
