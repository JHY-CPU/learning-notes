# 函数指针详解

## 一、什么是函数指针

函数指针（Function Pointer）是指**指向函数的指针变量**。函数在内存中也有地址，函数指针存储这个地址，通过函数指针可以调用函数。

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    // 定义函数指针
    int (*func_ptr)(int, int) = add;

    // 通过函数指针调用
    int result = func_ptr(3, 5);
    printf("结果: %d\n", result);  // 8

    return 0;
}
```

## 二、函数指针的定义语法

```
返回类型 (*指针名)(参数类型列表);
```

```c
// 指向 int func(int, int) 类型的函数
int (*fp)(int, int);

// 指向 void func(double*) 类型的函数
void (*fp2)(double*);

// 指向 char* func(const char*, int) 类型的函数
char* (*fp3)(const char*, int);
```

### 定义技巧

`(*fp)` 的括号必不可少：

```c
int *fp(int, int);   // 这是函数声明！返回 int*
int (*fp)(int, int); // 这是函数指针
```

## 三、函数指针的赋值

```c
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }

int main() {
    int (*fp)(int, int);

    fp = add;         // 赋值（函数名即地址）
    printf("%d\n", fp(10, 3));  // 13

    fp = sub;         // 切换函数
    printf("%d\n", fp(10, 3));  // 7

    // 函数名和 &函数名 两种写法等价
    fp = &add;        // 等价于 fp = add;

    return 0;
}
```

## 四、通过 typedef 简化

```c
// 使用 typedef 定义函数指针类型
typedef int (*MathFunc)(int, int);

int add(int a, int b) { return a + b; }
int mul(int a, int b) { return a * b; }

int main() {
    MathFunc fp = add;
    printf("%d\n", fp(4, 5));  // 9

    fp = mul;
    printf("%d\n", fp(4, 5));  // 20

    return 0;
}
```

## 五、函数指针数组

```c
typedef int (*OpFunc)(int, int);

int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }

int main() {
    OpFunc ops[3] = {add, sub, mul};
    const char *names[] = {"加法", "减法", "乘法"};

    for (int i = 0; i < 3; i++) {
        printf("%s: 10, 3 = %d\n", names[i], ops[i](10, 3));
    }

    return 0;
}
```

## 六、回调函数（Callback）

回调函数是指**将函数作为参数传递给另一个函数**，由后者在适当时候调用。

```c
// 比较函数的回调
int int_compare(const void *a, const void *b) {
    return *(const int*)a - *(const int*)b;
}

// 使用 qsort（标准库的快速排序）
int arr[] = {5, 2, 8, 1, 9};
qsort(arr, 5, sizeof(int), int_compare);

// 通用的查找函数
int* find_if(int *arr, int n, int (*predicate)(int)) {
    for (int i = 0; i < n; i++) {
        if (predicate(arr[i])) {
            return &arr[i];
        }
    }
    return NULL;
}

int is_even(int x) { return x % 2 == 0; }
int is_negative(int x) { return x < 0; }

// 调用
int arr[] = {1, 3, 4, 7, 8};
int *p = find_if(arr, 5, is_even);  // 找到第一个偶数
```

## 七、策略模式实现

```c
typedef double (*Strategy)(double, double);

double max(double a, double b) { return a > b ? a : b; }
double min(double a, double b) { return a < b ? a : b; }
double avg(double a, double b) { return (a + b) / 2.0; }

double apply_strategy(double a, double b, Strategy s) {
    return s(a, b);
}

int main() {
    printf("max: %.1f\n", apply_strategy(3.0, 7.0, max));  // 7.0
    printf("min: %.1f\n", apply_strategy(3.0, 7.0, min));  // 3.0
    printf("avg: %.1f\n", apply_strategy(3.0, 7.0, avg));  // 5.0
    return 0;
}
```

## 八、关键要点总结

> **核心概念**
> - 函数指针存储函数的入口地址
> - `返回类型 (*名)(参数列表)` 定义函数指针
> - 函数名本身就是地址，`fp = add` 和 `fp = &add` 等价

> **主要应用**
> - 回调函数（qsort 比较函数）
> - 策略模式（运行时选择算法）
> - 状态机和事件驱动系统
> - 插件架构

> **最佳实践**
> - 使用 typedef 简化复杂声明
> - 函数指针数组实现跳转表
> - 回调函数保持简洁和通用
