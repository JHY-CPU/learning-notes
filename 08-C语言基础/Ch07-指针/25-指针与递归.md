# 指针与递归

## 一、概述

递归和指针经常结合使用，特别是在处理链表、树等递归定义的数据结构时。指针使递归操作这些数据结构变得自然而高效。

## 二、链表的递归操作

### 递归遍历

```c
struct Node {
    int data;
    struct Node *next;
};

void print_list_recursive(struct Node *head) {
    if (head == NULL) {  // 基准情况
        printf("NULL\n");
        return;
    }
    printf("%d -> ", head->data);
    print_list_recursive(head->next);
}
```

### 递归计算长度

```c
int list_length(struct Node *head) {
    if (head == NULL) return 0;
    return 1 + list_length(head->next);
}
```

### 递归反转链表

```c
struct Node* reverse_recursive(struct Node *head) {
    if (head == NULL || head->next == NULL) {
        return head;
    }

    struct Node *new_head = reverse_recursive(head->next);
    head->next->next = head;
    head->next = NULL;
    return new_head;
}
```

### 递归查找

```c
struct Node* find_recursive(struct Node *head, int value) {
    if (head == NULL) return NULL;
    if (head->data == value) return head;
    return find_recursive(head->next, value);
}
```

## 三、二叉树的递归操作

### 节点定义

```c
struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
};
```

### 创建节点

```c
struct TreeNode* create_tree_node(int data) {
    struct TreeNode *node = (struct TreeNode*)malloc(sizeof(struct TreeNode));
    if (node == NULL) return NULL;
    node->data = data;
    node->left = NULL;
    node->right = NULL;
    return node;
}
```

### 先序遍历（根-左-右）

```c
void preorder(struct TreeNode *root) {
    if (root == NULL) return;
    printf("%d ", root->data);
    preorder(root->left);
    preorder(root->right);
}
```

### 中序遍历（左-根-右）

```c
void inorder(struct TreeNode *root) {
    if (root == NULL) return;
    inorder(root->left);
    printf("%d ", root->data);
    inorder(root->right);
}
```

### 后序遍历（左-右-根）

```c
void postorder(struct TreeNode *root) {
    if (root == NULL) return;
    postorder(root->left);
    postorder(root->right);
    printf("%d ", root->data);
}
```

### 计算树的高度

```c
int tree_height(struct TreeNode *root) {
    if (root == NULL) return 0;

    int left_height = tree_height(root->left);
    int right_height = tree_height(root->right);

    return 1 + (left_height > right_height ? left_height : right_height);
}
```

### 统计节点数

```c
int count_nodes(struct TreeNode *root) {
    if (root == NULL) return 0;
    return 1 + count_nodes(root->left) + count_nodes(root->right);
}
```

### 递归查找（BST）

```c
struct TreeNode* bst_search(struct TreeNode *root, int value) {
    if (root == NULL || root->data == value) {
        return root;
    }
    if (value < root->data) {
        return bst_search(root->left, value);
    }
    return bst_search(root->right, value);
}
```

### 递归插入（BST）

```c
struct TreeNode* bst_insert(struct TreeNode *root, int value) {
    if (root == NULL) {
        return create_tree_node(value);
    }
    if (value < root->data) {
        root->left = bst_insert(root->left, value);
    } else if (value > root->data) {
        root->right = bst_insert(root->right, value);
    }
    // 值已存在则不插入
    return root;
}
```

### 释放树（后序释放）

```c
void free_tree(struct TreeNode *root) {
    if (root == NULL) return;
    free_tree(root->left);
    free_tree(root->right);
    free(root);
}
```

## 四、指针参数的递归

### 递归修改指针

```c
// 递归释放链表（使用二级指针）
void free_list_recursive(struct Node **head) {
    if (*head == NULL) return;
    struct Node *temp = *head;
    *head = (*head)->next;
    free(temp);
    free_list_recursive(head);
}
```

### 递归构建数据结构

```c
// 从数组递归构建 BST
struct TreeNode* build_bst(int *arr, int start, int end) {
    if (start > end) return NULL;

    int mid = start + (end - start) / 2;
    struct TreeNode *root = create_tree_node(arr[mid]);
    root->left = build_bst(arr, start, mid - 1);
    root->right = build_bst(arr, mid + 1, end);

    return root;
}
```

## 五、递归深度与栈溢出

```c
// 深度递归可能导致栈溢出
void deep_recursion(int n) {
    int local_array[1000];  // 每层递归占用大量栈空间
    if (n <= 0) return;
    deep_recursion(n - 1);
}

// 对于极深的数据结构，考虑使用迭代
int list_length_iterative(struct Node *head) {
    int count = 0;
    while (head != NULL) {
        count++;
        head = head->next;
    }
    return count;
}
```

## 六、尾递归优化

```c
// 尾递归：递归调用是函数的最后操作
int factorial_tail(int n, int acc) {
    if (n <= 1) return acc;
    return factorial_tail(n - 1, n * acc);  // 尾调用
}

// 使用
int factorial(int n) {
    return factorial_tail(n, 1);
}
```

尾递归可以被编译器优化为循环，避免栈溢出。

## 七、关键要点总结

> **适用场景**
> - 链表的递归操作：遍历、反转、查找
> - 树的递归操作：遍历、搜索、插入、删除
> - 分治算法：二分查找、归并排序

> **注意事项**
> - 基准条件（base case）必须存在
> - 递归深度可能导致栈溢出
> - 尾递归可以被优化为迭代
> - 复杂操作（如树的释放）注意递归顺序
