# 指针的定义与初始化

## 一、指针定义语法

指针变量的定义格式为：

```
类型名 *指针变量名;
```

```c
int *p;          // p 是指向 int 类型的指针
char *str;       // str 是指向 char 类型的指针
float *fp;       // fp 是指向 float 类型的指针
double *dp;      // dp 是指向 double 类型的指针
```

### 定义时的注意事项

`*` 的位置有三种常见写法，含义相同但风格不同：

```c
int* p;    // 风格一：* 紧贴类型，强调"指针类型"
int *p;    // 风格二：* 紧贴变量名，强调"解引用时"
int * p;   // 风格三：两边都有空格（不常见）
```

**推荐使用 `int *p;` 风格**，因为在同一行定义多个指针时更清晰：

```c
int *p, *q;   // p 和 q 都是指针
int* p, q;    // p 是指针，q 是普通 int（容易误解！）
```

## 二、指针的初始化

### 1. 使用已有变量的地址初始化

```c
int x = 42;
int *p = &x;    // p 指向 x
```

### 2. 使用 NULL 初始化

```c
int *p = NULL;  // p 是空指针，不指向任何对象
```

### 3. 使用动态分配的内存初始化

```c
int *p = (int*)malloc(sizeof(int));
*p = 100;
free(p);
p = NULL;  // 释放后置空
```

### 4. 使用数组名初始化

```c
int arr[5] = {1, 2, 3, 4, 5};
int *p = arr;   // p 指向数组第一个元素
```

### 5. 使用另一个指针初始化

```c
int x = 10;
int *p = &x;
int *q = p;     // q 和 p 指向同一个地址
```

## 三、NULL 指针

### NULL 的定义

`NULL` 在 C 语言中被定义为 `((void*)0)` 或 `0`，是一个"空指针常量"。

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int *p = NULL;
    printf("NULL 的值: %p\n", (void*)p);  // 输出 0x0 或 (nil)
    return 0;
}
```

### NULL 的用途

1. **表示指针未初始化**或不指向有效对象
2. **安全检查**：解引用前判断指针是否有效
3. **哨兵值**：在链表等数据结构中标记结尾

```c
// 使用前检查指针有效性
int *p = NULL;
if (p != NULL) {
    printf("%d\n", *p);
} else {
    printf("指针为空，不能解引用\n");
}
```

## 四、野指针（Wild Pointer）

### 什么是野指针

野指针是指**指向不确定内存地址**的指针。使用未初始化的指针是最常见的野指针来源。

```c
int *p;      // p 是野指针，值不确定！
*p = 100;    // 未定义行为：可能崩溃、可能写入未知内存
```

### 野指针的常见来源

```c
// 来源一：未初始化
int *p;
// 此时 p 的值是随机的垃圾值

// 来源二：释放后未置空
int *p = (int*)malloc(sizeof(int));
free(p);
// 此时 p 仍然保存旧地址，是野指针！
*p = 10;  // 未定义行为

// 来源三：指针指向局部变量，该变量已出作用域
int* get_ptr() {
    int local = 42;
    return &local;  // local 在函数返回后销毁，返回的指针无效
}
```

### 如何避免野指针

```c
// 规则一：定义时立即初始化
int *p = NULL;

// 规则二：释放后立即置空
free(p);
p = NULL;

// 规则三：不返回局部变量的地址
// 使用 static 或动态分配代替
```

## 五、悬空指针（Dangling Pointer）

### 什么是悬空指针

悬空指针是指**曾经指向有效内存，但该内存已被释放**的指针。

```c
int *p = (int*)malloc(sizeof(int));
free(p);          // p 现在是悬空指针
// 此时再使用 *p 是未定义行为
```

### 典型场景

```c
// 场景一：free 后继续使用
free(p);
*p = 10;  // 错误！

// 场景二：返回局部变量地址
int* bad_func() {
    int temp = 100;
    return &temp;  // temp 已销毁
}

// 场景三：返回字符串字面量的局部指针
char* bad_str() {
    char buf[20] = "hello";
    return buf;  // buf 已销毁
}
```

## 六、指针初始化的最佳实践

```c
// 1. 始终初始化
int *p = NULL;

// 2. 使用前检查
if (p != NULL) {
    *p = 42;
}

// 3. 释放后置空
free(p);
p = NULL;

// 4. 检查 malloc 返回值
int *p = (int*)malloc(sizeof(int));
if (p == NULL) {
    // 内存分配失败处理
    fprintf(stderr, "内存分配失败\n");
    return -1;
}
```

## 七、关键要点总结

> **定义规范**
> - 使用 `类型 *变量名` 格式定义指针
> - 定义时立即初始化，至少赋值为 NULL
> - 区分指针类型和普通变量的混合定义

> **安全原则**
> - 未初始化的指针 = 野指针 = 定时炸弹
> - free 之后立即赋 NULL
> - 永远不要返回局部变量的地址
> - 解引用前检查指针是否为 NULL
