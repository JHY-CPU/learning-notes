# 指针与结构体

## 一、结构体指针基础

结构体指针是指**指向结构体变量的指针**。通过结构体指针可以访问结构体的成员。

```c
struct Student {
    char name[50];
    int age;
    float score;
};

int main() {
    struct Student stu = {"张三", 20, 85.5};
    struct Student *p = &stu;

    // 两种访问方式
    printf("姓名: %s\n", (*p).name);   // 方式一
    printf("年龄: %d\n", p->age);      // 方式二（推荐）
    printf("成绩: %.1f\n", p->score);

    return 0;
}
```

## 二、`->` 操作符

`->` 是访问结构体指针成员的专用操作符：

```
p->member  等价于  (*p).member
```

### 为什么推荐 `->`

```c
// 箭头操作符更清晰简洁
p->name       // 清晰
(*p).name     // 繁琐，需要括号

// 链式调用
p->next->data     // 清晰
(*(*p).next).data // 极其繁琐
```

## 三、结构体指针作为函数参数

### 传值 vs 传指针

```c
// 传值：会复制整个结构体（开销大）
void print_student(struct Student s) {
    printf("%s, %d, %.1f\n", s.name, s.age, s.score);
}

// 传指针：只传递一个地址（高效）
void print_student_p(const struct Student *s) {
    printf("%s, %d, %.1f\n", s->name, s->age, s->score);
}

// 通过指针修改结构体
void set_age(struct Student *s, int age) {
    s->age = age;
}
```

### 为什么用 const 指针

```c
// 输入参数：const 指针，只读
void display(const struct Student *s) {
    printf("%s\n", s->name);
    // s->age = 20;  // 编译错误
}

// 输出参数：普通指针，可写
void init(struct Student *s, const char *name, int age) {
    strncpy(s->name, name, 49);
    s->name[49] = '\0';
    s->age = age;
}
```

## 四、动态分配结构体

```c
struct Student *create_student(const char *name, int age, float score) {
    struct Student *s = (struct Student*)malloc(sizeof(struct Student));
    if (s == NULL) return NULL;

    strncpy(s->name, name, 49);
    s->name[49] = '\0';
    s->age = age;
    s->score = score;
    return s;
}

int main() {
    struct Student *stu = create_student("李四", 21, 90.0);
    if (stu != NULL) {
        printf("%s, %d\n", stu->name, stu->age);
        free(stu);
        stu = NULL;
    }
    return 0;
}
```

## 五、结构体指针与链表

### 链表节点定义

```c
struct Node {
    int data;
    struct Node *next;  // 指向下一个节点的指针
};
```

### 创建节点

```c
struct Node* create_node(int data) {
    struct Node *node = (struct Node*)malloc(sizeof(struct Node));
    if (node != NULL) {
        node->data = data;
        node->next = NULL;
    }
    return node;
}
```

### 头部插入

```c
void insert_front(struct Node **head, int data) {
    struct Node *new_node = create_node(data);
    if (new_node == NULL) return;
    new_node->next = *head;
    *head = new_node;
}
```

### 遍历链表

```c
void print_list(const struct Node *head) {
    const struct Node *current = head;
    while (current != NULL) {
        printf("%d -> ", current->data);
        current = current->next;
    }
    printf("NULL\n");
}
```

### 释放链表

```c
void free_list(struct Node *head) {
    struct Node *current = head;
    while (current != NULL) {
        struct Node *temp = current;
        current = current->next;
        free(temp);
    }
}
```

```c

// 初始化
```

## 九、关键要点总结

> **核心概念**
> - `->` 操作符访问结构体指针的成员
> - 传指针比传值更高效
> - const 指针参数保护数据不被修改

> **重要应用**
> - 链表：结构体中包含指向自身的指针
> - 动态结构体：malloc 创建，free 释放
> - 函数参数：用指针传递大型结构体

> **注意事项**
> - 自引用必须用指针，不能用值
> - 二级指针用于修改链表头节点
> - 动态分配后必须检查 NULL
