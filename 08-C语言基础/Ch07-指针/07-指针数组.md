# 指针数组

## 一、什么是指针数组

指针数组（Array of Pointers）是指**数组中的每个元素都是一个指针**。

```
定义格式：类型 *数组名[数组大小];
```

```c
int a = 10, b = 20, c = 30;
int *ptr_arr[3] = {&a, &b, &c};  // 3 个 int* 组成的数组

for (int i = 0; i < 3; i++) {
    printf("ptr_arr[%d] 指向的值: %d\n", i, *ptr_arr[i]);
}
```

### 内存布局

```
ptr_arr[0] -> a (值 10)
ptr_arr[1] -> b (值 20)
ptr_arr[2] -> c (值 30)
```

## 二、字符串指针数组

指针数组最常见的应用之一是存储多个字符串。

```c
#include <stdio.h>

int main() {
    const char *fruits[] = {
        "apple",
        "banana",
        "cherry",
        "date",
        "elderberry"
    };

    int n = sizeof(fruits) / sizeof(fruits[0]);
    for (int i = 0; i < n; i++) {
        printf("水果 %d: %s\n", i + 1, fruits[i]);
    }

    return 0;
}
```

### 与二维字符数组的对比

```c
// 方式一：指针数组（推荐）
const char *names[] = {"Alice", "Bob", "Charlie"};
// 每个字符串存储在只读区，数组只存储指针
// 节省内存，长度可变

// 方式二：二维字符数组
char names2[3][10] = {"Alice", "Bob", "Charlie"};
// 每行固定 10 字节，可能浪费空间
```

| 特性 | 指针数组 | 二维数组 |
|------|---------|---------|
| 内存分配 | 不连续 | 连续 |
| 长度灵活性 | 每个字符串长度任意 | 每行长度固定 |
| 可修改性 | 字符串字面量不可修改 | 可以修改 |
| 内存使用 | 更节省 | 可能浪费 |

## 三、`main` 函数的参数

命令行参数就是典型的指针数组应用。

```c
#include <stdio.h>

int main(int argc, char *argv[]) {
    // argc: 参数个数（包括程序名）
    // argv: 指针数组，每个元素指向一个参数字符串

    printf("参数个数: %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("argv[%d] = %s\n", i, argv[i]);
    }

    return 0;
}
```

运行示例：
```
$ ./program hello world 123
参数个数: 4
argv[0] = ./program
argv[1] = hello
argv[2] = world
argv[3] = 123
```

## 四、指针数组排序字符串

```c
#include <stdio.h>
#include <string.h>

int main() {
    const char *names[] = {"Charlie", "Alice", "Bob", "David"};
    int n = sizeof(names) / sizeof(names[0]);

    // 冒泡排序（交换指针，不移动字符串本身）
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (strcmp(names[j], names[j + 1]) > 0) {
                const char *temp = names[j];
                names[j] = names[j + 1];
                names[j + 1] = temp;
            }
        }
    }

    printf("排序后:\n");
    for (int i = 0; i < n; i++) {
        printf("  %s\n", names[i]);
    }

    return 0;
}
```

注意：这里交换的是指针，而非字符串内容，效率远高于移动字符串本身。

## 五、指向指针的指针数组

```c
int x = 1, y = 2, z = 3;
int *px = &x, *py = &y, *pz = &z;
int *arr[] = {px, py, pz};

// **arr 等价于 *(*arr) 等价于 *arr[0] 等价于 x
printf("%d\n", **arr);       // 1
printf("%d\n", *arr[1]);     // 2
printf("%d\n", **(arr + 2)); // 3
```

## 六、关键要点总结

> **核心概念**
> - 指针数组：`int *arr[n]`，n 个 int 指针组成的数组
> - 最常用于存储字符串集合
> - 比二维数组更灵活、更节省内存

> **应用场景**
> - 字符串集合的存储和排序
> - main 函数参数 argv
> - 查找表和跳转表

> **注意事项**
> - `int *arr[n]` 是指针数组（先结合 `[]`）
> - `int (*arr)[n]` 是数组指针（先结合 `*`）
