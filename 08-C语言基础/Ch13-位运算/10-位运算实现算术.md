# 10 - 位运算实现算术运算

## 概念

所有算术运算（加、减、乘、除）都可以**仅用位运算**来实现。这不仅是计算机硬件的基础原理，也是理解计算机如何真正执行计算的关键。

## 位运算实现加法

### 原理

二进制加法的核心：
- **不考虑进位的加法**：`a ^ b`（异或等价于无进位加法）
- **进位**：`(a & b) << 1`（与操作找出进位位，左移一位）

重复执行这两步，直到没有进位为止。

```
  5 + 3 的二进制加法过程：

  a = 0101  (5)
  b = 0011  (3)

  第1轮:
    无进位和 = 0101 ^ 0011 = 0110 (6)
    进位     = 0101 & 0011 = 0001 (1)，左移 = 0010 (2)

  第2轮:
    a = 0110, b = 0010
    无进位和 = 0110 ^ 0010 = 0100 (4)
    进位     = 0110 & 0010 = 0010 (2)，左移 = 0100 (4)

  第3轮:
    a = 0100, b = 0100
    无进位和 = 0100 ^ 0100 = 0000 (0)
    进位     = 0100 & 0100 = 0100 (4)，左移 = 1000 (8)

  第4轮:
    a = 0000, b = 1000
    无进位和 = 0000 ^ 1000 = 1000 (8)
    进位     = 0000 & 1000 = 0000 (0)，左移 = 0000

  进位为0，结束。结果 = 8
```

### 实现

```c
#include <stdio.h>

// 位运算实现加法
int add(int a, int b) {
    while (b != 0) {
        int carry = (a & b) << 1;  // 计算进位
        a = a ^ b;                  // 无进位加法
        b = carry;                  // 进位作为下一轮的加数
    }
    return a;
}

int main(void) {
    printf("5 + 3 = %d\n", add(5, 3));      // 8
    printf("10 + 20 = %d\n", add(10, 20));   // 30
    printf("0 + 0 = %d\n", add(0, 0));       // 0
    printf("-5 + 3 = %d\n", add(-5, 3));     // -2
    printf("100 + 200 = %d\n", add(100, 200)); // 300

    return 0;
}
```

## 位运算实现减法

### 原理

减法可以通过加法实现：`a - b = a + (-b)`

在补码系统中，`-b = ~b + 1`（取反加一）。

```c
#include <stdio.h>

int add(int a, int b) {
    while (b != 0) {
        int carry = (a & b) << 1;
        a = a ^ b;
        b = carry;
    }
    return a;
}

// 位运算实现减法
int subtract(int a, int b) {
    return add(a, add(~b, 1));  // a + (-b) = a + (~b + 1)
}

int main(void) {
    printf("10 - 3 = %d\n", subtract(10, 3));   // 7
    printf("5 - 8 = %d\n", subtract(5, 8));      // -3
    printf("100 - 100 = %d\n", subtract(100, 100)); // 0

    return 0;
}
```

## 位运算实现乘法

### 原理

乘法可以分解为**移位和加法**。类似手算乘法（竖式乘法）：

```
  5 * 3 的二进制乘法：

    101  (5)
  x 011  (3)
  -----
    101  (5 * 1，乘数第0位为1)
   000   (5 * 0，乘数第1位为0，左移1位)
  101    (5 * 1，乘数第2位为1，左移2位)
  -----
  1111   (15)
```

### 实现

```c
#include <stdio.h>

int add(int a, int b) {
    while (b != 0) {
        int carry = (a & b) << 1;
        a = a ^ b;
        b = carry;
    }
    return a;
}

// 位运算实现乘法（仅处理非负数）
unsigned int multiply(unsigned int a, unsigned int b) {
    unsigned int result = 0;

    while (b != 0) {
        if (b & 1) {          // 如果 b 的最低位为 1
            result = add(result, a);
        }
        a <<= 1;              // a 左移（乘以2）
        b >>= 1;              // b 右移（处理下一位）
    }

    return result;
}

int main(void) {
    printf("5 * 3 = %u\n", multiply(5, 3));       // 15
    printf("7 * 6 = %u\n", multiply(7, 6));       // 42
    printf("12 * 12 = %u\n", multiply(12, 12));   // 144
    printf("0 * 100 = %u\n", multiply(0, 100));   // 0

    return 0;
}
```

## 位运算实现除法

### 原理

除法通过**反复减法**（或移位减法）实现。类似于手算除法：

```c
#include <stdio.h>

int add(int a, int b) {
    while (b != 0) {
        int carry = (a & b) << 1;
        a = a ^ b;
        b = carry;
    }
    return a;
}

int subtract(int a, int b) {
    return add(a, add(~b, 1));
}

// 位运算实现除法（仅处理非负数）
unsigned int divide(unsigned int dividend, unsigned int divisor) {
    if (divisor == 0) return 0;  // 除零检查

    unsigned int quotient = 0;
    unsigned int remainder = dividend;

    // 找到 divisor 左移到不超过 dividend 的最大位置
    int shift = 31;
    while (shift >= 0) {
        if ((divisor << shift) <= remainder) {
            remainder = subtract(remainder, divisor << shift);
            quotient |= (1U << shift);
        }
        shift--;
    }

    return quotient;
}

int main(void) {
    printf("17 / 3 = %u\n", divide(17, 3));     // 5
    printf("100 / 7 = %u\n", divide(100, 7));   // 14
    printf("15 / 5 = %u\n", divide(15, 5));     // 3
    printf("7 / 10 = %u\n", divide(7, 10));     // 0

    return 0;
}
```

## 位运算实现取反（负数）

```c
// 取负：~x + 1
int negate(int x) {
    return add(~x, 1);
}

int main(void) {
    printf("-(42) = %d\n", negate(42));   // -42
    printf("-(-7) = %d\n", negate(-7));   // 7
    return 0;
}
```

## 位运算实现取模

```c
// 当 divisor 是 2 的幂时，取模非常简单
// a % (2^n) 等价于 a & ((1 << n) - 1)

#include <stdio.h>

int main(void) {
    unsigned int a = 123;

    printf("123 %% 8  = %u,  123 & 7  = %u\n", a % 8, a & 7);     // 3, 3
    printf("123 %% 16 = %u,  123 & 15 = %u\n", a % 16, a & 15);   // 11, 11
    printf("123 %% 32 = %u,  123 & 31 = %u\n", a % 32, a & 31);   // 27, 27

    return 0;
}
```

## 综合示例：完整计算器

```c
#include <stdio.h>

int bit_add(int a, int b) {
    while (b != 0) {
        int carry = (a & b) << 1;
        a = a ^ b;
        b = carry;
    }
    return a;
}

int bit_negate(int x) {
    return bit_add(~x, 1);
}

int bit_subtract(int a, int b) {
    return bit_add(a, bit_negate(b));
}

unsigned int bit_multiply(unsigned int a, unsigned int b) {
    unsigned int result = 0;
    while (b) {
        if (b & 1) result = bit_add(result, a);
        a <<= 1;
        b >>= 1;
    }
    return result;
}

int main(void) {
    int a = 42, b = 7;

    printf("%d + %d = %d\n", a, b, bit_add(a, b));
    printf("%d - %d = %d\n", a, b, bit_subtract(a, b));
    printf("%d * %d = %u\n", a, b, bit_multiply(a, b));

    return 0;
}
```

## 关键要点

- **加法**：异或（无进位加法）+ 与左移（进位），循环直到无进位
- **减法**：`a - b = a + (~b + 1)`，利用补码性质
- **乘法**：逐位检查乘数，移位后累加（类似竖式乘法）
- **除法**：移位减法，从高位开始尝试减去倍数
- 这些实现帮助理解计算机硬件如何执行算术运算
- 实际编程中应直接使用 `+` `-` `*` `/` 运算符，位运算实现仅用于学习
