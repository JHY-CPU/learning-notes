# 06 - 左移运算

## 概念

左移运算符 `<<` 将一个数的所有二进制位向左移动指定的位数，**右侧用 0 填充**。左移 n 位相当于乘以 2 的 n 次方。

### 运算示例

```
0000 1010 << 2
= 0010 1000

10（十进制）左移2位 = 40（十进制）
相当于 10 * 2^2 = 10 * 4 = 40
```

## 基本用法

```c
#include <stdio.h>

void print_binary_8(unsigned char n) {
    for (int i = 7; i >= 0; i--)
        printf("%d", (n >> i) & 1);
}

int main(void) {
    unsigned char a = 5;  // 00000101

    printf("原始值: ");
    print_binary_8(a);
    printf(" (%u)\n", a);

    printf("左移1位: ");
    print_binary_8(a << 1);
    printf(" (%u)\n", a << 1);  // 10

    printf("左移2位: ");
    print_binary_8(a << 2);
    printf(" (%u)\n", a << 2);  // 20

    printf("左移3位: ");
    print_binary_8(a << 3);
    printf(" (%u)\n", a << 3);  // 40

    return 0;
}
```

## 左移与乘法的关系

左移 n 位等价于乘以 2^n，这是位运算优化的基本原理。

```c
#include <stdio.h>

int main(void) {
    int n = 7;

    printf("%d * 2  = %d,  %d << 1 = %d\n", n, n * 2, n, n << 1);
    printf("%d * 4  = %d,  %d << 2 = %d\n", n, n * 4, n, n << 2);
    printf("%d * 8  = %d,  %d << 3 = %d\n", n, n * 8, n, n << 3);
    printf("%d * 16 = %d,  %d << 4 = %d\n", n, n * 16, n, n << 4);
    printf("%d * 32 = %d,  %d << 5 = %d\n", n, n * 32, n, n << 5);

    // 复杂的乘法可以拆解
    // n * 20 = n * (16 + 4) = (n << 4) + (n << 2)
    printf("%d * 20 = %d,  (%d<<4)+(%d<<2) = %d\n",
           n, n * 20, n, n, (n << 4) + (n << 2));

    return 0;
}
```

## 实际应用

### 1. 生成 2 的幂次

```c
#include <stdio.h>

int main(void) {
    // 1 << n 生成 2^n
    for (int i = 0; i < 16; i++) {
        printf("2^%d = %u\n", i, 1U << i);
    }

    // 常见用法：生成特定位置的位
    unsigned int bit0 = 1 << 0;  // 1     (00000001)
    unsigned int bit3 = 1 << 3;  // 8     (00001000)
    unsigned int bit7 = 1 << 7;  // 128   (10000000)

    return 0;
}
```

### 2. 设置位（替代乘法）

```c
#include <stdio.h>

int main(void) {
    // 设置特定位为 1
    unsigned int flags = 0;
    flags |= (1 << 2);   // 设置第2位
    flags |= (1 << 5);   // 设置第5位
    flags |= (1 << 7);   // 设置第7位
    printf("flags = %u (0x%02X)\n", flags, flags);  // 164 (0xA4)

    return 0;
}
```

### 3. 颜色/数据打包

```c
#include <stdio.h>

int main(void) {
    // 将颜色分量打包为 32 位 ARGB 值
    unsigned char alpha = 0xFF;
    unsigned char red   = 0xA5;
    unsigned char green = 0x3C;
    unsigned char blue  = 0x7B;

    unsigned int argb = (alpha << 24) | (red << 16) | (green << 8) | blue;
    printf("ARGB: 0x%08X\n", argb);  // 0xFFA53C7B

    return 0;
}
```

### 4. 构造掩码

```c
#include <stdio.h>

int main(void) {
    // 生成低 n 位全 1 的掩码
    int n = 5;
    unsigned int mask = (1U << n) - 1;
    printf("低%d位掩码: 0x%02X\n", n, mask);  // 0x1F（低5位全1）

    // 生成连续 n 个 1 的掩码（从第 start 位开始）
    int start = 3, width = 4;
    unsigned int field_mask = ((1U << width) - 1) << start;
    printf("从第%d位开始的%d位掩码: 0x%02X\n", start, width, field_mask);  // 0x78

    return 0;
}
```

## 移位溢出与未定义行为

### 1. 移位量 >= 类型宽度

当移位量大于或等于操作数类型的位宽时，行为是**未定义的**（Undefined Behavior）。

```c
#include <stdio.h>

int main(void) {
    int a = 1;

    // 以下行为未定义！
    // int b = a << 32;   // int 是32位，移32位是未定义行为
    // int c = a << 40;   // 更糟糕

    // 安全做法：确保移位量 < 类型位宽
    int shift = 31;  // 对于32位int，最大安全移位量是31
    int safe = a << shift;
    printf("1 << 31 = %d\n", safe);  // -2147483648（最小int值）

    return 0;
}
```

### 2. 有符号左移溢出

对有符号数进行左移可能导致溢出，产生未定义行为。

```c
#include <stdio.h>

int main(void) {
    // 使用无符号类型避免有符号溢出问题
    unsigned int a = 0x80000000;  // 最高位为1

    unsigned int b = a << 1;
    printf("0x80000000 << 1 = 0x%08X\n", b);  // 0x00000000（最高位丢失）

    // 注意：左移有符号数使其符号位改变是未定义行为
    // int x = 0x40000000;
    // int y = x << 1;  // 未定义：结果超过 int 范围

    return 0;
}
```

### 3. 安全的左移操作

```c
#include <stdio.h>
#include <limits.h>

// 安全的左移：检查是否会导致溢出
int safe_left_shift(int value, int shift, int *result) {
    if (shift < 0 || shift >= (int)sizeof(int) * 8) {
        return -1;  // 移位量非法
    }
    if (shift == 0) {
        *result = value;
        return 0;
    }
    // 检查是否会溢出
    if (value > (INT_MAX >> shift)) {
        return -2;  // 会溢出
    }
    *result = value << shift;
    return 0;
}

int main(void) {
    int result;
    if (safe_left_shift(1000000000, 2, &result) == 0) {
        printf("结果: %d\n", result);
    } else {
        printf("移位操作会导致溢出\n");
    }
    return 0;
}
```

## 关键要点

- `<<` 左移：所有位向左移动 n 位，右侧补 0
- 左移 n 位等价于乘以 2^n，且比乘法更快
- 移位量不能为负数，也不能 >= 操作数类型的位宽
- 有符号数左移溢出是未定义行为，建议使用无符号类型
- `1 << n` 是生成 2^n 的高效方式
- 左移常用于设置位、构建掩码、数据打包等操作
