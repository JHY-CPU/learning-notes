# 11 - 位计数问题

## 问题描述

位计数（Population Count / Hamming Weight）是指计算一个整数的二进制表示中**值为 1 的位的个数**。这是计算机科学中最基础也最常见的位操作问题之一。

## 方法一：逐位检查

最直观的方法：检查每一位是否为 1。

```c
#include <stdio.h>

// 方法1：逐位检查
int count_bits_v1(unsigned int n) {
    int count = 0;
    for (int i = 0; i < 32; i++) {
        if (n & (1U << i))
            count++;
    }
    return count;
}

// 方法2：右移逐位检查
int count_bits_v2(unsigned int n) {
    int count = 0;
    while (n) {
        count += (n & 1);
        n >>= 1;
    }
    return count;
}

int main(void) {
    unsigned int n = 0b10110101;
    printf("v1: %d 中有 %d 个1\n", n, count_bits_v1(n));  // 5
    printf("v2: %d 中有 %d 个1\n", n, count_bits_v2(n));  // 5
    return 0;
}
```

**时间复杂度**：O(位宽) = O(32) = O(1)

## 方法二：Brian Kernighan 算法

这是最优雅的算法之一，由 Brian Kernighan 提出。核心技巧：

> **`n & (n - 1)` 会清除 n 中最低位的 1**

```c
#include <stdio.h>

// Brian Kernighan 算法
// 每次 n & (n-1) 消除最低位的1，循环次数就是1的个数
int count_bits_kernighan(unsigned int n) {
    int count = 0;
    while (n) {
        n &= (n - 1);  // 清除最低位的1
        count++;
    }
    return count;
}

int main(void) {
    printf("%d\n", count_bits_kernighan(0b10110101));  // 5
    printf("%d\n", count_bits_kernighan(0xFF));         // 8
    printf("%d\n", count_bits_kernighan(0));            // 0
    return 0;
}
```

**原理演示**：

```
n = 10110101

第1次: n & (n-1) = 10110101 & 10110100 = 10110100  (消除最低位1)
第2次: n & (n-1) = 10110100 & 10110011 = 10110000
第3次: n & (n-1) = 10110000 & 10101111 = 10100000
第4次: n & (n-1) = 10100000 & 10011111 = 10000000
第5次: n & (n-1) = 10000000 & 01111111 = 00000000

循环5次，所以有5个1
```

**时间复杂度**：O(k)，k 是 1 的个数

## 方法三：分治法（查表 / 并行计算）

利用分治思想，将 32 位分成越来越小的组。

```c
#include <stdio.h>

int count_bits_parallel(unsigned int n) {
    // 第1步：每2位一组，统计1的个数
    // 0x55555555 = 01010101... (每2位中取第0位)
    n = n - ((n >> 1) & 0x55555555U);

    // 第2步：每4位一组，将相邻的2位组加起来
    // 0x33333333 = 00110011... (每4位中取低2位)
    n = (n & 0x33333333U) + ((n >> 2) & 0x33333333U);

    // 第3步：每8位一组
    n = (n + (n >> 4)) & 0x0F0F0F0FU;

    // 第4步：每16位一组
    n = n + (n >> 8);

    // 第5步：32位总和
    n = n + (n >> 16);

    return n & 0x3F;  // 结果最多32，6位足够
}

int main(void) {
    printf("%d\n", count_bits_parallel(0b10110101));  // 5
    printf("%d\n", count_bits_parallel(0xFFFFFFFF));   // 32
    return 0;
}
```

**时间复杂度**：O(log(位宽)) = O(5)，执行时间固定，无分支。

## 方法四：查表法

对于频繁计算的场景，预计算所有可能值。

```c
#include <stdio.h>

// 预计算表：每8位（一个字节）的1的个数
static const unsigned char bit_count_table[256] = {
    0,1,1,2,1,2,2,3,1,2,2,3,2,3,3,4,  // 0x00-0x0F
    1,2,2,3,2,3,3,4,2,3,3,4,3,4,4,5,  // 0x10-0x1F
    1,2,2,3,2,3,3,4,2,3,3,4,3,4,4,5,  // 0x20-0x2F
    2,3,3,4,3,4,4,5,3,4,4,5,4,5,5,6,  // 0x30-0x3F
    1,2,2,3,2,3,3,4,2,3,3,4,3,4,4,5,  // 0x40-0x4F
    2,3,3,4,3,4,4,5,3,4,4,5,4,5,5,6,  // 0x50-0x5F
    2,3,3,4,3,4,4,5,3,4,4,5,4,5,5,6,  // 0x60-0x6F
    3,4,4,5,4,5,5,6,4,5,5,6,5,6,6,7,  // 0x70-0x7F
    1,2,2,3,2,3,3,4,2,3,3,4,3,4,4,5,  // 0x80-0x8F
    2,3,3,4,3,4,4,5,3,4,4,5,4,5,5,6,  // 0x90-0x9F
    2,3,3,4,3,4,4,5,3,4,4,5,4,5,5,6,  // 0xA0-0xAF
    3,4,4,5,4,5,5,6,4,5,5,6,5,6,6,7,  // 0xB0-0xBF
    2,3,3,4,3,4,4,5,3,4,4,5,4,5,5,6,  // 0xC0-0xCF
    3,4,4,5,4,5,5,6,4,5,5,6,5,6,6,7,  // 0xD0-0xDF
    3,4,4,5,4,5,5,6,4,5,5,6,5,6,6,7,  // 0xE0-0xEF
    4,5,5,6,5,6,6,7,5,6,6,7,6,7,7,8   // 0xF0-0xFF
};

int count_bits_lookup(unsigned int n) {
    return bit_count_table[n & 0xFF]
         + bit_count_table[(n >> 8) & 0xFF]
         + bit_count_table[(n >> 16) & 0xFF]
         + bit_count_table[(n >> 24) & 0xFF];
}

int main(void) {
    printf("%d\n", count_bits_lookup(0b10110101));  // 5
    printf("%d\n", count_bits_lookup(0xFFFFFFFF));   // 32
    return 0;
}
```

**时间复杂度**：O(1)，只需 4 次查表和 3 次加法。

## C 标准库中的位计数

C23 标准引入了 `<stdbit.h>`，提供标准的位计数函数：

```c
// C23 标准（<stdbit.h>）
// stdc_popcount(x)  — 统计1的个数
// stdc_count_zeros(x) — 统计0的个数
```

GCC / Clang 内置函数：

```c
#include <stdio.h>

int main(void) {
    unsigned int n = 0b10110101;

    // GCC / Clang 内置函数
    printf("1的个数: %d\n", __builtin_popcount(n));     // 5
    printf("1的个数(64位): %d\n", __builtin_popcountll(0xFFFFFFFFFFFFFFFFULL)); // 64

    return 0;
}
```

## 应用场景

### 1. 计算汉明距离

两个等长字符串/整数之间对应位不同的个数。

```c
#include <stdio.h>

int hamming_distance(unsigned int a, unsigned int b) {
    return __builtin_popcount(a ^ b);
    // 不同的位在异或结果中为1，统计1的个数即可
}

int main(void) {
    printf("汉明距离: %d\n", hamming_distance(0b10110101, 0b11001010));  // 6
    return 0;
}
```

### 2. 判断是否为 2 的幂

```c
// n > 0 且 n & (n-1) == 0 则 n 是2的幂
// 因为2的幂只有一个1，n & (n-1) 消除这个1后为0
int is_power_of_two(unsigned int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
```

## 性能对比

| 方法 | 时间复杂度 | 特点 |
|------|-----------|------|
| 逐位检查 | O(32) | 简单易懂 |
| Brian Kernighan | O(k) | k为1的个数，最常用 |
| 分治并行 | O(5) | 无分支，适合SIMD |
| 查表法 | O(1) | 最快，需额外内存 |
| 编译器内置 | O(1) | 使用硬件指令，推荐 |

## 关键要点

- 位计数是统计整数二进制中 1 的个数的问题
- `n & (n-1)` 清除最低位的 1，是 Brian Kernighan 算法的核心
- 查表法以空间换时间，适合高频调用场景
- 实际项目中优先使用编译器内置函数（如 `__builtin_popcount`）
- 位计数在汉明距离、子集枚举、错误检测中有广泛应用
