# 27 - C23 位操作增强

## 概述

C23 标准引入了 `<stdbit.h>` 头文件，提供了**标准化的位操作函数**。在此之前，这些操作只能通过编译器内置函数（如 GCC 的 `__builtin_popcount`）或手动实现来完成。

## <stdbit.h> 头文件

C23 标准化了以下几类位操作：

### 1. 位计数函数

| 函数 | 功能 | 等效 GCC 内置 |
|------|------|---------------|
| `stdc_popcount(x)` | 统计 1 的个数 | `__builtin_popcount` |
| `stdc_count_zeros(x)` | 统计 0 的个数 | `类型位宽 - popcount` |

```c
#include <stdio.h>
#include <stdbit.h>

int main(void) {
    unsigned int x = 0b10110101;

    // C23 标准函数
    int ones = stdc_popcount(x);
    int zeros = stdc_count_zeros(x);

    printf("1的个数: %d\n", ones);   // 5
    printf("0的个数: %d\n", zeros);   // 27（32-5）

    return 0;
}
```

### 2. 前导零/一计数

| 函数 | 功能 | 等效 GCC 内置 |
|------|------|---------------|
| `stdc_leading_zeros(x)` | 最高位开始的 0 的个数 | `__builtin_clz` |
| `stdc_leading_ones(x)` | 最高位开始的 1 的个数 | 无直接等效 |

```c
#include <stdio.h>
#include <stdbit.h>

int main(void) {
    unsigned int x = 0b00010101;

    printf("前导零: %d\n", stdc_leading_zeros(x));  // 27
    printf("前导一: %d\n", stdc_leading_ones(0xFFFFFFFF)); // 32

    return 0;
}
```

### 3. 尾随零/一计数

| 函数 | 功能 | 等效 GCC 内置 |
|------|------|---------------|
| `stdc_trailing_zeros(x)` | 最低位开始的 0 的个数 | `__builtin_ctz` |
| `stdc_trailing_ones(x)` | 最低位开始的 1 的个数 | 无直接等效 |

```c
#include <stdio.h>
#include <stdbit.h>

int main(void) {
    unsigned int x = 0b10101100;

    printf("尾随零: %d\n", stdc_trailing_zeros(x));  // 2
    printf("尾随一: %d\n", stdc_trailing_ones(0b111));  // 3

    return 0;
}
```

### 4. 首位查找函数

| 函数 | 功能 | 说明 |
|------|------|------|
| `stdc_first_leading_one(x)` | 最高位的 1 的位置 | 返回值从 1 开始，0 表示无 |
| `stdc_first_trailing_one(x)` | 最低位的 1 的位置 | 返回值从 1 开始，0 表示无 |
| `stdc_first_leading_zero(x)` | 最高位的 0 的位置 | |
| `stdc_first_trailing_zero(x)` | 最低位的 0 的位置 | |

```c
#include <stdio.h>
#include <stdbit.h>

int main(void) {
    unsigned int x = 0b00101100;

    printf("最高位的1: 第%d位\n", stdc_first_leading_one(x));   // 6
    printf("最低位的1: 第%d位\n", stdc_first_trailing_one(x));  // 3

    return 0;
}
```

## 类型泛型版本

C23 提供了类型泛型宏，自动根据参数类型选择正确的函数：

```c
#include <stdio.h>
#include <stdbit.h>

int main(void) {
    unsigned char  c = 0xFF;
    unsigned short s = 0xFF00;
    unsigned int   i = 0xFFFF0000;
    unsigned long long ll = 0xFFFFFFFF00000000ULL;

    // 类型泛型宏自动适配类型
    printf("char:  %d 个1\n", stdc_popcount(c));   // 8
    printf("short: %d 个1\n", stdc_popcount(s));   // 8
    printf("int:   %d 个1\n", stdc_popcount(i));   // 16
    printf("long:  %d 个1\n", stdc_popcount(ll));  // 32

    return 0;
}
```

## C23 之前的替代方案

在 C23 之前，需要使用编译器内置函数或手动实现。

### GCC / Clang 内置函数

```c
#include <stdio.h>

int main(void) {
    unsigned int x = 0b10110101;

    // GCC/Clang 内置
    printf("popcount: %d\n", __builtin_popcount(x));
    printf("clz: %d\n", __builtin_clz(x));
    printf("ctz: %d\n", __builtin_ctz(x));
    printf("ffs: %d\n", __builtin_ffs(x));

    // 64位版本
    unsigned long long big = 0x8000000000000000ULL;
    printf("clzll: %d\n", __builtin_clzll(big));
    printf("popcountll: %d\n", __builtin_popcountll(0xFFFFFFFFFFFFFFFFULL));

    return 0;
}
```

### MSVC 内置函数

```c
#include <stdio.h>

// MSVC 需要包含 intrin.h
// #include <intrin.h>

int main(void) {
    // MSVC 内置函数（仅供参考）
    // __popcnt(x)    — 位计数
    // __lzcnt(x)     — 前导零
    // __tzcnt(x)     — 尾随零
    // _BitScanForward(&index, x) — 找最低位1
    // _BitScanReverse(&index, x) — 找最高位1

    printf("请在 MSVC 环境中测试\n");
    return 0;
}
```

### C11 及更早的手动实现

```c
#include <stdio.h>

// 手动实现 popcount（C11 兼容）
int my_popcount(unsigned int n) {
    n = n - ((n >> 1) & 0x55555555U);
    n = (n & 0x33333333U) + ((n >> 2) & 0x33333333U);
    n = (n + (n >> 4)) & 0x0F0F0F0FU;
    return (n * 0x01010101U) >> 24;
}

// 手动实现 clz
int my_clz(unsigned int n) {
    if (n == 0) return 32;
    int count = 0;
    if ((n & 0xFFFF0000) == 0) { count += 16; n <<= 16; }
    if ((n & 0xFF000000) == 0) { count += 8;  n <<= 8;  }
    if ((n & 0xF0000000) == 0) { count += 4;  n <<= 4;  }
    if ((n & 0xC0000000) == 0) { count += 2;  n <<= 2;  }
    if ((n & 0x80000000) == 0) { count += 1; }
    return count;
}

int main(void) {
    unsigned int x = 0b10110101;
    printf("popcount: %d\n", my_popcount(x));  // 5
    printf("clz: %d\n", my_clz(x));            // 24
    return 0;
}
```

## C23 其他位操作相关特性

### constexpr 关键字

C23 引入了 `constexpr`，可以在编译期求值位操作：

```c
#include <stdio.h>
#include <stdbit.h>

// C23 constexpr 示例（概念性）
// constexpr int BITS_IN_BYTE = 8;
// constexpr unsigned int MASK_LOW_8 = (1U << 8) - 1;

int main(void) {
    // 编译期求值
    enum { MASK = (1U << 4) - 1 };  // 0x0F
    printf("低4位掩码: 0x%X\n", MASK);

    return 0;
}
```

## 编译器支持情况

| 编译器 | `<stdbit.h>` 支持 | 版本要求 |
|--------|-------------------|----------|
| GCC | 部分支持 | GCC 14+ |
| Clang | 部分支持 | Clang 18+ |
| MSVC | 暂不支持 | - |

```c
// 检查是否支持 C23 位操作
#if defined(__STDC_VERSION__) && __STDC_VERSION__ >= 202311L
    #include <stdbit.h>
    #define HAS_STDBIT 1
#else
    #define HAS_STDBIT 0
#endif
```

## 关键要点

- C23 引入 `<stdbit.h>`，标准化了位操作函数
- 主要函数：`stdc_popcount`、`stdc_leading_zeros`、`stdc_trailing_zeros` 等
- 提供类型泛型宏，自动适配不同整数类型
- C23 之前使用编译器内置函数（GCC: `__builtin_*`，MSVC: `_BitScan*`）
- 编写可移植代码时，应提供手动实现作为后备方案
