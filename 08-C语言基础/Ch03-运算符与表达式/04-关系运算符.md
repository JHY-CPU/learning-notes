# 04 - 关系运算符

## 一、概述

关系运算符用于比较两个值的大小关系，共有6个：

| 运算符 | 名称 | 示例 | 含义 |
|--------|------|------|------|
| `>` | 大于 | `a > b` | a是否大于b |
| `<` | 小于 | `a < b` | a是否小于b |
| `>=` | 大于等于 | `a >= b` | a是否大于等于b |
| `<=` | 小于等于 | `a <= b` | a是否小于等于b |
| `==` | 等于 | `a == b` | a是否等于b |
| `!=` | 不等于 | `a != b` | a是否不等于b |

## 二、运算结果

关系运算的结果是 **`int` 类型**的值：
- 关系成立：结果为 `1`（真）
- 关系不成立：结果为 `0`（假）

```c
#include <stdio.h>

int main(void) {
    int a = 10, b = 20;

    printf("a > b  = %d\n", a > b);   // 0
    printf("a < b  = %d\n", a < b);   // 1
    printf("a >= b = %d\n", a >= b);  // 0
    printf("a <= b = %d\n", a <= b);  // 1
    printf("a == b = %d\n", a == b);  // 0
    printf("a != b = %d\n", a != b);  // 1

    return 0;
}
```

## 三、优先级

- `>` `<` `>=` `<=` 的优先级**高于** `==` `!=`。
- 所有关系运算符的优先级**低于**算术运算符。
- 所有关系运算符的优先级**高于**逻辑运算符。
- 结合性：**左结合**。

```c
// 等价于 (a + b) > (c + d)
int result = a + b > c + d;

// 等价于 ((a > b) == (c > d)) != 0
int result2 = a > b == c > d;

// 等价于 (a > b) && (c > d)
int result3 = a > b && c > d;
```

## 四、链式比较的陷阱

**C语言不支持数学中的链式写法！**

```c
int x = 5;

// 错误写法（语法正确但逻辑错误）
if (1 < x < 10) {
    // 这里永远为真！
    // 计算过程：1 < x 得到 1（真），然后 1 < 10 得到 1（真）
}

// 正确写法
if (x > 1 && x < 10) {
    // x在1到10之间
}
```

## 五、浮点数比较陷阱

浮点数由于精度问题，直接使用 `==` 比较可能产生意外结果。

```c
#include <stdio.h>
#include <math.h>

int main(void) {
    double a = 0.1 + 0.2;
    double b = 0.3;

    // 直接比较 —— 可能为假
    if (a == b) {
        printf("相等\n");
    } else {
        printf("不相等\n");  // 通常会执行这行
    }

    printf("a = %.20f\n", a);  // 0.30000000000000004441
    printf("b = %.20f\n", b);  // 0.29999999999999998890

    // 正确方式：比较差值是否在容差范围内
    double epsilon = 1e-9;
    if (fabs(a - b) < epsilon) {
        printf("在精度范围内相等\n");
    }

    return 0;
}
```

### 浮点比较推荐方法

```c
#include <math.h>

// 方式1：固定epsilon
int nearly_equal(double a, double b, double epsilon) {
    return fabs(a - b) < epsilon;
}

// 方式2：相对误差（适用于数值范围变化大的场景）
int nearly_equal_relative(double a, double b, double rel_eps) {
    double max_val = fabs(a) > fabs(b) ? fabs(a) : fabs(b);
    return fabs(a - b) <= max_val * rel_eps;
}
```

## 六、关系运算符在条件语句中的使用

```c
// if语句
if (score >= 60) {
    printf("及格\n");
}

// while循环
while (count < 10) {
    count++;
}

// 三目运算符
const char *result = (score >= 60) ? "及格" : "不及格";
```

## 七、关键要点

1. 关系运算的结果是 `int` 类型（`1` 或 `0`），不是布尔类型（C99之前没有 `_Bool`）。
2. `==` 和 `=` 容易混淆，建议将常量写在左边来避免笔误：`if (5 == x)`。
3. C语言**不支持链式比较**，如 `1 < x < 10` 需要写成 `x > 1 && x < 10`。
4. 浮点数比较应使用**容差法**，避免直接使用 `==`。
5. `>` `<` `>=` `<=` 的优先级高于 `==` `!=`。
6. 关系运算符可以链式用于逻辑表达式：`a > b && b > c`。
7. `char` 类型参与比较时使用的是其**数值**（ASCII码），而非字典序。
