# 17 - 掩码与标志位操作

## 一、概述

在系统编程中，经常需要使用一个整数的不同二进制位来表示多个**布尔标志（Flags）**。位运算提供了高效的标志位设置、清除、翻转和检测操作。

## 二、基本概念

### 2.1 掩码（Mask）

掩码是一个用于**选择性操作**特定二进制位的值。

```c
// 常用掩码定义方式
#define FLAG_READ    0x01  // 00000001
#define FLAG_WRITE   0x02  // 00000010
#define FLAG_EXEC    0x04  // 00000100
#define FLAG_APPEND  0x08  // 00001000

// 使用移位定义（更清晰）
#define FLAG_READ    (1 << 0)
#define FLAG_WRITE   (1 << 1)
#define FLAG_EXEC    (1 << 2)
#define FLAG_APPEND  (1 << 3)
```

### 2.2 四种基本操作

| 操作 | 运算符 | 示例 | 说明 |
|------|--------|------|------|
| 设置位 | `\|=` | `flags \|= MASK` | 将指定位置1 |
| 清除位 | `&= ~` | `flags &= ~MASK` | 将指定位置0 |
| 翻转位 | `^=` | `flags ^= MASK` | 将指定位取反 |
| 检测位 | `&` | `flags & MASK` | 检查指定位是否为1 |

## 三、详细操作示例

```c
#include <stdio.h>

#define FLAG_A (1 << 0)  // 0x01
#define FLAG_B (1 << 1)  // 0x02
#define FLAG_C (1 << 2)  // 0x04
#define FLAG_D (1 << 3)  // 0x08

void print_flags(unsigned int flags) {
    printf("flags = 0x%02X (", flags);
    if (flags & FLAG_A) printf("A");
    if (flags & FLAG_B) printf("B");
    if (flags & FLAG_C) printf("C");
    if (flags & FLAG_D) printf("D");
    if (flags == 0) printf("none");
    printf(")\n");
}

int main(void) {
    unsigned int flags = 0;

    // 设置标志位
    flags |= FLAG_A;
    print_flags(flags);  // flags = 0x01 (A)

    flags |= FLAG_C;
    print_flags(flags);  // flags = 0x05 (AC)

    // 清除标志位
    flags &= ~FLAG_A;
    print_flags(flags);  // flags = 0x04 (C)

    // 翻转标志位
    flags ^= FLAG_B;
    print_flags(flags);  // flags = 0x06 (BC)

    flags ^= FLAG_B;
    print_flags(flags);  // flags = 0x04 (C)

    // 检测标志位
    if (flags & FLAG_C) {
        printf("C标志位已设置\n");
    }

    if (!(flags & FLAG_A)) {
        printf("A标志位未设置\n");
    }

    // 设置多个标志位
    flags = FLAG_A | FLAG_B | FLAG_D;
    print_flags(flags);  // flags = 0x0B (ABD)

    // 清除多个标志位
    flags &= ~(FLAG_A | FLAG_D);
    print_flags(flags);  // flags = 0x02 (B)

    return 0;
}
```

## 四、检测多个标志位

```c
// 检测是否同时设置了多个标志（全部匹配）
if ((flags & (FLAG_A | FLAG_B)) == (FLAG_A | FLAG_B)) {
    printf("A和B都已设置\n");
}

// 检测是否设置了任一标志（至少一个匹配）
if (flags & (FLAG_A | FLAG_B)) {
    printf("A或B至少设置了一个\n");
}

// 检测是否没有设置任何指定标志
if (!(flags & (FLAG_A | FLAG_B))) {
    printf("A和B都未设置\n");
}

// 检测是否只设置了某个标志（排除其他）
if ((flags & FLAG_A) && !(flags & ~FLAG_A)) {
    printf("只设置了A标志\n");
}
```

## 五、结构体中的位域（Bit-field）

C语言支持在结构体中定义**位域**，可以更直观地操作特定位。

```c
#include <stdio.h>

struct Permissions {
    unsigned int read    : 1;  // 1位
    unsigned int write   : 1;  // 1位
    unsigned int execute : 1;  // 1位
    unsigned int reserved : 5; // 5位
};

int main(void) {
    struct Permissions perm = {0};

    perm.read = 1;
    perm.write = 1;

    if (perm.read) {
        printf("有读权限\n");
    }

    printf("size = %zu\n", sizeof(perm));  // 通常4字节

    return 0;
}
```

### 位域与手动位操作对比

```c
// 手动位操作方式
unsigned int perms = 0;
perms |= (1 << 0);   // 设置读权限
perms |= (1 << 1);   // 设置写权限
int can_read = (perms >> 0) & 1;

// 位域方式
struct { unsigned read:1; unsigned write:1; } p = {0};
p.read = 1;
p.write = 1;
int can_read2 = p.read;
```

**对比**：
- 位域：可读性好，编译器管理位操作。
- 手动位操作：更灵活，可移植性更好，效率可能更高。

## 六、实际应用示例

### 6.1 文件权限（类似Unix）

```c
#define S_IRUSR 0400  // 用户读
#define S_IWUSR 0200  // 用户写
#define S_IXUSR 0100  // 用户执行
#define S_IRGRP 0040  // 组读
#define S_IWGRP 0020  // 组写
#define S_IXGRP 0010  // 组执行
#define S_IROTH 0004  // 其他读
#define S_IWOTH 0002  // 其他写
#define S_IXOTH 0001  // 其他执行

int mode = S_IRUSR | S_IWUSR | S_IRGRP | S_IROTH;  // rw-r--r--

// 检查权限
if (mode & S_IWUSR) {
    printf("用户有写权限\n");
}
```

### 6.2 状态机

```c
#define STATE_IDLE    0x00
#define STATE_RUNNING 0x01
#define STATE_PAUSED  0x02
#define STATE_ERROR   0x04
#define STATE_DONE    0x08

unsigned int state = STATE_RUNNING | STATE_PAUSED;  // 运行但暂停

// 检测组合状态
if ((state & STATE_RUNNING) && (state & STATE_PAUSED)) {
    printf("程序处于暂停状态\n");
}
```

### 6.3 游戏中的道具系统

```c
#define ITEM_SWORD   (1 << 0)
#define ITEM_SHIELD  (1 << 1)
#define ITEM_POTION  (1 << 2)
#define ITEM_KEY     (1 << 3)
#define ITEM_MAP     (1 << 4)

unsigned int inventory = 0;

// 拾取道具
inventory |= ITEM_SWORD;
inventory |= ITEM_POTION;

// 使用药水
if (inventory & ITEM_POTION) {
    printf("使用了药水\n");
    inventory &= ~ITEM_POTION;  // 用完移除
}

// 检查是否有钥匙
if (inventory & ITEM_KEY) {
    printf("可以打开门\n");
} else {
    printf("需要先找到钥匙\n");
}
```

## 七、关键要点

1. 使用 `#define` 或移位 `(1 << n)` 定义掩码常量。
2. **设置位**用 `|=`，**清除位**用 `&= ~`，**翻转位**用 `^=`，**检测位**用 `&`。
3. 检测多个标志是否全部设置：`(flags & mask) == mask`。
4. 检测多个标志是否至少设置一个：`(flags & mask) != 0`。
5. 位域提供了更直观的位操作方式，但可移植性不如手动位操作。
6. 使用八进制（`0400`）或十六进制（`0x01`）表示掩码更直观。
7. 标志位操作在权限管理、状态机、配置选项等场景中非常常见。
