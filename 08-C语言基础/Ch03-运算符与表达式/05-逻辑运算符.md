# 05 - 逻辑运算符

## 一、概述

C语言提供三种逻辑运算符：

| 运算符 | 名称 | 操作数 | 说明 |
|--------|------|--------|------|
| `&&` | 逻辑与（AND） | 双目 | 两个操作数都为真时结果为真 |
| `\|\|` | 逻辑或（OR） | 双目 | 至少一个操作数为真时结果为真 |
| `!` | 逻辑非（NOT） | 单目 | 取反操作数的真值 |

## 二、真值判断规则

在C语言中，**零值为假，非零值为真**：

- **假（false）**：`0`、`0.0`、`'\0'`、`NULL`、空指针
- **真（true）**：任何非零值

逻辑运算的结果是 `int` 类型的 `1`（真）或 `0`（假）。

## 三、真值表

### 3.1 逻辑与 `&&`

| a | b | a && b |
|---|---|--------|
| 真 | 真 | **1** |
| 真 | 假 | 0 |
| 假 | 真 | 0 |
| 假 | 假 | 0 |

### 3.2 逻辑或 `||`

| a | b | a \|\| b |
|---|---|----------|
| 真 | 真 | **1** |
| 真 | 假 | **1** |
| 假 | 真 | **1** |
| 假 | 假 | 0 |

### 3.3 逻辑非 `!`

| a | !a |
|---|-----|
| 真 | 0 |
| 假 | 1 |

## 四、基本用法示例

```c
#include <stdio.h>

int main(void) {
    int a = 5, b = 0, c = 10;

    // 逻辑与
    printf("a && c = %d\n", a && c);  // 1（5和10都非零）
    printf("a && b = %d\n", a && b);  // 0（b为0，即假）

    // 逻辑或
    printf("a || b = %d\n", a || b);  // 1（a非零，为真）
    printf("b || 0 = %d\n", b || 0);  // 0（两个都为假）

    // 逻辑非
    printf("!a = %d\n", !a);    // 0（5非零取反为假）
    printf("!b = %d\n", !b);    // 1（0取反为真）
    printf("!!a = %d\n", !!a);  // 1（双重否定）

    return 0;
}
```

## 五、短路求值（Short-circuit Evaluation）

**这是逻辑运算符最重要的特性之一。**

### 5.1 `&&` 的短路特性

如果左操作数为假，**右操作数不会被求值**（因为结果已经确定为假）。

```c
int i = 0;

// 右边的 i++ 不会执行，因为左边 i == 0 为假
if (i != 0 && i++ > 0) {
    // 不会进入
}
printf("i = %d\n", i);  // 0（i++未执行）
```

### 5.2 `||` 的短路特性

如果左操作数为真，**右操作数不会被求值**（因为结果已经确定为真）。

```c
int i = 1;

// 右边的 i++ 不会执行，因为左边 i != 0 为真
if (i != 0 || i++ > 0) {
    // 进入
}
printf("i = %d\n", i);  // 1（i++未执行）
```

### 5.3 利用短路做边界检查

```c
// 经典用法：防止数组越界
if (index < size && array[index] == target) {
    // 安全：先检查index是否合法
}

// 经典用法：防止空指针解引用
if (ptr != NULL && ptr->value > 0) {
    // 安全：先检查ptr非空
}

// 经典用法：除零检查
if (denominator != 0 && numerator / denominator > 10) {
    // 安全：先检查分母非零
}

// 经典用法：字符串操作安全检查
if (str != NULL && str[0] != '\0') {
    // 安全：先检查指针，再检查内容
}
```

## 六、优先级

```
逻辑非 !  >  算术运算符  >  关系运算符  >  &&  >  ||  >  赋值运算符
```

```c
// ! 优先级最高（单目运算符）
!a > b    // 等价于 (!a) > b
a && b || c && d  // 等价于 (a && b) || (c && d)
```

## 七、综合示例

```c
#include <stdio.h>

int main(void) {
    int age = 25;
    int has_id = 1;
    int is_vip = 0;

    // 复合条件判断
    if (age >= 18 && has_id) {
        printf("可以入场\n");
    }

    // VIP或者有票都可以入场
    int has_ticket = 0;
    if (is_vip || has_ticket) {
        printf("VIP通道入场\n");
    } else {
        printf("请购票入场\n");
    }

    // 组合逻辑：年龄在18到65之间
    if (age >= 18 && age <= 65) {
        printf("工作年龄段\n");
    }

    // 取反：不是VIP
    if (!is_vip) {
        printf("普通客户\n");
    }

    return 0;
}
```

## 八、关键要点

1. 逻辑运算的结果是 `int` 类型的 `0` 或 `1`。
2. **零值为假，非零值为真**——这是C语言的核心规则。
3. `&&` 和 `||` 具有**短路求值**特性：一旦结果确定，右操作数不再求值。
4. 短路求值可以用来实现安全的**边界检查**和**空指针保护**。
5. `!` 的优先级很高（与 `++`、`sizeof` 同级），`&&` 的优先级高于 `||`。
6. 不要将逻辑运算符（`&&`、`||`）与位运算符（`&`、`|`）混淆。
7. `&&` 和 `||` 保证从左到右求值，且中间有序列点。
8. C99引入了 `<stdbool.h>`，可以使用 `bool`、`true`、`false`，但底层仍是 `int`。
