# 09 - 逗号运算符

## 一、概述

逗号运算符 `,` 是C语言中**优先级最低**的运算符，用于将多个表达式连接成一个表达式。

### 语法格式

```
表达式1, 表达式2, 表达式3, ..., 表达式n
```

### 求值规则

- 从左到右依次求值每个表达式。
- **整个逗号表达式的值等于最后一个表达式的值**。
- 前面所有表达式的副作用（如赋值、自增等）都会生效。

## 二、基本用法

```c
#include <stdio.h>

int main(void) {
    int a, b, result;

    // 逗号表达式的值是最后一个表达式的值
    result = (a = 1, b = 2, a + b);
    printf("result = %d\n", result);  // 3
    printf("a = %d, b = %d\n", a, b);  // a = 1, b = 2

    // 每个表达式都会被求值
    int x = 0;
    int y = (x = 5, x + 3, x * 2);
    printf("y = %d, x = %d\n", y, x);  // y = 10, x = 5

    return 0;
}
```

## 三、在 for 循环中使用

逗号运算符最常见的用法是在 `for` 循环的初始化和更新部分。

```c
#include <stdio.h>

int main(void) {
    // 同时初始化和更新两个变量
    for (int i = 0, j = 10; i < j; i++, j--) {
        printf("i = %d, j = %d\n", i, j);
    }
    // 输出:
    // i = 0, j = 10
    // i = 1, j = 9
    // i = 2, j = 8
    // i = 3, j = 7
    // i = 4, j = 6

    // 遍历两个数组
    int arr1[] = {1, 2, 3, 4, 5};
    int arr2[] = {5, 4, 3, 2, 1};
    for (int i = 0, j = 4; i < 5; i++, j--) {
        printf("arr1[%d]=%d, arr2[%d]=%d\n", i, arr1[i], j, arr2[j]);
    }

    return 0;
}
```

## 四、逗号表达式的值

```c
int a = 10;
int b = (a = 20, a + 5);  // b = 25（最后一个表达式 a+5 的值）
printf("b = %d\n", b);    // 25

// 逗号表达式的类型是最后一个表达式的类型
double d = (1, 2, 3.14);  // d = 3.14
printf("d = %f\n", d);
```

## 五、使用场景

### 5.1 在函数调用中（注意区分）

```c
// 注意：函数参数之间的逗号不是逗号运算符，而是分隔符
printf("%d %d\n", a, b);  // 这里 a 和 b 是两个独立的参数

// 如果想在参数中使用逗号表达式，必须加括号
printf("%d\n", (a = 1, b = 2, a + b));  // 输出3
// 不加括号会被解析为3个参数
```

### 5.2 在 while 循环中

```c
int ch;
while ((ch = getchar()) != EOF, ch != 'q') {
    // 先读取字符，再判断是否为'q'
    putchar(ch);
}
```

### 5.3 同时执行多个操作

```c
// 交换两个变量（利用逗号表达式组合多个操作）
int a = 10, b = 20;
int temp;
(temp = a, a = b, b = temp);
printf("a = %d, b = %d\n", a, b);  // a = 20, b = 10
```

## 六、逗号运算符 vs 函数参数分隔符

```c
// 函数参数中的逗号是分隔符，不是运算符！
func(a, b, c);       // 三个参数

// 逗号运算符——整个表达式作为第一个参数
func((a, b, c));     // 一个参数，值为c

// 逗号运算符+普通参数
func((a, b), c);     // 两个参数：第一个是b的值，第二个是c的值
```

## 七、逗号表达式在宏中的陷阱

```c
// 危险的宏定义
#define SWAP(a, b)  temp = a, a = b, b = temp

// 如果这样使用：
if (condition)
    SWAP(x, y);  // 展开为：temp = x, x = y, y = temp;
// 只有 temp = x 在if内部，后面两条语句在if外部！

// 安全的写法（用do-while包装）
#define SWAP_SAFE(a, b) do { temp = a; a = b; b = temp; } while(0)
```

## 八、关键要点

1. 逗号运算符是C语言中**优先级最低**的运算符。
2. 逗号表达式的值等于**最后一个表达式**的值。
3. 求值顺序**从左到右**，每个表达式都会被求值。
4. 最常见的用法是在 `for` 循环的初始化和更新部分。
5. 函数参数中的逗号是**分隔符**，不是逗号运算符。
6. 在宏定义中使用逗号表达式时要特别小心，建议用 `do-while` 包装。
7. 过度使用逗号表达式会降低代码可读性，应适度使用。
8. 逗号运算符保证有序列点——前面的表达式副作用完成后才执行后面的表达式。
