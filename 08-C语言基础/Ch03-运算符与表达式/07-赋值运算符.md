# 07 - 赋值运算符

## 一、概述

赋值运算符用于将一个值存储到变量中。C语言提供**基本赋值**和**复合赋值**两大类。

## 二、基本赋值运算符 `=`

```c
int a = 10;        // 声明并初始化
a = 20;            // 重新赋值
int b = a;         // 将a的值赋给b
```

### 2.1 赋值表达式的值

赋值本身是一个**表达式**，其值是赋给左操作数的值。

```c
int a, b, c;

// 赋值表达式的值就是被赋的值
a = (b = (c = 5));   // c=5的值是5，b=5的值是5，a=5的值是5
printf("%d %d %d\n", a, b, c);  // 5 5 5

// 利用赋值表达式的值
int x;
if ((x = getchar()) != EOF) {
    printf("读入字符: %c\n", x);
}
```

### 2.2 赋值的类型转换

赋值时，右操作数的值会被**自动转换**为左操作数的类型。

```c
int i = 3.14;      // i = 3（浮点截断为整数）
double d = 42;     // d = 42.0（整数提升为浮点）
char c = 65;       // c = 'A'（ASCII码65）
unsigned u = -1;   // u = 4294967295（有符号到无符号转换）
```

## 三、复合赋值运算符

复合赋值运算符将一个运算与赋值结合起来。

### 3.1 算术复合赋值

| 运算符 | 等价形式 | 示例 |
|--------|----------|------|
| `+=` | `a = a + b` | `a += 5` |
| `-=` | `a = a - b` | `a -= 5` |
| `*=` | `a = a * b` | `a *= 2` |
| `/=` | `a = a / b` | `a /= 2` |
| `%=` | `a = a % b` | `a %= 3` |

```c
#include <stdio.h>

int main(void) {
    int a = 10;

    a += 5;   printf("a += 5:  %d\n", a);  // 15
    a -= 3;   printf("a -= 3:  %d\n", a);  // 12
    a *= 2;   printf("a *= 2:  %d\n", a);  // 24
    a /= 4;   printf("a /= 4:  %d\n", a);  // 6
    a %= 4;   printf("a %%= 4: %d\n", a);  // 2

    return 0;
}
```

### 3.2 位运算复合赋值

| 运算符 | 等价形式 | 示例 |
|--------|----------|------|
| `&=` | `a = a & b` | `a &= mask` |
| `\|=` | `a = a \| b` | `a \|= flag` |
| `^=` | `a = a ^ b` | `a ^= mask` |
| `<<=` | `a = a << b` | `a <<= 2` |
| `>>=` | `a = a >> b` | `a >>= 2` |

```c
unsigned int flags = 0xFF;

flags &= 0x0F;   // 保留低4位：0x0F
printf("flags &= 0x0F: 0x%X\n", flags);  // 0x0F

flags |= 0xF0;   // 设置高4位
printf("flags |= 0xF0: 0x%X\n", flags);  // 0xFF

flags ^= 0xFF;   // 翻转所有位
printf("flags ^= 0xFF: 0x%X\n", flags);  // 0x00

int n = 1;
n <<= 4;          // 左移4位：1 * 16 = 16
printf("n <<= 4: %d\n", n);  // 16
n >>= 2;          // 右移2位：16 / 4 = 4
printf("n >>= 2: %d\n", n);  // 4
```

## 四、赋值的右结合性

赋值运算符是**右结合**的，这意味着连续赋值从右向左进行。

```c
int a, b, c;
a = b = c = 100;
// 等价于：a = (b = (c = 100))
// 执行顺序：c = 100 -> b = c -> a = b

printf("%d %d %d\n", a, b, c);  // 100 100 100
```

## 五、赋值与初始化的区别

```c
// 初始化：声明时赋予初始值（只执行一次）
int x = 10;

// 赋值：修改已有变量的值（可以多次执行）
x = 20;
x = 30;

// 注意：数组和字符串的初始化不是赋值
int arr[5] = {1, 2, 3, 4, 5};  // 初始化，正确
// int arr[5];
// arr = {1, 2, 3, 4, 5};      // 赋值，错误！数组名不可修改
```

## 六、常见应用场景

```c
// 1. 累加器
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum += i;   // 等价于 sum = sum + i
}

// 2. 计数器
int count = 0;
while (condition) {
    count++;
}

// 3. 字符串拼接模拟
char dest[100] = "Hello";
int len = 5;
dest[len++] = ' ';
dest[len++] = 'W';
dest[len] = '\0';

// 4. 位标志操作
unsigned int perms = 0;
perms |= 0x01;   // 设置读权限
perms |= 0x02;   // 设置写权限
perms &= ~0x01;  // 清除读权限
```

## 七、关键要点

1. `=` 是赋值，`==` 是比较——这是初学者最常见的混淆。
2. 赋值是**表达式**，有返回值（被赋的值），因此可以链式赋值。
3. 赋值运算符是**右结合**的，优先级仅高于逗号运算符。
4. 复合赋值运算符 `a op= b` 等价于 `a = a op b`，但 `a` 只被求值一次。
5. 赋值时会执行**隐式类型转换**，需要注意精度丢失和溢出问题。
6. 数组名、字符串字面量不能作为赋值的左操作数。
7. 左操作数必须是**可修改的左值**——不能对 `const` 变量、字面量赋值。
