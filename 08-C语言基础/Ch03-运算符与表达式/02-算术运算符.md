# 02 - 算术运算符

## 一、概述

C语言提供五种基本算术运算符：

| 运算符 | 名称 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 减法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法 | `7 / 3` | `2`（整数除法） |
| `%` | 取余（取模） | `7 % 3` | `1` |

它们都是**双目运算符**（需要两个操作数），且均为**左结合**。

## 二、加减乘法

加法、减法和乘法的用法与数学中一致，没有特殊陷阱。

```c
#include <stdio.h>

int main(void) {
    int a = 10, b = 3;

    printf("a + b = %d\n", a + b);   // 13
    printf("a - b = %d\n", a - b);   // 7
    printf("a * b = %d\n", a * b);   // 30

    return 0;
}
```

### 注意事项

- `+` 和 `-` 也可以作为**单目运算符**表示正号和负号：`+5`、`-a`。
- 整数运算可能溢出（overflow），需要注意值域范围。

## 三、除法运算符 `/`

除法是算术运算中最需要关注的运算符，因为**操作数的类型决定了结果**。

### 3.1 整数除法

当两个操作数都是整数时，执行**整数除法**，结果**截断小数部分**（向零取整）。

```c
#include <stdio.h>

int main(void) {
    printf("7 / 3 = %d\n", 7 / 3);      // 2（不是2.333...）
    printf("-7 / 3 = %d\n", -7 / 3);    // -2（向零取整）
    printf("7 / -3 = %d\n", 7 / -3);    // -2
    printf("-7 / -3 = %d\n", -7 / -3);  // 2

    return 0;
}
```

### 3.2 浮点除法

当至少一个操作数是浮点数时，执行**浮点除法**，结果保留小数部分。

```c
#include <stdio.h>

int main(void) {
    printf("7.0 / 3 = %f\n", 7.0 / 3);    // 2.333333
    printf("7 / 3.0 = %f\n", 7 / 3.0);    // 2.333333
    printf("7.0 / 3.0 = %f\n", 7.0 / 3.0);// 2.333333

    // 类型转换实现浮点除法
    int a = 7, b = 3;
    printf("(double)a / b = %f\n", (double)a / b);  // 2.333333

    return 0;
}
```

### 3.3 除零错误

**整数除以零是未定义行为（Undefined Behavior）**，程序会崩溃。浮点数除以零的结果是 `inf` 或 `-inf`（IEEE 754）。

```c
// 整数除零 —— 未定义行为，程序崩溃
// int x = 10 / 0;

// 浮点除零 —— 结果为inf
double y = 10.0 / 0.0;   // inf
double z = -10.0 / 0.0;  // -inf
```

## 四、取余运算符 `%`

取余运算符（也叫取模运算符）返回整数除法的**余数**。

### 4.1 基本用法

```c
#include <stdio.h>

int main(void) {
    printf("7 %% 3 = %d\n", 7 % 3);    // 1
    printf("10 %% 5 = %d\n", 10 % 5);  // 0（整除）
    printf("3 %% 7 = %d\n", 3 % 7);    // 3

    return 0;
}
```

### 4.2 取余的符号规则

C99/C11标准规定：**取余结果的符号与被除数（左操作数）相同**。

```c
printf("7 %% 3 = %d\n", 7 % 3);    // 1
printf("-7 %% 3 = %d\n", -7 % 3);  // -1（符号与-7一致）
printf("7 %% -3 = %d\n", 7 % -3);  // 1（符号与7一致）
printf("-7 %% -3 = %d\n", -7 % -3);// -1（符号与-7一致）
```

### 4.3 取余的限制

- `%` 的两个操作数**必须都是整数类型**。
- 不能对浮点数使用 `%`（需要使用 `fmod()` 函数）。
- 右操作数不能为零。

```c
#include <math.h>

// 浮点数取余使用 fmod()
double r = fmod(7.5, 2.3);  // 0.6
```

## 五、常见应用场景

```c
// 1. 判断奇偶
if (n % 2 == 0) {
    printf("%d 是偶数\n", n);
}

// 2. 取某一位数字
int num = 12345;
int last_digit = num % 10;  // 5

// 3. 循环数组索引
int index = (current + 1) % array_size;

// 4. 判断整除
if (year % 4 == 0 && (year % 100 != 0 || year % 400 == 0)) {
    printf("%d 是闰年\n", year);
}
```

## 六、关键要点

1. 整数除法会**截断小数**，不会四舍五入。
2. 要获得浮点除法结果，至少一个操作数必须是浮点类型。
3. `%` 只能用于整数，结果符号与被除数一致。
4. 整数除零是**未定义行为**，务必在运算前检查除数。
5. 注意整数运算的**溢出**问题，必要时使用更大的类型。
6. 如果需要四舍五入，可以使用 `(int)(x + 0.5)` 或 `round()` 函数。
