# 21 - 左值与右值

## 一、基本概念

### 1.1 左值（Lvalue）

**左值**（Left Value）是表示**存储在内存中的对象**的表达式，它有一个可识别的**存储位置**。左值可以出现在赋值运算符的**左边**。

```c
int x = 10;    // x 是左值
x = 20;        // x 可以出现在 = 左边

int arr[5];
arr[2] = 30;   // arr[2] 是左值

struct Point { int x; int y; };
struct Point p;
p.x = 5;       // p.x 是左值

int *ptr = &x;
*ptr = 100;    // *ptr 是左值
```

### 1.2 右值（Rvalue）

**右值**（Right Value）是**不是左值**的表达式，通常是临时的值或字面量，不能出现在赋值运算符的左边。

```c
int x = 10;     // 10 是右值
int y = x + 5;  // x + 5 的结果是右值

// 以下都是右值
42              // 整数字面量
3.14            // 浮点字面量
"hello"         // 字符串字面量（特殊：类型是char[]，可隐式转为char*）
x + y           // 算术表达式的结果
func()          // 函数返回值（非指针/数组类型时）
```

## 二、左值的分类

### 2.1 可修改的左值（Modifiable Lvalue）

可以被修改的左值，即可以出现在赋值运算符左边的表达式。

```c
int x = 10;
x = 20;         // x 是可修改的左值

int arr[5];
arr[0] = 1;     // arr[0] 是可修改的左值
```

### 2.2 不可修改的左值

```c
const int c = 10;
c = 20;         // 错误！c 是左值但不可修改（const限定）

// 数组名也是左值，但不可修改
int arr[5];
arr = NULL;     // 错误！arr 不可修改
```

## 三、哪些表达式是左值

```c
// 1. 变量名
int x;
x = 5;          // 左值

// 2. 解引用表达式
int *p = &x;
*p = 10;        // 左值

// 3. 数组下标表达式
int arr[5];
arr[0] = 1;     // 左值
0[arr] = 1;     // 也是左值（等价写法，arr[0] == *(arr+0) == *(0+arr) == 0[arr]）

// 4. 结构体/联合体成员（通过 . 或 ->）
struct S { int a; };
struct S s;
s.a = 5;        // 左值
(&s)->a = 5;    // 左值

// 5. 字符串字面量（特殊）
"hello"[0] = 'H';  // 合法但不推荐，字符串字面量是左值但修改是未定义行为

// 6. 逗号表达式（如果最右边是左值）
int a;
(a = 1, a) = 2;  // (a = 1, a) 是左值
```

## 四、哪些表达式是右值

```c
// 1. 字面量（除字符串字面量外）
42 = x;         // 错误！42 不是左值

// 2. 算术表达式的结果
int a = 5, b = 3;
a + b = 10;     // 错误！a + b 不是左值

// 3. 函数调用返回非引用类型
int func(void) { return 42; }
func() = 10;    // 错误！func() 不是左值

// 4. 关系和逻辑表达式的结果
(a > b) = 1;    // 错误！

// 5. 强制类型转换的结果
(int)3.14 = 5;  // 错误！

// 6. 赋值表达式的结果（在C中是右值！注意与C++不同）
int x;
(x = 5) = 10;   // C中错误！x = 5 的结果是右值
                // C++中合法！因为C++的赋值表达式返回左值
```

## 五、C语言中左值和右值的特殊规则

### 5.1 数组与指针的转换

```c
int arr[5];

// 数组名在大多数表达式中会"退化"为指向首元素的指针（右值）
int *p = arr;   // arr 转为 &arr[0]

// 但以下情况数组名不退化
sizeof(arr);    // 返回整个数组的大小
&arr;           // 返回指向数组的指针（类型 int (*)[5]）

// arr 本身不可修改
arr = NULL;     // 错误！arr 不可修改
```

### 5.2 函数指针

```c
void func(void) { }

// 函数名会自动转为函数指针
void (*fp)(void) = func;  // func 转为 &func

// 但函数名可以调用
func();         // 合法
(*func)();      // 也合法（先解引用再调用）
```

### 5.3 const 左值

```c
const int ci = 10;

// ci 是左值，但不可修改
ci = 20;            // 错误！

// 可以用 const 左值初始化其他值
int x = ci;         // 合法（读取ci的值）

// 对 const 对象取地址是合法的
const int *p = &ci; // 合法
```

## 六、C语言 vs C++ 的区别

```c
// C语言：赋值表达式的结果是右值
int x;
(x = 5) = 10;   // C中：错误！(x = 5)是右值

// C++：赋值表达式的结果是左值
// int x;
// (x = 5) = 10;  // C++中：合法！x最终为10

// C语言：条件表达式的结果是右值（除非两个分支都是同一左值）
int a, b;
((1) ? a : b) = 5;  // C中：错误
                     // C++中：合法
```

## 七、关键要点

1. **左值**表示内存中的对象（有存储位置），**右值**是临时值或字面量。
2. 变量名、解引用、数组下标、结构体成员是左值。
3. 算术表达式结果、函数返回值（非指针）、字面量是右值。
4. 赋值运算符的左操作数**必须是可修改的左值**。
5. `const` 左值是不可修改的左值。
6. C语言中赋值表达式的结果是**右值**（与C++不同）。
7. 数组名在大多数表达式中退化为指向首元素的指针。
8. 理解左值和右值对于掌握类型系统和指针非常重要。
