# 08 - 条件运算符

## 一、概述

条件运算符 `?:` 是C语言中**唯一的三目运算符**（需要三个操作数），也称为**三元运算符**或**三目条件运算符**。

### 语法格式

```
条件表达式 ? 表达式2 : 表达式3
```

**执行逻辑**：
- 如果条件表达式的值为**真（非零）**，则整个表达式的值为 `表达式2` 的值。
- 如果条件表达式的值为**假（零）**，则整个表达式的值为 `表达式3` 的值。

## 二、基本用法

```c
#include <stdio.h>

int main(void) {
    int a = 10, b = 20;

    // 求两个数的最大值
    int max = (a > b) ? a : b;
    printf("max = %d\n", max);  // 20

    // 判断奇偶
    int n = 7;
    printf("%d 是%s数\n", n, (n % 2 == 0) ? "偶" : "奇");

    // 绝对值
    int x = -15;
    int abs_x = (x >= 0) ? x : -x;
    printf("|%d| = %d\n", x, abs_x);  // 15

    return 0;
}
```

## 三、与 if-else 的对比

条件运算符可以看作简化版的 `if-else`：

```c
// if-else 写法
if (a > b) {
    max = a;
} else {
    max = b;
}

// 条件运算符写法（等价）
max = (a > b) ? a : b;
```

### 对比总结

| 特性 | 条件运算符 `?:` | if-else |
|------|----------------|---------|
| 代码量 | 一行搞定 | 需要多行 |
| 返回值 | 有返回值（表达式） | 无返回值（语句） |
| 嵌套 | 可以但不推荐 | 适合嵌套 |
| 可读性 | 简单场景更清晰 | 复杂逻辑更清晰 |
| 使用位置 | 可用于表达式内部 | 只能作为独立语句 |

## 四、在表达式中使用

条件运算符最大的优势是它是一个**表达式**，可以用在赋值、函数参数等位置。

```c
// 在printf中直接使用
printf("结果: %d\n", (x > 0) ? x : -x);

// 作为函数参数
void greet(const char *name) {
    printf("Hello, %s!\n", name);
}
greet((name != NULL) ? name : "World");

// 在数组初始化中使用
int arr[] = {(flag) ? 1 : 0, (flag) ? 2 : 0};

// 链式使用
const char *level = (score >= 90) ? "优秀" :
                    (score >= 80) ? "良好" :
                    (score >= 60) ? "及格" : "不及格";
```

## 五、嵌套条件运算符

条件运算符可以嵌套使用，但会降低可读性。

```c
#include <stdio.h>

int main(void) {
    int score = 85;

    // 嵌套条件运算符（不推荐过于复杂的嵌套）
    const char *grade = (score >= 90) ? "A" :
                        (score >= 80) ? "B" :
                        (score >= 70) ? "C" :
                        (score >= 60) ? "D" : "F";

    printf("成绩等级: %s\n", grade);  // B

    // 三数求最大值（嵌套）
    int a = 10, b = 20, c = 15;
    int max = (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);
    printf("最大值: %d\n", max);  // 20

    return 0;
}
```

### 嵌套建议

- 嵌套**不超过两层**，否则严重降低可读性。
- 复杂逻辑建议改用 `if-else if-else` 或 `switch` 语句。

## 六、类型要求

条件运算符的第二个和第三个操作数需要有**兼容的类型**或可以进行隐式类型转换。

```c
// 类型不同但兼容
double result = (flag) ? 1 : 2.5;  // 1被提升为1.0

// 类型不兼容的情况（需注意）
const char *s = (flag) ? "yes" : "no";  // 两个都是char*，没问题

// 混合类型 —— 遵循普通算术转换规则
int x = (flag) ? 3 : 4.5;  // x = 3 或 x = 4（4.5被截断）
```

## 七、条件运算符的短路特性

类似于 `&&` 和 `||`，条件运算符也保证**只对选中的分支求值**。

```c
int i = 0;

// 因为条件为真，只执行 i++，不执行 i--
int result = (i == 0) ? i++ : i--;
printf("result = %d, i = %d\n", result, i);  // result = 0, i = 1

// 安全的空指针操作
char *ptr = NULL;
const char *str = (ptr != NULL) ? ptr : "(null)";
// 如果ptr为NULL，不会访问ptr
```

## 八、关键要点

1. `?:` 是C语言唯一的三目运算符，语法：`条件 ? 真值 : 假值`。
2. 条件运算符是**表达式**，有返回值，可以用在赋值和函数参数中。
3. 简单的二选一场景使用条件运算符比 `if-else` 更简洁。
4. 避免过多嵌套，超过两层建议改用 `if-else`。
5. 条件运算符具有**右结合性**：`a ? b : c ? d : e` 等价于 `a ? b : (c ? d : e)`。
6. 条件运算符优先级仅高于赋值运算符和逗号运算符。
7. 只有被选中的分支会被求值（短路特性）。
