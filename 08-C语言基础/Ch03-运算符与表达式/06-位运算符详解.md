# 06 - 位运算符详解

## 一、概述

位运算符直接对整数的**二进制位**进行操作，效率极高，广泛应用于系统编程、嵌入式开发和底层操作。

C语言提供6个位运算符：

| 运算符 | 名称 | 类型 |
|--------|------|------|
| `&` | 按位与（AND） | 双目 |
| `\|` | 按位或（OR） | 双目 |
| `^` | 按位异或（XOR） | 双目 |
| `~` | 按位取反（NOT） | 单目 |
| `<<` | 左移 | 双目 |
| `>>` | 右移 | 双目 |

> **注意**：位运算符只能用于**整数类型**（`char`、`short`、`int`、`long`、`long long` 及其 unsigned 版本）。

## 二、按位与 `&`

两个对应位都为1时，结果位才为1。

```
  0b1100  (12)
& 0b1010  (10)
--------
  0b1000  (8)
```

```c
#include <stdio.h>

int main(void) {
    unsigned int a = 0b1100;  // 12
    unsigned int b = 0b1010;  // 10

    printf("a & b = %u\n", a & b);  // 8 (0b1000)

    // 常见用途：掩码操作，提取特定位
    unsigned int flags = 0xFF;  // 11111111
    unsigned int mask = 0x0F;   // 00001111
    printf("flags & mask = %u\n", flags & mask);  // 15 (低4位)

    return 0;
}
```

## 三、按位或 `|`

两个对应位只要有一个为1，结果位就为1。

```
  0b1100  (12)
| 0b1010  (10)
--------
  0b1110  (14)
```

```c
unsigned int a = 0b1100;  // 12
unsigned int b = 0b1010;  // 10

printf("a | b = %u\n", a | b);  // 14 (0b1110)

// 常见用途：设置特定位
unsigned int flags = 0x00;
flags |= (1 << 3);  // 将第3位置1
printf("flags = 0x%X\n", flags);  // 0x08
```

## 四、按位异或 `^`

两个对应位不同时，结果位为1；相同时为0。

```
  0b1100  (12)
^ 0b1010  (10)
--------
  0b0110  (6)
```

```c
unsigned int a = 0b1100;  // 12
unsigned int b = 0b1010;  // 10

printf("a ^ b = %u\n", a ^ b);  // 6 (0b0110)

// 异或的重要性质
printf("a ^ a = %u\n", a ^ a);      // 0（任何数与自身异或为0）
printf("a ^ 0 = %u\n", a ^ 0);      // 12（任何数与0异或为其自身）
printf("a ^ a ^ b = %u\n", a ^ a ^ b);  // 10（异或具有交换律和结合律）
```

## 五、按位取反 `~`

将每一位取反：0变1，1变0。（单目运算符）

```
~ 0b11001010
= 0b00110101
```

```c
unsigned char a = 0b11001010;  // 202
printf("~a = %u\n", (unsigned char)~a);  // 53 (0b00110101)

// 注意：对int类型取反要注意有符号数的表示
int b = 5;
printf("~b = %d\n", ~b);   // -6（因为 ~5 = ~0...00101 = 1...11010 = -6）
printf("~0 = %d\n", ~0);   // -1
```

## 六、左移 `<<`

将二进制位整体向左移动指定位数，**低位补0**。

```c
unsigned int a = 0b00001100;  // 12

printf("a << 1 = %u\n", a << 1);   // 24 (0b00011000)
printf("a << 2 = %u\n", a << 2);   // 48 (0b00110000)
printf("a << 3 = %u\n", a << 3);   // 96 (0b01100000)

// 左移n位等价于乘以2^n（在不溢出的前提下）
printf("12 << 3 = %u\n", 12u << 3);  // 96 = 12 * 8
```

### 左移注意事项

- 对**有符号数**左移，如果结果溢出或符号位被改变，则是**未定义行为**。
- 对**无符号数**左移，移出的位被丢弃，低位补0。
- 移位位数**不能为负数**，也不能大于等于类型位数（C99之前未定义，C99起对无符号数有明确定义）。

## 七、右移 `>>`

将二进制位整体向右移动指定位数。

```c
unsigned int a = 0b01100000;  // 96

printf("a >> 1 = %u\n", a >> 1);   // 48 (0b00110000)
printf("a >> 3 = %u\n", a >> 3);   // 12 (0b00001100)

// 右移n位等价于除以2^n
printf("96 >> 3 = %u\n", 96u >> 3);  // 12 = 96 / 8
```

### 右移的两种方式

- **逻辑右移**（无符号数）：高位补0。
- **算术右移**（有符号数）：高位补符号位（保持正负性）。C标准不强制规定，但几乎所有现代编译器对有符号数使用算术右移。

```c
// 有符号数右移
int neg = -8;  // 0xFFFFFFF8
printf("neg >> 1 = %d\n", neg >> 1);  // -4（算术右移，高位补1）

// 无符号数右移
unsigned int uneg = 0xFFFFFFF8;
printf("uneg >> 1 = %u\n", uneg >> 1);  // 0x7FFFFFFC（逻辑右移，高位补0）
```

## 八、综合示例

```c
#include <stdio.h>

void print_binary(unsigned int n) {
    for (int i = 31; i >= 0; i--) {
        printf("%d", (n >> i) & 1);
        if (i % 4 == 0) printf(" ");
    }
    printf("\n");
}

int main(void) {
    unsigned int a = 0b11001010;
    unsigned int b = 0b10101100;

    printf("a     = "); print_binary(a);
    printf("b     = "); print_binary(b);
    printf("a & b = "); print_binary(a & b);
    printf("a | b = "); print_binary(a | b);
    printf("a ^ b = "); print_binary(a ^ b);
    printf("~a    = "); print_binary(~a);
    printf("a << 2= "); print_binary(a << 2);
    printf("a >> 2= "); print_binary(a >> 2);

    return 0;
}
```

## 九、关键要点

1. 位运算符只对**整数类型**有效，不能用于浮点数。
2. `&` 用于掩码（提取位），`|` 用于设置位，`^` 用于翻转位。
3. 左移等价于乘以2的幂，右移等价于除以2的幂（无符号数或正数）。
4. 有符号数右移是**算术右移**（补符号位），无符号数右移是**逻辑右移**（补0）。
5. 移位操作中移位量为负数或大于等于类型宽度时行为可能未定义。
6. `~` 是单目运算符，优先级与 `!`、`++` 相同。
7. `&`、`|`、`^` 的优先级低于关系运算符，容易与逻辑运算符混淆。
