# 图的遍历

## 1. 概述

图的遍历是指从某个顶点出发，访问图中所有顶点且每个顶点只访问一次。主要有两种遍历方式：
- **深度优先搜索（DFS）**：尽可能深地搜索
- **广度优先搜索（BFS）**：按层次逐层搜索

## 2. 深度优先搜索（DFS）

### 2.1 思想

从起始顶点出发，沿着一条路径尽可能深入，直到无法继续时回溯。

```
图:      0---1---3
         |   |   |
         4---2---5

DFS从0出发: 0 -> 1 -> 2 -> 4 -> 3 -> 5（或类似顺序）
```

### 2.2 递归实现

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

#define MAX_V 100

// 邻接表
typedef struct AdjNode {
    int v;
    struct AdjNode *next;
} AdjNode;

AdjNode *heads[MAX_V];
int visited[MAX_V];

// 递归 DFS
void DFS(int v) {
    visited[v] = 1;
    printf("%d ", v);

    AdjNode *p = heads[v];
    while (p) {
        if (!visited[p->v]) {
            DFS(p->v);
        }
        p = p->next;
    }
}
```

### 2.3 非递归实现（用栈）

```c
void DFS_Iterative(int start, int n) {
    int visited[MAX_V] = {0};
    int stack[MAX_V];
    int top = -1;

    stack[++top] = start;

    while (top >= 0) {
        int v = stack[top--];
        if (visited[v]) continue;
        visited[v] = 1;
        printf("%d ", v);

        // 逆序压栈，保证左边的邻居先访问
        AdjNode *p = heads[v];
        int neighbors[MAX_V], cnt = 0;
        while (p) {
            neighbors[cnt++] = p->v;
            p = p->next;
        }
        for (int i = cnt - 1; i >= 0; i--) {
            if (!visited[neighbors[i]]) {
                stack[++top] = neighbors[i];
            }
        }
    }
}
```

## 3. 广度优先搜索（BFS）

### 3.1 思想

从起始顶点出发，先访问所有邻居，再访问邻居的邻居。

```
BFS从0出发: 0 -> 1 -> 4 -> 2 -> 3 -> 5
```

### 3.2 队列实现

```c
void BFS(int start, int n) {
    int visited[MAX_V] = {0};
    int queue[MAX_V];
    int front = 0, rear = 0;

    visited[start] = 1;
    queue[rear++] = start;

    while (front < rear) {
        int v = queue[front++];
        printf("%d ", v);

        AdjNode *p = heads[v];
        while (p) {
            if (!visited[p->v]) {
                visited[p->v] = 1;
                queue[rear++] = p->v;
            }
            p = p->next;
        }
    }
}
```

## 4. 连通分量

### 4.1 无向图的连通分量

使用 DFS 或 BFS 标记每个连通分量。

```c
void findConnectedComponents(int n) {
    int component[MAX_V];  // 每个顶点所属的连通分量编号
    int numComponents = 0;

    for (int i = 0; i < n; i++) component[i] = -1;

    for (int i = 0; i < n; i++) {
        if (component[i] == -1) {
            numComponents++;
            // DFS 标记连通分量
            int stack[MAX_V], top = -1;
            stack[++top] = i;
            while (top >= 0) {
                int v = stack[top--];
                if (component[v] != -1) continue;
                component[v] = numComponents;
                AdjNode *p = heads[v];
                while (p) {
                    if (component[p->v] == -1) {
                        stack[++top] = p->v;
                    }
                    p = p->next;
                }
            }
        }
    }

    printf("连通分量数: %d\n", numComponents);
    for (int c = 1; c <= numComponents; c++) {
        printf("分量%d: ", c);
        for (int i = 0; i < n; i++) {
            if (component[i] == c) printf("%d ", i);
        }
        printf("\n");
    }
}
```

### 4.2 判断两点是否连通

```c
bool isConnected(int u, int v, int n) {
    int visited[MAX_V] = {0};
    int queue[MAX_V];
    int front = 0, rear = 0;

    visited[u] = 1;
    queue[rear++] = u;

    while (front < rear) {
        int curr = queue[front++];
        if (curr == v) return true;
        AdjNode *p = heads[curr];
        while (p) {
            if (!visited[p->v]) {
                if (p->v == v) return true;
                visited[p->v] = 1;
                queue[rear++] = p->v;
            }
            p = p->next;
        }
    }
    return false;
}
```

## 5. 拓扑排序（有向无环图）

拓扑排序将有向无环图（DAG）的顶点排成线性序列，使得所有有向边都从前指向后。

### Kahn 算法（BFS 思想）

```c
void topoSort(int n) {
    int inDegree[MAX_V] = {0};
    int result[MAX_V], count = 0;

    // 计算入度
    for (int i = 0; i < n; i++) {
        AdjNode *p = heads[i];
        while (p) {
            inDegree[p->v]++;
            p = p->next;
        }
    }

    // 入度为0的顶点入队
    int queue[MAX_V], front = 0, rear = 0;
    for (int i = 0; i < n; i++) {
        if (inDegree[i] == 0) queue[rear++] = i;
    }

    while (front < rear) {
        int v = queue[front++];
        result[count++] = v;

        AdjNode *p = heads[v];
        while (p) {
            inDegree[p->v]--;
            if (inDegree[p->v] == 0) {
                queue[rear++] = p->v;
            }
            p = p->next;
        }
    }

    if (count != n) {
        printf("图中存在环，无法拓扑排序!\n");
    } else {
        printf("拓扑排序: ");
        for (int i = 0; i < count; i++) printf("%d ", result[i]);
        printf("\n");
    }
}
```

## 6. 判断图中是否有环

```c
// 有向图判环（DFS + 三色标记）
// 0=白色(未访问), 1=灰色(正在访问), 2=黑色(已完成)
int color[MAX_V];

bool hasCycleDirected(int v) {
    color[v] = 1;  // 标记为灰色
    AdjNode *p = heads[v];
    while (p) {
        if (color[p->v] == 1) return true;   // 遇到灰色，有环
        if (color[p->v] == 0 && hasCycleDirected(p->v)) return true;
        p = p->next;
    }
    color[v] = 2;  // 标记为黑色
    return false;
}

bool detectCycleDirected(int n) {
    for (int i = 0; i < n; i++) color[i] = 0;
    for (int i = 0; i < n; i++) {
        if (color[i] == 0 && hasCycleDirected(i)) return true;
    }
    return false;
}
```

## 7. 辅助函数和测试

```c
void addEdge(int u, int v) {
    AdjNode *node = (AdjNode *)malloc(sizeof(AdjNode));
    node->v = v; node->next = heads[u]; heads[u] = node;
}

void addUndirectedEdge(int u, int v) {
    addEdge(u, v);
    addEdge(v, u);
}

int main() {
    int n = 6;
    for (int i = 0; i < n; i++) heads[i] = NULL;

    // 构建无向图
    addUndirectedEdge(0, 1);
    addUndirectedEdge(0, 4);
    addUndirectedEdge(1, 2);
    addUndirectedEdge(1, 3);
    addUndirectedEdge(1, 4);
    addUndirectedEdge(2, 3);
    addUndirectedEdge(3, 5);

    printf("DFS: "); DFS(0); printf("\n");
    printf("BFS: "); BFS(0, n); printf("\n");
    findConnectedComponents(n);

    return 0;
}
```

## 8. 复杂度分析

| 遍历方式 | 邻接矩阵 | 邻接表 |
|---------|---------|--------|
| DFS | O(V^2) | O(V+E) |
| BFS | O(V^2) | O(V+E) |

## 9. 重点总结

- DFS 用**递归或栈**，BFS 用**队列**
- DFS 适合路径搜索、连通分量、拓扑排序
- BFS 适合最短路径（无权图）、层次遍历
- 有向图判环用三色标记（白-灰-黑）
- 拓扑排序只适用于有向无环图（DAG）
- 无向图用并查集也能判断连通性

> **注意**：DFS 的递归深度可能很大（图很大时），可能导致栈溢出。此时需要改用非递归实现。
