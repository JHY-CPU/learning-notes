# 栈的数组实现（顺序栈）

## 1. 概述

栈（Stack）是一种**后进先出（LIFO, Last In First Out）**的线性数据结构。只允许在栈顶进行插入（push）和删除（pop）操作。

### 生活中的栈
- 叠盘子：最后放上去的盘子最先被拿走
- 浏览器的后退按钮
- 函数调用栈

### 栈的基本操作
- **push**：将元素压入栈顶
- **pop**：弹出栈顶元素
- **peek/top**：查看栈顶元素（不弹出）
- **isEmpty**：判断栈是否为空

## 2. 数据结构定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

#define MAX_SIZE 100

// 顺序栈（静态数组实现）
typedef struct {
    int data[MAX_SIZE];  // 存储栈元素的数组
    int top;             // 栈顶指针（下标）
} ArrayStack;

// 动态顺序栈
typedef struct {
    int *data;           // 动态数组
    int top;             // 栈顶指针
    int capacity;        // 栈的容量
} DynamicStack;
```

## 3. 基本操作实现

### 3.1 初始化

```c
// 静态栈初始化
void initStack(ArrayStack *s) {
    s->top = -1;  // top = -1 表示栈空
}

// 动态栈创建
DynamicStack *createStack(int capacity) {
    DynamicStack *s = (DynamicStack *)malloc(sizeof(DynamicStack));
    if (!s) return NULL;
    s->data = (int *)malloc(capacity * sizeof(int));
    if (!s->data) {
        free(s);
        return NULL;
    }
    s->top = -1;
    s->capacity = capacity;
    return s;
}
```

### 3.2 push 操作

```c
// 压栈
bool push(ArrayStack *s, int value) {
    if (s->top >= MAX_SIZE - 1) {
        printf("栈满！无法压入 %d\n", value);
        return false;
    }
    s->data[++(s->top)] = value;
    return true;
}
```

```
push(10): top=-1 -> top=0, data[0]=10   栈: [10]
push(20): top=0  -> top=1, data[1]=20   栈: [10, 20]
push(30): top=1  -> top=2, data[2]=30   栈: [10, 20, 30]
```

### 3.3 pop 操作

```c
// 弹栈
bool pop(ArrayStack *s, int *value) {
    if (s->top < 0) {
        printf("栈空！无法弹出\n");
        return false;
    }
    *value = s->data[(s->top)--];
    return true;
}
```

```
pop: top=2 -> top=1, 返回 30   栈: [10, 20]
pop: top=1 -> top=0, 返回 20   栈: [10]
```

### 3.4 peek 操作

```c
// 查看栈顶元素
bool peek(ArrayStack *s, int *value) {
    if (s->top < 0) {
        return false;
    }
    *value = s->data[s->top];
    return true;
}
```

### 3.5 辅助操作

```c
// 判断栈空
bool isEmpty(ArrayStack *s) {
    return s->top < 0;
}

// 判断栈满
bool isFull(ArrayStack *s) {
    return s->top >= MAX_SIZE - 1;
}

// 获取栈大小
int size(ArrayStack *s) {
    return s->top + 1;
}

// 打印栈
void printStack(ArrayStack *s) {
    printf("栈底 -> ");
    for (int i = 0; i <= s->top; i++) {
        printf("%d", s->data[i]);
        if (i < s->top) printf(", ");
    }
    printf(" <- 栈顶 (共%d个元素)\n", size(s));
}
```

### 3.6 共享栈（两个栈共享一个数组）

```c
// 两个栈共享一个数组，分别从两端向中间生长
#define SHARED_SIZE 100

typedef struct {
    int data[SHARED_SIZE];
    int top1;  // 栈1的栈顶（从左向右）
    int top2;  // 栈2的栈顶（从右向左）
} SharedStack;

void initShared(SharedStack *s) {
    s->top1 = -1;
    s->top2 = SHARED_SIZE;
}

bool push1(SharedStack *s, int value) {
    if (s->top1 + 1 == s->top2) return false;  // 栈满
    s->data[++(s->top1)] = value;
    return true;
}

bool push2(SharedStack *s, int value) {
    if (s->top2 - 1 == s->top1) return false;
    s->data[--(s->top2)] = value;
    return true;
}
```

## 4. 完整测试代码

```c
int main() {
    ArrayStack stack;
    initStack(&stack);

    // 压栈
    push(&stack, 10);
    push(&stack, 20);
    push(&stack, 30);
    push(&stack, 40);
    printStack(&stack);  // 栈底 -> 10, 20, 30, 40 <- 栈顶

    // 查看栈顶
    int top;
    peek(&stack, &top);
    printf("栈顶元素: %d\n", top);  // 40

    // 弹栈
    int val;
    while (!isEmpty(&stack)) {
        pop(&stack, &val);
        printf("弹出: %d\n", val);
    }
    // 弹出: 40, 30, 20, 10

    // 尝试从空栈弹出
    pop(&stack, &val);  // 栈空！

    return 0;
}
```

## 5. 操作复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| push | O(1) | 直接在栈顶添加 |
| pop | O(1) | 直接移除栈顶 |
| peek | O(1) | 直接访问栈顶 |
| isEmpty | O(1) | 判断 top |
| isFull | O(1) | 判断 top |

所有操作都是 O(1)，这是顺序栈的最大优势。

## 6. 重点总结

- 栈是 LIFO 结构，只在栈顶操作
- 顺序栈用数组实现，栈顶指针 `top` 通常初始化为 `-1`
- `++top` 然后赋值（先加后存），`top--` 然后取值（先取后减）
- 共享栈充分利用空间，两个栈从两端向中间增长
- 顺序栈的缺点是容量固定（动态栈可解决）
- 函数调用、表达式求值、括号匹配都是栈的典型应用

> **注意**：`top` 的初始值选择（-1 或 0）会影响 push/pop 的写法。选择一种并保持一致即可。
