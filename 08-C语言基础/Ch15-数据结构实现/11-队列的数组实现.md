# 队列的数组实现（顺序队列）

## 1. 概述

队列（Queue）是一种**先进先出（FIFO, First In First Out）**的线性数据结构。在一端插入（队尾/rear），在另一端删除（队头/front）。

### 生活中的队列
- 排队买票：先来的人先买
- 打印机任务队列
- 消息队列

### 基本操作
- **enqueue**（入队）：在队尾添加元素
- **dequeue**（出队）：从队头移除元素
- **front/peek**：查看队头元素

## 2. 普通顺序队列的问题

```c
// 普通顺序队列 - 存在"假溢出"问题
typedef struct {
    int data[MAX_SIZE];
    int front;  // 队头下标
    int rear;   // 队尾下标（指向下一个要插入的位置）
} SimpleQueue;
```

**假溢出问题**：经过多次入队出队后，rear 到达数组末尾，但前面还有空位。

```
初始:     [ ][ ][ ][ ][ ][ ]
           f  r

入队A,B,C: [A][B][C][ ][ ][ ]
                 f        r

出队A,B:   [A][B][C][ ][ ][ ]
                       f  r

入队D,E,F: [A][B][C][D][E][F]
                             f  r  <- rear到达末尾，但前面有空位！
```

## 3. 循环队列（推荐方案）

循环队列通过取模运算，使数组首尾相连，解决假溢出问题。

### 3.1 数据结构定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

#define MAX_SIZE 6  // 实际可用空间为 MAX_SIZE - 1

typedef struct {
    int data[MAX_SIZE];
    int front;  // 队头下标
    int rear;   // 队尾下标（指向下一个要插入的位置）
} CircularQueue;
```

### 3.2 队空和队满判断

**关键问题**：如何区分队空和队满？

方法一：牺牲一个存储单元（推荐）：
- 队空：`front == rear`
- 队满：`(rear + 1) % MAX_SIZE == front`

方法二：使用 size 变量记录元素个数。

```c
void initQueue(CircularQueue *q) {
    q->front = 0;
    q->rear = 0;
}

bool isEmpty(CircularQueue *q) {
    return q->front == q->rear;
}

bool isFull(CircularQueue *q) {
    return (q->rear + 1) % MAX_SIZE == q->front;
}
```

### 3.3 入队操作

```c
bool enqueue(CircularQueue *q, int value) {
    if (isFull(q)) {
        printf("队满!\n");
        return false;
    }
    q->data[q->rear] = value;
    q->rear = (q->rear + 1) % MAX_SIZE;
    return true;
}
```

```
入队A: data=[A,_,_,_,_], front=0, rear=1
入队B: data=[A,B,_,_,_], front=0, rear=2
入队C: data=[A,B,C,_,_], front=0, rear=3
入队D: data=[A,B,C,D,_], front=0, rear=4
此时队满: (4+1)%5 == 0 == front
```

### 3.4 出队操作

```c
bool dequeue(CircularQueue *q, int *value) {
    if (isEmpty(q)) {
        printf("队空!\n");
        return false;
    }
    *value = q->data[q->front];
    q->front = (q->front + 1) % MAX_SIZE;
    return true;
}
```

### 3.5 查看队头

```c
bool peek(CircularQueue *q, int *value) {
    if (isEmpty(q)) return false;
    *value = q->data[q->front];
    return true;
}
```

### 3.6 辅助操作

```c
int size(CircularQueue *q) {
    return (q->rear - q->front + MAX_SIZE) % MAX_SIZE;
}

void printQueue(CircularQueue *q) {
    printf("队列 [大小=%d]: ", size(q));
    int i = q->front;
    while (i != q->rear) {
        printf("%d ", q->data[i]);
        i = (i + 1) % MAX_SIZE;
    }
    printf("\n");
}
```

## 4. 完整测试代码

```c
int main() {
    CircularQueue queue;
    initQueue(&queue);

    // 入队
    enqueue(&queue, 10);
    enqueue(&queue, 20);
    enqueue(&queue, 30);
    enqueue(&queue, 40);
    printQueue(&queue);  // 10 20 30 40

    // 此时 MAX_SIZE=6，可用5个位置，已用4个
    printf("队满? %s\n", isFull(&queue) ? "是" : "否");
    enqueue(&queue, 50);  // 入队成功
    enqueue(&queue, 60);  // 队满!

    // 出队
    int val;
    dequeue(&queue, &val);
    printf("出队: %d\n", val);  // 10
    dequeue(&queue, &val);
    printf("出队: %d\n", val);  // 20

    // 继续入队（利用回收的空间）
    enqueue(&queue, 60);
    enqueue(&queue, 70);
    printQueue(&queue);  // 30 40 50 60 70

    // 查看队头
    peek(&queue, &val);
    printf("队头: %d\n", val);  // 30

    return 0;
}
```

## 5. 不使用牺牲空间的方案

```c
// 方法二：使用 size 变量
typedef struct {
    int data[MAX_SIZE];
    int front;
    int rear;
    int size;    // 当前元素个数
    int capacity;
} QueueWithSize;

bool isFullSize(QueueWithSize *q) {
    return q->size == q->capacity;
}

bool isEmptySize(QueueWithSize *q) {
    return q->size == 0;
}

void enqueueSize(QueueWithSize *q, int value) {
    if (isFullSize(q)) return;
    q->data[q->rear] = value;
    q->rear = (q->rear + 1) % q->capacity;
    q->size++;
}

int dequeueSize(QueueWithSize *q) {
    if (isEmptySize(q)) return -1;
    int value = q->data[q->front];
    q->front = (q->front + 1) % q->capacity;
    q->size--;
    return value;
}
```

## 6. 操作复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| enqueue | O(1) | rear 后移一位 |
| dequeue | O(1) | front 后移一位 |
| peek | O(1) | 直接访问 front |
| isEmpty | O(1) | 判断 front == rear |
| isFull | O(1) | 判断 (rear+1)%MAX_SIZE == front |

## 7. 重点总结

- 循环队列通过**取模运算**实现首尾循环
- 牺牲一个存储单元来区分队空和队满是最常用的方法
- 队空条件：`front == rear`
- 队满条件：`(rear + 1) % MAX_SIZE == front`
- 可用容量为 `MAX_SIZE - 1`
- 入队是 `rear` 移动，出队是 `front` 移动

> **注意**：不要混淆 front 和 rear 的定义。本节中 front 指向队头元素，rear 指向下一个要插入的位置。有些教材的定义可能不同。
