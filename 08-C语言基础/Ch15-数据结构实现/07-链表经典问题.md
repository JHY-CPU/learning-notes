# 链表经典问题

## 1. 概述

链表是面试中的高频考点。本节整理链表的经典算法问题，包括反转链表、检测环、合并链表、找中间节点等。

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;
```

## 2. 反转链表

### 2.1 迭代法（三指针法）

核心思想：逐个改变指针方向，用三个指针维护当前状态。

```c
// 迭代反转链表 - O(n) 时间, O(1) 空间
Node *reverseList(Node *head) {
    Node *prev = NULL;
    Node *curr = head;
    Node *next = NULL;

    while (curr != NULL) {
        next = curr->next;   // 暂存下一个节点
        curr->next = prev;   // 反转指针
        prev = curr;         // prev 前进
        curr = next;         // curr 前进
    }
    return prev;  // prev 就是新的头节点
}
```

```
原链表: 1 -> 2 -> 3 -> 4 -> NULL
步骤1:  NULL <- 1    2 -> 3 -> 4 -> NULL
步骤2:  NULL <- 1 <- 2    3 -> 4 -> NULL
步骤3:  NULL <- 1 <- 2 <- 3    4 -> NULL
步骤4:  NULL <- 1 <- 2 <- 3 <- 4
结果:   4 -> 3 -> 2 -> 1 -> NULL
```

### 2.2 递归法

```c
// 递归反转链表 - O(n) 时间, O(n) 空间（栈空间）
Node *reverseListRecursive(Node *head) {
    // 递归终止条件
    if (head == NULL || head->next == NULL) {
        return head;
    }
    // 递归反转后面的链表
    Node *newHead = reverseListRecursive(head->next);
    // 反转当前节点
    head->next->next = head;
    head->next = NULL;
    return newHead;
}
```

## 3. 检测链表是否有环

### 3.1 快慢指针法（Floyd 判环算法）

核心思想：快指针每次走两步，慢指针每次走一步。如果有环，快指针一定会追上慢指针。

```c
// 检测是否有环 - O(n) 时间, O(1) 空间
int hasCycle(Node *head) {
    Node *slow = head;
    Node *fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;          // 慢指针走一步
        fast = fast->next->next;    // 快指针走两步
        if (slow == fast) {
            return 1;  // 有环
        }
    }
    return 0;  // 无环
}
```

### 3.2 找到环的入口

```c
// 找到环的入口节点
Node *findCycleEntry(Node *head) {
    Node *slow = head;
    Node *fast = head;

    // 第一步：检测是否有环
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) break;
    }
    if (fast == NULL || fast->next == NULL) return NULL;

    // 第二步：找入口
    // 一个指针从 head 出发，一个从相遇点出发，每次走一步
    // 两者必定在环入口相遇（数学证明）
    Node *p = head;
    while (p != slow) {
        p = p->next;
        slow = slow->next;
    }
    return p;
}
```

### 3.3 计算环的长度

```c
int getCycleLength(Node *head) {
    Node *slow = head;
    Node *fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) {
            // 找到相遇点，再走一圈计数
            int length = 1;
            Node *p = slow->next;
            while (p != slow) {
                length++;
                p = p->next;
            }
            return length;
        }
    }
    return 0;
}
```

## 4. 合并两个有序链表

```c
// 递归法合并
Node *mergeTwoLists(Node *l1, Node *l2) {
    if (l1 == NULL) return l2;
    if (l2 == NULL) return l1;

    if (l1->data <= l2->data) {
        l1->next = mergeTwoLists(l1->next, l2);
        return l1;
    } else {
        l2->next = mergeTwoLists(l1, l2->next);
        return l2;
    }
}

// 迭代法合并
Node *mergeTwoListsIter(Node *l1, Node *l2) {
    Node dummy;
    dummy.next = NULL;
    Node *tail = &dummy;

    while (l1 != NULL && l2 != NULL) {
        if (l1->data <= l2->data) {
            tail->next = l1;
            l1 = l1->next;
        } else {
            tail->next = l2;
            l2 = l2->next;
        }
        tail = tail->next;
    }
    tail->next = (l1 != NULL) ? l1 : l2;
    return dummy.next;
}
```

## 5. 找链表中间节点

```c
// 快慢指针法 - 快指针走两步，慢指针走一步
// 快指针到末尾时，慢指针正好在中间
Node *findMiddle(Node *head) {
    Node *slow = head;
    Node *fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}
```

```
链表: 1 -> 2 -> 3 -> 4 -> 5 -> NULL
slow: 1, 2, 3
fast: 1, 3, 5
中间节点: 3

链表: 1 -> 2 -> 3 -> 4 -> NULL
slow: 1, 2, 3
fast: 1, 3, NULL
中间节点: 3（偶数长度时返回第二个中间节点）
```

## 6. 判断链表是否为回文

```c
// 方法：找中间 -> 反转后半部分 -> 逐一比较
int isPalindrome(Node *head) {
    if (head == NULL || head->next == NULL) return 1;

    // 找中间节点
    Node *slow = head;
    Node *fast = head;
    while (fast->next != NULL && fast->next->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
    }

    // 反转后半部分
    Node *secondHalf = reverseList(slow->next);

    // 比较
    Node *p1 = head;
    Node *p2 = secondHalf;
    int result = 1;
    while (p2 != NULL) {
        if (p1->data != p2->data) {
            result = 0;
            break;
        }
        p1 = p1->next;
        p2 = p2->next;
    }

    // 恢复链表（可选）
    slow->next = reverseList(secondHalf);
    return result;
}
```

## 7. 删除链表倒数第 N 个节点

```c
// 快慢指针法：快指针先走 n 步，然后同时走
Node *removeNthFromEnd(Node *head, int n) {
    Node dummy;
    dummy.next = head;
    Node *fast = &dummy;
    Node *slow = &dummy;

    // fast 先走 n 步
    for (int i = 0; i < n; i++) {
        fast = fast->next;
    }

    // 同时走，直到 fast 到达末尾
    while (fast->next != NULL) {
        fast = fast->next;
        slow = slow->next;
    }

    // 删除 slow 后面的节点
    Node *temp = slow->next;
    slow->next = temp->next;
    free(temp);

    return dummy.next;
}
```

## 8. 相交链表的交点

```c
// 两个链表相交，找到交点
// 思路：让两个指针分别走完 A+B，最终会在交点相遇
Node *getIntersection(Node *headA, Node *headB) {
    if (headA == NULL || headB == NULL) return NULL;

    Node *pA = headA;
    Node *pB = headB;

    // pA 走完 A 走 B，pB 走完 B 走 A
    // 最终会在交点相遇（或同时到达 NULL）
    while (pA != pB) {
        pA = (pA == NULL) ? headB : pA->next;
        pB = (pB == NULL) ? headA : pB->next;
    }
    return pA;
}
```

## 9. 重点总结

| 问题 | 核心方法 | 时间 | 空间 |
|------|---------|------|------|
| 反转链表 | 三指针/递归 | O(n) | O(1)/O(n) |
| 检测环 | 快慢指针 | O(n) | O(1) |
| 环入口 | 快慢指针+数学 | O(n) | O(1) |
| 合并有序链表 | 迭代/递归 | O(n+m) | O(1)/O(n) |
| 中间节点 | 快慢指针 | O(n) | O(1) |
| 回文链表 | 反转+比较 | O(n) | O(1) |
| 倒数第N个 | 快慢指针 | O(n) | O(1) |
| 相交链表 | 双指针 | O(n+m) | O(1) |

> **核心技巧**：链表问题的三大利器——**快慢指针**、**三指针反转**、**dummy 头节点**。掌握这三种方法，大部分链表问题都可以迎刃而解。
