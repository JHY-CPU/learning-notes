# 二叉树基础

## 1. 概述

二叉树（Binary Tree）是每个节点**最多有两个子节点**的树形数据结构。两个子节点分别称为**左孩子**和**右孩子**。

### 与普通树的区别
- 每个节点的子节点有明确的左右之分
- 即使只有一个子节点，也要区分是左还是右

## 2. 基本术语

```
        A          <- 根节点 (root)
       / \
      B   C        <- 内部节点
     / \   \
    D   E   F      <- 叶子节点 (leaf)
```

| 术语 | 说明 |
|------|------|
| 根节点（Root） | 树的最顶层节点，没有父节点 |
| 父节点（Parent） | 某节点的上一层节点 |
| 子节点（Child） | 某节点的下一层节点 |
| 叶子节点（Leaf） | 没有子节点的节点 |
| 兄弟节点（Sibling） | 有相同父节点的节点 |
| 节点的度 | 节点拥有的子节点个数（0, 1, 2） |
| 树的度 | 树中所有节点度的最大值 |
| 层次/深度 | 根为第1层，依次递增 |
| 高度 | 叶子节点高度为1，向上递增 |
| 森林 | 多棵互不相交的树的集合 |

## 3. 特殊的二叉树

### 3.1 满二叉树（Perfect Binary Tree）
每一层的节点数都达到最大值。深度为 k 的满二叉树有 2^k - 1 个节点。

```
深度1:    *            1个节点
深度2:    *           3个节点
         / \
        *   *
深度3:   *           7个节点
        / \
       *   *
      / \ / \
     *  * *  *
```

### 3.2 完全二叉树（Complete Binary Tree）
除了最后一层，其他层都是满的，且最后一层的节点都靠左排列。

```
完全二叉树:        非完全二叉树:
      1                  1
     / \                / \
    2   3              2   3
   / \  /            / \    \
  4  5 6            4   5    6
```

完全二叉树可以用数组高效存储：
- 父节点下标：`(i - 1) / 2`
- 左孩子下标：`2 * i + 1`
- 右孩子下标：`2 * i + 2`

### 3.3 二叉搜索树（BST）
左子树所有节点的值 < 根节点的值 < 右子树所有节点的值。

## 4. 二叉树的性质

1. **第 i 层最多有 2^(i-1) 个节点**（i >= 1）
2. **深度为 k 的二叉树最多有 2^k - 1 个节点**
3. **叶子节点数 = 度为2的节点数 + 1**（n0 = n2 + 1）
4. **n 个节点的完全二叉树深度为 log2(n) + 1（向下取整）**

## 5. 节点定义

### 5.1 链式存储（最常用）

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct TreeNode {
    int data;                  // 数据域
    struct TreeNode *left;     // 左孩子
    struct TreeNode *right;    // 右孩子
} TreeNode;

// 创建新节点
TreeNode *createNode(int value) {
    TreeNode *node = (TreeNode *)malloc(sizeof(TreeNode));
    if (!node) return NULL;
    node->data = value;
    node->left = NULL;
    node->right = NULL;
    return node;
}
```

### 5.2 数组存储（适用于完全二叉树）

```c
#define MAX_NODES 100

typedef struct {
    int data[MAX_NODES];
    int size;
} ArrayBinaryTree;

// 父节点: (i - 1) / 2
// 左孩子: 2 * i + 1
// 右孩子: 2 * i + 2

int getParent(int i) { return (i - 1) / 2; }
int getLeftChild(int i) { return 2 * i + 1; }
int getRightChild(int i) { return 2 * i + 2; }
```

## 6. 构建一棵简单的二叉树

```c
TreeNode *buildSampleTree() {
    //       1
    //      / \
    //     2   3
    //    / \   \
    //   4   5   6
    //  /
    // 7

    TreeNode *root = createNode(1);
    root->left = createNode(2);
    root->right = createNode(3);
    root->left->left = createNode(4);
    root->left->right = createNode(5);
    root->right->right = createNode(6);
    root->left->left->left = createNode(7);
    return root;
}
```

## 7. 基本属性计算

```c
// 计算节点总数
int countNodes(TreeNode *root) {
    if (root == NULL) return 0;
    return 1 + countNodes(root->left) + countNodes(root->right);
}

// 计算叶子节点数
int countLeaves(TreeNode *root) {
    if (root == NULL) return 0;
    if (root->left == NULL && root->right == NULL) return 1;
    return countLeaves(root->left) + countLeaves(root->right);
}

// 计算树的高度
int treeHeight(TreeNode *root) {
    if (root == NULL) return 0;
    int leftHeight = treeHeight(root->left);
    int rightHeight = treeHeight(root->right);
    return 1 + (leftHeight > rightHeight ? leftHeight : rightHeight);
}

// 计算第 k 层的节点数
int countAtLevel(TreeNode *root, int k) {
    if (root == NULL) return 0;
    if (k == 1) return 1;
    return countAtLevel(root->left, k - 1) +
           countAtLevel(root->right, k - 1);
}
```

## 8. 判断完全二叉树

```c
// 使用层序遍历判断是否为完全二叉树
bool isCompleteTree(TreeNode *root) {
    if (!root) return true;
    TreeNode *queue[1000];
    int front = 0, rear = 0;
    bool hasNull = false;

    queue[rear++] = root;
    while (front < rear) {
        TreeNode *node = queue[front++];
        if (node == NULL) {
            hasNull = true;
        } else {
            if (hasNull) return false;  // 遇到非空节点在NULL之后
            queue[rear++] = node->left;
            queue[rear++] = node->right;
        }
    }
    return true;
}
```

## 9. 测试代码

```c
int main() {
    TreeNode *root = buildSampleTree();

    printf("节点总数: %d\n", countNodes(root));      // 7
    printf("叶子节点数: %d\n", countLeaves(root));    // 3
    printf("树的高度: %d\n", treeHeight(root));       // 4
    printf("第3层节点数: %d\n", countAtLevel(root, 3)); // 3
    printf("是否完全二叉树: %s\n",
           isCompleteTree(root) ? "是" : "否");

    return 0;
}
```

## 10. 重点总结

| 要点 | 说明 |
|------|------|
| 二叉树每个节点最多两个孩子 | 左孩子和右孩子有序 |
| 完全二叉树可用数组存储 | 高效利用空间，下标计算简单 |
| 叶子节点数 = 度为2节点数 + 1 | 重要性质 |
| n 个节点的完全二叉树深度 | floor(log2(n)) + 1 |
| 链式存储是最通用的方式 | 适合任意形态的二叉树 |

> **注意**：二叉树是递归定义的结构，几乎所有二叉树的操作都可以用递归来实现。理解递归是掌握二叉树的关键。
