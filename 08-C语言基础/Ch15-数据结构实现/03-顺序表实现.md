# 顺序表实现（用数组实现线性表）

## 1. 概述

顺序表（Sequential List）是线性表的一种顺序存储结构，使用**一段连续的存储单元**依次存储线性表的数据元素。在C语言中，通常用数组来实现。

### 特点
- 支持随机访问，按下标访问时间复杂度 O(1)
- 插入和删除需要移动元素，时间复杂度 O(n)
- 存储密度高，不需要额外的指针空间
- 需要预先分配连续的内存空间

## 2. 数据结构定义

### 2.1 静态顺序表

```c
#define MAXSIZE 100

typedef struct {
    int data[MAXSIZE];  // 存储数据元素的数组
    int length;         // 当前长度
} SeqList;
```

### 2.2 动态顺序表（支持扩容）

```c
#define INIT_SIZE 10    // 初始容量
#define INCREMENT 5     // 扩容增量

typedef struct {
    int *data;          // 指向动态分配数组的指针
    int length;         // 当前元素个数
    int capacity;       // 当前分配的容量
} DynamicList;
```

## 3. 基本操作实现

### 3.1 初始化

```c
#include <stdio.h>
#include <stdlib.h>

// 静态顺序表初始化
void initList(SeqList *list) {
    list->length = 0;
}

// 动态顺序表初始化
DynamicList *createList() {
    DynamicList *list = (DynamicList *)malloc(sizeof(DynamicList));
    if (!list) {
        printf("内存分配失败!\n");
        return NULL;
    }
    list->data = (int *)malloc(INIT_SIZE * sizeof(int));
    if (!list->data) {
        free(list);
        return NULL;
    }
    list->length = 0;
    list->capacity = INIT_SIZE;
    return list;
}
```

### 3.2 动态扩容

```c
// 扩容函数
int expandList(DynamicList *list) {
    int newCapacity = list->capacity + INCREMENT;
    int *newData = (int *)realloc(list->data, newCapacity * sizeof(int));
    if (!newData) {
        printf("扩容失败!\n");
        return 0;
    }
    list->data = newData;
    list->capacity = newCapacity;
    printf("扩容成功: %d -> %d\n", newCapacity - INCREMENT, newCapacity);
    return 1;
}
```

### 3.3 插入操作

```c
// 在第 pos 个位置插入元素（pos 从 0 开始）
int insertList(SeqList *list, int pos, int value) {
    // 检查位置是否合法
    if (pos < 0 || pos > list->length) {
        printf("插入位置不合法!\n");
        return 0;
    }
    // 检查是否已满
    if (list->length >= MAXSIZE) {
        printf("顺序表已满!\n");
        return 0;
    }
    // 从后往前移动元素
    for (int i = list->length; i > pos; i--) {
        list->data[i] = list->data[i - 1];
    }
    list->data[pos] = value;
    list->length++;
    return 1;
}

// 动态顺序表插入（自动扩容）
int insertDynamic(DynamicList *list, int pos, int value) {
    if (pos < 0 || pos > list->length) return 0;
    if (list->length >= list->capacity) {
        if (!expandList(list)) return 0;
    }
    for (int i = list->length; i > pos; i--) {
        list->data[i] = list->data[i - 1];
    }
    list->data[pos] = value;
    list->length++;
    return 1;
}
```

### 3.4 删除操作

```c
// 删除第 pos 个位置的元素
int deleteList(SeqList *list, int pos, int *deleted) {
    if (pos < 0 || pos >= list->length) {
        printf("删除位置不合法!\n");
        return 0;
    }
    *deleted = list->data[pos];
    // 从前往后移动元素
    for (int i = pos; i < list->length - 1; i++) {
        list->data[i] = list->data[i + 1];
    }
    list->length--;
    return 1;
}
```

### 3.5 查找操作

```c
// 按值查找，返回第一个匹配的下标，未找到返回 -1
int findList(SeqList *list, int value) {
    for (int i = 0; i < list->length; i++) {
        if (list->data[i] == value) {
            return i;
        }
    }
    return -1;
}

// 按下标查找，返回元素值
int getElem(SeqList *list, int pos, int *value) {
    if (pos < 0 || pos >= list->length) return 0;
    *value = list->data[pos];
    return 1;
}
```

### 3.6 修改操作

```c
// 将第 pos 个位置的元素修改为 value
int updateList(SeqList *list, int pos, int value) {
    if (pos < 0 || pos >= list->length) return 0;
    list->data[pos] = value;
    return 1;
}
```

### 3.7 遍历打印

```c
void printList(SeqList *list) {
    printf("顺序表 [长度=%d]: ", list->length);
    for (int i = 0; i < list->length; i++) {
        printf("%d ", list->data[i]);
    }
    printf("\n");
}

void printDynamicList(DynamicList *list) {
    printf("动态表 [长度=%d, 容量=%d]: ", list->length, list->capacity);
    for (int i = 0; i < list->length; i++) {
        printf("%d ", list->data[i]);
    }
    printf("\n");
}
```

### 3.8 销毁

```c
void destroyList(DynamicList *list) {
    if (list) {
        free(list->data);
        free(list);
    }
}
```

## 4. 完整测试代码

```c
int main() {
    // 测试静态顺序表
    SeqList list;
    initList(&list);

    insertList(&list, 0, 10);
    insertList(&list, 1, 20);
    insertList(&list, 2, 30);
    insertList(&list, 1, 15);  // 在位置1插入15
    printList(&list);          // 10 15 20 30

    int deleted;
    deleteList(&list, 2, &deleted);
    printf("删除的元素: %d\n", deleted);  // 20
    printList(&list);                     // 10 15 30

    int idx = findList(&list, 15);
    printf("元素15的位置: %d\n", idx);    // 1

    // 测试动态顺序表
    DynamicList *dList = createList();
    for (int i = 0; i < 15; i++) {
        insertDynamic(dList, i, (i + 1) * 10);
    }
    printDynamicList(dList);
    destroyList(dList);

    return 0;
}
```

## 5. 操作复杂度分析

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 按下标访问 | O(1) | 随机访问，直接计算地址 |
| 按值查找 | O(n) | 需要遍历 |
| 插入 | O(n) | 最好O(1)，最坏需要移动全部元素 |
| 删除 | O(n) | 最好O(1)，最坏需要移动全部元素 |
| 尾部插入 | O(1) | 不需要移动元素 |
| 尾部删除 | O(1) | 不需要移动元素 |

## 6. 重点总结

- 顺序表使用连续内存，支持 O(1) 的随机访问
- 插入和删除的平均时间复杂度是 O(n)，因为需要移动元素
- 动态扩容使用 `realloc`，扩容因子一般取 1.5 或 2
- 顺序表适合**读多写少**的场景
- 不适合频繁在中间位置插入删除的场景

> **注意**：动态扩容时，`realloc` 可能会将数据搬迁到新的内存地址，因此不要保存旧的指针。
