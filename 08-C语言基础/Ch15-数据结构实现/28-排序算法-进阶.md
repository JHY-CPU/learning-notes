# 排序算法 - 进阶篇

## 1. 总览

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 |
|------|---------|---------|------|--------|
| 归并排序 | O(n log n) | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) | O(n^2) | O(log n) | 不稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(1) | 不稳定 |

## 2. 归并排序（Merge Sort）

### 思想
分治策略：将数组分成两半，分别排序，然后合并两个有序数组。

```
[8, 3, 5, 1]
分成: [8, 3] [5, 1]
分成: [8] [3] [5] [1]
合并: [3, 8] [1, 5]
合并: [1, 3, 5, 8]
```

### 实现

```c
#include <stdio.h>
#include <stdlib.h>

// 合并两个有序子数组
void merge(int arr[], int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;

    // 创建临时数组
    int *L = (int *)malloc(n1 * sizeof(int));
    int *R = (int *)malloc(n2 * sizeof(int));

    for (int i = 0; i < n1; i++) L[i] = arr[left + i];
    for (int i = 0; i < n2; i++) R[i] = arr[mid + 1 + i];

    // 合并
    int i = 0, j = 0, k = left;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) {
            arr[k++] = L[i++];
        } else {
            arr[k++] = R[j++];
        }
    }
    while (i < n1) arr[k++] = L[i++];
    while (j < n2) arr[k++] = R[j++];

    free(L);
    free(R);
}

// 归并排序
void mergeSort(int arr[], int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    mergeSort(arr, left, mid);
    mergeSort(arr, mid + 1, right);
    merge(arr, left, mid, right);
}
```

### 复杂度分析
- **所有情况**：O(n log n)
- **空间**：O(n)（需要临时数组）
- **稳定**：相等元素保持相对顺序

### 迭代版归并排序

```c
void mergeSortIterative(int arr[], int n) {
    for (int width = 1; width < n; width *= 2) {
        for (int left = 0; left < n; left += 2 * width) {
            int mid = left + width - 1;
            int right = (left + 2 * width - 1 < n) ? left + 2 * width - 1 : n - 1;
            if (mid < right) {
                merge(arr, left, mid, right);
            }
        }
    }
}
```

## 3. 快速排序（Quick Sort）

### 思想
分治策略：选择一个基准值（pivot），将数组分为两部分（小于基准和大于基准），然后递归排序。

```
[8, 3, 5, 1, 9, 2, 7]  pivot=5
partition: [3, 1, 2] [5] [8, 9, 7]
递归排序左边和右边
```

### 实现

```c
// 分区函数
int partition(int arr[], int low, int high) {
    int pivot = arr[high];  // 选择最后一个元素作为基准
    int i = low - 1;

    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;
        }
    }
    // 将 pivot 放到正确位置
    int temp = arr[i + 1]; arr[i + 1] = arr[high]; arr[high] = temp;
    return i + 1;
}

// 快速排序
void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
```

### 优化技巧

```c
// 1. 三数取中选择 pivot
int medianOfThree(int arr[], int low, int high) {
    int mid = low + (high - low) / 2;
    if (arr[low] > arr[mid]) { int t=arr[low]; arr[low]=arr[mid]; arr[mid]=t; }
    if (arr[low] > arr[high]) { int t=arr[low]; arr[low]=arr[high]; arr[high]=t; }
    if (arr[mid] > arr[high]) { int t=arr[mid]; arr[mid]=arr[high]; arr[high]=t; }
    // 将 mid 放到 high-1 位置
    int t=arr[mid]; arr[mid]=arr[high]; arr[high]=t;
    return arr[high];
}

// 2. 小数组切换为插入排序
void quickSortOptimized(int arr[], int low, int high) {
    while (low < high) {
        if (high - low < 10) {
            // 小数组用插入排序
            for (int i = low + 1; i <= high; i++) {
                int key = arr[i], j = i - 1;
                while (j >= low && arr[j] > key) { arr[j+1] = arr[j]; j--; }
                arr[j+1] = key;
            }
            return;
        }
        int pi = partition(arr, low, high);
        // 尾递归优化：先处理较小的子数组
        if (pi - low < high - pi) {
            quickSortOptimized(arr, low, pi - 1);
            low = pi + 1;
        } else {
            quickSortOptimized(arr, pi + 1, high);
            high = pi - 1;
        }
    }
}
```

### 最坏情况
当数组已经有序或逆序时，每次 partition 都极度不平衡，退化为 O(n^2)。

## 4. 堆排序（Heap Sort）

### 思想
利用最大堆的性质：建立最大堆，每次取出堆顶（最大值）放到末尾。

```c
void heapify(int arr[], int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;

    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;

    if (largest != i) {
        int temp = arr[i]; arr[i] = arr[largest]; arr[largest] = temp;
        heapify(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    // 建堆（从最后一个非叶子节点开始）
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    // 排序
    for (int i = n - 1; i > 0; i--) {
        int temp = arr[0]; arr[0] = arr[i]; arr[i] = temp;
        heapify(arr, i, 0);
    }
}
```

### 特点
- 所有情况 O(n log n)
- 空间 O(1)（原地排序）
- **不稳定**

## 5. 完整测试

```c
void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");
}

int main() {
    int a1[] = {8, 3, 5, 1, 9, 2, 7, 4, 6};
    int a2[] = {8, 3, 5, 1, 9, 2, 7, 4, 6};
    int a3[] = {8, 3, 5, 1, 9, 2, 7, 4, 6};
    int n = 9;

    mergeSort(a1, 0, n - 1);
    printf("归并排序: "); printArray(a1, n);

    quickSort(a2, 0, n - 1);
    printf("快速排序: "); printArray(a2, n);

    heapSort(a3, n);
    printf("堆排序:   "); printArray(a3, n);

    return 0;
}
```

## 6. 三种排序对比

| 维度 | 归并排序 | 快速排序 | 堆排序 |
|------|---------|---------|--------|
| 平均时间 | O(n log n) | O(n log n) | O(n log n) |
| 最坏时间 | O(n log n) | O(n^2) | O(n log n) |
| 空间 | O(n) | O(log n) | O(1) |
| 稳定性 | 稳定 | 不稳定 | 不稳定 |
| 实际速度 | 中等 | 最快 | 较慢 |
| 适用场景 | 外部排序、链表排序 | 通用排序 | 内存受限 |

## 7. 重点总结

- **快速排序**是实践中最快的通用排序算法（常数因子小，缓存友好）
- **归并排序**是稳定的 O(n log n) 排序，适合外部排序
- **堆排序**是原地 O(n log n) 排序，适合内存受限场景
- C 标准库 `qsort` 的底层实现通常是**优化的快速排序**或**内省排序**（快排+堆排混合）
- 内省排序（IntroSort）：快排为主，递归深度过大时切换为堆排，小数组用插入排序

> **注意**：快速排序的最坏情况可以通过随机化 pivot 选择来避免。
