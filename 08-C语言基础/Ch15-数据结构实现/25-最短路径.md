# 最短路径算法

## 1. 概述

最短路径问题是图论中的经典问题：找到图中两个顶点之间路径长度之和最小的路径。

### 分类

| 问题类型 | 算法 | 适用场景 |
|---------|------|---------|
| 单源最短路径 | Dijkstra | 非负权图 |
| 单源最短路径 | Bellman-Ford | 可有负权边 |
| 全源最短路径 | Floyd | 任意图 |
| 无权图最短路径 | BFS | 边权为1 |

## 2. Dijkstra 算法

### 2.1 思想

贪心策略：每次从未确定的顶点中选择距离起点最近的顶点，更新其邻居的距离。

```
初始化: 起点距离=0, 其他距离=INF
重复:
  1. 选择距离最小的未访问顶点 u
  2. 标记 u 为已访问
  3. 更新 u 的所有邻居 v 的距离
     dist[v] = min(dist[v], dist[u] + weight(u,v))
```

### 2.2 邻接矩阵实现

```c
#include <stdio.h>
#include <stdbool.h>

#define MAX_V 100
#define INF 999999

// 邻接矩阵
int graph[MAX_V][MAX_V];
int dist[MAX_V];       // 起点到各顶点的最短距离
bool visited[MAX_V];   // 是否已确定最短距离
int prev[MAX_V];       // 前驱节点（用于还原路径）
int vertexCount;

void dijkstra(int start) {
    // 初始化
    for (int i = 0; i < vertexCount; i++) {
        dist[i] = INF;
        visited[i] = false;
        prev[i] = -1;
    }
    dist[start] = 0;

    for (int count = 0; count < vertexCount; count++) {
        // 找距离最小的未访问顶点
        int u = -1;
        int minDist = INF;
        for (int i = 0; i < vertexCount; i++) {
            if (!visited[i] && dist[i] < minDist) {
                minDist = dist[i];
                u = i;
            }
        }
        if (u == -1) break;  // 没有可达的未访问顶点

        visited[u] = true;

        // 更新邻居的距离
        for (int v = 0; v < vertexCount; v++) {
            if (!visited[v] && graph[u][v] != INF) {
                int newDist = dist[u] + graph[u][v];
                if (newDist < dist[v]) {
                    dist[v] = newDist;
                    prev[v] = u;
                }
            }
        }
    }
}

// 还原路径
void printPath(int start, int end) {
    if (dist[end] == INF) {
        printf("无路径\n");
        return;
    }
    int path[MAX_V], count = 0;
    for (int v = end; v != -1; v = prev[v]) {
        path[count++] = v;
    }
    printf("路径: ");
    for (int i = count - 1; i >= 0; i--) {
        printf("%d", path[i]);
        if (i > 0) printf(" -> ");
    }
    printf("\n距离: %d\n", dist[end]);
}
```

### 2.3 使用优先队列优化（简化版）

朴素 Dijkstra 的时间复杂度是 O(V^2)。使用优先队列（最小堆）可优化到 O((V+E)logV)。

```c
// 简化的优先队列（适用于顶点数较少的情况）
typedef struct {
    int vertex;
    int distance;
} PQNode;

void simplePQDijkstra(int adj[][MAX_V], int n, int start) {
    int dist[MAX_V];
    bool visited[MAX_V] = {false};

    for (int i = 0; i < n; i++) dist[i] = INF;
    dist[start] = 0;

    // 使用数组模拟最小堆
    PQNode pq[MAX_V * MAX_V];
    int pqSize = 0;

    // 插入起点
    pq[pqSize].vertex = start;
    pq[pqSize].distance = 0;
    pqSize++;

    while (pqSize > 0) {
        // 找最小值
        int minIdx = 0;
        for (int i = 1; i < pqSize; i++) {
            if (pq[i].distance < pq[minIdx].distance) minIdx = i;
        }
        int u = pq[minIdx].vertex;
        int d = pq[minIdx].distance;
        // 删除
        pq[minIdx] = pq[--pqSize];

        if (visited[u]) continue;
        visited[u] = true;

        for (int v = 0; v < n; v++) {
            if (adj[u][v] != INF && !visited[v]) {
                int newDist = d + adj[u][v];
                if (newDist < dist[v]) {
                    dist[v] = newDist;
                    pq[pqSize].vertex = v;
                    pq[pqSize].distance = newDist;
                    pqSize++;
                }
            }
        }
    }

    printf("从顶点%d的最短距离:\n", start);
    for (int i = 0; i < n; i++) {
        printf("  到%d: %s\n", i, dist[i] == INF ? "不可达" : "");
        if (dist[i] != INF) printf("    距离=%d\n", dist[i]);
    }
}
```

## 3. Floyd 算法（全源最短路径）

### 3.1 思想

动态规划：`dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])`

逐一尝试每个顶点作为中间点，更新所有点对之间的最短距离。

```c
int distF[MAX_V][MAX_V];  // 距离矩阵
int nextF[MAX_V][MAX_V];  // 路径矩阵

void floyd(int n) {
    // 初始化
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            nextF[i][j] = j;

    // 核心三重循环
    for (int k = 0; k < n; k++) {          // 中间点
        for (int i = 0; i < n; i++) {      // 起点
            for (int j = 0; j < n; j++) {  // 终点
                if (distF[i][k] + distF[k][j] < distF[i][j]) {
                    distF[i][j] = distF[i][k] + distF[k][j];
                    nextF[i][j] = nextF[i][k];
                }
            }
        }
    }
}

// 还原 Floyd 路径
void printFloydPath(int u, int v) {
    if (distF[u][v] == INF) {
        printf("无路径\n");
        return;
    }
    printf("%d", u);
    while (u != v) {
        u = nextF[u][v];
        printf(" -> %d", u);
    }
    printf(" (距离=%d)\n", distF[u][v]);
}
```

### 3.2 时间复杂度

| 算法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| Dijkstra（朴素） | O(V^2) | O(V) |
| Dijkstra（堆优化） | O((V+E)log V) | O(V) |
| Floyd | O(V^3) | O(V^2) |

## 4. 完整测试

```c
int main() {
    vertexCount = 5;

    // 初始化邻接矩阵
    for (int i = 0; i < vertexCount; i++)
        for (int j = 0; j < vertexCount; j++)
            graph[i][j] = (i == j) ? 0 : INF;

    // 添加边
    graph[0][1] = 10; graph[0][4] = 5;
    graph[1][2] = 1;  graph[1][4] = 2;
    graph[2][3] = 4;
    graph[3][0] = 7;  graph[3][2] = 6;
    graph[4][1] = 3;  graph[4][2] = 9; graph[4][3] = 2;

    printf("=== Dijkstra ===\n");
    dijkstra(0);
    printPath(0, 3);

    printf("\n=== Floyd ===\n");
    for (int i = 0; i < vertexCount; i++)
        for (int j = 0; j < vertexCount; j++)
            distF[i][j] = graph[i][j];
    floyd(vertexCount);
    printFloydPath(0, 3);

    return 0;
}
```

## 5. 重点总结

| 要点 | 说明 |
|------|------|
| Dijkstra | 贪心，非负权图，单源 |
| Floyd | 动态规划，任意图，全源 |
| Dijkstra 不能处理负权边 | 负权边需用 Bellman-Ford |
| Floyd 能检测负权环 | 对角线出现负值 |
| 路径还原需要前驱数组 | Dijkstra 用 prev，Floyd 用 next |

> **注意**：Dijkstra 的贪心策略保证每次选取的最近顶点的最短距离已确定，但前提是边权非负。负权边会破坏这个性质。
