# 双向链表实现

## 1. 概述

双向链表（Doubly Linked List）是每个节点包含**两个指针**的链表：一个指向前驱节点，一个指向后继节点。这使得双向链表可以**双向遍历**，并且在已知节点位置时删除操作更高效。

### 与单链表对比

| 特性 | 单链表 | 双向链表 |
|------|--------|---------|
| 每个节点指针数 | 1 | 2 |
| 空间开销 | 较小 | 较大（多一个指针） |
| 单方向遍历 | 支持 | 支持 |
| 双方向遍历 | 不支持 | 支持 |
| 已知节点时删除 | O(n) 需要找前驱 | O(1) 直接用 prev |
| 插入在节点前 | O(n) | O(1) |

## 2. 节点定义

```c
#include <stdio.h>
#include <stdlib.h>

// 双向链表节点
typedef struct DNode {
    int data;            // 数据域
    struct DNode *prev;  // 指向前驱节点
    struct DNode *next;  // 指向后继节点
} DNode;

// 双向链表结构
typedef struct {
    DNode *head;         // 头指针（带头节点）
    DNode *tail;         // 尾指针
    int length;          // 链表长度
} DoublyLinkedList;
```

## 3. 基本操作实现

### 3.1 创建与初始化

```c
// 创建新节点
DNode *createDNode(int value) {
    DNode *node = (DNode *)malloc(sizeof(DNode));
    if (!node) return NULL;
    node->data = value;
    node->prev = NULL;
    node->next = NULL;
    return node;
}

// 初始化双向链表
DoublyLinkedList *createDList() {
    DoublyLinkedList *list = (DoublyLinkedList *)malloc(sizeof(DoublyLinkedList));
    if (!list) return NULL;
    list->head = createDNode(0);  // 头节点
    list->tail = list->head;
    list->length = 0;
    return list;
}
```

### 3.2 尾部插入

```c
// 尾部插入
int insertDTail(DoublyLinkedList *list, int value) {
    DNode *newNode = createDNode(value);
    if (!newNode) return 0;
    newNode->prev = list->tail;
    newNode->next = NULL;
    list->tail->next = newNode;
    list->tail = newNode;
    list->length++;
    return 1;
}
```

```
插入前: head -> [A] -> [B] -> NULL
              (tail指向B)
插入C后: head -> [A] -> [B] -> [C] -> NULL
                       prev   prev   prev
              (tail指向C)
```

### 3.3 头部插入

```c
// 头部插入（在第一个数据节点之前）
int insertDHead(DoublyLinkedList *list, int value) {
    DNode *newNode = createDNode(value);
    if (!newNode) return 0;
    DNode *first = list->head->next;
    newNode->prev = list->head;
    newNode->next = first;
    list->head->next = newNode;
    if (first) {
        first->prev = newNode;
    } else {
        list->tail = newNode;  // 链表为空时更新尾指针
    }
    list->length++;
    return 1;
}
```

### 3.4 按位置插入

```c
// 在第 pos 个位置插入
int insertDAt(DoublyLinkedList *list, int pos, int value) {
    if (pos < 0 || pos > list->length) return 0;
    DNode *p = list->head;
    for (int i = 0; i < pos; i++) {
        p = p->next;
    }
    DNode *newNode = createDNode(value);
    if (!newNode) return 0;
    // 调整四个指针
    newNode->prev = p;
    newNode->next = p->next;
    if (p->next) {
        p->next->prev = newNode;
    } else {
        list->tail = newNode;  // 插在末尾，更新尾指针
    }
    p->next = newNode;
    list->length++;
    return 1;
}
```

### 3.5 删除操作

```c
// 删除第 pos 个节点
int deleteDAt(DoublyLinkedList *list, int pos, int *deleted) {
    if (pos < 0 || pos >= list->length) return 0;
    DNode *p = list->head->next;
    for (int i = 0; i < pos; i++) {
        p = p->next;
    }
    *deleted = p->data;
    // 修改前驱的 next
    p->prev->next = p->next;
    // 修改后继的 prev
    if (p->next) {
        p->next->prev = p->prev;
    } else {
        list->tail = p->prev;  // 删除的是尾节点
    }
    free(p);
    list->length--;
    return 1;
}

// 按值删除第一个匹配的节点
int deleteDByValue(DoublyLinkedList *list, int value) {
    DNode *p = list->head->next;
    while (p != NULL) {
        if (p->data == value) {
            p->prev->next = p->next;
            if (p->next) {
                p->next->prev = p->prev;
            } else {
                list->tail = p->prev;
            }
            free(p);
            list->length--;
            return 1;
        }
        p = p->next;
    }
    return 0;
}
```

### 3.6 双向遍历

```c
// 正向遍历
void printDListForward(DoublyLinkedList *list) {
    printf("正向遍历 [长度=%d]: ", list->length);
    DNode *p = list->head->next;
    while (p != NULL) {
        printf("%d", p->data);
        if (p->next) printf(" <-> ");
        p = p->next;
    }
    printf(" -> NULL\n");
}

// 反向遍历
void printDListBackward(DoublyLinkedList *list) {
    printf("反向遍历 [长度=%d]: ", list->length);
    DNode *p = list->tail;
    while (p != list->head) {
        printf("%d", p->data);
        if (p->prev != list->head) printf(" <-> ");
        p = p->prev;
    }
    printf(" -> NULL\n");
}
```

### 3.7 查找操作

```c
// 按值查找
DNode *findDByValue(DoublyLinkedList *list, int value) {
    DNode *p = list->head->next;
    while (p != NULL) {
        if (p->data == value) return p;
        p = p->next;
    }
    return NULL;
}

// 按位置查找
DNode *findDByIndex(DoublyLinkedList *list, int pos) {
    if (pos < 0 || pos >= list->length) return NULL;
    // 优化：判断从头还是从尾遍历更近
    DNode *p;
    if (pos <= list->length / 2) {
        p = list->head->next;
        for (int i = 0; i < pos; i++) p = p->next;
    } else {
        p = list->tail;
        for (int i = list->length - 1; i > pos; i--) p = p->prev;
    }
    return p;
}
```

### 3.8 销毁

```c
void destroyDList(DoublyLinkedList *list) {
    DNode *p = list->head;
    while (p != NULL) {
        DNode *temp = p;
        p = p->next;
        free(temp);
    }
    free(list);
}
```

## 4. 完整测试代码

```c
int main() {
    DoublyLinkedList *list = createDList();

    // 插入
    insertDTail(list, 10);
    insertDTail(list, 20);
    insertDTail(list, 30);
    insertDHead(list, 5);
    insertDAt(list, 2, 15);

    printDListForward(list);   // 5 <-> 10 <-> 15 <-> 20 <-> 30
    printDListBackward(list);  // 30 <-> 20 <-> 15 <-> 10 <-> 5

    // 删除
    int deleted;
    deleteDAt(list, 2, &deleted);
    printf("删除了: %d\n", deleted);  // 15
    printDListForward(list);          // 5 <-> 10 <-> 20 <-> 30

    deleteDByValue(list, 20);
    printDListForward(list);  // 5 <-> 10 <-> 30

    // 反向遍历验证
    printDListBackward(list); // 30 <-> 10 <-> 5

    destroyDList(list);
    return 0;
}
```

## 5. 操作复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 头部/尾部插入 | O(1) | 有头尾指针 |
| 按位置插入 | O(n) | 需要遍历 |
| 已知节点删除 | O(1) | 直接用 prev 指针 |
| 按位置删除 | O(n) | 需要遍历（优化后可O(n/2)） |
| 双向遍历 | O(n) | 可从任意方向 |

## 6. 重点总结

- 双向链表的节点有 `prev` 和 `next` 两个指针
- 可以双向遍历，在某些场景下比单链表更灵活
- 已知节点时，删除操作是 O(1)，这是相比单链表的最大优势
- 按位置查找时，可以从头或尾选择更近的方向遍历
- 注意维护 `prev` 指针，插入和删除时都要更新
- 空间开销比单链表大（每个节点多一个指针）

> **注意**：标准库中的 `list`（C++ STL）就是用双向链表实现的。Linux 内核中也大量使用了双向链表。
