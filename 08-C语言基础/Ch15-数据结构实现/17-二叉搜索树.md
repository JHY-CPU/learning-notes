# 二叉搜索树（BST）

## 1. 概述

二叉搜索树（Binary Search Tree，BST），又称二叉排序树，满足以下性质：
- 左子树中所有节点的值 **小于** 根节点的值
- 右子树中所有节点的值 **大于** 根节点的值
- 左右子树也分别是二叉搜索树

### BST 的中序遍历结果是**升序序列**，这是 BST 最重要的性质。

```
        8
       / \
      3   10
     / \    \
    1   6    14
       / \   /
      4   7 13
```

## 2. 节点定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

typedef struct BSTNode {
    int data;
    struct BSTNode *left;
    struct BSTNode *right;
} BSTNode;

BSTNode *createBSTNode(int value) {
    BSTNode *node = (BSTNode *)malloc(sizeof(BSTNode));
    node->data = value;
    node->left = NULL;
    node->right = NULL;
    return node;
}
```

## 3. 查找操作

### 3.1 递归查找

```c
BSTNode *search(BSTNode *root, int target) {
    if (root == NULL) return NULL;
    if (target == root->data) return root;
    if (target < root->data) return search(root->left, target);
    return search(root->right, target);
}
```

### 3.2 迭代查找

```c
BSTNode *searchIterative(BSTNode *root, int target) {
    while (root != NULL) {
        if (target == root->data) return root;
        else if (target < root->data) root = root->left;
        else root = root->right;
    }
    return NULL;
}
```

### 3.3 查找最小值和最大值

```c
// 最小值：最左节点
BSTNode *findMin(BSTNode *root) {
    while (root->left != NULL) {
        root = root->left;
    }
    return root;
}

// 最大值：最右节点
BSTNode *findMax(BSTNode *root) {
    while (root->right != NULL) {
        root = root->right;
    }
    return root;
}
```

## 4. 插入操作

```c
BSTNode *insert(BSTNode *root, int value) {
    if (root == NULL) {
        return createBSTNode(value);
    }
    if (value < root->data) {
        root->left = insert(root->left, value);
    } else if (value > root->data) {
        root->right = insert(root->right, value);
    }
    // 值已存在，不插入
    return root;
}
```

```
依次插入 8, 3, 10, 1, 6, 14, 4, 7, 13:

8 -> 8,3 -> 8,3,10 -> 8,3,10,1 -> ... -> 完整BST
```

## 5. 删除操作（最复杂）

删除节点有三种情况：

### 情况一：叶子节点
直接删除。

### 情况二：只有一个子节点
用子节点替代被删除节点。

### 情况三：有两个子节点
用**中序后继**（右子树的最小值）或**中序前驱**（左子树的最大值）替代被删除节点，然后删除那个替代节点。

```c
BSTNode *delete(BSTNode *root, int value) {
    if (root == NULL) return NULL;

    if (value < root->data) {
        root->left = delete(root->left, value);
    } else if (value > root->data) {
        root->right = delete(root->right, value);
    } else {
        // 找到要删除的节点
        // 情况一和二：没有子节点或只有一个子节点
        if (root->left == NULL) {
            BSTNode *temp = root->right;
            free(root);
            return temp;
        } else if (root->right == NULL) {
            BSTNode *temp = root->left;
            free(root);
            return temp;
        }
        // 情况三：有两个子节点
        // 找到右子树的最小值（中序后继）
        BSTNode *successor = findMin(root->right);
        root->data = successor->data;
        root->right = delete(root->right, successor->data);
    }
    return root;
}
```

```
删除节点3（有两个子节点）:
    前: 8,3,10,1,6     找到中序后继=4
    后: 8,4,10,1,6     (用4替代3，再删除原4的位置)
```

## 6. 中序遍历验证有序性

```c
void inorder(BSTNode *root) {
    if (root == NULL) return;
    inorder(root->left);
    printf("%d ", root->data);
    inorder(root->right);
}
```

## 7. 其他常用操作

### 7.1 求 BST 中第 k 小的元素

```c
// 使用中序遍历计数
int count = 0;
BSTNode *kthSmallest(BSTNode *root, int k) {
    if (root == NULL) return NULL;
    BSTNode *left = kthSmallest(root->left, k);
    if (left) return left;
    count++;
    if (count == k) return root;
    return kthSmallest(root->right, k);
}
```

### 7.2 验证是否为有效的 BST

```c
bool isValidBSTHelper(BSTNode *root, long minVal, long maxVal) {
    if (root == NULL) return true;
    if (root->data <= minVal || root->data >= maxVal) return false;
    return isValidBSTHelper(root->left, minVal, root->data) &&
           isValidBSTHelper(root->right, root->data, maxVal);
}

bool isValidBST(BSTNode *root) {
    return isValidBSTHelper(root, -2147483648L, 2147483647L);
}
```

### 7.3 找公共祖先（LCA）

```c
BSTNode *lowestCommonAncestor(BSTNode *root, int p, int q) {
    if (root == NULL) return NULL;
    if (p < root->data && q < root->data)
        return lowestCommonAncestor(root->left, p, q);
    if (p > root->data && q > root->data)
        return lowestCommonAncestor(root->right, p, q);
    return root;  // p 和 q 分居两侧，当前节点就是 LCA
}
```

## 8. 完整测试代码

```c
int main() {
    BSTNode *root = NULL;
    int values[] = {8, 3, 10, 1, 6, 14, 4, 7, 13};
    int n = sizeof(values) / sizeof(values[0]);

    for (int i = 0; i < n; i++) {
        root = insert(root, values[i]);
    }

    printf("中序遍历（升序）: ");
    inorder(root);  // 1 3 4 6 7 8 10 13 14
    printf("\n");

    // 查找
    BSTNode *found = search(root, 6);
    printf("查找6: %s\n", found ? "找到" : "未找到");

    printf("最小值: %d\n", findMin(root)->data);  // 1
    printf("最大值: %d\n", findMax(root)->data);  // 14

    // 删除
    root = delete(root, 3);
    printf("删除3后中序: ");
    inorder(root);  // 1 4 6 7 8 10 13 14
    printf("\n");

    printf("是否有效BST: %s\n", isValidBST(root) ? "是" : "否");

    return 0;
}
```

## 9. BST 的性能分析

| 操作 | 平均时间 | 最坏时间（退化成链表） |
|------|---------|---------------------|
| 查找 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

## 10. 重点总结

- BST 的中序遍历是有序的（升序）
- 查找类似于二分查找，每次排除一半
- 删除操作最复杂，分三种情况处理
- BST 在最坏情况下会退化为链表（插入有序数据时）
- 为了解决退化问题，引入了**平衡二叉搜索树**（AVL树、红黑树）

> **注意**：BST 不保证平衡。如果按 1, 2, 3, 4, 5 的顺序插入，BST 会退化为一条链。这是 BST 的主要缺点。
