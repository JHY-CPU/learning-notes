# 排序算法 - 特殊排序

## 1. 概述

前面介绍的排序算法都是基于**比较**的，时间复杂度下限为 O(n log n)。本节介绍的算法不依赖比较，可以突破这个限制。

## 2. 计数排序（Counting Sort）

### 思想
统计每个值出现的次数，然后按顺序输出。适用于**整数且范围不大**的情况。

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

void countingSort(int arr[], int n) {
    // 找最大值和最小值
    int max = arr[0], min = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
        if (arr[i] < min) min = arr[i];
    }

    int range = max - min + 1;
    int *count = (int *)calloc(range, sizeof(int));
    int *output = (int *)malloc(n * sizeof(int));

    // 统计每个元素出现的次数
    for (int i = 0; i < n; i++) {
        count[arr[i] - min]++;
    }

    // 累加计数（计算每个元素的最终位置）
    for (int i = 1; i < range; i++) {
        count[i] += count[i - 1];
    }

    // 从后往前填充（保证稳定性）
    for (int i = n - 1; i >= 0; i--) {
        output[count[arr[i] - min] - 1] = arr[i];
        count[arr[i] - min]--;
    }

    // 复制回原数组
    for (int i = 0; i < n; i++) {
        arr[i] = output[i];
    }

    free(count);
    free(output);
}
```

### 复杂度
- 时间：O(n + k)，k 是数据范围
- 空间：O(n + k)
- **稳定**

### 适用场景
- 数据范围小（k 不远大于 n）
- 非负整数（或可以映射为非负整数）

## 3. 基数排序（Radix Sort）

### 思想
从低位到高位，依次对每一位进行排序（通常使用计数排序作为子排序）。

```c
// 获取数组中的最大值
int getMax(int arr[], int n) {
    int max = arr[0];
    for (int i = 1; i < n; i++)
        if (arr[i] > max) max = arr[i];
    return max;
}

// 对指定位（exp=1,10,100...）进行计数排序
void countSortByDigit(int arr[], int n, int exp) {
    int *output = (int *)malloc(n * sizeof(int));
    int count[10] = {0};

    // 统计当前位的频次
    for (int i = 0; i < n; i++) {
        count[(arr[i] / exp) % 10]++;
    }

    // 累加
    for (int i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }

    // 从后往前填充
    for (int i = n - 1; i >= 0; i--) {
        output[count[(arr[i] / exp) % 10] - 1] = arr[i];
        count[(arr[i] / exp) % 10]--;
    }

    for (int i = 0; i < n; i++) arr[i] = output[i];
    free(output);
}

// LSD 基数排序（从最低位开始）
void radixSort(int arr[], int n) {
    int max = getMax(arr, n);
    // 对每一位进行计数排序
    for (int exp = 1; max / exp > 0; exp *= 10) {
        countSortByDigit(arr, n, exp);
    }
}
```

```
[170, 45, 75, 90, 802, 24, 2, 66]

按个位排序: [170, 90, 802, 2, 24, 45, 75, 66]
按十位排序: [802, 2, 24, 45, 66, 170, 75, 90]
按百位排序: [2, 24, 45, 66, 75, 90, 170, 802]
```

### 复杂度
- 时间：O(d * (n + k))，d 是位数，k 是基数（10）
- 空间：O(n + k)
- **稳定**

## 4. 桶排序（Bucket Sort）

### 思想
将数据分到多个桶中，每个桶内单独排序，最后合并。

```c
// 简化版桶排序（假设数据在 [0, 1) 范围内）
void bucketSort(float arr[], int n) {
    // 创建 n 个桶
    int *bucketSize = (int *)calloc(n, sizeof(int));
    float **buckets = (float **)malloc(n * sizeof(float *));
    for (int i = 0; i < n; i++) {
        buckets[i] = (float *)malloc(n * sizeof(float));
    }

    // 将元素分配到桶中
    for (int i = 0; i < n; i++) {
        int idx = (int)(n * arr[i]);  // 桶的索引
        if (idx >= n) idx = n - 1;
        buckets[idx][bucketSize[idx]++] = arr[i];
    }

    // 对每个桶排序（用插入排序）
    for (int i = 0; i < n; i++) {
        for (int j = 1; j < bucketSize[i]; j++) {
            float key = buckets[i][j];
            int k = j - 1;
            while (k >= 0 && buckets[i][k] > key) {
                buckets[i][k + 1] = buckets[i][k];
                k--;
            }
            buckets[i][k + 1] = key;
        }
    }

    // 合并
    int idx = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < bucketSize[i]; j++) {
            arr[idx++] = buckets[i][j];
        }
    }

    for (int i = 0; i < n; i++) free(buckets[i]);
    free(buckets);
    free(bucketSize);
}
```

### 复杂度
- 平均：O(n + k)，k 是桶的数量
- 最坏：O(n^2)（所有元素落入同一个桶）
- **稳定**（取决于桶内排序算法）

## 5. 特殊排序对比

| 算法 | 时间 | 空间 | 稳定 | 适用场景 |
|------|------|------|------|---------|
| 计数排序 | O(n+k) | O(n+k) | 是 | 整数，范围小 |
| 基数排序 | O(d(n+k)) | O(n+k) | 是 | 整数/字符串，位数固定 |
| 桶排序 | O(n+k) | O(n+k) | 是 | 均匀分布的数据 |

## 6. 完整测试

```c
void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");
}

int main() {
    int a1[] = {4, 2, 2, 8, 3, 3, 1};
    countingSort(a1, 7);
    printf("计数排序: "); printArray(a1, 7);

    int a2[] = {170, 45, 75, 90, 802, 24, 2, 66};
    radixSort(a2, 8);
    printf("基数排序: "); printArray(a2, 8);

    float a3[] = {0.42, 0.32, 0.53, 0.24, 0.15, 0.66, 0.77};
    bucketSort(a3, 7);
    printf("桶排序: ");
    for (int i = 0; i < 7; i++) printf("%.2f ", a3[i]);
    printf("\n");

    return 0;
}
```

## 7. 重点总结

- 非比较排序可以突破 O(n log n) 的限制
- **计数排序**：适合范围小的整数，统计频次
- **基数排序**：按位排序，从低位到高位（LSD）
- **桶排序**：分桶处理，适合均匀分布的数据
- 这些算法的局限性是需要额外空间，且适用条件有限
- 实际开发中，快排/归并等比较排序更通用

> **选择建议**：当数据特性满足条件时（整数、范围小、均匀分布），非比较排序效率更高。
