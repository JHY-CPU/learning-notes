# 33 - 跳表思想

## 概述

跳表（Skip List）是一种基于概率的有序数据结构，通过多层索引实现O(log n)的查找、插入和删除。它是平衡树的替代方案，实现简单且在并发场景下有优势。

---

## 1. 基本思想

跳表在有序链表的基础上，添加多层"快速通道"（索引层），每一层都是下一层的子集。

```
Level 3:  head ----------------------> 50 -------------------------> NULL
Level 2:  head ------> 20 -----------> 50 --------> 70 -----------> NULL
Level 1:  head -> 10 -> 20 -> 30 ----> 50 -> 60 -> 70 -> 80 -----> NULL
Level 0:  head -> 10 -> 20 -> 25 -> 30 -> 50 -> 60 -> 70 -> 80 -> 90 -> NULL
```

查找50的过程：
- Level 3: head -> 50（直接找到）

查找25的过程：
- Level 3: head -> 50（50 > 25，下移）
- Level 2: head -> 20 -> 50（20 < 25 < 50，下移）
- Level 1: 20 -> 30（30 > 25，下移）
- Level 0: 20 -> 25（找到）

## 2. 完整实现

```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
#include <limits.h>

#define MAX_LEVEL 16   // 最大层数
#define P 0.5          // 每层晋升概率

typedef struct SkipListNode {
    int key;
    int value;
    struct SkipListNode **forward;  // 每层的前向指针数组
} SkipListNode;

typedef struct {
    SkipListNode *header;
    int level;       // 当前最大层数
    int size;
} SkipList;

// 创建节点
SkipListNode *create_node(int level, int key, int value) {
    SkipListNode *node = (SkipListNode *)malloc(sizeof(SkipListNode));
    node->key = key;
    node->value = value;
    node->forward = (SkipListNode **)malloc((level + 1) * sizeof(SkipListNode *));
    for (int i = 0; i <= level; i++) {
        node->forward[i] = NULL;
    }
    return node;
}

// 创建跳表
SkipList *skiplist_create(void) {
    SkipList *list = (SkipList *)malloc(sizeof(SkipList));
    list->header = create_node(MAX_LEVEL, INT_MIN, 0);
    list->level = 0;
    list->size = 0;
    srand((unsigned)time(NULL));
    return list;
}

// 随机生成层数
int random_level(void) {
    int level = 0;
    while ((rand() & 0xFFFF) < (P * 0xFFFF) && level < MAX_LEVEL - 1) {
        level++;
    }
    return level;
}

// 查找
SkipListNode *skiplist_search(SkipList *list, int key) {
    SkipListNode *curr = list->header;
    for (int i = list->level; i >= 0; i--) {
        while (curr->forward[i] && curr->forward[i]->key < key) {
            curr = curr->forward[i];
        }
    }
    curr = curr->forward[0];
    if (curr && curr->key == key)
        return curr;
    return NULL;
}

// 插入
int skiplist_insert(SkipList *list, int key, int value) {
    SkipListNode *update[MAX_LEVEL + 1];  // 每层需要更新的节点
    SkipListNode *curr = list->header;

    // 从高层到低层查找插入位置
    for (int i = list->level; i >= 0; i--) {
        while (curr->forward[i] && curr->forward[i]->key < key) {
            curr = curr->forward[i];
        }
        update[i] = curr;
    }

    curr = curr->forward[0];

    // 如果key已存在，更新值
    if (curr && curr->key == key) {
        curr->value = value;
        return 0;
    }

    // 随机决定新节点的层数
    int new_level = random_level();

    // 如果新层数超过当前最大层数
    if (new_level > list->level) {
        for (int i = list->level + 1; i <= new_level; i++) {
            update[i] = list->header;
        }
        list->level = new_level;
    }

    // 创建新节点
    SkipListNode *new_node = create_node(new_level, key, value);

    // 更新每层的指针
    for (int i = 0; i <= new_level; i++) {
        new_node->forward[i] = update[i]->forward[i];
        update[i]->forward[i] = new_node;
    }

    list->size++;
    return 1;
}

// 删除
int skiplist_delete(SkipList *list, int key) {
    SkipListNode *update[MAX_LEVEL + 1];
    SkipListNode *curr = list->header;

    for (int i = list->level; i >= 0; i--) {
        while (curr->forward[i] && curr->forward[i]->key < key) {
            curr = curr->forward[i];
        }
        update[i] = curr;
    }

    curr = curr->forward[0];
    if (curr == NULL || curr->key != key)
        return 0;  // 未找到

    // 更新每层的指针
    for (int i = 0; i <= list->level; i++) {
        if (update[i]->forward[i] != curr)
            break;
        update[i]->forward[i] = curr->forward[i];
    }

    free(curr->forward);
    free(curr);

    // 更新最大层数
    while (list->level > 0 && list->header->forward[list->level] == NULL) {
        list->level--;
    }

    list->size--;
    return 1;
}

// 打印跳表
void skiplist_print(SkipList *list) {
    printf("跳表 (size=%d, level=%d):\n", list->size, list->level);
    for (int i = list->level; i >= 0; i--) {
        SkipListNode *node = list->header->forward[i];
        printf("Level %d: ", i);
        while (node) {
            printf("%d ", node->key);
            node = node->forward[i];
        }
        printf("\n");
    }
}

// 释放跳表
void skiplist_free(SkipList *list) {
    SkipListNode *curr = list->header->forward[0];
    while (curr) {
        SkipListNode *next = curr->forward[0];
        free(curr->forward);
        free(curr);
        curr = next;
    }
    free(list->header->forward);
    free(list->header);
    free(list);
}
```

## 3. 测试代码

```c
int main(void) {
    SkipList *list = skiplist_create();

    // 插入元素
    int keys[] = {3, 6, 7, 9, 12, 19, 17, 26, 21, 25};
    int n = sizeof(keys) / sizeof(keys[0]);

    for (int i = 0; i < n; i++) {
        skiplist_insert(list, keys[i], keys[i] * 10);
    }

    skiplist_print(list);

    // 查找
    int search_keys[] = {7, 15, 25};
    for (int i = 0; i < 3; i++) {
        SkipListNode *node = skiplist_search(list, search_keys[i]);
        if (node)
            printf("找到 %d: value=%d\n", search_keys[i], node->value);
        else
            printf("未找到 %d\n", search_keys[i]);
    }

    // 删除
    printf("\n删除 7 和 19 后:\n");
    skiplist_delete(list, 7);
    skiplist_delete(list, 19);
    skiplist_print(list);

    skiplist_free(list);
    return 0;
}
```

## 4. 复杂度分析

| 操作 | 平均时间 | 最坏时间 | 空间 |
|------|----------|----------|------|
| 查找 | O(log n) | O(n) | O(1) |
| 插入 | O(log n) | O(n) | O(log n) |
| 删除 | O(log n) | O(n) | O(1) |

- 平均层数：O(log n)
- 每层元素数约为上层的 1/P 倍
- 空间复杂度：O(n)（平均每节点约 1/(1-P) 个指针）

---

## 5. 与平衡树对比

| 特性 | 跳表 | 红黑树 |
|------|------|--------|
| 实现复杂度 | 简单 | 复杂 |
| 并发友好 | 是 | 否 |
| 范围查询 | 简单 | 需要特殊处理 |
| 内存局部性 | 较差（链表） | 较好 |
| 应用 | Redis, LevelDB | STL map/set |

---

## 要点总结

1. 跳表通过随机层数实现概率性平衡，避免了确定性平衡的复杂操作
2. 实现简单，天然支持范围查询
3. 在并发场景下，跳表比红黑树更容易实现无锁操作
4. Redis的有序集合（ZSET）和LevelDB都使用了跳表
