# 二叉树遍历

## 1. 概述

二叉树遍历是指按照某种规则访问树中的每个节点，使得每个节点被访问且仅被访问一次。主要有四种遍历方式。

### 遍历方式总览

```
        A
       / \
      B   C
     / \   \
    D   E   F
```

| 遍历方式 | 访问顺序 | 结果 |
|---------|---------|------|
| 前序遍历 | 根-左-右 | A B D E C F |
| 中序遍历 | 左-根-右 | D B E A C F |
| 后序遍历 | 左-右-根 | D E B F C A |
| 层序遍历 | 层层推进 | A B C D E F |

## 2. 递归遍历

### 2.1 前序遍历（Pre-order）

先访问根节点，再遍历左子树，最后遍历右子树。

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

// 前序遍历：根 -> 左 -> 右
void preorder(TreeNode *root) {
    if (root == NULL) return;
    printf("%d ", root->data);   // 访问根
    preorder(root->left);        // 遍历左子树
    preorder(root->right);       // 遍历右子树
}
```

### 2.2 中序遍历（In-order）

先遍历左子树，再访问根节点，最后遍历右子树。

```c
// 中序遍历：左 -> 根 -> 右
void inorder(TreeNode *root) {
    if (root == NULL) return;
    inorder(root->left);
    printf("%d ", root->data);
    inorder(root->right);
}
```

### 2.3 后序遍历（Post-order）

先遍历左子树，再遍历右子树，最后访问根节点。

```c
// 后序遍历：左 -> 右 -> 根
void postorder(TreeNode *root) {
    if (root == NULL) return;
    postorder(root->left);
    postorder(root->right);
    printf("%d ", root->data);
}
```

## 3. 非递归遍历（用栈模拟）

递归本质是利用了函数调用栈，非递归方法则显式地使用栈来模拟。

### 3.1 前序遍历非递归

```c
// 非递归前序遍历
void preorderIterative(TreeNode *root) {
    if (!root) return;
    TreeNode *stack[1000];
    int top = -1;
    stack[++top] = root;

    while (top >= 0) {
        TreeNode *node = stack[top--];
        printf("%d ", node->data);  // 访问

        // 先右后左入栈（因为栈是后进先出）
        if (node->right) stack[++top] = node->right;
        if (node->left) stack[++top] = node->left;
    }
}
```

### 3.2 中序遍历非递归

核心思想：一直向左走，将沿途节点入栈；到最左后弹出并访问，然后转向右子树。

```c
// 非递归中序遍历
void inorderIterative(TreeNode *root) {
    TreeNode *stack[1000];
    int top = -1;
    TreeNode *curr = root;

    while (curr != NULL || top >= 0) {
        // 一直向左走，沿途节点入栈
        while (curr != NULL) {
            stack[++top] = curr;
            curr = curr->left;
        }
        // 弹出栈顶并访问
        curr = stack[top--];
        printf("%d ", curr->data);
        // 转向右子树
        curr = curr->right;
    }
}
```

### 3.3 后序遍历非递归

方法：使用两个栈，或者在前序遍历基础上修改（根->右->左，然后反转）。

```c
// 非递归后序遍历（双栈法）
void postorderIterative(TreeNode *root) {
    if (!root) return;
    TreeNode *stack1[1000], *stack2[1000];
    int top1 = -1, top2 = -1;

    stack1[++top1] = root;
    while (top1 >= 0) {
        TreeNode *node = stack1[top1--];
        stack2[++top2] = node;

        if (node->left) stack1[++top1] = node->left;
        if (node->right) stack1[++top1] = node->right;
    }
    // stack2 中的顺序就是后序遍历
    while (top2 >= 0) {
        printf("%d ", stack2[top2--]->data);
    }
}
```

### 3.4 后序遍历（单栈法）

```c
// 非递归后序遍历（单栈法）
void postorderSingleStack(TreeNode *root) {
    if (!root) return;
    TreeNode *stack[1000];
    int top = -1;
    TreeNode *curr = root;
    TreeNode *lastVisited = NULL;

    while (curr != NULL || top >= 0) {
        while (curr != NULL) {
            stack[++top] = curr;
            curr = curr->left;
        }
        TreeNode *node = stack[top];
        // 如果右子树为空或已访问过，则访问当前节点
        if (node->right == NULL || node->right == lastVisited) {
            printf("%d ", node->data);
            lastVisited = node;
            top--;
        } else {
            curr = node->right;
        }
    }
}
```

## 4. 层序遍历（用队列）

按层次从上到下、从左到右访问每个节点。

```c
// 层序遍历
void levelOrder(TreeNode *root) {
    if (!root) return;
    TreeNode *queue[1000];
    int front = 0, rear = 0;

    queue[rear++] = root;

    while (front < rear) {
        TreeNode *node = queue[front++];
        printf("%d ", node->data);

        if (node->left) queue[rear++] = node->left;
        if (node->right) queue[rear++] = node->right;
    }
}

// 分层打印
void levelOrderByLevel(TreeNode *root) {
    if (!root) return;
    TreeNode *queue[1000];
    int front = 0, rear = 0;
    queue[rear++] = root;

    while (front < rear) {
        int levelSize = rear - front;
        for (int i = 0; i < levelSize; i++) {
            TreeNode *node = queue[front++];
            printf("%d ", node->data);
            if (node->left) queue[rear++] = node->left;
            if (node->right) queue[rear++] = node->right;
        }
        printf("\n");
    }
}
```

## 5. 根据遍历序列重建二叉树

### 5.1 前序 + 中序 重建

```c
// 在中序序列中查找值的位置
int findIndex(int *inorder, int inStart, int inEnd, int value) {
    for (int i = inStart; i <= inEnd; i++) {
        if (inorder[i] == value) return i;
    }
    return -1;
}

TreeNode *buildFromPreIn(int *preorder, int *inorder,
                         int inStart, int inEnd, int *preIndex) {
    if (inStart > inEnd) return NULL;

    int rootVal = preorder[(*preIndex)++];
    TreeNode *root = (TreeNode *)malloc(sizeof(TreeNode));
    root->data = rootVal;
    root->left = root->right = NULL;

    if (inStart == inEnd) return root;

    int inIndex = findIndex(inorder, inStart, inEnd, rootVal);
    root->left = buildFromPreIn(preorder, inorder, inStart, inIndex - 1, preIndex);
    root->right = buildFromPreIn(preorder, inorder, inIndex + 1, inEnd, preIndex);
    return root;
}
```

## 6. 完整测试代码

```c
TreeNode *buildSampleTree() {
    TreeNode *root = (TreeNode *)malloc(sizeof(TreeNode));
    root->data = 1;
    root->left = (TreeNode *)malloc(sizeof(TreeNode));
    root->left->data = 2;
    root->left->left = (TreeNode *)malloc(sizeof(TreeNode));
    root->left->left->data = 4;
    root->left->left->left = root->left->left->right = NULL;
    root->left->right = (TreeNode *)malloc(sizeof(TreeNode));
    root->left->right->data = 5;
    root->left->right->left = root->left->right->right = NULL;
    root->right = (TreeNode *)malloc(sizeof(TreeNode));
    root->right->data = 3;
    root->right->left = NULL;
    root->right->right = (TreeNode *)malloc(sizeof(TreeNode));
    root->right->right->data = 6;
    root->right->right->left = root->right->right->right = NULL;
    return root;
}

int main() {
    TreeNode *root = buildSampleTree();

    printf("前序遍历(递归): "); preorder(root); printf("\n");
    printf("中序遍历(递归): "); inorder(root); printf("\n");
    printf("后序遍历(递归): "); postorder(root); printf("\n");
    printf("前序遍历(迭代): "); preorderIterative(root); printf("\n");
    printf("中序遍历(迭代): "); inorderIterative(root); printf("\n");
    printf("后序遍历(迭代): "); postorderIterative(root); printf("\n");
    printf("层序遍历:       "); levelOrder(root); printf("\n");
    printf("\n分层打印:\n"); levelOrderByLevel(root);

    return 0;
}
```

## 7. 复杂度分析

| 遍历方式 | 时间复杂度 | 空间复杂度（递归/迭代） |
|---------|-----------|---------------------|
| 前序 | O(n) | O(h) / O(h) |
| 中序 | O(n) | O(h) / O(h) |
| 后序 | O(n) | O(h) / O(h) |
| 层序 | O(n) | O(w) |

> h = 树的高度，w = 树的最大宽度

## 8. 重点总结

- 前序遍历：**根在前**，第一个访问的是根节点
- 中序遍历：**根在中**，可用于BST排序输出
- 后序遍历：**根在后**，第一个访问的是最左下的节点，常用于释放内存
- 层序遍历：使用**队列**实现，不使用栈
- 非递归中序遍历是最常考的迭代遍历
- 根据前序+中序或中序+后序可以唯一确定一棵二叉树

> **记忆技巧**：前中后序的区别只在于"访问根节点"在递归调用的位置。放在前面是前序，中间是中序，后面是后序。
