# 栈的链表实现（链式栈）

## 1. 概述

链式栈（Linked Stack）使用链表来实现栈结构。将链表的头部作为栈顶，利用头插法实现 push，头删法实现 pop。链式栈没有容量限制（只受内存限制）。

### 顺序栈 vs 链式栈

| 特性 | 顺序栈 | 链式栈 |
|------|--------|--------|
| 存储方式 | 数组 | 链表 |
| 容量限制 | 固定（或需扩容） | 无限制 |
| 空间利用 | 可能浪费 | 按需分配 |
| 每个元素额外开销 | 无 | 一个指针 |
| 适用场景 | 大小可预估 | 大小不确定 |

## 2. 数据结构定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 栈节点
typedef struct StackNode {
    int data;
    struct StackNode *next;
} StackNode;

// 链式栈结构
typedef struct {
    StackNode *top;    // 栈顶指针
    int size;          // 栈中元素个数
} LinkedStack;
```

## 3. 基本操作实现

### 3.1 创建与初始化

```c
// 创建新节点
StackNode *createStackNode(int value) {
    StackNode *node = (StackNode *)malloc(sizeof(StackNode));
    if (!node) return NULL;
    node->data = value;
    node->next = NULL;
    return node;
}

// 初始化栈
LinkedStack *createStack() {
    LinkedStack *s = (LinkedStack *)malloc(sizeof(LinkedStack));
    if (!s) return NULL;
    s->top = NULL;
    s->size = 0;
    return s;
}
```

### 3.2 push 操作（头插法）

```c
// 压栈 - O(1)
bool push(LinkedStack *s, int value) {
    StackNode *newNode = createStackNode(value);
    if (!newNode) return false;
    newNode->next = s->top;
    s->top = newNode;
    s->size++;
    return true;
}
```

```
push(10): top -> [10] -> NULL
push(20): top -> [20] -> [10] -> NULL
push(30): top -> [30] -> [20] -> [10] -> NULL
```

### 3.3 pop 操作（头删法）

```c
// 弹栈 - O(1)
bool pop(LinkedStack *s, int *value) {
    if (s->top == NULL) {
        printf("栈空!\n");
        return false;
    }
    StackNode *temp = s->top;
    *value = temp->data;
    s->top = temp->next;
    free(temp);
    s->size--;
    return true;
}
```

```
pop: top -> [20] -> [10] -> NULL, 返回 30
pop: top -> [10] -> NULL,         返回 20
```

### 3.4 peek 操作

```c
// 查看栈顶 - O(1)
bool peek(LinkedStack *s, int *value) {
    if (s->top == NULL) return false;
    *value = s->top->data;
    return true;
}
```

### 3.5 辅助操作

```c
// 判断栈空
bool isEmpty(LinkedStack *s) {
    return s->top == NULL;
}

// 获取栈大小
int getSize(LinkedStack *s) {
    return s->size;
}

// 打印栈
void printStack(LinkedStack *s) {
    printf("栈顶 -> ");
    StackNode *p = s->top;
    while (p != NULL) {
        printf("[%d]", p->data);
        if (p->next) printf(" -> ");
        p = p->next;
    }
    printf(" -> NULL (共%d个元素)\n", s->size);
}

// 销毁栈
void destroyStack(LinkedStack *s) {
    StackNode *p = s->top;
    while (p != NULL) {
        StackNode *temp = p;
        p = p->next;
        free(temp);
    }
    free(s);
}
```

## 4. 完整测试代码

```c
int main() {
    LinkedStack *stack = createStack();

    // 压栈
    push(stack, 10);
    push(stack, 20);
    push(stack, 30);
    push(stack, 40);
    printStack(stack);  // 栈顶 -> [40] -> [30] -> [20] -> [10] -> NULL

    // 查看栈顶
    int top;
    peek(stack, &top);
    printf("栈顶元素: %d\n", top);  // 40
    printf("栈大小: %d\n", getSize(stack));  // 4

    // 弹栈
    int val;
    while (!isEmpty(stack)) {
        pop(stack, &val);
        printf("弹出: %d\n", val);
    }
    // 弹出: 40, 30, 20, 10

    destroyStack(stack);
    return 0;
}
```

## 5. 用链式栈实现栈的反转

```c
// 使用栈反转数组
void reverseArray(int arr[], int n) {
    LinkedStack *stack = createStack();
    // 全部压栈
    for (int i = 0; i < n; i++) {
        push(stack, arr[i]);
    }
    // 全部弹出（顺序相反）
    for (int i = 0; i < n; i++) {
        pop(stack, &arr[i]);
    }
    destroyStack(stack);
}
```

## 6. 多栈问题

在某些场景中，一个进程需要多个栈。使用链式栈可以更灵活地管理：

```c
// 两个独立的栈
typedef struct {
    LinkedStack *stack1;
    LinkedStack *stack2;
} DualStack;

DualStack *createDualStack() {
    DualStack *ds = (DualStack *)malloc(sizeof(DualStack));
    ds->stack1 = createStack();
    ds->stack2 = createStack();
    return ds;
}
```

## 7. 操作复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| push | O(1) | 头插法 |
| pop | O(1) | 头删法 |
| peek | O(1) | 访问头节点 |
| isEmpty | O(1) | 判断 top |
| getSize | O(1) | 维护 size 变量 |

## 8. 重点总结

- 链式栈没有容量上限，按需分配内存
- 栈顶就是链表头，push 和 pop 都是 O(1) 操作
- 不需要预先确定栈的最大容量
- 每个元素需要额外存储一个指针，空间开销略大
- 适合栈大小不确定或需要频繁创建销毁栈的场景

> **注意**：pop 时不要忘记释放被删除节点的内存。
