# 最小生成树

## 1. 概述

**最小生成树（Minimum Spanning Tree, MST）**是连通无向加权图的一棵生成树，使得所有边的权重之和最小。

### 生成树的性质
- 包含图中所有顶点
- 是一棵树（无环）
- 边数 = 顶点数 - 1

### 两种经典算法
- **Prim 算法**：从顶点出发，逐步扩展树
- **Kruskal 算法**：从边出发，按权重排序选择

## 2. Prim 算法

### 2.1 思想

类似 Dijkstra，从任意顶点开始，每次选择连接树内和树外的最小权重边，将对应顶点加入树。

```
初始: 选择任意顶点加入 MST
重复 V-1 次:
  1. 找到连接 MST 内部顶点和外部顶点的最小边
  2. 将该边和对应顶点加入 MST
```

### 2.2 实现

```c
#include <stdio.h>
#include <stdbool.h>

#define MAX_V 100
#define INF 999999

int graph[MAX_V][MAX_V];  // 邻接矩阵
int vertexCount;

void prim() {
    int parent[MAX_V];     // MST 中每个顶点的父节点
    int key[MAX_V];        // 连接到 MST 的最小边权重
    bool inMST[MAX_V];     // 是否在 MST 中

    // 初始化
    for (int i = 0; i < vertexCount; i++) {
        key[i] = INF;
        inMST[i] = false;
        parent[i] = -1;
    }
    key[0] = 0;  // 从顶点0开始

    for (int count = 0; count < vertexCount; count++) {
        // 找 key 值最小的不在 MST 中的顶点
        int u = -1;
        int minKey = INF;
        for (int v = 0; v < vertexCount; v++) {
            if (!inMST[v] && key[v] < minKey) {
                minKey = key[v];
                u = v;
            }
        }
        if (u == -1) break;

        inMST[u] = true;

        // 更新邻居的 key 值
        for (int v = 0; v < vertexCount; v++) {
            if (graph[u][v] != INF && !inMST[v] &&
                graph[u][v] < key[v]) {
                key[v] = graph[u][v];
                parent[v] = u;
            }
        }
    }

    // 输出 MST
    int totalWeight = 0;
    printf("Prim 最小生成树:\n");
    for (int i = 1; i < vertexCount; i++) {
        if (parent[i] != -1) {
            printf("  边: %d - %d, 权重: %d\n", parent[i], i, graph[parent[i]][i]);
            totalWeight += graph[parent[i]][i];
        }
    }
    printf("总权重: %d\n", totalWeight);
}
```

### 2.3 时间复杂度
- 朴素实现：O(V^2)
- 堆优化：O((V+E) log V)

## 3. Kruskal 算法

### 3.1 思想

将所有边按权重排序，从小到大选择边，如果加入该边不会形成环（两端点不在同一连通分量），则加入 MST。

```
1. 将所有边按权重从小到大排序
2. 初始化并查集
3. 遍历每条边 (u, v):
   如果 u 和 v 不在同一集合:
     选择这条边
     合并 u 和 v 所在集合
4. 选够 V-1 条边后结束
```

### 3.2 并查集实现

```c
int parentDSU[MAX_V];
int rankDSU[MAX_V];

void initDSU(int n) {
    for (int i = 0; i < n; i++) {
        parentDSU[i] = i;
        rankDSU[i] = 0;
    }
}

int findDSU(int x) {
    if (parentDSU[x] != x)
        parentDSU[x] = findDSU(parentDSU[x]);  // 路径压缩
    return parentDSU[x];
}

bool unionDSU(int x, int y) {
    int rx = findDSU(x);
    int ry = findDSU(y);
    if (rx == ry) return false;
    // 按秩合并
    if (rankDSU[rx] < rankDSU[ry]) {
        parentDSU[rx] = ry;
    } else if (rankDSU[rx] > rankDSU[ry]) {
        parentDSU[ry] = rx;
    } else {
        parentDSU[ry] = rx;
        rankDSU[rx]++;
    }
    return true;
}
```

### 3.3 边排序和 Kruskal

```c
typedef struct {
    int u, v, weight;
} Edge;

int compareEdge(const void *a, const void *b) {
    return ((Edge *)a)->weight - ((Edge *)b)->weight;
}

void kruskal(Edge edges[], int edgeCount, int n) {
    // 按权重排序
    qsort(edges, edgeCount, sizeof(Edge), compareEdge);

    initDSU(n);

    printf("Kruskal 最小生成树:\n");
    int totalWeight = 0;
    int edgeSelected = 0;

    for (int i = 0; i < edgeCount && edgeSelected < n - 1; i++) {
        if (unionDSU(edges[i].u, edges[i].v)) {
            printf("  边: %d - %d, 权重: %d\n",
                   edges[i].u, edges[i].v, edges[i].weight);
            totalWeight += edges[i].weight;
            edgeSelected++;
        }
    }

    if (edgeSelected < n - 1) {
        printf("图不连通，不存在最小生成树!\n");
    } else {
        printf("总权重: %d\n", totalWeight);
    }
}
```

### 3.4 时间复杂度
- O(E log E)，主要在于排序

## 4. Prim vs Kruskal

| 特性 | Prim | Kruskal |
|------|------|---------|
| 策略 | 从顶点扩展 | 从边选择 |
| 数据结构 | 优先队列/数组 | 并查集 |
| 时间复杂度 | O(V^2) 或 O(E log V) | O(E log E) |
| 适用场景 | 稠密图 | 稀疏图 |
| 需要图连通 | 是 | 否（可判断） |

## 5. 完整测试

```c
int main() {
    vertexCount = 5;

    // 初始化邻接矩阵
    for (int i = 0; i < vertexCount; i++)
        for (int j = 0; j < vertexCount; j++)
            graph[i][j] = (i == j) ? 0 : INF;

    // 添加无向边
    graph[0][1] = graph[1][0] = 2;
    graph[0][3] = graph[3][0] = 6;
    graph[1][2] = graph[2][1] = 3;
    graph[1][3] = graph[3][1] = 8;
    graph[1][4] = graph[4][1] = 5;
    graph[2][4] = graph[4][2] = 7;
    graph[3][4] = graph[4][3] = 9;

    printf("=== Prim 算法 ===\n");
    prim();

    printf("\n=== Kruskal 算法 ===\n");
    Edge edges[] = {
        {0, 1, 2}, {0, 3, 6}, {1, 2, 3},
        {1, 3, 8}, {1, 4, 5}, {2, 4, 7}, {3, 4, 9}
    };
    kruskal(edges, 7, vertexCount);

    return 0;
}
```

## 6. 重点总结

- 最小生成树是**边权和最小的生成树**
- Prim 算法：每次添加离 MST 最近的顶点，适合**稠密图**
- Kruskal 算法：按边权排序选择，用**并查集**判环，适合**稀疏图**
- 两种算法都是**贪心**策略
- MST 的总权重是唯一的，但生成树可能不唯一（等权边时）
- 如果图不连通，需要对每个连通分量分别求 MST

> **注意**：最小生成树只适用于**无向连通加权图**。有向图的类似问题是"最小树形图"（Edmonds 算法）。
