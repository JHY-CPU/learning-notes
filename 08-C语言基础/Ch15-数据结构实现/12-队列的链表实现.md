# 队列的链表实现（链式队列）

## 1. 概述

链式队列（Linked Queue）使用链表来实现队列结构。维护两个指针：**队头指针（front）**用于出队，**队尾指针（rear）**用于入队。

### 顺序队列 vs 链式队列

| 特性 | 顺序队列 | 链式队列 |
|------|---------|---------|
| 存储方式 | 数组 | 链表 |
| 容量 | 固定 | 无限制 |
| 空间利用 | 可能浪费 | 按需分配 |
| 额外开销 | 无 | 每个元素一个指针 |

## 2. 数据结构定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

// 队列节点
typedef struct QNode {
    int data;
    struct QNode *next;
} QNode;

// 链式队列
typedef struct {
    QNode *front;  // 队头指针
    QNode *rear;   // 队尾指针
    int size;      // 队列大小
} LinkedQueue;
```

## 3. 基本操作实现

### 3.1 创建与初始化

```c
// 创建节点
QNode *createQNode(int value) {
    QNode *node = (QNode *)malloc(sizeof(QNode));
    if (!node) return NULL;
    node->data = value;
    node->next = NULL;
    return node;
}

// 初始化队列
LinkedQueue *createQueue() {
    LinkedQueue *q = (LinkedQueue *)malloc(sizeof(LinkedQueue));
    if (!q) return NULL;
    q->front = NULL;
    q->rear = NULL;
    q->size = 0;
    return q;
}
```

### 3.2 入队操作

```c
// 入队（在队尾插入）- O(1)
bool enqueue(LinkedQueue *q, int value) {
    QNode *newNode = createQNode(value);
    if (!newNode) return false;

    if (q->rear == NULL) {
        // 队列为空
        q->front = newNode;
        q->rear = newNode;
    } else {
        q->rear->next = newNode;
        q->rear = newNode;
    }
    q->size++;
    return true;
}
```

```
空队列入队A: front -> [A] <- rear
入队B:       front -> [A] -> [B] <- rear
入队C:       front -> [A] -> [B] -> [C] <- rear
```

### 3.3 出队操作

```c
// 出队（从队头删除）- O(1)
bool dequeue(LinkedQueue *q, int *value) {
    if (q->front == NULL) {
        printf("队空!\n");
        return false;
    }
    QNode *temp = q->front;
    *value = temp->data;
    q->front = temp->next;

    // 如果出队后队列为空，rear 也要置 NULL
    if (q->front == NULL) {
        q->rear = NULL;
    }

    free(temp);
    q->size--;
    return true;
}
```

```
出队: front -> [B] -> [C] <- rear, 返回 A
出队: front -> [C] <- rear,        返回 B
出队: front = NULL, rear = NULL,    返回 C
```

### 3.4 查看队头

```c
// 查看队头 - O(1)
bool peek(LinkedQueue *q, int *value) {
    if (q->front == NULL) return false;
    *value = q->front->data;
    return true;
}
```

### 3.5 辅助操作

```c
// 判断队空
bool isEmpty(LinkedQueue *q) {
    return q->front == NULL;
}

// 获取大小
int getSize(LinkedQueue *q) {
    return q->size;
}

// 打印队列
void printQueue(LinkedQueue *q) {
    printf("队头 -> ");
    QNode *p = q->front;
    while (p != NULL) {
        printf("[%d]", p->data);
        if (p->next) printf(" -> ");
        p = p->next;
    }
    printf(" <- 队尾 (共%d个元素)\n", q->size);
}

// 销毁队列
void destroyQueue(LinkedQueue *q) {
    QNode *p = q->front;
    while (p != NULL) {
        QNode *temp = p;
        p = p->next;
        free(temp);
    }
    free(q);
}
```

## 4. 完整测试代码

```c
int main() {
    LinkedQueue *queue = createQueue();

    // 入队
    enqueue(queue, 10);
    enqueue(queue, 20);
    enqueue(queue, 30);
    enqueue(queue, 40);
    printQueue(queue);  // 队头 -> [10] -> [20] -> [30] -> [40] <- 队尾

    // 查看队头
    int front;
    peek(queue, &front);
    printf("队头元素: %d\n", front);  // 10
    printf("队列大小: %d\n", getSize(queue));  // 4

    // 出队
    int val;
    dequeue(queue, &val);
    printf("出队: %d\n", val);  // 10
    dequeue(queue, &val);
    printf("出队: %d\n", val);  // 20
    printQueue(queue);  // 队头 -> [30] -> [40] <- 队尾

    // 继续入队
    enqueue(queue, 50);
    printQueue(queue);  // 队头 -> [30] -> [40] -> [50] <- 队尾

    destroyQueue(queue);
    return 0;
}
```

## 6. 用单链表头部和尾部作为队列

如果使用带头节点的链表，入队和出队操作会更加统一：

```c
// 带头节点的链式队列
typedef struct {
    QNode *head;  // 头节点
    QNode *rear;  // 尾指针
    int size;
} HeaderQueue;

void initHeaderQueue(HeaderQueue *q) {
    q->head = createQNode(0);  // 哨兵节点
    q->rear = q->head;
    q->size = 0;
}

void enqueueHeader(HeaderQueue *q, int value) {
    QNode *newNode = createQNode(value);
    q->rear->next = newNode;
    q->rear = newNode;
    q->size++;
}

int dequeueHeader(HeaderQueue *q) {
    if (q->head->next == NULL) return -1;
    QNode *temp = q->head->next;
    int value = temp->data;
    q->head->next = temp->next;
    if (q->rear == temp) q->rear = q->head;  // 队列变空
    free(temp);
    q->size--;
    return value;
}
```

## 7. 操作复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| enqueue | O(1) | 在 rear 插入 |
| dequeue | O(1) | 从 front 删除 |
| peek | O(1) | 访问 front |
| isEmpty | O(1) | 判断 front |

## 8. 重点总结

- 链式队列没有容量限制，适合大小不确定的场景
- 维护 front 和 rear 两个指针，保证入队和出队都是 O(1)
- 出队后如果队列变空，要将 rear 也置为 NULL
- 不要忘记释放出队节点的内存
- 队列的典型应用：BFS、任务调度、生产者-消费者模型

> **注意**：入队在链表尾部，出队在链表头部。如果在尾部也出队（需要找前驱），时间复杂度会退化为 O(n)。
