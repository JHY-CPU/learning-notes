# 双端队列

## 1. 概述

双端队列（Deque, Double-ended Queue）是一种允许在**两端**进行插入和删除操作的线性数据结构。它是栈和队列的推广。

### 操作对比

| 操作 | 栈 | 队列 | 双端队列 |
|------|-----|------|---------|
| 一端插入 | push | enqueue | 支持 |
| 一端删除 | pop | dequeue | 支持 |
| 另一端插入 | 不支持 | 不支持 | 支持 |
| 另一端删除 | 不支持 | 不支持 | 支持 |

## 2. 数组实现（循环双端队列）

### 2.1 数据结构定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

#define MAX_SIZE 8

typedef struct {
    int data[MAX_SIZE];
    int front;  // 队头下标
    int rear;   // 队尾下标（指向队尾元素的下一个位置）
    int size;   // 当前元素个数
} Deque;
```

### 2.2 初始化

```c
void initDeque(Deque *dq) {
    dq->front = 0;
    dq->rear = 0;
    dq->size = 0;
}

bool isFull(Deque *dq) {
    return dq->size == MAX_SIZE;
}

bool isEmpty(Deque *dq) {
    return dq->size == 0;
}
```

### 2.3 前端操作

```c
// 在前端插入
bool pushFront(Deque *dq, int value) {
    if (isFull(dq)) {
        printf("双端队列满!\n");
        return false;
    }
    dq->front = (dq->front - 1 + MAX_SIZE) % MAX_SIZE;
    dq->data[dq->front] = value;
    dq->size++;
    return true;
}

// 从前端删除
bool popFront(Deque *dq, int *value) {
    if (isEmpty(dq)) {
        printf("双端队列空!\n");
        return false;
    }
    *value = dq->data[dq->front];
    dq->front = (dq->front + 1) % MAX_SIZE;
    dq->size--;
    return true;
}

// 查看前端
bool peekFront(Deque *dq, int *value) {
    if (isEmpty(dq)) return false;
    *value = dq->data[dq->front];
    return true;
}
```

### 2.4 后端操作

```c
// 在后端插入
bool pushBack(Deque *dq, int value) {
    if (isFull(dq)) {
        printf("双端队列满!\n");
        return false;
    }
    dq->data[dq->rear] = value;
    dq->rear = (dq->rear + 1) % MAX_SIZE;
    dq->size++;
    return true;
}

// 从后端删除
bool popBack(Deque *dq, int *value) {
    if (isEmpty(dq)) {
        printf("双端队列空!\n");
        return false;
    }
    dq->rear = (dq->rear - 1 + MAX_SIZE) % MAX_SIZE;
    *value = dq->data[dq->rear];
    dq->size--;
    return true;
}

// 查看后端
bool peekBack(Deque *dq, int *value) {
    if (isEmpty(dq)) return false;
    *value = dq->data[(dq->rear - 1 + MAX_SIZE) % MAX_SIZE];
    return true;
}
```

### 2.5 辅助操作

```c
int getSize(Deque *dq) {
    return dq->size;
}

void printDeque(Deque *dq) {
    printf("双端队列 [大小=%d]: 前端 -> ", dq->size);
    int idx = dq->front;
    for (int i = 0; i < dq->size; i++) {
        printf("%d ", dq->data[idx]);
        idx = (idx + 1) % MAX_SIZE;
    }
    printf("<- 后端\n");
}
```

## 3. 链表实现

```c
typedef struct DQNode {
    int data;
    struct DQNode *prev;
    struct DQNode *next;
} DQNode;

typedef struct {
    DQNode *front;
    DQNode *rear;
    int size;
} LinkedDeque;

LinkedDeque *createDeque() {
    LinkedDeque *dq = (LinkedDeque *)malloc(sizeof(LinkedDeque));
    dq->front = dq->rear = NULL;
    dq->size = 0;
    return dq;
}

// 前端插入
void dequePushFront(LinkedDeque *dq, int value) {
    DQNode *node = (DQNode *)malloc(sizeof(DQNode));
    node->data = value;
    node->prev = NULL;
    node->next = dq->front;
    if (dq->front) dq->front->prev = node;
    else dq->rear = node;
    dq->front = node;
    dq->size++;
}

// 后端插入
void dequePushBack(LinkedDeque *dq, int value) {
    DQNode *node = (DQNode *)malloc(sizeof(DQNode));
    node->data = value;
    node->next = NULL;
    node->prev = dq->rear;
    if (dq->rear) dq->rear->next = node;
    else dq->front = node;
    dq->rear = node;
    dq->size++;
}

// 前端删除
int dequePopFront(LinkedDeque *dq) {
    if (!dq->front) return -1;
    DQNode *temp = dq->front;
    int value = temp->data;
    dq->front = temp->next;
    if (dq->front) dq->front->prev = NULL;
    else dq->rear = NULL;
    free(temp);
    dq->size--;
    return value;
}

// 后端删除
int dequePopBack(LinkedDeque *dq) {
    if (!dq->rear) return -1;
    DQNode *temp = dq->rear;
    int value = temp->data;
    dq->rear = temp->prev;
    if (dq->rear) dq->rear->next = NULL;
    else dq->front = NULL;
    free(temp);
    dq->size--;
    return value;
}
```

## 4. 完整测试代码

```c
int main() {
    Deque dq;
    initDeque(&dq);

    // 后端插入（相当于普通队列入队）
    pushBack(&dq, 10);
    pushBack(&dq, 20);
    pushBack(&dq, 30);
    printDeque(&dq);  // 前端 -> 10 20 30 <- 后端

    // 前端插入
    pushFront(&dq, 5);
    pushFront(&dq, 1);
    printDeque(&dq);  // 前端 -> 1 5 10 20 30 <- 后端

    // 前端删除（相当于队列出队）
    int val;
    popFront(&dq, &val);
    printf("前端弹出: %d\n", val);  // 1

    // 后端删除（相当于栈弹出）
    popBack(&dq, &val);
    printf("后端弹出: %d\n", val);  // 30

    printDeque(&dq);  // 前端 -> 5 10 20 <- 后端

    return 0;
}
```

## 5. 双端队列的应用

### 5.1 用双端队列同时实现栈和队列

```c
// 当栈使用
void stackPush(Deque *dq, int value) { pushBack(dq, value); }
void stackPop(Deque *dq, int *val) { popBack(dq, val); }

// 当队列使用
void queueEnq(Deque *dq, int value) { pushBack(dq, value); }
void queueDeq(Deque *dq, int *val) { popFront(dq, val); }
```

### 5.2 回文检测

```c
bool isPalindrome(const char *str) {
    LinkedDeque *dq = createDeque();
    int len = 0;
    while (str[len]) {
        dequePushBack(dq, str[len]);
        len++;
    }
    while (dq->size > 1) {
        int front = dequePopFront(dq);
        int back = dequePopBack(dq);
        if (front != back) {
            // 清理并返回
            while (dq->size) dequePopFront(dq);
            free(dq);
            return false;
        }
    }
    free(dq);
    return true;
}
```

## 6. 操作复杂度

| 操作 | 数组实现 | 链表实现 |
|------|---------|---------|
| pushFront | O(1) | O(1) |
| pushBack | O(1) | O(1) |
| popFront | O(1) | O(1) |
| popBack | O(1) | O(1) |
| peek | O(1) | O(1) |

所有操作都是 O(1)。

## 7. 重点总结

- 双端队列两端都可以插入和删除
- 用循环数组实现时，使用 size 变量来判断空满
- 链表实现需要双向链表
- 双端队列可以模拟栈和队列的所有操作
- C++ STL 中的 `deque` 是一个支持动态扩容的双端队列
- 滑动窗口最大值问题需要用到双端队列

> **注意**：在算法面试中，"单调队列"问题通常需要使用双端队列来实现。
