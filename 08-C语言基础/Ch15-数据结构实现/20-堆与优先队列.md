# 堆与优先队列

## 1. 概述

堆（Heap）是一种特殊的**完全二叉树**，满足堆性质。堆是实现**优先队列**的经典数据结构。

### 两种堆
- **最大堆（Max Heap）**：每个节点的值 >= 其子节点的值。根节点是最大值。
- **最小堆（Min Heap）**：每个节点的值 <= 其子节点的值。根节点是最小值。

```
最大堆示例:          最小堆示例:
      90                   10
     /  \                 /  \
    80   70              20   30
   / \   /              / \   /
  50 60 40             40 50 60
```

## 2. 数组存储

由于堆是完全二叉树，可以用数组高效存储。

```
下标关系（从0开始）:
- 父节点:  (i - 1) / 2
- 左孩子:  2 * i + 1
- 右孩子:  2 * i + 2
```

```c
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

#define MAX_SIZE 100

typedef struct {
    int data[MAX_SIZE];
    int size;
} Heap;

void initHeap(Heap *h) {
    h->size = 0;
}
```

## 3. 最大堆操作

### 3.1 上浮（Sift Up / Bubble Up）

插入新元素后，如果比父节点大，就交换上去。

```c
// 上浮操作
void siftUp(Heap *h, int index) {
    while (index > 0) {
        int parent = (index - 1) / 2;
        if (h->data[index] <= h->data[parent]) break;
        // 交换
        int temp = h->data[index];
        h->data[index] = h->data[parent];
        h->data[parent] = temp;
        index = parent;
    }
}
```

```
插入 95 到最大堆:
    90            90            95
   /  \          /  \          /  \
  80   70  ->   80   95  ->   80   90
 / \   /  \    / \   /  \    / \   /  \
50 60 40  95  50 60 40  70  50 60 40  70
   (初始)      (95上浮)      (95再上浮)
```

### 3.2 下沉（Sift Down / Heapify Down）

删除堆顶后，用最后一个元素替换，然后下沉到正确位置。

```c
// 下沉操作
void siftDown(Heap *h, int index) {
    while (true) {
        int largest = index;
        int left = 2 * index + 1;
        int right = 2 * index + 2;

        if (left < h->size && h->data[left] > h->data[largest])
            largest = left;
        if (right < h->size && h->data[right] > h->data[largest])
            largest = right;

        if (largest == index) break;

        // 交换
        int temp = h->data[index];
        h->data[index] = h->data[largest];
        h->data[largest] = temp;
        index = largest;
    }
}
```

### 3.3 插入操作

```c
void heapInsert(Heap *h, int value) {
    if (h->size >= MAX_SIZE) {
        printf("堆满!\n");
        return;
    }
    h->data[h->size] = value;
    h->size++;
    siftUp(h, h->size - 1);
}
```

### 3.4 删除堆顶操作

```c
int heapExtract(Heap *h) {
    if (h->size <= 0) {
        printf("堆空!\n");
        return -1;
    }
    int max = h->data[0];
    h->data[0] = h->data[h->size - 1];
    h->size--;
    siftDown(h, 0);
    return max;
}
```

### 3.5 查看堆顶

```c
int heapPeek(Heap *h) {
    return h->data[0];
}
```

### 3.6 从数组建堆

```c
// 自底向上建堆 - O(n)
void buildHeap(Heap *h, int arr[], int n) {
    for (int i = 0; i < n; i++) {
        h->data[i] = arr[i];
    }
    h->size = n;
    // 从最后一个非叶子节点开始下沉
    for (int i = (n - 2) / 2; i >= 0; i--) {
        siftDown(h, i);
    }
}
```

## 4. 堆排序

利用最大堆的性质进行排序。

```c
// 堆排序 - O(n log n), 原地排序
void heapSort(int arr[], int n) {
    // 1. 建立最大堆
    for (int i = (n - 2) / 2; i >= 0; i--) {
        // siftDown 逻辑
        int index = i;
        while (true) {
            int largest = index;
            int left = 2 * index + 1;
            int right = 2 * index + 2;
            if (left < n && arr[left] > arr[largest]) largest = left;
            if (right < n && arr[right] > arr[largest]) largest = right;
            if (largest == index) break;
            int temp = arr[index];
            arr[index] = arr[largest];
            arr[largest] = temp;
            index = largest;
        }
    }

    // 2. 逐步取出最大值
    for (int i = n - 1; i > 0; i--) {
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        // 对剩余元素下沉
        int index = 0;
        while (true) {
            int largest = index;
            int left = 2 * index + 1;
            int right = 2 * index + 2;
            if (left < i && arr[left] > arr[largest]) largest = left;
            if (right < i && arr[right] > arr[largest]) largest = right;
            if (largest == index) break;
            temp = arr[index];
            arr[index] = arr[largest];
            arr[largest] = temp;
            index = largest;
        }
    }
}
```

## 5. 优先队列

优先队列中，出队顺序由优先级决定（而非入队顺序）。

```c
// 基于最大堆的优先队列
typedef struct {
    Heap heap;
} PriorityQueue;

void pqInit(PriorityQueue *pq) { initHeap(&pq->heap); }
void pqPush(PriorityQueue *pq, int value) { heapInsert(&pq->heap, value); }
int pqPop(PriorityQueue *pq) { return heapExtract(&pq->heap); }
int pqTop(PriorityQueue *pq) { return heapPeek(&pq->heap); }
bool pqEmpty(PriorityQueue *pq) { return pq->heap.size == 0; }
```

## 6. 应用：Top K 问题

找出 n 个数中最大的 K 个。

```c
// 用最小堆找最大的 K 个数
void siftUpMin(int heap[], int index) {
    while (index > 0) {
        int parent = (index - 1) / 2;
        if (heap[index] >= heap[parent]) break;
        int temp = heap[index];
        heap[index] = heap[parent];
        heap[parent] = temp;
        index = parent;
    }
}

void siftDownMin(int heap[], int size, int index) {
    while (true) {
        int smallest = index;
        int left = 2 * index + 1;
        int right = 2 * index + 2;
        if (left < size && heap[left] < heap[smallest]) smallest = left;
        if (right < size && heap[right] < heap[smallest]) smallest = right;
        if (smallest == index) break;
        int temp = heap[index];
        heap[index] = heap[smallest];
        heap[smallest] = temp;
        index = smallest;
    }
}

void topK(int arr[], int n, int k) {
    // 用前 k 个元素建立最小堆
    int heap[MAX_SIZE];
    for (int i = 0; i < k; i++) {
        heap[i] = arr[i];
        siftUpMin(heap, i);
    }

    // 遍历剩余元素
    for (int i = k; i < n; i++) {
        if (arr[i] > heap[0]) {
            heap[0] = arr[i];
            siftDownMin(heap, k, 0);
        }
    }

    printf("最大的%d个数: ", k);
    for (int i = 0; i < k; i++) printf("%d ", heap[i]);
    printf("\n");
}
```

## 7. 完整测试

```c
int main() {
    Heap h;
    initHeap(&h);

    // 插入
    int data[] = {10, 30, 20, 50, 40, 60, 70};
    for (int i = 0; i < 7; i++) {
        heapInsert(&h, data[i]);
    }

    printf("堆: ");
    for (int i = 0; i < h.size; i++) printf("%d ", h.data[i]);
    printf("\n");

    // 取出最大值
    printf("最大值: %d\n", heapExtract(&h));
    printf("最大值: %d\n", heapExtract(&h));

    // 堆排序
    int arr[] = {38, 27, 43, 3, 9, 82, 10};
    int n = 7;
    heapSort(arr, n);
    printf("堆排序: ");
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");

    return 0;
}
```

## 8. 复杂度分析

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 插入 | O(log n) | 上浮操作 |
| 删除堆顶 | O(log n) | 下沉操作 |
| 查看堆顶 | O(1) | 直接访问 |
| 建堆 | O(n) | 自底向上更高效 |
| 堆排序 | O(n log n) | 不稳定排序 |

## 9. 重点总结

- 堆是完全二叉树，用数组存储
- 最大堆：堆顶是最大值；最小堆：堆顶是最小值
- 插入用上浮（siftUp），删除用下沉（siftDown）
- 建堆时间复杂度是 O(n)，不是 O(n log n)
- 堆排序是原地排序，但不稳定
- Top K 问题用堆效率很高（最小堆求最大的K个，最大堆求最小的K个）
- 优先队列是堆的最主要应用场景

> **注意**：C 语言标准库没有堆的实现，需要手写。C++ 有 `priority_queue`，Java 有 `PriorityQueue`。
