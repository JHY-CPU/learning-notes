# 单链表实现

## 1. 概述

单链表（Singly Linked List）是线性表的链式存储结构，每个节点包含**数据域**和**指针域**。指针域存储指向下一个节点的地址，通过指针将各个节点串联起来。

### 与顺序表的对比

| 特性 | 顺序表 | 单链表 |
|------|--------|--------|
| 存储方式 | 连续内存 | 分散内存 |
| 访问方式 | 随机访问 O(1) | 顺序访问 O(n) |
| 插入/删除 | O(n) 移动元素 | O(1) 修改指针 |
| 空间利用 | 可能浪费 | 按需分配 |

## 2. 节点定义

```c
#include <stdio.h>
#include <stdlib.h>

// 单链表节点定义
typedef struct Node {
    int data;           // 数据域
    struct Node *next;  // 指针域，指向下一个节点
} Node;

// 链表结构（带头节点版本）
typedef struct {
    Node *head;         // 头指针
    int length;         // 链表长度
} LinkedList;
```

### 带头节点 vs 不带头节点

```
带头节点（推荐，简化操作）:
head -> [dummy] -> [10] -> [20] -> [30] -> NULL

不带头节点:
head -> [10] -> [20] -> [30] -> NULL
```

## 3. 基本操作实现

### 3.1 创建与初始化

```c
// 创建新节点
Node *createNode(int value) {
    Node *newNode = (Node *)malloc(sizeof(Node));
    if (!newNode) {
        printf("内存分配失败!\n");
        return NULL;
    }
    newNode->data = value;
    newNode->next = NULL;
    return newNode;
}

// 初始化链表（带头节点）
LinkedList *createList() {
    LinkedList *list = (LinkedList *)malloc(sizeof(LinkedList));
    if (!list) return NULL;
    list->head = createNode(0);  // 头节点数据域不使用
    if (!list->head) {
        free(list);
        return NULL;
    }
    list->length = 0;
    return list;
}
```

### 3.2 头插法

在链表头部插入新节点，时间复杂度 O(1)。

```c
// 头插法：新节点插入到第一个数据节点之前
int insertHead(LinkedList *list, int value) {
    Node *newNode = createNode(value);
    if (!newNode) return 0;
    newNode->next = list->head->next;
    list->head->next = newNode;
    list->length++;
    return 1;
}
```

```
插入前: head -> [A] -> [B] -> NULL
头插X:  head -> [X] -> [A] -> [B] -> NULL
```

### 3.3 尾插法

在链表尾部插入新节点，时间复杂度 O(n)（需要遍历找尾部）。

```c
// 尾插法
int insertTail(LinkedList *list, int value) {
    Node *newNode = createNode(value);
    if (!newNode) return 0;
    Node *p = list->head;
    while (p->next != NULL) {
        p = p->next;
    }
    p->next = newNode;
    list->length++;
    return 1;
}

// 尾插法优化版本（维护尾指针）
int insertTailOptimized(LinkedList *list, Node **tail, int value) {
    Node *newNode = createNode(value);
    if (!newNode) return 0;
    (*tail)->next = newNode;
    *tail = newNode;
    list->length++;
    return 1;
}
```

### 3.4 按位置插入

```c
// 在第 pos 个位置插入（pos 从 0 开始）
int insertAt(LinkedList *list, int pos, int value) {
    if (pos < 0 || pos > list->length) return 0;
    Node *p = list->head;
    for (int i = 0; i < pos; i++) {
        p = p->next;
    }
    Node *newNode = createNode(value);
    if (!newNode) return 0;
    newNode->next = p->next;
    p->next = newNode;
    list->length++;
    return 1;
}
```

### 3.5 删除操作

```c
// 删除第 pos 个节点
int deleteAt(LinkedList *list, int pos, int *deleted) {
    if (pos < 0 || pos >= list->length) return 0;
    Node *p = list->head;
    for (int i = 0; i < pos; i++) {
        p = p->next;
    }
    Node *temp = p->next;
    *deleted = temp->data;
    p->next = temp->next;
    free(temp);
    list->length--;
    return 1;
}

// 按值删除第一个匹配的节点
int deleteByValue(LinkedList *list, int value) {
    Node *p = list->head;
    while (p->next != NULL) {
        if (p->next->data == value) {
            Node *temp = p->next;
            p->next = temp->next;
            free(temp);
            list->length--;
            return 1;
        }
        p = p->next;
    }
    return 0;  // 未找到
}
```

### 3.6 查找操作

```c
// 按值查找，返回节点指针，未找到返回 NULL
Node *findByValue(LinkedList *list, int value) {
    Node *p = list->head->next;
    while (p != NULL) {
        if (p->data == value) return p;
        p = p->next;
    }
    return NULL;
}

// 按位置查找，返回节点指针
Node *findByIndex(LinkedList *list, int pos) {
    if (pos < 0 || pos >= list->length) return NULL;
    Node *p = list->head->next;
    for (int i = 0; i < pos; i++) {
        p = p->next;
    }
    return p;
}
```

### 3.7 遍历打印

```c
void printList(LinkedList *list) {
    printf("链表 [长度=%d]: ", list->length);
    Node *p = list->head->next;
    while (p != NULL) {
        printf("%d", p->data);
        if (p->next != NULL) printf(" -> ");
        p = p->next;
    }
    printf(" -> NULL\n");
}
```

### 3.8 销毁链表

```c
void destroyList(LinkedList *list) {
    Node *p = list->head;
    while (p != NULL) {
        Node *temp = p;
        p = p->next;
        free(temp);
    }
    free(list);
}
```

## 4. 完整测试代码

```c
int main() {
    LinkedList *list = createList();

    // 头插法
    insertHead(list, 30);
    insertHead(list, 20);
    insertHead(list, 10);
    printList(list);  // 10 -> 20 -> 30 -> NULL

    // 尾插法
    insertTail(list, 40);
    insertTail(list, 50);
    printList(list);  // 10 -> 20 -> 30 -> 40 -> 50 -> NULL

    // 按位置插入
    insertAt(list, 2, 25);
    printList(list);  // 10 -> 20 -> 25 -> 30 -> 40 -> 50 -> NULL

    // 删除
    int deleted;
    deleteAt(list, 3, &deleted);
    printf("删除了: %d\n", deleted);
    printList(list);  // 10 -> 20 -> 25 -> 40 -> 50 -> NULL

    deleteByValue(list, 40);
    printList(list);  // 10 -> 20 -> 25 -> 50 -> NULL

    // 查找
    Node *node = findByValue(list, 25);
    if (node) printf("找到了: %d\n", node->data);

    destroyList(list);
    return 0;
}
```

## 5. 操作复杂度分析

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 头部插入 | O(1) | 直接修改头节点的 next |
| 尾部插入 | O(n) | 需要遍历找到尾部（有尾指针时O(1)） |
| 按位置插入 | O(n) | 需要遍历到指定位置 |
| 头部删除 | O(1) | 直接删除第一个节点 |
| 按位置删除 | O(n) | 需要找到前驱节点 |
| 按值查找 | O(n) | 顺序遍历 |

## 6. 重点总结

- 单链表适合**频繁插入删除**的场景
- 头插法插入顺序与输入顺序相反，尾插法与输入顺序相同
- 使用头节点可以简化边界情况的处理
- 删除操作要注意释放内存，防止内存泄漏
- 单链表只能从前往后遍历，无法回退

> **注意**：实际面试和开发中，不带头节点的单链表也经常出现，需要特别注意处理头节点变化的情况。
