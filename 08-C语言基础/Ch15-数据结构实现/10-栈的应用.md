# 栈的应用

## 1. 概述

栈在计算机科学中有极其广泛的应用。本节介绍四个经典应用：括号匹配、表达式求值、进制转换和深度优先搜索（DFS）。

## 2. 应用一：括号匹配

判断一个字符串中的括号是否正确配对。包括 `()`, `[]`, `{}`。

### 算法思路
- 遇到左括号：压入栈
- 遇到右括号：弹出栈顶，检查是否匹配
- 遍历结束后，栈为空则匹配成功

```c
#include <stdio.h>
#include <string.h>
#include <stdbool.h>

#define MAX_SIZE 1000

bool isMatching(char left, char right) {
    return (left == '(' && right == ')') ||
           (left == '[' && right == ']') ||
           (left == '{' && right == '}');
}

bool isValidParentheses(const char *str) {
    char stack[MAX_SIZE];
    int top = -1;

    for (int i = 0; str[i] != '\0'; i++) {
        char c = str[i];
        if (c == '(' || c == '[' || c == '{') {
            stack[++top] = c;  // 左括号入栈
        } else if (c == ')' || c == ']' || c == '}') {
            if (top < 0) return false;  // 栈空，没有匹配的左括号
            if (!isMatching(stack[top--], c)) return false;  // 不匹配
        }
    }
    return top == -1;  // 栈空说明全部匹配
}

// 测试
// "({[]})" -> true
// "([)]"   -> false
// "{[}"    -> false
// ""       -> true
```

## 3. 应用二：表达式求值

### 3.1 中缀表达式转后缀表达式（逆波兰表达式）

中缀表达式：`3 + 4 * 2` （运算符在中间）
后缀表达式：`3 4 2 * +` （运算符在后面）

#### 转换规则
- 操作数：直接输出
- 运算符：与栈顶比较优先级
  - 栈空或栈顶是 `(` 或当前优先级更高：入栈
  - 否则：弹出栈顶并输出，再将当前运算符入栈
- `(`：入栈
- `)`：弹出并输出直到遇到 `(`

```c
// 运算符优先级
int priority(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    return 0;
}

void infixToPostfix(const char *infix, char *postfix) {
    char stack[MAX_SIZE];
    int top = -1;
    int j = 0;

    for (int i = 0; infix[i] != '\0'; i++) {
        char c = infix[i];

        if (c == ' ') continue;

        // 操作数（多位数字）直接输出
        if (c >= '0' && c <= '9') {
            while (infix[i] >= '0' && infix[i] <= '9') {
                postfix[j++] = infix[i++];
            }
            postfix[j++] = ' ';
            i--;  // 回退一位
        }
        // 左括号入栈
        else if (c == '(') {
            stack[++top] = c;
        }
        // 右括号：弹出直到左括号
        else if (c == ')') {
            while (top >= 0 && stack[top] != '(') {
                postfix[j++] = stack[top--];
                postfix[j++] = ' ';
            }
            top--;  // 弹出左括号，不输出
        }
        // 运算符
        else {
            while (top >= 0 && stack[top] != '(' &&
                   priority(stack[top]) >= priority(c)) {
                postfix[j++] = stack[top--];
                postfix[j++] = ' ';
            }
            stack[++top] = c;
        }
    }
    // 弹出剩余运算符
    while (top >= 0) {
        postfix[j++] = stack[top--];
        postfix[j++] = ' ';
    }
    postfix[j] = '\0';
}
```

### 3.2 后缀表达式求值

```c
int evaluatePostfix(const char *postfix) {
    int stack[MAX_SIZE];
    int top = -1;

    for (int i = 0; postfix[i] != '\0'; i++) {
        char c = postfix[i];
        if (c == ' ') continue;

        // 操作数入栈
        if (c >= '0' && c <= '9') {
            int num = 0;
            while (postfix[i] >= '0' && postfix[i] <= '9') {
                num = num * 10 + (postfix[i++] - '0');
            }
            stack[++top] = num;
            i--;
        }
        // 运算符：弹出两个操作数
        else {
            int b = stack[top--];
            int a = stack[top--];
            switch (c) {
                case '+': stack[++top] = a + b; break;
                case '-': stack[++top] = a - b; break;
                case '*': stack[++top] = a * b; break;
                case '/': stack[++top] = a / b; break;
            }
        }
    }
    return stack[top];
}

// 测试
// "3 4 2 * +" -> 4*2=8, 3+8=11
```

## 4. 应用三：进制转换

将十进制数转换为任意进制（2-16进制）。

```c
void decimalToBase(int num, int base) {
    if (num == 0) {
        printf("0\n");
        return;
    }

    char stack[MAX_SIZE];
    int top = -1;
    char digits[] = "0123456789ABCDEF";

    while (num > 0) {
        stack[++top] = digits[num % base];
        num /= base;
    }

    printf("转换结果: ");
    while (top >= 0) {
        printf("%c", stack[top--]);
    }
    printf("\n");
}

// decimalToBase(255, 16) -> FF
// decimalToBase(10, 2)   -> 1010
// decimalToBase(100, 8)  -> 144
```

## 5. 应用四：使用栈实现 DFS

深度优先搜索可以用栈来实现（替代递归）。

```c
// 用栈实现图的 DFS（邻接矩阵）
void DFS(int graph[][7], int n, int start) {
    int visited[7] = {0};
    int stack[MAX_SIZE];
    int top = -1;

    stack[++top] = start;
    visited[start] = 1;

    printf("DFS遍历: ");
    while (top >= 0) {
        int v = stack[top--];  // 弹出栈顶
        printf("%d ", v);

        // 将未访问的邻居压入栈（逆序，保证左边的邻居先访问）
        for (int i = n - 1; i >= 0; i--) {
            if (graph[v][i] && !visited[i]) {
                stack[++top] = i;
                visited[i] = 1;
            }
        }
    }
    printf("\n");
}
```

## 6. 应用五：单调栈（求下一个更大元素）

单调栈是栈的进阶应用，常用于求解"下一个更大元素"等问题。

```c
// 找每个元素右边第一个比它大的元素
void nextGreaterElement(int arr[], int n, int result[]) {
    int stack[MAX_SIZE];
    int top = -1;

    // 初始化结果为 -1
    for (int i = 0; i < n; i++) result[i] = -1;

    // 从右往左遍历
    for (int i = n - 1; i >= 0; i--) {
        // 弹出栈中所有小于等于当前元素的值
        while (top >= 0 && arr[stack[top]] <= arr[i]) {
            top--;
        }
        // 栈顶就是右边第一个比当前元素大的
        if (top >= 0) {
            result[i] = arr[stack[top]];
        }
        // 当前下标入栈
        stack[++top] = i;
    }
}

// 输入: [2, 1, 2, 4, 3]
// 输出: [4, 2, 4, -1, -1]
```

## 7. 重点总结

| 应用 | 核心思想 | 关键点 |
|------|---------|--------|
| 括号匹配 | 左括号入栈，右括号匹配弹出 | 注意边界条件 |
| 表达式求值 | 运算符优先级决定入栈时机 | 中缀转后缀再求值 |
| 进制转换 | 余数入栈，逆序输出 | 处理0的特殊情况 |
| DFS | 节点入栈，弹出后处理邻居 | 注意访问标记 |
| 单调栈 | 维护栈内元素的单调性 | 适用于"下一个更大/更小"类问题 |

> **核心思想**：栈的本质是"后进先出"，任何需要"回退"或"撤销"的场景都可以考虑使用栈。
