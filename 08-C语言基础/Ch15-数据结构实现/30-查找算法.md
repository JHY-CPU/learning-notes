# 30 - 查找算法

## 概述

查找（搜索）是最基本的操作之一。本节介绍顺序查找、二分查找、哈希查找和二叉搜索树查找。

---

## 1. 顺序查找（Linear Search）

### 基本思想

从头到尾逐个比较，是最简单的查找方法。

### 时间复杂度

- 最好：O(1)
- 最坏/平均：O(n)

```c
#include <stdio.h>

int linear_search(int arr[], int n, int target) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == target) return i;
    }
    return -1;  // 未找到
}

// 带哨兵的顺序查找（减少边界判断）
int linear_search_sentinel(int arr[], int n, int target) {
    if (n == 0) return -1;

    int last = arr[n - 1];
    arr[n - 1] = target;  // 设置哨兵

    int i = 0;
    while (arr[i] != target) i++;

    arr[n - 1] = last;  // 恢复

    if (i < n - 1 || arr[n - 1] == target)
        return i;
    return -1;
}
```

## 2. 二分查找（Binary Search）

### 基本思想

在**有序数组**中，每次将查找范围缩小一半。

### 时间复杂度

- 最好：O(1)
- 最坏/平均：O(log n)

### 代码实现

```c
// 迭代版
int binary_search(int arr[], int n, int target) {
    int left = 0, right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;  // 避免溢出

        if (arr[mid] == target)
            return mid;
        else if (arr[mid] < target)
            left = mid + 1;
        else
            right = mid - 1;
    }

    return -1;
}

// 递归版
int binary_search_recursive(int arr[], int left, int right, int target) {
    if (left > right) return -1;

    int mid = left + (right - left) / 2;

    if (arr[mid] == target) return mid;
    if (arr[mid] < target)
        return binary_search_recursive(arr, mid + 1, right, target);
    return binary_search_recursive(arr, left, mid - 1, target);
}

// 查找第一个等于target的位置
int lower_bound(int arr[], int n, int target) {
    int left = 0, right = n;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] < target)
            left = mid + 1;
        else
            right = mid;
    }
    return left;
}

// 查找第一个大于target的位置
int upper_bound(int arr[], int n, int target) {
    int left = 0, right = n;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] <= target)
            left = mid + 1;
        else
            right = mid;
    }
    return left;
}
```

## 3. 哈希查找（Hash Search）

### 基本思想

通过哈希函数将关键字映射到表中的位置，实现O(1)的平均查找。

### 哈希冲突处理

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#define TABLE_SIZE 101

// 链地址法实现
typedef struct HashNode {
    int key;
    int value;
    struct HashNode *next;
} HashNode;

typedef struct {
    HashNode *buckets[TABLE_SIZE];
} HashTable;

// 哈希函数
int hash(int key) {
    return ((key % TABLE_SIZE) + TABLE_SIZE) % TABLE_SIZE;
}

// 初始化
void ht_init(HashTable *ht) {
    memset(ht->buckets, 0, sizeof(ht->buckets));
}

// 插入
void ht_insert(HashTable *ht, int key, int value) {
    int idx = hash(key);
    HashNode *node = ht->buckets[idx];

    // 检查是否已存在
    while (node) {
        if (node->key == key) {
            node->value = value;  // 更新
            return;
        }
        node = node->next;
    }

    // 新建节点（头插法）
    node = (HashNode *)malloc(sizeof(HashNode));
    node->key = key;
    node->value = value;
    node->next = ht->buckets[idx];
    ht->buckets[idx] = node;
}

// 查找
int ht_search(HashTable *ht, int key, int *value) {
    int idx = hash(key);
    HashNode *node = ht->buckets[idx];
    while (node) {
        if (node->key == key) {
            *value = node->value;
            return 1;
        }
        node = node->next;
    }
    return 0;
}

// 删除
void ht_delete(HashTable *ht, int key) {
    int idx = hash(key);
    HashNode *node = ht->buckets[idx];
    HashNode *prev = NULL;

    while (node) {
        if (node->key == key) {
            if (prev)
                prev->next = node->next;
            else
                ht->buckets[idx] = node->next;
            free(node);
            return;
        }
        prev = node;
        node = node->next;
    }
}

// 释放
void ht_destroy(HashTable *ht) {
    for (int i = 0; i < TABLE_SIZE; i++) {
        HashNode *node = ht->buckets[i];
        while (node) {
            HashNode *tmp = node;
            node = node->next;
            free(tmp);
        }
    }
}
```

## 4. 二叉搜索树查找（BST Search）

### 基本思想

利用BST的性质：左子树所有节点小于根，右子树所有节点大于根。

### 时间复杂度

- 平均：O(log n)
- 最坏：O(n)（退化为链表）

```c
typedef struct BSTNode {
    int key;
    struct BSTNode *left;
    struct BSTNode *right;
} BSTNode;

// 创建节点
BSTNode *bst_create_node(int key) {
    BSTNode *node = (BSTNode *)malloc(sizeof(BSTNode));
    node->key = key;
    node->left = node->right = NULL;
    return node;
}

// 查找
BSTNode *bst_search(BSTNode *root, int key) {
    if (root == NULL || root->key == key)
        return root;
    if (key < root->key)
        return bst_search(root->left, key);
    return bst_search(root->right, key);
}

// 迭代查找
BSTNode *bst_search_iterative(BSTNode *root, int key) {
    while (root != NULL && root->key != key) {
        root = (key < root->key) ? root->left : root->right;
    }
    return root;
}

// 插入
BSTNode *bst_insert(BSTNode *root, int key) {
    if (root == NULL) return bst_create_node(key);
    if (key < root->key)
        root->left = bst_insert(root->left, key);
    else if (key > root->key)
        root->right = bst_insert(root->right, key);
    return root;
}

// 查找最小值
BSTNode *bst_min(BSTNode *root) {
    while (root && root->left) root = root->left;
    return root;
}

// 删除
BSTNode *bst_delete(BSTNode *root, int key) {
    if (root == NULL) return NULL;

    if (key < root->key) {
        root->left = bst_delete(root->left, key);
    } else if (key > root->key) {
        root->right = bst_delete(root->right, key);
    } else {
        // 找到节点
        if (root->left == NULL) {
            BSTNode *temp = root->right;
            free(root);
            return temp;
        }
        if (root->right == NULL) {
            BSTNode *temp = root->left;
            free(root);
            return temp;
        }
        // 两个子节点：用中序后继替换
        BSTNode *successor = bst_min(root->right);
        root->key = successor->key;
        root->right = bst_delete(root->right, successor->key);
    }
    return root;
}
```

## 5. 查找算法对比

| 算法 | 平均时间 | 最坏时间 | 前提条件 | 适用场景 |
|------|----------|----------|----------|----------|
| 顺序查找 | O(n) | O(n) | 无 | 小数据/无序 |
| 二分查找 | O(log n) | O(log n) | 有序数组 | 静态有序数据 |
| 哈希查找 | O(1) | O(n) | 哈希函数 | 频繁查找 |
| BST查找 | O(log n) | O(n) | BST结构 | 动态数据 |

---

## 要点总结

1. 二分查找是最常用的高效查找，但要求数据有序
2. 哈希查找平均O(1)，是最高效的查找方式
3. BST适合动态数据集，平衡BST可保证O(log n)
4. 实际选择需考虑数据特性（静态/动态、有序/无序）
