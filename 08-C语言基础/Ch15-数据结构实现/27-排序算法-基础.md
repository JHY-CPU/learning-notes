# 排序算法 - 基础篇

## 1. 排序算法总览

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 |
|------|---------|---------|------|--------|
| 冒泡排序 | O(n^2) | O(n^2) | O(1) | 稳定 |
| 选择排序 | O(n^2) | O(n^2) | O(1) | 不稳定 |
| 插入排序 | O(n^2) | O(n^2) | O(1) | 稳定 |

> **稳定**：相等元素排序后相对位置不变。

## 2. 冒泡排序（Bubble Sort）

### 思想
重复遍历数组，每次比较相邻元素，如果顺序错误就交换。每轮将最大值"冒泡"到末尾。

```c
#include <stdio.h>

void swap(int *a, int *b) {
    int t = *a; *a = *b; *b = t;
}

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int swapped = 0;  // 优化：如果没有交换，说明已有序
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(&arr[j], &arr[j + 1]);
                swapped = 1;
            }
        }
        if (!swapped) break;  // 已有序，提前退出
    }
}
```

```
初始:  [5, 3, 8, 4, 2]
第1轮: [3, 5, 4, 2, 8]  8冒泡到末尾
第2轮: [3, 4, 2, 5, 8]  5冒泡
第3轮: [3, 2, 4, 5, 8]  4冒泡
第4轮: [2, 3, 4, 5, 8]  3冒泡，排序完成
```

### 复杂度分析
- 最好：O(n)（已有序，一轮就退出）
- 最坏：O(n^2)（逆序）
- 平均：O(n^2)

## 3. 选择排序（Selection Sort）

### 思思
每轮从未排序部分选出最小值，放到已排序部分的末尾。

```c
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIdx]) {
                minIdx = j;
            }
        }
        if (minIdx != i) {
            swap(&arr[i], &arr[minIdx]);
        }
    }
}
```

```
初始: [5, 3, 8, 4, 2]
第1轮: [2, 3, 8, 4, 5]  找到最小值2，与arr[0]交换
第2轮: [2, 3, 8, 4, 5]  找到最小值3，已在位置
第3轮: [2, 3, 4, 8, 5]  找到最小值4，与arr[2]交换
第4轮: [2, 3, 4, 5, 8]  找到最小值5，与arr[3]交换
```

### 特点
- 比较次数固定：n(n-1)/2
- 交换次数少：最多 n-1 次
- **不稳定**：例如 [5, 5, 2] 排序后两个5的相对位置可能改变

## 4. 插入排序（Insertion Sort）

### 思想
将数组分为已排序和未排序两部分，每次从未排序部分取一个元素，插入到已排序部分的正确位置。

```c
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        // 向后移动比 key 大的元素
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
```

```
初始: [5, 3, 8, 4, 2]
i=1:  [3, 5, 8, 4, 2]  3插入到5前面
i=2:  [3, 5, 8, 4, 2]  8已在正确位置
i=3:  [3, 4, 5, 8, 2]  4插入到3和5之间
i=4:  [2, 3, 4, 5, 8]  2插入到最前面
```

### 特点
- **最好情况 O(n)**：数组基本有序时效率极高
- **稳定**：相等元素不会交换位置
- 适合小规模数据和基本有序的数据

## 5. 希尔排序（Shell Sort）

### 思想
插入排序的改进版。先按较大的间隔进行插入排序，逐步缩小间隔直到1（此时就是普通插入排序）。

```c
void shellSort(int arr[], int n) {
    // 初始间隔为 n/2，每次减半
    for (int gap = n / 2; gap > 0; gap /= 2) {
        // 对每个子序列做插入排序
        for (int i = gap; i < n; i++) {
            int key = arr[i];
            int j = i;
            while (j >= gap && arr[j - gap] > key) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            arr[j] = key;
        }
    }
}
```

```
gap=4: [5, 3, 8, 4, 2, 7, 1, 6]
       比较(5,2), (3,7), (8,1), (4,6) -> [2, 3, 1, 4, 5, 7, 8, 6]
gap=2: 比较(2,1,5,8), (3,4,7,6) -> [1, 3, 2, 4, 5, 6, 8, 7]
gap=1: 插入排序 -> [1, 2, 3, 4, 5, 6, 7, 8]
```

### 复杂度
- 平均：O(n^1.3) ~ O(n^1.5)（取决于间隔序列）
- 空间：O(1)
- **不稳定**

## 6. 完整测试

```c
void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");
}

int main() {
    int arr1[] = {5, 3, 8, 4, 2, 7, 1, 6};
    int arr2[] = {5, 3, 8, 4, 2, 7, 1, 6};
    int arr3[] = {5, 3, 8, 4, 2, 7, 1, 6};
    int arr4[] = {5, 3, 8, 4, 2, 7, 1, 6};
    int n = 8;

    bubbleSort(arr1, n);
    printf("冒泡排序: "); printArray(arr1, n);

    selectionSort(arr2, n);
    printf("选择排序: "); printArray(arr2, n);

    insertionSort(arr3, n);
    printf("插入排序: "); printArray(arr3, n);

    shellSort(arr4, n);
    printf("希尔排序: "); printArray(arr4, n);

    return 0;
}
```

## 7. 重点总结

- **冒泡排序**：最直观，但实际性能最差。优化点：提前退出和记录最后交换位置。
- **选择排序**：交换次数最少，但比较次数固定。**不稳定**。
- **插入排序**：对基本有序的数据性能最好。**稳定**，适合小数据集。
- **希尔排序**：插入排序的改进，间隔序列的选择很关键。
- 这些基础排序算法的时间复杂度都是 O(n^2)（希尔除外），不适合大规模数据。

> **实践建议**：当数据量较小时（n < 50），插入排序通常比快排等高级算法更快，因为常数因子更小。
