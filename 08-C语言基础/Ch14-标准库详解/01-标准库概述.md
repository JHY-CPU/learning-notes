# C标准库概述

## 1. 标准库简介

C标准库（C Standard Library）是C语言规范中定义的一组函数、宏和类型的集合。它为C程序提供了基础的输入输出、内存管理、字符串处理、数学计算等功能。标准库是每个符合标准的C编译器必须实现的部分。

## 2. C标准的演进

### 2.1 C89/C90（ANSI C / ISO C90）

C语言的第一个正式标准，定义了15个标准头文件：

| 头文件 | 功能 |
|--------|------|
| `<stdio.h>` | 标准输入输出 |
| `<stdlib.h>` | 通用工具函数 |
| `<string.h>` | 字符串处理 |
| `<ctype.h>` | 字符分类与转换 |
| `<math.h>` | 数学函数 |
| `<time.h>` | 时间日期函数 |
| `<assert.h>` | 断言宏 |
| `<errno.h>` | 错误码 |
| `<limits.h>` | 整数类型限制 |
| `<float.h>` | 浮点类型限制 |
| `<stdarg.h>` | 可变参数 |
| `<setjmp.h>` | 非局部跳转 |
| `<locale.h>` | 本地化 |
| `<signal.h>` | 信号处理 |
| `<stddef.h>` | 常用定义 |

### 2.2 C99 新增头文件

C99引入了许多新特性，新增了6个头文件：

| 头文件 | 功能 |
|--------|------|
| `<stdbool.h>` | 布尔类型支持 |
| `<stdint.h>` | 固定宽度整数类型 |
| `<inttypes.h>` | 整数格式化宏 |
| `<complex.h>` | 复数运算 |
| `<tgmath.h>` | 泛型数学宏 |
| `<fenv.h>` | 浮点环境控制 |

### 2.3 C11 新增头文件

C11重点加强了多线程和安全性支持：

| 头文件 | 功能 |
|--------|------|
| `<threads.h>` | 多线程支持 |
| `<stdatomic.h>` | 原子操作 |
| `<stdalign.h>` | 内存对齐控制 |
| `<stdnoreturn.h>` | `_Noreturn` 函数说明符 |

### 2.4 C17 变化

C17主要是缺陷修复版本，没有新增头文件，保持与C11的向后兼容性。

## 3. 标准库函数分类

### 3.1 输入输出（I/O）

```c
#include <stdio.h>

// 标准I/O
printf("格式化输出: %d, %s\n", 42, "hello");
scanf("%d", &num);

// 文件I/O
FILE *fp = fopen("data.txt", "r");
fclose(fp);
```

### 3.2 内存管理

```c
#include <stdlib.h>

int *arr = (int *)malloc(10 * sizeof(int));
int *arr2 = (int *)calloc(10, sizeof(int));
arr = (int *)realloc(arr, 20 * sizeof(int));
free(arr);
free(arr2);
```

### 3.3 字符串处理

```c
#include <string.h>

char dest[50];
char *src = "Hello, World!";
strcpy(dest, src);
size_t len = strlen(dest);
int cmp = strcmp(dest, src);
```

### 3.4 数学运算

```c
#include <math.h>

double result = sqrt(16.0);      // 4.0
double power = pow(2.0, 10.0);   // 1024.0
double angle = sin(M_PI / 6.0);  // 0.5
double val = fabs(-3.14);        // 3.14
```

### 3.5 时间日期

```c
#include <time.h>

time_t now = time(NULL);
struct tm *local = localtime(&now);
printf("当前时间: %d-%02d-%02d\n",
       local->tm_year + 1900,
       local->tm_mon + 1,
       local->tm_mday);
```

### 3.6 字符处理

```c
#include <ctype.h>

char ch = 'A';
if (isalpha(ch)) {
    char lower = tolower(ch);  // 'a'
}
if (isdigit('5')) {
    // 数字字符
}
```

## 5. 标准库实现

### 5.1 常见C标准库实现

| 实现 | 平台 | 特点 |
|------|------|------|
| **glibc** | Linux | GNU C库，功能最全 |
| **musl** | Linux | 轻量级，静态链接友好 |
| **MSVC CRT** | Windows | Microsoft Visual C++运行时 |
| **uClibc** | 嵌入式 | 面向嵌入式系统 |
| **BSD libc** | BSD系列 | FreeBSD/NetBSD/OpenBSD |

### 5.2 链接方式

```bash
# 静态链接
gcc -static program.c -o program

# 动态链接（默认）
gcc program.c -o program -lm  # 链接数学库
```

## 6. 重要注意事项

> **要点一**：标准库函数需要包含对应的头文件，否则可能导致未定义行为。

> **要点二**：标准库函数在出错时通常通过返回值或 `errno` 传递错误信息，务必检查返回值。

> **要点三**：`malloc` 等动态分配函数分配的内存必须使用 `free` 释放，否则造成内存泄漏。

> **要点四**：标准库函数的实现可能因平台而异，编写可移植代码时应遵循标准规范。

> **要点五**：C11引入了带 `_s` 后缀的安全版本函数（如 `gets_s`、`scanf_s`），但这些属于可选特性（Annex K），并非所有编译器都支持。

## 7. 标准库使用的最佳实践

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <errno.h>

// 1. 始终检查malloc返回值
int *data = malloc(100 * sizeof(int));
if (data == NULL) {
    fprintf(stderr, "内存分配失败: %s\n", strerror(errno));
    exit(EXIT_FAILURE);
}

// 2. 文件操作后检查错误
FILE *fp = fopen("test.txt", "r");
if (fp == NULL) {
    perror("打开文件失败");
    return 1;
}

// 3. 释放内存后置空指针
free(data);
data = NULL;

// 4. 关闭文件后检查返回值
if (fclose(fp) != 0) {
    perror("关闭文件失败");
}
```
