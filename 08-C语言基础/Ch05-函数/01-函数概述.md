# 01 - 函数概述

## 一、什么是函数

函数（Function）是 C 语言程序的基本组成单元，是一段完成特定功能的、可重复使用的代码块。函数将复杂的程序逻辑分解为若干个小的、可管理的部分，是实现**模块化编程**的核心手段。

```c
// 一个简单的函数示例
int add(int a, int b) {
    return a + b;
}
```

## 二、函数的作用

### 2.1 代码复用

函数可以被多次调用，避免重复编写相同的代码：

```c
#include <stdio.h>

void print_separator() {
    printf("================================\n");
}

int main() {
    print_separator();
    printf("第一部分\n");
    print_separator();
    printf("第二部分\n");
    print_separator();
    return 0;
}
```

### 2.2 模块化与分解

将大型程序拆分为多个功能独立的函数，每个函数只做一件事：

```c
// 程序被分解为多个功能函数
void input_data(int arr[], int n);
void sort_data(int arr[], int n);
void output_data(int arr[], int n);
void calculate_stat(int arr[], int n);
```

### 2.3 提高可读性和可维护性

函数名本身起到了"注释"的作用，让代码逻辑一目了然：

```c
// 不使用函数 —— 难以理解
for (int i = 0; i < n - 1; i++)
    for (int j = 0; j < n - 1 - i; j++)
        if (arr[j] > arr[j+1]) { int t = arr[j]; arr[j] = arr[j+1]; arr[j+1] = t; }

// 使用函数 —— 语义清晰
bubble_sort(arr, n);
```

## 三、函数的分类

### 3.1 库函数（Library Functions）

由 C 标准库或第三方库提供的、无需用户自行定义的函数。使用前需包含对应的头文件。

| 头文件       | 常用函数                                      |
|-------------|---------------------------------------------|
| `<stdio.h>` | `printf`, `scanf`, `fopen`, `fclose`, `fgets` |
| `<stdlib.h>`| `malloc`, `free`, `atoi`, `exit`, `rand`     |
| `<string.h>`| `strlen`, `strcpy`, `strcmp`, `memcpy`       |
| `<math.h>`  | `sqrt`, `pow`, `sin`, `cos`, `fabs`         |
| `<time.h>`  | `time`, `clock`, `difftime`                 |

```c
#include <stdio.h>
#include <math.h>

int main() {
    double x = 16.0;
    printf("sqrt(%.0f) = %.2f\n", x, sqrt(x));   // 库函数 sqrt
    printf("pow(2, 10) = %.0f\n", pow(2, 10));    // 库函数 pow
    return 0;
}
```

### 3.2 自定义函数（User-Defined Functions）

程序员根据需求自行编写的函数：

```c
// 自定义：判断是否为素数
int is_prime(int n) {
    if (n < 2) return 0;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return 0;
    }
    return 1;
}

int main() {
    for (int i = 2; i <= 50; i++) {
        if (is_prime(i))
            printf("%d ", i);
    }
    return 0;
}
```

### 3.3 无参函数与有参函数

```c
// 无参函数
void greet(void) {
    printf("Hello, World!\n");
}

// 有参函数
int max(int a, int b) {
    return (a > b) ? a : b;
}
```

### 3.4 有返回值函数与无返回值函数

```c
// 有返回值函数
double average(double a, double b) {
    return (a + b) / 2.0;
}

// 无返回值函数（void）
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}
```

## 四、使用函数的优势

| 优势           | 说明                                          |
|---------------|---------------------------------------------|
| **代码复用**   | 同一函数可在程序不同位置被反复调用               |
| **降低复杂度** | 将大问题分解为多个小问题，逐个击破               |
| **便于调试**   | 每个函数可以单独测试，定位错误更快速             |
| **团队协作**   | 不同开发者可以同时编写不同的函数模块             |
| **便于维护**   | 修改功能只需改动对应的函数，不影响其他部分        |
| **提高可读性** | 函数名具有描述性，使代码逻辑更加清晰             |

## 五、函数的基本结构

```c
// 函数定义的一般形式
返回值类型 函数名(参数列表) {
    声明部分;
    执行部分;
    return 返回值;  // void 函数可省略
}
```

```c
// 实例：计算两个整数的最大公约数
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

## 六、函数的调用流程

```
main() 调用 foo()
    |
    v
进入 foo() 执行
    |
    v
foo() 执行完毕，返回到 main()
    |
    v
main() 继续执行后续代码
```

```c
#include <stdio.h>

int square(int n) {
    printf("  进入 square(%d)\n", n);
    int result = n * n;
    printf("  离开 square，返回 %d\n", result);
    return result;
}

int main() {
    printf("调用 square(5) 前\n");
    int r = square(5);
    printf("调用 square(5) 后，结果 = %d\n", r);
    return 0;
}
```

## 七、要点总结

> **关键点：**
> 1. 函数是 C 语言程序的基本组成单元，是模块化编程的核心。
> 2. 函数分为**库函数**和**自定义函数**两大类。
> 3. 使用函数可以实现**代码复用**、**降低复杂度**、**便于调试和维护**。
> 4. 每个 C 程序至少包含一个函数——`main` 函数，它是程序的入口。
> 5. 函数应当遵循**单一职责原则**，一个函数只完成一个明确的功能。
> 6. 良好的函数命名本身就是最好的代码注释。
