# 09 - 函数指针进阶

## 一、函数指针数组

### 1.1 基本概念

函数指针数组是一个数组，其每个元素都是一个函数指针。语法：

```c
返回值类型 (*数组名[大小])(参数列表);
```

### 1.2 基本示例

```c
#include <stdio.h>

int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }
int divide(int a, int b) { return b ? a / b : 0; }

int main() {
    // 函数指针数组
    int (*ops[4])(int, int) = {add, sub, mul, divide};
    const char *names[] = {"加", "减", "乘", "除"};

    int a = 12, b = 4;
    for (int i = 0; i < 4; i++) {
        printf("%s: %d %s %d = %d\n",
               names[i], a, names[i], b, ops[i](a, b));
    }
    return 0;
}
```

### 1.3 实际应用：计算器

```c
#include <stdio.h>

typedef int (*Operation)(int, int);

int add(int a, int b)      { return a + b; }
int subtract(int a, int b) { return a - b; }
int multiply(int a, int b) { return a * b; }
int modulo(int a, int b)   { return b ? a % b : 0; }

int main() {
    Operation ops[256] = {NULL};  // 用字符做索引
    ops['+'] = add;
    ops['-'] = subtract;
    ops['*'] = multiply;
    ops['%'] = modulo;

    int a, b;
    char op;
    printf("输入: 数字 运算符 数字\n");
    while (scanf("%d %c %d", &a, &op, &b) == 3) {
        if (ops[(unsigned char)op]) {
            printf("= %d\n", ops[(unsigned char)op](a, b));
        } else {
            printf("未知运算符\n");
        }
    }
    return 0;
}
```

## 二、指向函数指针的指针

### 2.1 二级函数指针

```c
int add(int a, int b) { return a + b; }

int main() {
    int (*fp)(int, int) = add;      // 一级指针
    int (**pp)(int, int) = &fp;     // 二级指针（指向函数指针的指针）

    printf("%d\n", (*pp)(3, 5));    // 通过二级指针调用
    printf("%d\n", (**pp)(3, 5));   // 等价写法
    return 0;
}
```

### 2.2 函数指针数组的传递

```c
typedef void (*SignalHandler)(int);

// 传递函数指针数组到函数
void execute_all(SignalHandler handlers[], int n, int signal) {
    for (int i = 0; i < n; i++) {
        if (handlers[i])
            handlers[i](signal);
    }
}
```

## 三、回调函数（Callback）

### 3.1 什么是回调函数

回调函数是通过函数指针传递给另一个函数的函数，由后者在适当时机调用。

```c
#include <stdio.h>

// 框架函数——接受回调
void process_array(int arr[], int n, void (*callback)(int)) {
    for (int i = 0; i < n; i++) {
        callback(arr[i]);   // 在循环中调用回调
    }
}

// 回调函数 1
void print_value(int x) {
    printf("值: %d\n", x);
}

// 回调函数 2
void print_square(int x) {
    printf("平方: %d\n", x * x);
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};

    printf("=== 打印值 ===\n");
    process_array(arr, 5, print_value);

    printf("=== 打印平方 ===\n");
    process_array(arr, 5, print_square);

    return 0;
}
```

### 3.2 带上下文的回调

```c
// 回调函数可以访问外部传入的数据
typedef void (*ItemFunc)(int value, void *context);

void for_each(int arr[], int n, ItemFunc func, void *ctx) {
    for (int i = 0; i < n; i++)
        func(arr[i], ctx);
}

// 回调：累加到 context 指向的变量
void accumulate(int value, void *context) {
    int *sum = (int*)context;
    *sum += value;
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int total = 0;
    for_each(arr, 5, accumulate, &total);
    printf("总和: %d\n", total);  // 15
    return 0;
}
```

## 四、qsort 的比较函数

### 4.1 qsort 函数签名

```c
void qsort(void *base, size_t nmemb, size_t size,
           int (*compar)(const void *, const void *));
```

- `base`: 数组首地址
- `nmemb`: 元素个数
- `size`: 每个元素的大小（字节）
- `compar`: 比较函数指针

### 4.2 比较函数规则

```c
// 比较函数返回值：
//   < 0  表示 a 应排在 b 前面
//   == 0 表示 a 和 b 相等
//   > 0  表示 a 应排在 b 后面
```

### 4.3 排序整数数组

```c
#include <stdio.h>
#include <stdlib.h>

int compare_int(const void *a, const void *b) {
    return (*(const int*)a) - (*(const int*)b);
}

int compare_int_desc(const void *a, const void *b) {
    return (*(const int*)b) - (*(const int*)a);
}

int main() {
    int arr[] = {5, 2, 8, 1, 9, 3};
    int n = sizeof(arr) / sizeof(arr[0]);

    // 升序
    qsort(arr, n, sizeof(int), compare_int);

    // 降序
    qsort(arr, n, sizeof(int), compare_int_desc);

    return 0;
}
```

### 4.4 排序字符串数组

```c
#include <string.h>

int compare_string(const void *a, const void *b) {
    return strcmp(*(const char**)a, *(const char**)b);
}

int main() {
    const char *names[] = {"Charlie", "Alice", "Bob", "David"};
    int n = 4;
    qsort(names, n, sizeof(char*), compare_string);

    for (int i = 0; i < n; i++)
        printf("%s\n", names[i]);
    return 0;
}
```

### 4.5 排序结构体

```c
typedef struct {
    char name[32];
    int age;
    double score;
} Student;

int compare_by_score(const void *a, const void *b) {
    double sa = ((const Student*)a)->score;
    double sb = ((const Student*)b)->score;
    if (sa < sb) return -1;
    if (sa > sb) return 1;
    return 0;
}

int compare_by_name(const void *a, const void *b) {
    return strcmp(((const Student*)a)->name,
                  ((const Student*)b)->name);
}
```

## 五、函数指针的 typedef 进阶

## 六、要点总结

> **关键点：**
> 1. 函数指针数组用 `type (*arr[N])(params)` 声明，适合实现**命令分发**和**状态机**。
> 2. 回调函数是将函数作为参数传给另一个函数，由后者在适当时机调用。
> 3. `qsort` 是回调函数的经典应用，比较函数决定了排序的规则。
> 4. 使用 `const void*` 作为比较函数参数可以实现泛型比较。
> 5. `typedef` 可以大幅简化复杂的函数指针声明。
> 6. 回调函数结合 `void*` 上下文参数可以实现灵活的数据处理。
