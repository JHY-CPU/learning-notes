# 03 - 函数的调用与返回

## 一、函数调用的基本方式

### 1.1 调用语法

```c
函数名(实参1, 实参2, ...);
// 或
返回变量 = 函数名(实参1, 实参2, ...);
```

### 1.2 基本示例

```c
#include <stdio.h>

int square(int n) {
    return n * n;
}

int main() {
    // 方式一：将返回值赋给变量
    int result = square(5);
    printf("5 的平方 = %d\n", result);

    // 方式二：直接使用返回值作为表达式
    printf("3 的平方 = %d\n", square(3));

    // 方式三：将返回值作为参数传给另一个函数
    printf("平方的平方 = %d\n", square(square(2)));

    return 0;
}
```

### 1.3 调用时的参数匹配

函数调用时的参数称为**实参（Actual Argument）**，函数定义中的参数称为**形参（Formal Argument）**。

```c
int add(int x, int y) {    // x, y 是形参
    return x + y;
}

int main() {
    int a = 3, b = 5;
    int sum = add(a, b);   // a, b 是实参
    // 调用时：a 的值 3 赋给 x，b 的值 5 赋给 y
    return 0;
}
```

> **注意：** 实参和形参在内存中是独立的存储单元，修改形参不会影响实参（值传递）。

## 二、return 语句

### 2.1 return 的两种形式

```c
// 形式一：return 后带表达式（用于有返回值的函数）
int max(int a, int b) {
    return (a > b) ? a : b;
}

// 形式二：return 后不带表达式（用于 void 函数或提前退出）
void print_positive(int n) {
    if (n <= 0) {
        printf("不是正数\n");
        return;  // 提前退出函数
    }
    printf("正数: %d\n", n);
}
```

### 2.2 多个 return 语句

一个函数中可以有多个 `return` 语句，但只会执行其中**第一个到达的**：

```c
const char* grade(int score) {
    if (score >= 90) return "优秀";
    if (score >= 80) return "良好";
    if (score >= 70) return "中等";
    if (score >= 60) return "及格";
    return "不及格";
}
```

### 2.3 return 的执行流程

```c
int divide(int a, int b, int *ok) {
    if (b == 0) {
        *ok = 0;      // 标记失败
        return 0;     // 立即退出，不会执行后面的代码
    }
    *ok = 1;
    return a / b;     // 正常返回
}
```

> **关键理解：** `return` 语句执行后，函数立即结束，后续代码不再执行。

## 三、返回值类型

### 3.1 各种返回值类型

```c
// 返回 int
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

// 返回 double
double celsius_to_fahrenheit(double c) {
    return c * 9.0 / 5.0 + 32.0;
}

// 返回 char
char to_upper(char c) {
    if (c >= 'a' && c <= 'z')
        return c - 'a' + 'A';
    return c;
}

// 返回指针
char* find_char(const char *str, char c) {
    while (*str) {
        if (*str == c) return (char*)str;
        str++;
    }
    return NULL;
}
```

### 3.2 返回值的类型转换

如果 `return` 表达式的类型与函数声明的返回类型不同，编译器会进行**隐式类型转换**：

```c
double average(int a, int b) {
    return (a + b) / 2;   // 注意：整数除法结果为 int，自动转为 double
    // 问题：average(3, 4) 返回 3.0 而不是 3.5
}

// 正确写法
double average_fixed(int a, int b) {
    return (a + b) / 2.0;  // 使用浮点数除法
}
```

## 四、void 函数

### 4.1 无返回值的函数

当函数不需要返回任何结果时，使用 `void` 作为返回类型：

```c
void print_array(int arr[], int n) {
    printf("[");
    for (int i = 0; i < n; i++) {
        printf("%d", arr[i]);
        if (i < n - 1) printf(", ");
    }
    printf("]\n");
}

void clear_buffer(char buf[], int size) {
    for (int i = 0; i < size; i++)
        buf[i] = '\0';
}
```

### 4.2 void 函数中的 return

```c
void process(int value) {
    if (value < 0) {
        printf("无效值\n");
        return;  // 提前返回，不执行下面的代码
    }
    printf("处理值: %d\n", value);
    // 函数结束时自动返回，不需要 return
}
```

### 4.3 void 函数不能用于赋值

```c
void say_hello(void) {
    printf("Hello!\n");
}

int main() {
    // int x = say_hello();  // 错误！void 函数没有返回值
    say_hello();             // 正确
    return 0;
}
```

## 五、嵌套调用

函数可以嵌套调用，即一个函数的返回值作为另一个函数的参数：

```c
#include <stdio.h>

int add(int a, int b)     { return a + b; }
int multiply(int a, int b) { return a * b; }
int square(int n)          { return n * n; }

int main() {
    // 嵌套调用：square(add(2, 3)) = square(5) = 25
    int result = square(add(2, 3));
    printf("结果: %d\n", result);

    // 更复杂的嵌套
    // multiply(add(1, 2), square(3)) = multiply(3, 9) = 27
    printf("嵌套: %d\n", multiply(add(1, 2), square(3)));

    return 0;
}
```

> **注意：** C 语言**不支持嵌套定义**函数（即不能在一个函数内部定义另一个函数），但支持嵌套调用。

## 六、递归调用

函数可以调用自身，这称为**递归**（详见 Ch06）：

```c
// 递归计算阶乘
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);  // 函数调用自身
}
```

## 七、函数调用的执行过程

```
程序从 main() 开始
      |
      v
执行到 square(5) 调用
      |
      v
  保存 main 的现场（返回地址、局部变量）
  创建 square 的栈帧
  将实参 5 传给形参 n
      |
      v
  执行 square 的函数体
  计算 5 * 5 = 25
  执行 return 25
      |
      v
  销毁 square 的栈帧
  恢复 main 的现场
  返回值 25 填入调用位置
      |
      v
main 继续执行
```

## 八、要点总结

> **关键点：**
> 1. 函数调用时，实参的值会被传递给形参（值传递）。
> 2. `return` 语句用于返回值并立即结束函数执行。
> 3. 一个函数可以有多个 `return`，但只会执行第一个。
> 4. `void` 函数没有返回值，`return` 可以用于提前退出。
> 5. C 语言支持函数的嵌套调用，但不支持嵌套定义。
> 6. 函数返回类型与 `return` 表达式类型不匹配时，会发生隐式类型转换。
> 7. 注意整数除法与浮点除法的区别，避免精度丢失。
