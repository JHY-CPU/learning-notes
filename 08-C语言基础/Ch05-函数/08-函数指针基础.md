# 08 - 函数指针基础

## 一、什么是函数指针

**函数指针（Function Pointer）** 是指向函数的指针变量。函数在内存中也有地址（函数体代码的入口地址），函数指针存储的就是这个地址。

```c
// 函数定义
int add(int a, int b) {
    return a + b;
}

// 函数指针可以指向 add
int (*p)(int, int) = add;   // p 指向 add 函数
int result = p(3, 5);       // 通过指针调用函数，结果为 8
```

## 二、函数指针的定义语法

### 2.1 基本语法

```c
返回值类型 (*指针名)(参数类型列表);
```

```c
// 指向 int func(int, int) 类型函数的指针
int (*fp)(int, int);

// 指向 double func(double) 类型函数的指针
double (*gp)(double);

// 指向 void func(void) 类型函数的指针
void (*hp)(void);
```

### 2.2 注意括号的必要性

```c
int *fp(int, int);    // 这是函数声明！返回 int* 的函数 fp
int (*fp)(int, int);  // 这是函数指针！指向返回 int 的函数
```

> **记忆技巧：** `(*fp)` 的括号必不可少，否则 `*fp(int, int)` 会被解析为"函数声明"而非"指针声明"。

## 三、函数指针的赋值与调用

### 3.1 赋值

```c
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }

int main() {
    // 方式一：直接赋函数名
    int (*fp)(int, int) = add;

    // 方式二：取地址运算符（效果相同）
    int (*gp)(int, int) = &sub;

    // 函数名本身就是一个函数指针常量
    // add 和 &add 的值相同
    return 0;
}
```

### 3.2 通过函数指针调用

```c
int add(int a, int b) { return a + b; }

int main() {
    int (*fp)(int, int) = add;

    // 以下三种调用方式等价
    int r1 = fp(3, 5);        // 方式一：直接用指针名调用
    int r2 = (*fp)(3, 5);     // 方式二：先解引用再调用
    int r3 = add(3, 5);       // 方式三：直接调用原函数

    printf("%d %d %d\n", r1, r2, r3);  // 都是 8
    return 0;
}
```

### 3.3 函数指针的重新赋值

```c
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }

int main() {
    int (*fp)(int, int) = add;
    printf("add: %d\n", fp(10, 3));   // 13

    fp = sub;  // 指向另一个函数
    printf("sub: %d\n", fp(10, 3));   // 7

    fp = mul;
    printf("mul: %d\n", fp(10, 3));   // 30

    return 0;
}
```

## 四、typedef 简化函数指针

### 4.1 使用 typedef 定义别名

```c
// 定义 BinOp 为 "指向 int(int, int) 函数" 的类型
typedef int (*BinOp)(int, int);

// 现在 BinOp 就是一个类型名
BinOp fp = add;
int result = fp(3, 5);
```

### 4.2 更多示例

```c
typedef double (*MathFunc)(double);
typedef void (*Callback)(int);
typedef int (*Comparator)(const void*, const void*);

// 使用
MathFunc f = sin;
double val = f(3.14159 / 2);
```

### 4.3 复杂情况——函数指针作为参数

```c
// 不用 typedef（比较难读）
int compute(int (*op)(int, int), int a, int b) {
    return op(a, b);
}

// 用 typedef（更清晰）
typedef int (*BinOp)(int, int);
int compute(BinOp op, int a, int b) {
    return op(a, b);
}
```

## 五、函数指针作为函数参数

### 5.1 回调模式

```c
#include <stdio.h>

typedef void (*Action)(int);

void for_each(int arr[], int n, Action action) {
    for (int i = 0; i < n; i++) {
        action(arr[i]);
    }
}

void print_int(int x) {
    printf("%d ", x);
}

void print_square(int x) {
    printf("%d ", x * x);
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int n = 5;

    printf("原值: ");
    for_each(arr, n, print_int);
    printf("\n");

    printf("平方: ");
    for_each(arr, n, print_square);
    printf("\n");

    return 0;
}
```

## 六、比较函数指针

```c
#include <stdio.h>

int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }

int main() {
    int (*fp1)(int, int) = add;
    int (*fp2)(int, int) = add;
    int (*fp3)(int, int) = sub;

    // 指向同一函数的指针比较相等
    if (fp1 == fp2) printf("fp1 和 fp2 指向同一函数\n");

    // 指向不同函数的指针不等
    if (fp1 != fp3) printf("fp1 和 fp3 指向不同函数\n");

    return 0;
}
```

## 七、函数指针与 NULL

```c
int (*fp)(int, int) = NULL;

if (fp != NULL) {
    fp(3, 5);  // 仅在非空时调用
}

// 不安全的调用——可能导致段错误
// fp(3, 5);  // fp 是 NULL，段错误！
```

## 八、要点总结

> **关键点：**
> 1. 函数指针存储函数的入口地址，可以通过它调用函数。
> 2. 语法：`返回值类型 (*指针名)(参数列表)`，括号不能省略。
> 3. 函数名本身就是函数指针常量，`func` 和 `&func` 等价。
> 4. 使用 `typedef` 可以大幅简化复杂的函数指针声明。
> 5. 函数指针是**回调函数**和**策略模式**的基础。
> 6. 赋值和调用时必须确保函数签名（返回类型+参数类型）完全匹配。
> 7. 使用前检查函数指针是否为 `NULL`，防止段错误。
