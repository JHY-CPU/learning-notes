# 05 - 函数的参数与返回值

## 一、多参数函数

### 1.1 接受多个参数

C 语言函数可以接受任意数量的参数（受编译器限制）：

```c
// 计算多个值的加权平均
double weighted_avg(double a, double wa, double b, double wb) {
    return (a * wa + b * wb) / (wa + wb);
}

// 打印学生成绩
void print_student(const char *name, int age, double score) {
    printf("姓名: %s, 年龄: %d, 成绩: %.1f\n", name, age, score);
}
```

### 1.2 参数数量过多的问题

```c
// 参数太多，调用容易出错
void create_window(int x, int y, int width, int height,
                   const char *title, int border, int resizable,
                   int fullscreen, int vsync) {
    // ...
}

// 更好的方式：使用结构体封装参数
typedef struct {
    int x, y;
    int width, height;
    const char *title;
    int border;
    int resizable;
    int fullscreen;
    int vsync;
} WindowConfig;

void create_window(const WindowConfig *config) {
    // 通过 config->x, config->width 等访问
}
```

## 二、C 语言不支持默认参数

### 2.1 其他语言的默认参数（C 不支持）

```cpp
// C++ 支持默认参数
// void print_msg(const char *msg, int times = 1);

// C 语言不支持，需要其他方式模拟
```

### 2.2 模拟默认参数的方式

**方式一：函数重载（C 不支持，仅 C++）**

```cpp
// C++ 中的函数重载
void log(const char *msg);
void log(const char *msg, int level);
```

**方式二：用不同的函数名**

```c
// C 语言的替代方案
void log(const char *msg) {
    log_with_level(msg, 0);
}

void log_with_level(const char *msg, int level) {
    // 实际实现
    printf("[%d] %s\n", level, msg);
}
```

**方式三：传递特殊值表示"使用默认值"**

```c
// 用 -1 或 NULL 表示使用默认值
void connect(const char *host, int port) {
    if (port < 0) port = 80;  // 默认端口
    // ...
}

connect("example.com", -1);   // 使用默认端口 80
connect("example.com", 8080); // 使用指定端口
```

## 三、C 语言只有一个返回值

### 3.1 限制

C 语言函数只能通过 `return` 返回**一个值**：

```c
int add(int a, int b) {
    return a + b;  // 只能返回一个值
}
```

### 3.2 返回多个值的技巧

#### 技巧一：通过指针参数"返回"

```c
// 函数返回一个值，其余值通过指针参数传出
int divide(int a, int b, int *remainder) {
    if (b == 0) return -1;  // 错误码
    *remainder = a % b;     // 通过指针传出余数
    return a / b;           // 通过 return 返回商
}

int main() {
    int rem;
    int quot = divide(17, 5, &rem);
    printf("商 = %d, 余数 = %d\n", quot, rem);  // 商 = 3, 余数 = 2
    return 0;
}
```

#### 技巧二：通过结构体返回

```c
typedef struct {
    int quotient;
    int remainder;
    int valid;
} DivResult;

DivResult divide_v2(int a, int b) {
    DivResult result = {0, 0, 0};
    if (b == 0) return result;

    result.quotient = a / b;
    result.remainder = a % b;
    result.valid = 1;
    return result;
}

int main() {
    DivResult r = divide_v2(17, 5);
    if (r.valid)
        printf("商 = %d, 余数 = %d\n", r.quotient, r.remainder);
    return 0;
}
```

#### 技巧三：通过数组参数返回

```c
// 将计算结果写入数组
void solve_quadratic(double a, double b, double c, double roots[2]) {
    double disc = b * b - 4 * a * c;
    if (disc < 0) {
        roots[0] = roots[1] = NAN;
        return;
    }
    roots[0] = (-b + sqrt(disc)) / (2 * a);
    roots[1] = (-b - sqrt(disc)) / (2 * a);
}
```

#### 技巧四：通过全局变量（不推荐）

```c
// 全局变量存储额外返回值——不推荐使用
int g_error_code;

int risky_operation(int input) {
    if (input < 0) {
        g_error_code = -1;
        return 0;
    }
    g_error_code = 0;
    return input * 2;
}
```

## 四、错误码与返回值的结合

### 4.1 用返回值表示成功/失败

```c
// 返回 0 表示成功，非 0 表示错误
int read_config(const char *filename, Config *cfg) {
    FILE *f = fopen(filename, "r");
    if (!f) return -1;    // 文件打开失败

    if (fscanf(f, "%d", &cfg->value) != 1) {
        fclose(f);
        return -2;        // 读取失败
    }

    fclose(f);
    return 0;             // 成功
}
```

### 4.2 用指针参数返回实际数据

```c
// 返回错误码，数据通过指针传出
int get_user_info(int user_id, char *name, int name_size, int *age) {
    if (user_id <= 0) return -1;  // 无效 ID

    // 模拟数据库查询
    strncpy(name, "张三", name_size);
    *age = 25;
    return 0;  // 成功
}

int main() {
    char name[64];
    int age;
    if (get_user_info(1001, name, sizeof(name), &age) == 0) {
        printf("用户: %s, 年龄: %d\n", name, age);
    } else {
        printf("查询失败\n");
    }
    return 0;
}
```

## 六、参数的设计建议

| 建议                       | 说明                                          |
|---------------------------|---------------------------------------------|
| 参数数量不超过 3-5 个      | 参数过多说明函数职责不单一，考虑用结构体封装       |
| 输入参数用 `const` 修饰    | 明确表示该参数不会被修改，增强代码可读性           |
| 输出参数放在参数列表后面    | 如 `int func(int in1, int in2, int *out)`     |
| 返回值优先于输出参数        | 简单情况下，直接返回值比指针参数更直观             |
| 指针参数检查 NULL          | 防止空指针导致段错误                             |

```c
// 良好的参数设计
// const 修饰输入参数，指针用于输出参数
int compute_statistics(const double *data, int n,
                       double *mean, double *stddev) {
    if (!data || n <= 0 || !mean || !stddev) return -1;

    double sum = 0;
    for (int i = 0; i < n; i++) sum += data[i];
    *mean = sum / n;

    double var = 0;
    for (int i = 0; i < n; i++) {
        double diff = data[i] - *mean;
        var += diff * diff;
    }
    *stddev = sqrt(var / n);
    return 0;
}
```

## 七、要点总结

> **关键点：**
> 1. C 语言函数只能返回一个值，但可以通过指针参数"传出"额外的结果。
> 2. C 语言**不支持默认参数**，需要用不同的函数名或特殊值来模拟。
> 3. 使用结构体封装参数和返回值，可以处理复杂的数据传递需求。
> 4. 函数参数过多通常是设计不良的信号，考虑重构。
> 5. 用 `const` 修饰输入参数，用指针参数传递输出结果。
> 6. 指针参数在使用前应检查是否为 NULL。
