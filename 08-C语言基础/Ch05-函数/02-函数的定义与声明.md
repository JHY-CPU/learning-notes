# 02 - 函数的定义与声明

## 一、函数定义的语法

### 1.1 基本语法格式

```c
返回值类型 函数名(参数类型 参数名1, 参数类型 参数名2, ...) {
    // 函数体
    // 局部变量声明
    // 执行语句
    return 表达式;  // 如果返回类型不是 void
}
```

### 1.2 完整示例

```c
// 定义一个函数：计算两数之和
int add(int a, int b) {
    int sum = a + b;
    return sum;
}

// 定义一个无返回值的函数：打印分隔线
void print_line(char ch, int length) {
    for (int i = 0; i < length; i++) {
        putchar(ch);
    }
    putchar('\n');
}

// 定义一个无参数的函数：获取当前计数值
int get_counter(void) {
    static int count = 0;
    return ++count;
}
```

### 1.3 函数定义的各部分说明

| 组成部分       | 说明                                      | 示例                    |
|--------------|------------------------------------------|------------------------|
| 返回值类型     | 函数执行完毕后返回的数据类型               | `int`, `double`, `void` |
| 函数名         | 函数的标识符，遵循标识符命名规则           | `add`, `is_prime`      |
| 参数列表       | 函数接收的输入，可以为空                   | `(int a, int b)`       |
| 函数体         | 实现功能的具体代码                         | `{ ... }`              |
| return 语句   | 返回结果并结束函数执行                     | `return sum;`          |

## 二、函数声明（函数原型）

### 2.1 为什么需要函数声明

C 语言编译器按**从上到下**的顺序编译代码。如果函数定义在调用之后，编译器不认识这个函数，会产生**隐式声明警告**或**编译错误**。

```c
// 错误示例：函数调用在定义之前
int main() {
    int result = add(3, 5);  // 编译器不知道 add 是什么！
    return 0;
}

int add(int a, int b) {
    return a + b;
}
```

### 2.2 函数声明（原型）语法

```c
返回值类型 函数名(参数类型1, 参数类型2, ...);
```

**声明可以只写参数类型，不写参数名：**

```c
// 以下两种声明方式等价
int add(int a, int b);
int add(int, int);        // 省略参数名
```

### 2.3 正确的使用方式

```c
#include <stdio.h>

// 函数声明（原型）
int add(int, int);
void print_result(int);
double circle_area(double);

int main() {
    int sum = add(10, 20);
    print_result(sum);
    printf("圆的面积 = %.2f\n", circle_area(5.0));
    return 0;
}

// 函数定义
int add(int a, int b) {
    return a + b;
}

void print_result(int result) {
    printf("结果是: %d\n", result);
}

double circle_area(double radius) {
    return 3.14159265358979 * radius * radius;
}
```

## 三、在头文件中声明函数

### 3.1 将声明放入头文件

在实际项目中，函数声明通常放在 `.h` 头文件中，函数定义放在 `.c` 源文件中。

**math_utils.h（头文件——声明）**
```c
#ifndef MATH_UTILS_H
#define MATH_UTILS_H

int add(int a, int b);
int subtract(int a, int b);
int multiply(int a, int b);
double divide(int a, int b);

#endif
```

**math_utils.c（源文件——定义）**
```c
#include "math_utils.h"

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int multiply(int a, int b) {
    return a * b;
}

double divide(int a, int b) {
    if (b == 0) {
        fprintf(stderr, "错误：除数不能为零\n");
        return 0.0;
    }
    return (double)a / b;
}
```

**main.c（使用头文件）**
```c
#include <stdio.h>
#include "math_utils.h"

int main() {
    printf("add(3, 5) = %d\n", add(3, 5));
    printf("divide(10, 3) = %.2f\n", divide(10, 3));
    return 0;
}
```

### 3.2 编译多文件程序

```bash
gcc -o myprogram main.c math_utils.c
```

## 四、函数定义与声明的区别

| 特性       | 函数声明（Prototype）          | 函数定义（Definition）          |
|-----------|------------------------------|------------------------------|
| 作用       | 告诉编译器函数的存在和签名      | 提供函数的具体实现代码          |
| 函数体     | **没有**函数体，以分号结尾     | **有**函数体，用 `{}` 包裹    |
| 参数名     | 可以省略                      | 不能省略                      |
| 出现次数   | 可以出现多次                   | **只能出现一次**               |
| 放置位置   | 通常在头文件或源文件顶部        | 通常在源文件中                 |

## 五、常见错误

### 5.1 声明和定义不一致

```c
// 声明
double average(int a, int b);

// 定义 —— 返回类型不匹配！编译错误
int average(int a, int b) {
    return (a + b) / 2;
}
```

### 5.2 忘记函数声明

```c
// 如果不声明 foo，直接调用，C89 编译器会假设返回 int
int main() {
    double r = foo(3.14);  // 隐式声明，默认认为 foo 返回 int
    // 可能导致未定义行为
    return 0;
}

double foo(double x) {
    return x * 2;
}
```

> **注意：** C99 标准不再支持隐式函数声明，未声明就调用会直接报错。

### 5.3 多次定义同名函数

```c
int add(int a, int b) { return a + b; }
int add(int x, int y) { return x + y; }  // 重复定义！链接错误
```

## 六、K&R 风格声明（旧式风格，了解即可）

在 ANSI C 标准之前（K&R C），函数参数的类型写在括号外面：

```c
// K&R 风格（旧式，不推荐）
int add(a, b)
    int a;
    int b;
{
    return a + b;
}

// ANSI C 风格（现代，推荐）
int add(int a, int b) {
    return a + b;
}
```

## 七、要点总结

> **关键点：**
> 1. 函数定义包含完整的函数体，函数声明只告诉编译器函数的签名。
> 2. **先声明后使用**是 C 语言的基本规则，否则编译器会报错。
> 3. 函数声明中参数名可以省略，只保留类型。
> 4. 实际项目中，函数声明放在 `.h` 头文件中，定义放在 `.c` 源文件中。
> 5. 函数定义在程序中只能出现一次，但声明可以出现多次。
> 6. 声明和定义的返回类型、参数类型必须严格一致。
