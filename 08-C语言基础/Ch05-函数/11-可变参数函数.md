# 11 - 可变参数函数

## 一、什么是可变参数函数

**可变参数函数（Variadic Function）** 是参数数量可变的函数。最典型的例子是 `printf` 和 `scanf`，它们可以根据格式字符串接受不同数量的参数。

```c
// printf 的参数数量取决于格式字符串
printf("Hello\n");                     // 1 个参数
printf("x = %d\n", 42);               // 2 个参数
printf("%d + %d = %d\n", 1, 2, 3);    // 4 个参数
```

## 二、stdarg.h 头文件

C 语言通过 `<stdarg.h>` 头文件提供可变参数的处理机制。

### 2.1 核心宏

| 宏/类型        | 说明                                      |
|---------------|------------------------------------------|
| `va_list`     | 一种类型，用于存储可变参数的信息             |
| `va_start`    | 初始化 `va_list`，指定最后一个固定参数       |
| `va_arg`      | 获取下一个参数，需指定类型                  |
| `va_end`      | 清理 `va_list`，使用完毕后必须调用          |
| `va_copy`     | 复制一个 `va_list`（C99）                  |

### 2.2 使用步骤

```c
#include <stdarg.h>

返回值_type 函数名(固定参数, ...) {    // ... 表示可变参数
    va_list args;                       // 1. 声明 va_list
    va_start(args, 固定参数);            // 2. 初始化，指定最后一个固定参数
    类型 value = va_arg(args, 类型);     // 3. 逐个获取参数
    va_end(args);                       // 4. 清理
}
```

## 三、实现简单的可变参数函数

### 3.1 求任意个整数的和

```c
#include <stdio.h>
#include <stdarg.h>

// 求 n 个整数的和
int sum(int n, ...) {
    va_list args;
    va_start(args, n);        // n 是最后一个固定参数

    int total = 0;
    for (int i = 0; i < n; i++) {
        total += va_arg(args, int);  // 逐个取出 int 类型的参数
    }

    va_end(args);             // 清理
    return total;
}

int main() {
    printf("sum(3, 10, 20, 30) = %d\n", sum(3, 10, 20, 30));   // 60
    printf("sum(5, 1,2,3,4,5) = %d\n", sum(5, 1, 2, 3, 4, 5)); // 15
    return 0;
}
```

### 3.2 求任意个 double 的平均值

```c
#include <stdarg.h>

double average(int n, ...) {
    va_list args;
    va_start(args, n);

    double total = 0;
    for (int i = 0; i < n; i++) {
        total += va_arg(args, double);  // 注意：float 会被提升为 double
    }

    va_end(args);
    return n > 0 ? total / n : 0;
}

int main() {
    printf("avg = %.2f\n", average(4, 1.0, 2.0, 3.0, 4.0)); // 2.50
    return 0;
}
```

## 四、实现 printf 的变体

### 4.1 简单的日志函数

```c
#include <stdio.h>
#include <stdarg.h>
#include <time.h>

void log_message(const char *level, const char *fmt, ...) {
    // 打印时间戳和级别
    time_t now = time(NULL);
    struct tm *t = localtime(&now);
    printf("[%02d:%02d:%02d][%s] ",
           t->tm_hour, t->tm_min, t->tm_sec, level);

    // 处理格式字符串
    va_list args;
    va_start(args, fmt);
    vprintf(fmt, args);   // vprintf 接受 va_list 而不是 ...
    va_end(args);

    printf("\n");
}

int main() {
    log_message("INFO", "程序启动");
    log_message("DEBUG", "变量 x = %d, y = %f", 42, 3.14);
    log_message("ERROR", "文件 %s 不存在", "config.txt");
    return 0;
}
```

### 4.2 格式化到字符串

```c
#include <stdio.h>
#include <stdarg.h>

// 将格式化结果写入字符串
int my_snprintf(char *buf, size_t size, const char *fmt, ...) {
    va_list args;
    va_start(args, fmt);
    int len = vsnprintf(buf, size, fmt, args);
    va_end(args);
    return len;
}

int main() {
    char buf[256];
    my_snprintf(buf, sizeof(buf), "User: %s, Age: %d", "Alice", 25);
    printf("%s\n", buf);
    return 0;
}
```

## 五、使用哨兵值结束参数

当不预先传递参数个数时，可以用一个特殊值（哨兵）标记参数结束：

```c
#include <stdarg.h>
#include <stdio.h>

// 用 -1 作为结束标记
int sum_until_sentinel(int first, ...) {
    int total = first;

    va_list args;
    va_start(args, first);

    int val;
    while ((val = va_arg(args, int)) != -1) {
        total += val;
    }

    va_end(args);
    return total;
}

int main() {
    // 最后必须以 -1 结尾
    printf("sum = %d\n", sum_until_sentinel(10, 20, 30, -1));        // 60
    printf("sum = %d\n", sum_until_sentinel(1, 2, 3, 4, 5, -1));     // 15
    printf("sum = %d\n", sum_until_sentinel(100, -1));                // 100
    return 0;
}
```

## 六、不同类型的可变参数

```c
#include <stdarg.h>
#include <stdio.h>

// 打印不同类型的值
// 第一个参数是格式字符串，其中字符指定后续参数类型
// 'd'=int, 'f'=double, 's'=char*
void print_mixed(const char *fmt, ...) {
    va_list args;
    va_start(args, fmt);

    for (const char *p = fmt; *p; p++) {
        switch (*p) {
            case 'd':
                printf("%d ", va_arg(args, int));
                break;
            case 'f':
                printf("%.2f ", va_arg(args, double));  // float 提升为 double
                break;
            case 's':
                printf("%s ", va_arg(args, char*));
                break;
            default:
                printf("? ");
        }
    }
    printf("\n");
    va_end(args);
}

int main() {
    print_mixed("dfs", 42, 3.14, "hello");
    // 输出: 42 3.14 hello
    return 0;
}
```

## 七、va_list 的多次遍历

```c
#include <stdarg.h>
#include <stdio.h>

// 用 va_copy 保存参数列表的副本
double stats(int n, ...) {
    va_list args, args_copy;

    va_start(args, n);
    va_copy(args_copy, args);  // 复制一份

    // 第一次遍历：求和
    double sum = 0;
    for (int i = 0; i < n; i++)
        sum += va_arg(args, double);

    // 第二次遍历：求最大值
    double max_val = va_arg(args_copy, double);
    for (int i = 1; i < n; i++) {
        double v = va_arg(args_copy, double);
        if (v > max_val) max_val = v;
    }

    va_end(args);
    va_end(args_copy);

    printf("和: %.1f, 最大值: %.1f\n", sum, max_val);
    return sum;
}
```

## 八、注意事项

### 8.1 类型提升

C 语言的可变参数中，`float` 会被自动提升为 `double`，`char` 和 `short` 会被提升为 `int`：

```c
// va_arg 必须使用提升后的类型
float f = va_arg(args, double);   // 正确：float 提升为 double
// float f = va_arg(args, float); // 错误！
```

### 8.2 不能确定参数数量和类型

```c
// printf 通过格式字符串推断参数——但这是调用者的责任
// 如果格式字符串与实际参数不匹配，是未定义行为！
printf("%d %s\n", "hello", 42);   // 类型不匹配，未定义行为！
```

### 8.3 没有参数时的行为

```c
// 至少需要一个固定参数
// void bad(...);  // 不合法！至少需要一个固定参数
void ok(int count, ...);  // 合法
```

## 九、要点总结

> **关键点：**
> 1. 可变参数函数通过 `<stdarg.h>` 的宏来处理不定数量的参数。
> 2. `va_start(args, last_fixed)` 初始化，`va_arg(args, type)` 获取参数，`va_end(args)` 清理。
> 3. 使用 `vprintf`、`vsnprintf` 等函数可以传递 `va_list` 给标准库函数。
> 4. `float` 在可变参数中自动提升为 `double`，`char/short` 提升为 `int`。
> 5. 函数至少需要一个固定参数，不能只有 `...`。
> 6. 参数的数量和类型由调用者和被调用者之间的"约定"决定，编译器无法检查。
