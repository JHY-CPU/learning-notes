# if 语句基础

## 1. if 语句概述

`if` 语句是C语言中最基本的分支语句，用于实现**单分支选择结构**：当条件为真时执行指定的代码块，条件为假时跳过该代码块继续执行后续语句。

## 2. 基本语法

```c
if (表达式) {
    // 表达式为真（非零）时执行的语句
}
```

执行流程：

```
表达式 → 非零（真）→ 执行语句块 → 继续后续代码
    │
    └→ 零（假）────→ 跳过语句块 → 继续后续代码
```

**关键说明：**
- `if` 后面的括号是必须的，不能省略
- 括号内的表达式必须是一个标量类型（整数或指针），不能是结构体或数组
- 如果只有一条语句，花括号可以省略（但**强烈建议不要省略**）
- 花括号内可以有多条语句

## 3. 第一个 if 程序

```c
#include <stdio.h>

int main(void) {
    int temperature = 38;

    if (temperature > 37) {
        printf("体温偏高，注意休息\n");
    }

    printf("程序继续执行\n");
    return 0;
}
```

输出：
```
体温偏高，注意休息
程序继续执行
```

注意：无论条件是否成立，`"程序继续执行"` 都会输出，因为它是 if 语句之后的代码。

## 4. 条件表达式

`if` 后面的圆括号内是一个**表达式**，C语言会对这个表达式求值：
- 结果为 **非零值** → 条件为**真（true）**
- 结果为 **零** → 条件为**假（false）**

### 4.1 常见的条件表达式形式

```c
// 比较运算
if (x > 0)    { /* ... */ }
if (x == 0)   { /* ... */ }
if (x != y)   { /* ... */ }
if (x >= 10)  { /* ... */ }

// 逻辑运算
if (x > 0 && x < 100)  { /* ... */ }  // AND
if (x < 0 || x > 100)  { /* ... */ }  // OR
if (!done)             { /* ... */ }  // NOT

// 函数返回值
if (is_valid(data))  { /* ... */ }

// 变量本身（非零为真）
if (count)  { /* ... */ }  // 等价于 if (count != 0)
```

### 4.2 真值判断规则

C语言中，任何标量类型的值都可以作为条件：

```c
#include <stdio.h>

int main(void) {
    // 整数：0为假，非零为真
    if (1)    printf("1为真\n");       // 输出
    if (-1)   printf("-1为真\n");      // 输出（负数也是真）
    if (0)    printf("0为真\n");       // 不输出
    if (42)   printf("42为真\n");      // 输出

    // 浮点数：0.0为假，非零为真
    if (3.14) printf("3.14为真\n");    // 输出
    if (0.0)  printf("0.0为真\n");     // 不输出

    // 指针：NULL为假，非NULL为真
    int x = 10;
    int *p = &x;
    if (p)    printf("p非空\n");       // 输出

    int *q = NULL;
    if (q)    printf("q非空\n");       // 不输出

    // 字符：'\0'为假，其他为真
    if ('A')  printf("A为真\n");       // 输出
    if ('\0') printf("\\0为真\n");     // 不输出

    return 0;
}
```

**重要提示**：浮点数与0.0的比较由于精度问题可能会产生意外结果，通常需要使用误差范围进行比较：

```c
#include <math.h>

double value = 0.1 + 0.2 - 0.3;  // 理论上是0.0

// 不推荐：可能因浮点误差产生错误判断
if (value == 0.0) { /* ... */ }

// 推荐：使用误差范围
if (fabs(value) < 1e-9) { /* ... */ }
```

## 5. 单语句与复合语句

### 5.1 单语句（省略花括号）

```c
if (x > 0)
    printf("x是正数\n");
```

当if体内只有一条语句时，花括号可以省略。但这容易导致问题：

```c
// 危险的写法
if (x > 0)
    printf("x是正数\n");
    printf("x的值是%d\n", x);  // 这行不受if控制！
```

上面的代码，第二条 `printf` 总会执行，因为它不在if体内。这在代码缩进错误时极容易误导人。

### 5.2 复合语句（推荐写法）

```c
// 推荐的写法——始终使用花括号
if (x > 0) {
    printf("x是正数\n");
    printf("x的值是%d\n", x);  // 现在这两行都在if体内
}
```

**最佳实践**：无论if体内有多少条语句，都始终使用花括号包裹。这是一个防御性的编程习惯，能有效避免将来的修改引入bug。

## 6. if 语句的常见用法

### 6.1 边界检查

```c
#include <stdio.h>

int main(void) {
    int age;

    printf("请输入年龄: ");
    scanf("%d", &age);

    if (age < 0 || age > 150) {
        printf("输入的年龄无效！\n");
    }

    if (age >= 18) {
        printf("已成年\n");
    }

    return 0;
}
```

### 6.2 错误检测

```c
#include <stdio.h>

int main(void) {
    FILE *fp = fopen("data.txt", "r");

    if (fp == NULL) {
        printf("无法打开文件！\n");
        return 1;
    }

    // 文件操作...
    fclose(fp);
    return 0;
}
```

### 6.3 更新变量

```c
int max(int a, int b) {
    int result = a;       // 假设a是最大值

    if (b > a) {          // 如果b更大
        result = b;       // 更新最大值
    }

    return result;
}
```

### 6.4 设置标志

```c
#include <stdio.h>

int main(void) {
    int numbers[] = {3, 7, 2, 9, 5};
    int target = 9;
    int found = 0;        // 标志变量：0表示未找到
    int i;

    for (i = 0; i < 5; i++) {
        if (numbers[i] == target) {
            found = 1;    // 找到了，设置标志
        }
    }

    if (found) {
        printf("找到了 %d\n", target);
    } else {
        printf("未找到 %d\n", target);
    }

    return 0;
}
```

## 7. if 语句中的常见错误

### 7.1 赋值与比较混淆

```c
int x = 5;

// 错误：这是赋值，不是比较！
// if (x = 0) 把0赋给x，结果为0，条件为假

// 正确：使用 == 进行比较
if (x == 0) {
    printf("x等于0\n");
}
```

这是一个非常经典的C语言错误。编译器通常会对 `if (x = 0)` 发出警告，但不会报错。一种防御性写法是将常量放在左边：

```c
if (0 == x) {   // 如果不小心写成 0 = x，编译器会报错
    printf("x等于0\n");
}
```

### 7.2 忘记花括号

```c
// 看起来像是两条语句都在if体内
if (score >= 60)
    printf("及格\n");
    printf("恭喜\n");    // 这行永远会执行！
```

### 7.3 浮点数比较

```c
// 不可靠
if (x == 0.3) { /* ... */ }

// 可靠
if (fabs(x - 0.3) < 1e-9) { /* ... */ }
```

## 8. 要点总结

1. `if` 后的表达式求值结果：非零为真，零为假
2. 建议始终使用花括号，即使只有一条语句
3. 注意区分 `=`（赋值）和 `==`（相等比较）
4. 浮点数比较应使用误差范围，避免 `==`
5. C语言中没有布尔类型（C99之前），用整数表示真假：0为假，非零为真
6. 头文件 `<stdbool.h>`（C99引入）提供了 `bool`、`true`、`false` 的定义
