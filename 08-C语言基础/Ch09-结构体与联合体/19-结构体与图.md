# 19 - 结构体与图

## 一、图的基本概念

图（Graph）由**顶点（Vertex）** 和 **边（Edge）** 组成，是比树更通用的数据结构。图可以用多种结构体形式表示。

## 二、邻接矩阵表示

### 2.1 结构体定义

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#define MAX_VERTICES 100

typedef struct {
    int num_vertices;                    // 实际顶点数
    int adj[MAX_VERTICES][MAX_VERTICES]; // 邻接矩阵，adj[i][j] 表示边权
    char labels[MAX_VERTICES][32];       // 顶点标签（可选）
} GraphMatrix;
```

### 2.2 操作实现

```c
void graph_mat_init(GraphMatrix *g, int vertices) {
    g->num_vertices = vertices;
    memset(g->adj, 0, sizeof(g->adj));
}

// 添加无向边
void graph_mat_add_edge(GraphMatrix *g, int u, int v, int weight) {
    g->adj[u][v] = weight;
    g->adj[v][u] = weight;  // 无向图对称
}

// 添加有向边
void graph_mat_add_directed(GraphMatrix *g, int u, int v, int weight) {
    g->adj[u][v] = weight;
}

// 判断是否有边
int graph_mat_has_edge(GraphMatrix *g, int u, int v) {
    return g->adj[u][v] != 0;
}

// 打印邻接矩阵
void graph_mat_print(const GraphMatrix *g) {
    printf("邻接矩阵 (%d 个顶点):\n", g->num_vertices);
    for (int i = 0; i < g->num_vertices; i++) {
        for (int j = 0; j < g->num_vertices; j++) {
            printf("%3d ", g->adj[i][j]);
        }
        printf("\n");
    }
}

// 获取顶点 u 的所有邻居
void graph_mat_neighbors(const GraphMatrix *g, int u) {
    printf("顶点 %d 的邻居: ", u);
    for (int v = 0; v < g->num_vertices; v++) {
        if (g->adj[u][v] != 0) {
            printf("%d(权:%d) ", v, g->adj[u][v]);
        }
    }
    printf("\n");
}
```

**优缺点：**
- 优点：判断两顶点是否有边 O(1)，适合**稠密图**
- 缺点：空间 O(V^2)，对于**稀疏图**浪费严重

## 三、邻接表表示

### 3.1 结构体定义

```c
// 边节点
typedef struct EdgeNode {
    int dest;                // 目标顶点
    int weight;              // 边权
    struct EdgeNode *next;   // 下一条边
} EdgeNode;

// 顶点
typedef struct {
    char      label[32];
    EdgeNode *first_edge;    // 该顶点的第一条边
} Vertex;

// 图
typedef struct {
    int     num_vertices;
    Vertex  vertices[MAX_VERTICES];
} GraphList;
```

### 3.2 操作实现

```c
void graph_list_init(GraphList *g, int vertices) {
    g->num_vertices = vertices;
    for (int i = 0; i < vertices; i++) {
        g->vertices[i].first_edge = NULL;
        sprintf(g->vertices[i].label, "V%d", i);
    }
}

// 添加无向边
void graph_list_add_edge(GraphList *g, int u, int v, int weight) {
    // u -> v
    EdgeNode *e1 = malloc(sizeof(EdgeNode));
    e1->dest = v;
    e1->weight = weight;
    e1->next = g->vertices[u].first_edge;
    g->vertices[u].first_edge = e1;

    // v -> u（无向图）
    EdgeNode *e2 = malloc(sizeof(EdgeNode));
    e2->dest = u;
    e2->weight = weight;
    e2->next = g->vertices[v].first_edge;
    g->vertices[v].first_edge = e2;
}

// 打印邻接表
void graph_list_print(const GraphList *g) {
    for (int i = 0; i < g->num_vertices; i++) {
        printf("%s -> ", g->vertices[i].label);
        EdgeNode *e = g->vertices[i].first_edge;
        while (e != NULL) {
            printf("%s(权:%d) ", g->vertices[e->dest].label, e->weight);
            e = e->next;
        }
        printf("\n");
    }
}

// 释放邻接表
void graph_list_free(GraphList *g) {
    for (int i = 0; i < g->num_vertices; i++) {
        EdgeNode *e = g->vertices[i].first_edge;
        while (e != NULL) {
            EdgeNode *temp = e;
            e = e->next;
            free(temp);
        }
    }
}
```

## 四、图的遍历

### 4.1 DFS（深度优先搜索）

```c
void dfs(const GraphList *g, int v, int visited[]) {
    visited[v] = 1;
    printf("%s ", g->vertices[v].label);

    EdgeNode *e = g->vertices[v].first_edge;
    while (e != NULL) {
        if (!visited[e->dest]) {
            dfs(g, e->dest, visited);
        }
        e = e->next;
    }
}

void graph_dfs(const GraphList *g, int start) {
    int visited[MAX_VERTICES] = {0};
    printf("DFS 从 %s: ", g->vertices[start].label);
    dfs(g, start, visited);
    printf("\n");
}
```

### 4.2 BFS（广度优先搜索）

```c
void graph_bfs(const GraphList *g, int start) {
    int visited[MAX_VERTICES] = {0};
    int queue[MAX_VERTICES];
    int front = 0, rear = 0;

    visited[start] = 1;
    queue[rear++] = start;

    printf("BFS 从 %s: ", g->vertices[start].label);

    while (front < rear) {
        int v = queue[front++];
        printf("%s ", g->vertices[v].label);

        EdgeNode *e = g->vertices[v].first_edge;
        while (e != NULL) {
            if (!visited[e->dest]) {
                visited[e->dest] = 1;
                queue[rear++] = e->dest;
            }
            e = e->next;
        }
    }
    printf("\n");
}
```

## 五、完整示例

```c
int main(void) {
    GraphList g;
    graph_list_init(&g, 5);

    graph_list_add_edge(&g, 0, 1, 2);
    graph_list_add_edge(&g, 0, 2, 4);
    graph_list_add_edge(&g, 1, 3, 1);
    graph_list_add_edge(&g, 2, 3, 3);
    graph_list_add_edge(&g, 3, 4, 5);

    graph_list_print(&g);
    graph_dfs(&g, 0);
    graph_bfs(&g, 0);

    graph_list_free(&g);
    return 0;
}
```

## 六、关键要点

> **重要提示：**
>
> 1. 邻接矩阵适合稠密图，邻接表适合稀疏图。
> 2. 邻接表使用链表存储边，动态分配需要及时释放。
> 3. DFS 使用递归（或栈），BFS 使用队列。
> 4. 有向图和无向图的区别在于边是否对称添加。
> 5. 带权图的边存储权重信息，用于最短路径等算法。
> 6. 访问标记数组是防止重复访问的关键。
