# 06 - 结构体指针

## 一、指向结构体的指针

结构体指针是存储结构体变量**地址**的指针变量，是 C 语言中操作结构体最高效的方式。

```c
struct Student {
    int id;
    char name[32];
    float score;
};

struct Student s1 = {1001, "张三", 92.5};
struct Student *p = &s1;  // p 指向 s1
```

## 二、通过指针访问成员

### 2.1 `->` 操作符（推荐）

```c
struct Student *p = &s1;

// 读取
printf("学号: %d\n", p->id);
printf("姓名: %s\n", p->name);
printf("成绩: %.1f\n", p->score);

// 修改
p->score = 98.0;
p->id = 1002;
strcpy(p->name, "李四");
```

### 2.2 `(*p).member` 方式

```c
printf("学号: %d\n", (*p).id);
(*p).score = 98.0;
```

> **注意**：`*p.member` 是错误的，因为 `.` 优先级高于 `*`，必须写成 `(*p).member`。

### 2.3 两种方式的等价关系

```
p->member    等价于    (*p).member
```

## 三、结构体指针作为函数参数

这是结构体最常用的传参方式，原因有二：**高效**（只传递一个地址，通常 4/8 字节）和**可以修改原数据**。

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    int id;
    char name[32];
    float score;
} Student;

// 通过指针修改结构体
void set_score(Student *stu, float new_score) {
    if (stu == NULL) return;  // 防御性检查
    stu->score = new_score;
}

// 通过指针读取结构体（const 保证不修改）
void print_student(const Student *stu) {
    if (stu == NULL) return;
    printf("学号: %d, 姓名: %s, 成绩: %.1f\n",
           stu->id, stu->name, stu->score);
}

int main(void) {
    Student s1 = {1001, "张三", 92.5};

    print_student(&s1);       // 传递地址
    set_score(&s1, 98.0);     // 修改成绩
    print_student(&s1);       // 查看修改结果

    return 0;
}
```

### 值传递 vs 指针传递

```c
// 值传递：创建副本，大结构体时效率低
void func_by_value(Student s);
// 调用：func_by_value(s1);

// 指针传递：只传递地址（4/8 字节），高效
void func_by_pointer(const Student *s);
// 调用：func_by_pointer(&s1);
```

| 特性 | 值传递 | 指针传递 |
|------|--------|----------|
| 参数大小 | 结构体大小 | 4 或 8 字节 |
| 能否修改原数据 | 不能 | 能（去掉 const） |
| 大结构体效率 | 低（需拷贝） | 高 |
| 推荐场景 | 小结构体 | 大结构体 / 需要修改 |

## 四、结构体指针数组

```c
Student students[] = {
    {1001, "张三", 92.5},
    {1002, "李四", 88.0},
    {1003, "王五", 95.5}
};

// 指针数组：每个元素都是指向 Student 的指针
Student *ptrs[3];
for (int i = 0; i < 3; i++) {
    ptrs[i] = &students[i];
}

// 通过指针数组访问
for (int i = 0; i < 3; i++) {
    printf("%s: %.1f\n", ptrs[i]->name, ptrs[i]->score);
}
```

**用途**：可以在不移动结构体数据的情况下，对指针数组排序，实现间接排序。

## 五、动态分配结构体

```c
#include <stdlib.h>

// 动态分配单个结构体
Student *p = malloc(sizeof(Student));
if (p != NULL) {
    p->id = 1001;
    strcpy(p->name, "动态");
    p->score = 90.0;
    // ... 使用 ...
    free(p);
}

// 动态分配结构体数组
int n = 10;
Student *arr = malloc(n * sizeof(Student));
if (arr != NULL) {
    for (int i = 0; i < n; i++) {
        arr[i].id = 1000 + i;
    }
    free(arr);
}
```

## 六、指向结构体数组的指针

```c
Student class[] = {
    {1001, "张三", 92.5},
    {1002, "李四", 88.0},
    {1003, "王五", 95.5}
};

// p 指向数组第一个元素
Student *p = class;

// 指针算术运算
printf("%s\n", p->name);       // 张三（class[0]）
printf("%s\n", (p + 1)->name); // 李四（class[1]）
printf("%s\n", (p + 2)->name); // 王五（class[2]）

// 指针遍历
for (Student *p = class; p < class + 3; p++) {
    printf("%s: %.1f\n", p->name, p->score);
}
```

## 七、const 与结构体指针

```c
// 指向常量的指针：不能通过指针修改结构体
const Student *p1 = &s1;
// p1->score = 99.0;  // 编译错误
printf("%f\n", p1->score);  // 可以读取

// 常量指针：指针本身不能指向其他地址
Student *const p2 = &s1;
p2->score = 99.0;   // 可以修改
// p2 = &s2;         // 编译错误

// 指向常量的常量指针：都不能改
const Student *const p3 = &s1;
// p3->score = 99.0; // 编译错误
// p3 = &s2;         // 编译错误
```

## 八、关键要点

> **重要提示：**
>
> 1. `->` 是访问结构体指针成员最简洁的方式。
> 2. 传递结构体给函数时，优先使用**指针传递**（大结构体效率高）。
> 3. 不需要修改结构体的函数参数应加 `const` 限定。
> 4. 使用动态分配的结构体指针前，务必检查是否为 `NULL`。
> 5. 结构体指针可以进行算术运算（指向数组时）。
> 6. 链表、树等数据结构完全依赖结构体指针实现。
