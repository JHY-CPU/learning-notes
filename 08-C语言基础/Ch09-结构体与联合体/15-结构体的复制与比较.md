# 15 - 结构体的复制与比较

## 一、结构体的赋值复制

C 语言允许同类型结构体之间进行**整体赋值**，效果等同于逐字节浅拷贝。

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    int   id;
    char  name[32];
    float score;
} Student;

int main(void) {
    Student s1 = {1001, "张三", 92.5};
    Student s2;

    s2 = s1;  // 整体赋值 —— 浅拷贝

    // s2 是 s1 的完整副本
    printf("s2: %d, %s, %.1f\n", s2.id, s2.name, s2.score);

    // 修改 s2 不影响 s1
    s2.id = 1002;
    strcpy(s2.name, "李四");
    printf("s1: %d, %s, %.1f\n", s1.id, s1.name, s1.score);  // 不变

    return 0;
}
```

## 二、使用 memcpy 复制

```c
#include <string.h>

Student s1 = {1001, "张三", 92.5};
Student s2;

memcpy(&s2, &s1, sizeof(Student));
// 效果等同于 s2 = s1
```

### memcpy vs 赋值

| 特性 | `s2 = s1` | `memcpy(&s2, &s1, sizeof(Student))` |
|------|-----------|-------------------------------------|
| 效果 | 浅拷贝 | 浅拷贝 |
| 可读性 | 高 | 中 |
| 性能 | 编译器可能优化 | 通常更底层 |
| 包含指针时 | 浅拷贝指针值 | 浅拷贝指针值 |
| 推荐场景 | 一般情况 | 需要拷贝部分成员或批量操作 |

## 三、浅拷贝的问题

当结构体包含**指针成员**时，赋值复制只拷贝指针值（地址），不会拷贝指针指向的数据。

```c
typedef struct {
    int   id;
    char *name;    // 指针成员！
    float score;
} Student;

void shallow_copy_problem(void) {
    Student s1;
    s1.id = 1001;
    s1.name = malloc(32);
    strcpy(s1.name, "张三");
    s1.score = 92.5;

    Student s2 = s1;  // 浅拷贝

    printf("s1.name = %p\n", (void *)s1.name);
    printf("s2.name = %p\n", (void *)s2.name);
    // 两者地址相同！指向同一块内存

    free(s1.name);
    // s2.name 现在是悬垂指针！访问它会导致未定义行为
}
```

### 解决方案：深拷贝

```c
Student deep_copy(const Student *src) {
    Student dst;
    dst.id = src->id;
    dst.score = src->score;

    if (src->name != NULL) {
        size_t len = strlen(src->name) + 1;
        dst.name = malloc(len);
        if (dst.name != NULL) {
            memcpy(dst.name, src->name, len);
        }
    } else {
        dst.name = NULL;
    }

    return dst;
}

void deep_free(Student *s) {
    if (s != NULL) {
        free(s->name);
        s->name = NULL;
    }
}
```

## 四、结构体的比较

### 4.1 逐成员比较

C 语言**不支持**结构体的直接比较（不能用 `==`），必须逐成员比较。

```c
int student_equal(const Student *a, const Student *b) {
    return a->id == b->id &&
           a->score == b->score &&
           strcmp(a->name, b->name) == 0;  // 字符串用 strcmp
}

// 使用
if (student_equal(&s1, &s2)) {
    printf("两个学生信息相同\n");
}
```

### 4.2 为什么不能用 `==` 比较

```c
Student s1 = {1001, "张三", 92.5};
Student s2 = {1001, "张三", 92.5};

// if (s1 == s2)  // 编译错误！C 语言不支持结构体直接比较
```

原因：结构体可能包含**填充字节**，这些填充字节的内容是未定义的，直接按字节比较可能得到错误结果。

### 4.3 使用 memcmp 的风险

```c
#include <string.h>

Student s1 = {1001, "张三", 92.5};
Student s2 = {1001, "张三", 92.5};

// memcmp 可能返回非零！
// 因为 s1 和 s2 的填充字节内容可能不同
if (memcmp(&s1, &s2, sizeof(Student)) == 0) {
    // 不一定相等！填充字节不确定
}
```

**解决方法**：先用 `memset` 清零，再赋值。

```c
Student s1, s2;
memset(&s1, 0, sizeof(s1));
memset(&s2, 0, sizeof(s2));

s1.id = 1001;
s2.id = 1001;
// ... 逐成员赋值 ...

// 现在 memcmp 是安全的
if (memcmp(&s1, &s2, sizeof(Student)) == 0) {
    // 确实相等
}
```

## 五、完整的比较函数实现

```c
#include <string.h>
#include <math.h>

typedef struct {
    double x;
    double y;
} Point;

// 浮点数比较需要容差
int point_equal(const Point *a, const Point *b, double epsilon) {
    return fabs(a->x - b->x) < epsilon &&
           fabs(a->y - b->y) < epsilon;
}

// 返回 -1, 0, 1（可用于排序）
int point_compare(const Point *a, const Point *b) {
    if (a->x != b->x) return (a->x < b->x) ? -1 : 1;
    if (a->y != b->y) return (a->y < b->y) ? -1 : 1;
    return 0;
}
```

## 六、内存设置与初始化

```c
// 使用 memset 清零
Student s;
memset(&s, 0, sizeof(s));

// 使用 memset 填充特定值
memset(&s, 0xFF, sizeof(s));  // 所有字节设为 0xFF

// 注意：memset 按字节填充，不适合用非 0 值初始化整数
int arr[10];
memset(arr, 1, sizeof(arr));  // 每个 int 不是 1，而是 0x01010101 = 16843009
```

## 七、完整示例

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct {
    int    id;
    char   name[32];
    double scores[3];
    int    flags;
} Record;

void record_init_zero(Record *r) {
    memset(r, 0, sizeof(Record));
}

int record_equal(const Record *a, const Record *b) {
    if (a->id != b->id || a->flags != b->flags) return 0;
    if (strcmp(a->name, b->name) != 0) return 0;
    for (int i = 0; i < 3; i++) {
        if (a->scores[i] != b->scores[i]) return 0;
    }
    return 1;
}

Record record_copy(const Record *src) {
    Record dst;
    memcpy(&dst, src, sizeof(Record));  // 全部是值类型，memcpy 安全
    return dst;
}

int main(void) {
    Record r1;
    record_init_zero(&r1);
    r1.id = 1;
    strcpy(r1.name, "测试");
    r1.scores[0] = 90.0;

    Record r2 = record_copy(&r1);
    printf("相等: %s\n", record_equal(&r1, &r2) ? "是" : "否");

    r2.scores[0] = 85.0;
    printf("修改后相等: %s\n", record_equal(&r1, &r2) ? "是" : "否");

    return 0;
}
```

## 八、关键要点

> **重要提示：**
>
> 1. 结构体赋值是**浅拷贝**，包含指针成员时需要手动实现**深拷贝**。
> 2. C 语言**不支持**结构体的 `==` 比较，必须逐成员比较。
> 3. `memcmp` 用于结构体比较时，必须先 `memset` 清零以消除填充字节的影响。
> 4. 浮点数成员比较需要使用**容差**（epsilon），而非直接 `==`。
> 5. 字符串成员比较必须使用 `strcmp`，不能用 `==`。
> 6. 全部是值类型的结构体，`memcpy` 和赋值都是安全的。
