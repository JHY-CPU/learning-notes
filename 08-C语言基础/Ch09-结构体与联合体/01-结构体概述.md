# 01 - 结构体概述

## 一、什么是结构体

**结构体（Structure）** 是 C 语言中一种用户自定义的复合数据类型，它允许我们将**不同类型的数据**组合在一起，形成一个逻辑上的整体。

```c
// 结构体：描述一个学生的信息
struct Student {
    int id;            // 学号
    char name[32];     // 姓名
    float score;       // 成绩
    char gender;       // 性别
};
```

结构体中的每个变量称为**成员（Member）** 或**字段（Field）**，每个成员可以是不同的数据类型。

## 二、为什么需要结构体

### 2.1 现实世界的数据是复合的

现实中的一个"实体"往往包含多种属性。例如一个学生：

| 属性 | 数据类型 | 示例 |
|------|----------|------|
| 学号 | int | 1001 |
| 姓名 | 字符串 | "张三" |
| 成绩 | float | 92.5 |
| 性别 | char | 'M' |

如果没有结构体，我们需要用多个独立的变量来表示：

```c
// 不使用结构体 —— 散乱且难以管理
int stu_id = 1001;
char stu_name[32] = "张三";
float stu_score = 92.5;
char stu_gender = 'M';

// 如果有 100 个学生呢？需要 400 个变量！
int stu_id2 = 1002;
char stu_name2[32] = "李四";
// ... 这种方式完全不可行
```

### 2.2 结构体的优势

```c
// 使用结构体 —— 整洁、可扩展
struct Student {
    int id;
    char name[32];
    float score;
    char gender;
};

struct Student students[100];  // 轻松管理 100 个学生
```

**核心优势总结：**

1. **数据聚合**：将相关数据组织在一起，语义清晰
2. **代码复用**：定义一次，多次使用
3. **函数传参**：可以将一组相关数据作为整体传递
4. **数据建模**：能够精确地对现实世界实体建模

## 三、结构体与数组的对比

| 特性 | 数组 | 结构体 |
|------|------|--------|
| 元素类型 | 必须**相同** | 可以**不同** |
| 元素访问 | 通过下标 `[index]` | 通过成员名 `.member` |
| 内存布局 | 连续、等大元素 | 连续（可能有填充） |
| 主要用途 | 存储同类型集合 | 描述复合实体 |
| 大小计算 | `n * sizeof(element)` | 各成员大小之和（含对齐） |

```c
// 数组：存储同类型数据的集合
int scores[5] = {90, 85, 78, 92, 88};

// 结构体：存储不同类型数据的组合
struct Student {
    int id;
    char name[32];
    float score;
};
```

### 3.1 结构体与数组可以结合使用

```c
// 结构体数组：最常用的组合方式
struct Student class[50];

// 访问第 3 个学生的姓名
printf("姓名: %s\n", class[2].name);

// 访问第 i 个学生的成绩
class[i].score = 95.0;
```

### 3.2 结构体中也可以包含数组

```c
struct Student {
    int id;
    char name[32];      // 字符数组存储姓名
    float scores[5];    // 数组存储 5 门课成绩
};
```

## 四、结构体的本质理解

从内存角度看，结构体是一块**连续的内存区域**（可能因对齐而有填充字节），其中按顺序存放着各个成员。

```
struct Student 在内存中的示意（假设 4 字节对齐）：

地址偏移:  0     4      36      40    44
         +-----+-------+--------+-----+
内容:     | id  | name  | score  |gender|
         +-----+-------+--------+-----+
类型:     int   char[32] float   char
大小:     4B    32B     4B      1B(+3B填充)
```

## 五、关键要点

> **重要提示：**
>
> 1. 结构体是一种**类型定义**，不是变量声明。定义后需要声明变量才能使用。
> 2. 结构体成员按定义顺序存储在内存中（可能有对齐填充）。
> 3. 结构体的大小一般**大于等于**各成员大小之和（因为内存对齐）。
> 4. 结构体可以嵌套定义，也可以包含指向自身的指针（链表等数据结构的基础）。
> 5. C 语言的结构体不支持成员默认值、不支持成员函数（与 C++ 不同）。
> 6. 结构体是 C 语言实现**面向对象编程思想**的基础工具。

## 六、简单示例

```c
#include <stdio.h>

struct Student {
    int id;
    char name[32];
    float score;
};

int main(void) {
    // 声明并初始化一个结构体变量
    struct Student s1 = {1001, "张三", 92.5};

    // 访问并输出成员
    printf("学号: %d\n", s1.id);
    printf("姓名: %s\n", s1.name);
    printf("成绩: %.1f\n", s1.score);

    return 0;
}
```

输出结果：
```
学号: 1001
姓名: 张三
成绩: 92.5
```
