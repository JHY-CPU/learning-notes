# 04 - 结构体的初始化

## 一、顺序初始化

按照成员定义的顺序，依次提供初始值。这是最基础的初始化方式。

```c
struct Point {
    int x;
    int y;
    int z;
};

// 按照 x, y, z 的顺序提供值
struct Point p1 = {10, 20, 30};
```

### 1.1 部分初始化

提供的初始值少于成员数量时，剩余成员自动初始化为 **0**（对于数值类型）或 **'\0'**（对于字符类型）。

```c
struct Student {
    int id;
    char name[32];
    float score;
    int age;
};

// 只初始化前两个成员，其余为 0
struct Student s1 = {1001, "张三"};
// 实际效果：s1 = {1001, "张三", 0.0, 0}

// 只初始化第一个成员
struct Student s2 = {1002};
// 实际效果：s2 = {1002, "", 0.0, 0}
```

### 1.2 全零初始化

```c
struct Student s1 = {0};     // 所有成员初始化为零
struct Student s2 = {};      // C23 标准支持，GCC 也接受（作为扩展）
```

> **注意**：`{0}` 是 C 语言中唯一可移植的"将所有成员清零"写法。

## 二、指定初始化器（C99）

C99 标准引入了**指定初始化器（Designated Initializer）**，允许按成员名初始化，无需遵循定义顺序。

### 2.1 基本语法

```c
struct Student s1 = {
    .name = "李四",
    .id = 1002,
    .score = 88.5
};
// 顺序可以任意！
```

### 2.2 与顺序初始化混合使用

```c
// 混合使用：前两个按顺序，后面用指定初始化
struct Student s2 = {
    1003,            // id（按顺序）
    "王五",          // name（按顺序）
    .age = 20        // age（指定）
};
```

### 2.3 数组成员的指定初始化

```c
struct Classroom {
    int id;
    int students[5];
};

// 初始化数组成员
struct Classroom c1 = {
    .id = 1,
    .students = {80, 85, 90, 95, 100}
};

// 指定数组下标初始化（C99）
struct Classroom c2 = {
    .id = 2,
    .students = {[0] = 70, [2] = 85, [4] = 95}
    // students[1] 和 students[3] 为 0
};
```

### 2.4 嵌套结构体的指定初始化

```c
struct Address {
    char city[32];
    int zipcode;
};

struct Person {
    char name[32];
    int age;
    struct Address addr;
};

// 嵌套指定初始化
struct Person p1 = {
    .name = "赵六",
    .age = 25,
    .addr = {
        .city = "上海",
        .zipcode = 200000
    }
};
```

## 三、初始化的内存行为

```c
struct Student s1 = {1001, "张三", 92.5};
```

初始化时，编译器在栈上（如果是局部变量）分配内存，然后按照成员顺序写入初始值。

```
内存布局示意：

s1.id:    [01 00 00 00]        = 1001
s1.name:  [张三 \0 \0 ...]     = "张三"
s1.score: [00 00 B8 42]        = 92.5f
```

## 四、不同场景的初始化方式对比

```c
// 场景 1：全局变量（存储在 .data 或 .bss 段）
struct Student global_s = {1001, "全局", 90.0};

void example(void) {
    // 场景 2：局部变量（存储在栈上）
    struct Student local_s = {1002, "局部", 85.0};

    // 场景 3：静态局部变量（存储在 .data 段）
    static struct Student static_s = {1003, "静态", 80.0};

    // 场景 4：动态分配（存储在堆上）
    struct Student *heap_s = malloc(sizeof(struct Student));
    if (heap_s != NULL) {
        // 动态分配后只能逐成员赋值，不能用初始化器
        heap_s->id = 1004;
        strcpy(heap_s->name, "堆上");
        heap_s->score = 75.0;
        free(heap_s);
    }
}
```

## 五、使用 memset 清零

```c
#include <string.h>

struct Student s1;
memset(&s1, 0, sizeof(s1));  // 将整个结构体清零
```

**与 `{0}` 初始化的对比：**

| 方式 | 适用场景 | 注意事项 |
|------|----------|----------|
| `= {0}` | 声明时初始化 | 简洁、编译时检查 |
| `memset` | 运行时清零 | 可用于已声明的变量 |
| `= {}` | C23 标准 | 非标准编译器可能不支持 |

## 六、常见初始化错误

```c
// 错误 1：试图在声明后用初始化语法赋值
struct Student s1;
// s1 = {1001, "张三", 92.5};  // 编译错误！初始化语法只能在声明时使用

// 正确做法：
s1.id = 1001;
strcpy(s1.name, "张三");
s1.score = 92.5;

// 或者用复合字面量（C99）
s1 = (struct Student){1001, "张三", 92.5};  // C99 正确

// 错误 2：初始值数量超过成员数量
// struct Student s2 = {1, "A", 90.0, 20, 100};  // 编译器可能报错

// 错误 3：类型不匹配
// struct Student s3 = {"不是数字", "张三", 92.5};  // id 应该是 int
```

## 七、复合字面量（C99）

复合字面量允许在**任何表达式位置**创建匿名结构体值。

```c
struct Point make_point(int x, int y) {
    return (struct Point){x, y};  // 返回匿名结构体
}

void print_student(struct Student s);

// 直接传递匿名结构体
print_student((struct Student){1001, "张三", 92.5});

// 用于赋值
struct Student s1;
s1 = (struct Student){.id = 1002, .name = "李四", .score = 88.0};
```

## 八、完整示例

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    int hour;
    int minute;
    int second;
} Time;

typedef struct {
    char title[64];
    Time start_time;
    int duration;  // 分钟
} Meeting;

int main(void) {
    // 顺序初始化
    Meeting m1 = {"项目评审", {9, 30, 0}, 60};

    // 指定初始化
    Meeting m2 = {
        .title = "技术分享",
        .start_time = {.hour = 14, .minute = 0, .second = 0},
        .duration = 90
    };

    // 部分初始化
    Meeting m3 = {"站会"};
    // m3.start_time = {0,0,0}, m3.duration = 0

    // 复合字面量
    Meeting m4 = (Meeting){"午餐会", {12, 0, 0}, 45};

    printf("会议: %s, 开始: %02d:%02d, 时长: %d分钟\n",
           m1.title, m1.start_time.hour,
           m1.start_time.minute, m1.duration);
    printf("会议: %s, 开始: %02d:%02d, 时长: %d分钟\n",
           m2.title, m2.start_time.hour,
           m2.start_time.minute, m2.duration);
    printf("会议: %s, 开始: %02d:%02d, 时长: %d分钟\n",
           m4.title, m4.start_time.hour,
           m4.start_time.minute, m4.duration);

    return 0;
}
```

## 九、关键要点

> **重要提示：**
>
> 1. `{0}` 是将结构体所有成员清零的可移植写法。
> 2. C99 的指定初始化器（`.member = value`）大幅提高可读性。
> 3. 初始化语法**只能在声明时**使用，声明后需逐成员赋值或用复合字面量。
> 4. 部分初始化时，未指定的成员自动为零。
> 5. 复合字面量 `(struct Type){...}` 允许在表达式中创建结构体值。
> 6. `memset(&s, 0, sizeof(s))` 是运行时清零的常用方法。
> 7. 动态分配的结构体不能使用初始化器，必须逐成员赋值。
