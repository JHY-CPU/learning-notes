# 16 - 结构体与链表

## 一、单链表节点定义

链表是最基础的数据结构之一，完全依赖结构体指针实现。

```c
#include <stdio.h>
#include <stdlib.h>

// 单链表节点
typedef struct Node {
    int data;            // 数据域
    struct Node *next;   // 指针域：指向下一个节点
} Node;
```

**关键点**：结构体中包含指向**自身类型**的指针，这是允许的（但不能包含自身类型的实例）。

## 二、单链表基本操作

### 2.1 创建节点

```c
Node *create_node(int value) {
    Node *new_node = malloc(sizeof(Node));
    if (new_node == NULL) {
        perror("malloc");
        return NULL;
    }
    new_node->data = value;
    new_node->next = NULL;
    return new_node;
}
```

### 2.2 头插法

```c
// 在链表头部插入节点，返回新的头节点
Node *insert_head(Node *head, int value) {
    Node *new_node = create_node(value);
    if (new_node == NULL) return head;

    new_node->next = head;
    return new_node;  // 新节点成为新的头
}
```

### 2.3 尾插法

```c
Node *insert_tail(Node *head, int value) {
    Node *new_node = create_node(value);
    if (new_node == NULL) return head;

    if (head == NULL) {
        return new_node;
    }

    // 找到尾节点
    Node *curr = head;
    while (curr->next != NULL) {
        curr = curr->next;
    }
    curr->next = new_node;
    return head;
}
```

### 2.4 删除节点

```c
Node *delete_node(Node *head, int value) {
    if (head == NULL) return NULL;

    // 删除的是头节点
    if (head->data == value) {
        Node *temp = head->next;
        free(head);
        return temp;
    }

    // 查找要删除节点的前驱
    Node *curr = head;
    while (curr->next != NULL && curr->next->data != value) {
        curr = curr->next;
    }

    if (curr->next != NULL) {
        Node *temp = curr->next;
        curr->next = temp->next;
        free(temp);
    }

    return head;
}
```

### 2.5 遍历与打印

```c
void print_list(const Node *head) {
    const Node *curr = head;
    while (curr != NULL) {
        printf("%d -> ", curr->data);
        curr = curr->next;
    }
    printf("NULL\n");
}

void free_list(Node *head) {
    while (head != NULL) {
        Node *temp = head;
        head = head->next;
        free(temp);
    }
}
```

## 三、双向链表

```c
// 双向链表节点
typedef struct DNode {
    int data;
    struct DNode *prev;  // 指向前驱
    struct DNode *next;  // 指向后继
} DNode;

typedef struct {
    DNode *head;
    DNode *tail;
    int    size;
} DLinkedList;
```

### 3.1 双向链表插入

```c
// 在指定节点后插入
void dlist_insert_after(DLinkedList *list, DNode *pos, int value) {
    DNode *new_node = malloc(sizeof(DNode));
    if (new_node == NULL) return;

    new_node->data = value;
    new_node->prev = pos;
    new_node->next = pos->next;

    if (pos->next != NULL) {
        pos->next->prev = new_node;
    } else {
        list->tail = new_node;  // 插入在末尾，更新尾指针
    }
    pos->next = new_node;
    list->size++;
}

// 删除指定节点
void dlist_remove(DLinkedList *list, DNode *node) {
    if (node->prev != NULL) {
        node->prev->next = node->next;
    } else {
        list->head = node->next;  // 删除的是头节点
    }

    if (node->next != NULL) {
        node->next->prev = node->prev;
    } else {
        list->tail = node->prev;  // 删除的是尾节点
    }

    free(node);
    list->size--;
}
```

## 四、带头节点的链表

使用**哨兵节点（Sentinel/Dummy Node）** 可以简化边界条件处理。

```c
typedef struct {
    Node *sentinel;  // 哨兵节点，不存储数据
    int   size;
} LinkedList;

LinkedList *list_create(void) {
    LinkedList *list = malloc(sizeof(LinkedList));
    if (list == NULL) return NULL;

    list->sentinel = malloc(sizeof(Node));
    if (list->sentinel == NULL) {
        free(list);
        return NULL;
    }
    list->sentinel->next = NULL;
    list->size = 0;
    return list;
}

void list_push_front(LinkedList *list, int value) {
    Node *new_node = create_node(value);
    new_node->next = list->sentinel->next;
    list->sentinel->next = new_node;
    list->size++;
}
```

## 五、完整示例

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;

Node *create_node(int value) {
    Node *n = malloc(sizeof(Node));
    if (n) { n->data = value; n->next = NULL; }
    return n;
}

Node *insert_head(Node *head, int value) {
    Node *n = create_node(value);
    if (!n) return head;
    n->next = head;
    return n;
}

void print_list(const Node *head) {
    for (const Node *p = head; p; p = p->next)
        printf("%d -> ", p->data);
    printf("NULL\n");
}

int list_length(const Node *head) {
    int len = 0;
    for (const Node *p = head; p; p = p->next) len++;
    return len;
}

void free_list(Node *head) {
    while (head) {
        Node *t = head;
        head = head->next;
        free(t);
    }
}

int main(void) {
    Node *list = NULL;
    for (int i = 1; i <= 5; i++) {
        list = insert_head(list, i * 10);
    }
    print_list(list);
    printf("长度: %d\n", list_length(list));
    free_list(list);
    return 0;
}
```

输出：
```
50 -> 40 -> 30 -> 20 -> 10 -> NULL
长度: 5
```

## 六、关键要点

> **重要提示：**
>
> 1. 链表节点通过结构体中指向自身类型的指针实现。
> 2. 插入和删除操作需要仔细处理指针的修改顺序。
> 3. 头节点变化时，函数需要返回新的头节点（或使用二级指针）。
> 4. 动态分配的节点必须在不再使用时 `free`，避免内存泄漏。
> 5. 双向链表增加了 `prev` 指针，方便反向遍历但增加维护成本。
> 6. 哨兵节点简化了边界条件的处理。
