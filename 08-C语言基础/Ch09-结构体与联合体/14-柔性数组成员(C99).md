# 14 - 柔性数组成员（C99）

## 一、柔性数组的基本概念

**柔性数组成员（Flexible Array Member）** 是 C99 标准引入的特性，允许结构体的最后一个成员是**不指定大小的数组**。

```c
struct FlexArray {
    int    count;
    int    capacity;
    double data[];  // 柔性数组成员 —— 没有指定大小
};
```

**核心特点：**
- 必须是结构体的**最后一个**成员
- 必须**至少有一个**其他成员
- `sizeof(struct FlexArray)` **不包含**柔性数组的大小
- 不能直接声明变量（因为大小不确定）

## 二、为什么需要柔性数组

### 问题：变长数据存储

假设要存储一批学生数据，数量在运行时确定：

```c
// 方案 1：固定大小数组（浪费或不够用）
struct StudentList1 {
    int     count;
    Student students[100];  // 如果只有 3 个学生浪费空间，超过 100 个又装不下
};

// 方案 2：指针 + 动态分配（需要两次 malloc）
struct StudentList2 {
    int       count;
    Student  *students;  // 需要单独 malloc(students)
};

// 方案 3：柔性数组（一次 malloc，内存连续）
struct StudentList3 {
    int     count;
    Student students[];  // 柔性数组
};
```

### 方案对比

| 方案 | 内存分配次数 | 内存连续性 | 释放复杂度 |
|------|------------|-----------|-----------|
| 固定数组 | 0 | 连续 | 简单 |
| 指针方案 | 2 | 不连续 | 需要先 free 成员再 free 结构体 |
| 柔性数组 | 1 | 连续 | 一次 free 即可 |

## 三、柔性数组的使用方法

### 3.1 内存分配

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct {
    char  name[32];
    float score;
} Student;

typedef struct {
    int     count;
    Student students[];  // 柔性数组
} StudentList;

int main(void) {
    int n = 5;

    // 一次性分配：结构体头部 + 柔性数组空间
    StudentList *list = malloc(sizeof(StudentList) + n * sizeof(Student));
    if (list == NULL) {
        perror("malloc");
        return 1;
    }

    list->count = n;

    // 使用柔性数组，就像普通数组一样
    strcpy(list->students[0].name, "张三");
    list->students[0].score = 92.5;
    strcpy(list->students[1].name, "李四");
    list->students[1].score = 88.0;

    for (int i = 0; i < list->count; i++) {
        printf("%s: %.1f\n", list->students[i].name,
               list->students[i].score);
    }

    free(list);  // 只需一次 free
    return 0;
}
```

### 3.2 动态扩容

```c
StudentList *list_resize(StudentList *old, int new_count) {
    size_t new_size = sizeof(StudentList) + new_count * sizeof(Student);
    StudentList *new_list = realloc(old, new_size);
    if (new_list == NULL) {
        return old;  // 扩容失败，返回原指针
    }
    new_list->count = new_count;
    return new_list;
}

// 使用
StudentList *list = malloc(sizeof(StudentList) + 3 * sizeof(Student));
list->count = 3;
// ... 使用 ...

// 扩容到 10 个元素
list = list_resize(list, 10);
if (list != NULL) {
    // 继续使用 ...
}
```

## 四、柔性数组的常见使用模式

### 4.1 动态字符串

```c
typedef struct {
    size_t length;
    size_t capacity;
    char   data[];  // 柔性数组存储字符串
} DynString;

DynString *dstr_new(const char *init) {
    size_t len = strlen(init);
    size_t cap = len + 1;
    DynString *ds = malloc(sizeof(DynString) + cap);
    if (ds == NULL) return NULL;

    ds->length = len;
    ds->capacity = cap;
    memcpy(ds->data, init, len + 1);
    return ds;
}

void dstr_free(DynString *ds) {
    free(ds);
}
```

### 4.2 变长消息包

```c
#pragma pack(push, 1)
typedef struct {
    uint8_t  type;
    uint16_t length;
    uint8_t  payload[];  // 变长载荷
} Message;
#pragma pack(pop)

// 创建消息
Message *msg_create(uint8_t type, const void *data, uint16_t len) {
    Message *msg = malloc(sizeof(Message) + len);
    if (msg == NULL) return NULL;

    msg->type = type;
    msg->length = len;
    if (data != NULL && len > 0) {
        memcpy(msg->payload, data, len);
    }
    return msg;
}
```

### 4.3 动态数组

```c
typedef struct {
    size_t size;
    size_t capacity;
    int    data[];
} IntArray;

IntArray *arr_create(size_t initial_cap) {
    IntArray *arr = malloc(sizeof(IntArray) + initial_cap * sizeof(int));
    if (arr == NULL) return NULL;
    arr->size = 0;
    arr->capacity = initial_cap;
    return arr;
}

IntArray *arr_push(IntArray *arr, int value) {
    if (arr->size >= arr->capacity) {
        size_t new_cap = arr->capacity * 2;
        IntArray *new_arr = realloc(arr,
            sizeof(IntArray) + new_cap * sizeof(int));
        if (new_arr == NULL) return arr;  // 扩容失败
        new_arr->capacity = new_cap;
        arr = new_arr;
    }
    arr->data[arr->size++] = value;
    return arr;
}
```

## 五、柔性数组 vs 指针方案

```c
// 方案 A：柔性数组
typedef struct {
    int  count;
    char data[];
} FlexVersion;

// 方案 B：指针
typedef struct {
    int   count;
    char *data;
} PtrVersion;

void compare(void) {
    // 柔性数组：一次分配，连续内存
    FlexVersion *f = malloc(sizeof(FlexVersion) + 100);
    free(f);

    // 指针：两次分配，内存不连续
    PtrVersion *p = malloc(sizeof(PtrVersion));
    p->data = malloc(100);
    free(p->data);
    free(p);
}
```

| 对比项 | 柔性数组 | 指针方案 |
|-------|---------|---------|
| 分配次数 | 1 次 | 2 次 |
| 内存连续性 | 连续 | 可能不连续 |
| 缓存友好 | 好 | 一般 |
| 释放复杂度 | 1 次 free | 2 次 free（顺序敏感） |
| realloc 支持 | 整体 realloc | 只能 realloc 指针指向部分 |
| 编译器支持 | C99+ | 所有版本 |

## 六、历史替代方案（C99 之前）

在 C99 之前，开发者使用"零长度数组"（GCC 扩展）或"长度为 1 的数组"来实现类似效果：

```c
// GCC 扩展：零长度数组（非标准）
struct OldFlex {
    int  count;
    char data[0];  // GCC 特有，非标准
};

// C89 技巧：长度为 1 的数组
struct OldFlex2 {
    int  count;
    char data[1];  // 实际分配时分配更多
};
// 使用时: malloc(sizeof(struct OldFlex2) + n - 1)
```

> **推荐**：现代代码应使用 C99 柔性数组（`data[]`），而非旧方案。

## 七、关键要点

> **重要提示：**
>
> 1. 柔性数组必须是结构体的**最后一个**成员。
> 2. `sizeof` 不包含柔性数组部分的大小。
> 3. 使用 `malloc(sizeof(Struct) + n * sizeof(element))` 一次性分配。
> 4. 柔性数组与指针方案相比，内存连续、缓存友好、释放简单。
> 5. 可以用 `realloc` 对整个结构体进行扩容。
> 6. 零长度数组 `data[0]` 是 GCC 扩展，不是标准 C。
> 7. 包含柔性数组的结构体不能作为数组元素或嵌套在其他结构体中。
