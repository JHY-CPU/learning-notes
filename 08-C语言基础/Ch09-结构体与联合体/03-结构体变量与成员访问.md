# 03 - 结构体变量与成员访问

## 一、声明结构体变量

```c
struct Student {
    int id;
    char name[32];
    float score;
};

// 方式 1：先定义类型，再声明变量
struct Student s1;
struct Student s2, s3;

// 方式 2：定义时声明
struct Student2 {
    int id;
    char name[32];
} a, b;

// 方式 3：使用 typedef
typedef struct {
    int id;
    char name[32];
} Student;
Student s4;
```

## 二、使用 `.` 操作符访问成员

**点操作符（`.`）** 用于通过**结构体变量**直接访问成员。

### 2.1 读取成员

```c
struct Student s1 = {1001, "张三", 92.5};

printf("学号: %d\n", s1.id);
printf("姓名: %s\n", s1.name);
printf("成绩: %.1f\n", s1.score);
```

### 2.2 修改成员

```c
struct Student s1;

// 逐个赋值
s1.id = 1002;
strcpy(s1.name, "李四");
s1.score = 88.0;

// 修改已有成员
s1.score = 95.5;  // 更新成绩
s1.id++;          // 学号加 1
```

### 2.3 数组成员的特殊处理

```c
struct Student s1;

// 错误！数组名不是左值，不能直接赋值
// s1.name = "张三";  // 编译错误

// 正确方式 1：strcpy
strcpy(s1.name, "张三");

// 正确方式 2：逐字符赋值
s1.name[0] = '张';
s1.name[1] = '\0';
```

## 三、使用 `->` 操作符访问成员

**箭头操作符（`->`）** 用于通过**结构体指针**访问成员。

```c
struct Student s1 = {1001, "张三", 92.5};
struct Student *p = &s1;

// 方式 1：(*p).member  —— 先解引用，再访问
printf("学号: %d\n", (*p).id);

// 方式 2：p->member  —— 更简洁，推荐使用
printf("学号: %d\n", p->id);
printf("姓名: %s\n", p->name);
p->score = 98.0;
```

### `->` 操作符的本质

```c
p->member  等价于  (*p).member  等价于  *(p).member  // 注意优先级问题
```

> **注意**：`(*p).member` 中的括号**不可省略**，因为 `.` 的优先级高于 `*`。

## 四、结构体变量的初始化

### 4.1 顺序初始化

```c
struct Student s1 = {1001, "张三", 92.5};
```

### 4.2 使用指定初始化器（C99）

```c
struct Student s1 = {
    .name = "李四",
    .id = 1002,
    .score = 88.0
};
```

### 4.3 零初始化

```c
struct Student s1 = {0};  // 所有成员初始化为零
```

## 五、结构体变量的整体赋值

C 语言允许结构体变量之间的**整体赋值**（逐字节拷贝）。

```c
struct Student s1 = {1001, "张三", 92.5};
struct Student s2;

s2 = s1;  // 整体赋值，s2 是 s1 的完整副本

// s2 和 s1 是独立的，修改 s2 不影响 s1
strcpy(s2.name, "王五");
printf("%s\n", s1.name);  // 仍然输出 "张三"
printf("%s\n", s2.name);  // 输出 "王五"
```

**注意**：结构体整体赋值只能在**同类型**结构体之间进行。

## 六、结构体变量作为函数参数

```c
// 值传递：函数内修改不影响原变量
void print_student(struct Student s) {
    printf("学号: %d, 姓名: %s, 成绩: %.1f\n",
           s.id, s.name, s.score);
}

// 使用
struct Student s1 = {1001, "张三", 92.5};
print_student(s1);
```

## 七、完整示例

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    char brand[32];
    char model[32];
    int year;
    double price;
} Car;

int main(void) {
    // 声明并初始化
    Car car1 = {"Toyota", "Camry", 2023, 259800.0};

    // 用点操作符读取
    printf("品牌: %s\n", car1.brand);
    printf("型号: %s\n", car1.model);
    printf("年份: %d\n", car1.year);
    printf("价格: %.2f\n", car1.price);

    // 用点操作符修改
    car1.price = 249800.0;
    car1.year = 2024;
    strcpy(car1.model, "Camry Hybrid");

    printf("\n修改后:\n");
    printf("型号: %s\n", car1.model);
    printf("年份: %d\n", car1.year);
    printf("价格: %.2f\n", car1.price);

    // 使用指针和 -> 操作符
    Car *p = &car1;
    p->price = 239800.0;
    printf("\n指针修改后价格: %.2f\n", p->price);

    // 整体赋值
    Car car2 = car1;
    printf("\ncar2 品牌: %s\n", car2.brand);
    printf("car2 价格: %.2f\n", car2.price);

    return 0;
}
```

## 八、关键要点

> **重要提示：**
>
> 1. `.` 操作符用于结构体**变量**，`->` 操作符用于结构体**指针**。
> 2. 结构体的数组成员**不能直接赋值**，必须使用 `strcpy` 或 `memcpy`。
> 3. 结构体变量之间可以整体赋值，效果等同于逐字节拷贝。
> 4. `(*p).member` 中的括号不能省略，否则会因优先级出错。
> 5. 结构体指针在访问成员时，`->` 比 `(*p).` 更简洁、更常用。
> 6. 未初始化的结构体变量包含**垃圾值**，务必在使用前初始化。
