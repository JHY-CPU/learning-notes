# 08 - 字符串查找

## 一、strchr 查找字符

### 1.1 函数原型

```c
#include <string.h>
char *strchr(const char *s, int c);
```

- 在字符串 `s` 中**首次**出现字符 `c` 的位置
- `c` 可以是 `\0`
- 找到返回指向该位置的指针，未找到返回 `NULL`

### 1.2 基本示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World!";

    // 查找字符 'o'
    char *pos = strchr(str, 'o');
    if (pos) {
        printf("找到 'o' 在位置: %ld\n", pos - str);  // 4
        printf("从 'o' 开始: %s\n", pos);              // o, World!
    }

    // 查找不存在的字符
    pos = strchr(str, 'x');
    if (pos == NULL) {
        printf("未找到 'x'\n");
    }

    // 查找 \0（字符串结尾）
    pos = strchr(str, '\0');
    printf("结尾位置: %ld\n", pos - str);  // 13

    return 0;
}
```

### 1.3 实际应用：提取子串

```c
#include <stdio.h>
#include <string.h>

// 从邮箱地址中提取域名
void extract_domain(const char *email) {
    char *at = strchr(email, '@');
    if (at) {
        printf("用户名: %.*s\n", (int)(at - email), email);
        printf("域名: %s\n", at + 1);
    } else {
        printf("无效邮箱\n");
    }
}

int main() {
    extract_domain("user@example.com");
    // 用户名: user
    // 域名: example.com
    return 0;
}
```

## 二、strrchr 反向查找字符

### 2.1 函数原型

```c
#include <string.h>
char *strrchr(const char *s, int c);
```

- 从字符串末尾向前查找字符 `c` **最后一次**出现的位置

### 2.2 基本示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    char path[] = "/home/user/documents/file.txt";

    // 查找最后一个 '/'
    char *last_slash = strrchr(path, '/');
    if (last_slash) {
        printf("目录: %.*s\n", (int)(last_slash - path), path);
        printf("文件名: %s\n", last_slash + 1);
    }
    // 目录: /home/user/documents
    // 文件名: file.txt

    // 查找最后一个 '.'
    char *ext = strrchr(path, '.');
    if (ext) {
        printf("扩展名: %s\n", ext + 1);  // txt
    }

    // 对比 strchr 和 strrchr
    char str[] = "abcabc";
    printf("strchr:  %ld\n", strchr(str, 'a') - str);   // 0
    printf("strrchr: %ld\n", strrchr(str, 'a') - str);  // 3

    return 0;
}
```

## 三、strstr 查找子串

### 3.1 函数原型

```c
#include <string.h>
char *strstr(const char *haystack, const char *needle);
```

- 在 `haystack` 中查找子串 `needle` **首次**出现的位置
- `needle` 为空字符串时返回 `haystack`
- 找到返回指针，未找到返回 `NULL`

### 3.2 基本示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    char text[] = "The quick brown fox jumps over the lazy dog";

    // 查找子串
    char *pos = strstr(text, "fox");
    if (pos) {
        printf("找到 'fox' 在位置: %ld\n", pos - text);  // 16
        printf("上下文: %s\n", pos);                       // fox jumps...
    }

    // 查找不存在的子串
    pos = strstr(text, "cat");
    if (pos == NULL) {
        printf("未找到 'cat'\n");
    }

    // 查找自身
    pos = strstr(text, text);
    printf("找到自身: %s\n", pos);  // 完整字符串

    return 0;
}
```

### 3.3 实际应用：关键词过滤

```c
#include <stdio.h>
#include <string.h>

// 检查文本中是否包含敏感词
int contains_keyword(const char *text, const char *keywords[], int count) {
    for (int i = 0; i < count; i++) {
        if (strstr(text, keywords[i]) != NULL) {
            printf("发现敏感词: %s\n", keywords[i]);
            return 1;
        }
    }
    return 0;
}

int main() {
    const char *keywords[] = {"spam", "viagra", "lottery"};
    char message[] = "You won the lottery! Click here!";

    if (contains_keyword(message, keywords, 3)) {
        printf("此消息可能是垃圾信息\n");
    }

    return 0;
}
```

## 四、strpbrk 查找字符集合

### 4.1 函数原型

```c
#include <string.h>
char *strpbrk(const char *s, const char *accept);
```

- 在 `s` 中查找 `accept` 中**任意字符**首次出现的位置

```c
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World! 123";

    // 查找第一个出现的元音字母
    char *pos = strpbrk(str, "aeiouAEIOU");
    if (pos) {
        printf("第一个元音: '%c' 在位置 %ld\n", *pos, pos - str);
        // 第一个元音: 'e' 在位置 1
    }

    // 查找第一个数字
    pos = strpbrk(str, "0123456789");
    if (pos) {
        printf("第一个数字: '%c' 在位置 %ld\n", *pos, pos - str);
        // 第一个数字: '1' 在位置 15
    }

    return 0;
}
```

## 五、各查找函数对比

| 函数 | 查找方向 | 查找目标 | 返回 |
|------|----------|----------|------|
| `strchr` | 正向 | 单个字符（首次） | 指针或NULL |
| `strrchr` | 反向 | 单个字符（末次） | 指针或NULL |
| `strstr` | 正向 | 子串（首次） | 指针或NULL |
| `strpbrk` | 正向 | 字符集合（首次） | 指针或NULL |

## 六、重要注意事项

> **关键要点：**
> 1. **所有查找函数都可能返回 NULL**：使用前必须检查
> 2. **`strchr` 的第二个参数是 `int` 类型**：会自动转换为 `char`
> 3. **`strstr` 的 needle 为空串时返回 haystack**
> 4. **`strrchr` 的 'r' 代表 reverse（反向）**
> 5. **查找时注意字符串长度**：对于长文本频繁查找，考虑更高效的算法
> 6. **返回的指针指向原字符串内部位置**：不要对返回指针做 `free` 操作
