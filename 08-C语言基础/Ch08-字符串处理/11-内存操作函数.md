# 11 - 内存操作函数

## 一、memcpy 内存复制

### 1.1 函数原型

```c
#include <string.h>
void *memcpy(void *dest, const void *src, size_t n);
```

- 从 `src` 复制 `n` 个字节到 `dest`
- **不处理内存重叠**：重叠时行为未定义
- 返回值：`dest`

### 1.2 基本示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    // 复制整型数组
    int src[] = {1, 2, 3, 4, 5};
    int dest[5];
    memcpy(dest, src, sizeof(src));

    for (int i = 0; i < 5; i++) {
        printf("%d ", dest[i]);  // 1 2 3 4 5
    }
    printf("\n");

    // 复制结构体
    struct Point { int x, y; };
    struct Point p1 = {10, 20};
    struct Point p2;
    memcpy(&p2, &p1, sizeof(struct Point));
    printf("p2: (%d, %d)\n", p2.x, p2.y);  // (10, 20)

    // 复制字符串（含\0）
    char str1[] = "Hello";
    char str2[10];
    memcpy(str2, str1, strlen(str1) + 1);  // +1 包含\0
    printf("%s\n", str2);  // Hello

    return 0;
}
```

## 二、memmove 安全内存移动

### 2.1 函数原型

```c
#include <string.h>
void *memmove(void *dest, const void *src, size_t n);
```

- 与 `memcpy` 类似，但**能正确处理内存重叠**
- 实现方式：当 dest < src 时正向复制，否则反向复制

### 2.2 重叠问题示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World!";

    // ⚠️ memcpy 处理重叠可能出错
    // memcpy(str + 1, str, 5);  // 未定义行为！

    // ✅ memmove 正确处理重叠
    memmove(str + 1, str, 5);
    printf("%s\n", str);  // HHello World!

    // 实际应用：数组元素移动
    int arr[] = {1, 2, 3, 4, 5};
    memmove(arr + 2, arr, 3 * sizeof(int));  // 向右移动
    // arr: {1, 2, 1, 2, 3}

    for (int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);  // 1 2 1 2 3
    }

    return 0;
}
```

### 2.3 memcpy 与 memmove 的选择

```c
// 不确定是否重叠？用 memmove 更安全
// 确定不重叠？用 memcpy 可能更快（编译器优化）

// 推荐：除非确定不重叠且需要极致性能，否则使用 memmove
```

## 三、memcmp 内存比较

### 3.1 函数原型

```c
#include <string.h>
int memcmp(const void *s1, const void *s2, size_t n);
```

- 比较 `s1` 和 `s2` 的前 `n` 个字节
- 返回值：`< 0`、`0`、`> 0`（与 `strcmp` 类似）

### 3.2 基本示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    int a[] = {1, 2, 3};
    int b[] = {1, 2, 3};
    int c[] = {1, 2, 4};

    // 比较整型数组
    printf("%d\n", memcmp(a, b, sizeof(a)));  // 0 (相等)
    printf("%d\n", memcmp(a, c, sizeof(a)));  // 负数 (a < c)

    // 比较结构体（注意填充字节）
    struct Data { int a; char b; };
    struct Data d1 = {100, 'A'};
    struct Data d2 = {100, 'A'};
    // ⚠️ 结构体可能有填充字节，memcmp 可能不准确
    // 建议逐个成员比较

    return 0;
}
```

## 四、memset 内存设置

### 4.1 函数原型

```c
#include <string.h>
void *memset(void *s, int c, size_t n);
```

- 将 `s` 指向的内存的前 `n` 个字节设置为值 `c`
- 返回值：`s`

### 4.2 基本示例

```c
#include <stdio.h>
#include <string.h>

int main() {
    // 初始化数组为0
    int arr[100];
    memset(arr, 0, sizeof(arr));

    // 初始化字符数组
    char buf[50];
    memset(buf, 'A', sizeof(buf));
    buf[49] = '\0';  // 添加终止符
    printf("%s\n", buf);

    // 初始化为 -1（所有字节设为 0xFF）
    int nums[10];
    memset(nums, -1, sizeof(nums));
    // 每个int的4个字节都是0xFF
    // int值为 -1 (0xFFFFFFFF)

    // ⚠️ memset 按字节设置，不能用于设置非0/-1的int值
    memset(nums, 1, sizeof(nums));
    // 每个字节是 0x01
    // int值为 0x01010101 = 16843009，不是 1！

    // ✅ 正确初始化int数组为特定值
    for (int i = 0; i < 10; i++) {
        nums[i] = 1;  // 每个元素设为1
    }

    return 0;
}
```

### 4.3 常见用法

```c
// 清空结构体
struct Config {
    char name[50];
    int value;
    int flags;
};
struct Config cfg;
memset(&cfg, 0, sizeof(cfg));  // 全部清零

// 清空缓冲区
char buffer[1024];
memset(buffer, 0, sizeof(buffer));
```

## 五、memchr 内存中查找

### 5.1 函数原型

```c
#include <string.h>
void *memchr(const void *s, int c, size_t n);
```

- 在 `s` 的前 `n` 个字节中查找值为 `c` 的字节

```c
#include <stdio.h>
#include <string.h>

int main() {
    char data[] = "Hello, World!";

    // 查找字符 'W'
    char *pos = memchr(data, 'W', sizeof(data));
    if (pos) {
        printf("找到 'W' 在位置: %ld\n", pos - data);  // 7
    }

    // 在二进制数据中查找
    unsigned char bytes[] = {0x01, 0x02, 0xFF, 0x04};
    unsigned char *found = memchr(bytes, 0xFF, sizeof(bytes));
    if (found) {
        printf("找到 0xFF 在位置: %ld\n", found - bytes);  // 2
    }

    return 0;
}
```

## 六、内存函数对比总结

| 函数 | 功能 | 处理重叠 | 适用场景 |
|------|------|----------|----------|
| `memcpy` | 复制内存 | 不处理 | 通用复制（不重叠） |
| `memmove` | 移动内存 | 正确处理 | 可能重叠的复制 |
| `memcmp` | 比较内存 | - | 二进制数据比较 |
| `memset` | 设置内存 | - | 初始化/清零 |
| `memchr` | 查找字节 | - | 二进制数据查找 |

## 七、重要注意事项

> **关键要点：**
> 1. **不确定是否重叠就用 `memmove`**：更安全，性能差异通常可忽略
> 2. **`memset` 按字节设置**：只能可靠设置 0 和 -1（对于多字节类型）
> 3. **`memcmp` 比较结构体要小心**：填充字节可能不同，建议逐成员比较
> 4. **所有内存函数操作字节**：不关心数据类型，需要手动计算大小
> 5. **返回值可链式使用**：`memset(memcpy(dst, src, n), 0, n)` 不推荐，可读性差
> 6. **这些函数都在 `<string.h>` 中**：虽然操作内存，但不在 `<stdlib.h>`
