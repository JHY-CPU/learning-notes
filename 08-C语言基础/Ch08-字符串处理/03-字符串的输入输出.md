# 03 - 字符串的输入输出

## 一、字符串输出

### 1.1 printf 输出字符串

`printf` 是最常用的字符串输出函数，使用 `%s` 格式符。

```c
#include <stdio.h>

int main() {
    char str[] = "Hello, World!";
    char *ptr = "C Language";

    // 基本输出
    printf("%s\n", str);            // Hello, World!

    // 指定最小宽度（右对齐）
    printf("%20s\n", str);          //        Hello, World!

    // 左对齐
    printf("%-20s|\n", str);        // Hello, World!        |

    // 限制输出字符数
    printf("%.5s\n", str);          // Hello
    printf("%10.5s\n", str);        //      Hello

    // 输出指针指向的字符串
    printf("%s\n", ptr);            // C Language

    return 0;
}
```

### 1.2 puts 输出字符串

`puts` 自动在末尾添加换行符。

```c
#include <stdio.h>

int main() {
    char str[] = "Hello";

    puts(str);          // 输出: Hello\n
    puts("World");      // 输出: World\n

    // puts 遇到 \0 停止
    char buf[] = "Hello\0World";
    puts(buf);          // 只输出: Hello

    // puts 不能格式化输出
    // puts("Number: %d", 42);   // 错误！puts不支持格式化

    return 0;
}
```

### 1.3 fputs 输出字符串

`fputs` 不自动添加换行，可以指定输出流。

```c
#include <stdio.h>

int main() {
    // 输出到标准输出
    fputs("Hello", stdout);
    fputs("World\n", stdout);

    // 输出到文件
    FILE *fp = fopen("output.txt", "w");
    if (fp) {
        fputs("Line 1\n", fp);
        fputs("Line 2\n", fp);
        fclose(fp);
    }

    return 0;
}
```

### 1.4 putchar / fputc 逐字符输出

```c
void print_string(const char *str) {
    while (*str) {
        putchar(*str++);
    }
    putchar('\n');
}
```

## 二、字符串输入

### 2.1 scanf 读取字符串

```c
#include <stdio.h>

int main() {
    char str[100];

    // ⚠️ 危险：不指定宽度，可能缓冲区溢出
    // scanf("%s", str);

    // ✅ 安全：指定最大读取字符数（99 + \0）
    scanf("%99s", str);

    // scanf 遇到空白字符（空格、Tab、换行）停止
    // 输入 "Hello World"，str 只会得到 "Hello"

    printf("Read: %s\n", str);

    return 0;
}
```

**scanf 的问题：**
- 遇到空白字符停止
- 不检查缓冲区大小
- 可能导致缓冲区溢出

### 2.2 gets（已废弃，不要使用！）

```c
// ⚠️ gets 已在 C11 中移除，绝对不要使用！
// char str[10];
// gets(str);    // 极度危险！不检查缓冲区大小

// 为什么危险？
// 输入超过数组大小的字符 → 缓冲区溢出 → 程序崩溃或安全漏洞
```

### 2.3 fgets 安全输入（推荐）

`fgets` 是推荐的字符串输入方式。

```c
#include <stdio.h>
#include <string.h>

int main() {
    char str[100];

    // fgets(缓冲区, 最大大小, 输入流)
    // 最多读取 size-1 个字符，自动添加 \0
    if (fgets(str, sizeof(str), stdin) != NULL) {
        // 去除末尾的换行符
        size_t len = strlen(str);
        if (len > 0 && str[len - 1] == '\n') {
            str[len - 1] = '\0';
        }
        printf("You entered: %s\n", str);
    }

    return 0;
}
```

### 2.4 封装安全输入函数

```c
#include <stdio.h>
#include <string.h>

// 安全的字符串输入，自动去除换行符
char *safe_input(char *buffer, size_t size) {
    if (fgets(buffer, (int)size, stdin) == NULL) {
        return NULL;
    }

    // 去除末尾换行符
    size_t len = strlen(buffer);
    if (len > 0 && buffer[len - 1] == '\n') {
        buffer[len - 1] = '\0';
    } else {
        // 输入超长，清空剩余缓冲区
        int ch;
        while ((ch = getchar()) != '\n' && ch != EOF);
    }

    return buffer;
}

int main() {
    char name[50];

    printf("请输入姓名: ");
    if (safe_input(name, sizeof(name)) != NULL) {
        printf("你好, %s!\n", name);
    }

    return 0;
}
```

### 2.5 getchar 逐字符读取

```c
#include <stdio.h>

// 用 getchar 读取一行
int read_line(char *buf, int max_len) {
    int i = 0;
    int ch;

    while (i < max_len - 1 && (ch = getchar()) != '\n' && ch != EOF) {
        buf[i++] = (char)ch;
    }
    buf[i] = '\0';

    return i;  // 返回读取的字符数
}
```

## 三、各输入函数对比

| 函数 | 是否安全 | 遇空白停止 | 自动加\0 | 添加换行 |
|------|----------|------------|----------|----------|
| `scanf("%s")` | 不安全 | 是 | 是 | 否 |
| `scanf("%99s")` | 较安全 | 是 | 是 | 否 |
| `gets` | **极不安全** | 否 | 是 | 否 |
| `fgets` | **安全** | 否 | 是 | 保留\n |
| `getchar` | 安全 | 否 | 需手动 | 否 |

## 四、格式化输出详解

```c
#include <stdio.h>

int main() {
    char str[] = "Hello, World!";

    printf("|%s|\n", str);          // |Hello, World!|
    printf("|%20s|\n", str);        // |    Hello, World!|
    printf("|%-20s|\n", str);       // |Hello, World!    |
    printf("|%.5s|\n", str);        // |Hello|
    printf("|%20.5s|\n", str);      // |               Hello|
    printf("|%-20.5s|\n", str);     // |Hello               |

    // 打印地址
    printf("Address: %p\n", (void *)str);

    return 0;
}
```

## 五、重要注意事项

> **关键要点：**
> 1. **永远不要使用 `gets`**：已被C11标准移除，使用 `fgets` 替代
> 2. **`scanf("%s")` 要指定宽度**：如 `scanf("%99s", buf)` 防止溢出
> 3. **`fgets` 会保留换行符**：需要手动去除
> 4. **`scanf` 遇空白停止**：不能读取含空格的字符串，用 `fgets` 替代
> 5. **检查返回值**：输入函数可能失败（如遇到EOF），应检查返回值
> 6. **缓冲区大小永远要比实际需要多1**：留位置给 `\0`
