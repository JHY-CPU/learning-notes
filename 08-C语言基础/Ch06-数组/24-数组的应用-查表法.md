# 数组的应用：查表法

## 1. 概述

**查表法（Lookup Table）** 是用数组预存计算结果，用O(1)时间直接查询，替代运行时计算或冗长的分支判断。

核心思想：**用空间换时间**。

## 2. 替代 if-else 链

### 2.1 星期几的名称

```c
// 不用查表：if-else 链
const char* get_day_name(int day) {
    if (day == 0) return "星期日";
    else if (day == 1) return "星期一";
    else if (day == 2) return "星期二";
    else if (day == 3) return "星期三";
    else if (day == 4) return "星期四";
    else if (day == 5) return "星期五";
    else if (day == 6) return "星期六";
    else return "无效";
}

// 用查表法
const char* get_day_name_table(int day) {
    static const char *days[] = {
        "星期日", "星期一", "星期二", "星期三",
        "星期四", "星期五", "星期六"
    };
    if (day < 0 || day > 6) return "无效";
    return days[day];
}
```

### 2.2 月份天数

```c
// 查表法获取月份天数（非闰年）
int days_in_month(int month) {
    static const int days[] = {
        0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31
    };
    if (month < 1 || month > 12) return -1;
    return days[month];
}

// 支持闰年
int days_in_month_leap(int month, int is_leap) {
    static const int days[] = {
        0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31
    };
    if (month == 2 && is_leap) return 29;
    return days[month];
}
```

## 3. 预计算数学表

### 3.1 三角函数表

```c
#include <stdio.h>
#include <math.h>

#define TABLE_SIZE 360

// 预计算正弦表（0-359度）
void build_sin_table(double table[], int n) {
    for (int i = 0; i < n; i++) {
        table[i] = sin(i * M_PI / 180.0);
    }
}

int main(void) {
    double sin_table[TABLE_SIZE];
    build_sin_table(sin_table, TABLE_SIZE);

    // 查表获取 sin(45度)
    printf("sin(45°) = %.6f\n", sin_table[45]);
    printf("sin(90°) = %.6f\n", sin_table[90]);

    return 0;
}
```

### 3.2 平方/立方表

```c
int square_table[101];  // 0-100的平方

void build_square_table(void) {
    for (int i = 0; i <= 100; i++) {
        square_table[i] = i * i;
    }
}

// 使用
int sq = square_table[17];  // 289，无需计算
```

### 3.3 阶乘表

```c
// 0! 到 12! (13!超过int范围)
long long factorial_table[13];

void build_factorial_table(void) {
    factorial_table[0] = 1;
    for (int i = 1; i <= 12; i++) {
        factorial_table[i] = factorial_table[i - 1] * i;
    }
}

// 10! = 3628800，直接查表获得
```

## 4. 字符映射表

### 4.1 大小写转换

```c
// ASCII码映射表
char to_upper_table[256];
char to_lower_table[256];

void build_case_tables(void) {
    for (int i = 0; i < 256; i++) {
        to_upper_table[i] = (char)i;
        to_lower_table[i] = (char)i;
    }
    for (int i = 'a'; i <= 'z'; i++) {
        to_upper_table[i] = (char)(i - 32);
    }
    for (int i = 'A'; i <= 'Z'; i++) {
        to_lower_table[i] = (char)(i + 32);
    }
}

// 使用
char upper = to_upper_table['a'];  // 'A'
```

### 4.2 十六进制字符值

```c
int hex_value[256] = {0};

void build_hex_table(void) {
    for (int i = 0; i < 256; i++) hex_value[i] = -1;
    for (int i = '0'; i <= '9'; i++) hex_value[i] = i - '0';
    for (int i = 'a'; i <= 'f'; i++) hex_value[i] = i - 'a' + 10;
    for (int i = 'A'; i <= 'F'; i++) hex_value[i] = i - 'A' + 10;
}

// 使用
int v = hex_value['F'];  // 15
```

## 5. 状态机/转移表

```c
// 简易状态机：有限自动机判断是否接受
// 状态: 0=初始, 1=看到'a', 2=看到'ab'
// 输入: 'a'=0, 'b'=1, 其他=2

int transition[3][3] = {
    //   a    b   其他
    {   1,   0,   0  },  // 状态0
    {   1,   2,   0  },  // 状态1
    {   1,   0,   0  },  // 状态2 (接受态)
};

int is_accepting[3] = {0, 0, 1};  // 状态2是接受态

int accepts(const char *str) {
    int state = 0;
    for (int i = 0; str[i]; i++) {
        int input;
        if (str[i] == 'a') input = 0;
        else if (str[i] == 'b') input = 1;
        else input = 2;
        state = transition[state][input];
    }
    return is_accepting[state];
}
```

## 6. 查表法的优缺点

| 优点 | 缺点 |
|------|------|
| O(1) 查询时间 | 需要额外内存 |
| 避免重复计算 | 初始化有开销 |
| 代码简洁 | 大表可能浪费空间 |
| 无分支预测失败 | 不适合频繁变化的数据 |

---

## 重点总结

- 查表法的核心：**预计算 + 直接索引**，用空间换时间
- 最适合**数据范围固定**、**查询频繁**的场景
- 字符串映射用指针数组，数值映射用数值数组
- 数学函数查表要权衡**精度和空间**
- 状态机用二维数组实现转移表，代码清晰高效
