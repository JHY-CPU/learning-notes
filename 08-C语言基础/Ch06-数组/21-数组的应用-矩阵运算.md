# 数组的应用：矩阵运算

## 1. 矩阵的表示

矩阵可以用二维数组表示。一个 `m×n` 矩阵有 `m` 行 `n` 列。

```c
#define ROWS 3
#define COLS 3

int matrix[ROWS][COLS] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

## 2. 矩阵输出

```c
#include <stdio.h>

#define ROWS 3
#define COLS 3

void print_matrix(int m[][COLS], int rows) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < COLS; j++) {
            printf("%4d", m[i][j]);
        }
        printf("\n");
    }
}
```

## 3. 矩阵加法

两个同维矩阵对应元素相加。

```c
void matrix_add(int a[][COLS], int b[][COLS], int result[][COLS], int rows) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < COLS; j++) {
            result[i][j] = a[i][j] + b[i][j];
        }
    }
}
```

**完整示例：**

```c
int main(void) {
    int a[ROWS][COLS] = {{1,2,3},{4,5,6},{7,8,9}};
    int b[ROWS][COLS] = {{9,8,7},{6,5,4},{3,2,1}};
    int c[ROWS][COLS];

    matrix_add(a, b, c, ROWS);

    printf("A + B =\n");
    print_matrix(c, ROWS);
    //  10  10  10
    //  10  10  10
    //  10  10  10

    return 0;
}
```

## 4. 矩阵转置

行列互换。`A[i][j]` 变为 `A^T[j][i]`。

### 4.1 方阵原地转置

```c
void transpose_square(int m[][COLS], int n) {
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            int temp = m[i][j];
            m[i][j] = m[j][i];
            m[j][i] = temp;
        }
    }
}
```

### 4.2 非方阵转置

```c
// m×n 矩阵转置为 n×m 矩阵
void transpose(int src[][COLS], int dst[][ROWS], int rows, int cols) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            dst[j][i] = src[i][j];
        }
    }
}
```

## 5. 矩阵乘法

`C[i][j] = Σ A[i][k] × B[k][j]`，其中 A 是 `m×p`，B 是 `p×n`，C 是 `m×n`。

```c
void matrix_multiply(int a[][COLS], int b[][COLS],
                     int result[][COLS], int n) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            result[i][j] = 0;
            for (int k = 0; k < n; k++) {
                result[i][j] += a[i][k] * b[k][j];
            }
        }
    }
}
```

**完整示例：**

```c
int main(void) {
    int a[3][3] = {{1,2,3},{4,5,6},{7,8,9}};
    int b[3][3] = {{9,8,7},{6,5,4},{3,2,1}};
    int c[3][3];

    matrix_multiply(a, b, c, 3);

    printf("A × B =\n");
    print_matrix(c, 3);

    return 0;
}
```

输出：
```
A × B =
  30  24  18
  84  69  54
 138 114  90
```

## 6. 矩阵标量乘法

```c
void matrix_scale(int m[][COLS], int rows, int scalar) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < COLS; j++) {
            m[i][j] *= scalar;
        }
    }
}
```

## 7. 求矩阵的迹（Trace）

矩阵的迹 = 主对角线元素之和。

```c
int trace(int m[][COLS], int n) {
    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += m[i][i];
    }
    return sum;
}
```

## 8. 判断对称矩阵

对称矩阵满足 `A[i][j] == A[j][i]`。

```c
int is_symmetric(int m[][COLS], int n) {
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            if (m[i][j] != m[j][i]) return 0;
        }
    }
    return 1;
}
```

## 9. 完整示例

```c
#include <stdio.h>

#define N 3

void print_matrix(int m[][N], int n) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++)
            printf("%4d", m[i][j]);
        printf("\n");
    }
}

int main(void) {
    int a[N][N] = {{1,2,3},{4,5,6},{7,8,9}};

    printf("原矩阵:\n");
    print_matrix(a, N);

    printf("\n转置后:\n");
    for (int i = 0; i < N; i++)
        for (int j = i+1; j < N; j++) {
            int t = a[i][j]; a[i][j] = a[j][i]; a[j][i] = t;
        }
    print_matrix(a, N);

    printf("\n迹 = %d\n", 1+5+9);

    return 0;
}
```

---

## 重点总结

- 矩阵用二维数组表示，注意行和列的对应关系
- 矩阵乘法三重循环，时间复杂度 O(n³)
- 方阵转置只需交换上三角元素
- 注意矩阵运算的维度约束（加法需同维，乘法需内维相等）
- 大规模矩阵运算应考虑缓存优化和专用库（如BLAS）
