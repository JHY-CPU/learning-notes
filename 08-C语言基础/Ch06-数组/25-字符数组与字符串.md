# 字符数组与字符串

## 1. 字符数组

字符数组就是元素类型为 `char` 的数组。

```c
char arr[5] = {'H', 'e', 'l', 'l', 'o'};
// 这是字符数组，但不是字符串（没有'\0'结尾）
```

## 2. 字符串的本质

C语言中的字符串是以 **`'\0'`（空字符，ASCII码0）结尾**的字符数组。

```c
char str[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
// 这是一个字符串

char str2[] = "Hello";
// 编译器自动添加'\0'，等价于 char str2[6] = {'H','e','l','l','o','\0'};
```

```
字符数组（不是字符串）:
┌─────┬─────┬─────┬─────┬─────┐
│ 'H' │ 'e' │ 'l' │ 'l' │ 'o' │
└─────┴─────┴─────┴─────┴─────┘

字符串:
┌─────┬─────┬─────┬─────┬─────┬─────┐
│ 'H' │ 'e' │ 'l' │ 'l' │ 'o' │'\0' │
└─────┴─────┴─────┴─────┴─────┴─────┘
  必须有'\0'结尾
```

## 3. 字符串的初始化方式

```c
// 方式1：字符串字面量（最常用）
char s1[] = "Hello";          // 自动加'\0'，大小为6

// 方式2：逐字符初始化
char s2[] = {'H','e','l','l','o','\0'};

// 方式3：指定大小，部分初始化
char s3[10] = "Hello";        // s3[5]='\0'，其余为'\0'

// 方式4：指针指向字符串字面量
const char *s4 = "Hello";     // 字符串字面量存储在只读区

// 方式5：全部初始化为空字符串
char s5[100] = {0};           // 所有字符为'\0'
char s6[100] = "";            // 第一个字符为'\0'，其余为'\0'
```

## 4. 字符数组 vs 字符串指针

```c
// 可修改的字符数组
char arr[] = "Hello";
arr[0] = 'h';  // 合法：arr = "hello"

// 字符串字面量指针（不可修改！）
const char *ptr = "Hello";
// ptr[0] = 'h';  // 未定义行为！字符串字面量在只读区
```

| 特性 | `char arr[]` | `const char *ptr` |
|------|-------------|-------------------|
| 存储位置 | 栈上 | 只读数据段 |
| 可否修改 | 可以 | **不可以** |
| sizeof | 包含'\0'的数组大小 | 指针大小(4或8) |
| 赋值 | 不能整体赋值 | 可以重新指向 |

## 5. 常用字符串操作（基于字符数组）

### 5.1 计算字符串长度

```c
int my_strlen(const char s[]) {
    int len = 0;
    while (s[len] != '\0') {
        len++;
    }
    return len;
}

// 指针版本
int my_strlen_ptr(const char *s) {
    const char *p = s;
    while (*p) p++;
    return p - s;
}
```

### 5.2 字符串复制

```c
void my_strcpy(char dst[], const char src[]) {
    int i = 0;
    while (src[i] != '\0') {
        dst[i] = src[i];
        i++;
    }
    dst[i] = '\0';  // 别忘了结尾的'\0'
}
```

### 5.3 字符串连接

```c
void my_strcat(char dst[], const char src[]) {
    int len = 0;
    while (dst[len] != '\0') len++;  // 找dst的末尾

    int i = 0;
    while (src[i] != '\0') {
        dst[len + i] = src[i];
        i++;
    }
    dst[len + i] = '\0';
}
```

### 5.4 字符串比较

```c
int my_strcmp(const char s1[], const char s2[]) {
    int i = 0;
    while (s1[i] && s2[i]) {
        if (s1[i] != s2[i])
            return s1[i] - s2[i];
        i++;
    }
    return s1[i] - s2[i];
}
```

## 6. 标准库函数（string.h）

```c
#include <string.h>

char s[100];

strlen(s);              // 返回字符串长度（不含'\0'）
strcpy(s, "Hello");     // 复制字符串到s
strcat(s, " World");    // 连接字符串
strcmp(s, "Hello World"); // 比较字符串

strncpy(s, src, n);     // 安全复制，最多n个字符
strncat(s, src, n);     // 安全连接
strncmp(s1, s2, n);     // 比较前n个字符

strchr(s, 'o');         // 查找字符首次出现位置
strstr(s, "World");     // 查找子串
```

## 7. 常见陷阱

```c
// 陷阱1：忘记预留'\0'的空间
char buf[5];
strcpy(buf, "Hello");  // 需要6字节（含'\0'），溢出！

// 陷阱2：未初始化的字符数组
char s[10];
printf("%s\n", s);     // 未定义行为，没有'\0'

// 陷阱3：修改字符串字面量
char *p = "Hello";
p[0] = 'h';            // 未定义行为！

// 陷阱4：字符数组不等于字符串
char a[] = {'a','b','c'};  // 不是字符串！
printf("%s\n", a);          // 越界读取，未定义行为
```

---

## 重点总结

- C语言字符串 = 以 `'\0'` 结尾的字符数组
- `char arr[] = "Hello"` 自动添加 `'\0'`，大小为6
- 字符串字面量存储在只读区，**不可修改**
- `char[]` 可以修改内容，`const char *` 不能
- 标准库 `string.h` 提供丰富的字符串操作函数
- **永远记得为 `'\0'` 预留空间**
