# 数组的初始化技巧

## 1. 全零初始化

### 1.1 部分初始化技巧

在C语言中，**只要提供了至少一个初始化值**，剩余元素自动初始化为0。

```c
int a[100] = {0};        // 全部100个元素都是0
int b[100] = {1};        // b[0]=1, 其余99个都是0
int c[100] = {};         // GCC扩展，全零初始化（非标准C）
```

这是最简洁的**清零**写法，比 `memset` 更安全且编译器可优化。

### 1.2 不同类型的零初始化

```c
int     int_arr[10] = {0};        // 全部为 0
double  dbl_arr[10] = {0.0};      // 全部为 0.0
char    chr_arr[10] = {0};        // 全部为 '\0'
int    *ptr_arr[10] = {NULL};     // 全部为 NULL
```

## 2. 部分初始化

```c
// 只初始化前几个元素
int arr[10] = {1, 2, 3};
// 等价于: {1, 2, 3, 0, 0, 0, 0, 0, 0, 0}

// 初始化首尾元素，中间自动补零
int days[12] = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

// 稀疏数组初始化
int sparse[100] = {1, 0, 0, 0, 5};
// sparse[0]=1, sparse[4]=5, 其余为0
```

## 3. C99 指定初始化器

### 3.1 基本语法

```c
// 指定下标初始化
int a[10] = {[2] = 10, [5] = 20, [9] = 30};

// 值可以覆盖前面的初始化
int b[5] = {[1] = 100, [1] = 200};  // b[1] = 200
```

### 3.2 混合使用

```c
// 按顺序和指定混用
int a[] = {1, 2, [5] = 10, 11, [2] = 20};
// a[0]=1, a[1]=2, a[2]=20, a[5]=10, a[6]=11
// 数组长度为7（最大下标6 + 1）
```

### 3.3 实际应用

```c
// 初始化错误码表
enum { OK, ERR_FILE, ERR_MEM, ERR_NET, ERR_TIMEOUT };
int error_messages[10] = {
    [OK]       = 0,
    [ERR_FILE] = -1,
    [ERR_MEM]  = -2,
    [ERR_NET]  = -3
};

// 初始化月份天数（用下标1-12）
int days[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
// 或用指定初始化
int days2[13] = {
    [1] = 31, [2] = 28, [3] = 31, [4] = 30,
    [5] = 31, [6] = 30, [7] = 31, [8] = 31,
    [9] = 30, [10] = 31, [11] = 30, [12] = 31
};
```

## 4. 编译时常量初始化

```c
// 使用宏定义初始化值
#define INIT_VALUES {10, 20, 30, 40, 50}
int arr[5] = INIT_VALUES;

// 使用枚举
enum { N = 5 };
int b[N] = {1, 2, 3, 4, 5};

// 静态常量数组
const int LOOKUP[256] = {
    ['0'] = 0, ['1'] = 1, ['2'] = 2,
    ['A'] = 10, ['B'] = 11, ['C'] = 12
};
```

## 5. 运行时初始化

### 5.1 循环初始化

```c
int arr[100];
for (int i = 0; i < 100; i++) {
    arr[i] = i * i;  // 平方序列
}
```

### 5.2 memset 初始化

```c
#include <string.h>

int arr[100];
memset(arr, 0, sizeof(arr));        // 全部清零
memset(arr, 0xFF, sizeof(arr));     // 每字节设为0xFF
memset(arr, -1, sizeof(arr));       // 每字节设为0xFF（-1的补码）
```

> **注意**：`memset` 按字节设置，不能用于将int数组初始化为非0非-1的值。

```c
int arr[100];
memset(arr, 1, sizeof(arr));
// 每个int变为 0x01010101 = 16843009，而不是1！
```

### 5.3 拷贝初始化

```c
int src[5] = {1, 2, 3, 4, 5};
int dst[5];
memcpy(dst, src, sizeof(src));
```

## 6. 初始化陷阱

```c
// 陷阱1：局部数组未初始化
void func(void) {
    int arr[10];
    printf("%d\n", arr[0]);  // 未定义行为！值不确定
}

// 陷阱2：长度为0的数组
// int arr[0];  // C89不合法，C99 VLA中可以

// 陷阱3：初始化值过多
// int arr[3] = {1, 2, 3, 4};  // 编译错误

// 陷阱4：全局/静态 vs 局部
int global[10];           // 全局自动初始化为0
void func(void) {
    static int s[10];     // 静态自动初始化为0
    int local[10];        // 局部，值不确定！
}
```

---

## 重点总结

- `{0}` 是**最简洁的全零初始化**方式
- 部分初始化时，剩余元素自动为0
- C99 **指定初始化器** `[index] = value` 提供灵活的初始化方式
- `memset` 按字节操作，只能可靠地设置0和-1
- 局部数组**不自动初始化**，必须显式赋值或初始化
- 全局和静态数组**自动初始化为0**
