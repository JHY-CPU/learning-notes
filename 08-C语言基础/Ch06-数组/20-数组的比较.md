# 数组的比较

## 1. 不能直接比较

**数组不能用 `==` 或 `!=` 直接比较**。

```c
int a[5] = {1, 2, 3, 4, 5};
int b[5] = {1, 2, 3, 4, 5};

// if (a == b)    // 错误！比较的是地址，不是内容
// a > b          // 比较地址，无意义
```

数组名在表达式中退化为指针，`a == b` 比较的是两个数组在内存中的地址，而不是内容。

## 2. 逐元素比较

### 2.1 判断两个数组是否相等

```c
#include <stdio.h>

int arrays_equal(const int a[], const int b[], int n) {
    for (int i = 0; i < n; i++) {
        if (a[i] != b[i]) {
            return 0;  // 不相等
        }
    }
    return 1;  // 全部相等
}

int main(void) {
    int a[] = {1, 2, 3, 4, 5};
    int b[] = {1, 2, 3, 4, 5};
    int c[] = {1, 2, 3, 4, 6};

    printf("a == b: %s\n", arrays_equal(a, b, 5) ? "是" : "否");
    printf("a == c: %s\n", arrays_equal(a, c, 5) ? "是" : "否");

    return 0;
}
```

### 2.2 字典序比较

类似于字符串比较，逐元素按字典序比较。

```c
// 返回值: <0 表示a<b, 0表示相等, >0 表示a>b
int array_compare(const int a[], int na, const int b[], int nb) {
    int min_len = na < nb ? na : nb;

    for (int i = 0; i < min_len; i++) {
        if (a[i] != b[i]) {
            return a[i] - b[i];
        }
    }

    // 前面都相等，较短的数组"小于"较长的
    return na - nb;
}
```

```c
int a[] = {1, 2, 3};
int b[] = {1, 2, 4};
int c[] = {1, 2, 3, 4};

printf("%d\n", array_compare(a, 3, b, 3));  // -1 (3 < 4)
printf("%d\n", array_compare(a, 3, c, 4));  // -1 (a更短)
```

## 3. 使用 memcmp

`memcmp` 按字节比较两块内存。

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    int a[] = {1, 2, 3, 4, 5};
    int b[] = {1, 2, 3, 4, 5};
    int c[] = {1, 2, 3, 4, 6};

    int result1 = memcmp(a, b, sizeof(a));
    int result2 = memcmp(a, c, sizeof(a));

    printf("a 和 b: %s\n", result1 == 0 ? "相等" : "不相等");
    printf("a 和 c: %s\n", result2 == 0 ? "相等" : "不相等");
    printf("memcmp(a,c) = %d\n", result2);

    return 0;
}
```

### 3.1 memcmp 的返回值

- 返回 `0`：两块内存内容完全相同
- 返回 `< 0`：第一个不同字节，a 的值小于 b
- 返回 `> 0`：第一个不同字节，a 的值大于 b

### 3.2 memcmp 的注意事项

```c
// memcmp按字节比较，可能因填充字节产生误判
typedef struct {
    char c;    // 1字节
    // 3字节填充（对齐）
    int  n;    // 4字节
} S;

S s1 = {'a', 1};
S s2 = {'a', 1};

// 结构体中填充字节的值不确定，memcmp可能不为0
// 安全做法：逐字段比较，而非memcmp
```

## 4. 比较浮点数组

浮点数比较需要考虑精度问题。

```c
#include <math.h>

#define EPSILON 1e-9

int doubles_equal(double a, double b) {
    return fabs(a - b) < EPSILON;
}

int double_arrays_equal(const double a[], const double b[], int n) {
    for (int i = 0; i < n; i++) {
        if (!doubles_equal(a[i], b[i])) {
            return 0;
        }
    }
    return 1;
}
```

## 5. 查找数组差异

```c
// 找出两个数组中第一个不同元素的位置
int find_first_diff(const int a[], const int b[], int n) {
    for (int i = 0; i < n; i++) {
        if (a[i] != b[i]) return i;
    }
    return -1;  // 完全相同
}

// 统计两个数组中有多少个不同元素
int count_differences(const int a[], const int b[], int n) {
    int count = 0;
    for (int i = 0; i < n; i++) {
        if (a[i] != b[i]) count++;
    }
    return count;
}
```

---

## 重点总结

- 数组**不能用 `==` 比较内容**，比较的是地址
- 需要**逐元素比较**或使用 `memcmp`
- `memcmp` 按字节比较，注意**填充字节**问题
- 浮点数组不能用 `memcmp`，需考虑精度
- 结构体数组建议**逐字段比较**而非 `memcmp`
