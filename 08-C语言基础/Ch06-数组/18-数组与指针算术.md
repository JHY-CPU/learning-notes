# 数组与指针算术

## 1. 核心关系

`arr[i]` 本质上是 `*(arr + i)` 的语法糖。理解这一关系对掌握C语言数组至关重要。

```c
int arr[5] = {10, 20, 30, 40, 50};

// 以下表达式完全等价
arr[2]           // 下标访问
*(arr + 2)       // 指针算术 + 解引用
*(2 + arr)       // 加法交换律
2[arr]           // 合法但不要这样写！
```

## 2. 指针加减运算

### 2.1 指针 + 整数

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p = arr;  // p 指向 arr[0]

p = p + 1;     // p 指向 arr[1]，地址增加 sizeof(int)
p = p + 3;     // p 指向 arr[4]
```

**步长规则**：`ptr + n` 实际前进 `n × sizeof(*ptr)` 字节。

```c
int    *pi = arr;  pi + 1 前进 4 字节 (sizeof(int))
char   *pc = str;  pc + 1 前进 1 字节 (sizeof(char))
double *pd = dbl;  pd + 1 前进 8 字节 (sizeof(double))
```

### 2.2 指针 - 整数

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p = &arr[4];  // p 指向 arr[4]
p = p - 2;          // p 指向 arr[2]
```

### 2.3 指针 - 指针

两个指向同一数组的指针相减，得到**元素个数差**。

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p1 = &arr[1];
int *p2 = &arr[4];

ptrdiff_t diff = p2 - p1;  // diff = 3
printf("相差 %td 个元素\n", diff);
```

> 指向不同数组的指针相减是**未定义行为**。

### 2.4 指针比较

```c
int arr[5] = {10, 20, 30, 40, 50};
int *p1 = &arr[1];
int *p2 = &arr[3];

if (p1 < p2)  // p1 在 p2 之前
    printf("p1 在 p2 前面\n");
```

## 3. 用指针遍历数组

### 3.1 指针自增遍历

```c
#include <stdio.h>

int main(void) {
    int arr[] = {10, 20, 30, 40, 50};
    int len = sizeof(arr) / sizeof(arr[0]);

    // 方式1：指针自增
    for (int *p = arr; p < arr + len; p++) {
        printf("%d ", *p);
    }
    printf("\n");

    // 方式2：首尾指针
    int *begin = arr;
    int *end = arr + len;
    while (begin < end) {
        printf("%d ", *begin);
        begin++;
    }
    printf("\n");

    return 0;
}
```

### 3.2 下标与指针的等价性

```c
int arr[5] = {10, 20, 30, 40, 50};

// 下标方式
for (int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}

// 指针方式
for (int *p = arr; p < arr + 5; p++) {
    printf("%d ", *p);
}

// 混合方式：用指针+下标
int *p = arr;
for (int i = 0; i < 5; i++) {
    printf("%d ", p[i]);  // p[i] 等价于 *(p+i)
}
```

## 4. 指针算术在函数中的应用

```c
#include <stdio.h>

// 使用指针遍历数组的函数
int sum(const int *begin, const int *end) {
    int total = 0;
    while (begin < end) {
        total += *begin;
        begin++;
    }
    return total;
}

void fill(int *begin, int *end, int value) {
    while (begin < end) {
        *begin = value;
        begin++;
    }
}

int main(void) {
    int arr[] = {1, 2, 3, 4, 5};
    int len = sizeof(arr) / sizeof(arr[0]);

    printf("sum = %d\n", sum(arr, arr + len));  // 15

    fill(arr, arr + 3, 0);  // 前3个元素填0
    // arr = {0, 0, 0, 4, 5}

    return 0;
}
```

## 5. const 指针与数组

```c
const int arr[] = {1, 2, 3, 4, 5};

const int *p = arr;        // p指向const int，不能通过p修改
// *p = 10;                // 编译错误

int *const q = (int *)arr; // q是const指针，q不能指向别处
*q = 10;                   // 可以修改（但arr是const，实际上不应该）
```

## 6. 指针算术的限制

```c
int arr[5] = {10, 20, 30, 40, 50};

// 合法
int *p1 = arr;          // 指向第0个元素
int *p2 = arr + 5;      // 指向"最后一个元素之后"（可用于比较，不可解引用）
int *p3 = arr + 2;      // 指向第2个元素

// 非法（未定义行为）
int *p4 = arr - 1;      // 指向数组之前
int *p5 = arr + 6;      // 超过"尾后"位置
int *p6 = arr + 100;    // 严重越界
```

**允许的指针位置：**
```
arr[0]  arr[1]  arr[2]  arr[3]  arr[4]   (尾后)
  ↑       ↑       ↑       ↑       ↑        ↑
  p=arr   p+1     p+2     p+3     p+4      p+5 (合法但不能解引用)
```

---

## 重点总结

- `arr[i]` 等价于 `*(arr + i)`，下标访问是语法糖
- 指针加 `n` 前进 `n × sizeof(类型)` 字节
- 指针相减得到**元素个数差**，前提是指向同一数组
- 指针可以指向**数组末尾之后一位**，但不能解引用
- 指针遍历和下标遍历在性能上通常无差异，编译器会优化
