# 动态数组实现

## 1. 概述

C语言的数组大小固定，但通过 `malloc`/`realloc` 可以模拟**动态数组**（类似C++的 `std::vector`）。

核心思路：维护一个堆上分配的数组，在容量不足时**扩大容量**。

## 2. 基本数据结构

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#define INITIAL_CAPACITY 4

typedef struct {
    int *data;      // 数据存储区
    int size;       // 当前元素个数
    int capacity;   // 当前容量
} DynamicArray;
```

## 3. 核心操作实现

### 3.1 创建与销毁

```c
DynamicArray *da_create(void) {
    DynamicArray *da = malloc(sizeof(DynamicArray));
    if (!da) return NULL;

    da->data = malloc(INITIAL_CAPACITY * sizeof(int));
    if (!da->data) {
        free(da);
        return NULL;
    }

    da->size = 0;
    da->capacity = INITIAL_CAPACITY;
    return da;
}

void da_destroy(DynamicArray *da) {
    if (da) {
        free(da->data);
        free(da);
    }
}
```

### 3.2 扩容

```c
// 容量不足时扩大为原来的2倍
static int da_ensure_capacity(DynamicArray *da) {
    if (da->size < da->capacity) return 1;  // 容量足够

    int new_capacity = da->capacity * 2;
    int *new_data = realloc(da->data, new_capacity * sizeof(int));
    if (!new_data) return 0;  // 扩容失败

    da->data = new_data;
    da->capacity = new_capacity;
    return 1;
}
```

### 3.3 添加元素

```c
// 尾部添加
int da_push(DynamicArray *da, int value) {
    if (!da_ensure_capacity(da)) return 0;
    da->data[da->size++] = value;
    return 1;
}

// 指定位置插入
int da_insert(DynamicArray *da, int index, int value) {
    if (index < 0 || index > da->size) return 0;
    if (!da_ensure_capacity(da)) return 0;

    // 后移元素
    for (int i = da->size; i > index; i--) {
        da->data[i] = da->data[i - 1];
    }

    da->data[index] = value;
    da->size++;
    return 1;
}
```

### 3.4 删除元素

```c
// 尾部删除
int da_pop(DynamicArray *da) {
    if (da->size == 0) return 0;
    da->size--;
    return 1;
}

// 指定位置删除
int da_remove_at(DynamicArray *da, int index) {
    if (index < 0 || index >= da->size) return 0;

    for (int i = index; i < da->size - 1; i++) {
        da->data[i] = da->data[i + 1];
    }
    da->size--;
    return 1;
}
```

### 3.5 访问与查询

```c
// 获取元素（带边界检查）
int da_get(const DynamicArray *da, int index, int *out_value) {
    if (index < 0 || index >= da->size) return 0;
    *out_value = da->data[index];
    return 1;
}

// 设置元素
int da_set(DynamicArray *da, int index, int value) {
    if (index < 0 || index >= da->size) return 0;
    da->data[index] = value;
    return 1;
}

// 获取大小
int da_size(const DynamicArray *da) {
    return da->size;
}
```

## 4. 缩容策略

当元素数量远小于容量时，可以缩小容量以节省内存。

```c
void da_shrink_to_fit(DynamicArray *da) {
    if (da->size == da->capacity) return;

    int *new_data = realloc(da->data, da->size * sizeof(int));
    if (new_data) {
        da->data = new_data;
        da->capacity = da->size;
    }
}
```

## 5. 完整使用示例

```c
#include <stdio.h>
#include <stdlib.h>

// ... (上述所有函数定义)

int main(void) {
    DynamicArray *da = da_create();

    // 添加元素
    for (int i = 0; i < 10; i++) {
        da_push(da, (i + 1) * 10);
    }

    printf("容量: %d, 大小: %d\n", da->capacity, da->size);

    // 遍历
    printf("元素: ");
    for (int i = 0; i < da_size(da); i++) {
        int val;
        da_get(da, i, &val);
        printf("%d ", val);
    }
    printf("\n");

    // 插入和删除
    da_insert(da, 3, 35);   // 在下标3处插入35
    da_remove_at(da, 7);    // 删除下标7的元素

    // 缩容
    da_shrink_to_fit(da);
    printf("缩容后 - 容量: %d, 大小: %d\n", da->capacity, da->size);

    da_destroy(da);
    return 0;
}
```

## 6. 扩容策略分析

| 策略 | 说明 | 摊还复杂度 |
|------|------|-----------|
| 每次+1 | `realloc`每次加一个位置 | O(n) |
| 每次翻倍 | `capacity *= 2` | **O(1)** |
| 每次+固定值 | `capacity += 100` | O(n) |
| 黄金比例 | `capacity *= 1.618` | O(1) |

**翻倍策略**的摊还时间复杂度为O(1)，是最常用的策略。

```
容量变化: 4 → 8 → 16 → 32 → 64 → ...
虽然偶尔需要O(n)的扩容，但平均每次push仅为O(1)
```

---

## 重点总结

- 动态数组的核心：`malloc`初始分配 + `realloc`按需扩容
- **翻倍扩容**保证摊还O(1)的时间复杂度
- `realloc`可能移动内存，返回新地址，旧指针自动失效
- 必须提供 `destroy` 函数释放内存
- 通过**封装结构体**，可以实现安全的动态数组
