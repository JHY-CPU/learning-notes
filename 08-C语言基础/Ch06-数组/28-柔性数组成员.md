# 柔性数组成员（Flexible Array Member）

## 1. 概述

**柔性数组成员**是C99引入的特性，允许结构体的最后一个成员是**不指定大小的数组**。

```c
struct Packet {
    int length;      // 数据长度
    char data[];     // 柔性数组成员（必须是最后一个成员）
};
```

## 2. 基本用法

### 2.1 定义与内存分配

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

struct Packet {
    int length;
    char data[];
};

int main(void) {
    int data_len = 100;

    // 分配：结构体大小 + 柔性数组的大小
    struct Packet *pkt = malloc(sizeof(struct Packet) + data_len);
    if (!pkt) {
        fprintf(stderr, "内存分配失败\n");
        return 1;
    }

    pkt->length = data_len;
    memcpy(pkt->data, "Hello, Flexible Array Member!", 30);

    printf("长度: %d\n", pkt->length);
    printf("数据: %s\n", pkt->data);

    free(pkt);
    return 0;
}
```

### 2.2 内存布局

```
sizeof(struct Packet) = 4 (仅int length，柔性数组不计入)

分配 sizeof(struct Packet) + 100 字节后：

┌─────────────────────────────────────────────┐
│  int length (4字节)  │  char data[100字节]   │
│     pkt->length      │     pkt->data        │
└─────────────────────────────────────────────┘
  整个结构体在一块连续内存中
```

## 3. 与指针成员的对比

### 3.1 指针方式（两次分配）

```c
struct Packet_Ptr {
    int length;
    char *data;  // 指针成员
};

// 需要两次 malloc
struct Packet_Ptr *pkt = malloc(sizeof(struct Packet_Ptr));
pkt->data = malloc(100);
// 内存不连续，两次分配

// 需要两次 free
free(pkt->data);
free(pkt);
```

### 3.2 柔性数组方式（一次分配）

```c
struct Packet {
    int length;
    char data[];
};

// 只需一次 malloc
struct Packet *pkt = malloc(sizeof(struct Packet) + 100);
// 内存连续，一次分配

// 只需一次 free
free(pkt);
```

### 3.3 对比总结

| 特性 | 柔性数组 | 指针成员 |
|------|---------|---------|
| 内存分配 | 一次 malloc | 两次 malloc |
| 内存释放 | 一次 free | 两次 free |
| 内存布局 | 连续 | 不连续 |
| 缓存友好 | 好 | 差 |
| 内存碎片 | 少 | 多 |
| sizeof | 不含柔性数组 | 含指针(4/8字节) |

## 4. 实际应用场景

### 4.1 网络数据包

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <stdint.h>

typedef struct {
    uint32_t src_ip;
    uint32_t dst_ip;
    uint16_t src_port;
    uint16_t dst_port;
    uint32_t seq_num;
    uint32_t ack_num;
    uint16_t length;    // payload长度
    uint8_t  payload[]; // 柔性数组
} TcpPacket;

TcpPacket *create_packet(uint16_t payload_len) {
    TcpPacket *pkt = malloc(sizeof(TcpPacket) + payload_len);
    if (!pkt) return NULL;
    pkt->length = payload_len;
    memset(pkt->payload, 0, payload_len);
    return pkt;
}

void destroy_packet(TcpPacket *pkt) {
    free(pkt);  // 一次释放全部内存
}
```

### 4.2 可变长度消息

```c
typedef struct {
    int type;       // 消息类型
    int length;     // 消息体长度
    char body[];    // 消息体
} Message;

Message *create_message(int type, const char *text) {
    int len = strlen(text) + 1;
    Message *msg = malloc(sizeof(Message) + len);
    msg->type = type;
    msg->length = len;
    memcpy(msg->body, text, len);
    return msg;
}
```

### 4.3 图像像素数据

```c
typedef struct {
    int width;
    int height;
    int channels;    // 1=灰度, 3=RGB, 4=RGBA
    uint8_t pixels[]; // width * height * channels
} Image;

Image *create_image(int w, int h, int ch) {
    size_t data_size = (size_t)w * h * ch;
    Image *img = malloc(sizeof(Image) + data_size);
    img->width = w;
    img->height = h;
    img->channels = ch;
    memset(img->pixels, 0, data_size);
    return img;
}
```

## 5. 注意事项

### 5.1 语法限制

```c
// 必须是结构体的最后一个成员
struct Bad {
    char data[];  // 必须是最后一个！
    int length;   // 编译错误
};

// 结构体中只能有一个柔性数组成员
struct Bad2 {
    char data1[];
    char data2[];  // 编译错误
};

// 不能作为数组的元素
// struct Packet arr[10];  // 编译错误
```

### 5.2 sizeof 不包含柔性数组

```c
struct Packet {
    int length;
    char data[];
};

printf("%zu\n", sizeof(struct Packet));  // 输出 4 (仅int的大小)
```

### 5.3 不能初始化

```c
// 柔性数组成员不能在定义时初始化
struct Packet pkt = {100, "hello"};  // 编译错误或未定义行为
```

## 6. 完整示例

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct {
    int id;
    int name_len;
    char name[];
} Student;

Student *create_student(int id, const char *name) {
    int len = strlen(name) + 1;
    Student *s = malloc(sizeof(Student) + len);
    s->id = id;
    s->name_len = len;
    memcpy(s->name, name, len);
    return s;
}

void print_student(const Student *s) {
    printf("ID: %d, Name: %s\n", s->id, s->name);
}

int main(void) {
    Student *s1 = create_student(1, "Alice");
    Student *s2 = create_student(2, "Bob Smith");

    print_student(s1);
    print_student(s2);

    free(s1);
    free(s2);
    return 0;
}
```

---

## 重点总结

- 柔性数组成员：结构体末尾的**不定长数组**，C99引入
- 内存一次分配，连续存储，缓存友好
- **必须是结构体的最后一个成员**
- `sizeof` **不包含**柔性数组的大小
- 相比指针成员，减少内存分配次数和碎片
- 适用于网络协议、消息传递、可变长度数据等场景
