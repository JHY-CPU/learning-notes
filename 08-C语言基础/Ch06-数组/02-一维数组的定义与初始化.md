# 一维数组的定义与初始化

## 1. 一维数组的定义语法

```c
数据类型 数组名[数组长度];
```

- **数据类型**：数组中每个元素的类型
- **数组名**：遵循标识符命名规则
- **数组长度**：必须是**常量表达式**（C89/C90）或**整型表达式**（C99 VLA）

```c
// 合法定义
int    a[10];           // 10个int
double b[5];            // 5个double
char   c[100];          // 100个char

#define SIZE 20
int d[SIZE];            // 宏常量作为长度

enum { MAX = 50 };
int e[MAX];             // 枚举常量作为长度
```

### 1.1 定义时的常见错误

```c
// 错误：长度不能是C89中的变量
int n = 10;
int arr[n];             // C89错误，C99中VLA合法

// 错误：长度不能为负数或零（非VLA场景）
int b[-5];              // 编译错误
int c[0];               // C89中长度为0是未定义行为

// 错误：数组长度必须是整型
int d[3.5];             // 编译错误，不能是浮点数
```

## 2. 数组的初始化方式

### 2.1 完全初始化

列出所有元素的初始值，用花括号 `{}` 包裹，值之间用逗号分隔。

```c
int a[5] = {1, 2, 3, 4, 5};

// 编译器自动推断长度
int b[] = {10, 20, 30};    // 等价于 int b[3] = {10, 20, 30};
```

> **注意**：省略长度时，编译器根据初始值个数确定数组长度。

### 2.2 部分初始化

只提供部分初始值，**剩余元素自动初始化为0**。

```c
int a[5] = {1, 2};
// 等价于 int a[5] = {1, 2, 0, 0, 0};

int b[10] = {0};
// 所有元素都为0 —— 这是最常用的清零写法
```

**部分初始化的典型用法：**

```c
// 初始化计数器数组，全部清零
int count[26] = {0};

// 仅设置前几个值
int days[12] = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
```

### 2.3 零初始化

将数组所有元素初始化为0。

```c
// 方法1：全部写0
int a[5] = {0, 0, 0, 0, 0};

// 方法2：只写第一个0（推荐）
int b[5] = {0};

// 方法3：C99指定初始化
int c[5] = {[0 ... 4] = 0};  // GCC扩展语法

// 方法4：memset
int d[5];
memset(d, 0, sizeof(d));
```

### 2.4 C99 指定初始化器（Designated Initializer）

C99标准引入了**指定初始化器**，可以通过下标指定特定元素的值。

```c
// 指定下标初始化
int a[10] = {[2] = 10, [5] = 20, [8] = 30};
// a[2]=10, a[5]=20, a[8]=30, 其余为0

// 混合使用
int b[5] = {1, [3] = 40, 50};
// b[0]=1, b[3]=40, b[4]=50, 其余为0

// 范围初始化（GCC扩展，非标准C）
int c[10] = {[0 ... 4] = 1, [5 ... 9] = 2};
```

**实际应用场景：**

```c
// 初始化稀疏数组
int sparse[1000] = {[0] = 1, [100] = 2, [500] = 3, [999] = 4};

// 初始化查找表
int error_code[10] = {
    [0] = 0,        // 成功
    [1] = -1,       // 文件未找到
    [2] = -2,       // 权限不足
    [3] = -3,       // 内存不足
    [9] = -99       // 未知错误
};
```

## 3. 各种初始化方式对比

```c
#include <stdio.h>

int main(void) {
    // 完全初始化
    int a[5] = {1, 2, 3, 4, 5};

    // 部分初始化
    int b[5] = {10, 20};

    // 零初始化
    int c[5] = {0};

    // 自动推断长度
    int d[] = {100, 200, 300};

    // C99指定初始化
    int e[5] = {[1] = 50, [4] = 99};

    printf("a: ");
    for (int i = 0; i < 5; i++) printf("%d ", a[i]);
    printf("\n");  // a: 1 2 3 4 5

    printf("b: ");
    for (int i = 0; i < 5; i++) printf("%d ", b[i]);
    printf("\n");  // b: 10 20 0 0 0

    printf("c: ");
    for (int i = 0; i < 5; i++) printf("%d ", c[i]);
    printf("\n");  // c: 0 0 0 0 0

    printf("d的长度: %zu\n", sizeof(d) / sizeof(d[0]));  // 3

    printf("e: ");
    for (int i = 0; i < 5; i++) printf("%d ", e[i]);
    printf("\n");  // e: 0 50 0 0 99

    return 0;
}
```

## 4. 初始化的本质

```c
int a[5] = {1, 2, 3, 4, 5};
```

编译器在生成代码时，会将初始值写入数组对应的内存区域。对于全局数组和静态数组，初始化值在**编译时**确定；对于局部数组，初始化在**运行时**执行赋值操作。

```c
// 全局数组 —— 存储在数据段，编译时初始化
int global_arr[5] = {1, 2, 3, 4, 5};

void func(void) {
    // 静态数组 —— 存储在数据段，编译时初始化
    static int static_arr[5] = {1, 2, 3, 4, 5};

    // 局部数组 —— 存储在栈上，运行时初始化
    int local_arr[5] = {1, 2, 3, 4, 5};
}
```

---

## 重点总结

- 定义语法：`类型 数组名[长度];`，长度必须是**常量表达式**（C89）
- **完全初始化**时可省略长度：`int a[] = {1,2,3};`
- **部分初始化**时未指定元素自动为0：`int a[5] = {1,2};` 后三个为0
- `int a[n] = {0};` 是**清零**数组的简洁写法
- C99 **指定初始化器**允许按下标初始化：`[index] = value`
- 注意：`int a[];` 不带长度且不带初始化是**非法**的
