# 数组查找算法

## 1. 线性查找（Linear Search）

### 1.1 原理

从数组第一个元素开始，逐个与目标值比较，直到找到或遍历完整个数组。

### 1.2 实现

```c
#include <stdio.h>

int linear_search(const int arr[], int n, int target) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == target) {
            return i;  // 返回下标
        }
    }
    return -1;  // 未找到
}

// 返回所有匹配位置
int linear_search_all(const int arr[], int n, int target,
                      int indices[], int max_results) {
    int count = 0;
    for (int i = 0; i < n && count < max_results; i++) {
        if (arr[i] == target) {
            indices[count++] = i;
        }
    }
    return count;  // 返回找到的个数
}
```

### 1.3 复杂度分析

| 指标 | 值 |
|------|-----|
| 最好时间 | O(1)（第一个就是） |
| 最坏时间 | O(n)（在末尾或不存在） |
| 平均时间 | O(n/2) ≈ O(n) |
| 空间 | O(1) |
| 前提条件 | **无**，适用于任意数组 |

## 2. 二分查找（Binary Search）

### 2.1 前提条件

**数组必须是有序的！**

### 2.2 原理

每次取中间元素与目标值比较：
- 相等：找到
- 目标值 < 中间值：在左半部分继续
- 目标值 > 中间值：在右半部分继续

每次比较将搜索范围缩小一半。

```
在有序数组 [2, 5, 8, 12, 16, 23, 38, 56, 72, 91] 中查找 23

第1次: [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]
                    ↑mid=16, 23>16, 搜索右半

第2次:                      [23, 38, 56, 72, 91]
                               ↑mid=56, 23<56, 搜索左半

第3次:                      [23, 38]
                               ↑mid=23, 找到！下标5
```

### 2.3 实现

**迭代版本：**

```c
int binary_search(const int arr[], int n, int target) {
    int low = 0, high = n - 1;

    while (low <= high) {
        int mid = low + (high - low) / 2;  // 防止溢出

        if (arr[mid] == target)
            return mid;
        else if (arr[mid] < target)
            low = mid + 1;
        else
            high = mid - 1;
    }

    return -1;
}
```

**递归版本：**

```c
int binary_search_rec(const int arr[], int low, int high, int target) {
    if (low > high) return -1;

    int mid = low + (high - low) / 2;

    if (arr[mid] == target)
        return mid;
    else if (arr[mid] < target)
        return binary_search_rec(arr, mid + 1, high, target);
    else
        return binary_search_rec(arr, low, mid - 1, target);
}
```

### 2.4 防止整数溢出

```c
// 错误写法：可能溢出
int mid = (low + high) / 2;

// 正确写法：避免溢出
int mid = low + (high - low) / 2;
```

当 `low` 和 `high` 都很大时，`low + high` 可能超过 `int` 的范围。

### 2.5 复杂度分析

| 指标 | 值 |
|------|-----|
| 最好时间 | O(1) |
| 最坏时间 | O(log n) |
| 平均时间 | O(log n) |
| 空间 | O(1) 迭代 / O(log n) 递归 |
| 前提条件 | **数组必须有序** |

## 3. 查找第一个/最后一个出现位置

```c
// 查找第一个 >= target 的位置
int lower_bound(const int arr[], int n, int target) {
    int low = 0, high = n;
    while (low < high) {
        int mid = low + (high - low) / 2;
        if (arr[mid] < target)
            low = mid + 1;
        else
            high = mid;
    }
    return low;
}

// 查找第一个 > target 的位置
int upper_bound(const int arr[], int n, int target) {
    int low = 0, high = n;
    while (low < high) {
        int mid = low + (high - low) / 2;
        if (arr[mid] <= target)
            low = mid + 1;
        else
            high = mid;
    }
    return low;
}
```

## 4. 性能对比

```c
#include <stdio.h>
#include <time.h>

int linear_search(const int arr[], int n, int target);
int binary_search(const int arr[], int n, int target);

int main(void) {
    int n = 100000;
    int arr[n];
    for (int i = 0; i < n; i++) arr[i] = i * 2;  // 有序偶数序列

    int target = arr[n - 1];  // 最坏情况

    clock_t t1 = clock();
    for (int i = 0; i < 1000; i++)
        linear_search(arr, n, target);
    clock_t t2 = clock();

    clock_t t3 = clock();
    for (int i = 0; i < 1000; i++)
        binary_search(arr, n, target);
    clock_t t4 = clock();

    printf("线性查找: %.3f ms\n",
           (double)(t2 - t1) / CLOCKS_PER_SEC * 1000);
    printf("二分查找: %.3f ms\n",
           (double)(t4 - t3) / CLOCKS_PER_SEC * 1000);

    return 0;
}
```

---

## 重点总结

- **线性查找**：无需排序，O(n)，简单通用
- **二分查找**：需要有序数组，O(log n)，效率极高
- 二分查找的 `mid = low + (high - low) / 2` 防止整数溢出
- 对有序数组，二分查找是首选
- C标准库提供 `bsearch()` 函数用于二分查找
