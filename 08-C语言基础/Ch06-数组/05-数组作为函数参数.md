# 数组作为函数参数

## 1. 数组退化为指针

当数组作为函数参数传递时，**实际传递的是数组首元素的地址**（指针），而不是整个数组的副本。这个现象称为**数组退化（Array Decay）**。

```c
void func(int arr[5]) {
    // arr 在这里不是数组，而是一个 int* 指针
    printf("sizeof(arr) = %zu\n", sizeof(arr));
    // 在64位系统上输出 8（指针大小），而不是 20（5个int）
}

int main(void) {
    int a[5] = {1, 2, 3, 4, 5};
    printf("sizeof(a) = %zu\n", sizeof(a));  // 20
    func(a);
    return 0;
}
```

### 1.1 参数声明的等价写法

以下四种函数声明**完全等价**，编译器都会将参数视为 `int *`：

```c
void func(int arr[]);       // 写法1：不指定大小
void func(int arr[10]);     // 写法2：指定大小（被忽略）
void func(int arr[100]);    // 写法3：大小被忽略，仍是int*
void func(int *arr);        // 写法4：显式指针声明
```

> **注意**：`int arr[10]` 中的 `10` 会被编译器忽略，不影响参数类型。

## 2. 传递数组长度

由于函数内无法通过 `sizeof` 获取数组长度，**必须额外传递长度参数**。

```c
#include <stdio.h>

// 正确做法：同时传递数组和长度
void print_array(const int arr[], int len) {
    for (int i = 0; i < len; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main(void) {
    int data[] = {10, 20, 30, 40, 50};
    int len = sizeof(data) / sizeof(data[0]);

    print_array(data, len);

    return 0;
}
```

### 2.1 常见模式：指针 + 长度

```c
// 求数组元素之和
int sum(const int *arr, int n) {
    int total = 0;
    for (int i = 0; i < n; i++) {
        total += arr[i];
    }
    return total;
}

// 求数组最大值
int find_max(const int *arr, int n) {
    if (n <= 0) return 0;
    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
    }
    return max;
}
```

## 3. const 修饰数组参数

当函数**不需要修改**数组内容时，使用 `const` 修饰可以：
- 防止意外修改
- 允许传入 `const` 数组
- 编译器可进行更多优化

```c
// 只读函数，不会修改数组
void print_array(const int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
        // arr[i] = 0;  // 编译错误！const保护
    }
}

// 修改函数，需要修改数组
void zero_array(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        arr[i] = 0;  // 允许修改
    }
}
```

### 3.1 const 的实际意义

```c
#include <stdio.h>

// 计算数组平均值 —— 不修改数组
double average(const int arr[], int n) {
    if (n <= 0) return 0.0;
    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += arr[i];
    }
    return (double)sum / n;
}

// 数组元素乘以标量 —— 修改数组
void scale(int arr[], int n, int factor) {
    for (int i = 0; i < n; i++) {
        arr[i] *= factor;
    }
}

int main(void) {
    int data[] = {1, 2, 3, 4, 5};
    int len = 5;

    printf("平均值: %.2f\n", average(data, len));

    scale(data, len, 3);  // data变为{3,6,9,12,15}

    printf("平均值: %.2f\n", average(data, len));

    return 0;
}
```

## 4. 修改调用者数组的函数

由于数组退化为指针，函数内部的修改会**直接影响原始数组**。

```c
#include <stdio.h>

void reverse(int arr[], int n) {
    for (int i = 0; i < n / 2; i++) {
        int temp = arr[i];
        arr[i] = arr[n - 1 - i];
        arr[n - 1 - i] = temp;
    }
}

int main(void) {
    int data[] = {1, 2, 3, 4, 5};

    reverse(data, 5);

    for (int i = 0; i < 5; i++)
        printf("%d ", data[i]);   // 5 4 3 2 1

    return 0;
}
```

## 5. 指针形式的参数声明

```c
// 使用指针和长度参数
void fill(int *arr, int n, int value) {
    for (int i = 0; i < n; i++) {
        arr[i] = value;
    }
}

// 使用首尾指针（C++ STL风格）
int range_sum(const int *begin, const int *end) {
    int sum = 0;
    for (const int *p = begin; p != end; p++) {
        sum += *p;
    }
    return sum;
}

int main(void) {
    int arr[10];

    fill(arr, 10, 42);  // 全部填42

    int data[] = {1, 2, 3, 4, 5};
    int sum = range_sum(data, data + 5);
    printf("sum = %d\n", sum);  // 15

    return 0;
}
```

## 6. 返回数组修改结果

```c
// 原地修改：将数组每个元素取反
void negate(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        arr[i] = -arr[i];
    }
}

// 复制到新数组（调用者提供目标数组）
void square(const int src[], int dst[], int n) {
    for (int i = 0; i < n; i++) {
        dst[i] = src[i] * src[i];
    }
}

int main(void) {
    int a[] = {1, 2, 3, 4, 5};
    int b[5];

    square(a, b, 5);  // b = {1, 4, 9, 16, 25}

    negate(a, 5);     // a = {-1, -2, -3, -4, -5}

    return 0;
}
```

---

## 重点总结

- 数组传参时**退化为指针**，sizeof 无法获取数组长度
- 函数参数中 `int arr[]`、`int arr[N]`、`int *arr` **完全等价**
- 必须**额外传递长度参数**，这是C语言的标准做法
- 用 `const` 修饰**只读**数组参数，防止意外修改并提高代码可读性
- 函数内对数组的修改会**直接影响原始数组**（传址语义）
