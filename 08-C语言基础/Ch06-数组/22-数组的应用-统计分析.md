# 数组的应用：统计分析

## 1. 求和（Sum）

```c
#include <stdio.h>

int sum(const int arr[], int n) {
    int total = 0;
    for (int i = 0; i < n; i++) {
        total += arr[i];
    }
    return total;
}

// 指针版本
int sum_ptr(const int *begin, const int *end) {
    int total = 0;
    while (begin < end) {
        total += *begin++;
    }
    return total;
}
```

## 2. 求平均值（Mean）

```c
double mean(const int arr[], int n) {
    if (n <= 0) return 0.0;
    int total = sum(arr, n);
    return (double)total / n;
}
```

**注意**：先转为 `double` 再除法，避免整数除法截断。

## 3. 求最大值和最小值

```c
int find_max(const int arr[], int n) {
    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
    }
    return max;
}

int find_min(const int arr[], int n) {
    int min = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] < min) min = arr[i];
    }
    return min;
}

// 同时返回最大值和最小值
void find_min_max(const int arr[], int n, int *min, int *max) {
    *min = *max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] < *min) *min = arr[i];
        if (arr[i] > *max) *max = arr[i];
    }
}
```

## 4. 求方差与标准差

**方差** = 各数据与均值之差的平方的平均值
**标准差** = 方差的平方根

```c
#include <math.h>

double variance(const int arr[], int n) {
    if (n <= 1) return 0.0;

    double avg = mean(arr, n);
    double sum_sq = 0.0;

    for (int i = 0; i < n; i++) {
        double diff = arr[i] - avg;
        sum_sq += diff * diff;
    }

    return sum_sq / n;            // 总体方差
    // return sum_sq / (n - 1);   // 样本方差
}

double stddev(const int arr[], int n) {
    return sqrt(variance(arr, n));
}
```

## 5. 求中位数

中位数需要先排序，然后取中间值。

```c
#include <stdlib.h>

int compare_int(const void *a, const void *b) {
    return *(const int *)a - *(const int *)b;
}

double median(int arr[], int n) {
    // 需要一个副本，避免修改原数组
    int *copy = malloc(n * sizeof(int));
    for (int i = 0; i < n; i++) copy[i] = arr[i];

    qsort(copy, n, sizeof(int), compare_int);

    double result;
    if (n % 2 == 1)
        result = copy[n / 2];
    else
        result = (copy[n / 2 - 1] + copy[n / 2]) / 2.0;

    free(copy);
    return result;
}
```

## 6. 求众数（出现最频繁的值）

```c
int mode(const int arr[], int n) {
    int max_count = 0;
    int mode_val = arr[0];

    for (int i = 0; i < n; i++) {
        int count = 0;
        for (int j = 0; j < n; j++) {
            if (arr[j] == arr[i]) count++;
        }
        if (count > max_count) {
            max_count = count;
            mode_val = arr[i];
        }
    }
    return mode_val;
}
```

## 7. 完整统计分析示例

```c
#include <stdio.h>
#include <math.h>
#include <stdlib.h>

// ... (上述函数定义)

int main(void) {
    int data[] = {85, 92, 78, 90, 88, 76, 95, 82, 79, 91};
    int n = sizeof(data) / sizeof(data[0]);

    printf("数据: ");
    for (int i = 0; i < n; i++) printf("%d ", data[i]);
    printf("\n");

    printf("个数: %d\n", n);
    printf("总和: %d\n", sum(data, n));
    printf("平均: %.2f\n", mean(data, n));
    printf("最大: %d\n", find_max(data, n));
    printf("最小: %d\n", find_min(data, n));

    int min, max;
    find_min_max(data, n, &min, &max);
    printf("极差: %d\n", max - min);

    printf("方差: %.2f\n", variance(data, n));
    printf("标准差: %.2f\n", stddev(data, n));
    printf("中位数: %.1f\n", median(data, n));

    return 0;
}
```

输出：
```

个数: 10
总和: 856
平均: 85.60
最大: 95
最小: 76
极差: 19
方差: 36.84
标准差: 6.07
中位数: 86.5
```

---

## 重点总结

- 求和、求均值是所有统计分析的基础
- 方差衡量数据的**离散程度**，标准差与原数据同单位
- 中位数需要**排序**后取中间值
- 大数组的众数可以用**计数数组**优化（见直方图章节）
- 浮点运算注意**精度**问题
