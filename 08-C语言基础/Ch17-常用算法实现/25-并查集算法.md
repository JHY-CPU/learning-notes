# 并查集算法

## 1. 并查集基础

并查集（Union-Find / Disjoint Set Union）是一种用于处理不相交集合合并与查询的数据结构。支持两种操作：
- **Find**：查找元素所属集合的代表元素
- **Union**：合并两个集合

```c
#include <stdio.h>

#define MAXN 100005

int parent[MAXN];  // parent[i] 表示 i 的父节点
int rank_[MAXN];   // rank_[i] 表示以 i 为根的树的高度

// 初始化：每个元素自成一个集合
void init(int n) {
    for (int i = 0; i < n; i++) {
        parent[i] = i;
        rank_[i] = 0;
    }
}

// 查找（带路径压缩）
int find(int x) {
    if (parent[x] != x)
        parent[x] = find(parent[x]);  // 路径压缩
    return parent[x];
}

// 合并（按秩合并）
int unite(int x, int y) {
    int px = find(x), py = find(y);
    if (px == py) return 0;  // 已在同一集合
    if (rank_[px] < rank_[py]) { int t = px; px = py; py = t; }
    parent[py] = px;
    if (rank_[px] == rank_[py]) rank_[px]++;
    return 1;
}

// 判断是否在同一集合
int same(int x, int y) {
    return find(x) == find(y);
}
```

## 2. 并查集的优化

### 路径压缩（Path Compression）

```c
// 递归版
int findRecursive(int x) {
    if (parent[x] != x)
        parent[x] = findRecursive(parent[x]);
    return parent[x];
}

// 迭代版（两遍扫描法）
int findIterative(int x) {
    int root = x;
    while (parent[root] != root)
        root = parent[root];
    // 路径压缩：将路径上所有节点直接连到根
    while (x != root) {
        int next = parent[x];
        parent[x] = root;
        x = next;
    }
    return root;
}
```

### 按秩合并（Union by Rank）

```c
void unionByRank(int x, int y) {
    int px = find(x), py = find(y);
    if (px == py) return;
    // 将矮树合并到高树下
    if (rank_[px] < rank_[py]) {
        parent[px] = py;
    } else if (rank_[px] > rank_[py]) {
        parent[py] = px;
    } else {
        parent[py] = px;
        rank_[px]++;
    }
}
```

### 按大小合并（Union by Size）

```c
int size_[MAXN];

void initSize(int n) {
    for (int i = 0; i < n; i++) {
        parent[i] = i;
        size_[i] = 1;
    }
}

void unionBySize(int x, int y) {
    int px = find(x), py = find(y);
    if (px == py) return;
    if (size_[px] < size_[py]) { int t = px; px = py; py = t; }
    parent[py] = px;
    size_[px] += size_[py];
}

// 获取集合大小
int getSize(int x) { return size_[find(x)]; }
```

## 3. 连通分量计数

```c
int countComponents(int n, int edges[][2], int m) {
    init(n);
    int components = n;  // 初始n个连通分量
    for (int i = 0; i < m; i++) {
        if (unite(edges[i][0], edges[i][1]))
            components--;
    }
    return components;
}
```

## 4. 最小生成树（Kruskal算法）

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct { int u, v, w; } Edge;

int cmpEdge(const void *a, const void *b) {
    return ((Edge *)a)->w - ((Edge *)b)->w;
}

int kruskal(Edge edges[], int n, int m) {
    qsort(edges, m, sizeof(Edge), cmpEdge);
    init(n);
    int total = 0, cnt = 0;
    for (int i = 0; i < m && cnt < n - 1; i++) {
        if (unite(edges[i].u, edges[i].v)) {
            total += edges[i].w;
            cnt++;
        }
    }
    return (cnt == n - 1) ? total : -1;
}
```

## 5. 判断图中是否有环

```c
int hasCycle(int n, int edges[][2], int m) {
    init(n);
    for (int i = 0; i < m; i++) {
        if (find(edges[i][0]) == find(edges[i][1]))
            return 1;  // 两个端点已连通，加入此边会产生环
        unite(edges[i][0], edges[i][1]);
    }
    return 0;
}
```

## 6. 带权并查集

维护元素之间的相对关系。

```c
int parentW[MAXN];
double weight[MAXN];  // weight[x] 表示 x 到 parent[x] 的权值

void initWeighted(int n) {
    for (int i = 0; i < n; i++) {
        parentW[i] = i;
        weight[i] = 0;
    }
}

int findWeighted(int x) {
    if (parentW[x] != x) {
        int root = findWeighted(parentW[x]);
        weight[x] += weight[parentW[x]];  // 累加路径上的权值
        parentW[x] = root;
    }
    return parentW[x];
}

// 合并: value[x] + val = value[y]
void unionWeighted(int x, int y, double val) {
    int px = findWeighted(x), py = findWeighted(y);
    if (px == py) return;
    parentW[py] = px;
    weight[py] = weight[x] + val - weight[y];
}
```

## 7. 并查集应用总结

| 应用场景 | 说明 |
|---------|------|
| 连通分量 | 判断图的连通性、连通分量个数 |
| 最小生成树 | Kruskal算法 |
| 环检测 | 判断无向图是否有环 |
| 动态连通性 | 在线处理边的添加和连通性查询 |
| 最近公共祖先 | 离线LCA（Tarjan算法） |
| 等式约束 | 判断等式约束是否矛盾 |
| 图像处理 | 连通区域标记 |
