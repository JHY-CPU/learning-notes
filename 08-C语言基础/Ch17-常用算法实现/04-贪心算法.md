# 贪心算法

## 1. 贪心策略

贪心算法在每一步选择中都采取当前状态下最好或最优的选择，期望通过局部最优达到全局最优。贪心算法不一定能得到全局最优解，但在很多问题上确实有效。

### 适用条件
- **贪心选择性质**：一个全局最优解可以通过局部最优（贪心）选择来达到
- **最优子结构**：问题的最优解包含其子问题的最优解

## 2. 活动选择问题

有n个活动，每个活动有开始时间和结束时间，选出最多的互不重叠的活动。

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct { int start, end; } Activity;

int cmpActivity(const void *a, const void *b) {
    return ((Activity *)a)->end - ((Activity *)b)->end;
}

int activitySelection(Activity acts[], int n) {
    qsort(acts, n, sizeof(Activity), cmpActivity);
    int count = 1, lastEnd = acts[0].end;
    printf("选择活动: [%d, %d]\n", acts[0].start, acts[0].end);
    for (int i = 1; i < n; i++) {
        if (acts[i].start >= lastEnd) {
            count++;
            lastEnd = acts[i].end;
            printf("选择活动: [%d, %d]\n", acts[i].start, acts[i].end);
        }
    }
    return count;
}
```

## 3. 哈夫曼编码

哈夫曼编码是一种前缀编码，用于无损数据压缩。频率高的字符使用较短编码。

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct Node {
    char ch;
    int freq;
    struct Node *left, *right;
} Node;

Node *createNode(char ch, int freq) {
    Node *n = (Node *)malloc(sizeof(Node));
    n->ch = ch; n->freq = freq;
    n->left = n->right = NULL;
    return n;
}

// 简化版最小堆（实际应用中应使用完整堆实现）
Node *buildHuffmanTree(char chars[], int freqs[], int n) {
    Node **nodes = (Node **)malloc(n * sizeof(Node *));
    for (int i = 0; i < n; i++)
        nodes[i] = createNode(chars[i], freqs[i]);

    int size = n;
    while (size > 1) {
        // 找频率最小的两个节点
        int min1 = 0, min2 = 1;
        if (nodes[min1]->freq > nodes[min2]->freq) {
            int t = min1; min1 = min2; min2 = t;
        }
        for (int i = 2; i < size; i++) {
            if (nodes[i]->freq < nodes[min1]->freq) {
                min2 = min1; min1 = i;
            } else if (nodes[i]->freq < nodes[min2]->freq) {
                min2 = i;
            }
        }
        // 合并
        Node *parent = createNode('\0', nodes[min1]->freq + nodes[min2]->freq);
        parent->left = nodes[min1];
        parent->right = nodes[min2];
        nodes[min1] = parent;
        nodes[min2] = nodes[size - 1];
        size--;
    }
    Node *root = nodes[0];
    free(nodes);
    return root;
}

void printCodes(Node *root, char code[], int depth) {
    if (!root) return;
    if (!root->left && !root->right) {
        code[depth] = '\0';
        printf("字符 '%c': 频率 %d, 编码 %s\n", root->ch, root->freq, code);
        return;
    }
    code[depth] = '0';
    printCodes(root->left, code, depth + 1);
    code[depth] = '1';
    printCodes(root->right, code, depth + 1);
}
```

## 4. 区间调度问题（区间覆盖）

用最少的区间覆盖给定的范围。

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct { int start, end; } Interval;

int cmpInterval(const void *a, const void *b) {
    return ((Interval *)a)->start - ((Interval *)b)->start;
}

// 用最少的区间覆盖 [target_start, target_end]
int intervalCover(Interval intervals[], int n, int target_start, int target_end) {
    qsort(intervals, n, sizeof(Interval), cmpInterval);
    int count = 0, i = 0, cur_end = target_start;

    while (cur_end < target_end) {
        int farthest = cur_end;
        while (i < n && intervals[i].start <= cur_end) {
            if (intervals[i].end > farthest)
                farthest = intervals[i].end;
            i++;
        }
        if (farthest == cur_end) return -1; // 无法覆盖
        cur_end = farthest;
        count++;
    }
    return count;
}
```

## 5. 最小生成树

### Kruskal算法（基于贪心+并查集）

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct { int u, v, w; } Edge;

int cmpEdge(const void *a, const void *b) {
    return ((Edge *)a)->w - ((Edge *)b)->w;
}

int parent[10001], rank_[10001];

void initDSU(int n) {
    for (int i = 0; i < n; i++) { parent[i] = i; rank_[i] = 0; }
}

int find(int x) {
    if (parent[x] != x) parent[x] = find(parent[x]);
    return parent[x];
}

int unite(int x, int y) {
    int px = find(x), py = find(y);
    if (px == py) return 0;
    if (rank_[px] < rank_[py]) { int t = px; px = py; py = t; }
    parent[py] = px;
    if (rank_[px] == rank_[py]) rank_[px]++;
    return 1;
}

int kruskal(Edge edges[], int n, int m) {
    qsort(edges, m, sizeof(Edge), cmpEdge);
    initDSU(n);
    int total = 0, cnt = 0;
    for (int i = 0; i < m && cnt < n - 1; i++) {
        if (unite(edges[i].u, edges[i].v)) {
            total += edges[i].w;
            cnt++;
        }
    }
    return (cnt == n - 1) ? total : -1; // -1表示不连通
}
```

## 6. 贪心算法总结

| 问题 | 贪心策略 | 时间复杂度 |
|------|---------|-----------|
| 活动选择 | 按结束时间排序 | O(n log n) |
| 哈夫曼编码 | 每次合并频率最小的两个 | O(n log n) |
| 区间覆盖 | 贪心扩展最远覆盖 | O(n log n) |
| 最小生成树 | 按边权排序/选最短边 | O(E log E) |
| Dijkstra最短路 | 每次选最近未访问点 | O(V^2)或O(E log V) |

**关键提示**：使用贪心算法前必须证明其正确性，否则可能得到错误结果。
