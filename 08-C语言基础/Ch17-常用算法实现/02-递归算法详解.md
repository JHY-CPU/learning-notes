# 递归算法详解

## 1. 递归的基本概念

递归是一种在函数定义中调用自身的方法。递归将大问题分解为与原问题相同但规模更小的子问题，直到达到一个已知解的基线条件。

### 递归的两大要素
1. **递归终止条件（Base Case）**：问题可以直接求解的最小规模
2. **递归关系（Recursive Case）**：将大问题转化为小问题的方式

## 2. 递归设计的基本步骤

```c
// 递归函数的基本框架
返回类型 recursive(参数列表) {
    if (终止条件) {
        返回基本情况的解;    // 基线条件
    }
    // 将问题分解，递归调用
    return recursive(规模更小的参数);
}
```

## 3. 经典递归问题

### 3.1 阶乘计算

```c
#include <stdio.h>

// 递归计算阶乘: n! = n * (n-1)!
long long factorial(int n) {
    if (n <= 1) return 1;       // 基线条件
    return n * factorial(n - 1); // 递归关系
}

int main() {
    for (int i = 0; i <= 10; i++) {
        printf("%d! = %lld\n", i, factorial(i));
    }
    return 0;
}
```

### 3.2 汉诺塔问题

```c
#include <stdio.h>

int step = 0;

// 将n个盘子从from移动到to，借助aux
void hanoi(int n, char from, char aux, char to) {
    if (n == 1) {
        printf("第%d步: 将盘%d从%c移到%c\n", ++step, n, from, to);
        return;
    }
    hanoi(n - 1, from, to, aux);     // 将n-1个盘子移到辅助柱
    printf("第%d步: 将盘%d从%c移到%c\n", ++step, n, from, to);
    hanoi(n - 1, aux, from, to);     // 将n-1个盘子从辅助柱移到目标柱
}

int main() {
    hanoi(3, 'A', 'B', 'C');
    printf("共需要%d步\n", step);
    return 0;
}
```

## 4. 递推式与复杂度分析

### 主定理（Master Theorem）

对于形如 `T(n) = aT(n/b) + f(n)` 的递推式：
- 若 `f(n) = O(n^(log_b(a) - ε))`，则 `T(n) = Θ(n^(log_b(a)))`
- 若 `f(n) = Θ(n^(log_b(a)))`，则 `T(n) = Θ(n^(log_b(a)) * log n)`
- 若 `f(n) = Ω(n^(log_b(a) + ε))`，则 `T(n) = Θ(f(n))`

## 5. 记忆化搜索（Memoization）

递归中存在大量重复计算时，可以使用数组缓存已计算过的结果。

```c
#include <stdio.h>
#include <string.h>

#define MAXN 100

long long memo[MAXN];  // 记忆化数组

long long fib(int n) {
    if (n <= 1) return n;
    if (memo[n] != -1) return memo[n];  // 已计算过，直接返回
    memo[n] = fib(n - 1) + fib(n - 2);  // 计算并缓存
    return memo[n];
}

int main() {
    memset(memo, -1, sizeof(memo));  // 初始化为-1表示未计算
    for (int i = 0; i <= 40; i++) {
        printf("fib(%d) = %lld\n", i, fib(i));
    }
    return 0;
}
```

## 6. 尾递归优化

尾递归是指递归调用是函数的最后一个操作。编译器可以将尾递归优化为循环，避免栈溢出。

```c
// 普通递归计算阶乘（不是尾递归）
long long factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);  // 递归后还有乘法操作
}

// 尾递归版本（递归调用是最后操作）
long long factorial_tail(int n, long long acc) {
    if (n <= 1) return acc;
    return factorial_tail(n - 1, n * acc);  // 递归调用是最后操作
}

// 调用: factorial_tail(n, 1);

// 尾递归版斐波那契
long long fib_tail(int n, long long a, long long b) {
    if (n == 0) return a;
    if (n == 1) return b;
    return fib_tail(n - 1, b, a + b);
}

// 调用: fib_tail(n, 0, 1);
```

## 7. 递归转迭代

任何递归都可以用显式栈转为迭代，避免栈溢出风险。

```c
#include <stdio.h>

#define MAXN 1000

// 用栈模拟递归计算阶乘
typedef struct {
    int n;
    int state;  // 0: 第一次访问, 1: 已返回
} Frame;

long long factorial_iter(int n) {
    Frame stack[MAXN];
    int top = -1;
    long long result = 1;

    stack[++top] = (Frame){n, 0};

    while (top >= 0) {
        Frame *f = &stack[top];
        if (f->n <= 1) {
            top--;
            continue;
        }
        if (f->state == 0) {
            f->state = 1;
            stack[++top] = (Frame){f->n - 1, 0};
        } else {
            result *= f->n;
            top--;
        }
    }
    return result;
}
```

## 8. 递归的注意事项

1. **栈溢出风险**：深度过大的递归会导致栈溢出
2. **重复计算**：不做记忆化的递归可能有指数级复杂度
3. **尾递归优化**：优先使用尾递归形式
4. **调试技巧**：打印缩进可以帮助理解递归过程

```c
// 带调试信息的递归
void debug_recursion(int depth, const char *msg) {
    for (int i = 0; i < depth; i++) printf("  ");
    printf("%s\n", msg);
}
```
