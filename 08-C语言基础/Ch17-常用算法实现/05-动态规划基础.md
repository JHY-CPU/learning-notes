# 动态规划基础

## 1. 动态规划思想

动态规划（Dynamic Programming, DP）是一种通过把原问题分解为相对简单的子问题来求解复杂问题的方法。与分治法不同的是，DP的子问题往往**重叠**，通过存储子问题的解来避免重复计算。

### 两个核心性质
- **最优子结构**：问题的最优解包含子问题的最优解
- **重叠子问题**：递归求解时，相同的子问题被反复计算

## 2. DP的基本步骤

1. **定义状态**：用数组或变量表示子问题的解
2. **写出状态转移方程**：描述状态之间的递推关系
3. **确定边界条件**：初始状态的值
4. **确定计算顺序**：保证计算某状态时其依赖的状态已求出
5. **（可选）空间优化**：利用滚动数组等技巧降低空间复杂度

## 3. 斐波那契数列——DP入门

```c
#include <stdio.h>

// 朴素递归：O(2^n)，大量重复计算
long long fib_recursive(int n) {
    if (n <= 1) return n;
    return fib_recursive(n - 1) + fib_recursive(n - 2);
}

// DP解法：O(n)时间，O(n)空间
long long fib_dp(int n) {
    if (n <= 1) return n;
    long long dp[n + 1];
    dp[0] = 0; dp[1] = 1;
    for (int i = 2; i <= n; i++)
        dp[i] = dp[i - 1] + dp[i - 2];
    return dp[n];
}

// 空间优化：O(n)时间，O(1)空间
long long fib_optimized(int n) {
    if (n <= 1) return n;
    long long a = 0, b = 1, c;
    for (int i = 2; i <= n; i++) {
        c = a + b; a = b; b = c;
    }
    return c;
}
```

## 4. 爬楼梯问题

每次可以爬1或2个台阶，问爬到第n阶有多少种方法。

```c
#include <stdio.h>

// 状态转移: dp[i] = dp[i-1] + dp[i-2]
int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2, c;
    for (int i = 3; i <= n; i++) {
        c = a + b; a = b; b = c;
    }
    return b;
}
```

## 5. 最大子数组和

给定数组，找出和最大的连续子数组。

```c
#include <stdio.h>

// Kadane算法: O(n)
// dp[i] = 以arr[i]结尾的最大子数组和
// dp[i] = max(dp[i-1] + arr[i], arr[i])
int maxSubArray(int arr[], int n) {
    int maxSum = arr[0];
    int curSum = arr[0];
    for (int i = 1; i < n; i++) {
        if (curSum + arr[i] > arr[i])
            curSum = curSum + arr[i];
        else
            curSum = arr[i];
        if (curSum > maxSum)
            maxSum = curSum;
    }
    return maxSum;
}
```

## 6. 不同路径问题

从 m×n 网格左上角到右下角，每次只能向右或向下，有多少条路径？

```c
#include <stdio.h>

// 状态转移: dp[i][j] = dp[i-1][j] + dp[i][j-1]
int uniquePaths(int m, int n) {
    int dp[m][n];
    // 边界: 第一行和第一列都只有1条路径
    for (int i = 0; i < m; i++) dp[i][0] = 1;
    for (int j = 0; j < n; j++) dp[0][j] = 1;

    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1];

    return dp[m - 1][n - 1];
}

// 空间优化版本: 只用一行
int uniquePathsOptimized(int m, int n) {
    int dp[n];
    for (int j = 0; j < n; j++) dp[j] = 1;
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            dp[j] += dp[j - 1];
    return dp[n - 1];
}
```

## 7. 零钱兑换问题

给定面额数组，凑出目标金额的最少硬币数。

```c
#include <stdio.h>
#include <limits.h>

int coinChange(int coins[], int n, int amount) {
    int dp[amount + 1];
    dp[0] = 0;
    for (int i = 1; i <= amount; i++) {
        dp[i] = INT_MAX;
        for (int j = 0; j < n; j++) {
            if (coins[j] <= i && dp[i - coins[j]] != INT_MAX) {
                int val = dp[i - coins[j]] + 1;
                if (val < dp[i]) dp[i] = val;
            }
        }
    }
    return dp[amount] == INT_MAX ? -1 : dp[amount];
}
```

## 8. DP与贪心的区别

| 特征 | 动态规划 | 贪心算法 |
|------|---------|---------|
| 决策方式 | 考虑所有子问题 | 只做当前最优选择 |
| 子问题关系 | 重叠子问题 | 子问题独立 |
| 求解方式 | 自底向上或记忆化 | 一次性贪心选择 |
| 结果保证 | 一定得到最优解 | 不一定得到最优解 |

## 9. DP的两种实现方式

```c
// 自顶向下（记忆化搜索）
int dp_topdown(int n, int memo[]) {
    if (memo[n] != -1) return memo[n];
    // 递推计算
    memo[n] = dp_topdown(n-1, memo) + dp_topdown(n-2, memo);
    return memo[n];
}

// 自底向上（迭代）
int dp_bottomup(int n) {
    int dp[n + 1];
    dp[0] = 0; dp[1] = 1;
    for (int i = 2; i <= n; i++)
        dp[i] = dp[i-1] + dp[i-2];
    return dp[n];
}
```
