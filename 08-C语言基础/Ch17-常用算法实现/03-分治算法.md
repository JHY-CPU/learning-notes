# 分治算法

## 1. 分治思想

分治法（Divide and Conquer）的核心思想是将一个难以直接解决的大问题，分割成若干个规模较小的相同问题，逐个击破，分而治之。

### 三个步骤
1. **分解**：将原问题分解为若干个规模较小、相互独立的子问题
2. **求解**：递归地求解各子问题。若子问题规模足够小，则直接求解
3. **合并**：将各子问题的解合并为原问题的解

## 2. 归并排序（Merge Sort）

归并排序是分治思想的经典应用，时间复杂度为 O(n log n)。

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

void merge(int arr[], int temp[], int left, int mid, int right) {
    int i = left, j = mid + 1, k = left;
    while (i <= mid && j <= right) {
        if (arr[i] <= arr[j]) temp[k++] = arr[i++];
        else temp[k++] = arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    memcpy(arr + left, temp + left, (right - left + 1) * sizeof(int));
}

void mergeSort(int arr[], int temp[], int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    mergeSort(arr, temp, left, mid);
    mergeSort(arr, temp, mid + 1, right);
    merge(arr, temp, left, mid, right);
}

int main() {
    int arr[] = {38, 27, 43, 3, 9, 82, 10};
    int n = sizeof(arr) / sizeof(arr[0]);
    int *temp = (int *)malloc(n * sizeof(int));
    mergeSort(arr, temp, 0, n - 1);
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");
    free(temp);
    return 0;
}
```

## 3. 快速排序（Quick Sort）

快速排序通过选择一个基准元素，将数组分为两部分，左边小于基准，右边大于基准。

```c
#include <stdio.h>

void swap(int *a, int *b) {
    int t = *a; *a = *b; *b = t;
}

int partition(int arr[], int low, int high) {
    int pivot = arr[high];  // 选择最后一个元素作为基准
    int i = low - 1;
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(&arr[i], &arr[j]);
        }
    }
    swap(&arr[i + 1], &arr[high]);
    return i + 1;
}

void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

// 随机化快排：随机选择基准，避免最坏情况
#include <stdlib.h>
#include <time.h>

int randomizedPartition(int arr[], int low, int high) {
    int r = low + rand() % (high - low + 1);
    swap(&arr[r], &arr[high]);
    return partition(arr, low, high);
}
```

## 4. 大整数乘法（Karatsuba算法）

将大整数乘法从 O(n^2) 优化到 O(n^1.585)。

```c
#include <stdio.h>
#include <math.h>

// Karatsuba大整数乘法
// 将x和y各分为两部分: x = a*10^m + b, y = c*10^m + d
// x*y = ac*10^(2m) + (ad+bc)*10^m + bd
// 其中 ad+bc = (a+b)(c+d) - ac - bd (减少一次乘法)
long long karatsuba(long long x, long long y) {
    if (x < 10 || y < 10) return x * y;

    int n = (int)fmax(log10(x) + 1, log10(y) + 1);
    int m = n / 2;
    long long power = (long long)pow(10, m);

    long long a = x / power, b = x % power;
    long long c = y / power, d = y % power;

    long long ac = karatsuba(a, c);
    long long bd = karatsuba(b, d);
    long long adbc = karatsuba(a + b, c + d) - ac - bd;

    return ac * (long long)pow(10, 2 * m) + adbc * power + bd;
}
```

## 5. 最近点对问题

在平面上找出距离最近的两个点。

```c
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <float.h>

typedef struct { double x, y; } Point;

int cmpX(const void *a, const void *b) {
    double d = ((Point *)a)->x - ((Point *)b)->x;
    return (d > 0) - (d < 0);
}

int cmpY(const void *a, const void *b) {
    double d = ((Point *)a)->y - ((Point *)b)->y;
    return (d > 0) - (d < 0);
}

double dist(Point a, Point b) {
    return sqrt((a.x - b.x) * (a.x - b.x) + (a.y - b.y) * (a.y - b.y));
}

double bruteForce(Point pts[], int n) {
    double minD = DBL_MAX;
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            if (dist(pts[i], pts[j]) < minD)
                minD = dist(pts[i], pts[j]);
    return minD;
}

double stripClosest(Point strip[], int n, double d) {
    double minD = d;
    qsort(strip, n, sizeof(Point), cmpY);
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n && (strip[j].y - strip[i].y) < minD; j++)
            if (dist(strip[i], strip[j]) < minD)
                minD = dist(strip[i], strip[j]);
    return minD;
}

double closestUtil(Point pts[], int n) {
    if (n <= 3) return bruteForce(pts, n);
    int mid = n / 2;
    Point midPt = pts[mid];
    double dl = closestUtil(pts, mid);
    double dr = closestUtil(pts + mid, n - mid);
    double d = fmin(dl, dr);

    Point *strip = malloc(n * sizeof(Point));
    int j = 0;
    for (int i = 0; i < n; i++)
        if (fabs(pts[i].x - midPt.x) < d)
            strip[j++] = pts[i];
    double ds = stripClosest(strip, j, d);
    free(strip);
    return fmin(d, ds);
}
```

## 6. 分治法的适用条件

- 问题可以分解为若干相同类型的子问题
- 子问题之间相互独立（不共享状态）
- 子问题的解可以合并
- 当子问题规模足够小时可以直接求解
