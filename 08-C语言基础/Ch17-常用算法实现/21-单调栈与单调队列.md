# 单调栈与单调队列

## 1. 单调栈

单调栈是一种特殊的栈，栈内元素保持单调递增或递减。常用于解决**下一个更大/更小元素**问题。

### 下一个更大元素

```c
#include <stdio.h>

#define MAXN 100005

// 对每个元素找右边第一个比它大的元素
void nextGreaterElement(int arr[], int n, int result[]) {
    int stack[MAXN], top = -1;

    for (int i = 0; i < n; i++) {
        // 栈顶元素小于当前元素，当前元素就是栈顶的"下一个更大"
        while (top >= 0 && arr[stack[top]] < arr[i]) {
            result[stack[top]] = arr[i];
            top--;
        }
        stack[++top] = i;  // 入栈
    }
    // 栈中剩余元素右边没有更大的
    while (top >= 0) {
        result[stack[top]] = -1;
        top--;
    }
}

int main() {
    int arr[] = {2, 1, 2, 4, 3};
    int n = 5, result[MAXN];
    nextGreaterElement(arr, n, result);
    for (int i = 0; i < n; i++)
        printf("%d -> %d\n", arr[i], result[i]);
    return 0;
}
```

## 2. 直方图最大矩形

经典的单调栈应用。

```c
#include <stdio.h>

int largestRectangleArea(int heights[], int n) {
    int stack[MAXN], top = -1;
    int maxArea = 0;

    for (int i = 0; i <= n; i++) {
        int h = (i == n) ? 0 : heights[i];
        while (top >= 0 && heights[stack[top]] > h) {
            int height = heights[stack[top--]];
            int width = (top < 0) ? i : i - stack[top] - 1;
            int area = height * width;
            if (area > maxArea) maxArea = area;
        }
        stack[++top] = i;
    }
    return maxArea;
}

int main() {
    int heights[] = {2, 1, 5, 6, 2, 3};
    printf("最大矩形面积: %d\n",
           largestRectangleArea(heights, 6));  // 10
    return 0;
}
```

## 3. 接雨水问题

```c
#include <stdio.h>

// 单调栈解法
int trap(int height[], int n) {
    int stack[MAXN], top = -1;
    int water = 0;

    for (int i = 0; i < n; i++) {
        while (top >= 0 && height[i] > height[stack[top]]) {
            int bottom = height[stack[top--]];
            if (top < 0) break;
            int w = i - stack[top] - 1;
            int h = (height[i] < height[stack[top]] ? height[i] : height[stack[top]]) - bottom;
            water += w * h;
        }
        stack[++top] = i;
    }
    return water;
}
```

## 4. 单调队列

单调队列用于在滑动窗口中快速找到最大/最小值。

### 滑动窗口最小值

```c
#include <stdio.h>

#define MAXN 100005

void slidingWindowMin(int arr[], int n, int k) {
    int deque[MAXN];  // 存下标
    int front = 0, rear = 0;

    for (int i = 0; i < n; i++) {
        // 移除超出窗口的元素
        while (front < rear && deque[front] < i - k + 1)
            front++;

        // 维护单调递增队列（队头是最小值）
        while (front < rear && arr[deque[rear - 1]] > arr[i])
            rear--;

        deque[rear++] = i;

        // 输出窗口最小值
        if (i >= k - 1)
            printf("窗口[%d, %d]最小值: %d\n",
                   i - k + 1, i, arr[deque[front]]);
    }
}
```

### 滑动窗口最大值

```c
void slidingWindowMax(int arr[], int n, int k) {
    int deque[MAXN];
    int front = 0, rear = 0;

    for (int i = 0; i < n; i++) {
        while (front < rear && deque[front] < i - k + 1)
            front++;

        // 维护单调递减队列（队头是最大值）
        while (front < rear && arr[deque[rear - 1]] < arr[i])
            rear--;

        deque[rear++] = i;

        if (i >= k - 1)
            printf("窗口[%d, %d]最大值: %d\n",
                   i - k + 1, i, arr[deque[front]]);
    }
}
```

## 5. 单调队列优化DP

### 最大子数组和（限制长度不超过k）

```c
#include <stdio.h>
#include <limits.h>

int maxSubarraySumK(int arr[], int n, int k) {
    int prefix[MAXN];
    prefix[0] = 0;
    for (int i = 1; i <= n; i++)
        prefix[i] = prefix[i - 1] + arr[i - 1];

    int deque[MAXN], front = 0, rear = 0;
    int maxSum = INT_MIN;

    for (int i = 1; i <= n; i++) {
        // 维护窗口[i-k, i-1]
        while (front < rear && deque[front] < i - k)
            front++;

        // 找prefix[0..i-1]中的最小值
        if (front < rear)
            maxSum = (prefix[i] - prefix[deque[front]] > maxSum)
                    ? prefix[i] - prefix[deque[front]]
                    : maxSum;

        // 维护单调递增
        while (front < rear && prefix[deque[rear - 1]] > prefix[i])
            rear--;
        deque[rear++] = i;
    }
    return maxSum;
}
```

## 6. 单调栈应用总结

| 问题 | 方向 | 单调性 |
|------|------|--------|
| 下一个更大元素 | 从左到右 | 递减栈 |
| 下一个更小元素 | 从左到右 | 递增栈 |
| 上一个更大元素 | 从右到左 | 递减栈 |
| 上一个更小元素 | 从右到左 | 递增栈 |
| 直方图最大矩形 | 从左到右 | 递增栈 |
| 接雨水 | 从左到右 | 递减栈 |

## 7. 综合示例：每日温度

```c
// 给定每日温度，计算需要等几天才能遇到更高温度
void dailyTemperatures(int temps[], int n, int result[]) {
    int stack[MAXN], top = -1;
    for (int i = 0; i < n; i++) result[i] = 0;

    for (int i = 0; i < n; i++) {
        while (top >= 0 && temps[stack[top]] < temps[i]) {
            result[stack[top]] = i - stack[top];
            top--;
        }
        stack[++top] = i;
    }
}
```
