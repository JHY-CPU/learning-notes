# 15 - 类型转换

## 一、类型转换概述

C 语言中，当不同类型的值混合运算时，编译器会自动进行**类型转换**，也可以通过强制转换（cast）手动进行转换。

```
类型转换
├── 隐式转换（Implicit Conversion）
│   ├── 整型提升（Integer Promotion）
│   ├── 算术转换（Usual Arithmetic Conversion）
│   ├── 赋值转换
│   └── 函数参数转换
└── 显式转换（Explicit Conversion / Cast）
```

---

## 二、隐式类型转换

### 2.1 整型提升（Integer Promotion）

在表达式中，`char`、`short` 等小于 `int` 的类型会自动提升为 `int`（或 `unsigned int`）：

```c
char a = 10, b = 20;
int result = a + b;  // a 和 b 先提升为 int，再相加

// 整型提升规则：
// - 如果 int 能表示原类型的所有值 → 提升为 int
// - 否则 → 提升为 unsigned int
```

```c
#include <stdio.h>

int main(void) {
    char c = 'A';
    short s = 100;

    // c 和 s 在运算时提升为 int
    printf("sizeof(c + s) = %zu\n", sizeof(c + s));  // 4（int 的大小）
    printf("sizeof(c) = %zu\n", sizeof(c));            // 1
    printf("sizeof(s) = %zu\n", sizeof(s));            // 2

    return 0;
}
```

### 2.2 算术转换（Usual Arithmetic Conversion）

当二元运算符的两个操作数类型不同时，编译器将**较小的类型转换为较大的类型**：

```
转换方向（从低到高）：
char/short → int → unsigned int → long → unsigned long → long long → unsigned long long → float → double → long double
```

```c
int i = 5;
double d = 2.0;
double result = i + d;  // i 转换为 double，结果为 double

int x = 10;
unsigned int y = 20;
// x 转换为 unsigned int，结果为 unsigned int
```

### 2.3 赋值转换

赋值时，右侧表达式自动转换为左侧变量的类型：

```c
int a = 3.14;       // 截断：a = 3
double b = 42;      // b = 42.0
char c = 300;       // 溢出：c = 44（300 % 256）

unsigned int u = -1;  // u = 4294967295（UINT_MAX）
int i = 4294967295u;  // i = -1（回绕）
```

## 三、类型转换的方向与规则

### 3.1 整型之间的转换

```c
// 小类型 → 大类型（安全）
char c = 'A';
int i = c;           // 值保持不变

// 大类型 → 小类型（可能丢失数据）
int big = 300;
char small = big;    // small = 44（300 - 256）

// 有符号 ↔ 无符号
int neg = -1;
unsigned int pos = neg;  // pos = 4294967295
```

### 3.2 整型与浮点型

```c
// 浮点 → 整型（截断小数部分）
int a = 3.9;         // a = 3（不是四舍五入！）
int b = -3.9;        // b = -3

// 整型 → 浮点（可能损失精度）
float f = 16777217;  // float 只有 24 位尾数，可能不精确
```

### 3.3 有符号与无符号混合运算

```c
#include <stdio.h>

int main(void) {
    int a = -1;
    unsigned int b = 1;

    if (a < b) {
        printf("-1 < 1\n");      // 期望执行
    } else {
        printf("-1 >= 1\n");     // 实际执行！
    }
    // 原因：a 被转换为 unsigned int，-1 变成很大的正数

    return 0;
}
```

> **警告**：有符号和无符号混合运算时，有符号值被转换为无符号值，可能导致意外结果。

## 四、显式类型转换（Cast）

### 4.1 基本语法

```c
(type_name) expression
```

```c
int a = 10, b = 3;
double result = (double)a / b;   // 3.333333...
// 如果不转换：a / b = 3（整数除法）

int x = 256;
char c = (char)x;                // c = 0（截断）
```

### 4.2 常见使用场景

```c
// 1. 整数除法转为浮点除法
int a = 7, b = 2;
double avg = (double)a / b;    // 3.5（不是 3）

// 2. malloc 返回值转换
int *arr = (int *)malloc(10 * sizeof(int));

// 3. void* 转换
void *ptr = malloc(sizeof(int));
int *ip = (int *)ptr;

// 4. 类型截断
long long big = 123456789012345LL;
int small = (int)big;           // 截断
```

### 4.3 指针类型转换

```c
int x = 0x12345678;
char *cp = (char *)&x;      // 取 x 的第一个字节

// 取决于大端/小端
printf("第一个字节: 0x%02X\n", *cp);  // 0x78（小端）或 0x12（大端）
```

## 五、整型提升的详细规则

| 原始类型 | 提升后类型 | 条件 |
|----------|------------|------|
| `char` | `int` | 如果 `int` 能表示所有 `char` 值 |
| `signed char` | `int` | 通常 |
| `unsigned char` | `int` | 如果 `int` 能表示所有值 |
| `short` | `int` | 如果 `int` 能表示所有 `short` 值 |
| `unsigned short` | `int` | 如果 `int` 能表示所有值 |
| `unsigned short` | `unsigned int` | 如果 `int` 不能表示所有值 |

## 六、常见的类型转换陷阱

### 6.1 有符号/无符号比较

```c
int i = -1;
unsigned int u = 1;

// i 被转为 unsigned，-1 变成 UINT_MAX
if (i < u) {
    // 不会执行！
}
```

### 6.2 整数除法截断

```c
int a = 5, b = 2;
double c = a / b;      // c = 2.0（整数除法结果再转 double）
double d = (double)a / b;  // d = 2.5（正确）
```

### 6.3 char 符号性

```c
char c = 255;           // 如果 char 是有符号的，c = -1
int i = c;              // 符号扩展：i = -1（不是 255！）

unsigned char uc = 255;
int j = uc;             // j = 255（正确）
```

### 6.4 截断丢失数据

```c
double pi = 3.14159;
int truncated = (int)pi;   // truncated = 3（不是四舍五入）

long long big = 1LL << 40;
int small = (int)big;      // small 的值不确定
```

## 七、安全转换的建议

```c
#include <stdio.h>
#include <limits.h>

int main(void) {
    // 1. 检查转换是否会丢失数据
    long long big = 1000000LL;
    if (big >= INT_MIN && big <= INT_MAX) {
        int safe = (int)big;  // 安全
    }

    // 2. 浮点转整数前检查范围
    double d = 1e100;
    if (d >= INT_MIN && d <= INT_MAX) {
        int i = (int)d;
    }

    // 3. 避免有符号和无符号混合运算
    unsigned int len = 10;
    // 错误：for (int i = len - 1; i >= 0; i--)
    // 正确：
    for (unsigned int i = len; i-- > 0;) {
        printf("%u ", i);
    }

    return 0;
}
```

## 八、要点总结

1. **整型提升**：`char` 和 `short` 在运算时自动提升为 `int`
2. **算术转换**：混合类型运算时，较小类型转为较大类型
3. 有符号与无符号混合运算时，有符号值转为无符号值（可能出错）
4. 浮点转整数**截断**小数部分（不四舍五入）
5. 强制转换使用 `(type)` 语法，可能丢失数据
6. `char` 的符号性由实现定义，处理字符/字节时使用 `unsigned char`
7. 整数除法结果是整数，要得到浮点结果需要先转换操作数
8. 尽量避免隐式类型转换，使用显式转换表明意图
