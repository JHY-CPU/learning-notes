# 01 - 数据类型概述

## 一、什么是数据类型

在 C 语言中，**数据类型**决定了变量在内存中占用的空间大小、可存储的值范围以及可执行的操作。每一种数据类型都对应着一组特定的编码规则和运算方式。

正确理解数据类型，是编写高效、安全的 C 程序的基础。

---

## 二、C 语言数据类型分类

C 语言的数据类型可以分为以下四大类：

```
C 数据类型
├── 基本类型（Primitive Types）
│   ├── 整型：short, int, long, long long
│   ├── 字符型：char
│   ├── 浮点型：float, double, long double
│   └── 布尔型：_Bool（C99）
├── 构造类型（Derived / Composite Types）
│   ├── 数组（Array）
│   ├── 结构体（struct）
│   ├── 联合体（union）
│   └── 枚举（enum）
├── 指针类型（Pointer Type）
│   └── 如 int *, char *, void *
└── 空类型（void）
```

### 2.1 基本类型

基本类型是最基础的数据单元，直接由编译器和硬件支持：

| 类型 | 说明 | 典型大小 |
|------|------|----------|
| `char` | 字符型 | 1 字节 |
| `short` | 短整型 | 2 字节 |
| `int` | 整型 | 4 字节 |
| `long` | 长整型 | 4 或 8 字节 |
| `long long` | 长长整型（C99） | 8 字节 |
| `float` | 单精度浮点 | 4 字节 |
| `double` | 双精度浮点 | 8 字节 |
| `long double` | 扩展精度浮点 | 8/12/16 字节 |
| `_Bool` | 布尔型（C99） | 1 字节 |

> **注意**：以上大小是常见平台的典型值，C 标准只规定了最小范围，具体大小由实现决定。

### 2.2 构造类型

构造类型由基本类型组合而成：

- **数组**：相同类型元素的有序集合
- **结构体（struct）**：不同类型成员的集合，每个成员独立占用内存
- **联合体（union）**：多个成员共享同一段内存空间
- **枚举（enum）**：一组命名的整型常量

```c
// 结构体示例
struct Student {
    char name[32];
    int age;
    float score;
};

// 枚举示例
enum Color { RED, GREEN, BLUE };
```

### 2.3 指针类型

指针存储的是变量的**内存地址**，是 C 语言最强大也最危险的特性之一：

```c
int x = 42;
int *p = &x;   // p 存储 x 的地址

printf("x 的值: %d\n", *p);   // 通过指针解引用访问 x
printf("x 的地址: %p\n", (void *)p);
```

指针的大小在 32 位系统中为 4 字节，在 64 位系统中为 8 字节。

### 2.4 空类型（void）

`void` 表示"无类型"，主要用于：

- **函数返回值**：`void func(void)` 表示不返回任何值
- **通用指针**：`void *` 可以指向任意类型的数据
- **函数参数**：`void func(void)` 表示不接受任何参数

```c
void print_hello(void) {
    printf("Hello, World!\n");
}

// void* 通用指针
void *ptr = malloc(100);
int *ip = (int *)ptr;
```

## 三、类型系统的特点

### 3.1 静态类型

C 是**静态类型语言**，每个变量的类型在编译时就必须确定，且不能在运行时改变。

```c
int a = 10;
a = "hello";  // 编译错误！不能将字符串赋给 int 变量
```

### 3.2 弱类型 / 隐式转换

C 允许一定程度的**隐式类型转换**，这使得 C 被认为是一种"弱类型"语言：

```c
int i = 3.14;        // 隐式转换：i 的值为 3（小数部分被截断）
double d = 10;       // 隐式转换：d 的值为 10.0
char c = 65;         // 隐式转换：c 存储字符 'A'
```

### 3.3 类型大小依赖平台

C 标准只规定了类型的**最小范围**，而不是固定大小：

```c
// C 标准规定：
// sizeof(short) <= sizeof(int) <= sizeof(long) <= sizeof(long long)
// sizeof(char) == 1 （永远为 1）
```

### 3.4 有符号与无符号

整型和字符类型可以分为 **signed（有符号）** 和 **unsigned（无符号）** 两种：

```c
signed int a = -100;      // 可以存储负数
unsigned int b = 100;     // 只能存储非负数，但正数范围更大
```

## 四、类型限定符与存储说明符

C 语言还提供了一些关键字来修饰和限定类型：

| 关键字 | 作用 |
|--------|------|
| `const` | 声明常量，值不可修改 |
| `volatile` | 告诉编译器不要优化该变量的访问 |
| `restrict` | 限定指针，用于编译器优化（C99） |
| `static` | 控制变量/函数的链接属性和生命周期 |
| `extern` | 声明外部变量或函数 |
| `signed` | 指定有符号类型 |
| `unsigned` | 指定无符号类型 |

---

## 五、使用 `sizeof` 查看类型大小

`sizeof` 运算符可以查看任意类型或变量在当前平台上占用的字节数：

```c
#include <stdio.h>

int main(void) {
    printf("char:        %zu 字节\n", sizeof(char));
    printf("short:       %zu 字节\n", sizeof(short));
    printf("int:         %zu 字节\n", sizeof(int));
    printf("long:        %zu 字节\n", sizeof(long));
    printf("long long:   %zu 字节\n", sizeof(long long));
    printf("float:       %zu 字节\n", sizeof(float));
    printf("double:      %zu 字节\n", sizeof(double));
    printf("long double: %zu 字节\n", sizeof(long double));
    printf("char*:       %zu 字节\n", sizeof(char *));
    printf("void*:       %zu 字节\n", sizeof(void *));
    return 0;
}
```

---

## 六、要点总结

1. C 语言数据类型分为**基本类型、构造类型、指针类型、void 类型**四大类
2. C 是**静态类型**语言，类型在编译时确定
3. C 允许**隐式类型转换**，需要程序员自行注意类型安全
4. 类型大小**依赖于平台和编译器**，应使用 `sizeof` 获取实际大小
5. 整型分为 `signed` 和 `unsigned` 两种，影响取值范围
6. `void *` 是通用指针类型，可以指向任意类型
7. C99 新增了 `_Bool` 布尔类型和 `long long` 类型
8. 始终使用 `sizeof` 而非硬编码类型大小，提高代码的可移植性
