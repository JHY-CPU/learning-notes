# 09 - 常量详解

## 一、常量的概念

**常量**是在程序运行期间**值不能被改变**的量。C 语言提供了多种定义常量的方式。

---

## 二、字面常量（Literal Constants）

字面常量就是直接写在代码中的固定值：

```c
// 整型字面量
42           // int
3.14         // double
100L         // long
255U         // unsigned int
10LL         // long long

// 浮点字面量
3.14f        // float
2.71828      // double
1.0e10       // double（科学计数法）

// 字符字面量
'A'          // char
'\n'         // char（转义字符）

// 字符串字面量
"Hello"      // char 数组（含 '\0'）
""           // 空字符串
```

---

## 三、#define 宏常量

使用预处理器 `#define` 定义的符号常量：

```c
#include <stdio.h>

#define PI 3.14159
#define MAX_SIZE 1024
#define APP_NAME "MyApplication"
#define NEWLINE '\n'

int main(void) {
    double radius = 5.0;
    double area = PI * radius * radius;

    printf("%s: 面积 = %.2f%c", APP_NAME, area, NEWLINE);
    printf("最大尺寸: %d\n", MAX_SIZE);

    return 0;
}
```

### 3.1 #define 的特点

- 在**预处理阶段**进行文本替换
- 不占用内存（不是真正的变量）
- 没有类型信息
- 可以定义表达式和函数式宏

```c
#define SQUARE(x) ((x) * (x))
#define MAX(a, b) ((a) > (b) ? (a) : (b))

int result = SQUARE(5);     // 展开为 ((5) * (5)) = 25
int bigger = MAX(10, 20);   // 展开为 ((10) > (20) ? (10) : (20)) = 20
```

### 3.2 #define 的陷阱

```c
#define SQUARE(x) x * x       // 错误的写法！

int a = SQUARE(2 + 3);        // 展开为 2 + 3 * 2 + 3 = 11，不是 25！

// 正确写法：加括号
#define SQUARE(x) ((x) * (x))
```

## 四、const 常量

使用 `const` 关键字修饰的变量，值不可修改：

```c
#include <stdio.h>

int main(void) {
    const int MAX = 100;
    const double PI = 3.14159;
    const char GRADE = 'A';

    // MAX = 200;  // 编译错误！不能修改 const 变量

    printf("MAX = %d\n", MAX);
    printf("PI = %.5f\n", PI);

    return 0;
}
```

### 4.1 const 与 #define 的区别

| 特性 | `#define` | `const` |
|------|-----------|---------|
| 处理阶段 | 预处理（文本替换） | 编译 |
| 类型检查 | 无 | 有 |
| 内存 | 不占用 | 占用内存 |
| 作用域 | 从定义到文件末尾 | 遵循块作用域 |
| 调试 | 不可见（已替换） | 可调试 |
| 指针 | 不适用 | 可以用 const 修饰指针 |

```c
void func(void) {
    #define  CONST_A 10     // 作用域：从此处到文件末尾
    const int CONST_B = 20; // 作用域：仅在此函数内

    // CONST_A 在 func 结束后仍然可见！
    // CONST_B 在 func 结束后不可见
}
```

## 五、枚举常量

使用 `enum` 定义的一组命名整型常量：

```c
#include <stdio.h>

enum Color {
    RED,       // 0
    GREEN,     // 1
    BLUE       // 2
};

enum Weekday {
    MON = 1,   // 1（手动指定起始值）
    TUE,       // 2
    WED,       // 3
    THU,       // 4
    FRI,       // 5
    SAT = 6,   // 6（重新指定）
    SUN        // 7
};

int main(void) {
    enum Color favorite = BLUE;
    printf("RED = %d, GREEN = %d, BLUE = %d\n", RED, GREEN, BLUE);

    // 枚举值可用于 switch
    switch (favorite) {
        case RED:   printf("红色\n"); break;
        case GREEN: printf("绿色\n"); break;
        case BLUE:  printf("蓝色\n"); break;
    }

    return 0;
}
```

## 六、不同常量方式的比较

```c
// 方式一：#define
#define BUFFER_SIZE 1024

// 方式二：const
const int buffer_size = 1024;

// 方式三：enum
enum { BUFFER_SIZE = 1024 };

// 方式四：数组大小（仅限整型常量表达式）
char buffer[BUFFER_SIZE];
```

### 选择建议

| 场景 | 推荐方式 |
|------|----------|
| 整型常量集合 | `enum` |
| 单个类型化常量 | `const` |
| 简单替换或宏函数 | `#define` |
| 数组大小 | `enum` 或 `#define`（`const` 在 C 中不是常量表达式） |

> **重要**：在 C 语言中，`const` 变量**不是**编译时常量表达式，不能用于数组大小（C++ 可以）。

```c
const int N = 10;
int arr[N];        // C 中可能报错（取决于编译器）
// int arr[10];    // 正确
#define N 10
int arr[N];        // 正确
```

## 七、const 修饰的指针常量

```c
const int *p1;        // 指向常量的指针（不能通过 p1 修改值）
int const *p2;        // 同上
int * const p3 = &x;  // 常量指针（不能修改 p3 本身）
const int * const p4 = &x;  // 指向常量的常量指针
```

## 八、字符串字面量的特殊性

```c
char *str = "Hello";     // 指向字符串字面量的指针
char arr[] = "Hello";    // 字符数组，内容可修改

str[0] = 'h';            // 未定义行为！字符串字面量不可修改
arr[0] = 'h';            // 正确！arr 是可修改的数组
```

> 字符串字面量存储在**只读数据段**，修改它会导致未定义行为（可能崩溃）。

---

## 九、要点总结

1. C 语言有四种常量方式：**字面常量、#define、const、enum**
2. `#define` 在预处理阶段进行文本替换，无类型检查
3. `const` 有类型检查，遵循作用域规则，是更好的选择
4. `#define` 宏定义表达式时**必须加括号**，防止优先级问题
5. 枚举常量适合定义一组相关的整型值
6. 在 C 语言中，`const` 变量不是编译时常量表达式（与 C++ 不同）
7. 字符串字面量不可修改，应使用字符数组来存储可修改的字符串
8. 优先使用 `const` 而非 `#define` 定义单个常量
