# 02 - 整型类型详解

## 一、整型家族

C 语言提供了一系列整型类型，按存储大小从小到大排列：

| 类型 | 最小大小 | 典型大小 | 最小范围（C 标准） |
|------|----------|----------|---------------------|
| `short` (short int) | >= 16 位 | 2 字节 | -32767 ~ 32767 |
| `int` | >= 16 位 | 4 字节 | -32767 ~ 32767 |
| `long` (long int) | >= 32 位 | 4 或 8 字节 | -2147483647 ~ 2147483647 |
| `long long` | >= 64 位 | 8 字节 | -9223372036854775807 ~ 9223372036854775807 |

> **规则**：`sizeof(short) <= sizeof(int) <= sizeof(long) <= sizeof(long long)`

---

## 二、有符号与无符号

### 2.1 signed 与 unsigned

每种整型都可以分为有符号（signed）和无符号（unsigned）两种：

```c
signed int a = -42;        // 有符号，可以为负
unsigned int b = 42;       // 无符号，只能 >= 0
```

- **signed**（默认）：最高位为符号位，可表示正数、负数和零
- **unsigned**：所有位都用于表示数值，正数范围翻倍

### 2.2 简写形式

```c
// 以下两两等价
int a;              // 等价于 signed int
short b;            // 等价于 signed short int
long c;             // 等价于 signed long int
long long d;        // 等价于 signed long long int

unsigned int e;
unsigned short f;
unsigned long g;
unsigned long long h;
```

### 2.3 取值范围（以 32 位 int 为例）

| 类型 | 最小值 | 最大值 |
|------|--------|--------|
| `int` (32位) | -2,147,483,648 (-2^31) | 2,147,483,647 (2^31-1) |
| `unsigned int` (32位) | 0 | 4,294,967,295 (2^32-1) |
| `short` (16位) | -32,768 (-2^15) | 32,767 (2^15-1) |
| `unsigned short` (16位) | 0 | 65,535 (2^16-1) |

> 使用 `<limits.h>` 头文件中的宏可以获取各类型的精确范围。

---

## 三、limits.h 中的宏常量

```c
#include <stdio.h>
#include <limits.h>

int main(void) {
    printf("char 范围: %d ~ %d\n", CHAR_MIN, CHAR_MAX);
    printf("short 范围: %d ~ %d\n", SHRT_MIN, SHRT_MAX);
    printf("int 范围: %d ~ %d\n", INT_MIN, INT_MAX);
    printf("long 范围: %ld ~ %ld\n", LONG_MIN, LONG_MAX);
    printf("long long 范围: %lld ~ %lld\n", LLONG_MIN, LLONG_MAX);
    printf("unsigned int 最大值: %u\n", UINT_MAX);
    printf("unsigned long 最大值: %lu\n", ULONG_MAX);
    return 0;
}
```

常见宏定义（以 32 位 int 为例）：

| 宏 | 值 |
|----|-----|
| `INT_MIN` | -2147483648 |
| `INT_MAX` | 2147483647 |
| `UINT_MAX` | 4294967295 |
| `SHRT_MIN` | -32768 |
| `SHRT_MAX` | 32767 |
| `LONG_MIN` | -2147483648 (32位) 或 -9223372036854775808 (64位) |

## 四、整型字面量（常量）

### 4.1 不同进制的字面量

```c
int decimal = 42;        // 十进制
int octal = 052;         // 八进制（以 0 开头）
int hex = 0x2A;          // 十六进制（以 0x 开头）
int binary = 0b101010;   // 二进制（C23 标准，部分编译器提前支持）
```

### 4.2 整型后缀

通过后缀指定字面量的类型：

```c
int a = 100;             // 默认 int
long b = 100L;           // long 类型
long long c = 100LL;     // long long 类型

unsigned int d = 100U;   // unsigned int
unsigned long e = 100UL; // unsigned long
unsigned long long f = 100ULL; // unsigned long long
```

后缀可以组合使用，大小写均可（`l` 和 `L` 都合法）。

> **建议**：始终使用大写后缀（`L`、`U`、`LL`），因为小写 `l` 容易与数字 `1` 混淆。

## 五、sizeof 运算符查看整型大小

```c
#include <stdio.h>

int main(void) {
    printf("sizeof(short)        = %zu\n", sizeof(short));
    printf("sizeof(int)          = %zu\n", sizeof(int));
    printf("sizeof(long)         = %zu\n", sizeof(long));
    printf("sizeof(long long)    = %zu\n", sizeof(long long));
    printf("sizeof(unsigned int) = %zu\n", sizeof(unsigned int));
    return 0;
}
```

典型输出（64 位 Windows 系统）：
```
sizeof(short)        = 2
sizeof(int)          = 4
sizeof(long)         = 4
sizeof(long long)    = 8
sizeof(unsigned int) = 4
```

## 六、格式化输出占位符

| 类型 | printf 占位符 |
|------|--------------|
| `int` | `%d` |
| `unsigned int` | `%u` |
| `long` | `%ld` |
| `unsigned long` | `%lu` |
| `long long` | `%lld` |
| `unsigned long long` | `%llu` |
| `short` | `%hd` |
| `unsigned short` | `%hu` |

```c
int x = 42;
long y = 1000000L;
unsigned long long z = 18446744073709551615ULL;

printf("x = %d\n", x);
printf("y = %ld\n", y);
printf("z = %llu\n", z);
```

## 七、整型的选择建议

1. **一般整数运算**：使用 `int`，它是平台最高效的类型
2. **需要更大范围**：使用 `long long`
3. **需要节省内存**（如大量数据）：使用 `short`
4. **位操作和无符号运算**：使用 `unsigned int`
5. **明确大小需求**（如文件格式、网络协议）：使用 `<stdint.h>` 中的定宽类型

```c
#include <stdint.h>

int8_t   a;   // 确切 8 位有符号
uint8_t  b;   // 确切 8 位无符号
int16_t  c;   // 确切 16 位有符号
int32_t  d;   // 确切 32 位有符号
int64_t  e;   // 确切 64 位有符号
```

---

## 八、要点总结

1. C 语言整型有 `short`、`int`、`long`、`long long` 四种大小等级
2. 每种都可以加 `signed`（默认）或 `unsigned` 修饰
3. 类型大小满足：`sizeof(short) <= sizeof(int) <= sizeof(long) <= sizeof(long long)`
4. 使用 `<limits.h>` 中的宏获取类型的取值范围
5. 整型字面量可用后缀 `U`、`L`、`LL` 指定类型
6. 使用 `<stdint.h>` 的 `int32_t` 等类型确保跨平台一致性
7. 一般场景优先使用 `int`，它通常是平台最优的类型
8. 注意使用正确的 `printf` 占位符，否则会导致未定义行为
