# 21 - 字符编码

## 一、字符编码概述

**字符编码**是将字符映射为数字的规则。计算机只能处理数字，因此需要编码方案来表示文字。

```
字符 'A' → 编码值 65 (ASCII)
字符 '中' → 编码值 20013 (Unicode) / 0xD6D0 (GBK)
```

---

## 二、ASCII 编码

### 2.1 基本 ASCII

ASCII（American Standard Code for Information Interchange）使用 **7 位**编码，定义了 **128 个字符**：

| 范围 | 类别 | 说明 |
|------|------|------|
| 0 ~ 31 | 控制字符 | 不可打印（\0, \n, \t 等） |
| 32 ~ 126 | 可打印字符 | 空格、字母、数字、标点 |
| 127 | DEL | 删除控制符 |

```
数字 '0' ~ '9': 48 ~ 57
大写 'A' ~ 'Z': 65 ~ 90
小写 'a' ~ 'z': 97 ~ 122
```

### 2.2 扩展 ASCII

使用 **8 位**（1 字节），扩展到 **256 个字符**，高位 128 个字符在不同地区定义不同。

## 三、中文编码

### 3.1 GB2312

- 1980 年发布，收录 6763 个汉字
- 双字节编码，兼容 ASCII
- 高字节 0xA1~0xF7，低字节 0xA1~0xFE

### 3.2 GBK

- 1995 年发布，向下兼容 GB2312
- 收录 21003 个汉字
- 双字节编码，高字节 0x81~0xFE，低字节 0x40~0xFE

### 3.3 GB18030

- 2000 年发布，强制标准
- 可变长度：1、2、或 4 字节
- 完全兼容 Unicode

```c
// GBK 编码示例
char *str = "中文";    // GBK 环境下占用 5 字节：D6 D0 CE C4 \0
printf("字节数: %zu\n", strlen(str));  // 4
printf("字符数: 不确定（依赖编码）\n");
```

## 四、Unicode 编码

### 4.1 Unicode 概念

Unicode 为世界上每一个字符分配唯一的**码点（Code Point）**：

```
U+0041  'A'
U+4E2D  '中'
U+1F600 '😀'
```

Unicode 码点范围：U+0000 ~ U+10FFFF（超过 100 万个码点）。

### 4.2 编码方式

Unicode 有多种编码实现：

| 编码 | 字节序 | 说明 |
|------|--------|------|
| UTF-8 | 变长 1~4 字节 | 兼容 ASCII，最常用 |
| UTF-16 | 变长 2~4 字节 | Windows 内部使用 |
| UTF-32 | 固定 4 字节 | 简单但浪费空间 |

---

## 五、UTF-8 编码

### 5.1 编码规则

| 码点范围 | 字节数 | 编码格式 |
|----------|--------|----------|
| U+0000 ~ U+007F | 1 | 0xxxxxxx |
| U+0080 ~ U+07FF | 2 | 110xxxxx 10xxxxxx |
| U+0800 ~ U+FFFF | 3 | 1110xxxx 10xxxxxx 10xxxxxx |
| U+10000 ~ U+10FFFF | 4 | 11110xxx 10xxxxxx 10xxxxxx 10xxxxxx |

### 5.2 示例

```
'A' (U+0041): 01000001 → 0x41 (1字节)
'中' (U+4E2D): 0100 111000 101101 → 11100100 10111000 10101101 → 0xE4 0xB8 0xAD (3字节)
'😀' (U+1F600): 4字节
```

```c
char *utf8_str = "中";    // UTF-8 环境下是 3 字节 + '\0'
unsigned char *p = (unsigned char *)utf8_str;
printf("%02X %02X %02X\n", p[0], p[1], p[2]);  // E4 B8 AD
```

### 5.3 UTF-8 的优点

- 完全兼容 ASCII
- 自同步（可以通过首字节判断字符长度）
- 无字节序问题
- 空间效率高（英文 1 字节，常用中文 3 字节）

## 六、C 语言中的宽字符

### 6.1 wchar_t

`wchar_t` 是 C 标准定义的宽字符类型，大小由实现决定：

```c
#include <wchar.h>
#include <locale.h>

int main(void) {
    setlocale(LC_ALL, "");   // 设置本地化

    wchar_t wc = L'中';      // 宽字符
    wchar_t wstr[] = L"你好世界";  // 宽字符串

    printf("sizeof(wchar_t) = %zu\n", sizeof(wchar_t));  // 2 或 4

    wprintf(L"%ls\n", wstr);

    return 0;
}
```

### 6.2 wchar_t 的平台差异

| 平台 | wchar_t 大小 | 编码 |
|------|-------------|------|
| Windows | 2 字节 | UTF-16 |
| Linux | 4 字节 | UTF-32 |

## 七、多字节字符处理

### 7.1 标准库函数

```c
#include <stdlib.h>
#include <wchar.h>

// 多字节转宽字符
size_t mbtowc(wchar_t *pwc, const char *s, size_t n);
int mbtowc(wchar_t *pwc, const char *s, size_t n);

// 宽字符转多字节
size_t wcrtomb(char *s, wchar_t wc, mbstate_t *ps);

// 多字节字符串转宽字符串
size_t mbstowcs(wchar_t *dest, const char *src, size_t n);

// 宽字符串转多字节字符串
size_t wcstombs(char *dest, const wchar_t *src, size_t n);
```

### 7.2 示例

```c
#include <stdio.h>
#include <stdlib.h>
#include <locale.h>
#include <wchar.h>

int main(void) {
    setlocale(LC_ALL, "zh_CN.UTF-8");

    // UTF-8 → 宽字符
    char *utf8 = "你好";
    wchar_t wbuf[10];
    int count = mbstowcs(wbuf, utf8, 10);
    wprintf(L"宽字符数: %d\n", count);   // 2
    wprintf(L"%ls\n", wbuf);

    // 宽字符 → UTF-8
    char mb_buf[100];
    wcstombs(mb_buf, wbuf, 100);
    printf("%s\n", mb_buf);

    return 0;
}
```

## 八、判断 UTF-8 字符长度

```c
// 根据 UTF-8 首字节判断字符的字节数
int utf8_char_len(unsigned char c) {
    if (c < 0x80) return 1;        // 0xxxxxxx
    if ((c & 0xE0) == 0xC0) return 2;  // 110xxxxx
    if ((c & 0xF0) == 0xE0) return 3;  // 1110xxxx
    if ((c & 0xF8) == 0xF0) return 4;  // 11110xxx
    return -1;  // 无效的 UTF-8 首字节
}

// 计算 UTF-8 字符串的字符数
int utf8_strlen(const char *s) {
    int count = 0;
    while (*s) {
        int len = utf8_char_len((unsigned char)*s);
        if (len < 0) return -1;
        s += len;
        count++;
    }
    return count;
}
```

## 九、要点总结

1. ASCII 使用 7 位编码，定义 128 个字符，是所有编码的基础
2. 中文编码有 GB2312、GBK、GB18030 等，使用双字节或多字节
3. Unicode 为每个字符分配唯一码点，UTF-8 是最常用的编码方式
4. UTF-8 变长编码（1~4 字节），完全兼容 ASCII
5. `wchar_t` 在 Windows 上为 2 字节（UTF-16），Linux 上为 4 字节（UTF-32）
6. 使用 `setlocale()` 设置本地化环境
7. `mbstowcs()` 和 `wcstombs()` 用于多字节与宽字符的转换
8. 处理中文等多字节字符时，不要使用 `strlen` 计算字符数
