# 9_联邦学习概述

## 1. 联邦学习动机

联邦学习 (Federated Learning, FL) 是一种**分布式机器学习范式**，允许多个客户端在**不共享原始数据**的情况下协作训练模型。其核心动机是**隐私保护**。

### 1.1 传统 vs 联邦学习

| 特性 | 传统集中式学习 | 联邦学习 |
|------|--------------|----------|
| 数据位置 | 汇聚到中央服务器 | 保留在本地 |
| 隐私风险 | 高 (数据集中) | **低** (数据不共享) |
| 通信成本 | 传输原始数据 | 仅传输模型参数 |
| 法规合规 | 可能违反 GDPR | 满足数据本地化要求 |
| 数据利用 | 需要数据收集 | 可利用分散数据 |

### 1.2 应用驱动力

```
联邦学习需求驱动:

1. 法规要求
   - GDPR (欧盟): 数据不得跨境传输
   - 个人信息保护法: 数据最小化原则
   - 医疗数据: HIPAA 隐私规则

2. 数据特性
   - 手机用户数据 (如输入法)
   - 医院病历数据
   - 银行金融数据
   - 企业内部数据

3. 技术限制
   - 数据量大，传输成本高
   - 网络带宽受限
   - 实时性要求
```

## 2. FedAvg 算法

### 2.1 算法流程

FedAvg (Federated Averaging) 由 McMahan 等人 (2017) 提出，是联邦学习的基础算法。

```
FedAvg 流程:

Server:
  初始化全局模型 w_0
  for round t = 1, 2, ..., T:
      1. 随机选择 K 个客户端子集 S_t
      2. 将全局模型 w_t 广播给 S_t 中的客户端
      3. 等待客户端返回本地更新
      4. 聚合: w_{t+1} = Σ (n_k/n) * w_k

Client k (在第 t 轮):
  1. 接收全局模型 w_t
  2. 本地训练 E 个 epoch:
     for epoch = 1, ..., E:
         for batch (x, y) in local_data:
             w_k ← w_k - η∇L(w_k; x, y)
  3. 返回本地模型 w_k 给服务器
```

### 2.2 数学形式化

$$w_{t+1} = \sum_{k=1}^{K} \frac{n_k}{n} w_k^{(t+1)}$$

其中：
- $n_k$: 客户端 $k$ 的数据量
- $n = \sum_k n_k$: 总数据量
- $w_k^{(t+1)}$: 客户端 $k$ 本地训练后的模型

### 2.3 PyTorch 实现

```python
import torch
import torch.nn as nn
import torch.optim as optim
import copy

class FedAvgServer:
    """
    FedAvg 服务器端实现
    """
    def __init__(self, global_model, num_clients, num_rounds,
                 clients_per_round=10):
        self.global_model = global_model
        self.num_clients = num_clients
        self.num_rounds = num_rounds
        self.clients_per_round = clients_per_round

    def aggregate(self, client_models, client_weights):
        """
        加权平均聚合
        """
        global_dict = self.global_model.state_dict()
        total_weight = sum(client_weights)

        # 初始化聚合结果
        for key in global_dict:
            global_dict[key] = torch.zeros_like(global_dict[key])

        # 加权求和
        for model, weight in zip(client_models, client_weights):
            client_dict = model.state_dict()
            for key in global_dict:
                global_dict[key] += client_dict[key] * (weight / total_weight)

        self.global_model.load_state_dict(global_dict)

    def run(self, clients):
        """
        联邦学习主循环
        """
        results = []

        for round_idx in range(self.num_rounds):
            print(f"\n=== Round {round_idx + 1}/{self.num_rounds} ===")

            # 选择客户端
            selected = self.select_clients(clients)

            # 发送全局模型
            client_models = []
            client_weights = []

            for client in selected:
                # 克隆全局模型并发送给客户端
                local_model = copy.deepcopy(self.global_model)
                local_model = client.local_train(local_model)
                client_models.append(local_model)
                client_weights.append(client.num_samples)

            # 聚合
            self.aggregate(client_models, client_weights)

            # 评估
            acc = self.evaluate()
            results.append(acc)
            print(f"  Global Accuracy: {acc:.4f}")

        return results

    def select_clients(self, clients):
        """随机选择客户端"""
        import random
        return random.sample(clients, min(self.clients_per_round, len(clients)))

    def evaluate(self):
        """评估全局模型"""
        self.global_model.eval()
        correct = 0
        total = 0
        with torch.no_grad():
            for x, y in self.test_loader:
                output = self.global_model(x)
                _, pred = output.max(1)
                correct += pred.eq(y).sum().item()
                total += y.size(0)
        return correct / total


class FedAvgClient:
    """
    FedAvg 客户端实现
    """
    def __init__(self, client_id, local_data, num_samples,
                 local_epochs=5, lr=0.01, batch_size=32):
        self.client_id = client_id
        self.local_data = local_data
        self.num_samples = num_samples
        self.local_epochs = local_epochs
        self.lr = lr
        self.batch_size = batch_size

    def local_train(self, model):
        """
        本地训练
        """
        model.train()
        optimizer = optim.SGD(model.parameters(), lr=self.lr)
        dataloader = torch.utils.data.DataLoader(
            self.local_data, batch_size=self.batch_size, shuffle=True
        )

        for epoch in range(self.local_epochs):
            for x, y in dataloader:
                optimizer.zero_grad()
                output = model(x)
                loss = nn.functional.cross_entropy(output, y)
                loss.backward()
                optimizer.step()

        return model
```

## 3. 通信效率

### 3.1 通信瓶颈分析

联邦学习的通信是主要瓶颈，特别是在移动设备场景下：

| 场景 | 典型模型大小 | 通信频率 | 总通信量 |
|------|-------------|----------|----------|
| MNIST (MLP) | ~1 MB | 100 轮 | ~100 MB |
| CIFAR-10 (CNN) | ~10 MB | 500 轮 | ~5 GB |
| ImageNet (ResNet) | ~100 MB | 1000 轮 | ~100 GB |
| NLP (BERT) | ~400 MB | 2000 轮 | ~800 GB |

### 3.2 通信优化方法

```python
def compress_gradients(model, method='topk', ratio=0.01):
    """
    梯度压缩: 减少通信量
    """
    compressed = {}

    for name, param in model.named_parameters():
        if param.grad is not None:
            grad = param.grad.data

            if method == 'topk':
                # Top-K 稀疏化
                k = max(1, int(grad.numel() * ratio))
                values, indices = grad.abs().view(-1).topk(k)
                mask = torch.zeros_like(grad.view(-1), dtype=torch.bool)
                mask[indices] = True
                mask = mask.view(grad.shape)
                compressed[name] = {
                    'values': grad[mask],
                    'indices': indices,
                    'shape': grad.shape
                }

            elif method == 'quantize':
                # 量化到 1-bit
                sign = grad.sign()
                mean_abs = grad.abs().mean()
                compressed[name] = {
                    'sign': sign,
                    'scale': mean_abs
                }

    return compressed

def decompress_gradients(compressed, method='topk'):
    """
    梯度解压缩
    """
    decompressed = {}

    for name, data in compressed.items():
        if method == 'topk':
            grad = torch.zeros(data['shape']).view(-1)
            grad[data['indices']] = data['values']
            decompressed[name] = grad.view(data['shape'])

        elif method == 'quantize':
            decompressed[name] = data['sign'] * data['scale']

    return decompressed
```

## 4. 联邦学习的挑战

```
联邦学习核心挑战:

1. 数据异构性 (Non-IID)
   └── 不同客户端的数据分布不同

2. 通信效率
   └── 带宽受限、延迟高

3. 系统异构性
   └── 不同设备的计算能力和数据量差异大

4. 隐私与安全
   └── 模型更新可能泄露信息
   └── 恶意客户端可能发起攻击

5. 公平性
   └── 不同客户端的贡献和获益不均
```

## 5. 总结

| 要点 | 说明 |
|------|------|
| 核心动机 | 隐私保护、数据本地化 |
| 基础算法 | FedAvg: 本地训练 + 加权平均 |
| 通信模式 | 仅传输模型参数/梯度 |
| 核心挑战 | Non-IID、通信效率、安全 |
