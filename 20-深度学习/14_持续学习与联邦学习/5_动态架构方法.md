# 5_动态架构方法

## 1. 动态架构概述

动态架构方法通过**修改网络结构**来解决持续学习中的遗忘问题。其核心思想是为不同任务分配不同的网络参数，避免参数冲突。

### 1.1 方法分类

```
动态架构方法
├── 固定扩展
│   ├── Progressive Neural Networks (PNN)
│   └── 每个任务新增子网络
├── 选择性固定
│   ├── PackNet: 通过剪枝分配参数
│   └── Piggyback: 二值掩码
└── 动态增长
    ├── 动态扩展宽度
    └── 动态扩展深度
```

## 2. Progressive Neural Networks

### 2.1 算法原理

Rusu 等人 (2016) 提出的 PNN 为每个新任务添加一个新列 (column)，旧列的参数被冻结，新列通过**横向连接**访问旧列的特征。

```
PNN 架构:

Task 1          Task 2          Task 3
[Layer1]        [Layer1]        [Layer1]
   ↓               ↓↓              ↓↓↓
[Layer2]  →     [Layer2]   →    [Layer2]
   ↓               ↓↓              ↓↓↓
[Layer3]        [Layer3]        [Layer3]
   ↓               ↓↓              ↓↓↓
[Output]        [Output]        [Output]

← 冻结 →      ← 冻结 →        ← 新训练 →
         ↑ 横向连接      ↑ 横向连接
```

### 2.2 PyTorch 实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class PNNColumn(nn.Module):
    """PNN 的单个列"""
    def __init__(self, input_dim, hidden_dims, output_dim):
        super().__init__()
        self.layers = nn.ModuleList()

        # 构建层
        dims = [input_dim] + hidden_dims
        for i in range(len(dims) - 1):
            self.layers.append(nn.Linear(dims[i], dims[i+1]))

        self.output_layer = nn.Linear(hidden_dims[-1], output_dim)

    def forward(self, x):
        for layer in self.layers:
            x = F.relu(layer(x))
        return self.output_layer(x)

class ProgressiveNN(nn.Module):
    """
    Progressive Neural Networks 实现
    """
    def __init__(self, input_dim, hidden_dims, output_dim):
        super().__init__()
        self.input_dim = input_dim
        self.hidden_dims = hidden_dims
        self.output_dim = output_dim

        self.columns = nn.ModuleList()
        self.lateral_connections = nn.ModuleList()

        # 添加第一列
        first_col = PNNColumn(input_dim, hidden_dims, output_dim)
        self.columns.append(first_col)

    def add_column(self):
        """
        为新任务添加新列
        """
        num_cols = len(self.columns)
        new_col = PNNColumn(self.input_dim, self.hidden_dims, self.output_dim)

        # 为每一层添加横向连接 (从旧列到新列)
        laterals = nn.ModuleList()
        for layer_idx in range(len(self.hidden_dims)):
            # 横向连接: 将所有旧列的该层输出映射到新列
            lateral = nn.Linear(
                self.hidden_dims[layer_idx] * num_cols,
                self.hidden_dims[layer_idx]
            )
            laterals.append(lateral)

        self.columns.append(new_col)
        self.lateral_connections.append(laterals)

        # 冻结旧列参数
        for col in self.columns[:-1]:
            for param in col.parameters():
                param.requires_grad = False

        # 冻结旧横向连接
        for lateral_group in self.lateral_connections[:-1]:
            for param in lateral_group.parameters():
                param.requires_grad = False

    def forward(self, x, task_id=None):
        """
        前向传播
        """
        if task_id is None:
            task_id = len(self.columns) - 1

        col_outputs = []

        for col_idx in range(task_id + 1):
            h = x
            col = self.columns[col_idx]

            for layer_idx, layer in enumerate(col.layers):
                h = F.relu(layer(h))

                # 添加横向连接
                if col_idx > 0 and layer_idx < len(self.lateral_connections[col_idx-1]):
                    # 收集旧列在该层的输出
                    lateral_input = torch.cat(col_outputs[:col_idx], dim=-1)
                    lateral = self.lateral_connections[col_idx-1][layer_idx]
                    h = h + F.relu(lateral(lateral_input))

            col_outputs.append(h)

        # 输出层
        output = self.columns[task_id].output_layer(col_outputs[task_id])
        return output

    def freeze_old_columns(self):
        """冻结旧列"""
        for col in self.columns[:-1]:
            for param in col.parameters():
                param.requires_grad = False

    def count_parameters(self):
        """统计参数量"""
        total = sum(p.numel() for p in self.parameters())
        trainable = sum(p.numel() for p in self.parameters() if p.requires_grad)
        return {'total': total, 'trainable': trainable}
```

## 3. PackNet

### 3.1 算法原理

PackNet 通过**剪枝**将网络参数分配给不同任务。每个任务使用一部分参数，释放后的参数可用于新任务。

```
PackNet 流程:

Task 1 训练 → 剪枝 → 冻结已用参数 → 释放未用参数
Task 2 训练 → 剪枝 → 冻结已用参数 → 释放未用参数
...
```

### 3.2 实现

```python
class PackNet:
    """
    PackNet: 基于剪枝的参数分配
    """
    def __init__(self, model):
        self.model = model
        self.masks = {}  # 每个任务的参数掩码
        self.task_params = {}  # 每个任务使用的参数索引

    def train_task(self, task_id, train_loader, optimizer, epochs=10,
                   prune_ratio=0.5):
        """
        训练并分配参数给当前任务
        """
        # 正常训练
        for epoch in range(epochs):
            self.model.train()
            for x, y in train_loader:
                output = self.model(x.cuda())
                loss = F.cross_entropy(output, y.cuda())
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()

        # 剪枝: 保留权重最大的参数
        self._prune_and_allocate(task_id, prune_ratio)

    def _prune_and_allocate(self, task_id, prune_ratio):
        """
        剪枝并分配参数
        """
        task_mask = {}

        for name, param in self.model.named_parameters():
            if 'weight' in name:
                # 计算参数重要性 (绝对值)
                importance = param.data.abs()

                # 确定保留哪些参数
                num_keep = int(param.numel() * (1 - prune_ratio))
                _, keep_indices = importance.view(-1).topk(num_keep)

                # 创建掩码
                mask = torch.zeros_like(param.data, dtype=torch.bool)
                mask.view(-1)[keep_indices] = True

                # 排除已被旧任务占用的参数
                for old_mask in self.masks.values():
                    if name in old_mask:
                        mask = mask & (~old_mask[name])

                task_mask[name] = mask

                # 冻结被占用的参数
                param.requires_grad = False

        self.masks[task_id] = task_mask

    def apply_masks(self):
        """应用所有任务的掩码"""
        for name, param in self.model.named_parameters():
            if 'weight' in name:
                # 合并所有掩码
                combined_mask = torch.zeros_like(param.data, dtype=torch.bool)
                for task_mask in self.masks.values():
                    if name in task_mask:
                        combined_mask = combined_mask | task_mask[name]

                # 将不需要的参数置零
                param.data[~combined_mask] = 0

    def forward_with_mask(self, x, task_id):
        """使用特定任务的掩码前向传播"""
        # 临时应用掩码
        original_params = {}
        for name, param in self.model.named_parameters():
            if 'weight' in name and name in self.masks.get(task_id, {}):
                original_params[name] = param.data.clone()
                param.data = param.data * self.masks[task_id][name].float()

        output = self.model(x)

        # 恢复参数
        for name, data in original_params.items():
            dict(self.model.named_parameters())[name].data = data

        return output
```

## 4. Piggyback (二值掩码)

```python
class Piggyback:
    """
    Piggyback: 学习二值掩码适配新任务
    """
    def __init__(self, base_model):
        self.base_model = base_model
        self.masks = {}

    def learn_mask(self, task_id, train_loader, epochs=10):
        """
        为新任务学习二值掩码
        """
        # 初始化可学习的掩码参数
        mask_params = {}
        for name, param in self.base_model.named_parameters():
            if 'weight' in name:
                mask_params[name] = nn.Parameter(
                    torch.ones_like(param) * 5.0  # 初始化为大正值
                )

        optimizer = torch.optim.Adam(mask_params.values(), lr=0.01)

        for epoch in range(epochs):
            for x, y in train_loader:
                # 使用硬阈值化 (不可微) → 使用直通估计器
                output = self.forward_with_mask(x.cuda(), mask_params)
                loss = F.cross_entropy(output, y.cuda())

                optimizer.zero_grad()
                loss.backward()
                optimizer.step()

        # 二值化掩码
        task_mask = {}
        for name, mp in mask_params.items():
            task_mask[name] = (torch.sigmoid(mp) > 0.5).float()

        self.masks[task_id] = task_mask

    def forward_with_mask(self, x, mask_params):
        """带可学习掩码的前向传播"""
        # 直通估计器: 前向用硬阈值，反向用软阈值
        for name, param in self.base_model.named_parameters():
            if name in mask_params:
                # Hard sigmoid + straight-through estimator
                soft_mask = torch.sigmoid(mask_params[name])
                hard_mask = (soft_mask > 0.5).float()
                # 直通估计器
                mask = hard_mask + soft_mask - soft_mask.detach()
                param_masked = param * mask
                # 临时替换参数
                param.data = param_masked

        output = self.base_model(x)
        return output
```

## 5. 动态扩展宽度

```python
class DynamicExpansion(nn.Module):
    """
    动态扩展: 根据任务需求增加网络容量
    """
    def __init__(self, input_dim, base_hidden_dim=64, output_dim=10):
        super().__init__()
        self.input_dim = input_dim
        self.base_hidden_dim = base_hidden_dim
        self.output_dim = output_dim

        # 共享的骨干网络
        self.backbone = nn.Sequential(
            nn.Linear(input_dim, base_hidden_dim),
            nn.ReLU(),
        )

        # 任务特定的扩展模块
        self.expansion_modules = nn.ModuleList()
        self.task_classifiers = nn.ModuleList()

    def add_task(self, task_output_dim):
        """为新任务添加扩展模块"""
        # 添加扩展模块 (额外的隐藏层)
        expansion = nn.Sequential(
            nn.Linear(self.base_hidden_dim, self.base_hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(self.base_hidden_dim // 2, self.base_hidden_dim),
            nn.ReLU(),
        )
        self.expansion_modules.append(expansion)

        # 添加任务分类器
        classifier = nn.Linear(self.base_hidden_dim, task_output_dim)
        self.task_classifiers.append(classifier)

        # 冻结旧扩展模块
        for mod in self.expansion_modules[:-1]:
            for param in mod.parameters():
                param.requires_grad = False

    def forward(self, x, task_id=None):
        features = self.backbone(x)

        if task_id is not None:
            # 特定任务前向传播
            expanded = self.expansion_modules[task_id](features)
            return self.task_classifiers[task_id](expanded + features)
        else:
            outputs = []
            for i in range(len(self.expansion_modules)):
                expanded = self.expansion_modules[i](features)
                outputs.append(self.task_classifiers[i](expanded + features))
            return outputs
```

## 6. 方法对比

| 方法 | 参数共享 | 内存增长 | 遗忘 | 可扩展性 |
|------|----------|----------|------|----------|
| PNN | 无 (冻结旧列) | $O(T)$ | **零遗忘** | 有限 |
| PackNet | 部分 (参数复用) | $O(1)$ | 零遗忘 | 中等 |
| Piggyback | 全部 (掩码选择) | $O(T)$ (掩码) | 零遗忘 | 好 |
| 动态扩展 | 部分 (共享骨干) | $O(T)$ | 低 | 中等 |

## 7. 总结

| 要点 | 说明 |
|------|------|
| PNN | 新任务新列 + 横向连接，零遗忘 |
| PackNet | 剪枝分配参数，参数复用 |
| Piggyback | 学习二值掩码选择子网络 |
| 共同优势 | 理论上可避免灾难性遗忘 |
| 共同挑战 | 模型容量随任务数增长 |
