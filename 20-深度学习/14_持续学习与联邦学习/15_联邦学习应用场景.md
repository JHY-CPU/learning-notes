## 2. 医疗健康

### 2.1 多医院协作诊断

不同医院拥有不同类型的病例数据，通过联邦学习协作训练诊断模型：

```
联邦医疗诊断:

医院 A (CT扫描数据)  ──→  本地训练  ──→  ┐
医院 B (X光数据)      ──→  本地训练  ──→  ├→  聚合 → 全局诊断模型
医院 C (MRI数据)      ──→  本地训练  ──→  ┘

优势:
- 每家医院的数据留在本地
- 患者隐私受到保护
- 聚合多方知识提升诊断准确率
```

### 2.2 实现示例

```python
class FederatedMedicalDiagnosis:
    """
    联邦医疗诊断系统
    """
    def __init__(self, hospitals, global_model):
        self.hospitals = hospitals
        self.global_model = global_model

    def train(self, num_rounds=50):
        """联邦训练医疗诊断模型"""
        for round_idx in range(num_rounds):
            selected = np.random.choice(
                self.hospitals,
                size=min(5, len(self.hospitals)),
                replace=False
            )

            local_models = []
            for hospital in selected:
                local_model = copy.deepcopy(self.global_model)

                # 本地训练 (数据不出医院)
                local_model = hospital.local_train(
                    local_model,
                    epochs=3
                )

                local_models.append(local_model)

            # 安全聚合
            self.aggregate(local_models)

            # 评估
            avg_acc = self.evaluate_all_hospitals()
            print(f"Round {round_idx}: Avg Accuracy = {avg_acc:.4f}")

    def differential_privacy_round(self, client_update, epsilon=1.0):
        """差分隐私保护轮次"""
        # 添加噪声保护患者隐私
        sensitivity = 1.0
        noise_scale = sensitivity / epsilon

        for key in client_update:
            noise = torch.randn_like(client_update[key]) * noise_scale
            client_update[key] += noise

        return client_update
```

```python
class FederatedCreditScoring:
    """
    联邦信用评分
    """
    def __init__(self, banks, feature_mapping):
        """
        feature_mapping: 定义哪些特征属于哪家银行
        """
        self.banks = banks
        self.feature_mapping = feature_mapping

    def vertical_fl_training(self):
        """
        纵向联邦训练: 各银行持有不同特征
        银行 A: 存款记录、贷款记录
        银行 B: 信用卡消费、还款记录
        """
        # 1. 样本对齐 (PSI)
        common_ids = self.private_set_intersection()

        # 2. 拆分学习
        for epoch in range(100):
            for batch_ids in common_ids:
                # 各银行计算本地特征表示
                hidden_a = self.banks[0].compute_hidden(batch_ids)
                hidden_b = self.banks[1].compute_hidden(batch_ids)

                # 安全聚合 (使用安全多方计算)
                combined = self.secure_combine(hidden_a, hidden_b)

                # 计算损失并更新
                loss = self.compute_loss(combined)
                self.backward_and_update(loss)
```

## 4. Gboard 键盘预测

### 4.1 系统架构

Google 的 Gboard 是联邦学习最成功的商业应用之一：

```
Gboard 联邦学习:

用户手机 1:              用户手机 2:              用户手机 N:
[本地输入历史]          [本地输入历史]          [本地输入历史]
      ↓                       ↓                       ↓
本地语言模型更新        本地语言模型更新        本地语言模型更新
      ↓                       ↓                       ↓
      └──────────→ Google 服务器 ←──────────────┘
                          ↓
                   全局语言模型更新
                          ↓
                   推送到所有用户
```

### 4.2 关键优化

| 优化 | 描述 |
|------|------|
| 客户端选择 | 只在充电、WiFi、空闲时参与 |
| 压缩传输 | 梯度压缩减少通信量 |
| 差分隐私 | 保护用户输入习惯 |
| 安全聚合 | 服务器无法获取单个用户更新 |

## 5. 联邦 NLP

### 5.1 联邦预训练

```python
class FederatedPretraining:
    """
    联邦预训练语言模型
    """
    def __init__(self, clients, model_name='bert-base'):
        self.clients = clients
        self.model = AutoModel.from_pretrained(model_name)

    def federated_mlm_training(self, num_rounds=100):
        """
        联邦 Masked Language Model 训练
        """
        for round_idx in range(num_rounds):
            selected_clients = self.select_clients()

            local_updates = []
            for client in selected_clients:
                update = client.local_mlm_training(
                    copy.deepcopy(self.model),
                    local_epochs=1
                )
                local_updates.append(update)

            # FedAvg 聚合
            self.aggregate(local_updates)

            # 评估困惑度
            ppl = self.evaluate_perplexity()
            print(f"Round {round_idx}: Perplexity = {ppl:.2f}")
```

### 5.2 联邦微调

```python
def federated_finetuning(base_model, client_tasks):
    """
    联邦微调预训练模型
    各客户端有不同的下游任务
    """
    # 共享基础模型
    shared_model = base_model

    for round_idx in range(50):
        local_updates = []

        for client in client_tasks:
            # 冻结基础层，微调顶层
            local_model = copy.deepcopy(shared_model)
            local_model = client.finetune(local_model)
            local_updates.append(local_model)

        # 只聚合可训练参数
        shared_model = aggregate_trainable_params(
            shared_model, local_updates
        )

    return shared_model
```

```python
class FederatedPredictiveMaintenance:
    """
    联邦预测性维护
    多个工厂协作训练设备故障预测模型
    """
    def __init__(self, factories):
        self.factories = factories

    def train(self):
        """
        各工厂的设备传感器数据留在本地
        协作训练故障预测模型
        """
        global_model = self.initialize_model()

        for round_idx in range(100):
            updates = []
            for factory in self.factories:
                # 本地训练
                local_model = factory.train_on_sensor_data(
                    copy.deepcopy(global_model)
                )
                updates.append(local_model)

            # 鲁棒聚合 (工业场景需要对抗设备故障)
            global_model = self.robust_aggregate(updates)

        return global_model
```
