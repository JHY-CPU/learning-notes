# 0_时间序列分析基础

## 1. 什么是时间序列

时间序列（Time Series）是按时间顺序排列的一组观测值，通常以等间隔采样。时间序列分析的目标是从历史数据中提取模式，并用于预测未来值。

**时间序列的组成要素：**

```
原始序列 Y(t) = T(t) + S(t) + C(t) + R(t)

T(t): 趋势项 (Trend)        — 长期上升或下降的方向
S(t): 季节项 (Seasonality)   — 固定周期的重复模式（如每周、每年）
C(t): 周期项 (Cycle)         — 非固定周期的波动（如经济周期）
R(t): 残差项 (Residual)      — 随机噪声
```

## 2. 趋势 (Trend)

趋势反映时间序列在长期内的运动方向。

**线性趋势：**
$$Y(t) = a + b \cdot t + \epsilon$$

**指数趋势：**
$$Y(t) = a \cdot e^{bt} + \epsilon$$

**趋势检测方法：**
- Mann-Kendall 检验：非参数检验，检测单调趋势
- 线性回归斜率检验：判断斜率是否显著不为零

```python
import numpy as np
from scipy import stats

def detect_trend(series):
    """使用 Mann-Kendall 检验检测趋势"""
    n = len(series)
    s = 0
    for i in range(n - 1):
        for j in range(i + 1, n):
            s += np.sign(series[j] - series[i])
    # 计算方差
    var_s = n * (n - 1) * (2 * n + 5) / 18
    if s > 0:
        z = (s - 1) / np.sqrt(var_s)
    elif s < 0:
        z = (s + 1) / np.sqrt(var_s)
    else:
        z = 0
    p_value = 2 * (1 - stats.norm.cdf(abs(z)))
    return z, p_value
```

## 3. 季节性 (Seasonality)

季节性是指以固定频率重复出现的模式。常见周期：

| 数据类型 | 典型周期 |
|---------|---------|
| 小时数据 | 24（日周期）、168（周周期） |
| 日数据 | 7（周周期）、365（年周期） |
| 月数据 | 12（年周期） |

**季节性分解（STL 分解）：**

```python
from statsmodels.tsa.seasonal import STL
import matplotlib.pyplot as plt

# STL: Seasonal and Trend decomposition using Loess
stl = STL(series, period=12, robust=True)
result = stl.fit()

# 可视化分解结果
fig, axes = plt.subplots(4, 1, figsize=(12, 8))
axes[0].plot(result.observed);    axes[0].set_title('原始序列')
axes[1].plot(result.trend);       axes[1].set_title('趋势')
axes[2].plot(result.seasonal);    axes[2].set_title('季节性')
axes[3].plot(result.resid);       axes[3].set_title('残差')
plt.tight_layout()
```

## 4. 平稳性 (Stationarity)

平稳性是许多时序模型的前提假设。

**严平稳 (Strict Stationarity)：** 联合分布在时间平移下保持不变
$$F(x_{t_1}, x_{t_2}, \ldots, x_{t_k}) = F(x_{t_1+\tau}, x_{t_2+\tau}, \ldots, x_{t_k+\tau})$$

**弱平稳 (Weak Stationarity)：** 满足三个条件：
1. 均值恒定：$E[X_t] = \mu$，对所有 $t$
2. 方差恒定：$Var(X_t) = \sigma^2$，对所有 $t$
3. 自协方差仅依赖于滞后：$Cov(X_t, X_{t+k}) = \gamma_k$

### ADF 检验 (Augmented Dickey-Fuller)

ADF 检验是最常用的平稳性检验方法。其原假设为序列**存在单位根**（非平稳）。

**检验模型：**
$$\Delta y_t = \alpha + \beta t + \gamma y_{t-1} + \sum_{i=1}^{p} \delta_i \Delta y_{t-i} + \epsilon_t$$

- $H_0: \gamma = 0$（存在单位根，非平稳）
- $H_1: \gamma < 0$（平稳）

```python
from statsmodels.tsa.stattools import adfuller

def adf_test(series, name='Series'):
    result = adfuller(series, autolag='AIC')
    print(f'=== ADF 检验: {name} ===')
    print(f'ADF 统计量: {result[0]:.4f}')
    print(f'p 值:       {result[1]:.6f}')
    print(f'滞后阶数:   {result[2]}')
    print(f'临界值:')
    for key, value in result[4].items():
        print(f'  {key}: {value:.4f}')
    if result[1] < 0.05:
        print('结论: 序列平稳 (拒绝原假设)')
    else:
        print('结论: 序列非平稳 (无法拒绝原假设)')
    return result[1] < 0.05

# 差分使序列平稳
diff_series = np.diff(series, n=1)     # 一阶差分
diff2_series = np.diff(series, n=2)    # 二阶差分
seasonal_diff = series[12:] - series[:-12]  # 季节差分(周期12)
```

## 5. 自相关函数 ACF 与偏自相关函数 PACF

**自相关函数 (ACF)：** 度量 $Y_t$ 与 $Y_{t-k}$ 之间的相关性
$$\rho_k = \frac{\sum_{t=k+1}^{n}(Y_t - \bar{Y})(Y_{t-k} - \bar{Y})}{\sum_{t=1}^{n}(Y_t - \bar{Y})^2}$$

**偏自相关函数 (PACF)：** 去除中间变量影响后的纯相关
$$\phi_{kk} = Corr(Y_t, Y_{t-k} | Y_{t-1}, \ldots, Y_{t-k+1})$$

**ACF/PACF 在 ARMA 模型定阶中的应用：**

| 模型 | ACF 特征 | PACF 特征 |
|------|---------|-----------|
| AR(p) | 拖尾（指数衰减） | p 阶后截尾 |
| MA(q) | q 阶后截尾 | 拖尾（指数衰减） |
| ARMA(p,q) | 拖尾 | 拖尾 |

```python
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

fig, axes = plt.subplots(1, 2, figsize=(14, 4))
plot_acf(series, lags=40, ax=axes[0], title='ACF')
plot_pacf(series, lags=40, ax=axes[1], title='PACF', method='ywm')
plt.tight_layout()
```

## 6. 白噪声检验

白噪声序列是纯随机的，无法预测。在建模后需检验残差是否为白噪声。

**Ljung-Box 检验：**
$$Q = n(n+2) \sum_{k=1}^{h} \frac{\hat{\rho}_k^2}{n-k}$$

```python
from statsmodels.stats.diagnostic import acorr_ljungbox

lb_test = acorr_ljungbox(series, lags=[10, 20, 30])
print("Ljung-Box 检验结果:")
print(lb_test)
# 如果 p > 0.05，残差为白噪声，模型充分
```
