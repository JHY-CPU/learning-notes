# 时序预测与深度学习

## 一、传统时序方法

- **ARIMA**：自回归积分滑动平均模型
- **ETS**：指数平滑
- **Prophet**：Facebook的时序分解方法

---

## 二、深度学习时序模型

### 2.1 RNN/LSTM

循环结构天然适合序列建模。编码器-解码器结构支持多步预测。

### 2.2 TCN（时间卷积网络）

使用因果膨胀卷积：
- 因果性：不使用未来信息
- 膨胀卷积：增大感受野
- 残差连接：支持深层网络

### 2.3 Transformer变体

- **Informer**：ProbSparse自适应注意力
- **Autoformer**：序列分解 + 自相关机制
- **Fedformer**：频域增强
- **PatchTST**：时序patch + 通道独立性
- **iTransformer**：将每个变量视为token

### 2.4 时序预训练模型

- **TimesFM**：Google的时序基础模型
- **Chronos**：Amazon的时序预测模型
- **Lag-Llama**：基于LLaMA的时序模型

## 四、评估与实践

- **评估指标**：MSE、MAE、MAPE、sMAPE
