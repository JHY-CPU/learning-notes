# 4_序列到序列预测

## 1. Seq2Seq 架构概述

序列到序列 (Sequence-to-Sequence) 模型将变长输入序列映射到变长输出序列，核心由编码器和解码器组成：

```
编码器 (Encoder)                解码器 (Decoder)
┌───┐ ┌───┐ ┌───┐ ┌───┐       ┌───┐ ┌───┐ ┌───┐
│ x₁│→│ x₂│→│ x₃│→│ x₄│ ──→  │ s₀│→│ s₁│→│ s₂│→ y
└───┘ └───┘ └───┘ └───┘       └───┘ └───┘ └───┘
 历史观测序列                  上下文向量c   预测序列
```

**数学表述：**
- 编码器：$h_t = f(x_t, h_{t-1})$，生成上下文向量 $c = g(h_1, \ldots, h_T)$
- 解码器：$s_t = f(y_{t-1}, s_{t-1}, c)$，生成输出 $\hat{y}_t = \text{softmax}(W s_t)$

## 2. 上下文向量的传递方式

```python
import torch
import torch.nn as nn

class Seq2SeqForecaster(nn.Module):
    def __init__(self, input_size, hidden_size, output_size,
                 num_layers=2, dropout=0.1):
        super().__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers

        # 编码器
        self.encoder = nn.LSTM(
            input_size, hidden_size, num_layers,
            batch_first=True, dropout=dropout
        )

        # 解码器（接收编码器隐藏状态）
        self.decoder = nn.LSTM(
            input_size, hidden_size, num_layers,
            batch_first=True, dropout=dropout
        )

        # 输出投影
        self.fc_out = nn.Linear(hidden_size, output_size)

    def forward(self, src, trg_len, teacher_forcing_ratio=0.5, trg=None):
        """
        src: (batch, src_len, input_size) 编码器输入
        trg_len: int, 解码器输出长度
        trg: (batch, trg_len, output_size) 真实目标（训练时提供）
        """
        batch_size = src.size(0)

        # 编码
        enc_output, (hidden, cell) = self.encoder(src)

        # 解码器初始输入（最后一个编码器输入或零向量）
        decoder_input = src[:, -1:, :]  # (batch, 1, input_size)

        outputs = []
        for t in range(trg_len):
            dec_output, (hidden, cell) = self.decoder(
                decoder_input, (hidden, cell)
            )
            pred = self.fc_out(dec_output)  # (batch, 1, output_size)
            outputs.append(pred)

            # Teacher Forcing
            if trg is not None and torch.rand(1).item() < teacher_forcing_ratio:
                decoder_input = trg[:, t:t+1, :]
            else:
                decoder_input = pred

        return torch.cat(outputs, dim=1)  # (batch, trg_len, output_size)
```

## 3. Teacher Forcing

Teacher Forcing 是加速 Seq2Seq 训练的关键技术：训练时用真实值而非模型预测值作为解码器的下一步输入。

```
训练阶段 (Teacher Forcing=0.5):
  解码步1: 输入<START>     → 预测 y₁
  解码步2: 输入 y₁(真实)   → 预测 y₂   ← 50%概率用真实值
  解码步3: 输入 ŷ₂(预测)   → 预测 y₃   ← 50%概率用预测值

推理阶段 (无 Teacher Forcing):
  解码步1: 输入<START>  → 预测 ŷ₁
  解码步2: 输入 ŷ₁      → 预测 ŷ₂
  解码步3: 输入 ŷ₂      → 预测 ŷ₃
```

**Scheduled Sampling：** 逐步降低 teacher forcing 比率

```python
def get_teacher_forcing_ratio(epoch, total_epochs, start=1.0, end=0.0):
    """线性衰减 teacher forcing 比率"""
    return max(end, start - (start - end) * epoch / total_epochs)
```

## 4. 自回归预测 (Autoregressive)

自回归预测是推理时的标准做法：每次预测一步，然后将预测值作为下一步输入。

```python
def autoregressive_predict(model, src, horizon, max_val=None, min_val=None):
    """自回归多步预测"""
    model.eval()
    batch_size = src.size(0)

    with torch.no_grad():
        # 编码
        enc_output, (hidden, cell) = model.encoder(src)

        # 逐步解码
        input_step = src[:, -1:, :]
        predictions = []

        for _ in range(horizon):
            dec_output, (hidden, cell) = model.decoder(input_step, (hidden, cell))
            pred = model.fc_out(dec_output)
            predictions.append(pred)
            input_step = pred  # 自回归：用预测值作为下一步输入

    return torch.cat(predictions, dim=1)
```

## 5. 注意力机制增强 Seq2Seq

上下文向量是信息瓶颈，注意力机制让解码器能访问编码器所有时间步：

```python
class AttentionSeq2Seq(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__()
        self.encoder = nn.LSTM(input_size, hidden_size, batch_first=True)
        self.decoder = nn.LSTM(input_size + hidden_size, hidden_size, batch_first=True)

        # 注意力打分
        self.attn_score = nn.Linear(hidden_size * 2, 1)
        self.fc_out = nn.Linear(hidden_size * 2, output_size)

    def attention(self, decoder_hidden, encoder_outputs):
        """
        decoder_hidden: (batch, 1, hidden)
        encoder_outputs: (batch, src_len, hidden)
        """
        src_len = encoder_outputs.size(1)
        # 扩展 decoder_hidden 与每个编码器输出拼接
        dec_expanded = decoder_hidden.expand(-1, src_len, -1)
        combined = torch.cat([dec_expanded, encoder_outputs], dim=-1)

        # 计算注意力分数
        energy = torch.tanh(self.attn_score(combined))  # (batch, src_len, 1)
        weights = torch.softmax(energy, dim=1)

        # 加权上下文
        context = torch.sum(weights * encoder_outputs, dim=1, keepdim=True)
        return context, weights

    def forward(self, src, trg_len):
        batch_size = src.size(0)
        enc_out, (h, c) = self.encoder(src)

        decoder_input = src[:, -1:, :]
        outputs = []
        attn_weights_all = []

        for t in range(trg_len):
            dec_out, (h, c) = self.decoder(decoder_input, (h, c))
            context, attn_w = self.attention(dec_out, enc_out)

            # 拼接上下文和解码器输出
            combined = torch.cat([dec_out, context], dim=-1)
            pred = self.fc_out(combined)
            outputs.append(pred)
            attn_weights_all.append(attn_w)

            decoder_input = pred

        return torch.cat(outputs, dim=1), torch.cat(attn_weights_all, dim=1)
```

## 6. 多变量多步预测实战

```python
class MultiVarSeq2Seq(nn.Module):
    def __init__(self, n_features, hidden_size, forecast_horizon,
                 target_dim=1):
        super().__init__()
        self.encoder = nn.LSTM(n_features, hidden_size, 2,
                               batch_first=True, dropout=0.2)
        self.decoder = nn.LSTM(target_dim, hidden_size, 2,
                               batch_first=True, dropout=0.2)
        self.fc = nn.Linear(hidden_size, target_dim)
        self.horizon = forecast_horizon

    def forward(self, x, trg=None, tf_ratio=0.5):
        # x: (batch, lookback, n_features)
        _, (h, c) = self.encoder(x)

        # 解码
        if trg is not None:  # 训练
            dec_input = trg[:, 0:1, :]
        else:  # 推理
            dec_input = x[:, -1:, 0:1]  # 目标变量最后值

        outputs = []
        for t in range(self.horizon):
            out, (h, c) = self.decoder(dec_input, (h, c))
            pred = self.fc(out)
            outputs.append(pred)

            if trg is not None and torch.rand(1).item() < tf_ratio:
                dec_input = trg[:, t:t+1, :]
            else:
                dec_input = pred

        return torch.cat(outputs, dim=1)
```

## 7. Seq2Seq vs 其他多步策略

| 策略 | 实现方式 | 误差累积 | 计算成本 | 适用场景 |
|------|---------|---------|---------|---------|
| Seq2Seq + TF | 编码器-解码器 | 低 | 中 | 变长输出、中长期预测 |
| 递归预测 | 单步模型反复调用 | 高 | 低 | 短期预测 |
| 直接预测 | 每步独立模型 | 无 | 高 | 步数少 |
| 多输出 | 一次性输出所有步 | 无 | 中 | 固定步数 |

---

**要点总结：**
- Seq2Seq 通过编码器-解码器结构支持变长序列映射
- Teacher Forcing 加速训练但引入 exposure bias
- Scheduled Sampling 可缓解训练和推理的不一致
- 注意力机制打破了上下文向量的信息瓶颈
- 自回归推理是 Seq2Seq 模型的标准预测方式
