# 1_传统时序方法回顾

## 1. 概述

在深度学习兴起之前，统计方法在时序预测中占据主导地位。理解这些方法有助于：
- 作为深度学习模型的基线 (baseline)
- 理解时序数据的统计特性
- 在数据量有限时提供可靠的预测

## 2. ARIMA 模型

ARIMA(p, d, q) 由三部分组成：

| 组件 | 含义 | 参数 |
|------|------|------|
| AR(p) | 自回归 | p: 滞后阶数 |
| I(d) | 差分 | d: 差分次数 |
| MA(q) | 移动平均 | q: 误差滞后阶数 |

**数学表达：**
$$\phi(B)(1-B)^d Y_t = \theta(B) \epsilon_t$$

展开形式：
$$(1 - \sum_{i=1}^{p}\phi_i B^i)(1-B)^d Y_t = (1 + \sum_{j=1}^{q}\theta_j B^j)\epsilon_t$$

其中 $B$ 为滞后算子：$B Y_t = Y_{t-1}$

```python
from statsmodels.tsa.arima.model import ARIMA
import warnings
warnings.filterwarnings('ignore')

# 自动选择最优参数 (网格搜索)
def find_best_arima(series, p_range=(0,4), d_range=(0,2), q_range=(0,4)):
    best_aic = float('inf')
    best_order = None
    for p in range(*p_range):
        for d in range(*d_range):
            for q in range(*q_range):
                try:
                    model = ARIMA(series, order=(p, d, q))
                    fitted = model.fit()
                    if fitted.aic < best_aic:
                        best_aic = fitted.aic
                        best_order = (p, d, q)
                except:
                    continue
    print(f'最优参数: ARIMA{best_order}, AIC: {best_aic:.2f}')
    return best_order

order = find_best_arima(train_series)
model = ARIMA(train_series, order=order)
fitted = model.fit()

# 预测
forecast = fitted.forecast(steps=30)
print(fitted.summary())

# 残差诊断
residuals = fitted.resid
# 检验残差是否为白噪声
from statsmodels.stats.diagnostic import acorr_ljungbox
lb = acorr_ljungbox(residuals, lags=[10])
print(f'Ljung-Box p值: {lb["lb_pvalue"].values[0]:.4f}')
```

## 3. SARIMA 模型

SARIMA(p,d,q)(P,D,Q,m) 在 ARIMA 基础上加入季节性分量：

$$\phi_p(B)\Phi_P(B^m)(1-B)^d(1-B^m)^D Y_t = \theta_q(B)\Theta_Q(B^m)\epsilon_t$$

- $(p,d,q)$: 非季节性参数
- $(P,D,Q,m)$: 季节性参数，$m$ 为周期

```python
from statsmodels.tsa.statespace.sarimax import SARIMAX

# 月度数据，周期 m=12
model = SARIMAX(train_series,
                order=(1, 1, 1),
                seasonal_order=(1, 1, 1, 12),
                enforce_stationarity=False,
                enforce_invertibility=False)
fitted = model.fit(disp=False)
forecast = fitted.forecast(steps=24)

# 带置信区间的预测
pred = fitted.get_forecast(steps=24)
ci = pred.conf_int()
print(f'95% 置信区间宽度: {(ci.iloc[:,1] - ci.iloc[:,0]).mean():.2f}')
```

## 4. 指数平滑 (Exponential Smoothing)

指数平滑族模型的核心思想：**近期观测赋予更高权重**。

### 简单指数平滑 (SES)
$$\hat{Y}_{t+1} = \alpha Y_t + (1-\alpha)\hat{Y}_t$$

适用：无趋势、无季节性

### Holt 线性趋势方法
$$\ell_t = \alpha Y_t + (1-\alpha)(\ell_{t-1} + b_{t-1})$$
$$b_t = \beta^*(\ell_t - \ell_{t-1}) + (1-\beta^*)b_{t-1}$$
$$\hat{Y}_{t+h} = \ell_t + h \cdot b_t$$

### Holt-Winters 方法（三重指数平滑）

**加法模型：**
$$\ell_t = \alpha(Y_t - s_{t-m}) + (1-\alpha)(\ell_{t-1} + b_{t-1})$$
$$b_t = \beta(\ell_t - \ell_{t-1}) + (1-\beta)b_{t-1}$$
$$s_t = \gamma(Y_t - \ell_t - b_t) + (1-\gamma)s_{t-m}$$

```python
from statsmodels.tsa.holtwinters import ExponentialSmoothing

# Holt-Winters 加法模型
model = ExponentialSmoothing(train_series,
                              seasonal_periods=12,
                              trend='add',
                              seasonal='add')
fitted = model.fit(optimized=True)
print(f'平滑参数: alpha={fitted.params["smoothing_level"]:.3f}, '
      f'beta={fitted.params["smoothing_trend"]:.3f}, '
      f'gamma={fitted.params["smoothing_seasonal"]:.3f}')
forecast = fitted.forecast(24)
```

## 5. Prophet 模型

Facebook Prophet 是一个可分解的时间序列模型：

$$y(t) = g(t) + s(t) + h(t) + \epsilon_t$$

- $g(t)$: 趋势函数（分段线性或逻辑增长）
- $s(t)$: 傅里叶级数建模的季节性
- $h(t)$: 假日效应

```python
from prophet import Prophet

# Prophet 要求列名为 ds 和 y
df_prophet = pd.DataFrame({
    'ds': date_index,
    'y': series_values
})

model = Prophet(
    growth='linear',           # 或 'logistic'
    yearly_seasonality=True,
    weekly_seasonality=True,
    daily_seasonality=False,
    changepoint_prior_scale=0.05  # 趋势灵活度
)

# 添加自定义季节性
model.add_seasonality(name='monthly', period=30.5, fourier_order=5)

# 添加假日
model.add_country_holidays(country_name='CN')

model.fit(df_prophet)

# 创建未来日期并预测
future = model.make_future_dataframe(periods=60, freq='D')
forecast = model.predict(future)

# 可视化
model.plot(forecast)
model.plot_components(forecast)
```

## 6. 模型对比

| 方法 | 优势 | 劣势 | 适用场景 |
|------|------|------|---------|
| ARIMA | 理论成熟、可解释 | 仅单变量、线性假设 | 短期预测、经济数据 |
| SARIMA | 处理季节性 | 参数选择复杂 | 有明显季节性 |
| 指数平滑 | 简单快速 | 表达能力有限 | 快速基线 |
| Prophet | 自动化程度高、处理假日 | 不适合高频率 | 业务时序、含假日 |

## 7. 评估指标

```python
from sklearn.metrics import mean_absolute_error, mean_squared_error

def evaluate_forecast(actual, predicted):
    mae = mean_absolute_error(actual, predicted)
    rmse = np.sqrt(mean_squared_error(actual, predicted))
    mape = np.mean(np.abs((actual - predicted) / actual)) * 100
    smape = np.mean(2 * np.abs(actual - predicted) /
                    (np.abs(actual) + np.abs(predicted))) * 100
    return {'MAE': mae, 'RMSE': rmse, 'MAPE': mape, 'sMAPE': smape}
```

---

**要点总结：**
- ARIMA 适合平稳或差分后平稳的单变量序列
- SARIMA 处理季节性，但参数空间大，需 AIC/BIC 辅助选择
- Prophet 开箱即用，适合含有假日和多种季节性的业务数据
- 传统方法是很好的 baseline，深度学习模型应在基线上有显著提升才使用
