# 06_正定矩阵与二次型优化

## 核心概念

- **二次型 (Quadratic Form)**：形如 $f(\mathbf{x}) = \mathbf{x}^T A \mathbf{x} = \sum_{i,j} a_{ij} x_i x_j$ 的函数称为二次型，其中 $A$ 是对称矩阵。二次型是多元函数中仅次于线性函数的最简单形式。
- **正定矩阵 (Positive Definite Matrix)**：如果对所有非零向量 $\mathbf{x}$，都有 $\mathbf{x}^T A \mathbf{x} > 0$，则 $A$ 是正定矩阵（记作 $A \succ 0$）。所有特征值大于零。半正定（$\succeq 0$）允许等于零。
- **负定矩阵与不定矩阵**：所有特征值小于零为负定；既有正特征值又有负特征值为不定矩阵。不定矩阵对应的二次型在原点处是鞍点。
- **Cholesky 分解**：正定矩阵可以唯一分解为 $A = LL^T$，其中 $L$ 是下三角矩阵。这是求解正定线性方程组最高效的方法，复杂度为 $O(n^3/3)$，比 LU 分解快一倍。
- **二次型优化的椭球解释**：正定二次型 $f(\mathbf{x}) = \mathbf{x}^T A \mathbf{x}$ 的等值面是椭球，椭球轴的半长平方与 $A$ 的特征值成反比。条件数 $\kappa = \lambda_{\max}/\lambda_{\min}$ 决定了椭球"瘦长"程度。

## 数学推导

正定矩阵的判别条件：
- 所有特征值 $\lambda_i > 0$
- 所有顺序主子式 $> 0$（Sylvester 判据）
- 存在可逆矩阵 $B$ 使得 $A = B^T B$
- Cholesky 分解 $A = LL^T$ 存在

二次型的梯度与极值：
$$
f(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T A \mathbf{x} - \mathbf{b}^T \mathbf{x} + c
$$
$$
\nabla f(\mathbf{x}) = A\mathbf{x} - \mathbf{b} = \mathbf{0} \quad \Rightarrow \quad \mathbf{x}^* = A^{-1}\mathbf{b}
$$
由于 $\nabla^2 f(\mathbf{x}) = A \succ 0$，这是一个凸二次规划问题，$\mathbf{x}^*$ 是全局最小值。

瑞利商与特征值的关系：
$$
\min_{\mathbf{x} \neq 0} \frac{\mathbf{x}^T A \mathbf{x}}{\mathbf{x}^T \mathbf{x}} = \lambda_{\min}, \quad
\max_{\mathbf{x} \neq 0} \frac{\mathbf{x}^T A \mathbf{x}}{\mathbf{x}^T \mathbf{x}} = \lambda_{\max}
$$

## 直观理解

- **能量视角**：正定二次型 $\mathbf{x}^T A \mathbf{x}$ 可以理解为系统的"势能"。对于物理系统，势能总是非负的，且在平衡位置取最小值。这正是正定矩阵的物理直觉——在平衡点附近，势能可以用正定二次型近似。
- **椭球几何**：方程 $\mathbf{x}^T A \mathbf{x} = 1$ 在三维中是一个椭球面。$A$ 的特征向量指向椭球的主轴方向，特征值的倒数平方根是主轴半长。条件数大意味着椭球很"瘦"，优化时容易在长轴方向振荡。
- **等高线图**：在二维中，正定二次型的等高线是一族同心椭圆。梯度下降沿最陡方向下降，但路径呈"之"字形。而牛顿法利用曲率信息可以一步到达中心。

## 代码示例

```python
import numpy as np

# 创建一个正定矩阵
A = np.array([[3, 1], [1, 3]])
print(f"特征值: {np.linalg.eigvalsh(A)}")  # 都大于0

# Cholesky 分解
L = np.linalg.cholesky(A)
print(f"L:\n{L}")
print(f"LL^T:\n{L @ L.T}")
print(f"重建误差: {np.max(np.abs(A - L @ L.T)):.2e}")

# 验证 Sylvester 判据
def is_positive_definite(A):
    """检查矩阵是否正定"""
    try:
        np.linalg.cholesky(A)
        return True
    except np.linalg.LinAlgError:
        return False

# 对比不同矩阵
B = np.array([[1, 2], [2, 1]])  # 特征值: 3, -1 -> 不定
C = np.array([[-1, 0], [0, -2]])  # 特征值: -1, -2 -> 负定

print(f"A 正定? {is_positive_definite(A)}")
print(f"B 正定? {is_positive_definite(B)}")
print(f"C 正定? {is_positive_definite(C)}")

# 求解正定线性方程组 Ax = b (使用 Cholesky)
b = np.array([5, 7])
x = np.linalg.solve(A, b)  # 内部使用 Cholesky
print(f"方程 Ax=b 的解: {x}")

# 验证二次型最小值
x_opt = np.linalg.solve(A, b)  # 正定二次型 f(x) = 0.5x^T A x - b^T x 的最小值点
print(f"最小值点: {x_opt}")

# 瑞利商计算
def rayleigh_quotient(A, x):
    return (x @ A @ x) / (x @ x)

eigvals = np.linalg.eigvalsh(A)
print(f"最小瑞利商: {rayleigh_quotient(A, np.array([1, 1])):.4f} (λ_min={eigvals[0]:.4f})")
```

## 深度学习关联

- **误差曲面的局部近似**：在最优解附近，损失函数可以用二阶泰勒展开 $L(\theta) \approx L(\theta^*) + \frac{1}{2}(\theta-\theta^*)^T H (\theta-\theta^*)$ 近似。如果 Hessian 矩阵 $H \succ 0$，则局部是凸的，优化容易；如果有负特征值，则存在鞍点。
- **牛顿法 (Newton's Method)**：二阶优化方法 $\theta_{t+1} = \theta_t - H^{-1} \nabla L$ 利用 Hessian 的逆调整梯度方向。当 $H \succ 0$ 时，牛顿方向就是指向最小值的正确方向。但大规模深度学习中 Hessian 太大，无法直接计算，因此催生了拟牛顿法（如 L-BFGS）。
- **协方差矩阵正定性**：在多元高斯分布中，协方差矩阵 $\Sigma$ 必须是正定矩阵。在马氏距离 $d(\mathbf{x}, \boldsymbol{\mu}) = (\mathbf{x}-\boldsymbol{\mu})^T \Sigma^{-1} (\mathbf{x}-\boldsymbol{\mu})$ 中，$\Sigma^{-1}$ 的正定性确保了距离的非负性。
- **特征值正则化**：一些正则化技术通过惩罚 Hessian 的条件数或添加正定约束来改善优化 landscape，如让权重矩阵的奇异值更均衡，从而加速收敛并提高泛化能力。
