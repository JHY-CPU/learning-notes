# 12_梯度下降法的收敛性证明基础

## 核心概念

- **收敛性 (Convergence)**：梯度下降法能否保证找到最小值？如果可以，需要多少步？收敛性分析回答了这两个问题。不同假设条件下收敛速度不同。
- **Lipschitz 光滑性 (L-smooth)**：如果函数的梯度满足 $\|\nabla f(x) - \nabla f(y)\| \leq L \|x - y\|$，则称 $f$ 是 $L$-光滑的。$L$ 是 Hessian 最大特征值的上界，表示梯度的变化速率上限。
- **强凸性 (Strong Convexity)**：如果 $f(y) \geq f(x) + \nabla f(x)^T (y-x) + \frac{\mu}{2}\|y-x\|^2$，则称 $f$ 是 $\mu$-强凸的。$\mu$ 是 Hessian 最小特征值的下界，表示函数的"凸度"。
- **条件数 (Condition Number)**：$\kappa = L / \mu$，衡量优化问题的困难程度。$\kappa$ 越大，优化越困难（等值面越瘦长）。
- **收敛速度分类**：
  - **次线性收敛**：$f(x_k) - f(x^*) = O(1/k)$，梯度下降在凸函数上的典型速度
  - **线性收敛**：$f(x_k) - f(x^*) = O(\rho^k)$，$0<\rho<1$，强凸函数上的速度
  - **二次收敛**：$\|x_{k+1} - x^*\| = O(\|x_k - x^*\|^2)$，牛顿法在强凸函数上的速度

## 数学推导

梯度下降更新：$x_{k+1} = x_k - \eta \nabla f(x_k)$

### 凸函数 + L-光滑 的收敛性
假设 $f$ 是凸函数且 $L$-光滑，步长 $\eta = 1/L$：
$$
f(x_{k+1}) \leq f(x_k) - \frac{1}{2L}\|\nabla f(x_k)\|^2
$$

经过 $k$ 步后：
$$
f(x_k) - f(x^*) \leq \frac{2L\|x_0 - x^*\|^2}{k}
$$
这是次线性收敛（$O(1/k)$），意味着要获得 $\epsilon$ 精度需要 $O(1/\epsilon)$ 步。

### 强凸函数 + L-光滑 的收敛性
假设 $f$ 是 $\mu$-强凸且 $L$-光滑，步长 $\eta = 2/(\mu+L)$：
$$
\|x_k - x^*\|^2 \leq \left(1 - \frac{2\mu L}{\mu+L}\right)^k \|x_0 - x^*\|^2
$$

记 $\kappa = L/\mu$ 为条件数：
$$
\rho = 1 - \frac{2}{\kappa+1} \approx 1 - \frac{2}{\kappa}
$$
要获得 $\epsilon$ 精度需要 $O(\kappa \log(1/\epsilon))$ 步。

## 直观理解

- **凸函数的图像**：凸函数像碗。$L$-光滑限制碗壁不能太陡峭（梯度变化有上限），$\mu$-强凸限制碗壁不能太平坦（至少有一定的"碗深"）。条件数 $\kappa$ 小了碗是圆的（各向同性），大了碗是扁椭球（各向异性）。
- **条件数与优化难度**：想象一个深谷，东西方向平坦（小曲率），南北方向陡峭（大曲率）。梯度下降在东西方向走得很慢（收敛慢），在南北方向容易 overshoot。这就是条件数大的后果。牛顿法通过 Hessian 逆将问题"圆化"，消除了条件数的影响。
- **收敛速度的直观理解**：次线性收敛就像在平地上接近目标——前几步很快，但越靠近越慢。线性收敛像在有摩擦的斜坡上下滑——每一步缩小固定比例。二次收敛像从山顶加速滚下——越靠近目标越快。

## 代码示例

```python
import numpy as np

# 对比不同条件数下的梯度下降收敛
def gradient_descent(grad, x0, eta, n_iter):
    x = x0.copy()
    errors = []
    for _ in range(n_iter):
        x = x - eta * grad(x)
        errors.append(np.linalg.norm(x)**2)  # 假设最优解在原点
    return x, errors

# 两个二次函数：条件数不同
# f1(x,y) = x^2 + y^2 (条件数=1)
# f2(x,y) = x^2 + 100*y^2 (条件数=100)

def grad1(x):
    return np.array([2*x[0], 2*x[1]])

def grad2(x):
    return np.array([2*x[0], 200*x[1]])

x0 = np.array([5.0, 1.0])
n_iter = 100

# 条件数 = 1
_, errors1 = gradient_descent(grad1, x0, eta=0.1, n_iter=n_iter)
# 条件数 = 100
_, errors2 = gradient_descent(grad2, x0, eta=0.01, n_iter=n_iter)

print(f"条件数=1: 100步后误差={errors1[-1]:.6e}")
print(f"条件数=100: 100步后误差={errors2[-1]:.6e}")

# 证明 Lipschitz 光滑性
def is_l_smooth(f, grad, x, y, L):
    """验证 L-smooth 性质: f(y) <= f(x) + grad(x)^T(y-x) + L/2*||y-x||^2"""
    lhs = f(y)
    rhs = f(x) + grad(x) @ (y - x) + L/2 * np.linalg.norm(y-x)**2
    return lhs <= rhs + 1e-10  # 允许数值误差

# 验证强凸性
def is_strongly_convex(f, grad, x, y, mu):
    """验证 mu-强凸: f(y) >= f(x) + grad(x)^T(y-x) + mu/2*||y-x||^2"""
    lhs = f(y)
    rhs = f(x) + grad(x) @ (y - x) + mu/2 * np.linalg.norm(y-x)**2
    return lhs >= rhs - 1e-10

# 最优步长分析
L, mu = 200, 2  # 条件数 100
eta_opt = 2 / (L + mu)  # 最优固定步长
eta_small = 1 / L       # 保守步长

print(f"\n最优步长: {eta_opt:.4f}")
print(f"保守步长: {eta_small:.4f}")

# 验证 GD 的单调下降性质 (凸+L-smooth，eta <= 1/L)
f = lambda x: x[0]**2 + 100*x[1]**2
x = np.array([5.0, 1.0])
for t in range(20):
    x_new = x - 1/L * grad2(x)
    assert f(x_new) <= f(x) + 1e-10, f"第{t}步违背下降性质"
    x = x_new
print("GD 单调下降性质验证通过!")
```

## 深度学习关联

- **深度网络的非凸性**：深度学习的损失函数通常是非凸的，经典的凸优化收敛性理论不能直接应用。然而，实验发现 SGD 在非凸深度网络上仍能有效收敛。这催生了"非凸优化的隐式正则化"理论——SGD 本身有偏袒性地选择"平坦极小值"。
- **SGD 的收敛性**：随机梯度下降每次只用一批数据估计梯度，引入噪声。其收敛速度为 $O(1/\sqrt{k})$（在凸情形下），慢于全梯度下降的 $O(1/k)$。但 SGD 的噪声有助于逃离鞍点，在非凸深度学习中反而可能找到更好的解。
- **自适应方法的动机**：深度网络的 Hessian 在不同方向上的特征值差异极大（条件数巨大），固定学习率的 GD/SGD 收敛极慢。Adam 等自适应方法为每个参数维护独立的学习率，等价于隐式地估计并抵消了 Hessian 的对角线缩放效应。
- **批量归一化 (Batch Normalization)**：BN 层的一个重要效果是改善了优化 landscape 的条件数。BN 使梯度 Lipschitz 常数更小、梯度预测更可靠，从而允许使用更大的学习率并加速收敛。这间接验证了条件数对收敛速度的关键影响。
