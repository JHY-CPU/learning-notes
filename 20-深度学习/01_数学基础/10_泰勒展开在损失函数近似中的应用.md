# 11_泰勒展开在损失函数近似中的应用

## 核心概念

- **泰勒展开 (Taylor Expansion)**：将一个光滑函数在某点附近展开为多项式级数。一元函数在 $x=a$ 处的展开为 $f(x) = \sum_{k=0}^{\infty} \frac{f^{(k)}(a)}{k!}(x-a)^k$。阶数越高，近似越精确。
- **一阶近似与梯度下降**：$f(\mathbf{x}+\Delta\mathbf{x}) \approx f(\mathbf{x}) + \nabla f(\mathbf{x})^T \Delta\mathbf{x}$。在最陡下降方向选取 $\Delta\mathbf{x} = -\eta \nabla f$，这就是梯度下降法。
- **二阶近似与牛顿法**：$f(\mathbf{x}+\Delta\mathbf{x}) \approx f(\mathbf{x}) + \nabla f^T \Delta\mathbf{x} + \frac{1}{2}\Delta\mathbf{x}^T H \Delta\mathbf{x}$。令导数等于零得 $\Delta\mathbf{x} = -H^{-1}\nabla f$，即牛顿法。
- **拉格朗日余项**：泰勒展开的误差项为 $R_n = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-a)^{n+1}$，其中 $\xi$ 在 $x$ 和 $a$ 之间。理解误差大小对于选择学习率和判断近似有效性至关重要。
- **多元泰勒展开**：多元函数的泰勒展开涉及梯度（一阶）和海森矩阵（二阶），公式为 $f(\mathbf{x}+\mathbf{p}) = f(\mathbf{x}) + \mathbf{p}^T \nabla f + \frac{1}{2}\mathbf{p}^T H \mathbf{p} + O(\|\mathbf{p}\|^3)$。

## 数学推导

一元函数 $n$ 阶泰勒展开：
$$
f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \dots + \frac{f^{(n)}(a)}{n!}(x-a)^n + R_n
$$

多元函数在 $\mathbf{x}_0$ 附近的二阶泰勒展开：
$$
f(\mathbf{x}) \approx f(\mathbf{x}_0) + \nabla f(\mathbf{x}_0)^T (\mathbf{x} - \mathbf{x}_0) + \frac{1}{2}(\mathbf{x} - \mathbf{x}_0)^T H(\mathbf{x}_0) (\mathbf{x} - \mathbf{x}_0)
$$

展开为求和形式：
$$
f(x_1,\dots,x_n) \approx f(\mathbf{x}_0) + \sum_{i=1}^n \frac{\partial f}{\partial x_i}\Delta x_i + \frac{1}{2}\sum_{i=1}^n\sum_{j=1}^n \frac{\partial^2 f}{\partial x_i \partial x_j}\Delta x_i \Delta x_j
$$

## 直观理解

- **局部近似工具**：泰勒展开就好比用放大镜看曲线。一阶近似是用切线代替曲线，二阶近似是用抛物线（切线+曲率）代替曲线。离展开点越近，近似越准确。
- **为什么需要二阶信息**：一阶近似只能告诉你"往哪走能下降"，不能告诉你"走多大步合适"。二阶信息告诉你前方的曲率——曲率大（陡峭）的地方步子要小，曲率小（平坦）的地方步子可以大。
- **近似失效的情况**：在剧烈振荡的函数（如高频三角函数）附近，泰勒展开需要很高阶才能准确近似；在不可微点（如 ReLU 在 0 点附近）处，泰勒展开不适用。深度学习中使用次梯度或光滑化来处理。

## 代码示例

```python
import numpy as np

# 一元函数 f(x) = sin(x) 的泰勒展开
def taylor_sin(x, n_terms):
    """sin(x) 的泰勒展开，在 a=0 处展开"""
    approx = 0.0
    for k in range(n_terms):
        # sin 的 2k+1 阶导数在 0 处是 (-1)^k
        coeff = (-1)**k / np.math.factorial(2*k + 1)
        approx += coeff * x**(2*k + 1)
    return approx

x = 0.5
true_val = np.sin(x)
print(f"sin({x}) 真实值 = {true_val:.6f}")
for n in [1, 2, 3, 5]:
    approx = taylor_sin(x, n)
    print(f"  {n} 项展开: {approx:.6f}, 误差: {abs(true_val - approx):.2e}")

# 多元函数的二阶近似演示
def f(xy):
    x, y = xy
    return x**2 + 3*y**2 + x*y

def grad(xy):
    x, y = xy
    return np.array([2*x + y, 6*y + x])

def hessian(xy):
    return np.array([[2, 1], [1, 6]])

# 在 (1, 1) 处展开
x0 = np.array([1.0, 1.0])
delta = np.array([0.1, -0.1])

true_val = f(x0 + delta)
approx1 = f(x0) + grad(x0) @ delta
approx2 = approx1 + 0.5 * delta @ hessian(x0) @ delta

print(f"\n多元函数 f(x,y)=x^2+3y^2+xy")
print(f"真实值: {true_val:.6f}")
print(f"一阶近似: {approx1:.6f}, 误差: {abs(true_val - approx1):.2e}")
print(f"二阶近似: {approx2:.6f}, 误差: {abs(true_val - approx2):.2e}")

# 学习率与泰勒展开：过大的学习率导致一阶近似失效
def loss(x):
    return x**4  # 很陡的四次函数

x_cur = 1.0
g = 4 * x_cur**3  # 梯度

for lr in [0.01, 0.05, 0.1]:
    x_next = x_cur - lr * g
    true_dec = loss(x_cur) - loss(x_next)
    # 一阶近似预测的下降: lr * g^2 (正比于梯度范数的平方)
    pred_dec = lr * g**2
    print(f"lr={lr:.2f}: 真实下降={true_dec:.4f}, 一阶预测下降={pred_dec:.4f}, "
          f"比值={true_dec/pred_dec:.2f}")
```

## 深度学习关联

- **学习率选择的依据**：泰勒展开为学习率选择提供了理论基础。如果学习率 $\eta$ 太大，一阶近似失效，损失可能不降反升。理想学习率满足局部 Lipschitz 条件：$\eta < 2 / L$，其中 $L$ 是梯度 Lipschitz 常数（等价于 Hessian 最大特征值的上界）。
- **Polyak 动量 (Momentum)**：带动量的 SGD 可以视为泰勒展开到"二阶"的近似改进。动量项 $v_{t+1} = \beta v_t + \nabla L$ 利用了历史梯度的信息，相当于对损失函数的局部曲率进行了隐式建模，从而加速收敛。
- **二阶优化稀缺性**：尽管二阶近似能提供更好的收敛性，但深度学习中参数规模巨大（百万到千亿级），显式计算和存储 Hessian 矩阵不可行。因此实践中普遍使用一阶方法，并通过技巧（如 Adam 的自适应学习率、梯度裁剪）来弥补缺失的曲率信息。
- **Nesterov 加速梯度 (NAG)**：NAG 先利用动量"预览"下一步位置，再在该位置计算梯度，相当于在泰勒展开中用"未来点"的曲率信息修正当前梯度方向，提高了对损失曲面形状的适应性。
