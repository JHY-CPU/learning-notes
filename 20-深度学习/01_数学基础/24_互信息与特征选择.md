# 24_互信息与特征选择

## 核心概念

- **互信息 (Mutual Information)**：衡量两个随机变量 $X$ 和 $Y$ 之间共享的信息量，$I(X; Y) = \sum\sum p(x,y) \log \frac{p(x,y)}{p(x)p(y)}$。互信息等于 $H(X) - H(X|Y) = H(Y) - H(Y|X)$。
- **互信息与相关性**：互信息捕捉任何类型的关系（线性、非线性、高阶交互），而相关系数只捕捉线性关系。$I(X;Y) = 0$ 当且仅当 $X$ 和 $Y$ 独立。
- **互信息与 KL 散度**：$I(X;Y) = D_{\text{KL}}(p(x,y)\|p(x)p(y))$，即联合分布与边缘分布乘积之间的 KL 散度。这量化了"了解 $Y$ 让 $X$ 的不确定性减少多少"。
- **条件互信息**：$I(X;Y|Z) = H(X|Z) - H(X|Y,Z)$，衡量在已知 $Z$ 的条件下 $X$ 和 $Y$ 之间的互信息。用于理解变量间的条件依赖结构。
- **特征选择中的互信息**：互信息可以直接衡量特征 $X_i$ 与标签 $Y$ 的关联强度，$I(X_i;Y)$ 越大，该特征对预测越重要。这克服了相关系数只能捕捉线性关系的局限。

## 数学推导

互信息的定义：
$$
I(X; Y) = \sum_{x \in \mathcal{X}} \sum_{y \in \mathcal{Y}} p(x,y) \log \frac{p(x,y)}{p(x)p(y)}
$$

连续变量的互信息：
$$
I(X; Y) = \iint p(x,y) \log \frac{p(x,y)}{p(x)p(y)} dx dy
$$

互信息与熵的关系：
$$
I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)
$$

因此：$I(X; Y) \geq 0$，且 $I(X; Y) \leq \min(H(X), H(Y))$。

互信息与 KL 散度的关系：
$$
I(X; Y) = D_{\text{KL}}(p(x,y) \| p(x)p(y))
$$

信息瓶颈 (Information Bottleneck) 目标函数：
$$
\min_{p(t|x)} I(X; T) - \beta I(T; Y)
$$
其中 $T$ 是 $X$ 的压缩表示，$\beta$ 控制压缩与预测的权衡。

## 直观理解

- **信息重叠的维恩图**：互信息就像两个圆圈重叠的部分——$X$ 的信息和 $Y$ 的信息的公共部分。重叠越大，知道 $X$ 就获得了越多关于 $Y$ 的信息。
- **互信息 vs 相关性**：想象一个完美的二次关系 $Y = X^2$，$X \in [-1,1]$。相关系数可能为 0（线性无关），但互信息很大（知道 $X$ 就完全知道 $Y$）。互信息能够捕捉这种非线性依赖。
- **特征选择中的互信息**：在猫狗分类任务中，特征"是否有耳朵"与标签的互信息很小（两者都有耳朵），而"胡须长度"的互信息可能较大。互信息能自动找到最区分的特征，不需要人工设计。

## 代码示例

```python
import numpy as np
from sklearn.feature_selection import mutual_info_classif, mutual_info_regression
from sklearn.datasets import make_classification

# 1. 简单互信息计算（离散变量）
def discrete_mutual_information(x, y):
    """计算离散变量的互信息"""
    n = len(x)
    # 联合分布 p(x,y)
    joint = np.zeros((len(np.unique(x)), len(np.unique(y))))
    for xi, yi in zip(x, y):
        joint[xi, yi] += 1
    joint /= n
    
    # 边缘分布
    p_x = joint.sum(axis=1)
    p_y = joint.sum(axis=0)
    
    # 互信息
    mi = 0
    for i in range(len(p_x)):
        for j in range(len(p_y)):
            if joint[i,j] > 0:
                mi += joint[i,j] * np.log(joint[i,j] / (p_x[i] * p_y[j]))
    return mi

# 验证：完全相关的变量
x = np.array([0, 0, 1, 1, 0, 1])
y = np.array([0, 0, 1, 1, 0, 1])  # y = x
print(f"I(X; X) = {discrete_mutual_information(x, y):.4f} (应 ≈ H(X))")

# 验证：独立变量
np.random.seed(42)
x_ind = np.random.randint(0, 2, 1000)
y_ind = np.random.randint(0, 2, 1000)
print(f"I(X; Y) 独立变量 = {discrete_mutual_information(x_ind, y_ind):.4f} (应 ≈ 0)")

# 2. 连续特征的互信息（用于特征选择）
X, y = make_classification(n_samples=500, n_features=10, n_informative=4,
                            n_redundant=3, n_repeated=0, random_state=42)

mi_scores = mutual_info_classif(X, y, random_state=42)
print(f"\n特征选择: 互信息得分")
for i, score in enumerate(mi_scores):
    marker = "***" if score > 0.1 else "   "
    print(f"  特征 {i+1}: {score:.4f} {marker}")

# 3. 非线性相关检测
np.random.seed(42)
x_lin = np.random.randn(500)
y_lin = 2 * x_lin + 0.5 * np.random.randn(500)  # 线性关系

x_nonlin = np.random.randn(500)
y_nonlin = x_nonlin**2 + 0.2 * np.random.randn(500)  # 二次关系

x_noise = np.random.randn(500)
y_noise = np.random.randn(500)  # 独立

from scipy.stats import pearsonr

print("\n互信息 vs 相关系数 (非线性检测):")
for name, x, y in [("线性", x_lin, y_lin), ("二次", x_nonlin, y_nonlin), ("噪声", x_noise, y_noise)]:
    corr, _ = pearsonr(x, y)
    mi = mutual_info_regression(x.reshape(-1,1), y, random_state=42)[0]
    print(f"  {name}: 相关系数={corr:.4f}, 互信息={mi:.4f}")

# 4. 互信息的熵分解
from scipy.stats import entropy

# 近似计算连续变量的熵（用直方图）
def approximate_entropy(data, bins=20):
    hist, _ = np.histogram(data, bins=bins, density=True)
    hist = hist / hist.sum()
    return entropy(hist)

x_big = np.random.randn(10000)
y_big = x_big + 0.5 * np.random.randn(10000)

H_x = approximate_entropy(x_big)
H_y = approximate_entropy(y_big)
H_xy = approximate_entropy(x_big + y_big, bins=30)  # 近似联合熵

mi_approx = H_x + H_y - H_xy
print(f"\n熵分解 I(X;Y) = H(X)+H(Y)-H(X,Y):")
print(f"  I(X;Y) ≈ {mi_approx:.4f}")
```

## 深度学习关联

- **信息瓶颈理论 (Information Bottleneck)**：信息瓶颈理论（Tishby 等）认为深度神经网络的学习过程分为两个阶段：先是"拟合"阶段（$I(X;T)$ 和 $I(T;Y)$ 都增加），然后是"压缩"阶段（$I(X;T)$ 下降但 $I(T;Y)$ 继续上升或保持）。这意味着网络学会了丢弃输入中与任务无关的信息，只保留对预测有用的信息。
- **对比学习中的互信息最大化**：SimCLR、MoCo 等对比学习方法的核心是最大化同一图像不同视角表示之间的互信息 $I(z_i; z_j)$。通过对比正负样本，模型学习到的表示对数据增强方式保持不变（即丢弃了与语义无关的变体信息）。
- **互信息神经估计 (MINE)**：MINE (Mutual Information Neural Estimation) 使用神经网络直接估计高维变量之间的互信息 $I(X;Y) \approx \sup_\theta \mathbb{E}[T_\theta] - \log \mathbb{E}[e^{T_\theta}]$，无需知道分布形式。这在表示学习和生成模型中尤其有用。
- **特征选择与模型解释**：互信息可以用于解释神经网络的决策——计算输入特征与预测类别的互信息来识别哪些输入区域对决策最重要。这等价于基于信息论的解释性方法，如通过遮挡或扰动评估特征重要性。
