# 18_反向去噪过程与神经网络预测

## 核心概念

- **反向去噪过程**：从纯噪声 $x_T$ 开始，通过训练好的神经网络逐步去噪，最终得到干净样本 $x_0$。每一步去除一小部分噪声。
- **神经网络角色**：在 DDPM 中，神经网络 $\epsilon_\theta(x_t, t)$ 预测的是添加到 $x_t$ 的噪声 $\epsilon$，而非直接预测 $x_0$ 或 $x_{t-1}$。
- **时间步条件**：网络必须知道当前是第几步（$t$），因为不同时间步的噪声强度和去噪策略不同。使用正弦位置编码（类似 Transformer 的 positional encoding）将 $t$ 嵌入到网络中。
- **U-Net 架构**：标准扩散模型使用 U-Net 作为去噪网络——编码器提取多尺度特征，解码器恢复原尺寸，跳跃连接保留细节。
- **预测目标的选择**：DDPM 选择预测噪声 $\epsilon$；替代方案包括直接预测 $x_0$ 或预测得分 $\nabla_x \log p(x_t)$，三者之间存在数学等价关系。
- **降噪调度**：反向过程的方差 $\sigma_t^2$ 可以是固定的（$\beta_t$ 或 $\tilde{\beta}_t$）或可学习的，Improved DDPM 发现学习方差能提升对数似然。

## 数学推导

**反向去噪的更新公式**：

给定预测噪声 $\epsilon_\theta(x_t, t)$，去噪步骤如下：

$$
x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right) + \sigma_t z
$$

其中 $z \sim \mathcal{N}(0, I)$ 在 $t>1$ 时使用，$t=1$ 时 $z=0$。

**从噪声预测推导 $x_0$ 预测**：

如果预测了噪声 $\epsilon_\theta$，可以估计 $x_0$：

$$
\hat{x}_0 = \frac{x_t - \sqrt{1-\bar{\alpha}_t} \epsilon_\theta(x_t, t)}{\sqrt{\bar{\alpha}_t}}
$$

这个 $\hat{x}_0$ 可以用于计算更精确的去噪方向。

**U-Net 的结构**：

编码器：$x_t \to \text{Conv}(64) \to \text{Conv}(128) \to \text{Conv}(256) \to \text{Conv}(512)$

瓶颈：$\text{Self-Attention} \to \text{Conv}(512) \to \text{Self-Attention}$

解码器：$\text{ConvTranspose}(256) \to \text{ConvTranspose}(128) \to \text{ConvTranspose}(64) \to \text{Conv}(3)$

每层解码器通过跳跃连接拼接对应编码器的特征图。

**神经网络输入**：$x_t \in \mathbb{R}^{H \times W \times C}$（或潜空间维度）+ $t \in \mathbb{R}$（时间步嵌入）

**神经网络输出**：$\epsilon_\theta(x_t, t) \in \mathbb{R}^{H \times W \times C}$（预测的噪声，与输入尺寸相同）

**多尺度特征的重要性**：浅层特征保留细节（纹理、边缘），深层特征捕获语义（物体形状、场景结构）。U-Net 的跳跃连接使网络同时利用这两种信息。

## 直观理解

- **去噪网络 = 经验丰富的修复专家**：给专家一张严重损坏的旧照片（$x_t$），并告诉他"这是第 800 步的损坏程度"（$t$），他能判断出"损坏的随机噪声是什么样的"（$\epsilon_\theta$），然后据此修复。
- **为什么需要告诉网络当前步数**：同样的残缺照片，如果损坏程度不同，修复策略也不同——重度损坏需要大胆猜测整体结构，轻微损坏只需微调细节。如果不告诉网络步数，它会无所适从。
- **U-Net 的对称结构 = 素描与上色的配合**：编码器像画速写（提取重要特征），解码器像上色（恢复细节）。跳跃连接确保"上色"时不会丢失"速写"中的精细线条。
- **从粗到细的去噪过程**：早期步骤（大 $t$）去除大的噪声斑点，决定图像的整体结构；中期步骤（中 $t$）细化纹理；后期步骤（小 $t$）做最后的细节打磨。网络自动学会了这种分级去噪策略。

## 代码示例

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class TimeEmbedding(nn.Module):
    """时间步嵌入模块"""
    def __init__(self, dim):
        super().__init__()
        self.dim = dim
        self.mlp = nn.Sequential(
            nn.Linear(dim, dim * 4),
            nn.SiLU(),
            nn.Linear(dim * 4, dim),
        )
    
    def forward(self, t):
        # 正弦编码
        half = self.dim // 2
        freqs = torch.exp(-math.log(10000) * torch.arange(half, device=t.device) / half)
        emb = t[:, None].float() * freqs[None, :]
        emb = torch.cat([torch.sin(emb), torch.cos(emb)], dim=-1)
        return self.mlp(emb)

class ResBlock(nn.Module):
    """带有时间嵌入的残差块"""
    def __init__(self, in_ch, out_ch, time_emb_dim):
        super().__init__()
        self.time_proj = nn.Linear(time_emb_dim, out_ch)
        
        self.conv1 = nn.Conv2d(in_ch, out_ch, 3, 1, 1)
        self.norm1 = nn.GroupNorm(32, out_ch)
        self.conv2 = nn.Conv2d(out_ch, out_ch, 3, 1, 1)
        self.norm2 = nn.GroupNorm(32, out_ch)
        
        self.shortcut = nn.Conv2d(in_ch, out_ch, 1) if in_ch != out_ch else nn.Identity()
        self.act = nn.SiLU()
    
    def forward(self, x, t_emb):
        h = self.act(self.norm1(self.conv1(x)))
        # 注入时间信息
        time_shift = self.time_proj(self.act(t_emb))[:, :, None, None]
        h = h + time_shift
        h = self.act(self.norm2(self.conv2(h)))
        return self.shortcut(x) + h

class DenoisingUNet(nn.Module):
    """用于去噪的全功能 U-Net"""
    def __init__(self, in_channels=3, base_channels=64, time_emb_dim=256):
        super().__init__()
        self.time_embed = TimeEmbedding(time_emb_dim)
        
        # 编码器
        self.enc_in = nn.Conv2d(in_channels, base_channels, 3, 1, 1)
        self.enc1 = ResBlock(base_channels, base_channels, time_emb_dim)
        self.enc2 = nn.Sequential(
            nn.Conv2d(base_channels, base_channels * 2, 3, 2, 1),
            ResBlock(base_channels * 2, base_channels * 2, time_emb_dim),
        )
        self.enc3 = nn.Sequential(
            nn.Conv2d(base_channels * 2, base_channels * 4, 3, 2, 1),
            ResBlock(base_channels * 4, base_channels * 4, time_emb_dim),
        )
        
        # 瓶颈（自注意力）
        self.bottleneck = nn.Sequential(
            ResBlock(base_channels * 4, base_channels * 4, time_emb_dim),
            nn.MultiheadAttention(base_channels * 4, 4, batch_first=True),
            ResBlock(base_channels * 4, base_channels * 4, time_emb_dim),
        )
        
        # 解码器
        self.dec3 = nn.Sequential(
            nn.ConvTranspose2d(base_channels * 4, base_channels * 2, 4, 2, 1),
            ResBlock(base_channels * 2, base_channels * 2, time_emb_dim),
        )
        self.dec2 = nn.Sequential(
            nn.ConvTranspose2d(base_channels * 2, base_channels, 4, 2, 1),
            ResBlock(base_channels, base_channels, time_emb_dim),
        )
        self.dec1 = nn.Conv2d(base_channels, in_channels, 3, 1, 1)
    
    def forward(self, x, t):
        t_emb = self.time_embed(t)
        
        # 编码
        x = self.enc_in(x)
        s1 = self.enc1(x, t_emb)
        s2 = self.enc2[0](s1)
        s2 = self.enc2[1](s2, t_emb)
        s3 = self.enc3[0](s2)
        s3 = self.enc3[1](s3, t_emb)
        
        # 瓶颈
        b = self.bottleneck[0](s3, t_emb)
        B, C, H, W = b.shape
        b_flat = b.flatten(2).permute(0, 2, 1)
        b_attn, _ = self.bottleneck[1](b_flat, b_flat, b_flat)
        b = b_attn.permute(0, 2, 1).reshape(B, C, H, W)
        b = self.bottleneck[2](b, t_emb)
        
        # 解码（带跳跃连接）
        d3 = self.dec3[0](b)
        d3 = self.dec3[1](d3[:, :, :s2.size(2), :s2.size(3)] + s2, t_emb)
        d2 = self.dec2[0](d3)
        d2 = self.dec2[1](d2[:, :, :s1.size(2), :s1.size(3)] + s1, t_emb)
        out = self.dec1(d2)
        
        return out

# 演示反向去噪过程
def denoising_step(model, x_t, t, alpha_bar_t, alpha_bar_t_1, alpha_t, beta_t):
    """单步去噪：从 x_t 到 x_{t-1}"""
    t_tensor = torch.full((x_t.size(0),), t, device=x_t.device, dtype=torch.long)
    predicted_noise = model(x_t, t_tensor)
    
    # 方法1：通过预测噪声去噪
    x_0_estimated = (x_t - torch.sqrt(1 - alpha_bar_t) * predicted_noise) / torch.sqrt(alpha_bar_t)
    x_0_estimated = torch.clamp(x_0_estimated, -1, 1)
    
    # 计算 x_{t-1}
    coeff1 = 1 / torch.sqrt(alpha_t)
    coeff2 = beta_t / torch.sqrt(1 - alpha_bar_t)
    
    x_t_minus_1 = coeff1 * (x_t - coeff2 * predicted_noise)
    
    if t > 0:
        noise = torch.randn_like(x_t)
        x_t_minus_1 += torch.sqrt(beta_t) * noise
    
    return x_t_minus_1, predicted_noise

# 验证去噪网络输出形状
model = DenoisingUNet(in_channels=3)
x_t = torch.randn(4, 3, 32, 32)
t = torch.randint(0, 1000, (4,))
pred_noise = model(x_t, t)
print(f"输入形状: {x_t.shape}")
print(f"预测噪声形状: {pred_noise.shape}")
print(f"预测噪声范围: [{pred_noise.min():.3f}, {pred_noise.max():.3f}]")
print(f"模型总参数量: {sum(p.numel() for p in model.parameters()):,}")
```

## 深度学习关联

- **预测目标的选择**：DDPM 预测噪声 $\epsilon$，但后续工作如 ADM 和 DALL-E 2 也尝试直接预测 $x_0$。两者等价但各有优劣——$\epsilon$ 预测在大噪声时更稳定，$x_0$ 预测在小噪声时更准确。
- **Classifier-Free Guidance (CFG)**：将无条件与条件预测结合：$\tilde{\epsilon} = \epsilon_\theta(x_t, t, \emptyset) + w(\epsilon_\theta(x_t, t, c) - \epsilon_\theta(x_t, t, \emptyset))$，通过 $w>1$ 放大条件的影响，大幅提升文本到图像的对齐度。
- **U-Net 的替代架构**：DiT (Diffusion Transformer) 用纯 Transformer 替代 U-Net，将图像划分为 patch 后使用自注意力，在大规模计算下表现更好（scaling law）。
- **一致性模型 (Consistency Model)**：直接将 $x_T$ 映射到 $x_0$（一步生成），通过蒸馏预训练的扩散模型来学习这种映射，实现单步采样。
