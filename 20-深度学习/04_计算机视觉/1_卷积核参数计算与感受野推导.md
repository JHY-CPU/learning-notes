# 02_卷积核参数计算与感受野推导

## 核心概念

- **卷积层参数量**：对于一个卷积层，输入通道数为 $C_{in}$，输出通道数为 $C_{out}$，卷积核尺寸为 $K \times K$，参数量为 $C_{in} \times C_{out} \times K \times K + C_{out}$（含偏置）。
- **感受野（Receptive Field）定义**：感受野是指输出特征图上的一个像素点对应到输入图像上的区域大小。感受野越大，网络能捕捉的上下文信息越丰富。
- **感受野递推公式**：若第 $l$ 层卷积核大小为 $k_l$，步长为 $s_l$，则感受野的递推公式（从底层向顶层）为 $r_l = r_{l-1} + (k_l - 1) \times \prod_{i=1}^{l-1} s_i$，其中 $r_0 = 1$。
- **堆叠小卷积核的优势**：两个 $3\times3$ 卷积核堆叠可达到 $5\times5$ 的感受野，三个堆叠可达到 $7\times7$，但卷积核元素更少（$3\times3\times3=27$ vs $7\times7=49$，单通道情况下），且引入更多非线性。
- **有效感受野（Effective Receptive Field）**：理论感受野中心位置的像素贡献更大，边缘贡献趋于零，实际有效区域远小于理论值，呈高斯分布。
- **参数共享**：卷积核在输入空间上滑动时参数完全共享，使得参数量与输入尺寸无关，这是CNN相比全连接网络的关键优势。

## 数学推导

$$
\text{参数量（含偏置）} = C_{in} \times C_{out} \times K_h \times K_w + C_{out}
$$

**感受野递推公式（从底层向顶层）：**

$$
r_l = r_{l-1} + (k_l - 1) \times \prod_{i=1}^{l-1} s_i
$$

其中 $r_0 = 1$（输入层自身），$\prod_{i=1}^{0} s_i = 1$。

**等价展开形式：**

$$
r_l = r_{l-1} + (k_l - 1) \times \prod_{i=1}^{l-1} s_i
$$

**例子：VGG16 堆叠 3 个 $3\times3$ 卷积（步长=1）**
- Layer 1: $r_1 = 1 + (3-1) \times 1 = 3$
- Layer 2: $r_2 = 3 + (3-1) \times 1 = 5$
- Layer 3: $r_3 = 5 + (3-1) \times 1 = 7$

可见三个 $3\times3$ 卷积堆叠后的感受野为 $7\times7$，等价于一个 $7\times7$ 卷积。

## 直观理解

想象一下，你在看一张照片，但只能通过一个小孔观察。如果你移动小孔，你看到的是照片的不同局部区域。卷积核就像这个小孔。当你叠加多个卷积层时，相当于你通过多层小孔来观察——后一层能看到前一层多个位置的组合信息，从而"间接"看到了更大的原始图像区域。这就像通过拼接多个局部观察来还原全局画面。有效感受野的概念提醒我们，虽然理论上能看到的范围很大，但真正有用的信息往往集中在中心区域。

## 代码示例

```python
import torch.nn as nn

# 计算一个卷积层的参数量
conv_layer = nn.Conv2d(in_channels=3, out_channels=64, kernel_size=3, padding=1, bias=True)
total_params = sum(p.numel() for p in conv_layer.parameters())
print(f"3x3卷积 (3→64) 参数量: {total_params}")  # 输出: 3*64*3*3 + 64 = 1792

# 比较堆叠小卷积核 vs 大卷积核
stacked_small = nn.Sequential(
    nn.Conv2d(3, 64, 3, padding=1),  # 参数量: 3*64*9 + 64 = 1792
    nn.ReLU(),
    nn.Conv2d(64, 64, 3, padding=1), # 参数量: 64*64*9 + 64 = 36928
    nn.ReLU(),
    nn.Conv2d(64, 128, 3, padding=1) # 参数量: 64*128*9 + 128 = 73856
)
small_total = sum(p.numel() for p in stacked_small.parameters())
print(f"三个3x3卷积总参数量: {small_total}")

large_kernel = nn.Conv2d(3, 128, 7, padding=3)  # 直接7x7
large_total = sum(p.numel() for p in large_kernel.parameters())
print(f"一个7x7卷积参数量: {large_total}")
# 堆叠小卷积核通常参数量更少且非线性更强
```

## 深度学习关联

- **VGGNet的设计哲学**：VGGNet全面采用 $3\times3$ 小卷积核堆叠，证明了小卷积核堆叠在参数量效率和表达能力上均优于大卷积核，这一设计被后续几乎所有CNN模型沿用。
- **感受野与任务需求**：图像分类需要较大的感受野来识别整体物体，而语义分割需要在保持高分辨率的同时扩大感受野（如使用空洞卷积）。
- **现代架构中的感受野**：Vision Transformer (ViT) 通过全局自注意力实现整个图像的感受野，克服了CNN中感受野受限的问题；而Swin Transformer通过窗口注意力和移位窗口机制在局部和全局感受野之间取得平衡。
