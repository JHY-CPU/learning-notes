# 67_相机标定与内参及外参矩阵

## 核心概念

- **相机标定（Camera Calibration）**：确定相机内参（焦距、主点、畸变系数）和外参（相机在世界坐标系中的位置和朝向）的过程。标定是将2D图像像素坐标与3D世界坐标关联的基础。
- **内参矩阵（Intrinsic Matrix）**：描述相机内部几何特性的 $3\times3$ 矩阵 $K$，包含焦距 $(f_x, f_y)$、主点 $(c_x, c_y)$ 和倾斜因子 $s$。
- **外参矩阵（Extrinsic Matrix）**：描述相机在世界坐标系中的姿态——由一个 $3\times3$ 旋转矩阵 $R$ 和一个 $3\times1$ 平移向量 $t$ 组成，将世界坐标转换为相机坐标。
- **针孔相机模型**：描述3D点通过光心投影到2D像平面的过程。是相机标定的基本模型。包含投影中心（光心）、像平面和光轴。
- **畸变参数（Distortion Coefficients）**：真实相机存在镜头畸变，主要是径向畸变（桶形/枕形）和切向畸变（镜头与像面不平行）。畸变参数用于校正镜头畸变。
- **张正友标定法（Zhang's Method）**：最广泛使用的相机标定方法——拍摄多张不同姿态的棋盘格图像，利用棋盘格的已知几何结构（角点位置），通过闭合解+非线性优化求解内外参。

## 数学推导

**针孔相机模型（世界坐标→像素坐标）：**

世界坐标 $P_w = (X, Y, Z, 1)^T$ → 相机坐标 $P_c = (X_c, Y_c, Z_c)^T$ → 像素坐标 $(u, v)^T$：
$$
\begin{bmatrix} u \\ v \\ 1 \end{bmatrix} = K \cdot [R | t] \cdot \begin{bmatrix} X \\ Y \\ Z \\ 1 \end{bmatrix}
$$

**内参矩阵 $K$：**
$$
K = \begin{bmatrix} f_x & s & c_x \\ 0 & f_y & c_y \\ 0 & 0 & 1 \end{bmatrix}
$$

其中 $f_x = F / p_x$，$f_y = F / p_y$ 是以像素为单位的焦距（$F$ 是物理焦距，$p_x, p_y$ 是像素尺寸），$(c_x, c_y)$ 是像主点坐标（通常接近图像中心）。

**外参矩阵 $[R | t]$：**
$$
[R | t] = \begin{bmatrix} r_{11} & r_{12} & r_{13} & t_x \\ r_{21} & r_{22} & r_{23} & t_y \\ r_{31} & r_{32} & r_{33} & t_z \end{bmatrix}
$$

其中 $R$ 是 $3\times3$ 正交旋转矩阵，$t$ 是平移向量。

**径向畸变校正：**
$$
x_{corrected} = x(1 + k_1 r^2 + k_2 r^4 + k_3 r^6)
$$
$$
y_{corrected} = y(1 + k_1 r^2 + k_2 r^4 + k_3 r^6)
$$

其中 $(x, y)$ 是归一化图像坐标，$r^2 = x^2 + y^2$，$k_1, k_2, k_3$ 是径向畸变系数。

**切向畸变校正：**
$$
x_{corrected} = x + 2p_1 xy + p_2(r^2 + 2x^2)
$$
$$
y_{corrected} = y + p_1(r^2 + 2y^2) + 2p_2 xy
$$

其中 $p_1, p_2$ 是切向畸变系数。

## 直观理解

相机标定可以理解为"告诉计算机你的相机是如何工作的"。内参描述的是"从镜头到图像"的映射——焦距决定了视角大小（广角?长焦?），主点决定了图像的中心在哪，畸变系数描述了镜头导致的图像变形（像通过鱼缸看世界）。

外参描述的是"相机在空间中的位置和朝向"——相机是朝前还是朝下？在什么高度？朝向何方？就像告诉你"相机放在这里，朝那个方向看"。

张正友标定法用一个棋盘格来完成标定——棋盘格上的角点位置是已知的（等距排列的方格交点）。就像用一把已知尺寸的尺子在不同位置和角度拍照，通过图像中的"尺子"变形来反推相机的内外参数。

## 代码示例

```python
import torch
import math

class PinholeCamera:
    """针孔相机模型"""
    def __init__(self, fx, fy, cx, cy, img_width, img_height):
        self.fx = fx
        self.fy = fy
        self.cx = cx
        self.cy = cy
        self.img_width = img_width
        self.img_height = img_height
        
        # 内参矩阵
        self.K = torch.tensor([[fx, 0, cx],
                                [0, fy, cy],
                                [0, 0, 1]], dtype=torch.float32)
        self.K_inv = torch.inverse(self.K)
    
    def project(self, points_3d, R=None, t=None):
        """3D点投影到2D图像平面"""
        if R is None:
            R = torch.eye(3)
        if t is None:
            t = torch.zeros(3, 1)
        
        # 世界坐标 -> 相机坐标
        cam_points = R @ points_3d.T + t  # (3, N)
        
        # 透视投影
        u = cam_points[0] / cam_points[2] * self.fx + self.cx
        v = cam_points[1] / cam_points[2] * self.fy + self.cy
        
        return torch.stack([u, v], dim=1)

    def triangulate(self, points_2d_1, points_2d_2, R1, t1, R2, t2):
        """双目三角测量恢复3D点"""
        # 将2D点转换为归一化坐标
        pts1 = (points_2d_1 - torch.tensor([self.cx, self.cy])) / torch.tensor([self.fx, self.fy])
        pts2 = (points_2d_2 - torch.tensor([self.cx, self.cy])) / torch.tensor([self.fx, self.fy])
        
        # 构建线性方程组求解
        points_3d = []
        N = len(pts1)
        for i in range(N):
            A = torch.zeros(4, 4)
            A[0] = pts1[i, 0] * R1[2] - R1[0]
            A[1] = pts1[i, 1] * R1[2] - R1[1]
            A[2] = pts2[i, 0] * R2[2] - R2[0]
            A[3] = pts2[i, 1] * R2[2] - R2[1]
            _, _, V = torch.svd(A)
            points_3d.append(V[:, -1][:3] / V[:, -1][3])
        
        return torch.stack(points_3d)

def camera_calibration_demo():
    """相机标定模拟"""
    print("张正友标定法模拟:")
    print("1. 准备棋盘格 (已知角点3D坐标, Z=0)")
    print("2. 在不同位姿拍摄棋盘格图像")
    print("3. 检测棋盘格角点 (亚像素精度)")
    print("4. 利用PnP求解每张图像的外参")
    print("5. 利用所有图像联合优化内外参和畸变系数")
    print("6. 输出: K, R, t, 畸变系数(k1,k2,p1,p2,k3)")
    
    # 构造示例相机
    cam = PinholeCamera(fx=800, fy=800, cx=320, cy=240, 
                        img_width=640, img_height=480)
    
    # 模拟3D点
    points_3d = torch.randn(10, 3) * 2
    
    # 模拟外参
    angle = math.radians(15)
    R = torch.tensor([[math.cos(angle), 0, math.sin(angle)],
                      [0, 1, 0],
                      [-math.sin(angle), 0, math.cos(angle)]])
    t = torch.tensor([[0.1], [0.2], [3.0]])
    
    # 投影
    points_2d = cam.project(points_3d, R, t)
    print(f"\n3D点投影到2D:")
    for i in range(min(5, len(points_3d))):
        print(f"  3D({points_3d[i,0]:.2f}, {points_3d[i,1]:.2f}, {points_3d[i,2]:.2f}) "
              f"-> 2D({points_2d[i,0]:.1f}, {points_2d[i,1]:.1f})")
    
    print(f"\n内参矩阵 K:")
    print(cam.K)
    print(f"\n外参矩阵 [R|t]:")
    Rt = torch.cat([R, t], dim=1)
    print(Rt)
    print(f"\n投影矩阵 P = K[R|t]:")
    P = cam.K @ Rt
    print(P)

camera_calibration_demo()

print("\n畸变校正的关键公式:")
print("径向: x_corr = x * (1 + k1*r^2 + k2*r^4 + k3*r^6)")
print("切向: x_corr = x + 2*p1*xy + p2*(r^2 + 2*x^2)")
print("标定常用工具: OpenCV calibrateCamera(), MATLAB Camera Calibrator")
```

## 深度学习关联

- **深度学习中的几何先验**：相机标定参数在大量3D视觉深度学习方法中作为先验信息使用——单目深度估计（使用焦距信息解决尺度模糊）、NeRF（使用内外参生成光线）、3D重建（提供初始相机位姿）。
- **端到端相机标定**：深度学习方法可以直接从图像中学习相机标定参数（如CameraNet、PoseNet），无需棋盘格。这些方法使用大量合成数据或真实标注数据训练CNN直接预测相机内外参。
- **可微渲染中的相机参数**：在NeRF、3D Gaussian Splatting等可微渲染框架中，相机内外参构成了"渲染管线"的关键部分。由于这些参数参与梯度计算，它们可以是"可学习的"——在场景重建过程中同时优化3D内容和相机位姿（Bundle Adjustment的深度学习延伸）。
