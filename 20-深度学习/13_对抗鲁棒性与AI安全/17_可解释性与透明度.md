# 17_可解释性与透明度

## 1. 可解释性概述

可解释性 (Explainability / Interpretability) 旨在让人类理解深度学习模型的**决策依据和推理过程**，是建立对 AI 系统信任的关键。

### 1.1 为什么需要可解释性

| 需求 | 说明 |
|------|------|
| 信任建立 | 用户需要理解模型为何做出某决策 |
| 调试改进 | 发现模型学习了错误的特征 |
| 法规合规 | GDPR 等法规要求"解释权" |
| 公平性审计 | 检查决策是否基于敏感属性 |
| 安全验证 | 确保模型不会做出危险决策 |

### 1.2 方法分类

```
可解释性方法
├── 内在可解释模型
│   ├── 线性模型、决策树
│   └── 注意力机制 (有限)
├── 事后解释方法 (Post-hoc)
│   ├── 归因方法 (Attribution)
│   │   ├── SHAP
│   │   ├── LIME
│   │   ├── Integrated Gradients
│   │   └── Grad-CAM
│   ├── 概念解释
│   │   ├── TCAV
│   │   └── Concept Bottleneck
│   └── 示例解释
│       ├── 原型网络
│       └── 反事实解释
└── 可视化
    ├── 注意力可视化
    ├── 特征可视化
    └── 激活可视化
```

## 2. SHAP (SHapley Additive exPlanations)

### 2.1 理论基础

SHAP 基于博弈论中的 Shapley 值，将每个特征视为一个"玩家"，计算每个特征对预测结果的**边际贡献**：

$$\phi_i = \sum_{S \subseteq N \setminus \{i\}} \frac{|S|!(|N|-|S|-1)!}{|N|!} \left[ f(S \cup \{i\}) - f(S) \right]$$

其中 $S$ 是特征子集，$f(S)$ 是只使用特征 $S$ 的模型预测。

### 2.2 实现

```python
import shap
import torch
import numpy as np

def shap_explanation(model, background_data, test_data, feature_names=None):
    """
    SHAP 解释
    """
    # 创建解释器
    explainer = shap.DeepExplainer(model, background_data)

    # 计算 SHAP 值
    shap_values = explainer.shap_values(test_data)

    # 可视化
    shap.summary_plot(shap_values, test_data, feature_names=feature_names)

    return shap_values

def kernel_shap_explanation(predict_fn, background_data, instance):
    """
    Kernel SHAP: 模型无关的 SHAP 解释
    """
    explainer = shap.KernelExplainer(predict_fn, background_data)
    shap_values = explainer.shap_values(instance)
    return shap_values
```

## 3. LIME (Local Interpretable Model-agnostic Explanations)

### 3.1 核心思想

LIME 通过在目标样本附近生成扰动样本，用简单的可解释模型（如线性模型）来**局部近似**复杂模型的行为：

$$\xi(x) = \arg\min_{g \in G} \mathcal{L}(f, g, \pi_x) + \Omega(g)$$

其中 $g$ 是可解释模型，$\pi_x$ 是邻近度权重，$\Omega(g)$ 是复杂度惩罚。

### 3.2 实现

```python
import lime
from lime import lime_tabular, lime_image, lime_text

def lime_explanation_tabular(model, X_train, instance, feature_names):
    """
    表格数据的 LIME 解释
    """
    explainer = lime_tabular.LimeTabularExplainer(
        X_train,
        feature_names=feature_names,
        class_names=['Class 0', 'Class 1'],
        mode='classification'
    )

    explanation = explainer.explain_instance(
        instance,
        model.predict_proba,
        num_features=10
    )

    # 提取特征重要性
    feature_importance = explanation.as_list()

    return {
        'explanation': explanation,
        'feature_importance': feature_importance,
        'intercept': explanation.intercept
    }

def lime_explanation_image(model, image, num_samples=1000):
    """
    图像的 LIME 解释
    """
    explainer = lime_image.LimeImageExplainer()

    explanation = explainer.explain_instance(
        image,
        model.predict,
        top_labels=5,
        hide_color=0,
        num_samples=num_samples
    )

    # 获取最重要区域的掩码
    temp, mask = explanation.get_image_and_mask(
        explanation.top_labels[0],
        positive_only=True,
        num_features=5,
        hide_rest=False
    )

    return {'mask': mask, 'segments': explanation.segments}
```

## 4. Grad-CAM

### 4.1 原理

Grad-CAM (Gradient-weighted Class Activation Mapping) 利用卷积层的梯度信息生成**类激活热力图**：

$$L_{\text{Grad-CAM}}^c = \text{ReLU}\left( \sum_k \alpha_k^c A^k \right)$$

其中 $\alpha_k^c = \frac{1}{Z} \sum_i \sum_j \frac{\partial y^c}{\partial A_{ij}^k}$ 是特征图 $k$ 对类别 $c$ 的重要性权重。

### 4.2 实现

```python
import torch
import torch.nn.functional as F
import cv2
import numpy as np

class GradCAM:
    """
    Grad-CAM 实现
    """
    def __init__(self, model, target_layer):
        self.model = model
        self.target_layer = target_layer
        self.gradients = None
        self.activations = None

        # 注册钩子
        target_layer.register_forward_hook(self._save_activation)
        target_layer.register_backward_hook(self._save_gradient)

    def _save_activation(self, module, input, output):
        self.activations = output.detach()

    def _save_gradient(self, module, grad_input, grad_output):
        self.gradients = grad_output[0].detach()

    def generate_cam(self, input_image, target_class=None):
        """
        生成 Grad-CAM 热力图
        """
        # 前向传播
        output = self.model(input_image)

        if target_class is None:
            target_class = output.argmax(1).item()

        # 反向传播
        self.model.zero_grad()
        one_hot = torch.zeros_like(output)
        one_hot[0, target_class] = 1
        output.backward(gradient=one_hot)

        # 计算权重: 全局平均池化梯度
        weights = self.gradients.mean(dim=[2, 3], keepdim=True)

        # 加权组合
        cam = (weights * self.activations).sum(dim=1, keepdim=True)
        cam = F.relu(cam)  # 只保留正贡献

        # 归一化
        cam = cam - cam.min()
        cam = cam / (cam.max() + 1e-8)

        # 上采样到输入尺寸
        cam = F.interpolate(cam, size=input_image.shape[2:],
                           mode='bilinear', align_corners=False)

        return cam.squeeze().cpu().numpy()

    def visualize(self, input_image, target_class=None):
        """
        可视化 Grad-CAM
        """
        cam = self.generate_cam(input_image, target_class)

        # 转换为热力图
        heatmap = cv2.applyColorMap(np.uint8(255 * cam), cv2.COLORMAP_JET)
        heatmap = cv2.cvtColor(heatmap, cv2.COLOR_BGR2RGB)

        # 叠加到原图
        img_np = input_image.squeeze().permute(1, 2, 0).cpu().numpy()
        img_np = (img_np - img_np.min()) / (img_np.max() - img_np.min())
        img_np = np.uint8(255 * img_np)

        overlay = cv2.addWeighted(img_np, 0.6, heatmap, 0.4, 0)

        return overlay
```

## 5. 注意力可视化

### 5.1 Transformer 注意力

```python
def visualize_attention(model, tokenizer, text, layer_idx=-1):
    """
    可视化 Transformer 的注意力权重
    """
    inputs = tokenizer(text, return_tensors='pt')

    with torch.no_grad():
        outputs = model(**inputs, output_attentions=True)

    # 获取指定层的注意力
    attention = outputs.attentions[layer_idx]  # [batch, heads, seq, seq]
    attention = attention.squeeze(0)  # [heads, seq, seq]

    # 平均所有头的注意力
    avg_attention = attention.mean(dim=0)  # [seq, seq]

    # 获取 token
    tokens = tokenizer.convert_ids_to_tokens(inputs['input_ids'][0])

    return {
        'tokens': tokens,
        'attention_matrix': avg_attention.numpy(),
        'all_heads': attention.numpy()
    }
```

    cav = clf.coef_[0]

        'target_class': target_class,

## 7. 方法对比

| 方法 | 模型依赖 | 解释粒度 | 计算成本 | 适用数据 |
|------|----------|----------|----------|----------|
| SHAP | 模型无关 | 特征级 | 高 | 表格/文本 |
| LIME | 模型无关 | 特征级 | 中 | 通用 |
| Grad-CAM | 需要梯度 | 区域级 | 低 | 图像 |
| 注意力 | Transformer | Token级 | 低 | 文本 |
| TCAV | 需要梯度 | 概念级 | 中 | 图像 |
