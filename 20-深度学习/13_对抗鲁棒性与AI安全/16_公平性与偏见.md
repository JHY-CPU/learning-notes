# 16_公平性与偏见

## 1. 公平性问题概述

深度学习模型可能从训练数据中**继承或放大社会偏见**，对不同群体产生不公平的结果。公平性研究旨在识别、量化和消除这些偏见。

### 1.1 偏见来源

```
偏见来源
├── 数据偏见 (Data Bias)
│   ├── 表征不足: 某些群体数据量过少
│   ├── 历史偏见: 训练数据反映历史不公
│   ├── 标签偏见: 标注者的主观偏见
│   └── 采样偏见: 数据收集过程不均匀
├── 算法偏见 (Algorithmic Bias)
│   ├── 优化目标: 目标函数对某些群体不利
│   ├── 特征选择: 使用代理变量 (如邮编→种族)
│   └── 过拟合: 对少数群体过拟合
└── 部署偏见 (Deployment Bias)
    ├── 使用场景不匹配
    └── 反馈循环: 模型输出影响未来数据
```

### 1.2 典型案例

| 领域 | 偏见表现 | 影响 |
|------|----------|------|
| 招聘筛选 | 对女性求职者评分低 | 性别歧视 |
| 人脸识别 | 深肤色女性错误率高 | 种族歧视 |
| 贷款审批 | 对少数族裔拒贷率高 | 经济不平等 |
| 刑事司法 | 对黑人被告风险评分高 | 司法不公 |
| 医疗诊断 | 对少数群体诊断不准 | 健康不平等 |

## 2. 公平性指标

### 2.1 统计均等 (Demographic Parity)

要求模型对不同敏感群体的**正面预测率相同**：

$$P(\hat{Y}=1 | A=0) = P(\hat{Y}=1 | A=1)$$

其中 $A$ 为敏感属性 (如性别、种族)。

### 2.2 机会均等 (Equalized Odds)

要求模型对不同群体的**真正率和假正率相同**：

$$P(\hat{Y}=1 | A=0, Y=y) = P(\hat{Y}=1 | A=1, Y=y), \quad \forall y \in \{0, 1\}$$

### 2.3 预测均等 (Predictive Parity)

要求不同群体的**正预测值相同**：

$$P(Y=1 | \hat{Y}=1, A=0) = P(Y=1 | \hat{Y}=1, A=1)$$

### 2.4 指标计算

```python
import numpy as np

def compute_fairness_metrics(y_true, y_pred, sensitive_attr):
    """
    计算各种公平性指标

    Args:
        y_true: 真实标签
        y_pred: 预测标签
        sensitive_attr: 敏感属性 (0/1)
    """
    groups = [0, 1]
    metrics = {}

    for g in groups:
        mask = sensitive_attr == g
        group_true = y_true[mask]
        group_pred = y_pred[mask]

        # 正预测率
        metrics[f'positive_rate_g{g}'] = group_pred.mean()

        # 真正率 (TPR)
        tp = ((group_pred == 1) & (group_true == 1)).sum()
        fn = ((group_pred == 0) & (group_true == 1)).sum()
        metrics[f'tpr_g{g}'] = tp / (tp + fn) if (tp + fn) > 0 else 0

        # 假正率 (FPR)
        fp = ((group_pred == 1) & (group_true == 0)).sum()
        tn = ((group_pred == 0) & (group_true == 0)).sum()
        metrics[f'fpr_g{g}'] = fp / (fp + tn) if (fp + tn) > 0 else 0

    # 公平性差距
    metrics['demographic_parity_diff'] = abs(
        metrics['positive_rate_g0'] - metrics['positive_rate_g1']
    )
    metrics['equalized_odds_diff'] = max(
        abs(metrics['tpr_g0'] - metrics['tpr_g1']),
        abs(metrics['fpr_g0'] - metrics['fpr_g1'])
    )

    return metrics
```

## 3. 去偏方法

### 3.1 预处理去偏

```python
def reweight_samples(X, y, sensitive_attr):
    """
    样本重加权: 对少数群体赋予更高权重
    """
    n = len(y)
    weights = np.ones(n)

    for a in [0, 1]:
        for label in [0, 1]:
            mask = (sensitive_attr == a) & (y == label)
            count = mask.sum()
            expected = n * (sensitive_attr == a).mean() * (y == label).mean()
            if count > 0:
                weights[mask] = expected / count

    return weights

def data_augmentation_minority(X, y, sensitive_attr, target_group=0):
    """
    数据增强: 增加少数群体样本
    """
    minority_mask = sensitive_attr == target_group
    X_minority = X[minority_mask]

    # 使用 SMOTE 或过采样

    X_resampled, y_resampled = smote.fit_resample(

            # 公平性损失 (Demographic Parity)
            group_0_pred = pred[sensitive == 0]
            group_1_pred = pred[sensitive == 1]

            if len(group_0_pred) > 0 and len(group_1_pred) > 0:
                fairness_loss = (group_0_pred.mean() - group_1_pred.mean()) ** 2
            else:
                fairness_loss = 0

            total_loss = task_loss + lambda_fair * fairness_loss

            optimizer.zero_grad()
            total_loss.backward()
            optimizer.step()

        print(f"Epoch {epoch+1}: Task Loss={task_loss.item():.4f}, "
              f"Fairness Loss={fairness_loss.item() if isinstance(fairness_loss, torch.Tensor) else fairness_loss:.4f}")
```

### 3.3 后处理去偏

```python
def threshold_calibration(model, X_val, y_val, sensitive_val):
    """
    阈值校准: 为不同群体设置不同决策阈值
    """
    model.eval()
    with torch.no_grad():
        scores = F.softmax(model(X_val), dim=1)[:, 1].numpy()

    best_thresholds = {}
    best_accuracy = 0

    for t0 in np.arange(0.1, 0.9, 0.05):
        for t1 in np.arange(0.1, 0.9, 0.05):
            pred = np.where(sensitive_val == 0,
                           (scores > t0).astype(int),
                           (scores > t1).astype(int))

            # 计算公平性和准确率
            acc = (pred == y_val).mean()
            dp_diff = abs(
                pred[sensitive_val == 0].mean() -
                pred[sensitive_val == 1].mean()
            )

            # 公平性约束下的准确率
            if dp_diff < 0.05 and acc > best_accuracy:
                best_accuracy = acc
                best_thresholds = {'t0': t0, 't1': t1}

    return best_thresholds
```

## 4. 公平性-准确率权衡

| 公平性约束 | 准确率影响 | 适用场景 |
|------------|------------|----------|
| 无约束 | 最高 | 非敏感任务 |
| Demographic Parity | 中等下降 | 公共决策 |
| Equalized Odds | 轻微下降 | 医疗、金融 |
| 个体公平性 | 变化大 | 法律合规 |

## 5. 总结

| 要点 | 说明 |
|------|------|
| 偏见来源 | 数据、算法、部署 |
| 公平性指标 | Demographic Parity、Equalized Odds |
| 去偏阶段 | 预处理、训练时、后处理 |
| 核心权衡 | 公平性 vs 准确率 |
