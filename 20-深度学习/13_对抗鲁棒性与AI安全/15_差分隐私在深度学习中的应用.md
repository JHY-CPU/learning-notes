# 15_差分隐私在深度学习中的应用

## 1. 差分隐私基础

差分隐私 (Differential Privacy, DP) 是一种**数学上可证明的隐私保护框架**，通过在计算过程中添加校准噪声，确保单个数据样本的加入或移除不会显著改变输出结果。

### 1.1 形式化定义

一个随机化机制 $\mathcal{M}$ 满足 $(\epsilon, \delta)$-差分隐私，如果对于所有相邻数据集 $D, D'$ (仅相差一条记录) 和所有可能的输出集合 $S$：

$$\Pr[\mathcal{M}(D) \in S] \leq e^\epsilon \cdot \Pr[\mathcal{M}(D') \in S] + \delta$$

- $\epsilon$ (隐私预算): 控制隐私保护强度，越小保护越强
- $\delta$ (失败概率): 通常设为 $1/|D|^c$，$c > 1$

### 1.2 直观理解

```
差分隐私的核心思想:

数据集 A: {张三, 李四, 王五, ...}
数据集 B: {李四, 王五, ...}  (移除张三)

DP 保证: 模型在 A 和 B 上的输出分布非常相似
→ 攻击者无法判断张三是否在训练集中
```

## 2. DP-SGD 算法

### 2.1 核心机制

DP-SGD (Differentially Private Stochastic Gradient Descent) 是差分隐私在深度学习中的标准实现，由 Abadi 等人 (2016) 提出。

关键操作：
1. **梯度裁剪**: 限制每个样本梯度的范数
2. **噪声添加**: 在聚合梯度上添加高斯噪声
3. **隐私会计**: 跟踪累计隐私损失

### 2.2 完整实现

```python
import torch
import torch.nn as nn
import numpy as np

class DPSGD:
    """
    差分隐私随机梯度下降 (DP-SGD)
    """
    def __init__(self, model, optimizer, noise_multiplier=1.0,
                 max_grad_norm=1.0, batch_size=256):
        self.model = model
        self.optimizer = optimizer
        self.noise_multiplier = noise_multiplier
        self.max_grad_norm = max_grad_norm
        self.batch_size = batch_size

    def compute_per_sample_gradients(self, images, labels):
        """
        计算每个样本的梯度 (使用 functorch 或手动实现)
        """
        per_sample_grads = []

        for i in range(images.size(0)):
            self.optimizer.zero_grad()
            output = self.model(images[i:i+1])
            loss = nn.functional.cross_entropy(output, labels[i:i+1])
            loss.backward()

            # 收集每个参数的梯度
            grads = []
            for param in self.model.parameters():
                if param.grad is not None:
                    grads.append(param.grad.detach().clone())
                else:
                    grads.append(torch.zeros_like(param))
            per_sample_grads.append(grads)

        return per_sample_grads

    def clip_gradients(self, per_sample_grads):
        """
        裁剪每个样本的梯度
        """
        clipped_grads = []

        for sample_grads in per_sample_grads:
            # 计算该样本所有参数梯度的总范数
            total_norm = torch.sqrt(sum(
                g.norm() ** 2 for g in sample_grads
            ))

            # 裁剪
            clip_factor = min(1.0, self.max_grad_norm / (total_norm + 1e-6))
            clipped = [g * clip_factor for g in sample_grads]
            clipped_grads.append(clipped)

        return clipped_grads

    def add_noise(self, clipped_grads):
        """
        在聚合梯度上添加高斯噪声
        """
        # 聚合
        aggregated = []
        for param_idx in range(len(clipped_grads[0])):
            param_sum = sum(
                clipped_grads[i][param_idx] for i in range(len(clipped_grads))
            )

            # 添加高斯噪声
            noise_std = self.noise_multiplier * self.max_grad_norm
            noise = torch.randn_like(param_sum) * noise_std
            noisy_grad = param_sum + noise

            # 平均化
            aggregated.append(noisy_grad / self.batch_size)

        return aggregated

    def step(self, images, labels):
        """
        一步 DP-SGD 更新
        """
        # 1. 计算逐样本梯度
        per_sample_grads = self.compute_per_sample_gradients(images, labels)

        # 2. 裁剪
        clipped_grads = self.clip_gradients(per_sample_grads)

        # 3. 添加噪声并聚合
        noisy_grads = self.add_noise(clipped_grads)

        # 4. 更新参数
        for param, grad in zip(self.model.parameters(), noisy_grads):
            param.grad = grad
        self.optimizer.step()
```

### 2.3 使用 Opacus 库 (推荐)

```python
from opacus import PrivacyEngine
from opacus.validators import ModuleValidator

def train_with_dp(model, train_loader, optimizer, epochs=10,
                  epsilon=8.0, delta=1e-5, max_grad_norm=1.0):
    """
    使用 Opacus 库进行差分隐私训练
    """
    # 验证并修复模型兼容性
    model = ModuleValidator.fix(model)
    ModuleValidator.validate(model, strict=True)

    # 创建隐私引擎
    privacy_engine = PrivacyEngine()

    model, optimizer, train_loader = privacy_engine.make_private_with_epsilon(
        module=model,
        optimizer=optimizer,
        data_loader=train_loader,
        epochs=epochs,
        target_epsilon=epsilon,
        target_delta=delta,
        max_grad_norm=max_grad_norm,
    )

    for epoch in range(epochs):
        model.train()
        total_loss = 0
        correct = 0
        total = 0

        for images, labels in train_loader:
            optimizer.zero_grad()
            output = model(images)
            loss = nn.functional.cross_entropy(output, labels)
            loss.backward()
            optimizer.step()

            total_loss += loss.item()
            _, pred = output.max(1)
            correct += pred.eq(labels).sum().item()
            total += labels.size(0)

        # 获取当前隐私消耗
        epsilon_used = privacy_engine.get_epsilon(delta=delta)
        acc = 100.0 * correct / total

        print(f"Epoch {epoch+1}/{epochs} | "
              f"Loss: {total_loss/len(train_loader):.4f} | "
              f"Acc: {acc:.2f}% | "
              f"ε: {epsilon_used:.2f}")

    return model
```

## 3. 隐私预算与会计

### 3.1 Rényi 差分隐私 (RDP)

```python
def compute_rdp(alpha, noise_multiplier, steps):
    """
    计算 Rényi 差分隐私 (RDP) 会计
    """
    # RDP 损失: α / (2σ²) per step
    rdp = alpha / (2 * noise_multiplier ** 2) * steps
    return rdp

def rdp_to_dp(rdp_alpha, delta):
    """
    将 RDP 转换为 (ε, δ)-DP
    """
    epsilon = rdp_alpha + np.log(1/delta) / (rdp_alpha - 1)
    return epsilon

def compute_privacy_budget(noise_multiplier, batch_size,
                           dataset_size, epochs, delta=1e-5):
    """
    计算训练结束时的隐私预算
    """
    steps_per_epoch = dataset_size // batch_size
    total_steps = steps_per_epoch * epochs

    # 搜索最优 alpha
    best_epsilon = float('inf')
    for alpha in [1.1 + i * 0.1 for i in range(1, 100)]:
        rdp = compute_rdp(alpha, noise_multiplier, total_steps)
        epsilon = rdp_to_dp(rdp, delta)
        best_epsilon = min(best_epsilon, epsilon)

    return best_epsilon
```

### 3.2 参数选择指南

| 隐私目标 | $\epsilon$ 范围 | $\delta$ | 噪声乘数 | 效果 |
|----------|----------------|----------|----------|------|
| 强隐私 | < 1 | 1e-5 | > 2.0 | 准确率大幅下降 |
| 中等隐私 | 1-8 | 1e-5 | 0.5-2.0 | 准确率适度下降 |
| 弱隐私 | 8-30 | 1e-5 | 0.1-0.5 | 准确率轻微下降 |
| 实用隐私 | > 30 | 1e-5 | < 0.1 | 几乎无影响 |

## 4. DP 训练的实践挑战

### 4.1 准确率下降问题

```python
def dp_accuracy_comparison():
    """
    典型的 DP 训练 vs 标准训练准确率对比
    """
    results = {
```

### 4.2 缓解策略

| 策略 | 描述 | 效果 |
|------|------|------|
| 预训练 + DP 微调 | 在公开数据上预训练，DP 微调 | **显著** |
| 大批次训练 | 更大的 batch_size 减少噪声相对影响 | 高 |
| 组归一化 | 替代 BatchNorm | 中 |
| 分层裁剪 | 不同层使用不同裁剪阈值 | 中 |
| 增强数据 | 增加训练数据多样性 | 中 |
