# 多模态学习概述

## 1. 什么是模态 (Modality)

模态是指信息的表现形式或感知通道。人类通过多种模态感知世界——视觉、听觉、触觉、语言等。
在深度学习中，常见模态包括：

| 模态类型 | 数据形式 | 典型任务 |
|---------|---------|---------|
| 视觉 (Vision) | 图像、视频 | 目标检测、图像分类 |
| 语言 (Language) | 文本 | 情感分析、机器翻译 |
| 音频 (Audio) | 语音、音乐 | 语音识别、音乐生成 |
| 结构化数据 | 表格、图谱 | 推荐系统、知识推理 |

**多模态学习 (Multimodal Learning)** 的目标是让模型能够联合处理和理解来自多种模态的信息，
实现跨模态的推理、生成和交互。

## 2. 多模态学习的核心问题

多模态学习需要解决三个核心问题：

### 2.1 表示学习 (Representation)

如何将不同模态映射到统一或兼容的特征空间？

$$\mathbf{h}_v = f_v(\mathbf{x}_v), \quad \mathbf{h}_t = f_t(\mathbf{x}_t)$$

其中 $f_v$ 和 $f_t$ 分别为视觉编码器和文本编码器，$\mathbf{h}_v$ 和 $\mathbf{h}_t$ 为各自的表示。

### 2.2 对齐 (Alignment)

如何建立不同模态之间的语义对应关系？

$$\text{sim}(\mathbf{h}_v, \mathbf{h}_t) = \frac{\mathbf{h}_v^\top \mathbf{h}_t}{\|\mathbf{h}_v\| \cdot \|\mathbf{h}_t\|}$$

### 2.3 融合 (Fusion)

如何有效地组合多模态信息以完成下游任务？

## 3. 融合策略

### 3.1 早期融合 (Early Fusion)

在输入层或特征提取早期直接拼接多模态数据：

$$\mathbf{h}_{\text{fused}} = g([\mathbf{x}_v; \mathbf{x}_t])$$

```python
import torch
import torch.nn as nn

class EarlyFusion(nn.Module):
    def __init__(self, vision_dim, text_dim, hidden_dim):
        super().__init__()
        self.fusion = nn.Sequential(
            nn.Linear(vision_dim + text_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )

    def forward(self, vision_feat, text_feat):
        # 直接拼接特征
        fused = torch.cat([vision_feat, text_feat], dim=-1)
        return self.fusion(fused)
```

**优点**：简单直接，能捕获底层交互。
**缺点**：特征维度高，容易受噪声影响。

### 3.2 晚期融合 (Late Fusion)

各模态独立处理后再融合决策结果：

$$\hat{y} = \phi(g_v(\mathbf{x}_v), g_t(\mathbf{x}_t))$$

```python
class LateFusion(nn.Module):
    def __init__(self, vision_dim, text_dim, num_classes):
        super().__init__()
        self.vision_head = nn.Linear(vision_dim, num_classes)
        self.text_head = nn.Linear(text_dim, num_classes)
        self.weight = nn.Parameter(torch.tensor(0.5))

    def forward(self, vision_feat, text_feat):
        logits_v = self.vision_head(vision_feat)
        logits_t = self.text_head(text_feat)
        # 加权融合
        return self.weight * logits_v + (1 - self.weight) * logits_t
```

**优点**：各模态可独立优化，鲁棒性好。
**缺点**：无法捕获模态间的细粒度交互。

### 3.3 混合融合 (Hybrid Fusion)

结合早期和晚期融合的优点，在多个层级进行融合：

$$\mathbf{h}^{(l)} = \text{FusionBlock}(\mathbf{h}_v^{(l)}, \mathbf{h}_t^{(l)})$$

### 3.4 注意力融合 (Attention Fusion)

使用注意力机制动态加权不同模态的贡献：

$$\alpha = \text{softmax}\left(\frac{\mathbf{Q}_v \mathbf{K}_t^\top}{\sqrt{d_k}}\right), \quad \mathbf{h} = \alpha \mathbf{V}_t$$

```python
class AttentionFusion(nn.Module):
    def __init__(self, dim, num_heads=8):
        super().__init__()
        self.cross_attn = nn.MultiheadAttention(dim, num_heads)
        self.norm = nn.LayerNorm(dim)

    def forward(self, query_feat, kv_feat):
        # query_feat: 视觉特征, kv_feat: 文本特征
        attn_out, _ = self.cross_attn(query_feat, kv_feat, kv_feat)
        return self.norm(query_feat + attn_out)
```

## 4. 融合策略对比

| 策略 | 交互粒度 | 计算开销 | 鲁棒性 | 代表模型 |
|------|---------|---------|--------|---------|
| 早期融合 | 细粒度 | 高 | 低 | ViLBERT |
| 晚期融合 | 粗粒度 | 低 | 高 | 独立双塔 |
| 混合融合 | 多层级 | 中 | 中 | LXMERT |
| 注意力融合 | 动态 | 高 | 中 | CLIP, Flamingo |

## 5. 多模态学习的挑战

### 5.1 异构性差距 (Heterogeneity Gap)

不同模态的数据分布、维度和语义结构差异巨大。图像为高维连续信号，文本为离散符号序列，
如何建立二者之间的语义桥梁是核心难题。

### 5.2 对齐困难

不同模态间存在粒度不对齐的问题——一张图可能对应多句话，一个词可能对应图像中的多个区域。

### 5.3 缺失模态

实际应用中某些模态可能缺失，模型需要具备处理不完整输入的能力：

$$\hat{y} = \begin{cases} f([\mathbf{h}_v, \mathbf{h}_t]) & \text{both available} \\ f(\mathbf{h}_v) & \text{text missing} \\ f(\mathbf{h}_t) & \text{vision missing} \end{cases}$$

### 5.4 计算效率

多模态模型通常参数量巨大，推理速度慢，部署成本高。

### 5.5 评估困难

缺乏统一的多模态评估指标，不同任务需要不同的评估标准。

## 6. 多模态学习的发展脉络

```
早期手工特征融合 → 深度多模态网络 → 预训练范式 → 大规模基础模型
(2010-2015)        (2016-2018)      (2019-2021)    (2022-至今)
```

核心趋势：
- 从**任务特定**到**通用预训练**
- 从**小规模**到**大规模数据+模型**
- 从**模态特异**到**统一架构**
- 从**判别式**到**生成式**

多模态学习是当前 AI 最活跃的研究方向之一。从 CLIP 开创的对比学习范式，
到 GPT-4V/Gemini 等统一多模态大模型，这一领域正在经历从专用到通用的范式转变。
理解融合策略、对齐方法和架构设计是掌握多模态学习的关键基础。
