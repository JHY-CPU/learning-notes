# 对比学习基础

## 1. 对比学习的核心思想

对比学习 (Contrastive Learning) 是一种自监督学习范式，其核心思想是：
**拉近正样本对的表示距离，推远负样本对的表示距离**。

给定一个锚样本 $\mathbf{x}$，其正样本为 $\mathbf{x}^+$，负样本集合为 $\{\mathbf{x}_i^-\}$，
对比学习的目标为：

$$\mathcal{L} = -\log \frac{\exp(\text{sim}(\mathbf{h}, \mathbf{h}^+) / \tau)}{\exp(\text{sim}(\mathbf{h}, \mathbf{h}^+) / \tau) + \sum_i \exp(\text{sim}(\mathbf{h}, \mathbf{h}_i^-) / \tau)}$$

其中 $\tau$ 为温度参数，$\text{sim}(\cdot, \cdot)$ 通常为余弦相似度。

## 2. InfoNCE 损失

InfoNCE 是对比学习中最常用的损失函数：

$$\mathcal{L}_{\text{InfoNCE}} = -\log \frac{\exp(\mathbf{q} \cdot \mathbf{k}^+ / \tau)}{\sum_{i=0}^{K} \exp(\mathbf{q} \cdot \mathbf{k}_i / \tau)}$$

其中 $\mathbf{q}$ 为查询向量，$\mathbf{k}^+$ 为正样本键，$\{\mathbf{k}_i\}_{i=1}^K$ 为负样本键。

```python
import torch
import torch.nn.functional as F

def info_nce_loss(query, positive_key, negative_keys, temperature=0.07):
    """
    query: (B, D)
    positive_key: (B, D)
    negative_keys: (B, K, D)
    """
    # 正样本相似度
    pos_sim = F.cosine_similarity(query, positive_key, dim=-1)  # (B,)
    pos_sim = pos_sim / temperature

    # 负样本相似度
    neg_sim = torch.bmm(
        negative_keys, query.unsqueeze(-1)
    ).squeeze(-1) / temperature  # (B, K)

    # 拼接: [正样本, 负样本]
    logits = torch.cat([pos_sim.unsqueeze(1), neg_sim], dim=1)  # (B, 1+K)
    labels = torch.zeros(logits.size(0), dtype=torch.long, device=logits.device)

    return F.cross_entropy(logits, labels)
```

## 3. SimCLR

### 3.1 架构

SimCLR 采用对称的双编码器架构：

```
输入图像 x → 增强1 → 编码器f(·) → 投影头g(·) → z₁
    ↓
          增强2 → 编码器f(·) → 投影头g(·) → z₂
```

### 3.2 损失函数：NT-Xent

SimCLR 使用归一化温度标度交叉熵损失 (NT-Xent)：

$$\ell_{i,j} = -\log \frac{\exp(\text{sim}(\mathbf{z}_i, \mathbf{z}_j) / \tau)}{\sum_{k=1}^{2N} \mathbf{1}_{[k \neq i]} \exp(\text{sim}(\mathbf{z}_i, \mathbf{z}_k) / \tau)}$$

对于一个 batch 中的 $N$ 个样本，共有 $2(N-1)$ 个负样本。

```python
class SimCLRLoss(nn.Module):
    def __init__(self, temperature=0.5):
        super().__init__()
        self.temperature = temperature

    def forward(self, z_i, z_j):
        """
        z_i, z_j: (N, D) 同一样本的两个增强视图
        """
        N = z_i.size(0)
        z = torch.cat([z_i, z_j], dim=0)  # (2N, D)
        z = F.normalize(z, dim=1)

        # 相似度矩阵
        sim_matrix = z @ z.T / self.temperature  # (2N, 2N)

        # 掩码：去除自身对角线
        mask = torch.eye(2 * N, dtype=torch.bool, device=z.device)
        sim_matrix.masked_fill_(mask, -1e9)

        # 正样本对的索引
        pos_indices = torch.cat([
            torch.arange(N, 2 * N),
            torch.arange(0, N)
        ]).to(z.device)

        # 正样本相似度
        pos_sim = sim_matrix[torch.arange(2 * N), pos_indices]

        # NT-Xent 损失
        loss = -pos_sim + torch.logsumexp(sim_matrix, dim=1)
        return loss.mean()
```

### 3.3 关键设计

- **数据增强**：随机裁剪 + 颜色抖动 + 高斯模糊最为有效
- **投影头**：MLP 投影头比线性投影效果好，且投影前的特征更优
- **大 batch size**：需要大量负样本（通常 4096+）

## 4. MoCo (Momentum Contrast)

### 4.1 核心思想

MoCo 通过动量更新的编码器维护一个动态的负样本队列 (queue)，
解决了 SimCLR 需要大 batch size 的问题。

```
查询编码器 f_q ← 梯度更新
键编码器 f_k ← 动量更新: θ_k ← m·θ_k + (1-m)·θ_q
负样本队列 ← 动态队列，不断入队出队
```

### 4.2 动量更新

$$\theta_k \leftarrow m \cdot \theta_k + (1 - m) \cdot \theta_q, \quad m \in [0.99, 0.999]$$

```python
class MoCo(nn.Module):
    def __init__(self, encoder, dim=128, queue_size=65536, momentum=0.999, temperature=0.07):
        super().__init__()
        self.queue_size = queue_size
        self.momentum = momentum
        self.temperature = temperature

        # 查询编码器 (可训练)
        self.encoder_q = encoder
        self.mlp_q = nn.Sequential(
            nn.Linear(dim, dim), nn.ReLU(), nn.Linear(dim, dim)
        )

        # 键编码器 (动量更新)
        self.encoder_k = encoder.__class__()
        self.encoder_k.load_state_dict(encoder.state_dict())
        self.mlp_k = nn.Sequential(
            nn.Linear(dim, dim), nn.ReLU(), nn.Linear(dim, dim)
        )
        self.mlp_k.load_state_dict(self.mlp_q.state_dict())

        # 不需要梯度
        for param in self.encoder_k.parameters():
            param.requires_grad = False
        for param in self.mlp_k.parameters():
            param.requires_grad = False

        # 负样本队列
        self.register_buffer("queue", F.normalize(torch.randn(queue_size, dim), dim=1))
        self.register_buffer("queue_ptr", torch.zeros(1, dtype=torch.long))

    @torch.no_grad()
    def _momentum_update(self):
        for q_param, k_param in zip(self.encoder_q.parameters(), self.encoder_k.parameters()):
            k_param.data = self.momentum * k_param.data + (1 - self.momentum) * q_param.data
        for q_param, k_param in zip(self.mlp_q.parameters(), self.mlp_k.parameters()):
            k_param.data = self.momentum * k_param.data + (1 - self.momentum) * q_param.data

    def forward(self, x_q, x_k):
        # 查询
        q = F.normalize(self.mlp_q(self.encoder_q(x_q)), dim=1)

        with torch.no_grad():
            self._momentum_update()
            k = F.normalize(self.mlp_k(self.encoder_k(x_k)), dim=1)

        # 正样本相似度
        l_pos = (q * k).sum(dim=1, keepdim=True)  # (N, 1)

        # 负样本相似度
        l_neg = q @ self.queue.T  # (N, queue_size)

        logits = torch.cat([l_pos, l_neg], dim=1) / self.temperature
        labels = torch.zeros(logits.size(0), dtype=torch.long, device=logits.device)

        loss = F.cross_entropy(logits, labels)

        # 更新队列
        self._dequeue_and_enqueue(k)
        return loss

    @torch.no_grad()
    def _dequeue_and_enqueue(self, keys):
        batch_size = keys.size(0)
        ptr = int(self.queue_ptr)
        self.queue[ptr:ptr + batch_size] = keys
        ptr = (ptr + batch_size) % self.queue_size
        self.queue_ptr[0] = ptr
```

- 更强的数据增强
- 使用余弦学习率调度

## 5. BYOL (Bootstrap Your Own Latent)

### 5.1 无负样本对比

BYOL 的创新在于**不需要负样本**。通过在线网络 (online) 预测目标网络 (target) 的输出，
避免表示坍缩 (representation collapse)。

```
在线网络:  θ → 编码器 → 投影器 → 预测器 → p
目标网络:  ξ → 编码器 → 投影器          → z (停止梯度)
```

### 5.2 损失函数

$$\mathcal{L}_{\text{BYOL}} = \|p_\theta(z_v) - z'_\xi\|^2 + \|p_\theta(z'_\xi) - z_\xi\|^2$$

其中 $p_\theta$ 为预测器，目标网络通过 EMA 更新。

```python
class BYOL(nn.Module):
    def __init__(self, encoder, dim=256, pred_dim=128, momentum=0.996):
        super().__init__()
        self.momentum = momentum

        # 在线网络
        self.online_encoder = encoder
        self.online_projector = nn.Sequential(
            nn.Linear(dim, dim), nn.BatchNorm1d(dim), nn.ReLU(),
            nn.Linear(dim, pred_dim)
        )
        self.predictor = nn.Sequential(
            nn.Linear(pred_dim, pred_dim // 2), nn.BatchNorm1d(pred_dim // 2), nn.ReLU(),
            nn.Linear(pred_dim // 2, pred_dim)
        )

        # 目标网络 (无梯度)
        self.target_encoder = encoder.__class__()
        self.target_projector = nn.Sequential(
            nn.Linear(dim, dim), nn.BatchNorm1d(dim), nn.ReLU(),
            nn.Linear(dim, pred_dim)
        )
        self._update_target(0)  # 初始化

        for param in self.target_encoder.parameters():
            param.requires_grad = False
        for param in self.target_projector.parameters():
            param.requires_grad = False

    @torch.no_grad()
    def _update_target(self, m):
        for o, t in zip(self.online_encoder.parameters(), self.target_encoder.parameters()):
            t.data = m * t.data + (1 - m) * o.data
        for o, t in zip(self.online_projector.parameters(), self.target_projector.parameters()):
            t.data = m * t.data + (1 - m) * o.data

    def forward(self, x1, x2):
        # 在线网络
        z1_online = self.online_projector(self.online_encoder(x1))
        p1 = self.predictor(z1_online)
        z2_online = self.online_projector(self.online_encoder(x2))
        p2 = self.predictor(z2_online)

        # 目标网络 (停止梯度)
        with torch.no_grad():
            self._update_target(self.momentum)
            z1_target = self.target_projector(self.target_encoder(x1))
            z2_target = self.target_projector(self.target_encoder(x2))

        # 对称损失
        loss = F.mse_loss(p1, z2_target.detach()) + F.mse_loss(p2, z1_target.detach())
        return loss
```

### 5.3 为什么 BYOL 不会坍缩？

目标网络的动量更新和停止梯度操作提供了隐式的正则化，
使得在线网络无法简单地输出常数来最小化损失。

## 6. 方法对比总结

| 方法 | 负样本 | 损失函数 | 关键技巧 | 线性探测 Acc |
|------|--------|---------|---------|-------------|
| SimCLR | batch内 | NT-Xent | 大 batch + 强增强 | 69.3% |
| MoCo | 队列 | InfoNCE | 动量编码器 + 队列 | 68.9% |
| BYOL | 无 | MSE | 预测器 + EMA | 74.3% |
| SimSiam | 无 | 余弦 | 停止梯度 | 71.3% |

## 7. 温度参数的作用

温度 $\tau$ 控制分布的锐度：

- **较小的 $\tau$**：分布更尖锐，模型更关注困难负样本
- **较大的 $\tau$**：分布更平滑，所有负样本权重相近

$$\frac{\partial \mathcal{L}}{\partial \text{sim}} \propto \frac{1}{\tau} \left( \frac{\exp(\text{sim}/\tau)}{\sum \exp(\text{sim}/\tau)} \right)$$

通常 $\tau = 0.07$ 或 $0.5$ 效果较好。

## 8. 小结

对比学习为多模态学习提供了强大的表示学习基础。CLIP 等模型将对比学习
从单模态扩展到跨模态，实现了视觉和语言的语义对齐，这正是下一章要讨论的内容。
