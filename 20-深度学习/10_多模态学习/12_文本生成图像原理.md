# 文本生成图像原理

## 1. 概述

文本生成图像 (Text-to-Image Generation) 旨在根据文本描述生成对应的图像。
从 CLIP 的对比学习到 Stable Diffusion 的扩散模型，文本条件机制是实现高质量生成的关键。

核心挑战：**如何将文本的语义信息有效地注入到图像生成过程中**。

## 2. 文本条件机制

### 2.1 CLIP 引导的生成

CLIP 学习了图像和文本的共享表示空间，可以用来引导生成方向：

$$\mathcal{L}_{\text{CLIP-guided}} = -\text{sim}(E_v(G(z)), E_t(c))$$

其中 $G$ 为生成器，$z$ 为随机噪声，$c$ 为文本条件。

```python
class CLIPGuidedGenerator(nn.Module):
    def __init__(self, generator, clip_model):
        super().__init__()
        self.generator = generator
        self.clip = clip_model

    def generate(self, text, num_steps=100, lr=0.1):
        z = torch.randn(1, 128, requires_grad=True)
        optimizer = Adam([z], lr=lr)
        text_embed = self.clip.encode_text(text)

        for _ in range(num_steps):
            image = self.generator(z)
            image_embed = self.clip.encode_image(image)
            loss = -F.cosine_similarity(image_embed, text_embed).mean()
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

        return self.generator(z).detach()
```

### 2.2 Cross-Attention 条件注入

最常用的条件机制——在生成模型的每个 Transformer 块中加入交叉注意力：

$$Q = W_Q \cdot h, \quad K = W_K \cdot c, \quad V = W_V \cdot c$$
$$\text{Attn}(Q, K, V) = \text{softmax}\left(\frac{QK^\top}{\sqrt{d_k}}\right) V$$

```python
class CrossAttentionBlock(nn.Module):
    def __init__(self, hidden_dim, context_dim, num_heads=8):
        super().__init__()
        self.norm1 = nn.LayerNorm(hidden_dim)
        self.self_attn = nn.MultiheadAttention(hidden_dim, num_heads, batch_first=True)

        self.norm2 = nn.LayerNorm(hidden_dim)
        self.cross_attn = nn.MultiheadAttention(hidden_dim, num_heads, batch_first=True)

        self.norm3 = nn.LayerNorm(hidden_dim)
        self.ffn = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim * 4),
            nn.GELU(),
            nn.Linear(hidden_dim * 4, hidden_dim)
        )

        # 文本特征投影
        self.context_proj = nn.Linear(context_dim, hidden_dim)

    def forward(self, x, context):
        """
        x: (B, N, D) 图像特征
        context: (B, L, C) 文本特征
        """
        context = self.context_proj(context)

        # 自注意力
        normed = self.norm1(x)
        x = x + self.self_attn(normed, normed, normed)[0]

        # 交叉注意力 (文本条件)
        normed = self.norm2(x)
        x = x + self.cross_attn(normed, context, context)[0]

        # FFN
        x = x + self.ffn(self.norm3(x))
        return x
```

## 3. DALL-E 系列

### 3.1 DALL-E 1 (2021)

使用离散 VAE (dVAE) 将图像编码为离散 token，然后用自回归 Transformer 生成：

```
文本 Token ─┐
             ├→ GPT-like Transformer → 图像 Token → dVAE 解码器 → 图像
图像 Token ─┘
```

### 3.2 DALL-E 2 (2022)

使用 CLIP 文本编码 + 扩散模型：

```
文本 → CLIP 文本编码器 → 文本嵌入
    ↓
扩散 Prior 模型 → 图像嵌入 (CLIP 空间)
    ↓
扩散解码器 → 64x64 图像
    ↓
超分辨率模型 → 1024x1024 图像
```

```python
class DALLE2(nn.Module):
    def __init__(self, clip_model, prior_model, decoder, super_res):
        super().__init__()
        self.clip = clip_model
        self.prior = prior_model  # 扩散 Prior
        self.decoder = decoder    # 扩散解码器
        self.super_res = super_res  # 超分辨率

    def generate(self, text, guidance_scale=3.0):
        # 1. 文本编码
        text_embed = self.clip.encode_text(text)

        # 2. Prior: 文本嵌入 → 图像嵌入
        image_embed = self.prior.sample(text_embed)

        # 3. 扩散解码: 图像嵌入 → 64x64 图像
        image_64 = self.decoder.sample(image_embed, guidance_scale=guidance_scale)

        # 4. 超分辨率
        image_1024 = self.super_res.sample(image_64)

        return image_1024
```

## 4. Stable Diffusion

### 4.1 Latent Diffusion Model (LDM)

Stable Diffusion 的核心创新是在**潜空间**而非像素空间进行扩散：

$$\mathcal{L}_{\text{LDM}} = \mathbb{E}_{z, \epsilon \sim \mathcal{N}(0,I), t} \left[ \|\epsilon - \epsilon_\theta(z_t, t, c)\|^2 \right]$$

其中 $z = E(x)$ 为 VAE 编码的潜变量，$c$ 为文本条件。

```
文本 → CLIP 文本编码器 → 文本嵌入
                          ↓
图像 → VAE 编码器 → 潜变量 z → 加噪 → UNet(去噪, 条件=c) → 去噪潜变量 → VAE 解码器 → 图像
```

### 4.2 架构实现

```python
class StableDiffusionUNet(nn.Module):
    def __init__(self, in_channels=4, context_dim=768):
        super().__init__()

        # 下采样
        self.down_blocks = nn.ModuleList([
            DownBlock(64, 128, context_dim),
            DownBlock(128, 256, context_dim),
            DownBlock(256, 512, context_dim),
            DownBlock(512, 512, context_dim),
        ])

        # 中间层
        self.mid_block = MidBlock(512, context_dim)

        # 上采样
        self.up_blocks = nn.ModuleList([
            UpBlock(512, 512, context_dim),
            UpBlock(512, 256, context_dim),
            UpBlock(256, 128, context_dim),
            UpBlock(128, 64, context_dim),
        ])

        # 时间嵌入
        self.time_embed = TimestepEmbedding(320)

        # 输入输出
        self.conv_in = nn.Conv2d(in_channels, 64, 3, padding=1)
        self.conv_out = nn.Conv2d(64, in_channels, 3, padding=1)

    def forward(self, x, t, context):
        """
        x: (B, 4, H/8, W/8) 潜变量
        t: (B,) 时间步
        context: (B, L, 768) 文本条件
        """
        t_emb = self.time_embed(t)
        h = self.conv_in(x)

        # 下采样
        skips = [h]
        for block in self.down_blocks:
            h = block(h, t_emb, context)
            skips.append(h)

        # 中间
        h = self.mid_block(h, t_emb, context)

        # 上采样
        for block in self.up_blocks:
            h = torch.cat([h, skips.pop()], dim=1)
            h = block(h, t_emb, context)

        return self.conv_out(h)
```

### 4.3 Classifier-Free Guidance

$$\hat{\epsilon} = \epsilon_\theta(z_t, t, \varnothing) + w \cdot (\epsilon_\theta(z_t, t, c) - \epsilon_\theta(z_t, t, \varnothing))$$

其中 $w$ 为引导权重，$\varnothing$ 为空条件。

```python
def classifier_free_guidance(model, z_t, t, context, guidance_scale=7.5):
    # 有条件预测
    eps_cond = model(z_t, t, context)

    # 无条件预测
    eps_uncond = model(z_t, t, torch.zeros_like(context))

    # 引导
    eps = eps_uncond + guidance_scale * (eps_cond - eps_uncond)
    return eps
```

## 5. 生成流程

```python
@torch.no_grad()
def stable_diffusion_generate(model, text, tokenizer, num_steps=50, guidance_scale=7.5):
    # 1. 编码文本
    text_embed = encode_text(model, text, tokenizer)

    # 2. 初始化噪声
    z = torch.randn(1, 4, 64, 64)

    # 3. 逐步去噪
    for t in reversed(range(num_steps)):
        t_batch = torch.full((1,), t, dtype=torch.long)

        # Classifier-free guidance
        eps = classifier_free_guidance(model.unet, z, t_batch, text_embed, guidance_scale)

        # DDPM 采样
        z = ddpm_step(z, eps, t, num_steps)

    # 4. VAE 解码
    image = model.vae.decode(z)
    return image
```

## 6. 从 CLIP 到 Stable Diffusion 的演进

| 模型 | 年份 | 条件机制 | 生成方式 | 分辨率 |
|------|------|---------|---------|--------|
| CLIP | 2021 | 对比学习 | - | - |
| DALL-E 1 | 2021 | 自回归 | 离散 token | 256x256 |
| DALL-E 2 | 2022 | CLIP Prior | 扩散 | 1024x1024 |
| Imagen | 2022 | T5 文本编码 | 扩散 | 1024x1024 |
| Stable Diffusion | 2022 | CLIP + UNet | 潜扩散 | 512x512 |
| SDXL | 2023 | 双 CLIP | 潜扩散 | 1024x1024 |

## 7. 关键技术对比

| 技术 | 优点 | 缺点 |
|------|------|------|
| CLIP 引导 | 灵活，无需额外训练 | 需要迭代优化，速度慢 |
| Cross-Attention | 端到端训练，效果好 | 需要专门训练 |
| Classifier-Free Guidance | 质量高，可控 | 需要两次前向传播 |
| 潜空间扩散 | 计算高效 | 依赖 VAE 质量 |

## 8. 小结

文本生成图像的核心是将文本语义有效地注入到生成过程中。
从 CLIP 的对比学习空间出发，通过 Cross-Attention 和 Classifier-Free Guidance 等技术，
Stable Diffusion 实现了高质量、可控的文本到图像生成。
这一领域仍在快速发展，视频生成、3D 生成等方向是当前热点。
