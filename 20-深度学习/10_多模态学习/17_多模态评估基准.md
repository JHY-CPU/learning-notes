# 多模态评估基准

## 1. 概述

可靠的评估基准是衡量多模态模型能力的关键。
本章介绍主流的多模态评估基准，包括 MME、MMBench、SEED-Bench 等，
以及它们的评估维度和使用方法。

## 2. MME Benchmark

### 2.1 简介

MME (Multimodal Evaluation) 是首个系统性的多模态大模型评估基准，
包含感知 (Perception) 和认知 (Cognition) 两大类任务。

### 2.2 评估维度

```
MME
├── Perception (感知)
│   ├── 存在性 (Existence)
│   ├── 计数 (Count)
│   ├── 位置 (Position)
│   ├── 颜色 (Color)
│   ├── 海报 (Poster)
│   ├── Celebrity (名人识别)
│   ├── 场景 (Scene)
│   ├── 文本识别 (OCR)
│   └── 其他
└── Cognition (认知)
    ├── 常识推理
    ├── 数值计算
    ├── 文本翻译
    ├── 代码推理
    └── 其他
```

### 2.3 评估方式

MME 使用"是/否"问答形式，评估模型的准确率和幻觉率：

```python
class MMEEvaluator:
    def __init__(self, model):
        self.model = model

    def evaluate_perception(self, dataset):
        results = {}
        for task_name, task_data in dataset.items():
            correct = 0
            total = 0

            for image, question, gt_answer in task_data:
                # 用肯定问题测试
                pred_yes = self.model.answer(image, question)
                # 用否定问题测试
                neg_question = self.negate(question)
                pred_no = self.model.answer(image, neg_question)

                if pred_yes.lower() == gt_answer.lower() and \
                   pred_no.lower() != gt_answer.lower():
                    correct += 1
                total += 1

            accuracy = correct / total
            results[task_name] = {
                "accuracy": accuracy,
                "score": accuracy * 100  # 标准化到 0-100
            }

        return results

    def negate(self, question):
        """将肯定问题转为否定问题"""
        return question.replace("Is there", "Is there no")
```

### 2.4 计分方式

$$\text{Score}_{\text{task}} = \text{Acc}_{\text{yes}} + \text{Acc}_{\text{no}}$$

其中 $\text{Acc}_{\text{yes}}$ 为肯定问题准确率，$\text{Acc}_{\text{no}}$ 为否定问题准确率。

## 3. MMBench

### 3.1 设计理念

MMBench 采用循环评估策略 (CircularEval)，比简单选择题更可靠：

```python
class CircularEvaluation:
    """
    对于每个选项，将其移到不同位置测试模型的一致性
    """
    def __init__(self, model):
        self.model = model

    def circular_eval(self, image, question, options, correct_idx):
        """
        options: ["A. 猫", "B. 狗", "C. 鸟", "D. 鱼"]
        correct_idx: 正确答案索引
        """
        num_options = len(options)
        consistent_count = 0

        # 对每个循环排列测试
        for shift in range(num_options):
            # 重排选项
            shuffled_indices = [(correct_idx + shift) % num_options
                                for _ in range(num_options)]
            shuffled_options = [options[i] for i in shuffled_indices]

            # 预测
            pred = self.model.choice(image, question, shuffled_options)

            # 检查是否一致
            if pred == shuffled_indices.index(correct_idx):
                consistent_count += 1

        return consistent_count == num_options  # 完全一致
```

### 3.2 评估维度

| 维度 | 子维度 | 说明 |
|------|--------|------|
| 粗粒度 | 识别 | 物体/场景识别 |
| 粗粒度 | 计数 | 物体数量 |
| 粗粒度 | 空间 | 空间位置关系 |
| 细粒度 | 属性 | 颜色/大小/形状 |
| 细粒度 | 关系 | 物体间关系 |
| 逻辑 | 推理 | 常识/逻辑推理 |

### 4.3 评估代码

```python
class SEEDBenchEvaluator:
    def __init__(self, model):
        self.model = model

    def evaluate(self, dataset):
        results = {}

        for category, questions in dataset.items():
            correct = 0
            total = 0

            for item in questions:
                image = item['image']  # or video
                question = item['question']
                choices = item['choices']
                answer_idx = item['answer']

                pred = self.model.answer(image, question, choices)

                if pred == answer_idx:
                    correct += 1
                total += 1

            results[category] = correct / total

        results['overall'] = sum(results.values()) / len(results)
        return results
```

### 5.3 POPE 幻觉评估

```python
class POPEEvaluator:
    """评估模型的物体级幻觉"""
    def __init__(self, model):
        self.model = model

    def evaluate(self, image, existing_objects, non_existing_objects):
        """
        测试模型是否幻觉不存在的物体
        """
        tp, fp, tn, fn = 0, 0, 0, 0

        # 测试存在的物体
        for obj in existing_objects:
            answer = self.model.answer(image, f"Is there a {obj} in the image?")
            if answer.lower() == "yes":
                tp += 1
            else:
                fn += 1

        # 测试不存在的物体
        for obj in non_existing_objects:
            answer = self.model.answer(image, f"Is there a {obj} in the image?")
            if answer.lower() == "yes":
                fp += 1
            else:
                tn += 1

        precision = tp / (tp + fp) if (tp + fp) > 0 else 0
        recall = tp / (tp + fn) if (tp + fn) > 0 else 0
        f1 = 2 * precision * recall / (precision + recall) if (precision + recall) > 0 else 0
        accuracy = (tp + tn) / (tp + tn + fp + fn)

        return {"precision": precision, "recall": recall, "f1": f1, "accuracy": accuracy}
```

## 7. 评估最佳实践

### 7.1 选择合适的基准

```python
def select_benchmarks(task_type):
    if task_type == "general_understanding":
        return ["MMBench", "SEED-Bench", "MMStar"]
    elif task_type == "hallucination":
        return ["MME-Perception", "POPE", "MMHal-Bench"]
    elif task_type == "reasoning":
        return ["MMMU", "ScienceQA", "MathVista"]
    elif task_type == "generation":
        return ["LLaVA-Bench", "MM-Vet"]
```

### 7.2 注意事项

1. **避免数据泄露**：确保评估数据不在训练集中
2. **多次运行取平均**：某些基准对随机种子敏感
3. **检查评估协议**：不同基准的 prompt 格式不同
4. **关注子维度**：总分可能掩盖特定弱项

## 8. 小结

多模态评估基准从不同维度衡量模型能力，包括感知、认知、推理、幻觉等。
MME 擅长检测幻觉，MMBench 通过循环评估提供更可靠的判断，
SEED-Bench 覆盖图像和视频的全面评估。理解每个基准的评估维度和局限性，
对于准确评估和改进模型至关重要。
