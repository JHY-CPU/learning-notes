# 多模态安全与幻觉

## 1. 概述

多模态大模型的安全性和可靠性是实际部署的关键问题。
本章讨论视觉幻觉 (Visual Hallucination)、对抗攻击、数据安全和对齐问题。

核心问题：**模型可能生成与视觉内容不符的内容，或受到精心设计的攻击**。

## 2. 视觉幻觉

### 2.1 幻觉类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 对象幻觉 | 描述不存在的物体 | "图中有一只猫"（实际没有猫） |
| 属性幻觉 | 错误描述物体属性 | "红色的苹果"（实际是绿色） |
| 关系幻觉 | 错误描述物体关系 | "猫在桌子上"（实际在桌子下） |
| 计数幻觉 | 错误计数 | "3只猫"（实际2只） |
| 空间幻觉 | 错误的空间描述 | "左边的狗"（实际在右边） |

### 2.2 幻觉来源

1. **语言先验**：模型过度依赖语言统计模式
2. **视觉接地不足**：未充分对齐视觉信息
3. **训练数据噪声**：数据中存在不准确的描述
4. **对象混淆**：相似物体被混淆

```python
class HallucinationDetector:
    """检测多模态模型的幻觉"""
    def __init__(self, grounding_model, qa_model):
        self.grounding = grounding_model
        self.qa = qa_model

    def detect_object_hallucination(self, image, description):
        """检测对象级幻觉"""
        hallucinated = []

        # 提取描述中的对象
        objects = self.extract_objects(description)

        for obj in objects:
            # 尝试定位对象
            try:
                bbox = self.grounding.localize(image, obj)
                # 验证对象确实存在
                answer = self.qa.answer(image, f"Is there a {obj} in the image?")
                if answer.lower() == "no":
                    hallucinated.append(obj)
            except ObjectNotFound:
                hallucinated.append(obj)

        return hallucinated

    def detect_attribute_hallucination(self, image, description):
        """检测属性级幻觉"""
        hallucinated = []
        attributes = self.extract_attributes(description)

        for obj, attr in attributes:
            answer = self.qa.answer(image, f"Is the {obj} {attr}?")
            if answer.lower() == "no":
                hallucinated.append((obj, attr))

        return hallucinated
```

## 3. POPE 评估幻觉

### 3.1 采样策略

POPE (Polling-based Object Probing Evaluation) 使用三种采样策略：

```python
class POPESampler:
    def __init__(self, image, existing_objects, all_objects):
        self.image = image
        self.existing = existing_objects
        self.all_objects = all_objects

    def random_sample(self, num_negatives):
        """随机负样本"""
        non_existing = [obj for obj in self.all_objects if obj not in self.existing]
        return random.sample(non_existing, num_negatives)

    def popular_sample(self, num_negatives, popularity_scores):
        """流行度采样: 选择常见的物体作为负样本"""
        non_existing = [obj for obj in self.all_objects if obj not in self.existing]
        sorted_by_pop = sorted(non_existing, key=lambda x: popularity_scores[x], reverse=True)
        return sorted_by_pop[:num_negatives]

    def adversarial_sample(self, num_negatives, similarity_scores):
        """对抗采样: 选择与正样本相似的物体"""
        candidates = []
        for pos_obj in self.existing:
            similar = similarity_scores[pos_obj]  # 与 pos_obj 相似的物体
            for sim_obj in similar:
                if sim_obj not in self.existing and sim_obj not in candidates:
                    candidates.append(sim_obj)
        return candidates[:num_negatives]
```

### 3.2 评估结果解读

| 策略 | 含义 | 模型表现 |
|------|------|---------|
| Random | 随机负样本 | 通常最高 |
| Popular | 常见物体 | 中等 |
| Adversarial | 相似物体 | 通常最低 |

## 4. 对抗攻击

### 4.1 图像对抗攻击

在图像中添加不可见的扰动，误导模型输出：

$$x_{adv} = x + \epsilon \cdot \text{sign}(\nabla_x \mathcal{L}(x, y))$$

```python
def adversarial_attack(model, image, target_text, epsilon=0.03, steps=40):
    """FGSM/PGD 对抗攻击"""
    image_adv = image.clone().detach().requires_grad_(True)
    alpha = epsilon / steps

    for _ in range(steps):
        loss = model(image_adv, target_text)
        loss.backward()

        # PGD 更新
        with torch.no_grad():
            image_adv = image_adv + alpha * image_adv.grad.sign()
            # 投影到 epsilon 球内
            delta = image_adv - image
            delta = torch.clamp(delta, -epsilon, epsilon)
            image_adv = torch.clamp(image + delta, 0, 1).detach().requires_grad_(True)

    return image_adv.detach()
```

## 5. 防御策略

### 5.1 输入验证

```python
class InputValidator:
    def __init__(self):
        self.unsafe_patterns = [
            "忽略.*指令", "假装.*没有限制", "base64", "系统提示"
        ]

    def validate_text(self, text):
        for pattern in self.unsafe_patterns:
            if re.search(pattern, text, re.IGNORECASE):
                return False, "检测到不安全输入"
        return True, ""

    def validate_image(self, image):
        # 检测对抗性扰动
        noise_level = self.estimate_noise(image)
        if noise_level > self.threshold:
            return False, "检测到图像异常"
        return True, ""
```

### 5.2 幻觉缓解

```python
class HallucinationMitigator:
    """减少视觉幻觉的技术"""
    def __init__(self, model):
        self.model = model

    def contrastive_decoding(self, image, question, alpha=0.5):
        """
        对比解码: 去除语言先验的影响
        """
        # 有条件生成
        logits_cond = self.model(image, question)

        # 无图像生成 (语言先验)
        blank_image = torch.zeros_like(image)
        logits_uncond = self.model(blank_image, question)

        # 对比: 减去语言先验
        logits = logits_cond - alpha * logits_uncond
        return logits

    def visual_grounded_generation(self, image, question):
        """
        基于视觉接地的生成
        """
        # 先定位相关区域
        regions = self.model.localize(image, question)

        # 生成时关注定位区域
        attention_mask = self.create_spatial_mask(regions, image.shape)
        return self.model.generate(image, question, attention_mask=attention_mask)
```

### 5.3 安全对齐

```python
class SafetyAlignment:
    """安全对齐训练"""
    def __init__(self, model, safety_classifier):
        self.model = model
        self.safety_classifier = safety_classifier

    def rlhf_training(self, batch):
        """使用 RLHF 进行安全对齐"""
        # 生成多个回答
        responses = []
        for _ in range(4):
            resp = self.model.generate(batch['image'], batch['prompt'])
            responses.append(resp)

        # 安全评分
        scores = [self.safety_classifier.score(resp) for resp in responses]

        # DPO 损失
        best_idx = max(range(len(scores)), key=lambda i: scores[i])
        worst_idx = min(range(len(scores)), key=lambda i: scores[i])

        loss = -torch.log(torch.sigmoid(
            self.model.log_prob(responses[best_idx]) -
            self.model.log_prob(responses[worst_idx])
        ))

        return loss
```

## 6. 安全评估指标

| 指标 | 测量内容 |
|------|---------|
| 拒绝率 (Refusal Rate) | 模型拒绝不安全请求的比例 |
| 幻觉率 (Hallucination Rate) | 生成与图像不符内容的比例 |
| 越狱成功率 (Jailbreak Success Rate) | 攻击绕过安全措施的成功率 |
| 安全评分 (Safety Score) | 综合安全性评分 |

## 7. 常见安全问题与对策

| 问题 | 风险 | 对策 |
|------|------|------|
| 视觉幻觉 | 虚假信息 | 对比解码、视觉接地 |
| 提示注入 | 安全绕过 | 输入过滤、安全微调 |
| 对抗扰动 | 误导输出 | 对抗训练、输入净化 |
| 数据隐私 | 信息泄露 | 差分隐私、数据脱敏 |
| 偏见放大 | 歧视性输出 | 数据平衡、去偏训练 |

## 8. 小结

多模态安全是一个多层面的挑战。视觉幻觉是多模态模型的固有问题，
需要从训练数据、模型架构和推理策略多个角度进行缓解。
对抗攻击和提示注入则需要输入验证和安全对齐等防御措施。
建立完善的评估体系是确保多模态系统安全部署的前提。
