# 图文匹配与检索

## 1. 任务定义

图文匹配 (Image-Text Matching) 和图文检索 (Image-Text Retrieval) 是多模态学习的基础任务：
- **图文匹配**：判断图像和文本是否语义相关（二分类）
- **图文检索**：给定图像（或文本），从候选集合中找到最相关的文本（或图像）

## 2. 双塔模型 vs 单塔模型

### 2.1 双塔模型 (Dual-Tower)

两个独立的编码器分别处理图像和文本，通过计算嵌入相似度进行匹配：

```
图像 → 视觉编码器 → 视觉嵌入 v
文本 → 文本编码器 → 文本嵌入 t
匹配分数 = sim(v, t)
```

```python
class DualTowerModel(nn.Module):
    def __init__(self, vision_encoder, text_encoder, embed_dim=256):
        super().__init__()
        self.vision_encoder = vision_encoder
        self.text_encoder = text_encoder
        self.vision_proj = nn.Linear(768, embed_dim)
        self.text_proj = nn.Linear(768, embed_dim)

    def encode_image(self, images):
        features = self.vision_encoder(images)
        return F.normalize(self.vision_proj(features), dim=-1)

    def encode_text(self, input_ids, attention_mask):
        features = self.text_encoder(input_ids, attention_mask)
        return F.normalize(self.text_proj(features[:, 0, :]), dim=-1)

    def forward(self, images, input_ids, attention_mask):
        img_emb = self.encode_image(images)
        txt_emb = self.encode_text(input_ids, attention_mask)

        # 余弦相似度
        similarity = img_emb @ txt_emb.T
        return similarity
```

**优点**：推理快（可预先计算嵌入，适合大规模检索）
**缺点**：交叉信息有限

### 2.2 单塔模型 (Single-Tower)

将图像和文本拼接后送入同一个 Transformer，通过交叉注意力建模交互：

```python
class SingleTowerModel(nn.Module):
    def __init__(self, vision_encoder, text_encoder, hidden_size=768):
        super().__init__()
        self.vision_encoder = vision_encoder
        self.text_encoder = text_encoder

        # 交叉注意力层
        self.cross_attention = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(hidden_size, 12, batch_first=True),
            num_layers=3
        )
        self.classifier = nn.Linear(hidden_size, 2)

    def forward(self, images, input_ids, attention_mask):
        # 编码
        vis_feat = self.vision_encoder(images)  # (B, N, D)
        txt_feat = self.text_encoder(input_ids, attention_mask)  # (B, L, D)

        # 拼接
        combined = torch.cat([vis_feat, txt_feat], dim=1)

        # 交叉注意力
        fused = self.cross_attention(combined)

        # 分类
        return self.classifier(fused[:, 0, :])
```

**优点**：能建模细粒度交互
**缺点**：推理慢（每个图文对需要重新计算）

### 2.3 对比

| 特性 | 双塔 | 单塔 |
|------|------|------|
| 推理速度 | 快 (预计算) | 慢 (实时计算) |
| 交互建模 | 弱 | 强 |
| 适用场景 | 大规模检索 | 精细匹配 |
| 代表模型 | CLIP, ALIGN | ViLBERT, UNITER |

## 3. 对比损失

### 3.1 InfoNCE 损失

$$\mathcal{L}_{\text{InfoNCE}} = -\frac{1}{N} \sum_{i=1}^{N} \log \frac{\exp(\text{sim}(\mathbf{v}_i, \mathbf{t}_i)/\tau)}{\sum_{j=1}^{N} \exp(\text{sim}(\mathbf{v}_i, \mathbf{t}_j)/\tau)}$$

### 3.2 对称 InfoNCE

$$\mathcal{L} = \frac{1}{2}(\mathcal{L}_{v \to t} + \mathcal{L}_{t \to v})$$

### 3.3 Hard Negative Mining

```python
class HardNegativeContrastiveLoss(nn.Module):
    def __init__(self, temperature=0.07, num_hard_negatives=5):
        super().__init__()
        self.temperature = temperature
        self.num_hard_negatives = num_hard_negatives

    def forward(self, image_embeds, text_embeds):
        # 全局相似度矩阵
        sim_matrix = image_embeds @ text_embeds.T / self.temperature

        # 找到最难的负样本
        batch_size = sim_matrix.size(0)

        # 图像到文本
        i2t_neg = sim_matrix.clone()
        i2t_neg[torch.arange(batch_size), torch.arange(batch_size)] = -1e9
        _, hard_indices = i2t_neg.topk(self.num_hard_negatives, dim=1)

        # 只用正样本和困难负样本计算损失
        pos_sim = sim_matrix.diag()  # (B,)
        neg_sim = torch.gather(sim_matrix, 1, hard_indices)  # (B, K)

        logits = torch.cat([pos_sim.unsqueeze(1), neg_sim], dim=1)
        labels = torch.zeros(batch_size, dtype=torch.long, device=logits.device)

        return F.cross_entropy(logits, labels)
```

## 4. 图文检索评估指标

### 4.1 Recall@K

$$\text{Recall@K} = \frac{|\{\text{queries where correct result in top-K}\}|}{|\text{queries}|}$$

### 4.2 常用指标

| 指标 | 含义 |
|------|------|
| R@1 | Top-1 召回率 |
| R@5 | Top-5 召回率 |
| R@10 | Top-10 召回率 |
| MR | 平均排名 (Mean Rank) |
| MRR | 平均倒数排名 |
| mAP | 平均精确率均值 |

### 4.3 计算示例

```python
def compute_recall(similarity_matrix, k=1):
    """
    similarity_matrix: (N, N) 对角线为正样本
    """
    batch_size = similarity_matrix.size(0)
    # 图像检索文本
    _, indices = similarity_matrix.topk(k, dim=1)
    i2t_recall = (indices == torch.arange(batch_size).unsqueeze(1)).any(dim=1).float().mean()

    # 文本检索图像
    _, indices = similarity_matrix.T.topk(k, dim=1)
    t2i_recall = (indices == torch.arange(batch_size).unsqueeze(1)).any(dim=1).float().mean()

    return i2t_recall.item(), t2i_recall.item()
```

## 6. 检索流程实现

```python
class ImageTextRetriever:
    def __init__(self, model, image_database, text_database):
        self.model = model
        # 预计算数据库嵌入
        self.image_embeds = self._encode_all_images(image_database)
        self.text_embeds = self._encode_all_texts(text_database)

    @torch.no_grad()
    def retrieve_text(self, query_image, top_k=10):
        """给定图像，检索最相关的文本"""
        query_embed = self.model.encode_image(query_image.unsqueeze(0))
        similarity = query_embed @ self.text_embeds.T
        _, indices = similarity.topk(top_k, dim=1)
        return indices[0].tolist()

    @torch.no_grad()
    def retrieve_image(self, query_text, top_k=10):
        """给定文本，检索最相关的图像"""
        query_embed = self.model.encode_text(
            *self.model.tokenize(query_text)
        )
        similarity = query_embed @ self.image_embeds.T
        _, indices = similarity.topk(top_k, dim=1)
        return indices[0].tolist()

    def _encode_all_images(self, images):
        embeds = []
        for batch in DataLoader(images, batch_size=64):
            embeds.append(self.model.encode_image(batch))
        return torch.cat(embeds, dim=0)
```

## 8. 小结

图文匹配与检索是多模态学习的基础任务。双塔模型适合大规模检索（快速但粗粒度），
单塔模型适合精细匹配（慢但精确）。对比学习（特别是 InfoNCE 损失）是当前最有效的训练范式。
CLIP/ALIGN 等模型在检索任务上取得了显著进展，为下游应用奠定了基础。
