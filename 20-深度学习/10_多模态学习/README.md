# 多模态学习

## 一、基本概念

### 1.1 多模态

模态指数据的表现形式，如图像、文本、音频、视频。

多模态学习：利用多种模态信息进行学习和推理。

### 1.2 核心挑战

- **对齐**：不同模态间的语义对应
- **融合**：如何有效组合多模态信息
- **翻译**：跨模态生成（如图生文）
- **缺失模态**：推理时某些模态不可用

|------|------|----------|

## 三、视觉-语言模型

### 3.1 CLIP

对比学习预训练：图像-文本对的匹配。

- 图像编码器（ViT/ResNet）
- 文本编码器（Transformer）
- 对比损失：匹配对相似度高，不匹配对相似度低

### 3.2 ALIGN

与CLIP类似，使用更大规模的噪声图文对。

### 3.3 Flamingo

少样本视觉语言模型：
- 视觉编码器 + 语言模型
- 交叉注意力插入LLM层间
- 支持交错的图文输入

### 3.4 GPT-4V / Gemini

原生多模态大模型，统一处理图像、文本、音频、视频。

## 四、文生图

- **DALL-E 2/3**：CLIP + Diffusion
- **Stable Diffusion**：潜在空间扩散 + 文本条件
- **Midjourney**：商业级文生图
- **SDXL**：更大模型，更高质量
