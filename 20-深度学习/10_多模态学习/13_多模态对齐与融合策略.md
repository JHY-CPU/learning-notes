# 多模态对齐与融合策略

## 1. 概述

多模态对齐 (Multimodal Alignment) 旨在建立不同模态之间的语义对应关系，
多模态融合 (Multimodal Fusion) 则是将对齐后的信息进行有效组合。
这两者是多模态系统的核心组件。

## 2. 对齐粒度

### 2.1 粗粒度对齐

全局级别的对齐，将整个图像和整个文本映射到同一空间：

$$\text{sim}_{\text{global}}(\mathbf{v}, \mathbf{t}) = \frac{\mathbf{v}^\top \mathbf{t}}{\|\mathbf{v}\| \|\mathbf{t}\|}$$

### 2.2 细粒度对齐

区域/词级别的对齐，建立视觉区域和文本词之间的对应：

$$\text{sim}_{\text{fine}}(v_i, t_j) = \frac{v_i^\top t_j}{\|v_i\| \|t_j\|}$$

### 2.3 层级对齐

同时进行粗粒度和细粒度对齐：

$$\mathcal{L}_{\text{align}} = \mathcal{L}_{\text{global}} + \lambda \sum_{i,j} \mathcal{L}_{\text{fine}}(v_i, t_j)$$

```python
class HierarchicalAlignment(nn.Module):
    def __init__(self, visual_dim, text_dim, hidden_dim=256):
        super().__init__()
        # 全局对齐
        self.global_v_proj = nn.Linear(visual_dim, hidden_dim)
        self.global_t_proj = nn.Linear(text_dim, hidden_dim)

        # 细粒度对齐
        self.local_v_proj = nn.Linear(visual_dim, hidden_dim)
        self.local_t_proj = nn.Linear(text_dim, hidden_dim)

    def forward(self, visual_features, text_features):
        """
        visual_features: (B, N, D_v) - N 个视觉区域
        text_features: (B, L, D_t) - L 个文本 token
        """
        # 全局: 平均池化
        global_v = self.global_v_proj(visual_features.mean(dim=1))  # (B, H)
        global_t = self.global_t_proj(text_features.mean(dim=1))    # (B, H)

        # 细粒度
        local_v = self.local_v_proj(visual_features)  # (B, N, H)
        local_t = self.local_t_proj(text_features)    # (B, L, H)

        return global_v, global_t, local_v, local_t
```

## 3. 跨模态注意力

### 3.1 双向交叉注意力

视觉到文本 + 文本到视觉的双向交叉注意力：

```python
class BidirectionalCrossAttention(nn.Module):
    def __init__(self, dim, num_heads=8):
        super().__init__()
        # 视觉 → 文本
        self.v2t_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)
        # 文本 → 视觉
        self.t2v_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)

        self.norm_v = nn.LayerNorm(dim)
        self.norm_t = nn.LayerNorm(dim)

    def forward(self, visual_feat, text_feat):
        """
        visual_feat: (B, N, D)
        text_feat: (B, L, D)
        """
        # 文本 attend to 视觉
        t2v_out, t2v_weights = self.t2v_attn(text_feat, visual_feat, visual_feat)
        text_feat = self.norm_t(text_feat + t2v_out)

        # 视觉 attend to 文本
        v2t_out, v2t_weights = self.v2t_attn(visual_feat, text_feat, text_feat)
        visual_feat = self.norm_v(visual_feat + v2t_out)

        return visual_feat, text_feat, t2v_weights, v2t_weights
```

### 3.2 Co-Attention

视觉和文本同时进行注意力计算，共享注意力权重：

```python
class CoAttention(nn.Module):
    def __init__(self, dim, num_heads=8):
        super().__init__()
        self.num_heads = num_heads
        self.head_dim = dim // num_heads

        # 共享的 QKV 投影
        self.q_v = nn.Linear(dim, dim)
        self.k_v = nn.Linear(dim, dim)
        self.v_v = nn.Linear(dim, dim)

        self.q_t = nn.Linear(dim, dim)
        self.k_t = nn.Linear(dim, dim)
        self.v_t = nn.Linear(dim, dim)

    def forward(self, visual_feat, text_feat):
        B, N, D = visual_feat.shape
        _, L, _ = text_feat.shape

        # 视觉 Q, 文本 K,V
        Q_v = self.q_v(visual_feat).reshape(B, N, self.num_heads, self.head_dim).transpose(1, 2)
        K_t = self.k_t(text_feat).reshape(B, L, self.num_heads, self.head_dim).transpose(1, 2)
        V_t = self.v_t(text_feat).reshape(B, L, self.num_heads, self.head_dim).transpose(1, 2)

        attn_v = F.softmax((Q_v @ K_t.transpose(-2, -1)) / (self.head_dim ** 0.5), dim=-1)
        out_v = (attn_v @ V_t).transpose(1, 2).reshape(B, N, D)

        # 文本 Q, 视觉 K,V
        Q_t = self.q_t(text_feat).reshape(B, L, self.num_heads, self.head_dim).transpose(1, 2)
        K_v = self.k_v(visual_feat).reshape(B, N, self.num_heads, self.head_dim).transpose(1, 2)
        V_v = self.v_v(visual_feat).reshape(B, N, self.num_heads, self.head_dim).transpose(1, 2)

        attn_t = F.softmax((Q_t @ K_v.transpose(-2, -1)) / (self.head_dim ** 0.5), dim=-1)
        out_t = (attn_t @ V_v).transpose(1, 2).reshape(B, L, D)

        return out_v, out_t, attn_v, attn_t
```

## 4. 门控融合

### 4.1 简单门控

$$g = \sigma(W_g [v; t]), \quad h = g \odot v + (1-g) \odot t$$

### 4.2 多模态门控注意力

```python
class GatedMultimodalFusion(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.v_gate = nn.Sequential(nn.Linear(dim * 2, dim), nn.Sigmoid())
        self.t_gate = nn.Sequential(nn.Linear(dim * 2, dim), nn.Sigmoid())

        self.v_transform = nn.Linear(dim, dim)
        self.t_transform = nn.Linear(dim, dim)
        self.output_proj = nn.Linear(dim * 2, dim)

    def forward(self, v, t):
        # 门控
        concat = torch.cat([v, t], dim=-1)
        g_v = self.v_gate(concat)  # 视觉门
        g_t = self.t_gate(concat)  # 文本门

        # 门控变换
        h_v = g_v * self.v_transform(v)
        h_t = g_t * self.t_transform(t)

        # 融合
        return self.output_proj(torch.cat([h_v, h_t], dim=-1))
```

## 5. 投影对齐

### 5.1 线性投影对齐

最简单的对齐方式，将不同模态投影到共享空间：

$$\mathbf{z}_v = W_v \mathbf{h}_v, \quad \mathbf{z}_t = W_t \mathbf{h}_t$$

### 5.2 非线性投影对齐

```python
class ProjectionAlignment(nn.Module):
    def __init__(self, visual_dim, text_dim, shared_dim, num_layers=2):
        super().__init__()
        # 视觉投影
        v_layers = []
        in_dim = visual_dim
        for _ in range(num_layers - 1):
            v_layers.extend([nn.Linear(in_dim, shared_dim), nn.ReLU()])
            in_dim = shared_dim
        v_layers.append(nn.Linear(in_dim, shared_dim))
        self.v_proj = nn.Sequential(*v_layers)

        # 文本投影
        t_layers = []
        in_dim = text_dim
        for _ in range(num_layers - 1):
            t_layers.extend([nn.Linear(in_dim, shared_dim), nn.ReLU()])
            in_dim = shared_dim
        t_layers.append(nn.Linear(in_dim, shared_dim))
        self.t_proj = nn.Sequential(*t_layers)

    def forward(self, visual_feat, text_feat):
        return F.normalize(self.v_proj(visual_feat), dim=-1), \
               F.normalize(self.t_proj(text_feat), dim=-1)
```

### 5.3 MLP 投影 (LLaVA 风格)

```python
class MLPProjection(nn.Module):
    """LLaVA 使用的 2 层 MLP 投影"""
    def __init__(self, visual_dim, llm_dim):
        super().__init__()
        self.proj = nn.Sequential(
            nn.Linear(visual_dim, llm_dim),
            nn.GELU(),
            nn.Linear(llm_dim, llm_dim)
        )

    def forward(self, visual_features):
        return self.proj(visual_features)
```

## 6. 融合策略对比

| 策略 | 交互方式 | 参数量 | 效果 | 代表模型 |
|------|---------|--------|------|---------|
| 早期融合 (拼接) | 简单拼接 | 低 | 一般 | 简单双塔 |
| 注意力融合 | Cross-Attention | 高 | 好 | ViLBERT |
| 门控融合 | 门控加权 | 中 | 较好 | VisualBERT |
| 投影对齐 | 线性/MLP 映射 | 低 | 好 | LLaVA |
| Q-Former | 可学习查询 | 中 | 很好 | BLIP-2 |
| 对比学习 | 共享空间 | 低 | 好 | CLIP |

## 7. 对齐损失函数

| 损失函数 | 公式 | 特点 |
|---------|------|------|
| 对比损失 | $\mathcal{L}_{CL} = -\log \frac{e^{s^+/\tau}}{\sum e^{s_i/\tau}}$ | 拉近正对，推远负对 |
| 匹配损失 | $\mathcal{L}_{ITM} = \text{CE}(f(v,t), y)$ | 二分类判别 |
| 掩码重建 | $\mathcal{L}_{MR} = \|x - \hat{x}\|^2$ | 跨模态重建 |
| 生成损失 | $\mathcal{L}_{LM} = -\sum \log P(w_t)$ | 自回归生成 |

## 8. 实践建议

1. **简单任务**：线性投影 + 对比学习即可
2. **复杂理解**：需要 Cross-Attention 建模细粒度交互
3. **生成任务**：自回归生成 + 交叉注意力条件注入
4. **大规模预训练**：对比学习 + 匹配 + 生成的多任务训练

## 9. 小结

多模态对齐与融合是连接不同模态的关键技术。从简单的拼接融合到复杂的注意力机制，
从线性投影到 Q-Former，不同策略各有优劣。选择合适的对齐和融合方式取决于具体任务需求、
模型规模和计算预算。当前的趋势是使用轻量级的投影模块（如 MLP）桥接冻结的预训练模型。
