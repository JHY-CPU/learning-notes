# 视频理解多模态

## 1. 概述

视频理解需要同时处理视觉帧序列的时间动态和语义信息。
视频-文本多模态学习将视频与自然语言对齐，实现视频理解、检索和生成等任务。

核心挑战：**如何高效地编码时间维度的信息**。

## 2. 视频表示

### 2.1 帧级特征提取

将视频分解为独立帧，逐帧提取特征：

```python
class FrameFeatureExtractor(nn.Module):
    def __init__(self, backbone='clip-vit-l/14', num_frames=16):
        super().__init__()
        self.backbone = CLIPVisionModel.from_pretrained(backbone)
        self.num_frames = num_frames

    def forward(self, video):
        """
        video: (B, T, 3, H, W) 视频帧序列
        返回: (B, T, D) 帧级特征
        """
        B, T, C, H, W = video.shape
        # 采样固定数量的帧
        if T > self.num_frames:
            indices = torch.linspace(0, T - 1, self.num_frames).long()
            video = video[:, indices]
            T = self.num_frames

        # 逐帧编码
        frames = video.reshape(B * T, C, H, W)
        features = self.backbone(frames).last_hidden_state[:, 0, :]  # CLS token
        return features.reshape(B, T, -1)
```

### 2.2 时空特征提取

使用 3D 卷积或时空 Transformer 同时编码空间和时间：

```python
class SpatioTemporalEncoder(nn.Module):
    def __init__(self, patch_size=16, tubelet_size=2, dim=768):
        super().__init__()
        # 3D Patch Embedding: (T, H, W) -> (T/t, H/p, W/p) tokens
        self.patch_embed = nn.Conv3d(
            3, dim,
            kernel_size=(tubelet_size, patch_size, patch_size),
            stride=(tubelet_size, patch_size, patch_size)
        )
        self.pos_embed = nn.Parameter(torch.randn(1, 1568, dim) * 0.02)

        self.transformer = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(dim, 12, batch_first=True),
            num_layers=12
        )

    def forward(self, video):
        """
        video: (B, 3, T, H, W)
        """
        tokens = self.patch_embed(video)  # (B, D, T', H', W')
        tokens = tokens.flatten(2).transpose(1, 2)  # (B, N, D)
        tokens = tokens + self.pos_embed[:, :tokens.size(1), :]
        return self.transformer(tokens)
```

## 3. 视频-文本对齐

### 3.1 VideoCLIP

将 CLIP 范式扩展到视频，使用帧特征的时序聚合：

```python
class VideoCLIP(nn.Module):
    def __init__(self, clip_model, aggregation='mean'):
        super().__init__()
        self.clip = clip_model
        self.aggregation = aggregation

    def encode_video(self, video_frames):
        """
        video_frames: (B, T, 3, H, W)
        """
        B, T = video_frames.shape[:2]
        frames = video_frames.reshape(B * T, *video_frames.shape[2:])

        with torch.no_grad():
            frame_features = self.clip.encode_image(frames)  # (B*T, D)

        frame_features = frame_features.reshape(B, T, -1)

        if self.aggregation == 'mean':
            return frame_features.mean(dim=1)
        elif self.aggregation == 'attention':
            return self.temporal_attention(frame_features)

    def encode_text(self, text):
        return self.clip.encode_text(text)

    def temporal_attention(self, frame_features):
        """可学习的时间注意力池化"""
        weights = self.attn_proj(frame_features)  # (B, T, 1)
        weights = F.softmax(weights, dim=1)
        return (frame_features * weights).sum(dim=1)
```

### 3.2 时序建模方法

| 方法 | 描述 | 效率 | 效果 |
|------|------|------|------|
| Mean Pooling | 帧特征平均 | 高 | 一般 |
| Attention Pooling | 可学习注意力加权 | 中 | 较好 |
| Temporal Transformer | 时序 Transformer | 低 | 好 |
| Token Merging | 帧 token 合并 | 中 | 好 |

## 4. TimeChat 架构

TimeChat 是专为视频对话设计的多模态模型，支持时间定位：

```
视频帧序列 → 视觉编码器 → 帧级特征
                            ↓
                    时间感知 Q-Former
                            ↓
文本 Query → LLM → 时间定位 + 视频理解回答
```

```python
class TimeAwareQFormer(nn.Module):
    def __init__(self, hidden_dim=768, num_queries=32, num_layers=12):
        super().__init__()
        self.queries = nn.Parameter(torch.randn(1, num_queries, hidden_dim))

        # 时间位置编码
        self.time_embed = nn.Embedding(100, hidden_dim)  # 最多100个时间片段

        self.layers = nn.ModuleList([
            TimeAwareLayer(hidden_dim)
            for _ in range(num_layers)
        ])

        self.time_head = nn.Linear(hidden_dim, 2)  # 预测起止时间

    def forward(self, frame_features, frame_timestamps):
        """
        frame_features: (B, T, D)
        frame_timestamps: (B, T, 2) 起止时间
        """
        # 添加时间位置编码
        time_ids = torch.arange(frame_features.size(1), device=frame_features.device)
        frame_features = frame_features + self.time_embed(time_ids)

        queries = self.queries.expand(frame_features.size(0), -1, -1)

        for layer in self.layers:
            queries = layer(queries, frame_features)

        # 时间定位预测
        time_pred = self.time_head(queries)  # (B, Q, 2)

        return queries, time_pred


class TimeAwareLayer(nn.Module):
    def __init__(self, dim, num_heads=12):
        super().__init__()
        self.self_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)
        self.cross_attn = nn.MultiheadAttention(dim, num_heads, batch_first=True)
        self.ffn = nn.Sequential(
            nn.Linear(dim, dim * 4), nn.GELU(), nn.Linear(dim * 4, dim)
        )
        self.norms = nn.ModuleList([nn.LayerNorm(dim) for _ in range(3)])

    def forward(self, queries, frame_features):
        q = self.norms[0](queries)
        queries = queries + self.self_attn(q, q, q)[0]

        q = self.norms[1](queries)
        queries = queries + self.cross_attn(q, frame_features, frame_features)[0]

        queries = queries + self.ffn(self.norms[2](queries))
        return queries
```

### 5.2 视频对话模型

```python
class VideoChatModel(nn.Module):
    def __init__(self, vision_encoder, temporal_qformer, llm):
        super().__init__()
        self.vision_encoder = vision_encoder
        self.temporal_qformer = temporal_qformer
        self.llm = llm
        self.proj = nn.Linear(768, 4096)

    def forward(self, video, text_prompt):
        # 视频编码
        frame_features = self.vision_encoder(video)
        queries, time_pred = self.temporal_qformer(frame_features)

        # 投影到 LLM 空间
        video_tokens = self.proj(queries)

        # LLM 生成
        text_embeds = self.llm.get_input_embeddings()(tokenize(text_prompt))
        inputs_embeds = torch.cat([video_tokens, text_embeds], dim=1)

        return self.llm(inputs_embeds=inputs_embeds)
```

## 7. 时间定位

### 7.1 任务定义

给定文本描述，在视频中定位对应的时间片段：

$$\hat{s}, \hat{e} = \arg\max_{s,e} \text{sim}(\text{Video}(s:e), \text{Query})$$

```python
class TemporalGrounding(nn.Module):
    def __init__(self, video_encoder, text_encoder, hidden_dim=768):
        super().__init__()
        self.video_encoder = video_encoder
        self.text_encoder = text_encoder

        # 候选片段生成
        self.proposal_head = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 2)  # 预测起止时间偏移
        )

        # 匹配打分
        self.match_scorer = nn.Linear(hidden_dim * 2, 1)

    def forward(self, video_features, text_query):
        text_feat = self.text_encoder(text_query)  # (B, D)

        # 生成候选片段
        proposals = []
        for start in range(0, len(video_features), 4):
            for end in range(start + 4, len(video_features), 4):
                span_feat = video_features[start:end].mean(dim=0)
                proposals.append(span_feat)

        proposals = torch.stack(proposals)  # (N, D)

        # 匹配打分
        text_expanded = text_feat.unsqueeze(0).expand(len(proposals), -1)
        scores = self.match_scorer(torch.cat([proposals, text_expanded], dim=-1))

        return scores.squeeze(-1)
```

## 8. 小结

视频理解多模态需要处理时间维度的额外复杂性。从帧级特征提取到时空编码，
从视频-文本对齐到时间定位，每个环节都有独特的挑战。
VideoCLIP、TimeChat 等模型将视觉-语言的对比学习范式成功扩展到视频领域，
实现了视频理解、检索和对话等多种能力。长视频理解和高效的时间建模仍是未来研究重点。
