# 语音与文本多模态

## 1. 概述

语音-文本多模态学习旨在建立语音信号与文本之间的桥梁。
语音包含丰富的韵律、情感和说话人信息，而文本提供精确的语义。
代表性工作包括 OpenAI 的 Whisper 和 Facebook 的 wav2vec 系列。

## 2. 语音表示

### 2.1 语音信号处理

语音是连续的时间序列信号，需要经过预处理：

```
原始音频 (16kHz) → 帧化 (25ms窗口) → MFCC/Mel频谱 → 特征序列
```

```python
import torchaudio

class AudioProcessor:
    def __init__(self, sample_rate=16000, n_mels=80, n_fft=400, hop_length=160):
        self.sample_rate = sample_rate
        self.mel_transform = torchaudio.transforms.MelSpectrogram(
            sample_rate=sample_rate,
            n_mels=n_mels,
            n_fft=n_fft,
            hop_length=hop_length
        )

    def __call__(self, waveform):
        """
        waveform: (1, T) 原始音频波形
        返回: (n_mels, T') Mel 频谱图
        """
        mel_spec = self.mel_transform(waveform)
        log_mel = torch.log(mel_spec + 1e-9)
        return log_mel.squeeze(0)  # (80, T')
```

### 2.2 离散化表示

wav2vec 2.0 使用向量量化 (VQ) 将连续特征离散化：

```python
class VectorQuantizer(nn.Module):
    def __init__(self, num_codes=320, code_dim=256):
        super().__init__()
        self.codebook = nn.Embedding(num_codes, code_dim)
        self.num_codes = num_codes

    def forward(self, z):
        """
        z: (B, T, D) 连续特征
        返回: 离散编码 indices + 量化损失
        """
        # 计算到所有码本向量的距离
        distances = torch.cdist(z, self.codebook.weight)  # (B, T, num_codes)

        # 最近邻
        indices = distances.argmin(dim=-1)  # (B, T)

        # 量化
        z_q = self.codebook(indices)

        # VQ 损失
        commitment_loss = F.mse_loss(z, z_q.detach())
        codebook_loss = F.mse_loss(z_q, z.detach())

        # 直通梯度估计
        z_q = z + (z_q - z).detach()

        return z_q, indices, commitment_loss + codebook_loss
```

## 3. Whisper 架构

### 3.1 整体设计

Whisper 使用标准的编码器-解码器 Transformer：

```
音频 → 80维Mel频谱 → 编码器 → 解码器 → 文本
                                    ↑
                            特殊任务 tokens
```

```python
class WhisperModel(nn.Module):
    def __init__(self, n_mels=80, d_model=1280, n_head=20, n_enc_layers=32, n_dec_layers=32):
        super().__init__()

        # 音频编码器
        self.conv1 = nn.Conv1d(n_mels, d_model, kernel_size=3, padding=1)
        self.conv2 = nn.Conv1d(d_model, d_model, kernel_size=3, stride=2, padding=1)
        self.encoder = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_model, n_head, batch_first=True),
            num_layers=n_enc_layers
        )

        # 文本解码器
        self.decoder = nn.TransformerDecoder(
            nn.TransformerDecoderLayer(d_model, n_head, batch_first=True),
            num_layers=n_dec_layers
        )

        self.token_embedding = nn.Embedding(51865, d_model)
        self.output_proj = nn.Linear(d_model, 51865)

    def encode(self, mel_spectrogram):
        """
        mel_spectrogram: (B, 80, T)
        """
        x = F.gelu(self.conv1(mel_spectrogram))
        x = F.gelu(self.conv2(x))
        x = x.permute(0, 2, 1)  # (B, T/2, D)
        return self.encoder(x)

    def decode(self, tokens, encoder_out):
        x = self.token_embedding(tokens)
        return self.decoder(x, encoder_out)

    def forward(self, mel_spectrogram, tokens):
        encoder_out = self.encode(mel_spectrogram)
        decoder_out = self.decode(tokens, encoder_out)
        return self.output_proj(decoder_out)
```

### 3.2 任务特殊 Token

Whisper 通过特殊 token 控制行为：

| Token | 功能 |
|-------|------|
| `<\|startoftranscript\|>` | 开始转录 |
| `<\|en\|>`, `<\|zh\|>`, ... | 指定语言 |
| `<\|transcribe\|>` | 转录任务 |
| `<\|translate\|>` | 翻译任务 |
| `<\|notimestamps\|>` | 不输出时间戳 |
| `<\|0.00\|>` - `<\|30.00\|>` | 时间戳 token |

### 3.3 推理流程

```python
@torch.no_grad()
def transcribe(model, audio_path, language="en"):
    # 1. 音频预处理
    waveform, sr = torchaudio.load(audio_path)
    if sr != 16000:
        waveform = torchaudio.transforms.Resample(sr, 16000)(waveform)
    mel = audio_processor(waveform)

    # 2. 构建 prompt tokens
    tokens = [
        tokenizer.sot,
        tokenizer.language_token(language),
        tokenizer.transcribe,
        tokenizer.no_timestamps,
    ]

    # 3. 编码
    encoder_out = model.encode(mel.unsqueeze(0))

    # 4. 自回归解码
    for _ in range(448):
        logits = model.decode(torch.tensor([tokens]), encoder_out)
        next_token = logits[0, -1].argmax().item()
        if next_token == tokenizer.eot:
            break
        tokens.append(next_token)

    return tokenizer.decode(tokens[4:])  # 去掉 prompt tokens
```

## 4. 语音-语言对齐

### 4.1 对比学习对齐

类似 CLIP，将语音和文本映射到共享空间：

$$\mathcal{L}_{\text{speech-text}} = -\log \frac{\exp(\text{sim}(s_i, t_i)/\tau)}{\sum_j \exp(\text{sim}(s_i, t_j)/\tau)}$$

```python
class SpeechTextContrastive(nn.Module):
    def __init__(self, speech_encoder, text_encoder, embed_dim=512):
        super().__init__()
        self.speech_encoder = speech_encoder
        self.text_encoder = text_encoder
        self.speech_proj = nn.Linear(768, embed_dim)
        self.text_proj = nn.Linear(768, embed_dim)
        self.temperature = nn.Parameter(torch.tensor(0.07))

    def forward(self, audio, text_tokens):
        s = F.normalize(self.speech_proj(self.speech_encoder(audio)), dim=-1)
        t = F.normalize(self.text_proj(self.text_encoder(text_tokens)), dim=-1)

        logits = s @ t.T / self.temperature
        labels = torch.arange(logits.size(0), device=logits.device)
        return (F.cross_entropy(logits, labels) + F.cross_entropy(logits.T, labels)) / 2
```

### 4.2 ASR 辅助对齐

通过自动语音识别任务强制对齐语音和文本：

$$\mathcal{L}_{\text{ASR}} = -\sum_{t=1}^{T} \log P(w_t | w_{<t}, \mathbf{s})$$

## 5. 语音-文本模型对比

| 模型 | 类型 | 训练方式 | 主要能力 |
|------|------|---------|---------|
| wav2vec 2.0 | 纯编码器 | 自监督 | 语音表示 |

## 6. 多模态语音应用

### 6.1 语音对话系统

```python
class SpeechDialogueSystem:
    def __init__(self, asr_model, llm, tts_model):
        self.asr = asr_model
        self.llm = llm
        self.tts = tts_model

    def respond(self, audio_input):
        # 语音 → 文本
        text_input = self.asr.transcribe(audio_input)

        # 文本理解 & 生成
        text_output = self.llm.generate(text_input)

        # 文本 → 语音
        audio_output = self.tts.synthesize(text_output)
        return audio_output
```

model = whisper.load_model("large-v3")

## 8. 小结

语音-文本多模态学习将连续的语音信号与离散的文本语义对齐。
Whisper 通过大规模弱监督数据实现了强大的多语言语音识别能力，
而 wav2vec/HuBERT 的自监督预训练为语音表示学习提供了高效范式。
语音-语言对齐技术使端到端语音对话成为可能，推动了 AI 助手的发展。
