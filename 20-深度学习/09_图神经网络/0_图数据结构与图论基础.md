# 0_图数据结构与图论基础

## 1. 什么是图（Graph）

图是一种由**节点（Node/Vertex）**和**边（Edge）**组成的数据结构，用于描述实体之间的关系。

$$G = (V, E)$$

- $V = \{v_1, v_2, \dots, v_N\}$：节点集合，$N = |V|$ 为节点数
- $E = \{e_1, e_2, \dots, e_M\}$：边集合，$M = |E|$ 为边数

**直觉理解**：社交网络中每个人是节点，好友关系是边；分子中每个原子是节点，化学键是边。

## 2. 图的类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 无向图 | 边没有方向，$e_{ij} = e_{ji}$ | 社交好友关系 |
| 有向图 | 边有方向，$e_{ij} \neq e_{ji}$ | 关注关系（微博/Twitter） |
| 加权图 | 边带有权重 $w_{ij}$ | 道路网络（距离/时间） |
| 二部图 | 节点分两组，边只在组间 | 用户-商品购买关系 |
| 异构图 | 多种节点类型和边类型 | 知识图谱 |
| 超图 | 一条边可连接多个节点 | 论文-作者关系 |

## 3. 邻接矩阵（Adjacency Matrix）

邻接矩阵 $A \in \mathbb{R}^{N \times N}$ 是图的矩阵表示：

$$A_{ij} = \begin{cases} 1, & \text{若存在边 } (v_i, v_j) \\ 0, & \text{否则} \end{cases}$$

对于**加权图**，$A_{ij} = w_{ij}$（边的权重）。

**关键性质**：
- 无向图：$A$ 是对称矩阵，$A = A^T$
- 有向图：$A$ 一般不对称
- 自环：$A_{ii} = 1$ 表示节点 $v_i$ 有自连接

```python
import torch
import numpy as np

# 构建邻接矩阵（无向图）
# 节点: 0-1, 0-2, 1-2, 2-3 有边
num_nodes = 4
edges = [(0,1), (0,2), (1,2), (2,3)]

A = torch.zeros(num_nodes, num_nodes)
for i, j in edges:
    A[i, j] = 1.0
    A[j, i] = 1.0  # 无向图对称

print("邻接矩阵 A:")
print(A)
# tensor([[0., 1., 1., 0.],
#         [1., 0., 1., 0.],
#         [1., 1., 0., 1.],
#         [0., 0., 1., 0.]])
```

## 4. 度矩阵（Degree Matrix）

度矩阵 $D$ 是对角矩阵，对角元素为每个节点的度：

$$D_{ii} = \deg(v_i) = \sum_{j} A_{ij}$$

- **无向图**：$\deg(v_i)$ 是与 $v_i$ 相连的边数
- **有向图**：分为入度 $\deg^-(v_i) = \sum_j A_{ji}$ 和出度 $\deg^+(v_i) = \sum_j A_{ij}$

```python
# 度矩阵
D = torch.diag(A.sum(dim=1))
print("度矩阵 D:")
print(D)
# tensor([[2., 0., 0., 0.],
#         [0., 2., 0., 0.],
#         [0., 0., 3., 0.],
#         [0., 0., 0., 1.]])

# 归一化度矩阵的逆
D_inv = torch.diag(1.0 / A.sum(dim=1))
D_inv[D_inv == float('inf')] = 0
```

## 5. 拉普拉斯矩阵（Laplacian Matrix）

### 5.1 非归一化拉普拉斯

$$L = D - A$$

**性质**：
- $L$ 是对称半正定矩阵
- 最小特征值为 0，对应特征向量 $\mathbf{1}$
- 特征值个数等于连通分量数
- $\mathbf{x}^T L \mathbf{x} = \frac{1}{2} \sum_{i,j} A_{ij}(x_i - x_j)^2$

最后一个性质说明：**拉普拉斯矩阵度量信号在图上的平滑度**。

### 5.2 对称归一化拉普拉斯

$$L_{\text{sym}} = D^{-1/2} L D^{-1/2} = I - D^{-1/2} A D^{-1/2}$$

### 5.3 随机游走归一化拉普拉斯

$$L_{\text{rw}} = D^{-1} L = I - D^{-1} A$$

```python
# 非归一化拉普拉斯
L = D - A

# 对称归一化拉普拉斯
D_inv_sqrt = torch.diag(1.0 / torch.sqrt(A.sum(dim=1)))
D_inv_sqrt[D_inv_sqrt == float('inf')] = 0
L_sym = D_inv_sqrt @ L @ D_inv_sqrt

print("拉普拉斯矩阵 L:")
print(L)
print("\n对称归一化拉普拉斯 L_sym:")
print(L_sym)
```

## 7. 图上的信号与傅里叶变换

图上的信号 $\mathbf{x} \in \mathbb{R}^N$ 是定义在节点上的函数，$x_i$ 是节点 $v_i$ 的信号值。

图傅里叶变换以拉普拉斯矩阵的特征向量为基：

$$L = U \Lambda U^T$$

其中 $U$ 是特征向量矩阵，$\Lambda = \text{diag}(\lambda_1, \dots, \lambda_N)$ 是特征值矩阵。

信号 $\mathbf{x}$ 的图傅里叶变换：

$$\hat{\mathbf{x}} = U^T \mathbf{x}$$

**频率的含义**：特征值 $\lambda_l$ 越小，对应频率越低（信号越平滑）；特征值越大，频率越高（信号变化越剧烈）。

## 9. 关键概念速查

```python
# 节点数 N，特征维度 F，边数 M
# X ∈ R^{N×F}        节点特征矩阵
# A ∈ R^{N×N}         邻接矩阵
# E ∈ R^{2×M}         边索引（COO格式）
# H^{(l)} ∈ R^{N×d}   第l层节点表示
```

**核心公式速查**：
- 邻接矩阵：$A_{ij} = 1$ if $(v_i, v_j) \in E$
- 度矩阵：$D_{ii} = \sum_j A_{ij}$
- 拉普拉斯：$L = D - A$
- 归一化拉普拉斯：$L_{\text{sym}} = I - D^{-1/2} A D^{-1/2}$
- 图傅里叶变换：$\hat{x} = U^T x$
- 平滑性度量：$\mathbf{x}^T L \mathbf{x} = \frac{1}{2}\sum_{(i,j)\in E}(x_i - x_j)^2$
