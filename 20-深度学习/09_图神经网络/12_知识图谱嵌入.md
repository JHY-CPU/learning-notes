# 12_知识图谱嵌入

## 1. 知识图谱基础

知识图谱（Knowledge Graph, KG）是由三元组组成的有向异构图：

$$(h, r, t) \quad \text{(头实体, 关系, 尾实体)}$$

例如：(爱因斯坦, 出生于, 乌尔姆)、(乌尔曼, 位于, 德国)

**知识图谱补全**（链接预测）：给定 $(h, r, ?)$ 预测最可能的 $t$，或给定 $(?, r, t)$ 预测 $h$。

## 2. TransE

Bordes et al. (2013)：最经典的平移模型，假设关系是头尾实体间的平移向量。

$$h + r \approx t$$

**评分函数**：

$$f(h, r, t) = -\|h + r - t\|_{L_1/L_2}$$

得分越高（距离越小），三元组越可信。

**训练目标**：对比学习——让正样本得分高于负样本：

$$\mathcal{L} = \sum_{(h,r,t) \in \mathcal{S}} \sum_{(h',r',t') \in \mathcal{S}'} \max(0, \gamma + f(h,r,t) - f(h',r',t'))$$

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class TransE(nn.Module):
    def __init__(self, num_entities, num_relations, embed_dim, margin=1.0):
        super().__init__()
        self.entity_emb = nn.Embedding(num_entities, embed_dim)
        self.relation_emb = nn.Embedding(num_relations, embed_dim)
        self.margin = margin

        # 初始化：实体嵌入归一化
        nn.init.xavier_uniform_(self.entity_emb.weight.data)
        nn.init.xavier_uniform_(self.relation_emb.weight.data)

    def forward(self, pos_triples, neg_triples):
        # pos_triples: [B, 3] (h, r, t)
        # neg_triples: [B, 3] (h', r', t')
        h = self.entity_emb(pos_triples[:, 0])
        r = self.relation_emb(pos_triples[:, 1])
        t = self.entity_emb(pos_triples[:, 2])

        h_neg = self.entity_emb(neg_triples[:, 0])
        r_neg = self.relation_emb(neg_triples[:, 1])
        t_neg = self.entity_emb(neg_triples[:, 2])

        pos_score = -torch.norm(h + r - t, p=2, dim=-1)
        neg_score = -torch.norm(h_neg + r_neg - t_neg, p=2, dim=-1)

        loss = F.relu(self.margin + neg_score - pos_score).mean()
        return loss

    @torch.no_grad()
    def predict(self, h, r, candidates=None):
        """预测给定 (h, r) 的尾实体"""
        h_emb = self.entity_emb(h)  # [B, d]
        r_emb = self.relation_emb(r)  # [B, d]
        target = h_emb + r_emb  # [B, d]

        if candidates is not None:
            cand_emb = self.entity_emb(candidates)  # [B, K, d]
            scores = -torch.norm(cand_emb - target.unsqueeze(1), p=2, dim=-1)
        else:
            all_emb = self.entity_emb.weight  # [N, d]
            scores = -torch.norm(all_emb - target.unsqueeze(1), p=2, dim=-1)
        return scores
```

## 3. TransR

Lin et al. (2015)：不同关系映射到不同空间。

每种关系 $r$ 有投影矩阵 $M_r \in \mathbb{R}^{d_r \times d}$：

$$h_r = M_r h, \quad t_r = M_r t$$

$$f(h, r, t) = -\|h_r + r - t_r\|$$

```python
class TransR(nn.Module):
    def __init__(self, num_entities, num_relations, embed_dim, rel_dim, margin=1.0):
        super().__init__()
        self.entity_emb = nn.Embedding(num_entities, embed_dim)
        self.relation_emb = nn.Embedding(num_relations, rel_dim)
        self.proj_mat = nn.Embedding(num_relations, rel_dim * embed_dim)
        self.margin = margin
        self.embed_dim = embed_dim
        self.rel_dim = rel_dim

    def project(self, h, r):
        """将实体投影到关系空间"""
        M = self.proj_mat(r).view(-1, self.rel_dim, self.embed_dim)
        return torch.bmm(M, h.unsqueeze(-1)).squeeze(-1)

    def forward(self, pos_triples, neg_triples):
        h = self.entity_emb(pos_triples[:, 0])
        r = self.relation_emb(pos_triples[:, 1])
        t = self.entity_emb(pos_triples[:, 2])
        h_proj = self.project(h, pos_triples[:, 1])
        t_proj = self.project(t, pos_triples[:, 1])

        pos_score = -torch.norm(h_proj + r - t_proj, p=2, dim=-1)

        h_neg = self.entity_emb(neg_triples[:, 0])
        t_neg = self.entity_emb(neg_triples[:, 2])
        h_neg_proj = self.project(h_neg, neg_triples[:, 1])
        t_neg_proj = self.project(t_neg, neg_triples[:, 1])
        neg_score = -torch.norm(h_neg_proj + self.relation_emb(neg_triples[:, 1]) - t_neg_proj, p=2, dim=-1)

        return F.relu(self.margin + neg_score - pos_score).mean()
```

## 4. RotatE

Sun et al. (2019)：关系是复数空间中的旋转。

$$t = h \circ r, \quad |r_i| = 1$$

$$f(h, r, t) = -\|h \circ r - t\|$$

其中 $\circ$ 是逐元素 Hadamard 积，$r$ 在复数单位圆上。

RotatE 可以建模：
- **对称性**：$r = [1, 1, \dots, 1]$（旋转 0 度）
- **反对称性**：$r = [-1, -1, \dots, -1]$（旋转 180 度）
- **组合性**：$r_1 \circ r_2 = r_3$
- **层次关系**

```python
class RotatE(nn.Module):
    def __init__(self, num_entities, num_relations, embed_dim, margin=6.0):
        super().__init__()
        # 复数嵌入：实部和虚部分开存储
        self.entity_emb = nn.Embedding(num_entities, embed_dim * 2)
        self.relation_emb = nn.Embedding(num_relations, embed_dim)
        self.margin = margin
        self.embed_dim = embed_dim
        nn.init.uniform_(self.entity_emb.weight, -6 / (embed_dim ** 0.5), 6 / (embed_dim ** 0.5))
        nn.init.uniform_(self.relation_emb.weight, -6 / (embed_dim ** 0.5), 6 / (embed_dim ** 0.5))

    def forward(self, pos_triples, neg_triples):
        h = self.entity_emb(pos_triples[:, 0])
        r = self.relation_emb(pos_triples[:, 1])
        t = self.entity_emb(pos_triples[:, 2])

        h_re, h_im = h.chunk(2, dim=-1)
        t_re, t_im = t.chunk(2, dim=-1)
        r_phase = r

        # 旋转：(a+bi)(cos+sin i) = (a cos - b sin) + (a sin + b cos)i
        h_r_re = h_re * torch.cos(r_phase) - h_im * torch.sin(r_phase)
        h_r_im = h_re * torch.sin(r_phase) + h_im * torch.cos(r_phase)

        pos_score = -torch.sqrt((h_r_re - t_re) ** 2 + (h_r_im - t_im) ** 2 + 1e-8).sum(-1)

        # 负样本同理
        h_neg = self.entity_emb(neg_triples[:, 0])
        t_neg = self.entity_emb(neg_triples[:, 2])
        h_neg_re, h_neg_im = h_neg.chunk(2, dim=-1)
        t_neg_re, t_neg_im = t_neg.chunk(2, dim=-1)

        h_r_neg_re = h_neg_re * torch.cos(r_phase) - h_neg_im * torch.sin(r_phase)
        h_r_neg_im = h_neg_re * torch.sin(r_phase) + h_neg_im * torch.cos(r_phase)
        neg_score = -torch.sqrt((h_r_neg_re - t_neg_re) ** 2 + (h_r_neg_im - t_neg_im) ** 2 + 1e-8).sum(-1)

        return F.relu(self.margin + neg_score - pos_score).mean()
```

## 5. ComplEx

Trouillon et al. (2016)：复数空间中的双线性模型。

$$f(h, r, t) = \text{Re}\left(\sum_k h_k \cdot r_k \cdot \bar{t}_k\right)$$

可以建模**非对称关系**（因为复数乘法不可交换）。

```python
class ComplEx(nn.Module):
    def __init__(self, num_entities, num_relations, embed_dim):
        super().__init__()
        self.ent_re = nn.Embedding(num_entities, embed_dim)
        self.ent_im = nn.Embedding(num_entities, embed_dim)
        self.rel_re = nn.Embedding(num_relations, embed_dim)
        self.rel_im = nn.Embedding(num_relations, embed_dim)

    def score(self, h, r, t):
        h_re, h_im = self.ent_re(h), self.ent_im(h)
        r_re, r_im = self.rel_re(r), self.rel_im(r)
        t_re, t_im = self.ent_re(t), self.ent_im(t)

        # Re(h * r * conj(t))
        return (h_re * r_re * t_re + h_im * r_re * t_im +
                h_re * r_im * t_im - h_im * r_im * t_re).sum(-1)
```

## 6. 负采样策略

```python
def generate_negative_samples(pos_triples, num_entities, strategy='corrupt'):
    """
    生成负样本
    strategy: 'corrupt' - 随机替换头或尾实体
    """
    neg_triples = pos_triples.clone()
    B = pos_triples.size(0)
    mask = torch.rand(B) < 0.5  # 50% 概率替换头实体

    corrupt_head = torch.randint(0, num_entities, (mask.sum(),))
    corrupt_tail = torch.randint(0, num_entities, ((~mask).sum(),))

    neg_triples[mask, 0] = corrupt_head
    neg_triples[~mask, 2] = corrupt_tail
    return neg_triples
```

## 7. 模型对比

| 模型 | 评分函数 | 空间 | 可建模模式 | 参数量 |
|------|----------|------|-----------|--------|
| TransE | $-\|h+r-t\|$ | 实数 | 平移 | 小 |
| TransR | $-\|M_r h+r-M_r t\|$ | 关系子空间 | 跨关系 | 大 |
| RotatE | $-\|h \circ r - t\|$ | 复数 | 旋转对称 | 中等 |
| ComplEx | $\text{Re}(h \cdot r \cdot \bar{t})$ | 复数 | 非对称 | 中等 |

## 8. 小结

- 知识图谱嵌入将实体和关系映射到低维向量空间
- TransE 是最基础的平移模型：$h + r \approx t$
- TransR 引入关系特定的投影空间
- RotatE 用复数旋转建模对称性和组合性
- ComplEx 在复数空间做双线性评分，适合非对称关系
- 负采样是知识图谱嵌入训练的核心技术
