# 11_动态图与时空图网络

## 1. 动态图的动机

现实世界的图不断演化：
- 社交网络中用户不断建立/删除好友关系
- 金融交易网络中资金持续流动
- 交通网络中道路拥堵状况实时变化

**静态图的局限**：将所有时间信息压缩为一张图，丢失了演化模式。

**动态图表示**：$\{G_t = (V_t, E_t, X_t)\}_{t=1}^T$

## 2. 动态图的分类

| 类型 | 结构 | 特征 | 方法 |
|------|------|------|------|
| 离散动态图 | 按时间切片为快照序列 | 每张快照是静态图 | Snapshot GNN |
| 连续动态图 | 边带时间戳 | 流式更新 | TGAT, TGN |
| 时空图 | 固定拓扑 + 动态节点特征 | 空间 + 时间关联 | ST-GCN |

## 3. 离散动态图：快照序列方法

将时间离散化为 $\{t_1, t_2, \dots, t_T\}$，每个时间步一张图快照。

### 3.1 简单方法

对每个快照独立应用 GNN，然后用 RNN 聚合时间信息：

```python
import torch
import torch.nn as nn
from torch_geometric.nn import GCNConv

class SnapshotGNN(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim, num_snapshots=10):
        super().__init__()
        self.gcn = GCNConv(in_dim, hidden_dim)
        self.rnn = nn.GRU(hidden_dim, hidden_dim, batch_first=True)
        self.lin = nn.Linear(hidden_dim, out_dim)

    def forward(self, snapshot_list):
        # snapshot_list: list of (x_t, edge_index_t)
        h_seq = []
        for x_t, edge_index_t in snapshot_list:
            h_t = torch.relu(self.gcn(x_t, edge_index_t))
            h_seq.append(h_t.mean(dim=0))  # 图级别表示

        h_seq = torch.stack(h_seq).unsqueeze(0)  # [1, T, d]
        _, h_n = self.rnn(h_seq)  # h_n: [1, 1, d]
        return self.lin(h_n.squeeze())
```

## 4. 连续时间动态图

### 4.1 DyRep

Trivedi et al. (2019)：边事件的生成建模。

每个事件 $(v_i, v_j, t, r)$ 的强度：

$$\lambda_{ij}(t) = \sigma\left(h_{v_i}(t)^T h_{v_j}(t)\right)$$

节点表示通过事件更新：

$$h_{v_i}(t^+) = \text{UPDATE}(h_{v_i}(t), h_{v_j}(t))$$

```python
class DyRepUpdate(nn.Module):
    def __init__(self, hidden_dim):
        super().__init__()
        self.gru = nn.GRUCell(hidden_dim, hidden_dim)

    def forward(self, h_i, h_j, delta_t):
        # h_i: 源节点当前表示, h_j: 目标节点当前表示
        # 简化的更新：用目标节点信息更新源节点
        msg = torch.tanh(h_j)
        h_i_new = self.gru(msg, h_i)
        return h_i_new
```

### 4.2 TGAT（时间图注意力网络）

Xu et al. (2020)：将时间编码融入图注意力。

**时间编码**：使用与 Transformer 位置编码类似的方式编码时间差：

$$\Phi(\Delta t) = \left[\sin(\omega_1 \Delta t), \cos(\omega_1 \Delta t), \dots, \sin(\omega_d \Delta t), \cos(\omega_d \Delta t)\right]$$

其中 $\omega_k = \frac{1}{10000^{2k/d}}$。

**时间图注意力**：

$$e_{vu}(t) = \text{attn}(W h_v(t) + \Phi(t - t_v) \| W h_u(t) + \Phi(t - t_u))$$

```python
class TimeEncoding(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.dim = dim
        self.w = nn.Parameter(torch.randn(1, dim // 2))
        nn.init.xavier_uniform_(self.w)

    def forward(self, t):
        # t: [N] 时间差
        t = t.unsqueeze(-1)  # [N, 1]
        freq = t * self.w * 2 * 3.14159  # [N, d/2]
        return torch.cat([torch.sin(freq), torch.cos(freq)], dim=-1)  # [N, d]

class TGATLayer(nn.Module):
    def __init__(self, in_dim, out_dim, time_dim):
        super().__init__()
        self.time_enc = TimeEncoding(time_dim)
        self.W_q = nn.Linear(in_dim + time_dim, out_dim)
        self.W_k = nn.Linear(in_dim + time_dim, out_dim)
        self.W_v = nn.Linear(in_dim, out_dim)

    def forward(self, x, edge_index, t_src, t_dst, t_cur):
        # 时间编码
        te_src = self.time_enc(t_cur - t_src)
        te_dst = self.time_enc(t_cur - t_dst)

        # 拼接时间信息
        q = self.W_q(torch.cat([x[edge_index[0]], te_src], dim=-1))
        k = self.W_k(torch.cat([x[edge_index[1]], te_dst], dim=-1))
        v = self.W_v(x[edge_index[1]])

        # 注意力
        alpha = torch.softmax((q * k).sum(-1, keepdim=True), dim=0)
        return alpha * v
```

## 5. 时空图网络（ST-GNN）

交通预测是时空图的经典应用：道路网络是图，每个节点有时间序列特征。

### 5.1 ST-GCN 框架

$$H^{(l+1)} = \text{TCN}\left(\text{GCN}(H^{(l)})\right)$$

- **空间维度**：GCN 提取相邻节点的空间关联
- **时间维度**：时序卷积网络（TCN）或 RNN 捕捉时间模式

```python
class STGCNBlock(nn.Module):
    """时空图卷积块"""
    def __init__(self, in_dim, hidden_dim, out_dim, kernel_size=3):
        super().__init__()
        # 空间卷积
        self.spatial_conv = GCNConv(in_dim, hidden_dim)
        # 时间卷积 (1D 因果卷积)
        self.temporal_conv = nn.Conv1d(
            hidden_dim, hidden_dim,
            kernel_size=kernel_size,
            padding=kernel_size - 1,  # 因果填充
        )
        # 输出投影
        self.output_conv = nn.Conv1d(hidden_dim, out_dim, kernel_size=1)
        self.norm = nn.LayerNorm(out_dim)

    def forward(self, x_seq, edge_index):
        # x_seq: [N, T, d] 各节点的时间序列特征
        N, T, _ = x_seq.shape
        out_seq = []
        for t in range(T):
            # 空间卷积
            h_t = F.relu(self.spatial_conv(x_seq[:, t, :], edge_index))
            out_seq.append(h_t)

        # 堆叠后做时间卷积
        h = torch.stack(out_seq, dim=1)  # [N, T, d']
        h = h.permute(0, 2, 1)  # [N, d', T]
        h = F.relu(self.temporal_conv(h)[:, :, :T])  # 因果卷积：去掉未来信息
        h = F.relu(self.output_conv(h)).permute(0, 2, 1)  # [N, T, out_dim]
        return self.norm(h)

class STGCN(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim, num_nodes, pred_steps=12):
        super().__init__()
        self.block1 = STGCNBlock(in_dim, hidden_dim, hidden_dim)
        self.block2 = STGCNBlock(hidden_dim, hidden_dim, hidden_dim)
        self.pred_steps = pred_steps
        self.lin = nn.Linear(hidden_dim, out_dim * pred_steps)

    def forward(self, x, edge_index):
        # x: [N, T, d]
        h = self.block1(x, edge_index)
        h = self.block2(h, edge_index)
        h = h[:, -1, :]  # 取最后一个时间步
        out = self.lin(h)  # [N, pred_steps * out_dim]
        return out.view(-1, self.pred_steps, self.lin.out_features // self.pred_steps)
```

## 6. 动态图方法对比

| 方法 | 时间建模 | 更新方式 | 适用场景 |
|------|----------|----------|----------|
| DyRep | 连续时间 | 事件驱动 | 社交网络 |
| TGAT | 连续时间 | 邻居聚合+时间编码 | 通用 |
| TGN | 连续时间 | 记忆模块+消息 | 通用 |
| ST-GCN | 离散时间 | 空间+时间卷积 | 交通预测 |
| EvolveGCN | 离散快照 | RNN 演化权重 | 社区发现 |

## 7. 小结

- 动态图建模图的演化：离散快照 vs 连续时间事件
- DyRep 用强度函数建模事件发生概率
- TGAT 将时间编码融入注意力机制
- 时空图网络结合空间 GCN 和时间 TCN，适合交通预测
- 根据应用场景选择离散/连续时间建模方式
