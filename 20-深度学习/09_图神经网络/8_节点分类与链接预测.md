# 8_节点分类与链接预测

## 1. 图上的学习任务

| 任务 | 输入 | 输出 | 示例 |
|------|------|------|------|
| 节点分类 | 图 + 部分节点标签 | 每个节点的类别 | 论文分类、用户画像 |
| 链接预测 | 图（部分边可见） | 预测缺失边 | 推荐系统、知识补全 |
| 图分类 | 多张图 | 每张图的类别 | 分子属性、蛋白质分类 |

## 2. 节点分类：半监督学习

### 2.1 问题形式化

给定图 $G = (V, E)$，只有部分节点 $\{v_i : i \in \mathcal{L}\}$ 有标签，目标是预测未标注节点的标签。

$$\min_\theta \sum_{i \in \mathcal{L}} \mathcal{L}(f_\theta(G)_i, y_i)$$

图的结构本身提供了强大的正则化——**同质性假设**：相连节点往往标签相同。

### 2.2 转导式 vs 归纳式学习

**转导式（Transductive）**：训练时可以访问整个图（包括测试节点的特征和边），只预测测试节点的标签。

- GCN 是转导式的：归一化矩阵 $\hat{D}^{-1/2}\hat{A}\hat{D}^{-1/2}$ 依赖整个图结构
- 无法泛化到训练时未见过的新节点

**归纳式（Inductive）**：训练时只能访问训练图，需要泛化到全新的图或节点。

- GraphSAGE 是归纳式的：学到聚合函数，不依赖特定图结构
- 可以处理动态图和新增节点

```python
# 转导式：GCN 在 Cora 上
from torch_geometric.datasets import Planetoid
dataset = Planetoid(root='/tmp/Cora', name='Cora')
data = dataset[0]

# 训练时使用整个图结构
for epoch in range(200):
    model.train()
    optimizer.zero_grad()
    out = model(data.x, data.edge_index)  # 所有节点参与计算
    loss = F.nll_loss(out[data.train_mask], data.y[data.train_mask])
    loss.backward()
    optimizer.step()
```

```python
# 归纳式：GraphSAGE 使用邻居采样
from torch_geometric.loader import NeighborLoader

# 训练时每次只采样子图
train_loader = NeighborLoader(
    data,
    num_neighbors=[25, 10],  # 每层采样邻居数
    batch_size=256,
    input_nodes=data.train_mask,

for batch in train_loader:
    out = model(batch.x, batch.edge_index)
    loss = F.nll_loss(out[:batch.batch_size], batch.y[:batch.batch_size])
```

### 2.3 Cora 数据集实验

```python
import torch
import torch.nn.functional as F
from torch_geometric.nn import GCNConv, SAGEConv
from torch_geometric.datasets import Planetoid

class TransductiveGCN(torch.nn.Module):
    """转导式 GCN"""
    def __init__(self, in_dim, hidden_dim, out_dim):
        super().__init__()
        self.conv1 = GCNConv(in_dim, hidden_dim)
        self.conv2 = GCNConv(hidden_dim, out_dim)

    def forward(self, x, edge_index):
        x = F.relu(self.conv1(x, edge_index))
        x = F.dropout(x, p=0.5, training=self.training)
        return F.log_softmax(self.conv2(x, edge_index), dim=1)

class InductiveSAGE(torch.nn.Module):
    """归纳式 GraphSAGE"""
    def __init__(self, in_dim, hidden_dim, out_dim):
        super().__init__()
        self.conv1 = SAGEConv(in_dim, hidden_dim)
        self.conv2 = SAGEConv(hidden_dim, out_dim)

    def forward(self, x, edge_index):
        x = F.relu(self.conv1(x, edge_index))
        x = F.dropout(x, p=0.5, training=self.training)
        return F.log_softmax(self.conv2(x, edge_index), dim=1)
```

## 3. 链接预测

### 3.1 问题形式化

给定图 $G = (V, E)$，随机隐藏部分边，模型预测这些边是否存在。

$$P(\text{edge}_{ij} \text{ exists}) = \sigma(h_i^T h_j)$$

### 3.2 训练流程

1. 随机删除一定比例的边作为正样本
2. 随机采样不存在的边作为负样本
3. 用剩余图训练 GNN 得到节点嵌入
4. 用嵌入预测边是否存在

```python
from torch_geometric.transforms import RandomLinkSplit
from torch_geometric.utils import negative_sampling

# 划分边
transform = RandomLinkSplit(
    num_val=0.05,
    num_test=0.1,
    is_undirected=True,
    add_negative_train_samples=False,  # 训练时动态采样
)
train_data, val_data, test_data = transform(data)

class LinkPredictor(torch.nn.Module):
    def __init__(self, in_dim, hidden_dim):
        super().__init__()
        self.conv1 = GCNConv(in_dim, hidden_dim)
        self.conv2 = GCNConv(hidden_dim, hidden_dim)

    def encode(self, x, edge_index):
        x = F.relu(self.conv1(x, edge_index))
        return self.conv2(x, edge_index)

    def decode(self, z, edge_label_index):
        """解码器：点积"""
        return (z[edge_label_index[0]] * z[edge_label_index[1]]).sum(dim=-1)

    def forward(self, data):
        z = self.encode(data.x, data.edge_index)
        return self.decode(z, data.edge_label_index)
```

### 3.3 负采样策略

```python
# 随机负采样
neg_edge_index = negative_sampling(
    edge_index=train_data.edge_index,
    num_nodes=train_data.num_nodes,
    num_neg_samples=train_data.edge_label_index.size(1),  # 与正样本等量
)

# 混合正负样本
edge_label_index = torch.cat([
    train_data.edge_label_index,
    neg_edge_index,
], dim=-1)
edge_label = torch.cat([
    torch.ones(train_data.edge_label_index.size(1)),
    torch.zeros(neg_edge_index.size(1)),
])
```

### 3.4 训练与评估

```python
def train_link(model, data, optimizer):
    model.train()
    optimizer.zero_grad()

    # 动态负采样
    neg_edge_index = negative_sampling(
        data.edge_index, data.num_nodes,
        num_neg_samples=data.edge_label_index.size(1),
    )
    edge_label_index = torch.cat([data.edge_label_index, neg_edge_index], dim=-1)
    edge_label = torch.cat([
        torch.ones(data.edge_label_index.size(1)),
        torch.zeros(neg_edge_index.size(1)),
    ])

    out = model.decode(model.encode(data.x, data.edge_index), edge_label_index)
    loss = F.binary_cross_entropy_with_logits(out, edge_label)
    loss.backward()
    optimizer.step()
    return loss.item()

@torch.no_grad()
def test_link(model, data):
    model.eval()
    z = model.encode(data.x, data.edge_index)
    out = model.decode(z, data.edge_label_index).sigmoid()
    from sklearn.metrics import roc_auc_score
    return roc_auc_score(data.edge_label.cpu().numpy(), out.cpu().numpy())
```

## 4. 链接预测的解码器

| 解码器 | 公式 | 说明 |
|--------|------|------|
| 点积 | $\sigma(h_i^T h_j)$ | 简单高效 |
| DistMult | $\sigma(h_i \odot h_j \odot h_r)$ | 考虑关系类型 |
| MLP | $\sigma(\text{MLP}(h_i \| h_j))$ | 更强表达力 |
| 双线性 | $\sigma(h_i^T W h_j)$ | 可学习相似度度量 |

```python
class MLPPredictor(nn.Module):
    """MLP 解码器，表达力更强"""
    def __init__(self, hidden_dim):
        super().__init__()
        self.mlp = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1),
        )

    def forward(self, z, edge_index):
        src, dst = edge_index
        h = torch.cat([z[src], z[dst]], dim=-1)
        return self.mlp(h).squeeze(-1)
```

## 5. 链接预测评估指标

| 指标 | 说明 |
|------|------|
| ROC-AUC | 常用指标 |
| MRR (Mean Reciprocal Rank) | 排名质量 |
| Hits@K | 正样本排名前 K 的比例 |

```python
def hits_at_k(z, pos_edge_index, neg_edge_index, k=20):
    """计算 Hits@K"""
    pos_scores = (z[pos_edge_index[0]] * z[pos_edge_index[1]]).sum(-1)
    neg_scores = (z[neg_edge_index[0]] * z[neg_edge_index[1]]).sum(-1)

    hits = 0
    for pos_score in pos_scores:
        rank = (neg_scores >= pos_score).sum().item() + 1
        if rank <= k:
            hits += 1
    return hits / len(pos_scores)
```

## 6. 标签泄漏问题

链接预测中一个常见陷阱：**边泄漏（edge leakage）**。

如果在训练时使用了测试边来计算节点嵌入，评估结果会虚高。

**正确做法**：使用 `RandomLinkSplit` 自动处理边的划分，确保训练/验证/测试使用不同的边集。

```python
transform = RandomLinkSplit(
    num_val=0.05, num_test=0.1,
    is_undirected=True,
    split_labels=True,       # 分离边标签
    add_negative_train_samples=False,
)
train_data, val_data, test_data = transform(data)
# train_data.edge_index: 训练图结构（不含验证/测试边）
# train_data.edge_label_index: 训练正样本边
```

## 7. 小结

- 节点分类：半监督学习，利用图结构做正则化
- 转导式（GCN）依赖整个图结构；归纳式（GraphSAGE）可泛化到新节点
- 链接预测：编码器-解码器框架，常用点积或 MLP 解码器
- 负采样是链接预测训练的关键步骤
- 注意边泄漏问题，使用 `RandomLinkSplit` 正确划分数据
