# 10_异构图神经网络

## 1. 同构图 vs 异构图

**同构图（Homogeneous Graph）**：所有节点和边类型相同。

$$G = (V, E), \quad \text{单一种类的节点和边}$$

**异构图（Heterogeneous Graph）**：包含多种节点类型和边类型。

$$G = (V, E, \mathcal{T}_V, \mathcal{T}_E)$$

- $\mathcal{T}_V$：节点类型集合（如"用户"、"商品"、"评论"）
- $\mathcal{T}_E$：边类型集合（如"购买"、"撰写"、"属于"）

**示例——学术知识图谱**：
- 节点类型：论文 (Paper)、作者 (Author)、会议 (Venue)
- 边类型：撰写 (writes)、发表于 (published_in)、引用 (cites)

## 2. 元路径（Meta-path）

元路径是连接两个节点的**类型序列**，定义了特定的语义关系。

$$A_1 \xrightarrow{R_1} A_2 \xrightarrow{R_2} \cdots \xrightarrow{R_l} A_{l+1}$$

示例：
- **APVPA**：作者 → 论文 → 会议 → 论文 → 作者（同一会议发表论文的作者）
- **APA**：作者 → 论文 → 作者（合作关系）

元路径引导下的邻居聚合：

$$h_v^{(l+1)} = \text{AGG}\left(\{h_u : u \xrightarrow{\text{meta-path}} v\}\right)$$

## 3. R-GCN（关系图卷积网络）

Schlichtkrull et al. (2018)：为每种边类型学习独立的变换矩阵。

### 3.1 核心公式

$$h_v^{(l+1)} = \sigma\left(\sum_{r \in \mathcal{R}} \sum_{u \in \mathcal{N}_r(v)} \frac{1}{c_{v,r}} W_r^{(l)} h_u^{(l)} + W_0^{(l)} h_v^{(l)}\right)$$

其中：
- $\mathcal{R}$：关系类型集合
- $\mathcal{N}_r(v)$：$v$ 在关系 $r$ 下的邻居
- $W_r$：关系 $r$ 的权重矩阵
- $W_0$：自环权重
- $c_{v,r}$：归一化常数

### 3.2 参数共享（基分解）

每种关系一个 $W_r$ 会导致参数爆炸。R-GCN 使用**基分解（Basis Decomposition）**：

$$W_r = \sum_{b=1}^{B} a_{rb} V_b$$

- $V_b \in \mathbb{R}^{d' \times d}$：共享基矩阵
- $a_{rb}$：关系 $r$ 对基 $b$ 的系数

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import RGCNConv

class RGCN(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim, num_relations, num_bases=4):
        super().__init__()
        self.conv1 = RGCNConv(in_dim, hidden_dim, num_relations, num_bases=num_bases)
        self.conv2 = RGCNConv(hidden_dim, out_dim, num_relations, num_bases=num_bases)
        self.dropout = nn.Dropout(0.5)

    def forward(self, x, edge_index, edge_type):
        x = F.relu(self.conv1(x, edge_index, edge_type))
        x = self.dropout(x)
        return F.log_softmax(self.conv2(x, edge_index, edge_type), dim=1)

# 使用示例
# edge_type: [N_edges] 每条边的关系类型标签
# x: [N_nodes, in_dim] 节点特征
# edge_index: [2, N_edges]
```

### 3.3 R-GCN 用于知识图谱

R-GCN 可以作为知识图谱的编码器，配合解码器（如 DistMult）做链接预测：

```python
class RGCNEncoder(nn.Module):
    def __init__(self, num_nodes, hidden_dim, num_relations, num_bases=4):
        super().__init__()
        self.node_emb = nn.Embedding(num_nodes, hidden_dim)
        self.conv1 = RGCNConv(hidden_dim, hidden_dim, num_relations, num_bases)
        self.conv2 = RGCNConv(hidden_dim, hidden_dim, num_relations, num_bases)

    def forward(self, edge_index, edge_type):
        x = self.node_emb.weight
        x = F.relu(self.conv1(x, edge_index, edge_type))
        return self.conv2(x, edge_index, edge_type)

class DistMultDecoder(nn.Module):
    """解码器：DistMult 三元组评分"""
    def __init__(self, hidden_dim, num_relations):
        super().__init__()
        self.rel_emb = nn.Embedding(num_relations, hidden_dim)

    def forward(self, z, edge_index, edge_type):
        src, dst = edge_index
        rel = self.rel_emb(edge_type)
        return (z[src] * rel * z[dst]).sum(dim=-1)
```

## 4. HAN（异构图注意力网络）

Wang et al. (2019)：基于元路径的注意力机制。

### 4.1 两层注意力

**节点级注意力**：在同一条元路径下，学习邻居的重要性：

$$e_{ij}^{\Phi} = \text{attn}(h_i, h_j, \Phi)$$

**语义级注意力**：不同元路径的重要性：

$$w_{\Phi_i} = \frac{1}{|V|} \sum_{v \in V} q^T \tanh(W \cdot h_v^{\Phi_i} + b)$$

$$\beta_{\Phi_i} = \frac{\exp(w_{\Phi_i})}{\sum_j \exp(w_{\Phi_j})}$$

最终表示：

$$H = \sum_{\Phi} \beta_\Phi \cdot H^\Phi$$

```python
class SemanticAttention(nn.Module):
    """语义级注意力"""
    def __init__(self, hidden_dim, num_meta_paths):
        super().__init__()
        self.attn = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 4),
            nn.Tanh(),
            nn.Linear(hidden_dim // 4, 1, bias=False),
        )

    def forward(self, meta_path_embeds):
        # meta_path_embeds: list of [N, d]，每条元路径的节点嵌入
        w = []
        for h in meta_path_embeds:
            w.append(self.attn(h).mean(dim=0))  # [1]
        w = torch.softmax(torch.stack(w), dim=0)  # [num_paths]

        H = sum(w[i] * meta_path_embeds[i] for i in range(len(w)))
        return H
```

## 6. PyG 中的异构图模型

```python
from torch_geometric.nn import HANConv

class HANModel(nn.Module):
    def __init__(self, in_channels_dict, hidden_dim, out_dim, metadata):
        super().__init__()
        self.conv1 = HANConv(in_channels_dict, hidden_dim, heads=4, metadata=metadata)
        self.conv2 = HANConv(hidden_dim, out_dim, heads=4, metadata=metadata)
        self.lin = nn.Linear(out_dim, out_dim)

    def forward(self, x_dict, edge_index_dict):
        x_dict = self.conv1(x_dict, edge_index_dict)
        x_dict = {k: F.relu(v) for k, v in x_dict.items()}
        x_dict = self.conv2(x_dict, edge_index_dict)
        return self.lin(x_dict['paper'])
```

## 7. 小结

- 异构图包含多种节点类型和边类型，语义更丰富
- 元路径定义了节点间的语义关系，是异构图方法的核心概念
- R-GCN：为每种关系类型学习独立变换，用基分解控制参数量
- HAN：结合节点级和语义级注意力，自动学习元路径重要性
- PyG 的 `HeteroData` 和 `HANConv` 提供了便捷的异构图支持
