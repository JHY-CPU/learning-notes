# 7_图分类任务实践

## 1. 图分类任务概述

图分类要求对**整张图**进行预测：给定图 $G = (V, E)$ 和标签 $y$，学习映射 $f: \mathcal{G} \rightarrow \mathcal{Y}$。

**典型应用场景**：
- 分子属性预测（有毒/无毒）
- 蛋白质功能分类
- 社交网络分类
- 程序图分类（bug 检测）

**基本流程**：节点编码 → 图池化（读出） → 分类器

$$h_G = \text{READOUT}\left(\{h_v^{(L)} : v \in V\}\right), \quad \hat{y} = \text{MLP}(h_G)$$

## 2. 分子属性预测

### 2.1 分子图表示

分子天然适合表示为图：
- **节点**：原子（特征：原子类型、度数、电荷、杂化类型等）
- **边**：化学键（特征：键类型、是否共轭、是否在环中）

```python
from rdkit import Chem
import torch
from torch_geometric.data import Data

def mol_to_graph(smiles):
    mol = Chem.MolFromSmiles(smiles)
    # 原子特征
    atom_features = []
    for atom in mol.GetAtoms():
        features = [
            atom.GetAtomicNum(),          # 原子序数
            atom.GetDegree(),             # 度数
            atom.GetFormalCharge(),       # 形式电荷
            int(atom.GetHybridization()), # 杂化类型
            int(atom.GetIsAromatic()),    # 是否芳香
            atom.GetTotalNumHs(),         # 氢原子数
        ]
        atom_features.append(features)

    # 边索引
    edge_index = []
    edge_attr = []
    for bond in mol.GetBonds():
        i, j = bond.GetBeginAtomIdx(), bond.GetEndAtomIdx()
        edge_index.extend([[i, j], [j, i]])
        bond_type = int(bond.GetBondType())
        edge_attr.extend([bond_type, bond_type])

    x = torch.tensor(atom_features, dtype=torch.float)
    edge_index = torch.tensor(edge_index, dtype=torch.long).t().contiguous()
    edge_attr = torch.tensor(edge_attr, dtype=torch.float)
    return Data(x=x, edge_index=edge_index, edge_attr=edge_attr)
```

## 3. 图分类模型实现

### 3.1 GNN + 全局池化

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from torch_geometric.nn import GCNConv, global_mean_pool, global_max_pool

class GraphClassifier(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim, num_layers=3):
        super().__init__()
        self.convs = nn.ModuleList()
        self.bns = nn.ModuleList()
        self.convs.append(GCNConv(in_dim, hidden_dim))
        self.bns.append(nn.BatchNorm1d(hidden_dim))
        for _ in range(num_layers - 1):
            self.convs.append(GCNConv(hidden_dim, hidden_dim))
            self.bns.append(nn.BatchNorm1d(hidden_dim))

        self.lin1 = nn.Linear(hidden_dim * 2, hidden_dim)
        self.lin2 = nn.Linear(hidden_dim, out_dim)
        self.dropout = nn.Dropout(0.5)

    def forward(self, data):
        x, edge_index, batch = data.x, data.edge_index, data.batch
        for conv, bn in zip(self.convs, self.bns):
            x = F.relu(bn(conv(x, edge_index)))
            x = self.dropout(x)

        # 图级别读出
        h_mean = global_mean_pool(x, batch)
        h_max = global_max_pool(x, batch)
        h_graph = torch.cat([h_mean, h_max], dim=-1)

        h_graph = self.dropout(F.relu(self.lin1(h_graph)))
        return self.lin2(h_graph)
```

### 3.2 使用边特征的模型

分子图中边特征（化学键类型）很重要，可以使用 `GINEConv`（带边特征的 GIN）：

```python
from torch_geometric.nn import GINEConv, global_add_pool

class MoleculeGNN(nn.Module):
    def __init__(self, in_dim, edge_dim, hidden_dim, out_dim, num_layers=5):
        super().__init__()
        self.node_enc = nn.Linear(in_dim, hidden_dim)
        self.edge_enc = nn.Linear(edge_dim, hidden_dim)

        self.convs = nn.ModuleList()
        self.bns = nn.ModuleList()
        for _ in range(num_layers):
            mlp = nn.Sequential(
                nn.Linear(hidden_dim, hidden_dim * 2),
                nn.ReLU(),
                nn.Linear(hidden_dim * 2, hidden_dim),
            )
            self.convs.append(GINEConv(mlp, edge_dim=hidden_dim))
            self.bns.append(nn.BatchNorm1d(hidden_dim))

        self.lin = nn.Linear(hidden_dim, out_dim)

    def forward(self, data):
        x = self.node_enc(data.x)
        edge_attr = self.edge_enc(data.edge_attr)

        for conv, bn in zip(self.convs, self.bns):
            x = F.relu(bn(conv(x, data.edge_index, edge_attr)))

        h_graph = global_add_pool(x, data.batch)  # 求和池化（理论分析最优）
        return self.lin(h_graph)
```

## 5. 训练与评估

```python
def train(model, loader, optimizer, criterion):
    model.train()
    total_loss = 0
    for data in loader:
        optimizer.zero_grad()
        out = model(data)
        loss = criterion(out, data.y)
        loss.backward()
        optimizer.step()
        total_loss += loss.item() * data.num_graphs
    return total_loss / len(loader.dataset)

@torch.no_grad()
def evaluate(model, loader):
    model.eval()
    correct = 0
    for data in loader:
        pred = model(data).argmax(dim=1)
        correct += (pred == data.y).sum().item()
    return correct / len(loader.dataset)

# 训练循环
model = GraphClassifier(
    in_dim=dataset.num_node_features,
    hidden_dim=64,
    out_dim=dataset.num_classes,
)
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3, weight_decay=1e-5)
criterion = nn.CrossEntropyLoss()

for epoch in range(1, 201):
    loss = train(model, train_loader, optimizer, criterion)
    train_acc = evaluate(model, train_loader)
    test_acc = evaluate(model, test_loader)
    if epoch % 20 == 0:
        print(f'Epoch {epoch:03d}, Loss: {loss:.4f}, '
              f'Train: {train_acc:.4f}, Test: {test_acc:.4f}')
```

## 6. 训练技巧与注意事项

### 6.1 数据划分注意事项

分子数据集不能简单随机划分（避免信息泄漏）：
- 按 scaffold（骨架）划分：确保训练/测试的分子结构不同
- 按时间划分：模拟实际应用场景

### 6.2 评估指标

| 任务类型 | 常用指标 |
|----------|----------|
| 二分类 | ROC-AUC, PR-AUC |
| 多分类 | 准确率, F1 |
| 多标签 | 每任务分别计算 AUC，取平均 |
| 回归 | RMSE, MAE, $R^2$ |

## 7. 小结

- 图分类流程：节点编码 → 消息传递 → 图池化 → 分类器
- 分子图：原子为节点、化学键为边，使用带边特征的 GNN（GINE）
- 蛋白质分类：残基图或氨基酸图
- 池化策略：全局 mean+max 拼接简单有效；求和池化在理论上具有最强区分力
- 评估时注意数据划分策略，避免信息泄漏
