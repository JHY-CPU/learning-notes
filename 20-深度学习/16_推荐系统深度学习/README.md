# 推荐系统深度学习

## 一、推荐系统基础

### 1.1 范式

- **协同过滤**：基于用户-物品交互
- **内容过滤**：基于物品/用户特征
- **混合方法**：结合两者

### 1.2 评估指标

- Precision@K、Recall@K
- NDCG（归一化折损累积增益）
- MAP（平均精度均值）
- AUC

---

## 二、深度学习模型

### 2.1 Embedding方法

- **Matrix Factorization**：用户和物品的低维嵌入
- **Neural CF**：用神经网络替代内积

### 2.2 序列推荐

- **GRU4Rec**：用GRU建模用户行为序列
- **SASRec**：Self-Attention序列推荐
- **BERT4Rec**：用BERT做序列推荐

### 2.3 图神经网络推荐

- **LightGCN**：简化的图卷积
- **PinSage**：Pinterest的大规模GNN推荐

### 2.4 多任务学习

同时优化多个目标：点击率、转化率、停留时长等。

---

## 三、大规模推荐系统

### 3.1 召回-排序两阶段

```
候选集(百万级)
    ↓ 召回(多路召回，快速筛选)
候选集(千级)
    ↓ 粗排(简单模型)
候选集(百级)
    ↓ 精排(复杂模型)
    ↓ 重排(多样性、去重)
推荐结果
```

### 3.2 特征工程

- 用户特征：画像、历史行为
- 物品特征：属性、统计量
- 交叉特征：用户-物品交互
- 上下文特征：时间、位置、设备

### 3.3 工程优化

- 特征存储：Redis、特征库
- 模型服务：TensorFlow Serving
- 实时更新：在线学习
- A/B测试：效果验证
