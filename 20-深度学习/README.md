### 路径 A：系统学习（推荐）

适合从零开始系统掌握深度学习。

```
01_数学基础  ──▶  02_机器学习基础  ──▶  03_神经网络核心
        │                                              │
        │                              ┌───────────────┼───────────────┐
        │                              ▼               ▼               ▼
        │                   04_CV (视觉)   05_NLP (文本)   06_GenAI (生成)
        │                                              │
        │                              ┌───────────────┘
        │                              ▼
        └──▶  07_RL (强化学习)  ──▶  08_Engineering (工程部署)
```

### 路径 B：按方向选择

| 目标方向 | 必修目录 | 选修目录 |
|---------|---------|---------|
| 计算机视觉工程师 | 01 → 02 → 03 → **04** | 06（图像生成）、08（部署） |
