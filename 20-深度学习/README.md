```
        │                              ┌───────────────┼───────────────┐
        │                              ▼               ▼               ▼
        │                   04_CV (视觉)   05_NLP (文本)   06_GenAI (生成)
        │                                              │
        │                              ┌───────────────┘
        │                              ▼
        └──▶  07_RL (强化学习)  ──▶  08_Engineering (工程部署)
```

### 路径 B：按方向选择

| 目标方向 | 必修目录 | 选修目录 |
