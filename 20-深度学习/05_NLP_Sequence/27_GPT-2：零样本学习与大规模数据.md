# 27_GPT-2：零样本学习与大规模数据

## 核心概念

- **GPT-2**：OpenAI 于 2019 年发布，是 GPT 的升级版。最大版本 1.5B 参数，展现了对大规模数据和模型规模的扩展效果。
- **零样本学习 (Zero-shot Learning)**：GPT-2 展示了"未经特定任务微调，直接在测试任务上表现良好"的能力。模型可以仅通过"阅读"任务描述和格式就能执行任务。
- **WebText 数据集**：GPT-2 使用从 Reddit 上爬取的外链文章作为训练数据，经过精心筛选，获得约 40GB 的高质量文本。强调数据质量的重要性。
- **语言模型的"涌现能力"**：GPT-2 首次展示随着模型规模增大，语言模型开始表现出一些"意外"的能力——如基础问答、翻译、摘要等，是"涌现能力"概念的早期证据。
- **无监督多任务学习**：GPT-2 的目标不是针对单一任务优化，而是训练一个通用的语言模型，使其隐式地学习多种任务（翻译、QA、摘要等）的模式。
- **参数设置**：GPT-2 使用 48 层 Transformer、1600 维隐藏状态、25 个注意力头的配置。LayerNorm 移动到每个子层输入之前（Pre-Norm），提升训练稳定性。
- **"零样本"并非真正零样本**：虽然 GPT-2 没有针对下游任务微调，但其训练数据中不可避免包含了这些任务的示例（如双语网站、新闻问答），因此性能提升部分来自"记忆"。

## 数学推导

GPT-2 的零样本任务执行方式：将所有任务转化为条件概率最大化问题。

例如，翻译任务（法→英）：
$$
P_{\text{GPT-2}}(\text{english} | \text{"Translate French to English: Je suis étudiant →"})
$$

模型通过最大化上述条件概率，自回归生成英文翻译。

阅读理解任务：
$$
P_{\text{GPT-2}}(\text{answer} | \text{context + question + "答："})
$$

模型在给定上下文和问题时，生成最可能的回答。

## 直观理解

- **GPT-2 像"博览群书的通才"**：虽然没有专门训练做翻译、写摘要，但它读过的海量文本中包含了各种任务模式。就像一个人读过了无数中英对照的句子，看到"Translate to English"这种提示时，自然知道要输出英文。
- **零样本学习像"看一遍就会"**：没有专门训练过做单选题，但只要给出"问题：... 答案："的格式，GPT-2 就能理解这个模式并续写答案——就像你第一次看到新题型，看完说明就能做。
- **涌现能力像"熟读唐诗三百首"**：并没有人专门教 GPT-2 写诗，但读过的文本中包含诗歌，所以它也能"仿写"出类似诗歌的文本。量变引起质变。

## 代码示例

```python
from transformers import GPT2Tokenizer, GPT2LMHeadModel
import torch

# 加载 GPT-2 大版本
tokenizer = GPT2Tokenizer.from_pretrained('gpt2-xl')  # 1.5B 参数
model = GPT2LMHeadModel.from_pretrained('gpt2-xl')

# 零样本翻译
prompt = "Translate French to English: Je suis étudiant →"
inputs = tokenizer(prompt, return_tensors='pt')
with torch.no_grad():
    outputs = model.generate(
        inputs['input_ids'],
        max_new_tokens=20,
        pad_token_id=tokenizer.eos_token_id
    )
print("翻译结果:", tokenizer.decode(outputs[0], skip_special_tokens=True))

# 零样本问答
qa_prompt = (
    "Question: What is the capital of France?\n"
    "Answer:"
)
inputs = tokenizer(qa_prompt, return_tensors='pt')
with torch.no_grad():
    outputs = model.generate(
        inputs['input_ids'],
        max_new_tokens=10,
        do_sample=True,
        pad_token_id=tokenizer.eos_token_id
    )
print("\n问答结果:", tokenizer.decode(outputs[0], skip_special_tokens=True))
```

## 深度学习关联

- **零样本/少样本范式的奠基**：GPT-2 展示了 LLM 可以通过 prompt 格式直接执行未经微调的任务，为 GPT-3 的 in-context learning、ChatGPT 的对话式交互奠定了基础。
- **数据质量 vs 数据量**：GPT-2 的 WebText 强调了数据质量（Reddit 外链的社交媒体过滤），为后续训练数据构建提供了参考。LLaMA 等现代模型也继承了"选取高质量数据"的理念。
- **能力涌现的观察起点**：GPT-2 是"能力涌现"（emergent abilities）现象的最早观察者之一。随着模型规模增长，新能力并非平滑出现，而是在某个阈值后突然出现，这一现象至今仍是研究热点。
