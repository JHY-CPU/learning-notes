# 5_多视图3D重建

## 1. 多视图3D重建概述

多视图3D重建是从**多张不同视角拍摄的2D图像**中恢复三维场景结构的技术。它是计算机视觉中最经典的问题之一。

### 1.1 流水线

```
多张图像 → 特征提取与匹配 → 相机位姿估计(SfM) → 稠密重建(MVS) → 点云/网格
```

## 2. 运动恢复结构 (SfM)

### 2.1 Structure from Motion 基本流程

SfM 从无序图像集合中同时恢复**相机参数**和**3D结构**：

1. **特征提取**：SIFT / SuperPoint
2. **特征匹配**：最近邻 + 比率测试
3. **估计基础矩阵** $F$ / 本质矩阵 $E$
4. **三角化**初始3D点
5. **增量式/全局式**添加更多视图
6. **Bundle Adjustment** 光束法平差优化

### 2.2 特征匹配与几何验证

```python
import cv2
import numpy as np

def estimate_essential_matrix(kpts1, kpts2, K):
    """
    从匹配点估计本质矩阵
    kpts1, kpts2: 匹配的关键点 (N, 2)
    K: 相机内参矩阵 (3, 3)
    """
    # 计算基础矩阵
    F, mask = cv2.findFundamentalMat(kpts1, kpts2, cv2.FM_RANSAC)
    
    # 本质矩阵 E = K2^T @ F @ K1
    E = K.T @ F @ K
    
    # 分解本质矩阵得到R, t
    _, R, t, mask_pose = cv2.recoverPose(E, kpts1, kpts2, K)
    
    return R, t, mask_pose
```

### 2.3 三角化

已知两个相机的投影矩阵 $P_1 = K[I|0]$, $P_2 = K[R|t]$，以及对应点 $\mathbf{x}_1, \mathbf{x}_2$：

$$\begin{bmatrix} x_1 p_3^T - p_1^T \\ y_1 p_3^T - p_2^T \\ x_2 p_3'^T - p_1'^T \\ y_2 p_3'^T - p_2'^T \end{bmatrix} \mathbf{X} = \mathbf{0}$$

通过SVD求解。

```python
def triangulate_points(P1, P2, pts1, pts2):
    """DLT三角化"""
    pts4d = cv2.triangulatePoints(P1, P2, pts1.T, pts2.T)  # (4, N)
    pts3d = (pts4d[:3] / pts4d[3:]).T  # (N, 3)
    return pts3d
```

### 2.4 Bundle Adjustment

Bundle Adjustment (BA) 是SfM的核心优化步骤，同时优化所有相机参数和3D点坐标：

$$\min_{\{R_i, t_i\}, \{\mathbf{X}_j\}} \sum_{i,j} \rho\left(\|\mathbf{x}_{ij} - \pi(R_i \mathbf{X}_j + t_i)\|^2\right)$$

其中：
- $\pi$：投影函数
- $\rho$：鲁棒核函数（如 Huber loss）
- $\mathbf{x}_{ij}$：第 $j$ 个3D点在第 $i$ 个相机中的观测

```python
from scipy.optimize import least_squares

    """
```

### 2.5 增量式 vs 全局式SfM

| 特性 | 增量式 (COLMAP) | 全局式 |
|------|----------------|--------|
| 精度 | 高（逐步优化） | 较低（一步到位） |
| 速度 | 慢（N次BA） | 快（一次BA） |
| 漂移 | 存在 | 不存在 |
| 鲁棒性 | 高（逐步过滤外点） | 中 |
| 代表工具 | COLMAP, VisualSFM | Theia |

## 3. 多视图立体匹配 (MVS)

### 3.1 MVS 目标

在已知相机位姿后，MVS 的目标是**稠密重建**——为每个像素估计深度，生成稠密点云。

### 3.2 传统MVS分类

| 方法 | 思路 | 代表 |
|------|------|------|
| 体素法 | 将空间离散化为体素投票 | Space Carving |
| 深度图融合 | 每张图估计深度图后融合 | COLMAP MVS |
| 散布法 | 逐步扩散匹配 | PMVS |

### 3.3 PatchMatch MVS

COLMAP 使用的 PatchMatch 算法：

1. 随机初始化深度和法线
2. 空间传播：相邻像素传播好的假设
3. 视图传播：来自其他视图的假设
4. 随机扰动：避免局部最优
5. 迭代优化直到收敛

```python
def patchmatch_stereo(img_ref, imgs_src, poses, K, depth_range):

    H, W = img_ref.shape[:2]

                cost = compute_ncc(img_ref, imgs_src, x, y, d, n, poses, K)
                
                # 空间传播
                for dy, dx in [(0,-1), (0,1), (-1,0), (1,0)]:
                    ny, nx = y+dy, x+dx
                    if 0 <= ny < H and 0 <= nx < W:
                        neighbor_cost = evaluate(depth_map[ny,nx], normal_map[ny,nx])
                        if neighbor_cost < cost:
                            depth_map[y,x] = depth_map[ny,nx]
                            normal_map[y,x] = normal_map[ny,nx]
                            cost = neighbor_cost
                
                # 随机扰动
                d_new = d + np.random.uniform(-0.5, 0.5)
                if evaluate(d_new, n) < cost:
                    depth_map[y, x] = d_new
    
    return depth_map
```

## 4. 深度学习 MVS

### 4.1 MVSNet (Yao et al., 2019)

MVSNet 将MVS问题端到端地用神经网络解决：

```
输入: 参考图 + N张源图 + 相机位姿
    ↓
特征提取 (共享CNN)
    ↓
可微单应性变换 (Differentiable Homography)
    ↓
构建3D代价体
    ↓
3D CNN正则化
    ↓
深度回归 (Soft Argmin)
    ↓
深度图
```

### 4.2 可微单应性变换

关键创新：将不同视角的特征图通过**单应性变换** warp 到参考相机的前方平行平面：

$$H_i(d) = K_i R_i \left(I - \frac{(t_1 - t_i)\mathbf{n}_1^T}{d}\right) R_1^T K_1^{-1}$$

其中 $d$ 是假设的深度，$\mathbf{n}_1$ 是参考相机的主轴方向。

```python
def differentiable_homography(src_feat, depth, pose_src, pose_ref, K):
```

### 4.3 级联MVSNet

CasMVSNet (Gu et al., 2020) 使用**由粗到细**的深度假设策略：

```
粗阶段: D=8个深度假设 → 粗略深度图
  ↓ 上采样 + 缩小范围
中阶段: D=16个深度假设 → 中等深度图
  ↓ 上采样 + 缩小范围  
细阶段: D=32个深度假设 → 精细深度图
```

总假设数：$8 + 16 + 32 = 56$，远少于单阶段的 256 个。

## 5. 端到端3D重建

### 5.1 点云融合

多张深度图融合为统一的点云：

```python
def fuse_depth_maps(depth_maps, poses, K):
    """将多张深度图融合为点云"""
    all_points = []
    for depth, pose in zip(depth_maps, poses):
        H, W = depth.shape
        # 像素坐标 → 相机坐标 → 世界坐标
        v, u = np.mgrid[0:H, 0:W].reshape(2, -1)
        z = depth.flatten()
        valid = z > 0
        
        # 反投影
        x = (u[valid] - K[0,2]) * z[valid] / K[0,0]
        y = (v[valid] - K[1,2]) * z[valid] / K[1,1]
        pts_cam = np.stack([x, y, z[valid], np.ones_like(x)])
        
        # 变换到世界坐标
        pts_world = np.linalg.inv(pose) @ pts_cam
        all_points.append(pts_world[:3].T)
    
    return np.concatenate(all_points, axis=0)
```

### 5.2 泊松重建 (Poisson Surface Reconstruction)

从带法线的点云重建网格表面：

$$\Delta \chi = \nabla \cdot \vec{V}$$

其中 $\vec{V}$ 是由点云法线定义的向量场，$\chi$ 是指示函数。

## 7. 总结

| 阶段 | 任务 | 输入 | 输出 |
|------|------|------|------|
| SfM | 稀疏重建 | 无序图像 | 相机位姿 + 稀疏点云 |
| MVS | 稠密重建 | 图像 + 位姿 | 稠密深度图/点云 |
| 融合 | 表面重建 | 点云 | 网格模型 |

---

**关键要点**：
1. SfM通过特征匹配、三角化和Bundle Adjustment从无序图像恢复稀疏3D结构
2. MVS在已知位姿基础上进行稠密匹配，生成密集点云
3. MVSNet等深度学习方法实现了端到端深度估计，速度更快
4. COLMAP 是目前最成熟、最常用的开源重建工具
