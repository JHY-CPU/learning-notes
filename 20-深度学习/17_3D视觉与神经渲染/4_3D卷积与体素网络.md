# 4_3D卷积与体素网络

## 1. 体素化基础

体素 (Voxel) 是 "Volumetric Pixel" 的缩写，将3D空间均匀划分为规则网格，是2D卷积在3D的自然推广。

### 1.1 体素化过程

```python
import torch
import numpy as np

def voxelization(points, voxel_size, pc_range, max_points=35, max_voxels=20000):
    """
    将点云转换为体素
    points: (N, 3+C) C维附加特征
    voxel_size: (vx, vy, vz)
    pc_range: (xmin, ymin, zmin, xmax, ymax, zmax)
    返回: voxels (M, max_points, 3+C), coords (M, 3), num_points (M,)
    """
    # 计算每个点的体素坐标
    coords = ((points[:, :3] - pc_range[:3]) / voxel_size).long()
    
    # 过滤范围外的点
    mask = (coords >= 0).all(dim=1) & \
           (coords < torch.tensor(pc_range[3:]) // voxel_size).all(dim=1)
    coords = coords[mask]
    points = points[mask]
    
    # 按体素坐标分组
    voxel_grid = {}
    for i in range(len(coords)):
        key = tuple(coords[i].tolist())
        if key not in voxel_grid:
            voxel_grid[key] = []
        if len(voxel_grid[key]) < max_points:
            voxel_grid[key].append(points[i])
    
    # 填充输出张量
    M = min(len(voxel_grid), max_voxels)
    feat_dim = points.shape[-1]
    voxels = torch.zeros(M, max_points, feat_dim)
    coords_out = torch.zeros(M, 3, dtype=torch.long)
    num_points = torch.zeros(M, dtype=torch.long)
    
    for idx, (key, pts) in enumerate(voxel_grid.items()):
        if idx >= max_voxels:
            break
        pts = torch.stack(pts)
        voxels[idx, :len(pts)] = pts
        coords_out[idx] = torch.tensor(key)
        num_points[idx] = len(pts)
    
    return voxels, coords_out, num_points
```

## 2. 3D 卷积

### 2.1 标准3D卷积

$$y_{d,h,w} = \sum_{k_d,k_h,k_w} W_{k_d,k_h,k_w} \cdot x_{d+k_d, h+k_h, w+k_w}$$

- 输入：$(B, C_{in}, D, H, W)$
- 卷积核：$(C_{out}, C_{in}, K_d, K_h, K_w)$
- 输出：$(B, C_{out}, D', H', W')$

**计算量**：$O(C_{in} \cdot C_{out} \cdot K^3 \cdot D' \cdot H' \cdot W')$，相比2D卷积增加了一个维度。

### 2.2 稀疏3D卷积

实际3D数据（如点云体素化后）非常稀疏（90%+的体素为空）。标准3D卷积会浪费大量计算在空体素上。

**稀疏卷积**只对非空体素进行计算。

## 3. VoxelNet

### 3.1 架构

VoxelNet (Zhou & Tung, 2018) 是首个端到端的体素化3D检测网络：

```
点云 → 体素化 → VFE (逐体素特征提取) → 3D卷积 → 2D鸟瞰投影 → RPN → 检测框
```

### 3.2 Voxel Feature Encoding (VFE)

```python
class VFELayer(nn.Module):
    """体素特征编码层"""
    def __init__(self, in_channels, out_channels):

        self.bn = nn.BatchNorm1d(out_channels)

        # 局部最大池化（每个体素内）
        voxel_max = torch.max(point_feat, dim=1, keepdim=True)[0]

        concat = torch.cat([point_feat, voxel_max.expand_as(point_feat)], dim=-1)
```

## 4. 稀疏卷积 (Sparse Convolution)

### 4.1 原理

标准3D卷积在所有位置计算，但稀疏输入中大部分位置为零。**稀疏卷积** (Graham, 2015) 只在非零位置及其邻域计算。

```
标准3D卷积: 计算所有 D×H×W 个位置
稀疏卷积: 只计算 M 个非空体素的位置
```

### 4.2 MinkowskiEngine 实现

```python
import MinkowskiEngine as ME

class SparseConvNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.network = nn.Sequential(
            ME.MinkowskiConvolution(3, 32, 3, stride=1, dimension=3),
            ME.MinkowskiBatchNorm(32),
            ME.MinkowskiReLU(),
            ME.MinkowskiConvolution(32, 64, 3, stride=2, dimension=3),
            ME.MinkowskiBatchNorm(64),
            ME.MinkowskiReLU(),
            ME.MinkowskiConvolution(64, 128, 3, stride=2, dimension=3),
            ME.MinkowskiBatchNorm(128),
            ME.MinkowskiReLU(),
            ME.MinkowskiGlobalPooling(),  # 全局池化
        )
        self.cls = ME.MinkowskiLinear(128, 40)
    
    def forward(self, coords, feats):
        # coords: (N, 4) [batch_id, x, y, z]
        # feats: (N, C) 点的特征
        sparse_tensor = ME.SparseTensor(feats, coords)
        out = self.network(sparse_tensor)
        return self.cls(out)
```

### 4.3 稀疏卷积的规则

稀疏卷积的输出位置遵循**规则卷积规则**：只有当一个位置的所有输入邻域都非空时才计算输出。

**子流形稀疏卷积 (Submanifold)**：输出位置 = 输入位置，不扩大稀疏模式，适合特征提取。

## 5. 稀疏卷积网络 (SparseConvNet / MinkowskiNet)

### 5.1 MinkowskiNet 架构

```
稀疏输入体素
    ↓
SparseConvBlock ×N (逐层下采样)
    ↓
全局池化
    ↓
分类头 / 检测头
```

### 5.2 与标准3D卷积对比

| 特性 | 标准3D卷积 | 稀疏卷积 |
|------|-----------|----------|
| 计算量 | $O(D \cdot H \cdot W)$ | $O(M)$，M为非空体素数 |
| 内存 | 存储全部位置 | 只存储非空位置 |
| 速度 | 慢（稀疏率>90%时浪费严重） | 快（10-100倍加速） |
| 实现 | 标准PyTorch | 需专用库（MinkowskiEngine） |

## 7. 实用技巧

### 7.1 动态体素化

固定体素数量会导致截断。使用动态体素化，根据实际点数生成体素：

```python
from torch_scatter import scatter_mean

def dynamic_voxelization(points, voxel_size):
    coords = (points[:, :3] / voxel_size).long()
    # 唯一体素索引
    unique_coords, inverse = torch.unique(coords, dim=0, return_inverse=True)
    # 每个体素的平均特征
    voxel_feats = scatter_mean(points, inverse, dim=0)
    return unique_coords, voxel_feats
```

### 7.2 数据增强

- **随机翻转**：沿X/Y轴随机翻转
- **全局旋转**：$[-\pi/4, \pi/4]$ 随机旋转
- **全局缩放**：$[0.95, 1.05]$ 随机缩放
- **随机平移**：$[0.2, 0.2, 0.2]$ 范围随机平移

---

**关键要点**：
1. 体素化将不规则点云转为规则3D网格，可直接使用3D卷积
2. 稀疏卷积只计算非空体素，是实际应用中的标配
3. VFE层通过逐体素的PointNet聚合局部点特征
4. SECOND结合体素化和稀疏卷积，成为3D检测的经典基线
