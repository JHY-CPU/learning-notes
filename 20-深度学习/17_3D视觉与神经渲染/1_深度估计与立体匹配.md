# 1_深度估计与立体匹配

## 1. 深度估计的分类

深度估计是从图像中恢复场景深度信息的核心任务。

### 1.1 按输入分类

| 方法 | 输入 | 输出 | 代表模型 |
|------|------|------|----------|
| 单目深度估计 | 单张RGB图像 | 深度图 | MiDaS, DPT, MonoDepth |
| 双目立体匹配 | 左右图像对 | 视差图/深度图 | PSMNet, RAFT-Stereo |
| 多视图深度估计 | 多张图像+位姿 | 深度图/点云 | MVSNet, CasMVSNet |
| RGB-D融合 | RGB + 粗略深度 | 精细深度 | GuideDepth |

### 1.2 按输出类型

- **绝对深度**：预测真实距离（单位：米），适用于度量SLAM、3D重建
- **相对深度**：预测深度的相对顺序，适用于编辑、景深效果
- **仿射不变深度**：$d' = \alpha d + \beta$，适合跨场景通用

## 2. 双目立体视觉

### 2.1 极线约束

在双目系统中，左图中的一点 $\mathbf{p}_L$ 在右图中的对应点 $\mathbf{p}_R$ 一定落在**极线**上，将2D搜索降为1D。

极线约束方程：

$$\mathbf{p}_R^T F \mathbf{p}_L = 0$$

其中 $F$ 是基础矩阵（Fundamental Matrix）。

### 2.2 视差与深度的关系

对于校正后的双目图像，深度 $Z$ 与视差 $d$ 的关系：

$$Z = \frac{f \cdot B}{d}$$

- $f$：焦距（像素单位）
- $B$：基线长度（两个相机光心的距离）
- $d = x_L - x_R$：视差

**关键观察**：深度与视差成**反比**关系。近处物体视差大，远处物体视差小。

### 2.3 立体匹配的四个步骤

1. **代价计算 (Cost Computation)**：计算左右图特征的匹配代价
2. **代价聚合 (Cost Aggregation)**：在局部窗口内聚合代价
3. **视差计算 (Disparity Computation)**：选择最优视差（winner-take-all）
4. **视差优化 (Disparity Refinement)**：亚像素优化、左右一致性检查

```python
import torch
import torch.nn.functional as F

def compute_cost_volume(left_feat, right_feat, max_disp):
    """
    构建4D代价体
    left_feat, right_feat: (B, C, H, W)
    返回: (B, 2C, D, H, W) 其中 D = max_disp // 4
    """
    B, C, H, W = left_feat.shape
    cost_volume = torch.zeros(B, 2 * C, max_disp // 4, H, W).to(left_feat.device)
    for d in range(max_disp // 4):
        if d > 0:
            cost_volume[:, :C, d, :, d:] = left_feat[:, :, :, d:]
            cost_volume[:, C:, d, :, d:] = right_feat[:, :, :, :-d]
        else:
            cost_volume[:, :C, d, :, :] = left_feat
            cost_volume[:, C:, d, :, :] = right_feat
    return cost_volume
```

## 3. 深度学习立体匹配

### 3.1 PSMNet (Pyramid Stereo Matching Network, 2018)

核心思想：空间金字塔池化 (SPP) + 4D代价体 + 3D CNN

```
输入左右图像
    ↓
CNN特征提取 (共享权重)
    ↓
SPP模块 (多尺度上下文)
    ↓
构建4D代价体 (B, C, D, H, W)
    ↓
3D CNN编码-解码
    ↓
Soft Argmin → 视差图
```

**Soft Argmin**：将离散的视差选择变为连续可微操作

$$\hat{d} = \sum_{d=0}^{D_{max}} d \cdot \sigma(-c_d)$$

其中 $c_d$ 是视差为 $d$ 时的代价，$\sigma$ 是softmax。

### 3.2 RAFT-Stereo (2021)

借鉴光流估计的RAFT框架，通过迭代优化提升精度：

```python
# RAFT-Stereo 迭代更新伪代码
def raft_stereo_update(corr_volume, flow, hidden):
```

## 4. 单目深度估计

### 4.1 挑战

单目深度估计是一个**病态问题**：一张2D图像对应无穷多个可能的3D场景。需要依赖强大的先验知识。

### 4.2 MiDaS (2019-2023)

MiDaS 通过混合数据集训练实现了**零样本**相对深度估计。

**核心创新**：
1. **混合训练**：同时在多个深度数据集上训练（有监督+自监督）
2. **仿射不变损失**：对每个样本的尺度和偏移进行对齐

$$\mathcal{L}_{aff} = \min_{s, t} \|s \cdot \hat{d} + t - d_{gt}\|^2$$

```python
import torch

def affine_invariant_loss(pred, target):
    """仿射不变损失"""
    # 最小二乘拟合 s*pred + t ≈ target
    pred_flat = pred.reshape(pred.shape[0], -1)
    target_flat = target.reshape(target.shape[0], -1)
    
    # 解析求解最优 s, t
    mean_pred = pred_flat.mean(dim=1, keepdim=True)
    mean_target = target_flat.mean(dim=1, keepdim=True)
    
    cov = ((pred_flat - mean_pred) * (target_flat - mean_target)).mean(dim=1)
    var = ((pred_flat - mean_pred) ** 2).mean(dim=1)
    s = cov / (var + 1e-8)
    t = mean_target.squeeze() - s * mean_pred.squeeze()
    
    aligned = s.unsqueeze(-1) * pred_flat + t.unsqueeze(-1)
    return F.mse_loss(aligned, target_flat)
```

### 4.3 DPT (Dense Prediction Transformer, 2021)

使用 Vision Transformer 作为编码器，结合多尺度解码器：

```
ViT编码器 (预训练)
  ↓
多尺度特征 (4个层级)
  ↓
融合模块 (Progressive Fusion)
  ↓
预测头 → 深度图
```

### 4.4 Metric3D (2023-2024)

从相对深度扩展到**度量深度**（绝对深度），核心思路：

1. **Canonical Camera Transform**：将不同内参的图像变换到标准相机
2. **大规模数据集训练**：使用超过1000万张图像
3. **零样本泛化**：在未见过的场景中也能预测绝对深度

## 5. 深度估计损失函数

### 5.1 常用损失

```python
# 1. Scale-Invariant Loss (Eigen et al., 2014)
def scale_invariant_loss(pred, target):
    diff = torch.log(pred + 1e-8) - torch.log(target + 1e-8)
    return torch.mean(diff ** 2) - 0.5 * (torch.mean(diff)) ** 2

# 2. 梯度损失 (增强边缘)
def gradient_loss(pred, target):
    # 水平和垂直方向的梯度差
    diff_h = pred[:, :, :, 1:] - pred[:, :, :, :-1]
    diff_v = pred[:, :, 1:, :] - pred[:, :, :-1, :]
    gt_h = target[:, :, :, 1:] - target[:, :, :, :-1]
    gt_v = target[:, :, 1:, :] - target[:, :, :-1, :]
    return F.l1_loss(diff_h, gt_h) + F.l1_loss(diff_v, gt_v)

# 3. BerHu损失 (L1和L2的自适应混合)
def berhu_loss(pred, target, c=0.2):
    diff = torch.abs(pred - target)
    c_val = c * diff.max().detach()
    l2_part = (diff ** 2 + c_val ** 2) / (2 * c_val + 1e-8)
    return torch.where(diff <= c_val, diff, l2_part).mean()
```

### 5.2 损失对比

| 损失函数 | 特点 | 适用场景 |
|----------|------|----------|
| L1 | 鲁棒但梯度恒定 | 基础监督 |
| BerHu | 自适应L1/L2 | 深度回归 |
| Scale-Invariant | 不依赖绝对尺度 | 相对深度 |
| 仿射不变 | 尺度偏移对齐 | 零样本泛化 |
| 梯度损失 | 保留边缘 | 边缘锐利 |

## 6. 实践：使用MiDaS估计深度

```python
import torch
import cv2
import numpy as np

# 加载MiDaS模型
model_type = "DPT_Large"  # 或 MiDaS_small
midas = torch.hub.load("intel-isl/MiDaS", model_type)
midas.eval()

# 预处理
midas_transforms = torch.hub.load("intel-isl/MiDaS", "transforms")
transform = midas_transforms.dpt_transform

# 推理
img = cv2.imread("scene.jpg")
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
input_batch = transform(img_rgb)

with torch.no_grad():
    prediction = midas(input_batch)
    prediction = torch.nn.functional.interpolate(
        prediction.unsqueeze(1),
        size=img.shape[:2],
        mode="bicubic",
        align_corners=False
    ).squeeze()

depth_map = prediction.cpu().numpy()
# MiDaS输出的是逆深度（disparity），深度越大 = 物体越近
```

## 7. 评估指标

| 指标 | 公式 | 说明 |
|------|------|------|
| AbsRel | $\frac{1}{n}\sum\|\hat{d}-d\|/d$ | 相对误差 |
| SqRel | $\frac{1}{n}\sum(\hat{d}-d)^2/d$ | 平方相对误差 |
| RMSE | $\sqrt{\frac{1}{n}\sum(\hat{d}-d)^2}$ | 均方根误差 |
| $\delta < 1.25$ | $\max(\hat{d}/d, d/\hat{d}) < 1.25$ 的比例 | 阈值准确率 |

---

**关键要点**：
1. 双目立体匹配通过视差-深度几何关系精确计算深度，但需要多相机
2. 单目深度估计虽然病态，但深度学习通过强大先验实现了实用化
3. MiDaS的仿射不变损失是零样本深度估计的关键
4. 从相对深度到度量深度是当前研究热点
