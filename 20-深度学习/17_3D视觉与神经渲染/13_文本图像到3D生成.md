# 13_文本/图像到3D生成

## 1. 文本到3D生成概述

文本/图像到3D生成是3D视觉最前沿的研究方向，目标是从2D描述自动生成高质量3D内容。

### 1.1 技术路线

| 路线 | 方法 | 代表 |
|------|------|------|
| SDS优化 | 用2D扩散引导3D | DreamFusion, Magic3D |
| 直接生成 | 前馈网络直接出3D | Point-E, Shap-E |
| 视图扩展 | 从单图生成多视图 → 重建 | Zero-1-to-3, Wonder3D |
| 3D扩散 | 直接在3D空间扩散 | 3DShape2VecSet, DiT-3D |

## 2. Magic3D

### 2.1 粗到细生成

**Magic3D (Lin et al., CVPR 2023)** 采用两阶段策略解决DreamFusion分辨率低、速度慢的问题：

```
第一阶段 (低分辨率):
  文本 → SDS引导 → 粗糙3D网格 (低分辨率NeRF, 64²)
    ↓
提取网格 (Marching Cubes)
    ↓
第二阶段 (高分辨率):
  粗网格 → UV纹理初始化 → SDS引导 → 高分辨率纹理和法线 (512²)
```

### 2.2 关键创新

1. **显式网格引导**：第一阶段提取的网格提供几何约束
2. **Latent Diffusion**：使用Latent Diffusion而非Pixel Diffusion，加速10倍
3. **方向相关提示**：自动从文本生成方向描述

```python
def magic3d_coarse_to_fine(text_prompt):
    """Magic3D两阶段"""
    # 第一阶段：粗几何
    nerf = train_coarse_stage(
        prompt=text_prompt,
        resolution=64,
        num_steps=5000,
        guidance="SDS"  # Score Distillation Sampling
    )
    
    # 提取网格
    mesh = extract_mesh(nerf, resolution=128)  # Marching Cubes
    mesh = simplify_mesh(mesh, target_faces=50000)
    
    # 第二阶段：精细纹理和法线
    refined_mesh = train_fine_stage(
        mesh=mesh,
        prompt=text_prompt,
        resolution=512,
        num_steps=5000,
        guidance="SDS"
    )
    
    return refined_mesh
```

### 2.3 性能对比

| 方法 | 质量 (CLIP R-precision) | 速度 | 分辨率 |
|------|------------------------|------|--------|
| DreamFusion | 51.2% | ~2小时 | 64² |
| Magic3D | 61.2% | ~40分钟 | 512² |

## 3. Point-E

### 3.1 前馈生成

**Point-E (Nichol et al., 2022)** 是OpenAI提出的**前馈3D生成**方法，不需要逐样本优化：

```
文本 → 文本到图像扩散模型 → 2D图像
         ↓
    图像到点云扩散模型 → 3D点云 (1024点)
         ↓
    点云到网格 (可选) → 网格
```

### 3.2 两个扩散模型

1. **文本到图像**：使用预训练的 GLIDE 模型
2. **图像到点云**：专门训练的点云扩散模型

```python
class PointCloudDiffusion(nn.Module):
    """点云扩散模型"""
    def __init__(self, num_points=1024, num_timesteps=1000):
        super().__init__()
        self.num_points = num_points
        
        # 1D CNN处理点序列
        self.time_embed = SinusoidalPositionEmbed(256)
        self.conv_in = nn.Conv1d(3, 256, 1)
        
        self.blocks = nn.ModuleList([
            AttentionBlock(256) for _ in range(12)
        ])
        
        self.conv_out = nn.Conv1d(256, 3, 1)
    
    def forward(self, x, t, image_cond):
        """
        x: (B, N, 3) 噪声点云
        t: (B,) 时间步
        image_cond: (B, D) 图像特征
        """
        h = self.conv_in(x.transpose(1, 2))  # (B, 256, N)
        t_emb = self.time_embed(t)  # (B, 256)
        
        for block in self.blocks:
            h = block(h, t_emb, image_cond)
        
        noise_pred = self.conv_out(h).transpose(1, 2)  # (B, N, 3)
        return noise_pred
```

### 3.3 优势与局限

- **优势**：速度极快（~2分钟生成3D模型）
- **局限**：点云质量有限（1024点）、细节不够

## 4. Shap-E

### 4.1 隐式表示生成

**Shap-E (Jun & Nichol, 2023)** 改进了Point-E，直接生成**隐式表示**（NeRF/SDF）：

```
文本/图像 → 编码器 → 隐式函数参数 → 渲染/提取网格
```

### 4.2 编码器-解码器架构

- **编码器**：Transformer，将文本/图像编码为隐变量
- **解码器**：小型MLP，将隐变量转为隐式函数参数
- **训练**：对比学习 + 扩散模型

```python
class ShapE(nn.Module):
    def __init__(self):
        super().__init__()
        # 文本编码器
        self.text_encoder = TextTransformer(
            vocab_size=50000,
            hidden_dim=1024,
            num_layers=6
        )
        
        # 扩散解码器（生成隐式函数参数）
        self.diffusion_decoder = ConditionalDiffusion(
            input_dim=1024,
            hidden_dim=512,
            num_layers=8
        )
        
        # 隐式函数（NeRF + SDF）
        self.implicit_fn = ShapEImplicitFunction(
            hidden_dim=1024,
            num_layers=4
        )
    
    def generate(self, text_prompt):
        # 编码文本
        text_feat = self.text_encoder(text_prompt)  # (1, 1024)
        
        # 扩散生成
        implicit_params = self.diffusion_decoder.sample(text_feat)
        
        # 解码为隐式函数
        self.implicit_fn.load_params(implicit_params)
        
        return self.implicit_fn
```

### 4.3 与Point-E对比

| 特性 | Point-E | Shap-E |
|------|---------|--------|
| 输出 | 点云 (1024点) | 隐式函数 |
| 分辨率 | 有限 | 无限（连续） |
| 质量 | 中等 | 较好 |
| 速度 | ~2分钟 | ~1分钟 |
| 可微性 | 是 | 是 |

## 5. Zero-1-to-3

### 5.1 单图到3D

**Zero-1-to-3 (Liu et al., ICCV 2023)** 从单张RGB图像生成新视角，然后重建3D：

```
单张输入图像
    ↓
微调的扩散模型
    ↓
生成新视角 (novel views)
    ↓
多视图重建 (NeRF/3DGS)
    ↓
3D模型
```

### 5.2 视图条件扩散模型

在Stable Diffusion基础上，添加**相机位姿条件**：

```python
class ViewConditionedDiffusion(nn.Module):
    """视角条件扩散模型"""
    def __init__(self, pretrained_sd):
        super().__init__()
        # 复用Stable Diffusion的U-Net
        self.unet = pretrained_sd.unet
        
        # 添加相机条件编码
        self.camera_encoder = nn.Sequential(
            nn.Linear(4 + 3 + 3, 256),  # [rot_x, rot_y, rot_z, tx, ty, tz, fov]
            nn.ReLU(),
            nn.Linear(256, 768)  # 匹配CLIP特征维度
        )
        
        # 输入图像的CLIP特征作为条件
        self.clip_encoder = pretrained_sd.clip_image_encoder
    
    def forward(self, noisy_image, timestep, source_image, target_pose):
        """
        noisy_image: 目标视角的噪声图像
        source_image: 输入参考图像
        target_pose: 目标相机位姿
        """
        # 图像条件
        image_cond = self.clip_encoder(source_image)
        
        # 相机条件
        camera_cond = self.camera_encoder(target_pose)
        
        # 合并条件
        cond = image_cond + camera_cond
        
        # 预测噪声
        return self.unet(noisy_image, timestep, encoder_hidden_states=cond)
```

### 5.3 训练数据

使用大规模3D数据集 (Objaverse) 渲染多视角图像对进行训练。

## 6. Wonder3D

### 6.1 跨域扩散

**Wonder3D** 同时生成**法线图**和**彩色图**，利用法线信息提升几何质量：

```
单张图像
    ↓
跨域扩散模型
    ↓
多视角法线图 + 彩色图 (6个视角)
    ↓
深度融合 → 3D网格
```

### 6.2 关键创新

1. **跨域注意力**：法线和RGB在扩散过程中共享信息
2. **几何先验**：法线图提供更明确的几何约束
3. **快速收敛**：5分钟内生成高质量3D
