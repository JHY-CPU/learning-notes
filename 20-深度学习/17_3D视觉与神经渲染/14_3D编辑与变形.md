# 14_3D编辑与变形

## 1. 3D编辑概述

3D编辑是指在不重新训练的前提下，对已有的3D表示（NeRF、3D Gaussian等）进行局部修改、风格迁移、形状变形等操作。

### 1.1 挑战

与2D图像编辑不同，3D编辑需要保证**多视角一致性**：修改必须在所有视角下都合理。

## 2. Instruct-NeRF2NeRF

### 2.1 核心思想

**Instruct-NeRF2NeRF (Haque et al., ICCV 2023)** 使用图像编辑模型（如 InstructPix2Pix）逐视角编辑NeRF：

```
原始NeRF
    ↓
迭代编辑循环:
  1. 采样视角 → 渲染图像
  2. 用 InstructPix2Pix 编辑图像 (如 "让他戴上太阳镜")
  3. 用编辑后的图像微调NeRF
    ↓
编辑后的NeRF
```

### 2.2 编辑顺序

按CLIP相似度排序：先编辑与当前结果最相似的视角（最"正确"的），逐步传播编辑效果。

```python
def instruct_nerf2nerf(nerf, editing_model, instruction, num_steps=2000):
    """Instruct-NeRF2NeRF编辑流程"""
    
    # 获取所有训练视角
    train_poses = nerf.train_poses
    
    for step in range(num_steps):
        # 选择编辑顺序（按与原图的相似度）
        if step % 100 == 0:
            similarities = compute_clip_similarity(nerf, train_poses)
            sorted_indices = torch.argsort(similarities, descending=True)
        
        # 选择当前最相似的视角进行编辑
        edit_idx = sorted_indices[step % len(sorted_indices)]
        
        # 渲染当前视角
        rendered = nerf.render(train_poses[edit_idx])
        
        # 用图像编辑模型编辑
        edited = editing_model(rendered, instruction)
        
        # 用编辑后的图像微调NeRF（仅该视角）
        loss = F.mse_loss(nerf.render(train_poses[edit_idx]), edited)
        loss.backward()
        optimizer.step()
```

### 2.3 优势

- 无需3D编辑数据集
- 自然语言指令驱动
- 可利用强大的2D编辑模型

## 3. Gaussian Splatting 编辑

### 3.1 直接操作高斯

3DGS 的显式表示天然支持直接编辑：

```python
class GaussianEditor:
    """3D Gaussian编辑器"""
    
    def __init__(self, gaussian_model):
        self.model = gaussian_model
    
    def select_by_mask(self, mask_fn):
        """通过3D掩码函数选择高斯"""
        mask = mask_fn(self.model.means.data)
        return mask
    
    def delete(self, mask):
        """删除选中的高斯"""
        keep = ~mask
        self.model.means = nn.Parameter(self.model.means[keep])
        self.model.sh_coeffs = nn.Parameter(self.model.sh_coeffs[keep])
        self.model.scaling = nn.Parameter(self.model.scaling[keep])
        self.model.rotation = nn.Parameter(self.model.rotation[keep])
        self.model.opacity = nn.Parameter(self.model.opacity[keep])
    
    def move(self, mask, translation):
        """移动选中的高斯"""
        self.model.means.data[mask] += translation
    
    def recolor(self, mask, color):
        """重新着色"""
        # 修改0阶球谐系数（平均颜色）
        self.model.sh_coeffs.data[mask, 0, :] = color
    
    def scale(self, mask, scale_factor):
        """缩放"""
        self.model.scaling.data[mask] += torch.log(torch.tensor(scale_factor))
    
    def region_select_3d(self, center, radius):
        """球形区域选择"""
        dist = torch.norm(self.model.means - center, dim=-1)
        return dist < radius
```

### 3.2 基于SAM的3D分割

结合Segment Anything Model (SAM) 实现从2D到3D的分割：

```python
def segment_3d_with_sam(gaussian_model, cameras, sam_model, prompt):
    """用SAM在多个视角分割，投票得到3D分割"""
    votes = torch.zeros(len(gaussian_model.means))
    
    for cam in cameras:
        # 渲染该视角
        rendered, info = gaussian_model.render(cam)
        
        # SAM分割
        mask = sam_model.segment(rendered, prompt)  # 2D mask
        
        # 将2D mask反投影到3D高斯
        # 使用渲染权重（每个像素由哪些高斯贡献）
        gaussian_weights = info['weights']  # (H, W, K) K个高斯的权重
        
        for h in range(mask.shape[0]):
            for w in range(mask.shape[1]):
                if mask[h, w]:
                    gaussian_ids = info['gaussian_ids'][h, w]
                    weights = gaussian_weights[h, w]
                    for gid, w in zip(gaussian_ids, weights):
                        votes[gid] += w
    
    # 阈值化投票结果
    return votes > len(cameras) * 0.3
```

## 4. 文本驱动3D编辑

### 4.1 3D-aware编辑

```python
def text_driven_edit_3d(model, source_text, target_text, edit_strength=0.5):
    """
    文本驱动的3D编辑
    通过CLIP引导方向移动
    """
    # CLIP文本特征
    feat_source = clip_encode_text(source_text)
    feat_target = clip_encode_text(target_text)
    edit_direction = feat_target - feat_source
    
    # 对每个视角进行引导
    for cam in random_cameras(model, n=16):
        rendered = model.render(cam)
        
        # CLIP图像特征
        feat_img = clip_encode_image(rendered)
        
        # 计算引导方向
        similarity_source = F.cosine_similarity(feat_img, feat_source)
        similarity_target = F.cosine_similarity(feat_img, feat_target)
        
        # 损失：增大与目标的相似度
        loss = -similarity_target.mean()
        loss.backward()
    
    optimizer.step()
```

## 5. 形状变形

### 5.1 骨架驱动变形

```python
def skeleton_deformation(gaussian_model, skeleton, pose_params):
    """
    骨架驱动的高斯变形
    skeleton: 骨架定义 (关节位置, 父子关系)
    pose_params: 每个关节的旋转参数
    """
    # 计算每个高斯的骨骼权重（最近关节）
    joint_positions = skeleton.get_joint_positions(pose_params)  # (J, 3)
    
    # 到每个关节的距离
    dists = torch.cdist(gaussian_model.means, joint_positions)  # (N, J)
    weights = F.softmax(-dists / 0.1, dim=-1)  # (N, J) 蒙皮权重
    
    # 骨骼变换矩阵
    bone_transforms = skeleton.get_bone_transforms(pose_params)  # (J, 4, 4)
    
    # 混合变换
    # T = sum(w_j * T_j)
    T = (weights.unsqueeze(-1).unsqueeze(-1) * bone_transforms.unsqueeze(0)).sum(dim=1)
    
    # 应用变换
    means_h = torch.cat([gaussian_model.means, torch.ones(len(gaussian_model.means), 1)], dim=-1)
    deformed_means = (T @ means_h.unsqueeze(-1)).squeeze(-1)[:, :3]
    
    return deformed_means
```

## 6. 风格迁移

### 6.1 3D风格迁移

```python
def style_transfer_3d(model, style_image, content_weight=1.0, style_weight=1e4):
    """3D场景的风格迁移"""
    
    def gram_matrix(features):
        B, C, H, W = features.shape
        features = features.view(B, C, -1)
        return torch.bmm(features, features.transpose(1, 2)) / (C * H * W)
    
    # 提取风格特征
    style_features = vgg_extract(style_image)
    style_grams = [gram_matrix(f) for f in style_features]
    
    for cam in random_cameras(model):
        rendered = model.render(cam)
        content_features = vgg_extract(rendered)
        
        # 内容损失
        content_loss = F.mse_loss(content_features[2], content_features[2].detach())
        
        # 风格损失
        style_loss = 0
        for cf, sg in zip(content_features, style_grams):
            style_loss += F.mse_loss(gram_matrix(cf), sg)
        
        loss = content_weight * content_loss + style_weight * style_loss
        loss.backward()
    
    optimizer.step()
```
