# 0_3D视觉概述

## 1. 什么是3D视觉

3D视觉是计算机视觉的核心分支之一，目标是从2D图像或传感器数据中恢复、理解和生成三维世界的表示。与传统2D视觉不同，3D视觉需要处理**深度信息**、**几何结构**和**空间关系**。

### 1.1 核心任务

| 任务 | 输入 | 输出 | 典型方法 |
|------|------|------|----------|
| 深度估计 | 单张/多张图像 | 深度图 | MiDaS, DPT |
| 3D重建 | 多视图图像 | 点云/网格 | SfM, MVS, NeRF |
| 3D检测 | 点云/体素 | 3D边界框 | PointPillars, VoxelNet |
| 3D生成 | 文本/图像 | 3D模型 | DreamFusion, Shap-E |
| 位姿估计 | 图像 | 相机位姿 | PnP, SLAM |

## 2. 3D表示方式

### 2.1 点云 (Point Cloud)

点云是最直接的3D表示：一组无序的3D坐标点 $(x, y, z)$，可选附带颜色、法线等属性。

$$P = \{p_i = (x_i, y_i, z_i)\}_{i=1}^{N}$$

**优点**：简单、灵活、无需预定义拓扑结构
**缺点**：无表面信息、不规则结构难以处理

```python
import torch

# 点云基本操作
point_cloud = torch.randn(1024, 3)  # (N, 3) N个3D点
# 计算点对距离矩阵
dist = torch.cdist(point_cloud, point_cloud)  # (N, N)
# K近邻
k = 20
_, knn_idx = dist.topk(k, largest=False, dim=-1)  # (N, k)
```

### 2.2 网格 (Mesh)

网格由**顶点** $V \in \mathbb{R}^{N \times 3}$ 和**面** $F \in \mathbb{Z}^{M \times 3}$（三角形索引）组成，显式定义了物体表面。

**优点**：紧凑、有显式表面、支持纹理映射
**缺点**：拓扑固定、难以处理复杂形状变化

### 2.3 体素 (Voxel)

体素是像素在3D空间的推广，将空间均匀划分为 $D \times H \times W$ 的网格，每个格子存储占用概率或特征。

$$V \in \mathbb{R}^{D \times H \times W \times C}$$

**优点**：规则结构、可直接用3D卷积
**缺点**：内存随分辨率立方增长（$O(n^3)$）、稀疏数据浪费严重

```python
# 体素化示例
def points_to_voxel(points, voxel_size, pc_range):
    """
    将点云转换为体素
    points: (N, 3)
    voxel_size: (vx, vy, vz)
    pc_range: (xmin, ymin, zmin, xmax, ymax, zmax)
    """
    mask = torch.all((points >= pc_range[:3]) & (points < pc_range[3:]), dim=-1)
    points = points[mask]
    voxel_idx = ((points - pc_range[:3]) / voxel_size).long()
    return voxel_idx  # 每个点对应的体素索引
```

### 2.4 隐式函数 (Implicit Function)

隐式函数将3D形状编码为一个连续函数 $f: \mathbb{R}^3 \rightarrow \mathbb{R}$，其中：
- $f(\mathbf{x}) = 0$ 定义了物体表面（零等值面）
- $f(\mathbf{x}) < 0$ 表示物体内部
- $f(\mathbf{x}) > 0$ 表示物体外部

**代表方法**：DeepSDF, Occupancy Networks, NeRF

**优点**：分辨率无限、内存紧凑、可微分
**缺点**：推理慢（需查询大量点）、不能直接可视化

## 3. 深度估计基础

### 3.1 深度类型

- **绝对深度 (Metric Depth)**：真实的物理距离，单位为米
- **相对深度 (Relative Depth)**：深度的相对顺序，$d_i < d_j$
- **仿射深度 (Affine Depth)**：$d = \alpha \cdot d_{rel} + \beta$

### 3.2 双目视觉

通过两个已知基线 $b$ 的相机获取视差 $d$，利用三角测量计算深度：

$$Z = \frac{f \cdot b}{d}$$

其中 $f$ 是焦距，$d = x_L - x_R$ 是视差（左右图像对应点的水平偏移）。

### 3.3 单目深度估计

从单张图像推断深度是一个**病态问题 (ill-posed problem)**，因为同一2D图像可能对应无数3D场景。深度学习通过学习数据先验来解决此问题。

## 4. 多视图几何

### 4.1 对极几何 (Epipolar Geometry)

两个视图之间的几何关系由**本质矩阵** $E$ 或**基础矩阵** $F$ 描述：

$$\mathbf{x}_L^T F \mathbf{x}_R = 0$$

其中本质矩阵 $E = K_R^T F K_L$，$K$ 为相机内参矩阵。

### 4.2 本质矩阵分解

$$E = [t]_\times R$$

通过SVD分解 $E$ 可以恢复两个视图之间的旋转 $R$ 和平移 $t$（差一个尺度因子）。

### 4.3 三角化 (Triangulation)

已知两视图的投影矩阵 $P_1, P_2$ 和对应点 $\mathbf{x}_1, \mathbf{x}_2$，求3D点 $\mathbf{X}$：

$$\begin{cases} \mathbf{x}_1 \times (P_1 \mathbf{X}) = 0 \\ \mathbf{x}_2 \times (P_2 \mathbf{X}) = 0 \end{cases}$$

使用DLT (Direct Linear Transform) 求解。

## 6. 常用数据集与工具

### 6.1 数据集

- **ShapeNet**：大规模3D CAD模型数据集（55个类别，约51300个模型）
- **ModelNet**：3D物体分类（ModelNet10/40）
- **ScanNet**：室内场景RGB-D扫描数据
- **KITTI**：自动驾驶场景（图像、点云、3D标注）
- **nuScenes**：大规模自动驾驶数据集

### 6.2 工具库

- **Open3D**：3D数据处理与可视化
- **PyTorch3D**：Facebook的3D深度学习库
- **Kaolin**：NVIDIA的3D深度学习工具包
- **trimesh**：网格处理
- **PyVista**：3D绘图与分析
