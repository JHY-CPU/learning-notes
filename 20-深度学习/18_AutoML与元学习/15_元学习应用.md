## 2. 少样本图像分类

### 2.1 标准流程

```python
class FewShotClassifier:
    def __init__(self, meta_model, n_way=5, k_shot=1):
        self.model = meta_model
        self.n_way = n_way
        self.k_shot = k_shot
    
    def adapt_and_predict(self, support_images, support_labels, query_images):
        """
        support_images: (n_way * k_shot, C, H, W)
        query_images: (n_query, C, H, W)
        """
        # 如果是MAML类方法：内层适应
        if hasattr(self.model, 'adapt'):
            self.model.adapt(support_images, support_labels)
        
        # 预测
        with torch.no_grad():
            logits = self.model(support_images, support_labels, query_images, self.n_way)
            predictions = logits.argmax(dim=-1)
        
        return predictions
```

## 3. 机器人快速适应

### 3.1 MAML 在强化学习中的应用

**RL-MAML**：学习一个好的策略初始化，少量交互就能适应新环境。

```python
class MAMLRL:
    """MAML用于强化学习"""
    def __init__(self, policy_net, inner_lr=0.1, outer_lr=1e-3):
        self.policy = policy_net
        self.inner_lr = inner_lr
        self.meta_optimizer = torch.optim.Adam(policy_net.parameters(), lr=outer_lr)
    
    def meta_train_step(self, envs):
        """每个env代表一个不同任务"""
        meta_loss = 0
        
        for env in envs:
            # 内层：在该环境上收集轨迹
            trajectories = collect_trajectories(self.policy, env, n_episodes=5)
            inner_loss = compute_policy_loss(trajectories)
            
            # 计算梯度并更新
            grads = torch.autograd.grad(inner_loss, self.policy.parameters(),
                                         create_graph=True)
            adapted_params = [p - self.inner_lr * g 
                              for p, g in zip(self.policy.parameters(), grads)]
            
            # 外层：在适应后的策略上评估
            adapted_trajectories = collect_trajectories_with_params(
                self.policy, adapted_params, env, n_episodes=5
            )
            outer_loss = compute_policy_loss(adapted_trajectories)
            meta_loss += outer_loss
        
        meta_loss /= len(envs)
        self.meta_optimizer.zero_grad()
        meta_loss.backward()
        self.meta_optimizer.step()
```

## 4. 低资源NLP

### 4.1 少样本文本分类

```python
class MetaTextClassifier(nn.Module):
    def __init__(self, pretrained_bert, hidden_dim=768, n_way=5):
        super().__init__()
        self.encoder = pretrained_bert
        self.n_way = n_way
        
        # 元学习层
        self.meta_layers = nn.Sequential(
            nn.Linear(hidden_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 128)
        )
    
    def forward(self, support_x, support_y, query_x):
        # BERT编码
        support_emb = self.encoder(support_x).last_hidden_state[:, 0, :]  # [CLS]
        query_emb = self.encoder(query_x).last_hidden_state[:, 0, :]
        
        # 元学习嵌入
        support_emb = self.meta_layers(support_emb)
        query_emb = self.meta_layers(query_emb)
        
        # ProtoNet分类
        prototypes = compute_prototypes(support_emb, support_y, self.n_way)
        dists = torch.cdist(query_emb, prototypes)
        return -dists
```

## 7. 推荐系统

### 7.1 冷启动问题

新用户没有历史行为数据，传统推荐系统无法工作。元学习通过学习"如何快速了解用户偏好"来解决。

```python
class MetaRecommender(nn.Module):
    def __init__(self, user_encoder, item_encoder, meta_dim=64):
        super().__init__()
        self.user_encoder = user_encoder
        self.item_encoder = item_encoder
        
        # 元学习：从少量交互学习用户嵌入
        self.user_meta = nn.Sequential(
            nn.Linear(meta_dim * 2, meta_dim),  # 交互历史 → 用户偏好
            nn.ReLU(),
            nn.Linear(meta_dim, meta_dim),
        )
    
    def adapt_to_new_user(self, few_interactions):
        """
        few_interactions: 新用户的少量点击/评分
        返回: 用户嵌入
        """
        item_embeddings = self.item_encoder(few_interactions['items'])
        ratings = few_interactions['ratings']
        
        # 聚合交互信息
        weighted_emb = (item_embeddings * ratings.unsqueeze(-1)).mean(dim=0)
        interaction_feat = torch.cat([item_embeddings.mean(dim=0), weighted_emb])
        
        user_embedding = self.user_meta(interaction_feat)
        return user_embedding
```

## 8. 元学习的挑战与未来

### 8.1 当前挑战

| 挑战 | 说明 |
|------|------|
| 任务分布假设 | 需要训练和测试的任务来自同一分布 |
| 计算成本 | MAML 等方法需要大量计算 |
| 任务采样策略 | 如何选择有代表性的训练任务 |
| 从少样本到零样本 | 减少对support set的依赖 |

### 8.2 未来方向

1. **元学习 + 大模型**：用预训练大模型作为backbone，元学习做快速适配
2. **任务无关的元学习**：不需要显式任务定义
3. **持续元学习**：元学习器本身也能持续学习
4. **与LLM结合**：用大语言模型作为元学习器

---

**关键要点**：
1. 元学习在少样本分类、机器人、医学、药物发现、推荐系统等领域有广泛应用
2. 机器人领域：MAML-RL 学习策略初始化，少量交互适应新环境
3. NLP领域：结合预训练模型（BERT/GPT）+ 元学习处理低资源任务
4. 元学习的未来趋势是与大模型结合，实现更强的快速适应能力
