# 7_贝叶斯优化深入

## 1. 贝叶斯优化框架

### 1.1 伪代码

```
输入: 搜索空间 X, 目标函数 f, 初始点集 D_0
输出: 最优配置 x*

1. 初始化: 随机评估 N_0 个点, 得到 D_0 = {(x_i, y_i)}
2. 循环 t = 1, 2, ... T:
    a. 在 D_{0:t-1} 上拟合代理模型 M
    b. 选择 x_t = argmax_x α(x; M, D)  (采集函数)
    c. 评估 y_t = f(x_t)
    d. 更新数据 D_t = D_{t-1} ∪ {(x_t, y_t)}
3. 返回 x* = argmax_{(x,y)∈D} y
```

## 2. 高斯过程代理模型

### 2.1 高斯过程 (GP)

**高斯过程**是函数的分布：$f(x) \sim \mathcal{GP}(\mu(x), k(x, x'))$

- $\mu(x)$：均值函数（通常为0）
- $k(x, x')$：核函数（协方差函数），定义函数的平滑性

对于观测数据 $D = \{(x_i, y_i)\}_{i=1}^n$：

$$\mathbf{y} \sim \mathcal{N}(\boldsymbol{\mu}, K + \sigma_n^2 I)$$

其中 $K_{ij} = k(x_i, x_j)$。

### 2.2 GP预测

给定新点 $x_*$，预测分布为：

$$\mu(x_*) = \mathbf{k}_*^T (K + \sigma_n^2 I)^{-1} \mathbf{y}$$

$$\sigma^2(x_*) = k(x_*, x_*) - \mathbf{k}_*^T (K + \sigma_n^2 I)^{-1} \mathbf{k}_*$$

其中 $\mathbf{k}_* = [k(x_1, x_*), \ldots, k(x_n, x_*)]^T$。

### 2.3 常用核函数

```python
import torch
import gpytorch

class GPModel(gpytorch.models.ExactGP):
    """高斯过程模型"""
    def __init__(self, train_x, train_y, likelihood):
        super().__init__(train_x, train_y, likelihood)
        self.mean_module = gpytorch.means.ConstantMean()
        
        # RBF核 (平方指数)
        self.covar_module = gpytorch.kernels.ScaleKernel(
            gpytorch.kernels.RBFKernel()
        )
    
    def forward(self, x):
        mean = self.mean_module(x)
        covar = self.covar_module(x)
        return gpytorch.distributions.MultivariateNormal(mean, covar)
```

| 核函数 | 公式 | 特点 |
|--------|------|------|
| RBF (平方指数) | $\exp\left(-\frac{\|x-x'\|^2}{2l^2}\right)$ | 无穷可微，非常平滑 |
| Matérn 5/2 | $\left(1+\frac{\sqrt{5}r}{l}+\frac{5r^2}{3l^2}\right)\exp\left(-\frac{\sqrt{5}r}{l}\right)$ | 二阶可微，推荐 |
| Matérn 3/2 | $\left(1+\frac{\sqrt{3}r}{l}\right)\exp\left(-\frac{\sqrt{3}r}{l}\right)$ | 一阶可微 |

## 3. 采集函数

### 3.1 Expected Improvement (EI)

$$\alpha_{EI}(x) = \mathbb{E}\left[\max(f(x) - f^*, 0)\right] = \sigma(x)\left[Z \Phi(Z) + \phi(Z)\right]$$

其中 $Z = \frac{\mu(x) - f^*}{\sigma(x)}$，$f^*$ 是当前最优值。

```python
def expected_improvement(mu, sigma, f_best, xi=0.01):
    """
    Expected Improvement 采集函数
    mu: 预测均值
    sigma: 预测标准差
    f_best: 当前最优观测值
    xi: 探索-利用平衡参数
    """
    from scipy.stats import norm
    
    with np.errstate(divide='warn'):
        imp = mu - f_best - xi
        Z = imp / sigma
        ei = imp * norm.cdf(Z) + sigma * norm.pdf(Z)
        ei[sigma == 0.0] = 0.0
    return ei
```

### 3.2 Upper Confidence Bound (UCB)

$$\alpha_{UCB}(x) = \mu(x) + \kappa \cdot \sigma(x)$$

- $\kappa$ 大 → 偏好探索（高不确定性区域）
- $\kappa$ 小 → 偏好利用（高均值区域）

```python
def upper_confidence_bound(mu, sigma, kappa=2.576):
    """UCB采集函数"""
    return mu + kappa * sigma
```

### 3.3 Probability of Improvement (PI)

$$\alpha_{PI}(x) = P(f(x) > f^* + \xi) = \Phi\left(\frac{\mu(x) - f^* - \xi}{\sigma(x)}\right)$$

### 3.4 采集函数对比

| 函数 | 平衡方式 | 推荐度 |
|------|----------|--------|
| EI | 隐式（期望值） | 最推荐 |
| UCB | 显式（$\kappa$参数） | 简单好用 |
| PI | 隐式（$\xi$参数） | 偏保守 |

## 4. 完整贝叶斯优化实现

```python
import numpy as np
from sklearn.gaussian_process import GaussianProcessRegressor
from sklearn.gaussian_process.kernels import Matern

class BayesianOptimizer:
    """贝叶斯优化器"""
    def __init__(self, search_space, n_init=5):
        self.search_space = search_space
        self.n_init = n_init
        self.X_observed = []
        self.y_observed = []
        
        # GP代理模型
        self.gp = GaussianProcessRegressor(
            kernel=Matern(nu=2.5),
            alpha=1e-6,
            normalize_y=True,
            n_restarts_optimizer=5,
        )
    
    def suggest(self):
        """建议下一个评估点"""
        # 初始化阶段：随机采样
        if len(self.X_observed) < self.n_init:
            return self._random_sample()
        
        # 拟合GP
        X = np.array(self.X_observed)
        y = np.array(self.y_observed)
        self.gp.fit(X, y)
        
        # 最大化采集函数
        best_x = None
        best_ei = -1
        
        # 多起点优化
        candidates = [self._random_sample() for _ in range(100)]
        
        for x_cand in candidates:
            x_arr = np.array([x_cand])
            mu, sigma = self.gp.predict(x_arr, return_std=True)
            ei = expected_improvement(mu, sigma, y.max())
            
            if ei > best_ei:
                best_ei = ei
                best_x = x_cand
        
        return best_x
    
    def observe(self, x, y):
        """记录观测结果"""
        self.X_observed.append(x)
        self.y_observed.append(y)
    
    def _random_sample(self):
        """随机采样"""
        x = []
        for dim in self.search_space:
            if dim['type'] == 'log_uniform':
                x.append(np.exp(np.random.uniform(
                    np.log(dim['low']), np.log(dim['high'])
                )))
            elif dim['type'] == 'uniform':
                x.append(np.random.uniform(dim['low'], dim['high']))
            elif dim['type'] == 'choice':
                x.append(np.random.choice(dim['values']))
        return x

# 使用示例
def optimize_hyperparams(objective_fn, search_space, n_iter=50):
    """超参数优化"""
    optimizer = BayesianOptimizer(search_space)
    
    for i in range(n_iter):
        # 建议
        params = optimizer.suggest()
        
        # 评估
        score = objective_fn(params)
        
        # 记录
        optimizer.observe(params, score)
        
        print(f"Iter {i+1}: score={score:.4f}, params={params}")
    
    best_idx = np.argmax(optimizer.y_observed)
    return optimizer.X_observed[best_idx], optimizer.y_observed[best_idx]
```

## 5. GP 的局限与改进

### 5.1 计算复杂度

GP 的预测需要 $O(n^3)$ 的矩阵求逆，其中 $n$ 是观测点数。当 $n > 1000$ 时计算量不可接受。

### 5.2 解决方案

| 方案 | 思路 | 复杂度 |
|------|------|--------|
| 稀疏GP (SGPR) | 引入诱导点 $M \ll N$ | $O(NM^2)$ |
| 随机特征近似 | 用随机特征替代核矩阵 | $O(NMD)$ |
| 集成GP | 多个子集GP集成 | 并行 |

### 5.3 固定秩近似

## 6. 高维空间的贝叶斯优化

### 6.1 维度灾难

GP在高维（>20维）空间中效果下降，因为：
- 核函数在高维空间中变得平坦
- 需要指数级的观测数据

### 6.2 解决方案

| 方案 | 思路 |
|------|------|
| 随机嵌入 (REMBO) | 随机投影到低维 |
| Add-GP | 假设函数可加性分解 |
| SAASBO | 稀疏轴对齐子空间 |

---

**关键要点**：
1. 高斯过程是贝叶斯优化最常用的代理模型，提供预测均值和不确定性
2. EI是推荐的采集函数，自动平衡探索和利用
3. GP的 $O(n^3)$ 复杂度限制了评估次数，稀疏GP是常用解决方法
4. 高维空间需要额外处理（随机嵌入、可加性假设等）
