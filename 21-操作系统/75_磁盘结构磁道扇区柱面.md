# 75_磁盘结构磁道扇区柱面

## 核心概念

- **磁盘（Hard Disk）**：利用磁性材料存储数据的旋转式存储设备。
- 基本物理结构：**盘面（Platter）**、**磁道（Track）**、**扇区（Sector）**、**柱面（Cylinder）**。
- **磁盘地址** = 柱面号（C）+ 磁头号（H）+ 扇区号（S）——CHS 寻址。
- 408 考点：磁盘物理结构、CHS 寻址、磁盘容量计算。

## 原理分析

### 磁盘物理结构

```
              ┌──────────────────┐
              │     主轴（Spindle）│
              │        │         │
          ════╪════════╪═════════╪════  ← 盘面 0（上面）
          ════╪════════╪═════════╪════  ← 盘面 0（下面）
          ════╪════════╪═════════╪════  ← 盘面 1（上面）
          ════╪════════╪═════════╪════  ← 盘面 1（下面）
              │        │         │
              │   [磁头臂组件]    │
              └──────────────────┘
```

每个盘面被划分为同心圆环（**磁道**），每个磁道被划分为若干**扇区**。

```
         盘面俯视图
        ╭─────────────╮
       ╱   ╭───────╮   ╲
      │   ╱ ╭───╮ ╲   │
      │  │  │中心│  │  │   ← 同心圆 = 磁道
      │   ╲ ╰───╯ ╱   │   ← 每个磁道被扇形线分割 = 扇区
       ╲   ╰───────╯   ╱
        ╰─────────────╯
```

### 关键术语

**磁道（Track）**：
- 盘面上以主轴为圆心的同心圆环。
- 最外圈为 0 号磁道，向内编号递增。
- 典型硬盘有数千个磁道（如 50,000+）。

**扇区（Sector）**：
- 磁道被等分为若干弧段，每个弧段是一个扇区。
- 扇区是磁盘 I/O 的最小单位。
- 传统扇区大小 = 512 字节，现代高级格式 = 4KB。

**柱面（Cylinder）**：
- 所有盘面上相同编号的磁道组成一个柱面。
- 同一柱面上的磁头可以同时访问（不需要移动磁头臂）。
- 典型硬盘有数千个柱面。

**磁头（Head）**：
- 每个盘面有上下两个磁头（双面读写）。
- 所有磁头固定在同一磁头臂上，同时移动。
- 磁头数 = 盘面数 × 2。

### CHS 寻址

磁盘上的每个扇区由三元组 **(C, H, S)** 唯一标识：
- **C（Cylinder）**：柱面号（决定磁头臂位置）
- **H（Head）**：磁头号（选择哪个盘面的哪个磁头）
- **S（Sector）**：扇区号（磁道上的哪个扇区）

### 磁盘容量计算

$$\text{容量} = \text{柱面数} \times \text{磁头数} \times \text{每磁道扇区数} \times \text{扇区大小}$$

**例题**：某硬盘 2000 个柱面、4 个盘面（8 个磁头）、每磁道 500 个扇区、扇区 512B。

$$\text{容量} = 2000 \times 8 \times 500 \times 512 = 4{,}096{,}000{,}000\text{B} \approx 3.81\text{GB}$$

### LBA 寻址

现代磁盘使用 **LBA（Logical Block Addressing）** 代替 CHS：
- 将所有扇区线性编号：0, 1, 2, ..., N-1。
- 磁盘控制器负责将 LBA 转换为物理 CHS 地址。
- 突破 CHS 的 8GB 寻址限制。

$$\text{LBA} = C \times (\text{H数} \times \text{S数}) + H \times \text{S数} + (S - 1)$$

### 磁盘的读写过程

1. **寻道（Seek）**：磁头臂移动到目标柱面（最耗时）。
2. **旋转延迟（Rotational Delay）**：等待目标扇区旋转到磁头下方。
3. **数据传输（Transfer）**：读/写扇区数据。

$$T_{\text{total}} = T_{\text{seek}} + T_{\text{rotation}} + T_{\text{transfer}}$$

## 直观理解

- **磁盘** = 一摞留声机唱片，每张唱片的每一圈是一条磁道，磁道上的每段弧是一个扇区。
- **柱面** = 所有唱片在同一圈的位置——磁头臂不动就能访问同一柱面的所有扇区。
- **CHS 寻址** = "第几摞唱片的第几圈的第几段弧"。
