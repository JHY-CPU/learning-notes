# 58_页面置换缺页率计算

## 核心概念

- **缺页（Page Fault）**：访问的页面不在物理内存中，需从外存调入。
- **缺页率（Page Fault Rate）**：$f = \frac{\text{缺页次数}}{\text{总访问次数}}$。
- **命中率**：$h = 1 - f$。
- **有效访问时间（EAT）**：考虑缺页处理后的平均访问时间。
- 408 核心计算题：给定访问序列、物理块数、页面置换算法，计算缺页次数和缺页率。

## 原理分析

### 有效访问时间计算

$$\text{EAT} = (1 - f) \times t_{\text{mem}} + f \times (p \times t_{\text{swap}} + t_{\text{mem}})$$

其中：
- $t_{\text{mem}}$ = 内存访问时间（通常 $100\text{ns}$）
- $p$ = 污染概率（修改过的脏页比例，通常简化为 0 或给定值）
- $t_{\text{swap}}$ = 缺页处理时间（含磁盘 I/O，通常约 $10\text{ms}$）

简化版本（假设无脏页）：

$$\text{EAT} = (1 - f) \times t_{\text{mem}} + f \times (t_{\text{miss}} + t_{\text{mem}})$$

**例题**：$t_{\text{mem}} = 100\text{ns}$，$t_{\text{miss}} = 10\text{ms}$，要使 EAT 不超过 $200\text{ns}$，缺页率上限？

$$200 = (1-f) \times 100 + f \times (10{,}000{,}000 + 100)$$
$$200 = 100 - 100f + 10{,}000{,}100f$$
$$100 = 10{,}000{,}000f$$
$$f = \frac{100}{10{,}000{,}000} = 10^{-5}$$

即缺页率必须低于 $0.001\%$，说明对缺页率要求极高。

### OPT 算法缺页计算（最佳置换）

**规则**：淘汰未来最长时间不被使用的页面。

**例题**：页面访问序列 `2 3 2 1 5 2 4 5 3 2 5 2`，物理块数 $m=3$。

| 步骤 | 访问 | 块1 | 块2 | 块3 | 淘汰谁 | 缺页? |
|------|------|-----|-----|-----|--------|-------|
| 1 | 2 | 2 | - | - | - | Y |
| 2 | 3 | 2 | 3 | - | - | Y |
| 3 | 2 | 2 | 3 | - | - | N |
| 4 | 1 | 2 | 3 | 1 | - | Y |
| 5 | 5 | 2 | 5 | 1 | 淘汰3 | Y |
| 6 | 2 | 2 | 5 | 1 | - | N |
| 7 | 4 | 4 | 5 | 1 | 淘汰2 | Y |
| 8 | 5 | 4 | 5 | 1 | - | N |
| 9 | 3 | 4 | 5 | 3 | 淘汰1 | Y |
| 10 | 2 | 4 | 5 | 2 | 淘汰3→? | Y |

步骤 5：未来序列中 2 先出现，3 最后出现 → 淘汰 3。
步骤 7：未来序列中 5 先出现，1 最后出现 → 淘汰 1？不对，看 5 后面：5 出现后是 3,2,5,2，1 不再出现 → 淘汰 1。但 1 已经在步骤 5 中被淘汰了。重新分析：

步骤 7 的块状态是 {2,5,1}，未来访问序列是 4,5,3,2,5,2。1 不再出现 → 淘汰 1。得到 {2,5,4}。

步骤 9：块状态 {2,5,4}，未来 3,2,5,2。4 不再出现 → 淘汰 4。得到 {2,5,3}。

步骤 10：块状态 {2,5,3}，未来 5,2。3 不再出现 → 淘汰 3。得到 {2,5,4}——但 2 不是访问的。访问的是 2，2 已在块中。

重新修正：步骤 10 访问 2，2 已在块中 → 不缺页！

**正确结果**：缺页 6 次，缺页率 = $6/12 = 50\%$。

### LRU 算法缺页计算（最近最少使用）

**规则**：淘汰最近最久未被访问的页面。

**例题**：序列 `2 3 2 1 5 2 4 5 3 2 5 2`，$m=3$。

| 步骤 | 访问 | 块状态(新→旧) | 缺页? |
|------|------|-------------|-------|
| 1 | 2 | [2] | Y |
| 2 | 3 | [3,2] | Y |
| 3 | 2 | [2,3] | N |
| 4 | 1 | [1,2,3] | Y |
| 5 | 5 | [5,1,2] | Y(淘汰3) |
| 6 | 2 | [2,5,1] | N |
| 7 | 4 | [4,2,5] | Y(淘汰1) |
| 8 | 5 | [5,4,2] | N |
| 9 | 3 | [3,5,4] | Y(淘汰2) |
| 10 | 2 | [2,3,5] | Y(淘汰4) |
| 11 | 5 | [5,2,3] | N |
| 12 | 2 | [2,5,3] | N |

缺页 7 次，缺页率 = $7/12 \approx 58.3\%$。

### Clock 算法缺页计算（NRU / 二次机会）

**规则**：页面组织成循环队列，每页有访问位（A 位）。淘汰时扫描：
- A=0 → 淘汰该页。
- A=1 → 清零，给第二次机会，继续扫描。

**例题**：序列 `1 2 3 4 1 2 5 1 2 3 4 5`，$m=3$。

初始无页面，前 3 次调入 1,2,3（A 位全为 1）。

| 步骤 | 访问 | 操作 | 块状态 | 缺页? |
|------|------|------|--------|-------|
| 4 | 4 | 指针在1，A=1→清零；2，A=1→清零；3，A=1→清零；回1，A=0→淘汰 | [4,2,3] | Y |
| 5 | 1 | 指针在2，A=0→淘汰 | [4,1,3] | Y |
| 6 | 2 | 指针在3，A=0→淘汰 | [4,1,2] | Y |
| 7 | 5 | 指针在4，A=0→淘汰 | [5,1,2] | Y |
| 8 | 1 | 命中，A=1 | [5,1,2] | N |
| 9 | 2 | 命中，A=1 | [5,1,2] | N |
| 10 | 3 | 指针在5，A=0→淘汰 | [3,1,2] | Y |
| 11 | 4 | 指针在1，A=1→清零；2，A=1→清零；回1，A=0→淘汰 | [3,4,2] | Y |
| 12 | 5 | 指针在3，A=0→淘汰 | [5,4,2] | Y |

缺页 8 次。

### 改进型 Clock 算法

同时考虑**访问位 A** 和**修改位 M**，四类页面：
1. A=0, M=0：最佳淘汰候选（未访问未修改）。
2. A=0, M=1：次佳（未访问但修改过）。
3. A=1, M=0：较差（最近访问过但未修改）。
4. A=1, M=1：最差（最近访问且修改过）。

最多扫描 4 轮，减少磁盘 I/O。

## 直观理解

- OPT 是"开了天眼"，知道未来哪个页面不再用——理想但不可实现，用作性能上界。
- LRU 是"用最近的推断未来的"，好比翻书时把最近没翻的页先折起来。
- FIFO 是"只看谁来的最早"，不关心谁更有用。
- Clock 是"给一次机会"的 FIFO 改进版。

## 知识关联

- 与 57 FIFO 和 Belady 异常：FIFO 缺页率可能因物理块增多而升高。
- 与 55、56 分配置换策略：全局置换下的缺页率更难分析。
- 与 80 工作集与抖动：缺页率过高是抖动的直接表现。
- 408 考试常考：给出访问序列，手算 OPT、LRU、FIFO 的缺页次数。

## 代码实现

### OPT、LRU、FIFO、Clock 四种页面置换算法对比

```c
#include <stdio.h>
#include <stdbool.h>
#include <limits.h>

/*
 * 实现四种经典页面置换算法，并对比其缺页率
 * - OPT: 最佳置换（理论最优，需要预知未来）
 * - LRU: 最近最久未使用
 * - FIFO: 先进先出
 * - Clock: 时钟算法（NRU，LRU 的近似）
 */

int pages[] = {2, 3, 2, 1, 5, 2, 4, 5, 3, 2, 5, 2};
int n = 12;
int m = 3;  /* 物理块数 */

/* OPT: 淘汰未来最久不被使用的页面 */
int opt() {
    int frames[3] = {-1, -1, -1};
    int faults = 0;

    for (int i = 0; i < n; i++) {
        bool hit = false;
        for (int j = 0; j < m; j++)
            if (frames[j] == pages[i]) { hit = true; break; }

        if (!hit) {
            faults++;
            /* 找空闲位置 */
            int slot = -1;
            for (int j = 0; j < m; j++)
                if (frames[j] == -1) { slot = j; break; }

            if (slot == -1) {
                /* 找未来最久不使用的页面 */
                int farthest = -1, victim = 0;
                for (int j = 0; j < m; j++) {
                    int next_use = INT_MAX;
                    for (int k = i + 1; k < n; k++) {
                        if (pages[k] == frames[j]) {
                            next_use = k;
                            break;
                        }
                    }
                    if (next_use > farthest) {
                        farthest = next_use;
                        victim = j;
                    }
                }
                slot = victim;
            }
            frames[slot] = pages[i];
        }
    }
    return faults;
}

/* LRU: 淘汰最近最久未使用的页面 */
int lru() {
    int frames[3] = {-1, -1, -1};
    int last_used[3] = {0, 0, 0};
    int faults = 0;

    for (int i = 0; i < n; i++) {
        bool hit = false;
        for (int j = 0; j < m; j++) {
            if (frames[j] == pages[i]) {
                hit = true;
                last_used[j] = i;
                break;
            }
        }
        if (!hit) {
            faults++;
            int slot = -1;
            for (int j = 0; j < m; j++)
                if (frames[j] == -1) { slot = j; break; }
            if (slot == -1) {
                int oldest = i;
                for (int j = 0; j < m; j++) {
                    if (last_used[j] < oldest) {
                        oldest = last_used[j];
                        slot = j;
                    }
                }
            }
            frames[slot] = pages[i];
            last_used[slot] = i;
        }
    }
    return faults;
}

/* FIFO: 淘汰最早进入的页面 */
int fifo() {
    int frames[3] = {-1, -1, -1};
    int enter_time[3] = {0, 0, 0};
    int head = 0;
    int faults = 0;

    for (int i = 0; i < n; i++) {
        bool hit = false;
        for (int j = 0; j < m; j++)
            if (frames[j] == pages[i]) { hit = true; break; }
        if (!hit) {
            faults++;
            frames[head] = pages[i];
            enter_time[head] = i;
            head = (head + 1) % m;
        }
    }
    return faults;
}

/* Clock: 时钟算法，用访问位近似 LRU */
int clock_algo() {
    int frames[3] = {-1, -1, -1};
    bool ref[3] = {false, false, false};
    int pointer = 0;
    int faults = 0;

    for (int i = 0; i < n; i++) {
        bool hit = false;
        for (int j = 0; j < m; j++) {
            if (frames[j] == pages[i]) {
                hit = true;
                ref[j] = true;  /* 命中，置访问位 */
                break;
            }
        }
        if (!hit) {
            faults++;
            /* Clock 扫描：给第二次机会 */
            while (ref[pointer]) {
                ref[pointer] = false;
                pointer = (pointer + 1) % m;
            }
            frames[pointer] = pages[i];
            ref[pointer] = true;
            pointer = (pointer + 1) % m;
        }
    }
    return faults;
}

int main() {
    printf("=== 四种页面置换算法对比 ===\n");
    printf("访问序列: ");
    for (int i = 0; i < n; i++) printf("%d ", pages[i]);
    printf("\n物理块数: %d\n\n", m);

    printf("OPT:   缺页 %d 次（理论最优）\n", opt());
    printf("LRU:   缺页 %d 次\n", lru());
    printf("FIFO:  缺页 %d 次\n", fifo());
    printf("Clock: 缺页 %d 次（LRU近似）\n", clock_algo());

    return 0;
}
```

### 有效访问时间（EAT）计算

```c
#include <stdio.h>

/*
 * 有效访问时间计算
 * EAT = (1-f) * t_mem + f * (t_miss + t_mem)
 *
 * 缺页率对性能影响极大：
 * - t_mem = 100ns, t_miss = 10ms = 10,000,000ns
 * - 即使缺页率只有 0.1%，EAT 也会增加 100 倍
 */
int main() {
    double t_mem = 100.0;        /* 内存访问时间 (ns) */
    double t_miss = 10000000.0;  /* 缺页处理时间 10ms = 10^7 ns */

    printf("=== 有效访问时间计算 ===\n");
    printf("内存访问时间: %.0f ns\n", t_mem);
    printf("缺页处理时间: %.0f ns (%.0f ms)\n\n", t_miss, t_miss/1e6);

    double rates[] = {0.0, 0.000001, 0.00001, 0.0001, 0.001, 0.01, 0.1};
    int num = sizeof(rates) / sizeof(rates[0]);

    printf("| 缺页率    | EAT (ns)        | 性能影响 |\n");
    printf("|-----------|-----------------|----------|\n");

    for (int i = 0; i < num; i++) {
        double f = rates[i];
        double eat = (1 - f) * t_mem + f * (t_miss + t_mem);
        printf("| %.6f  | %15.1f | %.0fx     |\n", f, eat, eat / t_mem);
    }

    /* 反向计算：给定 EAT 上限，求缺页率上限 */
    double eat_limit = 200.0;  /* 要求 EAT ≤ 200ns */
    double f_max = (eat_limit - t_mem) / t_miss;
    printf("\n若要求 EAT ≤ %.0f ns，则缺页率必须 ≤ %.2e (%.6f%%)\n",
           eat_limit, f_max, f_max * 100);

    return 0;
}
```
