# 熔断降级最佳实践

## 一、实践清单

```yaml
配置:
  □ 网关层全局限流
  □ 服务间调用熔断
  □ 核心方法降级
  □ 超时控制合理
  □ 重试配合幂等

监控:
  □ 熔断触发告警
  □ 降级触发告警
  □ 规则动态调整
  □ 定期调优规则
```

## 二、选型建议

| 场景 | 推荐 |
|------|------|
| Spring Cloud | Sentinel |
| 轻量级 | Resilience4j |
| 全功能 | Sentinel |

## 三、注意事项

1. **熔断降级是微服务必备**
2. **Sentinel 国内首选**
3. **降级逻辑要保证核心功能**
4. **规则要持续调优**
5. **监控是关键**
