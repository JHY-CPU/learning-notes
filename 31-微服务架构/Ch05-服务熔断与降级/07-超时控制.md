# 超时控制

## 一、超时配置

```yaml
# Feign 超时
feign:
  client:
    config:
      default:
        connect-timeout: 5000
        read-timeout: 10000

# Resilience4j 超时
resilience4j:
  timelimiter:
    instances:
      userService:
        timeout-duration: 5s
```

## 二、注意事项

1. **超时时间要合理设置**
2. **连接超时和读取超时分开配置**
3. **核心服务超时短，非核心服务超时长**
4. **超时后要有降级逻辑**
5. **监控超时情况**
