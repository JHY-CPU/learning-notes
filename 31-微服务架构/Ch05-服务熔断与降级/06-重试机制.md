# 重试机制

## 一、重试策略

```java
// Resilience4j 重试
@Retry(name = "userService", fallbackMethod = "fallback")
public User getUser(Long id) {
    return userService.getUser(id);
}

// 重试配置
resilience4j:
  retry:
    instances:
      userService:
        max-attempts: 3
        wait-duration: 1s
        exponential-backoff-multiplier: 2
        retry-exceptions:
          - java.io.IOException
        ignore-exceptions:
          - com.example.BusinessException
```

## 二、注意事项

1. **重试要配合幂等**
2. **指数退避减少压力**
3. **区分可重试和不可重试异常**
4. **重试次数要有限制**
5. **重试记录要日志记录**
