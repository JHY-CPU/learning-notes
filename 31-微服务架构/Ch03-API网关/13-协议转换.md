# 协议转换

## 一、HTTP to gRPC

```yaml
spring:
  cloud:
    gateway:
      routes:
      - id: grpc-route
        uri: grpc://grpc-service:9090
        predicates:
        - Path=/api/grpc/**
```

## 二、注意事项

1. **网关可以做协议转换**
2. **gRPC 需要特殊支持**
3. **HTTP/2 是 gRPC 的基础**
4. **协议转换有性能开销**
5. **保持协议一致性减少转换**
