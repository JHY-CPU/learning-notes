# 灰度发布

## 一、灰度策略

```yaml
# 按 Header 灰度
spring:
  cloud:
    gateway:
      routes:
      - id: user-service-v1
        uri: lb://user-service-v1
        predicates:
        - Header=X-Version, v1
      - id: user-service-v2
        uri: lb://user-service-v2
        predicates:
        - Header=X-Version, v2
```

## 二、注意事项

1. **灰度比例要可配置**
2. **灰度规则要动态生效**
3. **灰度失败要能快速回滚**
4. **监控灰度版本的指标**
5. **灰度期间做好日志采集**
