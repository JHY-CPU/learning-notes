# 接口版本管理

## 一、版本策略

```java
// URI 版本
@GetMapping("/api/v1/users/{id}")
@GetMapping("/api/v2/users/{id}")

// Header 版本
@GetMapping(value = "/users/{id}", headers = "X-API-Version=1")
@GetMapping(value = "/users/{id}", headers = "X-API-Version=2")
```

## 二、注意事项

1. **版本策略要统一**
2. **向后兼容是原则**
3. **废弃接口要通知**
4. **版本切换要平滑**
5. **文档要同步更新**
