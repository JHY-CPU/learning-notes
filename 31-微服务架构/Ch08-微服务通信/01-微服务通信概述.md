# 微服务通信概述

## 一、通信模式

```yaml
同步通信:
  - REST HTTP - 简单通用
  - gRPC - 高性能二进制
  - GraphQL - 灵活查询

异步通信:
  - 消息队列 - Kafka/RabbitMQ/RocketMQ
  - 事件驱动 - Event Sourcing
```

## 二、模式选择

| 场景 | 推荐 |
|------|------|
| 查询数据 | REST HTTP |
| 内部调用 | gRPC |
| 事件通知 | MQ |
| 实时推送 | WebSocket |

## 三、注意事项

1. **强依赖用同步，弱依赖用异步**
2. **避免同步调用链过长**
3. **超时和重试必须配置**
4. **接口版本要管理**
5. **序列化格式要统一**
