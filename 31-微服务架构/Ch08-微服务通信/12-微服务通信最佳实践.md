# 微服务通信最佳实践

## 一、实践清单

```yaml
同步通信:
  □ Feign 声明式调用
  □ 超时必须配置
  □ 熔断降级要有
  □ 接口版本管理
  □ 错误码规范化

异步通信:
  □ 消息格式统一
  □ 消费幂等处理
  □ 死信队列兜底
  □ 消费延迟监控
  □ 消息轨迹追踪
```

## 二、选型建议

| 场景 | 推荐 |
|------|------|
| 查询数据 | REST HTTP |
| 内部调用 | gRPC |
| 事件通知 | MQ |
| 实时推送 | WebSocket |

## 三、注意事项

1. **强依赖用同步，弱依赖用异步**
2. **避免同步调用链过长**
3. **超时和重试必须配置**
4. **接口版本要管理**
5. **通信安全要做好**
