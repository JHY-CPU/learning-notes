# 超时与重试

## 一、超时配置

```yaml
feign:
  client:
    config:
      default:
        connect-timeout: 5000
        read-timeout: 10000
```

## 二、重试策略

```java
@Retry(name = "userService", fallbackMethod = "fallback")
public User getUser(Long id) {
    return userService.getUser(id);
}
```

## 三、注意事项

1. **超时必须配置**
2. **重试要配合幂等**
3. **指数退避减少压力**
4. **区分可重试和不可重试异常**
5. **重试次数要有限制**
