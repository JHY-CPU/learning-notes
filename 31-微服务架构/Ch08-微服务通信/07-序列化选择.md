# 序列化选择

## 一、序列化对比

| 格式 | 性能 | 体积 | 可读性 | 适用场景 |
|------|------|------|--------|---------|
| JSON | 中 | 大 | 好 | REST API |
| Protobuf | 高 | 小 | 差 | gRPC |
| Hessian | 中 | 中 | 差 | Dubbo |
| Kryo | 高 | 小 | 差 | Java 专用 |

## 二、注意事项

1. **JSON 适合对外 API**
2. **Protobuf 适合内部通信**
3. **序列化要向前兼容**
4. **大对象考虑压缩**
5. **统一序列化格式**
