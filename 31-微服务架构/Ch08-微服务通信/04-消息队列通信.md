# 消息队列通信

## 一、异步通信

```java
// 生产者
kafkaTemplate.send("order-events", order);

// 消费者
@KafkaListener(topics = "order-events")
public void onOrderCreated(Order order) {
    inventoryService.reserve(order);
}
```

## 二、注意事项

1. **异步通信解耦服务**
2. **消息要保证幂等消费**
3. **消息格式要统一**
4. **消费延迟要监控**
5. **死信队列处理失败消息**
