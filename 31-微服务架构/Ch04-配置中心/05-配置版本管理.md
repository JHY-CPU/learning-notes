# 配置版本管理

## 一、Git 存储配置

```yaml
spring:
  cloud:
    config:
      server:
        git:
          uri: https://github.com/org/config-repo
          default-label: main
          search-paths: '{application}'
```

## 二、注意事项

1. **配置变更要有记录**
2. **支持配置回滚**
3. **配置修改需要审批**
4. **敏感配置不要提交到 Git**
5. **配置变更要通知相关团队**
