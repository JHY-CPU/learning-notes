# 配置中心选型

## 一、选型对比

| 维度 | Nacos | Apollo | Consul | Config |
|------|-------|--------|--------|--------|
| 功能 | 注册+配置 | 配置 | 注册+配置 | 配置 |
| 动态刷新 | 支持 | 支持 | 支持 | 需重启 |
| 灰度 | 支持 | 支持 | 不支持 | 不支持 |
| 权限 | 基础 | 完善 | 基础 | 无 |
| 国内 | 最流行 | 流行 | 一般 | 少用 |

## 二、选型建议

```yaml
国内项目: Nacos（首选）
大型团队: Apollo
多数据中心: Consul
简单场景: Spring Cloud Config
```

## 三、注意事项

1. **Nacos 国内首选**
2. **Apollo 功能最全面**
3. **选择时考虑团队熟悉度**
4. **配置中心与注册中心最好统一**
5. **考虑与现有系统的集成**
