# 配置监听与刷新

## 一、配置监听

```java
@Component
public class ConfigListener {

    @NacosConfigListener(dataId = "app-config", group = "DEFAULT_GROUP")
    public void onConfigChange(String newConfig) {
        log.info("配置变更: {}", newConfig);
        // 重新加载配置
        reloadConfig(newConfig);
    }
}
```

## 二、注意事项

1. **配置变更要实时生效**
2. **监听器要做好异常处理**
3. **配置刷新要考虑线程安全**
4. **批量变更要合并处理**
5. **配置变更要有日志记录**
