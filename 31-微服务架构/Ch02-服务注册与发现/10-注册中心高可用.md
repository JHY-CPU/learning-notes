# 注册中心高可用

## 一、高可用架构

```yaml
Nacos 集群:
  节点1: nacos1:8848
  节点2: nacos2:8848
  节点3: nacos3:8848
  共识协议: Raft/JRaft

负载均衡:
  Nginx -> nacos1,nacos2,nacos3
  或
  VIP -> nacos1,nacos2,nacos3
```

## 二、注意事项

1. **注册中心至少 3 节点**
2. **客户端缓存服务列表**，注册中心挂了仍可工作
3. **定期备份注册数据**
4. **监控注册中心的健康状态**
5. **跨机房部署提高容灾能力**
