# 服务健康检查

## 一、健康检查方式

```java
// Spring Boot Actuator 健康端点
management:
  endpoints:
    web:
      exposure:
        include: health,info
  endpoint:
    health:
      show-details: always

// 自定义健康检查
@Component
public class DatabaseHealthIndicator implements HealthIndicator {
    @Override
    public Health health() {
        try {
            jdbcTemplate.queryForObject("SELECT 1", Integer.class);
            return Health.up().withDetail("database", "MySQL").build();
        } catch (Exception e) {
            return Health.down(e).build();
        }
    }
}
```

## 二、检查类型

| 类型 | 说明 | 频率 |
|------|------|------|
| 心跳检查 | 定期发送心跳 | 30 秒 |
| HTTP 检查 | 调用健康端点 | 10 秒 |
| TCP 检查 | 检查端口可达 | 5 秒 |
| 脚本检查 | 执行自定义脚本 | 按需 |

## 三、注意事项

1. **健康检查要检查所有依赖**
2. **检查频率要合理**，太频繁增加负载
3. **区分存活探针和就绪探针**
4. **K8s 中配置 liveness 和 readiness 探针**
5. **检查失败后要有自动恢复机制**
