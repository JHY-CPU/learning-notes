# 链路追踪最佳实践

## 一、实践清单

```yaml
配置:
  □ SkyWalking Agent 加载
  □ 采样率合理设置
  □ TraceID 全链路传递
  □ 日志关联 TraceID
  □ MQ 消息携带 TraceID

运维:
  □ OAP 高可用部署
  □ 存储用 ES 集群
  □ 数据定期清理
  □ 告警规则配置
  □ 监控 OAP 健康
```

## 二、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 链路断裂 | TraceID 丢失 | 检查 Header 传递 |
| 性能影响 | 采样率太高 | 降低采样率 |
| 存储不足 | 数据未清理 | 配置生命周期 |

## 三、注意事项

1. **SkyWalking 国内首选**
2. **TraceID 必须全链路传递**
3. **采样率要平衡性能和需求**
4. **日志要关联 TraceID**
5. **监控链路追踪系统本身**
