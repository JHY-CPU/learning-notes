# 告警规则

## 一、告警配置

```yaml
# SkyWalking 告警规则
rules:
  - name: response_time_rule
    expression: endpoint_avg > 1000
    period: 10
    count: 3
    message: "接口 {name} 平均响应时间超过 1 秒"

  - name: sla_rule
    expression: endpoint_sla < 95
    period: 10
    count: 2
    message: "接口 {name} SLA 低于 95%"
```

## 二、注意事项

1. **告警规则要合理设置**
2. **避免告警风暴**
3. **告警要有分级**
4. **告警要通知到责任人**
5. **定期审查告警规则**
