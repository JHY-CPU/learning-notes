# 链路追踪选型

## 一、方案对比

| 方案 | 侵入性 | 功能 | 性能 | 语言支持 |
|------|--------|------|------|---------|
| SkyWalking | 无侵入 | 全面 | 高 | 多语言 |
| Zipkin | 需埋点 | 基础 | 高 | 多语言 |
| Jaeger | 需埋点 | 中等 | 高 | 多语言 |
| Pinpoint | 无侵入 | 全面 | 中 | Java |

## 二、选型建议

```yaml
国内项目: SkyWalking（首选）
轻量级: Zipkin
云原生: Jaeger
Java 专用: Pinpoint
```

## 三、注意事项

1. **SkyWalking 国内首选**
2. **选择时考虑团队熟悉度**
3. **无侵入方案更适合存量系统**
4. **存储选择 ES 性能更好**
5. **考虑与日志、监控的集成**
