# 集成测试

## 一、Testcontainers

```java
@SpringBootTest
@Testcontainers
class OrderIntegrationTest {

    @Container
    static MySQLContainer<?> mysql = new MySQLContainer<>("mysql:8.0")
        .withDatabaseName("order_db")
        .withUsername("test")
        .withPassword("test");

    @Container
    static GenericContainer<?> redis = new GenericContainer<>("redis:7")
        .withExposedPorts(6379);

    @DynamicPropertySource
    static void configure(DynamicPropertyRegistry registry) {
        registry.add("spring.datasource.url", mysql::getJdbcUrl);
        registry.add("spring.redis.host", redis::getHost);
        registry.add("spring.redis.port",
            () -> redis.getMappedPort(6379));
    }

    @Autowired
    private OrderRepository orderRepository;

    @Test
    void shouldPersistOrderToDatabase() {
        Order order = new Order("C001", BigDecimal.valueOf(100));
        orderRepository.save(order);

        Optional<Order> found = orderRepository.findById(order.getId());
        assertThat(found).isPresent();
        assertThat(found.get().getCustomerId()).isEqualTo("C001");
    }
}
```

## 二、嵌入式替代方案

```java
// H2 内存数据库 - 轻量级测试
@SpringBootTest
@AutoConfigureTestDatabase(replace = Replace.NONE)
@TestPropertySource(properties = {
    "spring.datasource.url=jdbc:h2:mem:testdb",
    "spring.jpa.hibernate.ddl-auto=create-drop"
})
class LightWeightIntegrationTest { }

// Embedded Kafka - 消息集成测试
@SpringBootTest
@EmbeddedKafka(partitions = 1, topics = {"order-events"})
class KafkaIntegrationTest {

    @Autowired
    private KafkaTemplate<String, String> kafkaTemplate;

    @Test
    void shouldProduceAndConsumeMessage() {
        kafkaTemplate.send("order-events", "{\"id\":\"O001\"}");
        // 验证消费者接收到消息
    }
}
```

## 三、注意事项

1. **集成测试用容器保证环境一致性**
2. **测试之间要隔离**，每个测试独立数据
3. **合理使用 @Transactional 回滚**
4. **集成测试较慢，不要在每次构建都跑**
5. **CI 环境要支持 Docker**
