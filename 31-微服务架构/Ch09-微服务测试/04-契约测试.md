# 契约测试

## 一、Pact 契约测试

```java
// 消费者端 - 定义期望
@PactTestFor(pactVersion = PactVer)
@SpringBootTest
class OrderServiceConsumerPactTest {

    @Pact(consumer = "order-service", provider = "inventory-service")
    public RequestResponsePact checkStockPact(PactDslWithProvider builder) {
        return builder
            .given("商品P001库存为10")
            .uponReceiving("检查库存请求")
            .path("/api/inventory/check")
            .query("productId=P001&quantity=2")
            .method("GET")
            .willRespondWith()
            .status(200)
            .body("{\"available\": true}")
            .toPact();
    }

    @Test
    @PactTestFor(pactMethod = "checkStockPact")
    void testCheckStock(MockServer mockServer) {
        InventoryClient client = new InventoryClient(mockServer.getUrl());
        boolean available = client.check("P001", 2);
        assertThat(available).isTrue();
    }
}
```

```java
// 提供者端 - 验证契约
@Provider("inventory-service")
@PactBroker(url = "http://pact-broker:9292")
@SpringBootTest(webEnvironment = SpringBootTest.WebEnvironment.RANDOM_PORT)
class InventoryServiceProviderPactTest {

    @TestTemplate
    @ExtendWith(PactVerificationInvocationContextProvider.class)
    void pactVerificationTestTemplate(PactVerificationContext context) {
        context.verifyInteraction();
    }

    @State("商品P001库存为10")
    void setupInventoryState() {
        inventoryRepository.save(new Inventory("P001", 10));
    }
}
```

## 二、Spring Cloud Contract

```groovy
// Contract DSL (Groovy)
Contract.make {
    request {
        method 'GET'
        url '/api/inventory/check?productId=P001&quantity=2'
    }
    response {
        status 200
        body(available: true)
        headers {
            contentType(applicationJson())
        }
    }
}
```

## 三、注意事项

1. **契约测试保证服务间接口兼容**
2. **消费者驱动契约是主流模式**
3. **契约要版本化管理**
4. **Pact Broker 集中管理契约文件**
5. **CI 流水线中自动验证契约**
