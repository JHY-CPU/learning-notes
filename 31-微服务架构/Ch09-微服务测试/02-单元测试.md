# 单元测试

## 一、JUnit5 + Mockito

```java
@SpringBootTest
class OrderServiceTest {

    @MockBean
    private PaymentClient paymentClient;

    @MockBean
    private InventoryClient inventoryClient;

    @Autowired
    private OrderService orderService;

    @Test
    @DisplayName("创建订单成功")
    void shouldCreateOrderSuccessfully() {
        // Given
        OrderRequest request = new OrderRequest("P001", 2);
        Mockito.when(inventoryClient.check("P001", 2))
               .thenReturn(true);
        Mockito.when(paymentClient.deduct(any()))
               .thenReturn(new PaymentResult(true));

        // When
        Order order = orderService.createOrder(request);

        // Then
        assertThat(order.getStatus()).isEqualTo(OrderStatus.CREATED);
        Mockito.verify(inventoryClient).deduct("P001", 2);
    }

    @Test
    @DisplayName("库存不足时应抛出异常")
    void shouldThrowWhenInsufficientStock() {
        Mockito.when(inventoryClient.check("P001", 100))
               .thenReturn(false);

        assertThatThrownBy(() -> orderService.createOrder(
            new OrderRequest("P001", 100)))
            .isInstanceOf(InsufficientStockException.class);
    }
}
```

## 二、参数化测试

```java
@ParameterizedTest
@ValueSource(ints = {0, -1, -100})
@DisplayName("数量不能为非正数")
void shouldRejectNonPositiveQuantity(int quantity) {
    assertThatThrownBy(() -> orderService.createOrder(
        new OrderRequest("P001", quantity)))
        .isInstanceOf(IllegalArgumentException.class);
}

@ParameterizedTest
@CsvSource({"P001, 2, 100.00", "P002, 5, 250.00"})
void shouldCalculateTotalPrice(String productId, int qty, BigDecimal expected) {
    Order order = orderService.createOrder(new OrderRequest(productId, qty));
    assertThat(order.getTotalPrice()).isEqualByComparingTo(expected);
}
```

## 三、注意事项

1. **一个测试只验证一个行为**
2. **Given-When-Then 结构清晰**
3. **Mock 交互要验证**（verify）
4. **测试命名要描述业务语义**
5. **避免过度 Mock**，只 Mock 外部依赖
