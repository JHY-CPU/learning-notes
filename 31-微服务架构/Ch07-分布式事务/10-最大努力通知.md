# 最大努力通知

## 一、方案说明

```java
// 支付回调场景
public void onPaymentSuccess(Payment payment) {
    // 1. 更新本地状态
    orderMapper.updateStatus(payment.getOrderId(), "PAID");

    // 2. 通知下游服务
    notifyService.notify(payment);

    // 3. 定时重试通知
    @Scheduled(fixedDelay = 5000)
    public void retryNotify() {
        List<Notification> pending = notificationMapper.selectPending();
        for (Notification n : pending) {
            try {
                httpClient.post(n.getUrl(), n.getBody());
                notificationMapper.updateStatus(n.getId(), "SUCCESS");
            } catch (Exception e) {
                notificationMapper.incrementRetry(n.getId());
            }
        }
    }
}
```

## 二、注意事项

1. **适合跨系统通知场景**
2. **不要求强一致，允许通知失败**
3. **通知要有重试机制**
4. **接收方要幂等处理**
5. **通知次数有限制**
