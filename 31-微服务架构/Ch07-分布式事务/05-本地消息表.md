# 本地消息表

## 一、实现方案

```java
@Transactional
public void deductInventory(Order order) {
    // 本地事务
    inventoryMapper.deduct(order);
    // 保存消息记录
    messageMapper.insert(new Message("inventory-deducted", order));
}

// 定时投递
@Scheduled(fixedDelay = 1000)
public void deliverMessages() {
    List<Message> pending = messageMapper.selectPending(100);
    for (Message msg : pending) {
        kafkaTemplate.send(msg.getTopic(), msg.getContent());
        messageMapper.updateStatus(msg.getId(), "SENT");
    }
}
```

## 二、注意事项

1. **本地消息表是最简单的方案**
2. **消息投递要保证幂等**
3. **定时任务要合理配置**
4. **失败消息要有重试机制**
5. **适合大多数业务场景**
