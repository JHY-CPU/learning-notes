# 54-形式化方法

形式化方法使用数学技术对软件系统进行规约、设计和验证，以提高软件的正确性和可靠性。本模块覆盖形式化规约、模型检验和合约式设计三大主题。

## 目录结构

| 子目录 | 主题 | 说明 |
|--------|------|------|
| [01-形式化规约](./01-形式化规约/README.md) | 形式化规约 | Z语言、VDM、B方法、Alloy、OCL |
| [02-模型检验](./02-模型检验/README.md) | 模型检验 | 时序逻辑、SPIN、NuSMV、TLA+ |
| [03-合约式设计](./03-合约式设计/README.md) | 合约式设计 | 前置/后置条件、不变量、Eiffel、JML |

## 学习路径

1. **形式化规约**：学习用数学语言精确描述系统需求
2. **模型检验**：学习自动化验证系统行为是否满足规约
3. **合约式设计**：学习在代码中嵌入形式化约束
