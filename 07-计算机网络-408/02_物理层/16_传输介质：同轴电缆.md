# 16_传输介质：同轴电缆

## 核心概念

- **同轴电缆（Coaxial Cable）**：由内导体、绝缘层、外导体（屏蔽层）和外护套组成的同轴结构
- "同轴"指内外导体在同一中心轴线上
- 曾是以太网的主要传输介质，现在逐渐被双绞线和光纤取代

### 同轴电缆结构

```
        ┌────── 外护套
        │ ┌──── 外导体（屏蔽层/编织网）
        │ │ ┌── 绝缘层
        │ │ │ ┌ 内导体（铜芯线）
        │ │ │ │
    ════╪═╪═╪═╪════
        ════════
```

### 同轴电缆分类

| 类型 | 阻抗 | 特性阻抗 | 典型应用 |
|------|------|----------|----------|
| 基带同轴电缆 | 50Ω | 数字信号 | 早期以太网（10BASE2/10BASE5） |
| 宽带同轴电缆 | 75Ω | 模拟信号 | 有线电视（CATV） |

### 常见以太网标准

- **10BASE5**：粗缆，50Ω，最大段长500m，每段最多100个节点
- **10BASE2**：细缆，50Ω，最大段长185m，每段最多30个节点
- "10"=10Mbit/s，"BASE"=基带，"5/2"=约500/185米

## 原理分析

### 同轴电缆的传输特性

- **带宽**：比双绞线更高（可达数百MHz到数GHz）
- **抗干扰**：外导体屏蔽层提供良好的抗干扰能力
- **衰减**：随频率和距离增加
- **阻抗匹配**：终端必须接匹配电阻（50Ω或75Ω），否则信号反射

### 阻抗匹配原理

- 同轴电缆有固定的**特性阻抗**（50Ω或75Ω）
- 电缆末端必须接等值电阻，否则信号会**反射**
- 反射信号与原信号叠加会造成干扰
- 这就是为什么10BASE2和10BASE5的总线两端必须有**终端电阻**

### 同轴电缆的连接方式

- **总线拓扑**：所有设备接在同一根电缆上
- 使用**T型连接器**连接设备
- 通过**BNC接头**实现物理连接

## 直观理解

### 类比：水管

- 同轴电缆好比一根有内外两层管壁的水管
- 内管传输水流（信号），外管起保护和屏蔽作用
- 如果水管末端封死（没有终端电阻），水流会反弹（信号反射）
- 末端要接一个泄压阀（终端电阻），让水流平稳地流走

### 为什么逐渐被淘汰

- 比双绞线**贵**
- 比双绞线**粗重**，安装不便
- 总线拓扑，一台设备故障可能影响全网
- 被星形拓扑（交换机+双绞线）取代

## 协议关联

### 与以太网的关联

- 10BASE5 和 10BASE2 是最早的以太网标准
- 使用同轴电缆的总线拓扑
- 现代以太网使用双绞线（10BASE-T等）和光纤

### 与有线电视的关联

- CATV使用75Ω宽带同轴电缆
- 支持频分复用（FDM），可同时传多路电视信号
- Cable Modem利用同轴电缆提供宽带上网

### 考试要点

1. 50Ω用于数字传输（基带），75Ω用于模拟传输（宽带）
2. 需要阻抗匹配（终端电阻）
3. 10BASE5和10BASE2的参数
4. 同轴电缆的结构（内导体、绝缘层、外导体、外护套）
