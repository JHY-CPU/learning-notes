# 46_物理层协议对比_考试

## 核心概念

### 物理层协议选择题高频考点

1. EIA-232的电气特性（逻辑0为正电压）
2. RS-485的差分传输和多点能力
3. 各标准的四大特性对应关系
4. 物理层标准与数据链路层协议的区分

### 常见标准混淆辨析

| 容易混淆的 | 区分要点 |
|-----------|----------|
| EIA-232 vs RS-232 | 同一标准，不同命名时期 |
| EIA-232 vs RS-485 | 单端 vs 差分 |
| V.24 vs V.28 | V.24=功能/规程，V.28=电气 |
| 物理层标准 vs 数据链路层协议 | 只定义接口 vs 定义帧格式 |

### 考试中可能遇到的其他标准

| 标准 | 说明 |
|------|------|
| 10BASE-T | 10M以太网，双绞线，曼彻斯特编码 |
| 100BASE-TX | 100M以太网，5类线，4B/5B编码 |
| 1000BASE-T | 千兆以太网，超5类线，PAM-5编码 |
| SONET/SDH | 光纤同步传输标准 |
| USB | 通用串行总线 |
| IEEE 802.11 | WiFi无线局域网 |

## 原理分析

### 典型考题

**题目**：EIA-232标准中，逻辑"0"对应的电压范围是？
A. -3V ~ -15V  B. +3V ~ +15V  C. 0V ~ +5V  D. -5V ~ +5V

**答案**：B。EIA-232中逻辑"0"（空号）对应正电压，逻辑"1"（传号）对应负电压。

**题目**：下列物理层标准中，使用差分信号传输的是？
A. EIA-232  B. RS-485  C. V.28  D. 10BASE-T

**答案**：B。RS-485使用差分（平衡）信号传输。注意：10BASE-T虽然使用双绞线，但题目问的是标准层面。

**题目**：关于EIA-232和RS-485的比较，以下说法正确的是？
A. RS-485最大传输距离小于EIA-232
B. EIA-232支持多点通信
C. RS-485的抗干扰能力优于EIA-232
D. EIA-232使用差分信号

**答案**：C。RS-485使用差分信号，抗干扰能力更强。RS-485最大距离1200m > EIA-232的15m。

**题目**：物理层的四大接口特性中，规定引脚排列顺序的是？
A. 电气特性  B. 功能特性  C. 机械特性  D. 规程特性

**答案**：C。引脚排列属于机械特性。

### 综合判断题

**题目**：某工厂需要在100m距离上连接32个设备进行通信，应选择哪种物理层标准？

**答案**：RS-485。理由：
- 距离100m > EIA-232的15m，排除EIA-232
- 需要多点通信（32个设备），RS-485支持最多32个节点
- RS-485的差分信号在工业环境中抗干扰好

## 直观理解

### 快速判断表

```
需求判断：
├─ 短距离(<15m) + 点对点 → EIA-232
├─ 长距离(>100m) + 多点 → RS-485
├─ 数字线路 → X.21
├─ 局域网(100m) → 以太网(10BASE-T等)
├─ 无线 → 802.11 WiFi
└─ 光纤长距离 → SONET/SDH
```

## 协议关联

### 物理层与其他层的职责划分

| 功能 | 物理层 | 数据链路层 |
|------|--------|-----------|
| 信号编码 | 负责 | 不涉及 |
| 帧格式 | 不涉及 | 负责 |
| 差错检测 | 不涉及 | 负责 |
| 流量控制 | 不涉及 | 负责 |
| 接口特性 | 负责 | 不涉及 |

### 常见陷阱

1. **EIA-232逻辑0为正电压**——与直觉相反
2. **RS-485支持多点通信**——EIA-232只支持点对点
3. **物理层不负责差错控制**——这是数据链路层的功能
4. **机械特性包括引脚排列**——不是功能特性
5. **V.24和V.28是配合使用的**——V.24定义功能/规程，V.28定义电气
