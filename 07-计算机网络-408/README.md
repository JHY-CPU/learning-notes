## 建议学习顺序

408计算机网络在考试中约占25分，建议按照以下顺序学习：

1. **第一章 体系结构** -- 建立整体框架，理解分层模型和基本概念
2. **第二章 物理层** -- 掌握编码调制、奈氏准则、香农定理（常考计算题）
3. **第三章 数据链路层** -- 重点掌握滑动窗口、CSMA/CD、以太网、VLAN（高频考点）
4. **第四章 网络层** -- 最重要的章节，IP地址、子网划分、路由协议必考
5. **第五章 传输层** -- TCP三次握手/四次挥手、拥塞控制是核心考点
6. **第六章 应用层** -- DNS、HTTP、FTP等协议的工作原理
7. **第七章 补充专题** -- 网络安全、QoS、跨层综合题

每章笔记均包含「概念学习 -> 计算练习 -> 真题演练」三部分，建议按此顺序推进。

## 常用考试公式速查

### 物理层

| 公式 | 说明 |
|:---|:---|
| **奈氏准则**：$C = 2W \log_2 V$ (bps) | W为带宽(Hz)，V为信号电平数 |
| **香农定理**：$C = W \log_2(1 + S/N)$ (bps) | W为带宽(Hz)，S/N为信噪比 |
| **信噪比(dB)**：$10 \log_{10}(S/N)$ | 将比值转换为分贝 |
| **时延带宽积**：$传播时延 \times 带宽$ | 以比特为单位 |

### 数据链路层

| 公式 | 说明 |
|:---|:---|
| **信道利用率**：$\frac{T_d}{T_d + RTT + T_a}$ | Td为发送时延 |
| **纯ALOHA吞吐量**：$S = G \cdot e^{-2G}$ | 最大 S = 1/2e ≈ 0.184 |
| **时隙ALOHA吞吐量**：$S = G \cdot e^{-G}$ | 最大 S = 1/e ≈ 0.368 |
| **最小帧长**：$L_{min} = 2 \times 传播时延 \times 数据率$ | CSMA/CD |

### 网络层

| 公式 | 说明 |
|:---|:---|
| **CIDR主机数**：$2^n - 2$ | n为主机位数，减去网络和广播 |
| **IP分片**：每片数据部分为8的倍数 | 除最后一片外 |
| **RIP最大跳数**：15跳 | 16表示不可达 |

### 传输层

| 公式 | 说明 |
|:---|:---|
| **TCP序号**：32位，循环使用 | 理解序号与确认号的关系 |
| **拥塞窗口变化**：慢开始指数增长，拥塞避免线性增长 | 画图分析 |
| **发送窗口**：$min(拥塞窗口, 接收窗口)$ | 流量控制与拥塞控制结合 |
