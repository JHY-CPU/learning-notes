# 61_应用层端口号_考试

## 核心概念

- **408常考题型**：选择题为主，必须熟记端口号
- **高频考点**：常用协议的端口号
- **出题规律**：直接考端口号或给出端口号判断协议

## 原理分析

### 典型考题1：端口号匹配

**题目**：HTTP使用的端口号是（  ）
A. 21
B. 25
C. 53
D. 80

**答案**：D

### 典型考题2：协议判断

**题目**：端口号25对应的协议是（  ）
A. FTP
B. SMTP
C. POP3
D. DNS

**答案**：B

### 典型考题3：UDP端口

**题目**：以下使用UDP的协议是（  ）
A. HTTP
B. FTP
C. DNS
D. SMTP

**答案**：C

**解析**：
- HTTP、FTP、SMTP都使用TCP
- DNS主要使用UDP

### 典型考题4：端口填空

**题目**：完成以下表格：
| 协议 | 端口号 |
|------|--------|
| FTP控制 | ? |
| SMTP | ? |
| POP3 | ? |
| DNS | ? |
| HTTPS | ? |

**答案**：
| 协议 | 端口号 |
|------|--------|
| FTP控制 | 21 |
| SMTP | 25 |
| POP3 | 110 |
| DNS | 53 |
| HTTPS | 443 |

## 直观理解

**做题技巧**：
- "21控制20数据，25发邮件，53域名"
- "80看网页，443安全"
- "110取邮件，143也取"
- "67/68动态IP，161/162管理"

## 协议关联

- **端口与传输层**：端口标识应用层服务
- **端口与应用层**：每个协议有固定端口
- **408必考**：端口号是送分题，必须记住
