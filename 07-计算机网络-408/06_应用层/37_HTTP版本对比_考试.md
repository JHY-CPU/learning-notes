# 37_HTTP版本对比_考试

## 核心概念

- **408常考题型**：选择题为主，涉及HTTP各版本特点对比
- **关键考点**：
  - HTTP/1.0默认非持久，HTTP/1.1默认持久
  - HTTP/1.1必须包含Host头部
  - HTTP/2多路复用、头部压缩
  - 各版本解决的问题和遗留问题
- **易混淆点**：
  - HTTP/2解决了HTTP层队头阻塞，但TCP层仍有

## 原理分析

### 典型考题1：版本特性

**题目**：HTTP/1.1相比HTTP/1.0的主要改进是（  ）（多选）
A. 支持持久连接
B. 支持Host头部
C. 支持管道化
D. 使用UDP协议

**答案**：A, B, C

**解析**：
- HTTP/1.1默认持久连接（A正确）
- HTTP/1.1必须包含Host头部（B正确）
- HTTP/1.1支持管道化（C正确）
- HTTP/1.1仍使用TCP，不是UDP（D错误）

### 典型考题2：HTTP/2特性

**题目**：HTTP/2的新特性包括（  ）（多选）
A. 多路复用
B. 头部压缩
C. 服务器推送
D. 使用UDP协议

**答案**：A, B, C

**解析**：
- HTTP/2支持多路复用（A正确）
- HTTP/2支持HPACK头部压缩（B正确）
- HTTP/2支持服务器推送（C正确）
- HTTP/2仍使用TCP，不是UDP（D错误）

### 典型考题3：队头阻塞

**题目**：HTTP/1.1管道化存在队头阻塞问题，HTTP/2（  ）
A. 完全解决了队头阻塞
B. 解决了HTTP层队头阻塞
C. 没有解决队头阻塞
D. 增加了队头阻塞

**答案**：B

**解析**：
- HTTP/2多路复用解决了HTTP层的队头阻塞
- 但TCP层的队头阻塞仍然存在
- HTTP/3基于QUIC（UDP）才能完全解决

### 典型考题4：Host头部

**题目**：HTTP/1.1中Host头部是（  ）
A. 可选的
B. 必需的
C. 已废弃的
D. 不支持的

**答案**：B

## 直观理解

**做题技巧**：
- HTTP/1.0 = 非持久，无Host
- HTTP/1.1 = 持久，有Host，管道化
- HTTP/2 = 多路复用，头部压缩，服务器推送
- HTTP/2仍用TCP，HTTP/3用UDP

**常见错误**：
- 误认为HTTP/2使用UDP
- 误认为HTTP/2完全解决了队头阻塞
- 忘记HTTP/1.1必须包含Host头部

## 协议关联

- **HTTP版本与TCP**：HTTP/1.1和HTTP/2都使用TCP
- **HTTP/2与队头阻塞**：解决HTTP层阻塞，TCP层仍存在
- **HTTP/3与UDP**：HTTP/3基于QUIC（UDP）
- **408常见组合**：HTTP版本对比 + RTT计算 = 综合分析
