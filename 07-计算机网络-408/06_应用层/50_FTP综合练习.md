# 50_FTP综合练习

## 核心概念

- **FTP综合练习题型**：涉及FTP端口、连接过程、主动/被动模式
- **关键考点**：
  - 控制连接：端口21
  - 数据连接：端口20（主动模式）
  - FTP使用TCP
  - FTP是有状态协议
- **408考试重点**：FTP的双连接机制、主动/被动模式

## 原理分析

### 综合题1：FTP连接分析

**题目**：FTP客户端连接服务器，描述主动模式下的连接建立过程。

**参考答案**：
1. 客户端随机端口N → 服务器21端口：建立控制连接（TCP三次握手）
2. 客户端发送PORT命令，指定数据端口N+1
3. 服务器20端口 → 客户端N+1端口：建立数据连接（TCP三次握手）
4. 通过数据连接传输数据
5. 数据传输完毕，关闭数据连接
6. 控制连接保持，可进行后续操作

### 综合题2：主动与被动模式对比

**题目**：比较FTP主动模式和被动模式的数据连接建立方式。

**参考答案**：
| 特性 | 主动模式 | 被动模式 |
|------|---------|---------|
| 数据连接发起方 | 服务器 | 客户端 |
| 服务器数据端口 | 20 | 随机高端口 |
| 命令 | PORT | PASV |
| 适用场景 | 无防火墙 | 有防火墙 |

### 综合题3：FTP与HTTP对比

**题目**：从连接方式、状态维护、端口号三个方面比较FTP和HTTP。

**参考答案**：
| 特性 | FTP | HTTP |
|------|-----|------|
| 连接方式 | 双连接（控制+数据） | 单连接 |
| 状态维护 | 有状态 | 无状态 |
| 端口号 | 20/21 | 80 |
| 传输层 | TCP | TCP |

### 综合题4：端口分析

**题目**：在FTP主动模式中，数据连接的源端口和目的端口分别是多少？

**参考答案**：
- 源端口：20（服务器）
- 目的端口：客户端指定的端口（如N+1）
- 控制连接：客户端随机端口 → 服务器21

## 直观理解

**做题技巧**：
- 控制连接 = 端口21（始终存在）
- 数据连接 = 端口20（主动模式）
- 主动 = 服务器连客户端
- 被动 = 客户端连服务器

**常见错误**：
- 混淆控制连接和数据连接的端口
- 忘记被动模式数据端口是随机的
- 误认为FTP使用UDP

## 协议关联

- **FTP与TCP**：FTP使用TCP的可靠传输
- **FTP与HTTP**：FTP双连接有状态，HTTP单连接无状态
- **408常见组合**：FTP连接过程 + TCP三次握手 = 综合分析
