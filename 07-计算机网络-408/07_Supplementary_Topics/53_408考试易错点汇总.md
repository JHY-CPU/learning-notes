# 53_408考试易错点汇总

## 核心概念

- **本文件汇总408计算机网络考试中最常见的易错知识点**
- **按专题分类整理，考前快速复习**

## 原理分析

### 专题一：网络安全易错点

| 易错说法 | 正确理解 |
|---------|---------|
| 流量分析是主动攻击 | 错，流量分析是被动攻击 |
| 数字签名保证机密性 | 错，签名保证认证性/完整性/不可否认性 |
| 加密保证完整性 | 错，加密保证机密性，散列/MAC保证完整性 |
| 证书包含私钥 | 错，证书只包含公钥 |
| CA用公钥签名证书 | 错，CA用私钥签名 |
| AH提供加密功能 | 错，AH只提供完整性+认证 |
| IDS可以阻止攻击 | 错，IDS只检测，IPS才阻止 |

### 专题二：传输层易错点

| 易错说法 | 正确理解 |
|---------|---------|
| TCP提供可靠传输，所以不会丢包 | 错，TCP通过重传机制恢复丢包，不是不丢 |
| UDP完全不可靠 | 错，UDP只是不保证可靠，应用可自行实现可靠性 |
| TCP连接由端口号唯一标识 | 错，由四元组（源IP+源端口+目的IP+目的端口）标识 |
| 拥塞控制和流量控制是一回事 | 错，流量控制是端到端，拥塞控制是全局性 |
| 超时和3个重复ACK处理相同 | 错，超时→慢启动，3重复ACK→快恢复 |

### 专题三：网络层易错点

| 易错说法 | 正确理解 |
|---------|---------|
| ARP工作在网络层 | 错，ARP直接封装在以太网帧中（链路层辅助协议） |
| ICMP是传输层协议 | 错，ICMP封装在IP中，是网络层协议 |
| 路由器每个端口有独立IP | 对，路由器每个接口有独立IP地址 |
| 子网掩码可以不连续 | 标准子网掩码必须连续（408一般不考虑非连续） |

### 专题四：数据链路层易错点

| 易错说法 | 正确理解 |
|---------|---------|
| 交换机工作在网络层 | 错，交换机根据MAC转发，工作在链路层 |
| CSMA/CD用于无线网络 | 错，无线用CSMA/CA |
| 最小帧长无关紧要 | 错，最小帧长保证冲突能被检测到 |
| PPP需要CSMA/CD | 错，PPP是点对点，不需要多路访问控制 |

### 专题五：应用层易错点

| 易错说法 | 正确理解 |
|---------|---------|
| DNS只用UDP | 错，区域传输和大响应用TCP |
| HTTPS = 安全的HTTP协议 | 更准确：HTTPS = HTTP over TLS over TCP |
| SNMP使用TCP | 错，SNMP使用UDP（161/162） |
| SMTP只能发文本邮件 | 错，MIME扩展支持附件等非文本内容 |

## 直观理解

- **记忆方法**：每个易错点编一个"纠正口诀"
  - "流量分析是被动，主动攻击会修改"
  - "签名不加密，加密不签名"
  - "证书有公钥，私钥自己留"
  - "AH只验不密，ESP又验又密"
  - "超时回慢启动，重复ACK快恢复"
  - "ARP在链路帮IP，ICMP装在IP里"

## 协议关联

- **跨层易错**：ARP（链路层）为IP（网络层）服务
- **安全易错**：签名和加密的密钥使用方向相反
- **TCP易错**：流量控制 vs 拥塞控制
- **408考法**：易错点是选择题的高频陷阱
