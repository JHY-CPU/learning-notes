# 10_特殊IP地址_考试

## 核心概念

- **408常考特殊地址类型**：
  1. 网络地址（主机号全0）
  2. 广播地址（主机号全1）
  3. 回环地址（127.x.x.x）
  4. 私有地址（10.x.x.x, 172.16-31.x.x, 192.168.x.x）
  5. 0.0.0.0（本网络本主机）
  6. 255.255.255.255（受限广播）

- **考查形式**：
  - 判断某地址能否作为源/目的地址
  - 计算网络地址和广播地址
  - 判断特殊地址的含义

## 原理分析

### 真题类型分析

**题型1：地址合法性判断**

问：以下哪个IP地址可以分配给主机？
A. 192.168.1.0（C类网络地址）
B. 192.168.1.255（C类广播地址）
C. 192.168.1.1（合法主机地址）✓
D. 127.0.0.1（回环地址，不在网络上使用）

答案：C

**题型2：源地址和目的地址**

问：以下说法正确的是？
A. 255.255.255.255可以作为源地址
B. 0.0.0.0可以作为目的地址
C. 主机号全1的地址可以作为目的地址（用于广播）✓
D. 127.x.x.x可以出现在网络传输中

答案：C
- A错：受限广播不能作源地址
- B错：0.0.0.0不能作目的地址
- D错：127.x.x.x只在本机内部，不出现在网络上

**题型3：子网内地址计算**

问：某C类网络192.168.5.0/24，可用IP地址范围？
- 网络地址：192.168.5.0
- 广播地址：192.168.5.255
- 可用范围：192.168.5.1 ~ 192.168.5.254
- 可用数：$2^8 - 2 = 254$

**题型4：私有地址判断**

问：以下哪些是私有地址？（多选）
A. 10.1.1.1 ✓
B. 172.20.1.1 ✓
C. 172.15.1.1（B类私有是172.16~172.31）✗
D. 192.168.100.1 ✓

### 特殊地址全表速查

| 地址 | 网络号 | 主机号 | 用途 | 源 | 目的 |
|------|--------|--------|------|-----|------|
| 0.0.0.0 | 全0 | 全0 | 本主机 | ✓ | ✗ |
| 网络地址 | 特定 | 全0 | 标识网络 | ✗ | ✗ |
| 直接广播 | 特定 | 全1 | 特定网络广播 | ✗ | ✓ |
| 受限广播 | 全1 | 全1 | 本网络广播 | ✗ | ✓ |
| 127.x.x.x | 01111111 | 任意 | 回环测试 | ✓ | ✓ |

## 直观理解

**记忆技巧**：

"三不原则"：
- 网络地址（全0）**不能**分配给主机
- 广播地址（全1）**不能**分配给主机
- 回环地址（127）**不能**出现在网络上

"私有地址口诀"：
- A类私有：10（一个A类网络）
- B类私有：172.16到172.31（16个B类网络）
- C类私有：192.168（256个C类网络）

## 协议关联

### 各协议使用的特殊地址

- **DHCP Discover**：源0.0.0.0，目的255.255.255.255
- **ARP请求**：目的MAC为FF:FF:FF:FF:FF:FF（广播）
- **ICMP回显请求/应答**：使用127.0.0.1测试本地协议栈
- **RIP**：使用224.0.0.9（D类组播地址）

### 考试陷阱
- "10.0.0.1可以用于Internet通信" → 错，10.x.x.x是私有地址
- "网络地址可以作为分组的目的地址" → 错，网络地址标识网络，不转发到主机
