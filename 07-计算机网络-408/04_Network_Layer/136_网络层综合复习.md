# 网络层综合复习

## 核心概念

### 408网络层必考知识点清单
1. **IP地址与子网划分**
   - A/B/C类地址范围，私有地址范围
   - 子网掩码计算，可用主机数 $2^n - 2$
   - VLSM（可变长子网掩码）

2. **CIDR与路由聚合**
   - CIDR记法（IP/前缀长度）
   - 路由聚合：找最长公共前缀
   - 最长前缀匹配

3. **IP数据报格式**
   - 首部各字段含义（版本、首部长度、TTL、协议、校验和等）
   - 分片相关字段：标识、标志（DF/MF）、片偏移

4. **IP分片计算**
   - 每片数据部分必须是8字节的整数倍
   - 片偏移以8字节为单位

5. **路由协议**
   - RIP：距离向量，UDP 520，最大15跳
   - OSPF：链路状态，Dijkstra，IP协议号89
   - BGP：路径向量，TCP 179，AS-PATH防环

6. **ARP**
   - 广播请求，单播响应
   - 跨子网时解析网关MAC

7. **ICMP**
   - 差错报告和查询报文
   - ping（回显请求/应答）、traceroute（TTL超时）

8. **DHCP**
   - 四步交互：Discover→Offer→Request→ACK
   - UDP 67/68

9. **IPv6**
   - 128位地址，8组4位十六进制
   - 缩写规则（省略前导零，::替代连续全零组）
   - 固定40字节首部，取消校验和和路由器分片
   - 过渡技术：双栈、隧道、NAT64

10. **NAT**
    - 静态NAT、动态NAT、NAPT/PAT
    - NAPT使用端口号区分不同连接

## 原理分析

### 常考公式汇总
- 子网可用主机数：$2^n - 2$（n为主机位数）
- RIP最大跳数：15（16不可达）
- OSPF链路代价：Cost = 参考带宽 / 接口带宽
- NAPT最大连接数：约 $2^{16} = 65536$（每端口）
- CIDR聚合条件：连续地址块，公共前缀相同

### 易混淆概念
| 易混淆 | 区别 |
|--------|------|
| 子网划分 vs CIDR | 子网是大变小，CIDR是小聚大 |
| RIP vs OSPF | 距离向量 vs 链路状态 |
| ARP vs RARP | IP→MAC vs MAC→IP |
| AH vs ESP | 认证不加密 vs 加密+认证 |
| 传输模式 vs 隧道模式 | 端到端 vs VPN |
| SLAAC vs DHCPv6 | 无状态 vs 有状态 |

## 直观理解
- 网络层的核心是IP协议，所有其他协议都为IP服务
- 路由器是网络层的核心设备，负责路由选择和分组转发
- 地址解析（ARP）是网络层和数据链路层的桥梁

## 协议关联
- 网络层为传输层提供尽力而为的服务（Best Effort）
- TCP的拥塞控制与网络层的拥塞控制相互配合
- IPsec为IP数据报提供安全保护
- SDN将网络层的控制平面和数据平面分离
