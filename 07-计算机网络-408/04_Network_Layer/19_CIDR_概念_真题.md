# 19_CIDR_概念_真题

## 核心概念

- **408 CIDR真题考查形式**：
  - CIDR地址块范围计算
  - 路由聚合
  - 地址分配
  - 判断IP是否属于某地址块

## 原理分析

### 真题1：CIDR地址块范围

**题**：某ISP分配到地址块202.100.16.0/20，求可分配的IP地址范围和总地址数。

**解**：
- /20：前20位固定
- 最小地址：202.100.16.0
- 最大地址：202.100.31.255（16+15=31）
- 总地址数：$2^{12} = 4096$
- 可分配给主机：$4096 - 2 = 4094$

### 真题2：路由聚合

**题**：路由器R的4个接口分别连接以下网络：
- 接口1：192.168.0.0/24
- 接口2：192.168.1.0/24
- 接口3：192.168.2.0/24
- 接口4：192.168.3.0/24

路由器R向外部通告的聚合路由条目是什么？

**解**：
- 0 = 000000**00**, 1 = 000000**01**, 2 = 000000**10**, 3 = 000000**11**
- 前6位相同（000000）
- 共同前缀长度 = 8 + 8 + 6 = 22位
- 聚合结果：**192.168.0.0/22**

### 真题3：地址分配

**题**：某ISP拥有地址块200.10.0.0/16，需要分配给3个部门：X需要4096个地址，Y需要2048个地址，Z需要1024个地址。给出分配方案。

**解**：
- X: 4096 = $2^{12}$，需要/20
- Y: 2048 = $2^{11}$，需要/21
- Z: 1024 = $2^{10}$，需要/22

分配（从大到小，地址连续）：
| 部门 | 地址块 | 范围 |
|------|--------|------|
| X | 200.10.0.0/20 | 200.10.0.0~200.10.15.255 |
| Y | 200.10.16.0/21 | 200.10.16.0~200.10.23.255 |
| Z | 200.10.24.0/22 | 200.10.24.0~200.10.27.255 |

### 真题4：判断归属

**题**：目的地址172.16.50.100，路由表中有以下条目：
- 172.16.0.0/16 → 接口1
- 172.16.48.0/20 → 接口2
- 172.16.50.0/24 → 接口3

应从哪个接口转发？（最长前缀匹配）

**解**：
- 匹配/16：172.16.0.0/16 ✓
- 匹配/20：172.16.48.0/20，50在48~63之间 ✓
- 匹配/24：172.16.50.0/24，50=50 ✓
- 最长前缀匹配：**/24 → 接口3**

## 直观理解

**聚合步骤**：
1. 写出各地址的二进制
2. 从左到右比较，找到第一个不同的位
3. 共同前缀长度 = 相同的位数
4. 聚合地址 = 共同前缀 + 全0

**分配原则**：
- "先大后小，地址连续，不重叠"

## 协议关联

- CIDR路由聚合 → 减少Internet路由表条目
- 最长前缀匹配 → 路由器转发的核心机制
- CIDR + NAT → 延缓IPv4地址耗尽
