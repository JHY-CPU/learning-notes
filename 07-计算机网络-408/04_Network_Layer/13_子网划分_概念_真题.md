# 13_子网划分_概念_真题

## 核心概念

- **408真题考查形式**：
  - 给定IP地址和子网掩码，求网络地址/广播地址
  - 给定网络和需求子网数，设计子网掩码
  - 判断两个IP是否属于同一子网
  - 计算子网内可用主机数

## 原理分析

### 真题1：求网络地址和广播地址

**题**：IP地址为192.168.3.78，子网掩码为255.255.255.224，求：
（1）网络地址
（2）广播地址
（3）可用IP地址范围
（4）该子网最多可容纳多少台主机？

**解**：
- 224 = 11100000，借了3位，主机号5位
- 子网间隔 = $2^5 = 32$
- 78 ÷ 32 = 2 余 14
- （1）网络地址 = 192.168.3.64（2 × 32 = 64）
- （2）广播地址 = 192.168.3.95（64 + 31 = 95）
- （3）可用范围 = 192.168.3.65 ~ 192.168.3.94
- （4）可用主机数 = $2^5 - 2 = 30$

### 真题2：判断是否同一子网

**题**：192.168.1.100/26 和 192.168.1.150/26 是否在同一子网？

**解**：
- /26，间隔 = 64
- 100 ÷ 64 = 1 余 36，网络地址 192.168.1.64
- 150 ÷ 64 = 2 余 22，网络地址 192.168.1.128
- **不在同一子网**（网络地址不同）

### 真题3：子网设计

**题**：某公司申请到一个C类地址202.100.1.0/24，需要划分5个子网，每个子网至少25台主机。问是否可行？

**解**：
- 需要5个子网：$2^n \geq 5$，$n = 3$，可分8个子网
- 剩余主机位：$8 - 3 = 5$ 位
- 每子网主机数：$2^5 - 2 = 30 \geq 25$ ✓
- **可行**，子网掩码为 255.255.255.224（/27）

### 真题4：子网划分变化

**题**：B类网络150.100.0.0/16，借8位划分子网，求子网掩码和子网数。

**解**：
- 新前缀 = 16 + 8 = 24
- 子网掩码 = 255.255.255.0（/24）
- 子网数 = $2^8 = 256$
- 每子网主机数 = $2^8 - 2 = 254$

## 直观理解

**解题流程图**：
```
给定IP和掩码
  ↓
看掩码最后一个非255字节 → 查表得借用位数
  ↓
计算子网间隔 = 2^主机位数
  ↓
目标字节 ÷ 间隔 = 商...余数
  ↓
网络地址 = 商 × 间隔
广播地址 = 网络地址 + 间隔 - 1
可用范围 = 网络地址+1 ~ 广播地址-1
```

## 协议关联

- 子网划分与路由表：路由器的路由表中记录每个子网的网络地址和下一跳
- 子网划分与ARP：同一子网内用ARP获取MAC，不同子网将帧发给网关
- 考试常结合CIDR、路由转发一起出综合题
