# 25_IP分片_概念_计算

## 核心概念

- **分片计算的核心步骤**：
  1. 确定MTU和IP首部长度
  2. 计算每片最大数据量（8字节对齐）
  3. 计算分片数量
  4. 确定每个分片的偏移值和MF标志

- **关键公式**：
  - 每片最大数据 = $\lfloor(MTU - H)/8\rfloor \times 8$（H为首部长度）
  - 分片数 = $\lceil \text{总数据} / \text{每片最大数据} \rceil$
  - 片偏移 = 该片起始字节位置 / 8

## 原理分析

### 例1：基本分片计算

**题**：IP数据报总长度4000字节（含20字节首部），下一跳MTU=1500，求分片情况。

**解**：
- 数据部分 = 4000 - 20 = 3980字节
- 每片最大数据 = $\lfloor(1500-20)/8\rfloor \times 8 = \lfloor1480/8\rfloor \times 8 = 1480$
- 分片数 = $\lceil3980/1480\rceil = 3$

| 分片 | 总长度 | 数据长度 | 片偏移 | MF | 标识 |
|------|--------|---------|--------|-----|------|
| 1 | 1500 | 1480 | 0 | 1 | 同值 |
| 2 | 1500 | 1480 | 185 | 1 | 同值 |
| 3 | 1040 | 1020 | 370 | 0 | 同值 |

验证：
- 偏移0 + 1480 = 1480，偏移185×8 = 1480 ✓
- 偏移1480 + 1480 = 2960，偏移370×8 = 2960 ✓
- 总数据 = 1480 + 1480 + 1020 = 3980 ✓

### 例2：多级分片

**题**：原数据报数据2000字节，经过第一个路由器MTU=1000，第二个路由器MTU=500，求最终分片。

**解**：
第一级（MTU=1000）：
- 每片最大数据 = $\lfloor980/8\rfloor \times 8 = 976$
- 分片1：数据976，偏移0，MF=1
- 分片2：数据976，偏移122，MF=1
- 分片3：数据48，偏移244，MF=0

第二级（MTU=500，对第一级的每个分片）：
- 每片最大数据 = $\lfloor480/8\rfloor \times 8 = 480$
- 分片1(976) → 480+480+16（3片）
- 分片2(976) → 480+480+16（3片）
- 分片3(48) → 无需分片

### 例3：反向计算——从分片还原原数据报

**题**：收到3个分片：
- 分片1：总长度1500，偏移0，MF=1
- 分片2：总长度1500，偏移185，MF=1
- 分片3：总长度1040，偏移370，MF=0

求原数据报数据长度。

**解**：
- 分片1数据：1500-20=1480，偏移0
- 分片2数据：1500-20=1480，偏移185×8=1480
- 分片3数据：1040-20=1020，偏移370×8=2960
- 最后一个字节位置：2960+1020-1=3979
- 原数据长度 = 3980字节
- 原总长度 = 3980+20=4000字节

## 直观理解

**计算口诀**：
- "MTU减首部，除8取整乘8"
- "偏移等于起始位置除以8"
- "MF=1后面有，MF=0到尽头"

## 协议关联

- 分片与TCP MSS：TCP通过MSS协商避免IP分片
- 分片与ICMP：DF=1但需分片时返回ICMP差错
- 考试中分片计算常与MTU、TTL等联合考查
