# 08_IPv4地址分类_概念_考试

## 核心概念

- **408高频考点**：
  1. 根据IP地址第一字节判断类别
  2. 计算某类网络可用IP地址数量
  3. 特殊地址的识别（网络地址、广播地址、回环地址）
  4. IP地址与子网掩码配合确定网络号

- **重要数字必须记忆**：
  - A类：1~126，默认/8
  - B类：128~191，默认/16
  - C类：192~223，默认/24
  - 回环：127.x.x.x

- **易混淆点**：
  - 127不是A类（127是回环保留地址）
  - 0.x.x.x不是A类（0保留）
  - A类网络号最高位固定为0，但值从1开始

## 原理分析

### 典型真题解析

**题型1：地址类别判断**

问：IP地址172.16.1.1属于哪类地址？
- 第一字节172，128 ≤ 172 ≤ 191
- 属于**B类地址**
- 网络号：172.16.0.0，主机号：0.0.1.1

**题型2：计算可用主机数**

问：一个C类网络最多可容纳多少台主机？
$$2^8 - 2 = 256 - 2 = 254 \text{台}$$
- 减2：全0（网络地址）和全1（广播地址）

**题型3：网络地址和广播地址**

问：已知IP地址192.168.1.100，子网掩码255.255.255.0
- 网络地址：192.168.1.100 AND 255.255.255.0 = **192.168.1.0**
- 广播地址：192.168.1.**255**
- 可用IP范围：192.168.1.1 ~ 192.168.1.254

**题型4：判断地址是否合法**

问：以下哪个是有效的主机IP地址？
A. 192.168.1.255（C类广播地址）
B. 10.0.0.0（A类网络地址）
C. 172.16.1.1（有效的B类主机地址）✓
D. 224.0.0.1（D类组播地址）

答案：C。网络地址和广播地址不能分配给主机，D类是组播地址。

### 快速判断技巧

```
第一字节的二进制第一位：
0 → A类
1 → 再看第二位：
   0 → B类
   1 → 再看第三位：
      0 → C类
      1 → 再看第四位：
         0 → D类
         1 → E类
```

## 直观理解

**数字记忆口诀**：
- "A类1到126，B类128到191，C类192到223"
- "127是回环家，0是网络本家"

## 协议关联

### 与子网划分的关系

- 分类编址是子网划分的基础
- 子网划分 = 在主机号中借位作为子网号
- CIDR = 完全抛弃分类，自由指定前缀长度

### 考试常见陷阱
- "127.0.0.1是A类地址" → 错，127是回环测试保留
- "广播地址可以分配给主机" → 错，广播地址永远不能分配
- "网络地址可以分配给主机" → 错，全0代表网络本身
