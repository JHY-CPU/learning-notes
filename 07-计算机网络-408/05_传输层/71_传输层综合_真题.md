# 71_传输层综合_真题

## 核心概念

- **408传输层真题覆盖所有子主题**
- **高频考点**：TCP连接管理、拥塞控制计算、序号确认号、流量控制

### 408真题精选

**题目**：主机甲和乙建立了TCP连接，持续进行数据传输。甲的拥塞窗口cwnd=30KB，乙的接收窗口rwnd=20KB，最大报文段MSS=1KB。甲向乙发送数据的最大速率是多少？（RTT=10ms）

**解析**：
$$\text{发送窗口} = \min(\text{rwnd}, \text{cwnd}) = \min(20\text{KB}, 30\text{KB}) = 20\text{KB}$$
$$\text{最大速率} = \frac{20\text{KB}}{10\text{ms}} = 2\text{MB/s}$$

**题目**：TCP连接中，甲的初始序号为100，乙的初始序号为200。甲向乙发送数据：第一个报文段包含200字节，第二个报文段包含300字节。乙正确收到后发送确认。第二个确认报文段的确认号是多少？

**解析**：
- 三次握手消耗：SYN(100) + 第三次握手ACK(101) → 数据从102开始
- 第一个数据段：SEQ=102，200字节（102~301）
- 第二个数据段：SEQ=302，300字节（302~601）
- 第二个确认号 = 601 + 1 = **602**

**题目**：TCP连接的ssthresh=8 MSS，cwnd=1 MSS。假设在cwnd=16时收到3个冗余ACK，求cwnd和ssthresh。

**解析**：
- ssthresh = 16/2 = **8**
- cwnd = 8 + 3 = **11**
- 进入拥塞避免阶段

## 直观理解

- **做题关键**：分清超时和冗余ACK、记住SYN/FIN各消耗1序号、发送窗口取min(rwnd, cwnd)

## 协议关联

- **综合题常结合多个知识点**：连接管理+可靠传输+拥塞控制
- **408高频题型**：计算题（序号、cwnd变化）和概念题（状态转换、协议选择）
