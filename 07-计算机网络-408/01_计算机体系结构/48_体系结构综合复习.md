## 高频考点清单

### 必记概念

1. 协议三要素：语法、语义、同步
2. 服务与协议的区别：垂直 vs 水平
3. OSI各层名称和功能
4. TCP/IP各层名称和核心协议
5. 各层PDU名称
6. 发送时延和传播时延的公式

### 必记对比

| 对比项 | 选项A | 选项B |
|--------|-------|-------|
| OSI vs TCP/IP | 7层，理论标准 | 4层，事实标准 |
| 面向连接 vs 无连接 | TCP | UDP |
| 虚电路 vs 数据报 | 建立连接，同路径 | 无连接，独立路由 |
| 集线器 vs 交换机 vs 路由器 | 物理层 | 数据链路层 | 网络层 |

### 必会计算

1. 发送时延 = 数据长度 / 带宽
2. 传播时延 = 距离 / 传播速度
3. 总时延 = 发送时延 + 传播时延 + 处理时延 + 排队时延
4. 信道利用率 = 发送时延 / (发送时延 + RTT)
5. 时延带宽积 = 传播时延 x 带宽
6. 香农容量 = W * log2(1 + SNR)
7. 分组交换流水线时延 = (分组数 + 链路数 - 1) * 单分组发送时延

### 常见陷阱

1. "带宽"在模拟信号中是Hz，在网络中是b/s
2. TCP的PDU叫"报文段"，不叫"包"
3. 面向连接不一定等于可靠
4. 提高带宽只能减少发送时延，不减少传播时延
5. ARP协议的归属有争议，408一般归网络层
6. 片偏移以8字节为单位

## 直观理解

体系结构是整个计算机网络的"骨架"，理解了骨架，后续各层的"血肉"才能填充到位。
