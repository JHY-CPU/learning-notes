# 30_单工半双工全双工

## 核心概念

### 三种通信方式

| 方式 | 方向性 | 同时双向 | 典型例子 |
|------|--------|----------|----------|
| **单工** | 只能单向 | 不适用 | 广播电视、键盘→主机 |
| **半双工** | 可双向 | 不能同时 | 对讲机、集线器以太网 |
| **全双工** | 可双向 | 能同时 | 电话、交换机以太网 |

### 详细定义

**单工（Simplex）**：
- 数据只能沿一个方向传输
- 一方固定为发送方，另一方固定为接收方
- 无需反向信道
- 例：广播电台→收音机、电视发射台→电视机

**半双工（Half-Duplex）**：
- 数据可以双向传输，但**不能同时**进行
- 需要切换传输方向
- 可以使用一条信道（分时复用）或两条信道
- 例：对讲机、早期以太网（集线器）

**全双工（Full-Duplex）**：
- 数据可以**同时双向**传输
- 通常需要两条独立的信道（一条发、一条收）
- 例：电话、现代以太网（交换机端口）

## 原理分析

### 半双工的工作方式

```
A ──发送──→ ┌──────┐ ──接收──→ B
            │ 信道 │
A ←──接收── └──────┘ ←──发送── B

同一时刻只能一个方向传输
需要"转向"时间
```

### 全双工的工作方式

```
A ──发送──→ ┌──────信道1──────┐ ──接收──→ B
            │                 │
A ←──接收── └──────信道2──────┘ ←──发送── B

同时双向传输
```

### 以太网的双工模式

- **10BASE-T**（集线器）：半双工，需要CSMA/CD
- **10BASE-T**（交换机）：全双工，不需要CSMA/CD
- 100BASE-TX、1000BASE-T：支持全双工
- 全双工以太网的两端使用不同的线对（1/2发、3/6收）

### 半双工的效率损失

设转向时间为 $T_{turn}$，数据传输时间为 $T_{data}$：

$$\text{效率} = \frac{T_{data}}{T_{data} + T_{turn}}$$

## 直观理解

### 类比：三种交流方式

- **单工**：老师在台上讲课，学生只能听，不能回话
- **半双工**：学生举手→老师同意→学生发言→老师回应（交替进行）
- **全双工**：老师和学生讨论，双方可以同时发言

### 类比：道路

- **单工**：单行道
- **半双工**：单车道双向公路（需要交替通行）
- **全双工**：双车道公路（各行其道）

## 协议关联

### 与CSMA/CD的关系

- **半双工**以太网需要CSMA/CD来处理冲突
- **全双工**以太网不需要CSMA/CD（没有冲突）
- 现代以太网几乎都工作在全双工模式
- 这就是为什么千兆以太网及以上实际上不使用CSMA/CD

### 与交换机/集线器的关系

- 集线器：所有端口**半双工**共享
- 交换机：每个端口可独立**全双工**

### 考试要点

1. 半双工可以双向但不能同时
2. 全双工可以同时双向传输
3. 全双工以太网不需要CSMA/CD
4. 10BASE-T使用两对线：一对发一对收（实现全双工）
5. 电话是全双工的典型例子
