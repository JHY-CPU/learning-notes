# 40_物理层综合计算

## 核心概念

物理层综合计算题通常涉及以下公式：

### 公式总结

**奈奎斯特准则**（无噪声）：
$$C = 2W \log_2 M$$

**香农定理**（有噪声）：
$$C = W \log_2(1+\text{SNR})$$

**信噪比转换**：
$$\text{SNR} = 10^{\text{SNR}_{dB}/10}$$
$$\text{SNR}_{dB} = 10 \log_{10}(\text{SNR})$$

**波特率与比特率**：
$$R_b = R_B \cdot \log_2 M$$

**编码效率**：
$$R_{data} = R_{line} \times \text{编码效率}$$

**PCM编码速率**：
$$R = f_s \times n \quad \text{(采样频率 × 量化位数)}$$

**CDMA内积**：
$$S \cdot R = \frac{1}{m}\sum_{i=1}^{m} S_i \times R_i$$

## 原理分析

### 综合题型1：奈奎斯特+香农联合

**题目**：某信道带宽 4kHz，信噪比 30dB，采用 8 电平传输，求信道最大数据传输速率。

**解析**：

香农定理：
$$\text{SNR} = 10^{30/10} = 1000$$
$$C_1 = 4000 \times \log_2(1+1000) \approx 4000 \times 10 = 40000 \text{ bit/s}$$

奈奎斯特准则：
$$C_2 = 2 \times 4000 \times \log_2 8 = 8000 \times 3 = 24000 \text{ bit/s}$$

取较小值：
$$C = \min(40000, 24000) = 24000 \text{ bit/s} = 24 \text{ kbit/s}$$

### 综合题型2：PCM编码+TDM

**题目**：对 4kHz 最高频率的模拟信号进行 PCM 编码，量化级数为 256 级，采用 E1 标准传输，求：(1) 每路 PCM 信号的速率 (2) E1 可以复用多少路

**解析**：

(1) PCM 速率：
- 采样频率：$f_s = 2 \times 4000 = 8000$ Hz
- 量化级数 256 → 每样本 8 bit
- PCM 速率：$R = 8000 \times 8 = 64$ kbit/s

(2) E1 复用路数：
- E1 总速率：2.048 Mbit/s
- 可用时隙：30 个（TS0 同步，TS16 信令）
- 每路 64 kbit/s
- $30 \times 64$ kbit/s = 1.92 Mbit/s < 2.048 Mbit/s

所以 E1 可以复用 **30 路** PCM 信号。

### 综合题型3：编码+波特率

**题目**：某信道带宽 100kHz，无噪声，使用曼彻斯特编码，求最大数据传输速率。

**解析**：

奈奎斯特最大波特率：
$$R_B = 2W = 2 \times 100000 = 200000 \text{ Baud}$$

曼彻斯特编码效率 50%：
$$R_{data} = 200000 \times 50\% = 100000 \text{ bit/s} = 100 \text{ kbit/s}$$

### 综合题型4：传输时间计算

**题目**：某信道数据传输速率为 1 Mbit/s，传播时延为 20ms，发送 1000 bit 的数据，求总时延。

**解析**：

发送时延：
$$T_{send} = \frac{1000}{10^6} = 1 \text{ ms}$$

传播时延：
$$T_{prop} = 20 \text{ ms}$$

总时延：
$$T_{total} = T_{send} + T_{prop} = 1 + 20 = 21 \text{ ms}$$

## 直观理解

### 解题流程

```
读题
├─ 确定已知条件（带宽、信噪比、电平数、编码方式等）
├─ 判断用哪个公式
│  ├─ 无噪声 → 奈奎斯特
│  ├─ 有噪声 → 香农
│  └─ 都有 → 都算取min
├─ 注意单位换算（kHz→Hz, dB→比值）
└─ 如果有编码效率，最后乘以编码效率
```

## 协议关联

### 综合计算的常见组合

1. 奈奎斯特 + 香农
2. PCM + TDM（E1/T1）
3. 编码效率 + 奈奎斯特
4. CDMA 内积计算
5. 传输时延计算
