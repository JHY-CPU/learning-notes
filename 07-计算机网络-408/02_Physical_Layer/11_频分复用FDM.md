# 11_频分复用FDM

## 核心概念

- **频分复用（Frequency Division Multiplexing, FDM）**：将信道的频带划分为若干个互不重叠的子频带，每个用户占用一个子频带
- 各路信号**同时传输**，但频率不同
- 子频带之间留有**防护频带（Guard Band）**，防止信号间干扰

### FDM的关键参数

- **总带宽** $W$：信道的可用频带宽度
- **子频带宽度** $W_i$：每个用户分配的频带宽度
- **防护频带** $W_g$：相邻子频带之间的隔离频带
- **复用用户数**：

$$N = \left\lfloor \frac{W - W_g}{W_i + W_g} \right\rfloor + 1$$

简化计算（忽略首尾防护频带）：
$$N \approx \frac{W}{W_i + W_g}$$

### FDM的特点

| 优点 | 缺点 |
|------|------|
| 技术成熟，实现简单 | 子频带固定，灵活性差 |
| 适合模拟信号传输 | 频带利用率较低（防护频带开销） |
| 不需要严格同步 | 用户数增加时需要更窄的子频带 |

## 原理分析

### FDM频谱分配图

```
频率
↑
│  ┌────┐  ┌────┐  ┌────┐  ┌────┐
│  │ U1 │  │ U2 │  │ U3 │  │ U4 │
│  │    │  │    │  │    │  │    │
└──┴────┴──┴────┴──┴────┴──┴────┴──→
       ↑   ↑       ↑   ↑
     防护频带     防护频带
```

### FDM的工作流程

1. **发送端**：
   - 各用户信号分别用不同频率的载波进行调制
   - 调制后的信号在频域上互不重叠
   - 将所有调制信号合并，送入信道

2. **接收端**：
   - 使用带通滤波器分离各频带的信号
   - 分别解调，还原原始信号

### 典型应用：电话系统

- 传统模拟电话：每路电话 4kHz 带宽
- 基群：12路电话复用，带宽 48kHz
- 超群：60路电话复用
- 通过多级复用实现大容量传输

## 直观理解

### 类比：收音机

- FDM 就像**收音机的不同电台**
- 中央人民广播电台在某个频率，地方台在另一个频率
- 你调到哪个频率，就收听哪个台
- 电台之间留有间隔（防护频带），防止互相干扰
- 所有电台同时广播，但互不干扰

### 类比：小区停车位

- 小区 = 总信道带宽
- 每个车位 = 一个子频带
- 车位之间的隔离线 = 防护频带
- 每辆车占一个车位 = 每个用户占一个频带

## 协议关联

### 与调制技术的关系

- FDM 依赖**调制**技术将各路信号搬到不同频段
- 常用调制方式：AM、FM、SSB等
- 每路信号调制到不同载波频率

### 与WDM的关系

- WDM 是 FDM 在光纤领域的特例
- FDM 按频率划分，WDM 按波长划分
- 本质相同：波长 = 光速/频率

### 考试要点

1. FDM 是**模拟**复用技术（虽然也可以用于数字信号的频带传输）
2. 防护频带的概念和作用
3. 复用路数的计算
4. 与 TDM 的对比：FDM 同时不同频率，TDM 同频率不同时刻
