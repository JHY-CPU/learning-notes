# 27_编码方式_概念_真题

## 核心概念

本节汇总408真题中关于数字信号编码的典型考题。

### 高频考点

1. 曼彻斯特编码/差分曼彻斯特编码的波形判读
2. 编码效率与线路速率的计算
3. 不同编码方式的特性比较
4. 编码方式与以太网标准的对应关系

## 原理分析

### 真题类型1：波形判读

**题目**：下图所示的编码方式是？

```
电平
高  ┌─┐   ┌─────┐ ┌─┐
    │ │   │     │ │ │
低 ─┘ └───┘     └─┘ └─────
```

**解析**：每个比特中间都有跳变，这是**曼彻斯特编码**的特征。

### 真题类型2：从波形还原数据

**题目**：已知某信号采用曼彻斯特编码（1=高→低，0=低→高），波形如下，求原始数据。

```
高 ┌───┐ ┌─┐   ┌─────┐
   │   │ │ │   │     │
低 ─┘   └─┘ └───┘     └─────
```

**解析**：
- 第1个比特：高→低 = **1**
- 第2个比特：低→高 = **0**
- 第3个比特：高→低 = **1**
- 第4个比特：低→高 = **0**

数据为：**1010**

### 真题类型3：差分曼彻斯特判读

**题目**：已知采用差分曼彻斯特编码，初始状态为高电平，波形如下，求原始数据。

```
高 ┌───┐ ┌─┐ ┌───────┐
   │   │ │ │ │       │
低 ─┘   └─┘ └─┘       └─────
```

**解析**（规则：比特开始无跳变=1，有跳变=0）：
- 第1个比特：开始处从低跳到高（有跳变）= **0**
- 第2个比特：开始处从低跳到高（有跳变）= **0**
- 第3个比特：开始处无跳变 = **1**
- 第4个比特：开始处从低跳到高（有跳变）= **0**

数据为：**0010**

### 真题类型4：编码效率计算

**题目**：某10M以太网使用曼彻斯特编码，求线路波特率。

$$R_{line} = \frac{10 \text{ Mbit/s}}{50\%} = 20 \text{ Mbaud}$$

**题目**：某信道带宽为 4kHz，使用曼彻斯特编码，无噪声，求最大数据传输速率。

$$R_{B(max)} = 2W = 8000 \text{ Baud}$$
$$R_{data} = 8000 \times 50\% = 4000 \text{ bit/s}$$

## 直观理解

### 波形判读技巧

1. **看比特中间**：每个比特中间是否都有跳变？
   - 都有 → 曼彻斯特或差分曼彻斯特
   - 不一定有 → NRZ或NRZI

2. **看比特开始处**：比特开始处是否有跳变？
   - 有规律地跳变/不跳变 → 差分曼彻斯特
   - 绝对电平决定 → 曼彻斯特

3. **解题步骤**：
   ```
   a. 确定编码方式
   b. 确定比特边界（中间跳变处）
   c. 按规则解读每个比特
   ```

## 协议关联

### 易错点总结

1. **不同教材的曼彻斯特编码规则可能相反**：408考试以谢希仁教材为准（1=高→低）
2. **差分曼彻斯特看相对变化**：不是绝对电平，而是"有没有跳变"
3. **编码效率只针对编码方式**：与信道带宽无关
4. **曼彻斯特编码的线路速率是数据速率的2倍**：10M以太网实际线路速率20Mbaud

### 常见考试陷阱

1. 把曼彻斯特编码的线路速率当作数据速率
2. 混淆曼彻斯特和差分曼彻斯特的规则
3. 差分曼彻斯特忘了看"比特开始处"而只看中间
4. 忘记差分曼彻斯特需要确定初始电平状态
