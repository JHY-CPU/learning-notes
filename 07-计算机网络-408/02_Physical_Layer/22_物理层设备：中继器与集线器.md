# 22_物理层设备：中继器与集线器

## 核心概念

### 中继器（Repeater）

- **中继器**：工作在物理层的网络设备
- 功能：**再生和放大**衰减的数字信号，延长传输距离
- 不对信号做任何逻辑处理，只是简单地放大
- 不能连接两个不同的网络（不能进行协议转换）

### 中继器的特点

| 特性 | 说明 |
|------|------|
| 工作层次 | 物理层 |
| 功能 | 信号再生和放大 |
| 是否理解帧 | 否，只处理比特流 |
| 是否过滤 | 否，所有信号都转发 |
| 是否有MAC地址 | 否 |
| 冲突域 | 不隔离冲突域 |

### 集线器（Hub）

- **集线器**：多端口的中继器
- 功能：将一个端口收到的信号放大后转发到**所有其他端口**
- 采用**广播**方式工作
- 所有端口属于**同一个冲突域**

### 集线器的特点

| 特性 | 说明 |
|------|------|
| 工作层次 | 物理层 |
| 功能 | 信号放大 + 广播转发 |
| 端口关系 | 所有端口在同一冲突域 |
| 安全性 | 差（所有端口都能收到） |
| 带宽 | 共享（半双工） |

## 原理分析

### 中继器的工作原理

```
信号源 → [衰减的信号] → 中继器 → [再生的信号] → 目的地
              ↓
         信号波形畸变
              ↓
         中继器检测、放大、整形
              ↓
         还原为清晰的信号
```

### 集线器的工作方式

```
        端口1 ←→ 计算机A
        端口2 ←→ 计算机B
集线器 → 端口3 ←→ 计算机C
        端口4 ←→ 计算机D

当A向B发送数据时：
A → 集线器 → 广播到端口2,3,4
B收到（目标地址匹配），C和D收到（但丢弃）
```

### 5-4-3规则（408常考）

使用中继器/集线器扩展以太网时：
- 最多**5个网段**
- 最多**4个中继器**
- 最多**3个网段**可以连接计算机
- 另外2个网段只能用于扩展（不接计算机）

```
计算机—网段1—中继器—网段2—中继器—网段3—中继器—网段4—中继器—网段5—计算机
          ✓     1      ✓     2      ✓     3      ✗     4      ✗
```

## 直观理解

### 中继器类比：扩音器

- 中继器好比演讲时的**扩音器**
- 演讲者声音太小（信号衰减），远处的人听不清
- 扩音器接收声音→放大→重新播出
- 扩音器不关心你说的是什么语言（不理解内容），只是放大声音

### 集线器类比：传话筒

- 集线器好比一个**中央传话台**
- 任何人说的话，都会被转述给所有人
- 效率低（同一时间只有一个人能说话）
- 没有隐私（所有人都能听到）

### 为什么集线器被淘汰

1. **共享带宽**：10Mbit/s的集线器，8个端口共享10Mbit/s
2. **冲突域大**：所有端口在同一个冲突域，冲突频繁
3. **安全性差**：广播方式，数据可以被任意端口捕获
4. **交换机取代**：交换机每个端口是独立冲突域，提供独享带宽

## 协议关联

### 与冲突域/广播域的关系

- **中继器/集线器**：不隔离冲突域，不隔离广播域
- **交换机（二层）**：隔离冲突域，不隔离广播域
- **路由器（三层）**：隔离冲突域，隔离广播域

这是408必考的知识点。

### 与以太网的关系

- 早期以太网使用集线器连接（10BASE-T）
- 集线器构成的以太网是**半双工**的（需要CSMA/CD）
- 现代以太网使用交换机（全双工，不需要CSMA/CD）

### 考试要点

1. 中继器和集线器都是物理层设备
2. 集线器 = 多端口中继器
3. 集线器所有端口在同一冲突域
4. 5-4-3规则
5. 中继器不能连接异构网络（如以太网和令牌环网）
