# 36_ADSL技术概述

## 核心概念

- **ADSL（Asymmetric Digital Subscriber Line，非对称数字用户线）**：利用现有的电话线（铜双绞线）提供宽带上网服务
- **"非对称"**：下行速率远高于上行速率
- 利用电话线中未被语音使用的高频段传输数据

### ADSL的特点

| 参数 | 说明 |
|------|------|
| 传输介质 | 现有电话双绞线 |
| 下行速率 | 最高 8 Mbit/s（ADSL）/ 24 Mbit/s（ADSL2+） |
| 上行速率 | 最高 1 Mbit/s（ADSL）/ 3.3 Mbit/s（ADSL2+） |
| 最大距离 | 约 3~5 km |
| 频段划分 | 0~4kHz语音，25kHz~1.1MHz数据 |

### ADSL频段划分

```
频率
↑
│  ┌──────────────────┐
│  │   下行数据       │  200kHz ~ 1.1MHz
│  └──────────────────┘
│  ┌──────┐
│  │ 上行 │            25kHz ~ 200kHz
│  └──────┘
│  ┌────┐
│  │语音│              0 ~ 4kHz
│  └────┘
└────────────────────────→ 频率
```

### ADSL的核心技术

- **DMT（Discrete Multi-Tone，离散多音调）**：将频带划分为256个子信道（4kHz宽），每个子信道独立调制
- 使用**QAM调制**在每个子信道上调制数据
- 自适应调制：质量好的子信道用更高阶调制，质量差的子信道用低阶调制或不使用

## 原理分析

### DMT调制原理

- 将可用频带划分为256个4kHz子信道
- 每个子信道独立测量信道质量（信噪比）
- 信噪比高的子信道分配更多比特
- 信噪比低的子信道少分配或不分配比特

$$\text{总速率} = \sum_{i=1}^{256} b_i \times R_s$$

其中 $b_i$ 是第 $i$ 个子信道每符号携带的比特数，$R_s$ 是符号速率。

### ADSL的系统模型

```
用户端                                    局端
┌──────┐    电话线（双绞线）    ┌──────┐
│ DSL  │══════════════════════│ DSL  │──→ 互联网
│Modem │    ┌──语音──┐        │  AM  │
│      │    │分离器  │        │      │
└──┬───┘    └───┬────┘        └──────┘
   │            │
 计算机       电话机
```

- **分离器（Splitter）**：将语音信号（0~4kHz）和数据信号分离
- **DSLAM（DSL Access Multiplexer）**：局端设备，汇聚多个用户的ADSL信号

## 直观理解

### 类比：高速公路与辅路

- 电话线好比一条公路
- 语音通话走**辅路**（低频0~4kHz，窄但够用）
- 上网数据走**高速公路**（高频25kHz~1.1MHz，宽且快）
- 两辆车各行其道，互不干扰

### 为什么是"非对称"

- 用户上网主要行为：**下载**远多于**上传**
- 浏览网页：下载网页内容 >> 上传点击
- 看视频：下载视频 >> 几乎不上传
- 因此下行分配更多频带，上行分配较少频带

## 协议关联

### 与物理层的关系

- ADSL是物理层技术
- 使用频带传输方式
- 核心调制技术是DMT（离散多音调）

### 与其他接入技术的关系

- ADSL逐渐被光纤接入（FTTH）取代
- 电话线带宽有限，无法满足越来越高的速率需求
- 光纤接入可以提供数百Mbit/s甚至Gbit/s的速率

### 考试要点

1. ADSL利用电话线的高频段传输数据
2. 下行速率高于上行速率（非对称）
3. 使用DMT调制技术
4. 需要分离器分离语音和数据
5. 最大距离约3~5km
