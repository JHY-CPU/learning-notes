# 17_传输介质：光纤

## 核心概念

- **光纤（Optical Fiber）**：利用光的全反射原理传输光脉冲信号的介质
- 由**纤芯（Core）**和**包层（Cladding）**组成
- 纤芯折射率 > 包层折射率，光在纤芯内发生全反射

### 光纤结构

```
   ┌────── 外护套
   │ ┌──── 涂覆层
   │ │ ┌── 包层（折射率低）
   │ │ │ ┌ 纤芯（折射率高）
   ═══════════
   光 →→→→→→→→→ 光（全反射传播）
```

### 光纤分类

| 类型 | 纤芯直径 | 光源 | 传输距离 | 特点 |
|------|----------|------|----------|------|
| 多模光纤（MMF） | 50/62.5μm | LED | 较短（几百m~2km） | 模间色散大，成本低 |
| 单模光纤（SMF） | 8~10μm | 激光器 | 很长（几十km） | 无模间色散，成本高 |

### 光纤的通信窗口

- **850nm**：多模光纤常用
- **1310nm**：单模光纤，低色散
- **1550nm**：单模光纤，最低衰减，长距离传输

## 原理分析

### 全反射原理

光从高折射率介质射向低折射率介质时：
- 入射角大于**临界角**时，发生全反射
- 光在纤芯内不断全反射向前传播

临界角公式：
$$\theta_c = \arcsin\left(\frac{n_2}{n_1}\right)$$

- $n_1$：纤芯折射率
- $n_2$：包层折射率
- $n_1 > n_2$

### 数值孔径（NA）

$$NA = \sin\theta_{max} = \sqrt{n_1^2 - n_2^2}$$

- NA 越大，能接收光的角度范围越大
- 多模光纤 NA 较大，单模光纤 NA 较小

### 光纤的衰减

- 光在光纤中传输存在衰减
- 衰减系数：$\alpha$（dB/km）
- 850nm：约 2.5 dB/km（多模）
- 1310nm：约 0.35 dB/km（单模）
- 1550nm：约 0.2 dB/km（单模，最低）

## 直观理解

### 类比：管道中的弹珠

- 光纤好比一根弯曲的玻璃管道
- 光脉冲好比在管道中弹跳前进的弹珠
- 弹珠碰到管壁就反弹（全反射），永远不会跑出去
- 管道越细（单模），弹珠只有一个前进方向，不会互相碰撞（无模间色散）

### 光纤的优势

| 优点 | 说明 |
|------|------|
| 带宽极高 | 可达数十 Tbit/s |
| 衰减极低 | 长距离无需中继 |
| 抗电磁干扰 | 不受电磁干扰影响 |
| 保密性好 | 不易被窃听 |
| 重量轻 | 便于铺设 |

### 光纤的缺点

- 接续困难（需要专业设备）
- 弯曲半径有限制（弯折过度会增加损耗甚至断光）
- 设备成本较高

## 协议关联

### 与WDM的关系

- WDM（波分复用）利用光纤的巨大带宽
- 一根光纤同时传输多个波长的光信号
- DWDM可实现 80-160 个波长，总容量达 Tbit/s

### 与以太网的关系

- **1000BASE-SX**：千兆以太网，多模光纤，850nm
- **1000BASE-LX**：千兆以太网，单模光纤，1310nm
- **10GBASE-LR**：万兆以太网，单模光纤，1310nm

### 考试要点

1. 光纤利用全反射原理传输
2. 单模光纤与多模光纤的区别
3. 光纤的优点：带宽高、衰减低、抗电磁干扰
4. 光纤属于有导向传输介质
