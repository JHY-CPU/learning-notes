# 18_传输介质：无线

## 核心概念

- **无线传输介质**：利用电磁波在自由空间中传播来传输信号
- 属于**无导向传输介质**（没有固定的物理路径）
- 频率范围从无线电波到微波、红外线

### 电磁波频谱

| 频段 | 频率范围 | 波长 | 典型应用 |
|------|----------|------|----------|
| 无线电波 | 3Hz~3GHz | >10cm | AM/FM广播、手机 |
| 微波 | 3GHz~300GHz | 1mm~10cm | 卫星通信、WiFi |
| 红外线 | 300GHz~400THz | 750nm~1mm | 遥控器、IrDA |
| 可见光 | 400THz~800THz | 375~750nm | VLC可见光通信 |

### 无线传输方式

- **地面微波接力**：视距传输，需要中继站
- **卫星通信**：利用同步卫星中继，覆盖范围大
- **蜂窝移动通信**：2G/3G/4G/5G
- **无线局域网（WLAN）**：WiFi（IEEE 802.11）

### 无线传输的特点

| 优点 | 缺点 |
|------|------|
| 无需铺设线缆 | 易受干扰 |
| 移动性好 | 安全性较差 |
| 覆盖范围广 | 带宽有限 |
| 部署灵活 | 信号衰减和多径效应 |

## 原理分析

### 自由空间传播模型

$$P_r = \frac{P_t G_t G_r \lambda^2}{(4\pi d)^2 L}$$

- $P_r$：接收功率
- $P_t$：发射功率
- $G_t, G_r$：发射和接收天线增益
- $\lambda$：波长
- $d$：传播距离
- $L$：系统损耗

### 多径效应

- 无线信号经过多条路径到达接收端
- 不同路径的信号到达时间不同，相位不同
- 叠加后可能导致信号增强或衰落（多径衰落）
- 是无线通信的主要挑战之一

### 频率越高

- 带宽越大（容量越大）
- 衰减越大（覆盖越小）
- 方向性越强（天线可以更小）
- 这就是5G使用毫米波（高频）但需要更多基站的原因

## 直观理解

### 类比：广场上喊话

- 无线传输好比在广场上大声喊话
- 所有人都能听到（广播特性）
- 但人多嘈杂时（干扰），对方可能听不清
- 距离越远声音越小（衰减）
- 墙壁会反弹声音（多径效应），导致回声

### 各代移动通信的记忆

| 代际 | 技术核心 | 速率 | 特点 |
|------|----------|------|------|
| 1G | 模拟调频 | 低 | 只能打电话 |
| 2G | GSM/CDMA | 9.6~14.4 kbit/s | 可以发短信 |
| 3G | WCDMA/CDMA2000 | 数百kbit/s~数Mbit/s | 可以上网 |
| 4G | OFDM/LTE | 数十~数百Mbit/s | 流畅视频 |
| 5G | mmWave/NR | 数Gbit/s | 万物互联 |

## 协议关联

### 与物理层的关系

- 无线传输是物理层的传输介质之一
- 需要特殊的调制方式（OFDM、QAM等）
- 需要特殊的编码和差错控制

### 与WiFi的关系

- WiFi（IEEE 802.11）工作在2.4GHz和5GHz频段
- 使用OFDM调制
- 物理层使用DSSS或OFDM编码

### 考试要点

1. 无线属于无导向传输介质
2. 多径效应是无线通信的主要问题
3. 频率越高，带宽越大但衰减也越大
4. 卫星通信有较大的传播延迟（同步卫星约270ms）
