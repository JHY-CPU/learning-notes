# 47_物理层知识框架总结

## 核心概念

### 物理层知识体系全景图

```
物理层
├─ 1. 基本概念
│  ├─ 物理层的功能和地位
│  └─ 四大接口特性（机、电、功、规）
├─ 2. 数据通信基础
│  ├─ 信源、信道、信宿
│  ├─ 模拟信号 vs 数字信号
│  ├─ 波特率 vs 比特率：Rb = RB·log₂M
│  └─ 奈奎斯特准则 & 香农定理
├─ 3. 编码与调制
│  ├─ 数字→数字：NRZ、曼彻斯特、差分曼彻斯特
│  ├─ 数字→模拟：ASK、FSK、PSK、QAM
│  ├─ 模拟→数字：PCM（采样、量化、编码）
│  └─ 模拟→模拟：AM、FM
├─ 4. 信道复用
│  ├─ FDM（频分复用）
│  ├─ TDM（时分复用）：E1/T1标准
│  ├─ WDM（波分复用）
│  └─ CDM/CDMA（码分复用）
├─ 5. 传输介质
│  ├─ 双绞线（UTP/STP）
│  ├─ 同轴电缆（50Ω/75Ω）
│  ├─ 光纤（单模/多模）
│  └─ 无线（微波、卫星、红外）
├─ 6. 宽带接入
│  ├─ ADSL（DMT调制，非对称）
│  ├─ HFC（光纤+同轴，共享）
│  └─ FTTH/PON（光纤到户）
├─ 7. 物理层设备
│  ├─ 中继器（信号再生）
│  └─ 集线器（多端口中继器）
└─ 8. 物理层协议
   ├─ EIA-232（单端，点对点）
   ├─ RS-485（差分，多点）
   └─ X.21（数字线路）
```

### 408必背公式

| 公式 | 名称 | 条件 |
|------|------|------|
| $C = 2W\log_2 M$ | 奈奎斯特准则 | 无噪声 |
| $C = W\log_2(1+\text{SNR})$ | 香农定理 | 有噪声 |
| $R_b = R_B \cdot \log_2 M$ | 波特率→比特率 | — |
| $\text{SNR}_{dB} = 10\log_{10}\text{SNR}$ | 信噪比分贝转换 | — |
| $f_s \geq 2f_{max}$ | 采样定理 | PCM编码 |

### 知识点关联

```
编码调制 ──→ 曼彻斯特编码 ──→ 编码效率 ──→ 线路速率
    │
    └──→ PCM编码 ──→ 采样定理 ──→ E1/T1 ──→ TDM复用

奈奎斯特/香农 ──→ 信道容量 ──→ 与编码效率联合计算
    │
    └──→ 信噪比换算 ──→ 分贝与比值转换

传输介质 ──→ 光纤/双绞线 ──→ 以太网物理层标准
    │
    └──→ 无线 ──→ WiFi/移动通信
```

## 直观理解

### 考试解题策略

**选择题**：
1. 先判断题目考哪个知识点
2. 概念题：回忆关键区分点
3. 计算题：选择正确公式，注意单位

**计算题步骤**：
1. 提取已知条件
2. 判断用奈奎斯特还是香农（或两者都用）
3. 注意单位换算（kHz→Hz, dB→比值）
4. 有编码效率时最后乘效率
5. 两个公式都有时取较小值

## 协议关联

### 物理层与上层的交互

- 物理层向数据链路层提供**比特流传输**服务
- 数据链路层不关心物理层用什么编码方式
- 物理层的速率直接影响上层的传输效率

### 与实际网络的关联

- 局域网：双绞线+以太网物理层标准
- 广域网：光纤+SONET/SDH
- 接入网：FTTH/ADSL
- 无线：WiFi/蜂窝网络
