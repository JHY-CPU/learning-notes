# 14_信道复用_概念_考试

## 核心概念

### 复用方式总览

| 复用方式 | 划分维度 | 信号类型 | 典型应用 | 408考频 |
|----------|----------|----------|----------|---------|
| FDM | 频率 | 模拟为主 | 有线电视、广播 | 高 |
| TDM | 时间 | 数字 | E1/T1、SDH | 很高 |
| WDM | 波长 | 光信号 | 光纤骨干网 | 中 |
| CDM | 编码 | 数字 | 3G通信 | 很高 |
| STDM | 时间(动态) | 数字 | 统计复用 | 中 |

### 考试核心对比

**FDM vs TDM**：
- FDM：同时不同频率；TDM：同频率不同时刻
- FDM适合模拟信号；TDM适合数字信号
- FDM需要防护频带；TDM需要帧同步

**TDM vs STDM**：
- TDM固定分配时隙，STDM动态分配
- TDM可能有时隙空闲，STDM利用率更高
- STDM需要地址字段标识用户

### E1与T1标准（408重点）

| 标准 | E1（欧洲/中国） | T1（北美/日本） |
|------|----------------|----------------|
| 时隙数 | 32 | 24+1 |
| 每时隙比特数 | 8 | 8 |
| 帧长度 | 256 bit | 193 bit |
| 帧速率 | 8000 帧/s | 8000 帧/s |
| 总速率 | **2.048 Mbit/s** | **1.544 Mbit/s** |

## 原理分析

### 典型考题

**题目1**：E1系统的数据传输速率是多少？

解析：
$$R = 32 \times 8 \times 8000 = 2048000 \text{ bit/s} = 2.048 \text{ Mbit/s}$$

**题目2**：某FDM系统总带宽为 48kHz，每路信号需要 4kHz，防护频带为 0.5kHz，最多可复用多少路？

解析：
$$N = \left\lfloor \frac{48 - 0.5}{4 + 0.5} \right\rfloor + 1 = \left\lfloor \frac{47.5}{4.5} \right\rfloor + 1 = 10 + 1 = 11 \text{ 路}$$

更精确（两端不需要防护频带）：
$$N = \left\lfloor \frac{W + W_g}{W_i + W_g} \right\rfloor = \left\lfloor \frac{48 + 0.5}{4 + 0.5} \right\rfloor = \left\lfloor 10.78 \right\rfloor = 10 \text{ 路}$$

**题目3**：某TDM系统有8个用户，每个用户每帧发送 16 bit，帧速率为 1000 帧/s，求总数据速率和每用户速率。

解析：
- 每用户速率：$16 \times 1000 = 16 \text{ kbit/s}$
- 总速率：$8 \times 16 \times 1000 = 128 \text{ kbit/s}$

## 直观理解

### 复用方式选择记忆

```
信号类型判断：
├─ 模拟信号 → FDM
├─ 数字信号 → TDM 或 CDM
└─ 光信号 → WDM

应用判断：
├─ 有线电视 → FDM
├─ 电话骨干网 → TDM（E1/T1）
├─ 3G移动通信 → CDM
├─ 光纤骨干网 → WDM/DWDM
└─ WiFi/4G → OFDM
```

### 速记口诀

> 频F时T码C光W，四种复用各有招
> E1速率2兆零48，T1速率1兆544

## 协议关联

### 与OSI各层的关系

- 复用技术全部工作在**物理层**
- 但复用思想在上层也有体现：
  - 数据链路层的多路访问协议（争用、轮询）
  - 统计复用（STDM）的思想接近数据链路层的动态带宽分配

### 常见陷阱

1. **WDM和FDM的本质相同**：都是按频率划分，只是WDM特指光纤
2. **E1的TS0和TS16不传用户数据**：TS0用于同步，TS16用于信令
3. **CDM不是扩频通信**：CDM是复用技术，扩频是调制技术，两者相关但不同
4. **STDM不是固定分配**：强调"统计"，即按需动态分配
