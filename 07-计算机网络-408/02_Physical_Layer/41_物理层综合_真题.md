# 41_物理层综合_真题

## 核心概念

本节汇总408真题中物理层的综合计算和概念真题。

### 历年真题分布

物理层在408中通常出**1~2道选择题**，偶尔有计算题。考查内容：
1. 奈奎斯特/香农计算（高频）
2. CDMA码片计算（中频）
3. 曼彻斯特编码波形（中频）
4. 物理层接口特性（中频）
5. 复用技术对比（低频）

## 原理分析

### 真题1：综合计算

**题目（模拟408风格）**：某信道带宽为 3kHz，信噪比为 20dB。

(1) 求信道的最大数据传输速率（香农定理）。
(2) 若要达到 20 kbit/s 的数据传输速率，至少需要几个电平？

**解析**：

(1) 香农定理：
$$\text{SNR} = 10^{20/10} = 100$$
$$C = 3000 \times \log_2(1+100) = 3000 \times \log_2 101 \approx 3000 \times 6.66 = 19980 \text{ bit/s}$$

(2) 由奈奎斯特反推：
$$20000 = 2 \times 3000 \times \log_2 M$$
$$\log_2 M = \frac{20000}{6000} = 3.33$$
$$M = 2^{3.33} \approx 10.1$$

因此 $M$ 至少取 **16**（$\log_2 16 = 4$，此时速率 24000 bit/s > 19980 bit/s 也 < 香农极限...）

注意：实际速率不能超过香农极限 19980 bit/s。所以即使 $M=16$，实际最大速率也只能是 19980 bit/s。若题目条件允许近似，则 $M = 16$ 是奈奎斯特方面的要求，但实际受限于香农极限。

### 真题2：曼彻斯特编码

**题目（模拟408风格）**：10BASE-T以太网使用曼彻斯特编码，线路速率是多少？

A. 10 Mbaud  B. 20 Mbaud  C. 5 Mbaud  D. 40 Mbaud

**答案**：B。曼彻斯特编码效率50%，线路速率 = 10/50% = 20 Mbaud。

### 真题3：物理层设备

**题目（模拟408风格）**：一个集线器连接了4台计算机，另用一个集线器级联扩展，又连接了4台计算机。关于该网络的描述，正确的是？

A. 共有8个冲突域  B. 共有2个冲突域  C. 共有1个冲突域  D. 共有2个广播域

**答案**：C。集线器不隔离冲突域，级联后所有端口仍在同一个冲突域。也只有1个广播域。

### 真题4：PCM计算

**题目（模拟408风格）**：对最高频率为 4kHz 的信号进行PCM编码，采样频率为 8000Hz，每个样本用 8 bit 编码，求PCM编码后的数据速率。

**解析**：
$$R = 8000 \times 8 = 64000 \text{ bit/s} = 64 \text{ kbit/s}$$

这就是数字电话的标准速率。

### 真题5：复用技术

**题目（模拟408风格）**：E1系统的帧周期为 125μs，每帧 32 个时隙，每个时隙 8 bit，求E1的总数据传输速率。

**解析**：
$$R = \frac{32 \times 8}{125 \times 10^{-6}} = \frac{256}{125 \times 10^{-6}} = 2.048 \text{ Mbit/s}$$

## 直观理解

### 考试策略

1. 先看题目涉及哪个知识点
2. 计算题：先判断用奈奎斯特还是香农
3. 编码题：注意曼彻斯特编码效率50%
4. 设备题：记住冲突域/广播域的隔离关系
5. 复用题：E1/T1参数要背熟

## 协议关联

### 物理层知识点关联图

```
物理层
├─ 数据通信基础 → 奈奎斯特/香农（计算）
├─ 编码调制 → 曼彻斯特编码（波形/效率）
├─ 传输介质 → 光纤/双绞线（特性对比）
├─ 接口特性 → 四大特性（概念选择题）
├─ 复用技术 → FDM/TDM/CDMA（对比/计算）
├─ 宽带接入 → ADSL/HFC/FTTH（对比）
└─ 物理层设备 → 中继器/集线器（冲突域/广播域）
```

### 近年真题趋势

- 计算题趋向综合（奈奎斯特+香农+编码效率）
- CDMA码片计算偶有出现
- 概念题注重设备对比（集线器vs交换机vs路由器）
- 宽带接入技术的对比是新趋势
