# 28_数据传输方式

## 核心概念

### 数据传输的分类

数据传输可以从多个维度分类：

**按传输方向**：
- 单工（Simplex）
- 半双工（Half-Duplex）
- 全双工（Full-Duplex）

**按传输方式**：
- 串行传输（Serial）
- 并行传输（Parallel）

**按同步方式**：
- 同步传输（Synchronous）
- 异步传输（Asynchronous）

**按信号形式**：
- 基带传输（Baseband）
- 频带传输（Passband）
- 宽带传输（Broadband）

### 基本术语

- **数据传输速率**：单位时间内传输的比特数（bit/s）
- **波特率**：单位时间内传输的码元数（Baud）
- **信道容量**：信道的最大数据传输速率
- **吞吐量**：单位时间内通过某个网络的实际数据量

## 原理分析

### 各种传输方式的关系

```
数据传输方式
├─ 按方向分
│  ├─ 单工：只能一个方向（广播）
│  ├─ 半双工：可双向但不能同时（对讲机）
│  └─ 全双工：可同时双向（电话）
├─ 按方式分
│  ├─ 串行：一位一位传（远距离）
│  └─ 并行：多位同时传（近距离）
└─ 按同步分
   ├─ 同步：收发双方时钟同步
   └─ 异步：每个字符独立同步
```

### 传输速率的影响因素

$$C = W \log_2 M \quad \text{（奈奎斯特，无噪声）}$$
$$C = W \log_2(1+\text{SNR}) \quad \text{（香农，有噪声）}$$

传输速率受以下因素限制：
1. **信道带宽**：带宽越大，可支持的码元速率越高
2. **噪声**：噪声越大，信道容量越低
3. **编码方式**：编码效率影响有效数据速率
4. **调制方式**：每个码元携带的比特数

## 直观理解

### 类比：交流方式

- **单工**：电视广播，只能收看不能回话
- **半双工**：对讲机，说完"over"才能听对方说
- **全双工**：打电话，两个人可以同时说话

### 类比：排队与并排

- **串行传输**：单车道公路，车一辆一辆过
- **并行传输**：多车道公路，多辆车同时过
- 远距离选串行（布线成本低），近距离可选并行（速度快）

## 协议关联

### 与以太网的关系

- 传统以太网（10BASE-T）使用**半双工**传输（需要CSMA/CD）
- 现代以太网支持**全双工**传输（交换机端口之间）
- 全双工时不需要CSMA/CD

### 与串口的关系

- RS-232串口是**串行、全双工**的传输方式
- 一次传输1位数据
- TxD和RxD独立，实现全双工

### 考试要点

1. 区分单工、半双工、全双工
2. 串行适合远距离，适合高速传输
3. 并行适合近距离（如CPU与内存之间的总线）
4. 以太网的半双工/全双工模式
