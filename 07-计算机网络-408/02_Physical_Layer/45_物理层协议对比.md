# 45_物理层协议对比

## 核心概念

### 常见物理层标准汇总

| 标准 | 组织 | 应用场景 | 接口类型 |
|------|------|----------|----------|
| EIA-232 (RS-232) | EIA | 串口通信 | DB-9/DB-25 |
| RS-485 | EIA | 工业控制 | 差分 |
| V.24/V.28 | ITU-T | 调制解调器 | 与EIA-232类似 |
| X.21 | ITU-T | 数字线路 | 15针连接器 |
| RJ-45 | — | 以太网 | 8P8C |
| USB | USB-IF | 外设连接 | Type-A/B/C |

### EIA-232 vs RS-485

| 特性 | EIA-232 | RS-485 |
|------|---------|--------|
| 信号方式 | 单端 | 差分 |
| 最大距离 | 15m | 1200m |
| 最大速率 | 20 kbit/s | 10 Mbit/s |
| 节点数 | 点对点 | 多点（最多32个） |
| 抗干扰 | 差 | 好 |
| 典型应用 | 串口 | 工业总线 |

### V.24 与 EIA-232 的关系

- V.24 是 ITU-T 标准，定义了**功能特性和规程特性**
- V.28 定义了**电气特性**
- EIA-232 是 EIA 标准，包含了机械、电气、功能、规程四方面
- 两者高度兼容，引脚功能基本一致

### X.21 标准

- 用于**数字**线路的物理层接口
- 比 EIA-232 更高效，引脚更少
- 采用**平衡传输**（差分信号）
- 支持更高的数据速率
- 主要用于电信网络的数字接入

## 原理分析

### 各标准的四大特性对比

| 标准 | 机械特性 | 电气特性 | 功能特性 | 规程特性 |
|------|----------|----------|----------|----------|
| EIA-232 | DB-25/DB-9 | ±3V~±15V单端 | 定义25根引脚 | DTR/DSR/RTS/CTS流程 |
| RS-485 | 无规定 | 差分±1.5V~±5V | 两根信号线 | 多种协议可选 |
| V.24 | 配合V.28 | 配合V.28 | 定义电路功能 | 定义事件顺序 |
| X.21 | 15针连接器 | 平衡传输 | 精简引脚 | 同步规程 |

### 单端信号 vs 差分信号

**单端信号（EIA-232）**：
```
发送端          接收端
  TX ────────── RX
 GND ────────── GND
```
- 信号相对于地线的电压
- 易受共模干扰

**差分信号（RS-485）**：
```
发送端          接收端
  A  ────────── A
  B  ────────── B
```
- 信号是两根线之间的电压差
- A和B上的干扰相同，差分后抵消
- 抗干扰能力强

## 直观理解

### 类比：测量身高

- **单端信号**：以地面为参考测量身高。地面不平时测量不准
- **差分信号**：两个人互为参考测量身高差。地面起伏不影响结果

### 选择标准的考虑

```
需要什么标准？
├─ 短距离点对点 → EIA-232
├─ 长距离多点 → RS-485
├─ 数字线路 → X.21
├─ 高速率 → USB/以太网
└─ 工业环境 → RS-485（抗干扰）
```

## 协议关联

### 与OSI各层的关系

- 所有这些标准都是**物理层**标准
- 只定义接口特性，不定义数据格式
- 数据格式由数据链路层协议定义

### 与以太网物理层的关系

- 以太网也有物理层标准
- 10BASE-T：RJ-45 + 双绞线 + 曼彻斯特编码
- 100BASE-TX：RJ-45 + 5类线 + 4B/5B + MLT-3
- 1000BASE-LC：LC光纤连接器 + 单模光纤

### 考试要点

1. EIA-232是单端信号，RS-485是差分信号
2. EIA-232最大距离15m，RS-485最大距离1200m
3. V.24与EIA-232基本兼容
4. X.21用于数字线路
5. 各标准都定义了四大接口特性
