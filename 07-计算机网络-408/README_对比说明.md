# 计算机网络笔记版本对比说明

本仓库中存在两个计算机网络笔记目录，定位不同，内容互补：

## 版本对比

| 对比项 | `07-计算机网络/` | `07-计算机网络-408/` |
|:---|:---|:---|
| **目标** | Web开发网络基础 | 考研408计算机网络 |
| **格式** | HTML（单页笔记） | Markdown（分篇笔记） |
| **笔记数量** | 31 篇 | 550 篇 |
| **深度** | 概念理解为主 | 理论深度 + 计算 + 真题 |
| **覆盖范围** | TCP/IP、HTTP、DNS等核心协议 | OSI七层模型全面覆盖 |
| **适用场景** | 前端/后端开发面试 | 408统考、计算机考研 |
| **章节划分** | 无明确分层 | 按OSI七层划分7大章 |
| **题型覆盖** | 无 | 概念题 + 计算题 + 真题 |

## `07-计算机网络/`（Web开发版）

- 31个HTML笔记文件
- 涵盖：互联网工作原理、TCP/IP协议栈、HTTP/HTTPS、DNS、Cookie/Session、REST架构等
- 面向JS/Web开发者，侧重实际应用
- 编号从400开始（与其他前端/JS笔记编号体系一致）

## `07-计算机网络-408/`（考研408版）

- 550个Markdown笔记文件
- 7大章节完整覆盖：体系结构、物理层、数据链路层、网络层、传输层、应用层、补充专题
- 每个知识点包含「概念学习 -> 计算练习 -> 真题演练」三层递进
- 附带常用考试公式速查
- 面向408统考考生，理论严谨、计算详细、真题丰富

## 如何选择

- **准备考研408**：使用 `07-计算机网络-408/`，按建议学习顺序从体系结构开始
- **Web开发面试**：使用 `07-计算机网络/`，重点看HTTP、DNS、TCP相关章节
- **两者都需要**：先用408版打好理论基础，再用Web版补充实际应用视角
