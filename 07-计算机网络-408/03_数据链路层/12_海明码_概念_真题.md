# 12_海明码_概念_真题

## 核心概念

- **408高频考法**：给定数据位数求校验位数、给定数据计算海明码、收到码字进行纠错
- **易错点**：校验位的位置（2的幂次）、伴随式的计算、纠错时的位置确定
- **与CRC对比**：海明码纠错（改错位），CRC检错（不改错位，请求重传）

### 海明码 vs CRC 对比表

| 特性 | 海明码 | CRC |
|------|--------|-----|
| 类型 | 纠错码 | 检错码 |
| 纠错能力 | 有（1位） | 无 |
| 检错能力 | 有（2位） | 有（更强） |
| 冗余位 | $r = \lceil \log_2(k+r+1) \rceil$ | 取决于生成多项式 |
| 应用 | 内存ECC | 数据链路层 |

## 原理分析

### 真题1（求校验位数）

**题目**：要发送的数据为10位，采用海明码纠错，至少需要多少位校验位？

解答：
$$2^r \geq 10 + r + 1$$

- $r=3$: $8 \geq 14$? 不满足
- $r=4$: $16 \geq 15$? 满足！

答案：**4位校验位**

### 真题2（编码计算）

**题目**：数据 $D = 1010$，用海明码编码（偶校验）。

**Step1**：$k=4$，$r=3$（$2^3=8 \geq 4+3+1=8$），$n=7$

**Step2**：填入数据

| 位置 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|------|---|---|---|---|---|---|---|
| 类型 | P1 | P2 | d1 | P4 | d2 | d3 | d4 |
| 值 | ? | ? | 1 | ? | 0 | 1 | 0 |

**Step3**：计算校验位

- $P_1$（位置1,3,5,7）：$P_1 \oplus 1 \oplus 0 \oplus 0 = 0 \Rightarrow P_1 = 1$
- $P_2$（位置2,3,6,7）：$P_2 \oplus 1 \oplus 1 \oplus 0 = 0 \Rightarrow P_2 = 0$
- $P_4$（位置4,5,6,7）：$P_4 \oplus 0 \oplus 1 \oplus 0 = 0 \Rightarrow P_4 = 1$

海明码 = **1 0 1 1 0 1 0**

### 真题3（纠错）

**题目**：发送海明码 `1011010`，接收方收到 `1011110`，判断是否有错并纠正。

计算伴随式：
- $S_1 = 1 \oplus 1 \oplus 0 \oplus 0 = 0$（接收值 $S_1 = 1 \oplus 1 \oplus 1 \oplus 0 = 1$）
- $S_2 = 0 \oplus 1 \oplus 1 \oplus 0 = 0$（接收值 $S_2 = 0 \oplus 1 \oplus 1 \oplus 0 = 0$）
- $S_4 = 1 \oplus 0 \oplus 1 \oplus 0 = 0$（接收值 $S_4 = 1 \oplus 1 \oplus 1 \oplus 0 = 1$）

伴随式 $S_4S_2S_1 = 101_2 = 5$

出错位置为**第5位**，将第5位取反：$1 \to 0$

纠正后：`1011010` ✓

### 真题4（概念选择）

**题目**：以下关于海明码的描述，正确的是（ ）

A. 海明码能纠正任意多位错误
B. 海明码的编码效率一定高于CRC
C. 海明码能检测2位错误并纠正1位错误
D. 海明码不需要重传机制

**答案**：C、D

- A错：只能纠正1位错误
- B错：编码效率取决于k和r的关系，不一定高于CRC
- C对：这是海明码的标准能力
- D对：纠错码不需要重传

### 真题5（综合）

**题目**：某系统采用海明码纠错，数据8位，至少需要多少校验位？编码效率是多少？

解答：
$2^r \geq 8 + r + 1$

- $r=3$: $8 \geq 12$? 不满足
- $r=4$: $16 \geq 13$? 满足！

校验位数：4位
码字长度：12位
编码效率：$\eta = \frac{8}{12} = \frac{2}{3} \approx 66.7\%$

## 直观理解

- **海明码纠错就像拼图缺失**：3个校验位给你3条线索，组合起来能定位哪一块拼图放错了
- **伴随式**就像"错误定位密码"：密码的值直接告诉你出错的位置

## 协议关联

- 408中海明码和CRC常常**对比出题**
- 网络中主要用CRC（检错+重传），海明码主要在**存储系统**中使用
- 如果题目问"数据链路层用什么差错控制"，答案是**CRC**，不是海明码
