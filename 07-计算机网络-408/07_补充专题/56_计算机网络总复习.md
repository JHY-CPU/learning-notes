# 56_计算机网络总复习

## 核心概念

- **408计算机网络考试范围**：谢希仁《计算机网络》第7/8版为主
- **七大核心专题**：
  1. 计算机网络体系结构（OSI/TCP-IP模型）
  2. 物理层（信道容量、编码、多路复用）
  3. 数据链路层（以太网、CSMA/CD、PPP）
  4. 网络层（IP、路由、ARP、ICMP）
  5. 传输层（TCP、UDP、拥塞控制）
  6. 应用层（HTTP、DNS、FTP、SMTP）
  7. 网络安全（加密、签名、证书、防火墙、VPN）
- **分值分布（一般情况）**：
  - 选择题：约8-10题（16-20分）
  - 综合题：约1-2题（9-15分）
  - 总分约25-35分

## 原理分析

### 各专题必背公式

**物理层**：
- 奈奎斯特：$C = 2W \log_2 V$（无噪声信道最大速率）
- 香农：$C = W \log_2(1 + S/N)$（有噪声信道最大速率）
- 信噪比：$SNR_{dB} = 10 \log_{10}(S/N)$

**数据链路层**：
- 最小帧长：$L_{min} = 2 \times \tau \times R$
- 信道利用率（CSMA/CD）：$U = \frac{1}{1 + 2a}$，$a = \tau / T_0$

**网络层**：
- 子网可用主机数：$2^n - 2$
- CIDR最长前缀匹配

**传输层**：
- TCP窗口：$swnd = \min(rwnd, cwnd)$
- 慢启动：$cwnd = 2^n$
- 拥塞避免：$cwnd = cwnd + 1/RTT$

**安全**：
- RSA：$C = M^e \mod n$，$M = C^d \mod n$
- 生日攻击：$O(2^{n/2})$

### 考前快速检查清单

**必背端口号**：80(HTTP), 443(HTTPS), 21(FTP), 25(SMTP), 53(DNS), 22(SSH)

**必背状态码**：
- 200 OK, 301 Moved Permanently, 302 Found
- 400 Bad Request, 403 Forbidden, 404 Not Found
- 500 Internal Server Error, 503 Service Unavailable

**TCP状态机**：
- 三次握手：CLOSED→SYN_SENT→SYN_RCVD→ESTABLISHED
- 四次挥手：ESTABLISHED→FIN_WAIT_1→FIN_WAIT_2→TIME_WAIT→CLOSED

**ARP工作过程**：
- 有缓存→直接用
- 无缓存→广播ARP请求→单播ARP响应

**路由协议**：
- RIP：距离向量，16跳无穷大，30秒更新
- OSPF：链路状态，Dijkstra，触发更新
- BGP：路径向量，AS间

### 答题策略

1. **选择题**：
   - 先排除明显错误选项
   - 注意"不正确的是""错误的是"等否定表述
   - 协议端口号和层次归属必须秒答

2. **计算题**：
   - 子网划分：先确定子网掩码，再计算可用数
   - 香农/奈奎斯特：注意单位（Hz, bps, dB）
   - TCP窗口：画图标注cwnd变化
   - IP分片：每片数据长度必须是8的倍数

3. **综合题**：
   - 分析协议交互过程（如HTTP请求完整流程）
   - 抓住关键字段（端口、IP、序列号、确认号）

## 协议关联

- **全协议栈**：HTTP/DNS/FTP/SMTP → TCP/UDP → IP/ICMP/ARP → Ethernet/PPP
- **安全叠加**：TLS（传输层安全）+ IPsec（网络层安全）+ PGP（应用层安全）
- **管理叠加**：SNMP管理网络设备 + MIB存储管理数据
- **QoS叠加**：IntServ/DiffServ + 漏桶/令牌桶 + 拥塞控制
