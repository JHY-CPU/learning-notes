# 63_应用层协议对比_考试

## 核心概念

- **408常考题型**：选择题为主，对比各协议特点
- **关键考点**：协议端口号、传输层、状态特性
- **出题规律**：给出特征，判断协议

## 原理分析

### 典型考题1：协议对比

**题目**：以下使用TCP的协议有（  ）（多选）
A. HTTP
B. DNS
C. FTP
D. DHCP

**答案**：A, C

**解析**：
- HTTP使用TCP（A正确）
- DNS主要使用UDP（B错误）
- FTP使用TCP（C正确）
- DHCP使用UDP（D错误）

### 典型考题2：协议特性

**题目**：以下协议中有状态的是（  ）
A. HTTP
B. FTP
C. DNS
D. SMTP

**答案**：B

**解析**：
- HTTP无状态
- FTP有状态
- DNS无状态
- SMTP无状态

### 典型考题3：协议用途

**题目**：用于发送邮件的协议是（  ）
A. POP3
B. IMAP
C. SMTP
D. HTTP

**答案**：C

### 典型考题4：协议端口

**题目**：HTTPS的端口号是（  ）
A. 80
B. 443
C. 8080
D. 21

**答案**：B

## 直观理解

**做题技巧**：
- TCP协议：HTTP(80)、FTP(20/21)、SMTP(25)、POP3(110)、HTTPS(443)
- UDP协议：DNS(53)、DHCP(67/68)、SNMP(161)
- 有状态：FTP
- 无状态：HTTP

## 协议关联

- **协议与传输层**：TCP还是UDP
- **协议与端口**：每个协议有固定端口
- **408常见组合**：协议对比 + 端口号 = 选择题
