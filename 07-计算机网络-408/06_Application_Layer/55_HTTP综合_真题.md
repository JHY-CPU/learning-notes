# 55_HTTP综合_真题

## 核心概念

- **真题来源**：408统考及各校自命题中涉及HTTP综合
- **高频考点**：RTT计算、HTTP版本、报文分析
- **出题规律**：通常结合TCP和DNS综合考查

## 原理分析

### 真题1（计算题）

**题目**：HTTP/1.1持久连接非流水线，访问含1个HTML和2个图片的页面，需要多少RTT？

**答案**：4 RTT

**解析**：
- TCP握手：1 RTT
- HTML请求响应：1 RTT
- 2个图片：2 RTT
- 总计：1 + 1 + 2 = 4 RTT

### 真题2（选择题）

**题目**：HTTP/1.0默认使用（  ），HTTP/1.1默认使用（  ）。

**答案**：非持久连接；持久连接

### 真题3（计算题）

**题目**：DNS解析20ms，TCP握手30ms，HTTP请求响应20ms。访问含1个HTML和3个图片的页面（HTTP/1.1持久非流水线），总时间是多少？

**答案**：20 + 30 + 20 + 3×20 = 130ms

**解析**：
- DNS：20ms（一次性）
- TCP握手：30ms（一次性）
- HTML：20ms
- 3个图片：3 × 20ms = 60ms
- 总计：20 + 30 + 20 + 60 = 130ms

### 真题4（简答题）

**题目**：HTTP/1.1相比HTTP/1.0有哪些主要改进？

**参考答案**：
1. 默认持久连接
2. 支持Host头部（必需）
3. 支持管道化
4. 支持分块传输编码
5. 支持范围请求
6. 增加缓存控制头部

## 直观理解

**做题技巧**：
- 先确定HTTP版本（1.0非持久，1.1持久）
- 计算RTT时注意是否有缓存
- 结合DNS和TCP的总时间

## 协议关联

- **HTTP + TCP + DNS** = Web访问完整时间
- **HTTP + Cookie** = 状态管理
- **408常见组合**：HTTP版本 + RTT计算 = 综合题
