# 85_应用层传输层_考试

## 核心概念

- **408常考题型**：选择题为主，涉及应用层协议的传输层选择
- **关键考点**：
  - TCP协议：HTTP、FTP、SMTP、POP3、IMAP、SSH
  - UDP协议：DNS、DHCP、SNMP
  - DNS特殊情况：默认UDP，大报文用TCP
- **易混淆点**：
  - DNS既用UDP也用TCP
  - DHCP需要广播所以用UDP

## 原理分析

### 典型考题1：传输层选择

**题目**：以下使用TCP的协议是（  ）（多选）
A. HTTP
B. DNS
C. FTP
D. DHCP

**答案**：A, C

**解析**：
- HTTP使用TCP（A正确）
- DNS主要使用UDP（B错误）
- FTP使用TCP（C正确）
- DHCP使用UDP（D错误）

### 典型考题2：UDP协议

**题目**：以下使用UDP的协议是（  ）
A. HTTP
B. FTP
C. SMTP
D. DNS

**答案**：D

**解析**：
- HTTP、FTP、SMTP都使用TCP
- DNS主要使用UDP

### 典型考题3：DNS传输层

**题目**：关于DNS，以下说法正确的是（  ）
A. DNS只使用UDP
B. DNS只使用TCP
C. DNS主要使用UDP，大报文用TCP
D. DNS使用ICMP

**答案**：C

**解析**：
- DNS默认使用UDP（端口53）
- 响应超过512字节时使用TCP
- 区域传送使用TCP

### 典型考题4：选择依据

**题目**：DNS选择UDP的原因是（  ）
A. 需要可靠传输
B. 需要快速传输
C. 需要广播
D. 需要大文件传输

**答案**：B

## 直观理解

**做题技巧**：
- TCP = 可靠（HTTP、FTP、邮件）
- UDP = 快速（DNS、DHCP、SNMP）
- DNS特殊：默认UDP，大报文TCP

## 协议关联

- **应用层与传输层**：选择依据
- **TCP vs UDP**：各有适用场景
- **408常见组合**：协议 + 传输层 = 选择题高频考点
