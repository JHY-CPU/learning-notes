# 60_应用层端口号总结

## 核心概念

- **知名端口（Well-Known Ports）**：0-1023，分配给常用服务
- **注册端口（Registered Ports）**：1024-49151，分配给用户进程
- **动态端口（Dynamic Ports）**：49152-65535，临时使用
- **408考试重点**：必须熟记常用应用层协议的端口号

## 原理分析

### 常用应用层协议端口号

| 协议 | 端口号 | 传输层 | 用途 |
|------|--------|--------|------|
| FTP控制 | 21 | TCP | 文件传输控制 |
| FTP数据 | 20 | TCP | 文件传输数据 |
| SSH | 22 | TCP | 安全远程登录 |
| TELNET | 23 | TCP | 远程终端 |
| SMTP | 25 | TCP | 发送邮件 |
| DNS | 53 | UDP/TCP | 域名解析 |
| DHCP服务器 | 67 | UDP | 动态主机配置 |
| DHCP客户端 | 68 | UDP | 动态主机配置 |
| HTTP | 80 | TCP | Web服务 |
| POP3 | 110 | TCP | 接收邮件 |
| IMAP | 143 | TCP | 接收邮件 |
| HTTPS | 443 | TCP | 安全Web服务 |
| SNMP代理 | 161 | UDP | 网络管理 |
| SNMP Trap | 162 | UDP | 网络告警 |

### 按传输层分类

**TCP协议**：
- FTP（20/21）
- SSH（22）
- TELNET（23）
- SMTP（25）
- HTTP（80）
- POP3（110）
- IMAP（143）
- HTTPS（443）

**UDP协议**：
- DNS（53，大报文用TCP）
- DHCP（67/68）
- SNMP（161/162）

### 端口号记忆技巧

**TCP端口**：
- 20/21：FTP（文件传输）
- 22：SSH（安全）
- 23：TELNET（远程）
- 25：SMTP（发送邮件）
- 80：HTTP（Web）
- 110：POP3（收邮件）
- 143：IMAP（收邮件）
- 443：HTTPS（安全Web）

**UDP端口**：
- 53：DNS（域名）
- 67/68：DHCP（动态IP）
- 161/162：SNMP（管理）

## 直观理解

**记忆口诀**：
- "21控制20数据，22安全23终端，25发邮件53域名"
- "80看网页443安全，110取邮件143也取"
- "67/68动态IP，161/162管理"

**分类记忆**：
- 文件：FTP(20/21)
- 远程：SSH(22)、TELNET(23)
- 邮件：SMTP(25)、POP3(110)、IMAP(143)
- Web：HTTP(80)、HTTPS(443)
- 基础：DNS(53)、DHCP(67/68)

## 协议关联

- **端口与TCP/UDP**：每个端口对应一个传输层协议
- **端口与应用层**：端口号标识应用层服务
- **408考点**：常用端口号必须死记，常出选择题
- **陷阱**：DHCP端口67和68不可颠倒；DNS主要用UDP但也可用TCP
