# 81_DNS_HTTP_FTP综合_考试

## 核心概念

- **408常考题型**：选择题为主，涉及DNS/HTTP/FTP综合对比
- **关键考点**：
  - 各协议端口号
  - 传输层选择
  - 协议特点对比
- **出题规律**：给出特征，判断协议

## 原理分析

### 典型考题1：端口判断

**题目**：端口号80对应的协议是（  ）
A. DNS
B. HTTP
C. FTP
D. SMTP

**答案**：B

### 典型考题2：传输层判断

**题目**：以下使用UDP的协议是（  ）
A. HTTP
B. FTP
C. DNS
D. SMTP

**答案**：C

### 典型考题3：协议特点

**题目**：以下协议中有状态的是（  ）
A. HTTP
B. DNS
C. FTP
D. SMTP

**答案**：C

### 典型考题4：综合判断

**题目**：关于应用层协议，以下说法正确的是（  ）
A. HTTP使用两个TCP连接
B. FTP是无状态协议
C. DNS使用UDP端口53
D. HTTP使用端口21

**答案**：C

## 直观理解

**做题技巧**：
- DNS 53/UDP，HTTP 80/TCP，FTP 20+21/TCP
- HTTP无状态，FTP有状态
- DNS用UDP，HTTP/FTP用TCP

## 协议关联

- **DNS + HTTP + FTP** = 应用层核心协议
- **端口号 + 传输层** = 必考内容
- **408常见组合**：协议对比题是高频考点
