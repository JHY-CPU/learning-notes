# 87_应用层综合复习

## 核心概念

- **应用层综合复习**：408考试应用层重点知识点汇总
- **考试要求**：掌握所有应用层协议的特点、端口号、工作过程

## 必背知识点

### 1. 端口号（必须死记）

| 协议 | 端口 | 传输层 | 记忆口诀 |
|------|------|--------|---------|
| FTP控制 | 21 | TCP | "控制21" |
| FTP数据 | 20 | TCP | "数据20" |
| SSH | 22 | TCP | "安全22" |
| TELNET | 23 | TCP | "终端23" |
| SMTP | 25 | TCP | "发25" |
| DNS | 53 | UDP/TCP | "域53" |
| DHCP服务器 | 67 | UDP | "服务器67" |
| DHCP客户端 | 68 | UDP | "客户68" |
| HTTP | 80 | TCP | "网页80" |
| POP3 | 110 | TCP | "取110" |
| IMAP | 143 | TCP | "取143" |
| HTTPS | 443 | TCP | "安全443" |
| SNMP代理 | 161 | UDP | "管理161" |
| SNMP Trap | 162 | UDP | "告警162" |

### 2. 协议特点速记

**HTTP**：
- 端口80，TCP，无状态
- HTTP/1.0非持久，HTTP/1.1持久
- 方法：GET、POST、PUT、DELETE
- 状态码：2成功、3重定向、4客户端、5服务器

**FTP**：
- 端口20/21，TCP，有状态，双连接
- 控制连接21，数据连接20（主动）

**邮件系统**：
- SMTP(25)发送，POP3(110)/IMAP(143)接收
- DNS MX记录找邮件服务器

**DNS**：
- 端口53，UDP（大报文TCP）
- 递归查询：主机→本地DNS
- 迭代查询：本地DNS→其他DNS
- 记录：A、AAAA、CNAME、MX、NS、PTR

**DHCP**：
- 端口67/68，UDP
- DORA：Discover→Offer→Request→ACK

### 3. 高频考点

1. **端口号**：必考，送分题
2. **HTTP版本对比**：1.0 vs 1.1 vs 2.0
3. **RTT计算**：非持久/持久/流水线
4. **DNS查询**：递归vs迭代，缓存影响
5. **FTP双连接**：控制+数据
6. **邮件协议**：发送/接收协议
7. **P2P文件分发**：自扩展性，时间计算
8. **HTTPS**：端口443，SSL/TLS

### 4. 常见陷阱

- DNS既用UDP也用TCP
- DHCP端口67/68不可颠倒
- HTTP无状态，但Cookie可模拟状态
- HTTP/2解决HTTP层队头阻塞，TCP层仍阻塞
- FTP有状态，HTTP无状态
- SMTP是推协议，POP3/IMAP是拉协议

## 做题技巧

### 选择题技巧

1. **端口号题**：直接回忆口诀
2. **传输层题**：TCP=可靠，UDP=快速
3. **协议特点题**：无状态/有状态、单连接/双连接
4. **状态码题**：2成功、3重定向、4客户端、5服务器

### 计算题技巧

1. **RTT计算**：先画流程图，标注每段RTT
2. **持久/非持久**：确定HTTP版本
3. **缓存影响**：跳过已缓存的查询步骤
4. **P2P计算**：列三个约束，取最大值
