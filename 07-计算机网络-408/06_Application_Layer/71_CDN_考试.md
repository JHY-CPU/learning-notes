# 71_CDN_考试

## 核心概念

- **408常考题型**：选择题为主，涉及CDN基本概念
- **关键考点**：
  - CDN的作用：减少延迟、减轻源服务器
  - CDN通过DNS将用户导向最近的服务器
  - CDN是分布式Web缓存
- **易混淆点**：
  - CDN不只是缓存，还包含负载均衡等功能

## 原理分析

### 典型考题1：CDN作用

**题目**：CDN的主要作用是（  ）
A. 加密网络数据
B. 减少访问延迟
C. 管理用户认证
D. 路由网络数据

**答案**：B

**解析**：
- CDN将内容缓存到边缘服务器
- 用户从最近的服务器获取内容
- 显著减少访问延迟

### 典型考题2：CDN与DNS

**题目**：CDN通过什么技术将用户导向最近的服务器（  ）
A. TCP
B. DNS
C. HTTP
D. FTP

**答案**：B

**解析**：
- CDN使用DNS（CNAME）将用户导向最近的CDN服务器
- 根据用户位置返回最优服务器IP

### 典型考题3：CDN特点

**题目**：关于CDN，以下说法正确的是（  ）
A. CDN是集中式的
B. CDN可以减轻源服务器负担
C. CDN只用于视频
D. CDN不使用缓存

**答案**：B

## 直观理解

**做题技巧**：
- CDN = 减少延迟 + 减轻源服务器
- CDN通过DNS实现
- CDN是分布式缓存

## 协议关联

- **CDN与DNS**：CDN使用DNS重定向
- **CDN与HTTP**：CDN缓存HTTP内容
- **CDN与Web缓存**：CDN是分布式Web缓存
- **408常见组合**：CDN + DNS + HTTP = 内容分发综合题
