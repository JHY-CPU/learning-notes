# 53_邮件系统综合_真题

## 核心概念

- **真题来源**：408统考及各校自命题中涉及邮件系统
- **高频考点**：邮件协议端口号、发送/接收流程
- **出题规律**：通常给出场景，判断使用的协议

## 原理分析

### 真题1（选择题）

**题目**：用户代理从邮件服务器读取邮件使用的协议是（  ）
A. SMTP
B. POP3
C. HTTP
D. SNMP

**答案**：B

**解析**：
- POP3（或IMAP）用于从服务器读取邮件
- SMTP用于发送邮件
- HTTP用于Web访问
- SNMP用于网络管理

### 真题2（选择题）

**题目**：SMTP使用的端口号是（  ）
A. 110
B. 143
C. 25
D. 80

**答案**：C

**解析**：
- SMTP端口：25
- POP3端口：110
- IMAP端口：143
- HTTP端口：80

### 真题3（简答题）

**题目**：描述邮件从发送到接收涉及的所有协议和端口号。

**参考答案**：
1. 发件人UA → 发件方邮件服务器：SMTP（TCP 25）
2. 发件方邮件服务器 → 收件方邮件服务器：SMTP（TCP 25）
   - 需要DNS MX记录查询
3. 收件人UA → 收件方邮件服务器：POP3（TCP 110）或IMAP（TCP 143）

涉及协议：SMTP、DNS(MX)、POP3或IMAP

### 真题4（选择题）

**题目**：关于邮件协议，以下说法正确的是（  ）
A. SMTP用于接收邮件
B. POP3用于发送邮件
C. SMTP使用端口25
D. IMAP使用端口110

**答案**：C

## 直观理解

**真题做题技巧**：
- SMTP = 发送（端口25）
- POP3 = 接收（端口110）
- IMAP = 接收（端口143）
- DNS MX = 查找邮件服务器

## 协议关联

- **邮件系统与DNS**：MX记录是关键
- **邮件系统与TCP**：所有邮件协议使用TCP
- **408常见组合**：邮件协议 + 端口号 = 选择题高频考点
