# 83_电子邮件系统综合_考试

## 核心概念

- **408常考题型**：选择题为主，涉及邮件系统综合
- **关键考点**：
  - 邮件协议端口号
  - 发送/接收协议
  - MX记录
- **出题规律**：给出场景，判断协议

## 原理分析

### 典型考题1：邮件发送

**题目**：发送邮件使用的协议是（  ）
A. POP3
B. IMAP
C. SMTP
D. HTTP

**答案**：C

### 典型考题2：邮件接收

**题目**：从邮件服务器读取邮件使用的协议是（  ）（多选）
A. SMTP
B. POP3
C. IMAP
D. FTP

**答案**：B, C

### 典型考题3：MX记录

**题目**：MX记录用于（  ）
A. 域名解析
B. 邮件服务器定位
C. 文件传输
D. Web服务

**答案**：B

### 典型考题4：综合判断

**题目**：关于邮件系统，以下说法正确的是（  ）
A. SMTP端口是110
B. POP3用于发送邮件
C. SMTP端口是25
D. IMAP端口是80

**答案**：C

## 直观理解

**做题技巧**：
- SMTP = 发送（25）
- POP3 = 接收（110）
- IMAP = 接收（143）
- MX = 邮件服务器

## 协议关联

- **邮件 + DNS** = MX记录
- **邮件 + TCP** = 可靠传输
- **408常见组合**：邮件协议 + 端口号 = 选择题
