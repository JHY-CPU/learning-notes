# 77_数据链路层设备对比

## 核心概念

- **数据链路层设备**：网桥、交换机
- **相关设备**：中继器/集线器（物理层）、路由器（网络层）
- **核心对比**：工作层次、隔离冲突域/广播域的能力

### 设备对比总表

| 设备 | 层次 | 冲突域 | 广播域 | 转发依据 |
|------|------|--------|--------|---------|
| 中继器 | 物理层 | 不分割 | 不分割 | 信号再生 |
| 集线器 | 物理层 | 不分割 | 不分割 | 比特复制 |
| 网桥 | 数据链路层 | 分割 | 不分割 | MAC地址 |
| 交换机 | 数据链路层 | 分割 | 不分割 | MAC地址 |
| 路由器 | 网络层 | 分割 | 分割 | IP地址 |

### 各设备详解

**中继器/集线器**：
- 工作在物理层
- 不能识别帧结构
- 所有端口在同一冲突域和广播域
- 纯粹的信号放大/复制

**网桥**：
- 工作在数据链路层
- 能识别帧，根据MAC地址转发
- 每个端口是独立冲突域
- 所有端口在同一广播域
- 转发表通过学习源MAC地址建立

**交换机**：
- 多端口网桥
- 每个端口独立带宽
- 硬件转发（ASIC），速度快
- 每个端口独立冲突域
- 同一广播域（除非VLAN）

**路由器**：
- 工作在网络层
- 根据IP地址转发
- 每个端口独立冲突域和广播域
- 可连接异构网络

### 考试重点

- 各设备的工作层次
- 冲突域和广播域的分割
- 转发依据（MAC vs IP）

## 原理分析

### 冲突域计数

公式：冲突域数量 = 所有交换机/路由器端口数之和 + 集线器数量（每台1个）

**例题**：2台8口集线器 + 1台24口交换机 + 1台4口路由器

冲突域 = 2(集线器) + 24(交换机) + 4(路由器) = 30个

### 广播域计数

公式：广播域数量 = 路由器端口数 + VLAN数量（无VLAN时交换机=1）

**例题**：1台4口路由器 + 1台24口交换机（2个VLAN）

广播域 = 4(路由器) + 2(VLAN) = 6个

（假设路由器每个端口连接不同的广播域）

## 直观理解

- **集线器 = 大喇叭**：一个人说话，所有人听到
- **交换机 = 电话交换台**：点对点通话，互不干扰
- **路由器 = 邮局**：根据地址（IP）决定去向

## 协议关联

- 物理层设备不识别帧，数据链路层设备识别帧但不识别IP，网络层设备识别IP
- 三层交换机 = 二层交换 + 路由功能
- 设备对比是408考试的**高频考点**
