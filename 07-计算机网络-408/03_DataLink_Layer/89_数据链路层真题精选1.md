# 89_数据链路层真题精选1

## 核心概念

- **精选408真题**：涵盖CRC、滑动窗口、CSMA/CD、以太网帧结构
- **解题思路**：先判断考点，再套公式

## 原理分析

### 真题1（CRC）

**题目**（408 2013）：数据101001，生成多项式$G(x)=x^3+x^2+1$，求FCS。

**解答**：

$G(x) = 1101$，加3个0：$101001000$

模2除法：
```
       110100
    ___________
1101|101001000
     1101
     ----
      1110
      1101
      ----
       0111
       0000
       ----
        1110
        1101
        ----
         0110
         0000
         ----
          1100
          1101
          ----
           001
```

FCS = **001**

### 真题2（窗口约束）

**题目**（408改编）：数据链路层采用后退N帧协议（GBN），若序号字段为3比特，则发送窗口最大为（ ）

A. 3  B. 7  C. 8  D. 4

**答案**：B. 7

$W_s = 2^3 - 1 = 7$

### 真题3（信道利用率）

**题目**（408改编）：数据传输速率为4kbps，单向传播时延为20ms，要使停等协议的信道利用率达到80%，帧长至少为多少比特？

**解答**：

$$U = \frac{T_f}{T_f + 2T_p} = \frac{T_f}{T_f + 40} \geq 0.8$$

$T_f \geq 0.8(T_f + 40) = 0.8T_f + 32$

$0.2T_f \geq 32$

$T_f \geq 160$ ms

$L = 4000 \times 0.160 = \textbf{640 bit}$

### 真题4（CSMA/CD最小帧长）

**题目**（408改编）：以太网规定最小帧长为64字节，数据传输速率100Mbps，求争用期长度和最大冲突域直径（信号速度$2 \times 10^8$m/s）。

**解答**：

$L_{\min} = 64 \times 8 = 512$ bit

$2\tau = 512 / (100 \times 10^6) = 5.12\ \mu s$

$\tau = 2.56\ \mu s$

$d = 2.56 \times 10^{-6} \times 2 \times 10^8 = \textbf{512 m}$

### 真题5（设备对比）

**题目**（408改编）：若有1台8口集线器、1台24口交换机、1台3口路由器，求冲突域和广播域数（无VLAN）。

**解答**：

冲突域：1(集线器) + 24(交换机) + 3(路由器) = **28个**

广播域：3(路由器端口) = **3个**

## 直观理解

- **真题的特点**：看似复杂，其实都是基本公式的应用。关键是不要慌，列好已知条件再算

- CRC和信道利用率是计算题的两大支柱
