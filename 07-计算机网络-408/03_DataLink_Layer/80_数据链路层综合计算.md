# 80_数据链路层综合计算

## 核心概念

- **数据链路层计算题汇总**：CRC、海明码、信道利用率、窗口大小、最小帧长、ALOHA吞吐率
- **408考试中计算题通常占10-15分**
- **必须掌握**：公式、步骤、单位换算

### 计算公式速查

| 知识点 | 公式 |
|--------|------|
| CRC余数 | 模2除法 |
| 海明码校验位 | $2^r \geq k+r+1$ |
| 停等利用率 | $U = \frac{1}{1+2a}$ |
| GBN窗口 | $W_s \leq 2^n-1$ |
| SR窗口 | $W_s \leq 2^{n-1}$ |
| CSMA/CD最小帧长 | $L_{\min} = 2\tau \times R$ |
| 纯ALOHA吞吐率 | $S = Ge^{-2G}$，$S_{\max} = 1/2e$ |
| 时隙ALOHA吞吐率 | $S = Ge^{-G}$，$S_{\max} = 1/e$ |

## 原理分析

### 综合例题1（CRC + 帧长）

**题目**：数据 $M = 1101011011$，$G(x) = x^4+x+1$，求：
(1) FCS
(2) 发送帧
(3) 发送帧的总比特数

**解答**：

(1) $G(x) = 10011$，加4个0，模2除法得余数 $R = 1101$

(2) 发送帧 = $11010110111101$

(3) 总比特数 = 10 + 4 = **14 bit**

### 综合例题2（海明码）

**题目**：数据 1011，用海明码编码（偶校验）。

(1) 需要几个校验位？
(2) 海明码是什么？

**解答**：

(1) $2^r \geq 4+r+1$，$r=3$，$2^3=8 \geq 8$ ✓

(2) 位置分配：

| 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|---|---|---|---|---|---|---|
| P1 | P2 | 1 | P4 | 0 | 1 | 1 |

- $P_1$（1,3,5,7）：$P_1 \oplus 1 \oplus 0 \oplus 1 = 0$ → $P_1 = 0$
- $P_2$（2,3,6,7）：$P_2 \oplus 1 \oplus 1 \oplus 1 = 0$ → $P_2 = 1$
- $P_4$（4,5,6,7）：$P_4 \oplus 0 \oplus 1 \oplus 1 = 0$ → $P_4 = 0$

海明码 = **0110011**

### 综合例题3（信道利用率）

**题目**：数据率2Mbps，帧长2000bit，单向传播时延20ms。

(1) 停等协议利用率
(2) GBN（$W_s=7$）利用率
(3) 最小帧长（CSMA/CD，$2\tau=40\mu s$）

**解答**：

$T_f = 1$ ms，$T_p = 20$ ms

(1) $U = \frac{1}{1+40} = \frac{1}{41} \approx 2.44\%$

(2) $7 \times 1 = 7$ ms $< 41$ ms

$U = \frac{7}{41} \approx 17.07\%$

(3) $L_{\min} = 40 \times 10^{-6} \times 2 \times 10^6 = 80$ bit = **10字节**

### 综合例题4（ALOHA）

**题目**：时隙ALOHA，$G=0.5$，求吞吐量。

**解答**：

$S = 0.5 \times e^{-0.5} = 0.5 \times 0.6065 = 0.303$

吞吐率 = **30.3%**

### 综合例题5（窗口约束）

**题目**：序号5位，分别求GBN和SR的最大窗口。

**解答**：

GBN：$W_s = 2^5 - 1 = 31$

SR：$W_s = 2^{5-1} = 16$

## 直观理解

- **计算题解题步骤**：①列出已知条件 ②写出相关公式 ③代入计算 ④注意单位
- **单位换算**：距离用m、速度用m/s、时间用s、数据率用bps

## 协议关联

- 数据链路层计算题经常是408的"送分题"——只要公式记得对，计算不失误
- 建议考前**多做练习**，形成计算直觉
