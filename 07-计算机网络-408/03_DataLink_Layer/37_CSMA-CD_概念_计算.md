# 37_CSMA-CD_概念_计算

## 核心概念

- **CSMA/CD核心计算**：争用期 $2\tau$、最小帧长 $L_{\min}$、信道利用率
- **关键公式**：$L_{\min} = 2\tau \times R$，$U = \frac{T_f}{T_f + 2\tau}$（以太网停等等效）
- **二进制指数退避**：第k次重传等待 $\{0,...,2^k-1\}$ 个争用期

### 公式总结

$$2\tau = \frac{2 \times \text{最大距离}}{\text{传播速度}}$$

$$L_{\min} = 2\tau \times R$$

$$U = \frac{T_f}{T_f + 2\tau + T_{\text{IPG}}}$$

其中 $T_{\text{IPG}}$ = 帧间间隔 = 96 bit时间

## 原理分析

### 例题1（最小帧长）

**题目**：以太网数据率100Mbps，最大距离200m，信号传播速度 $2 \times 10^8$ m/s，求最小帧长。

**解答**：

$$\tau = \frac{200}{2 \times 10^8} = 1\ \mu s$$

$$2\tau = 2\ \mu s$$

$$L_{\min} = 2 \times 10^{-6} \times 100 \times 10^6 = 200\text{ bit} = 25\text{ 字节}$$

但以太网规定最小帧长为 **64字节**（512 bit），所以实际最短距离有更严格的限制。

### 例题2（反推最大距离）

**题目**：100Mbps以太网，最小帧长64字节，信号传播速度 $2 \times 10^8$ m/s，求最大冲突域直径。

**解答**：

$$L_{\min} = 64 \times 8 = 512\text{ bit}$$

$$2\tau = \frac{L_{\min}}{R} = \frac{512}{100 \times 10^6} = 5.12\ \mu s$$

$$\tau = 2.56\ \mu s$$

$$d = \tau \times v = 2.56 \times 10^{-6} \times 2 \times 10^8 = 512\text{ m}$$

但考虑中继器等延迟，实际规范最大距离为 100m（100BASE-TX）。

### 例题3（信道利用率）

**题目**：以太网100Mbps，帧长1500字节，帧间间隔96 bit时间，单向传播时延 $5\ \mu s$。

**解答**：

$$T_f = \frac{1500 \times 8}{10^8} = 120\ \mu s$$

$$T_{\text{IPG}} = \frac{96}{10^8} = 0.96\ \mu s$$

$$2\tau = 10\ \mu s$$

$$U = \frac{120}{120 + 10 + 0.96} = \frac{120}{130.96} \approx 91.6\%$$

### 例题4（退避计算）

**题目**：某站第3次重传，争用期 $51.2\ \mu s$，随机数选为5，求等待时间。

**解答**：

第3次重传，$k=3$，范围 $\{0, 1, ..., 2^3-1\} = \{0, 1, ..., 7\}$

随机数 $r = 5$

等待时间 = $5 \times 51.2 = 256\ \mu s$

### 例题5（综合）

**题目**：10Mbps以太网，最大距离2500m，帧长1000字节。

(1) 求争用期
(2) 求最小帧长
(3) 帧长是否满足最小帧长要求？
(4) 求信道利用率

**解答**：

(1) $\tau = 2500 / (2 \times 10^8) = 12.5\ \mu s$

$2\tau = 25\ \mu s$

(2) $L_{\min} = 25 \times 10^{-6} \times 10 \times 10^6 = 250$ bit = 31.25 字节

取整：**32字节**（以太网规定64字节）

(3) 1000字节 > 64字节，满足要求。

(4) $T_f = 1000 \times 8 / 10^7 = 800\ \mu s$

$U = \frac{800}{800 + 25} = \frac{800}{825} \approx 97\%$

## 直观理解

- **最小帧长**保证"说完之前能听到冲突"——就像保证你说话的时间足够长，让远处的回声能传回来
- **退避算法**就像"撞车后的等待"：撞的次数越多，等的时间范围越大

## 协议关联

- CSMA/CD的最小帧长计算是408必考题
- 以太网的最小帧长64字节就是基于CSMA/CD的需要
- 全双工模式下不需要CSMA/CD
