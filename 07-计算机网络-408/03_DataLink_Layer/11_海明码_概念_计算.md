# 11_海明码_概念_计算

## 核心概念

- **海明码计算三要素**：校验位位数 $r$、校验位位置（2的幂次）、各校验位的取值
- **核心公式**：$2^r \geq k + r + 1$（$k$为信息位数，$r$为校验位数）
- **纠错机制**：通过伴随式 $S_3S_2S_1$ 的值定位出错比特

### 计算步骤总结

1. 确定校验位数 $r$
2. 确定校验位位置（$2^0, 2^1, 2^2, ...$）
3. 将数据位填入非校验位位置
4. 计算各校验位的值（偶校验/奇校验）
5. 得到完整的海明码

### 校验位计算规则（偶校验）

每个校验位 $P_i$ 负责的位置：将位置编号写成二进制，**第 $i$ 位（从0开始数）为1**的所有位置。

## 原理分析

### 完整例题1

**题目**：信息码 $1011$，用海明码编码（偶校验）。

**Step1**：确定校验位数

$k = 4$，求最小 $r$ 使 $2^r \geq 4 + r + 1$

- $r=2$: $4 \geq 7$? 不满足
- $r=3$: $8 \geq 8$? 满足！

所以 $r = 3$，总长度 $n = 7$。

**Step2**：分配位置

| 位置 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|------|---|---|---|---|---|---|---|
| 二进制 | 001 | 010 | 011 | 100 | 101 | 110 | 111 |
| 类型 | P1 | P2 | d1 | P4 | d2 | d3 | d4 |
| 值 | ? | ? | 1 | ? | 0 | 1 | 1 |

**Step3**：计算校验位

- $P_1$：检查二进制位置第0位=1的位置 → 位置1, 3, 5, 7
  - $P_1 \oplus 1 \oplus 0 \oplus 1 = 0$（偶校验要求和为0）
  - $P_1 \oplus 0 = 0 \Rightarrow P_1 = 0$

- $P_2$：检查二进制位置第1位=1的位置 → 位置2, 3, 6, 7
  - $P_2 \oplus 1 \oplus 1 \oplus 1 = 0$
  - $P_2 \oplus 1 = 0 \Rightarrow P_2 = 1$

- $P_4$：检查二进制位置第2位=1的位置 → 位置4, 5, 6, 7
  - $P_4 \oplus 0 \oplus 1 \oplus 1 = 0$
  - $P_4 \oplus 0 = 0 \Rightarrow P_4 = 0$

**Step4**：海明码 = **0 1 1 0 0 1 1**

### 完整例题2（纠错）

**题目**：发送海明码 `0110011`，接收方收到 `0110111`（位置5出错），请纠错。

**Step1**：计算伴随式

- $S_1 = P_1 \oplus d_1 \oplus d_2 \oplus d_4 = 0 \oplus 1 \oplus 0 \oplus 1 = 0$?
  - 接收值：$S_1 = 0 \oplus 1 \oplus 1 \oplus 1 = 1$

- $S_2 = P_2 \oplus d_1 \oplus d_3 \oplus d_4 = 1 \oplus 1 \oplus 1 \oplus 1 = 0$?
  - 接收值：$S_2 = 1 \oplus 1 \oplus 1 \oplus 1 = 0$

- $S_4 = P_4 \oplus d_2 \oplus d_3 \oplus d_4 = 0 \oplus 0 \oplus 1 \oplus 1 = 0$?
  - 接收值：$S_4 = 0 \oplus 1 \oplus 1 \oplus 1 = 1$

**Step2**：伴随式 $S_4S_2S_1 = 101_2 = 5$

出错位置是**第5位**，将第5位取反：$0 \to 1$? 原来是0收到1，纠正为0。

纠正后：`0110011` ✓

### 例题3（求最小校验位）

**题目**：数据有11位，需要多少个校验位？

$2^r \geq 11 + r + 1 = 12 + r$

- $r=3$: $8 \geq 15$? 不满足
- $r=4$: $16 \geq 16$? 满足！

所以需要 **4个校验位**，总长度15位。

## 直观理解

- **伴随式就像GPS定位**：3个校验位就像3个信号塔，通过它们的"异常"组合确定出错位置
- **位置编号的二进制**天然编码了"哪些校验组包含该位置"，这就是海明码的精妙之处
- **记忆方法**：校验位在2的幂次位置（1,2,4,8），因为二进制中只有1个1

## 协议关联

- 海明码与CRC的根本区别：**纠错 vs 检错**
- 海明码校验位数：$r \approx \log_2 k$，编码效率约 $\frac{k}{k+\log_2 k}$
- 408考试中，海明码的计算题必须**步骤清晰**，否则容易出错
- 海明码纠错后不需要重传，但只能纠正1位错误

### 考试技巧

1. 先列位置表，把校验位位置标出来
2. 用二进制位置号判断每个校验位负责哪些位
3. 偶校验：各组XOR结果为0；奇校验：各组XOR结果为1
4. 纠错时：伴随式的值直接给出出错位置
