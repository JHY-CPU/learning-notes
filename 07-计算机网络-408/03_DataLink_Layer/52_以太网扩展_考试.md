# 52_以太网扩展_考试

## 核心概念

- **408考点**：冲突域/广播域数量计算、扩展设备的对比、碰撞检测范围
- **高频题型**：给定网络拓扑，计算冲突域和广播域数量

## 原理分析

### 真题1（冲突域计算）

**题目**：一个网络中有3台集线器和1台交换机，集线器各有8口，交换机有24口。问该网络有多少个冲突域？

**解答**：

- 3台集线器：每台1个冲突域 → 3个
- 1台交换机：24个端口24个冲突域 → 24个
- 总冲突域 = 3 + 24 = **27个**

（如果集线器连接到交换机，集线器的冲突域不变，仍然是1个）

### 真题2（广播域计算）

**题目**：1台24口交换机（不划分VLAN），2台8口集线器，1台3口路由器。求广播域数量。

**解答**：

- 交换机（无VLAN）：1个广播域
- 集线器：1个广播域
- 路由器：3个端口，3个广播域

但要注意连接关系。假设路由器的每个端口连接不同的网络：

如果交换机和集线器都在路由器的同一端口下，则共享1个广播域。

总广播域 = 路由器端口数 = **3个**

### 真题3（CSMA/CD范围）

**题目**：两台计算机通过集线器连接，CSMA/CD的冲突检测范围包括什么？

**解答**：

CSMA/CD的冲突检测范围 = 整个冲突域 = 集线器所有端口 + 连接的设备

如果多台集线器级联，整个级联系统是一个冲突域。

### 真题4（综合判断）

**题目**：以下说法正确的是（ ）

A. 集线器可以隔离冲突域
B. 交换机可以隔离广播域
C. 路由器可以隔离冲突域和广播域
D. 网桥工作在网络层

**答案**：C

- A错：集线器不隔离冲突域
- B错：交换机不隔离广播域（除非有VLAN）
- C对
- D错：网桥工作在数据链路层

### 真题5（网络设计）

**题目**：某公司有3个部门，要求各部门之间隔离广播域，使用什么设备最合理？

**答案**：使用**路由器**（每个部门一个端口），或使用**三层交换机划分VLAN**。

## 直观理解

- **计算冲突域**：找"共享介质"的设备。集线器=1个冲突域，交换机每端口=1个
- **计算广播域**：找"隔离广播"的设备。路由器每端口=1个广播域，VLAN=1个广播域

## 协议关联

- 冲突域只在半双工共享介质环境中有意义
- 全双工交换式以太网中没有冲突（每端口一个冲突域但只有一个设备）
- 路由器隔离广播域是网络分段的重要原因
