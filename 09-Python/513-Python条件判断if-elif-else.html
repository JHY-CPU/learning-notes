🔀 Python 条件判断 if-elif-else

if 语句、elif 多分支、三元表达式、match-case 模式匹配 (3.10+)、嵌套条件。

if 语句基础

// ========== 基本语法 ========== if condition: # 条件为 True 时执行 pass // ========== if-else ========== age = 18 if age >= 18: print("已成年") else: print("未成年") // ========== if-elif-else ========== score = 85 if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" elif score >= 60: grade = "D" else: grade = "F" // elif 是 else-if 的缩写 // 可以有任意多个 elif // else 可选,放在最后 // ========== 真值判断 ========== // Python 会自动将条件转换为 bool // ✅ Pythonic (隐式布尔): name = "Alice" if name: # name 非空 → True print(f"Hello, {name}") items = [] if not items: # items 为空 → True print("列表为空") count = 0 if count is not None: # 区分 0 和 None print(f"count = {count}") // ❌ 不推荐: if name != "": # 直接 if name: 更简洁 if len(items) > 0: # 直接 if items: 更简洁 if count == 0: # 如果本意是检查 None,用 is

条件表达式 (三元运算符)

// ========== 三元表达式 ========== // value_if_true if condition else value_if_false age = 20 status = "成年" if age >= 18 else "未成年" print(status) # "成年" // 相当于: if age >= 18: status = "成年" else: status = "未成年" // ========== 嵌套三元表达式 ========== score = 85 grade = "A" if score >= 90 else "B" if score >= 80 else "C" if score >= 70 else "D" if score >= 60 else "F" // 但嵌套三元可读性差,建议拆开! grade = ( "A" if score >= 90 else "B" if score >= 80 else "C" if score >= 70 else "D" if score >= 60 else "F" ) // ========== 短路替代 ========== // 简单的条件赋值可以用 or name = user_input or "default" // 等价于: name = user_input if user_input else "default" // ========== 实战用法 ========== def get_discount(price, is_vip): """VIP 打 8 折,非 VIP 满 100 打 9 折""" discount = 0.8 if is_vip else (0.9 if price >= 100 else 1.0) return price * discount get_discount(200, True) # 160.0 get_discount(200, False) # 180.0 get_discount(50, False) # 50.0

match-case 模式匹配

// ========== match-case (Python 3.10+) ========== // 类似其他语言的 switch-case,但更强大 // 基本: status_code = 404 match status_code: case 200: print("OK") case 201: print("Created") case 404: print("Not Found") case 500: print("Server Error") case _: # 默认分支,类似 default print("Unknown") // ========== 合并分支 ========== match status_code: case 200 | 201 | 204: # 多个值匹配同一分支 print("Success") case 301 | 302: print("Redirect") case 400 | 401 | 403 | 404: print("Client Error") case 500 | 502 | 503: print("Server Error") case _: print("Unknown") // ========== 解包匹配 ========== point = (3, 4) match point: case (0, 0): print("原点") case (x, 0): print(f"X 轴上: x={x}") case (0, y): print(f"Y 轴上: y={y}") case (x, y): print(f"坐标: ({x}, {y})") // ========== 守卫条件 ========== match value: case x if x < 0: print("负数") case 0: print("零") case x if x > 0: print("正数") // ========== 字典匹配 ========== user = {"name": "Alice", "role": "admin"} match user: case {"role": "admin"}: print("管理员") case {"role": "user"}: print("普通用户") case _: print("未知")

嵌套条件与最佳实践

// ========== 嵌套条件 ========== // 避免过深嵌套 // ❌ 深层嵌套: if user.is_active: if user.has_permission: if user.quota > 0: process(user) else: print("quota 不足") else: print("无权限") else: print("用户未激活") // ✅ 提前返回 (Guard Clause): if not user.is_active: print("用户未激活") return # 或 exit,或 raise if not user.has_permission: print("无权限") return if user.quota <= 0: print("quota 不足") return process(user) // ========== 条件简化 ========== // ❌ 冗余的 True/False 比较: if is_valid == True: # 直接 if is_valid: pass if is_empty == False: # 直接 if not is_empty: pass // ❌ 多余的三元: result = True if condition else False # 直接 result = bool(condition) // ✅ 用 in 简化多条件: if x == 1 or x == 2 or x == 3: # ❌ pass if x in (1, 2, 3): # ✅ pass // ✅ 用 any/all: if all(x > 0 for x in numbers): # 全部正数 pass if any(x < 0 for x in numbers): # 有负数 pass
💡 条件判断要点: (1) 用 if x: 代替 if x == True:; (2) 用 if not x: 代替 if x == False:; (3) 三元表达式适合简单条件; (4) match-case (3.10+) 比多重 elif 更清晰; (5) 提前返回代替深层嵌套。
