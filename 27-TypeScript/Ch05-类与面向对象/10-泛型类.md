# 泛型类

## 一、概念说明

泛型类在类定义时引入类型参数，使类可以存储和操作任意类型的值，同时保持类型安全。常见于集合类（Stack、Queue、Map 等）和容器类。

## 二、具体用法

### 2.1 基本泛型类

```typescript
class Stack<T> {
  private items: T[] = [];

  push(item: T): void {
    this.items.push(item);
  }

  pop(): T | undefined {
    return this.items.pop();
  }

  peek(): T | undefined {
    return this.items[this.items.length - 1];
  }

  size(): number {
    return this.items.length;
  }
}

const numStack = new Stack<number>();
numStack.push(1);
numStack.push(2);
numStack.push(3);
console.log(numStack.pop());  // 输出: 3
console.log(numStack.peek()); // 输出: 2

const strStack = new Stack<string>();
strStack.push("hello");
strStack.push("world");
console.log(strStack.pop()); // 输出: world
```

### 2.2 泛型约束

```typescript
class SortedList<T extends { compareTo(other: T): number }> {
  private items: T[] = [];

  add(item: T): void {
    this.items.push(item);
    this.items.sort((a, b) => a.compareTo(b));
  }

  getAll(): T[] {
    return [...this.items];
  }
}

class Number implements { compareTo(other: Number): number } {
  constructor(public value: number) {}
  compareTo(other: Number): number {
    return this.value - other.value;
  }
}

const list = new SortedList<Number>();
list.add(new Number(3));
list.add(new Number(1));
list.add(new Number(2));
console.log(list.getAll().map(n => n.value));
// 输出: [1, 2, 3]
```

## 三、注意事项与常见陷阱

1. **类级别泛型**：类型参数在整个类中共享
2. **静态成员不能用泛型**：静态方法/属性不能引用类的类型参数
3. **实例化时指定类型**：`new Stack<number>()`
4. **类型推断**：`new Stack()` 从初始数据推断类型
