# 类型保护与类

## 一、概念说明

在类的继承体系中，`instanceof` 操作符是最常用的类型保护，用于检查对象是否为某个类的实例。它在运行时检查原型链，同时帮助 TypeScript 收窄类型。类型保护让联合类型的类安全地访问各自的特有成员。

## 二、具体用法

### 2.1 instanceof 收窄

```typescript
class Cat {
  meow(): string { return "喵~"; }
}

class Dog {
  bark(): string { return "汪汪!"; }
}

function makeSound(animal: Cat | Dog): string {
  if (animal instanceof Cat) {
    return animal.meow(); // 收窄为 Cat
  }
  return animal.bark();   // 收窄为 Dog
}

console.log(makeSound(new Cat())); // 输出: 喵~
console.log(makeSound(new Dog())); // 输出: 汪汪!
```

### 2.2 错误处理中的 instanceof

```typescript
class ValidationError extends Error {
  constructor(public field: string, message: string) {
    super(message);
  }
}

class NetworkError extends Error {
  constructor(public statusCode: number, message: string) {
    super(message);
  }
}

function handleError(error: unknown): string {
  if (error instanceof ValidationError) {
    return `字段 "${error.field}" 验证失败: ${error.message}`;
  }
  if (error instanceof NetworkError) {
    return `网络错误 [${error.statusCode}]: ${error.message}`;
  }
  if (error instanceof Error) {
    return `未知错误: ${error.message}`;
  }
  return "未知异常";
}

console.log(handleError(new ValidationError("email", "格式无效")));
// 输出: 字段 "email" 验证失败: 格式无效
console.log(handleError(new NetworkError(404, "未找到")));
// 输出: 网络错误 [404]: 未找到
```

### 2.3 自定义类型谓词

```typescript
class Circle {
  constructor(public radius: number) {}
  isCircle(): this is Circle { return true; }
}

class Rectangle {
  constructor(public width: number, public height: number) {}
  isRectangle(): this is Rectangle { return true; }
}

// 使用自定义类型谓词
function getArea(shape: Circle | Rectangle): number {
  if ("radius" in shape) {
    // 使用 in 操作符收窄
    return Math.PI * shape.radius ** 2;
  }
  return shape.width * shape.height;
}
```

### 2.4 抽象类与类型保护

```typescript
abstract class Shape {
  abstract readonly kind: string;
  abstract area(): number;
}

class CircleShape extends Shape {
  kind = "circle" as const;
  constructor(public radius: number) { super(); }
  area() { return Math.PI * this.radius ** 2; }
}

class RectShape extends Shape {
  kind = "rectangle" as const;
  constructor(public width: number, public height: number) { super(); }
  area() { return this.width * this.height; }
}

function describe(shape: Shape): string {
  switch (shape.kind) {
    case "circle":
      return `圆形面积: ${shape.area().toFixed(2)}`;
    case "rectangle":
      return `矩形面积: ${shape.area().toFixed(2)}`;
  }
}
```

## 三、注意事项与常见陷阱

1. **`instanceof` 检查原型链**：继承关系会影响检查结果（子类实例也是父类实例）
2. **只对类有效**：接口没有运行时存在，不能用 `instanceof`
3. **多重继承不支持**：一个对象只能是一个类的实例
4. **自定义 `instanceof`**：可以定义 `[Symbol.hasInstance]` 自定义检查逻辑
5. **`in` 操作符**：可用于检查对象是否有某个属性，辅助类型收窄
6. **判别属性**：使用字面量类型属性（如 `kind`）配合 switch 进行穷尽检查
