# 组件类型

## 一、概念说明

Vue 3 提供了多种组件类型：`Component`、`DefineComponent`、`FunctionalComponent` 等。理解这些类型的区别和用法，有助于在动态组件、组件引用等场景中正确使用类型。

## 二、具体用法

### 2.1 DefineComponent 类型

```typescript
import type { DefineComponent } from 'vue';

// DefineComponent — defineComponent 的返回类型
// 泛型参数：Props, RawBindings, D, C, M, Mixins, Extends, E, EE
type MyComponent = DefineComponent<
  { title: string },           // Props
  {},                          // RawBindings
  {},                          // Data
  {},                          // Computed
  {},                          // Methods
  {},                          // Mixins
  {},                          // Extends
  { change: [value: string] }  // Emits
>;

// 使用
const components: Record<string, DefineComponent> = {
  UserCard: UserCardComponent,
  Profile: ProfileComponent,
};
```

### 2.2 动态组件类型

```vue
<script setup lang="ts">
import { shallowRef, markRaw } from 'vue';
import type { Component } from 'vue';
import UserCard from './UserCard.vue';
import Settings from './Settings.vue';

// Component 类型 — 通用组件类型
const tabs: Record<string, Component> = {
  user: markRaw(UserCard),
  settings: markRaw(Settings),
};

const currentTab = shallowRef<Component>(tabs.user);

const switchTab = (name: keyof typeof tabs) => {
  currentTab.value = tabs[name];
};
</script>

<template>
  <component :is="currentTab" />
</template>
```

### 2.3 组件引用类型

```vue
<script setup lang="ts">
import { ref } from 'vue';
import type { ComponentExposed } from 'vue-component-type-helpers';
import MyModal from './MyModal.vue';

// Ref 类型指向组件实例
const modalRef = ref<InstanceType<typeof MyModal>>();

// 或使用 ComponentExposed（需要 vue-component-type-helpers）
const modalRef2 = ref<ComponentExposed<typeof MyModal>>();

// 调用组件暴露的方法
const openModal = () => {
  modalRef.value?.open(); // 类型安全
};
</script>
```

### 2.4 FunctionalComponent 类型

```typescript
import type { FunctionalComponent } from 'vue';

// 函数式组件
const Greeting: FunctionalComponent<{ name: string }> = (props) => {
  return `Hello, ${props.name}`;
};

Greeting.props = {
  name: { type: String, required: true },
};
```

### 2.5 组件 props/emits 类型提取

```typescript
import type { ComponentProps, ComponentEmit, ComponentExposed } from 'vue-component-type-helpers';
import MyComponent from './MyComponent.vue';

// 提取组件的 Props 类型
type Props = ComponentProps<typeof MyComponent>;

// 提取组件的 Emits 类型
type Emits = ComponentEmit<typeof MyComponent>;

// 提取组件暴露的类型
type Exposed = ComponentExposed<typeof MyComponent>;
```

## 三、注意事项与常见陷阱

1. **`Component` 是最通用的组件类型**：适用于任何组件
2. **`DefineComponent` 包含完整的类型信息**：用于需要精确类型的场景
3. **`InstanceType<typeof Component>` 获取实例类型**：用于 ref 引用
4. **`markRaw` 防止组件被深度代理**：在动态组件中推荐使用
5. **`vue-component-type-helpers`**：官方提供类型提取工具
