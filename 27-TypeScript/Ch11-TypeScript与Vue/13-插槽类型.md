# 插槽类型

## 一、概念说明

Vue 3 的插槽（Slots）可以传递数据给父组件，TypeScript 支持插槽参数的类型推断。通过泛型可以定义插槽的作用域数据类型，确保父组件正确使用插槽参数。

## 二、具体用法

### 2.1 基本插槽类型

```vue
<!-- 子组件: UserList.vue -->
<script setup lang="ts">
interface User {
  id: number;
  name: string;
  email: string;
}

const props = defineProps<{
  users: User[];
}>();
</script>

<template>
  <ul>
    <li v-for="user in props.users" :key="user.id">
      <!-- 作用域插槽 — 传递 user 数据 -->
      <slot :user="user" :index="user.id"></slot>
    </li>
  </ul>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <UserList :users="users">
    <!-- slotProps 自动有类型 -->
    <template #default="{ user, index }">
      <!-- user: User, index: number -->
      <span>{{ index }}. {{ user.name }}</span>
    </template>
  </UserList>
</template>
```

### 2.2 具名插槽类型

```vue
<!-- 子组件: Layout.vue -->
<script setup lang="ts">
defineSlots<{
  default(): any;
  header(props: { title: string }): any;
  footer(props: { year: number }): any;
}>();
</script>

<template>
  <div class="layout">
    <header>
      <slot name="header" title="默认标题"></slot>
    </header>
    <main>
      <slot></slot>
    </main>
    <footer>
      <slot name="footer" :year="2024"></slot>
    </footer>
  </div>
</template>
```

```vue
<!-- 父组件 -->
<template>
  <Layout>
    <template #header="{ title }">
      <!-- title: string -->
      <h1>{{ title }}</h1>
    </template>

    <template #footer="{ year }">
      <!-- year: number -->
      <p>Copyright {{ year }}</p>
    </template>

    <p>主要内容</p>
  </Layout>
</template>
```

### 2.3 Slots 类型

```typescript
import type { Slots } from 'vue';

// 组件内部访问 slots 类型
const slots = useSlots();

// 检查插槽是否存在
if (slots.header) {
  // header 插槽存在
}

// 插槽函数类型
type SlotFunction = (props?: Record<string, unknown>) => VNode[];
```

### 2.4 条件插槽渲染

```vue
<script setup lang="ts">
const slots = useSlots();

// 动态检查插槽
const hasHeader = computed(() => !!slots.header);
const hasFooter = computed(() => !!slots.footer);
</script>

<template>
  <div>
    <header v-if="hasHeader">
      <slot name="header"></slot>
    </header>

    <main>
      <slot></slot>
    </main>

    <footer v-if="hasFooter">
      <slot name="footer"></slot>
    </footer>
  </div>
</template>
```

## 三、注意事项与常见陷阱

1. **`defineSlots` 是 Vue 3.3+ 特性**：旧版本需要手动处理类型
2. **插槽参数在模板中自动推断**：不需要显式标注
3. **`Slots` 类型用于组件内部**：检查插槽是否存在
4. **作用域插槽的参数名应有语义**：`{ user }` 比 `{ item }` 更清晰
5. **插槽的返回类型通常是 `any`**：因为可以返回任意 VNode
