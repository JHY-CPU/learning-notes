# 高级类型概述

## 一、概念说明

TypeScript 的类型系统是**图灵完备**的——可以在编译时执行任意计算。高级类型利用条件类型、映射类型、模板字面量类型等特性，在类型层面实现复杂的逻辑运算。

## 二、具体用法

### 2.1 类型系统能力

```typescript
// 条件判断
type If<C extends boolean, T, F> = C extends true ? T : F;
type R1 = If<true, string, number>;  // string
type R2 = If<false, string, number>; // number

// 递归
type DeepReadonly<T> = T extends object
  ? { readonly [K in keyof T]: DeepReadonly<T[K]> }
  : T;

// 字符串操作
type Greeting = `Hello, ${"World"}`; // "Hello, World"
```

### 2.2 类型编程模式

```typescript
// 组合类型操作
type Flatten<T> = T extends (infer U)[] ? U : T;
type R = Flatten<string[]>; // string

// 类型安全的链式操作
type Chain<T> = T extends { then: (fn: (v: infer V) => infer U) => any }
  ? Chain<U>
  : T;

console.log("类型系统可以做编译时计算");
// 输出: 类型系统可以做编译时计算
```

## 三、注意事项与常见陷阱

1. **类型编程只在编译时运行**：不影响运行时性能
2. **复杂类型可能拖慢编译**：过度递归会导致编译超时
3. **可读性优先**：代码是给人读的，类型也是
4. **渐进学习**：从基础类型操作开始，逐步掌握高级技巧
