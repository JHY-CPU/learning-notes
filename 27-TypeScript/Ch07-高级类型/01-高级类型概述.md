# 高级类型概述

## 一、概念说明

TypeScript 的类型系统是**图灵完备**的——可以在编译时执行任意计算。高级类型利用条件类型、映射类型、模板字面量类型等特性，在类型层面实现复杂的逻辑运算。掌握高级类型是成为 TypeScript 高级开发者的关键。

## 二、具体用法

### 2.1 类型系统能力

```typescript
// 条件判断
type If<C extends boolean, T, F> = C extends true ? T : F;
type R1 = If<true, string, number>;  // string
type R2 = If<false, string, number>; // number

// 递归
type DeepReadonly<T> = T extends object
  ? { readonly [K in keyof T]: DeepReadonly<T[K]> }
  : T;

// 字符串操作
type Greeting = `Hello, ${"World"}`; // "Hello, World"
```

### 2.2 类型编程模式

```typescript
// 组合类型操作
type Flatten<T> = T extends (infer U)[] ? U : T;
type R = Flatten<string[]>; string

// 类型安全的链式操作
type Chain<T> = T extends { then: (fn: (v: infer V) => infer U) => any }
  ? Chain<U>
  : T;

// 实用工具类型
type NonNullableProps<T> = {
  [K in keyof T]: NonNullable<T[K]>;
};
```

### 2.3 高级类型的实际应用

```typescript
// API 类型生成
type Endpoint<TMethod extends string, TPath extends string> = {
  method: TMethod;
  path: TPath;
};

type GetUsers = Endpoint<"GET", "/users">;
type CreateUser = Endpoint<"POST", "/users">;

// 类型安全的状态管理
type State = "idle" | "loading" | "success" | "error";
type Transition<S extends State> =
  S extends "idle" ? "loading" :
  S extends "loading" ? "success" | "error" :
  S extends "success" ? "idle" :
  S extends "error" ? "idle" : never;
```

### 2.4 常用高级类型模式

```typescript
// 1. 条件分支：T extends U ? X : Y
// 2. 类型推断：infer R
// 3. 属性遍历：[K in keyof T]
// 4. 键重映射：K as NewK
// 5. 递归处理：T extends object ? ... : T
// 6. 模板组合：`${A}-${B}`
```

## 三、注意事项与常见陷阱

1. **类型编程只在编译时运行**：不影响运行时性能，但可能影响编译速度
2. **复杂类型可能拖慢编译**：过度递归会导致编译超时
3. **可读性优先**：代码是给人读的，类型也是
4. **渐进学习**：从基础类型操作开始，逐步掌握高级技巧
5. **类型调试**：悬停查看类型、使用 `// @ts-expect-error` 测试
6. **文档化**：复杂高级类型必须有注释说明
