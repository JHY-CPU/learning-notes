# 模板字面量类型详解

## 一、概念说明

模板字面量类型使用模板字符串语法在类型层面进行字符串操作，支持拼接、联合展开和内置工具类型。

## 二、具体用法

### 2.1 类型级字符串拼接

```typescript
type CSS = `${number}px` | `${number}%` | `${number}rem`;
const size: CSS = "16px"; // ✅
// const bad: CSS = "16em"; // ❌
```

### 2.2 联合展开

```typescript
type Color = "red" | "green" | "blue";
type Size = "sm" | "md" | "lg";
type Combo = `${Color}-${Size}`;
// 9 种组合
```

### 2.3 内置工具类型

```typescript
type Upper = Uppercase<"hello">;     // "HELLO"
type Lower = Lowercase<"HELLO">;     // "hello"
type Cap = Capitalize<"hello">;      // "Hello"
type Uncap = Uncapitalize<"Hello">;  // "hello"
```

### 2.4 类型安全的 CSS 类名

```typescript
type Theme = "light" | "dark";
type Component = "btn" | "card" | "nav";
type State = "active" | "disabled";

type CSSClass = `${Theme}__${Component}--${State}`;
const cls: CSSClass = "light__btn--active";
```

## 三、注意事项与常见陷阱

1. **联合类型笛卡尔展开**：组合数量可能爆炸
2. **类型参数必须 `extends string`**
3. **与 `infer` 配合提取子串**
4. **性能注意**：过度展开影响编译速度
