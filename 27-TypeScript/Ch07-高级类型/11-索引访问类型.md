# 索引访问类型

## 一、概念说明

索引访问类型 `T[K]` 获取对象类型 T 中键 K 对应的值类型。类似于对象的属性访问，但在类型层面操作。

## 二、具体用法

### 2.1 基本索引访问

```typescript
interface User {
  id: number;
  name: string;
  metadata: { age: number; city: string };
}

type UserId = User["id"];          // number
type Name = User["name"];          // string
type Metadata = User["metadata"];  // { age: number; city: string }

// 嵌套访问
type Age = User["metadata"]["age"]; // number
```

### 2.2 联合键访问

```typescript
type StringOrNumber = User["id" | "name"]; // number | string
type AllValues = User[keyof User];         // number | string | { ... }
```

### 2.3 数组元素类型

```typescript
type Arr = string[];
type Elem = Arr[number]; // string

type Tuple = [string, number, boolean];
type TupleElem = Tuple[number]; // string | number | boolean
```

## 三、注意事项与常见陷阱

1. **用方括号**：`T[K]` 不是 `T.K`
2. **K 必须是 T 的键**：除非 T 有索引签名
3. **嵌套访问**：`T["a"]["b"]` 访问深层属性
4. **配合 `keyof`**：`T[keyof T]` 获取所有值类型的联合
