# 泛型函数

## 一、概念说明

泛型函数引入**类型参数**（如 `<T>`），使函数可以处理任意类型而保持类型安全。类型参数在调用时被推断或显式指定。泛型是创建通用、可复用代码的关键工具。

## 二、具体用法

### 2.1 基本泛型函数

```typescript
// T 是类型参数，在调用时确定
function identity<T>(arg: T): T {
  return arg;
}

// 自动推断类型参数
const str = identity("hello");   // T = string
const num = identity(42);        // T = number

console.log(str); // 输出: hello
console.log(num); // 输出: 42

// 显式指定类型参数
const result = identity<boolean>(true);
console.log(result); // 输出: true
```

### 2.2 泛型数组函数

```typescript
function firstElement<T>(arr: T[]): T | undefined {
  return arr[0];
}

const first1 = firstElement([1, 2, 3]);       // T = number
const first2 = firstElement(["a", "b", "c"]); // T = string

console.log(first1); // 输出: 1
console.log(first2); // 输出: a
```

### 2.3 多类型参数

```typescript
function pair<T, U>(first: T, second: U): [T, U] {
  return [first, second];
}

const p1 = pair("hello", 42);      // [string, number]
const p2 = pair(true, "world");    // [boolean, string]

console.log(p1); // 输出: ["hello", 42]
console.log(p2); // 输出: [true, "world"]
```

### 2.4 泛型映射

```typescript
function mapArray<T, U>(arr: T[], fn: (item: T, index: number) => U): U[] {
  return arr.map(fn);
}

const doubled = mapArray([1, 2, 3], (n) => n * 2);       // number[]
const strings = mapArray([1, 2, 3], (n) => `#${n}`);     // string[]

console.log(doubled); // 输出: [2, 4, 6]
console.log(strings); // 输出: ["#1", "#2", "#3"]
```

## 三、注意事项与常见陷阱

1. **类型参数在运行时不存在**：`T` 编译后被擦除
2. **尽量让编译器推断**：不必显式指定类型参数，除非推断不正确
3. **有意义的命名**：单字母 `T`、`U`、`K` 是惯例，复杂场景用有意义的名称
4. **泛型约束在需要时添加**：不要过早添加约束
