# 函数重载

## 一、概念说明

函数重载（Function Overloading）允许一个函数有**多个类型签名**，根据不同的参数类型或数量返回不同的类型。TypeScript 中通过多个重载签名和一个实现签名来实现。

## 二、具体用法

### 2.1 基本函数重载

```typescript
// 重载签名（只有类型声明，没有实现）
function format(value: string): string;
function format(value: number): string;

// 实现签名（包含实际逻辑）
function format(value: string | number): string {
  if (typeof value === "string") {
    return value.toUpperCase();
  }
  return value.toFixed(2);
}

console.log(format("hello")); // 输出: HELLO
console.log(format(3.14159)); // 输出: 3.14
```

### 2.2 不同参数数量的重载

```typescript
function createRange(end: number): number[];
function createRange(start: number, end: number): number[];
function createRange(start: number, end: number, step: number): number[];

function createRange(startOrEnd: number, end?: number, step: number = 1): number[] {
  const start = end === undefined ? 0 : startOrEnd;
  const actualEnd = end === undefined ? startOrEnd : end;
  const result: number[] = [];
  for (let i = start; i < actualEnd; i += step) {
    result.push(i);
  }
  return result;
}

console.log(createRange(5));        // 输出: [0, 1, 2, 3, 4]
console.log(createRange(2, 7));     // 输出: [2, 3, 4, 5, 6]
console.log(createRange(0, 10, 3)); // 输出: [0, 3, 6, 9]
```

### 2.3 根据参数类型返回不同类型

```typescript
function parse(value: string): string[];
function parse(value: string, asNumber: true): number[];

function parse(value: string, asNumber?: boolean): (string | number)[] {
  const parts = value.split(",");
  if (asNumber) {
    return parts.map((p) => parseFloat(p.trim()));
  }
  return parts.map((p) => p.trim());
}

const strings = parse("1, 2, 3");         // string[]
const numbers = parse("1, 2, 3", true);   // number[]

console.log(strings); // 输出: ["1", " 2", " 3"]
console.log(numbers); // 输出: [1, 2, 3]
```

## 三、注意事项与常见陷阱

1. **实现签名对调用者不可见**：调用时只能看到重载签名
2. **重载签名必须在实现签名之前**
3. **更具体的签名放在前面**：TypeScript 按顺序匹配
4. **实现签名必须兼容所有重载**：参数类型应为联合类型
