# 异步函数类型

## 一、概念说明

异步函数（`async function`）的返回类型是 `Promise<T>`，其中 T 是实际返回值的类型。TypeScript 可以自动推断异步函数的返回类型，也可以显式标注为 `Promise<T>`。

## 二、具体用法

### 2.1 基本异步函数类型

```typescript
// 自动推断返回类型为 Promise<string>
async function greet(name: string): Promise<string> {
  return `Hello, ${name}!`;
}

// 显式标注返回类型
async function fetchUser(id: number): Promise<{ id: number; name: string }> {
  return { id, name: "Alice" };
}

(async () => {
  const message = await greet("World");
  console.log(message);
  // 输出: Hello, World!

  const user = await fetchUser(1);
  console.log(user.name);
  // 输出: Alice
})();
```

### 2.2 异步函数类型别名

```typescript
type AsyncFetcher<T> = () => Promise<T>;
type AsyncProcessor<T, R> = (input: T) => Promise<R>;

const fetchUsers: AsyncFetcher<{ id: number; name: string }[]> = async () => {
  return [{ id: 1, name: "Alice" }, { id: 2, name: "Bob" }];
};

const processUser: AsyncProcessor<number, string> = async (id) => {
  return `处理用户 ${id}`;
};

(async () => {
  const users = await fetchUsers();
  console.log(users.length);
  // 输出: 2

  const result = await processUser(1);
  console.log(result);
  // 输出: 处理用户 1
})();
```

### 2.3 Awaited 工具类型

```typescript
// Awaited<T> 展开 Promise 的类型
type User = Awaited<ReturnType<typeof fetchUser>>;
// { id: number; name: string }

// 嵌套 Promise
type Nested = Promise<Promise<string>>;
type Flat = Awaited<Nested>; // string

console.log("Awaited 解包 Promise 类型");
// 输出: Awaited 解包 Promise 类型
```

## 三、注意事项与常见陷阱

1. **`Promise<T>` 必须显式标注**：`async () => T` 会自动包装为 `Promise<T>`
2. **`Awaited<T>` 展开类型**：提取 Promise 内部的值类型
3. **异步函数总是返回 Promise**：即使没有显式 `return`
4. **错误类型**：`Promise` 的 rejection 类型默认为 `unknown`
