# 类型谓词 is

## 一、概念说明

类型谓词是返回 `parameterName is Type` 形式的函数，用于自定义类型守卫。当函数返回 `true` 时，TypeScript 在后续代码中将参数收窄为指定的类型。

## 二、具体用法

### 2.1 基本类型谓词

```typescript
interface Cat {
  kind: "cat";
  meow(): void;
}

interface Dog {
  kind: "dog";
  bark(): void;
}

// 类型谓词函数
function isCat(animal: Cat | Dog): animal is Cat {
  return animal.kind === "cat";
}

function interact(animal: Cat | Dog): void {
  if (isCat(animal)) {
    // TypeScript 知道 animal 是 Cat
    animal.meow();
  } else {
    animal.bark();
  }
}

const cat: Cat = { kind: "cat", meow: () => console.log("喵~") };
interact(cat);
// 输出: 喵~
```

### 2.2 检查数组元素

```typescript
function isString(value: unknown): value is string {
  return typeof value === "string";
}

function filterStrings(values: unknown[]): string[] {
  return values.filter(isString);
}

const mixed = [1, "hello", true, "world", null];
const strings = filterStrings(mixed);
console.log(strings);
// 输出: ["hello", "world"]
```

### 2.3 检查对象属性

```typescript
interface ErrorResponse {
  error: string;
  code: number;
}

interface SuccessResponse {
  data: unknown;
}

function isError(res: ErrorResponse | SuccessResponse): res is ErrorResponse {
  return "error" in res;
}

function handleResponse(res: ErrorResponse | SuccessResponse): void {
  if (isError(res)) {
    console.log(`错误 [${res.code}]: ${res.error}`);
  } else {
    console.log("成功:", res.data);
  }
}

handleResponse({ error: "未找到", code: 404 });
// 输出: 错误 [404]: 未找到
handleResponse({ data: { id: 1 } });
// 输出: 成功: { id: 1 }
```

## 三、注意事项与常见陷阱

1. **谓词实现必须准确**：返回 `true` 时必须确保值确实是目标类型
2. **`is` 不是类型断言**：它是运行时检查，更安全
3. **可用于数组过滤**：`filter(isString)` 是类型安全的
4. **参数必须是单一变量**：不能写 `(a, b) => a is T`
