# 函数参数类型注解

## 一、概念说明

函数参数可以有**必选参数**、**可选参数**（`?`）和**默认参数**（`= value`）。TypeScript 严格检查参数的数量和类型，确保函数调用时传入正确的参数。

## 二、具体用法

### 2.1 必选参数

```typescript
function greet(name: string, age: number): string {
  return `${name}，${age}岁`;
}

console.log(greet("Alice", 25));
// 输出: Alice，25岁
// greet("Alice");       // ❌ 编译错误: 缺少参数
// greet("Alice", "25"); // ❌ 编译错误: 类型不匹配
```

### 2.2 可选参数

```typescript
// 可选参数放在必选参数之后
function createUser(name: string, email?: string): string {
  return email ? `${name} <${email}>` : name;
}

console.log(createUser("Alice"));
// 输出: Alice
console.log(createUser("Bob", "bob@example.com"));
// 输出: Bob <bob@example.com>
```

### 2.3 默认参数

```typescript
function connect(host: string = "localhost", port: number = 3000, ssl: boolean = false): string {
  const protocol = ssl ? "https" : "http";
  return `${protocol}://${host}:${port}`;
}

console.log(connect());
// 输出: http://localhost:3000
console.log connect("api.example.com", 443, true);
// 输出: https://api.example.com:443
console.log(connect("api.example.com"));
// 输出: http://api.example.com:3000
```

### 2.4 参数解构的类型

```typescript
interface Options {
  width: number;
  height: number;
  color?: string;
}

// 解构参数的类型注解
function draw({ width, height, color = "black" }: Options): void {
  console.log(`绘制 ${width}x${height} 的矩形，颜色: ${color}`);
}

draw({ width: 100, height: 200 });
// 输出: 绘制 100x200 的矩形，颜色: black
draw({ width: 50, height: 50, color: "red" });
// 输出: 绘制 50x50 的矩形，颜色: red
```

## 三、注意事项与常见陷阱

1. **可选参数必须在必选参数之后**：`f(a?: number, b: string)` 不合法
2. **默认参数自动推断类型**：`= "hello"` 推断参数类型为 `string`
3. **可选参数类型为 `T | undefined`**：使用时需注意 undefined
4. **`strictNullChecks` 下**：必须检查可选参数是否为 undefined
