# 可调用接口

## 一、概念说明

可调用接口（Callable Interface）是包含**调用签名**的接口，使接口描述的对象可以像函数一样被调用。同时可以拥有属性和方法，创建"增强函数"。这在库设计中常用于创建既可调用又有属性的函数对象。

## 二、具体用法

### 2.1 基本调用签名

```typescript
interface Greeter {
  (name: string): string;
}

const hello: Greeter = (name) => `Hello, ${name}!`;
console.log(hello("World"));
// 输出: Hello, World!
```

### 2.2 带属性的可调用接口

```typescript
interface Logger {
  // 调用签名
  (message: string): void;
  // 属性
  level: "debug" | "info" | "warn" | "error";
  // 方法
  setLevel(level: "debug" | "info" | "warn" | "error"): void;
}

function createLogger(): Logger {
  const logger = ((message: string) => {
    console.log(`[${logger.level}] ${message}`);
  }) as Logger;

  logger.level = "info";
  logger.setLevel = (level) => { logger.level = level; };

  return logger;
}

const log = createLogger();
log("普通消息");
log.setLevel("debug");
log("调试消息");
// 输出:
// [info] 普通消息
// [debug] 调试消息
```

### 2.3 构造签名

```typescript
interface Constructable {
  new (name: string): { name: string };
}

class User {
  constructor(public name: string) {}
}

const factory: Constructable = User;
const user = new factory("Alice");
console.log(user.name);
// 输出: Alice
```

### 2.4 重载调用签名

```typescript
// 接口支持多个调用签名（重载）
interface Formatter {
  (value: string): string;
  (value: number): string;
}

const format: Formatter = (value: string | number) => {
  if (typeof value === "string") {
    return value.trim().toUpperCase();
  }
  return value.toFixed(2);
};

console.log(format("  hello ")); // 输出: HELLO
console.log(format(3.14159));    // 输出: 3.14
```

### 2.5 实际应用：jQuery 风格

```typescript
// 类似 jQuery 的 $ 函数
interface JQuery {
  // 调用签名：选择元素
  (selector: string): JQueryInstance;
  // 静态方法
  ajax(url: string, options?: object): Promise<unknown>;
  version: string;
}

interface JQueryInstance {
  addClass(name: string): JQueryInstance;
  text(value?: string): JQueryInstance | string;
  on(event: string, handler: () => void): JQueryInstance;
}

// 简化实现
const $ = ((selector: string) => {
  console.log(`选择元素: ${selector}`);
  return {
    addClass: (name: string) => $,
    text: (value?: string) => value ?? "文本内容",
    on: (event: string, handler: () => void) => $,
  };
}) as JQuery;

$.version = "3.6.0";
$.ajax = async (url) => ({ data: [] });

console.log($("button").addClass("primary").text("点击"));
```

## 三、注意事项与常见陷阱

1. **调用签名语法**：`(参数) => 返回值` 在接口内部用 `:` 而非 `=>`
2. **可调用接口可有多个重载签名**：与函数重载规则一致
3. **构造签名用 `new`**：区分普通调用和构造调用
4. **与函数类型别名功能相似**：选择哪种取决于是否需要属性和声明合并
5. **`this` 在调用签名中**：可以在调用签名中声明 `this` 类型
6. **实现可调用接口**：需要用 `as` 断言，因为普通函数没有自定义属性
