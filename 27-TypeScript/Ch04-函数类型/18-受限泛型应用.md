# 受限泛型应用

## 一、概念说明

受限泛型通过 `extends` 约束类型参数，使泛型函数在保持灵活性的同时能安全地访问特定属性。常用于类型安全的 `keys`、`values`、`entries` 等工具函数。

## 二、具体用法

### 2.1 类型安全的 Object.keys

```typescript
// 标准 Object.keys 返回 string[]（丢失类型信息）
interface User {
  id: number;
  name: string;
  email: string;
}

const user: User = { id: 1, name: "Alice", email: "a@b.com" };
const keys = Object.keys(user); // string[]（不够精确）

// 受限泛型：返回 (keyof T)[]
function typedKeys<T extends object>(obj: T): (keyof T)[] {
  return Object.keys(obj) as (keyof T)[];
}

const betterKeys = typedKeys(user); // ("id" | "name" | "email")[]
console.log(betterKeys);
// 输出: ["id", "name", "email"]
```

### 2.2 类型安全的 Object.values

```typescript
function typedValues<T extends object>(obj: T): T[keyof T][] {
  return Object.values(obj) as T[keyof T][];
}

const values = typedValues(user); // (number | string)[]
console.log(values);
// 输出: [1, "Alice", "a@b.com"]
```

### 2.3 类型安全的 Object.entries

```typescript
function typedEntries<T extends object>(obj: T): [keyof T, T[keyof T]][] {
  return Object.entries(obj) as [keyof T, T[keyof T]][];
}

const entries = typedEntries(user);
entries.forEach(([key, value]) => {
  console.log(`${String(key)}: ${value}`);
});
// 输出:
// id: 1
// name: Alice
// email: a@b.com
```

### 2.4 类型安全的 pick

```typescript
function pick<T extends object, K extends keyof T>(
  obj: T,
  ...keys: K[]
): Pick<T, K> {
  const result = {} as Pick<T, K>;
  for (const key of keys) {
    result[key] = obj[key];
  }
  return result;
}

const nameAndEmail = pick(user, "name", "email");
console.log(nameAndEmail);
// 输出: { name: "Alice", email: "a@b.com" }
// pick(user, "age"); // ❌ "age" 不是 User 的键
```

## 三、注意事项与常见陷阱

1. **`extends object`**：约束类型参数必须是对象类型
2. **`keyof T` 保持精确**：比 `string[]` 更类型安全
3. **强制类型断言**：`Object.keys` 底层返回 `string[]`，需要 `as` 断言
4. **符号键**：`Object.keys` 不返回 Symbol 键
