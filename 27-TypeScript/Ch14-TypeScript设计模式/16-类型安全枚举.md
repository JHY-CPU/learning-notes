# 类型安全枚举

## 一、概念说明

TypeScript 中有多种方式实现类型安全的枚举：原生 `enum`、联合类型、`as const` 对象。每种方式有不同的权衡。

## 二、具体用法

### 2.1 联合类型枚举（推荐）

```typescript
type Direction = 'up' | 'down' | 'left' | 'right';
type HttpStatus = 200 | 201 | 400 | 404 | 500;
type Status = 'pending' | 'active' | 'inactive';

// 函数中使用
function move(direction: Direction) {
  switch (direction) {
    case 'up': return { y: -1 };
    case 'down': return { y: 1 };
    case 'left': return { x: -1 };
    case 'right': return { x: 1 };
  }
}
```

### 2.2 const 对象枚举

```typescript
const Direction = {
  Up: 'UP',
  Down: 'DOWN',
  Left: 'LEFT',
  Right: 'RIGHT',
} as const;

// 提取值类型
type Direction = (typeof Direction)[keyof typeof Direction];
// 'UP' | 'DOWN' | 'LEFT' | 'RIGHT'

// 使用
function move(direction: Direction) {
  // ...
}

move(Direction.Up);
```

### 2.3 有标签联合

```typescript
type Shape =
  | { type: 'circle'; radius: number }
  | { type: 'rectangle'; width: number; height: number }
  | { type: 'triangle'; base: number; height: number };

function area(shape: Shape): number {
  switch (shape.type) {
    case 'circle':
      return Math.PI * shape.radius ** 2;
    case 'rectangle':
      return shape.width * shape.height;
    case 'triangle':
      return (shape.base * shape.height) / 2;
  }
}
```

### 2.4 枚举工具函数

```typescript
function enumValues<T extends Record<string, string>>(obj: T): T[keyof T][] {
  return Object.values(obj);
}

function isEnumValue<T extends Record<string, string>>(
  obj: T,
  value: unknown
): value is T[keyof T] {
  return Object.values(obj).includes(value);
}
```

## 三、注意事项与常见陷阱

1. **联合类型优于 enum**：更简洁、兼容性更好
2. **`as const` 确保字面量类型**：不被拓宽
3. **有标签联合是最强大的枚举模式**
4. **`const enum` 有兼容性问题**：`isolatedModules` 不完全支持
5. **字符串枚举比数字枚举更好**：调试时有意义
