# 映射类型

## 一、概念说明

映射类型（Mapped Type）基于一个类型创建新类型，遍历其所有键并对每个键的值类型进行变换。语法为 `[K in keyof T]: NewType`。几乎所有内置工具类型（`Partial`、`Readonly`、`Pick` 等）都基于映射类型实现。

## 二、具体用法

### 2.1 基本映射类型

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

// 将所有属性变为 string
type StringUser = {
  [K in keyof User]: string;
};

const su: StringUser = { id: "1", name: "Alice", email: "a@b.com" };
console.log(su);
// 输出: { id: "1", name: "Alice", email: "a@b.com" }
```

### 2.2 属性修饰符

```typescript
// 添加 readonly
type ReadonlyUser = {
  readonly [K in keyof User]: User[K];
};

// 添加可选
type PartialUser = {
  [K in keyof User]?: User[K];
};

// 移除可选（Required）
type RequiredUser = {
  [K in keyof User]-?: User[K];
};

// 移除 readonly
type MutableUser = {
  -readonly [K in keyof ReadonlyUser]: ReadonlyUser[K];
};
```

### 2.3 键重映射（as）

```typescript
// TypeScript 4.1+ 支持 as 重映射键
type Getters<T> = {
  [K in keyof T as `get${Capitalize<string & K>}`]: () => T[K];
};

type UserGetters = Getters<User>;
// { getId: () => number; getName: () => string; getEmail: () => string; }

const getters: UserGetters = {
  getId: () => 1,
  getName: () => "Alice",
  getEmail: () => "a@b.com",
};

console.log(getters.getName());
// 输出: Alice
```

### 2.4 过滤键

```typescript
// 使用 as never 过滤特定类型的键
type OnlyStrings<T> = {
  [K in keyof T as T[K] extends string ? K : never]: T[K];
};

type Mixed = { name: string; age: number; email: string };
type StringKeys = OnlyStrings<Mixed>;
// { name: string; email: string }

const sk: StringKeys = { name: "Alice", email: "a@b.com" };
console.log(sk);
// 输出: { name: "Alice", email: "a@b.com" }
```

## 三、注意事项与常见陷阱

1. **必须遍历 `keyof T`**：没有 `keyof` 就不是映射类型
2. **`as` 键重映射是 TS 4.1+ 特性**：旧版本不支持
3. **`+` 和 `-` 修饰符**：`+` 添加（默认），`-` 移除
4. **映射类型保留可选性**：原始属性的 `?` 会被保留，除非显式移除
