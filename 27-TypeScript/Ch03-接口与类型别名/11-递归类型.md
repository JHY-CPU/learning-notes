# 递归类型

## 一、概念说明

递归类型是引用自身的类型，用于描述**树形结构**、**链表**、**嵌套 JSON** 等深度嵌套的数据。在接口和类型别名中都可以使用自引用。

## 二、具体用法

### 2.1 树形结构

```typescript
interface TreeNode {
  value: string;
  children?: TreeNode[];
}

const tree: TreeNode = {
  value: "根节点",
  children: [
    {
      value: "子节点1",
      children: [
        { value: "叶节点1" },
        { value: "叶节点2" },
      ],
    },
    { value: "子节点2" },
  ],
};

function traverse(node: TreeNode, depth = 0): void {
  console.log("  ".repeat(depth) + node.value);
  node.children?.forEach((child) => traverse(child, depth + 1));
}

traverse(tree);
// 输出:
// 根节点
//   子节点1
//     叶节点1
//     叶节点2
//   子节点2
```

### 2.2 JSON 类型

```typescript
type Json =
  | string
  | number
  | boolean
  | null
  | Json[]
  | { [key: string]: Json };

const data: Json = {
  name: "Alice",
  age: 25,
  hobbies: ["reading", "coding"],
  address: {
    city: "Beijing",
    zip: "100000",
  },
};

console.log(data);
// 输出: { name: "Alice", age: 25, hobbies: [...], address: {...} }
```

### 2.3 链表

```typescript
interface LinkedList<T> {
  value: T;
  next: LinkedList<T> | null;
}

const list: LinkedList<number> = {
  value: 1,
  next: {
    value: 2,
    next: {
      value: 3,
      next: null,
    },
  },
};

function toArray<T>(list: LinkedList<T>): T[] {
  const result: T[] = [];
  let current: LinkedList<T> | null = list;
  while (current) {
    result.push(current.value);
    current = current.next;
  }
  return result;
}

console.log(toArray(list));
// 输出: [1, 2, 3]
```

## 三、注意事项与常见陷阱

1. **递归深度限制**：TypeScript 对递归类型有深度限制，过深会导致编译错误
2. **性能影响**：复杂递归类型可能拖慢类型检查
3. **别名使用**：递归类型必须通过 `interface` 或 `type` 别名引用自己
4. **`interface` 和 `type` 都支持递归**
