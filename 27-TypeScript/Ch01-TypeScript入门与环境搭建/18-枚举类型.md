# 枚举类型

## 一、概念说明

枚举（Enum）是一种特殊的类型，用于定义一组命名常量。TypeScript 支持**数字枚举**和**字符串枚举**。枚举在编译后会生成实际的 JavaScript 对象代码。`const enum` 则会在编译时内联，不生成额外代码。枚举为常量集合提供了类型安全和可读性。

## 二、具体用法

### 2.1 数字枚举

```typescript
// 默认从 0 开始自增
enum Direction {
  Up,      // 0
  Down,    // 1
  Left,    // 2
  Right,   // 3
}

console.log(Direction.Up);     // 输出: 0
console.log(Direction[0]);     // 输出: "Up"（反向映射）

// 指定起始值
enum Status {
  Pending = 1,
  Active,    // 2
  Inactive,  // 3
}

console.log(Status.Active);
// 输出: 2
```

### 2.2 字符串枚举

```typescript
enum Color {
  Red = "RED",
  Green = "GREEN",
  Blue = "BLUE",
}

function paint(color: Color): string {
  switch (color) {
    case Color.Red:   return "涂成红色";
    case Color.Green: return "涂成绿色";
    case Color.Blue:  return "涂成蓝色";
  }
}

console.log(paint(Color.Red));
// 输出: 涂成红色
console.log(Color.Green);
// 输出: GREEN
```

### 2.3 const enum（推荐）

```typescript
// const enum 编译时内联，不生成 JS 对象
const enum Direction {
  Up = "UP",
  Down = "DOWN",
}

// 编译后直接内联为:
// const dir = "UP";
const dir = Direction.Up;
console.log(dir);
// 输出: UP
```

### 2.4 用 as const 替代枚举

```typescript
// 使用 as const 对象替代枚举（不生成运行时代码）
const Direction = {
  Up: "UP",
  Down: "DOWN",
  Left: "LEFT",
  Right: "RIGHT",
} as const;

// 提取值的联合类型
type Direction = (typeof Direction)[keyof typeof Direction];
// 等价于 "UP" | "DOWN" | "LEFT" | "RIGHT"

function move(dir: Direction): void {
  console.log(`移动方向: ${dir}`);
}

move(Direction.Up); // ✅
// move("INVALID"); // ❌ 编译错误
```

### 2.5 枚举的编译输出

```typescript
// 源码
enum Color {
  Red,
  Green,
  Blue,
}

// 编译后的 JavaScript（生成了 IIFE）
"use strict";
var Color;
(function (Color) {
  Color[Color["Red"] = 0] = "Red";
  Color[Color["Green"] = 1] = "Green";
  Color[Color["Blue"] = 2] = "Blue";
})(Color || (Color = {}));
```

## 三、注意事项与常见陷阱

1. **数字枚举有反向映射**：`Direction[0]` 返回 `"Up"`，字符串枚举没有反向映射
2. **推荐 `const enum`**：不生成额外代码，运行时更轻量
3. **`as const` 替代**：简单场景可用 `as const` 对象替代枚举，更轻量且支持 Tree Shaking
4. **枚举是运行时存在的**：普通枚举会生成 JS 代码，增加包体积
5. **枚举合并**：同名枚举会自动合并（Declaration Merging），但 `const enum` 不支持
6. **数字枚举是开放的**：可以传入数字值而不受枚举成员约束，字符串枚举更安全
