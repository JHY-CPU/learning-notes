# 枚举类型

## 一、概念说明

枚举（Enum）是一种特殊的类型，用于定义一组命名常量。TypeScript 支持**数字枚举**和**字符串枚举**。枚举在编译后会生成实际的 JavaScript 对象代码。`const enum` 则会在编译时内联，不生成额外代码。

## 二、具体用法

### 2.1 数字枚举

```typescript
// 默认从 0 开始自增
enum Direction {
  Up,      // 0
  Down,    // 1
  Left,    // 2
  Right,   // 3
}

console.log(Direction.Up);     // 输出: 0
console.log(Direction[0]);     // 输出: "Up"（反向映射）

// 指定起始值
enum Status {
  Pending = 1,
  Active,    // 2
  Inactive,  // 3
}

console.log(Status.Active);
// 输出: 2
```

### 2.2 字符串枚举

```typescript
enum Color {
  Red = "RED",
  Green = "GREEN",
  Blue = "BLUE",
}

function paint(color: Color): string {
  switch (color) {
    case Color.Red:   return "涂成红色";
    case Color.Green: return "涂成绿色";
    case Color.Blue:  return "涂成蓝色";
  }
}

console.log(paint(Color.Red));
// 输出: 涂成红色
console.log(Color.Green);
// 输出: GREEN
```

### 2.3 const enum（推荐）

```typescript
// const enum 编译时内联，不生成 JS 对象
const enum Direction {
  Up = "UP",
  Down = "DOWN",
}

// 编译后直接内联为:
// const dir = "UP";
const dir = Direction.Up;
console.log(dir);
// 输出: UP
```

## 三、注意事项与常见陷阱

1. **数字枚举有反向映射**：`Direction[0]` 返回 `"Up"`，字符串枚举没有
2. **推荐 `const enum`**：不生成额外代码，运行时更轻量
3. **`as const` 替代**：简单场景可用 `as const` 对象替代枚举
4. **枚举是运行时存在的**：普通枚举会生成 JS 代码，增加包体积
