# 可辨识联合类型

## 一、概念说明

可辨识联合类型（Tagged Union / Algebraic Data Type）是判别联合的正式名称。它要求联合类型的每个成员都有一个共同的**标签字段**（tag），TypeScript 据此在分支中自动收窄类型。这是函数式编程中代数数据类型在 TypeScript 中的实现。

## 二、具体用法

### 2.1 完整的可辨识联合

```typescript
// 定义 API 响应类型
type ApiResponse<T> =
  | { tag: "idle" }
  | { tag: "loading"; startTime: number }
  | { tag: "success"; data: T; duration: number }
  | { tag: "error"; error: string; code: number };

// 使用模式匹配处理不同状态
function handleResponse<T>(response: ApiResponse<T>): string {
  switch (response.tag) {
    case "idle":
      return "空闲状态";
    case "loading":
      return `加载中，开始时间: ${response.startTime}`;
    case "success":
      return `成功，耗时: ${response.duration}ms`;
    case "error":
      return `错误 [${response.code}]: ${response.error}`;
    default:
      // 穷尽检查
      const _exhaustive: never = response;
      return _exhaustive;
  }
}

const res: ApiResponse<string[]> = {
  tag: "success",
  data: ["item1", "item2"],
  duration: 150,
};

console.log(handleResponse(res));
// 输出: 成功，耗时: 150ms
```

### 2.2 函数式 Either 模式

```typescript
type Either<E, A> =
  | { _tag: "Left"; left: E }
  | { _tag: "Right"; right: A };

function mapEither<E, A, B>(
  either: Either<E, A>,
  f: (a: A) => B
): Either<E, B> {
  if (either._tag === "Left") {
    return either; // 错误直接传递
  }
  return { _tag: "Right", right: f(either.right) };
}

const success: Either<Error, number> = { _tag: "Right", right: 42 };
const doubled = mapEither(success, (n) => n * 2);

console.log(doubled);
// 输出: { _tag: "Right", right: 84 }
```

## 三、注意事项与常见陷阱

1. **标签字段命名**：常用 `tag`、`kind`、`type`、`_tag` 等
2. **不要共享标签值**：每个变体的标签值必须唯一
3. **结合泛型**：可辨识联合配合泛型可创建强大的数据类型
4. **穷尽检查是必须的**：确保新增变体时不会遗漏处理逻辑
