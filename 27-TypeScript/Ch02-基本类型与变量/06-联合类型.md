# 联合类型

## 一、概念说明

联合类型（Union Type）用 `|` 表示一个值可以是多种类型之一。语法为 `A | B | C`，表示"是 A 或者是 B 或者是 C"。联合类型常配合类型守卫进行类型收窄。

## 二、具体用法

### 2.1 基本联合类型

```typescript
// 参数可以是 string 或 number
function formatId(id: string | number): string {
  if (typeof id === "string") {
    return id.toUpperCase(); // 收窄为 string
  }
  return `#${id}`; // 收窄为 number
}

console.log(formatId("abc"));   // 输出: ABC
console.log(formatId(123));     // 输出: #123
```

### 2.2 联合类型与类型收窄

```typescript
type Result = string | number | Error;

function display(result: Result): void {
  if (typeof result === "string") {
    console.log(`文本: ${result}`);
  } else if (typeof result === "number") {
    console.log(`数字: ${result.toFixed(2)}`);
  } else {
    console.log(`错误: ${result.message}`);
  }
}

display("hello");
// 输出: 文本: hello
display(3.14159);
// 输出: 数字: 3.14
display(new Error("失败"));
// 输出: 错误: 失败
```

### 2.3 可辨识联合

```typescript
type Shape =
  | { kind: "circle"; radius: number }
  | { kind: "rectangle"; width: number; height: number };

function area(shape: Shape): number {
  switch (shape.kind) {
    case "circle":
      return Math.PI * shape.radius ** 2;
    case "rectangle":
      return shape.width * shape.height;
  }
}

console.log(area({ kind: "circle", radius: 5 }));
// 输出: 78.53981633974483
console.log(area({ kind: "rectangle", width: 4, height: 6 }));
// 输出: 24
```

## 三、注意事项与常见陷阱

1. **只能访问共有属性**：未收窄时只能访问联合类型所有成员共有的属性
2. **类型守卫是关键**：必须用 `typeof`、`instanceof` 或自定义守卫收窄
3. **联合类型分配律**：`T & (A | B)` 等于 `(T & A) | (T & B)`
4. **可辨识联合最佳实践**：用共同的"标签"属性区分不同变体
