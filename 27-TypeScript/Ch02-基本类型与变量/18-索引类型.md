# 索引类型

## 一、概念说明

索引类型包括 `keyof`（获取对象的键名联合类型）和索引访问类型 `T[K]`（获取对象属性的类型）。它们是 TypeScript 类型编程的基础工具，用于编写通用的、类型安全的工具函数。

## 二、具体用法

### 2.1 keyof 操作符

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

// keyof User = "id" | "name" | "email"
type UserKeys = keyof User;

function getProperty(obj: User, key: UserKeys) {
  return obj[key];
}

const user: User = { id: 1, name: "Alice", email: "a@b.com" };
console.log(getProperty(user, "name"));
// 输出: Alice
// getProperty(user, "age"); // ❌ 编译错误
```

### 2.2 索引访问类型 T[K]

```typescript
interface User {
  id: number;
  name: string;
  metadata: {
    age: number;
    city: string;
  };
}

// 获取单个属性类型
type UserId = User["id"];          // number
type UserName = User["name"];      // string
type Metadata = User["metadata"];  // { age: number; city: string }

// 嵌套访问
type Age = User["metadata"]["age"]; // number

// 获取多个属性类型的联合
type IdOrName = User["id" | "name"]; // number | string

const id: UserId = 42;
console.log(id);
// 输出: 42
```

### 2.3 配合泛型使用

```typescript
function pluck<T, K extends keyof T>(objects: T[], key: K): T[K][] {
  return objects.map((obj) => obj[key]);
}

const users: User[] = [
  { id: 1, name: "Alice", email: "a@b.com" },
  { id: 2, name: "Bob", email: "b@c.com" },
];

const names = pluck(users, "name"); // string[]
const ids = pluck(users, "id");     // number[]

console.log(names);
// 输出: ["Alice", "Bob"]
console.log(ids);
// 输出: [1, 2]
```

## 三、注意事项与常见陷阱

1. **`keyof` 只返回公共属性**：不包含 `private` 和 `protected` 属性
2. **`T[K]` 用方括号**：不是点语法，因为 K 是类型不是值
3. **`extends keyof T`**：约束泛型参数只能是 T 的键
4. **`keyof any`**：结果是 `string | number | symbol`
