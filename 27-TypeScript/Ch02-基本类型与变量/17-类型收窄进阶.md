# 类型收窄进阶

## 一、概念说明

类型收窄（Narrowing）是 TypeScript 根据代码逻辑推断出更具体类型的能力。除了基本的 `typeof`/`instanceof` 守卫，还有**自定义类型谓词**、**断言函数**、**判别属性**等进阶收窄方式。

## 二、具体用法

### 2.1 自定义类型谓词

```typescript
interface Cat {
  kind: "cat";
  meow(): void;
}

interface Dog {
  kind: "dog";
  bark(): void;
}

// 类型谓词：参数名 is 类型
function isCat(animal: Cat | Dog): animal is Cat {
  return animal.kind === "cat";
}

function interact(animal: Cat | Dog): void {
  if (isCat(animal)) {
    // TypeScript 知道 animal 是 Cat
    animal.meow();
  } else {
    animal.bark();
  }
}

const cat: Cat = { kind: "cat", meow: () => console.log("喵~") };
interact(cat);
// 输出: 喵~
```

### 2.2 断言函数（TS 3.7+）

```typescript
// 断言函数：asserts value is Type
function assertIsString(val: unknown): asserts val is string {
  if (typeof val !== "string") {
    throw new Error(`Expected string, got ${typeof val}`);
  }
}

function processInput(input: unknown): string {
  assertIsString(input);
  // 此后 TypeScript 知道 input 是 string
  return input.toUpperCase();
}

console.log(processInput("hello"));
// 输出: HELLO
// processInput(123); // 运行时抛出错误
```

### 2.3 真值收窄

```typescript
function process(value: string | null | undefined): string {
  if (value) {
    // 收窄为 string（排除了 null、undefined 和 ""）
    return value.toUpperCase();
  }
  return "默认值";
}

console.log(process("hello"));  // 输出: HELLO
console.log(process(null));     // 输出: 默认值
console.log(process(""));       // 输出: 默认值
```

### 2.4 赋值收窄

```typescript
let x: string | number = "hello";

x = 42; // 重新赋值后，类型收窄为 number
console.log(x.toFixed(1));
// 输出: 42.0

x = "world"; // 再次赋值，类型收窄为 string
console.log(x.toUpperCase());
// 输出: WORLD
```

## 三、注意事项与常见陷阱

1. **类型谓词的准确性**：实现必须正确，否则类型系统会骗你
2. **断言函数没有返回值**：它通过抛出异常来"终止"不符合的情况
3. **真值收窄的陷阱**：`0`、`""`、`false` 也会被排除
4. **收窄只在作用域内有效**：回调函数中收窄可能失效
