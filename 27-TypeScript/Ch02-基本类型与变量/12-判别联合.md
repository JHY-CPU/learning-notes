# 判别联合

## 一、概念说明

判别联合（Discriminated Union）是一种设计模式，在联合类型的每个成员中有一个**共同的字面量属性**（称为判别属性），通过检查这个属性来确定当前是哪个变体。这是 TypeScript 中处理多态数据的最佳实践。

## 二、具体用法

### 2.1 基本判别联合

```typescript
type Circle = {
  kind: "circle";
  radius: number;
};

type Rectangle = {
  kind: "rectangle";
  width: number;
  height: number;
};

type Triangle = {
  kind: "triangle";
  base: number;
  height: number;
};

type Shape = Circle | Rectangle | Triangle;

function getArea(shape: Shape): number {
  switch (shape.kind) {
    case "circle":
      return Math.PI * shape.radius ** 2;
    case "rectangle":
      return shape.width * shape.height;
    case "triangle":
      return (shape.base * shape.height) / 2;
  }
}

console.log(getArea({ kind: "circle", radius: 5 }));
// 输出: 78.53981633974483
```

### 2.2 状态机模式

```typescript
type LoadingState = {
  status: "loading";
};

type SuccessState = {
  status: "success";
  data: string[];
};

type ErrorState = {
  status: "error";
  error: string;
};

type State = LoadingState | SuccessState | ErrorState;

function render(state: State): string {
  switch (state.status) {
    case "loading":
      return "加载中...";
    case "success":
      return `数据: ${state.data.join(", ")}`;
    case "error":
      return `错误: ${state.error}`;
  }
}

console.log(render({ status: "loading" }));
// 输出: 加载中...
console.log(render({ status: "success", data: ["a", "b"] }));
// 输出: 数据: a, b
```

## 三、注意事项与常见陷阱

1. **判别属性必须是字面量类型**：`string` 无法区分不同变体
2. **穷尽检查**：在 `default` 分支用 `never` 确保处理所有情况
3. **一个判别属性就够**：不需要为每个变体定义不同的判别属性
4. **嵌套判别联合**：可以嵌套使用，创建复杂的状态类型
