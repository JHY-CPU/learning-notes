# 泛型约束详解

## 一、概念说明

泛型约束使用 `extends` 关键字限制类型参数的范围，使其必须满足特定条件。这允许在泛型函数内部安全地访问约束类型的属性。

## 二、具体用法

### 2.1 基本约束

```typescript
interface HasLength {
  length: number;
}

function logLength<T extends HasLength>(arg: T): T {
  console.log(`长度: ${arg.length}`);
  return arg;
}

logLength("hello");     // string 有 length
logLength([1, 2, 3]);   // 数组有 length
logLength({ length: 5, name: "x" }); // 对象有 length
// logLength(42);       // ❌ number 没有 length
// 输出: 长度: 5
```

### 2.2 keyof 约束

```typescript
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
  return obj[key];
}

const user = { name: "Alice", age: 25 };
console.log(getProperty(user, "name")); // 输出: Alice
// getProperty(user, "email"); // ❌
```

### 2.3 多重约束

```typescript
interface Printable { print(): void; }
interface Serializable { serialize(): string; }

function process<T extends Printable & Serializable>(item: T): string {
  item.print();
  return item.serialize();
}
```

## 三、注意事项与常见陷阱

1. **约束限制灵活性**：过于严格的约束限制泛型的通用性
2. **`extends` 在泛型中不是继承**：是类型约束
3. **`keyof` 约束最常用**：确保 key 是对象的有效键
4. **默认约束**：`<T extends any = string>` 提供默认类型
