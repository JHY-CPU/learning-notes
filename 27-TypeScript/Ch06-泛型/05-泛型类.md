# 泛型类

## 一、概念说明

泛型类在类定义时引入类型参数，使类的属性、方法可以使用泛型类型。实例化时指定具体类型，整个类实例的类型随之确定。泛型类常见于集合类、容器类、数据结构等。

## 二、具体用法

### 2.1 基本泛型类

```typescript
class Stack<T> {
  private items: T[] = [];

  push(item: T): void {
    this.items.push(item);
  }

  pop(): T | undefined {
    return this.items.pop();
  }

  peek(): T | undefined {
    return this.items[this.items.length - 1];
  }
}

const numStack = new Stack<number>();
numStack.push(1);
numStack.push(2);
console.log(numStack.pop()); // 输出: 2

const strStack = new Stack<string>();
strStack.push("hello");
console.log(strStack.peek()); // 输出: hello
```

### 2.2 泛型类约束

```typescript
class SortableList<T extends { toString(): string }> {
  private items: T[] = [];

  add(item: T): void {
    this.items.push(item);
  }

  sorted(): T[] {
    return [...this.items].sort((a, b) =>
      a.toString().localeCompare(b.toString())
    );
  }
}

const list = new SortableList<string>();
list.add("banana");
list.add("apple");
list.add("cherry");
console.log(list.sorted());
// 输出: ["apple", "banana", "cherry"]
```

### 2.3 泛型类继承

```typescript
class Queue<T> {
  protected items: T[] = [];

  enqueue(item: T): void {
    this.items.push(item);
  }

  dequeue(): T | undefined {
    return this.items.shift();
  }
}

// 子类指定具体类型
class NumberQueue extends Queue<number> {
  sum(): number {
    return this.items.reduce((a, b) => a + b, 0);
  }
}

// 子类继续泛型
class BoundedQueue<T> extends Queue<T> {
  constructor(private maxSize: number) { super(); }

  override enqueue(item: T): void {
    if (this.items.length < this.maxSize) {
      super.enqueue(item);
    }
  }
}
```

### 2.4 静态成员的限制

```typescript
class Container<T> {
  // ❌ 静态成员不能使用类型参数 T
  // static defaultValue: T;

  // ✅ 使用 unknown 或具体类型
  static count = 0;

  constructor(public value: T) {
    Container.count++;
  }
}
```

## 三、注意事项与常见陷阱

1. **静态成员不能用泛型**：`static item: T` 不合法，因为 T 是实例级别的
2. **实例化时指定类型**：`new Stack<number>()`，也可从构造函数参数推断
3. **类型参数在整个类中共享**：属性和方法共用同一个 `T`
4. **泛型类的继承**：子类可以指定具体类型或继续泛型
5. **`instanceof` 与泛型**：`new Stack<number>() instanceof Stack` 为 true（泛型擦除）
6. **类型推断**：`new Stack()` 从初始数据推断类型
