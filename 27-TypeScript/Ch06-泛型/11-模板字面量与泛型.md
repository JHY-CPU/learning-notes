# 模板字面量与泛型

## 一、概念说明

模板字面量类型结合泛型，可以在类型级别进行字符串操作。通过模板字符串语法 `` `${T}` `` 创建组合类型，配合 `infer` 实现类型级字符串解析。TypeScript 4.1+ 引入此特性。

## 二、具体用法

### 2.1 基本模板泛型

```typescript
type Greeting<T extends string> = `Hello, ${T}!`;

type G1 = Greeting<"World">; // "Hello, World!"
type G2 = Greeting<"TS">;    // "Hello, TS!"

console.log("模板字面量泛型");
// 输出: 模板字面量泛型
```

### 2.2 联合展开

```typescript
type Color = "red" | "green" | "blue";
type Size = "sm" | "md" | "lg";

type ClassName = `${Color}-${Size}`;
// "red-sm" | "red-md" | "red-lg" | "green-sm" | ...共 9 种
```

### 2.3 类型安全的事件系统

```typescript
type EventName = "click" | "focus" | "blur";
type HandlerName = `on${Capitalize<EventName>}`;
// "onClick" | "onFocus" | "onBlur"

interface Handlers {
  onClick: () => void;
  onFocus: () => void;
  onBlur: () => void;
}

// 自动生成事件处理器类型
type EventHandlers = {
  [K in EventName as `on${Capitalize<K>}`]: () => void;
};
```

### 2.4 路由参数提取

```typescript
type ExtractParams<T extends string> =
  T extends `${string}:${infer Param}/${infer Rest}`
    ? Param | ExtractParams<Rest>
    : T extends `${string}:${infer Param}`
    ? Param
    : never;

type Params = ExtractParams<"/users/:userId/posts/:postId">;
// "userId" | "postId"
```

### 2.5 内置字符串工具类型

```typescript
// Uppercase / Lowercase
type Upper = Uppercase<"hello">; // "HELLO"
type Lower = Lowercase<"HELLO">; // "hello"

// Capitalize / Uncapitalize
type Cap = Capitalize<"hello">;   // "Hello"
type Uncap = Uncapitalize<"Hello">; // "hello"

// 实际应用
type Getter<T extends string> = `get${Capitalize<T>}`;
type Setter<T extends string> = `set${Capitalize<T>}`;

type Props = "name" | "age";
type Methods = Getter<Props>; // "getName" | "getAge"
```

## 三、注意事项与常见陷阱

1. **联合类型会"笛卡尔展开"**：`A | B` 和 `C | D` 组合产生 4 种，数量可能爆炸
2. **类型参数必须 `extends string`**：否则不能用于模板
3. **内置工具类型**：`Uppercase`、`Lowercase`、`Capitalize`、`Uncapitalize`
4. **与 `infer` 配合**：可解析模板字符串类型，提取结构化信息
5. **模板字面量是字面量类型**：推断结果是精确的字符串字面量
6. **性能注意**：大量联合类型的模板字面量组合可能影响类型检查性能
