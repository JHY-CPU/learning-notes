# 模板字面量与泛型

## 一、概念说明

模板字面量类型结合泛型，可以在类型级别进行字符串操作。通过模板字符串语法 `` `${T}` `` 创建组合类型，配合 `infer` 实现类型级字符串解析。

## 二、具体用法

### 2.1 基本模板泛型

```typescript
type Greeting<T extends string> = `Hello, ${T}!`;

type G1 = Greeting<"World">; // "Hello, World!"
type G2 = Greeting<"TS">;    // "Hello, TS!"

console.log("模板字面量泛型");
// 输出: 模板字面量泛型
```

### 2.2 联合展开

```typescript
type Color = "red" | "green" | "blue";
type Size = "sm" | "md" | "lg";

type ClassName = `${Color}-${Size}`;
// "red-sm" | "red-md" | "red-lg" | "green-sm" | ...共 9 种
```

### 2.3 类型安全的事件系统

```typescript
type EventName = "click" | "focus" | "blur";
type HandlerName = `on${Capitalize<EventName>}`;
// "onClick" | "onFocus" | "onBlur"

interface Handlers {
  onClick: () => void;
  onFocus: () => void;
  onBlur: () => void;
}
```

## 三、注意事项与常见陷阱

1. **联合类型会"笛卡尔展开"**：组合数量可能爆炸
2. **类型参数必须 `extends string`**：否则不能用于模板
3. **内置工具类型**：`Uppercase`、`Lowercase`、`Capitalize`、`Uncapitalize`
4. **与 `infer` 配合**：可解析模板字符串类型
