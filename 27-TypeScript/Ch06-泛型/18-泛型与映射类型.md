# 泛型与映射类型

## 一、概念说明

泛型结合映射类型（`[K in keyof T]`）可以批量处理对象类型的属性，实现属性变换、过滤、增强等操作。这是 TypeScript 类型编程的核心模式，内置工具类型都基于此实现。

## 二、具体用法

### 2.1 批量属性转换

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

// 所有属性变为 string
type Stringify<T> = { [K in keyof T]: string };

type StringUser = Stringify<User>;
// { id: string; name: string; email: string; }
```

### 2.2 Getter 生成

```typescript
type Getters<T> = {
  [K in keyof T as `get${Capitalize<string & K>}`]: () => T[K];
};

type UserGetters = Getters<User>;
// { getId: () => number; getName: () => string; getEmail: () => string; }
```

### 2.3 可空属性

```typescript
type Nullable<T> = { [K in keyof T]: T[K] | null };

type NullableUser = Nullable<User>;
// { id: number | null; name: string | null; email: string | null; }
```

### 2.4 过滤属性类型

```typescript
type PickByType<T, U> = {
  [K in keyof T as T[K] extends U ? K : never]: T[K];
};

type StringProps = PickByType<User, string>;
// { name: string; email: string; }

type NumberProps = PickByType<User, number>;
// { id: number; }
```

### 2.5 实际应用

```typescript
// API 响应映射
type ApiResponse<T> = {
  [K in keyof T]: {
    data: T[K];
    loading: boolean;
    error: string | null;
  };
};

type ApiUser = ApiResponse<User>;
// {
//   id: { data: number; loading: boolean; error: string | null };
//   name: { data: string; loading: boolean; error: string | null };
//   email: { data: string; loading: boolean; error: string | null };
// }

// 表单字段类型
type FormFields<T> = {
  [K in keyof T]: {
    value: T[K];
    touched: boolean;
    dirty: boolean;
    error?: string;
  };
};
```

## 三、注意事项与常见陷阱

1. **`in keyof T` 是核心**：遍历对象的所有键
2. **`as` 可重映射键名**：TS 4.1+ 支持在映射中重命名键
3. **`as never` 过滤键**：条件为 `never` 的键被排除
4. **组合使用**：泛型 + 映射 + 条件类型 = 强大的类型操作能力
5. **性能注意**：大量属性的复杂映射类型可能影响类型检查速度
6. **映射类型保留可选和 readonly 修饰符**：`+?`、`-?`、`+readonly`、`-readonly`
