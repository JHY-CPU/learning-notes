# 变体泛型

## 一、概念说明

变体（Variance）描述泛型类型参数与类型之间的子类型关系。**协变**（Covariant）：`Dog[]` 是 `Animal[]` 的子类型。**逆变**（Contravariant）：反向关系。TypeScript 的泛型默认是协变的（除函数参数外）。

## 二、具体用法

### 2.1 协变

```typescript
class Animal { name = "animal"; }
class Dog extends Animal { breed = "金毛"; }

// 数组是协变的
const dogs: Dog[] = [new Dog()];
const animals: Animal[] = dogs; // ✅ 协变

// Promise 也是协变的
const dogPromise: Promise<Dog> = Promise.resolve(new Dog());
const animalPromise: Promise<Animal> = dogPromise; // ✅
```

### 2.2 函数参数的逆变

```typescript
type Fn<T> = (arg: T) => void;

type DogFn = Fn<Dog>;
type AnimalFn = Fn<Animal>;

const handleAnimal: AnimalFn = (animal) => console.log(animal.name);
const handleDog: DogFn = handleAnimal; // ✅ 逆变

handleDog(new Dog());
// 输出: animal
```

### 2.3 不变

```typescript
// 可读可写的引用通常是不变的
interface Ref<T> {
  get(): T;
  set(value: T): void;
}

// Ref<Dog> 不兼容 Ref<Animal>（不变）
```

## 三、注意事项与常见陷阱

1. **`strictFunctionTypes`**：开启后函数参数严格逆变
2. **数组协变有风险**：运行时可能插入错误类型
3. **`readonly` 数组是协变的**：可变数组更复杂
4. **理解变体有助于调试**：类型不兼容的原因通常是变体规则
