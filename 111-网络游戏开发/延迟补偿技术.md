# 延迟补偿技术

## 核心概念

延迟补偿（Lag Compensation）是对抗网络延迟影响的技术集合，核心目标是让玩家感觉操作"即时响应"。主要技术包括服务端回溯（Server Rewind）、命中判定补偿和插值缓冲。这些技术让高延迟玩家和低延迟玩家都能获得公平的游戏体验。

## 数学基础

### 延迟分解

网络延迟的总组成：

$$L_{total} = L_{propagation} + L_{transmission} + L_{queuing} + L_{processing}$$

- $L_{propagation}$：物理传播延迟（光速限制，约 1ms/200km）
- $L_{transmission}$：数据包传输延迟（取决于带宽和包大小）
- $L_{queuing}$：路由器排队延迟（受网络拥塞影响）
- $L_{processing}$：处理延迟（服务端/客户端计算时间）

### Peeker's Advantage

当玩家 A 探头射击玩家 B 时：

$$Advantage_A = RTT_B + T_{server\_tick}$$

这意味着探头方有约 50-100ms 的优势窗口。这是网络延迟的物理限制，无法完全消除。

### 服务端回溯时间

服务端回溯到客户端看到目标的时刻：

$$t_{rewind} = t_{server} - RTT_{client} - T_{interpolation}$$

其中 $T_{interpolation}$ 是客户端的插值缓冲延迟（通常 100ms）。

## 完整 C++ 延迟补偿系统

```cpp
#include <vector>
#include <deque>
#include <unordered_map>
#include <cstdint>
#include <glm/glm.hpp>
#include <glm/gtc/quaternion.hpp>

using Vec3 = glm::vec3;
using Quat = glm::quat;

// ===== 碰撞体定义 =====
struct Hitbox {
    Vec3 center;
    Vec3 halfExtents;  // Box 半尺寸
    Quat rotation;
    float radius;       // Capsule 半径
    int boneIndex;      // 关联的骨骼索引

    enum Type { SPHERE, CAPSULE, AABB };
    Type type = CAPSULE;
};

// ===== 历史状态 =====
struct HistoryEntry {
    uint32_t timestamp;          // 服务端时间戳
    Vec3 position;
    Quat rotation;
    std::vector<Hitbox> hitboxes;  // 该时刻所有碰撞体的位置
};

// ===== 射击请求 =====
struct ShotRequest {
    uint32_t shooterId;
    uint32_t clientTimestamp;    // 客户端看到射击的时间
    Vec3 origin;                 // 射击起点
    Vec3 direction;              // 射击方向
    uint32_t weaponId;
    uint16_t shotId;             // 射击序号（防重放）
};

// ===== 命中结果 =====
struct HitResult {
    bool hit = false;
    uint32_t targetId = 0;
    Vec3 hitPoint;
    int hitBone = -1;
    float damage = 0;
    float distance = 0;
};

// ===== 延迟补偿系统 =====
class LagCompensation {
public:
    static constexpr int HISTORY_DURATION_MS = 500;     // 历史记录保存 500ms
    static constexpr int MAX_REWIND_MS = 250;           // 最大回溯 250ms
    static constexpr int HISTORY_FPS = 60;              // 历史记录保存频率
    static constexpr float TICK_INTERVAL = 1.0f / HISTORY_FPS;

    // 每个玩家的历史状态
    std::unordered_map<uint32_t, std::deque<HistoryEntry>> playerHistory;

    // ===== 服务端：记录玩家状态 =====
    void RecordPlayerState(uint32_t playerId, const Vec3& position,
                           const Quat& rotation,
                           const std::vector<Hitbox>& hitboxes) {
        HistoryEntry entry;
        entry.timestamp = GetCurrentServerTime();
        entry.position = position;
        entry.rotation = rotation;
        entry.hitboxes = hitboxes;

        auto& history = playerHistory[playerId];
        history.push_back(entry);

        // 清理过旧的历史
        uint32_t cutoff = entry.timestamp - HISTORY_DURATION_MS;
        while (!history.empty() && history.front().timestamp < cutoff) {
            history.pop_front();
        }
    }

    // ===== 服务端：处理射击请求 =====
    HitResult ProcessShot(const ShotRequest& request) {
        HitResult result;

        // 1. 计算回溯时间
        uint32_t serverTime = GetCurrentServerTime();
        uint32_t rtt = serverTime - request.clientTimestamp;

        // 限制回溯上限
        if (rtt > MAX_REWIND_MS * 2) {
            // RTT 异常大，拒绝补偿
            rtt = MAX_REWIND_MS * 2;
        }

        // 回溯时间 = 客户端看到世界的时间
        // 客户端在 t_client 时看到目标，然后发送射击请求
        // 服务端在 t_server 收到请求，需要回溯到 t_client 的状态
        uint32_t rewindTime = request.clientTimestamp;

        // 2. 对每个玩家执行回溯命中检测
        float closestDist = FLT_MAX;
        HitResult closestHit;

        for (auto& [playerId, history] : playerHistory) {
            if (playerId == request.shooterId) continue;  // 不能击中自己

            // 找到最接近回溯时间的历史状态
            const HistoryEntry* entry = FindClosestHistory(history, rewindTime);
            if (!entry) continue;

            // 3. 使用历史状态进行射线-碰撞体检测
            HitResult playerHit = RayVsHitboxes(
                request.origin, request.direction, entry->hitboxes);

            if (playerHit.hit && playerHit.distance < closestDist) {
                closestDist = playerHit.distance;
                closestHit = playerHit;
                closestHit.targetId = playerId;
            }
        }

        return closestHit;
    }

    // ===== 批量回溯处理 =====
    std::vector<HitResult> ProcessShotgun(const ShotRequest& request,
                                           int pelletCount = 8) {
        std::vector<HitResult> results;

        for (int i = 0; i < pelletCount; i++) {
            // 每颗弹丸有随机散布
            Vec3 spreadDir = ApplySpread(request.direction, 2.0f);  // 2度散布

            ShotRequest spreadShot = request;
            spreadShot.direction = spreadDir;

            HitResult hit = ProcessShot(spreadShot);
            if (hit.hit) {
                hit.damage = GetWeaponDamage(request.weaponId) / pelletCount;
                results.push_back(hit);
            }
        }

        return results;
    }

private:
    // 查找最接近指定时间的历史状态
    const HistoryEntry* FindClosestHistory(
        const std::deque<HistoryEntry>& history,
        uint32_t targetTime) const
    {
        if (history.empty()) return nullptr;

        // 二分查找
        int left = 0, right = (int)history.size() - 1;
        int best = 0;
        uint32_t bestDiff = UINT32_MAX;

        while (left <= right) {
            int mid = (left + right) / 2;
            uint32_t diff = std::abs((int64_t)history[mid].timestamp - (int64_t)targetTime);
            if (diff < bestDiff) {
                bestDiff = diff;
                best = mid;
            }

            if (history[mid].timestamp < targetTime) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return &history[best];
    }

    // 射线与碰撞体检测
    HitResult RayVsHitboxes(const Vec3& origin, const Vec3& direction,
                             const std::vector<Hitbox>& hitboxes) {
        HitResult result;
        float closestDist = FLT_MAX;

        for (int i = 0; i < (int)hitboxes.size(); i++) {
            const auto& hb = hitboxes[i];

            float dist;
            if (RayVsCapsule(origin, direction, hb, dist)) {
                if (dist < closestDist) {
                    closestDist = dist;
                    result.hit = true;
                    result.hitBone = hb.boneIndex;
                    result.hitPoint = origin + direction * dist;
                    result.distance = dist;
                }
            }
        }

        return result;
    }

    // 射线与胶囊碰撞体检测
    bool RayVsCapsule(const Vec3& origin, const Vec3& direction,
                      const Hitbox& capsule, float& outDist) const {
        // 简化版：球体检测
        Vec3 oc = origin - capsule.center;
        float b = glm::dot(oc, direction);
        float c = glm::dot(oc, oc) - capsule.radius * capsule.radius;
        float discriminant = b * b - c;

        if (discriminant < 0) return false;

        float t = -b - std::sqrt(discriminant);
        if (t < 0) t = -b + std::sqrt(discriminant);
        if (t < 0) return false;

        outDist = t;
        return true;
    }

    Vec3 ApplySpread(const Vec3& direction, float spreadAngle) {
        // 随机散布（需要确定性随机数）
        float angle = (float)(rand() % 360) * 3.14159f / 180.0f;
        float spread = (float)(rand() % 100) / 100.0f * spreadAngle * 3.14159f / 180.0f;

        Vec3 right = glm::cross(direction, Vec3(0, 1, 0));
        Vec3 up = glm::cross(right, direction);

        Vec3 offset = right * std::cos(angle) * std::sin(spread)
                    + up * std::sin(angle) * std::sin(spread);

        return glm::normalize(direction + offset);
    }

    uint32_t GetCurrentServerTime() const { return 0; }  // 实际使用系统时钟
    float GetWeaponDamage(uint32_t weaponId) const { return 25.0f; }
};
```

## 插值缓冲系统

```cpp
// ===== 客户端插值缓冲 =====
class ClientInterpolationBuffer {
public:
    struct Snapshot {
        uint32_t timestamp;
        Vec3 position;
        Quat rotation;
        Vec3 velocity;
        uint8_t animState;
    };

    std::deque<Snapshot> buffer;
    float interpolationDelay = 0.1f;  // 100ms 延迟

    // 添加服务端快照
    void AddSnapshot(const Snapshot& snap) {
        // 按时间顺序插入
        auto it = buffer.begin();
        while (it != buffer.end() && it->timestamp < snap.timestamp) {
            ++it;
        }
        buffer.insert(it, snap);

        // 清理过旧数据
        uint32_t cutoff = snap.timestamp - (uint32_t)(interpolationDelay * 2000);
        while (!buffer.empty() && buffer.front().timestamp < cutoff) {
            buffer.pop_front();
        }
    }

    // 获取渲染时的插值状态
    bool GetRenderState(uint32_t currentTime, Vec3& outPos, Quat& outRot) const {
        if (buffer.size() < 2) {
            if (!buffer.empty()) {
                outPos = buffer.back().position;
                outRot = buffer.back().rotation;
                return true;
            }
            return false;
        }

        // 目标时间 = 当前时间 - 插值延迟
        uint32_t renderTime = currentTime - (uint32_t)(interpolationDelay * 1000);

        // 找到包围 renderTime 的两个快照
        for (size_t i = 0; i < buffer.size() - 1; i++) {
            if (buffer[i].timestamp <= renderTime && buffer[i + 1].timestamp >= renderTime) {
                float duration = (float)(buffer[i + 1].timestamp - buffer[i].timestamp);
                float t = (duration > 0) ?
                    (float)(renderTime - buffer[i].timestamp) / duration : 0.0f;
                t = std::clamp(t, 0.0f, 1.0f);

                outPos = glm::mix(buffer[i].position, buffer[i + 1].position, t);
                outRot = glm::slerp(buffer[i].rotation, buffer[i + 1].rotation, t);
                return true;
            }
        }

        // 超出缓冲范围
        if (renderTime < buffer.front().timestamp) {
            outPos = buffer.front().position;
            outRot = buffer.front().rotation;
        } else {
            outPos = buffer.back().position;
            outRot = buffer.back().rotation;
        }
        return true;
    }

    // 外推（当缓冲不足时）
    void Extrapolate(uint32_t currentTime, Vec3& outPos, Quat& outRot,
                     float maxExtrapolation = 0.2f) const {
        if (buffer.empty()) return;

        const auto& latest = buffer.back();
        float dt = (currentTime - latest.timestamp) / 1000.0f;
        dt = std::min(dt, maxExtrapolation);

        outPos = latest.position + latest.velocity * dt;
        outRot = latest.rotation;  // 旋转不外推
    }
};
```

## 常见陷阱与解决方案

1. **回溯时间超过历史缓冲**：高延迟玩家的射击回溯超出 500ms 缓冲。解决方案：设置最大回溯限制（通常 250ms），超出时使用最旧的历史数据。

2. **Peeker's Advantage 不公平**：探头方有额外的反应窗口。解决方案：无法物理消除，但可以通过游戏设计（如延迟射击惩罚）缓解。

3. **插值缓冲延迟导致操作不跟手**：100ms 缓冲 + 网络延迟 = 200ms+ 感知延迟。解决方案：本地玩家使用客户端预测，远端实体使用插值。

4. **回溯期间物理状态变化**：回溯时其他实体的位置/碰撞体已变化。解决方案：回溯期间使用完整的历史世界状态，而非只回溯目标实体。

## 实际游戏案例

- **CS2**：经典 Lag Compensation 实现，peeker's advantage 约 40ms（64 tick）
- **守望先锋**：客户端命中判定 + 服务端验证的混合方案
- **堡垒之夜**：射击回溯 + 建筑系统的即时响应
- **彩虹六号**：精确的命中框回溯系统，支持墙体穿透检测
- **Valorant**：服务端回溯 + 128 tick 服务器，极低延迟补偿
