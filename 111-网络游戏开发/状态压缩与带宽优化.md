# 状态压缩与带宽优化

## 核心概念

状态压缩与带宽优化旨在最小化网络传输数据量，核心手段包括量化（降低精度）、Delta 压缩（只传输变化）、位压缩（紧凑打包）和脏标记（按需同步）。在保持游戏体验的前提下，可将带宽降低 80% 以上。

## 数学基础

### 量化公式

将浮点数映射到 $n$ 位整数：

$$q = \text{round}\left(\frac{x - x_{min}}{x_{max} - x_{min}} \times (2^n - 1)\right)$$

反量化：

$$\hat{x} = \frac{q}{2^n - 1} \times (x_{max} - x_{min}) + x_{min}$$

量化误差：

$$e_{max} = \frac{x_{max} - x_{min}}{2 \times (2^n - 1)}$$

对于位置（范围 -1000~1000，16 位）：$e_{max} = 2000 / 131070 \approx 0.015m$，精度足够。

### 四元数量化

四元数 $q = (w, x, y, z)$ 满足 $w^2 + x^2 + y^2 + z^2 = 1$，因此只需编码 3 个分量，第 4 个可推导：

选出绝对值最大的分量 $q_m$，只编码其余 3 个分量，每个 10 bit（共 32 bit）：

$$\hat{q}_m = \text{sign}(q_m) \times \sqrt{1 - \hat{q}_a^2 - \hat{q}_b^2 - \hat{q}_c^2}$$

### Delta 编码增益

假设属性值每帧变化的概率为 $p$，则 Delta 编码的平均位宽：

$$\text{bits}_{avg} = p \times \text{bits}_{full} + (1-p) \times 1$$

当 $p = 0.1$（10% 变化率），完整编码 32 bit：$\text{bits}_{avg} = 0.1 \times 32 + 0.9 \times 1 = 4.1$ bit，压缩比 87%。

## 完整 C++ 状态压缩系统

```cpp
#include <vector>
#include <cstdint>
#include <cstring>
#include <cmath>
#include <glm/glm.hpp>
#include <glm/gtc/quaternion.hpp>

using Vec3 = glm::vec3;
using Quat = glm::quat;

// ===== 位写入器 =====
class BitWriter {
public:
    std::vector<uint8_t> buffer;
    int bytePos = 0;
    int bitPos = 0;

    BitWriter(int capacity = 1024) {
        buffer.resize(capacity, 0);
    }

    // 写入指定位数的值
    void WriteBits(uint32_t value, int numBits) {
        for (int i = 0; i < numBits; i++) {
            if (value & (1u << i)) {
                buffer[bytePos] |= (1u << bitPos);
            }
            bitPos++;
            if (bitPos >= 8) {
                bitPos = 0;
                bytePos++;
                if (bytePos >= (int)buffer.size()) {
                    buffer.resize(buffer.size() * 2, 0);
                }
            }
        }
    }

    // 写入布尔值
    void WriteBool(bool value) {
        WriteBits(value ? 1 : 0, 1);
    }

    // 量化写入浮点数
    void WriteFloat(float value, float min, float max, int bits) {
        float normalized = (value - min) / (max - min);
        normalized = std::clamp(normalized, 0.0f, 1.0f);
        uint32_t quantized = (uint32_t)(normalized * ((1u << bits) - 1));
        WriteBits(quantized, bits);
    }

    // 量化写入向量
    void WriteVec3(const Vec3& v, float min, float max, int bitsPerComponent) {
        WriteFloat(v.x, min, max, bitsPerComponent);
        WriteFloat(v.y, min, max, bitsPerComponent);
        WriteFloat(v.z, min, max, bitsPerComponent);
    }

    // 量化写入四元数（Smallest Three 压缩）
    void WriteQuaternion(const Quat& q) {
        // 找出绝对值最大的分量
        float values[4] = {q.x, q.y, q.z, q.w};
        int largest = 0;
        float largestVal = std::abs(values[0]);
        for (int i = 1; i < 4; i++) {
            if (std::abs(values[i]) > largestVal) {
                largest = i;
                largestVal = std::abs(values[i]);
            }
        }

        // 确保最大分量为正（利用 q 和 -q 等价性）
        if (values[largest] < 0) {
            for (int i = 0; i < 4; i++) values[i] = -values[i];
        }

        // 编码最大分量索引（2 bit）
        WriteBits(largest, 2);

        // 编码其余 3 个分量（每个 10 bit）
        float scale = 1.0f / (std::sqrt(2.0f) * 2.0f);  // 归一化因子
        int idx = 0;
        for (int i = 0; i < 4; i++) {
            if (i == largest) continue;
            float normalized = (values[i] * scale + 0.5f);
            normalized = std::clamp(normalized, 0.0f, 1.0f);
            uint32_t quantized = (uint32_t)(normalized * 1023);
            WriteBits(quantized, 10);
            idx++;
        }
    }

    // 获取打包后的数据
    const uint8_t* GetData() const { return buffer.data(); }
    int GetByteSize() const { return bytePos + (bitPos > 0 ? 1 : 0); }
    int GetBitSize() const { return bytePos * 8 + bitPos; }
};

// ===== 位读取器 =====
class BitReader {
public:
    const uint8_t* buffer;
    int bytePos = 0;
    int bitPos = 0;
    int totalBits;

    BitReader(const uint8_t* data, int bitCount)
        : buffer(data), totalBits(bitCount) {}

    uint32_t ReadBits(int numBits) {
        uint32_t value = 0;
        for (int i = 0; i < numBits; i++) {
            if (bytePos * 8 + bitPos >= totalBits) break;
            if (buffer[bytePos] & (1u << bitPos)) {
                value |= (1u << i);
            }
            bitPos++;
            if (bitPos >= 8) {
                bitPos = 0;
                bytePos++;
            }
        }
        return value;
    }

    bool ReadBool() {
        return ReadBits(1) != 0;
    }

    float ReadFloat(float min, float max, int bits) {
        uint32_t quantized = ReadBits(bits);
        float normalized = (float)quantized / ((1u << bits) - 1);
        return min + normalized * (max - min);
    }

    Vec3 ReadVec3(float min, float max, int bitsPerComponent) {
        return Vec3(
            ReadFloat(min, max, bitsPerComponent),
            ReadFloat(min, max, bitsPerComponent),
            ReadFloat(min, max, bitsPerComponent)
        );
    }

    Quat ReadQuaternion() {
        int largest = ReadBits(2);
        float components[4] = {0};

        float scale = std::sqrt(2.0f) * 2.0f;
        float sumSq = 0;
        int idx = 0;
        for (int i = 0; i < 4; i++) {
            if (i == largest) continue;
            uint32_t quantized = ReadBits(10);
            float normalized = (float)quantized / 1023.0f;
            components[i] = (normalized - 0.5f) * scale;
            sumSq += components[i] * components[i];
            idx++;
        }

        // 推导最大分量
        components[largest] = std::sqrt(1.0f - sumSq);

        return Quat(components[3], components[0], components[1], components[2]);
    }
};

// ===== Delta 压缩器 =====
class DeltaCompressor {
public:
    // 完整实体状态
    struct EntityState {
        uint32_t entityId;
        Vec3 position;
        Quat rotation;
        Vec3 velocity;
        float health;
        uint8_t flags;  // 各种状态标志
    };

    // 上次同步的状态
    struct LastSyncState {
        Vec3 position;
        Quat rotation;
        Vec3 velocity;
        float health;
        uint8_t flags;
    };

    std::unordered_map<uint32_t, LastSyncState> lastStates;

    // 打包增量状态
    int PackDelta(BitWriter& writer, const EntityState& current) {
        int bitsWritten = 0;

        auto it = lastStates.find(current.entityId);
        bool isNew = (it == lastStates.end());

        // 写入实体 ID 和是否为新实体
        writer.WriteBits(current.entityId, 16);
        writer.WriteBool(isNew);
        bitsWritten += 17;

        if (isNew) {
            // 新实体：写入完整状态
            writer.WriteVec3(current.position, -1000, 1000, 16);
            writer.WriteQuaternion(current.rotation);
            writer.WriteVec3(current.velocity, -50, 50, 12);
            writer.WriteFloat(current.health, 0, 100, 8);
            writer.WriteBits(current.flags, 8);
            bitsWritten += 16*3 + 32 + 12*3 + 8 + 8;
        } else {
            const auto& last = it->second;

            // 位置变化检查
            bool posChanged = glm::length(current.position - last.position) > 0.01f;
            writer.WriteBool(posChanged);
            bitsWritten++;
            if (posChanged) {
                // 编码位置差值（更小范围，更多精度）
                Vec3 deltaPos = current.position - last.position;
                writer.WriteVec3(deltaPos, -10, 10, 12);  // 差值范围小，用更少位
                bitsWritten += 36;
            }

            // 旋转变化检查
            float rotDiff = 1.0f - std::abs(glm::dot(current.rotation, last.rotation));
            bool rotChanged = rotDiff > 0.001f;
            writer.WriteBool(rotChanged);
            bitsWritten++;
            if (rotChanged) {
                writer.WriteQuaternion(current.rotation);
                bitsWritten += 32;
            }

            // 速度变化
            bool velChanged = glm::length(current.velocity - last.velocity) > 0.1f;
            writer.WriteBool(velChanged);
            bitsWritten++;
            if (velChanged) {
                writer.WriteVec3(current.velocity, -50, 50, 12);
                bitsWritten += 36;
            }

            // 生命值变化
            bool healthChanged = std::abs(current.health - last.health) > 0.5f;
            writer.WriteBool(healthChanged);
            bitsWritten++;
            if (healthChanged) {
                writer.WriteFloat(current.health, 0, 100, 8);
                bitsWritten += 8;
            }

            // 标志变化
            bool flagsChanged = current.flags != last.flags;
            writer.WriteBool(flagsChanged);
            bitsWritten++;
            if (flagsChanged) {
                writer.WriteBits(current.flags, 8);
                bitsWritten += 8;
            }
        }

        // 更新上次同步状态
        LastSyncState newState;
        newState.position = current.position;
        newState.rotation = current.rotation;
        newState.velocity = current.velocity;
        newState.health = current.health;
        newState.flags = current.flags;
        lastStates[current.entityId] = newState;

        return bitsWritten;
    }

    // 压缩效果统计
    struct CompressionStats {
        int totalBitsFull;
        int totalBitsDelta;
        float compressionRatio;
    };

    CompressionStats GetStats() const {
        // 统计压缩效果
        return {0, 0, 0};
    }
};

// ===== 脏标记同步系统 =====
class DirtyFlagSync {
public:
    enum DirtyFlags : uint8_t {
        DIRTY_POSITION  = 1 << 0,
        DIRTY_ROTATION  = 1 << 1,
        DIRTY_VELOCITY  = 1 << 2,
        DIRTY_HEALTH    = 1 << 3,
        DIRTY_ANIMATION = 1 << 4,
        DIRTY_INVENTORY = 1 << 5,
    };

    struct SyncedEntity {
        uint32_t entityId;
        uint8_t dirtyFlags = 0;
        Vec3 position;
        Quat rotation;
        Vec3 velocity;
        float health;
        uint8_t animState;
    };

    std::unordered_map<uint32_t, SyncedEntity> entities;

    // 设置属性并标记脏位
    void SetPosition(uint32_t id, Vec3 pos) {
        entities[id].position = pos;
        entities[id].dirtyFlags |= DIRTY_POSITION;
    }

    void SetHealth(uint32_t id, float hp) {
        entities[id].health = hp;
        entities[id].dirtyFlags |= DIRTY_HEALTH;
    }

    // 同步到客户端
    void SyncToClient(BitWriter& writer, uint32_t playerId) {
        for (auto& [id, entity] : entities) {
            if (entity.dirtyFlags == 0) continue;

            // 打包脏标记
            writer.WriteBits(id, 16);
            writer.WriteBits(entity.dirtyFlags, 6);

            if (entity.dirtyFlags & DIRTY_POSITION)
                writer.WriteVec3(entity.position, -1000, 1000, 16);
            if (entity.dirtyFlags & DIRTY_ROTATION)
                writer.WriteQuaternion(entity.rotation);
            if (entity.dirtyFlags & DIRTY_VELOCITY)
                writer.WriteVec3(entity.velocity, -50, 50, 12);
            if (entity.dirtyFlags & DIRTY_HEALTH)
                writer.WriteFloat(entity.health, 0, 100, 8);
            if (entity.dirtyFlags & DIRTY_ANIMATION)
                writer.WriteBits(entity.animState, 8);

            // 清除脏标记
            entity.dirtyFlags = 0;
        }
    }
};
```

## 压缩效果对比

| 属性 | 原始大小 | 量化后 | Delta 压缩后 | 压缩比 |
|------|---------|--------|-------------|-------|
| 位置 (Vec3) | 12 字节 | 6 字节 (16bit/轴) | 2-4 字节 (Delta) | 67-83% |
| 旋转 (Quat) | 16 字节 | 4 字节 (Smallest Three) | 4 字节 / 1 bit (无变化) | 75-99% |
| 速度 (Vec3) | 12 字节 | 4.5 字节 (12bit/轴) | 1-4 字节 | 67-92% |
| 生命值 (float) | 4 字节 | 1 字节 (8bit) | 1 bit (无变化) | 75-99% |
| 整体实体 | ~64 字节 | ~20 字节 | ~3-10 字节 | 84-95% |

## 常见陷阱与解决方案

1. **量化精度过低导致位置抖动**：16 位量化精度 0.015m，在远处仍可能可见。解决方案：对远处实体使用更低精度，近处使用高精度。

2. **Delta 累积误差**：浮点误差随时间累积。解决方案：定期发送完整状态快照作为基准。

3. **位压缩字节对齐**：跨字节边界的位操作容易出错。解决方案：使用经过充分测试的 BitWriter/BitReader 库。

4. **脏标记未在正确时机清除**：同步前清除会导致数据丢失，同步后不清除会导致重复发送。解决方案：在数据包发送确认后清除脏标记。

## 实际游戏案例

- **Quake 系列**：经典的位压缩 + Delta 压缩方案，奠定了业界基础
- **Source 引擎**：Source TV 的 Delta 快照压缩，支持观战和回放
- **Unreal Engine**：内置属性复制 + 条件压缩 + 优先级系统
- **守望先锋**：高效的状态压缩，100 玩家战场带宽控制在合理范围
- **CS2**：GoldSrc/Source 引擎的演进，高效的增量同步
