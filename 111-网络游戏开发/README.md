# 111-网络游戏开发

网络游戏开发涉及客户端-服务端架构、实时同步、延迟处理等复杂技术。本模块覆盖从同步架构到大规模在线服务的完整知识体系。

## 目录

1. [帧同步与状态同步](./帧同步与状态同步.md) - 两种核心同步架构
2. [客户端预测](./客户端预测.md) - 本地预测与回滚纠正
3. [延迟补偿技术](./延迟补偿技术.md) - 服务端回溯与命中判定
4. [网络同步策略](./网络同步策略.md) - 属性同步与兴趣管理
5. [房间匹配系统](./房间匹配系统.md) - ELO/MMR与房间管理
6. [反作弊与安全](./反作弊与安全.md) - 服务端权威与输入校验
7. [网络协议选择](./网络协议选择.md) - TCP/UDP/QUIC协议选型
8. [大世界同步](./大世界同步.md) - AOI与动态区域加载
9. [聊天与社交系统](./聊天与社交系统.md) - 消息分发与好友系统
10. [状态压缩与带宽优化](./状态压缩与带宽优化.md) - 数据压缩与增量同步

## 学习路径

```
基础层: 帧同步/状态同步 -> 客户端预测 -> 延迟补偿
进阶层: 同步策略 -> 房间匹配 -> 反作弊
高级层: 大世界同步 -> 协议选择 -> 带宽优化
```

## 推荐框架

- **引擎**: Unity Netcode、Unreal NetDriver、Godot MultiplayerAPI
- **中间件**: Photon、Mirror、Nakama、PlayFab
- **自研**: ENet/KCP + Protobuf/FlatBuffers
