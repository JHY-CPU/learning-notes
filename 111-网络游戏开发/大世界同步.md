# 大世界同步

## 核心概念

大世界同步解决 MMO 或开放世界游戏中海量实体的网络同步问题。核心技术是兴趣区域（AOI, Area of Interest）管理，只同步玩家附近和关注的实体，大幅减少带宽消耗。同时需要处理世界分区、动态加载、玩家分流等复杂问题。

## 数学基础

### AOI 范围模型

玩家的兴趣区域可以用圆形或矩形表示：

圆形 AOI：$|\mathbf{p}_{entity} - \mathbf{p}_{player}| \le R_{aoi}$

矩形 AOI：$|x_e - x_p| \le W/2$ 且 $|z_e - z_p| \le H/2$

### 网格分区效率

将世界划分为 $cellSize \times cellSize$ 的网格，查询时只需检查玩家所在格子及其周围格子：

格子索引：$cell_x = \lfloor x / cellSize \rfloor$，$cell_y = \lfloor z / cellSize \rfloor$

查询范围：$\Delta = \lceil R_{aoi} / cellSize \rceil + 1$

对比暴力遍历，网格法的加速比：

$$speedup = \frac{N_{total}}{N_{cells} \times density}$$

### 四叉树空间分割

四叉树递归地将空间四等分，直到每个叶子节点的实体数小于阈值。查询复杂度 $O(\log N + k)$，其中 $k$ 是结果数量。

## 完整 C++ 大世界同步系统

```cpp
#include <vector>
#include <unordered_map>
#include <unordered_set>
#include <queue>
#include <cmath>
#include <glm/glm.hpp>
#include <cstdint>

using Vec3 = glm::vec3;
using EntityId = uint32_t;
using PlayerId = uint32_t;
using ChunkId = uint64_t;

// ===== 空间哈希网格 =====
class SpatialGrid {
public:
    struct Cell {
        std::unordered_set<EntityId> entities;
    };

    float cellSize = 10.0f;
    std::unordered_map<int64_t, Cell> grid;

    // 将世界坐标转换为格子键
    int64_t WorldToCellKey(float x, float z) const {
        int cx = (int)std::floor(x / cellSize);
        int cz = (int)std::floor(z / cellSize);
        return ((int64_t)cx << 32) | (uint32_t)cz;
    }

    // 添加实体
    void AddEntity(EntityId eid, const Vec3& pos) {
        int64_t key = WorldToCellKey(pos.x, pos.z);
        grid[key].entities.insert(eid);
        entityCells[eid] = key;
    }

    // 移动实体
    void MoveEntity(EntityId eid, const Vec3& newPos) {
        int64_t newKey = WorldToCellKey(newPos.x, newPos.z);
        auto it = entityCells.find(eid);
        if (it != entityCells.end() && it->second != newKey) {
            // 从旧格子移除
            grid[it->second].entities.erase(eid);
            if (grid[it->second].entities.empty()) {
                grid.erase(it->second);
            }
            // 添加到新格子
            grid[newKey].entities.insert(eid);
            it->second = newKey;
        }
    }

    // 移除实体
    void RemoveEntity(EntityId eid) {
        auto it = entityCells.find(eid);
        if (it != entityCells.end()) {
            grid[it->second].entities.erase(eid);
            if (grid[it->second].entities.empty()) {
                grid.erase(it->second);
            }
            entityCells.erase(it);
        }
    }

    // 查询范围内的实体
    std::vector<EntityId> QueryRange(const Vec3& center, float radius) const {
        std::vector<EntityId> result;
        int cellRange = (int)std::ceil(radius / cellSize);
        int cx = (int)std::floor(center.x / cellSize);
        int cz = (int)std::floor(center.z / cellSize);

        for (int dx = -cellRange; dx <= cellRange; dx++) {
            for (int dz = -cellRange; dz <= cellRange; dz++) {
                int64_t key = ((int64_t)(cx + dx) << 32) | (uint32_t)(cz + dz);
                auto it = grid.find(key);
                if (it != grid.end()) {
                    for (EntityId eid : it->second.entities) {
                        // 精确距离检查
                        Vec3 entityPos = GetEntityPosition(eid);
                        Vec3 diff = entityPos - center;
                        diff.y = 0;  // 忽略高度
                        if (glm::length(diff) <= radius) {
                            result.push_back(eid);
                        }
                    }
                }
            }
        }
        return result;
    }

private:
    std::unordered_map<EntityId, int64_t> entityCells;

    Vec3 GetEntityPosition(EntityId eid) const {
        return Vec3(0);  // 实际从实体系统获取
    }
};

// ===== AOI 兴趣区域管理器 =====
class AOIManager {
public:
    float aoiRadius = 50.0f;      // 兴趣区域半径
    float hysteresisRadius = 55.0f; // 迟滞半径（避免频繁进出）

    SpatialGrid grid;

    struct PlayerAOI {
        std::unordered_set<EntityId> visibleEntities;  // 当前可见实体
        std::unordered_set<EntityId> pendingAdd;       // 待加入
        std::unordered_set<EntityId> pendingRemove;    // 待移除
    };

    std::unordered_map<PlayerId, PlayerAOI> playerAOIs;

    // 每帧更新玩家的 AOI
    void UpdatePlayerAOI(PlayerId player, const Vec3& playerPos) {
        auto& aoi = playerAOIs[player];

        // 查询当前范围内的实体
        auto nearby = grid.QueryRange(playerPos, hysteresisRadius);

        std::unordered_set<EntityId> currentSet(nearby.begin(), nearby.end());

        // 找出需要新增的实体
        for (EntityId eid : currentSet) {
            if (!aoi.visibleEntities.count(eid)) {
                aoi.pendingAdd.insert(eid);
            }
        }

        // 找出需要移除的实体（使用迟滞半径避免频繁进出）
        std::unordered_set<EntityId> inRange;
        auto inRadius = grid.QueryRange(playerPos, aoiRadius);
        inRange.insert(inRadius.begin(), inRadius.end());

        for (EntityId eid : aoi.visibleEntities) {
            if (!inRange.count(eid)) {
                aoi.pendingRemove.insert(eid);
            }
        }

        // 应用变化
        for (EntityId eid : aoi.pendingAdd) {
            aoi.visibleEntities.insert(eid);
        }
        for (EntityId eid : aoi.pendingRemove) {
            aoi.visibleEntities.erase(eid);
        }

        aoi.pendingAdd.clear();
        aoi.pendingRemove.clear();
    }

    // 获取需要同步给玩家的实体
    std::unordered_set<EntityId> GetVisibleEntities(PlayerId player) const {
        auto it = playerAOIs.find(player);
        return (it != playerAOIs.end()) ? it->second.visibleEntities
                                         : std::unordered_set<EntityId>();
    }
};

// ===== 世界区块系统 =====
class WorldChunkSystem {
public:
    static constexpr float CHUNK_SIZE = 100.0f;  // 每个区块 100x100 米

    enum ChunkState {
        UNLOADED,
        LOADING,
        LOADED,
        ACTIVE
    };

    struct Chunk {
        ChunkId id;
        ChunkState state = UNLOADED;
        std::vector<EntityId> entities;
        std::vector<PlayerId> observers;  // 在此区块附近的玩家
    };

    std::unordered_map<ChunkId, Chunk> chunks;
    std::unordered_map<PlayerId, std::unordered_set<ChunkId>> playerLoadedChunks;

    // 计算区块 ID
    ChunkId GetChunkId(float x, float z) const {
        int cx = (int)std::floor(x / CHUNK_SIZE);
        int cz = (int)std::floor(z / CHUNK_SIZE);
        return ((uint64_t)(uint32_t)cx << 32) | (uint32_t)cz;
    }

    // 获取玩家附近的区块
    std::vector<ChunkId> GetNearbyChunks(const Vec3& pos, int radius = 2) const {
        std::vector<ChunkId> result;
        int cx = (int)std::floor(pos.x / CHUNK_SIZE);
        int cz = (int)std::floor(pos.z / CHUNK_SIZE);

        for (int dx = -radius; dx <= radius; dx++) {
            for (int dz = -radius; dz <= radius; dz++) {
                ChunkId id = ((uint64_t)(uint32_t)(cx + dx) << 32) | (uint32_t)(cz + dz);
                result.push_back(id);
            }
        }
        return result;
    }

    // 更新玩家的区块加载
    void UpdatePlayerChunks(PlayerId player, const Vec3& pos) {
        auto nearby = GetNearbyChunks(pos, 2);
        auto& loaded = playerLoadedChunks[player];

        std::unordered_set<ChunkId> nearbySet(nearby.begin(), nearby.end());

        // 加载新进入范围的区块
        for (ChunkId cid : nearbySet) {
            if (!loaded.count(cid)) {
                LoadChunk(cid);
                loaded.insert(cid);
            }
        }

        // 卸载离开范围的区块
        for (auto it = loaded.begin(); it != loaded.end(); ) {
            if (!nearbySet.count(*it)) {
                UnloadChunk(*it);
                it = loaded.erase(it);
            } else {
                ++it;
            }
        }
    }

private:
    void LoadChunk(ChunkId id) {
        auto& chunk = chunks[id];
        if (chunk.state == UNLOADED) {
            chunk.state = LOADING;
            chunk.id = id;
            // 异步加载区块数据
            // LoadChunkAsync(id);
            chunk.state = LOADED;
        }
    }

    void UnloadChunk(ChunkId id) {
        auto it = chunks.find(id);
        if (it != chunks.end() && it->second.observers.empty()) {
            it->second.state = UNLOADED;
        }
    }
};

// ===== 玩家分流系统 =====
class PlayerSharding {
public:
    struct Shard {
        uint32_t shardId;
        std::string worldName;
        int playerCount;
        int maxPlayers;
        float loadFactor;  // 负载因子 0-1
    };

    std::vector<Shard> shards;

    // 分配玩家到最优分线
    uint32_t AssignShard(PlayerId player, const std::string& worldName,
                         const Vec3& preferredPos = Vec3(0)) {
        // 查找同世界的所有分线
        std::vector<Shard*> candidates;
        for (auto& shard : shards) {
            if (shard.worldName == worldName && shard.playerCount < shard.maxPlayers) {
                candidates.push_back(&shard);
            }
        }

        if (candidates.empty()) {
            // 需要创建新分线
            return CreateNewShard(worldName);
        }

        // 选择负载最低的分线
        std::sort(candidates.begin(), candidates.end(),
            [](const Shard* a, const Shard* b) {
                return a->loadFactor < b->loadFactor;
            });

        candidates[0]->playerCount++;
        candidates[0]->loadFactor = (float)candidates[0]->playerCount / candidates[0]->maxPlayers;
        return candidates[0]->shardId;
    }

private:
    uint32_t CreateNewShard(const std::string& worldName) {
        Shard newShard;
        newShard.shardId = (uint32_t)shards.size();
        newShard.worldName = worldName;
        newShard.playerCount = 0;
        newShard.maxPlayers = 100;
        newShard.loadFactor = 0;
        shards.push_back(newShard);
        return newShard.shardId;
    }
};
```

## 常见陷阱与解决方案

1. **AOI 半径过小**：边缘实体频繁进出同步，产生大量增删消息。解决方案：使用迟滞半径（加入时用大半径，移除时用小半径）。

2. **网格大小不当**：格子太小导致大量空格子，太大导致每个格子内实体过多。经验公式：$cellSize \approx 2 \times R_{aoi} / 5$。

3. **动态加载延迟**：玩家移动速度过快时，区块加载跟不上。解决方案：预测性加载（根据移动方向预加载前方区块）。

4. **分流边界实体丢失**：分线切换时实体可能重复或丢失。解决方案：使用无缝分区 + 交接区域。

## 实际游戏案例

- **魔兽世界**：区域分线 + AOI 同步 + 跨服技术
- **原神**：单人世界 + 合作模式的同步方案
- **EVE Online**：单实例大世界 + Time Dilation（时间膨胀）机制应对高负载
- **新世界 (Amazon)**：无缝分区 + 动态负载均衡
- **最终幻想14**：区域分线 + 动态实例
