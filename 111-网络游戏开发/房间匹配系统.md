# 房间匹配系统

## 核心概念

房间匹配系统负责将玩家配对到合适的游戏房间，核心挑战是在匹配速度和实力均衡之间取得平衡。常用 ELO 或 MMR 评分系统衡量玩家水平，不同游戏模式需要不同的匹配策略。

## 数学基础

### ELO 评分系统

ELO 系统由 Arpad Elo 于 1960 年代发明，核心公式：

期望得分（玩家 A 对玩家 B 的期望胜率）：

$$E_A = \frac{1}{1 + 10^{(R_B - R_A)/400}}$$

$$E_B = 1 - E_A$$

评分更新（实际结果 $S_A$, $S_B$ 为 1/0.5/0）：

$$R_A' = R_A + K \cdot (S_A - E_A)$$

$$R_B' = R_B + K \cdot (S_B - E_B)$$

其中 $K$ 是 K 因子：
- 新手（< 30 局）：$K = 40$（快速定位）
- 中级玩家：$K = 20$（标准）
- 高手（> 2000 分）：$K = 10$（稳定）

### 团队 ELO

团队 ELO 取全队平均值：

$$R_{team} = \frac{1}{N} \sum_{i=1}^{N} R_i$$

更精确的方案考虑团队内的方差：

$$R_{team} = \bar{R} + \alpha \cdot \sigma_R$$

其中 $\sigma_R$ 是团队评分标准差，$\alpha$ 是方差惩罚系数（防止高分带低分）。

### 匹配质量函数

匹配的质量可以用两个团队的评分差距衡量：

$$quality = \frac{1}{1 + |R_{team\_A} - R_{team\_B}| / scale}$$

当差距为 0 时质量最高（1.0），差距越大质量越低。

## 完整 C++ 匹配系统

```cpp
#include <vector>
#include <queue>
#include <algorithm>
#include <unordered_map>
#include <cmath>
#include <string>
#include <cstdint>

using PlayerId = uint32_t;
using RoomId = uint32_t;

// ===== ELO 评分系统 =====
class ELORatingSystem {
public:
    struct PlayerRating {
        PlayerId playerId;
        int rating;           // 显示分数
        int hiddenRating;     // 隐藏分数（用于匹配）
        int gamesPlayed;
        int wins;
        int losses;
        float kFactor;
    };

    std::unordered_map<PlayerId, PlayerRating> ratings;

    // 添加新玩家
    void AddPlayer(PlayerId playerId, int initialRating = 1000) {
        ratings[playerId] = {playerId, initialRating, initialRating, 0, 0, 0, 40.0f};
    }

    // 计算期望胜率
    float GetExpectedScore(int ratingA, int ratingB) const {
        return 1.0f / (1.0f + std::pow(10.0f, (ratingB - ratingA) / 400.0f));
    }

    // 更新评分
    void UpdateRating(PlayerId winnerId, PlayerId loserId) {
        auto& winner = ratings[winnerId];
        auto& loser = ratings[loserId];

        float expectedWin = GetExpectedScore(winner.hiddenRating, loser.hiddenRating);

        // 更新隐藏分
        winner.hiddenRating += (int)(winner.kFactor * (1.0f - expectedWin));
        loser.hiddenRating -= (int)(loser.kFactor * expectedWin * 0.5f);  // 败者扣分较少

        // 更新显示分
        winner.rating += (int)(winner.kFactor * (1.0f - expectedWin) * 0.8f);
        loser.rating = std::max(0, loser.rating - (int)(loser.kFactor * expectedWin * 0.4f));

        // 更新统计
        winner.gamesPlayed++;
        winner.wins++;
        loser.gamesPlayed++;
        loser.lossures++;

        // 根据场次调整 K 因子
        UpdateKFactor(winner);
        UpdateKFactor(loser);
    }

    // 团队评分更新
    void UpdateTeamRating(const std::vector<PlayerId>& winners,
                          const std::vector<PlayerId>& losers) {
        float teamRatingA = GetTeamRating(winners);
        float teamRatingB = GetTeamRating(losers);

        float expectedA = GetExpectedScore((int)teamRatingA, (int)teamRatingB);

        for (PlayerId pid : winners) {
            auto& player = ratings[pid];
            float personalExpected = GetExpectedScore(player.hiddenRating, (int)teamRatingB);
            // 个人贡献加权
            float contribution = 0.5f + 0.5f * (personalExpected - expectedA);
            player.hiddenRating += (int)(player.kFactor * (1.0f - expectedA) * contribution);
            player.rating += (int)(player.kFactor * (1.0f - expectedA) * 0.6f);
            player.gamesPlayed++;
            player.wins++;
            UpdateKFactor(player);
        }

        for (PlayerId pid : losers) {
            auto& player = ratings[pid];
            player.hiddenRating -= (int)(player.kFactor * expectedA * 0.4f);
            player.rating = std::max(0, player.rating - (int)(player.kFactor * expectedA * 0.3f));
            player.gamesPlayed++;
            player.lossures++;
            UpdateKFactor(player);
        }
    }

    int GetRating(PlayerId playerId) const {
        auto it = ratings.find(playerId);
        return (it != ratings.end()) ? it->second.rating : 1000;
    }

    int GetHiddenRating(PlayerId playerId) const {
        auto it = ratings.find(playerId);
        return (it != ratings.end()) ? it->second.hiddenRating : 1000;
    }

private:
    float GetTeamRating(const std::vector<PlayerId>& team) const {
        float sum = 0;
        for (PlayerId pid : team) {
            sum += GetHiddenRating(pid);
        }
        return sum / team.size();
    }

    void UpdateKFactor(PlayerRating& player) {
        if (player.gamesPlayed < 30) player.kFactor = 40.0f;
        else if (player.hiddenRating > 2000) player.kFactor = 10.0f;
        else player.kFactor = 20.0f;
    }
};

// ===== 玩家匹配条目 =====
struct MatchPlayer {
    PlayerId id;
    int mmr;                // 匹配评分
    int region;             // 区域代码
    float queueTime;        // 已等待时间
    int partySize;          // 组队人数
    std::vector<PlayerId> partyMembers;  // 队友列表
    int behaviorScore;      // 行为分
    int gameMode;           // 游戏模式
};

// ===== 匹配结果 =====
struct MatchResult {
    RoomId roomId;
    std::vector<PlayerId> teamA;
    std::vector<PlayerId> teamB;
    float matchQuality;     // 匹配质量评分 (0-1)
    float avgMMRDifference; // 平均 MMR 差距
};

// ===== 匹配系统 =====
class MatchmakingSystem {
public:
    struct Config {
        int teamSize = 5;                 // 每队人数
        int baseMMRRange = 200;           // 基础 MMR 范围
        float rangeExpandRate = 50.0f;    // 每秒扩大的范围
        float maxWaitTime = 180.0f;       // 最大等待时间（秒）
        int maxMMRRange = 1000;           // 最大 MMR 范围
        float behaviorThreshold = 0.6f;   // 行为分阈值
        bool requireSameRegion = true;    // 是否必须同区域
    };

    Config config;
    ELORatingSystem ratingSystem;

    // 匹配队列（按模式分组）
    std::unordered_map<int, std::vector<MatchPlayer>> queues;

    // 加入匹配队列
    void Enqueue(const MatchPlayer& player) {
        queues[player.gameMode].push_back(player);
    }

    // 取消匹配
    void Dequeue(PlayerId playerId, int gameMode) {
        auto& queue = queues[gameMode];
        queue.erase(
            std::remove_if(queue.begin(), queue.end(),
                [playerId](const MatchPlayer& p) { return p.id == playerId; }),
            queue.end());
    }

    // 执行匹配（每秒调用一次）
    std::vector<MatchResult> RunMatchmaking(int gameMode) {
        auto& queue = queues[gameMode];
        if (queue.size() < config.teamSize * 2) return {};

        // 1. 按隐藏 MMR 排序
        std::sort(queue.begin(), queue.end(),
            [](const MatchPlayer& a, const MatchPlayer& b) {
                return a.mmr < b.mmr;
            });

        std::vector<MatchResult> results;
        std::vector<bool> matched(queue.size(), false);

        // 2. 滑动窗口匹配
        for (int i = 0; i + config.teamSize * 2 - 1 < (int)queue.size(); i++) {
            if (matched[i]) continue;

            // 动态 MMR 范围
            int mmrRange = GetDynamicRange(queue[i].queueTime);

            // 检查窗口内是否满足条件
            if (CanFormMatch(queue, i, matched, mmrRange)) {
                MatchResult result = FormMatch(queue, i, matched);
                results.push_back(result);

                // 标记为已匹配
                for (int j = 0; j < config.teamSize * 2; j++) {
                    matched[i + j] = true;
                }
            }
        }

        // 3. 更新队列中未匹配玩家的等待时间
        UpdateQueueTimes(queue, matched);

        return results;
    }

private:
    int GetDynamicRange(float waitTime) const {
        int range = config.baseMMRRange + (int)(waitTime * config.rangeExpandRate);
        return std::min(range, config.maxMMRRange);
    }

    bool CanFormMatch(const std::vector<MatchPlayer>& queue, int startIdx,
                      const std::vector<bool>& matched, int mmrRange) {
        int endIdx = startIdx + config.teamSize * 2 - 1;
        if (endIdx >= (int)queue.size()) return false;

        // 检查 MMR 差距
        if (queue[endIdx].mmr - queue[startIdx].mmr > mmrRange) return false;

        // 检查区域一致性
        if (config.requireSameRegion) {
            int region = queue[startIdx].region;
            for (int j = startIdx; j <= endIdx; j++) {
                if (queue[j].region != region) return false;
            }
        }

        // 检查行为分
        for (int j = startIdx; j <= endIdx; j++) {
            float behaviorNorm = queue[j].behaviorScore / 10000.0f;
            if (behaviorNorm < config.behaviorThreshold) return false;
        }

        return true;
    }

    MatchResult FormMatch(const std::vector<MatchPlayer>& queue, int startIdx,
                          std::vector<bool>& matched) {
        MatchResult result;
        result.roomId = GenerateRoomId();

        // 交替分配到两队（平衡 MMR）
        for (int j = 0; j < config.teamSize; j++) {
            result.teamA.push_back(queue[startIdx + j * 2].id);
            result.teamB.push_back(queue[startIdx + j * 2 + 1].id);
        }

        // 计算匹配质量
        float mmrA = 0, mmrB = 0;
        for (auto id : result.teamA) mmrA += ratingSystem.GetHiddenRating(id);
        for (auto id : result.teamB) mmrB += ratingSystem.GetHiddenRating(id);
        mmrA /= result.teamA.size();
        mmrB /= result.teamB.size();

        result.avgMMRDifference = std::abs(mmrA - mmrB);
        result.matchQuality = 1.0f / (1.0f + result.avgMMRDifference / 200.0f);

        return result;
    }

    void UpdateQueueTimes(std::vector<MatchPlayer>& queue, const std::vector<bool>& matched) {
        for (size_t i = 0; i < queue.size(); i++) {
            if (!matched[i]) {
                queue[i].queueTime += 1.0f;  // 每秒增加
                // 超时处理
                if (queue[i].queueTime > config.maxWaitTime) {
                    // 可以降低匹配标准或通知玩家
                }
            }
        }
    }

    RoomId GenerateRoomId() {
        static RoomId nextId = 1;
        return nextId++;
    }
};
```

## 常见陷阱与解决方案

1. **热门时段队列拥堵**：大量玩家同时排队导致等待时间长。解决方案：动态调整匹配范围，或使用预匹配缓冲池。

2. **组队时 MMR 不合理**：高分玩家带低分玩家（代练）。解决方案：使用团队最高 MMR 或加权平均，而非简单平均。

3. **匹配成功后玩家不准备**：频繁重新匹配浪费资源。解决方案：设置准备超时，超时未准备则移出队列并施加冷却惩罚。

4. **跨区域匹配延迟差异**：不同区域玩家匹配到一起导致延迟不一致。解决方案：优先同区域匹配，跨区域匹配时增加延迟警告。

## 实际游戏案例

- **王者荣耀**：多段位系统 + 匹配分 + 行为分综合匹配
- **CS2**：ELO 变体 + Prime 匹配 + 信任因子（Trust Factor）
- **DOTA2**：基于 MMR 的匹配 + 行为分数系统（Conduct Summary）
- **Apex Legends**：段位 + 匹配分双重系统，赛季重置
- **英雄联盟**：隐藏 MMR + 段位系统 + 位置匹配
