# 反作弊与安全

## 核心概念

反作弊是网络游戏的核心安全挑战。基本原则是"永不信任客户端"，所有关键逻辑（伤害判定、物品生成、资源消耗）必须在服务端执行，并对客户端输入进行严格校验。安全体系需要在多个层面同时防护：服务端权威、输入校验、行为检测、客户端防护。

## 数学基础

### 行为异常检测

玩家行为可以用统计模型描述。设某操作的正常频率为 $\mu$，标准差为 $\sigma$，当实际频率 $x$ 超过阈值时标记为异常：

$$z = \frac{x - \mu}{\sigma}$$

若 $|z| > 3$（即超出 3 个标准差，概率 0.27%），判定为异常：

$$\text{suspicious} = \begin{cases} \text{true} & |z| > 3 \\ \text{false} & |z| \le 3 \end{cases}$$

### 移动速度校验

合法移动距离的上限：

$$d_{max} = v_{max} \cdot \Delta t \cdot tolerance$$

其中 $v_{max}$ 是最大移动速度，$\Delta t$ 是时间间隔，$tolerance$ 是容差因子（通常 1.2-1.5）。

### 命中率统计

作弊检测的命中率模型：

$$accuracy_{suspicious} = \frac{hits_{headshot}}{shots_{total}}$$

正常玩家的爆头率通常 15-30%，超过 50% 持续多局则高度可疑。

#include <vector>
#include <unordered_map>
#include <string>
#include <cmath>
#include <cstdint>
#include <glm/glm.hpp>

using Vec3 = glm::vec3;
using PlayerId = uint32_t;

// ===== 服务端权威移动系统 =====
class ServerAuthorityMovement {
public:
    static constexpr float TOLERANCE = 1.3f;      // 移动距离容差
    static constexpr float WALL_CHECK_DIST = 0.5f; // 穿墙检测距离

    struct PlayerState {
        Vec3 position;
        Vec3 velocity;
        float maxSpeed;
        float jumpForce;
        bool onGround;
        uint32_t lastMoveFrame;
    };

    std::unordered_map<PlayerId, PlayerState> playerStates;

    // 处理移动请求
    bool ProcessMoveRequest(PlayerId playerId, Vec3 requestedPos,
                            uint32_t clientFrame, float deltaTime) {
        auto it = playerStates.find(playerId);
        if (it == playerStates.end()) return false;

        auto& state = it->second;

        // 1. 速度校验
        Vec3 deltaMove = requestedPos - state.position;
        float moveDist = glm::length(deltaMove);
        float maxAllowedMove = state.maxSpeed * deltaTime * TOLERANCE;

        if (moveDist > maxAllowedMove) {
            LogSuspicious(playerId, "Speed hack", moveDist, maxAllowedMove);
            // 纠正：限制到最大允许距离
            if (moveDist > 0.001f) {
                requestedPos = state.position + (deltaMove / moveDist) * maxAllowedMove;
            }
            return false;  // 拒绝该移动
        }

        // 2. 穿墙检测
        if (RaycastWall(state.position, requestedPos)) {
            LogSuspicious(playerId, "Wall hack", 0, 0);
            return false;
        }

        // 3. 地面检测（防止空中无限跳跃）
        if (!state.onGround) {
            requestedPos.y = std::min(requestedPos.y, state.position.y);
        }

        // 4. 世界边界检测
        requestedPos.x = std::clamp(requestedPos.x, -1000.0f, 1000.0f);
        requestedPos.z = std::clamp(requestedPos.z, -1000.0f, 1000.0f);

        // 5. 传送检测（瞬移过大距离）
        if (moveDist > state.maxSpeed * deltaTime * 5.0f) {
            LogSuspicious(playerId, "Teleport hack", moveDist, 0);
            // 可以选择拒绝或标记审查
        }

        state.position = requestedPos;
        state.lastMoveFrame = clientFrame;
        return true;
    }

private:
    bool RaycastWall(const Vec3& from, const Vec3& to) {
        // 物理引擎射线检测（简化）
        return false;  // 实际使用物理引擎
    }

    void LogSuspicious(PlayerId pid, const char* reason, float actual, float expected) {
        // 记录可疑行为
    }
};

// ===== 输入频率限制器 =====
class InputRateLimiter {
public:
    struct PlayerLimits {
        int moveCount = 0;
        int attackCount = 0;
        int skillCount = 0;
        int jumpCount = 0;
        float lastResetTime = 0;

        // 滑动窗口统计
        std::vector<float> attackTimestamps;
        std::vector<float> moveTimestamps;
    };

    std::unordered_map<PlayerId, PlayerLimits> limits;

    // 验证攻击频率
    bool ValidateAttack(PlayerId playerId, float currentTime, int maxPerSecond) {
        auto& limit = limits[playerId];

        // 清理过期的时间戳
        float windowStart = currentTime - 1.0f;
        limit.attackTimestamps.erase(
            std::remove_if(limit.attackTimestamps.begin(), limit.attackTimestamps.end(),
                [windowStart](float t) { return t < windowStart; }),
            limit.attackTimestamps.end());

        // 检查是否超过限制
        if ((int)limit.attackTimestamps.size() >= maxPerSecond * 1.2f) {
            return false;  // 拒绝
        }

        limit.attackTimestamps.push_back(currentTime);
        return true;
    }

    // 验证移动频率
    bool ValidateMove(PlayerId playerId, float currentTime, int maxPerSecond) {
        auto& limit = limits[playerId];

        float windowStart = currentTime - 1.0f;
        limit.moveTimestamps.erase(
            std::remove_if(limit.moveTimestamps.begin(), limit.moveTimestamps.end(),
                [windowStart](float t) { return t < windowStart; }),
            limit.moveTimestamps.end());

        if ((int)limit.moveTimestamps.size() >= maxPerSecond * 1.5f) {
            return false;
        }

        limit.moveTimestamps.push_back(currentTime);
        return true;
    }
};

// ===== 行为分析系统 =====
class BehaviorAnalyzer {
public:
    struct PlayerStats {
        // 射击统计
        int totalShots = 0;
        int headshots = 0;
        int bodyshots = 0;
        int misses = 0;

        // 移动统计
        float totalDistance = 0;
        float maxSpeed = 0;

        // 评分
        float suspicionScore = 0;  // 0-100, 越高越可疑
        std::vector<std::string> flags;
    };

    std::unordered_map<PlayerId, PlayerStats> stats;

    // 记录射击
    void RecordShot(PlayerId shooter, PlayerId target, bool isHeadshot,
                    float distance, Vec3 aimDirection) {
        auto& s = stats[shooter];
        s.totalShots++;

        if (isHeadshot) {
            s.headshots++;
        } else if (target != 0) {
            s.bodyshots++;
        } else {
            s.misses++;
        }

        // 分析异常
        AnalyzeShotPattern(shooter, isHeadshot, distance);
    }

    // 分析射击模式
    void AnalyzeShotPattern(PlayerId playerId, bool isHeadshot, float distance) {
        auto& s = stats[playerId];
        if (s.totalShots < 20) return;  // 样本太少

        // 爆头率异常检测
        float headshotRate = (float)s.headshots / s.totalShots;
        float expectedRate = 0.2f;  // 正常玩家约 20%
        float stdDev = 0.1f;

        float zScore = (headshotRate - expectedRate) / stdDev;
        if (zScore > 3.0f) {
            s.suspicionScore += 10;
            s.flags.push_back("Abnormal headshot rate: " + std::to_string(headshotRate));
        }

        // 远距离爆头异常
        if (isHeadshot && distance > 50.0f) {
            float longRangeHSRate = GetLongRangeHeadshotRate(playerId);
            if (longRangeHSRate > 0.5f) {
                s.suspicionScore += 15;
                s.flags.push_back("Suspicious long-range headshots");
            }
        }
    }

    // 获取综合可疑度评分
    float GetSuspicionScore(PlayerId playerId) const {
        auto it = stats.find(playerId);
        return (it != stats.end()) ? it->second.suspicionScore : 0;
    }

private:
    float GetLongRangeHeadshotRate(PlayerId playerId) const {
        return 0;  // 实际实现中计算远距离爆头率
    }
};

// ===== 完整反作弊管理器 =====
class AntiCheatManager {
public:
    ServerAuthorityMovement movement;
    InputRateLimiter rateLimiter;
    BehaviorAnalyzer behavior;

    // 信任分级
    enum TrustLevel {
        TRUSTED,        // 正常玩家
        MONITORING,     // 正在监控
        RESTRICTED,     // 受限（降低优先级）
        SUSPENDED,      // 暂停
        BANNED          // 封禁
    };

    std::unordered_map<PlayerId, TrustLevel> trustLevels;

    // 处理玩家操作（统一入口）
    bool ValidateAction(PlayerId playerId, const PlayerAction& action,
                        float currentTime) {
        // 1. 检查信任级别
        if (trustLevels[playerId] == BANNED) return false;
        if (trustLevels[playerId] == SUSPENDED) return false;

        // 2. 输入频率检查
        switch (action.type) {
            case ACTION_ATTACK:
                if (!rateLimiter.ValidateAttack(playerId, currentTime,
                    GetMaxAttackRate(playerId))) {
                    FlagPlayer(playerId, "Attack rate exceeded");
                    return false;
                }
                break;
            case ACTION_MOVE:
                if (!rateLimiter.ValidateMove(playerId, currentTime, 60)) {
                    FlagPlayer(playerId, "Move rate exceeded");
                    return false;
                }
                break;
        }

        // 3. 行为分析
        float suspicion = behavior.GetSuspicionScore(playerId);
        if (suspicion > 80) {
            trustLevels[playerId] = SUSPENDED;
            AlertAdmin(playerId, "High suspicion score: " + std::to_string(suspicion));
        } else if (suspicion > 50) {
            trustLevels[playerId] = MONITORING;
        }

        return true;
    }

    // 获取玩家的信任惩罚因子（受监控玩家的参数更严格）
    float GetTrustFactor(PlayerId playerId) const {
        switch (trustLevels.at(playerId)) {
            case TRUSTED: return 1.0f;
            case MONITORING: return 0.8f;
            case RESTRICTED: return 0.5f;
            default: return 0.0f;
        }
    }

private:
    int GetMaxAttackRate(PlayerId playerId) const {
        float trust = GetTrustFactor(playerId);
        return (int)(10 * trust);  // 信任度越低，允许频率越低
    }

    void FlagPlayer(PlayerId playerId, const std::string& reason) {
        behavior.stats[playerId].suspicionScore += 5;
        behavior.stats[playerId].flags.push_back(reason);
    }

    void AlertAdmin(PlayerId playerId, const std::string& message) {
        // 通知管理员或自动处理系统
    }
};

## 常见陷阱与解决方案

| 陷阱 | 现象 | 解决方案 |
|------|------|---------|
| 过度校验 | 正常玩家因网络波动被误判 | 使用统计模型而非单帧判断 |
| 客户端反作弊被破解 | 无法远程更新防护 | 服务端权威 + 行为检测为主 |
| 惩罚过于严厉 | 误封引发玩家不满 | 分级惩罚（警告->限制->暂封->封禁） |
| 服务端校验开销大 | 影响服务器性能 | 空间分区加速碰撞检测 |

## 实际游戏案例

- **CS2**：VAC（Valve Anti-Cheat）+ Overwatch（玩家审查）+ Trust Factor（信任因子）
- **Valorant**：Vanguard 驱动级反作弊，Ring 0 级别内核模块
- **堡垒之夜**：EasyAntiCheat + 服务端权威
- **EAC/BattlEye**：业界主流第三方反作弊方案
- **王者荣耀**：服务端权威 + 行为检测 + 举报系统
