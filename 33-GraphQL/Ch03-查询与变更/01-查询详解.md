# 查询详解 (Query)

## 一、基本查询

```graphql
# 简单字段查询
query {
  user(id: "1") {
    name
    email
  }
}

# 命名查询 (推荐)
query GetUser($id: ID!) {
  user(id: $id) {
    id
    name
    email
    createdAt
  }
}

# 多个根查询
query GetDashboard {
  totalUsers: users {
    totalCount
  }
  recentOrders(first: 5) {
    edges {
      node {
        id
        total
      }
    }
  }
  systemHealth {
    status
    uptime
  }
}
```

## 二、变量与默认值

```graphql
# 变量类型声明
query GetUsers(
  $limit: Int = 10          # 默认值
  $offset: Int = 0
  $status: UserStatus       # 可选变量
  $search: String!          # 必填变量
) {
  users(
    limit: $limit
    offset: $offset
    status: $status
    search: $search
  ) {
    totalCount
    items {
      id
      name
    }
  }
}

# 变量值
{
  "limit": 20,
  "offset": 0,
  "search": "张三"
}
```

## 三、内联片段

```graphql
# 类型条件片段
query {
  node(id: "1") {
    id
    ... on User {
      name
      email
    }
    ... on Product {
      name
      price
    }
  }
}

# 带条件的片段
query GetUser($detailed: Boolean!) {
  user(id: "1") {
    name
    ... @include(if: $detailed) {
      email
      phone
      address {
        city
      }
    }
  }
}
```

## 四、注意事项

1. **命名查询优于匿名查询**
2. **变量类型要完整声明**
3. **必填变量用 ! 标注**
4. **多个变量用逗号分隔**
5. **内联片段用于多态场景**
