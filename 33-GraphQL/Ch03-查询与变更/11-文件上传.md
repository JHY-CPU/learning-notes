# 文件上传

## 一、GraphQL Multipart Request

```graphql
# Schema 定义
scalar Upload

type Mutation {
  uploadAvatar(file: Upload!): User!
  uploadFiles(files: [Upload!]!): [File!]!
}
```

```javascript
// Apollo Server + graphql-upload
import { graphqlUploadExpress } from 'graphql-upload';

app.use(graphqlUploadExpress({ maxFileSize: 10_000_000, maxFiles: 5 }));

const resolvers = {
  Mutation: {
    uploadAvatar: async (_, { file }) => {
      const { createReadStream, filename, mimetype } = await file;
      const stream = createReadStream();
      // 上传到 OSS
      const url = await oss.upload(stream, filename);
      return userService.updateAvatar(userId, url);
    },
  },
};
```

## 二、客户端上传

```javascript
// Apollo Client 上传
import { createUploadLink } from 'apollo-upload-client';

const client = new ApolloClient({
  link: createUploadLink({ uri: '/graphql' }),
  cache: new InMemoryCache(),
});

// 使用
const UPLOAD = gql`
  mutation UploadAvatar($file: Upload!) {
    uploadAvatar(file: $file) {
      id
      avatar
    }
  }
`;

function AvatarUpload() {
  const [upload] = useMutation(UPLOAD);

  const handleChange = async (e) => {
    const file = e.target.files[0];
    await upload({ variables: { file } });
  };

  return <input type="file" onChange={handleChange} />;
}
```

## 三、替代方案: 预签名 URL

```graphql
# 方案: 先获取上传 URL，再直传
type Mutation {
  getUploadUrl(filename: String!, contentType: String!): UploadUrl!
}

type UploadUrl {
  url: String!
  fields: JSON       # POST 表单字段
  expiresAt: DateTime!
}

# 流程:
# 1. 调用 getUploadUrl 获取签名 URL
# 2. 客户端直接上传到 OSS
# 3. 上传完成后提交文件 URL
```

## 四、注意事项

1. **GraphQL 上传用 multipart/form-data**，不是 JSON
2. **限制文件大小和类型**
3. **大文件建议用预签名 URL**
4. **文件上传走独立的存储服务**
5. **病毒扫描要集成**
