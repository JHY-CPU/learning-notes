# 订阅详解 (Subscription)

## 一、基础订阅

```graphql
# 单事件订阅
subscription OnNewMessage($chatId: ID!) {
  newMessage(chatId: $chatId) {
    id
    content
    sender {
      id
      name
      avatar
    }
    createdAt
  }
}

# 多事件订阅
subscription OnOrderUpdate($orderId: ID!) {
  orderUpdate(orderId: $orderId) {
    event           # CREATED | PAID | SHIPPED | DELIVERED
    order {
      id
      status
      updatedAt
    }
  }
}
```

## 三、客户端订阅

```javascript
// Apollo Client 订阅
import { useSubscription, gql } from '@apollo/client';

const MESSAGE_SUB = gql`
  subscription OnNewMessage($chatId: ID!) {
    newMessage(chatId: $chatId) {
      id
      content
      sender { name }
    }
  }
`;

function ChatMessages({ chatId }) {
  const { data, loading } = useSubscription(MESSAGE_SUB, {
    variables: { chatId },
    onData: ({ data }) => {
      // 每次收到新消息
      console.log('新消息:', data.data.newMessage);
    },
  });

  return <div>{/* 消息列表 */}</div>;
}
```

## 四、注意事项

1. **订阅通过 WebSocket 长连接**
2. **生产用 Redis PubSub 跨实例同步**
3. **鉴权在连接建立时做**
4. **客户端要处理断线重连**
5. **订阅过滤减少无用推送**
