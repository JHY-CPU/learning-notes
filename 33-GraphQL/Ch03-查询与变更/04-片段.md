# 片段 (Fragments)

## 一、命名片段

```graphql
# 定义片段
fragment UserFields on User {
  id
  name
  email
  avatar
}

fragment OrderFields on Order {
  id
  orderNo
  total
  status
  createdAt
}

# 使用片段
query GetUserWithOrders($userId: ID!) {
  user(id: $userId) {
    ...UserFields
    orders {
      ...OrderFields
      items {
        product { name }
        quantity
      }
    }
  }
}
```

## 二、内联片段

```graphql
# 内联片段 - 多态查询
query SearchResults($keyword: String!) {
  search(keyword: $keyword) {
    ... on User {
      name
      email
    }
    ... on Product {
      name
      price
      category { name }
    }
    ... on Order {
      orderNo
      total
    }
  }
}

# 带类型条件
query GetNode($id: ID!) {
  node(id: $id) {
    id
    ... on User {
      name
      role
    }
    ... on Product @include(if: $showProducts) {
      name
      price
    }
  }
}
```

## 三、片段组合

```graphql
# 片片可以引用其他片段
fragment UserBasic on User {
  id
  name
}

fragment UserDetail on User {
  ...UserBasic
  email
  phone
  address {
    city
    street
  }
}

fragment UserWithOrders on User {
  ...UserDetail
  orders(first: 10) {
    edges {
      node {
        id
        total
        status
      }
    }
  }
}

query {
  user(id: "1") {
    ...UserWithOrders
  }
}
```

## 四、注意事项

1. **片段用于复用字段集合**
2. **命名片段定义在文件或共享模块中**
3. **内联片段适合多态场景**
4. **片段之间不要循环引用**
5. **GraphQL Codegen 会为片段生成 TypeScript 类型**
