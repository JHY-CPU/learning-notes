# 枚举类型

## 一、基本定义

```graphql
# 枚举定义
enum OrderStatus {
  CREATED
  PAID
  SHIPPED
  DELIVERED
  CANCELLED
}

enum UserRole {
  ADMIN
  CUSTOMER
  VENDOR
  SUPPORT
}

enum SortDirection {
  ASC
  DESC
}

# 使用枚举
type Order {
  id: ID!
  status: OrderStatus!
}

type User {
  id: ID!
  role: UserRole!
}

type Query {
  orders(
    status: OrderStatus
    sort: SortDirection = DESC
  ): [Order!]!
}
```

## 二、枚举值描述

```graphql
enum OrderStatus {
  """订单已创建，等待支付"""
  CREATED
  """订单已支付，等待发货"""
  PAID
  """商品已发货"""
  SHIPPED
  """商品已送达"""
  DELIVERED
  """订单已取消"""
  CANCELLED
}
```

## 三、服务端映射

```java
// Java 枚举
public enum OrderStatus {
    CREATED, PAID, SHIPPED, DELIVERED, CANCELLED
}

// graphql-java 自动映射枚举
// 或手动注册
@Bean
public RuntimeWiring.Builder runtimeWiring() {
    return RuntimeWiring.newRuntimeWiring()
        .type("OrderStatus", builder ->
            builder.enumValues(
                name -> OrderStatus.valueOf(name)
            ))
        .type("Query", builder ->
            builder.dataFetcher("orders", env -> {
                OrderStatus status = env.getArgument("status");
                return orderRepository.findByStatus(status);
            }));
}
```

## 四、注意事项

1. **枚举值默认全大写 SNAKE_CASE**
2. **枚举值不能有前导数字**
3. **新增枚举值是安全的**（非破坏性变更）
4. **删除枚举值是破坏性的**
5. **枚举适合固定集合**，不要用 String 替代
