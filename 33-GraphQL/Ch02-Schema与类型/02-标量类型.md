# 标量类型

## 一、内置标量

```graphql
scalar Int      # 32位有符号整数
scalar Float    # 双精度浮点数
scalar String   # UTF-8 字符串
scalar Boolean  # true / false
scalar ID       # 唯一标识符（序列化为 String）
```

## 二、自定义标量

```graphql
# SDL 定义
scalar DateTime
scalar JSON
scalar Email
scalar URL
scalar BigInt
scalar Decimal
```

```java
// graphql-java 自定义标量
@Configuration
public class ScalarConfig {

    @Bean
    public GraphQLScalarType dateTimeScalar() {
        return GraphQLScalarType.newScalar()
            .name("DateTime")
            .description("ISO-8601 日期时间")
            .coercing(new Coercing<Instant, String>() {
                @Override
                public String serialize(Object dataFetcherResult) {
                    if (dataFetcherResult instanceof Instant) {
                        return ((Instant) dataFetcherResult)
                            .toString();
                    }
                    throw new CoercingSerializeException("不是有效的 DateTime");
                }

                @Override
                public Instant parseValue(Object input) {
                    return Instant.parse(input.toString());
                }

                @Override
                public Instant parseLiteral(Value<?> input) {
                    if (input instanceof StringValue) {
                        return Instant.parse(
                            ((StringValue) input).getValue());
                    }
                    throw new CoercingParseLiteralException("不是有效的 DateTime 字面量");
                }
            })
            .build();
    }
}
```

## 三、常用自定义标量库

```yaml
graphql-scalars (npm):
  DateTime:    ISO-8601 日期时间
  Date:        日期
  Time:        时间
  JSON:        任意 JSON
  Email:       邮箱格式
  URL:         URL 格式
  NonEmptyString: 非空字符串
  PositiveInt: 正整数
  BigInt:      大整数
  Currency:    货币代码
```

## 四、注意事项

1. **自定义标量需要序列化/反序列化/字面量解析**
2. **金额建议用 String 或自定义 Decimal**
3. **DateTime 统一用 ISO-8601 格式**
4. **ID 虽然内部是 String**，但语义上是标识符
5. **自定义标量要验证输入格式**
