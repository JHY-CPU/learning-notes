# 代码生成 (GraphQL Codegen)

## 一、安装配置

```bash
npm install -D @graphql-codegen/cli @graphql-codegen/typescript @graphql-codegen/typescript-operations @graphql-codegen/typescript-react-apollo
```

```yaml
# codegen.yml
overwrite: true
schema: "http://localhost:4000/graphql"
documents: "src/**/*.{ts,tsx}"
generates:
  src/generated/graphql.ts:
    plugins:
      - typescript
      - typescript-operations
      - typescript-react-apollo
    config:
      withHooks: true
      withComponent: false
      scalars:
        DateTime: string
        JSON: Record<string, unknown>
```

## 二、生成的类型

```typescript
// 自动生成的类型
export type User = {
  __typename?: 'User';
  id: string;
  name: string;
  email: string;
};

export type GetUsersQueryVariables = Exact<{
  limit: Scalars['Int'];
}>;

export type GetUsersQuery = {
  __typename?: 'Query';
  users: {
    __typename?: 'UserConnection';
    edges: Array<{
      __typename?: 'UserEdge';
      node: {
        __typename?: 'User';
        id: string;
        name: string;
      };
    }>;
  };
};

// 自动生成的 Hook
export function useGetUsersQuery(
  options?: Omit<Urql.UseQueryArgs<GetUsersQueryVariables>, 'query'>
) {
  return Urql.useQuery<GetUsersQuery>({ query: GetUsersDocument, ...options });
}
```

## 三、使用生成的代码

```tsx
import { useGetUsersQuery, useCreateUserMutation } from './generated/graphql';

function UserList() {
  // 完全类型安全
  const { data, loading } = useGetUsersQuery({
    variables: { limit: 10 },
  });

  const [createUser] = useCreateUserMutation();

  const handleCreate = async () => {
    // 变量类型自动推断
    const result = await createUser({
      variables: {
        input: { name: '张三', email: 'zhang@test.com' },
      },
    });
    // result.data?.createUser.id 自动补全
  };

  if (loading) return <Spinner />;

  return data?.users.edges.map(({ node }) => (
    <div key={node.id}>{node.name}</div>
  ));
}
```

## 四、npm scripts

```json
{
  "scripts": {
    "codegen": "graphql-codegen --config codegen.yml",
    "codegen:watch": "graphql-codegen --config codegen.yml --watch"
  }
}
```

## 五、注意事项

1. **Codegen 是必备工具**，保证类型安全
2. **schema 可以从服务端或 .graphql 文件获取**
3. **watch 模式开发时自动更新类型**
4. **scalars 映射要正确配置**
5. **CI 中要检查生成的类型是否更新**
