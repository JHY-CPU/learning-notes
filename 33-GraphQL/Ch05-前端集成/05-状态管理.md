# 状态管理

## 一、Apollo Cache 作为状态管理

```typescript
// Apollo Cache 可以替代 Redux/Zustand 的大部分功能
const client = new ApolloClient({
  link: httpLink,
  cache: new InMemoryCache({
    typePolicies: {
      Query: {
        fields: {
          // 合并策略 - 无限滚动
          feed: {
            keyArgs: false,
            merge(existing = { edges: [] }, incoming, { args }) {
              if (args?.after) {
                return {
                  ...incoming,
                  edges: [...existing.edges, ...incoming.edges],
                };
              }
              return incoming;
            },
          },
          // 本地状态字段
          isLoggedIn: {
            read() {
              return !!localStorage.getItem('token');
            },
          },
        },
      },
      User: {
        keyFields: ['id'],
        fields: {
          // 计算字段
          fullName: {
            read(user) {
              return `${user.firstName} ${user.lastName}`;
            },
          },
        },
      },
    },
  }),
});
```

## 二、本地状态管理

```typescript
// 本地字段
const typeDefs = gql`
  extend type Query {
    isLoggedIn: Boolean!
    cartItems: [ID!]!
  }

  extend type Mutation {
    login(token: String!): Boolean!
    logout: Boolean!
    addToCart(id: ID!): Boolean!
  }
`;

// 本地 Mutation
const LOGIN = gql`
  mutation Login($token: String!) {
    login(token: $token) @client
  }
`;

// 使用
function Login() {
  const [login] = useMutation(LOGIN);

  const handleLogin = async (token) => {
    localStorage.setItem('token', token);
    await login({ variables: { token } });
  };
}
```

## 三、与传统状态管理结合

```typescript
// 适合用 Apollo Cache:
// - 服务器数据
// - 缓存的查询结果
// - 需要同步的实体数据

// 适合用 Zustand/Redux:
// - UI 状态 (模态框、侧边栏)
// - 表单状态
// - 临时状态
// - 纯客户端数据

// 混合使用示例
import { create } from 'zustand';

// UI 状态用 Zustand
const useUIStore = create((set) => ({
  sidebarOpen: false,
  modal: null,
  toggleSidebar: () => set((s) => ({ sidebarOpen: !s.sidebarOpen })),
}));

// 服务器数据用 Apollo
function App() {
  const { data } = useQuery(GET_USER);
  const { sidebarOpen } = useUIStore();
}
```

## 四、注意事项

1. **Apollo Cache 可以充当状态管理器**
2. **本地字段用 @client 指令**
3. **typePolicies 定义合并和读取逻辑**
4. **纯 UI 状态用轻量状态管理库**
5. **不要过度使用 cache.modify**
