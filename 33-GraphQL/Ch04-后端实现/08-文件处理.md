# 文件处理

## 一、GraphQL 上传

```java
// Spring GraphQL 文件上传
@Controller
public class FileController {

    @MutationMapping
    public User uploadAvatar(@Argument("file") Part file,
                            @ContextValue UserContext ctx)
            throws IOException {
        String filename = file.getSubmittedFileName();
        String contentType = file.getContentType();

        // 验证
        if (!contentType.startsWith("image/")) {
            throw new IllegalArgumentException("只支持图片文件");
        }
        if (file.getSize() > 5 * 1024 * 1024) {
            throw new IllegalArgumentException("文件大小不能超过5MB");
        }

        // 上传到 OSS
        String url = ossService.upload(file.getInputStream(),
            filename, contentType);

        return userService.updateAvatar(ctx.getUserId(), url);
    }

    @MutationMapping
    public List<FileUploadResult> uploadFiles(
            @Argument("files") List<Part> files) {
        return files.stream().map(file -> {
            try {
                String url = ossService.upload(
                    file.getInputStream(),
                    file.getSubmittedFileName(),
                    file.getContentType());
                return new FileUploadResult(true, url, null);
            } catch (Exception e) {
                return new FileUploadResult(false, null, e.getMessage());
            }
        }).collect(Collectors.toList());
    }
}
```

## 二、预签名 URL 方案

```graphql
type Mutation {
  getUploadUrl(input: UploadUrlInput!): UploadUrlResult!
}

input UploadUrlInput {
  filename: String!
  contentType: String!
  size: Int!
}

type UploadUrlResult {
  uploadUrl: String!
  fileUrl: String!
  expiresAt: DateTime!
  headers: JSON
}
```

```java
@MutationMapping
public UploadUrlResult getUploadUrl(@Argument UploadUrlInput input) {
    // 验证
    validateFile(input);

    // 生成预签名 URL
    String key = generateFileKey(input.getFilename());
    PresignedUrl url = ossService.generatePresignedUrl(key,
        input.getContentType(), Duration.ofMinutes(15));

    return new UploadUrlResult(
        url.getUploadUrl(),
        url.getFileUrl(),
        url.getExpiresAt(),
        url.getHeaders()
    );
}
```

## 三、下载处理

```graphql
type Query {
  downloadFile(id: ID!): DownloadResult!
}

type DownloadResult {
  url: String!
  filename: String!
  contentType: String!
  size: Int!
}
```

## 四、注意事项

1. **大文件用预签名 URL 而非 GraphQL 上传**
2. **文件大小和类型必须验证**
3. **上传到独立存储服务（OSS/S3）**
4. **文件名要消毒处理**
5. **权限检查（谁能上传/下载）**
