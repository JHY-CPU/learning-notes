# 分页与游标

## 一、Relay 分页规范

```graphql
# Relay Connection 模式
type UserConnection {
  edges: [UserEdge!]!
  pageInfo: PageInfo!
  totalCount: Int!
}

type UserEdge {
  node: User!
  cursor: String!
}

type PageInfo {
  hasNextPage: Boolean!
  hasPreviousPage: Boolean!
  startCursor: String
  endCursor: String
}

# 查询
type Query {
  users(
    first: Int
    after: String
    last: Int
    before: String
  ): UserConnection!
}
```

## 二、使用示例

```graphql
# 前向分页
query GetUsers($first: Int!, $after: String) {
  users(first: $first, after: $after) {
    edges {
      cursor
      node {
        id
        name
        email
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
    totalCount
  }
}

# 变量
{ "first": 10, "after": "YXJyYXljb25uZWN0aW9uOjEw" }

# 获取下一页
{ "first": 10, "after": "YXJyYXljb25uZWN0aW9uOjIw" }
```

## 三、服务端实现

```java
// Spring GraphQL 分页
@Controller
public class UserController {

    @QueryMapping
    public Connection<User> users(
            @Argument Integer first,
            @Argument String after,
            @Argument Integer last,
            @Argument String before) {

        PageRequest pageRequest = buildPageRequest(first, after, last, before);
        Page<User> page = userRepository.findAll(pageRequest);

        List<Edge<User>> edges = page.getContent().stream()
            .map(user -> new DefaultEdge<>(
                user, CursorUtil.toCursor(user.getId())))
            .collect(Collectors.toList());

        PageInfo pageInfo = new DefaultPageInfo(
            page.hasNext(),
            page.hasPrevious(),
            edges.isEmpty() ? null : edges.get(0).getCursor(),
            edges.isEmpty() ? null : edges.get(edges.size() - 1).getCursor()
        );

        return new DefaultConnection<>(edges, pageInfo);
    }
}
```

## 四、注意事项

1. **游标用 Base64 编码**，不要暴露数据库 ID
2. **first/last 要设置最大限制**，防止一次取太多
3. **排序要稳定**，否则游标会失效
4. **totalCount 可选**，计算成本高
5. **Relay Connection 是社区标准**
