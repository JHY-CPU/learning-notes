# 查询语法

## 一、基础查询

```graphql
# 简单查询
{
  user(id: "1") {
    name
    email
  }
}

# 带别名 - 同一字段多次查询
{
  admin: user(id: "1") {
    name
  }
  customer: user(id: "2") {
    name
  }
}

# 片段 - 复用字段集合
fragment UserFields on User {
  id
  name
  email
  avatar
}

{
  user1: user(id: "1") {
    ...UserFields
  }
  user2: user(id: "2") {
    ...UserFields
    role
  }
}
```

## 二、变量

```graphql
# 带变量的查询
query GetUser($id: ID!, $includeOrders: Boolean!) {
  user(id: $id) {
    name
    orders @include(if: $includeOrders) {
      id
      total
    }
  }
}

# 变量值
{
  "id": "1",
  "includeOrders": true
}
```

## 三、指令

```graphql
# @include - 条件包含
query GetUser($id: ID!, $withEmail: Boolean!) {
  user(id: $id) {
    name
    email @include(if: $withEmail)
  }
}

# @skip - 条件排除
query GetUser($id: ID!, $skipAvatar: Boolean!) {
  user(id: $id) {
    name
    avatar @skip(if: $skipAvatar)
  }
}

# @deprecated - 使用已废弃字段时的警告
type User {
  oldField: String @deprecated(reason: "Use newField instead")
}
```

## 四、嵌套与关联查询

```graphql
# 深度嵌套查询
query {
  user(id: "1") {
    name
    orders(first: 5) {
      edges {
        node {
          id
          total
          items {
            product {
              name
              price
              category {
                name
              }
            }
            quantity
          }
        }
      }
      pageInfo {
        hasNextPage
        endCursor
      }
    }
  }
}
```

## 五、注意事项

1. **查询命名建议使用 PascalCase**
2. **变量类型要标注非空 !**
3. **别名适合同一字段多次查询**
4. **片段避免过度抽象**
5. **嵌套深度要控制**，防止 N+1 问题
