# 变更语法 (Mutation)

## 一、基础变更

```graphql
# 创建
mutation CreateUser($input: CreateUserInput!) {
  createUser(input: $input) {
    id
    name
    email
    createdAt
  }
}

# 变量
{
  "input": {
    "name": "张三",
    "email": "zhangsan@example.com",
    "password": "secret123"
  }
}

# 返回
{
  "data": {
    "createUser": {
      "id": "U001",
      "name": "张三",
      "email": "zhangsan@example.com",
      "createdAt": "2024-01-01T00:00:00Z"
    }
  }
}
```

## 二、更新与删除

```graphql
# 更新
mutation UpdateUser($id: ID!, $input: UpdateUserInput!) {
  updateUser(id: $id, input: $input) {
    id
    name
    email
  }
}

# 删除
mutation DeleteUser($id: ID!) {
  deleteUser(id: $id) {
    success
    message
  }
}

# 批量操作
mutation BatchCreateUsers($inputs: [CreateUserInput!]!) {
  batchCreateUsers(inputs: $inputs) {
    successCount
    failedCount
    errors { field message }
  }
}
```

## 三、Input 类型设计

```graphql
# 输入类型
input CreateUserInput {
  name: String!
  email: String!
  password: String!
  role: UserRole = CUSTOMER
  profile: CreateProfileInput
}

input CreateProfileInput {
  avatar: String
  bio: String
  phone: String
}

input UpdateUserInput {
  name: String
  email: String
  profile: UpdateProfileInput
}

input UpdateProfileInput {
  avatar: String
  bio: String
  phone: String
}

# Mutation 定义
type Mutation {
  createUser(input: CreateUserInput!): User!
  updateUser(id: ID!, input: UpdateUserInput!): User!
  deleteUser(id: ID!): DeleteResult!
}
```

## 四、Resolver 实现

```javascript
const resolvers = {
  Mutation: {
    createUser: async (_, { input }, context) => {
      // 1. 输入验证
      validateEmail(input.email);
      validatePassword(input.password);

      // 2. 唯一性检查
      const existing = await db.users.findByEmail(input.email);
      if (existing) {
        throw new GraphQLError('邮箱已被注册', {
          extensions: { code: 'DUPLICATE_EMAIL' },
        });
      }

      // 3. 业务逻辑
      const hashedPassword = await bcrypt.hash(input.password, 10);
      const user = await db.users.create({
        ...input,
        password: hashedPassword,
      });

      // 4. 返回结果
      return user;
    },

    updateUser: async (_, { id, input }, context) => {
      // 权限检查
      if (context.currentUser.id !== id && !context.currentUser.isAdmin) {
        throw new GraphQLError('无权修改', {
          extensions: { code: 'FORBIDDEN' },
        });
      }

      return db.users.update(id, input);
    },

    deleteUser: async (_, { id }, context) => {
      // 管理员权限
      if (!context.currentUser.isAdmin) {
        throw new GraphQLError('需要管理员权限', {
          extensions: { code: 'FORBIDDEN' },
        });
      }

      await db.users.delete(id);
      return { success: true, message: '用户已删除' };
    },
  },
};
```

## 五、事务处理

```javascript
// 复杂 Mutation 的数据库事务
const resolvers = {
  Mutation: {
    placeOrder: async (_, { input }, context) => {
      return await db.transaction(async (trx) => {
        // 1. 扣减库存
        for (const item of input.items) {
          const product = await trx.products.findById(item.productId);
          if (product.stock < item.quantity) {
            throw new GraphQLError('库存不足', {
              extensions: { code: 'INSUFFICIENT_STOCK' },
            });
          }
          await trx.products.decrementStock(item.productId, item.quantity);
        }

        // 2. 创建订单
        const order = await trx.orders.create({
          userId: context.currentUser.id,
          items: input.items,
          addressId: input.addressId,
        });

        // 3. 清空购物车
        await trx.cart.clear(context.currentUser.id);

        return order;
      });
    },
  },
};
```

## 六、注意事项

1. **Mutation 命名用动词 + 名词**（createUser, updateOrder）
2. **输入参数用 Input 类型包装**
3. **更新操作的字段都用可选类型**
4. **返回更新后的完整对象**
5. **删除操作返回操作结果而非被删对象**
6. **复杂操作要加数据库事务**
7. **Mutation 是顺序执行的**，不能并行
