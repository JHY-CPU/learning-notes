# 安全加固

## 一、查询限制

```yaml
安全措施:
  查询深度限制:
    - 防止嵌套攻击
    - 建议: maxDepth: 10

  查询复杂度限制:
    - 防止资源耗尽
    - 建议: maxComplexity: 1000

  查询频率限制:
    - 防止 DDoS
    - 按用户/IP 限流

  自省禁用:
    - 生产关闭
    - 防止 Schema 泄露
```

```typescript
// Apollo Server 安全配置
import depthLimit from 'graphql-depth-limit';
import { createComplexityLimitRule } from 'graphql-validation-complexity';

const server = new ApolloServer({
  schema,
  validationRules: [
    depthLimit(10),
    createComplexityLimitRule(1000),
  ],
  introspection: process.env.NODE_ENV !== 'production',
  plugins: [
    // 请求频率限制
    {
      async requestDidStart() {
        return {
          async didResolveOperation(ctx) {
            const ip = ctx.request.http.headers.get('x-forwarded-for');
            await rateLimiter.check(ip, ctx.request.operationName);
          },
        };
      },
    },
  ],
});
```

## 二、认证与授权

```typescript
// JWT 认证中间件
const context = async ({ req }) => {
  const token = req.headers.authorization?.replace('Bearer ', '');
  if (!token) return { user: null };

  try {
    const decoded = jwt.verify(token, SECRET);
    return { user: await userService.findById(decoded.id) };
  } catch {
    return { user: null };
  }
};

// 字段级授权指令
const authDirective = (role) => (resolver) => (parent, args, ctx, info) => {
  if (!ctx.user) throw new AuthenticationError('未登录');
  if (role && ctx.user.role !== role) throw new ForbiddenError('权限不足');
  return resolver(parent, args, ctx, info);
};
```

## 三、输入验证

```typescript
import { GraphQLError } from 'graphql';
import validator from 'validator';

const resolvers = {
  Mutation: {
    createUser: (_, { input }) => {
      // 验证邮箱
      if (!validator.isEmail(input.email)) {
        throw new GraphQLError('邮箱格式不正确', {
          extensions: { code: 'VALIDATION_ERROR' },
        });
      }
      // 验证密码强度
      if (!validator.isStrongPassword(input.password)) {
        throw new GraphQLError('密码强度不够', {
          extensions: { code: 'WEAK_PASSWORD' },
        });
      }
      return userService.create(input);
    },
  },
};
```

## 四、注意事项

1. **生产环境关闭 Playground**
2. **自省查询生产禁用**
3. **查询深度/复杂度必须限制**
4. **敏感字段要字段级授权**
5. **输入验证要严格**
