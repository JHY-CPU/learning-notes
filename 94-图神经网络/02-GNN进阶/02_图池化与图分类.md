# 图池化与图分类


## 1. 图分类任务概述


图分类（Graph Classification）是图级任务，目标是为每张图预测一个标签。典型应用包括分子性质预测、蛋白质功能分类、社交网络社区检测等。


### 与节点分类的区别


| 特性 | 节点分类 | 图分类 |
| --- | --- | --- |
| 输入 | 一张大图 | 多张小图 |
| 输出 | 每个节点一个标签 | 每张图一个标签 |
| 挑战 | 半监督、标签稀缺 | 将节点特征聚合为图特征 |
| 核心操作 | 消息传递 | 消息传递 + 池化（Readout） |


### 图分类的完整流程


1. 对每张图执行多层消息传递，更新节点表示
2. 通过池化（Readout）将节点表示聚合为图表示
3. 用图表示进行分类（如MLP分类器）


$$
hG = READOUT({hv(K) | v ∈ V})
                ŷ = MLP(hG)
$$


## 2. 全局池化（Global Pooling）


全局池化是最简单的Readout操作，对所有节点表示进行全局聚合。


### 三种基本全局池化


| 池化方式 | 公式 | 特点 | 适用场景 |
| --- | --- | --- | --- |
| Mean | h~G~ = (1/\|V\|) Σ h~v~ | 保留整体分布信息 | 图大小差异大 |
| Max | h~G~ = max~v~ h~v~ | 捕获最显著特征 | 关键子结构很重要 |
| Sum | h~G~ = Σ h~v~ | 保留图大小信息 | 子结构数量重要 |


> **Example:** #### 选择建议
>
>
> - **Mean**
>    ：通用选择，对图大小不敏感，适合图大小差异大的场景
> - **Sum**
>    ：当子结构数量与图标签相关时使用（如分子中某类官能团的数量）
> - **Max**
>    ：当只需关注最显著的结构特征时使用
> - **组合使用**
>    ：可以将三种池化结果拼接 [mean||max||sum]，获得更丰富的图表示


> **Note:** **局限性：**
> 全局池化丢失了图的层次结构信息，无法区分具有相同节点集但不同结构的图（在某些极端情况下）。


## 3. 层次池化（Hierarchical Pooling）


层次池化通过多步逐步压缩图的规模，保留图的层次结构信息，类似CNN中的池化层。


### DiffPool


Ying et al. (2018) 提出的可微分池化方法，通过学习聚类分配矩阵 S 将节点映射到更粗粒度的超节点。


$$
S(l) = softmax(GNNpool(A(l), H(l)))  // 聚类分配矩阵
                H(l+1) = S(l)T H(l)  // 粗化节点特征
                A(l+1) = S(l)T A(l) S(l)  // 粗化邻接矩阵
$$


### SAGPool


Self-Attention Graph Pooling，通过注意力分数选择重要节点。


$$
z = σ(GNN(A, X))  // 通过GNN计算注意力分数
                idx = top-rank(z, k)  // 选择前k个重要节点
                Xpool = Xidx ⊙ zidx  // 注意力加权的节点特征
$$


### TopKPool


基于可学习的投影向量选择前k个重要节点，计算高效。


| 方法 | 原理 | 复杂度 | 优点 | 缺点 |
| --- | --- | --- | --- | --- |
| DiffPool | 学习聚类分配 | O(n^2^) | 可端到端学习 | 内存开销大 |
| SAGPool | 注意力选择 | O(n) | 自适应选择 | 可能丢失信息 |
| TopKPool | 投影排序 | O(n) | 简单高效 | 投影向量固定 |
| MinCutPool | 最小割 | O(n^2^) | 有理论保证 | 训练不稳定 |


## 4. Readout 操作详解


Readout（读出）操作将多层GNN得到的节点表示聚合为图表示。与池化不同，Readout通常在每层GNN之后都进行，最后拼接各层的图表示。


### 分层 Readout


$$
hG(k) = READOUT({hv(k) | v ∈ V}), k = 1, ..., K
                hG = CONCAT(hG(1), hG(2), ..., hG(K))
$$


分层Readout的优势在于同时利用了局部（浅层）和全局（深层）的结构信息。


> **Example:** #### 为什么需要分层Readout？
>
>
> 浅层GNN学到的是局部结构（如化学键模式），深层GNN学到的是全局结构（如官能团）。只使用最终层的Readout会丢失局部信息，分层拼接保留了多尺度信息。


## 5. 图分类完整流程


1. **输入**
   ：一批图，每个图有节点特征和边索引
2. **消息传递**
   ：多层GNN更新节点表示
3. **Readout**
   ：将每层的节点表示聚合为图表示
4. **分类**
   ：将图表示输入分类器预测标签
5. **训练**
   ：使用交叉熵损失端到端训练


## 6. PyG 代码：图分类任务


```
import torch
import torch.nn.functional as F
from torch_geometric.nn import GCNConv, global_mean_pool, global_max_pool
from torch_geometric.loader import DataLoader
from torch_geometric.datasets import TUDataset

# ========== 加载数据集 ==========
dataset = TUDataset(root='/tmp/MUTAG', name='MUTAG')
print(f"数据集: {dataset}")
print(f"图数量: {len(dataset)}, 类别数: {dataset.num_classes}")
print(f"特征维度: {dataset.num_node_features}")

# 划分训练/测试集
train_dataset = dataset[:150]
test_dataset = dataset[150:]
train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True)
test_loader = DataLoader(test_dataset, batch_size=32)

# ========== 图分类模型 ==========
class GraphClassifier(torch.nn.Module):
    def __init__(self, in_channels, hidden_channels, out_channels, num_layers=3):
        super().__init__()
        self.convs = torch.nn.ModuleList()
        self.convs.append(GCNConv(in_channels, hidden_channels))
        for _ in range(num_layers - 1):
            self.convs.append(GCNConv(hidden_channels, hidden_channels))
        self.classifier = torch.nn.Sequential(
            torch.nn.Linear(hidden_channels * 2, hidden_channels),  # *2 因为拼接mean和max
            torch.nn.ReLU(),
            torch.nn.Dropout(0.5),
            torch.nn.Linear(hidden_channels, out_channels)
        )

    def forward(self, data):
        x, edge_index, batch = data.x, data.edge_index, data.batch
        # 多层GNN + 分层Readout
        h_list = []
        for conv in self.convs:
            x = conv(x, edge_index)
            x = F.relu(x)
            h_list.append(x)

        # 分层Readout: 每层做池化后拼接
        # 这里简化为只用最后一层
        h_mean = global_mean_pool(x, batch)  # [num_graphs, hidden]
        h_max = global_max_pool(x, batch)    # [num_graphs, hidden]
        h_graph = torch.cat([h_mean, h_max], dim=1)  # [num_graphs, hidden*2]

        return self.classifier(h_graph)


# ========== 训练与评估 ==========
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model = GraphClassifier(
    in_channels=dataset.num_node_features,
    hidden_channels=64,
    out_channels=dataset.num_classes
).to(device)
optimizer = torch.optim.Adam(model.parameters(), lr=0.01)

def train():
    model.train()
    total_loss = 0
    for data in train_loader:
        data = data.to(device)
        optimizer.zero_grad()
        out = model(data)
        loss = F.cross_entropy(out, data.y)
        loss.backward()
        optimizer.step()
        total_loss += loss.item() * data.num_graphs
    return total_loss / len(train_dataset)

@torch.no_grad()
def test(loader):
    model.eval()
    correct = 0
    for data in loader:
        data = data.to(device)
        pred = model(data).argmax(dim=1)
        correct += (pred == data.y).sum().item()
    return correct / len(loader.dataset)

for epoch in range(1, 201):
    loss = train()
    train_acc = test(train_loader)
    test_acc = test(test_loader)
    if epoch % 50 == 0:
        print(f'Epoch: {epoch:03d}, Loss: {loss:.4f}, '
              f'Train Acc: {train_acc:.4f}, Test Acc: {test_acc:.4f}')
```


## 总结


- 图分类 = 多层消息传递 + 池化（Readout）+ 分类器
- 全局池化（Mean/Max/Sum）简单但可能丢失结构信息
- 层次池化（DiffPool/SAGPool/TopKPool）保留层次结构
- 分层Readout利用多尺度信息，通常比仅用最终层效果好
- Mean+Max拼接是实践中简单有效的Readout策略
