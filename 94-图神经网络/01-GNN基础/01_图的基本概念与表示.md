# 图的基本概念与表示


## 1. 图的定义


图（Graph）是一种由**节点（顶点）**和**边**组成的非线性数据结构，用于描述对象之间的关系。


### 数学定义


$$
G = (V, E)
                其中 V = {v1, v2, ..., vn} 为节点集合，E = {e1, e2, ..., em} 为边集合
$$


### 核心概念


- **节点（Node/Vertex）**
   ：图中的基本单元，代表实体对象
- **边（Edge）**
   ：连接两个节点的线，表示节点之间的关系
- **邻接矩阵（Adjacency Matrix）A**
   ：n×n 矩阵，A
   ~ij~
   =1 表示节点 i 和 j 之间有边
- **度矩阵（Degree Matrix）D**
   ：对角矩阵，D
   ~ii~
   表示节点 i 的度（邻居数量）


> **Example:** ### 示例：一个包含4个节点的图
>
>
> 节点: V = {1, 2, 3, 4}，边: E = {(1,2), (1,3), (2,3), (3,4)}
>
>
> |  | 节点1 | 节点2 | 节点3 | 节点4 |
> | --- | --- | --- | --- | --- |
> | 节点1 | 0 | 1 | 1 | 0 |
> | 节点2 | 1 | 0 | 1 | 0 |
> | 节点3 | 1 | 1 | 0 | 1 |
> | 节点4 | 0 | 0 | 1 | 0 |
>
>
> 度矩阵 D = diag(2, 2, 3, 1)


## 2. 图的分类


### 有向图 vs 无向图


| 特性 | 无向图 | 有向图 |
| --- | --- | --- |
| 边的方向 | 无方向 | 有方向（从源到目标） |
| 邻接矩阵 | 对称矩阵 A=A^T^ | 不一定对称 |
| 度 | 单一度 | 入度 + 出度 |
| 示例 | 社交网络好友关系 | 关注关系、网页链接 |


### 加权图


每条边带有权重值，邻接矩阵 A~ij~ 存储权重值而非0/1。


- 交通网络：权重表示距离或时间
- 通信网络：权重表示带宽


### 二部图（Bipartite Graph）


节点可分为两个不相交的集合 U 和 V，每条边连接 U 中的节点和 V 中的节点。


- 用户-商品购买关系
- 作者-论文发表关系


> **Note:** **注意：**
> 在图神经网络中，我们通常处理的是无向图，但可以通过将有向边替换为两条无向边来处理有向图。


## 3. 图的表示方法


### 邻接矩阵（Adjacency Matrix）


最直观的表示方式，用 n×n 矩阵表示。适用于稠密图，空间复杂度 O(n²)。


$$
Aij = { 1, 如果 (i,j) ∈ E; 0, 否则 }
$$


### 邻接表（Adjacency List）


为每个节点维护一个邻居列表。适用于稀疏图，空间复杂度 O(n + m)。


### 边列表（Edge List）


直接存储所有边的列表，格式为 [(u~1~,v~1~), (u~2~,v~2~), ...]。GNN框架中常用 COO 格式存储。


| 表示方法 | 空间复杂度 | 查询邻居 | 添加边 | 适用场景 |
| --- | --- | --- | --- | --- |
| 邻接矩阵 | O(n²) | O(n) | O(1) | 稠密图、矩阵运算 |
| 邻接表 | O(n+m) | O(degree) | O(1) | 稀疏图、遍历 |
| 边列表 | O(m) | O(m) | O(1) | GNN框架输入 |


## 4. 图的特征


### 节点特征矩阵 X


每个节点有一个 d 维的特征向量，所有节点特征组成特征矩阵 X ∈ R^n×d^。


- 社交网络：用户的年龄、性别、兴趣等
- 分子图：原子类型、电荷等
- 引文网络：论文的词袋表示


### 图拉普拉斯矩阵（Graph Laplacian）


$$
L = D - A
$$


其中 D 为度矩阵，A 为邻接矩阵。拉普拉斯矩阵具有以下性质：


- L 是对称半正定矩阵
- 最小特征值为 0，对应特征向量为全 1 向量
- 特征值为 0 的重数等于图的连通分量数


### 归一化拉普拉斯矩阵


$$
对称归一化: Lsym = D-1/2 L D-1/2 = I - D-1/2 A D-1/2
                随机游走归一化: Lrw = D-1 L = I - D-1 A
$$


> **Note:** **图信号处理视角：**
> 拉普拉斯矩阵的特征分解对应图上的傅里叶变换，特征向量构成频域基，特征值对应频率。


## 5. 图上的任务分类


| 任务层级 | 任务描述 | 典型应用 | 输出 |
| --- | --- | --- | --- |
| 节点级 | 预测每个节点的属性 | 节点分类、节点回归 | y ∈ R^n^ |
| 边级 | 预测节点对之间的关系 | 链接预测、推荐系统 | y ∈ R^n×n^ |
| 图级 | 预测整个图的属性 | 分子性质预测、图分类 | y ∈ R |


### 半监督学习场景


在引文网络等场景中，只有少量节点有标签，需要利用图结构信息推断未标记节点的类别。GNN通过消息传递机制，将标签信息沿图结构传播。


### 训练范式


- **直推式学习（Transductive）**
   ：训练和测试在同一张图上进行（如GCN在Cora上的节点分类）
- **归纳式学习（Inductive）**
   ：训练时未见过的新图也可以推断（如GraphSAGE）


## 6. Python代码：NetworkX基本操作


```
import networkx as nx
import numpy as np
import matplotlib.pyplot as plt

# ========== 创建图 ==========
G = nx.Graph()
G.add_nodes_from([1, 2, 3, 4, 5])
G.add_edges_from([(1,2), (1,3), (2,3), (3,4), (4,5)])

print(f"节点数: {G.number_of_nodes()}")
print(f"边数: {G.number_of_edges()}")
print(f"节点度: {dict(G.degree())}")

# ========== 邻接矩阵 ==========
A = nx.adjacency_matrix(G).toarray()
print(f"邻接矩阵:\n{A}")

# 度矩阵
D = np.diag([G.degree(i) for i in G.nodes()])
print(f"度矩阵:\n{D}")

# 拉普拉斯矩阵
L = D - A
print(f"拉普拉斯矩阵:\n{L}")

# ========== 给节点添加特征 ==========
for node in G.nodes():
    G.nodes[node]['feature'] = np.random.randn(4)  # 4维特征
    G.nodes[node]['label'] = np.random.randint(0, 3)  # 3类标签

# 特征矩阵 X
X = np.array([G.nodes[n]['feature'] for n in G.nodes()])
print(f"特征矩阵形状: {X.shape}")

# ========== 常用图算法 ==========
print(f"聚类系数: {nx.clustering(G)}")
print(f"最短路径(1->5): {nx.shortest_path(G, 1, 5)}")
print(f"是否连通: {nx.is_connected(G)}")

# ========== 可视化 ==========
plt.figure(figsize=(8, 6))
pos = nx.spring_layout(G, seed=42)
colors = [G.nodes[n]['label'] for n in G.nodes()]
nx.draw(G, pos, node_color=colors, cmap=plt.cm.Set1,
        with_labels=True, node_size=600, font_size=16)
plt.title("图的可视化")
plt.tight_layout()
plt.savefig("graph_visualization.png", dpi=150)
plt.show()
```


> **Important:** **关键要点：**
> NetworkX 是最常用的 Python 图分析库，但不适合大规模图计算。对于 GNN 训练，推荐使用 PyTorch Geometric (PyG) 或 DGL。


## 总结


- 图 G=(V,E) 由节点和边组成，可用邻接矩阵 A、邻接表、边列表表示
- 图拉普拉斯矩阵 L=D-A 是谱图理论和GCN的核心工具
- 图上的任务分为节点级、边级和图级三个层次
- 特征矩阵 X 编码节点的属性信息，邻接矩阵 A 编码结构信息
- 直推式学习和归纳式学习是两种重要的训练范式
