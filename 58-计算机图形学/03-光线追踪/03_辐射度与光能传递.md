# 辐射度与光能传递


## 辐射度与光能传递

一、辐射度方法概述

## 一、辐射度方法概述


辐射度（Radiosity）是一种基于热辐射理论的全局光照算法，专门处理纯漫反射表面之间的能量交换。它将场景离散化为面片（patch），计算每个面片接收和发出的辐射能量。


### 1.1 适用条件


- 所有表面均为理想漫反射（Lambertian）
- 能量在面片之间均匀分布
- 适合室内场景（墙面、天花板、地板均为漫反射）


### 1.2 辐射度方程

**辐射度方程：**


`B_i = E_i + ρ_i × Σ_j F_ij × B_j × (A_j / A_i)`


B_i = 面片i的辐射度（出射能量密度）


E_i = 面片i的自发光


ρ_i = 面片i的反射率


F_ij = 形状因子（form factor），i接收到j辐射的比例


A_i, A_j = 面片面积
二、形状因子

## 二、形状因子（Form Factor）


形状因子F_ij描述了从面片i能"看到"面片j的程度，即面片i接收的来自面片j的辐射能量占面片j总辐射的比例。


### 2.1 形状因子定义

**微分形式因子：**


`F_ij = (1/A_i) × ∫_Ai ∫_Aj (cos θ_i × cos θ_j) / (π × r²) × V(i,j) dA_j dA_i`


θ_i = 连线与面片i法线的夹角


θ_j = 连线与面片j法线的夹角


r = 两个微面之间的距离


V(i,j) = 可见性函数（0或1，被遮挡则为0）

### 2.2 形状因子性质


| 性质 | 说明 |
| --- | --- |
| 互易性 | A_i × F_ij = A_j × F_ji |
| 归一性 | Σ_j F_ij = 1（封闭环境） |
| 非负性 | 0 ≤ F_ij ≤ 1 |
| 对称性 | 相对面片之间对称 |


### 2.3 计算方法


- **解析法：**
   仅适用于简单几何体（平行面、同轴圆盘等）
- **半立方体法：**
   在面片中心放置半立方体，将对面片投影到立方体面上
- **蒙特卡洛法：**
   随机采样射线，统计命中率
- **线性光源法：**
   利用解析公式加速计算

三、求解方法

## 三、辐射度求解方法


### 3.1 逐步求精法（Progressive Refinement）


逐步求精法不需要构建完整的形状因子矩阵，而是一次"射击"一个面片的能量到整个场景，逐步收敛。

**逐步求精算法：**


1. 初始化所有面片的辐射度 B_i = E_i


2. 选择未辐射能量最大的面片 k


3. 对每个其他面片 i，计算并累加 ΔB_i = ρ_i × F_ik × ΔB_k


4. 更新面片 k 的未辐射能量为 0


5. 重复步骤 2-4 直到能量变化小于阈值


优点：每步O(n)，交互式预览，不需要O(n²)内存存储完整矩阵

### 3.2 与光线追踪对比


| 维度 | 辐射度 | 光线追踪 |
| --- | --- | --- |
| 适用材质 | 仅漫反射 | 任意BRDF |
| 视图相关 | 否（预计算） | 是（逐像素） |
| 软阴影 | 自然产生 | 需要特殊处理 |
| 颜色渗色 | 自然产生 | 需要多次反弹 |
| 镜面反射 | 不支持 | 原生支持 |
| 适用场景 | 室内漫反射场景 | 通用场景 |

四、光子映射

## 四、光子映射（Photon Mapping）


光子映射是Henrik Jensen提出的两-pass全局光照算法，特别擅长处理焦散（caustics）和漫反射间接光照。


### 4.1 算法流程


1. **Pass 1 - 光子发射：**
   从光源发射大量光子，在表面碰撞点存储光子信息（位置、入射方向、能量）到光子图（KD-Tree）
2. **Pass 2 - 渲染：**
   从相机追踪光线，在漫反射碰撞点查询附近光子，用核密度估计计算辐射度


### 4.2 光子图类型


| 类型 | 存储位置 | 用途 |
| --- | --- | --- |
| 焦散光子图 | 第一次碰撞为漫反射，后续为镜面/折射 | 焦散效果 |
| 全局光子图 | 所有碰撞点 | 漫反射间接光照 |

**光子密度估计：**


`L(p, ω_o) ≈ (1 / πr²) × Σ f_r × ΔΦ_i × δ(ω_i, ω_o)`


找到最近的N个光子，计算覆盖半径r，用核函数加权求和估计辐射度。


N越大越平滑但越模糊，N越小越精确但噪声越大。

### 4.3 焦散效果


焦散（Caustics）是光线通过镜面反射或折射后聚焦在漫反射表面上形成的明亮图案。光子映射特别擅长表现这种效果，因为从光源发射的光子会在焦散区域密集分布。


> **Note:** **为什么路径追踪难以处理焦散：**
>
>
> 焦散需要一条"漫反射→镜面→光源"的光路。从相机随机追踪很难恰好命中这个极小的概率事件。光子映射从光源出发，自然地将光子集中在焦散区域。

五、现代全局光照

## 五、现代全局光照技术


### 5.1 VCM（Vertex Connection and Merging）


VCM统一了BDPT和光子映射：先尝试直接连接路径顶点（BDPT），如果距离太近则合并（光子映射），综合两种方法的优势。


### 5.2 实时光线追踪


| 技术 | 原理 | 应用 |
| --- | --- | --- |
| 屏幕空间GI（SSGI） | 利用屏幕空间信息估计间接光照 | UE4/Lumen |
| DDGI（漫反射GI探针） | 空间中放置探针存储辐射度 | NVRTX |
| ReSTIR | 时空域重用光追样本 | 实时路径追踪 |
| Lumen（UE5） | 软件光追+表面缓存+SDF | 游戏引擎 |
