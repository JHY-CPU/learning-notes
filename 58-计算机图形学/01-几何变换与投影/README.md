# 01 - 几何变换与投影

> 几何变换与投影是计算机图形学的数学基础，负责将三维世界中的物体映射到二维屏幕。

---

## 1. 二维变换

### 1.1 平移变换（Translation）

平移将点沿指定方向移动固定距离：

```
x' = x + tx
y' = y + ty
```

平移不是线性变换（不能用 2×2 矩阵表示），因此引入齐次坐标。

### 1.2 旋转变换（Rotation）

绕原点逆时针旋转 θ 角：

```
x' = x·cosθ - y·sinθ
y' = x·sinθ + y·cosθ
```

旋转矩阵 R(θ) 是正交矩阵，其逆等于转置，即 R(-θ)。

### 1.3 缩放变换（Scaling）

相对原点进行缩放：

```
x' = sx · x
y' = sy · y
```

- sx > 1 水平拉伸，sx < 1 水平压缩
- sy 为负值时包含翻转

### 1.4 错切变换（Shear）

保持 y 坐标不变，x 坐标随 y 值偏移：

```
x' = x + shx · y
y' = y
```

常用于制作倾斜效果。

---

## 2. 齐次坐标（Homogeneous Coordinates）

为统一处理平移和线性变换，引入齐次坐标：

- **点** 表示为 (x, y, 1)
- **向量** 表示为 (x, y, 0)

齐次坐标的优点：
- 平移可以用矩阵乘法表示
- 所有变换（平移、旋转、缩放、错切）统一为 3×3 矩阵
- w ≠ 0 时，(x, y, w) 与 (x/w, y/w, 1) 表示同一个点
- 无穷远点可表示为 (x, y, 0)

### 齐次坐标下的变换矩阵

| 变换类型 | 矩阵形式 |
|---------|---------|
| 平移 | T(tx, ty) = [[1,0,tx],[0,1,ty],[0,0,1]] |
| 旋转 | R(θ) = [[cosθ,-sinθ,0],[sinθ,cosθ,0],[0,0,1]] |
| 缩放 | S(sx,sy) = [[sx,0,0],[0,sy,0],[0,0,1]] |

---

## 3. 变换矩阵的复合

多个变换可以组合为一个矩阵。关键原则：

- **变换顺序影响结果**：先旋转后平移 ≠ 先平移后旋转
- **矩阵乘法从右向左应用**：P' = M₃·M₂·M₁·P 中，M₁ 先作用
- **绕任意点旋转** = 平移至原点 → 旋转 → 平移回去
  - `M = T(cx,cy) · R(θ) · T(-cx,-cy)`

复合的优势：对 n 个点做 k 次变换，复合后只需 n 次矩阵乘法而非 n×k 次。

---

## 4. 坐标系变换流水线

图形渲染中存在多个坐标系，点依次经过以下变换：

```
模型坐标系 → 世界坐标系 → 观察坐标系 → 裁剪坐标系 → NDC → 屏幕坐标系
 (Model)    (World)      (View)       (Clip)       (NDC)   (Screen)
```

### 各阶段说明

| 阶段 | 说明 |
|------|------|
| **模型→世界** | 将物体从局部坐标放入世界场景，使用模型矩阵 M |
| **世界→观察** | 将世界坐标变换到以相机为原点的坐标系，使用观察矩阵 V |
| **观察→裁剪** | 应用投影矩阵 P，进入齐次裁剪空间 |
| **裁剪→NDC** | 透视除法（除以 w 分量），坐标范围归一化到 [-1,1] |
| **NDC→屏幕** | 视口变换，映射到像素坐标 |

完整的 **MVP 变换**：`P_clip = P × V × M × P_model`

## 5. 三维变换

### 5.1 3D 平移与缩放

齐次坐标下使用 4×4 矩阵：

- 平移 T(tx, ty, tz)
- 缩放 S(sx, sy, sz)

### 5.2 绕坐标轴旋转

- 绕 X 轴：x 不变，y-z 平面内旋转
- 绕 Y 轴：y 不变，x-z 平面内旋转
- 绕 Z 轴：z 不变，x-y 平面内旋转

### 5.3 绕任意轴旋转（Rodrigues 公式）

给定单位轴 **n** 和角度 θ，旋转矩阵为：

```
R = cosθ·I + (1-cosθ)·n·nᵀ + sinθ·[n]×
```

其中 [n]× 是 **n** 的反对称矩阵。该公式是三维旋转的核心工具。

---

## 6. 观察变换（View Transform）

### 6.1 相机描述

相机由位置 **eye**、观察方向 **look**、上方向 **up** 三个参数定义。

### 6.2 Look-At 矩阵

1. 计算相机坐标系的三个基向量：
   - **w** = -normalize(look - eye)  （相机看向 -w 方向）
   - **u** = normalize(up × w)
   - **v** = w × u
2. 构造旋转矩阵 R 和平移矩阵 T
3. View 矩阵 V = R · T(-eye)

---

## 7. 投影变换

### 7.1 正交投影（Orthographic Projection）

平行投影，物体大小不随距离变化：
- 将长方体 [l,r]×[b,t]×[f,n] 映射到标准立方体 [-1,1]³
- 先平移到原点，再缩放到标准范围

正交投影保留平行线的平行性，常用于工程制图和 CAD。

### 7.2 透视投影（Perspective Projection）

近大远小的投影效果：
- 所有投影线汇聚于一点（灭点）
- 一点透视：一个灭点（一个主消隐方向）
- 两点透视：两个灭点
- 三点透视：三个灭点

### 7.3 透视投影矩阵

将视锥体 [l,r]×[b,t]×[n,f] 变换到裁剪空间：

```
M_proj = [[2n/(r-l),  0,        (r+l)/(r-l),    0       ],
          [0,         2n/(t-b), (t+b)/(t-b),    0       ],
          [0,         0,       -(f+n)/(f-n),   -2fn/(f-n)],
          [0,         0,       -1,              0       ]]
```

透视投影矩阵的特点：
- 第 4 行将 -z 写入 w 分量，为透视除法做准备
- 透视除法后，z 被映射到 [-1,1]

---

## 8. 视锥体与裁剪

### 8.1 视锥体（View Frustum）

视锥体定义了相机可见的空间范围，由六个裁剪平面围成：
- 近裁剪面（Near）、远裁剪面（Far）
- 左、右、上、下四个侧面

### 8.2 裁剪算法

- **Cohen-Sutherland 编码**：用于 2D 线段裁剪，快速判断完全可见/完全不可见/部分可见
- **Liang-Barsky 参数化裁剪**：用参数方程更高效地计算交点
- **齐次裁剪**：在裁剪空间中对图元进行六面裁剪，是 GPU 的标准做法

---

## 9. 背面剔除（Back-Face Culling）

### 9.1 原理

对于不透明封闭物体，朝向相机背面的面片不可见，可以直接丢弃以减少渲染负担。

### 9.2 判断方法

计算面片法线 **n** 与视线方向的点积：
- **n** · (camera - point) > 0：面向相机，保留
- **n** · (camera - point) < 0：背向相机，剔除

在观察坐标系中，若投影后多边形顶点为顺时针（或逆时针，取决于约定），则为背面。

### 9.3 局限性

- 仅适用于封闭凸物体
- 开放曲面或半透明物体不能简单剔除
- 需要与深度测试配合使用
