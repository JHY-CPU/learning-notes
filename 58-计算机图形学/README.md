## 知识体系概览

```
计算机图形学
├── 几何变换与投影        — 将 3D 世界映射到 2D 屏幕的数学基础
├── 光栅化与着色          — 将几何图元转化为像素并赋予颜色
├── 光线追踪              — 基于物理光学的高质量渲染方法
└── GPU 与渲染管线        — 硬件加速图形处理的架构与 API
```

## 推荐学习路径

1. **几何变换与投影** — 理解坐标系、矩阵变换和投影原理，是后续所有内容的基础。
2. **光栅化与着色** — 掌握实时渲染核心：光栅化、光照、纹理和阴影。
3. **GPU 与渲染管线** — 学习现代 GPU 架构和可编程管线，能够编写 Shader。
4. **光线追踪** — 深入离线渲染与实时光追技术，理解全局光照。
