# 着色模型


## 着色模型

一、光照基础

## 一、光照基础


着色模型计算物体表面每个点的颜色，模拟光与材质的交互效果。


### 1.1 光的分量


| 分量 | 英文 | 特点 |
| --- | --- | --- |
| 环境光 | Ambient | 来自四面八方的间接光照 |
| 漫反射 | Diffuse | 光线均匀向各方向散射 |
| 镜面反射 | Specular | 光线沿反射方向集中反射 |

二、Lambert漫反射

## 二、Lambert漫反射模型


Lambert模型描述理想漫反射表面的光照效果，光强与入射角的余弦成正比。

**漫反射公式：**


`I_diffuse = Kd × I_light × max(0, N · L)`


Kd = 漫反射系数（材质颜色）


I_light = 光源强度


N = 表面法线（单位向量）


L = 光源方向（单位向量）

> **Note:** **Lambert余弦定律：**
> 表面接收到的光能与入射角的余弦成正比。当光线垂直照射时接收到最多光能。

三、Phong模型

## 三、Phong着色模型


Phong模型是经典的局部光照模型，结合环境光、漫反射和镜面反射三种分量。


### 3.1 Phong反射方程

`I = Ka × Ia + Kd × Id × max(0, N·L) + Ks × Is × max(0, R·V)^n`


Ka, Kd, Ks = 环境光、漫反射、镜面反射系数


Ia, Id, Is = 环境光、漫反射光、镜面反射光强度


R = 反射方向 = 2(N·L)N - L


V = 观察方向


n = 光泽度指数（shininess）

### 3.2 光泽度指数的影响


| n值 | 效果 | 材质示例 |
| --- | --- | --- |
| 1-10 | 高光范围大，柔和 | 粗糙塑料 |
| 20-50 | 中等高光 | 光滑塑料 |
| 100+ | 高光尖锐，接近镜面 | 金属、镜子 |

四、Blinn-Phong模型

## 四、Blinn-Phong模型


Blinn-Phong是Phong模型的改进版，使用半程向量替代反射向量，计算效率更高。


### 4.1 半程向量

**半程向量（Halfway Vector）：**


`H = normalize(L + V)`


**Blinn-Phong镜面反射：**


`I_specular = Ks × Is × max(0, N·H)^n`

### 4.2 Phong vs Blinn-Phong


| 维度 | Phong | Blinn-Phong |
| --- | --- | --- |
| 计算 | 反射向量R与V点积 | 半程向量H与N点积 |
| 效率 | 需要计算R | 直接计算H，更高效 |
| 效果 | 高光较为集中 | 高光更自然 |
| 应用 | 传统渲染 | OpenGL默认、游戏引擎 |

五、着色频率

## 五、着色频率（Shading Frequency）


| 频率 | 方法 | 特点 |
| --- | --- | --- |
| Flat Shading | 每三角形计算一次 | 面片感明显，计算最少 |
| Gouraud Shading | 每顶点计算，片段间插值 | 较平滑，但高光可能失真 |
| Phong Shading | 每片段计算，法线插值 | 最平滑，计算量最大 |


> **Note:** **注意：**
> Phong着色（Shading）指的是逐片段计算光照，与Phong模型（Model）是不同的概念。Phong着色可以使用任何光照模型。

六、纹理映射

## 六、纹理映射（Texture Mapping）


纹理映射将2D图像（纹理）贴到3D模型表面，增加表面细节而不增加几何复杂度。


### 6.1 纹理坐标


- **UV坐标：**
   (u,v) ∈ [0,1]²，定义纹理上的采样位置
- 每个顶点关联一个UV坐标
- 三角形内部的UV通过重心坐标插值


### 6.2 纹理过滤


| 方法 | 说明 | 效果 |
| --- | --- | --- |
| 最近邻（Nearest） | 取最近的纹素 | 像素化，快速 |
| 双线性（Bilinear） | 4个纹素插值 | 平滑 |
| 三线性（Trilinear） | 双线性 + Mipmap层间插值 | 最平滑 |


### 6.3 Mipmap

**Mipmap：**
预先计算纹理的多级分辨率版本，每级分辨率减半。


解决远处纹理采样的走样（Aliasing）问题。


额外存储空间增加33%。

### 6.4 纹理寻址模式


| 模式 | 说明 |
| --- | --- |
| Wrap / Repeat | 重复纹理 |
| Clamp / Edge | 使用边缘像素填充 |
| Mirror | 镜像重复 |
| Border | 使用指定边界色 |
