# 纹理映射与采样


## 纹理映射与采样

一、纹理映射概述

## 一、纹理映射概述


纹理映射（Texture Mapping）是将2D图像（纹理）贴到3D模型表面的技术。它在不增加几何复杂度的前提下大幅增强场景的视觉细节。


### 1.1 基本流程


1. 为每个顶点指定纹理坐标（UV坐标）
2. 光栅化时对UV坐标进行插值（透视校正）
3. 对每个像素（fragment），根据UV坐标从纹理中采样颜色

**UV坐标：**
纹理坐标通常用(u, v)表示，范围[0, 1]。u=0对应纹理左边缘，u=1对应右边缘；v=0对应下边缘，v=1对应上边缘（OpenGL约定）。
二、UV映射

## 二、UV映射方式


UV映射定义了如何将2D纹理坐标分配给3D模型的顶点。不同几何形状需要不同的映射策略。


### 2.1 常见映射方式


| 方式 | 原理 | 适用模型 |
| --- | --- | --- |
| 平面映射 | 沿一个方向投影到平面 | 地面、墙面、纸张 |
| 柱面映射 | 将纹理包裹到圆柱面 | 瓶子、圆柱体 |
| 球面映射 | 使用球坐标(u=θ/2π, v=φ/π) | 球体、地球仪 |
| 立方体映射 | 六个面分别映射 | 天空盒、环境映射 |
| 展开映射（UV Unwrap） | 手动展开网格到2D平面 | 复杂角色、建筑 |


### 2.2 UV展开注意事项


- **接缝（Seam）：**
   UV不连续的边界，需要尽量隐藏在不显眼的位置
- **拉伸（Stretching）：**
   UV拉伸会导致纹理变形，应保持面积和角度一致性
- **纹理利用率：**
   尽量填满[0,1]空间，减少浪费
- **纹理密度：**
   不同区域可使用不同密度（重要部位用更多纹素）

三、纹理采样

## 三、纹理采样（Texture Sampling）


纹理采样是从连续的UV坐标映射到离散的纹素（texel）的过程。由于屏幕像素和纹素通常不是一一对应关系，需要插值算法。


### 3.1 最近邻采样（Nearest Neighbor）


取最近的纹素作为采样结果。速度快但会产生明显的锯齿（aliasing）。

**最近邻采样：**


`texel = texture[int(u * width), int(v * height)]`


优点：计算简单，适合像素风格


缺点：放大时马赛克，缩小远处严重锯齿

### 3.2 双线性插值（Bilinear Interpolation）


取最近的四个纹素进行双线性插值，产生平滑过渡。

**双线性插值步骤：**


1. 根据UV找到包围的四个纹素(u0,v0), (u1,v0), (u0,v1), (u1,v1)


2. 计算在四个纹素中的相对位置(s, t)


3. 水平方向插值：c0 = lerp(T(u0,v0), T(u1,v0), s)


4. 水平方向插值：c1 = lerp(T(u0,v1), T(u1,v1), s)


5. 垂直方向插值：result = lerp(c0, c1, t)


`lerp(a, b, t) = a × (1-t) + b × t`
四、Mipmapping

## 四、Mipmap（多级渐远纹理）


Mipmap通过预计算纹理的多级缩小版本来解决远处物体纹理采样的走样问题。每一级的分辨率是上一级的一半。


### 4.1 Mipmap层级


| 层级 | 分辨率 | 纹素数量 |
| --- | --- | --- |
| Level 0 | 256 × 256 | 65536 |
| Level 1 | 128 × 128 | 16384 |
| Level 2 | 64 × 64 | 4096 |
| Level 3 | 32 × 32 | 1024 |
| ... | ... | ... |
| Level n | 1 × 1 | 1 |


> **Note:** **Mipmap内存开销：**
> 所有层级的总内存约为原始纹理的4/3倍（1 + 1/4 + 1/16 + ... = 4/3）。这是可以接受的额外开销。


### 4.2 Mipmap层级选择

**层级D的计算：**


D = log₂(max(|du/dx|, |dv/dx|, |du/dy|, |dv/dy|))


du/dx和dv/dx表示u、v在屏幕x方向的变化率


D=0表示纹理比屏幕像素大（近处），需要原始分辨率


D>0表示纹理比屏幕像素小（远处），使用缩小版本

### 4.3 三线性过滤（Trilinear Filtering）


三线性过滤在Mipmap的两个相邻层级上分别进行双线性插值，然后在层级之间再次插值，实现最平滑的过渡。


1. 根据像素覆盖面积计算Mipmap层级D
2. 在Level floor(D)上做双线性插值，得到color1
3. 在Level ceil(D)上做双线性插值，得到color2
4. 对color1和color2根据D的小数部分插值

五、各向异性过滤

## 五、各向异性过滤（Anisotropic Filtering）


Mipmap假设纹理在屏幕空间中各方向均匀缩放，但实际中（如地面纹理沿视线方向）缩放是各向异性的，Mipmap会导致过度模糊。


### 5.1 问题分析


当地面纹理与视线成倾斜角度时，纹理在屏幕x和y方向的缩放比例不同。例如远端的地面纹理在y方向被严重压缩，但x方向变化不大。


### 5.2 各向异性过滤原理

**各向异性过滤：**


- 沿最长的缩放方向采集多个样本（rip-map或EWA滤波）


- 各向异性比率（如4x、8x、16x）表示最大采样倍数


- 4x表示沿一个方向最多采样4个纹素再平均


- 效果：远处地面纹理清晰度大幅提升


- 代价：额外的纹理采样，性能略有下降

| 过滤方式 | 质量 | 性能 | 适用场景 |
| --- | --- | --- | --- |
| 最近邻 | 最差 | 最快 | 像素风格 |
| 双线性 | 一般 | 快 | 近处纹理 |
| 三线性 | 好 | 中等 | 通用 |
| 各向异性 4x | 很好 | 较慢 | 地面、墙面 |
| 各向异性 16x | 最佳 | 最慢 | 高质量渲染 |

六、法线贴图

## 六、法线贴图（Normal Mapping）


法线贴图通过在纹理中存储表面法线信息，在不增加几何细节的情况下模拟凹凸效果。法线信息存储在切线空间中。


### 6.1 切线空间


切线空间是以模型表面为参考的局部坐标系，由三个向量组成：


- **T（Tangent，切线）：**
   沿纹理u方向
- **B（Bitangent，副切线）：**
   沿纹理v方向
- **N（Normal，法线）：**
   垂直于表面


### 6.2 法线贴图编码

**法线编码方式：**


- 切线空间中的法线(Nx, Ny, Nz)范围是[-1, 1]


- 存储到RGB纹理需要映射到[0, 255]：


`color = normal × 0.5 + 0.5`


- 读取时反向映射：


`normal = color × 2.0 - 1.0`


- 平坦表面法线=(0,0,1)对应颜色=(0.5, 0.5, 1.0)即浅蓝色

### 6.3 法线贴图着色


```
// GLSL片段着色器 - 法线贴图
uniform sampler2D normalMap;
uniform sampler2D diffuseMap;

in vec2 fragUV;
in vec3 fragT;  // 切线
in vec3 fragB;  // 副切线
in vec3 fragN;  // 法线

void main() {
    // 从法线贴图读取切线空间法线
    vec3 normalTS = texture(normalMap, fragUV).rgb * 2.0 - 1.0;

    // 构建TBN矩阵，将法线变换到世界空间
    mat3 TBN = mat3(normalize(fragT), normalize(fragB), normalize(fragN));
    vec3 normalWS = normalize(TBN * normalTS);

    // 使用世界空间法线进行光照计算
    vec3 lightDir = normalize(lightPos - fragPos);
    float diff = max(dot(normalWS, lightDir), 0.0);
    vec3 color = texture(diffuseMap, fragUV).rgb * diff;
}
```

七、其他常见纹理类型

## 七、其他常见纹理类型


| 纹理类型 | 通道 | 内容 | 用途 |
| --- | --- | --- | --- |
| 漫反射贴图（Diffuse） | RGB | 物体基础颜色 | 表面颜色 |
| 法线贴图（Normal） | RGB | 表面法线方向 | 凹凸细节 |
| 高光贴图（Specular） | R/G/RGB | 高光强度和颜色 | 高光控制 |
| 高度贴图（Height） | R | 表面高度值 | 视差映射 |
| 粗糙度贴图（Roughness） | R | 表面粗糙度 | PBR材质 |
| 金属度贴图（Metallic） | R | 表面金属程度 | PBR材质 |
| 环境光遮蔽（AO） | R | 环境光遮蔽值 | 全局光照近似 |
| 环境贴图（Environment） | RGB | 周围环境颜色 | 反射/折射 |
