## 渲染管线概述

一、渲染管线概述

## 一、渲染管线（Rendering Pipeline）


渲染管线定义了将3D场景数据转换为2D屏幕图像的一系列处理阶段。现代GPU采用高度并行化的管线架构。


### 1.1 管线总体流程

**顶点数据**
→
**顶点着色器**
→ (曲面细分) → (几何着色器) →
**光栅化**
→
**片段着色器**
→
**帧缓冲**
二、固定管线 vs 可编程管线

## 二、固定管线 vs 可编程管线


| 维度 | 固定管线 | 可编程管线 |
| --- | --- | --- |
| 灵活性 | 预定义功能 | 自定义着色器程序 |
| 时代 | OpenGL 1.x-2.x | OpenGL 3.0+ / DirectX 10+ |
| 可编程阶段 | 无 | 顶点/几何/片段着色器 |
| 效果能力 | 有限的固定效果 | 任意视觉效果 |
| 性能 | 硬件优化固定 | 开发者可深度优化 |

三、着色器阶段

## 三、可编程着色器阶段


### 3.1 顶点着色器（Vertex Shader）


- **输入：**
   单个顶点的属性（位置、法线、纹理坐标等）
- **执行次数：**
   每个顶点执行一次
- **核心任务：**

   - 应用MVP变换，将顶点从模型空间变换到裁剪空间
   - 传递顶点属性到后续阶段
   - 顶点动画（骨骼动画、变形）


### 3.2 曲面细分着色器（Tessellation Shader）


- **细分控制着色器（TCS）：**
   控制细分程度
- **细分器（Tessellator）：**
   硬件固定功能，生成新顶点
- **细分评估着色器（TES）：**
   计算新顶点的位置
- **用途：**
   LOD自适应、平滑曲面


### 3.3 几何着色器（Geometry Shader）


- **输入：**
   一个完整图元（点、线、三角形）
- **输出：**
   零个或多个图元
- **能力：**
   创建或销毁几何体
- **用途：**
   粒子系统、毛发、Billboard


> **Note:** **注意：**
> 几何着色器性能开销较大，在性能敏感的场景中应谨慎使用。


### 3.4 片段着色器（Fragment Shader）


- **输入：**
   插值后的片段属性（来自光栅化）
- **执行次数：**
   每个片段执行一次
- **核心任务：**

   - 计算光照（Phong/Blinn-Phong/PBR）
   - 纹理采样和混合
   - 阴影计算
   - 后处理效果

**性能关键：**
片段着色器是管线中最耗时的阶段，因为执行次数最多（每个像素可能执行多次）。
四、管线详细阶段

## 四、管线详细阶段


### 4.1 应用阶段（CPU）


- 场景管理和剔除（视锥剔除、遮挡剔除）
- 准备渲染命令和数据
- 设置着色器和资源状态
- 提交绘制调用（Draw Call）


### 4.2 几何阶段（GPU）


1. 顶点着色器：变换顶点
2. 曲面细分（可选）
3. 几何着色器（可选）
4. 裁剪：移除视锥体外的图元
5. 背面剔除：移除背对相机的面
6. 透视除法和视口变换


### 4.3 光栅化阶段（GPU）


1. 三角形设置：计算边方程
2. 三角形遍历：生成片段
3. 属性插值：计算每个片段的属性


### 4.4 片段处理阶段（GPU）


1. 片段着色器：计算片段颜色
2. Early-Z测试（在着色器前）
3. 模板测试
4. 深度测试（Z-Buffer）
5. 混合（Blending）
6. 写入帧缓冲

五、现代渲染技术

## 五、现代渲染技术


### 5.1 计算着色器（Compute Shader）


- 通用GPU计算，不绑定渲染管线
- 支持任意数据读写
- 用途：粒子模拟、后处理、光线追踪加速


### 5.2 延迟渲染（Deferred Rendering）


- 第一遍：将几何信息渲染到G-Buffer
- 第二遍：基于G-Buffer计算光照
- 优势：光照复杂度与几何复杂度解耦


### 5.3 基于物理的渲染（PBR）


- 基于能量守恒和微面元理论
- 使用金属度和粗糙度等物理参数
- 更真实的材质表现


### 5.4 光线追踪管线


| 着色器 | 功能 |
| --- | --- |
| Ray Generation Shader | 生成初始光线 |
| Intersection Shader | 自定义求交逻辑 |
| Any Hit Shader | 任意命中处理（透明等） |
| Closest Hit Shader | 最近命中处理 |
| Miss Shader | 光线未命中时处理 |
