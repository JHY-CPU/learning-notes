# 片元着色器与光照模型

## 1. 核心理论

片元着色器（Fragment Shader / Pixel Shader）是渲染管线中决定每个像素最终颜色的阶段。对光栅化产生的每个片元（潜在像素）独立执行，通过光照计算、纹理采样等操作输出最终颜色。

### 1.1 光照的物理基础

光与表面的交互可以用**双向反射分布函数（BRDF）**来描述：

```
BRDF: f_r(ω_i, ω_o, n) = dL_o(ω_o) / dE_i(ω_i)
其中：
  ω_i = 入射光方向
  ω_o = 出射光方向（观察方向）
  n = 表面法线
  dL_o = 反射辐射亮度
  dE_i = 入射辐照度

渲染方程（简化版）：
L_o(ω_o) = ∫ f_r(ω_i, ω_o) × L_i(ω_i) × cos(θ_i) × dω_i
其中 L_i(ω_i) 是来自ω_i方向的入射光，cos(θ_i) = n·ω_i
```

## 2. Lambert漫反射模型

### 2.1 数学原理

Lambert余弦定律：理想漫反射表面的反射光强度与入射光和法线夹角的余弦成正比。

```
漫反射公式：
I_diffuse = k_d × I_light × max(N · L, 0)

其中：
  k_d = 漫反射系数（材质颜色）
  I_light = 光源颜色和强度
  N = 表面法线（归一化）
  L = 光源方向（归一化，从表面指向光源）
  N · L = dot(N, L) = cos(θ)
  max()确保背面不被照亮
```

### 2.2 GLSL实现

```glsl
#version 330 core

in vec3 v_Normal;
in vec3 v_WorldPos;
in vec2 v_TexCoord;

uniform vec3 u_LightDir;      // 光源方向（从表面指向光源）
uniform vec3 u_LightColor;    // 光源颜色
uniform vec3 u_BaseColor;     // 材质基础颜色

out vec4 fragColor;

void main()
{
    vec3 N = normalize(v_Normal);
    vec3 L = normalize(u_LightDir);

    // Lambert漫反射
    float NdotL = max(dot(N, L), 0.0);
    vec3 diffuse = u_BaseColor * u_LightColor * NdotL;

    // 环境光（简化模拟间接光照）
    vec3 ambient = u_BaseColor * 0.1;

    fragColor = vec4(ambient + diffuse, 1.0);
}
```

### 2.3 HLSL实现（Unity URP）

```hlsl
half4 LambertFrag(Varyings input) : SV_Target
{
    half4 texColor = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, input.uv);
    half3 baseColor = texColor.rgb * _BaseColor.rgb;

    Light mainLight = GetMainLight();
    half3 N = normalize(input.normalWS);
    half3 L = mainLight.direction;

    // Lambert
    half NdotL = saturate(dot(N, L));
    half3 diffuse = baseColor * mainLight.color * NdotL;

    // 环境光
    half3 ambient = SampleSH(N) * baseColor;

    return half4(ambient + diffuse, texColor.a);
}
```

## 3. Phong光照模型

### 3.1 数学原理

Phong模型将光照分解为三个分量：环境光（Ambient）、漫反射（Diffuse）、镜面反射（Specular）。

```
Phong公式：
I = I_ambient + I_diffuse + I_specular

I_ambient = k_a × I_a
I_diffuse = k_d × I_light × max(N · L, 0)
I_specular = k_s × I_light × max(R · V, 0)^shininess

其中：
  k_a, k_d, k_s = 环境光/漫反射/镜面反射系数
  R = reflect(-L, N) = 反射方向 = 2×(N·L)×N - L
  V = 观察方向（从表面指向相机）
  shininess = 高光指数（越大高光越小越锐利）
```

### 3.2 完整GLSL实现

```glsl
#version 330 core

struct Material {
    vec3 ambient;
    vec3 diffuse;
    vec3 specular;
    float shininess;
};

struct Light {
    vec3 position;      // 点光源位置
    vec3 color;         // 光源颜色
    float intensity;    // 光源强度
    float constant;     // 衰减常数项
    float linear;       // 衰减线性项
    float quadratic;    // 衰减二次项
};

uniform Material u_Material;
uniform Light u_Light;
uniform vec3 u_CameraPos;

in vec3 v_Normal;
in vec3 v_WorldPos;
in vec2 v_TexCoord;

out vec4 fragColor;

// 光源衰减计算
float CalcAttenuation(float distance, float constant, float linear, float quadratic)
{
    return 1.0 / (constant + linear * distance + quadratic * distance * distance);
}

vec3 CalcPhong(vec3 normal, vec3 fragPos, vec3 viewDir)
{
    vec3 lightDir = normalize(u_Light.position - fragPos);
    float distance = length(u_Light.position - fragPos);
    float attenuation = CalcAttenuation(distance, u_Light.constant, u_Light.linear, u_Light.quadratic);

    // 环境光
    vec3 ambient = u_Material.ambient * u_Light.color * 0.1;

    // 漫反射
    float diff = max(dot(normal, lightDir), 0.0);
    vec3 diffuse = u_Material.diffuse * u_Light.color * diff * attenuation * u_Light.intensity;

    // 镜面反射（Phong）
    vec3 reflectDir = reflect(-lightDir, normal);
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), u_Material.shininess);
    vec3 specular = u_Material.specular * u_Light.color * spec * attenuation * u_Light.intensity;

    return ambient + diffuse + specular;
}

void main()
{
    vec3 N = normalize(v_Normal);
    vec3 V = normalize(u_CameraPos - v_WorldPos);

    vec3 result = CalcPhong(N, v_WorldPos, V);

    // 采样纹理
    vec3 texColor = texture(u_DiffuseMap, v_TexCoord).rgb;
    result *= texColor;

    fragColor = vec4(result, 1.0);
}
```

## 4. Blinn-Phong光照模型

### 4.1 数学原理

Blinn-Phong用**半程向量（Halfway Vector）**替代反射向量，计算更快且高光更自然。

```
Blinn-Phong公式：
I_specular = k_s × I_light × max(N · H, 0)^shininess

其中：
  H = normalize(L + V) = 半程向量（光方向和视线方向的中间方向）

与Phong的区别：
  Phong:    spec = pow(max(R · V, 0), n)      // R = reflect(-L, N)
  Blinn:    spec = pow(max(N · H, 0), n)      // H = normalize(L + V)

  当视线与反射方向一致时 Phong 和 Blinn 等价
  但 Blinn 在掠射角度下高光衰减更平滑，更接近真实物理
  OpenGL默认使用 Blinn-Phong
```

### 4.2 GLSL实现

```glsl
vec3 CalcBlinnPhong(vec3 normal, vec3 fragPos, vec3 viewDir)
{
    vec3 lightDir = normalize(u_Light.position - fragPos);

    // 环境光
    vec3 ambient = u_Material.ambient * u_Light.color * 0.1;

    // 漫反射
    float diff = max(dot(normal, lightDir), 0.0);
    vec3 diffuse = u_Material.diffuse * u_Light.color * diff;

    // 高光（Blinn-Phong）
    vec3 halfDir = normalize(lightDir + viewDir);
    float spec = pow(max(dot(normal, halfDir), 0.0), u_Material.shininess);
    vec3 specular = u_Material.specular * u_Light.color * spec;

    return ambient + diffuse + specular;
}
```

### 4.3 HLSL完整实现（Unity URP）

```hlsl
half4 BlinnPhongFrag(Varyings input) : SV_Target
{
    half4 texColor = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, input.uv);
    half3 baseColor = texColor.rgb * _BaseColor.rgb;

    Light mainLight = GetMainLight();
    half3 N = normalize(input.normalWS);
    half3 L = mainLight.direction;
    half3 V = normalize(_WorldSpaceCameraPos - input.positionWS);
    half3 H = normalize(L + V);

    // 环境光
    half3 ambient = SampleSH(N) * baseColor;

    // 漫反射
    half NdotL = saturate(dot(N, L));
    half3 diffuse = baseColor * mainLight.color * NdotL;

    // Blinn-Phong高光
    half NdotH = saturate(dot(N, H));
    half spec = pow(NdotH, _Gloss);
    half3 specular = mainLight.color * _SpecColor.rgb * spec;

    // 阴影衰减
    half shadow = mainLight.shadowAttenuation;

    half3 finalColor = ambient + (diffuse + specular) * shadow;
    return half4(finalColor, texColor.a);
}
```

## 5. PBR（基于物理的渲染）

### 5.1 理论基础

PBR基于两个核心原则：
1. **能量守恒**：反射光的总能量不超过入射光能量
2. **微表面理论**：微观层面的表面由大量微小的完美镜面组成

```
Cook-Torrance BRDF:
f_r = k_d × (c/π) + k_s × (D × F × G) / (4 × (N·V) × (N·L))

其中：
  D = 法线分布函数（微表面法线的统计分布）
  F = 菲涅尔项（不同角度的反射率变化）
  G = 几何遮蔽函数（微表面间的自遮挡）
  c = 反照率（albedo）
  k_d = 1 - k_s（能量守恒）
```

### 5.2 GGX法线分布函数（D项）

```
D_GGX(N, H, α) = α² / (π × ((N·H)² × (α² - 1) + 1)²)

其中 α = roughness²（粗糙度的平方，使参数变化更线性）
```

```glsl
float DistributionGGX(vec3 N, vec3 H, float roughness)
{
    float a = roughness * roughness;
    float a2 = a * a;
    float NdotH = max(dot(N, H), 0.0);
    float NdotH2 = NdotH * NdotH;

    float denom = NdotH2 * (a2 - 1.0) + 1.0;
    float denom2 = denom * denom;

    return a2 / (3.14159265 * denom2 + 0.0001);
}
```

### 5.3 Schlick-GGX几何遮蔽函数（G项）

```
G_SchlickGGX(N, V, k) = (N·V) / ((N·V) × (1-k) + k)

其中 k = (roughness + 1)² / 8 （用于直接光照）
  或 k = roughness² / 2 （用于IBL）

G_Smith(N, V, L, k) = G_SchlickGGX(N, V, k) × G_SchlickGGX(N, L, k)
```

```glsl
float GeometrySchlickGGX(float NdotV, float roughness)
{
    float r = roughness + 1.0;
    float k = (r * r) / 8.0;
    return NdotV / (NdotV * (1.0 - k) + k);
}

float GeometrySmith(vec3 N, vec3 V, vec3 L, float roughness)
{
    float NdotV = max(dot(N, V), 0.0);
    float NdotL = max(dot(N, L), 0.0);
    float ggx1 = GeometrySchlickGGX(NdotV, roughness);
    float ggx2 = GeometrySchlickGGX(NdotL, roughness);
    return ggx1 * ggx2;
}
```

### 5.4 Schlick菲涅尔近似（F项）

```
F_Schlick(H, V, F0) = F0 + (1 - F0) × (1 - (H·V))⁵

其中 F0 = 基础反射率
  金属：F0 = albedo（反照率即反射率）
  非金属：F0 = 0.04（约4%的反射率）
  通过 metallic 参数在两者之间插值：
  F0 = mix(vec3(0.04), albedo, metallic)
```

```glsl
vec3 fresnelSchlick(float cosTheta, vec3 F0)
{
    return F0 + (1.0 - F0) * pow(clamp(1.0 - cosTheta, 0.0, 1.0), 5.0);
}
```

### 5.5 完整PBR着色器（GLSL）

```glsl
#version 330 core

#define PI 3.14159265359
#define MAX_LIGHTS 4

struct PBRLight {
    vec3 position;
    vec3 color;
    float intensity;
};

uniform sampler2D u_AlbedoMap;
uniform sampler2D u_NormalMap;
uniform sampler2D u_MetallicRoughnessMap; // R=metallic, G=roughness
uniform sampler2D u_AOMap;

uniform PBRLight u_Lights[MAX_LIGHTS];
uniform int u_LightCount;
uniform vec3 u_CameraPos;

in vec3 v_WorldPos;
in vec3 v_Normal;
in vec2 v_TexCoord;
in mat3 v_TBN;

out vec4 fragColor;

// D项 - GGX法线分布
float DistributionGGX(vec3 N, vec3 H, float roughness) {
    float a = roughness * roughness;
    float a2 = a * a;
    float NdotH = max(dot(N, H), 0.0);
    float denom = NdotH * NdotH * (a2 - 1.0) + 1.0;
    return a2 / (PI * denom * denom + 0.0001);
}

// G项 - Schlick-GGX
float GeometrySchlickGGX(float NdotV, float k) {
    return NdotV / (NdotV * (1.0 - k) + k);
}
float GeometrySmith(vec3 N, vec3 V, vec3 L, float roughness) {
    float r = roughness + 1.0;
    float k = (r * r) / 8.0;
    float NdotV = max(dot(N, V), 0.0);
    float NdotL = max(dot(N, L), 0.0);
    return GeometrySchlickGGX(NdotV, k) * GeometrySchlickGGX(NdotL, k);
}

// F项 - Schlick近似
vec3 fresnelSchlick(float cosTheta, vec3 F0) {
    return F0 + (1.0 - F0) * pow(clamp(1.0 - cosTheta, 0.0, 1.0), 5.0);
}

// 法线贴图采样
vec3 GetNormal() {
    vec3 normalTS = texture(u_NormalMap, v_TexCoord).rgb * 2.0 - 1.0;
    return normalize(v_TBN * normalTS);
}

void main()
{
    // 采样PBR贴图
    vec3 albedo = pow(texture(u_AlbedoMap, v_TexCoord).rgb, vec3(2.2)); // sRGB→线性
    float metallic = texture(u_MetallicRoughnessMap, v_TexCoord).r;
    float roughness = texture(u_MetallicRoughnessMap, v_TexCoord).g;
    float ao = texture(u_AOMap, v_TexCoord).r;

    vec3 N = GetNormal();
    vec3 V = normalize(u_CameraPos - v_WorldPos);

    // F0计算：非金属约0.04，金属用反照率
    vec3 F0 = mix(vec3(0.04), albedo, metallic);

    vec3 Lo = vec3(0.0); // 出射光累加

    for (int i = 0; i < u_LightCount; i++)
    {
        vec3 L = normalize(u_Lights[i].position - v_WorldPos);
        vec3 H = normalize(V + L);

        float dist = length(u_Lights[i].position - v_WorldPos);
        float attenuation = 1.0 / (dist * dist); // 物理衰减
        vec3 radiance = u_Lights[i].color * u_Lights[i].intensity * attenuation;

        // Cook-Torrance BRDF
        float D = DistributionGGX(N, H, roughness);
        float G = GeometrySmith(N, V, L, roughness);
        vec3 F = fresnelSchlick(max(dot(H, V), 0.0), F0);

        vec3 numerator = D * G * F;
        float denominator = 4.0 * max(dot(N, V), 0.0) * max(dot(N, L), 0.0) + 0.0001;
        vec3 specular = numerator / denominator;

        // 能量守恒
        vec3 kS = F;                    // 镜面反射比例
        vec3 kD = vec3(1.0) - kS;       // 漫反射比例
        kD *= 1.0 - metallic;            // 金属没有漫反射

        float NdotL = max(dot(N, L), 0.0);
        Lo += (kD * albedo / PI + specular) * radiance * NdotL;
    }

    // 简化环境光（实际应用中使用IBL）
    vec3 ambient = vec3(0.03) * albedo * ao;
    vec3 color = ambient + Lo;

    // 色调映射（Reinhard）
    color = color / (color + vec3(1.0));
    // Gamma校正
    color = pow(color, vec3(1.0 / 2.2));

    fragColor = vec4(color, 1.0);
}
```

## 6. 法线贴图与TBN矩阵

### 6.1 TBN矩阵构建

```glsl
// 顶点着色器中构建TBN矩阵
layout(location = 0) in vec3 a_Position;
layout(location = 1) in vec3 a_Normal;
layout(location = 2) in vec2 a_TexCoord;
layout(location = 3) in vec4 a_Tangent; // xyz=切线方向, w=手性

out mat3 v_TBN;

void main()
{
    vec3 T = normalize(mat3(u_Model) * a_Tangent.xyz);
    vec3 N = normalize(mat3(u_NormalMatrix) * a_Normal);
    // 重正化（确保切线垂直于法线）
    T = normalize(T - dot(T, N) * N);
    vec3 B = cross(N, T) * a_Tangent.w; // w控制手性

    v_TBN = mat3(T, B, N); // 切线空间 → 世界空间
}
```

### 6.2 法线贴图采样

```glsl
vec3 GetNormalFromMap()
{
    // 法线贴图存储在[0,1]范围，需要映射到[-1,1]
    vec3 normalTS = texture(u_NormalMap, v_TexCoord).rgb * 2.0 - 1.0;
    // 从切线空间变换到世界空间
    return normalize(v_TBN * normalTS);
}
```

## 7. 菲涅尔效应

菲涅尔效应描述了物体在不同视角下的反射率变化：正面反射率低，掠射角度反射率高。

```glsl
// Schlick菲涅尔近似
vec3 FresnelSchlick(float cosTheta, vec3 F0)
{
    return F0 + (1.0 - F0) * pow(1.0 - cosTheta, 5.0);
}

// 应用示例：边缘发光效果
float rim = 1.0 - saturate(dot(N, V));
vec3 fresnel = FresnelSchlick(rim, F0);
```

## 9. 性能提示

- Phong/Blinn-Phong：每次片元约20-30条ALU指令，移动端友好
- PBR Cook-Torrance：每次片元约100-150条ALU指令，适合PC/主机
- 使用`half`精度代替`float`可节省移动端约30%的着色器时间
- `pow()`是最昂贵的标量函数，减少使用次数或用查找表（LUT）替代
- 法线贴图采样（纹理读取）比ALU运算更慢，纹理缓存未命中是主要瓶颈
- 多光源循环应尽量展开或使用固定循环次数以避免动态分支

## 10. 常见问题与调试

**问题1：高光在背面出现**
- 添加`max(dot(N, L), 0.0)`检查
- 或使用`step(0.0, dot(N, L))`消除分支

**问题2：PBR材质看起来太暗或太亮**
- 检查纹理是否在sRGB空间（需要gamma到线性转换）
- 检查光源强度是否使用了物理单位

**问题3：法线贴图导致表面出现接缝**
- 检查模型切线空间是否一致
- 使用`dFdx/dFdy`在片元着色器中计算切线

**问题4：不同光照模型切换时颜色跳变**
- 实现过渡混合：`final = lerp(phong, pbr, blendFactor)`

## 11. 实际使用案例

- Unity URP默认使用简化的Blinn-Phong变体（Lambert + 高光）
- Unity HDRP使用完整PBR管线（GGX BRDF + IBL + SSS）
- 《使命召唤》系列从Phong切换到GGX BRDF后材质质量显著提升
- 《王者荣耀》移动端使用简化Phong + 低精度half计算保证60fps
- 《荒野大镖客2》使用多层PBR叠加（皮肤SSS + 布料Sheen + 毛发Marschner）
- Three.js的MeshStandardMaterial使用WebGL实现完整PBR
