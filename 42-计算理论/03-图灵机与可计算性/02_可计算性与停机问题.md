# 可计算性与停机问题


## 一、可判定性 (Decidability)


**可判定语言：**存在图灵机 M，对所有输入 w：


- w ∈ L 时 M
   **停机并接受**
- w ∉ L 时 M
   **停机并拒绝**


关键：M 必须对所有输入**都停机**（不能死循环）。


### 相关概念区分


| 概念 | 定义 | TM行为 |
| --- | --- | --- |
| **可判定** (Decidable) | 存在总停机的TM识别 | 对所有输入停机 |
| **图灵可识别** (Turing-recognizable) | 存在TM识别（半判定） | w∈L时停机接受，w∉L时可能死循环 |
| **图灵不可识别** | 补集不是图灵可识别的 | — |


关系：可判定 ⊂ 图灵可识别 ⊂ 递归可枚举语言


## 二、停机问题 (Halting Problem)


> **Theorem:** **定理：**停机问题是**不可判定的**。
>
>
> 形式化：语言 `A~TM~ = {⟨M, w⟩ | M是图灵机且M在输入w上停机}` 是图灵可识别但不可判定的。


### 对角化证明（反证法）


**假设**存在TM H 判定停机问题，即：


- H(⟨M, w⟩) = accept，若 M 在 w 上停机
- H(⟨M, w⟩) = reject，若 M 在 w 上不停机


**构造**一个新的TM D：


```
D(⟨M⟩):
    运行 H(⟨M, ⟨M⟩⟩)    # 问H：M以自身编码为输入会停机吗？
    if H 返回 accept:
        进入死循环        # 故意不停机
    else:
        停机并接受        # 故意停机
```


**矛盾：**考虑 D(⟨D⟩) 的行为：


- 若 D(⟨D⟩) 停机 → H 返回 accept → D 死循环 →
   **矛盾！**
- 若 D(⟨D⟩) 不停机 → H 返回 reject → D 停机 →
   **矛盾！**


因此假设不成立，**H 不存在**。∎


## 三、Rice定理


> **Theorem:** **Rice定理：**对图灵可识别语言的任何**非平凡**语义性质，判断一个TM的语言是否具有该性质是**不可判定的**。


**非平凡：**不是所有图灵可识别语言都具有，也不是都不具有。


### Rice定理的应用


| 问题 | 是否可判定 | Rice定理适用？ |
| --- | --- | --- |
| M的语言是否为空？ | 不可判定 | 是（空语言是语义性质） |
| M的语言是否有限？ | 不可判定 | 是 |
| M的语言是否包含某串w？ | 不可判定 | 是 |
| M是否在100步内停机？ | 可判定 | 否（语法性质，非语义） |
| M是否在任何输入上死循环？ | 不可判定 | 是 |


## 四、归约 (Reduction)


**归约的核心思想：**如果问题 A 可归约到问题 B，且 A 不可判定，则 B 也不可判定。


`A ≤~m~ B` 表示：存在可计算函数 f，使得 x ∈ A ⇔ f(x) ∈ B。


### 经典归约示例


```
ATM (停机问题)
↓ 归约
HALT (停机判定问题)
↓ 归约
ETM (空语言问题)
↓ 归约
ALLTM (全语言问题)
↓ 归约
EQTM (等价性问题)
```


以上所有问题都是**不可判定**的。

## 六、康托尔对角化方法


停机问题证明使用的核心思想来自康托尔证明实数不可数的方法。


### 实数不可数证明


> **Example:** **假设** (0,1) 中的实数可数，排列为 r₁, r₂, r₃, ...
>
>
> 构造新的实数 d，其第 i 位小数 ≠ rᵢ 的第 i 位小数。
>
>
> d 不在列表中，**矛盾！**故实数不可数。∎


同样的"对角线"技巧被用于停机问题：D 的行为总是与假设中 H 的判断相反。
