图灵机 (Turing Machine)

一、图灵机模型

图灵机是阿兰·图灵在1936年提出的计算模型，由以下组件构成：

物理组件

  • 无限长纸带 (Tape) — 分为一个个格子，每个格子存放一个符号。纸带向右无限延伸（或双向无限）
  • 读写头 (Head) — 每步读取当前格子的符号，可以写入新符号，并向左或右移动一格
  • 有限状态控制器 — 有限状态集合，控制转移逻辑
  • 转移函数 — 基于当前状态和读入符号决定下一步操作

二、形式化定义

图灵机是七元组 M = (Q, Σ, Γ, δ, q₀, qaccept, qreject)：

  • Q — 有穷状态集合
  • Σ — 有穷输入字母表（不含空白符）
  • Γ — 有穷纸带字母表（Σ ⊂ Γ，包含空白符 _）
  • δ — 转移函数：δ: Q × Γ → Q × Γ × {L, R}
  • q₀ ∈ Q — 初始状态
  • qaccept ∈ Q — 接受状态
  • qreject ∈ Q — 拒绝状态（qaccept ≠ qreject）

转移函数 δ(q, X) = (p, Y, D) 的含义

  • 若当前状态为 q，读写头下符号为 X
  • 则：转移到状态 p，在格子中写入 Y，读写头向 D 方向移动一格（L=左，R=右）

丘奇-图灵论题：任何"有效可计算"的函数都可以由图灵机计算。即图灵机是计算能力的上限。

六、通用图灵机 (UTM)

通用图灵机是一台可以模拟任何图灵机的图灵机：

  • 输入：图灵机 M 的编码 ⟨M⟩ + M 的输入 w
  • 行为：模拟 M 在 w 上的运行
  • 相当于现代计算机的冯·诺依曼架构（程序即数据）
# 通用图灵机概念
def universal_turing_machine(machine_description, input_string):
    """
    machine_description: 图灵机M的形式化编码
    input_string: M的输入w
    返回: M在w上的计算结果
    """
    tape = init_tape(input_string)
    state = machine_description['start']
    head = 0

    while state not in ('accept', 'reject'):
        symbol = tape[head]
        transition = machine_description['delta'][(state, symbol)]
        state, write_symbol, direction = transition
        tape[head] = write_symbol
        head += 1 if direction == 'R' else -1

    return state == 'accept'
