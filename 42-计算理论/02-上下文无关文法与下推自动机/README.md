# 02-上下文无关文法与下推自动机

## 1. 上下文无关文法（CFG）的形式定义

**定义**：上下文无关文法是四元组 $G = (V, \Sigma, R, S)$，其中：

- $V$：非终结符号的有穷集合（变量）
- $\Sigma$：终结符号的有穷集合（$V \cap \Sigma = \emptyset$）
- $R$：产生式规则的有穷集合，每条规则形如 $A \to \alpha$，其中 $A \in V$，$\alpha \in (V \cup \Sigma)^*$
- $S \in V$：起始符号

**例**：$G = (\{S\}, \{a, b\}, \{S \to aSb, S \to \varepsilon\}, S)$，生成语言 $\{a^n b^n \mid n \geq 0\}$。

**"上下文无关"的含义**：产生式 $A \to \alpha$ 中 $A$ 的替换不依赖于 $A$ 的上下文，无论 $A$ 出现在什么位置都可以替换。

---

## 2. 推导与语法分析树

**推导（Derivation）**：若 $G$ 中有产生式 $A \to \gamma$，则 $\alpha A \beta \Rightarrow_G \alpha \gamma \beta$（一步推导）。用 $\Rightarrow_G^*$ 表示零步或多步推导。

**最左推导**：每一步替换最左边的非终结符。记为 $\Rightarrow_{lm}$。

**最右推导**（规范推导）：每一步替换最右边的非终结符。记为 $\Rightarrow_{rm}$。

**句型**：若 $S \Rightarrow_G^* \alpha$，则 $\alpha$ 是 $G$ 的一个句型。句型由终结符和非终结符组成。

**语法分析树（Parse Tree）**：

- 根节点标记为起始符号 $S$
- 内部节点标记为非终结符
- 叶节点标记为终结符或 $\varepsilon$
- 从左到右读出叶节点得到推导出的字符串
- 树的高度反映了推导的效率

**定理**：若 $G$ 是 CFG 且 $S \Rightarrow_G^* w$（$w \in \Sigma^*$），则存在以 $S$ 为根、$w$ 为叶的语法分析树。

---

## 3. 歧义文法与消歧方法

**定义**：CFG $G$ 对于字符串 $w$ 是歧义的，如果存在 $w$ 的两棵不同的语法分析树（等价地，两个不同的最左推导）。

**例**：算术表达式文法 $E \to E+E \mid E*E \mid (E) \mid id$ 对 $id + id * id$ 有两棵分析树，取决于 $+$ 和 $*$ 的优先级。

**固有歧义性**：语言 $L$ 是固有歧义的，如果每个生成 $L$ 的 CFG 都是歧义的。例：$\{a^i b^j c^k \mid i = j \text{ 或 } j = k\}$ 是固有歧义的。

**消歧方法**：

- **引入优先级**：将文法改写为层次化结构
  ```
  E → E + T | T
  T → T * F | F
  F → (E) | id
  ```
- **引入结合性**：左递归表示左结合，右递归表示右结合
- **使用优先级和结合性声明**（在 parser generator 中）

## 4. Chomsky 文法分类

**Chomsky 层级**（按限制从强到弱）：

| 类型 | 名称 | 产生式限制 | 识别器 |
|------|------|-----------|--------|
| Type 3 | 正则文法 | $A \to aB$ 或 $A \to a$ 或 $A \to \varepsilon$ | DFA/NFA |
| Type 2 | 上下文无关文法 | $A \to \alpha$（$A \in V$） | PDA |
| Type 1 | 上下文有关文法 | $\alpha A \beta \to \alpha \gamma \beta$，$|\gamma| \geq 1$ | LBA |
| Type 0 | 无限制文法 | $\alpha \to \beta$（$|\alpha| \geq 1$） | 图灵机 |

**包含关系**：Type 3 $\subset$ Type 2 $\subset$ Type 1 $\subset$ Type 0。每一类都是严格包含——存在语言属于外层但不属于内层。

## 5. CFG 的化简

### 5.1 无用符号消除

**无用符号**：不出现在任何从 $S$ 推导出终结串的推导中的符号。无用符号分为两类：

1. **不可生成（non-generating）**：不能推导出终结串
2. **不可到达（non-reachable）**：从 $S$ 出发无法到达

**算法**：

1. 标记所有可生成的符号（从终结符开始，$A \to \alpha$ 且 $\alpha$ 中所有符号已标记则标记 $A$）
2. 标记所有可达的符号（从 $S$ 开始沿产生式搜索）
3. 删除未被两步都标记的符号及其相关产生式

**注意**：必须先消除不可生成符号，再消除不可到达符号。

### 5.2 ε-产生式消除

**ε-产生式**：形如 $A \to \varepsilon$ 的产生式。

**可空符号**：$A$ 是可空的，如果 $A \Rightarrow_G^* \varepsilon$。

**算法**：

1. 找出所有可空的非终结符
2. 对每个产生式 $A \to X_1 X_2 \cdots X_k$，对 $\{X_1, \ldots, X_k\}$ 的每个可空子集生成新的产生式（替换可空符号为 $\varepsilon$）
3. 删除所有 $\varepsilon$-产生式（除非 $S$ 本身可空且 $S$ 不在任何产生式右侧）

### 5.3 单一产生式消除

**单一产生式**：形如 $A \to B$ 的产生式（$B \in V$）。

**算法**：

1. 对每对非终结符 $A, B$，若 $A \Rightarrow^* B$ 且 $B \to \gamma$ 非单一，则添加 $A \to \gamma$
2. 删除所有单一产生式

## 6. Chomsky 范式（CNF）与 Greibach 范式（GNF）

### Chomsky 范式（CNF）

**定义**：每个产生式形如 $A \to BC$ 或 $A \to a$（允许 $S \to \varepsilon$ 若 $\varepsilon \in L(G)$）。

**定理**：每个不含 $\varepsilon$ 的 CFL 都有等价的 CNF 文法。

**转换步骤**：

1. 消除无用符号、ε-产生式、单一产生式
2. 对 $A \to B_1 B_2 \cdots B_k$（$k \geq 3$），引入新非终结符拆分：$A \to B_1 C_1$，$C_1 \to B_2 C_2$，...
3. 对 $A \to \alpha B \beta$ 中终结符 $a$，引入 $N_a \to a$ 替换

**推论**：CNF 下的推导步数最多为 $2|w| - 1$（$|w| \geq 2$）。这给出了 CYK 算法的正确性基础。

### Greibach 范式（GNF）

**定义**：每个产生式形如 $A \to a\alpha$，其中 $a \in \Sigma$，$\alpha \in V^*$。

**定理**：每个不含 $\varepsilon$ 的 CFL 都有等价的 GNF 文法。

**意义**：GNF 下每步推导恰好消耗一个终结符，因此 $n$ 步推导产生长度为 $n$ 的字符串。

## 7. 下推自动机（PDA）的形式定义

**定义**：PDA 是七元组 $M = (Q, \Sigma, \Gamma, \delta, q_0, Z_0, F)$，其中：

- $Q$：有穷状态集
- $\Sigma$：输入字母表
- $\Gamma$：栈字母表（$\Sigma \subseteq \Gamma$）
- $\delta: Q \times (\Sigma \cup \{\varepsilon\}) \times \Gamma \to 2^{Q \times \Gamma^*}$：转移函数
- $q_0 \in Q$：初始状态
- $Z_0 \in \Gamma$：初始栈符号
- $F \subseteq Q$：接受状态集

**转移的含义**：$\delta(q, a, X) \ni (p, \gamma)$ 表示"在状态 $q$，读入输入 $a$，栈顶为 $X$ 时，可以转移到状态 $p$，并将 $X$ 替换为 $\gamma$（$\gamma = Y_1 Y_2 \cdots Y_k$，$Y_1$ 成为新的栈顶）"。

**两种接受方式**：

1. **接受状态接受**：读完输入后停在接受状态
2. **空栈接受**：读完输入后栈为空

**定理**：两种接受方式等价——对每个接受状态接受的 PDA，存在空栈接受的 PDA 接受相同语言，反之亦然。

---

## 8. PDA 与 CFG 的等价性

**定理**：语言 $L$ 是上下文无关语言，当且仅当存在 PDA $M$ 使得 $L = L(M)$。

### CFG $\to$ PDA（识别方式）

构造单状态 PDA，用栈模拟最左推导：

1. 初始时将 $S$ 压入栈
2. 若栈顶是非终结符 $A$，选择 $A \to \alpha$，将 $\alpha$ 压入栈（替换 $A$）
3. 若栈顶是终结符 $a$，读入输入中的 $a$，弹出 $a$
4. 当栈空时接受

**正确性**：PDA 接受 $w$ $\iff$ $S \Rightarrow_{lm}^* w$。

### PDA $\to$ CFG

构造等价文法：

1. 引入非终结符 $[p, X, q]$ 表示"从状态 $p$ 到状态 $q$，读入栈顶 $X$ 并将其消耗掉所推导出的串"
2. 对每个转移 $\delta(p, a, X) \ni (r, Y_1 Y_2 \cdots Y_k)$，添加产生式 $[p, X, q_k] \to a[r, Y_1, q_1][q_1, Y_2, q_2] \cdots [q_{k-1}, Y_k, q_k]$
3. 起始符号 $S = [q_0, Z_0, q]$ 对所有 $q \in F$

## 9. 上下文无关语言的泵引理

**定理**：若 $L$ 是 CFL，则存在泵长度 $p$，使得对任意 $w \in L$ 且 $|w| \geq p$，存在分解 $w = uvxyz$ 满足：

1. $|vxy| \leq p$
2. $|vy| > 0$（$v$ 和 $y$ 不同时为空）
3. 对所有 $i \geq 0$，$uv^i xy^i z \in L$

**证明思路**：利用 CNF 下的语法分析树。若 $|w| \geq 2^{p-1}$（$p$ 为非终结符数），则树中必有某个非终结符重复出现（鸽巢原理）。沿该重复路径"泵"即可。

**应用——证明语言不是 CFL**：

**例题**：证明 $L = \{a^n b^n c^n \mid n \geq 0\}$ 不是 CFL。

假设 $L$ 是 CFL，取 $w = a^p b^p c^p$。由 $|vxy| \leq p$，$vxy$ 不能同时包含 $a$、$b$、$c$ 三种符号。若 $v$ 或 $y$ 包含两种以上的符号，泵一次会导致交叉，$uv^2 xy^2 z$ 不是 $a^* b^* c^*$ 的形式。若 $vxy$ 只包含一种或两种符号，泵 $i = 0$ 后三种符号的个数不相等，$uv^0 xy^0 z \notin L$。矛盾。

**奥格登引理（Ogden's Lemma）**：更强的泵引理版本——可指定 $w$ 中的"标记"位置，保证泵的位置在标记区域内。用于证明某些泵引理无法处理的语言非 CFL。

## 10. CFL 的封闭性与判定性质

### 封闭性

CFL 对以下运算是**封闭**的：

| 运算 | 证明方法 |
|------|---------|
| 并集 $L_1 \cup L_2$ | 新起始符号 $S \to S_1 \mid S_2$ |
| 连接 $L_1 \cdot L_2$ | 新起始符号 $S \to S_1 S_2$ |
| Kleene 星 $L^*$ | 新起始符号 $S \to SS_1 \mid \varepsilon$ |
| 同态 $h(L)$ | 替换终结符 |
| 反转 $L^R$ | 反转每条产生式 |
| 与正则语言的交 | PDA 与 DFA 的乘积构造 |

**重要不封闭性**：CFL 对**交集**和**补集**不封闭。

**证明交集不封闭**：$L_1 = \{a^n b^n c^m\}$ 和 $L_2 = \{a^m b^n c^n\}$ 都是 CFL，但 $L_1 \cap L_2 = \{a^n b^n c^n\}$ 不是 CFL。

### 判定性质

| 问题 | 可判定性 |
|------|---------|
| $w \in L(G)$？ | **可判定**（CYK 算法，$O(n^3)$） |
| $L(G) = \emptyset$？ | **可判定** |
| $L(G)$ 无穷？ | **可判定**（检查泵引理中的循环） |
| $L(G_1) \subseteq L(G_2)$？ | **不可判定** |
| $L(G_1) = L(G_2)$？ | **不可判定** |
| $G$ 是否歧义？ | **不可判定** |
| $L(G) = \Sigma^*$？ | **不可判定** |
