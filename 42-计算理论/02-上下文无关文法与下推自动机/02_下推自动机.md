# 下推自动机 (PDA)


## 一、下推自动机的定义


PDA是带有**栈存储**的有限自动机，是识别上下文无关语言的计算模型。


PDA是一个七元组 `P = (Q, Σ, Γ, δ, q₀, Z₀, F)`：


- **Q**
   — 有穷状态集合
- **Σ**
   — 有穷输入字母表
- **Γ**
   — 有穷栈字母表
- **δ**
   — 转移函数：
   `δ: Q × (Σ ∪ {ε}) × Γ → P(Q × Γ*)`
- **q₀**
   — 初始状态
- **Z₀**
   — 初始栈底符号
- **F**
   — 接受状态集合


### PDA转移的含义


δ(q, a, X) 包含 (p, γ) 表示：


- 当前状态为 q
- 读入输入符号 a（或 ε 不读入）
- 栈顶符号为 X
- 转移到状态 p，弹出 X，压入串 γ


## 二、栈操作详解


### 压栈 (Push)

a
b
底

δ(q, x, b) → (p, ab)


弹出b，压入ab


### 弹栈 (Pop)

b
底

δ(q, x, a) → (p, ε)


弹出a，不压入


### 替换 (Replace)

c
b
底

δ(q, x, a) → (p, cb)


弹出a，压入cb


## 三、PDA的两种接受方式


| 接受方式 | 条件 | 特点 |
| --- | --- | --- |
| **终态接受** | 输入读完后，当前状态 ∈ F | 栈内容不限 |
| **空栈接受** | 输入读完后，栈为空 | 当前状态不限 |


> **Theorem:** **定理：**终态接受和空栈接受是等价的。对任意终态接受的PDA，可构造等价的空栈接受PDA，反之亦然。


## 四、经典示例：PDA接受 {0ⁿ1ⁿ | n ≥ 0}


### 设计思路


1. 读到 0 时，往栈里压入一个标记（如 X）
2. 读到 1 时，从栈中弹出一个标记
3. 输入读完且栈为空（只剩底符号），则接受


### 形式化描述


```
P = (Q, Σ, Γ, δ, q₀, Z₀, F)
Q = {q₀, q₁, q₂}
Σ = {0, 1}
Γ = {X, Z₀}
F = {q₂}
Z₀ = Z₀  # 栈底标记

# 转移函数
δ(q₀, 0, Z₀) = {(q₀, XZ₀)}   # 读0，栈底时压X
δ(q₀, 0, X)  = {(q₀, XX)}     # 读0，栈顶X时再压X
δ(q₀, 1, X)  = {(q₁, ε)}      # 读1，弹出X，进入q₁
δ(q₁, 1, X)  = {(q₁, ε)}      # 读1，继续弹出X
δ(q₁, ε, Z₀) = {(q₂, Z₀)}     # 栈空（只剩底），接受
```


### 输入 "0011" 的运行过程


| 步骤 | 状态 | 输入剩余 | 栈（顶在左） | 操作 |
| --- | --- | --- | --- | --- |
| 1 | q₀ | 0011 | Z₀ | 读0，压X |
| 2 | q₀ | 011 | XZ₀ | 读0，压X |
| 3 | q₀ | 11 | XXZ₀ | 读1，弹X→q₁ |
| 4 | q₁ | 1 | XZ₀ | 读1，弹X |
| 5 | q₁ | ε | Z₀ | ε-转移，接受 |
| 接受！ |


## 五、PDA与CFG的等价性


> **Theorem:** **定理：**一个语言是上下文无关的，当且仅当存在某个PDA接受它。


### CFG → PDA（模拟最左推导）


1. 栈顶是终结符且匹配输入：弹出，读入
2. 栈顶是变量 A：非确定地选择一条产生式 A→α，弹出A，压入α
3. 输入读完且栈只剩底符号：接受


### PDA → CFG（构造等价文法）


1. 变量 A
   ~pq~
   表示"从状态p（栈空）到状态q（栈空）的计算"
2. 将PDA的栈操作编码为产生式规则
3. 起始变量 = A
   ~q₀q~f~~
   （从初始状态到某个接受状态）


## 六、确定性PDA (DPDA)


DPDA要求对每组 (q, a, X)，δ(q, a, X) 至多包含**一个**元素，且若 δ(q, ε, X) 非空则 δ(q, a, X) 必须为空。


| 特性 | NFA vs DFA | PDA vs DPDA |
| --- | --- | --- |
| 等价性 | 等价 | **不等价** |
| 语言类 | 正则语言 | DPDA语言 ⊊ CFL |
| 例 | — | {wwᴿ}是DPDA语言，{ww}不是 |


注意 DPDA接受的语言类 = **确定性上下文无关语言** (DCFL)，是CFL的真子集。
