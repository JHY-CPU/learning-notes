空间复杂性与其他复杂性

一、空间复杂性类

空间复杂度衡量图灵机在计算过程中使用的纸带格子数。

复杂性类定义典型问题
L (LOGSPACE)确定性 O(log n) 空间图的连通性（有向/无向）
NL (NLOGSPACE)非确定性 O(log n) 空间有向图可达性
PSPACE确定性多项式空间TQBF（真量化布尔公式）
NPSPACE非确定性多项式空间—

Savitch定理：NPSPACE = PSPACE

即非确定性不节省空间！（对比：P vs NP 中不确定是否节省时间）

证明：用递归模拟非确定性选择，空间从 O(f(n)) 降到 O(f(n)²)。

二、L vs NL 问题

类比P vs NP：L = NL ？这是空间复杂度中的核心问题。

有向图可达性问题 (PATH)

问题：给定有向图 G 和节点 s, t，是否存在从 s 到 t 的路径？

  • 在 NL 中：猜测路径上的每个节点，用 O(log n) 空间存储当前节点编号
  • 在 P 中：BFS/DFS 可解（需要 O(n) 空间）
  • 是否在 L 中？未知！（这是 L vs NL 的核心）

Immerman-Szelepcsényi定理：NL = co-NL

在空间复杂度中，非确定性和其补集是等价的。这是令人惊讶的结果，因为 NP vs co-NP 仍是未解问题。

三、层次定理 (Hierarchy Theorems)

时间层次定理

若 f(n) ≥ n log n，则存在语言在 DTIME(f(n)) 中但不在 DTIME(f(n)/log f(n)) 中。

推论：P ⊊ EXPTIME（确定性指数时间）

空间层次定理

若 f(n) ≥ log n，则存在语言在 DSPACE(f(n)) 中但不在 DSPACE(f(n)-1) 中。

推论：L ⊊ PSPACE ⊊ EXPSPACE

完整的关系图

  L ⊆ NL ⊆ P ⊆ NP ⊆ PSPACE ⊆ EXPTIME ⊆ NEXPTIME ⊆ ...

  已证明的真包含关系：
  ✓ P ⊊ EXPTIME        （时间层次定理）
  ✓ PSPACE ⊊ EXPSPACE   （空间层次定理）
  ✓ L ⊊ PSPACE

  未解的关系：
  ? L = NL?
  ? P = NP?
  ? NP = PSPACE?
  ? P = PSPACE?

四、Savitch定理详解

核心思想：用二分搜索 + 递归模拟非确定性选择。

有向图可达性的递归判定

def reachable(G, s, t, n):
    """判断G中是否存在长度≤n的s到t的路径"""
    if n == 1:
        return s == t or (s,t) in G.edges

    for each node m in G:
        # 二分：是否存在s到m的长度≤n/2路径，且m到t的长度≤n/2路径？
        if reachable(G, s, m, n//2) and reachable(G, m, t, n//2):
            return True
    return False

# 空间分析：递归深度 O(log n)，每层存节点编号 O(log n)
# 总空间：O(log² n) — 多项式空间！

这个算法证明了 NL ⊆ DSPACE(log² n) ⊆ PSPACE。

六、概率复杂性

随机化算法的复杂性类

类定义关系
BPP有界误差多项式时间（误差 ≤ 1/3）P ⊆ BPP ⊆ P/poly，实际中≈P
RP单侧误差，NO永远正确P ⊆ RP ⊆ BPP
ZPP零误差期望多项式时间ZPP = RP ∩ co-RP
PP概率多项式时间（概率 > 1/2）NP ⊆ PP

示例：Miller-Rabin素性测试

判断数字 n 是否为素数：

  • 随机选择 a ∈ [2, n-2]
  • 检查费马小定理相关条件
  • 重复 k 次，错误概率 ≤ 4-k
  • 属于 BPP，实际中被认为等同于 P（2002年AKS证明素性在P中）

七、电路复杂性与P/poly

P/poly：可以用多项式大小布尔电路族判定的语言类。

  • P ⊆ P/poly（任何多项式时间算法可编译为电路）
  • P/poly 包含不可判定的语言！（因此 P/poly ≠ P）
  • 若 NP ⊆ P/poly，则多项式层次 (PH) 坍缩——广泛认为不可能

电路下界

证明某个问题需要超多项式大小的电路，是证明 P ≠ NP 的一种可能途径，但至今进展甚微。
