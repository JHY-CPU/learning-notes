P与NP问题

一、时间复杂性类

P类（多项式时间可判定）

P = { L | 存在确定性TM在 O(nᵏ) 时间内判定 L }

即：存在多项式时间算法的问题。

NP类（非确定性多项式时间可验证）

NP = { L | 存在非确定性TM在多项式时间内接受 L }

等价定义：存在多项式时间验证器——给定输入 w 和证书 c，能在多项式时间内验证 w ∈ L。

NP难与NP完全

  • NP-hard（NP难）：所有NP问题都可以多项式归约到它（可能不在NP中）
  • NP-complete（NP完全）：在NP中，且是NP-hard的（NP中最难的问题）

核心问题：P = NP ？

这是计算机科学中最重要的未解问题，千禧年七大难题之一（奖金100万美元）。

当前主流猜想：P ≠ NP（NP完全问题没有多项式时间算法）

三、Cook-Levin定理（SAT是NP完全的）

Cook-Levin定理 (1971)：布尔可满足性问题 (SAT) 是NP完全的。

这是历史上第一个被证明为NP完全的问题。

SAT问题

给定布尔公式 φ（合取范式），是否存在变量赋值使其为真？

例：φ = (x₁ ∨ x₂ ∨ ¬x₃) ∧ (¬x₁ ∨ x₂) ∧ (x₃ ∨ ¬x₂)

取 x₁=True, x₂=False, x₃=True：φ = (T∨F∨F) ∧ (F∨F) ∧ (T∨T) = T ∧ F ∧ T = False

取 x₁=True, x₂=True, x₃=True：φ = (T∨T∨F) ∧ (F∨T) ∧ (T∨F) = T ∧ T ∧ T = True（满足）

Cook定理证明思路

  1. 任取NP中的语言L，有非确定性多项式时间TM M判定L
  2. 对任意输入w，构造布尔公式 φw
  3. φw 可满足 ⇔ M接受w
  4. 构造过程只用多项式时间
  5. 故 L ≤P SAT，SAT是NP完全的

五、多项式时间归约

问题 A 多项式时间归约到问题 B（记作 A ≤P B）：

  • 存在多项式时间可计算的函数 f
  • 对所有 x：x ∈ A ⇔ f(x) ∈ B
  • 即：B的多项式算法可直接用于解A

归约链

SAT ≤ₚ 3-SAT ≤ₚ 团问题 ≤ₚ 顶点覆盖 ≤ₚ 哈密顿回路 ≤ₚ TSP

只要链中有一个是多项式可解的，链中所有问题都是多项式可解的！

证明NPC的方法

  1. 证明问题在NP中（给出多项式时间验证器）
  2. 选择一个已知的NPC问题A
  3. 构造A到该问题的多项式归约
  4. 由传递性，该问题也是NPC的

六、面对NP难问题的策略

策略描述适用场景
精确算法指数时间但保证最优解小规模实例
近似算法多项式时间，有近似比保证可接受近似解
启发式算法贪心、模拟退火、遗传算法大规模实际问题
固定参数可解f(k)·nᵏ 时间（k为参数）参数较小
随机化算法随机选择，期望最优快速近似
