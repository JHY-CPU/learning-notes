# 正则表达式与正则语言


## 一、正则表达式的定义


正则表达式是描述**正则语言**的代数记法，递归定义如下：


1. **基础情况：**

   - 空集
      `∅`
      是正则表达式（表示空语言）
   - 空串
      `ε`
      是正则表达式（表示 {ε}）
   - 字母表中每个符号
      `a`
      是正则表达式（表示 {a}）
2. **归纳步骤：**
   若 R 和 S 是正则表达式，则以下也是：

   - `(R|S)`
      — 并集（选择）
   - `(RS)`
      — 连接（concatenation）
   - `(R*)`
      — 克林闭包（零次或多次重复）


## 二、运算优先级与常见模式


| 运算符 | 含义 | 优先级 | 示例 |
| --- | --- | --- | --- |
| `*` | 克林闭包 | 最高 (3) | `a*` = {ε, a, aa, aaa, ...} |
| 连接 | 拼接 | 中 (2) | `ab` = {ab} |
| `\|` | 选择（或） | 最低 (1) | `a\|b` = {a, b} |


### 常见正则表达式模式


> **Example:** | 模式 | 描述 | 匹配示例 |
> | --- | --- | --- |
> | `(0\|1)*` | 所有二进制串 | ε, 0, 101, 1110 |
> | `0(0\|1)*1` | 以0开头以1结尾的串 | 01, 001, 01011 |
> | `(0\|1)*01(0\|1)*` | 包含子串"01" | 01, 1010, 0011 |
> | `(00\|11)*` | 偶数长度的相同对 | ε, 00, 1100, 0011 |
> | `1(0\|1)*0` | 以1开头0结尾 | 10, 110, 1010 |


## 三、正则语言的封闭性


正则语言在以下运算下**封闭**（即运算结果仍是正则语言）：


| 运算 | 说明 | 方法 |
| --- | --- | --- |
| **并集** ∪ | L₁ ∪ L₂ | NFA添加新初始状态，ε转移到两个旧初始状态 |
| **连接** ∘ | L₁ ∘ L₂ | NFA将L₁接受状态ε转移到L₂初始状态 |
| **克林闭包** * | L* | NFA添加新初始/接受状态，旧接受状态ε回旧初始 |
| **交集** ∩ | L₁ ∩ L₂ | 乘积构造（两个DFA同步运行） |
| **补集** | ¬L | DFA交换接受与非接受状态 |
| **反转** | Lᴿ | NFA反转所有转移箭头，初始/接受状态互换 |


## 五、正则语言、DFA、NFA的等价性


以下三个定义描述了**完全相同的语言类**：


- **DFA**
   识别的语言
- **NFA**
   识别的语言
- **正则表达式**
   描述的语言


### 转换关系


```
正则表达式                DFA               NFA
    ┌──────────────┐   Thompson   ┌─────┐ 子集构造  ┌─────┐
    │  R ───────────────────────►│ NFA │────────►│ DFA │
    │  (直接递归构造)│   构造法     └─────┘  法      └─────┘
    └──────────────┘                 ▲                 │
         ▲                           │                 │ 最小化
         │ 状态标记法                 │                 ▼
         │                      ┌────┴────┐      ┌─────────┐
         └──────────────────────│  DFA    │◄─────│最小 DFA │
                                └─────────┘      └─────────┘
```


Thompson构造法 将正则表达式转为NFA；子集构造法 将NFA转为DFA；Hopcroft算法 最小化DFA。


## 六、Myhill-Nerode定理


语言 L 的**等价关系** ~~L~：x ~~L~ y 当且仅当对所有串 z，xz ∈ L ⇔ yz ∈ L。


> **Theorem:** **定理：**L 是正则语言，当且仅当 ~~L~ 的等价类数目有穷。
>
>
> 等价类数目 = 识别 L 的最小DFA的状态数。


### 应用示例


> **Example:** 证明 L = {0ⁿ1ⁿ | n ≥ 0} 不是正则语言：
>
>
> 考虑串 0^i^ 和 0^j^（i ≠ j），取 z = 1^i^，则 0^i^1^i^ ∈ L 但 0^j^1^i^ ∉ L（j ≠ i）。
>
>
> 因此所有 0^i^ 形式的串属于不同等价类，等价类无穷多，L 非正则。∎


## 七、正则表达式转NFA (Python示例)


```
# 简化的正则表达式引擎框架
class NFA:
    """用Thompson构造法构建NFA"""
    def __init__(self, start, accept, transitions):
        self.start = start       # 初始状态
        self.accept = accept     # 接受状态
        self.delta = transitions # dict: (state, char) -> set of states

    def epsilon_closure(self, states):
        """计算 ε-闭包"""
        stack = list(states)
        closure = set(states)
        while stack:
            s = stack.pop()
            for nxt in self.delta.get((s, ''), set()):
                if nxt not in closure:
                    closure.add(nxt)
                    stack.append(nxt)
        return closure

    def accepts(self, input_str):
        """判断NFA是否接受输入串"""
        current = self.epsilon_closure({self.start})
        for ch in input_str:
            next_states = set()
            for s in current:
                next_states |= self.delta.get((s, ch), set())
            current = self.epsilon_closure(next_states)
        return bool(current & self.accept)

# 例：正则表达式 0(0|1)*1
# 对应NFA可接受 "01", "001", "01011" 等
# 不接受 "10", "00", "111"
```
