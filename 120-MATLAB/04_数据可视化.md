# 数据可视化

## 一、二维图形

### 1.1 基本 plot 函数

```matlab
% 基本折线图
x = linspace(0, 2*pi, 100);
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1);
xlabel('x');
ylabel('y');
title('正弦函数');
grid on;

% 多条曲线同时绘制
figure;
plot(x, y1, x, y2);
legend('sin(x)', 'cos(x)');

% 使用 hold on 叠加绘制
figure;
plot(x, y1, 'b-', 'LineWidth', 2);
hold on;
plot(x, y2, 'r--', 'LineWidth', 2);
hold off;
legend('sin(x)', 'cos(x)');
title('正弦与余弦');
```

### 1.2 线型、标记和颜色

```matlab
x = linspace(0, 2*pi, 20);
y = sin(x);

% 线型指定格式：plot(x, y, '颜色+线型+标记')
plot(x, y, 'r-o');        % 红色实线 + 圆圈标记
plot(x, y, 'g--s');       % 绿色虚线 + 方形标记
plot(x, y, 'b:^');        % 蓝色点线 + 三角标记
plot(x, y, 'm-.d');       % 品红点划线 + 菱形标记

% 颜色代码
% 'r' 红  'g' 绿  'b' 蓝  'c' 青
% 'm' 品红 'y' 黄  'k' 黑  'w' 白

% 标记代码
% 'o' 圆圈  's' 方形  'd' 菱形
% '^' 上三角 'v' 下三角 '>' 右三角 '<' 左三角
% 'p' 五角星 'h' 六角星  '+' 加号  'x' 叉号  '*' 星号

% RGB 自定义颜色
plot(x, y, 'Color', [0.2 0.6 0.8], 'LineWidth', 3);

% 使用 Name-Value 对设置属性
plot(x, y, ...
    'Color', [0.8 0.2 0.2], ...
    'LineStyle', '--', ...
    'LineWidth', 2, ...
    'Marker', 'o', ...
    'MarkerSize', 8, ...
    'MarkerFaceColor', 'y');
```

### 1.3 scatter 散点图

```matlab
% 基本散点图
x = randn(500, 1);
y = randn(500, 1);
figure;
scatter(x, y);

% 自定义散点图
figure;
scatter(x, y, 50, 'filled', 'MarkerFaceColor', [0.3 0.5 0.8]);

% 气泡图（大小和颜色编码）
sizes = rand(500, 1) * 100;       % 点的大小
colors = sqrt(x.^2 + y.^2);      % 点的颜色（距离原点的远近）
figure;
scatter(x, y, sizes, colors, 'filled');
colorbar;
title('气泡图：大小和颜色编码');
xlabel('X'); ylabel('Y');
colormap(jet);
```

### 1.4 bar 柱状图

```matlab
% 基本柱状图
data = [23 45 56 78 33 67];
figure;
bar(data);
set(gca, 'XTickLabel', {'一月','二月','三月','四月','五月','六月'});
ylabel('销售额（万元）');
title('月度销售');

% 分组柱状图
data2 = [23 34 45; 45 56 67; 56 67 78; 78 89 90];
figure;
bar(data2, 'grouped');
legend('产品A', '产品B', '产品C');
set(gca, 'XTickLabel', {'Q1','Q2','Q3','Q4'});

% 堆叠柱状图
figure;
bar(data2, 'stacked');
legend('产品A', '产品B', '产品C');

% 水平柱状图
figure;
barh(data);
```

### 1.5 其他二维图形

```matlab
% 饼图
sales = [35 25 20 15 5];
labels = {'华东', '华南', '华北', '西南', '西北'};
figure;
```

## 二、三维图形

### 2.1 三维曲线

```matlab
% 三维螺旋线
t = linspace(0, 10*pi, 500);
x = cos(t);
y = sin(t);
z = t;
figure;
plot3(x, y, z, 'b-', 'LineWidth', 2);
xlabel('X'); ylabel('Y'); zlabel('Z');
title('三维螺旋线');
grid on;
view(45, 30);  % 设置视角（方位角, 仰角）
```

### 2.2 三维曲面

```matlab
% 网格曲面
[X, Y] = meshgrid(-3:0.1:3);
Z = sin(X) .* cos(Y) .* exp(-0.1*(X.^2 + Y.^2));

figure;
surf(X, Y, Z);
xlabel('X'); ylabel('Y'); zlabel('Z');
title('三维曲面');
colorbar;
shading interp;  % 平滑着色

% 线框图
figure;
mesh(X, Y, Z);
title('线框图');

% 等高线图
figure;
contour(X, Y, Z, 20);     % 20条等高线
colorbar;
title('二维等高线');

figure;
contour3(X, Y, Z, 20);    % 三维等高线
title('三维等高线');

% 填充等高线
figure;
contourf(X, Y, Z, 15);
colorbar;
title('填充等高线');
```

### 2.3 三维可视化进阶

```matlab
% peaks 函数（MATLAB 内置示例）
[X, Y, Z] = peaks(50);

% surfc：曲面 + 等高线投影
figure;
surfc(X, Y, Z);
colormap(hot);

% meshc：线框 + 等高线投影
figure;
meshc(X, Y, Z);

% waterfall
figure;
waterfall(X, Y, Z);

% slice：体数据切片
[x, y, z] = meshgrid(-2:0.2:2);
v = x .* exp(-x.^2 - y.^2 - z.^2);
figure;
slice(x, y, z, v, [-1 0 1], 0, [-1 0.5 1]);
colorbar;
xlabel('X'); ylabel('Y'); zlabel('Z');
title('三维体数据切片');
```

## 三、subplot 子图

```matlab
% 基本子图布局
x = linspace(0, 2*pi, 100);

figure;
subplot(2, 2, 1);     % 2行2列，第1个
plot(x, sin(x));
title('sin(x)');

subplot(2, 2, 2);     % 第2个
plot(x, cos(x));
title('cos(x)');

subplot(2, 2, 3);     % 第3个
plot(x, tan(x));
ylim([-10 10]);
title('tan(x)');

subplot(2, 2, 4);     % 第4个
plot(x, exp(-x));
title('exp(-x)');

sgtitle('三角函数与指数函数');  % 总标题

% 不规则布局
figure;
subplot(2, 3, [1 2]);  % 占据第1、2个位置
plot(x, sin(x));
title('sin(x)');

subplot(2, 3, 3);
plot(x, cos(x));
title('cos(x)');

subplot(2, 3, [4 5 6]);  % 占据第4、5、6个位置
plot(x, sin(x).*cos(x));
title('sin(x)*cos(x)');
```

## 四、图形属性与美化

### 4.1 Figure 与 Axes 属性

```matlab
x = linspace(0, 2*pi, 100);
figure('Name', '我的图形', ...
       'NumberTitle', 'off', ...
       'Position', [100 100 800 600], ...  % [left, bottom, width, height]
       'Color', 'w');                      % 白色背景

plot(x, sin(x), 'b-', 'LineWidth', 2);

% 设置坐标轴属性
ax = gca;  % 获取当前坐标轴
ax.FontSize = 14;
ax.FontName = 'Times New Roman';
ax.XLabel.String = '时间 (s)';
ax.YLabel.String = '幅值';
ax.Title.String = '正弦波';
ax.XLim = [0 2*pi];
ax.YLim = [-1.5 1.5];
ax.XTick = 0:pi/2:2*pi;
ax.XTickLabel = {'0', '\pi/2', '\pi', '3\pi/2', '2\pi'};
ax.Box = 'on';
ax.LineWidth = 1.5;
ax.GridLineStyle = '--';
ax.GridColor = [0.5 0.5 0.5];
grid on;
```

### 4.2 字体与注释

```matlab
x = linspace(0, 2*pi, 100);
figure;
plot(x, sin(x), 'b-', 'LineWidth', 2);
hold on;
plot(x, cos(x), 'r--', 'LineWidth', 2);

% 图例
legend({'sin(x)', 'cos(x)'}, ...
    'Location', 'northeast', ...
    'FontSize', 12, ...
    'Box', 'on');

% 标注
text(pi, 0, '  (\pi, 0)', 'FontSize', 14);
text(pi/2, 1, '  峰值', 'FontSize', 12, 'Color', 'b');

% 箭头注释
annotation('arrow', [0.3 0.25], [0.7 0.6], 'LineWidth', 2);

% 双Y轴
yyaxis left;
plot(x, sin(x), 'b-');
ylabel('sin(x)');

yyaxis right;
plot(x, exp(x/5), 'r-');
ylabel('exp(x/5)');
xlabel('x');
title('双Y轴图');
```

## 五、图形导出

```matlab
% 保存为图像文件
figure;
peaks;

% 保存为多种格式
saveas(gcf, 'figure.png');
saveas(gcf, 'figure.fig');   % MATLAB 图形格式（可编辑）
saveas(gcf, 'figure.pdf');

% 使用 print 函数（更灵活）
print(gcf, 'figure_hires', '-dpng', '-r300');   % 300 DPI PNG
print(gcf, 'figure', '-dpdf');                    % PDF 矢量图
print(gcf, 'figure', '-depsc');                   % 彩色 EPS
print(gcf, 'figure', '-dsvg');                    % SVG 矢量图
print(gcf, 'figure', '-dtiff', '-r600');          % 600 DPI TIFF

% exportgraphics（R2020a+ 推荐）
fig = figure;
peaks;
exportgraphics(fig, 'figure.pdf', 'ContentType', 'vector');
exportgraphics(fig, 'figure.png', 'Resolution', 300);

% 复制到剪贴板（用于粘贴到 Word 等）
print(gcf, '-clipboard', '-dmeta');  % EMF 格式
```

## 六、动画

```matlab
% 使用 drawnow 创建动画
figure;
x = linspace(0, 2*pi, 200);
h = plot(x, sin(x), 'b-', 'LineWidth', 2);
axis([0 2*pi -1.5 1.5]);

for phase = linspace(0, 4*pi, 200)
    set(h, 'YData', sin(x + phase));
    drawnow;
    pause(0.02);
end

% 使用 getframe + movie
figure;
nFrames = 60;
for k = 1:nFrames
    [X, Y, Z] = peaks(100);
    Z = Z * sin(2*pi*k/nFrames);
    surf(X, Y, Z);
    shading interp;
    axis([-3 3 -3 3 -10 10]);
    view(k*6, 30);
    F(k) = getframe(gcf);
end
movie(F, 2, 30);  % 播放2次，每秒30帧

% 保存动画为 GIF
for k = 1:length(F)
    [imind, cm] = rgb2ind(frame2im(F(k)), 256);
    if k == 1
        imwrite(imind, cm, 'animation.gif', 'gif', 'Loopcount', inf, 'DelayTime', 0.03);
    else
        imwrite(imind, cm, 'animation.gif', 'gif', 'WriteMode', 'append', 'DelayTime', 0.03);
    end
end
```
