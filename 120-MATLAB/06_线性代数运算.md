# 线性代数运算

## 一、矩阵分解

### 1.1 LU 分解

```matlab
% LU 分解：A = L*U（下三角 × 上三角）
% 或带主元：P*A = L*U

A = [2 1 1; 4 3 3; 8 7 9];

% 基本 LU 分解
[L, U] = lu(A);
fprintf('L * U - A 的范数: %.2e\n', norm(L*U - A));

% 带置换矩阵
[L, U, P] = lu(A);
fprintf('P*A - L*U 的范数: %.2e\n', norm(P*A - L*U));

disp('L ='); disp(L);
disp('U ='); disp(U);
disp('P ='); disp(P);

% 应用：求解线性方程组 Ax = b
b = [1; 2; 3];
% 步骤1：求解 Ly = Pb（前代）
y = L \ (P * b);
% 步骤2：求解 Ux = y（回代）
x = U \ y;

fprintf('LU 分解求解: x = [%.6f; %.6f; %.6f]\n', x(1), x(2), x(3));
fprintf('直接求解: x = [%.6f; %.6f; %.6f]\n', (A\b));

% 大矩阵求解时 LU 分解更高效
% 需要求解多组 Ax=b 时，只需做一次 LU 分解
```

### 1.2 QR 分解

```matlab
% QR 分解：A = Q*R（正交矩阵 × 上三角矩阵）

A = [1 2 3; 4 5 6; 7 8 10];

% 经济型 QR 分解
[Q, R] = qr(A, 0);
fprintf('Q*R - A 的范数: %.2e\n', norm(Q*R - A));
fprintf('Q 的正交性检验: %.2e\n', norm(Q'*Q - eye(size(Q'*Q))));

% 完整 QR 分解
[Q_full, R_full] = qr(A);

% 应用1：求解最小二乘问题
% 超定系统 Ax = b，最小化 ||Ax - b||
A_over = rand(10, 3);
b_over = rand(10, 1);
[Q, R] = qr(A_over, 0);
x_qr = R \ (Q' * b_over);   % QR 分解解法
x_backslash = A_over \ b_over;  % 左除解法
fprintf('QR 解与左除解差异: %.2e\n', norm(x_qr - x_backslash));

% 应用2：Gram-Schmidt 正交化
v1 = [1; 1; 0]; v2 = [1; 0; 1]; v3 = [0; 1; 1];
V = [v1 v2 v3];
[Q_gs, R_gs] = qr(V, 0);
disp('正交化后的基:');
disp(Q_gs);
```

### 1.3 SVD 奇异值分解

```matlab
% SVD：A = U*S*V'（最通用的矩阵分解）
A = rand(5, 3);

% 基本 SVD
[U, S, V] = svd(A);

% 验证
fprintf('U*S*V'' - A 的范数: %.2e\n', norm(U*S*V' - A));

% 经济型 SVD（适合瘦高矩阵）
[U_e, S_e, V_e] = svd(A, 'econ');
fprintf('经济型 SVD 重建误差: %.2e\n', norm(U_e*S_e*V_e' - A));

% 奇异值
sigma = svd(A);  % 只返回奇异值（降序排列）
disp('奇异值:');
disp(sigma');

% 也可以从 S 矩阵提取
sigma_from_S = diag(S);

% 应用：图像压缩
% 用 rank-k 近似原矩阵
k = 2;  % 保留前 k 个奇异值
A_approx = U_e(:, 1:k) * S_e(1:k, 1:k) * V_e(:, 1:k)';
fprintf('rank-%d 近似误差: %.2e\n', k, norm(A - A_approx));

% 伪逆矩阵
A_pinv = pinv(A);   % 内部使用 SVD
A_pinv_manual = V * inv(S) * U';  % 手动计算（仅作理解用）

% 条件数
fprintf('条件数: %.4f\n', max(sigma)/min(sigma));
fprintf('matlab cond: %.4f\n', cond(A));
```

### 1.4 特征值分解

```matlab
% 特征值分解：A*V = V*D，D 为特征值对角矩阵
A = [4 -2; 1 1];

% 基本特征值求解
[V, D] = eig(A);
lambda = diag(D);

disp('特征值:');
disp(lambda');
disp('特征向量:');
disp(V);

% 验证 A*V = V*D
fprintf('A*V - V*D 的范数: %.2e\n', norm(A*V - V*D));

% 对称矩阵的特征值分解（更快、更稳定）
A_sym = [2 1; 1 3];
[V_sym, D_sym] = eig(A_sym);
fprintf('对称矩阵特征向量正交性: %.2e\n', norm(V_sym'*V_sym - eye(2)));

% 广义特征值：A*v = lambda*B*v
A_ge = [2 1; 1 3];
B_ge = [1 0; 0 2];
[V_ge, D_ge] = eig(A_ge, B_ge);
lambda_ge = diag(D_ge);
disp('广义特征值:');
disp(lambda_ge');

% 只计算特征值（不计算特征向量，更快）
eigenvalues = eig(A);

% 最大/最小特征值
lambda_max = eigs(A, 1, 'largestabs');
lambda_min = eigs(A, 1, 'smallestabs');

% 应用：PageRank 简化示例
% 链接矩阵的最大特征值对应的特征向量
L = [0 1/2 1/2; 1/3 0 1/3; 1/3 1/2 0];
[V_pagerank, D_pagerank] = eig(L');
[~, idx] = max(real(diag(D_pagerank)));
pagerank = real(V_pagerank(:, idx));
pagerank = pagerank / sum(pagerank);  % 归一化
disp('PageRank:');
disp(pagerank');
```

## 二、特殊矩阵

```matlab
% 正定矩阵检验
A_posdef = [4 2; 2 3];
eigenvalues = eig(A_posdef);
isPosDef = all(eigenvalues > 0);
fprintf('特征值: %s, 是否正定: %d\n', mat2str(eigenvalues'), isPosDef);

% Cholesky 分解（仅适用于正定矩阵）
L_chol = chol(A_posdef, 'lower');
fprintf('Cholesky: L*L'' - A 的范数: %.2e\n', norm(L_chol*L_chol' - A_posdef));

% Cholesky 解线性方程组
b = [1; 2];
y = L_chol \ b;
x = L_chol' \ y;
fprintf('Cholesky 解: x = [%.6f; %.6f]\n', x(1), x(2));

% 对称化
A_nonsym = [1 2 3; 4 5 6; 7 8 9];
A_symmetrized = (A_nonsym + A_nonsym') / 2;

% 正交化
% 生成随机正交矩阵
[Q, ~] = qr(randn(5));
fprintf('Q 正交性: %.2e\n', norm(Q'*Q - eye(5)));
```

## 三、矩阵的秩与零空间

```matlab
% 秩
A = [1 2 3; 4 5 6; 7 8 9];
r = rank(A);
fprintf('矩阵 A 的秩: %d\n', r);

% 零空间（Null Space）
N = null(A);
disp('零空间基:');
disp(N);
fprintf('A*N 范数: %.2e\n', norm(A*N));

% 有理化显示
N_rational = null(A, 'rational');
disp('零空间（有理化）:');
disp(N_rational);

% 列空间
[Q_col, ~] = qr(A, 0);
col_space = Q_col(:, 1:rank(A));

% 行空间
[~, ~, V_row] = svd(A);
row_space = V_row(:, 1:rank(A))';

% 应用：判断方程组解的情况
% Ax = b 的通解 = 特解 + 零空间的任意线性组合
b = [1; 2; 3];
x_particular = pinv(A) * b;  % 最小范数特解
N = null(A);
% 通解: x = x_particular + N*c（c 为任意系数）
fprintf('特解范数: %.4f\n', norm(x_particular));
fprintf('验证 A*x_particular ≈ b: %.2e\n', norm(A*x_particular - b));
```

## 四、矩阵函数

```matlab
% 矩阵指数
A = [0 1; -2 -3];
expA = expm(A);
fprintf('矩阵指数 e^A:\n');
disp(expA);

% 矩阵对数
logA = logm(expA);
fprintf('矩阵对数 log(e^A) ≈ A:\n');
disp(logA);
fprintf('误差: %.2e\n', norm(logA - A));

% 矩阵开方
sqrtA = sqrtm(A);
fprintf('矩阵平方根:\n');
disp(sqrtA);
fprintf('sqrtm(A)^2 - A 范数: %.2e\n', norm(sqrtA^2 - A));

% 通用矩阵函数
fA = funm(A, @sin);
fprintf('矩阵正弦 sin(A):\n');
disp(fA);

% 矩阵多项式
p = [1 2 3];  % 多项式系数 x^2 + 2x + 3
polyvalm(p, A);  % A^2 + 2*A + 3*I
```

## 五、稀疏矩阵

```matlab
% 创建稀疏矩阵
n = 1000;
A_sparse = spdiags([-ones(n,1) 2*ones(n,1) -ones(n,1)], [-1 0 1], n, n);

% 与稠密矩阵对比
A_dense = full(A_sparse);

% 存储对比
fprintf('稀疏矩阵非零元素: %d\n', nnz(A_sparse));
fprintf('稀疏矩阵占用内存: %.2f KB\n', whos('A_sparse').bytes / 1024);
fprintf('稠密矩阵占用内存: %.2f KB\n', whos('A_dense').bytes / 1024);

% 稀疏矩阵运算
b = ones(n, 1);
tic;
x_sparse = A_sparse \ b;
t_sparse = toc;

tic;
x_dense = A_dense \ b;
t_dense = toc;

fprintf('稀疏求解时间: %.4f 秒\n', t_sparse);
fprintf('稠密求解时间: %.4f 秒\n', t_dense);
fprintf('解的差异: %.2e\n', norm(x_sparse - x_dense));

% 稀疏矩阵可视化
figure;
spy(A_sparse);
title('稀疏矩阵结构 (spy)');

% 创建其他稀疏矩阵
I = [1 2 3 4];
J = [2 3 4 1];
V = [5 6 7 8];
A_triplet = sparse(I, J, V, 5, 5);  % 从三元组创建
disp(full(A_triplet));

% 常用稀疏矩阵函数
speye(100)           % 稀疏单位矩阵
sprand(100, 100, 0.1) % 10% 非零的随机稀疏矩阵
spones(A_sparse)     % 非零元素替换为1
nnz(A_sparse)        % 非零元素个数
nonzeros(A_sparse)   % 非零元素值
sparsity = 1 - nnz(A_sparse)/numel(A_sparse);  % 稀疏度
```

## 六、矩阵变换

```matlab
% 正交投影
A = [1 1 0; 0 1 1; 1 0 1];
b = [1; 2; 3];

% 最小二乘投影
x_proj = A \ b;
b_proj = A * x_proj;  % b 在 A 列空间上的投影
residual = b - b_proj;  % 残差（垂直于列空间）

fprintf('投影 b_proj = [%.4f; %.4f; %.4f]\n', b_proj(1), b_proj(2), b_proj(3));
fprintf('残差范数: %.4f\n', norm(residual));

% Householder 变换（QR 分解的基础）
% 将向量反射到某个坐标轴
v = [3; 4];
u = v / norm(v);
H = eye(2) - 2 * u * u';  % Householder 矩阵
fprintf('Householder 反射:\n');
disp(H);
fprintf('H*v = [%.6f; %.6f]\n', (H*v));
% H*v 应该是 v 的镜像

% Givens 旋转
theta = pi/6;
G = [cos(theta) -sin(theta); sin(theta) cos(theta)];
x_rot = G * [1; 0];  % 旋转 [1,0] 角度 theta
fprintf('旋转后: [%.4f, %.4f]\n', x_rot(1), x_rot(2));
```
