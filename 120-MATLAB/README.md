## 学习路线

**入门阶段（01-03）：** 掌握 MATLAB 环境、矩阵操作和基本编程结构

**进阶阶段（04-07）：** 学习可视化、数值方法、线性代数和信号处理

**专题阶段（08-12）：** 符号计算、文件操作、Simulink、Python 集成和数学建模

## 环境要求

- MATLAB R2020a 或更高版本（部分功能需要更早版本即可）
- Signal Processing Toolbox（信号处理章节）
- Optimization Toolbox（优化章节）
- Statistics and Machine Learning Toolbox（统计章节）
- Symbolic Math Toolbox（符号计算章节）
- Simulink（Simulink 章节）

## 代码说明

所有代码块使用 `matlab` 语言标记，可直接复制到 MATLAB 命令窗口或编辑器中运行。建议在 Live Script（.mlx）中运行以获得最佳体验。
