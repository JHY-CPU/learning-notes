# 数值计算

## 一、方程求解

### 1.1 线性方程组

```matlab
% 直接求解 Ax = b
A = [3 2 -1; 2 -2 4; -1 0.5 -1];
b = [1; -2; 0];

% 方法1：左除运算符（推荐，最高效）
x = A \ b;
fprintf('解: x1=%.4f, x2=%.4f, x3=%.4f\n', x(1), x(2), x(3));

% 方法2：求逆（不推荐，效率低且数值不稳定）
x2 = inv(A) * b;

% 方法3：使用 linsolve
x3 = linsolve(A, b);

% 验证残差
residual = norm(A*x - b);
fprintf('残差: %.2e\n', residual);

% 超定方程组（最小二乘）
A_over = rand(5, 3);
b_over = rand(5, 1);
x_ls = A_over \ b_over;   % 最小二乘解
x_pinv = pinv(A_over) * b_over;  % 伪逆解

% 欠定方程组（无穷多解，返回最小范数解）
A_under = rand(2, 5);
b_under = rand(2, 1);
x_min = A_under \ b_under;
```

### 1.2 非线性方程求解

```matlab
% fzero：求单变量函数的零点
f = @(x) x.^3 - 6*x.^2 + 11*x - 6;
x0 = 1.5;  % 初始猜测
root = fzero(f, x0);
fprintf('零点: x = %.6f\n', root);

% 指定搜索区间
root2 = fzero(f, [2.5 4]);
fprintf('区间[2.5,4]内的零点: x = %.6f\n', root2);

% 作图辅助查看零点位置
figure;
x = linspace(0, 4, 500);
plot(x, f(x), 'b-', 'LineWidth', 2);
hold on;
plot(root, 0, 'ro', 'MarkerSize', 10, 'MarkerFaceColor', 'r');
plot(root2, 0, 'gs', 'MarkerSize', 10, 'MarkerFaceColor', 'g');
yline(0, 'k--');
xlabel('x'); ylabel('f(x)');
title('f(x) = x^3 - 6x^2 + 11x - 6');
grid on;

% fsolve：多变量非线性方程组
options = optimoptions('fsolve', 'Display', 'iter');
F = @(x) [x(1)^2 + x(2)^2 - 1;     % 单位圆
           x(1) - x(2)];              % 直线 y = x
x0 = [0.5; 0.5];
[sol, fval, exitflag] = fsolve(F, x0, options);
fprintf('交点: (%.6f, %.6f)\n', sol(1), sol(2));
fprintf('残差: %.2e\n', norm(fval));

% 多项式求根
p = [1 -6 11 -6];  % x^3 - 6x^2 + 11x - 6
roots(p)            % 返回所有根（包括复数根）
```

## 二、数值微分与积分

### 2.1 数值微分

```matlab
% diff：差分近似导数
x = linspace(0, 2*pi, 1000);
y = sin(x);

% 一阶导数（前向差分）
dy = diff(y) ./ diff(x);
x_mid = (x(1:end-1) + x(2:end)) / 2;

figure;
plot(x, cos(x), 'b-', 'LineWidth', 2); hold on;
plot(x_mid, dy, 'r--', 'LineWidth', 1.5);
legend('cos(x) 理论值', '数值导数');
title('sin(x) 的导数');

% gradient：中心差分（更精确）
dy_grad = gradient(y, x(2)-x(1));

% 二阶导数
d2y = gradient(dy_grad, x(2)-x(1));
figure;
plot(x, -sin(x), 'b-', x, d2y, 'r--');
legend('-sin(x)', '数值二阶导数');
title('sin(x) 的二阶导数');
```

### 2.2 数值积分

```matlab
% integral（推荐）：自适应积分
f = @(x) exp(-x.^2);
I = integral(f, -inf, inf);
fprintf('高斯积分 ∫exp(-x²)dx 从 -∞ 到 +∞ = %.10f (理论值: √π = %.10f)\n', I, sqrt(pi));

% 有限区间积分
g = @(x) sin(x).^2;
I2 = integral(g, 0, pi);
fprintf('∫sin²(x)dx 从 0 到 π = %.6f (理论值: π/2 = %.6f)\n', I2, pi/2);

% 积分并获取误差估计
[I3, err] = integral(@(x) 1./(x.^3 - 2*x - 5), 0, 2, 'AbsTol', 1e-12);
fprintf('积分值: %.10f, 估计误差: %.2e\n', I3, err);

% trapz：梯形法则（已知离散数据点时使用）
x = linspace(0, pi, 1000);
y = sin(x);
I_trap = trapz(x, y);
fprintf('trapz 结果: %.6f\n', I_trap);

% cumtrapz：累积积分
F_cumul = cumtrapz(x, y);
figure;
plot(x, F_cumul, 'r-', 'LineWidth', 2);
hold on;
plot(x, 1-cos(x), 'b--', 'LineWidth', 1.5);
legend('累积积分', '1-cos(x)');

% 二重积分
f2 = @(x,y) exp(-x.^2 - y.^2);
I_double = integral2(f2, -inf, inf, -inf, inf);
fprintf('二重积分: %.6f (理论值: π = %.6f)\n', I_double, pi);

% 三重积分
f3 = @(x,y,z) x.^2 + y.^2 + z.^2;
I_triple = integral3(f3, -1, 1, -1, 1, -1, 1);
fprintf('三重积分: %.6f\n', I_triple);
```

## 三、常微分方程（ODE）

### 3.1 ode45 基本用法

```matlab
% 一阶 ODE：dy/dt = -2y + sin(t), y(0) = 1
f = @(t, y) -2*y + sin(t);
tspan = [0 10];
y0 = 1;

[t, y] = ode45(f, tspan, y0);

figure;
plot(t, y, 'b-', 'LineWidth', 2);
xlabel('t'); ylabel('y(t)');
title("dy/dt = -2y + sin(t), y(0) = 1");
grid on;

% 数值验证（与解析解对比）
y_exact = (2*sin(t) - cos(t) + 5*exp(-2*t)) / 5;
figure;
plot(t, abs(y - y_exact), 'r-', 'LineWidth', 1.5);
xlabel('t'); ylabel('|误差|');
title('ode45 数值误差');
```

### 3.2 二阶 ODE 转换

```matlab
% 二阶 ODE：y'' + 2y' + 5y = 0, y(0)=1, y'(0)=0
% 转换为方程组：令 z(1)=y, z(2)=y'
% z'(1) = z(2)
% z'(2) = -2*z(2) - 5*z(1)

ode2nd = @(t, z) [z(2); -2*z(2) - 5*z(1)];
tspan = [0 10];
z0 = [1; 0];  % [y(0); y'(0)]

[t, z] = ode45(ode2nd, tspan, z0);

figure;
plot(t, z(:,1), 'b-', 'LineWidth', 2); hold on;
plot(t, z(:,2), 'r--', 'LineWidth', 2);
legend('y(t)', "y'(t)");
xlabel('t');
title("y'' + 2y' + 5y = 0");
grid on;
```

### 3.3 高级 ODE 求解器

```matlab
% 设置精度选项
options = odeset('RelTol', 1e-8, 'AbsTol', 1e-10, 'MaxStep', 0.01);
[t, y] = ode45(f, tspan, y0, options);

% 刚性方程使用 ode15s 或 ode23s
% Van der Pol 振荡器（刚性，mu=1000）
mu = 1000;
vdp = @(t, y) [y(2); mu*(1-y(1)^2)*y(2) - y(1)];
tspan = [0 3000];
y0 = [2; 0];

% ode45 可能非常慢或失败
% [t, y] = ode45(vdp, tspan, y0);  % 慢
options = odeset('RelTol', 1e-6);
[t, y] = ode15s(vdp, tspan, y0, options);  % 刚性求解器

figure;
plot(t, y(:,1), 'b-', 'LineWidth', 1.5);
xlabel('t'); ylabel('y_1');
title('Van der Pol 振荡器 (\mu = 1000)');

% ODE 求解器对比
% ode45：非刚性问题首选（Runge-Kutta 4/5阶）
% ode23：非刚性，低精度要求（Bogacki-Shampine 2/3阶）
% ode113：非刚性，高精度（Adams-Bashforth-Moulton）
% ode15s：刚性问题首选（NDF/BDF）
% ode23s：刚性，低精度（Modified Rosenbrock）
% ode23t：中等刚性（梯形法则）
% ode23tb：刚性，低精度（TR-BDF2）
```

### 3.4 相平面分析

```matlab
% 简谐振子：x'' + ω²x = 0
omega = 2;
oscillator = @(t, z) [z(2); -omega^2*z(1)];

figure;
hold on;
for x0 = [0.5 1 1.5 2]
    for v0_sign = [-1 1]
        [t, z] = ode45(oscillator, [0 10], [x0; x0*v0_sign*omega]);
        plot(z(:,1), z(:,2), 'b-', 'LineWidth', 1);
    end
end
xlabel('x'); ylabel("x'");
title('简谐振子相平面图');
axis equal; grid on;

% 添加方向箭头
[X, V] = meshgrid(-2.5:0.5:2.5, -5:1:5);
dX = V;
dV = -omega^2*X;
quiver(X, V, dX, dV, 'r', 'LineWidth', 0.5);
```

## 四、插值

### 4.1 一维插值

```matlab
% 已知数据点
x = [0 1 2 3 4 5];
y = [0 1 4 9 16 25];  % y = x²

% 插值查询点
xq = linspace(0, 5, 100);

% 不同插值方法
y_linear = interp1(x, y, xq, 'linear');     % 线性插值（默认）
y_spline = interp1(x, y, xq, 'spline');     % 三次样条
y_pchip = interp1(x, y, xq, 'pchip');       % 分段三次 Hermite
y_nearest = interp1(x, y, xq, 'nearest');   % 最近邻

figure;
plot(x, y, 'ko', 'MarkerSize', 10, 'MarkerFaceColor', 'k'); hold on;
plot(xq, y_linear, 'b-', 'LineWidth', 1.5);
plot(xq, y_spline, 'r--', 'LineWidth', 1.5);
plot(xq, y_pchip, 'g:', 'LineWidth', 2);
plot(xq, y_nearest, 'm-.', 'LineWidth', 1);
legend('数据点', 'linear', 'spline', 'pchip', 'nearest');
title('一维插值方法对比');

% 样条插值对象（可重复使用）
pp = spline(x, y);
y_pp = ppval(pp, xq);

% pchip 更适合单调数据（避免过冲）
```

### 4.2 二维插值

```matlab
% 二维网格数据插值
[X, Y] = meshgrid(-3:1:3);
Z = peaks(X, Y);

% 插值到更细的网格
[Xq, Yq] = meshgrid(-3:0.1:3);
Zq_linear = interp2(X, Y, Z, Xq, Yq, 'linear');
Zq_spline = interp2(X, Y, Z, Xq, Yq, 'spline');

figure;
subplot(1, 2, 1);
surf(Xq, Yq, Zq_linear);
title('线性插值');
shading interp;

subplot(1, 2, 2);
surf(Xq, Yq, Zq_spline);
title('样条插值');
shading interp;

% scatteredInterpolant：散点数据插值
rng(42);
x_scatter = rand(100, 1) * 6 - 3;
y_scatter = rand(100, 1) * 6 - 3;
z_scatter = peaks(x_scatter, y_scatter);

F = scatteredInterpolant(x_scatter, y_scatter, z_scatter, 'natural');
[Xg, Yg] = meshgrid(-3:0.1:3);
Zg = F(Xg, Yg);

figure;
surf(Xg, Yg, Zg); shading interp;
hold on;
scatter3(x_scatter, y_scatter, z_scatter, 50, 'r', 'filled');
title('散点插值');
```

## 五、数值逼近

### 5.1 多项式拟合

```matlab
% 生成含噪声数据
x = linspace(0, 2*pi, 50);
y = sin(x) + 0.2*randn(size(x));

% 多项式拟合（3阶）
p = polyfit(x, y, 3);
y_fit = polyval(p, x);

figure;
plot(x, y, 'bo', 'MarkerSize', 6); hold on;
plot(x, y_fit, 'r-', 'LineWidth', 2);
plot(x, sin(x), 'g--', 'LineWidth', 1.5);
legend('含噪声数据', '3阶多项式拟合', '真实函数');
title('多项式拟合');

% 计算 R²（决定系数）
SS_res = sum((y - y_fit).^2);
SS_tot = sum((y - mean(y)).^2);
R2 = 1 - SS_res/SS_tot;
fprintf('R² = %.6f\n', R2);

% 不同阶数对比
figure;
plot(x, y, 'ko', 'MarkerSize', 5); hold on;
for deg = [1 3 5 10]
    p = polyfit(x, y, deg);
    y_f = polyval(p, linspace(0, 2*pi, 200));
    plot(linspace(0, 2*pi, 200), y_f, 'LineWidth', 1.5);
end
legend('数据', '1阶', '3阶', '5阶', '10阶');
title('不同阶数多项式拟合');
```

### 5.2 curvefit 自定义曲线拟合

```matlab
% 自定义模型拟合
x = linspace(0, 5, 100)';
y = 2.5*exp(-0.8*x) .* sin(3*x) + 0.3*randn(size(x));

% 定义模型函数
model = @(b, x) b(1)*exp(-b(2)*x) .* sin(b(3)*x);

% 初始参数猜测
b0 = [1, 1, 1];

% 使用 lsqcurvefit
options = optimoptions('lsqcurvefit', 'Display', 'iter');
[b_fit, resnorm] = lsqcurvefit(model, b0, x, y, [], [], options);

fprintf('拟合参数: A=%.4f, lambda=%.4f, omega=%.4f\n', b_fit(1), b_fit(2), b_fit(3));

figure;
plot(x, y, 'b.', 'MarkerSize', 8); hold on;
plot(x, model(b_fit, x), 'r-', 'LineWidth', 2);
legend('数据', '拟合曲线');
title('自定义曲线拟合');
```

## 六、数值稳定性与误差

```matlab
% 浮点精度问题
fprintf('0.1 + 0.2 == 0.3: %d\n', 0.1 + 0.2 == 0.3);  % false!
fprintf('0.1 + 0.2 = %.20f\n', 0.1 + 0.2);
fprintf('eps = %.2e\n', eps);

% 安全的浮点比较
isEqual = abs((0.1 + 0.2) - 0.3) < eps * 10;
fprintf('近似相等: %d\n', isEqual);

% 累积误差示例
n = 1e6;
x = single(0.1);   % 单精度
sum_single = 0;
for i = 1:n
    sum_single = sum_single + x;
end
fprintf('单精度累加: %.2f (期望: %.2f)\n', sum_single, n*x);

sum_builtin = sum(repmat(single(0.1), n, 1));
fprintf('内置sum: %.2f\n', sum_builtin);

% 条件数与数值稳定性
A = [1 1; 1 1.0001];
fprintf('矩阵条件数: %.2e\n', cond(A));
% 条件数很大说明矩阵接近奇异，数值求解不稳定
```
