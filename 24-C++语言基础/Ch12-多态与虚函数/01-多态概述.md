# 多态概述

## 一、概念说明

**多态**（Polymorphism）指同一接口在不同上下文中表现出不同行为。C++支持两种多态：

- **静态多态**（编译期）：函数重载、运算符重载、模板
- **动态多态**（运行时）：虚函数、继承

### 1.1 核心区别

| 特性 | 静态多态 | 动态多态 |
|------|---------|---------|
| 绑定时机 | 编译期 | 运行时 |
| 实现方式 | 模板/重载 | 虚函数/vtable |
| 性能 | 更快（可内联） | 有间接调用开销 |
| 灵活性 | 编译时确定类型 | 运行时可扩展 |

## 二、具体用法

### 2.1 动态多态

```cpp
#include <iostream>

class Animal {
public:
    virtual void speak() const { std::cout << "..." << std::endl; }
    virtual ~Animal() = default;
};

class Dog : public Animal {
public:
    void speak() const override { std::cout << "汪汪!" << std::endl; }
};

class Cat : public Animal {
public:
    void speak() const override { std::cout << "喵喵!" << std::endl; }
};

void letItSpeak(const Animal& a) { a.speak(); }

int main() {
    Dog dog;
    Cat cat;
    letItSpeak(dog);  // 汪汪!
    letItSpeak(cat);  // 喵喵!
    return 0;
}
```

**输出：**
```
汪汪!
喵喵!
```

### 2.2 静态多态（模板）

```cpp
#include <iostream>

template<typename T>
void process(T& obj) {
    obj.speak();  // 编译期绑定
}

int main() {
    Dog dog;
    Cat cat;
    process(dog);  // 编译期确定调用Dog::speak
    process(cat);
    return 0;
}
```

## 三、注意事项与常见陷阱

- 动态多态需要**基类引用或指针**才能触发
- 动态多态必须有**虚函数**和**继承**
- 静态多态不支持异构容器（不能放不同类型的对象）
- 性能敏感场景考虑静态多态
- 可扩展性需求选择动态多态
