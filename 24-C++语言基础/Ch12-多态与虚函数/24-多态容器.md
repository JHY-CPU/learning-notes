# 多态容器

## 一、概念说明

存储多态对象需要使用**指针或引用**（因为容器按值存储会发生切片）。常用方案：`std::vector<std::unique_ptr<Base>>`。

## 二、具体用法

### 2.1 多态对象集合

```cpp
#include <iostream>
#include <vector>
#include <memory>

class Shape {
public:
    virtual void draw() const = 0;
    virtual double area() const = 0;
    virtual ~Shape() = default;
};

class Circle : public Shape {
    double r;
public:
    Circle(double r) : r(r) {}
    void draw() const override { std::cout << "绘制圆" << std::endl; }
    double area() const override { return 3.14159 * r * r; }
};

class Rectangle : public Shape {
    double w, h;
public:
    Rectangle(double w, double h) : w(w), h(h) {}
    void draw() const override { std::cout << "绘制矩形" << std::endl; }
    double area() const override { return w * h; }
};

int main() {
    std::vector<std::unique_ptr<Shape>> shapes;
    shapes.push_back(std::make_unique<Circle>(5));
    shapes.push_back(std::make_unique<Rectangle>(3, 4));

    double totalArea = 0;
    for (const auto& s : shapes) {
        s->draw();
        totalArea += s->area();
    }
    std::cout << "总面积: " << totalArea << std::endl;
    return 0;
}
```

**输出：**
```
绘制圆
绘制矩形
总面积: 90.5398
```

## 三、注意事项与常见陷阱

- 不要使用`std::vector<Base>`（会切片）
- 使用`std::unique_ptr`自动管理内存
- 注意多态对象的排序和比较
- `std::reference_wrapper`可存储引用而非拷贝
