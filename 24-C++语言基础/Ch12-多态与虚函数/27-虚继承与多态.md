# 虚继承与多态

## 一、概念说明

虚继承中虚基类子对象位于对象末尾，通过**虚基类表**（vbtable）的偏移量定位。虚继承+多态的内存布局更复杂，包含vptr和vbptr。

## 二、具体用法

### 2.1 虚继承的vtable

```cpp
#include <iostream>

class A {
public:
    virtual void foo() { std::cout << "A::foo" << std::endl; }
    int a_val = 1;
};

class B : virtual public A {
public:
    void foo() override { std::cout << "B::foo" << std::endl; }
    int b_val = 2;
};

class C : virtual public A {
public:
    void foo() override { std::cout << "C::foo" << std::endl; }
    int c_val = 3;
};

class D : public B, public C {
public:
    void foo() override { std::cout << "D::foo" << std::endl; }
    int d_val = 4;
};

int main() {
    D d;
    d.A::a_val = 100;

    A* pa = &d;
    pa->foo();  // D::foo（多态）

    std::cout << "sizeof(A)=" << sizeof(A) << std::endl;
    std::cout << "sizeof(B)=" << sizeof(B) << std::endl;
    std::cout << "sizeof(D)=" << sizeof(D) << std::endl;
    return 0;
}
```

**输出：**
```
D::foo
sizeof(A)=16
sizeof(B)=32
sizeof(D)=48
```

## 三、注意事项与常见陷阱

- 虚继承的内存布局更复杂（vptr + vbptr）
- 虚基类子对象位于派生类对象的末尾
- 虚继承的虚函数调用需要两次间接访问
- `sizeof`会因虚继承显著增大
