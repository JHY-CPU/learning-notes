# 多态与智能指针

## 一、概念说明

`std::unique_ptr<Base>`和`std::shared_ptr<Base>`可以指向`Derived`对象，实现多态管理。智能指针自动管理内存，析构时正确调用派生类析构函数。

## 二、具体用法

### 2.1 unique_ptr多态

```cpp
#include <iostream>
#include <memory>
#include <vector>

class Resource {
public:
    virtual void use() = 0;
    virtual ~Resource() = default;
};

class Database : public Resource {
public:
    void use() override { std::cout << "使用数据库" << std::endl; }
    ~Database() override { std::cout << "关闭数据库连接" << std::endl; }
};

class Cache : public Resource {
public:
    void use() override { std::cout << "使用缓存" << std::endl; }
    ~Cache() override { std::cout << "清理缓存" << std::endl; }
};

int main() {
    std::vector<std::unique_ptr<Resource>> resources;
    resources.push_back(std::make_unique<Database>());
    resources.push_back(std::make_unique<Cache>());

    for (auto& r : resources) r->use();
    // 自动析构，正确调用派生类析构
    return 0;
}
```

**输出：**
```
使用数据库
使用缓存
清理缓存
关闭数据库连接
```

### 2.2 shared_ptr多态

```cpp
#include <iostream>
#include <memory>

class Base {
public:
    virtual ~Base() { std::cout << "~Base" << std::endl; }
};

class Derived : public Base {
public:
    ~Derived() override { std::cout << "~Derived" << std::endl; }
};

int main() {
    std::shared_ptr<Base> p = std::make_shared<Derived>();
    // 引用计数管理生命周期，析构时正确调用~Derived
    return 0;
}
```

**输出：**
```
~Derived
~Base
```

## 三、注意事项与常见陷阱

- `shared_ptr<Base>`的析构正确调用派生类析构（即使Base析构非virtual，但应保持virtual）
- `unique_ptr<Base>`转移所有权用`std::move`
- 循环引用用`std::weak_ptr`打破
- 智能指针不能混用`unique_ptr`和`shared_ptr`
