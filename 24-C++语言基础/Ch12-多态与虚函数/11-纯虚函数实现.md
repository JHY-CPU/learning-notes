# 纯虚函数实现

## 一、概念说明

纯虚函数虽然标记为`= 0`，但**可以提供函数体**。派生类仍必须显式重写，但可以通过`Base::func()`调用基类提供的默认实现。

## 二、具体用法

### 2.1 纯虚函数带默认实现

```cpp
#include <iostream>

class Logger {
public:
    // 纯虚函数但有实现（提供默认日志格式）
    virtual void log(const std::string& msg) = 0;
    virtual ~Logger() = default;

    // 纯虚函数的默认实现
    void defaultLog(const std::string& msg) {
        std::cout << "[LOG] " << msg << std::endl;
    }
};

class ConsoleLogger : public Logger {
public:
    void log(const std::string& msg) override {
        defaultLog(msg);  // 使用基类提供的实现
    }
};

class FileLogger : public Logger {
public:
    void log(const std::string& msg) override {
        std::cout << "[FILE] " << msg << std::endl;  // 完全自定义
    }
};

int main() {
    ConsoleLogger cl;
    FileLogger fl;
    cl.log("控制台消息");
    fl.log("文件消息");
    return 0;
}
```

**输出：**
```
[LOG] 控制台消息
[FILE] 文件消息
```

## 三、注意事项与常见陷阱

- 纯虚函数的函数体必须在类外定义（或在类内`= 0 { }`）
- 派生类仍然必须重写纯虚函数
- 纯虚析构函数**必须**提供实现
- 提供默认实现可以减少代码重复
