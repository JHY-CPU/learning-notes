# 类型擦除

## 一、概念说明

**类型擦除**（Type Erasure）将不同类型对象的共同行为通过统一接口暴露，隐藏具体类型信息。`std::function`、`std::any`、`std::shared_ptr`都是类型擦除的典型应用。类型擦除结合了模板（编译期多态）和虚函数（运行时多态）的优点。

### 1.1 类型擦除的结构

```
外部接口：统一类型（如std::function<void()>）
内部实现：
  ├── Concept（抽象接口）：虚函数
  └── Model<T>（模板实现）：持有具体类型T
```

## 二、具体用法

### 2.1 手动实现类型擦除

```cpp
#include <iostream>
#include <memory>
#include <string>
#include <vector>

class AnyDrawable {
    // 内部抽象接口
    struct Concept {
        virtual ~Concept() = default;
        virtual void draw() const = 0;
        virtual std::unique_ptr<Concept> clone() const = 0;
    };

    // 模板实现：持有具体类型
    template<typename T>
    struct Model : Concept {
        T value;
        Model(const T& v) : value(v) {}
        void draw() const override { value.draw(); }
        std::unique_ptr<Concept> clone() const override {
            return std::make_unique<Model>(value);
        }
    };

    std::unique_ptr<Concept> ptr;

public:
    // 任意类型都可以构造AnyDrawable
    template<typename T>
    AnyDrawable(const T& v) : ptr(std::make_unique<Model<T>>(v)) {}

    // 值语义的拷贝
    AnyDrawable(const AnyDrawable& o) : ptr(o.ptr->clone()) {}
    AnyDrawable& operator=(const AnyDrawable& o) {
        ptr = o.ptr->clone();
        return *this;
    }

    void draw() const { ptr->draw(); }
};

// 完全不同的类型，但都有draw()
struct Circle { void draw() const { std::cout << "圆形" << std::endl; } };
struct Square { void draw() const { std::cout << "方形" << std::endl; } };
struct Star   { void draw() const { std::cout << "星形" << std::endl; } };

int main() {
    std::vector<AnyDrawable> shapes;
    shapes.push_back(Circle{});
    shapes.push_back(Square{});
    shapes.push_back(Star{});

    for (const auto& s : shapes)
        s.draw();

    // 值语义拷贝
    AnyDrawable copy = shapes[0];
    copy.draw();

    return 0;
}
```

### 2.2 std::function的类型擦除

```cpp
#include <iostream>
#include <functional>
#include <vector>

// std::function就是类型擦除
void freeFunc() { std::cout << "自由函数" << std::endl; }

struct Functor {
    void operator()() const { std::cout << "函数对象" << std::endl; }
};

int main() {
    std::vector<std::function<void()>> tasks;

    // 不同类型，统一接口
    tasks.push_back(freeFunc);                          // 自由函数
    tasks.push_back(Functor{});                         // 函数对象
    tasks.push_back([]() { std::cout << "lambda" << std::endl; });  // lambda
    tasks.push_back(std::bind([](int x) { std::cout << "bind: " << x << std::endl; }, 42));

    for (auto& task : task) task();

    return 0;
}
```

### 2.3 类型擦除的any实现

```cpp
#include <iostream>
#include <memory>
#include <string>
#include <typeindex>

class Any {
    struct Base {
        virtual ~Base() = default;
        virtual std::unique_ptr<Base> clone() const = 0;
    };

    template<typename T>
    struct Holder : Base {
        T value;
        Holder(const T& v) : value(v) {}
        std::unique_ptr<Base> clone() const override {
            return std::make_unique<Holder>(value);
        }
    };

    std::unique_ptr<Base> holder;
    std::type_index type;

public:
    template<typename T>
    Any(const T& v)
        : holder(std::make_unique<Holder<T>>(v))
        , type(typeid(T)) {}

    template<typename T>
    T* cast() {
        if (type != typeid(T)) return nullptr;
        return &static_cast<Holder<T>*>(holder.get())->value;
    }

    std::type_index getType() const { return type; }
};

int main() {
    Any a1(42);
    Any a2(std::string("hello"));
    Any a1(3.14);

    if (auto* p = a1.cast<int>())
        std::cout << "int: " << *p << std::endl;

    if (auto* p = a2.cast<std::string>())
        std::cout << "string: " << *p << std::endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **类型擦除有运行时开销**：虚函数调用+堆分配
2. **`std::function`内部也是类型擦除实现**：理解其原理有助于正确使用
3. **类型擦除提供运行时多态的灵活性**：不需要继承关系
4. **注意自定义类型擦除的拷贝语义**：深拷贝需要clone
5. **类型擦除与`std::variant`互补**：variant要求编译时知道所有类型
6. **类型擦除是duck typing的C++实现**：任何有draw()的类型都可以
