# 虚函数基础

## 一、概念说明

**虚函数**（Virtual Function，C++标准 §10.3）通过`virtual`关键字声明，实现动态绑定（运行时多态）。通过基类指针或引用调用虚函数时，实际调用的是对象**真实类型**（动态类型）的版本，而非指针/引用的静态类型。

### 1.1 虚函数 vs 非虚函数

| 特性 | 虚函数 | 非虚函数 |
|------|--------|---------|
| 绑定时机 | 运行时（动态） | 编译期（静态） |
| 调用方式 | 通过vtable间接调用 | 直接调用 |
| 可重写 | 可以（override） | 只能隐藏 |
| 内联 | 通常不能 | 可以 |
| 性能 | 略慢 | 更快 |

## 二、具体用法

### 2.1 虚函数调用

```cpp
#include <iostream>

class Base {
public:
    virtual void foo() { std::cout << "Base::foo()" << std::endl; }
    virtual void bar() { std::cout << "Base::bar()" << std::endl; }
    void baz() { std::cout << "Base::baz()" << std::endl; }  // 非虚
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    void foo() override { std::cout << "Derived::foo()" << std::endl; }
    void bar() override { std::cout << "Derived::bar()" << std::endl; }
    void baz() { std::cout << "Derived::baz()" << std::endl; }  // 隐藏，不是重写
};

int main() {
    Derived d;
    Base* p = &d;

    p->foo();  // Derived::foo() — 动态绑定（虚函数）
    p->bar();  // Derived::bar() — 动态绑定（虚函数）
    p->baz();  // Base::baz()    — 静态绑定（非虚函数）

    d.foo();   // Derived::foo()
    d.baz();   // Derived::baz()  — 直接调用Derived版本

    return 0;
}
```

### 2.2 虚函数的重写规则

```cpp
#include <iostream>

class Base {
public:
    virtual void func(int x) { std::cout << "Base(" << x << ")" << std::endl; }
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    // 正确：签名完全匹配
    void func(int x) override {
        std::cout << "Derived(" << x << ")" << std::endl;
    }

    // 错误示例（有override时编译失败）：
    // void func(double x) override {}  // 参数类型不同
    // void func(int x) const override {}  // 多了const
};

int main() {
    Derived d;
    Base* p = &d;
    p->func(42);  // Derived(42)
    return 0;
}
```

### 2.3 虚函数与引用/指针

```cpp
#include <iostream>

class Base {
public:
    virtual void show() const { std::cout << "Base" << std::endl; }
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    void show() const override { std::cout << "Derived" << std::endl; }
};

// 通过引用传参实现多态
void callShow(const Base& b) {
    b.show();  // 多态调用
}

// 通过指针传参实现多态
void callShow(Base* b) {
    b->show();  // 多态调用
}

int main() {
    Derived d;

    // 指针
    Base* p = &d;
    p->show();       // Derived

    // 引用
    Base& r = d;
    r.show();        // Derived

    // 对象赋值（切片，非多态）
    Base b = d;
    b.show();        // Base

    // 函数传参
    callShow(d);     // Derived（引用传参）
    callShow(&d);    // Derived（指针传参）

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **虚函数需要通过指针或引用调用才有多态**：值传递发生切片
2. **派生类重写应使用`override`关键字**：防止签名不匹配导致隐藏
3. **非虚函数通过指针调用是静态绑定**：根据指针类型决定
4. **析构函数应声明为`virtual`**：否则通过基类指针delete是未定义行为
5. **虚函数可以有默认参数，但默认参数是静态绑定的**：根据指针类型
6. **`virtual`关键字只在声明时使用**：派生类用`override`替代
