# 虚函数基础

## 一、概念说明

**虚函数**通过`virtual`关键字声明，实现动态绑定（运行时多态）。通过基类指针或引用调用虚函数时，实际调用的是对象**真实类型**的版本。

## 二、具体用法

### 2.1 虚函数调用

```cpp
#include <iostream>

class Base {
public:
    virtual void foo() { std::cout << "Base::foo()" << std::endl; }
    void bar() { std::cout << "Base::bar()" << std::endl; }  // 非虚
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    void foo() override { std::cout << "Derived::foo()" << std::endl; }
    void bar() { std::cout << "Derived::bar()" << std::endl; }  // 遮蔽
};

int main() {
    Derived d;
    Base* p = &d;

    p->foo();  // Derived::foo() — 动态绑定（虚函数）
    p->bar();  // Base::bar() — 静态绑定（非虚函数）
    return 0;
}
```

**输出：**
```
Derived::foo()
Base::bar()
```

## 三、注意事项与常见陷阱

- 虚函数需要通过**指针或引用**调用才有多态
- 派生类重写应使用`override`关键字
- 非虚函数通过指针调用是静态绑定
- 析构函数应声明为`virtual`
- 虚函数可以有默认参数，但默认参数是静态绑定的
