# 纯虚函数

## 一、概念说明

**纯虚函数**用`= 0`声明，使类成为**抽象类**，不能被实例化。纯虚函数强制派生类提供实现，定义了接口契约。

## 二、具体用法

### 2.1 纯虚函数定义接口

```cpp
#include <iostream>

class Serializer {
public:
    virtual void serialize(const std::string& data) = 0;  // 纯虚
    virtual ~Serializer() = default;
};

class JsonSerializer : public Serializer {
public:
    void serialize(const std::string& data) override {
        std::cout << "{ \"data\": \"" << data << "\" }" << std::endl;
    }
};

class XmlSerializer : public Serializer {
public:
    void serialize(const std::string& data) override {
        std::cout << "<data>" << data << "</data>" << std::endl;
    }
};

void save(Serializer& s, const std::string& data) {
    s.serialize(data);
}

int main() {
    JsonSerializer json;
    XmlSerializer xml;
    save(json, "Hello");
    save(xml, "World");
    return 0;
}
```

**输出：**
```
{ "data": "Hello" }
<data>World</data>
```

## 三、注意事项与常见陷阱

- 包含纯虚函数的类不能实例化
- 派生类必须实现所有纯虚函数才能实例化
- 纯虚函数**可以有函数体**（`= 0 { ... }`）
- 纯虚析构函数**必须提供实现**
- 抽象类可以有数据成员和非纯虚函数
