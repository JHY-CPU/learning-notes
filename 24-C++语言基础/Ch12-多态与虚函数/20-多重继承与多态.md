# 多重继承与多态

## 一、概念说明

多重继承中，派生类有多个vptr（每个含虚函数的基类一个）。多态调用通过各自基类的vptr/vtable进行，需要正确的偏移计算。

## 二、具体用法

### 2.1 多基类虚函数

```cpp
#include <iostream>

class A {
public:
    virtual void foo() { std::cout << "A::foo" << std::endl; }
    virtual ~A() = default;
};

class B {
public:
    virtual void bar() { std::cout << "B::bar" << std::endl; }
    virtual ~B() = default;
};

class C : public A, public B {
public:
    void foo() override { std::cout << "C::foo" << std::endl; }
    void bar() override { std::cout << "C::bar" << std::endl; }
};

int main() {
    C c;
    A* pa = &c;
    B* pb = &c;  // pb的地址与pa不同（vptr偏移）

    pa->foo();  // C::foo
    pb->bar();  // C::bar
    return 0;
}
```

**输出：**
```
C::foo
C::bar
```

## 三、注意事项与常见陷阱

- 多继承对象有多个vptr
- 不同基类指针指向同一对象时地址可能不同
- 菱形继承中虚继承的vtable更复杂
- `dynamic_cast`可以进行交叉转型
