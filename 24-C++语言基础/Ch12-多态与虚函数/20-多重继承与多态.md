# 多重继承与多态

## 一、概念说明

多重继承中（C++标准 §10.1），派生类有多个vptr（每个含虚函数的基类一个）。多态调用通过各自基类的vptr/vtable进行，需要正确的偏移计算。不同基类指针指向同一对象时，指针值可能不同。

### 1.1 多继承的vptr布局

```
class C : public A, public B
内存布局：
[C的A子对象: vptr_A] → A的vtable（C的函数地址）
[C的A子对象: A的成员]
[C的B子对象: vptr_B] → B的vtable（C的函数地址）
[C的B子对象: B的成员]
[C的成员]
```

## 二、具体用法

### 2.1 多基类虚函数

```cpp
#include <iostream>

class A {
public:
    virtual void foo() { std::cout << "A::foo" << std::endl; }
    virtual ~A() = default;
};

class B {
public:
    virtual void bar() { std::cout << "B::bar" << std::endl; }
    virtual ~B() = default;
};

class C : public A, public B {
public:
    void foo() override { std::cout << "C::foo" << std::endl; }
    void bar() override { std::cout << "C::bar" << std::endl; }
};

int main() {
    C c;
    A* pa = &c;
    B* pb = &c;

    // 指针值不同！（因为vptr偏移）
    std::cout << "&c  = " << (void*)&c << std::endl;
    std::cout << "pa  = " << (void*)pa << std::endl;
    std::cout << "pb  = " << (void*)pb << std::endl;

    pa->foo();  // C::foo（通过A的vtable）
    pb->bar();  // C::bar（通过B的vtable）

    return 0;
}
```

### 2.2 多继承中的交叉转型

```cpp
#include <iostream>

class A {
public:
    virtual void aFunc() { std::cout << "A" << std::endl; }
    virtual ~A() = default;
};

class B {
public:
    virtual void bFunc() { std::cout << "B" << std::endl; }
    virtual ~B() = default;
};

class C : public A, public B {
public:
    void aFunc() override { std::cout << "C::aFunc" << std::endl; }
    void bFunc() override { std::cout << "C::bFunc" << std::endl; }
};

int main() {
    C c;
    A* pa = &c;

    // 交叉转型：从A*转到B*
    B* pb = dynamic_cast<B*>(pa);
    if (pb) {
        std::cout << "A* → B* 交叉转型成功" << std::endl;
        pb->bFunc();  // C::bFunc

        // 指针值被调整
        std::cout << "pa  = " << (void*)pa << std::endl;
        std::cout << "pb  = " << (void*)pb << std::endl;
    }

    return 0;
}
```

### 2.3 多继承中的析构

```cpp
#include <iostream>

class A {
public:
    virtual ~A() { std::cout << "~A" << std::endl; }
};

class B {
public:
    virtual ~B() { std::cout << "~B" << std::endl; }
};

class C : public A, public B {
public:
    ~C() override { std::cout << "~C" << std::endl; }
};

int main() {
    C c;
    A* pa = &c;
    B* pb = &c;

    // 通过A*删除
    // delete pa;  // 正确：调用~C → ~B → ~A
    // 通过B*删除
    // delete pb;  // 也正确：编译器调整指针

    // 栈对象自动析构
    std::cout << "=== 析构 ===" << std::endl;
    return 0;
}
// 输出：~B → ~A（析构顺序：B → A）
```

## 三、注意事项与常见陷阱

1. **多继承对象有多个vptr**：每个含虚函数的基类子对象一个
2. **不同基类指针指向同一对象时地址可能不同**：vptr偏移导致
3. **菱形继承中虚继承的vtable更复杂**：增加虚基类偏移信息
4. **`dynamic_cast`可以进行交叉转型**：自动调整指针值
5. **析构时指针自动调整**：通过任意基类指针delete都能正确析构
6. **多继承增加理解和调试难度**：优先使用单继承+接口
