# 协变返回类型

## 一、概念说明

**协变返回类型**允许派生类虚函数的返回类型是基类返回类型的派生类。这是C++中唯一允许改变虚函数返回类型的情况。

## 二、具体用法

### 2.1 协变返回类型

```cpp
#include <iostream>
#include <memory>

class Base {
public:
    virtual Base* clone() const {
        std::cout << "Base::clone" << std::endl;
        return new Base(*this);
    }
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    // 协变：返回Derived*而非Base*
    Derived* clone() const override {
        std::cout << "Derived::clone" << std::endl;
        return new Derived(*this);
    }
};

int main() {
    Derived d;
    Base* b = &d;
    Base* copy = b->clone();  // 调用Derived::clone
    delete copy;

    Derived* d2 = d.clone();  // 直接得到Derived*
    delete d2;
    return 0;
}
```

**输出：**
```
Derived::clone
Derived::clone
```

## 三、注意事项与常见陷阱

- 协变仅适用于**指针**或**引用**返回类型
- 基类返回`Base*`，派生类可返回`Derived*`
- 基类返回`Base&`，派生类可返回`Derived&`
- 返回值类型不能是不相关的类型
- 协变返回类型是Clone模式的基础
