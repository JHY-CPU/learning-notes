# 协变返回类型

## 一、概念说明

**协变返回类型**（Covariant Return Type，C++标准 §10.3）允许派生类虚函数的返回类型是基类返回类型的派生类。这是C++中唯一允许改变虚函数返回类型的情况。协变仅适用于**指针**或**引用**返回类型。

### 1.1 协变规则

| 基类返回类型 | 派生类返回类型 | 是否合法 |
|-------------|--------------|---------|
| `Base*` | `Derived*` | 合法（协变） |
| `Base&` | `Derived&` | 合法（协变） |
| `int` | `double` | 不合法 |
| `std::string` | `const char*` | 不合法 |

## 二、具体用法

### 2.1 Clone模式中的协变

```cpp
#include <iostream>
#include <memory>

class Base {
public:
    virtual Base* clone() const {
        std::cout << "Base::clone" << std::endl;
        return new Base(*this);
    }
    virtual void show() const { std::cout << "Base" << std::endl; }
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    // 协变：返回Derived*而非Base*
    Derived* clone() const override {
        std::cout << "Derived::clone" << std::endl;
        return new Derived(*this);
    }
    void show() const override { std::cout << "Derived" << std::endl; }
};

int main() {
    Derived d;

    // 通过基类指针调用
    Base* b = &d;
    Base* copy1 = b->clone();  // 调用Derived::clone，返回Derived*
    copy1->show();             // Derived

    // 直接调用：返回Derived*，无需向下转型
    Derived* copy2 = d.clone();
    copy2->show();             // Derived

    delete copy1;
    delete copy2;
    return 0;
}
```

### 2.2 工厂模式中的协变

```cpp
#include <iostream>
#include <memory>

class Product {
public:
    virtual Product* create() const = 0;
    virtual void use() const = 0;
    virtual ~Product() = default;
};

class ConcreteProduct : public Product {
public:
    ConcreteProduct* create() const override {
        return new ConcreteProduct();
    }
    void use() const override {
        std::cout << "使用ConcreteProduct" << std::endl;
    }
};

class SpecialProduct : public Product {
public:
    SpecialProduct* create() const override {
        return new SpecialProduct();
    }
    void use() const override {
        std::cout << "使用SpecialProduct" << std::endl;
    }
};

// 工厂函数：利用协变
void factoryDemo(const Product& prototype) {
    auto* product = prototype.create();  // 返回实际类型的指针
    product->use();
    delete product;
}

int main() {
    ConcreteProduct cp;
    SpecialProduct sp;
    factoryDemo(cp);  // 使用ConcreteProduct
    factoryDemo(sp);  // 使用SpecialProduct
    return 0;
}
```

### 2.3 引用的协变

```cpp
#include <iostream>

class Base {
public:
    virtual const Base& self() const {
        std::cout << "Base::self()" << std::endl;
        return *this;
    }
    virtual ~Base() = default;
};

class Derived : public Base {
public:
    const Derived& self() const override {
        std::cout << "Derived::self()" << std::endl;
        return *this;  // 返回Derived&，不是Base&
    }

    void derivedOnly() const {
        std::cout << "Derived特有方法" << std::endl;
    }
};

int main() {
    Derived d;
    const Derived& ref = d.self();  // 直接得到Derived&，无需转换
    ref.derivedOnly();

    Base* b = &d;
    const Base& baseRef = b->self();  // 多态调用，返回Derived的引用
    return 0;
}
```

## 三、注意事项与常见陷阱

1. **协变仅适用于指针或引用返回类型**：值类型不支持协变
2. **基类返回`Base*`，派生类可返回`Derived*`**：派生关系必须成立
3. **基类返回`Base&`，派生类可返回`Derived&`**：同上
4. **返回值类型不能是不相关的类型**：必须有继承关系
5. **协变返回类型是Clone模式的基础**：避免类型转换的样板代码
6. **`std::unique_ptr<Base>`返回类型不支持协变**：因为`unique_ptr<Derived>`不是`unique_ptr<Base>`的派生
