# 继承与友元

## 一、概念说明

**友元关系不可继承**。如果类A是类B的友元，A的派生类不是B的友元。如果需要派生类也能访问，必须在基类中显式声明。

## 二、具体用法

### 2.1 友元不继承

```cpp
#include <iostream>

class Secret {
    int value;
    friend class Friend;  // 只有Friend是友元
public:
    Secret(int v) : value(v) {}
};

class Friend {
public:
    void show(const Secret& s) {
        std::cout << "Friend访问: " << s.value << std::endl;
    }
};

class ChildOfFriend : public Friend {
public:
    void show(const Secret& s) {
        // std::cout << s.value;  // 编译错误！ChildOfFriend不是友元
        std::cout << "不能直接访问私有成员" << std::endl;
    }
};

int main() {
    Secret s(42);
    Friend f;
    f.show(s);        // OK
    ChildOfFriend cf;
    cf.show(s);       // 不能访问s.value
    return 0;
}
```

**输出：**
```
Friend访问: 42
不能直接访问私有成员
```

## 三、注意事项与常见陷阱

- 友元关系不能被继承
- 友元关系不能传递（A是B友元，B是C友元，A不是C友元）
- 可以将友元函数声明为模板函数
- 过度使用友元破坏封装性
