# 私有继承

## 一、概念说明

**私有继承**将基类的所有public/protected成员变为派生类的private成员。它表达"用...实现"（is-implemented-in-terms-of）语义，与组合功能相似但有一些区别。

## 二、具体用法

### 2.1 私有继承 vs 组合

```cpp
#include <iostream>
#include <vector>

// 私有继承：复用std::vector的实现
class IntSet : private std::vector<int> {
public:
    using std::vector<int>::push_back;
    using std::vector<int>::size;
    using std::vector<int>::begin;
    using std::vector<int>::end;

    bool contains(int val) const {
        for (const auto& x : *this)
            if (x == val) return true;
        return false;
    }

    void add(int val) {
        if (!contains(val))
            push_back(val);
    }
};

int main() {
    IntSet s;
    s.add(3); s.add(1); s.add(3);  // 重复不添加
    s.add(2);
    for (auto it = s.begin(); it != s.end(); ++it)
        std::cout << *it << " ";
    std::cout << "\n大小: " << s.size() << std::endl;
    return 0;
}
```

**输出：**
```
3 1 2
大小: 3
```

## 三、注意事项与常见陷阱

- 私有继承不能向上转型到基类
- 私有继承可以访问基类的protected成员（组合不行）
- 优先使用组合，除非需要访问protected成员或重写虚函数
- 可用`using`声明选择性地暴露基类接口
