# 纯虚函数与抽象类

## 一、概念说明

**纯虚函数**用`= 0`声明，表示该函数没有默认实现（但可以有函数体）。包含纯虚函数的类称为**抽象类**，不能直接实例化，只能作为基类被继承。

## 二、具体用法

### 2.1 定义抽象类

```cpp
#include <iostream>
#include <cmath>

class Shape {
public:
    // 纯虚函数
    virtual double area() const = 0;
    virtual double perimeter() const = 0;
    virtual void describe() const {
        std::cout << "面积=" << area() << ", 周长=" << perimeter() << std::endl;
    }
    virtual ~Shape() = default;
};

class Circle : public Shape {
    double radius;
public:
    Circle(double r) : radius(r) {}
    double area() const override { return M_PI * radius * radius; }
    double perimeter() const override { return 2 * M_PI * radius; }
};

class Rectangle : public Shape {
    double w, h;
public:
    Rectangle(double w, double h) : w(w), h(h) {}
    double area() const override { return w * h; }
    double perimeter() const override { return 2 * (w + h); }
};

int main() {
    // Shape s;  // 编译错误！抽象类不能实例化
    Circle c(5);
    Rectangle r(3, 4);
    c.describe();
    r.describe();

    // 通过基类指针使用
    Shape* shapes[] = {&c, &r};
    for (auto* s : shapes)
        s->describe();
    return 0;
}
```

**输出：**
```
面积=78.5398, 周长=31.4159
面积=12, 周长=14
面积=78.5398, 周长=31.4159
面积=12, 周长=14
```

## 三、注意事项与常见陷阱

- 抽象类可以有数据成员和非纯虚函数
- 纯虚函数可以有函数体（`virtual void f() = 0 { ... }`）
- 派生类必须实现所有纯虚函数才能实例化
- 析构函数可以是纯虚的，但必须提供实现（`= 0`后加函数体）
