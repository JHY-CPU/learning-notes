# 模板方法模式

## 一、概念说明

**模板方法模式**在基类中定义算法的骨架（不可重写的公共函数），将可变步骤声明为虚函数，由派生类实现。这是"控制反转"的体现——基类控制流程，派生类提供细节。

## 二、具体用法

### 2.1 模板方法实现

```cpp
#include <iostream>

class DataProcessor {
public:
    // 模板方法：定义算法骨架（非虚接口）
    void process() {
        loadData();
        validate();
        transform();
        saveResults();
    }

    virtual ~DataProcessor() = default;

protected:
    virtual void loadData() = 0;
    virtual void validate() { std::cout << "默认验证..." << std::endl; }
    virtual void transform() = 0;
    virtual void saveResults() { std::cout << "保存结果..." << std::endl; }
};

class CSVProcessor : public DataProcessor {
protected:
    void loadData() override {
        std::cout << "加载CSV数据" << std::endl;
    }
    void transform() override {
        std::cout << "CSV格式转换" << std::endl;
    }
};

class JSONProcessor : public DataProcessor {
protected:
    void loadData() override {
        std::cout << "加载JSON数据" << std::endl;
    }
    void transform() override {
        std::cout << "JSON格式转换" << std::endl;
    }
    void validate() override {
        std::cout << "JSON Schema验证" << std::endl;
    }
};

int main() {
    CSVProcessor csv;
    std::cout << "=== CSV处理 ===" << std::endl;
    csv.process();

    JSONProcessor json;
    std::cout << "=== JSON处理 ===" << std::endl;
    json.process();
    return 0;
}
```

**输出：**
```
=== CSV处理 ===
加载CSV数据
默认验证...
CSV格式转换
保存结果...
=== JSON处理 ===
加载JSON数据
JSON Schema验证
JSON格式转换
保存结果...
```

## 三、注意事项与常见陷阱

- 模板方法应声明为非虚函数（NVI模式）
- 虚函数提供钩子方法供派生类定制
- 基类提供默认实现的步骤可不重写
- 纯虚函数步骤必须由派生类实现
