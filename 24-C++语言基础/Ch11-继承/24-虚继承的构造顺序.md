# 虚继承的构造顺序

## 一、概念说明

虚继承中（C++标准 §12.6.2），**虚基类由最远派生类**（most derived class）负责构造。无论继承层次多深，虚基类只构造一次，且在所有非虚基类之前构造。中间类对虚基类的初始化参数被忽略。

### 1.1 完整构造顺序（含虚继承）

```
1. 虚基类（按声明顺序，深度优先从左到右）
2. 非虚基类（按声明顺序）
3. 非静态成员（按声明顺序）
4. 派生类构造函数体
```

## 二、具体用法

### 2.1 虚基类构造顺序

```cpp
#include <iostream>

class A {
public:
    A(int v) { std::cout << "A(" << v << ")" << std::endl; }
};

class B : virtual public A {
public:
    B() : A(1) { std::cout << "B()" << std::endl; }
    // ↑ 这个A(1)在B不是最远派生类时被忽略
};

class C : virtual public A {
public:
    C() : A(2) { std::cout << "C()" << std::endl; }
    // ↑ 这个A(2)在C不是最远派生类时被忽略
};

class D : public B, public C {
public:
    // D是最远派生类，负责构造虚基类A
    // B()和C()中的A(1)、A(2)都被忽略
    D() : A(99), B(), C() {
        std::cout << "D()" << std::endl;
    }
};

int main() {
    D d;
    // 输出：
    // A(99)   ← D指定的参数
    // B()
    // C()
    // D()
    return 0;
}
```

### 2.2 多层虚继承

```cpp
#include <iostream>

class Root {
public:
    Root() { std::cout << "Root默认构造" << std::endl; }
    Root(int v) { std::cout << "Root(" << v << ")" << std::endl; }
};

class A : virtual public Root {
public:
    A() : Root(1) { std::cout << "A()" << std::endl; }
};

class B : virtual public Root {
public:
    B() : Root(2) { std::cout << "B()" << std::endl; }
};

class C : public A, public B {
public:
    C() : Root(99), A(), B() {
        // Root(99)生效，A和B中的Root(1)、Root(2)被忽略
        std::cout << "C()" << std::endl;
    }
};

// 单独创建A时，A负责构造Root
// A a;  → Root(1) → A()

int main() {
    C c;
    // 输出：
    // Root(99)
    // A()
    // B()
    // C()
    return 0;
}
```

### 2.3 单独创建中间类

```cpp
#include <iostream>

class A {
public:
    A(int v) { std::cout << "A(" << v << ")" << std::endl; }
};

class B : virtual public A {
public:
    B() : A(10) { std::cout << "B()" << std::endl; }
};

class C : virtual public A {
public:
    C() : A(20) { std::cout << "C()" << std::endl; }
};

class D : public B, public C {
public:
    D() : A(99) { std::cout << "D()" << std::endl; }
};

int main() {
    std::cout << "=== 创建B ===" << std::endl;
    B b;  // B是最远派生类，由B构造A
    // 输出：A(10) → B()

    std::cout << "\n=== 创建D ===" << std::endl;
    D d;  // D是最远派生类，由D构造A
    // 输出：A(99) → B() → C() → D()

    std::cout << "\n=== 创建C ===" << std::endl;
    C c;  // C是最远派生类，由C构造A
    // 输出：A(20) → C()

    return 0;
}
```

### 2.4 钻石继承的完整构造顺序

```cpp
#include <iostream>

class V {
public:
    V(const std::string& s) { std::cout << "V(" << s << ")" << std::endl; }
};

class B1 : virtual public V {
public:
    B1() : V("B1") { std::cout << "B1()" << std::endl; }
};

class B2 : virtual public V {
public:
    B2() : V("B2") { std::cout << "B2()" << std::endl; }
};

class C1 : public B1 {
public:
    C1() : V("C1") { std::cout << "C1()" << std::endl; }
};

class C2 : public B2 {
public:
    C2() : V("C2") { std::cout << "C2()" << std::endl; }
};

class D : public C1, public C2 {
public:
    D() : V("D"), C1(), C2() {
        std::cout << "D()" << std::endl;
    }
};

int main() {
    D d;
    // 输出：
    // V(D)    ← 最远派生类D构造虚基类V
    // B1()
    // C1()
    // B2()
    // C2()
    // D()
    return 0;
}
```

## 三、注意事项与常见陷阱

1. **最远派生类的构造参数决定虚基类的初始化**：中间类的参数被忽略
2. **中间类对虚基类的初始化被忽略**：即使写了也无效（但必须合法）
3. **构造顺序：虚基类 → 非虚基类（声明顺序） → 成员 → 自身**
4. **单独创建中间类时由该类负责构造虚基类**：`B b;`中B构造A
5. **虚基类必须有可访问的构造函数**：最远派生类必须能调用它
6. **C++17的保证复制消除减少了虚继承中的临时对象问题**
