# 类型兼容规则

## 一、概念说明

**向上转型**（Upcasting）允许基类指针/引用指向派生类对象。这是多态的基础。向下转型需要`dynamic_cast`进行安全检查。

## 二、具体用法

### 2.1 向上转型

```cpp
#include <iostream>

class Animal {
public:
    virtual void speak() { std::cout << "Animal sound" << std::endl; }
    virtual ~Animal() = default;
};

class Dog : public Animal {
public:
    void speak() override { std::cout << "汪汪!" << std::endl; }
    void fetch() { std::cout << "捡球!" << std::endl; }
};

void makeSound(Animal& a) {  // 基类引用
    a.speak();  // 多态调用
}

int main() {
    Dog dog;

    // 向上转型：安全且自动
    Animal& ref = dog;     // 引用
    Animal* ptr = &dog;    // 指针
    Animal  obj = dog;     // 对象切片！（危险）

    ref.speak();   // "汪汪!"（多态）
    ptr->speak();  // "汪汪!"（多态）
    obj.speak();   // "Animal sound"（切片后调用基类版本）

    makeSound(dog);  // "汪汪!"
    return 0;
}
```

**输出：**
```
汪汪!
汪汪!
Animal sound
汪汪!
```

## 三、注意事项与常见陷阱

- 指针/引用向上转型保留多态行为
- 对象赋值会发生**对象切片**（slicing），丢失派生类信息
- 向下转型必须使用`dynamic_cast`，并检查结果
- 虚函数保证基类引用/指针调用正确的派生类版本
