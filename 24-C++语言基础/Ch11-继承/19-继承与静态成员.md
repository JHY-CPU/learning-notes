# 继承与静态成员

## 一、概念说明

基类的静态成员（C++标准 §10.2/§9.4.2）被所有派生类**共享**——整个继承层次只有一份副本。静态成员函数可以被继承和隐藏（非虚函数），但不能是虚函数，不支持多态。

### 1.1 静态成员在继承中的行为

| 特性 | 行为 |
|------|------|
| 静态数据成员 | 所有派生类共享同一份 |
| 静态成员函数 | 可被继承、隐藏（非重写） |
| 虚函数 | 静态函数不能是虚函数 |
| 访问方式 | `Base::member` 或 `Derived::member` |

## 二、具体用法

### 2.1 静态成员共享

```cpp
#include <iostream>

class Base {
public:
    static int count;
    static void showCount() { std::cout << "count=" << count << std::endl; }
};

int Base::count = 0;

class Derived1 : public Base {};
class Derived2 : public Base {};

int main() {
    Base::count = 10;
    Derived1::showCount();    // 输出10，共享同一份
    Derived2::count = 20;
    Base::showCount();        // 输出20

    // 验证是同一份
    std::cout << "同一份: "
              << (&Base::count == &Derived1::count) << std::endl;  // 1

    // 所有路径访问同一地址
    std::cout << "&Base::count    = " << &Base::count << std::endl;
    std::cout << "&Derived1::count = " << &Derived1::count << std::endl;
    std::cout << "&Derived2::count = " << &Derived2::count << std::endl;

    return 0;
}
```

### 2.2 静态成员函数的继承与隐藏

```cpp
#include <iostream>

class Base {
public:
    static void func() { std::cout << "Base::func()" << std::endl; }
    static void func(int x) { std::cout << "Base::func(int " << x << ")" << std::endl; }
};

class Derived : public Base {
public:
    // 隐藏Base的所有func（名称遮蔽）
    static void func(const std::string& s) {
        std::cout << "Derived::func(" << s << ")" << std::endl;
    }

    // 使用using恢复基类的重载
    using Base::func;
};

int main() {
    Derived::func("hello");  // Derived::func(string)
    Derived::func(42);       // Base::func(int) — using使之可见
    Derived::func();         // Base::func() — using使之可见

    return 0;
}
```

### 2.3 CRTP实现静态多态

```cpp
#include <iostream>

// CRTP: Curiously Recurring Template Pattern
// 基类模板，派生类将自己作为模板参数
template<typename Derived>
class Counter {
    static int count;
public:
    Counter() { ++count; }
    ~Counter() { --count; }
    static int getCount() { return count; }

    // 静态多态：编译时绑定
    void interface() {
        static_cast<Derived*>(this)->implementation();
    }
};

template<typename Derived>
int Counter<Derived>::count = 0;

class Widget : public Counter<Widget> {
public:
    void implementation() {
        std::cout << "Widget实现" << std::endl;
    }
};

class Gadget : public Counter<Gadget> {
public:
    void implementation() {
        std::cout << "Gadget实现" << std::endl;
    }
};

int main() {
    Widget w1, w2;
    Gadget g1;

    std::cout << "Widget数量: " << Counter<Widget>::getCount() << std::endl;  // 2
    std::cout << "Gadget数量: " << Counter<Gadget>::getCount() << std::endl;  // 1

    w1.interface();  // Widget实现
    g1.interface();  // Gadget实现

    return 0;
}
```

### 2.4 静态工厂模式

```cpp
#include <iostream>
#include <memory>
#include <string>
#include <unordered_map>

class Product {
public:
    virtual std::string name() const = 0;
    virtual ~Product() = default;

    // 静态工厂注册
    using Creator = std::unique_ptr<Product>(*)();
    static std::unordered_map<std::string, Creator>& registry() {
        static std::unordered_map<std::string, Creator> reg;
        return reg;
    }

    static std::unique_ptr<Product> create(const std::string& type) {
        auto it = registry().find(type);
        if (it != registry().end()) return it->second();
        return nullptr;
    }
};

class ConcreteA : public Product {
public:
    std::string name() const override { return "ConcreteA"; }
    static std::unique_ptr<Product> create() { return std::make_unique<ConcreteA>(); }
};

// 注册
struct AutoRegisterA {
    AutoRegisterA() { Product::registry()["A"] = ConcreteA::create; }
} autoRegisterA;

int main() {
    auto p = Product::create("A");
    if (p) std::cout << p->name() << std::endl;
    return 0;
}
```

## 三、注意事项与常见陷阱

1. **静态成员在整个继承体系中只有一份**：所有派生类共享基类的静态成员
2. **静态函数不能是虚函数**：`virtual static`是非法组合
3. **可以通过派生类访问基类的静态成员**：`Derived::BaseStaticMember`
4. **使用CRTP实现静态多态**：编译时多态替代虚函数，消除运行时开销
5. **静态成员的初始化顺序**：跨编译单元的静态初始化顺序未定义
6. **`inline static`（C++17）可在类内初始化非const静态成员**：简化语法
