# 继承与静态成员

## 一、概念说明

基类的静态成员被所有派生类**共享**——整个继承层次只有一份副本。静态成员函数可以被继承和重写（隐藏），但不支持虚函数的多态。

## 二、具体用法

### 2.1 静态成员共享

```cpp
#include <iostream>

class Base {
public:
    static int count;
    static void showCount() { std::cout << "count=" << count << std::endl; }
};

int Base::count = 0;

class Derived1 : public Base {};
class Derived2 : public Base {};

int main() {
    Base::count = 10;
    Derived1::showCount();   // 输出10，共享同一份
    Derived2::count = 20;
    Base::showCount();       // 输出20
    std::cout << "都是同一份: "
              << (&Base::count == &Derived1::count) << std::endl;
    return 0;
}
```

**输出：**
```
count=10
count=20
都是同一份: 1
```

## 三、注意事项与常见陷阱

- 静态成员在整个继承体系中只有一份
- 静态函数不能是虚函数
- 可以通过派生类访问基类的静态成员
- 使用"Curiously Recurring Template Pattern"（CRTP）实现静态多态
