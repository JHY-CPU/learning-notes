# 单继承基础

## 一、概念说明

**单继承**指一个类只从一个基类派生。它是继承的最简单形式，语法为`class Derived : public Base`。继承方式（public/protected/private）决定了基类成员在派生类中的访问级别。

## 二、具体用法

### 2.1 基本单继承

```cpp
#include <iostream>
#include <string>

class Person {
protected:
    std::string name;
    int age;
public:
    Person(const std::string& n, int a) : name(n), age(a) {}
    void display() const {
        std::cout << "姓名: " << name << ", 年龄: " << age << std::endl;
    }
};

class Student : public Person {
    int studentId;
public:
    Student(const std::string& n, int a, int id)
        : Person(n, a), studentId(id) {}

    void study(const std::string& subject) const {
        std::cout << name << "在学习" << subject << std::endl;
    }

    void display() const {
        Person::display();  // 调用基类版本
        std::cout << "学号: " << studentId << std::endl;
    }
};

int main() {
    Student s("张三", 20, 1001);
    s.display();
    s.study("C++");
    return 0;
}
```

**输出：**
```
姓名: 张三, 年龄: 20
学号: 1001
张三在学习C++
```

## 三、注意事项与常见陷阱

- 派生类**不能**访问基类的`private`成员
- 基类构造函数不会被继承（C++11可用`using Base::Base`继承）
- 派生类对象包含完整的基类子对象
- 基类引用/指针可以指向派生类对象（向上转型）
