# 单继承基础

## 一、概念说明

**单继承**（Single Inheritance，C++标准 §10.1）指一个类只从一个直接基类派生。它是继承的最简单形式，语法为`class Derived : access-specifier Base`。单继承形成线性的类层次结构，每个类最多有一个直接基类。

继承方式（public/protected/private）决定了基类成员在派生类中的访问级别，其中`public`继承最常用，保持is-a语义。

## 二、具体用法

### 2.1 基本单继承

```cpp
#include <iostream>
#include <string>

class Person {
protected:
    std::string name;
    int age;
public:
    Person(const std::string& n, int a) : name(n), age(a) {}
    void display() const {
        std::cout << "姓名: " << name << ", 年龄: " << age << std::endl;
    }
};

class Student : public Person {
    int studentId;
    double gpa;
public:
    Student(const std::string& n, int a, int id, double g)
        : Person(n, a), studentId(id), gpa(g) {}

    void study(const std::string& subject) const {
        std::cout << name << "在学习" << subject << std::endl;
    }

    void display() const {
        Person::display();  // 调用基类版本
        std::cout << "学号: " << studentId << ", GPA: " << gpa << std::endl;
    }

    bool isHonorStudent() const { return gpa >= 3.5; }
};

int main() {
    Student s("张三", 20, 1001, 3.8);
    s.display();
    s.study("C++");
    std::cout << "荣誉学生: " << (s.isHonorStudent() ? "是" : "否") << std::endl;
    return 0;
}
```

**输出：**
```
姓名: 张三, 年龄: 20
学号: 1001, GPA: 3.8
张三在学习C++
荣誉学生: 是
```

### 2.2 多层单继承

```cpp
#include <iostream>
#include <string>

class Animal {
protected:
    std::string name;
public:
    Animal(const std::string& n) : name(n) {}
    virtual void eat() { std::cout << name << "在吃东西" << std::endl; }
    virtual ~Animal() = default;
};

class Pet : public Animal {
protected:
    std::string owner;
public:
    Pet(const std::string& n, const std::string& o)
        : Animal(n), owner(o) {}
    void showOwner() const { std::cout << name << "的主人是" << owner << std::endl; }
};

class Dog : public Pet {
    std::string breed;
public:
    Dog(const std::string& n, const std::string& o, const std::string& b)
        : Pet(n, o), breed(b) {}
    void bark() const { std::cout << name << ": 汪汪!" << std::endl; }
    void showInfo() const {
        showOwner();
        std::cout << "品种: " << breed << std::endl;
    }
};

int main() {
    Dog dog("旺财", "李四", "柴犬");
    dog.eat();       // 来自Animal（两层之上）
    dog.showOwner(); // 来自Pet
    dog.bark();      // Dog特有
    dog.showInfo();
    return 0;
}
```

### 2.3 向上转型与基类指针

```cpp
#include <iostream>
#include <vector>
#include <memory>

class Shape {
public:
    virtual double area() const = 0;
    virtual void draw() const {
        std::cout << "绘制图形, 面积=" << area() << std::endl;
    }
    virtual ~Shape() = default;
};

class Circle : public Shape {
    double radius;
public:
    Circle(double r) : radius(r) {}
    double area() const override { return 3.14159 * radius * radius; }
};

class Rectangle : public Shape {
    double w, h;
public:
    Rectangle(double w, double h) : w(w), h(h) {}
    double area() const override { return w * h; }
};

int main() {
    // 基类指针管理派生类对象
    std::vector<std::unique_ptr<Shape>> shapes;
    shapes.push_back(std::make_unique<Circle>(5.0));
    shapes.push_back(std::make_unique<Rectangle>(3.0, 4.0));

    for (const auto& s : shapes)
        s->draw();  // 多态调用

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **派生类不能访问基类的`private`成员**：需要通过基类的public/protected接口
2. **基类构造函数不会被继承**：C++11可用`using Base::Base`显式继承
3. **派生类对象包含完整的基类子对象**：基类部分完全初始化后才初始化派生类部分
4. **基类引用/指针可以指向派生类对象**：这是多态的基础（向上转型）
5. **单继承形成线性层次**：每个类只有一个直接基类，结构清晰易理解
6. **使用`Base::`显式调用基类版本**：派生类重写函数时可通过作用域运算符调用基类实现
