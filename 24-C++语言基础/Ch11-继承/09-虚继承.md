# 虚继承

## 一、概念说明

**虚继承**（Virtual Inheritance）通过`virtual`关键字解决菱形继承问题，确保共享基类只有一个实例。被虚继承的类称为**虚基类**。

## 二、具体用法

### 2.1 解决菱形问题

```cpp
#include <iostream>

class Animal {
public:
    int legs;
    Animal(int l = 0) : legs(l) {
        std::cout << "Animal(" << legs << ")" << std::endl;
    }
};

// 虚继承
class Mammal : virtual public Animal {
public:
    Mammal() { std::cout << "Mammal" << std::endl; }
};

class WingedAnimal : virtual public Animal {
public:
    WingedAnimal() { std::cout << "WingedAnimal" << std::endl; }
};

class Bat : public Mammal, public WingedAnimal {
public:
    Bat() : Animal(2) {  // 最远派生类负责构造虚基类
        std::cout << "Bat" << std::endl;
    }
};

int main() {
    Bat bat;
    bat.legs = 2;  // OK！只有一份Animal
    std::cout << "bat.legs=" << bat.legs << std::endl;
    return 0;
}
```

**输出：**
```
Animal(2)
Mammal
WingedAnimal
Bat
bat.legs=2
```

## 三、注意事项与常见陷阱

- 虚基类由**最远派生类**负责构造
- 虚继承有少量性能开销（间接访问虚基类）
- 不是所有中间类都需要虚继承，只需直接继承虚基类的那层
- 虚继承的内存布局更复杂，使用vtable/vptr间接访问
- 除非解决菱形问题，否则不需要虚继承
