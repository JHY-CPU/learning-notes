# 基类成员访问

## 一、概念说明

派生类访问基类成员有多种方式：直接访问（`protected`/`public`成员）、通过基类作用域`Base::`、以及使用`using`声明将基类成员引入派生类作用域。

## 二、具体用法

### 2.1 using声明继承基类成员

```cpp
#include <iostream>

class Base {
public:
    void foo(int x) { std::cout << "Base::foo(int " << x << ")" << std::endl; }
    void foo(double x) { std::cout << "Base::foo(double " << x << ")" << std::endl; }
};

class Derived : public Base {
public:
    // 使用using将Base的foo引入Derived作用域
    using Base::foo;
    void foo(const char* s) { std::cout << "Derived::foo(" << s << ")" << std::endl; }
};

int main() {
    Derived d;
    d.foo(42);        // Base::foo(int)
    d.foo(3.14);      // Base::foo(double)
    d.foo("hello");   // Derived::foo(const char*)
    return 0;
}
```

**输出：**
```
Base::foo(int 42)
Base::foo(double 3.14)
Derived::foo(hello)
```

### 2.2 作用域遮蔽

```cpp
#include <iostream>

class Base {
public:
    int value = 10;
    void show() { std::cout << "Base::show()" << std::endl; }
};

class Derived : public Base {
public:
    int value = 20;  // 遮蔽Base::value
    void show() { std::cout << "Derived::show()" << std::endl; }

    void test() {
        std::cout << "Derived::value=" << value << std::endl;
        std::cout << "Base::value=" << Base::value << std::endl;
        show();
        Base::show();
    }
};

int main() {
    Derived d;
    d.test();
    return 0;
}
```

**输出：**
```
Derived::value=20
Base::value=10
Derived::show()
Base::show()
```

## 三、注意事项与常见陷阱

- 派生类同名成员会**遮蔽**（hide）基类成员（不是覆盖）
- 使用`Base::member`显式访问被遮蔽的基类成员
- `using Base::func;`可将基类的重载函数全部引入
- 作用域遮蔽与虚函数覆盖不同，遮蔽是静态绑定的
