# 数据类型概述

## 一、概念说明

C++是**静态类型语言**，每个变量在使用前必须声明其类型。数据类型决定了变量占用的内存大小、取值范围和可执行的操作。C++的数据类型分为**基本类型**、**复合类型**和**用户自定义类型**三大类。

## 二、具体用法

### 2.1 基本类型分类

```cpp
#include <iostream>
#include <limits>
using namespace std;

int main() {
    // 整型
    int a = 42;
    short b = 100;
    long c = 1000000L;
    long long d = 1000000000000LL;

    // 浮点型
    float e = 3.14f;
    double f = 3.14159265358979;
    long double g = 3.14159265358979323846L;

    // 字符型
    char h = 'A';
    wchar_t i = L'中';

    // 布尔型
    bool j = true;

    // 空类型
    // void 表示无类型，不能声明void变量

    cout << "int大小: " << sizeof(int) << " 字节" << endl;
    cout << "double大小: " << sizeof(double) << " 字节" << endl;
    cout << "char大小: " << sizeof(char) << " 字节" << endl;
    cout << "bool大小: " << sizeof(bool) << " 字节" << endl;

    return 0;
}
```

输出（典型64位系统）：
```
int大小: 4 字节
double大小: 8 字节
char大小: 1 字节
bool大小: 1 字节
```

### 2.2 类型大小总览表

```cpp
#include <iostream>
#include <climits>
#include <cfloat>
using namespace std;

int main() {
    cout << "========== 整数类型 ==========" << endl;
    cout << "short:     " << sizeof(short) << "字节, 范围: "
         << SHRT_MIN << " ~ " << SHRT_MAX << endl;
    cout << "int:       " << sizeof(int) << "字节, 范围: "
         << INT_MIN << " ~ " << INT_MAX << endl;
    cout << "long:      " << sizeof(long) << "字节, 范围: "
         << LONG_MIN << " ~ " << LONG_MAX << endl;
    cout << "long long: " << sizeof(long long) << "字节, 范围: "
         << LLONG_MIN << " ~ " << LLONG_MAX << endl;

    cout << "\n========== 浮点类型 ==========" << endl;
    cout << "float:  " << sizeof(float) << "字节, 精度: "
         << FLT_DIG << "位有效数字" << endl;
    cout << "double: " << sizeof(double) << "字节, 精度: "
         << DBL_DIG << "位有效数字" << endl;

    return 0;
}
```

输出（64位Linux/GCC）：
```
========== 整数类型 ==========
short:     2字节, 范围: -32768 ~ 32767
int:       4字节, 范围: -2147483648 ~ 2147483647
long:      8字节, 范围: -9223372036854775808 ~ 9223372036854775807
long long: 8字节, 范围: -9223372036854775808 ~ 9223372036854775807

========== 浮点类型 ==========
float:  4字节, 精度: 6位有效数字
double: 8字节, 精度: 15位有效数字
```

### 2.3 C++11固定大小类型

```cpp
#include <iostream>
#include <cstdint>
using namespace std;

int main() {
    // 固定大小的整数类型（跨平台一致）
    int8_t   i8  = 127;           // 1字节有符号
    uint8_t  u8  = 255;           // 1字节无符号
    int16_t  i16 = 32767;         // 2字节有符号
    int32_t  i32 = 2147483647;    // 4字节有符号
    int64_t  i64 = 9223372036854775807LL;  // 8字节有符号

    cout << "int8_t:  " << sizeof(int8_t) << " 字节" << endl;
    cout << "int16_t: " << sizeof(int16_t) << " 字节" << endl;
    cout << "int32_t: " << sizeof(int32_t) << " 字节" << endl;
    cout << "int64_t: " << sizeof(int64_t) << " 字节" << endl;

    return 0;
}
```

输出：
```
int8_t:  1 字节
int16_t: 2 字节
int32_t: 4 字节
int64_t: 8 字节
```

## 三、注意事项与常见陷阱

1. **类型大小不固定**：`int`和`long`的大小依赖于平台和编译器，不要假设其大小
2. **使用固定大小类型**：需要确切大小时使用`<cstdint>`中的`int32_t`等类型
3. **有符号与无符号**：默认整型是有符号的，加`unsigned`前缀变为无符号
4. **隐式转换**：不同类型的混合运算会发生隐式转换，可能导致数据丢失
5. **sizeof是编译期运算符**：`sizeof`的结果在编译时确定，不是运行时操作
