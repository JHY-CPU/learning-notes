# 整数类型详解

## 一、概念说明

整数类型用于存储不含小数部分的数值。C++提供了多种整数类型以适应不同的大小需求，分为**有符号**和**无符号**两种变体。

## 二、具体用法

### 2.1 整数类型族

```cpp
#include <iostream>
#include <climits>
using namespace std;

int main() {
    // 从小到大：short → int → long → long long
    short       s = 32767;           // 至少16位
    int         i = 2147483647;      // 至少16位，通常32位
    long        l = 2147483647L;     // 至少32位
    long long   ll = 9223372036854775807LL;  // 至少64位

    // 无符号变体
    unsigned short      us = 65535;
    unsigned int        ui = 4294967295U;
    unsigned long       ul = 4294967295UL;
    unsigned long long  ull = 18446744073709551615ULL;

    cout << "signed:   " << s << ", " << i << ", " << l << ", " << ll << endl;
    cout << "unsigned: " << us << ", " << ui << ", " << ul << ", " << ull << endl;

    return 0;
}
```

输出：
```
signed:   32767, 2147483647, 2147483647, 9223372036854775807
unsigned: 65535, 4294967295, 4294967295, 18446744073709551615
```

### 2.2 signed与unsigned

```cpp
#include <iostream>
using namespace std;

int main() {
    // signed（默认）：可以表示正数、负数和零
    int a = -42;
    int b = 0;
    int c = 42;

    // unsigned：只能表示非负数，范围更大
    unsigned int d = 0;
    unsigned int e = 4294967295U;  // 最大值

    cout << "signed int范围: " << INT_MIN << " ~ " << INT_MAX << endl;
    cout << "unsigned int范围: 0 ~ " << UINT_MAX << endl;

    // char的符号性是实现定义的
    char ch = -1;           // 可能是有符号的
    signed char sc = -1;    // 明确有符号
    unsigned char uc = 255; // 明确无符号

    cout << "char是否为有符号: " << (CHAR_MIN < 0 ? "是" : "否") << endl;

    return 0;
}
```

输出：
```
signed int范围: -2147483648 ~ 2147483647
unsigned int范围: 0 ~ 4294967295
char是否为有符号: 是
```

### 2.3 整数溢出

```cpp
#include <iostream>
#include <climits>
using namespace std;

int main() {
    // 有符号整数溢出：未定义行为（Undefined Behavior）
    int maxInt = INT_MAX;
    cout << "INT_MAX = " << maxInt << endl;
    // int overflow = maxInt + 1;  // 未定义行为！不要这样写

    // 无符号整数溢出：定义良好（回绕）
    unsigned int maxUint = UINT_MAX;
    cout << "UINT_MAX = " << maxUint << endl;
    unsigned int wrapped = maxUint + 1;  // 回绕到0
    cout << "UINT_MAX + 1 = " << wrapped << endl;

    // 无符号数下溢
    unsigned int zero = 0;
    unsigned int underflow = zero - 1;  // 回绕到最大值
    cout << "0 - 1 = " << underflow << endl;

    return 0;
}
```

输出：
```
INT_MAX = 2147483647
UINT_MAX = 4294967295
UINT_MAX + 1 = 0
0 - 1 = 4294967295
```

### 2.4 整数字面量

```cpp
#include <iostream>
using namespace std;

int main() {
    // 十进制（默认）
    int dec = 42;

    // 八进制：以0开头
    int oct = 052;  // 8*5 + 2 = 42

    // 十六进制：以0x或0X开头
    int hex = 0x2A; // 16*2 + 10 = 42

    // 二进制（C++14）：以0b或0B开头
    int bin = 0b101010; // 32+8+2 = 42

    cout << "十进制: " << dec << endl;
    cout << "八进制: " << oct << " (输出为八进制)" << endl;
    cout << "十六进制: " << hex << " (输出为十六进制)" << endl;

    // 字面量后缀
    auto a = 42;        // int
    auto b = 42U;       // unsigned int
    auto c = 42L;       // long
    auto d = 42UL;      // unsigned long
    auto e = 42LL;      // long long
    auto f = 42ULL;     // unsigned long long

    cout << "a类型大小: " << sizeof(a) << endl;
    cout << "e类型大小: " << sizeof(e) << endl;

    return 0;
}
```

输出：
```
十进制: 42
八进制: 52 (输出为八进制)
十六进制: 2a (输出为十六进制)
a类型大小: 4
e类型大小: 8
```

## 三、注意事项与常见陷阱

1. **有符号溢出是未定义行为**：程序可能出现任何结果，包括看似正常工作
2. **混合signed和unsigned**：有符号数会被转换为无符号数，负数变成巨大的正数
3. **不要用八进制字面量**：以`0`开头会被误解为八进制，`int x = 012`结果是10而非12
4. **整数字面量默认是int**：大的字面量需要加后缀（如`42LL`）确保类型正确
5. **long的大小不固定**：Windows上`long`是4字节，Linux 64位上是8字节，用`int64_t`更可靠
