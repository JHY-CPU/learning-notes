# 字面量与用户定义字面量

## 一、概念说明

字面量（literal）是源代码中直接表示**固定值**的符号。C++支持整型、浮点型、字符和字符串字面量，C++11还支持用户自定义字面量后缀。

## 二、具体用法

### 2.1 内置字面量

```cpp
#include <iostream>
using namespace std;

int main() {
    // 整型字面量
    int dec = 42;           // 十进制
    int oct = 052;          // 八进制（以0开头）
    int hex = 0x2A;         // 十六进制
    int bin = 0b101010;     // 二进制（C++14）

    // 整型后缀
    auto a = 42;            // int
    auto b = 42U;           // unsigned int
    auto c = 42L;           // long
    auto d = 42LL;          // long long
    auto e = 42UL;          // unsigned long
    auto f = 42ULL;         // unsigned long long

    // 浮点型字面量
    double pi1 = 3.14;      // double（默认）
    float pi2 = 3.14f;      // float
    long double pi3 = 3.14L; // long double

    // 字符和字符串字面量
    char ch = 'A';
    const char* str = "Hello";
    wchar_t wch = L'A';
    const wchar_t* wstr = L"Hello";
    auto u8str = u8"UTF-8";  // C++11
    auto u16str = u"UTF-16"; // C++11
    auto u32str = U"UTF-32"; // C++11

    // 特殊字符
    const char* raw = R"(原始字符串\n不转义\t保持原样)";
    cout << raw << endl;

    return 0;
}
```

输出：
```
原始字符串\n不转义\t保持原样
```

### 2.2 用户定义字面量（C++11）

```cpp
#include <iostream>
#include <string>
using namespace std;

// 用户定义字面量：operator""后缀

// 编译期字面量：接受char...模板参数
constexpr long double operator"" _deg(long double deg) {
    return deg * 3.14159265358979323846L / 180.0L;
}

// 接受unsigned long long
constexpr unsigned long long operator"" _KB(unsigned long long kb) {
    return kb * 1024;
}
constexpr unsigned long long operator"" _MB(unsigned long long mb) {
    return mb * 1024 * 1024;
}

// 接受const char*（运行时）
long double operator"" _rad(const char* str) {
    // 简化版：直接用strtod
    return stold(str);
}

// 字符串字面量
string operator"" _s(const char* str, size_t len) {
    return string(str, len);
}

int main() {
    // 使用自定义字面量
    double angle = 90.0_deg;
    cout << "90度 = " << angle << " 弧度" << endl;

    cout << "1KB = " << 1_KB << " 字节" << endl;
    cout << "1MB = " << 1_MB << " 字节" << endl;

    string name = "张三"_s;  // 自动转为std::string
    cout << "姓名: " << name << endl;

    return 0;
}
```

输出：
```
90度 = 1.5708 弧度
1KB = 1024 字节
1MB = 1048576 字节
姓名: 张三
```

### 2.3 常见用户定义字面量

```cpp
#include <iostream>
#include <chrono>
using namespace std;
using namespace std::chrono_literals;

int main() {
    // 标准库的用户定义字面量（C++14）
    auto timeout = 5s;           // 5秒
    auto duration = 100ms;       // 100毫秒
    auto period = 2min;          // 2分钟
    auto hour = 1h;              // 1小时

    cout << "5秒 = " << timeout.count() << " 毫秒" << endl;
    cout << "100毫秒 = " << duration.count() << " 毫秒" << endl;

    // std::complex_literals
    using namespace std::complex_literals;
    auto z = 3.0 + 4.0i;
    cout << "复数: " << z << endl;

    // std::string_literals
    using namespace std::string_literals;
    auto s = "Hello"s;           // std::string而非const char*
    cout << "字符串: " << s << endl;

    return 0;
}
```

输出：
```
5秒 = 5000 毫秒
100毫秒 = 100 毫秒
复数: (3,4)
字符串: Hello
```

## 三、注意事项与常见陷阱

1. **用户定义字面量必须以_开头**：不以下划线开头的后缀是为标准库保留的
2. **编译期字面量**：`operator""`接受`char...`模板参数时在编译期求值
3. **整型溢出**：字面量超过类型范围可能导致截断或编译错误
4. **八进制陷阱**：`012`是八进制的10，不是十进制的12
5. **原始字符串**：`R"(...)"`中的内容不会转义，适合正则表达式等场景
