# 类型大小与对齐

## 一、概念说明

`sizeof`运算符返回类型的**字节大小**，`alignof`（C++11）返回类型的**对齐要求**，`alignas`指定变量的对齐方式。内存对齐影响结构体大小和CPU访问效率。

## 二、具体用法

### 2.1 sizeof运算符

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    // 基本类型大小
    cout << "char:     " << sizeof(char) << " 字节" << endl;
    cout << "short:    " << sizeof(short) << " 字节" << endl;
    cout << "int:      " << sizeof(int) << " 字节" << endl;
    cout << "long:     " << sizeof(long) << " 字节" << endl;
    cout << "long long:" << sizeof(long long) << " 字节" << endl;
    cout << "float:    " << sizeof(float) << " 字节" << endl;
    cout << "double:   " << sizeof(double) << " 字节" << endl;
    cout << "bool:     " << sizeof(bool) << " 字节" << endl;
    cout << "指针:     " << sizeof(void*) << " 字节" << endl;

    // sizeof用于变量
    int x = 42;
    cout << "x的大小: " << sizeof x << endl;  // 括号可省略
    cout << "x的大小: " << sizeof(x) << endl;

    // sizeof数组
    int arr[10];
    cout << "arr总大小: " << sizeof(arr) << " 字节" << endl;
    cout << "arr元素数: " << sizeof(arr) / sizeof(arr[0]) << endl;

    // 注意：sizeof(指针)不是数组大小
    int* ptr = arr;
    cout << "ptr大小: " << sizeof(ptr) << " 字节（不是数组大小）" << endl;

    return 0;
}
```

输出（64位系统）：
```
char:     1 字节
short:    2 字节
int:      4 字节
long:     8 字节
long long:8 字节
float:    4 字节
double:   8 字节
bool:     1 字节
指针:     8 字节
x的大小: 4
x的大小: 4
arr总大小: 40 字节
arr元素数: 10
ptr大小: 8 字节（不是数组大小）
```

### 2.2 结构体大小与对齐

```cpp
#include <iostream>
using namespace std;

// 没有填充的结构体
struct NoPadding {
    char a;     // 1字节
    int b;      // 4字节
    char c;     // 1字节
};
// 实际大小：12字节（3字节填充 + 1 + 4 + 1 + 3字节填充）

// 优化排列的结构体
struct Optimized {
    int b;      // 4字节
    char a;     // 1字节
    char c;     // 1字节
    // 2字节填充
};
// 实际大小：8字节

// 紧密排列（无填充）
struct Packed {
    char a, b, c;
    // 1字节填充
};
// 实际大小：4字节

int main() {
    cout << "NoPadding大小: " << sizeof(NoPadding) << " 字节" << endl;
    cout << "Optimized大小: " << sizeof(Optimized) << " 字节" << endl;
    cout << "Packed大小: " << sizeof(Packed) << " 字节" << endl;

    // 查看成员偏移
    cout << "\nNoPadding偏移:" << endl;
    cout << "  a: " << offsetof(NoPadding, a) << endl;
    cout << "  b: " << offsetof(NoPadding, b) << endl;
    cout << "  c: " << offsetof(NoPadding, c) << endl;

    return 0;
}
```

输出：
```
NoPadding大小: 12 字节
Optimized大小: 8 字节
Packed大小: 4 字节

NoPadding偏移:
  a: 0
  b: 4
  c: 8
```

### 2.3 alignof与alignas（C++11）

```cpp
#include <iostream>
using namespace std;

// alignas指定对齐方式
struct alignas(16) AlignedStruct {
    int x;     // 4字节
    int y;     // 4字节
    // 8字节填充到16字节对齐
};

// alignas也可以用于变量
alignas(64) int cacheLine[16];  // 缓存行对齐

struct NormalStruct {
    int x;
    int y;
};

int main() {
    cout << "NormalStruct对齐: " << alignof(NormalStruct) << " 字节" << endl;
    cout << "NormalStruct大小: " << sizeof(NormalStruct) << " 字节" << endl;

    cout << "AlignedStruct对齐: " << alignof(AlignedStruct) << " 字节" << endl;
    cout << "AlignedStruct大小: " << sizeof(AlignedStruct) << " 字节" << endl;

    // 常见类型的对齐
    cout << "\nchar对齐: " << alignof(char) << endl;
    cout << "int对齐: " << alignof(int) << endl;
    cout << "double对齐: " << alignof(double) << endl;
    cout << "指针对齐: " << alignof(void*) << endl;

    return 0;
}
```

输出：
```
NormalStruct对齐: 4 字节
NormalStruct大小: 8 字节
AlignedStruct对齐: 16 字节
AlignedStruct大小: 16 字节

char对齐: 1
int对齐: 4
double对齐: 8
指针对齐: 8
```

### 2.4 使用numeric_limits

```cpp
#include <iostream>
#include <limits>
#include <climits>
#include <cfloat>
using namespace std;

int main() {
    // C++方式：numeric_limits
    cout << "int最大值: " << numeric_limits<int>::max() << endl;
    cout << "int最小值: " << numeric_limits<int>::min() << endl;
    cout << "double精度位数: " << numeric_limits<double>::digits10 << endl;
    cout << "double epsilon: " << numeric_limits<double>::epsilon() << endl;
    cout << "float无穷大: " << numeric_limits<float>::infinity() << endl;

    // C方式：climits/cfloat宏
    cout << "\nINT_MAX: " << INT_MAX << endl;
    cout << "INT_MIN: " << INT_MIN << endl;
    cout << "FLT_MAX: " << FLT_MAX << endl;
    cout << "DBL_EPSILON: " << DBL_EPSILON << endl;

    return 0;
}
```

输出：
```
int最大值: 2147483647
int最小值: -2147483648
double精度位数: 15
double epsilon: 2.22045e-16
float无穷大: inf

INT_MAX: 2147483647
INT_MIN: -2147483648
FLT_MAX: 3.40282e+38
DBL_EPSILON: 2.22045e-16
```

## 三、注意事项与常见陷阱

1. **sizeof是编译期运算符**：结果在编译时确定，不计算表达式的值
2. **结构体对齐**：成员按对齐要求排列，大小通常是最大对齐的倍数
3. **32位vs64位**：指针大小在32位系统是4字节，64位是8字节
4. **数组退化为指针**：函数参数中的数组退化为指针，`sizeof`得到指针大小
5. **对齐影响性能**：未对齐的内存访问可能导致性能下降甚至崩溃（某些架构）
