# 列表初始化详解

## 一、概念说明

C++11引入的**列表初始化**（List Initialization）使用花括号`{}`进行初始化，提供了统一的初始化语法。它能防止窄化转换、避免"最令人烦恼的解析"问题。

## 二、具体用法

### 2.1 基本列表初始化

```cpp
#include <iostream>
#include <vector>
#include <map>
using namespace std;

int main() {
    // 内置类型
    int a{42};
    double b{3.14};
    bool c{true};

    // 数组
    int arr[]{1, 2, 3, 4, 5};

    // STL容器
    vector<int> nums{10, 20, 30};
    map<string, int> ages{{"Alice", 25}, {"Bob", 30}};

    // 自定义类型
    struct Point { double x, y; };
    Point p{1.0, 2.0};

    cout << "a: " << a << ", b: " << b << endl;
    cout << "arr[0]: " << arr[0] << endl;
    cout << "nums[0]: " << nums[0] << endl;
    cout << "p: (" << p.x << ", " << p.y << ")" << endl;

    return 0;
}
```

输出：
```
a: 42, b: 3.14
arr[0]: 1
nums[0]: 10
p: (1, 2)
```

### 2.2 防止窄化转换

```cpp
#include <iostream>
using namespace std;

int main() {
    // 窄化转换：精度丢失或值域溢出
    int a = 3.14;      // OK但警告：截断为3
    // int b{3.14};     // 编译错误：禁止窄化

    int c = 300;
    // char d{c};       // 编译错误：300超出char范围

    long long big = 1LL << 40;
    // int small{big};  // 编译错误：超出int范围

    // 安全的转换仍然允许
    int e{3};           // OK：精确转换
    double f{3};        // OK：扩大精度
    char g{'A'};        // OK：字符字面量

    cout << "e: " << e << ", f: " << f << ", g: " << g << endl;

    return 0;
}
```

输出：
```
e: 3, f: 3, g: A
```

### 2.3 解决"最令人烦恼的解析"

```cpp
#include <iostream>
#include <vector>
using namespace std;

struct Timer {
    Timer(int) { cout << "Timer构造" << endl; }
};

struct Widget {
    Widget() { cout << "Widget构造" << endl; }
};

int main() {
    // 问题：C++中声明优先于表达式
    // Timer t1(int());  // 被解析为函数声明！不是变量定义

    // 解决：使用列表初始化
    Timer t1{int()};   // 明确是变量定义，调用Timer(int)
    Widget w1{};       // 明确调用默认构造函数

    // 其他例子
    vector<int> v1(10, 2);   // 10个元素，每个值为2
    vector<int> v2{10, 2};   // 2个元素：10和2

    cout << "v1大小: " << v1.size() << ", v1[0]: " << v1[0] << endl;
    cout << "v2大小: " << v2.size() << ", v2[0]: " << v2[0] << endl;

    return 0;
}
```

输出：
```
Timer构造
Widget构造
v1大小: 10, v1[0]: 2
v2大小: 2, v2[0]: 10
```

### 2.4 std::initializer_list

```cpp
#include <iostream>
#include <initializer_list>
#include <vector>
using namespace std;

// 自定义类支持列表初始化
class IntArray {
    vector<int> data;
public:
    // 接受initializer_list的构造函数
    IntArray(initializer_list<int> list) : data(list) {
        cout << "从列表构造，大小: " << list.size() << endl;
    }

    void print() const {
        for (int x : data) cout << x << " ";
        cout << endl;
    }

    int size() const { return data.size(); }
};

// 使用initializer_list的函数
int sum(initializer_list<int> values) {
    int total = 0;
    for (int v : values) total += v;
    return total;
}

int main() {
    // 创建IntArray
    IntArray arr{1, 2, 3, 4, 5};
    arr.print();

    // 使用initializer_list函数
    cout << "sum(1,2,3,4,5) = " << sum({1, 2, 3, 4, 5}) << endl;

    // vector也支持initializer_list
    vector<int> nums = {10, 20, 30, 40};
    cout << "vector大小: " << nums.size() << endl;

    return 0;
}
```

输出：
```
从列表构造，大小: 5
1 2 3 4 5
sum(1,2,3,4,5) = 15
vector大小: 4
```

## 三、注意事项与常见陷阱

1. **() vs {}**：对`vector`等容器，`()`调用构造函数，`{}`使用initializer_list
2. **auto推导**：`auto x{42}`在C++11是`initializer_list<int>`，C++14以后是`int`
3. **空列表初始化**：`int x{};`等价于`int x = 0;`，调用默认初始化
4. **嵌套列表**：嵌套列表可以初始化多维数组和嵌套容器
5. **优先级**：如果类型有initializer_list构造函数，列表初始化会优先选择它
