# 引用变量基础

## 一、概念说明

引用是变量的**别名**，一旦绑定到一个变量，就不能再绑定到其他变量。引用不是新对象，只是已有对象的另一个名字。C++中有左值引用（`&`）和右值引用（`&&`）。

## 二、具体用法

### 2.1 左值引用基本用法

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int& ref = x;  // ref是x的别名

    cout << "x = " << x << endl;
    cout << "ref = " << ref << endl;

    // 通过引用修改原变量
    ref = 100;
    cout << "修改ref后 x = " << x << endl;

    // 引用和原变量是同一块内存
    cout << "&x = " << &x << endl;
    cout << "&ref = " << &ref << endl;

    // 引用必须初始化
    // int& badRef;  // 编译错误：引用必须初始化

    // 引用不能重新绑定
    int y = 200;
    ref = y;  // 这是赋值，不是重新绑定！将y的值赋给x
    cout << "赋值后 x = " << x << ", ref = " << ref << endl;

    return 0;
}
```

输出：
```
x = 42
ref = 42
修改ref后 x = 100
&x = 0x7ffd...
&ref = 0x7ffd...
赋值后 x = 200, ref = 200
```

### 2.2 引用作为函数参数

```cpp
#include <iostream>
#include <string>
using namespace std;

// 值传递：修改不影响原变量
void byValue(int x) {
    x = 999;
}

// 引用传递：修改影响原变量
void byReference(int& x) {
    x = 999;
}

// const引用：避免拷贝，同时防止修改
void printInfo(const string& name, const int& age) {
    cout << "姓名: " << name << ", 年龄: " << age << endl;
    // name = "修改";  // 编译错误
}

// 交换两个变量的值
void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 42;

    byValue(x);
    cout << "值传递后 x = " << x << endl;  // 42

    byReference(x);
    cout << "引用传递后 x = " << x << endl;  // 999

    int a = 10, b = 20;
    swap(a, b);
    cout << "交换后: a = " << a << ", b = " << b << endl;

    string name = "张三";
    int age = 25;
    printInfo(name, age);

    return 0;
}
```

输出：
```
值传递后 x = 42
引用传递后 x = 999
交换后: a = 20, b = 10
姓名: 张三, 年龄: 25
```

### 2.3 引用与指针的区别

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int y = 100;

    // 引用
    int& ref = x;       // 必须初始化，不能为null
    // int& bad;         // 编译错误

    // 指针
    int* ptr = &x;      // 可以不初始化，可以为null
    int* nullPtr = nullptr;

    // 引用不能重新绑定
    ref = y;            // 赋值：将y的值赋给x（x=100）
    cout << "x = " << x << endl;  // 100

    // 指针可以重新指向
    ptr = &y;           // ptr现在指向y
    cout << "*ptr = " << *ptr << endl;  // 100

    // 引用的sizeof是原对象的大小
    cout << "sizeof(ref) = " << sizeof(ref) << endl;  // sizeof(int)

    // 指针的sizeof是地址大小
    cout << "sizeof(ptr) = " << sizeof(ptr) << endl;  // 8（64位系统）

    return 0;
}
```

输出：
```
x = 100
*ptr = 100
sizeof(ref) = 4
sizeof(ptr) = 8
```

### 2.4 引用作为返回值

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 返回引用：允许修改容器元素
int& getElement(vector<int>& vec, size_t index) {
    return vec[index];  // 返回元素的引用
}

// 返回const引用：只读访问
const string& getMax(const string& a, const string& b) {
    return (a > b) ? a : b;
}

// 危险：返回局部变量的引用（悬垂引用）
// int& badFunc() {
//     int local = 42;
//     return local;  // 编译警告！local将在函数返回后销毁
// }

int main() {
    vector<int> nums = {1, 2, 3, 4, 5};

    // 通过返回的引用修改元素
    getElement(nums, 2) = 999;
    cout << "nums[2] = " << nums[2] << endl;

    string s1 = "Hello";
    string s2 = "World";
    cout << "最大值: " << getMax(s1, s2) << endl;

    return 0;
}
```

输出：
```
nums[2] = 999
最大值: World
```

## 三、注意事项与常见陷阱

1. **引用必须初始化**：`int& ref;`是编译错误
2. **引用不能为null**：保证引用始终指向有效对象，比指针更安全
3. **引用不能重新绑定**：`ref = x`是赋值操作，不是重新绑定
4. **不要返回局部变量的引用**：局部变量在函数返回后销毁，返回其引用导致悬垂引用
5. **const引用绑定临时值**：`const int& ref = 42;`是合法的，延长了临时对象的生命周期
