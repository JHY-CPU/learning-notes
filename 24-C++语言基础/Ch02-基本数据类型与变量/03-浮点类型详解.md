# 浮点类型详解

## 一、概念说明

浮点类型用于表示带小数部分的数值，遵循**IEEE 754**浮点数标准。C++提供三种浮点类型：`float`、`double`和`long double`，精度依次递增。

## 二、具体用法

### 2.1 三种浮点类型

```cpp
#include <iostream>
#include <cfloat>
#include <cmath>
using namespace std;

int main() {
    // float：单精度，约6-7位有效数字
    float f = 3.14159265f;

    // double：双精度，约15-16位有效数字（推荐使用）
    double d = 3.14159265358979323846;

    // long double：扩展精度，精度取决于实现
    long double ld = 3.14159265358979323846L;

    cout << "float:        " << f << endl;
    cout << "double:       " << d << endl;
    cout << "long double:  " << ld << endl;

    cout << "\nfloat精度:  " << FLT_DIG << " 位有效数字" << endl;
    cout << "double精度: " << DBL_DIG << " 位有效数字" << endl;
    cout << "long double精度: " << LDBL_DIG << " 位有效数字" << endl;

    cout << "\nfloat大小:  " << sizeof(float) << " 字节" << endl;
    cout << "double大小: " << sizeof(double) << " 字节" << endl;
    cout << "long double大小: " << sizeof(long double) << " 字节" << endl;

    return 0;
}
```

输出：
```
float:        3.14159
double:       3.14159
long double:  3.14159

float精度:  6 位有效数字
double精度: 15 位有效数字
long double精度: 18 位有效数字

float大小:  4 字节
double大小: 8 字节
long double大小: 16 字节
```

### 2.2 精度丢失问题

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>
using namespace std;

int main() {
    // 经典精度丢失示例
    double a = 0.1;
    double b = 0.2;
    double c = a + b;

    cout << fixed << setprecision(20);
    cout << "0.1 = " << a << endl;
    cout << "0.2 = " << b << endl;
    cout << "0.1 + 0.2 = " << c << endl;
    cout << "0.1 + 0.2 == 0.3 ? " << (c == 0.3 ? "true" : "false") << endl;

    // 累积误差
    float sum = 0.0f;
    for (int i = 0; i < 1000000; i++) {
        sum += 0.000001f;
    }
    cout << "\n累加100万次0.000001 = " << sum << endl;
    cout << "期望值 = 1.0, 误差 = " << abs(sum - 1.0f) << endl;

    return 0;
}
```

输出（示例）：
```
0.1 = 0.10000000000000000555
0.2 = 0.20000000000000001110
0.1 + 0.2 = 0.30000000000000004441
0.1 + 0.2 == 0.3 ? false

累加100万次0.000001 = 0.95367431640625000000
期望值 = 1.0, 误差 = 0.0463256835937500
```

### 2.3 浮点数比较

```cpp
#include <iostream>
#include <cmath>
#include <cfloat>
using namespace std;

// 安全的浮点数比较函数
bool isClose(double a, double b, double rel_tol = 1e-9, double abs_tol = 0.0) {
    return abs(a - b) <= max(rel_tol * max(abs(a), abs(b)), abs_tol);
}

int main() {
    double x = 0.1 + 0.2;
    double y = 0.3;

    // 错误的比较方式
    cout << "直接比较: " << (x == y ? "相等" : "不等") << endl;

    // 正确方式1：epsilon比较
    cout << "epsilon比较: " << (abs(x - y) < DBL_EPSILON ? "相等" : "不等") << endl;

    // 正确方式2：相对误差比较
    cout << "相对误差比较: " << (isClose(x, y) ? "相等" : "不等") << endl;

    return 0;
}
```

输出：
```
直接比较: 不等
epsilon比较: 相等
相对误差比较: 相等
```

### 2.4 特殊浮点值

```cpp
#include <iostream>
#include <cmath>
#include <limits>
using namespace std;

int main() {
    // 无穷大
    double inf = numeric_limits<double>::infinity();
    double neg_inf = -numeric_limits<double>::infinity();

    // 非数字（NaN）
    double nan_val = numeric_limits<double>::quiet_NaN();
    double nan_from_op = 0.0 / 0.0;
    double nan_from_sqrt = sqrt(-1.0);

    cout << "正无穷: " << inf << endl;
    cout << "负无穷: " << neg_inf << endl;
    cout << "NaN: " << nan_val << endl;

    // NaN的特殊性质：与任何值比较都为false
    cout << "NaN == NaN ? " << (nan_val == nan_val ? "true" : "false") << endl;
    cout << "NaN != NaN ? " << (nan_val != nan_val ? "true" : "false") << endl;

    // 检查特殊值
    cout << "isinf(inf): " << (isinf(inf) ? "true" : "false") << endl;
    cout << "isnan(nan): " << (isnan(nan_val) ? "true" : "false") << endl;
    cout << "isfinite(42): " << (isfinite(42.0) ? "true" : "false") << endl;

    return 0;
}
```

输出：
```
正无穷: inf
负无穷: -inf
NaN: nan
NaN == NaN ? false
NaN != NaN ? true
isinf(inf): true
isnan(nan): true
isfinite(42): true
```

## 三、注意事项与常见陷阱

1. **不要用==比较浮点数**：由于精度限制，应该使用误差范围比较
2. **默认使用double**：除非有特殊需要，否则不要使用float，精度损失可能带来严重问题
3. **避免浮点数累加**：大量累加会累积误差，使用Kahan求和算法等补偿方法
4. **NaN不等于任何值**：包括NaN自身，必须用`isnan()`函数检查
5. **字面量后缀**：float字面量加`f`后缀（如`3.14f`），否则默认是double
