# 变量初始化方式

## 一、概念说明

C++提供了多种变量初始化方式，不同方式有不同的语义和安全保证。C++11引入的**列表初始化**（花括号初始化）是最推荐的方式。

## 二、具体用法

### 2.1 默认初始化与值初始化

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int globalVar;  // 全局变量：零初始化

int main() {
    // 默认初始化：内置类型值不确定，类类型调用默认构造
    int a;              // 未初始化，值不确定！
    string s;           // 调用string默认构造函数，空字符串
    vector<int> v;      // 空vector

    // 值初始化（加括号）：内置类型零初始化
    int b = int();      // b = 0
    int c{};            // c = 0（列表初始化）

    cout << "全局变量: " << globalVar << endl;  // 0
    cout << "值初始化: " << b << endl;          // 0
    cout << "列表初始化: " << c << endl;        // 0
    cout << "字符串: \"" << s << "\"" << endl;

    // cout << "未初始化: " << a << endl;  // 未定义行为！

    return 0;
}
```

输出：
```
全局变量: 0
值初始化: 0
列表初始化: 0
字符串: ""
```

### 2.2 拷贝初始化与直接初始化

```cpp
#include <iostream>
#include <string>
using namespace std;

class MyClass {
    int value;
public:
    MyClass(int v) : value(v) {
        cout << "构造函数: " << v << endl;
    }
    MyClass(const MyClass& other) : value(other.value) {
        cout << "拷贝构造: " << value << endl;
    }
    int getValue() const { return value; }
};

int main() {
    // 拷贝初始化（=）
    string s1 = "Hello";        // 可能调用构造+移动
    MyClass obj1 = 42;          // 调用构造函数（非拷贝构造）

    // 直接初始化（()）
    string s2("World");         // 调用构造函数
    MyClass obj2(100);          // 调用构造函数

    cout << "obj1: " << obj1.getValue() << endl;
    cout << "obj2: " << obj2.getValue() << endl;

    return 0;
}
```

输出：
```
构造函数: 42
构造函数: 100
obj1: 42
obj2: 100
```

### 2.3 列表初始化（C++11）

```cpp
#include <iostream>
#include <vector>
#include <initializer_list>
using namespace std;

int main() {
    // 列表初始化（统一初始化语法）
    int a = {42};
    int b{42};
    int c = {0};
    double d{3.14};

    // 窄化转换被禁止
    int x = 3.14;         // OK：截断为3（警告）
    // int y{3.14};        // 编译错误：禁止窄化转换
    int z = {3};          // OK：没有窄化

    // 聚合初始化
    int arr[]{1, 2, 3, 4, 5};
    struct Point { int x, y; };
    Point p{10, 20};

    // 容器初始化
    vector<int> nums{1, 2, 3, 4, 5};
    vector<string> names{"Alice", "Bob", "Charlie"};

    cout << "a: " << a << endl;
    cout << "arr[0]: " << arr[0] << endl;
    cout << "p: (" << p.x << ", " << p.y << ")" << endl;
    cout << "nums大小: " << nums.size() << endl;

    return 0;
}
```

输出：
```
a: 42
arr[0]: 1
p: (10, 20)
nums大小: 5
```

### 2.4 聚合初始化

```cpp
#include <iostream>
#include <cstring>
using namespace std;

// 聚合类型：没有用户定义构造函数、没有私有/保护成员、
// 没有虚函数、没有基类的类/数组
struct Student {
    char name[20];
    int age;
    double gpa;
};

struct Point3D {
    double x, y, z;
};

int main() {
    // 聚合初始化
    Student s1 = {"张三", 20, 3.8};
    Student s2{"李四", 22, 3.9};

    Point3D origin{0, 0, 0};
    Point3D p{1.0, 2.0, 3.0};

    // 部分初始化：未指定的成员零初始化
    Student s3 = {"王五"};  // age=0, gpa=0.0

    cout << s1.name << ", " << s1.age << ", " << s1.gpa << endl;
    cout << s2.name << ", " << s2.age << ", " << s2.gpa << endl;
    cout << "p: (" << p.x << ", " << p.y << ", " << p.z << ")" << endl;
    cout << s3.name << ", " << s3.age << ", " << s3.gpa << endl;

    return 0;
}
```

输出：
```
张三, 20, 3.8
李四, 22, 3.9
p: (1, 2, 3)
王五, 0, 0
```

## 三、注意事项与常见陷阱

1. **未初始化变量是定时炸弹**：局部内置类型变量不初始化，其值是不确定的，可能导致随机bug
2. **列表初始化禁止窄化**：`int x{3.14}`会编译错误，比`int x = 3.14`更安全
3. **auto的陷阱**：`auto x{42}`推导为`initializer_list<int>`（C++11）或`int`（C++14以后）
4. **() vs {}**：`vector<int> v(10)`创建10个0的vector，`vector<int> v{10}`创建包含10这一个元素的vector
5. **类成员初始化**：优先使用成员初始化列表（`: member(val)`），而非构造函数体内赋值
