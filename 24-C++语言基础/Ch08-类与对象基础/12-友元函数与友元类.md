# 友元函数与友元类

## 一、概念说明

`friend`关键字允许指定的函数或类访问另一个类的私有和保护成员。友元关系**不能传递**、**不能继承**、**不能交换**。常用于运算符重载和紧密协作的类之间。

## 二、具体用法

### 2.1 友元函数

```cpp
class Box {
    double width, height;
public:
    Box(double w, double h) : width(w), height(h) {}
    friend double area(const Box& b);
};

double area(const Box& b) {
    return b.width * b.height;  // 直接访问私有成员
}

Box box(3, 4);
std::cout << area(box) << std::endl;  // 输出: 12
```

### 2.2 友元类

```cpp
class Engine {
    int rpm;
    Engine(int r) : rpm(r) {}
    friend class Car;  // Car可访问Engine的私有成员
};

class Car {
public:
    void showRPM(const Engine& e) {
        std::cout << "RPM: " << e.rpm << std::endl;
    }
};

Engine e(3000);
Car car;
car.showRPM(e);
// 输出: RPM: 3000
```

### 2.3 运算符重载中的友元

```cpp
class Vector2D {
    double x, y;
public:
    Vector2D(double x, double y) : x(x), y(y) {}

    // operator<<必须是友元（左操作数是ostream）
    friend std::ostream& operator<<(std::ostream& os, const Vector2D& v) {
        return os << "(" << v.x << ", " << v.y << ")";
    }

    // operator+作为友元支持对称性
    friend Vector2D operator+(const Vector2D& a, const Vector2D& b) {
        return Vector2D(a.x + b.x, a.y + b.y);
    }
};

Vector2D v1(1, 2), v2(3, 4);
std::cout << v1 + v2 << std::endl;  // 输出: (4, 6)
```

### 2.4 友元声明的位置

```cpp
class A {
    int secret = 42;
    friend class B;  // 声明B是A的友元
};

class B {
public:
    void accessA(const A& a) {
        std::cout << a.secret << std::endl;  // 可访问
    }
};
```

## 三、注意事项与常见陷阱

- 友元关系不能传递：A友元B，B友元C，不代表A友元C
- 友元关系不能继承：派生类不是基类友元的友元
- 友元关系不能交换：A友元B不代表B友元A
- `operator<<`和`operator>>`通常需要声明为友元
- 过多友元说明类的接口设计可能需要改进
