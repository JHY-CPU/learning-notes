# 嵌套类

## 一、概念说明

嵌套类是在另一个类内部定义的类。嵌套类是一个独立的类，只是作用域在外围类内。嵌套类可以访问外围类的私有成员（在C++11中），但外围类不能访问嵌套类的私有成员。

## 二、具体用法

### 2.1 基本嵌套类

```cpp
class LinkedList {
public:
    // 嵌套类：链表节点
    class Node {
    public:
        int data;
        Node* next;
        Node(int d) : data(d), next(nullptr) {}
    };

private:
    Node* head = nullptr;

public:
    void pushFront(int val) {
        Node* newNode = new Node(val);
        newNode->next = head;
        head = newNode;
    }

    void print() const {
        for (Node* cur = head; cur; cur = cur->next) {
            std::cout << cur->data << " ";
        }
        std::cout << std::endl;
    }

    ~LinkedList() {
        while (head) {
            Node* temp = head;
            head = head->next;
            delete temp;
        }
    }
};

LinkedList list;
list.pushFront(3);
list.pushFront(2);
list.pushFront(1);
list.print();
// 输出: 1 2 3
```

### 2.2 嵌套类作为实现细节

```cpp
class Tree {
public:
    class Node;  // 前向声明

private:
    Node* root = nullptr;

public:
    class Node {
    public:
        int key;
        Node* left = nullptr;
        Node* right = nullptr;
        Node(int k) : key(k) {}
    };

    void insert(int key) {
        root = insertImpl(root, key);
    }
private:
    Node* insertImpl(Node* node, int key) {
        if (!node) return new Node(key);
        if (key < node->key) node->left = insertImpl(node->left, key);
        else node->right = insertImpl(node->right, key);
        return node;
    }
};
```

### 2.3 public嵌套类型

```cpp
class Container {
public:
    // 暴露给外部的嵌套类型
    using value_type = int;
    using iterator = int*;
    using const_iterator = const int*;

    // 外部可使用 Container::value_type
};

Container::value_type x = 42;  // 等价于 int x = 42
```

## 三、注意事项与常见陷阱

- 嵌套类的作用域在外围类内，需通过`Outer::Inner`访问
- 嵌套类默认不能访问外围类的非静态成员（除非声明为友元）
- 嵌套类的前向声明必须在public或protected区域
- 嵌套类和外围类无特殊的内存关系
- 嵌套类常用于隐藏实现细节（类似PIMPL）
