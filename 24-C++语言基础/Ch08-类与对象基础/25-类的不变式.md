# 类的不变式

## 一、概念说明

类的**不变式**（Invariant）是对象在任何时刻都必须满足的条件。不变式在构造时建立，在每个成员函数调用前后保持。RAII是不变式的常见应用：资源的获取和释放构成不变式。

## 二、具体用法

### 2.1 维护不变式

```cpp
class Date {
    int year, month, day;
public:
    Date(int y, int m, int d) : year(y), month(m), day(d) {
        if (m < 1 || m > 12)
            throw std::invalid_argument("月份无效");
        if (d < 1 || d > daysInMonth(y, m))
            throw std::invalid_argument("日期无效");
    }

    void setMonth(int m) {
        if (m < 1 || m > 12)
            throw std::invalid_argument("月份无效");
        month = m;
        // 如果当前日期超过新月份的最大天数，调整
        if (day > daysInMonth(year, month))
            day = daysInMonth(year, month);
    }

private:
    static int daysInMonth(int y, int m) {
        int days[] = {31,28,31,30,31,30,31,31,30,31,30,31};
        if (m == 2 && (y % 4 == 0 && (y % 100 != 0 || y % 400 == 0)))
            return 29;
        return days[m - 1];
    }
};

Date d(2024, 2, 29);  // 闰年2月29日
std::cout << "日期有效\n";  // 输出: 日期有效
// Date bad(2024, 13, 1);  // 抛出异常
```

### 2.2 RAII实现资源不变式

```cpp
class FileGuard {
    FILE* file;
public:
    // 不变式：file要么为nullptr，要么指向有效文件
    FileGuard(const char* name) : file(fopen(name, "r")) {
        if (!file) throw std::runtime_error("打开文件失败");
    }

    ~FileGuard() { if (file) fclose(file); }

    bool isOpen() const { return file != nullptr; }

    // 禁止拷贝，保证不变式
    FileGuard(const FileGuard&) = delete;
    FileGuard& operator=(const FileGuard&) = delete;
};
```

### 2.3 不变式与类设计

```cpp
class Stack {
    std::vector<int> data;
    static constexpr size_t MAX_SIZE = 1000;
public:
    // 不变式：size <= MAX_SIZE
    void push(int val) {
        if (data.size() >= MAX_SIZE)
            throw std::overflow_error("栈满");
        data.push_back(val);
    }

    int pop() {
        if (data.empty())
            throw std::underflow_error("栈空");
        int val = data.back();
        data.pop_back();
        return val;
    }

    size_t size() const { return data.size(); }
};
```

## 三、注意事项与常见陷阱

- 构造函数是建立不变式的关键时刻
- 构造失败时抛异常，对象不会"部分构造"
- 不变式应在每个public函数执行前后保持
- 使用private封装数据，通过public接口维护不变式
- 析构函数也应维护不变式（释放所有资源）
