# 静态成员函数

## 一、概念说明

静态成员函数属于**类本身**，不依赖对象实例调用。它**没有`this`指针**，因此只能访问静态成员。静态函数常用于工厂方法、工具函数、单例获取等场景。

## 二、具体用法

### 2.1 基本静态函数

```cpp
class Math {
public:
    static double pi() { return 3.14159265; }
    static double circleArea(double r) {
        return pi() * r * r;
    }
    static int max(int a, int b) {
        return (a > b) ? a : b;
    }
};

// 通过类名调用（无需对象）
std::cout << Math::pi() << std::endl;          // 输出: 3.14159
std::cout << Math::circleArea(5) << std::endl;  // 输出: 78.5398
std::cout << Math::max(3, 7) << std::endl;      // 输出: 7
```

### 2.2 工厂方法

```cpp
class Connection {
    std::string host;
    int port;
    Connection(const std::string& h, int p) : host(h), port(p) {}

public:
    // 静态工厂方法
    static Connection create(const std::string& host, int port) {
        return Connection(host, port);
    }

    void show() const {
        std::cout << host << ":" << port << std::endl;
    }
};

auto conn = Connection::create("localhost", 8080);
conn.show();
// 输出: localhost:8080
```

### 2.3 单例中的静态函数

```cpp
class Logger {
    Logger() = default;
public:
    static Logger& getInstance() {
        static Logger instance;  // C++11保证线程安全
        return instance;
    }

    void log(const std::string& msg) {
        std::cout << "[LOG] " << msg << std::endl;
    }
};

Logger::getInstance().log("程序启动");
// 输出: [LOG] 程序启动
```

### 2.4 静态函数不能访问非静态成员

```cpp
class Example {
    int value;  // 非静态成员
public:
    static void bad() {
        // value = 10;  // 编译错误：无this指针
        std::cout << "静态函数\n";
    }
};
```

## 三、注意事项与常见陷阱

- 静态成员函数不能是`const`、`virtual`、或有`this`指针
- 静态函数可以访问静态成员和调用其他静态函数
- 通过对象调用静态函数是合法的但不推荐
- 静态函数可作为回调函数（签名匹配时）
- 局部静态变量在C++11后保证线程安全初始化
