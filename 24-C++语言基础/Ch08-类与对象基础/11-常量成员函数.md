# 常量成员函数

## 一、概念说明

`const`成员函数承诺不修改对象的状态（数据成员）。在`const`成员函数中，`this`指针的类型是`const ClassName*`。`mutable`关键字允许特定成员在`const`函数中被修改。

## 二、具体用法

### 2.1 const成员函数

```cpp
class Account {
    double balance;
public:
    Account(double b) : balance(b) {}

    // const成员函数：不修改对象
    double getBalance() const {
        // balance = 0;  // 编译错误
        return balance;
    }

    // 非const成员函数：可修改
    void deposit(double amount) {
        balance += amount;
    }
};

const Account acc(1000);  // const对象
std::cout << acc.getBalance() << std::endl;  // 输出: 1000
// acc.deposit(500);  // 编译错误：const对象不能调用非const函数
```

### 2.2 mutable关键字

```cpp
class Cached {
    std::string data;
    mutable bool dirty;  // 可在const函数中修改
    mutable int accessCount;  // 逻辑上不影响const语义

public:
    Cached(const std::string& d) : data(d), dirty(false), accessCount(0) {}

    std::string getData() const {
        ++accessCount;  // mutable成员可修改
        return data;
    }

    int getAccessCount() const { return accessCount; }
};

Cached c("Hello");
c.getData();
c.getData();
std::cout << c.getAccessCount() << std::endl;  // 输出: 2
```

### 2.3 const重载

```cpp
class Text {
    std::string data;
public:
    Text(const std::string& d) : data(d) {}

    // 非const版本：返回可修改引用
    char& operator[](size_t i) {
        std::cout << "非const\n";
        return data[i];
    }

    // const版本：返回const引用
    const char& operator[](size_t i) const {
        std::cout << "const\n";
        return data[i];
    }
};

Text t("Hello");
t[0] = 'h';        // 调用非const版本

const Text ct("World");
char c = ct[0];    // 调用const版本
```

### 2.4 const成员函数的等价理解

```cpp
class Example {
    int x;
public:
    // 以下两个声明等价
    int getX() const;
    // int getX(const Example* this);  // 伪代码
};
```

## 三、注意事项与常见陷阱

- `const`对象只能调用`const`成员函数
- `mutable`成员在逻辑上不影响对象的const语义
- const和非const成员函数可以构成重载
- `const`成员函数可返回非const值（不修改对象自身）
- 常用`const`成员函数实现**逻辑const**而非**位const**
