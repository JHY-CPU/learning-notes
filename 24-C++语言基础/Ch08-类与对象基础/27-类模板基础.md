# 类模板基础

## 一、概念说明

类模板允许创建**与类型无关**的类。使用时通过模板参数指定具体类型，编译器为每个使用的类型生成独立的类定义。`std::vector<T>`就是类模板的典型例子。

## 二、具体用法

### 2.1 基本类模板

```cpp
template <typename T>
class Box {
    T value;
public:
    Box(T v) : value(v) {}
    T get() const { return value; }
    void set(T v) { value = v; }
};

Box<int> intBox(42);
Box<std::string> strBox("Hello");

std::cout << intBox.get() << std::endl;  // 输出: 42
std::cout << strBox.get() << std::endl;  // 输出: Hello
```

### 2.2 多模板参数

```cpp
template <typename K, typename V>
class Pair {
    K first;
    V second;
public:
    Pair(K k, V v) : first(k), second(v) {}
    K getKey() const { return first; }
    V getValue() const { return second; }
};

Pair<std::string, int> p("age", 25);
std::cout << p.getKey() << ": " << p.getValue() << std::endl;
// 输出: age: 25
```

### 2.3 非类型模板参数

```cpp
template <typename T, size_t N>
class FixedArray {
    T data[N];
public:
    T& operator[](size_t i) { return data[i]; }
    const T& operator[](size_t i) const { return data[i]; }
    size_t size() const { return N; }
};

FixedArray<int, 5> arr;
for (size_t i = 0; i < arr.size(); ++i) arr[i] = i * 10;
std::cout << arr[3] << std::endl;  // 输出: 30
```

### 2.4 模板特化

```cpp
// 通用版本
template <typename T>
class Printer {
public:
    void print(const T& val) {
        std::cout << val << std::endl;
    }
};

// 针对bool的特化
template <>
class Printer<bool> {
public:
    void print(bool val) {
        std::cout << (val ? "true" : "false") << std::endl;
    }
};

Printer<int> pi;
pi.print(42);  // 输出: 42

Printer<bool> pb;
pb.print(true);  // 输出: true
```

### 2.5 类内简化（C++17）

```cpp
// C++17类模板参数推导
Box intBox2(42);        // 推导为 Box<int>
Pair p2("key", 100);    // 推导为 Pair<const char*, int>
```

## 三、注意事项与常见陷阱

- 类模板通常定义在头文件中（需要完整定义来实例化）
- 每个模板参数类型生成独立的类（代码膨胀）
- 模板参数必须支持模板中使用的操作
- C++17允许类模板参数推导（CTAD）
- 模板特化应在原模板可见后定义
