# 对象创建与销毁顺序

## 一、概念说明

对象的创建和销毁顺序因存储位置而异：栈对象按声明顺序创建、逆序销毁；堆对象按new/delete顺序；全局对象在main前创建、main后销毁；静态局部变量首次执行时创建、程序结束时销毁。

## 二、具体用法

### 2.1 栈对象

```cpp
{
    std::cout << "进入作用域\n";
    StackObj a("A"), b("B"), c("C");
    std::cout << "作用域中\n";
}
// 输出:
// 进入作用域
// A 构造
// B 构造
// C 构造
// 作用域中
// C 析构（逆序）
// B 析构
// A 析构
```

### 2.2 成员初始化顺序

```cpp
class Container {
    Member m1, m2, m3;
public:
    Container() : m3("C"), m1("A"), m2("B") {}
    // 实际构造顺序：m1 → m2 → m3（按声明顺序）
};
// 输出:
// A 构造
// B 构造
// C 构造
```

### 2.3 基类与派生类

```cpp
class Base {
public:
    Base() { std::cout << "Base 构造\n"; }
    ~Base() { std::cout << "Base 析构\n"; }
};

class Derived : public Base {
    Member mem;
public:
    Derived() { std::cout << "Derived 构造\n"; }
    ~Derived() { std::cout << "Derived 析构\n"; }
};

// Derived d; 输出:
// Base 构造
// Member 构造
// Derived 构造
// Derived 析构
// Member 析构
// Base 析构
```

### 2.4 全局与静态对象

```cpp
// 全局对象：main之前构造，main之后析构
struct Global {
    Global() { std::cout << "Global 构造\n"; }
    ~Global() { std::cout << "Global 析构\n"; }
};

Global g;

int main() {
    std::cout << "main 开始\n";

    // 静态局部：首次执行时构造
    static std::string s = "Static Local";
    std::cout << s << std::endl;

    std::cout << "main 结束\n";
    return 0;
}
// 输出:
// Global 构造
// main 开始
// Static Local
// main 结束
// Global 析构
```

## 三、注意事项与常见陷阱

- 构造顺序：基类 → 成员（按声明顺序）→ 派生类
- 析构顺序是构造的**严格逆序**
- 成员初始化列表的顺序不影响实际初始化顺序
- 全局对象的初始化顺序在不同翻译单元中未定义
- 构造函数中调用虚函数不会触发动态分派（对象未完全构造）
