# 构造函数基础

## 一、概念说明

构造函数在对象创建时自动调用，负责初始化对象。C++支持多种构造函数：默认构造、参数化构造、拷贝构造、移动构造。**初始化列表**是构造函数中初始化成员的推荐方式。

## 二、具体用法

### 2.1 默认构造与参数化构造

```cpp
class Student {
    std::string name;
    int age;
public:
    // 默认构造函数
    Student() : name("Unknown"), age(0) {}

    // 参数化构造函数
    Student(const std::string& n, int a) : name(n), age(a) {}

    void display() const {
        std::cout << name << ", " << age << "岁" << std::endl;
    }
};

Student s1;                // 默认构造
Student s2("Alice", 20);   // 参数化构造
s1.display();  // 输出: Unknown, 0岁
s2.display();  // 输出: Alice, 20岁
```

### 2.2 初始化列表

```cpp
class Point {
    const int x;  // const成员必须在初始化列表中初始化
    int& ref;     // 引用成员必须在初始化列表中初始化
    int y;
public:
    // 初始化列表：在构造函数体执行前初始化成员
    Point(int x, int& ref, int y)
        : x(x), ref(ref), y(y) {
        // 构造函数体（此时成员已初始化）
    }
};

int shared = 100;
Point p(1, shared, 3);
```

### 2.3 成员初始化顺序

```cpp
class Order {
    int a;
    int b;
public:
    // 初始化顺序由成员声明顺序决定，不是初始化列表顺序
    // a先于b声明，所以a先初始化
    Order(int val) : b(val), a(b) {}
    // a初始化时b尚未初始化！a的值未定义

    void show() const {
        std::cout << "a=" << a << " b=" << b << std::endl;
    }
};
```

### 2.4 explicit防止隐式转换

```cpp
class Wrapper {
    int value;
public:
    explicit Wrapper(int v) : value(v) {}
};

void func(Wrapper w) { }

// Wrapper w = 42;  // 编译错误：explicit阻止隐式转换
Wrapper w(42);       // OK：显式构造
// func(42);         // 编译错误
func(Wrapper(42));   // OK
```

## 三、注意事项与常见陷阱

- 成员初始化顺序由**声明顺序**决定，不是初始化列表顺序
- `const`和引用成员**必须**在初始化列表中初始化
- 如果定义了任何构造函数，编译器不会自动生成默认构造函数
- 委托构造函数（C++11）可避免重复代码
- 构造函数不能声明为`const`、`virtual`或`static`
