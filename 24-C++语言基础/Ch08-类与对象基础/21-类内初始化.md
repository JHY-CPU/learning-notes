# 类内初始化

## 一、概念说明

C++11允许在类内声明数据成员时提供**默认初始化值**（Default Member Initializer，C++11标准 §9.2）。如果构造函数的初始化列表未初始化该成员，则使用类内初始值。

优先级规则：**构造函数初始化列表 > 类内默认值 > 默认初始化**

## 二、具体用法

### 2.1 基本类内初始化

```cpp
#include <iostream>
#include <string>
using namespace std;

class Config {
    int port = 8080;
    string host = "localhost";
    bool debug = false;
    double timeout = 30.0;

public:
    Config() = default;

    Config(int p) : port(p) {}  // 只覆盖port，其余用类内默认值

    Config(int p, const string& h, bool d)
        : port(p), host(h), debug(d) {}

    void show() const {
        cout << host << ":" << port
             << " debug=" << debug
             << " timeout=" << timeout << endl;
    }
};

int main() {
    Config c1;
    c1.show();  // localhost:8080 debug=0 timeout=30

    Config c2(3000);
    c2.show();  // localhost:3000 debug=0 timeout=30

    Config c2(3000, "example.com", true);
    c3.show();  // example.com:3000 debug=1 timeout=30

    return 0;
}
```

### 2.2 与初始化列表的优先级

```cpp
#include <iostream>
using namespace std;

class Example {
    int a = 1;   // 类内默认值
    int b = 2;
    int c = 3;

public:
    // 初始化列表优先于类内初始化
    Example() : a(10) {}
    // 结果：a = 10 (初始化列表), b = 2 (类内), c = 3 (类内)

    Example(int v) : a(v), b(v * 2) {}
    // 结果：a = v, b = v*2, c = 3 (类内)

    void show() const {
        cout << "a=" << a << " b=" << b << " c=" << c << endl;
    }
};

int main() {
    Example e1;
    e1.show();  // a=10 b=2 c=3

    Example e2(5);
    e2.show();  // a=5 b=10 c=3

    return 0;
}
```

### 2.3 花括号和等号初始化

```cpp
#include <iostream>
#include <vector>
#include <array>
using namespace std;

class Container {
    // 等号初始化
    int count = 0;

    // 花括号初始化（推荐用于容器）
    vector<int> data = {1, 2, 3};

    // 圆括号初始化（注意歧义！）
    string name{"default"};  // 花括号OK

    // string name("default");  // 类内不能用圆括号初始化非静态成员（歧义为函数声明）

    // array初始化
    array<int, 3> scores = {90, 85, 92};

public:
    void show() const {
        cout << "name=" << name << " count=" << count << endl;
        cout << "data: ";
        for (int d : data) cout << d << " ";
        cout << endl;
    }
};

int main() {
    Container c;
    c.show();

    return 0;
}
```

### 2.4 static成员的类内初始化

```cpp
#include <iostream>
using namespace std;

class Counter {
public:
    // C++17 inline static：可以在类内初始化
    static inline int totalCount = 0;
    static inline const int MAX_INSTANCES = 1000;

    // C++17之前需要类外定义：
    // class内声明：static int totalCount;
    // 类外定义：int Counter::totalCount = 0;

    Counter() { ++totalCount; }
    ~Counter() { --totalCount; }
};

int main() {
    Counter a, b, c;
    cout << "实例数: " << Counter::totalCount << endl;  // 3
    cout << "最大实例: " << Counter::MAX_INSTANCES << endl;  // 1000

    return 0;
}
```

### 2.5 初始化顺序

```cpp
#include <iostream>
using namespace std;

class OrderDemo {
    // 初始化顺序由声明顺序决定，不是书写顺序！
    int x;
    int y;
    int z;

public:
    // 即使写成 y(2), x(1), z(y + 1) 也没用
    // 实际按声明顺序：x → y → z
    OrderDemo() : z(3), y(2), x(1) {}

    // 危险示例：z用y初始化，但y可能还未初始化
    // class Bad {
    //     int y = z + 1;  // z尚未初始化！UB
    //     int z = 10;
    // };

    void show() const {
        cout << "x=" << x << " y=" << y << " z=" << z << endl;
    }
};

int main() {
    OrderDemo d;
    d.show();  // x=1 y=2 z=3

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **类内初始化不能使用圆括号初始化非静态成员**：与函数声明歧义，用 `int x{42}` 或 `int x = 42`
2. **初始化顺序由成员声明顺序决定**：不是类内初始值的书写顺序，也不是初始化列表的顺序
3. **const和引用成员可在类内初始化**：C++11起支持
4. **类内初始化简化了多构造函数的代码**：避免重复初始化共同成员
5. **注意成员初始化顺序依赖**：一个成员的初始值依赖另一个成员时，必须确保被依赖者先声明
6. **聚合初始化与类内初始化不兼容**：有类内初始值的类不是聚合类型（C++14前）
