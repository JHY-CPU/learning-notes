# 静态成员变量

## 一、概念说明

静态数据成员属于**类本身**而非某个对象，所有对象共享同一份静态成员。它在类内声明、类外定义（分配存储），且只有一个实例。

## 二、具体用法

### 2.1 基本静态成员

```cpp
class Counter {
public:
    static int count;  // 类内声明
    Counter() { ++count; }
    ~Counter() { --count; }
};

// 类外定义（分配存储）
int Counter::count = 0;

Counter a, b, c;
std::cout << "对象数: " << Counter::count << std::endl;  // 输出: 对象数: 3
// 也可以通过对象访问：a.count
```

### 2.2 static constexpr（C++17起可类内初始化）

```cpp
class Config {
public:
    static constexpr int MaxSize = 1024;
    static constexpr double Pi = 3.14159265;
    static constexpr const char* Name = "MyApp";
};

std::cout << Config::MaxSize << std::endl;  // 输出: 1024
std::cout << Config::Name << std::endl;     // 输出: MyApp
```

### 2.3 单例模式中的应用

```cpp
class Singleton {
    static Singleton* instance;
    Singleton() {}

public:
    static Singleton* getInstance() {
        if (!instance) {
            instance = new Singleton();
        }
        return instance;
    }
};

Singleton* Singleton::instance = nullptr;
auto s = Singleton::getInstance();
```

### 2.4 计数与追踪

```cpp
class Widget {
    static int nextId;
    int id;
public:
    Widget() : id(nextId++) {}
    int getId() const { return id; }
};

int Widget::nextId = 0;

Widget w1, w2, w3;
std::cout << w1.getId() << " " << w2.getId() << " " << w3.getId() << std::endl;
// 输出: 0 1 2
```

## 三、注意事项与常见陷阱

- 静态成员**必须**在类外定义（C++17的`inline static`除外）
- `static constexpr`可在类内初始化，不需要类外定义
- 静态成员不属于任何对象，没有`this`指针
- 静态成员的初始化顺序在不同翻译单元中未定义
- 头文件中的静态变量可能导致多个实例（用`inline static`解决，C++17）
