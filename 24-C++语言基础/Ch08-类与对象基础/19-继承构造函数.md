# 继承构造函数

## 一、概念说明

C++11的`using`声明可以将基类的构造函数引入派生类，避免在派生类中重复编写转发构造函数。派生类会生成与基类构造函数签名匹配的构造函数。

## 二、具体用法

### 2.1 基本继承构造

```cpp
class Base {
public:
    Base() { std::cout << "Base()\n"; }
    Base(int x) { std::cout << "Base(" << x << ")\n"; }
    Base(int x, int y) { std::cout << "Base(" << x << "," << y << ")\n"; }
};

class Derived : public Base {
public:
    using Base::Base;  // 继承Base的所有构造函数
};

Derived d1;          // 输出: Base()
Derived d2(42);      // 输出: Base(42)
Derived d3(1, 2);    // 输出: Base(1,2)
```

### 2.2 继承构造与成员初始化

```cpp
class Base {
public:
    Base(int x) { std::cout << "Base(" << x << ")\n"; }
};

class Derived : public Base {
    int extra;
public:
    using Base::Base;  // 继承Base(int)
    // extra使用默认初始化（int为0）
};

Derived d(10);  // 输出: Base(10)
// d.extra = 0 (默认初始化)
```

### 2.3 继承构造与覆盖

```cpp
class Base {
public:
    Base() { std::cout << "Base()\n"; }
    Base(int x) { std::cout << "Base(int)\n"; }
};

class Derived : public Base {
    int val;
public:
    using Base::Base;  // 继承Base()和Base(int)

    // 可以同时定义自己的构造函数
    Derived(int x, int y) : Base(x), val(y) {
        std::cout << "Derived(int,int)\n";
    }
};

Derived d1;        // 输出: Base()
Derived d2(42);    // 输出: Base(int)
Derived d3(1, 2);  // 输出: Base(int) Derived(int,int)
```

### 2.4 实际应用

```cpp
class std_exception : public std::exception {
public:
    using std::exception::exception;  // 继承std::exception的构造函数
};

// 简化自定义异常类
class MyError : public std::runtime_error {
    using std::runtime_error::runtime_error;
};
```

## 三、注意事项与常见陷阱

- 继承构造不继承默认参数值
- 继承构造的派生类成员使用默认初始化
- 如果基类构造函数是private，`using`声明无法继承
- 继承构造不影响派生类自己定义的构造函数
- 多继承时只能继承一个基类的构造函数
