# 对象赋值与初始化区别

## 一、概念说明

**初始化**（Initialization）是在创建对象时设定初始值，调用构造函数。**赋值**（Assignment）是修改已有对象的值，调用`operator=`。两者语义不同，性能也不同。

## 二、具体用法

### 2.1 初始化 vs 赋值

```cpp
class Big {
    std::vector<int> data;
public:
    Big() : data(1000000) {
        std::cout << "默认构造\n";
    }
    Big(const Big& other) : data(other.data) {
        std::cout << "拷贝构造\n";
    }
    Big& operator=(const Big& other) {
        data = other.data;
        std::cout << "拷贝赋值\n";
        return *this;
    }
};

// 初始化：调用一次拷贝构造
Big b1;
Big b2 = b1;  // 拷贝构造

// 赋值：先构造再赋值（两次操作）
Big b3;       // 默认构造
b3 = b1;      // 拷贝赋值
```

### 2.2 直接初始化 vs 拷贝初始化

```cpp
class Widget {
public:
    Widget(int) { std::cout << "Widget(int)\n"; }
    Widget(const Widget&) { std::cout << "拷贝构造\n"; }
};

// 直接初始化：调用匹配的构造函数
Widget w1(42);        // Widget(int)

// 拷贝初始化：要求可访问的拷贝构造
Widget w2 = Widget(42);  // Widget(int)，可能优化掉拷贝

// Widget w3 = 42;  // 尝试隐式转换：Widget(int) → Widget
```

### 2.3 列表初始化

```cpp
// 直接列表初始化
std::vector<int> v1{1, 2, 3};   // initializer_list构造

// 拷贝列表初始化
std::vector<int> v2 = {1, 2, 3}; // 同上

// 统一初始化（C++11）
int x = {42};  // 拷贝列表初始化
int y{42};     // 直接列表初始化
```

### 2.4 编译器优化

```cpp
class Obj {
public:
    Obj() { std::cout << "构造\n"; }
    Obj(const Obj&) { std::cout << "拷贝\n"; }
};

Obj create() {
    Obj o;        // 构造
    return o;     // 拷贝（可能被RVO优化掉）
}

Obj result = create();
// 可能只输出: 构造 （RVO优化掉了拷贝）
```

## 三、注意事项与常见陷阱

- 初始化只调用构造函数，赋值先构造再赋值
- `auto x = value;` 是拷贝初始化
- 列表初始化`{}`禁止窄化转换
- 编译器优化（RVO）可能消除额外的拷贝
- `const`和引用成员只能初始化，不能赋值
