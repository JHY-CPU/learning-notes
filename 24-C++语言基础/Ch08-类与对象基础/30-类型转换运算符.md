# 类型转换运算符

## 一、概念说明

类型转换运算符允许自定义类型转换为其他类型。C++11起推荐使用`explicit`转换运算符防止意外的隐式转换。`operator bool()`是最常见的转换运算符。

## 二、具体用法

### 2.1 基本转换运算符

```cpp
class Celsius {
    double temp;
public:
    explicit Celsius(double t) : temp(t) {}

    // 转换为double
    explicit operator double() const { return temp; }

    // 转换为Fahrenheit（自定义类型）
    operator double() const { return temp; }
};

Celsius c(100.0);
double d = static_cast<double>(c);
std::cout << d << std::endl;  // 输出: 100
```

### 2.2 operator bool（最常用）

```cpp
class File {
    FILE* handle;
public:
    File(const char* name) : handle(fopen(name, "r")) {}

    explicit operator bool() const {
        return handle != nullptr;
    }

    ~File() { if (handle) fclose(handle); }
};

File f("test.txt");
if (f) {  // 上文转换（contextual conversion）：OK
    std::cout << "文件打开成功\n";
}

// bool b = f;  // 编译错误：explicit阻止隐式转换
```

### 2.3 标准库中的应用

```cpp
// std::optional的operator bool
std::optional<int> opt = 42;
if (opt) {  // 检查是否有值
    std::cout << *opt << std::endl;  // 输出: 42
}

// std::shared_ptr的operator bool
std::shared_ptr<int> sp = std::make_shared<int>(42);
if (sp) {  // 检查是否非空
    std::cout << *sp << std::endl;  // 输出: 42
}
```

### 2.4 安全的转换

```cpp
class SafeInt {
    int value;
public:
    SafeInt(int v) : value(v) {}

    explicit operator int() const { return value; }
    explicit operator bool() const { return value != 0; }

    // 不提供operator char*等危险转换
};

SafeInt si(42);
int n = static_cast<int>(si);  // OK：显式
// int n2 = si;  // 编译错误：explicit

if (si) std::cout << "非零\n";  // 输出: 非零
```

## 三、注意事项与常见陷阱

- `explicit`转换运算符只在特定上下文中允许隐式转换（if/while/for）
- 提供`operator bool()`时注意避免与算术运算的意外交互
- C++11起`std::istream`的`operator bool()`替代了旧的`operator void*()`
- 转换运算符可以是`const`（推荐）
- 不要提供过多的转换运算符，每个转换都增加复杂性
