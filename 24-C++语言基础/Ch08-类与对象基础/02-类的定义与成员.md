# 类的定义与成员

## 一、概念说明

类的成员包括**数据成员**（存储状态）和**成员函数**（定义行为）。成员函数可以在类内定义（自动inline），也可以在类外定义。

## 二、具体用法

### 2.1 完整类定义

```cpp
class Rectangle {
    // 数据成员
    double width;
    double height;

public:
    // 构造函数
    Rectangle(double w, double h) : width(w), height(h) {}

    // 成员函数
    double area() const {
        return width * height;
    }

    double perimeter() const {
        return 2 * (width + height);
    }

    void scale(double factor) {
        width *= factor;
        height *= factor;
    }
};

Rectangle rect(3.0, 4.0);
std::cout << "面积: " << rect.area() << std::endl;        // 输出: 面积: 12
std::cout << "周长: " << rect.perimeter() << std::endl;   // 输出: 周长: 14
```

### 2.2 类内定义 vs 类外定义

```cpp
class Circle {
    double radius;
public:
    Circle(double r);  // 类内声明

    // 类内定义：自动inline
    double getRadius() const { return radius; }
};

// 类外定义
Circle::Circle(double r) : radius(r) {}

double Circle::area() const {
    return 3.14159 * radius * radius;
}
```

### 2.3 数据成员初始化

```cpp
class Config {
    // C++11类内初始化
    int maxRetries = 3;
    double timeout = 30.0;
    std::string name = "default";

public:
    // 构造函数初始化列表优先于类内初始化
    Config(const std::string& n) : name(n) {}
};

Config cfg("myConfig");
std::cout << cfg.name << std::endl;  // 输出: myConfig
```

### 2.4 成员类型

```cpp
class Container {
public:
    // 类型别名
    using ValueType = int;
    using Iterator = int*;

    // 静态成员
    static constexpr int MaxSize = 1000;

    // 嵌套类
    class Node {
        ValueType data;
        Node* next;
    };
};
```

## 三、注意事项与常见陷阱

- 数据成员声明顺序决定了初始化顺序（非初始化列表顺序）
- 类内初始化的成员在构造函数初始化列表之后应用
- 成员函数中可直接访问同类其他成员（包括private）
- 空类的大小至少为1字节（保证不同对象地址不同）
- `mutable`成员可在const成员函数中修改
