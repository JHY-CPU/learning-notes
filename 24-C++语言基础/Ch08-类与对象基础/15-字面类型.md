# 字面类型

## 一、概念说明

字面类型（Literal Type）是可以在编译期上下文中使用的类型，允许`constexpr`函数和对象使用。字面类型的要求包括：所有非静态数据成员和基类必须是字面类型，必须有constexpr析构函数（C++20）。

## 二、具体用法

### 2.1 constexpr构造函数

```cpp
class Point {
    double x_, y_;
public:
    constexpr Point(double x, double y) : x_(x), y_(y) {}
    constexpr double x() const { return x_; }
    constexpr double y() const { return y_; }
    constexpr double distance() const {
        return x_ * x_ + y_ * y_;
    }
};

constexpr Point origin(0, 0);
constexpr Point p(3.0, 4.0);

static_assert(p.distance() == 25.0, "distance should be 25");
std::cout << p.distance() << std::endl;  // 输出: 25
```

### 2.2 编译期对象操作

```cpp
class Rational {
    int num, den;
public:
    constexpr Rational(int n, int d = 1) : num(n), den(d) {}

    constexpr Rational operator+(const Rational& other) const {
        return Rational(num * other.den + other.num * den,
                        den * other.den);
    }

    constexpr double value() const {
        return static_cast<double>(num) / den;
    }
};

constexpr Rational r1(1, 2);
constexpr Rational r2(1, 3);
constexpr Rational sum = r1 + r2;

static_assert(sum.value() > 0.8 && sum.value() < 0.9, "1/2+1/3=5/6");
```

### 2.3 编译期数组

```cpp
template <size_t N>
class FixedArray {
    int data[N] = {};
public:
    constexpr FixedArray() = default;
    constexpr int& operator[](size_t i) { return data[i]; }
    constexpr const int& operator[](size_t i) const { return data[i]; }
    constexpr size_t size() const { return N; }
};

constexpr FixedArray<5> makeSequence() {
    FixedArray<5> arr;
    for (size_t i = 0; i < arr.size(); ++i) {
        arr[i] = static_cast<int>(i + 1);
    }
    return arr;
}

constexpr auto seq = makeSequence();
static_assert(seq[0] == 1 && seq[4] == 5, "sequence check");
```

### 2.4 C++20 constexpr改进

```cpp
// C++20放宽：允许虚函数、union、try-catch等在constexpr中使用
class C20Class {
public:
    constexpr C20Class(int v) : val(v) {}
    constexpr ~C20Class() {}  // C++20: constexpr析构函数

private:
    int val;
};
```

## 三、注意事项与常见陷阱

- constexpr对象的所有成员必须是constexpr可构造的
- C++20要求字面类型有constexpr析构函数
- constexpr构造函数不保证对象在编译期构造（取决于上下文）
- 编译期对象不能使用动态内存（C++20之前）
- 字面类型是编译期计算的基础
