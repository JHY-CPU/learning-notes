# 类与对象概述

## 一、概念说明

**类**（Class）是C++面向对象编程的核心（C++11标准 §9），是用户自定义类型的蓝图，封装了**数据成员**（属性）和**成员函数**（方法）。**对象**（Object）是类的实例。

OOP四大特性：
- **封装**：隐藏实现细节，暴露接口
- **继承**：复用已有代码，建立类型层次
- **多态**：统一接口，不同实现
- **抽象**：提取共同特征，忽略无关细节

## 二、基本概念

### 2.1 类的定义与实例化

```cpp
#include <iostream>
#include <string>
using namespace std;

class Dog {
public:
    string name;
    int age;

    void bark() const {
        cout << name << " says: Woof!" << endl;
    }
};

int main() {
    Dog myDog;           // 实例化对象（栈上）
    myDog.name = "Buddy";
    myDog.age = 3;
    myDog.bark();        // Buddy says: Woof!

    // 堆上实例化
    Dog* heapDog = new Dog{"Rex", 5};
    heapDog->bark();     // Rex says: Woof!
    delete heapDog;

    return 0;
}
```

### 2.2 封装与访问控制

```cpp
#include <iostream>
using namespace std;

class BankAccount {
private:
    double balance;         // 私有数据：外部不可直接访问

protected:
    string ownerName;       // 受保护：子类可访问

public:
    BankAccount(const string& name, double initial)
        : ownerName(name), balance(initial) {}

    void deposit(double amount) {
        if (amount > 0) balance += amount;
    }

    bool withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            return true;
        }
        return false;
    }

    double getBalance() const { return balance; }

    void display() const {
        cout << ownerName << " 余额: " << balance << endl;
    }
};

int main() {
    BankAccount acc("Alice", 1000);
    acc.deposit(500);
    acc.withdraw(200);
    acc.display();  // Alice 余额: 1300

    // acc.balance = 999999;  // 编译错误：balance是private

    return 0;
}
```

### 2.3 this指针

```cpp
class Rectangle {
    double width, height;
public:
    Rectangle(double w, double h) : width(w), height(h) {}

    // this指针：指向当前对象
    double area() const { return width * height; }

    // 返回*this支持链式调用
    Rectangle& setWidth(double w) {
        width = w;
        return *this;
    }

    Rectangle& setHeight(double h) {
        height = h;
        return *this;
    }

    bool isSquare() const {
        return width == height;
    }
};

int main() {
    Rectangle r(3, 4);
    r.setWidth(5).setHeight(5);  // 链式调用
    cout << "面积: " << r.area() << endl;    // 25
    cout << "正方形: " << r.isSquare() << endl;  // 1

    return 0;
}
```

### 2.4 const成员函数

```cpp
class DataHolder {
    int value;
public:
    DataHolder(int v) : value(v) {}

    // const成员函数：不能修改成员变量
    int getValue() const {
        // value = 10;  // 编译错误
        return value;
    }

    // 非const成员函数：可以修改
    void setValue(int v) {
        value = v;
    }

    // mutable成员：即使在const函数中也可修改
    mutable int accessCount = 0;
    int getValueCounted() const {
        ++accessCount;  // OK：mutable
        return value;
    }
};

void print(const DataHolder& dh) {
    // dh.setValue(10);  // 编译错误：const引用只能调用const函数
    cout << dh.getValue() << endl;  // OK
}
```

## 三、本章内容概览

- **基础篇**：类定义、访问控制（public/private/protected）、this指针、构造/析构函数
- **特殊成员**：拷贝构造/赋值运算符、静态成员、友元函数/类
- **高级特性**：委托构造函数、explicit防止隐式转换、default/delete
- **设计原则**：不变式（invariant）、PIMPL模式、SOLID原则
- **模板与多态**：类模板、CRTP、值语义 vs 引用语义

## 四、注意事项与常见陷阱

1. **类定义以分号结尾**：`class Foo { };`（容易忘记）
2. **成员函数在类内定义默认inline**：类外定义则不是
3. **const对象只能调用const成员函数**：非const对象可以调用两种
4. **mutable成员在const函数中也可修改**：用于缓存计数等场景
5. **类的访问控制是编译期检查**：不影响运行时性能
