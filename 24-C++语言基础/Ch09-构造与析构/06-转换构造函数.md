# 转换构造函数

## 一、概念说明

**转换构造函数**（Converting Constructor）是能用单个参数调用的构造函数，它定义了从参数类型到类类型的**隐式转换**规则。使用`explicit`关键字可以禁止隐式转换，只允许显式转换。

### 1.1 隐式转换的危险

隐式转换可能导致意外的类型转换，产生难以发现的bug。

## 二、具体用法

### 2.1 隐式转换

```cpp
#include <iostream>

class Fraction {
private:
    int numerator;
    int denominator;
public:
    // 单参数构造函数：允许隐式转换 int -> Fraction
    Fraction(int num, int den = 1) : numerator(num), denominator(den) {
        if (den == 0) throw std::invalid_argument("分母不能为0");
    }

    void display() const {
        std::cout << numerator << "/" << denominator << std::endl;
    }
};

void printFraction(Fraction f) {
    f.display();
}

int main() {
    Fraction f1 = 5;    // 隐式转换：5 -> Fraction(5, 1)
    f1.display();

    printFraction(10);  // 隐式转换：10 -> Fraction(10, 1)

    Fraction f2(3, 4);  // 直接初始化，不是隐式转换
    f2.display();
    return 0;
}
```

**输出：**
```
5/1
10/1
3/4
```

### 2.2 explicit禁止隐式转换

```cpp
#include <iostream>

class Meter {
private:
    double value;
public:
    explicit Meter(double v) : value(v) {}

    double getValue() const { return value; }

    friend Meter operator+(const Meter& a, const Meter& b) {
        return Meter(a.value + b.value);
    }
};

void measure(Meter m) {
    std::cout << m.getValue() << " 米" << std::endl;
}

int main() {
    Meter m1(5.0);          // OK：直接初始化
    Meter m2 = Meter(3.0);  // OK：显式转换
    // Meter m3 = 2.0;      // 编译错误！禁止隐式转换
    // measure(10.0);        // 编译错误！禁止隐式转换
    // Meter m4 = m1 + 4.0; // 编译错误！

    measure(Meter(10.0));   // OK：显式转换
    Meter m5 = m1 + Meter(4.0);  // OK
    std::cout << "m5 = " << m5.getValue() << " 米" << std::endl;
    return 0;
}
```

**输出：**
```
m5 = 9 米
```

### 2.3 多参数构造与聚合初始化

```cpp
#include <iostream>

class Point {
public:
    double x, y;
    // C++11: explicit多参数构造
    explicit Point(double x = 0, double y = 0) : x(x), y(y) {}
};

int main() {
    Point p1(1.0, 2.0);          // OK
    // Point p2 = {1.0, 2.0};    // 编译错误！explicit阻止
    Point p3 = Point(3.0, 4.0);  // OK
    std::cout << "p3: (" << p3.x << ", " << p3.y << ")" << std::endl;
    return 0;
}
```

**输出：**
```
p3: (3, 4)
```

## 三、注意事项与常见陷阱

- **建议所有单参数构造函数都加`explicit`**，除非你确实需要隐式转换
- 隐式转换链不被允许（A可转B，B可转C，但A不能一步转C）
- 标准库中`std::string(const char*)`是非explicit的，允许隐式转换
- `explicit`也可用于类型转换运算符（`explicit operator T()`）
- 使用`static_cast<Target>(val)`或`Target(val)`进行显式转换
