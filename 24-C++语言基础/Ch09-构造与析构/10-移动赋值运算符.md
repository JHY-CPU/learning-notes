# 移动赋值运算符

## 一、概念说明

**移动赋值运算符**（Move Assignment Operator）用于将一个右值对象的资源转移给已有对象。与移动构造类似，它"窃取"源对象的资源而非深拷贝，但需要处理**自赋值检查**和**旧资源释放**。

## 二、具体用法

### 2.1 基本实现

```cpp
#include <iostream>
#include <algorithm>
#include <utility>

class Buffer {
private:
    int* data;
    int size;
public:
    Buffer(int s) : data(new int[s]()), size(s) {}

    Buffer(Buffer&& other) noexcept : data(other.data), size(other.size) {
        other.data = nullptr;
        other.size = 0;
    }

    // 移动赋值运算符
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {       // 自赋值检查
            delete[] data;          // 释放旧资源
            data = other.data;      // 转移资源
            size = other.size;
            other.data = nullptr;   // 源对象置空
            other.size = 0;
        }
        return *this;
    }

    ~Buffer() { delete[] data; }

    void fill(int val) { std::fill(data, data + size, val); }
    int getSize() const { return size; }
};

int main() {
    Buffer a(10), b(5);
    a.fill(1); b.fill(2);

    std::cout << "移动前: a.size=" << a.getSize() << ", b.size=" << b.getSize() << std::endl;

    a = std::move(b);  // 移动赋值：b的资源转移给a

    std::cout << "移动后: a.size=" << a.getSize() << ", b.size=" << b.getSize() << std::endl;
    return 0;
}
```

**输出：**
```
移动前: a.size=10, b.size=5
移动后: a.size=5, b.size=0
```

### 2.2 移动赋值的swap实现

```cpp
#include <iostream>
#include <utility>

class Container {
private:
    int* ptr;
    int size;
public:
    Container(int s) : ptr(new int[s]()), size(s) {}

    Container(Container&& other) noexcept : ptr(other.ptr), size(other.size) {
        other.ptr = nullptr;
        other.size = 0;
    }

    // 使用swap实现移动赋值
    Container& operator=(Container&& other) noexcept {
        Container temp(std::move(other));  // 移动构造临时对象
        std::swap(ptr, temp.ptr);         // 交换资源
        std::swap(size, temp.size);
        return *this;                     // temp析构释放旧资源
    }

    ~Container() { delete[] ptr; }
    int getSize() const { return size; }
};

int main() {
    Container c1(100), c2(50);
    c1 = std::move(c2);
    std::cout << "c1.size=" << c1.getSize() << std::endl;
    return 0;
}
```

**输出：**
```
c1.size=50
```

## 三、注意事项与常见陷阱

- 移动赋值必须处理**自赋值**（虽然`std::move(x)`不应等于`&x`，但防御性编程仍需检查）
- 移动赋值必须标记`noexcept`
- 移动赋值后源对象处于有效但未指定状态
- 移动赋值与移动构造通常一起定义（Rule of Five）
- 不要忘记释放被赋值对象的旧资源
