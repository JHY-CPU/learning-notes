# 默认构造函数

## 一、概念说明

**默认构造函数**（Default Constructor）是不接受任何参数（或所有参数都有默认值）的构造函数。当创建对象不指定初始值时，编译器会自动调用默认构造函数。

### 1.1 编译器合成默认构造函数

如果用户没有定义**任何**构造函数，编译器会自动合成一个默认构造函数。但合成的默认构造函数执行以下操作：
- 调用基类的默认构造函数（如果有）
- 调用成员变量的默认构造函数（如果是类类型）
- **不初始化**内置类型的成员（int、double等保持未定义值）

### 1.2 =default与=delete

C++11引入了`=default`显式要求编译器生成默认实现，`=delete`显式禁止生成。

## 二、具体用法

### 2.1 编译器合成的默认构造函数

```cpp
#include <iostream>
#include <string>

class Student {
public:
    std::string name;  // string有默认构造，会被初始化为空串
    int age;           // 内置类型，未初始化（未定义值！）
    // 编译器合成的默认构造函数不会初始化age
};

int main() {
    Student s;  // 调用合成的默认构造函数
    std::cout << "name: \"" << s.name << "\"" << std::endl;
    std::cout << "age: " << s.age << std::endl;  // 未定义行为
    return 0;
}
```

**输出（示例）：**
```
name: ""
age: -858993460  // 未定义值，每次运行可能不同
```

### 2.2 显式定义默认构造函数

```cpp
class Rectangle {
private:
    double width = 1.0;   // C++11类内初始值
    double height = 1.0;
public:
    // 方式一：手动定义
    Rectangle() {
        std::cout << "手动默认构造" << std::endl;
    }
};

class Circle {
private:
    double radius;
public:
    // 方式二：=default（推荐，编译器生成最高效实现）
    Circle() = default;

    Circle(double r) : radius(r) {}

    double area() const { return 3.14159 * radius * radius; }
};

int main() {
    Rectangle r;
    Circle c1;       // 调用默认构造
    Circle c2(5.0);  // 调用参数化构造
    std::cout << "c2面积: " << c2.area() << std::endl;
    return 0;
}
```

**输出：**
```
手动默认构造
c2面积: 78.5398
```

### 2.3 禁用默认构造函数

```cpp
class NoDefault {
public:
    NoDefault() = delete;             // 禁止默认构造
    NoDefault(int v) : value(v) {}    // 只允许带参构造

private:
    int value;
};

int main() {
    // NoDefault nd;  // 编译错误！默认构造被删除
    NoDefault nd(42);  // OK
    return 0;
}
```

## 三、注意事项与常见陷阱

- **一旦定义了任何构造函数**，编译器就不再自动生成默认构造函数
- 使用`=default`可以在类内或类外声明，类外定义需写`ClassName() = default;`
- 内置类型成员不会被默认初始化，建议使用类内初始值`int x = 0;`
- `=default`比手动写空函数体更高效，编译器可能将其视为trivial构造
- 合成的默认构造函数只初始化**类类型**成员和**有类内初始值**的成员
