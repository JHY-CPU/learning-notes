# 移动构造函数

## 一、概念说明

**移动构造函数**（Move Constructor）是C++11引入的特性，用于"窃取"临时对象（右值）的资源，而非进行深拷贝。移动后源对象处于**有效但未指定**的状态，通常资源被转移为空状态。

### 1.1 右值引用

移动构造函数的参数是**右值引用**（`ClassName&&`），它绑定到临时对象或被`std::move`显式转换的对象。

### 1.2 noexcept的重要性

移动构造函数应标记为`noexcept`，因为标准库容器（如`std::vector`）在扩容时只有在移动构造是`noexcept`的情况下才会使用移动而非拷贝。

## 二、具体用法

### 2.1 基本移动构造

```cpp
#include <iostream>
#include <algorithm>
#include <utility>

class Buffer {
private:
    int* data;
    int size;
public:
    Buffer(int s) : data(new int[s]()), size(s) {
        std::cout << "构造: size=" << size << std::endl;
    }

    // 拷贝构造（深拷贝）
    Buffer(const Buffer& other) : data(new int[other.size]), size(other.size) {
        std::copy(other.data, other.data + size, data);
        std::cout << "拷贝构造: size=" << size << std::endl;
    }

    // 移动构造（资源转移）
    Buffer(Buffer&& other) noexcept
        : data(other.data), size(other.size) {
        other.data = nullptr;  // 源对象置空，防止析构时释放
        other.size = 0;
        std::cout << "移动构造: size=" << size << std::endl;
    }

    ~Buffer() {
        delete[] data;
        std::cout << "析构: size=" << size << std::endl;
    }
};

int main() {
    std::cout << "--- 1. 直接构造 ---" << std::endl;
    Buffer b1(100);

    std::cout << "\n--- 2. 拷贝构造 ---" << std::endl;
    Buffer b2 = b1;

    std::cout << "\n--- 3. 移动构造 ---" << std::endl;
    Buffer b3 = std::move(b1);  // b1资源被转移

    std::cout << "\n--- 结束 ---" << std::endl;
    return 0;
}
```

**输出：**
```
--- 1. 直接构造 ---
构造: size=100

--- 2. 拷贝构造 ---
拷贝构造: size=100

--- 3. 移动构造 ---
移动构造: size=100

--- 结束 ---
析构: size=0
析构: size=100
析构: size=100
```

### 2.2 移动返回局部对象

```cpp
#include <iostream>
#include <utility>
#include <string>

class Document {
private:
    std::string content;
public:
    Document(const std::string& c) : content(c) {
        std::cout << "普通构造" << std::endl;
    }
    Document(Document&& other) noexcept : content(std::move(other.content)) {
        std::cout << "移动构造" << std::endl;
    }
    Document(const Document& other) : content(other.content) {
        std::cout << "拷贝构造" << std::endl;
    }

    const std::string& getContent() const { return content; }
};

Document createDocument() {
    Document doc("Hello, World!");
    return doc;  // C++17强制拷贝消除，不会调用移动/拷贝构造
}

int main() {
    Document d = createDocument();
    std::cout << "内容: " << d.getContent() << std::endl;
    return 0;
}
```

**输出：**
```
普通构造
内容: Hello, World!
```

## 三、注意事项与常见陷阱

- 移动后源对象必须处于**有效但未指定**状态，保证可以安全析构
- 移动构造必须标记`noexcept`以配合容器优化
- `std::move`不移动任何东西，只是将左值转为右值引用
- 移动后不要对源对象做任何假设（除析构和重新赋值外）
- 基本类型（int等）没有移动语义，移动等价于拷贝
