# 完美转发构造

## 一、概念说明

**完美转发构造**（Perfect Forwarding Constructor）使用可变参数模板和完美转发，将参数原封不动地传递给成员的构造函数。这使得一个构造函数可以接受任意类型和数量的参数，适用于**包装类**和**工厂类**。

### 1.1 核心语法

- `Args&&... args`：可变参数模板 + 万能引用
- `std::forward<Args>(args)...`：完美转发，保持左值/右值属性

## 二、具体用法

### 2.1 基本完美转发构造

```cpp
#include <iostream>
#include <string>
#include <utility>

class Person {
    std::string name;
    int age;
public:
    // 完美转发构造：接受任意参数组合
    template<typename... Args>
    explicit Person(Args&&... args)
        : name(std::forward<Args>(args)...) {}

    // 多成员的完美转发需要多个构造函数或tuple技巧
    void display() const {
        std::cout << "姓名: " << name << std::endl;
    }
};

int main() {
    Person p1("张三");           // const char* → string
    std::string s = "李四";
    Person p2(s);               // 左值string
    Person p3(std::move(s));    // 右值string
    p1.display(); p2.display(); p3.display();
    return 0;
}
```

**输出：**
```
姓名: 张三
姓名: 李四
姓名: 李四
```

### 2.2 emplace风格构造

```cpp
#include <iostream>
#include <tuple>
#include <utility>
#include <string>

class Widget {
    std::string name;
    int value;
public:
    // 使用tuple存储多组转发参数
    Widget(const std::string& n, int v) : name(n), value(v) {
        std::cout << "构造: " << name << "=" << value << std::endl;
    }
};

template<typename T, typename... Args>
T* constructInPlace(void* location, Args&&... args) {
    return new (location) T(std::forward<Args>(args)...);
}

int main() {
    alignas(Widget) char buffer[sizeof(Widget)];
    Widget* w = constructInPlace<Widget>(buffer, "测试", 42);
    std::cout << w->name << std::endl;
    w->~Widget();
    return 0;
}
```

**输出：**
```
构造: 测试=42
测试
```

## 三、注意事项与常见陷阱

- 完美转发构造可能与拷贝/移动构造产生冲突，需用SFINAE或`requires`约束
- 使用`std::enable_if`或C++20 concepts限制模板构造的适用范围
- 万能引用`T&&`在模板中才能实现完美转发
- `std::forward`必须显式指定模板参数：`std::Forward<T>(arg)`
- `emplace_back`等容器方法内部使用的就是完美转发构造
