# 参数化构造函数

## 一、概念说明

**参数化构造函数**是接受一个或多个参数的构造函数，用于在创建对象时提供初始值。通过不同的参数组合，可以实现**构造函数重载**，为对象提供多种初始化方式。

### 1.1 为什么要参数化构造

默认构造函数无法提供有意义的初始状态，参数化构造函数允许我们在创建对象时指定关键属性值，保证对象从创建起就处于有效状态。

## 二、具体用法

### 2.1 单参数构造函数

```cpp
#include <iostream>
#include <string>

class Counter {
private:
    int count;
public:
    explicit Counter(int c) : count(c) {  // explicit防止隐式转换
        std::cout << "Counter构造: count = " << count << std::endl;
    }
    int getCount() const { return count; }
};

int main() {
    Counter c1(10);        // 直接初始化
    Counter c2 = Counter(20);  // 拷贝初始化
    // Counter c3 = 30;    // 编译错误：explicit禁止隐式转换
    std::cout << "c1: " << c1.getCount() << std::endl;
    std::cout << "c2: " << c2.getCount() << std::endl;
    return 0;
}
```

**输出：**
```
Counter构造: count = 10
Counter构造: count = 20
c1: 10
c2: 20
```

### 2.2 多参数构造函数

```cpp
#include <iostream>
#include <string>
#include <cmath>

class Student {
private:
    std::string name;
    int age;
    double score;
public:
    // 带默认参数值的构造函数
    Student(const std::string& n = "未知", int a = 18, double s = 0.0)
        : name(n), age(a), score(s) {}

    void display() const {
        std::cout << name << ", " << age << "岁, 成绩: " << score << std::endl;
    }
};

class Vector2D {
private:
    double x, y;
public:
    Vector2D(double x = 0.0, double y = 0.0) : x(x), y(y) {}

    double length() const { return std::sqrt(x * x + y * y); }

    friend std::ostream& operator<<(std::ostream& os, const Vector2D& v) {
        return os << "(" << v.x << ", " << v.y << ")";
    }
};

int main() {
    Student s1;                        // 全部使用默认值
    Student s2("张三", 20, 95.5);       // 全部指定
    Student s3("李四");                 // 只指定姓名

    s1.display(); s2.display(); s3.display();

    Vector2D v1(3.0, 4.0);
    std::cout << v1 << " 长度: " << v1.length() << std::endl;
    return 0;
}
```

**输出：**
```
未知, 18岁, 成绩: 0
张三, 20岁, 成绩: 95.5
李四, 18岁, 成绩: 0
(3, 4) 长度: 5
```

### 2.3 使用列表初始化（C++11）

```cpp
#include <initializer_list>
#include <iostream>

class IntArray {
private:
    int* data;
    int size;
public:
    // 使用initializer_list构造
    IntArray(std::initializer_list<int> list)
        : data(new int[list.size()]), size(list.size()) {
        int i = 0;
        for (auto& val : list) {
            data[i++] = val;
        }
    }
    ~IntArray() { delete[] data; }

    void print() const {
        std::cout << "[";
        for (int i = 0; i < size; ++i) {
            if (i > 0) std::cout << ", ";
            std::cout << data[i];
        }
        std::cout << "]" << std::endl;
    }
};

int main() {
    IntArray arr = {1, 2, 3, 4, 5};
    arr.print();
    return 0;
}
```

**输出：**
```
[1, 2, 3, 4, 5]
```

## 三、注意事项与常见陷阱

- 优先使用**成员初始化列表**而非构造函数体内赋值
- `explicit`关键字防止意外的隐式类型转换
- 默认参数值让一个构造函数同时充当默认构造函数的角色
- 使用`initializer_list`支持花括号初始化语法
- 注意：`initializer_list`构造函数优先级很高，可能导致歧义
