# 原子操作进阶

## 一、概念说明

`compare_exchange_weak`和`compare_exchange_strong`实现CAS（Compare-And-Swap），是无锁编程的核心操作。CAS检查当前值是否等于期望值，如果是则设置新值，否则更新期望值为当前值。

- `weak`版本可能伪失败（spurious failure）：即使值相等也可能返回false，性能更好
- `strong`版本不会伪失败：但性能略低

```cpp
#include <iostream>
#include <atomic>

int main() {
    std::atomic<int> val{10};

    int expected = 10;
    int desired = 20;

    // CAS操作
    bool success = val.compare_exchange_strong(expected, desired);
    std::cout << "CAS结果: " << success
              << ", 值: " << val.load() << std::endl;

    // 值不匹配时
    expected = 10; // 期望10，但实际是20
    success = val.compare_exchange_strong(expected, desired);
    std::cout << "CAS结果: " << success
              << ", expected被更新为: " << expected << std::endl;

    return 0;
}
```

**输出：**
```
CAS结果: 1, 值: 20
CAS结果: 0, expected被更新为: 20
```

## 二、具体用法

### 2.1 无锁栈

```cpp
#include <iostream>
#include <atomic>
#include <memory>

template <typename T>
class LockFreeStack {
    struct Node {
        T data;
        Node* next;
        Node(T d) : data(std::move(d)), next(nullptr) {}
    };

    std::atomic<Node*> head{nullptr};

public:
    void push(T item) {
        auto* node = new Node(std::move(item));
        node->next = head.load();
        // CAS循环：如果head还是node->next，就更新为node
        while (!head.compare_exchange_weak(node->next, node)) {
            // 失败时node->next被更新为当前head
        }
    }

    bool pop(T& result) {
        Node* oldHead = head.load();
        while (oldHead && !head.compare_exchange_weak(oldHead, oldHead->next)) {
            // oldHead被更新为当前head
        }
        if (oldHead) {
            result = std::move(oldHead->data);
            delete oldHead;
            return true;
        }
        return false;
    }

    ~LockFreeStack() {
        T item;
        while (pop(item));
    }
};

int main() {
    LockFreeStack<int> stack;
    stack.push(1);
    stack.push(2);
    stack.push(3);

    int val;
    while (stack.pop(val)) {
        std::cout << val << " ";
    }
    std::cout << std::endl;

    return 0;
}
```

**输出：**
```
3 2 1
```

### 2.2 CAS实现自旋锁

```cpp
#include <iostream>
#include <thread>
#include <atomic>

class SpinLock {
    std::atomic_flag flag = ATOMIC_FLAG_INIT;
public:
    void lock() {
        while (flag.test_and_set(std::memory_order_acquire)) {
            // 自旋等待，可加pause指令降低功耗
        }
    }
    void unlock() {
        flag.clear(std::memory_order_release);
    }
};

SpinLock spin;
int shared = 0;

void increment() {
    for (int i = 0; i < 10000; ++i) {
        spin.lock();
        ++shared;
        spin.unlock();
    }
}

int main() {
    std::thread t1(increment);
    std::thread t2(increment);
    t1.join();
    t2.join();
    std::cout << "shared = " << shared << std::endl;
    return 0;
}
```

**输出：**
```
shared = 20000
```

## 三、注意事项与常见陷阱

- **`weak`版本需要循环**：因为可能伪失败。
- **CAS循环应限制次数**：避免极端情况下活锁。
- **ABA问题**：值从A变B再变A，CAS认为没变。用版本号解决。
- **`compare_exchange`自动更新expected**：失败时expected被设为当前值。
- **无锁数据结构很难正确实现**：优先使用mutex，除非性能确实需要。
