# 线程管理

## 一、概念说明

线程管理包括：创建、等待（`join`）、分离（`detach`）、移动、获取ID、获取硬件并发数等。C++的`thread`对象拥有线程所有权，RAII管理可确保线程正确结束。

```cpp
#include <iostream>
#include <thread>
#include <vector>

void worker(int id) {
    std::cout << "线程 " << id << " ID: " << std::this_thread::get_id() << std::endl;
    std::this_thread::sleep_for(std::chrono::milliseconds(50));
}

int main() {
    std::vector<std::thread> threads;

    // 批量创建线程
    for (int i = 0; i < 4; ++i) {
        threads.emplace_back(worker, i);
    }

    std::cout << "硬件并发数: " << std::thread::hardware_concurrency() << std::endl;

    // 等待所有线程
    for (auto& t : threads) {
        if (t.joinable()) t.join();
    }

    std::cout << "所有线程完成" << std::endl;
    return 0;
}
```

**输出（顺序可能不同）：**
```
硬件并发数: 8
线程 0 ID: 140234567890
线程 1 ID: 140234567891
线程 2 ID: 140234567892
线程 3 ID: 140234567893
所有线程完成
```

## 二、具体用法

### 2.1 RAII线程守卫

```cpp
#include <iostream>
#include <thread>
#include <vector>

class ThreadGuard {
    std::thread& t;
public:
    explicit ThreadGuard(std::thread& t_) : t(t_) {}
    ~ThreadGuard() {
        if (t.joinable()) {
            t.join();
            std::cout << "ThreadGuard自动join" << std::endl;
        }
    }
    ThreadGuard(const ThreadGuard&) = delete;
    ThreadGuard& operator=(const ThreadGuard&) = delete;
};

void task() {
    std::this_thread::sleep_for(std::chrono::milliseconds(100));
    std::cout << "任务完成" << std::endl;
}

int main() {
    {
        std::thread t(task);
        ThreadGuard guard(t);
        // 即使这里抛异常，guard析构也会join
    }
    std::cout << "作用域结束" << std::endl;
    return 0;
}
```

**输出：**
```
任务完成
ThreadGuard自动join
作用域结束
```

### 2.2 jthread（C++20）

```cpp
#include <iostream>
#include <thread>

// C++20的jthread自动join
void demo() {
    // std::jthread自动在析构时join
    // std::jthread t([](){
    //     std::cout << "jthread自动管理" << std::endl;
    // });
    // 不需要手动join

    // 等价的C++11写法
    std::thread t([](){
        std::cout << "手动管理thread" << std::endl;
    });
    t.join();
}

int main() {
    demo();
    std::cout << "完成" << std::endl;
    return 0;
}
```

**输出：**
```
手动管理thread
完成
```

## 三、注意事项与常见陷阱

- **析构前必须`join`或`detach`**：否则调用`std::terminate()`。
- **`joinable()`检查线程是否可join**：已join或detach后返回false。
- **移动线程所有权**：`t2 = std::move(t1)`。
- **`jthread`（C++20）自动join**：还支持`stop_token`协作取消。
- **不要在容器销毁后访问线程**：先join再销毁。
