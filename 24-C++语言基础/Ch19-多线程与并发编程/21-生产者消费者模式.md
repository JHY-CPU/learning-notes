# 生产者消费者模式

## 一、概念说明

生产者-消费者模式是最经典的并发模式：生产者生成数据放入队列，消费者从队列取出数据处理。使用条件变量实现高效等待，避免忙轮询。

```cpp
#include <iostream>
#include <thread>
#include <mutex>
#include <condition_variable>
#include <queue>

template <typename T>
class BoundedQueue {
    std::queue<T> queue_;
    size_t capacity_;
    std::mutex mtx_;
    std::condition_variable notFull_, notEmpty_;

public:
    explicit BoundedQueue(size_t cap) : capacity_(cap) {}

    void push(T item) {
        std::unique_lock<std::mutex> lock(mtx_);
        notFull_.wait(lock, [this] { return queue_.size() < capacity_; });
        queue_.push(std::move(item));
        notEmpty_.notify_one();
    }

    T pop() {
        std::unique_lock<std::mutex> lock(mtx_);
        notEmpty_.wait(lock, [this] { return !queue_.empty(); });
        T item = std::move(queue_.front());
        queue_.pop();
        notFull_.notify_one();
        return item;
    }
};

int main() {
    BoundedQueue<int> q(5);

    std::thread producer([&q] {
        for (int i = 0; i < 10; ++i) {
            q.push(i);
            std::cout << "生产: " << i << std::endl;
        }
    });

    std::thread consumer([&q] {
        for (int i = 0; i < 10; ++i) {
            int val = q.pop();
            std::cout << "消费: " << val << std::endl;
        }
    });

    producer.join();
    consumer.join();
    return 0;
}
```

**输出（顺序可能不同）：**
```
生产: 0
消费: 0
生产: 1
消费: 1
...
```

## 二、具体用法

### 2.1 多生产者多消费者

```cpp
#include <iostream>
#include <thread>
#include <mutex>
#include <condition_variable>
#include <queue>
#include <atomic>

std::mutex coutMtx;

int main() {
    std::mutex mtx;
    std::condition_variable cv;
    std::queue<int> q;
    std::atomic<bool> done{false};

    auto producer = [&](int id) {
        for (int i = 0; i < 5; ++i) {
            {
                std::lock_guard<std::mutex> lock(mtx);
                q.push(id * 100 + i);
            }
            cv.notify_one();
        }
    };

    auto consumer = [&](int id) {
        while (true) {
            std::unique_lock<std::mutex> lock(mtx);
            cv.wait(lock, [&] { return !q.empty() || done; });
            if (q.empty() && done) break;
            int val = q.front();
            q.pop();
            lock.unlock();
            {
                std::lock_guard<std::mutex> lk(coutMtx);
                std::cout << "消费者" << id << " 处理: " << val << std::endl;
            }
        }
    };

    std::thread p1(producer, 1), p2(producer, 2);
    std::thread c1(consumer, 1), c2(consumer, 2);
    p1.join(); p2.join();
    done = true;
    cv.notify_all();
    c1.join(); c2.join();
    return 0;
}
```

**输出（顺序可能不同）：**
```
消费者1 处理: 100
消费者2 处理: 200
...
```

## 三、注意事项与常见陷阱

- **队列容量限制**：防止内存无限增长。
- **优雅退出**：用标志位通知消费者结束。
- **`notify`不需要持有锁**：但条件修改需在锁保护下。
- **虚假唤醒**：始终用谓词版本的`wait`。
- **多个消费者竞争**：每次`notify_one`只唤醒一个。
