# 关联容器概述

## 一、概念说明

关联容器通过键（key）组织数据，提供高效的查找操作。分为**有序关联容器**（基于红黑树，O(log n)）和**无序关联容器**（基于哈希表，平均O(1)）。有序容器保证元素按序排列，无序容器提供更快的查找。

## 二、分类

### 2.1 有序关联容器

```cpp
#include <map>
#include <set>

// map：键值对，键唯一
std::map<std::string, int> scores;

// multimap：键值对，键可重复
std::multimap<std::string, int> grades;

// set：只有键，键唯一
std::set<int> unique_nums;

// multiset：只有键，键可重复
std::multiset<int> bag;
```

### 2.2 无序关联容器

```cpp
#include <unordered_map>
#include <unordered_set>

// 基于哈希表
std::unordered_map<std::string, int> hash_map;
std::unordered_multimap<std::string, int> hash_mmap;
std::unordered_set<int> hash_set;
std::unordered_multiset<int> hash_mset;
```

### 2.3 选择指南

```cpp
/*
| 需求                     | 选择                              |
|------------------------|----------------------------------|
| 需要有序遍历            | map / set                        |
| 需要O(log n)稳定查找    | map / set                        |
| 需要最快查找            | unordered_map / unordered_set    |
| 允许重复键              | multimap / multiset              |
| 需要键值对              | map / unordered_map              |
| 只需要键                | set / unordered_set              |
*/
```

### 2.4 基本操作示例

```cpp
void demo() {
    // map插入
    std::map<std::string, int> m;
    m["alice"] = 90;          // 插入或更新
    m.insert({"bob", 85});    // 插入
    m.emplace("charlie", 95); // 原地构造

    // 查找
    auto it = m.find("alice");
    if (it != m.end()) {
        std::cout << it->first << ": " << it->second << std::endl;
    }

    // 检查存在
    if (m.count("alice") > 0) { /* 存在 */ }
    if (m.contains("alice")) { /* C++20 */ }
}
```

## 三、注意事项与常见陷阱

- 有序容器的键类型必须支持`operator<`或提供自定义比较器
- 无序容器的键类型必须可哈希（`std::hash`特化）
- `map::operator[]`在键不存在时会插入默认值
- 无序容器的遍历顺序不确定
- 有序容器的迭代器在插入删除后一般仍有效（被删元素除外）
- 自定义类型作为键需要特化`std::hash`或提供比较函数
