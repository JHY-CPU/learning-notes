# 自定义比较函数

## 一、概念说明

有序关联容器（map、set等）使用比较函数来确定元素顺序（C++标准 §23.2.4）。默认使用`std::less<Key>`（即`operator<`）。可以使用仿函数、lambda表达式或函数指针自定义比较逻辑。比较函数必须实现**严格弱序**（strict weak ordering）。

### 1.1 严格弱序要求

| 条件 | 数学含义 |
|------|---------|
| 非自反性 | `!(a < a)` |
| 非对称性 | `a < b` → `!(b < a)` |
| 传递性 | `a < b && b < c` → `a < c)` |
| 等价传递性 | `a ≡ b && b ≡ c` → `a ≡ c` |

其中`a ≡ b`定义为`!(a < b) && !(b < a)`。

## 二、具体用法

### 2.1 仿函数比较器

```cpp
#include <set>
#include <iostream>
#include <string>

struct CaseInsensitiveLess {
    bool operator()(const std::string& a, const std::string& b) const {
        return std::lexicographical_compare(
            a.begin(), a.end(), b.begin(), b.end(),
            [](char x, char y) { return std::tolower(x) < std::tolower(y); }
        );
    }
};

int main() {
    std::set<std::string, CaseInsensitiveLess> words;
    words.insert("Apple");
    words.insert("apple");  // 被忽略（大小写不敏感）
    words.insert("BANANA");

    for (const auto& w : words) std::cout << w << " ";
    // Apple BANANA（只有两个元素）
}
```

### 2.2 Lambda比较器（C++20）

```cpp
#include <map>

void lambda_comparator() {
    // C++20：lambda可作为模板参数
    auto greater = [](int a, int b) { return a > b; };

    // 需要声明类型
    std::map<int, std::string, decltype(greater)> desc(greater);
    desc[1] = "one";
    desc[2] = "two";
    desc[3] = "three";

    for (const auto& [k, v] : desc)
        std::cout << k << ": " << v << std::endl;
    // 3: three, 2: two, 1: one
}
```

### 2.3 使用std::greater

```cpp
#include <set>
#include <functional>

void std_greater_demo() {
    // 降序set
    std::set<int, std::greater<int>> desc = {1, 2, 3, 4, 5};
    for (int v : desc) std::cout << v << " ";  // 5 4 3 2 1

    // 降序map
    std::map<std::string, int, std::greater<std::string>> desc_map;
    desc_map["alice"] = 90;
    desc_map["bob"] = 85;
    // 按key降序排列
}
```

### 2.4 成员变量比较

```cpp
#include <set>
#include <tuple>

struct Employee {
    std::string name;
    int id;
    int department;

    // 用id排序
    bool operator<(const Employee& other) const {
        return id < other.id;
    }

    // 多字段排序：使用tuple的operator<
    static bool multi_field_less(const Employee& a, const Employee& b) {
        return std::tie(a.department, a.id) <
               std::tie(b.department, b.id);
    }
};

void member_compare() {
    std::set<Employee> employees;  // 使用operator<
    employees.insert({"Alice", 1002, 1});
    employees.insert({"Bob", 1001, 1});

    // 按id排序
    for (const auto& e : employees)
        std::cout << e.id << ": " << e.name << std::endl;
    // 1001: Bob, 1002: Alice

    // 使用自定义比较器（多字段排序）
    auto cmp = [](const Employee& a, const Employee& b) {
        return std::tie(a.department, a.id) <
               std::tie(b.department, b.id);
    };
    std::set<Employee, decltype(cmp)> multi_sorted(cmp);
}
```

### 2.5 指定比较器实例

```cpp
// 带状态的比较器
struct RangeComparator {
    int tolerance;
    explicit RangeComparator(int t) : tolerance(t) {}

    bool operator()(int a, int b) const {
        return std::abs(a - b) < tolerance ? false : a < b;
    }
};

void stateful_comparator() {
    // 必须传入比较器实例
    std::set<int, RangeComparator> s(RangeComparator(5));
    s.insert(10);
    s.insert(12);  // 与10差值<5，可能被视为等价
}
```

## 三、注意事项与常见陷阱

1. **比较函数必须实现严格弱序**：不满足会导致未定义行为（死循环、崩溃）
2. **`a < b`为true时，`b < a`必须为false**：违反会导致无限循环
3. **相等的元素应满足`!(a < b) && !(b < a)`**
4. **lambda比较器需要在声明时传入容器构造函数**：类型和实例都需要
5. **set/multiset的元素是const的**：比较器不能修改元素
6. **使用`std::tie`简化多字段比较**：自动按字段顺序比较
7. **比较器影响容器的行为**：不同的比较器会产生不同的元素顺序
