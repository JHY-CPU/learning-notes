# 自定义比较函数

## 一、概念说明

有序关联容器（map、set等）使用比较函数来确定元素顺序。可以使用仿函数、lambda表达式或函数指针自定义比较逻辑。

## 二、具体用法

### 2.1 仿函数比较器

```cpp
#include <set>
#include <iostream>

struct CaseInsensitiveLess {
    bool operator()(const std::string& a, const std::string& b) const {
        return std::lexicographical_compare(
            a.begin(), a.end(), b.begin(), b.end(),
            [](char x, char y) { return std::tolower(x) < std::tolower(y); }
        );
    }
};

int main() {
    std::set<std::string, CaseInsensitiveLess> words;
    words.insert("Apple");
    words.insert("apple");  // 被忽略（大小写不敏感）
    words.insert("BANANA");

    for (const auto& w : words) std::cout << w << " ";
    // Apple BANANA（只有两个元素）
}
```

### 2.2 Lambda比较器

```cpp
#include <map>

void lambda_comparator() {
    // 降序排列
    auto greater = [](int a, int b) { return a > b; };
    std::map<int, std::string, decltype(greater)> desc(greater);
    desc[1] = "one";
    desc[2] = "two";
    desc[3] = "three";

    for (const auto& [k, v] : desc) {
        std::cout << k << ": " << v << std::endl;
    }
    // 3: three, 2: two, 1: one
}
```

### 2.3 成员变量比较

```cpp
struct Employee {
    std::string name;
    int id;

    // 用id排序
    bool operator<(const Employee& other) const {
        return id < other.id;
    }
};

void member_compare() {
    std::set<Employee> employees;
    employees.insert({"Alice", 1002});
    employees.insert({"Bob", 1001});

    // 按id排序
    for (const auto& e : employees) {
        std::cout << e.id << ": " << e.name << std::endl;
    }
}
```

## 三、注意事项与常见陷阱

- 比较函数必须实现严格弱序（strict weak ordering）
- `a < b`为true时，`b < a`必须为false
- 相等的元素应满足`!(a < b) && !(b < a)`
- lambda比较器需要在声明时传入容器构造函数
- set/multiset的元素是const的，比较器不能修改元素
