# 容器内存管理

## 一、概念说明

STL容器通过分配器（Allocator）管理内存。默认使用`std::allocator`，也可以自定义分配器实现内存池、统计等需求。

## 二、具体用法

### 2.1 默认分配器

```cpp
#include <vector>
#include <memory>

int main() {
    // 默认使用std::allocator
    std::vector<int, std::allocator<int>> v = {1, 2, 3};

    // 内存管理
    v.reserve(100);  // 预分配
    v.shrink_to_fit();  // 释放多余
}
```

### 2.2 自定义分配器

```cpp
#include <vector>
#include <cstddef>
#include <iostream>

// 简单的计数分配器
template <typename T>
struct CountingAllocator {
    using value_type = T;
    static size_t alloc_count;

    T* allocate(size_t n) {
        alloc_count += n;
        return static_cast<T*>(::operator new(n * sizeof(T)));
    }

    void deallocate(T* p, size_t n) noexcept {
        alloc_count -= n;
        ::operator delete(p);
    }

    template <typename U>
    CountingAllocator(const CountingAllocator<U>&) noexcept {}
    CountingAllocator() = default;
};

template <typename T>
size_t CountingAllocator<T>::alloc_count = 0;

int main() {
    std::vector<int, CountingAllocator<int>> v;
    for (int i = 0; i < 1000; ++i) v.push_back(i);
    std::cout << "分配: " << CountingAllocator<int>::alloc_count << std::endl;
}
```

### 2.3 pmr分配器（C++17）

```cpp
#include <vector>
#include <memory_resource>

void pmr_demo() {
    // 使用单调缓冲区（避免系统调用）
    char buffer[1024];
    std::pmr::monotonic_buffer_resource pool{std::data(buffer), std::size(buffer)};

    std::pmr::vector<int> v{&pool};
    for (int i = 0; i < 100; ++i) v.push_back(i);
    // 内存从buffer分配，快速且确定性
}
```

## 三、注意事项

- 自定义分配器需要满足Allocator要求
- pmr（C++17）提供了更灵活的内存管理
- 内存池分配器可减少碎片和系统调用
- 分配器相等的容器才能互操作
