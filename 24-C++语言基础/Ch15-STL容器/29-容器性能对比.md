# 容器性能对比

## 一、操作复杂度对比表

```cpp
/*
| 容器              | 随机访问 | 插入(头) | 插入(尾) | 插入(中) | 查找    |
|-------------------|---------|---------|---------|---------|--------|
| vector            | O(1)    | O(n)    | O(1)*   | O(n)    | O(n)   |
| deque             | O(1)    | O(1)    | O(1)    | O(n)    | O(n)   |
| list              | O(n)    | O(1)    | O(1)    | O(1)**  | O(n)   |
| forward_list      | O(n)    | O(1)    | N/A     | O(1)**  | O(n)   |
| set/map           | N/A     | N/A     | N/A     | O(log n)| O(log n)|
| unordered_set/map | N/A     | N/A     | N/A     | O(1)*** | O(1)***|
| array             | O(1)    | N/A     | N/A     | N/A     | O(n)   |
*/
```

## 二、实际性能测试

```cpp
#include <vector>
#include <list>
#include <unordered_set>
#include <set>
#include <chrono>
#include <iostream>

int main() {
    const int N = 1000000;

    // vector顺序插入
    auto start = std::chrono::high_resolution_clock::now();
    std::vector<int> vec;
    for (int i = 0; i < N; ++i) vec.push_back(i);
    auto end = std::chrono::high_resolution_clock::now();
    std::cout << "vector push_back: "
              << std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count()
              << "ms" << std::endl;

    // vector随机访问
    start = std::chrono::high_resolution_clock::now();
    volatile int sum = 0;
    for (int i = 0; i < N; ++i) sum += vec[i];
    end = std::chrono::high_resolution_clock::now();
    std::cout << "vector随机访问: "
              << std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count()
              << "ms" << std::endl;

    // unordered_set查找
    std::unordered_set<int> uset;
    for (int i = 0; i < N; ++i) uset.insert(i);
    start = std::chrono::high_resolution_clock::now();
    for (int i = 0; i < N; ++i) uset.count(i);
    end = std::chrono::high_resolution_clock::now();
    std::cout << "unordered_set查找: "
              << std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count()
              << "ms" << std::endl;
}
```

## 三、选择建议

- 默认使用vector（缓存友好）
- 需要有序用set/map
- 需要快速查找用unordered_set/map
- 需要稳定引用用list
- 固定大小用array
