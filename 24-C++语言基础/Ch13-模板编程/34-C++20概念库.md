# C++20概念库

## 一、概念说明

`<concepts>`头文件提供了C++20标准库预定义的概念（Concept），涵盖了比较、算术、对象生命周期等基本约束。这些概念可以直接用于模板约束，无需自己定义。

## 二、具体用法

### 2.1 核心语言概念

```cpp
#include <concepts>

int main() {
    std::cout << std::boolalpha;

    // 基本类型概念
    std::cout << std::integral<int> << std::endl;              // true
    std::cout << std::integral<double> << std::endl;           // false
    std::cout << std::floating_point<float> << std::endl;      // true
    std::cout << std::signed_integral<int> << std::endl;       // true
    std::cout << std::unsigned_integral<unsigned> << std::endl;// true

    // 派生关系
    struct Base {};
    struct Derived : Base {};
    std::cout << std::derived_from<Derived, Base> << std::endl; // true

    // 可转换性
    std::cout << std::convertible_to<int, double> << std::endl;  // true
    std::cout << std::convertible_to<int*, void*> << std::endl;  // true
}
```

### 2.2 比较概念

```cpp
template <typename T>
    requires std::equality_comparable<T>
bool find(const std::vector<T>& vec, const T& target) {
    for (const auto& x : vec)
        if (x == target) return true;
    return false;
}

template <typename T>
    requires std::totally_ordered<T>
T find_min(const std::vector<T>& vec) {
    return *std::min_element(vec.begin(), vec.end());
}

int main() {
    std::vector<int> v{3, 1, 4, 1, 5};
    std::cout << find(v, 4) << std::endl;       // true
    std::cout << find_min(v) << std::endl;       // 1
}
```

### 2.3 对象概念

```cpp
template <typename T>
    requires std::default_initializable<T> && std::copyable<T>
class ValueHolder {
    T value{};
public:
    ValueHolder() = default;
    ValueHolder(T v) : value(v) {}
    T get() const { return value; }
    void set(T v) { value = v; }
};

int main() {
    ValueHolder<int> vh1;          // default_initializable
    ValueHolder<int> vh2(42);
    ValueHolder<int> vh3 = vh2;    // copyable
    std::cout << vh3.get() << std::endl;  // 42
}
```

### 2.4 可调用概念

```cpp
template <typename F, typename T>
    requires std::invocable<F, T>
auto apply_and_print(F func, T val) {
    if constexpr (std::invocable<F, T>) {
        auto result = func(val);
        std::cout << "结果: " << result << std::endl;
        return result;
    }
}

int main() {
    apply_and_print([](int x) { return x * 2; }, 21);   // 结果: 42
    apply_and_print([](double x) { return x + 1; }, 3.0); // 结果: 4
}
```

### 2.5 movable和copyable

```cpp
template <typename T>
    requires std::movable<T>
class MovableOnly {
    T value;
public:
    MovableOnly(T v) : value(std::move(v)) {}
    MovableOnly(MovableOnly&&) = default;
    MovableOnly& operator=(MovableOnly&&) = default;
    T get() const { return value; }
};

int main() {
    MovableOnly<std::string> mo1("hello");
    MovableOnly<std::string> mo2 = std::move(mo1);
    std::cout << mo2.get() << std::endl;  // hello
}
```

## 三、注意事项与常见陷阱

- `<concepts>`头文件需要C++20支持
- `same_as<T,U>`检查精确相等，`convertible_to<T,U>`检查隐式转换
- `invocable<F, Args...>`检查函数对象是否可以用指定参数调用
- `regular<T>`要求：default_initializable + copyable + equality_comparable
- `semiregular<T>`要求：default_initializable + copyable
- 概念可以组合：`requires T { requires integral<T> && copyable<T>; }`
- 检查概念是否满足可以用`std::integral<T>`（直接作为bool值）
