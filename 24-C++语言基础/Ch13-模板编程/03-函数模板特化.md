# 函数模板特化

## 一、概念说明

函数模板特化（Specialization）允许为特定类型提供不同于通用模板的实现。当通用模板的算法对某些类型不适用或效率不高时，特化提供了定制化的能力。特化分为**全特化**（所有参数都指定具体类型）。

## 二、具体用法

### 2.1 全特化

```cpp
// 通用模板
template <typename T>
bool compare(T a, T b) {
    std::cout << "通用模板" << std::endl;
    return a == b;
}

// 全特化：const char* 版本
template <>
bool compare<const char*>(const char* a, const char* b) {
    std::cout << "const char* 特化" << std::endl;
    return std::strcmp(a, b) == 0;
}

int main() {
    compare(10, 20);                    // 通用模板 → false
    compare("abc", "abc");              // const char* 特化 → true
    compare(std::string("hi"), std::string("hi")); // 通用模板 → true
}
```

### 2.2 特化与重载的优先级

```cpp
template <typename T>
void process(T val) {
    std::cout << "模板版本: " << val << std::endl;
}

// 全特化
template <>
void process<int*>(int* val) {
    std::cout << "指针特化: " << *val << std::endl;
}

// 非模板重载（优先级最高）
void process(double val) {
    std::cout << "非模板版本: " << val << std::endl;
}

int main() {
    process(3.14);     // 非模板版本
    process(42);       // 模板版本（int）
    int x = 100;
    process(&x);       // 指针特化
}
```

### 2.3 常见应用场景

```cpp
// 字符串比较特化
template <typename T>
int compare_val(T a, T b) {
    if (a < b) return -1;
    if (b < a) return 1;
    return 0;
}

template <>
int compare_val<const char*>(const char* a, const char* b) {
    return std::strcmp(a, b);  // 用strcmp替代指针比较
}

// 输出特化
template <typename T>
void print(T val) {
    std::cout << val;
}

template <>
void print<bool>(bool val) {
    std::cout << (val ? "true" : "false");  // bool特殊输出
}

int main() {
    std::cout << compare_val(3, 5) << std::endl;         // -1
    std::cout << compare_val("abc", "def") << std::endl; // -1 (strcmp)

    print(true);   // true
    print(42);     // 42
}
```

## 三、注意事项与常见陷阱

- 函数模板不支持偏特化（Partial Specialization），只有全特化
- 对于偏特化需求，可使用函数重载、类模板或`if constexpr`替代
- 特化必须在通用模板声明之后
- 特化版本的函数签名必须与通用模板完全匹配（包括const引用等）
- 优先级：非模板函数 > 全特化 > 通用模板
- 考虑用`std::string_view`替代`const char*`特化以获得更好的类型安全
