# 函数模板基础

## 一、概念说明

函数模板允许定义一个通用函数，编译器根据调用时的实参类型自动推导模板参数并生成具体函数。函数模板本身不是函数，而是生成函数的"蓝图"。

## 二、具体用法

### 2.1 template关键字语法

```cpp
// 基本语法：template <typename T> 或 template <class T>
template <typename T>
T add(T a, T b) {
    return a + b;
}

// 多个模板参数
template <typename T, typename U>
auto multiply(T a, U b) -> decltype(a * b) {
    return a * b;
}
```

### 2.2 类型推导

```cpp
template <typename T>
void print_type(T value) {
    std::cout << typeid(value).name() << ": " << value << std::endl;
}

int main() {
    // 编译器自动推导类型
    print_type(42);          // int: 42
    print_type(3.14);        // double: 3.14
    print_type("hello");     // const char*: hello

    // 显式指定类型（无法推导或需要转换时）
    std::cout << add<int>(3, 4.5) << std::endl;  // 7（4.5被截断为4）
    std::cout << add<double>(3, 4.5) << std::endl; // 7.5
}
```

### 2.3 模板实例化

```cpp
// 隐式实例化：编译器自动根据使用情况生成
template <typename T>
T square(T x) { return x * x; }

// 显式实例化：强制编译器生成特定类型的版本
template int square<int>(int);       // 显式实例化int版本
template double square<double>(double); // 显式实例化double版本

int main() {
    std::cout << square(5) << std::endl;    // 25
    std::cout << square(2.5) << std::endl;  // 6.25
}
```

### 2.4 模板重载

```cpp
// 普通函数（非模板）优先于模板函数
int max_val(int a, int b) {
    std::cout << "普通版本" << std::endl;
    return (a > b) ? a : b;
}

template <typename T>
T max_val(T a, T b) {
    std::cout << "模板版本" << std::endl;
    return (a > b) ? a : b;
}

int main() {
    max_val(1, 2);        // 普通版本（精确匹配非模板）
    max_val(1.0, 2.0);    // 模板版本（double）
    max_val('a', 'b');    // 模板版本（char）
}
```

## 三、注意事项与常见陷阱

- 函数模板的模板参数可以隐式推导，类模板不可以（C++17之前）
- 当模板参数无法推导时，必须显式指定：`func<int>(args...)`
- 引用和const修饰符会影响类型推导的结果
- 数组会退化为指针进行推导（`int[3]` 推导为 `int*`）
- 重载决议中，非模板函数优先于模板函数，更特化的模板优先于更通用的模板
