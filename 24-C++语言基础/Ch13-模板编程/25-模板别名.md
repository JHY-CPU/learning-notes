# 模板别名

## 一、概念说明

模板别名（Alias Template）是C++11引入的`using`语法，允许为模板类型创建别名。相比传统的`typedef`，`using`可以模板化，支持默认参数，语法也更清晰。

## 二、具体用法

### 2.1 基本别名模板

```cpp
// typedef的限制：不能模板化
typedef std::vector<int> IntVec;  // OK，但固定了类型

// using可以模板化
template <typename T>
using Vec = std::vector<T>;

// 别名模板带默认参数
template <typename T, typename Allocator = std::allocator<T>>
using Deque = std::deque<T, Allocator>;

int main() {
    Vec<int> v1{1, 2, 3};
    Vec<std::string> v2{"hello", "world"};

    std::cout << v1.size() << std::endl;  // 3
    std::cout << v2[0] << std::endl;      // hello
}
```

### 2.2 简化复杂类型

```cpp
// 简化迭代器类型
template <typename Container>
using Iterator = typename Container::iterator;

// 简化智能指针
template <typename T>
using Unique = std::unique_ptr<T>;

template <typename T>
using Shared = std::shared_ptr<T>;

// 简化函数类型
template <typename Ret, typename... Args>
using Function = std::function<Ret(Args...)>;

int main() {
    std::vector<int> v{1, 2, 3};
    Iterator<std::vector<int>> it = v.begin();
    std::cout << *it << std::endl;  // 1

    Unique<int> p = std::make_unique<int>(42);
    std::cout << *p << std::endl;   // 42

    Function<int, int, int> add = [](int a, int b) { return a + b; };
    std::cout << add(3, 4) << std::endl;  // 7
}
```

### 2.3 与SFINAE配合

```cpp
// 条件别名
template <typename T>
using EnableIfIntegral = std::enable_if_t<std::is_integral_v<T>, T>;

template <typename T>
using EnableIfFloat = std::enable_if_t<std::is_floating_point_v<T>, T>;

// 使用
template <typename T>
EnableIfIntegral<T> safe_div(T a, T b) {
    return b ? a / b : 0;
}

template <typename T>
EnableIfFloat<T> safe_div(T a, T b) {
    return b ? a / b : 0.0;
}

int main() {
    std::cout << safe_div(10, 3) << std::endl;    // 3
    std::cout << safe_div(10.0, 3.0) << std::endl; // 3.33333
}
```

### 2.4 类型映射别名

```cpp
// 类型特征简化
template <typename T>
using RemoveCVRef = std::remove_cv_t<std::remove_reference_t<T>>;

// 编译期类型选择
template <bool IsBig>
using Storage = std::conditional_t<IsBig, std::vector<uint8_t>, std::array<uint8_t, 64>>;

int main() {
    RemoveCVRef<const int&> x = 42;  // x是int类型
    std::cout << x << std::endl;     // 42

    Storage<false> small_buf{};  // array<uint8_t, 64>
    Storage<true> big_buf(1000); // vector<uint8_t>
    std::cout << sizeof(small_buf) << std::endl;  // 64
}
```

## 三、注意事项与常见陷阱

- `using`别名模板不能特化（可以使用类模板特化替代）
- 别名模板不是新类型，只是已有类型的别名
- 别名模板在模板参数推导中会被展开
- typedef不能模板化，`using`是唯一选择
- 别名模板可以出现在类作用域中（作为成员别名）
- C++11之前可用类模板+内部typedef模拟，但更繁琐
