# 变参模板进阶

## 一、概念说明

变参模板的进阶用法包括参数包的各种展开技巧、完美转发组合使用、以及递归继承等高级模式。这些技巧是实现`std::tuple`、`std::variant`等标准库组件的基础。

## 二、具体用法

### 2.1 完美转发与变参模板

```cpp
// std::make_unique 的简化实现
template <typename T, typename... Args>
std::unique_ptr<T> make_object(Args&&... args) {
    // 完美转发：保持参数的值类别（左值/右值）
    return std::unique_ptr<T>(new T(std::forward<Args>(args)...));
}

struct Point {
    double x, y;
    Point(double x, double y) : x(x), y(y) {}
    void show() const { std::cout << "(" << x << ", " << y << ")" << std::endl; }
};

int main() {
    auto p = make_object<Point>(3.0, 4.0);
    p->show();  // (3, 4)
}
```

### 2.2 初始化列表展开

```cpp
// 使用初始化列表保证求值顺序
template <typename... Args>
void ordered_print(Args... args) {
    // initializer_list保证从左到右求值
    (void)std::initializer_list<int>{
        (std::cout << args << " ", 0)...
    };
    std::cout << std::endl;
}

// 计算多个值的和
template <typename... Args>
auto sum_all(Args... args) {
    using CommonType = std::common_type_t<Args...>;
    CommonType result = 0;
    (void)std::initializer_list<CommonType>{
        (result += args, CommonType{0})...
    };
    return result;
}

int main() {
    ordered_print(1, "hello", 3.14);  // 1 hello 3.14
    std::cout << sum_all(1, 2, 3, 4) << std::endl;  // 10
}
```

### 2.3 递归继承（实现tuple）

```cpp
// 简化版tuple实现
template <typename... Types>
class MyTuple;

// 基本情况
template <>
class MyTuple<> {};

// 递归情况：继承包含剩余类型的tuple
template <typename Head, typename... Tail>
class MyTuple<Head, Tail...> : private MyTuple<Tail...> {
    using Base = MyTuple<Tail...>;
    Head value;
public:
    MyTuple() = default;
    MyTuple(const Head& h, const Tail&... t) : Base(t...), value(h) {}

    Head& head() { return value; }
    const Head& head() const { return value; }
    Base& tail() { return *this; }
};

int main() {
    MyTuple<int, double, char> t(42, 3.14, 'A');
    std::cout << t.head() << std::endl;              // 42
    std::cout << t.tail().head() << std::endl;       // 3.14
    std::cout << t.tail().tail().head() << std::endl; // A
}
```

### 2.4 索引序列展开

```cpp
// 编译期索引序列
template <std::size_t... Is>
struct index_sequence {};

template <std::size_t N, std::size_t... Is>
struct make_index_sequence_impl : make_index_sequence_impl<N-1, N-1, Is...> {};

template <std::size_t... Is>
struct make_index_sequence_impl<0, Is...> {
    using type = index_sequence<Is...>;
};

template <std::size_t N>
using make_index_sequence = typename make_index_sequence_impl<N>::type;

// 用索引序列调用函数
template <typename Tuple, std::size_t... Is>
void print_tuple_impl(const Tuple& t, index_sequence<Is...>) {
    ((std::cout << std::get<Is>(t) << " "), ...);
    std::cout << std::endl;
}

template <typename... Args>
void print_tuple(const std::tuple<Args...>& t) {
    print_tuple_impl(t, make_index_sequence<sizeof...(Args)>{});
}

int main() {
    auto t = std::make_tuple(10, 3.14, "hello");
    print_tuple(t);  // 10 3.14 hello
}
```

## 三、注意事项与常见陷阱

- 完美转发`Args&&...`结合`std::forward`是变参模板的常用模式
- 初始化列表的逗号表达式`(expr, 0)...`是展开副作用的常用技巧
- 递归继承会为每个参数类型创建一个基类，注意空基类优化
- `std::index_sequence`是C++14标准库提供的，无需手动实现
- 编译器对变参模板的递归深度有限制，超大参数包可能需要其他方案
- 折叠表达式（C++17）可以替代大部分初始化列表展开的场景
