# 类模板特化

## 一、概念说明

类模板特化允许为特定类型提供不同的类实现。与函数模板不同，类模板同时支持**全特化**和**偏特化**。全特化为所有模板参数指定具体类型，偏特化则只为部分参数或满足特定条件的类型提供定制。

## 二、具体用法

### 2.1 全特化

```cpp
// 通用模板
template <typename T>
class Storage {
    T value;
public:
    Storage(T v) : value(v) {}
    void print() const { std::cout << "Value: " << value << std::endl; }
};

// 全特化：bool类型
template <>
class Storage<bool> {
    bool value;
public:
    Storage(bool v) : value(v) {}
    void print() const {
        std::cout << "Bool: " << (value ? "true" : "false") << std::endl;
    }
    bool flip() { value = !value; return value; }
};

// 全特化：const char*
template <>
class Storage<const char*> {
    std::string value;
public:
    Storage(const char* v) : value(v ? v : "") {}
    void print() const { std::cout << "String: \"" << value << "\"" << std::endl; }
    size_t length() const { return value.length(); }
};

int main() {
    Storage<int> si(42);
    si.print();                  // Value: 42

    Storage<bool> sb(true);
    sb.print();                  // Bool: true
    sb.flip();
    sb.print();                  // Bool: false

    Storage<const char*> ss("hello");
    ss.print();                  // String: "hello"
    std::cout << ss.length();    // 5
}
```

### 2.2 全特化的匹配规则

```cpp
template <typename T>
class TypeName {
public:
    static std::string name() { return "unknown"; }
};

template <>
class TypeName<int> {
public:
    static std::string name() { return "int"; }
};

template <>
class TypeName<double> {
public:
    static std::string name() { return "double"; }
};

int main() {
    std::cout << TypeName<int>::name() << std::endl;     // int
    std::cout << TypeName<double>::name() << std::endl;  // double
    std::cout << TypeName<char>::name() << std::endl;    // unknown
}
```

## 三、注意事项与常见陷阱

- 全特化必须在通用模板声明之后定义
- 特化版本必须提供完整的类定义（不能只特化部分成员）
- 编译器选择最匹配的模板版本：全特化 > 通用模板
- 特化的模板参数列表`<>`中不写`typename`/`class`
- 全特化的类不再是一个模板类，它是一个普通类
- 如果只需特化部分参数，应使用偏特化而非全特化
