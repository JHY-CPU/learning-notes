# 折叠表达式（C++17）

## 一、概念说明

折叠表达式（Fold Expression）是C++17引入的语法，用于简洁地对参数包中的所有参数应用同一个二元运算符。它替代了以往需要递归模板或初始化列表技巧来展开参数包的繁琐写法。

## 二、具体用法

### 2.1 四种折叠形式

```cpp
// 一元右折叠: ( pack op ... )
template <typename... Args>
auto sum_right(Args... args) {
    return (args + ...);  // arg1 + (arg2 + arg3)
}

// 一元左折叠: ( ... op pack )
template <typename... Args>
auto sum_left(Args... args) {
    return (... + args);  // (arg1 + arg2) + arg3
}

// 二元右折叠: ( pack op ... op init )
template <typename... Args>
auto concat(Args... args) {
    return (args + ... + std::string("!"));  // arg1 + (arg2 + ... + "!")
}

// 二元左折叠: ( init op ... op pack )
template <typename... Args>
auto sum_init(Args... args) {
    return (100 + ... + args);  // ((100 + arg1) + arg2) + ...
}

int main() {
    std::cout << sum_right(1, 2, 3, 4) << std::endl;    // 10
    std::cout << sum_left(1, 2, 3, 4) << std::endl;     // 10

    std::cout << concat("a", "b", "c") << std::endl;    // abc!
    std::cout << sum_init(1, 2, 3) << std::endl;        // 106
}
```

### 2.2 常用运算符折叠

```cpp
// 逻辑运算
template <typename... Args>
bool all_true(Args... args) {
    return (args && ...);  // 所有参数都为true
}

template <typename... Args>
bool any_true(Args... args) {
    return (args || ...);  // 任一参数为true
}

// 比较运算
template <typename... Args>
bool all_equal(Args... args) {
    // 对于单参数直接返回true
    if constexpr (sizeof...(args) <= 1) return true;
    else {
        auto first = std::get<0>(std::tuple<Args...>(args...));
        return ((args == first) && ...);
    }
}

int main() {
    std::cout << std::boolalpha;
    std::cout << all_true(true, true, true) << std::endl;   // true
    std::cout << all_true(true, false, true) << std::endl;  // false
    std::cout << any_true(false, false, true) << std::endl; // true
}
```

### 2.3 逗号折叠（副作用）

```cpp
// 打印所有参数
template <typename... Args>
void print_all(Args... args) {
    ((std::cout << args << " "), ...);  // 逗号折叠
    std::cout << std::endl;
}

// 对每个参数调用函数
template <typename Func, typename... Args>
void for_each(Func func, Args&&... args) {
    (func(std::forward<Args>(args)), ...);
}

int main() {
    print_all(1, 2.5, "hello", 'A');  // 1 2.5 hello A

    for_each([](auto x) { std::cout << x * 2 << " "; },
             1, 2, 3, 4);
    // 输出: 2 4 6 8
}
```

### 2.4 二元折叠与初始值

```cpp
// 带初始值的折叠更安全
template <typename... Args>
auto safe_sum(Args... args) {
    return (0 + ... + args);  // 空参数包返回0
}

// 字符串拼接
template <typename... Strings>
std::string concat_all(Strings... strs) {
    return (std::string{} + ... + strs);
}

int main() {
    std::cout << safe_sum() << std::endl;          // 0（空参数包）
    std::cout << safe_sum(1, 2, 3) << std::endl;   // 6

    std::cout << concat_all("Hello", " ", "World") << std::endl;  // Hello World
}
```

## 三、注意事项与常见陷阱

- 一元折叠对空参数包的行为：`&&`→true，`||`→false，`,`→`void()`，其他运算符报错
- 使用二元折叠提供初始值可以安全处理空参数包的情况
- 左折叠和右折叠对于非结合运算符（如`-`、`/`）结果不同
- 逗号折叠`(expr, ...)`是最常用的副作用展开方式
- 折叠表达式可以大大简化变参模板代码，优先考虑使用
- 不能混用一元和二元折叠语法
