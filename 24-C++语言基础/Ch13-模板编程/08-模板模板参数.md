# 模板模板参数

## 一、概念说明

模板模板参数（Template Template Parameter）允许将一个模板本身作为另一个模板的参数传递。这使得可以编写接受"模板的模板"的泛型代码，常用于容器适配器、策略类等设计模式中。

## 二、具体用法

### 2.1 基本语法

```cpp
// Container 是一个模板模板参数
template <typename T, template <typename, typename> class Container>
class Adapter {
    Container<T, std::allocator<T>> data;
public:
    void add(const T& val) { data.push_back(val); }
    void show() const {
        for (const auto& v : data) std::cout << v << " ";
        std::cout << std::endl;
    }
    std::size_t size() const { return data.size(); }
};

int main() {
    // 传入 vector 作为容器
    Adapter<int, std::vector> vecAdapter;
    vecAdapter.add(1); vecAdapter.add(2); vecAdapter.add(3);
    vecAdapter.show();  // 1 2 3

    // 传入 deque 作为容器
    Adapter<int, std::deque> dequeAdapter;
    dequeAdapter.add(10); dequeAdapter.add(20);
    dequeAdapter.show();  // 10 20
}
```

### 2.2 简化写法（C++17）

```cpp
// C++17 可以用 template <typename> class Container 简化
template <typename T, template <typename> class Wrapper>
class ValueHolder {
    Wrapper<T> val;
public:
    // 使用 Wrapper<T> 的接口
    void set(const T& v) { /* val = v; */ }
    T get() const { /* return *val; */ }
};

// C++17 中 class 和 typename 等价
template <typename T, template <typename> typename Container>
void process() {
    Container<T> instance;
    std::cout << typeid(instance).name() << std::endl;
}
```

### 2.3 实际应用：策略容器

```cpp
template <typename T>
class AscendingSort {
public:
    bool operator()(const T& a, const T& b) const { return a < b; }
};

template <typename T>
class DescendingSort {
public:
    bool operator()(const T& a, const T& b) const { return a > b; }
};

template <typename T, template <typename> class Compare = AscendingSort>
class SortableVector {
    std::vector<T> data;
    Compare<T> comp;
public:
    void push(const T& val) { data.push_back(val); }
    void sort() { std::sort(data.begin(), data.end(), comp); }
    void print() const {
        for (const auto& v : data) std::cout << v << " ";
        std::cout << std::endl;
    }
};

int main() {
    SortableVector<int, AscendingSort> sv1;
    sv1.push(30); sv1.push(10); sv1.push(20);
    sv1.sort(); sv1.print();  // 10 20 30

    SortableVector<int, DescendingSort> sv2;
    sv2.push(30); sv2.push(10); sv2.push(20);
    sv2.sort(); sv2.print();  // 30 20 10
}
```

## 三、注意事项与常见陷阱

- 模板模板参数的签名必须与实际传入模板的签名匹配
- C++17之前必须用`class`关键字，C++17起可用`typename`
- 传入`std::vector`时需注意其有两个模板参数（元素类型+分配器）
- 模板模板参数增加了模板的抽象层次，会提升代码复杂度
- 常见用途：策略模式、容器适配器、分配器配置
- 使用`template <typename> class`语法时注意参数个数匹配
