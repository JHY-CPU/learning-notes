# 类模板基础

## 一、概念说明

类模板（Class Template）允许定义一个通用的类，其中某些类型作为模板参数。使用类模板时必须显式指定类型参数（C++17之前），编译器为每种类型组合生成独立的类。

## 二、具体用法

### 2.1 基本定义与使用

```cpp
template <typename T>
class Pair {
    T first, second;
public:
    Pair(T a, T b) : first(a), second(b) {}
    T getFirst() const { return first; }
    T getSecond() const { return second; }
    T max() const { return (first > second) ? first : second; }
};

int main() {
    Pair<int> p1(3, 7);
    std::cout << p1.max() << std::endl;           // 7

    Pair<std::string> p2("apple", "banana");
    std::cout << p2.max() << std::endl;           // banana

    // C++17 类模板参数推导（CTAD）
    Pair p3(1.5, 2.5);  // 推导为 Pair<double>
    std::cout << p3.max() << std::endl;           // 2.5
}
```

### 2.2 多参数类模板

```cpp
template <typename K, typename V>
class KeyValue {
    K key;
    V value;
public:
    KeyValue(const K& k, const V& v) : key(k), value(v) {}
    void print() const {
        std::cout << key << " => " << value << std::endl;
    }
};

int main() {
    KeyValue<std::string, int> entry("age", 25);
    entry.print();  // age => 25

    KeyValue<double, std::string> map(3.14, "pi");
    map.print();    // 3.14 => pi
}
```

### 2.3 成员函数的实例化

```cpp
template <typename T>
class Container {
    T data;
public:
    Container(const T& d) : data(d) {}
    void show() const { std::cout << data << std::endl; }

    // 只有被调用的成员函数才会被实例化
    void unused() const {
        // 即使T没有某些操作，只要不调用此函数就不会报错
        std::cout << data.nonexistent() << std::endl;
    }
};

int main() {
    Container<int> c(42);
    c.show();      // 42 — show()被实例化
    // c.unused(); // 编译错误——如果取消注释，会尝试实例化unused()
}
```

### 2.4 类模板中的静态成员

```cpp
template <typename T>
class Counter {
    static int count;
public:
    Counter() { ++count; }
    static int getCount() { return count; }
};

// 每种类型的静态成员需要单独定义
template <typename T> int Counter<T>::count = 0;

int main() {
    Counter<int> a, b;
    Counter<double> c;
    std::cout << Counter<int>::getCount() << std::endl;    // 2
    std::cout << Counter<double>::getCount() << std::endl; // 1
}
```

## 三、注意事项与常见陷阱

- 类模板的成员函数只有在被调用时才实例化（懒实例化）
- 静态成员需要在类外用`template<typename T>`前缀定义
- 每种模板参数组合生成完全独立的类，`vector<int>`和`vector<double>`无继承关系
- C++17的类模板参数推导（CTAD）可以省略显式类型指定
- 类模板中使用嵌套类型需要加`typename`关键字（如`typename Container::iterator`）
