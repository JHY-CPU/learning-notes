# 非类型模板参数

## 一、概念说明

非类型模板参数（Non-Type Template Parameter, NTTP）允许将编译期常量值作为模板参数传递。与类型参数不同，NTTP传递的是值而非类型。这在需要编译期确定大小、策略等场景中非常有用。

## 二、具体用法

### 2.1 整型NTTP

```cpp
// 固定大小的数组容器
template <typename T, std::size_t N>
class FixedArray {
    T data[N];
public:
    T& operator[](std::size_t i) { return data[i]; }
    constexpr std::size_t size() const { return N; }

    void fill(const T& val) {
        for (std::size_t i = 0; i < N; ++i) data[i] = val;
    }
};

int main() {
    FixedArray<int, 5> arr;
    arr.fill(0);
    arr[2] = 42;
    std::cout << arr.size() << std::endl;    // 5
    std::cout << arr[2] << std::endl;        // 42

    FixedArray<double, 100> big;
    std::cout << big.size() << std::endl;    // 100
}
```

### 2.2 指针和引用NTTP

```cpp
// 引用NTTP：编译期全局配置
template <int& Config>
void apply_config() {
    std::cout << "配置值: " << Config << std::endl;
}

int global_config = 42;

// 指针NTTP
template <const char* Name>
void print_name() {
    std::cout << "名称: " << Name << std::endl;
}
```

### 2.3 C++20字面类型NTTP

C++20放宽了NTTP的限制，允许使用字面类型（有constexpr构造函数的类）作为NTTP：

```cpp
// C++20：类类型作为NTTP
struct Config {
    int width;
    int height;
    constexpr Config(int w, int h) : width(w), height(h) {}
};

template <Config cfg>
class Window {
public:
    static void info() {
        std::cout << "窗口: " << cfg.width << "x" << cfg.height << std::endl;
    }
};

int main() {
    Window<Config{800, 600}>::info();   // 窗口: 800x600
    Window<Config{1920, 1080}>::info(); // 窗口: 1920x1080
}
```

### 2.4 auto作为NTTP类型（C++17）

```cpp
// C++17：auto推导NTTP类型
template <auto N>
struct ValueWrapper {
    static constexpr auto value = N;
    void print() const {
        std::cout << "值: " << N << " (类型: " << typeid(N).name() << ")" << std::endl;
    }
};

int main() {
    ValueWrapper<42> v1;
    ValueWrapper<3.14> v2;
    v1.print();  // 值: 42 (类型: int)
    v2.print();  // 值: 3.14 (类型: double)
}
```

## 三、注意事项与常见陷阱

- NTTP必须是编译期常量表达式，不能使用运行时变量
- 浮点型NTTP在C++20之前不被标准支持（部分编译器有扩展）
- 类类型NTTP必须是字面类型（C++20）
- 字符串字面量不能直接作为NTTP，但字符数组可以
- NTTP使得模板实例化更加多样化，`FixedArray<int,3>`和`FixedArray<int,4>`是不同类型
- C++20允许使用`auto`简化NTTP声明
