# 类模板偏特化

## 一、概念说明

偏特化（Partial Specialization）允许只特化部分模板参数，或对满足某种模式的类型进行特化。偏特化仍然是模板，只是比通用模板更具体。这是类模板独有的能力，函数模板不支持偏特化。

## 二、具体用法

### 2.1 部分参数特化

```cpp
// 通用模板：两个类型参数
template <typename T1, typename T2>
class MyPair {
public:
    T1 first;
    T2 second;
    void typeInfo() const {
        std::cout << "通用: 两个不同类型" << std::endl;
    }
};

// 偏特化：两个参数类型相同
template <typename T>
class MyPair<T, T> {
public:
    T first, second;
    void typeInfo() const {
        std::cout << "偏特化: 两个相同类型" << std::endl;
    }
    T sum() const { return first + second; }  // 额外功能
};

int main() {
    MyPair<int, double> p1;
    p1.typeInfo();  // 通用: 两个不同类型

    MyPair<int, int> p2;
    p2.typeInfo();  // 偏特化: 两个相同类型
    p2.first = 10; p2.second = 20;
    std::cout << p2.sum() << std::endl;  // 30
}
```

### 2.2 指针类型特化

```cpp
template <typename T>
class SmartContainer {
    std::vector<T> data;
public:
    void add(const T& val) { data.push_back(val); }
    void show() const {
        for (const auto& v : data) std::cout << v << " ";
        std::cout << std::endl;
    }
};

// 偏特化：指针类型
template <typename T>
class SmartContainer<T*> {
    std::vector<T*> data;
public:
    void add(T* val) { data.push_back(val); }
    void show() const {
        for (const auto* v : data) std::cout << *v << " ";
        std::cout << std::endl;
    }
    ~SmartContainer() {
        for (auto* p : data) delete p;  // 自动管理指针
    }
};

int main() {
    SmartContainer<int> sc1;
    sc1.add(10); sc1.add(20);
    sc1.show();  // 10 20

    SmartContainer<int*> sc2;
    sc2.add(new int(30));
    sc2.add(new int(40));
    sc2.show();  // 30 40
}
```

### 2.3 引用类型特化

```cpp
template <typename T>
class Wrapper {
    T value;
public:
    Wrapper(T v) : value(v) {}
    T get() const { return value; }
};

// 偏特化：引用类型（存储指针而非引用）
template <typename T>
class Wrapper<T&> {
    T* ptr;
public:
    Wrapper(T& v) : ptr(&v) {}
    T& get() const { return *ptr; }
    void set(T& v) { ptr = &v; }
};

int main() {
    int x = 42;
    Wrapper<int> w1(x);
    std::cout << w1.get() << std::endl;  // 42（值拷贝）

    Wrapper<int&> w2(x);
    x = 99;
    std::cout << w2.get() << std::endl;  // 92（引用，看到x的变化）
}
```

## 三、注意事项与常见陷阱

- 偏特化只能用于类模板，不能用于函数模板
- 偏特化的模板参数个数可以与通用模板不同
- 编译器选择最特化的版本：全特化 > 偏特化 > 通用模板
- 偏特化的参数列表必须与通用模板"模式匹配"
- 指针特化`T*`、引用特化`T&`、`const T`特化都是常见的偏特化形式
- 多个偏特化可能导致歧义，需确保没有两个偏特化同样匹配
