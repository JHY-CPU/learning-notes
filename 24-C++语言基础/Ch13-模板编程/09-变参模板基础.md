# 变参模板基础

## 一、概念说明

变参模板（Variadic Template）是C++11引入的特性，允许模板接受任意数量和类型的参数。通过**参数包**（Parameter Pack）和**递归展开**实现，彻底解决了以往需要为不同参数数量编写多个重载的问题。

## 二、具体用法

### 2.1 参数包语法

```cpp
// Args 是模板参数包，args 是函数参数包
template <typename... Args>
void print_all(Args... args) {
    std::cout << "参数个数: " << sizeof...(args) << std::endl;
}

int main() {
    print_all();                    // 参数个数: 0
    print_all(1);                   // 参数个数: 1
    print_all(1, 3.14, "hello");    // 参数个数: 3
}
```

### 2.2 递归展开

```cpp
// 基本情况：无参数
void print_recursive() {
    std::cout << "（结束）" << std::endl;
}

// 递归情况：至少一个参数
template <typename T, typename... Args>
void print_recursive(T first, Args... rest) {
    std::cout << first << " ";
    print_recursive(rest...);  // 递归调用，每次减少一个参数
}

int main() {
    print_recursive(1, 2.5, "hello", 'A');
    // 输出: 1 2.5 hello A （结束）
}
```

### 2.3 折叠表达式（C++17简化）

```cpp
// C++17 折叠表达式替代递归
template <typename... Args>
auto sum(Args... args) {
    return (args + ...);  // 一元右折叠
}

template <typename... Args>
void print_fold(Args... args) {
    ((std::cout << args << " "), ...);  // 逗号折叠
    std::cout << std::endl;
}

int main() {
    std::cout << sum(1, 2, 3, 4, 5) << std::endl;  // 15
    std::cout << sum(1.1, 2.2, 3.3) << std::endl;   // 6.6
    print_fold("hello", 42, 3.14, 'A');  // hello 42 3.14 A
}
```

### 2.4 包展开位置

```cpp
// 参数包可以在多种上下文中展开
template <typename... Args>
class TupleHolder {
public:
    // 用参数包初始化基类链
    static void demo(Args... args) {
        // 1. 初始化列表展开
        int arr[] = { static_cast<int>(args)... };
        for (auto v : arr) std::cout << v << " ";
        std::cout << std::endl;

        // 2. 函数调用展开
        (void)std::initializer_list<int>{
            (std::cout << args << " ", 0)...
        };
        std::cout << std::endl;
    }
};

int main() {
    TupleHolder<int,int,int>::demo(10, 20, 30);
    // 10 20 30
    // 10 20 30
}
```

## 三、注意事项与常见陷阱

- `sizeof...(args)`返回参数包中的参数个数（不含展开）
- 递归展开必须有一个基本情况（无参数版本）来终止递归
- C++17折叠表达式大大简化了变参模板的编写
- 递归展开时参数包名后面必须加`...`表示展开
- 变参模板是实现完美转发、`std::make_tuple`、`std::function`等的基础
- 空参数包`Args...`也是合法的（零参数情况）
