# 变量模板（C++14）

## 一、概念说明

变量模板（Variable Template）是C++14引入的特性，允许定义模板化的变量。它使得编译期常量和类型相关值可以参数化，替代了以往用类模板静态成员`::value`的繁琐写法。

## 二、具体用法

### 2.1 基本变量模板

```cpp
// 编译期数学常量
template <typename T>
constexpr T pi = T(3.14159265358979323846);

template <typename T>
constexpr T e = T(2.71828182845904523536);

int main() {
    std::cout << std::setprecision(15);
    std::cout << pi<double> << std::endl;  // 3.14159265358979
    std::cout << pi<float> << std::endl;   // 3.14159274101257 (单精度)
    std::cout << e<double> << std::endl;   // 2.71828182845905

    // 用于计算面积
    double r = 5.0;
    std::cout << "圆面积: " << pi<double> * r * r << std::endl;  // 78.5398
}
```

### 2.2 替代类模板静态成员

```cpp
// 传统写法（类模板静态成员）
template <typename T>
struct IsIntegralOld {
    static constexpr bool value = std::is_integral<T>::value;
};

// C++14 变量模板
template <typename T>
constexpr bool IsIntegral = std::is_integral_v<T>;

// 标准库中的 _v 后缀就是变量模板
// std::is_integral_v<T> 等价于 std::is_integral<T>::value

int main() {
    std::cout << std::boolalpha;
    std::cout << IsIntegral<int> << std::endl;     // true
    std::cout << IsIntegral<double> << std::endl;   // false
    std::cout << std::is_same_v<int, int32_t> << std::endl;  // true
}
```

### 2.3 类型相关的编译期值

```cpp
// 不同类型的缓冲区大小
template <typename T>
constexpr std::size_t buffer_size = 1024;

template <>
constexpr std::size_t buffer_size<double> = 2048;

template <>
constexpr std::size_t buffer_size<std::string> = 4096;

int main() {
    std::cout << buffer_size<int> << std::endl;          // 1024
    std::cout << buffer_size<double> << std::endl;       // 2048
    std::cout << buffer_size<std::string> << std::endl;  // 4096
}
```

### 2.4 配合constexpr函数

```cpp
// 变量模板+constexpr函数
template <typename T>
constexpr T degrees_to_radians(T deg) {
    return deg * pi<T> / T(180);
}

template <typename T>
constexpr T golden_ratio = (T(1) + std::sqrt(T(5))) / T(2);

int main() {
    std::cout << std::setprecision(6);
    std::cout << degrees_to_radians(180.0) << std::endl;  // 3.14159
    std::cout << golden_ratio<double> << std::endl;       // 1.61803
}
```

## 三、注意事项与常见陷阱

- 变量模板在头文件中定义时需要`inline`（C++17）避免多重定义
- `constexpr`变量模板隐含`inline`（C++17起）
- 变量模板不能被重载（每个名称只能有一个模板）
- 标准库的`_v`后缀（如`is_integral_v<T>`）就是变量模板的应用
- 变量模板可以被特化（全特化和偏特化）
- 变量模板使得编译期常量的使用更加直观简洁
