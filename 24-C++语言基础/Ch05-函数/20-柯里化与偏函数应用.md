# 柯里化与偏函数应用

## 一、概念说明

**偏函数应用**（Partial Function Application）是将函数的部分参数固定，产生一个接受剩余参数的新函数。**柯里化**（Currying）是将多参数函数转换为一系列单参数函数的技术。

C++没有原生柯里化支持，但可通过lambda、`std::bind`或模板实现。

## 二、具体用法

### 2.1 偏函数应用（lambda方式）

```cpp
auto add = [](int a, int b) { return a + b; };

// 固定第一个参数
auto add5 = [add](int x) { return add(5, x); };

std::cout << add5(3) << std::endl;   // 输出: 8
std::cout << add5(10) << std::endl;  // 输出: 15
```

### 2.2 偏函数应用（std::bind）

```cpp
int multiply(int a, int b) { return a * b; }

auto triple = std::bind(multiply, 3, std::placeholders::_1);
std::cout << triple(7) << std::endl;  // 输出: 21
```

### 2.3 柯里化实现

```cpp
// 返回lambda的lambda
auto curriedAdd = [](int a) {
    return [a](int b) {
        return [a, b](int c) {
            return a + b + c;
        };
    };
};

auto add10 = curriedAdd(10);
auto add10and20 = add10(20);
std::cout << add10and20(30) << std::endl;  // 输出: 60

// 一步到位
std::cout << curriedAdd(1)(2)(3) << std::endl;  // 输出: 6
```

### 2.4 模板化通用偏函数应用

```cpp
template <typename F, typename... BoundArgs>
auto partial(F func, BoundArgs... bound) {
    return [func, bound...](auto&&... args) {
        return func(bound..., std::forward<decltype(args)>(args)...);
    };
}

int divide(int a, int b) { return a / b; }
auto halfOf = partial(divide, std::placeholders::_1, 2);
// 使用bind实现
auto half = std::bind(divide, std::placeholders::_1, 2);
std::cout << half(10) << std::endl;  // 输出: 5
```

## 三、注意事项与常见陷阱

- 柯里化函数的调用必须严格按照参数顺序传递
- 偏函数应用固定部分参数，柯里化每次只接受一个参数
- lambda捕获会使柯里化的函数不可拷贝（如有不可拷贝对象）
- 实际项目中偏函数应用比柯里化更常用
- 函数式编程库（如Boost）提供了更完整的柯里化支持
