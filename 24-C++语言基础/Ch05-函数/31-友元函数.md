# 友元函数

## 一、概念说明

`friend`关键字允许特定的非成员函数或类访问另一个类的**私有和保护成员**。友元函数不是类的成员函数，但拥有访问类私有成员的权限。友元关系**不能传递**（A是B的友元，B是C的友元，不代表A是C的友元）且**不能继承**。

## 二、具体用法

### 2.1 友元函数访问私有成员

```cpp
class BankAccount {
    double balance;
public:
    BankAccount(double b) : balance(b) {}
    // 声明友元函数
    friend void audit(const BankAccount& acc);
};

void audit(const BankAccount& acc) {
    // 可直接访问私有成员balance
    std::cout << "余额: " << acc.balance << std::endl;
}

BankAccount acc(10000.0);
audit(acc);
// 输出: 余额: 10000
```

### 2.2 友元函数实现运算符重载

```cpp
class Vector2D {
    double x, y;
public:
    Vector2D(double x, double y) : x(x), y(y) {}
    friend Vector2D operator+(const Vector2D& a, const Vector2D& b);
    friend std::ostream& operator<<(std::ostream& os, const Vector2D& v);
};

Vector2D operator+(const Vector2D& a, const Vector2D& b) {
    return Vector2D(a.x + b.x, a.y + b.y);
}

std::ostream& operator<<(std::ostream& os, const Vector2D& v) {
    return os << "(" << v.x << ", " << v.y << ")";
}

Vector2D v1(1, 2), v2(3, 4);
std::cout << v1 + v2 << std::endl;
// 输出: (4, 6)
```

### 2.3 友元类

```cpp
class Engine {
    int rpm;
public:
    Engine(int r) : rpm(r) {}
    friend class Car;  // Car可访问Engine的私有成员
};

class Car {
public:
    void showEngine(const Engine& e) {
        std::cout << "RPM: " << e.rpm << std::endl;
    }
};
```

## 三、注意事项与常见陷阱

- 友元关系破坏了封装，应谨慎使用
- 友元不能被继承，子类不是父类友元的友元
- 友元声明在类内，但函数定义在类外（不加friend）
- `operator<<`和`operator>>`通常需要声明为友元（左操作数不是类对象）
- 过多友元说明接口设计可能有问题
