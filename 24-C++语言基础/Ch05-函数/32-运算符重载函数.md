# 运算符重载函数

## 一、概念说明

运算符重载允许为自定义类型定义运算符的行为。重载可以作为**成员函数**或**非成员函数**实现。选择原则：一元运算符通常为成员函数，二元运算符若需要左操作数隐式转换则为非成员函数。

## 二、具体用法

### 2.1 成员函数重载

```cpp
class Fraction {
    int num, den;
public:
    Fraction(int n, int d) : num(n), den(d) {}

    // 二元运算符作为成员函数（左操作数是this）
    Fraction operator+(const Fraction& rhs) const {
        return Fraction(num * rhs.den + rhs.num * den,
                        den * rhs.den);
    }

    // 一元运算符
    Fraction operator-() const {
        return Fraction(-num, den);
    }

    // 复合赋值运算符
    Fraction& operator+=(const Fraction& rhs) {
        *this = *this + rhs;
        return *this;
    }

    friend std::ostream& operator<<(std::ostream& os, const Fraction& f);
};

std::ostream& operator<<(std::ostream& os, const Fraction& f) {
    return os << f.num << "/" << f.den;
}

Fraction a(1, 2), b(1, 3);
std::cout << a + b << std::endl;   // 输出: 5/6
std::cout << -a << std::endl;      // 输出: -1/2
```

### 2.2 非成员函数重载（对称性）

```cpp
class Vector {
public:
    double x, y;
    Vector(double x, double y) : x(x), y(y) {}

    // 非成员函数实现 double * Vector
    friend Vector operator*(double scalar, const Vector& v);
    friend Vector operator*(const Vector& v, double scalar);
};

Vector operator*(double s, const Vector& v) {
    return Vector(s * v.x, s * v.y);
}

Vector operator*(const Vector& v, double s) {
    return s * v;  // 复用上面的版本
}

Vector v(1, 2);
std::cout << (2.0 * v).x << ", " << (2.0 * v).y << std::endl;
// 输出: 2, 4
```

### 2.3 比较运算符（C++20飞船运算符）

```cpp
#include <compare>

class Version {
    int major_, minor_, patch_;
public:
    Version(int maj, int min, int pat)
        : major_(maj), minor_(min), patch_(pat) {}

    // C++20: 一个运算符生成==, !=, <, >, <=, >=
    auto operator<=>(const Version&) const = default;
};

Version v1(1, 2, 3), v2(1, 2, 4);
std::cout << (v1 < v2) << std::endl;   // 输出: 1
std::cout << (v1 == v2) << std::endl;  // 输出: 0
```

### 2.4 下标运算符

```cpp
class SafeArray {
    int data[10];
public:
    int& operator[](size_t i) {
        if (i >= 10) throw std::out_of_range("越界");
        return data[i];
    }
    const int& operator[](size_t i) const {
        if (i >= 10) throw std::out_of_range("越界");
        return data[i];
    }
};

SafeArray arr;
arr[0] = 42;
std::cout << arr[0] << std::endl;  // 输出: 42
```

## 三、注意事项与常见陷阱

- 不能创建新的运算符，只能重载已有运算符
- 部分运算符不能重载：`.`、`::`、`sizeof`、`?:`、`typeid`
- `=`、`[]`、`()`、`->`必须是成员函数
- 重载运算符应保持语义一致性（`+`不应修改对象）
- `++`前置和后置通过参数区分：`operator++()` vs `operator++(int)`
