# 变参函数

## 一、概念说明

变参函数（Variadic Functions/Templates）允许接受**不定数量**的参数。C++有两套变参机制：
- **C风格 `va_list`**（C++11标准 §18.10）：类型不安全，不推荐
- **C++11变参模板**（Variadic Templates）：类型安全，现代C++首选

C++17引入**折叠表达式**（Fold Expressions），极大简化了参数包的展开写法。

## 二、具体用法

### 2.1 C风格变参（不推荐，仅供了解）

```cpp
#include <iostream>
#include <cstdarg>
using namespace std;

double average(int count, ...) {
    va_list args;
    va_start(args, count);
    double sum = 0;
    for (int i = 0; i < count; ++i) {
        sum += va_arg(args, double);  // 必须知道确切类型
    }
    va_end(args);
    return sum / count;
}

int main() {
    cout << average(3, 1.0, 2.0, 3.0) << endl;  // 2
    // average(3, 1, 2, 3);  // 错误！va_arg读取double但传入int
    return 0;
}
```

**C风格变参的缺陷**：无类型检查、需手动传递参数个数、整型提升规则复杂。

### 2.2 C++11变参模板 — 递归展开

```cpp
#include <iostream>
using namespace std;

// 递归终止条件（基函数）
void print() {
    cout << endl;
}

// 参数包展开：处理第一个，递归处理剩余
template <typename T, typename... Args>
void print(T first, Args... rest) {
    cout << first;
    if constexpr (sizeof...(rest) > 0) cout << " ";
    print(rest...);  // 递归调用，参数包减一
}

int main() {
    print(1, 2.5, "hello", 'A');
    // 输出: 1 2.5 hello A

    print("单一参数");
    // 输出: 单一参数

    return 0;
}
```

### 2.3 C++17折叠表达式（推荐）

```cpp
#include <iostream>
using namespace std;

// 一元右折叠
template <typename... Args>
auto sum(Args... args) {
    return (args + ...);  // ((a + b) + c) + d
}

// 一元左折叠
template <typename... Args>
auto sumLeft(Args... args) {
    return (... + args);  // a + (b + (c + d))
}

// 带初始值的折叠（空参数包安全）
template <typename... Args>
auto sumSafe(Args... args) {
    return (args + ... + 0);  // 空包返回0
}

// 逗号折叠：执行每个参数的副作用
template <typename... Args>
void printAll(Args... args) {
    ((cout << args << " "), ...);  // 逗号折叠
    cout << endl;
}

// 参数个数
template <typename... Args>
size_t countArgs(Args... args) {
    return sizeof...(args);
}

int main() {
    cout << sum(1, 2, 3, 4, 5) << endl;       // 15
    cout << sumSafe() << endl;                  // 0（空包）
    printAll(1, "hello", 3.14, 'A');           // 1 hello 3.14 A
    cout << "参数个数: " << countArgs(1, 'a', 3.14) << endl;  // 3

    return 0;
}
```

### 2.4 完美转发参数包

```cpp
#include <iostream>
#include <memory>
#include <utility>
using namespace std;

// 工厂函数：完美转发构造参数
template <typename T, typename... Args>
unique_ptr<T> make(Args&&... args) {
    return unique_ptr<T>(new T(forward<Args>(args)...));
}

// 可变参数的emplace式操作
template <typename Container, typename... Args>
void emplaceBack(Container& c, Args&&... args) {
    c.emplace_back(forward<Args>(args)...);
}

int main() {
    auto str = make<string>("hello", 5, 'x');  // string("hello", 5, 'x')
    cout << *str << endl;  // helloxxxxx

    return 0;
}
```

### 2.5 编译期参数包处理

```cpp
#include <iostream>
#include <type_traits>
using namespace std;

// 检查所有参数是否为同一类型
template <typename T, typename... Args>
bool allSameType() {
    return (is_same_v<T, Args> && ...);  // 折叠表达式
}

// 静态断言：所有参数必须是算术类型
template <typename... Args>
auto safeSum(Args... args) {
    static_assert((is_arithmetic_v<Args> && ...), "所有参数必须是算术类型");
    return (args + ...);
}

int main() {
    cout << allSameType<int, int, int>() << endl;    // 1
    cout << allSameType<int, double, int>() << endl;  // 0
    cout << safeSum(1, 2.5, 3) << endl;              // 6.5

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **C风格变参不进行类型检查**：容易出错，现代C++应避免使用
2. **变参模板的参数包展开有多种方式**：递归（C++11）、折叠表达式（C++17，推荐）、逗号展开
3. **sizeof...(args)返回参数个数但不展开参数包**：编译期常量
4. **空参数包的递归需要提供终止函数**：折叠表达式则可用初始值处理
5. **折叠表达式是C++17最推荐的参数包处理方式**：简洁且无递归开销
6. **参数包转发用std::forward<Args>(args)...**：保持值类别
7. **变参模板可用于类模板**：如 `tuple<T...>`、`variant<T...>`
