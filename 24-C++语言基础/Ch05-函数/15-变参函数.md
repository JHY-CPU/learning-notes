# 变参函数

## 一、概念说明

变参函数允许接受**不定数量**的参数。C++有两种变参机制：C风格的`va_list`（类型不安全）和C++11引入的**变参模板**（variadic templates，类型安全）。

现代C++应优先使用变参模板。

## 二、具体用法

### 2.1 C风格变参（不推荐）

```cpp
#include <cstdarg>

double average(int count, ...) {
    va_list args;
    va_start(args, count);
    double sum = 0;
    for (int i = 0; i < count; ++i) {
        sum += va_arg(args, double);
    }
    va_end(args);
    return sum / count;
}

std::cout << average(3, 1.0, 2.0, 3.0) << std::endl;
// 输出: 2
```

### 2.2 C++11变参模板

```cpp
// 递归终止条件
void print() {
    std::cout << std::endl;
}

// 递归展开参数包
template <typename T, typename... Args>
void print(T first, Args... rest) {
    std::cout << first << " ";
    print(rest...);  // 递归调用，参数包减一
}

print(1, 2.5, "hello", 'A');
// 输出: 1 2.5 hello A
```

### 2.3 折叠表达式（C++17）

```cpp
// C++17简化了参数包展开
template <typename... Args>
auto sum(Args... args) {
    return (args + ...);  // 一元右折叠
}

std::cout << sum(1, 2, 3, 4, 5) << std::endl;
// 输出: 15

// 参数个数
template <typename... Args>
size_t countArgs(Args... args) {
    return sizeof...(args);
}

std::cout << countArgs(1, 'a', 3.14) << std::endl;
// 输出: 3
```

### 2.4 转发参数包

```cpp
template <typename... Args>
void forwardTo(Args&&... args) {
    otherFunc(std::forward<Args>(args)...);
}
```

## 三、注意事项与常见陷阱

- C风格变参不进行类型检查，容易出错
- 变参模板的参数包展开有多种方式：递归、折叠表达式、逗号展开
- `sizeof...(args)`返回参数个数但不展开参数包
- 空参数包的递归需要提供终止函数
- 折叠表达式是C++17最推荐的参数包处理方式
