# 默认参数

## 一、概念说明

默认参数（Default Arguments，C++11标准 §8.3.6）允许函数在调用时省略部分实参，编译器自动填入预设的默认值。这在需要向后兼容或提供便捷API时非常有用。

核心规则：
- 默认参数必须从**右向左**连续指定，中间不能跳过
- 默认值只能在**声明**处指定，不能在定义处重复指定
- 默认参数在**调用点**求值，不是定义点

## 二、具体用法

### 2.1 基本用法

```cpp
#include <iostream>
#include <string>
using namespace std;

void printMessage(const string& msg,
                  const string& prefix = "[INFO]",
                  bool newline = true) {
    cout << prefix << " " << msg;
    if (newline) cout << endl;
}

int main() {
    printMessage("Hello");                  // [INFO] Hello
    printMessage("Error", "[ERROR]");       // [ERROR] Error
    printMessage("Debug", "[DBG]", false);  // [DBG] Debug (无换行)

    return 0;
}
```

### 2.2 声明与定义分离

```cpp
// ========== header.h ==========
// 头文件中指定默认值（只需在声明处指定）
void connect(const string& host,
             int port = 8080,
             int timeout = 30);

// ========== source.cpp ==========
// 源文件中不再指定默认值
void connect(const string& host, int port, int timeout) {
    cout << "Connecting to " << host << ":" << port
         << " timeout=" << timeout << "s" << endl;
}

// 使用
int main() {
    connect("localhost");        // port=8080, timeout=30
    connect("localhost", 3306);  // timeout=30
    return 0;
}
```

### 2.3 从右向左连续规则

```cpp
// 正确：从右向左连续
void f(int a, int b = 2, int c = 3);

// 错误：中间跳过
// void f(int a, int b = 2, int c);  // 编译错误

// 正确：可以只给部分参数默认值
void g(int a, int b = 0, int c = 0);  // OK
```

### 2.4 默认参数与重载

```cpp
#include <iostream>
using namespace std;

// 用默认参数可以替代简单的重载
void log(const string& msg, int level = 1) {
    cout << "[L" << level << "] " << msg << endl;
}
// 等价于同时提供 log(string) 和 log(string, int) 两个版本

// 但注意歧义：
// void log(const string& msg) { }  // 与上面的log(string)歧义

int main() {
    log("系统启动");       // level=1
    log("警告信息", 3);   // level=3
    return 0;
}
```

### 2.5 默认参数的求值时机

```cpp
#include <iostream>
using namespace std;

int counter = 0;

int getCounter() {
    return ++counter;
}

void show(int id, int seq = getCounter()) {
    cout << "id=" << id << " seq=" << seq << endl;
}

int main() {
    // 默认参数在每次调用时求值
    show(1);    // id=1 seq=1 (counter变为1)
    show(2);    // id=2 seq=2 (counter变为2)
    show(3, 99); // id=3 seq=99 (不调用getCounter)

    return 0;
}
```

## 三、虚函数与默认参数

```cpp
#include <iostream>
using namespace std;

struct Base {
    virtual void f(int x = 10) { cout << "Base: " << x << endl; }
};

struct Derived : Base {
    void f(int x = 20) override { cout << "Derived: " << x << endl; }
};

int main() {
    Derived d;
    Base& b = d;

    b.f();      // 调用Derived::f，但默认参数是Base的 → 输出: Derived: 10
    d.f();      // 调用Derived::f，默认参数是Derived的 → 输出: Derived: 20

    // 警告：虚函数的默认参数根据静态类型决定，可能导致意外结果
    return 0;
}
```

## 四、注意事项与常见陷阱

1. **默认值只能在声明处指定**：头文件中声明时给默认值，源文件定义处不能再给
2. **参数必须从右向左连续默认**：`void f(int a, int b=1, int c)` 是编译错误
3. **默认参数不参与重载决议**：与同名函数可能引起歧义
4. **默认参数表达式在调用点求值**：每次调用时重新求值，不是定义时一次求值
5. **虚函数的默认参数根据静态类型决定**：通过基类引用调用时使用基类的默认值，可能非预期
6. **默认参数不能是局部变量**：必须是能在调用点访问的表达式
7. **lambda不能有默认参数**：但可以捕获默认值
