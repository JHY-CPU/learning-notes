# 回调函数模式

## 一、概念说明

回调函数是一种设计模式：将一个函数（回调）作为参数传递给另一个函数，在特定事件或条件发生时被调用。C++中实现回调的方式包括：函数指针、`std::function`、lambda和仿函数。

## 二、具体用法

### 2.1 函数指针回调

```cpp
// 排序时的比较回调
int compareAsc(const void* a, const void* b) {
    return (*(int*)a - *(int*)b);
}

int arr[] = {5, 3, 1, 4, 2};
std::qsort(arr, 5, sizeof(int), compareAsc);
for (int x : arr) std::cout << x << " ";
// 输出: 1 2 3 4 5
```

### 2.2 std::function回调（推荐）

```cpp
class EventSystem {
    std::vector<std::function<void(const std::string&)>> handlers;
public:
    void addHandler(std::function<void(const std::string&)> cb) {
        handlers.push_back(std::move(cb));
    }
    void emit(const std::string& event) {
        for (auto& h : handlers) h(event);
    }
};

EventSystem events;
events.addHandler([](const std::string& e) {
    std::cout << "Logger: " << e << std::endl;
});
events.addHandler([](const std::string& e) {
    std::cout << "UI: show " << e << std::endl;
});
events.emit("button_clicked");
// 输出:
// Logger: button_clicked
// UI: show button_clicked
```

### 2.3 回调与异步操作

```cpp
void asyncDownload(const std::string& url,
                   std::function<void(int, const std::string&)> onComplete) {
    // 模拟异步操作
    int status = 200;
    std::string data = "downloaded content";
    onComplete(status, data);
}

asyncDownload("https://example.com", [](int code, const std::string& data) {
    std::cout << "Status: " << code << ", Size: " << data.size() << std::endl;
});
// 输出: Status: 200, Size: 17
```

## 三、注意事项与常见陷阱

- 回调中捕获的引用需注意对象生命周期
- 回调的执行顺序通常不被保证（取决于实现）
- 避免回调地狱（callback hell），考虑使用future/promise或协程
- `std::function`有性能开销，性能敏感场景考虑模板参数或函数指针
- 回调可能在不同线程执行，需注意线程安全
