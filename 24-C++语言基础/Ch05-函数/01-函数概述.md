# 函数概述

## 一、概念说明

函数是C++程序的基本构建块，用于封装一段可重复使用的代码逻辑。函数将输入（参数）经过处理后产生输出（返回值），实现代码的模块化和复用。

C++中函数的使用包括三个环节：**声明**（告诉编译器函数的存在）、**定义**（提供函数的具体实现）、**调用**（在需要的地方执行函数）。

```cpp
// 函数声明（函数原型）
int add(int a, int b);

// 函数定义
int add(int a, int b) {
    return a + b;
}

// 函数调用
int result = add(3, 5);  // result = 8
```

## 二、具体用法

### 2.1 函数定义的完整语法

```cpp
返回类型 函数名(参数列表) {
    // 函数体
    return 返回值;  // void函数可省略return
}

// 示例：计算圆的面积
double circleArea(double radius) {
    const double PI = 3.14159265;
    return PI * radius * radius;
}
```

```cpp
// 输出示例
double area = circleArea(5.0);
// area = 78.5398
```

### 2.2 函数声明与定义分离

大型项目中，声明通常放在头文件（`.h`），定义放在源文件（`.cpp`）。

```cpp
// math_utils.h —— 声明
#ifndef MATH_UTILS_H
#define MATH_UTILS_H

int factorial(int n);
double power(double base, int exp);

#endif

// math_utils.cpp —— 定义
#include "math_utils.h"

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```

### 2.3 void函数（无返回值）

```cpp
void printMessage(const std::string& msg) {
    std::cout << msg << std::endl;
}

void printMessage("Hello, C++!");
// 输出: Hello, C++!
```

## 三、注意事项与常见陷阱

- 函数必须先声明或定义后使用，否则编译器报错
- 非`void`函数必须有`return`语句，否则行为未定义
- 函数不能嵌套定义（C++不允许函数中定义函数）
- 返回局部变量的引用或指针是危险操作，会导致悬垂引用
- 函数声明可以省略参数名，但定义中通常需要参数名
