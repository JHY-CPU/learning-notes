# 函数返回值

## 一、概念说明

函数的返回值是函数执行后传递给调用者的结果。C++提供了多种返回方式：返回值、返回引用、返回右值引用（移动语义），以及编译器优化（RVO/NRVO）机制。

**RVO（Return Value Optimization）** 是编译器的重要优化，允许直接在调用者的栈空间中构造返回对象，省去拷贝/移动操作。

## 二、具体用法

### 2.1 基本返回值

```cpp
int max(int a, int b) {
    return (a > b) ? a : b;
}

std::string greet(const std::string& name) {
    return "Hello, " + name + "!";
}

std::cout << greet("Alice") << std::endl;
// 输出: Hello, Alice!
```

### 2.2 返回引用

```cpp
// 返回数组元素的引用，允许修改
int& getElement(std::vector<int>& vec, size_t index) {
    return vec[index];
}

std::vector<int> nums = {10, 20, 30};
getElement(nums, 1) = 99;  // nums[1]变为99
std::cout << nums[1] << std::endl;
// 输出: 99
```

### 2.3 RVO与NRVO

```cpp
class BigObject {
    std::vector<int> data;
public:
    BigObject() { std::cout << "构造\n"; }
    BigObject(const BigObject&) { std::cout << "拷贝\n"; }
    BigObject(BigObject&&) { std::cout << "移动\n"; }
};

// NRVO（Named RVO）：编译器优化掉拷贝
BigObject create() {
    BigObject obj;    // 命名对象
    return obj;
}

// 调用 create() 输出：构造 （无拷贝/移动）
```

### 2.4 返回移动语义

```cpp
std::vector<int> createVector() {
    std::vector<int> result = {1, 2, 3, 4, 5};
    return result;  // 隐式移动（C++11起）
}
```

## 三、注意事项与常见陷阱

- 不要返回局部变量的引用或指针（悬垂引用）
- RVO是C++标准允许的优化，即使有副作用也可执行
- 返回`std::move(local_var)`反而会**阻止**NRVO优化
- `auto`返回类型推导（C++14）：函数体中必须有return语句
- 返回引用的函数不能返回临时对象的引用
