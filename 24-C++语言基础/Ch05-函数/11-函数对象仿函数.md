# 函数对象（仿函数）

## 一、概念说明

函数对象（Functor/仿函数）是**重载了`operator()`运算符的类的对象**。它可以像函数一样被调用，同时保留了类的特性——可以拥有状态（成员变量）、构造函数等。

相比普通函数指针，仿函数通常更容易被编译器内联优化。

## 二、具体用法

### 2.1 基本仿函数

```cpp
struct Multiplier {
    int factor;
    Multiplier(int f) : factor(f) {}

    int operator()(int value) const {
        return value * factor;
    }
};

Multiplier doubler(2);
Multiplier tripler(3);

std::cout << doubler(5) << std::endl;   // 输出: 10
std::cout << tripler(5) << std::endl;   // 输出: 15
```

### 2.2 有状态仿函数

```cpp
class Counter {
    int count = 0;
public:
    void operator()() { count++; }
    int getCount() const { return count; }
};

Counter counter;
counter();
counter();
counter();
std::cout << "Count: " << counter.getCount() << std::endl;
// 输出: Count: 3
```

### 2.3 在STL中使用

```cpp
struct CompareDesc {
    bool operator()(int a, int b) const {
        return a > b;
    }
};

std::vector<int> nums = {3, 1, 4, 1, 5, 9};
std::sort(nums.begin(), nums.end(), CompareDesc());

for (int n : nums) std::cout << n << " ";
// 输出: 9 5 4 3 1 1
```

### 2.4 标准库预定义仿函数

```cpp
#include <functional>

std::plus<int> add;
std::cout << add(3, 5) << std::endl;           // 输出: 8

std::greater<int> gt;
std::cout << gt(5, 3) << std::endl;            // 输出: 1 (true)

std::multiplies<int> mul;
std::cout << mul(4, 7) << std::endl;           // 输出: 28
```

## 三、注意事项与常见陷阱

- 仿函数的`operator()`通常应声明为`const`，除非需要修改状态
- 每个仿函数类型是**独立的类型**，即使行为相同
- 仿函数比函数指针更容易被编译器内联优化
- C++11后lambda常替代简单的仿函数，但复杂逻辑仍适合仿函数
- 仿函数作为模板参数时必须是类型，不能是函数指针
