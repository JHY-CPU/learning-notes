# 函数对象与排序

## 一、概念说明

在STL排序算法中，自定义**比较器**（comparator）是常见需求。比较器可以是函数指针、lambda表达式或仿函数，用于定义元素之间的排序规则。

`std::sort`要求比较器满足**严格弱序**（strict weak ordering）：不可自反、反对称、可传递。

## 二、具体用法

### 2.1 Lambda排序

```cpp
std::vector<int> nums = {5, 2, 8, 1, 9, 3};

// 升序（默认）
std::sort(nums.begin(), nums.end());

// 降序
std::sort(nums.begin(), nums.end(), [](int a, int b) {
    return a > b;
});

for (int n : nums) std::cout << n << " ";
// 输出: 9 8 5 3 2 1
```

### 2.2 按结构体成员排序

```cpp
struct Student {
    std::string name;
    int score;
};

std::vector<Student> students = {
    {"Alice", 85}, {"Bob", 92}, {"Charlie", 78}
};

// 按分数降序
std::sort(students.begin(), students.end(),
    [](const Student& a, const Student& b) {
        return a.score > b.score;
    });

for (const auto& s : students)
    std::cout << s.name << ": " << s.score << std::endl;
// 输出:
// Bob: 92
// Alice: 85
// Charlie: 78
```

### 2.3 仿函数作为比较器

```cpp
struct CompareByAbs {
    bool operator()(int a, int b) const {
        return std::abs(a) < std::abs(b);
    }
};

std::vector<int> vals = {-5, 3, -2, 7, -1};
std::sort(vals.begin(), vals.end(), CompareByAbs());

for (int v : vals) std::cout << v << " ";
// 输出: -1 -2 3 -5 7
```

### 2.4 多条件排序

```cpp
// 先按分数降序，分数相同按姓名升序
std::sort(students.begin(), students.end(),
    [](const Student& a, const Student& b) {
        if (a.score != b.score) return a.score > b.score;
        return a.name < b.name;
    });
```

## 三、注意事项与常见陷阱

- 比较器必须满足严格弱序，`comp(a, a)`必须为`false`
- `std::sort`要求随机访问迭代器，list用`list::sort()`
- 稳定排序使用`std::stable_sort`，保持相等元素的相对顺序
- 比较器的`operator()`应声明为`const`
- 自定义比较器比`operator<`更灵活，是STL的重要设计模式
