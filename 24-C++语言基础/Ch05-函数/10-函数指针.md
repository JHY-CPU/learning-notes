# 函数指针

## 一、概念说明

函数指针（Function Pointer，C++11标准 §8.3.3）是指向函数的指针变量，存储函数的入口地址。通过函数指针可以在运行时动态选择要调用的函数，实现回调机制和策略模式。

函数指针的类型由**返回类型**和**参数列表**共同决定。函数名可隐式转换为函数指针（退化）。

现代C++中，`std::function`和lambda提供了更灵活的替代方案，但函数指针更轻量（无类型擦除开销）。

## 二、具体用法

### 2.1 基本函数指针

```cpp
#include <iostream>
using namespace std;

int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }

int main() {
    // 声明函数指针：指向返回int、接受两个int参数的函数
    int (*op)(int, int);

    op = add;
    cout << "add(5,3) = " << op(5, 3) << endl;  // 8

    op = sub;
    cout << "sub(5,3) = " << op(5, 3) << endl;  // 2

    op = mul;
    cout << "mul(5,3) = " << op(5, 3) << endl;  // 15

    return 0;
}
```

### 2.2 使用typedef/using简化

```cpp
// typedef方式（C风格）
typedef int (*BinaryOp)(int, int);

// using方式（C++11推荐，更清晰）
using BinaryOp = int (*)(int, int);

// 使用简化后的类型
BinaryOp func = add;
cout << func(10, 20) << endl;  // 30

// 函数指针类型别名可以用于参数
void applyOp(BinaryOp op, int a, int b) {
    cout << op(a, b) << endl;
}
```

### 2.3 函数指针作为参数（回调函数）

```cpp
#include <iostream>
using namespace std;

// 函数指针作为参数：回调模式
void applyOperation(int* arr, size_t len, int (*op)(int)) {
    for (size_t i = 0; i < len; ++i) {
        arr[i] = op(arr[i]);
    }
}

int doubleValue(int x) { return x * 2; }
int negate(int x) { return -x; }

int main() {
    int nums[] = {1, 2, 3, 4, 5};

    applyOperation(nums, 5, doubleValue);
    for (int n : nums) cout << n << " ";  // 2 4 6 8 10
    cout << endl;

    applyOperation(nums, 5, negate);
    for (int n : nums) cout << n << " ";  // -2 -4 -6 -8 -10
    cout << endl;

    return 0;
}
```

### 2.4 函数指针数组

```cpp
#include <iostream>
using namespace std;

int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }

int main() {
    // 函数指针数组
    using Op = int(*)(int, int);
    Op operations[] = {add, sub, mul};
    const char* names[] = {"add", "sub", "mul"};

    for (int i = 0; i < 3; ++i) {
        cout << names[i] << "(10,3) = " << operations[i](10, 3) << endl;
    }
    // add(10,3) = 13
    // sub(10,3) = 7
    // mul(10,3) = 30

    return 0;
}
```

### 2.5 与现代C++替代方案对比

```cpp
#include <iostream>
#include <functional>
#include <vector>
using namespace std;

int main() {
    int offset = 10;

    // 函数指针：零开销，但不能捕获状态
    using FuncPtr = int(*)(int);
    FuncPtr doubleIt = [](int x) { return x * 2; };  // 无捕获OK

    // std::function：可捕获，有类型擦除开销
    function<int(int)> addOffset = [offset](int x) { return x + offset; };

    // lambda直接使用：零开销，编译器推导类型
    auto addOffset2 = [offset](int x) { return x + offset; };

    // 性能排序：函数指针 ≈ lambda直接使用 > std::function
    cout << doubleIt(5) << endl;     // 10
    cout << addOffset(5) << endl;    // 15
    cout << addOffset2(5) << endl;   // 15

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **函数指针不能指向重载函数**：除非使用显式转换 `static_cast<int(*)(int)>(&overloaded)`
2. **lambda无捕获时可隐式转换为函数指针**：有捕获则不能
3. **函数指针与std::function不同**：前者更轻量（直接调用），后者有类型擦除开销
4. **空函数指针调用是未定义行为**：调用前必须检查非空
5. **成员函数指针语法不同于普通函数指针**：需要 `ReturnType (ClassName::*ptr)(Args)`
6. **函数指针可作为模板非类型参数**：`template<int(*F)(int)>` 在编译期绑定
7. **std::function可替代几乎所有函数指针场景**：除非对性能有极致要求
