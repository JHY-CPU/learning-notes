# 函数指针

## 一、概念说明

函数指针是指向函数的指针变量，存储函数的入口地址。通过函数指针可以在运行时动态选择要调用的函数，实现回调机制和策略模式。

函数指针的类型由**返回类型**和**参数列表**共同决定。

## 二、具体用法

### 2.1 基本函数指针

```cpp
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }

// 声明函数指针
int (*op)(int, int);

op = add;
std::cout << "add(5,3) = " << op(5, 3) << std::endl;
// 输出: add(5,3) = 8

op = sub;
std::cout << "sub(5,3) = " << op(5, 3) << std::endl;
// 输出: sub(5,3) = 2
```

### 2.2 使用typedef/using简化

```cpp
// typedef方式
typedef int (*BinaryOp)(int, int);

// using方式（推荐，更清晰）
using BinaryOp = int (*)(int, int);

// 使用简化后的类型
BinaryOp func = add;
std::cout << func(10, 20) << std::endl;
// 输出: 30
```

### 2.3 函数指针作为参数（回调函数）

```cpp
void applyOperation(int* arr, size_t len, int (*op)(int)) {
    for (size_t i = 0; i < len; ++i) {
        arr[i] = op(arr[i]);
    }
}

int doubleValue(int x) { return x * 2; }

int nums[] = {1, 2, 3, 4, 5};
applyOperation(nums, 5, doubleValue);
for (int n : nums) std::cout << n << " ";
// 输出: 2 4 6 8 10
```

### 2.4 函数指针数组

```cpp
using Op = int(int, int);
Op* operations[] = {add, sub};
std::cout << operations[0](10, 3) << std::endl;  // 输出: 13
std::cout << operations[1](10, 3) << std::endl;  // 输出: 7
```

## 三、注意事项与常见陷阱

- 函数指针不能指向重载函数，除非使用显式转换
- lambda无捕获时可隐式转换为函数指针
- 函数指针与`std::function`不同，前者更轻量但功能受限
- 空函数指针调用是未定义行为
- 成员函数指针语法不同于普通函数指针
