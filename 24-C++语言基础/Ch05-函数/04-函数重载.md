# 函数重载

## 一、概念说明

函数重载允许在同一作用域内定义多个**同名函数**，只要它们的**参数列表不同**（类型、数量或顺序不同）。编译器根据调用时的实参类型自动选择最佳匹配的版本，称为**重载决议**（overload resolution）。

返回类型不同**不能**作为重载的依据。

## 二、具体用法

### 2.1 参数类型不同

```cpp
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

std::string add(const std::string& a, const std::string& b) {
    return a + b;
}

std::cout << add(3, 5) << std::endl;           // 输出: 8
std::cout << add(1.5, 2.3) << std::endl;       // 输出: 3.8
std::cout << add("Hi ", "there") << std::endl;  // 输出: Hi there
```

### 2.2 参数数量不同

```cpp
int multiply(int a, int b) { return a * b; }
int multiply(int a, int b, int c) { return a * b * c; }

std::cout << multiply(2, 3) << std::endl;      // 输出: 6
std::cout << multiply(2, 3, 4) << std::endl;   // 输出: 24
```

### 2.3 const重载

```cpp
class Text {
    std::string data;
public:
    // 非const版本
    char& operator[](size_t i) {
        std::cout << "非const版本\n";
        return data[i];
    }
    // const版本
    const char& operator[](size_t i) const {
        std::cout << "const版本\n";
        return data[i];
    }
};
```

## 三、注意事项与常见陷阱

- 仅返回类型不同不构成重载，编译报错
- 默认参数与重载可能产生歧义（ambiguous）调用
- `const`和非`const`成员函数可构成重载（基于`this`指针类型）
- 整数字面量默认`int`，浮点默认`double`，注意类型匹配
- 重载决议优先精确匹配，其次类型提升，最后标准转换
