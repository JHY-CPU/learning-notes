# IDE使用技巧 - Visual Studio

## 一、概念说明

Visual Studio（简称VS）是微软开发的**功能最完整**的C++ IDE。社区版免费，提供强大的编辑器、调试器、性能分析器和项目管理工具，是Windows平台C++开发的首选。

## 二、具体用法

### 2.1 创建项目

```
文件 → 新建 → 项目 → Visual C++ → 空项目
或者
文件 → 新建 → 项目 → CMake → CMake项目
```

项目类型说明：
- **控制台应用**：命令行程序，有`main`函数入口
- **空项目**：不含任何文件的空项目模板
- **CMake项目**：使用CMake管理构建
- **静态库/动态库**：创建`.lib`或`.dll`库文件

### 2.2 项目属性配置

```
右键项目 → 属性 → 配置属性
```

```
常规 → C++语言标准 → ISO C++17标准 (/std:c++17)
C/C++ → 常规 → 警告等级 → 等级4 (/W4)
C/C++ → 常规 → 将警告视为错误 → 是 (/WX)
C/C++ → 优化 → 优化 → 已禁用 (/Od)（调试）/ 最大优化 (/O2)（发布）
C/C++ → 预处理器 → 预处理器定义 → DEBUG_MODE
链接器 → 输入 → 附加依赖项 → ws2_32.lib
```

### 2.3 调试功能

```cpp
// 调试示例代码
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> data = {5, 2, 8, 1, 9, 3};

    // 断点：点击行号左侧的灰色区域，或按F9
    // 条件断点：右键断点 → 条件 → data[i] > 5
    // 数据断点：调试 → 新建数据断点 → 监视内存地址

    for (int i = 0; i < data.size(); i++) {
        if (data[i] > 5) {
            cout << "找到大数: " << data[i] << endl;
        }
    }

    // 即时窗口：调试时可以输入表达式求值
    // 监视窗口：添加变量持续监视
    // 自动窗口：自动显示当前行涉及的变量
    // 调用堆栈：查看函数调用链

    return 0;
}
```

### 2.4 性能分析器

```
调试 → 性能探查器 (Alt+F2)
```

可用的分析工具：
- **CPU使用率**：分析各函数的CPU占用
- **内存使用量**：检测内存泄漏
- **GPU使用率**：图形程序性能分析
- **检测**：详细的函数调用计时

### 2.6 IntelliCode智能提示

```
工具 → 获取工具和功能 → 单个组件 → IntelliCode
```

IntelliCode功能：
- 基于AI的代码补全排序（最可能的选项排在最前）
- 智能参数提示
- 自动using语句导入
- 代码风格建议

## 三、注意事项与常见陷阱

1. **版本选择**：社区版（Community）免费且功能完整，个人和小团队足够使用
2. **配置和平台**：注意区分Debug/Release和x86/x64配置，不同配置有不同的编译选项
3. **预编译头**：默认创建的项目可能使用预编译头（pch.h/pch.cpp），初学者可以关闭此功能
4. **MSVC特性**：MSVC有一些非标准扩展，使用`/permissive-`选项开启严格标准符合模式
5. **项目文件不要手动编辑**：`.vcxproj`文件由VS管理，手动修改容易导致项目损坏
