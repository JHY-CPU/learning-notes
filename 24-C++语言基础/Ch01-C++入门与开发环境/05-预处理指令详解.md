# 预处理指令详解

## 一、概念说明

预处理指令是在**编译之前**由预处理器处理的指令，以`#`开头。预处理器不理解C++语法，只进行文本替换和条件包含等操作。

## 二、具体用法

### 2.1 #include 包含头文件

```cpp
// 方式一：尖括号 —— 从系统/标准库路径查找
#include <iostream>
#include <vector>
#include <string>

// 方式二：双引号 —— 先从当前目录查找，再到系统路径查找
#include "myheader.h"
#include "utils/helper.h"
```

### 2.2 #define 宏定义

```cpp
// 对象式宏：简单的文本替换
#define MAX_SIZE 1024
#define PI 3.14159265358979
#define DEBUG_MODE true

// 函数式宏：带参数的宏
#define SQUARE(x) ((x) * (x))
#define MAX(a, b) ((a) > (b) ? (a) : (b))

// 多行宏：使用反斜杠续行
#define PRINT_DEBUG(msg) \
    std::cout << "[DEBUG] " << __FILE__ << ":" \
    << __LINE__ << " " << msg << std::endl

int main() {
    int arr[MAX_SIZE];
    double area = PI * SQUARE(5.0);
    int bigger = MAX(10, 20);

    std::cout << "面积: " << area << std::endl;
    std::cout << "较大值: " << bigger << std::endl;
    PRINT_DEBUG("程序运行中");
    return 0;
}
```

输出：
```
面积: 78.5398
较大值: 20
[DEBUG] main.cpp:17 程序运行中
```

### 2.3 预定义宏

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "文件: " << __FILE__ << endl;
    cout << "行号: " << __LINE__ << endl;
    cout << "日期: " << __DATE__ << endl;
    cout << "时间: " << __TIME__ << endl;
    cout << "函数: " << __func__ << endl;
    cout << "C++标准: " << __cplusplus << endl;
    return 0;
}
```

输出（示例）：
```
文件: main.cpp
行号: 8
日期: May  6 2026
时间: 14:30:00
函数: main
C++标准: 201703
```

### 2.4 条件编译

```cpp
#include <iostream>

// 根据平台条件编译
#ifdef _WIN32
    #include <windows.h>
    #define PLATFORM "Windows"
#elif defined(__linux__)
    #include <unistd.h>
    #define PLATFORM "Linux"
#elif defined(__APPLE__)
    #include <TargetConditionals.h>
    #define PLATFORM "macOS"
#else
    #define PLATFORM "Unknown"
#endif

// 防止头文件重复包含
#ifndef MY_HEADER_H
#define MY_HEADER_H
// 头文件内容
#endif

// #if 表达式
#if __cplusplus >= 202002L
    #define CPP_VERSION "C++20或更高"
#elif __cplusplus >= 201703L
    #define CPP_VERSION "C++17"
#else
    #define CPP_VERSION "C++14或更低"
#endif

int main() {
    std::cout << "平台: " << PLATFORM << std::endl;
    std::cout << "标准: " << CPP_VERSION << std::endl;
    return 0;
}
```

### 2.5 #pragma 指令

```cpp
// 防止头文件重复包含（非标准但广泛支持）
#pragma once

// 禁用特定警告（MSVC）
#pragma warning(disable: 4996)

// 设置结构体对齐
#pragma pack(push, 1)
struct PackedStruct {
    char a;
    int b;
};
#pragma pack(pop)

// GCC/Clang中使用
#pragma GCC diagnostic push
#pragma GCC diagnostic ignored "-Wunused-variable"
    int unused_var = 42;
#pragma GCC diagnostic pop
```

### 2.6 #error 和 #warning

```cpp
// 编译时产生错误
#ifndef __cplusplus
    #error "此文件需要C++编译器"
#endif

// 编译时产生警告（GCC/Clang）
#if __cplusplus < 201703L
    #warning "建议使用C++17或更高版本"
#endif
```

## 三、注意事项与常见陷阱

1. **宏不是函数**：`#define SQUARE(x) x * x`在`SQUARE(1+2)`时会展开为`1+2*1+2=5`而非9，必须加括号
2. **宏没有类型检查**：宏是纯文本替换，不会检查类型，容易出错，建议优先使用`constexpr`和内联函数
3. **#define vs const**：常量应使用`const`或`constexpr`而非`#define`
4. **#pragma once非标准**：虽然被绝大多数编译器支持，但不是C++标准的一部分
5. **预处理器不理解C++**：它只做文本处理，不会检查语法正确性
