# 开发环境搭建

## 一、概念说明

C++开发环境由**编译器**、**编辑器/IDE**和**构建工具**三部分组成。选择合适的开发环境能显著提高开发效率。

## 二、具体用法

### 2.1 VS Code配置（推荐初学者）

VS Code是一款轻量级编辑器，通过扩展可以成为强大的C++开发环境。

**安装步骤：**
1. 下载并安装 [VS Code](https://code.visualstudio.com/)
2. 安装扩展：`C/C++`（Microsoft）、`C/C++ Extension Pack`
3. 安装编译器：Windows下安装MinGW-w64或使用WSL

**配置文件 `.vscode/tasks.json`：**

```json
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "build",
            "type": "shell",
            "command": "g++",
            "args": [
                "-g",
                "-std=c++17",
                "-Wall",
                "${file}",
                "-o",
                "${fileDirname}/${fileBasenameNoExtension}"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            }
        }
    ]
}
```

**配置文件 `.vscode/launch.json`：**

```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Debug",
            "type": "cppdbg",
            "request": "launch",
            "program": "${fileDirname}/${fileBasenameNoExtension}",
            "args": [],
            "stopAtEntry": false,
            "cwd": "${workspaceFolder}",
            "environment": [],
            "externalConsole": false,
            "MIMode": "gdb"
        }
    ]
}
```

### 2.2 Visual Studio（Windows推荐）

Visual Studio是微软提供的功能完整的IDE，适合大型项目开发。

```cpp
// Visual Studio中创建项目后自动生成的代码
// 文件 -> 新建 -> 项目 -> 控制台应用

#include <iostream>

int main() {
    std::cout << "Hello Visual Studio!" << std::endl;
    return 0;
}
```

输出：
```
Hello Visual Studio!
```

**Visual Studio的优势：**
- 集成MSVC编译器，Windows平台性能优秀
- 内置强大的调试器和性能分析器
- IntelliSense代码补全非常智能
- 支持CMake项目直接打开

### 2.3 CLion配置（JetBrains出品）

CLion是专业的C/C++ IDE，对CMake有原生支持。

```cmake
# CLion项目默认的CMakeLists.txt
cmake_minimum_required(VERSION 3.20)
project(MyProject)

set(CMAKE_CXX_STANDARD 17)

add_executable(MyProject main.cpp)
```

### 2.4 编译器安装验证

```bash
# 检查g++版本
g++ --version

# 检查clang++版本
clang++ --version

# 编译并运行一个测试程序
g++ -std=c++17 -o hello hello.cpp && ./hello
```

输出（示例）：
```
g++ (GCC) 13.2.0
Copyright (C) 2023 Free Software Foundation, Inc.
```

## 三、注意事项与常见陷阱

1. **编译器路径**：确保`g++`或`clang++`在系统PATH环境变量中，否则VS Code无法找到编译器
2. **标准版本**：始终在编译选项中指定`-std=c++17`或更高版本，避免使用编译器默认值
3. **Windows用户**：推荐使用MinGW-w64而非原生MSYS2，或者直接使用Visual Studio
4. **编码问题**：Windows终端默认GBK编码，源文件建议保存为UTF-8并添加`-finput-charset=UTF-8`编译选项
5. **不要同时安装多个编译器版本**：初学者容易造成路径冲突，选择一个即可
