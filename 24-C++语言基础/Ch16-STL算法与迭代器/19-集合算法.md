# 集合算法

## 一、概念说明

集合算法（C++标准 §25.4.5）对已排序的范围执行集合运算：并集、交集、差集、对称差集。输入必须是**已排序**的，结果也自动有序。这些算法利用有序性实现O(n)复杂度。

### 1.1 四种集合运算

| 算法 | 结果 |
|------|------|
| `set_union` | A ∪ B |
| `set_intersection` | A ∩ B |
| `set_difference` | A - B |
| `set_symmetric_difference` | (A-B) ∪ (B-A) |

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> a = {1, 2, 3, 4, 5};
    std::vector<int> b = {3, 4, 5, 6, 7};
    std::vector<int> result;

    // 并集
    std::set_union(a.begin(), a.end(), b.begin(), b.end(),
                   std::back_inserter(result));
    // result: {1, 2, 3, 4, 5, 6, 7}

    // 交集
    result.clear();
    std::set_intersection(a.begin(), a.end(), b.begin(), b.end(),
                          std::back_inserter(result));
    // result: {3, 4, 5}

    // 差集（在a中但不在b中）
    result.clear();
    std::set_difference(a.begin(), a.end(), b.begin(), b.end(),
                        std::back_inserter(result));
    // result: {1, 2}

    // 对称差集
    result.clear();
    std::set_symmetric_difference(a.begin(), a.end(), b.begin(), b.end(),
                                   std::back_inserter(result));
    // result: {1, 2, 6, 7}

    // includes：检查包含关系
    bool is_sub = std::includes(a.begin(), a.end(),
                                 std::vector<int>{2, 3}.begin(),
                                 std::vector<int>{2, 3}.end());
    std::cout << "a包含{2,3}: " << is_sub << std::endl;  // true
}
```

## 三、注意事项

1. **输入范围必须已排序**：否则结果未定义
2. **输出用back_inserter**：大小未知
3. **结果自动有序**
4. **相等元素的处理**：`set_union`保留较多集合中的个数
5. **`includes`检查一个范围是否包含另一个**
6. **复杂度O(n+m)**：n和m是两个输入范围的大小
