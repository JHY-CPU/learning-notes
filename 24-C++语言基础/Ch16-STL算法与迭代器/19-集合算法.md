# 集合算法

## 一、概念说明

集合算法对已排序的范围执行集合运算：并集、交集、差集、对称差集。输入必须是已排序的。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> a = {1, 2, 3, 4, 5};
    std::vector<int> b = {3, 4, 5, 6, 7};
    std::vector<int> result;

    // 并集
    result.clear();
    std::set_union(a.begin(), a.end(), b.begin(), b.end(),
                   std::back_inserter(result));
    // result: {1, 2, 3, 4, 5, 6, 7}

    // 交集
    result.clear();
    std::set_intersection(a.begin(), a.end(), b.begin(), b.end(),
                          std::back_inserter(result));
    // result: {3, 4, 5}

    // 差集（在a中但不在b中）
    result.clear();
    std::set_difference(a.begin(), a.end(), b.begin(), b.end(),
                        std::back_inserter(result));
    // result: {1, 2}

    // 对称差集（仅在其中一个集合中）
    result.clear();
    std::set_symmetric_difference(a.begin(), a.end(), b.begin(), b.end(),
                                   std::back_inserter(result));
    // result: {1, 2, 6, 7}

    // includes：检查包含关系
    bool is_sub = std::includes(a.begin(), a.end(),
                                 std::vector<int>{2, 3}.begin(),
                                 std::vector<int>{2, 3}.end());
    std::cout << "a包含{2,3}: " << is_sub << std::endl;  // true
}
```

## 三、注意事项

- 输入范围必须已排序
- 输出也需要back_inserter（大小未知）
- 结果自动有序
- 相等元素的处理见各函数文档
