# 移动迭代器

## 一、概念说明

`std::make_move_iterator`将普通迭代器转换为移动迭代器，使算法使用移动语义而非拷贝。适用于将元素从一个容器转移到另一个容器。

## 二、具体用法

```cpp
#include <iterator>
#include <vector>
#include <string>
#include <algorithm>
#include <iostream>

int main() {
    std::vector<std::string> src = {"hello", "world", "foo", "bar"};
    std::vector<std::string> dst;

    // 移动而非拷贝
    dst.reserve(src.size());
    std::copy(std::make_move_iterator(src.begin()),
              std::make_move_iterator(src.end()),
              std::back_inserter(dst));

    // src中的字符串已被移动（可能为空）
    std::cout << "src[0]: \"" << src[0] << "\"" << std::endl;  // 空

    // dst包含移动后的字符串
    for (const auto& s : dst) std::cout << s << " ";
    // hello world foo bar
}
```

### 2.1 移动构造容器

```cpp
void move_container() {
    std::vector<std::unique_ptr<int>> src;
    src.push_back(std::make_unique<int>(1));
    src.push_back(std::make_unique<int>(2));

    // unique_ptr不能拷贝，只能移动
    std::vector<std::unique_ptr<int>> dst(
        std::make_move_iterator(src.begin()),
        std::make_move_iterator(src.end())
    );

    // src中的unique_ptr已移出
    std::cout << "dst[0]: " << *dst[0] << std::endl;  // 1
}
```

## 三、注意事项

- 移动后源对象处于有效但未指定状态
- 只对支持移动的类型有意义
- 原始迭代器转换为移动迭代器后不能混用
- 配合`std::move`和`back_inserter`使用
