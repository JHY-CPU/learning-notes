# 查找算法

## 一、概念说明

STL提供了多种查找算法：`find`（线性查找）、`binary_search`（二分查找）、`lower_bound`（下界）、`upper_bound`（上界）等。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5, 3, 2};

    // find：线性查找
    auto it = std::find(v.begin(), v.end(), 3);
    if (it != v.end()) {
        std::cout << "找到: " << std::distance(v.begin(), it) << std::endl;  // 2
    }

    // find_if：条件查找
    auto it2 = std::find_if(v.begin(), v.end(), [](int x) { return x > 3; });
    if (it2 != v.end()) std::cout << "第一个>3: " << *it2 << std::endl;  // 4

    // count / count_if
    std::cout << "3的数量: " << std::count(v.begin(), v.end(), 3) << std::endl;
    std::cout << "偶数数量: " << std::count_if(v.begin(), v.end(),
        [](int x) { return x % 2 == 0; }) << std::endl;

    // binary_search：O(log n)，需要有序
    std::sort(v.begin(), v.end());
    bool found = std::binary_search(v.begin(), v.end(), 3);
    std::cout << "binary_search: " << found << std::endl;

    // lower_bound：>= value的第一个位置
    auto lo = std::lower_bound(v.begin(), v.end(), 3);
    std::cout << "lower_bound(3): " << *lo << std::endl;

    // upper_bound：> value的第一个位置
    auto hi = std::upper_bound(v.begin(), v.end(), 3);
    std::cout << "upper_bound(3): " << *hi << std::endl;

    // equal_range：[lower_bound, upper_bound)
    auto [begin, end] = std::equal_range(v.begin(), v.end(), 3);
    std::cout << "3的数量: " << std::distance(begin, end) << std::endl;
}
```

## 三、注意事项

- `find`是O(n)，`binary_search`是O(log n)但需要有序
- `lower_bound`和`upper_bound`只对有序序列有意义
- 未找到时返回`end()`迭代器
- `binary_search`只返回bool，不返回位置
