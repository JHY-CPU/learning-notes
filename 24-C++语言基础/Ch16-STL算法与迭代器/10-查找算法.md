# 查找算法

## 一、概念说明

STL提供了多种查找算法（C++标准 §25.3-§25.4.9）：`find`（线性查找，O(n)）、`binary_search`（二分查找，O(log n)）、`lower_bound`（下界）、`upper_bound`（上界）等。选择取决于容器是否有序以及需要什么信息（是否存在、位置、范围）。

### 1.1 查找算法对比

| 算法 | 复杂度 | 要求 | 返回值 |
|------|--------|------|--------|
| `find` | O(n) | 无 | 迭代器 |
| `find_if` | O(n) | 无 | 迭代器 |
| `binary_search` | O(log n) | 有序 | bool |
| `lower_bound` | O(log n) | 有序 | 迭代器 |
| `upper_bound` | O(log n) | 有序 | 迭代器 |
| `equal_range` | O(log n) | 有序 | 迭代器对 |

## 二、具体用法

### 2.1 线性查找

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5, 3, 2};

    // find：线性查找第一个匹配
    auto it = std::find(v.begin(), v.end(), 3);
    if (it != v.end())
        std::cout << "找到: 位置" << std::distance(v.begin(), it) << std::endl;  // 2

    // find_if：条件查找
    auto it2 = std::find_if(v.begin(), v.end(), [](int x) { return x > 3; });
    if (it2 != v.end())
        std::cout << "第一个>3: " << *it2 << std::endl;  // 4

    // find_if_not：反向条件
    auto it3 = std::find_if_not(v.begin(), v.end(), [](int x) { return x < 4; });

    // count / count_if
    std::cout << "3的数量: " << std::count(v.begin(), v.end(), 3) << std::endl;
    std::cout << "偶数数量: " << std::count_if(v.begin(), v.end(),
        [](int x) { return x % 2 == 0; }) << std::endl;

    // all_of / any_of / none_of
    bool all_pos = std::all_of(v.begin(), v.end(), [](int x) { return x > 0; });
    bool any_neg = std::any_of(v.begin(), v.end(), [](int x) { return x < 0; });
}
```

### 2.2 二分查找

```cpp
void binary_search_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5, 5, 5, 6, 7};

    // binary_search：只返回是否存在
    bool found = std::binary_search(v.begin(), v.end(), 5);
    std::cout << "找到5: " << found << std::endl;  // true

    // lower_bound：>= value的第一个位置
    auto lo = std::lower_bound(v.begin(), v.end(), 5);
    std::cout << "lower_bound(5): 位置" << std::distance(v.begin(), lo)
              << " 值=" << *lo << std::endl;  // 位置4 值=5

    // upper_bound：> value的第一个位置
    auto hi = std::upper_bound(v.begin(), v.end(), 5);
    std::cout << "upper_bound(5): 位置" << std::distance(v.begin(), hi)
              << " 值=" << *hi << std::endl;  // 位置7 值=6

    // equal_range：[lower_bound, upper_bound)
    auto [begin, end] = std::equal_range(v.begin(), v.end(), 5);
    std::cout << "5的数量: " << std::distance(begin, end) << std::endl;  // 3
}
```

### 2.3 范围查找

```cpp
void range_search() {
    std::vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8, 9};

    // 查找范围 [3, 7]
    auto lo = std::lower_bound(v.begin(), v.end(), 3);  // 第一个>=3
    auto hi = std::upper_bound(v.begin(), v.end(), 7);  // 第一个>7

    std::cout << "范围[3,7]: ";
    for (auto it = lo; it != hi; ++it)
        std::cout << *it << " ";  // 3 4 5 6 7

    // 自定义比较器的查找
    struct Person { std::string name; int age; };
    std::vector<Person> people = {{"A", 20}, {"B", 25}, {"C", 30}};
    std::sort(people.begin(), people.end(),
              [](const Person& a, const Person& b) { return a.age < b.age; });

    auto it = std::lower_bound(people.begin(), people.end(), 25,
                               [](const Person& p, int age) { return p.age < age; });
}
```

## 三、注意事项

1. **`find`是O(n)，`binary_search`是O(log n)但需要有序**
2. **`lower_bound`和`upper_bound`只对有序序列有意义**
3. **未找到时返回`end()`迭代器**
4. **`binary_search`只返回bool**：需要位置用`lower_bound`
5. **`equal_range`返回一对迭代器**：[lower_bound, upper_bound)
6. **自定义比较器需与排序比较器一致**
