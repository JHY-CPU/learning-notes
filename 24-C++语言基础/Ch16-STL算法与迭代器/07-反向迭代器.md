# 反向迭代器

## 一、概念说明

`std::reverse_iterator`（C++标准 §24.5.1.1）将正向迭代器转换为反向迭代器，使算法可以逆序遍历容器。`rbegin()`和`rend()`提供便捷的反向迭代器访问。反向迭代器的`++`操作实际是向反方向移动。

### 1.1 与正向迭代器的关系

```
正向: begin() → end()
      [1] [2] [3] [4] [5]
反向: rbegin() ← rend()

rbegin() 指向最后一个元素（5）
rend()   指向第一个元素之前的位置
反向++   = 正向--
```

## 二、具体用法

### 2.1 基本用法

```cpp
#include <vector>
#include <algorithm>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // rbegin/rend：反向遍历
    for (auto it = v.rbegin(); it != v.rend(); ++it)
        std::cout << *it << " ";  // 5 4 3 2 1
    std::cout << std::endl;

    // 反向排序（降序）
    std::sort(v.rbegin(), v.rend());
    for (int x : v) std::cout << x << " ";  // 5 4 3 2 1

    // crbegin/crend：const反向迭代器
    for (auto it = v.crbegin(); it != v.crend(); ++it)
        std::cout << *it << " ";
}
```

### 2.2 反向查找

```cpp
#include <algorithm>

void reverse_find() {
    std::vector<int> v = {1, 2, 3, 2, 1};

    // 从后往前找2
    auto rit = std::find(v.rbegin(), v.rend(), 2);
    if (rit != v.rend()) {
        // base()：转换回正向迭代器
        // 注意：base()返回的迭代器偏移一位！
        auto forward_it = rit.base();
        std::cout << "位置: " << std::distance(v.begin(), forward_it) - 1
                  << std::endl;  // 3
    }

    // 最后一个偶数
    auto last_even = std::find_if(v.rbegin(), v.rend(),
                                   [](int x) { return x % 2 == 0; });
    if (last_even != v.rend())
        std::cout << "最后一个偶数: " << *last_even << std::endl;  // 2
}
```

### 2.3 base()的偏移问题

```cpp
void base_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // rbegin() 对应 end()-1
    // rend()   对应 begin()-1（不存在）

    auto rit = v.rbegin();  // 指向5
    auto fit = rit.base();  // 指向end()（5之后）

    // rit.base() 比 rit 指向的位置靠后一位
    // 所以用base()做插入时：
    v.insert(rit.base(), 99);  // 在5之前插入
    // v: {1, 2, 3, 4, 99, 5}

    // 删除rit指向的元素：
    v.erase(std::prev(rit.base()));  // 或 v.erase(rit.base() - 1)
}
```

## 三、注意事项

1. **`rbegin()`指向最后一个元素，`rend()`指向第一个元素之前**
2. **`base()`将反向迭代器转换为正向迭代器**：偏移一位
3. **反向迭代器的`++`是向反方向移动**
4. **只有双向或随机访问迭代器支持反向**：forward_list不支持
5. **反向排序用`sort(rbegin(), rend())`**：比`greater<>`更简洁
6. **`base()`的偏移在做插入/删除时需注意**
