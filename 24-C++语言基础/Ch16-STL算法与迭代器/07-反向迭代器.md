# 反向迭代器

## 一、概念说明

`std::reverse_iterator`将正向迭代器转换为反向迭代器，使算法可以逆序遍历容器。`rbegin()`和`rend()`提供便捷的反向迭代器访问。

## 二、具体用法

```cpp
#include <vector>
#include <algorithm>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // rbegin/rend：反向遍历
    for (auto it = v.rbegin(); it != v.rend(); ++it) {
        std::cout << *it << " ";  // 5 4 3 2 1
    }
    std::cout << std::endl;

    // 反向排序
    std::sort(v.rbegin(), v.rend());  // 降序
    for (int x : v) std::cout << x << " ";  // 5 4 3 2 1

    // crbegin/crend：const反向迭代器
    for (auto it = v.crbegin(); it != v.crend(); ++it) {
        std::cout << *it << " ";
    }
}
```

### 2.1 反向查找

```cpp
void reverse_find() {
    std::vector<int> v = {1, 2, 3, 2, 1};

    // 从后往前找2
    auto it = std::find(v.rbegin(), v.rend(), 2);
    if (it != v.rend()) {
        // base()：转换回正向迭代器
        auto forward_it = it.base();
        std::cout << "位置: " << std::distance(v.begin(), forward_it) - 1 << std::endl;  // 3
    }
}
```

## 三、注意事项

- `rbegin()`指向最后一个元素，`rend()`指向第一个元素之前
- `base()`将反向迭代器转换为正向迭代器（偏移一位）
- 反向迭代器的`++`是向反方向移动
- 只有双向或随机访问迭代器支持反向
