# 归约算法

## 一、概念说明

归约算法将范围内的元素合并为单个值。`std::accumulate`是经典归约，C++17引入`std::reduce`和`std::transform_reduce`支持并行执行。

## 二、具体用法

```cpp
#include <numeric>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // accumulate：累加
    int sum = std::accumulate(v.begin(), v.end(), 0);
    std::cout << "sum: " << sum << std::endl;  // 15

    // 自定义操作
    int product = std::accumulate(v.begin(), v.end(), 1,
                                  [](int a, int b) { return a * b; });
    std::cout << "product: " << product << std::endl;  // 120

    // reduce（C++17）：类似accumulate但可并行
    int sum2 = std::reduce(v.begin(), v.end(), 0);

    // transform_reduce（C++17）：先变换再归约
    int sq_sum = std::transform_reduce(v.begin(), v.end(), 0,
                                        std::plus<>(),
                                        [](int x) { return x * x; });
    std::cout << "平方和: " << sq_sum << std::endl;  // 55
}
```

### 2.1 字符串拼接

```cpp
#include <string>

void string_reduce() {
    std::vector<std::string> words = {"Hello", " ", "World"};
    std::string result = std::accumulate(
        words.begin(), words.end(), std::string(""));
    std::cout << result << std::endl;  // Hello World
}
```

## 三、注意事项

- `accumulate`在`<numeric>`中，不在`<algorithm>`
- `reduce`允许并行执行，不保证求值顺序
- `transform_reduce`适合先map再reduce的场景
- 默认操作是`plus<>`，可自定义为任何二元操作
