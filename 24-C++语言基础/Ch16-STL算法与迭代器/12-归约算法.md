# 归约算法

## 一、概念说明

归约算法（C++标准 §26.7-§26.8）将范围内的元素合并为单个值。`std::accumulate`是经典归约（C++98），C++17引入`std::reduce`和`std::transform_reduce`支持并行执行。归约是函数式编程中`fold`或`reduce`操作的C++实现。

### 1.1 accumulate vs reduce

| 特性 | accumulate | reduce |
|------|-----------|--------|
| 头文件 | `<numeric>` | `<numeric>` |
| 求值顺序 | 左到右（确定） | 不确定（可并行） |
| 并行支持 | 否 | 是（C++17） |
| 引入版本 | C++98 | C++17 |

## 二、具体用法

### 2.1 accumulate

```cpp
#include <numeric>
#include <vector>
#include <iostream>
#include <string>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 累加
    int sum = std::accumulate(v.begin(), v.end(), 0);
    std::cout << "sum: " << sum << std::endl;  // 15

    // 累乘
    int product = std::accumulate(v.begin(), v.end(), 1,
                                  [](int a, int b) { return a * b; });
    std::cout << "product: " << product << std::endl;  // 120

    // 字符串拼接
    std::vector<std::string> words = {"Hello", " ", "World"};
    std::string result = std::accumulate(words.begin(), words.end(), std::string(""));
    std::cout << result << std::endl;  // Hello World

    // 字符串拼接（带分隔符）
    std::string with_sep = std::accumulate(
        std::next(words.begin()), words.end(), words[0],
        [](std::string a, const std::string& b) { return a + ", " + b; });
}
```

### 2.2 reduce（C++17）

```cpp
void reduce_demo() {
    std::vector<int> v(1000000, 1);

    // reduce：类似accumulate但可并行
    long long sum1 = std::reduce(v.begin(), v.end(), 0LL);

    // 并行reduce
    long long sum2 = std::reduce(std::execution::par, v.begin(), v.end(), 0LL);

    // 自定义操作
    int product = std::reduce(v.begin(), v.end(), 1, std::multiplies<>());
}
```

### 2.3 transform_reduce（C++17）

```cpp
void transform_reduce_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 先平方再求和
    int sq_sum = std::transform_reduce(v.begin(), v.end(), 0,
                                        std::plus<>(),
                                        [](int x) { return x * x; });
    std::cout << "平方和: " << sq_sum << std::endl;  // 55

    // 向量点积
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    int dot = std::transform_reduce(a.begin(), a.end(), b.begin(), 0);
    std::cout << "点积: " << dot << std::endl;  // 32

    // 并行transform_reduce
    int parallel_sq_sum = std::transform_reduce(
        std::execution::par, v.begin(), v.end(), 0,
        std::plus<>(), [](int x) { return x * x; });
}
```

## 三、注意事项

1. **`accumulate`在`<numeric>`中，不在`<algorithm>`**
2. **`reduce`允许并行执行，不保证求值顺序**
3. **`transform_reduce`适合先map再reduce的场景**
4. **默认操作是`plus<>`，可自定义为任何二元操作**
5. **初始值类型决定返回类型**：`accumulate(v.begin(), v.end(), 0.0)`返回`double`
6. **字符串拼接用accumulate**：reduce不保证顺序
