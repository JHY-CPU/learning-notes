# 复制算法

## 一、概念说明

`std::copy`（C++标准 §25.3.1）是最常用的复制算法，将元素从源范围复制到目标范围。变体包括`copy_if`（条件复制）、`copy_backward`（逆向复制）、`copy_n`（复制n个）。对trivial类型，编译器可能优化为`memcpy`。

### 1.1 复制算法族

| 算法 | 功能 |
|------|------|
| `copy` | 全部复制 |
| `copy_if` | 条件复制 |
| `copy_n` | 复制n个 |
| `copy_backward` | 逆向复制（处理重叠） |

## 二、具体用法

### 2.1 copy

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst(5);

    // 基本复制
    std::copy(src.begin(), src.end(), dst.begin());
    // dst: {1, 2, 3, 4, 5}

    // 部分复制
    std::vector<int> part(3);
    std::copy(src.begin() + 1, src.begin() + 4, part.begin());
    // part: {2, 3, 4}
}
```

### 2.2 copy_if

```cpp
void copy_if_demo() {
    std::vector<int> src = {1, 2, 3, 4, 5, 6, 7, 8};

    // 条件复制
    std::vector<int> evens;
    std::copy_if(src.begin(), src.end(), std::back_inserter(evens),
                 [](int x) { return x % 2 == 0; });
    // evens: {2, 4, 6, 8}

    // 复制大于5的
    std::vector<int> large;
    std::copy_if(src.begin(), src.end(), std::back_inserter(large),
                 [](int x) { return x > 5; });
    // large: {6, 7, 8}
}
```

### 2.3 copy_n

```cpp
void copy_n_demo() {
    std::vector<int> src = {1, 2, 3, 4, 5};

    // 复制前3个
    std::vector<int> first3(3);
    std::copy_n(src.begin(), 3, first3.begin());
    // first3: {1, 2, 3}
}
```

### 2.4 copy_backward

```cpp
void copy_backward_demo() {
    // 处理重叠范围
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 将前3个元素移到后3个位置
    // 如果用copy，目标范围可能被覆盖
    std::copy_backward(v.begin(), v.begin() + 3, v.end());
    // v: {1, 2, 1, 2, 3}

    // 对比：用copy会导致部分数据丢失
    std::vector<int> v2 = {1, 2, 3, 4, 5};
    std::copy(v2.begin(), v2.begin() + 3, v2.begin() + 2);
    // v2: {1, 2, 1, 2, 3}（可能正确，但依赖实现）
}
```

## 三、注意事项

1. **目标范围必须有足够的空间**：或使用`back_inserter`
2. **源和目标重叠时用`copy_backward`**
3. **`copy_if`需要back_inserter**：大小未知
4. **copy是O(n)**：对trivial类型可能优化为`memcpy`
5. **不要用copy覆盖自身重叠区域**：结果未定义
6. **C++20 ranges版本更简洁**：`std::ranges::copy(src, dst.begin())`
