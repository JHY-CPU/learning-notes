# 复制算法

## 一、概念说明

`std::copy`是最常用的复制算法，将元素从源范围复制到目标范围。变体包括`copy_if`（条件复制）、`copy_backward`（逆向复制）。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst(5);

    // copy
    std::copy(src.begin(), src.end(), dst.begin());

    // copy_if：条件复制
    std::vector<int> evens;
    std::copy_if(src.begin(), src.end(), std::back_inserter(evens),
                 [](int x) { return x % 2 == 0; });
    // evens: {2, 4}

    // copy_n：复制n个元素
    std::vector<int> first3(3);
    std::copy_n(src.begin(), 3, first3.begin());

    // copy_backward：从后往前复制（处理重叠）
    std::vector<int> v = {1, 2, 3, 4, 5};
    std::copy_backward(v.begin(), v.begin() + 3, v.end());
    // v: {1, 2, 1, 2, 3}

    for (int x : dst) std::cout << x << " ";
}
```

## 三、注意事项

- 目标范围必须有足够的空间
- 源和目标重叠时用`copy_backward`
- `copy_if`需要back_inserter（大小未知）
- copy是O(n)，对trivial类型可能优化为memcpy
