# 比较算法

## 一、概念说明

STL提供范围比较算法：`equal`（相等比较）、`lexicographical_compare`（字典序比较）、`mismatch`（找到第一个不匹配位置）。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> a = {1, 2, 3, 4, 5};
    std::vector<int> b = {1, 2, 3, 4, 5};
    std::vector<int> c = {1, 2, 3, 4, 6};

    // equal：检查是否相等
    std::cout << "a == b: " << std::equal(a.begin(), a.end(), b.begin()) << std::endl;
    std::cout << "a == c: " << std::equal(a.begin(), a.end(), c.begin()) << std::endl;

    // lexicographical_compare：字典序比较
    std::vector<int> x = {1, 2, 3};
    std::vector<int> y = {1, 2, 4};
    std::cout << "x < y: "
              << std::lexicographical_compare(x.begin(), x.end(), y.begin(), y.end())
              << std::endl;  // true

    // mismatch：找到第一个不匹配的位置
    auto [it1, it2] = std::mismatch(a.begin(), a.end(), c.begin());
    if (it1 != a.end()) {
        std::cout << "不匹配: " << *it1 << " vs " << *it2 << std::endl;  // 4 vs 6
    }

    // 字符串字典序
    std::string s1 = "apple";
    std::string s2 = "banana";
    std::cout << "apple < banana: "
              << std::lexicographical_compare(s1.begin(), s1.end(),
                                               s2.begin(), s2.end())
              << std::endl;  // true
}
```

## 三、注意事项

- `equal`需要第二个范围至少和第一个一样长
- `mismatch`返回一对迭代器
- `lexicographical_compare`是字符串比较的通用版本
- 可以自定义比较器
