# 流迭代器

## 一、概念说明

流迭代器将流（文件、控制台）与STL算法连接，允许用算法读写流数据。`istream_iterator`用于读取，`ostream_iterator`用于写入。

## 二、具体用法

```cpp
#include <iterator>
#include <vector>
#include <algorithm>
#include <iostream>
#include <sstream>

int main() {
    // ostream_iterator：输出到流
    std::vector<int> v = {1, 2, 3, 4, 5};
    std::copy(v.begin(), v.end(), std::ostream_iterator<int>(std::cout, " "));
    // 1 2 3 4 5

    // istream_iterator：从流读取
    std::istringstream iss("10 20 30 40 50");
    std::vector<int> nums;
    std::copy(std::istream_iterator<int>(iss),
              std::istream_iterator<int>(),
              std::back_inserter(nums));
    // nums: {10, 20, 30, 40, 50}

    // 文件流示例
    // std::ifstream file("data.txt");
    // std::copy(std::istream_iterator<std::string>(file),
    //           std::istream_iterator<std::string>(),
    //           std::back_inserter(words));
}
```

### 2.1 字符串解析

```cpp
#include <string>

void parse_string() {
    std::string data = "hello world foo bar";
    std::istringstream iss(data);
    std::vector<std::string> words;

    std::copy(std::istream_iterator<std::string>(iss),
              std::istream_iterator<std::string>(),
              std::back_inserter(words));
    // words: {"hello", "world", "foo", "bar"}
}
```

## 三、注意事项

- 默认构造的`istream_iterator`表示流结束
- `ostream_iterator`的第二个参数是分隔符
- 流迭代器是单遍的，不能多次读取
- 适用于简单的流输入输出场景
