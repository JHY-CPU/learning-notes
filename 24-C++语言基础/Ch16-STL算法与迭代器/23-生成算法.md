# 生成算法

## 一、概念说明

生成算法填充范围：`fill`（用固定值填充）、`generate`（用函数生成值）、`iota`（递增填充）。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>
#include <random>

int main() {
    // fill：用固定值填充
    std::vector<int> v(5);
    std::fill(v.begin(), v.end(), 42);
    // v: {42, 42, 42, 42, 42}

    // fill_n：填充n个
    std::fill_n(v.begin(), 3, 10);
    // v: {10, 10, 10, 42, 42}

    // generate：用函数生成
    int counter = 0;
    std::generate(v.begin(), v.end(), [&counter]() { return counter++; });
    // v: {0, 1, 2, 3, 4}

    // generate_n：生成n个
    std::vector<int> v2;
    std::generate_n(std::back_inserter(v2), 5, [n = 0]() mutable { return n++; });
    // v2: {0, 1, 2, 3, 4}

    // 随机数生成
    std::random_device rd;
    std::mt19937 gen(rd());
    std::uniform_int_distribution<> dis(1, 100);
    std::generate(v.begin(), v.end(), [&]() { return dis(gen); });

    for (int x : v) std::cout << x << " ";
}
```

## 三、注意事项

- `fill`用固定值，`generate`用函数生成
- 目标范围必须有足够的空间
- `generate`的函数对象每次调用返回新值
- 配合随机数生成器使用很常见
