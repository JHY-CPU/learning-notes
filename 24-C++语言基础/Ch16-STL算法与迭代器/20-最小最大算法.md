# 最小最大算法

## 一、概念说明

STL提供`min`、`max`、`minmax`、`min_element`、`max_element`等算法，以及C++17的`clamp`（夹值）。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    // min / max
    std::cout << std::min(3, 7) << std::endl;     // 3
    std::cout << std::max(3, 7) << std::endl;     // 7

    // initializer_list版本
    std::cout << std::min({5, 3, 1, 4, 2}) << std::endl;  // 1
    std::cout << std::max({5, 3, 1, 4, 2}) << std::endl;  // 5

    // minmax：返回pair
    auto [lo, hi] = std::minmax({5, 3, 1, 4, 2});
    std::cout << lo << " " << hi << std::endl;  // 1 5

    // 范围版本
    std::vector<int> v = {5, 3, 1, 4, 2};
    auto min_it = std::min_element(v.begin(), v.end());
    auto max_it = std::max_element(v.begin(), v.end());
    std::cout << "min: " << *min_it << std::endl;  // 1
    std::cout << "max: " << *max_it << std::endl;  // 5

    // clamp（C++17）：限制在范围内
    std::cout << std::clamp(5, 1, 10) << std::endl;   // 5
    std::cout << std::clamp(-5, 0, 100) << std::endl;  // 0
    std::cout << std::clamp(150, 0, 100) << std::endl; // 100

    // 自定义比较
    auto cmp = [](int a, int b) { return abs(a) < abs(b); };
    std::cout << std::min(-7, 3, cmp) << std::endl;  // 3
}
```

## 三、注意事项

- `min`/`max`对相等值返回第一个参数
- `min_element`/`max_element`返回迭代器
- `clamp`需要min <= max，否则未定义行为
- 自定义比较器需满足严格弱序
