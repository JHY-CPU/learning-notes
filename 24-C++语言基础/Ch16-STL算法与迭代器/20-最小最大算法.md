# 最小最大算法

## 一、概念说明

STL提供`min`、`max`、`minmax`、`min_element`、`max_element`等算法（C++标准 §25.4.7-§25.4.8），以及C++17的`clamp`（夹值）。这些算法广泛用于比较和边界检查。

### 1.1 算法族

| 算法 | 功能 | 返回值 |
|------|------|--------|
| `min`/`max` | 两个值的最小/最大 | 值 |
| `min_element`/`max_element` | 范围的最小/最大 | 迭代器 |
| `minmax` | 两个值的最小和最大 | pair |
| `minmax_element` | 范围的最小和最大 | pair<迭代器> |
| `clamp`（C++17） | 限制在范围内 | 值 |

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    // min / max
    std::cout << std::min(3, 7) << std::endl;     // 3
    std::cout << std::max(3, 7) << std::endl;     // 7

    // initializer_list版本
    std::cout << std::min({5, 3, 1, 4, 2}) << std::endl;  // 1
    std::cout << std::max({5, 3, 1, 4, 2}) << std::endl;  // 5

    // minmax：返回pair
    auto [lo, hi] = std::minmax({5, 3, 1, 4, 2});
    std::cout << lo << " " << hi << std::endl;  // 1 5

    // 范围版本
    std::vector<int> v = {5, 3, 1, 4, 2};
    auto min_it = std::min_element(v.begin(), v.end());
    auto max_it = std::max_element(v.begin(), v.end());
    std::cout << "min: " << *min_it << std::endl;  // 1
    std::cout << "max: " << *max_it << std::endl;  // 5

    // minmax_element
    auto [min_e, max_e] = std::minmax_element(v.begin(), v.end());
    std::cout << *min_e << " " << *max_e << std::endl;  // 1 5

    // clamp（C++17）：限制在范围内
    std::cout << std::clamp(5, 1, 10) << std::endl;   // 5
    std::cout << std::clamp(-5, 0, 100) << std::endl;  // 0
    std::cout << std::clamp(150, 0, 100) << std::endl; // 100

    // 自定义比较
    auto cmp = [](int a, int b) { return abs(a) < abs(b); };
    std::cout << std::min(-7, 3, cmp) << std::endl;  // 3（绝对值更小）
}
```

## 三、注意事项

1. **`min`/`max`对相等值返回第一个参数**
2. **`min_element`/`max_element`返回迭代器**：需解引用
3. **`clamp`需要min <= max**：否则未定义行为
4. **自定义比较器需满足严格弱序**
5. **`min_element`对空范围返回`end()`**
6. **`clamp`常用于边界检查**：如坐标限制、参数验证
