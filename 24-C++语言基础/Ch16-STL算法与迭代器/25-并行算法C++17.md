# 并行算法（C++17）

## 一、概念说明

C++17引入了执行策略（Execution Policy，C++标准 §25.2.2），允许标准算法并行执行。通过指定策略，算法可以利用多线程或SIMD指令加速。大数据量时并行算法可显著提升性能。

### 1.1 三种执行策略

| 策略 | 说明 |
|------|------|
| `std::execution::seq` | 顺序执行（默认） |
| `std::execution::par` | 并行执行（多线程） |
| `std::execution::par_unseq` | 并行+向量化（SIMD） |

## 二、具体用法

```cpp
#include <algorithm>
#include <execution>
#include <vector>
#include <iostream>
#include <numeric>

int main() {
    std::vector<int> v(1000000);
    std::iota(v.begin(), v.end(), 1);

    // 顺序执行（默认）
    std::sort(std::execution::seq, v.begin(), v.end());

    // 并行执行（多线程）
    std::sort(std::execution::par, v.begin(), v.end());

    // 并行+向量化
    std::sort(std::execution::par_unseq, v.begin(), v.end());

    // 并行for_each
    std::for_each(std::execution::par, v.begin(), v.end(),
                   [](int& x) { x *= 2; });

    // 并行reduce
    long long sum = std::reduce(std::execution::par, v.begin(), v.end(), 0LL);

    // 并行transform
    std::vector<int> dst(v.size());
    std::transform(std::execution::par, v.begin(), v.end(), dst.begin(),
                    [](int x) { return x * x; });

    // 并行copy_if
    std::vector<int> filtered;
    std::copy_if(std::execution::par, v.begin(), v.end(),
                 std::back_inserter(filtered),
                 [](int x) { return x % 2 == 0; });
}
```

## 四、注意事项

1. **并行算法需要线程安全的容器和函数**：函数对象不能有副作用
2. **小数据量时并行反而更慢**：线程开销
3. **函数对象不能有副作用**：或需要同步（互斥锁等）
4. **不是所有算法都支持并行策略**
5. **需要支持C++17的编译器**：可能需要链接TBB等库
6. **`par_unseq`要求函数对象可向量化**：不能有同步
