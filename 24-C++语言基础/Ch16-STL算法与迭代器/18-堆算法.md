# 堆算法

## 一、概念说明

堆是一种特殊的完全二叉树，满足堆性质（大顶堆：父节点>=子节点）。STL提供`make_heap`、`push_heap`、`pop_heap`、`sort_heap`等操作。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

    // make_heap：构建大顶堆
    std::make_heap(v.begin(), v.end());
    std::cout << "堆顶: " << v.front() << std::endl;  // 9

    // push_heap：先push_back再push_heap
    v.push_back(10);
    std::push_heap(v.begin(), v.end());
    std::cout << "堆顶: " << v.front() << std::endl;  // 10

    // pop_heap：先pop_heap再pop_back
    std::pop_heap(v.begin(), v.end());  // 堆顶移到末尾
    int top = v.back();
    v.pop_back();
    std::cout << "弹出: " << top << std::endl;  // 10

    // is_heap：检查是否是堆
    std::cout << "是堆: " << std::is_heap(v.begin(), v.end()) << std::endl;

    // sort_heap：堆排序
    std::sort_heap(v.begin(), v.end());
    for (int x : v) std::cout << x << " ";
    // 有序输出

    // 小顶堆
    std::vector<int> v2 = {3, 1, 4, 1, 5};
    std::make_heap(v2.begin(), v2.end(), std::greater<int>());
    std::cout << "\n小顶堆: " << v2.front() << std::endl;  // 1
}
```

## 三、注意事项

- STL堆默认是大顶堆
- push前先`push_back`，pop前先`pop_heap`
- 堆排序是O(n log n)但不稳定
- `priority_queue`提供了更高级的堆接口
