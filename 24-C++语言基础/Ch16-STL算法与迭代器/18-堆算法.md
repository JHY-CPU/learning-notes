# 堆算法

## 一、概念说明

堆（C++标准 §25.4.6）是一种特殊的完全二叉树，满足堆性质（大顶堆：父节点>=子节点）。STL提供`make_heap`、`push_heap`、`pop_heap`、`sort_heap`等操作。这些算法直接操作vector等随机访问容器，是`priority_queue`的底层实现。

### 1.1 堆的性质

```
大顶堆（默认）：
- 每个节点 >= 其子节点
- 根节点最大
- 底层存储：vector（完全二叉树数组表示）
- 父节点：(i-1)/2
- 左子：2*i+1，右子：2*i+2
```

## 二、具体用法

### 2.1 基本操作

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

    // make_heap：构建大顶堆
    std::make_heap(v.begin(), v.end());
    std::cout << "堆顶: " << v.front() << std::endl;  // 9

    // 检查是否是堆
    std::cout << "是堆: " << std::is_heap(v.begin(), v.end()) << std::endl;

    // push_heap：先push_back再push_heap
    v.push_back(10);
    std::push_heap(v.begin(), v.end());
    std::cout << "堆顶: " << v.front() << std::endl;  // 10

    // pop_heap：先pop_heap再pop_back
    std::pop_heap(v.begin(), v.end());  // 堆顶移到末尾
    int top = v.back();
    v.pop_back();
    std::cout << "弹出: " << top << std::endl;  // 10
}
```

### 2.2 小顶堆

```cpp
void min_heap_demo() {
    std::vector<int> v = {3, 1, 4, 1, 5};

    // 小顶堆
    std::make_heap(v.begin(), v.end(), std::greater<int>());
    std::cout << "小顶堆: " << v.front() << std::endl;  // 1

    v.push_back(0);
    std::push_heap(v.begin(), v.end(), std::greater<int>());
    std::cout << "小顶堆: " << v.front() << std::endl;  // 0
}
```

### 2.3 堆排序

```cpp
void heap_sort_demo() {
    std::vector<int> v = {5, 3, 1, 4, 2};

    // 堆排序：O(n log n)
    std::make_heap(v.begin(), v.end());
    std::sort_heap(v.begin(), v.end());
    // v: {1, 2, 3, 4, 5}
}
```

### 2.4 自定义比较器

```cpp
struct Task {
    int priority;
    std::string name;
    bool operator<(const Task& other) const {
        return priority < other.priority;
    }
};

void custom_compare() {
    std::vector<Task> tasks = {{1, "低"}, {3, "高"}, {2, "中"}};
    std::make_heap(tasks.begin(), tasks.end());

    while (!tasks.empty()) {
        std::pop_heap(tasks.begin(), tasks.end());
        std::cout << tasks.back().name << std::endl;  // 高、中、低
        tasks.pop_back();
    }
}
```

## 三、注意事项

1. **STL堆默认是大顶堆**：`std::greater<>`实现小顶堆
2. **push前先`push_back`，pop前先`pop_heap`**
3. **堆排序是O(n log n)但不稳定**
4. **`priority_queue`提供了更高级的堆接口**
5. **`is_heap`（C++11）检查是否满足堆性质**
6. **`is_heap_until`找到第一个破坏堆性质的位置**
