# 数值算法

## 一、概念说明

`<numeric>`头文件（C++标准 §26.7-§26.8）提供数值计算算法：`iota`（填充递增）、`accumulate`（累加）、`inner_product`（内积）、`adjacent_difference`（相邻差）、`partial_sum`（前缀和）。这些算法在数学计算和数据分析中广泛使用。

### 1.1 算法族

| 算法 | 功能 |
|------|------|
| `iota` | 从初始值递增填充 |
| `accumulate` | 累加（归约） |
| `inner_product` | 内积（点积） |
| `adjacent_difference` | 相邻元素差 |
| `partial_sum` | 前缀和 |

## 二、具体用法

```cpp
#include <numeric>
#include <vector>
#include <iostream>

int main() {
    // iota：填充递增值
    std::vector<int> v(5);
    std::iota(v.begin(), v.end(), 1);
    // v: {1, 2, 3, 4, 5}

    // accumulate：累加
    int sum = std::accumulate(v.begin(), v.end(), 0);
    std::cout << "sum: " << sum << std::endl;  // 15

    // inner_product：内积
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    int ip = std::inner_product(a.begin(), a.end(), b.begin(), 0);
    std::cout << "内积: " << ip << std::endl;  // 32 (1*4+2*5+3*6)

    // adjacent_difference：相邻差
    std::vector<int> src = {1, 3, 6, 10, 15};
    std::vector<int> diff(src.size());
    std::adjacent_difference(src.begin(), src.end(), diff.begin());
    // diff: {1, 2, 3, 4, 5}

    // partial_sum：前缀和
    std::vector<int> ps(src.size());
    std::partial_sum(src.begin(), src.end(), ps.begin());
    // ps: {1, 4, 10, 20, 35}
}
```

## 三、注意事项

1. **这些算法都在`<numeric>`中，不在`<algorithm>`**
2. **`iota`从初始值开始递增填充**
3. **`accumulate`的初始值决定返回类型**
4. **`inner_product`可用于点积计算**
5. **`adjacent_difference`第一个元素直接拷贝**
6. **`partial_sum`计算前缀和**

## 四、高级用法

### 4.1 自定义操作

```cpp
#include <numeric>
#include <vector>
#include <string>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // accumulate 用乘法代替加法
    int product = std::accumulate(v.begin(), v.end(), 1,
        [](int acc, int x) { return acc * x; });
    // product = 120 (5!)

    // 字符串拼接
    std::vector<std::string> words = {"Hello", " ", "World"};
    std::string sentence = std::accumulate(words.begin(), words.end(), std::string(""));
    // sentence = "Hello World"

    // inner_product 自定义操作
    std::vector<int> a = {1, 2, 3}, b = {4, 5, 6};
    // 曼哈顿距离: sum(|a_i - b_i|)
    int manhattan = std::inner_product(a.begin(), a.end(), b.begin(), 0,
        std::plus<>(), [](int x, int y) { return std::abs(x - y); });
    // manhattan = 6

    // 计算加权平均
    std::vector<double> values = {90, 85, 92};
    std::vector<double> weights = {0.3, 0.3, 0.4};
    double weightedAvg = std::inner_product(values.begin(), values.end(),
        weights.begin(), 0.0) / std::accumulate(weights.begin(), weights.end(), 0.0);
    std::cout << "加权平均: " << weightedAvg << std::endl;
}
```

### 4.2 partial_sum 的扩展用法

```cpp
#include <numeric>
#include <vector>
#include <iostream>

int main() {
    // 前缀积
    std::vector<int> v = {2, 3, 4, 5};
    std::vector<int> prefixProduct(v.size());
    std::partial_sum(v.begin(), v.end(), prefixProduct.begin(),
        [](int a, int b) { return a * b; });
    // prefixProduct: {2, 6, 24, 120}

    // 差分还原（partial_sum 的逆操作）
    std::vector<int> diff = {1, 2, 3, 4, 5};  // 相邻差
    std::vector<int> original(diff.size());
    std::partial_sum(diff.begin(), diff.end(), original.begin());
    // original: {1, 3, 6, 10, 15}  // 还原为前缀和
}
```

## 五、实际应用场景

| 场景 | 使用算法 | 说明 |
| --- | --- | --- |
| 前缀和查询 | partial_sum | O(1) 回答区间和 |
| 数组求和 | accumulate | 简洁的一行求和 |
| 向量点积 | inner_product | 机器学习中的特征计算 |
| 序列差分 | adjacent_difference | 信号处理中的微分近似 |
| 序列填充 | iota | 生成索引数组 |

## 六、注意事项补充

- **`accumulate` 的初始值类型决定了返回值类型**：用 `0LL` 得到 `long long`
- **浮点精度**：`accumulate` 对浮点数使用 `+` 可能有精度问题
- **`inner_product` 两个范围长度必须一致**：否则行为未定义
- **C++17 并行算法**：`std::reduce` 是 `accumulate` 的并行版本，不保证顺序
