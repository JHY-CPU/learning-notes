# 数值算法

## 一、概念说明

`<numeric>`头文件提供数值计算算法：`iota`（填充递增）、`accumulate`（累加）、`inner_product`（内积）、`adjacent_difference`（相邻差）。

## 二、具体用法

```cpp
#include <numeric>
#include <vector>
#include <iostream>

int main() {
    // iota：填充递增值
    std::vector<int> v(5);
    std::iota(v.begin(), v.end(), 1);
    // v: {1, 2, 3, 4, 5}

    // accumulate：累加
    int sum = std::accumulate(v.begin(), v.end(), 0);
    std::cout << "sum: " << sum << std::endl;  // 15

    // inner_product：内积
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    int ip = std::inner_product(a.begin(), a.end(), b.begin(), 0);
    std::cout << "内积: " << ip << std::endl;  // 32

    // adjacent_difference：相邻差
    std::vector<int> src = {1, 3, 6, 10, 15};
    std::vector<int> diff(src.size());
    std::adjacent_difference(src.begin(), src.end(), diff.begin());
    // diff: {1, 2, 3, 4, 5}

    // partial_sum：前缀和
    std::vector<int> ps(src.size());
    std::partial_sum(src.begin(), src.end(), ps.begin());
    // ps: {1, 4, 10, 20, 35}

    for (int x : v) std::cout << x << " ";
}
```

## 三、注意事项

- 这些算法都在`<numeric>`中，不在`<algorithm>`
- `iota`从初始值开始递增填充
- `accumulate`的初始值决定返回类型
- `inner_product`可用于点积计算
