# 变换算法

## 一、概念说明

`std::transform`（C++标准 §25.3.4）对输入范围的每个元素应用函数，结果输出到目标范围。提供一元版本（单输入）和二元版本（双输入）。这是函数式编程中`map`操作的C++实现。

### 1.1 两种版本

| 版本 | 输入 | 签名 |
|------|------|------|
| 一元 | 单个范围 | `f(x) -> y` |
| 二元 | 两个范围 | `f(x1, x2) -> y` |

## 二、具体用法

### 2.1 一元transform

```cpp
#include <algorithm>
#include <vector>
#include <iostream>
#include <string>

int main() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst;

    // 变换到新容器
    dst.resize(src.size());
    std::transform(src.begin(), src.end(), dst.begin(),
                   [](int x) { return x * 2; });
    // dst: {2, 4, 6, 8, 10}

    // 原地变换
    std::transform(src.begin(), src.end(), src.begin(),
                   [](int x) { return x + 10; });
    // src: {11, 12, 13, 14, 15}

    // 字符串变换
    std::string s = "hello world";
    std::transform(s.begin(), s.end(), s.begin(), ::toupper);
    // s: "HELLO WORLD"
}
```

### 2.2 二元transform

```cpp
void binary_transform() {
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    std::vector<int> c(3);

    // 两个输入范围
    std::transform(a.begin(), a.end(), b.begin(), c.begin(),
                   [](int x, int y) { return x + y; });
    // c: {5, 7, 9}

    // 向量点积（变换+归约）
    std::vector<int> products(3);
    std::transform(a.begin(), a.end(), b.begin(), products.begin(),
                   [](int x, int y) { return x * y; });
    int dot = std::accumulate(products.begin(), products.end(), 0);
    // 1*4 + 2*5 + 3*6 = 32
}
```

### 2.3 配合back_inserter

```cpp
void with_back_inserter() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst;

    // 大小未知时用back_inserter
    std::transform(src.begin(), src.end(), std::back_inserter(dst),
                   [](int x) { return x * x; });
    // dst: {1, 4, 9, 16, 25}

    // 配合filter（先变换再过滤）
    std::vector<int> filtered;
    std::copy_if(dst.begin(), dst.end(), std::back_inserter(filtered),
                 [](int x) { return x > 10; });
    // filtered: {16, 25}
}
```

### 2.4 C++20 ranges版本

```cpp
#include <ranges>

void ranges_transform() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 惰性变换（视图）
    auto squares = v | std::views::transform([](int x) { return x * x; });

    // 组合管道
    auto result = v
        | std::views::transform([](int x) { return x * 2; })
        | std::views::filter([](int x) { return x > 5; });

    for (int x : result) std::cout << x << " ";  // 6 8 10
}
```

## 三、注意事项

1. **目标范围必须有足够的空间**：或使用`back_inserter`
2. **函数对象应返回新值，不修改输入**
3. **一元和二元版本分别处理单输入和双输入**
4. **配合lambda使用最灵活**
5. **原地变换时输入输出可以是同一范围**
6. **C++20 ranges提供惰性变换**：不立即计算
