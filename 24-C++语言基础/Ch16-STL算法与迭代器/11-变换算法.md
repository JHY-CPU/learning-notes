# 变换算法

## 一、概念说明

`std::transform`对输入范围的每个元素应用函数，结果输出到目标范围。提供一元版本（单输入）和二元版本（双输入）。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst;

    // 一元transform
    dst.resize(src.size());
    std::transform(src.begin(), src.end(), dst.begin(),
                   [](int x) { return x * 2; });
    // dst: {2, 4, 6, 8, 10}

    // 原地变换
    std::transform(src.begin(), src.end(), src.begin(),
                   [](int x) { return x + 10; });
    // src: {11, 12, 13, 14, 15}

    // 二元transform
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    std::vector<int> c(3);
    std::transform(a.begin(), a.end(), b.begin(), c.begin(),
                   [](int x, int y) { return x + y; });
    // c: {5, 7, 9}

    for (int x : dst) std::cout << x << " ";
}
```

### 2.1 字符串变换

```cpp
#include <string>

void string_transform() {
    std::string s = "hello world";
    std::transform(s.begin(), s.end(), s.begin(), ::toupper);
    std::cout << s << std::endl;  // HELLO WORLD
}
```

## 三、注意事项

- 目标范围必须有足够的空间（或使用back_inserter）
- 函数对象应返回新值，不修改输入
- 一元和二元版本分别处理单输入和双输入
- 配合lambda使用最灵活
