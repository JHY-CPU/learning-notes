# 排序算法

## 一、概念说明

STL提供了多种排序算法（C++标准 §25.4.1-§25.4.5）：`sort`（快速排序，O(n log n)）、`stable_sort`（稳定排序，O(n log n)）、`partial_sort`（部分排序）、`nth_element`（第n小元素）、`is_sorted`（检查是否有序）等。选择合适的排序算法取决于是否需要稳定性、是否只需要部分排序等。

### 1.1 排序算法对比

| 算法 | 复杂度 | 稳定 | 说明 |
|------|--------|------|------|
| `sort` | O(n log n) | 否 | 通用排序 |
| `stable_sort` | O(n log n) | 是 | 保留相等元素顺序 |
| `partial_sort` | O(n log k) | 否 | 只排序前k个 |
| `nth_element` | O(n) | 否 | 第n小元素就位 |
| `is_sorted` | O(n) | - | 检查是否有序 |

## 二、具体用法

### 2.1 sort

```cpp
#include <algorithm>
#include <vector>
#include <iostream>
#include <string>

int main() {
    std::vector<int> v = {5, 3, 1, 4, 2};

    // 默认升序
    std::sort(v.begin(), v.end());
    // v: {1, 2, 3, 4, 5}

    // 自定义比较（降序）
    std::sort(v.begin(), v.end(), std::greater<int>());
    // v: {5, 4, 3, 2, 1}

    // lambda比较器
    std::sort(v.begin(), v.end(), [](int a, int b) {
        return a % 10 < b % 10;  // 按个位数排序
    });
}
```

### 2.2 对象排序

```cpp
struct Person {
    std::string name;
    int age;
};

void sort_objects() {
    std::vector<Person> people = {{"Alice", 30}, {"Bob", 25}, {"Charlie", 35}};

    // 按age排序
    std::sort(people.begin(), people.end(),
              [](const Person& a, const Person& b) { return a.age < b.age; });

    // 多字段排序
    std::sort(people.begin(), people.end(),
              [](const Person& a, const Person& b) {
                  return std::tie(a.age, a.name) < std::tie(b.age, b.name);
              });

    // C++20 ranges投影
    // std::ranges::sort(people, {}, &Person::age);
}
```

### 2.3 stable_sort

```cpp
void stable_demo() {
    std::vector<std::pair<int, std::string>> data = {
        {2, "B"}, {1, "A"}, {2, "C"}, {1, "D"}
    };

    // 按first排序，保留原始顺序
    std::stable_sort(data.begin(), data.end(),
                     [](const auto& a, const auto& b) {
                         return a.first < b.first;
                     });
    // {1,"A"}, {1,"D"}, {2,"B"}, {2,"C"}
    // 相等元素保持原来顺序：A在D前，B在C前
}
```

### 2.4 partial_sort与nth_element

```cpp
void partial_and_nth() {
    std::vector<int> v = {5, 3, 1, 4, 2, 9, 7, 8, 6};

    // partial_sort：只排序前3个最小的
    std::partial_sort(v.begin(), v.begin() + 3, v.end());
    // v: {1, 2, 3, ...} 前3个有序，其余无序

    // nth_element：第3小的元素就位
    v = {5, 3, 1, 4, 2, 9, 7, 8, 6};
    std::nth_element(v.begin(), v.begin() + 2, v.end());
    // v[2] == 3，前面都<=3，后面都>=3

    // 应用：找中位数
    std::vector<int> data = {5, 3, 1, 4, 2};
    auto mid = data.begin() + data.size() / 2;
    std::nth_element(data.begin(), mid, data.end());
    std::cout << "中位数: " << *mid << std::endl;  // 3
}
```

### 2.5 is_sorted

```cpp
void check_sorted() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    bool sorted = std::is_sorted(v.begin(), v.end());
    std::cout << "已排序: " << sorted << std::endl;  // true

    // 检查部分有序
    bool partial = std::is_sorted_until(v.begin(), v.end()) == v.end();
}
```

## 三、注意事项

1. **`sort`需要随机访问迭代器**：vector、deque、array可用，list不行
2. **`stable_sort`保留相等元素的相对顺序**：但额外开销
3. **比较函数必须满足严格弱序**：不满足是未定义行为
4. **`is_sorted`（C++11）快速检查是否有序**
5. **`nth_element`是O(n)平均**：比完全排序快
6. **list有自己的`sort`成员函数**：因为迭代器不是随机访问
