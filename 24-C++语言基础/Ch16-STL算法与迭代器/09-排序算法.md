# 排序算法

## 一、概念说明

STL提供了多种排序算法：`sort`（快速排序）、`stable_sort`（稳定排序）、`partial_sort`（部分排序）、`nth_element`（第n小元素）等。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {5, 3, 1, 4, 2};

    // std::sort：O(n log n)，不稳定
    std::sort(v.begin(), v.end());
    // v: {1, 2, 3, 4, 5}

    // 自定义比较
    std::sort(v.begin(), v.end(), std::greater<int>());
    // v: {5, 4, 3, 2, 1}

    // std::stable_sort：O(n log n)，稳定排序
    std::stable_sort(v.begin(), v.end());

    // is_sorted：检查是否已排序
    std::cout << std::boolalpha << std::is_sorted(v.begin(), v.end()) << std::endl;

    // partial_sort：只排序前n个
    std::vector<int> v2 = {5, 3, 1, 4, 2};
    std::partial_sort(v2.begin(), v2.begin() + 3, v2.end());
    // v2: {1, 2, 3, ...} 前3个有序

    // nth_element：第n小的元素就位
    std::nth_element(v2.begin(), v2.begin() + 2, v2.end());
    std::cout << "第3小: " << v2[2] << std::endl;
}
```

### 2.1 排序对象

```cpp
struct Person {
    std::string name;
    int age;
};

void sort_objects() {
    std::vector<Person> people = {{"Alice", 30}, {"Bob", 25}, {"Charlie", 35}};

    std::sort(people.begin(), people.end(),
              [](const Person& a, const Person& b) { return a.age < b.age; });
}
```

## 三、注意事项

- `sort`需要随机访问迭代器（vector, deque, array）
- `stable_sort`保留相等元素的相对顺序
- 比较函数必须满足严格弱序
- `is_sorted`（C++11）快速检查是否有序
