# 函数对象与算法

## 一、概念说明

算法接受函数对象（仿函数）或lambda作为谓词。函数对象是重载了`operator()`的类，可以保存状态。

## 二、具体用法

### 2.1 标准函数对象

```cpp
#include <functional>
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {5, 3, 1, 4, 2};

    // 标准仿函数
    std::sort(v.begin(), v.end(), std::greater<int>());  // 降序
    std::sort(v.begin(), v.end(), std::less<int>());     // 升序

    // 算术仿函数
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    std::vector<int> c(3);
    std::transform(a.begin(), a.end(), b.begin(), c.begin(), std::plus<int>());
    // c: {5, 7, 9}

    // 比较仿函数
    bool result = std::all_of(v.begin(), v.end(),
                               std::bind(std::greater<int>(), std::placeholders::_1, 0));
}
```

### 2.2 自定义仿函数

```cpp
struct IsEven {
    bool operator()(int x) const { return x % 2 == 0; }
};

struct Multiplier {
    int factor;
    Multiplier(int f) : factor(f) {}
    int operator()(int x) const { return x * factor; }
};

void custom_functors() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 用仿函数做谓词
    int even_count = std::count_if(v.begin(), v.end(), IsEven());

    // 用仿函数做变换
    std::transform(v.begin(), v.end(), v.begin(), Multiplier(2));
}
```

### 2.3 Lambda表达式

```cpp
void lambda_usage() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 简单lambda
    std::for_each(v.begin(), v.end(), [](int x) { std::cout << x << " "; });

    // 带捕获的lambda
    int threshold = 3;
    auto count = std::count_if(v.begin(), v.end(),
                                [threshold](int x) { return x > threshold; });

    // mutable lambda
    int sum = 0;
    std::for_each(v.begin(), v.end(), [&sum](int x) { sum += x; });
}
```

## 三、注意事项

- 仿函数应是无状态或轻量的
- lambda比仿函数更简洁，但不能有模板参数
- `<functional>`提供标准仿函数（`plus`、`greater`等）
- bind已被lambda大部分替代
