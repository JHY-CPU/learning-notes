# 函数对象与算法

## 一、概念说明

算法接受函数对象（仿函数）或lambda作为谓词（C++标准 §25.2）。函数对象是重载了`operator()`的类，可以保存状态。`<functional>`头文件提供标准仿函数（`plus`、`greater`等）。现代C++中lambda比仿函数更常用，但仿函数在需要模板参数时仍不可替代。

### 1.1 三种可调用对象

| 类型 | 状态 | 模板 | 简洁性 |
|------|------|------|--------|
| 函数指针 | 无 | 否 | 中 |
| 仿函数 | 可有 | 可以 | 低 |
| lambda | 可有 | C++20 | 最高 |

## 二、具体用法

### 2.1 标准仿函数

```cpp
#include <functional>
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {5, 3, 1, 4, 2};

    // 标准仿函数
    std::sort(v.begin(), v.end(), std::greater<int>());  // 降序
    std::sort(v.begin(), v.end(), std::less<int>());     // 升序

    // 算术仿函数
    std::vector<int> a = {1, 2, 3};
    std::vector<int> b = {4, 5, 6};
    std::vector<int> c(3);
    std::transform(a.begin(), a.end(), b.begin(), c.begin(), std::plus<int>());
    // c: {5, 7, 9}

    // 逻辑仿函数
    std::vector<bool> flags = {true, false, true};
    std::vector<bool> result(3);
    std::transform(flags.begin(), flags.end(), result.begin(), std::logical_not<bool>());
}
```

### 2.2 自定义仿函数

```cpp
struct IsEven {
    bool operator()(int x) const { return x % 2 == 0; }
};

struct Multiplier {
    int factor;
    explicit Multiplier(int f) : factor(f) {}
    int operator()(int x) const { return x * factor; }
};

void custom_functors() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 用仿函数做谓词
    int even_count = std::count_if(v.begin(), v.end(), IsEven());

    // 用仿函数做变换
    std::transform(v.begin(), v.end(), v.begin(), Multiplier(2));

    // 有状态仿函数
    struct Accumulator {
        int sum = 0;
        void operator()(int x) { sum += x; }
    };
    Accumulator acc = std::for_each(v.begin(), v.end(), Accumulator());
    std::cout << "总和: " << acc.sum << std::endl;
}
```

### 2.3 Lambda表达式

```cpp
void lambda_usage() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 简单lambda
    std::for_each(v.begin(), v.end(), [](int x) { std::cout << x << " "; });

    // 带捕获的lambda
    int threshold = 3;
    auto count = std::count_if(v.begin(), v.end(),
                                [threshold](int x) { return x > threshold; });

    // mutable lambda
    int sum = 0;
    std::for_each(v.begin(), v.end(), [&sum](int x) { sum += x; });

    // 泛型lambda（C++14）
    auto print = [](const auto& container) {
        for (const auto& x : container) std::cout << x << " ";
    };
}
```

## 三、注意事项

1. **仿函数应是无状态或轻量的**：拷贝开销影响性能
2. **lambda比仿函数更简洁**：但不能有模板参数（C++20前）
3. **`<functional>`提供标准仿函数**：`plus`、`greater`、`not_fn`等
4. **bind已被lambda大部分替代**
5. **有状态仿函数可在`for_each`后获取结果**
6. **`std::function`提供类型擦除**：但有性能开销
