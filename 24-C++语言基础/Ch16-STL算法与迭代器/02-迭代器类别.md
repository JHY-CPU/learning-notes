# 迭代器类别

## 一、概念说明

迭代器按能力分为五个类别（C++标准 §24.2），形成层次结构。较高级别的迭代器具备较低级别的所有能力。算法根据迭代器类别选择最优实现——例如`std::sort`要求随机访问迭代器，而`std::find`只需要输入迭代器。

### 1.1 层次结构

```
输入迭代器 ──→ 前向迭代器 ──→ 双向迭代器 ──→ 随机访问迭代器 ──→ 连续迭代器(C++20)
输出迭代器 ↗
```

## 二、各类别详解

### 2.1 能力对比表

| 类别 | 解引用 | 自增 | 自减 | 随机跳转 | 多遍 | 内存连续 |
|------|--------|------|------|---------|------|---------|
| 输入迭代器 | 只读 | ++ | - | - | 否 | - |
| 输出迭代器 | 只写 | ++ | - | - | 否 | - |
| 前向迭代器 | 读写 | ++ | - | - | 是 | - |
| 双向迭代器 | 读写 | ++ | -- | - | 是 | - |
| 随机访问迭代器 | 读写 | ++ | -- | +/-/[] | 是 | - |
| 连续迭代器 | 读写 | ++ | -- | +/-/[] | 是 | 是 |

### 2.2 各类别示例

```cpp
#include <iterator>
#include <vector>
#include <list>
#include <forward_list>
#include <set>
#include <iostream>
#include <sstream>

int main() {
    // 随机访问迭代器：支持 +/-/[]，O(1)跳转
    std::vector<int> v = {1, 2, 3, 4, 5};
    auto vit = v.begin();
    vit += 3;              // O(1)
    std::cout << *vit;     // 4
    std::cout << vit[1];   // 5
    auto dist = vit - v.begin();  // 3

    // 双向迭代器：支持 ++/--，O(n)跳转
    std::list<int> lst = {1, 2, 3, 4, 5};
    auto lit = lst.begin();
    ++lit; --lit;          // OK
    // lit += 3;           // 编译错误！list不支持随机访问

    // 前向迭代器：只支持 ++
    std::forward_list<int> fl = {1, 2, 3};
    auto fit = fl.begin();
    ++fit;                 // OK
    // --fit;              // 编译错误！

    // 输入/输出迭代器：单遍
    std::istringstream iss("1 2 3");
    auto iit = std::istream_iterator<int>(iss);
    int val = *iit;        // 读取
    ++iit;                 // 前进（不可回退）
}
```

### 2.3 迭代器Traits检查

```cpp
#include <iterator>
#include <type_traits>
#include <vector>
#include <list>

template<typename Iter>
void print_category() {
    using Cat = typename std::iterator_traits<Iter>::iterator_category;
    if constexpr (std::is_same_v<Cat, std::random_access_iterator_tag>)
        std::cout << "随机访问" << std::endl;
    else if constexpr (std::is_same_v<Cat, std::bidirectional_iterator_tag>)
        std::cout << "双向" << std::endl;
    else if constexpr (std::is_same_v<Cat, std::forward_iterator_tag>)
        std::cout << "前向" << std::endl;
}

void check_categories() {
    print_category<std::vector<int>::iterator>();  // 随机访问
    print_category<std::list<int>::iterator>();    // 双向
}
```

### 2.4 算法对迭代器类别的要求

```cpp
/*
| 算法              | 要求的迭代器类别    |
|-------------------|-------------------|
| find, count       | 输入迭代器         |
| for_each          | 输入迭代器         |
| copy              | 输入+输出迭代器    |
| sort              | 随机访问迭代器     |
| reverse           | 双向迭代器         |
| binary_search     | 前向迭代器         |
| lower_bound       | 前向迭代器         |
| stable_sort       | 随机访问迭代器     |
| nth_element       | 随机访问迭代器     |
| rotate            | 前向迭代器         |
*/
```

## 三、注意事项

1. **算法要求特定迭代器类别**：否则编译错误
2. **随机访问迭代器的算法通常比双向迭代器的快**
3. **`advance(it, n)`根据类别选择O(1)或O(n)实现**
4. **自定义迭代器需要定义`iterator_category`**
5. **C++20的连续迭代器（contiguous_iterator_tag）**：比随机访问多一个内存连续保证
