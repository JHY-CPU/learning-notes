# 迭代器类别

## 一、概念说明

迭代器按能力分为五个类别，形成层次结构。较高级别的迭代器具备较低级别的所有能力。算法根据迭代器类别选择最优实现。

## 二、类别层次

```cpp
/*
输入迭代器 ──→ 前向迭代器 ──→ 双向迭代器 ──→ 随机访问迭代器
输出迭代器 ↗                                     ↓
                                          连续迭代器(C++20)
*/
```

### 2.1 各类别能力

```cpp
#include <iterator>
#include <vector>
#include <list>
#include <forward_list>
#include <iostream>

int main() {
    // 随机访问迭代器：支持 +/-/[]，O(1)跳转
    std::vector<int> v = {1, 2, 3, 4, 5};
    auto vit = v.begin();
    vit += 3;              // O(1)
    std::cout << *vit;     // 4
    std::cout << vit[1];   // 5

    // 双向迭代器：支持 ++/--，O(n)跳转
    std::list<int> lst = {1, 2, 3, 4, 5};
    auto lit = lst.begin();
    ++lit; --lit;          // OK
    // lit += 3;           // 编译错误！

    // 前向迭代器：只支持 ++
    std::forward_list<int> fl = {1, 2, 3};
    auto fit = fl.begin();
    ++fit;                 // OK
    // --fit;              // 编译错误！

    // 输入/输出迭代器：单遍扫描
    // istream_iterator, ostream_iterator
}
```

### 2.2 迭代器Traits

```cpp
#include <iterator>
#include <type_traits>

void check_category() {
    using VecIter = std::vector<int>::iterator;
    using ListIter = std::list<int>::iterator;

    // 检查迭代器类别
    std::cout << std::boolalpha;
    std::cout << "vector随机访问: "
              << std::is_same_v<
                    std::iterator_traits<VecIter>::iterator_category,
                    std::random_access_iterator_tag>
              << std::endl;  // true

    std::cout << "list双向: "
              << std::is_same_v<
                    std::iterator_traits<ListIter>::iterator_category,
                    std::bidirectional_iterator_tag>
              << std::endl;  // true
}
```

## 三、注意事项

- 算法要求特定迭代器类别，否则编译错误
- 随机访问迭代器的算法通常比双向迭代器的快
- `advance(it, n)`根据类别选择O(1)或O(n)实现
- 自定义迭代器需要定义`iterator_category`
