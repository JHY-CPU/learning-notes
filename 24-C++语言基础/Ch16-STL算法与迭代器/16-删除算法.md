# 删除算法

## 一、概念说明

`std::remove`（C++标准 §25.3.8）不真正删除元素，而是将未删除的元素移到前面，返回新的逻辑末尾。需要配合`erase`真正删除，这称为**erase-remove惯用法**。C++20引入了更简洁的`std::erase`和`std::erase_if`。

### 1.1 为什么不直接删除

```
算法不能改变容器大小（只通过迭代器操作）
remove通过移动元素实现"逻辑删除"
实际删除需要调用容器的erase方法
```

## 二、具体用法

### 2.1 remove

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 2, 4, 2, 5};

    // remove：移除等于值的元素（逻辑移除）
    auto new_end = std::remove(v.begin(), v.end(), 2);
    // v: {1, 3, 4, 5, ?, ?, ?} new_end指向第5个位置
    // 剩余元素的值未指定

    // erase-remove惯用法
    v.erase(new_end, v.end());
    // v: {1, 3, 4, 5}
}
```

### 2.2 remove_if

```cpp
void remove_if_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5, 6};

    // erase-remove_if惯用法
    v.erase(std::remove_if(v.begin(), v.end(),
            [](int x) { return x % 2 == 0; }), v.end());
    // v: {1, 3, 5}
}
```

### 2.3 remove_copy

```cpp
void remove_copy_demo() {
    std::vector<int> src = {1, 2, 3, 2, 4};
    std::vector<int> dst;

    // 复制时删除（不修改原数据）
    std::remove_copy(src.begin(), src.end(), std::back_inserter(dst), 2);
    // dst: {1, 3, 4}
    // src不变

    // remove_copy_if
    std::vector<int> dst2;
    std::remove_copy_if(src.begin(), src.end(), std::back_inserter(dst2),
                        [](int x) { return x > 2; });
    // dst2: {1, 2}
}
```

### 2.4 C++20 erase / erase_if

```cpp
// C++20更简洁的写法
void modern_erase() {
    std::vector<int> v = {1, 2, 3, 2, 4, 2, 5};

    // C++20: std::erase（成员函数）
    std::erase(v, 2);  // 移除所有值为2的元素
    // v: {1, 3, 4, 5}

    // C++20: std::erase_if（成员函数）
    std::erase_if(v, [](int x) { return x > 3; });
    // v: {1, 3}

    // string的erase
    std::string s = "hello world";
    std::erase(s, 'l');  // 移除所有'l'
    // s: "heo word"
}
```

## 三、注意事项

1. **`remove`不改变容器大小**：只移动元素并返回新末尾
2. **必须配合`erase`才能真正删除**：erase-remove惯用法
3. **剩余元素的值未指定**：不要依赖
4. **C++20提供了更简洁的`erase`和`erase_if`**
5. **`remove`对每个元素调用赋值**：trivial类型可能优化为memcpy
6. **`remove_if`的谓词只调用一次**：性能可控
