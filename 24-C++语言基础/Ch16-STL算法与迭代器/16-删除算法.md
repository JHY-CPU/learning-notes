# 删除算法

## 一、概念说明

`std::remove`不真正删除元素，而是将未删除的元素移到前面，返回新的逻辑末尾。需要配合`erase`真正删除，这称为**erase-remove惯用法**。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 2, 4, 2, 5};

    // remove：移除等于值的元素（逻辑移除）
    auto new_end = std::remove(v.begin(), v.end(), 2);
    // v: {1, 3, 4, 5, ?, ?, ?} new_end指向第5个位置

    // erase-remove惯用法
    v.erase(new_end, v.end());
    // v: {1, 3, 4, 5}

    // remove_if：条件移除
    std::vector<int> v2 = {1, 2, 3, 4, 5, 6};
    v2.erase(std::remove_if(v2.begin(), v2.end(),
              [](int x) { return x % 2 == 0; }), v2.end());
    // v2: {1, 3, 5}

    // remove_copy / remove_copy_if
    std::vector<int> v3 = {1, 2, 3, 2, 4};
    std::vector<int> result;
    std::remove_copy(v3.begin(), v3.end(), std::back_inserter(result), 2);
    // result: {1, 3, 4}

    for (int x : v2) std::cout << x << " ";
}
```

### 2.1 C++20 erase / erase_if

```cpp
// C++20更简洁的写法
void modern_erase() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // C++20: std::erase
    std::erase(v, 2);  // 移除所有值为2的元素

    // C++20: std::erase_if
    std::erase_if(v, [](int x) { return x > 3; });
}
```

## 三、注意事项

- `remove`不改变容器大小，只移动元素并返回新末尾
- 必须配合`erase`才能真正删除
- 剩余元素的值未指定
- C++20提供了更简洁的`erase`和`erase_if`
