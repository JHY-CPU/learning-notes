# 替换算法

## 一、概念说明

替换算法（C++标准 §25.3.5）修改范围中满足条件的元素。`replace`替换等于特定值的元素，`replace_if`按条件替换。`replace_copy`和`replace_copy_if`在复制时替换，不修改原始范围。

### 1.1 替换算法族

| 算法 | 功能 | 修改原数据 |
|------|------|-----------|
| `replace` | 替换等于值的元素 | 是 |
| `replace_if` | 按条件替换 | 是 |
| `replace_copy` | 复制时替换值 | 否 |
| `replace_copy_if` | 复制时按条件替换 | 否 |

## 二、具体用法

### 2.1 replace

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 2, 4, 2, 5};

    // replace：替换等于old_value的元素
    std::replace(v.begin(), v.end(), 2, 20);
    // v: {1, 20, 3, 20, 4, 20, 5}

    // 替换所有0为-1
    std::replace(v.begin(), v.end(), 0, -1);
}
```

### 2.2 replace_if

```cpp
void replace_if_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8};

    // 条件替换：所有>5的替换为0
    std::replace_if(v.begin(), v.end(),
                    [](int x) { return x > 5; }, 0);
    // v: {1, 2, 3, 4, 5, 0, 0, 0}

    // 替换负数为绝对值
    std::vector<int> v2 = {-1, 2, -3, 4, -5};
    std::replace_if(v2.begin(), v2.end(),
                    [](int x) { return x < 0; }, 0);
}
```

### 2.3 replace_copy

```cpp
void replace_copy_demo() {
    std::vector<int> src = {1, 2, 3, 2, 4};
    std::vector<int> dst(5);

    // 复制时替换
    std::replace_copy(src.begin(), src.end(), dst.begin(), 2, 99);
    // dst: {1, 99, 3, 99, 4}
    // src不变: {1, 2, 3, 2, 4}

    // 配合back_inserter
    std::vector<int> dst2;
    std::replace_copy_if(src.begin(), src.end(), std::back_inserter(dst2),
                         [](int x) { return x % 2 == 0; }, 0);
    // dst2: {1, 0, 3, 0, 0}
}
```

### 2.4 实用示例

```cpp
void text_processing() {
    std::string s = "Hello, World! Hello, C++!";

    // 替换所有逗号为空格
    std::replace(s.begin(), s.end(), ',', ' ');
    // s: "Hello  World! Hello  C++!"

    // 条件替换：大写变小写
    std::string s2 = "ABCdef123";
    std::replace_if(s2.begin(), s2.end(),
                    [](char c) { return std::isupper(c); }, '*');
    // s2: "***def123"
}
```

## 三、注意事项

1. **`replace`就地修改，`replace_copy`复制时修改**
2. **比较使用`==`运算符**
3. **大小不变，不会插入或删除元素**
4. **适用于批量值替换场景**
5. **`replace_copy_if`需要back_inserter（大小未知时）**
