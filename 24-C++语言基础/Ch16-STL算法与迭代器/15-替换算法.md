# 替换算法

## 一、概念说明

替换算法修改范围中满足条件的元素。`replace`替换等于特定值的元素，`replace_if`按条件替换。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 2, 4, 2, 5};

    // replace：替换等于old_value的元素
    std::replace(v.begin(), v.end(), 2, 20);
    // v: {1, 20, 3, 20, 4, 20, 5}

    // replace_if：条件替换
    std::replace_if(v.begin(), v.end(),
                    [](int x) { return x > 10; }, 0);
    // v: {1, 0, 3, 0, 4, 0, 5}

    // replace_copy：复制时替换
    std::vector<int> src = {1, 2, 3, 2, 4};
    std::vector<int> dst(5);
    std::replace_copy(src.begin(), src.end(), dst.begin(), 2, 99);
    // dst: {1, 99, 3, 99, 4}

    // replace_copy_if：复制时条件替换
    std::vector<int> dst2(5);
    std::replace_copy_if(src.begin(), src.end(), dst2.begin(),
                         [](int x) { return x % 2 == 0; }, 0);
    // dst2: {1, 0, 3, 0, 0}

    for (int x : v) std::cout << x << " ";
}
```

## 三、注意事项

- `replace`就地修改，`replace_copy`复制时修改
- 比较使用`==`运算符
- 大小不变，不会插入或删除元素
