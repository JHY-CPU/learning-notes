# 插入迭代器

## 一、概念说明

插入迭代器（Insert Iterator）是一种特殊的输出迭代器，将赋值操作转换为容器的插入操作。它们让算法可以自动扩展容器，避免手动管理大小。

## 二、具体用法

```cpp
#include <iterator>
#include <vector>
#include <list>
#include <algorithm>
#include <iostream>

int main() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst;

    // back_inserter：尾部插入
    std::copy(src.begin(), src.end(), std::back_inserter(dst));
    // dst: {1, 2, 3, 4, 5}

    // front_inserter：头部插入（需要list/deque）
    std::list<int> lst;
    std::copy(src.begin(), src.end(), std::front_inserter(lst));
    // lst: {5, 4, 3, 2, 1}（逆序）

    // inserter：指定位置插入
    std::vector<int> v2 = {10, 20};
    std::copy(src.begin(), src.end(), std::inserter(v2, v2.begin() + 1));
    // v2: {10, 1, 2, 3, 4, 5, 20}

    for (int x : dst) std::cout << x << " ";
}
```

### 2.1 与reserve配合

```cpp
void efficient_copy() {
    std::vector<int> src(1000, 42);
    std::vector<int> dst;

    // 先reserve避免多次扩容
    dst.reserve(src.size());
    std::copy(src.begin(), src.end(), std::back_inserter(dst));
}
```

## 三、注意事项

- `back_inserter`需要容器支持`push_back`
- `front_inserter`需要容器支持`push_front`（list, deque）
- 插入迭代器使算法可以扩展容器大小
- 不reserve时，每次插入可能触发扩容（性能损失）
- 插入迭代器本身是输出迭代器，只能写不能读
