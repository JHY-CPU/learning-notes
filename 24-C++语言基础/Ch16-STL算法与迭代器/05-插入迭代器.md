# 插入迭代器

## 一、概念说明

插入迭代器（Insert Iterator，C++标准 §24.5.2）是一种特殊的输出迭代器，将赋值操作转换为容器的插入操作。它们让算法可以自动扩展容器，避免手动管理大小。三种插入迭代器分别对应容器的三种插入位置：尾部、头部、指定位置。

### 1.1 三种插入迭代器

| 类型 | 操作 | 要求 |
|------|------|------|
| `back_inserter` | 调用`push_back` | 容器支持`push_back` |
| `front_inserter` | 调用`push_front` | 容器支持`push_front` |
| `inserter` | 调用`insert(pos, val)` | 容器有`insert`成员 |

## 二、具体用法

### 2.1 back_inserter

```cpp
#include <iterator>
#include <vector>
#include <algorithm>
#include <iostream>

int main() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> dst;

    // back_inserter：尾部插入
    std::copy(src.begin(), src.end(), std::back_inserter(dst));
    // dst: {1, 2, 3, 4, 5}

    // 配合transform
    std::vector<int> doubled;
    std::transform(src.begin(), src.end(), std::back_inserter(doubled),
                   [](int x) { return x * 2; });
    // doubled: {2, 4, 6, 8, 10}

    // 配合copy_if
    std::vector<int> evens;
    std::copy_if(src.begin(), src.end(), std::back_inserter(evens),
                 [](int x) { return x % 2 == 0; });
    // evens: {2, 4}
}
```

### 2.2 front_inserter

```cpp
#include <list>

void front_inserter_demo() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::list<int> lst;

    // front_inserter：头部插入（结果逆序！）
    std::copy(src.begin(), src.end(), std::front_inserter(lst));
    // lst: {5, 4, 3, 2, 1}

    // front_inserter需要push_front
    // std::vector不支持push_front
    // std::vector<int> v;
    // std::copy(src.begin(), src.end(), std::front_inserter(v));  // 编译错误！
}
```

### 2.3 inserter

```cpp
void inserter_demo() {
    std::vector<int> src = {1, 2, 3, 4, 5};
    std::vector<int> v2 = {10, 20};

    // inserter：在指定位置插入
    std::copy(src.begin(), src.end(), std::inserter(v2, v2.begin() + 1));
    // v2: {10, 1, 2, 3, 4, 5, 20}

    // 对set/map使用inserter
    std::set<int> s = {1, 2, 3};
    std::vector<int> more = {4, 5, 6};
    std::copy(more.begin(), more.end(), std::inserter(s, s.end()));
    // s: {1, 2, 3, 4, 5, 6}
}
```

### 2.4 与reserve配合

```cpp
void efficient_copy() {
    std::vector<int> src(1000, 42);
    std::vector<int> dst;

    // 先reserve避免多次扩容
    dst.reserve(src.size());
    std::copy(src.begin(), src.end(), std::back_inserter(dst));
    // 性能提升：只分配一次内存
}
```

## 三、注意事项

1. **`back_inserter`需要容器支持`push_back`**：vector、deque、list、string
2. **`front_inserter`需要容器支持`push_front`**：list、deque（vector不支持）
3. **插入迭代器使算法可以扩展容器大小**：普通迭代器不会
4. **不reserve时，每次插入可能触发扩容**：性能损失显著
5. **插入迭代器本身是输出迭代器**：只能写不能读
6. **`front_inserter`结果是逆序的**：因为每次在头部插入
