# 迭代器操作

## 一、概念说明

迭代器提供类似指针的操作：解引用（`*`）、自增（`++`）、自减（`--`）、随机访问（`+`/`-`/`[]`）。此外，STL还提供了辅助函数`advance`、`distance`、`next`、`prev`等。不同类别的迭代器支持不同操作，了解这些操作的时间复杂度对性能优化至关重要。

### 1.1 操作复杂度

| 操作 | 随机访问 | 双向 | 前向 |
|------|---------|------|------|
| `++it` | O(1) | O(1) | O(1) |
| `--it` | O(1) | O(1) | 不支持 |
| `it + n` | O(1) | 不支持 | 不支持 |
| `advance(it, n)` | O(1) | O(n) | O(n) |
| `distance(first, last)` | O(1) | O(n) | O(n) |

## 二、具体用法

### 2.1 基本操作

```cpp
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {10, 20, 30, 40, 50};

    // 创建迭代器
    auto it = v.begin();

    // 解引用
    std::cout << *it << std::endl;     // 10

    // 自增
    ++it;
    std::cout << *it << std::endl;     // 20

    // 自减（双向/随机访问）
    --it;
    std::cout << *it << std::endl;     // 10

    // 随机访问
    it += 3;
    std::cout << *it << std::endl;     // 40

    // 算术
    auto it2 = v.end();
    std::cout << std::distance(v.begin(), it2) << std::endl;  // 5

    // 比较
    if (it < it2) std::cout << "it在end之前" << std::endl;
}
```

### 2.2 std::advance

```cpp
#include <list>

void advance_demo() {
    std::list<int> lst = {1, 2, 3, 4, 5};
    auto it = lst.begin();

    // advance：前进n步（根据类别选择实现）
    std::advance(it, 3);  // list是O(n)，vector是O(1)
    std::cout << *it << std::endl;  // 4

    // advance可以后退（负数）
    std::advance(it, -2);
    std::cout << *it << std::endl;  // 2

    // advance对随机访问是O(1)，对双向是O(n)
}
```

### 2.3 std::distance

```cpp
void distance_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // distance：计算两个迭代器之间的距离
    auto d = std::distance(v.begin(), v.end());
    std::cout << d << std::endl;  // 5

    // 对随机访问是O(1)，对其他是O(n)
    d = std::distance(v.begin(), v.begin() + 3);
    std::cout << d << std::endl;  // 3

    // 注意：first必须能到达last
    // std::distance(v.end(), v.begin());  // 未定义行为！
}
```

### 2.4 std::next / std::prev（C++11）

```cpp
void next_prev_demo() {
    std::list<int> lst = {1, 2, 3, 4, 5};

    // next：不修改原迭代器，返回新迭代器
    auto next_it = std::next(lst.begin(), 2);
    std::cout << *next_it << std::endl;  // 3

    // prev：不修改原迭代器
    auto prev_it = std::prev(lst.end(), 1);
    std::cout << *prev_it << std::endl;  // 5

    // 默认步长为1
    auto it = std::next(lst.begin());
    std::cout << *it << std::endl;  // 2

    // 对比advance（修改原迭代器）
    auto it2 = lst.begin();
    std::advance(it2, 2);  // it2被修改
}
```

### 2.5 常用模式

```cpp
void patterns() {
    std::vector<int> v = {1, 2, 3, 4, 5};

    // 获取倒数第n个元素
    auto last3 = std::prev(v.end(), 3);
    std::cout << *last3 << std::endl;  // 3

    // 安全地获取中间元素
    auto mid = std::next(v.begin(), std::distance(v.begin(), v.end()) / 2);
    std::cout << *mid << std::endl;  // 3

    // 检查是否至少有n个元素
    if (std::distance(v.begin(), v.end()) >= 3) {
        auto third = std::next(v.begin(), 2);
        std::cout << *third << std::endl;  // 3
    }
}
```

## 三、注意事项

1. **`advance`对随机访问迭代器是O(1)，对双向是O(n)**
2. **`distance`要求两个迭代器可到达**：`first`必须在`last`之前
3. **`end()`指向最后一个元素之后，不能解引用**
4. **`cbegin()`/`cend()`返回const_iterator**
5. **`next`/`prev`不修改原迭代器**：比`advance`更安全
6. **负数步长只对双向/随机访问迭代器有效**
