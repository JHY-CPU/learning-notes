# 迭代器操作

## 一、概念说明

迭代器提供类似指针的操作：解引用（`*`）、自增（`++`）、自减（`--`）、随机访问（`+`/`-`/`[]`）。不同类别的迭代器支持不同操作。

## 二、具体用法

### 2.1 基本操作

```cpp
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {10, 20, 30, 40, 50};

    // 创建迭代器
    auto it = v.begin();

    // 解引用
    std::cout << *it << std::endl;     // 10

    // 自增
    ++it;
    std::cout << *it << std::endl;     // 20

    // 自减（双向/随机访问）
    --it;
    std::cout << *it << std::endl;     // 10

    // 随机访问
    it += 3;
    std::cout << *it << std::endl;     // 40

    // 算术
    auto it2 = v.end();
    std::cout << std::distance(v.begin(), it2) << std::endl;  // 5

    // 比较
    if (it < it2) std::cout << "it在end之前" << std::endl;
}
```

### 2.2 std::advance / std::distance

```cpp
#include <list>

void advance_demo() {
    std::list<int> lst = {1, 2, 3, 4, 5};
    auto it = lst.begin();

    // advance：前进n步（根据类别选择实现）
    std::advance(it, 3);  // list是O(n)，vector是O(1)
    std::cout << *it << std::endl;  // 4

    // distance：计算两个迭代器之间的距离
    auto d = std::distance(lst.begin(), lst.end());
    std::cout << d << std::endl;  // 5

    // next / prev（C++11）：不修改原迭代器
    auto next_it = std::next(lst.begin(), 2);
    auto prev_it = std::prev(lst.end(), 1);
}
```

## 三、注意事项

- `advance`对随机访问迭代器是O(1)，对双向是O(n)
- `distance`要求两个迭代器可到达
- `end()`指向最后一个元素之后，不能解引用
- `cbegin()`/`cend()`返回const_iterator
