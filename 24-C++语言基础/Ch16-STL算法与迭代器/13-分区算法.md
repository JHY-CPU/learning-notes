# 分区算法

## 一、概念说明

分区算法将范围分为满足条件和不满足条件的两部分。`partition`不稳定，`stable_partition`保留相对顺序。

## 二、具体用法

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8};

    // partition：不稳定分区
    auto it = std::partition(v.begin(), v.end(),
                              [](int x) { return x % 2 == 0; });
    // 前半部分是偶数，后半部分是奇数
    std::cout << "偶数: ";
    for (auto i = v.begin(); i != it; ++i) std::cout << *i << " ";
    std::cout << "\n奇数: ";
    for (auto i = it; i != v.end(); ++i) std::cout << *i << " ";

    // is_partitioned：检查是否已分区
    std::cout << "\nis_partitioned: "
              << std::is_partitioned(v.begin(), v.end(),
                                      [](int x) { return x % 2 == 0; });

    // stable_partition：稳定分区（保留相对顺序）
    std::vector<int> v2 = {1, 2, 3, 4, 5, 6};
    std::stable_partition(v2.begin(), v2.end(),
                          [](int x) { return x % 2 == 0; });

    // partition_point：找到分区点
    auto pp = std::partition_point(v.begin(), v.end(),
                                    [](int x) { return x % 2 == 0; });
}
```

## 三、注意事项

- `partition`不保证相对顺序，`stable_partition`保证
- `partition`返回分区点（第一个不满足条件的元素）
- `partition_point`需要范围已分区
- 应用：快速排序、top-k问题
