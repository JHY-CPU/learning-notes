# 分区算法

## 一、概念说明

分区算法（C++标准 §25.4.1.3）将范围分为满足条件和不满足条件的两部分。`partition`不稳定（O(n)），`stable_partition`保留相对顺序（O(n log n)）。分区是快速排序的核心操作。

### 1.1 分区 vs 排序

| 操作 | 结果 | 复杂度 |
|------|------|--------|
| 分区 | 满足条件的在前，不满足的在后 | O(n) |
| 排序 | 完全有序 | O(n log n) |

## 二、具体用法

### 2.1 partition

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8};

    // partition：不稳定分区
    auto pivot = std::partition(v.begin(), v.end(),
                                 [](int x) { return x % 2 == 0; });
    // 前半部分是偶数，后半部分是奇数

    std::cout << "偶数: ";
    for (auto i = v.begin(); i != pivot; ++i) std::cout << *i << " ";
    std::cout << "\n奇数: ";
    for (auto i = pivot; i != v.end(); ++i) std::cout << *i << " ";
    std::cout << std::endl;

    // is_partitioned：检查是否已分区
    bool is_part = std::is_partitioned(v.begin(), v.end(),
                                        [](int x) { return x % 2 == 0; });
    std::cout << "已分区: " << is_part << std::endl;
}
```

### 2.2 stable_partition

```cpp
void stable_demo() {
    std::vector<int> v = {1, 2, 3, 4, 5, 6};

    // stable_partition：保留相对顺序
    std::stable_partition(v.begin(), v.end(),
                          [](int x) { return x % 2 == 0; });
    // v: {2, 4, 6, 1, 3, 5}  偶数和奇数都保持原顺序
}
```

### 2.3 partition_point

```cpp
void partition_point_demo() {
    std::vector<int> v = {2, 4, 6, 1, 3, 5};
    // 假设已分区

    // partition_point：找到分区点
    auto pp = std::partition_point(v.begin(), v.end(),
                                    [](int x) { return x % 2 == 0; });
    std::cout << "分区点位置: " << std::distance(v.begin(), pp) << std::endl;  // 3
}
```

### 2.4 应用：快速排序中的分区

```cpp
template<typename Iter>
Iter partition_impl(Iter first, Iter last) {
    auto pivot = *std::prev(last);
    auto bound = first;
    for (auto it = first; it != std::prev(last); ++it) {
        if (*it < pivot) {
            std::iter_swap(bound, it);
            ++bound;
        }
    }
    std::iter_swap(bound, std::prev(last));
    return bound;
}
```

## 三、注意事项

1. **`partition`不保证相对顺序**：`stable_partition`保证
2. **`partition`返回分区点**：第一个不满足条件的元素
3. **`partition_point`需要范围已分区**
4. **应用**：快速排序、top-k问题、数据分类
5. **`partition`是O(n)**：比完全排序快
