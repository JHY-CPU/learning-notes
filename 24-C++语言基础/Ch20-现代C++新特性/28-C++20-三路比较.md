# C++20 三路比较 operator<=>

## 一、概念说明

`operator<=>`（三路比较运算符，Spaceship Operator，C++20 §7.6.8, §11.10）允许编译器自动生成所有六种比较运算符（`==`, `!=`, `<`, `>`, `<=`, `>=`）。它返回比较结果类型，指示操作数的顺序关系。

### 1.1 比较类别

| 类型 | 含义 | 例子 |
|------|------|------|
| `std::strong_ordering` | 完全有序、可替换 | 整数、字符串 |
| `std::weak_ordering` | 有序但等价不完全相同 | 大小写不敏感比较 |
| `std::partial_ordering` | 可能无序 | 浮点数（NaN） |

```cpp
#include <iostream>
#include <compare>
#include <string>

struct Student {
    std::string name;
    int score;

    // 一行生成所有比较运算符
    auto operator<=>(const Student&) const = default;
};

int main() {
    Student a{"Alice", 95};
    Student b{"Bob", 87};
    Student c{"Alice", 95};

    std::cout << "a > b: " << (a > b) << std::endl;
    std::cout << "a == c: " << (a == c) << std::endl;
    std::cout << "b < a: " << (b < a) << std::endl;

    // 三路比较结果
    auto result = a <=> b;
    std::cout << "a <=> b: " << (result > 0 ? "a > b" : "a < b") << std::endl;

    return 0;
}
```

**输出：**
```
a > b: 1
a == c: 1
b < a: 1
a <=> b: a > b
```

## 二、具体用法

### 2.1 自定义比较逻辑

```cpp
#include <iostream>
#include <compare>

struct Version {
    int major, minor, patch;

    // 只比较major和minor，patch不参与排序
    auto operator<=>(const Version& other) const {
        if (auto cmp = major <=> other.major; cmp != 0) return cmp;
        if (auto cmp = minor <=> other.minor; cmp != 0) return cmp;
        return std::strong_ordering::equal;
    }

    bool operator==(const Version& other) const {
        return major == other.major && minor == other.minor;
    }
};

int main() {
    Version v1{1, 2, 3};
    Version v2{1, 2, 5}; // patch不同但比较相等
    Version v3{1, 3, 0};

    std::cout << "v1 == v2: " << (v1 == v2) << std::endl;
    std::cout << "v1 < v2: " << (v1 < v2) << std::endl;
    std::cout << "v1 < v3: " << (v1 < v3) << std::endl;

    return 0;
}
```

**输出：**
```
v1 == v2: 1
v1 < v2: 0
v1 < v3: 1
```

### 2.2 浮点数比较

```cpp
#include <iostream>
#include <compare>
#include <cmath>

struct Point {
    double x, y;

    // partial_ordering：因为浮点数可能有NaN
    std::partial_ordering operator<=>(const Point& other) const {
        if (auto cmp = x <=> other.x; cmp != 0) return cmp;
        return y <=> other.y;
    }

    bool operator==(const Point& other) const {
        return x == other.x && y == other.y;
    }
};

int main() {
    Point p1{1.0, 2.0};
    Point p2{1.0, 3.0};

    std::cout << "p1 < p2: " << (p1 < p2) << std::endl;

    // NaN情况
    Point p_nan{std::nan(""), 0.0};
    auto cmp = p1 <=> p_nan;
    std::cout << "p1 <=> NaN: unordered=" << (cmp == std::partial_ordering::unordered) << std::endl;

    return 0;
}
```

### 2.3 简化类的比较

```cpp
#include <iostream>
#include <compare>
#include <string>

// = default自动生成按成员声明顺序的比较
struct Person {
    std::string name;
    int age;
    double salary;

    auto operator<=>(const Person&) const = default;
    // 自动生成：先比较name，再比较age，再比较salary
};

int main() {
    Person a{"Alice", 30, 50000};
    Person b{"Bob", 25, 60000};

    std::cout << "a < b: " << (a < b) << std::endl;   // Alice < Bob
    std::cout << "a > b: " << (a > b) << std::endl;

    Person c{"Alice", 30, 50000};
    std::cout << "a == c: " << (a == c) << std::endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **`= default`自动生成所有比较**：按成员声明顺序，不一定是逻辑重要性顺序。
2. **`operator==`也可以`= default`**：编译器自动生成。
3. **浮点数用`partial_ordering`**：因为NaN的存在，NaN与任何值无序。
4. **`operator<=>`在C++20引入**：需要GCC 10+、Clang 10+、MSVC 19.22+。
5. **自动生成的比较按成员声明顺序**：如果有逻辑优先级需求，需要手写。
6. **`<compare>`头文件**：需要包含才能使用比较类别类型。
