# C++20 三路比较 operator<=>

## 一、概念说明

`operator<=>`（三路比较运算符，spaceship operator）允许编译器自动生成所有六种比较运算符。它返回一个比较结果类型（`std::strong_ordering`、`std::weak_ordering`、`std::partial_ordering`）。

```cpp
#include <iostream>
#include <compare>
#include <string>

struct Student {
    std::string name;
    int score;

    // 一行生成所有比较运算符
    auto operator<=>(const Student&) const = default;
};

int main() {
    Student a{"Alice", 95};
    Student b{"Bob", 87};
    Student c{"Alice", 95};

    std::cout << "a > b: " << (a > b) << std::endl;
    std::cout << "a == c: " << (a == c) << std::endl;
    std::cout << "b < a: " << (b < a) << std::endl;

    // 三路比较结果
    auto result = a <=> b;
    std::cout << "a <=> b: " << (result > 0 ? "a > b" : "a < b") << std::endl;

    return 0;
}
```

**输出：**
```
a > b: 1
a == c: 1
b < a: 1
a <=> b: a > b
```

## 二、具体用法

### 2.1 自定义比较

```cpp
#include <iostream>
#include <compare>

struct Version {
    int major, minor, patch;

    // 只比较major和minor，patch不参与比较
    auto operator<=>(const Version& other) const {
        if (auto cmp = major <=> other.major; cmp != 0) return cmp;
        if (auto cmp = minor <=> other.minor; cmp != 0) return cmp;
        return std::strong_ordering::equal;
    }

    bool operator==(const Version& other) const {
        return major == other.major && minor == other.minor;
    }
};

int main() {
    Version v1{1, 2, 3};
    Version v2{1, 2, 5}; // patch不同但比较相等

    std::cout << "v1 == v2: " << (v1 == v2) << std::endl;
    std::cout << "v1 < v2: " << (v1 < v2) << std::endl;

    Version v3{1, 3, 0};
    std::cout << "v1 < v3: " << (v1 < v3) << std::endl;

    return 0;
}
```

**输出：**
```
v1 == v2: 1
v1 < v2: 0
v1 < v3: 1
```

### 2.2 比较类别

| 类型 | 含义 | 例子 |
|------|------|------|
| `strong_ordering` | 完全有序、可替换 | 整数、字符串 |
| `weak_ordering` | 有序但等价不完全相同 | 大小写不敏感比较 |
| `partial_ordering` | 可能无序 | 浮点数（NaN） |

## 三、注意事项与常见陷阱

- **`= default`自动生成所有比较**：按成员声明顺序。
- **`operator==`也可以`= default`**：编译器自动生成。
- **浮点数用`partial_ordering`**：因为NaN的存在。
- **`operator<=>`在C++20引入**：需要编译器支持。
- **自动生成的比较按成员声明顺序**：不按逻辑重要性。
