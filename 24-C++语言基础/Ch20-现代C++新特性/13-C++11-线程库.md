# C++11 线程库

## 一、概念说明

C++11在`<thread>`、`<mutex>`、`<condition_variable>`、`<future>`、`<atomic>`中提供了标准线程支持，结束了依赖平台API的历史。

```cpp
#include <iostream>
#include <thread>
#include <mutex>
#include <atomic>

std::atomic<int> counter{0};
std::mutex mtx;

void increment(int times) {
    for (int i = 0; i < times; ++i) {
        counter.fetch_add(1);
    }
}

int main() {
    std::thread t1(increment, 100000);
    std::thread t2(increment, 100000);
    t1.join();
    t2.join();

    std::cout << "counter=" << counter.load() << std::endl;
    return 0;
}
```

**输出：**
```
counter=200000
```

## 二、具体用法

### 2.1 核心组件一览

| 头文件 | 组件 |
|--------|------|
| `<thread>` | `thread`, `this_thread` |
| `<mutex>` | `mutex`, `lock_guard`, `unique_lock`, `scoped_lock` |
| `<condition_variable>` | `condition_variable`, `condition_variable_any` |
| `<future>` | `future`, `promise`, `async`, `packaged_task` |
| `<atomic>` | `atomic`, `atomic_flag` |

## 三、注意事项与常见陷阱

- **`thread`析构前必须`join`或`detach`**。
- **`mutex`不能拷贝**。
- **`future`析构会阻塞**（如果关联`async`任务）。
- **`atomic`只保证单个变量的原子性**。
- **详细内容参见Ch19多线程章节**。
