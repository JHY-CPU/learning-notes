# C++11 右值与移动

## 一、概念说明

C++11引入右值引用（`T&&`）和移动语义，允许"窃取"临时对象的资源，避免昂贵的深拷贝（C++11 §5.2.11, §12.8）。`std::move`将左值转为右值引用，`std::forward`实现完美转发。

### 1.1 值类别

| 类别 | 特征 | 示例 |
|------|------|------|
| 左值（lvalue） | 有名字、可取地址 | 变量、解引用 |
| 将亡值（xvalue） | 即将被移动的对象 | `std::move(x)` |
| 纯右值（prvalue） | 临时对象、字面量 | `42`、`func()` |

```cpp
#include <iostream>
#include <utility>
#include <string>

class Buffer {
    int* data;
    size_t size_;
public:
    Buffer(size_t n) : data(new int[n]), size_(n) {
        std::cout << "构造(size=" << n << ")" << std::endl;
    }

    // 拷贝构造：深拷贝
    Buffer(const Buffer& o) : data(new int[o.size_]), size_(o.size_) {
        std::copy(o.data, o.data + size_, data);
        std::cout << "拷贝构造" << std::endl;
    }

    // 移动构造：窃取资源
    Buffer(Buffer&& o) noexcept : data(o.data), size_(o.size_) {
        o.data = nullptr;
        o.size_ = 0;
        std::cout << "移动构造" << std::endl;
    }

    // 拷贝赋值
    Buffer& operator=(const Buffer& o) {
        if (this != &o) {
            delete[] data;
            size_ = o.size_;
            data = new int[size_];
            std::copy(o.data, o.data + size_, data);
        }
        return *this;
    }

    // 移动赋值
    Buffer& operator=(Buffer&& o) noexcept {
        if (this != &o) {
            delete[] data;
            data = o.data;
            size_ = o.size_;
            o.data = nullptr;
            o.size_ = 0;
        }
        return *this;
    }

    ~Buffer() { delete[] data; }
    size_t size() const { return size_; }
};

int main() {
    Buffer a(100);
    Buffer b = std::move(a); // 移动构造
    std::cout << "b.size=" << b.size() << ", a.size=" << a.size() << std::endl;

    Buffer c(50);
    c = std::move(b);        // 移动赋值
    std::cout << "c.size=" << c.size() << ", b.size=" << b.size() << std::endl;

    return 0;
}
```

**输出：**
```
构造(size=100)
移动构造
b.size=100, a.size=0
构造(size=50)
移动赋值
c.size=100, b.size=0
```

## 二、具体用法

### 2.1 完美转发

`std::forward`在模板中保持参数的值类别，实现完美转发。

```cpp
#include <iostream>
#include <utility>

void process(int& x) { std::cout << "左值: " << x << std::endl; }
void process(int&& x) { std::cout << "右值: " << x << std::endl; }
void process(const int& x) { std::cout << "const左值: " << x << std::endl; }

template <typename T>
void wrapper(T&& arg) {
    process(std::forward<T>(arg)); // 保持值类别
}

int main() {
    int x = 42;
    wrapper(x);            // 左值：T推导为int&
    wrapper(100);          // 右值：T推导为int
    wrapper(std::move(x)); // 右值：T推导为int
    return 0;
}
```

**输出：**
```
左值: 42
右值: 100
右值: 42
```

### 2.2 引用折叠规则

```
T&  &  -> T&
T&  && -> T&
T&& &  -> T&
T&& && -> T&&
```

```cpp
#include <iostream>
#include <type_traits>

template <typename T>
void check(T&& arg) {
    if constexpr (std::is_lvalue_reference_v<T>) {
        std::cout << "推导为左值引用" << std::endl;
    } else {
        std::cout << "推导为右值引用" << std::endl;
    }
}

int main() {
    int x = 42;
    check(x);   // T推导为int&，T&& = int& && = int&
    check(42);  // T推导为int，T&& = int&&
    return 0;
}
```

**输出：**
```
推导为左值引用
推导为右值引用
```

### 2.3 移动语义在容器中的应用

```cpp
#include <iostream>
#include <vector>
#include <string>

int main() {
    std::vector<std::string> v;

    std::string s = "hello world";

    // 拷贝：s仍有效
    v.push_back(s);
    std::cout << "拷贝后: s=" << s << std::endl;

    // 移动：s被掏空
    v.push_back(std::move(s));
    std::cout << "移动后: s=" << s << " (可能为空)" << std::endl;

    // emplace直接构造（最高效）
    v.emplace_back("in-place");

    std::cout << "容器: ";
    for (const auto& item : v) std::cout << "[" << item << "] ";
    std::cout << std::endl;

    return 0;
}
```

**输出：**
```
拷贝后: s=hello world
移动后: s= (可能为空)
容器: [hello world] [] [in-place]
```

## 三、注意事项与常见陷阱

1. **`std::move`不移动任何东西**：只是类型转换（左值转为右值引用），真正的移动由移动构造/赋值完成。
2. **移动后对象处于有效但未指定状态**：只能析构或赋新值，不能依赖之前的值。
3. **`noexcept`对移动非常重要**：`std::vector`扩容时只使用`noexcept`的移动构造函数。
4. **万能引用（Universal Reference）只出现在模板中**：`auto&&`也是万能引用。
5. **完美转发需要`std::forward`**：保持参数原始的值类别，不能用`std::move`。
6. **移动不总是更快**：对于`int`等POD类型，移动和拷贝效率相同。
