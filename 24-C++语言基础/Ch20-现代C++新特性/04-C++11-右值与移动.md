# C++11 右值与移动

## 一、概念说明

C++11引入右值引用（`T&&`）和移动语义，允许"窃取"临时对象的资源，避免昂贵的深拷贝。`std::move`将左值转为右值引用，`std::forward`实现完美转发。

- **左值**：有名字、可取地址的表达式
- **右值**：临时对象、字面量、`std::move`的结果
- **万能引用**：模板中的`T&&`可推导为左值或右值引用

```cpp
#include <iostream>
#include <utility>
#include <string>

class Buffer {
    int* data;
    size_t size_;
public:
    Buffer(size_t n) : data(new int[n]), size_(n) {
        std::cout << "构造(size=" << n << ")" << std::endl;
    }

    // 拷贝构造：深拷贝
    Buffer(const Buffer& o) : data(new int[o.size_]), size_(o.size_) {
        std::copy(o.data, o.data + size_, data);
        std::cout << "拷贝构造" << std::endl;
    }

    // 移动构造：窃取资源
    Buffer(Buffer&& o) noexcept : data(o.data), size_(o.size_) {
        o.data = nullptr;
        o.size_ = 0;
        std::cout << "移动构造" << std::endl;
    }

    ~Buffer() { delete[] data; }
    size_t size() const { return size_; }
};

int main() {
    Buffer a(100);
    Buffer b = std::move(a); // 移动构造
    std::cout << "b.size=" << b.size() << ", a.size=" << a.size() << std::endl;

    return 0;
}
```

**输出：**
```
构造(size=100)
移动构造
b.size=100, a.size=0
```

## 二、具体用法

### 2.1 完美转发

```cpp
#include <iostream>
#include <utility>

void process(int& x) { std::cout << "左值: " << x << std::endl; }
void process(int&& x) { std::cout << "右值: " << x << std::endl; }

template <typename T>
void wrapper(T&& arg) {
    process(std::forward<T>(arg)); // 保持值类别
}

int main() {
    int x = 42;
    wrapper(x);            // 左值
    wrapper(100);          // 右值
    wrapper(std::move(x)); // 右值
    return 0;
}
```

**输出：**
```
左值: 42
右值: 100
右值: 42
```

### 2.2 引用折叠规则

```
T&  &  -> T&
T&  && -> T&
T&& &  -> T&
T&& && -> T&&
```

```cpp
#include <iostream>
#include <type_traits>

template <typename T>
void check(T&& arg) {
    if constexpr (std::is_lvalue_reference_v<T>) {
        std::cout << "推导为左值引用" << std::endl;
    } else {
        std::cout << "推导为右值引用" << std::endl;
    }
}

int main() {
    int x = 42;
    check(x);   // T推导为int&，T&& = int& && = int&
    check(42);  // T推导为int，T&& = int&&
    return 0;
}
```

**输出：**
```
推导为左值引用
推导为右值引用
```

## 三、注意事项与常见陷阱

- **`std::move`不移动任何东西**：只是类型转换。
- **移动后对象处于有效但未指定状态**：只能析构或赋新值。
- **`noexcept`对移动很重要**：`std::vector`扩容时只用`noexcept`移动。
- **万能引用只出现在模板中**：`auto&&`也是万能引用。
- **完美转发需要`std::forward`**：保持参数原始的值类别。
