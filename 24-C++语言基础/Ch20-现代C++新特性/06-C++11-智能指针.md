# C++11 智能指针

## 一、概念说明

C++11引入三种智能指针（`<memory>`头文件），自动管理动态内存：
- `unique_ptr`：独占所有权，零开销
- `shared_ptr`：共享所有权，引用计数
- `weak_ptr`：观察者，不增加引用计数

```cpp
#include <iostream>
#include <memory>

struct Resource {
    std::string name;
    Resource(std::string n) : name(std::move(n)) {
        std::cout << name << " 创建" << std::endl;
    }
    ~Resource() { std::cout << name << " 销毁" << std::endl; }
};

int main() {
    // unique_ptr：独占所有权
    auto u1 = std::make_unique<Resource>("独占资源");
    // auto u2 = u1; // 错误：不能拷贝
    auto u2 = std::move(u1); // 移动所有权

    // shared_ptr：共享所有权
    auto s1 = std::make_shared<Resource>("共享资源");
    {
        auto s2 = s1; // 拷贝，引用计数+1
        std::cout << "引用计数: " << s1.use_count() << std::endl;
    } // s2销毁，引用计数-1

    // weak_ptr：弱引用
    std::weak_ptr<Resource> w = s1;
    if (auto locked = w.lock()) {
        std::cout << "lock成功: " << locked->name << std::endl;
    }

    std::cout << "离开作用域..." << std::endl;
    return 0;
}
```

**输出：**
```
独占资源 创建
共享资源 创建
引用计数: 2
lock成功: 共享资源
离开作用域...
共享资源 销毁
独占资源 销毁
```

## 二、具体用法

### 2.1 推荐用法

```cpp
#include <iostream>
#include <memory>

// 工厂函数返回unique_ptr
std::unique_ptr<int> createInt(int val) {
    return std::make_unique<int>(val);
}

// 函数参数传递策略
void takeOwnership(std::unique_ptr<int> p) {
    std::cout << "获取所有权: " << *p << std::endl;
}

void borrowRef(const std::shared_ptr<int>& p) {
    std::cout << "借用: " << *p << std::endl;
}

int main() {
    // 工厂模式
    auto p = createInt(42);

    // 转移所有权
    takeOwnership(std::move(p));

    // 共享
    auto sp = std::make_shared<int>(99);
    borrowRef(sp);

    return 0;
}
```

**输出：**
```
获取所有权: 42
借用: 99
```

## 三、注意事项与常见陷阱

- **优先使用`make_unique`/`make_shared`**：避免直接`new`。
- **循环引用用`weak_ptr`打破**：A持有B的`shared_ptr`，B持有A的`weak_ptr`。
- **不要用同一裸指针创建多个`shared_ptr`**：会双重释放。
- **`unique_ptr`大小等于裸指针**：零额外开销。
- **`make_shared`合并分配**：对象和控制块在一起，`weak_ptr`引用时整个块不释放。
