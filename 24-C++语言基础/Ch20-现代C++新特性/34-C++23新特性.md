# C++23新特性

## 一、概念说明

C++23继续完善标准库，主要新增：
- `std::expected<T, E>`：预期值或错误
- `std::print`/`std::println`：格式化输出
- `std::flat_map`/`std::flat_set`：基于排序数组的容器
- 多维下标`operator[]`
- `std::generator`：协程生成器

```cpp
#include <iostream>
#include <expected>
#include <string>
#include <cmath>

// expected: 表示可能失败的操作
std::expected<double, std::string> safeSqrt(double x) {
    if (x < 0) return std::unexpected("负数无实数平方根");
    return std::sqrt(x);
}

int main() {
    // expected
    auto r1 = safeSqrt(4.0);
    auto r2 = safeSqrt(-1.0);

    if (r1) std::cout << "sqrt(4) = " << *r1 << std::endl;
    if (!r2) std::cout << "错误: " << r2.error() << std::endl;

    // print (C++23)
    // std::print("Hello, {}!\n", "C++23");
    // std::println("值: {}", 42);

    // 多维下标
    // std::array<std::array<int, 3>, 3> matrix;
    // matrix[1][2] = 42;

    return 0;
}
```

**输出：**
```
sqrt(4) = 2
错误: 负数无实数平方根
```

## 二、具体用法

### 2.1 expected使用

```cpp
#include <iostream>
#include <expected>
#include <string>

std::expected<int, std::string> parseInt(const std::string& s) {
    try {
        return std::stoi(s);
    } catch (...) {
        return std::unexpected("解析失败: " + s);
    }
}

int main() {
    auto results = {parseInt("42"), parseInt("abc"), parseInt("100")};

    for (const auto& r : results) {
        if (r) std::cout << "值: " << *r << std::endl;
        else std::cout << "错误: " << r.error() << std::endl;
    }

    // monadic操作
    auto doubled = parseInt("21").transform([](int x) { return x * 2; });
    if (doubled) std::cout << "翻倍: " << *doubled << std::endl;

    return 0;
}
```

**输出：**
```
值: 42
错误: 解析失败: abc
值: 100
翻倍: 42
```

### 2.2 C++23特性一览

| 特性 | 说明 |
|------|------|
| `std::print` | 格式化输出 |
| `std::expected` | 预期值或错误 |
| `flat_map`/`flat_set` | 基于数组的有序容器 |
| 多维`operator[]` | `arr[1,2,3]` |
| `std::generator` | 协程生成器 |
| `std::to_underlying` | 枚举转底层类型 |
| `auto(x)` / `auto{x}` | 显式衰变拷贝 |

## 三、注意事项与常见陷阱

- **编译器支持有限**：GCC 13+、Clang 17+、MSVC 2022 17.6+部分支持。
- **`expected`类似Rust的`Result`**：类型安全的错误处理。
- **`print`不加`\n`不换行**：用`println`自动换行。
- **`flat_map`比`std::map`缓存友好**：适合小量数据。
- **`std::generator`需要协程支持**：`co_yield`语法。
