# C++23新特性

## 一、概念说明

C++23（ISO/IEC 14882:2024）继续完善标准库和语言特性，主要新增：`std::expected<T, E>`、`std::print`/`std::println`、`std::flat_map`/`std::flat_set`、多维下标、`std::generator`等。

### 1.1 C++23核心特性

| 特性 | 说明 | 头文件 |
|------|------|--------|
| `std::expected` | 预期值或错误 | `<expected>` |
| `std::print` | 格式化输出 | `<print>` |
| `flat_map`/`flat_set` | 基于排序数组的容器 | `<flat_map>`/`<flat_set>` |
| 多维`operator[]` | `arr[1, 2, 3]` | 语言特性 |
| `std::generator` | 协程生成器 | `<generator>` |
| `std::to_underlying` | 枚举转底层类型 | `<utility>` |
| `auto(x)` | 显式衰变拷贝 | 语言特性 |

```cpp
#include <iostream>
#include <expected>
#include <string>
#include <cmath>

// expected: 表示可能失败的操作
std::expected<double, std::string> safeSqrt(double x) {
    if (x < 0) return std::unexpected("负数无实数平方根");
    return std::sqrt(x);
}

int main() {
    // expected
    auto r1 = safeSqrt(4.0);
    auto r2 = safeSqrt(-1.0);

    if (r1) std::cout << "sqrt(4) = " << *r1 << std::endl;
    if (!r2) std::cout << "错误: " << r2.error() << std::endl;

    // print (C++23)
    // std::print("Hello, {}!\n", "C++23");
    // std::println("值: {}", 42);

    return 0;
}
```

**输出：**
```
sqrt(4) = 2
错误: 负数无实数平方根
```

## 二、具体用法

### 2.1 std::expected

`std::expected`类似Rust的`Result`类型，提供类型安全的错误处理。

```cpp
#include <iostream>
#include <expected>
#include <string>

std::expected<int, std::string> parseInt(const std::string& s) {
    try {
        return std::stoi(s);
    } catch (...) {
        return std::unexpected("解析失败: " + s);
    }
}

int main() {
    auto results = {parseInt("42"), parseInt("abc"), parseInt("100")};

    for (const auto& r : results) {
        if (r) std::cout << "值: " << *r << std::endl;
        else std::cout << "错误: " << r.error() << std::endl;
    }

    // monadic操作
    auto doubled = parseInt("21").transform([](int x) { return x * 2; });
    if (doubled) std::cout << "翻倍: " << *doubled << std::endl;

    // and_then链式调用
    auto result = parseInt("42")
        .and_then([](int x) -> std::expected<int, std::string> {
            if (x > 0) return x * 2;
            return std::unexpected("必须为正数");
        });

    if (result) std::cout << "结果: " << *result << std::endl;

    return 0;
}
```

**输出：**
```
值: 42
错误: 解析失败: abc
值: 100
翻倍: 42
结果: 84
```

### 2.3 flat_map / flat_set

```cpp
#include <iostream>
// #include <flat_map>  // C++23
#include <vector>
#include <map>

int main() {
    // flat_map: 基于排序数组的map，缓存友好
    // std::flat_map<std::string, int> scores{
    //     {"Alice", 95}, {"Bob", 87}, {"Charlie", 92}
    // };
    // for (const auto& [name, score] : scores) {
    //     std::println("{}: {}", name, score);

        std::cout << name << ": " << score << std::endl;

    return 0;
}
```

### 2.4 其他特性

```cpp
#include <iostream>
#include <utility>
#include <cstdint>

enum class Color : uint8_t { Red, Green, Blue };

int main() {
    // std::to_underlying: 枚举转底层类型
    auto val = std::to_underlying(Color::Red);
    std::cout << "Color::Red = " << static_cast<int>(val) << std::endl;

    // auto(x): 显式衰变拷贝（C++23）
    const int ci = 42;
    auto copy = auto(ci); // int, 去const
    std::cout << "auto(ci) = " << copy << std::endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **编译器支持有限**：GCC 13+、Clang 17+、MSVC 2022 17.6+部分支持。
2. **`expected`类似Rust的`Result`**：类型安全的错误处理，替代异常或错误码。
3. **`print`不加`\n`不换行**：用`println`自动换行。
4. **`flat_map`比`std::map`缓存友好**：适合小量数据和频繁遍历场景。
5. **`std::generator`需要协程支持**：使用`co_yield`语法。
6. **`std::expected`的monadic操作**：`transform`、`and_then`、`or_else`支持函数式风格。
