# C++20 日期与时区

## 一、概念说明

C++20扩展了`<chrono>`库（C++20 §29.7），添加了日历日期、时区、格式化支持。从此C++可以原生处理日期计算和时区转换，不再依赖第三方库（如Howard Hinnant的date库）。

### 1.1 核心类型

| 类型 | 用途 |
|------|------|
| `year`, `month`, `day` | 日历组成部分 |
| `year_month_day` | 完整日期 |
| `sys_days` | 系统时钟天数 |
| `hh_mm_ss` | 时间（时:分:秒） |
| `zoned_time` | 带时区的时间 |
| `time_zone` | 时区表示 |

```cpp
#include <iostream>
#include <chrono>

using namespace std::chrono;

int main() {
    // 日历日期
    auto date = 2026y / May / 6;
    std::cout << "日期: " << date << std::endl;

    // 年-月-日
    year_month_day ymd{year{2026}, month{5}, day{6}};
    std::cout << "年=" << ymd.year() << " 月=" << ymd.month()
              << " 日=" << ymd.day() << std::endl;

    // 当前时间
    auto now = system_clock::now();
    auto today = floor<days>(now);
    year_month_day today_ymd{today};
    std::cout << "今天: " << today_ymd << std::endl;

    return 0;
}
```

**输出：**
```
日期: 2026-05-06
年=2026 月=May 日=06
今天: 2026-05-06
```

## 二、具体用法

### 2.1 日期运算

```cpp
#include <iostream>
#include <chrono>

using namespace std::chrono;

int main() {
    auto date = 2026y / May / 6;

    // 天数运算
    auto nextWeek = sys_days{date} + days{7};
    year_month_day next{nextWeek};
    std::cout << "一周后: " << next << std::endl;

    // 月运算
    auto nextMonth = date + months{1};
    std::cout << "一月后: " << nextMonth << std::endl;

    // 年运算
    auto nextYear = date + years{1};
    std::cout << "一年后: " << nextYear << std::endl;

    // 日期差值
    auto date2 = 2026y / December / 25;
    auto diff = sys_days{date2} - sys_days{date};
    std::cout << "到圣诞节还有: " << diff.count() << " 天" << std::endl;

    // 星期几
    weekday wd{sys_days{date}};
    std::cout << "星期: " << wd << std::endl;

    return 0;
}
```

**输出：**
```
一周后: 2026-05-13
一月后: 2026-06-06
一年后: 2027-05-06
到圣诞节还有: 233 天
星期: Wed
```

### 2.2 时区处理

```cpp
#include <iostream>
#include <chrono>

using namespace std::chrono;

int main() {
    // 带时区的时间
    zoned_time zt{"Asia/Shanghai", system_clock::now()};
    std::cout << "上海: " << zt << std::endl;

    // 转换时区
    zoned_time ny{"America/New_York", zt};
    std::cout << "纽约: " << ny << std::endl;

    zoned_time london{"Europe/London", zt};
    std::cout << "伦敦: " << london << std::endl;

    // 获取本地时间
    auto local = zt.get_local_time();
    std::cout << "本地时间点: " << local.time_since_epoch().count() << std::endl;

    return 0;
}
```

**输出：**
```
上海: 2026-05-06 10:30:00.000 CST
纽约: 2026-05-05 22:30:00.000 EDT
伦敦: 2026-05-06 03:30:00.000 BST
本地时间点: ...
```

### 2.3 日期格式化

```cpp
#include <iostream>
#include <chrono>
#include <format>

using namespace std::chrono;

int main() {
    auto date = 2026y / May / 6;
    auto now = system_clock::now();

    // 使用std::format格式化日期（C++20+）
    // 注意：需要编译器支持
    // std::cout << std::format("{:%Y-%m-%d}", date) << std::endl;
    // std::cout << std::format("{:%H:%M:%S}", now) << std::endl;

    // 日期字面量
    auto birthday = 1990y / January / 15;
    std::cout << "生日: " << birthday << std::endl;

    // 本月最后一天
    auto last = 2026y / May / last;
    std::cout << "五月最后一天: " << last << std::endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **需要时区数据库**：系统需包含IANA时区数据，否则时区功能不可用。
2. **编译器支持不完整**：GCC 13+、Clang 14+部分支持，MSVC支持较好。
3. **`hh_mm_ss`用于时间格式化**：配合`std::format`使用。
4. **`system_clock`是UTC**：本地时间需通过时区转换获得。
5. **日期字面量需`using namespace std::chrono`**：`2026y/May/6`。
6. **`last`关键字**：表示某月的最后一天，如`2026y/May/last`。
