# C++20 日期与时区

## 一、概念说明

C++20扩展了`<chrono>`库，添加了日历日期、时区、格式化支持。核心类型：
- `std::chrono::year/month/day`：日历日期
- `std::chrono::hh_mm_ss`：时间
- `std::chrono::zoned_time`：带时区的时间
- `std::chrono::time_zone`：时区表示

```cpp
#include <iostream>
#include <chrono>

using namespace std::chrono;

int main() {
    // 日历日期
    auto date = 2026y / May / 6;
    std::cout << "日期: " << date << std::endl;

    // 年-月-日
    year_month_day ymd{year{2026}, month{5}, day{6}};
    std::cout << "年=" << ymd.year() << " 月=" << ymd.month()
              << " 日=" << ymd.day() << std::endl;

    // 星期几
    auto weekday = sys_days{ymd};
    std::cout << "星期: " << weekday.time_since_epoch().count() << std::endl;

    // 当前时间
    auto now = system_clock::now();
    auto today = floor<days>(now);
    year_month_day today_ymd{today};
    std::cout << "今天: " << today_ymd << std::endl;

    return 0;
}
```

**输出：**
```
日期: 2026-05-06
年=2026 月=May 日=06
今天: 2026-05-06
```

## 二、具体用法

### 2.1 时区处理

```cpp
#include <iostream>
#include <chrono>

using namespace std::chrono;

int main() {
    // 带时区的时间
    zoned_time zt{"Asia/Shanghai", system_clock::now()};
    std::cout << "上海: " << zt << std::endl;

    // 转换时区
    zoned_time ny{"America/New_York", zt};
    std::cout << "纽约: " << ny << std::endl;

    // 日期运算
    auto date = 2026y / May / 6;
    auto nextWeek = sys_days{date} + days{7};
    year_month_day next{nextWeek};
    std::cout << "一周后: " << next << std::endl;

    // 年/月运算
    auto nextMonth = date + months{1};
    std::cout << "一月后: " << nextMonth << std::endl;

    return 0;
}
```

**输出：**
```
上海: 2026-05-06 10:30:00 CST
纽约: 2026-05-05 22:30:00 EDT
一周后: 2026-05-13
一月后: 2026-06-06
```

## 三、注意事项与常见陷阱

- **需要时区数据库**：系统需包含IANA时区数据。
- **编译器支持不完整**：GCC 13+、Clang 14+部分支持。
- **`hh_mm_ss`用于时间格式化**：配合`std::format`。
- **`system_clock`是UTC**：本地时间需通过时区转换。
- **日期字面量`2026y/May/6`需`using namespace std::chrono`**。
