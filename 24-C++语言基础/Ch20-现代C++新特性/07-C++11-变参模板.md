# C++11 变参模板

## 一、概念说明

变参模板（Variadic Templates，C++11 §14.5.3）支持可变数量的模板参数。使用参数包（parameter pack）`typename... Args`声明，通过递归展开或折叠表达式（C++17）处理。变参模板是`std::tuple`、`std::function`、`std::variant`等标准库组件的基础。

### 1.1 参数包操作

| 操作 | 语法 | 说明 |
|------|------|------|
| 声明 | `typename... Args` | 模板参数包 |
| 展开 | `Args...` | 函数参数包 |
| 大小 | `sizeof...(Args)` | 编译期常量 |
| 折叠 | `(args + ...)` | C++17折叠表达式 |

```cpp
#include <iostream>
#include <string>

// 递归终止条件
void print() {
    std::cout << std::endl;
}

// 递归展开
template <typename T, typename... Args>
void print(T first, Args... rest) {
    std::cout << first;
    if constexpr (sizeof...(rest) > 0) std::cout << ", ";
    print(rest...);
}

int main() {
    print(1, 2.5, "hello", 'A');
    print("单一参数");
    print();
    return 0;
}
```

**输出：**
```
1, 2.5, hello, A
单一参数

```

## 二、具体用法

### 2.1 折叠表达式（C++17）

C++17引入折叠表达式，彻底替代递归展开模式。

```cpp
#include <iostream>
#include <vector>
#include <sstream>

// 一元右折叠
template <typename... Args>
auto sum(Args... args) {
    return (args + ...); // (a + (b + (c + d)))
}

// 逗号折叠：副作用
template <typename... Args>
void printAll(Args... args) {
    ((std::cout << args << " "), ...);
    std::cout << std::endl;
}

// 二元折叠：带初始值
template <typename... Args>
auto sumWithInit(int init, Args... args) {
    return (init + ... + args);
}

// 容器批量操作
template <typename Container, typename... Values>
void addAll(Container& c, Values... values) {
    (c.push_back(values), ...);
}

int main() {
    std::cout << "sum(1,2,3,4,5)=" << sum(1, 2, 3, 4, 5) << std::endl;
    printAll("hello", 42, 3.14, true);
    std::cout << "sumWithInit(100,1,2,3)=" << sumWithInit(100, 1, 2, 3) << std::endl;

    std::vector<int> vec;
    addAll(vec, 1, 2, 3, 4, 5);
    for (int v : vec) std::cout << v << " ";
    std::cout << std::endl;

    return 0;
}
```

**输出：**
```
sum(1,2,3,4,5)=15
hello 42 3.14 1
sumWithInit(100,1,2,3)=106
1 2 3 4 5
```

### 2.2 tuple实现原理

```cpp
#include <iostream>
#include <tuple>

// 简化的tuple实现（递归继承）
template <typename... Ts>
struct SimpleTuple;

template <>
struct SimpleTuple<> {};

template <typename T, typename... Rest>
struct SimpleTuple<T, Rest...> : SimpleTuple<Rest...> {
    T value;
    SimpleTuple(T v, Rest... rest) : SimpleTuple<Rest...>(rest...), value(v) {}
};

// 递归访问
template <size_t I, typename... Ts>
auto get(const SimpleTuple<Ts...>& t) {
    if constexpr (I == 0) return t.value;
    else return get<I - 1>(static_cast<const SimpleTuple<Ts...>&>(t));
}

int main() {
    // 使用标准库tuple
    auto t = std::make_tuple(42, 3.14, "hello");
    std::cout << std::get<0>(t) << ", "
              << std::get<1>(t) << ", "
              << std::get<2>(t) << std::endl;

    // 结构化绑定（C++17）
    auto [a, b, c] = t;
    std::cout << a << ", " << b << ", " << c << std::endl;

    return 0;
}
```

**输出：**
```
42, 3.14, hello
42, 3.14, hello
```

### 2.3 完美转发变参

```cpp
#include <iostream>
#include <utility>
#include <memory>

template <typename T, typename... Args>
std::unique_ptr<T> make(Args&&... args) {
    return std::unique_ptr<T>(new T(std::forward<Args>(args)...));
}

struct Widget {
    int x, y;
    Widget(int x, int y) : x(x), y(y) {
        std::cout << "Widget(" << x << "," << y << ")" << std::endl;
    }
};

int main() {
    auto w = make<Widget>(10, 20);
    std::cout << "w=(" << w->x << "," << w->y << ")" << std::endl;
    return 0;
}
```

**输出：**
```
Widget(10,20)
w=(10,20)
```

## 三、注意事项与常见陷阱

1. **`sizeof...(args)`获取参数包大小**：编译期常量，可在`if constexpr`中使用。
2. **递归展开需要终止条件**：空参数包的重载版本。
3. **C++17折叠表达式更简洁**：强烈推荐替代递归展开。
4. **完美转发变参**：`Args&&... args` + `std::forward<Args>(args)...`。
5. **参数包不能直接遍历**：需要递归或折叠表达式展开。
6. **参数包展开顺序**：C++17保证从左到右，之前是未指定的。
