# C++11 变参模板

## 一、概念说明

变参模板（Variadic Templates）支持可变数量的模板参数。使用参数包（parameter pack）和递归展开或折叠表达式（C++17）处理。

```cpp
#include <iostream>
#include <string>

// 递归终止条件
void print() {
    std::cout << std::endl;
}

// 递归展开
template <typename T, typename... Args>
void print(T first, Args... rest) {
    std::cout << first;
    if constexpr (sizeof...(rest) > 0) std::cout << ", ";
    print(rest...);
}

int main() {
    print(1, 2.5, "hello", 'A');
    print("单一参数");
    print();
    return 0;
}
```

**输出：**
```
1, 2.5, hello, A
单一参数
```

## 二、具体用法

### 2.1 折叠表达式（C++17）

```cpp
#include <iostream>

// C++17折叠表达式替代递归
template <typename... Args>
auto sum(Args... args) {
    return (args + ...); // 一元右折叠
}

template <typename... Args>
void printAll(Args... args) {
    ((std::cout << args << " "), ...); // 逗号折叠
    std::cout << std::endl;
}

int main() {
    std::cout << "sum(1,2,3,4,5)=" << sum(1, 2, 3, 4, 5) << std::endl;
    printAll("hello", 42, 3.14, true);
    return 0;
}
```

**输出：**
```
sum(1,2,3,4,5)=15
hello 42 3.14 1
```

### 2.2 tuple实现原理

```cpp
#include <iostream>
#include <tuple>

// 简化的tuple实现
template <typename... Ts>
struct SimpleTuple;

template <>
struct SimpleTuple<> {};

template <typename T, typename... Rest>
struct SimpleTuple<T, Rest...> : SimpleTuple<Rest...> {
    T value;
    SimpleTuple(T v, Rest... rest) : SimpleTuple<Rest...>(rest...), value(v) {}
};

template <size_t I, typename... Ts>
auto get(const SimpleTuple<Ts...>& t) {
    if constexpr (I == 0) return t.value;
    else return get<I - 1>(static_cast<const SimpleTuple<Ts...>&>(t));
}

int main() {
    // 使用标准库tuple
    auto t = std::make_tuple(42, 3.14, "hello");
    std::cout << std::get<0>(t) << ", "
              << std::get<1>(t) << ", "
              << std::get<2>(t) << std::endl;

    // 结构化绑定（C++17）
    auto [a, b, c] = t;
    std::cout << a << ", " << b << ", " << c << std::endl;

    return 0;
}
```

**输出：**
```
42, 3.14, hello
42, 3.14, hello
```

## 三、注意事项与常见陷阱

- **`sizeof...(args)`获取参数包大小**：编译期常量。
- **递归展开需要终止条件**：空参数包的重载。
- **C++17折叠表达式更简洁**：推荐替代递归展开。
- **完美转发变参**：`Args&&... args` + `std::forward<Args>(args)...`。
- **参数包不能直接遍历**：需要递归或折叠表达式展开。
