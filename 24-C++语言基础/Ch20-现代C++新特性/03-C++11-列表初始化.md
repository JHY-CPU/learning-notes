# C++11 列表初始化

## 一、概念说明

C++11引入统一初始化语法（花括号`{}`），适用于所有类型：基本类型、聚合类型、STL容器、自定义类型。它消除了C++中初始化语法的不一致性。

```cpp
#include <iostream>
#include <vector>
#include <string>

struct Point {
    int x, y;
};

int main() {
    // 基本类型
    int a{42};
    double b{3.14};

    // 数组
    int arr[]{1, 2, 3, 4, 5};

    // STL容器
    std::vector<int> vec{1, 2, 3};
    std::string str{"Hello"};

    // 聚合类型
    Point p{10, 20};

    // 动态分配
    int* ptr = new int{99};

    std::cout << "a=" << a << ", b=" << b << std::endl;
    std::cout << "arr[0]=" << arr[0] << std::endl;
    std::cout << "vec[0]=" << vec[0] << ", str=" << str << std::endl;
    std::cout << "p=(" << p.x << "," << p.y << ")" << std::endl;
    std::cout << "*ptr=" << *ptr << std::endl;

    delete ptr;
    return 0;
}
```

**输出：**
```
a=42, b=3.14
arr[0]=1
vec[0]=1, str=Hello
p=(10,20)
*ptr=99
```

## 二、具体用法

### 2.1 窄化转换检查

```cpp
#include <iostream>

int main() {
    // 列表初始化禁止窄化转换
    int x = 42;
    // double d{x};  // 错误：int到double是窄化

    int a = 3.14;    // OK：截断为3
    // int b{3.14};   // 错误：禁止窄化

    // char c{300};   // 错误：超出char范围

    std::cout << "列表初始化防止窄化转换" << std::endl;
    return 0;
}
```

**输出：**
```
列表初始化防止窄化转换
```

### 2.2 initializer_list

```cpp
#include <iostream>
#include <initializer_list>
#include <vector>

class MyVector {
    std::vector<int> data;
public:
    MyVector(std::initializer_list<int> init) : data(init) {
        std::cout << "initializer_list构造, 大小=" << data.size() << std::endl;
    }

    void print() const {
        for (int x : data) std::cout << x << " ";
        std::cout << std::endl;
    }
};

int main() {
    MyVector v1{1, 2, 3, 4, 5};
    v1.print();

    MyVector v2 = {10, 20, 30};
    v2.print();

    // initializer_list的陷阱
    std::vector<std::string> vs1{3, "hi"}; // 3个"hi"
    std::vector<std::string> vs2(3, "hi"); // 也是3个"hi"
    std::cout << "vs1大小=" << vs1.size() << ", vs2大小=" << vs2.size() << std::endl;

    return 0;
}
```

**输出：**
```
initializer_list构造, 大小=5
1 2 3 4 5
initializer_list构造, 大小=3
10 20 30
vs1大小=3, vs2大小=3
```

## 三、注意事项与常见陷阱

- **`auto x{1}`在C++11中推导为`initializer_list<int>`**：C++17改为`int`。
- **`vector<int> v{10}`是1个元素10，不是10个元素**：用`vector<int>(10)`。
- **列表初始化禁止窄化**：提高类型安全性。
- **聚合初始化需要满足聚合条件**：无私有成员、无虚函数等。
- **`std::initializer_list`是轻量级代理**：不拥有所引用的元素。
