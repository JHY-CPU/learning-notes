# C++11 列表初始化

## 一、概念说明

C++11引入统一初始化语法（花括号`{}`），适用于所有类型：基本类型、聚合类型、STL容器、自定义类型（C++11 §8.5）。它消除了C++中初始化语法的不一致性，并提供窄化转换检查。

### 1.1 初始化方式对比

| 方式 | 语法 | 窄化检查 | 优先匹配 |
|------|------|---------|---------|
| 列表初始化 | `T obj{args}` | 是 | initializer_list > 构造函数 |
| 直接初始化 | `T obj(args)` | 否 | 构造函数 |
| 拷贝初始化 | `T obj = args` | 否 | 拷贝/移动构造 |

```cpp
#include <iostream>
#include <vector>
#include <string>

struct Point {
    int x, y;
};

int main() {
    // 基本类型
    int a{42};
    double b{3.14};

    // 数组
    int arr[]{1, 2, 3, 4, 5};

    // STL容器
    std::vector<int> vec{1, 2, 3};
    std::string str{"Hello"};

    // 聚合类型
    Point p{10, 20};

    // 动态分配
    int* ptr = new int{99};

    std::cout << "a=" << a << ", b=" << b << std::endl;
    std::cout << "vec[0]=" << vec[0] << ", str=" << str << std::endl;
    std::cout << "p=(" << p.x << "," << p.y << ")" << std::endl;
    std::cout << "*ptr=" << *ptr << std::endl;

    delete ptr;
    return 0;
}
```

**输出：**
```
a=42, b=3.14
vec[0]=1, str=Hello
p=(10,20)
*ptr=99
```

## 二、具体用法

### 2.1 窄化转换检查

列表初始化禁止可能丢失信息的转换，提高类型安全性。

```cpp
#include <iostream>

int main() {
    int x = 42;

    // 允许的初始化
    int a = x;          // OK：同类型
    int b{x};           // OK：同类型

    // 禁止的窄化转换
    // double d{x};     // 错误：int到double是窄化
    // int c{3.14};     // 错误：double到int是窄化
    // char c{300};     // 错误：超出char范围

    // 对比：非列表初始化允许窄化
    double e = x;       // OK（无窄化检查）
    int f = 3.14;       // OK但截断为3（无检查）

    std::cout << "列表初始化防止窄化转换" << std::endl;
    return 0;
}
```

### 2.2 initializer_list构造

当类有`initializer_list`构造函数时，花括号初始化优先匹配它。

```cpp
#include <iostream>
#include <initializer_list>
#include <vector>

class MyVector {
    std::vector<int> data;
public:
    MyVector(int size, int value) : data(size, value) {
        std::cout << "构造(size=" << size << ", value=" << value << ")" << std::endl;
    }

    MyVector(std::initializer_list<int> init) : data(init) {
        std::cout << "initializer_list构造, 大小=" << data.size() << std::endl;
    }

    void print() const {
        for (int x : data) std::cout << x << " ";
        std::cout << std::endl;
    }
};

int main() {
    MyVector v1{1, 2, 3, 4, 5};   // initializer_list
    v1.print();

    MyVector v2(3, 7);             // 构造函数：3个7
    v2.print();

    MyVector v3 = {10, 20, 30};   // initializer_list
    v3.print();

    // vector的陷阱
    std::vector<std::string> vs1{3, "hi"}; // initializer_list: 3个"hi"
    std::vector<std::string> vs2(3, "hi"); // 构造函数: 3个"hi"
    std::cout << "vs1大小=" << vs1.size() << ", vs2大小=" << vs2.size() << std::endl;

    return 0;
}
```

**输出：**
```
initializer_list构造, 大小=5
1 2 3 4 5
构造(size=3, value=7)
7 7 7
initializer_list构造, 大小=3
10 20 30
vs1大小=3, vs2大小=3
```

### 2.3 空初始化

```cpp
#include <iostream>
#include <vector>

int main() {
    // 值初始化（零初始化）
    int x{};           // x = 0
    double d{};        // d = 0.0
    int* p{};          // p = nullptr

    std::vector<int> v(5); // 5个0

    std::cout << "x=" << x << ", d=" << d << ", p=" << p << std::endl;
    return 0;
}
```

**输出：**
```
x=0, d=0, p=0
```

## 三、注意事项与常见陷阱

1. **`auto x{1}`在C++11中推导为`initializer_list<int>`**：C++17改为`int`，行为统一。
2. **`vector<int> v{10}`是1个元素值为10，不是10个元素**：用`vector<int>(10)`表示10个元素。
3. **列表初始化禁止窄化**：提高类型安全性，但可能需要显式转换。
4. **聚合初始化需要满足聚合条件**：无私有成员、无虚函数、无用户声明的构造函数（C++17放宽）。
5. **`std::initializer_list`是轻量级代理**：不拥有所引用的元素，只提供对编译期数组的访问。
6. **花括号初始化与`initializer_list`的优先级**：有initializer_list构造函数时优先匹配。
