# C++17 并行算法

## 一、概念说明

C++17为标准库算法添加了执行策略参数，支持顺序、并行和向量化执行。定义在`<execution>`头文件中。

```cpp
#include <iostream>
#include <algorithm>
#include <execution>
#include <vector>
#include <numeric>

int main() {
    std::vector<int> data(100000);
    std::iota(data.begin(), data.end(), 0);

    // 顺序执行
    auto sum1 = std::reduce(std::execution::seq, data.begin(), data.end(), 0LL);

    // 并行执行
    auto sum2 = std::reduce(std::execution::par, data.begin(), data.end(), 0LL);

    // 并行+向量化
    auto sum3 = std::reduce(std::execution::par_unseq, data.begin(), data.end(), 0LL);

    std::cout << "seq: " << sum1 << std::endl;
    std::cout << "par: " << sum2 << std::endl;
    std::cout << "par_unseq: " << sum3 << std::endl;

    return 0;
}
```

**输出：**
```
seq: 4999950000
par: 4999950000
par_unseq: 4999950000
```

## 二、具体用法

### 2.1 支持的算法

```cpp
#include <iostream>
#include <algorithm>
#include <execution>
#include <vector>

int main() {
    std::vector<int> v = {5, 3, 1, 4, 2, 8, 7, 6};

    // 并行排序
    std::sort(std::execution::par, v.begin(), v.end());

    // 并行for_each
    std::for_each(std::execution::par, v.begin(), v.end(),
                  [](int& x) { x *= 2; });

    // 并行查找
    auto it = std::find(std::execution::par, v.begin(), v.end(), 8);

    // 并行transform
    std::vector<int> result(v.size());
    std::transform(std::execution::par, v.begin(), v.end(), result.begin(),
                   [](int x) { return x * x; });

    std::cout << "并行算法完成" << std::endl;
    return 0;
}
```

**输出：**
```
并行算法完成
```

## 三、注意事项与常见陷阱

- **需要链接并行后端**：GCC需要TBB（`-ltbb`）或`libpstl`。
- **算法函数必须线程安全**：lambda不能有数据竞争。
- **数据量太小并行开销得不偿失**：通常需要数千元素以上。
- **`par_unseq`允许向量化**：函数不能有同步操作。
- **详细内容参见Ch19并行算法章节**。
