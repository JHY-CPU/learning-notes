# C++17 类模板推导 (CTAD)

## 一、概念说明

C++17引入类模板参数推导（CTAD），允许在构造类模板对象时不显式指定模板参数。编译器从构造函数参数自动推导。

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <mutex>
#include <pair>

int main() {
    // C++17之前：需要显式指定
    std::pair<int, double> p1(42, 3.14);
    std::vector<int> v1 = {1, 2, 3};
    std::mutex m1;
    std::lock_guard<std::mutex> lock1(m1);

    // C++17 CTAD：自动推导
    std::pair p2(42, 3.14);               // pair<int, double>
    std::vector v2 = {1, 2, 3};           // vector<int>
    std::lock_guard lock2(m1);            // lock_guard<mutex>

    std::cout << "p2: " << p2.first << ", " << p2.second << std::endl;
    std::cout << "v2大小: " << v2.size() << std::endl;

    return 0;
}
```

**输出：**
```
p2: 42, 3.14
v2大小: 3
```

## 二、具体用法

### 2.1 自定义推导指引

```cpp
#include <iostream>
#include <string>

template <typename T>
class Wrapper {
    T value;
public:
    Wrapper(T v) : value(std::move(v)) {}
    const T& get() const { return value; }
};

// 推导指引
template <typename T>
Wrapper(const char*) -> Wrapper<std::string>;

int main() {
    Wrapper w1(42);             // Wrapper<int>
    Wrapper w2(3.14);           // Wrapper<double>
    Wrapper w3("hello");        // Wrapper<std::string>（通过指引）

    std::cout << w1.get() << std::endl;
    std::cout << w2.get() << std::endl;
    std::cout << w3.get() << std::endl;

    return 0;
}
```

**输出：**
```
42
3.14
hello
```

## 三、注意事项与常见陷阱

- **推导指引需要与类模板在同一命名空间**。
- **聚合类型也可以推导**：`std::array a{1, 2, 3}` 推导为 `array<int, 3>`。
- **C++20允许从initializer_list推导**。
- **推导失败时编译器报错**：需要显式指定。
- **CTAD让代码更简洁但可能降低可读性**：复杂类型建议显式指定。
