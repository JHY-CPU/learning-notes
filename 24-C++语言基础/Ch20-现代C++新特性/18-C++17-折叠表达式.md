# C++17 折叠表达式

## 一、概念说明

折叠表达式（Fold Expression）简化变参模板中的参数包展开。四种形式：
- **一元右折叠**：`(pack op ...)` → `(a op (b op c))`
- **一元左折叠**：`(... op pack)` → `((a op b) op c)`
- **二元右折叠**：`(pack op ... op init)`
- **二元左折叠**：`(init op ... op pack)`

```cpp
#include <iostream>

// 一元右折叠：求和
template <typename... Args>
auto sum(Args... args) {
    return (args + ...); // (a + (b + (c + d)))
}

// 逗号折叠：打印所有参数
template <typename... Args>
void printAll(Args... args) {
    ((std::cout << args << " "), ...);
    std::cout << std::endl;
}

// 逻辑折叠
template <typename... Args>
bool allTrue(Args... args) {
    return (args && ...);
}

int main() {
    std::cout << sum(1, 2, 3, 4, 5) << std::endl;      // 15
    printAll("hello", 42, 3.14, true);                   // hello 42 3.14 1
    std::cout << allTrue(true, true, false) << std::endl; // 0

    return 0;
}
```

**输出：**
```
15
hello 42 3.14 1
0
```

## 二、具体用法

### 2.1 二元折叠（带初始值）

```cpp
#include <iostream>
#include <vector>

// 二元左折叠：带初始值
template <typename... Args>
auto sumWithInit(int init, Args... args) {
    return (init + ... + args); // init + a + b + c
}

// 容器操作
template <typename Container, typename... Values>
void addAll(Container& c, Values... values) {
    (c.push_back(values), ...); // 逐个push_back
}

int main() {
    std::cout << sumWithInit(100, 1, 2, 3) << std::endl; // 106

    std::vector<int> vec;
    addAll(vec, 1, 2, 3, 4, 5);
    for (int v : vec) std::cout << v << " ";
    std::cout << std::endl;

    return 0;
}
```

**输出：**
```
106
1 2 3 4 5
```

### 2.2 支持的运算符

| 运算符 | 示例 |
|--------|------|
| 算术 | `+`, `-`, `*`, `/`, `%` |
| 位运算 | `&`, `\|`, `^`, `<<`, `>>` |
| 比较 | `==`, `!=`, `<`, `>`, `<=`, `>=` |
| 逻辑 | `&&`, `\|\|` |
| 逗号 | `,` |
| 赋值 | `=`, `+=`, `-=`, `*=`等 |

## 三、注意事项与常见陷阱

- **空参数包的处理**：一元折叠需要运算符有单位元，或用二元折叠。
- **`&&`的单位元是`true`**：空包`(args && ...)`返回true。
- **`\|\|`的单位元是`false`**：空包`(args \|\| ...)`返回false。
- **`,`运算符折叠特别有用**：执行多个操作。
- **折叠表达式只在变参模板中有意义**。
