# C++17 折叠表达式

## 一、概念说明

折叠表达式（Fold Expression，C++17 §8.1.6）简化变参模板中的参数包展开。替代了C++11/14中复杂的递归展开模式，使变参模板代码更简洁、更易读。

### 1.1 四种折叠形式

| 形式 | 语法 | 求值顺序 | 示例（args=a,b,c） |
|------|------|---------|-------------------|
| 一元右折叠 | `(pack op ...)` | 右到左 | `(a op (b op c))` |
| 一元左折叠 | `(... op pack)` | 左到右 | `((a op b) op c)` |
| 二元右折叠 | `(pack op ... op init)` | 右到左 | `(a op (b op (c op init)))` |
| 二元左折叠 | `(init op ... op pack)` | 左到右 | `(((init op a) op b) op c)` |

```cpp
#include <iostream>

// 一元右折叠：求和
template <typename... Args>
auto sum(Args... args) {
    return (args + ...); // (a + (b + (c + d)))
}

// 逗号折叠：打印所有参数
template <typename... Args>
void printAll(Args... args) {
    ((std::cout << args << " "), ...);
    std::cout << std::endl;
}

// 逻辑折叠
template <typename... Args>
bool allTrue(Args... args) {
    return (args && ...);
}

int main() {
    std::cout << sum(1, 2, 3, 4, 5) << std::endl;      // 15
    printAll("hello", 42, 3.14, true);                   // hello 42 3.14 1
    std::cout << allTrue(true, true, false) << std::endl; // 0

    return 0;
}
```

**输出：**
```
15
hello 42 3.14 1
0
```

## 二、具体用法

### 2.1 二元折叠（带初始值）

```cpp
#include <iostream>
#include <vector>
#include <sstream>

// 二元左折叠：带初始值
template <typename... Args>
auto sumWithInit(int init, Args... args) {
    return (init + ... + args); // init + a + b + c
}

// 容器批量操作
template <typename Container, typename... Values>
void addAll(Container& c, Values... values) {
    (c.push_back(values), ...); // 逐个push_back
}

// 字符串拼接
template <typename... Args>
std::string concat(Args... args) {
    std::ostringstream oss;
    ((oss << args), ...);
    return oss.str();
}

int main() {
    std::cout << sumWithInit(100, 1, 2, 3) << std::endl; // 106

    std::vector<int> vec;
    addAll(vec, 1, 2, 3, 4, 5);
    for (int v : vec) std::cout << v << " ";
    std::cout << std::endl;

    std::cout << concat("Hello", ' ', "World", '!', 42) << std::endl;

    return 0;
}
```

**输出：**
```
106
1 2 3 4 5
Hello World!42
```

### 2.2 支持的运算符

折叠表达式支持以下运算符：

| 类别 | 运算符 |
|------|--------|
| 算术 | `+`, `-`, `*`, `/`, `%` |
| 位运算 | `&`, `|`, `^`, `<<`, `>>` |
| 比较 | `==`, `!=`, `<`, `>`, `<=`, `>=` |
| 逻辑 | `&&`, `||` |
| 逗号 | `,` |
| 赋值 | `=`, `+=`, `-=`, `*=`, `/=`, `%=`, `&=`, `|=`, `^=`, `<<=`, `>>=` |

### 2.3 实际应用

```cpp
#include <iostream>
#include <functional>

// 多重检查
template <typename T, typename... Predicates>
bool allMatch(const T& val, Predicates... preds) {
    return (preds(val) && ...);
}

// 链式调用
template <typename T, typename... Funcs>
auto applyAll(const T& val, Funcs... funcs) {
    return (funcs(val), ...);
}

// 参数验证
template <typename... Validators>
bool validate(int value, Validators... validators) {
    return (validators(value) && ...);
}

int main() {
    // 多重谓词检查
    int x = 42;
    bool result = allMatch(x,
        [](int v) { return v > 0; },
        [](int v) { return v < 100; },
        [](int v) { return v % 2 == 0; }
    );
    std::cout << "x满足所有条件: " << result << std::endl;

    // 参数验证
    bool valid = validate(25,
        [](int v) { return v >= 0; },
        [](int v) { return v <= 100; }
    );
    std::cout << "25有效: " << valid << std::endl;

    return 0;
}
```

**输出：**
```
x满足所有条件: 1
25有效: 1
```

## 三、注意事项与常见陷阱

1. **空参数包的处理**：一元折叠需要运算符有单位元，或用二元折叠提供初始值。
2. **`&&`的单位元是`true`**：空包`(args && ...)`返回`true`。
3. **`||`的单位元是`false`**：空包`(args || ...)`返回`false`。
4. **`,`运算符折叠特别有用**：执行多个副作用操作。
5. **`+`折叠空包是未定义行为**：使用二元折叠或确保至少有一个参数。
6. **折叠表达式只在变参模板中有意义**：不能用于普通函数。
