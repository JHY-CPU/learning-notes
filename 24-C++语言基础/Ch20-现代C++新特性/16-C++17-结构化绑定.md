# C++17 结构化绑定

## 一、概念说明

结构化绑定（Structured Binding，C++17 §11.5）允许从结构体、数组、`tuple`/`pair`中一次性解构出多个变量。语法：`auto [a, b, c] = expression;`。这大幅简化了多返回值的使用。

### 1.1 支持的类型

| 类型 | 示例 |
|------|------|
| 结构体/类 | `struct Point{int x,y}; auto [x,y] = p;` |
| 数组 | `int arr[3]; auto [a,b,c] = arr;` |
| `std::pair` | `auto [k,v] = *map.begin();` |
| `std::tuple` | `auto [a,b,c] = make_tuple(1,2.0,"3");` |

```cpp
#include <iostream>
#include <tuple>
#include <map>

struct Point { double x, y; };

int main() {
    // 从pair解构
    std::pair<int, std::string> p{42, "hello"};
    auto [id, name] = p;
    std::cout << "id=" << id << ", name=" << name << std::endl;

    // 从tuple解构
    auto [a, b, c] = std::make_tuple(1, 2.5, "three");
    std::cout << "a=" << a << ", b=" << b << ", c=" << c << std::endl;

    // 从结构体解构
    Point pt{3.0, 4.0};
    auto [x, y] = pt;
    std::cout << "x=" << x << ", y=" << y << std::endl;

    // 从数组解构
    int arr[]{10, 20, 30};
    auto [d, e, f] = arr;
    std::cout << "d=" << d << ", e=" << e << std::endl;

    // 在循环中使用（遍历map）
    std::map<std::string, int> scores{{"Alice", 95}, {"Bob", 87}};
    for (const auto& [key, value] : scores) {
        std::cout << key << ": " << value << std::endl;
    }

    return 0;
}
```

**输出：**
```
id=42, name=hello
a=1, b=2.5, c=three
x=3, y=4
d=10, e=20
Alice: 95
Bob: 87
```

## 二、具体用法

### 2.1 返回多个值

```cpp
#include <iostream>
#include <tuple>
#include <optional>
#include <string>

// 返回tuple（C++11风格）
std::tuple<bool, std::string, int> fetchData(int id) {
    if (id > 0) return {true, "data_" + std::to_string(id), 200};
    return {false, "", 404};
}

// 结构化返回
struct Result {
    bool success;
    std::string data;
    int code;
};

Result process(int id) {
    if (id > 0) return {true, "processed", 200};
    return {false, "invalid", 400};
}

int main() {
    // 结构化绑定tuple
    auto [success, data, code] = fetchData(42);
    if (success) {
        std::cout << "成功: " << data << " (code=" << code << ")" << std::endl;
    }

    // 结构化绑定结构体
    auto [ok, result, status] = process(1);
    std::cout << "处理: " << result << " status=" << status << std::endl;

    // 带引用的绑定（修改原tuple）
    std::tuple<int, double> t{10, 3.14};
    auto& [ri, rd] = t;
    ri = 100;
    std::cout << "修改后: " << std::get<0>(t) << std::endl;

    return 0;
}
```

**输出：**
```
成功: data_42 (code=200)
处理: processed status=200
修改后: 100
```

### 2.2 遍历关联容器

```cpp
#include <iostream>
#include <map>
#include <unordered_map>
#include <set>

int main() {
    // map遍历
    std::map<std::string, int> wordCount{
        {"hello", 5}, {"world", 3}, {"cpp", 10}
    };

    for (const auto& [word, count] : wordCount) {
        std::cout << word << ": " << count << std::endl;
    }

    // set遍历（set元素是const的）
    std::set<std::pair<int, std::string>> items{
        {1, "first"}, {2, "second"}, {3, "third"}
    };

    for (const auto& [id, name] : items) {
        std::cout << id << " -> " << name << std::endl;
    }

    return 0;
}
```

### 2.3 if/switch初始化（C++17）

```cpp
#include <iostream>
#include <map>
#include <string>

int main() {
    std::map<int, std::string> m{{1, "one"}, {2, "two"}};

    // if语句中的初始化器和结构化绑定
    if (auto [it, inserted] = m.insert({3, "three"}); inserted) {
        std::cout << "插入成功: " << it->second << std::endl;
    } else {
        std::cout << "已存在: " << it->second << std::endl;
    }

    // 错误码处理
    auto [code, msg] = std::make_tuple(404, std::string("Not Found"));
    if (code >= 400) {
        std::cout << "错误: " << code << " " << msg << std::endl;
    }

    return 0;
}
```

**输出：**
```
插入成功: three
错误: 404 Not Found
```

## 三、注意事项与常见陷阱

1. **绑定数量必须匹配**：`auto [a, b] = tuple<int,int,int>`不合法。
2. **绑定的变量是副本或引用**：`auto`是副本，`auto&`是引用，`const auto&`是常引用。
3. **不能绑定到private成员**：除非提供`get<I>`函数或作为友元。
4. **`const`和`volatile`修饰符有效**：`const auto& [a,b] = ...`是常见模式。
5. **绑定数组需要C++17**：`int arr[3]; auto [a,b,c] = arr;`。
6. **绑定的变量名是新变量**：不是原成员的别名，但通过引用绑定可以修改原值。
