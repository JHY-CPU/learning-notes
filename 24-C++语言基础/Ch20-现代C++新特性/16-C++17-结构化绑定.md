# C++17 结构化绑定

## 一、概念说明

结构化绑定（Structured Binding）允许从结构体、数组、`tuple`/`pair`中一次性解构出多个变量。语法：`auto [a, b, c] = expression;`

```cpp
#include <iostream>
#include <tuple>
#include <map>

struct Point { double x, y; };

int main() {
    // 从pair解构
    std::pair<int, std::string> p{42, "hello"};
    auto [id, name] = p;
    std::cout << "id=" << id << ", name=" << name << std::endl;

    // 从tuple解构
    auto [a, b, c] = std::make_tuple(1, 2.5, "three");
    std::cout << "a=" << a << ", b=" << b << ", c=" << c << std::endl;

    // 从结构体解构
    Point pt{3.0, 4.0};
    auto [x, y] = pt;
    std::cout << "x=" << x << ", y=" << y << std::endl;

    // 从数组解构
    int arr[]{10, 20, 30};
    auto [d, e, f] = arr;
    std::cout << "d=" << d << ", e=" << e << std::endl;

    // 在循环中使用（遍历map）
    std::map<std::string, int> scores{{"Alice", 95}, {"Bob", 87}};
    for (const auto& [key, value] : scores) {
        std::cout << key << ": " << value << std::endl;
    }

    return 0;
}
```

**输出：**
```
id=42, name=hello
a=1, b=2.5, c=three
x=3, y=4
d=10, e=20
Alice: 95
Bob: 87
```

## 二、具体用法

### 2.1 返回多个值

```cpp
#include <iostream>
#include <tuple>
#include <optional>

// 返回tuple（C++11风格）
std::tuple<bool, std::string, int> fetchData(int id) {
    if (id > 0) return {true, "data_" + std::to_string(id), 200};
    return {false, "", 404};
}

int main() {
    // C++17结构化绑定
    auto [success, data, code] = fetchData(42);
    if (success) {
        std::cout << "成功: " << data << " (code=" << code << ")" << std::endl;
    }

    // 带引用的绑定
    std::tuple<int, double> t{10, 3.14};
    auto& [ri, rd] = t;
    ri = 100; // 修改原tuple
    std::cout << "修改后: " << std::get<0>(t) << std::endl;

    return 0;
}
```

**输出：**
```
成功: data_42 (code=200)
修改后: 100
```

## 三、注意事项与常见陷阱

- **绑定数量必须匹配**：`auto [a, b] = tuple<int,int,int>`不合法。
- **绑定的变量是副本或引用**：`auto`是副本，`auto&`是引用。
- **不能绑定到private成员**：除非提供`get<I>`函数。
- **`const`和`volatile`修饰符有效**：`const auto& [a,b] = ...`。
- **绑定数组需要C++17**：`int arr[3]; auto [a,b,c] = arr;`。
