# C++14新特性

## 一、概念说明

C++14（ISO/IEC 14882:2014）是C++11的"完善版"，没有革命性变化，但填补了C++11的诸多空白。核心改进：泛型lambda、变量模板、返回类型推导、`make_unique`、`constexpr`增强等。

### 1.1 C++14特性一览

| 特性 | 说明 | 示例 |
|------|------|------|
| 泛型lambda | `auto`参数 | `[](auto x){ return x; }` |
| 变量模板 | `template<T> constexpr T pi` | `pi<double>` |
| 返回类型推导 | 函数自动推导返回类型 | `auto f(){ return 42; }` |
| `make_unique` | 安全创建unique_ptr | `make_unique<int>(42)` |
| 二进制字面量 | `0b1010` | `int x = 0b1111;` |
| 数字分隔符 | 提高可读性 | `1'000'000` |
| `deprecated`属性 | 标记弃用 | `[[deprecated]]` |

```cpp
#include <iostream>
#include <memory>
#include <vector>
#include <algorithm>
#include <string>

int main() {
    // 泛型lambda
    auto generic = [](auto a, auto b) { return a + b; };
    std::cout << generic(1, 2) << std::endl;
    std::cout << generic(1.5, 2.5) << std::endl;
    std::cout << generic(std::string("Hello "), std::string("World")) << std::endl;

    // make_unique（C++14新增）
    auto ptr = std::make_unique<int>(42);

    // 返回类型推导
    auto lambda = [](int x) { return x * 2; };

    // 二进制字面量
    int binary = 0b1010;
    std::cout << "0b1010 = " << binary << std::endl;

    // 数字分隔符
    long big = 1'000'000'000;
    std::cout << "1'000'000'000 = " << big << std::endl;

    return 0;
}
```

**输出：**
```
3
4
Hello World
42
0b1010 = 10
1'000'000'000 = 1000000000
```

## 二、具体用法

### 2.1 变量模板

```cpp
#include <iostream>
#include <cmath>

// 变量模板
template <typename T>
constexpr T pi = T(3.14159265358979323846);

template <typename T>
constexpr T e = T(2.71828182845904523536);

// 物理常量
template <typename T>
constexpr T speed_of_light = T(299792458); // m/s

int main() {
    std::cout << "pi<float> = " << pi<float> << std::endl;
    std::cout << "pi<double> = " << pi<double> << std::endl;
    std::cout << "e<double> = " << e<double> << std::endl;

    // 用变量模板计算
    auto area = pi<double> * 5.0 * 5.0;
    std::cout << "半径5的圆面积: " << area << std::endl;

    return 0;
}
```

**输出：**
```
pi<float> = 3.14159
pi<double> = 3.14159
e<double> = 2.71828
半径5的圆面积: 78.5398
```

### 2.2 泛型lambda应用

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

int main() {
    // 通用打印
    auto print = [](const auto& container) {
        for (const auto& item : container)
            std::cout << item << " ";
        std::cout << std::endl;
    };

    std::vector<int> vi = {1, 2, 3};
    std::vector<std::string> vs = {"a", "b", "c"};
    print(vi);
    print(vs);

    // 通用排序
    auto sortDesc = [](auto& container) {
        std::sort(container.begin(), container.end(), std::greater<>{});
    };

    sortDesc(vi);
    std::cout << "降序: ";
    print(vi);

    // 通用容器求和
    auto sum = [](const auto& container) {
        typename std::decay_t<decltype(container)>::value_type total{};
        for (const auto& item : container) total += item;
        return total;
    };

    std::vector<double> vd = {1.1, 2.2, 3.3};
    std::cout << "sum(vd) = " << sum(vd) << std::endl;

    return 0;
}
```

**输出：**
```
1 2 3
a b c
降序: 3 2 1
sum(vd) = 6.6
```

### 2.3 返回类型推导

C++14允许普通函数（非递归）自动推导返回类型。

```cpp
#include <iostream>

// C++14: 返回类型推导
auto add(int a, int b) {
    return a + b;  // 推导为int
}

auto divide(double a, double b) {
    return a / b;  // 推导为double
}

// decltype(auto)保留引用性
int& getRef(int* p) {
    return *p;
}

int main() {
    auto x = add(3, 4);
    auto y = divide(10.0, 3.0);

    std::cout << "add(3,4)=" << x << std::endl;
    std::cout << "divide(10,3)=" << y << std::endl;

    int val = 42;
    decltype(auto) ref = getRef(&val);
    ref = 100;
    std::cout << "val=" << val << std::endl;

    return 0;
}
```

**输出：**
```
add(3,4)=7
divide(10,3)=3.33333
val=100
```

## 三、注意事项与常见陷阱

1. **泛型lambda不能直接使用`decltype`推导返回类型**（C++14有限制，C++17改善）。
2. **变量模板的特化和偏特化**：和类模板规则类似。
3. **`make_unique`是C++14特性**：C++11需手写或使用Boost。
4. **二进制字面量和数字分隔符只是语法糖**：不影响运行时行为。
5. **`[[deprecated]]`属性**：标记弃用的API，编译器产生警告。
6. **递归函数不能使用返回类型推导**：需要至少一个非递归调用先确定类型。
