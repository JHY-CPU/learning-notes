# C++14新特性

## 一、概念说明

C++14是C++11的"完善版"，主要改进：泛型lambda、变量模板、返回类型推导、`make_unique`、`constexpr`增强等。

```cpp
#include <iostream>
#include <memory>
#include <vector>
#include <algorithm>

int main() {
    // 泛型lambda
    auto generic = [](auto a, auto b) { return a + b; };
    std::cout << generic(1, 2) << std::endl;
    std::cout << generic(1.5, 2.5) << std::endl;
    std::cout << generic(std::string("Hello "), std::string("World")) << std::endl;

    // make_unique（C++14新增）
    auto ptr = std::make_unique<int>(42);

    // 返回类型推导
    auto lambda = [](int x) { return x * 2; }; // 自动推导返回int

    // 二进制字面量
    int binary = 0b1010;
    std::cout << "0b1010 = " << binary << std::endl;

    // 数字分隔符
    long big = 1'000'000'000;
    std::cout << "1'000'000'000 = " << big << std::endl;

    return 0;
}
```

**输出：**
```
3
4
Hello World
42
0b1010 = 10
1'000'000'000 = 1000000000
```

## 二、具体用法

### 2.1 变量模板

```cpp
#include <iostream>
#include <cmath>

// C++14: 变量模板
template <typename T>
constexpr T pi = T(3.14159265358979323846);

template <typename T>
constexpr T e = T(2.71828182845904523536);

int main() {
    std::cout << "pi<float> = " << pi<float> << std::endl;
    std::cout << "pi<double> = " << pi<double> << std::endl;
    std::cout << "e<double> = " << e<double> << std::endl;

    // 用变量模板计算圆面积
    auto area = pi<double> * 5.0 * 5.0;
    std::cout << "半径5的圆面积: " << area << std::endl;

    return 0;
}
```

**输出：**
```
pi<float> = 3.14159
pi<double> = 3.14159
e<double> = 2.71828
半径5的圆面积: 78.5398
```

### 2.2 generic lambda应用

```cpp
#include <iostream>
#include <vector>
#include <string>

int main() {
    // 通用打印
    auto print = [](const auto& container) {
        for (const auto& item : container)
            std::cout << item << " ";
        std::cout << std::endl;
    };

    std::vector<int> vi = {1, 2, 3};
    std::vector<std::string> vs = {"a", "b", "c"};
    print(vi);
    print(vs);

    // 通用排序
    auto sortDesc = [](auto& container) {
        std::sort(container.begin(), container.end(), std::greater<>{});
    };

    sortDesc(vi);
    std::cout << "降序: ";
    print(vi);

    return 0;
}
```

**输出：**
```
1 2 3
a b c
降序: 3 2 1
```

## 三、注意事项与常见陷阱

- **泛型lambda不能使用`auto`返回类型+`decltype`推导**（C++14有限制）。
- **变量模板的特化和偏特化**：和类模板类似。
- **`make_unique`是C++14特性**：C++11需手写。
- **二进制字面量和数字分隔符只是语法糖**：不影响运行时。
- **`deprecated`属性**：`[[deprecated("use X instead")]]`标记弃用。
