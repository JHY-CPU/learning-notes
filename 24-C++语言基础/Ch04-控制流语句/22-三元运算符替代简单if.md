# 三元运算符替代简单if

## 一、概念说明

对于简单的二选一场景，三元运算符`? :`可以替代`if-else`，使代码更紧凑。但不应过度使用。

## 二、具体用法

### 2.1 适合用三元的场景

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 简单赋值
    int x = 10;
    string type = (x > 0) ? "正数" : "非正数";
    cout << type << endl;

    // 函数返回值
    auto absValue = [](int n) {
        return (n >= 0) ? n : -n;
    };
    cout << "|-5| = " << absValue(-5) << endl;

    // const初始化（if无法做到）
    int y = 42;
    const int limit = (y > 100) ? 200 : 50;
    cout << "limit = " << limit << endl;

    // 输出中的条件
    cout << "状态: " << (x > 5 ? "正常" : "异常") << endl;

    return 0;
}
```

输出：
```
正数
|-5| = 5
limit = 50
状态: 正常
```

### 2.2 不适合用三元的场景

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 85;

    // 不适合：复杂逻辑用if-else
    // 嵌套三元可读性差
    string grade = (score >= 90) ? "A"
                 : (score >= 80) ? "B"
                 : (score >= 70) ? "C"
                 : (score >= 60) ? "D" : "F";
    // 虽然能写，但if-else更清晰

    // 适合的写法
    string grade2;
    if (score >= 90) grade2 = "A";
    else if (score >= 80) grade2 = "B";
    else if (score >= 70) grade2 = "C";
    else if (score >= 60) grade2 = "D";
    else grade2 = "F";

    cout << "成绩: " << grade << " / " << grade2 << endl;

    // 不适合：有副作用的操作
    // if (x > 0) { process(x); log(x); } 更清晰
    // 而非 (x > 0) ? (process(x), log(x)) : void();

    return 0;
}
```

输出：
```
成绩: B / B
```

## 三、注意事项与常见陷阱

1. **简单场景用三元**：一个值的二选一适合三元
2. **复杂逻辑用if**：有副作用、多条语句用if-else
3. **类型必须兼容**：三元的两个分支需要有公共类型
4. **可读性第一**：如果三元使代码更难理解，改用if
5. **const初始化**：三元可以在const变量声明时使用，if做不到
