# 结构化绑定 C++17

## 一、概念说明

结构化绑定（Structured Binding）允许用`auto [a, b, c]`语法将聚合类型（结构体、数组、tuple等）的成员解包为独立变量。

## 二、具体用法

### 2.1 基本用法

```cpp
#include <iostream>
#include <tuple>
#include <map>
#include <string>
using namespace std;

pair<string, int> getPerson() {
    return {"Alice", 25};
}

int main() {
    // 解包pair
    auto [name, age] = getPerson();
    cout << name << ", " << age << endl;

    // 解包数组
    int arr[] = {1, 2, 3};
    auto [a, b, c] = arr;
    cout << a << ", " << b << ", " << c << endl;

    // 解包结构体
    struct Point { double x, y; };
    Point p{3.0, 4.0};
    auto [x, y] = p;
    cout << "(" << x << ", " << y << ")" << endl;

    // 解包tuple
    auto [status, code, msg] = make_tuple(true, 200, "OK");
    cout << status << ", " << code << ", " << msg << endl;

    return 0;
}
```

输出：
```
Alice, 25
1, 2, 3
(3, 4)
1, 200, OK
```

### 2.2 在循环中使用

```cpp
#include <iostream>
#include <map>
#include <string>
#include <vector>
using namespace std;

int main() {
    // 遍历map
    map<string, int> scores = {{"Alice", 95}, {"Bob", 87}, {"Charlie", 92}};

    for (const auto& [name, score] : scores) {
        cout << name << ": " << score << endl;
    }

    // 遍历vector of pairs
    vector<pair<int, string>> items = {{1, "one"}, {2, "two"}, {3, "three"}};
    for (const auto& [num, word] : items) {
        cout << num << " -> " << word << endl;
    }

    // 配合if初始化语句
    if (auto [it, inserted] = scores.insert({"David", 88}); inserted) {
        cout << "插入成功: " << it->first << endl;
    } else {
        cout << "已存在: " << it->first << endl;
    }

    return 0;
}
```

输出：
```
Alice: 95
Bob: 87
Charlie: 92
1 -> one
2 -> two
3 -> three
插入成功: David
```

## 三、注意事项与常见陷阱

1. **绑定的是副本还是引用**：`auto`是副本，`auto&`是引用，`const auto&`是只读引用
2. **成员数量必须匹配**：`auto [a, b] = triplet`是编译错误
3. **绑定到引用可以修改原对象**：`auto& [x, y] = p; x = 100;`
4. **只适用于公开成员**：私有成员不能直接结构化绑定
5. **C++17特性**：需要`-std=c++17`
