# 提前返回模式

## 一、概念说明

提前返回（Early Return / Guard Clause）通过在函数开头检查异常条件并提前返回，避免深层嵌套的if-else结构，使代码更扁平、更易读。

## 二、具体用法

### 2.1 深层嵌套 vs 提前返回

```cpp
#include <iostream>
#include <string>
using namespace std;

// 传统方式：深层嵌套（箭头代码）
string processTraditional(int* ptr, int size) {
    if (ptr != nullptr) {
        if (size > 0) {
            if (size <= 100) {
                if (*ptr > 0) {
                    return "处理成功";
                } else {
                    return "值必须为正";
                }
            } else {
                return "大小超出限制";
            }
        } else {
            return "大小必须为正";
        }
    } else {
        return "指针为空";
    }
}

// 提前返回方式：扁平结构
string processEarlyReturn(int* ptr, int size) {
    if (ptr == nullptr) return "指针为空";
    if (size <= 0) return "大小必须为正";
    if (size > 100) return "大小超出限制";
    if (*ptr <= 0) return "值必须为正";

    // 主逻辑：没有嵌套
    return "处理成功";
}

int main() {
    int val = 42;
    cout << processEarlyReturn(&val, 10) << endl;
    cout << processEarlyReturn(nullptr, 10) << endl;
    cout << processEarlyReturn(&val, -1) << endl;
    cout << processEarlyReturn(&val, 200) << endl;

    return 0;
}
```

输出：
```
处理成功
指针为空
大小必须为正
大小超出限制
```

### 2.2 实际应用

```cpp
#include <iostream>
#include <vector>
#include <optional>
using namespace std;

optional<int> findMax(const vector<int>& data) {
    // Guard clause
    if (data.empty()) return nullopt;

    // 主逻辑
    int maxVal = data[0];
    for (size_t i = 1; i < data.size(); i++) {
        if (data[i] > maxVal) maxVal = data[i];
    }
    return maxVal;
}

void processUser(const string& name, int age) {
    if (name.empty()) {
        cout << "错误: 名字不能为空" << endl;
        return;
    }
    if (age < 0 || age > 150) {
        cout << "错误: 年龄无效" << endl;
        return;
    }

    // 正常处理
    cout << "处理用户: " << name << ", " << age << "岁" << endl;
}

int main() {
    processUser("Alice", 25);
    processUser("", 25);
    processUser("Bob", -1);

    vector<int> data = {3, 1, 4, 1, 5, 9};
    if (auto max = findMax(data)) {
        cout << "最大值: " << *max << endl;
    }

    return 0;
}
```

输出：
```
处理用户: Alice, 25岁
错误: 名字不能为空
错误: 年龄无效
最大值: 9
```

## 三、注意事项与常见陷阱

1. **先处理异常情况**：函数开头集中处理错误/边界条件
2. **减少嵌套层级**：目标是函数主体不缩进或只缩进一层
3. **资源清理**：提前返回前确保资源已释放，或使用RAII
4. **单一退出点争论**：现代C++不强制单一出口，多个return更清晰
5. **适用于条件检查多的函数**：如果只有一个条件，普通if-else即可
