# 循环中的常见错误

## 一、概念说明

循环是bug的高发区域。常见的错误包括差一错误、死循环、越界访问和迭代器失效等。

## 二、具体用法

### 2.1 差一错误

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> data = {1, 2, 3, 4, 5};

    // 错误：<=导致越界
    // for (int i = 0; i <= data.size(); i++) {
    //     cout << data[i] << " ";  // data[5]越界！
    // }

    // 正确：使用 <
    for (size_t i = 0; i < data.size(); i++) {
        cout << data[i] << " ";
    }
    cout << endl;

    // 差一错误：n个元素应该迭代n次
    int n = 5;
    int count = 0;
    for (int i = 1; i <= n; i++) {  // 从1到n，共n次
        count++;
    }
    cout << "迭代次数: " << count << endl;  // 5

    return 0;
}
```

输出：
```
1 2 3 4 5
迭代次数: 5
```

### 2.2 死循环

```cpp
#include <iostream>
using namespace std;

int main() {
    // 错误1：条件永远为真
    int i = 0;
    // while (i < 10) { /* 忘记 i++ */ }

    // 错误2：有符号/无符号比较
    // for (int i = 0; i < vec.size(); i++)
    // 如果size()返回unsigned，i转为unsigned可能出问题

    // 错误3：continue跳过迭代更新
    int j = 0;
    while (j < 5) {
        if (j == 2) {
            j++;
            continue;  // 必须在continue前更新j
        }
        cout << j << " ";
        j++;
    }
    cout << endl;

    return 0;
}
```

输出：
```
0 1 3 4
```

### 2.3 迭代器失效

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> nums = {1, 2, 3, 4, 5};

    // 错误：遍历时删除导致迭代器失效
    // for (auto it = nums.begin(); it != nums.end(); ++it) {
    //     if (*it == 3) {
    //         nums.erase(it);  // it失效！
    //     }
    // }

    // 正确方式
    for (auto it = nums.begin(); it != nums.end(); ) {
        if (*it == 3) {
            it = nums.erase(it);  // erase返回下一个有效迭代器
        } else {
            ++it;
        }
    }

    // C++20: std::erase
    // std::erase(nums, 3);

    for (int n : nums) cout << n << " ";
    cout << endl;

    return 0;
}
```

输出：
```
1 2 4 5
```

## 三、注意事项与常见陷阱

1. **<= vs <**：`i <= n`迭代n+1次，`i < n`迭代n次
2. **有符号/无符号混合**：`int i`与`size_t`比较可能出问题
3. **while中continue跳过更新**：确保迭代变量在continue前已更新
4. **迭代器失效**：修改容器时迭代器可能失效
5. **空容器**：循环前检查容器是否为空
