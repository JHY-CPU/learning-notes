# 算法中的循环模式

## 一、概念说明

算法编程中常见的循环模式包括遍历、搜索、累加和变换。掌握这些模式是解决算法问题的基础。

## 二、具体用法

### 2.1 遍历模式

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> data = {5, 3, 8, 1, 9, 2, 7};

    // 正向遍历
    for (int x : data) cout << x << " ";
    cout << endl;

    // 反向遍历
    for (auto it = data.rbegin(); it != data.rend(); ++it) {
        cout << *it << " ";
    }
    cout << endl;

    // 带索引遍历
    for (size_t i = 0; i < data.size(); i++) {
        cout << "[" << i << "]=" << data[i] << " ";
    }
    cout << endl;

    // 双指针遍历（对撞指针）
    int left = 0, right = data.size() - 1;
    while (left < right) {
        cout << data[left] << "," << data[right] << " ";
        left++;
        right--;
    }
    cout << endl;

    return 0;
}
```

输出：
```
5 3 8 1 9 2 7
7 2 9 1 8 3 5
[0]=5 [1]=3 [2]=8 [3]=1 [4]=9 [5]=2 [6]=7
5,7 3,2 8,9
```

### 2.2 搜索与累加模式

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> data = {5, 3, 8, 1, 9, 2, 7};

    // 线性搜索
    int target = 8;
    int foundIdx = -1;
    for (size_t i = 0; i < data.size(); i++) {
        if (data[i] == target) {
            foundIdx = i;
            break;
        }
    }
    cout << "找到" << target << "在索引" << foundIdx << endl;

    // 累加/累乘
    int sum = 0;
    for (int x : data) sum += x;
    cout << "总和: " << sum << endl;

    // 条件累加
    int evenSum = 0;
    for (int x : data) {
        if (x % 2 == 0) evenSum += x;
    }
    cout << "偶数和: " << evenSum << endl;

    // 变换
    vector<int> squared;
    for (int x : data) squared.push_back(x * x);
    for (int x : squared) cout << x << " ";
    cout << endl;

    // STL算法（推荐）
    int stlSum = 0;
    for_each(data.begin(), data.end(), [&stlSum](int x) { stlSum += x; });
    cout << "STL总和: " << stlSum << endl;

    return 0;
}
```

输出：
```
找到8在索引2
总和: 35
偶数和: 10
25 9 64 1 81 4 49
STL总和: 35
```

## 三、注意事项与常见陷阱

1. **选择合适的模式**：搜索用break，变换用新容器，累加用accumulator
2. **STL算法优先**：`std::find`、`std::accumulate`、`std::transform`等更安全
3. **双指针适合有序数据**：无序数据的双指针需要排序或其他预处理
4. **边界条件**：空容器、单元素容器要特殊处理
5. **时间复杂度意识**：嵌套循环是O(n²)，哈希表查找是O(1)
