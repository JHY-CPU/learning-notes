# 控制流概述

## 一、概念说明

控制流语句决定程序执行的**路径**。C++有三种基本控制结构：**顺序结构**（按顺序执行）、**分支结构**（条件选择）和**循环结构**（重复执行）。所有复杂的程序逻辑都可以用这三种结构组合实现。

## 二、具体用法

### 2.1 三大控制结构

```cpp
#include <iostream>
using namespace std;

int main() {
    // 1. 顺序结构：按书写顺序依次执行
    cout << "步骤1" << endl;
    cout << "步骤2" << endl;
    cout << "步骤3" << endl;

    // 2. 分支结构：根据条件选择执行路径
    int score = 85;
    if (score >= 90) {
        cout << "优秀" << endl;
    } else if (score >= 60) {
        cout << "及格" << endl;
    } else {
        cout << "不及格" << endl;
    }

    // 3. 循环结构：重复执行直到条件不满足
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i;
    }
    cout << "1到10的和: " << sum << endl;

    return 0;
}
```

输出：
```
步骤1
步骤2
步骤3
及格
1到10的和: 55
```

### 2.2 控制流语句总览

| 类别 | 语句 | 用途 |
|------|------|------|
| 分支 | `if` / `else if` / `else` | 条件分支 |
| 分支 | `switch` / `case` | 多路分支 |
| 循环 | `for` | 计数循环 |
| 循环 | `while` | 条件循环 |
| 循环 | `do-while` | 至少执行一次的循环 |
| 跳转 | `break` | 跳出循环或switch |
| 跳转 | `continue` | 跳过本次迭代 |
| 跳转 | `return` | 从函数返回 |
| 跳转 | `goto` | 无条件跳转（不推荐） |
| C++17 | `if constexpr` | 编译期条件 |
| C++17 | 结构化绑定 | `auto [a, b] = pair` |

## 三、注意事项与常见陷阱

1. **所有控制流都可以互相嵌套**：if中可以有for，for中可以有if
2. **花括号习惯**：即使只有一条语句也建议加花括号，避免后续添加时遗漏
3. **缩进不代表作用域**：C++只认花括号，不认缩进
4. **避免深层嵌套**：超过3层嵌套应考虑重构（提取函数、guard clause等）
5. **break只跳出一层**：嵌套循环中break只跳出最近的一层
