# 无限循环与退出

## 一、概念说明

无限循环（死循环）是条件永远为true的循环。服务器、游戏主循环、事件处理等场景需要真正的无限循环，通过`break`、`return`或外部信号退出。

## 二、具体用法

### 2.1 无限循环写法

```cpp
#include <iostream>
using namespace std;

int main() {
    // 写法1：while(true)
    int count = 0;
    while (true) {
        if (count >= 3) break;
        cout << "while(true): " << count << endl;
        count++;
    }

    // 写法2：for(;;)
    count = 0;
    for (;;) {
        if (count >= 3) break;
        cout << "for(;;): " << count << endl;
        count++;
    }

    // 写法3：do-while(true)
    count = 0;
    do {
        if (count >= 3) break;
        cout << "do-while(true): " << count << endl;
        count++;
    } while (true);

    return 0;
}
```

输出：
```
while(true): 0
while(true): 1
while(true): 2
for(;;): 0
for(;;): 1
for(;;): 2
do-while(true): 0
do-while(true): 1
do-while(true): 2
```

### 2.2 实际应用

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 游戏主循环简化版
    bool running = true;
    int frame = 0;

    while (running) {
        // 处理输入
        // string input = getUserInput();

        // 模拟：3帧后退出
        frame++;
        if (frame > 3) {
            running = false;
            continue;
        }

        // 更新游戏状态
        cout << "帧 " << frame << ": 更新状态" << endl;

        // 渲染
        cout << "帧 " << frame << ": 渲染" << endl;
    }

    cout << "游戏结束" << endl;

    return 0;
}
```

输出：
```
帧 1: 更新状态
帧 1: 渲染
帧 2: 更新状态
帧 2: 渲染
帧 3: 更新状态
帧 3: 渲染
游戏结束
```

## 三、注意事项与常见陷阱

1. **确保有退出条件**：没有退出条件的循环会永远运行
2. **避免CPU 100%占用**：长时间等待的循环应包含sleep/yield
3. **信号处理**：用信号处理函数优雅退出
4. **while(true)更常见**：`for(;;)`功能相同但可读性略差
5. **调试时设置断点**：在循环退出条件处设置断点便于调试
