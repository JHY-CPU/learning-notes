# if语句详解

## 一、概念说明

`if`语句根据条件表达式的真假选择执行路径。C++支持`if`、`if-else`和`if-else if-else`多种形式。

## 二、具体用法

### 2.1 基本if语句

```cpp
#include <iostream>
using namespace std;

int main() {
    int temperature = 35;

    // 简单if
    if (temperature > 30) {
        cout << "天气很热！" << endl;
    }

    // if-else
    int age = 15;
    if (age >= 18) {
        cout << "成年人" << endl;
    } else {
        cout << "未成年人" << endl;
    }

    // if-else if-else链
    int score = 85;
    if (score >= 90) {
        cout << "A - 优秀" << endl;
    } else if (score >= 80) {
        cout << "B - 良好" << endl;
    } else if (score >= 70) {
        cout << "C - 中等" << endl;
    } else if (score >= 60) {
        cout << "D - 及格" << endl;
    } else {
        cout << "F - 不及格" << endl;
    }

    return 0;
}
```

输出：
```
天气很热！
未成年人
B - 良好
```

### 2.2 嵌套if与else匹配

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 20;

    // else与最近的if匹配
    if (x > 0)
        if (y > 0)
            cout << "x和y都为正" << endl;
        else  // 这个else属于内层if
            cout << "x为正但y非正" << endl;

    // 推荐：始终使用花括号明确意图
    if (x > 0) {
        if (y > 0) {
            cout << "x和y都为正" << endl;
        } else {
            cout << "x为正但y非正" << endl;
        }
    }

    return 0;
}
```

输出：
```
x和y都为正
x和y都为正
```

### 2.3 if中的初始化语句（C++17）

```cpp
#include <iostream>
#include <string>
#include <map>
using namespace std;

int main() {
    map<string, int> scores = {{"Alice", 95}, {"Bob", 87}};

    // C++17：if中包含初始化
    if (auto it = scores.find("Alice"); it != scores.end()) {
        cout << "找到: " << it->first << " = " << it->second << endl;
    } else {
        cout << "未找到" << endl;
    }

    // it在这里不可见
    // cout << it->first;  // 编译错误

    // 实用场景：锁的作用域
    // if (lock_guard<mutex> lock(mtx); condition) {
    //     // 持有锁的操作
    // }

    return 0;
}
```

输出：
```
找到: Alice = 95
```

## 三、注意事项与常见陷阱

1. **else悬挂**：`else`与最近的`if`匹配，使用花括号明确
2. **= vs ==**：`if (x = 5)`是赋值，开启`-Wparentheses`警告
3. **条件必须是bool或可转为bool**：`if (ptr)`检查指针是否非空
4. **短路求值**：`&&`和`||`在if中支持短路
5. **C++17初始化语句**：限制变量的作用域，避免名称泄漏
