# 正则表达式基础

## 一、概念说明

C++11引入了`<regex>`库，提供了标准的正则表达式支持。主要功能包括：**匹配**（整个字符串是否符合模式）、**搜索**（在字符串中查找模式）、**替换**（用新模式替换匹配部分）。

## 二、具体用法

### 2.1 匹配（regex_match）

```cpp
#include <regex>

std::string s = "12345";
std::regex pattern(R"(\d+)");  // 一个或多个数字

if (std::regex_match(s, pattern)) {
    std::cout << s << " 是纯数字\n";
}
// 输出: 12345 是纯数字

// 带捕获组
std::string email = "user@example.com";
std::regex emailPat(R"((\w+)@(\w+)\.(\w+))");
std::smatch m;
if (std::regex_match(email, m, emailPat)) {
    std::cout << "用户: " << m[1] << std::endl;
    std::cout << "域名: " << m[2] << std::endl;
}
// 输出:
// 用户: user
// 域名: example
```

### 2.2 搜索（regex_search）

```cpp
std::string text = "Order #123: $45.67";
std::regex numPat(R"(\d+)");

std::smatch match;
if (std::regex_search(text, match, numPat)) {
    std::cout << "找到数字: " << match[0] << std::endl;
}
// 输出: 找到数字: 123

// 查找所有匹配
auto begin = std::sregex_iterator(text.begin(), text.end(), numPat);
auto end = std::sregex_iterator();
for (auto it = begin; it != end; ++it) {
    std::cout << "匹配: " << it->str() << std::endl;
}
// 输出:
// 匹配: 123
// 匹配: 45
// 匹配: 67
```

### 2.3 替换（regex_replace）

```cpp
std::string text = "Hello 123 World 456";
std::regex digits(R"(\d+)");

// 替换所有数字为"#"
std::string result = std::regex_replace(text, digits, "#");
std::cout << result << std::endl;
// 输出: Hello # World #
```

### 2.4 常用正则语法

```cpp
// 元字符: . * + ? ^ $ | \ [] {} ()
// \d 数字  \w 单词字符  \s 空白  . 任意字符
// + 一次或多次  * 零次或多次  ? 零次或一次

std::regex datePat(R"(\d{4}-\d{2}-\d{2})");
std::string text = "Date: 2024-01-15";
if (std::regex_search(text, datePat)) {
    std::cout << "找到日期\n";
}
// 输出: 找到日期
```

## 三、注意事项与常见陷阱

- 正则表达式编译有开销，频繁使用应复用`std::regex`对象
- `std::regex_match`要求**整个**字符串匹配，`regex_search`只需部分匹配
- 使用`R"(...)"`原始字符串避免转义困扰
- 正则语法默认为ECMAScript，可通过标志指定其他语法
- 复杂正则可能导致性能问题（灾难性回溯）
