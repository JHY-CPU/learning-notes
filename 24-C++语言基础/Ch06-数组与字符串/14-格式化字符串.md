# 格式化字符串

## 一、概念说明

C++中格式化字符串的方式包括C风格的`sprintf`/`snprintf`和C++20引入的`std::format`。`std::format`提供了类型安全、高性能的格式化方案，是现代C++的推荐方式。

## 二、具体用法

### 2.1 sprintf / snprintf（C风格）

```cpp
char buf[100];

// sprintf（不安全，可能溢出）
sprintf(buf, "Name: %s, Age: %d", "Alice", 25);
std::cout << buf << std::endl;
// 输出: Name: Alice, Age: 25

// snprintf（安全，限制写入长度）
snprintf(buf, sizeof(buf), "PI = %.4f", 3.14159);
std::cout << buf << std::endl;
// 输出: PI = 3.1416

// 格式化到string
std::string result;
result.resize(50);
int len = snprintf(result.data(), result.size(),
                   "Score: %d/%d", 85, 100);
result.resize(len);
std::cout << result << std::endl;
// 输出: Score: 85/100
```

### 2.2 std::format（C++20）

```cpp
#include <format>

// 基本格式化
std::string s = std::format("Hello, {}!", "World");
std::cout << s << std::endl;  // 输出: Hello, World!

// 多参数
std::cout << std::format("{} + {} = {}", 3, 5, 8) << std::endl;
// 输出: 3 + 5 = 8

// 指定格式
std::cout << std::format("PI = {:.2f}", 3.14159) << std::endl;
// 输出: PI = 3.14

// 十六进制
std::cout << std::format("0x{:08X}", 255) << std::endl;
// 输出: 0x000000FF
```

### 2.3 std::format高级用法

```cpp
// 参数索引
std::cout << std::format("{1} is {0}", "Alice", 25) << std::endl;
// 输出: 25 is Alice

// 对齐和填充
std::cout << std::format("{:>10}", "right") << std::endl;
// 输出:      right
std::cout << std::format("{:*<10}", "left") << std::endl;
// 输出: left******

// 科学计数法
std::cout << std::format("{:e}", 1234567.0) << std::endl;
// 输出: 1.234567e+06
```

### 2.4 格式化到输出流

```cpp
// C++23: std::print
// std::print("Hello, {}!\n", "World");

// 等价方式
std::cout << std::format("Hello, {}!", "World") << std::endl;
// 输出: Hello, World!
```

## 三、注意事项与常见陷阱

- `sprintf`不安全，始终使用`snprintf`并检查返回值
- `std::format`的`{}`顺序从0开始，类型安全
- C++20之前可使用`fmt`库（`std::format`的设计来源）
- `snprintf`返回需要的缓冲区大小（不含`\0`）
- `std::format`格式化浮点数精度更可控
