# 字符串比较与排序

## 一、概念说明

字符串比较基于**字典序**（lexicographical order），逐字符比较ASCII/Unicode值。`std::string`的`<`、`>`、`==`运算符和`compare()`方法都基于字典序。对于特殊排序需求（如大小写不敏感、locale感知），需要自定义比较器。

## 二、具体用法

### 2.1 基本比较

```cpp
std::string a = "apple";
std::string b = "banana";

std::cout << (a == b) << std::endl;   // 输出: 0
std::cout << (a < b) << std::endl;    // 输出: 1 (a的'a' < b的'b')

// compare()返回值
// <0: *this < other,  0: 相等,  >0: *this > other
std::cout << a.compare(b) << std::endl;  // 输出: 负数
std::cout << a.compare(a) << std::endl;  // 输出: 0
```

### 2.2 字符串排序

```cpp
std::vector<std::string> names = {
    "Charlie", "alice", "Bob", "DAVID"
};

// 默认字典序（大写 < 小写）
std::sort(names.begin(), names.end());
for (const auto& n : names) std::cout << n << " ";
// 输出: Bob Charlie DAVID alice
```

### 2.3 大小写不敏感比较

```cpp
bool caseInsensitive(const std::string& a, const std::string& b) {
    return std::lexicographical_compare(
        a.begin(), a.end(),
        b.begin(), b.end(),
        [](char ca, char cb) {
            return std::tolower(ca) < std::tolower(cb);
        });
}

std::vector<std::string> names = {"Charlie", "alice", "Bob", "DAVID"};
std::sort(names.begin(), names.end(), caseInsensitive);
for (const auto& n : names) std::cout << n << " ";
// 输出: alice Bob Charlie DAVID
```

### 2.4 自然排序（数字感知）

```cpp
// "file2" < "file10" 而非 "file10" < "file2"
bool naturalCompare(const std::string& a, const std::string& b) {
    size_t i = 0, j = 0;
    while (i < a.size() && j < b.size()) {
        if (std::isdigit(a[i]) && std::isdigit(b[j])) {
            // 提取数字并比较
            long na = 0, nb = 0;
            while (i < a.size() && std::isdigit(a[i]))
                na = na * 10 + (a[i++] - '0');
            while (j < b.size() && std::isdigit(b[j]))
                nb = nb * 10 + (b[j++] - '0');
            if (na != nb) return na < nb;
        } else {
            if (a[i] != b[j]) return a[i] < b[j];
            ++i; ++j;
        }
    }
    return a.size() < b.size();
}

std::vector<std::string> files = {"file10", "file2", "file1", "file20"};
std::sort(files.begin(), files.end(), naturalCompare);
for (const auto& f : files) std::cout << f << " ";
// 输出: file1 file2 file10 file20
```

## 三、注意事项与常见陷阱

- 默认字符串排序大写字母在小写字母前面（ASCII值更小）
- `std::locale`可用于locale感知的比较和排序
- Unicode字符串的排序更复杂，需要考虑规范化
- 自然排序没有标准库实现，需要自行编写或使用第三方库
- `std::hash<std::string>`可用于哈希表中的字符串键
