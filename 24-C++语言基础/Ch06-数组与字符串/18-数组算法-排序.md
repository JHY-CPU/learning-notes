# 数组算法 - 排序

## 一、概念说明

`<algorithm>`头文件提供了丰富的排序算法。`std::sort`是最常用的通用排序，基于**内省排序**（introsort），平均时间复杂度O(n log n)。还可使用`std::stable_sort`（稳定排序）、`std::partial_sort`（部分排序）等。

## 二、具体用法

### 2.1 基本排序

```cpp
#include <algorithm>
#include <vector>

std::vector<int> v = {5, 2, 8, 1, 9, 3};

// 升序排序
std::sort(v.begin(), v.end());
for (int x : v) std::cout << x << " ";
// 输出: 1 2 3 5 8 9
```

### 2.2 自定义比较器

```cpp
std::vector<int> v = {5, 2, 8, 1, 9, 3};

// 降序
std::sort(v.begin(), v.end(), std::greater<int>());
for (int x : v) std::cout << x << " ";
// 输出: 9 8 5 3 2 1

// lambda比较器
std::sort(v.begin(), v.end(), [](int a, int b) {
    return a % 10 < b % 10;  // 按个位数排序
});
for (int x : v) std::cout << x << " ";
// 输出: 1 2 3 5 8 9 (个位: 1 2 3 5 8 9)
```

### 2.3 稳定排序

```cpp
std::vector<std::pair<int, std::string>> items = {
    {3, "C"}, {1, "A"}, {3, "B"}, {2, "D"}
};

// 稳定排序：相等元素保持原始顺序
std::stable_sort(items.begin(), items.end(),
    [](const auto& a, const auto& b) {
        return a.first < b.first;
    });

for (const auto& [num, name] : items)
    std::cout << "(" << num << "," << name << ") ";
// 输出: (1,A) (2,D) (3,C) (3,B)  -- B在C前面保持了原始顺序
```

### 2.4 部分排序与第N大元素

```cpp
std::vector<int> v = {5, 2, 8, 1, 9, 3, 7, 4, 6};

// 找前3个最小元素
std::partial_sort(v.begin(), v.begin() + 3, v.end());
for (int i = 0; i < 3; ++i) std::cout << v[i] << " ";
// 输出: 1 2 3

// 第N大元素（nth_element）
std::vector<int> nums = {5, 2, 8, 1, 9};
std::nth_element(nums.begin(), nums.begin() + 2, nums.end());
std::cout << "第3小: " << nums[2] << std::endl;
// 输出: 第3小: 5
```

## 三、注意事项与常见陷阱

- `std::sort`要求**随机访问迭代器**，list用`list::sort()`
- 比较器必须满足**严格弱序**：`comp(a,a)`为false
- `std::sort`可能改变相等元素的相对顺序，需要稳定用`stable_sort`
- 排序前检查容器是否为空
- 对部分有序的数据，`std::sort`仍是O(n log n)，`std::stable_sort`可能更快
