# 编译期字符串

## 一、概念说明

C++17/20增强了编译期字符串操作能力。`constexpr`字符串操作允许在编译期执行字符串查找、比较、哈希等，结果直接嵌入程序代码，零运行时开销。

## 二、具体用法

### 2.1 constexpr字符串比较

```cpp
constexpr bool strEqual(const char* a, const char* b) {
    while (*a && *a == *b) { ++a; ++b; }
    return *a == *b;
}

static_assert(strEqual("hello", "hello"), "should be equal");
static_assert(!strEqual("hello", "world"), "should differ");

// 编译期结果
constexpr bool same = strEqual("abc", "abc");
std::cout << same << std::endl;  // 输出: 1
```

### 2.2 编译期字符串哈希

```cpp
constexpr uint32_t fnv1aHash(const char* str, uint32_t hash = 2166136261u) {
    return (*str == '\0') ? hash
        : fnv1aHash(str + 1, (hash ^ static_cast<uint32_t>(*str)) * 16777619u);
}

// 编译期计算哈希
constexpr auto h1 = fnv1aHash("hello");
constexpr auto h2 = fnv1aHash("hello");
static_assert(h1 == h2, "same string same hash");

std::cout << "hash: " << h1 << std::endl;
// 输出: hash: 1335831723 (示例值)
```

### 2.3 constexpr字符串类（C++20）

```cpp
#include <algorithm>

struct ConstString {
    char data[32] = {};
    size_t len = 0;

    constexpr ConstString(const char* s) {
        while (*s && len < 31) {
            data[len++] = *s++;
        }
    }

    constexpr size_t size() const { return len; }
    constexpr char operator[](size_t i) const { return data[i]; }

    constexpr bool contains(char c) const {
        for (size_t i = 0; i < len; ++i)
            if (data[i] == c) return true;
        return false;
    }
};

constexpr ConstString s("Hello World");
static_assert(s.contains('W'), "should contain W");
static_assert(s.size() == 11, "length should be 11");
```

### 2.4 编译期字符串查找

```cpp
constexpr size_t strFind(const char* str, char target, size_t pos = 0) {
    if (str[pos] == '\0') return static_cast<size_t>(-1);  // npos
    if (str[pos] == target) return pos;
    return strFind(str, target, pos + 1);
}

constexpr size_t pos = strFind("Hello World", 'W');
static_assert(pos == 6, "W is at position 6");
std::cout << "位置: " << pos << std::endl;  // 输出: 位置: 6
```

### 2.5 编译期字符串拼接

```cpp
template <size_t N1, size_t N2>
constexpr auto concatStrings(const char (&a)[N1], const char (&b)[N2]) {
    char result[N1 + N2 - 1] = {};
    for (size_t i = 0; i < N1 - 1; ++i) result[i] = a[i];
    for (size_t i = 0; i < N2; ++i) result[N1 - 1 + i] = b[i];
    // 返回数组需要包装
    // 实际项目可使用 fixed_string 等类型
    return result;
}
```

## 三、注意事项与常见陷阱

- `constexpr`字符串函数在C++20之前不能使用`std::string`
- 编译期字符串操作受限于constexpr的限制（无动态内存分配等）
- `consteval`可强制编译期求值（C++20）
- 编译期哈希可用于`switch`语句中的字符串匹配
- 实际项目中可使用`constexpr_string`等第三方库
