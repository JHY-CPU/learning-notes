# 数组算法 - 查找

## 一、概念说明

`<algorithm>`提供了多种查找算法：`std::find`（线性查找）、`std::binary_search`（二分查找）、`std::lower_bound`/`upper_bound`（边界查找）、`std::find_if`（条件查找）等。

## 二、具体用法

### 2.1 线性查找：std::find

```cpp
std::vector<int> v = {10, 20, 30, 40, 50};

auto it = std::find(v.begin(), v.end(), 30);
if (it != v.end()) {
    std::cout << "找到: " << *it
              << " 位置: " << std::distance(v.begin(), it) << std::endl;
}
// 输出: 找到: 30 位置: 2

// 查找不存在的元素
auto end = std::find(v.begin(), v.end(), 99);
if (end == v.end()) {
    std::cout << "未找到\n";
}
// 输出: 未找到
```

### 2.2 条件查找：std::find_if

```cpp
std::vector<int> v = {1, 4, 6, 8, 3};

// 找第一个偶数
auto it = std::find_if(v.begin(), v.end(),
    [](int x) { return x % 2 == 0; });

if (it != v.end()) {
    std::cout << "第一个偶数: " << *it << std::endl;
}
// 输出: 第一个偶数: 4
```

### 2.3 二分查找（需已排序）

```cpp
std::vector<int> v = {1, 3, 5, 7, 9, 11};

// binary_search：只返回是否存在
bool found = std::binary_search(v.begin(), v.end(), 7);
std::cout << "7存在: " << found << std::endl;  // 输出: 7存在: 1

// lower_bound：第一个 >= value 的位置
auto lo = std::lower_bound(v.begin(), v.end(), 6);
std::cout << "lower_bound(6): " << *lo << std::endl;
// 输出: lower_bound(6): 7

// upper_bound：第一个 > value 的位置
auto up = std::upper_bound(v.begin(), v.end(), 5);
std::cout << "upper_bound(5): " << *up << std::endl;
// 输出: upper_bound(5): 7
```

### 2.4 计数与极值

```cpp
std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

// 计数
int cnt = std::count(v.begin(), v.end(), 1);
std::cout << "1出现次数: " << cnt << std::endl;  // 输出: 1出现次数: 2

// 最大最小
auto [minIt, maxIt] = std::minmax_element(v.begin(), v.end());
std::cout << "最小: " << *minIt << ", 最大: " << *maxIt << std::endl;
// 输出: 最小: 1, 最大: 9
```

## 三、注意事项与常见陷阱

- `std::find`是O(n)，`std::binary_search`是O(log n)但要求**已排序**
- 所有查找失败都返回`end()`迭代器
- `lower_bound`和`upper_bound`可用于实现插入排序位置
- `std::find_if`可接受lambda，实现任意条件查找
- 对频繁查找的数据，考虑使用`std::set`或`std::unordered_set`
