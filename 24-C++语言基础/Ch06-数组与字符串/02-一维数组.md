# 一维数组

## 一、概念说明

一维数组是同一类型元素的**连续存储**序列，大小在编译时确定（静态数组）或运行时确定（动态数组）。数组名在大多数表达式中**退化为指向首元素的指针**。

## 二、具体用法

### 2.1 声明与初始化

```cpp
// 默认初始化（值未定义）
int arr1[5];

// 聚合初始化
int arr2[5] = {10, 20, 30, 40, 50};

// 部分初始化（剩余元素为0）
int arr3[5] = {1, 2};  // {1, 2, 0, 0, 0}

// 全部初始化为0
int arr4[5] = {};

// 自动推导大小
int arr5[] = {1, 2, 3, 4};  // 大小为4

std::cout << "arr2[0]=" << arr2[0]
          << ", arr2[4]=" << arr2[4] << std::endl;
// 输出: arr2[0]=10, arr2[4]=50
```

### 2.2 访问与遍历

```cpp
int nums[] = {10, 20, 30, 40, 50};

// 下标访问（从0开始）
std::cout << nums[2] << std::endl;  // 输出: 30

// 范围for循环
for (int n : nums) {
    std::cout << n << " ";
}
// 输出: 10 20 30 40 50

// 计算元素个数
size_t len = sizeof(nums) / sizeof(nums[0]);
std::cout << "\n长度: " << len << std::endl;
// 输出: 长度: 5
```

### 2.3 动态数组

```cpp
// 运行时确定大小
int n = 10;
int* dynArr = new int[n];

for (int i = 0; i < n; ++i) {
    dynArr[i] = i * i;
}

std::cout << "dynArr[5] = " << dynArr[5] << std::endl;
// 输出: dynArr[5] = 25

delete[] dynArr;  // 必须用delete[]释放
```

### 2.4 越界访问

```cpp
int arr[3] = {1, 2, 3};
// arr[5] = 10;  // 未定义行为！编译器通常不检查
// 使用at()可以检查（std::array）
```

## 三、注意事项与常见陷阱

- 数组下标从0开始，越界是**未定义行为**（UB）
- 数组名不是指针常量，`sizeof(arr)`返回整个数组大小
- 局部数组不初始化，值是垃圾值
- `new[]`分配的数组必须用`delete[]`释放（不能用`delete`）
- 数组不能直接赋值：`arr1 = arr2` 是错误的
