# 一维数组

## 一、概念说明

一维数组是同一类型元素的**连续存储**序列（C++11标准 §8.3.4），大小在编译时确定（静态数组）或运行时确定（动态数组）。数组名在大多数表达式中**退化为指向首元素的指针**。

现代C++中推荐使用 `std::array`（固定大小）和 `std::vector`（动态大小）替代裸数组。

| 容器 | 大小 | 头文件 | 引入版本 |
|------|------|--------|---------|
| C风格数组 `T[N]` | 编译期 | 无 | C |
| `std::array<T,N>` | 编译期 | `<array>` | C++11 |
| `std::vector<T>` | 运行期 | `<vector>` | C++98 |
| `std::span<T>` | 运行期 | `<span>` | C++20 |

## 二、具体用法

### 2.1 声明与初始化

```cpp
#include <iostream>
using namespace std;

int main() {
    // 默认初始化（局部数组值未定义，全局/静态数组为0）
    int arr1[5];              // 垃圾值

    // 聚合初始化
    int arr2[5] = {10, 20, 30, 40, 50};

    // 部分初始化（剩余元素为0）
    int arr3[5] = {1, 2};     // {1, 2, 0, 0, 0}

    // 全部初始化为0
    int arr4[5] = {};         // {0, 0, 0, 0, 0}

    // 自动推导大小
    int arr5[] = {1, 2, 3, 4};  // 大小为4

    // C++17 std::array
    #include <array>
    array<int, 5> stdArr = {10, 20, 30, 40, 50};
    stdArr.fill(0);  // 全部置零

    cout << "arr2[0]=" << arr2[0] << ", arr2[4]=" << arr2[4] << endl;
    // 输出: arr2[0]=10, arr2[4]=50

    return 0;
}
```

### 2.2 访问与遍历

```cpp
#include <iostream>
using namespace std;

int main() {
    int nums[] = {10, 20, 30, 40, 50};

    // 下标访问（从0开始）
    cout << nums[2] << endl;  // 30

    // 范围for循环（C++11）
    for (int n : nums) {
        cout << n << " ";
    }
    cout << endl;  // 10 20 30 40 50

    // 指针遍历
    for (int* p = begin(nums); p != end(nums); ++p) {
        cout << *p << " ";
    }
    cout << endl;

    // 计算元素个数
    size_t len = size(nums);  // C++17 std::size，或 sizeof(nums)/sizeof(nums[0])
    cout << "长度: " << len << endl;  // 5

    // std::array遍历更安全
    #include <array>
    array<int, 5> arr = {1, 2, 3, 4, 5};
    cout << "arr.at(2) = " << arr.at(2) << endl;  // 带边界检查
    // arr.at(10);  // 抛出 std::out_of_range

    return 0;
}
```

### 2.3 动态数组

```cpp
#include <iostream>
#include <memory>
#include <vector>
using namespace std;

int main() {
    // 传统动态数组（不推荐）
    int n = 10;
    int* dynArr = new int[n]();  // ()初始化为0
    for (int i = 0; i < n; ++i) dynArr[i] = i * i;
    cout << "dynArr[5] = " << dynArr[5] << endl;  // 25
    delete[] dynArr;  // 必须用delete[]

    // 现代C++：unique_ptr数组
    auto smartArr = make_unique<int[]>(n);
    for (int i = 0; i < n; ++i) smartArr[i] = i * i;
    // 自动释放

    // 最佳：std::vector
    vector<int> vec(n);
    for (int i = 0; i < n; ++i) vec[i] = i * i;
    vec.push_back(100);  // 可动态增长

    return 0;
}
```

### 2.4 数组退化与传递

```cpp
#include <iostream>
using namespace std;

// 数组退化为指针：丢失大小信息
void printArr(int arr[]) {  // 实际是 int* arr
    cout << "sizeof(arr) = " << sizeof(arr) << endl;  // 指针大小！
}

// 使用引用保留大小信息
template <size_t N>
void printArrRef(const int (&arr)[N]) {
    cout << "数组大小: " << N << endl;
    for (size_t i = 0; i < N; ++i)
        cout << arr[i] << " ";
    cout << endl;
}

int main() {
    int nums[] = {1, 2, 3, 4, 5};
    printArr(nums);     // sizeof可能是8（指针大小）
    printArrRef(nums);  // 数组大小: 5

    return 0;
}
```

### 2.5 数组算法

```cpp
#include <iostream>
#include <algorithm>
#include <numeric>
using namespace std;

int main() {
    int arr[] = {5, 2, 8, 1, 9, 3};

    // 排序
    sort(begin(arr), end(arr));  // {1, 2, 3, 5, 8, 9}

    // 查找
    auto it = find(begin(arr), end(arr), 5);
    if (it != end(arr))
        cout << "找到5，位置: " << distance(begin(arr), it) << endl;

    // 累加
    int sum = accumulate(begin(arr), end(arr), 0);
    cout << "sum = " << sum << endl;  // 28

    // 最大值
    cout << "max = " << *max_element(begin(arr), end(arr)) << endl;  // 9

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **数组下标从0开始，越界是未定义行为**（UB）：编译器通常不检查
2. **数组名不是指针常量**：`sizeof(arr)`返回整个数组大小，不是指针大小
3. **局部数组不初始化，值是垃圾值**：全局/静态数组会初始化为0
4. **new[]分配的数组必须用delete[]释放**：用delete是UB
5. **数组不能直接赋值**：`arr1 = arr2` 编译错误，需用 `memcpy` 或循环
6. **数组退化丢失大小信息**：函数参数传递时退化为指针，模板引用可保留大小
7. **优先使用std::array和std::vector**：提供边界检查、迭代器、STL算法兼容性
