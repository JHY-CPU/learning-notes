# 字符串操作函数

## 一、概念说明

C标准库`<cstring>`提供了一系列C风格字符串操作函数。这些函数以`\0`作为字符串结束标志，不进行边界检查，使用时需特别注意安全性。

现代C++推荐使用`std::string`替代这些函数。

## 二、具体用法

### 2.1 长度 strlen

```cpp
const char* str = "Hello";
std::cout << "长度: " << strlen(str) << std::endl;
// 输出: 长度: 5

char buf[20] = "World";
std::cout << "长度: " << strlen(buf) << std::endl;
// 输出: 长度: 5 （不含'\0'）
```

### 2.2 拷贝 strcpy / strncpy

```cpp
char dest[20];

strcpy(dest, "Hello");  // 不安全：不检查dest大小
std::cout << dest << std::endl;  // 输出: Hello

// strncpy更安全，但可能不添加'\0'
char safe[5];
strncpy(safe, "Hello World", 4);  // 拷贝最多4个字符
safe[4] = '\0';  // 必须手动添加终止符
std::cout << safe << std::endl;  // 输出: Hell
```

### 2.3 连接 strcat / strncat

```cpp
char str[50] = "Hello";
strcat(str, ", ");     // 连接
strcat(str, "World!");
std::cout << str << std::endl;
// 输出: Hello, World!

// strncat更安全
char buf[10] = "Hi";
strncat(buf, " there!!!", 3);  // 最多追加3个字符
std::cout << buf << std::endl;  // 输出: Hi th
```

### 2.4 比较 strcmp

```cpp
const char* a = "apple";
const char* b = "banana";

int result = strcmp(a, b);
if (result < 0)
    std::cout << a << " < " << b << std::endl;
else if (result > 0)
    std::cout << a << " > " << b << std::endl;
else
    std::cout << a << " == " << b << std::endl;
// 输出: apple < banana

// 返回值：<0 前小, 0 相等, >0 前大
```

### 2.5 查找 strchr / strstr

```cpp
const char* str = "Hello World";

// 查找字符
const char* pos = strchr(str, 'W');
if (pos) std::cout << "位置: " << (pos - str) << std::endl;
// 输出: 位置: 6

// 查找子串
const char* sub = strstr(str, "World");
if (sub) std::cout << "找到: " << sub << std::endl;
// 输出: 找到: World
```

## 三、注意事项与常见陷阱

- 所有`str*`函数都不做边界检查，容易缓冲区溢出
- `strncpy`不保证添加`\0`，需手动处理
- `strcmp`返回值不是简单的-1/0/1，而是负/0/正
- 优先使用`std::string`或安全版本（`strncpy_s`等）
- 空指针传入这些函数会导致崩溃
