# 字符串字面量

## 一、概念说明

C++支持多种字符串字面量：普通字符串`"..."`、宽字符串`L"..."`、UTF-8字符串`u8"..."`、UTF-16`u"..."`、UTF-32`U"..."`，以及原始字符串`R"(...)"`。不同前缀产生不同类型的字符数组。

## 二、具体用法

### 2.1 编码前缀

```cpp
// 普通字符串（char）
const char* s1 = "Hello";

// 宽字符（wchar_t，平台相关大小）
const wchar_t* s2 = L"Hello";

// UTF-8（char8_t，C++20）
// const char8_t* s3 = u8"Hello";  // C++20

// UTF-16（char16_t）
const char16_t* s4 = u"Hello";

// UTF-32（char32_t）
const char32_t* s5 = U"Hello";

std::cout << s1 << std::endl;  // 输出: Hello
```

### 2.2 原始字符串字面量（C++11）

```cpp
// 原始字符串：不处理转义字符
std::string path = R"(C:\Users\name\file.txt)";
std::cout << path << std::endl;
// 输出: C:\Users\name\file.txt

// 包含引号和括号
std::string json = R"({"name": "Alice", "age": 25})";
std::cout << json << std::endl;
// 输出: {"name": "Alice", "age": 25}

// 自定义定界符（处理包含 )" 的情况）
std::string code = R"delim(printf(")");)delim";
std::cout << code << std::endl;
// 输出: printf(")");
```

### 2.3 字符串拼接

```cpp
// 相邻字面量自动拼接
std::string multi =
    "This is a "
    "very long "
    "string.";
std::cout << multi << std::endl;
// 输出: This is a very long string.

// 注意类型匹配
auto concat = "Hello " "World";  // "Hello World"
```

### 2.4 后缀 s（C++14）

```cpp
using namespace std::string_literals;

auto str = "Hello"s;  // 直接得到std::string
std::cout << str.size() << std::endl;  // 输出: 5

// 区别
std::cout << sizeof("Hello") << std::endl;  // 输出: 6 (char数组)
std::cout << "Hello"s.size() << std::endl;  // 输出: 5 (string)
```

### 2.5 后缀 sv（C++17）

```cpp
using namespace std::string_view_literals;

auto sv = "Hello"sv;  // 得到std::string_view
std::cout << sv.size() << std::endl;  // 输出: 5
```

## 三、注意事项与常见陷阱

- 原始字符串中`\n`不会被解释为换行
- `u8`前缀在C++20中返回`char8_t`，与`char`不兼容
- 字符串字面量存储在只读内存区，通过`char*`修改是UB
- 跨平台开发注意`wchar_t`大小差异（Windows 2字节，Linux 4字节）
- 相邻字符串拼接时类型必须兼容
