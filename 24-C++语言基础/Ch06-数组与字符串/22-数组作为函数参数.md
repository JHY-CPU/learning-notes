# 数组作为函数参数

## 一、概念说明

C++中数组不能按值传递给函数（会发生**退化**为指针）。有多种方式传递数组：退化为指针、传递数组大小的引用、使用模板保留大小信息，或使用STL容器。

## 二、具体用法

### 2.1 退化为指针

```cpp
// 以下三种写法等价，都会退化为 int*
void print1(int arr[]);
void print2(int arr[10]);   // 10被忽略！
void print3(int* arr);

void print1(int arr[]) {
    // sizeof(arr) 返回指针大小，不是数组大小
    std::cout << "sizeof(arr) = " << sizeof(arr) << std::endl;
}

int nums[10];
print1(nums);
// 输出: sizeof(arr) = 8 (64位系统)
```

### 2.2 传递大小

```cpp
void printArray(int* arr, size_t size) {
    for (size_t i = 0; i < size; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;
}

int nums[] = {1, 2, 3, 4, 5};
printArray(nums, sizeof(nums) / sizeof(nums[0]));
// 输出: 1 2 3 4 5
```

### 2.3 引用传递（保留大小）

```cpp
// 模板引用传递，保留数组大小
template <size_t N>
void printArray(int (&arr)[N]) {
    std::cout << "数组大小: " << N << std::endl;
    for (size_t i = 0; i < N; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;
}

int nums[] = {10, 20, 30};
printArray(nums);
// 输出:
// 数组大小: 3
// 10 20 30
```

### 2.4 使用std::array或std::span

```cpp
#include <array>

// std::array可按值传递，自带size()
void print(std::array<int, 5> arr) {
    for (int x : arr) std::cout << x << " ";
    std::cout << std::endl;
}

// C++20 std::span：通用连续序列视图
// void print(std::span<int> sp) {
//     for (int x : sp) std::cout << x << " ";
// }
```

## 三、注意事项与常见陷阱

- `int arr[10]`作为参数时`10`被忽略，实际是`int*`
- 引用传递数组可以保留大小信息但限制了灵活性
- 模板方式对不同大小的数组生成不同函数实例
- `std::span`（C++20）是传递数组/容器的最佳方案
- C风格数组传参时总要同时传递大小信息
