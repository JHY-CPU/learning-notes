# 数组算法 - 变换

## 一、概念说明

`<algorithm>`提供了变换和遍历算法：`std::transform`对每个元素应用函数并存储结果，`std::for_each`对每个元素执行操作，`std::fill`填充指定值，`std::generate`用生成器填充。

## 二、具体用法

### 2.1 std::transform

```cpp
std::vector<int> src = {1, 2, 3, 4, 5};
std::vector<int> dst(src.size());

// 一元变换：每个元素乘以2
std::transform(src.begin(), src.end(), dst.begin(),
    [](int x) { return x * 2; });

for (int x : dst) std::cout << x << " ";
// 输出: 2 4 6 8 10

// 原地变换
std::transform(src.begin(), src.end(), src.begin(),
    [](int x) { return x * x; });
for (int x : src) std::cout << x << " ";
// 输出: 1 4 9 16 25
```

### 2.2 二元变换

```cpp
std::vector<int> a = {1, 2, 3};
std::vector<int> b = {10, 20, 30};
std::vector<int> result(a.size());

std::transform(a.begin(), a.end(), b.begin(), result.begin(),
    [](int x, int y) { return x + y; });

for (int x : result) std::cout << x << " ";
// 输出: 11 22 33
```

### 2.3 std::for_each

```cpp
std::vector<int> v = {1, 2, 3, 4, 5};

int sum = 0;
std::for_each(v.begin(), v.end(), [&sum](int x) {
    sum += x;
});
std::cout << "总和: " << sum << std::endl;  // 输出: 总和: 15

// 修改元素
std::for_each(v.begin(), v.end(), [](int& x) { x *= 10; });
for (int x : v) std::cout << x << " ";
// 输出: 10 20 30 40 50
```

### 2.4 fill 与 generate

```cpp
std::vector<int> v(5);

// fill：填充固定值
std::fill(v.begin(), v.end(), 42);
for (int x : v) std::cout << x << " ";
// 输出: 42 42 42 42 42

// generate：用函数生成值
int counter = 0;
std::generate(v.begin(), v.end(), [&counter]() { return counter++; });
for (int x : v) std::cout << x << " ";
// 输出: 0 1 2 3 4

// iota（<numeric>）：递增填充
std::iota(v.begin(), v.end(), 10);
for (int x : v) std::cout << x << " ";
// 输出: 10 11 12 13 14
```

### 2.5 copy 与 copy_if

```cpp
std::vector<int> src = {1, 2, 3, 4, 5};
std::vector<int> dst;

// copy_if：复制满足条件的元素
std::copy_if(src.begin(), src.end(), std::back_inserter(dst),
    [](int x) { return x % 2 == 0; });

for (int x : dst) std::cout << x << " ";
// 输出: 2 4
```

## 三、注意事项与常见陷阱

- `transform`的输出范围必须已分配空间，或使用`back_inserter`
- `for_each`返回传入的函数对象，可获取有状态的结果
- `transform`可做映射，`for_each`更适合副作用操作
- `fill`对POD类型效率高，`generate`更灵活
- 范围for循环是`for_each`的语法糖，更易读
