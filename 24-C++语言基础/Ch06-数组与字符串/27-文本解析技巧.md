# 文本解析技巧

## 一、概念说明

文本解析是将结构化文本（CSV、日志、配置文件等）拆分为有意义的数据。C++提供了多种工具：`stringstream`、`getline`、`std::regex`、`std::string`的查找方法等。

## 二、具体用法

### 2.1 逐词读取

```cpp
#include <sstream>

std::string text = "Hello World C++ Programming";
std::istringstream iss(text);
std::string word;

while (iss >> word) {
    std::cout << word << std::endl;
}
// 输出:
// Hello
// World
// C++
// Programming
```

### 2.2 CSV解析

```cpp
std::string csv = "Alice,25,95.5\nBob,30,88.0\nCharlie,22,91.5";
std::istringstream stream(csv);
std::string line;

while (std::getline(stream, line)) {
    std::istringstream lineStream(line);
    std::string name;
    int age;
    double score;

    std::getline(lineStream, name, ',');
    lineStream >> age;
    lineStream.ignore();
    lineStream >> score;

    std::cout << name << " age=" << age
              << " score=" << score << std::endl;
}
// 输出:
// Alice age=25 score=95.5
// Bob age=30 score=88
// Charlie age=22 score=91.5
```

### 2.3 split实现

```cpp
std::vector<std::string> split(const std::string& s, char delimiter) {
    std::vector<std::string> tokens;
    std::istringstream stream(s);
    std::string token;
    while (std::getline(stream, token, delimiter)) {
        tokens.push_back(token);
    }
    return tokens;
}

auto words = split("one,two,three,four", ',');
for (const auto& w : words) std::cout << w << " ";
// 输出: one two three four
```

### 2.4 使用正则解析

```cpp
#include <regex>

std::string log = "2024-01-15 10:30:45 [ERROR] File not found";
std::regex pat(R"((\d{4}-\d{2}-\d{2}) (\d{2}:\d{2}:\d{2}) \[(\w+)\] (.+))");
std::smatch m;

if (std::regex_match(log, m, pat)) {
    std::cout << "日期: " << m[1] << std::endl;
    std::cout << "时间: " << m[2] << std::endl;
    std::cout << "级别: " << m[3] << std::endl;
    std::cout << "消息: " << m[4] << std::endl;
}
// 输出:
// 日期: 2024-01-15
// 时间: 10:30:45
// 级别: ERROR
// 消息: File not found
```

### 2.5 按行读取文件

```cpp
#include <fstream>

std::ifstream file("data.txt");
std::string line;
while (std::getline(file, line)) {
    // 逐行处理
    std::cout << line << std::endl;
}
```

## 三、注意事项与常见陷阱

- `getline`读取整行时会消费换行符但不包含在结果中
- 解析失败后流进入错误状态，需`clear()`重置
- 正则解析开销大，简单场景用`find`/`substr`更快
- CSV中包含逗号或引号的字段需要特殊处理
- 编码问题可能导致中文等非ASCII字符解析异常
