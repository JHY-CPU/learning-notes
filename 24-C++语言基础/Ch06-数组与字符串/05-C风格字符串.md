# C风格字符串

## 一、概念说明

C风格字符串是以**NUL字符（`\0`）**结尾的`char`数组（C++11标准 §2.14.5）。字符串字面量（如`"hello"`）是C风格字符串的常量形式，自动在末尾添加`\0`。

现代C++中推荐使用 `std::string` 替代C风格字符串。

| 特性 | C风格字符串 | std::string |
|------|-----------|-------------|
| 内存管理 | 手动 | 自动 |
| 大小管理 | 手动维护\0 | 自动跟踪长度 |
| 安全性 | 易溢出 | 带边界检查 |
| 性能 | 直接操作 | 轻微开销 |
| 适用场景 | C接口交互 | 日常字符串操作 |

## 二、具体用法

### 2.1 声明与初始化

```cpp
#include <iostream>
#include <cstring>
using namespace std;

int main() {
    // 字符数组初始化
    char str1[] = "Hello";           // 大小为6（含'\0'）
    char str2[10] = "World";         // 大小为10，后面补'\0'
    char str3[] = {'H', 'i', '\0'};  // 显式添加'\0'
    // char str4[5] = "Hello";       // 编译错误！需要6个位置

    cout << "str1: " << str1 << endl;           // Hello
    cout << "sizeof(str1): " << sizeof(str1) << endl;  // 6
    cout << "strlen(str1): " << strlen(str1) << endl;  // 5

    // 部分初始化：其余自动补'\0'
    char buf[20] = "Short";
    cout << "strlen(buf): " << strlen(buf) << endl;  // 5
    cout << "sizeof(buf): " << sizeof(buf) << endl;  // 20

    return 0;
}
```

### 2.2 字符串字面量

```cpp
#include <iostream>
using namespace std;

int main() {
    // 字面量存储在只读区域
    const char* ptr = "Hello";  // 推荐加const

    // 字面量合并（相邻字符串自动拼接）
    const char* merged = "Hello "
                         "World";  // 等价于 "Hello World"
    cout << merged << endl;  // Hello World

    // C++11 raw string literal：保留转义字符
    const char* raw = R"(path\to\file\n)";
    cout << raw << endl;  // path\to\file\n

    // C++11 UTF-8字符串
    const char* utf8 = u8"你好世界";
    cout << utf8 << endl;

    return 0;
}
```

### 2.3 常用字符串函数

```cpp
#include <iostream>
#include <cstring>
using namespace std;

int main() {
    char dst[20];
    const char* src = "Hello";

    // strlen：字符串长度（不含'\0'）
    cout << "长度: " << strlen(src) << endl;  // 5

    // strcpy：复制（注意溢出风险）
    strcpy(dst, src);
    cout << "dst: " << dst << endl;  // Hello

    // strncpy：安全复制
    strncpy(dst, "VeryLongStringThatExceeds", sizeof(dst) - 1);
    dst[sizeof(dst) - 1] = '\0';  // 手动添加终止符

    // strcat：拼接
    char buf[20] = "Hello";
    strcat(buf, " World");
    cout << buf << endl;  // Hello World

    // strcmp：比较
    if (strcmp("abc", "abd") < 0) cout << "abc < abd" << endl;

    // strchr / strstr：查找
    const char* pos = strchr("Hello World", 'W');
    if (pos) cout << "找到W在位置: " << pos << endl;  // World

    return 0;
}
```

### 2.4 输入输出

```cpp
#include <iostream>
using namespace std;

int main() {
    char name[50];

    // cin >> 遇空格停止，可能溢出
    cout << "输入名字: ";
    // cin >> name;  // 不安全

    // getline：读取整行（更安全）
    cin.getline(name, 50);
    cout << "你好, " << name << endl;

    // gets已被C++14移除（极不安全）
    // fgets：C方式的安全读取
    // fgets(name, 50, stdin);

    return 0;
}
```

### 2.5 C风格字符串与std::string互转

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // C++ → C
    string cppStr = "Hello";
    const char* cStr = cppStr.c_str();  // 返回只读指针
    cout << cStr << endl;

    // 需要可修改副本
    char* mutableCopy = new char[cppStr.size() + 1];
    strcpy(mutableCopy, cppStr.c_str());
    mutableCopy[0] = 'h';  // OK
    delete[] mutableCopy;

    // C → C++
    const char* raw = "World";
    string cppStr2(raw);
    cout << cppStr2 << endl;  // World

    return 0;
}
```

### 2.6 常见安全问题

```cpp
#include <iostream>
#include <cstring>
using namespace std;

// 危险：缓冲区溢出
void unsafe() {
    char buf[10];
    strcpy(buf, "ThisStringIsTooLong");  // 溢出！未定义行为
}

// 安全：使用strncpy或std::string
void safe() {
    char buf[10];
    strncpy(buf, "ThisStringIsTooLong", sizeof(buf) - 1);
    buf[sizeof(buf) - 1] = '\0';
    cout << buf << endl;  // ThisStrin

    // 最佳：使用std::string
    string s = "ThisStringIsTooLong";
    if (s.size() > 9) s.resize(9);
    cout << s << endl;
}
```

## 三、注意事项与常见陷阱

1. **忘记'\0'终止符**：导致字符串函数越界读取或写入
2. **char str[5] = "Hello"编译错误**：需要6个位置放'\0'
3. **字符串字面量存储在只读内存区**：修改是未定义行为
4. **sizeof返回数组大小，strlen返回字符串长度**：两者含义不同
5. **strcpy/strcat不检查缓冲区大小**：极其危险，优先用strncpy或std::string
6. **gets已从C++14移除**：使用fgets或cin.getline
7. **c_str()返回的指针有效期有限**：string修改后可能失效
8. **c++字符串字面量默认是const char***：C++11起不能赋值给非const的char*
