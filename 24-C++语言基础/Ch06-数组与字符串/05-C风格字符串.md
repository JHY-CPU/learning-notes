# C风格字符串

## 一、概念说明

C风格字符串是以**NUL字符（`\0`）**结尾的`char`数组。字符串字面量（如`"hello"`）是C风格字符串的常量形式，自动在末尾添加`\0`。

C风格字符串的长度取决于`\0`的位置，而非数组大小。

## 二、具体用法

### 2.1 声明与初始化

```cpp
// 字符数组初始化
char str1[] = "Hello";        // 大小为6（含'\0'）
char str2[10] = "World";      // 大小为10，后面补'\0'
char str3[] = {'H', 'i', '\0'};  // 显式添加'\0'
char str4[5] = "Hello";       // 编译错误！需要6个位置

std::cout << str1 << std::endl;  // 输出: Hello
std::cout << sizeof(str1) << std::endl;  // 输出: 6
```

### 2.2 字符串字面量

```cpp
// 字面量存储在只读区域
const char* ptr = "Hello";  // 推荐加const

// 字面量合并
const char* merged = "Hello "
                     "World";  // 等价于 "Hello World"

std::cout << merged << std::endl;  // 输出: Hello World
```

### 2.3 修改字符串

```cpp
// 字符数组可修改
char str[] = "Hello";
str[0] = 'h';
std::cout << str << std::endl;  // 输出: hello

// 指向字面量的指针不可修改（未定义行为）
// char* ptr = "Hello";
// ptr[0] = 'h';  // 崩溃！
```

### 2.4 输入输出

```cpp
char name[50];
std::cout << "输入名字: ";
std::cin >> name;  // 遇空格停止，可能溢出
std::cout << "你好, " << name << std::endl;

// 更安全的方式
std::cin.getline(name, 50);  // 读取整行
```

## 三、注意事项与常见陷阱

- 忘记`\0`终止符会导致字符串函数越界读取
- `char str[5] = "Hello"` 编译错误（需要6个位置放`\0`）
- 字符串字面量存储在只读内存区，修改是未定义行为
- `sizeof(str)`返回数组大小，`strlen(str)`返回字符串长度（不含`\0`）
- `scanf("%s", str)` 不安全，应使用`fgets`或`std::cin.getline`
