# 字符串编码

## 一、概念说明

字符编码决定了字符如何表示为字节序列。C++支持多种编码：ASCII（1字节）、UTF-8（变长1-4字节）、UTF-16（2或4字节）、UTF-32（固定4字节）。不同平台的`wchar_t`大小不同（Windows为2字节，Linux为4字节）。

## 二、具体用法

### 2.1 字符串字面量前缀

```cpp
// ASCII / UTF-8
const char* s1 = "Hello";            // char
const char* s2 = u8"Hello";          // C++20: char8_t, 之前为char

// 宽字符
const wchar_t* s3 = L"Hello";        // wchar_t（平台相关）

// UTF-16 / UTF-32
const char16_t* s4 = u"Hello";       // UTF-16
const char32_t* s5 = U"Hello";       // UTF-32

// 中文示例
const char* cn = u8"你好";           // UTF-8编码
std::cout << cn << std::endl;         // 输出: 你好（终端需UTF-8支持）
```

### 2.2 宽字符字符串

```cpp
#include <locale>
#include <codecvt>

std::wstring ws = L"Hello";
std::wcout << ws << std::endl;  // 输出: Hello

// Windows常用
// const wchar_t* path = L"C:\\Users\\文件.txt";
```

### 2.3 编码转换（C++17已弃用codecvt）

```cpp
// C++17之前
#include <codecvt>
#include <locale>

std::wstring_convert<std::codecvt_utf8<wchar_t>> converter;

// UTF-8 → wstring
std::wstring wide = converter.from_bytes("Hello 你好");

// wstring → UTF-8
std::string utf8 = converter.to_bytes(wide);

// 注意：C++17标记为deprecated，推荐使用第三方库如ICU
```

### 2.4 字节与字符的区别

```cpp
std::string utf8 = u8"你好世界";
std::cout << "字节数: " << utf8.size() << std::endl;
// 输出: 字节数: 12 (每个中文3字节UTF-8)

// 注意：size()返回字节数，不是字符数
// 正确计算UTF-8字符数需要解析编码
```

## 三、注意事项与常见陷阱

- `std::string`存储的是字节序列，不感知编码
- `wchar_t`在Windows和Linux大小不同，跨平台注意
- `std::codecvt`在C++17标记为弃用
- 终端/文件编码需与程序编码一致，否则显示乱码
- 推荐使用UTF-8作为统一编码，避免编码转换问题
