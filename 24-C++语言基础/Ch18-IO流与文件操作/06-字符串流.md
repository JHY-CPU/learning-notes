# 字符串流

## 一、概念说明

字符串流（`<sstream>`）将字符串作为IO目标，支持类型安全的格式化和解析。三种类型：
- `ostringstream`：向字符串写入（类似sprintf）
- `istringstream`：从字符串读取（类似sscanf）
- `stringstream`：双向读写

```cpp
#include <iostream>
#include <sstream>
#include <string>

int main() {
    // ostringstream: 格式化到字符串
    std::ostringstream oss;
    oss << "年龄: " << 25 << ", 身高: " << 1.75;
    std::string result = oss.str();
    std::cout << result << std::endl;

    // istringstream: 从字符串解析
    std::string data = "42 3.14 hello";
    std::istringstream iss(data);
    int a; double b; std::string s;
    iss >> a >> b >> s;
    std::cout << "a=" << a << ", b=" << b << ", s=" << s << std::endl;

    // stringstream: 双向
    std::stringstream ss;
    ss << 100 << " " << 200;
    int x, y;
    ss >> x >> y;
    std::cout << "x=" << x << ", y=" << y << std::endl;

    return 0;
}
```

**输出：**
```
年龄: 25, 身高: 1.75
a=42, b=3.14, s=hello
x=100, y=200
```

## 二、具体用法

### 2.1 类型转换

```cpp
#include <iostream>
#include <sstream>

// int转string
std::string intToString(int n) {
    std::ostringstream oss;
    oss << n;
    return oss.str();
}

// string转int
int stringToInt(const std::string& s) {
    std::istringstream iss(s);
    int n;
    iss >> n;
    return n;
}

// double转string（指定精度）
std::string doubleToString(double d, int precision) {
    std::ostringstream oss;
    oss << std::fixed << std::setprecision(precision) << d;
    return oss.str();
}

int main() {
    std::cout << intToString(42) << std::endl;
    std::cout << stringToInt("123") << std::endl;
    std::cout << doubleToString(3.14159, 3) << std::endl;

    return 0;
}
```

**输出：**
```
42
123
3.142
```

### 2.2 字符串分割

```cpp
#include <iostream>
#include <sstream>
#include <vector>
#include <string>

std::vector<std::string> split(const std::string& str, char delimiter = ' ') {
    std::vector<std::string> tokens;
    std::istringstream iss(str);
    std::string token;
    while (std::getline(iss, token, delimiter)) {
        if (!token.empty()) tokens.push_back(token);
    }
    return tokens;
}

int main() {
    std::string csv = "apple,banana,,cherry,date";
    auto parts = split(csv, ',');

    std::cout << "分割结果:" << std::endl;
    for (const auto& p : parts) {
        std::cout << "  '" << p << "'" << std::endl;
    }

    return 0;
}
```

**输出：**
```
分割结果:
  'apple'
  'banana'
  'cherry'
  'date'
```

## 三、注意事项与常见陷阱

- **`str()`返回副本**：修改不会影响流的内容。
- **`str(new_string)`替换流内容**：可以重用同一个流。
- **`clear()`清除状态但不清除内容**：重用流时需要先`clear()`再`str("")`。
- **`ostringstream`比`sprintf`更安全**：类型检查、自动内存管理。
- **大字符串频繁拼接用`ostringstream`**：比`+=`效率高。
