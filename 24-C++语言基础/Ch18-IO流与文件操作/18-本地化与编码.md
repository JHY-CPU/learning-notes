# 本地化与编码

## 一、概念说明

`<locale>`库支持国际化：数字/日期格式、货币、字符分类、字符串排序等。宽字符流（`wcout`/`wcin`）和`<codecvt>`（C++17已弃用）用于多字节编码转换。

C++17后推荐使用第三方库（如ICU）处理Unicode，标准库的编码支持有限。

```cpp
#include <iostream>
#include <locale>
#include <iomanip>

int main() {
    // 设置全局locale
    std::locale loc(""); // 系统默认locale
    std::cout.imbue(loc);

    // 不同locale的数字格式
    std::locale us("en_US.UTF-8");
    std::locale de("de_DE.UTF-8");
    std::locale cn("zh_CN.UTF-8");

    double price = 1234567.89;

    std::cout << "默认: " << price << std::endl;
    std::cout.imbue(us);
    std::cout << "美国: " << price << std::endl;
    std::cout.imbue(de);
    std::cout << "德国: " << price << std::endl;

    return 0;
}
```

**输出（取决于系统locale支持）：**
```
默认: 1.23457e+06
美国: 1,234,567.89
德国: 1.234.567,89
```

## 二、具体用法

### 2.1 宽字符流

```cpp
#include <iostream>
#include <locale>
#include <string>

int main() {
    // 设置全局locale以支持宽字符
    std::locale::global(std::locale(""));
    std::wcout.imbue(std::locale());

    // 宽字符输出
    std::wstring msg = L"Hello World";
    std::wcout << msg << std::endl;

    // 宽字符输入
    // std::wstring input;
    // std::wcin >> input;

    // 字符分类
    std::locale loc;
    std::wcout << L"A是字母: " << std::isalpha(L'A', loc) << std::endl;
    std::wcout << L"5是数字: " << std::isdigit(L'5', loc) << std::endl;

    return 0;
}
```

**输出：**
```
Hello World
A是字母: 1
5是数字: 1
```


## 三、注意事项与常见陷阱

- **`<codecvt>`在C++17中已弃用**：应使用第三方库处理Unicode。
- **`wcout`在Windows下行为特殊**：需要`_setmode`设置控制台编码。
- **locale名称因系统而异**：`"en_US.UTF-8"`在Windows可能是`"English_United States"`。
- **默认locale通常是`"C"`**：不支持千位分隔符等格式化。
- **`imbue`只影响单个流**：`locale::global()`设置全局默认。
- **字符串排序受locale影响**：不同语言的排序规则不同。
