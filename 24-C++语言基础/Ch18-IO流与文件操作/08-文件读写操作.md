# 文件读写操作

## 一、概念说明

文件读写分为**文本模式**和**二进制模式**。文本模式自动处理换行符转换（Windows `\r\n` ↔ `\n`），二进制模式不做任何转换。

常用读取方式：`>>`格式化读取、`getline`整行读取、`read`/`write`二进制读写。

```cpp
#include <iostream>
#include <fstream>
#include <string>

int main() {
    // 写入文本文件
    {
        std::ofstream ofs("data.txt");
        ofs << "Alice 90" << std::endl;
        ofs << "Bob 85" << std::endl;
        ofs << "Charlie 92" << std::endl;
    }

    // 逐行读取
    {
        std::ifstream ifs("data.txt");
        std::string line;
        int lineNum = 0;
        while (std::getline(ifs, line)) {
            ++lineNum;
            std::cout << lineNum << ": " << line << std::endl;
        }
    }

    // 格式化读取
    {
        std::ifstream ifs("data.txt");
        std::string name;
        int score;
        while (ifs >> name >> score) {
            std::cout << name << " 得了 " << score << " 分" << std::endl;
        }
    }

    return 0;
}
```

**输出：**
```
1: Alice 90
2: Bob 85
3: Charlie 92
Alice 得了 90 分
Bob 得了 85 分
Charlie 得了 92 分
```

## 二、具体用法

### 2.1 追加写入

```cpp
#include <iostream>
#include <fstream>

int main() {
    // 追加模式
    {
        std::ofstream ofs("log.txt", std::ios::app);
        ofs << "日志1: 启动" << std::endl;
    }
    {
        std::ofstream ofs("log.txt", std::ios::app);
        ofs << "日志2: 运行中" << std::endl;
    }

    // 读取全部内容
    std::ifstream ifs("log.txt");
    std::string content((std::istreambuf_iterator<char>(ifs)),
                        std::istreambuf_iterator<char>());
    std::cout << content;

    return 0;
}
```

**输出：**
```
日志1: 启动
日志2: 运行中
```

### 2.2 整文件读取

```cpp
#include <iostream>
#include <fstream>
#include <sstream>
#include <string>

// 方法1：istreambuf_iterator（最快）
std::string readFile1(const std::string& path) {
    std::ifstream ifs(path);
    return std::string(std::istreambuf_iterator<char>(ifs),
                       std::istreambuf_iterator<char>());
}

// 方法2：stringstream（常用）
std::string readFile2(const std::string& path) {
    std::ifstream ifs(path);
    std::stringstream ss;
    ss << ifs.rdbuf();
    return ss.str();
}

// 方法3：逐行读取
std::string readFile3(const std::string& path) {
    std::ifstream ifs(path);
    std::string result, line;
    while (std::getline(ifs, line)) {
        result += line + '\n';
    }
    return result;
}

int main() {
    // 创建测试文件
    {
        std::ofstream ofs("test.txt");
        ofs << "Hello\nWorld\nC++" << std::endl;
    }

    std::cout << "方法1: " << readFile1("test.txt");
    std::cout << "方法2: " << readFile2("test.txt");
    std::cout << "方法3: " << readFile3("test.txt");

    return 0;
}
```

**输出：**
```
方法1: Hello
World
C++
方法2: Hello
World
C++
方法3: Hello
World
C++
```

## 三、注意事项与常见陷阱

- **`getline`不包含换行符**：需要的话需手动添加。
- **读取失败时检查`eof()`和`fail()`**：区分正常结束和错误。
- **二进制模式用`ios::binary`**：避免换行符自动转换。
- **大文件不要一次读入内存**：使用流式处理或内存映射。
- **Windows文本模式自动转换`\r\n`为`\n`**：二进制模式不转换。
