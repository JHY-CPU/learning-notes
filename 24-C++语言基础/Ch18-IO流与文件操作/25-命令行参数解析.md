# 命令行参数解析

## 一、概念说明

C++的`main`函数接收命令行参数：`int main(int argc, char* argv[])`。`argc`是参数个数（包括程序名），`argv`是参数字符串数组。

手动解析适合简单场景，复杂参数推荐使用第三方库如`getopt`（POSIX）、`cxxopts`、`CLI11`等。

```cpp
#include <iostream>
#include <string>
#include <cstring>

int main(int argc, char* argv[]) {
    std::cout << "程序名: " << argv[0] << std::endl;
    std::cout << "参数个数: " << argc << std::endl;

    // 手动解析简单参数
    std::string inputFile;
    std::string outputFile;
    bool verbose = false;
    int count = 1;

    for (int i = 1; i < argc; ++i) {
        std::string arg = argv[i];

        if (arg == "-v" || arg == "--verbose") {
            verbose = true;
        } else if (arg == "-i" || arg == "--input") {
            if (i + 1 < argc) inputFile = argv[++i];
        } else if (arg == "-o" || arg == "--output") {
            if (i + 1 < argc) outputFile = argv[++i];
        } else if (arg == "-n") {
            if (i + 1 < argc) count = std::stoi(argv[++i]);
        } else if (arg == "-h" || arg == "--help") {
            std::cout << "用法: program -i <输入> -o <输出> [-v] [-n <数量>]" << std::endl;
            return 0;
        }
    }

    std::cout << "输入: " << inputFile << std::endl;
    std::cout << "输出: " << outputFile << std::endl;
    std::cout << "详细: " << verbose << std::endl;
    std::cout << "数量: " << count << std::endl;

    return 0;
}
```

**输出（运行`./program -i data.txt -o result.txt -v -n 5`）：**
```
程序名: ./program
参数个数: 7
输入: data.txt
输出: result.txt
详细: 1
数量: 5
```

## 二、具体用法

### 2.1 使用POSIX getopt

```cpp
#include <iostream>
#include <unistd.h>  // POSIX only

int main(int argc, char* argv[]) {
    int opt;
    std::string input, output;
    bool verbose = false;

    // optstring: "i:o:vh" - i和o需要参数, v和h不需要
    while ((opt = getopt(argc, argv, "i:o:vh")) != -1) {
        switch (opt) {
            case 'i': input = optarg; break;
            case 'o': output = optarg; break;
            case 'v': verbose = true; break;
            case 'h':
                std::cout << "Usage: " << argv[0] << " -i <input> -o <output> [-v]" << std::endl;
                return 0;
            default:
                std::cerr << "未知选项" << std::endl;
                return 1;
        }
    }

    std::cout << "input=" << input << ", output=" << output
              << ", verbose=" << verbose << std::endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

- **`argv[0]`是程序名**：不是第一个参数。
- **参数中的空格需要引号**：`./prog "hello world"` 作为单个参数。
- **`getopt`会重排`argv`**：选项在前，非选项参数在后。
- **`optind`指向第一个非选项参数**：`argv[optind]`是剩余参数。
- **Windows没有`getopt`**：使用`GetCommandLine`或第三方库。
