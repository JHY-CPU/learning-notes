# 文件定位

## 一、概念说明

文件定位用于在文件中移动读写位置。C++提供了两组指针：
- **读指针（get pointer）**：`seekg`移动、`tellg`获取位置
- **写指针（put pointer）**：`seekp`移动、`tellp`获取位置

定位方式：`ios::beg`（从头）、`ios::cur`（当前位置）、`ios::end`（从尾）。

```cpp
#include <iostream>
#include <fstream>

int main() {
    // 创建文件
    {
        std::ofstream ofs("data.txt");
        ofs << "0123456789ABCDEFGHIJ";
    }

    // 定位读取
    {
        std::ifstream ifs("data.txt");

        // 获取文件大小
        ifs.seekg(0, std::ios::end);
        auto fileSize = ifs.tellg();
        std::cout << "文件大小: " << fileSize << " 字节" << std::endl;

        // 回到开头
        ifs.seekg(0, std::ios::beg);

        // 读取前5个字符
        char buf[6] = {};
        ifs.read(buf, 5);
        std::cout << "前5个: " << buf << std::endl;

        // 定位到第10个字节
        ifs.seekg(10, std::ios::beg);
        char c;
        ifs.get(c);
        std::cout << "位置10: " << c << std::endl;

        // 从当前位置后移3个字节
        ifs.seekg(3, std::ios::cur);
        ifs.get(c);
        std::cout << "位置14: " << c << std::endl;
    }

    return 0;
}
```

**输出：**
```
文件大小: 20 字节
前5个: 01234
位置10: A
位置14: E
```

## 二、具体用法

### 2.1 随机访问记录

```cpp
#include <iostream>
#include <fstream>

struct Record {
    int id;
    double value;
};

void createFile(const char* name) {
    std::ofstream ofs(name, std::ios::binary);
    for (int i = 0; i < 5; ++i) {
        Record r = {i, i * 1.5};
        ofs.write(reinterpret_cast<const char*>(&r), sizeof(r));
    }
}

Record readRecord(const char* name, int index) {
    std::ifstream ifs(name, std::ios::binary);
    // 定位到第index条记录
    ifs.seekg(index * sizeof(Record), std::ios::beg);
    Record r;
    ifs.read(reinterpret_cast<char*>(&r), sizeof(r));
    return r;
}

int main() {
    createFile("records.dat");

    // 随机读取第3条记录（索引2）
    auto r = readRecord("records.dat", 2);
    std::cout << "记录2: id=" << r.id << ", value=" << r.value << std::endl;

    // 读取最后一条
    r = readRecord("records.dat", 4);
    std::cout << "记录4: id=" << r.id << ", value=" << r.value << std::endl;

    return 0;
}
```

**输出：**
```
记录2: id=2, value=3
记录4: id=4, value=6
```

### 2.2 修改文件中间内容

```cpp
#include <iostream>
#include <fstream>

int main() {
    // 创建文件
    {
        std::ofstream ofs("editable.txt");
        ofs << "ABCDEFGHIJ";
    }

    // 修改第3-5个字符
    {
        std::fstream fs("editable.txt", std::ios::in | std::ios::out);
        fs.seekp(2, std::ios::beg);
        fs.write("XYZ", 3);
    }

    // 验证
    std::ifstream ifs("editable.txt");
    std::string content;
    ifs >> content;
    std::cout << "修改后: " << content << std::endl;

    return 0;
}
```

**输出：**
```
修改后: ABXYZFGHIJ
```

## 三、注意事项与常见陷阱

- **文本模式下`seekg`/`tellg`不可靠**：因换行符转换，位置可能不准确。用二进制模式。
- **`seekg`和`seekp`在`fstream`中独立维护**：但某些实现中可能同步。
- **超出文件大小的定位**：读取会失败，写入会在中间填充零字节。
- **`tellg()`返回`-1`表示失败**：通常是流处于错误状态。
- **追加模式下`seekp`无效**：`ios::app`强制在尾部写入。
