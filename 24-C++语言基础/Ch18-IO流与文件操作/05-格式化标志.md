# 格式化标志

## 一、概念说明

`ios_base`维护一组格式化标志（`fmtflags`），控制流的格式化行为。可以用`setiosflags`/`resetiosflags`操纵符或`setf`/`unsetf`成员函数设置。

常用标志：`left`、`right`、`scientific`、`fixed`、`hex`、`oct`、`dec`、`showbase`、`showpoint`、`uppercase`、`boolalpha`、`unitbuf`。

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 使用setiosflags/resetiosflags
    std::cout << std::setiosflags(std::ios::left | std::ios::showbase);
    std::cout << std::hex << std::setw(10) << 255 << std::endl;

    std::cout << std::resetiosflags(std::ios::left);
    std::cout << std::setiosflags(std::ios::right);
    std::cout << std::setw(10) << 255 << std::endl;

    // 使用成员函数setf/unsetf
    std::cout.setf(std::ios::showpos);
    std::cout << std::dec << 42 << std::endl; // +42
    std::cout.unsetf(std::ios::showpos);

    // 查询当前标志
    auto flags = std::cout.flags();
    std::cout << "当前标志值: " << flags << std::endl;

    return 0;
}
```

**输出：**
```
0xff
       0xff
+42
当前标志值: 16386
```

## 二、具体用法

### 2.1 保存和恢复格式状态

```cpp
#include <iostream>
#include <iomanip>

class FormatGuard {
    std::ostream& os;
    std::ios::fmtflags flags;
    std::streamsize precision;
    std::streamsize width;
    char fill;
public:
    FormatGuard(std::ostream& s)
        : os(s), flags(s.flags()), precision(s.precision()),
          width(s.width()), fill(s.fill()) {}
    ~FormatGuard() {
        os.flags(flags);
        os.precision(precision);
        os.width(width);
        os.fill(fill);
    }
};

int main() {
    std::cout << "默认: " << 3.14159 << std::endl;

    {
        FormatGuard guard(std::cout);
        std::cout << std::fixed << std::setprecision(2) << std::showpos;
        std::cout << "修改后: " << 3.14159 << std::endl;
    } // guard析构，恢复格式

    std::cout << "恢复后: " << 3.14159 << std::endl;

    return 0;
}
```

**输出：**
```
默认: 3.14159
修改后: +3.14
恢复后: 3.14159
```

### 2.2 常用fmtflags一览

| 标志 | 作用 | 示例 |
|------|------|------|
| `ios::left` | 左对齐 | `"Hi      "` |
| `ios::right` | 右对齐 | `"      Hi"` |
| `ios::dec` | 十进制 | `255` |
| `ios::hex` | 十六进制 | `ff` |
| `ios::oct` | 八进制 | `377` |
| `ios::showbase` | 显示进制前缀 | `0xff` |
| `ios::showpos` | 正数显示+号 | `+42` |
| `ios::uppercase` | 大写 | `0XFF` |
| `ios::fixed` | 固定小数 | `3.140000` |
| `ios::scientific` | 科学计数 | `3.14e+00` |
| `ios::boolalpha` | true/false | `true` |
| `ios::showpoint` | 显示小数点 | `3.` |

## 三、注意事项与常见陷阱

- **`flags()`返回当前全部标志**：可以用它保存和恢复状态。
- **`setf`不覆盖已有标志**：`flags(new_flags)`会完全替换。
- **`setiosflags`和`resetiosflags`是对称的操纵符**：更方便使用。
- **格式标志对所有后续输出生效**：直到被修改。
- **用RAII守卫（FormatGuard）保护格式状态**：避免函数调用污染全局格式。
