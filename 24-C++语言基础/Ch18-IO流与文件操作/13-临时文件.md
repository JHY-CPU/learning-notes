# 临时文件

## 一、概念说明

临时文件用于存储程序运行时的临时数据，程序结束时应清理。C标准库提供`tmpfile()`和`tmpnam()`，C++17的`<filesystem>`提供更现代的方案。

`tmpfile()`创建临时文件并返回`FILE*`，关闭时自动删除。`tmpnam()`生成临时文件名但不创建文件（有安全风险）。

```cpp
#include <iostream>
#include <cstdio>
#include <filesystem>

int main() {
    // tmpfile: 自动删除的临时文件
    FILE* tmp = std::tmpfile();
    if (tmp) {
        std::fputs("临时数据\n", tmp);
        std::rewind(tmp);

        char buf[256];
        std::fgets(buf, sizeof(buf), tmp);
        std::cout << "tmpfile内容: " << buf;

        std::fclose(tmp); // 自动删除
        std::cout << "tmpfile已自动删除" << std::endl;
    }

    // C++17 filesystem临时路径
    auto tempDir = std::filesystem::temp_directory_path();
    std::cout << "系统临时目录: " << tempDir << std::endl;

    return 0;
}
```

**输出：**
```
tmpfile内容: 临时数据
tmpfile已自动删除
系统临时目录: C:\Users\...\Temp
```

## 二、具体用法

### 2.1 创建临时文件路径

```cpp
#include <iostream>
#include <fstream>
#include <filesystem>
#include <random>
#include <sstream>

// 安全的临时文件名生成
std::string generateTempPath(const std::string& prefix = "tmp",
                              const std::string& ext = ".txt") {
    auto tempDir = std::filesystem::temp_directory_path();

    std::random_device rd;
    std::mt19937_64 gen(rd());
    std::uniform_int_distribution<long long> dist;

    std::ostringstream oss;
    oss << tempDir.string() << "/" << prefix << "_" << dist(gen) << ext;
    return oss.str();
}

// RAII临时文件
class TempFile {
    std::string path_;
public:
    TempFile(const std::string& prefix = "tmp")
        : path_(generateTempPath(prefix)) {
        std::cout << "创建临时文件: " << path_ << std::endl;
    }

    ~TempFile() {
        if (std::filesystem::exists(path_)) {
            std::filesystem::remove(path_);
            std::cout << "删除临时文件: " << path_ << std::endl;
        }
    }

    const std::string& path() const { return path_; }
};

int main() {
    {
        TempFile tmp("myapp");
        std::ofstream ofs(tmp.path());
        ofs << "临时内容" << std::endl;
    } // 自动删除

    std::cout << "临时文件已清理" << std::endl;
    return 0;
}
```

**输出：**
```
创建临时文件: ...\tmp_123456789.txt
删除临时文件: ...\tmp_123456789.txt
临时文件已清理
```

## 三、注意事项与常见陷阱

- **`tmpnam`有竞态安全问题**：生成名字和创建文件之间可能被抢占。
- **`tmpfile`关闭时自动删除**：适合用完即弃的场景。
- **临时文件应设合适权限**：避免其他用户读取敏感数据。
- **程序异常退出时临时文件可能残留**：用RAII或信号处理清理。
- **`std::filesystem::temp_directory_path()`跨平台获取临时目录**。
