# 格式化输出

## 一、概念说明

C++ IO流提供了丰富的格式化控制，包括：宽度（`setw`）、精度（`setprecision`）、填充（`setfill`）、进制（`hex`/`oct`/`dec`）、对齐等。这些定义在`<iomanip>`和`<ios>`头文件中。

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 设置宽度和填充
    std::cout << "|" << std::setw(10) << 42 << "|" << std::endl;
    std::cout << "|" << std::setw(10) << std::setfill('*') << 42 << "|" << std::endl;

    // 浮点精度
    double pi = 3.141592653589793;
    std::cout << "默认精度: " << pi << std::endl;
    std::cout << "精度2: " << std::setprecision(2) << pi << std::endl;
    std::cout << "精度10: " << std::setprecision(10) << pi << std::endl;

    // 固定和科学计数法
    std::cout << "fixed: " << std::fixed << std::setprecision(4) << 12345.6789 << std::endl;
    std::cout << "scientific: " << std::scientific << 12345.6789 << std::endl;
    std::cout << "defaultfloat: " << std::defaultfloat << 12345.6789 << std::endl;

    return 0;
}
```

**输出：**
```
|        42|
|********42|
默认精度: 3.14159
精度2: 3.1
精度10: 3.141592654
fixed: 12345.6789
scientific: 1.234568e+04
defaultfloat: 12345.7
```

## 二、具体用法

### 2.1 表格格式化输出

```cpp
#include <iostream>
#include <iomanip>
#include <vector>

struct Student {
    std::string name;
    int score;
};

int main() {
    std::vector<Student> students = {
        {"张三", 95}, {"李四", 87}, {"王五", 100}, {"赵六", 72}
    };

    // 表头
    std::cout << std::left
              << std::setw(10) << "姓名"
              << std::setw(8) << "分数"
              << "等级" << std::endl;
    std::cout << std::setfill('-') << std::setw(24) << "" << std::setfill(' ') << std::endl;

    // 数据行
    for (const auto& s : students) {
        char grade = s.score >= 90 ? 'A' : s.score >= 80 ? 'B' : s.score >= 60 ? 'C' : 'D';
        std::cout << std::left << std::setw(10) << s.name
                  << std::right << std::setw(8) << s.score
                  << std::setw(6) << grade << std::endl;
    }

    return 0;
}
```

**输出：**
```
姓名      分数    等级
------------------------
张三          95     A
李四          87     B
王五         100     A
赵六          72     C
```

### 2.2 数字格式化

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 十六进制大写 + 前缀
    std::cout << std::uppercase << std::showbase << std::hex;
    std::cout << "0xFF = " << 255 << std::endl;

    // 百分比格式
    double ratio = 0.856;
    std::cout << std::fixed << std::setprecision(1);
    std::cout << "完成度: " << ratio * 100 << "%" << std::endl;

    // 正号显示
    std::cout << std::showpos;
    std::cout << "+42: " << 42 << std::endl;
    std::cout << "-42: " << -42 << std::endl;
    std::cout << std::noshowpos;

    return 0;
}
```

**输出：**
```
0xFF = 0XFF
完成度: 85.6%
+42: +42
-42: -42
```

## 三、注意事项与常见陷阱

- **`setw`只对下一次输出有效**：需每次输出前重新设置。
- **`setprecision`是持久的**：会一直影响后续浮点输出。
- **`fixed`/`scientific`是状态标志**：切换后需要`defaultfloat`恢复。
- **`setfill`也是持久的**：用完后记得恢复默认填充字符。
- **`hex`不会自动加前缀**：需要`showbase`才会显示`0x`。
