# 流缓冲区

## 一、概念说明

`std::streambuf`是流的底层缓冲区实现，负责实际的字符读写。流对象（`istream`/`ostream`）通过缓冲区与外部设备（文件、字符串、控制台）交互。

自定义`streambuf`可以实现：日志过滤、加密传输、网络IO、自定义格式化等。

```cpp
#include <iostream>
#include <streambuf>

// 自定义缓冲区：将输出转为大写
class UpperBuf : public std::streambuf {
    std::streambuf* sink;
protected:
    int_type overflow(int_type c) override {
        if (c != traits_type::eof()) {
            char ch = static_cast<char>(c);
            if (ch >= 'a' && ch <= 'z') ch = ch - 'a' + 'A';
            sink->sputc(ch);
        }
        return c;
    }
public:
    UpperBuf(std::streambuf* s) : sink(s) {}
};

int main() {
    UpperBuf upper(std::cout.rdbuf()); // 获取cout的缓冲区
    std::ostream os(&upper);

    os << "hello world!" << std::endl;
    os << "this is uppercase output" << std::endl;

    return 0;
}
```

**输出：**
```
HELLO WORLD!
THIS IS UPPERCASE OUTPUT
```

## 二、具体用法

### 2.1 重定向流缓冲区

```cpp
#include <iostream>
#include <sstream>

int main() {
    // 保存原始缓冲区
    auto* originalBuf = std::cout.rdbuf();

    // 重定向cout到字符串流
    std::stringstream ss;
    std::cout.rdbuf(ss.rdbuf());

    std::cout << "这不会显示在屏幕上" << std::endl;

    // 恢复原始缓冲区
    std::cout.rdbuf(originalBuf);

    std::cout << "从字符串流获取: " << ss.str();
    std::cout << "这会显示在屏幕上" << std::endl;

    return 0;
}
```

**输出：**
```
从字符串流获取: 这不会显示在屏幕上
这会显示在屏幕上
```

### 2.2 带时间戳的缓冲区

```cpp
#include <iostream>
#include <streambuf>
#include <chrono>
#include <ctime>
#include <iomanip>

class TimestampBuf : public std::streambuf {
    std::streambuf* sink;
    bool atLineStart = true;

protected:
    int_type overflow(int_type c) override {
        if (c != traits_type::eof()) {
            if (atLineStart) {
                auto now = std::chrono::system_clock::now();
                auto t = std::chrono::system_clock::to_time_t(now);
                std::tm tm = *std::localtime(&t);

                std::ostringstream oss;
                oss << "[" << std::put_time(&tm, "%H:%M:%S") << "] ";
                std::string ts = oss.str();
                for (char ch : ts) sink->sputc(ch);

                atLineStart = false;
            }
            sink->sputc(static_cast<char>(c));
            if (c == '\n') atLineStart = true;
        }
        return c;
    }

public:
    TimestampBuf(std::streambuf* s) : sink(s) {}
};

int main() {
    TimestampBuf tsBuf(std::cout.rdbuf());
    std::ostream logger(&tsBuf);

    logger << "应用启动" << std::endl;
    logger << "加载配置" << std::endl;
    logger << "就绪" << std::endl;

    return 0;
}
```

**输出（示例）：**
```
[10:30:00] 应用启动
[10:30:00] 加载配置
[10:30:00] 就绪
```

## 三、注意事项与常见陷阱

- **`overflow`在缓冲区满或`flush`时调用**：是核心重写方法。
- **`sync`方法用于刷新缓冲区**：需正确实现。
- **`underflow`用于输入缓冲区**：`overflow`用于输出缓冲区。
- **重定向`cout`后`cerr`不受影响**：它们使用不同的缓冲区。
- **析构自定义缓冲区前恢复原缓冲区**：避免悬挂指针。
