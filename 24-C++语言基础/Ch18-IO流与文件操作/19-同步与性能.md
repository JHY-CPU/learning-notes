# 同步与性能

## 一、概念说明

C++的IO流默认与C的`stdio`同步，以确保`cout`和`printf`可以安全混用。但同步有性能开销，在高性能场景中应关闭。

关键函数：
- `std::ios::sync_with_stdio(false)`：关闭C/C++ IO同步
- `cin.tie(nullptr)`：解除`cin`与`cout`的绑定（每次`cin >>`前不再自动刷新`cout`）

```cpp
#include <iostream>
#include <chrono>

void benchmarkSync() {
    auto start = std::chrono::high_resolution_clock::now();

    for (int i = 0; i < 100000; ++i) {
        std::cout << "";
    }

    auto end = std::chrono::high_resolution_clock::now();
    auto ms = std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count();
    std::cout << "同步开启耗时: " << ms << "ms" << std::endl;
}

void benchmarkNoSync() {
    std::ios::sync_with_stdio(false);

    auto start = std::chrono::high_resolution_clock::now();

    for (int i = 0; i < 100000; ++i) {
        std::cout << "";
    }

    auto end = std::chrono::high_resolution_clock::now();
    auto ms = std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count();
    std::cout << "同步关闭耗时: " << ms << "ms" << std::endl;
}

int main() {
    benchmarkSync();
    benchmarkNoSync();
    return 0;
}
```

**输出（示例）：**
```
同步开启耗时: 15ms
同步关闭耗时: 3ms
```

## 二、具体用法

### 2.1 竞赛编程中的快速IO

```cpp
#include <iostream>
#include <vector>

int main() {
    // 竞赛编程的快速IO设置
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);
    std::cout.tie(nullptr);

    int n;
    std::cin >> n;

    std::vector<int> arr(n);
    for (int i = 0; i < n; ++i) {
        std::cin >> arr[i];
    }

    long long sum = 0;
    for (int x : arr) sum += x;

    std::cout << sum << '\n'; // 用'\n'不用endl

    return 0;
}
```

**输出（输入"5\n1 2 3 4 5"）：**
```
15
```

### 2.2 性能对比

```cpp
#include <iostream>
#include <sstream>
#include <chrono>

// 使用endl（每次刷新）
void testEndl() {
    auto start = std::chrono::high_resolution_clock::now();
    std::ostringstream oss;
    for (int i = 0; i < 100000; ++i) {
        oss << i << std::endl;
    }
    auto end = std::chrono::high_resolution_clock::now();
    auto us = std::chrono::duration_cast<std::chrono::microseconds>(end - start).count();
    std::cout << "endl: " << us << "us" << std::endl;
}

// 使用'\n'（不刷新）
void testNewline() {
    auto start = std::chrono::high_resolution_clock::now();
    std::ostringstream oss;
    for (int i = 0; i < 100000; ++i) {
        oss << i << '\n';
    }
    auto end = std::chrono::high_resolution_clock::now();
    auto us = std::chrono::duration_cast<std::chrono::microseconds>(end - start).count();
    std::cout << "'\\n': " << us << "us" << std::endl;
}

int main() {
    testEndl();
    testNewline();
    return 0;
}
```

**输出（示例）：**
```
endl: 8500us
'\n': 2100us
```

## 三、注意事项与常见陷阱

- **关闭同步后不能混用C/C++ IO**：`printf`和`cout`输出顺序不确定。
- **`sync_with_stdio`只在程序开始调用一次**：之后再调用可能无效。
- **`tie(nullptr)`解除绑定后需手动flush**：如需要确保提示语先显示。
- **`endl` = `\n` + `flush`**：性能差距可达数倍。
- **文件IO不需要此设置**：文件流没有同步开销。
