# 二进制文件操作

## 一、概念说明

二进制模式直接读写字节，不做任何转换，适合存储结构化数据、图片、音频等。使用`read()`和`write()`方法，配合`reinterpret_cast`。

```cpp
#include <iostream>
#include <fstream>
#include <cstring>

struct Record {
    int id;
    double value;
    char name[32];
};

int main() {
    const char* filename = "records.dat";

    // 写入二进制文件
    {
        std::ofstream ofs(filename, std::ios::binary);
        Record records[2] = {
            {1, 3.14, "记录A"},
            {2, 2.71, "记录B"}
        };
        ofs.write(reinterpret_cast<const char*>(records), sizeof(records));
    }

    // 读取二进制文件
    {
        std::ifstream ifs(filename, std::ios::binary);
        Record r;
        while (ifs.read(reinterpret_cast<char*>(&r), sizeof(Record))) {
            std::cout << "ID=" << r.id << ", 值=" << r.value
                      << ", 名称=" << r.name << std::endl;
        }
    }

    return 0;
}
```

**输出：**
```
ID=1, 值=3.14, 名称=记录A
ID=2, 值=2.71, 名称=记录B
```

## 二、具体用法

### 2.1 读写单个对象

```cpp
#include <iostream>
#include <fstream>
#include <vector>

struct Point {
    float x, y, z;
};

void writePoints(const std::string& path, const std::vector<Point>& points) {
    std::ofstream ofs(path, std::ios::binary);
    size_t count = points.size();
    ofs.write(reinterpret_cast<const char*>(&count), sizeof(count));
    ofs.write(reinterpret_cast<const char*>(points.data()), count * sizeof(Point));
}

std::vector<Point> readPoints(const std::string& path) {
    std::ifstream ifs(path, std::ios::binary);
    size_t count;
    ifs.read(reinterpret_cast<char*>(&count), sizeof(count));
    std::vector<Point> points(count);
    ifs.read(reinterpret_cast<char*>(points.data()), count * sizeof(Point));
    return points;
}

int main() {
    std::vector<Point> data = {{1.0f, 2.0f, 3.0f}, {4.0f, 5.0f, 6.0f}, {7.0f, 8.0f, 9.0f}};

    writePoints("points.dat", data);

    auto loaded = readPoints("points.dat");
    for (const auto& p : loaded) {
        std::cout << "(" << p.x << ", " << p.y << ", " << p.z << ")" << std::endl;
    }

    return 0;
}
```

**输出：**
```
(1, 2, 3)
(4, 5, 6)
(7, 8, 9)
```

### 2.2 文件尾部检测

```cpp
#include <iostream>
#include <fstream>

int main() {
    // 写入
    {
        std::ofstream ofs("nums.dat", std::ios::binary);
        int nums[] = {10, 20, 30, 40, 50};
        ofs.write(reinterpret_cast<const char*>(nums), sizeof(nums));
    }

    // 读取直到EOF
    {
        std::ifstream ifs("nums.dat", std::ios::binary);
        int val;
        int count = 0;
        while (ifs.read(reinterpret_cast<char*>(&val), sizeof(int))) {
            std::cout << "val[" << count++ << "] = " << val << std::endl;
        }
        std::cout << "共读取 " << count << " 个整数" << std::endl;
    }

    return 0;
}
```

**输出：**
```
val[0] = 10
val[1] = 20
val[2] = 30
val[3] = 40
val[4] = 50
共读取 5 个整数
```

## 三、注意事项与常见陷阱

- **必须使用`ios::binary`模式**：否则文本模式会转换换行符。
- **结构体包含指针时不能直接写入**：指针值无效，需序列化。
- **不同平台/编译器的结构体布局可能不同**：注意padding和对齐。
- **字节序（大小端）问题**：跨平台时需要统一字节序。
- **`read`/`write`不检查数据有效性**：需自行验证。
- **`gcount()`返回实际读取的字节数**：部分读取时很有用。
