# 对象生命周期管理

## 一、概念说明

C++中对象的生命周期由其存储类别和所有权语义决定：
- **自动存储**：栈上局部变量，作用域结束自动销毁
- **动态存储**：堆上对象，由`new`/`delete`或智能指针管理
- **静态存储**：全局/静态变量，程序启动到结束
- **线程存储**：`thread_local`变量，线程开始到结束

所有权语义定义了"谁负责释放对象"：
- **独占所有权**（`unique_ptr`）：只有一个拥有者
- **共享所有权**（`shared_ptr`）：多个拥有者，最后一个释放
- **弱引用**（`weak_ptr`）：观察但不拥有

```cpp
#include <iostream>
#include <memory>

class Resource {
    std::string name;
public:
    Resource(std::string n) : name(std::move(n)) {
        std::cout << "Resource '" << name << "' 创建" << std::endl;
    }
    ~Resource() { std::cout << "Resource '" << name << "' 销毁" << std::endl; }
    const std::string& getName() const { return name; }
};

void lifecycleDemo() {
    // 1. 自动存储 - 栈上
    Resource stackRes("栈对象");

    // 2. 独占所有权 - unique_ptr
    auto uniqueRes = std::make_unique<Resource>("独占对象");

    // 3. 共享所有权 - shared_ptr
    auto sharedRes = std::make_shared<Resource>("共享对象");
    auto sharedCopy = sharedRes; // 共享

    // 4. 弱引用 - weak_ptr
    std::weak_ptr<Resource> weakRes = sharedRes;

    std::cout << "共享引用计数: " << sharedRes.use_count() << std::endl;
    // 离开作用域：栈对象 -> 独占对象 -> 共享对象（计数归零）
}

int main() {
    lifecycleDemo();
    std::cout << "函数返回，所有对象已释放" << std::endl;
    return 0;
}
```

**输出：**
```
Resource '栈对象' 创建
Resource '独占对象' 创建
Resource '共享对象' 创建
共享引用计数: 2
Resource '共享对象' 销毁
Resource '独占对象' 销毁
Resource '栈对象' 销毁
函数返回，所有对象已释放
```

## 二、具体用法

### 2.1 所有权转移模式

```cpp
#include <iostream>
#include <memory>

class Connection {
    int id;
public:
    Connection(int i) : id(i) {
        std::cout << "Connection(" << id << ") 建立" << std::endl;
    }
    ~Connection() { std::cout << "Connection(" << id << ") 断开" << std::endl; }
    int getId() const { return id; }
};

// 工厂：创建并返回独占所有权
std::unique_ptr<Connection> createConnection(int id) {
    return std::make_unique<Connection>(id);
}

// 接收独占所有权
void process(std::unique_ptr<Connection> conn) {
    std::cout << "处理连接 #" << conn->getId() << std::endl;
    // conn在此销毁
}

// 不获取所有权（借用）
void inspect(Connection* conn) {
    std::cout << "检查连接 #" << conn->getId() << std::endl;
    // 不负责释放
}

int main() {
    auto conn = createConnection(1);
    inspect(conn.get()); // 借用
    process(std::move(conn)); // 转移所有权
    // conn现在为空
    return 0;
}
```

**输出：**
```
Connection(1) 建立
检查连接 #1
处理连接 #1
Connection(1) 断开
```

## 三、注意事项与常见陷阱

- **明确所有权语义**：函数参数和返回值的文档应说明是否获取所有权。
- **栈上对象不要用`delete`释放**：栈对象由编译器管理。
- **`shared_ptr`的引用计数不等于"被引用次数"**：`weak_ptr`不计入。
- **移动语义是所有权转移的核心**：`std::move`不移动任何东西，只是类型转换。
- **`std::enable_shared_from_this`允许对象安全获取自身的`shared_ptr`**。
- **避免在容器中存储裸指针**：除非明确不需要管理生命周期。
