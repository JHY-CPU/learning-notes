# 智能指针与数组

## 一、概念说明

C++11中`unique_ptr`支持数组形式`unique_ptr<T[]>`，自动调用`delete[]`。C++14起`shared_ptr`也支持数组，但需要指定删除器或直接使用`shared_ptr<T[]>`(C++17)。

`make_unique<T[]>`(C++14)和`make_shared<T[]>`(C++20)分别用于创建数组智能指针。

```cpp
#include <iostream>
#include <memory>

int main() {
    // unique_ptr数组（C++11起支持）
    auto arr = std::make_unique<int[]>(5);
    for (int i = 0; i < 5; ++i) {
        arr[i] = (i + 1) * 10;
    }

    std::cout << "unique_ptr数组: ";
    for (int i = 0; i < 5; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;

    // shared_ptr数组（C++17起支持T[]形式）
    std::shared_ptr<int[]> sp(new int[3]{1, 2, 3});
    std::cout << "shared_ptr数组: ";
    for (int i = 0; i < 3; ++i) {
        std::cout << sp[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}
```

**输出：**
```
unique_ptr数组: 10 20 30 40 50
shared_ptr数组: 1 2 3
```

## 二、具体用法

### 2.1 自定义删除器管理数组

```cpp
#include <iostream>
#include <memory>

struct MyClass {
    int id;
    MyClass(int i) : id(i) { std::cout << "MyClass(" << id << ") 构造" << std::endl; }
    ~MyClass() { std::cout << "MyClass(" << id << ") 析构" << std::endl; }
};

int main() {
    // C++14方式：shared_ptr + 自定义删除器
    std::shared_ptr<MyClass> sp(
        new MyClass[3]{MyClass(1), MyClass(2), MyClass(3)},
        [](MyClass* p) { delete[] p; }
    );

    std::cout << "shared_ptr持有数组，引用计数: " << sp.use_count() << std::endl;

    // 注意：不能用sp[i]访问，因为底层是单个指针
    // 需要自行管理偏移量

    return 0;
}
```

**输出：**
```
MyClass(1) 构造
MyClass(2) 构造
MyClass(3) 构造
shared_ptr持有数组，引用计数: 1
MyClass(3) 析构
MyClass(2) 析构
MyClass(1) 析构
```

### 2.2 unique_ptr数组的高级用法

```cpp
#include <iostream>
#include <memory>
#include <algorithm>

int main() {
    // 创建并填充
    size_t n = 5;
    auto data = std::make_unique<double[]>(n);

    for (size_t i = 0; i < n; ++i) {
        data[i] = (i + 1) * 1.5;
    }

    // 排序需要get()获取裸指针
    std::sort(data.get(), data.get() + n, std::greater<double>());

    std::cout << "降序排列: ";
    for (size_t i = 0; i < n; ++i) {
        std::cout << data[i] << " ";
    }
    std::cout << std::endl;

    // reset释放
    data.reset();
    std::cout << "已释放: " << (data == nullptr) << std::endl;

    return 0;
}
```

**输出：**
```
降序排列: 7.5 6 4.5 3 1.5
已释放: 1
```

## 三、注意事项与常见陷阱

- **`unique_ptr<T[]>`支持`[]`下标访问**：`shared_ptr<T[]>`在C++17起支持。
- **`get()`返回的裸指针可用于STL算法**：如`sort(data.get(), data.get()+n)`。
- **不能用`make_shared`创建数组**（C++20之前）：需手动指定删除器。
- **数组智能指针的`sizeof`与单元素相同**：不额外存储数组大小。
- **推荐使用`std::vector`代替数组智能指针**：除非需要固定生命周期控制。
