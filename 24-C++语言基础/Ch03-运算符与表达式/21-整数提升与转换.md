# 整数提升与转换

## 一、概念说明

C++在混合类型的运算中自动进行**整数提升**和**算术转换**。小整数类型提升为`int`，不同大小的类型按规则转换为公共类型。

## 二、具体用法

### 2.1 整数提升

```cpp
#include <iostream>
using namespace std;

int main() {
    // 整数提升：小于int的类型提升为int
    char c = 'A';
    short s = 100;
    bool b = true;

    // char和short在运算时提升为int
    auto result1 = c + s;   // int + int = int
    cout << "char + short 的类型大小: " << sizeof(result1) << endl;  // 4

    // bool提升为int（false=0, true=1）
    auto result2 = b + 5;
    cout << "true + 5 = " << result2 << endl;  // 6

    // unsigned char也提升为int
    unsigned char uc = 255;
    cout << "unsigned char 255 + 1 = " << (uc + 1) << endl;  // 256（int）

    return 0;
}
```

输出：
```
char + short 的类型大小: 4
true + 5 = 6
unsigned char 255 + 1 = 256
```

### 2.2 Usual Arithmetic Conversions

```cpp
#include <iostream>
using namespace std;

int main() {
    // 常规算术转换：决定混合类型的公共类型
    // 规则（简化）：
    // 1. 有long double → 另一个转为long double
    // 2. 有double → 另一个转为double
    // 3. 有float → 另一个转为float
    // 4. 都是整数：先整数提升，然后：
    //    a. 两个都是signed或都是unsigned → 小的转为大的
    //    b. unsigned ≥ signed → signed转为unsigned
    //    c. signed > unsigned且signed能表示所有unsigned值 → unsigned转为signed

    int i = -1;
    unsigned int u = 1;

    // 有符号和无符号混合：int转为unsigned int
    // -1变成UINT_MAX
    cout << "int vs unsigned:" << endl;
    cout << "  -1 < 1? " << (i < u ? "true" : "false") << endl;  // false!
    cout << "  -1 as unsigned: " << i + u << endl;  // 0（UINT_MAX + 1）

    // 安全做法：显式转换
    cout << "显式比较: " << ((int)i < (int)u ? "true" : "false") << endl;

    // double和int：int转为double
    double d = 3.14;
    auto mixed = d + i;
    cout << "double + int 类型: " << sizeof(mixed) << endl;  // 8

    return 0;
}
```

输出：
```
int vs unsigned:
  -1 < 1? false
  -1 as unsigned: 0
显式比较: true
double + int 类型: 8
```

## 三、注意事项与常见陷阱

1. **有符号和无符号混合**：有符号数转为无符号数，负数变成巨大正数
2. **提升到int**：所有小于int的类型运算前先提升为int
3. **类型截断**：大类型赋值给小类型可能截断
4. **开启警告**：`-Wconversion`可以检测隐式转换
5. **显式转换更安全**：混合类型运算时使用static_cast明确意图
