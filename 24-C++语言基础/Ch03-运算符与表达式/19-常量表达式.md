# 常量表达式

## 一、概念说明

常量表达式（constant expression）是在**编译时就能求值**的表达式（C++11标准 §5.19）。使用 `constexpr` 关键字声明的变量或函数可以在编译期被求值，从而提升性能并启用编译期编程能力。

C++17引入 `if constexpr`，C++20引入 `consteval` 和 `constinit`，进一步完善了编译期计算体系。

关键字对比：

| 关键字 | 引入版本 | 含义 |
|--------|---------|------|
| `const` | C++98 | 运行时常量，值不可变 |
| `constexpr` | C++11 | 可以在编译期求值 |
| `consteval` | C++20 | 必须在编译期求值（立即函数） |
| `constinit` | C++20 | 变量必须在编译期初始化，但之后可变 |

## 二、具体用法

### 2.1 constexpr变量

```cpp
#include <iostream>
#include <array>
using namespace std;

int main() {
    // 编译期常量
    constexpr int x = 42;
    constexpr double pi = 3.14159265358979;
    constexpr int arrSize = 10;

    // 用于需要编译期常量的场景
    int cArr[arrSize];                    // C风格数组大小
    array<int, arrSize> stdArr;           // std::array模板参数
    int* p = new int[arrSize];            // 动态分配大小也可以
    delete[] p;

    // const 和 constexpr 的区别
    const int runtimeVal = []() { return 42; }();  // OK：const可在运行时求值
    // constexpr int ceVal = []() { return 42; }(); // C++20 OK，C++17前不行

    // constexpr变量自动具有内部链接（除非显式extern）
    constexpr int internal = 10;          // 内部链接
    extern constexpr int external = 20;   // 外部链接

    cout << "x = " << x << endl;
    cout << "arrSize = " << arrSize << endl;

    return 0;
}
```

### 2.2 constexpr函数

```cpp
#include <iostream>
#include <array>
using namespace std;

// C++11：constexpr函数只能包含return语句
constexpr int square_c11(int x) {
    return x * x;
}

// C++14：可以包含循环、局部变量、if语句等
constexpr int factorial(int n) {
    int result = 1;
    for (int i = 2; i <= n; ++i) {
        result *= i;
    }
    return result;
}

// constexpr函数可以递归
constexpr int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

// constexpr函数可以操作自定义类型
struct Point {
    double x, y;
    constexpr Point(double x, double y) : x(x), y(y) {}
    constexpr double distanceFromOrigin() const {
        return x * x + y * y;  // 简化：不调sqrt以保持constexpr
    }
};

int main() {
    // 编译期求值
    constexpr int sq = square_c11(5);           // 25
    constexpr int fact = factorial(10);         // 3628800
    constexpr int fib = fibonacci(10);          // 55
    array<int, sq> arr1;                        // 大小25
    array<int, fact % 100> arr2;                // 大小0（取模结果）

    // 运行期调用同一函数（上下文不要求编译期时）
    int n;
    // cin >> n;
    // int runtimeSq = square_c11(n);  // OK，运行时调用

    // 自定义类型的constexpr
    constexpr Point p(3.0, 4.0);
    constexpr double dist = p.distanceFromOrigin();  // 25.0

    cout << "factorial(10) = " << fact << endl;
    cout << "fibonacci(10) = " << fib << endl;
    cout << "distance² = " << dist << endl;

    return 0;
}
```

### 2.3 if constexpr — 编译期分支（C++17）

```cpp
#include <iostream>
#include <type_traits>
#include <string>
using namespace std;

template <typename T>
string toString(const T& value) {
    if constexpr (is_integral_v<T>) {
        return to_string(value);        // 整数走这条分支
    } else if constexpr (is_floating_point_v<T>) {
        return to_string(value);        // 浮点走这条分支
    } else {
        return string(value);          // 其他类型走这条分支
    }
}

// 递归终止：if constexpr使得无需额外重载
template <typename T, typename... Args>
auto sum(T first, Args... rest) {
    if constexpr (sizeof...(rest) == 0) {
        return first;
    } else {
        return first + sum(rest...);
    }
}

int main() {
    cout << toString(42) << endl;         // "42"
    cout << toString(3.14) << endl;       // "3.140000"
    cout << toString("hello"s) << endl;   // "hello"

    cout << "sum = " << sum(1, 2, 3, 4, 5) << endl;  // 15

    return 0;
}
```

### 2.4 consteval 和 constinit（C++20）

```cpp
#include <iostream>
using namespace std;

// consteval：立即函数，必须在编译期求值
consteval int compileTimeSquare(int x) {
    return x * x;
}

// constinit：保证编译期初始化，但变量本身不是const
constinit int globalCounter = compileTimeSquare(10);  // 编译期初始化为100

int main() {
    // compileTimeSquare不能在运行时调用
    int arr[compileTimeSquare(5)];  // OK：25

    constexpr int ce = compileTimeSquare(7);  // OK：49

    // int n; cin >> n;
    // compileTimeSquare(n);  // 编译错误！n不是常量表达式

    // constinit变量可以修改
    globalCounter += 1;  // OK
    cout << "globalCounter = " << globalCounter << endl;  // 101

    return 0;
}
```

### 2.5 编译期计算的实际应用

```cpp
#include <array>
#include <iostream>
using namespace std;

// 编译期生成查找表
template <size_t N>
constexpr auto generateSquareTable() {
    array<int, N> table{};
    for (size_t i = 0; i < N; ++i) {
        table[i] = static_cast<int>(i * i);
    }
    return table;
}

int main() {
    // 编译期生成0~9的平方表
    constexpr auto squares = generateSquareTable<10>();

    for (size_t i = 0; i < squares.size(); ++i) {
        cout << i << "² = " << squares[i] << endl;
    }

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **constexpr变量必须用常量表达式初始化**：`const int a = f();` 可以（f运行时调用），但 `constexpr int a = f();` 要求f是constexpr且参数是常量表达式
2. **constexpr函数不一定编译期求值**：取决于调用上下文——编译期需要就编译期求，否则运行时求
3. **C++17起lambda可以是constexpr**：`constexpr auto square = [](int x){ return x*x; };`
4. **constexpr函数在C++11和C++14差异很大**：C++11只能有一个return语句，C++14允许循环、局部变量、if
5. **虚函数不能是constexpr的**（C++20前）；C++20开始允许constexpr虚函数
6. **consteval不能用在虚函数中**：立即函数不能被覆盖
7. **constinit不能用于局部变量**：仅用于具有静态或线程存储期的变量
8. **避免在constexpr函数中使用未定义行为**：编译期检测到UB会导致编译错误
