# 常量表达式

## 一、概念说明

常量表达式是**在编译时就能求值**的表达式。`constexpr`关键字表明一个表达式或函数可以在编译期求值。

## 二、具体用法

### 2.1 constexpr表达式

```cpp
#include <iostream>
#include <array>
using namespace std;

int main() {
    // 编译期常量
    constexpr int x = 42;
    constexpr double pi = 3.14159;
    constexpr int arrSize = 10;

    // 用于需要编译期常量的场景
    int arr[arrSize];                  // C风格数组大小
    array<int, arrSize> stdarr;        // std::array大小

    // 编译期计算
    constexpr int factorial(int n) {
        return n <= 1 ? 1 : n * factorial(n - 1);
    }

    constexpr int fact5 = factorial(5);
    int arr2[fact5];  // 大小120，在编译期确定

    cout << "5! = " << fact5 << endl;
    cout << "arr2大小: " << sizeof(arr2) / sizeof(arr2[0]) << endl;

    return 0;
}
```

输出：
```
5! = 120
arr2大小: 120
```

## 三、注意事项与常见陷阱

1. **constexpr变量必须用编译期常量初始化**
2. **constexpr函数不一定编译期求值**：取决于调用上下文
3. **C++14放宽了constexpr函数的限制**：可以有循环和局部变量
4. **consteval（C++20）**：强制编译期求值
