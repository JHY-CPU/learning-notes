# 位运算符详解

## 一、概念说明

位运算符直接操作整数的**二进制位**。包括按位与`&`、或`|`、异或`^`、取反`~`、左移`<<`和右移`>>`。位运算在系统编程、嵌入式开发和算法竞赛中广泛使用。

## 二、具体用法

### 2.1 六种位运算

```cpp
#include <iostream>
#include <bitset>
using namespace std;

int main() {
    unsigned int a = 0b1010;  // 10
    unsigned int b = 0b1100;  // 12

    cout << "a = " << bitset<8>(a) << " (" << a << ")" << endl;
    cout << "b = " << bitset<8>(b) << " (" << b << ")" << endl;

    // 按位与：两位都为1时结果为1
    unsigned int andResult = a & b;
    cout << "a & b = " << bitset<8>(andResult) << " (" << andResult << ")" << endl;

    // 按位或：任一位为1时结果为1
    unsigned int orResult = a | b;
    cout << "a | b = " << bitset<8>(orResult) << " (" << orResult << ")" << endl;

    // 按位异或：两位不同时结果为1
    unsigned int xorResult = a ^ b;
    cout << "a ^ b = " << bitset<8>(xorResult) << " (" << xorResult << ")" << endl;

    // 按位取反：0变1，1变0
    unsigned int notA = ~a;
    cout << "~a = " << bitset<8>(notA) << " (" << notA << ")" << endl;

    // 左移：所有位向左移动指定位数
    unsigned int leftShift = a << 2;
    cout << "a << 2 = " << bitset<8>(leftShift) << " (" << leftShift << ")" << endl;

    // 右移：所有位向右移动指定位数
    unsigned int rightShift = b >> 1;
    cout << "b >> 1 = " << bitset<8>(rightShift) << " (" << rightShift << ")" << endl;

    return 0;
}
```

输出：
```
a = 00001010 (10)
b = 00001100 (12)
a & b = 00001000 (8)
a | b = 00001110 (14)
a ^ b = 00000110 (6)
~a = 11110101 (245)
a << 2 = 00101000 (40)
b >> 1 = 00000110 (6)
```

### 2.2 常用位操作技巧

```cpp
#include <iostream>
using namespace std;

int main() {
    // 判断奇偶：最低位为1是奇数
    int n = 42;
    cout << n << " 是" << ((n & 1) ? "奇数" : "偶数") << endl;

    // 乘以2的幂：左移
    cout << "5 * 8 = " << (5 << 3) << endl;   // 5 * 2^3 = 40

    // 除以2的幂：右移
    cout << "100 / 4 = " << (100 >> 2) << endl;  // 100 / 2^2 = 25

    // 交换两个数（不用临时变量）
    int x = 10, y = 20;
    x ^= y;
    y ^= x;
    x ^= y;
    cout << "交换后: x=" << x << ", y=" << y << endl;

    // 取绝对值（适用于有符号整数）
    int neg = -42;
    int mask = neg >> 31;  // 全0（正数）或全1（负数）
    int absVal = (neg ^ mask) - mask;
    cout << "|-42| = " << absVal << endl;

    return 0;
}
```

输出：
```
42 是偶数
5 * 8 = 40
100 / 4 = 25
交换后: x=20, y=10
|-42| = 42
```

### 2.3 位掩码操作

```cpp
#include <iostream>
#include <bitset>
using namespace std;

int main() {
    // 设置特定位（置1）
    unsigned char flags = 0b00000000;
    flags |= (1 << 3);  // 设置第3位
    cout << "设置第3位: " << bitset<8>(flags) << endl;

    // 清除特定位（置0）
    flags &= ~(1 << 3);  // 清除第3位
    cout << "清除第3位: " << bitset<8>(flags) << endl;

    // 翻转特定位
    flags = 0b10101010;
    flags ^= (1 << 2);   // 翻转第2位
    cout << "翻转第2位: " << bitset<8>(flags) << endl;

    // 检查特定位
    flags = 0b10101010;
    bool bit3Set = (flags >> 3) & 1;
    bool bit0Set = (flags >> 0) & 1;
    cout << "第3位是否为1: " << bit3Set << endl;
    cout << "第0位是否为1: " << bit0Set << endl;

    // 清除最低的1
    int x = 0b10101000;
    int lowest = x & (-x);  // 提取最低的1
    x &= (x - 1);           // 清除最低的1
    cout << "最低的1在位置: " << bitset<8>(lowest) << endl;
    cout << "清除最低1后: " << bitset<8>(x) << endl;

    return 0;
}
```

输出：
```
设置第3位: 00001000
清除第3位: 00000000
翻转第2位: 10101110
第3位是否为1: 1
第0位是否为1: 0
最低的1在位置: 00001000
清除最低1后: 10100000
```

## 三、注意事项与常见陷阱

1. **有符号右移**：有符号数右移是实现定义的（算术移位），无符号数是逻辑移位
2. **移位超出范围**：`x << n`中`n >= 类型位数`是未定义行为
3. **左移负数**：左移有符号负数是未定义行为
4. **优先级陷阱**：`==`优先级高于`&`，`flags & 1 == 0`应写为`(flags & 1) == 0`
5. **不要用位运算代替逻辑运算**：`&&`/`||`支持短路，`&`/`|`不支持
