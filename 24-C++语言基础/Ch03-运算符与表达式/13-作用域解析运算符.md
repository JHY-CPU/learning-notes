# 作用域解析运算符

## 一、概念说明

`::`（作用域解析运算符）用于指定标识符所属的**作用域**。它可以访问全局变量、命名空间成员、类的静态成员等。

## 二、具体用法

### 2.1 访问全局作用域

```cpp
#include <iostream>
using namespace std;

int value = 100;  // 全局变量

int main() {
    int value = 42;  // 局部变量

    cout << "局部 value: " << value << endl;    // 42
    cout << "全局 value: " << ::value << endl;  // 100

    // 修改全局变量
    ::value = 200;
    cout << "修改后全局 value: " << ::value << endl;

    return 0;
}
```

输出：
```
局部 value: 42
全局 value: 100
修改后全局 value: 200
```

### 2.2 类作用域

```cpp
#include <iostream>
using namespace std;

class Math {
public:
    static const double PI;

    static double square(double x) {
        return x * x;
    }

    double cube(double x) const;
};

const double Math::PI = 3.14159265358979;

// 类外定义成员函数
double Math::cube(double x) const {
    return x * x * x;
}

int main() {
    // 通过类名访问静态成员
    cout << "PI: " << Math::PI << endl;
    cout << "square(5): " << Math::square(5) << endl;

    // 通过对象访问
    Math m;
    cout << "cube(3): " << m.cube(3) << endl;

    return 0;
}
```

输出：
```
PI: 3.14159
square(5): 25
cube(3): 27
```

### 2.3 命名空间作用域

```cpp
#include <iostream>
using namespace std;

namespace Graphics {
    void draw() { cout << "Graphics::draw" << endl; }

    namespace Internal {
        void helper() { cout << "Internal helper" << endl; }
    }
}

namespace Audio {
    void draw() { cout << "Audio::draw" << endl; }
}

int main() {
    // 使用作用域解析访问不同命名空间的同名函数
    Graphics::draw();
    Audio::draw();
    Graphics::Internal::helper();

    return 0;
}
```

输出：
```
Graphics::draw
Audio::draw
Internal helper
```

## 三、注意事项与常见陷阱

1. **全局作用域`::`**：不加前缀的`::name`访问全局命名空间
2. **类外定义必须用`::`**：`void Class::func()`在类外定义成员
3. **静态成员必须用`::`**：通过类名而非对象访问静态成员
4. **嵌套命名空间**：`A::B::C`访问嵌套命名空间的成员
5. **不要用`::`访问局部变量**：`::`只能访问外层作用域的命名实体
