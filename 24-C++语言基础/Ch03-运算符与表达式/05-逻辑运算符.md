# 逻辑运算符

## 一、概念说明

逻辑运算符用于组合和取反布尔表达式：`&&`（逻辑与）、`||`（逻辑或）、`!`（逻辑非）。它们支持**短路求值**，即在结果已确定时停止求值。

## 二、具体用法

### 2.1 基本逻辑运算

```cpp
#include <iostream>
using namespace std;

int main() {
    bool a = true, b = false;

    cout << boolalpha;
    cout << "true && true:  " << (true && true) << endl;    // true
    cout << "true && false: " << (true && false) << endl;   // false
    cout << "false && true: " << (false && true) << endl;   // false
    cout << "false && false:" << (false && false) << endl;  // false

    cout << "true || true:  " << (true || true) << endl;    // true
    cout << "true || false: " << (true || false) << endl;   // true
    cout << "false || false:" << (false || false) << endl;  // false

    cout << "!true:  " << (!true) << endl;   // false
    cout << "!false: " << (!false) << endl;  // true

    // 组合使用
    int age = 25;
    bool hasID = true;
    bool canEnter = (age >= 18) && hasID;
    cout << "可以进入: " << canEnter << endl;

    return 0;
}
```

输出：
```
true && true:  true
true && false: false
false && true: false
false && false:false
true || true:  true
true || false: true
false || false:false
!true:  false
!false: true
可以进入: true
```

### 2.2 短路求值

```cpp
#include <iostream>
using namespace std;

int sideEffect(int val, const char* name) {
    cout << "  " << name << "被求值" << endl;
    return val;
}

int main() {
    // &&短路：左侧为false时，右侧不求值
    cout << "false && (副作用表达式):" << endl;
    bool result1 = sideEffect(0, "左侧") && sideEffect(1, "右侧");
    cout << "结果: " << result1 << endl;

    // ||短路：左侧为true时，右侧不求值
    cout << "\ntrue || (副作用表达式):" << endl;
    bool result2 = sideEffect(1, "左侧") || sideEffect(0, "右侧");
    cout << "结果: " << result2 << endl;

    // 利用短路防止空指针解引用
    int* ptr = nullptr;
    // 如果ptr为null，右侧的*ptr不会执行
    if (ptr != nullptr && *ptr > 0) {
        cout << "值为正数" << endl;
    } else {
        cout << "指针为空或值非正" << endl;
    }

    return 0;
}
```

输出：
```
false && (副作用表达式):
  左侧被求值
结果: 0

true || (副作用表达式):
  左侧被求值
结果: 1
指针为空或值非正
```

### 2.3 布尔转换

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 非布尔类型自动转换为bool
    // 0、nullptr、空指针 → false
    // 非0值、非空指针 → true

    int x = 0;
    int y = 42;
    int* p = nullptr;
    string s = "";

    cout << boolalpha;
    cout << "0 -> " << (bool)x << endl;         // false
    cout << "42 -> " << (bool)y << endl;        // true
    cout << "nullptr -> " << (bool)p << endl;   // false
    cout << "\"\" -> " << (bool)s.size() << endl; // false

    // 实际应用
    int count = 0;
    if (!count) {
        cout << "没有项目" << endl;
    }

    int* ptr = new int(42);
    if (ptr) {
        cout << "指针有效: " << *ptr << endl;
    }
    delete ptr;

    return 0;
}
```

输出：
```
0 -> false
42 -> true
nullptr -> false
"" -> false
没有项目
指针有效: 42
```

## 三、注意事项与常见陷阱

1. **短路求值保证顺序**：`&&`和`||`保证从左到右求值，且可能不求值右侧
2. **利用短路防止错误**：`ptr && *ptr > 0`避免空指针解引用
3. **副作用在短路时可能不执行**：`false && i++`中`i++`不会执行
4. **非布尔转bool**：整数0转false，非0转true；指针nullptr转false
5. **优先级**：`!`优先级高于`&&`，`&&`优先级高于`||`
