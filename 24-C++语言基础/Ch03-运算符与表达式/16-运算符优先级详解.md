# 运算符优先级详解

## 一、概念说明

运算符优先级决定了**表达式中运算的执行顺序**。优先级高的运算符先执行。当优先级相同时，结合性决定执行顺序。

## 二、具体用法

### 2.1 优先级表（从高到低）

```cpp
#include <iostream>
using namespace std;

int main() {
    // 优先级1（最高）：作用域
    // ::

    // 优先级2：成员访问和后缀
    // ()  []  ->  .  后置++ --

    // 优先级3：一元运算符
    // 前置++ --  + -  ! ~  * &  sizeof  typeid  cast

    // 优先级4：乘除模
    // *  /  %
    int a = 2 + 3 * 4;  // 14, 不是20
    cout << "2 + 3 * 4 = " << a << endl;

    // 优先级5：加减
    // +  -
    int b = 10 - 4 + 2;  // 8, 从左到右
    cout << "10 - 4 + 2 = " << b << endl;

    // 优先级6：移位
    // <<  >>
    int c = 1 << 2 + 1;  // 1 << 3 = 8, +优先级高于<<
    cout << "1 << 2 + 1 = " << c << endl;

    // 优先级7-8：关系和相等
    // < <= > >=  == !=
    int d = 5 > 3 && 2 < 4;  // true && true = 1
    cout << "5 > 3 && 2 < 4 = " << d << endl;

    // 优先级9-11：位运算
    // &  ^  |

    // 优先级12-13：逻辑
    // &&  ||
    int e = 1 | 2 && 4;  // (1|2) && 4 = 3 && 4 = 1
    cout << "1 | 2 && 4 = " << e << endl;

    // 优先级14：条件
    // ? :

    // 优先级15：赋值
    // =  += -= *= /= 等

    // 优先级16（最低）：逗号
    // ,

    return 0;
}
```

输出：
```
2 + 3 * 4 = 14
10 - 4 + 2 = 8
1 << 2 + 1 = 8
5 > 3 && 2 < 4 = 1
1 | 2 && 4 = 1
```

### 2.2 常见混淆示例

```cpp
#include <iostream>
using namespace std;

int main() {
    // 混淆1：* 和 ++
    int x = 5;
    int* p = &x;
    int a = *p++;    // 先取值，再++指针（后置++优先级高于*）
    int b = (*p)++;  // 先取值，再++该值
    int c = *++p;    // 先++指针，再取值
    // int d = ++*p; // 先取值，再++该值

    // 混淆2：& 和 ==
    int flags = 0x0F;
    // if (flags & 0x01 == 0)  // 错误：==优先级高于&
    // 应写为：
    if ((flags & 0x01) == 0) {
        cout << "最低位为0" << endl;
    }

    // 混淆3：<< 和 +
    cout << "1 + 2: " << 1 + 2 << endl;     // OK
    // cout << 1 + 2 << 3;  // 错误：<< 被当作移位

    // 建议：不确定优先级时，用括号
    int result = (a + b) * (c - 1);
    cout << "结果: " << result << endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **记不住就加括号**：这是最重要的建议
2. **== 优先级高于 &**：`flags & mask == 0`应写为`(flags & mask) == 0`
3. **<< 优先级低于算术**：`cout << a + b`是`cout << (a + b)`
4. **后置++优先级最高**：`*p++`是`*(p++)`不是`(*p)++`
5. **&& 高于 ||**：`a || b && c`是`a || (b && c)`
