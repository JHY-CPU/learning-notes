# 左值与右值

## 一、概念说明

C++中每个表达式都有**值类别**（value category）。基本分类为**左值**（lvalue）和**右值**（rvalue），C++11进一步细分为左值、纯右值（prvalue）、亡值（xvalue）等。

## 二、具体用法

### 2.1 左值与右值的基本概念

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 左值：有名字、有地址、可以出现在赋值号左边
    int x = 42;        // x是左值
    int& ref = x;      // ref是左值引用，绑定到左值x
    x = 100;           // OK：x是左值

    // 右值：临时的、没有持久地址、不能出现在赋值号左边
    int y = x + 1;     // x + 1是右值（纯右值）
    // (x + 1) = 10;   // 编译错误：不能给右值赋值

    // 字面量是右值
    // 42 = x;          // 编译错误：42是右值
    // "hello" = "bye"; // 编译错误

    // 字符串字面量是左值（特殊）
    const char* str = "hello";  // "hello"是左值

    // 左值引用只能绑定左值
    // int& bad = 42;   // 编译错误：不能将右值绑定到非const左值引用

    // const左值引用可以绑定右值
    const int& cref = 42;  // OK：const引用延长临时对象生命
    cout << "cref = " << cref << endl;

    return 0;
}
```

输出：
```
cref = 42
```

### 2.2 右值引用（C++11）

```cpp
#include <iostream>
#include <string>
#include <utility>
using namespace std;

class MyString {
    char* data;
    size_t len;

public:
    // 构造函数
    MyString(const char* s = "") {
        len = strlen(s);
        data = new char[len + 1];
        strcpy(data, s);
        cout << "构造: " << data << endl;
    }

    // 拷贝构造（左值引用参数）
    MyString(const MyString& other) {
        len = other.len;
        data = new char[len + 1];
        strcpy(data, other.data);
        cout << "拷贝构造: " << data << endl;
    }

    // 移动构造（右值引用参数）
    MyString(MyString&& other) noexcept {
        data = other.data;
        len = other.len;
        other.data = nullptr;
        other.len = 0;
        cout << "移动构造" << endl;
    }

    ~MyString() {
        delete[] data;
    }
};

int main() {
    // std::move将左值转为右值引用
    MyString s1("Hello");
    MyString s2 = std::move(s1);  // 调用移动构造

    // 临时对象使用移动构造
    MyString s3 = MyString("World");

    return 0;
}
```

输出：
```
构造: Hello
移动构造
构造: World
```

### 2.3 亡值（xvalue）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 亡值（xvalue）：即将被销毁的对象
    // 通过std::move或返回右值引用的函数产生

    string s1 = "Hello";
    string s2 = std::move(s1);  // s1变为亡值

    cout << "s2: " << s2 << endl;
    cout << "s1: " << s1 << " (可能为空)" << endl;

    // 亡值可以绑定到右值引用
    string&& rref = std::move(s2);

    // 亡值也可以调用成员函数
    cout << "size: " << rref.size() << endl;

    return 0;
}
```

输出：
```
s2: Hello
s1:  (可能为空)
size: 5
```

### 2.4 完美转发

```cpp
#include <iostream>
#include <utility>
using namespace std;

template<typename T>
void wrapper(T&& arg) {  // 万能引用
    // std::forward保持值类别
    cout << "转发: " << std::forward<T>(arg) << endl;
}

int main() {
    int x = 42;

    wrapper(x);           // x是左值，T推导为int&，forward保持左值
    wrapper(42);          // 42是右值，T推导为int，forward保持右值
    wrapper(std::move(x)); // 亡值，T推导为int，forward保持右值

    return 0;
}
```

输出：
```
转发: 42
转发: 42
转发: 42
```

## 三、注意事项与常见陷阱

1. **左值有地址，右值是临时的**：变量名是左值，表达式结果常是右值
2. **move不移动**：`std::move`只是类型转换，不实际移动任何东西
3. **move后对象处于有效但未指定状态**：可以销毁或赋新值，但不应假设其内容
4. **右值引用变量本身是左值**：`int&& r = 5;`中`r`本身是左值
5. **完美转发用forward**：模板中用`std::forward<T>`保持参数的值类别
