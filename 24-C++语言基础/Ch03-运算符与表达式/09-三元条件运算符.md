# 三元条件运算符

## 一、概念说明

三元条件运算符`? :`是C++中唯一的三目运算符，语法为`condition ? expr1 : expr2`。如果condition为true返回expr1，否则返回expr2。它是`if-else`语句的简洁形式。

## 二、具体用法

### 2.1 基本用法

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int a = 10, b = 20;

    // 基本三元运算
    int max = (a > b) ? a : b;
    cout << "较大值: " << max << endl;

    // 用于输出
    cout << (a % 2 == 0) ? "偶数" : "奇数";
    cout << endl;

    // 字符串
    string status = (a > 0) ? "正数" : "非正数";
    cout << "a是: " << status << endl;

    // 嵌套（不推荐过度嵌套）
    int x = 5;
    string category = (x > 0) ? "正" : (x < 0) ? "负" : "零";
    cout << "x = " << category << endl;

    return 0;
}
```

输出：
```
较大值: 20
a是: 正数
x = 正
```

### 2.2 类型要求

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // 两个分支必须是兼容类型
    int x = 42;

    // OK：都是int
    int a = (x > 0) ? 1 : -1;

    // OK：double和int，结果是double
    double b = (x > 0) ? 3.14 : 2;

    // OK：都可转为string
    string s = (x > 0) ? "yes" : "no";

    // 编译错误：类型不兼容
    // auto bad = (x > 0) ? "string" : 42;  // const char* vs int

    // 返回引用
    int left = 10, right = 20;
    int& ref = (x > 0) ? left : right;
    ref = 999;
    cout << "left = " << left << ", right = " << right << endl;

    // 三元运算的结果是左值（如果两个分支都是左值）
    ((x > 0) ? left : right) = 100;
    cout << "赋值后 left = " << left << endl;

    return 0;
}
```

输出：
```
left = 999, right = 20
赋值后 left = 100
```

### 2.3 与if-else的对比

```cpp
#include <iostream>
using namespace std;

// 三元运算符适合简单的二选一
int absValue(int x) {
    return (x >= 0) ? x : -x;  // 简洁
}

// if-else适合复杂逻辑
string grade(int score) {
    if (score >= 90) {
        return "优秀";
    } else if (score >= 80) {
        return "良好";
    } else if (score >= 60) {
        return "及格";
    } else {
        return "不及格";
    }
}

// 三元运算符可以用于const初始化
void demo() {
    const int timeout = true ? 30 : 60;  // const必须在声明时确定值
    cout << "超时: " << timeout << endl;

    // if-else无法用于const初始化
    // const int t;
    // if (condition) t = 30; else t = 60;  // 编译错误
}

int main() {
    cout << "|-5| = " << absValue(-5) << endl;
    cout << "85分: " << grade(85) << endl;
    demo();

    return 0;
}
```

输出：
```
|-5| = 5
85分: 良好
超时: 30
```

## 三、注意事项与常见陷阱

1. **不要过度嵌套**：`a ? b : c ? d : e`可读性差，改用if-else
2. **类型必须兼容**：两个分支的类型必须可以转换为公共类型
3. **短路求值**：只有一个分支会被求值，类似`&&`和`||`
4. **三元的结果可以是左值**：如果两个分支都是左值表达式
5. **可用于const初始化**：`const int x = cond ? a : b;`是合法的
