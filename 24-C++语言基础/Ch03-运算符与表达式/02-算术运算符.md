# 算术运算符

## 一、概念说明

算术运算符执行基本的数学运算：加`+`、减`-`、乘`*`、除`/`、取模`%`。整数和浮点数的除法行为不同，需特别注意。

## 二、具体用法

### 2.1 基本运算

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 17, b = 5;

    cout << "a + b = " << a + b << endl;   // 22
    cout << "a - b = " << a - b << endl;   // 12
    cout << "a * b = " << a * b << endl;   // 85

    // 整数除法：截断小数部分
    cout << "a / b = " << a / b << endl;   // 3（不是3.4）

    // 取模：余数
    cout << "a % b = " << a % b << endl;   // 2

    // 浮点除法
    double x = 17.0, y = 5.0;
    cout << "x / y = " << x / y << endl;   // 3.4

    // 混合运算：整数提升为浮点
    double result = a / static_cast<double>(b);
    cout << "精确除法: " << result << endl; // 3.4

    return 0;
}
```

输出：
```
a + b = 22
a - b = 12
a * b = 85
a / b = 3
a % b = 2
x / y = 3.4
精确除法: 3.4
```

### 2.2 整数除法与取模

```cpp
#include <iostream>
using namespace std;

int main() {
    // 整数除法总是向零截断
    cout << "17 / 5 = " << 17 / 5 << endl;     // 3
    cout << "-17 / 5 = " << -17 / 5 << endl;   // -3（向零截断）
    cout << "17 / -5 = " << 17 / -5 << endl;   // -3

    // 取模：a % b = a - (a/b)*b
    cout << "17 % 5 = " << 17 % 5 << endl;     // 2
    cout << "-17 % 5 = " << -17 % 5 << endl;   // -2（符号与被除数相同）
    cout << "17 % -5 = " << 17 % -5 << endl;   // 2

    // 取模的实际应用
    // 判断奇偶
    int n = 42;
    cout << n << " 是" << ((n % 2 == 0) ? "偶数" : "奇数") << endl;

    // 循环数组索引
    int index = 10;
    int size = 7;
    cout << "循环索引: " << index % size << endl;  // 3

    // 注意：浮点数不能用%
    // double x = 3.14 % 2.0;  // 编译错误
    // 应使用 fmod() 函数

    return 0;
}
```

输出：
```
17 / 5 = 3
-17 / 5 = -3
17 / -5 = -3
17 % 5 = 2
-17 % 5 = -2
17 % -5 = 2
42 是偶数
循环索引: 3
```

### 2.3 算术溢出

```cpp
#include <iostream>
#include <climits>
using namespace std;

int main() {
    // 有符号整数溢出：未定义行为
    int maxInt = INT_MAX;
    cout << "INT_MAX = " << maxInt << endl;
    // int overflow = maxInt + 1;  // UB！可能产生任意结果

    // 无符号整数溢出：定义良好（回绕）
    unsigned int maxUint = UINT_MAX;
    cout << "UINT_MAX = " << maxUint << endl;
    unsigned int wrapped = maxUint + 1;
    cout << "UINT_MAX + 1 = " << wrapped << endl;  // 0

    unsigned int zero = 0;
    unsigned int underflow = zero - 1;
    cout << "0 - 1 = " << underflow << endl;  // UINT_MAX

    // 溢出检测方法
    auto safeAdd = [](int a, int b, int& result) -> bool {
        if (b > 0 && a > INT_MAX - b) return false;
        if (b < 0 && a < INT_MIN - b) return false;
        result = a + b;
        return true;
    };

    int result;
    if (safeAdd(2000000000, 2000000000, result)) {
        cout << "安全加法: " << result << endl;
    } else {
        cout << "加法溢出！" << endl;
    }

    return 0;
}
```

输出：
```
INT_MAX = 2147483647
UINT_MAX = 4294967295
UINT_MAX + 1 = 0
0 - 1 = 4294967295
加法溢出！
```

## 三、注意事项与常见陷阱

1. **整数除法截断**：`7 / 2`结果是3不是3.5，需要精确结果时转换为浮点
2. **除以零**：整数除以零是未定义行为，浮点除以零得到inf或-inf
3. **取模的符号**：C++11起`a % b`的符号与`a`相同
4. **有符号溢出是UB**：不要依赖有符号溢出的行为，使用溢出检测
5. **混合类型运算**：int和double混合运算时，int自动提升为double
