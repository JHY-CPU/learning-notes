# 成员访问运算符

## 一、概念说明

`.`（点运算符）和`->`（箭头运算符）用于访问结构体或类的成员。`.`用于对象，`->`用于指针。

## 二、具体用法

### 2.1 . 和 -> 的使用

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Point {
    double x, y;

    double distance() const {
        return x * x + y * y;
    }
};

int main() {
    Point p1{3.0, 4.0};
    Point* p2 = new Point{5.0, 12.0};

    // . 用于对象
    cout << "p1.x = " << p1.x << endl;
    cout << "p1.distance = " << p1.distance() << endl;

    // -> 用于指针（等价于 (*p2).x）
    cout << "p2->x = " << p2->x << endl;
    cout << "p2->distance = " << p2->distance() << endl;

    // (*p2).x 和 p2->x 完全等价
    cout << "(*p2).x = " << (*p2).x << endl;

    delete p2;
    return 0;
}
```

输出：
```
p1.x = 3
p1.distance = 25
p2->x = 5
p2->distance = 169
(*p2).x = 5
```

### 2.2 优先级陷阱

```cpp
#include <iostream>
using namespace std;

struct Node {
    int value;
    Node* next;
};

int main() {
    Node a{1, nullptr};
    Node b{2, &a};
    Node* head = &b;

    // -> 优先级高于 *
    // *head.next 是 *(head.next)，编译错误
    // (*head).next 才正确
    // head->next 等价于 (*head).next

    cout << "head->value = " << head->value << endl;
    cout << "head->next->value = " << head->next->value << endl;

    // 链式访问
    // head->next->next 是安全的（这里next->next是nullptr）
    if (head->next->next) {
        cout << "有第三个节点" << endl;
    } else {
        cout << "只有两个节点" << endl;
    }

    return 0;
}
```

输出：
```
head->value = 2
head->next->value = 1
只有两个节点
```

## 三、注意事项与常见陷阱

1. **空指针用->会崩溃**：`nullptr->member`是未定义行为
2. **->优先级高于***：`*p.member`是`*(p.member)`，不是`(*p).member`
3. **智能指针用->**：`unique_ptr`和`shared_ptr`重载了`->`运算符
4. **.不能重载，->可以重载**：自定义智能指针需要重载`->`
5. **链式访问**：`a->b->c`中任何一个`->`的左侧为null都会崩溃
