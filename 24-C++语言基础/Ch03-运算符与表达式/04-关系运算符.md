# 关系运算符

## 一、概念说明

关系运算符比较两个值的关系，返回`bool`结果。包括`==`（等于）、`!=`（不等于）、`>`（大于）、`<`（小于）、`>=`（大于等于）、`<=`（小于等于）。

## 二、具体用法

### 2.1 基本比较

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;

    cout << boolalpha;
    cout << "a == b: " << (a == b) << endl;  // false
    cout << "a != b: " << (a != b) << endl;  // true
    cout << "a > b:  " << (a > b) << endl;   // false
    cout << "a < b:  " << (a < b) << endl;   // true
    cout << "a >= b: " << (a >= b) << endl;  // false
    cout << "a <= b: " << (a <= b) << endl;  // true

    // 字符串比较
    string s1 = "apple", s2 = "banana";
    cout << "\n\"apple\" < \"banana\": " << (s1 < s2) << endl;  // true（字典序）

    // 指针比较
    int x = 42;
    int* p1 = &x;
    int* p2 = &x;
    cout << "p1 == p2: " << (p1 == p2) << endl;  // true（相同地址）

    return 0;
}
```

输出：
```
a == b: false
a != b: true
a > b:  false
a < b:  true
a >= b: false
a <= b: true

"apple" < "banana": true
p1 == p2: true
```

### 2.2 浮点数比较陷阱

```cpp
#include <iostream>
#include <cmath>
#include <cfloat>
using namespace std;

// 安全的浮点数比较
bool approxEqual(double a, double b, double epsilon = 1e-9) {
    return abs(a - b) < epsilon;
}

bool relativeEqual(double a, double b, double relTol = 1e-9) {
    return abs(a - b) <= relTol * max(abs(a), abs(b));
}

int main() {
    // 浮点数精度问题
    double a = 0.1 + 0.2;
    double b = 0.3;

    cout << "0.1 + 0.2 = " << a << endl;
    cout << "0.1 + 0.2 == 0.3? " << (a == b ? "true" : "false") << endl;

    // 正确的比较方式
    cout << "近似相等? " << (approxEqual(a, b) ? "true" : "false") << endl;
    cout << "相对相等? " << (relativeEqual(a, b) ? "true" : "false") << endl;

    // 特殊值比较
    double inf = numeric_limits<double>::infinity();
    double nan_val = numeric_limits<double>::quiet_NaN();

    cout << "\nNaN == NaN: " << (nan_val == nan_val) << endl;  // false!
    cout << "NaN != NaN: " << (nan_val != nan_val) << endl;   // true!
    cout << "inf > 0: " << (inf > 0) << endl;                 // true

    // 检查NaN
    cout << "isnan(NaN): " << isnan(nan_val) << endl;

    return 0;
}
```

输出：
```
0.1 + 0.2 = 0.3
0.1 + 0.2 == 0.3? false
近似相等? true
相对相等? true

NaN == NaN: false
NaN != NaN: true
inf > 0: true
isnan(NaN): true
```

### 2.3 连接比较的陷阱

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 10, c = 15;

    // 错误：数学中的 a < b < c 在C++中不是预期行为
    // a < b < c 先计算 a < b（得到true=1），再计算 1 < c（true）
    // 即使 a > b，结果仍然可能是true！
    cout << "5 < 10 < 15: " << (a < b < c) << endl;  // true（但不是你想的那样）

    // 错误示例：a=50时
    a = 50;
    cout << "50 < 10 < 15: " << (a < b < c) << endl;  // true!（50<10=false=0，0<15=true）

    // 正确方式：使用逻辑与
    a = 5;
    cout << "5 < 10 && 10 < 15: " << (a < b && b < c) << endl;  // true

    return 0;
}
```

输出：
```
5 < 10 < 15: true
50 < 10 < 15: true
5 < 10 && 10 < 15: true
```

## 三、注意事项与常见陷阱

1. **浮点数不能用==比较**：由于精度问题，应使用误差范围比较
2. **a < b < c不是你想的那样**：C++中这是`(a<b)<c`，不是数学中的区间比较
3. **NaN的特殊性**：`NaN == NaN`为false，必须用`isnan()`检测
4. **== vs =**：`if (x = 5)`是赋值，`if (x == 5)`是比较
5. **无符号数比较**：`unsigned(-1) > 0`为true，因为-1转换为最大的无符号数
