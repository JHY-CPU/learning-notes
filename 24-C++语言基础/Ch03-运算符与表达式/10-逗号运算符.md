# 逗号运算符

## 一、概念说明

逗号运算符`,`从左到右依次计算每个表达式，返回**最右侧表达式的值**。它是优先级最低的运算符。虽然能写出紧凑的代码，但过度使用会降低可读性。

## 二、具体用法

### 2.1 基本用法

```cpp
#include <iostream>
using namespace std;

int main() {
    // 逗号运算符：依次求值，返回最后一个
    int a = (1, 2, 3, 4, 5);
    cout << "a = " << a << endl;  // 5

    // 实际执行：1（丢弃）, 2（丢弃）, 3（丢弃）, 4（丢弃）, 5（返回）

    // 带副作用
    int x = 10;
    int y = (x++, x += 5, x * 2);
    cout << "x = " << x << ", y = " << y << endl;
    // x++ → x=11, x+=5 → x=16, x*2 → 32

    // 在for循环中常见用法
    for (int i = 0, j = 10; i < j; i++, j--) {
        cout << "i=" << i << ", j=" << j << endl;
    }

    return 0;
}
```

输出：
```
a = 5
x = 16, y = 32
i=0, j=10
i=1, j=9
i=2, j=8
i=3, j=7
i=4, j=6
```

### 2.2 常见应用场景

```cpp
#include <iostream>
using namespace std;

// 在宏中使用逗号运算符
#define SWAP(a, b) (tmp = a, a = b, b = tmp)

// 函数返回多个值的旧方式
int minMax(int arr[], int n, int& outMax) {
    int minVal = arr[0], maxVal = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] < minVal) minVal = arr[i];
        if (arr[i] > maxVal) maxVal = arr[i];
    }
    outMax = maxVal;
    return minVal;
}

int main() {
    // for循环中的多变量操作
    for (int i = 0, count = 0; i < 5; i++, count++) {
        // i和count同时递增
    }

    // while循环中的更新
    int i = 0;
    int sum = 0;
    while (i < 5, sum < 20) {  // 注意：条件部分用的是sum < 20
        sum += i;
        i++;
    }
    cout << "i=" << i << ", sum=" << sum << endl;

    // 使用minMax
    int arr[] = {3, 7, 1, 9, 4};
    int maxVal;
    int minVal = minMax(arr, 5, maxVal);
    cout << "min=" << minVal << ", max=" << maxVal << endl;

    return 0;
}
```

输出：
```
i=7, sum=21
min=1, max=9
```

### 2.3 优先级陷阱

```cpp
#include <iostream>
using namespace std;

int main() {
    // 逗号运算符优先级最低！
    int a, b;
    int c;

    // 这里逗号是分隔符，不是运算符
    a = 1, b = 2, c = 3;
    cout << "a=" << a << ", b=" << b << ", c=" << c << endl;

    // 想用逗号运算符时，必须加括号
    int x = (a = 1, b = 2, c = 3);
    // 等价于：a=1; b=2; x = (c=3);
    cout << "x = " << x << endl;

    // 陷阱：函数参数中的逗号不是运算符
    // cout << (1, 2, 3) << endl;  // 输出3
    // cout << 1, 2, 3 << endl;    // 编译错误！逗号被当作参数分隔符

    // 正确写法
    cout << (1, 2, 3) << endl;  // 3

    return 0;
}
```

输出：
```
a=1, b=2, c=3
x = 3
3
```

## 三、注意事项与常见陷阱

1. **优先级最低**：`a = 1, 2`中逗号是分隔符不是运算符，`a = (1, 2)`中才是运算符
2. **可读性差**：除for循环外，尽量避免使用逗号运算符
3. **函数参数中的逗号**：是参数分隔符，不是逗号运算符
4. **只有最后一个值有效**：前面表达式的值被丢弃，只有副作用保留
5. **用分号替代**：大多数情况下，用分号分隔语句更清晰
