# 取地址与解引用

## 一、概念说明

`&`（取地址）获取变量的内存地址，`*`（解引用）通过指针访问所指向的值。注意`&`和`*`在不同上下文中有不同含义。

## 二、具体用法

### 2.1 取地址与解引用

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;

    // &取地址
    int* ptr = &x;  // ptr存储x的地址
    cout << "x的值: " << x << endl;
    cout << "x的地址: " << &x << endl;
    cout << "ptr的值（x的地址）: " << ptr << endl;

    // *解引用
    cout << "通过ptr访问x: " << *ptr << endl;

    // 通过指针修改值
    *ptr = 100;
    cout << "修改后 x = " << x << endl;

    // 多级指针
    int** pptr = &ptr;  // 指向指针的指针
    cout << "**pptr = " << **pptr << endl;

    // 指针和引用的区别
    int& ref = x;       // 引用：别名
    int* p = &x;        // 指针：存储地址

    // &和*在不同上下文的含义
    // 声明中的&表示引用：int& ref = x;
    // 表达式中的&表示取地址：int* p = &x;
    // 声明中的*表示指针：int* p;
    // 表达式中的*表示解引用：int val = *p;

    return 0;
}
```

输出：
```
x的值: 42
x的地址: 0x7ffd...
ptr的值（x的地址）: 0x7ffd...
通过ptr访问x: 42
修改后 x = 100
**pptr = 100
```

### 2.2 指针运算

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int* p = arr;  // 数组名退化为指针

    cout << "p[0] = " << *p << endl;
    cout << "p[1] = " << *(p + 1) << endl;
    cout << "p[2] = " << *(p + 2) << endl;

    // 指针算术
    p++;  // 移动到下一个int（4字节）
    cout << "p++后: " << *p << endl;

    // 指针减法
    int* q = &arr[4];
    cout << "q - p = " << (q - p) << " 个元素" << endl;

    // 遍历数组
    for (int* it = arr; it != arr + 5; ++it) {
        cout << *it << " ";
    }
    cout << endl;

    return 0;
}
```

输出：
```
p[0] = 10
p[1] = 20
p[2] = 30
p++后: 20
q - p = 3 个元素
10 20 30 40 50
```

## 三、注意事项与常见陷阱

1. **空指针解引用**：`*nullptr`是未定义行为，程序会崩溃
2. **悬垂指针**：指向已释放内存的指针不能解引用
3. **&在不同上下文**：声明中是引用，表达式中是取地址
4. **指针运算单位**：`p + 1`移动的是一个元素的大小，不是1字节
5. **取地址运算符不能重载**：`&`的行为是固定的
