# 赋值运算符

## 一、概念说明

赋值运算符将右侧表达式的值存储到左侧变量中。简单赋值`=`之外，C++还提供了复合赋值运算符`+=`、`-=`等，使代码更简洁。

## 二、具体用法

### 2.1 简单赋值与复合赋值

```cpp
#include <iostream>
using namespace std;

int main() {
    // 简单赋值
    int x = 10;
    x = 20;  // 重新赋值

    // 复合赋值运算符
    int a = 100;
    a += 10;   // a = a + 10  → 110
    a -= 20;   // a = a - 20  → 90
    a *= 2;    // a = a * 2   → 180
    a /= 3;    // a = a / 3   → 60
    a %= 7;    // a = a % 7   → 4

    // 位运算复合赋值
    unsigned int b = 0xFF;
    b &= 0x0F;   // b = b & 0x0F → 0x0F
    b |= 0xF0;   // b = b | 0xF0 → 0xFF
    b ^= 0xFF;   // b = b ^ 0xFF → 0x00
    b <<= 4;     // b = b << 4   → 0x00
    b = 0x80;
    b >>= 2;     // b = b >> 2   → 0x20

    cout << "a = " << a << endl;
    cout << "b = " << b << endl;

    // 连续赋值（从右到左）
    int c, d, e;
    c = d = e = 42;
    cout << "c=" << c << ", d=" << d << ", e=" << e << endl;

    return 0;
}
```

输出：
```
a = 4
b = 32
c=42, d=42, e=42
```

### 2.2 类的赋值运算符

```cpp
#include <iostream>
#include <cstring>
using namespace std;

class String {
    char* data;
    int length;

public:
    String(const char* str = "") {
        length = strlen(str);
        data = new char[length + 1];
        strcpy(data, str);
    }

    // 拷贝赋值运算符
    String& operator=(const String& other) {
        if (this != &other) {  // 自赋值检查
            delete[] data;
            length = other.length;
            data = new char[length + 1];
            strcpy(data, other.data);
        }
        return *this;  // 返回*this支持链式赋值
    }

    // 移动赋值运算符（C++11）
    String& operator=(String&& other) noexcept {
        if (this != &other) {
            delete[] data;
            data = other.data;
            length = other.length;
            other.data = nullptr;
            other.length = 0;
        }
        return *this;
    }

    ~String() { delete[] data; }

    void print() const { cout << data << endl; }
};

int main() {
    String s1("Hello");
    String s2("World");
    String s3;

    s3 = s1;  // 拷贝赋值
    s3.print();

    s3 = String("C++");  // 移动赋值
    s3.print();

    // 链式赋值
    s1 = s2 = s3;
    s1.print();
    s2.print();

    return 0;
}
```

输出：
```
Hello
C++
C++
C++
```

### 2.3 引用返回的赋值

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 返回引用允许赋值操作
int& getElement(vector<int>& vec, int index) {
    return vec[index];
}

int main() {
    vector<int> nums = {1, 2, 3, 4, 5};

    // 通过返回的引用赋值
    getElement(nums, 2) = 999;
    cout << "nums[2] = " << nums[2] << endl;

    // 甚至可以复合赋值
    getElement(nums, 0) += 100;
    cout << "nums[0] = " << nums[0] << endl;

    return 0;
}
```

输出：
```
nums[2] = 999
nums[0] = 101
```

## 三、注意事项与常见陷阱

1. **= vs ==**：`if (x = 5)`是赋值，`if (x == 5)`是比较
2. **自赋值检查**：类的赋值运算符必须检查`this != &other`
3. **返回*this**：赋值运算符应返回`*this`的引用以支持链式赋值
4. **复合赋值效率**：`a += b`可能比`a = a + b`更高效（少一次拷贝）
5. **const不能赋值**：`const int x = 5; x = 10;`是编译错误
