# 短路求值陷阱

## 一、概念说明

`&&`和`||`支持短路求值：`&&`左侧为false时跳过右侧，`||`左侧为true时跳过右侧。这既是特性也是陷阱来源。

## 二、具体用法

### 2.1 短路求值的副作用

```cpp
#include <iostream>
using namespace std;

int callCount = 0;

bool check() {
    callCount++;
    cout << "check()第" << callCount << "次调用" << endl;
    return callCount > 2;
}

int main() {
    // &&短路：左侧false时右侧不执行
    callCount = 0;
    if (false && check()) {
        cout << "不会执行" << endl;
    }
    cout << "check被调用: " << callCount << " 次" << endl;

    // ||短路：左侧true时右侧不执行
    callCount = 0;
    if (true || check()) {
        cout << "执行但check不调用" << endl;
    }
    cout << "check被调用: " << callCount << " 次" << endl;

    // 混合短路
    callCount = 0;
    bool result = check() && check() && check();
    cout << "三个check: 调用" << callCount << "次, 结果=" << result << endl;

    return 0;
}
```

输出：
```
check被调用: 0 次
执行但check不调用
check被调用: 0 次
check()第1次调用
check()第2次调用
check()第3次调用
三个check: 调用3次, 结果=1
```

### 2.2 利用短路防止错误

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    // 防止空指针解引用
    int* ptr = nullptr;
    if (ptr != nullptr && *ptr > 0) {
        cout << "正数" << endl;
    } else {
        cout << "安全：短路保护了空指针" << endl;
    }

    // 防止数组越界
    vector<int> vec = {1, 2, 3};
    int index = 10;
    if (index >= 0 && index < (int)vec.size() && vec[index] > 0) {
        cout << "有效" << endl;
    } else {
        cout << "安全：短路保护了越界访问" << endl;
    }

    return 0;
}
```

输出：
```
安全：短路保护了空指针
安全：短路保护了越界访问
```

## 三、注意事项与常见陷阱

1. **依赖短路的副作用可能不执行**：`false && func()`中func不会被调用
2. **检查顺序很重要**：`ptr && *ptr > 0`必须先检查ptr
3. **不要在短路表达式中执行必要操作**：初始化等必须执行的操作不应放在短路表达式中
4. **没有序列点问题**：`&&`和`||`保证从左到右求值
5. **性能考虑**：把更可能提前终止的条件放在左边
