# 指向成员的指针

## 一、概念说明

成员指针是指向类的成员（数据成员或成员函数）的特殊指针。它存储的是**成员在类中的偏移量**，而非绝对地址。使用时需通过对象实例解引用。

## 二、具体用法

### 2.1 成员数据指针

```cpp
class Point {
public:
    int x, y;
    Point(int x, int y) : x(x), y(y) {}
};

// 声明：指向Point的int成员的指针
int Point::*px = &Point::x;
int Point::*py = &Point::y;

Point p(10, 20);
std::cout << p.*px << std::endl;  // 输出: 10
std::cout << p.*py << std::endl;  // 输出: 20

// 通过指针使用
Point* ptr = &p;
std::cout << ptr->*px << std::endl;  // 输出: 10
```

### 2.2 成员函数指针

```cpp
class Calculator {
public:
    int add(int a, int b) const { return a + b; }
    int mul(int a, int b) const { return a * b; }
};

// 声明：指向Calculator的int(int,int) const成员函数的指针
int (Calculator::*op)(int, int) const = &Calculator::add;

Calculator calc;
std::cout << (calc.*op)(3, 4) << std::endl;  // 输出: 7

op = &Calculator::mul;
std::cout << (calc.*op)(3, 4) << std::endl;  // 输出: 12
```

### 2.3 使用using简化

```cpp
using MemberFunc = int (Calculator::*)(int, int) const;

MemberFunc f = &Calculator::add;
Calculator calc;
std::cout << (calc.*f)(5, 6) << std::endl;  // 输出: 11
```

### 2.4 成员指针表

```cpp
struct Animal {
    void speak() { std::cout << "...\n"; }
    void eat()   { std::cout << "munch\n"; }
    void sleep() { std::cout << "zzz\n"; }
};

using Action = void (Animal::*)();
Action actions[] = {&Animal::speak, &Animal::eat, &Animal::sleep};

Animal pet;
for (auto act : actions) {
    (pet.*act)();
}
// 输出:
// ...
// munch
// zzz
```

## 三、注意事项与常见陷阱

- `&ClassName::member`必须用`&`，不能省略
- `.*`用于对象，`->*`用于指针
- 成员函数指针与虚函数兼容（通过指针调用虚函数仍触发动态分派）
- 静态成员使用普通指针，不是成员指针
- 成员指针不能指向重载函数，除非显式转换
