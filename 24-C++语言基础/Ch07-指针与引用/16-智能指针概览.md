# 智能指针概览

## 一、概念说明

C++11提供了三种智能指针，定义在`<memory>`中：`unique_ptr`（独占所有权）、`shared_ptr`（共享所有权）、`weak_ptr`（弱引用，不增加引用计数）。智能指针通过**RAII**自动管理动态内存，避免内存泄漏。

## 二、具体用法

### 2.1 unique_ptr（独占）

```cpp
#include <memory>

// 推荐用make_unique（C++14）
auto p1 = std::make_unique<int>(42);
std::cout << *p1 << std::endl;  // 输出: 42

// 独占所有权：不可拷贝
auto p2 = std::move(p1);  // 移动所有权
// p1现在为空
std::cout << (p1 == nullptr) << std::endl;  // 输出: 1
std::cout << *p2 << std::endl;  // 输出: 42
// 离开作用域自动释放
```

### 2.2 shared_ptr（共享）

```cpp
auto p1 = std::make_shared<int>(42);
{
    auto p2 = p1;  // 共享所有权，引用计数+1
    std::cout << "引用计数: " << p1.use_count() << std::endl;
    // 输出: 引用计数: 2
}
// p2销毁，引用计数-1
std::cout << "引用计数: " << p1.use_count() << std::endl;
// 输出: 引用计数: 1
```

### 2.3 weak_ptr（弱引用）

```cpp
std::weak_ptr<int> wp;
{
    auto sp = std::make_shared<int>(42);
    wp = sp;

    std::cout << "未过期: " << !wp.expired() << std::endl;
    // 输出: 未过期: 1

    if (auto locked = wp.lock()) {
        std::cout << *locked << std::endl;  // 输出: 42
    }
}
// sp销毁
std::cout << "已过期: " << wp.expired() << std::endl;
// 输出: 已过期: 1
```

### 2.4 选择指南

```cpp
// 唯一所有权 → unique_ptr
auto file = std::unique_ptr<FILE, decltype(&fclose)>(
    fopen("data.txt", "r"), &fclose);

// 共享所有权 → shared_ptr
auto data = std::make_shared<BigData>();
auto viewer1 = data;  // 共享
auto viewer2 = data;  // 共享

// 避免循环引用 → weak_ptr
struct Node {
    std::shared_ptr<Node> next;
    std::weak_ptr<Node> prev;  // 避免循环引用
};
```

## 三、注意事项与常见陷阱

- 优先使用`make_unique`/`make_shared`而非`new`
- `shared_ptr`引用计数有开销，不需要共享时用`unique_ptr`
- 循环引用导致`shared_ptr`内存泄漏，用`weak_ptr`打破
- 不要用裸指针创建多个`shared_ptr`，会导致双重释放
- `unique_ptr`是零开销抽象，性能等同裸指针
