# 多级指针

## 一、概念说明

多级指针是指向指针的指针。`int** pp`存储的是`int*`变量的地址。二级指针常用于函数中修改指针本身（而非指针指向的值），以及动态二维数组。

## 二、具体用法

### 2.1 二级指针基础

```cpp
int x = 42;
int* p = &x;
int** pp = &p;

std::cout << "x  = " << x << std::endl;      // 输出: 42
std::cout << "*p = " << *p << std::endl;      // 输出: 42
std::cout << "**pp = " << **pp << std::endl;  // 输出: 42

// 修改值
**pp = 100;
std::cout << "x = " << x << std::endl;  // 输出: 100
```

### 2.2 用于函数中修改指针

```cpp
// 通过二级指针修改调用者的指针
void allocate(int** pp, int size) {
    *pp = new int[size];
    for (int i = 0; i < size; ++i) {
        (*pp)[i] = i * 10;
    }
}

int* arr = nullptr;
allocate(&arr, 5);
for (int i = 0; i < 5; ++i) std::cout << arr[i] << " ";
// 输出: 0 10 20 30 40
delete[] arr;
```

### 2.3 动态二维数组

```cpp
int rows = 3, cols = 4;
int** mat = new int*[rows];
for (int i = 0; i < rows; ++i) {
    mat[i] = new int[cols];
    for (int j = 0; j < cols; ++j) {
        mat[i][j] = i * cols + j;
    }
}

std::cout << mat[1][2] << std::endl;  // 输出: 6

// 释放
for (int i = 0; i < rows; ++i) delete[] mat[i];
delete[] mat;
```

### 2.4 三级及以上（罕见）

```cpp
int x = 42;
int* p = &x;
int** pp = &p;
int*** ppp = &pp;

std::cout << ***ppp << std::endl;  // 输出: 42
// 实际项目中极少需要三级以上指针
```

## 三、注意事项与常见陷阱

- 多级指针使代码难以理解，优先考虑使用引用或智能指针
- 二级指针用于修改指针时，引用更清晰（C++中推荐）
- 动态二维数组用`std::vector<std::vector<int>>`更安全
- 逐级解引用时每级都需检查是否为空
- 多级指针的声明从右向左阅读：`int** pp` = 指向`int*`的指针
