# 多级指针

## 一、概念说明

多级指针是指向指针的指针。`int** pp`存储的是`int*`变量的地址。二级指针常用于函数中修改指针本身（而非指针指向的值），以及动态二维数组。

声明规则：每增加一级，多一个`*`。`int***`是三级指针（指向`int**`的指针）。

在现代C++中，多级指针的使用场景大幅减少，可以用引用、智能指针和标准容器替代。

## 二、具体用法

### 2.1 二级指针基础

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int* p = &x;
    int** pp = &p;

    // 三级解引用
    cout << "x  = " << x << endl;      // 42
    cout << "*p = " << *p << endl;      // 42
    cout << "**pp = " << **pp << endl;  // 42

    // 通过二级指针修改值
    **pp = 100;
    cout << "x = " << x << endl;       // 100

    // 内存布局
    cout << "pp  = " << pp << endl;    // p的地址
    cout << "*pp = " << *pp << endl;   // x的地址（即p的值）
    cout << "&p  = " << &p << endl;    // 同上

    return 0;
}
```

### 2.2 用于函数中修改指针

```cpp
#include <iostream>
using namespace std;

// 通过二级指针修改调用者的指针
void allocate(int** pp, int size) {
    *pp = new int[size];
    for (int i = 0; i < size; ++i) {
        (*pp)[i] = i * 10;
    }
}

// 现代C++替代：使用引用或智能指针
void allocateModern(int*& pRef, int size) {
    pRef = new int[size];
    for (int i = 0; i < size; ++i) {
        pRef[i] = i * 10;
    }
}

// 最佳：使用unique_ptr
#include <memory>
unique_ptr<int[]> allocateSmart(int size) {
    auto arr = make_unique<int[]>(size);
    for (int i = 0; i < size; ++i) {
        arr[i] = i * 10;
    }
    return arr;
}

int main() {
    int* arr = nullptr;
    allocate(&arr, 5);
    for (int i = 0; i < 5; ++i) cout << arr[i] << " ";
    cout << endl;  // 0 10 20 30 40
    delete[] arr;

    // 引用版本
    int* arr2 = nullptr;
    allocateModern(arr2, 5);
    delete[] arr2;

    // unique_ptr版本
    auto arr3 = allocateSmart(5);
    for (int i = 0; i < 5; ++i) cout << arr3[i] << " ";
    cout << endl;

    return 0;
}
```

### 2.3 动态二维数组

```cpp
#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 4;

    // 指针数组方式
    int** mat = new int*[rows];
    for (int i = 0; i < rows; ++i) {
        mat[i] = new int[cols];
        for (int j = 0; j < cols; ++j) {
            mat[i][j] = i * cols + j;
        }
    }

    cout << mat[1][2] << endl;  // 6

    // 释放（逆序）
    for (int i = 0; i < rows; ++i) delete[] mat[i];
    delete[] mat;

    // 现代替代：vector<vector<int>>
    #include <vector>
    vector<vector<int>> vmat(rows, vector<int>(cols, 0));
    vmat[1][2] = 6;
    // 自动释放

    return 0;
}
```

### 2.4 三级及以上（罕见）

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int* p = &x;
    int** pp = &p;
    int*** ppp = &pp;

    cout << ***ppp << endl;  // 42

    // 实际项目中极少需要三级以上指针
    // 如果需要，通常说明设计有问题，应考虑重构

    return 0;
}
```

### 2.5 指针数组 vs 数组指针

```cpp
#include <iostream>
using namespace std;

int main() {
    // 指针数组：数组的元素是指针
    int a = 1, b = 2, c = 3;
    int* ptrArr[3] = {&a, &b, &c};  // 3个int*的数组

    for (int i = 0; i < 3; ++i)
        cout << *ptrArr[i] << " ";  // 1 2 3
    cout << endl;

    // 数组指针：指向数组的指针
    int arr[3] = {10, 20, 30};
    int (*arrPtr)[3] = &arr;  // 指向含3个int的数组的指针

    cout << (*arrPtr)[1] << endl;  // 20

    // 二维数组与指针
    int matrix[2][3] = {{1,2,3}, {4,5,6}};
    int (*row)[3] = matrix;  // 指向第一行（含3个int的数组）
    cout << row[1][2] << endl;  // 6

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **多级指针使代码难以理解**：优先考虑使用引用或智能指针
2. **二级指针用于修改指针时，引用更清晰**：`void f(int*& p)` 比 `void f(int** pp)` 更直观
3. **动态二维数组用vector<vector<int>>更安全**：自动管理内存
4. **逐级解引用时每级都需检查是否为空**：避免空指针解引用
5. **多级指针的声明从右向左阅读**：`int** pp` = 指向`int*`的指针
6. **指针数组和数组指针语法不同**：`int* arr[3]` vs `int (*arr)[3]`
7. **现代C++中几乎不需要二级以上的指针**：用引用+智能指针替代
