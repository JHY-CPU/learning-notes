# 函数指针数组

## 一、概念说明

函数指针数组存储多个函数指针，允许通过下标选择要调用的函数。常用于实现**调度表**（dispatch table）、状态机、命令处理器等模式。

## 二、具体用法

### 2.1 基本函数指针数组

```cpp
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }

// 函数指针数组
using Op = int(*)(int, int);
Op ops[] = {add, sub, mul};
const char* names[] = {"+", "-", "*"};

int a = 10, b = 3;
for (int i = 0; i < 3; ++i) {
    std::cout << a << " " << names[i] << " " << b
              << " = " << ops[i](a, b) << std::endl;
}
// 输出:
// 10 + 3 = 13
// 10 - 3 = 7
// 10 * 3 = 30
```

### 2.2 命令调度器

```cpp
void cmdHelp()  { std::cout << "帮助信息\n"; }
void cmdQuit()  { std::cout << "退出\n"; }
void cmdRun()   { std::cout << "运行中...\n"; }

using CmdFunc = void(*)();
struct Command { const char* name; CmdFunc func; };

Command commands[] = {
    {"help", cmdHelp},
    {"quit", cmdQuit},
    {"run",  cmdRun}
};

// 查找并执行命令
std::string input = "run";
for (const auto& cmd : commands) {
    if (input == cmd.name) {
        cmd.func();
        break;
    }
}
// 输出: 运行中...
```

### 2.3 状态机

```cpp
enum State { IDLE, RUNNING, PAUSED };

void idleAction()   { std::cout << "空闲\n"; }
void runAction()    { std::cout << "运行\n"; }
void pausedAction() { std::cout << "暂停\n"; }

using Action = void(*)();
Action stateActions[] = {idleAction, runAction, pausedAction};

State current = RUNNING;
stateActions[current]();  // 输出: 运行
```

### 2.4 lambda数组

```cpp
// 无捕获lambda可放入函数指针数组
using Handler = void(*)(int);
Handler handlers[] = {
    [](int x) { std::cout << "处理A: " << x << std::endl; },
    [](int x) { std::cout << "处理B: " << x << std::endl; }
};

handlers[0](10);  // 输出: 处理A: 10
handlers[1](20);  // 输出: 处理B: 20
```

## 三、注意事项与常见陷阱

- 函数指针数组的元素必须是同一类型（返回类型+参数列表一致）
- 数组下标越界是未定义行为
- 有捕获lambda不能放入函数指针数组，需用`std::function`数组
- `std::array`比原生数组更安全（自带size和迭代器）
- 大量函数选择时switch语句可能更高效（编译器优化）
