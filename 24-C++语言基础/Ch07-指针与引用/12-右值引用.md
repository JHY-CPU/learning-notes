# 右值引用

## 一、概念说明

右值引用（`T&&`）是C++11引入的新引用类型，用于绑定到**右值**（临时对象、`std::move`后的对象）。右值引用的主要用途是实现**移动语义**——高效地转移资源所有权而非拷贝。

## 二、具体用法

### 2.1 基本右值引用

```cpp
int x = 42;

// 左值引用绑定左值
int& lref = x;  // OK

// 右值引用绑定右值
int&& rref = 100;  // OK：字面量是右值
// int&& rref2 = x;  // 编译错误：x是左值

std::cout << rref << std::endl;  // 输出: 100
```

### 2.2 移动构造函数

```cpp
class MyString {
    char* data;
    size_t len;
public:
    // 拷贝构造：深拷贝
    MyString(const MyString& other) : len(other.len) {
        data = new char[len + 1];
        std::memcpy(data, other.data, len + 1);
        std::cout << "拷贝构造\n";
    }

    // 移动构造：转移资源所有权
    MyString(MyString&& other) noexcept
        : data(other.data), len(other.len) {
        other.data = nullptr;  // 源对象置空
        other.len = 0;
        std::cout << "移动构造\n";
    }
};

MyString s1("Hello");
MyString s2 = std::move(s1);  // 调用移动构造
// 输出: 移动构造
```

### 2.3 std::move

```cpp
std::string s1 = "Hello World";
std::string s2 = std::move(s1);  // 移动s1到s2

std::cout << "s2: " << s2 << std::endl;
// 输出: s2: Hello World

// s1处于"已移动"状态，有效但未指定
std::cout << "s1: '" << s1 << "'" << std::endl;
// 输出: s1: '' (可能为空)
```

### 2.4 移动 vs 拷贝

```cpp
std::vector<std::string> v1 = {"one", "two", "three"};
std::vector<std::string> v2;

// 拷贝：复制所有字符串
v2 = v1;  // 三次string拷贝

// 移动：转移所有字符串（更快）
v2 = std::move(v1);  // 三次string移动
// v1现在为空
```

## 三、注意事项与常见陷阱

- `std::move`不移动任何东西，只是将左值转为右值引用
- 移动后的对象处于**有效但未指定**状态，只可销毁或赋新值
- 移动操作应标记`noexcept`，否则容器可能选择拷贝
- 右值引用本身是**左值**（有名字）
- 右值引用延长了临时对象的寿命
