# 指针与引用概述

## 一、概念说明

**指针**是存储内存地址的变量（C++11标准 §8.3.1），通过`*`解引用运算符访问指向的对象。**引用**是已有变量的别名（C++11标准 §8.3.2），创建后不可重新绑定。两者都提供了间接访问对象的方式，但语义和安全性有显著区别。

## 二、指针 vs 引用对比表

| 特性 | 指针（T*） | 引用（T&） | 右值引用（T&&） |
|------|-----------|-----------|----------------|
| 可为空 | 可以（nullptr） | 不可以 | 不可以 |
| 可重新绑定 | 可以 | 不可以 | 不可以 |
| 需要解引用 | 需要（*ptr） | 不需要 | 不需要 |
| 占用空间 | 指针大小（4/8字节） | 通常实现为指针 | 同左值引用 |
| 多级 | 可以（T**） | 不可以 | 不可以 |
| 数组运算 | 支持指针算术 | 不支持 | 不支持 |
| 绑定到临时对象 | 不可以 | const引用可以 | 可以 |

## 三、选择策略

```cpp
#include <iostream>
#include <memory>
#include <string>
#include <vector>
using namespace std;

// 函数参数：优先const引用（零拷贝，不可修改）
void process(const string& str) {
    cout << str << endl;
}

// 需要修改原值：非const左值引用
void increment(int& value) {
    value++;
}

// 可选参数或可为空：指针或智能指针
void setCallback(void (*cb)(int)) {  // 可以为nullptr
    if (cb) cb(42);
}

// 多态：指针或引用
struct Shape { virtual ~Shape() = default; virtual void draw() const = 0; };
void render(const Shape& shape) { shape.draw(); }   // 引用：不能为空
void renderOpt(Shape* shape) { if (shape) shape->draw(); }  // 指针：可为空

// 资源管理：智能指针
auto widget = make_unique<string>("独占所有权");
auto shared = make_shared<string>("共享所有权");

// 移动语义：右值引用
void consume(string&& s) {
    cout << "消费: " << s << endl;
}
```

## 四、本章内容概览

- **基础篇**：指针基础、运算、与数组关系、多级指针、void指针
- **引用篇**：引用基础、const引用、右值引用、万能引用（forwarding reference）

- **安全与实践**：悬垂指针/引用、指针安全、最佳实践、结构体指针

## 五、现代C++建议

1. **优先使用引用而非指针**：引用不可为空，更安全
2. **优先使用智能指针管理动态内存**：避免手动new/delete
3. **使用`std::span`（C++20）代替指针+大小的函数参数**：安全的数组视图
4. **使用`std::optional`代替空指针语义**：明确表达"可能没有值"
