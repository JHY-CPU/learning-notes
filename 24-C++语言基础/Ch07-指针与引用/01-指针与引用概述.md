# 指针与引用概述

## 一、概念说明

**指针**是存储内存地址的变量，**引用**是已有变量的别名。两者都提供了间接访问对象的方式，但语义和用法有显著区别。

本章全面覆盖指针基础、智能指针、引用分类（左值/右值/万能引用）、移动语义、以及各种高级指针技术。

## 二、指针 vs 引用对比表

| 特性 | 指针（T*） | 引用（T&） |
|------|-----------|-----------|
| 可为空 | 可以（nullptr） | 不可以 |
| 可重新绑定 | 可以 | 不可以 |
| 需要解引用 | 需要（*ptr） | 不需要 |
| 占用空间 | 指针大小（4/8字节） | 通常不额外占用 |
| 多级 | 可以（T**） | 不可以 |
| 数组 | 指针算术 | 不支持 |

## 三、选择策略

```cpp
// 函数参数：优先const引用
void process(const std::string& str);  // 推荐

// 需要修改原值：非const引用
void increment(int& value);

// 可选参数或可为空：指针或智能指针
void setCallback(void (*cb)(int));  // 可以为nullptr

// 多态：指针或引用
void draw(const Shape& shape);  // 引用
void draw(Shape* shape);        // 指针（可为空）

// 资源管理：智能指针
auto ptr = std::make_unique<Widget>();  // 独占所有权
auto shared = std::make_shared<Widget>();  // 共享所有权
```
