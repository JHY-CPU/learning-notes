# 结构体指针

## 一、概念说明

结构体指针用于间接访问结构体成员，通过`->`运算符替代`(*ptr).member`。结构体指针在链表、树等数据结构中广泛使用。

## 二、具体用法

### 2.1 基本用法

```cpp
struct Point {
    int x, y;
};

Point p = {3, 4};
Point* ptr = &p;

// -> 运算符
std::cout << ptr->x << ", " << ptr->y << std::endl;
// 输出: 3, 4

// 等价于
std::cout << (*ptr).x << ", " << (*ptr).y << std::endl;
// 输出: 3, 4

// 修改成员
ptr->x = 10;
std::cout << p.x << std::endl;  // 输出: 10
```

### 2.2 链表实现

```cpp
struct Node {
    int data;
    Node* next;
    Node(int d) : data(d), next(nullptr) {}
};

// 创建链表
Node* head = new Node(1);
head->next = new Node(2);
head->next->next = new Node(3);

// 遍历
for (Node* cur = head; cur != nullptr; cur = cur->next) {
    std::cout << cur->data << " ";
}
// 输出: 1 2 3

// 释放内存
while (head) {
    Node* temp = head;
    head = head->next;
    delete temp;
}
```

### 2.3 动态分配结构体

```cpp
struct Student {
    std::string name;
    int age;
};

Student* s = new Student{"Alice", 20};
std::cout << s->name << ", " << s->age << std::endl;
// 输出: Alice, 20

delete s;
```

### 2.4 结构体指针数组

```cpp
Point* points[3];
points[0] = new Point{1, 2};
points[1] = new Point{3, 4};
points[2] = new Point{5, 6};

for (auto* p : points) {
    std::cout << "(" << p->x << "," << p->y << ") ";
}
// 输出: (1,2) (3,4) (5,6)

for (auto* p : points) delete p;
```

## 三、注意事项与常见陷阱

- `->`是`(*ptr).member`的语法糖
- 链表等数据结构删除节点时注意不要遗漏中间节点的释放
- 结构体指针的`sizeof`返回指针大小（4或8字节），不是结构体大小
- `nullptr`检查在访问指针前应进行
- 现代C++中链表等考虑使用智能指针
