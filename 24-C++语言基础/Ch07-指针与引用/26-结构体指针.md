# 结构体指针

## 一、概念说明

结构体指针用于间接访问结构体成员，通过`->`运算符替代`(*ptr).member`（C++11标准 §5.2.5）。结构体指针在链表、树等数据结构中广泛使用。

`->`运算符的本质：`ptr->member` 等价于 `(*ptr).member`，是语法糖。

## 二、具体用法

### 2.1 基本用法

```cpp
#include <iostream>
using namespace std;

struct Point {
    int x, y;
};

int main() {
    Point p = {3, 4};
    Point* ptr = &p;

    // -> 运算符
    cout << ptr->x << ", " << ptr->y << endl;  // 3, 4

    // 等价写法
    cout << (*ptr).x << ", " << (*ptr).y << endl;  // 3, 4

    // 修改成员
    ptr->x = 10;
    cout << "p.x = " << p.x << endl;  // 10

    return 0;
}
```

### 2.2 链表实现

```cpp
#include <iostream>
using namespace std;

struct Node {
    int data;
    Node* next;
    Node(int d) : data(d), next(nullptr) {}
};

int main() {
    // 创建链表
    Node* head = new Node(1);
    head->next = new Node(2);
    head->next->next = new Node(3);

    // 遍历
    cout << "链表: ";
    for (Node* cur = head; cur != nullptr; cur = cur->next) {
        cout << cur->data << " ";
    }
    cout << endl;  // 1 2 3

    // 在头部插入
    Node* newNode = new Node(0);
    newNode->next = head;
    head = newNode;

    // 释放内存
    while (head) {
        Node* temp = head;
        head = head->next;
        delete temp;
    }

    return 0;
}
```

### 2.3 动态分配结构体

```cpp
#include <iostream>
#include <string>
#include <memory>
using namespace std;

struct Student {
    string name;
    int age;
    void display() const {
        cout << name << ", " << age << "岁" << endl;
    }
};

int main() {
    // 传统方式
    Student* s = new Student{"Alice", 20};
    s->display();  // Alice, 20岁
    delete s;

    // 现代C++：智能指针
    auto student = make_unique<Student>("Bob", 22);
    student->display();  // Bob, 22岁
    // 自动释放

    // make_unique的->用法与裸指针相同
    student->name = "Charlie";
    student->display();  // Charlie, 22岁

    return 0;
}
```

### 2.4 结构体指针数组

```cpp
#include <iostream>
using namespace std;

struct Point {
    int x, y;
};

int main() {
    // 指针数组：每个元素指向一个结构体
    Point* points[3];
    points[0] = new Point{1, 2};
    points[1] = new Point{3, 4};
    points[2] = new Point{5, 6};

    for (int i = 0; i < 3; ++i) {
        cout << "(" << points[i]->x << "," << points[i]->y << ") ";
    }
    cout << endl;  // (1,2) (3,4) (5,6)

    // 释放
    for (int i = 0; i < 3; ++i) delete points[i];

    // 现代替代：vector
    #include <vector>
    vector<Point> vec = {{1,2}, {3,4}, {5,6}};
    for (const auto& pt : vec) {
        cout << "(" << pt.x << "," << pt.y << ") ";
    }
    cout << endl;

    return 0;
}
```

### 2.5 链表高级操作

```cpp
#include <iostream>
using namespace std;

struct Node {
    int data;
    Node* next;
    Node(int d, Node* n = nullptr) : data(d), next(n) {}
};

// 反转链表
Node* reverse(Node* head) {
    Node* prev = nullptr;
    while (head) {
        Node* next = head->next;
        head->next = prev;
        prev = head;
        head = next;
    }
    return prev;
}

// 查找中间节点
Node* findMiddle(Node* head) {
    Node* slow = head;
    Node* fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}

int main() {
    Node* head = new Node(1, new Node(2, new Node(3, new Node(4))));

    head = reverse(head);  // 4 3 2 1
    Node* mid = findMiddle(head);
    cout << "中间节点: " << mid->data << endl;  // 2

    // 清理
    while (head) {
        Node* temp = head;
        head = head->next;
        delete temp;
    }

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **`->`是`(*ptr).member`的语法糖**：两者等价
2. **链表删除节点时注意不要遗漏中间节点的释放**：先保存next再删除当前
3. **结构体指针的sizeof返回指针大小**（4或8字节），不是结构体大小
4. **nullptr检查在访问指针前应进行**：空指针解引用是UB
5. **现代C++中链表等考虑使用智能指针**：`unique_ptr<Node>`自动管理链表内存
6. **结构体指针可以指向派生类对象**：多态的基础
7. **C++中struct默认public，class默认private**：除此之外完全等价
