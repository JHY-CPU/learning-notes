# 引用基础

## 一、概念说明

引用是变量的**别名**，创建后不可重新绑定到其他变量。引用不是新对象，不占用额外存储（编译器通常实现为指针，但语义不同）。C++中引用分为**左值引用**（`T&`）、**右值引用**（`T&&`）和**const引用**（`const T&`）。

## 二、具体用法

### 2.1 左值引用基础

```cpp
int x = 42;
int& ref = x;  // ref是x的别名

std::cout << "x = " << x << std::endl;      // 输出: 42
std::cout << "ref = " << ref << std::endl;  // 输出: 42

ref = 100;  // 通过引用修改x
std::cout << "x = " << x << std::endl;      // 输出: 100

// ref和x的地址相同
std::cout << "&x = " << &x << std::endl;
std::cout << "&ref = " << &ref << std::endl;  // 同一地址
```

### 2.2 引用必须初始化

```cpp
// int& ref;  // 编译错误：引用必须初始化
int x = 10;
int& ref = x;  // 正确

// 引用不能为空
// int& ref = nullptr;  // 编译错误
```

### 2.3 不可重新绑定

```cpp
int a = 10, b = 20;
int& ref = a;

ref = b;  // 这是将b的值赋给a，不是重新绑定！
std::cout << "a = " << a << std::endl;  // 输出: 20
std::cout << "b = " << b << std::endl;  // 输出: 20
```

### 2.4 引用与函数

```cpp
void increment(int& value) {
    value++;
}

int num = 5;
increment(num);
std::cout << num << std::endl;  // 输出: 6
```

### 2.5 引用作为返回值

```cpp
int& getElement(std::vector<int>& vec, size_t index) {
    return vec[index];
}

std::vector<int> v = {1, 2, 3};
getElement(v, 1) = 99;  // 可以赋值
std::cout << v[1] << std::endl;  // 输出: 99
```

## 三、注意事项与常见陷阱

- 引用必须在声明时初始化，之后不可重新绑定
- 不要返回局部变量的引用（悬垂引用）
- `int& ref = a; ref = b;` 是赋值不是重新绑定
- 引用不是指针，不能为`nullptr`，不能指向数组
- 函数返回引用时确保被引用对象的生命周期足够长
