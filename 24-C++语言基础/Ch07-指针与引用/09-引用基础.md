# 引用基础

## 一、概念说明

引用是变量的**别名**（C++11标准 §8.3.2），创建后不可重新绑定到其他变量。引用不是新对象，不占用额外存储空间（语义上，编译器通常实现为const指针）。C++中引用分为：

| 引用类型 | 语法 | 绑定对象 | 典型用途 |
|---------|------|---------|---------|
| 左值引用 | `T&` | 可修改的左值 | 函数输出参数、操作符重载 |
| const左值引用 | `const T&` | 左值和右值 | 函数输入参数（零拷贝） |
| 右值引用 | `T&&` | 右值（临时对象） | 移动语义、完美转发 |
| 万能引用 | `T&&`（推导中） | 左值或右值 | 完美转发 |

## 二、具体用法

### 2.1 左值引用基础

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int& ref = x;  // ref是x的别名

    cout << "x = " << x << endl;      // 42
    cout << "ref = " << ref << endl;  // 42

    ref = 100;  // 通过引用修改x
    cout << "x = " << x << endl;      // 100

    // ref和x的地址相同
    cout << "&x = " << &x << endl;
    cout << "&ref = " << &ref << endl;  // 同一地址

    return 0;
}
```

### 2.2 引用必须初始化且不可重新绑定

```cpp
#include <iostream>
using namespace std;

int main() {
    // int& ref;  // 编译错误：引用必须初始化
    // int& ref = nullptr;  // 编译错误：引用不能为空

    int a = 10, b = 20;
    int& ref = a;

    ref = b;  // 这是将b的值赋给a，不是重新绑定！
    cout << "a = " << a << endl;  // 20
    cout << "b = " << b << endl;  // 20
    cout << "&ref = " << &ref << endl;  // 仍是a的地址

    return 0;
}
```

### 2.3 const引用

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // const引用可以绑定到临时对象
    const int& ref1 = 42;        // OK：临时int绑定到const引用
    const int& ref2 = 3.14;      // OK：double转int后绑定（临时对象延长生命周期）
    const string& ref3 = "hi";   // OK：字面量创建临时string

    // 非const引用不能绑定临时对象
    // int& bad = 42;  // 编译错误

    // const引用延长临时对象生命周期
    const string& s = string("hello") + " world";
    cout << s << endl;  // hello world
    // 临时对象生命周期延长到s的生命周期结束

    return 0;
}
```

### 2.4 引用与函数

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

// const引用输入参数（零拷贝）
void process(const string& str) {
    cout << "长度: " << str.size() << endl;
}

// 非const引用输出参数
void increment(int& value) {
    value++;
}

// 引用返回值
int& getElement(vector<int>& vec, size_t index) {
    return vec[index];
}

int main() {
    string s = "hello";
    process(s);  // 不拷贝

    int x = 5;
    increment(x);
    cout << x << endl;  // 6

    vector<int> v = {1, 2, 3};
    getElement(v, 1) = 99;  // 可以赋值
    cout << v[1] << endl;  // 99

    return 0;
}
```

### 2.5 右值引用（C++11）

```cpp
#include <iostream>
#include <utility>
using namespace std;

void process(int& x)  { cout << "左值: " << x << endl; }
void process(int&& x) { cout << "右值: " << x << endl; }

int main() {
    int x = 42;

    process(x);       // 左值版本
    process(42);      // 右值版本
    process(move(x)); // 右值版本（move将左值转为右值引用）

    // 右值引用变量本身是左值
    int&& rref = 100;
    process(rref);    // 左值版本（rref有名字，是左值）

    return 0;
}
```

### 2.6 引用折叠规则（C++11）

```cpp
// 引用折叠规则（用于模板和完美转发）
// T&  &  → T&
// T&  && → T&
// T&& &  → T&
// T&& && → T&&
// 规律：出现一个&，结果就是&
```

## 三、注意事项与常见陷阱

1. **引用必须在声明时初始化**：之后不可重新绑定到其他对象
2. **不要返回局部变量的引用**：局部变量在函数返回后销毁，导致悬垂引用
3. **`int& ref = a; ref = b;` 是赋值不是重新绑定**：a的值变为b的值，ref仍指向a
4. **引用不是指针，不能为nullptr，不能指向数组**：更安全但受限
5. **函数返回引用时确保被引用对象的生命周期足够长**：通常返回成员变量或参数的引用
6. **const引用可以绑定临时对象并延长其生命周期**：这是const引用的重要特性
7. **右值引用变量本身是左值**：有名字的右值引用变量是左值表达式
