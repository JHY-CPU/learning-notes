# 指针运算

## 一、概念说明

指针支持有限的算术运算（C++11标准 §5.7）：自增（`++`）、自减（`--`）、加减整数（`+n`、`-n`）、指针相减（`p1 - p2`）、指针比较。这些运算以**指向类型的大小**为单位，而非字节。

指针运算仅对指向数组元素或数组末尾后一个位置的指针有定义，越界操作是未定义行为。

## 二、具体用法

### 2.1 指针自增自减

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int* p = arr;  // 指向arr[0]

    cout << *p << endl;    // 10
    p++;                    // 移动sizeof(int)字节，到arr[1]
    cout << *p << endl;    // 20
    p += 2;                 // 移动2个int，到arr[3]
    cout << *p << endl;    // 40
    p--;                    // 回到arr[2]
    cout << *p << endl;    // 30

    // double指针：每次移动8字节（64位系统）
    double dArr[] = {1.1, 2.2, 3.3};
    double* dp = dArr;
    dp++;
    cout << *dp << endl;   // 2.2

    return 0;
}
```

### 2.2 指针加减整数

```cpp
#include <iostream>
using namespace std;

int main() {
    double arr[] = {1.1, 2.2, 3.3, 4.4};
    double* p = arr;

    cout << *(p + 0) << endl;  // 1.1
    cout << *(p + 2) << endl;  // 3.3
    cout << *(p + 3) << endl;  // 4.4

    // p + n 等价于 &arr[n]
    // *(p + n) 等价于 arr[n]
    cout << arr[2] << endl;       // 3.3
    cout << *(arr + 2) << endl;   // 3.3

    // 负数偏移（需确保指针仍在数组范围内）
    double* end = &arr[3];
    cout << *(end - 1) << endl;  // 3.3

    return 0;
}
```

### 2.3 指针相减

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int* p1 = &arr[1];
    int* p2 = &arr[4];

    ptrdiff_t distance = p2 - p1;  // ptrdiff_t是有符号整数类型
    cout << "距离: " << distance << endl;  // 3（4-1=3个int单位）

    // 注意：只有指向同一数组的指针才能相减
    int other = 99;
    int* p3 = &other;
    // p2 - p3;  // 未定义行为：不在同一数组

    return 0;
}
```

### 2.4 指针遍历数组

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int* begin = arr;
    int* end = arr + 5;  // 指向末尾后一位（哨兵）

    // 指针遍历（STL迭代器的底层原理）
    for (int* p = begin; p != end; ++p) {
        cout << *p << " ";
    }
    cout << endl;  // 10 20 30 40 50

    // 反向遍历
    for (int* p = end - 1; p >= begin; --p) {
        cout << *p << " ";
    }
    cout << endl;  // 50 40 30 20 10

    // 使用std::begin/std::end（C++11）
    for (int* p = std::begin(arr); p != std::end(arr); ++p) {
        cout << *p << " ";
    }
    cout << endl;

    return 0;
}
```

### 2.5 指针比较

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {1, 2, 3};
    int* p1 = &arr[0];
    int* p2 = &arr[2];

    // 相等性比较
    cout << (p1 == &arr[0]) << endl;  // 1

    // 关系比较（仅对同一数组有意义）
    cout << (p1 < p2) << endl;   // 1（p1在p2前面）
    cout << (p1 > p2) << endl;   // 0

    // nullptr比较
    int* np = nullptr;
    if (!np) cout << "np为空\n";
    if (np == nullptr) cout << "np为nullptr\n";

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **指针运算以类型大小为单位**：`p+1`移动`sizeof(T)`字节，不是1字节
2. **对不指向同一数组的指针进行减法或比较是未定义行为**
3. **指针超出数组范围后进行解引用是未定义行为**：但可以指向末尾后一位作为哨兵
4. **`p[n]`等价于`*(p+n)`**：下标运算对指针也有效
5. **`void*`不支持算术运算**：不知道类型大小，无法计算偏移
6. **`ptrdiff_t`是指针相减的结果类型**：有符号整数，可能与`int`大小不同
7. **使用`std::begin`/`std::end`替代手动计算**：更安全、更清晰
