# 指针运算

## 一、概念说明

指针支持有限的算术运算：自增（`++`）、自减（`--`）、加减整数（`+n`、`-n`）、指针相减（`p1 - p2`）。这些运算以**指向类型的大小**为单位，而非字节。

## 二、具体用法

### 2.1 指针自增自减

```cpp
int arr[] = {10, 20, 30, 40, 50};
int* p = arr;  // 指向arr[0]

std::cout << *p << std::endl;    // 输出: 10
p++;                                // 移动到arr[1]
std::cout << *p << std::endl;    // 输出: 20
p += 2;                             // 移动到arr[3]
std::cout << *p << std::endl;    // 输出: 40
p--;                                // 回到arr[2]
std::cout << *p << std::endl;    // 输出: 30
```

### 2.2 指针加减整数

```cpp
double arr[] = {1.1, 2.2, 3.3, 4.4};
double* p = arr;

std::cout << *(p + 0) << std::endl;  // 输出: 1.1
std::cout << *(p + 2) << std::endl;  // 输出: 3.3
std::cout << *(p + 3) << std::endl;  // 输出: 4.4

// p + n 等价于 &arr[n]
```

### 2.3 指针相减

```cpp
int arr[] = {10, 20, 30, 40, 50};
int* p1 = &arr[1];
int* p2 = &arr[4];

std::cout << "距离: " << (p2 - p1) << std::endl;
// 输出: 距离: 3 （4-1=3个int单位）
```

### 2.4 指针遍历数组

```cpp
int arr[] = {10, 20, 30, 40, 50};
int* begin = arr;
int* end = arr + 5;

for (int* p = begin; p != end; ++p) {
    std::cout << *p << " ";
}
// 输出: 10 20 30 40 50
```

### 2.5 指针比较

```cpp
int arr[] = {1, 2, 3};
int* p1 = &arr[0];
int* p2 = &arr[2];

std::cout << (p1 < p2) << std::endl;   // 输出: 1 (p1在前面)
std::cout << (p1 == &arr[0]) << std::endl;  // 输出: 1
```

## 三、注意事项与常见陷阱

- 指针运算以**类型大小**为单位，`p+1`移动`sizeof(T)`字节
- 对不指向同一数组的指针进行减法或比较是未定义行为
- 指针超出数组范围后进行解引用是未定义行为
- `p[n]`等价于`*(p+n)`，下标运算对指针也有效
- `void*`不支持算术运算（不知道类型大小）
