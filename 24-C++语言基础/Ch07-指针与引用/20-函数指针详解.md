# 函数指针详解

## 一、概念说明

函数指针存储函数的入口地址，允许在运行时动态选择要调用的函数。函数指针的类型由**返回类型**和**参数列表**共同决定，与函数名无关。

## 二、具体用法

### 2.1 声明与赋值

```cpp
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }

// 声明函数指针
int (*op)(int, int) = add;

std::cout << op(10, 3) << std::endl;  // 输出: 13

op = sub;
std::cout << op(10, 3) << std::endl;  // 输出: 7
```

### 2.2 using/typedef简化

```cpp
// using（推荐）
using BinaryOp = int(*)(int, int);

// typedef
typedef int (*BinaryOp2)(int, int);

BinaryOp fn = add;
std::cout << fn(5, 3) << std::endl;  // 输出: 8
```

### 2.3 函数指针数组（调度表）

```cpp
using Op = int(int, int);
Op* table[] = {add, sub};

std::cout << table[0](20, 5) << std::endl;  // 输出: 25
std::cout << table[1](20, 5) << std::endl;  // 输出: 15
```

### 2.4 回调函数

```cpp
void apply(int* arr, size_t n, int (*transform)(int)) {
    for (size_t i = 0; i < n; ++i) {
        arr[i] = transform(arr[i]);
    }
}

int square(int x) { return x * x; }

int data[] = {1, 2, 3, 4, 5};
apply(data, 5, square);
for (int x : data) std::cout << x << " ";
// 输出: 1 4 9 16 25
```

### 2.5 Lambda转函数指针

```cpp
// 无捕获lambda可转为函数指针
int (*fn)(int) = [](int x) { return x * 2; };
std::cout << fn(21) << std::endl;  // 输出: 42

// 有捕获lambda不可转为函数指针
// int (*fn2)(int) = [factor](int x) { return x * factor; };  // 编译错误
```

### 2.6 返回函数指针的函数

```cpp
using Op = int(*)(int, int);

Op getOp(const std::string& name) {
    if (name == "add") return add;
    if (name == "sub") return sub;
    return nullptr;
}

auto fn = getOp("add");
if (fn) std::cout << fn(3, 4) << std::endl;  // 输出: 7
```

## 三、注意事项与常见陷阱

- 函数指针类型严格匹配返回类型和参数列表
- 空函数指针调用是未定义行为
- 成员函数指针语法不同（需要`&ClassName::func`）
- 无捕获lambda可隐式转换为函数指针
- `std::function`可包装有捕获lambda和仿函数
