# 指针基础

## 一、概念说明

指针是存储**内存地址**的变量（C++11标准 §8.3.1）。通过`&`取地址运算符获取变量地址，通过`*`解引用运算符访问地址指向的值。指针的类型决定了如何解释该地址处的数据（类型安全）。

关键概念：
- 指针本身也是一个变量，占用内存（32位系统4字节，64位系统8字节）
- 指针类型决定解引用时读取多少字节以及如何解释
- `nullptr`是C++11引入的空指针字面量，类型为`std::nullptr_t`

## 二、具体用法

### 2.1 声明与初始化

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int* ptr = &x;  // ptr存储x的地址

    cout << "x的值: " << x << endl;         // 42
    cout << "x的地址: " << &x << endl;      // 0x7ffd...
    cout << "ptr的值: " << ptr << endl;     // 同上
    cout << "ptr指向的值: " << *ptr << endl; // 42

    // 指针自身的地址
    cout << "ptr的地址: " << &ptr << endl;   // ptr自身在内存中的位置

    return 0;
}
```

### 2.2 解引用与修改

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    int* p = &x;

    *p = 20;        // 通过指针修改x
    cout << "x = " << x << endl;  // 20

    *p += 5;
    cout << "x = " << x << endl;  // 25

    // 解引用是左值表达式，可以出现在赋值左侧
    (*p)++;
    cout << "x = " << x << endl;  // 26

    return 0;
}
```

### 2.3 空指针

```cpp
#include <iostream>
using namespace std;

int main() {
    int* p1 = nullptr;     // C++11推荐
    int* p2 = NULL;        // C风格（不推荐）
    int* p3 = 0;           // 等价于NULL（不推荐）

    // 检查空指针
    if (p1 == nullptr) {
        cout << "p1是空指针\n";
    }

    // 安全解引用
    if (p1 != nullptr) {
        cout << *p1 << endl;  // 不会执行
    }

    // C++11后始终使用nullptr，不要用NULL或0
    // nullptr可以隐式转换为任何指针类型，但不能转为整数
    int x = 42;
    p1 = &x;
    if (p1) cout << "p1非空: " << *p1 << endl;  // 42

    return 0;
}
```

### 2.4 指针类型

```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 42;
    double d = 3.14;

    int* pi = &i;
    double* pd = &d;
    // pi = pd;  // 编译错误：类型不兼容

    // void指针可指向任意类型（但丢失类型信息）
    void* pv = &i;
    int* pi2 = static_cast<int*>(pv);  // 需要显式转换
    cout << *pi2 << endl;  // 42

    // const指针与指针const
    const int ci = 100;
    const int* cp = &ci;       // 指向const int的指针（不能通过指针修改值）
    int* const pc = &i;        // const指针（指针不能重新指向）
    const int* const ccp = &ci; // 指向const的const指针

    *const_cast<int*>(cp) = 200;  // 去掉const（仅原始非const对象安全）

    return 0;
}
```

### 2.5 指针与sizeof

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 42;
    int* p = &x;

    cout << "sizeof(x) = " << sizeof(x) << endl;   // 4（int大小）
    cout << "sizeof(p) = " << sizeof(p) << endl;   // 8（64位系统指针大小）
    cout << "sizeof(*p) = " << sizeof(*p) << endl;  // 4（int大小）

    // 指针大小与类型无关
    char* cp = nullptr;
    double* dp = nullptr;
    cout << "sizeof(char*) = " << sizeof(cp) << endl;   // 8
    cout << "sizeof(double*) = " << sizeof(dp) << endl; // 8

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **未初始化的指针（野指针）包含随机地址**：解引用是未定义行为，务必初始化为nullptr
2. **`*`的位置不影响语义**：`int* p`、`int *p`、`int * p`都正确，但推荐 `int* p`
3. **声明多个指针时每个都需要`*`**：`int *p1, p2;` 中p2是int不是指针
4. **解引用空指针或已释放指针导致程序崩溃**：使用前始终检查
5. **指针值可以用cout输出为地址**：用 `hex` 控制格式
6. **const的三种位置有不同含义**：指针本身const vs 指向内容const vs 两者都const
7. **悬垂指针（dangling pointer）**：指向已被释放的对象，解引用是UB
