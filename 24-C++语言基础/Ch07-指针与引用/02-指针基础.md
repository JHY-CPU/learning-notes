# 指针基础

## 一、概念说明

指针是存储**内存地址**的变量。通过`&`取地址运算符获取变量地址，通过`*`解引用运算符访问地址指向的值。指针的类型决定了如何解释该地址处的数据。

## 二、具体用法

### 2.1 声明与初始化

```cpp
int x = 42;
int* ptr = &x;  // ptr存储x的地址

std::cout << "x的值: " << x << std::endl;
std::cout << "x的地址: " << &x << std::endl;
std::cout << "ptr的值: " << ptr << std::endl;
std::cout << "ptr指向的值: " << *ptr << std::endl;
// 输出:
// x的值: 42
// x的地址: 0x7ffd... (某地址)
// ptr的值: 0x7ffd... (同上)
// ptr指向的值: 42
```

### 2.2 解引用与修改

```cpp
int x = 10;
int* p = &x;

*p = 20;  // 通过指针修改x
std::cout << "x = " << x << std::endl;
// 输出: x = 20

*p += 5;
std::cout << "x = " << x << std::endl;
// 输出: x = 25
```

### 2.3 空指针

```cpp
int* p1 = nullptr;     // C++11推荐
int* p2 = NULL;        // C风格（不推荐）
int* p3 = 0;           // 等价于NULL（不推荐）

// 检查空指针
if (p1 == nullptr) {
    std::cout << "p1是空指针\n";
}
// 输出: p1是空指针

// 安全解引用
if (p1 != nullptr) {
    std::cout << *p1 << std::endl;
}
```

### 2.4 指针类型

```cpp
int i = 42;
double d = 3.14;

int* pi = &i;      // int指针
double* pd = &d;   // double指针
// pi = pd;         // 编译错误：类型不兼容

// void指针可指向任意类型
void* pv = &i;
int* pi2 = static_cast<int*>(pv);  // 需要显式转换
std::cout << *pi2 << std::endl;     // 输出: 42
```

## 三、注意事项与常见陷阱

- 未初始化的指针（野指针）包含随机地址，解引用是未定义行为
- `*`的位置不影响语义：`int* p`、`int *p`、`int * p`都正确
- 声明多个指针时每个都需要`*`：`int *p1, *p2;`
- 解引用空指针或已释放指针导致程序崩溃
- 指针值可以用`std::cout`输出为地址，用`std::hex`控制格式
