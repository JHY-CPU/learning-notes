# 赋值运算符重载

## 一、概念说明

`operator=`（C++11标准 §13.5.3）是特殊的运算符，只能作为**成员函数**重载。它管理已有对象的状态更新，通常返回`*this`的引用以支持链式赋值。

| 类型 | 签名 | 用途 |
|------|------|------|
| 拷贝赋值 | `T& operator=(const T&)` | 深拷贝赋值 |
| 移动赋值 | `T& operator=(T&&) noexcept` | 转移资源所有权 |
| 拷贝交换 | `T& operator=(T)` | 异常安全的赋值 |

## 二、具体用法

### 2.1 拷贝赋值与移动赋值

```cpp
#include <iostream>
#include <algorithm>
#include <utility>
using namespace std;

class Buffer {
    int* data;
    size_t size_;
public:
    Buffer(size_t s) : data(new int[s]()), size_(s) {}
    ~Buffer() { delete[] data; }

    // 拷贝构造
    Buffer(const Buffer& other) : data(new int[other.size_]), size_(other.size_) {
        copy(other.data, other.data + size_, data);
    }

    // 移动构造
    Buffer(Buffer&& other) noexcept : data(other.data), size_(other.size_) {
        other.data = nullptr;
        other.size_ = 0;
    }

    // 拷贝赋值：拷贝交换惯用法（异常安全）
    Buffer& operator=(Buffer other) {  // 按值传参触发拷贝/移动构造
        swap(data, other.data);
        swap(size_, other.size_);
        return *this;
    }

    size_t size() const { return size_; }
};

int main() {
    Buffer a(10), b(5), c(3);
    a = b;              // 拷贝赋值
    b = move(c);        // 移动赋值
    cout << "a.size=" << a.size() << endl;  // 5
    cout << "b.size=" << b.size() << endl;  // 3

    // 链式赋值
    Buffer d(1), e(2), f(3);
    d = e = f;
    cout << "d.size=" << d.size() << endl;  // 3

    return 0;
}
```

### 2.2 自赋值安全

```cpp
#include <iostream>
#include <cstring>
using namespace std;

class String {
    char* str;
    size_t len;
public:
    String(const char* s = "") : len(strlen(s)) {
        str = new char[len + 1];
        strcpy(str, s);
    }

    ~String() { delete[] str; }

    // 自赋值安全的拷贝赋值
    String& operator=(const String& other) {
        if (this != &other) {  // 检查自赋值
            delete[] str;
            len = other.len;
            str = new char[len + 1];
            strcpy(str, other.str);
        }
        return *this;
    }

    friend ostream& operator<<(ostream& os, const String& s) {
        return os << s.str;
    }
};

int main() {
    String s1("hello"), s2("world");
    s1 = s2;        // 正常赋值
    cout << s1 << endl;  // world

    s1 = s1;        // 自赋值：安全处理
    cout << s1 << endl;  // world

    return 0;
}
```

### 2.3 禁止赋值

```cpp
#include <iostream>
using namespace std;

class NonCopyable {
    int* data;
public:
    NonCopyable(int v) : data(new int(v)) {}
    ~NonCopyable() { delete data; }

    // 禁止拷贝和赋值
    NonCopyable(const NonCopyable&) = delete;
    NonCopyable& operator=(const NonCopyable&) = delete;

    // 允许移动
    NonCopyable(NonCopyable&& other) noexcept : data(other.data) {
        other.data = nullptr;
    }
    NonCopyable& operator=(NonCopyable&& other) noexcept {
        if (this != &other) {
            delete data;
            data = other.data;
            other.data = nullptr;
        }
        return *this;
    }

    int get() const { return data ? *data : 0; }
};

int main() {
    NonCopyable a(42);
    // NonCopyable b = a;  // 编译错误：拷贝已删除
    // NonCopyable c(10); c = a;  // 编译错误：赋值已删除

    NonCopyable b = move(a);  // OK：移动
    cout << b.get() << endl;  // 42

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **赋值运算符必须返回 `*this` 的引用**：支持 `a = b = c` 链式赋值
2. **必须处理自赋值情况**：`a = a` 不应出错
3. **拷贝交换惯用法天然提供异常安全**：按值传参+swap
4. **赋值运算符只能是成员函数**：不能作为非成员重载
5. **`=delete`可以禁止赋值操作**：用于不可复制的资源管理类
6. **移动赋值应标记noexcept**：标准库容器要求以启用移动优化
