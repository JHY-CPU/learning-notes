# 赋值运算符重载

## 一、概念说明

`operator=`是特殊的运算符，只能作为**成员函数**重载。它管理已有对象的状态更新，通常返回`*this`的引用以支持链式赋值。

## 二、具体用法

### 2.1 拷贝赋值与移动赋值

```cpp
#include <iostream>
#include <algorithm>
#include <utility>

class Buffer {
    int* data;
    int size_;
public:
    Buffer(int s) : data(new int[s]()), size_(s) {}
    ~Buffer() { delete[] data; }

    // 拷贝赋值：拷贝交换惯用法
    Buffer& operator=(Buffer other) {  // 按值传参触发拷贝构造
        std::swap(data, other.data);
        std::swap(size_, other.size_);
        return *this;
    }

    // 移动赋值
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            delete[] data;
            data = other.data;
            size_ = other.size_;
            other.data = nullptr;
            other.size_ = 0;
        }
        return *this;
    }

    int size() const { return size_; }
};

int main() {
    Buffer a(10), b(5), c(3);
    a = b;              // 拷贝赋值
    b = std::move(c);   // 移动赋值
    std::cout << "a.size=" << a.size() << std::endl;
    std::cout << "b.size=" << b.size() << std::endl;
    return 0;
}
```

**输出：**
```
a.size=5
b.size=3
```

## 三、注意事项与常见陷阱

- 赋值运算符**必须**返回`*this`的引用
- 必须处理**自赋值**情况
- 拷贝交换惯用法天然提供异常安全
- 赋值运算符只能是成员函数，不能是非成员
- `=delete`可以禁止赋值操作
