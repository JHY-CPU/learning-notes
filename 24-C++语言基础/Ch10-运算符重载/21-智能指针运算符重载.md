# 智能指针运算符重载

## 一、概念说明

智能指针需要重载`*`（解引用）、`->`（成员访问）、`=`（赋值）等运算符（C++11标准 §20.7.1），模拟裸指针的行为同时自动管理资源生命周期。

标准库智能指针：
- `std::unique_ptr`：独占所有权，禁止拷贝，允许移动
- `std::shared_ptr`：共享所有权，引用计数
- `std::weak_ptr`：弱引用，不增加引用计数

## 二、具体用法

### 2.1 unique_ptr简化实现

```cpp
#include <iostream>
#include <utility>
using namespace std;

template<typename T>
class UniquePtr {
    T* ptr;
public:
    explicit UniquePtr(T* p = nullptr) : ptr(p) {}
    ~UniquePtr() { delete ptr; }

    // 禁止拷贝
    UniquePtr(const UniquePtr&) = delete;
    UniquePtr& operator=(const UniquePtr&) = delete;

    // 移动语义
    UniquePtr(UniquePtr&& o) noexcept : ptr(o.ptr) { o.ptr = nullptr; }
    UniquePtr& operator=(UniquePtr&& o) noexcept {
        if (this != &o) { delete ptr; ptr = o.ptr; o.ptr = nullptr; }
        return *this;
    }

    // 解引用运算符
    T& operator*() const { return *ptr; }
    T* operator->() const { return ptr; }

    // 布尔转换
    explicit operator bool() const { return ptr != nullptr; }

    // 相等比较
    bool operator==(const UniquePtr& o) const { return ptr == o.ptr; }
    bool operator!=(const UniquePtr& o) const { return ptr != o.ptr; }
    bool operator==(nullptr_t) const { return ptr == nullptr; }

    // 下标运算符（数组特化场景）
    T& operator[](size_t i) const { return ptr[i]; }

    T* get() const { return ptr; }
    T* release() { T* p = ptr; ptr = nullptr; return p; }
    void reset(T* p = nullptr) { delete ptr; ptr = p; }
};

struct Point { int x, y; };

int main() {
    UniquePtr<Point> p1(new Point{3, 4});
    cout << "p1->x=" << p1->x << ", (*p1).y=" << (*p1).y << endl;

    UniquePtr<Point> p2 = move(p1);
    cout << "p1为空: " << !p1 << endl;
    cout << "p2->x=" << p2->x << endl;

    // 链式操作
    p2.reset(new Point{5, 6});
    cout << "p2重置: (" << p2->x << "," << p2->y << ")" << endl;

    return 0;
}
```

### 2.2 shared_ptr核心机制（引用计数）

```cpp
#include <iostream>
#include <atomic>
using namespace std;

// 简化版shared_ptr，展示核心运算符
template<typename T>
class SharedPtr {
    T* ptr;
    atomic<int>* refCount;

    void release() {
        if (refCount && --(*refCount) == 0) {
            delete ptr;
            delete refCount;
        }
    }

public:
    explicit SharedPtr(T* p = nullptr)
        : ptr(p), refCount(p ? new atomic<int>(1) : nullptr) {}

    // 拷贝：增加引用计数
    SharedPtr(const SharedPtr& o) : ptr(o.ptr), refCount(o.refCount) {
        if (refCount) ++(*refCount);
    }

    // 拷贝赋值
    SharedPtr& operator=(const SharedPtr& o) {
        if (this != &o) {
            release();
            ptr = o.ptr;
            refCount = o.refCount;
            if (refCount) ++(*refCount);
        }
        return *this;
    }

    ~SharedPtr() { release(); }

    T& operator*() const { return *ptr; }
    T* operator->() const { return ptr; }
    explicit operator bool() const { return ptr != nullptr; }

    int use_count() const { return refCount ? refCount->load() : 0; }
};

int main() {
    SharedPtr<int> sp1(new int(42));
    cout << "sp1=" << *sp1 << " count=" << sp1.use_count() << endl;  // 42, 1

    {
        SharedPtr<int> sp2 = sp1;  // 拷贝，引用计数+1
        cout << "sp2=" << *sp2 << " count=" << sp1.use_count() << endl;  // 42, 2
    }  // sp2析构，引用计数-1

    cout << "sp1 count=" << sp1.use_count() << endl;  // 1

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **operator->的特殊递归语义**：返回指针后编译器自动`->`
2. **explicit operator bool()防止意外与整数比较**：只允许在布尔上下文中使用
3. **unique_ptr禁止拷贝，shared_ptr允许拷贝增加引用计数**
4. **实际项目中使用std::unique_ptr和std::shared_ptr**：自定义版本仅用于学习
5. **自定义智能指针需要正确处理线程安全**：shared_ptr内部使用原子操作
6. **数组版本的智能指针需要operator[]**：`unique_ptr<T[]>` 的特化
