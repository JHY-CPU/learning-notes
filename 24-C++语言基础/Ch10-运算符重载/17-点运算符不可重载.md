# 点运算符不可重载

## 一、概念说明

**点运算符`.`和指向成员的点运算符`.*`不可重载**（C++11标准 §13.5.1）。这是为了保证对象成员访问的基本语义不会被改变。如果允许重载`.`，将无法可靠地访问对象的真实成员。

不可重载的原因：
- `.` 是访问成员的基础操作，重载会破坏语言的基本语义
- 编译器需要`.`来查找其他成员（包括可能的`operator->`）
- 会导致无限递归问题

## 二、具体用法

### 2.1 使用operator->间接实现代理

```cpp
#include <iostream>
#include <string>
using namespace std;

// 不能重载. 但可以重载-> 实现代理模式
class StringProxy {
    string* str;
public:
    StringProxy(string* s) : str(s) {}

    // 通过operator->代理访问
    string* operator->() { return str; }
    const string* operator->() const { return str; }

    // operator* 也可以代理
    string& operator*() { return *str; }
    const string& operator*() const { return *str; }
};

int main() {
    string s = "Hello World";
    StringProxy proxy(&s);

    // proxy. 不能重载，但proxy->可以访问string的方法
    cout << proxy->size() << endl;          // 11
    cout << proxy->substr(0, 5) << endl;    // Hello
    cout << proxy->find("World") << endl;   // 6

    // operator*代理
    (*proxy) += "!";
    cout << *proxy << endl;  // Hello World!

    return 0;
}
```

### 2.2 reference_wrapper的代理模式

```cpp
#include <iostream>
#include <functional>
using namespace std;

struct Widget {
    int id;
    string name;
    void display() const {
        cout << "Widget(" << id << ", " << name << ")" << endl;
    }
};

int main() {
    Widget w{42, "Button"};

    // std::reference_wrapper 使用 operator T& 间接实现代理
    reference_wrapper<Widget> ref(w);

    // ref.get() 返回引用
    ref.get().display();           // Widget(42, Button)
    ref.get().name = "Label";

    // 隐式转换为引用
    Widget& wref = ref;
    wref.display();                // Widget(42, Label)

    return 0;
}
```

### 2.3 可能的C++23/26扩展

```cpp
#include <iostream>
using namespace std;

// C++23增加了operator.的实验性提案（P0416R1 / P2728R0）
// 但截至C++23尚未正式进入标准

// 如果operator.可重载，语法可能是：
// class Proxy {
//     Widget* w;
// public:
//     Widget& operator.() { return *w; }  // 假设语法
// };

// 目前的替代方案：
// 1. operator-> 和 operator*
// 2. 隐式类型转换 operator T&()
// 3. 命名访问函数（如 .get()）

int main() {
    cout << "点运算符不可重载是C++的语言规范" << endl;
    cout << "使用operator->和operator*实现代理模式" << endl;
    return 0;
}
```

## 三、注意事项与常见陷阱

1. **`.`和 `.*` 不可重载是语言硬性规定**：没有例外
2. **可通过 `operator->` 和 `operator*` 实现智能指针和代理模式**：标准替代方案
3. **C++23增加了operator.的实验性提案**（P0416/P2728），但尚未正式进入标准
4. **std::reference_wrapper使用 `operator T&` 间接实现类似代理**：隐式转换方式
5. **代理模式中不要过度使用隐式转换**：可能导致意外行为
