# 流运算符重载

## 一、概念说明

`operator<<`（输出流）和`operator>>`（输入流）必须作为**非成员函数**重载，因为左操作数是`std::ostream`/`std::istream`，不能修改标准库类。通常声明为友元以访问私有成员。

## 二、具体用法

### 2.1 输出运算符

```cpp
#include <iostream>
#include <string>

class Person {
    std::string name;
    int age;
public:
    Person(const std::string& n, int a) : name(n), age(a) {}

    // 友元输出运算符
    friend std::ostream& operator<<(std::ostream& os, const Person& p) {
        return os << "Person(" << p.name << ", " << p.age << ")";
    }
};

int main() {
    Person p("张三", 25);
    std::cout << p << std::endl;       // 直接输出
    std::cout << p << " 完成" << std::endl;  // 支持链式调用
    return 0;
}
```

**输出：**
```
Person(张三, 25)
Person(张三, 25) 完成
```

### 2.2 输入输出配对

```cpp
#include <iostream>
#include <sstream>

class Point {
public:
    double x, y;
    Point(double x = 0, double y = 0) : x(x), y(y) {}

    friend std::ostream& operator<<(std::ostream& os, const Point& p) {
        return os << "(" << p.x << ", " << p.y << ")";
    }

    friend std::istream& operator>>(std::istream& is, Point& p) {
        char ch;
        return is >> ch >> p.x >> ch >> p.y >> ch;  // 格式：(x, y)
    }
};

int main() {
    Point p;
    std::istringstream iss("(3.14, 2.71)");
    iss >> p;
    std::cout << "读取: " << p << std::endl;
    return 0;
}
```

**输出：**
```
读取: (3.14, 2.71)
```

## 三、注意事项与常见陷阱

- 流运算符**必须**是非成员函数
- 返回流的引用以支持链式调用
- `operator>>`的第二参数必须是非const引用
- 输入运算符应处理格式错误
- 可通过`friend`声明在类内定义以访问私有成员
