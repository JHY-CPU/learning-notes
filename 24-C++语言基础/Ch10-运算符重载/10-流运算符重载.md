# 流运算符重载

## 一、概念说明

`operator<<`（输出流）和`operator>>`（输入流，C++11标准 §13.5.6）必须作为**非成员函数**重载，因为左操作数是`std::ostream`/`std::istream`，不能修改标准库类。通常声明为友元以访问私有成员。

关键设计：
- 返回流的引用以支持链式调用 `cout << a << b`
- `operator>>`的第二参数必须是非const引用
- 输入运算符应处理格式错误

## 二、具体用法

### 2.1 输出运算符

```cpp
#include <iostream>
#include <string>
using namespace std;

class Person {
    string name;
    int age;
public:
    Person(const string& n, int a) : name(n), age(a) {}

    // 友元输出运算符
    friend ostream& operator<<(ostream& os, const Person& p) {
        return os << "Person(" << p.name << ", " << p.age << ")";
    }
};

int main() {
    Person p("Alice", 25);
    cout << p << endl;             // Person(Alice, 25)
    cout << p << " OK" << endl;    // Person(Alice, 25) OK（链式调用）

    // 输出到文件流也适用
    // ofstream file("out.txt");
    // file << p;

    return 0;
}
```

### 2.2 输入输出配对

```cpp
#include <iostream>
#include <sstream>
#include <string>
using namespace std;

class Point {
public:
    double x, y;
    Point(double x = 0, double y = 0) : x(x), y(y) {}

    friend ostream& operator<<(ostream& os, const Point& p) {
        return os << "(" << p.x << ", " << p.y << ")";
    }

    friend istream& operator>>(istream& is, Point& p) {
        char ch;
        return is >> ch >> p.x >> ch >> p.y >> ch;  // 格式：(x, y)
    }
};

int main() {
    // 输出
    Point p1(3.14, 2.71);
    cout << p1 << endl;  // (3.14, 2.71)

    // 输入
    Point p2;
    istringstream iss("(1.5, 2.5)");
    iss >> p2;
    cout << "读取: " << p2 << endl;  // 读取: (1.5, 2.5)

    return 0;
}
```

### 2.3 类层次中的流运算符

```cpp
#include <iostream>
using namespace std;

class Shape {
public:
    virtual ~Shape() = default;
    virtual void print(ostream& os) const = 0;

    friend ostream& operator<<(ostream& os, const Shape& s) {
        s.print(os);  // 多态调用
        return os;
    }
};

class Circle : public Shape {
    double radius;
public:
    Circle(double r) : radius(r) {}
    void print(ostream& os) const override {
        os << "Circle(r=" << radius << ")";
    }
};

class Rectangle : public Shape {
    double w, h;
public:
    Rectangle(double w, double h) : w(w), h(h) {}
    void print(ostream& os) const override {
        os << "Rectangle(" << w << "x" << h << ")";
    }
};

int main() {
    Circle c(5);
    Rectangle r(3, 4);

    cout << c << endl;  // Circle(r=5)
    cout << r << endl;  // Rectangle(3x4)

    // 多态输出
    Shape* shapes[] = {&c, &r};
    for (auto* s : shapes) cout << *s << " ";  // Circle(r=5) Rectangle(3x4)
    cout << endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **流运算符必须是非成员函数**：左操作数是流对象
2. **返回流的引用以支持链式调用**：`cout << a << b`
3. **operator>>的第二参数必须是非const引用**：需要修改输入对象
4. **输入运算符应处理格式错误**：检查流状态
5. **可通过friend声明在类内定义以访问私有成员**：常见模式
6. **不要重载 `&&` 和 `||` 为流运算符**：会丢失短路语义
