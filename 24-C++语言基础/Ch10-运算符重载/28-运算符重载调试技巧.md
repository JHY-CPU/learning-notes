# 运算符重载调试技巧

## 一、概念说明

调试运算符重载时，可以在运算符函数中添加日志来追踪调用链和参数值。利用`__PRETTY_FUNCTION__`或`__FUNCSIG__`可以自动输出函数签名。

## 二、具体用法

### 2.1 调试日志包装

```cpp
#include <iostream>
#include <string>

class DebugVec {
    int x, y;
public:
    DebugVec(int x = 0, int y = 0) : x(x), y(y) {}

    friend DebugVec operator+(const DebugVec& a, const DebugVec& b) {
        std::cerr << "[DEBUG] operator+: ("
                  << a.x << "," << a.y << ") + ("
                  << b.x << "," << b.y << ")" << std::endl;
        return DebugVec(a.x + b.x, a.y + b.y);
    }

    friend bool operator==(const DebugVec& a, const DebugVec& b) {
        std::cerr << "[DEBUG] operator== called" << std::endl;
        return a.x == b.x && a.y == b.y;
    }

    friend std::ostream& operator<<(std::ostream& os, const DebugVec& v) {
        return os << "(" << v.x << "," << v.y << ")";
    }
};

int main() {
    DebugVec a(1, 2), b(3, 4);
    DebugVec c = a + b;
    std::cout << "c = " << c << std::endl;
    std::cout << "a == b: " << (a == b) << std::endl;
    return 0;
}
```

**输出：**
```
[DEBUG] operator+: (1,2) + (3,4)
c = (4,6)
[DEBUG] operator== called
a == b: 0
```

## 三、注意事项与常见陷阱

- 使用`std::cerr`输出调试信息（不缓冲）
- 可以通过宏`#ifdef DEBUG`控制调试输出
- 注意调试日志可能影响运算符的性能语义
- 考虑使用代理类在调试和非调试模式间切换
