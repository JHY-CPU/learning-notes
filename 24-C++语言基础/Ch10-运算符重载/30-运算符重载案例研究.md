# 运算符重载案例研究

## 一、概念说明

综合运用多种运算符重载实现几个经典的数据类型：**复数类**、**矩阵类**和**有理数类**，展示运算符重载在实际中的综合应用。

## 二、具体用法

### 2.1 有理数类（Rational）

```cpp
#include <iostream>
#include <numeric>
#include <stdexcept>

class Rational {
    long long num, den;
    void normalize() {
        if (den == 0) throw std::invalid_argument("分母不能为零");
        if (den < 0) { num = -num; den = -den; }
        long long g = std::gcd(std::abs(num), den);
        num /= g; den /= g;
    }
public:
    Rational(long long n = 0, long long d = 1) : num(n), den(d) { normalize(); }

    Rational operator+(const Rational& r) const {
        return Rational(num * r.den + r.num * den, den * r.den);
    }
    Rational operator-(const Rational& r) const {
        return Rational(num * r.den - r.num * den, den * r.den);
    }
    Rational operator*(const Rational& r) const {
        return Rational(num * r.num, den * r.den);
    }
    Rational operator/(const Rational& r) const {
        return Rational(num * r.den, den * r.num);
    }
    bool operator==(const Rational& r) const {
        return num == r.num && den == r.den;
    }
    bool operator<(const Rational& r) const {
        return num * r.den < r.num * den;
    }

    friend std::ostream& operator<<(std::ostream& os, const Rational& r) {
        if (r.den == 1) return os << r.num;
        return os << r.num << "/" << r.den;
    }
};

int main() {
    Rational a(1, 3), b(1, 6);
    std::cout << "a + b = " << a + b << std::endl;
    std::cout << "a - b = " << a - b << std::endl;
    std::cout << "a * b = " << a * b << std::endl;
    std::cout << "a / b = " << a / b << std::endl;
    std::cout << "a < b: " << (a < b) << std::endl;
    return 0;
}
```

**输出：**
```
a + b = 1/2
a - b = 1/6
a * b = 1/18
a / b = 2
a < b: 0
```

### 2.2 简易矩阵类

```cpp
#include <iostream>
#include <vector>

class Matrix {
    int rows, cols;
    std::vector<double> data;
public:
    Matrix(int r, int c) : rows(r), cols(c), data(r * c, 0) {}
    double& operator()(int r, int c) { return data[r * cols + c]; }
    double operator()(int r, int c) const { return data[r * cols + c]; }

    Matrix operator+(const Matrix& m) const {
        Matrix result(rows, cols);
        for (size_t i = 0; i < data.size(); ++i)
            result.data[i] = data[i] + m.data[i];
        return result;
    }

    friend std::ostream& operator<<(std::ostream& os, const Matrix& m) {
        for (int i = 0; i < m.rows; ++i) {
            for (int j = 0; j < m.cols; ++j)
                os << m(i, j) << " ";
            os << "\n";
        }
        return os;
    }
};

int main() {
    Matrix m(2, 2);
    m(0, 0) = 1; m(0, 1) = 2;
    m(1, 0) = 3; m(1, 1) = 4;
    std::cout << m + m << std::endl;
    return 0;
}
```

**输出：**
```
2 4
6 8
```

## 三、注意事项与常见陷阱

- 有理数必须化简以保证比较正确
- 矩阵乘法的`operator*`注意维度匹配
- 工业级矩阵库（如Eigen）使用表达式模板优化
- 运算符重载应保证数学性质（交换律、结合律等）
