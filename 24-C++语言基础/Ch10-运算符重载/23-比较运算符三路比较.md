# C++20三路比较运算符

## 一、概念说明

**`operator<=>`**（Spaceship Operator）是C++20引入的三路比较运算符，定义一次即可自动推导出`==`、`!=`、`<`、`>`、`<=`、`>=`六种比较运算符。

### 1.1 返回类型

- `std::strong_ordering`：完全有序（int、string）
- `std::weak_ordering`：等价但不可区分（大小写不敏感比较）
- `std::partial_ordering`：部分有序（浮点数，NaN不可比）

## 二、具体用法

### 2.1 默认三路比较

```cpp
#include <iostream>
#include <compare>
#include <string>

struct Student {
    std::string name;
    int score;

    // =default 自动生成：按字典序比较name再比较score
    auto operator<=>(const Student&) const = default;
};

int main() {
    Student a{"Alice", 90}, b{"Bob", 85}, c{"Alice", 95};

    std::cout << "a < b: " << (a < b) << std::endl;
    std::cout << "a == b: " << (a == b) << std::endl;
    std::cout << "a < c: " << (a < c) << std::endl;    // 同名比分数
    return 0;
}
```

**输出：**
```
a < b: 1
a == b: 0
a < c: 1
```

### 2.2 自定义三路比较

```cpp
#include <iostream>
#include <compare>

class Version {
    int major_, minor_, patch_;
public:
    Version(int ma, int mi, int p) : major_(ma), minor_(mi), patch_(p) {}

    std::strong_ordering operator<=>(const Version& o) const {
        if (auto c = major_ <=> o.major_; c != 0) return c;
        if (auto c = minor_ <=> o.minor_; c != 0) return c;
        return patch_ <=> o.patch_;
    }

    bool operator==(const Version& o) const {
        return major_ == o.major_ && minor_ == o.minor_ && patch_ == o.patch_;
    }
};

int main() {
    Version v1(1, 2, 3), v2(1, 2, 4), v3(2, 0, 0);
    std::cout << "v1 < v2: " << (v1 < v2) << std::endl;
    std::cout << "v1 >= v2: " << (v1 >= v2) << std::endl;
    std::cout << "v1 <=> v3: " << ((v1 <=> v3) < 0) << std::endl;
    return 0;
}
```

**输出：**
```
v1 < v2: 1
v1 >= v2: 0
v1 <=> v3: 1
```

## 三、注意事项与常见陷阱

- `default`的`<=>`按成员声明顺序逐个比较
- 自定义`<=>`时通常也需定义`==`（`<=>`不自动生成`==`）
- C++20起`a != b`自动推导为`!(a == b)`
- `partial_ordering`不能用于关联容器（不满足严格弱序）
- `std::strong_ordering`适合绝大多数场景
