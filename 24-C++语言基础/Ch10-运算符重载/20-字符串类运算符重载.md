# 字符串类运算符重载

## 一、概念说明

自定义字符串类是运算符重载的经典案例，涉及`+`（连接）、`==`（比较）、`[]`（下标）、`<<`（输出）等运算符的综合运用。

## 二、具体用法

### 2.1 简易字符串类

```cpp
#include <iostream>
#include <cstring>
#include <algorithm>

class MyString {
    char* data;
    size_t len;
public:
    MyString(const char* s = "") : len(strlen(s)), data(new char[len + 1]) {
        strcpy(data, s);
    }
    MyString(const MyString& o) : len(o.len), data(new char[o.len + 1]) {
        strcpy(data, o.data);
    }
    ~MyString() { delete[] data; }

    // 拷贝赋值
    MyString& operator=(MyString o) {
        std::swap(data, o.data);
        std::swap(len, o.len);
        return *this;
    }

    // 连接
    MyString operator+(const MyString& rhs) const {
        MyString result;
        result.len = len + rhs.len;
        result.data = new char[result.len + 1];
        strcpy(result.data, data);
        strcat(result.data, rhs.data);
        return result;
    }

    // 下标
    char& operator[](size_t i) { return data[i]; }
    const char& operator[](size_t i) const { return data[i]; }

    // 比较
    bool operator==(const MyString& o) const { return strcmp(data, o.data) == 0; }
    bool operator!=(const MyString& o) const { return !(*this == o); }
    bool operator<(const MyString& o) const { return strcmp(data, o.data) < 0; }

    // 输出
    friend std::ostream& operator<<(std::ostream& os, const MyString& s) {
        return os << s.data;
    }

    size_t size() const { return len; }
};

int main() {
    MyString a("Hello"), b(" World");
    MyString c = a + b;
    std::cout << c << std::endl;
    std::cout << "长度: " << c.size() << std::endl;
    std::cout << "a == b: " << (a == b) << std::endl;
    std::cout << "c[0]: " << c[0] << std::endl;
    return 0;
}
```

**输出：**
```
Hello World
长度: 11
a == b: 0
c[0]: H
```

## 三、注意事项与常见陷阱

- 连接运算符应返回新对象而非修改原对象
- 注意内存管理和深拷贝
- 比较运算符保持一致的排序关系
- 实际项目中优先使用`std::string`
