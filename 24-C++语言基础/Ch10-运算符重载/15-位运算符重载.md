# 位运算符重载

## 一、概念说明

位运算符（`& | ^ ~ << >>`）重载常用于**位集合**（bitset）或**标志位枚举**的实现，使位操作更直观。

## 二、具体用法

### 2.1 位标志集合

```cpp
#include <iostream>

class Permissions {
    unsigned bits;
public:
    constexpr Permissions(unsigned b = 0) : bits(b) {}

    static constexpr unsigned READ  = 1 << 0;
    static constexpr unsigned WRITE = 1 << 1;
    static constexpr unsigned EXEC  = 1 << 2;

    Permissions operator|(const Permissions& rhs) const {
        return Permissions(bits | rhs.bits);
    }
    Permissions operator&(const Permissions& rhs) const {
        return Permissions(bits & rhs.bits);
    }
    Permissions operator^(const Permissions& rhs) const {
        return Permissions(bits ^ rhs.bits);
    }
    Permissions operator~() const {
        return Permissions(~bits);
    }
    Permissions& operator|=(const Permissions& rhs) {
        bits |= rhs.bits;
        return *this;
    }

    bool has(Permissions flag) const {
        return (bits & flag.bits) == flag.bits;
    }

    friend std::ostream& operator<<(std::ostream& os, const Permissions& p) {
        os << "[";
        if (p.has(Permissions(READ)))  os << "R";
        if (p.has(Permissions(WRITE))) os << "W";
        if (p.has(Permissions(EXEC)))  os << "X";
        return os << "]";
    }
};

int main() {
    auto perm = Permissions(Permissions::READ) | Permissions(Permissions::WRITE);
    std::cout << "权限: " << perm << std::endl;
    std::cout << "有读: " << perm.has(Permissions(Permissions::READ)) << std::endl;
    std::cout << "有执行: " << perm.has(Permissions(Permissions::EXEC)) << std::endl;
    return 0;
}
```

**输出：**
```
权限: [RW]
有读: 1
有执行: 0
```

## 三、注意事项与常见陷阱

- 位运算适合表示标志位集合
- 使用`constexpr`允许编译期计算
- 位运算符通常作为非成员函数实现对称性
- 注意移位运算符的移位量不能超过位宽
