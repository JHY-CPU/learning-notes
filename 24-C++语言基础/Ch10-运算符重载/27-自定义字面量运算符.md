# 自定义字面量运算符

## 一、概念说明

**用户自定义字面量**（User-Defined Literals，C++11标准 §13.5.8）通过`operator""`后缀为字面量添加自定义类型。格式为`字面量_后缀名`，如`42_km`、`3.14_rad`。

字面量运算符参数类型：
- 整数字面量：`unsigned long long`
- 浮点字面量：`long double`
- 字符串字面量：`const char*, size_t`
- 字符字面量：`char`

## 二、具体用法

### 2.1 基本自定义字面量

```cpp
#include <iostream>
#include <string>
using namespace std;

class Distance {
    double meters;
public:
    constexpr Distance(double m) : meters(m) {}
    constexpr double toMeters() const { return meters; }

    friend ostream& operator<<(ostream& os, const Distance& d) {
        return os << d.meters << "m";
    }
};

// 字面量运算符：数值+_后缀
constexpr Distance operator""_km(long double v) {
    return Distance(v * 1000.0);
}
constexpr Distance operator""_m(long double v) {
    return Distance(v);
}
constexpr Distance operator""_cm(long double v) {
    return Distance(v * 0.01);
}
constexpr Distance operator""_mm(long double v) {
    return Distance(v * 0.001);
}

int main() {
    auto d1 = 5.0_km;
    auto d2 = 200.0_m;
    auto d3 = 50.0_cm;
    auto d4 = 1000.0_mm;

    cout << d1 << endl;  // 5000m
    cout << d2 << endl;  // 200m
    cout << d3 << endl;  // 0.5m
    cout << d4 << endl;  // 1m

    // 编译期计算
    constexpr auto distance = 2.5_km;
    cout << distance << endl;  // 2500m

    return 0;
}
```

### 2.2 字符串字面量

```cpp
#include <iostream>
#include <string>
using namespace std;

// 字符串字面量运算符
string operator""_s(const char* str, size_t len) {
    return string(str, len);
}

// 二进制字面量（整数）
constexpr unsigned long long operator""_b(const char* str) {
    unsigned long long result = 0;
    for (; *str; ++str) {
        result = result * 2 + (*str - '0');
    }
    return result;
}

int main() {
    // 字符串拼接
    auto greeting = "Hello"_s + " World";
    cout << greeting << endl;  // Hello World

    // 二进制字面量
    auto bits = 1011_b;
    cout << "1011(b) = " << bits << endl;  // 11

    return 0;
}
```

### 2.3 标准库字面量

```cpp
#include <iostream>
#include <chrono>
#include <string>
using namespace std;
using namespace std::chrono_literals;
using namespace std::string_literals;

int main() {
    // <chrono>字面量
    auto duration = 500ms + 2s;
    cout << chrono::duration_cast<chrono::milliseconds>(duration).count() << "ms" << endl;
    // 2500ms

    // <string>字面量
    auto str = "Hello"s + " World"s;
    cout << str << endl;  // Hello World

    // <complex>字面量（C++14）
    // auto c = 1.0 + 2.0i;

    return 0;
}
```

### 2.4 编译期字面量

```cpp
#include <iostream>
#include <ratio>
using namespace std;

// 编译期安全的单位转换
struct Meters {
    double value;
    constexpr explicit Meters(double v) : value(v) {}
};

struct Kilometers {
    double value;
    constexpr explicit Kilometers(double v) : value(v) {}
};

constexpr Meters operator""_m(long double v) {
    return Meters(static_cast<double>(v));
}

constexpr Kilometers operator""_km(long double v) {
    return Kilometers(static_cast<double>(v));
}

// 单位转换
constexpr Meters toMeters(Kilometers km) {
    return Meters(km.value * 1000.0);
}

int main() {
    constexpr auto d1 = 1.5_km;
    constexpr auto d2 = toMeters(d1);
    cout << d2.value << "m" << endl;  // 1500m

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **后缀名必须以下划线`_`开头**：不带下划线的保留给标准库使用
2. **参数类型有限制**：整数→`unsigned long long`、浮点→`long double`、字符串→`const char*, size_t`
3. **字面量运算符应声明为constexpr**：允许编译期求值
4. **标准库提供`<chrono>`、`<string>`等的字面量**：`using namespace std::chrono_literals;`
5. **不能同时定义同名的整数和浮点字面量运算符**：会导致歧义
6. **数字和后缀之间不能有空格**：`5 _km` 是错误的
