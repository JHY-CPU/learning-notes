# 自定义字面量运算符

## 一、概念说明

**用户自定义字面量**（User-Defined Literals）通过`operator""`后缀为字面量添加自定义类型。格式为`字面量_后缀名`，如`42_km`、`3.14_rad`。

## 二、具体用法

### 2.1 基本自定义字面量

```cpp
#include <iostream>

class Distance {
    double meters;
public:
    constexpr Distance(double m) : meters(m) {}
    constexpr double toMeters() const { return meters; }

    friend std::ostream& operator<<(std::ostream& os, const Distance& d) {
        return os << d.meters << "m";
    }
};

// 字面量运算符：数值+_后缀
constexpr Distance operator""_km(long double v) {
    return Distance(v * 1000.0);
}
constexpr Distance operator""_m(long double v) {
    return Distance(v);
}
constexpr Distance operator""_cm(long double v) {
    return Distance(v * 0.01);
}

int main() {
    auto d1 = 5.0_km;
    auto d2 = 200.0_m;
    auto d3 = 50.0_cm;
    std::cout << "d1 = " << d1 << std::endl;
    std::cout << "d2 = " << d2 << std::endl;
    std::cout << "d3 = " << d3 << std::endl;
    return 0;
}
```

**输出：**
```
d1 = 5000m
d2 = 200m
d3 = 0.5m
```

### 2.2 字符串字面量

```cpp
#include <iostream>
#include <string>

std::string operator""_s(const char* str, size_t len) {
    return std::string(str, len);
}

int main() {
    auto greeting = "Hello"_s + " World";
    std::cout << greeting << std::endl;
    return 0;
}
```

**输出：**
```
Hello World
```

## 三、注意事项与常见陷阱

- 后缀名必须以下划线`_`开头（不带下划线的保留给标准库）
- 参数类型有限制：整数→`unsigned long long`、浮点→`long double`、字符串→`const char*, size_t`
- 字面量运算符应声明为`constexpr`
- 标准库提供`<chrono>`的字面量如`using namespace std::chrono_literals`
