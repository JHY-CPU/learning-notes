# 迭代器运算符重载

## 一、概念说明

迭代器需要重载`*`、`->`、`++`、`--`、`==`、`!=`、`+`、`-`、`[]`等运算符以模拟指针行为。随机访问迭代器需要实现所有运算符，输入迭代器只需实现部分。

## 二、具体用法

### 2.1 随机访问迭代器

```cpp
#include <iostream>
#include <iterator>

class IntArray {
    int* data;
    int size_;
public:
    class Iterator {
        int* ptr;
    public:
        using iterator_category = std::random_access_iterator_tag;
        using value_type = int;
        using difference_type = std::ptrdiff_t;
        using pointer = int*;
        using reference = int&;

        Iterator(int* p = nullptr) : ptr(p) {}

        reference operator*() const { return *ptr; }
        pointer operator->() const { return ptr; }
        reference operator[](difference_type n) const { return ptr[n]; }

        Iterator& operator++() { ++ptr; return *this; }
        Iterator operator++(int) { Iterator tmp = *this; ++ptr; return tmp; }
        Iterator& operator--() { --ptr; return *this; }
        Iterator operator--(int) { Iterator tmp = *this; --ptr; return tmp; }

        Iterator operator+(difference_type n) const { return Iterator(ptr + n); }
        Iterator operator-(difference_type n) const { return Iterator(ptr - n); }
        difference_type operator-(const Iterator& o) const { return ptr - o.ptr; }

        bool operator==(const Iterator& o) const { return ptr == o.ptr; }
        bool operator!=(const Iterator& o) const { return ptr != o.ptr; }
        bool operator<(const Iterator& o) const { return ptr < o.ptr; }
    };

    IntArray(int s) : data(new int[s]()), size_(s) {}
    ~IntArray() { delete[] data; }

    Iterator begin() { return Iterator(data); }
    Iterator end() { return Iterator(data + size_); }

    int& operator[](int i) { return data[i]; }
};

int main() {
    IntArray arr(5);
    for (int i = 0; i < 5; ++i) arr[i] = (i + 1) * 10;

    for (auto it = arr.begin(); it != arr.end(); ++it)
        std::cout << *it << " ";
    std::cout << std::endl;

    auto it = arr.begin();
    std::cout << "it[2] = " << it[2] << std::endl;
    return 0;
}
```

**输出：**
```
10 20 30 40 50
it[2] = 30
```

## 三、注意事项与常见陷阱

- 迭代器应提供正确的`iterator_category`类型别名
- `operator[]`在随机访问迭代器中是必需的
- 前置`++`比后置`++`更高效（避免拷贝）
- 使用`std::iterator_traits`获取迭代器属性
