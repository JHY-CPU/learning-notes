# 运算符重载与继承

## 一、概念说明

运算符重载可以在继承体系中使用（C++11标准 §13.5），但需要特别注意**虚运算符**和**类型兼容**问题。通常将比较等通用运算符声明为虚函数以支持多态比较。

关键问题：
- 运算符可以是virtual的，支持多态
- 比较运算符需要 `dynamic_cast` 检查实际类型
- 拷贝赋值运算符不应声明为虚函数（但clone模式可替代）

## 二、具体用法

### 2.1 继承中的虚运算符

```cpp
#include <iostream>
#include <memory>
#include <cmath>
using namespace std;

class Shape {
public:
    virtual ~Shape() = default;
    virtual double area() const = 0;
    virtual unique_ptr<Shape> clone() const = 0;

    // 虚比较运算符
    virtual bool operator==(const Shape& other) const {
        return typeid(*this) == typeid(other) && area() == other.area();
    }

    bool operator!=(const Shape& other) const {
        return !(*this == other);
    }

    virtual void print(ostream& os) const = 0;

    friend ostream& operator<<(ostream& os, const Shape& s) {
        s.print(os);
        return os;
    }
};

class Circle : public Shape {
    double radius;
public:
    Circle(double r) : radius(r) {}

    double area() const override { return M_PI * radius * radius; }

    unique_ptr<Shape> clone() const override {
        return make_unique<Circle>(*this);
    }

    bool operator==(const Shape& other) const override {
        auto* c = dynamic_cast<const Circle*>(&other);
        return c && abs(c->radius - radius) < 1e-9;
    }

    void print(ostream& os) const override {
        os << "Circle(r=" << radius << ")";
    }
};

class Rectangle : public Shape {
    double w, h;
public:
    Rectangle(double w, double h) : w(w), h(h) {}

    double area() const override { return w * h; }

    unique_ptr<Shape> clone() const override {
        return make_unique<Rectangle>(*this);
    }

    bool operator==(const Shape& other) const override {
        auto* r = dynamic_cast<const Rectangle*>(&other);
        return r && abs(r->w - w) < 1e-9 && abs(r->h - h) < 1e-9;
    }

    void print(ostream& os) const override {
        os << "Rect(" << w << "x" << h << ")";
    }
};

int main() {
    Circle c1(5), c2(5), c3(3);
    cout << c1 << " == " << c2 << ": " << (c1 == c2) << endl;  // 1
    cout << c1 << " == " << c3 << ": " << (c1 == c3) << endl;  // 0

    // 基类引用比较（多态）
    const Shape& s1 = c1;
    const Shape& s2 = c3;
    cout << s1 << " == " << s2 << ": " << (s1 == s2) << endl;  // 0

    return 0;
}
```

### 2.2 二元运算符的类型对称性

```cpp
#include <iostream>
using namespace std;

class Base {
public:
    virtual ~Base() = default;

    // 派生类之间的运算符需要特殊处理
    virtual bool equals(const Base& other) const = 0;

    friend bool operator==(const Base& a, const Base& b) {
        return a.equals(b);
    }
};

class Derived : public Base {
    int value;
public:
    Derived(int v) : value(v) {}

    bool equals(const Base& other) const override {
        if (auto* d = dynamic_cast<const Derived*>(&other)) {
            return d->value == value;
        }
        return false;
    }
};

int main() {
    Derived d1(42), d2(42), d3(99);
    cout << (d1 == d2) << endl;  // 1
    cout << (d1 == d3) << endl;  // 0

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **运算符可以声明为virtual以支持多态**：但需要在基类中定义统一接口
2. **比较运算符需要dynamic_cast检查实际类型**：不同派生类之间比较需谨慎
3. **拷贝赋值运算符不应声明为虚函数**：但clone模式可替代多态拷贝
4. **继承中的运算符签名必须一致才能正确覆盖**：参数类型和const性必须匹配
5. **C++20的<=>运算符可以自动生成==和!=**：简化比较运算符的实现
