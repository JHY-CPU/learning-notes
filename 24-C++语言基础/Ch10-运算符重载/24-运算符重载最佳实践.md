# 运算符重载最佳实践

## 一、概念说明

运算符重载应遵循**最小惊讶原则**（Principle of Least Surprise），保持与内建类型一致的语义。以下是经过实践检验的最佳实践规则。

## 二、核心规则速查表

| 运算符 | 实现方式 | 返回类型 | 说明 |
|--------|---------|---------|------|
| `+ - * /` | 非成员函数 | 新对象（值） | 对称性 |
| `+= -= *= /=` | 成员函数 | `T&` | 修改并返回*this |
| `== != < > <= >=` | 非成员或成员 | `bool` | 一致性 |
| `++ --` 前置 | 成员函数 | `T&` | 先变后用 |
| `++ --` 后置 | 成员函数 | `T`（旧值） | 哑参数int |
| `[]` | 成员函数 | `T&` / `const T&` | 非const和const版本 |
| `()` | 成员函数 | 取决于用途 | 仿函数 |
| `<< >>` | 非成员友元 | 流引用 | 链式调用 |
| `=` | 成员函数 | `T&` | 链式赋值 |
| `->` | 成员函数 | 指针 | 智能指针/迭代器 |

## 三、具体用法

### 3.1 完整的Vec3实现

```cpp
#include <iostream>
#include <cmath>
using namespace std;

class Vec3 {
public:
    double x, y, z;
    Vec3(double x = 0, double y = 0, double z = 0) : x(x), y(y), z(z) {}

    // ===== 复合赋值：成员函数，返回引用 =====
    Vec3& operator+=(const Vec3& v) { x += v.x; y += v.y; z += v.z; return *this; }
    Vec3& operator-=(const Vec3& v) { x -= v.x; y -= v.y; z -= v.z; return *this; }
    Vec3& operator*=(double s) { x *= s; y *= s; z *= s; return *this; }
    Vec3& operator/=(double s) { x /= s; y /= s; z /= s; return *this; }

    // ===== 二元算术：非成员，基于复合赋值 =====
    friend Vec3 operator+(Vec3 a, const Vec3& b) { return a += b; }
    friend Vec3 operator-(Vec3 a, const Vec3& b) { return a -= b; }
    friend Vec3 operator*(Vec3 v, double s) { return v *= s; }
    friend Vec3 operator*(double s, Vec3 v) { return v *= s; }
    friend Vec3 operator/(Vec3 v, double s) { return v /= s; }

    // ===== 一元运算符：成员函数 =====
    Vec3 operator-() const { return {-x, -y, -z}; }
    Vec3 operator+() const { return *this; }

    // ===== 比较：一致定义 == 和 != =====
    bool operator==(const Vec3& v) const {
        return x == v.x && y == v.y && z == v.z;
    }
    bool operator!=(const Vec3& v) const { return !(*this == v); }

    // ===== 流运算符：非成员友元 =====
    friend ostream& operator<<(ostream& os, const Vec3& v) {
        return os << "(" << v.x << "," << v.y << "," << v.z << ")";
    }

    // ===== 实用函数 =====
    double length() const { return sqrt(x*x + y*y + z*z); }
};

int main() {
    Vec3 a(1, 2, 3), b(4, 5, 6);

    cout << "a + b = " << a + b << endl;      // (5,7,9)
    cout << "a * 2 = " << a * 2.0 << endl;    // (2,4,6)
    cout << "2 * a = " << 2.0 * a << endl;    // (2,4,6)
    cout << "-a = " << -a << endl;             // (-1,-2,-3)
    cout << "a == b: " << (a == b) << endl;    // 0
    cout << "|a| = " << a.length() << endl;    // 3.74166

    return 0;
}
```

### 3.2 C++20三路比较运算符

```cpp
#include <iostream>
#include <compare>
using namespace std;

// C++20：使用<=>自动生成所有比较运算符
class Version {
    int major_, minor_, patch_;
public:
    Version(int m, int n, int p) : major_(m), minor_(n), patch_(p) {}

    // <=> 自动生成 ==, !=, <, >, <=, >=
    auto operator<=>(const Version&) const = default;

    friend ostream& operator<<(ostream& os, const Version& v) {
        return os << v.major_ << "." << v.minor_ << "." << v.patch_;
    }
};

int main() {
    Version v1(1, 2, 3), v2(1, 2, 4);

    cout << v1 << " < " << v2 << ": " << (v1 < v2) << endl;   // 1
    cout << v1 << " == " << v2 << ": " << (v1 == v2) << endl;  // 0
    cout << v1 << " >= " << v2 << ": " << (v1 >= v2) << endl;  // 0

    return 0;
}
```

## 四、注意事项与常见陷阱

1. **不要滥用运算符重载，语义必须直观**：`+`应该是某种"加法"，`*`应该是某种"乘法"
2. **保持运算符的数学性质**：`+`的交换性（`a+b == b+a`）、结合性
3. **基于复合赋值实现二元算术避免重复代码**：`operator+` 基于 `operator+=`
4. **始终同时定义 `==` 和 `!=`**：C++20前不自动生成
5. **考虑使用C++20的 `<=>` 减少比较运算符代码**：一行生成6个比较运算符
6. **运算符的优先级和结合性不能改变**：重载只是提供实现，不改变语法
7. **不能创建新的运算符符号**：只能重载已有的运算符
