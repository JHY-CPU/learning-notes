# 算术运算符重载

## 一、概念说明

算术运算符（`+ - * /`）重载通常返回一个**新对象**，不修改操作数。推荐以**非成员函数**形式实现对称性（`5 + obj`和`obj + 5`都能工作）。

## 二、具体用法

### 2.1 复数类算术运算符

```cpp
#include <iostream>

class Complex {
public:
    double real, imag;
    Complex(double r = 0, double i = 0) : real(r), imag(i) {}

    friend Complex operator+(const Complex& a, const Complex& b) {
        return Complex(a.real + b.real, a.imag + b.imag);
    }

    friend Complex operator-(const Complex& a, const Complex& b) {
        return Complex(a.real - b.real, a.imag - b.imag);
    }

    friend Complex operator*(const Complex& a, const Complex& b) {
        return Complex(a.real * b.real - a.imag * b.imag,
                       a.real * b.imag + a.imag * b.real);
    }

    friend std::ostream& operator<<(std::ostream& os, const Complex& c) {
        return os << c.real << " + " << c.imag << "i";
    }
};

int main() {
    Complex a(3, 4), b(1, 2);
    std::cout << "a + b = " << a + b << std::endl;
    std::cout << "a - b = " << a - b << std::endl;
    std::cout << "a * b = " << a * b << std::endl;
    return 0;
}
```

**输出：**
```
a + b = 4 + 6i
a - b = 2 + 2i
a * b = -5 + 10i
```

### 2.2 基于operator+实现operator+=

```cpp
#include <iostream>

class Vector2D {
public:
    double x, y;
    Vector2D(double x = 0, double y = 0) : x(x), y(y) {}

    // 复合赋值：修改自身，返回引用
    Vector2D& operator+=(const Vector2D& rhs) {
        x += rhs.x;
        y += rhs.y;
        return *this;
    }

    // 二元算术：基于+=实现（高效复用）
    friend Vector2D operator+(Vector2D lhs, const Vector2D& rhs) {
        lhs += rhs;  // lhs是拷贝，修改安全
        return lhs;
    }

    friend std::ostream& operator<<(std::ostream& os, const Vector2D& v) {
        return os << "(" << v.x << ", " << v.y << ")";
    }
};

int main() {
    Vector2D a(1, 2), b(3, 4);
    std::cout << "a + b = " << a + b << std::endl;
    a += b;
    std::cout << "a += b → " << a << std::endl;
    return 0;
}
```

**输出：**
```
a + b = (4, 6)
a += b → (4, 6)
```

## 三、注意事项与常见陷阱

- 算术运算符应返回**新对象**（值），不修改操作数
- 优先基于复合赋值实现二元算术运算符
- 使用`friend`实现非成员运算符以支持对称操作
- 算术运算符应该是`const`的（不修改操作数）
- 考虑边界情况（如除以零）
