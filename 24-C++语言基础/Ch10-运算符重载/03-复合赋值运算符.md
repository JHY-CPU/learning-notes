# 复合赋值运算符

## 一、概念说明

复合赋值运算符（`+= -= *= /=`等）修改左操作数并返回其引用。它们是其他运算符的基础——通常先实现`+=`，再基于它实现`+`。

## 二、具体用法

### 2.1 完整实现

```cpp
#include <iostream>

class Fraction {
    int num, den;
    void simplify() { int g = gcd(abs(num), abs(den)); num /= g; den /= g; }
public:
    Fraction(int n = 0, int d = 1) : num(n), den(d) { simplify(); }

    Fraction& operator+=(const Fraction& rhs) {
        num = num * rhs.den + rhs.num * den;
        den *= rhs.den;
        simplify();
        return *this;
    }

    Fraction& operator-=(const Fraction& rhs) {
        num = num * rhs.den - rhs.num * den;
        den *= rhs.den;
        simplify();
        return *this;
    }

    Fraction& operator*=(const Fraction& rhs) {
        num *= rhs.num;
        den *= rhs.den;
        simplify();
        return *this;
    }

    friend Fraction operator+(Fraction lhs, const Fraction& rhs) { return lhs += rhs; }
    friend Fraction operator-(Fraction lhs, const Fraction& rhs) { return lhs -= rhs; }
    friend Fraction operator*(Fraction lhs, const Fraction& rhs) { return lhs *= rhs; }

    friend std::ostream& operator<<(std::ostream& os, const Fraction& f) {
        return os << f.num << "/" << f.den;
    }
};

int main() {
    Fraction a(1, 3), b(1, 6);
    std::cout << "a + b = " << a + b << std::endl;
    std::cout << "a - b = " << a - b << std::endl;
    std::cout << "a * b = " << a * b << std::endl;
    return 0;
}
```

**输出：**
```
a + b = 1/2
a - b = 1/6
a * b = 1/18
```

## 三、注意事项与常见陷阱

- 复合赋值运算符必须返回`*this`的**引用**（非const）
- 复合赋值是**成员函数**（因为修改了左操作数）
- 基于复合赋值实现二元算术可避免代码重复
- 注意运算优先级和结合性保持不变
