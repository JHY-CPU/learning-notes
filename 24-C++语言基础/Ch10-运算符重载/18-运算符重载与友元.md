# 运算符重载与友元

## 一、概念说明

某些运算符作为**非成员函数**重载更自然（如`+`需要支持`5 + obj`），此时需要`friend`声明来访问类的私有成员。非成员运算符提供**对称性**：左右操作数都可以隐式转换。

## 二、具体用法

### 2.1 非成员运算符实现对称性

```cpp
#include <iostream>

class Meter {
    double value;
public:
    explicit Meter(double v) : value(v) {}

    // 非成员友元：支持 double + Meter 和 Meter + double
    friend Meter operator+(const Meter& lhs, const Meter& rhs) {
        return Meter(lhs.value + rhs.value);
    }

    friend Meter operator*(const Meter& m, double scalar) {
        return Meter(m.value * scalar);
    }
    friend Meter operator*(double scalar, const Meter& m) {
        return Meter(scalar * m.value);  // 对称性
    }

    friend std::ostream& operator<<(std::ostream& os, const Meter& m) {
        return os << m.value << "m";
    }
};

int main() {
    Meter a(1.5), b(2.5);
    std::cout << "a + b = " << a + b << std::endl;
    std::cout << "a * 2 = " << a * 2.0 << std::endl;
    std::cout << "2 * a = " << 2.0 * a << std::endl;
    return 0;
}
```

**输出：**
```
a + b = 4m
a * 2 = 3m
2 * a = 3m
```

## 三、注意事项与常见陷阱

- 成员运算符的左操作数不能隐式转换
- 非成员友元运算符两个操作数都可以隐式转换
- 赋值类运算符（`= +=`等）必须是成员函数
- `<<`和`>>`必须是非成员函数
- friend声明不破坏封装性，而是类接口的一部分
