# 运算符重载与友元

## 一、概念说明

某些运算符作为**非成员函数**重载更自然（C++11标准 §11.3、§13.5）。例如`+`需要支持 `5 + obj`（左操作数不是类类型），此时需要`friend`声明来访问类的私有成员。非成员运算符提供**对称性**：左右操作数都可以隐式转换。

| 实现方式 | 优点 | 缺点 |
|---------|------|------|
| 成员函数 | 可访问this | 左操作数不能隐式转换 |
| 非成员friend | 左右都可隐式转换 | 需要friend声明 |

## 二、具体用法

### 2.1 非成员运算符实现对称性

```cpp
#include <iostream>
using namespace std;

class Meter {
    double value;
public:
    explicit Meter(double v) : value(v) {}

    // 非成员友元：支持 double + Meter 和 Meter + double
    friend Meter operator+(const Meter& lhs, const Meter& rhs) {
        return Meter(lhs.value + rhs.value);
    }

    friend Meter operator-(const Meter& lhs, const Meter& rhs) {
        return Meter(lhs.value - rhs.value);
    }

    // 对称的乘法：支持 m * 2.0 和 2.0 * m
    friend Meter operator*(const Meter& m, double scalar) {
        return Meter(m.value * scalar);
    }
    friend Meter operator*(double scalar, const Meter& m) {
        return Meter(scalar * m.value);
    }

    // 流运算符必须是友元非成员
    friend ostream& operator<<(ostream& os, const Meter& m) {
        return os << m.value << "m";
    }

    // 比较运算符
    friend bool operator==(const Meter& lhs, const Meter& rhs) {
        return lhs.value == rhs.value;
    }
    friend bool operator<(const Meter& lhs, const Meter& rhs) {
        return lhs.value < rhs.value;
    }
};

int main() {
    Meter a(1.5), b(2.5);
    cout << "a + b = " << a + b << endl;    // 4m
    cout << "a * 2 = " << a * 2.0 << endl;  // 3m
    cout << "2 * a = " << 2.0 * a << endl;  // 3m
    cout << "a == b: " << (a == b) << endl;  // 0

    return 0;
}
```

### 2.2 成员 vs 非成员的选择

```cpp
#include <iostream>
using namespace std;

class Vec2 {
public:
    double x, y;
    Vec2(double x = 0, double y = 0) : x(x), y(y) {}

    // 必须是成员：修改左操作数
    Vec2& operator+=(const Vec2& v) { x += v.x; y += v.y; return *this; }
    Vec2& operator*=(double s) { x *= s; y *= s; return *this; }

    // 推荐非成员：对称性
    friend Vec2 operator+(Vec2 a, const Vec2& b) { return a += b; }
    friend Vec2 operator*(Vec2 v, double s) { return v *= s; }
    friend Vec2 operator*(double s, Vec2 v) { return v *= s; }

    // 必须非成员：左操作数是ostream
    friend ostream& operator<<(ostream& os, const Vec2& v) {
        return os << "(" << v.x << "," << v.y << ")";
    }

    // 可以是成员：只有一个参数有意义
    bool operator==(const Vec2& v) const { return x == v.x && y == v.y; }
};

int main() {
    Vec2 a(1, 2), b(3, 4);
    cout << a + b << endl;       // (4,6)
    cout << a * 2.0 << endl;     // (2,4)
    cout << 2.0 * a << endl;     // (2,4)

    return 0;
}
```

### 2.3 通过友元避免getter/setter

```cpp
#include <iostream>
using namespace std;

class Complex {
    double real, imag;
public:
    Complex(double r = 0, double i = 0) : real(r), imag(i) {}

    // 友元直接访问私有成员，无需getter
    friend Complex operator+(const Complex& a, const Complex& b) {
        return Complex(a.real + b.real, a.imag + b.imag);
    }

    friend ostream& operator<<(ostream& os, const Complex& c) {
        os << c.real;
        if (c.imag >= 0) os << "+";
        return os << c.imag << "i";
    }

    friend bool operator==(const Complex& a, const Complex& b) {
        return a.real == b.real && a.imag == b.imag;
    }
};

int main() {
    Complex a(1, 2), b(3, -4);
    cout << a << " + " << b << " = " << (a + b) << endl;
    // 1+2i + 3-4i = 4-2i

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **成员运算符的左操作数不能隐式转换**：`5 + obj` 不行，但 `obj + 5` 可以
2. **非成员友元运算符两个操作数都可以隐式转换**：提供对称性
3. **赋值类运算符（`= +=`等）必须是成员函数**：语言要求
4. **`<<`和`>>`必须是非成员函数**：左操作数是流对象
5. **friend声明不破坏封装性**：是类接口的一部分，类自己选择暴露给谁
