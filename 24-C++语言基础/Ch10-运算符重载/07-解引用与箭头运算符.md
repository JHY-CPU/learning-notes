# 解引用与箭头运算符

## 一、概念说明

`operator*`和`operator->`（C++11标准 §13.5.2）用于使自定义类表现为**智能指针**或**迭代器**。`operator->`的特殊之处在于它被**递归调用**直到返回真正的指针。

| 运算符 | 签名 | 返回类型 | 用途 |
|--------|------|---------|------|
| `operator*` | `T& operator*()` | 引用 | 解引用访问对象 |
| `operator->` | `T* operator->()` | 指针 | 成员访问 |

## 二、具体用法

### 2.1 简单智能指针

```cpp
#include <iostream>
#include <utility>
using namespace std;

template<typename T>
class UniquePtr {
    T* ptr;
public:
    explicit UniquePtr(T* p = nullptr) : ptr(p) {}
    ~UniquePtr() { delete ptr; }

    // 禁止拷贝
    UniquePtr(const UniquePtr&) = delete;
    UniquePtr& operator=(const UniquePtr&) = delete;

    // 移动语义
    UniquePtr(UniquePtr&& other) noexcept : ptr(other.ptr) {
        other.ptr = nullptr;
    }
    UniquePtr& operator=(UniquePtr&& other) noexcept {
        if (this != &other) {
            delete ptr;
            ptr = other.ptr;
            other.ptr = nullptr;
        }
        return *this;
    }

    // 解引用：返回引用
    T& operator*() { return *ptr; }
    const T& operator*() const { return *ptr; }

    // 箭头：返回指针
    T* operator->() { return ptr; }
    const T* operator->() const { return ptr; }

    // 布尔转换
    explicit operator bool() const { return ptr != nullptr; }

    T* get() const { return ptr; }
    T* release() { T* p = ptr; ptr = nullptr; return p; }
};

struct Widget {
    int id;
    string name;
    void display() const {
        cout << "Widget(" << id << ", " << name << ")" << endl;
    }
};

int main() {
    UniquePtr<Widget> w(new Widget{42, "Button"});

    // operator*() 使用
    (*w).display();  // Widget(42, Button)
    (*w).name = "Label";

    // operator->() 使用（更常用）
    w->display();    // Widget(42, Label)
    cout << "id=" << w->id << endl;  // id=42

    // 移动语义
    UniquePtr<Widget> w2 = move(w);
    cout << "w为空: " << !w << endl;   // 1
    w2->display();                     // Widget(42, Label)

    return 0;
}
```

### 2.2 迭代器中的应用

```cpp
#include <iostream>
using namespace std;

class SimpleIter {
    int* ptr;
public:
    SimpleIter(int* p) : ptr(p) {}

    int& operator*() const { return *ptr; }
    int* operator->() const { return ptr; }

    SimpleIter& operator++() { ++ptr; return *this; }
    SimpleIter operator++(int) {
        SimpleIter old = *this;
        ++ptr;
        return old;
    }

    bool operator!=(const SimpleIter& other) const {
        return ptr != other.ptr;
    }
};

struct Node {
    int value;
    Node* next;
};

int main() {
    int arr[] = {10, 20, 30, 40, 50};

    // 指针风格的迭代器
    for (SimpleIter it = arr; it != SimpleIter(arr + 5); ++it) {
        cout << *it << " ";  // 10 20 30 40 50
    }
    cout << endl;

    return 0;
}
```

### 2.3 operator->的递归语义

```cpp
#include <iostream>
using namespace std;

struct Point {
    int x, y;
    void show() const { cout << "(" << x << "," << y << ")" << endl; }
};

// 代理类：包装智能指针
class PtrProxy {
    Point* ptr;
public:
    PtrProxy(Point* p) : ptr(p) {}

    // 返回另一个有operator->的对象
    Point* operator->() { return ptr; }
    // 编译器递归调用：proxy->show()
    // → proxy.operator->() 返回 Point*
    // → Point*->show()
};

int main() {
    Point p{3, 4};
    PtrProxy proxy(&p);
    proxy->show();  // 编译器自动递归调用operator->

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **operator->返回指针后编译器自动继续`->`调用**：递归语义，直到得到真正的指针
2. **智能指针应禁止或正确实现拷贝/移动操作**：unique_ptr禁止拷贝，shared_ptr允许
3. **std::unique_ptr和std::shared_ptr已实现了这些运算符**：实际项目应使用标准库
4. **自定义迭代器需要同时实现`*`和`->`**：STL要求
5. **operator->只能返回指针或有operator->的对象**：不能返回引用
6. **explicit operator bool()防止意外整数转换**：只允许在布尔上下文中使用
