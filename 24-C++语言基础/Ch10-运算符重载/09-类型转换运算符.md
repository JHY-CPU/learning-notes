# 类型转换运算符

## 一、概念说明

**类型转换运算符**（Conversion Operator，C++11标准 §12.3.2）`operator T()`定义了从类类型到其他类型的转换。C++11起可用`explicit`关键字防止意外的隐式转换。

| 特性 | 隐式转换 | 显式转换（explicit） |
|------|---------|---------------------|
| 调用方式 | 自动 | static_cast或直接初始化 |
| 安全性 | 可能意外 | 更安全 |
| 推荐度 | 不推荐 | 推荐 |

## 二、具体用法

### 2.1 隐式与显式转换

```cpp
#include <iostream>
#include <string>
using namespace std;

class Celsius {
    double temp;
public:
    explicit Celsius(double t) : temp(t) {}

    // 显式转换运算符（推荐）
    explicit operator double() const { return temp; }

    // 显式转换到string
    explicit operator string() const {
        return to_string(temp) + "°C";
    }
};

void printTemp(double t) { cout << t << " 度" << endl; }

int main() {
    Celsius c(36.6);

    // 显式转换
    double d = static_cast<double>(c);
    cout << d << endl;  // 36.6

    string s = static_cast<string>(c);
    cout << s << endl;  // 36.600000°C

    // printTemp(c);  // 编译错误！explicit阻止隐式转换
    printTemp(static_cast<double>(c));  // OK：显式转换

    return 0;
}
```

### 2.2 bool转换（最佳实践）

```cpp
#include <iostream>
#include <fstream>
using namespace std;

class File {
    bool isOpen;
public:
    File(const char* filename) : isOpen(true) {
        // 模拟打开文件
        cout << "打开文件: " << filename << endl;
    }
    ~File() { if (isOpen) cout << "关闭文件" << endl; }

    // explicit operator bool：用于条件判断
    explicit operator bool() const { return isOpen; }

    void close() { isOpen = false; }
};

int main() {
    File f("test.txt");

    if (f) {                      // OK：条件上下文允许隐式转换
        cout << "文件已打开" << endl;
    }

    if (!f) {                     // OK：!运算符允许
        cout << "文件未打开" << endl;
    }

    f.close();
    if (!f) {
        cout << "文件已关闭" << endl;
    }

    // bool b = f;  // 编译错误！explicit阻止赋值
    // int i = f;   // 编译错误！explicit阻止

    return 0;
}
```

### 2.3 多个转换运算符的歧义

```cpp
#include <iostream>
using namespace std;

// 危险：定义多个转换运算符可能导致歧义
class Ambiguous {
    int value;
public:
    Ambiguous(int v) : value(v) {}

    // 如果两个都不是explicit
    // operator int() const { return value; }
    // operator double() const { return value * 1.0; }
    // Ambiguous a(42);
    // double d = a;  // 歧义：int还是double？

    // 解决方案：都用explicit
    explicit operator int() const { return value; }
    explicit operator double() const { return value * 1.0; }
};

int main() {
    Ambiguous a(42);
    int i = static_cast<int>(a);       // OK
    double d = static_cast<double>(a); // OK
    cout << i << " " << d << endl;

    return 0;
}
```

### 2.4 实际应用：安全包装类

```cpp
#include <iostream>
#include <optional>
#include <string>
using namespace std;

class SafeString {
    optional<string> data;
public:
    SafeString(const string& s) : data(s) {}
    SafeString() : data(nullopt) {}

    // explicit bool：检查是否有值
    explicit operator bool() const { return data.has_value(); }

    // explicit string：获取值
    explicit operator string() const {
        return data.value_or("<empty>");
    }

    // 获取值的另一种方式
    const string& value() const {
        return data.value();
    }

    const string& value_or(const string& fallback) const {
        return data.value_or(fallback);
    }
};

int main() {
    SafeString s1("hello");
    SafeString s2;

    if (s1) {
        cout << static_cast<string>(s1) << endl;  // hello
    }

    if (!s2) {
        cout << "s2为空" << endl;
    }

    cout << s2.value_or("默认值") << endl;  // 默认值

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **建议转换运算符加explicit防止意外隐式转换**：C++11起最佳实践
2. **operator bool()应加explicit防止与整数混用**：避免 `int i = obj;` 这样的错误
3. **避免定义多个转换运算符**：可能导致歧义，编译器无法选择
4. **标准库std::optional、std::ifstream都使用explicit operator bool**：业界标准
5. **转换运算符可以是虚函数**：支持多态转换
6. **转换运算符不能有参数**：`operator T()` 只能是无参的
