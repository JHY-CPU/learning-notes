# 运算符重载与模板

## 一、概念说明

模板与运算符重载结合可以编写**通用代码**。使用SFINAE或C++20 concepts可以在编译期检测类型是否支持特定运算符，提供更好的错误信息和重载决议。

## 二、具体用法

### 2.1 SFINAE检测运算符

```cpp
#include <iostream>
#include <type_traits>

// 检测是否有operator+
template<typename T, typename = void>
struct has_plus : std::false_type {};

template<typename T>
struct has_plus<T, std::void_t<decltype(std::declval<T>() + std::declval<T>())>>
    : std::true_type {};

// 检测是否有operator<<
template<typename T, typename = void>
struct is_printable : std::false_type {};

template<typename T>
struct is_printable<T, std::void_t<decltype(std::cout << std::declval<T>())>>
    : std::true_type {};

int main() {
    std::cout << "int有operator+: " << has_plus<int>::value << std::endl;
    std::cout << "int可打印: " << is_printable<int>::value << std::endl;
    return 0;
}
```

**输出：**
```
int有operator+: 1
int可打印: 1
```

### 2.2 C++20 concepts约束

```cpp
#include <iostream>
#include <concepts>

template<typename T>
concept Addable = requires(T a, T b) {
    { a + b } -> std::convertible_to<T>;
};

template<Addable T>
T sum(T a, T b) {
    return a + b;
}

int main() {
    std::cout << "sum(1,2)=" << sum(1, 2) << std::endl;
    std::cout << "sum(1.5,2.5)=" << sum(1.5, 2.5) << std::endl;
    // sum("a","b") → 编译错误，const char*没有合适的operator+
    return 0;
}
```

**输出：**
```
sum(1,2)=3
sum(1.5,2.5)=4
```

## 三、注意事项与常见陷阱

- SFINAE是C++11/14/17检测运算符的标准方法
- C++20 concepts提供更清晰的约束语法
- 模板运算符重载注意ADL（参数依赖查找）规则
- 友元函数模板在类内定义时需要注意可见性
