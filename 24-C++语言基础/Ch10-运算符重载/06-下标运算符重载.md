# 下标运算符重载

## 一、概念说明

`operator[]`（C++11标准 §13.5.5）提供数组风格的元素访问。应提供**const版本**（只读）和**非const版本**（可读写），const版本返回const引用。

设计原则：
- 非const版本返回 `T&`：支持 `arr[i] = val`
- const版本返回 `const T&`：防止修改
- 两个版本构成重载，根据对象的const性自动选择

## 二、具体用法

### 2.1 const与非const版本

```cpp
#include <iostream>
#include <stdexcept>
using namespace std;

class IntArray {
    int* data;
    int size_;
public:
    IntArray(int s) : data(new int[s]()), size_(s) {}
    ~IntArray() { delete[] data; }

    // 非const版本：可读可写
    int& operator[](int idx) {
        if (idx < 0 || idx >= size_) throw out_of_range("越界");
        return data[idx];
    }

    // const版本：只读
    const int& operator[](int idx) const {
        if (idx < 0 || idx >= size_) throw out_of_range("越界");
        return data[idx];
    }

    int size() const { return size_; }
};

void printArray(const IntArray& arr) {
    // 调用const版本的operator[]
    for (int i = 0; i < arr.size(); ++i)
        cout << arr[i] << " ";
    cout << endl;
}

int main() {
    IntArray arr(5);
    for (int i = 0; i < arr.size(); ++i)
        arr[i] = (i + 1) * 10;  // 调用非const版本

    printArray(arr);  // 调用const版本：10 20 30 40 50

    return 0;
}
```

### 2.2 带边界检查的at()方法

```cpp
#include <iostream>
#include <stdexcept>
using namespace std;

class SafeArray {
    int* data;
    size_t size_;
public:
    SafeArray(size_t s) : data(new int[s]()), size_(s) {}
    ~SafeArray() { delete[] data; }

    // operator[]：不做检查（性能优先）
    int& operator[](size_t idx) { return data[idx]; }
    const int& operator[](size_t idx) const { return data[idx]; }

    // at()：做边界检查
    int& at(size_t idx) {
        if (idx >= size_) throw out_of_range("SafeArray::at 越界");
        return data[idx];
    }
    const int& at(size_t idx) const {
        if (idx >= size_) throw out_of_range("SafeArray::at 越界");
        return data[idx];
    }

    size_t size() const { return size_; }
};

int main() {
    SafeArray arr(5);
    arr[2] = 42;         // 无检查，快速
    cout << arr[2] << endl;  // 42

    try {
        arr.at(10) = 99;  // 抛出异常
    } catch (const out_of_range& e) {
        cout << "异常: " << e.what() << endl;
    }

    return 0;
}
```

### 2.3 多维下标（C++23）

```cpp
#include <iostream>
using namespace std;

// C++23支持多维下标：operator[](int, int)
// 此前需要链式调用或代理类

class Matrix {
    int data[3][4];
public:
    // 传统方式：返回引用的引用
    int* operator[](int row) { return data[row]; }
    const int* operator[](int row) const { return data[row]; }

    // C++23多维下标（概念上）
    // int& operator[](int row, int col) { return data[row][col]; }
};

int main() {
    Matrix m;
    m[1][2] = 42;  // m[1]返回int*，然后[2]访问元素
    cout << m[1][2] << endl;  // 42

    return 0;
}
```

### 2.4 字符串索引的高级用法

```cpp
#include <iostream>
#include <string>
#include <map>
using namespace std;

// 字典风格的下标
class Config {
    map<string, string> data;
public:
    // 非const：不存在则创建
    string& operator[](const string& key) {
        return data[key];
    }

    // const：不存在则抛异常
    const string& operator[](const string& key) const {
        auto it = data.find(key);
        if (it == data.end())
            throw out_of_range("键不存在: " + key);
        return it->second;
    }
};

int main() {
    Config cfg;
    cfg["host"] = "localhost";  // 非const版本
    cfg["port"] = "8080";

    const Config& ccfg = cfg;
    cout << ccfg["host"] << ":" << ccfg["port"] << endl;  // localhost:8080

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **非const版本返回引用以支持 `arr[i] = val`**：赋值左侧需要左值
2. **const版本返回const引用防止修改**：保护数据完整性
3. **两个版本的函数签名不同**：基于对象的const性构成重载
4. **考虑使用at()方法提供带边界检查的访问**：如std::vector的设计
5. **C++23支持多维下标：`arr[i, j]`**：通过 `operator[](int, int)` 实现
6. **operator[]只能是成员函数**：不能作为非成员函数重载
