# 下标运算符重载

## 一、概念说明

`operator[]`提供数组风格的元素访问。应提供**const版本**（只读）和**非const版本**（可读写），const版本返回const引用。

## 二、具体用法

### 2.1 const与非const版本

```cpp
#include <iostream>
#include <stdexcept>

class IntArray {
    int* data;
    int size_;
public:
    IntArray(int s) : data(new int[s]()), size_(s) {}
    ~IntArray() { delete[] data; }

    // 非const版本：可读可写
    int& operator[](int idx) {
        if (idx < 0 || idx >= size_) throw std::out_of_range("越界");
        return data[idx];
    }

    // const版本：只读
    const int& operator[](int idx) const {
        if (idx < 0 || idx >= size_) throw std::out_of_range("越界");
        return data[idx];
    }

    int size() const { return size_; }
};

void printArray(const IntArray& arr) {
    // 调用const版本的operator[]
    for (int i = 0; i < arr.size(); ++i)
        std::cout << arr[i] << " ";
    std::cout << std::endl;
}

int main() {
    IntArray arr(5);
    for (int i = 0; i < arr.size(); ++i)
        arr[i] = (i + 1) * 10;  // 调用非const版本

    printArray(arr);  // 调用const版本
    return 0;
}
```

**输出：**
```
10 20 30 40 50
```

## 三、注意事项与常见陷阱

- 非const版本返回**引用**以支持`arr[i] = val`
- const版本返回**const引用**防止修改
- 两个版本的函数签名不同，构成重载
- 考虑使用`at()`方法提供带边界检查的访问
- C++23支持多维下标：`arr[i, j]`通过`operator[](int, int)`
