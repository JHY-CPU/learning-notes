# 函数调用运算符

## 一、概念说明

**`operator()`**（C++11标准 §13.5.4）使得对象可以像函数一样被调用，这样的对象称为**仿函数**（Functor）。Lambda表达式的本质就是编译器生成的匿名仿函数类。

仿函数的优势：
- 可以有状态（成员变量），比函数指针更灵活
- 编译器可以内联调用，性能优于std::function
- 可以模板化，支持泛型

## 二、具体用法

### 2.1 基本仿函数

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

// 有状态的比较器
class CompareByField {
    bool ascending;
public:
    CompareByField(bool asc = true) : ascending(asc) {}

    bool operator()(int a, int b) const {
        return ascending ? (a < b) : (a > b);
    }
};

// 有状态的累加器
class Accumulator {
    int sum = 0;
    int count = 0;
public:
    int operator()(int val) {
        sum += val;
        ++count;
        return sum;
    }

    double average() const {
        return count > 0 ? static_cast<double>(sum) / count : 0.0;
    }

    int getSum() const { return sum; }
};

int main() {
    // 排序
    vector<int> v = {5, 2, 8, 1, 9};
    sort(v.begin(), v.end(), CompareByField(false));  // 降序
    for (int x : v) cout << x << " ";  // 9 8 5 2 1
    cout << endl;

    // 累加器
    Accumulator acc;
    for (int x : {10, 20, 30, 40}) {
        cout << "累加: " << acc(x) << endl;
    }
    cout << "平均: " << acc.average() << endl;  // 25

    return 0;
}
```

### 2.2 Lambda的本质

```cpp
#include <iostream>
using namespace std;

int main() {
    // Lambda表达式
    auto square = [](int x) { return x * x; };

    // 编译器生成的等价类（概念上）：
    // class __LambdaSquare {
    // public:
    //     int operator()(int x) const { return x * x; }
    // };

    cout << "square(5) = " << square(5) << endl;  // 25

    // 带捕获的lambda
    int offset = 10;
    auto addOffset = [offset](int x) { return x + offset; };

    // 等价类：
    // class __LambdaAddOffset {
    //     int offset;  // 捕获的变量作为成员
    // public:
    //     __LambdaAddOffset(int o) : offset(o) {}
    //     int operator()(int x) const { return x + offset; }
    // };

    cout << "addOffset(5) = " << addOffset(5) << endl;  // 15

    return 0;
}
```

### 2.3 用于STL算法

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <numeric>
using namespace std;

// 自定义仿函数用于transform
struct Negate {
    int operator()(int x) const { return -x; }
};

// 带阈值的过滤器
class Threshold {
    int limit;
public:
    Threshold(int l) : limit(l) {}
    bool operator()(int x) const { return x > limit; }
};

int main() {
    vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8};

    // 使用仿函数的transform
    vector<int> negated(v.size());
    transform(v.begin(), v.end(), negated.begin(), Negate());
    // negated: -1, -2, -3, -4, -5, -6, -7, -8

    // 使用仿函数的count_if
    int above5 = count_if(v.begin(), v.end(), Threshold(5));
    cout << ">5的个数: " << above5 << endl;  // 3

    // 使用lambda替代仿函数（更简洁）
    int above3 = count_if(v.begin(), v.end(), [](int x) { return x > 3; });
    cout << ">3的个数: " << above3 << endl;  // 5

    return 0;
}
```

### 2.4 std::function包装

```cpp
#include <iostream>
#include <functional>
#include <vector>
using namespace std;

int main() {
    // std::function可以包装任意可调用对象
    vector<function<int(int, int)>> operations;

    // 仿函数
    struct Add { int operator()(int a, int b) const { return a + b; } };
    operations.push_back(Add{});

    // Lambda
    operations.push_back([](int a, int b) { return a * b; });

    // 函数
    int subtract(int a, int b) { return a - b; }
    operations.push_back(subtract);

    for (auto& op : operations) {
        cout << op(10, 3) << " ";  // 13 7 7
    }
    cout << endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **仿函数可以有状态**：lambda也可通过捕获实现
2. **std::function可以包装任意仿函数/lambda**：但有类型擦除开销
3. **operator()可以重载**：不同参数列表
4. **仿函数通常比函数指针更高效**：编译器可以内联
5. **constexpr operator()允许编译期调用**：C++17起
6. **无状态仿函数可以与函数指针互转**：但实际很少这样做
