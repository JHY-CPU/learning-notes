# 函数调用运算符

## 一、概念说明

**`operator()`**使得对象可以像函数一样被调用，这样的对象称为**仿函数**（Functor）。Lambda表达式的本质就是编译器生成的匿名仿函数类。

## 二、具体用法

### 2.1 基本仿函数

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

// 仿函数：状态化的比较器
class CompareBy {
    std::string field;
public:
    CompareBy(const std::string& f) : field(f) {}

    bool operator()(int a, int b) const {
        if (field == "abs") return abs(a) < abs(b);
        return a < b;
    }
};

// 有状态的累加器
class Accumulator {
    int sum = 0;
public:
    int operator()(int val) { sum += val; return sum; }
    int getSum() const { return sum; }
};

int main() {
    std::vector<int> v = {3, -1, 4, -1, 5, -9};
    std::sort(v.begin(), v.end(), CompareBy("abs"));
    for (int x : v) std::cout << x << " ";
    std::cout << std::endl;

    Accumulator acc;
    for (int x : {1, 2, 3, 4, 5}) acc(x);
    std::cout << "总和: " << acc.getSum() << std::endl;
    return 0;
}
```

**输出：**
```
-1 -1 3 4 5 -9
总和: 15
```

### 2.2 Lambda的本质

```cpp
#include <iostream>

// Lambda等价于：
auto lambda = [](int x) { return x * x; };

// 编译器生成：
class LambdaGenerated {
public:
    auto operator()(int x) const { return x * x; }
};

int main() {
    std::cout << "lambda(5) = " << lambda(5) << std::endl;
    // 等价于 LambdaGenerated{}(5)
    return 0;
}
```

**输出：**
```
lambda(5) = 25
```

## 三、注意事项与常见陷阱

- 仿函数可以有状态（成员变量），lambda也可通过捕获实现
- `std::function`可以包装任意仿函数/lambda
- `operator()`可以重载（不同参数列表）
- 仿函数通常比函数指针更高效（可内联）
- `constexpr operator()`允许编译期调用
