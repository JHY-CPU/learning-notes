# 比较运算符重载

## 一、概念说明

比较运算符（`== != < > <= >=`）返回`bool`值。C++20引入了**三路比较运算符**（`<=>`，spaceship operator），可以自动推导出所有六种比较运算符。

## 二、具体用法

### 2.1 传统方式

```cpp
#include <iostream>

class Version {
    int major_, minor_, patch_;
public:
    Version(int ma, int mi, int p) : major_(ma), minor_(mi), patch_(p) {}

    bool operator==(const Version& o) const {
        return major_ == o.major_ && minor_ == o.minor_ && patch_ == o.patch_;
    }
    bool operator!=(const Version& o) const { return !(*this == o); }
    bool operator<(const Version& o) const {
        if (major_ != o.major_) return major_ < o.major_;
        if (minor_ != o.minor_) return minor_ < o.minor_;
        return patch_ < o.patch_;
    }
    bool operator>(const Version& o) const { return o < *this; }
    bool operator<=(const Version& o) const { return !(o < *this); }
    bool operator>=(const Version& o) const { return !(*this < o); }
};

int main() {
    Version v1(1, 2, 3), v2(1, 2, 4);
    std::cout << "v1 == v2: " << (v1 == v2) << std::endl;
    std::cout << "v1 < v2: " << (v1 < v2) << std::endl;
    return 0;
}
```

**输出：**
```
v1 == v2: 0
v1 < v2: 1
```

### 2.2 C++20三路比较

```cpp
#include <iostream>
#include <compare>

class Point {
public:
    double x, y;
    Point(double x = 0, double y = 0) : x(x), y(y) {}

    // 一个运算符自动生成所有比较
    auto operator<=>(const Point&) const = default;
};

int main() {
    Point a(1, 2), b(1, 3);
    std::cout << "a == b: " << (a == b) << std::endl;
    std::cout << "a != b: " << (a != b) << std::endl;
    std::cout << "a < b: " << (a < b) << std::endl;
    return 0;
}
```

**输出：**
```
a == b: 0
a != b: 1
a < b: 1
```

## 三、注意事项与常见陷阱

- 比较运算符应该是`const`成员
- `==`和`!=`应该保持一致
- `<`是关联容器（`std::set`、`std::map`）排序的基础
- C++20的`<=>`可大幅减少样板代码
- `default`的`<=>`使用**字典序**比较所有成员
