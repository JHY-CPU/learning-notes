# 表达式模板

## 一、概念说明

**表达式模板**（Expression Templates）通过延迟求值避免创建不必要的临时对象，常用于线性代数库（如Eigen）。运算符不立即计算结果，而是返回一个"表达式对象"，在最终赋值时一次性求值。

## 二、具体用法

### 2.1 延迟求值向量

```cpp
#include <iostream>
#include <vector>

class Vec {
    std::vector<double> data;
public:
    Vec(size_t n, double val = 0) : data(n, val) {}
    Vec(const std::vector<double>& d) : data(d) {}

    size_t size() const { return data.size(); }
    double& operator[](size_t i) { return data[i]; }
    double operator[](size_t i) const { return data[i]; }

    // 返回表达式模板而非立即计算
    template<typename E>
    Vec& operator=(const E& expr) {
        for (size_t i = 0; i < size(); ++i)
            data[i] = expr[i];  // 延迟到此时才计算
        return *this;
    }

    friend std::ostream& operator<<(std::ostream& os, const Vec& v) {
        os << "[";
        for (size_t i = 0; i < v.size(); ++i) {
            if (i > 0) os << ", ";
            os << v.data[i];
        }
        return os << "]";
    }
};

// 表达式模板：VecAdd只保存引用，不计算
template<typename L, typename R>
struct VecAdd {
    const L& lhs;
    const R& rhs;
    VecAdd(const L& l, const R& r) : lhs(l), rhs(r) {}
    double operator[](size_t i) const { return lhs[i] + rhs[i]; }
};

template<typename L, typename R>
VecAdd<L, R> operator+(const L& l, const R& r) {
    return VecAdd<L, R>(l, r);
}

int main() {
    Vec a({1, 2, 3}), b({4, 5, 6}), c({7, 8, 9});
    Vec result(3);
    result = a + b + c;  // 一次循环完成，无临时对象
    std::cout << result << std::endl;
    return 0;
}
```

**输出：**
```
[12, 15, 18]
```

## 三、注意事项与常见陷阱

- 表达式模板增加了代码复杂度
- 仅在性能关键场景使用（如大型矩阵运算）
- 注意悬空引用问题（表达式模板保存引用而非拷贝）
- 现代编译器的优化可能已经接近表达式模板的性能
