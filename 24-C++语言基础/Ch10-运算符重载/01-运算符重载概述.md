# 运算符重载概述

## 一、概念说明

**运算符重载**（Operator Overloading）允许为自定义类型定义运算符的行为，使代码更直观、接近数学表达。C++中大多数运算符都可以重载，但需要遵循基本原则。

### 1.1 可重载运算符

| 类别 | 运算符 |
|------|--------|
| 算术 | `+ - * / %` |
| 位运算 | `& | ^ ~ << >>` |
| 比较 | `== != < > <= >=` |
| 逻辑 | `&& || !` |
| 赋值 | `= += -= *= /= ...` |
| 自增减 | `++ --` |
| 其他 | `[] () -> , new delete` |

### 1.2 不可重载运算符

`.`（点）、`.*`（成员指针）、`::`（作用域）、`?:`（三元）、`sizeof`、`typeid`、`alignof`

### 1.3 基本原则

- 至少一个操作数必须是**用户自定义类型**
- 不能创建新运算符符号
- 不能改变运算符的**优先级**和**结合性**
- 不能改变运算符的**操作数个数**

## 二、具体用法

### 2.1 成员函数 vs 非成员函数

```cpp
#include <iostream>

class Point {
public:
    double x, y;
    Point(double x = 0, double y = 0) : x(x), y(y) {}

    // 成员函数重载：左操作数是this
    Point operator+(const Point& rhs) const {
        return Point(x + rhs.x, y + rhs.y);
    }

    // 非成员函数重载：对称性更好
    friend Point operator-(const Point& lhs, const Point& rhs);
};

Point operator-(const Point& lhs, const Point& rhs) {
    return Point(lhs.x - rhs.x, lhs.y - rhs.y);
}

int main() {
    Point a(1, 2), b(3, 4);
    Point c = a + b;   // a.operator+(b)
    Point d = a - b;   // operator-(a, b)
    std::cout << "c=(" << c.x << "," << c.y << ")" << std::endl;
    std::cout << "d=(" << d.x << "," << d.y << ")" << std::endl;
    return 0;
}
```

**输出：**
```
c=(4,6)
d=(-2,-2)
```

## 三、注意事项与常见陷阱

- 不要过度重载，保持语义直观（`+`应该有"加法"的含义）
- 赋值类运算符通常应返回`*this`的引用
- 比较运算符通常应返回`bool`
- 成员函数重载时左操作数必须是`this`类型
- 流运算符`<< >>`必须作为非成员（友元）函数重载
