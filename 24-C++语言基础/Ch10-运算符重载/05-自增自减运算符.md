# 自增自减运算符

## 一、概念说明

自增运算符有**前置**（`++i`）和**后置**（`i++`）两种形式，它们的签名不同：前置返回引用，后置返回旧值的拷贝。通过**哑参数**`int`区分两种重载。

## 二、具体用法

### 2.1 前置与后置实现

```cpp
#include <iostream>

class Counter {
    int value;
public:
    Counter(int v = 0) : value(v) {}

    // 前置++：先增后用，返回引用
    Counter& operator++() {
        ++value;
        return *this;
    }

    // 后置++：先用后增，返回旧值（哑参数int区分）
    Counter operator++(int) {
        Counter old = *this;
        ++(*this);  // 复用前置++
        return old;
    }

    // 前置--
    Counter& operator--() {
        --value;
        return *this;
    }

    // 后置--
    Counter operator--(int) {
        Counter old = *this;
        --(*this);
        return old;
    }

    int getValue() const { return value; }
    friend std::ostream& operator<<(std::ostream& os, const Counter& c) {
        return os << c.value;
    }
};

int main() {
    Counter c(5);
    std::cout << "初始: " << c << std::endl;
    std::cout << "c++: " << c++ << std::endl;   // 输出5，然后c变成6
    std::cout << "之后: " << c << std::endl;
    std::cout << "++c: " << ++c << std::endl;   // c先变成7，输出7
    std::cout << "c--: " << c-- << std::endl;   // 输出7，然后c变成6
    std::cout << "之后: " << c << std::endl;
    return 0;
}
```

**输出：**
```
初始: 5
c++: 5
之后: 6
++c: 7
c--: 7
之后: 6
```

## 三、注意事项与常见陷阱

- 前置返回**引用**，后置返回**值**
- 后置通常效率更低（需要保存旧值的拷贝）
- 建议后置基于前置实现以复用逻辑
- 迭代器类必须实现`++`和`--`运算符
- 优先使用前置形式`++i`除非需要后置语义
