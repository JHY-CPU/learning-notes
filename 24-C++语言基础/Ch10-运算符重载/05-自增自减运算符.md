# 自增自减运算符

## 一、概念说明

自增运算符有**前置**（`++i`）和**后置**（`i++`）两种形式（C++11标准 §13.5.7）。它们的签名不同：前置返回引用，后置返回旧值的拷贝。通过**哑参数**（dummy parameter）`int`区分两种重载。

| 形式 | 签名 | 返回类型 | 语义 |
|------|------|---------|------|
| 前置++ | `T& operator++()` | 引用 | 先增后用 |
| 后置++ | `T operator++(int)` | 值（旧值） | 先用后增 |
| 前置-- | `T& operator--()` | 引用 | 先减后用 |
| 后置-- | `T operator--(int)` | 值（旧值） | 先用后减 |

## 二、具体用法

### 2.1 前置与后置实现

```cpp
#include <iostream>
using namespace std;

class Counter {
    int value;
public:
    Counter(int v = 0) : value(v) {}

    // 前置++：先增后用，返回引用
    Counter& operator++() {
        ++value;
        return *this;
    }

    // 后置++：先用后增，返回旧值（哑参数int区分）
    Counter operator++(int) {
        Counter old = *this;
        ++(*this);  // 复用前置++（避免代码重复）
        return old;
    }

    // 前置--
    Counter& operator--() {
        --value;
        return *this;
    }

    // 后置--
    Counter operator--(int) {
        Counter old = *this;
        --(*this);
        return old;
    }

    int getValue() const { return value; }

    friend ostream& operator<<(ostream& os, const Counter& c) {
        return os << c.value;
    }
};

int main() {
    Counter c(5);
    cout << "初始: " << c << endl;        // 5
    cout << "c++: " << c++ << endl;        // 输出5，然后c变成6
    cout << "之后: " << c << endl;         // 6
    cout << "++c: " << ++c << endl;        // c先变成7，输出7
    cout << "c--: " << c-- << endl;        // 输出7，然后c变成6
    cout << "之后: " << c << endl;         // 6
    cout << "--c: " << --c << endl;        // c先变成5，输出5

    return 0;
}
```

### 2.2 迭代器中的自增自减

```cpp
#include <iostream>
using namespace std;

class SimpleIter {
    int* ptr;
public:
    SimpleIter(int* p) : ptr(p) {}

    int& operator*() const { return *ptr; }

    // 前置++：返回引用，效率更高
    SimpleIter& operator++() {
        ++ptr;
        return *this;
    }

    // 后置++：返回旧值拷贝，效率较低
    SimpleIter operator++(int) {
        SimpleIter old = *this;
        ++ptr;
        return old;
    }

    bool operator!=(const SimpleIter& other) const {
        return ptr != other.ptr;
    }
};

int main() {
    int arr[] = {10, 20, 30, 40, 50};

    for (SimpleIter it = arr; it != SimpleIter(arr + 5); ++it) {
        cout << *it << " ";  // 10 20 30 40 50
    }
    cout << endl;

    return 0;
}
```

### 2.3 性能对比

```cpp
#include <iostream>
using namespace std;

class Heavy {
    int* data;
    size_t size;
public:
    Heavy(size_t s) : data(new int[s]), size(s) {}
    ~Heavy() { delete[] data; }

    // 前置++：无额外拷贝，返回引用
    Heavy& operator++() {
        for (size_t i = 0; i < size; ++i) ++data[i];
        return *this;
    }

    // 后置++：需要拷贝整个对象
    Heavy operator++(int) {
        Heavy old(*this);  // 拷贝（昂贵！）
        ++(*this);
        return old;
    }
};

int main() {
    // 前置++更高效：无拷贝开销
    // 循环中始终使用 ++i 而非 i++（除非需要后置语义）

    Heavy h(10000);
    ++h;     // 高效
    // h++;  // 低效（拷贝10000个int）

    cout << "前置++无拷贝开销" << endl;

    return 0;
}
```

## 三、注意事项与常见陷阱

1. **前置返回引用，后置返回值**：这是语义要求
2. **后置通常效率更低**：需要保存旧值的拷贝
3. **建议后置基于前置实现**：复用逻辑，避免代码重复
4. **迭代器类必须实现++和--运算符**：STL要求
5. **优先使用前置形式++i**：除非需要后置语义
6. **后置的哑参数int不参与计算**：仅用于区分重载，不需要参数名
7. **operator--的实现模式与operator++完全对称**
