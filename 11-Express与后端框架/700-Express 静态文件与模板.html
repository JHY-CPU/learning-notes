静态文件服务

// ========== express.static ========== // 内置中间件, 服务静态文件 (CSS/JS/图片) // ========== 基本使用 ========== // public/ 目录下的文件可以直接访问: app.use(express.static('public')); // public/style.css → http://localhost:3000/style.css // public/js/app.js → http://localhost:3000/js/app.js // public/images/logo.png → http://localhost:3000/images/logo.png // ========== 虚拟路径前缀 ========== // 添加路径前缀, 不暴露真实目录结构: app.use('/static', express.static('public')); // 现在访问: http://localhost:3000/static/style.css // ========== 多目录 ========== // 多个静态目录, 按顺序查找: app.use(express.static('public')); app.use(express.static('uploads')); // ========== 高级配置 ========== const path = require('path'); app.use('/static', express.static(path.join(__dirname, 'public'), { maxAge: '1d', // 缓存 1 天 etag: true, // ETag 支持 lastModified: true, // Last-Modified setHeaders: (res, filePath) => { // 对 CSS/JS 设置不同缓存: if (filePath.endsWith('.css')) { res.setHeader('Cache-Control', 'public, max-age=86400'); } if (filePath.endsWith('.js')) { res.setHeader('Cache-Control', 'public, max-age=604800'); } // 禁止 HTML 缓存: if (filePath.endsWith('.html')) { res.setHeader('Cache-Control', 'no-cache'); } }, dotfiles: 'ignore', // 隐藏文件 index: false // 禁用目录索引 })); // ========== 安全注意 ========== // ❌ 不要将 node_modules 暴露! // ❌ 不要将 .env 放在 public 目录 // ✅ 用 path.join(__dirname, 'public') 绝对路径 // ✅ 使用虚拟路径前缀隐藏真实路径

EJS 的 <%- 注意 XSS 风险, 只信任数据
    💡 静态文件要点: express.static() 配置缓存策略; 用虚拟路径前缀隐藏目录; 多目录时顺序查找; 不要暴露敏感文件。模板: EJS 最常用 (类 HTML); Pug 更简洁但缩进敏感; SSR 首屏快/SEO 好; 生产启用缓存; res.locals 传递全局模板变量。
